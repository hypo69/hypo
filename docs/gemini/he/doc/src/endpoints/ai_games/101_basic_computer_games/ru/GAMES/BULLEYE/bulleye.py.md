# Bulleye

## סקירה כללית

משחק "בולט" הוא משחק שבו השחקן מנסה לנחש את הקואורדינטות על מטרה. המטרה היא עיגול המחולק למספר אזורים, כל אחד מהם מביא לשחקן כמות שונה של נקודות. השחקן מזין קואורדינטות (x, y) ומקבל נקודות בהתאם לאיזה אזור במטרה הוא פגע. המשחק מסתיים כאשר השחקן צובר 100 נקודות או יותר.

## פונקציות

### `math`
הקוד משתמש במודול math כדי לבצע פעולות מתמטיות כמו חישוב שורש ריבועי.
### `input`
הקוד משתמש בפונקציה input כדי לקבל קלט מהמשתמש, כמו קואורדינטות.
### `float`
הקוד משתמש בפונקציה float כדי להמיר קלט משתמש למספרים עשרוניים.
### `print`
הקוד משתמש בפונקציה print כדי להציג הודעות למשתמש, כולל ניקוד משתמש ותוצאה סופית.

### `math.sqrt(x**2 + y**2)`

**Description**: הפונקציה מחשבת את המרחק מהנקודה (x, y) למרכז (0, 0).

**Parameters**:
- `x` (float): קואורדינטת ה-x.
- `y` (float): קואורדינטת ה-y.

**Returns**:
- `float`: המרחק מהנקודה (x, y) למרכז (0, 0).
