# POKER

## Обзор

Этот модуль реализует простую игру в покер для одного игрока против компьютера. Игрок начинает с 200$, может делать ставки до 10$ за раунд. Цель игры - получить более сильную покерную комбинацию, чем у компьютера, и накопить больше денег. Игра заканчивается, когда у игрока заканчиваются деньги или он решает прекратить играть.

## Функции

### `create_deck`

**Описание**: Создает и возвращает новую колоду карт.

**Параметры**:
- Нет параметров.

**Возвращает**:
- `list`: Список строк, представляющих карты в колоде (например, "AH" - туз червей).

### `deal_cards`

**Описание**: Раздает указанное количество карт из колоды.

**Параметры**:
- `deck` (list): Колода карт, из которой нужно раздавать.
- `num_cards` (int): Количество карт для раздачи.

**Возвращает**:
- `list`: Список карт, розданных из колоды.

### `evaluate_hand`

**Описание**: Оценивает покерную комбинацию руки и возвращает ее ранг.

**Параметры**:
- `hand` (list): Список карт в руке (например, ["AH", "KH", "QH", "JH", "10H"]).

**Возвращает**:
- `int`: Ранг комбинации (от 1 до 9, где 9 - наивысший ранг).
   - 1: High Card
   - 2: One Pair
   - 3: Two Pair
   - 4: Three of a Kind
   - 5: Straight
   - 6: Flush
   - 7: Full House
   - 8: Four of a Kind
   - 9: Straight Flush

### `get_hand_rank_name`

**Описание**: Получает ранг покерной комбинации и возвращает ее строковое название.

**Параметры**:
- `rank` (int): Ранг покерной комбинации (число от 1 до 9).

**Возвращает**:
- `str`: Строковое название покерной комбинации (например, "One Pair", "Straight Flush").

### `display_hand`

**Описание**: Выводит на экран карты, находящиеся в руке игрока, в виде строк.

**Параметры**:
- `hand` (list): Список карт в руке игрока.

**Возвращает**:
- `None`

### `format_card`

**Описание**: Форматирует карту для вывода на экран, заменяя обозначения мастей на символы.

**Параметры**:
- `card` (str): Строка, представляющая карту (например, "KH").

**Возвращает**:
- `str`: Отформатированная строка карты с символом масти (например, "K♥").

### `display_formatted_hand`

**Описание**: Отображает карты в руке игрока с использованием форматирования.

**Параметры**:
- `hand` (list): Список карт в руке.

**Возвращает**:
- `None`

### `play_poker`

**Описание**: Основная функция, запускающая игру в покер.

**Параметры**:
- Нет параметров.

**Возвращает**:
- `None`
