## Функции

### `get_troupe_data`

**Описание**: Функция для получения данных о труппе.

**Параметры**:
* `troupe_id` (str): Идентификатор труппы.
* `optional_param` (Optional[dict], optional): Дополнительные параметры. По умолчанию `None`.

**Возвращает**:
* `dict | None`: Данные о труппе в формате словаря, или `None`, если данные не найдены.

**Вызывает исключения**:
* `ValueError`: Если передан неверный тип данных для `troupe_id`.
* `APIError`: Если произошла ошибка при запросе к API.
* `ConnectionError`: Если произошла ошибка при подключении к серверу.


### `create_troupe`

**Описание**: Функция для создания новой труппы.

**Параметры**:
* `troupe_name` (str): Название труппы.
* `description` (str): Описание труппы.
* `members` (list[str]): Список имен членов труппы.

**Возвращает**:
* `str | None`: Идентификатор созданной труппы, или `None`, если создание не удалось.

**Вызывает исключения**:
* `ValueError`: Если передан неверный тип данных для параметров.
* `APIError`: Если произошла ошибка при запросе к API.
* `ConflictError`: Если уже существует труппа с таким названием.
