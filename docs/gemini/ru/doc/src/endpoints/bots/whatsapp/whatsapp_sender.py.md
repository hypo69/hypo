# Модуль для отправки сообщений WhatsApp

## Обзор

Этот модуль предоставляет функциональность для автоматической отправки сообщений WhatsApp с использованием библиотеки `pywhatkit` и `pyautogui`. Он позволяет отправлять заданное количество сообщений на указанный номер телефона в случайное время в пределах заданного временного интервала.

## Подробней

Модуль предназначен для автоматизации отправки сообщений через WhatsApp Web. Он использует `pywhatkit` для инициирования отправки сообщения и `pyautogui` для эмуляции нажатия клавиши Enter, чтобы отправить сообщение.
Дополнительно, модуль содержит банк сообщений с эмодзи, из которых случайным образом выбирается сообщение для отправки.

## Функции

### `send_whatsapp_message`

```python
def send_whatsapp_message(phone_number, message, hour, minutes):
    """Sends a WhatsApp message using pywhatkit and pyautogui."""
    ...
```

**Назначение**: Отправляет сообщение WhatsApp с использованием `pywhatkit` и `pyautogui`.

**Параметры**:
- `phone_number` (str): Номер телефона получателя с кодом страны.
- `message` (str): Текст сообщения для отправки.
- `hour` (int): Час отправки сообщения (0-23).
- `minutes` (int): Минута отправки сообщения (0-59).

**Возвращает**:
- `None`: Функция ничего не возвращает.

**Вызывает исключения**:
- `Exception`: Если происходит ошибка при отправке сообщения.

**Как работает функция**:

1. Функция использует `pywhatkit.sendwhatmsg` для планирования отправки сообщения через WhatsApp Web.
2.  Функция ожидает 20 секунд, чтобы дать время WhatsApp Web загрузиться.
3.  После загрузки WhatsApp Web, функция эмулирует нажатие клавиши Enter с помощью `pyautogui`, чтобы отправить сообщение.
4.  В случае возникновения ошибки, функция логирует ошибку с использованием `logger.error`.

**Примеры**:
```python
send_whatsapp_message('+79991234567', 'Привет!', 10, 30)
```
```python
send_whatsapp_message('+79991234567', 'Как дела?', 14, 0)
```

## Переменные модуля

- `messages` (list): Список сообщений для отправки.
- `phone_number` (str): Номер телефона получателя с кодом страны.
- `num_messages` (int): Количество сообщений для отправки.
- `start_hour` (int): Начальный час временного интервала для отправки сообщений (0-23).
- `end_hour` (int): Конечный час временного интервала для отправки сообщений (0-23).
- `message_count` (int): Счетчик отправленных сообщений.
- `random_hour` (int): Случайный час отправки сообщения в пределах заданного интервала.
- `random_minutes` (int): Случайная минута отправки сообщения.
- `random_message` (str): Случайное сообщение из списка `messages`.
