# AngularJS v1.8.2 Документация

## Обзор

Этот файл содержит минифицированный код библиотеки AngularJS версии 1.8.2.  Он реализует множество функций для работы с DOM, обработкой данных, шаблонизацией, маршрутизацией, HTTP-запросами и т.д.  Этот файл не содержит Python-кода, поэтому документация не включает описание функций в стиле Python. Вместо этого, данная документация предоставляет общее представление о возможностях и структуре AngularJS.
