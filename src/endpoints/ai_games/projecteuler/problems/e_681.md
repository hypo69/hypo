# Задача No 681
## Числа Кармайкла

## Условия
Число Кармайкла – это составное число `n`, которое удовлетворяет сравнению `b^(n-1) ≡ 1 (mod n)` для всех целых чисел `b`, взаимно простых с `n`.

Число Кармайкла можно проверить, разложив его на простые множители, и если оно удовлетворяет следующим условиям:
*  `n` свободно от квадратов (т.е. не делится ни на какой квадрат простого числа).
*  Для каждого простого множителя `p` числа `n`, `p-1` является делителем `n-1`.

Например, 561 = 3 * 11 * 17 является числом Кармайкла.

Пусть `C(n)` – n-е число Кармайкла (C(1) = 561).

Далее, пусть `S(n)` – сумма первых `n` чисел Кармайкла.

Например:
`C(1) = 561`
`C(2) = 1105`
`C(3) = 1729`
`C(10) = 10585`
`S(10) = 40099`

Найдите `S(10^6)`.
