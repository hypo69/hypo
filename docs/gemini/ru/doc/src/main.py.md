# Модуль `main`

## Обзор

Модуль `main` предоставляет интерактивное меню для запуска предопределенных скриптов.
Он принимает ввод пользователя для выбора и выполнения скриптов 1 или 2.

## Функции

### `script1`

**Описание**: Выполняет скрипт 1.

### `script2`

**Описание**: Выполняет скрипт 2.

### `show_help`

**Описание**: Выводит справочную информацию о доступных командах.

### `interactive_menu`

**Описание**: Интерактивное меню для выбора и запуска скриптов.

### `main`

**Описание**: Основная функция для обработки аргументов командной строки и запуска меню.

**Параметры**:
- `--help` (bool, optional): Выводит справочную информацию. По умолчанию `False`.
