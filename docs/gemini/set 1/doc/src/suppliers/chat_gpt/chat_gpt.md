# Модуль `hypotez/src/suppliers/chat_gpt/chat_gpt.py`

## Обзор

Этот модуль предоставляет класс `ChatGpt` для работы с данными чат-ботов, в частности, для получения HTML-представлений диалогов.

## Классы

### `ChatGpt`

**Описание**: Класс для работы с данными чат-ботов.

**Методы**:

- `yeld_conversations_htmls`


## Методы

### `yeld_conversations_htmls`

**Описание**:  Возвращает генератор HTML-файлов диалогов.

**Возвращает**:
- `str`:  Генерирует HTML-представления диалогов.


**Пример использования:**

```python
from hypotez.src.suppliers.chat_gpt.chat_gpt import ChatGpt

chat_gpt_instance = ChatGpt()
for html_file in chat_gpt_instance.yeld_conversations_htmls():
    # Обработка каждого HTML файла
    print(html_file)
```

**Детали реализации:**

1. Определяет путь к директории с диалогами (`conversation_directory`).
2. Использует метод `glob` для получения списка всех файлов с расширением `.html` в указанной директории.
3. Возвращает генератор, который поочередно предоставляет содержимое каждого найденного HTML-файла.

**Возможные исключения**:

-  Любые исключения, которые могут возникнуть при работе с файловой системой.
