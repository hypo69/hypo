# Модуль hypotez/src/suppliers/kualastyle/header.py

## Обзор

Этот модуль содержит функции для определения корневого каталога проекта и загрузки настроек проекта. Он также предоставляет доступ к различным переменным, связанным с проектом, таким как имя проекта, версия, документация и т.д.

## Функции

### `set_project_root`

**Описание**:  Определяет корневой каталог проекта, начиная с текущего каталога и двигаясь вверх по иерархии каталогов.  Ищет файлы-маркеры (pyproject.toml, requirements.txt, .git) для определения корня.

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или каталогов, используемых для определения корневого каталога проекта. По умолчанию (`pyproject.toml`, `requirements.txt`, `.git`).

**Возвращает**:

- `Path`: Путь к корневому каталогу проекта, если найден. В противном случае возвращает путь к текущему каталогу.

**Вызывает исключения**:
Нет.


## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта.

**Тип**: `Path`


### `settings`

**Описание**: Словарь настроек проекта, загруженный из файла `settings.json`.

**Тип**: `dict` или `None`


### `doc_str`

**Описание**: Строка с документацией проекта (README.MD).

**Тип**: `str` или `None`


### `__project_name__`

**Описание**: Имя проекта.  Получает значение из `settings`, если доступно, иначе используется значение по умолчанию - 'hypotez'.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта. Получает значение из `settings`, если доступно, иначе пустая строка.

**Тип**: `str`


### `__doc__`

**Описание**: Документация проекта (содержимое README.MD).  Получает значение из `doc_str`, если доступно, иначе пустая строка.

**Тип**: `str`


### `__details__`

**Описание**: Дополнительные детали проекта. Пустая строка по умолчанию.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта. Получает значение из `settings`, если доступно, иначе пустая строка.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права проекта. Получает значение из `settings`, если доступно, иначе пустая строка.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка на возможность поддержать разработчика.  Получает значение из `settings`, если доступно, иначе используется строка с ссылкой.

**Тип**: `str`


## Обработка исключений


Обработка исключений `FileNotFoundError` и `json.JSONDecodeError` выполняется для безопасной загрузки настроек и документации из файлов `settings.json` и `README.MD` соответственно. При возникновении исключений, соответствующие переменные (`settings`, `doc_str`) будут иметь значение `None`.
