# Модуль `hypotez/src/endpoints/advertisement/facebook/start_posting.py`

## Обзор

Модуль `start_posting.py` отвечает за запуск процесса отправки рекламных объявлений в группы Facebook. Он использует класс `FacebookPromoter` для управления этим процессом.

## Глобальные переменные

### `MODE`

Строковая переменная, хранящая режим работы (в данном случае 'dev').

### `filenames`

Список строк, представляющих пути к файлам с данными о группах Facebook.

### `excluded_filenames`

Список строк, содержащих пути к файлам, которые следует исключить из обработки.

### `campaigns`

Список строк, представляющих названия рекламных кампаний.

### `promoter`

Объект класса `FacebookPromoter`, используемый для отправки объявлений.

## Классы

### `Driver`

Описание класса `Driver` отсутствует. Требуется подробная документация для данного класса.

### `Chrome`

Описание класса `Chrome` отсутствует. Требуется подробная документация для данного класса.

### `FacebookPromoter`

Описание класса `FacebookPromoter` отсутствует. Требуется подробная документация для данного класса.

## Функции

### `run_campaigns`

**Описание**: Функция запускает рекламные кампании.

**Параметры**:
- `campaigns` (list): Список названий рекламных кампаний.
- `group_file_paths` (list): Список путей к файлам с данными о группах Facebook.

**Возвращает**:
- Нет значения возврата.

**Вызывает исключения**:
- Не определены потенциальные исключения.
