# Модуль `src.suppliers.hb.graber`

## Обзор

Модуль `src.suppliers.hb.graber` предназначен для сбора информации о товарах с веб-сайта `hb.co.il`. Он наследует функциональность базового класса `Graber` и предоставляет возможность переопределять методы для нестандартной обработки полей.

## Подробней

Этот модуль является частью системы для сбора данных о товарах от различных поставщиков. Он специализируется на поставщике `hb.co.il`. Для каждого поля товара на сайте `hb.co.il` предусмотрена функция обработки, которая может быть переопределена для нестандартной обработки. Также, модуль предоставляет возможность выполнения предварительных действий перед отправкой запроса к веб-драйверу с использованием декораторов.

## Классы

### `Graber`

**Описание**: Класс `Graber` предназначен для сбора данных о товарах с веб-сайта `hb.co.il`.

**Как работает класс**:
Класс наследуется от базового класса `Graber` (`Grbr`). В конструкторе класса устанавливается префикс поставщика (`hb`) и вызывается конструктор родительского класса. Также, устанавливаются глобальные настройки через `Context`.

**Методы**:
- `__init__`: Инициализация класса.

#### `__init__`

**Описание**: Инициализирует экземпляр класса `Graber`.

```python
def __init__(self, driver: Driver, lang_index)
```

**Параметры**:
- `driver` (`Driver`): Экземпляр веб-драйвера для управления браузером.
- `lang_index` (`int`): Индекс языка.

**Как работает функция**:
В конструкторе класса устанавливается префикс поставщика (`hb`) и вызывается конструктор родительского класса. Также, устанавливаются глобальные настройки через `Context`. `Context.locator_for_decorator = None` - указывает на то, что если будет установлено значение, то оно выполнится в декораторе `@close_pop_up`.

**Примеры**:
```python
from src.webdriver.driver import Driver
driver = Driver()
graber = Graber(driver, 0)
```
