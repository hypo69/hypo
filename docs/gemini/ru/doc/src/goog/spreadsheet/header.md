# Модуль hypotez/src/goog/spreadsheet/header.py

## Обзор

Данный модуль содержит функцию `set_project_root` для определения корневой директории проекта и переменные, содержащие информацию о проекте (название, версия, описание и т.д.).  Он загружает настройки из файла `settings.json` и документацию из `README.MD`.

## Функции

### `set_project_root`

**Описание**:  Функция находит корневую директорию проекта, начиная с текущего файла и двигаясь вверх по директориям, пока не найдет папку, содержащую один из указанных маркеров (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, которые указывают на корневую директорию проекта. По умолчанию - `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь до корневой директории проекта, если она найдена. В противном случае возвращает директорию, где расположен текущий скрипт.

**Вызывает исключения**:

- Нет.


## Переменные

### `__root__`

**Описание**:  Переменная, хранящая путь к корневой директории проекта, полученный с помощью функции `set_project_root()`.

**Тип**: `Path`

### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженные из файла `settings.json`.

**Тип**: `dict`

**Примечание**: Если файл `settings.json` не найден или содержит некорректный JSON, переменная `settings` останется `None`.

### `doc_str`

**Описание**: Строка, содержащая текст документации, загруженный из файла `README.MD`.

**Тип**: `str`

**Примечание**: Если файл `README.MD` не найден или при чтении возникла ошибка, переменная `doc_str` останется `None`.

### `__project_name__`

**Описание**: Название проекта.

**Тип**: `str`

**Значение по умолчанию**: `hypotez`

**Примечание**: Получает значение из `settings` или устанавливает значение по умолчанию.

### `__version__`

**Описание**: Версия проекта.

**Тип**: `str`

**Значение по умолчанию**: `''`

**Примечание**: Получает значение из `settings` или устанавливает значение по умолчанию.

### `__doc__`

**Описание**: Текст документации.

**Тип**: `str`

**Значение по умолчанию**: `''`

**Примечание**: Получает значение из `doc_str` или устанавливает значение по умолчанию.

### `__details__`

**Описание**: Дополнительные подробности о проекте.

**Тип**: `str`

**Значение по умолчанию**: `''`

### `__author__`

**Описание**: Автор проекта.

**Тип**: `str`

**Значение по умолчанию**: `''`

**Примечание**: Получает значение из `settings` или устанавливает значение по умолчанию.

### `__copyright__`

**Описание**: Авторские права на проект.

**Тип**: `str`

**Значение по умолчанию**: `''`

**Примечание**: Получает значение из `settings` или устанавливает значение по умолчанию.

### `__cofee__`

**Описание**: Ссылка на страницу пожертвования чашки кофе для разработчика.

**Тип**: `str`
