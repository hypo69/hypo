# Модуль `src.endpoints.hypo69`

## Обзор

Модуль предоставляет эндпоинты для разработчика, включающие в себя: `small_talk_bot` (бот с чатом модели ИИ), `code_assistant` (модуль обучения модели коду проекта) и `psychologist_bot` (ранняя разработка модуля парсинга диалогов).

## Подробнее

Этот модуль содержит основные эндпоинты, используемые для различных задач, таких как общение с ИИ, помощь в разработке кода и анализ диалогов. Он является частью проекта `hypotez` и предоставляет разработчикам инструменты для взаимодействия с различными компонентами системы.
