### `parse_product(product)`

**Описание:**

Обрабатывает объект `product` и преобразует его в другой формат.  Функция извлекает значения из атрибута `product_small_image_urls` и преобразует его в строковый тип.

**Аргументы:**

* `product`: Объект, представляющий данные о продукте. Ожидается, что объект содержит атрибут `product_small_image_urls`.

**Возвращаемое значение:**

Объект `product`, преобразованный  с измененным `product_small_image_urls`.


### `parse_products(products)`

**Описание:**

Обрабатывает список объектов `products` и преобразует каждый элемент списка с помощью функции `parse_product()`.

**Аргументы:**

* `products`: Список объектов `product`.

**Возвращаемое значение:**

Новый список `new_products`, содержащий обработанные объекты `product`.
