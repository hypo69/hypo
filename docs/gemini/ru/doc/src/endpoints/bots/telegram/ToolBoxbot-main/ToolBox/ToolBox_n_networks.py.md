# Модуль `ToolBox_n_networks`

## Обзор

Модуль `ToolBox_n_networks` предоставляет класс `neural_networks`, который включает в себя методы для взаимодействия с различными нейронными сетями, такими как FLUX.1-schnell, Mistral Large и GPT-4o-mini. Он позволяет генерировать изображения на основе текстовых запросов и получать текстовые ответы от языковых моделей.

## Классы

### `neural_networks`

**Описание**: Класс, предоставляющий методы для взаимодействия с различными нейронными сетями.

#### Методы

##### `_FLUX_schnell`

**Описание**: Метод для отправки запроса к модели FLUX.1-schnell для генерации изображения на основе текстового описания.

**Параметры**:
- `prompt` (str): Текстовое описание для генерации изображения.
- `size` (list[int, int]): Список, содержащий ширину и высоту изображения.
- `seed` (int): Зерно для генерации изображения.
- `num_inference_steps` (int): Количество шагов инференса.

**Возвращает**:
- `str | None`: Возвращает объект `PIL.Image` или `None`, если запрос не удался.

##### `__mistral_large_2407`

**Описание**: Метод для отправки запроса к модели Mistral Large для генерации текстового ответа на основе списка сообщений.

**Параметры**:
- `prompt` (list[dict[str, str]]): Список словарей, каждый из которых представляет сообщение.

**Возвращает**:
- `tuple[str, int, int] | str`: Возвращает кортеж, содержащий текстовый ответ, количество токенов в запросе и количество токенов в ответе. Возвращает строку в случае ошибки.

##### `_free_gpt_4o_mini`

**Описание**: Метод для отправки запроса к модели GPT-4o-mini для генерации текстового ответа на основе списка сообщений. В случае неудачи запроса к GPT-4o-mini использует `__mistral_large_2407`.

**Параметры**:
- `prompt` (list[dict[str, str]]): Список словарей, каждый из которых представляет сообщение.

**Возвращает**:
- `tuple[str, int, int] | str`: Возвращает кортеж, содержащий текстовый ответ, количество токенов в запросе и количество токенов в ответе. Возвращает строку в случае ошибки.
