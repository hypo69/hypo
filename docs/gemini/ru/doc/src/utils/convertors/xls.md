# Модуль hypotez/src/utils/convertors/xls.py

## Обзор

Этот модуль предоставляет функции для работы с файлами в формате XLS. Он содержит функции для чтения данных из XLS-файлов в формате словаря и сохранения данных в XLS-файл.

## Функции

### `xls2dict`

**Описание**: Функция `xls2dict` читает XLS-файл и возвращает данные в формате словаря.

**Параметры**:

- `xls_file` (str | Path): Путь к файлу XLS.

**Возвращает**:

- `dict | None`: Словарь с данными из файла XLS или `None`, если произошла ошибка.

**Вызывает исключения**:

- Любые исключения, которые могут быть возбуждены функцией `read_xls_as_dict` из модуля `src.utils.xls`.
