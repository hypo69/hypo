## Введение

В данном документе описывается, как запустить и выполнить тесты, проверяющие функциональность классов `Driver` и `ExecuteLocator`, находящихся в файле `test_driver_executor.py`.  Тесты охватывают различные сценарии использования и взаимодействия между этими классами.


## Структура тестов

Файл `test_driver_executor.py` содержит тесты, нацеленные на проверку корректности работы методов классов `Driver` и `ExecuteLocator`.  Тесты покрывают различные аспекты функциональности, включая навигацию, получение веб-элементов, отправку сообщений, взаимодействие с элементами и обработку исключений.


## Тестируемые методы и функции

- `test_navigate_to_page`: Проверка корректной загрузки указанной страницы.
- `test_get_webelement_by_locator_single_element`: Проверка корректного возврата элемента по локатору.
- `test_get_webelement_by_locator_no_element`: Проверка обработки случаев, когда элемент не найден.
- `test_send_message`: Проверка корректной отправки сообщения элементу.
- `test_get_attribute_by_locator`: Проверка корректного получения атрибута элемента.
- `test_execute_locator_event`: Проверка выполнения события на локаторе.
- `test_get_locator_keys`: Проверка возврата корректных ключей локатора.
- `test_navigate_and_interact`: Проверка последовательности навигации и взаимодействия с элементами на другой странице.
- `test_invalid_locator`: Проверка обработки некорректных локаторов и выброса соответствующего исключения.


## Запуск тестов

### Установка зависимостей

Перед запуском тестов убедитесь, что у вас установлены все необходимые зависимости.  Выполните команду:

```bash
pip install -r requirements.txt
```

Убедитесь, что в `requirements.txt` указаны все необходимые пакеты, включая `pytest`, `selenium` и др.


### Настройка WebDriver

Тесты используют Chrome WebDriver. Убедитесь, что у вас установлен ChromeDriver и укажите его путь в вашем коде:

```python
from selenium import webdriver
from selenium.webdriver.chrome.service import Service

service = Service(executable_path="/path/to/chromedriver") # Укажите корректный путь
driver = webdriver.Chrome(service=service)
```


### Запуск тестов

Для запуска тестов используйте команду:

```bash
pytest src/webdriver/_pytest/test_driver_executor.py
```

Эта команда запустит все тесты, определённые в файле `test_driver_executor.py`.

## Отчеты о тестах

`pytest` генерирует отчет с результатами тестирования. Для получения более подробного отчета используйте флаги:

- **Подробный текстовый отчет:**
```bash
pytest src/webdriver/_pytest/test_driver_executor.py -v
```

- **HTML-отчет:** (требуется `pytest-html`)
```bash
pip install pytest-html
pytest src/webdriver/_pytest/test_driver_executor.py --html=report.html
```
Это создаст HTML-отчёт в файле `report.html`.
