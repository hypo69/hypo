# Документация для `test_jupyter_examples.py`

## Обзор

Файл `test_jupyter_examples.py` содержит тесты для автоматического запуска и проверки Jupyter Notebooks. Он использует `pytest` для параметризации тестов и `nbformat` и `nbconvert` для обработки и запуска ноутбуков. Основная цель - убедиться, что все примеры ноутбуков выполняются без ошибок.

## Функции

### `get_notebooks`

**Описание**: Извлекает все файлы Jupyter Notebook из указанной папки.

**Параметры**:
- `folder` (str): Путь к папке, содержащей ноутбуки.

**Возвращает**:
- `list`: Список путей к файлам Jupyter Notebook в указанной папке.

### `test_notebook_execution`

**Описание**: Выполняет Jupyter Notebook и проверяет, что не возникают исключения.

**Параметры**:
- `notebook_path` (str): Путь к файлу Jupyter Notebook.

**Возвращает**:
- `None`

**Вызывает исключения**:
- `pytest.fail`: Если во время выполнения ноутбука возникает исключение.
