# Документация для Flask приложения

## Обзор

Данный модуль содержит минимальную конфигурацию для создания Flask-приложения. Он инициализирует приложение Flask и указывает путь к папке с шаблонами HTML.

## Подробней

Этот код создает экземпляр Flask-приложения, который будет использоваться для обработки HTTP-запросов и рендеринга веб-страниц. Путь к папке с шаблонами указан относительно текущего файла.

## Функции

### `Flask(__name__, template_folder='./../client/html')`

```python
app = Flask(__name__, template_folder='./../client/html')
```

**Назначение**: Инициализация Flask-приложения.

**Параметры**:
- `__name__` (str): Имя текущего модуля. Используется Flask для определения корневой директории приложения.
- `template_folder` (str): Путь к папке, содержащей HTML-шаблоны. В данном случае, папка находится на один уровень выше (`./../`) в директории `client` в подпапке `html`.

**Возвращает**:
- `Flask`: Экземпляр Flask-приложения.

**Как работает функция**:

1.  Инициализация Flask-приложения: Создается экземпляр класса `Flask`, который принимает имя текущего модуля и путь к папке с шаблонами.
2.  Установка пути к шаблонам: Указывается папка, где Flask будет искать HTML-шаблоны для рендеринга веб-страниц.

```
Инициализация Flask приложения
│
└─── Установка пути к шаблонам
```

**Примеры**:

```python
from flask import Flask

app = Flask(__name__, template_folder='./../client/html')
