# Модуль `printer`

## Обзор

Этот модуль предоставляет функции для красивого вывода данных в формате человека и стилизации текста, включая цвета, фоны и шрифты.


## Функции

### `_color_text`

**Описание**: Функция для применения стилей цвета, фона и шрифта к тексту. Использует ANSI escape коды для стилизации.

**Параметры**:
- `text` (str): Текст, которому нужно применить стили.
- `text_color` (str, optional): Цвет текста. По умолчанию пустая строка (без цвета). Доступные цвета из словаря `TEXT_COLORS`.
- `bg_color` (str, optional): Цвет фона. По умолчанию пустая строка (без фона). Доступные цвета из словаря `BG_COLORS`.
- `font_style` (str, optional): Стили шрифта. По умолчанию пустая строка (без стилей). Доступные стили из словаря `FONT_STYLES`.

**Возвращает**:
- `str`: Стилизованный текст.

**Примеры**:
```python
_color_text("Hello, World!", text_color="green", font_style="bold")
```
Возвращает: `'\033[1m\033[32mHello, World!\033[0m'`

### `pprint`

**Описание**: Функция для красивого вывода данных с опциональными стилями цвета, фона и шрифта. Форматирует входные данные в зависимости от их типа и выводит их в консоль.

**Параметры**:
- `print_data` (Any, optional): Данные для вывода. Может быть `None`, `dict`, `list`, `str` или `Path`.
- `text_color` (str, optional): Цвет текста. По умолчанию "white".  См. словарь `TEXT_COLORS`.
- `bg_color` (str, optional): Цвет фона. По умолчанию пустая строка (нет фона). См. словарь `BG_COLORS`.
- `font_style` (str, optional): Стили шрифта. По умолчанию пустая строка (нет стилей). См. словарь `FONT_STYLES`.


**Возвращает**:
- `None`

**Вызывает исключения**:
- `Exception`: Если тип данных не поддерживается или при выводе произошла ошибка.


**Примеры**:

```python
pprint({"name": "Alice", "age": 30}, text_color="green")
pprint(["apple", "banana", "cherry"], text_color="blue", font_style="bold")
pprint("text example", text_color="yellow", bg_color="bg_red", font_style="underline")
```

##  Словари констант

### `TEXT_COLORS`

Словарь, содержащий ANSI escape коды для различных цветов текста.


### `BG_COLORS`

Словарь, содержащий ANSI escape коды для различных цветов фона.


### `FONT_STYLES`

Словарь, содержащий ANSI escape коды для различных стилей шрифта.


### `RESET`

Строка, содержащая ANSI escape код для сброса стилей.


##  Примечание

Функция `pprint` умеет обрабатывать различные типы данных, включая `dict`, `list`, `str`, и файлы (проверяя расширения `.csv` и `.xls`).  Она также умеет обрабатывать `None`.
