# Модуль для извлечения ссылок на чаты с ChatGPT

## Обзор

Модуль `grab_lilnks_to_chats.py` предназначен для автоматического извлечения ссылок на отдельные чаты с платформы ChatGPT. Он использует веб-драйвер для навигации по сайту и поиска необходимых элементов, определенных в файле локаторов.

## Подробней

Этот модуль является частью проекта `hypotez` и используется для автоматизации сбора данных с ChatGPT. Он позволяет получить список ссылок на чаты, что может быть полезно для дальнейшего анализа или обработки. Модуль использует драйверы `Chrome` или `Firefox` для взаимодействия с веб-страницей, а также файл локаторов (`chats_list.json`) для определения элементов, содержащих ссылки на чаты.

## Функции

### `get_links`

```python
def get_links(d: Driver) -> list:
    """
    Извлекает ссылки на отдельные чаты с платформы ChatGPT, используя предоставленный веб-драйвер.

    Args:
        d (Driver): Экземпляр веб-драйвера, который будет использоваться для поиска элементов на странице.

    Returns:
        list: Список веб-элементов, представляющих ссылки на отдельные чаты.

    Raises:
        Exception: Если возникает ошибка при выполнении локатора или при поиске элементов на странице.
    """
```

**Как работает функция**:

1.  Функция `get_links` принимает объект `Driver` в качестве аргумента. Этот объект представляет собой веб-драйвер, который используется для взаимодействия с веб-страницей.
2.  Функция вызывает метод `execute_locator` объекта `Driver`, передавая ему локатор `locator.link`. Этот локатор определяет, как найти элементы, содержащие ссылки на чаты, на веб-странице.
3.  Метод `execute_locator` возвращает список веб-элементов, соответствующих локатору.
4.  Функция возвращает полученный список веб-элементов.

### Внутренние функции:

В данной функции внутренних функций нет.

**Примеры**:

```python
from src.webdriver.driver import Driver
from src.webdriver.firefox import Firefox
from src.utils.jjson import j_loads_ns
from src import gs

locator = j_loads_ns(gs.path.src / 'suppliers' / 'chat_gpt' / 'locators' / 'chats_list.json')
d = Driver(Firefox)
d.get_url('https://chatgpt.com/')
links = get_links(d)
if links:
    print(f'Найдено {len(links)} ссылок на чаты.')
else:
    print('Ссылки на чаты не найдены.')
```

## Главный блок `if __name__ == '__main__'`

В главном блоке выполняется следующий код:

1.  Создается экземпляр веб-драйвера `Firefox` с помощью `d = Driver(Firefox)`.
2.  Открывается URL `https://chatgpt.com/` с помощью `d.get_url('https://chatgpt.com/')`.
3.  Вызывается функция `get_links(d)` для получения списка ссылок на чаты.
4.  Результат сохраняется в переменной `links`.
5.  Выводятся результаты.
