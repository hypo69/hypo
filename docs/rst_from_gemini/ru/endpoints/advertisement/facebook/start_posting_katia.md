# start_posting_katia.py

Файл: `hypotez/src/endpoints/advertisement/facebook/start_posting_katia.py`

**Описание:**

Этот скрипт отвечает за отправку рекламных объявлений в группы Facebook. Использует фреймворк для работы с браузером, а также класс `FacebookPromoter` для управления рекламной кампанией.

**Константы:**

* `MODE = 'debug'`:  Устанавливает режим работы скрипта (debug).

**Импорты:**

* `header`:  Предполагается, что это модуль, содержащий общие заголовки или настройки.
* `Driver`, `Chrome`: Классы из модуля `webdriver` для работы с браузером Chrome.
* `FacebookPromoter`:  Класс из модуля `src.endpoints.advertisement.facebook`, отвечающий за реализацию логики создания и запуска рекламных кампаний.
* `logger`: Модуль для логирования.

**Логика работы:**

1. **Инициализация:**
   - Создается экземпляр `Driver` с использованием `Chrome`.
   - Открывается сайт facebook.com.
   - Определяются файлы с данными о группах (`filenames`).
   - Определяются названия рекламных кампаний (`campaigns`).
   - Создается экземпляр класса `FacebookPromoter`, принимающий драйвер и пути к файлам с данными о группах.
   - В конструктор FacebookPromoter передается `no_video=False`, что подразумевает, что планируется использовать видео.


2. **Запуск рекламных кампаний:**
   - Вызывается метод `promoter.run_campaigns(campaigns)`, который инициирует процесс создания и запуска объявлений для всех указанных кампаний.

3. **Обработка прерываний:**
   - `try...except KeyboardInterrupt`:  Блок обработки прерываний, позволяющий корректно завершить выполнение скрипта при нажатии Ctrl+C.  В этом случае выводится сообщение в лог.


**Возможные улучшения:**

* **Уточнить структуру данных:**  Описание данных в `filenames` (например, формата файлов `.json`) и `campaigns`.
* **Детализация процесса `run_campaigns`:** Добавить описание внутренних действий метода `run_campaigns`, например, последовательности создания, публикации и мониторинга объявлений.
* **Обработка ошибок:** Добавить обработку более широкого спектра исключений (например, `ConnectionError`, `TimeoutException`).
* **Параметризация:** Разделить константы `filenames` и `campaigns` на отдельные файлы или использовать конфигурационный файл.
* **Изменение режима:** Добавьте возможность изменить режим (`MODE`) на 'production' и корректировать поведение программы в соответствии с этим.
* **Задержка между объявлениями:** Возможно, необходимо добавить временные задержки, чтобы избежать перегрузки сервера Facebook.
* **Обработка результатов:** Добавить логику обработки результатов рекламной кампании (статистика, отчёты и т.д.).
* **Валидация данных:** Проверить корректность данных, полученных из файлов.
* **Документация методов:** Документировать методы класса `FacebookPromoter`.


**Использование:**

Запуск скрипта выполняется стандартно через интерпретатор Python.

```bash
python start_posting_katia.py
```
