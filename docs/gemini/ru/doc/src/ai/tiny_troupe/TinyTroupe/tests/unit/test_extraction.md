# Модуль `test_extraction.py`

## Обзор

Этот модуль содержит тесты для модуля `extraction` проекта `tiny_troupe`.  Тесты проверяют корректность экспорта артефактов в различные форматы (JSON, текст, docx) и работу нормализации концептов.

## Фикстуры

### `exporter`

**Описание**: Фикстура, возвращающая экземпляр класса `ArtifactExporter` с заданным базовым каталогом для экспорта.

**Аргументы**:

-  Нет аргументов.

**Возвращает**:
-  `ArtifactExporter`: Экземпляр класса `ArtifactExporter`.


## Тесты

### `test_export_json`

**Описание**: Тест, проверяющий экспорт артефакта в формате JSON.

**Аргументы**:
- `exporter`: Экземпляр класса `ArtifactExporter`.

**Возвращает**:
-  Нет возвращаемого значения.

-  Нет исключений.

**Описание**: Тест, проверяющий экспорт артефакта в формате TXT.

**Аргументы**:
- `exporter`: Экземпляр класса `ArtifactExporter`.

**Возвращает**:
-  Нет возвращаемого значения.

**Аргументы**:
- `exporter`: Экземпляр класса `ArtifactExporter`.

**Возвращает**:
-  Нет возвращаемого значения.

### `test_normalizer`

**Описание**: Тест, проверяющий работу нормализатора концептов. Проверяет корректное создание нормализованных элементов и заполнение `normalizing_map`.

**Аргументы**:
-  Нет аргументов.

**Возвращает**:
-  Нет возвращаемого значения.

**Вызывает исключения**:
- Нет исключений.


## Функционал модуля `extraction`

Этот модуль предоставляет инструменты для экспорта артефактов и нормализации концептов.  Протестированны методы экспорта в json, txt и docx, а также нормализация наборов концептов.


## Использование

Для запуска тестов необходимо установить необходимые библиотеки (pytest, docx) и запустить pytest из командной строки в корне проекта:

```bash
pip install -r requirements.txt
pytest
```
