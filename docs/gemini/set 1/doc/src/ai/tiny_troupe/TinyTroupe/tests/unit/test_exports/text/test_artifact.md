# Модуль test_artifact.txt

## Обзор

Этот файл содержит тесты для модуля экспорта артефактов. Он предоставляет набор тестов, проверяющих корректность работы функций экспорта в формате текстового файла.

## Тесты

### `test_export_artifact`

**Описание**: Этот тест проверяет экспорт текстового артефакта.

**Параметры**:

- `input_data`: (dict): Входные данные для экспорта.
- `expected_output`: (str): Ожидаемый результат экспорта.

**Возвращает**:

- `None`:  Функция не возвращает значение.

**Вызывает исключения**:

- `ValueError`: Возникает в случае некорректных входных данных.
- `IOError`: Возникает в случае ошибок при работе с файлом.
