# Модуль `header`

## Обзор

Модуль `header` содержит общие настройки и переменные для проекта. Он определяет путь к корневой директории проекта, загружает настройки из `settings.json` и документацию из `README.MD`, а также предоставляет общие переменные, такие как имя проекта, версия, автор и т.д.

## Функции

### `set_project_root`

**Описание**:
Находит корневую директорию проекта, начиная с директории текущего файла, и ищет вверх по дереву каталогов, пока не найдет директорию, содержащую один из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple, optional): Кортеж имен файлов или каталогов, которые идентифицируют корневую директорию проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена. В противном случае возвращает директорию, в которой находится скрипт.

## Переменные

### `MODE`

**Описание**:
Режим работы приложения. По умолчанию равен `dev`.

### `__root__`

**Описание**:
Путь к корневой директории проекта. Устанавливается функцией `set_project_root`.

### `settings`

**Описание**:
Словарь настроек проекта, загруженных из файла `settings.json`. Может быть `None`, если файл не найден или содержит ошибки JSON.

### `doc_str`

**Описание**:
Строка с документацией, загруженная из файла `README.MD`. Может быть `None`, если файл не найден или содержит ошибки.

### `__project_name__`

**Описание**:
Имя проекта. По умолчанию 'hypotez'. Если в `settings.json` присутствует ключ `project_name`, то используется его значение.

### `__version__`

**Описание**:
Версия проекта. По умолчанию ''. Если в `settings.json` присутствует ключ `version`, то используется его значение.

### `__doc__`

**Описание**:
Документация проекта, загруженная из файла `README.MD`.

### `__details__`

**Описание**:
Детали проекта. Изначально пустая строка.

### `__author__`

**Описание**:
Автор проекта. Если в `settings.json` присутствует ключ `author`, то используется его значение. В противном случае пустая строка.

### `__copyright__`

**Описание**:
Информация об авторских правах проекта. Если в `settings.json` присутствует ключ `copyrihgnt`, то используется его значение. В противном случае пустая строка.

### `__cofee__`

**Описание**:
Строка с призывом поддержать разработчика. Если в `settings.json` присутствует ключ `cofee`, то используется его значение. В противном случае используется значение по умолчанию.
