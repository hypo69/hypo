# Локаторы для страницы входа Facebook

## Описание

Этот файл `login.json` содержит определения локаторов для элементов страницы входа в Facebook. Локаторы используются для поиска и взаимодействия с элементами на веб-странице во время автоматизированного тестирования или веб-скрейпинга.

## Обзор

Файл `login.json` содержит JSON-объект, который определяет локаторы для элементов на странице входа в Facebook. Каждый локатор включает в себя информацию о способе поиска элемента (`by`), селекторе (`selector`), дополнительных параметрах, таких как `if_list`, `use_mouse`, `timeout`, `timeout_for_event`, `event`, `mandatory`, и `locator_description`. Эти локаторы предназначены для автоматизации действий на странице входа в Facebook, таких как ввод электронной почты, пароля и нажатие кнопки входа.

## Локаторы

### `email`

**Описание**: Локатор для поля ввода электронной почты или номера телефона пользователя.

**Параметры**:

-   `attribute`: `null` - атрибут элемента не используется.
-   `by`: `"XPATH"` - метод поиска элемента по XPath.
-   `selector`: `"//input[@name = 'email']"` - XPath-селектор для поиска элемента.
-   `if_list`: `"first"` - если найдено несколько элементов, используется первый.
-   `use_mouse`: `false` - не используется взаимодействие с мышью.
-   `timeout`: `0` - таймаут для ожидания элемента равен 0 секундам.
-   `timeout_for_event`: `"presence_of_element_located"` - событие ожидания - присутствие элемента на странице.
-   `event`: `"click()"` - событие для активации элемента - клик.
-   `mandatory`: `true` - поле обязательно для заполнения.
-   `locator_description`: `"user email or phone"` - описание локатора.

### `password`

-   `if_list`: `"first"` - если найдено несколько элементов, используется первый.
-   `use_mouse`: `[false, false]` - не используется взаимодействие с мышью.
-   `timeout`: `0` - таймаут для ожидания элемента равен 0 секундам.
-   `timeout_for_event`: `"presence_of_element_located"` - событие ожидания - присутствие элемента на странице.

-   `mandatory`: `[true, true]` - поле обязательно для заполнения.
-   `locator_description`: `"user email or phone"` - описание локатора.

-   `use_mouse`: `false` - не используется взаимодействие с мышью.
-   `timeout`: `0` - таймаут для ожидания элемента равен 0 секундам.
-   `timeout_for_event`: `"presence_of_element_located"` - событие ожидания - присутствие элемента на странице.

-   `mandatory`: `true` - кнопка обязательна для взаимодействия.
