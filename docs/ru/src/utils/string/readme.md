## Обзор

Модуль предоставляет удобные утилиты для нормализации и обработки данных. Он может быть использован для:
- Удаления HTML-тегов из строк.
- Преобразования строк в числовые или логические значения.
- Очистки строк от специальных символов.
- Преобразования списков строк в одну нормализованную строку.

---

## Функции модуля

### `normalize_boolean`

**Назначение**:
Преобразует входное значение в логическое значение.

**Параметры**:
- `input_data` (Any): Данные, которые могут представлять логическое значение (строка, число, логический тип).

**Возвращает**:
- `bool`: Преобразованное логическое значение.

**Пример**:
```python
normalize_boolean('yes')  # Результат: True
normalize_boolean(0)      # Результат: False
```

**Как работает функция**:

1. Функция принимает входные данные (`input_data`) любого типа.
2. Преобразует эти данные в логическое значение (`True` или `False`).

```ascii
Входные данные (input_data)
    ↓
Преобразование в bool
    ↓
Возвращает bool
```

### `normalize_string`

**Назначение**:
Преобразует строку или список строк в нормализованную строку, удаляя лишние пробелы, HTML-теги и специальные символы.

**Параметры**:
- `input_data` (str | list): Строка или список строк.

**Возвращает**:
- `str`: Очищенная строка в кодировке UTF-8.

**Пример**:
```python
normalize_string(['  Example string  ', '<b>with HTML</b>'])  # Результат: 'Example string with HTML'
```

**Как работает функция**:

1. Функция принимает строку или список строк (`input_data`).
2. Удаляет лишние пробелы в начале и конце каждой строки.
3. Удаляет HTML-теги из строк.
4. Удаляет специальные символы из строк.
5. Возвращает очищенную строку в кодировке UTF-8.

```ascii
Входные данные (input_data)
    ↓
Удаление лишних пробелов
    ↓
Удаление HTML-тегов
    ↓
Удаление специальных символов
    ↓
Возвращает очищенную строку UTF-8
```

### `normalize_int`

**Назначение**:
Преобразует входное значение в целое число.

**Параметры**:
- `input_data` (str | int | float | Decimal): Число или его строковое представление.

**Возвращает**:
- `int`: Преобразованное целое число.

**Пример**:
```python
normalize_int('42')  # Результат: 42
normalize_int(3.14)  # Результат: 3
```

**Как работает функция**:

1. Функция принимает число или его строковое представление (`input_data`).
2. Преобразует это значение в целое число.

```ascii
Входные данные (input_data)
    ↓
Преобразование в int
    ↓
Возвращает int
```

### `normalize_float`

**Назначение**:
Преобразует входное значение в число с плавающей точкой.

**Параметры**:
- `value` (Any): Число, строка или список чисел.

**Возвращает**:
- `float | List[float] | None`: Число с плавающей точкой, список чисел с плавающей точкой или `None` в случае ошибки.

**Пример**:
```python
normalize_float('3.14')         # Результат: 3.14
normalize_float([1, '2.5', 3])  # Результат: [1.0, 2.5, 3.0]
```

**Как работает функция**:

1. Функция принимает число, строку или список чисел (`value`).
2. Преобразует это значение в число с плавающей точкой или список чисел с плавающей точкой.
3. Возвращает результат или `None` в случае ошибки.

```ascii
Входные данные (value)
    ↓
Преобразование в float
    ↓
Возвращает float или List[float] или None
```

### `remove_line_breaks`

**Назначение**:
Удаляет символы новой строки из строки.

**Параметры**:
- `input_str` (str): Входная строка.

**Возвращает**:
- `str`: Строка без символов новой строки.

**Пример**:
```python
remove_line_breaks('String\nwith line breaks\r')  # Результат: 'String with line breaks'
```

**Как работает функция**:

1. Функция принимает строку (`input_str`).
2. Удаляет символы новой строки (`\n` и `\r`) из строки.
3. Возвращает строку без символов новой строки.

```ascii
Входная строка (input_str)
    ↓
Удаление символов новой строки
    ↓
Возвращает строку без символов новой строки
```

remove_html_tags('<p>Example text</p>')  # Результат: 'Example text'

**Как работает функция**:

2. Удаляет HTML-теги из строки.
3. Возвращает строку без HTML-тегов.

```ascii
Входная строка с HTML-тегами (input_html)
    ↓
Удаление HTML-тегов
    ↓
Возвращает строку без HTML-тегов
```

### `remove_special_characters`

**Назначение**:
Удаляет специальные символы из строки или списка строк.

**Параметры**:
- `input_str` (str | list): Строка или список строк.

**Возвращает**:
- `str | list`: Строка или список строк без специальных символов.

**Пример**:
```python
remove_special_characters('Hello@World!')  # Результат: 'HelloWorld'
```

**Как работает функция**:

1. Функция принимает строку или список строк (`input_str`).
2. Удаляет специальные символы из строки или списка строк.
3. Возвращает строку или список строк без специальных символов.

```ascii
Входная строка или список строк (input_str)
    ↓
Удаление специальных символов
    ↓
Возвращает строку или список строк без специальных символов
```

### `normalize_sql_date`

**Назначение**:
Преобразует строку или объект datetime в стандартный формат SQL-даты (`YYYY-MM-DD`).

**Параметры**:
- `input_data` (str | datetime): Строка или объект datetime, представляющий дату.

**Возвращает**:
- `str`: Нормализованная SQL-дата в виде строки в формате `YYYY-MM-DD`.

**Пример**:
```python
normalize_sql_date('2024-12-06')  # Результат: '2024-12-06'
normalize_sql_date(datetime(2024, 12, 6))  # Результат: '2024-12-06'
```

**Как работает функция**:

1. Функция принимает строку или объект datetime (`input_data`).
2. Преобразует дату в формат `YYYY-MM-DD`.
3. Возвращает нормализованную SQL-дату в виде строки.

```ascii
Входная строка или объект datetime (input_data)
    ↓
Преобразование в формат YYYY-MM-DD
    ↓
Возвращает нормализованную SQL-дату
```

## Пример использования

```python
from src.utils.string.normalizer import normalize_string, normalize_boolean, normalize_int, normalize_float, normalize_sql_date

# Нормализация строки
clean_str = normalize_string(['<h1>Header</h1>', '  text with spaces  '])
print(clean_str)  # 'Header text with spaces'

# Нормализация логического значения
is_active = normalize_boolean('Yes')
print(is_active)  # True

# Нормализация целого числа
integer_value = normalize_int('42')
print(integer_value)  # 42

# Нормализация числа с плавающей точкой
float_value = normalize_float('3.14159')
print(float_value)  # 3.14159

# Нормализация SQL-даты
sql_date = normalize_sql_date('2024-12-06')
print(sql_date)  # '2024-12-06'
```

## Логирование

Все ошибки и предупреждения логируются через `logger`:
- Ошибки логируются с использованием `logger.error`.
- Неожиданные значения логируются с использованием `logger.debug` или `logger.warning`.
