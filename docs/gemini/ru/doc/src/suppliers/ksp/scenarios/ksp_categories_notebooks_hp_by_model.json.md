# Документация для `ksp_categories_notebooks_hp_by_model.json`

## Обзор

Файл `ksp_categories_notebooks_hp_by_model.json` содержит конфигурационные данные для сценариев, определяющих соответствие моделей ноутбуков HP категориям PrestaShop на основе URL-адресов KSP. Каждый сценарий включает информацию о бренде, URL, параметрах фильтрации, а также о соответствующих категориях PrestaShop.

## Структура данных

### Сценарии

Словарь `scenarios` содержит наборы данных для каждого конкретного сценария. Ключом является название модели ноутбука (например, `"HP Laptop 14 I7"`).

#### Пример сценария

```json
"HP Laptop 14 I7": {
  "brand": "HP",
  "url": "https://ksp.co.il/web/cat/162..268..271..7975..5315",
  "checkbox": false,
  "active": true,
  "condition": "new",
  "presta_categories": {
    "template": {
      "hp": [ "LAPTOPS INTEL I7", "14" ]
    }
  }
}
```

#### Описание полей

-   `brand` (str): Бренд ноутбука, в данном случае всегда `"HP"`.
-   `url` (str): URL-адрес категории ноутбука на сайте KSP.
-   `checkbox` (bool): Булево значение, указывающее на необходимость использования чекбокса. Всегда установлено в `false`.
-   `active` (bool): Булево значение, определяющее, активен ли данный сценарий. Всегда установлено в `true`.
-   `condition` (str): Состояние товара, в данном случае всегда `"new"`.
-   `presta_categories` (dict): Словарь, содержащий категории PrestaShop, к которым относится модель ноутбука.
    -   `template` (dict):  шаблон для категорий
       - `hp` (list): Список категорий PrestaShop, к которым относится модель. В примере это `"LAPTOPS INTEL I7"` и `"14"`.
