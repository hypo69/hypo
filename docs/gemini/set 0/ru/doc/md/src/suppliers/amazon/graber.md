# Модуль hypotez/src/suppliers/amazon/graber.py

## Обзор

Модуль `graber.py` содержит класс `Graber`, предназначенный для сбора данных о товарах со страницы `amazon.com`. Класс наследуется от базового класса `Graber` из модуля `src.suppliers` и предоставляет функции для извлечения различных полей товара.  В нём реализованы функции для обработки конкретных полей страницы товара, а также декоратор (теперь закомментирован) для закрытия всплывающих окон перед извлечением данных.  Этот модуль интегрирован с веб-драйвером и логикой обработки данных из модулей `src`, `src.product`, `src.webdriver`, `src.utils.jjson`, `src.logger`, `src.logger.exceptions`.  Модуль использует асинхронное программирование для повышения производительности.

## Классы

### `Graber`

**Описание**: Класс `Graber` наследует функциональность из базового класса `Graber` из модуля `src.suppliers`, добавляя специализированные методы для извлечения данных со страницы товара Amazon.  

**Атрибуты**:

- `supplier_prefix`: Строка, определяющая префикс поставщика ('amazon').

**Методы**:

#### `__init__`

**Описание**: Инициализирует экземпляр класса.

**Параметры**:

- `driver (Driver)`: Экземпляр класса `Driver`, представляющий веб-драйвер.

#### `grab_page`

**Описание**: Асинхронный метод для извлечения данных о товаре.

**Параметры**:

- `driver (Driver)`: Экземпляр класса `Driver`.

**Возвращает**:

- `ProductFields`: Объект, содержащий собранные поля товара.
