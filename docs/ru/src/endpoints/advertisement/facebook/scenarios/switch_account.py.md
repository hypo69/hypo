# Модуль `switch_account`

## Обзор

Модуль предназначен для переключения между аккаунтами в Facebook с использованием Selenium WebDriver. Он содержит функцию `switch_account`, которая проверяет наличие кнопки "Переключить" и нажимает на неё, если она присутствует.

## Подробней

Этот модуль является частью системы автоматизации для работы с Facebook рекламой. Он использует JSON-файл с локаторами элементов страницы для поиска кнопки переключения аккаунта. Функция `switch_account` принимает объект `Driver` (инстанс Selenium WebDriver) и использует его для взаимодействия с веб-страницей.

## Функции

### `switch_account`

```python
def switch_account(driver: Driver):
    """ Если есть кнопка `Переключить` - нажимаю её  """
```

**Описание**: Функция для переключения между аккаунтами в Facebook.

**Как работает функция**:
1. Функция `switch_account` принимает объект `driver` типа `Driver`, который представляет собой экземпляр Selenium WebDriver, используемый для управления браузером.
2. Внутри функции вызывается метод `execute_locator` объекта `driver`. Этот метод выполняет поиск элемента на веб-странице с использованием локатора, переданного в аргументе `locator.switch_to_account_button`. Локатор указывает на кнопку "Переключить аккаунт".
3. Если элемент, соответствующий локатору `locator.switch_to_account_button`, найден на странице, метод `execute_locator` нажимает на этот элемент, что приводит к переключению аккаунта.
4. Если элемент не найден, метод `execute_locator` ничего не делает, и функция завершается.

**Параметры**:
- `driver` (Driver): Объект `Driver`, представляющий собой экземпляр Selenium WebDriver.

**Возвращает**:
- `None`: Функция ничего не возвращает.

**Вызывает исключения**:
- Отсутствуют явные исключения, но могут возникнуть исключения, связанные с работой Selenium WebDriver, такие как `NoSuchElementException`, если элемент не найден.

**Примеры**:

```python
from src.webdriver.driver import Driver
from selenium import webdriver

# Пример использования функции switch_account
# Предположим, что у вас уже есть настроенный драйвер WebDriver
driver = Driver(webdriver.Chrome())  # Инициализация Chrome WebDriver
driver.get("https://www.facebook.com")  # Открытие Facebook

# Вызов функции switch_account для переключения аккаунта
switch_account(driver)

# Закрытие браузера после выполнения
driver.quit()
