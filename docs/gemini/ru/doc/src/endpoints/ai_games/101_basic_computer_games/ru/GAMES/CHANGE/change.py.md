# Игра "Хёркл"

## Обзор

Данный модуль реализует игру "Хёркл", в которой игрок пытается найти компьютер, "прячущийся" в случайном месте на сетке 10x10. Игрок получает подсказки о направлении и расстоянии до Хёркла после каждой попытки.

**Описание**:
Основная функция, реализующая игровой процесс "Хёркл". Игра продолжается, пока игрок не найдет Хёркла.

**Параметры**:
  - Нет

**Возвращает**:
  - Нет

**Вызывает исключения**:
  - `ValueError`: Возникает, если ввод пользователя не является целым числом.
