# Модуль `gradio`

## Обзор

Модуль `gradio` предоставляет интерфейс для создания простых веб-приложений с использованием библиотеки Gradio. В этом модуле определена функция `greet`, которая возвращает приветствие, и веб-интерфейс для этой функции.

## Оглавление

- [Функции](#функции)
  - [`greet`](#greet)

## Функции

### `greet`

**Описание**: Возвращает приветствие для переданного имени.

**Параметры**:
- `name` (str): Имя, которое нужно поприветствовать.

**Возвращает**:
- `str`: Приветствие в виде "Hello {name}!".

```python
def greet(name):
    return "Hello " + name + "!"
```
