# Модуль `src.suppliers.kualastyle.graber`

## Обзор

Модуль `src.suppliers.kualastyle.graber` представляет собой класс для сбора данных о товарах с сайта `kualastyle.co.il`. Он расширяет функциональность базового класса `Graber`, предоставляя специфическую логику обработки полей товаров. Модуль также включает в себя механизм для выполнения предварительных действий перед отправкой запроса к веб-драйверу через декораторы.

## Классы

### `Graber`

**Описание**: Класс `Graber` предназначен для сбора данных о товарах с сайта `kualastyle.co.il`. Он наследуется от класса `Graber` (переименованного в `Grbr`), предоставляя базовую функциональность и расширяя ее специфическими методами для данного поставщика.

**Методы**:

- `__init__`: Инициализирует экземпляр класса `Graber`.
    
    **Параметры**:
    - `driver` (`Driver`): Экземпляр веб-драйвера для взаимодействия с сайтом.

-   `supplier_prefix`: (str) Префикс поставщика `kualastyle`.

```python
    def __init__(self, driver: Driver):
        """Инициализация класса сбора полей товара."""
        self.supplier_prefix = 'kualastyle'
        super().__init__(supplier_prefix=self.supplier_prefix, driver=driver)
        # Устанавливаем глобальные настройки через Context
        
        Context.locator_for_decorator = None # <- если будет уастановлено значение - то оно выполнится в декораторе `@close_pop_up`
```
