Данный модуль содержит функцию `get_product_id`, предназначенную для извлечения идентификатора продукта из предоставленного текста. Функция использует вспомогательную функцию `extract_prod_ids` из модуля `src.suppliers.aliexpress.utils.extract_product_id` для более эффективного и надежного извлечения. В случае, если идентификатор продукта не найден, функция вызывает исключение `ProductIdNotFoundException`.

```python
""" module: src.suppliers.aliexpress.api.tools """
"""Some useful tools."""

from ..errors import ProductIdNotFoundException
from src.suppliers.aliexpress.utils.extract_product_id import extract_prod_ids
import re


def get_product_id(raw_product_id: str) -> str:
    """Возвращает идентификатор продукта из заданного текста.
    Возвращает None, если идентификатор не найден.
    Возвращает исключение ProductIdNotFoundException, если идентификатор не удалось найти.


    Args:
        raw_product_id: Текст, содержащий идентификатор продукта.

    Returns:
        Идентификатор продукта в виде строки, или None, если идентификатор не найден.
       Raises:
           ProductIdNotFoundException: Если идентификатор продукта не найден.
    """
    try:
        return extract_prod_ids(raw_product_id)
    except Exception as e:
        raise ProductIdNotFoundException(f"Ошибка при извлечении id: {e}")

```
