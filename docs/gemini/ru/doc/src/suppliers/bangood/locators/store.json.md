# Файл: store.json

## Обзор

Файл `store.json` содержит JSON-объект, представляющий структуру локаторов для страницы магазина на сайте Banggood. Он содержит в себе локаторы для различных элементов, таких как меню категорий, элементы на странице, кнопки и т.д.

## Структура

Файл представляет собой JSON-объект, который может содержать следующие поля:

-   `menu`: Локаторы для элементов меню.
-   `catalog`: Локаторы для элементов каталога.
-    `page`: Локаторы для элементов страницы.
-   `button`: Локаторы для кнопок.
-    `input`: Локаторы для полей ввода.

## Описание ключей

### `menu`

**Описание**: Содержит локаторы для элементов меню на странице магазина.

-   `category_list`: Локатор для контейнера списка категорий.
-   `category_item`: Локатор для элемента категории.

### `catalog`

**Описание**: Содержит локаторы для элементов каталога.

-   `catalog_items`: Локатор для контейнера элементов каталога.
-   `item_card`: Локатор для карточки товара в каталоге.
-    `item_link`: Локатор для ссылки на товар.
-   `item_title`: Локатор для заголовка товара.
-    `item_price`: Локатор для цены товара.
-    `item_image`: Локатор для изображения товара.
-    `item_discount`: Локатор для скидки на товар.

### `page`

**Описание**: Содержит локаторы для общих элементов страницы магазина.
-   `main_content`: Локатор для основного контента страницы.
-   `modal_window`: Локатор для модального окна.
-   `modal_close`: Локатор для кнопки закрытия модального окна.

### `button`

**Описание**: Содержит локаторы для кнопок на странице магазина.

-   `add_to_cart`: Локатор для кнопки "Добавить в корзину".
-   `buy_now`: Локатор для кнопки "Купить сейчас".
-   `close`: Локатор для кнопки "Закрыть".
-   `submit`: Локатор для кнопки "Подтвердить".
-  `add_to_wish_list`: Локатор для кнопки "Добавить в избранное".
-   `view_cart`: Локатор для кнопки "Просмотреть корзину".

### `input`

**Описание**: Содержит локаторы для полей ввода.
-   `search_input`: Локатор для поля ввода поиска.
