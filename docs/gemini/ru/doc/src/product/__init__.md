# Модуль `hypotez/src/product`

## Обзор

Модуль `src.product` содержит классы и функции для работы с продуктами. Он предоставляет средства для работы с продуктами, включая их поля, перевод многоязычных полей и представление в плоском формате.

## Классы

### `Product`

**Описание**: Класс `Product` содержит методы и атрибуты, связанные с продуктом. Подробное описание в файле `product.py`.

### `ProductFields`

**Описание**: Класс `ProductFields` описывает поля продукта. Подробное описание в файле `product_fields.py`.


## Функции

### `translate_presta_fields_dict`

**Описание**: Функция переводит многоязычные поля из класса `ProductFields`.

**Параметры**:
- Нет параметров.

**Возвращает**:
- `dict`: Словарь переведенных полей.

**Вызывает исключения**:
- Нет описанных исключений.


## Атрибуты

### `record`

**Описание**: Словарь, представляющий поля продукта в плоском формате (без вложенности).

### `MODE`

**Описание**: Строковая переменная, содержащая режим работы (`dev`).


## Модули

### `product`

**Описание**: Модуль, содержащий класс `Product`.

### `product_fields`

**Описание**: Модуль, содержащий класс `ProductFields` и функцию `translate_presta_fields_dict`.
