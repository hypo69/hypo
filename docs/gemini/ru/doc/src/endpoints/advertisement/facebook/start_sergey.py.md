# Модуль `start_sergey.py`

## Обзор

Модуль предназначен для запуска рекламных кампаний в Facebook, нацеленных на русско- и ивритоязычные группы. Он использует драйвер браузера для автоматизации процесса размещения объявлений и управляет циклами запуска кампаний.

## Константы

### `MODE`
**Описание**: Режим работы, по умолчанию установлен в `dev`.
- **Тип**: `str`
- **Значение**: `dev`

### `group_file_paths_ru`
**Описание**: Список путей к файлам с группами для русскоязычной аудитории.
- **Тип**: `list[str]`
- **Значение**: `["sergey_pages.json"]`

### `adv_file_paths_ru`
**Описание**: Список путей к файлам с рекламными объявлениями для русскоязычной аудитории.
- **Тип**: `list[str]`
- **Значение**: `["ru_ils.json"]`

### `group_file_paths_he`
**Описание**: Список путей к файлам с группами для ивритоязычной аудитории.
- **Тип**: `list[str]`
- **Значение**: `["sergey_pages.json"]`

### `adv_file_paths_he`
**Описание**: Список путей к файлам с рекламными объявлениями для ивритоязычной аудитории.
- **Тип**: `list[str]`
- **Значение**: `["he_ils.json"]`

### `group_categories_to_adv`
**Описание**: Список категорий групп, в которых будет размещаться реклама.
- **Тип**: `list[str]`
- **Значение**: `['sales', 'biz']`


## Функции

### `run_campaign`

**Описание**: Запускает рекламную кампанию.

**Параметры**:
- `d` (`Driver`): Экземпляр драйвера.
- `promoter_name` (`str`): Имя рекламодателя.
- `campaigns` (`list`): Список кампаний.
- `group_file_paths` (`list`): Пути к файлам с группами.
- `language` (`str`): Язык рекламной кампании.
- `currency` (`str`): Валюта рекламной кампании.

**Возвращает**:
- `None`: Функция ничего не возвращает.

### `campaign_cycle`

**Описание**:  Управляет циклом запуска рекламных кампаний для различных языков и рекламодателей.

**Параметры**:
- `d` (`Driver`): Экземпляр драйвера.

**Возвращает**:
- `bool`: Возвращает `True` после завершения цикла.

### `main`

**Описание**: Основная функция для запуска рекламных кампаний. Инициализирует драйвер, запускает циклы рекламных кампаний и обрабатывает прерывания.

**Параметры**:
- `None`: Функция не принимает параметров.

**Возвращает**:
- `None`: Функция ничего не возвращает.

**Вызывает исключения**:
- `KeyboardInterrupt`: Возникает при прерывании выполнения программы пользователем.
