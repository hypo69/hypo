# Модуль hypotez/src/suppliers/ivory/header.py

## Обзор

Данный модуль содержит функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и чтения файла `README.MD`.  Он также определяет переменные, содержащие информацию о проекте, такие как название, версия, описание, автор и авторские права.

## Функции

### `set_project_root`

**Описание**: Определяет корневую директорию проекта, начиная от текущей директории файла, ищет вверх по директориям до тех пор, пока не найдет директорию содержащую один из указанных файлов.

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или каталогов, которые используются для определения корня проекта.

**Возвращает**:

- `Path`: Объект `Path` представляющий корневую директорию проекта, или текущую директорию, если корневая не найдена.

**Вызывает исключения**:

- Никаких исключений не генерируется.

### `set_project_root`

**Описание**:  Возвращает корневую директорию проекта.

**Параметры**:
Нет.


**Возвращает**:
- `Path`: Путь до корневой директории проекта


**Вызывает исключения**:
- Никаких.

## Переменные

### `__root__`

**Описание**: Переменная хранит путь к корневой директории проекта, полученный функцией `set_project_root`.

**Тип**: `Path`

### `settings`

**Описание**: Словарь настроек, загруженный из файла `src/settings.json`.

**Тип**: `dict`


### `doc_str`

**Описание**: Строковое представление содержимого файла `README.MD`

**Тип**: `str`


### `__project_name__`

**Описание**: Имя проекта, полученное из настроек (по умолчанию 'hypotez').

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта, полученная из настроек (по умолчанию пустая строка).

**Тип**: `str`

### `__doc__`

**Описание**:  Описание проекта (документация), полученная из файла `README.MD` (по умолчанию пустая строка).

**Тип**: `str`


### `__details__`

**Описание**:  Дополнительная информация о проекте (по умолчанию пустая строка).

**Тип**: `str`

### `__author__`

**Описание**: Автор проекта, полученный из настроек (по умолчанию пустая строка).

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права проекта, полученные из настроек (по умолчанию пустая строка).

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка на страницу для пожертвований на кофе разработчику.

**Тип**: `str`
