# Модуль `header.py`

## Обзор

Модуль `header.py` предназначен для настройки окружения проекта, загрузки конфигураций и базовой информации, такой как имя проекта, версия и документация. Он также определяет корневую директорию проекта и добавляет её в системный путь.

## Функции

### `set_project_root`

**Описание**:
Находит корневую директорию проекта, начиная с директории текущего файла, ищет вверх и останавливается в первой директории, содержащей один из маркерных файлов.

**Параметры**:
- `marker_files` (tuple): Имена файлов или директорий, идентифицирующие корень проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена, иначе директория, где расположен скрипт.

## Переменные

### `__root__`

**Описание**:
Путь к корневой директории проекта. Определяется с помощью функции `set_project_root`.

### `config`

**Описание**:
Словарь, содержащий конфигурацию проекта, загруженную из файла `config.json`.

### `doc_str`

**Описание**:
Строка, содержащая документацию проекта, загруженную из файла `README.MD`.

### `__project_name__`

**Описание**:
Имя проекта. По умолчанию 'hypotez'. Если `config` существует, берётся из ключа "project_name".

### `__version__`

**Описание**:
Версия проекта. По умолчанию пустая строка. Если `config` существует, берётся из ключа "version".

### `__doc__`

**Описание**:
Документация проекта. По умолчанию пустая строка. Если `doc_str` существует, берётся из него.

### `__details__`

**Описание**:
Дополнительные детали проекта. По умолчанию пустая строка. На данный момент не используется.

### `__author__`

**Описание**:
Автор проекта. По умолчанию пустая строка. Если `config` существует, берётся из ключа "author".

### `__copyright__`

**Описание**:
Авторские права проекта. По умолчанию пустая строка. Если `config` существует, берётся из ключа "copyrihgnt".

### `__cofee__`

**Описание**:
Сообщение для поддержки разработчика. По умолчанию "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69".
