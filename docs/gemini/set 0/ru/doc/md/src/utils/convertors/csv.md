# Модуль `hypotez/src/utils/convertors/csv.py`

## Обзор

Модуль `hypotez/src/utils/convertors/csv.py` предоставляет утилиты для конвертации данных в формате CSV в различные форматы, включая словари Python и объекты `SimpleNamespace`. Также модуль включает функцию для сохранения данных CSV в формате JSON.

## Функции

### `csv2dict`

**Описание**: Преобразует данные из CSV файла в словарь Python.

**Параметры**:
- `csv_file` (str | Path): Путь к файлу CSV.

**Возвращает**:
- `dict | None`: Словарь Python, содержащий данные из CSV, или `None` при ошибке.

**Вызывает исключения**:
- `Exception`: Возникает, если не удалось прочитать CSV файл.

### `csv2ns`

**Описание**: Преобразует данные из CSV файла в объекты `SimpleNamespace`.

**Параметры**:
- `csv_file` (str | Path): Путь к файлу CSV.

**Возвращает**:
- `SimpleNamespace | None`: Объект `SimpleNamespace`, содержащий данные из CSV, или `None` при ошибке.

**Вызывает исключения**:
- `Exception`: Возникает, если не удалось прочитать CSV файл.

### `csv_to_json`

**Описание**: Конвертирует CSV файл в JSON формат и сохраняет результат в новый JSON файл.

**Параметры**:
- `csv_file_path` (str | Path): Путь к файлу CSV.
- `json_file_path` (str | Path): Путь к файлу JSON, куда будет сохранен результат.
- `exc_info` (bool, optional): Если True, включает информацию об ошибке в логи. По умолчанию True.

**Возвращает**:
- `List[Dict[str, str]] | None`: Список словарей, представляющих данные в формате JSON, или `None`, если преобразование завершилось неудачно.

**Примеры использования**:
```python
>>> json_data = csv_to_json('dialogue_log.csv', 'dialogue_log.json')
>>> print(json_data)
[{'role': 'user', 'content': 'Hello'}, {'role': 'assistant', 'content': 'Hi there!'}]
```

**Вызывает исключения**:
- `Exception`: Любые ошибки при чтении/записи файлов или конвертации.  Логирование ошибок происходит с помощью `logger` из `src.logger`.
