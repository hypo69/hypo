# webdriver/_docs/executor.2.ru.md

## Обзор

Этот модуль предоставляет функции для взаимодействия с веб-элементами на веб-страницах. Он включает в себя методы для поиска элементов, отправки сообщений, получения атрибутов и работы с URL.

## Функции

### `execute_locator`

**Описание**: Выполняет действия на веб-элементе, используя указанный локатор.

**Параметры**:
- `locator` (dict): Словарь или объект с информацией о локаторе (например, тип поиска и селектор). Требуется.
- `message` (str, optional): Сообщение для отправки элементу (например, текст для ввода). По умолчанию ''.
- `typing_speed` (float, optional): Скорость набора текста, если отправляется сообщение. По умолчанию 0.0.
- `continue_on_error` (bool, optional): Флаг для продолжения выполнения в случае ошибки. По умолчанию True.

**Возвращаемое значение**:
- `any`: Результат выполнения локатора, который может быть веб-элементом, списком элементов, значением атрибута или результатом действия.

**Возможные исключения**:
- `WebDriverException`: Возникает при ошибках взаимодействия с драйвером.
- `NoSuchElementException`: Возникает, если элемент, определенный локатором, не найден.
- `InvalidLocatorException`: Возникает, если локатор имеет неправильный формат.
- `ElementNotVisibleException`: Возникает, если элемент, определенный локатором, не виден на странице.
- `StaleElementReferenceException`: Возникает, если элемент, на который ссылается локатор, был удалён или изменён.
- `TimeoutException`: Возникает, если действие не выполнено в течение заданного времени.

### `get_webelement_by_locator`

**Описание**: Находит и возвращает веб-элемент(ы) на странице, используя локатор.

**Параметры**:
- `locator` (dict): Словарь или объект с информацией о локаторе. Требуется.

**Возвращаемое значение**:
- `any`: Один или несколько веб-элементов, найденных по локатору. Возможен возврат списка элементов или одного элемента, в зависимости от типа локатора.

**Возможные исключения**:
- те же, что и у `execute_locator`.

### `get_attribute_by_locator`

**Описание**: Получает значение атрибута веб-элемента, найденного по локатору.

**Параметры**:
- `locator` (dict): Словарь или объект с информацией о локаторе. Требуется.
- `message` (str, optional): Сообщение для отправки элементу перед получением атрибута. По умолчанию ''.

**Возвращаемое значение**:
- `any`: Значение атрибута веб-элемента. Возвращает `None`, если элемент не найден или атрибут не существует.

**Возможные исключения**:
- те же, что и у `execute_locator`.

### `send_message`

**Описание**: Отправляет сообщение (например, текст) веб-элементу, используя локатор.

**Параметры**:
- `locator` (dict): Словарь или объект с информацией о локаторе. Требуется.
- `message` (str): Сообщение для отправки элементу. Требуется.
- `typing_speed` (float, optional): Скорость набора текста, если сообщение отправляется по частям. По умолчанию 0.0.
- `continue_on_error` (bool, optional): Флаг для продолжения выполнения в случае ошибки. По умолчанию True.

**Возвращаемое значение**:
- `bool`: `True`, если сообщение успешно отправлено, или `False`, если произошла ошибка.

**Возможные исключения**:
- те же, что и у `execute_locator`.


### `get_url`

**Описание**: Загружает HTML-контент с указанного URL-адреса или локального файла.

**Параметры**:
- `url` (str): URL-адрес или путь к файлу для получения HTML-контента. Требуется.
- `protocol` (str, optional): Протокол для URL (по умолчанию `https://`).

**Возвращаемое значение**:
- `bool`: `True`, если контент успешно загружен, или `False`, если произошла ошибка.

**Возможные исключения**:
- `requests.exceptions.RequestException`: Возникает при проблемах с запросом к URL (например, сетевые ошибки).
- `FileNotFoundError`: Возникает, если указанный файл не найден.
- `URLError`: Возникает при ошибках, связанных с URL.
