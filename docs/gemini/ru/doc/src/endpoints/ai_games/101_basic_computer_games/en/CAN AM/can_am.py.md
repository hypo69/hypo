# CAN AM

## Обзор

Игра "CAN AM" представляет собой симуляцию гонки по трассе. Игрок управляет машиной, определяя, сколько раз нужно нажать на газ, чтобы достичь финиша. Игрок должен правильно рассчитать ускорение, чтобы не вылететь с трассы или не остановиться слишком рано.

## Функции

### `time.sleep()`

**Описание**: Функция используется для приостановки выполнения программы на заданное количество секунд.

### `input()`

**Описание**: Функция используется для получения ввода от пользователя.

### `int()`

**Описание**: Функция используется для преобразования введенных данных в целое число.

## Переменные

### `position`

**Описание**: Переменная `position` хранит текущую позицию автомобиля на трассе. Инициализируется значением 0.

### `speed`

**Описание**: Переменная `speed` хранит текущую скорость автомобиля. Инициализируется значением 0.

### `trackLength`

**Описание**: Переменная `trackLength` хранит длину трассы, равную 200.

### `acceleration`

**Описание**: Переменная `acceleration` хранит значение ускорения, введенное пользователем.

## Игровой цикл

**Описание**: Основной игровой цикл `while True:` продолжается до тех пор, пока игрок не достигнет финиша или не вылетит с трассы.

## Обработка ввода

**Описание**: 
Блок `try...except ValueError` обрабатывает возможные ошибки ввода. Если пользователь введет не целое число, то будет выведено сообщение об ошибке. 
Также проверяется ввод на диапазон допустимых значений (0-9) и, если ввод не корректен, выводит сообщение об ошибке и переходит к следующей итерации цикла.

## Обновление состояния

**Описание**:

- `speed += acceleration`: Увеличивает текущую скорость автомобиля на значение введенного ускорения.
- `position += speed`: Обновляет текущую позицию автомобиля, прибавляя к ней текущую скорость.

## Проверка условий

**Описание**:

-   `if position < 0 or position > trackLength`: Проверяет, вылетела ли машина за пределы трассы. Если да, выводит сообщение "CRASH!" и завершает игру.
-   `if position >= trackLength`: Проверяет, достигла ли машина финиша. Если да, выводит сообщение "WINNER!" и завершает игру.
