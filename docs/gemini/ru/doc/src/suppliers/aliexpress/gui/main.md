# Модуль `hypotez/src/suppliers/aliexpress/gui/main.py`

## Обзор

Данный модуль предоставляет основной интерфейс приложения для управления рекламными кампаниями AliExpress. Он реализован с использованием PyQt6 и позволяет создавать, редактировать и загружать данные рекламных кампаний в формате JSON. Модуль включает в себя вкладки для работы с разными типами данных (JSON редактор, редактор кампаний, редактор продуктов).  Также реализованы функции для открытия, сохранения и загрузки файлов.

## Классы

### `MainApp`

**Описание**: Класс `MainApp` представляет собой главное окно приложения. Он управляет вкладами и предоставляет основные функции для работы с файлами и редактированием данных.

**Методы**:

- `__init__(self)`:
    **Описание**: Инициализирует главное окно приложения с вкладками для редактирования данных JSON, кампаний и продуктов. Настраивает геометрию окна, добавляет вкладки и создаёт меню.
    **Параметры**:
    - `self`: ссылка на экземпляр класса.


- `create_menubar(self)`:
    **Описание**: Создаёт главное меню приложения с пунктами для операций с файлами (открытие, сохранение, выход) и редактирования. Подключает обработчики событий для этих пунктов меню.
    **Параметры**:
    - `self`: ссылка на экземпляр класса.


- `open_file(self)`:
    **Описание**: Открывает диалоговое окно для выбора файла JSON. Загружает выбранный файл в соответствующую вкладку.
    **Параметры**:
    - `self`: ссылка на экземпляр класса.


- `save_file(self)`:
    **Описание**: Сохраняет текущий файл в выбранном формате.
    **Параметры**:
    - `self`: ссылка на экземпляр класса.


- `exit_application(self)`:
    **Описание**: Закрывает приложение.
    **Параметры**:
    - `self`: ссылка на экземпляр класса.


- `copy(self)`:
    **Описание**: Копирует выделенный текст в буфер обмена.
    **Параметры**:
    - `self`: ссылка на экземпляр класса.


- `paste(self)`:
    **Описание**: Вставляет текст из буфера обмена.
    **Параметры**:
    - `self`: ссылка на экземпляр класса.


- `load_file(self, campaign_file)`:
    **Описание**: Загружает JSON-файл в JSON редактор.
    **Параметры**:
    - `self`: ссылка на экземпляр класса.
    - `campaign_file`: путь к загружаемому файлу.
    **Возвращает**:
       - Возвращает `None`.
    **Исключения**:
    - `Exception`: Возникает при ошибке загрузки файла. Сообщение об ошибке отображается пользователю.
