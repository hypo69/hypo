# Модуль `header.py`

## Обзор

Файл `header.py` содержит общие настройки и импорты, необходимые для других скриптов в директории `src/suppliers/aliexpress/campaign/_examples`. Он устанавливает режим работы, добавляет корневую и рабочую директории проекта в `sys.path` для корректного импорта модулей.

## Содержание

- [Обзор](#обзор)
- [Переменные](#переменные)
- [Импорты](#импорты)

## Переменные

### `MODE`

**Описание**: Определяет режим работы скрипта.

**Значение**: `'dev'`.

### `dir_root`

**Описание**: Абсолютный путь к корневой директории проекта.

**Значение**: Вычисляется динамически на основе текущей рабочей директории.

### `dir_src`

**Описание**: Абсолютный путь к директории `src`.

**Значение**:  Вычисляется на основе `dir_root`.

## Импорты

В данном файле импортируются следующие модули:

### `os`

**Описание**: Предоставляет функции для взаимодействия с операционной системой.

### `sys`

**Описание**: Предоставляет доступ к некоторым переменным и функциям, используемым или поддерживаемым интерпретатором.

### `pathlib.Path`

**Описание**: Предоставляет способ представления путей файловой системы в виде объектов.
