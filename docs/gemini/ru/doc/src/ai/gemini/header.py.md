# Модуль `header.py`

## Обзор

Модуль `header.py` представляет собой часть проекта `hypotez`, предназначенную для настройки и управления глобальными переменными и путями проекта. Он отвечает за определение корневой директории проекта, загрузку конфигурации из файла `config.json`, чтение документации из `README.MD` и установку основных атрибутов проекта, таких как имя, версия, автор и т.д.

## Функции

### `set_project_root`

**Описание**: 
Функция `set_project_root` определяет корневую директорию проекта, начиная поиск от директории текущего файла, поднимаясь вверх по иерархии. Поиск завершается при нахождении директории, содержащей один из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple): Кортеж с именами файлов или директорий, которые используются как маркеры для идентификации корневой директории проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Объект `Path`, представляющий путь к корневой директории проекта. Если корневая директория не найдена, возвращается директория текущего файла.

## Глобальные переменные

### `__root__`

**Описание**: 
`__root__` содержит объект `Path`, который представляет путь к корневой директории проекта. Эта переменная инициализируется при вызове функции `set_project_root`.

### `config`

**Описание**: 
`config` является словарем, содержащим настройки проекта, загруженные из файла `config.json`. Если файл не найден или не может быть декодирован, значение переменной будет `None`.

### `doc_str`

**Описание**: 
`doc_str` содержит строку с содержимым файла `README.MD`, которая обычно используется для документирования проекта. Если файл не найден или не может быть прочитан, значение переменной будет `None`.

### `__project_name__`

**Описание**: 
`__project_name__` содержит имя проекта, полученное из файла конфигурации `config.json` или значение `hypotez` по умолчанию, если конфигурация не найдена.

### `__version__`

**Описание**: 
`__version__` содержит версию проекта, полученную из файла конфигурации `config.json` или пустую строку, если конфигурация не найдена.

### `__doc__`

**Описание**: 
`__doc__` содержит строку документации проекта, загруженную из файла `README.MD` или пустую строку, если документ не найден.

### `__details__`

**Описание**: 
`__details__` содержит строку с дополнительной информацией о проекте. На текущий момент она установлена в пустую строку.

### `__author__`

**Описание**: 
`__author__` содержит имя автора проекта, полученное из файла конфигурации `config.json` или пустую строку, если конфигурация не найдена.

### `__copyright__`

**Описание**: 
`__copyright__` содержит информацию об авторских правах, полученную из файла конфигурации `config.json` или пустую строку, если конфигурация не найдена.

### `__cofee__`

**Описание**: 
`__cofee__` содержит сообщение с предложением угостить разработчика чашкой кофе. Значение по умолчанию: `"Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69"`. Значение может быть получено из файла настроек, если он доступен.
