Описание модуля
---------------

Этот модуль содержит класс `AliCampaignGoogleSheet` для работы с Google Sheets в рамках рекламных кампаний AliExpress.  Он предоставляет методы для управления листами, записи данных о категориях и продуктах, а также форматирования листов. Модуль использует библиотеку gspread для взаимодействия с Google Sheets и другие вспомогательные библиотеки.
