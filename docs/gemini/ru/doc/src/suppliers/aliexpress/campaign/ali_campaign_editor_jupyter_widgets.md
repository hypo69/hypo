# Модуль `hypotez/src/suppliers/aliexpress/campaign/ali_campaign_editor_jupyter_widgets.py`

## Обзор

Этот модуль предоставляет виджеты Jupyter для работы с редактором кампаний AliExpress в интерактивных Jupyter ноутбуках.  Он позволяет выбирать кампании, категории, языки, инициализировать редактор, сохранять кампании и отображать продукты.


## Классы

### `JupyterCampaignEditorWidgets`

**Описание**: Класс предоставляет виджеты для взаимодействия с редактором кампаний AliExpress. Позволяет выбирать кампанию, категорию и язык, инициализировать редактор, сохранять изменения и отображать продукты.

**Атрибуты**:

- `language: str`: Язык кампании.
- `currency: str`: Валюта кампании.
- `campaign_name: str`: Название выбранной кампании.
- `category_name: str`: Название выбранной категории.
- `category: SimpleNamespace`: Объект, содержащий данные категории.
- `campaign_editor: AliCampaignEditor`: Объект редактора кампаний.
- `products: list[SimpleNamespace]`: Список продуктов в выбранной категории.

**Методы**:

#### `__init__(self)`

**Описание**: Инициализирует виджеты и настраивает редактор кампаний. Создает выпадающие списки для выбора кампаний, категорий и языков/валют, а также кнопки для инициализации, сохранения и отображения продуктов. Устанавливает обработчики событий для взаимодействий с виджетами.

#### `initialize_campaign_editor(self, _)`

**Описание**: Инициализирует редактор кампаний на основе выбранных значений кампании, категории и языка.  Обновляет `campaign_editor`, `category` и `products` на основе выбора.

**Параметры**:

- `_`: Пустой аргумент, необходимый для корректной работы с обработчиками кнопок.

#### `update_category_dropdown(self, campaign_name: str)`

**Описание**: Обновляет выпадающий список категорий на основе выбранной кампании.

**Параметры**:

- `campaign_name (str)`: Название кампании.


#### `on_campaign_name_change(self, change: dict[str, str])`

**Описание**: Обрабатывает изменения в выпадающем списке выбора кампаний. Обновляет список категорий и инициализирует редактор кампаний.

**Параметры**:

- `change (dict[str, str])`: Словарь изменений, содержащий новое значение.


#### `on_category_change(self, change: dict[str, str])`

**Описание**: Обрабатывает изменения в выпадающем списке категорий. Инициализирует редактор кампаний с новой категорией.

**Параметры**:

- `change (dict[str, str])`: Словарь изменений, содержащий новое значение.


#### `on_language_change(self, change: dict[str, str])`

**Описание**: Обрабатывает изменения в выпадающем списке выбора языка/валюты. Инициализирует редактор кампаний с новым языком/валютой.

**Параметры**:

- `change (dict[str, str])`: Словарь изменений, содержащий новое значение.


#### `save_campaign(self, _)`

**Описание**: Сохраняет выбранную кампанию и ее категории.

**Параметры**:

- `_`: Пустой аргумент, необходимый для корректной работы с обработчиками кнопок.


#### `show_products(self, _)`

**Описание**: Отображает продукты в выбранной категории.

**Параметры**:

- `_`: Пустой аргумент, необходимый для корректной работы с обработчиками кнопок.


#### `open_spreadsheet(self, _)`

**Описание**: Открывает Google Таблицу в браузере.

**Параметры**:

- `_`: Пустой аргумент, необходимый для корректной работы с обработчиками кнопок.


#### `setup_callbacks(self)`

**Описание**: Настраивает обработчики событий для виджетов. Связывает методы обратного вызова с изменениями в выпадающих списках и кнопках.


#### `display_widgets(self)`

**Описание**: Отображает виджеты в Jupyter ноутбуке для взаимодействия. Инициализирует редактор кампаний с первой выбранной кампанией.
