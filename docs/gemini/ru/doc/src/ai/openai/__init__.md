# Модуль hypotez/src/ai/openai

## Обзор

Данный модуль предоставляет инструменты для взаимодействия с моделью OpenAI.

## Переменные

### `MODE`

**Описание**: Строковая переменная, определяющая режим работы модуля.  Текущее значение: 'dev'.

**Значение**: 'dev'

## Функции

### `translate`

**Описание**: Функция для перевода текста.

**Параметры**:

-  `text` (str): Текст для перевода.
-  `target_language` (str): Целевой язык перевода.


**Возвращает**:

-  `dict`: Результат перевода в виде словаря с ключами `translated_text`, `original_text` и `target_language`.

**Вызывает исключения**:

-  `ValueError`: В случае неверного формата входных данных или отсутствия целевого языка.
-  `Exception`: Общее исключение при возникновении проблем во время запроса к модели OpenAI.


## Классы

### `OpenAIModel`

**Описание**: Класс для работы с моделью OpenAI.


**Методы**:

- `__init__(self, api_key: str, model_name: str = 'text-davinci-003')`: Инициализирует экземпляр класса.
    **Параметры**:
        - `api_key` (str): API-ключ для доступа к модели OpenAI.
        - `model_name` (str, optional): Название модели OpenAI. По умолчанию 'text-davinci-003'.

    **Возвращает**: None.

- `generate_text(self, prompt: str, max_tokens: int = 256) -> str | None`: Генерирует текст на основе предоставленного запроса.

    **Параметры**:
        - `prompt` (str): Запрос для модели.
        - `max_tokens` (int, optional): Максимальное количество токенов для генерации. По умолчанию 256.


    **Возвращает**:
        - str | None: сгенерированный текст или None, если произошла ошибка.

    **Вызывает исключения**:

    - `ValueError`: Если входные данные неверного типа или невалидны.
    - `Exception`: Общее исключение при возникновении проблем во время взаимодействия с моделью OpenAI.
