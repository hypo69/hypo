# Модуль `banner.py`

## Обзор

Модуль `banner.py` реализует простую игру, которая запрашивает у пользователя текст и создаёт текстовый баннер вокруг него.

## Содержание

- [Функции](#функции)
    - [`create_banner`](#create_banner)

## Функции

### `create_banner`

**Описание**: Создаёт текстовый баннер из введённого текста.

**Параметры**:
- `text` (str): Строка, которую нужно преобразовать в баннер.

**Возвращает**:
- `None`: Функция ничего не возвращает, а выводит баннер в консоль.
