# Модуль `graber.py`

## Обзор

Модуль `graber.py` содержит класс `Graber`, предназначенный для сбора информации о товарах с сайта `grandadvanse.co.il`. Этот класс наследует функциональность от родительского класса `Graber` и переопределяет или дополняет методы для обработки специфичных полей товаров данного поставщика. В модуле также присутствует заготовка декоратора для предварительных действий перед запросом к веб-драйверу, который можно настроить для закрытия всплывающих окон или выполнения других подготовительных шагов.

## Переменные

### `MODE`
- **Описание**: Режим работы модуля, по умолчанию установлен в `dev`.
- **Тип**: `str`

## Декораторы

### `close_pop_up`
**Описание**: Заготовка для декоратора, который должен выполнять предварительные действия, такие как закрытие всплывающих окон перед выполнением основной функции. Заготовка закоментирована, для активации ее нужно раскоментировать и переопределить ее поведение.

**Параметры**:\
    - `value` (Any, optional): Дополнительное значение для декоратора. По умолчанию `None`.

**Возвращает**:\
    - `Callable`: Декоратор, оборачивающий функцию.
