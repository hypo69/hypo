# Модуль PrestaShop

## Обзор

Данный модуль предоставляет инструменты для работы с API платформы PrestaShop. Он содержит классы и функции для управления различными ресурсами PrestaShop, включая категории, клиентов, языки, прайс-листы, продукты, магазины, поставщиков, склады и др.  Модуль также включает вспомогательные функции для работы с API, включая построение схем и примеры использования.

## Структура модуля

Этот модуль организован в несколько директорий:

* **`PrestaShop`:** Основной каталог модуля, содержащий модули для управления разными ресурсами PrestaShop.
* **`_examples`:** Директория с примерами использования модуля.
* **`api`:** Директория с модулями для работы с API PrestaShop.
* **`api_schemas`:** Директория со схемами JSON для ресурсов API PrestaShop.
* **`domains`:** Директория, содержащая конфигурации для различных доменов, таких как `ecat.co.il`, `emildesign.com` и `sergey.mymaster.co.il`.


## Компоненты

* **`category.py`:** Управление категориями продуктов.
* **`customer.py`:** Управление клиентами.
* **`language.py`:** Управление языками.
* **`pricelist.py`:** Управление прайс-листами.
* **`product.py`:** Управление продуктами.
* **`shop.py`:** Управление магазинами.
* **`supplier.py`:** Управление поставщиками.
* **`warehouse.py`:** Управление складами.
* **`api.py`:** Модуль для работы с API PrestaShop, содержащий функции для выполнения запросов и обработки ответов.
* **`api_schemas_builder.py`:** Скрипт для построения схем API.
* **`api_resourses_list.py`:**  Список доступных ресурсов API.


## Примеры использования

```python
from PrestaShop.product import Product

# Инициализация класса Product
product = Product()

# Пример получения данных о продукте с ID 12345
product_data = product.get_product_data(product_id="12345")

print(product_data)
```
