## Обзор

Данный JSON-файл содержит конфигурацию для сценария, связанного с категориями принтеров поставщика Morlevi. Файл определяет URL-адрес, условия и связи с категориями PrestaShop.

## Структура JSON

### `scenarios`

**Описание**: Корневой объект, содержащий настройки для сценария.

**Параметры**:

-   `url` (str): URL-адрес, с которого будут извлекаться данные. В данном случае, `"https://www.morlevi.co.il/Cat/60"`.
-   `checkbox` (bool): Логическое значение, указывающее на наличие/отсутствие чекбокса. В данном случае `false`.
-   `active` (bool): Логическое значение, указывающее, активен ли сценарий. В данном случае `true`.
-   `condition` (str): Условие для применения сценария. В данном случае `"new"`.
-   `presta_categories` (str): Строка, содержащая ID категорий PrestaShop, разделенных запятыми. В данном случае `"151,209"`.
