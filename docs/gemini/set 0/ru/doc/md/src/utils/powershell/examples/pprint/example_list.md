# Модуль example_list

## Обзор

Этот модуль содержит примеры работы с списками.

## Функции

### `pprint_list`

**Описание**: Функция выводит список в отформатированном виде.

**Параметры**:

- `data` (list): Список, который необходимо вывести.

**Возвращает**:

- `None`: Данная функция не возвращает значение.
