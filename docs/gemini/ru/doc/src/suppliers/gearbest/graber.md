# Модуль hypotez/src/suppliers/gearbest/graber.py

## Обзор

Модуль `hypotez/src/suppliers/gearbest/graber.py` содержит класс `Graber`, предназначенный для сбора данных о товарах с сайта gearbest.com. Класс наследуется от класса `Grbr` из родительского модуля `src.suppliers.graber`. Он предоставляет функции для обработки полей товара, позволяя переопределять стандартную обработку полей для конкретных требований.  Модуль использует веб-драйвер для взаимодействия с сайтом и содержит логику для обработки всплывающих окон.

* Модуль использует декоратор `@close_pop_up` для закрытия всплывающих окон перед выполнением основной логики. Декоратор находится в родительском классе, но может быть переопределен в дочерних классах.
