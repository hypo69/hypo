Данный файл содержит JSON-конфигурацию сценариев для парсинга категорий часов бренда HONOR с сайта KSP. В нем определены сценарии для конкретных моделей часов, а также их настройки для парсинга и классификации.

## Сценарии

### Honor Magic Watch 2
**Описание**: Конфигурация для парсинга категорий часов Honor Magic Watch 2 с сайта KSP.
- **`brand`** (str): Бренд часов - `HONOR`.
- **`url`** (str): URL-адрес страницы с часами на сайте KSP: `https://ksp.co.il/web/cat/2085..28432..28440`.
- **`checkbox`** (bool): Логическое значение, указывающее на состояние чекбокса (не используется в данном контексте, `false`).
-  **`active`** (bool): Логическое значение, указывающее на активность сценария (`true`).
-  **`condition`** (str): Строка, указывающая на состояние товара (`new`).
- **`presta_categories`** (dict): Словарь соответствий идентификаторов категорий PrestaShop и их текстовых представлений:
  - `"3405"`: `"GOOGLE PIXEL PRO"`.
  - `"3198"`: `"CONSUMER ELECTRONICS"`.
  - `"3202"`: `"computer,smartphone,gaming console,smart device"`.
  - `"6471"`: `"Smartphones"`.
  - `"3403"`: `"GOOGLE"`.
