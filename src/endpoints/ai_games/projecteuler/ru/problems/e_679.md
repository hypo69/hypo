# Задача No 679
## Калькулятор с переменными
## Условия
У вас есть калькулятор, который работает с переменными, представленными буквами от `a` до `z`.
Калькулятор поддерживает три вида операций:
   *  `a=b+c` - присваивает `a` сумму `b` и `c`
   *  `a=b-c` - присваивает `a` разность `b` и `c`
   *  `a=b` - присваивает `a` значение `b`
   
В начале все переменные равны `0`. Вам предоставляется список операций. После выполнения всех операций, какие значения будут у переменных?

Значения переменных не ограничиваются целыми числами и могут быть отрицательными.

Вам даётся входной файл `operations.txt` с операциями, каждая операция на отдельной строке.
Например:
```
a=b+c
b=1
c=2
d=a-b
e=d
a=e-b
```

Ваша цель - найти значения переменных после выполнения всех операций из файла `operations.txt`.
Выведите сумму значений всех переменных после выполнения всех операций.
