# `header.py`

## Обзор

Модуль `header.py` предназначен для настройки среды проекта, включая определение корневой директории, загрузку настроек из `settings.json`, чтение документации из `README.md` и инициализацию глобальных переменных проекта.

## Функции

### `set_project_root`

**Описание**:
Находит корневую директорию проекта, начиная с директории текущего файла, и добавляет ее в `sys.path`, если она не добавлена.

**Параметры**:
- `marker_files` (tuple, optional): Список файлов или директорий, по наличию которых определяется корневая директория проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории проекта.

## Глобальные переменные

### `__root__`

**Описание**:
Путь к корневой директории проекта, определенный функцией `set_project_root`.

### `settings`

**Описание**:
Словарь с настройками проекта, загруженный из файла `settings.json`. Если файл не найден или содержит ошибку, значение остается `None`.

### `doc_str`

**Описание**:
Строка с содержимым файла `README.MD`, если он существует. В противном случае `None`.

### `__project_name__`

**Описание**:
Название проекта. По умолчанию 'hypotez', или значение из `settings.json`.

### `__version__`

**Описание**:
Версия проекта. По умолчанию пустая строка, или значение из `settings.json`.

### `__doc__`

**Описание**:
Содержимое документации проекта (из `README.md`). Пустая строка, если файл не найден.

### `__details__`

**Описание**:
Детали проекта. Пустая строка по умолчанию.

### `__author__`

**Описание**:
Автор проекта. Пустая строка по умолчанию, или значение из `settings.json`.

### `__copyright__`

**Описание**:
Информация о копирайте. Пустая строка по умолчанию, или значение из `settings.json`.

### `__cofee__`

**Описание**:
Сообщение для поощрения разработчика чашкой кофе. По умолчанию  "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69", или значение из `settings.json`.
