# Модуль hypotez/src/endpoints/prestashop/__init__.py

## Обзор

Этот модуль предоставляет интерфейс для работы с API Престашоп. Он содержит классы для работы с различными сущностями: продуктами, поставщиками, категориями, складами, языками, магазинами, прайс-листами и клиентами.  Модуль импортирует необходимые классы из соответствующих файлов.

## Импортированные модули

* `PrestaShop`: Класс для работы с основным API Престашоп.
* `PrestaProduct`: Класс для работы с продуктами.
* `PrestaSupplier`: Класс для работы с поставщиками.
* `PrestaCategory`: Класс для работы с категориями.
* `PrestaWarehouse`: Класс для работы со складами.
* `PrestaLanguage`: Класс для работы с языками.
* `PrestaShopShop`: Класс для работы с магазинами.
* `PriceListRequester`: Класс для работы с прайс-листами.
* `PrestaCustomer`: Класс для работы с клиентами.

## Константы

* `MODE`: Строка, определяющая режим работы (например, 'dev' или 'prod').

## Дополнительные сведения


Этот модуль предоставляет удобный способ взаимодействия с API Престашоп для выполнения различных задач.
