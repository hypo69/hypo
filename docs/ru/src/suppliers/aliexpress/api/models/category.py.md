# Модуль `category`

## Обзор

Модуль `category` содержит модели данных для категорий товаров, используемые при взаимодействии с API AliExpress. Он включает в себя базовую модель `Category` и модель `ChildCategory` для представления дочерних категорий.

    - [`ChildCategory`](#childcategory)

## Классы

### `Category`

**Описание**: Базовый класс для представления категории товара.

**Атрибуты**:
- `category_id` (int): Идентификатор категории.
- `category_name` (str): Название категории.

### `ChildCategory`

**Описание**: Класс для представления дочерней категории, наследуемый от `Category`.

**Атрибуты**:
- `category_id` (int): Идентификатор категории.
- `category_name` (str): Название категории.
- `parent_category_id` (int): Идентификатор родительской категории.
