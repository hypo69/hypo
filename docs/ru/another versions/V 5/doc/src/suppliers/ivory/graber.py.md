# Модуль `src.suppliers.ivory.graber`

## Обзор

Модуль `src.suppliers.ivory.graber` содержит класс `Graber`, который предназначен для сбора информации о товарах с сайта `ivory.co.il`.
Этот класс наследуется от базового класса `Graber` (`Grbr`) и переопределяет некоторые его методы для нестандартной обработки полей, специфичных для данного поставщика.

## Подробнее

Модуль предоставляет функциональность для извлечения данных о товарах с веб-страниц `ivory.co.il`. Он использует веб-драйвер для взаимодействия с сайтом и извлекает необходимые данные, такие как наименование, описание, характеристики и цены товаров. Если для какого-либо поля требуется нестандартная обработка, соответствующий метод переопределяется в классе `Graber`.

Перед отправкой запроса к веб-драйверу можно выполнить предварительные действия с использованием декоратора `@close_pop_up`. Этот декоратор позволяет закрывать всплывающие окна или выполнять другие необходимые действия перед обработкой страницы товара.

## Классы

### `Graber`

**Описание**: Класс `Graber` предназначен для сбора информации о товарах с сайта `ivory.co.il`.

**Как работает класс**:
Класс наследуется от базового класса `Graber` (`Grbr`) и инициализируется с указанием префикса поставщика `'ivory'`, драйвера веб-браузера и индекса языка. Он также устанавливает значение `Context.locator_for_decorator` в `None`, что указывает на отсутствие необходимости выполнения каких-либо действий в декораторе `@close_pop_up` по умолчанию.

**Методы**:
- `__init__`: Инициализирует экземпляр класса `Graber`.

#### `__init__`

```python
def __init__(self, driver: Driver, lang_index):
    """Инициализация класса сбора полей товара."""
```

**Описание**: Инициализирует экземпляр класса `Graber`.

**Как работает функция**:
Конструктор класса `Graber` выполняет следующие действия:
1. Устанавливает атрибут `supplier_prefix` равным `'ivory'`.
2. Вызывает конструктор родительского класса `Graber` (`Grbr`) с указанием префикса поставщика, драйвера веб-браузера и индекса языка.
3. Устанавливает атрибут `Context.locator_for_decorator` в `None`.

**Параметры**:
- `driver` (Driver): Драйвер веб-браузера, используемый для взаимодействия с сайтом.
- `lang_index` (int): Индекс языка, используемый для выбора языковой версии сайта.
