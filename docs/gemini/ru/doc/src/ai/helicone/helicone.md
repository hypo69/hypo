# Модуль helicone.py

## Обзор

Модуль `helicone.py` содержит конфигурацию для взаимодействия с сервисом Helicone AI.  Он определяет переменную `MODE`, которая, предположительно, задаёт режим работы (например, 'dev', 'prod').  В настоящий момент, документация содержит лишь заготовки для описания. Необходима подробная информация о функциональности модуля.

## Переменные

### `MODE`

**Описание**: Переменная, определяющая режим работы приложения.  Предполагается, что она может принимать значения типа `str`, например, 'dev' или 'prod'.  В настоящий момент установлено значение `'dev'`.

**Значение**: `'dev'`
