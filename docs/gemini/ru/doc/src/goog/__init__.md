# Модуль hypotez/src/goog

## Обзор

Этот модуль предоставляет интерфейс для работы с Google Spreadsheets.

## Переменные

### `MODE`

**Описание**:  Устанавливает режим работы модуля, например, 'dev' или 'prod'.

**Значение**:  `'dev'` по умолчанию.

## Классы

### `Spreadsheet`

**Описание**: Класс для работы с Google Spreadsheets. Подробнее см. [документацию класса](/docs/hypotez/src/goog/spreadsheet.md).
