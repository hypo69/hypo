# Модуль: src.suppliers.amazon.scenario

## Обзор

Модуль предназначен для сбора информации о товарах с сайта Amazon, начиная со страниц категорий. Он включает в себя функции для извлечения списка категорий и товаров, а также для обработки информации о каждом товаре. Модуль адаптирован для работы с веб-драйвером и поддерживает уникальные сценарии обработки для каждого поставщика.

## Подробней

Этот модуль является частью системы сбора данных о товарах от поставщиков для проекта `hypotez`. Он специализируется на работе с Amazon, автоматизируя процесс извлечения данных о товарах из категорий. Основная цель модуля - собрать как можно больше информации о товарах, представленных на Amazon, для последующей обработки и интеграции с другими частями системы, такими как PrestaShop.

## Функции

### `get_list_products_in_category`

```python
def get_list_products_in_category(s) -> list[str,str,None]:
    """ Returns list of products urls from category page

    @returns list or one of products urls or None

```

**Описание**: Извлекает список URL товаров со страницы категории на Amazon.

**Как работает функция**:
1. Функция принимает экземпляр класса `Supplier` в качестве аргумента, содержащего необходимые инструменты и настройки для взаимодействия с сайтом Amazon.
2. Извлекает локаторы элементов страницы из атрибута `locators['category']` объекта `Supplier`. Локаторы используются для поиска элементов на странице.
3. Прокручивает страницу для загрузки всех товаров, используя метод `scroll()` драйвера.
4. Извлекает список URL товаров, используя локатор `product_links` и метод `execute_locator()`.
5. Проверяет, найдены ли ссылки на товары. Если нет, записывает предупреждение в лог.
6. Преобразует результат в список, если он представлен строкой.
7. Записывает информацию о количестве найденных товаров в лог.

**Параметры**:
- `s` (Supplier): Экземпляр класса `Supplier`, содержащий настройки и инструменты для работы с сайтом.

**Возвращает**:
- `list[str,str,None]`: Список URL товаров или `None`, если товары не найдены.
