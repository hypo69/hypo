# Модуль hypotez/src/logger/header.py

## Обзор

Этот модуль отвечает за определение корневого пути к проекту `hypotez`, что используется для корректного импорта модулей. Он также загружает настройки из файла `settings.json` и документацию из `README.MD`.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с текущей директории и двигаясь вверх по дереву каталогов. Останавливается на первом каталоге, содержащем один из указанных файлов-маркеров (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или каталогов, используемых для определения корневого каталога проекта. По умолчанию содержит ('pyproject.toml', 'requirements.txt', '.git').

**Возвращает**:

- `Path`: Путь к корневому каталогу проекта, если он найден. В противном случае возвращает путь к директории, где расположен текущий скрипт.

**Вызывает исключения**:

- Нет.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root`.

**Тип**: `Path`

### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json`.

**Тип**: `dict` | `None`

### `doc_str`

**Описание**: Строка с документацией проекта, загруженная из файла `README.MD`.

**Тип**: `str` | `None`

### `__project_name__`

**Описание**: Название проекта, полученное из настроек. Если настройки не найдены, используется значение по умолчанию - 'hypotez'.

**Тип**: `str`

### `__version__`

**Описание**: Версия проекта, полученная из настроек. Если настройки не найдены, используется значение по умолчанию - пустая строка.

**Тип**: `str`

### `__doc__`

**Описание**: Документация проекта, полученная из файла `README.MD`. Если файл не найден, используется значение по умолчанию - пустая строка.

**Тип**: `str`

### `__details__`

**Описание**: Подробная информация о проекте (по умолчанию пустая строка).

**Тип**: `str`

### `__author__`

**Описание**: Автор проекта, полученный из настроек. Если настройки не найдены, используется значение по умолчанию - пустая строка.

**Тип**: `str`

### `__copyright__`

**Описание**: Авторские права на проект, полученные из настроек. Если настройки не найдены, используется значение по умолчанию - пустая строка.

**Тип**: `str`

### `__cofee__`

**Описание**: Ссылка на страницу для пожертвований (кофе) автору. Если настройки не найдены, используется значение по умолчанию.

**Тип**: `str`
