# Tiny Utils - описание библиотеки

## Описание

**Tiny Utils** - это библиотека утилит, предоставляющая набор легких вспомогательных функций для различных распространенных задач. Библиотека включает в себя утилиты для преобразования форматов данных, работы с текстом и файлами, строковыми операциями, форматированием дат и времени, обработкой изображений и т.д. Она организована в несколько модулей для удобного доступа к определенным функциональным возможностям.

## Установка

Для использования **Tiny Utils** необходимо клонировать репозиторий и установить необходимые зависимости, указанные в файле `requirements.txt`.

```bash
git clone https://github.com/hypo69/tiny-utils.git
cd tiny_utils
pip install -r requirements.txt
```

## Обзор модулей

Библиотека содержит несколько подмодулей, каждый из которых отвечает за определенную задачу:

* **Преобразователи**: Модули для преобразования форматов данных, такие как текст в изображение, webp в png, JSON, XML, кодирование Base64 и т.д.
* **Утилиты для строк**: Инструменты для расширенной работы со строками.
* **Операции с файлами**: Функции для работы с файлами (чтение, запись, копирование, удаление, перемещение).
* **Утилиты для дат и времени**: Инструменты для форматирования дат и времени.
* **Утилиты FTP**: Функции для работы с FTP-серверами (подключение, загрузка, скачивание, управление файлами).
* **Утилиты для изображений**: Базовые функции обработки изображений (изменение размера, обрезка, преобразование форматов, применение фильтров).
* **Утилиты для PDF**: Утилиты для работы с PDF-файлами (преобразование, слияние, разделение, извлечение текста).
* **Утилиты для принтера**: Функции для отправки данных на принтер.

## Описание модулей

### Преобразователи

Модуль `convertors` содержит утилиты для преобразования данных между форматами. Модули могут обрабатывать различные типы данных, от CSV до JSON и текста до изображений.

#### Файлы:

* `text2png.py`: Преобразует текстовые данные в файл изображения PNG.
* `tts.py`: Преобразует текст в речь и сохраняет его как аудиофайл.
* `webp2png.py`: Преобразует изображения из формата WebP в PNG.
* `xls.py`: Обработка и манипуляции с XLS-файлами.
* `xml2dict.py`: Преобразует XML-данные в словарь Python.
* `base64.py`: Кодирование и декодирование данных с использованием Base64.
* `csv.py`: Инструменты для парсинга и обработки CSV-файлов.
* `dict.py`: Утилиты для работы со словарями Python.
* `html.py`: Преобразует HTML-контент в различные форматы.
* `json.py`: Утилиты для парсинга и обработки JSON-данных.
* `md2dict.py`: Преобразует Markdown-контент в словарь.
* `ns.py`: Специализированные утилиты для преобразования пространств имён.

(Другие модули описаны аналогично)

## Примеры использования

### Преобразование текста в изображение PNG

```python
from tiny_utils.convertors import text2png

text = "Привет, мир!"
output_path = "выходное_изображение.png"
text2png.convert(text, output_path)
```

### Преобразование XML в словарь

```python
from tiny_utils.convertors import xml2dict

xml_data = "<корень><элемент>Привет</элемент></корень>"
словарь = xml2dict.convert(xml_data)
print(словарь)
```

### Парсинг и обработка JSON

```python
from tiny_utils.convertors import json

json_data = '{"имя": "Иван", "возраст": 30}'
обработанные_данные = json.parse(json_data)
print(обработанные_данные)
```
