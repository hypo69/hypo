# Модуль `graber.py`

## Обзор

Модуль `graber.py` предназначен для сбора значений полей на странице товара веб-сайта `cdata.co.il`. Он содержит класс `Graber`, который наследуется от родительского класса `Graber` (обозначенного как `Grbr`). Для каждого поля товара создана функция обработки, которая может быть перегружена в этом классе для нестандартной обработки.

Модуль использует декоратор для выполнения предварительных действий перед отправкой запроса к веб-драйверу. Декоратор по умолчанию находится в родительском классе, и для его активации необходимо передать значение в `Context.locator`.

## Подробнее

Модуль предназначен для автоматизированного сбора данных о товарах с сайта `cdata.co.il`. Он использует веб-драйвер для взаимодействия с сайтом и извлечения необходимой информации.

Класс `Graber` переопределяет или использует функции обработки полей из родительского класса для адаптации к структуре данных на сайте `cdata.co.il`. Декоратор используется для обработки всплывающих окон или других предварительных действий, которые необходимо выполнить перед основным сбором данных.

## Классы

### `Graber`

**Описание**: Класс для операций захвата данных с сайта `cdata.co.il`.

**Методы**:
- `__init__`: Инициализация класса сбора полей товара.

#### `__init__`

```python
def __init__(self, driver: Driver, lang_index:int):
    """Инициализация класса сбора полей товара."""
```

**Описание**: Инициализирует экземпляр класса `Graber`.

**Параметры**:
- `driver` (Driver): Экземпляр веб-драйвера для взаимодействия с сайтом.
- `lang_index` (int): Индекс языка, используемого на сайте.

**Примеры**:
```python
from src.webdriver.driver import Driver
from src.suppliers.cdata.graber import Graber

# Пример инициализации класса Graber
driver = Driver()
graber = Graber(driver=driver, lang_index=1)
```
