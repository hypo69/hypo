# Модуль helpers

## Обзор

Этот модуль предоставляет функции для работы с цветовыми форматами, включая перевод между HEX и десятичным форматами, а также преобразование HEX в RGB.

## Функции

### `hex_color_to_decimal`

**Описание**: Функция переводит цвет из шестнадцатеричного (HEX) формата в десятичный формат.

**Параметры**:
- `letters` (str): Шестнадцатеричное представление цвета (например, "A", "AA", "ABC").

**Возвращает**:
- `int`: Десятичное представление цвета.

**Вызывает исключения**:
- Нет


### `decimal_color_to_hex`

**Описание**: Функция переводит цвет из десятичного формата в шестнадцатеричный (HEX) формат.

**Параметры**:
- `number` (int): Десятичное представление цвета.

**Возвращает**:
- `str`: Шестнадцатеричное представление цвета.

**Вызывает исключения**:
- Нет


### `hex_to_rgb`

**Описание**: Функция переводит цвет из шестнадцатеричного (HEX) формата в RGB формат.

**Параметры**:
- `hex` (str): Шестнадцатеричное представление цвета (например, "#FF0000", "FF0000").

**Возвращает**:
- `tuple`: RGB представление цвета (кортеж из трех целых чисел).

**Вызывает исключения**:
- Нет
