grandadvance Graber Module
==========================

This module contains the `Graber` class for grabbing product data from Grandadvance. It utilizes the `Driver` class for web interaction, and conforms to the `ProductFields` structure for data representation.  The module includes asynchronous functions for fetching various product details, alongside a decorator for handling pop-up windows.
