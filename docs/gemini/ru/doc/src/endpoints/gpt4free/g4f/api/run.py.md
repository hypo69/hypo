# Модуль запуска API g4f

## Обзор

Модуль предназначен для запуска API g4f (GPT4Free), который предоставляет доступ к различным моделям искусственного интеллекта. В основном, он импортирует и вызывает функцию для запуска API.

## Подробней

Данный модуль является точкой входа для запуска API GPT4Free. Он использует функцию `run_api` из модуля `g4f.api` для инициализации и запуска API-сервера. Параметр `debug=True` включает режим отладки, что полезно для разработки и тестирования.

## Функции

### `run_api`

```python
def run_api(debug: bool = False):
    """
    Запускает API GPT4Free.

    Args:
        debug (bool): Флаг, указывающий, следует ли запускать API в режиме отладки. По умолчанию `False`.

    Returns:
        None

    Raises:
        Exception: Если возникает ошибка при запуске API.
    """
    ...
```

**Назначение**: Запускает API GPT4Free.

**Параметры**:
- `debug` (bool): Флаг, указывающий, следует ли запускать API в режиме отладки. По умолчанию `False`.

**Возвращает**:
- `None`: Функция ничего не возвращает.

**Вызывает исключения**:
- `Exception`: Если возникает ошибка при запуске API.

**Как работает функция**:

1. Функция `run_api` инициализирует и запускает API GPT4Free.
2. Если `debug=True`, API запускается в режиме отладки, что облегчает отладку и тестирование.
3. Функция ничего не возвращает.

**Примеры**:

```python
import g4f.api

# Запуск API в режиме отладки
g4f.api.run_api(debug=True)

# Запуск API без режима отладки
g4f.api.run_api()
```
