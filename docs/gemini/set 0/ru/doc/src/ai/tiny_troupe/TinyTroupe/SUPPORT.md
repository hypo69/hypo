# ИНСТРУКЦИЯ ПО ИСПОЛЬЗОВАНИЮ МОДУЛЯ SUPPORT

## Обзор

Данный модуль содержит инструкции по подаче запросов на поддержку и получению помощи по проекту TinyTroupe.

## Как подать заявку на поддержку

Для отслеживания ошибок и запросов на новые функции используется система GitHub Issues. Перед созданием новой заявки, пожалуйста, проверьте наличие существующих, чтобы избежать дублирования. Для новых запросов создавайте новую задачу (issue) с описанием ошибки или запроса на новую функцию.

## Получение помощи

Для получения помощи и ответов на вопросы об использовании проекта, пожалуйста, обратитесь к нашей Wiki или нашему форуму обсуждений, которые расположены на GitHub.

## Политика поддержки Microsoft

Поддержка TinyTroupe ограничена ресурсами, перечисленными выше.
