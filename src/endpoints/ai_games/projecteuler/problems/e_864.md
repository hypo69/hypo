# Задача No 864
## Суммы по модулю

## Условия
Пусть *S*(*n*) — это сумма всех *x* таких, что 1 ≤ *x* < *n* и  *x*<sup>2</sup> ≡ 1 (mod *n*).
Например, *S*(10) = 1 + 9 = 10.

Пусть *SS*(*n*) = Σ *S*(*k*) для 2 ≤ *k* ≤ *n*.
Например, *SS*(10) = *S*(2) + *S*(3) + ... + *S*(10) = 0 + 4 + 0 + 5 + 0 + 12 + 0 + 13 + 10 = 44.

Известно, что *SS*(10<sup>4</sup>) = 28392910.

Найдите *SS*(10<sup>8</sup>).
