# Модуль: Помощник по программированию

Этот модуль содержит класс `CodeAssistant`, используемый для взаимодействия с различными моделями ИИ, такими как Google Gemini и OpenAI, для задач обработки кода.

## Примеры использования

Пример использования класса `CodeAssistant`:

```python
assistant = CodeAssistant(роль='code_checker', язык='ru', модель=['gemini'])
результат = assistant.обработать_файлы(файлы=['файл1.py', 'файл2.py'], параметры={})
```

## Описание

Модуль предоставляет инструменты для анализа и генерации документации кода.  Поддерживаются различные языки программирования и модели ИИ.

## Платформы

Поддерживаются платформы, на которых работают модели ИИ.  Точный перечень платформ может изменяться.

## Сводка

Модуль предоставляет функционал для взаимодействия с ИИ-моделями, позволяющий анализировать и документировать код.


## Классы

### `CodeAssistant`

Класс `CodeAssistant` используется для взаимодействия с различными моделями ИИ, такими как Google Gemini, и предоставляет методы для анализа и генерации документации для кода.

#### Атрибуты

- `роль`: Роль помощника (например, 'code_checker').
- `язык`: Язык, который будет использовать помощник (например, 'ru').
- `модель`: Список используемых моделей ИИ (например, `['gemini']`).


#### Методы

##### `обработать_файлы`

Метод для обработки файлов кода.

**Описание**: Анализирует и обрабатывает файлы кода, используя заданные параметры.

**Параметры**:
- `файлы` (list): Список файлов для обработки.
- `параметры` (dict, опционально): Дополнительные параметры для настройки обработки.

**Возвращает**:
- list: Результат обработки в виде списка проанализированных данных.

**Вызывает исключения**:
- `FileNotFoundError`: Если указанный файл не найден.
- `ValueError`: В случае некорректного формата входных данных.
- `Exception`: Общее исключение, если произошла непредвиденная ошибка.

**Пример использования**:

```python
assistant = CodeAssistant(роль='code_checker', язык='ru', модель=['gemini'])
результат = assistant.обработать_файлы(файлы=['файл1.py', 'файл2.py'], параметры={})
```
