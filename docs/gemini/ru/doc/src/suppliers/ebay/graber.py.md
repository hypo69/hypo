# Модуль `graber` для eBay

## Обзор

Модуль предназначен для сбора данных о товарах с сайта `ebay.com`. Он содержит класс `Graber`, который наследуется от базового класса `Graber` (`Grbr`) и переопределяет некоторые методы для специфичной обработки данных с eBay. Модуль использует декораторы для выполнения предварительных действий перед запросом к веб-драйверу, например, для закрытия всплывающих окон.

## Подробнее

Этот модуль является частью системы для сбора данных с различных онлайн-площадок. Он специализируется на `ebay.com`, предоставляя методы для извлечения информации о товарах, учитывая особенности структуры страниц этого сайта. Класс `Graber` содержит логику для обработки каждого поля товара, и при необходимости методы обработки могут быть переопределены для нестандартных ситуаций.

## Классы

### `Graber`

**Описание**: Класс `Graber` предназначен для сбора информации о товарах с сайта `ebay.com`. Он наследуется от базового класса `Graber` (`Grbr`) и адаптирует методы сбора данных под особенности eBay.

**Методы**:
- `__init__`: Инициализирует экземпляр класса `Graber`, устанавливает префикс поставщика и вызывает конструктор родительского класса.

**Параметры**:
- `driver` (Driver): Экземпляр веб-драйвера для взаимодействия с сайтом eBay.
- `lang_index`: Индекс языка, используемый для локализации (если применимо).

**Примеры**
```python
from src.webdriver.driver import Driver
from src.suppliers.ebay.graber import Graber

# Пример инициализации класса Graber
driver = Driver()  # Предполагается, что класс Driver уже инициализирован
lang_index = 0
ebay_graber = Graber(driver, lang_index)
```
