# Модуль `src.suppliers.morlevi`

## Обзор

Модуль `src.suppliers.morlevi` предназначен для работы с поставщиком данных Morlevi. В настоящее время он содержит импорт класса `Graber`, который, вероятно, отвечает за извлечение данных с использованием специфичной логики для этого поставщика.

## Подробней

Этот модуль служит точкой входа для функциональности, связанной с Morlevi. Он импортирует класс `Graber`, который, по всей видимости, реализует логику сбора и обработки данных от этого поставщика. Модуль предоставляет интерфейс для доступа к этому функционалу, упрощая его использование в других частях проекта `hypotez`.
