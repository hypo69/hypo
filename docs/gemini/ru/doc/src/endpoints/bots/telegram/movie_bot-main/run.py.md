# Модуль запуска Telegram-бота для работы с фильмами

## Обзор

Этот модуль содержит основной код для запуска Telegram-бота, предназначенного для работы с фильмами. Он использует библиотеку `aiogram` для управления ботом, `python-dotenv` для загрузки переменных окружения, а также настраивает логирование и применяет middleware для контроля частоты запросов.

## Подробней

Модуль выполняет следующие задачи:

- Загружает переменные окружения из файла `.env`.
- Инициализирует бота и диспетчер `aiogram`.
- Подключает middleware для ограничения частоты запросов.
- Регистрирует обработчики сообщений из модуля `apps.hendlers`.
- Запускает бота в режиме polling.
- Настраивает логирование для отслеживания работы бота.
