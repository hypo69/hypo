# Задача No 278
## Выражения с тремя простыми числами
## Условия
Рассмотрим выражение вида *an + bm + cp*, где *a*, *b* и *c* - попарно различные простые числа, а *n*, *m* и *p* - положительные целые числа.

Известно, что любое достаточно большое число можно выразить в такой форме.

Например, при *a* = 2, *b* = 3 и *c* = 5:
- 10 = 2 * 2 + 3 * 2 + 5 * 0
- 11 = 2 * 3 + 3 * 0 + 5 * 1
- 12 = 2 * 1 + 3 * 2 + 5 * 1
...

Наименьшее число, которое не может быть выражено в виде *an + bm + cp*, называется числом Фробениуса *F(a, b, c)*.

Например, *F*(2, 3, 5) = 1. Тот факт, что 1 не может быть представлен, очевиден. Также можно проверить, что все числа, большие 1, могут быть представлены.

Если *a*, *b* и *c* простые числа, для которых *a* < *b* < *c*, то *F(a, b, c)* = 11 для *a* = 3, *b* = 5 и *c* = 7 и *F(a, b, c)* = 61 для *a* = 5, *b* = 11 и *c* = 17.

Найдите $\sum F(a, b, c)$, где *a*, *b* и *c* - простые числа, для которых *a* < *b* < *c* < 5000.
