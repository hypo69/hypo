# Модуль для определения токена

## Обзор

Модуль содержит функцию `is_token`, предназначенную для определения, является ли предоставленная строка токеном определенного формата.

## Подробней

Модуль `tok_rec.py` содержит функцию, проверяющую, является ли строка токеном, основываясь на ее длине и наличии определенного символа в определенной позиции.

## Функции

### `is_token`

```python
def is_token(token):
    """
    Определяет, является ли предоставленная строка токеном, основываясь на ее длине и формате.

    Args:
        token (str): Строка для проверки.

    Returns:
        bool: `True`, если строка является токеном, `False` в противном случае.

    """
```

**Как работает функция**:
1. **Разделение токена**: Строка `token` разделяется на части по пробелам, и берется последний элемент.
2. **Получение длины токена**: Определяется длина полученного токена.
3. **Проверка длины**: Проверяется, равна ли длина токена 62 символам.
4. **Проверка формата**: Если длина токена равна 62, проверяется, является ли второй символ символом `/`.
5. **Возврат результата**: Если длина и формат соответствуют, возвращается `True`, иначе `False`.

**Параметры**:
- `token` (str): Строка, которую необходимо проверить на соответствие формату токена.

**Возвращает**:
- `bool`: `True`, если строка является токеном, `False` в противном случае.

**Примеры**:

```python
# Пример 1: Токен с правильным форматом
token1 = "Some string /ThisIsAValidTokenWithLength62AndSlashAtSecondPosition"
result1 = is_token(token1)
print(result1)  # Вывод: False

# Пример 2: Токен с неправильной длиной
token2 = "InvalidTokenWithLengthLessThan62"
result2 = is_token(token2)
print(result2)  # Вывод: False

# Пример 3: Токен с неправильным символом на второй позиции
token3 = "Some string AThisIsAValidTokenWithLength62ButNoSlash"
result3 = is_token(token3)
print(result3)  # Вывод: False
