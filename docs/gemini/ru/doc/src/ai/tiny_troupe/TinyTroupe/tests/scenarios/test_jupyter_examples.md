# Модуль `test_jupyter_examples.py`

## Обзор

Данный модуль содержит тесты для выполнения Jupyter ноутбуков. Он предназначен для автоматической проверки корректности и работоспособности примеров в папке с ноутбуками.

## Функции

### `get_notebooks`

**Описание**: Функция для получения списка путей к Jupyter ноутбукам в заданной папке.

**Параметры**:

- `folder` (str): Путь к папке с Jupyter ноутбуками.

**Возвращает**:

- list[str]: Список путей к файлам Jupyter ноутбуков, удовлетворяющих условиям (`.ipynb`, отсутствие строк ".executed." и ".local." в названии).


### `test_notebook_execution`

**Описание**: Функция для выполнения Jupyter ноутбука и проверки отсутствия исключений.

**Параметры**:

- `notebook_path` (str): Путь к файлу Jupyter ноутбука.

**Возвращает**:

- None

**Вызывает исключения**:

- `pytest.fail`: Если во время выполнения ноутбука произойдет исключение.  В сообщении будет содержаться информация об исключении и пути к ноутбуку.

**Примечания**:

- Функция использует `ExecutePreprocessor` для выполнения ноутбука с заданным таймаутом и именем ядра.
- После выполнения сохраняет копию исполненного ноутбука, добавляя префикс ".executed.local." к имени файла.


## Переменные

### `NOTEBOOK_FOLDER`

**Описание**: Путь к папке, содержащей Jupyter ноутбуки, которые необходимо проверить.

**Значение**: Строка, представляющая путь.

### `TIMEOUT`

**Описание**: Максимальное время выполнения Jupyter ноутбука в секундах.

**Значение**: Целое число.

### `KERNEL_NAME`

**Описание**: Имя ядра для выполнения ноутбука.

**Значение**: Строка, представляющая имя ядра (в данном случае "python3").

## Модули


### `nbformat`

**Описание**: Модуль для работы с форматом Jupyter Notebook.

### `ExecutePreprocessor`

**Описание**: Модуль из `nbconvert` для выполнения Jupyter ноутбуков.

### `pytest`

**Описание**: Модуль для написания и выполнения тестов.

### `os`

**Описание**: Модуль для работы с операционной системой.

### `sys`

**Описание**: Модуль для доступа к системным переменным.


## Примечания

- Необходимо убедиться, что указанный путь `NOTEBOOK_FOLDER` корректен и содержит файлы Jupyter ноутбуков.
- Значение `TIMEOUT` должно быть достаточным для выполнения ноутбуков, но не слишком большим, чтобы предотвратить зависание тестов.
- Ядро `KERNEL_NAME` должно быть совместимо с ноутбуками.
