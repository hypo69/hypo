# Модуль grender

## Обзор

Этот модуль предоставляет инструменты для форматирования и рендеринга таблиц Google Sheets. Он включает классы для работы с ячейками, форматированием, слиянием ячеек, а также настройкой таблиц.

## Классы

### `GSRender`

**Описание**: Класс `GSRender` содержит методы для рендеринга таблиц Google Sheets. Он позволяет задавать различные параметры форматирования, включая цвет фона, выравнивание текста и шрифт.

**Методы**:

#### `__init__`

**Описание**: Инициализирует объект `GSRender`.

**Параметры**:
- `*args`: произвольное количество аргументов.
- `**kwards`: произвольное количество именованных аргументов.

**Возвращает**:
- `None`

#### `render_header`

**Описание**: Рисует заголовок таблицы в первой строке.

**Параметры**:
- `self`: экземпляр класса `GSRender`.
- `ws` (Worksheet): таблица в книге.
- `world_title` (str): заголовок Google таблицы.
- `range` (str = 'A1:Z1'): диапазон ячеек.
- `merge_type` (str('MERGE_ALL') | str('MERGE_COLUMNS') | str('MERGE_ROWS') = 'MERGE_ALL'): тип слияния ('MERGE_ALL', 'MERGE_COLUMNS', 'MERGE_ROWS').

**Возвращает**:
- `None`

#### `merge_range`

**Описание**: Объединяет ячейки в указанном диапазоне в соответствии с типом слияния.

**Параметры**:
- `self`: экземпляр класса `GSRender`.
- `ws` (Worksheet): таблица в книге.
- `range` (str): диапазон ячеек.
- `merge_type` (str('MERGE_ALL') | str('MERGE_COLUMNS') | str('MERGE_ROWS') = 'MERGE_ALL'): тип слияния ('MERGE_ALL', 'MERGE_COLUMNS', 'MERGE_ROWS').

**Возвращает**:
- `None`


#### `set_worksheet_direction`

**Описание**: Устанавливает направление текста в листе.

**Параметры**:
- `self`: экземпляр класса `GSRender`.
- `sh` (Spreadsheet): книга.
- `ws` (Worksheet): лист.
- `direction` (str('ltr') | str('rtl') = 'rtl'): направление ('ltr' или 'rtl').

**Возвращает**:
- `None`


#### `header`

**Описание**: Добавляет строку заголовка в таблицу.

**Параметры**:
- `self`: экземпляр класса `GSRender`.
- `ws` (Worksheet): таблица в книге.
- `ws_header` (str | list): заголовок (строка или список значений).
- `row` (int = None): номер строки для вставки.

**Возвращает**:
- `None`

#### `write_category_title`

**Описание**: Добавляет строку категории в таблицу.

**Параметры**:
- `self`: экземпляр класса `GSRender`.
- `ws` (Worksheet): таблица в книге.
- `ws_category_title` (str | list): заголовок категории (строка или список значений).
- `row` (int = None): номер строки для вставки.

**Возвращает**:
- `None`

#### `get_first_empty_row`

**Описание**: Возвращает номер первой пустой строки в таблице.

**Параметры**:
- `self`: экземпляр класса `GSRender`.
- `ws` (Worksheet): таблица в книге.
- `by_col` (int = None): номер столбца для поиска.

**Возвращает**:
- `int`: номер первой пустой строки.
