# Модуль `src.utils.powershell.examples.pprint`

## Обзор

Данный модуль предназначен для демонстрации работы функций печати с использованием `pprint` и пользовательской функции `pprint` из модуля `src.printer`.

## Содержание

- [Обзор](#обзор)
- [Импорты](#импорты)
- [Функции](#функции)
  - [`pprint`](#pprint)

## Импорты

Модуль импортирует следующие компоненты:
- `header`
- `pprint` как `pretty_print` из стандартной библиотеки `pprint`
- `pprint` из `src.printer`

## Функции

### `pprint`

**Описание**: Выводит строку "Hello, world!" на экран, используя функцию `pprint` из модуля `src.printer`.

**Пример использования:**

```python
pprint("Hello, world!")
```
