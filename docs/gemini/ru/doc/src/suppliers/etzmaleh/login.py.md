# Модуль `login` для поставщика `etzmaleh`

## Обзор

Модуль `login` предназначен для выполнения авторизации поставщика `etzmaleh` через веб-интерфейс. Он содержит функцию `login`, которая принимает объект поставщика и возвращает `True` в случае успешной авторизации. В текущей реализации функция просто логирует факт авторизации и возвращает `True`.
