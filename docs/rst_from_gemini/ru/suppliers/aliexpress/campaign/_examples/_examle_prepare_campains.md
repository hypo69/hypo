Этот скрипт предоставляет примеры использования функций для подготовки кампаний на AliExpress. Он импортирует функции из модуля `prepare_campaigns` и демонстрирует их применение в различных сценариях.

* **Пример 1: Обработка одной категории кампании:**

```python
process_campaign_category("SummerSale", "Electronics", "EN", "USD", force=True)
```

Этот пример обрабатывает категорию "Electronics" для кампании "SummerSale" на английском языке (EN) с использованием доллара США (USD). `force=True` указывает на принудительное пересоздание данных, если они уже существуют.


* **Пример 2: Обработка конкретной кампании:**

```python
process_campaign("WinterSale", categories=["Clothing", "Toys"], language="EN", currency="USD", force=False)
```

Этот пример обрабатывает кампанию "WinterSale", используя категории "Clothing" и "Toys", на английском языке (EN) с использованием доллара США (USD). `force=False` означает, что данные не будут пересозданы, если они уже существуют.


* **Пример 3: Обработка всех кампаний:**

```python
process_all_campaigns(language="EN", currency="USD", force=True)
```

Этот пример обрабатывает все кампании на английском языке (EN) с использованием доллара США (USD).  `force=True` заставляет переработать все данные.
