# Модуль `request_parameters.py`

## Обзор

Модуль `request_parameters.py` содержит определения классов, представляющих собой перечисления (enums) для различных параметров запросов к API AliExpress. Эти классы используются для задания типов продуктов, способов сортировки и типов ссылок при формировании запросов к API.

## Подробней

Модуль определяет классы `ProductType`, `SortBy` и `LinkType`, каждый из которых содержит константы, представляющие собой возможные значения соответствующих параметров API AliExpress. Это позволяет унифицировать и упростить использование API, а также избежать ошибок при формировании запросов.

## Классы

### `ProductType`

**Описание**: Класс, представляющий типы продуктов.

**Методы**: Нет.

**Параметры**: Нет.

**Атрибуты**:
- `ALL` (str): Представляет все типы продуктов (`'ALL'`).
- `PLAZA` (str): Представляет продукты, доступные на площадке Plaza (`'PLAZA'`).
- `TMALL` (str): Представляет продукты, доступные на площадке Tmall (`'TMALL'`).

**Примеры**:
```python
product_type = ProductType()
print(product_type.ALL) # ALL
print(product_type.PLAZA) # PLAZA
```

### `SortBy`

**Описание**: Класс, представляющий способы сортировки продуктов.

**Методы**: Нет.

**Параметры**: Нет.

**Атрибуты**:
- `SALE_PRICE_ASC` (str): Сортировка по возрастанию цены (`'SALE_PRICE_ASC'`).
- `SALE_PRICE_DESC` (str): Сортировка по убыванию цены (`'SALE_PRICE_DESC'`).
- `LAST_VOLUME_ASC` (str): Сортировка по возрастанию объема продаж (`'LAST_VOLUME_ASC'`).
- `LAST_VOLUME_DESC` (str): Сортировка по убыванию объема продаж (`'LAST_VOLUME_DESC'`).

**Примеры**:
```python
sort_by = SortBy()
print(sort_by.SALE_PRICE_ASC) # SALE_PRICE_ASC
print(sort_by.LAST_VOLUME_DESC) # LAST_VOLUME_DESC
```

### `LinkType`

**Описание**: Класс, представляющий типы ссылок.

**Методы**: Нет.

**Параметры**: Нет.

**Атрибуты**:
- `NORMAL` (int): Представляет обычную ссылку (`0`).
- `HOTLINK` (int): Представляет партнерскую ссылку (`2`).

**Примеры**:
```python
link_type = LinkType()
print(link_type.NORMAL) # 0
print(link_type.HOTLINK) # 2
