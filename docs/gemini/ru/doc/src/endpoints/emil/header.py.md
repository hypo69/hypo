# Модуль `header`

## Обзор

Модуль `header.py` предназначен для инициализации основных настроек проекта, таких как корневая директория, загрузка параметров из `settings.json`, а также чтение документации из `README.MD`. Он также предоставляет информацию о проекте, такую как имя, версия, авторские права и ссылку на поддержку разработчика.

## Функции

### `set_project_root`

**Описание**:
Находит корневую директорию проекта, начиная с директории текущего файла, двигаясь вверх и останавливаясь в первой директории, содержащей один из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или директорий для идентификации корневой директории проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена, иначе директория, где расположен скрипт.

## Переменные

### `MODE`

**Описание**:
Определяет режим работы приложения (например, `'dev'` для разработки).

### `__root__`

**Описание**:
Содержит путь к корневой директории проекта, полученный с помощью функции `set_project_root`.

### `settings`

**Описание**:
Словарь, загруженный из `settings.json`, содержащий настройки проекта. Может быть `None`, если файл не найден или произошла ошибка декодирования JSON.

### `doc_str`

**Описание**:
Строка, содержащая содержимое файла `README.MD`. Может быть `None`, если файл не найден или произошла ошибка чтения.

### `__project_name__`

**Описание**:
Имя проекта, полученное из `settings.json` или `'hypotez'` по умолчанию.

### `__version__`

**Описание**:
Версия проекта, полученная из `settings.json` или пустая строка по умолчанию.

### `__doc__`

**Описание**:
Документация проекта, полученная из `README.MD` или пустая строка по умолчанию.

### `__details__`

**Описание**:
Дополнительная информация о проекте. В настоящее время всегда является пустой строкой.

### `__author__`

**Описание**:
Автор проекта, полученный из `settings.json` или пустая строка по умолчанию.

### `__copyright__`

**Описание**:
Авторские права проекта, полученные из `settings.json` или пустая строка по умолчанию.

### `__cofee__`

**Описание**:
Строка, содержащая ссылку на поддержку разработчика, полученная из `settings.json` или стандартное сообщение по умолчанию.
