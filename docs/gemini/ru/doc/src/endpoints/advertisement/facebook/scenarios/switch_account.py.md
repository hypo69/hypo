# Модуль для переключения между аккаунтами в Facebook

## Обзор

Модуль `switch_account.py` предназначен для автоматического переключения между учетными записями в Facebook с использованием Selenium WebDriver. Он содержит функцию `switch_account`, которая выполняет переключение аккаунта, если на странице присутствует соответствующая кнопка.

## Подробней

Этот модуль является частью системы автоматизации рекламы в Facebook и используется для переключения между различными аккаунтами, которые могут использоваться для размещения рекламы. Он использует JSON-файл с локаторами элементов страницы для поиска кнопки переключения аккаунта.

## Функции

### `switch_account`

```python
def switch_account(driver: Driver):
    """ Если есть кнопка `Переключить` - нажимаю её  """
```

**Назначение**: Выполняет переключение между аккаунтами в Facebook, если на странице присутствует кнопка "Переключить".

**Как работает функция**:
1. Функция принимает объект `driver` класса `Driver`, который представляет собой инстанс Selenium WebDriver, настроенный для управления браузером.
2. Функция использует метод `execute_locator` объекта `driver` для поиска элемента на странице, соответствующего локатору `locator.switch_to_account_button`.
3. Если элемент найден и является кнопкой "Переключить", функция нажимает на эту кнопку, что приводит к переключению между аккаунтами.

**Параметры**:
- `driver` (Driver): Объект `Driver`, представляющий собой инстанс Selenium WebDriver.

**Возвращает**:
- `None`: Функция ничего не возвращает.

**Вызывает исключения**:
- Функция не вызывает исключений напрямую, но может возникнуть исключение, если не удается найти элемент на странице.

**Примеры**:
```python
from src.webdriver.driver import Driver
from selenium import webdriver

# Создание инстанса драйвера (пример с Chrome)
driver = Driver(webdriver.Chrome)
driver.get("https://www.facebook.com")  # Открываем Facebook (или другую страницу)

# Вызов функции для переключения аккаунта
switch_account(driver)
