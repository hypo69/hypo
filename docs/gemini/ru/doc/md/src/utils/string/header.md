# Модуль hypotez/src/logger

## Обзор

Модуль `src.logger` определяет корневой путь к проекту. Все импорты строятся относительно этого пути. В дальнейшем предполагается перенести определение корневого пути в системную переменную.


## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла, ищет вверх по иерархии директорий, пока не найдет директорию, содержащую один из указанных маркеров файлов.

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, которые используются в качестве маркеров для определения корневой директории проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена, иначе путь к директории, в которой находится текущий скрипт.

**Пример использования**:

```python
project_root = set_project_root()
print(project_root)
```

**Примечания**:

- Добавляет корневой путь в `sys.path`, если он еще не присутствует. Это важно для импорта модулей, находящихся в корневом каталоге проекта.


### `__project_name__`

**Описание**: Получает имя проекта из файла settings.json, по умолчанию 'hypotez'.


**Возвращает**:
- `str`: Имя проекта, полученное из файла settings.json, или значение по умолчанию.


### `__version__`

**Описание**: Получает версию проекта из файла settings.json, по умолчанию ''.


**Возвращает**:
- `str`: Версия проекта, полученная из файла settings.json, или значение по умолчанию.


### `__doc__`

**Описание**: Получает описание проекта из файла README.MD, по умолчанию ''.


**Возвращает**:
- `str`: Описание проекта, полученное из файла README.MD, или значение по умолчанию.


### `__details__`

**Описание**: Получает дополнительные детали из файла settings.json, по умолчанию ''.


**Возвращает**:
- `str`: Дополнительные детали проекта, полученные из файла settings.json, или значение по умолчанию.


### `__author__`

**Описание**: Получает имя автора проекта из файла settings.json, по умолчанию ''.


**Возвращает**:
- `str`: Имя автора проекта, полученное из файла settings.json, или значение по умолчанию.


### `__copyright__`

**Описание**: Получает копирайт проекта из файла settings.json, по умолчанию ''.


**Возвращает**:
- `str`: Копирайт проекта, полученное из файла settings.json, или значение по умолчанию.


### `__cofee__`

**Описание**: Получает ссылку на поддержку автора через Boosty из файла settings.json, по умолчанию ссылка на поддержку.


**Возвращает**:
- `str`: Ссылка на поддержку автора, полученная из файла settings.json, или значение по умолчанию.


**Обработка исключений**:

- `FileNotFoundError`: В случае, если файл `settings.json` или `README.MD` не найден.
- `json.JSONDecodeError`: В случае, если файл `settings.json` имеет неправильный формат JSON.


## Переменные

### `MODE`

**Описание**: Строковая переменная, хранящая режим работы (например, 'dev', 'prod').


### `__root__`

**Описание**: Путь к корневому каталогу проекта, полученный функцией `set_project_root()`.


### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженный из файла `settings.json`. Может быть `None`, если файл не найден или некорректен.


### `doc_str`

**Описание**: Строка, содержащая описание проекта, загруженная из файла `README.MD`. Может быть `None`, если файл не найден или некорректен.
