## Список эндпоинтов

| Метод        | URL                                  | Описание                                                                                                  |
|---------------|-----------------------------------------|-------------------------------------------------------------------------------------------------------------|
| GET           | `/aviad/flights`                       | Получение списка всех рейсов.                                                                              |
| GET           | `/aviad/flights/{flightId}`             | Получение информации о конкретном рейсе по его ID.                                                           |
| POST          | `/aviad/flights`                       | Создание нового рейса.                                                                                      |
| PUT           | `/aviad/flights/{flightId}`             | Обновление данных о рейсе.                                                                                   |
| DELETE        | `/aviad/flights/{flightId}`             | Удаление рейса.                                                                                          |
| GET           | `/aviad/users`                       | Получение списка всех пользователей.                                                                         |
| GET           | `/aviad/users/{userId}`                | Получение информации о конкретном пользователе по его ID.                                                    |
| POST          | `/aviad/users`                       | Создание нового пользователя.                                                                                  |
| PUT           | `/aviad/users/{userId}`                | Обновление данных о пользователе.                                                                               |
| DELETE        | `/aviad/users/{userId}`                | Удаление пользователя.                                                                                      |
| GET           | `/aviad/bookings`                      | Получение списка всех бронирований.                                                                         |
| POST          | `/aviad/bookings`                      | Создание новой брони.                                                                                       |
| GET           | `/aviad/bookings/{bookingId}`           | Получение информации о конкретном бронировании по его ID.                                                       |
| ...          | ...                                      | ...                                                                                                   |


## Описание параметров

Подробное описание параметров, которые принимают и возвращают эндпоинты.
Например:

* **`/aviad/flights` (GET):**
    * `startDate`: Дата начала (формат YYYY-MM-DD).
    * `endDate`: Дата окончания (формат YYYY-MM-DD).
    * `destination`: Название пункта назначения (строка).

## Возможные ошибки

Список возможных ошибок и их кодов состояния.

* **400 Bad Request:** Некорректные параметры запроса.
* **404 Not Found:** Запрашиваемый ресурс не найден.
* **401 Unauthorized:** Необходима авторизация.
* **500 Internal Server Error:** Внутренняя ошибка сервера.
