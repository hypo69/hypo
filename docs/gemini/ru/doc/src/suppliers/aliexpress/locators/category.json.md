# Документация для `category.json`

## Обзор

Файл `category.json` содержит JSON-структуру, определяющую локаторы для извлечения данных о товарах и навигации по страницам категорий на AliExpress. Он включает в себя локаторы для ссылок на товары и для пагинации (переключения страниц).

## Структура JSON

### `product_links`

**Описание**:
Содержит информацию для локализации ссылок на страницы товаров в категории.

**Поля**:
- `attribute` (str): Атрибут, из которого нужно извлечь ссылку. В данном случае - `href`.
- `by` (str): Метод локализации элемента. В данном случае - `XPATH`.
- `selector` (str): Строка XPATH для поиска элемента.
- `if_list` (str):  Указывает, как обрабатывать список элементов. В данном случае - `first` (брать первый элемент).
- `use_mouse` (bool): Указывает, нужно ли использовать мышь для взаимодействия. В данном случае `false`.
- `timeout` (int): Время ожидания элемента. В данном случае - `0`.
- `timeout_for_event` (str): Условие таймаута, событие, при котором нужно ожидать появления элемента.  В данном случае - `presence_of_element_located`.
- `event` (str | null): Событие, которое нужно вызвать после определения элемента. В данном случае - `null`.
- `mandatory` (bool): Указывает, является ли локатор обязательным для работы. В данном случае - `true`.
- `locator_description` (str): Описание локатора.

### `pagination`

**Описание**:
Содержит информацию для навигации по страницам категории. Включает в себя локаторы для контейнера пагинации и для кнопки "следующая страница".

#### `ul`

**Описание**:
Содержит информацию для локализации контейнера пагинации.

**Поля**:
- `attribute` (str | null): Атрибут, который нужно извлечь. В данном случае - `null`.
- `by` (str): Метод локализации элемента. В данном случае - `XPATH`.
- `selector` (str): Строка XPATH для поиска элемента.
-  `timeout` (int): Время ожидания элемента. В данном случае - `0`.
- `timeout_for_event` (str): Условие таймаута, событие, при котором нужно ожидать появления элемента. В данном случае - `presence_of_element_located`.
- `event` (str): Событие, которое нужно вызвать после определения элемента. В данном случае - `click()`.
- `mandatory` (bool): Указывает, является ли локатор обязательным для работы. В данном случае - `false`.
- `locator_description` (str): Описание локатора.
