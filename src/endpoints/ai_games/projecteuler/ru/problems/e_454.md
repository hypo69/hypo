# Задача No 454
## Гиперболические диофантовы уравнения
## Условия
Рассмотрим диофантово уравнение 1/x + 1/y = 1/n, где x, y и n являются целыми положительными числами.

Для n = 12 существует 3 решения:
1/13 + 1/156 = 1/12
1/16 + 1/48 = 1/12
1/18 + 1/36 = 1/12

Обратите внимание, что решения 1/156 + 1/13, 1/48 + 1/16 и 1/36 + 1/18 считаются теми же решениями, что и приведенные выше.

Мы определим F(N) как количество решений для 1/x + 1/y = 1/n, с 1 ≤ n ≤ N.

Мы можем проверить, что F(1000) = 1069.

Найдите F(10<sup>6</sup>).
