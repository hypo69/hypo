# Модуль `header.py`

## Обзор

Модуль `header.py` предназначен для настройки среды и импорта необходимых модулей. Он также определяет константу `MODE` для указания режима работы приложения (например, `dev` для разработки).

## Содержание

1. [Обзор](#обзор)
2. [Константы](#константы)
3. [Импорты](#импорты)

## Константы

### `MODE`

**Описание**:
Константа, определяющая режим работы приложения.

**Значение**:
`'dev'` (режим разработки)

## Импорты

### `sys`
**Описание**:
Модуль `sys` предоставляет доступ к некоторым переменным и функциям, взаимодействующим с интерпретатором Python.

### `os`
**Описание**:
Модуль `os` предоставляет способ использовать операционные системы зависимые функции.

### `pathlib.Path`
**Описание**:
Модуль `pathlib.Path` предоставляет способ работать с путями файловой системы в более объектно-ориентированном стиле.

## Переменные

### `__root__`
**Описание**:
Абсолютный путь к корневому каталогу проекта.

**Тип**:
`pathlib.Path`
