# PIZZA

## סקירה כללית

המשחק "פיצה" מדמה את תהליך ההזמנה והמשלוח של פיצה. על השחקן לענות על מספר שאלות בנוגע לכמות הפיצות, המחיר שלהן, גובה הטיפ וכו'. בסוף התוכנית מוצגת העלות הכוללת של ההזמנה, כולל מס וטיפ.
