### `test_repeat_on_error`

**Описание**: Тестирует декоратор `repeat_on_error` для повторного вызова функции в случае возникновения указанных исключений.

**Параметры**:
- Нет параметров.

**Возвращает**:
- Нет возвращаемого значения.

**Вызывает исключения**:
- `DummyException`: Если функция вызывает `DummyException` и достигается максимальное количество повторов.
- `RuntimeError`: Если функция вызывает `RuntimeError` и это исключение не указано в списке исключений для повтора.
