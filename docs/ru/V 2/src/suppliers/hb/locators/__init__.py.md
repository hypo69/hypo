# Модуль `src.suppliers.hb.locators`

## Обзор

Модуль `src.suppliers.hb.locators` содержит определения локаторов для работы с поставщиком HB. Модуль также включает информацию о версии и изменениях в локаторах.

- [Версия и документация](#версия-и-документация)
- [Изменения в локаторах](#изменения-в-локаторах)
- [Импорты](#импорты)

## Версия и документация

В модуле присутствует информация о версии, подробное описание и общая документация. 
- `__version__`: Версия модуля.
- `__doc__`: Общая документация модуля.
- `__details__`: Подробное описание модуля.

## Изменения в локаторах

Модуль содержит комментарий, предупреждающий об осторожном применении изменений в локаторах.

## Импорты

В модуле производятся следующие импорты:

- `Version` из `packaging.version`
- `__version__`, `__doc__`, `__details__` из `.version`
- `locator` из `.locator`
