# Модуль `prepare_all_camapaigns.py`

## Обзор

Модуль `prepare_all_camapaigns.py` предназначен для запуска процесса обработки всех рекламных кампаний AliExpress. Он импортирует необходимые модули и функции для выполнения этой задачи.

## Подробней

Основная цель этого модуля - обеспечить запуск процесса создания или обновления партнерских ссылок (affiliate) для рекламных кампаний на платформе AliExpress. Если рекламная кампания не существует, она будет создана. Модуль использует функцию `process_all_campaigns` из подмодуля `src.suppliers.aliexpress.campaign` для выполнения этой задачи. Расположение файла в структуре проекта указывает на его роль в подготовке и обработке рекламных кампаний AliExpress.
