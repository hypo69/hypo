# Модуль для обработки кода с помощью помощника по программированию

## Обзор

Этот модуль содержит класс `CodeAssistant`, предназначенный для взаимодействия с различными моделями ИИ, такими как Google Gemini и OpenAI, для задач обработки кода.  Модуль позволяет анализировать и генерировать документацию для кода.

## Примеры использования

```python
from ai.prompts.developer.code_assistant import CodeAssistant

assistant = CodeAssistant(role='code_checker', lang='ru', model=['gemini'])
result = assistant.process_files(files=['file1.py', 'file2.py'], options={})
print(result)
```

## Плаформа и сводка

Модуль поддерживает платформы, на которых работают модели ИИ, используемые для обработки кода.  Сводка модуля — обеспечение взаимодействия с моделями ИИ для анализа кода и создания документации.


## Классы

### `CodeAssistant`

**Описание**: Класс для работы с помощником по программированию. Используется для взаимодействия с различными моделями ИИ, такими как Google Gemini, и обеспечивает методы для анализа и генерации документации для кода.


**Атрибуты**:

- `role` (str): Роль помощника (например, 'code_checker').
- `lang` (str): Язык, который будет использовать помощник (например, 'ru').
- `model` (list): Список используемых моделей ИИ (например, ['gemini']).


**Методы**:

#### `process_files`

**Описание**: Метод для обработки файлов кода.

**Параметры**:

- `files` (list): Список файлов для обработки.
- `options` (dict, optional): Дополнительные параметры для настройки обработки. По умолчанию `{}`.

**Возвращает**:

- list: Список результатов обработки, представленный в виде анализированных данных.


**Примеры использования**:

```python
assistant = CodeAssistant(role='code_checker', lang='ru', model=['gemini'])
result = assistant.process_files(files=['file1.py', 'file2.py'], options={})
print(result)
```

**Возможные исключения**:

- `ValueError`: Возникает при некорректном формате входных данных.
- `Exception`:  Возникает при других необработанных ошибках во время работы с моделями ИИ.
