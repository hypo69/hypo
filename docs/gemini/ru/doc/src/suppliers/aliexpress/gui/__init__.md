# Модуль hypotez/src/suppliers/aliexpress/gui/__init__.py

## Обзор

Данный модуль предоставляет инициализацию для графического интерфейса (GUI) поставщика AliExpress.

## Постоянные значения

### `MODE`

**Описание:**  Переменная, определяющая режим работы GUI.

**Значение:**  'dev' (в данном примере).

**Примечание:**  Значение может быть изменено для различных режимов работы (например, 'prod' для производства).
