# Модуль hypotez/src/webdriver/firefox/extentions/test_extention - Copy/html/popup.html

## Обзор

Данный HTML-файл представляет собой шаблон всплывающего окна (popup) для расширения Firefox. Он отображает заголовок "Hypotez" и абзац с приветствием.

## Содержание

### Структура HTML

Файл `popup.html` содержит стандартную структуру HTML-документа:

- `<!DOCTYPE html>`: Декларация типа документа.
- `<html lang="en">`: Корневой элемент HTML.
- `<head>`: Мета-данные страницы.
- `<meta charset="UTF-8">`: Установка кодировки символов.
- `<meta name="viewport" content="width=device-width, initial-scale=1.0">`: Определение настроек отображения.
- `<title>Hypotez</title>`: Заголовок страницы.
- `<body>`: Основное содержимое страницы.
- `<h1>Hypotez</h1>`: Заголовок первого уровня.
- `<p>Привет, Это Давидка. Я обучаю модель</p>`: Абзац с текстом.


## Переменные

### `MODE`

**Тип**: `str`

**Описание**: Переменная, хранящая режим работы, в данном случае `'debug'`.  Значение может использоваться для настройки поведения приложения.
