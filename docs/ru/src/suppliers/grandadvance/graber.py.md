# Модуль для сбора данных о товарах с сайта grandadvance.co.il

## Обзор

Модуль содержит класс `Graber`, который предназначен для сбора информации о товарах с сайта [grandadvance.co.il](https://grandadvance.co.il). Он наследует функциональность от базового класса `Graber` и переопределяет методы для обработки специфических полей, если это необходимо.

## Подробней

Этот модуль является частью системы сбора данных о товарах с различных онлайн-магазинов. Он использует веб-драйвер для взаимодействия с сайтом grandadvance.co.il и извлекает необходимую информацию о товарах, такую как наименование, описание, характеристики и цену. Собранные данные могут быть использованы для анализа рынка, сравнения цен и других целей.

## Классы

### `Graber`

**Описание**: Класс `Graber` наследуется от класса `Graber` (который, вероятно, определен в модуле `src.suppliers.graber`) и предназначен для сбора данных о товарах с сайта grandadvance.co.il. Он использует веб-драйвер для навигации по сайту и извлечения информации о товарах.

**Как работает класс**:

1.  **Инициализация**: При создании экземпляра класса `Graber` происходит инициализация конфигурации и локаторов, необходимых для работы с сайтом grandadvance.co.il.
2.  **Наследование**: Класс наследует методы от базового класса `Graber` для выполнения стандартных операций, таких как навигация по сайту, извлечение данных и обработка ошибок.
3.  **Переопределение методов**: При необходимости класс может переопределять методы базового класса для выполнения специфических задач, связанных с сайтом grandadvance.co.il. Например, можно переопределить метод для извлечения цены товара, если она отображается на сайте нестандартным образом.
4.  **Использование декораторов**: Для выполнения предварительных действий перед отправкой запроса к веб-драйверу можно использовать декораторы. Декоратор по умолчанию находится в родительском классе. Для того, чтобы декоратор сработал, нужно передать значение в `Context.locator`. Если необходимо реализовать свой декоратор, нужно раскомментировать строки с декоратором и переопределить его поведение.

**Методы**:

*   `__init__`: Конструктор класса.

    *   **Описание**: Инициализирует экземпляр класса `Graber`.
    *   **Параметры**:

        *   `driver` (`Driver`): Экземпляр веб-драйвера, используемый для взаимодействия с сайтом.
        *   `lang_index` (`int`): Индекс языка, используемого на сайте.
    *   **Как работает**:

        1.  Загружает конфигурацию из JSON-файла `grandadvance.json`.
        2.  Загружает локаторы элементов из JSON-файла `product.json`.
        3.  Вызывает конструктор родительского класса `Graber` с указанием префикса поставщика (`ENDPOINT`), драйвера и индекса языка.
        4.  Устанавливает локатор для декоратора, который будет использоваться для выполнения предварительных действий перед отправкой запроса к веб-драйверу.
