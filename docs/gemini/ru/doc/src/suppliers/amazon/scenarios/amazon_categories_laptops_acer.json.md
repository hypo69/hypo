# Документация для `amazon_categories_laptops_acer.json`

## Обзор

Данный файл содержит JSON-конфигурацию для сценариев сбора данных о ноутбуках Acer с сайта Amazon. Конфигурация включает информацию о магазине, а также набор сценариев для различных моделей ноутбуков Acer, классифицированных по процессорам (Intel Celeron, Intel i3, Intel i5, Ryzen 3) и размерам экрана.

## store
  
  **Описание**:  Содержит информацию о магазине на Amazon, откуда будут собираться данные.

  -   `store_id` (str): Идентификатор магазина. В данном случае пустая строка.
  -   `supplier_id` (str): Идентификатор поставщика. В данном случае пустая строка.
  -   `get store banners` (bool): Флаг, указывающий необходимость получения баннеров магазина. В данном случае `true`.
  -   `description` (str): Описание магазина. В данном случае "Asus laptops".
  -   `about` (str): Дополнительная информация о магазине. В данном случае "Asus laptops".
  -   `url` (str): URL-адрес страницы магазина на Amazon.
  -   `shop categories page` (str): URL-адрес страницы с категориями магазина. В данном случае пустая строка.
  -   `shop categories json file` (str): Путь к JSON-файлу с категориями магазина. В данном случае пустая строка.

## scenarios

Раздел содержит список сценариев для сбора данных о конкретных моделях ноутбуков Acer. Каждый сценарий представляет собой объект, ключом которого является название сценария, а значением - подробная конфигурация.

### `LAPTOP ACER INTEL CELERON 11`

**Описание**: Сценарий для сбора данных о ноутбуках Acer с процессором Intel Celeron и размером экрана 11 дюймов.

**Параметры**:
- `brand` (str): Бренд ноутбука. В данном случае `"ACER"`.
- `url` (str): URL-адрес страницы Amazon с соответствующими ноутбуками.
- `active` (bool): Флаг, указывающий, активен ли сценарий. В данном случае `true`.
- `condition` (str): Состояние товаров (например, `"new"`).
- `presta_categories` (dict): Категории PrestaShop, в которые будут отнесены собранные данные. В данном случае `{"template": { "acer": [ "LAPTOPS INTEL CELERON", "11" ] }}`.
- `checkbox` (bool): Флаг, используемый для UI. В данном случае `false`.
- `price_rule` (int): Правило для расчета цены. В данном случае `1`.
