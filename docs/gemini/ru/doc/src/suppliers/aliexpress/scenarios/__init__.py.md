# Модуль `hypotez/src/suppliers/aliexpress/scenarios/__init__.py`

## Обзор

Модуль содержит сценарии для работы с AliExpress. Он предоставляет набор функций и классов для автоматизации различных процессов на платформе AliExpress, например, вход в систему.

## Содержание

- [Обзор](#обзор)
- [Переменные](#переменные)
- [Импорты](#импорты)

## Переменные

### `MODE`

**Описание**: Определяет режим работы модуля. По умолчанию установлен в `dev`.

## Импорты

### `from .login import login`

**Описание**: Импортирует модуль `login` из текущей директории, который содержит сценарий для входа в систему AliExpress.
