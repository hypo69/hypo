# Модуль `grender.py`

## Обзор

Модуль `grender.py` предназначен для рендеринга таблиц Google Sheets, предоставляя возможности для стилизации заголовков, объединения ячеек и установки направления текста. Он использует библиотеки `gspread`, `gspread_formatting` и другие вспомогательные модули для работы с Google Sheets API.

## Классы

### `GSRender`

**Описание**: Класс `GSRender` предоставляет методы для рендеринга и стилизации таблиц Google Sheets.

#### `__init__`

**Описание**: Конструктор класса `GSRender`.

**Параметры**:
- `*args`: Произвольные позиционные аргументы.
- `**kwards`: Произвольные именованные аргументы.

**Возвращает**:
- `None`: Метод ничего не возвращает.

#### `render_header`

**Описание**: Метод для отрисовки заголовка таблицы в первой строке.

**Параметры**:
- `ws` (`Worksheet`): Таблица в Google Sheet.
- `world_title` (`str`): Заголовок таблицы.
- `range` (`str`, optional): Диапазон ячеек для заголовка, по умолчанию `A1:Z1`.
- `merge_type` (`str`, optional): Тип объединения ячеек, по умолчанию `MERGE_ALL`. Возможные значения: `MERGE_ALL`, `MERGE_COLUMNS`, `MERGE_ROWS`.

**Возвращает**:
- `None`: Метод ничего не возвращает.

#### `merge_range`

**Описание**: Метод для объединения ячеек в заданном диапазоне.

**Параметры**:
- `ws` (`Worksheet`): Таблица в Google Sheet.
- `range` (`str`): Диапазон ячеек для объединения.
- `merge_type` (`str`, optional): Тип объединения ячеек, по умолчанию `MERGE_ALL`. Возможные значения: `MERGE_ALL`, `MERGE_COLUMNS`, `MERGE_ROWS`.

**Возвращает**:
- `None`: Метод ничего не возвращает.

#### `set_worksheet_direction`

**Описание**: Метод для установки направления текста на листе (справа налево или слева направо).

**Параметры**:
- `sh` (`Spreadsheet`): Объект таблицы Google Sheets.
- `ws` (`Worksheet`): Таблица в Google Sheet.
- `direction` (`str`, optional): Направление текста, по умолчанию `rtl`. Возможные значения: `ltr`, `rtl`.

**Возвращает**:
- `None`: Метод ничего не возвращает.

#### `header`

**Описание**: Метод для добавления заголовка в таблицу.

**Параметры**:
- `ws` (`Worksheet`): Таблица в Google Sheet.
- `ws_header` (`str | list`): Заголовок таблицы (строка или список строк).
- `row` (`int`, optional): Номер строки для вставки заголовка, по умолчанию `None`. Если `None`, заголовок добавляется в первую пустую строку.

**Возвращает**:
- `None`: Метод ничего не возвращает.

#### `write_category_title`

**Описание**: Метод для добавления заголовка категории в таблицу.

**Параметры**:
- `ws` (`Worksheet`): Таблица в Google Sheet.
- `ws_category_title` (`str | list`): Заголовок категории (строка или список строк).
- `row` (`int`, optional): Номер строки для вставки заголовка категории, по умолчанию `None`. Если `None`, заголовок добавляется в первую пустую строку.

**Возвращает**:
- `None`: Метод ничего не возвращает.

#### `get_first_empty_row`

**Описание**: Метод для получения номера первой пустой строки в таблице.

**Параметры**:
- `ws` (`Worksheet`): Таблица в Google Sheet.
- `by_col` (`int`, optional): Номер колонки, по которой нужно искать пустую строку, по умолчанию `None`. Если `None`, поиск ведется по всей таблице.

**Возвращает**:
- `int`: Номер первой пустой строки.
