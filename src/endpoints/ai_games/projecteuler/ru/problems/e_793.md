# Задача No 793
## Упрощенный вариант игры 2048
## Условия
Рассмотрим упрощенную версию популярной игры 2048.

Игровое поле представляет собой строку из N (1 ≤ N ≤ 10) целых чисел. Каждый ход игрок выбирает один из двух вариантов:

* **Движение влево**: Все ненулевые числа сдвигаются влево, а соседние одинаковые числа заменяются их суммой. Пустые места заполняются нулями.
* **Движение вправо**: Все ненулевые числа сдвигаются вправо, а соседние одинаковые числа заменяются их суммой. Пустые места заполняются нулями.

Например, если поле изначально равно \[1, 2, 2, 3, 0], то после движения влево оно станет \[1, 4, 3, 0, 0], а после движения вправо - \[0, 1, 4, 3, 0].

Начальное состояние поля - строка из N единиц. Цель игры - получить поле, содержащее хотя бы одно число, которое больше или равно заданному значению K (2 ≤ K ≤ 65536).

Определите наименьшее количество ходов, необходимое для достижения цели.

Если цель недостижима, выведите -1.
