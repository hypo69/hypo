# Конфигурационный файл `config.ini`

## Обзор

Файл `config.ini` содержит параметры конфигурации для взаимодействия с OpenAI или Azure OpenAI, а также настройки для моделирования и логирования. Он определяет тип API, параметры модели, настройки кэширования, параметры предотвращения вредоносного контента, а также уровень логирования.

## Раздел `OpenAI`

### **Описание**:
Содержит настройки для взаимодействия с API OpenAI или Azure OpenAI.

### Параметры:

- `API_TYPE` (str): Тип используемого API. Может быть `openai` или `azure`. Значение по умолчанию: `openai`.
- `AZURE_API_VERSION` (str): Версия API Azure OpenAI. Используется, если `API_TYPE` установлен в `azure`. По умолчанию `2023-05-15`.
- `MODEL` (str): Идентификатор используемой модели OpenAI. По умолчанию `gpt-4o`.
- `MAX_TOKENS` (int): Максимальное количество токенов, которое может использовать модель. По умолчанию `4000`.
- `TEMPERATURE` (float): Параметр температуры модели, влияющий на случайность ответов. По умолчанию `0.3`.
- `FREQ_PENALTY` (float): Штраф за частотное повторение токенов. По умолчанию `0.0`.
- `PRESENCE_PENALTY` (float): Штраф за наличие токенов в ответе. По умолчанию `0.0`.
- `TIMEOUT` (int): Время ожидания запроса к API в секундах. По умолчанию `60`.
- `MAX_ATTEMPTS` (int): Максимальное количество попыток повтора запроса в случае ошибки. По умолчанию `5`.
- `WAITING_TIME` (int): Время ожидания между попытками запроса в секундах. По умолчанию `1`.
- `EXPONENTIAL_BACKOFF_FACTOR` (int): Коэффициент экспоненциального увеличения времени ожидания между попытками. По умолчанию `5`.
- `EMBEDDING_MODEL` (str): Идентификатор модели для получения векторных представлений текста. По умолчанию `text-embedding-3-small`.
- `CACHE_API_CALLS` (bool): Указывает, нужно ли кэшировать вызовы API. По умолчанию `False`.
- `CACHE_FILE_NAME` (str): Имя файла для кэширования вызовов API. По умолчанию `openai_api_cache.pickle`.
- `MAX_CONTENT_DISPLAY_LENGTH` (int): Максимальная длина отображаемого текста. По умолчанию `1024`.

## Раздел `Simulation`

### **Описание**:
Содержит настройки для моделирования, в том числе предотвращение вредоносного контента и нарушения авторских прав.

### Параметры:
- `RAI_HARMFUL_CONTENT_PREVENTION` (bool): Включает или выключает предотвращение вредоносного контента. По умолчанию `True`.
- `RAI_COPYRIGHT_INFRINGEMENT_PREVENTION` (bool): Включает или выключает предотвращение нарушения авторских прав. По умолчанию `True`.

## Раздел `Logging`

### **Описание**:
Содержит настройки логирования.

### Параметры:
- `LOGLEVEL` (str): Уровень логирования. Возможные значения: `ERROR`, `WARNING`, `INFO`, `DEBUG`. По умолчанию `ERROR`.
