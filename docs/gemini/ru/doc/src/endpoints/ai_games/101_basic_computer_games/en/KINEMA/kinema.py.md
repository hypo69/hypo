# KINEMA

## Обзор

Данный модуль реализует текстовую игру "Кинема", в которой игрок управляет точкой на плоскости, перемещая ее с помощью команд. Игра позволяет перемещать точку влево, вправо, вверх и вниз, а также отображает текущие координаты.
