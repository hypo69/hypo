## Классы

### `OpenAIAPI`

**Описание**: Класс для взаимодействия с API OpenAI.
Включает методы для генерации текста, изображений и обработки входящих данных от пользователя.

**Методы**:
- `__init__`: Конструктор класса, инициализирует клиента OpenAI и загружает ключи API.
- `get_user_input`: Метод для получения пользовательского ввода.
- `generate_text`: Метод для генерации текста с использованием модели OpenAI.
- `generate_image`: Метод для генерации изображения с использованием модели OpenAI.

## Функции

### `create_chat_completion`

**Описание**: Функция для создания чат-комплишена с использованием API OpenAI.

**Параметры**:
- `client` (openai.OpenAI): Экземпляр клиента OpenAI.
- `messages` (list): Список сообщений для чата.
- `model` (str, optional): Модель, используемая для генерации. По умолчанию "gpt-3.5-turbo".
- `temperature` (float, optional): Температура для генерации. По умолчанию 1.
- `max_tokens` (int, optional): Максимальное количество токенов для ответа. По умолчанию 100.

**Возвращает**:
- `openai.types.chat.chat_completion.ChatCompletion | None`: Объект ChatCompletion или None в случае ошибки.

**Вызывает исключения**:
- `Exception`: Ошибка при обращении к API.

### `create_image`

**Описание**: Функция для создания изображения с использованием API OpenAI.

**Параметры**:
- `client` (openai.OpenAI): Экземпляр клиента OpenAI.
- `prompt` (str): Текст запроса для генерации изображения.
- `size` (str, optional): Размер изображения. По умолчанию "1024x1024".
- `quality` (str, optional): Качество изображения. По умолчанию "standard".
- `style` (str, optional): Стиль изображения. По умолчанию "vivid".

**Возвращает**:
- `str | None`: URL изображения или None в случае ошибки.

**Вызывает исключения**:
- `Exception`: Ошибка при обращении к API.

### `get_answer_from_chat`

**Описание**: Функция для извлечения ответа из объекта ChatCompletion.

**Параметры**:
- `chat_completion` (openai.types.chat.chat_completion.ChatCompletion): Объект ChatCompletion.

**Возвращает**:
- `str | None`: Текст ответа или None если нет ответа.
