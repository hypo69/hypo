# Шаблон для создания HTML отчета из сценария мехирона

## Обзор

Этот шаблон описывает процесс создания HTML отчета на основе данных, полученных из сценария мехирона. Он предназначен для структурирования данных, вывода информации в табличном формате и создания визуализации.

## Структура данных

Структура данных, ожидаемых для обработки, должна быть определена в соответствии с особенностями сценария мехирона.  Подробный формат должен быть представлен в отдельной части документации.

## Функции

### `generate_html_report(data: list, columns: list) -> str`

**Описание**: Функция генерирует HTML строку с отчетом, построенным на данных из сценария мехирона.

**Параметры**:
- `data` (list): Список словарей, где каждый словарь представляет строку данных.
- `columns` (list): Список имен столбцов, соответствующих ключам словарей в `data`.

**Возвращает**:
- `str`: HTML строка, содержащая отчет.

**Вызывает исключения**:
- `TypeError`: Если `data` или `columns` имеют неверный тип.
- `ValueError`: Если не все ключи в `data` соответствуют значениям в `columns`.
