# Модуль alirequests

## Обзор

Модуль `alirequests` представляет собой экспериментальный код для работы с AliExpress. В текущей версии он включает в себя импорт необходимых библиотек и инициализацию веб-драйвера для взаимодействия с сайтом AliExpress.

## Подробней

Этот модуль, вероятно, предназначен для автоматизации каких-либо действий на сайте AliExpress, таких как сбор данных, выполнение поиска или другие операции, требующие взаимодействия с веб-страницами. На данный момент он содержит только базовую настройку для запуска браузера и открытия главной страницы AliExpress.

## Переменные

### `d`

**Описание**: Экземпляр класса `Driver`, используемый для управления браузером.

**Как работает**: Создается экземпляр драйвера `Firefox`, который позволяет управлять браузером Firefox для автоматизации действий на веб-странице.

**Примеры**:

```python
from src.webdriver.driver import Driver, Chrome, Firefox

d = Driver(Firefox)
d.get_url(r"https://www.aliexpress.com")
