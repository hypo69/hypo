# Задача No 883
## Счастливые квадраты
## Условия
Назовем число *счастливым*, если его можно записать как сумму двух квадратов, т.е. если существуют целые числа *a* и *b*, такие, что *n* = *a*² + *b*².

Рассмотрим следующее множество:

    S(n) = {1, 2, ..., n}

Также рассмотрим множество *F*(n), которое содержит все счастливые числа из *S*(n). Например:

    F(10) = {1, 2, 4, 5, 8, 9, 10}

Определим функцию *g*(n) как разницу между *n* и количеством чисел в *F*(n). Например:

    g(10) = 10 - 7 = 3

Далее, определим функцию *G*(n) как сумму *g(i)* для всех *i* от 1 до *n*. Например:

    G(10) = g(1) + g(2) + ... + g(10) = 0 + 0 + 1 + 1 + 1 + 2 + 2 + 2 + 2 + 3 = 14

Известно, что *G*(10⁶) = 63184968.

Найдите *G*(10⁸).
