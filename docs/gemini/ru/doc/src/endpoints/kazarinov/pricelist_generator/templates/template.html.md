# Шаблон HTML для генерации прайс-листа

## Обзор

Данный HTML-файл представляет собой шаблон для генерации прайс-листа. Он использует Jinja2 для динамического заполнения данных о товарах, заголовка, описания и общей стоимости. Шаблон предназначен для отображения списка товаров с их изображениями, описаниями, характеристиками и ценами.

## Структура шаблона

Шаблон состоит из следующих основных частей:

1.  **Заголовок (head)**:
    *   Содержит метаданные, такие как кодировка, viewport и заголовок документа.
    *   Подключает стили CSS для оформления страницы.
2.  **Тело (body)**:
    *   Содержит контейнер для основного содержимого.
    *   **Заголовок страницы (h1)**:
        *   Отображает заголовок прайс-листа, динамически подставляемый через переменную `title`.
    *   **Описание (p class="lead")**:
        *   Отображает описание прайс-листа, динамически подставляемое через переменную `description`.
    *   **Контейнер для товаров (div class="row")**:
        *   Использует flexbox для отображения карточек товаров.
        *   Содержит цикл `{% for product in products %}`, который перебирает список товаров и генерирует карточку для каждого из них.
    *   **Карточка товара (div class="product-card")**:
        *   Отображает заголовок товара (`product.product_title`).
        *   Содержит изображение товара (`product.image_local_saved_path`).
        *   Отображает описание товара (`product.product_description`).
        *   Отображает спецификацию товара (`product.specification`).
    *   **Подвал (div class="footer")**:
        *   Отображает общую стоимость товаров (`price`) и валюту (`currency`).

## Стили CSS

В разделе `<style>` определены следующие стили:

*   **body**:
    *   Устанавливает белый фон и черный текст.
    *   Задает шрифт по умолчанию.
    *   Убирает отступы по умолчанию.
*   **h1**:
    *   Выравнивает заголовок по центру.
    *   Устанавливает размер шрифта.
*   **.lead**:
    *   Выравнивает текст по центру.
    *   Устанавливает размер шрифта.
*   **.container**:
    *   Устанавливает ширину и центрирует контейнер.
*   **.row**:
    *   Использует flexbox для отображения карточек товаров.
*   **.product-card**:
    *   Устанавливает светлый фон и рамку.
    *   Задает отступы и ширину.
*   **.product-card h3**:
    *   Убирает верхний отступ у заголовка.
*   **.product-card img**:
    *   Устанавливает ширину и высоту изображения.
    *   Сохраняет пропорции изображения.
    *   Устанавливает отступ справа от изображения.
*   **.product-info**:
    *   Занимает оставшееся пространство в карточке.
*   **.price-tag**:
    *   Устанавливает цвет фона и текста для ценника.
    *   Задает отступы и скругление.
*   **.footer**:
    *   Выравнивает текст по центру.
    *   Устанавливает верхний отступ и рамку сверху.

## Переменные Jinja2

В шаблоне используются следующие переменные Jinja2:

*   `title`: Заголовок прайс-листа.
*   `description`: Описание прайс-листа.
*   `products`: Список товаров, каждый из которых имеет следующие атрибуты:
    *   `product_title`: Название товара.
    *   `image_local_saved_path`: Путь к изображению товара.
    *   `product_description`: Описание товара.
    *   `specification`: Спецификация товара.
*   `price`: Общая стоимость товаров.
*   `currency`: Валюта.

## Использование

Этот шаблон используется для генерации HTML-страницы с прайс-листом. Переменные Jinja2 заполняются данными о товарах, и результирующий HTML отображается в браузере.

## Примеры использования:

```python
from jinja2 import Environment, FileSystemLoader

# Пример данных для прайс-листа
data = {
    "title": "Прайс-лист на товары",
    "description": "Список товаров с ценами",
    "products": [
        {
            "product_title": "Товар 1",
            "image_local_saved_path": "/path/to/image1.jpg",
            "product_description": "Описание товара 1",
            "specification": "Характеристики товара 1"
        },
        {
            "product_title": "Товар 2",
            "image_local_saved_path": "/path/to/image2.jpg",
            "product_description": "Описание товара 2",
            "specification": "Характеристики товара 2"
        }
    ],
    "price": "1000",
    "currency": "₽"
}

# Загрузка шаблона
env = Environment(loader=FileSystemLoader("."))
template = env.get_template("template.html")

# Генерация HTML
html_output = template.render(data)

# Вывод или сохранение HTML
print(html_output)
```
