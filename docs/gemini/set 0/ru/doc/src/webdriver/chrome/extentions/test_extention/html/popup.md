# Модуль hypotez/src/webdriver/chrome/extentions/test_extention/html/popup.html

## Обзор

Данный файл `popup.html` представляет собой HTML-шаблон для всплывающего окна расширения браузера. Он отображает заголовок "hypotez" и краткое описание функции расширения - сбор данных с текущей веб-страницы при нажатии на иконку.

## Структура файла

Файл `popup.html` содержит стандартный HTML-документ с тегами `<html>`, `<head>`, `<body>`, содержащими заголовок, стиль и текст для отображения в всплывающем окне.


## Стиль

### Стили CSS

Файл содержит CSS-стили для настройки внешнего вида всплывающего окна:

```html
<style>
    body {
        width: 200px;
        padding: 10px;
    }
</style>
```
Это задаёт ширину страницы в 200 пикселей и отступы вокруг содержимого в 10 пикселей.

## Содержимое

### Заголовок

```html
<h1>hypotez</h1>
```

Заголовок "hypotez" отображается в начале всплывающего окна.

### Текст описания

```html
<p>Click the extension icon to collect data from the current webpage.</p>
```
