# Модуль hypotez/src/webdriver/bs/__init__.py

## Обзор

Данный модуль `hypotez/src/webdriver/bs/__init__.py` служит для инициализации и импорта компонентов веб-драйвера, связанных с библиотекой BS (вероятно, Beautiful Soup).  Он определяет глобальную переменную `MODE`, которая устанавливает режим работы (например, `dev` или `prod`).

## Глобальные переменные

### `MODE`

**Описание**: Переменная, хранящая режим работы (`dev` по умолчанию).  Вероятно, используется для настройки поведения других компонентов.

**Значение**: `'dev'`


## Импорты

### `from .bs import BS`

**Описание**: Импортирует класс `BS` из модуля `bs.py` внутри той же папки (`./bs`). Вероятно, этот класс представляет собой реализацию работы с Beautiful Soup.
