# Модуль `header.py`

## Обзор

Модуль `header.py` содержит основную конфигурацию и метаданные для проекта, а также вспомогательные функции для определения корневой директории проекта.

## Функции

### `set_project_root`

**Описание**:
Находит корневую директорию проекта, начиная с директории текущего файла и двигаясь вверх, пока не найдет директорию, содержащую маркерные файлы.

**Параметры**:
- `marker_files` (tuple, optional): Список имен файлов или директорий, которые идентифицируют корень проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена. В противном случае возвращает директорию, где находится скрипт.

## Глобальные переменные

### `MODE`

**Описание**:
Режим работы приложения (по умолчанию `'dev'`).

### `__root__`

**Описание**:
Путь к корневой директории проекта. Определяется функцией `set_project_root`.

### `config`

**Описание**:
Словарь, содержащий конфигурацию проекта, загруженную из `config.json`. Если файл не найден или не является валидным JSON, значение будет `None`.

### `doc_str`

**Описание**:
Строка, содержащая содержимое файла `README.md`. Если файл не найден, значение будет `None`.

### `__project_name__`

**Описание**:
Название проекта. Значение берется из `config.json`, если оно есть, иначе по умолчанию `'hypotez'`.

### `__version__`

**Описание**:
Версия проекта. Значение берется из `config.json`, если оно есть, иначе по умолчанию `''`.

### `__doc__`

**Описание**:
Содержимое файла `README.md`. Если файл не найден, значение будет `''`.

### `__details__`

**Описание**:
Дополнительные детали проекта (по умолчанию `''`).

### `__author__`

**Описание**:
Автор проекта. Значение берется из `config.json`, если оно есть, иначе по умолчанию `''`.

### `__copyright__`

**Описание**:
Информация об авторских правах. Значение берется из `config.json`, если оно есть, иначе по умолчанию `''`.

### `__cofee__`

**Описание**:
Сообщение с призывом поддержать разработчика.
