# ИНСТРУКЦИЯ по использованию модуля SUPPORT

## Обзор

Данный модуль содержит информацию о том, как отправить запрос на поддержку, получить помощь или сообщить об ошибке проекта TinyTroupe.

## Руководство по подаче отчетов об ошибках и запросов на новые функции

Этот проект использует GitHub Issues для отслеживания ошибок и запросов на новые функции. Перед тем, как создать новый запрос, пожалуйста, проверьте, не существует ли уже подобного запроса или ошибки. Для новых ошибок или запросов на новые функции создайте новый Issue.

Для получения помощи и ответов на вопросы об использовании этого проекта, пожалуйста, обратитесь к нашей Wiki или обсудите вопрос на форуме обсуждений, которые находятся прямо здесь, на GitHub.

## Политика поддержки Microsoft

Поддержка TinyTroupe ограничена ресурсами, указанными выше.
