# Модуль `hypotez/src/suppliers/morlevi/graber.py`

## Обзор

Модуль содержит класс `Graber`, предназначенный для сбора данных с веб-страницы товара на сайте `morlevi.co.il`.  Класс наследуется от базового класса `Graber` и предоставляет методы для обработки специфичных данных этого поставщика.  Включает в себя (или может включать) обработку изображений и закрытие всплывающих окон.


## Классы

### `Graber`

**Описание**:  Класс `Graber` реализует логику сбора данных с сайта `morlevi.co.il`.  Он обрабатывает специфичные поля товаров, используя переопределенные методы из базового класса.  Поддерживает предварительные действия (например, закрытие всплывающих окон) перед основным сбором данных.

**Методы**:

- `__init__(self, driver: Driver)`:
    **Описание**: Инициализирует экземпляр класса.
    **Параметры**:
        - `driver` (Driver): Экземпляр класса `Driver` для взаимодействия с веб-драйвером.
    **Возвращает**:
        - `None`

- `local_saved_image(self, value: Any = None)`:
    **Описание**:  Загружает и сохраняет изображение с веб-страницы товара локально.  Сохраняет путь к изображению в поле `ProductFields.local_saved_image`.
    **Параметры**:
        - `value` (Any, опционально): Дополнительные данные, которые можно передать в функцию при вызове. Если `value` передан, его значение записывается в `ProductFields.local_saved_image`.
    **Возвращает**:
        - `bool`: `True`, если изображение сохранено успешно; `False` иначе.
