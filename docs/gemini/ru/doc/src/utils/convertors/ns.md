# Модуль hypotez/src/utils/convertors/ns.py

## Обзор

Модуль `ns.py` предоставляет функции для преобразования объектов `SimpleNamespace` в различные форматы: словари, JSON, CSV, XML и XLS.  Он обрабатывает вложенные структуры данных, обеспечивая конвертацию объектов `SimpleNamespace`, словарей и списков.

## Функции

### `ns2dict`

**Описание**: Функция преобразует объект `SimpleNamespace` в словарь. Рекурсивно обрабатывает вложенные структуры, включая другие объекты `SimpleNamespace`, словари и списки.

**Параметры**:

- `ns_obj` (SimpleNamespace): Объект `SimpleNamespace`, который необходимо преобразовать.

**Возвращает**:

- `Dict[str, Any]`: Преобразованный словарь с вложенными структурами.

### `ns2json`

**Описание**: Пока не реализована.


**Параметры**:

- `ns_obj` (SimpleNamespace): Объект SimpleNamespace, который нужно преобразовать.


**Возвращает**:

- `str`: Строка в формате JSON.


**Вызывает исключения**:

- `ValueError`: В случае ошибок при преобразовании.


### `ns2csv`

**Описание**: Функция преобразует объект `SimpleNamespace` в CSV-формат и сохраняет его в указанный файл.

**Параметры**:

- `ns_obj` (SimpleNamespace): Объект `SimpleNamespace`, который нужно преобразовать.
- `csv_file_path` (str | Path): Путь к файлу для сохранения CSV-данных.

**Возвращает**:

- `bool`: `True`, если преобразование и сохранение выполнены успешно, `False` иначе.

**Вызывает исключения**:

- `Exception`: Общие исключения, возникающие при работе с файлами или данными.


### `ns2xml`

**Описание**: Функция преобразует объект `SimpleNamespace` в XML-формат.

**Параметры**:

- `ns_obj` (SimpleNamespace): Объект `SimpleNamespace`, который нужно преобразовать.
- `root_tag` (str, optional): Имя тега корневого элемента XML. По умолчанию "root".

**Возвращает**:

- `str`: Строка, содержащая XML-представление объекта.

**Вызывает исключения**:

- `Exception`: Общие исключения, возникающие при работе с данными или XML-парсером.


### `ns2xls`

**Описание**: Функция преобразует объект `SimpleNamespace` в формат XLS (Excel) и сохраняет его в указанный файл.

**Параметры**:

- `ns_obj` (SimpleNamespace): Объект `SimpleNamespace`, который необходимо преобразовать.
- `xls_file_path` (str | Path): Путь к файлу для сохранения XLS-файла.

**Возвращает**:

- `bool`: `True`, если преобразование и сохранение выполнены успешно, `False` иначе.


**Вызывает исключения**:

- `Exception`: Общие исключения, возникающие при работе с файлами или данными.
