# Модуль myai

## Обзор

Модуль `myai` предоставляет базовые настройки для проекта.

## Постоянные значения

### `MODE`

**Описание**: Строковая переменная, определяющая режим работы (например, 'dev', 'prod').  В текущей реализации значение установлено в 'dev'.
