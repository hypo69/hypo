# Задача No 194
## Комбинации сетей с соединителями
## Условия

Рассмотрим сети, состоящие из узлов, соединенных между собой ребрами. Для этой задачи мы рассматриваем сети, состоящие из двух типов узлов: A и B.

Для данной сети существует три основных правила:

1. В сети есть хотя бы один узел A.
2. В сети есть хотя бы один узел B.
3. Каждый узел соединен хотя бы с одним другим узлом.

Например, узел A соединен с узлом B. Эта простейшая сеть представлена схемой A-B.

Теперь мы вводим два соединителя для создания более сложных сетей:

- Соединитель типа 1 связывает два узла одного типа.
- Соединитель типа 2 связывает два узла разных типов.

Соединители нумеруются, чтобы отличать их друг от друга. Например, A-1-A - это сеть, полученная путем соединения двух узлов типа A через соединитель 1. А-1-B-2-A - это сеть, полученная путем соединения узлов A и B через соединитель 1, а затем соединения полученного узла (A-1-B) с узлом A через соединитель 2.

Определим C(a,b,c) как количество различных сетей, которые можно получить, используя 'a' узлов A, 'b' узлов B и 'c' соединителей.

Например:

- C(2,2,3) = 10
- C(2,2,4) = 119
- C(3,2,5) = 1546
- C(3,3,5) = 1089
- C(3,2,6) = 17199
- C(4,2,6) = 21512
- C(4,3,6) = 15888

Пусть F(n) = ∑ C(a,b,n) для 1 ≤ a,b ≤ n.

Например:

- F(3) = 10 + 1546 + 1089 = 2645
- F(7) = 2595750

Найдите F(10000) mod 1000000087.
