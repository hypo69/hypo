# Модуль send_data.js

## Обзор

Данный скрипт JavaScript отвечает за сбор данных с загруженной веб-страницы и отправку их на сервер.  Он использует `fetch` API для отправки POST запроса.

## Функции

### `onPageLoad`

**Описание**: Обработчик события загрузки страницы. Сбор заголовка, URL и содержимого страницы, формирование объекта данных и отправка запроса на сервер.

**Обрабатывает события**:
- `load` (в контексте `window`)

**Действия**:
1. Считывает `title`, `url` и `body` текущей страницы.
2. Формирует объект `data` с собранной информацией.
3. Отправляет POST запрос на `http://127.0.0.1/hypotez.online/api/` с данными `data` в формате JSON.
4. Обрабатывает ответ от сервера, выводит результат в консоль (`console.log`) или логгирует ошибки (`console.error`).

**Возвращает**:
- Ничего не возвращает.

**Возможные ошибки**:
- `Network response was not ok`: В случае, если сервер вернул ошибку (код ответа не 2xx). Обрабатывается блоком `.catch`.
- Прочие ошибки при выполнении `fetch` или обработке ответа. Обрабатывается блоком `.catch`.
