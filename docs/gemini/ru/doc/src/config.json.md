# Конфигурационный файл `config.json`

## Обзор

Данный файл `config.json` содержит конфигурационные параметры для проекта `hypotez`. Он включает информацию об авторе, названии проекта, режимах работы, настройках git, путях к различным директориям, формате временных меток, версии и авторских правах.

## Описание полей

### `author`

**Описание**: Имя автора проекта.

**Значение**: `"Hypo69"`

### `project_name`

**Описание**: Название проекта.

**Значение**: `"hypotez"`

### `mode`

**Описание**: Текущий режим работы приложения.

**Значение**: `"dev"`

### `avaible_modes`

**Описание**: Список доступных режимов работы.

**Значение**: `[ "dev", "debug", "test", "prod" ]`

### `git`

**Описание**: Имя git репозитория.

**Значение**: `"hypo"`

### `git_user`

**Описание**: Имя пользователя git.

**Значение**: `"hypo69"`

### `path`

**Описание**: Объект, содержащий пути к различным директориям.

#### `external_storage`

**Описание**: Путь к директории внешнего хранилища.

**Значение**: `"E:\\\\hypo69\\\\data"`

#### `google_drive`

**Описание**: Путь к директории Google Drive.

**Значение**: `"H:\\\\My Drive\\\\hypotez\\\\data"`

#### `log`

**Описание**: Путь к директории для хранения лог-файлов.

**Значение**: `"E:\\\\hypo69\\\\log"`

#### `tmp`

**Описание**: Путь к директории для временных файлов.

**Значение**: `"E:\\\\hypo69\\\\tmp"`

### `timestamp_format`

**Описание**: Формат временной метки.

**Значение**: `"%y%m%d%H%M%S%f"`

### `release`

**Описание**: Версия релиза.

**Значение**: `"0.1"`

### `version`

**Описание**: Версия проекта.

**Значение**: `"0.11"`

### `copyright`

**Описание**: Авторские права.

**Значение**: `"2024, hypo69"`

### `cofee`

**Описание**: Призыв поддержать разработчика чашкой кофе.

**Значение**: `"Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69"`
