# Модуль `hypotez/src/suppliers/etzmaleh/header.py`

## Обзор

Данный модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json` и документации из файла `README.MD`. Модуль также инициализирует переменные, содержащие информацию о проекте (название, версия, описание, автор, права и т.д.).


## Функции

### `set_project_root`

**Описание**:  Находит корневую директорию проекта, начиная с текущей директории файла и двигаясь вверх по дереву директорий.  Поиск останавливается на первой директории, содержащей указанные маркерные файлы.

**Параметры**:

- `marker_files` (tuple): Кортеж из имен файлов или каталогов, используемых для определения корневой директории проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена; в противном случае, путь к директории текущего файла.

**Вызывает исключения**:

- Нет.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root()`.

**Тип**: `Path`


### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженные из файла `settings.json`.

**Тип**: `dict` | `None`


### `doc_str`

**Описание**: Строка, содержащая текст документации, загруженный из файла `README.MD`.

**Тип**: `str` | `None`


### `__project_name__`

**Описание**: Название проекта. Получается из словаря `settings`. По умолчанию 'hypotez'.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта. Получается из словаря `settings`. По умолчанию пустая строка.

**Тип**: `str`


### `__doc__`

**Описание**: Описание проекта. Получается из строки `doc_str`. По умолчанию пустая строка.

**Тип**: `str`


### `__details__`

**Описание**: Дополнительные детали о проекте. По умолчанию пустая строка.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта. Получается из словаря `settings`. По умолчанию пустая строка.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права проекта. Получается из словаря `settings`. По умолчанию пустая строка.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка на страницу для поддержки разработчика (пожертвования кофе). Получается из словаря `settings`. По умолчанию строка со ссылкой.

**Тип**: `str`
