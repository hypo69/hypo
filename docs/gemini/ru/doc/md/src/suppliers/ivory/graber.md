# Модуль hypotez/src/suppliers/ivory/graber.py

## Обзор

Модуль `hypotez/src/suppliers/ivory/graber.py` содержит класс `Graber`, предназначенный для сбора данных о товарах с сайта `ivory.co.il`. Класс наследуется от базового класса `Graber` и предоставляет методы для извлечения различных полей товара, включая обработку специфичных для этого поставщика данных.  В классе реализованы асинхронные методы для извлечения данных, обеспечивая параллельную работу с веб-драйвером.  Также присутствует декоратор для автоматического закрытия всплывающих окон.


## Классы

### `Graber`

**Описание**: Класс `Graber` расширяет базовый класс `Grbr` для специфической обработки данных с сайта `ivory.co.il`.  Реализует методы для извлечения различных полей товара.

**Методы**:

- `__init__(self, driver: Driver)`:
    **Описание**: Инициализирует объект `Graber`.
    **Параметры**:
        - `driver` (Driver): Экземпляр веб-драйвера.
    **Возвращает**:
        - `None`


- `grab_page(self, driver: Driver) -> ProductFields`
    **Описание**: Асинхронный метод для сбора данных о товаре.
    **Параметры**:
        - `driver` (Driver): Экземпляр веб-драйвера.
    **Возвращает**:
        - `ProductFields`: Объект, содержащий собранные данные о товаре.
    **Обрабатываемые исключения**:
        - `ExecuteLocatorException`: Возникает при ошибке выполнения локатора.
