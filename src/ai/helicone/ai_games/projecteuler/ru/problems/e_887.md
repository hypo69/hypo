# Задача No 887
## Числа с повторяющимися цифрами

## Условия

Натуральное число называется числом с повторяющимися цифрами, если в его десятичном представлении есть по крайней мере две одинаковые цифры. Например, 1232 является числом с повторяющимися цифрами, а 1234 - нет.

Пусть $R(n)$ будет количеством $n$-значных чисел с повторяющимися цифрами. Например, $R(2) = 9$ и $R(3) = 252$.

Пусть $S(n)$ будет суммой всех $n$-значных чисел с повторяющимися цифрами. Например, $S(2) = 4860$ и $S(3) = 1419330$.

Пусть $T(n)$ будет суммой всех $n$-значных чисел, которые не являются числами с повторяющимися цифрами. Например, $T(2) = 4455$ и $T(3) = 181425$.

Пусть $U(n) = S(n) - T(n)$. Например, $U(2) = 405$ и $U(3) = 1237905$.

Найдите $\sum_{n=2}^{40} U(n)$.
