## Обзор

Этот файл `cdata_categories_monitors_apple.json` содержит конфигурационные данные для мониторов бренда Apple. Данные организованы в виде JSON-объекта, где ключом является `scenarios`, а значением — словарь, в котором ключи являются названиями сценариев (моделей мониторов Apple), а значениями —  наборы параметров для каждого сценария.

## Структура JSON

JSON файл имеет следующую структуру:

```json
{
  "scenarios": {
    "SCENARIO_NAME_1": {
      "brand": "BRAND_NAME",
      "url": "URL_STRING",
      "checkbox": BOOLEAN,
      "active": BOOLEAN,
      "condition":"CONDITION_STRING",
      "presta_categories": "CATEGORY_IDS_STRING"
    },
    "SCENARIO_NAME_2": {
       // ... Другие сценарии
    }
  }
}
```

## Описание полей

### `scenarios`
- **Описание**: Основной раздел, содержащий все сценарии для мониторов Apple.
- **Тип**: `object` (словарь).
- **Ключи**: Названия сценариев (например, `APPLE 18`, `APPLE 21.5`).
- **Значения**: Объекты (словари), представляющие параметры конкретного сценария.

### Каждый сценарий

Каждый сценарий (например, `"APPLE 18"`) представляет собой словарь со следующими полями:

- #### `brand`
  - **Описание**: Бренд монитора.
  - **Тип**: `string`.
  - **Пример**: `"APPLE"`

- #### `url`
  - **Описание**: URL, связанный со сценарием. Может быть ссылкой на страницу товара или просто строкой-идентификатором.
  - **Тип**: `string`.
  - **Пример**: `"---------------------APPLE 18----------------------"` или `"https://reseller.c-data.co.il/%D7%9E%D7%A1%D7%9B%D7%99%D7%9D#/specFilters=215m!#-!6360&manFilters=3"`

- #### `checkbox`
  - **Описание**: Флаг, определяющий состояние чекбокса.
  - **Тип**: `boolean`.
  - **Пример**: `false`

- #### `active`
  - **Описание**: Флаг, определяющий, активен ли сценарий.
  - **Тип**: `boolean`.
  - **Пример**: `true`
  
- #### `condition`
  - **Описание**:  Состояние товара, например, "new".
  - **Тип**: `string`.
  - **Пример**: `"new"`
  
- #### `presta_categories`
  - **Описание**:  Категории PrestaShop, к которым относится товар, перечисленные через запятую.
  - **Тип**: `string`.
  - **Пример**: `"127,241"`
