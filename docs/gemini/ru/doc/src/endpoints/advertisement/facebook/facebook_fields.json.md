# Документация для `facebook_fields.json`

## Обзор

Файл `facebook_fields.json` содержит структуру JSON, описывающую поля, используемые для создания объявлений в Facebook. Он включает поля для обычных постов и событий, а также настройки для тайм-аута и типа события.

## Структура JSON

### Поля для поста

В разделе `"post"` определены поля, необходимые для создания обычного поста в Facebook.

-   `"title"` (string): Заголовок поста.
-   `"description"` (string): Описание поста.
-   `"language"` (string): Язык поста.
-   `"currency"` (string): Валюта, используемая в посте.
-   `"price"` (string): Цена, указанная в посте.
-   `"category"` (string): Категория поста.
-   `"products"` (array): Массив продуктов, связанных с постом.

### Настройки тайм-аута

-   `"timeout"` (number): Время ожидания в секундах.
-   `"timeout_for_event"` (string): Тип события для ожидания тайм-аута (например, `"presence_of_element_located"`).

### Поля для события

В разделе `"event"` определены поля, необходимые для создания события в Facebook.

-   `"title"` (string): Заголовок события.
-   `"description"` (string): Описание события.
-   `"language"` (string): Язык события.
-   `"currency"` (string): Валюта, используемая в событии.
-   `"price"` (string): Цена события.
-   `"start_date"` (string): Дата начала события.
-   `"start_time"` (string): Время начала события.
-   `"end_date"` (string): Дата окончания события.
