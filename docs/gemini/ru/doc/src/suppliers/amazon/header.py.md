# Модуль `header`

## Обзор

Модуль `header` предназначен для определения и настройки корневой директории проекта, а также для загрузки основных настроек и документации из файлов `settings.json` и `README.MD` соответственно. Он также устанавливает различные глобальные переменные, такие как имя проекта, версия, авторские права и т.д., которые используются в других частях проекта `hypotez`.

## Подробней

Модуль `header` выполняет следующие задачи:

1.  Определяет корневую директорию проекта, начиная с директории текущего файла и двигаясь вверх по иерархии, пока не найдет один из маркерных файлов ('.git' или '\_\_root\_\_').
2.  Загружает настройки из файла `settings.json` в корневой директории проекта.
3.  Читает содержимое файла `README.MD` для использования в качестве документации проекта.
4.  Устанавливает глобальные переменные, такие как имя проекта, версия, автор, копирайт и сообщение для поддержки разработчиков, используя данные из `settings.json` или значения по умолчанию.

Эти переменные используются для настройки и предоставления информации о проекте в других модулях.

## Функции

### `set_project_root`

```python
def set_project_root(marker_files: tuple = ('__root__', '.git')) -> Path:
    """
    Args:
        marker_files (tuple): Filenames or directory names to identify the project root.
    
    Returns:
        Path: Path to the root directory if found, otherwise the directory where the script is located.
    """
```

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла, и останавливается на первой директории, содержащей любой из маркерных файлов.

**Параметры**:

*   `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для идентификации корневой директории проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:

*   `Path`: Путь к корневой директории, если она найдена, иначе директория, где расположен скрипт.

**Примеры**:

```python
from pathlib import Path
root_path = set_project_root()
print(f"Root path: {root_path}")
