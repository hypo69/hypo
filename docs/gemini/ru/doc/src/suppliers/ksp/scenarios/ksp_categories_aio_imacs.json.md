# Документация для `ksp_categories_aio_imacs.json`

## Обзор

Данный JSON-файл содержит конфигурации сценариев для сбора данных о продуктах Apple iMac различных моделей с сайта KSP. Каждый сценарий определяет конкретную модель iMac, ее бренд, URL, статус активности и сопоставление с категориями PrestaShop.

## Структура файла

Файл представляет собой JSON-объект с одним ключом `scenarios`, значением которого является объект, содержащий наборы сценариев. Каждый сценарий описан как ключ-значение, где ключ является именем сценария, а значение - объект с детальными параметрами.

## Сценарии

### `3425 IMac 24 inch`

**Описание**: Сценарий для сбора данных о 24-дюймовых iMac.

**Параметры**:
- `brand` (str): Бренд товара, в данном случае "APPLE".
- `url` (str): URL-адрес страницы с товаром на сайте KSP.
- `checkbox` (bool): Флаг, указывающий на наличие чекбокса (в данном случае `false`).
- `active` (bool): Флаг, указывающий, активен ли сценарий (в данном случае `true`).
- `condition` (str): Состояние товара (в данном случае `new`).
- `presta_categories` (dict): Словарь соответствия категорий для PrestaShop.

**Значение `presta_categories`**:
- `template` (dict): Словарь, где ключ `"apple"` сопоставлен с категорией `"iMac 24"`.
