# Модуль `defaults.py`

## Обзор

Модуль содержит определения заголовков по умолчанию для HTTP-запросов, используемых в проекте `hypotez`. В частности, он определяет заголовки для обычных запросов (`DEFAULT_HEADERS`) и запросов, выполняемых через WebView (`WEBVIEW_HAEDERS`). Также модуль проверяет наличие библиотеки `brotli` и соответствующим образом настраивает заголовки.

## Подробней

Этот модуль важен для настройки HTTP-запросов, отправляемых проектом. Заголовки, определённые здесь, влияют на то, как серверы обрабатывают запросы, например, какой формат кодирования данных поддерживается. Использование `j_loads` или `j_loads_ns` здесь не требуется, так как модуль не читает JSON или конфигурационные файлы.

## Переменные

### `has_brotli`

Определяет, доступна ли библиотека `brotli`.

**Описание**: Булева переменная, показывающая наличие библиотеки `brotli`.

**Назначение**: Указывает, может ли быть использовано сжатие Brotli в HTTP-запросах.

**Примеры**:

```python
if has_brotli:
    print("Brotli доступен")
else:
    print("Brotli не доступен")
```

### `DEFAULT_HEADERS`

Словарь, содержащий заголовки HTTP-запроса по умолчанию.

**Описание**: Словарь, определяющий стандартные HTTP-заголовки для обычных запросов.

**Ключи и значения**:

-   `"accept"` (str): Типы контента, которые клиент может обрабатывать (`"*/*"` означает любой тип).
-   `"accept-encoding"` (str): Схемы кодирования, поддерживаемые клиентом (`"gzip, deflate"` плюс `"br"`, если доступен `brotli`).
-   `"accept-language"` (str): Предпочитаемые языки.
-   `"referer"` (str): URL страницы, с которой был сделан запрос (пустая строка по умолчанию).
-   `"sec-ch-ua"` (str): Информация о браузере.
-   `"sec-ch-ua-mobile"` (str): Указывает, является ли устройство мобильным.
-   `"sec-ch-ua-platform"` (str): Платформа устройства.
-   `"sec-fetch-dest"` (str): Назначение запроса (`"empty"`).
-   `"sec-fetch-mode"` (str): Режим запроса (`"cors"`).
-   `"sec-fetch-site"` (str): Сайт, с которого был сделан запрос (`"same-origin"`).
-   `"user-agent"` (str): Строка user-agent браузера.

**Примеры**:

```python
print(DEFAULT_HEADERS["user-agent"])
```

### `WEBVIEW_HAEDERS`

Словарь, содержащий заголовки HTTP-запроса по умолчанию для WebView.

**Описание**: Словарь, определяющий стандартные HTTP-заголовки для запросов, выполняемых через WebView.

**Ключи и значения**:

-   `"Accept"` (str): Типы контента, которые клиент может обрабатывать (`"*/*"` означает любой тип).
-   `"Accept-Encoding"` (str): Схемы кодирования, поддерживаемые клиентом (`"gzip, deflate, br"`).
-   `"Accept-Language"` (str): Предпочитаемые языки.
-   `"Referer"` (str): URL страницы, с которой был сделан запрос (пустая строка по умолчанию).
-   `"Sec-Fetch-Dest"` (str): Назначение запроса (`"empty"`).
-   `"Sec-Fetch-Mode"` (str): Режим запроса (`"cors"`).
-   `"Sec-Fetch-Site"` (str): Сайт, с которого был сделан запрос (`"same-origin"`).
-   `"User-Agent"` (str): Строка user-agent браузера.

**Примеры**:

```python
print(WEBVIEW_HAEDERS["User-Agent"])
```
