# Модуль `gworksheets.py`

## Обзор

Модуль `gworksheets.py` предоставляет класс `GWorksheet`, который расширяет возможности работы с листами в Google Sheets. Он включает в себя функциональность для создания, получения, настройки направления, добавления заголовков и категорий для листов.

## Классы

### `GWorksheet`

**Описание**: Класс `GWorksheet` предназначен для работы с листами Google Sheets. Он наследует функциональность от класса `Worksheet` и добавляет собственные методы для управления листами.

**Наследует**:

-   `Worksheet`: базовый класс для работы с листами

**Атрибуты**:

-   `sh`: Ссылка на объект `Spreadsheet`.
-   `ws`: Ссылка на объект `Worksheet`.
-  `render`: Ссылка на объект `GSRender`, используется для отрисовки элементов.

## Функции

### `__init__`

**Описание**: Конструктор класса `GWorksheet`. Инициализирует объект класса, получая ссылку на spreadsheet и название листа, а также другие параметры для настройки листа.

**Параметры**:
- `sh`: Spreadsheet, с которым будет работать данный класс.
- `ws_title` (str, optional): Название листа. По умолчанию 'new'.
- `rows` (int, optional): Кол-во строк на листе. По умолчанию `None`.
- `cols` (int, optional): Кол-во колонок на листе. По умолчанию `None`.
- `direcion` (str, optional): Направление текста на листе ('rtl', 'ltr'). По умолчанию 'rtl'.
- `wipe_if_exist` (bool, optional): Флаг, указывающий нужно ли очистить лист, если он существует. По умолчанию `True`.
- `*args`: Произвольные аргументы.
- `**kwards`: Произвольные именованные аргументы.

**Возвращает**:
    - `None`

### `get`

**Описание**: Получает или создает лист в текущем spreadsheet.

**Параметры**:
- `sh`: Spreadsheet, в котором будет получен или создан лист.
- `ws_title` (str, optional): Название листа. По умолчанию 'new'.
- `rows` (int, optional): Кол-во строк, если лист создаётся. По умолчанию 100.
- `cols` (int, optional): Кол-во колонок, если лист создаётся. По умолчанию 100.
- `direction` (str, optional): Направление текста на листе ('rtl', 'ltr'). По умолчанию 'rtl'.
- `wipe_if_exist` (bool, optional): Флаг, указывающий нужно ли очистить лист, если он существует. По умолчанию `True`.

### `header`

**Описание**: Добавляет заголовок к листу.

**Параметры**:
- `world_title` (str): Заголовок.
- `range` (str, optional): Диапазон ячеек для заголовка. По умолчанию 'A1:Z1'.
- `merge_type` (str, optional): Тип объединения ячеек для заголовка. По умолчанию 'MERGE_ALL'.

**Возвращает**:
    - `None`

### `category`

**Описание**: Добавляет заголовок категории на лист.

**Параметры**:
-   `ws_category_title`: Заголовок категории.

**Возвращает**:
    - `None`

### `direction`

**Описание**: Устанавливает направление листа.

**Параметры**:
- `direction` (str, optional): Направление текста на листе ('rtl', 'ltr'). По умолчанию 'rtl'.

**Возвращает**:
    - `None`
