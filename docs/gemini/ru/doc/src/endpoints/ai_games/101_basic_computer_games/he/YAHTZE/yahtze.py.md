# YAHTZE - משחק קוביות

## Обзор

Этот модуль реализует классическую игру в кости "Yahtzee". Игроки бросают 5 кубиков и пытаются набрать очки, комбинируя результаты бросков в различных категориях. Цель игры - набрать максимальное количество очков, заполняя все 13 категорий.

## Функции

### `roll_die`

**Описание**:
Генерирует случайное число от 1 до 6, представляющее собой бросок кубика.

**Возвращает**:
- `int`: Случайное целое число от 1 до 6.

### `roll_dice`

**Описание**:
Генерирует 5 случайных чисел от 1 до 6, представляющих собой бросок 5 кубиков.

**Возвращает**:
- `List[int]`: Список из 5 целых чисел, каждое из которых представляет результат броска одного кубика.

### `print_dice`

**Описание**:
Выводит результаты броска кубиков на экран.

**Параметры**:
- `dice` (List[int]): Список целых чисел, представляющих результаты броска кубиков.

### `select_dice_to_reroll`

**Описание**:
Позволяет игроку выбрать, какие кубики перебросить.

**Параметры**:
- `dice` (List[int]): Список целых чисел, представляющих результаты броска кубиков.

**Возвращает**:
- `List[int]`: Список индексов кубиков, которые нужно перебросить.

**Вызывает исключения**:
- `ValueError`: Возникает, если введен некорректный формат данных.

### `calculate_score`

**Описание**:
Вычисляет очки для выбранной категории.

**Параметры**:
- `dice` (List[int]): Список целых чисел, представляющих результаты броска кубиков.
- `category` (int): Номер выбранной категории.

**Возвращает**:
- `int`: Количество очков, полученных для выбранной категории.

### `display_available_categories`

**Описание**:
Выводит список доступных категорий для игрока.

**Параметры**:
- `categories_available` (List[bool]): Список, указывающий, какие категории доступны.

### `get_category_name`

**Описание**:
Возвращает имя категории по ее номеру.

**Параметры**:
- `category_number` (int): Номер категории.

**Возвращает**:
- `str`: Имя категории.

### `get_category_choice`

**Описание**:
Запрашивает у игрока выбор категории.

**Параметры**:
- `categories_available` (List[bool]): Список, указывающий, какие категории доступны.

**Возвращает**:
- `int`: Номер выбранной категории.

**Вызывает исключения**:
- `ValueError`: Возникает, если введен некорректный формат данных.

### `play_yahtze`

**Описание**:
Управляет игровым процессом "Yahtzee".
