# Модуль `login.py`

## Обзор

Модуль `login.py` предназначен для реализации функций авторизации поставщика Kualastyle. Включает функции для входа в систему и закрытия всплывающих окон.

## Оглавление

- [Функции](#Функции)
    - [`login`](#login)
    - [`close_pop_up`](#close_pop_up)

## Функции

### `login`

**Описание**: Функция для входа в систему поставщика.

**Параметры**:
- `s` (Supplier): Объект поставщика.

**Возвращает**:
- `bool`: `True`, если вход выполнен успешно, иначе `False`.

### `close_pop_up`

**Описание**: Функция для закрытия всплывающего окна на сайте поставщика.

**Параметры**:
- `s` (Supplier): Объект поставщика.

**Возвращает**:
- `bool`: `True`, если окно закрыто успешно, иначе `False`.

**Вызывает исключения**:
- `Exception`: Возникает при ошибке закрытия всплывающего окна. В случае ошибки, сообщение будет записано в лог с уровнем `warning`.
