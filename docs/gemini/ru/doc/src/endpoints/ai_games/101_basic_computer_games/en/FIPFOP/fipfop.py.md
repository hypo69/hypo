# FIPFOP

## Обзор

Данный модуль представляет собой реализацию игры FIPFOP, в которой игрок поочередно вводит числа, пока их общая сумма не достигнет 100. После достижения 100 игра завершается, и выводится количество ходов, потребовавшихся для достижения цели.

## Функции

В данном модуле нет функций, только основной игровой цикл.

## Переменные

### `totalSum`
**Описание**: Сумма введенных пользователем чисел. Инициализируется 0.

### `numberOfTurns`
**Описание**: Количество ходов, сделанных пользователем. Инициализируется 0.

### `userNumber`
**Описание**: Число, введенное пользователем на текущем ходу.

## Основной игровой цикл

### Описание
Основной игровой цикл продолжается до тех пор, пока `totalSum` не станет больше или равна 100.

### Логика цикла
1. **Ввод числа:** Пользователю предлагается ввести целое число. Если ввод не является целым числом, выводится сообщение об ошибке, и цикл продолжается.
2. **Увеличение счетчика ходов:** Переменная `numberOfTurns` увеличивается на 1.
3. **Добавление числа к сумме:** Введенное число добавляется к общей сумме `totalSum`.
4. **Проверка условия:** Цикл завершается, когда `totalSum` становится больше или равна 100.

### Вывод результата
После завершения цикла выводится сообщение, указывающее, сколько ходов было сделано для достижения суммы 100.
