# Модуль DeepInfraChat

## Обзор

Модуль `DeepInfraChat.py` предназначен для интеграции с платформой DeepInfra Chat через API OpenAI. Он определяет класс `DeepInfraChat`, который наследуется от класса `OpenaiTemplate` и предоставляет конфигурацию для взаимодействия с различными моделями, размещенными на DeepInfra.

## Подробней

Этот модуль является частью системы g4f (GPT4Free), целью которой является предоставление бесплатного доступа к различным моделям искусственного интеллекта. Он определяет параметры подключения и список поддерживаемых моделей для DeepInfra Chat, упрощая взаимодействие с этими моделями через стандартизированный интерфейс.

## Классы

### `DeepInfraChat(OpenaiTemplate)`

**Описание**:
Класс `DeepInfraChat` наследуется от `OpenaiTemplate` и предназначен для настройки и использования моделей чата, размещенных на платформе DeepInfra. Он содержит атрибуты, определяющие URL, базовый API, состояние работоспособности, модели по умолчанию, модели для работы с изображениями и список всех поддерживаемых моделей.

**Принцип работы**:

1.  **Наследование от `OpenaiTemplate`**: Класс наследует основные методы и атрибуты для взаимодействия с API OpenAI, что позволяет использовать общую логику для отправки запросов и обработки ответов.
2.  **Определение атрибутов класса**: Устанавливаются специфические для DeepInfra Chat значения, такие как URL, API base, список моделей и их алиасы.
3.  **Предоставление списка моделей**: Атрибуты `models` и `vision_models` содержат списки идентификаторов моделей, доступных на платформе DeepInfra, включая как основные модели, так и модели для обработки изображений.
4.  **Создание алиасов моделей**: Атрибут `model_aliases` позволяет использовать короткие и понятные имена для обращения к моделям, упрощая их выбор и конфигурацию.

**Атрибуты**:

*   `url` (str): URL для доступа к DeepInfra Chat. Значение: `"https://deepinfra.com/chat"`.
*   `api_base` (str): Базовый URL для API OpenAI на DeepInfra. Значение: `"https://api.deepinfra.com/v1/openai"`.
*   `working` (bool): Указывает, находится ли провайдер в рабочем состоянии. Значение: `True`.
*   `default_model` (str): Модель, используемая по умолчанию. Значение: `'deepseek-ai/DeepSeek-V3'`.
*   `default_vision_model` (str): Модель для работы с изображениями, используемая по умолчанию. Значение: `'openbmb/MiniCPM-Llama3-V-2_5'`.
*   `vision_models` (List[str]): Список поддерживаемых моделей для работы с изображениями. Значения: `[default_vision_model, 'meta-llama/Llama-3.2-90B-Vision-Instruct']`.
*   `models` (List[str]): Список поддерживаемых моделей.
*   `model_aliases` (Dict[str, str]): Словарь, содержащий псевдонимы моделей для удобства использования.

**Примеры**:

```python
from g4f.Provider.DeepInfraChat import DeepInfraChat

# Создание экземпляра класса DeepInfraChat
deepinfra_chat = DeepInfraChat()

# Доступ к атрибутам класса
url = deepinfra_chat.url
api_base = deepinfra_chat.api_base
models = deepinfra_chat.models
model_aliases = deepinfra_chat.model_aliases

print(f"URL: {url}")
print(f"API Base: {api_base}")
print(f"Доступные модели: {models}")
print(f"Алиасы моделей: {model_aliases}")
```
