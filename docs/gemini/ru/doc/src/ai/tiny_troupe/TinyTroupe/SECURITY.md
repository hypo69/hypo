# SECURITY.md

## Обзор

Данный файл содержит политику безопасности Microsoft относительно сообщения о найденных уязвимостях в репозиториях GitHub.  Файл описывает, как сообщать о потенциальных уязвимостях, предоставляя необходимую информацию для эффективного анализа.

## Сообщения об уязвимостях

**Не следует сообщать об уязвимостях через общедоступные проблемы GitHub.**

Вместо этого, пожалуйста, обратитесь в Microsoft Security Response Center (MSRC) по адресу [https://msrc.microsoft.com/create-report](https://aka.ms/security.md/msrc/create-report).

Для отправки сообщения без входа в систему отправьте электронное письмо на адрес [secure@microsoft.com](mailto:secure@microsoft.com).  Если это возможно, зашифруйте свое сообщение с помощью нашего ключа PGP; пожалуйста, загрузите его со страницы [страницы ключа PGP Microsoft Security Response Center](https://aka.ms/security.md/msrc/pgp).


Вы должны получить ответ в течение 24 часов. Если по какой-либо причине вы этого не сделаете, пожалуйста, свяжитесь с нами повторно по электронной почте, чтобы убедиться, что мы получили ваше первоначальное сообщение. Дополнительную информацию можно найти на [microsoft.com/msrc](https://www.microsoft.com/msrc).


Пожалуйста, предоставьте следующую информацию, чтобы помочь нам лучше понять характер и масштаб потенциальной проблемы:

* Тип проблемы (например, переполнение буфера, SQL-инъекция, межсайтовая атака с подделкой запроса, и т.д.)
* Полные пути исходных файлов, связанных с проявлением проблемы
* Местоположение затронутого исходного кода (тег/ветвь/коммит или прямой URL)
* Любая специальная конфигурация, необходимая для воспроизведения проблемы
* Инструкции по пошаговому воспроизведению проблемы
* Прототип или код эксплоита (если это возможно)
* Воздействие проблемы, включая то, как злоумышленник может использовать проблему

Эта информация поможет нам быстрее обработать ваш отчет.

Если вы сообщаете об ошибке в рамках программы поощрения обнаружения ошибок, более полные отчеты могут способствовать получению более высокой награды. Пожалуйста, посетите страницу нашей [программы поощрения обнаружения ошибок Microsoft](https://aka.ms/security.md/msrc/bounty), чтобы узнать больше о наших активных программах.
