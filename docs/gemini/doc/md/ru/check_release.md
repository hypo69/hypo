# Модуль check_release

## Обзор

Этот модуль предоставляет функцию `check_latest_release` для проверки последней версии релиза на GitHub.


## Функции

### `check_latest_release`

**Описание**: Проверяет последнюю версию релиза на GitHub по заданному владельцу и репозиторию.

**Параметры**:
- `owner` (str): Владелец репозитория на GitHub.
- `repo` (str): Название репозитория на GitHub.

**Возвращает**:
- `str`: Версия последнего релиза в формате строки, если доступна. Иначе возвращает `None`.


**Вызывает исключения**:
- Возможны ошибки при запросе к API GitHub, которые не обрабатываются.
