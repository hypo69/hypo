# Модуль `src.suppliers.hb`

## Обзор

Модуль `src.suppliers.hb` предоставляет функциональность для сбора данных с использованием класса `Graber`. Он содержит импорт класса `Graber` из подмодуля `graber`.

## Подробней

Этот модуль служит точкой входа для работы с функциональностью сбора данных от поставщика HB. Он импортирует класс `Graber`, который, вероятно, содержит логику для взаимодействия с API HB, обработки данных и т.д. Модуль предназначен для упрощения доступа к функциональности сбора данных HB через единую точку импорта.
