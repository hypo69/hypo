# src.suppliers.aliexpress.scenarios.login

## Обзор

Модуль `login.py` предназначен для автоматизации процесса входа в систему AliExpress с использованием Selenium WebDriver. Он содержит функцию `login`, которая принимает объект поставщика (`Supplier`) с настроенным веб-драйвером и локаторами элементов для выполнения действий по входу в систему.

## Подорбней

Этот модуль является частью системы автоматизации для AliExpress и предназначен для автоматического входа в учетную запись пользователя. Он использует веб-драйвер для взаимодействия с веб-страницей AliExpress, заполняет поля электронной почты и пароля, а также нажимает кнопку входа. Модуль также обрабатывает принятие файлов cookie и может быть расширен для настройки языка, валюты и страны доставки.
