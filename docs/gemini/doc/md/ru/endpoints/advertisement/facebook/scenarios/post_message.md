# Модуль `hypotez/src/endpoints/advertisement/facebook/scenarios/post_message.py`

## Обзор

Этот модуль предоставляет функции для публикации сообщений в Facebook Ads Manager.  Он содержит функции для ввода заголовка и описания, загрузки медиа-файлов, добавления подписей и публикации сообщения.

## Оглавление

- [Функции](#функции)
    - [`post_title`](#post_title)
    - [`upload_media`](#upload_media)
    - [`update_images_captions`](#update_images_captions)
    - [`publish`](#publish)
    - [`promote_post`](#promote_post)
    - [`post_message`](#post_message)

## Функции

### `post_title`

**Описание**: Отправляет заголовок и описание кампании в поле для сообщения.

**Параметры**:
- `d` (Driver): Экземпляр драйвера, используемый для взаимодействия с веб-страницей.
- `message` (SimpleNamespace | str): Объект SimpleNamespace или строка, содержащая заголовок и описание сообщения.  Если SimpleNamespace, ожидается атрибут `title` и `description`.


**Возвращает**:
- `bool`: `True`, если заголовок и описание были отправлены успешно, иначе `None`.


### `upload_media`

**Описание**: Загружает медиа-файлы в раздел изображений и обновляет подписи.

**Параметры**:
- `d` (Driver): Экземпляр драйвера.
- `media` (SimpleNamespace | List[SimpleNamespace] | str | list[str]): Медиа-файл(ы) для загрузки. Может быть строкой, путём к файлу, объектом `SimpleNamespace` с атрибутом `local_saved_image` (или `local_saved_video`) или списком таких объектов.
- `no_video` (bool, optional): Признак, что нужно игнорировать загрузку видео. По умолчанию `False`.
- `without_captions` (bool, optional): Признак, что не нужно добавлять подписи к изображениям. По умолчанию `False`.

**Возвращает**:
- `bool`: `True`, если медиа-файлы загружены успешно, иначе `None`.

**Вызывает исключения**:
- `Exception`: Если возникла ошибка во время загрузки медиа или обновления подписей.


### `update_images_captions`

**Описание**: Добавляет описания к загруженным медиа-файлам.

**Параметры**:
- `d` (Driver): Экземпляр драйвера.
- `media` (List[SimpleNamespace]): Список объектов SimpleNamespace, содержащих данные для обновления подписей.
- `textarea_list` (List[WebElement]): Список текстовых областей, куда добавляются подписи.

**Вызывает исключения**:
- `Exception`: Если произошла ошибка при обновлении подписей.


### `publish`

**Описание**: Публикует сообщение.

**Параметры**:
- `d` (Driver): Экземпляр драйвера.
- `attempts` (int, optional): Максимальное количество попыток публикации. По умолчанию 5.

**Возвращает**:
- `bool`: `True` если сообщение опубликовано, иначе `False` или `None`.


### `promote_post`

**Описание**: Управляет процессом продвижения поста с заголовком, описанием и медиа-файлами.

**Параметры**:
- `d` (Driver): Экземпляр драйвера.
- `category` (SimpleNamespace): Детали категории, используемые для заголовка и описания поста.
- `products` (List[SimpleNamespace]): Список продуктов, содержащих медиа и детали для публикации.


**Возвращает**:
- `bool`: `True` если пост успешно опубликован, иначе `None`.

### `post_message`

**Описание**: Управляет процессом публикации поста с заголовком, описанием и медиа-файлами.

**Параметры**:
- `d` (Driver): Экземпляр драйвера.
- `message` (SimpleNamespace): Объект SimpleNamespace, содержащий данные для публикации.
- `no_video` (bool, optional): Признак, что нужно игнорировать загрузку видео. По умолчанию `False`.
- `images` (Optional[str | list[str]], optional): Путь или список путей к изображениям. По умолчанию `None`.
- `without_captions` (bool, optional): Признак, что не нужно добавлять подписи к изображениям. По умолчанию `False`.


**Возвращает**:
- `bool`: `True` если пост успешно опубликован, иначе `None`.
