# Модуль `header.py`

## Обзор

Модуль `header.py` предназначен для настройки основных параметров проекта, таких как определение корневой директории, загрузка настроек из файла `settings.json`, получение документации из файла `README.MD` и определения метаданных проекта.

## Функции

### `set_project_root`

**Описание**:
Находит корневой каталог проекта, начиная с каталога текущего файла, просматривая вверх и останавливаясь на первом каталоге, содержащем любой из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple, optional): Имена файлов или каталогов для идентификации корня проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневому каталогу, если он найден, в противном случае каталог, в котором расположен скрипт.

## Глобальные переменные

### `MODE`

**Описание**:
Режим работы приложения. По умолчанию установлен в `'dev'`.

### `__root__`

**Описание**:
Путь к корневому каталогу проекта. Определяется функцией `set_project_root`.

### `settings`

**Описание**:
Словарь с настройками проекта, загруженный из файла `settings.json`. Если файл не найден или не может быть декодирован, значение будет `None`.

### `doc_str`

**Описание**:
Строка с содержимым файла `README.MD`, содержащая документацию проекта. Если файл не найден или не может быть прочитан, значение будет `None`.

### `__project_name__`

**Описание**:
Название проекта, загруженное из `settings.json` или `'hypotez'` по умолчанию.

### `__version__`

**Описание**:
Версия проекта, загруженная из `settings.json` или `''` по умолчанию.

### `__doc__`

**Описание**:
Документация проекта, загруженная из файла `README.MD` или `''` по умолчанию.

### `__details__`

**Описание**:
Детальная информация о проекте, по умолчанию `''`.

### `__author__`

**Описание**:
Автор проекта, загруженный из `settings.json` или `''` по умолчанию.

### `__copyright__`

**Описание**:
Информация об авторских правах, загруженная из `settings.json` или `''` по умолчанию.

### `__cofee__`

**Описание**:
Строка с предложением поддержать разработчика чашкой кофе, загруженная из `settings.json` или стандартная строка по умолчанию.
