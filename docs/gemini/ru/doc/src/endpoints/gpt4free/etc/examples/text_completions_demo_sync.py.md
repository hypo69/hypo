# Модуль демонстрации текстового автозавершения (синхронный режим)

## Обзор

Этот модуль демонстрирует синхронное использование библиотеки `g4f` для выполнения текстового автозавершения с использованием модели `gpt-4o`. Он отправляет запрос в модель и выводит полученный ответ.

## Подробнее

Модуль предназначен для демонстрации простого примера взаимодействия с API `g4f` в синхронном режиме. Он показывает, как отправить запрос к модели `gpt-4o` и получить ответ, используя клиент `Client` из библиотеки `g4f`. Этот пример может быть использован для быстрого прототипирования и тестирования функциональности текстового автозавершения.

## Функции

### `create`

```python
response = client.chat.completions.create(
    model="gpt-4o",
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "how does a court case get to the Supreme Court?"}
    ],
)
```

**Назначение**: Отправляет запрос на автозавершение текста к модели `gpt-4o`.

**Параметры**:
- `model` (str): Имя модели, используемой для автозавершения текста (в данном случае, "gpt-4o").
- `messages` (list): Список сообщений, содержащих контекст для автозавершения. Каждое сообщение представляет собой словарь с ключами `role` и `content`.

**Возвращает**:
- `response`: Объект ответа, содержащий результаты автозавершения.

**Как работает функция**:
1. Функция вызывает метод `create` объекта `client.chat.completions` для отправки запроса к модели `gpt-4o`.
2. В параметре `messages` передается список сообщений, содержащих системное сообщение (определение роли ассистента) и пользовательский запрос (вопрос о том, как дело попадает в Верховный суд).
3. Результат выполнения запроса сохраняется в переменной `response`.

```
Запрос на автозавершение текста
     │
     ▼
Отправка запроса к модели gpt-4o
     │
     ▼
Получение ответа от модели
     │
     ▼
Сохранение ответа в переменной response
```

**Примеры**:
```python
from g4f.client import Client

client = Client()

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "how does a court case get to the Supreme Court?"}
    ],
)

print(response.choices[0].message.content)
```

### `print`

```python
print(response.choices[0].message.content)
```

**Назначение**: Выводит содержимое сообщения из первого варианта выбора в ответе.

**Параметры**:
- `response.choices[0].message.content` (str): Содержимое сообщения из первого варианта выбора в ответе.

**Возвращает**:
- Ничего. Функция просто выводит текст в консоль.

**Как работает функция**:
1. Функция извлекает содержимое первого сообщения из первого варианта выбора в ответе, полученного от модели `gpt-4o`.
2. Выводит извлеченное содержимое в консоль.

```
Извлечение содержимого сообщения из ответа
     │
     ▼
Вывод содержимого в консоль
```

**Примеры**:
```python
from g4f.client import Client

client = Client()

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "how does a court case get to the Supreme Court?"}
    ],
)

print(response.choices[0].message.content)
