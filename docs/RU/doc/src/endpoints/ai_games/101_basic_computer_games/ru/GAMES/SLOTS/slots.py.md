# Модуль "Слоты"

## Обзор

Модуль `slots.py` реализует простую игру "Слоты", имитирующую игровой автомат с тремя барабанами. На каждом барабане случайным образом выпадает один из четырех символов: вишня (C), слива (P), колокольчик (B) или звездочка (*). Игрок выигрывает, если на всех трех барабанах выпадают одинаковые символы.

## Функции

### `get_symbol`

**Описание**:
Преобразует число в соответствующий символ для игрового автомата.

**Параметры**:
- `number` (int): Число от 1 до 4, представляющее барабан.

**Возвращает**:
- `str`: Символ, соответствующий числу ("C", "P", "B", "*"). Возвращает "?" в случае неизвестного номера.

```python
def get_symbol(number):
    """
    Преобразует число в соответствующий символ для игрового автомата.

    Args:
        number (int): Число от 1 до 4, представляющее барабан.

    Returns:
        str: Символ, соответствующий числу ("C", "P", "B", "*").
    """
    if number == 1:
        return "C"  # Вишня
    elif number == 2:
        return "P"  # Слива
    elif number == 3:
        return "B"  # Колокольчик
    elif number == 4:
        return "*"  # Звездочка
    else:
        return "?" # Неизвестно
```

- Нет параметров.
