# Автоматизированная отправка сообщений WhatsApp

## Обзор

Этот документ описывает создание скрипта на Python для автоматической отправки сообщений WhatsApp. Он предназначен для разработчиков с базовыми знаниями Python и опытом работы с командной строкой.

## Подробней

Данный скрипт позволяет автоматизировать отправку сообщений в WhatsApp, настраивая количество отправляемых сообщений и временной диапазон. Он использует библиотеки `pywhatkit`, `pyautogui`, `pynput`, `emoji`, `random` и `time` для взаимодействия с веб-версией WhatsApp, автоматизации действий и генерации случайных сообщений. Скрипт предназначен для автоматизации маркетинговых кампаний, уведомлений или других задач, требующих массовой отправки сообщений.

## Основной цикл отправки

**Описание**: Цикл, который отправляет сообщения в заданном количестве и временном диапазоне.

**Примеры**:

```python
message_count = 0
while message_count < num_messages:
    hour = random.randint(start_hour, end_hour)
    minutes = random.randint(0, 59)
    message = random.choice(messages)
    send_whatsapp_message(phone_number, message, hour, minutes)
    message_count += 1
```
