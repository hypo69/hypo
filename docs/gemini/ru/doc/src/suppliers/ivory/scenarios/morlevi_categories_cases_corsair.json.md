# Документация для `morlevi_categories_cases_corsair.json`

## Обзор

Данный JSON-файл содержит конфигурацию сценариев для парсинга категорий корпусов от производителя CORSAIR на сайте Morlevi. Каждый сценарий определяет параметры для фильтрации и сопоставления категорий товаров на сайте с категориями в PrestaShop.

## Структура JSON

Файл представляет собой JSON-объект, содержащий единственный ключ `scenarios`, значением которого является объект, где каждый ключ - это название сценария, а значение - объект с параметрами данного сценария.

## Описание сценариев

### `CORSAIR MID TOWER`

**Описание**:
Сценарий для парсинга корпусов CORSAIR типа MID TOWER.

**Параметры**:

- `brand` (str): Производитель товара - "CORSAIR".
- `template` (str): Пустой шаблон (не используется).
- `url` (str): URL-адрес страницы с категорией товаров на сайте Morlevi, которая будет использована для парсинга.
- `checkbox` (bool): Флаг, указывающий, что не нужно использовать чекбокс.
- `active` (bool): Флаг, указывающий, что сценарий активен.
- `condition` (str): Состояние товара - "new".
- `presta_categories` (dict): Объект, содержащий соответствие между категориями производителя и категориями PrestaShop.
    - `template` (dict): Объект с соответствиями категорий, где ключ `corsair` соответствует категории "MINI ITX" в PrestaShop.
