# Модуль `hypotez/src/ai/myai/header.py`

## Обзор

Этот модуль предоставляет функции для определения корневой директории проекта и загрузки настроек проекта. Он также инициализирует переменные, описывающие проект (название, версия, описание и т.д.).

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла. Поиск происходит вверх по директориям до тех пор, пока не будет найдена директория, содержащая указанные маркерные файлы (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или каталогов, которые используются для определения корневой директории проекта. По умолчанию включает `'pyproject.toml'`, `'requirements.txt'`, и `.git`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена, иначе путь к директории текущего файла.

**Вызывает исключения**:
Нет.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта. Инициализируется функцией `set_project_root`.

**Тип**: `Path`


### `settings`

**Описание**: Словарь с настройками проекта. Загружается из файла `src/settings.json`.

**Тип**: `dict` | `None`

**Инициализация**:
Инициализируется при успешной загрузке файла `settings.json`.


### `doc_str`

**Описание**: Строка с описанием проекта, загруженная из файла `README.MD`.

**Тип**: `str` | `None`

**Инициализация**:
Инициализируется при успешной загрузке файла `README.MD`.

### `__project_name__`

**Описание**: Название проекта, полученное из настроек. По умолчанию `'hypotez'`.

**Тип**: `str`

**Инициализация**:
Инициализируется значением из настроек или значением по умолчанию.

### `__version__`

**Описание**: Версия проекта, полученная из настроек. По умолчанию пустая строка.

**Тип**: `str`

**Инициализация**:
Инициализируется значением из настроек или значением по умолчанию.

### `__doc__`

**Описание**: Описание проекта, полученное из `doc_str`. По умолчанию пустая строка.

**Тип**: `str`

**Инициализация**:
Инициализируется значением из `doc_str` или значением по умолчанию.

### `__details__`

**Описание**: Дополнительные детали о проекте (по умолчанию пустая строка).

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта, полученный из настроек. По умолчанию пустая строка.

**Тип**: `str`

**Инициализация**:
Инициализируется значением из настроек или значением по умолчанию.


### `__copyright__`

**Описание**: Авторские права на проект, полученные из настроек. По умолчанию пустая строка.

**Тип**: `str`

**Инициализация**:
Инициализируется значением из настроек или значением по умолчанию.

### `__cofee__`

**Описание**: Ссылка на спонсорскую страницу, предлагающую кофе для разработчика. По умолчанию ссылка на страницу Boosty.

**Тип**: `str`

**Инициализация**:
Инициализируется значением из настроек или значением по умолчанию.
