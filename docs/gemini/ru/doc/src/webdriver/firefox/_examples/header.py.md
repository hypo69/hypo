# Модуль `header.py`

## Обзор

Данный модуль устанавливает корень проекта в системный путь.

## Оглавление

- [Обзор](#обзор)
- [Импорт](#импорт)
- [Переменные](#переменные)

## Импорт

В модуле импортируются следующие библиотеки:

- `sys`: Используется для манипуляций с системными переменными и путями.
- `os`: Используется для взаимодействия с операционной системой, например, для работы с путями.
- `pathlib.Path`: Используется для создания объектов, представляющих пути к файлам и каталогам.

```python
import sys,os
from pathlib import Path
```

## Переменные

### `__root__`

```python
__root__ : Path
```

**Описание**:
   - Переменная `__root__` типа `pathlib.Path` хранит путь к корневой директории проекта. Она вычисляется путем нахождения последней позиции подстроки `hypotez` в текущей рабочей директории и обрезания строки до этого места, а также добавления длины подстроки `hypotez` (7 символов).

**Пример**:
- Если текущая рабочая директория `/home/user/projects/hypotez/src/webdriver/firefox/_examples`, то значение `__root__` будет `/home/user/projects/hypotez/`.

### Системный путь

```python
sys.path.append (__root__)
```

**Описание**:
   - В системный путь добавляется переменная `__root__`. Это позволяет импортировать модули и пакеты из корневой директории проекта.
