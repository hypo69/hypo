# Модуль `hypotez/src/suppliers/aliexpress/api/_examples/rest/__init__.py`

## Обзор

Данный модуль инициализирует пакет `rest`, импортируя классы запросов API AliExpress. Он предоставляет доступ к различным запросам, связанным с продуктами, заказами, горячими товарами, акциями и категориями.

## Импортированные классы

### `AliexpressAffiliateProductSmartmatchRequest`

**Описание**: Класс для запроса смарт-подбора продуктов AliExpress.

### `AliexpressAffiliateOrderGetRequest`

**Описание**: Класс для получения информации о конкретном заказе AliExpress.

### `AliexpressAffiliateOrderListRequest`

**Описание**: Класс для получения списка заказов AliExpress.

### `AliexpressAffiliateHotproductDownloadRequest`

**Описание**: Класс для загрузки горячих товаров AliExpress.

### `AliexpressAffiliateProductdetailGetRequest`

**Описание**: Класс для получения детальной информации о продукте AliExpress.

### `AliexpressAffiliateHotproductQueryRequest`

**Описание**: Класс для запроса горячих товаров AliExpress.

### `AliexpressAffiliateFeaturedpromoProductsGetRequest`

**Описание**: Класс для получения списка товаров из избранных акций AliExpress.

### `AliexpressAffiliateFeaturedpromoGetRequest`

**Описание**: Класс для получения информации об избранных акциях AliExpress.

### `AliexpressAffiliateProductQueryRequest`

**Описание**: Класс для запроса продуктов AliExpress.

### `AliexpressAffiliateCategoryGetRequest`

**Описание**: Класс для получения списка категорий AliExpress.

### `AliexpressAffiliateOrderListbyindexRequest`

**Описание**: Класс для получения списка заказов AliExpress с использованием индекса.

### `AliexpressAffiliateLinkGenerateRequest`

**Описание**: Класс для генерации партнерских ссылок AliExpress.
