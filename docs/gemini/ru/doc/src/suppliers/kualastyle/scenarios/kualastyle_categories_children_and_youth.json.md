# Документация для `kualastyle_categories_children_and_youth.json`

## Обзор

Данный JSON-файл содержит информацию о категории товаров "ילדים ונוער" (Дети и молодежь) для сайта Kualastyle. Он указывает, что у данной категории есть подкатегории, но не содержит конкретных сценариев для них.

## Структура файла

### Ключ `category name on site`
**Описание**: Название категории товаров на сайте.
- **Тип данных**: `string`
- **Значение**: `"ילדים ונוער"` (Дети и молодежь)

### Ключ `have subcategories`
**Описание**: Указывает, есть ли у данной категории подкатегории.
- **Тип данных**: `boolean`
- **Значение**: `true`

### Ключ `scenarios`
**Описание**: Объект, содержащий сценарии для категории.
- **Тип данных**: `object`
- **Значение**: Пустой объект `{}`. Это означает, что конкретные сценарии для этой категории не определены в данном файле.
