# Модуль `src.suppliers.aliexpress.scenarios`

## Обзор

Модуль `src.suppliers.aliexpress.scenarios` содержит сценарии для работы с AliExpress. Он включает в себя различные функции и классы, предназначенные для выполнения определенных действий на платформе AliExpress.
