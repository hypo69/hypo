grandadvance_categories_matherboards_msi.json
==========================================

This file contains a JSON object describing MSI motherboards categorized by socket type.  Each key represents a socket type, and the corresponding value is a dictionary with details such as brand, URL, active status, and associated PrestaShop categories.
