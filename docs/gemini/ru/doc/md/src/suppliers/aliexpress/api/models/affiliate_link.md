# Модуль hypotez/src/suppliers/aliexpress/api/models/affiliate_link.py

## Обзор

Модуль `affiliate_link.py` содержит определение класса `AffiliateLink`, представляющего данные о партнерской ссылке AliExpress.  Класс содержит атрибуты для хранения ссылки и источника данных.

## Оглавление

* [AffiliateLink](#affiliate-link)


## Классы

### `AffiliateLink`

**Описание**: Класс `AffiliateLink` описывает структуру данных для представления партнерской ссылки.

**Атрибуты**:

- `promotion_link` (str): Ссылка на промо акцию.
- `source_value` (str): Источник данных о ссылке.
