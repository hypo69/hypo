# Модуль `hypotez/src/suppliers/chat_gpt/scenarios/grab_lilnks_to_chats.py`

## Обзор

Данный модуль содержит функции для получения ссылок на отдельные чаты с использованием веб-драйвера. Он использует предварительно загруженный локатор, определённый в файле `chats_list.json`. Модуль предполагает наличие установленных веб-драйверов (Chrome, Firefox).

## Определения

### Переменная `MODE`

```python
MODE = 'dev'
```

**Описание**:  Переменная, скорее всего, задает режим работы. Значение `'dev'` предполагает, что это режим разработки.


## Функции

### `get_links`

**Описание**: Получает список ссылок на отдельные чаты.

**Параметры**:

- `d` (Driver): Экземпляр класса веб-драйвера.

**Возвращает**:

- `list[str]`: Список ссылок на отдельные чаты.

**Вызывает исключения**:

- Любые исключения, которые могут быть подняты методами класса `Driver`.


##  Примеры использования

```python
if __name__ == '__main__':
    d = Driver(Firefox)
    d.get_url('https://chatgpt.com/')
    links = get_links(d)
    # ... дальнейшая обработка списка ссылок `links`
```

**Примечание**: Данный пример предполагает, что функция `get_links` возвращает список ссылок. При отсутствии результата, обработка должна учитывать `None` или пустой список.
