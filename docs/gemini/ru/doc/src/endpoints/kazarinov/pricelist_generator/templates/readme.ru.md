# Шаблон для создания HTML отчета из сценария мехирона

## Обзор

Данный шаблон описывает создание HTML отчета на основе данных, полученных из сценария мехирона.  Он предназначен для структурирования результатов в удобном для чтения формате.

## Структура шаблона

Шаблон предполагает наличие данных в формате словаря или списка словарей.  Каждый элемент данных содержит ключевые значения, которые будут использоваться для формирования таблиц и других элементов отчета.

## Функции

### `generate_html_report`

**Описание**: Функция генерирует HTML отчет на основе входных данных.

**Параметры**:

- `data` (list[dict]): Список словарей, содержащих данные для отчета. Каждый словарь представляет одну строку данных. Ожидается, что все словари имеют одинаковые ключи.
- `report_title` (str): Заголовок отчета.
- `table_headers` (list[str]): Список заголовков таблицы, соответствующих ключам в словарях `data`.


**Возвращает**:

- `str`: HTML код отчета. Возвращает строку, содержащую HTML-код.

**Вызывает исключения**:

- `TypeError`: Если входные данные не соответствуют ожидаемому формату (список словарей).
- `ValueError`: Если `table_headers` не соответствует ключам в словарях `data`.


```python
def generate_html_report(data: list[dict], report_title: str, table_headers: list[str]) -> str:
    """
    Args:
        data (list[dict]): Список словарей, содержащих данные для отчета. Каждый словарь представляет одну строку данных. Ожидается, что все словари имеют одинаковые ключи.
        report_title (str): Заголовок отчета.
        table_headers (list[str]): Список заголовков таблицы, соответствующих ключам в словарях `data`.

    Returns:
        str: HTML код отчета. Возвращает строку, содержащую HTML-код.

    Raises:
        TypeError: Если входные данные не соответствуют ожидаемому формату (список словарей).
        ValueError: Если `table_headers` не соответствует ключам в словарях `data`.
    """
    # ... (Реализация функции, генерирующей HTML код) ...
    # Пример:
    html_report = f"<html><head><title>{report_title}</title></head><body><h1>{report_title}</h1><table><tr>"
    for header in table_headers:
        html_report += f"<th>{header}</th>"
    html_report += "</tr>"
    for row in data:
        html_report += "<tr>"
        for header in table_headers:
            html_report += f"<td>{row.get(header, 'N/A')}</td>"  # Обработка отсутствующих значений
        html_report += "</tr>"
    html_report += "</table></body></html>"
    return html_report
```

## Пример использования

```python
# Пример данных
data = [
    {'column1': 'value1', 'column2': 'value2'},
    {'column1': 'value3', 'column2': 'value4'},
]

# Заголовки таблицы
headers = ['column1', 'column2']

# Генерируем отчет
report = generate_html_report(data, "Мой отчет", headers)
print(report)
```
