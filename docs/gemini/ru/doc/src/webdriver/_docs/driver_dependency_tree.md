# Модуль `src.webdriver.driver`

## Обзор

Этот модуль предоставляет базовый класс `Driver` и классы драйверов для различных браузеров (Chrome, Firefox, Edge).  Он обеспечивает абстракцию для взаимодействия с браузером через Selenium, включая навигацию, взаимодействие с элементами, получение данных и другие действия.

## Классы

### `Driver`

**Описание**: Базовый класс для взаимодействия с браузером.  Этот класс использует метакласс `DriverMeta` для динамического создания классов драйверов.


**Атрибуты:**
* `previous_url: str`: Предыдущий URL страницы.
* `referrer: str`: URL страницы-источника.
* `page_lang: str`: Язык страницы.
* `ready_state`: Состояние загрузки страницы.
* `get_page_lang()`: Возвращает язык страницы.
* `unhide_DOM_element()`: Раскрывает элемент DOM.
* `get_referrer()`: Возвращает ссылку-источника.
* `window_focus()`: Фокусирует текущее окно.
* `execute_locator()`: Выполняет локатор.
* `click()`: Кликает по элементу.
* `get_webelement_as_screenshot()`: Возвращает скриншот элемента.
* `get_attribute_by_locator()`: Возвращает атрибут элемента.
* `send_message()`: Отправляет сообщение.
* `send_key_to_webelement()`: Отправляет текст в элемент.



**Методы:**

* `driver_payload(self)`: Возвращает данные драйвера.

* `scroll(self, scrolls: int, frame_size: int, direction: str, delay: float) -> None | bool`: Прокручивает страницу.
  * **Параметры:**
      * `scrolls` (int): Количество прокруток.
      * `frame_size` (int): Размер фрейма.
      * `direction` (str): Направление прокрутки ('up' или 'down').
      * `delay` (float): Задержка между прокрутками.
  * **Возвращает:** `None | bool`: Возвращает `None`, если операция не выполнена или `True`, если выполнена успешно.

* `locale(self) -> None | str`: Возвращает локаль.

* `get_url(self, url: str) -> bool`: Переходит на указанный URL.
  * **Параметры:**
      * `url` (str): URL для перехода.
  * **Возвращает:** `bool`: `True`, если переход успешен, `False` иначе.

* `extract_domain(self, url: str) -> str`: Извлекает домен из URL.
  * **Параметры:**
      * `url` (str): URL для извлечения домена.
  * **Возвращает:** `str`: Домен.

* `_save_cookies_localy(self, to_file: str | Path) -> bool`: Сохраняет куки в локальный файл.
  * **Параметры:**
      * `to_file` (str | Path): Путь к файлу для сохранения куки.
  * **Возвращает:** `bool`: `True`, если сохранение прошло успешно.

* `page_refresh(self) -> bool`: Обновляет страницу.
  * **Возвращает:** `bool`: `True`, если обновление прошло успешно.

* `wait(self, interval: float)`: Ожидает заданное время.
  * **Параметры:**
      * `interval` (float): Интервал ожидания в секундах.

* `delete_driver_logs(self) -> bool`: Очищает логи драйвера.
  * **Возвращает:** `bool`: `True`, если логи удалены, `False` иначе.


### `DriverMeta`

**Описание**: Метакласс для динамического создания классов драйверов.

**Методы:**

* `__call__(cls, webdriver_cls, *args, **kwargs)`: Метод вызова метакласса.
