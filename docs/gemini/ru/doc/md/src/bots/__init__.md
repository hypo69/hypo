# Модуль hypotez/src/bots/__init__.py

## Обзор

Этот модуль содержит инициализацию для ботов, предоставляя доступ к классам ботов, например, TelegramBot.  Он устанавливает режим работы (MODE) и импортирует необходимые классы из других файлов.

## Переменные

### `MODE`

**Описание**: Переменная, определяющая режим работы бота (например, 'dev', 'prod').

**Значение**:  'dev' (по умолчанию).


## Классы

### `TelegramBot`

**Описание**: Класс, представляющий Telegram бота.  Подробная информация о классе находится в файле `telegram.py`.
