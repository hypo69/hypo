Module Description
------------------

This module contains the `AliAffiliatedProducts` class for collecting product data from AliExpress,
generating affiliate links, and saving associated images and videos. It leverages the `AliApi` class for API interactions and
includes utility functions for handling URLs, saving media, and data serialization.  It's designed for processing a list of product IDs or URLs.
