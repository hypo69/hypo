## HTML-структура

### `<html>`

Корневой элемент HTML-документа.
- `lang="en"`: Атрибут, указывающий, что язык содержимого - английский.

### `<head>`

Раздел, содержащий метаданные и ссылки на внешние ресурсы.
- `<meta charset="UTF-8">`: Устанавливает кодировку символов как UTF-8.
- `<meta name="viewport" content="width=device-width, initial-scale=1.0">`: Настройка области просмотра для адаптивного дизайна.
- `<title>Hypotez</title>`: Устанавливает заголовок вкладки браузера.

### `<body>`

Раздел, содержащий отображаемое содержимое страницы.
- `<h1>Hypotez</h1>`: Заголовок первого уровня, отображающий название расширения.
- `<p>Привет, Это Давидка. Я обучаю модель</p>`: Параграф, содержащий приветственное сообщение.

## Переменные

### `MODE`

- **Описание**: Глобальная переменная, устанавливающая режим работы расширения.
- **Тип**: `str`
- **Значение**: `"debug"`
