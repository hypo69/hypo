# Документация для `product.json`

## Обзор

Данный файл `product.json` содержит определения локаторов и параметров для извлечения данных о продуктах с веб-сайта. Он определяет, какие атрибуты, селекторы, методы поиска и другие настройки используются для каждого поля продукта.
