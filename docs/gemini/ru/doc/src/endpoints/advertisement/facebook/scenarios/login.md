# Модуль hypotez/src/endpoints/advertisement/facebook/scenarios/login.py

## Обзор

Данный модуль содержит сценарий для входа на Facebook. Он использует экземпляр драйвера `Driver` для взаимодействия с веб-элементами, ввода логина и пароля, а также нажатия кнопки входа. Модуль загружает необходимые локаторы из файла `login.json` и использует библиотеку `gs`.


## Функции

### `login`

**Описание**: Функция для входа на Facebook.  Использует драйвер для заполнения формы авторизации.

**Параметры**:

- `d` (Driver): Экземпляр драйвера для взаимодействия с веб-элементами.

**Возвращает**:

- `bool`: `True`, если авторизация прошла успешно, иначе `False`.

**Вызывает исключения**:

- `Exception`: В случае возникновения ошибок при вводе логина, пароля или нажатии кнопки.  Подробности ошибки выводятся в лог.


**Детали реализации**:

Функция `login` получает данные авторизации (`credentials`) из переменной `gs.facebook_credentials`.  Затем последовательно выполняет следующие действия:
1. Ввод логина в поле email с помощью `d.send_key_to_webelement`. Обработка исключений `Exception` важна для предотвращения аварийного завершения программы.
2. Ожидание (1.3 секунды) с помощью `d.wait`.
3. Ввод пароля в поле password с помощью `d.send_key_to_webelement`. Аналогично, обработка исключений `Exception` важна.
4. Ожидание (0.5 секунды) с помощью `d.wait`.
5. Нажатие кнопки входа с помощью `d.execute_locator`. Обработка исключений `Exception` важна.

В случае успешного выполнения всех операций функция возвращает `True`. В противном случае, возвращает `False` и записывает сообщение об ошибке в лог.
