## Примеры использования драйвера

### Функция `main`

**Описание**: Главная функция, демонстрирующая примеры работы с драйверами.

**Примеры:**

#### Создание экземпляра Chrome драйвера и навигация по URL

```python
chrome_driver = Driver(Chrome)
if chrome_driver.get_url("https://www.example.com"):
    print("Успешно перешли на URL")
```

**Описание**: Создает экземпляр класса `Driver` с типом `Chrome` и пытается перейти на указанный URL.  Выводит сообщение об успехе, если переход выполнен успешно.

#### Извлечение домена из URL

```python
domain = chrome_driver.extract_domain("https://www.example.com/path/to/page")
print(f"Извлеченный домен: {domain}")
```

**Описание**: Извлекает доменную часть из переданного URL.

#### Сохранение куки в локальный файл

```python
success = chrome_driver._save_cookies_localy()
if success:
    print("Куки были сохранены успешно")
```

**Описание**: Сохраняет куки в локальный файл.

#### Обновление текущей страницы

```python
if chrome_driver.page_refresh():
    print("Страница успешно обновлена")
```

**Описание**: Обновляет текущую страницу в браузере.

#### Прокрутка страницы вниз

```python
if chrome_driver.scroll(scrolls=3, direction='forward', frame_size=1000, delay=1):
    print("Страница успешно прокручена вниз")
```

**Описание**: Прокручивает страницу вниз на определенное количество прокруток.

#### Получение языка текущей страницы

```python
page_language = chrome_driver.locale
print(f"Язык страницы: {page_language}")
```

**Описание**: Получает язык текущей страницы.

#### Установка пользовательского User-Agent

```python
user_agent = {
    'user-agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/100.0.4896.127 Safari/537.36'
}
custom_chrome_driver = Driver(Chrome, user_agent=user_agent)
if custom_chrome_driver.get_url("https://www.example.com"):
    print("Успешно перешли на URL с пользовательским User-Agent")
```

**Описание**: Устанавливает пользовательский User-Agent для драйвера.

#### Поиск элемента по CSS селектору

```python
element = chrome_driver.find_element(By.CSS_SELECTOR, 'h1')
if element:
    print(f"Найден элемент с текстом: {element.text}")
```

**Описание**: Находит элемент на странице по CSS селектору.

#### Получение текущего URL

```python
current_url = chrome_driver.current_url
print(f"Текущий URL: {current_url}")
```

**Описание**: Получает текущий адрес URL.

#### Фокусировка окна

```python
chrome_driver.window_focus()
print("Окно сфокусировано")
```

**Описание**: Устанавливает фокус на окно браузера.
