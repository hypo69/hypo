# מודול Gomoku

## סקירה כללית

מודול זה מיישם את משחק הלוח האסטרטגי גומוקו (Gomoku), הידוע גם בשם "חמישה בשורה". המשחק מתנהל בין שני שחקנים, שמתחרים על ידי הצבת אסימונים על לוח 15x15, במטרה ליצור רצף רציף של חמישה אסימונים משלהם, בין אם אופקית, אנכית או אלכסונית. מודול זה כולל גם בינה מלאכותית בסיסית המאפשרת לשחקן אחד להתחרות מול המחשב.

## פונקציות

### `print_board`

**תיאור**: פונקציה זו מדפיסה את מצב הלוח הנוכחי לקונסולה.

**פרמטרים**:
- `board` (list): רשימה דו-ממדית המייצגת את לוח המשחק.

**החזרות**:
- `None`: פונקציה זו אינה מחזירה ערך.

### `check_winner`

**תיאור**: פונקציה זו בודקת אם יש מנצח במשחק.

**פרמטרים**:
- `board` (list): רשימה דו-ממדית המייצגת את לוח המשחק.
- `player` (int): מספר השחקן (1 או 2) שיש לבדוק אם ניצח.

**החזרות**:
- `bool`: מחזיר `True` אם יש מנצח, אחרת `False`.

### `check_full_board`

**תיאור**: פונקציה זו בודקת אם הלוח מלא.

**פרמטרים**:
- `board` (list): רשימה דו-ממדית המייצגת את לוח המשחק.

**החזרות**:
- `bool`: מחזירה `True` אם הלוח מלא, אחרת `False`.
