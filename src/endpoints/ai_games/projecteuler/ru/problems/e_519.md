# Задача No 519
## Идеальные квадраты

## Условия

Пусть S(n) — количество идеальных квадратов (т.е. целых чисел, являющихся квадратами других целых чисел), меньших или равных n.

Например, S(16) = 4 (1, 4, 9, 16).

Рассмотрим функцию F(n,k), которая возвращает сумму S(n) + S(n²) + S(n³) + ... + S(nᵏ).

Таким образом, F(10, 3) = S(10) + S(100) + S(1000) = 3 + 10 + 31 = 44.

Найдите F(10⁶, 10¹⁸) mod 10⁹+7.

**Примечание:** 
Может оказаться полезным, что для целых положительных n, S(n) = ⌊√n⌋.
