# Модуль `hypotez/src/suppliers/aliexpress/api/helpers/products.py`

## Обзор

Этот модуль содержит функции для обработки данных о продуктах, полученных с API AliExpress.  Функции предназначены для подготовки данных в формате, удобном для дальнейшего использования.


## Функции

### `parse_product`

**Описание**:  Функция обрабатывает отдельный продукт, преобразуя URL изображений в строковый формат.

**Параметры**:
- `product`: (Объект): Объект, представляющий данные о продукте.  Ожидается, что у него есть атрибут `product_small_image_urls`, который является объектом, содержащим URL изображений, и можно преобразовать к строке.

**Возвращает**:
- `product`: (Объект): Объект продукта с обновленным атрибутом `product_small_image_urls`, содержащим строку URL.

**Вызывает исключения**:
- Нет.  Возможные ошибки, связанные с некорректным форматом данных, должны быть обработаны вызывающей функцией.


### `parse_products`

**Описание**: Функция обрабатывает список продуктов, применяя функцию `parse_product` к каждому элементу.

**Параметры**:
- `products`: (Список): Список объектов, представляющих данные о продуктах.

**Возвращает**:
- `new_products`: (Список): Новый список обработанных объектов продуктов.

**Вызывает исключения**:
- Нет. Возможные ошибки, связанные с некорректным форматом данных внутри списка, должны быть обработаны вызывающей функцией.
