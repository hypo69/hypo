# Модуль hypotez/src/suppliers/aliexpress/gapi

## Обзор

Этот модуль содержит инициализационные функции и переменные для работы с API AliExpress. Он определяет режим работы (MODE) и импортирует необходимые модули, включая версию API и документацию.

## Переменные

### `MODE`

**Описание**: Строковая переменная, определяющая режим работы. В данном случае, `MODE = 'dev'`.

**Тип**: `str`

**Значение**: `'dev'`


## Модули

### `packaging.version`

**Описание**: Модуль для работы с версиями пакетов.

### `aliexpress.gapi.version`

**Описание**: Модуль, содержащий информацию о версии API AliExpress. Содержит переменные `__version__`, `__doc__`, `__details__`.

В этом модуле нет функций.
