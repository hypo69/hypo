# Шаблон таблицы прайса (ru)

## Обзор

Данный файл `template_table_ru.html` представляет собой шаблон HTML для генерации таблицы прайса. Шаблон отображает информацию о продуктах, включая заголовок, описание, технические характеристики, изображение и цену.

## Структура шаблона

### Тег `<html>`

Содержит всю разметку HTML документа.

### Тег `<head>`

Содержит мета-теги, заголовок страницы (`title`) и стили (`<style>`). Стили определяют внешний вид таблицы, включая цвета, шрифты, размеры, отступы и др.


### Тег `<body>`

Содержит основную контентную часть страницы.

#### Див `.container`

Оборачивает весь контент в контейнер, обеспечивающий центрирование и макетирование.

##### Заголовок `<h1>`

Отображает заголовок, который будет отображаться в начале страницы.

##### Параграф `<p class="lead">`

Отображает описание, заданное параметром `description`.

##### Таблица `<table>`

Выводит таблицу с информацией о продуктах. Каждая строка таблицы `.product-card` представляет собой карточку товара.

###### Ячейка `<td>`

Содержит:
* Заголовок `<h3>` — заголовок продукта.
* Вложенную таблицу `<table>` — для выравнивания изображения и описания.
* Изображение `<img>` — выводит изображение товара.
* Блок `.product-info` — содержит описание и технические характеристики продукта.
* Блок `.price-tag` — отображает цену продукта.

#### Див `.footer`

Отображает общую стоимость всех продуктов с использованием класса `.price-tag`.

## Переменные шаблона

Шаблон использует следующие переменные:

* **`title`**: Заголовок страницы (строка).
* **`description`**: Описание (строка).
* **`products`**: Список объектов данных о продуктах.
    * Каждый объект `product` содержит следующие атрибуты:
        * `product_title`: Название продукта (строка).
        * `image_local_saved_path`: Путь к локальному изображению продукта (строка).
        * `product_description`: Описание продукта (строка).
        * `specification`: Технические характеристики (строка).
* **`price`**: Общая стоимость (число).
* **`currency`**: Валюта (строка).

## Обработка данных

Шаблон использует цикл `{% for product in products %}` для итерации по списку продуктов и вывода информации о каждом из них в отдельной строке таблицы.

## Стиль (CSS)

Стиль задает отображение элементов страницы, включая:

* Цвет фона.
* Цвет текста.
* Шрифт.
* Центрирование элементов.
* Размеры элементов.
* Отступы.
* Стиль карточек товаров `.product-card`.
* Стиль изображения товара `<img>`.
* Стиль блока с информацией о товаре `.product-info`.
* Стиль блока с ценой `.price-tag`.
* Стиль подвала `.footer`.
