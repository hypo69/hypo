# Модуль tinytroupe/__init__.py

## Обзор

Этот модуль инициализирует TinyTroupe, устанавливая необходимые библиотеки, читает конфигурационный файл, инициализирует логгирование и предоставляет функции для форматирования вывода конфигурации и добавления пользовательского CSS в Jupyter Notebook.


## Импорты

- `os`: Для взаимодействия с операционной системой.
- `logging`: Для ведения журнала.
- `configparser`: Для чтения конфигурационных файлов.
- `rich`: Для форматированного вывода в консоли.
- `rich.jupyter`: Для интеграции с Jupyter Notebook.
- `sys`: Для манипуляций с системными переменными.
- `utils`: Для использования функций из модуля `utils`.


## Функции

### `read_config_file`

**Описание**: Читает конфигурационный файл и возвращает объект `ConfigParser`.

**Параметры**:
-  Нет.

**Возвращает**:
- `configparser.ConfigParser`: Объект, содержащий данные конфигурации.

**Вызывает исключения**:
- `FileNotFoundError`: Если конфигурационный файл не найден.
- `configparser.Error`: Если есть ошибки в формате файла конфигурации.


### `pretty_print_config`

**Описание**: Форматированно выводит конфигурацию в консоль.

**Параметры**:
- `config (configparser.ConfigParser)`: Конфигурация для вывода.


**Возвращает**:
-  Не имеет возвращаемого значения.

**Вызывает исключения**:
-  Нет.


### `start_logger`

**Описание**: Инициализирует логгер, настраивая его уровень логирования и обработку ошибок.

**Параметры**:
- `config (configparser.ConfigParser)`: Конфигурация, содержащая настройки логгера.

**Возвращает**:
-  Не имеет возвращаемого значения.

**Вызывает исключения**:
- `ValueError`: Если конфигурация содержит неверные настройки для логгера.


### `inject_html_css_style_prefix`

**Описание**: Добавляет префикс CSS стилей к строке HTML.

**Параметры**:
- `html_format (str)`: Строка HTML, к которой необходимо добавить префикс.
- `css_style_prefix (str)`: Префикс CSS стиля, который нужно добавить.

**Возвращает**:
- `str`: Строка HTML с добавленным префиксом CSS.

**Вызывает исключения**:
- `TypeError`: Если входные параметры имеют неправильный тип.
