hypotez.src.utils.file
=======================

.. module:: hypotez.src.utils.file
    :platform: Windows, Unix
    :synopsis: Module for file operations


This module provides functions for various file operations, including saving data to files, reading file contents, getting filenames in a directory, and recursively searching and reading files.  It also includes functionality to remove Byte Order Marks (BOM) from text files, primarily for Python files within a given directory structure.
