# Модуль hypotez/src/suppliers/ivory/login.py

## Обзор

Данный модуль содержит константу `MODE`, которая, предположительно, задаёт режим работы системы.

## Константы

### `MODE`

**Описание**: Константа, определяющая режим работы системы.  Предполагается, что она может принимать значение 'dev'.

**Значение**: `'dev'`
