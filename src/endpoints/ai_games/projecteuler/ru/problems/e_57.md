# Задача No 57
## Квадратные корни
## Условия
Рассмотрим последовательность приближений для квадратного корня из 2:

1 + 1/2 = 3/2 = 1.5

1 + 1/(2 + 1/2) = 7/5 = 1.4

1 + 1/(2 + 1/(2 + 1/2)) = 17/12 = 1.41666...

1 + 1/(2 + 1/(2 + 1/(2 + 1/2))) = 41/29 = 1.41379...

Таким образом, первыми восемью членами последовательности являются:

1: 3/2
2: 7/5
3: 17/12
4: 41/29
5: 99/70
6: 239/169
7: 577/408
8: 1393/985

Дробь n-го члена последовательности будет иметь вид (числитель)/(знаменатель). 
В первых 1000 членах последовательности, сколько дробей имеют числитель с большим количеством цифр, чем знаменатель?
