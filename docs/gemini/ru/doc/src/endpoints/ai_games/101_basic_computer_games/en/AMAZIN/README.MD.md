# AMAZIN DRAW A MAZE

## Обзор

Программа генерирует уникальный лабиринт при каждом запуске, гарантируя наличие только одного пути через него. Пользователь может задать размеры лабиринта, то есть количество клеток в ширину и длину.

## Описание

Программа `AMAZIN DRAW A MAZE` предназначена для генерации лабиринтов. При каждом запуске создается новый лабиринт, имеющий только один путь от начала до конца. Размеры лабиринта могут быть заданы пользователем, что позволяет настраивать сложность и внешний вид генерируемых лабиринтов.

## Принцип работы

1.  **Генерация:** Программа использует алгоритм для генерации лабиринта, гарантируя наличие одного и только одного пути.
2.  **Настройка размеров:** Пользователь может задать размеры лабиринта (количество клеток в ширину и длину).
3.  **Вывод:** Сгенерированный лабиринт выводится на экран.

## Параметры

- **Ширина лабиринта:** Задается пользователем, определяя количество клеток в ширину.
- **Длина лабиринта:** Задается пользователем, определяя количество клеток в длину.

## Пример использования

Для запуска программы необходимо выполнить скрипт и указать желаемые размеры лабиринта.
```
python your_script_name.py --width 10 --height 15
```
При этом будет сгенерирован и выведен на экран лабиринт шириной 10 клеток и длиной 15 клеток.
