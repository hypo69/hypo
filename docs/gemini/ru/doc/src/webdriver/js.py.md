# Модуль `src.webdriver.js`

## Обзор

Модуль `src.webdriver.js` предоставляет утилиты на JavaScript для взаимодействия с веб-страницей. Он расширяет возможности Selenium WebDriver, добавляя общие функции на JavaScript для манипуляций видимостью элементов, извлечения информации о странице и управления фокусом браузера.

**Основные возможности:**

1.  Делает невидимые DOM-элементы видимыми для взаимодействия.
2.  Получает метаданные, такие как статус готовности документа, реферер или язык страницы.
3.  Управляет фокусом окна браузера программно.

## Классы

### `JavaScript`

**Описание**: Предоставляет утилиты на JavaScript для взаимодействия с веб-страницей.

**Методы**:
- `__init__`: Инициализирует класс `JavaScript`.
- `unhide_DOM_element`: Делает невидимый DOM элемент видимым.
- `ready_state`: Получает статус загрузки документа.
- `window_focus`: Устанавливает фокус на окно браузера.
- `get_referrer`: Получает URL-адрес реферера текущего документа.
- `get_page_lang`: Получает язык текущей страницы.

**Параметры**:

- `driver` (WebDriver): Экземпляр Selenium WebDriver для выполнения JavaScript.

## Функции

### `unhide_DOM_element`

**Описание**: Делает невидимый DOM элемент видимым, изменяя его свойства стиля.

**Параметры**:
- `element` (WebElement): DOM-элемент, который нужно сделать видимым.

**Возвращает**:
- `bool`: `True`, если скрипт выполнен успешно, `False` в противном случае.

**Вызывает исключения**:
- `Exception`: Возникает при ошибке выполнения JavaScript.

### `ready_state`

**Описание**: Возвращает статус загрузки документа.

**Возвращает**:
- `str`: `'loading'`, если документ все еще загружается, `'complete'`, если загрузка завершена.
   Возвращает пустую строку `''`, если не удалось получить статус документа.

**Вызывает исключения**:
- `Exception`: Возникает при ошибке выполнения JavaScript.

### `window_focus`

**Описание**: Устанавливает фокус на окно браузера, выводя его на передний план.

**Параметры**:
- Нет

**Возвращает**:
- `None`: Функция ничего не возвращает.

**Вызывает исключения**:
- `Exception`: Возникает при ошибке выполнения JavaScript.

### `get_referrer`

**Описание**: Возвращает URL-адрес реферера текущего документа.

**Возвращает**:
- `str`: URL-адрес реферера или пустую строку, если он недоступен.

**Вызывает исключения**:
- `Exception`: Возникает при ошибке выполнения JavaScript.

### `get_page_lang`

**Описание**: Возвращает язык текущей страницы.

**Возвращает**:
- `str`: Код языка страницы или пустую строку, если он недоступен.

**Вызывает исключения**:
- `Exception`: Возникает при ошибке выполнения JavaScript.
