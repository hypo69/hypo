# OpenAI Model Interface Extension - background.js

## Обзор

Данный файл `background.js` является фоновым скриптом расширения Chrome для интерфейса модели OpenAI. Он отвечает за прослушивание событий установки расширения и выводит сообщение в консоль при установке.

## События

### `chrome.runtime.onInstalled`

**Описание**:
Событие, срабатывающее при установке расширения.

**Параметры**:
- `addListener` (function): Функция обратного вызова, выполняемая при установке расширения.

**Возвращает**:
- `None`: Функция не возвращает значения.

**Действия**:
- Выводит сообщение `OpenAI Model Interface Extension Installed` в консоль браузера при установке расширения.
