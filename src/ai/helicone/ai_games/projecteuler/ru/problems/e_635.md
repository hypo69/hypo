# Задача No 635
## Простая арифметика
## Условия
Пусть f(n, k) - функция, определенная следующим образом:
- f(n, 1) = n
- f(n, k) = n - f(n, k-1), если k > 1

Например, f(6, 1) = 6, f(6, 2) = 6 - 6 = 0, f(6, 3) = 6 - 0 = 6, f(6, 4) = 6 - 6 = 0.

Пусть S(n) - сумма f(n, k) при 1 <= k <= n.
Например, S(6) = f(6, 1) + f(6, 2) + f(6, 3) + f(6, 4) + f(6, 5) + f(6, 6) = 6 + 0 + 6 + 0 + 6 + 0 = 18

Определим T(n) = S(1) + S(2) + ... + S(n).

Вычислено, что T(10) = 95 и T(100) = 30350.

Найдите T(10^12) mod 1000000007.
