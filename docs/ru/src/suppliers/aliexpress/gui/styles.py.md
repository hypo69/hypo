# Модуль стилей для графического интерфейса AliExpress
## Обзор

Модуль `styles.py` содержит функции для стилизации элементов графического интерфейса (GUI) приложения для работы с AliExpress. В частности, он предоставляет функцию `set_fixed_size` для установки фиксированного размера виджетов PyQt6.

## Подробнее

Модуль предназначен для централизации общих функций стилизации, чтобы обеспечить единообразный внешний вид пользовательского интерфейса AliExpress. Он использует библиотеку PyQt6 для создания и управления элементами интерфейса. Функция `set_fixed_size` упрощает установку фиксированных размеров виджетов, что может быть полезно для обеспечения консистентности отображения элементов интерфейса на различных устройствах и платформах.

## Функции

### `set_fixed_size`

```python
def set_fixed_size(widget: QtWidgets.QWidget, width: int, height: int):
    """ Set a fixed size for a given widget """
    widget.setFixedSize(width, height)
```

**Назначение**: Устанавливает фиксированный размер для заданного виджета.

**Параметры**:
- `widget` (QtWidgets.QWidget): Виджет, для которого необходимо установить фиксированный размер.
- `width` (int): Ширина виджета в пикселях.
- `height` (int): Высота виджета в пикселях.

**Возвращает**: None

**Вызывает исключения**:
-   `TypeError`: Если переданы неправильные типы аргументов.
-   `ValueError`: Если `width` или `height` имеют отрицательные значения.

**Как работает функция**:
1.  Функция `set_fixed_size` принимает виджет `widget` и желаемые значения ширины `width` и высоты `height`.
2.  Вызывает метод `setFixedSize(width, height)` виджета `widget`, чтобы установить его фиксированный размер.

```
      A: Начало
      │
      B: Вызов setFixedSize(width, height) у виджета
      │
      C: Завершение
```

**Примеры**:

```python
from PyQt6 import QtWidgets
from PyQt6.QtWidgets import QPushButton, QWidget
app = QtWidgets.QApplication([])
button = QPushButton("Кнопка")

# Установка фиксированного размера для кнопки
set_fixed_size(button, 100, 50)

button.show()
app.exec()
