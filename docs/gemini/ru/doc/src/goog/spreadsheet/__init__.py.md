# Модуль `hypotez/src/goog/spreadsheet/__init__.py`

## Обзор

Модуль `hypotez/src/goog/spreadsheet` предназначен для работы с Google Sheets. Он предоставляет классы и функции для взаимодействия с электронными таблицами, включая чтение, запись и другие операции.

## Переменные

### `MODE`

- **Описание**: Режим работы модуля. По умолчанию установлен в `'dev'`.

## Импорты

### `from .spreadsheet import SpreadSheet`

- **Описание**: Импортирует класс `SpreadSheet` из модуля `spreadsheet.py`. Предназначен для базовой работы с таблицами.

### `from .reach_spreadsheet import ReachSpreadsheet`

- **Описание**: Импортирует класс `ReachSpreadsheet` из модуля `reach_spreadsheet.py`. Предназначен для более продвинутой работы с таблицами, включая чтение и запись данных.
