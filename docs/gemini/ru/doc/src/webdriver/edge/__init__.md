# Модуль hypotez/src/webdriver/edge/__init__.py

## Обзор

Этот модуль предоставляет начальную точку входа для работы с драйвером Edge WebDriver. Он импортирует необходимый класс `Edge` из подмодуля `edge`.

## Переменные

### `MODE`

**Описание**: Переменная, определяющая режим работы (например, 'dev' или 'prod').  Текущее значение - 'dev'.

**Тип**: `str`


## Импорты

### `from .edge import Edge`

**Описание**: Импортирует класс `Edge` из модуля `edge`.  Этот импорт позволяет использовать класс `Edge` в других частях проекта.
