# Модуль `src.suppliers.ivory.login`

## Обзор

Данный модуль `login.py` предназначен для работы с поставщиком IVORY. В текущей версии не содержит исполняемого кода, но является частью проекта `hypotez`, отвечающего за поставщиков. Модуль включает необходимые метаданные для идентификации и организации в рамках проекта.
