# Задача No 293
## Замечательные целые числа

## Условия

Положительное целое число называется *замечательным*, если его можно представить в виде суммы различных степеней 2. Например, 13 является замечательным, поскольку 13 = 2³ + 2² + 2⁰.

Пусть *f(n)* будет наименьшим замечательным числом *m*, таким, что *n + m* является простым числом. 
Например, *f(4) = 1*, поскольку 4 + 1 = 5, которое является простым числом, а 1 - наименьшее замечательное число. Также *f(100) = 3*, поскольку 100 + 3 = 103.

Пусть *S(n)* будет суммой всех различных значений *f(i)* для 1 ≤ *i* ≤ *n*.
Например, *S(100) = 1074*.

Найдите *S(10 000 000)*.
