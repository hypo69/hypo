# Задача No 9
## Специальная тройка Пифагора
## Условия
Тройка Пифагора — это набор из трех натуральных чисел a < b < c, для которых выполняется равенство:

a² + b² = c²

Например, 3² + 4² = 9 + 16 = 25 = 5².

Существует ровно одна тройка Пифагора, для которой a + b + c = 1000.
Найдите произведение abc.
