# Модуль нормализации строк и числовых данных

## Обзор

Этот модуль предоставляет функции для нормализации строк, булевых значений, целых и чисел с плавающей точкой. Он также содержит вспомогательные методы для обработки текста, включая удаление HTML-тегов и специальных символов.

## Функции

### `normalize_boolean`

**Описание**: Преобразует входные данные в булево значение.

**Параметры**:
- `input_data` (Any): Данные, которые могут быть представлены в виде булевого значения (например, bool, строка, целое число).

**Возвращает**:
- `bool`: Булево представление входных данных.

**Пример**:
```python
>>> normalize_boolean('yes')
True
```
### `normalize_string`

**Описание**: Нормализует строку или список строк.

**Параметры**:
- `input_data` (str | list): Входные данные, которые могут быть либо строкой, либо списком строк.

**Возвращает**:
- `str`: Очищенная и нормализованная строка в кодировке UTF-8.

**Вызывает исключения**:
- `TypeError`: Если `input_data` не является строкой или списком.

**Пример**:
```python
>>> normalize_string(['Hello', '  World!  '])
'Hello World!'
```

### `normalize_int`

**Описание**: Преобразует входные данные в целое число.

**Параметры**:
- `input_data` (str | int | float | Decimal): Входные данные, которые могут быть числом или его строковым представлением.

**Возвращает**:
- `int`: Целочисленное представление входных данных.

**Пример**:
```python
>>> normalize_int('42')
42
```

### `normalize_float`

**Описание**: Безопасно преобразует входные данные в float или список float.

**Параметры**:
- `value` (Any): Входное значение для преобразования. Может быть единичным значением (число или строка) или итерируемым объектом (список/кортеж).

**Возвращает**:
- `float | List[float] | None`: Значение float, список значений float, или `None`, если преобразование не удалось.

**Пример**:
```python
>>> normalize_float("3.14")
3.14
>>> normalize_float([1, '2.5', 3])
[1.0, 2.5, 3.0]
```

### `normalize_sql_date`

**Описание**: Нормализует входные данные в формат даты SQL (YYYY-MM-DD).

**Параметры**:
- `input_data` (str): Данные, которые могут представлять дату (например, строка, объект datetime).

**Возвращает**:
- `str`: Нормализованная дата в формате SQL (YYYY-MM-DD) или исходное значение, если преобразование не удалось.

**Пример**:
```python
>>> normalize_sql_date('2024-12-06')
'2024-12-06'
>>> normalize_sql_date('12/06/2024')
'2024-12-06'
```

### `simplify_string`

**Описание**: Упрощает входную строку, оставляя только буквы и цифры, а пробелы заменяет на подчеркивания.

**Параметры**:
- `input_str` (str): Строка для упрощения.

**Возвращает**:
- `str`: Упрощенная строка.

**Пример**:
```python
example_str = "It's a test string with 'single quotes', numbers 123 and symbols!"
simplified_str = simplify_string(example_str)
print(simplified_str)  # Output: Its_a_test_string_with_single_quotes_numbers_123_and_symbols
```

### `remove_line_breaks`

**Описание**: Удаляет переносы строк из входной строки.

**Параметры**:
- `input_str` (str): Входная строка.

**Возвращает**:
- `str`: Строка без переносов строк.

### `remove_html_tags`

**Описание**: Удаляет HTML-теги из входной строки.

**Параметры**:
- `input_html` (str): Входная HTML-строка.

**Возвращает**:
- `str`: Строка без HTML-тегов.

### `remove_special_characters`

**Описание**: Удаляет указанные специальные символы из строки или списка строк.

**Параметры**:
- `input_str` (str | list): Входная строка или список строк.
- `chars` (list[str], optional): Список символов для удаления. По умолчанию `None`.

**Возвращает**:
- `str | list`: Обработанная строка или список строк с удаленными символами.

### `normalize_sku`

**Описание**: Нормализует SKU, удаляя специфические ключевые слова на иврите и любые не буквенно-цифровые символы.

**Параметры**:
- `input_str` (str): Входная строка, содержащая SKU.

**Возвращает**:
- `str`: Нормализованная строка SKU.

**Пример**:
```python
>>> normalize_sku("מקט: 303235")
'303235'
>>> normalize_sku("מק''ט: 12345")
'12345'
```
