# Модуль подготовки AI-кампании для AliExpress

## Обзор

Модуль `prepare_ai_campaign.py` предназначен для подготовки и обработки AI-кампаний на платформе AliExpress. Он включает в себя функциональность для редактирования кампаний, обработки категорий, а также общей обработки всех кампаний. Модуль использует различные утилиты для работы с файлами и директориями, а также модуль логирования для записи информации о процессе выполнения.

## Подробнее

Этот модуль является частью системы управления рекламными кампаниями на AliExpress. Он предоставляет инструменты для автоматизации процесса создания и настройки рекламных кампаний с использованием AI.

## Функции

### `AliCampaignEditor.process_ai_campaign`

```python
campaign_editor.process_ai_campaign(campaign_name)
```

**Назначение**: Запускает процесс обработки AI-кампании для указанного имени кампании.

**Параметры**:
- `campaign_name` (str): Имя кампании, которую необходимо обработать.

**Возвращает**:
- `None`: Функция ничего не возвращает.

**Как работает функция**:

1.  Функция `process_ai_campaign` запускает процесс обработки AI-кампании. Она использует имя кампании (`campaign_name`) для определения файлов конфигурации и других ресурсов, необходимых для обработки кампании.
2.  Внутри функции происходит вызов других методов и функций, необходимых для обработки кампании, включая чтение данных, обработку категорий и обновление информации о кампании.

**Примеры**:

```python
campaign_editor.process_ai_campaign('lighting')
```

В данном примере вызывается функция `process_ai_campaign` для обработки кампании с именем 'lighting'.
