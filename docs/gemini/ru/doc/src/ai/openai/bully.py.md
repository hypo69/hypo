# Модуль `src.ai.openai.bully`

## Обзор

Модуль `bully.py` предназначен для демонстрации того, как можно заставить модель ChatGPT генерировать грубые высказывания, имитируя поведение булли. Он использует API OpenAI для взаимодействия с моделью, предоставляя системный промпт и пользовательские сообщения для получения ответа в стиле булли.

## Функции

### `bully`

**Описание**: Функция отправляет сообщение в OpenAI API и получает ответ в стиле булли.

**Параметры**:
- `user_message` (str, optional): Сообщение пользователя. По умолчанию "Hello!".
- `messages` (list, optional): Список сообщений, включая системный промпт. По умолчанию `[{"system": "user", "content": system_prompt}]`.

**Возвращает**:
- `list`: Список сообщений, включая ответ от модели.

**Пример использования:**
```python
import src.ai.openai.bully as bully_module

result = bully_module.bully(user_message="Ты глупый!")
print(result)
```
