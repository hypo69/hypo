# Модуль `hypotez/src/utils/xls.py`

## Обзор

Модуль `xls.py` предоставляет функции для преобразования файлов Excel (`xls`) в формат JSON и обратно. Он умеет обрабатывать несколько листов Excel и сохранять данные в формате JSON.

## Функции

### `read_xls_as_dict`

**Описание**: Читает файл Excel и преобразует его в словарь Python.  По желанию преобразует определённый лист и сохраняет результат в файл JSON. Обрабатывает ошибки.

**Параметры**:

- `xls_file` (str): Путь к файлу Excel.
- `json_file` (str, необязательно): Путь к файлу JSON для сохранения результата. По умолчанию `None`.
- `sheet_name` (Union[str, int], необязательно): Имя листа для обработки. По умолчанию `None` (обрабатываются все листы).

**Возвращает**:

- Union[Dict, List[Dict], bool]: Словарь, содержащий данные из файла Excel.  Ключи словаря - имена листов, значения - списки словарей, представляющие строки. Возвращает `False` в случае ошибки.


**Вызывает исключения**:

- `FileNotFoundError`: Если файл Excel не найден.
- `Exception`: При любых других ошибках при чтении файла или работе с данными.


### `save_xls_file`

**Описание**: Сохраняет данные в формате JSON в файл Excel. Обрабатывает ошибки.

**Параметры**:

- `data` (Dict[str, List[Dict]]): Словарь, содержащий данные для сохранения. Ключи словаря - имена листов, значения - списки словарей, представляющие строки.
- `file_path` (str): Путь к файлу Excel для сохранения.

**Возвращает**:

- bool: `True`, если сохранение прошло успешно, `False` в случае ошибки.

**Вызывает исключения**:

- `Exception`: При любых ошибках при сохранении файла Excel.
