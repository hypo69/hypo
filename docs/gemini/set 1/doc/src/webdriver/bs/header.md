# Модуль hypotez/src/webdriver/bs/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек проекта из файла `settings.json`.  Он также загружает описание проекта из файла `README.MD`.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с текущей директории и двигаясь вверх по дереву директорий. Останавливается на первой директории, содержащей один из указанных маркеров файлов.

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, которые используются для определения корневой директории проекта.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращается директория, в которой находится текущий скрипт.

**Вызывает исключения**:

- Нет


### <ins>Описание глобальных переменных</ins>

**Описание**: Глобальные переменные, хранящие корневую директорию проекта, настройки проекта, описание проекта и другие метаданные.
* `__root__`: Путь к корневой директории проекта (Path)
* `settings`: Словарь с настройками проекта (dict)
* `doc_str`: Строка с описанием проекта из файла README.MD (str)
* `__project_name__`: Название проекта (str)
* `__version__`: Версия проекта (str)
* `__doc__`: Документация проекта (str)
* `__details__`: Дополнительные детали о проекте (str)
* `__author__`: Автор проекта (str)
* `__copyright__`: Авторские права (str)
* `__cofee__`: Ссылка для поддержки разработчика (str)
