# Модуль `header.py`

## Обзор

Модуль `header.py` содержит определения переменных и функций, используемых для настройки и инициализации проекта. Он включает в себя функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и документации из файла `README.MD`, а также определение общих переменных проекта.

## Функции

### `set_project_root`

**Описание**:
Находит корневую директорию проекта, начиная с директории текущего файла, и останавливается на первой директории, содержащей любой из маркерных файлов.

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или директорий для идентификации корневой директории проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена, в противном случае - директория, где находится скрипт.

### Переменные

### `MODE`

**Описание**:
Режим работы приложения (по умолчанию 'dev').

### `__root__`

**Описание**:
Корневая директория проекта, полученная из функции `set_project_root`.

### `settings`

**Описание**:
Словарь с настройками проекта, загруженный из файла `settings.json`.

### `doc_str`

**Описание**:
Строка, содержащая документацию проекта, загруженная из файла `README.MD`.

### `__project_name__`

**Описание**:
Название проекта, взятое из настроек `settings.json` или 'hypotez', если настройки не загружены.

### `__version__`

**Описание**:
Версия проекта, взятая из настроек `settings.json` или пустая строка, если настройки не загружены.

### `__doc__`

**Описание**:
Документация проекта, взятая из `doc_str`.

### `__details__`

**Описание**:
Детали проекта (в данный момент пустая строка).

### `__author__`

**Описание**:
Автор проекта, взятый из настроек `settings.json` или пустая строка, если настройки не загружены.

### `__copyright__`

**Описание**:
Авторские права на проект, взятые из настроек `settings.json` или пустая строка, если настройки не загружены.

### `__cofee__`

**Описание**:
Строка с призывом угостить разработчика чашечкой кофе, взятая из настроек `settings.json` или дефолтная строка, если настройки не загружены.
