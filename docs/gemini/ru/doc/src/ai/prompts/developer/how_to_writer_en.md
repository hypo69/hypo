# Инструкция по использованию кода для работы с входящими данными

## Обзор

Этот файл содержит инструкции по использованию кода для обработки входящих данных. Он описывает шаги, необходимые для выполнения кода, а также предоставляет примеры использования.

## Функции

### `process_input_data`

**Описание**: Данная функция обрабатывает входящие данные, проверяет их на корректность и выполняет дальнейшие действия в зависимости от результата проверки.

**Параметры**:

- `input_data` (dict): Словарь с входящими данными. Необходим для выполнения дальнейшей обработки.

**Возвращает**:

- `result` (dict | None): Словарь с результатами обработки данных или `None`, если произошла ошибка.

**Вызывает исключения**:

- `ValueError`: Возникает, если входящие данные не соответствуют ожидаемому формату.
- `TypeError`: Возникает, если тип входящих данных не соответствует ожидаемому.


**Подробное описание:**

Функция `process_input_data` ожидает словарь `input_data` как аргумент.  Внутри функции происходит проверка входящих данных. Если данные некорректны, генерируется исключение `ValueError` или `TypeError`.  В противном случае данные передаются на дальнейшую обработку. Результат обработки возвращается в виде словаря `result`. В случае возникновения ошибок, функция возвращает `None`.


**Пример использования**:

```python
import json

input_data = {
    "name": "John Doe",
    "age": 30,
    "city": "New York"
}

try:
    result = process_input_data(input_data)
    if result:
        print(json.dumps(result, indent=2))
except (ValueError, TypeError) as ex:
    print(f"Ошибка: {ex}")

```

## Классы

### `InputValidator`

**Описание**: Класс `InputValidator` отвечает за валидацию входящих данных.

**Методы**:

- `validate_data(input_data: dict) -> bool`: Проверяет, соответствуют ли входящие данные определенным критериям. Возвращает `True`, если данные валидны, и `False` в противном случае.

**Подробное описание**:

Метод `validate_data` принимает на вход словарь `input_data`.  Внутри метода происходит проверка, соответствуют ли поля в словаре заданным типам и значениям. Если проверка успешна, метод возвращает `True`. В противном случае возвращается `False`.


```python
class InputValidator:

    def validate_data(self, input_data: dict) -> bool:
        """
        Проверяет корректность входящих данных.

        Args:
            input_data (dict): Входящие данные.

        Returns:
            bool: True, если данные валидны, False в противном случае.
        """
        try:
            if not isinstance(input_data, dict):
                raise TypeError("Входящие данные должны быть словарем")
            if not all(key in input_data for key in ['name', 'age', 'city']):
                raise ValueError("Отсутствуют необходимые поля в данных")
            if not isinstance(input_data.get('age'), int):
                raise TypeError("Возраст должен быть целым числом")

            return True
        exсept (ValueError, TypeError) as e:
            print(f"Ошибка валидации: {e}")
            return False
```
