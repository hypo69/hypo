# ИНСТРУКЦИЯ

## Обзор

Данный файл содержит список команд бота, написанного на Python.  Каждая команда имеет своё назначение и принимает определённые параметры.

## Команды

### `!hi`

**Описание**: Приветствует пользователя.

**Использование**: `!hi`

**Возвращает**: Текстовое приветствие.


### `!train`

**Описание**: Обучает модель на предоставленных данных.

**Использование**: `!train <data> <data_dir> <positive> <attachment>`

**Параметры**:
* `data` (строка): Путь к файлу с данными для обучения.
* `data_dir` (строка): Путь к директории с данными для обучения.
* `positive` (строка): Флаг, указывающий на позитивный класс данных (например, "true" или "1").
* `attachment` (строка): Путь к прикреплённому файлу с данными для обучения.

**Возвращает**: Сообщение об успешном обучении или об ошибке.


### `!test`

**Описание**: Тестирует модель на предоставленных тестовых данных.

**Использование**: `!test <test_data>`

**Параметры**:
* `test_data` (строка): Путь к файлу с тестовыми данными в формате JSON.


**Возвращает**: Результаты тестирования в формате JSON или сообщение об ошибке.

**Возможные исключения**:
* `JSONDecodeError`: Если тестовые данные не соответствуют формату JSON.
* `FileNotFoundError`: Если файл с тестовыми данными не найден.



### `!archive`

**Описание**: Архивирует файлы в указанной директории.

**Использование**: `!archive <directory>`

**Параметры**:
* `directory` (строка): Путь к директории, которую необходимо заархивировать.

**Возвращает**: Сообщение об успешном архивировании или об ошибке.


### `!select_dataset`

**Описание**: Выбирает набор данных для обучения из указанной директории.

**Использование**: `!select_dataset <path_to_dir_positive> <positive>`

**Параметры**:
* `path_to_dir_positive` (строка): Путь к директории с положительными образцами данных.
* `positive` (строка): Флаг, указывающий на позитивный класс данных (например, "true" или "1").


**Возвращает**:  Список путей к выбранным наборам данных или сообщение об ошибке.


### `!instruction`

**Описание**: Выводит сообщение с инструкцией.

**Использование**: `!instruction`

**Возвращает**:  Текст инструкции.
