# Задача No 449
## Заполняя треугольник
## Условия
Начнем с равностороннего треугольника, стороны которого имеют длину *n*. Затем добавим еще один треугольник, приклеив его по одной из сторон исходного, получив ромб. Затем мы добавляем еще один треугольник, приклеивая его к одной из открытых сторон фигуры. Мы продолжаем добавлять треугольники таким образом, пока не получится фигура, стороны которой имеют длину *n*.

На рисунке показан процесс, в котором равносторонние треугольники с длиной стороны 1 добавлены, чтобы получить шестиугольник.

![Рисунок процесса](https://projecteuler.net/project/images/p449_triangle.gif)

Обозначим через *T(n)* количество разных фигур, которые могут быть сформированы таким образом, при условии, что длина стороны треугольника равна *n*.

Например, *T(1) = 1, T(2) = 3, T(3) = 13, T(10) = 5188398*

Найдите *T(10<sup>12</sup>)* mod 10<sup>8</sup>.
