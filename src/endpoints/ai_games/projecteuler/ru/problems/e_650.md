# Задача No 650
## Дивизоры сумм факториалов
## Условия
Пусть $d(n)$ обозначает количество делителей числа $n$.
Пусть $f(n) = \sum_{k=1}^{n} k!$.

Дано $S(n) = \sum_{k=1}^{n} d(f(k))$.
Например, $f(3) = 1! + 2! + 3! = 1 + 2 + 6 = 9$, и $d(9) = 3$,
и, следовательно, $S(3) = d(f(1)) + d(f(2)) + d(f(3)) = d(1) + d(3) + d(9) = 1 + 2 + 3 = 6$.

Также, $S(10) = 135$, $S(100) = 15810$.

Найдите $S(10^8)$ по модулю $1\,000\,000\,007$.
