## Создание рекламной кампании

### 1. Инициализация кампании

**Описание**: Настройка базовых параметров кампании (имя, язык, валюта).

**Параметры**:
- `campaign_name` (str): Имя рекламной кампании.
- `language` (str): Язык кампании.
- `currency` (str): Валюта кампании.

**Возвращает**:
- `None`: Данные не возвращаются.


### 2. Создание директорий для кампании

**Описание**: Создание директорий для хранения данных кампании и категорий.

**Параметры**:
- `campaign_name` (str): Имя рекламной кампании.
- `categories` (list[str]): Список категорий для кампании.

**Возвращает**:
- `None`: Данные не возвращаются.


### 3. Сохранение конфигурации кампании

**Описание**: Сохранение конфигурации кампании в файле.

**Параметры**:
- `campaign_name` (str): Имя рекламной кампании.
- `campaign_config` (dict): Словарь с конфигурацией кампании.

**Возвращает**:
- `None`: Данные не возвращаются.

**Возможные исключения**:
- `IOError`: Ошибка при работе с файлами.


### 4. Сбор данных о продуктах

**Описание**: Загрузка данных о продуктах для кампании.

**Параметры**:
- `product_urls` (list[str]): Список URL-адресов продуктов.

**Возвращает**:
- `dict`: Словарь с данными о продуктах.


### 5. Сохранение данных о продуктах

**Описание**: Сохранение данных о продуктах в файле.

**Параметры**:
- `campaign_name` (str): Имя рекламной кампании.
- `product_data` (dict): Данные о продуктах.

**Возвращает**:
- `None`: Данные не возвращаются.


### 6. Создание рекламных материалов

**Описание**: Создание рекламных материалов на основе данных о продуктах.

**Параметры**:
- `campaign_name` (str): Имя рекламной кампании.
- `product_data` (dict): Данные о продуктах.

**Возвращает**:
- `None`: Данные не возвращаются.


### 7. Просмотр и публикация кампании

**Описание**: Просмотр и публикация кампании.

**Параметры**:
- `campaign_name` (str): Имя рекламной кампании.

**Возвращает**:
- `None`: Данные не возвращаются.
