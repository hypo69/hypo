# Задача No 680
## Разложение на простые числа факториалов
## Условия
Пусть p(n) — наименьший простой делитель числа n.
Например, p(10) = 2, p(11) = 11, а p(121) = 11.

Пусть P(n) — это сумма p(k) для всех k от 2 до n включительно.
Например, P(10) = 2 + 3 + 2 + 5 + 2 + 7 + 2 + 3 + 2 + 11 = 39.

Пусть F(n) — сумма P(k!) для всех k от 2 до n включительно.
Например, F(4) = P(2!) + P(3!) + P(4!) = P(2) + P(6) + P(24) = (2) + (2+2+3) + (2+2+2+3+2+3) = 2 + 7 + 14 = 23.

Найдите F(500).
