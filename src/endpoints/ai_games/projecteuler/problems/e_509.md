# Задача No 509
##  Последовательности Эрдёша-Тюрана

## Условия
Последовательность $s_1, s_2, s_3, \dots$ называется последовательностью Эрдёша-Тюрана, если для всех $i, j > 0$ существует целое число $k > 0$ такое, что $s_i + s_j = s_k$.

Например, последовательность 1, 2, 3, 4, 5, ... — последовательность Эрдёша-Тюрана, потому что для любых $s_i$ и $s_j$ мы можем выбрать $k = i + j$, что дает $s_i + s_j = s_{i+j}$.

Пусть $f(N)$ — количество последовательностей Эрдёша-Тюрана таких, что $s_i$ — целое число, $1 \le s_i \le N$ и $s_i < s_{i+1}$ для всех $i > 0$.

Известно, что $f(5) = 7$, последовательности:
*   {1, 2, 3, 4, 5}
*   {1, 2, 3, 4}
*   {1, 2, 3, 5}
*   {1, 2, 4, 5}
*   {1, 3, 4, 5}
*   {2, 3, 4, 5}
*   {3, 4, 5}

Также известно, что $f(10) = 178$.

Найдите $f(1000000)$ по модулю $10^9 + 7$.
