# Задача No 357
## Простое порождение делителей

## Условия

Рассмотрим число *n*. Если для каждого делителя *d* числа *n* число *d + n/d* является простым, то *n* называется простым порождающим делители числом.

Например, 30 является простым порождающим делители числом:
* Делители 30: 1, 2, 3, 5, 6, 10, 15, 30.
* 1 + 30/1 = 31, простое.
* 2 + 30/2 = 17, простое.
* 3 + 30/3 = 13, простое.
* 5 + 30/5 = 11, простое.
* 6 + 30/6 = 11, простое.
* 10 + 30/10 = 13, простое.
* 15 + 30/15 = 17, простое.
* 30 + 30/30 = 31, простое.

С другой стороны, 20 не является простым порождающим делители числом:
* Делители 20: 1, 2, 4, 5, 10, 20.
* 1 + 20/1 = 21, не простое.

Найдите сумму всех простых порождающих делители чисел, не превосходящих 100 000 000.
