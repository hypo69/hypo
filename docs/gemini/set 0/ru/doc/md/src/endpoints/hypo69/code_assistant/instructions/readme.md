# ИНСТРУКЦИЯ ПО ОБРАБОТКЕ PYTHON-ФУНКЦИЙ

## Обзор

Данный документ описывает инструкцию для обработки Python-файлов с целью создания документации в формате Markdown.  Документация включает в себя описание функций и классов, а также их параметров, возвращаемых значений и возможных исключений.

## Инструкции по созданию документации:

1. **Формат:** Используется стандарт Markdown (.md).  Каждый файл должен начинаться с заголовка и краткого описания.  Для функций и методов применяются Docstrings с указанием параметров, возвращаемых значений и возможных исключений, согласно примеру в инструкции. Замените `ex` на `e` в блоках обработки исключений.

2. **Структура:** Каждый файл должен содержать разделы "Оглавление", "Классы", "Функции", и т.д. Оглавление должно представлять собой ссылки на эти разделы для удобной навигации.

3. **Форматирование:** Используйте правильный Markdown синтаксис для заголовков, списков и ссылок.  Структурируйте описания функций и методов с использованием разделов "Описание", "Параметры", "Возвращает" и "Вызывает исключения".

4. **Заголовки:** Используйте заголовки первого (`#`), второго (`##`), третьего (`###`) и четвертого (`####`) уровней,  согласно примеру.

5. **Пример структуры документации:**

```markdown
# Модуль обработки данных

## Оглавление

- [Обзор](#обзор)
- [Функции](#функции)
  - [Функция 1](#функция-1)
  - [Функция 2](#функция-2)
- [Классы](#классы)
  - [Класс 1](#класс-1)
    - [Метод 1](#метод-1)

## Обзор

Краткое описание модуля обработки данных.

## Функции

### Функция 1

**Описание**:  Функция для обработки данных.

**Параметры**:
- `input_data` (list): Список входных данных.
- `param2` (int, optional): Вспомогательный параметр. По умолчанию 10.

**Возвращает**:
- `list`: Список обработанных данных.

**Вызывает исключения**:
- `ValueError`: Если `input_data` не является списком.

```

## Обработка входных данных

Последовательность обработки входных данных:

1. Проверяется тип входного файла.
2. Парсится код.
3. Создается структура документации.
4. Форматируется результат.

Модель пока не поддерживает обработку сложных структур кода, таких как вложенные циклы или рекурсивные функции.
