# Модуль hypotez/src/webdriver/bs/__init__.py

Этот модуль содержит инициализацию для подмодуля `bs`.  Он предоставляет константу `MODE` и импортирует класс `BS` из подмодуля `bs`.

## Константы

### `MODE`

**Описание**:  Переменная, хранящая режим работы (`dev`).

**Значение**:  `'dev'`

## Модули

### `bs`

**Описание**: Подмодуль, содержащий класс `BS`.

## Импорты

### `from .bs import BS`

**Описание**: Импортирует класс `BS` из подмодуля `bs`.
