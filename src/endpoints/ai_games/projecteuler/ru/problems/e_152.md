# Задача No 152
## Запись суммы в виде суммы обратных квадратов

## Условия
Рассмотрим сумму обратных квадратов натуральных чисел:
$$\frac{1}{1^2} + \frac{1}{2^2} + \frac{1}{3^2} + \dots = \frac{\pi^2}{6}$$
В частности, если взять только первые 4 члена этой суммы, то мы получим:
$$\frac{1}{1^2} + \frac{1}{2^2} + \frac{1}{3^2} + \frac{1}{4^2} = \frac{205}{144} = 1.42361\dots$$
Это число близко к $\frac{\pi^2}{6} = 1.64493\dots$

Теперь мы хотели бы найти несколько таких сумм из обратных квадратов *различных* чисел, которые в точности равны 1.
Один из простейших примеров:
$$\frac{1}{2^2} + \frac{1}{3^2} + \frac{1}{4^2} + \frac{1}{5^2} + \frac{1}{6^2} + \frac{1}{7^2} + \frac{1}{12^2} = 1$$
Интересно, что все знаменатели в этой сумме не превышают 12.

Сколько существует таких сумм из обратных квадратов различных натуральных чисел, которые в точности равны 1, и все знаменатели не превышают 80?
