# Модуль подготовки рекламных кампаний AliExpress

## Обзор

Модуль `prepare_all_camapaigns.py` предназначен для проверки и создания партнерских рекламных кампаний AliExpress. Если рекламная кампания не существует, она будет создана.

## Подробней

Этот модуль является частью системы работы с AliExpress в проекте `hypotez`. Он гарантирует, что для каждой необходимой рекламной кампании существует соответствующая партнерская кампания, автоматически создавая ее при необходимости. Модуль использует функциональность из `src.suppliers.aliexpress.campaign` для обработки всех кампаний.
