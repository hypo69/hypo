# Модуль facebook_groups_widgets.py

## Обзор

Модуль `facebook_groups_widgets.py` предоставляет класс `FacebookGroupsWidget` для создания и отображения выпадающего списка групп Facebook, загруженных из JSON-файла.  Этот виджет полезен для выбора группы при подаче объявления.

## Классы

### `FacebookGroupsWidget`

**Описание**: Класс `FacebookGroupsWidget` создает выпадающий список с URL групп Facebook, загруженных из предоставленного JSON-файла.

**Методы**:

#### `__init__`

**Описание**: Инициализирует объект `FacebookGroupsWidget`.

**Параметры**:

- `json_file_path` (Path): Путь к файлу JSON, содержащему информацию о группах Facebook.

**Возвращает**:

-  (None):  Метод не возвращает значение.

#### `create_dropdown`

**Описание**: Создает и возвращает виджет выпадающего списка `Dropdown` с URL групп.

**Параметры**:

- (None): Метод не принимает параметры.

**Возвращает**:

- `Dropdown`: Виджет выпадающего списка с URL групп Facebook.

#### `display_widget`

**Описание**: Отображает созданный виджет выпадающего списка на основе данных из JSON.

**Параметры**:

- (None): Метод не принимает параметры.

**Возвращает**:

- (None): Метод не возвращает значение.



## Переменные

### `MODE`

**Описание**: Переменная, хранящая режим работы (в данном случае, 'dev').


## Импорты

- `header`:  Предполагается, что импортируется другой модуль (возможно, для базовых функций или настроек).
- `IPython.display`:  Для отображения виджетов.
- `ipywidgets`:  Для работы с виджетами (Dropdown).
- `src.utils.jjson`:  Для загрузки данных из JSON-файла.
- `types`:  Для использования типа `SimpleNamespace`.
- `pathlib`: Для работы с путями к файлам.


##  Замечания

Этот модуль использует `IPython.display` и `ipywidgets` для создания интерактивного виджета.  Рекомендуется, чтобы окружение, в котором будет работать этот код, имело установленные эти библиотеки.
