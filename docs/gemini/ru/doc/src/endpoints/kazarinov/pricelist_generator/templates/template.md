# Шаблон HTML для генерации прайс-листа

## Обзор

Данный шаблон HTML (`template.html`) предназначен для генерации прайс-листа. Он позволяет отображать информацию о продуктах, включая заголовок, описание, спецификации, изображения и общую стоимость. Шаблон поддерживает адаптивную верстку для разных устройств.

## Структура шаблона

Шаблон состоит из нескольких основных секций:

### `<head>`

Содержит мета-теги, название страницы (`title`), и стили CSS для форматирования содержимого.

### `<body>`

Содержит основное содержимое страницы:

#### Заголовок (`<h1>`)

Отображает заголовок прайс-листа.

#### Описание (`<p class="lead">`)

Отображает краткое описание прайс-листа.

#### Блок `products` (`<div class="row">`)

Блок для отображения карточек с информацией о каждом продукте. Каждая карточка имеет структуру:

##### `<div class="product-card">`

*   Заголовок продукта (`<h3>`)
*   `div` для `product-content`

    *   Изображение продукта (`<img>`)
    *   `div` для `product-info`

        *   Описание продукта (`<p>`)
        *   Спецификации продукта (`<p>`)
        
##### Итерация по продуктам (`{% for product in products %}`)

Цикл, который обрабатывает каждый продукт из списка `products` и отображает его данные в карточке.

#### Подвал (`<div class="footer">`)

Содержит общую стоимость заказа с использованием стилизованного блока `price-tag`.

## Стиль (`<style>`)

CSS-стили определяют внешний вид страницы, включая:

*   Цвета фона и текста
*   Шрифт
*   Разметку элементов
*   Стиль для заголовков
*   Стиль для карточек с продуктами, включая изображение, описание и цены
*   Стиль для общей стоимости

## Переменные шаблона

Шаблон использует следующие динамические переменные:

*   `title`: Заголовок прайс-листа.
*   `description`: Описание прайс-листа.
*   `products`: Список словарей, каждый из которых представляет собой продукт с полями:
    *   `product_title`: Название продукта
    *   `image_local_saved_path`: Путь к локальному изображению продукта
    *   `product_description`: Описание продукта
    *   `specification`: Спецификации продукта
*   `price`: Общая стоимость всех продуктов.
*   `currency`: Валюта общей стоимости.


## Пример использования

```html
<!-- Пример использования шаблона -->
<!DOCTYPE html>
<html>
  <head>
    <!-- ... -->
  </head>
  <body>
    <!-- ... -->
    <div class="row">
    {% for product in products %}
    <div class="product-card">
        <h3>{{ product.product_title }}</h3>
        <div class="product-content">
            <img src="{{ product.image_local_saved_path }}" alt="{{ product.product_title }}" />
            <div class="product-info">
                <p>{{ product.product_description }}</p>
                <p>{{ product.specification }}</p>
            </div>
        </div>
    </div>
    {% endfor %}
    </div>
     <!-- ... -->
  </body>
</html>
```

В данном примере показано, как использовать переменные `product.product_title`, `product.image_local_saved_path`, `product.product_description` и `product.specification` для отображения данных продукта в карточке.

## Заключение

Шаблон `template.html` предоставляет структуру для отображения прайс-листа, включая адаптивность, стили и возможность интеграции с данными продуктов.
