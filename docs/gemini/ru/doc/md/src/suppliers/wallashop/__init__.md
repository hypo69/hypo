# Модуль hypotez/src/suppliers/wallashop/__init__.py

## Обзор

Данный модуль предоставляет базовый интерфейс и настройки для работы с поставщиком WallaShop.  Модуль содержит константу MODE и импортирует класс `Graber` из файла `graber.py`.

## Константы

### `MODE`

**Описание**:  Переменная, хранящая режим работы (например, 'dev', 'prod').

**Значение**: `'dev'`


## Классы

### `Graber`

**Описание**:  Класс для работы с API WallaShop. Подробная информация доступна в файле `graber.py`.
