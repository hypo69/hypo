# CHIEF

## סקירה כללית

**CHIEF** הוא משחק מתמטיקה שנועד לאמן בארבע פעולות (חיבור, חיסור, כפל, חילוק). המשחק כולל גם אלמנטים של כיף: כאשר השחקן מזין תשובה שגויה, התוכנית מסבירה כיצד היה צריך לפתור את הבעיה. המשחק מועיל לילדים ויכול לשמש מורים כדי לבדוק ידע בחשבון.

## הוראות שלב אחר שלב למימוש

### 1. אתחול המשחק

- המשחק מתחיל בכך שהמחשב (בתפקיד הצ'יף) שואל אותך שאלה:
  ```
  אני צ'יף המתמטיקה, האל ההודי הגדול של המתמטיקה.
  האם אתה מוכן לקבל את המבחן שלי?
  ```
- השחקן חייב לענות "YES" או "NO" כדי להמשיך או לסיים את המשחק.

### 2. לולאת המשחק הראשית

- **משימה:**
  1. הצ'יף נותן משימה אריתמטית מורכבת. לדוגמה:
     ```
     קח מספר, הוסף 3, חלק ב-5, כפל ב-8, חלק ב-5 והוסף את אותו המספר. החסר 1. מה התוצאה?
     ```
  2. השחקן מזין את תשובתו, והתוכנית בודקת אותה.

- **בדיקת תשובה:**
  1. אם התשובה שגויה, התוכנית מסבירה כיצד היה צריך לפתור את הבעיה:
     ```
     התשובה שלך הייתה שגויה! בואו נבדוק את השלבים:
     (1) 12 ועוד 3 = 15
     (2) חלקי 5 = 3
     (3) כפול 8 = 24
     (4) חלקי 5 = 4.8
     (5) הוסף 3 = 7.8
     (6) חיסור 1 = 6.8
     ```
  2. אם התשובה נכונה, התוכנית משבחת את השחקן וממשיכה במשחק:
     ```
     אתה צודק! זו התשובה הנכונה.
     ```

### 3. ספירת המנצח

- למשחק אין מנצח מסורתי, מכיוון שהמטרה היא תרגול מיומנויות אריתמטיות. המשחק נמשך עד שהשחקן פותר את המשימות או עד שהוא מחליט לצאת.

### 4. סיום המשחק

- לאחר השלמת כל המשימות או לפי החלטת השחקן, התוכנית תציע לשחק שוב:
  ```
  רוצה לשחק שוב? (YES/NO)
  ```

## דוגמה לפעולת התוכנית

1. **תחילת המשחק:**
   ```
   אני צ'יף המתמטיקה, האל ההודי הגדול של המתמטיקה.
   האם אתה מוכן לקבל את המבחן שלי?
   > YES
   ```

2. **משימה:**
   ```
   קח מספר, הוסף 3, חלק ב-5, כפל ב-8, חלק ב-5 והוסף את אותו המספר. החסר 1. מה התוצאה?
   > 6.8
   ```

3. **בדיקת תשובה:**
   ```
   התשובה שלך הייתה שגויה! בואו נבדוק את השלבים:
   (1) 12 ועוד 3 = 15
   (2) חלקי 5 = 3
   (3) כפול 8 = 24
   (4) חלקי 5 = 4.8
   (5) הוסף 3 = 7.8
   (6) חיסור 1 = 6.8
   אתה צודק! זו התשובה הנכונה.
   ```

4. **סיום המשחק:**
   ```
   רוצה לשחק שוב? (YES/NO)
   > NO
   תודה ששיחקת!
   ```

## מגבלות אפשריות

- התוכנית בודקת רק פעולות אריתמטיות, ואינה פותרת בעיות מתמטיות מורכבות.
- יש לשים לב היטב לכללים כדי למנוע טעות בחישובים.

## מימוש

המשחק מיושם באמצעות חישובים אריתמטיים בסיסיים ומאפשר לשחקן לקבל הסברים על כל שלב, אם הוא טעה.
