# Модуль `hypotez/src/suppliers/wallashop/header.py`

## Обзор

Этот модуль предоставляет функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`. Он также загружает README.md для использования в других частях проекта.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с текущего файла и двигаясь вверх по директориям, пока не найдет директорию, содержащую файлы-маркеры (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж с именами файлов или директорий, которые используются для определения корневой директории проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.


**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена. В противном случае возвращает директорию, в которой находится текущий скрипт.


**Вызывает исключения**:

- Нет


### `__root__`
**Описание**: Переменная, хранящая путь к корневой директории проекта, полученный путем вызова `set_project_root()`.

**Значение**:
- `Path`: Путь к корневой директории проекта.

## Загрузка настроек

**Описание**: Загружает настройки из файла `settings.json` в корневой директории проекта.

**Обработка ошибок**:

- Обрабатывает `FileNotFoundError`: Если файл `settings.json` не найден.
- Обрабатывает `json.JSONDecodeError`: Если файл `settings.json` поврежден или содержит некорректный JSON.


## Загрузка README.md

**Описание**: Загружает содержимое файла README.md в корневой директории проекта.

**Обработка ошибок**:

- Обрабатывает `FileNotFoundError`: Если файл `README.MD` не найден.
- Обрабатывает `json.JSONDecodeError`: В данном случае, это некорректно, так как читается строка, а не JSON.




## Переменные

### `__project_name__`

**Описание**: Получает имя проекта из `settings.json`, по умолчанию `hypotez`.

**Тип**: `str`

### `__version__`

**Описание**: Получает версию проекта из `settings.json`, по умолчанию пустая строка.

**Тип**: `str`

### `__doc__`

**Описание**: Получает содержимое README.md.

**Тип**: `str`

### `__details__`

**Описание**: Получает дополнительные данные из `settings.json`, по умолчанию пустая строка.

**Тип**: `str`


### `__author__`

**Описание**: Получает имя автора из `settings.json`, по умолчанию пустая строка.

**Тип**: `str`

### `__copyright__`

**Описание**: Получает авторские права из `settings.json`, по умолчанию пустая строка.

**Тип**: `str`

### `__cofee__`

**Описание**: Получает ссылку на помощь автору проекта, по умолчанию задана строка.

**Тип**: `str`
