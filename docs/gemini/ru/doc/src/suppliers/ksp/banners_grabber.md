# Модуль banners_grabber

## Обзор

Модуль `banners_grabber` отвечает за загрузку баннеров от поставщика KSP.  Этот модуль предоставляет функцию для получения данных баннеров.

## Оглавление

## Функции

### `get_banners`

**Описание**: Функция `get_banners` возвращает информацию о баннерах.

**Возвращает**:
- `bool`: Возвращает `True`, если данные успешно получены.

**Вызывает исключения**:
- Нет описанных исключений.
