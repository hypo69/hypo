# Модуль src.utils.string

## Обзор

Этот модуль содержит утилиты для работы со строками.

## Функции

### `string_to_int`

**Описание**: Преобразует строку в целое число.

**Параметры**:
- `input_string` (str): Входная строка.

**Возвращает**:
- `int`: Преобразованное целое число.
- `ValueError`: Возникает, если строка не может быть преобразована в целое число.
