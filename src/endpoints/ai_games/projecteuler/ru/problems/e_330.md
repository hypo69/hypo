# Задача No 330
## Перестановки с ограничениями
## Условия
Определим функцию f(n,k) следующим образом:
- f(n,k) = 0, если n < 0.
- f(0,k) = 1.
- f(n,k) = Σ f(n-i,k-1), где i пробегает все целые числа, которые делят n, включая 1 и n.

Например:
- f(3,2) = f(2,1) + f(1,1) + f(0,1) = (f(1,0) + f(0,0)) + (f(0,0)) + 1 = (0+1) + 1 + 1 = 3.
- f(3,3) = f(2,2) + f(1,2) + f(0,2) = (f(1,1) + f(0,1)) + (f(0,1)) + 1 = (f(0,0) + 1) + 1 + 1 = (1+1) + 1 + 1 = 4.

Известно, что f(10,10) = 8349.

Найдите f(10<sup>15</sup>, 15) mod 10<sup>9</sup> + 7.
