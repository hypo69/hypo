# Задача No 545
## Область сумм

## Условия
Пусть $S(n)$ - множество всех сумм целых чисел от 1 до $n$.
Например, $S(3) = \{1, 2, 3, 1+2, 1+3, 2+3, 1+2+3\} = \{1, 2, 3, 3, 4, 5, 6\} = \{1, 2, 3, 4, 5, 6\}$.

Обозначим $f(n)$ как наибольшее значение $k$, для которого все целые числа от 1 до $k$ содержатся в $S(n)$.
Таким образом, $f(3)=6$.

Пусть $A(n) = \sum f(i)$ при $1 \le i \le n$.
Например,
$S(1) = \{1\}$, $f(1) = 1$,
$S(2) = \{1, 2, 3\}$, $f(2) = 3$,
$S(3) = \{1, 2, 3, 4, 5, 6\}$, $f(3) = 6$,
$A(3) = 1 + 3 + 6 = 10$.

Известно, что $A(10) = 150$, $A(1000) = 340745$ и $A(10^6) = 333333833332$.

Найдите $A(10^{12})$.
