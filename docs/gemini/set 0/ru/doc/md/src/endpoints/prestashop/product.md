# Модуль hypotez/src/endpoints/prestashop/product.py

## Обзор

Данный модуль предоставляет инструменты для работы с товарами в системе PrestaShop через API. Он реализует класс `PrestaProduct`, который наследуется от класса `PrestaShop`, обеспечивая функции проверки наличия товара, расширенного поиска и получения информации по ID.

## Классы

### `PrestaProduct`

**Описание**: Класс `PrestaProduct` представляет собой объект для работы с товарами в PrestaShop. Он предоставляет методы для взаимодействия с API, выполняя операции проверки наличия товара, поиска и получения информации о товаре по ID.

**Методы**:

- `check(product_reference: str)`:
    **Описание**: Проверяет наличие товара в базе данных PrestaShop по указанному `product_reference` (SKU, MKT).
    **Параметры**:
        - `product_reference` (str): Строка, содержащая `product_reference` (SKU, MKT) товара.
    **Возвращает**:
        - dict: Словарь, содержащий информацию о товаре, если товар найден.
        - False: Если товар не найден.
    **Вызывает исключения**:
        - Возможные исключения, поднимаемые `PrestaShop`.

- `search(filter: str, value: str)`:
    **Описание**: Выполняет расширенный поиск в базе данных PrestaShop по указанному `filter` и `value`.
    **Параметры**:
        - `filter` (str): Параметр фильтра для поиска.
        - `value` (str): Значение для фильтра.
    **Возвращает**:
        - Возвращаемое значение API поиска, обычно список товаров.
    **Вызывает исключения**:
        - Возможные исключения, поднимаемые `PrestaShop`.

- `get(id_product)`:
    **Описание**: Возвращает информацию о товаре по указанному `id_product`.
    **Параметры**:
        - `id_product` (int): ID товара.
    **Возвращает**:
        - dict: Словарь с информацией о товаре, если товар найден.
        - Возможные ошибки API PrestaShop.
    **Вызывает исключения**:
        - Возможные исключения, поднимаемые `PrestaShop`.


### `__init__`

**Описание**: Конструктор класса `PrestaProduct`.

**Параметры**:
    - `credentials` (Optional[dict | SimpleNamespace], optional): Словарь или объект `SimpleNamespace` с параметрами `api_domain` и `api_key`. По умолчанию `None`.
    - `api_domain` (Optional[str], optional): Домен API. По умолчанию `None`.
    - `api_key` (Optional[str], optional): Ключ API. По умолчанию `None`.
    
**Возвращает**:
    - Нет возвращаемого значения.

**Вызывает исключения**:
    - `ValueError`: Если не указаны оба параметра `api_domain` и `api_key`, или если `credentials` не содержат необходимые данные.
