    -   התוכנית יוצרת שני לוחות משחק ריקים בגודל 10x10 עבור כל שחקן, כאשר כל תא יכול להיות ריק או להכיל חלק מספינה.

    -   השחקן מזין את קואורדינטות ההתחלה והסוף של כל ספינה, לדוגמה, `A1-A5`.
