# Модуль hypotez/src/ai/gemini/header.py

## Обзор

Этот модуль предоставляет функции для работы с конфигурацией проекта, получения корневого каталога проекта и доступа к документации. Он определяет переменные, содержащие имя проекта, версию, документацию, описание, автора, копирайт и ссылку на поддержку разработчика.

## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная с текущего файла и идя вверх по директориям, пока не найдет директорию, содержащую указанные файлы-маркеры (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж с именами файлов или директорий, используемых для определения корневого каталога проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневому каталогу проекта, если он найден, в противном случае - путь к каталогу текущего файла.

**Вызывает исключения**:

-  Не вызывает исключений.


## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта, полученный с помощью функции `set_project_root()`.

**Тип**: `Path`


### `config`

**Описание**: Словарь с настройками проекта, загруженный из файла `config.json`.

**Тип**: `dict`


### `doc_str`

**Описание**: Строка с документацией проекта, загруженная из файла `README.MD`.

**Тип**: `str`


### `__project_name__`

**Описание**: Имя проекта, взятое из `config.json`. По умолчанию 'hypotez'.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта, взятая из `config.json`. По умолчанию пустая строка.

**Тип**: `str`


### `__doc__`

**Описание**: Документация проекта, взятая из `README.MD`. По умолчанию пустая строка.

**Тип**: `str`


### `__details__`

**Описание**: Подробное описание проекта. По умолчанию пустая строка.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта, взятый из `config.json`. По умолчанию пустая строка.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права на проект, взятые из `config.json`. По умолчанию пустая строка.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка на возможность поддержать разработчика.


**Тип**: `str`
