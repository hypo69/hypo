Jupyter Widgets for AliExpress Campaign Editor
============================================

.. module:: hypotez.src.suppliers.aliexpress.campaign.ali_campaign_editor_jupyter_widgets
    :platform: Windows, Unix
    :synopsis: Jupyter widgets for the AliExpress campaign editor.

This module provides Jupyter widgets for managing AliExpress campaigns, enabling interaction
with campaign selection, category management, language/currency setting, initialization,
saving, product display, and Google Spreadsheet access.
