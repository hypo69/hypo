# Модуль `hypotez/src/suppliers/aliexpress/api/helpers/__init__.py`

## Обзор

Данный модуль содержит вспомогательные функции для работы с API AliExpress.  Он предоставляет инструменты для обработки запросов, получения и обработки данных о продуктах и категориях.

## Функции

### `api_request`

**Описание**: Функция для выполнения запросов к API AliExpress.

**Параметры**:
- `url` (str): URL запроса.
- `params` (Optional[dict], optional): Параметры запроса. По умолчанию `None`.
- `headers` (Optional[dict], optional): Заголовки запроса. По умолчанию `None`.


**Возвращает**:
- `dict | None`: Ответ от API в формате словаря или `None` в случае ошибки.


**Вызывает исключения**:
- `requests.exceptions.RequestException`: Возникает при проблемах с выполнением запроса.


### `get_list_as_string`

**Описание**: Функция для преобразования списка в строку, подходящую для использования в запросах API.

**Параметры**:
- `data` (list): Список значений.


**Возвращает**:
- `str`: Строковое представление списка, подходящее для использования в запросах API.


**Вызывает исключения**:
- `TypeError`: Возникает, если `data` не является списком.


### `get_product_ids`

**Описание**: Функция для извлечения списка идентификаторов продуктов.

**Параметры**:
- `data` (dict): Словарь данных, содержащий список продуктов.


**Возвращает**:
- `list | None`: Список идентификаторов продуктов или `None` в случае ошибки.


**Вызывает исключения**:
- `TypeError`: Возникает, если `data` не является словарем.


### `parse_products`

**Описание**: Функция для парсинга данных о продуктах, полученных от API.

**Параметры**:
- `data` (dict): Словарь данных о продуктах.


**Возвращает**:
- `list | None`: Список обработанных данных о продуктах или `None` в случае ошибки.


**Вызывает исключения**:
- `TypeError`: Возникает, если `data` не является словарем.


### `filter_parent_categories`

**Описание**: Функция для фильтрации родительских категорий.

**Параметры**:
- `data` (list): Список категорий.


**Возвращает**:
- `list | None`: Список родительских категорий или `None` в случае ошибки.


**Вызывает исключения**:
- `TypeError`: Возникает, если `data` не является списком.
