# Модуль hypotez/src/webdriver/header.py

## Обзор

Этот модуль содержит код для определения корневой директории проекта и загрузки настроек из файла `settings.json`. Он также обрабатывает возможные исключения, такие как `FileNotFoundError` и `json.JSONDecodeError`.  Модуль предоставляет переменные, содержащие информацию о проекте (название, версия, описание и др.) для использования другими частями приложения.


## Функции

### `set_project_root`

**Описание**: Функция находит корневую директорию проекта, начиная с директории текущего файла и идя вверх по дереву директорий.  Она останавливается на первой директории, содержащей один из файлов-маркеров (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для определения корневой директории проекта.


**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена, иначе - директория, где расположен скрипт.


**Вызывает исключения**:

- Не вызывает никаких исключений.


### Переменные

**Описание**: Переменные, определяющие параметры проекта, такие как имя, версия и т.д.

**Детали**:

* `__root__` (Path): Путь к корневой директории проекта.  Получается с помощью вызова функции `set_project_root()`.
* `settings` (dict): Словарь настроек проекта, загруженный из файла `settings.json`.
* `doc_str` (str): Содержимое файла `README.MD`.
* `__project_name__` (str): Название проекта, полученное из настроек или установленное по умолчанию ('hypotez').
* `__version__` (str): Версия проекта, полученная из настроек или по умолчанию (пустая строка).
* `__doc__` (str): Описание проекта, полученное из файла `README.MD` или по умолчанию (пустая строка).
* `__details__` (str): Дополнительные детали о проекте (по умолчанию пустая строка).
* `__author__` (str): Автор проекта, полученное из настроек или по умолчанию (пустая строка).
* `__copyright__` (str): Авторские права, полученное из настроек или по умолчанию (пустая строка).
* `__cofee__` (str): Ссылка для пожертвований разработчику (по умолчанию ссылка).


## Обработка исключений

**Описание**:  Код содержит обработку исключений `FileNotFoundError` и `json.JSONDecodeError` для безопасного доступа к файлам настроек и документации.

**Детали**: При попытке открыть файл настроек или файла документации, если файл не найден или содержимое файла некорректно (не валидный JSON), то переменные `settings` и `doc_str` соответственно остаются `None` или пустой строкой.
