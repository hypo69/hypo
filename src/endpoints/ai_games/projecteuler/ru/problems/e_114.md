# Задача No 114
## Комбинации размещений красных блоков I
## Условия
Черный блок шириной в одну единицу и красный блок шириной, как минимум, в три единицы, располагаются в ряд из n единиц, так, чтобы ни один из красных блоков не соприкасался друг с другом.
Обозначим за f(n) число способов, которыми можно разместить эти блоки.
Например, при n = 8, есть следующие 17 вариантов:

```
# # # # # # # #
R # # # # # # #
# R # # # # # #
# # R # # # # #
# # # R # # # #
# # # # R # # #
# # # # # R # #
# # # # # # R #
R # R # # # # #
R # # R # # # #
R # # # R # # #
R # # # # R # #
# R # R # # # #
# R # # R # # #
# # R # R # # #
R # R # R # # #
R # # R # R # #
```

Сколько существует способов заполнить строку из n=50 единиц?
