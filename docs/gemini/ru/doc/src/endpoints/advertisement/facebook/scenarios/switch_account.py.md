# Модуль `switch_account`

## Обзор

Модуль `switch_account.py` предназначен для переключения между аккаунтами в Facebook. Он использует Selenium WebDriver для взаимодействия с веб-интерфейсом и JSON-файл с локаторами элементов для навигации.

### `switch_account`

**Описание**: Функция для переключения между аккаунтами Facebook. Находит кнопку "Переключить" и кликает по ней.

- `None`: Функция не возвращает явного значения.
