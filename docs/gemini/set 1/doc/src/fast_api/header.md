# Модуль `hypotez/src/fast_api/header.py`

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`. Он также предоставляет доступ к информации о проекте (название, версия, описание, автор и т.д.).

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для идентификации корневой директории проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена, иначе директория, в которой находится скрипт.

**Вызывает исключения**:

- Нет.


### Загрузка настроек

**Описание**: Загружает настройки из файла `settings.json`. Если файл не найден или содержит некорректные данные, переменная `settings` сохранит значение `None`.

**Вызывает исключения**:

- `FileNotFoundError`: Если файл `settings.json` не найден.
- `json.JSONDecodeError`: Если содержимое файла `settings.json` не является валидным JSON.


### Загрузка документации

**Описание**: Загружает содержимое файла `README.MD` в переменную `doc_str`.  Если файл не найден или содержит некорректные данные, переменная `doc_str` сохранит значение `None`.

**Вызывает исключения**:

- `FileNotFoundError`: Если файл `README.MD` не найден.
- `json.JSONDecodeError`: Если содержимое файла не является валидным JSON.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта.

**Тип**: `Path`


### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженные из файла `settings.json`.  Может быть `None`, если файл не найден или поврежден.

**Тип**: `dict` или `None`


### `doc_str`

**Описание**: Содержимое файла `README.MD`, содержащее описание проекта. Может быть `None`, если файл не найден или поврежден.

**Тип**: `str` или `None`


### `__project_name__`

**Описание**: Название проекта. Получается из `settings`, если доступен, иначе имеет значение `'hypotez'`.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта. Получается из `settings`, если доступен, иначе имеет значение `''`.

**Тип**: `str`


### `__doc__`

**Описание**: Документация к проекту. Получается из `doc_str`, если доступен, иначе имеет значение `''`.

**Тип**: `str`


### `__details__`

**Описание**: Подробная информация о проекте. По умолчанию пустая строка.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта. Получается из `settings`, если доступен, иначе имеет значение `''`.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права на проект. Получается из `settings`, если доступен, иначе имеет значение `''`.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка для поддержки разработчика. Получается из `settings`, если доступен, иначе имеет значение `"Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69"`.

**Тип**: `str`
