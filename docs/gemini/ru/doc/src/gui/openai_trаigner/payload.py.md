# Документация модуля `payload.py`

## Обзор

Файл `payload.py` предназначен для определения основных параметров и констант, используемых в модуле `openai_trainer`. В настоящее время файл содержит определение режима работы, устанавливая его в значение `dev`.

## Константы

### `MODE`
**Описание**: Константа `MODE` определяет режим работы приложения.
**Значение**:
- `dev`: Режим разработки.
