# Модуль `header`

## Обзор

Модуль `header` предназначен для определения корневой директории проекта, загрузки настроек из файла `settings.json`, чтения документации из `README.MD` и установки основных метаданных проекта.

## Функции

### `set_project_root`

**Описание**:
Находит корневую директорию проекта, начиная с директории текущего файла, ища вверх по дереву каталогов до первого каталога, содержащего один из указанных файлов-маркеров.

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или директорий для идентификации корневой директории проекта. По умолчанию (`'__root__'`, `'.git'`).

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена, иначе директория, где расположен скрипт.

## Глобальные переменные

### `__root__`

**Описание**:
Путь к корневой директории проекта.

### `settings`

**Описание**:
Словарь, содержащий настройки проекта, загруженные из файла `settings.json`.

### `doc_str`

**Описание**:
Строка, содержащая текст документации, считанный из файла `README.MD`.

### `__project_name__`

**Описание**:
Имя проекта, взятое из `settings.json` или "hypotez" по умолчанию.

### `__version__`

**Описание**:
Версия проекта, взятая из `settings.json` или пустая строка по умолчанию.

### `__doc__`

**Описание**:
Строка с документацией проекта, считанная из `README.MD`, или пустая строка по умолчанию.

### `__details__`

**Описание**:
Строка с дополнительной информацией о проекте.

### `__author__`

**Описание**:
Автор проекта, взятый из `settings.json` или пустая строка по умолчанию.

### `__copyright__`

**Описание**:
Информация об авторских правах, взятая из `settings.json` или пустая строка по умолчанию.

### `__cofee__`

**Описание**:
Строка с ссылкой на возможность угостить разработчика кофе, взятая из `settings.json` или ссылка на Boosty по умолчанию.
