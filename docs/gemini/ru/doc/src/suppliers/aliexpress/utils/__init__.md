# Модуль hypotez/src/suppliers/aliexpress/utils

## Обзор

Этот модуль предоставляет вспомогательные функции для работы с данными AliExpress. Он содержит функции для извлечения идентификаторов продуктов, перевода URL-адресов в HTTPS и работы с локалями.

## Функции

### `extract_prod_ids`

**Описание**: Функция извлекает идентификаторы продуктов из текста.

**Параметры**:
- `text` (str): Текст, из которого необходимо извлечь идентификаторы продуктов.

**Возвращает**:
- `list[str]`: Список извлеченных идентификаторов продуктов.

**Вызывает исключения**:
- `ValueError`: Если входной текст не является строкой.
- `TypeError`: Если в тексте не найдены корректные идентификаторы продуктов.


### `ensure_https`

**Описание**: Функция преобразует URL-адрес в HTTPS, если это необходимо.

**Параметры**:
- `url` (str): URL-адрес, который необходимо проверить.

**Возвращает**:
- `str`: URL-адрес в формате HTTPS.


**Вызывает исключения**:
- `TypeError`: Если входной параметр `url` не является строкой.
- `ValueError`: Если входной URL-адрес не содержит схемы (например, http:// или https://).


### `locales`

**Описание**: Функция, предоставляющая доступ к спискам локалей.

**Параметры**:
-  Не принимает параметры.

**Возвращает**:
- `dict`: Словарь, содержащий списки локалей.


**Вызывает исключения**:
-  `TypeError`: Если внутренние структуры данных не являются ожидаемыми типами.
