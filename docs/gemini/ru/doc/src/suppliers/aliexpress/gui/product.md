# Модуль `hypotez/src/suppliers/aliexpress/gui/product.py`

## Обзор

Этот модуль предоставляет класс `ProductEditor`, предназначенный для редактирования данных продуктов AliExpress. Он использует PyQt6 для создания графического интерфейса пользователя (GUI) и взаимодействует с классом `AliCampaignEditor` для подготовки данных продукта. Модуль позволяет загружать JSON-файлы с данными продукта, отображать их и запускать асинхронную подготовку продукта.


## Классы

### `ProductEditor`

**Описание**: Класс `ProductEditor` представляет собой виджет PyQt6, предназначенный для редактирования данных продуктов AliExpress. Он загружает JSON-данные, отображает заголовок и подробные сведения о продукте и позволяет запускать асинхронную подготовку.

**Атрибуты**:

- `data: SimpleNamespace`: Загруженные данные продукта из JSON-файла.
- `language: str`: Язык продукта (по умолчанию "EN").
- `currency: str`: Валюта продукта (по умолчанию "USD").
- `file_path: str`: Путь к загруженному JSON-файлу.
- `editor: AliCampaignEditor`: Экземпляр класса `AliCampaignEditor`, используемый для подготовки данных продукта.
- `main_app`: ссылка на приложение, содержащее ProductEditor.


**Методы**:

#### `__init__(self, parent=None, main_app=None)`

**Описание**: Инициализирует виджет `ProductEditor`.

**Параметры**:

- `parent`: Родительский виджет.
- `main_app`: ссылка на приложение, содержащее ProductEditor.


#### `setup_ui(self)`

**Описание**: Настраивает пользовательский интерфейс (UI). Создает кнопки "Открыть JSON-файл", "Подготовить продукт" и метку для отображения пути к файлу.

#### `setup_connections(self)`

**Описание**: Настраивает связи между элементами UI (сигналами и слотами).

#### `open_file(self)`

**Описание**: Открывает диалоговое окно выбора файла для загрузки JSON-данных.

**Возвращает**:
 - Если файл не выбран, ничего не возвращает.


#### `load_file(self, file_path)`

**Описание**: Загружает JSON-данные из указанного файла.

**Параметры**:
- `file_path` (str): Путь к файлу.

**Возможные исключения**:
 - `Exception`: При возникновении ошибок при загрузке JSON-данных.


#### `create_widgets(self, data)`

**Описание**: Создает виджеты на основе загруженных данных из JSON-файла.

**Параметры**:
- `data` (SimpleNamespace): данные из файла.


#### `prepare_product_async(self)`

**Описание**: Асинхронно подготавливает продукт, используя `AliCampaignEditor`.

**Возможные исключения**:
 - `Exception`: При возникновении ошибок при подготовке продукта.
