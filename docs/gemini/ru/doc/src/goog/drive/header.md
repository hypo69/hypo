# Модуль hypotez/src/goog/drive/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`.  Он также предоставляет доступ к информации о проекте (имя, версия, описание и др.) и ссылке на подписку для поддержки разработчика.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная от текущей директории и двигаясь вверх по дереву директорий.

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, которые используются для определения корневой директории проекта. По умолчанию содержит `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращает директорию, где находится скрипт.

**Пример использования**:

```python
root_path = set_project_root()
print(root_path)
```


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта.  Инициализируется функцией `set_project_root()`.

**Тип**: `Path`

**Значение**: Путь к корневой директории проекта.

### `settings`

**Описание**: Словарь настроек проекта, загруженный из файла `settings.json`.

**Тип**: `dict` или `None`


### `doc_str`

**Описание**: Строка документации проекта, загруженная из файла `README.MD`.

**Тип**: `str` или `None`


### `__project_name__`

**Описание**: Имя проекта. Получается из настроек, либо имеет значение по умолчанию.

**Тип**: `str`

**Значение**: Значение, полученное из настроек или значение по умолчанию (`hypotez`).

### `__version__`

**Описание**: Версия проекта. Получается из настроек, либо имеет значение по умолчанию.

**Тип**: `str`

**Значение**: Значение, полученное из настроек или пустая строка.


### `__doc__`

**Описание**: Документация проекта. Получается из файла `README.MD`, либо имеет значение по умолчанию.

**Тип**: `str`

**Значение**: Значение, полученное из `doc_str` или пустая строка.


### `__details__`

**Описание**: Дополнительные детали о проекте.

**Тип**: `str`

**Значение**: Пустая строка.


### `__author__`

**Описание**: Автор проекта. Получается из настроек, либо имеет значение по умолчанию.

**Тип**: `str`

**Значение**: Значение, полученное из настроек или пустая строка.


### `__copyright__`

**Описание**: Авторские права на проект. Получается из настроек, либо имеет значение по умолчанию.

**Тип**: `str`

**Значение**: Значение, полученное из настроек или пустая строка.

### `__cofee__`

**Описание**: Ссылка для поддержки разработчика (например, ссылка на Boosty). Получается из настроек, либо имеет значение по умолчанию.

**Тип**: `str`

**Значение**: Значение, полученное из настроек или строка по умолчанию.


**Обработка исключений**:

Функции `set_project_root`, `load_settings`, `load_doc`,  используют блоки `try...excepts`, чтобы обработать потенциальные ошибки `FileNotFoundError` и `json.JSONDecodeError`.  Это предотвращает сбой программы при отсутствии необходимых файлов или некорректном формате данных.
