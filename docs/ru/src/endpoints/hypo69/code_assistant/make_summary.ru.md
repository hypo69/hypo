# Модуль `make_summary.py`

## Обзор

Модуль `make_summary.py` предназначен для автоматического создания файла `SUMMARY.md`, используемого для компиляции документации с помощью инструментов, таких как `mdbook`. Он рекурсивно обходит указанную директорию с исходными файлами `.md` и генерирует оглавление, фильтруя файлы в зависимости от указанного языка.

## Подробней

Этот модуль автоматизирует процесс создания оглавления для документации, что упрощает поддержку и обновление структуры документации проекта. Пути строятся относительно корня проекта, что обеспечивает устойчивость к изменениям в структуре директорий.

**Как работает функция**:

1.  **Обработка аргументов командной строки**: Функция анализирует аргументы командной строки для определения языка фильтрации и пути к директории с `.md` файлами.
2.  **Поиск файлов**: Функция рекурсивно обходит указанную директорию в поисках `.md` файлов, принимая во внимание указанный язык фильтрации.
3.  **Создание оглавления**: Функция создает файл `SUMMARY.md` в директории `docs` и записывает в него оглавление, содержащее ссылки на найденные `.md` файлы.

Внутри функции происходят следующие действия и преобразования:

*   **Аргументы командной строки** – Анализируются аргументы командной строки, переданные при запуске скрипта.
*   **Получение списка файлов** – Рекурсивно обходится указанная директория для поиска `.md` файлов.
*   **Создание файла SUMMARY.md** – Создается файл `SUMMARY.md` и заполняется оглавлением.
*   **Запись оглавления в файл** – Записывается сформированное оглавление в файл `SUMMARY.md`.

**Примеры**:

*   Запуск скрипта для создания `SUMMARY.md` с фильтрацией по русскому языку:

    ```bash
    python src/endpoints/hypo69/code_assistant/make_summary.py -lang ru src
    ```

*   Запуск скрипта для создания `SUMMARY.md` с фильтрацией по английскому языку:

    ```bash
    python src/endpoints/hypo69/code_assistant/make_summary.py -lang en src
    ```
