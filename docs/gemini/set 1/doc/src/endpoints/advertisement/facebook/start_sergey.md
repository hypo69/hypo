# Модуль `hypotez/src/endpoints/advertisement/facebook/start_sergey.py`

## Обзор

Этот модуль отвечает за запуск рекламных кампаний на Facebook для разных рекламодателей. Он предоставляет функции для управления кампаниями, включая выбор групп, языков и валют. Модуль использует драйвер для взаимодействия с веб-приложением Facebook.  Также реализован циклический запуск кампаний с определённой периодичностью.

## Функции

### `run_campaign`

**Описание**: Функция запускает рекламную кампанию.

**Параметры**:

- `d` (Driver): Экземпляр драйвера веб-драйвера. Обязан поддерживать методы для взаимодействия с сайтом Facebook.
- `promoter_name` (str): Имя рекламодателя.  
- `campaigns` (list): Список названий кампаний, которые нужно запустить.
- `group_file_paths` (list): Список путей к файлам с данными о целевых группах.
- `language` (str): Язык рекламной кампании.
- `currency` (str): Валюта рекламной кампании.


**Возвращает**:

-  `None`: Функция не возвращает значения.

**Вызывает исключения**:

- Любые исключения, которые могут быть подняты методами класса `FacebookPromoter`.


### `campaign_cycle`

**Описание**: Функция для управления циклом запуска кампаний.

**Параметры**:

- `d` (Driver): Экземпляр драйвера.

**Возвращает**:

- `bool`: `True`, если цикл завершен без ошибок.

**Вызывает исключения**:

- Любые исключения, которые могут быть подняты внутри цикла, включая исключения, генерируемые функцией `run_campaign`.


### `main`

**Описание**: Основная функция для запуска рекламных кампаний.

**Параметры**:

- Нет

**Возвращает**:

- `None`: Функция не возвращает значения.


**Вызывает исключения**:

- `KeyboardInterrupt`: Если пользователь прерывает выполнение программы.
- Любые исключения, которые могут быть подняты функцией `campaign_cycle`, включая исключения, генерируемые `Driver`.


## Переменные

### `MODE`

**Описание**:  Переменная, определяющая режим работы скрипта. В текущей реализации она хранит строку 'dev'.

### `group_file_paths_ru`, `adv_file_paths_ru`, `group_file_paths_he`, `adv_file_paths_he`

**Описание**:  Список путей к файлам с данными о группах и рекламных объявлениях для разных языков (русский и иврит).

### `group_categories_to_adv`

**Описание**:  Список категорий групп, в которые необходимо направить рекламу.

### `language_currency_pairs`

**Описание**: Список словарей, связывающих языки с валютами, используемых для циклического запуска кампаний.


## Замечания

*   Модуль использует классы и функции из других модулей (например, `FacebookPromoter`, `Driver`, `Chrome`, `logger`).  Необходимо убедиться, что эти модули доступны.
*   Функция `campaign_cycle` включает в себя многократный запуск `run_campaign` для различных кампаний и языков.
*   В коде присутствует механизм задержки (`time.sleep`) для предотвращения перегрузки серверов Facebook.
*   Используется обработка исключений `KeyboardInterrupt` для корректного завершения работы скрипта.
*   В коде присутствует много условий, проверяющих язык и валюту.  Важно правильно определить соответствия между именами кампаний и файлами конфигурации.
*   Логирование используется через `logger`.
*   Использование `copy.copy` при копировании списков.

*   Предполагается, что файлы с данными о группах и рекламных объявлениях (например, `sergey_pages.json` и `ru_ils.json`) содержат необходимую информацию в формате, понятном классу `FacebookPromoter`.
*   `interval()` является неявной функцией, её определение не представлено в предоставленном коде. Необходимо понимать её логику, чтобы оценить работоспособность кода.
