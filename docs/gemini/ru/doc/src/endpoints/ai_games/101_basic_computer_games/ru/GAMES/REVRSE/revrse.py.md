# REVRSE

## Обзор

Игра "REVRSE" предлагает игроку угадать последовательность из 4-х чисел, цифры которых расположены в определенном порядке, при этом числа могут повторяться. После каждого хода игрока, компьютер выводит количество правильно угаданных цифр и количество цифр, которые присутствуют в загаданной последовательности, но не находятся на своих позициях. Игра продолжается, пока игрок не угадает последовательность цифр, или пока не закончится число попыток.

## Функции

### `generate_target_sequence`

**Описание**: Функция для генерации случайной последовательности из 4 цифр от 1 до 6.

**Параметры**:
- Нет

**Возвращает**:
- `list[int]`: Список из 4 случайных целых чисел от 1 до 6.

### `compare_sequences`

**Описание**: Функция для сравнения двух последовательностей и подсчета прямых и косвенных попаданий.

**Параметры**:
- `target` (list[int]): Загаданная последовательность чисел.
- `user` (list[int]): Последовательность, введенная пользователем.

**Возвращает**:
- `tuple[int, int]`: Кортеж, содержащий количество прямых и косвенных попаданий.

### `play_reverse_game`

**Описание**: Основная функция, реализующая логику игры REVRSE.

**Параметры**:
- Нет

**Возвращает**:
- `None`: Функция ничего не возвращает, а выводит результаты игры в консоль.

**Вызывает исключения**:
- `ValueError`: Если ввод пользователя не является целым числом или не соответствует формату.
