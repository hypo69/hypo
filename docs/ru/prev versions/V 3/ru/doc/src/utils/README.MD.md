# Документация для модуля `Tiny Utils`

## Обзор

`Tiny Utils` - это библиотека утилит, предоставляющая набор легковесных вспомогательных функций для различных общих задач. Эта библиотека включает утилиты для преобразования форматов данных, манипулирования текстом и файлами, строковых операций, форматирования даты и времени, обработки изображений и многого другого. Она организована в несколько модулей для облегчения доступа к определенным функциям.

## Подробней

Этот файл `README.MD` предоставляет общее описание библиотеки `Tiny Utils`, включая информацию об установке, обзоре модулей, примерах использования и лицензии. Он служит отправной точкой для понимания возможностей и использования библиотеки. Этот файл содержит не код, а документацию к библиотеке.

### Module Descriptions

Подробное описание каждого модуля библиотеки.

#### Convertors

Модуль `convertors` содержит утилиты для преобразования данных между форматами.

##### Files:

- **text2png.py**: Преобразует текст в изображение PNG.
- **tts.py**: Преобразует текст в речь и сохраняет как аудиофайл.
- **webp2png.py**: Преобразует изображения из формата WebP в формат PNG.
- **xls.py**: Обрабатывает преобразования и манипуляции с файлами XLS.
- **xml2dict.py**: Преобразует XML-данные в словарь Python.
- **base64.py**: Кодирует или декодирует данные, используя кодировку Base64.
- **csv.py**: Предоставляет инструменты для разбора и манипулирования CSV.
- **dict.py**: Утилиты для работы со словарями Python.
- **html.py**: Преобразует HTML-контент в различные форматы.
- **json.py**: Утилиты для разбора и манипулирования JSON.
- **md2dict.py**: Преобразует контент Markdown в словарь.
- **ns.py**: Специализированные утилиты для преобразования пространств имен.

#### String Utilities

Модуль `string` включает расширенные функции для манипулирования строками.

#### File Operations

Модуль `file.py` включает утилиты для обработки файлов.

#### Date-Time Utilities

Модуль `date_time.py` предоставляет различные утилиты для работы с датой и временем.

#### FTP Utilities

Модуль `ftp.py` включает функции для обработки операций FTP.

#### Image Utilities

Модуль `image.py` предоставляет базовые инструменты для обработки изображений.

#### PDF Utilities

Модуль `pdf.py` предлагает утилиты для работы с PDF.

#### Printer Utilities

Модуль `printer.py` включает функции для отправки файлов или отформатированных данных на принтер.
