# Модуль `hypotez/src/suppliers/hb/category.py`

## Обзор

Модуль `hypotez/src/suppliers/hb/category.py` отвечает за сбор информации о товарах из категорий на сайте поставщика hb.co.il. Он использует веб-драйвер для взаимодействия с сайтом и реализует методы для получения списков категорий и товаров в рамках этих категорий.

## Функции

### `get_list_products_in_category`

**Описание**: Возвращает список URL-адресов страниц товаров, находящихся в заданной категории.  Функция обрабатывает возможность наличия пагинации на странице категории.

**Параметры**:

- `s` (Supplier): Экземпляр класса `Supplier`, содержащий информацию о поставщике, веб-драйвере и локаторах.

**Возвращает**:

- list[str]: Список URL-адресов страниц товаров или `None`, если список пуст или произошла ошибка.

**Вызывает исключения**:

- Возможны исключения, связанные с взаимодействием с веб-драйвером (например, `NoSuchElementException`).


### `paginator`

**Описание**: Обрабатывает навигацию по страницам пагинации.

**Параметры**:

- `d` (Driver): Экземпляр класса `Driver` для управления веб-драйвером.
- `locator` (dict): Словарь локаторов для элементов пагинации.
- `list_products_in_category` (list): Список URL-адресов товаров (может быть изменён).

**Возвращает**:

- bool: `True`, если пагинация есть и следующая страница была успешно загружена; `False`, если пагинации нет или возникла ошибка.

**Вызывает исключения**:

- Возможны исключения, связанные с взаимодействием с веб-драйвером (например, `NoSuchElementException`).


### `get_list_categories_from_site`

**Описание**: Сбор списка категорий с сайта поставщика.
