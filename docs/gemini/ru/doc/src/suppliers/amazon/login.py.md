# Модуль `login` для Amazon

## Обзор

Модуль `login.py` предназначен для автоматизации процесса авторизации на сайте Amazon с использованием веб-драйвера. Он содержит функцию `login`, которая принимает объект поставщика (`Supplier`) в качестве аргумента и выполняет шаги, необходимые для входа в систему.

## Подробней

Этот модуль играет важную роль в проекте `hypotez`, автоматизируя вход в аккаунт Amazon для выполнения дальнейших задач, таких как сбор данных о товарах, отслеживание цен и другие операции, требующие авторизации. Функция `login` использует локаторы элементов страницы, хранящиеся в объекте поставщика, для взаимодействия с веб-страницей Amazon. В случае неуспешной авторизации, в лог записывается сообщение об ошибке.
