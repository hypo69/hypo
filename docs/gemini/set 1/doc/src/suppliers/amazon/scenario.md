# Модуль `hypotez/src/suppliers/amazon/scenario.py`

## Обзор

Этот модуль содержит сценарий обработки категорий товаров для поставщика Amazon. Он отвечает за сбор информации о категориях и товарах, а также за взаимодействие с веб-драйвером для получения данных.

## Локальные переменные

### `MODE`

**Описание**: Переменная, хранящая режим работы (например, 'dev', 'prod').

## Функции

### `get_list_products_in_category`

**Описание**: Функция собирает список ссылок на страницы товаров на странице категории.

**Параметры**:

- `s` (Supplier): Экземпляр класса `Supplier`, содержащий информацию о поставщике и веб-драйвере.

**Возвращает**:

- `list[str,str,None]`: Список ссылок на страницы товаров или `None`, если список не был получен или локаторы не найдены.

**Обрабатываемые исключения**:

- Нет описания обработки исключений.


**Подробное описание**:

Функция `get_list_products_in_category` принимает объект `Supplier` и извлекает список ссылок на страницы товаров со страницы категории.  В случае отсутствия локаторов или списка товаров, она возвращает `None`.
