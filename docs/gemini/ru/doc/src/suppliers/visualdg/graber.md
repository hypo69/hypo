# Модуль `hypotez/src/suppliers/visualdg/graber.py`

## Обзор

Модуль `hypotez/src/suppliers/visualdg/graber.py` содержит класс `Graber`, предназначенный для сбора данных с веб-страниц товара на сайте `visualdg.co.il`. Класс наследуется от `Graber` из родительского модуля и предоставляет методы для обработки данных с веб-страницы.  Реализована возможность добавления предварительной обработки перед запросом к веб-драйверу через декоратор `close_pop_up`.

## Классы

### `Graber`

**Описание**: Класс `Graber` отвечает за сбор информации с веб-страницы товара на сайте `visualdg.co.il`.  Предоставляет возможность перегрузки функций обработки отдельных полей, если стандартная обработка не подходит.


**Методы**:

- `__init__`: Инициализирует экземпляр класса.

```python
    def __init__(self, driver: Driver):
        """Инициализация класса сбора полей товара.

        Args:
            driver (Driver): Экземпляр класса драйвера.
        """
        self.supplier_prefix = 'visualdg'
        super().__init__(supplier_prefix=self.supplier_prefix, driver=driver)
        # Устанавливаем глобальные настройки через Context
        Context.locator_for_decorator = None  # <- если будет уастановлено значение - то оно выполнится в декораторе `@close_pop_up`

## Функции (внешние)

Обратите внимание, что функции `close_pop_up` и `decorator` являются заглушками и не реализованы в этом коде. Их реализации находятся в родительском классе.

### `close_pop_up`

**Описание**:  Заглушка декоратора для закрытия всплывающих окон перед выполнением основной логики функции.

**Параметры**:
- `value (Any)`: Дополнительные данные для декоратора. По умолчанию `None`.

**Возвращает**:
- `Callable`: Декоратор, оборачивающий функцию.


### `decorator`

**Описание**: Внутренняя функция декоратора.


**Параметры**:
- `func (Callable)`: Функция, которая должна быть обернута декоратором.

**Возвращает**:
- `Callable`: Обёрнутая функция.
