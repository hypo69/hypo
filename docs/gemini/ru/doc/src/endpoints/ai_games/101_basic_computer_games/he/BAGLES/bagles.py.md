# BAGLES

## Обзор

Этот модуль реализует игру "Бейглс", в которой компьютер генерирует секретное число, а игрок пытается его угадать, получая подсказки в виде "FERMI", "PICO" или "BAGELS".

## Функции

### `generate_secret_number`

**Описание**:
Создает случайное секретное число с уникальными цифрами.

**Параметры**:
- `num_digits` (int): Количество цифр в секретном числе.

**Возвращает**:
- `str`: Секретное число в виде строки.

### `get_clues`

**Описание**:
Генерирует подсказки для введенного пользователем числа на основе секретного числа.

**Параметры**:
- `user_guess` (str): Введенное пользователем число.
- `secret_number` (str): Секретное число.

**Возвращает**:
- `str`: Подсказки в виде строки ("FERMI", "PICO" или "BAGELS").

### `play_bagels_game`

**Описание**:
Запускает игру "Бейглс".

**Параметры**:
- Нет

**Возвращает**:
- `None`
