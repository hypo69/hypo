# АНАЛИЗ ИНСТРУКЦИИ ПО ИГРЕ "BOXING"

## Инструкция по реализации игры "BOXING" (Олимпийский бокс)

### Описание
Симуляция олимпийского бокса, состоящая из трёх раундов. Игрок тренирует своего боксера, выбирая его сильнейший удар и уязвимость. Компьютер управляет вторым боксером. Игроки поочередно выполняют удары и защиты. Цель — победить в двух из трёх раундов, нанося удары или блокируя атаки противника.

---

### Пошаговая инструкция для реализации

#### 1. Инициализация игры
   -   В начале игры игрок выбирает имя своего бойца и его сильнейший удар.
   -   Игрок задаёт уязвимость бойца (слабое место на теле, куда противник может нанести удар).
   -   На экране отображается информация о бойце игрока и его противнике.

#### 2. Основной цикл игры
   -   **Раунд 1, 2 и 3:**
     1.  Каждый раунд состоит из нескольких этапов. Игрок и компьютер поочередно наносят удары.
     2.  Удары могут быть: хук, апперкот, джеб и т. д.
     3.  Каждому удару присваивается вероятность попадания и нанесения урона в зависимости от силы удара и уязвимости противника.
     4.  Игрок может использовать блокировку или уклонение от удара.
     5.  В конце раунда отображается информация о здоровье обоих бойцов.

#### 3. Подсчёт победителя
   -   После каждого раунда вычисляется количество нанесённых и полученных ударов.
   -   Победитель определяется на основе очков, набранных по завершению двух раундов из трёх.
   -   При ничьей раунд повторяется.

#### 4. Завершение игры
   -   После завершения игры выводится итоговый счёт и победитель:
     ```
     Поздравляем! Ваш боец победил!
     ```
   -   Предложение сыграть снова:
     ```
     Хотите сыграть снова? (да/нет)
     ```

---

### Пример работы программы

1.  **Начало игры:**
   ```
   Добро пожаловать в Олимпийский бокс!
   Введите имя вашего бойца: Джон
   Введите лучший удар вашего бойца (1 — хук, 2 — апперкот, 3 — джеб): 2
   Введите уязвимость вашего бойца (1 — слабая голова, 2 — корпус, 3 — слабые ноги): 1
   Ваш боец — Джон, его лучший удар — апперкот, уязвимость — голова.
   Противник: Супермен.
   ```

2.  **Раунд 1:**
   ```
   Раунд 1 начинается!
   Ваш ход. Введите тип удара (1 — хук, 2 — апперкот, 3 — джеб): 2
   Вы нанесли апперкот! Супермен блокирует!
   Супермен наносит хук. Вы блокируете!
   ```

3.  **Завершение игры:**
   ```
   Раунд 3. Супермен атакует! Удар в голову! Джон теряет здоровье.
   Поздравляем! Супермен победил!
   Хотите сыграть снова? (да/нет)
   > нет
   До свидания!
   ```
