# Модуль hypotez/src/suppliers/morlevi/__init__.py

## Обзор

Этот модуль предоставляет базовые настройки и импорты для поставщика данных Morlevi.  Он содержит константу `MODE` и импортирует класс `Graber`.

## Константы

### `MODE`

**Описание**:  Переменная, хранящая режим работы (например, "dev", "prod").

**Значение**: `"dev"` по умолчанию.


## Классы

### `Graber`

**Описание**: Класс `Graber` предназначен для работы с источником данных Morlevi. Подробности реализации находятся в файле `morlevi/graber.py`.
