# Документация для `manifest.json`

## Обзор

Файл `manifest.json` описывает метаданные для расширения Chrome "Try xpath". Он включает в себя информацию о версии, имени, описании, иконках, разрешениях, действиях, фоновом скрипте и контентных скриптах, необходимых для работы расширения.

## Структура файла

### `manifest_version`

**Описание**: Указывает версию манифеста.
- **Тип**: `number`
- **Значение**: `3`

### `name`

**Описание**: Название расширения.
- **Тип**: `string`
- **Значение**: `"Try xpath"`

### `description`

**Описание**: Описание расширения.
- **Тип**: `string`
- **Значение**: `"This extension displays the result of evaluating xpath expression or CSS selector."`

### `version`

**Описание**: Версия расширения.
- **Тип**: `string`
- **Значение**: `"1.3.5"`

### `icons`

**Описание**: Иконки расширения.
- **Тип**: `object`
- **Содержимое**: 
  - `"48"`: Путь к иконке размером 48x48 пикселей, `"icons/icon_48.png"`

### `permissions`

**Описание**: Разрешения, необходимые для работы расширения.
- **Тип**: `array`
- **Содержимое**:
  - `"<all_urls>"`: Разрешение на доступ ко всем URL-адресам.
  - `"storage"`: Разрешение на использование API storage.

### `action`

**Описание**: Настройки действия расширения (отображается на панели инструментов).
- **Тип**: `object`
- **Содержимое**:
    - `"default_icon"`: Путь к иконке действия, `"icons/icon_48.png"`.
    - `"default_title"`: Заголовок действия, `"Try xpath"`.
    - `"default_popup"`: Путь к HTML-файлу всплывающего окна действия, `"popup/popup.html"`.

### `background`

**Описание**: Настройки фонового скрипта расширения.
- **Тип**: `object`
- **Содержимое**:
    - `"service_worker"`: Путь к фоновому JavaScript-файлу, `"background/try_xpath_background.js"`.

### `content_scripts`

**Описание**: Настройки контентных скриптов расширения.
- **Тип**: `array`
- **Содержимое**:
  - **Объект настроек контентного скрипта**:
     - `"matches"`: Массив шаблонов URL-адресов, к которым будет применен контентный скрипт, `["<all_urls>"]`.
     - `"js"`: Массив путей к JavaScript-файлам контентного скрипта, `["scripts/try_xpath_functions.js"]`.
