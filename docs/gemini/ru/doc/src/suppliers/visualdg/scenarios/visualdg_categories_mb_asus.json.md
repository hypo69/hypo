## Обзор
Данный JSON-файл содержит сценарии для материнских плат ASUS, используемые в проекте `hypotez`. Файл определяет параметры и категории для различных моделей материнских плат ASUS, включая их URL-адреса, активность, состояние и связанные категории PrestaShop.

## Структура JSON

Файл состоит из одного основного объекта, содержащего поле `scenarios`, которое является словарем. Ключами этого словаря являются названия сценариев (модели материнских плат), а значениями — словари с параметрами каждого сценария.
