# Модуль post_message_async

## Обзор

Данный модуль предоставляет функции для публикации сообщений в Facebook, используя Selenium WebDriver.  Модуль включает функции для заполнения заголовка и описания поста, загрузки медиа-файлов (изображений и видео) и публикации готового поста. Модуль поддерживает асинхронную загрузку и обработку изображений, а также настройку языка для отображения текста.

## Оглавление

- [post_title](#функция-post-title)
- [upload_media](#функция-upload-media)
- [update_images_captions](#функция-update-images-captions)
- [promote_post](#функция-promote-post)

## Функции

### `post_title`

**Описание**: Отправляет заголовок и описание кампании в поле публикации поста.

**Параметры**:
- `d` (Driver): Экземпляр драйвера, используемый для взаимодействия с веб-страницей.
- `category` (SimpleNamespace): Объект, содержащий заголовок и описание, которые необходимо отправить.

**Возвращает**:
- `bool`: `True`, если заголовок и описание были успешно отправлены, иначе `None`.

**Примеры**:
```python
driver = Driver(...)
category = SimpleNamespace(title="Campaign Title", description="Campaign Description")
post_title(driver, category)
```

**Обработка исключений**:
- Возможные ошибки обработки `logger.error()`, такие как проблемы со скроллингом или открытием поля ввода.


### `upload_media`

**Описание**: Загружает медиа-файлы (изображения и видео) в раздел изображений и обновляет подписи.

**Параметры**:
- `d` (Driver): Экземпляр драйвера, используемый для взаимодействия с веб-страницей.
- `products` (List[SimpleNamespace]): Список продуктов, содержащий пути к медиа-файлам.
- `no_video` (bool, optional): Признак, что не нужно загружать видео, по умолчанию `False`.

**Возвращает**:
- `bool`: `True`, если медиа-файлы были загружены успешно, иначе `None`.

**Вызывает исключения**:
- `Exception`: Возникает при ошибках во время загрузки медиа или обновления подписей.

**Примеры**:
```python
driver = Driver(...)
products = [SimpleNamespace(local_saved_image='path/to/image.jpg', ...)]
await upload_media(driver, products)
```


### `update_images_captions`

**Описание**: Асинхронно добавляет описания к загруженным медиа-файлам.

**Параметры**:
- `d` (Driver): Экземпляр драйвера, используемый для взаимодействия с веб-страницей.
- `products` (List[SimpleNamespace]): Список продуктов с данными для обновления.
- `textarea_list` (List[WebElement]): Список областей ввода, куда добавляются подписи.


**Вызывает исключения**:
- `Exception`: Возникает при ошибках в процессе обновления подписей медиа.

**Обработка исключений**:
- Логирование ошибок при генерации сообщения и отправке его в текстовое поле.


### `promote_post`

**Описание**: Управляет процессом продвижения поста с заголовком, описанием и медиа-файлами.

**Параметры**:
- `d` (Driver): Экземпляр драйвера, используемый для взаимодействия с веб-страницей.
- `category` (SimpleNamespace): Подробные данные категории, используемые для заголовка и описания поста.
- `products` (List[SimpleNamespace]): Список продуктов, содержащих медиа и детали, которые необходимо опубликовать.
- `no_video` (bool, optional): Признак, что не нужно загружать видео, по умолчанию `False`.

**Возвращает**:
- `bool`: `True`, если пост был успешно опубликован, иначе `None`.

**Примеры**:
```python
driver = Driver(...)
category = SimpleNamespace(title="Campaign Title", description="Campaign Description")
products = [SimpleNamespace(local_saved_image='path/to/image.jpg', ...)]
await promote_post(driver, category, products)
```
