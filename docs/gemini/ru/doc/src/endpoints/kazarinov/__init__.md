# Модуль hypotez/src/endpoints/kazarinov/__init__.py

## Обзор

Этот модуль инициализирует и предоставляет доступ к классам, связанным с ботом KazarinovTelegramBot.

## Постоянные значения

### `MODE`

**Описание:** Строковая переменная, определяющая режим работы приложения. В данном случае, значение `'dev'`.

## Импорты

### `from .kazarinov_bot import KazarinovTelegramBot`

**Описание:** Импортирует класс `KazarinovTelegramBot` из файла `kazarinov_bot.py` в текущем каталоге.
