# Модуль `facebook_groups_widgets`

## Обзор

Модуль `facebook_groups_widgets.py` предназначен для создания и отображения выпадающего списка с URL групп Facebook, используя данные из JSON-файла. Этот виджет используется для выбора групп при подаче объявлений.

## Классы

### `FacebookGroupsWidget`

**Описание**: Класс для создания выпадающего списка с URL групп Facebook из JSON-файла.

**Методы**:

- `__init__`: Инициализирует виджет, загружая данные из JSON и создавая выпадающий список.
- `create_dropdown`: Создает и возвращает виджет выпадающего списка на основе данных групп.
- `display_widget`: Отображает виджет выпадающего списка.

#### `__init__`

**Описание**: Инициализация виджета с выпадающим списком для групп Facebook.

**Параметры**:
- `json_file_path` (Path): Путь к JSON-файлу, содержащему информацию о группах Facebook.

#### `create_dropdown`

**Описание**: Создает и возвращает виджет выпадающего списка на основе данных групп.

**Возвращает**:
- `Dropdown`: Виджет выпадающего списка с URL групп Facebook.

#### `display_widget`

**Описание**: Отображает виджет выпадающего списка.

## Функции
В данном модуле нет отдельных функций, вся логика реализована в методах класса `FacebookGroupsWidget`.
