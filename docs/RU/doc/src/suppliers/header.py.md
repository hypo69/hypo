# Модуль `src.suppliers.header`

## Обзор

Модуль `src.suppliers.header` предназначен для определения корневой директории проекта, загрузки настроек из `settings.json` и документации из `README.MD`, а также для хранения метаданных проекта, таких как имя, версия, автор и т.д.

## Функции

### `set_project_root`

**Описание**:
Находит корневую директорию проекта, начиная с директории текущего файла, и поднимаясь вверх, останавливаясь на первой директории, содержащей любой из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple, optional): Имена файлов или директорий для идентификации корня проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена, иначе директория, где расположен скрипт.

## Переменные модуля

### `__root__`
**Описание**:
Путь к корневой директории проекта. Определяется с помощью функции `set_project_root`.

### `settings`
**Описание**:
Словарь с настройками проекта, загруженный из файла `settings.json`. Изначально имеет значение `None`.

### `doc_str`
**Описание**:
Строка, содержащая документацию проекта, загруженную из файла `README.MD`. Изначально имеет значение `None`.

### `__project_name__`
**Описание**:
Имя проекта. По умолчанию `hypotez`. Значение загружается из `settings.json` или устанавливается по умолчанию.

### `__version__`
**Описание**:
Версия проекта. Значение загружается из `settings.json` или устанавливается пустой строкой по умолчанию.

### `__doc__`
**Описание**:
Документация проекта. Значение загружается из `README.MD` или устанавливается пустой строкой по умолчанию.

### `__details__`
**Описание**:
Дополнительные сведения о проекте. В данный момент установлено пустой строкой.

### `__author__`
**Описание**:
Автор проекта. Значение загружается из `settings.json` или устанавливается пустой строкой по умолчанию.

### `__copyright__`
**Описание**:
Информация об авторских правах проекта. Значение загружается из `settings.json` или устанавливается пустой строкой по умолчанию.

### `__cofee__`
**Описание**:
Сообщение о возможности угостить разработчика кофе. Значение загружается из `settings.json` или устанавливается значением по умолчанию: "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69".
