# Слоты

## Обзор

Этот модуль реализует простую азартную игру "Слоты", имитирующую игровой автомат с тремя барабанами. На каждом барабане случайным образом выпадает один из четырех символов: вишня (C), слива (P), колокольчик (B) или звездочка (*). Если на всех трех барабанах выпадают одинаковые символы, игрок выигрывает, в противном случае - проигрывает.

## Функции

### `get_symbol`

**Описание**:
Преобразует число в соответствующий символ для игрового автомата.

**Параметры**:
- `number` (int): Число от 1 до 4, представляющее барабан.

**Возвращает**:
- `str`: Символ, соответствующий числу ("C", "P", "B", "*").

### `play_slots`

**Описание**:
Имитирует игровой автомат.
Генерирует случайные символы на трех барабанах и определяет, выиграл ли игрок.

**Параметры**:
Отсутствуют.

**Возвращает**:
- `None`: Функция ничего не возвращает, а выводит результаты игры в консоль.
