# Модуль hypotez/src/endpoints/kazarinov/react/header.py

## Обзор

Данный модуль содержит функции для работы с настройками проекта, включая поиск корневой директории, чтение настроек из файла `settings.json` и документации из `README.MD`.  Он инициализирует переменные, содержащие информацию о проекте, такие как имя, версия, описание, автор и т.д.,  получая их из загруженных настроек.

## Функции

### `set_project_root`

**Описание**:  Находит корневую директорию проекта, начиная с текущей директории и итеративно поднимаясь по дереву директорий.  Останавливается на первой директории, содержащей один из указанных маркеров файлов (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, по которым определяется корневая директория проекта.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена, иначе — путь к директории, где находится текущий скрипт.


**Пример использования**:

```python
project_root = set_project_root()
print(project_root)
```

### Чтение настроек из `settings.json`


**Описание**:  Читает настройки из файла `settings.json` в корневой директории проекта и сохраняет их в переменной `settings`.

**Возможные исключения**:

- `FileNotFoundError`: Возникает, если файл `settings.json` не найден.
- `json.JSONDecodeError`: Возникает, если содержимое файла `settings.json` не является корректным JSON-объектом.

### Чтение документации из `README.MD`


**Описание**: Читает содержимое файла `README.MD` в корневой директории проекта и сохраняет его в переменной `doc_str`.

**Возможные исключения**:

- `FileNotFoundError`: Возникает, если файл `README.MD` не найден.
- `json.JSONDecodeError`: Возникает, если содержимое файла `README.MD` содержит синтаксические ошибки.

## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученной функцией `set_project_root()`.


### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженные из `settings.json`. Значение `None`, если файл не найден или некорректен.


### `doc_str`

**Описание**: Строка, содержащая содержимое файла `README.MD` проекта. Значение `None`, если файл не найден или некорректен.


### `__project_name__`

**Описание**: Имя проекта, полученное из настроек (`settings.json`). По умолчанию "hypotez".


### `__version__`

**Описание**: Версия проекта, полученная из настроек (`settings.json`). По умолчанию пустая строка.


### `__doc__`

**Описание**: Документация к проекту, полученная из `README.MD`. По умолчанию пустая строка.


### `__details__`

**Описание**: Дополнительные детали о проекте. По умолчанию пустая строка.


### `__author__`

**Описание**: Автор проекта, полученный из настроек (`settings.json`). По умолчанию пустая строка.


### `__copyright__`

**Описание**: Авторские права на проект, полученные из настроек (`settings.json`). По умолчанию пустая строка.


### `__cofee__`

**Описание**: Ссылка на платформа для поддержки разработчика.  По умолчанию ссылка на Boosty.
