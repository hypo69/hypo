# Шаблон чата Gemini

## Обзор

Этот шаблон HTML (`chat.html`) предоставляет интерфейс для взаимодействия с моделью Gemini.  Он использует Bootstrap для стилизации и jQuery для асинхронной обработки запросов. Шаблон отображает историю чата, поле ввода пользователя и кнопку отправки.  При нажатии кнопки, пользовательский ввод отправляется на сервер для обработки моделью Gemini, а полученный ответ отображается в окне чата.

## Файл

Файл `chat.html` представляет собой шаблон HTML для веб-страницы чата.  Он включает в себя заголовок, основную часть (содержащую чат-лог и форму ввода), а также скрипты JavaScript для обработки взаимодействия с пользователем.


## Стили

Этот шаблон использует Bootstrap для стилизации элементов.  Дополнительно можно подключить пользовательские CSS файлы, как показано в коде.


## Элементы

### `<h1>`

Заголовок страницы "Kazarinov AI Chat".

### `.chat-box`

Диви, содержащий чат-лог.  Используется `overflow-y: scroll;` для автоматического прокрутки при заполнении.

### `#chat-log`

ID контейнера для отображения сообщений (как пользователя, так и AI).

### `#user-input`

Поле ввода для ввода сообщений пользователем.

### `#chat-form`

Форма, содержащая поле ввода и кнопку отправки.  Обработка отправки выполняется с помощью jQuery.

### Кнопка "Отправить"

Кнопка для отправки сообщения.

## JavaScript

### Обработчик событий `submit` формы

Обработчик событий обрабатывает отправку формы:

* Получает ввод пользователя из поля `#user-input`.
* Добавляет сообщение пользователя в чат-лог.
* Отправляет асинхронный POST-запрос на сервер `/ask` с данными пользователя.
* После получения ответа от сервера добавляет ответ AI в чат-лог.
* Очищает поле ввода.
