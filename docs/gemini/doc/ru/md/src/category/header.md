# Модуль hypotez/src/category/header.py

## Обзор

Модуль `hypotez/src/category/header.py` определяет корневой путь к проекту `hypotez`. Все импорты строятся относительно этого пути.  В дальнейшем планируется перенести логику определения корневого пути в системные переменные.

## Функции

### `set_project_root`

**Описание**: Ищет корневую директорию проекта, начиная от текущего файла. Поиск идёт вверх по директориям до тех пор, пока не будет найдена директория, содержащая один из файлов-маркеров (по умолчанию `pyproject.toml`, `requirements.txt`, `.git`). Если корневая директория найдена, она добавляется в `sys.path`.

**Параметры**:

- `marker_files` (tuple, опционально): Список файлов или директорий, по которым определяется корневая директория проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена; иначе - путь к директории, в которой находится данный скрипт.

**Вызывает исключения**:

- Нет


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный функцией `set_project_root()`.

**Тип**: `Path`


### `settings`

**Описание**: Словарь со значениями настроек из файла `settings.json`.

**Тип**: `dict` или `None`


### `doc_str`

**Описание**: Строковое содержимое файла `README.MD`.

**Тип**: `str` или `None`


### `__project_name__`

**Описание**: Название проекта. Получено из файла `settings.json`, по умолчанию `'hypotez'`.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта. Получено из файла `settings.json`, по умолчанию пустая строка.

**Тип**: `str`


### `__doc__`

**Описание**: Документация проекта. Получено из файла `README.MD`, по умолчанию пустая строка.

**Тип**: `str`


### `__details__`

**Описание**: Подробная информация о проекте. По умолчанию пустая строка.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта. Получено из файла `settings.json`, по умолчанию пустая строка.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права. Получено из файла `settings.json`, по умолчанию пустая строка.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка для пожертвования разработчику. Получено из файла `settings.json`, по умолчанию ссылка на Boosty.

**Тип**: `str`


## Обработка исключений

Обработка `FileNotFoundError` и `json.JSONDecodeError` при чтении файлов `settings.json` и `README.MD`. Если файлы не найдены или некорректно отформатированы, переменные `settings`, `doc_str` и соответствующие поля проекта инициализируются значением `None` или пустой строкой.
