# Конфигурация Chrome WebDriver

## Обзор

Этот файл содержит конфигурационные параметры для Chrome WebDriver, включая настройки запуска браузера, пути к исполняемым файлам и профилям, а также заголовки запросов.

## Опции

Здесь определены опции запуска браузера Chrome.

- **`log-level`**: Уровень логирования. Установлен в "5".
- **`disable-dev-shm-usage`**: Отключает использование `/dev/shm`.
- **`remote-debugging-port`**: Порт для удаленной отладки. Установлен в "0".
- **`arguments`**: Список аргументов командной строки для запуска Chrome. Включает "--kiosk" и "--disable-gpu".

## Отключенные опции

Здесь перечислены отключенные опции запуска.

- **`headless`**: Опция headless (запуск без графического интерфейса) отключена.

## Директории профилей

Здесь определены пути к директориям профилей Chrome для разных сценариев.

- **`os`**: Путь к профилю пользователя Chrome для операционной системы.
    - `%LOCALAPPDATA%\\\\Google\\\\Chrome\\\\User Data`
- **`internal`**: Внутренний путь к профилю по умолчанию.
    - `webdriver\\\\chrome\\\\profiles\\\\default`
- **`testing`**: Путь к профилю для тестирования.
   - `%LOCALAPPDATA%\\\\Google\\\\Chrome for Testing\\\\User Data`

## Расположение бинарных файлов

Здесь определены пути к исполняемым файлам Chrome и ChromeDriver.

- **`os`**: Путь к исполняемому файлу Chrome для операционной системы.
    - `C:\\\\Program Files\\\\Google\\\\Chrome\\\\Application\\\\chrome.exe`
- **`exe`**: Путь к исполняемому файлу ChromeDriver.
     - `bin\\\\webdrivers\\\\chrome\\\\125.0.6422.14\\\\chromedriver.exe`
- **`binary`**: Путь к исполняемому файлу Chrome.
    - `bin\\\\webdrivers\\\\chrome\\\\125.0.6422.14\\\\win64-125.0.6422.14\\\\chrome-win64\\\\chrome.exe`
- **`chromium`**: Путь к исполняемому файлу Chromium.
    - `bin\\\\webdrivers\\\\chromium\\\\chrome-win\\\\chrome.exe`

## Заголовки

Здесь определены заголовки HTTP запросов, которые будут использоваться при взаимодействии с веб-сайтами.

- **`User-Agent`**: Значение заголовка User-Agent.
- **`Accept`**: Значение заголовка Accept.
- **`Accept-Charset`**: Значение заголовка Accept-Charset.
- **`Accept-Encoding`**: Значение заголовка Accept-Encoding.
- **`Accept-Language`**: Значение заголовка Accept-Language.
- **`Connection`**: Значение заголовка Connection.

## Прокси

Здесь указано, включен ли прокси.

- **`proxy_enabled`**: Указывает, включен ли прокси. Установлено в `false`.
