# Модуль hypotez/src/goog/quickstart.py

## Обзор

Данный модуль предоставляет функции для взаимодействия с Google Apps Script API. Он позволяет создавать новые проекты скриптов, загружать файлы (например, код JavaScript и манифесты) в эти проекты и выводить URL проекта. Модуль использует аутентификацию, хранение токенов доступа и обработку потенциальных ошибок API.

## Функции

### `main`

**Описание**: Функция `main` является точкой входа программы. Она выполняет аутентификацию, создает новый проект Google Apps Script, загружает файлы кода в проект и выводит URL проекта.

**Параметры**:

Нет параметров.

**Возвращает**:

Нет возвращаемых значений.

**Вызывает исключения**:

- `googleapiclient.errors.HttpError`: Возникает, если API Google Apps Script возвращает ошибку. Содержимое ошибки будет выведено в консоль.


## Переменные

### `MODE`

**Описание**:  Переменная `MODE` содержит строковое значение, предположительно, определяющее режим работы (например, 'dev', 'prod').


### `SCOPES`

**Описание**: Список необходимых областей доступа к Google Apps Script API.

**Значение**: `['https://www.googleapis.com/auth/script.projects']`

### `SAMPLE_CODE`

**Описание**: Пример кода JavaScript.

**Значение**:  Текст JavaScript кода (функции `helloWorld`).

### `SAMPLE_MANIFEST`

**Описание**: Пример манифеста Google Apps Script (JSON).

**Значение**:  Текст JSON-манифеста.

### `token_path`

**Описание**: Путь к файлу `token.json`, содержащему данные авторизации.

**Значение**:  `gs.path.tmp / 'e-cat-346312-137284f4419e.json'`

## Модули

### `header`

**Описание**:  Предполагается, что этот модуль содержит вспомогательные функции или константы.

### `gs`

**Описание**:  Предполагается, что этот модуль содержит функции для работы с файловой системой (например, `gs.path`).


## Зависимости

- `google.auth.transport.requests`
- `google.oauth2.credentials`
- `google_auth_oauthlib.flow`
- `googleapiclient`
- `googleapiclient.discovery`
- `pathlib`

## Особенности

- Использует `InstalledAppFlow` для аутентификации.
- Хранит токены авторизации в файле `token.json`.
- Обрабатывает ошибки, возвращаемые API.
- Подключает обработку исключений `HttpError`, предоставляя более информативную обратную связь.
