# Модуль `src.webdriver.edge.extentions.version`

## Обзор

Модуль `src.webdriver.edge.extentions.version` предназначен для хранения информации о версии, имени, документации и другой дополнительной информации о модуле или пакете. Модуль также содержит информацию об авторе и деталях версии.

## Переменные модуля

### `__name__`

**Описание**:
Содержит имя модуля. Если скрипт запускается напрямую, значение будет `"__main__"`.

### `__version__`

**Описание**:
Содержит версию модуля или пакета.

**Тип**:
`str`

**Значение**:
`"3.12.0.0.0.4"`

### `__doc__`

**Описание**:
Документационная строка модуля.

**Тип**:
`str`

### `__details__`

**Описание**:
Содержит дополнительные сведения о модуле, в данном случае, детали версии.

**Тип**:
`str`

**Значение**:
`"Details about version for module or class"`

### `__annotations__`

**Описание**:
Содержит аннотации типов для переменных и функций в модуле.

### `__author__`

**Описание**:
Содержит имя автора(ов) модуля.

**Тип**:
`str`

**Значение**:
`'hypotez '`
