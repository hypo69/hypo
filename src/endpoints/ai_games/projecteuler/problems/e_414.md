# Задача No 414
## Совершенные степенные числа
## Условия

Определение: Совершенное степенное число — это положительное целое число, которое может быть записано в виде целой степени другого положительного целого числа, большего 1.

Например, 256 — совершенное степенное число, поскольку оно равно 2⁸ или 4⁴ или 16².

Первые несколько совершенных степенных чисел: 4, 8, 9, 16, 25, 27, 32, 36, 49, 64, 81, 100, 121, 125, ...

Мы также можем определить последовательность {aₙ} следующим образом:
* a₁ = 1
* Для n > 1, aₙ = наименьшее целое число > aₙ₋₁, такое, что сумма aₙ + aₙ₋₁ является совершенным степенным числом.

Первые несколько членов этой последовательности: 1, 3, 5, 28, 30, 44, 105, 107, 158, 160, 207, 377, 379, 672, 674, 1065, 1067, ...

Пусть S(n) обозначает сумму первых n членов последовательности {aₙ}.
Таким образом, S(10) = 1148 и S(100) = 1110176.

Найдите S(10⁴).
