## Функции

### `start_supplier`

```python
def start_supplier(supplier_prefix: str = 'aliexpress', locale: str = 'en') -> Supplier:
    """ Старт поставщика """
```

**Как работает функция**:
Функция `start_supplier` предназначена для запуска поставщика с заданными параметрами. Она принимает префикс поставщика (`supplier_prefix`) и локаль (`locale`) в качестве аргументов, формирует словарь параметров и возвращает экземпляр класса `Supplier`, инициализированный с этими параметрами.

**Параметры**:
- `supplier_prefix` (str, optional): Префикс поставщика. По умолчанию `'aliexpress'`.
- `locale` (str, optional): Локаль. По умолчанию `'en'`.

**Возвращает**:
- `Supplier`: Экземпляр класса `Supplier`, инициализированный с переданными параметрами.

**Примеры**:

```python
supplier = start_supplier(supplier_prefix='bangood', locale='ru')
# Возвращает экземпляр класса Supplier для Banggood с русской локалью.
```
```python
supplier = start_supplier()
# Возвращает экземпляр класса Supplier для AliExpress с английской локалью (значения по умолчанию).
```
```python
supplier = start_supplier(locale='de')
# Возвращает экземпляр класса Supplier для AliExpress с немецкой локалью.
