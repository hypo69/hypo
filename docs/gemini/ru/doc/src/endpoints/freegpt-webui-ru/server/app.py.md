# Документация для `app.py`

## Обзор

Файл `app.py` инициализирует Flask-приложение, которое будет использоваться для веб-интерфейса FreeGPT на русском языке. В данном файле создается экземпляр Flask с указанием папки для HTML-шаблонов.

## Подробней

Данный код является отправной точкой для веб-приложения FreeGPT на русском языке. Он создает экземпляр Flask, который будет обрабатывать HTTP-запросы и возвращать HTML-страницы. Расположение файла `hypotez/src/endpoints/freegpt-webui-ru/server/app.py` говорит о том, что это серверная часть веб-интерфейса. Указание `template_folder='./../client/html'` сообщает Flask, где искать HTML-шаблоны для отображения веб-страниц.

## Переменные

### `app`

**Описание**: Экземпляр Flask-приложения.

**Принцип работы**: Создается с использованием `Flask(__name__, template_folder='./../client/html')`. Параметр `__name__` указывает на имя текущего модуля, а `template_folder` задает путь к папке с HTML-шаблонами, относительно текущего файла.

**Примеры**:

```python
from flask import Flask

app = Flask(__name__, template_folder='./../client/html')
```
