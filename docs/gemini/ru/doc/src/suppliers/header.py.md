# Модуль `header`

## Обзор

Модуль `header.py` предназначен для определения и установки основных параметров проекта, таких как корневой каталог, имя проекта, версия, автор и т.д. Он также загружает настройки проекта из файла `settings.json` и документацию из `README.MD`.

## Функции

### `set_project_root`

**Описание**:
    Находит корневой каталог проекта, начиная с директории текущего файла и поднимаясь вверх, останавливаясь на первом каталоге, содержащем любой из маркерных файлов.

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или каталогов, которые идентифицируют корень проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневому каталогу, если найден, иначе путь к каталогу, где расположен скрипт.

## Переменные

### `MODE`

**Описание**:
    Режим работы приложения. По умолчанию установлен в `'dev'`.

### `__root__`

**Описание**:
    Путь к корневому каталогу проекта. Определяется функцией `set_project_root`.

### `settings`

**Описание**:
    Словарь настроек проекта, загруженный из файла `settings.json`. Если файл не найден или имеет неверный формат, значение будет `None`.

### `doc_str`

**Описание**:
    Строка документации проекта, загруженная из файла `README.MD`. Если файл не найден или имеет неверный формат, значение будет `None`.

### `__project_name__`

**Описание**:
    Имя проекта, загруженное из настроек или по умолчанию `'hypotez'`.

### `__version__`

**Описание**:
    Версия проекта, загруженная из настроек или пустая строка `''`.

### `__doc__`

**Описание**:
    Описание проекта, загруженное из файла `README.MD`, или пустая строка `''`, если файл не найден.

### `__details__`

**Описание**:
    Детальная информация о проекте. Изначально установлена как пустая строка `''`.

### `__author__`

**Описание**:
    Автор проекта, загруженный из настроек или пустая строка `''`.

### `__copyright__`

**Описание**:
    Авторские права проекта, загруженные из настроек или пустая строка `''`.

### `__cofee__`

**Описание**:
    Сообщение с предложением угостить разработчика кофе, загруженное из настроек или строка по умолчанию: "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69".
