# Голосовой помощник ChatGPT - Telegram

## Обзор

Модуль реализует голосового помощника ChatGPT, интегрированного с Telegram.  Он позволяет пользователям общаться с ChatGPT через Telegram-бот.

## Функции


### `process_message`

**Описание**: Обрабатывает входящие сообщения из Telegram.

**Параметры**:
- `update` (object): Объект, содержащий данные о входящем сообщении из Telegram API.

**Возвращает**:
- `None`: Функция не возвращает значения.  Возможные действия: отправка ответа в Telegram или обработка ошибок.

**Вызывает исключения**:
- `Exception`: Обрабатываются общие исключения.

## Классы

### `ChatGPTBot`

**Описание**: Класс, представляющий Telegram бота, интегрированного с ChatGPT.

**Методы**:
- `__init__(self, api_key: str, token: str)`: Инициализирует бота с API ключом ChatGPT и токеном Telegram бота.
    - **Параметры**:
        - `api_key` (str): API ключ для доступа к ChatGPT.
        - `token` (str): Токен Telegram бота.
    - **Возвращает**:
        - `None`
- `handle_update(self, update: object)`: Обрабатывает входящие обновления (сообщения) от Telegram.
    - **Параметры**:
        - `update` (object): Объект, содержащий данные о входящем сообщении из Telegram API.
    - **Возвращает**:
        - `None`
    - **Вызывает исключения**:
        - `Exception`: Обрабатываются общие исключения.
