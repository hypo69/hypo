# Модуль `src.suppliers.ebay.graber`

## Обзор

Модуль `src.suppliers.ebay.graber` представляет собой класс, предназначенный для сбора значений полей со страниц товаров на сайте `ebay.com`. Для каждого поля товара предусмотрена функция обработки, которая может быть перегружена в этом классе для нестандартной обработки. Модуль также включает возможность выполнения предварительных действий перед отправкой запроса к веб-драйверу с использованием декораторов.
