# Модуль hypotez/src/logger

## Обзор

Модуль `hypotez/src/logger` определяет корневой путь к проекту `hypotez`.  Все импорты строятся относительно этого пути.  В будущем этот путь будет перенесен в системную переменную.  Текущая реализация находит корень проекта, начиная с текущего файла, двигаясь вверх по директориям до тех пор, пока не обнаружит файлы `pyproject.toml`, `requirements.txt` или `.git`.  Он также добавляет этот корневой путь в `sys.path` для корректного импорта модулей.  Модуль загружает настройки из файла `settings.json` и документ из `README.MD`.

## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная от текущего файла.

**Параметры**:

- `marker_files` (tuple): Список файлов/папок, которые указывают на корневой каталог проекта.  По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневому каталогу проекта, или к каталогу текущего файла, если корень не найден.

**Вызывает исключения**:

- Нет.


## Глобальные переменные

### `__root__`

**Описание**: Корневой путь к проекту.

**Тип**: `Path`

**Примечание**: Значение устанавливается функцией `set_project_root`.

### `settings`

**Описание**: Словарь настроек проекта, загруженный из файла `settings.json`.

**Тип**: `dict`

**Примечание**: Возможные значения: `None` - в случае ошибки загрузки файла.

### `doc_str`

**Описание**: Строка документации проекта, загруженная из файла `README.MD`.

**Тип**: `str`

**Примечание**: Возможные значения: `None` - в случае ошибки загрузки файла.

### `__project_name__`

**Описание**: Название проекта, полученное из настроек или имеющее значение по умолчанию `hypotez`.

**Тип**: `str`

**Примечание**: Зависит от `settings`.

### `__version__`

**Описание**: Версия проекта, полученная из настроек или пустая строка по умолчанию.

**Тип**: `str`

**Примечание**: Зависит от `settings`.

### `__doc__`

**Описание**: Документация проекта (из `README.MD` или пустая строка по умолчанию).

**Тип**: `str`

**Примечание**: Зависит от `doc_str`.

### `__details__`

**Описание**: Подробное описание проекта (пустая строка по умолчанию).

**Тип**: `str`

### `__author__`

**Описание**: Автор проекта (из настроек или пустая строка по умолчанию).

**Тип**: `str`

**Примечание**: Зависит от `settings`.

### `__copyright__`

**Описание**: Авторские права проекта (из настроек или пустая строка по умолчанию).

**Тип**: `str`

**Примечание**: Зависит от `settings`.

### `__cofee__`

**Описание**: Ссылка на помощь разработчикам (из настроек или значение по умолчанию).

**Тип**: `str`

**Примечание**: Зависит от `settings`.
