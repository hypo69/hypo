## Использование


Для использования функций и классов из этого модуля, импортируйте их из подмодуля `drive`:

```python
from .drive import GoogleDrive
```

Затем вы можете использовать объект `GoogleDrive` для работы с Google Диском.  Подробная документация по классу `GoogleDrive` находится в модуле `hypotez/src/goog/drive/drive.py`.
