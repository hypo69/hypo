# Модуль `request_parameters.py`

## Обзор

Модуль содержит классы, определяющие параметры запросов к API AliExpress, такие как типы продуктов, способы сортировки и типы ссылок. Он предоставляет константы для использования в запросах к API, обеспечивая типобезопасность и удобство использования.

## Подробнее

Этот модуль предназначен для стандартизации параметров, используемых при взаимодействии с API AliExpress. Определенные здесь классы и константы помогают избежать ошибок при формировании запросов и обеспечивают консистентность в рамках всего проекта `hypotez`.

## Классы

### `ProductType`

**Описание**: Класс, определяющий типы продуктов, которые можно запросить через API AliExpress.

**Как работает класс**: Класс содержит константы, представляющие различные типы продуктов, такие как `ALL`, `PLAZA` и `TMALL`. Эти константы используются для указания типа продукта в запросах к API.

**Методы**: Нет методов.

**Параметры**: Нет параметров.

**Примеры**:
```python
product_type = ProductType.ALL
print(product_type)  # Вывод: ALL
```

### `SortBy`

**Описание**: Класс, определяющий способы сортировки результатов поиска товаров в API AliExpress.

**Как работает класс**: Класс содержит константы, представляющие различные способы сортировки, такие как сортировка по возрастанию цены (`SALE_PRICE_ASC`), по убыванию цены (`SALE_PRICE_DESC`), по возрастанию объема продаж (`LAST_VOLUME_ASC`) и по убыванию объема продаж (`LAST_VOLUME_DESC`).

**Методы**: Нет методов.

**Параметры**: Нет параметров.

**Примеры**:
```python
sort_order = SortBy.SALE_PRICE_ASC
print(sort_order)  # Вывод: SALE_PRICE_ASC
```

### `LinkType`

**Описание**: Класс, определяющий типы ссылок, используемых в API AliExpress.

**Как работает класс**: Класс содержит константы, представляющие различные типы ссылок, такие как `NORMAL` (обычная ссылка) и `HOTLINK` (горячая ссылка). Эти константы используются для указания типа ссылки в запросах к API.

**Методы**: Нет методов.

**Параметры**: Нет параметров.

**Примеры**:
```python
link_type = LinkType.NORMAL
print(link_type)  # Вывод: 0
```
