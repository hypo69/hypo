# Модуль hypotez/src/endpoints/kazarinov/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`. Он также предоставляет доступ к документации проекта (README.MD).

## Функции

### `set_project_root`

**Описание**: Определяет корневую директорию проекта, начиная от директории текущего файла и идя вверх по директориям, пока не найдет директорию, содержащую один из файлов-маркеров (pyproject.toml, requirements.txt, .git).

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или директорий, по которым определяется корень проекта. По умолчанию это кортеж ('pyproject.toml', 'requirements.txt', '.git').

**Возвращает**:
- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращает директорию, в которой расположен текущий скрипт.

**Вызывает исключения**:
- Нет

### `set_project_root`

**Описание**: Эта функция поиска корневой директории проекта  (см. описание выше)

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или директорий, по которым определяется корень проекта. По умолчанию это кортеж ('pyproject.toml', 'requirements.txt', '.git').


**Возвращает**:
- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращает директорию, в которой расположен текущий скрипт.


**Вызывает исключения**:
- Нет



## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root`.

**Тип**: `Path`


### `settings`

**Описание**: Словарь настроек проекта, загруженный из файла `settings.json`.

**Тип**: `dict`


### `doc_str`

**Описание**: Строковое представление содержимого файла README.MD, если он найден.

**Тип**: `str`


### `__project_name__`

**Описание**: Имя проекта, взятое из настроек (`settings.json`) или установленное по умолчанию (`hypotez`).

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта, взятая из настроек (`settings.json`) или установленное по умолчанию (пустая строка).

**Тип**: `str`


### `__doc__`

**Описание**: Документация проекта (содержимое README.MD), если оно найдено.

**Тип**: `str`


### `__details__`

**Описание**: Подробности проекта (пустая строка по умолчанию).

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта, взятый из настроек (`settings.json`) или установленное по умолчанию (пустая строка).

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права проекта, взятые из настроек (`settings.json`) или установленное по умолчанию (пустая строка).

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка для пожертвования на поддержку проекта.

**Тип**: `str`
