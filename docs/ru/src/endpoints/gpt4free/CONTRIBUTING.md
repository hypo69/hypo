# Документация для разработчика gpt4free/CONTRIBUTING.md

## Обзор

Этот файл содержит инструкции для контрибьюторов проекта `gpt4free`, описывающие шаги для добавления новых веб-сайтов в список поддерживаемых.

## Подробнее

Этот файл предоставляет информацию о том, как внести свой вклад в проект `gpt4free`. Он описывает процесс реверс-инжиниринга веб-сайтов, добавления их в список для тестирования и рефакторинга для интеграции в основную библиотеку `g4f`.

## Инструкции по контрибьюции

1.  **Реверс-инжиниринг веб-сайта:**

    *   Выберите веб-сайт из списка [sites-to-reverse](https://github.com/xtekky/gpt4free/issues/40), который нужно добавить в проект.
    *   Изучите его структуру и API для понимания принципов его работы.
2.  **Добавление в список для тестирования:**

    *   Добавьте веб-сайт в каталог [./etc/unittest/](https://github.com/xtekky/gpt4free/tree/main/etc/unittest/), чтобы его можно было протестировать.
3.  **Рефакторинг и добавление в g4f:**

    *   Проведите рефакторинг кода для соответствия стандартам проекта.
    *   Добавьте реверс-инжиниринговый код в каталог [./g4f](https://github.com/xtekky/gpt4free/tree/main/g4f) для интеграции в основную библиотеку.

## Заключение

Разработчики проекта будут благодарны за любой вклад в развитие `gpt4free`.
