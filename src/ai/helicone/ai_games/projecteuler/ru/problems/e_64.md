# Задача No 64
## Нечетные периоды квадратных корней
## Условия
Все квадратные корни не являются целыми числами, и их можно представить в виде непрерывных дробей. Рассмотрим, к примеру, √23:

√23 = 4 + 1/(1 + 1/(3 + 1/(1 + 1/(8 + ...))))

Оказывается, что непрерывное разложение квадратного корня любого неквадратного числа становится периодическим. Рассмотрим разложения для √2, √3, √5, √6 и √7:

√2 = [1;(2)], период=1
√3 = [1;(1,2)], период=2
√5 = [2;(4)], период=1
√6 = [2;(2,4)], период=2
√7 = [2;(1,1,1,4)], период=4

С первыми десятью неквадратными числами мы получаем следующие периоды:

2, 1, 2, 1, 4, 2, 1, 2, 2, 6

Очевидно, что периоды могут быть сколь угодно длинными.

Найдите количество квадратных корней среди первых десяти тысяч натуральных чисел, которые имеют нечетный период.
