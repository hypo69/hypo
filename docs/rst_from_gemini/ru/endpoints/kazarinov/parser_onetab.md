# Модуль `parser_onetab.py`

Файл: `C:\Users\user\Documents\repos\hypotez\src\endpoints\kazarinov\parser_onetab.py`

## Описание

Этот модуль предназначен для парсинга URL-адресов из страниц OneTab. Он извлекает список ссылок, содержащихся в HTML-странице OneTab, и дополнительно парсит информацию о цене и названии из этой страницы.

## Функции

* **`prepare_one_tab(target_page_url: str) -> Tuple[int, str, List[str]] | bool`:**
    * Вызывает функцию `fetch_target_urls_onetab` для получения данных.
    * Возвращает кортеж `(цена, название, список_ссылок)` или `False` в случае ошибки.

* **`fetch_target_urls_onetab(target_page_url: str) -> Tuple[int, str, List[str]] | bool`:**
    * **Цель:** Извлекает целевые URL-адреса с указанной страницы OneTab.
    * **Логика:**
        1. Выполняет GET-запрос к `target_page_url`. Обрабатывает возможные ошибки запроса (`requests.exceptions.RequestException`) и ошибки кодировки.
        2. Использует `BeautifulSoup` для парсинга HTML-контента.
        3. Извлекает список ссылок (`urls`) из тегов `<a>` с классом `tabLink`.
        4. Извлекает данные о цене и названии из тега `<div>` с классом `tabGroupLabel`.
            * Обрабатывает случай, когда `div` не найден.
            * Разбивает строку с данными на цену и имя.
            * Парсит цену как целое число (`int`). Обрабатывает `ValueError` при ошибке преобразования.
            * Если имя не найдено, использует значение `gs.now`.
        5. Возвращает кортеж `(цена, имя, список_ссылок)` или `False` в случае ошибки.
    * **Обработка ошибок:**
        * Использует `try...except` для обработки исключений `requests.exceptions.RequestException` и `ValueError`.
        * Использует `logger` для записи сообщений об ошибках в лог.
        * Возвращает `False`, если произошла ошибка.
    * **Важные замечания:**
        * Проверяет статус ответа (`response.status_code`) на 200.
        *  Обрабатывает возможный случай отсутствия элемента `div` с классом `tabGroupLabel`, возвращая `False`.


## Использование

Для использования модуля необходимо:

1. Импортировать функции из модуля в другой скрипт.
2. Передать URL страницы OneTab в функцию `prepare_one_tab`.
3. Обработать возвращаемое значение (кортеж или `False`).

```python
from hypotez.src.endpoints.kazarinov.parser_onetab import prepare_one_tab

target_url = "ваш_url_onetab"

result = prepare_one_tab(target_url)

if result:
    price, name, urls = result
    print(f"Цена: {price}, Имя: {name}, Ссылки: {urls}")
else:
    print("Ошибка при парсинге.")
```
