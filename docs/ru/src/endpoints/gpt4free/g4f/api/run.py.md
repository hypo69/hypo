# Модуль запуска API g4f

## Обзор

Этот модуль отвечает за запуск API g4f (GPT4Free). Он импортирует модуль `g4f.api` и запускает API с возможностью включения режима отладки.

## Подробнее

Модуль `run.py` предназначен для упрощения запуска API GPT4Free.  Основное назначение - предоставить простой способ запуска API с параметрами конфигурации, такими как режим отладки.  В данном случае API запускается только при прямом выполнении этого скрипта (то есть, когда он не импортирован как модуль в другой скрипт).

## Функции

### `__main__`

```python
if __name__ == "__main__":
    g4f.api.run_api(debug=True)
```

**Назначение**: Условный блок, который выполняется только при непосредственном запуске скрипта. Он вызывает функцию `run_api` из модуля `g4f.api` для запуска API.

**Как работает функция**:

1.  **Условие `if __name__ == "__main__":`**: Проверяет, является ли текущий скрипт главным запускаемым файлом. Если да, то выполняется код внутри блока.
2.  **Вызов `g4f.api.run_api(debug=True)`**: Запускает API, используя функцию `run_api` из модуля `g4f.api`. Параметр `debug=True` включает режим отладки, что полезно для разработки и тестирования.
