Firefox WebDriver
=================

.. module:: hypotez.src.webdriver.firefox
    :platform: Windows, Unix
    :synopsis: Firefox WebDriver

This module defines a subclass of `selenium.webdriver.Firefox` called `Firefox`.
It provides additional functionality such as launching Firefox in kiosk mode
and setting up a Firefox profile for the WebDriver.  It also includes
integration with `ExecuteLocator` and `JavaScript` for enhanced
interaction with the browser.
