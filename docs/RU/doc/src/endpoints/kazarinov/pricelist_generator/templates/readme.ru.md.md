# Шаблон для создания HTML отчета из сценария мехирона

## Обзор

Данный шаблон предназначен для генерации HTML-отчета на основе данных, полученных из сценария мехирона. Он представляет собой базовую структуру для создания отчета, который может включать в себя таблицы, графики и другой визуальный контент.

## Структура шаблона

Шаблон включает в себя следующие основные блоки:

1. **Заголовок отчета (`report_title`)**: Задает заголовок для HTML-страницы.

2. **Стили (`report_style`)**: Включает в себя CSS-стили, определяющие внешний вид элементов отчета.

3. **Тело отчета (`report_body`)**: Содержит основной контент отчета, включая таблицы и другие элементы.

   - **Таблица (`table_data`)**: Представляет собой HTML-таблицу, в которой отображаются данные.
      - **Заголовки таблицы (`table_headers`)**: Устанавливает заголовки столбцов таблицы.
      - **Строки таблицы (`table_rows`)**: Устанавливает данные строк таблицы.

## Использование шаблона

1. **Заголовок отчета:** Заголовок задается с помощью переменной `report_title`. Измените значение этой переменной, чтобы установить необходимый заголовок.

2. **Стили:** Стили CSS находятся в переменной `report_style`. Вы можете настроить стили, чтобы изменить внешний вид таблицы и других элементов отчета.

3. **Тело отчета:** Основное содержание HTML-отчета находится в переменной `report_body`. Вы можете добавлять в тело таблицы (`table_data`) данные, используя переменные `table_headers` и `table_rows`.

## Пример

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>{{ report_title }}</title>
    <style>
        {{ report_style }}
    </style>
</head>
<body>
    {{ report_body }}
    <table class="report_table">
        <thead>
            <tr>
            {% for header in table_headers %}
                <th>{{ header }}</th>
            {% endfor %}
            </tr>
        </thead>
        <tbody>
            {% for row in table_rows %}
                <tr>
                    {% for cell in row %}
                        <td>{{ cell }}</td>
                    {% endfor %}
                </tr>
            {% endfor %}
        </tbody>
    </table>
</body>
</html>
```

## Зависимости

Шаблон использует синтаксис шаблонизатора Jinja2, который позволяет вставлять динамические данные в HTML.

## Заключение

Этот шаблон предоставляет простую основу для создания HTML-отчетов из данных мехирона. Вы можете расширять и настраивать его для удовлетворения конкретных требований к отчетам.
