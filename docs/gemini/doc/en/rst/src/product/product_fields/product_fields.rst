hypotez/src/product/product_fields/product_fields.py
===================================================

.. module:: hypotez.src.product.product_fields
   :platform: Windows, Unix
   :synopsis: Contains the `ProductFields` class for describing product fields in PrestaShop tables.

This module defines the `ProductFields` class, which encapsulates the data for product fields in PrestaShop tables.  It includes methods for loading field data, handling various data types (including optional types and dictionaries), and logging errors during field assignment.  The module also provides support for language detection using the `langdetect` library.
