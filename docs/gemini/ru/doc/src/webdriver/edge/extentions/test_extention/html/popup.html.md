# Документация для `popup.html`

## Обзор

Файл `popup.html` представляет собой HTML-страницу, которая отображается во всплывающем окне расширения браузера. Это простой HTML-файл, который предназначен для отображения информации пользователю о функциональности расширения.

## HTML-структура

HTML-структура файла `popup.html` включает следующие элементы:
- `<!DOCTYPE html>`: Объявление типа документа HTML5.
- `<html>`: Корневой элемент HTML-документа.
- `<head>`: Содержит метаданные документа, такие как заголовок и стили.
- `<title>`: Задает заголовок документа, отображаемый на вкладке браузера.
- `<style>`: Включает в себя встроенные CSS-стили для страницы.
- `<body>`: Содержит основное содержание HTML-документа, видимое пользователю.

## CSS-стили

CSS-стили внутри `<style>` устанавливают следующие параметры:

- `body`: Ширина установлена в 200 пикселей, с внутренними отступами в 10 пикселей. Это обеспечивает ограниченный размер и отступы для содержимого всплывающего окна.

## Заголовок

```html
<h1>hypotez</h1>
```

Заголовок `<h1>` отображает название расширения "hypotez".

## Текст

```html
<p>Click the extension icon to collect data from the current webpage.</p>
```

Текст абзаца `<p>` объясняет пользователю, что для сбора данных со страницы нужно нажать на значок расширения.
