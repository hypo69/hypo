# Модуль: src.suppliers.bangood._experiments

## Обзор

Модуль содержит экспериментальный код, связанный с поставщиком Banggood. Включает импорты необходимых библиотек, настройку путей и определение функции для старта поставщика.

## Подробней

Этот модуль, по-видимому, является частью процесса экспериментов и отладки для интеграции с поставщиком Banggood. Он содержит необходимые импорты и базовую функцию для инициализации поставщика, что позволяет тестировать и проверять различные аспекты взаимодействия с Banggood.

## Функции

### `start_supplier`

```python
def start_supplier(supplier_prefix: str = 'aliexpress', locale: str = 'en') -> Supplier:
    """Старт поставщика
    
    Args:
        supplier_prefix (str, optional): Префикс поставщика. По умолчанию 'aliexpress'.
        locale (str, optional): Локаль. По умолчанию 'en'.
    
    Returns:
        Supplier: Объект поставщика.
    
    """
```

**Назначение**:
Функция `start_supplier` предназначена для инициализации и запуска объекта поставщика с заданными параметрами.

**Параметры**:
- `supplier_prefix` (str, optional): Префикс поставщика, используемый для идентификации поставщика. По умолчанию `'aliexpress'`.
- `locale` (str, optional): Локаль, определяющая язык и региональные настройки поставщика. По умолчанию `'en'`.

**Возвращает**:
- `Supplier`: Объект поставщика, созданный на основе переданных параметров.

**Как работает функция**:

1. **Определение параметров**: Функция принимает два параметра: `supplier_prefix` и `locale`, которые используются для конфигурации поставщика.
2. **Создание словаря параметров**: Создается словарь `params`, содержащий переданные параметры `supplier_prefix` и `locale`.
3. **Инициализация и возврат объекта `Supplier`**: Функция создает и возвращает экземпляр класса `Supplier`, передавая словарь `params` в качестве аргументов для инициализации.

**Примеры**:
```python
supplier = start_supplier(supplier_prefix='bangood', locale='ru')
# Возвращает объект Supplier, сконфигурированный для Banggood с русской локалью.
```
```python
supplier = start_supplier()
# Возвращает объект Supplier с настройками по умолчанию (aliexpress, en).
```
```python
supplier = start_supplier(locale='de')
# Возвращает объект Supplier с префиксом по умолчанию (aliexpress) и немецкой локалью.
```
```python
supplier = start_supplier(supplier_prefix='bangood')
# Возвращает объект Supplier для Banggood с локалью по умолчанию (en).
```
```python
supplier = start_supplier(supplier_prefix='bangood', locale='ru')
# Возвращает объект Supplier для Banggood с русской локалью.
