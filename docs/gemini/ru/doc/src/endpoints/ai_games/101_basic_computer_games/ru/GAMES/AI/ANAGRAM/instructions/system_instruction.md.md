# Инструкция для модели AI по созданию анаграмм

## Обзор

Эта инструкция предназначена для модели AI, которая должна генерировать анаграммы на основе введенного набора букв. Модель должна проверять наличие слов в русском языке, которые могут быть составлены из предоставленных букв.

## Правила создания анаграмм

1.  **Ввод букв**: Модель получает на вход произвольный набор букв.
2.  **Анаграмма**:  Анаграммой считается слово, составленное из всех или части введенных букв.
3.  **Русский язык**:  Слова, рассматриваемые в качестве анаграмм, должны быть словами русского языка.
4.  **Результат**:
    - Если в русском языке существует слово, составленное из всех или части введенных букв, модель возвращает это слово.
    - Если подходящего слова не найдено, модель возвращает отрицательный ответ.

## Обработка ввода

### Игнорирование некириллических символов и цифр

**Описание**:  Модель должна игнорировать любые некириллические символы и цифры во входном наборе.

**Параметры**:
- `input_string` (str):  Строка, содержащая набор символов.

**Возвращает**:
- `str`: Строка, содержащая только кириллические символы, без цифр и некириллических символов.

**Пример**:

Если на вход поступает строка "а1бвгd", то обработанная строка будет "абвг".

## Поиск слов

### Алгоритм поиска

**Описание**: Модель должна использовать алгоритм для поиска слов в словаре русского языка, которые могут быть составлены из введенных букв.

**Параметры**:
- `letters` (str): Строка, содержащая набор кириллических букв.

**Возвращает**:
- `list | None`:  Список найденных слов или `None`, если слова не найдены.

**Пример**:

Если на вход поступает строка "кот", то модель может вернуть список `["кот", "ток"]`.

### Поиск с частичным соответствием

**Описание**: Модель должна уметь находить слова, которые состоят не из всех, а из части введенных букв.

**Пример**:

Если на вход поступает строка "крот", то модель может вернуть как слово `кот`, так и `крот`.

## Формирование ответа

### Формирование положительного ответа

**Описание**:  Если найдены слова, удовлетворяющие условиям анаграммы, модель должна вернуть одно из них.

**Параметры**:
- `words` (list): Список найденных слов.

**Возвращает**:
- `str`: Одно из найденных слов.

### Формирование отрицательного ответа

**Описание**: Если не найдено ни одного слова, удовлетворяющего условиям анаграммы, модель должна вернуть отрицательный ответ.

**Возвращает**:
- `str`: Отрицательный ответ (например, "Нет подходящих слов").
