# Задача No 282
## Функция Аккермана
## Условия
Для неотрицательных целых чисел *m* и *n* функция Аккермана A(*m*, *n*) определяется следующим образом:
* A(0, *n*) = *n* + 1
* A(*m*, 0) = A(*m* - 1, 1) если *m* > 0
* A(*m*, *n*) = A(*m* - 1, A(*m*, *n* - 1)) если *m*, *n* > 0

Например, A(1, 0) = A(0, 1) = 2.
Функция Аккермана быстро растет, например, A(4, 2) – это число из 19 729 десятичных цифр.

Найдите последнюю цифру числа A(n, n) при *n* = 10<sup>19</sup>.
