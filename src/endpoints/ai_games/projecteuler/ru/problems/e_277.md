# Задача No 277
## Последовательности с тройным правилом

## Условия

Последовательность целых чисел начинается с a₁ = 1 и продолжается следующим образом:

Если aₙ делится на 3, то aₙ₊₁ = aₙ / 3.
Если остаток от деления aₙ на 3 равен 1, то aₙ₊₁ = (4aₙ + 2) / 3.
Если остаток от деления aₙ на 3 равен 2, то aₙ₊₁ = (2aₙ - 1) / 3.

Последовательность продолжается, пока не достигнет значения 1.
Рассмотрим последовательность, начинающуюся с 1:

1, 2, 1, 2, 1, 2, 1, 2, 1, 2, ...

Поскольку последовательность не достигает значения 1, необходимо использовать другое начальное значение.

Теперь рассмотрим последовательность, начинающуюся с 10:

10, 14, 10, 14, 10, 14, 10, 14, ...

И эта последовательность также не достигает значения 1, поэтому опять нужно другое начальное значение.

Теперь рассмотрим последовательность, начинающуюся с 25:

25, 17, 11, 7, 5, 3, 1.

В данном примере последовательность достигает 1 за 6 шагов.

Задано начальное число.

Необходимо закодировать эту последовательность следующим образом:
Для каждого шага, когда вы берете aₙ, применяя тройное правило, запишите «d», если aₙ делится на 3, «u», если остаток от деления aₙ на 3 равен 1, и «n», если остаток от деления aₙ на 3 равен 2.
Итак, для примера с 25 получим последовательность кода «uunnnd».

Теперь рассмотрим последовательность, которая генерирует код «ddnndduddu».
Найдите наименьшее целое число, которое генерирует эту последовательность.

## Пример

Как описано в условиях, последовательность, начинающаяся с 25, дает код "uunnnd".
