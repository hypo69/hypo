# Модуль `src.templates._examples`

## Обзор

Модуль `src.templates._examples` содержит информацию о версиях и документацию.

## Содержание

- [Переменные](#Переменные)

## Переменные

### `__version__`

**Описание**: Версия пакета.

**Тип**: `packaging.version.Version`

### `__doc__`

**Описание**: Документация пакета.

**Тип**: `str`

### `__details__`

**Описание**: Детали пакета.

**Тип**: `str`
