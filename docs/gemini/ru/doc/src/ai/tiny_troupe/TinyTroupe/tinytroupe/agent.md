# Модуль tinytroupe.agent

## Обзор

Данный модуль предоставляет основные классы и функции для агентов TinyTroupe. Агенты — ключевое понятие в TinyTroupe. Они представляют собой моделируемых людей или сущностей, способных взаимодействовать с другими агентами и окружающей средой, получая стимулы и производя действия. Агенты обладают внутренним состоянием, которое обновляется по мере их взаимодействия со средой и другими агентами. Они могут хранить и извлекать информацию из памяти и выполнять действия в среде. В отличие от агентов, призванных поддерживать AI-ассистентов или аналогичные инструменты повышения производительности, агенты TinyTroupe **нацелены на представление поведения, подобного человеческому**, которое включает особенности, эмоции и другие человеческие черты, которых не стоит ожидать от инструмента повышения производительности.

Основной дизайн вдохновлен когнитивной психологией, поэтому агенты обладают различными внутренними когнитивными состояниями, такими как внимание, эмоции и цели. Также это объясняет, почему память агентов, в отличие от других платформ агентов на основе LLMs, имеет тонкие внутренние разделения, особенно между эпизодической и семантической памятью. Также присутствуют некоторые концепции бихевиоризма, такие как "стимул" и "ответ" в методах `listen` и `act`, которые являются ключевыми абстракциями для понимания взаимодействия агентов с окружающей средой и другими агентами.

## Параметры по умолчанию

### `default`

Словарь, содержащий параметры по умолчанию для моделей, таких как `embedding_model` и `max_content_display_length`, которые могут быть переопределены конфигурационными файлами.

## Классы

### `TinyPerson`

**Описание**: Моделирует человека в вселенной TinyTroupe.

**Атрибуты**:
* `name` (str): Имя агента.
* `episodic_memory` (EpisodicMemory): Эпизодическая память агента.
* `semantic_memory` (SemanticMemory): Семантическая память агента.
* `_mental_faculties` (list): Список когнитивных способностей агента.

**Методы**:

#### `__init__`

**Описание**: Инициализирует агента.

**Параметры**:
* `name` (str): Имя агента.
* `episodic_memory` (EpisodicMemory, optional): Реализация эпизодической памяти. По умолчанию `EpisodicMemory()`.
* `semantic_memory` (SemanticMemory, optional): Реализация семантической памяти. По умолчанию `SemanticMemory()`.
* `mental_faculties` (list, optional): Список когнитивных способностей агента.


#### `_post_init`

**Описание**: Выполняется после `__init__`. Используется для инициализации дополнительных атрибутов агента.

#### `generate_agent_prompt`

**Описание**: Генерирует запрос для агента.

**Возвращает**:
* str: Сгенерированный запрос.


#### `reset_prompt`

**Описание**: Сбрасывает запрос для агента.

#### `get`

**Описание**: Возвращает значение атрибута конфигурации.

**Параметры**:
* `key` (str): Ключ атрибута.

**Возвращает**:
* Any: Значение атрибута.


#### `define`

**Описание**: Определяет значение атрибута конфигурации.

**Параметры**:
* `key` (str): Ключ атрибута.
* `value` (Any): Значение атрибута.
* `group` (str, optional): Группа атрибута (если нужно).


#### `define_several`

**Описание**: Определяет несколько значений атрибутов конфигурации в одной группе.

**Параметры**:
* `group` (str): Имя группы.
* `records` (list): Список записей.


#### `define_relationships`

**Описание**: Определяет или обновляет отношения агента.

**Параметры**:
* `relationships` (list or dict): Отношения.
* `replace` (bool, optional): Заменить существующие отношения. По умолчанию `True`.

**Возвращает**:
* `TinyPerson`: Сам агент, для цепочки вызовов.


#### `clear_relationships`

**Описание**: Очищает отношения агента.

#### `related_to`

**Описание**: Определяет отношение между двумя агентами.

**Параметры**:
* `other_agent` (TinyPerson): Другой агент.
* `description` (str): Описание отношения.
* `symmetric_description` (str, optional): Описание симметричного отношения.

**Возвращает**:
* `TinyPerson`: Сам агент, для цепочки вызовов.
