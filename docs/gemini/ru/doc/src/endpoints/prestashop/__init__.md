# Модуль hypotez/src/endpoints/prestashop/__init__.py

## Обзор

Этот модуль предоставляет интерфейс для работы с API Престашоп.  Он экспортирует классы для взаимодействия с различными ресурсами, такими как продукты, поставщики, категории, склады и т.д.  Модуль инициализируется константой `MODE`, которая указывает режим работы.

## Переменные

### `MODE`

**Описание**: Строковая константа, определяющая режим работы. В данном случае используется значение `'dev'`.

## Классы

### `PrestaShop`

**Описание**: Класс для работы с основными настройками и информацией о магазине Престашоп.

### `PrestaProduct`

**Описание**: Класс для работы с продуктами в Престашоп.

### `PrestaSupplier`

**Описание**: Класс для работы с поставщиками в Престашоп.

### `PrestaCategory`

**Описание**: Класс для работы с категориями в Престашоп.

### `PrestaWarehouse`

**Описание**: Класс для работы со складами в Престашоп.

### `PrestaLanguage`

**Описание**: Класс для работы с языками в Престашоп.

### `PrestaShopShop`

**Описание**: Класс для работы с магазинами Престашоп.

### `PriceListRequester`

**Описание**: Класс для запроса прайс-листов.

### `PrestaCustomer`

**Описание**: Класс для работы с клиентами Престашоп.
