# Модуль `login`

## Обзор

Модуль `login.py` содержит функции для выполнения входа в систему AliExpress с использованием веб-драйвера. Он включает в себя взаимодействие с элементами веб-страницы для ввода учетных данных и принятия cookies.

## Функции

### `login`

**Описание**: Выполняет вход в систему AliExpress с использованием веб-драйвера.

**Параметры**:
- `s` (`Supplier`): Класс поставщика с запущенным веб-драйвером.

**Возвращает**:
- `bool`: Возвращает `True` в случае успешного входа. В текущей реализации всегда возвращает `True` (debug).

**Вызывает исключения**:
- Не вызывает исключений, но включает в себя TODO комментарии для будущей обработки ошибок.
