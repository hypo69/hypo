# Модуль hypotez/src/product/header.py

## Обзор

Данный модуль определяет корневой путь к проекту `hypotez`. Все импорты строятся относительно этого пути. В дальнейшем планируется перенести определение корневого пути в системную переменную.  Модуль также загружает настройки из файла `settings.json` и содержит документацию из `README.MD`, если таковая присутствует.

## Функции

### `set_project_root`

**Описание**:  Находит корневую директорию проекта, начиная от директории текущего файла. Поиск выполняется вверх по дереву директорий, пока не найдена директория, содержащая один из указанных маркеров (файлов или директорий).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для определения корневой директории проекта. По умолчанию: `('pyproject.toml', 'requirements.txt', '.git')`.


**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена, иначе путь к директории, где находится скрипт.


**Вызывает исключения**:

- Нет.


## Глобальные переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный вызовом функции `set_project_root()`.


### `settings`

**Описание**: Словарь настроек проекта, загруженный из файла `src/settings.json`. Возможные значения: `dict`, `None`.

### `doc_str`

**Описание**: Строка документации проекта, загруженная из файла `src/README.MD`. Возможные значения: `str`, `None`.

### `__project_name__`

**Описание**: Имя проекта, полученное из `settings.json` или значение по умолчанию `hypotez`. Тип: `str`.

### `__version__`

**Описание**: Версия проекта, полученная из `settings.json` или значение по умолчанию `''`. Тип: `str`.

### `__doc__`

**Описание**: Документация проекта, полученная из `README.MD` или значение по умолчанию `''`. Тип: `str`.

### `__details__`

**Описание**: Подробная информация о проекте (по умолчанию пустая строка). Тип: `str`.

### `__author__`

**Описание**: Автор проекта, полученный из `settings.json` или значение по умолчанию `''`. Тип: `str`.

### `__copyright__`

**Описание**: Авторские права на проект, полученные из `settings.json` или значение по умолчанию `''`. Тип: `str`.

### `__cofee__`

**Описание**: Ссылка для поддержки разработчика чашкой кофе. Получен из `settings.json`, или значение по умолчанию. Тип: `str`.
