# Модуль `login`

## Обзор

Модуль `login.py` предназначен для автоматизации процесса авторизации на сайте Amazon с использованием веб-драйвера. Он содержит функцию `login`, которая принимает объект поставщика (Supplier) в качестве аргумента и выполняет шаги, необходимые для входа в систему. Модуль использует локаторы, хранящиеся в объекте поставщика, для взаимодействия с элементами веб-страницы.

## Подробнее

Этот модуль является частью процесса автоматизации для поставщиков, работающих с Amazon. Он использует веб-драйвер для эмуляции действий пользователя, таких как ввод адреса электронной почты и пароля, а также нажатие кнопок для входа в систему. Модуль включает в себя обработку ситуаций, когда элементы не сразу доступны на странице, и использует логирование для отслеживания процесса входа в систему и регистрации ошибок.
