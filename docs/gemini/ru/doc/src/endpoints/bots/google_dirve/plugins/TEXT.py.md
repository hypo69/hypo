# Настройки бота для Google Drive

## Обзор

В этом модуле определены настройки и константы, используемые ботом для загрузки файлов в Google Drive. Он включает в себя параметры авторизации, пути загрузки, сообщения для пользователя и флаги поддержки различных платформ.

## Подробней

Этот файл содержит важные параметры конфигурации, которые определяют поведение бота. Здесь задаются учетные данные для Mega, имя папки в Google Drive, текстовые сообщения для взаимодействия с пользователем и настройки поддержки различных сервисов загрузки файлов.

## Переменные

### `drive_folder_name`

**Описание**: Имя папки в Google Drive, куда будут загружаться файлы.

### `MEGA_EMAIL`

**Описание**: Email для доступа к аккаунту Mega.

### `MEGA_PASSWORD`

**Описание**: Пароль для доступа к аккаунту Mega.

### `START`

**Описание**: Приветственное сообщение для пользователя при запуске бота.

### `HELP`

**Описание**: Справочное сообщение с информацией о командах и возможностях бота.

### `DP_DOWNLOAD`

**Описание**: Сообщение о начале загрузки файла из Dropbox.

### `OL_DOWNLOAD`

**Описание**: Сообщение о начале загрузки файла из Openload.

### `PROCESSING`

**Описание**: Сообщение об обработке запроса пользователя.

### `DOWN_TWO`

**Описание**: Флаг, определяющий поведение при загрузке (значение `True`).

### `DOWNLOAD`

**Описание**: Сообщение о начале загрузки файла.

### `DOWN_MEGA`

**Описание**: Сообщение о начале загрузки файла из Mega.

### `DOWN_COMPLETE`

**Описание**: Сообщение об успешном завершении загрузки файла.

### `NOT_AUTH`

**Описание**: Сообщение об отсутствии авторизации у пользователя.

### `REVOKE_FAIL`

**Описание**: Сообщение о неудачной попытке отзыва авторизации.

### `AUTH_SUCC`

**Описание**: Сообщение об успешной авторизации пользователя.

### `ALREADY_AUTH`

**Описание**: Сообщение о том, что пользователь уже авторизован.

### `AUTH_URL`

**Описание**: Форматированная строка, содержащая URL для авторизации в Google Drive.

### `UPLOADING`

**Описание**: Сообщение о начале загрузки файла в Google Drive.

### `REVOKE_TOK`

**Описание**: Сообщение об успешном отзыве токена авторизации.

### `DOWN_PATH`

**Описание**: Путь к папке для сохранения загруженных файлов (Linux-путь).

### `DOWNLOAD_URL`

**Описание**: Форматированная строка, содержащая информацию об успешной загрузке файла и ссылку на скачивание.

### `AUTH_ERROR`

**Описание**: Сообщение об ошибке авторизации.

### `OPENLOAD`

**Описание**: Флаг поддержки загрузки файлов из Openload (значение `True`).

### `DROPBOX`

**Описание**: Флаг поддержки загрузки файлов из Dropbox (значение `True`).

### `MEGA`

**Описание**: Флаг поддержки загрузки файлов из Mega (значение `True`).

### `UPDATE`

**Описание**: Информация об обновлении бота.
