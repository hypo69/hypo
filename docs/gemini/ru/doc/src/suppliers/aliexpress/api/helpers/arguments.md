# Модуль `hypotez/src/suppliers/aliexpress/api/helpers/arguments.py`

## Обзор

Данный модуль содержит вспомогательные функции для обработки аргументов, используемых в API AliExpress. Он предоставляет инструменты для преобразования и валидации входных данных, таких как списки и строки, для работы с идентификаторами продуктов.

## Функции

### `get_list_as_string`

**Описание**: Преобразует входное значение в строку, разделённую запятыми, если это список; в противном случае возвращает исходное значение или `None`, если входное значение `None`.

**Параметры**:

- `value`: (любой тип) значение для преобразования.

**Возвращает**:

- `str | None`: строковое представление входного значения, если оно является списком; в противном случае возвращает исходное значение или `None` при `None`.

**Вызывает исключения**:

- `InvalidArgumentException`: Если входное значение не является ни списком, ни строкой.


### `get_product_ids`

**Описание**: Преобразует входные данные в список идентификаторов продуктов.

**Параметры**:

- `values`: (строка или список) входные данные, содержащие идентификаторы продуктов, разделенные запятыми, или список строк с идентификаторами.

**Возвращает**:

- `list`: список идентификаторов продуктов.

**Вызывает исключения**:

- `InvalidArgumentException`: Если входное значение не является ни списком, ни строкой.
