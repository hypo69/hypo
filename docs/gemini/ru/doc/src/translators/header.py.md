# Модуль `header.py`

## Обзор

Модуль `header.py` предназначен для определения и настройки основных параметров проекта, таких как корневая директория, имя проекта, версия, автор, и т.д. Он также загружает настройки из файла `settings.json` и документацию из файла `README.MD`.

## Функции

### `set_project_root`

**Описание**:
Находит корневую директорию проекта, начиная с директории текущего файла, идя вверх по дереву каталогов до первой директории, содержащей один из указанных файлов-маркеров.

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или директорий для определения корневой директории проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена. В противном случае возвращает путь к директории, где расположен скрипт.

## Глобальные переменные

### `__root__`

**Описание**:
Путь к корневой директории проекта. Определяется с помощью функции `set_project_root`.

### `settings`

**Описание**:
Словарь настроек проекта, загруженный из файла `settings.json`. Значение `None`, если файл не найден или произошла ошибка JSON.

### `doc_str`

**Описание**:
Строка, содержащая документацию, загруженную из файла `README.MD`. Значение `None`, если файл не найден или произошла ошибка чтения.

### `__project_name__`

**Описание**:
Имя проекта, извлеченное из настроек (`settings.json`). По умолчанию `hypotez`, если параметр не найден в настройках.

### `__version__`

**Описание**:
Версия проекта, извлеченная из настроек (`settings.json`). Пустая строка, если параметр не найден в настройках.

### `__doc__`

**Описание**:
Документация проекта, загруженная из файла `README.MD`. Пустая строка, если файл не найден.

### `__details__`

**Описание**:
Дополнительная информация о проекте. На данный момент пустая строка.

### `__author__`

**Описание**:
Автор проекта, извлеченный из настроек (`settings.json`). Пустая строка, если параметр не найден в настройках.

### `__copyright__`

**Описание**:
Информация об авторских правах проекта, извлеченная из настроек (`settings.json`). Пустая строка, если параметр не найден в настройках.

### `__cofee__`

**Описание**:
Строка с предложением поддержать разработчика чашкой кофе.  Содержит ссылку на boosty.to.
