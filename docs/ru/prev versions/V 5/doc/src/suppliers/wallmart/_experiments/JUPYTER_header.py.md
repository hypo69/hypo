# Модуль `src.suppliers.wallmart._experiments.JUPYTER_header`

## Обзор

Модуль содержит набор экспериментов, связанных с поставщиком Wallmart, и включает в себя функциональность для взаимодействия с веб-драйвером, обработки продуктов и категорий, а также интеграцию с PrestaShop. Модуль предоставляет инструменты для запуска поставщика и работы с различными аспектами электронной коммерции.

## Подробней

Этот файл является частью экспериментального кода, связанного с поставщиком Wallmart. Он включает в себя импорты различных модулей и классов, необходимых для работы с данными о продуктах, категориях и поставщиках. Основная цель этого кода - предоставить возможность запуска поставщика с определенными параметрами, такими как префикс поставщика и локаль.
В коде производятся различные операции, включая добавление корневой директории проекта в `sys.path` для обеспечения доступа к другим модулям проекта. Также, выполняется импорт необходимых модулей, таких как `Driver`, `Supplier`, `Product`, `Category` и другие, используемые для работы с данными и взаимодействием с веб-драйвером.

## Функции

### `start_supplier`

```python
def start_supplier(supplier_prefix: str = 'aliexpress', locale: str = 'en') -> Supplier:
    """ Старт поставщика """
    params: dict = {
        'supplier_prefix': supplier_prefix,
        'locale': locale
    }
    
    return Supplier(**params)
```

**Описание**: Запускает поставщика с заданными параметрами.

**Как работает функция**:
Функция принимает префикс поставщика и локаль в качестве аргументов, создает словарь с этими параметрами и использует его для инициализации объекта `Supplier`. Затем возвращает созданный объект `Supplier`.

**Параметры**:
- `supplier_prefix` (str, optional): Префикс поставщика. По умолчанию `'aliexpress'`.
- `locale` (str, optional): Локаль поставщика. По умолчанию `'en'`.

**Возвращает**:
- `Supplier`: Объект поставщика, созданный с заданными параметрами.

**Примеры**:
```python
supplier = start_supplier(supplier_prefix='wallmart', locale='fr')
# Возвращает объект Supplier, инициализированный с префиксом 'wallmart' и локалью 'fr'.
```
```python
supplier = start_supplier()
# Возвращает объект Supplier, инициализированный с префиксом 'aliexpress' и локалью 'en'.
