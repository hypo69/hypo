# Модуль hypotez/src/gui/context_menu/header.py

## Обзор

Данный модуль `hypotez/src/gui/context_menu/header.py` отвечает за инициализацию путей к бинарным файлам GTK, FFmpeg и Graphviz, а также добавление их в системный путь. Он настраивает переменную окружения для WeasyPrint и подавляет предупреждения GTK.


## Постоянные переменные

### `MODE`

**Описание**:  Переменная, хранящая режим работы. В данном случае это строка 'dev'.

**Значение**: `'dev'`
