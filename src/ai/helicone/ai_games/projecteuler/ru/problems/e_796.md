# Задача No 796
## Обобщенные числа Фибоначчи
## Условия
Определим *обобщенные числа Фибоначчи* следующим образом:

Начнем с двух натуральных чисел $a$ и $b$, при этом $a \le b$.
Определим последовательность $F(a, b)$ как $f_1 = a$, $f_2 = b$ и $f_n = f_{n-1} + f_{n-2}$ для $n \ge 3$.
Пусть $S(a, b)$ будет суммой всех *различных* простых чисел, которые являются делителями любого из членов последовательности $F(a, b)$.

Например:
$F(3, 5) = 3, 5, 8, 13, 21, 34, 55, 89, 144, 233, 377, 610, 987, 1597, 2584, 4181, 6765, 10946, \dots$
Простые делители членов этой последовательности равны: 2, 3, 5, 13, 89, 233, 1597, 4181.
Следовательно, $S(3, 5) = 2 + 3 + 5 + 13 + 89 + 233 + 1597 + 4181 = 6123$.

Известно, что $S(1, 1) = 2$, $S(2, 3) = 2 + 3 + 5 + 7 + 13 + 17 + 23 + 29 + 47 = 146$.

Найдите $\sum S(p, q)$ для всех пар простых чисел $p$ и $q$ таких, что $2 \le p \le q \le 100$.
