# Модуль `header.py`

## Обзор

Модуль `header.py` предназначен для определения корневой директории проекта, загрузки настроек из файла `settings.json`, чтения документации из `README.MD`, а также для хранения основных метаданных проекта, таких как имя, версия, автор и т.д.

## Функции

### `set_project_root`

**Описание**: 
Находит корневую директорию проекта, начиная с директории текущего файла, ища вверх и останавливаясь на первой директории, содержащей любой из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple, optional): Кортеж имен файлов или директорий, по которым определяется корень проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если найдена, иначе — директория, где расположен скрипт.

## Переменные

### `__root__`

**Описание**:
Путь к корневой директории проекта. Определяется функцией `set_project_root`.

### `settings`

**Описание**:
Словарь с настройками, загруженными из файла `settings.json`. Если файл не найден или содержит ошибку, значение будет `None`.

### `doc_str`

**Описание**:
Строка с содержимым файла `README.MD`. Если файл не найден или возникает ошибка чтения, значение будет `None`.

### `__project_name__`

**Описание**:
Имя проекта, извлекаемое из настроек, по умолчанию 'hypotez'.

### `__version__`

**Описание**:
Версия проекта, извлекаемая из настроек. По умолчанию пустая строка.

### `__doc__`

**Описание**:
Содержание документации проекта, загруженное из `README.MD`.

### `__details__`

**Описание**:
Дополнительные детали проекта. В данном коде не используется и имеет пустую строку.

### `__author__`

**Описание**:
Автор проекта, извлекаемый из настроек. По умолчанию пустая строка.

### `__copyright__`

**Описание**:
Информация об авторских правах, извлекаемая из настроек. По умолчанию пустая строка.

### `__cofee__`

**Описание**:
Сообщение с призывом поддержать разработчика, извлекаемое из настроек. По умолчанию "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69".
