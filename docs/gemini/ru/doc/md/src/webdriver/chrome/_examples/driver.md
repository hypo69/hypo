# Модуль `hypotez/src/webdriver/chrome/_examples/driver.py`

## Обзор

Данный модуль предоставляет примеры использования классов `Driver` и `Chrome` для управления веб-драйвером Chrome.  В нем показаны различные методы, включая навигацию по страницам, извлечение домена, сохранение куки, обновление страницы, прокрутку, получение языка страницы, установку пользовательского агента, поиск элементов и получение текущего URL.


## Функции

### `main`

**Описание**: Основная функция, демонстрирующая примеры использования `Driver` и `Chrome`.

**Возвращает**:
- `None`: Функция не возвращает значения.


### `get_url`

**Описание**: Метод класса `Driver`, возвращает True, если навигация по URL успешна, иначе - False.


**Параметры**:
- `url (str)`: URL адрес для навигации.


**Возвращает**:
- `bool`: True, если навигация успешна, иначе False.


### `extract_domain`

**Описание**: Метод класса `Driver`, извлекает домен из предоставленного URL.


**Параметры**:
- `url (str)`: URL адрес для извлечения домена.


**Возвращает**:
- `str`: Извлеченный домен.


### `_save_cookies_localy`

**Описание**: Метод класса `Driver`, сохраняет куки в локальный файл.


**Возвращает**:
- `bool`: True, если сохранение прошло успешно, иначе False.


### `page_refresh`

**Описание**: Метод класса `Driver`, обновляет текущую страницу.


**Возвращает**:
- `bool`: True, если обновление прошло успешно, иначе False.


### `scroll`

**Описание**: Метод класса `Driver`, прокручивает страницу.


**Параметры**:
- `scrolls (int)`: Количество прокруток.
- `direction (str)`: Направление прокрутки ('forward' или 'backward').
- `frame_size (int)`: Размер фрейма для прокрутки.
- `delay (int)`: Задержка между прокрутками.


**Возвращает**:
- `bool`: True, если прокрутка прошла успешно, иначе False.


### `locale`

**Описание**:  Свойство класса `Driver`, возвращающее язык текущей страницы.

**Возвращает**:
- `str`: Язык текущей страницы.


### `find_element`

**Описание**: Метод класса `Driver`, находит элемент по селектору.


**Параметры**:
- `by (selenium.webdriver.common.by.By)`: Тип селектора.
- `selector (str)`: Значение селектора.


**Возвращает**:
- `selenium.webdriver.remote.webelement.WebElement`: Найденный элемент, или `None` если не найден.


### `current_url`

**Описание**: Свойство класса `Driver`, возвращающее текущий URL.

**Возвращает**:
- `str`: Текущий URL.


### `window_focus`

**Описание**: Метод класса `Driver`, фокусирует окно драйвера.

**Возвращает**:
- `None`: Функция не возвращает значения.
