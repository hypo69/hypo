# Модуль logger

## Обзор

Модуль `logger` предоставляет инструменты для логгирования событий телеграм-ботов.  Он настроен для работы в режиме разработки (`MODE = 'dev'`).

## Оглавление

* [Обзор](#обзор)
* [Константы](#константы)


## Константы

### `MODE`

**Описание**:  Переменная, определяющая режим работы логгирования. В данном случае, режим разработки ('dev').

```python
MODE = 'dev'
```
