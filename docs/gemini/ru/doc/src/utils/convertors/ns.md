# Модуль hypotez/src/utils/convertors/ns.py

## Обзор

Модуль `ns.py` предоставляет функции для конвертации объектов `SimpleNamespace` в различные форматы: `dict`, `JSON`, `CSV`, `XML` и `XLS`.  Он предназначен для удобной работы с объектами `SimpleNamespace` и их преобразования в более удобные для обработки структуры данных.

## Функции

### `ns2dict`

**Описание**: Функция рекурсивно преобразует объект `SimpleNamespace` в словарь.  Поддерживает вложенные структуры `SimpleNamespace`, `dict` и `list`.

**Параметры**:
- `ns_obj` (SimpleNamespace): Объект `SimpleNamespace` для преобразования.

**Возвращает**:
- `Dict[str, Any]`: Преобразованный словарь с вложенными структурами.

### `ns2csv`

**Описание**: Функция конвертирует объект `SimpleNamespace` в формат CSV и сохраняет его в указанный файл.

**Параметры**:
- `ns_obj` (SimpleNamespace): Объект `SimpleNamespace` для конвертации.
- `csv_file_path` (str | Path): Путь к файлу CSV для сохранения данных.

**Возвращает**:
- `bool`: `True`, если конвертация и сохранение успешны, `False` в противном случае.

**Обрабатывает исключения**:
- `Exception`: Возникает при проблемах с сохранением в файл CSV.  Подробная информация об ошибке записывается в лог.

### `ns2xml`

**Описание**: Функция конвертирует объект `SimpleNamespace` в XML-формат.

**Параметры**:
- `ns_obj` (SimpleNamespace): Объект `SimpleNamespace` для конвертации.
- `root_tag` (str, optional): Имя корневого тега в XML. По умолчанию `"root"`.

**Возвращает**:
- `str`: Строка XML-представления объекта.

**Обрабатывает исключения**:
- `Exception`: Возникает при проблемах с конвертацией в XML.  Подробная информация об ошибке записывается в лог.


### `ns2xls`

**Описание**: Функция конвертирует объект `SimpleNamespace` в формат XLS и сохраняет его в указанный файл.

**Параметры**:
- `data` (SimpleNamespace): Объект `SimpleNamespace` для конвертации.
- `xls_file_path` (str | Path): Путь к файлу XLS для сохранения данных.

**Возвращает**:
- `bool`: `True`, если конвертация и сохранение успешны, `False` в противном случае.

**Обрабатывает исключения**:
- `Exception`: Возникает при проблемах с сохранением в файл XLS. Подробная информация об ошибке записывается в лог.
