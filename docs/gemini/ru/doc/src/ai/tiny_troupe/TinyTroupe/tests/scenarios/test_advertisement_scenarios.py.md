# `test_advertisement_scenarios.py`

## Обзор

Этот файл содержит тесты для проверки сценариев, связанных с рекламными объявлениями, их оценкой и созданием, а также сценарий для профилирования потребителей. В тестах используются агенты `TinyPerson` для моделирования поведения различных людей, принимающих решения на основе представленной им информации.
