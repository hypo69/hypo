# Модуль hypotez/src/goog/drive/__init__.py

## Обзор

Этот модуль предоставляет инициализацию для взаимодействия с Google Drive.  Он импортирует необходимые классы и функции из подмодуля `drive`.

## Переменные

### `MODE`

**Описание**:  Переменная, определяющая режим работы (например, `dev`, `prod`).  Текущее значение: `'dev'`.

**Тип**: `str`


## Модули

### `google.drive`

**Описание**: Модуль, содержащий реализацию взаимодействия с Google Drive.

## Импорты

### `from .drive import GoogleDrive`

**Описание**: Импортирует класс `GoogleDrive` из модуля `drive`.
