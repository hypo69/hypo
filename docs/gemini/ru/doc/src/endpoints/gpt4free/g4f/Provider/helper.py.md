# Модуль helper

## Обзор

Модуль `helper` предоставляет вспомогательные функции для работы с провайдерами, cookies и асинхронными HTTP-запросами в контексте проекта `hypotez`. Он включает импорт необходимых функций и компонентов из других модулей проекта для обеспечения удобства и переиспользования кода.

## Подробнее

Модуль предназначен для упрощения работы с различными частями проекта, такими как управление провайдерами, обработка cookies и выполнение асинхронных запросов. Импортируя необходимые функции и классы из других модулей, `helper` обеспечивает централизованный доступ к ним и упрощает их использование в других частях проекта.

## Функции

### `get_cookies`

**Назначение**: Получение cookies.

**Параметры**:
- Отсутствуют.

**Возвращает**:
- `dict | None`: Возвращает словарь с cookies или `None` в случае ошибки.

**Как работает функция**:

1. Функция вызывает `get_cookies` из модуля `..cookies`, чтобы получить cookies.

**Примеры**:

```python
from ..cookies import get_cookies
cookies = get_cookies()
if cookies:
    print(f"Cookies: {cookies}")
else:
    print("Failed to retrieve cookies.")
```

### `get_connector`

**Назначение**: Получение коннектора для асинхронных HTTP-запросов.

**Параметры**:
- Отсутствуют.

**Возвращает**:
- `aiohttp.TCPConnector | None`: Возвращает TCPConnector для выполнения асинхронных HTTP-запросов или `None` в случае ошибки.

**Как работает функция**:

1. Функция вызывает `get_connector` из модуля `..requests.aiohttp`, чтобы получить коннектор.

**Примеры**:

```python
from ..requests.aiohttp import get_connector
connector = get_connector()
if connector:
    print(f"Connector: {connector}")
else:
    print("Failed to retrieve connector.")
```
