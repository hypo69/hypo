# Документация модуля `gomoko.py`

## Обзор

Этот модуль реализует базовую версию игры Гомоку (Gomoku), также известной как "крестики-нолики на доске 19x19". Игра предназначена для двух игроков, которые по очереди ставят свои символы ("X" и "O") на доску, стремясь выстроить линию из пяти своих символов по горизонтали, вертикали или диагонали.

## Константы

### `BOARD_SIZE`

- **Описание**: Размер стороны квадратной доски для игры.
- **Тип**: `int`
- **Значение**: 19

## Функции

### `create_board`

**Описание**: Создаёт и возвращает пустую доску для игры.

**Параметры**:
  - Нет параметров.

**Возвращает**:
  - `list`: Двумерный список, представляющий доску игры.

### `print_board`

**Описание**: Выводит текущее состояние доски на экран.

**Параметры**:
  - `board` (`list`): Двумерный список, представляющий доску игры.

**Возвращает**:
  - Нет возвращаемого значения.

### `place_stone`

**Описание**: Размещает камень игрока на указанной позиции доски.

**Параметры**:
  - `board` (`list`): Двумерный список, представляющий доску игры.
  - `row` (`int`): Номер строки, где размещается камень.
  - `col` (`int`): Номер столбца, где размещается камень.
  - `player` (`str`): Символ текущего игрока ('X' или 'O').

**Возвращает**:
  - Нет возвращаемого значения.

### `check_win`

**Описание**: Проверяет, выиграл ли текущий игрок, собрав линию из пяти своих камней.

**Параметры**:
  - `board` (`list`): Двумерный список, представляющий доску игры.
  - `row` (`int`): Номер строки последнего размещённого камня.
  - `col` (`int`): Номер столбца последнего размещённого камня.
  - `player` (`str`): Символ текущего игрока ('X' или 'O').

**Возвращает**:
  - `bool`: `True`, если игрок выиграл, `False` в противном случае.

### `is_board_full`

**Описание**: Проверяет, заполнена ли доска.

**Параметры**:
  - `board` (`list`): Двумерный список, представляющий доску игры.

**Возвращает**:
  - `bool`: `True`, если доска заполнена, `False` в противном случае.

### `get_user_input`

**Описание**: Запрашивает ввод координат от текущего игрока.

**Параметры**:
  - `player` (`str`): Символ текущего игрока ('X' или 'O').

**Возвращает**:
  - `tuple` | `None`: Кортеж из двух целых чисел (строка, столбец), если ввод корректен, `None` в противном случае.

**Вызывает исключения**:
  - `ValueError`: Если введено некорректное значение (не число).

### `play_gomoku`

**Описание**: Запускает основной игровой цикл, управляя ходами игроков, проверками победы и ничьей.

**Параметры**:
  - Нет параметров.

**Возвращает**:
  - Нет возвращаемого значения.
