# Модуль hypotez/src/endpoints/advertisement/facebook/scenarios/switch_account.py

## Обзор

Модуль `switch_account.py` содержит функцию для переключения учетных записей в Facebook.  В нем используется предварительно загруженный локатор для кнопки "Переключить".

## Функции

### `switch_account`

**Описание**: Функция переключения учетной записи в Facebook. Если на странице есть кнопка "Переключить", она нажимается.

**Параметры**:

- `driver` (Driver): Объект драйвера веб-драйвера, необходимый для взаимодействия с браузером.

**Возвращает**:

-  `None`: Функция не возвращает никакого значения.

**Вызывает исключения**:

-  Любые исключения, возникающие при работе с веб-драйвером или локаторами (например, `NoSuchElementException`).


## Локаторы

Локаторы для элементов страницы загружаются из файла `post_message.json` в директории `locators`.  Они определяют, как найти элемент "Переключить учетную запись".  Необходимо убедиться, что файл `post_message.json` содержит корректные локаторы для поиска кнопки.

**Примечание**:  Этот код предполагает, что `gs.path` и `src.webdriver.Driver` и `src.utils.j_loads_ns` определены в других модулях. Необходимы соответствующие импорты в файле, если они отсутствуют.  Также необходимо, чтобы переменная `MODE` была определена и имела значение 'dev'.
