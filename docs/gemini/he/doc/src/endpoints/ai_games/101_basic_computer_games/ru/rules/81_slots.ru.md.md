# SLOTS (משחק מכונות מזל)

## סקירה כללית

**SLOTS** הוא משחק המדמה פעולה של מכונת מזל. השחקן מבצע הימורים ומסובב את התופים כדי לקבל שילובים מנצחים. מטרת המשחק היא לצבור כמה שיותר נקודות על ידי ביצוע הימורים מוצלחים וקבלת שילובים מנצחים.

## הוראות שלב אחר שלב למימוש

### 1. אתחול המשחק

- התוכנית מציגה הודעת פתיחה ומסבירה את הכללים:
  ```
  ברוכים הבאים ל-SLOTS!
  המשימה שלכם היא לבצע הימורים ולסובב את התופים כדי לקבל שילובים מנצחים.
  המשחק נמשך עד שנגמרות לכם הנקודות לביצוע הימורים.
  בהצלחה!
  ```

- התוכנית יוצרת שלושה תופים עם סמלים (לדוגמה, תפוח, בננה, דובדבן, לימון, שבע).
- השחקן מתחיל עם כמות מסוימת של נקודות (לדוגמה, 100 נקודות).
- השחקן מבצע הימורים ומסובב את התופים.

---

### 2. תהליך משחק עיקרי

#### 2.1. הימור של שחקן:

- השחקן מציין את סכום ההימור.
- התוכנית בודקת האם יש לשחקן מספיק נקודות להימור:
  - אם יש מספיק נקודות, התוכנית מקבלת את ההימור.
  - אם אין מספיק נקודות, התוכנית מדווחת על שגיאה ומציעה לשחקן לבצע הימור קטן יותר:
    ```
    אין לכם מספיק נקודות עבור הימור זה. נסו שוב.
    ```
- התוכנית מציגה את כמות הנקודות הנוכחית של השחקן:
  ```
  יש לכם 100 נקודות.
  ```

#### 2.2. סיבוב התופים:
- התוכנית בוחרת באופן אקראי סמלים על התופים.
- התוכנית מדווחת על התוצאה:
  ```
  תופים: תפוח, בננה, דובדבן.
  ```

#### 2.3. ספירת תוצאות:
- התוכנית בודקת האם השחקן זכה:
  - אם השחקן קיבל שלושה סמלים זהים, התוכנית מוסיפה זכייה (לדוגמה, 10 נקודות עבור שלושה תפוחים).
  - אם השחקן קיבל שני סמלים זהים, התוכנית מוסיפה זכייה (לדוגמה, 5 נקודות עבור שתי בננות).
  - אם השחקן לא קיבל שילוב מנצח, התוכנית גובה את ההימור.
- התוכנית מעדכנת את כמות הנקודות של השחקן:
  ```
  זכיתם ב-10 נקודות!
  עכשיו יש לכם 110 נקודות.
  ```

#### 2.4. בדיקת תנאי סיום המשחק:
- המשחק מסתיים אם לשחקן נגמרות הנקודות לביצוע הימורים.
- התוכנית מכריזה על התוצאה הסופית:
  ```
  המשחק הסתיים! יש לכם 0 נקודות.
  ```

---

### 3. סיום המשחק

- לאחר סיום המשחק התוכנית מציעה לשחק שוב:
  ```
  רוצים לשחק שוב? (כן/לא)
  ```

- אם השחקן בוחר "כן", המשחק מתחיל מחדש עם כמות נקודות חדשה.

## דוגמה לפעולת התוכנית

1.  **תחילת המשחק:**
    ```
    ברוכים הבאים ל-SLOTS!
    יש לכם 100 נקודות.
    הכניסו את סכום ההימור:
    > 10
    ההימור התקבל.
    ```
2.  **תהליך המשחק:**
    ```
    מסובבים את התופים...
    תופים: תפוח, בננה, דובדבן.
    הפסדתם 10 נקודות.
    עכשיו יש לכם 90 נקודות.

    הכניסו את סכום ההימור:
    > 20
    ההימור התקבל.
    ```

3.  **סיום המשחק:**
    ```
    מסובבים את התופים...
    תופים: שבע, שבע, שבע.
    זכיתם ב-100 נקודות!
    עכשיו יש לכם 190 נקודות.

    רוצים לשחק שוב? (כן/לא):
    > לא
    תודה ששיחקתם!
    ```

## מגבלות אפשריות

- השחקן חייב להכניס את סכום ההימור בפורמט הנכון.
- התוכנית חייבת לטפל בקלט לא נכון ולהציע לנסות שוב.
- השחקן לא יכול לבצע הימור שגדול מכמות הנקודות שלו.

## מימוש

ניתן לממש את המשחק ב-Python באמצעות האפשרויות הבאות:

- **מודול `random`** לבחירת סמלים אקראיים על התופים.
- **לולאות ותנאים** לבדיקת קלט השחקן ולטיפול בהימורים שלו.
- **פונקציות** לספירת תוצאות ועדכון נקודות.
