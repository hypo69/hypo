# Модуль `hypotez/src/suppliers/aliexpress/header.py`

## Обзор

Этот модуль содержит функцию `set_project_root` для определения корневой директории проекта.  Он также загружает настройки из файла `settings.json` в переменную `settings`.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла и идя вверх по дереву директорий, пока не обнаружится директория, содержащая один из файлов или директорий из переданного списка `marker_files`.

**Параметры**:

- `marker_files` (tuple): Список файлов или директорий, по наличию которых определяется корневая директория. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращает директорию, в которой находится скрипт.

**Вызывает исключения**:

- Не вызывает никаких исключений.


### Загрузка настроек (`settings`)

**Описание**:  Загружает настройки из файла `settings.json`, находящегося в корневой директории проекта.

**Обработка исключений**:

- `FileNotFoundError`: Если файл `settings.json` не найден.
- `json.JSONDecodeError`: Если файл `settings.json` содержит невалидный JSON. В обоих случаях выполняется `...`, что означает пропуск обработки и сохранение значения `settings` как `None`.

**Параметры**: Нет.

**Возвращает**:
- `dict | None`: Словарь с настройками (`dict`), если файл успешно загружен. В противном случае возвращает `None`.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root()`.


### `MODE`

**Описание**: Значение переменной `MODE`, хранящей режим работы (например, `dev`, `prod`).

### `settings`

**Описание**: Словарь, содержащий настройки, загруженные из файла `settings.json`.
