# Модуль try_xpath_check_frame.js

## Обзор

Этот файл содержит определения для проверки наличия фрейма в контексте `tryxpath`. Он определяет переменную `tryxpath` и устанавливает свойство `isContentLoaded`.


## Переменные

### `tryxpath`

**Описание**: Объект, содержащий методы и свойства для проверки XPath.

**Тип данных**: Объект

##  Используемые переменные
### `tryxpath.isContentLoaded`
**Описание**: Переменная, обозначающая загружен ли контент, необходимый для работы. Возможные значения: `true`/`false`.
