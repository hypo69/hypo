# Задача No 263
## Реципрокные циклы
## Условия
Рациональная дробь с числителем 1 называется единичной дробью. Десятичные представления единичных дробей имеют повторяющиеся десятичные части.

Например, 
1/2 = 0.5
1/3 = 0.(3)
1/7 = 0.(142857)
1/11 = 0.(09)

где (09) означает, что цифры 09 повторяются бесконечно. В единичной дроби 1/n длину повторяющейся десятичной части будем называть длиной реципрокного цикла.

Выясняется, что при n < 10 длина реципрокного цикла для 1/n принимает максимальное значение 6 при n = 7.

Найдите сумму всех чисел n, для которых n < 100 000 и длина реципрокного цикла для 1/n наибольшая.
