# Документация для `cdata_categories_gaming_laptops_dell.json`

## Обзор

Данный файл содержит JSON-конфигурацию для сценариев, связанных с игровыми ноутбуками Dell. Каждый сценарий определяет марку, URL, состояние, активность и категории PrestaShop. Файл предназначен для использования в системе автоматизации обработки данных поставщиков.

### `DELL GAMING 14 I5`

**Описание**: Конфигурация для игрового ноутбука Dell 14 дюймов с процессором Intel Core i5.

**Поля**:
- `brand` (str): Марка ноутбука - `DELL`.
- `url` (str): URL-адрес, относящийся к данному сценарию. 
- `checkbox` (bool): Флаг, определяющий состояние чекбокса, по умолчанию `false`.
- `active` (bool): Флаг, определяющий активность сценария, по умолчанию `true`.
- `condition` (str): Состояние товара, `new`.
- `presta_categories` (str): Строка с идентификаторами категорий PrestaShop через запятую: `3,49,50,23,93,10,5`.
