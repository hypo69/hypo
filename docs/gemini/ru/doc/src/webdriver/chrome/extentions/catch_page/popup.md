# Модуль popup.js

## Обзор

Этот скрипт JavaScript обрабатывает клики по кнопке "Отправить URL" в всплывающем окне расширения Chrome.  Он получает текущий URL активной вкладки, отправляет его в фоновый скрипт и отображает соответствующее сообщение об успехе или ошибке.

## Функции

### `addEventListener`

**Описание**: Регистрирует обработчик события для клика по кнопке с идентификатором "sendUrlButton".

**Параметры**:
- `event`: (click): Событие клика.

**Возвращает**:
-  Не имеет возвращаемого значения.

**Обработка исключений**:
- Не обрабатывает исключения.



### `chrome.tabs.query`

**Описание**: Выполняет запрос к API Chrome для получения информации об активной вкладке.

**Параметры**:
- `options`: Объект с параметрами запроса. `active: true, currentWindow: true` указывают на необходимость получения активной вкладки в текущем окне.

**Возвращает**:
- `tabs`: Массив объектов `tabs`, содержащих информацию о вкладке.

**Обработка исключений**:
- Не обрабатывает исключения.



### `chrome.runtime.sendMessage`

**Описание**: Отправляет сообщение в фоновый скрипт.

**Параметры**:
- `message`: Объект с данными сообщения.  Здесь `action: "sendUrl", url: activeTabUrl`.
- `callback`: Функция обратного вызова, которая будет вызвана после получения ответа от фонового скрипта.

**Возвращает**:
- Не имеет возвращаемого значения.

**Обработка исключений**:
- Не обрабатывает исключения.


### Анонимная функция (внутри `addEventListener`)

**Описание**: Обработчик события клика по кнопке.

**Параметры**:
- Не принимает параметров.

**Возвращает**:
- Не имеет возвращаемого значения.

**Обработка исключений**:
- Не обрабатывает исключения.

**Детали**:
- Получает активную вкладку.
- Получает URL активной вкладки.
- Отправляет URL в фоновый скрипт с помощью `chrome.runtime.sendMessage`.
- Обрабатывает ответ от фонового скрипта, отображая сообщения об успехе или неудаче.


##  Обработка событий

Этот скрипт реагирует на клик кнопки, получая информацию о текущей вкладке и отправляя запрос в фоновый скрипт для обработки URL.
