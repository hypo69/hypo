# Модуль `graber.py`

## Обзор

Модуль `graber.py` является частью проекта `hypotez` и предназначен для сбора данных о товарах с сайта `wallashop.co.il`. Он содержит класс `Graber`, который наследуется от базового класса `Graber` (Grbr) и переопределяет некоторые методы для специфичной обработки полей товаров на сайте `wallashop`.

## Подорбней

Этот модуль играет важную роль в процессе автоматического сбора данных о товарах. Он использует веб-драйвер для взаимодействия с сайтом и извлечения необходимой информации. Класс `Graber` содержит логику для обработки различных полей товаров, и при необходимости может переопределять методы родительского класса для нестандартной обработки. Модуль также использует декораторы для выполнения предварительных действий перед отправкой запросов к веб-драйверу.

## Классы

### `Graber`

**Описание**: Класс `Graber` предназначен для сбора данных о товарах с сайта `wallashop.co.il`. Он наследуется от класса `Graber` (Grbr) и переопределяет некоторые методы для специфичной обработки полей товаров на этом сайте.

**Методы**:
- `__init__`: Инициализирует экземпляр класса `Graber`.

**Параметры**:
- `driver` (Driver): Экземпляр веб-драйвера для взаимодействия с сайтом.
- `lang_index` (int): Индекс языка, используемого на сайте.

**Примеры**
```python
from src.webdriver.driver import Driver
from src.suppliers.wallashop.graber import Graber

# Пример создания экземпляра класса Graber
driver = Driver()
graber = Graber(driver=driver, lang_index=0)
```
