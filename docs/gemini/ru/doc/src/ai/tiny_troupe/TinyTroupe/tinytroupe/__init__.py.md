## Импорты

Модуль импортирует следующие библиотеки:

- `os`: Для взаимодействия с операционной системой.
- `logging`: Для настройки логирования.
- `configparser`: Для парсинга конфигурационных файлов.
- `rich`: Для красивого вывода в консоль.
- `rich.jupyter`: Для интеграции `rich` с Jupyter Notebook.
- `sys`: Для доступа к параметрам и функциям, специфичным для интерпретатора.

Также добавляется текущий путь в `sys.path` и импортируются утилиты из `tinytroupe.utils`.

## Конфигурация и логирование

###  `config = utils.read_config_file()`

**Описание**:
Считывает конфигурационный файл.

###  `utils.pretty_print_config(config)`

**Описание**:
Выводит конфигурацию в консоль с красивым форматированием.

**Параметры**:
- `config` (configparser.ConfigParser): Конфигурация, которая будет выведена.

###  `utils.start_logger(config)`

**Описание**:
Инициализирует систему логирования на основе конфигурации.

**Параметры**:
- `config` (configparser.ConfigParser): Конфигурация, содержащая параметры логирования.

## Исправление стилей Rich для Jupyter

### `rich.jupyter.JUPYTER_HTML_FORMAT = utils.inject_html_css_style_prefix(rich.jupyter.JUPYTER_HTML_FORMAT, "margin:0px;")`

**Описание**:
Внедряет CSS-стиль для удаления отступов в выводе `rich` в Jupyter Notebook, что позволяет корректно отображать вывод без лишних полей.
