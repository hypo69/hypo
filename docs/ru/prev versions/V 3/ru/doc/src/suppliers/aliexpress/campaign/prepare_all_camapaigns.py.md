# Модуль `prepare_all_camapaigns`

## Обзор

Этот модуль предназначен для запуска процесса подготовки всех рекламных кампаний на AliExpress. Он импортирует необходимые модули и функции для выполнения этой задачи. Основная цель модуля - обеспечить создание или обновление партнерских ссылок для рекламных кампаний, инициируя функцию `process_all_campaigns`.

## Подробней

Модуль `prepare_all_camapaigns` играет роль точки входа для процесса подготовки рекламных кампаний AliExpress. Он использует функцию `process_all_campaigns` из модуля `src.suppliers.aliexpress.campaign` для обработки всех кампаний. В проекте этот модуль важен для автоматизации процесса создания и обновления партнерских ссылок, что необходимо для корректного отслеживания и монетизации рекламных кампаний.
