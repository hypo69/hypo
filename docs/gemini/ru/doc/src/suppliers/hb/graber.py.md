# Модуль `hb.graber`

## Обзор

Модуль `hb.graber` представляет собой класс `Graber`, предназначенный для сбора данных о товарах с сайта `hb.co.il`. Он наследует функциональность от базового класса `Graber` и предоставляет возможность переопределения стандартных методов обработки полей. Модуль также поддерживает использование декораторов для выполнения предварительных действий перед отправкой запросов к вебдрайверу.

## Классы

### `Graber`

**Описание**: Класс `Graber` предназначен для операций захвата данных с сайта `hb.co.il`. Он наследует функциональность от класса `Graber` и предоставляет возможность переопределять методы обработки полей.

**Методы**:

-   `__init__`: Инициализирует экземпляр класса `Graber`.
    
    **Описание**:
    Инициализирует экземпляр класса `Graber` с указанным драйвером, устанавливает префикс поставщика и настраивает локатор для декоратора.

    **Параметры**:
    -   `driver` (`Driver`): Драйвер для управления браузером.

**Константы**:

### `MODE`

**Описание**: Константа `MODE` определяет режим работы (например, `dev` для разработки).

## Импорты

### `MODE`

**Описание**: Константа, определяющая режим работы модуля.

### `Any`

**Описание**: Тип, представляющий любое значение. Используется для типизации.

### `header`

**Описание**: Модуль, предоставляющий функциональность для работы с заголовками.

### `Graber`

**Описание**: Базовый класс для сбора данных, импортированный как `Grbr`.

### `Context`

**Описание**: Класс для хранения контекста выполнения, импортированный из `src.suppliers.graber`.

### `close_pop_up`

**Описание**: Функция для закрытия всплывающих окон, импортированная из `src.suppliers.graber`.

### `Driver`

**Описание**: Класс для управления веб-драйвером, импортированный из `src.webdriver.driver`.

### `logger`

**Описание**: Объект для логирования, импортированный из `src.logger.logger`.
