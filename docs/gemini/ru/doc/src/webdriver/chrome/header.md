# Модуль hypotez/src/webdriver/chrome/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и получения документации из файла `README.MD`. Он инициализирует переменные, необходимые для доступа к проекту и его ресурсам.

## Функции

### `set_project_root`

**Описание**: Функция находит корневую директорию проекта, начиная от директории текущего файла и проходя вверх по дереву директорий.

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или каталогов, используемых для определения корня проекта.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращает путь к директории, где находится текущий скрипт.

**Вызывает исключения**:

- Не вызывает исключений.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта. Инициализируется функцией `set_project_root`.

**Тип**: `Path`


### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженные из файла `settings.json`.

**Тип**: `dict`


### `doc_str`

**Описание**: Строка, содержащая текст документации, загруженная из файла `README.MD`.

**Тип**: `str`


### `__project_name__`

**Описание**: Имя проекта. Получается из словаря `settings`. По умолчанию `'hypotez'`.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта. Получается из словаря `settings`. По умолчанию `''`.

**Тип**: `str`


### `__doc__`

**Описание**: Текст документации. Получается из переменной `doc_str`. По умолчанию `''`.

**Тип**: `str`


### `__details__`

**Описание**: Подробная информация о проекте. По умолчанию `''`.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта. Получается из словаря `settings`. По умолчанию `''`.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права на проект. Получается из словаря `settings`. По умолчанию `''`.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка на страницу для поддержки разработчика. Получается из словаря `settings`. По умолчанию ссылка на страницу поддержки.

**Тип**: `str`


## Обработка ошибок

Функции модуля обрабатывают `FileNotFoundError` и `json.JSONDecodeError` для корректной работы при отсутствии или неправильном формате файлов настроек и документации.
