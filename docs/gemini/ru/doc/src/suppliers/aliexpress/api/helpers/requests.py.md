# Модуль `src.suppliers.aliexpress.api.helpers.requests`

## Обзор

Модуль содержит вспомогательные функции для выполнения запросов к API AliExpress и обработки ответов. Основная функция `api_request` отвечает за выполнение запроса, обработку возможных ошибок и преобразование ответа в удобный формат.

## Функции

### `api_request`

**Описание**: Выполняет запрос к API, обрабатывает ответ и возвращает результат.

**Параметры**:
- `request` (any): Объект запроса, предоставляющий метод `getResponse()` для получения ответа.
- `response_name` (str): Имя ключа в ответе, содержащего полезные данные.
- `attemps` (int, optional): Количество попыток выполнения запроса. По умолчанию `1`.

**Возвращает**:
- `any | None`: Возвращает результат из ответа API, если запрос выполнен успешно и код ответа равен 200, или `None` в случае ошибки или некорректного кода ответа.

**Вызывает исключения**:
- `ApiRequestException`: Возникает при ошибке запроса.
- `ApiRequestResponseException`: Возникает при ошибке обработки ответа.
