# Задача No 271
## Суммы остатков
## Условия

Рассмотрим сравнение:

$$n^3 \equiv 1 \pmod {p}$$

где $p$ - простое число.

Пусть $S(p)$ - множество решений $n$ в интервале $0 < n < p$.
Например, для $p = 7$, решениями будут $n=1, 2, 4$. Следовательно, $S(7) = \{1, 2, 4\}$.
Сумма элементов в $S(7)$ равна 1 + 2 + 4 = 7.

Пусть $N(p)$ - сумма элементов в $S(p)$.
Рассмотрим $N(p)$ для всех простых чисел $p$, где $5 \le p \le 100$.

Сумма $N(p)$ для этих простых чисел равна 4171.

Найдите сумму $N(p)$ для всех простых чисел $p$, где $5 \le p < 10^7$.
