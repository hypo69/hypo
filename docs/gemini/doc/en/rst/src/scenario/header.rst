.. module:: hypotez.src.scenario.header
   :platform: Windows, Unix
   :synopsis: This module contains functions for setting the project root and loading settings.

Module Description
------------------

This module defines a function `set_project_root` that finds the root directory of the project. It reads settings from a JSON file and provides access to various project metadata such as name, version, documentation, author, copyright, and a developer coffee link.
