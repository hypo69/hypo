# Модуль _experiments

## Обзор

Модуль `_experiments` предназначен для проведения экспериментов, связанных с поставщиком `visualdg` в проекте `hypotez`. Он содержит вспомогательные функции и настройки, необходимые для тестирования и отладки различных аспектов работы с поставщиками, продуктами и категориями.

## Подробней

Этот модуль используется для быстрого прототипирования и тестирования изменений, связанных с интеграцией поставщиков, обработки данных о продуктах и категориях. Он предоставляет возможность запуска поставщика с определенными параметрами, что упрощает процесс отладки и экспериментов.

## Классы

В данном модуле классы не определены.

## Функции

### `start_supplier`

```python
def start_supplier(supplier_prefix: str = 'aliexpress', locale: str = 'en') -> Supplier:
    """ Старт поставщика """
```

**Назначение**: Функция `start_supplier` создает и возвращает экземпляр класса `Supplier` с заданными параметрами.

**Параметры**:

-   `supplier_prefix` (str, optional): Префикс поставщика. По умолчанию `'aliexpress'`.
-   `locale` (str, optional): Локаль поставщика. По умолчанию `'en'`.

**Возвращает**:

-   `Supplier`: Экземпляр класса `Supplier`, сконфигурированный с переданными параметрами.

**Как работает функция**:

1.  Формирует словарь `params` с параметрами `supplier_prefix` и `locale`.
2.  Создает экземпляр класса `Supplier` с использованием словаря `params` в качестве аргументов.
3.  Возвращает созданный экземпляр класса `Supplier`.

**ASCII flowchart**:

```
Начало
  ↓
Создание словаря params = {supplier_prefix, locale}
  ↓
Создание экземпляра Supplier с параметрами из params
  ↓
Возврат экземпляра Supplier
  ↓
Конец
```

**Примеры**:

```python
from src.suppliers import Supplier  # Предполагается, что Supplier импортирован

# Пример 1: Запуск поставщика с префиксом 'aliexpress' и локалью 'en'
supplier = start_supplier(supplier_prefix='aliexpress', locale='en')
print(type(supplier))  # Вывод: <class 'src.suppliers.Supplier'>

# Пример 2: Запуск поставщика только с указанием префикса, локаль по умолчанию 'en'
supplier = start_supplier(supplier_prefix='amazon')
print(supplier.locale)  # Вывод: en
