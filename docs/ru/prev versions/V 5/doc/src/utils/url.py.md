# Модуль для работы с URL

## Обзор

Модуль `src.utils.string.url` предоставляет набор функций для работы с URL-адресами. Он включает в себя функции для извлечения параметров запроса из URL, проверки URL на валидность и сокращения длинных URL-адресов с использованием сервиса TinyURL.

## Подробней

Этот модуль предназначен для упрощения работы с URL-адресами в проекте `hypotez`. Он предоставляет удобные инструменты для выполнения общих задач, таких как извлечение параметров и проверка валидности URL. Использование этого модуля позволяет избежать повторной реализации этих функций в разных частях проекта, обеспечивая консистентность и упрощая поддержку кода.

## Функции

### `extract_url_params`

```python
def extract_url_params(url: str) -> dict | None:
    """ Извлекает параметры из строки URL.

    Args:
        url (str): Строка URL для парсинга.

    Returns:
        dict | None: Словарь параметров запроса и их значений или `None`, если URL не содержит параметров.
    """
```

**Описание**: Извлекает параметры запроса из переданной URL-строки.

**Как работает функция**:
1. Использует `urlparse` для разбора URL на составные части.
2. Извлекает строку запроса из URL.
3. Использует `parse_qs` для преобразования строки запроса в словарь параметров.
4. Если параметры присутствуют, преобразует значения параметров из списка в строку, если параметр имеет только одно значение.
5. Возвращает словарь параметров или `None`, если URL не содержит параметров.

**Параметры**:
- `url` (str): URL-строка, из которой нужно извлечь параметры.

**Возвращает**:
- `dict | None`: Словарь, содержащий параметры запроса и их значения, или `None`, если URL не содержит параметров.

**Примеры**:

```python
url = "https://example.com?param1=value1&param2=value2"
params = extract_url_params(url)
print(params)  # Вывод: {'param1': 'value1', 'param2': 'value2'}

url = "https://example.com"
params = extract_url_params(url)
print(params)  # Вывод: None
```

### `is_url`

```python
def is_url(text: str) -> bool:
    """ Проверяет, является ли переданный текст валидным URL с использованием библиотеки validators.

    Args:
        text (str): Строка для проверки.

    Returns:
        bool: `True` если строка является валидным URL, иначе `False`.
    """
```

**Описание**: Проверяет, является ли переданная строка валидным URL-адресом.

**Как работает функция**:
1. Использует функцию `validators.url` для проверки, является ли переданная строка валидным URL.
2. Возвращает `True`, если строка является валидным URL, и `False` в противном случае.

**Параметры**:
- `text` (str): Строка, которую необходимо проверить на валидность URL.

**Возвращает**:
- `bool`: `True`, если строка является валидным URL, иначе `False`.

**Примеры**:

```python
url = "https://example.com"
is_valid = is_url(url)
print(is_valid)  # Вывод: True

url = "not a url"
is_valid = is_url(url)
print(is_valid)  # Вывод: False
```

### `url_shortener`

```python
def url_shortener(long_url: str) -> str | None:
    """ Сокращает длинный URL с использованием сервиса TinyURL.

    Args:
        long_url (str): Длинный URL для сокращения.

    Returns:
        str | None: Сокращённый URL или `None`, если произошла ошибка.
    """
```

**Описание**: Сокращает длинный URL, используя сервис TinyURL.

**Как работает функция**:
1. Формирует URL для запроса к API TinyURL, передавая длинный URL в качестве параметра.
2. Отправляет GET-запрос к API TinyURL.
3. Если запрос успешен (код состояния 200), возвращает сокращенный URL из ответа.
4. В случае ошибки возвращает `None`.

**Параметры**:
- `long_url` (str): Длинный URL, который нужно сократить.

**Возвращает**:
- `str | None`: Сокращенный URL или `None`, если произошла ошибка.

**Примеры**:

```python
long_url = "https://example.com/very/long/url/to/shorten"
short_url = url_shortener(long_url)
if short_url:
    print(f"Сокращенный URL: {short_url}")
else:
    print("Ошибка при сокращении URL.")
