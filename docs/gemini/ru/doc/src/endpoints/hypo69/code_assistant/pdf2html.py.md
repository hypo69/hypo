## Функции

### `pdf2html`

**Описание**: 
Конвертирует PDF-файл в HTML-файл, используя утилиту `PDFUtils`.

**Параметры**:
- `pdf_file` (str): Путь к исходному PDF-файлу.
- `html_file` (str): Путь к результирующему HTML-файлу.

**Возвращает**:
- `None`: Функция ничего не возвращает явно.
