# Модуль `header.py`

## Обзор

Модуль `header.py` содержит общие настройки и переменные для проекта, включая определение корневой директории, загрузку настроек из `settings.json`, чтение документации из `README.md` и установку основных метаданных проекта.

## Функции

### `set_project_root`

**Описание**:
Функция `set_project_root` определяет корневую директорию проекта, начиная поиск от директории текущего файла. Поиск останавливается при обнаружении одного из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple, optional): Кортеж имен файлов или директорий, которые используются для идентификации корневой директории проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращает путь к директории, в которой находится текущий скрипт.

## Переменные

### `MODE`

**Описание**:
Режим работы приложения. По умолчанию установлено значение `'dev'`.

### `__root__`

**Описание**:
Путь к корневой директории проекта. Определяется с помощью функции `set_project_root`.

### `settings`

**Описание**:
Словарь, содержащий настройки проекта, загруженные из файла `settings.json`.

### `doc_str`

**Описание**:
Строка, содержащая содержимое файла `README.md`.

### `__project_name__`

**Описание**:
Название проекта, взятое из настроек или установлено значение по умолчанию `'hypotez'`.

### `__version__`

**Описание**:
Версия проекта, взятая из настроек или установлена пустая строка `''`.

### `__doc__`

**Описание**:
Содержимое документации проекта, взятое из файла `README.md` или установлено пустой строкой `''`.

### `__details__`

**Описание**:
Детали проекта. В данный момент установлена пустая строка `''`.

### `__author__`

**Описание**:
Автор проекта, взятый из настроек или установлена пустая строка `''`.

### `__copyright__`

**Описание**:
Авторские права, взятые из настроек или установлена пустая строка `''`.

### `__cofee__`

**Описание**:
Сообщение с предложением поддержать разработчика чашечкой кофе, взятое из настроек или установлено значение по умолчанию: `"Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69"`
