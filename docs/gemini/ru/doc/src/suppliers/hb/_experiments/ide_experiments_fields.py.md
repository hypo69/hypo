# Модуль ide_experiments_fields.py

## Обзор

Модуль предназначен для сбора и обработки данных о товарах с сайта поставщика HB (hbdeadsea.co.il) с целью их дальнейшей передачи в систему PrestaShop. Он включает в себя функции для извлечения информации о товарах со страниц категорий и страниц отдельных товаров, а также для нормализации и подготовки этих данных для импорта в PrestaShop.

## Подробнее

Модуль выполняет следующие основные задачи:

1.  **Сбор данных о товарах**: Извлекает информацию о товарах, такую как название, описание, цена, состав, изображения и другие характеристики, непосредственно со страниц сайта HB с использованием Selenium WebDriver.
2.  **Нормализация данных**: Приводит извлеченные данные к единому формату, удаляет лишние символы и форматирует строки для соответствия требованиям PrestaShop.
3.  **Подготовка данных для PrestaShop**: Формирует словари `presta_fields_dict` и `assist_fields_dict`, содержащие данные о товарах, которые могут быть переданы в API PrestaShop для создания или обновления товаров.
4.  **Автоматизация работы с веб-сайтом**: Использует Selenium WebDriver для автоматического перехода по страницам категорий и товаров, закрытия баннеров и выполнения других действий, необходимых для сбора данных.

Модуль предназначен для использования в рамках процесса автоматизированного импорта данных о товарах от поставщика HB в систему PrestaShop.

## Классы

### `grab_product_page`

**Описание**: Собирает со страницы товара значения веб-элементов и преобразует их в поля ProductFields.

**Параметры**:

*   `supplier` (Supplier): Класс поставщика. Веб-драйвер должен быть установлен на странице товара. В учетной записи должна быть видна линейка "Affiliate links", из которой берется информация о партнерской ссылке.
*   `async_run` (bool, optional): Флаг для асинхронного запуска. По умолчанию `True`.

**Возвращает**:

*   `ProductFields`: Объект класса `ProductFields`, содержащий собранные и обработанные данные о товаре.

**Как работает функция**:

1.  Инициализация глобальных переменных: Устанавливает глобальные переменные `s` (Supplier), `p` (Product), `f` (ProductFields), `d` (Driver) и `l` (locator).
2.  Закрытие баннера: Закрывает баннер на странице товара с помощью метода `execute_locator` драйвера.
3.  Прокрутка страницы: Прокручивает страницу товара для загрузки элементов, подгружаемых через AJAX.
4.  Вызов функций, специфичных для поставщика: Вызывает внутренние функции для извлечения и обработки определенных полей товара.
5.  Установка ссылок: Вызывает функцию `set_references` для установки идентификаторов товара.
6.  Извлечение дополнительных полей: Вызывает функции для извлечения дополнительных полей товара, таких как стоимость доставки, короткая партнерская ссылка и т.д.
7.  Извлечение URL изображений: Извлекает URL дополнительных изображений товара.
8.  Извлечение имени и ссылки товара: Извлекает название товара и формирует ссылку на товар.
9.  Возврат объекта ProductFields: Возвращает объект `ProductFields`, содержащий все собранные и обработанные данные о товаре.

**Внутренние функции**:

#### `product_reference_and_volume_and_price_for_100`

**Назначение**: Извлекает объем, артикул поставщика и цену за единицу товара.

**Параметры**: Нет

**Возвращает**: Нет

**Как работает функция**:

1.  Извлекает список веб-элементов, содержащих информацию об объеме, артикуле и цене за 100 мл.
2.  Перебирает веб-элементы и извлекает соответствующие значения.
3.  Записывает извлеченные значения в поля объекта `f` (ProductFields).

#### `set_references`

**Назначение**: Устанавливает идентификаторы товара.

**Параметры**:

*   `f` (ProductFields): Объект класса ProductFields.
*   `s` (Supplier): Объект класса Supplier.

**Возвращает**: Нет

**Как работает функция**:

1.  Устанавливает `id_supplier` из объекта `s.supplier_id`.
2.  Формирует `reference` товара, объединяя `s.supplier_id` и `f.supplier_reference`.

**ASCII flowchart**:

```
    Начало
    │
    ├── Вызов product_reference_and_volume_and_price_for_100()
    │
    ├── Вызов set_references(f, s)
    │
    ├── Вызов field_additional_shipping_cost()
    │
    ├── Вызов field_affiliate_short_link()
    │
    ├── Вызов field_available_for_order()
    │
    ├── Извлечение изображений _images_urls
    │
    ├── Вызов field_description()
    │
    ├── Вызов field_how_to_use()
    │
    ├── Вызов field_id_category_default()
    │
    ├── Вызов field_id_manufacturer()
    │
    ├── Вызов field_ingredients()
    │
    ├── Извлечение name
    │
    └── Конец
```

**Примеры**:

```python
s: Supplier = Supplier(supplier_prefix='hb')
product_fields: ProductFields = grab_product_page(s)
print(product_fields.presta_fields_dict)
```
