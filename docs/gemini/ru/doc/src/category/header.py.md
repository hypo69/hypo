# src.category.header

## Обзор

Модуль `header.py` определяет корневой путь к проекту. Все импорты в проекте строятся относительно этого пути.

## Функции

### `set_project_root`

**Описание**:
Находит корневой каталог проекта, начиная с каталога текущего файла, двигаясь вверх по структуре каталогов, пока не будет найдена папка, содержащая один из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple): Список файлов или каталогов, наличие которых определяет корневой каталог проекта.

**Возвращает**:
- `Path`: Путь к корневому каталогу, если он найден, иначе — путь к каталогу, где расположен скрипт.

## Переменные

### `__root__`
**Описание**:
Путь к корневому каталогу проекта.

### `settings`
**Описание**:
Словарь с настройками проекта, загруженный из `settings.json`. Если загрузка не удалась, значение равно `None`.

### `doc_str`
**Описание**:
Строка с содержимым файла `README.MD`. Если чтение не удалось, значение равно `None`.

### `__project_name__`
**Описание**:
Имя проекта, извлеченное из `settings.json` или значение по умолчанию `'hypotez'`, если `settings.json` не загружен.

### `__version__`
**Описание**:
Версия проекта, извлеченная из `settings.json` или пустая строка, если `settings.json` не загружен.

### `__doc__`
**Описание**:
Содержимое файла `README.MD`, или пустая строка, если чтение не удалось.

### `__details__`
**Описание**:
Строка с дополнительной информацией о проекте (в текущей версии всегда пустая).

### `__author__`
**Описание**:
Автор проекта, извлеченный из `settings.json` или пустая строка, если `settings.json` не загружен.

### `__copyright__`
**Описание**:
Информация об авторских правах, извлеченная из `settings.json` или пустая строка, если `settings.json` не загружен.

### `__cofee__`
**Описание**:
Строка с информацией о том, как можно поддержать разработчика чашкой кофе, извлеченная из `settings.json` или значение по умолчанию.
