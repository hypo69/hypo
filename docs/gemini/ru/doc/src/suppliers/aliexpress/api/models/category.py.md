# Модуль: src.suppliers.aliexpress.api.models

## Обзор

Модуль содержит классы для представления категорий товаров AliExpress.  Классы предназначены для хранения информации об идентификаторах и наименованиях категорий, а также для представления дочерних категорий с указанием идентификатора родительской категории.

## Подробнее

Этот модуль предоставляет структуры данных для работы с категориями товаров, получаемыми через API AliExpress. Он определяет два класса: `Category` и `ChildCategory`. `Category` содержит базовую информацию о категории, а `ChildCategory` наследует эту информацию и добавляет идентификатор родительской категории.

## Классы

### `Category`

**Описание**: Базовый класс для представления категории товара.

**Принцип работы**:
Класс `Category` предназначен для хранения идентификатора и наименования категории. Он служит базовым классом для класса `ChildCategory`.

**Атрибуты**:

-   `category_id` (int): Идентификатор категории.
-   `category_name` (str): Наименование категории.

### `ChildCategory`

**Описание**: Класс для представления дочерней категории товара.

**Принцип работы**:
Класс `ChildCategory` наследует атрибуты класса `Category` и добавляет атрибут, указывающий на идентификатор родительской категории.

**Наследует**:

-   `Category`: Наследует атрибуты `category_id` и `category_name` от класса `Category`.

**Атрибуты**:

-   `parent_category_id` (int): Идентификатор родительской категории.
-   `category_id` (int): Идентификатор категории.
-   `category_name` (str): Наименование категории.
