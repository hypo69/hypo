# Задача No 443
## Гигантское НОК

## Условия
Пусть $L(n)$ обозначает наименьшее общее кратное первых $n$ натуральных чисел, то есть $L(n) = \text{lcm}(1, 2, 3, ..., n)$.

Известно, что $L(1) = 1$, $L(2) = 2$, $L(3) = 6$, $L(4) = 12$, $L(5) = 60$.

Пусть $A(n) = L(n+1) / L(n)$.

Известно, что $A(1) = 2$, $A(2) = 3$, $A(3) = 2$, $A(4) = 5$, $A(5) = 1$.

Пусть $S(n) = \sum_{i=1}^n A(i)$.

Известно, что $S(5) = 13$.

Найдите $S(10^6)$.
