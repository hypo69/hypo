# Модуль `xls`

## Обзор

Модуль `xls` предоставляет функции для конвертации `.xls` файлов в формат словаря (`dict`). В основном, он использует функции из модуля `src.utils.xls` для чтения данных из `.xls` файлов.

## Подробней

Модуль предназначен для упрощения процесса извлечения данных из `.xls` файлов и представления их в удобном формате словаря. Это может быть полезно для дальнейшей обработки данных, анализа или интеграции с другими частями проекта.

## Функции

### `xls2dict`

```python
def xls2dict(xls_file: str | Path) -> dict | None:
    """
    Args:
        xls_file (str | Path): Путь к .xls файлу.

    Returns:
        dict | None: Словарь, содержащий данные из .xls файла, или None в случае ошибки.
    """
```

**Описание**: Преобразует `.xls` файл в словарь.

**Параметры**:

- `xls_file` (str | Path): Путь к `.xls` файлу, который требуется преобразовать.

**Возвращает**:

- `dict | None`: Словарь, содержащий данные из `.xls` файла. Возвращает `None`, если произошла ошибка при чтении файла.

**Примеры**:

```python
from pathlib import Path
xls_file = Path('example.xls')
data = xls2dict(xls_file)
if data:
    print(data)
