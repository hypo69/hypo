# Модуль `hypotez/src/suppliers/aliexpress/gui/category.py`

## Обзор

Этот модуль предоставляет класс `CategoryEditor`, который служит графическим интерфейсом для подготовки рекламных кампаний на AliExpress.  Класс взаимодействует с данными кампаний, загруженными из JSON-файлов, и предоставляет функции для подготовки категорий кампаний.

## Класс `CategoryEditor`

### Описание

Класс `CategoryEditor` представляет собой окно приложения, предназначенное для редактирования и подготовки категорий рекламных кампаний AliExpress. Он загружает данные из JSON-файла, отображает информацию о кампании и категориях, и предоставляет кнопки для подготовки всех категорий или отдельной.

### Методы

#### `__init__(self, parent=None, main_app=None)`

**Описание**: Инициализирует окно редактора категорий.

**Параметры**:
* `parent` (QtWidgets.QWidget, необязательно): Родительское окно.
* `main_app` (объект MainApp, необязательно): Ссылка на главное приложение.

**Возвращает**:
- None


#### `setup_ui(self)`

**Описание**: Настраивает пользовательский интерфейс (UI).

**Параметры**:
- Нет

**Возвращает**:
- None


#### `setup_connections(self)`

**Описание**: Устанавливает связи между элементами UI.

**Параметры**:
- Нет

**Возвращает**:
- None


#### `open_file(self)`

**Описание**: Открывает диалог выбора файла JSON для загрузки данных кампании.

**Параметры**:
- Нет

**Возвращает**:
- None


#### `load_file(self, campaign_file)`

**Описание**: Загружает данные кампании из указанного JSON-файла.

**Параметры**:
* `campaign_file` (str): Путь к файлу JSON.

**Возвращает**:
- None

**Возможные исключения**:
* `Exception`: При ошибке загрузки JSON-файла выводится сообщение об ошибке.

**Описание**: Создает виджеты, отображающие данные из загруженного JSON-файла.

**Параметры**:

**Возвращает**:
- None

**Описание**: Асинхронно подготавливает все категории в загруженной кампании.

- Нет

**Возвращает**:
- None

**Возможные исключения**:

#### `prepare_category_async(self)`

**Параметры**:
