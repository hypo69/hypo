# Модуль hypotez/src/suppliers/bangood/header.py

## Обзор

Этот модуль предоставляет функции для получения и работы с настройками проекта, корневым каталогом и документацией. Он находит корневой каталог проекта, загружает настройки из файла `settings.json` и загружает строку документации из файла `README.MD`.

## Функции

### `set_project_root`

**Описание**: Определяет корневой каталог проекта, начиная с текущей директории и двигаясь вверх по иерархии директорий, пока не найдёт директорию, содержащую один из файлов-маркеров (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж из строк, представляющих имена файлов или директорий, которые используются для определения корневого каталога проекта.

**Возвращает**:

- `Path`: Путь до корневого каталога проекта, если он найден. В противном случае возвращает путь к текущей директории, где находится скрипт.

**Вызывает исключения**:

- Нет


###  `get_settings`


**Описание**: Функция для получения настроек проекта из файла `settings.json` находящегося в корневой директории проекта.

**Возвращает**:
- `dict`: Словарь настроек проекта.
- `None`: Если файл настроек не найден или не удалось его прочитать.


### `get_documentation`

**Описание**: Функция для загрузки документации проекта из файла `README.MD` находящегося в корневой директории проекта.


**Возвращает**:
- `str`: Строка документации проекта.
- `None`: Если файл документации не найден или не удалось его прочитать.


## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта, полученный с помощью функции `set_project_root`.

**Тип**: `Path`


### `__project_name__`

**Описание**: Название проекта, полученное из файла `settings.json`. По умолчанию `hypotez`.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта, полученная из файла `settings.json`. По умолчанию пустая строка.

**Тип**: `str`


### `__doc__`

**Описание**: Документация проекта, полученная из файла `README.MD`. По умолчанию пустая строка.

**Тип**: `str`


### `__details__`

**Описание**: Подробное описание проекта. По умолчанию пустая строка.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта, полученный из файла `settings.json`. По умолчанию пустая строка.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права проекта, полученные из файла `settings.json`. По умолчанию пустая строка.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка на помощь разработчику. По умолчанию ссылка на Boosty.to.

**Тип**: `str`


##  Примечание

- Модуль использует `gs.path.root`, предположительно, это функция из другого модуля, отвечающая за получение корневой директории проекта.
- Обработка исключений `FileNotFoundError` и `json.JSONDecodeError` предотвращает сбой программы при отсутствии или повреждении файлов.
- Использование `Optional` и типов данных (например, `dict | None`) улучшает читаемость и безопасность кода.
