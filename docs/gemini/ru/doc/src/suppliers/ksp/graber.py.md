# Модуль `graber.py`

## Обзор

Модуль `graber.py` предназначен для сбора информации о товарах с сайта `ksp.co.il`. Он содержит класс `Graber`, который наследуется от класса `Graber` (возможно, из модуля `src.suppliers.graber`). Класс `Graber` переопределяет методы родительского класса для обработки специфических полей и элементов на страницах товаров `ksp.co.il`.

## Подробней

Этот модуль разработан для автоматизации процесса сбора данных о товарах с веб-сайта `ksp.co.il`. Он использует веб-драйвер для взаимодействия со страницами сайта и извлекает необходимую информацию о товарах. Для обработки различных полей на странице товара используются функции, которые могут быть переопределены в классе `Graber` для нестандартной обработки. Модуль также предоставляет возможность выполнения предварительных действий перед отправкой запроса к веб-драйверу с использованием декораторов.
