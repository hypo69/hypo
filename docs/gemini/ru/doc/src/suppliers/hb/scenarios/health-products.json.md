# Документация для `health-products.json`

## Обзор

Данный файл `health-products.json` содержит конфигурацию для сценария обработки товаров категории "מוצרי בריאות" (товары для здоровья) с сайта hbdeadsea.co.il. Он определяет URL страницы, название категории, состояние товаров и категории PrestaShop, к которым следует отнести товары.

## Структура файла

### `scenarios`
**Описание**: Объект, содержащий настройки для сценария обработки товаров.

#### `url`
**Описание**: URL страницы, с которой следует собирать данные о товарах.

**Тип**: `string`

**Значение**: `"https://hbdeadsea.co.il/product-category/health-products/"`

#### `name`
**Описание**: Название категории товаров.

**Тип**: `string`

**Значение**: `"מוצרי בריאות"`

#### `condition`
**Описание**: Состояние товаров (например, "new" для новых товаров).

**Тип**: `string`

**Значение**: `"new"`

#### `presta_categories`
**Описание**: Объект, содержащий настройки для категорий PrestaShop.

##### `default_category`
**Описание**: ID категории по умолчанию в PrestaShop, к которой следует отнести товары.

**Тип**: `integer`

**Значение**: `11111`

##### `additional_categories`
**Описание**: Массив дополнительных категорий PrestaShop, к которым следует отнести товары. В данном случае, массив пустой.

**Тип**: `array`

**Значение**: `[""]`
