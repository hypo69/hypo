# Задача No 61
## Циклические числа-многоугольники
## Условия
Треугольные, квадратные, пятиугольные, шестиугольные, семиугольные и восьмиугольные числа генерируются по следующим формулам:
    
|Тип числа|Формула|
|--|--|
|Треугольные|P<sub>3,n</sub>=n(n+1)/2|
|Квадратные|P<sub>4,n</sub>=n<sup>2</sup>|
|Пятиугольные|P<sub>5,n</sub>=n(3n-1)/2|
|Шестиугольные|P<sub>6,n</sub>=n(2n-1)|
|Семиугольные|P<sub>7,n</sub>=n(5n-3)/2|
|Восьмиугольные|P<sub>8,n</sub>=n(3n-2)|
 
Наборы из четырехзначных чисел, где каждое число является треугольным, квадратным, пятиугольным, шестиугольным, семиугольным и восьмиугольным, а также являются циклическими (т.е. последние два разряда каждого числа совпадают с первыми двумя разрядами следующего числа) можно сформировать.

Например, набор 8128, 2882, 8281 – имеет три элемента, каждый из которых является циклическим с предыдущим числом.

Найдите единственный набор из шести четырехзначных чисел, для которого каждое число является треугольным, квадратным, пятиугольным, шестиугольным, семиугольным и восьмиугольным числом, и который также является циклическим.

Найдите сумму чисел в этом наборе.
