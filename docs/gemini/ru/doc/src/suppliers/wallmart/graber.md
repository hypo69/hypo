# Модуль hypotez/src/suppliers/wallmart/graber.py

## Обзор

Модуль `graber.py` содержит класс `Graber`, предназначенный для сбора данных о товарах с сайта walmart.com. Класс наследуется от базового класса `Grbr` и реализует методы для извлечения различных полей товара, таких как цена, название, описание и другие.  Класс предоставляет асинхронные методы для обработки данных с веб-страницы.  Модуль использует веб-драйвер и предоставляет механизм обработки всплывающих окон.
