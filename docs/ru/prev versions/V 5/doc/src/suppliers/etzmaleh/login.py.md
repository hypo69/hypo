# Модуль `login`

## Обзор

Модуль `login.py` предназначен для выполнения авторизации поставщика Etzmaleh. Он содержит функцию `login`, которая принимает объект поставщика и возвращает `True`, если авторизация прошла успешно, и `False` в противном случае.

## Подробней

Этот модуль играет важную роль в процессе сбора данных с сайта поставщика Etzmaleh, обеспечивая вход в систему для последующего извлечения информации. Он использует веб-драйвер для взаимодействия с сайтом.
`login` является частью подсистемы `suppliers` проекта `hypotez` и отвечает за начальный этап взаимодействия с поставщиком Etzmaleh.

## Функции

### `login`

```python
def login(s) -> bool:
    """ Функция логин. 
   @param
        s - Supplier
    @returns
        True if login else False

   """
```

**Как работает функция**:

Функция `login` принимает объект поставщика `s` в качестве аргумента. Внутри функции происходит логирование успешной авторизации с использованием `logger.info`. На данный момент функция всегда возвращает `True`, предполагая успешную авторизацию.

**Параметры**:

- `s`: Объект поставщика (Supplier), для которого выполняется вход в систему.

**Возвращает**:

- `bool`: `True`, если вход в систему выполнен успешно, `False` в противном случае. В текущей реализации всегда возвращает `True`.

**Примеры**:

```python
from src.suppliers.etzmaleh.login import login

class Supplier:
    pass

s = Supplier()
result = login(s)
print(result)  # Вывод: True
