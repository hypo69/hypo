# Модуль `switch_account`

## Обзор

Модуль предназначен для переключения между аккаунтами в Facebook с использованием веб-драйвера. Он содержит функцию `switch_account`, которая использует локаторы для поиска и нажатия кнопки переключения аккаунта.

## Подробнее

Данный модуль является частью системы автоматизации для работы с Facebook. Он использует веб-драйвер для имитации действий пользователя в браузере, в частности, для переключения между разными аккаунтами. Файл `post_message.json` содержит локаторы элементов интерфейса, необходимых для навигации и взаимодействия со страницей Facebook.

## Функции

### `switch_account`

```python
def switch_account(driver: Driver):
    """ Если есть кнопка `Переключить` - нажимаю её  """
```

**Назначение**:
Функция выполняет переключение между аккаунтами в Facebook, нажимая на кнопку "Переключить", если она доступна.

**Параметры**:
- `driver` (Driver): Экземпляр веб-драйвера, используемый для взаимодействия с браузером.

**Возвращает**:
- Ничего (None). Функция выполняет действие и ничего не возвращает явно.

**Как работает функция**:

1. Функция принимает объект `driver` типа `Driver`, который представляет собой инстанс веб-драйвера, используемый для управления браузером.
2. Использует метод `execute_locator` объекта `driver` для поиска и нажатия элемента, соответствующего локатору `locator.switch_to_account_button`. Этот локатор определен в файле `post_message.json` и указывает на кнопку переключения аккаунта.

**ASCII flowchart**:

```
A: Получение driver
|
B: Выполнение локатора switch_to_account_button
|
C: Нажатие на кнопку "Переключить" (если она существует)
```

Где:

- `A`: Получение инстанса веб-драйвера.
- `B`: Выполнение локатора для поиска кнопки переключения аккаунта.
- `C`: Клик по кнопке "Переключить", если элемент найден.

**Примеры**:

```python

from src.endpoints.advertisement.facebook.scenarios.switch_account import switch_account

# Пример использования функции switch_account
driver = Driver(Chrome)  # Создание инстанса драйвера Chrome
switch_account(driver)    # Вызов функции для переключения аккаунта
