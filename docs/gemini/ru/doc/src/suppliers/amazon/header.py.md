## Функции

### `set_project_root`

**Описание**:
Находит корневой каталог проекта, начиная с каталога текущего файла,
ища вверх и останавливаясь на первом каталоге, содержащем любой из маркерных файлов.

**Параметры**:
- `marker_files` (tuple): Имена файлов или каталогов для идентификации корня проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневому каталогу, если он найден, иначе каталог, где расположен скрипт.

## Переменные

### `MODE`

**Описание**:
Режим работы приложения. По умолчанию установлен в `dev`.
### `__root__`

**Описание**:
Путь к корневой директории проекта.

### `settings`

**Описание**:
Словарь с настройками, загруженный из файла `settings.json`. Может быть `None`, если файл не найден или произошла ошибка при чтении.

### `doc_str`

**Описание**:
Содержимое файла `README.MD`. Может быть `None`, если файл не найден или произошла ошибка при чтении.

### `__project_name__`

**Описание**:
Название проекта. Значение берется из `settings.json` или установлено в `hypotez`, если `settings.json` не найден.

### `__version__`

**Описание**:
Версия проекта. Значение берется из `settings.json` или установлено в пустую строку, если `settings.json` не найден.

### `__doc__`

**Описание**:
Описание проекта. Значение берется из файла `README.MD` или установлено в пустую строку, если `README.MD` не найден.

### `__details__`

**Описание**:
Детальная информация о проекте. В данный момент всегда пустая строка.

### `__author__`

**Описание**:
Автор проекта. Значение берется из `settings.json` или установлено в пустую строку, если `settings.json` не найден.

### `__copyright__`

**Описание**:
Информация о копирайте проекта. Значение берется из `settings.json` или установлено в пустую строку, если `settings.json` не найден.

### `__cofee__`

**Описание**:
Сообщение для поддержки разработчика. Значение берется из `settings.json` или установлено в стандартное сообщение, если `settings.json` не найден.
