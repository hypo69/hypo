# Модуль login.py

## Обзор

Модуль `login.py` предназначен для автоматизации процесса входа в систему AliExpress с использованием WebDriver. Он содержит функцию `login`, которая принимает объект поставщика и использует его для выполнения шагов входа, таких как ввод учетных данных и принятие файлов cookie.

## Подробней

Этот модуль является частью проекта `hypotez` и отвечает за автоматизацию входа в AliExpress. Он использует Selenium WebDriver для управления браузером и выполнения действий на странице входа. Модуль предназначен для работы с классом `Supplier`, который предоставляет необходимые данные, такие как драйвер WebDriver и локаторы элементов страницы.
