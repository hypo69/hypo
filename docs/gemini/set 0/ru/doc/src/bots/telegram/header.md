# Модуль hypotez/src/logger/header.py

## Обзор

Модуль `hypotez/src/logger/header.py` определяет корневой путь к проекту `hypotez`.  Все импорты строятся относительно этого пути.  В дальнейшем этот путь планируется перенести в системную переменную.  Модуль также загружает настройки из файла `settings.json` и, при наличии, информацию из файла `README.MD` для использования в других частях проекта.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная от директории текущего файла. Поиск производится вверх по дереву директорий, и останавливается на первой директории, содержащей один из указанных файлов или каталогов.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или каталогов, используемых для определения корневой директории. По умолчанию - ('pyproject.toml', 'requirements.txt', '.git').

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена. В противном случае, возвращает путь к директории, где расположен текущий скрипт.

**Примечания**: Добавляет корневой путь проекта в системный путь (`sys.path`), если его там еще нет.

## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный вызовом функции `set_project_root()`.

**Тип**: `Path`

### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженный из файла `gs.path.root / 'src' / 'settings.json'`.

**Тип**: `dict` или `None`, если файл не найден или некорректно отформатирован.

### `doc_str`

**Описание**: Строка, содержащая информацию из файла `gs.path.root / 'src' / 'README.MD'`.

**Тип**: `str` или `None`, если файл не найден или некорректно отформатирован.

### `__project_name__`

**Описание**: Имя проекта, полученное из настроек `settings`. По умолчанию - `hypotez`.

**Тип**: `str`

### `__version__`

**Описание**: Версия проекта, полученная из настроек `settings`. По умолчанию - пустая строка.

**Тип**: `str`

### `__doc__`

**Описание**: Текстовое описание проекта, полученное из файла `README.MD`. По умолчанию - пустая строка.

**Тип**: `str`

### `__details__`

**Описание**: Дополнительные данные о проекте. По умолчанию - пустая строка.

**Тип**: `str`

### `__author__`

**Описание**: Автор проекта, полученный из настроек `settings`. По умолчанию - пустая строка.

**Тип**: `str`

### `__copyright__`

**Описание**: Авторские права на проект, полученные из настроек `settings`. По умолчанию - пустая строка.

**Тип**: `str`

### `__cofee__`

**Описание**: Ссылка на страницу для поддержки разработчика чашкой кофе.

**Тип**: `str`
