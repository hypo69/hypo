# Модуль для авторизации поставщика HB

## Обзор

Модуль `login.py` предназначен для авторизации поставщика HB. Он содержит функцию `login`, которая выполняет вход в систему поставщика.

## Подробнее

Этот модуль является частью подсистемы работы с поставщиками проекта `hypotez`. Функция `login` используется для автоматизации процесса входа в систему HB.
