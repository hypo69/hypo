# Задача No 817
## Реконструируя матрицу

## Условия
Рассмотрим целочисленную матрицу A размера m x n. Предположим, что все элементы A известны.
Теперь возьмем другую матрицу B того же размера, что и A. Сначала B заполнена нулями.

Процедура вычисления значений B заключается в итерации по всем парам (r,c), где 1 ≤ r ≤ m и 1 ≤ c ≤ n.

Для каждой пары (r,c), мы находим все элементы A(i,j), такие что 1 ≤ i ≤ r и 1 ≤ j ≤ c. Затем мы вычисляем сумму всех этих элементов. Обозначим эту сумму как S. Затем мы присваиваем B(r,c) = S.

Например, пусть A является матрицей 2x3:
```
  1  2  3
  4  5  6
```
Тогда B вычисляется следующим образом:
```
B(1,1) = 1
B(1,2) = 1 + 2 = 3
B(1,3) = 1 + 2 + 3 = 6
B(2,1) = 1 + 4 = 5
B(2,2) = 1 + 2 + 4 + 5 = 12
B(2,3) = 1 + 2 + 3 + 4 + 5 + 6 = 21
```
Таким образом, матрица B будет:
```
 1  3  6
 5 12 21
```

Вместо того, чтобы вычислять B из A, мы теперь рассмотрим обратную задачу: зная B, восстановить A. В предыдущем примере, зная матрицу B, мы можем восстановить матрицу A, следующим образом:
```
A(1,1) = B(1,1) = 1
A(1,2) = B(1,2) - B(1,1) = 3 - 1 = 2
A(1,3) = B(1,3) - B(1,2) = 6 - 3 = 3
A(2,1) = B(2,1) - B(1,1) = 5 - 1 = 4
A(2,2) = B(2,2) - B(1,2) - B(2,1) + B(1,1) = 12 - 3 - 5 + 1 = 5
A(2,3) = B(2,3) - B(1,3) - B(2,2) + B(1,2) = 21 - 6 - 12 + 3 = 6
```

Теперь рассмотрим матрицу C размером 5x5, где C вычисляется таким же образом, как B из A. Элементы C определены следующим образом:
```
C(r,c) = r^3 + c^3 + rc
```
Например, C(1,1) = 1^3 + 1^3 + 1 * 1 = 3 и C(2,3) = 2^3 + 3^3 + 2 * 3 = 8 + 27 + 6 = 41.

Пусть S — сумма всех элементов в матрице A, которую мы можем реконструировать из C.
Найдите S.
