# Модуль `src.goog`

## Обзор

Модуль `src.goog` предоставляет функциональность для работы с Google Spreadsheets.

## Оглавление

1. [Классы](#классы)
    - [`SpreadSheet`](#spreadsheet)

## Классы

### `SpreadSheet`

**Описание**:
Класс `SpreadSheet` предоставляет интерфейс для взаимодействия с Google Spreadsheets. Подробнее о функциональности класса можно найти в документации к модулю `spreadsheet`.

## Импортированные модули

Модуль импортирует класс `SpreadSheet` из модуля `src.goog.spreadsheet`.
