# Модуль hypotez/src/goog/text_to_speech/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и документации из файла `README.MD`. Он также определяет переменные, содержащие информацию о проекте, такие как имя, версия, описание, автор и информацию о поддержке.

## Функции

### `set_project_root`

**Описание**: Определяет корневую директорию проекта, начиная с директории текущего файла и поднимаясь вверх по дереву директорий, пока не найдет директорию, содержащую файлы маркеров (pyproject.toml, requirements.txt, .git).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, которые используются для определения корневой директории проекта. По умолчанию содержит ('pyproject.toml', 'requirements.txt', '.git').

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращает директорию, где расположен текущий файл.


**Вызывает исключения**:

- Нет


### Загрузка настроек и документации

**Описание**: Загружает настройки из файла `settings.json` и документацию из файла `README.MD`.

**Параметры**:

- Нет

**Возвращает**:

- Нет


**Вызывает исключения**:

- `FileNotFoundError`: Возникает, если файл `settings.json` или `README.MD` не найден.
- `json.JSONDecodeError`: Возникает, если содержимое файла `settings.json` не является валидным JSON.

## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта.

**Тип**: `Path`

### `settings`

**Описание**: Словарь настроек из файла `settings.json`.

**Тип**: `dict`

### `doc_str`

**Описание**: Строка с документацией из файла `README.MD`.

**Тип**: `str`

### `__project_name__`

**Описание**: Название проекта. Значение взято из настроек, если они доступны, иначе установлено значение `'hypotez'`.

**Тип**: `str`

### `__version__`

**Описание**: Версия проекта. Значение взято из настроек, если они доступны, иначе установлено значение `''`.

**Тип**: `str`

### `__doc__`

**Описание**: Документация проекта. Значение взято из настроек, если они доступны, иначе установлено значение `''`.

**Тип**: `str`

### `__details__`

**Описание**: Подробная информация о проекте.

**Тип**: `str`

### `__author__`

**Описание**: Автор проекта. Значение взято из настроек, если они доступны, иначе установлено значение `''`.

**Тип**: `str`

### `__copyright__`

**Описание**: Авторские права проекта. Значение взято из настроек, если они доступны, иначе установлено значение `''`.

**Тип**: `str`

### `__cofee__`

**Описание**: Ссылка для поддержки разработчика. Значение взято из настроек, если они доступны, иначе установлено значение по умолчанию.

**Тип**: `str`
