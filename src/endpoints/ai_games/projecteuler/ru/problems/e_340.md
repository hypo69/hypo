# Задача No 340
## Странная функция

## Условия
Определим функцию f(n) следующим образом:

* f(n) = n, если n < 3
* f(n) = f(n - 1) + f(n - 2) + f(n - 3), если n ≥ 3

Пусть также g(a, b, c) будет суммой всех различных значений f(n) таких, что a ≤ n ≤ b и f(n) mod 10<sup>9</sup> = c.

Например:
f(0) = 0, f(1) = 1, f(2) = 2, f(3) = 3, f(4) = 6, f(5) = 11, f(6) = 20, f(7) = 37, f(8) = 68, f(9) = 125, f(10) = 230
Таким образом, g(0, 10, 0) = 0, g(0, 10, 2) = 2, g(0, 10, 3) = 3, g(0, 10, 6) = 6, g(0, 10, 11) = 11, g(0, 10, 20) = 20, g(0, 10, 37) = 37, g(0, 10, 68) = 68, g(0, 10, 125) = 125, g(0, 10, 230) = 230.

Пусть S(n) = ∑ g(0, 10<sup>k</sup>, k), где k принимает все значения от 0 до n.

Вычислите S(18) mod 10<sup>9</sup>.
