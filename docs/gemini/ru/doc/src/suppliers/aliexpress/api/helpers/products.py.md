# Модуль `products`

## Обзор

Модуль `products` содержит вспомогательные функции для обработки данных о продуктах, полученных от AliExpress API. Основная задача модуля —  парсинг и приведение данных о продуктах к нужному формату.

## Оглавление

1. [Функции](#функции)
    - [`parse_product`](#parse_product)
    - [`parse_products`](#parse_products)

## Функции

### `parse_product`

**Описание**:
Функция для обработки данных о продукте. Извлекает строку из `product.product_small_image_urls`.

**Параметры**:
- `product`: Объект, содержащий данные о продукте.

**Возвращает**:
- Обработанный объект `product`.

**Пример**
```python
   product = {
        "product_small_image_urls": <bs4.element.Tag>
   }
   parse_product(product)
   product = {
       "product_small_image_urls": 'url'
   }
```

### `parse_products`

**Описание**:
Функция для обработки списка продуктов. Применяет функцию `parse_product` к каждому элементу списка.

**Параметры**:
- `products`: Список объектов, содержащих данные о продуктах.

**Возвращает**:
- `list`: Новый список обработанных объектов продуктов.
