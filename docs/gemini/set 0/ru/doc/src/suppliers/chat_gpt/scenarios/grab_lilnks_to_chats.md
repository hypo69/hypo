# Модуль grab_lilnks_to_chats

## Обзор

Этот модуль предназначен для получения ссылок на отдельные чаты с использованием веб-драйвера. Он использует данные из файла `chats_list.json` для поиска ссылок.

## Функции

### `get_links`

**Описание**: Функция извлекает ссылки на отдельные чаты, используя заданный веб-драйвер.

**Параметры**:

- `d` (Driver): Веб-драйвер для взаимодействия с веб-страницей.

**Возвращает**:

- `list[str] | None`: Список ссылок на отдельные чаты или `None`, если возникла ошибка.

**Вызывает исключения**:

- Возможные исключения, связанные с работой веб-драйвера (например, `NoSuchElementException`, `TimeoutException`).

## Использование

Пример использования функции `get_links`:

```python
from src.webdriver.driver import Driver
from src.webdriver.firefox import Firefox
# ... (импорт других необходимых модулей)

if __name__ == '__main__':
    d = Driver(Firefox)
    d.get_url('https://chatgpt.com/')
    links = get_links(d)
    if links:
        for link in links:
            print(link)
    else:
        print("Список ссылок пуст или произошла ошибка.")
```

**Примечания**:

- Файл `chats_list.json` должен содержать данные в формате JSON, соответствующем ожидаемому формату локейтеров.
- Необходимо установить необходимые библиотеки (например, `selenium`, `json`).
- Этот код не будет работать корректно без правильной настройки веб-драйвера (например, установлены и настроены Firefox).
- Модуль использует локейтор `link` из файла `chats_list.json`.  Обратите внимание, что этот локейтор должен быть правильно сконфигурирован для успешного поиска ссылок.
