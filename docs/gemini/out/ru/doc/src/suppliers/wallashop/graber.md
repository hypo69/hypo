# Модуль `hypotez/src/suppliers/wallashop/graber.py`

## Обзор

Модуль содержит класс `Graber`, предназначенный для сбора данных с веб-страниц товаров на сайте `wallashop.co.il`. Класс наследуется от базового класса `Graber` из модуля `src.suppliers.graber`.  В классе реализованы методы для обработки полей товаров, а также возможность использования декоратора для предварительных действий перед отправкой запросов к веб-драйверу.

## Классы

**Описание**: Класс для сбора данных с сайта `wallashop.co.il`. Наследуется от класса `Grbr` из модуля `src.suppliers.graber` и предоставляет методы для обработки полей товаров.  Возможность применения декоратора для предварительных действий.
