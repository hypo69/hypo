# Задача No 484
## Цепные числа Фибоначчи

## Условия
Рассмотрим последовательность Фибоначчи F<sub>n</sub>, где F<sub>0</sub> = 0, F<sub>1</sub> = 1 и F<sub>n</sub> = F<sub>n-1</sub> + F<sub>n-2</sub> для n ≥ 2.

Последовательность цепных чисел Фибоначчи G<sub>n</sub> определяется следующим образом:
* G<sub>0</sub> = 0
* G<sub>1</sub> = 1
* G<sub>n</sub> = G<sub>n-1</sub> + G<sub>n-2</sub>, если F<sub>n</sub> четно
* G<sub>n</sub> = G<sub>n-1</sub> - G<sub>n-2</sub>, если F<sub>n</sub> нечетно

Первые несколько значений F<sub>n</sub> и G<sub>n</sub>:

| n | F<sub>n</sub> | G<sub>n</sub> |
|---|---|---|
| 0 | 0 | 0 |
| 1 | 1 | 1 |
| 2 | 1 | 1 - 0 = 1 |
| 3 | 2 | 1 + 1 = 2 |
| 4 | 3 | 2 - 1 = 1 |
| 5 | 5 | 1 - 2 = -1 |
| 6 | 8 | -1 + 1 = 0 |
| 7 | 13 | 0 - (-1) = 1 |
| 8 | 21 | 1 - 0 = 1 |
| 9 | 34 | 1 + 1 = 2 |
|10 | 55 | 2 - 1 = 1 |

Пусть S(n) = Σ G<sub>i</sub> для 0 ≤ i ≤ n.

Известно, что S(10) = 8, S(100) = 4944, и S(1000) = 4292212.

Найдите S(10<sup>15</sup>) mod 10<sup>9</sup>.
