# Модуль hypotez/src/webdriver/chrome/extentions/version.py

## Обзор

Этот модуль содержит константы и метаданные, связанные с версией расширения Chrome.

## Переменные

### `MODE`

**Описание**:  Переменная, хранящая режим работы (например, 'dev', 'prod').

**Значение**: 'dev'


## Дополнительные данные

### `__version__`

**Описание**: Строковая переменная, содержащая версию модуля.

**Значение**: "3.12.0.0.0.4"

### `__doc__`

**Описание**: Строковое описание модуля.


### `__details__`

**Описание**: Подробная информация о версии модуля.

**Значение**: "Details about version for module or class"

### `__name__`

**Описание**: Имя модуля. При запуске скрипта напрямую значение равно `"__main__"`.

**Тип**: str


### `__author__`

**Описание**: Автор(ы) модуля.


**Значение**: 'hypotez'


### `__annotations__`

**Описание**: Словарь, содержащий аннотации типов для переменных и функций.

**Тип**: dict
