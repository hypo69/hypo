# Модуль `utils.py`

## Обзор

Модуль содержит набор низкоуровневых утилитных функций, используемых для шифрования, дешифрования, кодирования и декодирования данных. Эти функции необходимы для взаимодействия с API Mega.co.nz, обеспечивая безопасную передачу данных.

## Подробней

Этот модуль предоставляет инструменты для выполнения криптографических операций, таких как шифрование AES в режиме CBC, а также для кодирования и декодирования данных в формате Base64. Он также включает функции для преобразования данных между различными форматами, такими как массивы 32-битных целых чисел и строки. Модуль важен для обеспечения безопасности и целостности данных при работе с Mega.
