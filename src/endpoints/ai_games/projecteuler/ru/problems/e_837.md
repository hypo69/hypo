# Задача No 837
## Золотая пропорция в последовательности Падована
## Условия
Последовательность Падована *P(n)* определяется следующим образом:

*   *P(0) = 1*
*   *P(1) = 1*
*   *P(2) = 1*
*   *P(n) = P(n-2) + P(n-3)* для *n > 2*.

Первые несколько членов последовательности: 1, 1, 1, 2, 2, 3, 4, 5, 7, 9, 12, 16, 21, 28, ...

Определим золотую пропорцию для последовательности Падована как предел отношения двух последовательных членов:

φ = lim(*n* → ∞) *P(n)* / *P(n-1)*.

Можно показать, что этот предел существует, и φ является решением уравнения x³ = x + 1, т.е. φ ≈ 1.324718.

Пусть *A(n)* – округленное значение *P(n)* / *P(n-1)*. Например, *A(10)* = round(12/9) = 1.

Найдите количество значений *n* для *n* между 2 и 10 000 000, таких что |*A(n)* - φ| < 10⁻⁹.
