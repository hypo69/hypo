# Модуль languages

## Обзор

Модуль `languages` определяет класс `Language`, который содержит константы для различных языков. Эти константы используются для указания языка при взаимодействии с API AliExpress.

## Подробней

Этот модуль предоставляет централизованный способ определения поддерживаемых языков, что упрощает использование и поддержку языковых параметров в запросах к API AliExpress. Он позволяет избежать использования строковых литералов, что уменьшает вероятность ошибок и облегчает рефакторинг.

## Классы

### `Language`

**Описание**: Класс, содержащий статические атрибуты, представляющие коды языков.

**Методы**:
- Нет.

**Параметры**:
- Нет.

**Примеры**
```python
from src.suppliers.aliexpress.api.models.languages import Language

# Использование константы для английского языка
english_language = Language.EN
print(english_language)
```
