# Игра "Виселица"

## Обзор

Этот модуль реализует игру "Виселица". Компьютер выбирает случайное слово, а игрок пытается отгадать его по буквам, получая штрафы за неверные попытки.

## Функции

### `draw_hangman`

**Описание**: Функция для отображения виселицы в зависимости от количества ошибок.

**Параметры**:
- `errors` (int): Количество ошибок, совершенных игроком.

**Возвращает**:
- `None`: Функция ничего не возвращает.

### `play_hangman`

**Описание**: Основная функция игры в виселицу.

**Параметры**:
- `None`: Функция не принимает параметров.

**Возвращает**:
- `None`: Функция ничего не возвращает.

**Логика работы**:
1.  Выбирает случайное слово из списка `WORDS`.
2.  Создает строку `guess_string` из прочерков, соответствующую длине слова.
3.  Инициализирует счетчик ошибок `number_of_errors` в 0.
4.  Запускает цикл, пока количество ошибок меньше 6 и слово не отгадано:
    - Выводит текущее состояние слова.
    - Запрашивает ввод буквы у пользователя.
    - Проверяет наличие введенной буквы в слове:
      - Если буква есть, обновляет `guess_string`, показывая букву на всех ее местах.
      - Проверяет, угадано ли слово. Если да, выводит поздравление и завершает игру.
      - Если буквы нет, увеличивает счетчик ошибок и рисует виселицу.
5. Проверяет, не проиграл ли игрок (достигнуто ли 6 ошибок). Если да, выводит сообщение о проигрыше и загаданное слово.
