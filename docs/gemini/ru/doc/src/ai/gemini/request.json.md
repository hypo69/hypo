# Документация для `request.json`

## Обзор

Этот файл (`request.json`) содержит схему JSON, описывающую структуру запроса к API Gemini. Он определяет формат данных, необходимых для взаимодействия с моделью Gemini, включая параметры контента, инструкции, инструменты, настройки безопасности и конфигурации генерации.

## Подробней

Файл `request.json` определяет структуру запросов к API Gemini. Этот файл используется для валидации запросов и обеспечивает единообразный формат обмена данными с моделью. Он содержит определения для различных полей, таких как содержимое запроса (`contents`), инструкции (`systemInstruction`), инструменты (`tools`), настройки безопасности (`safetySettings`) и параметры генерации (`generationConfig`).

## Объекты

### `cachedContent`

**Описание**: Закешированный контент.

**Параметры**:
- `string`: Строковое значение, представляющее закешированный контент.

### `contents`

**Описание**: Массив объектов, содержащих контент запроса.

**Параметры**:
- `role` (string): Роль контента.
- `parts` (массив): Массив частей контента.

### `parts`

**Описание**: Массив частей контента, каждая из которых может быть текстом, inline-данными, данными файла или метаданными видео.

**Параметры**:
- `text` (string): Текстовое содержимое.
- `inlineData` (объект): Встроенные данные.
  - `mimeType` (string): MIME-тип данных.
  - `data` (string): Данные в формате строки.
- `fileData` (объект): Данные файла.
  - `mimeType` (string): MIME-тип файла.
  - `fileUri` (string): URI файла.
- `videoMetadata` (объект): Метаданные видео.
  - `startOffset` (объект): Смещение начала видео.
    - `seconds` (integer): Секунды смещения.
    - `nanos` (integer): Наносекунды смещения.
  - `endOffset` (объект): Смещение конца видео.
    - `seconds` (integer): Секунды смещения.
    - `nanos` (integer): Наносекунды смещения.

### `systemInstruction`

**Описание**: Инструкции для системы.

**Параметры**:
- `role` (string): Роль инструкции.
- `parts` (массив): Массив частей инструкции.
  - `text` (string): Текст инструкции.

### `tools`

**Описание**: Массив инструментов.

**Параметры**:
- `functionDeclarations` (массив): Массив объявлений функций.
  - `name` (string): Имя функции.
  - `description` (string): Описание функции.
  - `parameters` (объект): Параметры функции.

### `safetySettings`

**Описание**: Настройки безопасности.

**Параметры**:
- `category` (enum): Категория безопасности.
- `threshold` (enum): Порог блокировки.

### `generationConfig`

**Описание**: Конфигурация генерации.

**Параметры**:
- `temperature` (number): Температура.
- `topP` (number): Top P.
- `topK` (number): Top K.
- `candidateCount` (integer): Количество кандидатов.
- `maxOutputTokens` (integer): Максимальное количество выходных токенов.
- `presencePenalty` (float): Штраф за присутствие.
- `frequencyPenalty` (float): Штраф за частоту.
- `stopSequences` (массив): Последовательности остановки.
- `responseMimeType` (string): MIME-тип ответа.
- `responseSchema` (схема): Схема ответа.
- `seed` (integer): Зерно.
- `responseLogprobs` (boolean): Лог вероятности ответа.
- `logprobs` (integer): Лог вероятности.
- `audioTimestamp` (boolean): Временная метка аудио.

### `labels`

**Описание**: Метки.

**Параметры**:
- `string` (string): Строковое значение метки.
