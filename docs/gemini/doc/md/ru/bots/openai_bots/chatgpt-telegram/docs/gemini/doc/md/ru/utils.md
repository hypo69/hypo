# Модуль removeFile

## Обзор

Данный модуль предоставляет функцию для асинхронного удаления файла.

## Функции

### `removeFile`

**Описание**: Асинхронная функция для удаления файла по указанному пути.

**Параметры**:
- `path` (str): Путь к файлу, который необходимо удалить.

**Возвращает**:
-  `None`: Функция не возвращает явных значений.

**Вызывает исключения**:
- `TypeError`: Возникает, если передан некорректный тип `path`.
- `Error`: Общее исключение, которое может возникнуть при удалении файла.  Это может быть связанно с проблемой доступа к файлу, отсутствием файла или другими операционными ошибками.


```python
import { unlink } from 'fs/promises'
export async function removeFile(path) {
    try {
        await unlink(path)
    } catch (e) {
        console.log('Error while removing file', e.message)
    }
}
```
