PsychologistBot
=============

This module defines the `PsychologistTelgrambot` class, a custom Telegram bot designed for psychological support. It utilizes the Gemini API for generating responses to user input, handles various message types (text, voice, documents), and incorporates a system for routing URLs to specific handlers.  The bot is customized to work with a specific Telegram chat.
