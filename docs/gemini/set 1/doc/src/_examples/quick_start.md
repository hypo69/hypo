# Модуль hypotez/src/_examples/quick_start.py

## Обзор

Данный модуль предоставляет пример быстрого запуска библиотеки `hypotez`. Он содержит константу `MODE`, которая, предположительно, задаёт режим работы (например, `dev` для разработки).


## Константы

### `MODE`

**Описание**: Константа, определяющая режим работы приложения.

**Значение**: `'dev'`

В этом модуле нет функций.
