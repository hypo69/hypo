# Модуль `hypotez/src/webdriver/_examples/_example_executor_2.py`

## Обзор

Данный модуль содержит примеры использования класса `ExecuteLocator` для различных сценариев тестирования. Примеры демонстрируют создание экземпляра `ExecuteLocator`, выполнение различных задач с его помощью, включая получение элементов по XPath, отправку сообщений, обработку исключений и использование списков локаторов.

## Функции

### `execute_locator`

**Описание**: Функция `execute_locator` выполняет заданный локатор, используя `WebDriver`.

**Параметры**:

- `locator (dict):` Словарь, описывающий локатор.  Требуется информация о типе поиска (`by`), селекторе (`selector`), атрибуте (`attribute`), ивенте (`event`), и других параметрах.
- `continue_on_error (bool, optional):` Флаг, определяющий, нужно ли продолжать выполнение при ошибке. По умолчанию `False`.

**Возвращает**:

- `dict | None:` Возвращает результат выполнения локатора или `None`, если произошла ошибка.

**Вызывает исключения**:

- `ExecuteLocatorException`: Возникает при ошибках выполнения локатора, например, при отсутствии элемента или ошибке в селекторе.


### `send_message`

**Описание**: Функция `send_message` отправляет сообщение в заданное поле ввода.

**Параметры**:

- `locator (dict):` Словарь, описывающий локатор.
- `message (str):` Текст сообщения для отправки.
- `typing_speed (float, optional):` Скорость набора текста (в секундах). По умолчанию `0.1`
- `continue_on_error (bool, optional):` Флаг, определяющий, нужно ли продолжать выполнение при ошибке. По умолчанию `False`.


**Возвращает**:

- `str | None:` Возвращает `None`, если произошла ошибка, иначе возвращает `True`.

**Вызывает исключения**:

- `ExecuteLocatorException`: Возникает при ошибках выполнения локатора.


### `evaluate_locator`

**Описание**: Функция `evaluate_locator` выполняет локатор и возвращает значение атрибута.

**Параметры**:

- `attribute_name (str):` Имя атрибута для получения значения.


**Возвращает**:

- `str | None:` Возвращает значение атрибута или `None`, если произошла ошибка.

**Вызывает исключения**:

- `ExecuteLocatorException`: Возникает при ошибках выполнения локатора.
