# Модуль `header`

## Обзор

Модуль `header` содержит функции для определения корневой директории проекта. Это необходимо для правильной настройки путей и импорта модулей внутри проекта `hypotez`.

## Подорбней

Этот модуль предоставляет функцию `set_project_root`, которая автоматически определяет корень проекта, начиная с текущего местоположения файла (`header.py`). Он ищет вверх по дереву каталогов, пока не найдет один из файлов-маркеров (например, `__root__` или `.git`). Если корневой каталог найден, он добавляется в `sys.path`, что позволяет импортировать модули из этого каталога.
