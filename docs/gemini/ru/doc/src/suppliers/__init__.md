# Модуль `hypotez/src/suppliers`

## Обзор

Данный модуль предоставляет инструменты для работы с поставщиками данных. Он содержит класс `Supplier`, который служит базовым классом для работы с различными поставщиками.  Для каждого поставщика могут быть определены специфичные методы извлечения данных, которые дополняют базовый класс `Supplier`.  Эти методы расположены в директориях, названия которых соответствуют префиксу имени поставщика (например, `amazon`, `aliexpress`).  Префикс задается при создании нового поставщика.
