# Модуль hypotez/src/suppliers/ksp/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json` и документации из `README.MD`. Он также предоставляет доступ к глобальным переменным, содержащим информацию о проекте (имя проекта, версия, описание и др.).

## Функции

### `set_project_root`

**Описание**: Функция находит корневую директорию проекта, начиная от текущего файла.  Она ищет вверх по директориям, пока не найдет директорию, содержащую указанные файлы (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:
- `marker_files` (tuple): Кортеж файлов или директорий, по которым определяется корень проекта. По умолчанию содержит ('pyproject.toml', 'requirements.txt', '.git').

**Возвращает**:
- `Path`: Путь к корневой директории проекта. Если корень не найден, возвращает директорию, где расположен текущий скрипт.

**Примечания**: Добавляет найденный корень в системный путь `sys.path` для корректной импортации модулей из проекта.


### `set_project_root`

**Описание**:  Функция находит корневую директорию проекта.

**Параметры**:
- `marker_files`: Список файлов или директорий, которые используются для поиска корня проекта. По умолчанию содержит `'pyproject.toml'`, `'requirements.txt'`, и `'.git'`.


**Возвращает**:
- `Path`: Путь к корню проекта.


## Глобальные переменные

### `__root__`

**Описание**: Путь к корневой директории проекта.


### `settings`

**Описание**: Словарь настроек проекта, загруженный из файла `settings.json`. Может быть `None`, если файл не найден или повреждён.


### `doc_str`

**Описание**: Строка с содержимым файла `README.MD`. Может быть `None`, если файл не найден или повреждён.


### `__project_name__`

**Описание**: Имя проекта. Значение по умолчанию — `hypotez`.


### `__version__`

**Описание**: Версия проекта. Значение по умолчанию — пустая строка.


### `__doc__`

**Описание**: Документация проекта (содержимое файла `README.MD`). Значение по умолчанию — пустая строка.


### `__details__`

**Описание**: Дополнительные детали о проекте. Значение по умолчанию — пустая строка.


### `__author__`

**Описание**: Автор проекта. Значение по умолчанию — пустая строка.


### `__copyright__`

**Описание**: Авторские права на проект. Значение по умолчанию — пустая строка.


### `__cofee__`

**Описание**: Ссылка на способ поддержки разработчиков. Значение по умолчанию — ссылка на Boosty.


## Обработка исключений

- Используются блоки `try...except` для обработки `FileNotFoundError` и `json.JSONDecodeError`,  чтобы избежать аварийного завершения программы при отсутствии или повреждении файла `settings.json` или `README.MD`.
- Исключения `FileNotFoundError` и `json.JSONDecodeError` обрабатываются, и в случае ошибки соответствующая переменная устанавливается в `None` или пустую строку, что позволяет программе продолжить работу без аварийного завершения.
