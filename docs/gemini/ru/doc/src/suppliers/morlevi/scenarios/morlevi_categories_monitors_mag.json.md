## Структура файла
Файл содержит JSON-объект с одним ключом `scenarios`, значением которого является словарь, содержащий сценарии парсинга. Каждый сценарий представлен в виде отдельного объекта, где ключ — это имя сценария (например, `"MAG 22"`), а значение — это словарь с параметрами сценария.

## Сценарии

### `MAG 22`

**Описание**: Сценарий для парсинга мониторов MAG с диагональю 22 дюйма.

**Параметры**:
- `brand` (str): Бренд монитора - `"MAG"`.
- `url` (str): URL страницы с фильтром для мониторов 22 дюйма на сайте morlevi.co.il.
- `checkbox` (bool): Флаг использования чекбокса (в данном случае `false`).
- `active` (bool): Флаг активности сценария - `true`.
- `condition` (str): Условие товара - `"new"`.
- `presta_categories` (str): Список идентификаторов категорий PrestaShop через запятую - `"127,128,529"`.

## Заключение

Данный файл `morlevi_categories_monitors_mag.json` служит для настройки парсинга товаров категории "Мониторы" бренда "MAG" с веб-сайта morlevi.co.il. Каждый сценарий представлен отдельным блоком и содержит всю необходимую информацию для выполнения задачи парсинга. В файле присутствуют заглушки для url в сценариях `MAG 34` и `MAG 49`, необходимо заменить их на корректные значения url.
