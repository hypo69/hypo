# C-Data Мониторы HP: Сценарии

## Обзор

Этот файл содержит JSON-структуру, определяющую сценарии для парсинга мониторов бренда HP с веб-сайта C-Data. Каждый сценарий включает в себя информацию о бренде, URL-адресе для парсинга, настройках чекбокса, активности, состоянии товара и категориях PrestaShop.

## Сценарии

### `HP 18`

**Описание**: Сценарий для парсинга 18-дюймовых мониторов HP.

**Параметры**:
- `brand` (str): Бренд монитора - "HP".
- `url` (str): URL-адрес страницы с мониторами на сайте C-Data.
- `checkbox` (bool): Флаг, указывающий на необходимость использования чекбокса. По умолчанию `false`.
- `active` (bool): Флаг, указывающий на активность сценария. По умолчанию `true`.
- `condition` (str): Состояние товара - "new".
- `presta_categories` (str): Категории PrestaShop, к которым относится товар, разделенные запятой - "127,241".
