# Модуль `src.goog.spreadsheet.bberyakov`

## Обзор

Модуль `src.goog.spreadsheet.bberyakov` предоставляет инструменты для работы с Google Sheets API, включая классы для управления электронными таблицами и листами, а также для рендеринга данных.

## Классы

### `GSpreadsheet`

**Описание**: Класс для работы с Google Spreadsheet.

### `GWorksheet`

**Описание**: Класс для работы с отдельными листами Google Spreadsheet.

### `GSRenderr`

**Описание**: Класс для рендеринга данных из Google Sheets.
