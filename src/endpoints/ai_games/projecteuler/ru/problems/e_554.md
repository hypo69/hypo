# Задача No 554
## Достижения игрока

## Условия
Пусть *n* представляет собой целое положительное число.
Рассмотрим набор *S*( *n* ) как множество всех возможных последовательностей длин ходов, которые игрок может сделать в игре, состоящей из *n* ходов, где длина каждого хода является целым положительным числом, и каждая длина хода меньше или равна предыдущей длине хода.

Например, *S*(4) содержит следующие 15 последовательностей:
{1,1,1,1}, {2,1,1,1}, {1,2,1,1}, {1,1,2,1}, {1,1,1,2},
{3,1,1,1}, {2,2,1,1}, {2,1,2,1}, {1,3,1,1}, {1,2,2,1},
{4,1,1,1}, {3,2,1,1}, {3,1,2,1}, {2,3,1,1}, {2,2,2,1}.

Пусть *f*( *n* ) будет размером *S*( *n* ), то есть количеством возможных последовательностей в *S*( *n* ).
Из вышеприведенного примера, *f*(4) = 15.
Также известно, что *f*(5) = 51 и *f*(10) = 627.

Пусть *F*( *m* ) будет суммой *f*( *n* ) для *n* = 1 до *m*.
Таким образом, *F*(5) = 1 + 2 + 4 + 15 + 51 = 73.

Найдите *F*(10<sup>18</sup>) mod 1 000 000 007.
