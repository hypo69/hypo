# Модуль `base64`

## Обзор

Модуль `base64` предоставляет функции для кодирования и декодирования данных в формате Base64, а также для записи декодированного контента во временный файл.

## Функции

### `base64_to_tmpfile`

**Описание**: 
Преобразует контент, закодированный в Base64, во временный файл.

Эта функция декодирует контент, закодированный в Base64, и записывает его во временный файл с тем же расширением, что и у предоставленного имени файла. Возвращается путь к временному файлу.

**Параметры**:
- `content` (str): Контент, закодированный в Base64, который будет декодирован и записан в файл.
- `file_name` (str): Имя файла, используемое для извлечения расширения файла для временного файла.

**Возвращает**:
- `str`: Путь к временному файлу.

**Пример**:
```python
>>> base64_content = "SGVsbG8gd29ybGQh"  # Base64 encoded content "Hello world!"
>>> file_name = "example.txt"
>>> tmp_file_path = base64_to_tmpfile(base64_content, file_name)
>>> print(f"Temporary file created at: {tmp_file_path}")
Temporary file created at: /tmp/tmpfile.txt
```

### `base64encode`

**Описание**:
Функция для кодирования изображения в формат Base64.

**Параметры**:
- `image_path` (str): Путь к изображению.

**Возвращает**:
- `str`: Изображение, закодированное в Base64.
