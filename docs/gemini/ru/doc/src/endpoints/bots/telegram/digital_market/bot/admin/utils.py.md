# Модуль utils

## Обзор

Модуль `utils` содержит утилиту `process_dell_text_msg`, которая предназначена для удаления последнего сообщения пользователя и текущего сообщения в Telegram боте.

## Подробней

Этот модуль используется для очистки истории сообщений в чате с ботом, удаляя предыдущее сообщение бота и сообщение пользователя. Это может быть полезно для управления взаимодействием с пользователем, например, при переходе к новому этапу диалога или для удаления устаревшей информации. Функция `process_dell_text_msg` извлекает идентификатор последнего сообщения из контекста FSM (Finite State Machine) и использует его для удаления сообщения бота.
