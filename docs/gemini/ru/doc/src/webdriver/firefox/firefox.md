# Модуль firefox.py

## Обзор

Модуль `firefox.py` предоставляет класс `Firefox`, расширяющий функционал стандартного `webdriver.Firefox`. Он позволяет настраивать пользовательский профиль, запускать браузер в режиме киоска и устанавливать пользовательские настройки, включая прокси.  Модуль также загружает и использует дополнительные функции, такие как JavaScript манипуляции и поиск элементов.

## Класс Firefox

### `Firefox`

**Описание**: Расширенный класс для работы с Firefox WebDriver.  Позволяет настраивать профиль, geckodriver, версию Firefox, пользовательский агент и прокси.

**Конструктор**:

### `__init__`

**Описание**: Инициализирует экземпляр класса `Firefox`.

**Параметры**:

- `profile_name` (Optional[str], optional): Имя пользовательского профиля Firefox. По умолчанию `None`.
- `geckodriver_version` (Optional[str], optional): Версия geckodriver. По умолчанию `None`.
- `firefox_version` (Optional[str], optional): Версия Firefox. По умолчанию `None`.
- `user_agent` (Optional[str], optional): Пользовательский агент в формате строки. По умолчанию `None`.
- `proxy_file_path` (Optional[str], optional): Путь к файлу с прокси. По умолчанию `None`.
- `*args`, `**kwargs`:  Дополнительные параметры, передаваемые в конструктор базового класса `WebDriver`.

**Возвращает**:
- `None`

**Обрабатываемые исключения**:

- `WebDriverException`: Возникает при ошибках запуска WebDriver (например, неверный путь к geckodriver, отсутствует Firefox).
- `Exception`: Возникает при других ошибках во время инициализации.

### `set_proxy`

**Описание**: Настройка прокси из предоставленного словаря.

**Параметры**:

- `options` (Options): Объект `Options` для настройки прокси в Firefox.


**Возвращает**:
- `None`

**Обрабатываемые исключения**:

- `TypeError`: Возникает при неверном типе данных.
- `KeyError`: Возникает при отсутствии ключа в словаре.


### `_payload`

**Описание**: Загрузка исполнителей для локейторов и сценариев JavaScript.

**Параметры**:
-  Нет

**Возвращает**:
- `None`

**Обрабатываемые исключения**:
- `Exception`: Возникает при других ошибках.
