# Модуль `hypotez/src/suppliers/aliexpress/api/_examples/rest/AliexpressAffiliateCategoryGetRequest.py`

## Обзор

Модуль содержит класс `AliexpressAffiliateCategoryGetRequest`, представляющий запрос для получения информации о категориях на платформе AliExpress. Этот класс наследуется от базового класса `RestApi`, предоставляя методы для работы с API.

## Классы

### `AliexpressAffiliateCategoryGetRequest`

**Описание**: Класс реализует запрос к API AliExpress для получения данных о категориях.

**Методы**:

- `__init__`: Инициализирует объект запроса.
- `getapiname`: Возвращает имя API-метода.

**Параметры метода `__init__`**:

- `domain` (str, опционально): Доменное имя API (по умолчанию "api-sg.aliexpress.com").
- `port` (int, опционально): Порт API (по умолчанию 80).


**Метод `getapiname`**:

**Описание**: Возвращает имя метода API, который используется для запроса.

**Возвращает**:
- str: Название метода API ("aliexpress.affiliate.category.get").


**Примечания**:  Класс использует базовый класс `RestApi`, который, предположительно, содержит дополнительные методы для обработки запросов к API, такие как отправка запроса и обработка ответа.  Необходимы подробности о классе `RestApi`.
