# Модуль `battle.py`

## Обзор

Этот модуль представляет собой простую игру в битву между двумя игроками (один из которых - компьютер). Игроки начинают с заданным количеством солдат, и каждый ход выбирают, сколько солдат отправить в атаку. Побеждает тот, кто уничтожит всех солдат противника.
