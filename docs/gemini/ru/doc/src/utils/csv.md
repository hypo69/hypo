# Модуль `hypotez/src/utils/csv.py`

## Обзор

Модуль `hypotez/src/utils/csv.py` предоставляет функции для работы с CSV и JSON файлами.  Он включает в себя операции по сохранению данных в CSV, чтению данных из CSV в список словарей, преобразованию CSV в JSON и чтению CSV в формат словаря.

## Функции

### `save_csv_file`

**Описание**: Сохраняет список словарей в CSV файл.  Возможность добавления данных в существующий файл ('a') или перезаписи ('w').

**Параметры**:

- `data` (List[Dict[str, str]]): Список словарей, которые нужно сохранить в CSV файл. Должен быть списком словарей.
- `file_path` (Union[str, Path]): Путь к CSV файлу.
- `mode` (str, optional): Режим открытия файла ('a' - добавить, 'w' - перезаписать). По умолчанию 'a'.
- `exc_info` (bool, optional): Включает информацию об исключении в логи. По умолчанию True.

**Возвращает**:

- `bool`: `True`, если операция прошла успешно, иначе `False`.

**Возможные исключения**:

- `TypeError`: Если входные данные `data` не являются списком словарей.
- `ValueError`: Если входные данные `data` пустые.


### `read_csv_file`

**Описание**: Читает содержимое CSV файла и возвращает его как список словарей.

**Параметры**:

- `file_path` (Union[str, Path]): Путь к CSV файлу.
- `exc_info` (bool, optional): Включает информацию об исключении в логи. По умолчанию True.

**Возвращает**:

- `List[Dict[str, str]] | None`: Список словарей, представляющих содержимое CSV файла, или `None`, если произошла ошибка.

**Возможные исключения**:

- `FileNotFoundError`: Если файл не найден.


### `read_csv_as_json`

**Описание**: Преобразует CSV файл в JSON и сохраняет его в указанный файл.

**Параметры**:

- `csv_file_path` (Union[str, Path]): Путь к CSV файлу.
- `json_file_path` (Union[str, Path]): Путь для сохранения JSON файла.
- `exc_info` (bool, optional): Включает информацию об исключении в логи. По умолчанию True.

**Возвращает**:

- `bool`: `True` если преобразование успешно, иначе `False`.

**Возможные исключения**:

- Любые исключения, возникающие во время работы с файлами или JSON.


### `read_csv_as_dict`

**Описание**: Преобразует содержимое CSV файла в словарь.

**Параметры**:

- `csv_file` (Union[str, Path]): Путь к CSV файлу.

**Возвращает**:

- `dict | None`: Словарь, представляющий содержимое CSV файла, или `None`, если произошла ошибка.

**Возможные исключения**:

- Любые исключения, возникающие во время работы с файлом.


### `read_csv_as_ns`

**Описание**: Загружает данные из CSV файла в список словарей с использованием Pandas.

**Параметры**:

- `file_path` (Union[str, Path]): Путь к CSV файлу.

**Возвращает**:

- `List[dict]`: Список словарей, представляющих содержимое CSV файла.  Возвращает пустой список если возникла ошибка или файл не найден.

**Возможные исключения**:

- `FileNotFoundError`: Если файл не найден.
