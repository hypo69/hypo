# Модуль `salvo.py`

## Обзор

Этот модуль реализует игру "Морской бой" (SALVO), в которой игрок пытается потопить корабли противника, расположенные на сетке 10x10. Игрок вводит координаты для выстрела, и игра сообщает о попадании, промахе или потоплении корабля. Цель игры — потопить все корабли противника за минимальное количество ходов.

## Функции

### `create_board`

**Описание**: Создает игровое поле заданного размера, заполненное нулями.

**Параметры**:
- `size` (int): Размер игрового поля.

**Возвращает**:
- `list[list[int]]`: Двумерный список, представляющий игровое поле.

### `place_ships`

**Описание**: Размещает корабли на игровом поле. Корабли размещаются случайным образом (горизонтально или вертикально).

**Параметры**:
- `board` (list[list[int]]): Игровое поле.
- `ships_lengths` (list[int]): Список длин кораблей для размещения.

**Возвращает**:
- `list[tuple[int, int, str, int]]`: Список кортежей с информацией о размещенных кораблях (строка, столбец, ориентация, длина).

### `is_sunk`

**Описание**: Проверяет, потоплен ли корабль.

**Параметры**:
- `board` (list[list[str]]): Игровое поле с отметками о попаданиях.
- `ship` (tuple[int, int, str, int]): Кортеж с информацией о корабле (строка, столбец, ориентация, длина).

**Возвращает**:
- `bool`: `True`, если корабль потоплен, иначе `False`.

### `print_board`

**Описание**: Выводит игровое поле в консоль, скрывая расположение кораблей. Вместо кораблей отображаются '~'.

**Параметры**:
- `board` (list[list[str]]): Игровое поле.

**Возвращает**:
- `None`

### `play_salvo`

**Описание**: Основная функция игры Salvo. Управляет процессом игры, включая ввод координат от пользователя, проверку попаданий, потоплений, и вывод результатов.

**Параметры**:
- `None`

**Возвращает**:
- `None`

**Вызывает исключения**:
- `ValueError`: Если пользователь вводит некорректные координаты (не числа или значения вне диапазона игрового поля).
