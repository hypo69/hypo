# Модуль hypotez/src/utils/autodoc.py

## Обзор

Модуль `autodoc` содержит декоратор `autodoc`, который автоматически обновляет строку документации функции, добавляя время последнего вызова. Это позволяет отслеживать историю вызовов функций.

## Функции

### `autodoc`

**Описание**: Декоратор, который обновляет строку документации функции с добавлением времени последнего вызова.

**Параметры**:

- Нет.


**Возвращает**:
- Обновленная функция.


**Вызывает исключения**:
- Нет.


### `update_docstring`

**Описание**: Функция, которая обновляет строку документации функции.

**Параметры**:
- `func` (функция): Функция, для которой нужно обновить документацию.

**Возвращает**:
- Нет.


**Вызывает исключения**:
- Нет.


### `example_function`

**Описание**: Пример функции, использующей декоратор `autodoc`.  При каждом вызове её docstring обновляется, добавляя информацию о времени последнего вызова.

**Параметры**:
- `param1` (int): Первое значение.
- `param2` (str): Второе значение.

**Возвращает**:
- None.

**Вызывает исключения**:
- Нет.


## Пример использования

```python
@autodoc
def example_function(param1: int, param2: str) -> None:
    """Пример функции.

    Args:
        param1 (int): Первое значение.
        param2 (str): Второе значение.
    """
    print(f"Processing {param1} and {param2}")

example_function(1, "test")
print(example_function.__doc__)  # Вывод обновленного docstring
example_function(2, "another test")
print(example_function.__doc__)  # Вывод обновленного docstring
```
