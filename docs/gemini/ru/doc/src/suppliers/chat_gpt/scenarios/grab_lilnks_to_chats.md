# Модуль `hypotez/src/suppliers/chat_gpt/scenarios/grab_lilnks_to_chats.py`

## Обзор

Модуль `grab_lilnks_to_chats.py` содержит функции для получения ссылок на чаты с использованием веб-драйвера.  Он использует конфигурацию из файла `chats_list.json` для определения локеров и осуществляет взаимодействие с браузером через драйвер.

## Переменные

### `MODE`

**Описание**:  Переменная, хранящая режим работы (`'dev'` в текущем примере).

## Функции

### `get_links(d: Driver)`

**Описание**: Получает список ссылок на отдельные чаты, используя переданный веб-драйвер.

**Параметры**:

- `d` (Driver): Экземпляр класса `Driver`, предоставляющий доступ к веб-драйверу (например, Chrome или Firefox).

**Возвращает**:

- `list`: Список строк, где каждая строка представляет собой URL-адрес чата. Возвращает `None`, если возникла ошибка.


**Вызывает исключения**:

- Любые исключения, генерируемые функциями веб-драйвера.


## Использование (Пример)

```python
if __name__ == '__main__':
    d = Driver(Firefox)
    d.get_url('https://chatgpt.com/')
    links = get_links(d)
    # Обработка полученных ссылок (links)
    ...
```

**Примечание**: Данный пример предполагает, что модуль `header`, `gs`, `Driver`, `Chrome`, `Firefox`, `j_loads_ns` уже импортированы и доступны. Файл `chats_list.json` должен содержать корректные данные для локеров. Обратите внимание на обработку потенциальных ошибок при использовании `get_links`.
