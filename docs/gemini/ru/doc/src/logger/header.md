# Модуль hypotez/src/logger/header.py

## Обзор

Данный модуль отвечает за инициализацию корневого пути проекта и загрузку настроек из файла `settings.json`. Он также предоставляет доступ к документации проекта (README.md).

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла. Поиск ведется вверх по иерархии директорий до первого обнаружения файлов-маркеров (по умолчанию: `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Список файлов или директорий, которые используются для определения корневой директории проекта. По умолчанию: (`'pyproject.toml'`, `'requirements.txt'`, `'.git'`).


**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена. В противном случае возвращает директорию текущего файла.


**Вызывает исключения**:

- Нет


### `__init__`

**Описание**:  Инициализирует модуль.

**Параметры**:

- Нет

**Возвращает**:

- Нет

**Вызывает исключения**:

- `FileNotFoundError`: Если файл `settings.json` не найден.
- `json.JSONDecodeError`: Если файл `settings.json` поврежден или некорректен.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный функцией `set_project_root()`.

**Тип**: `Path`


### `settings`

**Описание**: Словарь настроек проекта, загруженный из файла `settings.json`.

**Тип**: `dict` или `None`


### `doc_str`

**Описание**: Текст документации проекта, загруженный из файла `README.MD`.

**Тип**: `str` или `None`


### `__project_name__`

**Описание**: Название проекта, полученное из настроек. Если настройка отсутствует, используется значение по умолчанию `hypotez`.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта, полученная из настроек. Если настройка отсутствует, используется значение по умолчанию `''`.

**Тип**: `str`


### `__doc__`

**Описание**: Документация проекта (README.md), если файл найден, иначе пустая строка.

**Тип**: `str`


### `__details__`

**Описание**: Подробная информация о проекте (пока не используется).

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта, полученный из настроек. Если настройка отсутствует, используется значение по умолчанию `''`.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права проекта, полученные из настроек. Если настройка отсутствует, используется значение по умолчанию `''`.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка на страницу поддержки разработчика.

**Тип**: `str`
