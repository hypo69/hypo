# Модуль `get_product_id`

## Обзор

Модуль предоставляет инструменты для извлечения идентификатора продукта из входной строки, такой как URL или непосредственно идентификатор.

## Оглавление

1. [Функции](#Функции)
    - [`get_product_id`](#get_product_id)

## Функции

### `get_product_id`

**Описание**: Возвращает идентификатор продукта из заданного текста. Вызывает исключение `ProductIdNotFoundException` в случае неудачи.

**Параметры**:
- `raw_product_id` (str): Строка, содержащая URL или идентификатор продукта.

**Возвращает**:
- `str`: Идентификатор продукта.

**Вызывает исключения**:
- `ProductIdNotFoundException`: Вызывается, если не удается извлечь идентификатор продукта из входной строки.
