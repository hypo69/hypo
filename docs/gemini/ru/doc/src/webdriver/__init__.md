# Модуль hypotez/src/webdriver

## Обзор

Данный модуль `webdriver` предоставляет абстракцию для управления различными веб-драйверами, такими как Chrome, Firefox, Edge и др. Он предоставляет интерфейс для инициализации и управления сессиями браузера, а также включает в себя базовые методы для взаимодействия с веб-страницами.
