# Документация модуля `creds.py`

## Обзор

Модуль `creds.py` предназначен для хранения учетных данных, необходимых для работы с Telegram ботом и Google Drive API. Он содержит класс `Creds`, в котором определены статические переменные для хранения токена Telegram бота, ID папки TeamDrive и ID TeamDrive. Этот модуль позволяет удобно управлять и конфигурировать доступ к необходимым ресурсам.

## Подробнее

Этот модуль предназначен для хранения и предоставления учетных данных, используемых приложением для взаимодействия с Telegram и Google Drive. Он содержит класс `Creds` с полями для хранения токенов и идентификаторов, необходимых для аутентификации и авторизации. Код используется для централизованного управления учетными данными, что упрощает их изменение и предотвращает их утечку. Расположение файла `hypotez/src/endpoints/bots/google_dirve/creds.py` указывает на то, что он используется для настройки интеграции с Google Drive в рамках функциональности Telegram-бота.
