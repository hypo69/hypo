# HOCKEY

## סקירה כללית

מודול זה מדמה משחק הוקי בסיסי בין שני שחקנים. כל שחקן יכול לבחור מבין שלוש פעולות בכל סיבוב: זריקה, הגנה או מסירה. הנקודות מוענקות בהתאם לצירוף הפעולות של שני השחקנים. המשחק ממשיך עד שאחד השחקנים מגיע ל-10 נקודות.

## פונקציות

### `main`

**תיאור**:
פונקציה זו מפעילה את משחק ההוקי. היא מאפשרת לשני שחקנים לבחור פעולות (זריקה, הגנה או מסירה) בכל סיבוב, מעדכנת את הניקוד בהתאם לחוקי המשחק, ומכריזה על המנצח לאחר שאחד השחקנים מגיע ל-10 נקודות.

**פרמטרים**:
אין פרמטרים לפונקציה.

**מחזירה**:
אין ערך מוחזר.

**הערות**:
הקוד משתמש בלולאת `while` כדי לנהל את המשחק עד שאחד השחקנים מגיע ל-10 נקודות.
הקוד משתמש בבלוק `try-excep` לטיפול בשגיאות קלט מהמשתמש. אם המשתמש יזין קלט שאינו מספר שלם, תוצג הודעת שגיאה.
