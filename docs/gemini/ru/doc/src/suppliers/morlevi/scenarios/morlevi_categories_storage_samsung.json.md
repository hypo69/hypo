# Документация для `morlevi_categories_storage_samsung.json`

## Обзор

Данный файл содержит JSON-конфигурацию сценариев для парсинга категорий товаров Samsung со страницы поставщика Morlevi. Каждый сценарий включает в себя информацию о бренде, URL-адресе страницы, настройках чекбокса, статусе активности, условии товара и сопоставлении с категориями PrestaShop.

## Структура JSON

### `scenarios`

Объект, содержащий сценарии для различных категорий товаров Samsung.

### Название сценария

Ключ, представляющий уникальное имя сценария, например `"SAMSUNG NVME GEN4 512GB"`.

#### `brand`

**Описание**: Бренд товара.

**Тип**: `str`

**Пример**: `"SAMSUNG"`

#### `url`

**Описание**: URL-адрес страницы товара у поставщика Morlevi.

**Тип**: `str`

**Пример**: `"https://www.morlevi.co.il/Cat/314?p_315=28&p_175=826&sort=datafloat2%2Cprice&keyword="`

#### `checkbox`

**Описание**: Флаг, указывающий, использовать ли чекбокс при парсинге.

**Тип**: `bool`

**Пример**: `false`

#### `active`

**Описание**: Флаг, указывающий, активен ли сценарий.

**Тип**: `bool`

**Пример**: `true`

#### `condition`

**Описание**: Условие товара.

**Тип**: `str`

**Пример**: `"new"`

#### `presta_categories`

**Описание**: Объект, содержащий сопоставления категорий PrestaShop.

##### `template`

**Описание**: Объект, содержащий сопоставления шаблонов.

**Тип**: `dict`

**Пример**:
```json
{
  "samsung": "SSD NVME GEN4 512GB"
}
```

В данном примере ключ `"samsung"` представляет шаблон для PrestaShop, а значение `"SSD NVME GEN4 512GB"` соответствует категории товара в PrestaShop.
