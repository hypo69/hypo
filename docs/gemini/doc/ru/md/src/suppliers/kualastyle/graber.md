# Модуль hypotez/src/suppliers/kualastyle/graber.py

## Обзор

Модуль `graber.py` содержит класс `Graber`, предназначенный для сбора данных о товарах с сайта `kualastyle.co.il`.  Класс наследуется от базового класса `Graber` (из `src.suppliers`) и переопределяет методы для специфической обработки данных с целевого сайта.  Предоставляет асинхронную функцию `grab_page` для извлечения данных о товаре. Включает в себя декоратор `@close_pop_up` для обработки всплывающих окон перед выполнением задач.


## Классы

### `Graber`

**Описание**: Класс `Graber` реализует логику сбора данных о товарах с сайта `kualastyle.co.il`. Наследуется от `Grbr` (из `src.suppliers`).

- `grab_page(self, driver: Driver) -> ProductFields`:
    **Описание**: Асинхронная функция для извлечения данных о товаре.
    **Параметры**:

    **Возвращает**:
        - `ProductFields`: Объект содержащий извлеченные поля товара.
    **Вызывает исключения**:
        - Возможные исключения, генерируемые методами, вызываемыми внутри функции.
