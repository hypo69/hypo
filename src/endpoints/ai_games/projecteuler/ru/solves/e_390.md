## Ответ на Задачу No 390: Triangles with Integer Sides and Integer Area

### 1. Анализ задачи и решение
**Понимание задачи:**
* Нам нужно посчитать количество треугольников с целочисленными сторонами и целочисленной площадью, у которых длина самой длинной стороны не превышает заданного значения `n`.
* Для проверки, что площадь треугольника является целым числом, мы будем использовать формулу Герона и проверять, что подкоренное выражение (после умножения на 16) является полным квадратом.
* Так как у нас ограничение на самую длинную сторону, то можно перебрать все возможные стороны `a`, `b`, и `c` , где `a <= b <= c <= n`.

**Решение:**
1. **Инициализация:** Создаем переменную `count`, которая будет подсчитывать количество подходящих треугольников, и инициализируем ее значением 0.
2. **Цикл по c:** Итерируемся по всем возможным значениям самой длинной стороны `c` от 1 до `n`.
3. **Цикл по b:** Для каждого `c` итерируемся по всем возможным значениям средней стороны `b` от 1 до `c`.
4. **Цикл по a:** Для каждого `c` и `b` итерируемся по всем возможным значениям самой маленькой стороны `a` от 1 до `b`.
5. **Проверка условия треугольника:** Проверяем, что стороны `a`, `b` и `c` удовлетворяют неравенству треугольника (a + b > c).
6. **Вычисление площади:**
    * Вычисляем полупериметр `s = (a + b + c) / 2`.
    * Проверяем, является ли `16 * s * (s - a) * (s - b) * (s - c)` полным квадратом. Это эквивалентно проверки того, что площадь является целым числом.
7. **Увеличение счетчика:** Если все условия выполняются, то увеличиваем `count` на 1.
8. **Возврат результата:** После прохождения всех циклов возвращаем значение `count`.

### 2. Алгоритм решения
1. Начать
2. Инициализировать переменную `count` со значением 0
3. Для каждого `c` от 1 до `n`:
    * Для каждого `b` от 1 до `c`:
        * Для каждого `a` от 1 до `b`:
            * Если `a + b > c` (неравенство треугольника):
                * Вычислить `s = (a + b + c) / 2`
                * Вычислить `area_squared = 16 * s * (s - a) * (s - b) * (s - c)`
                * Если `area_squared` является полным квадратом:
                    * Увеличить `count` на 1
4. Вернуть `count`
5. Конец

### 3. Реализация на Python 3.12
```python
import math

def is_perfect_square(n):
  """Checks if a number is a perfect square."""
  if n < 0:
    return False
  root = int(math.sqrt(n))
  return root * root == n

def count_integer_triangles(max_side):
  """Counts the number of triangles with integer sides and integer area."""
  triangle_count = 0
  for c in range(1, max_side + 1):
    for b in range(1, c + 1):
      for a in range(1, b + 1):
        if a + b > c:
            s = (a + b + c) / 2
            area_squared = 16 * s * (s - a) * (s - b) * (s - c)
            if is_perfect_square(area_squared):
                triangle_count += 1
  return triangle_count

# Пример использования:
result = count_integer_triangles(1000000)
print(result)
```
