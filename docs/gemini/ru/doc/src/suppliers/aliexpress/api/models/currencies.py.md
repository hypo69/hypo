# Модуль `src.suppliers.aliexpress.api.models.currencies`

## Обзор

Модуль `currencies` содержит класс `Currency`, который определяет набор статических переменных, представляющих различные валюты.

## Подробнее

Этот модуль предоставляет удобный способ для определения валют, используемых при работе с API AliExpress. Каждая валюта представлена строковой константой, что облегчает использование и сравнение валют в коде.

## Классы

### `Currency`

**Описание**: Класс `Currency` содержит статические переменные, представляющие различные валюты.

**Принцип работы**:
Класс `Currency` предназначен для хранения строковых кодов валют в виде статических переменных. Это позволяет использовать их как константы, не создавая экземпляры класса.

**Методы**:
- Нет методов, только статические переменные.

**Статические переменные**:
- `USD`: Доллар США.
- `GBP`: Фунт стерлингов.
- `CAD`: Канадский доллар.
- `EUR`: Евро.
- `UAH`: Украинская гривна.
- `MXN`: Мексиканское песо.
- `TRY`: Турецкая лира.
- `RUB`: Российский рубль.
- `BRL`: Бразильский реал.
- `AUD`: Австралийский доллар.
- `INR`: Индийская рупия.
- `JPY`: Японская иена.
- `IDR`: Индонезийская рупия.
- `SEK`: Шведская крона.
- `KRW`: Южнокорейская вона.
- `ILS`: Израильский новый шекель.

**Примеры**:

```python
from src.suppliers.aliexpress.api.models.currencies import Currency

usd = Currency.USD
print(usd)  # Вывод: USD

eur = Currency.EUR
print(eur)  # Вывод: EUR
```
