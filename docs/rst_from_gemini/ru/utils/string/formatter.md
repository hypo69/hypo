**Функции:**

* **`remove_line_breaks(input_str: str) -> str`:** Удаляет символы перевода строки (`\n`, `\r`) из входной строки.
* **`remove_htmls(input_html: str) -> str`:** Удаляет HTML-теги из входной строки.

* **`remove_special_characters(input_str: str | list) -> str | list`:** Удаляет все символы, кроме букв, цифр и пробелов из входной строки или списка строк.
* **`clear_numbers(input_str: str) -> str`:** Оставляет в строке только цифры и точки, удаляя все остальные символы.  (Полезно для извлечения чисел из строк)
