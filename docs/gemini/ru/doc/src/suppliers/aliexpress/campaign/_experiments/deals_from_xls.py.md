# Модуль для парсинга сделок из XLS файлов, сгенерированных AliExpress

## Обзор

Модуль предназначен для извлечения информации о сделках из XLS файлов, которые были сгенерированы в личном кабинете на сайте portals.aliexpress.com. Он использует класс `DealsFromXLS` для парсинга данных и предоставляет возможность итерации по сделкам.

## Подробнее

Модуль предназначен для автоматизации процесса сбора данных о сделках с платформы AliExpress. Он облегчает задачу получения информации из XLS файлов, предоставляя удобный интерфейс для итерации по каждой сделке.

## Функции

```python
    Raises:
        Нет исключений.

    """
    ...
```

**Как работает функция**:

1.  Функция `get_next_deal` извлекает следующую сделку из XLS файла, который был обработан парсером `DealsFromXLS`.
2.  Информация о сделке возвращается в виде словаря, который содержит различные атрибуты сделки, такие как название продукта, цена, скидка и т. д.

**Пример**:

```python
from src.suppliers.aliexpress import DealsFromXLS
from src.utils.printer import pprint

deals_parser = DealsFromXLS(language='EN', currency= 'USD')

for deal in deals_parser.get_next_deal():
    pprint(deal)
```

В данном примере создается экземпляр класса `DealsFromXLS`, затем в цикле извлекаются и печатаются все доступные сделки из XLS файла.

## Классы

### `DealsFromXLS`

**Описание**: Класс предназначен для парсинга XLS файлов, сгенерированных на платформе AliExpress, и извлечения информации о сделках.

**Методы**:

*   `__init__(language: str, currency: str)`: Инициализирует экземпляр класса `DealsFromXLS` с указанным языком и валютой.
*   `get_next_deal()`: Возвращает следующую сделку из XLS файла.

**Примеры**:

```python
from src.suppliers.aliexpress import DealsFromXLS

deals_parser = DealsFromXLS(language='EN', currency='USD')
```

В данном примере создается экземпляр класса `DealsFromXLS` для парсинга XLS файлов на английском языке с валютой USD.
