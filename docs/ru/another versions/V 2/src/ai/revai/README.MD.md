# revai (rev.com)

## Обзор

Модуль `revai` предназначен для работы с API rev.com, сервиса для транскрибации аудио и видео. Он предоставляет функциональность для взаимодействия с API, включая отправку файлов на транскрибацию, получение результатов и управление заданиями.

## Классы

### `RevAIClient`

**Описание**: Класс для взаимодействия с API Rev.ai.

**Методы**:

- [`__init__`](#__init__)
- [`submit_job`](#submit_job)
- [`get_job`](#get_job)
- [`get_transcript`](#get_transcript)
- [`get_alignment`](#get_alignment)
- [`delete_job`](#delete_job)

#### `__init__`

**Описание**: Инициализирует экземпляр `RevAIClient`.

**Параметры**:
- `access_token` (str): API-ключ для доступа к rev.ai.

#### `submit_job`

**Описание**: Отправляет файл на транскрибацию.

**Параметры**:
- `file_path` (str): Путь к файлу для транскрибации.
- `metadata` (Optional[str], optional): Дополнительные метаданные. По умолчанию `None`.
- `callback_url` (Optional[str], optional): URL для обратного вызова. По умолчанию `None`.
- `skip_diarization` (Optional[bool], optional): Флаг для пропуска диаризации. По умолчанию `None`.
- `skip_punctuation` (Optional[bool], optional): Флаг для пропуска пунктуации. По умолчанию `None`.

**Возвращает**:
- `dict`: Ответ от API с информацией о задании.

**Вызывает исключения**:
- `requests.exceptions.RequestException`: Если произошла ошибка при отправке запроса.

**Параметры**:
- `job_id` (str): Идентификатор задания.

**Возвращает**:
- `dict`: Ответ от API с информацией о задании.

**Вызывает исключения**:
- `requests.exceptions.RequestException`: Если произошла ошибка при отправке запроса.

#### `get_transcript`

#### `get_alignment`

**Параметры**:
