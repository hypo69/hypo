# Модуль make_summary

## Обзор

Модуль `make_summary` предназначен для создания файла `SUMMARY.md`, который используется для компиляции документации средствами `mdbook`. Он рекурсивно обходит папку с исходными файлами `.md` и генерирует список ссылок на все файлы в `SUMMARY.md`.


## Функции

### `make_summary`

**Описание**: Создает файл `SUMMARY.md`, рекурсивно обходя папку.

**Параметры**:

- `docs_dir` (Path): Путь к исходной директории с файлами `.md`.

**Возвращает**:

- `None`: Функция не возвращает значение.

**Вызывает исключения**:

- Любые исключения, возникающие во время работы с файлами.


### `_make_summary`

**Описание**: Рекурсивно обходит папку и создает файл `SUMMARY.md` с главами на основе `.md` файлов.

**Параметры**:

- `src_dir` (Path): Путь к папке с исходными `.md` файлами.
- `summary_file` (Path): Путь для сохранения файла `SUMMARY.md`.

**Возвращает**:

- `bool`: `True`, если файл `SUMMARY.md` успешно создан, `False` в противном случае.

**Вызывает исключения**:

- Любые исключения, возникающие во время работы с файлами.


### `prepare_summary_path`

**Описание**: Формирует путь к файлу `SUMMARY.md`, заменяя часть пути `src` на `docs` и добавляя имя файла.

**Параметры**:

- `src_dir` (Path): Исходный путь с `src`.
- `file_name` (str, optional): Имя файла, который нужно создать. По умолчанию `'SUMMARY.md'`.

**Возвращает**:

- `Path`: Новый путь к файлу `SUMMARY.md`.

**Вызывает исключения**:

- Нет.


## Особенности

- Модуль использует `Pathlib` для работы с путями, что делает его более переносимым.
- Проверяет существование файла `SUMMARY.md` перед его перезаписью.
- Обрабатывает возможные исключения (например, ошибки ввода-вывода) и выводит информацию об ошибках в консоль.
- Рекурсивно обходит папку, собирая все `.md` файлы.
- Сортирует ссылки на файлы в `SUMMARY.md` по имени файла.
- Создает родительскую директорию, если она не существует.


## Использование

Для использования модуля необходимо передать путь к директории с файлами `.md` в качестве аргумента функции `make_summary`. Например:

```python
from pathlib import Path
from hypotez.src.endpoints.hypo69.code_assistant import make_summary

docs_dir = Path("./src/endpoints/hypo69/code_assistant")
make_summary(docs_dir)
```

Это создаст файл `SUMMARY.md` в соответствующей подпапке `docs`.
