# Задача No 87
## Простые кубы, квадраты и степени
## Условия
Число 50 можно записать в виде суммы квадрата простого числа, куба простого числа и четвертой степени простого числа следующим образом: 7² + 2³ + 2⁴.

Фактически, существуют ровно четыре числа меньше пятидесяти, которые могут быть представлены таким образом:

* 8 = 2² + 2³ + 2⁴
* 33 = 5² + 2³ + 2⁴
* 43 = 7² + 3³ + 2⁴
* 50 = 7² + 2³ + 2⁴

Сколько чисел меньше пятидесяти миллионов могут быть записаны в виде суммы квадрата простого числа, куба простого числа и четвертой степени простого числа?
