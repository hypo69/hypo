# Модуль определения языков для AliExpress API

## Обзор

Модуль `languages.py` содержит класс `Language`, который определяет константы для представления различных языков, используемых в AliExpress API.

## Подробней

Этот модуль предоставляет централизованный способ определения поддерживаемых языков в API AliExpress. Он улучшает читаемость и удобство сопровождения кода, за счет использования констант вместо строк.
Языковые коды определены как статические атрибуты класса `Language`.

## Классы

### `Language`

**Описание**: Класс `Language` содержит статические атрибуты, представляющие коды различных языков.

**Как работает класс**:
Класс `Language` предназначен только для хранения констант, представляющих различные языки. Он не имеет методов и служит в качестве пространства имен для языковых кодов.

**Атрибуты**:
- `EN`: Код для английского языка ('EN').
- `RU`: Код для русского языка ('RU').
- `PT`: Код для португальского языка ('PT').
- `ES`: Код для испанского языка ('ES').
- `FR`: Код для французского языка ('FR').
- `ID`: Код для индонезийского языка ('ID').
- `IT`: Код для итальянского языка ('IT').
- `TH`: Код для тайского языка ('TH').
- `JA`: Код для японского языка ('JA').
- `AR`: Код для арабского языка ('AR').
- `VI`: Код для вьетнамского языка ('VI').
- `TR`: Код для турецкого языка ('TR').
- `DE`: Код для немецкого языка ('DE').
- `HE`: Код для иврита ('HE').
- `KO`: Код для корейского языка ('KO').
- `NL`: Код для нидерландского языка ('NL').
- `PL`: Код для польского языка ('PL').
- `MX`: Код для мексиканского испанского языка ('MX').
- `CL`: Код для чилийского испанского языка ('CL').
- `IW`: Код для иврита (альтернативное представление) ('IW').
- `IN`: Код для индийского языка ('IN').

**Примеры**:

```python
from src.suppliers.aliexpress.api.models.languages import Language

# Использование константы для английского языка
english_code = Language.EN
print(english_code)  # Вывод: EN

# Использование константы для русского языка
russian_code = Language.RU
print(russian_code)  # Вывод: RU
```
