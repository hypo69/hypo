# `src/suppliers/aliexpress/gui/header.py`

## Обзор

Этот модуль содержит вспомогательные функции и настройки для работы с GUI в контексте поставщика AliExpress. Основное назначение модуля - определение корневой директории проекта и загрузка общих настроек приложения.

## Функции

### `set_project_root`

**Описание**:
Функция находит корневую директорию проекта, начиная с директории текущего файла и двигаясь вверх, останавливаясь на первой директории, содержащей любой из маркерных файлов.

**Параметры**:
- `marker_files` (tuple, optional): Кортеж имен файлов или директорий, идентифицирующих корень проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена, иначе путь к директории, где расположен скрипт.

## Переменные

### `__root__`

**Описание**:
Путь к корневой директории проекта. Устанавливается при вызове функции `set_project_root`.

### `settings`

**Описание**:
Словарь с настройками приложения. Загружается из файла `settings.json` в корневой директории проекта. Если файл не найден или не удалось распарсить JSON, то присваивается `None`.
