# Модуль hypotez/src/logger

## Обзор

Модуль `hypotez/src/logger` определяет корневой путь к проекту `hypotez`. Все импорты строятся относительно этого пути. В дальнейшем предполагается перенесение этой логики в системную переменную.

## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная с директории текущего файла и поднимаясь по иерархии каталогов. Останавливается на первом каталоге, содержащем указанные файлы-маркеры.

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или каталогов, которые используются для определения корневого каталога проекта. По умолчанию содержит ('pyproject.toml', 'requirements.txt', '.git').

**Возвращает**:

- `Path`: Путь к корневому каталогу проекта, если он найден. В противном случае возвращает директорию, где находится текущий файл.

**Вызывает исключения**:

- Нет.

## Глобальные переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта, полученный с помощью функции `set_project_root()`.

**Тип**: `Path`

### `settings`

**Описание**: Словарь настроек, загруженный из файла `src/settings.json`.

**Тип**: `dict` или `None`

**Примечание**:  Возможные ошибки при загрузке (FileNotFoundError, json.JSONDecodeError) обрабатываются.

### `doc_str`

**Описание**: Строка с содержимым файла `README.MD`.

**Тип**: `str` или `None`

**Примечание**: Возможные ошибки при загрузке (FileNotFoundError, json.JSONDecodeError) обрабатываются.

### `__project_name__`

**Описание**: Название проекта, полученное из `settings.json` или установленное по умолчанию.

**Тип**: `str`

### `__version__`

**Описание**: Версия проекта, полученная из `settings.json` или значение по умолчанию.

**Тип**: `str`


### `__doc__`

**Описание**: Содержимое файла `README.MD`, если он найден.

**Тип**: `str`

### `__details__`

**Описание**: Строка с дополнительными деталями проекта.

**Тип**: `str`

### `__author__`

**Описание**: Автор проекта, полученный из `settings.json` или значение по умолчанию.

**Тип**: `str`

### `__copyright__`

**Описание**: Авторские права, полученные из `settings.json` или значение по умолчанию.

**Тип**: `str`

### `__cofee__`

**Описание**: Ссылка на площадку для поддержки разработчика.

**Тип**: `str`
