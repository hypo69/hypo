# Задача No 622
## Перетасовка карт
## Условия
Рассмотрим колоду из $n$ карт, пронумерованных от 1 до $n$.

Процедуру *совершенной перетасовки* выполняют следующим образом:
колоду карт делят на две равные стопки, верхняя половина карт идет в первую стопку, нижняя половина - во вторую.
Затем карты снова соединяют в одну колоду, чередуя карты из первой и второй стопок.
Если число карт нечетно, то нижняя половина содержит на одну карту больше.
Например, для $n = 5$, карты $\{1, 2, 3, 4, 5\}$ будут перетасованы следующим образом:
первая стопка: $\{1, 2\}$
вторая стопка: $\{3, 4, 5\}$
результат: $\{1, 3, 2, 4, 5\}$

Пусть $p(n)$ - наименьшее количество перетасовок, необходимых для возвращения колоды в исходное состояние.

Например, $p(5) = 4$.
$ \{1, 2, 3, 4, 5\} \rightarrow \{1, 3, 2, 4, 5\} \rightarrow \{1, 2, 4, 3, 5\} \rightarrow \{1, 4, 2, 3, 5\} \rightarrow \{1, 2, 3, 4, 5\} $

Найдите $\sum_{n=1}^{10000} p(n)$.
