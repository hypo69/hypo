# Шаблон страницы чата

## Обзор

Этот файл представляет собой шаблон HTML для страницы чата, взаимодействующей с моделью искусственного интеллекта. Он использует Bootstrap для стилизации и jQuery для асинхронного обмена данными с сервером.

## Оглавление

* [Шаблон страницы чата](#шаблон-страницы-чата)
* [Описание](#описание)

* [Использование](#использование)

## Описание

Страница отображает чат-бокс, поле ввода для пользователя и кнопку отправки. Получение ответа от AI происходит с помощью AJAX запроса на серверный маршрут `/ask`.

## Функциональность

1. **Ввод пользователя:** Пользователь вводит сообщение в поле ввода и нажимает кнопку "Отправить".

2. **Отправка запроса:**  Сообщение пользователя отправляется на сервер с помощью AJAX запроса POST методом на маршрут `/ask`.

3. **Обработка сервером:** Серверная часть обрабатывает запрос и возвращает ответ AI.

4. **Вывод ответа:**  Ответ AI отображается в чат-боксе.

5. **Очистка поля ввода:** После отправки сообщения поле ввода очищается.

## Использование

Шаблон использует Bootstrap для стилизации интерфейса и jQuery для реализации AJAX.  Серверная часть должна быть реализована таким образом, чтобы обрабатывать POST-запросы на `/ask` и возвращать JSON ответ в формате `{ "response": "ответ_ai" }`.
