# Модуль hypotez/src/suppliers/aliexpress/api/version.py

## Обзор

Модуль `version.py` содержит информацию о версии API AliExpress.  Он предоставляет переменные, хранящие информацию о версии, имени модуля, документации, дополнительных данных, типах параметров/возвращаемых значений и авторах.

## Переменные

### `__version__`

**Описание**: Переменная, содержащая версию модуля.

**Значение**: "3.12.0.0.0.4"

### `__name__`

**Описание**: Имя модуля. Если скрипт запускается напрямую, значение будет `"__main__"`.

**Тип**: `str`

### `__doc__`

**Описание**: Строка документации модуля.

**Тип**: `str`


### `__details__`

**Описание**: Дополнительные данные о версии модуля или класса.

**Значение**: "Details about version for module or class"

**Тип**: `str`


### `__annotations__`

**Описание**: Содержит аннотации типов для переменных и функций в модуле.

**Тип**: `Any` (тип аннотаций)

### `__author__`

**Описание**: Имя(на) автора(ов) модуля.

**Значение**: `'hypotez '`

**Тип**: `str`
