# Модуль `src.suppliers.kualastyle`

## Обзор

Модуль `src.suppliers.kualastyle` предоставляет функциональность для работы с поставщиком Kualastyle. Включает в себя инструменты для сбора данных и интеграции с их API.

## Оглавление

- [Переменные](#переменные)
- [Импорты](#импорты)

## Переменные

### `MODE`
- **Описание**: Режим работы модуля. По умолчанию установлен в `dev`.
- **Тип**: `str`

## Импорты

### `from .graber import Graber`

- **Описание**: Импортирует класс `Graber` из модуля `graber`, который находится в текущей директории. Класс `Graber` предназначен для выполнения сбора данных с сайта Kualastyle.
