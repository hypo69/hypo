# Модуль html_escapes

## Обзор

Этот модуль предоставляет словарь для преобразования символов UTF-8 в их HTML-экранированные аналоги.

## Словарь html_escapes

### `html_escapes`

**Описание**: Словарь, содержащий соответствия между символами UTF-8 и их HTML-экранированными представлениями.

**Пример использования**:

```python
from hypotez.src.utils.string.html_escapes import html_escapes

escaped_ampersand = html_escapes['&']  # Получение экранированного символа '&'
```

**Ключи**: Символы UTF-8.

**Значения**: Экранированные HTML-представления ключей.

**Содержание словаря**: Словарь содержит большое количество соответствий для различных символов, включая латинские буквы, знаки препинания, математические символы и др.  Полный список значений см. в коде модуля.
