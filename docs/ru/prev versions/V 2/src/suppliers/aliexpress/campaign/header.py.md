# Модуль `header.py`

## Обзор

Модуль `header.py` предназначен для определения корневой директории проекта, загрузки настроек из файла `settings.json`, чтения документации из `README.MD`, а также определения основных метаданных проекта, таких как имя, версия, авторские права и т.д.

## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная с текущего каталога файла, ища вверх по иерархии, пока не найдет каталог, содержащий один из маркерных файлов.

**Параметры**:
- `marker_files` (tuple, optional): Имена файлов или каталогов для определения корня проекта. По умолчанию (`'__root__'`, `'.git'`).

**Возвращает**:
- `Path`: Путь к корневому каталогу, если он найден, в противном случае — путь к каталогу, где расположен скрипт.

## Глобальные переменные

### `__root__`

**Описание**: Содержит путь к корневой директории проекта. Определяется функцией `set_project_root`.

### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженные из файла `settings.json`.  Если файл не найден или содержит ошибку JSON, значение остается `None`.

### `doc_str`

**Описание**: Строка, содержащая содержимое файла `README.MD`. Если файл не найден или содержит ошибку, значение остается `None`.

### `__project_name__`

**Описание**: Имя проекта. По умолчанию `'hypotez'`, но может быть переопределено из настроек.

### `__version__`

**Описание**: Версия проекта. Загружается из настроек, если они доступны. По умолчанию пустая строка `''`.

### `__doc__`

**Описание**: Документация проекта, загруженная из `README.MD`. Если `README.MD` не найден, то это пустая строка `''`.

### `__details__`

**Описание**: Детали проекта. В текущей версии всегда пустая строка `''`.

### `__author__`

**Описание**: Автор проекта. Загружается из настроек, если они доступны. По умолчанию пустая строка `''`.

### `__copyright__`

**Описание**: Авторские права на проект. Загружаются из настроек, если они доступны. По умолчанию пустая строка `''`.

### `__cofee__`

**Описание**: Сообщение, призывающее поддержать разработчика. Загружается из настроек, если они доступны, иначе используется значение по умолчанию: "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69".
