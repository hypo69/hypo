# Модуль: src.suppliers.aliexpress.api.models

## Обзор

Модуль `src.suppliers.aliexpress.api.models` содержит классы и структуры данных, используемые для взаимодействия с API AliExpress. Он определяет модели данных для языков, валют, параметров запросов, партнерских ссылок, горячих продуктов, продуктов и категорий.

## Подробнее

Этот модуль предоставляет необходимые модели данных для работы с API AliExpress, упрощая процесс обработки и обмена данными между приложением и API. Он определяет структуры данных, представляющие различные сущности, такие как продукты, категории, валюты и языки, а также параметры запросов и ответы API.
