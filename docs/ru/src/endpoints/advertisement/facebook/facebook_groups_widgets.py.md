# Модуль для создания выпадающего списка групп Facebook

## Обзор

Модуль `facebook_groups_widgets.py` предназначен для создания и отображения выпадающего списка групп Facebook. Он использует библиотеку `ipywidgets` для создания интерактивного элемента интерфейса, позволяющего пользователю выбирать группы из списка, загруженного из JSON-файла.

## Подробней

Этот модуль предоставляет класс `FacebookGroupsWidget`, который инициализируется путем указания пути к JSON-файлу, содержащему информацию о группах Facebook. Класс создает выпадающий список на основе этих данных и отображает его. Это полезно для создания пользовательских интерфейсов, где необходимо выбирать группы Facebook для различных целей, например, для размещения рекламы или сбора данных.

## Классы

### `FacebookGroupsWidget`

**Описание**: Класс `FacebookGroupsWidget` создает выпадающий список с URL групп Facebook из предоставленного JSON-файла.

**Как работает класс**:

1.  При инициализации класса `FacebookGroupsWidget` загружаются данные о группах Facebook из JSON-файла, путь к которому передается в конструктор.
2.  Метод `create_dropdown` создает виджет выпадающего списка (`Dropdown`) на основе загруженных данных. Ключи из JSON-файла используются в качестве опций для выпадающего списка.
3.  Метод `display_widget` отображает созданный виджет выпадающего списка, делая его видимым в интерфейсе пользователя.

**Методы**:

*   `__init__(self, json_file_path: Path)`: Инициализирует виджет с выпадающим списком для групп Facebook.
*   `create_dropdown(self) -> Dropdown`: Создает и возвращает виджет выпадающего списка на основе данных групп.
*   `display_widget(self)`: Отображает виджет выпадающего списка.

#### `__init__(self, json_file_path: Path)`

```python
    def __init__(self, json_file_path: Path):
        """
        Инициализация виджета с выпадающим списком для групп Facebook.

        Args:
            json_file_path (Path): Путь к JSON-файлу, содержащему информацию о группах Facebook.
        """
```

**Назначение**: Инициализация экземпляра класса `FacebookGroupsWidget`.

**Как работает функция**:

1.  Функция принимает путь к JSON-файлу, содержащему информацию о группах Facebook.
2.  Загружает данные из JSON-файла, используя функцию `j_loads_ns` из модуля `src.utils.jjson`, и сохраняет их в атрибуте `groups_data` как `SimpleNamespace`.
3.  Вызывает метод `create_dropdown` для создания виджета выпадающего списка на основе загруженных данных и сохраняет его в атрибуте `dropdown`.

**Параметры**:

*   `json_file_path` (Path): Путь к JSON-файлу, содержащему информацию о группах Facebook.

#### `create_dropdown(self) -> Dropdown`

```python
    def create_dropdown(self) -> Dropdown:
        """ Создает и возвращает виджет выпадающего списка на основе данных групп.

        Returns:
            Dropdown: Виджет выпадающего списка с URL групп Facebook.
        """
```

**Назначение**: Создает и возвращает виджет выпадающего списка на основе данных групп.

**Как работает функция**:

1.  Извлекает ключи (URL групп Facebook) из атрибута `groups_data.__dict__`.
2.  Создает экземпляр класса `Dropdown` из библиотеки `ipywidgets`.
3.  Устанавливает параметры выпадающего списка:
    *   `options`: Список URL групп Facebook, полученных из данных.
    *   `description`: Текст описания выпадающего списка ('Facebook Groups:').
    *   `disabled`: Устанавливается в `False`, чтобы выпадающий список был активен.
4.  Возвращает созданный виджет выпадающего списка.

**Возвращает**:

*   `Dropdown`: Виджет выпадающего списка с URL групп Facebook.

#### `display_widget(self)`

```python
    def display_widget(self):
        """ Отображает виджет выпадающего списка."""
```

**Назначение**: Отображает виджет выпадающего списка.

**Как работает функция**:

1.  Использует функцию `display` из модуля `IPython.display` для отображения виджета выпадающего списка, хранящегося в атрибуте `self.dropdown`.

**Параметры**:
