# Задача No 281
## Замечательные фракталы

## Условия

Пусть *f*(n) — наименьшее значение *m*, такое что все положительные целые числа, не превосходящие *n*, можно представить в виде суммы не более чем *m* чисел, каждое из которых равно либо некоторой целой степени 2, либо некоторой целой степени 3.

Например, первые несколько значений *f*(n) таковы:

*   f(1) = 1
*   f(2) = 1
*   f(3) = 1
*   f(4) = 2
*   f(5) = 2
*   f(6) = 2
*   f(7) = 2
*   f(8) = 2
*   f(9) = 2
*   f(10) = 2
*   f(11) = 3
*   f(12) = 2
*   f(13) = 3
*   f(14) = 3
*   f(15) = 3

Сумма *f*(n) для 1 ≤ *n* ≤ 100 равна 402.

Найдите сумму *f*(n) для 1 ≤ *n* ≤ 10<sup>6</sup>.
