# Модуль `hypotez/src/main.py`

## Обзор

Этот модуль предоставляет интерактивное меню для запуска предопределенных скриптов. Он принимает ввод пользователя для выбора и выполнения скриптов 1 или 2.

## Функции

### `script1()`

**Описание**: Выполняет скрипт 1.

**Вызывает исключения**:
- Нет описанных исключений.

### `script2()`

**Описание**: Выполняет скрипт 2.

**Вызывает исключения**:
- Нет описанных исключений.

### `show_help()`

**Описание**: Отображает справку по доступным командам.

**Вызывает исключения**:
- Нет описанных исключений.


### `interactive_menu()`

**Описание**: Интерактивное меню для выбора и запуска скриптов.

**Вызывает исключения**:
- Нет описанных исключений.


### `main()`

**Описание**: Основная функция для обработки аргументов командной строки и запуска меню.

**Параметры**:
- Нет описанных параметров.

**Вызывает исключения**:
- Нет описанных исключений.


## Использование

Пример использования модуля:

```python
python main.py  # Запускает интерактивное меню
python main.py --help # Отображает справку
```

## Модули

- `src.utils.jjson`: Модуль для работы с JSON данными.
- `src.logger`: Модуль для ведения логов.
