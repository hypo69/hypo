# Модуль `header`

## Обзор

Модуль `header` определяет основные настройки и пути для проекта, а также загружает общие данные, такие как настройки из `settings.json` и документацию из `README.MD`. Он также устанавливает корень проекта для корректной работы с путями.

## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная с каталога текущего файла, и останавливается при первом обнаружении маркеров.

**Параметры**:
- `marker_files` (tuple, optional):  Кортеж имен файлов или каталогов для идентификации корня проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневому каталогу, если найден, иначе путь к каталогу, где расположен скрипт.

## Глобальные переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта. Определяется функцией `set_project_root`.

### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json`. Если файл не найден или имеет неверный формат, значение будет `None`.

### `doc_str`

**Описание**: Строка с документацией, загруженная из файла `README.MD`. Если файл не найден или имеет неверный формат, значение будет `None`.

### `__project_name__`

**Описание**: Название проекта, извлеченное из файла настроек или "hypotez" по умолчанию.

### `__version__`

**Описание**: Версия проекта, извлеченная из файла настроек или пустая строка по умолчанию.

### `__doc__`

**Описание**: Документация проекта, загруженная из файла `README.MD` или пустая строка, если файл не найден.

### `__details__`

**Описание**: Строка для дополнительных деталей, пока пустая.

### `__author__`

**Описание**: Автор проекта, извлеченный из файла настроек или пустая строка по умолчанию.

### `__copyright__`

**Описание**: Авторское право проекта, извлеченное из файла настроек или пустая строка по умолчанию.

### `__cofee__`

**Описание**: Сообщение с предложением поддержать разработчика чашечкой кофе. Загружается из настроек или используется значение по умолчанию.
