# Задача No 99
## Наибольшее число в списке

## Условия

Сравнивая два целых положительных числа, записанных в виде a^b и c^d, не всегда легко определить, какое из них больше. Например,

2^11 = 2048 < 3^7 = 2187

Однако

632382^518061 > 519432^525806.

Первая строка текстового файла, `base_exp.txt`, содержит 1000 строк, каждая из которых представляет пару чисел, основание и показатель, разделенных запятой.

Какая строка в этом файле содержит наибольшее значение a^b?

**Примечание:**
Файл `base_exp.txt` можно получить по ссылке: https://projecteuler.net/project/resources/p099_base_exp.txt
