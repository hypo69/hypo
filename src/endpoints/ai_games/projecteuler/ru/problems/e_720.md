# Задача No 720
## Размещения с повторениями
## Условия
Пусть *S*(*n*, *k*) - количество *k*-элементных размещений с повторениями из набора *n* элементов. Например, *S*(3, 2) = 9, поскольку существует 9 двузначных чисел, которые могут быть сформированы из набора {1, 2, 3}:

11, 12, 13, 21, 22, 23, 31, 32, 33.

Аналогично, *S*(3, 3) = 27.

Пусть *F*(*n*, *k*, *m*) - количество *k*-элементных размещений с повторениями из набора *n* элементов, где *m* - наибольшее значение, которое встречается в размещении, например:

*   *F*(3, 2, 1) = 1, поскольку есть только одно такое размещение: 11.
*   *F*(3, 2, 2) = 4, поскольку есть 4 размещения: 11, 12, 21, 22.
*   *F*(3, 2, 3) = 9, поскольку есть 9 размещений: 11, 12, 13, 21, 22, 23, 31, 32, 33.

Пусть *G*(*n*, *k*) = Σ(*F*(*n*, *k*, *m*))<sup>2</sup> для всех *m* от 1 до *n*.
Например, *G*(3, 2) = 1<sup>2</sup> + 4<sup>2</sup> + 9<sup>2</sup> = 98.

Известно, что *G*(7, 10) mod 1,000,000,007 = 916,855,915.

Найдите *G*(10<sup>7</sup>, 10<sup>12</sup>) mod 1,000,000,007.
