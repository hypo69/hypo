# Задача No 392
## Заполнение многоугольников
## Условия
Пусть S - множество точек (x, y), где x и y - целые числа, 0 ≤ x ≤ n, 0 ≤ y ≤ n.
Пусть C - множество всех прямых отрезков между любыми двумя различными точками из S.
Мы выбираем подмножество T из C, так что:

- Никакие два отрезка в T не пересекаются (за исключением, возможно, в конечных точках отрезков).
- Любой отрезок в T имеет по крайней мере одну конечную точку (с координатами x и y), где x = 0 или x = n.
- Для любой точки из S, которая не является конечной точкой отрезка в T, существует хотя бы два отрезка в T, конечные точки которых содержат эту точку.

Назовем T *заполнением* множества S.

Пусть F(n) - количество отрезков в T, который является заполнением S, с наименьшей возможной длиной, и пусть L(n) - длина заполнения T.

Например:

- F(2) = 4
- F(3) = 6
- F(10) = 30
- L(2) = 4 + 2√2 ≈ 6.828427125
- L(3) = 6 + 4√2 ≈ 11.65685425
- L(5) = 10 + 8√2 ≈ 21.31370850
- L(100) ≈ 2828.427124746

Найдите L(10<sup>5</sup>) и приведите ответ с точностью до 8 знаков после запятой.
