## Обзор

Этот документ описывает структуру и назначение файла `morlevi_categories_printers.json`, который содержит настройки для сценария парсинга категорий принтеров с сайта morlevi.co.il.

## Структура JSON

Файл содержит JSON-объект с одним ключом `"scenarios"`, который, в свою очередь, содержит объект с настройками сценария.

```json
{
  "scenarios": {
    "url": "https://www.morlevi.co.il/Cat/60",
    "checkbox": false,
    "active": true,
    "condition":"new",
    "presta_categories": "151,209"
  }
}
```

## Описание полей

### `scenarios`

-   **Описание**: Объект, содержащий настройки для сценария парсинга.

#### `url`

-   **Описание**: URL-адрес страницы, с которой необходимо парсить данные.
-   **Тип**: `string`
-   **Значение по умолчанию**: `null`
-   **Пример**: `"https://www.morlevi.co.il/Cat/60"`

#### `checkbox`

-   **Описание**: Флаг, указывающий, нужно ли использовать чекбокс.
-   **Тип**: `boolean`
-   **Значение по умолчанию**: `false`
-   **Пример**: `false`

#### `active`

-   **Описание**: Флаг, указывающий, активен ли сценарий.
-   **Тип**: `boolean`
-   **Значение по умолчанию**: `false`
-   **Пример**: `true`

#### `condition`
-   **Описание**: Состояние товара.
-   **Тип**: `string`
-   **Значение по умолчанию**: `null`
-   **Пример**: `"new"`

#### `presta_categories`

-   **Описание**: Строка, содержащая идентификаторы категорий PrestaShop, к которым необходимо привязать товары.
-   **Тип**: `string`
-  **Значение по умолчанию**: `null`
-   **Пример**: `"151,209"`
