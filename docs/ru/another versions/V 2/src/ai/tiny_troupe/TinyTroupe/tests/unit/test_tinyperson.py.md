# Тестирование модуля `test_tinyperson.py`

## Обзор

Данный файл содержит набор модульных тестов для проверки корректности работы класса `TinyPerson`. Тесты охватывают различные аспекты поведения агента, включая взаимодействие с окружением, обработку стимулов, управление целями и контекстом, а также сохранение и загрузку спецификаций агента.

### `test_act`

**Описание**:
Проверяет, что агент корректно реагирует на запрос, выполняя как минимум одно действие, включая `TALK` и завершающее действие `DONE`.

**Параметры**:
- `setup`: Fixture для настройки окружения.

**Возвращает**:
- `None`: Функция ничего не возвращает.

**Вызывает исключения**:
- `AssertionError`: Возникает, если агент не выполняет необходимые действия.
