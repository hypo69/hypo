# Модуль hypotez/src/logger/header.py

## Обзор

Этот модуль содержит функции для определения корневого каталога проекта и загрузки настроек из файла `settings.json`. Он также предоставляет доступ к документации проекта (README.md).

## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная с текущего файла и поднимаясь вверх по директориям. Поиск останавливается на первой директории, содержащей один из указанных файлов маркеров.

**Параметры**:
- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, используемых для определения корневого каталога проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневому каталогу проекта, если он найден. В противном случае возвращает путь к директории, где находится текущий файл.

**Вызывает исключения**:
- Нет


### `get_project_settings`


**Описание**:  Загружает настройки проекта из файла `settings.json` в корне проекта.

**Параметры**:
- Нет

**Возвращает**:
- `dict | None`: Словарь с настройками проекта, если файл `settings.json` существует и содержит валидные данные. В противном случае возвращает `None`.


## Константы

### `MODE`

**Описание**: Строковая константа, хранящая режим работы, например 'dev'.

**Значение**: `'dev'`


## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта.

**Тип**: `Path`

### `settings`

**Описание**: Словарь с настройками проекта.

**Тип**: `dict` | `None`


### `doc_str`

**Описание**: Содержимое файла README.md.


**Тип**: `str` | `None`


### `__project_name__`

**Описание**: Название проекта. Извлекается из настроек, по умолчанию 'hypotez'.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта. Извлекается из настроек, по умолчанию пустая строка.

**Тип**: `str`


### `__doc__`

**Описание**: Документация проекта (README.md). Извлекается из файла, по умолчанию пустая строка.

**Тип**: `str`


### `__details__`

**Описание**:  Дополнительные детали проекта (по умолчанию пустая строка).

**Тип**: `str`

### `__author__`

**Описание**: Автор проекта. Извлекается из настроек, по умолчанию пустая строка.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права. Извлекается из настроек, по умолчанию пустая строка.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка для поддержки разработчика. Извлекается из настроек, по умолчанию ссылка на Boosty.

**Тип**: `str`

## Обработка исключений

В коде используется обработка исключений `try...excep` для перехвата `FileNotFoundError` и `json.JSONDecodeError` при работе с файлами настроек. При возникновении исключения, соответствующие переменные инициализируются значением по умолчанию.


## Использование

Этот модуль предназначен для использования внутри проекта для получения доступа к корневому каталогу и настройкам проекта.  Он используется для корректного импорта других модулей и доступа к необходимым ресурсам.
