# Модуль `test_ali_campaign_editor_jupyter_widgets.py`

## Обзор

Этот модуль содержит тесты для функций, связанных с обработкой файлов в проекте.  Включает тесты для сохранения текста в файл (`save_text_file`), чтения текста из файла (`read_text_file`), получения списка файлов из директории (`get_filenames`) и получения списка директорий из пути (`get_directory_names`). Все тесты основаны на использовании патчинга (mock-объектов) для имитации работы внешних зависимостей.

## Функции

### `test_save_text_file`

**Описание**: Тестирует функцию `save_text_file`. Использует патчинг для имитации работы функций `Path.open`, `Path.mkdir` и `logger`.

**Аргументы**:

- `mock_logger` (MagicMock): Мок-объект для `logger`.
- `mock_mkdir` (MagicMock): Мок-объект для `Path.mkdir`.
- `mock_file_open` (MagicMock): Мок-объект для `Path.open`.

**Пример использования**:

```python
>>> test_save_text_file()
```

**Проверяемые действия**:

- Функция `save_text_file` должна корректно открыть файл на запись в кодировке UTF-8.
- Функция `save_text_file` должна корректно записать переданный текст в файл.
- Функция `save_text_file` должна вызвать `Path.mkdir` для создания директории, если она не существует.


### `test_read_text_file`

**Описание**: Тестирует функцию `read_text_file`. Использует патчинг для имитации работы функции `Path.open`.

**Аргументы**:

- `mock_file_open` (MagicMock): Мок-объект для `Path.open`.


**Возвращаемое значение**:

- str: Содержимое файла.


**Пример использования**:

```python
>>> content: str = test_read_text_file()
>>> print(content)
'This is a test.'
```

**Проверяемые действия**:

- Функция `read_text_file` должна корректно открыть файл на чтение в кодировке UTF-8.
- Функция `read_text_file` должна корректно прочитать текст из файла.
- Функция должна возвращать содержимое файла.


### `test_get_filenames`

**Описание**: Тестирует функцию `get_filenames`. Использует патчинг для имитации работы функции `Path.iterdir`.

**Возвращаемое значение**:

- list[str]: Список имен файлов в директории.


**Пример использования**:

```python
>>> filenames: list[str] = test_get_filenames()
>>> print(filenames)
['file1.txt', 'file2.txt']
```

**Проверяемые действия**:

- Функция `get_filenames` должна корректно получить список имен файлов из переданной директории.


### `test_get_directory_names`

**Описание**: Тестирует функцию `get_directory_names`. Использует патчинг для имитации работы функции `Path.iterdir`.

**Возвращаемое значение**:

- list[str]: Список имен директорий.


**Пример использования**:

```python
>>> directories: list[str] = test_get_directory_names()
>>> print(directories)
['dir1', 'dir2']
```

**Проверяемые действия**:

- Функция `get_directory_names` должна корректно получить список имен директорий из переданного пути.
