# Модуль `affiliate_link`

## Обзор

Модуль `affiliate_link` определяет класс `AffiliateLink`, который используется для представления партнерской ссылки с AliExpress. Класс содержит информацию о самой ссылке и источнике ее получения.

## Оглавление

- [Классы](#классы)
    - [`AffiliateLink`](#affiliatelink)

## Классы

### `AffiliateLink`

**Описание**: Класс `AffiliateLink` представляет партнерскую ссылку с AliExpress.

**Атрибуты**:

- `promotion_link` (str): Партнерская ссылка.
- `source_value` (str): Источник получения партнерской ссылки.
