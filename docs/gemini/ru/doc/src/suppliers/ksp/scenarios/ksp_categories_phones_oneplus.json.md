## Обзор

Файл `ksp_categories_phones_oneplus.json` содержит конфигурацию для сбора данных о телефонах марки ONEPLUS с сайта KSP. Каждый сценарий описывает конкретную модель телефона, включая её URL на сайте, состояние (новый), категорию PrestaShop и другие параметры.

## Структура
Файл представляет собой JSON-объект с единственным ключом `scenarios`, значение которого является словарем, где ключи — это названия моделей телефонов, а значения — словари с параметрами для каждого сценария.
