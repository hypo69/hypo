# Модуль borderify.js

## Обзор

Данный скрипт изменяет стиль границы элемента `document.body`. Он устанавливает границу в 5 пикселей красного цвета.  Также вызывает всплывающее окно `alert()`.


## Функции

### `null`

**Описание**: В данном файле нет функций, определённых как функции. Все действия выполняются при выполнении скрипта.


**Действия**:
* Устанавливает свойство `border` элемента `document.body` со значением "5px solid red".
* Вызывает всплывающее окно `alert()`.
