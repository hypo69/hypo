# UGLY: משחק ניחושים פשוט

## Обзор

Этот файл реализует простую игру-угадайку, в которой компьютер выбирает случайное число от 1 до 10, а игрок пытается его угадать. Игра заканчивается, как только игрок правильно угадывает число.

## Функции

### `main`

**Описание**: Основная логика игры. Инициализирует случайное число, запрашивает ввод пользователя и проверяет, угадал ли пользователь число.

**Параметры**:
- Нет параметров.

**Возвращает**:
- Нет возвращаемого значения.

**Вызывает исключения**:
- `ValueError`: Возникает, если пользователь вводит некорректное значение (не целое число).
```python
import random

# אתחול המספר המוגרל באופן אקראי בין 1 ל-10
targetNumber = random.randint(1, 10)

# קבלת ניחוש מהמשתמש
try:
    userGuess = int(input("נחש מספר בין 1 ל-10: "))
except ValueError as ex:
    print("אנא הזן מספר שלם.")
    exit() # סיום התוכנית במקרה של קלט שגוי.

# בדיקה האם הניחוש נכון
if userGuess == targetNumber:
    print("YOU GOT IT!") # הודעה על ניצחון
else:
    print("YOU UGLY LOSER!") # הודעה על הפסד
```
