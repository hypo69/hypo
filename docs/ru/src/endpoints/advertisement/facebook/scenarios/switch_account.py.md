# Модуль `switch_account`

## Обзор

Модуль предназначен для переключения между аккаунтами в Facebook с использованием Selenium WebDriver. Он содержит функцию `switch_account`, которая проверяет наличие кнопки "Переключить" и нажимает на неё.

## Подробнее

Этот модуль является частью системы автоматизации для работы с Facebook. Он использует предопределенные локаторы элементов интерфейса для поиска и взаимодействия с кнопкой переключения аккаунта.

## Функции

### `switch_account`

```python
def switch_account(driver: Driver):
    """Если есть кнопка `Переключить` - нажимаю её"""
```

**Назначение**: Переключает аккаунт в Facebook, если доступна кнопка "Переключить".

**Как работает функция**:
1. Функция принимает инстанс WebDriver в качестве аргумента.
2. Использует метод `execute_locator` для поиска кнопки "Переключить".
3. Если кнопка найдена, функция нажимает на неё, переключая аккаунт.

**Параметры**:
- `driver` (Driver): Инстанс WebDriver, используемый для управления браузером.

**Возвращает**:
- `None`: Функция ничего не возвращает.

**Вызывает исключения**:
- Функция не вызывает явных исключений, но может генерировать исключения, если элемент не найден или недоступен.

**Примеры**:

```python
# Пример использования функции switch_account
from src.webdriver.driver import Driver
from selenium import webdriver

# Создание инстанса драйвера
driver = Driver(webdriver.Chrome)

# Вызов функции переключения аккаунта
switch_account(driver)
