# Модуль `hypotez/src/suppliers/aliexpress/scenarios/affiliate_links_shortener_scenario.py`

## Обзор

Этот модуль содержит функцию `get_short_affiliate_link`, предназначенную для получения сокращенной ссылки из ссылки-аффилиата AliExpress с использованием веб-драйвера. Модуль использует локаторы, загруженные из файла `affiliate_links_shortener.json`, для взаимодействия с веб-сайтом.

## Функции

### `get_short_affiliate_link`

**Описание**: Функция генерирует сокращенную ссылку для аффилиатной ссылки AliExpress. Она взаимодействует с веб-страницей через веб-драйвер, вводит исходную ссылку, нажимает кнопку получения сокращенной ссылки, ожидает обновления страницы, извлекает полученную сокращенную ссылку, открывает новую вкладку с этой ссылкой, проверяет корректность ссылки и возвращает ее, или выводит сообщение об ошибке и возвращает пустую строку при возникновении проблем.

**Параметры**:

- `d` (Driver): Объект веб-драйвера, необходимый для взаимодействия с браузером.
- `url` (str): Исходная ссылка-аффилиат.

**Возвращает**:

- `str`: Сокращенная ссылка или пустая строка в случае ошибки.

**Вызывает исключения**:

- Нет.  Возможные ошибки обрабатываются внутри функции, вызывая логирование, но не прерывая дальнейшее выполнение программы.
