# Описание файла `store.json`

## Обзор

Файл `store.json` содержит JSON-объект с локаторами элементов для страницы магазина поставщика Ivory.
Этот файл используется для определения путей к различным элементам страницы, таким как кнопки, текстовые поля и другие интерактивные элементы, необходимые для автоматизации тестирования и взаимодействия с веб-интерфейсом.

## Структура файла

Файл содержит JSON-объект с следующей структурой:

```json
{
    "store": {
        "catalog": {
             "name": "...",
             "category_button": "...",
             "show_all_button":"...",
             "product_card": "..."

        },
        "product": {
            "price": "...",
            "add_to_cart_button": "...",
            "quantity_field": "..."
        },
        "cart": {
             "product_name":"...",
             "product_price":"...",
             "product_quantity":"...",
             "remove_button":"...",
             "checkout_button":"...",
             "cart_empty_text":"..."
        },
         "modal": {
            "close_button": "..."
        },
        "auth":{
            "email_field": "...",
            "password_field": "...",
            "submit_button":"..."
        },
        "profile":{
           "profile_button":"...",
           "logout_button":"..."
        },
         "search":{
            "input":"...",
            "submit_button":"..."
         },
        "main_page":{
            "cart_button":"..."
        }
    }
}
```

## Описание ключей

### `store`

Главный ключ, содержащий все локаторы для магазина.

#### `catalog`

Локаторы для элементов каталога товаров.

-   `name`: Локатор для имени каталога.
-   `category_button`: Локатор для кнопки категории товара.
-   `show_all_button`: Локатор для кнопки показа всех товаров.
-   `product_card`: Локатор для карточки товара.

#### `product`

Локаторы для элементов страницы продукта.

-   `price`: Локатор для цены товара.
-   `add_to_cart_button`: Локатор для кнопки добавления товара в корзину.
-   `quantity_field`: Локатор для поля ввода количества товара.

#### `cart`

Локаторы для элементов страницы корзины.

-   `product_name`: Локатор для имени товара в корзине.
-   `product_price`: Локатор для цены товара в корзине.
-   `product_quantity`: Локатор для количества товара в корзине.
-   `remove_button`: Локатор для кнопки удаления товара из корзины.
-   `checkout_button`: Локатор для кнопки оформления заказа.
-    `cart_empty_text`: Локатор для текста, когда корзина пуста.

#### `modal`

Локаторы для элементов модального окна.

-   `close_button`: Локатор для кнопки закрытия модального окна.

#### `auth`

Локаторы для элементов формы аутентификации.

-   `email_field`: Локатор для поля ввода email.
-   `password_field`: Локатор для поля ввода пароля.
-   `submit_button`: Локатор для кнопки отправки формы.

#### `profile`

Локаторы для элементов профиля пользователя.

-   `profile_button`: Локатор для кнопки перехода в профиль.
-   `logout_button`: Локатор для кнопки выхода из профиля.

#### `search`

Локаторы для элементов поиска.

-   `input`: Локатор для поля ввода поискового запроса.
-   `submit_button`: Локатор для кнопки отправки поискового запроса.

#### `main_page`

Локаторы для элементов главной страницы.

-   `cart_button`: Локатор для кнопки перехода в корзину.
