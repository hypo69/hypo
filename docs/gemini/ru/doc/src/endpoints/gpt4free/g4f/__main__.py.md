# Модуль запуска API GPT4Free

## Обзор

Этот модуль предназначен для запуска API GPT4Free. Он использует аргументы командной строки, чтобы определить, как запустить API, включая возможность запуска графического интерфейса.

## Подробнее

Этот модуль является точкой входа для запуска API GPT4Free. Он анализирует аргументы командной строки, предоставленные пользователем, и использует их для настройки и запуска API. Если графический интерфейс не указан в аргументах, он будет включен по умолчанию.
