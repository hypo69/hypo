# Модуль `experimentation`

## Обзор

Этот модуль предоставляет классы для проведения экспериментов, в частности, для рандомизации выбора между двумя вариантами и последующего дерандомизации.

## Классы

### `ABRandomizer`

**Описание**: Класс `ABRandomizer` используется для рандомизации выбора между двумя вариантами (например, A и B) и последующего дерандомизации.  Этот класс сохраняет информацию о рандомизации, чтобы иметь возможность восстановить исходный порядок.

**Методы**:

#### `__init__`

**Описание**: Инициализирует экземпляр класса `ABRandomizer`.

**Параметры**:
- `real_name_1` (str): Название первого варианта в исходных данных.
- `real_name_2` (str): Название второго варианта в исходных данных.
- `blind_name_a` (str): Название первого варианта, видимое пользователю.
- `blind_name_b` (str): Название второго варианта, видимое пользователю.
- `passtrough_name` (list): Список названий, которые не подлежат рандомизации и всегда возвращаются как есть.
- `random_seed` (int): Зерно для генератора случайных чисел.

#### `randomize`

**Описание**: Рандомизирует выбор между двумя вариантами для заданного индекса.

**Параметры**:
- `i` (int): Индекс элемента.
- `a` (str): Первый вариант.
- `b` (str): Второй вариант.

**Возвращает**:
- tuple(str, str): Кортеж, содержащий рандомизированные варианты.


#### `derandomize`

**Описание**: Дерандомизирует выбор для заданного индекса.

**Параметры**:
- `i` (int): Индекс элемента.
- `a` (str): Первый вариант.
- `b` (str): Второй вариант.

**Возвращает**:
- tuple(str, str): Кортеж, содержащий исходные варианты.

**Вызывает исключения**:
- `Exception`: Если рандомизация для данного элемента не найдена.

#### `derandomize_name`

**Описание**: Восстанавливает исходное значение из рандомизированного имени.

**Параметры**:
- `i` (int): Индекс элемента.
- `blind_name` (str): Название варианта, выбранное пользователем.


**Возвращает**:
- str: Исходное название варианта.

**Вызывает исключения**:
- `Exception`: Если выбранное название не распознано.

### `Intervention`

**Описание**: Класс `Intervention` предназначен для реализации вмешательств в системе.

**Методы**:

#### `__init__`

**Описание**: Инициализирует экземпляр класса `Intervention`.

**Параметры**:
- `agent` (TinyPerson): Агент, на котором будет произведено вмешательство.
- `agents` (list): Список агентов, на которых будет произведено вмешательство.
- `environment` (TinyWorld): Окружение, на котором будет произведено вмешательство.
- `environments` (list): Список окружений, на которых будет произведено вмешательство.


**Вызывает исключения**:
- `Exception`: Если предоставлены и `agent`, и `agents`, или `environment` и `environments`.
- `Exception`: Если ни один из параметров не предоставлен.

#### `check_precondition`

**Описание**: Проверяет выполнение условий для вмешательства.


**Вызывает исключения**:
- `NotImplementedError`: Метод не реализован.

#### `apply`

**Описание**: Применяет вмешательство.


**Вызывает исключения**:
- `NotImplementedError`: Метод не реализован.

#### `set_textual_precondition`

**Описание**: Устанавливает текстовое условие для вмешательства.

**Параметры**:
- `text` (str): Текст условия.


#### `set_functional_precondition`

**Описание**: Устанавливает функциональное условие для вмешательства.

**Параметры**:
- `func` (function): Функция условия.


#### `set_effect`

**Описание**: Устанавливает эффект вмешательства.

**Параметры**:
- `effect_func` (function): Функция, определяющая эффект вмешательства.
