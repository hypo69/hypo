# Модуль hypotez/src/bots/discord

## Обзор

Этот модуль содержит константы и настройки для бота Discord.  Он предоставляет переменную `MODE`, которая определяет режим работы бота.

## Константы

### `MODE`

**Описание**: Переменная, определяющая режим работы бота.

**Значение**:  `'development'`

## Использование

Этот модуль предоставляет лишь константу `MODE` и не содержит функций или классов.  Для использования просто импортируйте необходимую константу.
