# dot.py - Преобразование DOT-файлов в PNG

Файл: `C:\Users\user\Documents\repos\hypotez\src\utils\convertors\dot.py`

**Описание:**

Этот скрипт преобразует файлы в формате DOT в изображения PNG с помощью библиотеки Graphviz. Он предназначен для создания визуальных представлений графов, заданных в DOT-нотации.

**Функции:**

* **`dot2png(dot_file: str, png_file: str) -> None`:**
    * Принимает путь к файлу DOT (`dot_file`) и путь для сохранения выходного файла PNG (`png_file`).
    * Читает содержимое файла DOT.
    * Создаёт объект `Source` из библиотеки `graphviz` с прочитанным содержимым.
    * Устанавливает формат вывода в PNG.
    * Сохраняет изображение в указанный файл (`png_file`).
    * **Обработка ошибок:**
        * `FileNotFoundError`: Если файл DOT не найден.
        * `Exception`: Для других ошибок при преобразовании. Выводит подробную информацию об ошибке.
    * **Документация:** Предоставляет подробное описание, включая пример использования, синтаксис DOT и примеры вызовов из командной строки.  **Важно:** Пример использования DOT-файла в документации некорректный. Пример `example.dot` демонстрирует _ориентированный_ граф.

**Использование из командной строки:**

```bash
python dot.py <путь_к_файлу_dot> <путь_к_файлу_png>
```

Например:

```bash
python dot.py example.dot output.png
```

**Требования:**

* Библиотека `graphviz` должна быть установлена.

**Примечания:**

* Код корректно обрабатывает ошибки и предоставляет полезные сообщения об ошибках.
* Документация является исчерпывающей и включает примеры.
* Пример DOT-файла в документации должен быть скорректирован для корректной работы.


**Рекомендации по улучшению:**
