# Модуль background.js

## Обзор

Этот модуль отвечает за обработку кликов по значку расширения в адресной строке браузера Firefox. При нажатии на значок выполняется сценарий, находящийся в файле `contentScript.js`, для текущей вкладки.

## Функции

### `browser.browserAction.onClicked.addListener`

**Описание**: Этот слушатель реагирует на клики по значку расширения.

**Параметры**:

- `(tab)` (object): Объект, содержащий информацию о текущей вкладке, на которой был нажат значок.

**Возвращает**:

-  (не используется):  Функция не возвращает значения.

**Вызывает исключения**:

-  (не применимо):  Функция не вызывает исключений.


### `browser.scripting.executeScript`

**Описание**: Выполняет JavaScript-код, находящийся в указанных файлах, в контексте указанной вкладки.

**Параметры**:

- `(options)` (object): Объект настроек для выполнения скрипта.
    - `target` (object): Параметры для определения контекста выполнения скрипта.
        - `tabId` (int): ID вкладки, в контексте которой должен быть выполнен скрипт.
    - `files` (array): Массив путей к файлам с JavaScript-кодом.


**Возвращает**:

- (не используется): Функция не возвращает значения.


**Вызывает исключения**:

- (не применимо): Функция не вызывает исключений.
