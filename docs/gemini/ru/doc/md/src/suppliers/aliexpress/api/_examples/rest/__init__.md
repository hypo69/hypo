# Модуль `hypotez/src/suppliers/aliexpress/api/_examples/rest/__init__.py`

## Обзор

Данный модуль предоставляет импорты для различных запросов API AliExpress, предназначенных для работы с партнёрской программой.

## Импорты

Этот модуль импортирует следующие классы, представляющие запросы к API AliExpress:

### `AliexpressAffiliateProductSmartmatchRequest`

**Описание**: Класс для запроса подбора продуктов по ключевым словам.


### `AliexpressAffiliateOrderGetRequest`

**Описание**: Класс для запроса информации о конкретном заказе.


### `AliexpressAffiliateOrderListRequest`

**Описание**: Класс для запроса списка заказов.


### `AliexpressAffiliateHotproductDownloadRequest`

**Описание**: Класс для запроса скачивания горячих продуктов.


### `AliexpressAffiliateProductdetailGetRequest`

**Описание**: Класс для запроса подробной информации о продукте.


### `AliexpressAffiliateHotproductQueryRequest`

**Описание**: Класс для запроса поиска горячих продуктов.


### `AliexpressAffiliateFeaturedpromoProductsGetRequest`

**Описание**: Класс для запроса списка продуктов с акциями.


### `AliexpressAffiliateFeaturedpromoGetRequest`

**Описание**: Класс для запроса информации об акциях.


### `AliexpressAffiliateProductQueryRequest`

**Описание**: Класс для запроса поиска продуктов.


### `AliexpressAffiliateCategoryGetRequest`

**Описание**: Класс для запроса списка категорий продуктов.


### `AliexpressAffiliateOrderListbyindexRequest`

**Описание**: Класс для запроса списка заказов по индексу.


### `AliexpressAffiliateLinkGenerateRequest`

**Описание**: Класс для генерации ссылок на продукты.
