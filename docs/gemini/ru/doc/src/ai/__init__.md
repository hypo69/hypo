# Модуль src.ai

## Обзор

Модуль `src.ai` предоставляет интерфейс для взаимодействия с моделями искусственного интеллекта, включая Google Gemini и OpenAI.

## Переменные

### `MODE`

**Описание**: Строковая переменная, определяющая режим работы модуля. В текущем примере значение `'dev'` указывает на режим разработки.

## Модули

### `gemini`

**Описание**: Модуль содержит класс `GoogleGenerativeAI`, предоставляющий доступ к моделям Google Gemini.

### `openai`

**Описание**: Модуль содержит класс `OpenAIModel`, предоставляющий доступ к моделям OpenAI.
