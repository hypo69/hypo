# Модуль hypotez/src/suppliers/aliexpress/campaign/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json` и документации из файла `README.MD`.  Модуль использует модуль `gs` для работы с путями и предоставляет переменные, содержащие информацию о проекте, такие как имя, версия, описание и прочее.


## Функции

### `set_project_root`

**Описание**: Функция находит корневую директорию проекта, начиная с текущего файла и идя вверх по директориям, пока не найдет директорию содержащую файлы `pyproject.toml`, `requirements.txt` или `.git`.

**Параметры**:

- `marker_files` (tuple): Кортеж файлов или директорий, по которым определяется корневая директория проекта.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена, иначе путь к текущей директории.

**Вызывает исключения**:

-  (Не указаны)


### Функции для загрузки настроек и документации

**Описание**: Эти функции реализуют логику загрузки данных из файлов `settings.json` и `README.MD`.

**Параметры**:

- Отсутствуют.

**Возвращает**:

- Отсутствует.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный из функции `set_project_root`. Тип `Path`.

### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженный из файла `settings.json`. Тип `dict`.

### `doc_str`

**Описание**: Строка, содержащая содержимое файла `README.MD`.  Тип `str`.

### `__project_name__`

**Описание**: Имя проекта.  Получается из `settings.json` или имеет значение по умолчанию `hypotez`. Тип `str`.

### `__version__`

**Описание**: Версия проекта. Получается из `settings.json` или имеет значение по умолчанию `''`. Тип `str`.

### `__doc__`

**Описание**: Описание проекта. Получается из `README.MD` или имеет значение по умолчанию `''`. Тип `str`.

### `__details__`

**Описание**: Дополнительные детали проекта.  Значение по умолчанию `''`. Тип `str`.

### `__author__`

**Описание**: Автор проекта. Получается из `settings.json` или имеет значение по умолчанию `''`. Тип `str`.

### `__copyright__`

**Описание**: Авторские права. Получается из `settings.json` или имеет значение по умолчанию `''`. Тип `str`.

### `__cofee__`

**Описание**: Ссылка на поддержку разработчика. Получается из `settings.json` или имеет значение по умолчанию.  Тип `str`.

**Примечание**: Переменные, имеющие подчеркивание в начале и конце имени, обычно являются константами или внутренними переменными, используемыми в модуле.
