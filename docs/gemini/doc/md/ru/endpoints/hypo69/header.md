# Модуль hypotez/src/endpoints/hypo69/header.py

## Обзор

Данный модуль содержит вспомогательные функции для работы с проектом, такие как определение корневой директории проекта, загрузка настроек из файла `settings.json` и чтение файла `README.MD`.  Он также определяет переменные, содержащие информацию о проекте (название, версия, описание, автор и т.д.).


## Функции

### `get_project_root`

**Описание**: Функция находит корневую директорию проекта, начиная от текущей директории и двигаясь вверх по дереву директорий. Она останавливается на первой директории, содержащей один из указанных файлов маркеров (например, `pyproject.toml`, `requirements.txt`, `.git`). Если корневая директория не найдена, возвращается директория, в которой находится текущий файл.  Добавляет корневую директорию в `sys.path` для импорта модулей.

**Параметры**:
- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, по которым определяется корневая директория проекта.

**Возвращает**:
- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращается путь к директории, в которой расположен текущий файл.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта. Инициализируется функцией `get_project_root()`.


### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженные из файла `settings.json`. Значение по умолчанию `None`.

**Обработка ошибок**:
- `FileNotFoundError`: Исключение возникает, если файл `settings.json` не найден.
- `json.JSONDecodeError`: Исключение возникает, если файл `settings.json` содержит некорректный JSON.


### `doc_str`

**Описание**: Строка, содержащая содержимое файла `README.MD`. Значение по умолчанию `None`.

**Обработка ошибок**:
- `FileNotFoundError`: Исключение возникает, если файл `README.MD` не найден.
- `json.JSONDecodeError`: Исключение возникает, если файл `README.MD` содержит некорректный JSON.


### `__project_name__`

**Описание**: Имя проекта. Значение берется из настроек или задаётся по умолчанию как 'hypotez'.


### `__version__`

**Описание**: Версия проекта. Значение берется из настроек или задаётся по умолчанию как пустая строка.


### `__doc__`

**Описание**: Документация к проекту. Значение берется из файла `README.MD` или задаётся по умолчанию как пустая строка.


### `__details__`

**Описание**: Дополнительная информация о проекте. Значение по умолчанию пустая строка.


### `__author__`

**Описание**: Автор проекта. Значение берется из настроек или задаётся по умолчанию как пустая строка.


### `__copyright__`

**Описание**: Авторские права на проект. Значение берется из настроек или задаётся по умолчанию как пустая строка.


### `__cofee__`

**Описание**: Ссылка для поддержки разработчика. Значение берется из настроек или задаётся по умолчанию.
