# Модуль `hypotez/src/suppliers/amazon/scenario.py`

## Обзор

Этот модуль содержит сценарий для сбора данных о товарах с сайта Amazon. Он реализует функции для получения списка категорий, списка товаров в категории и извлечения данных о конкретных товарах.  Модуль ориентирован на работу с веб-драйвером для взаимодействия с сайтом.

## Константы

### `MODE`

**Описание**: Переменная, хранящая режим работы модуля (например, 'dev', 'prod').

## Функции

### `get_list_products_in_category(s)`

**Описание**: Функция возвращает список URL-адресов товаров, находящихся на странице категории.

**Параметры**:

- `s` (объект): Объект поставщика (Supplier), содержащий информацию о поставщике, веб-драйвере и локаторах.

**Возвращает**:

- `list[str, str, None]`: Список URL-адресов товаров или `None` при ошибке.

**Вызывает исключения**:

- `None`: Возможны ошибки, не обработанные в функции, которые могут приводить к возврату `None`.

**Подробное описание**:

Функция `get_list_products_in_category` извлекает ссылки на страницы товаров, представленные на странице категории.  Она использует веб-драйвер для работы со страницей и извлекает ссылки на товары, с использованием локаторов, хранящихся в `s.locators['category']`. Обработка ошибок и проверка корректности локаторов, а также  обработка случая, когда список товаров пуст, реализованы внутри функции.

**Примечания**:

- Функция пока не реализует обработку случаев пролистывания страниц с товарами.
- Функция не использует проверки наличия товара в базе данных (`PrestaShopProduct.check`). Эта часть кода закомментирована.
