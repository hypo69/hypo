# Модуль hypotez/src/webdriver/edge/edge.py

## Обзор

Модуль содержит класс `Edge`, который расширяет класс `selenium.webdriver.Edge` для работы с веб-драйвером Edge. Он упрощает конфигурацию драйвера, используя `fake_useragent` для генерации случайных user-agent'ов.  Класс предоставляет дополнительные методы для работы с JavaScript и локаторами элементов.

## Классы

### `Edge`

**Описание**:  Класс `Edge` расширяет базовый класс `selenium.webdriver.Edge`, добавляя функциональность для работы с JavaScript и локаторами элементов.

**Атрибуты**:

- `driver_name` (str): Имя веб-драйвера (по умолчанию `'edge'`).

**Методы**:

#### `__init__`

**Описание**: Инициализирует экземпляр класса `Edge`.

**Параметры**:

- `user_agent` (Optional[dict], optional): Словарь для указания user-agent. Если `None`, генерируется случайный user-agent.

**Возвращает**:
- `None`

**Вызывает исключения**:
- `WebDriverException`: Возникает, если не удалось запустить веб-драйвер Edge.
- `Exception`: Возникает при возникновении других ошибок при запуске или работе веб-драйвера.


#### `_payload`

**Описание**: Загружает исполнители для локаторов и JavaScript-сценариев.

**Параметры**:

- Нет

**Возвращает**:
- `None`

#### `set_options`

**Описание**: Создает и настраивает опции запуска для веб-драйвера Edge.

**Параметры**:

- `opts` (Optional[List[str]], optional): Список опций для добавления к веб-драйверу Edge. По умолчанию `None`.

**Возвращает**:
- `EdgeOptions`: Настроенный объект `EdgeOptions`.


## Модульные переменные

### `MODE`

**Описание**:  Переменная, хранящая режим работы, вероятно, для разработки или производства.
