# Tiny Utils

## Обзор

**Tiny Utils** — это библиотека утилит, предоставляющая набор легких вспомогательных функций для различных общих задач. Библиотека включает утилиты для преобразования форматов данных, работы с текстом и файлами, строковыми операциями, форматированием даты и времени, обработкой изображений и др. Она организована в несколько модулей для удобного доступа к конкретным функциям.

## Установка

Для использования **Tiny Utils** клонируйте репозиторий и установите необходимые зависимости, как указано в файле `requirements.txt`.

```bash
git clone https://github.com/hypo69/tiny-utils.git
cd tiny_utils
pip install -r requirements.txt
```

## Обзор модулей

Библиотека содержит несколько подмодулей, каждый из которых отвечает за определённую задачу:

- **Преобразователи**: Модули для преобразования форматов данных, таких как текст-изображение, webp-png, JSON, XML, кодирование Base64 и др.
- **Утилиты для строк**: Инструменты для продвинутой работы со строками.
- **Утилиты для работы с файлами**: Функции для работы с файлами и их манипулирования.
- **Утилиты для работы с датой и временем**: Инструменты для форматирования даты и времени.
- **Утилиты для работы с FTP**: Функции для работы с FTP-серверами.
- **Утилиты для работы с изображениями**: Базовые функции обработки изображений.
- **Утилиты для работы с PDF**: Утилиты для работы с PDF-файлами, включая преобразование, слияние, разделение и извлечение текста.
- **Утилиты для работы с принтером**: Функции для отправки данных на принтер.


## Описание модулей

### Преобразователи

Модуль `convertors` содержит утилиты для преобразования данных между форматами. Эти модули могут обрабатывать различные типы данных, от CSV до JSON и текста до изображений.


#### Файлы:

- **text2png.py**: Преобразует текстовые данные в изображение PNG.
- **tts.py**: Преобразует текст в речь и сохраняет аудиофайл.
- **webp2png.py**: Преобразует изображения из формата WebP в PNG.
- **xls.py**: Обрабатывает преобразования и манипуляции с XLS-файлами.
- **xml2dict.py**: Преобразует XML-данные в словарь Python.
- **base64.py**: Кодирует или декодирует данные с помощью кодирования Base64.
- **csv.py**: Предоставляет инструменты для парсинга и манипулирования CSV.
- **dict.py**: Утилиты для работы со словарями Python.
- **html.py**: Преобразует HTML-контент в различные форматы.
- **json.py**: Утилиты для парсинга и манипулирования JSON.
- **md2dict.py**: Преобразует Markdown-контент в словарь.
- **ns.py**: Специализированные утилиты для преобразования пространств имён.
