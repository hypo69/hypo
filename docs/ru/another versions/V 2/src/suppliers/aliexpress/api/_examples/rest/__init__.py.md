# Модуль `src.suppliers.aliexpress.api._examples.rest`

## Обзор

Данный модуль содержит примеры запросов к API AliExpress, включая запросы для получения продуктов, заказов, категорий, горячих товаров и прочих данных.

## Содержание

- [Импортированные классы](#импортированные-классы)

## Импортированные классы

### `AliexpressAffiliateProductSmartmatchRequest`
**Описание:** Класс для запроса смарт-подбора товаров.

### `AliexpressAffiliateOrderGetRequest`
**Описание:** Класс для запроса информации о заказе.

### `AliexpressAffiliateOrderListRequest`
**Описание:** Класс для запроса списка заказов.

### `AliexpressAffiliateHotproductDownloadRequest`
**Описание:** Класс для запроса загрузки горячих товаров.

### `AliexpressAffiliateProductdetailGetRequest`
**Описание:** Класс для запроса детальной информации о товаре.

### `AliexpressAffiliateHotproductQueryRequest`
**Описание:** Класс для запроса горячих товаров.

### `AliexpressAffiliateFeaturedpromoProductsGetRequest`
**Описание:** Класс для запроса акционных товаров.

### `AliexpressAffiliateFeaturedpromoGetRequest`
**Описание:** Класс для запроса информации об акциях.

### `AliexpressAffiliateProductQueryRequest`
**Описание:** Класс для запроса товаров.

### `AliexpressAffiliateCategoryGetRequest`
**Описание:** Класс для запроса категорий товаров.

### `AliexpressAffiliateOrderListbyindexRequest`
**Описание:** Класс для запроса списка заказов по индексу.

### `AliexpressAffiliateLinkGenerateRequest`
**Описание:** Класс для генерации партнерских ссылок.
