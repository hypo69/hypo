# Маршруты Fast API

## Обзор

Модуль `routes.py` предназначен для манипулирования маршрутами в Fast API сервере. Он определяет обработчики для входящих запросов, в частности, для обработки сообщений от Telegram бота.

## Подробнее

Этот модуль настраивает маршруты, связывая URL-адреса с соответствующими функциями обработки. В текущей версии модуля реализован обработчик для сообщений Telegram, который делегирует обработку классу `BotHandler`. Это позволяет организовать логику обработки сообщений Telegram в отдельном компоненте и обеспечивает гибкость в добавлении новых обработчиков или изменении существующих.

## Классы

### `Routes`

**Описание**: Класс `Routes` предназначен для определения и настройки маршрутов в Fast API приложении. Он содержит методы для регистрации обработчиков различных типов запросов.

**Методы**:
- `tegram_message_handler`: Регистрирует обработчик для входящих сообщений Telegram.
