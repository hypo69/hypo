# Модуль hypotez/src/scenario/header.py

## Обзор

Данный модуль отвечает за инициализацию проекта, нахождение корневой директории, загрузку настроек и документации из файлов `settings.json` и `README.MD` соответственно. Он также определяет переменные, описывающие проект, такие как имя, версия, описание и автор.

- [Функции](#функции)


## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с текущей директории и двигаясь вверх по дереву директорий, пока не найдет директорию, содержащую один из указанных маркеров.

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, которые служат маркерами корневой директории проекта. По умолчанию это кортеж (`'pyproject.toml'`, `'requirements.txt'`, `'.git'`).

**Возвращает**:

- `Path`: Объект `Path` представляющий путь к корневой директории проекта. Если корневая директория не найдена, возвращает путь к директории, где находится текущий файл.

**Вызывает исключения**:

-  Не вызывает исключения.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта. Инициализируется вызовом функции `set_project_root()`.


### `settings`

**Описание**: Словарь настроек проекта, загруженный из файла `settings.json`. Инициализируется при помощи `json.load()` из файла `settings.json`, если файл найден и корректный. В противном случае, переменная `settings` остается `None`.


### `doc_str`

**Описание**: Строковое представление документации из файла `README.MD`. Инициализируется при помощи `settings_file.read()`, если файл найден и корректный. В противном случае, переменная `doc_str` остается `None`.


### `__project_name__`

**Описание**: Имя проекта. Получается из словаря настроек `settings`, если он не пустой. Иначе, используется значение по умолчанию — `'hypotez'`.


### `__version__`

**Описание**: Версия проекта. Получается из словаря настроек `settings`, если он не пустой. Иначе, используется значение по умолчанию — пустая строка.


### `__doc__`

**Описание**: Документация проекта. Получается из переменной `doc_str`, если она не пустая. Иначе, используется значение по умолчанию — пустая строка.


### `__details__`

**Описание**: Подробное описание проекта. Инициализируется пустой строкой.


### `__author__`

**Описание**: Автор проекта. Получается из словаря настроек `settings`, если он не пустой. Иначе, используется значение по умолчанию — пустая строка.


### `__copyright__`

**Описание**: Авторские права. Получается из словаря настроек `settings`, если он не пустой. Иначе, используется значение по умолчанию — пустая строка.


### `__cofee__`

**Описание**: Ссылка на страницу для поддержки разработчика (кофе). Получается из словаря настроек `settings`, если он не пустой. Иначе, используется значение по умолчанию.
