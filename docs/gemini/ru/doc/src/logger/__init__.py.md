# Модуль `src.logger`

## Обзор

Модуль `src.logger` предназначен для настройки и использования системы логирования в приложении. Он включает в себя функциональность для создания и управления логгерами, обработки исключений, связанных с логированием, а также импортирует необходимые модули.

## Импортированные модули

### `logger`

Модуль `logger` импортируется из `src.logger.logger` и предназначен для управления процессом логирования.

### Исключения

Модуль импортирует следующие исключения из `src.logger.exceptions`:

- `ExecuteLocatorException`: Исключение, возникающее при проблемах с локатором.
- `DefaultSettingsException`: Исключение, возникающее при проблемах с настройками по умолчанию.
- `CredentialsError`: Исключение, возникающее при ошибках аутентификации.
- `PrestaShopException`: Исключение, возникающее при ошибках, связанных с PrestaShop.
- `PayloadChecksumError`: Исключение, возникающее при ошибках контрольной суммы данных.
