# Модуль `hypo69`

## Обзор

Модуль `hypo69` содержит эндпоинты для разработчика, предоставляя доступ к различным функциональным блокам, включая бота с чатом модели ИИ, модуль обучения модели коду проекта и раннюю разработку модуля парсинга диалогов.

## Оглавление

* [**small_talk_bot**](#small-talk-bot)
* [**code_assistant**](#code-assistant)
* [**psychologist_bot**](#psychologist-bot)


## `small_talk_bot`

### Описание

Бот для чата с моделью ИИ. Предоставляет функционал для взаимодействия с моделью и получения ответов.


## `code_assistant`

### Описание

Модуль обучения модели коду проекта. Предоставляет инструменты для обучения модели на данных проекта.


## `psychologist_bot`

### Описание

Ранняя разработка модуля парсинга диалогов. Предназначен для анализа и обработки диалогов.
