# Модуль `hypotez/src/suppliers/aliexpress/utils/locales.py`

## Обзор

Модуль `locales.py` предназначен для загрузки данных локализации из файла JSON.  Он предоставляет функцию для загрузки данных и построения словарей языков и валют.

## Функции

### `get_locales`

**Описание**: Загружает данные локализации из файла JSON.

**Параметры**:
- `locales_path` (Path | str): Путь к файлу JSON, содержащему данные локализации.

**Возвращает**:
- `list[dict[str, str]] | None`: Список словарей, где каждый словарь содержит пару "язык" (ключ) и "валюта" (значение). Возвращает `None`, если файл не найден или некорректно сформирован.

**Примеры**:
```python
from src.suppliers.aliexpress.utils.locales import load_locales_data
locales = load_locales_data(Path('/path/to/locales.json'))
print(locales)
```

**Примечания**:
Функция использует `j_loads_ns` из `src.utils.jjson` для загрузки данных.  Она ожидает, что файл JSON будет содержать ключ `locales` со списком словарей, содержащих пары "язык-валюта".

## Постоянные переменные

### `MODE`

**Описание**: Переменная, хранящая режим работы (в данном случае 'dev').


## Импорты

- `pathlib`: Модуль для работы с путями.
- `gs`: Модуль (скорее всего, из другого модуля), вероятно, содержит конфигурационные данные.
- `j_loads`, `j_loads_ns`: Функции для загрузки данных из JSON (из модуля `src.utils.jjson`).


## Примечания

- Файл `locales.json` должен быть расположен в указанном пути, чтобы функция `get_locales` могла корректно загрузить данные.
- Модуль использует библиотеку `jjson`, чтобы загрузить данные JSON. Ожидается, что она будет установлена в системе.
- В коде используется импорт из несуществующих модулей, `src.utils.jjson` и `gs`. Для корректной работы нужно определить, где эти модули расположены и импортировать их.
