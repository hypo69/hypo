# Модуль `hypotez/src/suppliers/aliexpress/api/helpers/products.py`

## Обзор

Этот модуль содержит функции для обработки данных о продуктах, полученных с API AliExpress.  Функции предназначены для извлечения и подготовки данных о продуктах для дальнейшего использования.

## Оглавление

- [Функции](#функции)
    - [`parse_product`](#parse_product)
    - [`parse_products`](#parse_products)


## Функции

### `parse_product`

**Описание**: Эта функция преобразует данные о продукте, изменяя поле `product_small_image_urls`.

**Параметры**:

- `product` (объект): Объект, содержащий данные о продукте.  Ожидается, что у объекта есть атрибут `product_small_image_urls`, который является `HTML-элементом <string>` (предположительно, содержащим ссылки на изображения).

**Возвращает**:

- `product` (объект): Обновленный объект `product` с изменённым атрибутом `product_small_image_urls`. Значение `product_small_image_urls` теперь является строкой, извлечённой из исходного HTML-элемента.


### `parse_products`

**Описание**:  Функция обрабатывает список продуктов, применяя к каждому продукту функцию `parse_product`.

**Параметры**:

- `products` (список): Список объектов, каждый из которых представляет данные о продукте (аналогично параметру `product` в функции `parse_product`).

**Возвращает**:

- `new_products` (список): Новый список объектов продуктов,  каждый из которых имеет изменённое поле `product_small_image_urls`. Если входной список `products` пуст, функция возвращает пустой список.
