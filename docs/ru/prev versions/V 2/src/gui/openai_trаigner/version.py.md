# src.gui.openai_trainer.version.py

## Обзор

Данный модуль `version.py` содержит информацию о версии, авторе, лицензии и другую метаинформацию для проекта `openai_trainer`.

## Содержание

- [Обзор](#обзор)
- [Переменные](#переменные)

## Переменные

### `__version__`

**Описание**: Строка, представляющая текущую версию проекта.

**Тип**: `str`

**Значение**: `'v1.1'`

### `__doc__`

**Описание**: Строка документации модуля. На данный момент пустая строка.

**Тип**: `str`

**Значение**: `''`

### `__details__`

**Описание**: Строка для дополнительных деталей модуля. На данный момент пустая строка.

**Тип**: `str`

**Значение**: `''`

### `__author__`

**Описание**: Строка, представляющая имя автора проекта.

**Тип**: `str`

**Значение**: `'hypo69'`

### `__copyright__`

**Описание**: Строка, содержащая информацию о лицензии и авторских правах.

**Тип**: `str`

**Значение**:

```
## License

Copyright (c) 2024 hypo69

This project is licensed under the MIT License. See the [MIT License](https://opensource.org/licenses/MIT) for details.

Commercial use of the code is prohibited without prior permission from the authors.
```

### `__cofee__`

**Описание**: Строка, содержащая ссылку на страницу поддержки автора.

**Тип**: `str`

**Значение**: `"Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69"`
