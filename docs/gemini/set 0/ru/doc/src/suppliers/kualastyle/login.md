# Модуль hypotez/src/suppliers/kualastyle/login.py

## Обзор

Этот модуль содержит функции для авторизации поставщика Kualastyle.  Он включает в себя функции `login` и `close_pop_up` для взаимодействия с веб-драйвером и закрытия всплывающих окон.

## Функции

### `login`

**Описание**: Функция для входа в систему поставщика.

**Параметры**:

- `s` (Supplier): Объект, представляющий поставщика, содержащий необходимую информацию и драйвер.


**Возвращает**:

- `bool`: `True`, если вход успешен, `False` иначе.


### `close_pop_up`

**Описание**: Функция для закрытия всплывающих окон (pop-up) на сайте Kualastyle.

**Параметры**:

- `s` (Supplier): Объект, представляющий поставщика, содержащий необходимую информацию и драйвер.


**Возвращает**:

- `bool`: `True`, если окно закрыто, `False` иначе.


**Обработка исключений**:

- `Exception`: Если возникает ошибка при выполнении действий, связанных с закрытием pop-up, логируется предупреждение с описанием ошибки. Сообщение лога записывается в `logger` и имеет формат: `"Не закрыл попап"`.


**Детали**:

- Функция получает ссылку `https://www.kualastyle.com` с помощью `s.driver.get_url`.
- Функция использует `s.locators['close_pop_up_locator']` для получения локейторов для закрытия pop-up.
- Использование метода `s.driver.execute_locator` для выполнения действий с локейтором.
- `s.driver.wait(5)` - ожидание в течение 5 секунд (для загрузки страницы).
- `s.driver.window_focus(s.driver)` - фокус на текущем окне.
- `...` -  указывает на то, что код для обработки других сценариев (например, при ошибках или некорректных данных) не представлен в данном фрагменте.
