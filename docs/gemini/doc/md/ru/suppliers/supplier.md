# Модуль `hypotez/src/suppliers/supplier.py`

## Обзор

Модуль `supplier.py` содержит класс `Supplier`, предназначенный для управления и выполнения сценариев для различных поставщиков.  Класс предоставляет методы для загрузки настроек, входа на сайт поставщика и запуска сценариев.

## Классы

### `Supplier`

**Описание**: Класс `Supplier` представляет собой поставщика, который отвечает за выполнение сценариев. Он хранит информацию о поставщике, включая его идентификатор, префикс, локаль, правила расчета цен, связанные модули, сценарии, локаторы, веб-драйвер и текущий сценарий.

**Методы**:

- `__init__(self, **data)`: Инициализирует поставщика, загружает конфигурацию.
- `_payload(self)`: Загружает параметры поставщика из файла настроек.
- `login(self)`: Выполняет вход на сайт поставщика.
- `run_scenario_files(self, scenario_files: Optional[str | List[str]] = None)`: Выполняет один или несколько файлов сценариев.
- `run_scenarios(self, scenarios: dict | List[dict])`: Выполняет список или один сценарий.
- `check_supplier_prefix(cls, value: str)`: Валидатор префикса поставщика.


## Функции


### `check_supplier_prefix`

**Описание**: Функция-валидатор, проверяет, что префикс поставщика не пустой.

**Параметры**:

- `value` (str): Префикс поставщика.

**Возвращает**:

- `str`: Префикс поставщика.

**Вызывает исключения**:

- `ValueError`: Если префикс поставщика пустой.


## Атрибуты

### `MODE`

**Описание**: Переменная, содержащая режим работы приложения ('development' по умолчанию).


## Обработка исключений

В модуле используется обработка исключений.

### `DefaultSettingsException`

**Описание**: Исключение, которое возникает при ошибке загрузки настроек поставщика.

### `ModuleNotFoundError`

**Описание**: Исключение, которое возникает при попытке импортировать несуществующий модуль.


## Дополнительная информация

- Класс использует `pydantic` для валидации данных.
- Использование `j_loads_ns` для парсинга JSON данных.
-  Используется `logger` для регистрации событий.
-  Используется `gs.path.src` для построения путей.
