# Модуль create_report

## Обзор

Модуль предназначен для генерации отчетов на основе данных, полученных от различных поставщиков. Он включает в себя функциональность для извлечения, разбора и обработки данных о продуктах, подготовки данных, обработки с использованием ИИ и интеграции с Facebook для размещения продуктов. Модуль использует класс `ReportGenerator` для создания отчетов в форматах HTML и PDF.

## Подробней

Модуль `create_report.py` является частью подсистемы формирования отчетов, которая используется для автоматизации процесса создания отчетов о продуктах. Он принимает данные о продуктах в различных форматах, обрабатывает их и генерирует отчеты, готовые для публикации. Основная цель модуля - упростить и ускорить процесс создания отчетов, а также обеспечить их соответствие требованиям.

## Переменные

### `report_generator`

**Описание**: Экземпляр класса `ReportGenerator`, используемый для создания отчетов.

### `html_file_he`

**Описание**: Объект `Path`, представляющий путь к HTML-файлу для отчета на иврите.

### `pdf_file_he`

**Описание**: Объект `Path`, представляющий путь к PDF-файлу для отчета на иврите.

### `html_file_ru`

**Описание**: Объект `Path`, представляющий путь к HTML-файлу для отчета на русском языке.

### `pdf_file_ru`

**Описание**: Объект `Path`, представляющий путь к PDF-файлу для отчета на русском языке.

## Функции

### `create_report`

```python
def create_report(response_dict: dict, lang: str, html_file: Path, pdf_file: Path):
    """
    Создает отчет на основе предоставленных данных и сохраняет его в формате HTML и PDF.

    Args:
        response_dict (dict): Словарь с данными для отчета.
        lang (str): Язык отчета ('he' для иврита, 'ru' для русского).
        html_file (Path): Путь для сохранения HTML-файла.
        pdf_file (Path): Путь для сохранения PDF-файла.

    Returns:
        None

    Raises:
        Exception: Если возникает ошибка при создании отчета.

    Example:
        >>> report_generator = ReportGenerator()
        >>> data = {'ключ': 'значение'}
        >>> html_file = Path('report.html')
        >>> pdf_file = Path('report.pdf')
        >>> report_generator.create_report(data, 'ru', html_file, pdf_file)
    """
```

**Описание**: Функция создает отчет на основе предоставленных данных и сохраняет его в форматах HTML и PDF.

**Параметры**:

- `response_dict` (dict): Словарь с данными для отчета.
- `lang` (str): Язык отчета ('he' для иврита, 'ru' для русского).
- `html_file` (Path): Путь для сохранения HTML-файла.
- `pdf_file` (Path): Путь для сохранения PDF-файла.

**Возвращает**:

- `None`

**Вызывает исключения**:

- `Exception`: Если возникает ошибка при создании отчета.

**Примеры**:

```python
report_generator = ReportGenerator()
data = {'ключ': 'значение'}
html_file = Path('report.html')
pdf_file = Path('report.pdf')
report_generator.create_report(data, 'ru', html_file, pdf_file)
