# Модуль `src.suppliers.bangood.graber`

## Обзор

Модуль `src.suppliers.bangood.graber` представляет собой класс, который предназначен для сбора данных о товарах с сайта `bangood.com`. Класс наследует функциональность от родительского класса `Graber` и переопределяет методы для специфической обработки полей, если это требуется. Модуль также включает механизм для предварительных действий перед запросом к веб-драйверу через декораторы.

## Функции

### `__init__`

**Описание**: Инициализирует класс `Graber` для сбора полей товара. Устанавливает префикс поставщика и вызывает конструктор родительского класса. Также устанавливает настройки для декоратора через `Context`.

**Параметры**:
- `driver` (`Driver`): Экземпляр веб-драйвера для взаимодействия со страницей.

**Возвращает**:
- `None`: Метод ничего не возвращает.
