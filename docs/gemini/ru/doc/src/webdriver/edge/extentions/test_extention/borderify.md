# Модуль borderify.js

## Обзор

Этот скрипт устанавливает красную границу толщиной 5 пикселей вокруг всего документа.  Также отображает диалоговое окно alert().

## Функции

### `document.body.style.border = "5px solid red";`

**Описание**: Устанавливает свойство `border` элемента `body` на красную границу толщиной 5 пикселей.

**Подробности**: Эта строка кода напрямую изменяет стиль элемента `body` в документе.


### `alert();`

**Описание**: Выводит диалоговое окно с сообщением.

**Подробности**: Эта функция отображает модальное диалоговое окно "alert" пользователю.  Пользователь может продолжить работу лишь после закрытия диалогового окна.
