# Модуль `hypotez/src/suppliers/gearbest/header.py`

## Обзор

Этот модуль содержит функции для определения корневого каталога проекта, загрузки настроек из файла `settings.json` и получения информации о проекте, такой как название, версия, описание и автор.

## Функции

### `set_project_root`

**Описание**:  Находит корневой каталог проекта, начиная с текущего каталога, ищет вверх по иерархии каталогов, пока не найдет каталог, содержащий один из файлов или папок, указанных в списке `marker_files`.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или каталогов, используемых для определения корневого каталога проекта. По умолчанию: `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневому каталогу проекта, если он найден. В противном случае возвращает каталог, в котором находится скрипт.

**Вызывает исключения**:

- Нет указанных исключений.


## Переменные

### `__root__`

**Описание**:  Путь к корневому каталогу проекта, полученный вызовом функции `set_project_root()`.


### `settings`

**Описание**:  Словарь с настройками проекта, загруженный из файла `settings.json`. `None`, если файл не найден или не удалось загрузить данные.


### `doc_str`

**Описание**:  Строка с описанием проекта, загруженная из файла `README.MD`. `None`, если файл не найден или не удалось загрузить данные.


### `__project_name__`

**Описание**: Имя проекта, полученное из настроек. По умолчанию: `'hypotez'`.


### `__version__`

**Описание**: Версия проекта, полученная из настроек. По умолчанию: пустая строка.


### `__doc__`

**Описание**: Документация проекта, полученная из `doc_str`. По умолчанию: пустая строка.


### `__details__`

**Описание**: Дополнительные детали о проекте. По умолчанию: пустая строка.


### `__author__`

**Описание**: Автор проекта, полученный из настроек. По умолчанию: пустая строка.


### `__copyright__`

**Описание**: Авторские права проекта, полученные из настроек. По умолчанию: пустая строка.


### `__cofee__`

**Описание**: Ссылка на площадку для поддержки разработчика. По умолчанию: ссылка на Boosty.
