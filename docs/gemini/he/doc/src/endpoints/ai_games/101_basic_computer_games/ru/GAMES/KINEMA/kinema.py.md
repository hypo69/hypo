# kinema.py

## סקירה כללית

קובץ זה מיישם משחק טקסט פשוט בשם "קינמה", בו השחקן מזיז נקודה במישור באמצעות פקודות. המשחק מתחיל בנקודה (0, 0) ומאפשר לשחקן להזיז את הנקודה שמאלה, ימינה, למעלה או למטה עד שהוא בוחר לצאת.
