# Модуль `hypotez/src/endpoints/hypo69/psychologist_bot/header.py`

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек проекта. Он также определяет переменные, содержащие информацию о проекте (название, версия, описание, автор и др.).

## Функции

### `set_project_root`

**Описание**: Определяет корневую директорию проекта, начиная с директории текущего файла и идя вверх по дереву директорий.

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или директорий, которые указывают на корневую директорию проекта. По умолчанию содержит ('pyproject.toml', 'requirements.txt', '.git').


**Возвращает**:
- `Path`: Путь к корневой директории проекта, если найдена, иначе путь к директории текущего файла.


**Вызывает исключения**:
- Нет

### Загрузить настройки

Функция загружает настройки из файла `settings.json` в корневой директории проекта.


**Вызывает исключения**:
- `FileNotFoundError`: Если файл `settings.json` не найден.
- `json.JSONDecodeError`: Если файл `settings.json` содержит некорректный JSON.

## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта.

**Тип**: `Path`


### `settings`

**Описание**: Словарь настроек проекта, загруженный из файла `settings.json`.

**Тип**: `dict` или `None`, если файл не найден или некорректен.


### `doc_str`

**Описание**: Содержимое файла `README.MD` в корневой директории проекта.

**Тип**: `str` или `None`, если файл не найден или некорректен.

### `__project_name__`

**Описание**: Название проекта.

**Тип**: `str`

### `__version__`

**Описание**: Версия проекта.

**Тип**: `str`


### `__doc__`

**Описание**: Описание проекта (из файла `README.MD`).

**Тип**: `str`


### `__details__`

**Описание**: Дополнительная информация о проекте.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка на возможность поддержать разработчика.

**Тип**: `str`
