# Модуль _experiments

## Обзор

Модуль `_experiments` содержит экспериментальный код, связанный с поставщиком KSP (Kramer Sales Promotion). Включает в себя различные импорты и определения для работы с данными, веб-драйверами, продуктами, категориями и другими компонентами системы.

## Подробней

Этот модуль, расположенный в `src/suppliers/ksp/_experiments/JUPYTER_header.py`, предназначен для экспериментов и прототипирования. Он импортирует множество модулей и классов, используемых в проекте `hypotez`. Модуль, вероятно, используется для тестирования новых идей и подходов, связанных с поставщиком KSP, прежде чем они будут интегрированы в основную кодовую базу. В частности, он включает в себя функциональность для запуска поставщика с заданными параметрами.

## Переменные

### `dir_root`

```python
dir_root : Path = Path (os.getcwd()[:os.getcwd().rfind('hypotez')+7])
```

**Описание**: Определяет корневой каталог проекта `hypotez`, обрезая текущий рабочий каталог до нужной глубины.

### `dir_src`

```python
dir_src = Path (dir_root, 'src')
```

**Описание**: Определяет путь к каталогу `src` внутри корневого каталога проекта.

## Функции

### `start_supplier`

```python
def start_supplier(supplier_prefix: str = 'aliexpress', locale: str = 'en' ) -> Supplier:
    """ Старт поставщика """
    params: dict = \
    {
        'supplier_prefix': supplier_prefix,
        'locale': locale
    }
    
    return Supplier(**params)
```

**Описание**: Функция `start_supplier` предназначена для инициализации и запуска поставщика (Supplier) с заданными параметрами.

**Как работает функция**:
1. Функция принимает два параметра: `supplier_prefix` (префикс поставщика, по умолчанию `'aliexpress'`) и `locale` (локаль, по умолчанию `'en'`).
2. Создается словарь `params`, содержащий переданные параметры `supplier_prefix` и `locale`.
3. Создается и возвращается экземпляр класса `Supplier` с использованием параметров из словаря `params`.

**Параметры**:
- `supplier_prefix` (str, optional): Префикс поставщика. По умолчанию `'aliexpress'`.
- `locale` (str, optional): Локаль поставщика. По умолчанию `'en'`.

**Возвращает**:
- `Supplier`: Экземпляр класса `Supplier`, инициализированный с заданными параметрами.

**Примеры**:

```python
from src.suppliers import Supplier

# Запуск поставщика с параметрами по умолчанию
supplier = start_supplier()

# Запуск поставщика с указанием префикса и локали
supplier = start_supplier(supplier_prefix='ksp', locale='ru')
```
