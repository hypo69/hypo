**Описание:**

Этот файл определяет пользовательские исключения для модуля `aliexpress.api.errors`. Он предоставляет набор исключений, специфичных для работы с API AliExpress, позволяя более четко обрабатывать различные ошибки, возникающие при взаимодействии с ним.

**Классы исключений:**

* **`AliexpressException`:**  Базовый класс для всех исключений, связанных с API AliExpress. Принимает `reason` (строка) для описания причины ошибки.


* **`InvalidArgumentException`:** Исключение, генерируемое, когда входные аргументы не соответствуют ожидаемому формату или типу.


* **`ProductIdNotFoundException`:** Исключение, генерируемое, если не удалось найти продукт с указанным ID.


* **`ApiRequestException`:** Исключение, генерируемое, если запрос к API AliExpress завершился неудачей.


* **`ApiRequestResponseException`:** Исключение, генерируемое, если ответ от API AliExpress не соответствует ожидаемому формату или структуре.


* **`ProductsNotFoudException`:** Исключение, генерируемое, если не удалось найти ни одного продукта по заданным критериям.


* **`CategoriesNotFoudException`:** Исключение, генерируемое, если не удалось найти ни одной категории по заданным критериям.


* **`InvalidTrackingIdException`:** Исключение, генерируемое, если предоставленный трекинг-ID некорректен или отсутствует.
