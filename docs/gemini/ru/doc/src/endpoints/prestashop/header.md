# Модуль hypotez/src/logger/header.py

## Обзор

Модуль `hypotez/src/logger/header.py` определяет корневой путь к проекту. Все импорты строятся относительно этого пути.  В дальнейшем этот функционал планируется перенести в системные переменные.

## Функции

### `set_project_root`

**Описание**: Определяет корневой каталог проекта, начиная с текущей директории и двигаясь вверх по дереву каталогов.  Останавливается на первой директории, содержащей файлы маркеров.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или каталогов, используемых для определения корневого каталога проекта. По умолчанию - (`'pyproject.toml'`, `'requirements.txt'`, `.git`).

**Возвращает**:

- `Path`: Путь к корневому каталогу проекта, если найден, иначе - путь к директории, где расположен данный скрипт.

**Пример использования**:

```python
root_path = set_project_root()
print(root_path)
```

**Обрабатываемые исключения**:

- Нет обработки исключений.

## Глобальные переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта, полученный с помощью функции `set_project_root()`.

**Тип**: `Path`


### `settings`

**Описание**: Словарь настроек, загруженный из файла `settings.json` в корневой директории проекта.

**Тип**: `dict` или `None`

**Обрабатываемые исключения**:

- `FileNotFoundError`: Если файл `settings.json` не найден.
- `json.JSONDecodeError`: Если содержимое файла `settings.json` не является валидным JSON.


### `doc_str`

**Описание**: Строка документации, загруженная из файла `README.MD` в корневой директории проекта.

**Тип**: `str` или `None`

**Обрабатываемые исключения**:

- `FileNotFoundError`: Если файл `README.MD` не найден.
- `json.JSONDecodeError`: (неприменимо в данном случае)


### `__project_name__`

**Описание**: Название проекта. Получается из настроек `settings`, по умолчанию - 'hypotez'.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта. Получается из настроек `settings`, по умолчанию - пустая строка.

**Тип**: `str`


### `__doc__`

**Описание**: Документация проекта. Получается из `doc_str`, по умолчанию - пустая строка.

**Тип**: `str`


### `__details__`

**Описание**: Дополнительные детали. По умолчанию - пустая строка.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта. Получается из настроек `settings`, по умолчанию - пустая строка.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права. Получается из настроек `settings`, по умолчанию - пустая строка.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка на пожертвование автору за чашечку кофе. Получается из настроек `settings`, по умолчанию - ссылка на боости.

**Тип**: `str`
