# Модуль hypotez/src/webdriver/edge/__init__.py

## Обзор

Этот модуль предоставляет начальную точку входа для работы с драйвером Edge.

## Постоянные значения

### `MODE`

**Описание**:  Устанавливает режим работы (например, 'dev', 'prod').

**Значение**: 'dev'

## Импорты

### `from .edge import Edge`

**Описание**: Импортирует класс `Edge` из модуля `edge.py` внутри текущей директории.
