# Модуль: src.utils.instructions.about_utils

## Обзор

Данный модуль содержит общие инструкции для модулей `src.utils` и `src.utils.converters`.  Он описывает принципы работы этих модулей и запрещает использование некоторых функций.


## Инструкции

### Модули `src.utils` и `src.utils.converters`

- Модули `src.utils` и `src.utils.converters` предназначены для выполнения служебных функций проекта.
- Они разработаны так, чтобы минимизировать использование внутренних утилит.
- В этих модулях **запрещено** использование функций `j_loads`, `j_loads_ns` и `j_dumps`.
-  Для работы с JSON необходимо использовать стандартный модуль `json`.
