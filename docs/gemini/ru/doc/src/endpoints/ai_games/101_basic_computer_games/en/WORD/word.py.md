# Игра "Фибоначчи в степени"

## Обзор

Данный модуль реализует игру "Фибоначчи в степени", в которой компьютер выбирает случайное число, а игрок пытается его угадать. Число, с которым сравнивается ввод пользователя, вычисляется путем возведения случайного числа в степень числа Фибоначчи, соответствующего номеру попытки.

## Функции

### `fibonacci`

**Описание**: Вычисляет n-е число Фибоначчи.

**Параметры**:
- `n` (int): Номер числа Фибоначчи для вычисления.

**Возвращает**:
- int: n-е число Фибоначчи.

**Пример:**
```python
fibonacci(5) # Возвращает 5
fibonacci(10) # Возвращает 55
```

## Основная логика игры

1. **Инициализация переменных**:
   - `numberOfGuesses = 0`:  Счетчик количества попыток устанавливается в 0.
   - `targetNumber = random.randint(1, 100)`: Генерируется случайное целое число от 1 до 100.

2. **Основной игровой цикл (`while True`)**:
   - Увеличивает `numberOfGuesses` на 1.
   - Вычисляет `fibonacciNumber` с помощью функции `fibonacci(numberOfGuesses)`.
   - Вычисляет `poweredNumber` как `targetNumber` в степени `fibonacciNumber`.
   - Запрашивает ввод числа у пользователя.
   - Обрабатывает ошибку `ValueError`, если пользователь вводит не целое число.
   - Проверяет, равно ли введенное число (`userGuess`) вычисленному (`poweredNumber`):
     - Если числа равны, выводит сообщение о победе и заканчивает игру.
     - Если числа не равны, выводит сообщение "Попробуйте еще раз!".
