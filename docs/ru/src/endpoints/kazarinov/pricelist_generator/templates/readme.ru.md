# Шаблон для создания HTML отчета из сценария мехирона

## Обзор

Данный шаблон представляет собой структуру для создания HTML-отчета на основе сценария мехирона. Он предназначен для использования в качестве основы для построения отчетов с информацией, полученной из сценариев.

## Структура HTML

Шаблон включает в себя следующие элементы:

-   **`<html>`**: Корневой элемент HTML-документа.
-   **`<head>`**: Заголовочная часть документа, включающая:
    -   **`<meta charset="UTF-8">`**: Установка кодировки UTF-8 для корректного отображения символов.
    -   **`<title>`**: Заголовок документа, отображаемый во вкладке браузера (здесь: `Мехирон отчет`).
    -   **`<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bulma@0.9.4/css/bulma.min.css">`**: Ссылка на CSS-фреймворк Bulma для стилизации элементов.
-   **`<body>`**: Тело документа, содержащее основное содержимое:
    -   **`<section class="section">`**: Секция для основного содержимого.
        -   **`<div class="container">`**: Контейнер для центрирования содержимого.
            -   **`<h1 class="title">Мехирон отчет</h1>`**: Заголовок отчета.
            -   **`<h2 class="subtitle">Отчет с описанием сценария</h2>`**: Подзаголовок отчета.
            -   **`<div class="content">`**: Контейнер для контента.
                -   **`{{ content }}`**: Место для вставки сгенерированного HTML-контента из сценария мехирона.

## Использование

1.  Скопируйте данный шаблон в файл `template.html`.
2.  Используйте этот шаблон как основу для генерации HTML-отчета.
3.  Подставьте сгенерированный HTML-контент из сценария мехирона в место, указанное как `{{ content }}`.
4.  Полученный HTML-файл можно открыть в браузере для просмотра отчета.
