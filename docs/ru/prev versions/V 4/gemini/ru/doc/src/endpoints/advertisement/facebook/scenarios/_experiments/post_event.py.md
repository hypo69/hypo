# Модуль `post_event`

## Обзор

Модуль `post_event` предназначен для обработки и отправки информации о мероприятиях (events) в группы Facebook. Он взаимодействует с данными о мероприятиях, хранящимися в JSON-файлах, и использует класс `FacebookPromoter` для публикации этих мероприятий в указанные группы Facebook. Модуль также предоставляет функциональность для отправки мероприятий в определенную группу Facebook.

## Подробней

Этот модуль является частью системы автоматизации маркетинга в Facebook и используется для автоматической публикации информации о событиях, полученных от AliExpress, в группах Facebook. Он позволяет загружать данные о мероприятиях из JSON-файлов, расположенных в Google Drive, и отправлять их в указанные группы Facebook. Это упрощает процесс распространения информации о мероприятиях и экономит время маркетологов.
