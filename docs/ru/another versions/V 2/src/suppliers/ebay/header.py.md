# Модуль `src.suppliers.ebay.header`

## Обзор

Модуль `header.py` предназначен для настройки корневой директории проекта, загрузки настроек из файла `settings.json`, а также чтения документации из файла `README.MD`. Также определяет основные метаданные проекта, такие как имя, версия, автор и т.д.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла, и останавливается на первой директории, содержащей любой из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple, optional): Кортеж с именами файлов или директорий, определяющих корень проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена, иначе путь к директории, где расположен скрипт.

## Глобальные переменные

### `__root__`

**Описание**: Путь к корневой директории проекта.

### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json`. Если файл не найден или содержит ошибку JSON, значение будет `None`.

### `doc_str`

**Описание**: Строка с содержимым документации, загруженной из файла `README.MD`. Если файл не найден или содержит ошибку, значение будет `None`.

### `__project_name__`

**Описание**: Имя проекта, берется из `settings.json` или равно `hypotez` по умолчанию.

### `__version__`

**Описание**: Версия проекта, берется из `settings.json` или пустая строка по умолчанию.

### `__doc__`

**Описание**: Документация проекта, загруженная из `README.MD` или пустая строка по умолчанию.

### `__details__`

**Описание**: Детали проекта, пустая строка по умолчанию.

### `__author__`

**Описание**: Автор проекта, берется из `settings.json` или пустая строка по умолчанию.

### `__copyright__`

**Описание**: Копирайт проекта, берется из `settings.json` или пустая строка по умолчанию.

### `__cofee__`

**Описание**: Сообщение с призывом угостить разработчика кофе, берется из `settings.json` или текст по умолчанию.
