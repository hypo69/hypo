# Модуль `_experiments`

## Обзор

Модуль `_experiments` предназначен для экспериментов и прототипирования в рамках работы с поставщиком Wallashop. Содержит вспомогательные функции и импорты для проведения экспериментов, связанных с обработкой данных о продуктах, категориях и взаимодействием с веб-драйвером.

## Подробней

Этот модуль содержит различные импорты, необходимые для работы с поставщиками, продуктами, категориями, веб-драйверами и утилитами, такими как форматирование строк, нормализация строк и печать. Также в модуле определена функция `start_supplier`, которая создает и возвращает объект поставщика на основе переданных параметров.

## Функции

### `start_supplier`

```python
def start_supplier(supplier_prefix: str = 'aliexpress', locale: str = 'en') -> Supplier:
    """ Старт поставщика """
```

**Назначение**: Функция `start_supplier` создает и возвращает объект поставщика (`Supplier`) на основе переданных параметров.

**Параметры**:

*   `supplier_prefix` (str, optional): Префикс поставщика. По умолчанию `'aliexpress'`.
*   `locale` (str, optional): Локаль поставщика. По умолчанию `'en'`.

**Возвращает**:

*   `Supplier`: Объект поставщика с заданными параметрами.

**Как работает функция**:

1.  Функция принимает префикс поставщика (`supplier_prefix`) и локаль (`locale`) в качестве аргументов.
2.  Создает словарь `params`, содержащий эти параметры.
3.  Создает экземпляр класса `Supplier` с использованием переданных параметров и возвращает его.

```
A (Получение параметров)
|
B (Создание словаря параметров)
|
C (Создание объекта Supplier)
|
D (Возврат объекта Supplier)
```

**Примеры**:

```python
# Пример 1: Запуск поставщика с префиксом 'aliexpress' и локалью 'en'
