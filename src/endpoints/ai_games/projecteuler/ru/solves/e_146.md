## Ответ на Задачу No 146: Исследование простых чисел с определенными разностями

### 1. Анализ задачи и решение
**Понимание задачи:**
* Задача состоит в поиске всех целых чисел `n` в диапазоне `0 < n < 150 000 000`, таких, что все шесть чисел последовательности `n² + 1`, `n² + 3`, `n² + 7`, `n² + 9`, `n² + 13` и `n² + 27` являются простыми.
* Необходимо перебрать все `n` в заданном диапазоне, для каждого `n` проверить все шесть чисел на простоту, и если все они простые, добавить `n` к общей сумме.

**Решение:**
1. **Определение функции для проверки простоты:** Создаем функцию `is_prime(number)`, которая проверяет, является ли заданное число простым. Мы можем использовать стандартный алгоритм проверки, перебирая делители до квадратного корня из числа.
2. **Перебор n:** Итерируем по всем `n` от 1 до 149 999 999.
3. **Проверка последовательности:** Для каждого `n` вычисляем все шесть чисел последовательности и проверяем их на простоту, используя нашу функцию `is_prime()`.
4. **Суммирование:** Если все шесть чисел простые, добавляем `n` к общей сумме `total_sum`.
5. **Возврат результата:** После окончания цикла возвращаем `total_sum`.

### 2. Алгоритм решения
1. Начать
2. Инициализировать переменную `total_sum` со значением 0
3. Определить функцию `is_prime(number)`:
   * Если `number` меньше 2, вернуть `False`
   * Для каждого `i` от 2 до квадратного корня из `number`:
     * Если `number` делится на `i` без остатка, вернуть `False`
   * Вернуть `True`
4. Для каждого числа `n` от 1 до 149 999 999:
    * Вычислить `n_squared = n * n`
    * Вычислить `num1 = n_squared + 1`, `num2 = n_squared + 3`, `num3 = n_squared + 7`, `num4 = n_squared + 9`, `num5 = n_squared + 13`, `num6 = n_squared + 27`
    * Если `is_prime(num1)` и `is_prime(num2)` и `is_prime(num3)` и `is_prime(num4)` и `is_prime(num5)` и `is_prime(num6)`:
        * Прибавить `n` к `total_sum`
5. Вернуть `total_sum`
6. Конец

### 3. Реализация на Python 3.12
```python
import math

def is_prime(number):
    if number < 2:
        return False
    for i in range(2, int(math.sqrt(number)) + 1):
        if number % i == 0:
            return False
    return True

def sum_of_n_with_prime_sequence(limit):
    total_sum = 0
    for n in range(1, limit):
        n_squared = n * n
        num1 = n_squared + 1
        num2 = n_squared + 3
        num3 = n_squared + 7
        num4 = n_squared + 9
        num5 = n_squared + 13
        num6 = n_squared + 27
        if (is_prime(num1) and is_prime(num2) and is_prime(num3) and 
            is_prime(num4) and is_prime(num5) and is_prime(num6)):
            total_sum += n
    return total_sum

# Пример использования:
limit = 150_000_000
result = sum_of_n_with_prime_sequence(limit)
print(result)
```
