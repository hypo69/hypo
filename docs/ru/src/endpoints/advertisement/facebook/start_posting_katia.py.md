# Модуль для отправки рекламных объявлений в группы Facebook (Katia?)

## Обзор

Модуль предназначен для автоматической отправки рекламных объявлений в различные группы Facebook. Он использует Selenium WebDriver для управления браузером и взаимодействия с Facebook.

## Подробнее

Этот код предназначен для автоматизации процесса размещения рекламных объявлений в группах Facebook. Он использует классы `Driver` и `FacebookPromoter` для управления браузером и выполнения основных действий по размещению.

## Функции

В данном модуле напрямую не определены функции. Однако используются методы класса `FacebookPromoter` для выполнения задач.

## Классы

### `FacebookPromoter`

**Описание**: Класс `FacebookPromoter` отвечает за продвижение рекламных кампаний в Facebook.

**Как работает класс**:
Класс `FacebookPromoter` инициализируется драйвером браузера, путями к файлам с информацией о группах и флагом, указывающим на наличие видео. Он использует драйвер для навигации по Facebook и размещения объявлений в указанных группах.

**Методы**:

- `run_campaigns(campaigns: list)`: Запускает рекламные кампании.

     **Параметры**:
     - `campaigns` (list): Список названий кампаний для запуска.

     **Как работает функция**:
     1. Функция перебирает список кампаний (`campaigns`).
     2. Для каждой кампании вызывается метод `run_campaign` класса `FacebookPromoter`.
     3. В случае прерывания с клавиатуры, регистрируется сообщение в логе.

     **Возвращает**:
     - `None`: Функция ничего не возвращает.

     **Вызывает исключения**:
     - `KeyboardInterrupt`: Возникает при прерывании процесса с клавиатуры.

### `Driver`

**Описание**: Класс `Driver` отвечает за управление экземпляром веб-браузера с использованием Selenium WebDriver.

**Как работает класс**:
Класс `Driver` инициализируется типом браузера (например, Chrome). Он предоставляет методы для навигации по веб-страницам и выполнения действий с веб-элементами.

**Методы**:

- `get_url(url: str)`: Открывает указанный URL в браузере.

     **Параметры**:
     - `url` (str): URL для открытия.

     **Как работает функция**:
     1. Функция использует драйвер Selenium WebDriver для открытия указанного URL в браузере.

     **Возвращает**:
     - `None`: Функция ничего не возвращает.

     **Вызывает исключения**:
     - Отсутствуют явные исключения.

- `execute_locator(l: dict)`: Возвращает веб-элемент по локатору.

     **Параметры**:
     - `l` (dict): Словарь с параметрами локатора.

     **Как работает функция**:
     1. Функция использует локатор `l` для поиска веб-элемента на странице.
     2. Возвращает найденный веб-элемент.

     **Возвращает**:
     - Веб-элемент: Найденный веб-элемент.

     **Вызывает исключения**:
     - Отсутствуют явные исключения.

## Переменные

- `d`: Экземпляр класса `Driver`, используемый для управления браузером. Инициализируется с типом браузера Chrome.
- `filenames`: Список имен файлов, содержащих информацию о группах Facebook.
- `campaigns`: Список названий рекламных кампаний.
- `promoter`: Экземпляр класса `FacebookPromoter`, используемый для запуска рекламных кампаний.

## Логирование

В коде используется модуль `logger` из `src.logger.logger` для записи информации о ходе выполнения программы и ошибок.

Пример использования:

```python
from src.logger.logger import logger

try:
    promoter.run_campaigns(campaigns)
except KeyboardInterrupt:
    logger.info("Campaign promotion interrupted.")
```
Здесь `logger.info` используется для записи сообщения о прерывании рекламной кампании.
