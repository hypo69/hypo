# Задача No 779
## Минимальная сумма

## Условия

Рассмотрим прямоугольную сетку размером *n* × *m*, где *n* обозначает количество строк, а *m* — количество столбцов. Предположим, что мы начинаем в левом верхнем углу сетки. Мы хотим добраться до правого нижнего угла, перемещаясь только вправо или вниз.

Для каждого возможного пути можно рассчитать сумму чисел, через которые мы проходим. Нас интересуют пути с наименьшей возможной суммой.

Например, для сетки 3×4, заданная матрицей
```
  1 1 5 9
  1 1 5 9
  1 1 1 1
```
сумма наименьшего пути равна 8.
```
  1+1+1+1+1+1+1+1 = 8
```

Для сетки 3×4, заданной матрицей
```
  1 1 5 9
  1 1 1 9
  1 1 1 1
```
сумма наименьшего пути равна 7.
```
  1+1+1+1+1+1+1 = 7
```
Обозначим эту минимальную сумму для сетки *n* × *m* через *f(n, m)*.

Сетка создается с использованием следующей псевдослучайной последовательности.
```
s[1] = 290797
s[k+1] = s[k]^2 mod 50515093
```
Затем значения в сетке генерируются с использованием
```
a[n, m] = 1 + (s[m+n] mod 20)
```

Определим *S(N)* как сумму *f(n, m)* для 1 ≤ *n*, *m* ≤ *N*.

Известно, что *S(5) = 1486*.

Найдите *S(500)*.
