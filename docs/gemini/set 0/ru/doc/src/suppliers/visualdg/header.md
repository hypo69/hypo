# Модуль hypotez/src/suppliers/visualdg/header.py

## Обзор

Этот модуль содержит функции для определения корневого каталога проекта и загрузки настроек из файла `settings.json`. Он также предоставляет доступ к информации о проекте, такой как имя, версия, документация, автор и ссылка для пожертвований.

## Функции

### `set_project_root`

**Описание**: Определяет корневой каталог проекта, начиная с текущей директории и поднимаясь по дереву директорий до тех пор, пока не обнаружит файлы-маркеры (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или каталогов, которые будут использоваться для поиска корневого каталога. По умолчанию это (`'pyproject.toml'`, `'requirements.txt'`, `.git`).

**Возвращает**:

- `Path`: Путь к корневому каталогу проекта, если найден, иначе текущий каталог.

**Вызывает исключения**:

- Нет


### `load_settings`

**Описание**: Загружает настройки проекта из файла `settings.json`.

**Параметры**:

- Нет

**Возвращает**:

- `dict | None`: Словарь с настройками проекта или `None`, если файл не найден или поврежден.

**Вызывает исключения**:

- `FileNotFoundError`: Если файл `settings.json` не найден.
- `json.JSONDecodeError`: Если содержимое файла `settings.json` не является валидным JSON.


## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта, полученный с помощью функции `set_project_root`.

**Тип**: `Path`


### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json`.

**Тип**: `dict | None`


### `doc_str`

**Описание**: Строка с текстом документации проекта, загруженная из файла `README.MD`.

**Тип**: `str | None`


### `__project_name__`

**Описание**: Имя проекта, полученное из настроек проекта.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта, полученная из настроек проекта.

**Тип**: `str`


### `__doc__`

**Описание**: Документация проекта, полученная из файла `README.MD`.

**Тип**: `str`


### `__details__`

**Описание**: Подробности о проекте.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта, полученный из настроек проекта.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права проекта, полученные из настроек проекта.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка для пожертвований на кофе автору проекта.

**Тип**: `str`


## Импорты

Этот модуль использует следующие импорты:

- `sys`
- `json`
- `packaging.version`
- `pathlib`
- `src.gs`


## Замечания

- Модуль использует функцию `set_project_root` для корректного поиска корневого каталога проекта.
- Он обрабатывает потенциальные ошибки при чтении файла `settings.json` и `README.MD`, предотвращая аварийный выход программы.
- Значения переменных `__project_name__`, `__version__`, `__doc__`, `__author__`, `__copyright__`, и `__cofee__` инициализируются значениями из настроек, но возвращают значение по умолчанию, если соответствующие ключи отсутствуют.
