# Модуль hypotez/src/ai/gemini/header.py

## Обзор

Этот модуль предоставляет функции для определения корневой директории проекта и загрузки конфигурации из файла `config.json`. Он также предоставляет доступ к документации, хранящейся в файле `README.MD`.

## Функции

### `set_project_root`

**Описание**:  Находит корневую директорию проекта, начиная с текущей директории и проходя вверх по древу директорий. Останавливается на первой директории, содержащей один из указанных файлов-маркеров.

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, используемых для определения корневой директории. По умолчанию включает ('pyproject.toml', 'requirements.txt', '.git').

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена. В противном случае возвращает директорию, в которой расположен текущий файл.

**Вызывает исключения**:

- Нет.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный вызовом `set_project_root()`.

**Тип**: `Path`


### `config`

**Описание**: Словарь, содержащий конфигурацию проекта, загруженную из файла `config.json`.

**Тип**: `dict` | `None`


### `doc_str`

**Описание**: Строка, содержащая содержимое файла `README.MD`.

**Тип**: `str` | `None`


### `__project_name__`

**Описание**: Имя проекта, полученное из `config`. По умолчанию 'hypotez'.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта, полученная из `config`. По умолчанию пустая строка.

**Тип**: `str`


### `__doc__`

**Описание**: Документация проекта, полученная из `doc_str`. По умолчанию пустая строка.

**Тип**: `str`


### `__details__`

**Описание**: Подробная информация о проекте. По умолчанию пустая строка.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта, полученный из `config`. По умолчанию пустая строка.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права проекта, полученные из `config`. По умолчанию пустая строка.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка на способ поддержания проекта.


**Тип**: `str`


## Обработка исключений

Обработка ошибок `FileNotFoundError` и `json.JSONDecodeError` при чтении файлов `config.json` и `README.MD` предотвращает падение программы при их отсутствии.
