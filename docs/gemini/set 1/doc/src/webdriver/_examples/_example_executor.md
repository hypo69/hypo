# Модуль `hypotez/src/webdriver/_examples/_example_executor.py`

## Обзор

Данный модуль содержит примеры использования класса `ExecuteLocator` для управления веб-драйвером. Он демонстрирует различные сценарии, включая простые и сложные локеи, обработку ошибок, отправку сообщений и работу с несколькими локерами. Примеры иллюстрируют как получать текст с элемента, отправлять сообщения, использовать разные события, и обрабатывать исключения при выполнении локеров. Модуль демонстрирует использование класса `ExecuteLocator` для локализации элементов на веб-странице и работы с ними.

## Функции

### `main`

**Описание**: Функция `main` содержит примеры использования `ExecuteLocator`. Она создает экземпляр веб-драйвера, инициализирует `ExecuteLocator`, выполняет различные локеи, обрабатывает исключения и закрывает драйвер.

**Параметры**:
- Не принимает параметры.

**Возвращает**:
- Не возвращает значение.

**Вызывает исключения**:
- `ExecuteLocatorException`: Возникает, если при выполнении локеров возникают ошибки.


### `execute_locator`

**Описание**:  Метод `execute_locator` из класса `ExecuteLocator`.  Выполняет заданный локер.

**Параметры**:
- `locator (dict)`: Словарь с настройками локера.  Должен содержать ключи 'by', 'selector', 'attribute', и т.д.
- `continue_on_error (bool, optional):`  Если `True`, то при ошибке выполнение не прерывается, а продолжается. По умолчанию `False`.

**Возвращает**:
- `dict | None`: Словарь с результатом выполнения локера или `None`, если произошла ошибка или локер не найден.

**Вызывает исключения**:
- `ExecuteLocatorException`: Возникает, если при выполнении локера возникают ошибки.

### `send_message`

**Описание**:  Метод `send_message` из класса `ExecuteLocator`. Отправляет сообщение в элемент.

**Параметры**:
- `locator (dict)`: Словарь с настройками локера.
- `message (str)`: Текст сообщения.
- `typing_speed (float, optional):` Скорость печати сообщения. По умолчанию `None`.
- `continue_on_error (bool, optional):` Если `True`, то при ошибке выполнение не прерывается, а продолжается. По умолчанию `False`.

**Возвращает**:
- `dict | None`: Словарь с результатом выполнения или `None`, если произошла ошибка или локер не найден.

**Вызывает исключения**:
- `ExecuteLocatorException`: Возникает, если при выполнении локера возникают ошибки.


### `evaluate_locator`

**Описание**: Метод `evaluate_locator` из класса `ExecuteLocator`.  Оценивает локер и возвращает значение атрибута.

**Параметры**:
- `attribute (str)`: Имя атрибута для оценки.

**Возвращает**:
- `str | None`: Значение атрибута или `None`, если атрибут не найден или произошла ошибка.

**Вызывает исключения**:
- `ExecuteLocatorException`: Возникает, если при выполнении локера возникают ошибки.
