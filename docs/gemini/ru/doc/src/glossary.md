# Глоссарий

## Обзор

Данный документ содержит глоссарий, описывающий ключевые понятия и структуры, используемые в проекте.

## Классы

### `Driver`

**Описание**: Базовый класс для управления драйверами веб-браузеров (например, Chrome, Firefox).

**Атрибуты**:

- `Executor`: Объект для выполнения действий.
- `drivers`: Список доступных драйверов (Chrome, Firefox и т.д.).

### `Supplier`

**Описание**: Класс для взаимодействия с поставщиками данных (например, Amazon, Aliexpress, Morlevi).

**Атрибуты**:

- `suppliers`: Список доступных поставщиков.
- `Graber`: Объект для извлечения данных.

### `Product`

**Описание**: Класс, представляющий продукт.

**Атрибуты**:

- `ProductFields`:  Объект, содержащий поля продукта.
