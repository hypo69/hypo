# Модуль schemas.py

## Обзор

Модуль `schemas.py` содержит Pydantic-схемы для валидации и типизации данных, связанных с продуктами в цифровом магазине Telegram-бота. Он определяет модели данных для идентификации продукта, создания и обновления информации о продукте.

## Подробней

Этот модуль предоставляет структуры данных, которые используются для определения формата запросов и ответов при взаимодействии с API бота, касающегося управления продуктами. Схемы обеспечивают проверку типов данных, минимальную длину строк и другие ограничения, чтобы гарантировать корректность данных.

## Классы

### `ProductIDModel`

**Описание**: Модель для представления идентификатора продукта.

**Как работает класс**:
Этот класс используется для передачи и валидации идентификатора продукта. Он содержит только одно поле - `id`, которое должно быть целым числом.

**Параметры**:
- `id` (int): Идентификатор продукта.

**Примеры**:
```python
product_id = ProductIDModel(id=123)
print(product_id.id)  # Вывод: 123
```

### `ProductModel`

**Описание**: Модель для представления информации о продукте.

**Как работает класс**:
Этот класс используется для валидации данных продукта, таких как имя, описание, цена, ID категории, ID файла и скрытый контент. Он определяет минимальную длину для имени и описания, а также проверяет, что цена и ID категории больше нуля.

**Параметры**:
- `name` (str): Название продукта (минимальная длина 5 символов).
- `description` (str): Описание продукта (минимальная длина 5 символов).
- `price` (int): Цена продукта (должна быть больше 0).
- `category_id` (int): ID категории продукта (должен быть больше 0).
- `file_id` (str | None): ID файла продукта (может быть `None`).
- `hidden_content` (str): Скрытый контент продукта (минимальная длина 5 символов).

**Примеры**:
```python
product_data = ProductModel(
    name='Awesome Product',
    description='This is an amazing product',
    price=99,
    category_id=1,
    file_id='file123',
    hidden_content='Secret content'
)
print(product_data.name)  # Вывод: Awesome Product
```
```python
from pydantic import BaseModel, Field

class ProductModel(BaseModel):
    name: str = Field(..., min_length=5)
    description: str = Field(..., min_length=5)
    price: int = Field(..., gt=0)
    category_id: int = Field(..., gt=0)
    file_id: str | None = None
    hidden_content: str = Field(..., min_length=5)

    # Пример создания экземпляра класса
    product = ProductModel(
        name="Новый продукт",
        description="Описание нового продукта",
        price=100,
        category_id=10,
        file_id="file_id_123",
        hidden_content="Скрытый контент продукта"
    )

    # Пример доступа к атрибутам экземпляра класса
    print(f"Название продукта: {product.name}")
    print(f"Цена продукта: {product.price}")
