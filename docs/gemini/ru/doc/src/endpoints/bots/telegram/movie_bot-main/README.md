# Документация для Movie Bot

## Обзор

Этот документ предоставляет информацию о телеграм-боте Movie Bot, предназначенном для поиска ссылок на бесплатный просмотр фильмов и сериалов. Бот разработан на основе библиотеки Aiogram и использует сервисы Google для ассоциативного поиска, а также библиотеку Кинопоиска и сервис [w2.kpfr/wiki](https://w2.kpfr.wiki/). В боте реализована защита от флудинга с использованием middlewares, а для парсинга используется BeautifulSoup.

## Подорбней

Movie Bot предоставляет пользователям возможность находить ссылки на бесплатный просмотр фильмов и сериалов через Telegram. Он использует различные источники и технологии для обеспечения релевантных результатов поиска. Защита от флуда реализована для обеспечения стабильной работы бота при большом количестве запросов.

## Быстрый старт

### Клонирование проекта

1.  Склонируйте проект на свой компьютер.

### Настройка `.env` файла

2.  Создайте файл `.env` и наполните его в предложенном формате:

```
TOKEN=123456789 # Токен вашего telegram бота
```

### Создание и активация виртуального окружения

3.  Если у вас нет виртуального окружения, создайте и активируйте его:

```shell
python -m venv venv
```

```shell
venv\\Scripts\\activate.bat
```

### Установка зависимостей

4.  Обновите pip и установите необходимые зависимости:

```shell
pip install --upgrade pip
```

```shell
pip install -r requirements.txt
```

### Запуск бота

5.  Запустите `run.py`:

```shell
python run.py
```
