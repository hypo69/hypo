# Модуль hypotez/src/goog/header.py

## Обзор

Этот модуль содержит функцию `set_project_root` для определения корневого каталога проекта и настройку пути поиска. Он также загружает настройки из файла `settings.json` и документацию из `README.MD`, если они существуют.  Модуль инициализирует переменные, представляющие имя проекта, версию, документацию, автора, копирайт и ссылку на пожертвования.


## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная с текущей директории и двигаясь вверх по иерархии директорий, пока не найдёт файлы-маркеры (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж имён файлов или директорий, по которым определяется корень проекта.

**Возвращает**:

- `Path`: Путь к корневому каталогу проекта, если найден, в противном случае — путь к каталогу, где расположен скрипт.


**Вызывает исключения**:

- Нет


###  Загрузка настроек и документации

Модуль включает обработку исключений `FileNotFoundError` и `json.JSONDecodeError` при чтении файлов `settings.json` и `README.MD`.  Это предотвращает ошибку выполнения при отсутствии указанных файлов или некорректном формате данных в них.


## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта. Инициализируется функцией `set_project_root()`.

**Тип**: `Path`


### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженный из файла `settings.json`.

**Тип**: `dict`


### `doc_str`

**Описание**: Строка, содержащая текст документации, загруженная из файла `README.MD`.

**Тип**: `str`


### `__project_name__`

**Описание**: Имя проекта, полученное из настроек или заданное по умолчанию.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта, полученная из настроек или пустая строка по умолчанию.

**Тип**: `str`


### `__doc__`

**Описание**: Текст документации, полученный из файла `README.MD` или пустая строка по умолчанию.

**Тип**: `str`


### `__details__`

**Описание**: Подробная информация о проекте, по умолчанию пустая строка.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта, полученный из настроек или пустая строка по умолчанию.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права проекта, полученные из настроек или пустая строка по умолчанию.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка на пожертвование (на cup of coffee), полученная из настроек или строка по умолчанию.

**Тип**: `str`
