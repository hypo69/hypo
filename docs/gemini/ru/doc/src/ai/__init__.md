# Модуль `hypotez/src/ai`

## Обзор

Данный модуль предоставляет интерфейсы для взаимодействия с моделями искусственного интеллекта, такими как Gemini и OpenAI.  Он определяет константу `MODE`, которая используется для выбора режима работы (например, `dev` или `prod`).  Модуль импортирует классы `GoogleGenerativeAI` и `OpenAIModel` из соответствующих подмодулей.

## Константы

### `MODE`

**Описание**: Строковая константа, определяющая режим работы модуля.  По умолчанию установлено значение 'dev'.


## Классы

### `GoogleGenerativeAI`

**Описание**:  Класс, предоставляющий интерфейс для работы с моделями Gemini.  Подробная документация находится в файле `hypotez/src/ai/gemini.py`.


### `OpenAIModel`

**Описание**: Класс, предоставляющий интерфейс для работы с моделями OpenAI.  Подробная документация находится в файле `hypotez/src/ai/openai.py`.
