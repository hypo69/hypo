# Задача No 728
## Запоминание последовательностей

## Условия

Определим последовательность *a*<sub>*n*</sub> следующим образом:

*  *a*<sub>1</sub> = 1
*  *a*<sub>*n*</sub> = *n* - *a*<sub>*a*<sub>*n*-1</sub></sub>, для *n* > 1.

Первые несколько членов последовательности:
1, 2, 1, 3, 2, 4, 3, 5, 4, 6, 5, 7, 6, 8, 7, 9, 8, ...

Определим *S*( *N*) как сумму *a*<sub>*i*</sub>, где 1 ≤ *i* ≤ *N*.
Например, *S*(10) = 1 + 2 + 1 + 3 + 2 + 4 + 3 + 5 + 4 + 6 = 31.

Пусть *f*( *M*, *K*) будет суммой *S*( *n*) для всех *n*, таких что *n* mod *M* = *K*.

Например, *f*(5, 3) = *S*(3) + *S*(8) + *S*(13) + *S*(18) + ...

Известно, что *f*(5, 3) = 471050015.

Найдите *f*(10<sup>6</sup>, 99991).
