# Модуль hypotez/src/utils/convertors/base64.py

## Обзор

Модуль `hypotez/src/utils/convertors/base64.py` предоставляет функцию для декодирования содержимого, закодированного в Base64, и записи его во временный файл с указанным расширением.

## Функции

### `base64_to_tmpfile`

**Описание**: Преобразует содержимое, закодированное в Base64, во временный файл. Функция декодирует содержимое, закодированное в Base64, и записывает его во временный файл с тем же расширением, что и в предоставленном имени файла. Возвращает путь к временному файлу.

**Параметры**:
- `content` (str): Содержимое, закодированное в Base64, которое нужно декодировать и записать в файл.
- `file_name` (str): Имя файла, используемое для извлечения расширения для временного файла.

**Возвращает**:
- `str`: Путь к временному файлу.

**Примеры использования**:

```python
>>> base64_content = "SGVsbG8gd29ybGQh"  # Base64 encoded content "Hello world!"
>>> file_name = "example.txt"
>>> tmp_file_path = base64_to_tmpfile(base64_content, file_name)
>>> print(f"Temporary file created at: {tmp_file_path}")
Temporary file created at: /tmp/tmpfile.txt
```

**Примечания**:
- Функция использует `tempfile.NamedTemporaryFile` для создания временных файлов, что гарантирует их удаление при выходе из функции или при необходимости.
- Обработка ошибок (например, при некорректном Base64 кодировании) не реализована.


### `base64encode`

**Описание**: Функция для кодирования изображения в Base64.

**Параметры**:
- `image_path` (str): Путь к изображению.

**Возвращает**:
- `str`: Строка, представляющая изображение, закодированное в Base64.
