# `popup.html`

## Обзор

Этот HTML-файл представляет собой всплывающее окно расширения Chrome, предназначенное для отправки URL текущей страницы. Он включает в себя кнопку "Send URL", которая при нажатии отправляет URL текущей страницы.

## Структура HTML

### Основные элементы

- `<!DOCTYPE html>`: Объявление типа документа HTML5.
- `<html>`: Корневой элемент HTML.
- `<head>`: Содержит метаданные документа, такие как заголовок и ссылки на скрипты.
  - `<title>URL Sender</title>`: Заголовок всплывающего окна.
  - `<script src="popup.js"></script>`: Ссылка на JavaScript файл `popup.js`, который обрабатывает логику отправки URL.
- `<body>`: Содержит видимый контент страницы.
  - `<button id="sendUrlButton">Send URL</button>`: Кнопка, при нажатии на которую запускается отправка URL.

## Скрипты

### `popup.js`
