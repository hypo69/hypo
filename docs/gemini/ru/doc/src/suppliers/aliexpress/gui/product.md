# Модуль `hypotez/src/suppliers/aliexpress/gui/product.py`

## Обзор

Этот модуль предоставляет класс `ProductEditor` для редактирования данных продуктов, полученных из файла JSON.  Класс позволяет загружать JSON-файлы, отображать основные данные продукта и запускать асинхронную подготовку данных.

## Классы

### `ProductEditor`

**Описание**: Класс `ProductEditor` представляет собой виджет PyQt6, предназначенный для редактирования данных продуктов, загруженных из JSON-файлов.  Он предоставляет интерфейс для выбора файла, загрузки данных, отображения заголовка и описания продукта, а также запуска асинхронной подготовки продукта.

**Атрибуты**:

- `data: SimpleNamespace`: Данные загруженного продукта в формате `SimpleNamespace`.
- `language: str`: Язык продукта (по умолчанию 'EN').
- `currency: str`: Валюта продукта (по умолчанию 'USD').
- `file_path: str`: Путь к загруженному JSON-файлу.
- `editor: AliCampaignEditor`: Экземпляр класса `AliCampaignEditor` для подготовки продукта.

**Методы**:

#### `__init__(self, parent=None, main_app=None)`

**Описание**: Инициализирует виджет `ProductEditor`.

**Параметры**:

- `parent (Optional[QWidget], optional)`: Родительский виджет. По умолчанию `None`.
- `main_app (Optional[MainApp], optional)`: Экземпляр главного приложения. По умолчанию `None`.

**Возвращает**:  None

#### `setup_ui(self)`

**Описание**: Настраивает пользовательский интерфейс виджета.

**Возвращает**: None

#### `setup_connections(self)`

**Описание**: Настраивает связи сигналов и слотов.

**Возвращает**: None

#### `open_file(self)`

**Описание**: Открывает диалог выбора файла JSON.

**Возвращает**: None

#### `load_file(self, file_path)`

**Описание**: Загружает JSON-файл по заданному пути.

**Параметры**:

- `file_path (str)`: Путь к файлу JSON.

**Возвращает**: None

**Обрабатывает исключения**:

- `Exception`:  Возникает в случае ошибки загрузки файла JSON.

#### `create_widgets(self, data)`

**Описание**: Создает виджеты, отображающие данные продукта из загруженного JSON.

**Параметры**:

- `data (SimpleNamespace)`: Данные продукта.

**Возвращает**: None


#### `prepare_product_async(self)`

**Описание**: Асинхронная подготовка продукта.

**Возвращает**: None

**Обрабатывает исключения**:

- `Exception`: Возникает в случае ошибки при подготовке продукта.
