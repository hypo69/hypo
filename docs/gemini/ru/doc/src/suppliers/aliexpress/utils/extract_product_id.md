# Модуль extract_product_id

## Обзор

Этот модуль предоставляет функцию для извлечения идентификаторов продуктов из URL-адресов или списка URL-адресов.  Он может принимать как строку с URL, так и список строк, а также непосредственно идентификаторы продуктов. Функция обрабатывает различные форматы входных данных, включая URL-адреса с и без префикса `/item/`, и возвращает список извлечённых идентификаторов или сам идентификатор в случае, если вход был строкой.  Если для входных данных не найден допустимый идентификатор, возвращается `None`.


## Функции

### `extract_prod_ids`

**Описание**: Функция извлекает идентификаторы продуктов из списка URL-адресов или непосредственно возвращает идентификатор, если он предоставлен.

**Параметры**:
- `urls` (str | list[str]): URL-адрес, список URL-адресов или идентификатор продукта.

**Возвращает**:
- str | list[str] | None: Список извлечённых идентификаторов продуктов, единственный идентификатор или `None`, если не найден допустимый идентификатор.

**Примеры**:
```python
>>> extract_prod_ids("https://www.aliexpress.com/item/123456.html")
'123456'

>>> extract_prod_ids(["https://www.aliexpress.com/item/123456.html", "7891011.html"])
['123456', '7891011']

>>> extract_prod_ids(["https://www.example.com/item/123456.html", "https://www.example.com/item/abcdef.html"])
['123456']

>>> extract_prod_ids("7891011")
'7891011'

>>> extract_prod_ids("https://www.example.com/item/abcdef.html")
None
```

### `extract_id`

**Описание**: Извлекает идентификатор продукта из URL-адреса или проверяет, является ли входной строкой допустимым идентификатором.

**Параметры**:
- `url` (str): URL-адрес или идентификатор продукта.

**Возвращает**:
- str | None: Извлеченный идентификатор продукта или сам ввод, если это допустимый идентификатор, или `None`, если не найден допустимый идентификатор.

**Примеры**:
```python
>>> extract_id("https://www.aliexpress.com/item/123456.html")
'123456'

>>> extract_id("7891011")
'7891011'

>>> extract_id("https://www.example.com/item/abcdef.html")
None
```

**Примечание**: Функция `extract_id` используется внутри `extract_prod_ids` для обработки отдельных URL-адресов или идентификаторов.  Она выполняет основную задачу по извлечению числового идентификатора из строки.
