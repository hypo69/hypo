# Модуль `switch_account`

## Обзор

Модуль предназначен для переключения между аккаунтами в Facebook, используя WebDriver.

## Оглавление

1.  [Функции](#Функции)
    *   [`switch_account`](#switch_account)

## Функции

### `switch_account`

**Описание**: Функция для переключения между аккаунтами, если есть кнопка "Переключить".

**Параметры**:
- `driver` (Driver): Экземпляр WebDriver для управления браузером.

**Возвращает**:
- `None`: Функция не возвращает значения.

**Вызывает исключения**:
- `WebDriverException`: Возникает при проблемах с управлением браузером.
