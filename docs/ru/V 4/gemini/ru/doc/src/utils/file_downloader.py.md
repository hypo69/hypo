# Модуль для скачивания файлов

## Обзор

Модуль содержит функцию `download_file`, которая позволяет скачивать файлы по указанному URL-адресу и сохранять их на диск. Это полезно для получения данных или ресурсов из сети для дальнейшей обработки или использования в проекте.

## Подробнее

Данный модуль предоставляет простой способ загрузки файлов из интернета. Функция `download_file` принимает URL файла и путь для сохранения в качестве аргументов. Она выполняет HTTP-запрос и сохраняет содержимое ответа в указанный файл. Модуль может быть использован для автоматической загрузки необходимых ресурсов, таких как конфигурационные файлы, наборы данных или другие файлы, необходимые для работы приложения.

## Функции

### `download_file`

```python
def download_file(url, destination):
    """
    Args:
        url (str): URL-адрес файла для скачивания.
        destination (str): Путь для сохранения скачанного файла на диск.

    Returns:
        None: Функция ничего не возвращает.

    Raises:
        None: Функция не вызывает исключений напрямую, но может возникнуть `requests.exceptions.RequestException` при проблемах с HTTP-запросом.

    Example:
        >>> file_url = 'https://example.com/path/to/file.txt'
        >>> save_as = 'downloaded_file.txt'
        >>> download_file(file_url, save_as)
        Файл успешно загружен!
    """
    # Отправляем GET-запрос на сервер с указанным URL и передаем флаг stream=True для постепенной загрузки файла
    response = requests.get(url, stream=True)

    # Проверяем, успешен ли запрос (код ответа 200 означает успех)
    if response.status_code == 200:
        # Открываем файл для записи в бинарном режиме (wb)
        with open(destination, 'wb') as file:
            # Скачиваем файл по частям (по 1024 байта), чтобы избежать проблем с памятью при больших файлах
            for chunk in response.iter_content(chunk_size=1024):
                file.write(chunk)  # Записываем каждую часть в файл
        print("Файл успешно загружен!")  # Сообщаем об успешной загрузке
    else:
        # Если код ответа не 200, выводим сообщение об ошибке
        print("Ошибка загрузки файла!")
```

**Описание**: Функция скачивает файл по указанному URL и сохраняет его в указанное место на диске.

**Параметры**:
- `url` (str): URL-адрес файла для скачивания.
- `destination` (str): Путь для сохранения скачанного файла.

**Возвращает**:
- `None`: Функция ничего не возвращает.

**Вызывает исключения**:
- `requests.exceptions.RequestException`: Может возникнуть при проблемах с HTTP-запросом.
