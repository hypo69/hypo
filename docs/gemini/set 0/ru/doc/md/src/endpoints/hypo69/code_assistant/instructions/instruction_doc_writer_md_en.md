# Модуль `example_module`

## Обзор

Этот модуль предоставляет функции для работы с числами.

## Функции

### `add_numbers`

**Описание**: Функция складывает два числа.

**Параметры**:
- `num1` (int): Первое число.
- `num2` (int): Второе число.

**Возвращает**:
- `int`: Сумма двух чисел.

**Вызывает исключения**:
- `TypeError`: Если на вход поданы не числовые значения.
