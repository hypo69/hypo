## Обзор

`popup.html` - это основной HTML-файл для расширения Chrome, предназначенного для взаимодействия с моделями OpenAI. Этот файл включает в себя пользовательский интерфейс, который позволяет пользователям отправлять сообщения в чат с выбранным ассистентом и управлять обучением модели. Файл использует AngularJS для управления динамическим контентом и событиями.

## Структура HTML

HTML-структура разделена на несколько основных секций:

1.  **`<head>`**: 
    *   Содержит заголовок страницы (`<title>`), подключает необходимые скрипты (`angular.min.js`, `jquery-3.5.1.slim.min.js`, `popup.js`) и стили (`style.css`).
2.  **`<body>`**: 
    *   Использует директивы AngularJS `ng-app` и `ng-controller` для инициализации приложения и контроллера.
    *   Содержит заголовок (`<h1>`), навигационные вкладки (`<ul>`, `<li>`).
    *   Раздел для чата (`<div ng-show="isTabActive(\'chat\')">`) содержит:
        *   Выпадающий список для выбора ассистента (`<select>`).
        *   Поле ввода сообщения (`<textarea>`).
        *   Кнопку отправки сообщения (`<button>`).
        *   Область для отображения ответа модели (`<div>`).
    *   Раздел для обучения модели (`<div ng-show="isTabActive(\'model\')">`) содержит:
        *   Поле ввода данных для обучения (`<textarea>`).
        *   Кнопку для запуска обучения (`<button>`).
        *   Область для отображения статуса обучения (`<p>`).

## Скрипты

Файл подключает следующие JavaScript-скрипты:

*   `angular.min.js`: Фреймворк AngularJS для управления интерфейсом.
*   `jquery-3.5.1.slim.min.js`: Библиотека jQuery для упрощения работы с DOM.
*   `popup.js`: Пользовательский скрипт, содержащий логику приложения (контроллер, функции).

## Стили

Стили для элементов управления и оформления находятся в файле `style.css`.

## Интерактивные элементы

### Навигационные вкладки

*   `<li>` элементы с директивами `ng-class` и `ng-click` переключают активную вкладку между чатом и моделью.
*   Функция `isTabActive(tabName)` в `popup.js` определяет, какая вкладка активна.
*   Функция `setActiveTab(tabName)` в `popup.js` устанавливает активную вкладку.

### Вкладка "Chat"

*   `select` с `ng-model` и `ng-options` позволяет выбирать ассистента.
*   `textarea` с `ng-model` для ввода сообщения.
*   `button` с `ng-click` вызывает функцию `sendMessage()` в `popup.js`.
*   `div` с `id="response"` и директивой AngularJS отображает ответ от модели.

### Вкладка "Model"

*   `textarea` с `ng-model` для ввода данных для обучения.
*   `button` с `ng-click` вызывает функцию `trainModel()` в `popup.js`.
*   `p` отображает текущий статус обучения.
