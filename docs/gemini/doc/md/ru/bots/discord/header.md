# Модуль hypotez/src/bots/discord/header.py

## Обзор

Этот модуль содержит вспомогательные функции для работы с проектом hypotez, в том числе определение корневой директории проекта и загрузку настроек из файла `settings.json`.

## Функции

### `get_project_root`

**Описание**:  Находит корневую директорию проекта, начиная с текущей директории. Поиск происходит вверх по иерархии директорий, и завершается в первой директории, содержащей файлы или директории в списке `marker_files`.

**Параметры**:
- `marker_files` (tuple): Кортеж файлов или директорий, которые указывают на корневую директорию проекта. По умолчанию (`pyproject.toml`, `requirements.txt`, `.git`).

**Возвращает**:
- `Path`: Путь к корневой директории проекта, если найдена. В противном случае возвращается путь к директории, где находится данный файл.

**Вызывает исключений**:
- Нет


### `get_settings`

**Описание**: Загружает настройки из файла `settings.json` в корневой директории проекта.

**Возвращает**:
- `dict`: Словарь с настройками, загруженными из файла. Если файл не найден или содержимое файла не является валидным JSON, возвращается `None`.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта.

**Тип**: `Path`


### `settings`

**Описание**: Словарь с настройками, загруженными из файла `settings.json`.

**Тип**: `dict` или `None`

### `doc_str`

**Описание**: Строка с содержимым файла `README.MD` из корневой директории проекта.

**Тип**: `str` или `None`


### `__project_name__`

**Описание**: Название проекта, полученное из настроек (`settings.json`). По умолчанию `hypotez`.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта, полученная из настроек (`settings.json`). По умолчанию пустая строка.

**Тип**: `str`


### `__doc__`

**Описание**: Документация к проекту, полученная из файла `README.MD`. По умолчанию пустая строка.

**Тип**: `str`


### `__details__`

**Описание**: Дополнительная информация о проекте.


**Тип**: `str`


### `__author__`

**Описание**: Автор проекта, полученный из настроек (`settings.json`). По умолчанию пустая строка.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права на проект, полученные из настроек (`settings.json`). По умолчанию пустая строка.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка на способ поддержать автора проекта.


**Тип**: `str`
