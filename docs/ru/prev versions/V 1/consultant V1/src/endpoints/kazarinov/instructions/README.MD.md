**Оптимизированный код**:

```rst
Этот каталог содержит инструкции и системные подсказки для AI модели.
=====================================================================

Этот каталог содержит инструкции и системные подсказки, предназначенные для использования
с AI моделями, такими как Google Gemini или OpenAI.

Файлы
-------
* ``system_instruction_mexiron.md``:
    - Системная инструкция, загружаемая при инициализации модели.
    - Описывает поведение и основные принципы работы модели.

* ``command_instruction_mexiron.md``:
    - Инструкция для модели.
    - Определяет формат ответа и шаблоны для генерации текста.
    - Используется для управления выходными данными модели.

Использование
------------
Данные инструкции и промпты предназначены для использования в процессе инициализации
и работы AI-модели, чтобы обеспечить корректное поведение и ожидаемые результаты.
Системная инструкция устанавливает базовое поведение модели, а инструкция команд определяет
формат и структуру ответов.

Пример
------
    Например, если вы используете Google Gemini, то ``system_instruction_mexiron.md``
    будет задавать контекст для модели, а ``command_instruction_mexiron.md``
    укажет, как модель должна генерировать ответы на запросы.

```
