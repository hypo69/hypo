# Модуль `search.py`

## Обзор

Модуль `search.py` предназначен для выполнения поисковых запросов на сайте Kinopoisk через Google. Он использует библиотеки `BeautifulSoup` и `requests` для парсинга результатов поиска и извлечения необходимой информации о фильмах или сериалах.

## Подробней

Модуль `search.py`  используется для поиска информации о фильмах или сериалах на сайте Kinopoisk. Он выполняет поисковый запрос через Google, парсит результаты поиска и извлекает ссылку, заголовок и описание найденного фильма или сериала.
Этот код важен для реализации функциональности поиска фильмов и сериалов в боте. Он позволяет пользователям находить нужную информацию о фильмах, используя поисковые запросы.

## Функции

### `search_query`

```python
def search_query(query: str, type_movie: str = 'series') -> dict | None:
    """
    Args:
        query (str): Поисковый запрос.
        type_movie (str, optional): Тип искомого контента ('series' или другой). По умолчанию 'series'.

    Returns:
        dict | None: Словарь с информацией о фильме/сериале (ссылка, заголовок, описание) или None, если ничего не найдено.

    Raises:
        Нет.
    """
```

**Описание**: Выполняет поисковый запрос на Kinopoisk через Google и возвращает информацию о найденном фильме или сериале.

**Параметры**:

-   `query` (str): Поисковый запрос.
-   `type_movie` (str, optional): Тип искомого контента ('series' или другой). По умолчанию 'series'.

**Возвращает**:

-   `dict | None`: Словарь с информацией о фильме/сериале (ссылка, заголовок, описание) или `None`, если ничего не найдено.

**Примеры**:

```python
search_query('теория большого взрыва')
```
