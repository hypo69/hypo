# Модуль `graber.py`

## Обзор

Модуль `graber.py` представляет собой базовый класс `Graber` для сбора данных со страниц HTML поставщиков. Он использует веб-драйвер (`Driver`) для извлечения целевых полей, таких как название, описание, спецификация, артикул и цена. Местоположение полей определяется с помощью локаторов, хранящихся в JSON-файлах в директории `locators` каждого поставщика. Для нестандартной обработки полей товара необходимо переопределить соответствующие функции в своем классе, наследовав `Graber`.
