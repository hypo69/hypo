# Модуль `src.suppliers.chat_gpt`

## Обзор

Модуль `src.suppliers.chat_gpt` предоставляет функциональность для интеграции с ChatGPT через Google Sheets. Он включает в себя класс `GptGs`, который позволяет взаимодействовать с ChatGPT и Google Sheets.
