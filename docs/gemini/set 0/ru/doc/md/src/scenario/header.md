# Модуль hypotez/src/scenario/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и получения документации из файла `README.MD`. Он также устанавливает переменные, содержащие информацию о проекте, например, имя, версию, описание, автора, копирайт и ссылку на платформу для поддержки разработчика.

## Функции

### `set_project_root`

**Описание**:  Находит корневую директорию проекта, начиная с директории текущего файла, ищет вверх по дереву директорий, пока не найдёт директорию, содержащую один из файлов-маркеров (например, `pyproject.toml`, `requirements.txt` или `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж имён файлов или директорий, используемых для определения корневой директории проекта. По умолчанию включает `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена, иначе путь к директории, где находится текущий скрипт.

**Вызывает исключения**:

- Нет.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root`.

**Тип**: `Path`

### `settings`

**Описание**: Словарь настроек проекта, загруженный из файла `settings.json`. Если файл не найден или в нём некорректные данные, то `settings` будет иметь значение `None`.

**Тип**: `dict` или `None`

### `doc_str`

**Описание**: Строка документации проекта, загруженная из файла `README.MD`. Если файл не найден или в нём некорректные данные, то `doc_str` будет иметь значение `None`.

**Тип**: `str` или `None`

### `__project_name__`

**Описание**: Имя проекта. По умолчанию `hypotez`. Получается из настроек или устанавливается в значение по умолчанию, если настройки не загружены.

**Тип**: `str`

### `__version__`

**Описание**: Версия проекта. По умолчанию пустая строка. Получается из настроек или устанавливается в значение по умолчанию, если настройки не загружены.

**Тип**: `str`

### `__doc__`

**Описание**: Документация проекта (из `README.MD`). По умолчанию пустая строка.

**Тип**: `str`

### `__details__`

**Описание**:  Дополнительные детали о проекте (значение по умолчанию пустая строка).

**Тип**: `str`

### `__author__`

**Описание**: Автор проекта. По умолчанию пустая строка. Получается из настроек или устанавливается в значение по умолчанию, если настройки не загружены.

**Тип**: `str`

### `__copyright__`

**Описание**: Авторские права на проект. По умолчанию пустая строка. Получается из настроек или устанавливается в значение по умолчанию, если настройки не загружены.

**Тип**: `str`

### `__cofee__`

**Описание**: Ссылка на страницу для поддержки разработчика. По умолчанию ссылка на Boosty. Получается из настроек или устанавливается в значение по умолчанию, если настройки не загружены.

**Тип**: `str`
