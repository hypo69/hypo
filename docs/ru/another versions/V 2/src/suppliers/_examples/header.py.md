# `header.py`

## Обзор

Данный модуль содержит код для настройки путей и импорта необходимых библиотек для работы с проектом. Он определяет корневую директорию проекта и добавляет её в `sys.path`, что позволяет импортировать модули из других директорий проекта.

## Импорты

Модуль импортирует следующие библиотеки:

-   `os`: Для работы с операционной системой, в частности для получения текущей рабочей директории.
-   `sys`: Для взаимодействия с интерпретатором Python, используется для изменения `sys.path`.
-   `pathlib.Path`: Для более удобной работы с путями файловой системы.

## Переменные

### `dir_root`

**Описание**: Переменная `dir_root` типа `Path` хранит путь к корневой директории проекта.

**Инициализация**:

-   Получает текущую рабочую директорию с помощью `os.getcwd()`.
-   Ищет индекс последнего вхождения `hypotez` в строке пути, чтобы выделить корневую директорию проекта.
-   Создает объект `Path` из корневого пути.

### `dir_src`

**Описание**: Переменная `dir_src` типа `Path` хранит путь к директории `src` внутри корневой директории проекта.

**Инициализация**:

- Создает объект `Path` из пути к директории `src`, которая находится внутри `dir_root`.

**Использование:**

-   Путь к корневой директории проекта (`dir_root`) добавляется в `sys.path`.
-   Путь к директории `src` добавляется в `sys.path`.

Это обеспечивает возможность импорта модулей из этих директорий в дальнейшем коде проекта.
