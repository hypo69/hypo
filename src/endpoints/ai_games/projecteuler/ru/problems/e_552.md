# Задача No 552
## Числа Хэмминга в интервале
## Условия
Число Хэмминга – это число, все простые делители которого не превышают 5.
Пусть $H(n)$ обозначает $n$-е число Хэмминга (где $H(1) = 1$).
Рассмотрим числа Хэмминга $H(1) = 1, H(2) = 2, H(3) = 3, H(4) = 4, H(5) = 5, H(6) = 6, H(7) = 8, H(8) = 9, H(9) = 10, H(10) = 12, H(11) = 15, ...$.

Назовем число Хэмминга $H(n)$ *универсальным числом Хэмминга*, если $H(n) - 1$ и $H(n) + 1$ также являются числами Хэмминга.

Вы можете проверить, что есть 20 универсальных чисел Хэмминга меньше 1000: 
2, 3, 4, 5, 8, 9, 10, 15, 20, 32, 36, 48, 63, 75, 80, 125, 243, 250, 312, 625.

Пусть $UH(a, b)$ обозначает количество универсальных чисел Хэмминга между $H(a)$ и $H(b)$, включительно.

Вычислите $UH(10^6, 10^9)$.
