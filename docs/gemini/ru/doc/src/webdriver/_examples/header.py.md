# Модуль `header.py`

## Обзор

Данный модуль содержит вспомогательный код для настройки путей в проекте `hypotez`. Он добавляет корневую директорию проекта и директорию `src` в `sys.path`, что позволяет импортировать модули из этих директорий.

## Содержание

1. [Обзор](#обзор)
2. [Импорты](#импорты)
3. [Переменные](#переменные)

## Импорты

В этом модуле используются следующие импорты:

- `os`: для работы с операционной системой.
- `sys`: для работы с системными переменными и функциями.
- `pathlib.Path`: для удобной работы с путями к файлам и директориям.

```python
import os
import sys
from pathlib import Path
```

## Переменные

### `dir_root`

```python
dir_root : Path = Path (os.getcwd()[:os.getcwd().rfind('hypotez')+7])
```
**Описание**:
Переменная `dir_root` представляет собой корневую директорию проекта. Она вычисляется путем получения текущей рабочей директории, нахождения в ней подстроки 'hypotez', и обрезки строки до этого места.
Тип: `pathlib.Path`
### `dir_src`

```python
dir_src = Path (dir_root, 'src')
```
**Описание**:
Переменная `dir_src` представляет собой директорию `src` относительно корневой директории проекта.
Тип: `pathlib.Path`

### Изменения `sys.path`
```python
sys.path.append (str (dir_root) )  # Добавляю корневую директорию в sys.path
sys.path.append (str (dir_root) ) # Добавляю рабочую директорию в sys.path
```
**Описание**:
В `sys.path` добавляются `dir_root` (корневая директория)  два раза.
