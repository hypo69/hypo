# Модуль `kbs.py`

## Обзор

Модуль `kbs.py` содержит функции для создания различных клавиатур (инлайн и основных) для Telegram-бота, используемых в цифровом магазине. Клавиатуры включают кнопки для навигации, совершения покупок и доступа к различным разделам бота.

## Функции

### `main_user_kb`

**Описание**: Создает главную клавиатуру пользователя с кнопками для перехода к профилю, каталогу, информации о магазине, поддержки автора и админ-панели (если пользователь является администратором).

**Параметры**:
- `user_id` (int): Идентификатор пользователя.

**Возвращает**:
- `InlineKeyboardMarkup`: Сформированная инлайн-клавиатура.

### `catalog_kb`

**Описание**: Создает инлайн-клавиатуру для отображения категорий каталога.

**Параметры**:
- `catalog_data` (List[Category]): Список объектов `Category`, представляющих категории товаров.

**Возвращает**:
- `InlineKeyboardMarkup`: Сформированная инлайн-клавиатура.

### `purchases_kb`

**Описание**: Создает инлайн-клавиатуру для просмотра покупок пользователя.

**Параметры**:
- Нет

**Возвращает**:
- `InlineKeyboardMarkup`: Сформированная инлайн-клавиатура.

### `product_kb`

**Описание**: Создает инлайн-клавиатуру для отображения вариантов покупки товара.

**Параметры**:
- `product_id` (int): Идентификатор товара.
- `price` (int): Цена товара в рублях.
- `stars_price` (int): Цена товара в звездах.

**Возвращает**:
- `InlineKeyboardMarkup`: Сформированная инлайн-клавиатура.

### `get_product_buy_youkassa`

**Описание**: Создает инлайн-клавиатуру для оплаты товара через ЮKassa.

**Параметры**:
- `price` (int): Цена товара.

**Возвращает**:
- `InlineKeyboardMarkup`: Сформированная инлайн-клавиатура.

### `get_product_buy_robocassa`

**Описание**: Создает инлайн-клавиатуру для оплаты товара через Robocassa.

**Параметры**:
- `price` (int): Цена товара в рублях.
- `payment_link` (str): Ссылка на страницу оплаты Robocassa.

**Возвращает**:
- `InlineKeyboardMarkup`: Сформированная инлайн-клавиатура.

### `get_product_buy_stars`

**Описание**: Создает инлайн-клавиатуру для оплаты товара звездами.

**Параметры**:
- `price` (int): Цена товара в звездах.

**Возвращает**:
- `InlineKeyboardMarkup`: Сформированная инлайн-клавиатура.
