# examples.py

## Обзор

Этот файл содержит примеры использования библиотеки `tinytroupe` для создания агентов. Примеры можно использовать напрямую или немного модифицировать для создания собственных агентов.

    - [`create_lila_the_linguist`](#create_lila_the_linguist)

## Функции

### `create_oscar_the_architect`

**Описание**: Создает агента по имени Оскар, архитектора.

**Возвращает**:
- `TinyPerson`: Объект агента Оскар.

### `create_lisa_the_data_scientist`

**Описание**: Создает агента по имени Лиза, специалиста по данным.

**Возвращает**:
- `TinyPerson`: Объект агента Лиза.

### `create_marcos_the_physician`

**Описание**: Создает агента по имени Маркос, врача.

**Возвращает**:
- `TinyPerson`: Объект агента Маркос.

### `create_lila_the_linguist`

**Описание**: Создает агента по имени Лила, лингвиста.

**Возвращает**:
- `TinyPerson`: Объект агента Лила.
