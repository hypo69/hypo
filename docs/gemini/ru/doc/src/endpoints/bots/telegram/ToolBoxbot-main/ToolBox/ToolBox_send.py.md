# ToolBox_send.py

## Обзор

Этот модуль содержит код для отправки сообщения пользователям Telegram с использованием промокода. Он использует библиотеку `telebot` для взаимодействия с Telegram API и `sqlite3` для работы с базой данных пользователей.

## Подробнее

Этот скрипт предназначен для рассылки промокодов пользователям Telegram-бота. Он подключается к базе данных `UsersData.db`, извлекает идентификаторы пользователей, у которых еще не активирован промокод, и отправляет им сообщение с информацией о промокоде. Если отправка сообщения не удалась, в консоль выводится сообщение об ошибке.
