## HeliconeAI

**Описание**: Класс `HeliconeAI` упрощает использование API Helicone.ai и OpenAI. Он инициализирует необходимые клиенты и предоставляет методы для различных задач.

**Атрибуты**:
- `helicone`: Объект для взаимодействия с Helicone.ai.
- `client`: Объект для взаимодействия с OpenAI.

**Методы**:
- `generate_poem`: Генерирует стихотворение.
- `analyze_sentiment`: Анализирует тональность текста.
- `summarize_text`: Создает краткое изложение текста.
- `translate_text`: Переводит текст.

## Функции

### `generate_poem`

**Описание**: Генерирует стихотворение на основе заданного промпта.

**Параметры**:
- `prompt` (str): Текст-промпт для генерации стихотворения.

**Возвращает**:
- `str`: Сгенерированное стихотворение.

**Вызывает исключения**:
- Возможны исключения, связанные с API OpenAI или Helicone.ai.


### `analyze_sentiment`

**Описание**: Анализирует тональность заданного текста.

**Параметры**:
- `text` (str): Текст для анализа тональности.

**Возвращает**:
- `str`: Анализ тональности (например, "положительный", "отрицательный", "нейтральный").

**Вызывает исключения**:
- Возможны исключения, связанные с API OpenAI или Helicone.ai.


### `summarize_text`

**Описание**: Создает краткое изложение заданного текста.

**Параметры**:
- `text` (str): Текст для создания краткого изложения.

**Возвращает**:
- `str`: Краткое изложение текста.

**Вызывает исключения**:
- Возможны исключения, связанные с API OpenAI или Helicone.ai.


### `translate_text`

**Описание**: Переводит заданный текст на указанный целевой язык.

**Параметры**:
- `text` (str): Текст для перевода.
- `target_language` (str): Целевой язык перевода (например, "русский", "английский").

**Возвращает**:
- `str`: Переведенный текст.

**Вызывает исключения**:
- Возможны исключения, связанные с API OpenAI или Helicone.ai.
