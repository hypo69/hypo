# hypo69 Module: Эндпоинты для разработчика

## Обзор

Модуль `hypo69` предоставляет набор эндпоинтов для взаимодействия с различными сервисами, включая чат-бота, модуль обучения модели коду и раннюю разработку модуля парсинга диалогов.


## Сервисы

### `small_talk_bot`

**Описание**: Бот с чатом модели ИИ.  Обеспечивает взаимодействие с моделью чата в формате диалога.

### `code_assistant`

**Описание**: Модуль обучения модели коду проекта.  Предоставляет инструменты и API для обучения модели на примерах кода, улучшения ее качества и взаимодействия.

### `psychologist_bot`

**Описание**:  Ранняя разработка модуля парсинга диалогов.  Предварительные функции и API для анализа и обработки диалогов, направленные на понимание и интерпретацию пользовательских запросов.
