# Модуль hypotez/src/suppliers/etzmaleh/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и чтения документации из файла `README.MD`.  Он устанавливает корневую директорию в `sys.path`, что позволяет импортировать модули из проекта.  Модуль также предоставляет переменные, содержащие информацию о проекте (название, версия, автор, лицензия и т.д.).


## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с текущей директории и перемещаясь вверх по дереву каталогов, пока не обнаружит файлы, указанные в аргументе `marker_files`.

**Параметры**:

- `marker_files` (tuple): Кортеж с именами файлов или директорий, по которым будет определяться корень проекта. По умолчанию содержит `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена; в противном случае - директория, в которой расположен текущий скрипт.


**Вызывает исключения**:

- Нет.


###  (Дополнительные переменные, доступные в модуле)


**__root__ (Path):**  Путь к корневой директории проекта, полученный с помощью функции `set_project_root()`.

**settings (dict):** Словарь настроек, загруженный из файла `settings.json`, или `None`, если файл не найден или содержит ошибки.

**doc_str (str):**  Содержимое файла `README.MD`, или `None` если файл не найден или содержит ошибки.

**__project_name__ (str):** Название проекта, полученное из `settings.json`, по умолчанию `hypotez`.

**__version__ (str):** Версия проекта, полученная из `settings.json`, по умолчанию пустая строка.

**__doc__ (str):**  Документация проекта, полученная из `README.MD`, по умолчанию пустая строка.

**__details__ (str):** Подробная информация о проекте, по умолчанию пустая строка.

**__author__ (str):** Автор проекта, полученный из `settings.json`, по умолчанию пустая строка.

**__copyright__ (str):** Авторские права, полученные из `settings.json`, по умолчанию пустая строка.

**__cofee__ (str):** Ссылка для пожертвований на кофе, полученная из `settings.json`, по умолчанию содержит ссылку.
