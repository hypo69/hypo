# revai (rev.com - модель, которая умеет работать с звуковыми файлами переговоров, совещаний, звонков и т.п.)

## Обзор

Данный модуль предоставляет функциональность для взаимодействия с API rev.com, сервисом для транскрибирования аудиофайлов. Он включает в себя методы для управления заданиями транскрибирования, получения результатов и другой связанной информации.
