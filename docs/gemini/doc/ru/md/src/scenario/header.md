# Модуль hypotez/src/scenario/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`. Он также загружает README.md для документации.  Модуль инициализирует переменные, такие как имя проекта, версию, описание, и другие детали проекта, полученные из настроек.

## Функции

### `set_project_root`

**Описание**:  Находит корневую директорию проекта, начиная от текущей директории и двигаясь вверх по дереву директорий, пока не найдет директорию, содержащую один из файлов, указанных в аргументе `marker_files`.

**Параметры**:
- `marker_files` (tuple): Кортеж с именами файлов или директорий, которые используются для определения корневой директории проекта.  По умолчанию: `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории проекта. Если корневая директория не найдена, возвращается директория, где находится скрипт.

**Примеры использования**:
```python
project_root = set_project_root()
print(project_root)
```


### Подготовка окружения

**Описание**:  Выполняет предварительные действия по подготовке среды выполнения, например, устанавливает переменную окружения, загружает файлы конфигурации и т.д.

**Действия**:
- Ищет корневую директорию проекта, используя функцию `set_project_root`.
- Добавляет корневую директорию в `sys.path` для импорта модулей.
- Загружает настройки из файла `settings.json` в переменную `settings`.
- Загружает содержимое файла `README.MD` в переменную `doc_str`.
- Инициализирует переменные проекта: `__project_name__`, `__version__`, `__doc__`, `__details__`, `__author__`, `__copyright__`, `__cofee__`.


**Обработка ошибок**:
- Если файл `settings.json` не найден или не может быть декодирован, `settings` устанавливается в `None`.
- Если файл `README.MD` не найден или не может быть декодирован, `doc_str` устанавливается в `None`.


## Переменные

**Описание**: Переменные, содержащие информацию о проекте.

- `__root__` (Path): Путь к корневой директории проекта.
- `settings` (dict | None): Настройки проекта, загруженные из `settings.json`.
- `doc_str` (str | None): Содержимое файла `README.MD`.
- `__project_name__` (str): Имя проекта, взятое из настроек. По умолчанию `'hypotez'`.
- `__version__` (str): Версия проекта, взятая из настроек. По умолчанию `''`.
- `__doc__` (str):  Описание проекта, взятое из файла `README.MD`. По умолчанию `''`.
- `__details__` (str):  Дополнительные данные о проекте. По умолчанию `''`.
- `__author__` (str): Автор проекта, взятый из настроек. По умолчанию `''`.
- `__copyright__` (str): Авторские права, взятые из настроек. По умолчанию `''`.
- `__cofee__` (str): Ссылка на поддержку разработчика. По умолчанию ссылка на Boosty.
