# Модуль hypotez/src/webdriver/firefox/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`. Он также загружает описание проекта из файла `README.MD`.  Модуль инициализирует глобальные переменные, содержащие информацию о проекте, такие как имя, версия, описание, автор и т.д.

## Функции

### `set_project_root`

**Описание**:  Находит корневую директорию проекта, начиная с текущего файла и двигаясь вверх по директориям.  Останавливается на первой директории, содержащей один из указанных файлов/каталогов (по умолчанию это `pyproject.toml`, `requirements.txt` и `.git`). Если корневой каталог не найден, возвращает директорию, где находится текущий файл. Также добавляет корневой каталог в системный путь `sys.path`, если его там еще нет.

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или каталогов, по которым определяется корень проекта.  По умолчанию это `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Объект `Path` представляющий путь к корневой директории проекта.


### Функции для работы с настройками и описанием проекта

**Описание**: Функции для обработки настроек из файла `settings.json` и документации проекта из `README.MD`.

**Возвращаемые значения**:

Эти функции (не указанные отдельно) могут возвращать `None`, если соответствующий файл не найден или не содержит корректные данные (в случае с `json.JSONDecodeError` или `FileNotFoundError`).

**Обрабатываемые исключения**:

- `FileNotFoundError`: Возникает, если файл `settings.json` или `README.MD` не найден.
- `json.JSONDecodeError`: Возникает, если содержимое файла `settings.json` не соответствует формату JSON.


## Глобальные переменные

**Описание**: Глобальные переменные, инициализируемые данным модулем.

- `__root__` (Path): Путь к корневой директории проекта. Инициализируется функцией `set_project_root()`.
- `settings` (dict): Словарь, содержащий настройки проекта, загруженные из `settings.json`. Может иметь значение `None`.
- `doc_str` (str): Строка с описанием проекта, загруженная из `README.MD`. Может иметь значение `None`.
- `__project_name__` (str): Имя проекта.  Используется значение из `settings.json` или значение по умолчанию `hypotez` , если настройки не загружены.
- `__version__` (str): Версия проекта.  Используется значение из `settings.json` или значение по умолчанию `''` , если настройки не загружены.
- `__doc__` (str): Документация проекта. Значение `doc_str` или пустая строка, если  `doc_str` не найден.
- `__details__` (str):  Дополниетльная информация. По умолчанию пустая строка.
- `__author__` (str): Автор проекта.  Значение из `settings.json` или пустая строка, если настройки не загружены.
- `__copyright__` (str): Авторские права.  Значение из `settings.json` или пустая строка, если настройки не загружены.
- `__cofee__` (str): Ссылка на Boosty. Значение из `settings.json` или значение по умолчанию, если настройки не загружены.
