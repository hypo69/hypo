# Модуль hypotez/src/_examples/header.py

## Обзор

Данный модуль содержит вспомогательные функции для определения корневой директории проекта и загрузки настроек. Он определяет переменные, хранящие информацию о проекте, такие как имя, версия, описание и другие метаданные, полученные из файла настроек `settings.json` и файла `README.MD`.

## Оглавление

* [Функции](#функции)
    * [`set_project_root`](#set_project_root)

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с текущей директории и идя вверх по иерархии директорий. Останавливается на первой директории, содержащей указанные файлы-маркеры.

**Параметры**:

* `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для идентификации корневой директории проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

* `Path`: Путь к корневой директории проекта, если найдена, иначе - путь к директории, где расположен скрипт.

**Обрабатывает исключения**:

* Никаких исключений не обрабатывается.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root`.

**Тип**: `Path`

### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженный из файла `settings.json`.

**Тип**: `dict`

**Значение по умолчанию**: `None`

### `doc_str`

**Описание**: Строка, содержащая текст из файла `README.MD`.

**Тип**: `str`

**Значение по умолчанию**: `None`

### `__project_name__`

**Описание**: Имя проекта, полученное из настроек (`settings.json`) или имеющее значение по умолчанию `'hypotez'`.

**Тип**: `str`

**Значение по умолчанию**: `'hypotez'`


### `__version__`

**Описание**: Версия проекта, полученная из настроек (`settings.json`) или имеющая значение по умолчанию `''`.

**Тип**: `str`

**Значение по умолчанию**: `''`

### `__doc__`

**Описание**: Описание проекта, полученное из файла `README.MD` или имеющее значение по умолчанию `''`.

**Тип**: `str`

**Значение по умолчанию**: `''`


### `__details__`

**Описание**: Дополнительные детали проекта. По умолчанию пустая строка.

**Тип**: `str`

**Значение по умолчанию**: `''`


### `__author__`

**Описание**: Автор проекта, полученный из настроек (`settings.json`) или имеющее значение по умолчанию `''`.

**Тип**: `str`

**Значение по умолчанию**: `''`


### `__copyright__`

**Описание**: Авторские права проекта, полученный из настроек (`settings.json`) или имеющее значение по умолчанию `''`.

**Тип**: `str`

**Значение по умолчанию**: `''`

### `__cofee__`

**Описание**: Ссылка для поддержки разработчика. Получена из настроек (`settings.json`) или имеет значение по умолчанию.

**Тип**: `str`

**Значение по умолчанию**: "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69"
