# Задача No 560
## Отношение наименьшего общего кратного к наибольшему общему делителю
## Условия
Пусть lcm(a, b) обозначает наименьшее общее кратное a и b, а gcd(a, b) обозначает наибольший общий делитель a и b.
Для данного целого числа n пусть $M(n) = \frac{\text{lcm}(1, n) + \text{lcm}(2, n) + \dots + \text{lcm}(n, n)}{\text{gcd}(1, n) + \text{gcd}(2, n) + \dots + \text{gcd}(n, n)}$.

Например, M(10) = 438/26.

Пусть $S(n) = \sum_{k=1}^{n} M(k)$.

Известно, что S(10) = 15593/72.

Найдите $S(10^8)$, дайте свой ответ в виде строки "p/q" в несократимой форме.
