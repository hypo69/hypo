# Документация для `kualastyle_locators.json`

## Обзор

Файл `kualastyle_locators.json` содержит JSON-структуру, описывающую локаторы элементов веб-страницы для различных сценариев взаимодействия. Эти локаторы используются для автоматизации тестирования и сбора данных. Файл определяет различные элементы, такие как меню, категории товаров, информацию о продуктах и т.д.

## Основные секции

### `infinity_scroll`
   
**Описание**: Определяет, используется ли бесконечная прокрутка на странице.

**Тип**: `boolean`

**Значение**:
- `false`: Бесконечная прокрутка не используется.

### `checkboxes_for_categories`

**Описание**: Определяет, используются ли чекбоксы для выбора категорий.

**Тип**: `boolean`

**Значение**:
- `false`: Чекбоксы для выбора категорий не используются.

### `close_pop_up_locator`

**Описание**: Локатор для кнопки закрытия всплывающего окна.

**Тип**: `object`

**Свойства**:
- `attribute` (`null`): Атрибут, используемый для локатора. В данном случае не используется.
- `by` (`XPATH`): Метод поиска элемента.
- `selector` (`string`): XPATH-селектор для поиска кнопки закрытия.
- `selector tmp` (`string`): Альтернативный XPATH-селектор.
- `timeout` (`int`): Тайм-аут ожидания элемента.
- `timeout_for_event` (`string`): Условие для ожидания элемента.
- `event` (`string`): Событие для локатора.
- `if_list` (`string`): Определение, какой элемент из списка использовать.
- `use_mouse` (`boolean`): Определяет, использовать ли мышь для взаимодействия.
