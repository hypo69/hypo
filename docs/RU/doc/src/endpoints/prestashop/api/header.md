# Модуль hypotez/src/endpoints/prestashop/api/header.py

## Обзор

Этот модуль содержит вспомогательные функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и получения информации о проекте из файла `README.MD`.  Он также предоставляет переменные, содержащие информацию о проекте, такие как имя, версия, описание, автор и т.д.

## Функции

### `set_project_root`

**Описание**: Определяет корневую директорию проекта, начиная с директории текущего файла и идя вверх по дереву директорий. Поиск завершается, когда находится директория, содержащая один из заданных файлов-маркеров (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple, опционально): Кортеж имен файлов или директорий, по которым определяется корневая директория. По умолчанию это `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена, иначе путь к текущей директории.

**Вызывает исключения**:

- Нет.

### `Загрузка настроек`

**Описание**: Загружает настройки из файла `settings.json` в переменную `settings`.


**Возвращает**:

- `dict`: Словарь с настройками проекта, если файл найден и корректен.
- `None`: Если файл не найден или некорректный (например, невалидный JSON).

**Вызывает исключения**:

- `FileNotFoundError`: Если файл `settings.json` не найден.
- `json.JSONDecodeError`: Если содержимое файла `settings.json` не является валидным JSON.


### `Загрузка документации`

**Описание**: Загружает содержимое файла `README.MD` в переменную `doc_str`.


**Возвращает**:

- `str`: Содержимое файла `README.MD`, если файл найден и корректен.
- `None`: Если файл не найден или некорректный.

**Вызывает исключения**:

- `FileNotFoundError`: Если файл `README.MD` не найден.
- `json.JSONDecodeError`: (В данном случае не актуально, так как читается текст, а не JSON).


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта.

**Тип**: `Path`

### `__project_name__`

**Описание**: Имя проекта.  Получается из настроек (`settings.json`) или имеет значение 'hypotez' по умолчанию.

**Тип**: `str`

### `__version__`

**Описание**: Версия проекта.  Получается из настроек (`settings.json`) или пустая строка по умолчанию.

**Тип**: `str`

### `__doc__`

**Описание**: Описание проекта, содержимое `README.MD`.  Получается из файла `README.MD` или пустая строка по умолчанию.

**Тип**: `str`

### `__details__`

**Описание**: Дополнительные детали проекта.  По умолчанию пустая строка.

**Тип**: `str`

### `__author__`

**Описание**: Автор проекта.  Получается из настроек (`settings.json`) или пустая строка по умолчанию.

**Тип**: `str`

### `__copyright__`

**Описание**: Авторские права. Получается из настроек (`settings.json`) или пустая строка по умолчанию.

**Тип**: `str`

### `__cofee__`

**Описание**: Ссылка для поддержки разработчика. Получается из настроек (`settings.json`) или строка по умолчанию.

**Тип**: `str`
