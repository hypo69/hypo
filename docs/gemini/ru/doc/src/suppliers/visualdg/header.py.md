# Модуль header

## Обзор

Модуль `header.py` предназначен для определения настроек и метаданных проекта `hypotez`. Он выполняет поиск корневой директории проекта, загружает настройки из файла `settings.json` и документацию из файла `README.MD`, а также определяет основные переменные, такие как имя проекта, версия, авторские права и т. д.

## Подорбней

Этот модуль выполняет следующие задачи:

1.  Определяет корневую директорию проекта, начиная с текущей директории файла, и останавливается на первой директории, содержащей файлы-маркеры (`__root__`, `.git`).
2.  Загружает настройки из файла `settings.json`, расположенного в корневой директории проекта.
3.  Считывает содержимое файла `README.MD` для получения основной документации проекта.
4.  Определяет основные переменные проекта, такие как имя, версия, автор, авторские права и сообщение для поддержки разработчика.

Этот модуль важен для инициализации основных параметров проекта и обеспечения доступа к ним из других модулей.

**Описание**: Находит корневой каталог проекта, начиная с каталога текущего файла, и выполняет поиск вверх, останавливаясь на первом каталоге, содержащем любой из файлов-маркеров.

*   `marker_files` (tuple): Кортеж с именами файлов или каталогов, которые идентифицируют корень проекта. По умолчанию `('__root__', '.git')`.

*   `Path`: Путь к корневому каталогу, если он найден, в противном случае - каталог, в котором расположен скрипт.
