## Ответ на Задачу No 162: Шестнадцатеричные числа, содержащие 0, 1 и A

### 1. Анализ задачи и решение
**Понимание задачи:**
* Необходимо найти количество шестнадцатеричных чисел, не превышающих 16<sup>16</sup>, которые содержат хотя бы одну цифру 0, одну цифру 1 и одну цифру A.
* Число 16<sup>16</sup>  в шестнадцатеричной системе имеет вид `10000000000000000`, то есть это 17-значное число, начинающиеся с 1 и все остальные цифры 0.
* Мы можем использовать принцип включения-исключения. Всего шестнадцатеричных чисел не более 16<sup>16</sup> (17-значные числа, где первая цифра может быть от 0 до F, а остальные от 0 до F). Это 16<sup>16</sup>.  Из этого количества нужно вычесть числа, в которых отсутствует 0, 1, или A, и прибавить те, которые не содержат два из этих символов, и снова вычесть те, которые не содержат все три.
* Удобно работать с количеством цифр в 16-ричном числе. Максимальная длина числа, не превышающая 16<sup>16</sup> равна 17.

**Решение:**

1. **Общее количество чисел:** Всего 16-ричных чисел, не превосходящих 16<sup>16</sup> равно 16<sup>16</sup>. Но, проще считать общее количество 17-значных чисел (ведущий 0 не влияет на результат, так как мы ищем количество чисел, а не их сумму) - это 16<sup>16</sup>, так как все 17 цифр, кроме первой могут быть от 0 до F,  а у нас их 16 (от 0 до F).  Ведущая цифра тоже может быть 0, в таком случае это будет число с меньшим количеством разрядов.
2.  **Числа без 0:** Количество 17-значных 16-ричных чисел, не содержащих 0, равно 15<sup>17</sup>.
3.  **Числа без 1:** Количество 17-значных 16-ричных чисел, не содержащих 1, равно 15<sup>17</sup>.
4.  **Числа без A:** Количество 17-значных 16-ричных чисел, не содержащих A, равно 15<sup>17</sup>.
5.  **Числа без 0 и 1:** Количество 17-значных 16-ричных чисел, не содержащих 0 и 1, равно 14<sup>17</sup>.
6.  **Числа без 0 и A:** Количество 17-значных 16-ричных чисел, не содержащих 0 и A, равно 14<sup>17</sup>.
7.  **Числа без 1 и A:** Количество 17-значных 16-ричных чисел, не содержащих 1 и A, равно 14<sup>17</sup>.
8.  **Числа без 0, 1 и A:** Количество 17-значных 16-ричных чисел, не содержащих 0, 1 и A, равно 13<sup>17</sup>.

Итоговый подсчет по принципу включения исключения:
`Общее количество` - (`без 0` + `без 1` + `без A`) + (`без 0 и 1` + `без 0 и A` + `без 1 и A`) - `без 0, 1 и A`

Количество нужных чисел = `16**17` - `3 * 15**17` + `3 * 14**17` - `13**17`

### 2. Алгоритм решения

1. Начать.
2. Инициализировать:
   - `total_count` = 16<sup>17</sup> (общее количество чисел)
   - `count_without_0` = 15<sup>17</sup>
   - `count_without_1` = 15<sup>17</sup>
   - `count_without_A` = 15<sup>17</sup>
   - `count_without_0_and_1` = 14<sup>17</sup>
   - `count_without_0_and_A` = 14<sup>17</sup>
   - `count_without_1_and_A` = 14<sup>17</sup>
   - `count_without_0_1_and_A` = 13<sup>17</sup>
3. Вычислить количество чисел, содержащих хотя бы один 0, один 1 и один A:
   `result` = `total_count` - (`count_without_0` + `count_without_1` + `count_without_A`) + (`count_without_0_and_1` + `count_without_0_and_A` + `count_without_1_and_A`) - `count_without_0_1_and_A`
4. Преобразовать `result` в шестнадцатеричное число в верхнем регистре.
5. Вывести шестнадцатеричное число.
6. Конец.

### 3. Реализация на Python 3.12
```python
def solve():
    total_count = 16**17
    count_without_0 = 15**17
    count_without_1 = 15**17
    count_without_A = 15**17
    count_without_0_and_1 = 14**17
    count_without_0_and_A = 14**17
    count_without_1_and_A = 14**17
    count_without_0_1_and_A = 13**17

    result = total_count - (count_without_0 + count_without_1 + count_without_A) + \
             (count_without_0_and_1 + count_without_0_and_A + count_without_1_and_A) - \
             count_without_0_1_and_A

    return hex(result)[2:].upper()

print(solve())

```

* **Вычислить Result:**  Вычисляем итоговое количество чисел по принципу включения-исключения
* **Преобразовать в Hex:** Конвертируем результат из десятичного представления в шестнадцатеричное.
* **Вывести Hex:** Выводим итоговое шестнадцатеричное число.
