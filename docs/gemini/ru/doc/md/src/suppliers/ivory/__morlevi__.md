# Модуль `hypotez/src/suppliers/ivory/__morlevi__.py`

## Обзор

Данный модуль содержит функции для работы с поставщиком Morlevi, включая авторизацию, получение данных о продуктах и категориях.

## Функции

### `login`

**Описание**: Функция осуществляет логин на сайте поставщика Morlevi.

**Параметры**:
- `supplier` (объект): Объект, представляющий поставщика, содержащий информацию о драйвере браузера, локаторах и т.д.

**Возвращает**:
- `bool`: `True`, если логин успешен, `None` в противном случае.

**Вызывает исключения**:
- `Exception`: Возникает при возникновении ошибок во время выполнения логина, включая ошибки при попытках закрыть модальные окна.


### `_login`

**Описание**: Вспомогательная функция для осуществления логина на сайте.

**Параметры**:
- `_s` (объект): Объект, представляющий поставщика.

**Возвращает**:
- `bool`: `True`, если логин успешен, `None` в противном случае.

**Вызывает исключения**:
- `Exception`: Возникает при возникновении ошибок во время попытки логина.


### `grab_product_page`

**Описание**: Функция для получения данных о конкретном продукте на сайте Morlevi.

**Параметры**:
- `s` (объект): Объект, представляющий поставщика.

**Возвращает**:
- `Product`: Объект класса `Product`, содержащий данные о продукте.

**Вызывает исключения**:
- `Exception`: Возникает при возникновении ошибок при сборе данных о продукте, в т.ч. при отсутствии цены.


### `list_products_in_category_from_pagination`

**Описание**: Функция для получения списка ссылок на все продукты в заданной категории, используя пагинацию.

**Параметры**:
- `supplier` (объект): Объект, представляющий поставщика.

**Возвращает**:
- `list`: Список ссылок на продукты в категории.  Возвращает пустой список, если товары в категории отсутствуют.

**Вызывает исключения**:
- `Exception`: Возникает при возникновении ошибок во время получения ссылок на продукты или при переходе между страницами.


### `get_list_products_in_category`

**Описание**: Функция для получения списка продуктов из категории.

**Параметры**:
- `s` (объект): Объект, представляющий поставщика.
- `scenario` (JSON):  JSON-объект с настройками.
- `presath` (PrestaShopWebServiceDict):  Словарь настроек PrestaShopWebService.

**Возвращает**:
- `list`: Список продуктов.


### `get_list_categories_from_site`

**Описание**: Функция для получения списка категорий с сайта.

**Параметры**:
- `s` (объект): Объект, представляющий поставщика.
- `scenario_file` (строка):  Имя файла со сценарием.
- `brand` (строка, опционально): Фильтр по бренду.
