# Модуль `header`

## Обзор

Модуль `header` предназначен для определения корневой директории проекта, загрузки настроек из файла `settings.json` и чтения документации из файла `README.MD`. Он также определяет основные переменные проекта, такие как имя, версию, автора и т.д.

## Подробней

Этот модуль выполняет несколько важных задач для инициализации проекта `hypotez`. Во-первых, он определяет корневую директорию проекта с помощью функции `set_project_root`, которая ищет маркерные файлы, такие как `__root__` или `.git`. Это позволяет модулю работать независимо от текущего местоположения скрипта. Во-вторых, модуль пытается загрузить настройки из файла `settings.json` и документацию из `README.MD`. Если эти файлы не найдены или содержат ошибки, соответствующие переменные остаются пустыми или получают значения по умолчанию.

## Функции

### `set_project_root`

```python
def set_project_root(marker_files: tuple = ('__root__', '.git')) -> Path:
    """
    Args:
        marker_files (tuple): Filenames or directory names to identify the project root.

    Returns:
        Path: Path to the root directory if found, otherwise the directory where the script is located.
    """
```

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла, и останавливается на первой директории, содержащей любой из маркерных файлов.

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или каталогов для идентификации корня проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена, иначе директория, где расположен скрипт.

**Примеры**:

```python
from pathlib import Path
root_path = set_project_root()
print(f"Root path: {root_path}")
```

## Переменные модуля

### `__root__`

**Описание**: Путь к корневой директории проекта.

### `settings`

**Описание**: Словарь с настройками проекта, загруженными из `settings.json`.

### `doc_str`

**Описание**: Строка с содержимым файла `README.MD`.

### `__project_name__`

**Описание**: Имя проекта, полученное из настроек или значение по умолчанию 'hypotez'.

### `__version__`

**Описание**: Версия проекта, полученная из настроек или пустая строка.

### `__doc__`

**Описание**: Документация проекта, полученная из `README.MD` или пустая строка.

### `__details__`

**Описание**: Детали проекта, в данном коде всегда пустая строка.

### `__author__`

**Описание**: Автор проекта, полученный из настроек или пустая строка.

### `__copyright__`

**Описание**: Информация об авторских правах, полученная из настроек или пустая строка.

### `__cofee__`

**Описание**: Сообщение с предложением поддержать разработчика чашкой кофе, полученное из настроек или значение по умолчанию.
