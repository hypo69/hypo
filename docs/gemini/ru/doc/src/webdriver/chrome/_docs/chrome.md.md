# Модуль `chrome`

## Обзор

Модуль `chrome.py` определяет класс `Chrome`, который является подклассом `selenium.webdriver.Chrome`. Этот класс предоставляет дополнительные функциональные возможности для настройки и конфигурирования экземпляра Chrome WebDriver. Он разработан для продвинутых сценариев использования и включает методы для поиска свободных портов, установки параметров и инициализации WebDriver.

## Классы

### `Chrome`

**Описание**: Подкласс `selenium.webdriver.Chrome`, предоставляющий дополнительные функции.

**Методы**:
- `__init__`: Инициализирует Chrome WebDriver с заданными параметрами и профилем.
- `find_free_port`: Находит свободный порт в заданном диапазоне.
- `set_options`: Устанавливает параметры Chrome WebDriver.

#### `__init__`

**Описание**: Инициализирует экземпляр `Chrome` с заданными параметрами, включая настройки User-Agent.

**Параметры**:
- `user_agent` (dict, optional): Настройки User-Agent для Chrome WebDriver. По умолчанию `None`.

**Возвращает**:
- `None`

**Вызывает исключения**:
- `WebDriverException`: Если возникла ошибка при инициализации WebDriver.
- `Exception`: Если произошла общая ошибка или сбой WebDriver.

#### `find_free_port`

**Описание**: Находит свободный порт в указанном диапазоне.

**Параметры**:
- `start_port` (int): Начальный порт диапазона.
- `end_port` (int): Конечный порт диапазона.

**Возвращает**:
- `int | None`: Свободный порт, если он найден, иначе `None`.

**Вызывает исключения**:
- `OSError`: Если порт занят.

#### `set_options`

**Описание**: Устанавливает опции запуска для Chrome WebDriver.

**Параметры**:
- `settings` (list | dict | None, optional): Настройки конфигурации для опций Chrome. По умолчанию `None`.

**Возвращает**:
- `ChromeOptions`: Объект `ChromeOptions` с заданными опциями запуска.
