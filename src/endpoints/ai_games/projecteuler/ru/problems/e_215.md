# Задача No 215
## Стены из кирпичей

## Условия

Рассмотрим стену из кирпичей, которая имеет ширину 9 единиц и высоту 3 единицы. Стена строится из кирпичей размера 2x1 и 3x1 (кирпичи размера 1x2 и 1x3 считаются вариантами тех же самых кирпичей).

Есть много способов построить такую стену, при условии, что все кирпичи выровнены, а между ними нет промежутков.

Определенный тип стены нас интересует особо. Тип стены определяется наличием *сквозных трещин*. Сквозная трещина - это горизонтальная линия, которая проходит через всю ширину стены и не пересекает ни один кирпич. Для стены шириной 9 и высотой 3, описанной выше, следующие два варианта имеют сквозные трещины:

```
+---+---+---+
|   |   |   |
+---+---+---+
|   +---+   |
+---+---+---+
|   |   |   |
+---+---+---+

+---+---+---+
|   |   +---+
+---+---+---+
|   |   |   |
+---+---+---+
|   +---+   |
+---+---+---+
```

в то время как следующие два варианта не имеют сквозных трещин:

```
+---+---+---+
|   |   |   |
+---+---+---+
|   +---+---+
+---+---+---+
|   |   |   |
+---+---+---+

+---+---+---+
|   |   +---+
+---+---+---+
|   +---+   |
+---+---+---+
|   |   |   |
+---+---+---+
```

Пусть $M(n)$ будет количеством способов построить стену шириной $n$ и высотой 1, так чтобы не было сквозных трещин.

Пусть $C(w,h)$ будет количеством способов построить стену шириной $w$ и высотой $h$ так, чтобы не было сквозных трещин.

Вы можете проверить, что $M(3) = 1$, $M(4) = 1$, $M(5) = 2$, $M(6) = 3$, $M(7) = 5$ и $M(8) = 8$.
Также $C(3,2) = 3$, $C(9,3) = 8$.
Также известно, что $C(9,4) = 100$.

Найдите $C(32, 10)$
