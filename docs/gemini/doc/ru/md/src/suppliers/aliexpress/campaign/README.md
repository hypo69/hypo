# Модуль создания рекламной кампании на AliExpress

## Обзор

Данный модуль описывает процесс создания рекламной кампании на платформе AliExpress.  Он разбивает весь процесс на последовательные шаги, начиная с инициализации данных (имя кампании, язык, валюта) и заканчивая публикацией готовой кампании.

## Этапы создания кампании

Следующая диаграмма описывает этапы создания рекламной кампании:

```
+-------------------------+
| Start                   |
| Создание рекламной      |
| кампании                |
+-----------+-------------+
            |
            v
+-----------+---------------+
| Initialize Campaign Name, |
| Language, and Currency    |
+-----------+---------------+
            |
            v
+-----------+-------------+
| Create Campaign and     |
| Category Directories    |
+-----------+-------------+
            |
            v
+-----------+-----------------+\n
| Save Campaign Configuration |
+-----------+-----------------+\n
            |
            v
+-----------+-------------+
| Collect Product Data    |
+-----------+-------------+
            |
            v
+-----------+-------------+
| Save Product Data       |
+-----------+-------------+
            |
            v
+-----------+------------------+
| Create Promotional Materials |
+-----------+------------------+
            |
            v
+-----------+-------------+
| Review Campaign         |
+-----------+-------------+
            |
            v
+-----------+-------------+
| Is Campaign Ready?      |
+-----------+-------------+
   | Yes / No
   v      v
+-----------+-------------+
| Publish Campaign        |
+-----------+-------------+
   |
   v
+-----------+-------------+
| End                     |
| Создание рекламной      |
| кампании                |
+-------------------------+
```

## Функции

### `initialize_campaign_data`

**Описание**: Инициализирует данные кампании (имя, язык, валюта).

**Параметры**:

- `campaign_name` (str): Название кампании.
- `language` (str): Язык кампании.
- `currency` (str): Валюта кампании.

**Возвращает**:

- `dict`: Словарь с инициализированными данными кампании, или `None`, если произошла ошибка.

**Вызывает исключения**:

- `ValueError`: Если предоставлены некорректные данные.


### `create_campaign_directories`

**Описание**: Создает директории для кампании и категорий.

**Параметры**:

- `campaign_name` (str): Название кампании.


**Возвращает**:

- `bool`: True, если директории успешно созданы, иначе False.


**Вызывает исключения**:

- `IOError`: Если возникла ошибка при создании директорий.

### `save_campaign_config`

**Описание**: Сохраняет конфигурацию кампании.

**Параметры**:

- `config_data` (dict): Данные конфигурации кампании.

**Возвращает**:

- `bool`: True, если сохранение прошло успешно, иначе False.

**Вызывает исключения**:

- `IOError`: Если возникла ошибка при сохранении данных.



### `collect_product_data`

**Описание**: Сбор данных о продуктах для кампании.

**Параметры**:

- `campaign_id` (int): Идентификатор кампании.


**Возвращает**:

- `list`: Список данных о продуктах, или `None`, если данные не собраны.


**Вызывает исключения**:

- `APIError`: Если возникла ошибка при взаимодействии с API.

### `save_product_data`

**Описание**: Сохранение данных о продуктах в хранилище.

**Параметры**:

- `product_data` (list): Список данных о продуктах.


**Возвращает**:

- `bool`: True, если данные успешно сохранены, иначе False.


**Вызывает исключения**:

- `StorageError`: Если возникла ошибка при сохранении данных.

### `create_promotional_materials`

**Описание**: Создание рекламных материалов.

**Параметры**:

- `product_data` (list): Список данных о продуктах.


**Возвращает**:

- `bool`: True, если материалы успешно созданы, иначе False.


**Вызывает исключения**:

- `MaterialCreationError`: Если возникла ошибка при создании материалов.

### `review_campaign`

**Описание**: Проверка кампании.

**Параметры**:

- `campaign_id` (int): Идентификатор кампании.

**Возвращает**:

- `bool`: True, если кампания готова к публикации, иначе False.

**Вызывает исключения**:

- `ReviewError`: Если произошла ошибка при проверке.


### `is_campaign_ready`


**Описание**: Проверка готовности кампании к публикации.

**Параметры**:

- `campaign_id` (int): Идентификатор кампании.

**Возвращает**:

- `bool`: True, если кампания готова, иначе False.

**Вызывает исключения**:

- `ValidationError`: Если кампания не соответствует требованиям.


### `publish_campaign`

**Описание**: Публикация кампании.

**Параметры**:

- `campaign_id` (int): Идентификатор кампании.


**Возвращает**:

- `bool`: True, если кампания опубликована, иначе False.

**Вызывает исключения**:

- `PublishingError`: Если возникла ошибка при публикации.
