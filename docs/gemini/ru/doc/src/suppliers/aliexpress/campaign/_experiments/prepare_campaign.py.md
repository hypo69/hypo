# Модуль `prepare_campaign.py`

## Обзор

Модуль `prepare_campaign.py` предназначен для подготовки и запуска рекламных кампаний на AliExpress. Он включает в себя логику для проверки существования рекламной кампании и, если она не найдена, создает новую.

## Подробней

Модуль используется для автоматизации процесса создания и запуска рекламных кампаний на платформе AliExpress. Он проверяет, существует ли уже кампания с заданным именем, и если нет, то создает новую, используя параметры, такие как язык и валюта.
Анализ предоставленного кода показывает, что модуль является частью более крупной системы, отвечающей за управление рекламными кампаниями поставщиков на AliExpress.
Этот код служит отправной точкой для запуска процесса создания или обновления рекламной кампании, используя функциональность, предоставляемую модулем `process_campaign`.

## Переменные

- `locales (dict)`: Словарь, содержащий соответствия между языками и валютами. Например, для английского языка (`EN`) указана валюта `USD`, для иврита (`HE`) и русского (`RU`) - `ILS`.
- `language (str)`: Язык, используемый в рекламной кампании. По умолчанию установлен как `'EN'`.
- `currency (str)`: Валюта, используемая в рекламной кампании. По умолчанию установлена как `'USD'`.
- `campaign_name (str)`: Название рекламной кампании. По умолчанию установлено как `'brands'`.
