# Задача No 255
## Последовательности округленных квадратных корней
## Условия

Рассмотрим следующую последовательность:
   n<sub>0</sub> = 12345
   n<sub>k+1</sub> = ⌊√(n<sub>k</sub>)⌋

Пусть n<sub>0</sub> = n и n<sub>m</sub> = 1. Определим m как *индекс округленного квадратного корня* для n.

Например, для n = 6561, последовательность такова: 6561, 81, 9, 3, 1. Следовательно, индекс округленного квадратного корня для 6561 равен 4.
Аналогично, для n = 2, индекс округленного квадратного корня равен 1.

Сумма всех чисел n, 1 ≤ n < 10<sup>100</sup>, для которых индекс округленного квадратного корня равен 15, равна s.

Найдите последние 10 цифр s.
