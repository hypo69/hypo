**Описание:**

Этот модуль определяет набор пользовательских исключений для работы с API AliExpress.  Он позволяет структурировать и обрабатывать ошибки, возникающие при взаимодействии с этим API, предоставляя более информативные сообщения об ошибках.

**Классы исключений:**

* **`AliexpressException`:**
    * **Базовый класс:** Общий базовый класс для всех исключений, связанных с API AliExpress. Принимает аргумент `reason` для объяснения причины ошибки.

* **`InvalidArgumentException`:**
    * **Описание:** Подкласс `AliexpressException`. Возникает, когда переданные аргументы некорректны.

* **`ProductIdNotFoundException`:**
    * **Описание:** Подкласс `AliexpressException`. Возникает, если продукт с заданным ID не найден.

* **`ApiRequestException`:**
    * **Описание:** Подкласс `AliexpressException`. Возникает, если запрос к API AliExpress завершается ошибкой.

* **`ApiRequestResponseException`:**
    * **Описание:** Подкласс `AliexpressException`. Возникает, если ответ от API AliExpress некорректен.

* **`ProductsNotFoudException`:**
    * **Описание:** Подкласс `AliexpressException`. Возникает, если по запросу не найдено ни одного продукта.

* **`CategoriesNotFoudException`:**
    * **Описание:** Подкласс `AliexpressException`. Возникает, если по запросу не найдено ни одной категории.

* **`InvalidTrackingIdException`:**
    * **Описание:** Подкласс `AliexpressException`. Возникает, если идентификатор отслеживания недействителен или отсутствует.
