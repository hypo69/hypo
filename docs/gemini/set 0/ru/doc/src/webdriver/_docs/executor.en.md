## Основные функции

### Обработка локеров

Класс `ExecuteLocator` предоставляет методы для работы с локерами, описывающими элементы на веб-странице.  Локатор задается в виде словаря, содержащего различные параметры для поиска и взаимодействия с элементами.

#### `execute_locator(self, locator: dict, message: str = None, typing_speed: float = 0, continue_on_error: bool = True)`

Выполняет действия, заданные в локаторе.

**Параметры:**

- `locator` (dict): Словарь с параметрами для действий.
- `message` (str, опционально): Сообщение для отправки, если необходимо.
- `typing_speed` (float, опционально): Скорость ввода для отправки сообщений. По умолчанию 0.
- `continue_on_error` (bool, опционально): Флаг, указывающий на продолжение выполнения, если произошла ошибка. По умолчанию `True`.

**Возвращает:**

- `Union[str, list, dict, WebElement, bool]`: Результат выполнения действий (строка, список, словарь, элемент веб-страницы или `bool`).

#### `get_webelement_by_locator(self, locator: dict | SimpleNamespace, message: str = None) -> WebElement | List[WebElement] | bool`

Возвращает элемент или список элементов, соответствующих заданному локатору.

**Параметры:**

- `locator` (dict | SimpleNamespace): Локатор, определяющий элемент.
- `message` (str, опционально): Сообщение для отладки.

**Возвращает:**

- `WebElement | List[WebElement] | bool`: Найденный элемент или список элементов или `False` в случае неудачи.


#### `get_attribute_by_locator(self, locator: dict | SimpleNamespace, message: str = None) -> str | list | dict | bool`

Возвращает атрибут элемента, найденного по локатору.

**Параметры:**

- `locator` (dict | SimpleNamespace): Локатор для поиска элемента.
- `message` (str, опционально): Сообщение для отладки.

**Возвращает:**

- `str | list | dict | bool`: Значение атрибута.

#### `send_message(self, locator: dict | SimpleNamespace, message: str, typing_speed: float, continue_on_error:bool) -> bool`

Отправляет текст сообщению.

**Параметры:**

- `locator` (dict | SimpleNamespace): Локатор для отправки сообщения.
- `message` (str): Текст сообщения.
- `typing_speed` (float): Скорость ввода.
- `continue_on_error` (bool): Флаг, указывающий на продолжение выполнения при ошибке.

**Возвращает:**

- `bool`: `True` если успешно, `False` в противном случае.

## Обработка ошибок

Класс `ExecuteLocator` использует блоки `try...except` для обработки различных исключений, таких как `NoSuchElementException` и `TimeoutException`, возникающих при поиске элементов.  Исключения логгируются.
