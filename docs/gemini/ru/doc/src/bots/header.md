# Модуль hypotez/src/bots/header.py

## Обзор

Этот модуль содержит вспомогательные функции для работы с проектом hypotez, в частности, для определения корневой директории проекта и загрузки настроек из файла `settings.json`. Он также обрабатывает случай, когда файл настроек отсутствует или поврежден.  В случае неудачи, модуль использует значения по умолчанию.  Модуль загружает также описание проекта из файла `README.MD`.

## Функции

### `set_project_root`

**Описание**: Функция находит корневую директорию проекта, начиная от текущей директории и идя вверх по дереву директорий, пока не найдет директорию, содержащую один из файлов-маркеров (`pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, которые используются для определения корневой директории проекта. По умолчанию заданы: `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена, в противном случае путь к директории, где находится данный скрипт.

**Вызывает исключения**:

- Нет

## Переменные

### `__root__`

**Описание**: Переменная, хранящая путь к корневой директории проекта. Инициализируется функцией `set_project_root()`.


### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженные из файла `settings.json` в корневой директории проекта.

**Тип данных**: `dict`

**Значение по умолчанию**: `None`

### `doc_str`

**Описание**: Строка, содержащая описание проекта, загруженная из файла `README.MD` в корневой директории проекта.

**Тип данных**: `str`

**Значение по умолчанию**: `None`


### `__project_name__`

**Описание**: Имя проекта.  Получается из настроек (`settings`), или имеет значение по умолчанию `'hypotez'`.

**Тип данных**: `str`


### `__version__`

**Описание**: Версия проекта. Получается из настроек (`settings`), или имеет значение по умолчанию `''`.

**Тип данных**: `str`

### `__doc__`

**Описание**: Документация проекта. Получается из файла `README.MD` (`doc_str`), или имеет значение по умолчанию `''`.

**Тип данных**: `str`

### `__details__`

**Описание**: Дополнительные детали проекта. Значение по умолчанию `''`.

**Тип данных**: `str`


### `__author__`

**Описание**: Автор проекта. Получается из настроек (`settings`), или имеет значение по умолчанию `''`.

**Тип данных**: `str`


### `__copyright__`

**Описание**: Авторские права проекта. Получается из настроек (`settings`), или имеет значение по умолчанию `''`.

**Тип данных**: `str`


### `__cofee__`

**Описание**: Ссылка на поддержку разработчика. Получается из настроек (`settings`), или имеет значение по умолчанию "Ссылка на поддержку разработчика".

**Тип данных**: `str`
