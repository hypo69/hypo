# Модуль для определения пользовательских исключений для GPT4Free
==============================================================

Модуль содержит определения пользовательских исключений, используемых в библиотеке `gpt4free`. Эти исключения позволяют более точно обрабатывать ошибки, возникающие при работе с различными провайдерами и моделями.

## Обзор

В данном модуле определены различные классы исключений, которые могут возникать в процессе работы с библиотекой `gpt4free`. Каждое исключение предназначено для конкретной ситуации, что позволяет более точно идентифицировать и обрабатывать ошибки.

## Классы

### `ProviderNotFoundError`

**Описание**:
Исключение, возникающее, когда не удается найти запрошенного провайдера.

### `ProviderNotWorkingError`

**Описание**:
Исключение, возникающее, когда провайдер не работает или недоступен.

### `StreamNotSupportedError`

**Описание**:
Исключение, возникающее, когда потоковая передача не поддерживается.

### `ModelNotFoundError`

**Описание**:
Исключение, возникающее, когда не удается найти запрошенную модель.

### `ModelNotAllowedError`

**Описание**:
Исключение, возникающее, когда использование модели не разрешено.

### `RetryProviderError`

**Описание**:
Исключение, возникающее при необходимости повторной попытки использования провайдера.

### `RetryNoProviderError`

**Описание**:
Исключение, возникающее при отсутствии доступных провайдеров для повторной попытки.

### `VersionNotFoundError`

**Описание**:
Исключение, возникающее, когда не удается найти запрошенную версию.

### `ModelNotSupportedError`

**Описание**:
Исключение, возникающее, когда модель не поддерживается.

### `MissingRequirementsError`

**Описание**:
Исключение, возникающее при отсутствии необходимых зависимостей.

### `NestAsyncioError`

**Описание**:
Исключение, возникающее при проблемах с асинхронностью (вложенные циклы событий).
**Наследует**: `MissingRequirementsError`

### `MissingAuthError`

**Описание**:
Исключение, возникающее при отсутствии необходимой аутентификации.

### `PaymentRequiredError`

**Описание**:
Исключение, возникающее, когда требуется оплата за использование сервиса.

### `NoMediaResponseError`

**Описание**:
Исключение, возникающее, когда отсутствует ответ с медиа-контентом.

### `ResponseError`

**Описание**:
Исключение, возникающее при общей ошибке ответа.

### `ResponseStatusError`

**Описание**:
Исключение, возникающее при ошибке в статусе ответа.

### `RateLimitError`

**Описание**:
Исключение, возникающее при превышении лимита запросов (ограничение скорости).
**Наследует**: `ResponseStatusError`

### `NoValidHarFileError`

**Описание**:
Исключение, возникающее при отсутствии валидного HAR-файла.

### `TimeoutError`

**Описание**:
Исключение, возникающее при истечении времени ожидания во время API-запросов.

### `ConversationLimitError`

**Описание**:
Исключение, возникающее при превышении лимита разговоров во время API-запросов к AI-сервису.
