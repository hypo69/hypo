# Модуль _experiments

## Обзор

Модуль `_experiments` предназначен для проведения экспериментов в рамках работы с поставщиком eBay. Он содержит различные импорты и настройки путей для облегчения экспериментов и отладки кода.

## Подробней

Этот модуль, вероятно, используется для тестирования различных функций и подходов к парсингу или обработке данных с eBay. Расположение файла в директории `src/suppliers/ebay/_experiments` указывает на то, что он не является частью основного функционала, а служит площадкой для экспериментов.

## Импорты

В данном модуле выполняются следующие импорты:

- `sys`: Используется для работы с системными параметрами и функциями.
- `os`: Используется для взаимодействия с операционной системой, например, для работы с путями к файлам.
- `pathlib.Path`: Используется для представления путей к файлам и директориям в объектно-ориентированном стиле.
- `json`: Используется для работы с данными в формате JSON.
- `re`: Используется для работы с регулярными выражениями.
- `src.webdriver.driver.Driver`: Класс для управления веб-драйвером (например, Selenium).
- `src.suppliers.Supplier`: Класс, представляющий поставщика (в данном случае, вероятно, eBay).
- `src.product.Product`, `src.product.ProductFields`: Классы и перечисления, связанные с представлением продукта.
- `src.category.Category`: Класс, представляющий категорию продукта.
- `src.utils.StringFormatter`, `src.utils.StringNormalizer`: Классы для форматирования и нормализации строк.
- `src.utils.printer.pprint`: Функция для "красивой" печати данных.
- `src.endpoints.PrestaShop.Product`: Класс для взаимодействия с PrestaShop API.
- `src.endpoints.PrestaShop`: Модуль для взаимодействия с PrestaShop.

## Переменные

- `dir_root (Path)`: Определяет корневую директорию проекта ("hypotez") и добавляет её в `sys.path`, чтобы обеспечить возможность импорта модулей из проекта.
- `dir_src (Path)`: Определяет путь к директории `src` внутри проекта и добавляет её в `sys.path`.

## Функции

### `start_supplier`

```python
def start_supplier(supplier_prefix: str = 'aliexpress', locale: str = 'en') -> Supplier:
    """ Старт поставщика """
```

**Описание**: Функция для инициализации и запуска поставщика.

**Как работает функция**:
Функция принимает префикс поставщика и локаль, создает словарь параметров и инициализирует объект класса `Supplier` с этими параметрами.

**Параметры**:
- `supplier_prefix` (str, optional): Префикс поставщика. По умолчанию 'aliexpress'.
- `locale` (str, optional): Локаль. По умолчанию 'en'.

**Возвращает**:
- `Supplier`: Объект класса `Supplier`.

**Примеры**:

```python
supplier = start_supplier(supplier_prefix='ebay', locale='ru')
```
```python
supplier = start_supplier()
```
```python
supplier = start_supplier(locale='de')
```
```python
supplier = start_supplier(supplier_prefix='amazon', locale='en')
