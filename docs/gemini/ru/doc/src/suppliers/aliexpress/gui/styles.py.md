# Модуль `styles.py`

## Обзор

Модуль `styles.py` содержит функции для стилизации элементов графического интерфейса (GUI) в приложении, использующем библиотеку PyQt6. В частности, он предоставляет функцию `set_fixed_size` для установки фиксированного размера виджетов. Расположен в `src/suppliers/aliexpress/gui`, что указывает на его использование для стилизации интерфейса, связанного с поставщиком AliExpress.

## Подробней

Модуль предоставляет инструменты для стандартизации внешнего вида элементов интерфейса, что облегчает разработку и поддержку единообразного стиля приложения. Функция `set_fixed_size` позволяет задать конкретные размеры виджетов, что может быть полезно для обеспечения консистентности отображения на различных устройствах и разрешениях.

## Функции

### `set_fixed_size`

```python
def set_fixed_size(widget: QtWidgets.QWidget, width: int, height: int):
    """Set a fixed size for a given widget"""
```

**Описание**: Устанавливает фиксированный размер для указанного виджета.

**Параметры**:

- `widget` (QtWidgets.QWidget): Виджет, для которого необходимо установить размер.
- `width` (int): Ширина виджета.
- `height` (int): Высота виджета.

**Возвращает**:

- `None`: Функция ничего не возвращает.

**Вызывает исключения**:

- Отсутствуют явные исключения.

**Примеры**:

```python
from PyQt6 import QtWidgets

widget = QtWidgets.QPushButton('Кнопка')
set_fixed_size(widget, 100, 30)
```
```python
from PyQt6 import QtWidgets

label = QtWidgets.QLabel('Надпись')
set_fixed_size(label, 150, 25)
