# Модуль `hypotez/src/endpoints/advertisement/facebook/start_posting.py`

## Обзор

Этот модуль отвечает за отправку рекламных объявлений в группы Facebook. Он использует веб-драйвер для авторизации и взаимодействия с платформой Facebook, а также предопределенные JSON-файлы для конфигурации кампаний. Модуль включает в себя цикл для периодической запуска кампаний.

## Переменные

### `MODE`

**Тип**: `str`

**Описание**: Строковое значение, определяющее режим работы (`dev`).

### `filenames`

**Тип**: `list[str]`

**Описание**: Список путей к JSON-файлам, содержащим информацию о целевых группах Facebook.

### `excluded_filenames`

**Тип**: `list[str]`

**Описание**: Список исключаемых путей к JSON-файлам.

### `campaigns`

**Тип**: `list`

**Описание**: Список названий кампаний, которые необходимо запустить.

**Описание**: Класс, представляющий веб-драйвер Chrome (не определён в данном файле, предполагается импорт из `src.webdriver.driver`).

**Описание**: Класс для запуска рекламных кампаний в Facebook (не определён в данном файле, предполагается импорт из `src.endpoints.advertisement.facebook`).


## Функции

### `run_campaigns`

**Описание**: Функция запуска рекламных кампаний.

**Параметры**:

- `campaigns` (`list`): Список названий кампаний.
- `group_file_paths` (`list[str]`): Список путей к JSON-файлам с информацией о группах.

**Возвращает**:

- `None`

**Вызывает исключения**:

- (Не определены)


## Инициализация

### Создание драйвера

`d = Driver(Chrome)`

**Описание**: Создаёт экземпляр класса `Driver` с типом `Chrome`.

`d.get_url(r"https://facebook.com")`

**Описание**: Переход на страницу `https://facebook.com` с помощью драйвера `d`.

### Создание `FacebookPromoter`

`promoter:FacebookPromoter = FacebookPromoter(d, group_file_paths=filenames, no_video = True)`

**Описание**: Инициализация объекта класса `FacebookPromoter` с передачей веб-драйвера `d`, списком путей к файлам с группами `filenames` и флагом `no_video`.

## Цикл запуска кампаний

```python
try:
    while True:
        promoter.run_campaigns(campaigns = copy.copy(campaigns), group_file_paths = filenames)
        print(f"Going sleep {time.localtime}")
        time.sleep(180)
        ...
except KeyboardInterrupt:
    logger.info("Campaign promotion interrupted.")
```

**Описание**:  Бесконечный цикл, запускающий кампании с помощью `promoter.run_campaigns`. После каждого цикла производится пауза в 180 секунд. Цикл прерывается при нажатии Ctrl+C.  Обработка `KeyboardInterrupt` позволяет корректно завершить работу, записывая сообщение в лог.


**Примечание**: Код содержит `...`, указывающий на отсутствие детальной реализации внутренних частей цикла.
