# Модуль initial_revision

## Обзор

Модуль содержит определения для миграции базы данных, используемой Alembic. Он включает в себя функции `upgrade` и `downgrade`, которые определяют изменения, вносимые в базу данных при обновлении и откате версии соответственно.

## Подробней

Этот модуль является частью системы миграции базы данных Alembic, используемой для управления изменениями схемы базы данных. Он содержит функции для обновления (`upgrade`) и отката (`downgrade`) базы данных до определенной версии. В данном случае, это initial revision, то есть начальная версия базы данных.

## Переменные

- `revision` (str): Идентификатор текущей ревизии (версии) миграции. В данном случае: `'47f559ec82bb'`.
- `down_revision` (Union[str, None]): Идентификатор предыдущей ревизии. Для initial revision это `None`.
- `branch_labels` (Union[str, Sequence[str], None]): Метки веток для миграции. В данном случае: `None`.
- `depends_on` (Union[str, Sequence[str], None]): Список ревизий, от которых зависит данная миграция. В данном случае: `None`.
