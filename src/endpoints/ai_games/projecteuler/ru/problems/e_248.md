# Задача No 248
## Числа генерации
## Условия
Пусть задано множество простых чисел, `S`, и определим `f(n)` как количество способов, которыми n может быть записано в виде произведения элементов из `S` (порядок множителей не имеет значения).

Например, если `S` = {2, 3}, то:

*   `f(1) = 1` (произведение 0 элементов из `S`)
*   `f(2) = 1`
*   `f(3) = 1`
*   `f(4) = 1` (2 * 2)
*   `f(5) = 0`
*   `f(6) = 2` (2 * 3 и 3 * 2)
*   `f(8) = 1`
*   `f(9) = 1`
*   `f(10) = 0`
*   `f(12) = 2` (2 * 2 * 3 и 2 * 3 * 2 и 3 * 2 * 2)

Определим `g(n)` как следующее:

*   `g(1) = 1`
*   `g(n)` - минимальное число `k` такое, что `f(k) = n`.

Например, для `S = {2, 3}`:

*   `g(1) = 1`
*   `g(2) = 6`

Обозначим `S(n)` как множество первых n простых чисел.

Найдите `g(10^12)` для `S = S(100)`.

**Другими словами, требуется найти наименьшее число, которое может быть разложено на произведение первых 100 простых чисел ровно `10^12` способами.**
