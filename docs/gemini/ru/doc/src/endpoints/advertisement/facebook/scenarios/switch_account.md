# Модуль `hypotez/src/endpoints/advertisement/facebook/scenarios/switch_account.py`

## Обзор

Этот модуль содержит функцию `switch_account`, предназначенную для переключения учетных записей в приложении Facebook. Функция взаимодействует с драйвером, используя предопределенные локаторы.


## Функции

### `switch_account`

**Описание**: Функция переключает учетные записи в приложении Facebook, если такая кнопка присутствует.

**Параметры**:

- `driver` (Driver): Объект драйвера, предоставляющий методы для взаимодействия с веб-драйвером.


**Возвращает**:

-  Функция ничего не возвращает.


**Вызывает исключения**:

-  Возможные исключения, связанные с выполнением действий в браузере, будут обрабатываться внутри функции `driver.execute_locator`.
