## АНАЛИЗ КОДА: Игра LETTER (Угадай букву)

### <алгоритм>

1. **Начало игры:**
   - Выводится приветствие и правила игры.
   - Генерируется случайная буква (secret_letter) в диапазоне A-Z.
   - Инициализируется счётчик попыток (attempts) значением 0.
     - *Пример:* `secret_letter` = "K", `attempts` = 0

2. **Игровой цикл:**
   - Запрашивается ввод от игрока (guess_letter).
   - Увеличивается счётчик попыток `attempts` на 1.
     - *Пример:* `guess_letter` = "M", `attempts` = 1
   - Проверяется корректность ввода:
     - Если ввод не является одной буквой верхнего регистра (A-Z), выводится сообщение об ошибке.
       - *Пример:* ввод "123" -> ошибка, ввод "abc" -> ошибка, ввод "M" -> продолжение.
   - Сравнивается введенная буква с загаданной `secret_letter`:
     - Если `guess_letter` < `secret_letter`, выводится "Загаданная буква выше по алфавиту."
       - *Пример:* `guess_letter` = "M", `secret_letter` = "P"
     - Если `guess_letter` > `secret_letter`, выводится "Загаданная буква ниже по алфавиту."
       - *Пример:* `guess_letter` = "M", `secret_letter` = "K"
     - Если `guess_letter` == `secret_letter`, переход к завершению игры.
       - *Пример:* `guess_letter` = "K", `secret_letter` = "K"

3. **Завершение игры:**
   - Выводится поздравительное сообщение с количеством попыток.
   - Спрашивается, хочет ли игрок играть снова (play_again):
     - Если `play_again` == "да", начинается новый игровой цикл с пункта 1.
     - Если `play_again` == "нет", выводится сообщение "Спасибо за игру!" и программа завершается.

   **Пример потока данных:**
   ```
   Start --> Инициализация(Приветствие, выбор буквы, attempts=0)
   Инициализация --> Input(guess_letter)
   Input --> Проверка ввода
   Проверка ввода -->  Увеличение attempts
   Увеличение attempts --> Сравнение букв(guess_letter, secret_letter)
   Сравнение букв --guess_letter < secret_letter--> Подсказка("Загаданная буква выше")
   Подсказка --> Input
   Сравнение букв --guess_letter > secret_letter--> Подсказка("Загаданная буква ниже")
   Подсказка --> Input
   Сравнение букв --guess_letter == secret_letter--> Завершение игры(поздравление, кол-во попыток)
   Завершение игры --> Играть снова?(да/нет)
   Играть снова?(да) --> Инициализация
   Играть снова?(нет) --> End
   ```

**Описание зависимостей в `mermaid`:**

-   `Start`: Начало игры.
-   `Initialize`: Инициализация игры, включает в себя приветствие, генерацию случайной буквы (`secret_letter`) и сброс счетчика попыток (`attempts` = 0).
-   `InputLetter`: Запрос ввода буквы от пользователя (`guess_letter`).
-   `ValidateInput`: Проверка корректности введенной буквы, должна быть одной буквой в верхнем регистре.
-    `InputError`: Выводится сообщение об ошибке, если ввод некорректный.
-   `IncrementAttempts`: Увеличение счетчика попыток (`attempts`) на 1.
-   `CompareLetters`: Сравнение введенной буквы (`guess_letter`) с загаданной (`secret_letter`).
-   `HintHigher`: Вывод подсказки "Загаданная буква выше", если `guess_letter` меньше `secret_letter`.
-   `HintLower`: Вывод подсказки "Загаданная буква ниже", если `guess_letter` больше `secret_letter`.
-   `GameEnd`: Завершение игры при угадывании, включает в себя поздравление и вывод количества попыток (`attempts`).
-    `PlayAgain`: Запрос на повторную игру.
-    `End`: Завершение игры.
