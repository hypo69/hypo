# Модуль hypotez/src/goog/gtranslater/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и получения документации из файла `README.MD`.  Он также устанавливает корневую директорию в переменную `sys.path` для правильной работы импорта модулей.

## Функции

### `set_project_root`

**Описание**: Определяет корневую директорию проекта, начиная с текущей директории и идя вверх по дереву директорий, пока не найдет директорию, содержащую указанные файлы-маркеры (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж имён файлов или директорий, по которым будет определяться корневая директория проекта. По умолчанию использует (`'pyproject.toml'`, `'requirements.txt'`, `.git`).

**Возвращает**:

- `Path`: Объект `Path` содержащий путь к корневой директории проекта. Если корневая директория не найдена, возвращает путь к директории, в которой находится текущий файл.

**Вызывает исключения**:

- Не вызывает исключения.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root`.

**Тип**: `Path`

### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженный из файла `settings.json`.

**Тип**: `dict` или `None`.

### `doc_str`

**Описание**: Строка, содержащая текст документации из файла `README.MD`.

**Тип**: `str` или `None`.

### `__project_name__`

**Описание**: Имя проекта, полученное из настроек проекта. По умолчанию `hypotez`.

**Тип**: `str`

### `__version__`

**Описание**: Версия проекта, полученное из настроек проекта. По умолчанию пустая строка.

**Тип**: `str`

### `__doc__`

**Описание**: Строка документации проекта.


**Тип**: `str`

### `__details__`

**Описание**: Подробная информация о проекте.

**Тип**: `str`

### `__author__`

**Описание**: Автор проекта, полученное из настроек проекта. По умолчанию пустая строка.

**Тип**: `str`

### `__copyright__`

**Описание**: Авторские права, полученное из настроек проекта. По умолчанию пустая строка.

**Тип**: `str`

### `__cofee__`

**Описание**: Ссылка на поддержку разработчика через Boosty.


**Тип**: `str`

**Примечание**: Переменные `__project_name__`, `__version__`, `__doc__`, `__details__`, `__author__`, `__copyright__`, `__cofee__` могут принимать значения `None` если соответствующие ключи не найдены в файле `settings.json` или если файл `settings.json` не найден.
