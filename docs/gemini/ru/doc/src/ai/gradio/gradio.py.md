# Модуль `src.ai.gradio.gradio`

## Обзор

Модуль `src.ai.gradio.gradio` представляет собой простой пример использования библиотеки Gradio для создания веб-интерфейса. Он содержит одну функцию `greet`, которая принимает имя в качестве параметра и возвращает приветствие. Модуль запускает веб-интерфейс с помощью Gradio.

    - [`greet`](#greet)

## Функции

### `greet`

**Описание**: Функция принимает имя в качестве параметра и возвращает приветствие.

**Параметры**:
- `name` (str): Имя пользователя.

**Возвращает**:
- `str`: Приветствие в виде строки "Hello " + name + "!".
