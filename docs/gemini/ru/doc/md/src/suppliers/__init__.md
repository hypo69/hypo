# Модуль `hypotez/src/suppliers`

## Обзор

Данный модуль содержит инструменты для работы с поставщиками данных. Он предоставляет базовый класс `Supplier` и инфраструктуру для подключения специфичных методов извлечения данных для различных поставщиков.  Класс `Supplier` является абстрактным и должен быть расширен для каждого конкретного поставщика.  Дополняющие функции для извлечения данных хранятся в подкаталогах, именованных в соответствии с префиксом поставщика (например, `amazon`, `aliexpress`).

## Классы

### `Supplier`

**Описание**: Базовый класс для работы с поставщиками.  Представляет собой абстрактный класс, который должен быть расширен для каждого конкретного поставщика.

**Методы**:

- `get_data()`: Извлекает данные от поставщика.


### `Graber`

**Описание**: Класс для извлечения данных с помощью конкретной стратегии.
**Методы**:

- `__init__(self, url: str)`: Конструктор класса.
- `get_products(self, filter: Optional[str]=None) -> list | None`: Получает список товаров.
- `close_pop_up(self)`: Закрывает всплывающие окна.

### `Context`

**Описание**: Класс для работы с контекстом извлечения данных.
