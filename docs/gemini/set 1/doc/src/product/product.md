# Модуль `hypotez/src/product/product.py`

## Обзор

Модуль `src.product` определяет поведение продукта в проекте. Он взаимодействует с сайтом, продуктом и API PrestaShop, используя данные, полученные из grabber'a.

## Классы

### `Product`

**Описание**: Класс `Product` управляет манипуляциями с продуктом. Изначально запрашивает данные с страницы продукта, а затем работает с API PrestaShop. Наследует классы `ProductFields` и `PrestaShop`.

**Методы**:

- `__init__(self, *args, **kwargs)`: Инициализирует объект `Product`.

  **Параметры**:
    - `*args`: Список аргументов переменной длины.
    - `**kwargs`: Словарь произвольных именованных аргументов.

### `Category`

**Описание**: Класс `Category`  не описан в предоставленном коде, но используется в методе `get_parent_categories`.

**Методы**:

- `get_parents(id_category: int, dept: int = 0) -> list`:  Возвращает список родительских категорий для заданной категории.  Метод реализован в другом модуле.

## Функции

### `get_parent_categories(id_category: int, dept: int = 0) -> list`

**Описание**:  Возвращает список родительских категорий для заданной категории.  Дублирует функционал функции `get_parents` из класса `Category`.

**Параметры**:
    - `id_category` (int): Идентификатор категории.
    - `dept` (int, optional): Глубина категории. По умолчанию 0.

**Возвращает**:
    - list: Список родительских категорий.

**Вызывает исключения**:
    - `TypeError`: Если `id_category` не является целым числом.
