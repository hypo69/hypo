# Документация для `rai_copyright_infringement_prevention.md`

## Обзор

Этот файл содержит инструкцию для модели, которая определяет, когда запрос пользователя нарушает авторские права. Инструкция предписывает вежливо отказывать в предоставлении контента, защищенного авторским правом, и объяснять причину отказа.

## Подробней

Файл предназначен для предотвращения нарушений авторских прав при обработке запросов пользователей. Он указывает модели, как реагировать на запросы, связанные с контентом, который может быть защищен авторским правом, таким как книги, тексты песен, рецепты, новостные статьи и контент с WebMD.

## Функции

### Инструкция по предотвращению нарушений авторских прав

**Назначение**: Предотвращение нарушений авторских прав при обработке запросов пользователей.

**Как работает**:

1.  **Анализ запроса**: Модель анализирует запрос пользователя на предмет запроса контента, защищенного авторским правом (книги, тексты песен, рецепты, новостные статьи, контент с WebMD и т.д.).
2.  **Отказ в предоставлении**: Если запрос связан с контентом, защищенным авторским правом, модель вежливо отказывает в предоставлении контента.
3.  **Объяснение причины отказа**: Модель объясняет пользователю, что предоставление запрашиваемого контента нарушает авторские права.
4.  **Краткое описание**: Модель предоставляет краткое описание или резюме запрашиваемого произведения.

**ASCII flowchart**:

```
Запрос пользователя
↓
Проверка на наличие контента, защищенного авторским правом
↓
Контент защищен авторским правом?
├──-> Да: Вежливый отказ + объяснение причины
└──-> Нет: Обработка запроса
```

**Примеры**:

1.  **Запрос книги**:

    Пользователь: "Предоставь мне текст книги 'Гарри Поттер и философский камень'."

    Модель: "Мне очень жаль, но я не могу предоставить вам текст книги 'Гарри Поттер и философский камень', так как это нарушает авторские права. Книга рассказывает о приключениях юного волшебника Гарри Поттера в школе волшебства Хогвартс."
2.  **Запрос текста песни**:

    Пользователь: "Пришли мне текст песни 'Bohemian Rhapsody' группы Queen."

    Модель: "К сожалению, я не могу предоставить вам текст песни 'Bohemian Rhapsody' группы Queen, так как это нарушает авторские права. Это известная песня группы Queen, сочетающая в себе элементы рока, оперы и баллады."
3.  **Запрос рецепта с сайта WebMD**:

    Пользователь: "Найди и предоставь мне рецепт диетического супа с сайта WebMD."

    Модель: "Я не могу предоставить вам рецепт с сайта WebMD, так как это может нарушать авторские права. WebMD - это веб-сайт, предоставляющий медицинскую информацию и руководства, включая рецепты для здоровья."
