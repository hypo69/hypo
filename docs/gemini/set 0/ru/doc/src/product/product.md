# Модуль `hypotez/src/product/product.py`

## Обзор

Модуль `src.product` определяет поведение продукта в проекте. Он взаимодействует с веб-сайтом, продуктом и API PrestaShop, используя данные, полученные из продукта.

## Классы

### `Product`

**Описание**: Класс `Product` реализует манипуляции с продуктом.  Изначально он получает данные с страницы продукта, а затем взаимодействует с API PrestaShop. Наследует классы `ProductFields` и `PrestaShop`.

**Методы**:

- `__init__`: Инициализирует объект `Product`.

    **Описание**: Инициализирует объект `Product`, принимая переменное число аргументов и произвольные ключевые аргументы.

    **Параметры**:
    - `*args`: Переменная длина списка аргументов.
    - `**kwargs`: Произвольные ключевые аргументы.


- `get_parent_categories`: Получает родительские категории для заданной категории.

    **Описание**:  Функция получает родительские категории для указанной категории по ее ID. Дублирует функцию `get_parents` из класса `Category`.

    **Параметры**:
    - `id_category` (int): ID категории.
    - `dept` (int, optional): Глубина категории. По умолчанию 0.

    **Возвращает**:
    - `list`: Список родительских категорий.

    **Вызывает исключения**:
    - `TypeError`: Если `id_category` не является целым числом.
