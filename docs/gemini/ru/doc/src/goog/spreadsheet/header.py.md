# Модуль `header.py`

## Обзор

Модуль `header.py` предназначен для настройки окружения проекта, определения корневой директории, загрузки настроек из `settings.json`, чтения документации из `README.MD` и определения основных метаданных проекта.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла, и останавливается на первой директории, содержащей маркерные файлы.

**Параметры**:
- `marker_files` (tuple, optional): Список имен файлов или каталогов для идентификации корневой директории. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если найдена. В противном случае возвращается путь к директории, где расположен скрипт.

## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта.

### `settings`

**Описание**: Словарь с настройками проекта, загруженными из файла `settings.json`. Если файл не найден или невалиден, то значение `None`.

### `doc_str`

**Описание**: Строка с содержимым файла `README.MD`. Если файл не найден или невалиден, то значение `None`.

### `__project_name__`

**Описание**: Название проекта. Значение по умолчанию `'hypotez'`, если в настройках не указано иное.

### `__version__`

**Описание**: Версия проекта. Значение по умолчанию `''`, если в настройках не указано иное.

### `__doc__`

**Описание**: Содержимое файла `README.MD`. Если `doc_str` не удалось получить, то значение `''`.

### `__details__`

**Описание**: Дополнительная информация о проекте (на данный момент не используется). Значение по умолчанию `''`.

### `__author__`

**Описание**: Имя автора проекта. Значение по умолчанию `''`, если в настройках не указано иное.

### `__copyright__`

**Описание**: Информация об авторских правах. Значение по умолчанию `''`, если в настройках не указано иное.

### `__cofee__`

**Описание**: Строка, содержащая предложение поддержать разработчика чашкой кофе. Если в настройках не указано иное, то используется значение по умолчанию: `"Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69"`.
