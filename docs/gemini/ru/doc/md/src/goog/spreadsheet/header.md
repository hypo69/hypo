# Модуль hypotez/src/goog/spreadsheet/header.py

## Обзор

Данный модуль `header.py` отвечает за инициализацию и получение основных настроек проекта, таких как корневой каталог, имя проекта, версию, описание и авторские права.  Он также загружает настройки из файла `settings.json` и документацию из `README.MD`, если они существуют.

## Функции

### `set_project_root`

**Описание**: Функция находит корневой каталог проекта, начиная от текущей директории и идя вверх по иерархии директорий. Она останавливается на первой директории, содержащей файлы-маркеры (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для определения корневой директории проекта.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена, иначе путь к директории, где находится скрипт.


**Вызывает исключения**:

- Не вызывает исключений.


## Переменные

### `__root__`

**Описание**: Переменная, хранящая путь к корневому каталогу проекта, полученный функцией `set_project_root`.

**Тип**: `Path`


### `settings`

**Описание**: Переменная, содержащая словарь настроек проекта, загруженный из файла `settings.json`.

**Тип**: `dict`


### `doc_str`

**Описание**: Переменная, содержащая строку с документацией проекта, загруженную из файла `README.MD`.

**Тип**: `str`

### `__project_name__`

**Описание**: Переменная, хранящая имя проекта.  Получается из словаря настроек `settings` или имеет значение по умолчанию `hypotez`.

**Тип**: `str`


### `__version__`

**Описание**: Переменная, хранящая версию проекта. Получается из словаря настроек `settings` или имеет значение по умолчанию `''`.

**Тип**: `str`


### `__doc__`

**Описание**: Переменная, хранящая описание проекта. Получается из переменной `doc_str` или имеет значение по умолчанию `''`.

**Тип**: `str`


### `__details__`

**Описание**: Переменная, хранящая дополнительные сведения о проекте. Имеет значение по умолчанию `''`.

**Тип**: `str`


### `__author__`

**Описание**: Переменная, хранящая имя автора проекта. Получается из словаря настроек `settings` или имеет значение по умолчанию `''`.

**Тип**: `str`


### `__copyright__`

**Описание**: Переменная, хранящая авторские права проекта. Получается из словаря настроек `settings` или имеет значение по умолчанию `''`.

**Тип**: `str`


### `__cofee__`

**Описание**: Переменная, хранящая ссылку на поддержку разработчика через платформа для сбора пожертвований. Получается из словаря настроек `settings` или имеет значение по умолчанию.

**Тип**: `str`


## Обработка исключений

Функции чтения файла настроек и документации (считывание `settings.json` и `README.MD`) содержат обработку исключений `FileNotFoundError` и `json.JSONDecodeError`. Это позволяет программе продолжить работу, даже если указанные файлы отсутствуют или содержат некорректные данные.  В случае ошибки соответствующие переменные (settings и doc_str) будут иметь значения по умолчанию.
