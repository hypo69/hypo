# Модуль hypotez/src/suppliers/morlevi/__init__.py

## Обзор

Модуль `morlevi` предоставляет инструменты для работы с поставщиком данных Morlevi.  В настоящее время содержит только импорт класса `Graber`.

## Содержимое

Этот модуль содержит конфигурацию режима работы (`MODE`) и импортирует класс `Graber` из подмодуля `graber`.


## Переменные

### `MODE`

**Описание**: Переменная, определяющая режим работы.

**Значение**:  `'dev'`


## Классы

**Описание**:  Класс для работы с API поставщика Morlevi.  (Полное описание должно быть в файле `graber.py`).
