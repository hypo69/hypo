# Модуль `src.suppliers.ebay`

## Обзор

Модуль `src.suppliers.ebay` предназначен для работы с поставщиком eBay. Он содержит основной класс `Graber`, который отвечает за взаимодействие с API eBay.

## Оглавление

- [Обзор](#обзор)
- [Переменные](#переменные)
- [Импорт](#импорт)
    - [Graber](#graber)

## Переменные

### `MODE`

- **Описание**: Указывает режим работы модуля. В данном случае установлено значение `'dev'`.
- **Тип**: `str`
- **Значение по умолчанию**: `'dev'`

## Импорт

### `Graber`

- **Описание**: Класс `Graber` для взаимодействия с API eBay. Импортирован из модуля `graber`.
- **Путь импорта**: `from .graber import Graber`
