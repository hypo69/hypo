# GUNNER

## Обзор

Игра "GUNNER" - это симуляция стрельбы по мишени. Игрок вводит угол выстрела и его скорость, а компьютер рассчитывает, попадет ли снаряд в цель. Игрок имеет 5 попыток и после каждой попытки, получает подсказку: "TOO LOW", "TOO HIGH" или "BINGO".

## Функции

В данном файле нет объявленных функций, так как весь код реализован в виде скрипта.

## Переменные

### `attemptsLeft`

**Описание**: Количество оставшихся попыток у игрока.

**Тип**: `int`

**Начальное значение**: `5`

### `targetDistance`

**Описание**: Случайное расстояние до цели, генерируемое в начале игры.

**Тип**: `int`

**Диапазон**: от 100 до 1000 (футов)

## Игровой цикл

Основной игровой цикл выполняется, пока у игрока есть оставшиеся попытки (`while attemptsLeft > 0:`). В каждой итерации цикла происходит ввод данных, расчет дальности и проверка попадания.

## Обработка ввода

### `try...except ValueError`

**Описание**: Блок `try-except` обрабатывает возможные ошибки ввода.

**Args**:

- `ex (ValueError)`: Перехватывает ошибку, если пользователь вводит не числовое значение.

**Действия**:
- Выводит сообщение об ошибке: "Пожалуйста, введите числовые значения."
- Продолжает выполнение цикла с новой итерации.

### Ввод угла выстрела

**Описание**: Запрашивает у пользователя угол выстрела в градусах (от 0 до 90).

**Тип**: `float`

### Ввод скорости выстрела

**Описание**: Запрашивает у пользователя скорость выстрела в футах в секунду.

**Тип**: `float`

### Проверка ввода угла

**Описание**: Проверяет, находится ли введенный угол в допустимом диапазоне (от 0 до 90 градусов).

**Действия**:
- Если угол вне диапазона, выводит сообщение об ошибке: "Угол должен быть в диапазоне от 0 до 90 градусов."
- Переходит к следующей итерации цикла.

## Расчет дальности полета

### `angle_radians`

**Описание**: Угол выстрела, преобразованный из градусов в радианы.

**Тип**: `float`

**Использование**: `math.radians(angle)`

### `distance`

**Описание**: Расстояние полета снаряда, вычисленное по формуле.

**Тип**: `float`

**Формула**: `(speed**2 * math.sin(2 * angle_radians)) / 32.2`

## Проверка попадания

**Описание**: Проверяет, попал ли снаряд в цель.

**Действия**:
- Использует `abs()` для получения абсолютного значения разницы между вычисленной дальностью и расстоянием до цели.
- Сравнивает разницу с небольшой погрешностью `0.01` для учета неточностей представления чисел с плавающей точкой.
- Если попадание, выводит сообщение "BINGO!" и завершает игру.
- Если не долет, выводит "TOO LOW".
- Если перелет, выводит "TOO HIGH".

## Уменьшение количества попыток

**Описание**: Уменьшает количество оставшихся попыток на 1 после каждого выстрела.

**Действия**: `attemptsLeft -= 1`

## Вывод сообщения о проигрыше

**Описание**: Выводит сообщение "YOU LOSE" если количество попыток закончилось.
