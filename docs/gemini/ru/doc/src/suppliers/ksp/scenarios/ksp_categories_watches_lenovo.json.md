## Раздел `scenarios`

Внутри ключа `scenarios` находится объект, где каждый ключ представляет собой название сценария. В данном случае, есть только один сценарий с ключом `LENOVO WATHES`.

### Сценарий `LENOVO WATHES`

Это объект, описывающий сценарий для парсинга умных часов LENOVO.

#### Параметр `brand`

- **Описание**: Бренд товара, для которого предназначен данный сценарий.
- **Тип**: `строка`
- **Значение**: `"LENOVO"`

#### Параметр `url`

- **Описание**: URL-адрес страницы на сайте KSP, с которой будут парситься данные.
- **Тип**: `строка`
- **Значение**: `"https://ksp.co.il/web/cat/2085..159"`

#### Параметр `checkbox`

- **Описание**: Флаг, указывающий, нужно ли использовать чекбокс при парсинге.
- **Тип**: `логическое значение`
- **Значение**: `false`

#### Параметр `active`

- **Описание**: Флаг, указывающий, активен ли данный сценарий.
- **Тип**: `логическое значение`
- **Значение**: `true`

#### Параметр `condition`

- **Описание**: Состояние товара (например, "new" для нового).
- **Тип**: `строка`
- **Значение**: `"new"`

#### Параметр `presta_categories`

- **Описание**: Объект, сопоставляющий ID категорий PrestaShop с их названиями или ключевыми словами.
- **Тип**: `объект`
    - `"3405"`: `"GOOGLE PIXEL PRO"`
    - `"3198"`: `"CONSUMER ELECTRONICS"`
    - `"3202"`: `"computer,smartphone,gaming console,smart device"`
    - `"6471"`: `"Smartphones"`
    - `"3403"`: `"GOOGLE"`
