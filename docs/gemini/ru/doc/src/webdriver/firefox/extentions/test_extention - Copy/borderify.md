# Модуль borderify.js

## Обзор

Этот модуль отвечает за изменение стилей границы элемента `document.body`.  Он устанавливает для границы `document.body` толщину 5 пикселей и красный цвет.  Так же, он отображает всплывающее окно с помощью `alert()`.


## Функции

### `null`

**Описание**: Данный модуль не содержит функций в классическом понимании.  Он содержит только операторы присваивания и вызова функций.


## Методы

**Описание**: Данный модуль не содержит методов.


## Операторы

### `document.body.style.border = "5px solid red";`

**Описание**: Устанавливает свойство `border` элемента `document.body` со значением `5px solid red`. Это изменяет ширину, тип и цвет границы вокруг тела документа.

**Используемые объекты**:

* `document.body`: Ссылка на корневой элемент `body` HTML-документа.

**Используемые свойства**:

* `style.border`: Свойство, определяющее стили границы элемента.


### `alert();`

**Описание**: Вызывает всплывающее окно с предупреждением (`alert`).


**Используемые объекты**:

* `window`: Ссылка на текущее окно браузера.  Это встроенный объект, который предоставляет доступ к информации о браузере и его контексте.
