# ИНСТРУКЦИЯ по использованию бота для обучения и тестирования модели

## Описание

Данный документ содержит инструкцию для использования Discord бота, предназначенного для обучения и тестирования моделей. Он описывает шаги по подготовке данных, обучению модели, проверке статуса обучения и тестированию ее работы.

## Шаги по обучению модели

### Шаг 1: Убедитесь, что бот запущен

Убедитесь, что ваш бот запущен. В консоли должно появиться сообщение, указывающее на вход бота в систему.

```plaintext
Logged in as YourBotName#1234
```

### Шаг 2: Приглашение бота в сервер

Убедитесь, что бот приглашен на ваш сервер с необходимыми правами на чтение и отправку сообщений.

### Шаг 3: Подготовка данных для обучения

Вы можете обучить модель, используя текстовые данные или файлы, содержащие данные для обучения.

1. **Обучение с текстовыми данными**: Подготовьте строку текстовых данных, которые вы хотите использовать для обучения.

2. **Обучение с файлом**: Подготовьте файл, содержащий данные для обучения. Убедитесь, что файл доступен на вашем локальном компьютере.

### Шаг 4: Использование команды обучения

**Метод 1: Использование текстовых данных напрямую**

1. В канале Discord, где бот имеет доступ, введите следующую команду:
   ```plaintext
   !train "Ваши данные для обучения" positive=True
   ```
   Пример:
   ```plaintext
   !train "Примерные данные для обучения" positive=True
   ```

**Метод 2: Загрузка файла**

1. Прикрепите файл с данными для обучения в сообщении.
2. В том же сообщении введите следующую команду и отправьте ее:
   ```plaintext
   !train positive=True
   ```
   Пример:
   ```plaintext
   !train positive=True
   ```

Бот сохранит файл и начнет обучение модели с предоставленными данными.

### Шаг 5: Отслеживание обучения

После отправки команды обучения бот должен ответить сообщением, указывающим на статус задачи обучения:

```plaintext
Обучение модели начато. ID задачи: <job_id>
```

### Шаг 6: Проверка статуса обучения

Вы можете добавить дополнительные команды в бота для проверки статуса задачи обучения, если это необходимо. Это обычно включает в себя запрос объекта модели для статуса задачи.

### Шаг 7: Тестирование модели

После обучения модели вы можете протестировать ее с помощью команды `!test`.

1. Подготовьте JSON-строку тестовых данных.
2. В канале Discord, где бот имеет доступ, введите следующую команду:
   ```plaintext
   !test {"ключ_теста": "значение_теста"}
   ```
   Пример:
   ```plaintext
   !test {"вход": "Тестовые входные данные"}
   ```

Бот ответит предсказаниями модели.

### Шаг 8: Использование дополнительных команд

Ваш бот также поддерживает другие команды, такие как архивирование файлов и выбор наборов данных. Используйте эти команды аналогично для управления вашими данными и моделью.

**Архивирование файлов**:
```plaintext
!archive <путь_к_каталогу>
```
Пример:
```plaintext
!archive /путь/к/каталогу
```

**Выбор набора данных**:
```plaintext
!select_dataset <путь_к_каталогу_положительных_данных> positive=True
```
Пример:
```plaintext
!select_dataset /путь/к/положительным_данным positive=True
```

## Резюме

1. **Запуск бота**: Убедитесь, что ваш бот запущен.
2. **Приглашение бота**: Убедитесь, что бот находится на вашем сервере Discord.
3. **Подготовка данных**: Подготовьте свои данные для обучения в виде текста или файла.
4. **Обучение модели**: Используйте команду `!train` с текстовыми данными или прикрепленным файлом.
5. **Отслеживание обучения**: Следите за ответом бота о статусе задачи обучения.
6. **Тестирование модели**: Используйте команду `!test` с тестовыми данными для проверки производительности модели.
7. **Управление данными**: Используйте команды `!archive` и `!select_dataset`, если это необходимо.


##  Добавление команды вопросов и ответов

### Инструкция по добавлению команды "задать вопрос"

1. **Запуск бота**: Убедитесь, что ваш бот запущен.

2. **Задаём вопрос**: В канале Discord, где бот имеет доступ, введите следующую команду:

```plaintext
!ask Какой город является столицей Франции?
```

3. **Получение ответа**: Бот должен ответить ответом модели:

```plaintext
Ответ модели: Столицей Франции является Париж.
```

### Резюме

1. **Добавление команды `ask`**:
   - Обновите скрипт бота, чтобы добавить команду `ask`.
   - Реализуйте метод `ask` в вашем классе `Model`, чтобы запросить модель и получить ответ.

2. **Запуск бота**:
   - Запустите свой бота, чтобы он был доступен на вашем сервере Discord.

3. **Задавайте вопросы**:
   - Используйте команду `!ask`, чтобы взаимодействовать с обученной моделью и получать ответы.
