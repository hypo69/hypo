# Модуль `src.suppliers.amazon`

## Обзор

Модуль `src.suppliers.amazon` предназначен для работы с поставщиком Amazon. Он включает в себя функционал для сбора данных (граббер) и определения сценариев взаимодействия, например, получение списка товаров в определенной категории.

## Константы

### `MODE`
- **Описание**: Определяет режим работы модуля (например, `'dev'` для режима разработки).

## Импортированные модули

- `graber`: Модуль, содержащий класс `Graber` для сбора данных с сайта Amazon.
- `scenario`: Модуль, содержащий функцию `get_list_products_in_category` для получения списка товаров в заданной категории.

## Функции

### `get_list_products_in_category`

**Описание**: Функция для получения списка товаров в определенной категории.

- Подробности смотрите в модуле `scenario`.

### `Graber`

**Описание**: Класс для сбора данных с сайта Amazon.

- Подробности смотрите в модуле `graber`.
