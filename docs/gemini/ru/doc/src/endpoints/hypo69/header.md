# Модуль hypotez/src/endpoints/hypo69/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json` и документации из файла `README.MD`.  Он также инициализирует переменные, содержащие информацию о проекте, такие как название, версия, автор, и т.д.


## Функции

### `set_project_root`

**Описание**: Определяет корневую директорию проекта, начиная от текущей директории файла и идя вверх по иерархии директорий, пока не найдёт директорию, содержащую один из указанных файлов маркеров (pyproject.toml, requirements.txt, .git).

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, которые используются для определения корневой директории проекта. По умолчанию содержит ('pyproject.toml', 'requirements.txt', '.git').

**Возвращает**:

- `Path`: Объект Path, представляющий корневую директорию проекта. Если корневая директория не найдена, возвращает директорию, где расположен текущий скрипт.

**Вызывает исключения**:

- Нет


## Переменные

### `__root__`

**Описание**: Переменная, хранящая корневую директорию проекта, полученную с помощью функции `set_project_root`.

**Тип**: `Path`

### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженный из файла `settings.json` в корневой директории проекта.

**Тип**: `dict` или `None`.  `None`, если файл `settings.json` не найден или содержит недопустимый JSON.


### `doc_str`

**Описание**: Строка, содержащая текст документации проекта, загруженная из файла `README.MD` в корневой директории проекта.

**Тип**: `str` или `None`.  `None`, если файл `README.MD` не найден или содержит недопустимый формат.

### `__project_name__`

**Описание**: Строка, содержащая имя проекта. Получается из настроек, если они доступны, иначе устанавливается как `'hypotez'`.

**Тип**: `str`

### `__version__`

**Описание**: Строка, содержащая версию проекта. Получается из настроек, если они доступны, иначе устанавливается как пустая строка.

**Тип**: `str`

### `__doc__`

**Описание**: Строка, содержащая документацию проекта. Получается из `doc_str`, если он доступен, иначе устанавливается как пустая строка.

**Тип**: `str`

### `__details__`

**Описание**: Строка, содержащая подробности о проекте. По умолчанию пустая строка.

**Тип**: `str`

### `__author__`

**Описание**: Строка, содержащая имя автора проекта. Получается из настроек, если они доступны, иначе устанавливается как пустая строка.

**Тип**: `str`

### `__copyright__`

**Описание**: Строка, содержащая копирайт проекта. Получается из настроек, если они доступны, иначе устанавливается как пустая строка.

**Тип**: `str`


### `__cofee__`

**Описание**: Строка, содержащая ссылку на спонсорскую платформу для поддержки разработчика. Получается из настроек, если они доступны, иначе устанавливается как значение по умолчанию.

**Тип**: `str`
