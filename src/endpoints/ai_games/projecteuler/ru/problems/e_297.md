# Задача No 297
## Сумма целых чисел, имеющих представление Фибоначчи

## Условия
Каждое натуральное число может быть однозначно представлено в виде суммы различных чисел Фибоначчи таким образом, что никакие два соседних числа Фибоначчи не используются в сумме. Например:
```
100 = 3 + 8 + 89
```
Такое представление называется представлением Зекендорфа.

Для натурального числа n обозначим через f(n) количество чисел Фибоначчи в его представлении Зекендорфа.
Например: f(100) = 3 и f(123456) = 4.

Найдите $\sum_{i=1}^{10^{17}} f(i)$.
