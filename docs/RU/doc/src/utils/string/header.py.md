# Модуль `header.py`

## Обзор

Модуль `header.py` предназначен для определения корневого пути к проекту, загрузки настроек из файла `settings.json`, а также чтения документации из файла `README.MD`. Все импорты в проекте строятся относительно корневого пути, найденного этим модулем.

## Функции

### `set_project_root`

**Описание**:
Находит корневую директорию проекта, начиная с директории текущего файла, ища вверх по дереву каталогов до первого каталога, содержащего любой из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple, optional): Кортеж имен файлов или каталогов, которые идентифицируют корень проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена, иначе - директорию, где находится скрипт.

## Переменные

### `MODE`

**Описание**:
Режим работы приложения (`dev` - разработка).

### `__root__`

**Описание**:
Корневой путь к проекту (тип `pathlib.Path`).

### `settings`

**Описание**:
Словарь с настройками проекта, загруженный из файла `settings.json`.

### `doc_str`

**Описание**:
Строка, содержащая документацию проекта, загруженную из файла `README.MD`.

### `__project_name__`

**Описание**:
Название проекта, взятое из настроек или значение по умолчанию 'hypotez'.

### `__version__`

**Описание**:
Версия проекта, взятая из настроек.

### `__doc__`

**Описание**:
Документация проекта, загруженная из `README.MD`.

### `__details__`

**Описание**:
Дополнительные детали проекта (в текущей реализации пустая строка).

### `__author__`

**Описание**:
Автор проекта, взятый из настроек.

### `__copyright__`

**Описание**:
Информация о копирайте, взятая из настроек.

### `__cofee__`

**Описание**:
Строка, содержащая ссылку на поддержку разработчика чашечкой кофе (или аналогичную поддержку).
