# Acey Ducey Card Game

## Обзор

Данный модуль представляет собой симуляцию карточной игры Acey Ducey. Игрок делает ставки, пытаясь угадать, будет ли следующая карта по значению между двумя открытыми картами. Игра продолжается до тех пор, пока у игрока не закончатся деньги или пока игрок не прервет игру.

## Функции

### `create_deck`

**Описание**: Создает и перемешивает колоду карт для игры.

**Параметры**:

- Нет параметров

**Возвращает**:
- `list`: Список целых чисел, представляющих значения карт в колоде. Значения от 2 до 14 (туз = 14). Колода состоит из 52 карт (4 масти).

### `card_name`

**Описание**: Преобразует числовое значение карты в ее текстовое представление (например, "Валет", "Дама", "Король", "Туз").

**Параметры**:
- `value` (int): Числовое значение карты.

**Возвращает**:
- `str`: Строковое представление значения карты.

### `play_acey_ducey`

**Описание**: Запускает основной игровой цикл Acey Ducey.

**Параметры**:
- Нет параметров

**Возвращает**:
- `None`: Функция ничего не возвращает, но выводит результат игры и сообщения в консоль.

**Вызывает исключения**:
- `ValueError`: Если пользователь вводит некорректное значение ставки.
