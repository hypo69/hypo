# Модуль для загрузки примеров спецификаций агентов и фрагментов

## Обзор

Модуль предоставляет функции для загрузки примеров спецификаций агентов и фрагментов, а также для получения списка доступных примеров. Этот модуль полезен для демонстрации и тестирования функциональности системы tiny_troupe.

## Подробнее

Модуль содержит функции для загрузки спецификаций агентов и фрагментов из файлов JSON, расположенных в каталогах `agents` и `fragments` соответственно. Он также предоставляет функции для получения списка доступных примеров агентов и фрагментов. Расположение файла в проекте `hypotez` указывает на то, что он используется для загрузки конфигураций агентов и фрагментов, вероятно, для целей тестирования или демонстрации.

## Функции

### `load_example_agent_specification`

```python
def load_example_agent_specification(name: str) -> dict:
    """
    Load an example agent specification.

    Args:
        name (str): The name of the agent.

    Returns:
        dict: The agent specification.
    """
    ...
```

**Назначение**: Загружает пример спецификации агента.

**Параметры**:
- `name` (str): Имя агента, для которого требуется загрузить спецификацию.

**Возвращает**:
- `dict`: Словарь, представляющий спецификацию агента.

**Как работает функция**:

1.  Функция принимает имя агента в качестве входного параметра.
2.  Она конструирует путь к файлу спецификации агента на основе имени, предполагая, что файлы находятся в подкаталоге `agents`.
3.  Использует `json.load` для загрузки JSON-данных из указанного файла и возвращает их в виде словаря.

```
A: Принятие имени агента
|
B: Построение пути к файлу спецификации агента
|
C: Загрузка JSON-данных из файла
|
D: Возвращение спецификации агента в виде словаря
```

**Примеры**:

```python
agent_spec = load_example_agent_specification('my_agent')
print(agent_spec)
# {'name': 'My Agent', 'description': 'Пример агента'}
```

### `list_example_agents`

```python
def list_example_agents() -> list:
    """
    List the available example agents.

    Returns:
        list: A list of the available example agents.
    """
    ...
```

**Назначение**: Получает список доступных примеров агентов.

**Возвращает**:
- `list`: Список имен доступных примеров агентов.

**Как работает функция**:

1.  Функция получает список файлов в подкаталоге `agents`.
2.  Для каждого файла удаляет расширение `.agent.json`, чтобы получить имя агента.
3.  Возвращает список имен агентов.

```
A: Получение списка файлов в подкаталоге агентов
|
B: Удаление расширения .agent.json для получения имени агента
|
C: Возвращение списка имен агентов
```

**Примеры**:

```python
agents = list_example_agents()
print(agents)
# ['agent1', 'agent2', 'agent3']
```
