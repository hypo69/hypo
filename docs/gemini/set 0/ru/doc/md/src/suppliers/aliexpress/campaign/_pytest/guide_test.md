### `prepare_campaigns.py`

**Описание**: Этот файл содержит функции для подготовки материалов кампании, включая обновление категорий и обработку кампаний по категориям.

**Функции**:

#### `update_category(category_data: dict, category_file: str) -> bool | None`

**Описание**: Обновляет категорию в JSON файле.

**Параметры**:
* `category_data` (dict): Данные для обновления категории.
* `category_file` (str): Путь к файлу с категориями.

**Возвращает**:
* `bool | None`: `True`, если обновление прошло успешно, `False` - при ошибке, `None` - если возникла непредвиденная ошибка.

**Вызывает исключения**:
* `FileNotFoundError`: Если файл не найден.
* `JSONDecodeError`: Если файл не является корректным JSON.
* Другие ошибки при работе с файловой системой или JSON.


#### `process_campaign_category(campaign_data: dict, category_data: dict) -> dict | None`

**Описание**: Обработка конкретной категории в рамках кампании.

**Параметры**:
* `campaign_data` (dict): Данные о кампании.
* `category_data` (dict): Данные о категории.

**Возвращает**:
* `dict | None`: Результат обработки категории или `None` при ошибке.


#### `process_campaign(campaign_data: dict) -> list | None`

**Описание**: Обработка всей кампании по всем категориям.

**Параметры**:
* `campaign_data` (dict): Данные о кампании.

**Возвращает**:
* `list | None`: Список результатов обработки каждой категории или `None` при ошибке.


#### `main(campaign_data: dict) -> None`

**Описание**: Асинхронная основная функция для обработки кампании.

**Параметры**:
* `campaign_data` (dict): Данные о кампании.

**Возвращает**:
* `None`

## Инструкции по тестированию

1. **Установка зависимостей**: Убедитесь, что все необходимые зависимости установлены. Выполните команду:
   ```bash
   pip install -r requirements.txt
   ```

2. **Запуск тестов**: Для запуска всех тестов используйте команду:
   ```bash
   pytest test_campaign_integration.py
   ```

3. **Проверка тестов**: Убедитесь, что все тесты проходят успешно. В выводе команды `pytest` должно быть указано, что все тесты пройдены (`PASSED`).
