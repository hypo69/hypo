# Модуль `products.py`

## Обзор

Модуль `products.py` содержит функции для обработки данных о продуктах, полученных от AliExpress API. Основная цель - преобразование и структурирование данных о продуктах для дальнейшего использования в проекте `hypotez`. Модуль предоставляет функции для обработки одного продукта и списка продуктов.

## Подробней

Модуль предназначен для работы с данными, которые возвращает AliExpress API. Он содержит функции, которые приводят данные к нужному формату, например, преобразуют `product_small_image_urls` в строковый тип. Это необходимо для унификации данных и упрощения работы с ними в дальнейшем.

## Функции

### `parse_product`

```python
def parse_product(product):
    """
    Преобразует данные об отдельном продукте.

    Args:
        product: Объект продукта, полученный от AliExpress API.

    Returns:
        product: Объект продукта с преобразованными данными.

    """
```

**Описание**: Преобразует данные об отдельном продукте, полученные от AliExpress API. В частности, преобразует поле `product_small_image_urls` в строковый тип.

**Параметры**:
- `product`: Объект продукта, полученный от AliExpress API.

**Возвращает**:
- `product`: Объект продукта с преобразованными данными.

### `parse_products`

```python
def parse_products(products):
    """
    Преобразует список продуктов, полученных от AliExpress API.

    Args:
        products: Список объектов продуктов, полученных от AliExpress API.

    Returns:
        new_products: Список объектов продуктов с преобразованными данными.

    """
```

**Описание**: Преобразует список продуктов, полученных от AliExpress API, применяя функцию `parse_product` к каждому элементу списка.

- `products`: Список объектов продуктов, полученных от AliExpress API.

**Возвращает**:
- `new_products`: Список объектов продуктов с преобразованными данными.
