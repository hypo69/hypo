# Модуль hypotez/src/webdriver/_pytest/test_driver_executor.py

## Обзор

Этот модуль содержит тесты для проверки функциональности класса `ExecuteLocator` и взаимодействия с WebDriver. Тесты охватывают навигацию по страницам, поиск элементов, отправку сообщений, получение атрибутов и обработку исключений.

## Фикстуры

### `driver`

**Описание**: Фикстура для инициализации и завершения работы WebDriver.

**Возвращает**:
- `webdriver.Chrome`: Экземпляр WebDriver.

**Установки**:
- Создает экземпляр `webdriver.Chrome` с заданными опциями (включая headless режим).
- Назначает путь к chromedriver.
- Открывает страницу `http://example.com`.

**Разрушения**:
- Закрывает WebDriver.

### `execute_locator`

**Описание**: Фикстура для создания экземпляра `ExecuteLocator`.

**Аргументы**:
- `driver`: Экземпляр WebDriver, полученный из фикстуры `driver`.

**Возвращает**:
- `ExecuteLocator`: Экземпляр класса `ExecuteLocator` с подключенным к нему WebDriver.

## Функции

### `test_navigate_to_page`

**Описание**: Проверка корректной навигации WebDriver к указанной странице.

**Аргументы**:
- `execute_locator`: Экземпляр `ExecuteLocator`.
- `driver`: Экземпляр `WebDriver`.

**Проверяет**:
- Что текущий URL WebDriver равен `http://example.com`.


### `test_get_webelement_by_locator_single_element`

**Описание**: Проверка корректного получения элемента по локатору.

**Аргументы**:
- `execute_locator`: Экземпляр `ExecuteLocator`.
- `driver`: Экземпляр `WebDriver`.

**Проверяет**:
- Что полученный элемент является экземпляром `WebElement`.
- Что текст элемента равен ожидаемому ("Example Domain").


### `test_get_webelement_by_locator_no_element`

**Описание**: Проверка возвращаемого значения при отсутствии элемента по локатору.

**Аргументы**:
- `execute_locator`: Экземпляр `ExecuteLocator`.
- `driver`: Экземпляр `WebDriver`.

**Проверяет**:
- Что метод возвращает `False`, если элемент не найден.


### `test_send_message`

**Описание**: Проверка отправки сообщения элементу.

**Аргументы**:
- `execute_locator`: Экземпляр `ExecuteLocator`.
- `driver`: Экземпляр `WebDriver`.

**Проверяет**:
- Что отправка сообщения проходит успешно (возвращает `True`).


### `test_get_attribute_by_locator`

**Описание**: Проверка получения атрибута элемента.

**Аргументы**:
- `execute_locator`: Экземпляр `ExecuteLocator`.
- `driver`: Экземпляр `WebDriver`.

**Проверяет**:
- Что значение атрибута `href` элемента соответствует ожидаемому.


### `test_execute_locator_event`

**Описание**: Проверка выполнения события на элементе.

**Аргументы**:
- `execute_locator`: Экземпляр `ExecuteLocator`.
- `driver`: Экземпляр `WebDriver`.

**Проверяет**:
- Что выполнение события (например, клик) проходит успешно.


### `test_get_locator_keys`

**Описание**: Проверка получения ключей локатора.

**Аргументы**:
- `execute_locator`: Экземпляр `ExecuteLocator`.
- `driver`: Экземпляр `WebDriver`.

**Проверяет**:
- Что возвращаемый набор ключей соответствует ожидаемому.


### `test_navigate_and_interact`

**Описание**: Проверка навигации и взаимодействия с элементами на другой странице.

**Аргументы**:
- `execute_locator`: Экземпляр `ExecuteLocator`.
- `driver`: Экземпляр `WebDriver`.

**Проверяет**:
- Навигацию к новой странице.
- Взаимодействие с элементами на новой странице.
- Проверку загрузки результатов поиска.


### `test_invalid_locator`

**Описание**: Проверка обработки некорректных локаторов.

**Аргументы**:
- `execute_locator`: Экземпляр `ExecuteLocator`.
- `driver`: Экземпляр `WebDriver`.

**Проверяет**:
- Что при некорректном локаторе возникает исключение `ExecuteLocatorException`.
