### `login(self)`

**Описание:**

Функция осуществляет вход в систему на сайте `visualdg.co.il`.

**Параметры:**

* `self`: Объект, содержащий методы для работы с браузером (например, `selenium`-webdriver). Предполагается, что в `self` уже заполнены данные `locators`, необходимые для нахождения элементов на странице.

**Возвращаемое значение:**

* `True`: Если вход успешен.

**Детали реализации:**

1. Открывает страницу входа `https://www.visualdg.co.il/customer_login`.
2. Получает значения `email` и `password` из `self.locators['login']`.
3. Находит поля `email` и `password` на странице с использованием данных из `self.locators['login']`.  **Важно:** Используется структура `locators` для хранения информации о селекторах, что повышает читаемость и поддерживаемость кода. Необходимо документально описать, как формируются данные в `self.locators`.
4. Вводит `email` и `password` в соответствующие поля.
5. Ждет 1 секунду ( `self.wait(1)`).
6. Нажимает кнопку входа.
7. Ждет 1 секунду.
8. Выводит сообщение `VDG logged in` в лог.
