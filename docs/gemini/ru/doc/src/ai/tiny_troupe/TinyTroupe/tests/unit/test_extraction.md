# Модуль `test_extraction.py`

## Обзор

Этот модуль содержит тесты для модуля `extraction`, проверяя корректность экспорта данных в различные форматы (JSON, текст, docx) и нормализацию концепций.

## Фикстуры

### `exporter`

**Описание**: Фикстура, возвращающая экземпляр класса `ArtifactExporter` с заданным каталогом вывода.

**Возвращает**:
- `ArtifactExporter`: Экземпляр класса `ArtifactExporter`.


## Функции

### `test_export_json(exporter)`

**Описание**: Тестирует экспорт данных в формате JSON.

**Параметры**:
- `exporter` (`ArtifactExporter`): Экземпляр класса `ArtifactExporter`.


**Описание**: Тестирует экспорт данных в формате JSON.

**Параметры**:
- `exporter` (`ArtifactExporter`): Экземпляр класса `ArtifactExporter`.
- `artifact_data` (dict): Данные, которые необходимо экспортировать в формате JSON.

**Возвращает**:
- Ничего.

**Вызывает исключения**:
- Никаких.

**Описание**: Тестирует экспорт данных в формате текст.

**Параметры**:
- `exporter` (`ArtifactExporter`): Экземпляр класса `ArtifactExporter`.
- `artifact_data` (str): Данные, которые необходимо экспортировать в формате текста.


**Возвращает**:

**Параметры**:
- `exporter` (`ArtifactExporter`): Экземпляр класса `ArtifactExporter`.
- `artifact_data` (str): Данные, которые необходимо экспортировать в формате Markdown.


**Возвращает**:

**Вызывает исключения**:
- Никаких.


### `test_normalizer`

**Описание**: Тестирует функцию нормализации концепций.

**Параметры**:
- `concepts` (list): Список концепций, которые нужно нормализовать.
- `n` (int): Количество нормализованных элементов, которые нужно вернуть.
- `verbose` (bool): Флаг для вывода отладочной информации.

**Возвращает**:
- Ничего.

**Вызывает исключения**:
- Никаких.


## Дополнительные замечания

- Тесты проверяют существование экспортированных файлов и соответствие их содержимого ожидаемым данным.
- Для проверки docx используется библиотека `docx`.
- Тест `test_normalizer` проверяет, что количество нормализованных элементов соответствует заданному значению `n`, и что все элементы входного списка присутствуют в словаре `normalizing_map`.
