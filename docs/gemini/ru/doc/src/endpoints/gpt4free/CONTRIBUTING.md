# Документация для разработчика: CONTRIBUTING.md

## Обзор

Этот файл содержит инструкции для тех, кто хочет внести свой вклад в проект `gpt4free`. Он описывает шаги, которые необходимо выполнить для добавления новых веб-сайтов в список поддерживаемых источников.

## Подробней

Этот файл служит руководством для контрибьюторов, объясняя процесс добавления новых веб-сайтов в проект `gpt4free`. Он включает в себя реверс-инжиниринг веб-сайта, добавление его в список модульных тестов и рефакторинг кода для интеграции в основную библиотеку `g4f`.

## Инструкции по внесению вклада

1.  **Реверс-инжиниринг веб-сайта:**

    *   Выберите веб-сайт из списка [sites-to-reverse](https://github.com/xtekky/gpt4free/issues/40), который вы хотите добавить в проект.

2.  **Добавление в модульные тесты:**

    *   Добавьте информацию о выбранном веб-сайте в директорию [./etc/unittest/](https://github.com/xtekky/gpt4free/tree/main/etc/unittest/). Это необходимо для обеспечения стабильности и корректности работы нового источника.

3.  **Рефакторинг и интеграция:**

    *   Выполните рефакторинг кода и добавьте его в директорию [./g4f](https://github.com/xtekky/gpt4free/tree/main/g4f). Этот шаг включает в себя адаптацию кода веб-сайта для работы с `gpt4free` и интеграцию его в основную библиотеку.

## Благодарность

Разработчики проекта выражают благодарность всем, кто готов внести свой вклад в развитие `gpt4free`.
