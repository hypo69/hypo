# Модуль hypotez/src/webdriver/edge/extentions/__init__.py

## Обзор

Этот модуль содержит константы, вероятно, определяющие режим работы (MODE) и импортирует информацию о версии из файла `version.py`.


## Константы

### `MODE`

**Описание**:  Переменная, содержащая строковое значение, скорее всего, определяющее режим работы (например, 'dev', 'prod').

**Значение**: `'dev'`


## Модули

### `packaging.version`

**Описание**:  Модуль для работы с версиями пакетов.


### `hypotez.src.webdriver.edge.extentions.version`

**Описание**: Модуль, содержащий информацию о версии, документацию и дополнительные детали.

**Импортируемые переменные**:
- `__version__`: Версия модуля.
- `__doc__`: Документация модуля.
- `__details__`: Дополнительная информация о модуле.
