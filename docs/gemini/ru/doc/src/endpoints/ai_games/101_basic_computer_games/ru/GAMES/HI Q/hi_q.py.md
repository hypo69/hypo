# Игра HI Q

## Обзор

Данный модуль реализует игру "HI Q" - головоломку, в которой игрок удаляет шарики с доски, перепрыгивая через них. Цель игры - оставить на доске только один шарик. Игрок вводит координаты шарика и направление для перепрыгивания. Игра завершается, когда не остается возможных ходов или на доске остается только один шарик.
