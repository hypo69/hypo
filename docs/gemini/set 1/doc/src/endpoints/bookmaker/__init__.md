# Модуль `hypotez/src/endpoints/__init__.py`

## Обзор

Этот модуль (`hypotez/src/endpoints/__init__.py`) служит для инициализации и импорта различных конечных точек (endpoints) из приложения `hypotez`. Он определяет константу `MODE` с текущим режимом работы.


## Константы

### `MODE`

**Описание**: Константа, определяющая режим работы приложения.  В данном случае, задано значение `'dev'`, что указывает на режим разработки.
