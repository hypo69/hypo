## Переменные

### `MODE`

**Описание**:
Определяет режим работы модуля.

**Тип**:
`str`

**Значение по умолчанию**:
`dev`

### `__name__`

**Описание**:
Содержит имя модуля. Если скрипт запущен напрямую, значение будет `"__main__"`.

**Тип**:
`str`

### `__version__`

**Описание**:
Хранит версию модуля или пакета.

**Тип**:
`str`

**Значение по умолчанию**:
`"3.12.0.0.0.4"`

### `__doc__`

**Описание**:
Строка документации модуля.

**Тип**:
`str`

### `__details__`

**Описание**:
Содержит дополнительные сведения о версии модуля или класса.

**Тип**:
`str`

**Значение по умолчанию**:
`"Details about version for module or class"`

### `__annotations__`

**Описание**:
Содержит аннотации типов для переменных и функций в модуле.

**Тип**:
`any`

### `__author__`

**Описание**:
Имя(имена) автора(ов) модуля.

**Тип**:
`str`

**Значение по умолчанию**:
`'hypotez '`
