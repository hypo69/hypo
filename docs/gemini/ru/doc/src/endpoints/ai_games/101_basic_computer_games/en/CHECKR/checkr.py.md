# CHECKR

## Обзор

Данный модуль представляет собой текстовую реализацию упрощенной версии игры в шашки, где игрок играет против компьютера. Игра происходит на доске 8x8, целью которой является достижение противоположного конца доски одной из своих шашек, избегая при этом захвата шашек противника.

## Глобальные переменные

- `BOARD_SIZE` (int): Размер доски (8x8).
- `EMPTY` (str): Символ для пустой клетки (`.`).
- `PLAYER` (str): Символ для шашки игрока (`1`).
- `COMPUTER` (str): Символ для шашки компьютера (`2`).

## Функции

### `initialize_board`

**Описание**: Инициализирует доску 8x8 с начальным расположением шашек.

**Параметры**:
- Нет.

**Возвращает**:
- `list[list[str]]`: Двумерный список, представляющий доску, где размещены шашки игрока и компьютера в начальных позициях.

### `draw_board`

**Описание**: Отрисовывает текущее состояние доски в консоль.

**Параметры**:
- `board` (list[list[str]]): Двумерный список, представляющий текущее состояние доски.

**Возвращает**:
- `None`: Функция ничего не возвращает, а только выводит доску в консоль.

### `is_valid_move`

**Описание**: Проверяет, является ли ход игрока или компьютера допустимым.

**Параметры**:
- `board` (list[list[str]]): Текущее состояние доски.
- `row` (int): Строка текущей позиции шашки.
- `col` (int): Столбец текущей позиции шашки.
- `new_row` (int): Строка новой позиции шашки.
- `new_col` (int): Столбец новой позиции шашки.
- `player` (str): Игрок, для которого проверяется ход (`PLAYER` или `COMPUTER`).

**Возвращает**:
- `bool`: `True`, если ход допустим, `False` в противном случае.

### `update_board`

**Описание**: Обновляет доску после хода.

**Параметры**:
- `board` (list[list[str]]): Текущее состояние доски.
- `row` (int): Строка текущей позиции шашки.
- `col` (int): Столбец текущей позиции шашки.
- `new_row` (int): Строка новой позиции шашки.
- `new_col` (int): Столбец новой позиции шашки.

**Возвращает**:
- `None`: Функция изменяет доску напрямую.

### `check_win`

**Описание**: Проверяет, достиг ли игрок или компьютер победы.

**Параметры**:
- `board` (list[list[str]]): Текущее состояние доски.
- `player` (str): Игрок, для которого проверяется победа (`PLAYER` или `COMPUTER`).

**Возвращает**:
- `bool`: `True`, если игрок или компьютер победили, `False` в противном случае.

### `get_computer_moves`

**Описание**: Находит все возможные ходы компьютера.

**Параметры**:
- `board` (list[list[str]]): Текущее состояние доски.

**Возвращает**:
- `list[tuple[int, int, int, int]]`: Список кортежей с доступными ходами компьютера в формате (row, col, new_row, new_col).

### `computer_turn`

**Описание**: Выполняет ход компьютера.

**Параметры**:
- `board` (list[list[str]]): Текущее состояние доски.

**Возвращает**:
- `None`: Функция изменяет доску напрямую.

### `player_turn`

**Описание**: Запрашивает и выполняет ход игрока.

**Параметры**:
- `board` (list[list[str]]): Текущее состояние доски.

**Возвращает**:
- `None`: Функция изменяет доску напрямую.

### `play_checkers`

**Описание**: Основная функция игры в шашки.

**Параметры**:
- Нет.

**Возвращает**:
- `None`: Функция управляет игровым процессом и не возвращает значение.
