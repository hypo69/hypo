# Модуль facebook

## Обзор

Данный модуль содержит API-эндпоинты для работы с рекламой на Facebook.  Он предоставляет инструменты для создания, обновления и получения информации о рекламных кампаниях, объявлениях и других связанных ресурсах.

## Функции

### `create_campaign`

**Описание**: Функция для создания новой рекламной кампании на Facebook.

**Параметры**:
- `campaign_data` (dict): Словарь с данными кампании. Обязательный параметр.  Требуется информация о целевой аудитории, бюджете, и других важных настройках. Должен соответствовать спецификации Facebook API.
- `access_token` (str): Токен доступа к Facebook API. Обязательный параметр.

**Возвращает**:
- `dict | None`: Словарь с данными созданной кампании, если операция выполнена успешно, иначе `None`.  В случае ошибки может содержать информацию об ошибке.


**Вызывает исключения**:
- `FacebookAPIError`: Возникает в случае ошибки при взаимодействии с Facebook API.  Например, при неверном токен доступа, или отсутствии необходимых прав.
- `ValueError`: Возникает при неверных входных данных (некорректный формат `campaign_data` или `access_token`).


### `get_campaign_details`

**Описание**: Функция для получения подробных данных о рекламной кампании.

**Параметры**:
- `campaign_id` (str): Идентификатор кампании. Обязательный параметр.
- `access_token` (str): Токен доступа к Facebook API. Обязательный параметр.

**Возвращает**:
- `dict | None`: Словарь с деталями кампании, если запрос успешен, иначе `None`.  В случае ошибки может содержать информацию об ошибке.


**Вызывает исключения**:
- `FacebookAPIError`: Возникает в случае ошибки при взаимодействии с Facebook API. Например, если кампания не найдена или нет доступа.
- `ValueError`: Возникает при некорректном идентификаторе кампании или `access_token`.
