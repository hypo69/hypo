# Модуль стилей для UI элементов AliExpress

## Обзор

Модуль `styles.py` содержит общие функции для стилизации элементов пользовательского интерфейса (UI) в проекте `hypotez`, в частности, для работы с поставщиком AliExpress. Он предоставляет возможность устанавливать фиксированный размер для виджетов PyQt6.

## Подробнее

Этот модуль предназначен для стандартизации внешнего вида элементов интерфейса, что упрощает поддержку и улучшает визуальную консистентность приложения. Он содержит функции, которые могут быть использованы для настройки размеров виджетов, обеспечивая единообразный стиль отображения.

## Функции

### `set_fixed_size`

```python
def set_fixed_size(widget: QtWidgets.QWidget, width: int, height: int):
    """ Set a fixed size for a given widget """
```

**Назначение**: Устанавливает фиксированный размер для заданного виджета PyQt6.

**Параметры**:
- `widget` (QtWidgets.QWidget): Виджет, для которого необходимо установить фиксированный размер.
- `width` (int): Ширина виджета в пикселях.
- `height` (int): Высота виджета в пикселях.

**Возвращает**:
- `None`: Функция ничего не возвращает.

**Вызывает исключения**:
- Отсутствуют явные исключения.

**Как работает функция**:

1. Функция `set_fixed_size` принимает виджет `widget`, ширину `width` и высоту `height` в качестве аргументов.
2. Она использует метод `setFixedSize` объекта `widget` для установки фиксированного размера виджета.

**ASCII flowchart**:

```
A[Принимает виджет, ширину, высоту]
|
B[Устанавливает фиксированный размер виджета с использованием setFixedSize]
```

**Примеры**:

```python
from PyQt6 import QtWidgets
from PyQt6.QtWidgets import QApplication, QWidget, QPushButton
import sys
from src.suppliers.aliexpress.gui.styles import set_fixed_size

app = QApplication(sys.argv)
window = QWidget()
button = QPushButton("Click me")
set_fixed_size(button, 100, 50)  # Устанавливает фиксированный размер кнопки 100x50 пикселей
window.show()
sys.exit(app.exec())
