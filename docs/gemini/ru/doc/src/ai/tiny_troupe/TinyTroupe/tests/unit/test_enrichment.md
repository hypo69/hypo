# Модуль `test_enrichment.py`

## Обзор

Этот файл содержит модульные тесты для класса `TinyEnricher`, проверяющие корректность обогащения контента. Тесты проверяют, что результат обогащения соответствует заданным требованиям, включая увеличение размера контента в 3 раза.

## Функции

### `test_enrich_content`

**Описание**:  Функция тестирует метод `enrich_content` класса `TinyEnricher`. Она принимает исходный контент, требования к обогащению, тип контента и другую контекстную информацию, и проверяет корректность результата.

**Параметры**:

- `content_to_enrich` (str): Исходный контент, который необходимо обогатить. Представлен в виде Markdown документа.
- `requirements` (str): Требования к обогащению, описывающие желаемый формат и детализацию результата.
- `content_type` (str): Тип контента, в данном случае "Document".
- `context_info` (str): Контекстная информация о происхождении или контексте контента.
- `context_cache` (Optional[dict], optional): Кэшированная информация, если необходимо, иначе None.
- `verbose` (bool): Флаг, указывающий на необходимость вывода дополнительных сообщений (в данном случае, True).

**Возвращает**:

- `result` (str | None): Результат обогащения контента. Возвращает строку или `None`, если обогащение не выполнилось.

**Вызывает исключения**:

- `AssertionError`: Возникает, если результат обогащения не соответствует ожидаемым требованиям (например, если размер результата меньше 3-х кратного исходного).
