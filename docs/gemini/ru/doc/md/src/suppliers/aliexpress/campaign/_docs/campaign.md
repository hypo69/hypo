# Модуль `campaign` для AliExpress

## Обзор

Модуль `campaign` предназначен для управления и редактирования рекламных кампаний на AliExpress, взаимодействия с Google Таблицами для получения и обработки данных, а также подготовки данных кампании для использования.  Этот документ описывает структуру, алгоритм работы и функциональность модуля.

## Классы

### `AliPromoCampaign`

**Описание**: Класс `AliPromoCampaign` отвечает за работу с одной рекламной кампанией, включая загрузку данных из Google Таблиц, взаимодействие с категориями и продуктами.

**Методы**:

- `__init__(campaign_name, category_name, language, currency, force_update)`:
    **Описание**: Инициализирует объект кампании.
    **Параметры**:
        - `campaign_name` (str): Название кампании.
        - `category_name` (str): Название категории.
        - `language` (str): Язык.
        - `currency` (str): Валюта.
        - `force_update` (bool, optional): Принудительно обновлять данные. По умолчанию `False`.
    **Возвращает**:
        - `None`

- `initialize_campaign()`:
    **Описание**: Инициализирует внутренние данные кампании.
    **Параметры**:
        - `None`
    **Возвращает**:
        - `None`


- `get_category_from_campaign()`:
    **Описание**: Извлекает данные о категории из объекта кампании.
    **Параметры**:
        - `None`
    **Возвращает**:
        - `SimpleNamespace` или `None`: Данные о категории.

- `get_category_products(force_update)`:
    **Описание**: Извлекает продукты для указанной категории.
    **Параметры**:
        - `force_update` (bool): Принудительно обновлять данные.
    **Возвращает**:
        - `list`: Список продуктов.


- `create_product_namespace(**kwargs)`:
    **Описание**: Создает объект `SimpleNamespace` с данными продукта.
    **Параметры**:
        - `**kwargs`: Данные продукта.
    **Возвращает**:
        - `SimpleNamespace`

- `create_campaign_namespace(**kwargs)`:
    **Описание**: Создает объект `SimpleNamespace` с данными кампании.
    **Параметры**:
        - `**kwargs`: Данные кампании.
    **Возвращает**:
        - `SimpleNamespace`


- `prepare_products()`:
    **Описание**: Подготавливает данные продуктов для дальнейшего использования.
    **Параметры**:
        - `None`
    **Возвращает**:
        - `None`
