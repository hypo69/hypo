# Модуль hypotez/src/webdriver/edge/header.py

## Обзор

Данный модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`.  Также он загружает документацию из файла `README.MD`.

## Функции

### `set_project_root`

**Описание**:  Ищет корневую директорию проекта, начиная от текущей директории файла и двигаясь вверх по иерархии директорий.  Останавливается на первой директории, содержащей один из файлов маркеров (указанных в аргументе).

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, которые будут использованы в качестве маркеров корневой директории. По умолчанию - `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращает путь к директории, в которой расположен текущий файл.

**Вызывает исключения**:

- Нет.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root`.

**Тип**: `Path`.


## Дополнительные сведения

### Загрузка настроек

Модуль пытается загрузить настройки из файла `settings.json` в переменную `settings`.  Если файл не найден или произошла ошибка декодирования JSON, переменная `settings` остается равной `None`.


### Загрузка документации

Модуль пытается загрузить содержимое файла `README.MD` в переменную `doc_str`. Если файл не найден или произошла ошибка чтения, переменная `doc_str` остается равной `None`.


### Константные значения (Переменные с фиксированными значениями)

Следующие переменные содержат значения, загруженные из файла настроек (`settings.json`), или содержат значение по умолчанию, если файл не найден или соответствующий параметр отсутствует:

- `__project_name__` (str):  Название проекта (значение из `settings.json` или 'hypotez' по умолчанию).
- `__version__` (str): Версия проекта (значение из `settings.json` или пустая строка по умолчанию).
- `__doc__` (str): Документация проекта (значение из `README.MD` или пустая строка по умолчанию).
- `__details__` (str): Дополнительные данные о проекте (пустая строка по умолчанию).
- `__author__` (str): Автор проекта (значение из `settings.json` или пустая строка по умолчанию).
- `__copyright__` (str): Авторские права (значение из `settings.json` или пустая строка по умолчанию).
- `__cofee__` (str): Ссылка для поддержки разработчика (значение из `settings.json` или значение по умолчанию).
