# Модуль `src.suppliers.kualastyle.login`

## Обзор

Модуль `src.suppliers.kualastyle.login` содержит функции для авторизации поставщика Kualastyle, включая логин и закрытие всплывающих окон.

## Содержание

1.  [Функции](#Функции)
    -   [`login`](#login)
    -   [`close_pop_up`](#close_pop_up)

## Функции

### `login`

**Описание**: Функция выполняет вход в систему поставщика.

**Параметры**:
-   `s` (Supplier): Объект поставщика.

**Возвращает**:
-   `bool`: `True`, если вход выполнен успешно, иначе `False`.

### `close_pop_up`

**Описание**: Функция закрывает всплывающее окно на сайте поставщика.

**Параметры**:
-   `s` (Supplier): Объект поставщика.

**Возвращает**:
-   `bool`: `True`, если окно закрыто успешно, иначе `False`.

**Вызывает исключения**:
- `Exception`: Выводит предупреждение в лог, если не удалось закрыть всплывающее окно.
