# Модуль `start_sergey`

## Обзор

Модуль предназначен для автоматической публикации рекламных объявлений в группах Facebook. Он конфигурирует и запускает рекламные кампании, используя заданные параметры, такие как язык, валюта и список групп для публикации. Модуль поддерживает как рекламные кампании "kazarinov", так и "aliexpress".

## Подробней

Данный модуль является частью системы автоматизации маркетинга в Facebook. Он использует драйвер браузера для доступа к Facebook и публикации рекламных объявлений в соответствии с предварительно заданными настройками. Модуль предназначен для работы с несколькими языками и валютами, а также поддерживает различные типы рекламных кампаний.

## Функции

### `run_campaign`

```python
def run_campaign(d: Driver, promoter_name: str, campaigns: list | str, group_file_paths: list, language: str, currency: str):
    """Запуск рекламной кампании.

    Args:
        d (Driver): Экземпляр драйвера.
        promoter_name (str): Имя рекламодателя.
        campaigns (list): Список кампаний.
        group_file_paths (list): Пути к файлам с группами.
        language (str): Язык рекламной кампании.
        currency (str): Валюта рекламной кампании.
    """
```

**Описание**: Запускает рекламную кампанию с заданными параметрами, используя указанный драйвер и имя рекламодателя.

**Параметры**:
- `d` (Driver): Экземпляр драйвера, используемый для управления браузером.
- `promoter_name` (str): Имя рекламодателя.
- `campaigns` (list | str): Список кампаний для запуска.
- `group_file_paths` (list): Список путей к файлам, содержащим информацию о группах Facebook.
- `language` (str): Язык, на котором будет отображаться реклама.
- `currency` (str): Валюта, используемая в рекламной кампании.

**Возвращает**: None

**Вызывает исключения**: None
