# Модуль hypotez/src/gui/header.py

## Обзор

Этот модуль определяет корневой путь к проекту `hypotez`.  Все импорты строятся относительно этого пути.  В нем также происходит загрузка настроек из файла `settings.json` и документации из файла `README.MD` находящихся в корне проекта.

## Оглавление

- [Функции](#функции)
    - [get_project_root](#get_project_root)


## Функции

### `get_project_root`

**Описание**: Находит корневую директорию проекта, начиная от директории текущего файла, поднимаясь вверх по иерархии директорий, и останавливаясь на первой директории, содержащей один из указанных файлов-маркеров.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для определения корневой директории проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории, если найдена, в противном случае — директория, где расположен текущий скрипт.

**Пример использования**:

```python
root_path = get_project_root()
print(root_path)
```

**Возможные исключения**:

Не обрабатываются.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `get_project_root`.  Тип `Path`.

### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json`.  Тип `dict` или `None`, если файл не найден или содержит невалидный JSON.

### `doc_str`

**Описание**:  Строка с текстом документации, загруженная из файла `README.MD`. Тип `str` или `None`, если файл не найден или содержит невалидный текст.


### `__project_name__`

**Описание**: Имя проекта. Тип `str`.  Получается из настроек проекта (`settings`) или имеет значение 'hypotez' по умолчанию.

### `__version__`

**Описание**: Версия проекта. Тип `str`. Получается из настроек проекта или значение пустая строка по умолчанию.

### `__doc__`

**Описание**: Документация проекта. Тип `str`. Получается из `doc_str` или значение пустая строка по умолчанию.

### `__details__`

**Описание**: Дополнительные детали проекта. Тип `str`. Значение пустая строка по умолчанию.

### `__author__`

**Описание**: Автор проекта. Тип `str`. Получается из настроек проекта или значение пустая строка по умолчанию.

### `__copyright__`

**Описание**: Авторские права. Тип `str`. Получается из настроек проекта или значение пустая строка по умолчанию.

### `__cofee__`

**Описание**: Ссылка на бусти, где можно поддержать разработчика. Тип `str`.  Получается из настроек проекта или значение по умолчанию.
