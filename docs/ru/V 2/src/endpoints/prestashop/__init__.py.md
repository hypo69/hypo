# Модуль `src.endpoints.prestashop`

## Обзор

Модуль содержит инициализацию для подмодуля `prestashop`.

## Содержание

- [Обзор](#обзор)
- [Модули](#модули)

## Модули

Модуль импортирует следующие подмодули, но на данный момент они закомментированы.

<!--
- `product_fields`: Закомментированный импорт для работы с полями продукта.
- `api`: Закомментированный импорт для работы с API PrestaShop (синхронная и асинхронная версии).
- `product_async`: Закомментированный импорт для асинхронной работы с продуктами.
- `supplier`: Закомментированный импорт для работы с поставщиками.
- `category`: Закомментированный импорт для работы с категориями (синхронная и асинхронная версии).
- `warehouse`: Закомментированный импорт для работы со складами.
- `language`: Закомментированный импорт для работы с языками.
- `shop`: Закомментированный импорт для работы с магазинами.
- `pricelist`: Закомментированный импорт для работы с прайс-листами.
- `customer`: Закомментированный импорт для работы с клиентами.
-->

**Примечание**: На данный момент ни один из подмодулей не импортируется, что означает, что они не будут доступны для использования. Для их использования необходимо раскомментировать соответствующие строки импорта.
