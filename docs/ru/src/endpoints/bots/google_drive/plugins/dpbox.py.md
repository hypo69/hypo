# Модуль для преобразования ссылок Dropbox

## Обзор

Модуль содержит функцию `DPBOX`, которая преобразует различные типы ссылок Dropbox в прямые ссылки для скачивания.

## Подробней

Функция `DPBOX` анализирует URL-адрес Dropbox и преобразует его в прямую ссылку для скачивания, добавляя или изменяя параметр `dl=1`. Это позволяет пользователям напрямую скачивать файлы из Dropbox, минуя страницу предварительного просмотра.

## Функции

### `DPBOX`

```python
def DPBOX(url: str) -> str:
    """Преобразует URL-адрес Dropbox в прямую ссылку для скачивания.

    Args:
        url (str): URL-адрес Dropbox.

    Returns:
        str: Прямая ссылка для скачивания файла из Dropbox.
    """
```

**Как работает функция**:

Функция `DPBOX` принимает URL-адрес Dropbox в качестве входных данных и выполняет следующие действия:

1.  **Проверяет наличие "dl.dropbox.com" в URL**:
    *   Если "dl.dropbox.com" присутствует, функция проверяет наличие параметров "?dl=0" или "?dl=1".
    *   Если присутствует "?dl=0", заменяет его на "?dl=1".
    *   Если отсутствует и "?dl=0", и "?dl=1", добавляет "?dl=1" к URL.
2.  **Если в URL "www.dropbox.com"**:
    *   Заменяет "www.dropbox.com" на "dl.dropbox.com".
    *   Далее выполняет те же действия, что и в первом пункте, проверяя и изменяя параметры "?dl=0" и "?dl=1".
3.  **Иначе (если в URL нет ни "dl.dropbox.com", ни "www.dropbox.com")**:
    *   Проверяет наличие параметров "?dl=0" или "?dl=1".
    *   Если присутствует "?dl=0", заменяет его на "?dl=1".
    *   Если отсутствует и "?dl=0", и "?dl=1", добавляет "?dl=1" к URL.

**Параметры**:

*   `url` (str): URL-адрес Dropbox, который необходимо преобразовать.

**Возвращает**:

*   `str`: Прямая ссылка для скачивания файла из Dropbox.

**Примеры**:

```python
>>> DPBOX("https://www.dropbox.com/s/example/file.txt?dl=0")
'https://dl.dropbox.com/s/example/file.txt?dl=1'

>>> DPBOX("https://dl.dropbox.com/s/example/file.txt")
'https://dl.dropbox.com/s/example/file.txt?dl=1'

>>> DPBOX("https://www.dropbox.com/s/example/file.txt")
'https://dl.dropbox.com/s/example/file.txt?dl=1'
```
