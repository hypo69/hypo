# Модуль `hypotez/src/suppliers/etzmaleh/header.py`

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`. Он также предоставляет доступ к информации о проекте, такой как имя, версия, описание и другие детали.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная от директории текущего файла и поднимаясь по дереву директорий, пока не найдет директорию, содержащую указанные файлы-маркеры.

**Параметры**:

- `marker_files` (tuple): Кортеж из имен файлов или директорий, которые будут использоваться для определения корневой директории проекта.  По умолчанию задано `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращает директорию, где расположен текущий скрипт.

**Вызывает исключения**:

- Нет.


## Переменные

### `__root__`

**Описание**: Переменная, содержащая путь к корневой директории проекта.  Инициализируется вызовом функции `set_project_root`.

**Тип**: `Path`


### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженный из файла `settings.json`.  Если файл не найден или содержит невалидный JSON, переменная имеет значение `None`.

**Тип**: `dict`


### `doc_str`

**Описание**: Строка, содержащая описание проекта (README.md), загруженная из файла `README.MD`. Если файл не найден или содержит невалидный текст, переменная имеет значение `None`.

**Тип**: `str`


### `__project_name__`

**Описание**: Имя проекта. Получается из настроек (`settings`) или по умолчанию установлено как 'hypotez'.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта. Получается из настроек (`settings`) или по умолчанию установлено как пустая строка.

**Тип**: `str`


### `__doc__`

**Описание**: Документация проекта (считывается из `README.MD`).

**Тип**: `str`


### `__details__`

**Описание**:  Дополнительные детали проекта (пустая строка по умолчанию).

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта. Получается из настроек (`settings`) или по умолчанию установлено как пустая строка.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права на проект. Получается из настроек (`settings`) или по умолчанию установлено как пустая строка.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка на страницу для пожертвований на чашку кофе для разработчика. Получается из настроек (`settings`) или по умолчанию установлена ссылка.

**Тип**: `str`
