# Модуль: src.suppliers.wallashop._experiments

## Обзор

Данный модуль содержит эксперименты, связанные с поставщиком Wallashop. Включает в себя различные импорты, настройки путей и функцию для запуска поставщика. Модуль, по-видимому, предназначен для экспериментов и отладки, связанных с интеграцией поставщика Wallashop.

## Подробней

Этот код содержит настройки путей, импорты библиотек и определение функции `start_supplier`. Код необходим для запуска и отладки интеграции с поставщиком Wallashop. Он настраивает пути к директориям проекта, импортирует необходимые модули и содержит функцию для инициализации поставщика.

## Функции

### `start_supplier`

def start_supplier(supplier_prefix: str = 'aliexpress', locale: str = 'en') -> Supplier:
    """ Старт поставщика

        supplier_prefix (str, optional): Префикс поставщика. По умолчанию 'aliexpress'.
        locale (str, optional): Локаль поставщика. По умолчанию 'en'.

    Returns:
        Supplier: Объект поставщика.

    ...
```

**Назначение**: Запускает поставщика с указанным префиксом и локалью.

**Параметры**:
- `supplier_prefix` (str, optional): Префикс поставщика. По умолчанию `'aliexpress'`.
- `locale` (str, optional): Локаль поставщика. По умолчанию `'en'`.

**Возвращает**:
- `Supplier`: Объект поставщика.

**Как работает функция**:

1. Функция `start_supplier` принимает два параметра: `supplier_prefix` и `locale`, которые определяют префикс и локаль поставщика соответственно.
2. Внутри функции создается словарь `params`, содержащий переданные параметры `supplier_prefix` и `locale`.
3. Функция возвращает экземпляр класса `Supplier`, инициализированный с помощью словаря `params`. Это позволяет настроить и запустить конкретного поставщика с заданными параметрами.

**Примеры**:

```python
