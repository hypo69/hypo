# Модуль arguments

## Обзор

Модуль `arguments` содержит вспомогательные функции для обработки и валидации аргументов, используемых в API для работы с AliExpress. Он включает функции для преобразования списков в строки и извлечения идентификаторов продуктов.

## Подробней

Этот модуль предоставляет функции, которые помогают стандартизировать входные данные для API AliExpress. `get_list_as_string` преобразует список элементов в строку, разделенную запятыми, что полезно для передачи нескольких значений в одном параметре запроса. `get_product_ids` извлекает и валидирует идентификаторы продуктов из списка или строки, обеспечивая правильность данных перед отправкой запроса к API.
Расположение файла в проекте указывает на то, что он используется для подготовки аргументов при запросах к API AliExpress.

## Функции

### `get_list_as_string`

```python
def get_list_as_string(value):
    """
    Args:
        value: 

    Returns:
        None:
    """
```

**Описание**: Преобразует входное значение в строку, если это список или строка. Если значение равно `None`, функция возвращает `None`.

**Параметры**:
- `value`: Значение для преобразования. Может быть строкой, списком или `None`.

**Возвращает**:
- `str | None`: Строка, полученная из списка, разделенного запятыми, или исходная строка, если на входе была строка. Возвращает `None`, если входное значение `None`.

**Вызывает исключения**:
- `InvalidArgumentException`: Если аргумент не является строкой, списком или `None`.

**Примеры**:
```python
# Пример 1: Преобразование списка в строку
input_list = ['item1', 'item2', 'item3']
result = get_list_as_string(input_list)
print(result)  # Вывод: 'item1,item2,item3'

# Пример 2: Возвращение исходной строки
input_string = 'already_a_string'
result = get_list_as_string(input_string)
print(result)  # Вывод: 'already_a_string'

# Пример 3: Обработка значения None
input_none = None
result = get_list_as_string(input_none)
print(result)  # Вывод: None
```

### `get_product_ids`

```python
def get_product_ids(values):
    """
    Args:
        values: 

    Returns:
        list:
    """
```

**Описание**: Извлекает и валидирует идентификаторы продуктов из списка или строки, разделенной запятыми.

**Параметры**:
- `values`: Список или строка идентификаторов продуктов.

**Возвращает**:
- `list[str]`: Список идентификаторов продуктов после обработки.

**Вызывает исключения**:
- `InvalidArgumentException`: Если аргумент не является списком или строкой.

**Примеры**:
```python
# Пример 1: Получение идентификаторов продуктов из строки
input_string = "12345,67890"
product_ids = get_product_ids(input_string)
print(product_ids) # Вывод: ['12345', '67890']

# Пример 2: Получение идентификаторов продуктов из списка
input_list = ["12345", "67890"]
product_ids = get_product_ids(input_list)
print(product_ids) # Вывод: ['12345', '67890']
