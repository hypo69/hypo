## Функции

### `assemble_computer`

**Описание**: Функция принимает JSON-данные о компонентах компьютера и генерирует описание сборки на русском и иврите, включая определение типа сборки (геймерская, офисная, рабочая станция) и перевод названий компонентов.

**Параметры**:

- `input_data` (dict): Словарь с JSON-данными о компонентах компьютера, например: [{"product_id": "<id>", "product_title": "<name>", "product_description": "<description>", "image_local_saved_path": "<path>"}]
- `locale` (str, optional): Язык вывода (по умолчанию `ru`). Допустимые значения: `ru`, `he`.


**Возвращает**:

- dict: Словарь с результатами, содержащий описания сборки на русском и иврите. Структура аналогична предоставленному в примере выходному формату JSON.

**Вызывает исключения**:

- `ValueError`: Если входные данные имеют некорректный формат.
- `TypeError`: Если входные данные содержат несовместимые типы.
- `TranslationError`: Если не удалось перевести какие-либо термины.
