# Модуль: Помощник по коду

Этот модуль содержит класс `CodeAssistant`, предназначенный для взаимодействия с различными моделями ИИ, такими как Google Gemini и OpenAI, для задач обработки кода.  Модуль предназначен для анализа и генерации документации для кода на Python.


## Пример использования

Пример использования класса `CodeAssistant`:

```python
assistant = CodeAssistant(role='code_checker', lang='ru', model=['gemini'])
result = assistant.process_files(files=['file1.py', 'file2.py'], options={'verbose': True})
print(result)
```


## Поддерживаемые платформы

- Linux
- macOS
- Windows


## Синтаксис

Модуль поддерживает стандартный синтаксис Python.


## Классы

### `CodeAssistant`

Класс `CodeAssistant` используется для взаимодействия с различными моделями ИИ, такими как Google Gemini, и предоставляет методы для анализа и генерации документации для кода.


#### Атрибуты

- `role`: Роль помощника (например, 'code_checker').
- `lang`: Язык, который будет использовать помощник (например, 'ru').
- `model`: Список используемых моделей ИИ (например, `['gemini']`).


#### Методы

##### `process_files`

Метод для обработки файлов кода.

```python
def process_files(self, files: list, options: dict = None) -> list | None:
    """
    Args:
        files (list): Список файлов для обработки.
        options (dict, optional): Дополнительные параметры для настройки обработки. По умолчанию None.

    Returns:
        list | None: Результат обработки в виде списка проанализированных данных. Возвращает None в случае ошибки.

    Raises:
        FileNotFoundError: Если файл не найден.
        ValueError: Если передан некорректный тип данных для параметра files.
        Exception: Общее исключение.
    """
    pass
```

**Описание:** Метод `process_files` принимает список файлов и опции для настройки обработки.  Обрабатывает каждый файл в списке, используя выбранные модели ИИ, и возвращает список результатов.  Обрабатывает исключения, связанные с файлами и параметрами.

**Параметры:**

- `files` (list): Список путей к файлам.
- `options` (dict, optional): Словарь с дополнительными опциями для обработки.


**Возвращает:**

- list: Список результатов обработки каждого файла.

**Возможные исключения:**

- `FileNotFoundError`: Если указанный файл не найден.
- `ValueError`: Если тип данных `files` не соответствует ожидаемому.
- `Exception`: В случае возникновения других ошибок.
