##  Установка пути

**Описание**:  Функционал установки корня проекта в системный путь.


```python
import sys,os
from pathlib import Path
__root__ : Path = os.getcwd() [:os.getcwd().rfind(r'hypotez')+7]
sys.path.append (__root__)
```

**Описание**:  Этот код устанавливает переменную `__root__` как родительскую директорию текущего рабочего каталога, содержащую папку `hypotez`.  Затем, он добавляет этот путь в системный путь поиска модулей (`sys.path`), что позволяет импортировать модули из родительской директории.  Эта операция важна для приложений, которые могут использовать модули из родительского каталога.
