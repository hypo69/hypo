# Модуль src.suppliers.ivory._experiments

## Обзор

Модуль содержит различные эксперименты и наработки, связанные с поставщиком ivory. Включает в себя импорты необходимых библиотек и классов для работы с веб-драйвером, поставщиками, продуктами, категориями и утилитами.

## Подробней

Этот модуль, вероятно, используется для тестирования и разработки новых функций или улучшений для поставщика ivory. Он включает в себя настройку путей для импорта модулей из проекта `hypotez` и содержит функции для запуска поставщика.

## Функции

### `start_supplier`

```python
def start_supplier(supplier_prefix: str = 'aliexpress', locale: str = 'en'):
    """ Старт поставщика
    Args:
        supplier_prefix (str, optional): Префикс поставщика. По умолчанию 'aliexpress'.
        locale (str, optional): Локаль поставщика. По умолчанию 'en'.

    Returns:
        Supplier: Инстанс класса `Supplier`.

    """
```

**Назначение**: Функция создает и возвращает инстанс класса `Supplier` с заданными параметрами.

**Параметры**:
- `supplier_prefix` (str, optional): Префикс поставщика, используемый для определения конкретного поставщика. По умолчанию `'aliexpress'`.
- `locale` (str, optional): Локаль поставщика, определяющая язык и региональные настройки. По умолчанию `'en'`.

**Возвращает**:
- `Supplier`: Объект класса `Supplier`, инициализированный с переданными параметрами.

**Как работает функция**:

1.  Функция принимает два параметра: `supplier_prefix` и `locale`, которые определяют, какого поставщика и с какой локалью нужно запустить.
2.  Создается словарь `params`, содержащий эти параметры.
3.  Создается и возвращается экземпляр класса `Supplier` с использованием параметров из словаря `params`.

```python
params: dict = {
        'supplier_prefix': supplier_prefix,
        'locale': locale
    }
```

Где:

*   `supplier_prefix` - префикс поставщика.
*   `locale` - локаль поставщика.

```python
return Supplier(**params)
