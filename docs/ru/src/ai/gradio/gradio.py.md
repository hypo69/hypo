# Модуль для запуска Gradio интерфейса

## Обзор

Модуль содержит минимальный пример использования библиотеки Gradio для создания простого веб-интерфейса. Этот интерфейс принимает текстовый ввод от пользователя и возвращает приветствие.

## Подробнее

Этот модуль демонстрирует базовый способ создания интерактивного веб-приложения с использованием Gradio. Он полезен для быстрой демонстрации моделей машинного обучения или других функций, требующих пользовательского ввода.

## Функции

### `greet`

```python
def greet(name: str) -> str:
    """
    Возвращает приветствие для заданного имени.

    Args:
        name (str): Имя пользователя.

    Returns:
        str: Приветствие в формате "Hello [имя]!".

    Example:
        >>> greet("World")
        'Hello World!'
    """
```

**Назначение**: Функция `greet` принимает имя пользователя в качестве входных данных и возвращает приветствие, содержащее это имя.

**Параметры**:
- `name` (str): Имя, которое нужно включить в приветствие.

**Возвращает**:
- `str`: Строка приветствия.

**Как работает функция**:

1. Функция принимает строковый аргумент `name`.
2. Формирует строку приветствия "Hello " + `name` + "!".
3. Возвращает сформированную строку.

```text
Имя пользователя
   ↓
Формирование приветствия ("Hello " + name + "!")
   ↓
Возврат приветствия
```

**Примеры**:

```python
>>> greet("World")
'Hello World!'
>>> greet("Alice")
'Hello Alice!'
```

### `demo`

```python
demo = gr.Interface(fn=greet, inputs="text", outputs="text")
```

**Описание**: Создает экземпляр интерфейса Gradio, который связывает функцию `greet` с текстовым полем ввода и текстовым полем вывода.

**Принцип работы**:
   - Использует `gr.Interface` для создания веб-интерфейса.
   - Параметр `fn` указывает функцию, которая будет вызываться при взаимодействии с интерфейсом (в данном случае, `greet`).
   - Параметр `inputs="text"` указывает, что интерфейс должен содержать текстовое поле для ввода.
   - Параметр `outputs="text"` указывает, что интерфейс должен отображать текстовый вывод.

### `demo.launch()`

```python
demo.launch()
```

**Описание**: Запускает веб-интерфейс Gradio.

**Принцип работы**:
   - Метод `launch()` запускает локальный веб-сервер, на котором размещается интерфейс Gradio.
   - Пользователь может взаимодействовать с интерфейсом через веб-браузер.

## Порядок работы кода:
1. **Определение функции `greet`**:
   - Определяется функция `greet`, которая принимает имя пользователя в качестве входных данных и возвращает приветствие.
2. **Создание интерфейса Gradio**:
   - Создается экземпляр `gr.Interface`, который связывает функцию `greet` с текстовыми полями ввода и вывода.
3. **Запуск интерфейса Gradio**:
   - Вызывается метод `launch()` для запуска локального веб-сервера и отображения интерфейса в браузере.
