# Игра "Buzzword"

## Обзор

Этот модуль представляет собой реализацию простой игры "Buzzword", в которой компьютер выбирает случайное слово из списка, перемешивает его буквы и предлагает пользователю угадать исходное слово.

## Функции

### `shuffle_word`

**Описание**:
Функция для перемешивания букв слова.

**Параметры**:
- `word` (str): Слово, буквы которого нужно перемешать.

**Возвращает**:
- `str`: Слово с перемешанными буквами.

## Переменные

### `buzzwords`

**Описание**:
Список слов, из которых выбирается случайное слово для игры.

### `target_word`

**Описание**:
Случайно выбранное слово из списка `buzzwords`.

### `shuffled_word`

**Описание**:
Слово `target_word` с перемешанными буквами, созданное с помощью функции `shuffle_word`.

## Основной цикл игры

- Пользователю выводится перемешанное слово.
- Пользователь вводит свой вариант ответа.
- Если ответ пользователя совпадает с исходным словом, игра завершается с сообщением о победе.
- В противном случае пользователю предлагается попробовать еще раз.

## Разбор кода

1.  **Импорт модуля `random`**:
    - `import random`: Импортирует модуль random, который используется для случайного выбора слова и перемешивания букв.
2.  **Список слов `buzzwords`**:
    - `buzzwords = ["COMPUTER", "PROGRAM", "SYSTEM", "SOFTWARE", "ALGORITHM"]`: Список слов, из которого будет выбрано слово для игры.
3.  **Выбор случайного слова**:
    - `target_word = random.choice(buzzwords)`: Случайный выбор слова из списка и его сохранение в переменной `target_word`.
4.  **Функция `shuffle_word(word)`**:
    - Функция, принимающая слово и возвращающая его с перемешанными буквами.
    - `word_list = list(word)`: Преобразует слово в список символов.
    - `random.shuffle(word_list)`: Случайным образом перемешивает символы в списке.
    - `return ''.join(word_list)`: Возвращает перемешанные символы как строку.
5.  **Перемешивание слова**:
    - `shuffled_word = shuffle_word(target_word)`: Вызов функции `shuffle_word` для перемешивания букв выбранного слова.
6.  **Вывод перемешанного слова**:
    - `print("נחש את המילה:")`: Вывод сообщения пользователю.
    - `print(shuffled_word)`: Вывод перемешанного слова, которое нужно угадать.
7.  **Игровой цикл `while True`**:
    - Бесконечный цикл, который продолжается до тех пор, пока пользователь не угадает слово.
    - `user_guess = input("הכנס את הניחוש שלך: ").upper()`: Запрос ввода пользователя и преобразование его в верхний регистр.
    - **Проверка ответа**:
        - `if user_guess == target_word:`: Проверка, совпадает ли ввод пользователя с исходным словом.
        - `print("כל הכבוד! ניחשת נכון!")`: Вывод сообщения о победе, если пользователь угадал слово.
        - `break`: Завершение цикла (и игры) при правильном ответе.
    - `else:`: Если ответ неверный, выводится сообщение "ניסיון נוסף...", и цикл продолжается.
