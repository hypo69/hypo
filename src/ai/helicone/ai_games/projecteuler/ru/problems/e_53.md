# Задача No 53
## Комбинаторные выборы

## Условия
Существует десять способов выбрать два объекта из пяти, 12345:

12, 13, 14, 15, 23, 24, 25, 34, 35, 45

В общем виде, количество способов выбрать *r* объектов из *n* объектов, также известное как комбинации, записывается как:

nCr = n! / (r! * (n-r)!), где n! = n × (n-1) × ... × 3 × 2 × 1.

В частности,

5C3 = 5! / (3! * 2!) = 10.

Считается, что существует более миллиона таких комбинаций, значения которых превышают один миллион.

Сколько существует таких комбинаций *nCr*, для 1 ≤ *n* ≤ 100, значения которых превышают один миллион?
