# Модуль `hypotez/src/suppliers/wallashop/header.py`

## Обзор

Данный модуль содержит функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и получения описания проекта из файла `README.MD`.  Он устанавливает переменные, содержащие информацию о проекте, такие как название, версия, описание, автор, авторские права и ссылку на пожертвования.


## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная от текущей директории файла. Поиск происходит вверх по иерархии директорий до тех пор, пока не будет найдена директория, содержащая один из указанных файлов или папок.

**Параметры**:

- `marker_files` (tuple): Кортеж из имён файлов или папок, по наличию которых в директории будет определяться корень проекта.


**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращает директорию, в которой расположен текущий файл.


**Вызывает исключения**:

-  Не вызывает исключений.


### `set_project_root`


**Описание**: Функция `set_project_root` ищет корневой каталог проекта, начиная с текущей директории.

**Параметры**:

- `marker_files` (tuple):  Список файлов и папок, которые указывают на корневую директорию проекта.


**Возвращает**:

- `Path`: Путь к корневой директории. Если корневая директория не найдена, возвращает директорию текущего файла.


**Вызывает исключения**:

- Не вызывает исключений.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root`.  


### `settings`

**Описание**: Словарь настроек, загруженный из файла `settings.json` в корневой директории проекта. Если файл не найден или содержит некорректные данные, `settings` будет `None`.


### `doc_str`

**Описание**: Строка с описанием проекта, прочитанная из файла `README.MD` в корневой директории проекта. Если файл не найден, `doc_str` будет `None`.

### `__project_name__`

**Описание**: Название проекта, полученное из настроек `settings`, по умолчанию `hypotez`.

### `__version__`

**Описание**: Версия проекта, полученное из настроек `settings`, по умолчанию пустая строка.


### `__doc__`

**Описание**: Описание проекта, полученное из файла `README.MD`, по умолчанию пустая строка.


### `__details__`

**Описание**: Дополнительные детали о проекте, по умолчанию пустая строка.


### `__author__`

**Описание**: Автор проекта, полученное из настроек `settings`, по умолчанию пустая строка.

### `__copyright__`

**Описание**: Авторские права на проект, полученное из настроек `settings`, по умолчанию пустая строка.

### `__cofee__`

**Описание**: Ссылка на страницу для пожертвований на кофе разработчику, полученное из настроек `settings`, по умолчанию содержит ссылку на boosty.to/hypo69.
