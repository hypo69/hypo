# Модуль `hypotez/src/goog/spreadsheet`

## Обзор

Данный модуль предоставляет классы для работы с Google Spreadsheets. Он содержит классы `SpreadSheet` и `ReachSpreadsheet`, предоставляющие интерфейсы для взаимодействия с различными аспектами электронной таблицы Google.
