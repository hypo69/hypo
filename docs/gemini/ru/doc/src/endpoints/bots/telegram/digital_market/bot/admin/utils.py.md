# Модуль utils

## Обзор

Модуль `utils` содержит функцию `process_dell_text_msg`, которая предназначена для удаления сообщений в Telegram боте. Функция удаляет как последнее сообщение, сохраненное в FSMContext, так и текущее сообщение пользователя, инициировавшее вызов функции.

## Подробней

Функция `process_dell_text_msg` используется для очистки истории сообщений в чате с ботом, что может быть полезно для улучшения пользовательского опыта и поддержания порядка в диалоге. Идентификатор последнего сообщения, которое нужно удалить, берется из хранилища данных FSMContext.
