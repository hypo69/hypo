# Задача No 345
## Сумма в матрице
## Условия
Задан квадратный массив размером n×n, состоящий из неотрицательных целых чисел.
Выберите по одному числу из каждой строки так, чтобы сумма этих n чисел была максимальной.

Например, рассмотрим следующий массив размером 4×4:

```
  1  2  3  4
  2  3  4  5
  3  4  5  6
  4  5  6  7
```

Наибольшая возможная сумма, достигаемая путём выбора по одному числу из каждой строки, равна 4 + 5 + 6 + 7 = 22.

Найдите наибольшую возможную сумму, полученную путём выбора по одному числу из каждой строки для заданного массива matrix.txt, содержащего массив размером 15×15.
