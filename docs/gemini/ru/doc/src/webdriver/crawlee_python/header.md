# Модуль `hypotez/src/webdriver/crawlee_python/header.py`

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`.  Также, он предоставляет доступ к информации о проекте, такой как имя, версия, документация, автор и информация об авторах.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная от текущей директории и продвигаясь вверх по дереву директорий, пока не найдет директорию, содержащую один из указанных файлов или директорий.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для определения корневой директории проекта. По умолчанию: `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена, в противном случае путь к директории, в которой расположен скрипт.

**Обрабатываемые исключения**:

- Нет указанных исключений.


### Загрузка настроек

**Описание**:  Модуль пытается загрузить настройки из файла `settings.json` в переменную `settings`.

**Обрабатываемые исключения**:

- `FileNotFoundError`: Возникает, если файл `settings.json` не найден.
- `json.JSONDecodeError`: Возникает, если файл `settings.json` содержит недопустимый JSON.

### Загрузка документации

**Описание**: Модуль пытается загрузить описание проекта из файла `README.MD` в переменную `doc_str`.

**Обрабатываемые исключения**:

- `FileNotFoundError`: Возникает, если файл `README.MD` не найден.
- `json.JSONDecodeError`: Возникает, если файл `README.MD` содержит недопустимый формат.


## Переменные

### `__root__`

**Описание**:  Путь к корневой директории проекта.

**Тип**: `Path`

### `settings`

**Описание**: Словарь с настройками проекта.

**Тип**: `dict` или `None`

### `doc_str`

**Описание**: Строка с описанием проекта, полученная из файла `README.MD`.

**Тип**: `str` или `None`

### `__project_name__`

**Описание**: Имя проекта. Значение берется из `settings['project_name']`, или устанавливается по умолчанию `'hypotez'`, если `settings` равен `None`.

**Тип**: `str`

### `__version__`

**Описание**: Версия проекта. Значение берется из `settings['version']`, или устанавливается по умолчанию `''`, если `settings` равен `None`.

**Тип**: `str`

### `__doc__`

**Описание**: Документация проекта. Значение берется из `doc_str`, или устанавливается по умолчанию `''`, если `doc_str` равен `None`.

**Тип**: `str`

### `__details__`

**Описание**: Детали проекта.

**Тип**: `str`

### `__author__`

**Описание**: Автор проекта. Значение берется из `settings['author']`, или устанавливается по умолчанию `''`, если `settings` равен `None`.

**Тип**: `str`

### `__copyright__`

**Описание**: Авторские права проекта. Значение берется из `settings['copyrihgnt']`, или устанавливается по умолчанию `''`, если `settings` равен `None`.

**Тип**: `str`

### `__cofee__`

**Описание**: Ссылка для поддержки разработчика чашкой кофе. Значение берется из `settings['cofee']`, или устанавливается по умолчанию `Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69`

**Тип**: `str`
