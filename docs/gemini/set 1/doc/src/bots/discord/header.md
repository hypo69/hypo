# Модуль `hypotez/src/bots/discord/header.py`

## Обзор

Данный модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`.  Также он предоставляет доступ к документации проекта (README.md).

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная от директории текущего файла. Поиск происходит вверх по дереву директорий, пока не будет найдена директория, содержащая один из указанных маркеров (файлы или директории).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, которые используются для определения корневой директории проекта. По умолчанию - `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта. Если корневая директория не найдена, возвращается путь к директории, где находится текущий файл.

**Вызывает исключения**:

-  Не вызывает никаких исключений.


### `<неопределенные функции>`

**Описание**:  Необходимо добавить описание функций, которые еще не описаны, если они существуют в коде.

## Переменные

### `MODE`

**Описание**: Переменная, хранящая режим работы приложения. В данном примере значение - 'dev'.

### `__root__`

**Описание**:  Путь к корневой директории проекта, полученный с помощью функции `set_project_root()`.

### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженные из файла `settings.json`.  Значение по умолчанию - `None`.


### `doc_str`

**Описание**: Строка, содержащая содержимое файла `README.MD`, если он найден. Значение по умолчанию - `None`.


### `__project_name__`

**Описание**: Имя проекта. Загружается из настроек `settings`, значение по умолчанию - 'hypotez'.

### `__version__`

**Описание**: Версия проекта. Загружается из настроек `settings`, значение по умолчанию - ''.

### `__doc__`

**Описание**: Документация проекта (содержимое README.MD), если файл найден. Значение по умолчанию - ''.

### `__details__`

**Описание**: Дополнительные детали о проекте. Значение по умолчанию - ''.

### `__author__`

**Описание**: Автор проекта. Загружается из настроек `settings`, значение по умолчанию - ''.

### `__copyright__`

**Описание**: Авторские права на проект. Загружается из настроек `settings`, значение по умолчанию - ''.

### `__cofee__`

**Описание**: Ссылка на способ поддержать разработчиков. Загружается из настроек `settings`, значение по умолчанию - ссылка на Boosty.


## Обработка исключений

### Обработка `FileNotFoundError` и `json.JSONDecodeError`

**Описание**:  В коде присутствуют блоки `try...except`, которые обрабатывают исключения `FileNotFoundError` и `json.JSONDecodeError` при чтении файлов `settings.json` и `README.MD`. В случае возникновения исключения, соответствующие переменные (settings, doc_str) остаются со значениями по умолчанию, что предотвращает падение программы.
