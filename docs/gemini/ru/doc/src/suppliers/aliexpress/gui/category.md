# Модуль `hypotez/src/suppliers/aliexpress/gui/category.py`

## Обзор

Данный модуль предоставляет интерфейс для подготовки рекламных кампаний на AliExpress.  Он реализует класс `CategoryEditor`, позволяющий загружать JSON-файлы с данными кампаний, отображать информацию о них и запускать асинхронные операции подготовки категорий.

## Классы

### `CategoryEditor`

**Описание**: Класс `CategoryEditor` представляет собой окно для работы с категориями рекламных кампаний.  Он позволяет загружать JSON-файлы, отображать данные кампаний и запускать асинхронные задачи подготовки категорий.

**Атрибуты**:

- `campaign_name`: Имя кампании (str).
- `data`: Данные кампании в формате `SimpleNamespace` (SimpleNamespace).
- `language`: Язык кампании (str), по умолчанию 'EN'.
- `currency`: Валюта кампании (str), по умолчанию 'USD'.
- `file_path`: Путь к загруженному файлу (str).
- `editor`: Экземпляр класса `AliCampaignEditor` для работы с кампанией.

**Методы**:

#### `__init__(self, parent=None, main_app=None)`

**Описание**: Инициализирует главное окно.

**Параметры**:
- `parent` (optional): Родительское окно (QWidget).
- `main_app` (optional): Экземпляр основного приложения (MainApp).

**Возвращает**:  None.


#### `setup_ui(self)`

**Описание**: Настраивает пользовательский интерфейс.

**Возвращает**: None.

#### `setup_connections(self)`

**Описание**: Устанавливает связи между элементами интерфейса.

**Возвращает**: None.


#### `open_file(self)`

**Описание**: Открывает диалоговое окно для выбора JSON-файла.

**Возвращает**: None.


#### `load_file(self, campaign_file)`

**Описание**: Загружает JSON-файл и инициализирует необходимые данные.

**Параметры**:
- `campaign_file` (str): Путь к JSON-файлу.

**Возвращает**: None.

**Обрабатывает исключения**:
- `Exception`:  Возникает при ошибке загрузки JSON-файла. Ошибка выводится в диалоговом окне.


#### `create_widgets(self, data)`

**Описание**: Создает виджеты на основе загруженных данных.

**Параметры**:
- `data` (SimpleNamespace): Данные кампании.

**Возвращает**: None.


#### `prepare_all_categories_async(self)`

**Описание**: Асинхронно подготавливает все категории.

**Возвращает**: None.

**Обрабатывает исключения**:
- `Exception`: Возникает при ошибке подготовки категорий. Ошибка выводится в диалоговом окне.


#### `prepare_category_async(self)`

**Описание**: Асинхронно подготавливает определенную категорию.

**Возвращает**: None.

**Обрабатывает исключения**:
- `Exception`: Возникает при ошибке подготовки категории. Ошибка выводится в диалоговом окне.
