# Модуль hypotez/src/logger/header.py

## Обзор

Этот модуль отвечает за инициализацию и настройку параметров проекта, такие как корневой путь, настройки из файла `settings.json`, и документацию из файла `README.MD`.  Он также добавляет корневой путь к системе поиска модулей `sys.path`.

## Функции

### `set_project_root`

**Описание**:  Находит корневой каталог проекта, начиная от текущего файла и двигаясь вверх по директориям. Останавливается на первой директории, содержащей один из заданных файлов-маркеров (например, `pyproject.toml`, `requirements.txt`, `.git`). Если корневой каталог не найден, возвращает текущую директорию, в которой находится скрипт.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, которые используются для определения корневого каталога проекта.

**Возвращает**:

- `Path`: Путь к корневому каталогу проекта, если найден, иначе - путь к текущей директории.

**Вызывает исключения**:

-  Не вызывает никаких исключений.


### `__main__`

**Описание**:  (не функция, а блок кода)
Этот блок кода выполняется, когда скрипт запускается напрямую.  Он находит корневой путь проекта, загружает настройки из `settings.json` и документацию из `README.MD`,  и инициализирует соответствующие переменные.

**Параметры**:

- Нет явных параметров.

**Возвращает**:

- Нет явного возвращаемого значения.

**Вызывает исключения**:

- `FileNotFoundError`: Возникает, если файлы `settings.json` или `README.MD` не найдены.
- `json.JSONDecodeError`: Возникает, если содержимое `settings.json` не может быть декодировано как JSON.


## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта. Инициализируется функцией `set_project_root`. Тип: `Path`.

### `settings`

**Описание**: Словарь с настройками проекта, загруженными из файла `settings.json`. Тип: `dict` или `None`.

### `doc_str`

**Описание**: Строка с содержимым файла `README.MD`, если он найден. Тип: `str` или `None`.

### `__project_name__`

**Описание**: Имя проекта, взятое из настроек (`settings.json`) или установленное по умолчанию. Тип: `str`.

### `__version__`

**Описание**: Версия проекта, взятая из настроек (`settings.json`) или установленная по умолчанию. Тип: `str`.

### `__doc__`

**Описание**: Документация проекта, взятая из файла `README.MD` или установленная по умолчанию. Тип: `str`.

### `__details__`

**Описание**: Дополнительные детали проекта. Тип: `str`.

### `__author__`

**Описание**: Автор проекта, взятый из настроек (`settings.json`) или установленный по умолчанию. Тип: `str`.

### `__copyright__`

**Описание**: Авторские права проекта, взятые из настроек (`settings.json`) или установленные по умолчанию. Тип: `str`.

### `__cofee__`

**Описание**: Ссылка на поддержку разработчика. Тип: `str`.
