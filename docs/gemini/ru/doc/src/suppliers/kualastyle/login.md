# Модуль hypotez/src/suppliers/kualastyle/login.py

## Обзор

Этот модуль содержит функции для авторизации поставщика Kualastyle. Он предоставляет методы для работы с веб-драйвером, управления всплывающими окнами и обработки возможных исключений.

## Функции

### `login`

**Описание**: Функция для входа поставщика.

**Параметры**:

- `s` (Supplier): Объект, представляющий поставщика.

**Возвращает**:

- `bool`: `True`, если вход успешен, иначе `False`.

**Вызывает исключения**:

Нет


### `close_pop_up`

**Описание**: Функция для закрытия всплывающего окна.

**Параметры**:

- `s` (Supplier): Объект, представляющий поставщика.

**Возвращает**:

- `bool`: `True`, если окно закрыто, иначе `False`.

**Вызывает исключения**:

- `Exception`:  Возникает при ошибке выполнения команды закрытия всплывающего окна.  В логах записывается сообщение о том, что попап не был закрыт.
