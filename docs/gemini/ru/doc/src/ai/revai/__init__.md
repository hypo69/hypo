# Модуль hypotez/src/ai/revai

## Обзор

Данный модуль предоставляет интерфейс для взаимодействия с API сервиса Rev.ai.  Он содержит константы и, вероятно, функции для настройки и инициализации работы с API.

## Константы

### `MODE`

**Описание**:  Переменная, определяющая режим работы (например, `dev`, `prod`).

**Значение**:  `'dev'` по умолчанию.
