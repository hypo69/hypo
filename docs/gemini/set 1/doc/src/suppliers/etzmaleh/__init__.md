Данный модуль предоставляет базовые настройки и импорты для поставщика данных etzmaleh.  Он определяет константу `MODE` и импортирует класс `Graber` из подмодуля `graber`.

## Модульные константы

### `MODE`

**Описание**: Строковая константа, определяющая режим работы. В текущем случае установлено значение 'dev'.

**Описание**:  Класс `Graber` отвечает за получение данных от поставщика.  Подробная документация находится в файле `graber.py`.
