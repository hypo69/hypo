# Модуль ai

## Обзор

Модуль `ai` — это интерфейс для управления различными моделями ИИ. Он обеспечивает взаимодействие с внешними API, обрабатывает конфигурации для анализа данных и обработки естественного языка. Модуль включает в себя подмодули для работы с разными моделями ИИ.

## Подмодули

### `anthropic`

**Описание**: Обеспечивает интеграцию с моделями ИИ Anthropic.  Позволяет выполнять задачи, связанные с продвинутым пониманием языка и генерацией ответов.

[Перейти к документации модуля](https://github.com/hypo69/hypo/blob/master/src/ai/anthropic/readme.ru.md)


### `dialogflow`

**Описание**: Интегрируется с Google Dialogflow, поддерживает обработку естественного языка (NLU) и функции разговорного ИИ для создания интерактивных приложений.

[Перейти к документации модуля](https://github.com/hypo69/hypo/blob/master/src/ai/dialogflow/readme.ru.md)


### `gemini`

**Описание**: Управляет соединениями с моделями ИИ Gemini. Предоставляет поддержку для приложений, которые требуют уникальных возможностей ИИ Gemini.

[Перейти к документации модуля](https://github.com/hypo69/hypo/blob/master/src/ai/gemini/readme.ru.md)


### `helicone`

**Описание**: Подключается к моделям Helicone, предоставляя доступ к специализированным функциям для настройки решений на базе ИИ.

[Перейти к документации модуля](https://github.com/hypo69/hypo/blob/master/src/ai/helicone/readme.ru.md)


### `llama`

**Описание**: Интерфейс для LLaMA (Large Language Model Meta AI). Предназначен для задач, связанных с пониманием и генерацией естественного языка в различных приложениях.

[Перейти к документации модуля](https://github.com/hypo69/hypo/blob/master/src/ai/llama/readme.ru.md)


### `myai`

**Описание**: Кастомный подмодуль ИИ, разработанный для специализированных конфигураций моделей и реализации, обеспечивающий уникальные функции ИИ, специфичные для проекта.

[Перейти к документации модуля](https://github.com/hypo69/hypo/blob/master/src/ai/myai/readme.ru.md)


### `openai`

**Описание**: Интегрируется с API OpenAI, предоставляя доступ к их набору моделей (например, GPT) для задач, таких как генерация текста, классификация, перевод и другие.

[Перейти к документации модуля](https://github.com/hypo69/hypo/blob/master/src/ai/openai/readme.ru.md)


### `tiny_troupe`

**Описание**: Обеспечивает интеграцию с моделями ИИ от Microsoft, предлагая решения для обработки естественного языка и задач анализа данных с использованием маленьких моделей, оптимизированных для производительности.

[Перейти к документации модуля](https://github.com/hypo69/hypo/blob/master/src/ai/tiny_troupe/readme.ru.md)


### `revai`

**Описание**: Интегрируется с моделью от rev.com, которая специализируется на работе с аудиофайлами, такими как записи переговоров, совещаний, звонков и других аудио-материалов.

[Перейти к документации модуля](https://github.com/hypo69/hypo/blob/master/src/ai/revai/readme.ru.md)


### `prompts`

**Описание**:  Системные и командные промпты в формате Markdown для моделей ИИ.
