# Модуль translate_product_fields

## Обзор

Данный модуль предоставляет функции для работы с переводами полей товаров. Он обеспечивает связь между словарем полей товара, таблицей переводов и переводчиками.  Модуль позволяет получать, вставлять и переводить записи о переводах.


## Функции

### `get_translations_from_presta_translations_table`

**Описание**: Функция извлекает переводы полей товара из таблицы переводов PrestaShop.

**Параметры**:
- `product_reference` (str): Идентификатор товара (например, `SKU`).
- `credentials` (dict): Словарь с данными для авторизации и подключения к базе данных переводов PrestaShop.
- `i18n` (str, optional): Код языка перевода (например, `en_EN`, `he_HE`, `ru-RU`). По умолчанию `None`.

**Возвращает**:
- `list`: Список словарей с переводами. Возвращает пустой список, если переводы не найдены.


**Вызывает исключения**:
- Возможные исключения, связанные с подключением к базе данных или обработкой запросов, должны быть обработаны в вызывающем коде.


### `insert_new_translation_to_presta_translations_table`

**Описание**: Функция для добавления новой записи перевода в таблицу переводов PrestaShop.

**Параметры**:
- `record` (dict): Словарь с данными для новой записи перевода.
- `credentials` (dict): Словарь с данными для авторизации и подключения к базе данных переводов PrestaShop.


**Возвращает**:
-  `None` - функция не возвращает результат.

**Вызывает исключения**:
- Возможные исключения, связанные с подключением к базе данных или обработкой запросов, должны быть обработаны в вызывающем коде.


### `translate_record`

**Описание**: Функция для перевода полей товара.

**Параметры**:
- `record` (dict): Словарь с полями товара для перевода.
- `from_locale` (str): Код исходного языка (например, `fr_FR`).
- `to_locale` (str): Код языка перевода (например, `en_EN`).

**Возвращает**:
- `dict`: Словарь с переведенными полями товара.


**Вызывает исключения**:
- Возможные исключения из функции `translate` из модуля `src.ai` (e.g., `TranslationError`).
- Возможные исключения при обработке переведенной записи.
