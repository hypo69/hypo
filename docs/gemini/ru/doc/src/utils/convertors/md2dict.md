# Модуль md2dict

## Обзор

Модуль `md2dict` предназначен для конвертации строк Markdown в структурированные словари Python.  Он поддерживает извлечение вложенного JSON содержимого, если оно присутствует в строке Markdown.


## Функции

### `md2dict`

**Описание**: Конвертирует строку Markdown в структурированный словарь.  Извлекает JSON-контент, если он найден в строке, иначе парсит Markdown и возвращает словарь секций.

**Параметры**:

- `md_string` (str): Строка Markdown для конвертации.

**Возвращает**:

- `Dict[str, dict | list]`: Структурированное представление Markdown содержимого. Возвращает словарь с ключом "json", если найден JSON контент, или словарь с секциями Markdown.


**Возможные исключения**:

- `Exception`:  Возникает при ошибках в процессе парсинга Markdown или JSON.  Подробная информация о  ошибке логгируется с помощью `logger`.


### `extract_json_from_string`

**Описание**: Извлекает JSON контент из строки, если он присутствует.

**Параметры**:

- `text` (str): Строка для поиска JSON.

**Возвращает**:

- `dict | None`: Извлеченный JSON контент или `None`, если JSON не найден.


**Возможные исключения**:

- `Exception`: Возникает при ошибках в процессе парсинга JSON.  Подробная информация о  ошибке логгируется с помощью `logger`.


## Использование

```python
from hypotez.src.utils.convertors.md2dict import md2dict

markdown_string = """
# Заголовок 1

## Заголовок 2

```json
{"key": "value"}
```
"""

result = md2dict(markdown_string)
print(result)
```

Этот пример демонстрирует использование функции `md2dict`.  В результате будет выведен словарь, содержащий JSON-контент в виде вложенного словаря.


##  Примечания

- Модуль использует библиотеку `markdown2` для парсинга Markdown.
- Для обработки ошибок и логгирования используется `logger` из модуля `src.logger.logger`.  Важно правильно настроить `logger` для корректной работы в вашем проекте.
- Используется регулярное выражение `re` для извлечения JSON из строки.
- Пример использования функции `eval` для парсинга JSON. В реальных проектах необходимо быть внимательным к возможным проблемам безопасности при использовании `eval` для парсинга данных из внешних источников.
