# Модуль `src.fast_api`

## Обзор

Модуль `src.fast_api` представляет собой точку входа для FastAPI-приложения. В данный момент модуль определяет константу `MODE` для указания режима работы приложения (разработка или продакшн).

## Оглавление

1. [Переменные](#переменные)

## Переменные

### `MODE`

**Описание**:  Константа, определяющая режим работы приложения. В данный момент установлена в `'dev'`, что указывает на режим разработки.

**Тип**: `str`
