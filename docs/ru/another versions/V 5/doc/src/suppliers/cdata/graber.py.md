# Модуль `graber.py`

## Обзор

Модуль `graber.py` предназначен для сбора значений полей на странице товара сайта `cdata.co.il`. Он расширяет функциональность родительского класса `Graber` из модуля `src.suppliers.graber`, предоставляя возможность переопределения стандартных методов обработки полей для специфических требований сайта `cdata.co.il`.
Модуль использует декоратор для выполнения предварительных действий перед отправкой запроса к веб-драйверу.

## Подробней

Этот модуль является частью системы для сбора и обработки данных о товарах с различных онлайн-магазинов. Он специализируется на сайте `cdata.co.il`, адаптируя общие механизмы сбора данных (`Graber`) к особенностям структуры этого конкретного сайта. Использование декоратора позволяет выполнять такие действия, как закрытие всплывающих окон, перед началом сбора данных, что повышает стабильность и надежность работы.

## Классы

### `Graber`

**Описание**: Класс `Graber` предназначен для операций сбора данных с сайта `cdata.co.il`. Он наследуется от класса `Graber` (Grbr) из модуля `src.suppliers.graber` и переопределяет некоторые методы для адаптации к специфике сайта.

**Как работает класс**:
1.  При инициализации класса устанавливается префикс поставщика (`supplier_prefix`) как `cdata`.
2.  Вызывается конструктор родительского класса `Graber` для инициализации общих параметров, таких как веб-драйвер и индекс языка.
3.  Устанавливается значение `Context.locator_for_decorator` в `None`, что отключает выполнение декоратора `@close_pop_up`. Если установить значение, то декоратор выполнится.

**Методы**:

*   `__init__`: Инициализирует экземпляр класса `Graber`, устанавливая префикс поставщика и вызывая конструктор родительского класса.

**Параметры**:

*   `driver` (Driver): Экземпляр веб-драйвера для взаимодействия с сайтом.
*   `lang_index` (int): Индекс языка, используемый для локализации.

### `__init__`

```python
def __init__(self, driver: Driver, lang_index:int):
    """Инициализация класса сбора полей товара."""
```

**Описание**: Инициализирует экземпляр класса `Graber`, устанавливая префикс поставщика и вызывая конструктор родительского класса.

**Как работает функция**:

1.  Устанавливает атрибут `supplier_prefix` равным `'cdata'`.
2.  Вызывает конструктор родительского класса `Graber` с указанием префикса поставщика, драйвера и индекса языка.
3.  Устанавливает значение `Context.locator_for_decorator` в `None`.

**Параметры**:

*   `driver` (Driver): Экземпляр веб-драйвера для взаимодействия с сайтом.
*   `lang_index` (int): Индекс языка, используемый для локализации.

**Примеры**:

```python
from src.webdriver.driver import Driver
from src.suppliers.cdata.graber import Graber

# Пример инициализации класса Graber
driver = Driver()  # Предполагается, что класс Driver инициализируется без параметров
lang_index = 0
graber = Graber(driver, lang_index)
```
