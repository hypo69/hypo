# Модуль `graber`

## Обзор

Модуль `graber` предназначен для сбора значений полей на странице товара веб-сайта `visualdg.co.il`. Он расширяет функциональность родительского класса `Graber` и предоставляет возможность переопределения методов обработки полей для нестандартных случаев.

## Подробней

Этот модуль является частью системы сбора данных о товарах с веб-сайтов поставщиков. Он специализируется на извлечении информации с сайта `visualdg.co.il`. Для стандартизированной обработки полей используются функции родительского класса, а для специфических случаев предоставляется возможность их переопределения. Модуль также поддерживает предварительные действия перед отправкой запроса к веб-драйверу с использованием декораторов.

## Классы

### `Graber`

**Описание**: Класс `Graber` предназначен для операций захвата данных с сайта `visualdg.co.il`. Он наследуется от класса `Graber` (Grbr) из модуля `src.suppliers.graber`.

**Как работает класс**:
1.  При инициализации класса устанавливается префикс поставщика (`supplier_prefix`) как `'visualdg'`.
2.  Вызывается конструктор родительского класса `Graber` с указанием префикса поставщика, драйвера и индекса языка.
3.  Устанавливаются глобальные настройки через класс `Context`, в частности, сбрасывается значение `Context.locator_for_decorator`.

**Методы**:

*   `__init__`: Инициализирует экземпляр класса `Graber`.

**Параметры**:

*   `driver` (`Driver`): Экземпляр веб-драйвера для управления браузером.
*   `lang_index` (int): Индекс языка, используемый при сборе данных.

### `__init__`

```python
    def __init__(self, driver: Driver, lang_index):
        """Инициализация класса сбора полей товара."""
```

**Описание**: Инициализирует экземпляр класса `Graber`.

**Как работает функция**:

1.  Устанавливает атрибут `supplier_prefix` равным `'visualdg'`.
2.  Вызывает конструктор родительского класса (`Graber` из `src.suppliers.graber`) с переданными параметрами.
3.  Устанавливает `Context.locator_for_decorator` в `None`.

**Параметры**:

*   `driver` (`Driver`): Экземпляр веб-драйвера для управления браузером.
*   `lang_index` (int): Индекс языка, используемый при сборе данных.
