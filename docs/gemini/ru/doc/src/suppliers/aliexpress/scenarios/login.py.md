# Модуль `login.py`

## Обзор

Модуль `login.py` содержит функцию `login`, которая отвечает за авторизацию пользователя на сайте AliExpress через веб-драйвер Selenium.

## Содержание

- [Функции](#функции)
    - [`login`](#login)

## Функции

### `login`

**Описание**: Функция выполняет авторизацию пользователя на сайте AliExpress с использованием веб-драйвера.

**Параметры**:
- `s` (`Supplier`): Объект поставщика с уже инициализированным веб-драйвером.

**Возвращает**:
- `bool`: Возвращает `True` в случае успешной авторизации.

**Пример**
```python
    def login(s)->bool:
        """ login to aliexpress via webdriver
```
