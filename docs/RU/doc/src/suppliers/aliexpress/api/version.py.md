# Модуль `version.py`

## Обзор

Модуль `version.py` содержит информацию о версии, имя модуля, документацию, дополнительные сведения и авторов проекта.

## Оглавление

- [Переменные](#переменные)

## Переменные

### `__name__`

**Описание**: Содержит имя модуля. Если скрипт запускается напрямую, значение будет `"__main__"`.

### `__version__`

**Описание**: Хранит версию модуля или пакета.

### `__doc__`

**Описание**: Строка документации модуля.

### `__details__`

**Описание**: Содержит дополнительные сведения о модуле или классе.

### `__annotations__`

**Описание**: Содержит аннотации типов для переменных и функций в модуле.

### `__author__`

**Описание**: Имя(имена) автора(ов) модуля.
