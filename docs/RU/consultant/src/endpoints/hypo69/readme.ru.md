**hypo69**: эндпоинты для разработчика
==============================================

**small_talk_bot** - бот с чатом модели ии
Модуль предоставляет функциональность для создания чат-бота с использованием модели искусственного интеллекта. Основное назначение - обработка и ответ на текстовые запросы пользователей.
**code_assistant** - модуль обучения модели коду проекта
Модуль предназначен для обучения модели искусственного интеллекта коду текущего проекта. Это позволяет модели лучше понимать структуру и специфику проекта.
**psychologist_bot** - ранняя разработка модуля парсинга диалогов
Модуль находится в стадии ранней разработки и предназначен для парсинга и анализа диалогов. Он может быть использован для обработки и понимания текстовых разговоров.
