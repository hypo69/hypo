# Задача No 558
##  Итерационная функция

## Условия

Определим функцию *f(n)* следующим образом:
*   *f(n)* = 0, если *n* < 2
*   *f(n)* = 1 + *f*(n - *f*(n - 1)), если *n* ≥ 2

Также определим *g(n)* = сумма *f(i)* для всех *i* от 0 до *n*.
Например, первые 10 значений *f(i)* равны 0, 0, 1, 1, 2, 3, 4, 4, 5, 6, а соответствующие первые 10 значений *g(i)* равны 0, 0, 1, 2, 4, 7, 11, 15, 20, 26.

Найдите *g*(10<sup>18</sup>) mod 1 000 000 007.
