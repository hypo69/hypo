# Модуль `hypotez/src/suppliers/header.py`

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`.  Он также обрабатывает возможные ошибки при чтении файла настроек и файла README.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с текущей директории скрипта и идя вверх по дереву директорий, пока не найдет директорию, содержащую один из указанных маркеров файлов.

**Параметры**:
- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, которые используются для определения корневой директории проекта. По умолчанию включает `pyproject.toml`, `requirements.txt` и `.git`.

**Возвращает**:
- `Path`: Объект `Path` представляющий путь к корневой директории проекта. Если корневая директория не найдена, возвращает директорию, где расположен текущий файл.

**Вызывает исключения**:
- Нет


### Загрузка настроек

**Описание**: Загружает настройки из файла `settings.json` в глобальную переменную `settings`.

**Параметры**:
- Нет

**Возвращает**:
- Нет


**Вызывает исключения**:
- `FileNotFoundError`: Если файл `settings.json` не найден.
- `json.JSONDecodeError`: Если содержимое файла `settings.json` не является валидным JSON.


### Загрузка документации

**Описание**: Загружает контент из файла `README.MD` в глобальную переменную `doc_str`.

**Параметры**:
- Нет

**Возвращает**:
- Нет

**Вызывает исключения**:
- `FileNotFoundError`: Если файл `README.MD` не найден.
- `json.JSONDecodeError`: Если содержимое файла `README.MD` не является валидным текстом.


## Глобальные переменные

### `__root__`

**Описание**: Путь к корневой директории проекта.

**Тип**: `Path`

### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженные из файла `settings.json`.

**Тип**: `dict`

### `doc_str`

**Описание**: Строка, содержащая контент файла `README.MD`.

**Тип**: `str`

### `__project_name__`

**Описание**: Название проекта, полученное из настроек или установленное по умолчанию.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта, полученная из настроек или установленная по умолчанию.

**Тип**: `str`


### `__doc__`

**Описание**: Документация проекта, полученная из файла `README.MD` или установленная по умолчанию.

**Тип**: `str`


### `__details__`

**Описание**: Подробное описание проекта, полученное из настроек или установленное по умолчанию (пустая строка по умолчанию).

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта, полученный из настроек или установленный по умолчанию.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права проекта, полученные из настроек или установленные по умолчанию.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка на способ пожертвования автору проекта (чашечка кофе).

**Тип**: `str`
