# Модуль `src.goog`

## Обзор

Модуль `src.goog` предоставляет функциональность для работы с Google Sheets. Он включает в себя класс `SpreadSheet`, который инкапсулирует логику взаимодействия с таблицами Google.

## Оглавление

1. [Классы](#классы)
    - [`SpreadSheet`](#spreadsheet)

## Классы

### `SpreadSheet`

**Описание**: 
Класс `SpreadSheet` предоставляет интерфейс для взаимодействия с Google Sheets.

**Импортируется из:**
`from .spreadsheet import SpreadSheet`
