# Задача No 411
## Окрашенные квадраты

## Условия

Рассмотрим бесконечную плоскость, разбитую на единичные квадраты, как показано на рисунке ниже.

![Изображение сетки квадратов](https://projecteuler.net/images/p411_squares.gif)

Мы будем называть квадрат с нижним левым углом в точке (x,y) квадратом (x,y).

Для заданного целого n ≥ 1, определим множество Sₙ следующим образом:
Sₙ = { (x,y) : x,y являются целыми, 0 ≤ x,y < n }.

Мы также определим функцию f(n) как число способов покрасить квадраты множества Sₙ, используя один из четырех цветов, таким образом, что никакие два квадрата, имеющие общую сторону, не имеют одинаковый цвет. Другими словами, для любых двух квадратов (x₁, y₁) и (x₂, y₂), таких что |x₁-x₂| + |y₁-y₂| = 1, эти квадраты должны иметь разный цвет.

Например, f(1) = 4, f(2) = 12, f(3) = 24.

Найдите f(10 000 000) mod 100 000 007.
