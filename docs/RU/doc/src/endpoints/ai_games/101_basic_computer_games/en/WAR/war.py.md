# WAR

## Обзор

Этот модуль реализует карточную игру "Война" для двух игроков. Игра использует стандартную колоду из 52 карт. Игроки делят колоду пополам и поочередно выкладывают по одной карте. Игрок, карта которого имеет более высокое значение, выигрывает раунд и забирает обе карты. Если карты одинакового значения, объявляется "война".

## Функции

### `create_deck`

**Описание**:
Создает стандартную колоду из 52 игральных карт.

**Параметры**:
- Нет параметров.

**Возвращает**:
- `list`: Список строк, представляющих карты, например, "2C", "AH".

### `deal_cards`

**Описание**:
Перемешивает колоду и раздает карты поровну двум игрокам.

**Параметры**:
- `deck` (list): Список карт, представляющих колоду.

**Возвращает**:
- `tuple`: Кортеж из двух списков, представляющих колоды первого и второго игроков.

### `card_value`

**Описание**:
Определяет числовое значение карты для сравнения.

**Параметры**:
- `card` (str): Строка, представляющая карту, например, "2C", "AH".

**Возвращает**:
- `int`: Числовое значение карты (2-14).

### `war`

**Описание**:
Реализует логику "войны" в игре, когда карты игроков равны.

**Параметры**:
- `player1_deck` (list): Список карт первого игрока.
- `player2_deck` (list): Список карт второго игрока.

**Возвращает**:
- `tuple`: Кортеж, содержащий:
  - `int`: Номер победителя (1 или 2) или 0, если никто не выиграл.
  - `list`: Карты, выигранные первым игроком.
  - `list`: Карты, выигранные вторым игроком.
  - Если война не состоялась возвращает пустые списки для карт.

### `play_war`

**Описание**:
Основная функция игры "Война", которая управляет процессом игры, включая раздачу карт, сравнение карт и определение победителя.

**Параметры**:
- Нет параметров.

**Возвращает**:
- Нет возвращаемого значения. Функция выводит результаты игры в консоль.
