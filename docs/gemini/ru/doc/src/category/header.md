# Модуль `hypotez/src/category/header.py`

## Обзор

Модуль `header.py` определяет корневой путь к проекту `hypotez`. Все импорты строятся относительно этого пути.  В дальнейшем предполагается перенесение логики определения корневого пути в системные переменные.

## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная с текущей директории, ищет вверх по дереву каталогов до тех пор, пока не найдет каталог, содержащий указанные маркерные файлы.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или каталогов, которые используются для определения корневого каталога проекта. По умолчанию это ('pyproject.toml', 'requirements.txt', '.git').

**Возвращает**:

- `Path`: Путь к корневому каталогу проекта, если он найден. В противном случае возвращает каталог, в котором расположен скрипт.

**Вызывает исключения**:

- Нет


## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта, полученный с помощью функции `set_project_root()`.


## Обработка данных

### Чтение настроек

**Описание**: Читает файл настроек `settings.json` из каталога `src`.

**Действия**:
- Попытка открыть файл `src/settings.json` в режиме чтения.
- Разбор JSON-данных из файла в словарь `settings`.
- Обработка исключений:
    - `FileNotFoundError`: Если файл `settings.json` не найден, `settings` остается `None`.
    - `json.JSONDecodeError`: Если данные в файле `settings.json` некорректно отформатированы, `settings` остается `None`.


### Чтение документации

**Описание**: Читает файл README.MD из каталога `src` для получения документации.

**Действия**:
- Попытка открыть файл `src/README.MD` в режиме чтения.
- Чтение содержимого файла в строковую переменную `doc_str`.
- Обработка исключений:
    - `FileNotFoundError`: Если файл `README.MD` не найден, `doc_str` становится пустой строкой.
    - `json.JSONDecodeError`: Если файл не является валидным Markdown, `doc_str` становится пустой строкой.



## Переменные проекта

**Описание**: Переменные, содержащие информацию о проекте, полученную из файла настроек `settings.json`.  Если файл настроек не найден или не содержит нужные данные, значения по умолчанию устанавливаются.

- `__project_name__` (str): Название проекта. По умолчанию 'hypotez'.
- `__version__` (str): Версия проекта. По умолчанию пустая строка.
- `__doc__` (str): Документация из `README.MD`. По умолчанию пустая строка.
- `__details__` (str): Дополнительные данные. По умолчанию пустая строка.
- `__author__` (str): Автор проекта. По умолчанию пустая строка.
- `__copyright__` (str): Авторские права. По умолчанию пустая строка.
- `__cofee__` (str): Ссылка для поддержки разработчика. По умолчанию ссылка для поддержки разработчика.
