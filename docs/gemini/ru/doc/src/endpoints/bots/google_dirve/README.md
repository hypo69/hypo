# Документация модуля `google_dirve`

## Обзор

Этот модуль представляет собой документацию для Telegram-бота, предназначенного для загрузки файлов в Google Drive. Он содержит инструкции по настройке, использованию и расширению функциональности бота.

## Подробней

Этот бот позволяет пользователям загружать файлы в Google Drive, используя прямые ссылки, а также ссылки на Mega.nz, openload и Dropbox. Документация содержит информацию об установке необходимых модулей, настройке Google Drive API, получении токена Telegram-бота и использовании основных команд бота.

## Установка

### Установка модулей

```
sudo pip3 install -r requirements.txt
```

## Использование

### Запуск бота

```
python3 bot.py
```

### Авторизация

1.  Авторизуйте бота, используя команду `/auth`.
2.  Сгенерируйте ключ и отправьте его боту.
3.  Теперь вы можете отправлять поддерживаемые ссылки боту.

## Команды

-   `/start` - Приветственное сообщение
-   `/auth` - Авторизация пользователя
-   `/revoke` - Удаление сохраненных учетных данных
-   `/help` - Справочный текст

## Поддерживаемые ссылки

-   Прямые ссылки
-   Mega.nz ссылки
-   Dropbox ссылки

## Требования

-   [Google Drive API Credentials](https://console.cloud.google.com/apis/credentials) (Other type) - Обязательно
-   Telegram Bot Token (Используется BotFather) - Обязательно
-   Openload FTP логин и ключ - Опционально
-   Mega Email и Password - Опционально

## Настройка

1.  Создайте [Google Drive API Credentials](https://console.cloud.google.com/apis/credentials) (other type) и скачайте JSON-файл.
2.  Поместите JSON-файл в корневую директорию бота и переименуйте его в `client_secrets.json`.
3.  Замените токен вашего бота в файле [`creds.py`](./creds.py).
