# Модуль `src.webdriver`

## Обзор

Модуль `src.webdriver` предназначен для управления различными веб-драйверами. Он обеспечивает абстракцию над различными браузерами и инструментами автоматизации, такими как Chrome, Firefox, Edge, BrowserStack, Playwright и Crawlee.

## Переменные

### `MODE`
**Описание**: Определяет режим работы модуля. По умолчанию установлено значение `'dev'`.
- `MODE` (str): Режим работы модуля (например, 'dev' - режим разработки).
