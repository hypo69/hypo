# Задача No 431
## Номера Ферма

## Условия

Числа Ферма - это целые числа вида F<sub>n</sub> = 2<sup>2<sup>n</sup></sup> + 1, где n ≥ 0.
Для первых нескольких чисел Ферма:

* F<sub>0</sub> = 3,
* F<sub>1</sub> = 5,
* F<sub>2</sub> = 17,
* F<sub>3</sub> = 257,
* F<sub>4</sub> = 65537.

Может показаться, что все числа Ферма являются простыми, однако F<sub>5</sub> = 4294967297 = 641 * 6700417, поэтому F<sub>5</sub> составное.
Известно, что если n ≥ m, то F<sub>n</sub> ≡ 1 (mod F<sub>m</sub>).

Кроме того, если k ≠ n, то gcd(F<sub>n</sub>, F<sub>k</sub>) = 1.

Факторизация чисел Ферма является сложной задачей и в то же время важной, поскольку они используются для генерации псевдослучайных чисел и в криптографии.

Пусть A(n) - общее количество простых делителей (не обязательно различных) числа F<sub>n</sub>.

Известно, что:

* A(0) = 1, поскольку F<sub>0</sub> = 3,
* A(1) = 1, поскольку F<sub>1</sub> = 5,
* A(2) = 1, поскольку F<sub>2</sub> = 17,
* A(3) = 1, поскольку F<sub>3</sub> = 257,
* A(4) = 1, поскольку F<sub>4</sub> = 65537,
* A(5) = 2, поскольку F<sub>5</sub> = 641 × 6700417.

Пусть S(N) = Σ A(n) для 0 ≤ n ≤ N.

Известно, что S(5) = 7 и S(10) = 28.

Найдите S(20).
