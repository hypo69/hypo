# Задача No 465
## Множество наибольших простых делителей

## Условия

Пусть *g(n)* обозначает наибольший простой делитель числа *n*.
Пусть *S(n)* обозначает множество всех натуральных чисел *k*, таких что *g(k) < g(k+1)*, *k* ≤ *n*.

Например, *S(10)* = {1, 2, 3, 6, 10}, и его элементы в порядке возрастания:
*   *g(1)=1*
*   *g(2)=2*
*   *g(3)=3*
*   *g(4)=2*
*   *g(5)=5*
*   *g(6)=3*
*   *g(7)=7*
*   *g(8)=2*
*   *g(9)=3*
*   *g(10)=5*

Поэтому *S(10)* = {1, 2, 3, 6, 10}, поскольку
* *g(1) < g(2)*
* *g(2) < g(3)*
* *g(3) > g(4)*
* *g(4) < g(5)*
* *g(5) > g(6)*
* *g(6) < g(7)*
* *g(7) > g(8)*
* *g(8) < g(9)*
* *g(9) > g(10)*
* *g(10) < g(11)*

Пусть *s(n)* обозначает сумму элементов множества *S(n)*.
Например, *s(10)* = 1+2+3+6+10 = 22.

Дано, что *s(10<sup>3</sup>)* = 828860.

Найдите *s(10<sup>7</sup>)*.
