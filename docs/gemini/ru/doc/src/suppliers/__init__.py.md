# Модуль `src.suppliers`

## Обзор

Модуль `src.suppliers` представляет собой набор инструментов для работы с поставщиками. Он включает в себя базовый класс `Supplier` и механизмы для добавления специфичных методов для каждого конкретного поставщика. Методы для каждого поставщика размещаются в директориях с префиксом имени поставщика, например `amazon`, `aliexpress`, `morlevi` и т.д.  Префикс задается при создании нового поставщика в системе.
