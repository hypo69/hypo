# Модуль `hypotez/src/suppliers/grandadvance/graber.py`

## Обзор

Этот модуль содержит класс `Graber`, предназначенный для сбора данных о товарах со страницы `grandadvanse.co.il`.  Класс реализует асинхронный захват данных, используя веб-драйвер, и обрабатывает различные поля товара.  Он предоставляет методы для получения информации о товаре, включая изображения, и сохраняет их локально.  Модуль использует декоратор `close_pop_up` для обработки всплывающих окон.


## Классы

### `Graber`

**Описание**: Класс `Graber` наследуется от `Grbr` и предназначен для сбора данных о товарах.  Он инициализируется экземпляром веб-драйвера и содержит методы для извлечения информации о товаре.

**Атрибуты**:

* `supplier_prefix`: Строка, определяющая префикс поставщика данных.

**Методы**:

#### `__init__(self, driver: Driver)`

**Описание**: Инициализирует экземпляр класса `Graber`.

**Параметры**:

* `driver` (Driver): Экземпляр класса `Driver`, предоставляющий доступ к веб-драйверу.


#### `grab_page(self, driver: Driver) -> ProductFields`

**Описание**: Асинхронный метод для захвата полей продукта.

**Параметры**:

* `driver` (Driver): Экземпляр класса `Driver`.

**Возвращает**:

* `ProductFields`: Объект содержащий собранные поля товара.

**Вызывает исключения**:
* Возможны исключения, связанные с выполнением локатора (`ExecuteLocatorException`).


#### `local_saved_image(self, value: Any = None)`

**Описание**: Метод для загрузки и сохранения изображения локально.

**Параметры**:

* `value` (Any, необязательно): Дополнительное значение. Если передано, его значение используется для заполнения поля `ProductFields.local_saved_image`.

**Возвращает**:
* `bool`: `True` если изображение успешно сохранено, иначе `None`
