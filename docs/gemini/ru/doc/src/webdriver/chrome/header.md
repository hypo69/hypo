# Модуль hypotez/src/webdriver/chrome/header.py

## Обзор

Данный модуль содержит код для определения корневой директории проекта, загрузки настроек из файла `settings.json` и получения документации из `README.MD`.  Он инициализирует глобальные переменные, необходимые для работы других частей проекта, такие как `__root__`, `settings`, `__project_name__`, `__version__`, `__doc__`, `__details__`, `__author__`, `__copyright__`, и `__cofee__`.


## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с текущей директории, ищет вверх по дереву директорий до тех пор, пока не найдёт директорию, содержащую один из указанных файлов или папок.

**Параметры**:

- `marker_files` (tuple): Кортеж имён файлов или директорий, которые используются для определения корня проекта. По умолчанию это `('pyproject.toml', 'requirements.txt', '.git')`.


**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена, иначе путь к директории, где расположен скрипт.


**Вызывает исключения**:

- Нет


## Глобальные переменные

### `__root__`

**Описание**: Путь к корневой директории проекта. Инициализируется функцией `set_project_root`.

**Тип**: `Path`

### `settings`

**Описание**: Словарь настроек, загруженный из файла `settings.json`.

**Тип**: `dict`

### `doc_str`

**Описание**: Строка, содержащая текст из файла `README.MD`.

**Тип**: `str`


### `__project_name__`

**Описание**: Имя проекта, полученное из настроек или по умолчанию `hypotez`.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта, полученная из настроек или по умолчанию пустая строка.

**Тип**: `str`


### `__doc__`

**Описание**: Документация проекта, полученная из файла `README.MD` или по умолчанию пустая строка.

**Тип**: `str`


### `__details__`

**Описание**: Дополнительные сведения о проекте (по умолчанию пустая строка).

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта, полученный из настроек или по умолчанию пустая строка.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права на проект, полученные из настроек или по умолчанию пустая строка.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка на страницу сбора пожертвований на чашку кофе для разработчика.

**Тип**: `str`
