# Модуль `hypotez/src/endpoints/advertisement/facebook/scenarios/post_event.py`

## Обзор

Данный модуль предоставляет функции для публикации календарных событий в группах Facebook.  Он использует Selenium WebDriver для взаимодействия с веб-сайтом Facebook и JSON-файлы для хранения локаторов элементов страницы.

## Функции

### `post_title`

**Описание**: Отправляет заголовок события в форму для публикации.

**Параметры**:

- `d` (Driver): Экземпляр класса `Driver`, используемый для взаимодействия с веб-страницей.
- `title` (str): Заголовок события.

**Возвращает**:

- `bool`: `True`, если заголовок был отправлен успешно, иначе `None`.

**Вызывает исключения**:
- Возможны ошибки при взаимодействии с веб-драйвером, которые обрабатываются в блоке `try...except`.

### `post_event`

**Описание**: Управляет процессом публикации события с заголовком, описанием и медиафайлами.

**Параметры**:

- `d` (Driver): Экземпляр класса `Driver`, используемый для взаимодействия с веб-страницей.
- `event` (SimpleNamespace): Объект, содержащий данные события (заголовок, описание, дату, время и ссылку).

**Возвращает**:

- `bool`: `True`, если событие было опубликовано успешно, иначе `None`.

**Вызывает исключения**:
- Возможны ошибки при выполнении каждой из функций `post_title`, `post_date`, `post_time` и `post_description`.


##  Локаторы (locator)

Локаторы элементов на странице Facebook загружаются из файла `post_event.json` в каталоге `locators`.  Этот файл должен содержать информацию о расположении необходимых элементов.

## Использование

Для использования функций данного модуля необходимо иметь экземпляр класса `Driver` для взаимодействия с веб-сайтом Facebook.  Функции ожидают `SimpleNamespace` объект `event`, содержащий необходимые поля, такие как `title`, `description`, и `start`.
