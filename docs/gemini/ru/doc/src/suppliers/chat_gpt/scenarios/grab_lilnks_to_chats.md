# Модуль grab_lilnks_to_chats

## Обзор

Модуль `grab_lilnks_to_chats` предназначен для получения ссылок на отдельные чаты с помощью веб-драйвера.  Он использует заранее определённые локейторы для нахождения ссылок и предоставляет функцию для их извлечения.

## Функции

### `get_links`

**Описание**: Функция извлекает ссылки на отдельные чаты.

**Параметры**:
* `d` (Driver): Объект веб-драйвера, необходимый для взаимодействия с браузером.


**Возвращает**:
* `list`: Список ссылок на отдельные чаты. Возвращает пустой список, если ссылки не найдены или возникла ошибка.

**Вызывает исключения**:
* Возможны исключения, связанные с работой веб-драйвера (например, ошибки при выполнении локейтора),  которые будут подниматься, если они возникнут. Обработка ошибок должна быть реализована в вызывающей функции.
