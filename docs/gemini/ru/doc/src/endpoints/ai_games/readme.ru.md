# Игры, которые легко научить машину играть

## Обзор

Данный документ описывает выбор игр для обучения искусственного интеллекта (ИИ).  Выбор игры зависит от целей обучения, сложности правил, количества возможных состояний и других факторов.

## Игры для обучения ИИ

### Игры с полной информацией и дискретными действиями

* **Шашки и шахматы:**  Классические игры с чёткими правилами и ограниченным количеством возможных ходов. ИИ достигает впечатляющих результатов в этих играх.
* **Крестики-нолики:** Простая игра, используемая для обучения базовым алгоритмам машинного обучения.
* **Го:**  Игра с огромным количеством возможных позиций, успешно осваиваемая современными алгоритмами глубокого обучения.
* **Реверси:** Стратегическая игра, похожая на шашки.
* **Судоку:**  Задача заполнения пустых клеток таблицы цифрами от 1 до 9, при этом каждая цифра должна встречаться в каждой строке, каждом столбце и каждом квадрате 3х3.


### Игры с неполной информацией

* **Покер:**  Игры с неполной информацией, где игроки не знают карт соперников, что усложняет обучение ИИ.  Однако, современные алгоритмы машинного обучения успешно применяются для игры в покер.
* **Блэкджек:**  Карточная игра с элементами случайности и принятия решений.


### Игры с элементами случайности

* **Морской бой:**  Игра, где часть информации скрыта от игрока, добавляет элемент случайности.
* **Угадай слово:**  Задача игрока - отгадать загаданное слово, задавая вопросы.


### Игры в реальном времени

* **StarCraft:** Сложная стратегическая игра в реальном времени, требующая от ИИ принятия быстрых и эффективных решений.
* **Dota 2:** Популярная киберспортивная игра, требующая от ИИ не только стратегического мышления, но и координации действий нескольких юнитов.


### Игры с большим количеством возможных состояний

* **Шахматы на больших досках:** Увеличение размера доски значительно повышает сложность игры.
* **Го на больших досках:** Аналогично шахматам.


## Методы обучения

* **Подкрепление:** ИИ обучается методом проб и ошибок, получая вознаграждение за правильные действия и наказание за неправильные.
* **Обучение с учителем:** ИИ обучается на большом количестве данных, где для каждого состояния указан оптимальный ход.
* **Обучение без учителя:** ИИ самостоятельно изучает структуру данных и находит закономерности.


## Преимущества обучения ИИ на играх

* **Четко определенные правила:** Легко формализовать правила игры и создать среду для обучения.
* **Возможность симуляции:**  Проведение большого количества игр для обучения модели.
* **Измеримый результат:**  Легко оценить успешность обучения, сравнивая результаты ИИ с результатами профессиональных игроков.


## Игры, которые могут быть сложнее для обучения

* **Игры с бесконечным количеством возможных состояний:** Например, игры с открытым миром.
* **Игры с сильной зависимостью от физики:** Игры, где важна точная симуляция физических процессов.
* **Игры с социальным взаимодействием:** Игры, где важны психологические факторы и умение общаться с другими игроками.
