### Название игры: **FURS** (Торговля шкурами)

---

#### Описание
**FURS** — это экономическая игра, в которой игрок торгует шкурами с индейцами, пытаясь заработать наибольшее количество денег. В игре участвует несколько торговцев, и каждый ход игрок решает, сколько шкур и по какой цене купить или продать. Цель игры — накопить как можно больше денег, управляя своими покупками и продажами шкур.

---

### Пошаговая инструкция для реализации

#### 1. **Инициализация игры**
   - При запуске игры программа приветствует игрока и объясняет правила.
   - Игрок получает начальный капитал (например, 100 долларов).
   - Программа генерирует случайные цены на шкуру, которые могут варьироваться в зависимости от сезона или состояния рынка.
   - Игрок может выбрать количество торговцев, с которыми он будет работать (например, от 2 до 5).

#### 2. **Основной процесс игры**
   - **Торговля шкурами:**
     1. Игрок может покупать шкуру по цене, установленной на рынке, или продавать её с выгодой.
     2. Каждый товар (шкура) имеет свою цену, которая может изменяться в зависимости от спроса и предложения.
     3. Цена покупки и продажи шкур варьируется и зависит от состояния рынка и торговых отношений с индейцами.

     - Игрок решает, сколько шкур он покупает или продаёт за определённую сумму денег.

#### 3. **Условия победы**

"""
Модуль, описывающий игру "FURS" (Торговля шкурами)
====================================================

Этот модуль содержит описание экономической игры, в которой игрок торгует шкурами
с индейцами. Цель игры — заработать как можно больше денег,
управляя покупками и продажами шкур.

"""

from typing import List, Dict
from src.logger.logger import logger #  Импортируем logger для логирования ошибок
import random # Импортируем модуль random для генерации случайных значений


def initialize_game() -> Dict:
    """
    Инициализирует игру, устанавливая начальные значения для игрока и рынка.

    :return: Словарь с начальными данными игры (капитал игрока, цены на шкуры).
    :rtype: dict
    """
    try:
        #  Начальный капитал игрока.
        player_capital = 100
        #  Генерация случайной цены на шкуру.
        fur_price = random.randint(15, 30)
        #  Создание словаря с начальными данными игры.
        game_data = {
            'player_capital': player_capital,
            'fur_price': fur_price,
            'player_furs': 0
        }
        #  Выводим приветственное сообщение и начальные условия игры.
        print('Добро пожаловать в игру FURS!')
        print(f'У вас ${player_capital}.')
        print('Торгуйте шкурами с индейцами и постарайтесь заработать как можно больше денег.')
        print('Программа генерирует случайные цены на шкуру.')
        print('Начните торговлю!\n')
        return game_data
    except Exception as ex:
        logger.error('Ошибка инициализации игры', ex)
        return {}


def trade_furs(game_data: Dict, action: str, quantity: int) -> Dict:
    """
    Выполняет торговые операции (покупку или продажу шкур).

    :param game_data: Словарь с данными игры.
    :type game_data: dict
    :param action: Действие игрока ('buy' или 'sell').
    :type action: str
    :param quantity: Количество шкур для покупки или продажи.
    :type quantity: int
    :return: Обновленный словарь с данными игры.
    :rtype: dict
    """
    try:
        #  Получение текущих данных из словаря.
        player_capital = game_data['player_capital']
        fur_price = game_data['fur_price']
        player_furs = game_data['player_furs']
        #  Проверяем, что количество шкур больше 0.
        if quantity <= 0:
            print('Количество шкур должно быть больше 0.')
            return game_data
        #  Выполняем операцию покупки шкур.
        if action == 'buy':
            #  Рассчитываем общую стоимость покупки.
            total_cost = fur_price * quantity
            #  Проверяем, что у игрока достаточно денег для покупки.
            if player_capital < total_cost:
                print('Недостаточно денег для покупки.')
                return game_data
            #  Обновляем капитал и количество шкур игрока.
            player_capital -= total_cost
            player_furs += quantity
            print(f'Вы купили {quantity} шкуры по ${fur_price}. Остаток: ${player_capital}.')
            print(f'Ваши шкуры: {player_furs}')
        #  Выполняем операцию продажи шкур.
        elif action == 'sell':
            #  Проверяем, что у игрока есть шкуры для продажи.
            if player_furs < quantity:
                print('Недостаточно шкур для продажи.')
                return game_data
             # Обновляем капитал и количество шкур игрока.
            player_capital += fur_price * quantity
            player_furs -= quantity
            print(f'Вы продали {quantity} шкуры по ${fur_price}. Остаток: ${player_capital}.')
            print(f'Ваши шкуры: {player_furs}')

        #  Обновляем данные игры.
        game_data['player_capital'] = player_capital
        game_data['player_furs'] = player_furs
        return game_data
    except Exception as ex:
        logger.error('Ошибка при торговле шкурами', ex)
        return game_data


def check_win_condition(game_data: Dict) -> bool:
    """
    Проверяет, выполнил ли игрок условия победы.

    :param game_data: Словарь с данными игры.
    :type game_data: dict
    :return: True, если игрок выиграл, иначе False.
    :rtype: bool
    """
    try:
        #  Получаем капитал игрока из данных игры.
        player_capital = game_data['player_capital']
        player_furs = game_data['player_furs']
        #  Проверяем условия победы (накопление 500$ или 50 шкур).
        if player_capital >= 500 or player_furs >= 50:
            print(f'Поздравляем! Вы накопили ${player_capital} и {player_furs} шкур.')
            return True
        return False
    except Exception as ex:
         logger.error('Ошибка проверки условий победы', ex)
         return False


def play_again() -> bool:
    """
    Предлагает игроку сыграть еще раз.

    :return: True, если игрок хочет сыграть еще раз, иначе False.
    :rtype: bool
    """
    while True:
        #  Запрашиваем ввод игрока.
        answer = input('Хотите сыграть ещё раз? (да/нет): ').lower()
        if answer == 'да':
            return True
        elif answer == 'нет':
            print('Спасибо за игру!')
            return False
        else:
            print('Некорректный ввод. Пожалуйста, введите "да" или "нет".')


def main():
    """
    Основная функция для запуска и управления игровым процессом.
    """
    #  Запускаем игру и получаем начальные данные.
    game_data = initialize_game()
    if not game_data:
        return # если не удалось инициализировать игру, то выходим
    #  Основной цикл игры.
    while True:
        #  Генерируем случайную цену на шкуру.
        game_data['fur_price'] = random.randint(15, 30)
        print(f'Цена за шкуру: ${game_data["fur_price"]}')
        #  Запрашиваем действие игрока.
        action = input('Купить или продать шкуры? (buy/sell/exit): ').lower()
        #  Проверяем ввод игрока.
        if action == 'exit':
            print('Спасибо за игру!')
            break
        if action not in ['buy', 'sell']:
             print('Некорректный ввод. Пожалуйста, введите "buy", "sell" или "exit".')
             continue
        #  Запрашиваем количество шкур.
        try:
            quantity = int(input('Сколько шкур вы хотите? '))
        except ValueError:
            print('Некорректный ввод. Пожалуйста, введите целое число.')
            continue
        #  Выполняем торговую операцию.
        game_data = trade_furs(game_data, action, quantity)
        #  Проверяем условия победы.
        if check_win_condition(game_data):
            #  Предлагаем сыграть еще раз.
            if not play_again():
                break
            else:
               # Если игрок хочет сыграть еще раз, начинаем игру сначала.
                game_data = initialize_game()
                if not game_data:
                    break
            
# Код запуска программы
if __name__ == '__main__':
    main()
