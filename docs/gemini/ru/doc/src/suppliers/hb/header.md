# Модуль hypotez/src/suppliers/hb/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и чтения документации из файла `README.MD`. Он также предоставляет доступ к глобальным переменным, хранящим информацию о проекте (имя, версия, описание, автор, права).

## Функции

### `set_project_root`

**Описание**: Определяет корневую директорию проекта, начиная с текущей директории и проходя вверх по директориям, пока не найдет директорию, содержащую один из указанных файлов или каталогов.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или каталогов, которые используются для определения корневой директории проекта.

**Возвращает**:

- `Path`: Путь к корневой директории проекта. Если корневая директория не найдена, возвращает директорию, в которой находится данный скрипт.

**Вызывает исключения**:

- Не вызывает исключения.


## Глобальные переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root`.

**Тип**: `Path`


### `settings`

**Описание**: Словарь настроек проекта, загруженный из файла `settings.json`.

**Тип**: `dict` или `None`


### `doc_str`

**Описание**: Текст документации из файла `README.MD`.

**Тип**: `str` или `None`


### `__project_name__`

**Описание**: Название проекта, взятое из настроек (или `hypotez` по умолчанию).

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта, взятая из настроек (или пустая строка по умолчанию).

**Тип**: `str`


### `__doc__`

**Описание**: Текст документации, взятый из `README.MD` (или пустая строка по умолчанию).

**Тип**: `str`


### `__details__`

**Описание**: Детали проекта (пустая строка по умолчанию).

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта, взятый из настроек (или пустая строка по умолчанию).

**Тип**: `str`


### `__copyright__`

**Описание**: Права на проект, взятые из настроек (или пустая строка по умолчанию).

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка для поддержки автора проекта.

**Тип**: `str`
