# BIORHYTHM

## Обзор

Этот модуль реализует игру "Биоритмы", которая вычисляет биоритмы пользователя на основе его даты рождения и заданной даты. Биоритмы рассчитываются на основе трех циклов: физического, эмоционального и интеллектуального, каждый из которых имеет свою периодичность.

## Функции

### `calculate_biorhythm`

**Описание**:
Вычисляет биоритмы для заданной даты рождения и даты проверки.

**Параметры**:
- `birth_date` (date): Дата рождения пользователя.
- `check_date` (date): Дата для проверки.

**Возвращает**:
- `None`: Функция выводит таблицу с результатами биоритмов.

**Вызывает исключения**:
- `None`: Функция не вызывает исключений.
```python
def calculate_biorhythm(birth_date, check_date):
    """
    מחשב את הביוריתמוסים עבור תאריך לידה ותאריך בדיקה.

    :param birth_date: תאריך הלידה של המשתמש.
    :param check_date: התאריך לבדיקה.
    :return: טבלה עם תוצאות הביוריתמוס.
    """
    # חישוב מספר הימים בין תאריכי הלידה והבדיקה
    delta = check_date - birth_date
    days = delta.days

    # חישוב הביוריתמוסים
    physical = math.sin(2 * math.pi * days / 23)
    emotional = math.sin(2 * math.pi * days / 28)
    intellectual = math.sin(2 * math.pi * days / 33)

    # יצירת טבלה של התוצאות
    print("---------------------------------------------------")
    print("|        תאריך לידה      |      תאריך בדיקה    |      ביוריתמוס       |")
    print("---------------------------------------------------")
    print(f"| {birth_date.strftime('%d/%m/%Y')}    | {check_date.strftime('%d/%m/%Y')}  | פיזי: {physical:.2f}   |")
    print(f"|                        |                     | רגשי: {emotional:.2f}   |")
    print(f"|                        |                     | אינטלקטואלי: {intellectual:.2f} |")
    print("---------------------------------------------------")
```
