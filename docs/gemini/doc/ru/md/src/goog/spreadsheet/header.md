# Модуль `hypotez/src/goog/spreadsheet/header.py`

## Обзор

Этот модуль содержит функции для определения корневого каталога проекта и загрузки настроек проекта из файла `settings.json`.  Он также загружает описание проекта из файла `README.MD`.  Модуль необходим для инициализации проекта и доступа к его конфигурации.

## Функции

### `set_project_root`

**Описание**: Определяет корневой каталог проекта, начиная с текущей директории и проходя вверх по древу каталогов.  Останавливается на первом каталоге, содержащем файлы-маркеры.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или каталогов, используемых для определения корневого каталога проекта. По умолчанию (`pyproject.toml`, `requirements.txt`, `.git`).

**Возвращает**:

- `Path`: Путь к корневому каталогу проекта, если найден, иначе текущий каталог.

**Вызывает исключения**:

- Нет.


## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта. Инициализируется функцией `set_project_root()`.

### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json`. Если файл не найден или некорректен, переменная устанавливается в `None`.


### `doc_str`

**Описание**: Строка, содержащая текст документации проекта из файла `README.MD`. Если файл не найден или некорректен, переменная устанавливается в `None`.


### `__project_name__`

**Описание**: Название проекта, полученное из настроек.  По умолчанию `hypotez`.

### `__version__`

**Описание**: Версия проекта, полученная из настроек. По умолчанию пустая строка.

### `__doc__`

**Описание**: Строка документации проекта, полученная из файла `README.MD`. По умолчанию пустая строка.

### `__details__`

**Описание**: Дополнительные детали о проекте. По умолчанию пустая строка.

### `__author__`

**Описание**: Автор проекта, полученный из настроек. По умолчанию пустая строка.

### `__copyright__`

**Описание**: Авторские права проекта, полученные из настроек. По умолчанию пустая строка.

### `__cofee__`

**Описание**: Ссылка на страницу для поддержки разработчика. По умолчанию  `Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69`.
