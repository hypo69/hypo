# Модуль hypotez/src/fast_api/header.py

## Обзор

Этот модуль содержит функции для определения корневого каталога проекта и загрузки настроек из файла `settings.json`.  Он также обрабатывает потенциальные ошибки при чтении файла настроек и документации.  Модуль инициализирует переменные, описывающие проект (название, версия, автор, копирайт, описание, ссылка на поддержку).

## Функции

### `set_project_root`

**Описание**:  Находит корневой каталог проекта, начиная от текущего файла и идя вверх по директориям, пока не найдет директорию, содержащую указанные файлы-маркеры (pyproject.toml, requirements.txt, .git).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для определения корневого каталога проекта. По умолчанию содержит ('pyproject.toml', 'requirements.txt', '.git').


**Возвращает**:

- `Path`: Путь к корневому каталогу проекта, если он найден. В противном случае возвращает путь к директории, где находится текущий файл.


**Вызывает исключения**:

- Нет


### `load_settings`

**Описание**: (Уточнение:  Функция `load_settings` не определена, но описывается логика в коде) Функция загружает настройки из файла `settings.json`.

**Параметры**:

- Нет


**Возвращает**:

- `dict`: Словарь с настройками проекта, если файл `settings.json` найден и загружен без ошибок.


**Вызывает исключения**:

- `FileNotFoundError`: Если файл `settings.json` не найден.
- `json.JSONDecodeError`: Если содержимое файла `settings.json` не является валидным JSON.


### `load_documentation`

**Описание**: (Уточнение: Функция `load_documentation` не определена, но описывается логика в коде) Загружает текстовое содержимое файла `README.MD`.

**Параметры**:

- Нет


**Возвращает**:

- `str`: Текстовое содержимое файла `README.MD` , если он найден и прочитан без ошибок. `None` в противном случае.


**Вызывает исключения**:

- `FileNotFoundError`: Если файл `README.MD` не найден.
- `json.JSONDecodeError`:  (НЕПРИМЕНИМО к чтению текстовых файлов, ошибка приведена для соответствия формату, но не является корректной в данном контексте.)



## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта. Инициализируется функцией `set_project_root()`.

**Тип**: `Path`

### `settings`

**Описание**: Словарь с настройками проекта. Загружается из файла `settings.json`.

**Тип**: `dict`

### `doc_str`

**Описание**: Текстовое содержимое файла `README.MD`. Загружается из файла `README.MD`.

**Тип**: `str`

### `__project_name__`

**Описание**: Название проекта. Получается из настроек (`settings`). По умолчанию "hypotez".

**Тип**: `str`

### `__version__`

**Описание**: Версия проекта. Получается из настроек (`settings`). По умолчанию "".

**Тип**: `str`

### `__doc__`

**Описание**: Описание проекта. Получается из файла `README.MD`. По умолчанию "".

**Тип**: `str`

### `__details__`

**Описание**: Дополнительные детали проекта. По умолчанию "".

**Тип**: `str`

### `__author__`

**Описание**: Автор проекта. Получается из настроек (`settings`). По умолчанию "".

**Тип**: `str`

### `__copyright__`

**Описание**: Авторские права проекта. Получается из настроек (`settings`). По умолчанию "".

**Тип**: `str`

### `__cofee__`

**Описание**: Ссылка на поддержку разработчика. Получается из настроек (`settings`). По умолчанию ссылка на Boosty.

**Тип**: `str`
