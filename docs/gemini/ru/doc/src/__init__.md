## Обзор

Этот модуль является корневым модулем проекта `hypotez/src`. Он предоставляет структурированный обзор других модулей, описывая их основные функциональности и роли.  В этом модуле определен константа `MODE`, имеющая значение `'dev'`. Также он импортирует модуль `credentials`.


## Переменные

### `MODE`

**Описание**: Константа, определяющая режим работы проекта. В данном случае, значение `'dev'`.
