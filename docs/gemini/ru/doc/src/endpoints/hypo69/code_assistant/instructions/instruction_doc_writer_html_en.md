# Документация для кода

## Обзор

Этот файл содержит примеры генерации HTML документации для Python кода.  Документация сгенерирована на основе входных данных, но не основана на реальном Python коде.

## Функции

### `function_name`

**Описание**: Краткое описание функции.

**Параметры**:
- `param` (str): Описание параметра `param`.
- `param1` (Optional[str | dict | str], optional): Описание параметра `param1`. По умолчанию `None`.

**Возвращает**:
- `dict | None`: Описание возвращаемого значения. Возвращает словарь или `None`.

**Вызывает исключения**:
- `SomeError`: Описание ситуации, в которой возникает исключение `SomeError`.


### `another_function`

**Описание**:  Другая функция.

**Параметры**:
- `arg1` (int): Целочисленное значение.
- `arg2` (str): Строковое значение.

**Возвращает**:
- `bool`: Значение булевого типа.


## Классы

### `ClassName`

**Описание**: Краткое описание класса.

**Методы**:
- `method_name`: Краткое описание метода.
- `another_method`:  Краткое описание другого метода.

**Параметры**:
- `param` (str): Описание параметра `param`.
- `param1` (Optional[str | dict | str], optional): Описание параметра `param1`. По умолчанию `None`.

### `AnotherClass`

**Описание**:  Описание другого класса.

**Методы**:
- `method_one`: Описание метода.


##  Пример использования

**Описание**: Примеры использования функций и методов.

```html
<p>Пример использования функции <code>function_name</code>:</p>

<pre><code>
result = function_name("some_value", {"key": "value"})
print(result)
</code></pre>

<p>Пример использования класса <code>ClassName</code>:</p>

<pre><code>
obj = ClassName("instance_value")
obj.method_name("another_value")
</code></pre>
```
