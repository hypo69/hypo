# Задача No 335
## Gather the coins

## Условия

Представьте, что имеется доска, состоящая из квадратов, расположенных в виде треугольника с n рядами. Например, если n=3, доска будет выглядеть следующим образом:

```
  O
 O O
O O O
```

В каждой клетке находится монета.

В начале игры все монеты располагаются на нижнем ряду.

Каждый ход можно выбрать любую монету и переместить её в соседнюю клетку (соседней считается клетка, разделяющая с исходной общую сторону).

Целью является собрать все монеты в верхнюю клетку.

Пусть M(n) будет минимальным количеством ходов, необходимых для того, чтобы собрать все монеты в верхнюю клетку, для треугольника с n рядами.

Например, M(3) = 20.

Известно, что:
M(10) = 572
M(100) = 52328
M(1000) = 5023328

Найдите M(100000).
