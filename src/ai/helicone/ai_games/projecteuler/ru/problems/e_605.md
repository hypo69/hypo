# Задача No 605
## Нечетные подпоследовательности

## Условия
Назовем последовательность целых чисел *нечетной*, если каждый член последовательности является нечетным числом.

Пусть F(*n*) – количество нечетных подпоследовательностей последовательности {1, 2, 3, ..., *n*}, которые не являются подпоследовательностью последовательности Фибоначчи.

Например:
*   {1, 3, 5} - нечетная подпоследовательность {1, 2, 3, 4, 5}.
*   {1, 5} является подпоследовательностью Фибоначчи.
*   {1, 3} не является подпоследовательностью Фибоначчи.
*   F(5) = 7, нечетными подпоследовательностями {1, 2, 3, 4, 5} являются {}, {1}, {3}, {5}, {1,3}, {1,5}, {3,5}, {1,3,5}.  Из них {1}, {5}, {1,5} являются подпоследовательностями Фибоначчи.

Известно, что F(10) = 124, F(100) = 158968 и F(1000) mod 123456789 = 81065389.

Найдите F(10<sup>18</sup>) mod 123456789.
