# Задача No 886
## Доли суммы
## Условия

Пусть S(N) - это сумма, полученная суммированием всех чисел, у которых нет цифр, отличных от 0, 1 или 2, и которые меньше или равны N.
Например, S(25) = 1 + 2 + 10 + 11 + 12 + 20 + 21 + 22 = 99.

Пусть T(N) - сумма значений S(n), для всех 1 ≤ n ≤ N.
Например, T(5) = S(1) + S(2) + S(3) + S(4) + S(5) = 1 + 3 + 3 + 3 + 3 = 13.
T(10) = 13 + 33 + 33 + 33 + 33 + 105 + 105 + 105 + 105 + 105 = 668

Найдите T(10<sup>18</sup>) mod 1 000 000 007.
