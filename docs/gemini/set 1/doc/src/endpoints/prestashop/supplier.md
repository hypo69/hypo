# Модуль `hypotez/src/endpoints/prestashop/supplier.py`

## Обзор

Данный модуль предоставляет класс `PrestaSupplier` для работы с поставщиками в системе PrestaShop.  Он наследуется от класса `PrestaShop`, расширяя функциональность для работы с поставщиками. Модуль использует API PrestaShop для взаимодействия с системой.

## Классы

### `PrestaSupplier`

**Описание**: Класс `PrestaSupplier` предназначен для работы с поставщиками в PrestaShop. Он расширяет функциональность базового класса `PrestaShop`.

**Методы**:

- `__init__`: Инициализирует экземпляр класса.

**Параметры**:

- `credentials` (Optional[dict | SimpleNamespace], optional): Словарь или объект SimpleNamespace с параметрами `api_domain` и `api_key`. По умолчанию `None`.
- `api_domain` (Optional[str], optional): Домен API. По умолчанию `None`.
- `api_key` (Optional[str], optional): Ключ API. По умолчанию `None`.

**Возвращает**:
-  Не имеет возвращаемого значения.

**Вызывает исключения**:

- `ValueError`: Если не указаны оба параметра `api_domain` и `api_key`, или они имеют пустые значения.
