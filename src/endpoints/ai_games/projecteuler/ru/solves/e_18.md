## Ответ на Задачу No 18: Максимальная сумма по пути I

### 1. Анализ задачи и решение
**Понимание задачи:**
* Нам дан треугольник чисел. Нужно найти максимальную сумму, двигаясь от вершины к основанию, при этом на каждом шаге можно переходить только к соседним числам в нижнем ряду.
* Задача сводится к динамическому программированию: вычисляем максимальную сумму для каждой позиции, идя снизу вверх.

**Решение:**
1. **Инициализация:** Начинаем с предпоследнего ряда.
2. **Итерация:** Идем снизу вверх по треугольнику. Для каждого числа в текущем ряду находим максимальную сумму, добавляя к нему максимальное из двух чисел, находящихся под ним.
3. **Результат:** После прохода по всем рядам, максимальная сумма будет в вершине треугольника.

### 2. Алгоритм решения
1. Начать
2. Скопировать входной треугольник в матрицу `triangle`
3. Для каждой строки `row` начиная с предпоследней до первой (снизу вверх):
   * Для каждого элемента `element` в текущей строке:
     * Обновить значение `element` как `element` + `max(triangle[row+1][index], triangle[row+1][index+1])`, где `index` индекс текущего элемента в строке `row`
4. Вернуть значение `triangle[0][0]` (значение в вершине треугольника)
5. Конец

### 3. Реализация на Python 3.12
```python
def max_path_sum(triangle):
    """
    Calculates the maximum sum of a path from the top to the bottom of a triangle.

    Args:
      triangle: A list of lists representing the triangle of numbers.

    Returns:
      The maximum path sum.
    """
    num_rows = len(triangle)
    
    # Iterate from the second to last row up to the first row
    for row in range(num_rows - 2, -1, -1):
        for index in range(len(triangle[row])):
           triangle[row][index] += max(triangle[row+1][index], triangle[row+1][index+1])

    return triangle[0][0]

# Пример использования:
triangle = [
    [75],
    [95, 64],
    [17, 47, 82],
    [18, 35, 87, 10],
    [20, 4, 82, 47, 65],
    [19, 1, 23, 75, 3, 34],
    [88, 2, 77, 73, 7, 63, 67],
    [99, 65, 4, 28, 6, 16, 70, 92],
    [41, 41, 26, 56, 83, 40, 80, 70, 33]
]


result = max_path_sum(triangle)
print(result)
```

### 4. Блок-схема в формате mermaid
```mermaid
flowchart TD
    Start((Начало)) --> CopyTriangle((Скопировать треугольник в matrix))
    CopyTriangle --> LoopRows((Начать цикл по строкам снизу вверх: row))
    LoopRows --> LoopElements((Начать цикл по элементам в строке: element))
    LoopElements --> UpdateElementValue((Обновить значение element: element = element + max(element_низу_слева, element_низу_справа) ))
    UpdateElementValue --> LoopElements
    LoopElements --> LoopRows
    LoopRows --> ReturnResult((Вернуть значение triangle[0][0]))
    ReturnResult --> End((Конец))
```

**Legenda:**
* **Начало, Конец:** Начало и конец алгоритма.
* **Скопировать треугольник в matrix:** Копирование входного треугольника для работы с ним.
* **Начать цикл по строкам снизу вверх:** Начало цикла итерации по строкам треугольника, начиная с предпоследней строки.
* **Начать цикл по элементам в строке:**  Начало вложенного цикла итерации по элементам текущей строки.
* **Обновить значение element:** Обновление текущего элемента, добавляя к нему максимальное из двух соседних элементов следующей строки.
* **Вернуть значение triangle[0][0]:** Возвращаем значение верхнего элемента, которое содержит максимальную сумму пути.
