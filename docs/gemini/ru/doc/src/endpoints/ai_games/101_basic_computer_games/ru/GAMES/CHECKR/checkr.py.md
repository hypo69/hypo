# CHECKR

## Обзор

Игра "Шашки" - это текстовая реализация упрощенной версии игры в шашки. Игра происходит на доске 8x8, где игрок играет против компьютера. Цель игры - достичь противоположного конца доски одной из своих шашек, избегая при этом захвата шашек противника.

## Глобальные переменные

- `BOARD_SIZE`: Размер доски (8x8).
- `EMPTY`: Символ для пустой клетки (`.`).
- `PLAYER`: Символ для шашки игрока (`1`).
- `COMPUTER`: Символ для шашки компьютера (`2`).

## Функции

### `initialize_board`

**Описание**: Инициализирует доску 8x8 с начальным расположением шашек.

**Возвращает**:
- `list[list[str]]`: Двумерный список, представляющий доску с начальным расположением шашек.

### `draw_board`

**Описание**: Отрисовывает текущее состояние доски в консоль.

**Параметры**:
- `board` (list[list[str]]): Двумерный список, представляющий текущее состояние доски.

**Возвращает**:
- `None`: Функция ничего не возвращает, а выводит доску в консоль.

### `is_valid_move`

**Описание**: Проверяет, является ли ход игрока допустимым.

**Параметры**:
- `board` (list[list[str]]): Двумерный список, представляющий текущее состояние доски.
- `row` (int): Текущая строка шашки.
- `col` (int): Текущий столбец шашки.
- `new_row` (int): Новая строка, куда игрок хочет переместить шашку.
- `new_col` (int): Новый столбец, куда игрок хочет переместить шашку.
- `player` (str): Символ игрока ('1' или '2').

**Возвращает**:
- `bool`: Возвращает `True`, если ход допустим, `False` в противном случае.

### `update_board`

**Описание**: Обновляет доску после хода.

**Параметры**:
- `board` (list[list[str]]): Двумерный список, представляющий текущее состояние доски.
- `row` (int): Текущая строка шашки.
- `col` (int): Текущий столбец шашки.
- `new_row` (int): Новая строка, куда игрок хочет переместить шашку.
- `new_col` (int): Новый столбец, куда игрок хочет переместить шашку.

**Возвращает**:
- `None`: Функция ничего не возвращает, а изменяет переданную доску напрямую.

### `check_win`

**Описание**: Проверяет, достиг ли игрок или компьютер победы.

**Параметры**:
- `board` (list[list[str]]): Двумерный список, представляющий текущее состояние доски.
- `player` (str): Символ игрока ('1' или '2').

**Возвращает**:
- `bool`: Возвращает `True`, если игрок или компьютер достиг победы, `False` в противном случае.

### `get_computer_moves`

**Описание**: Находит все возможные ходы компьютера.

**Параметры**:
- `board` (list[list[str]]): Двумерный список, представляющий текущее состояние доски.

**Возвращает**:
- `list[tuple[int, int, int, int]]`: Список кортежей, представляющих возможные ходы компьютера в формате (row, col, new_row, new_col).

### `computer_turn`

**Описание**: Выполняет ход компьютера.

**Параметры**:
- `board` (list[list[str]]): Двумерный список, представляющий текущее состояние доски.

**Возвращает**:
- `None`: Функция ничего не возвращает, а изменяет переданную доску напрямую.

### `player_turn`

**Описание**: Запрашивает и выполняет ход игрока.

**Параметры**:
- `board` (list[list[str]]): Двумерный список, представляющий текущее состояние доски.

**Возвращает**:
- `None`: Функция ничего не возвращает, а изменяет переданную доску напрямую.

**Вызывает исключения**:
- `ValueError`: Возникает, если введены некорректные данные (не целые числа).

### `play_checkers`

**Описание**: Основная функция игры в шашки.

**Возвращает**:
- `None`: Функция ничего не возвращает, а управляет игровым процессом и выводит результаты.
