# Модуль helicone

## Обзор

Модуль `helicone` содержит константы и, вероятно, функции для работы с платформой Helicone AI.  Он предназначен для взаимодействия с API Helicone и инициализации подключений.

## Константы

### `MODE`

**Описание**:  Константа, хранящая режим работы.  В данном случае, установлено значение 'dev'.

**Значение**: `'dev'`
