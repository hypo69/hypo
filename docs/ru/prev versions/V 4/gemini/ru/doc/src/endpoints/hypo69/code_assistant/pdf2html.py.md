# Модуль конвертации PDF в HTML

## Обзор

Модуль предназначен для конвертации PDF-файлов в HTML-формат. В основном использует функциональность класса `PDFUtils` для выполнения преобразования. Расположен в подсистеме `hypo69` и используется для обработки файлов.

## Подробней

Этот модуль берет PDF-файл и конвертирует его в HTML-файл, что позволяет просматривать содержимое PDF в веб-браузере. Он использует `PDFUtils.pdf_to_html` для выполнения преобразования. Модуль задает пути к PDF и HTML файлам в файловой системе проекта.

## Функции

### `pdf2html`

```python
def pdf2html(pdf_file, html_file):
    """ """
```

**Описание**:
Функция выполняет конвертацию PDF-файла в HTML-файл с использованием `PDFUtils.pdf_to_html`.

**Параметры**:
- `pdf_file` (str): Путь к PDF-файлу.
- `html_file` (str): Путь к HTML-файлу, в который будет записан результат конвертации.

**Возвращает**:
- `None`: Функция ничего не возвращает.

**Вызывает исключения**:
- Отсутствуют явные исключения, но могут возникнуть исключения, связанные с работой `PDFUtils.pdf_to_html`.

**Примеры**:

```python
pdf_file = gs.path.root / 'assets' / 'materials' / '101_BASIC_Computer_Games_Mar75.pdf'
html_file = gs.path.root / 'assets' / 'materials' / '101_BASIC_Computer_Games_Mar75.html'

pdf2html(pdf_file, html_file)
```
