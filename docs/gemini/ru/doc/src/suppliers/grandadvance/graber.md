# Модуль hypotez/src/suppliers/grandadvance/graber.py

## Обзор

Этот модуль содержит класс `Graber`, предназначенный для сбора данных с веб-страниц товара на сайте `grandadvanse.co.il`. Класс наследуется от базового класса `Graber` из модуля `src.suppliers.graber`, добавляя специфическую логику для этого поставщика. В нём реализованы функции для обработки полей товаров, а также опциональный декоратор для закрытия всплывающих окон.

## Классы

### `Graber`

**Описание**: Класс `Graber` отвечает за сбор данных со страниц товаров от поставщика `grandadvance`. Он расширяет функциональность базового класса `Graber` из модуля `src.suppliers`.

**Атрибуты**:

- `supplier_prefix`: Строковое значение, представляющее префикс для идентификации поставщика (`grandadvance`).

**Методы**:

- `__init__(self, driver: Driver)`:
    **Описание**: Инициализирует экземпляр класса `Graber`.
    **Параметры**:
    - `driver (Driver)`: Объект драйвера веб-драйвера, используемый для взаимодействия с браузером.
    **Возвращает**:
    - Не имеет возвращаемого значения.
    **Примечания**: Вызывает конструктор базового класса `super().__init__(...)` для инициализации общих атрибутов и методов. Устанавливает `supplier_prefix`. Устанавливает глобальные настройки через `Context`.
