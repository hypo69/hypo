# Модуль `scenario.py`

## Обзор

Модуль `scenario.py` предназначен для сбора данных о товарах с сайта поставщика Banggood через веб-драйвер. Он содержит функции для получения списка категорий товаров и списка товаров в каждой категории.

## Подробней

Этот модуль играет ключевую роль в процессе извлечения информации о товарах с сайта Banggood. Он использует веб-драйвер для навигации по сайту, сбора ссылок на товары и передачи этих ссылок для дальнейшей обработки. Модуль также содержит функциональность для обработки возможных изменений в структуре сайта Banggood, таких как добавление, переименование или удаление категорий.
