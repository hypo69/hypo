# Модуль конвертации PDF в HTML

## Обзор

Этот модуль предназначен для конвертации PDF-файлов в HTML-формат. Он использует утилиты из `src.utils.pdf` и `src.gs` для обработки файлов.

## Подробней

Модуль предоставляет функцию `pdf2html`, которая принимает путь к PDF-файлу и путь для сохранения HTML-файла. В основном используется для преобразования документов в формат, удобный для веб-просмотра.

## Функции

### `pdf2html`

def pdf2html(pdf_file, html_file):
    """ """
```

**Как работает функция**:
Функция `pdf2html` принимает два параметра: путь к PDF-файлу (`pdf_file`) и путь к HTML-файлу (`html_file`). Она использует метод `pdf_to_html` из класса `PDFUtils` для конвертации PDF в HTML.

**Параметры**:
- `pdf_file`: Путь к PDF-файлу, который необходимо конвертировать.
- `html_file`: Путь, по которому будет сохранен HTML-файл.

**Возвращает**:
- None

**Вызывает исключения**:
- Отсутствуют явные исключения, но могут быть вызваны исключения внутри `PDFUtils.pdf_to_html`.

**Примеры**:

```python
pdf_file = gs.path.root / 'assets' / 'materials' / '101_BASIC_Computer_Games_Mar75.pdf'
html_file = gs.path.root / 'assets' / 'materials' / '101_BASIC_Computer_Games_Mar75.html'

pdf2html(pdf_file, html_file)
