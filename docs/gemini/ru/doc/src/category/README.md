# Модуль категоризации данных

## Обзор

Этот модуль предоставляет инструменты для категоризации данных о продуктах, полученных от поставщиков.  Он включает в себя класс `Product` для представления данных о продукте и класс `Supplier` для представления данных о поставщике.

## Оглавление

* [Модуль категоризации данных](#модуль-категоризации-данных)
* [Класс Product](#класс-product)
* [Класс Supplier](#класс-supplier)


## Класс Product

### `Product`

**Описание**: Представляет данные о продукте, включая его идентификатор, название и категорию.

**Атрибуты**:

- `product_id` (int): Идентификатор продукта.
- `name` (str): Название продукта.
- `category` (str): Категория продукта.


## Класс Supplier

### `Supplier`

**Описание**: Представляет данные о поставщике, включая его идентификатор и имя.

**Атрибуты**:

- `supplier_id` (int): Идентификатор поставщика.
- `name` (str): Название поставщика.
