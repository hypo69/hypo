# Модуль для логирования времени выполнения функций

## Обзор

Модуль содержит функции для измерения и логирования времени выполнения других функций. Предоставляет три варианта логирования: для асинхронных функций, функций-генераторов и обычных функций.

## Подробней

Этот модуль предназначен для упрощения процесса измерения времени выполнения функций в проекте `hypotez`. Он предоставляет удобные декораторы (`log_time_async`, `log_time_yield` и `log_time`) для автоматического логирования времени выполнения функций различных типов (асинхронных, генераторов и обычных). Функции возвращают результат работы декорируемой функции вместе с информацией о времени выполнения.

## Функции

### `log_time_async`

```python
async def log_time_async(method: callable, **kwargs):
    """ Асинхронно измеряет время выполнения метода и возвращает результат вместе с временем.

    Args:
        method (callable): Асинхронный метод, время выполнения которого необходимо измерить.
        **kwargs: Произвольные ключевые аргументы, передаваемые в метод.

    Returns:
        str: Результат выполнения метода, объединенный со строкой, содержащей время выполнения в секундах. Если результат отсутствует, возвращается только строка времени.

    Как работает функция:
    1. Фиксирует время начала выполнения метода.
    2. Вызывает асинхронный метод с переданными аргументами и ожидает его завершения.
    3. Вычисляет время выполнения метода.
    4. Форматирует время выполнения в строку.
    5. Если результат выполнения метода существует, объединяет его со строкой времени и возвращает. В противном случае возвращает только строку времени.

    Внутри функции происходят следующие действия и преобразования:
     start_time -> result_method -> end_time -> result_time

     Где `start_time` - фиксируется время начала выполнения, `result_method` - выполнение асинхронного метода и получение результата, `end_time` - фиксируется время окончания выполнения, `result_time` - вычисление времени выполнения и форматирование результата.
    """
```

### `log_time_yield`

```python
def log_time_yield(method: callable, **kwargs):
    """ Измеряет время выполнения метода-генератора и возвращает генератор, выдающий результаты вместе с временем выполнения.

    Args:
        method (callable): Метод-генератор, время выполнения которого необходимо измерить.
        **kwargs: Произвольные ключевые аргументы, передаваемые в метод.

    Yields:
        str: Результаты выполнения метода-генератора с добавленной информацией о времени выполнения в конце.

    Как работает функция:
    1. Фиксирует время начала выполнения метода-генератора.
    2. Вызывает метод-генератор с переданными аргументами.
    3. Передает результаты, возвращаемые генератором, вызывающей стороне.
    4. После завершения работы генератора вычисляет время выполнения.
    5. Добавляет информацию о времени выполнения к последнему результату, возвращаемому генератором.

    Внутри функции происходят следующие действия и преобразования:
     start_time -> result_method -> end_time -> result_time

     Где `start_time` - фиксируется время начала выполнения, `result_method` - выполнение метода-генератора и получение результата, `end_time` - фиксируется время окончания выполнения, `result_time` - вычисление времени выполнения и добавление информации о времени к результату.
    """
```
