# –ú–Ω–æ–∂–µ—Å—Ç–≤–∞ (–Ω–∞–±–æ—Ä—ã) –≤ python

**1. –í–≤–µ–¥–µ–Ω–∏–µ: –ß—Ç–æ —Ç–∞–∫–æ–µ –º–Ω–æ–∂–µ—Å—Ç–≤–∞?**

–í –∏–Ω—Ñ–æ—Ä–º–∞—Ç–∏–∫–µ –∏ –º–∞—Ç–µ–º–∞—Ç–∏–∫–µ –º–Ω–æ–∂–µ—Å—Ç–≤–∞ ‚Äì —ç—Ç–æ —Å–ø–æ—Å–æ–± –ø—Ä–µ–¥—Å—Ç–∞–≤–∏—Ç—å –Ω–∞–±–æ—Ä—ã —É–Ω–∏–∫–∞–ª—å–Ω—ã—Ö —ç–ª–µ–º–µ–Ω—Ç–æ–≤. –í–∞–∂–Ω–æ –ø–æ–Ω–∏–º–∞—Ç—å, —á—Ç–æ:

*   **–£–Ω–∏–∫–∞–ª—å–Ω–æ—Å—Ç—å:** –ö–∞–∂–¥—ã–π —ç–ª–µ–º–µ–Ω—Ç –≤ –º–Ω–æ–∂–µ—Å—Ç–≤–µ –¥–æ–ª–∂–µ–Ω –±—ã—Ç—å —É–Ω–∏–∫–∞–ª—å–Ω—ã–º. –ü–æ–≤—Ç–æ—Ä–µ–Ω–∏—è –Ω–µ –¥–æ–ø—É—Å–∫–∞—é—Ç—Å—è.
*   **–ù–µ—É–ø–æ—Ä—è–¥–æ—á–µ–Ω–Ω–æ—Å—Ç—å:** –ü–æ—Ä—è–¥–æ–∫ —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –≤ –º–Ω–æ–∂–µ—Å—Ç–≤–µ –Ω–µ –∏–º–µ–µ—Ç –∑–Ω–∞—á–µ–Ω–∏—è.


**2. –ú–Ω–æ–∂–µ—Å—Ç–≤–∞ –∏ —Ñ—Ä—É–∫—Ç—ã**

–ú—ã –ø—Ä–µ–¥—Å—Ç–∞–≤–∏–º —Å–µ–±–µ, —á—Ç–æ —É –Ω–∞—Å –µ—Å—Ç—å —Ç–æ–ª—å–∫–æ —Å–∞–º–∏ —Ñ—Ä—É–∫—Ç—ã:

*   üçé (—è–±–ª–æ–∫–æ)
*   üçê (–≥—Ä—É—à–∞)
*   üçâ (–¥—ã–Ω—è)
*   üß∫ (–∫–æ—Ä–∑–∏–Ω–∫–∞)

–í–∞–∂–Ω–æ, —á—Ç–æ:

1.  **–í –Ω–∞–±–æ—Ä–µ *–Ω–µ—Ç* –æ–¥–∏–Ω–∞–∫–æ–≤—ã—Ö —Ñ—Ä—É–∫—Ç–æ–≤:** –ï—Å–ª–∏ –≤ –Ω–∞–±–æ—Ä–µ –µ—Å—Ç—å —è–±–ª–æ–∫–æ, —Ç–æ —Ç–∞–º –Ω–µ –±—É–¥–µ—Ç –¥—Ä—É–≥–æ–≥–æ —Ç–∞–∫–æ–≥–æ –∂–µ —è–±–ª–æ–∫–∞. –ö–∞–∂–¥—ã–π —Ñ—Ä—É–∫—Ç —É–Ω–∏–∫–∞–ª–µ–Ω –≤ —Å–≤–æ–µ–º –Ω–∞–±–æ—Ä–µ.
2.  **–ü–æ—Ä—è–¥–æ–∫ —Ñ—Ä—É–∫—Ç–æ–≤ –Ω–µ –≤–∞–∂–µ–Ω:** –ï—Å–ª–∏ –≤ –Ω–∞–±–æ—Ä–µ –µ—Å—Ç—å —è–±–ª–æ–∫–æ –∏ –≥—Ä—É—à–∞, —Ç–æ —ç—Ç–æ —Ç–æ –∂–µ —Å–∞–º–æ–µ, —á—Ç–æ –µ—Å–ª–∏ –±—ã —Ç–∞–º –±—ã–ª–∏ –≥—Ä—É—à–∞ –∏ —è–±–ª–æ–∫–æ. –ü–æ—Ä—è–¥–æ–∫ –Ω–µ –∏–º–µ–µ—Ç –∑–Ω–∞—á–µ–Ω–∏—è.

–ù–∞–ø—Ä–∏–º–µ—Ä, `{üçé, üçê, üçâ}` ‚Äî —ç—Ç–æ –º–Ω–æ–∂–µ—Å—Ç–≤–æ, –≥–¥–µ –µ—Å—Ç—å —è–±–ª–æ–∫–æ, –≥—Ä—É—à–∞ –∏ –¥—ã–Ω—è.

**3. –ü–æ—á–µ–º—É –º–Ω–æ–∂–µ—Å—Ç–≤–∞ —Ç—Ä–µ–±—É—é—Ç —É–Ω–∏–∫–∞–ª—å–Ω–æ—Å—Ç–∏ —ç–ª–µ–º–µ–Ω—Ç–æ–≤?**

*   –ú–Ω–æ–∂–µ—Å—Ç–≤–∞ –ø—Ä–µ–¥–Ω–∞–∑–Ω–∞—á–µ–Ω—ã –¥–ª—è –æ—Ç—Å–ª–µ–∂–∏–≤–∞–Ω–∏—è *–Ω–∞–ª–∏—á–∏—è* —ç–ª–µ–º–µ–Ω—Ç–æ–≤, –∞ –Ω–µ –∏—Ö *–∫–æ–ª–∏—á–µ—Å—Ç–≤–∞*.
*   –£–Ω–∏–∫–∞–ª—å–Ω–æ—Å—Ç—å —ç–ª–µ–º–µ–Ω—Ç–æ–≤ —É–ø—Ä–æ—â–∞–µ—Ç –≤—ã–ø–æ–ª–Ω–µ–Ω–∏–µ –æ–ø–µ—Ä–∞—Ü–∏–π –Ω–∞–¥ –º–Ω–æ–∂–µ—Å—Ç–≤–∞–º–∏.
*   –ú–Ω–æ–∂–µ—Å—Ç–≤–∞ –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è –¥–ª—è –∏—Å–∫–ª—é—á–µ–Ω–∏—è –∏–∑–±—ã—Ç–æ—á–Ω–æ—Å—Ç–∏ –¥–∞–Ω–Ω—ã—Ö.

**4. –û–ø–µ—Ä–∞—Ü–∏–∏ –Ω–∞–¥ –º–Ω–æ–∂–µ—Å—Ç–≤–∞–º–∏ (–Ω–∞–±–æ—Ä–∞–º–∏ —Ñ—Ä—É–∫—Ç–æ–≤):**

1.  **–û–±—ä–µ–¥–∏–Ω–µ–Ω–∏–µ (Union): "–°–æ–±–∏—Ä–∞–µ–º –≤—Å–µ —Ñ—Ä—É–∫—Ç—ã –≤ –æ–¥–∏–Ω –Ω–∞–±–æ—Ä"**
    *   –û–±—ä–¥–∏–Ω—è—é –¥–≤–∞ –Ω–∞–±–æ—Ä–∞ —Ñ—Ä—É–∫—Ç–æ–≤, —Å–æ–±—Ä–∞–≤ –≤—Å–µ —Ñ—Ä—É–∫—Ç—ã –≤ –Ω–æ–≤—ã–π –Ω–∞–±–æ—Ä. –ï—Å–ª–∏ –≤ –¥–≤—É—Ö –Ω–∞–±–æ—Ä–∞—Ö –µ—Å—Ç—å –æ–¥–∏–Ω–∞–∫–æ–≤—ã–µ —Ñ—Ä—É–∫—Ç—ã, –≤ –Ω–æ–≤–æ–º –Ω–∞–±–æ—Ä–µ –±—É–¥–µ—Ç —Ç–æ–ª—å–∫–æ –æ–¥–∏–Ω —Ç–∞–∫–æ–π —Ñ—Ä—É–∫—Ç.
    *   –ï—Å–ª–∏ –≤ –Ω–∞–±–æ—Ä–µ A –µ—Å—Ç—å {üçé, üçê}, –∞ –≤ –Ω–∞–±–æ—Ä–µ B –µ—Å—Ç—å {üçê, üçâ}, —Ç–æ –≤ –Ω–∞–±–æ—Ä–µ A ‚à™ B –±—É–¥–µ—Ç {üçé, üçê, üçâ}.

2.  **–ü–µ—Ä–µ—Å–µ—á–µ–Ω–∏–µ (Intersection): "–ù–∞—Ö–æ–¥–∏–º –æ–±—â–∏–µ —Ñ—Ä—É–∫—Ç—ã"**
    *   –ò—â—É *—Ç–æ–ª—å–∫–æ* —Ç–µ —Ñ—Ä—É–∫—Ç—ã, –∫–æ—Ç–æ—Ä—ã–µ –µ—Å—Ç—å –∏ –≤ –Ω–∞–±–æ—Ä–µ A, –∏ –≤ –Ω–∞–±–æ—Ä–µ B.
    *   –ï—Å–ª–∏ –≤ –Ω–∞–±–æ—Ä–µ A –µ—Å—Ç—å {üçé, üçê}, –∞ –≤ –Ω–∞–±–æ—Ä–µ B –µ—Å—Ç—å {üçê, üçâ}, —Ç–æ –≤ –Ω–∞–±–æ—Ä–µ A ‚à© B –±—É–¥–µ—Ç —Ç–æ–ª—å–∫–æ {üçê}.

3.  **–†–∞–∑–Ω–æ—Å—Ç—å (Difference): "–§—Ä—É–∫—Ç—ã, –∫–æ—Ç–æ—Ä—ã–µ –µ—Å—Ç—å –≤ –æ–¥–Ω–æ–º –Ω–∞–±–æ—Ä–µ, –Ω–æ –Ω–µ—Ç –≤ –¥—Ä—É–≥–æ–º"**
    *   –ò—â—É *—Ç–æ–ª—å–∫–æ* —Ç–µ —Ñ—Ä—É–∫—Ç—ã, –∫–æ—Ç–æ—Ä—ã–µ –µ—Å—Ç—å –≤ –Ω–∞–±–æ—Ä–µ A, –Ω–æ –∫–æ—Ç–æ—Ä—ã—Ö –Ω–µ—Ç –≤ –Ω–∞–±–æ—Ä–µ B.
    *   –ï—Å–ª–∏ –≤ –Ω–∞–±–æ—Ä–µ A –µ—Å—Ç—å {üçé, üçê}, –∞ –≤ –Ω–∞–±–æ—Ä–µ B –µ—Å—Ç—å {üçê, üçâ}, —Ç–æ –≤ –Ω–∞–±–æ—Ä–µ A - B –±—É–¥–µ—Ç {üçé}, –∞ –≤ –Ω–∞–±–æ—Ä–µ B - A –±—É–¥–µ—Ç {üçâ}.

4.  **–°–∏–º–º–µ—Ç—Ä–∏—á–µ—Å–∫–∞—è —Ä–∞–∑–Ω–æ—Å—Ç—å (Symmetric Difference): "–§—Ä—É–∫—Ç—ã, –∫–æ—Ç–æ—Ä—ã–µ –µ—Å—Ç—å —Ç–æ–ª—å–∫–æ –≤ –æ–¥–Ω–æ–º –∏–∑ –Ω–∞–±–æ—Ä–æ–≤"**
    *   –ò—â—É *–≤—Å–µ* —Ñ—Ä—É–∫—Ç—ã, –∫–æ—Ç–æ—Ä—ã–µ –µ—Å—Ç—å –ª–∏–±–æ –≤ –Ω–∞–±–æ—Ä–µ A, –ª–∏–±–æ –≤ –Ω–∞–±–æ—Ä–µ B, –Ω–æ –Ω–µ –≤ –æ–±–æ–∏—Ö –æ–¥–Ω–æ–≤—Ä–µ–º–µ–Ω–Ω–æ.
    *   –ï—Å–ª–∏ –≤ –Ω–∞–±–æ—Ä–µ A –µ—Å—Ç—å {üçé, üçê}, –∞ –≤ –Ω–∞–±–æ—Ä–µ B –µ—Å—Ç—å {üçê, üçâ}, —Ç–æ –≤ –Ω–∞–±–æ—Ä–µ A ^ B –±—É–¥–µ—Ç {üçé, üçâ}.

5.  **–ü–æ–¥–º–Ω–æ–∂–µ—Å—Ç–≤–æ (Subset): "–í—Å–µ –ª–∏ —Ñ—Ä—É–∫—Ç—ã –∏–∑ –æ–¥–Ω–æ–≥–æ –Ω–∞–±–æ—Ä–∞ –µ—Å—Ç—å –≤ –¥—Ä—É–≥–æ–º?"**
    *   –ü—Ä–æ–≤–µ—Ä—è—é, —è–≤–ª—è–µ—Ç—Å—è –ª–∏ –Ω–∞–±–æ—Ä A –ø–æ–¥–º–Ω–æ–∂–µ—Å—Ç–≤–æ–º –Ω–∞–±–æ—Ä–∞ B. –≠—Ç–æ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ –≤—Å–µ —Ñ—Ä—É–∫—Ç—ã –∏–∑ –Ω–∞–±–æ—Ä–∞ A –µ—Å—Ç—å –∏ –≤ –Ω–∞–±–æ—Ä–µ B.
    *   **–ü—Ä–∏–º–µ—Ä:** –ï—Å–ª–∏ –≤ –Ω–∞–±–æ—Ä–µ A –µ—Å—Ç—å {üçé, üçê}, –∞ –≤ –Ω–∞–±–æ—Ä–µ B –µ—Å—Ç—å {üçé, üçê, üçâ}, —Ç–æ A <= B.

6.  **–ù–∞–¥–º–Ω–æ–∂–µ—Å—Ç–≤–æ (Superset): "–°–æ–¥–µ—Ä–∂–∏—Ç –ª–∏ –æ–¥–∏–Ω –Ω–∞–±–æ—Ä –≤—Å–µ —Ñ—Ä—É–∫—Ç—ã –∏–∑ –¥—Ä—É–≥–æ–≥–æ?"**
    *   –ü—Ä–æ–≤–µ—Ä—è—é, —è–≤–ª—è–µ—Ç—Å—è –ª–∏ –Ω–∞–±–æ—Ä A –Ω–∞–¥–º–Ω–æ–∂–µ—Å—Ç–≤–æ–º –Ω–∞–±–æ—Ä–∞ B. –≠—Ç–æ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ –≤—Å–µ —Ñ—Ä—É–∫—Ç—ã –∏–∑ –Ω–∞–±–æ—Ä–∞ B –µ—Å—Ç—å –∏ –≤ –Ω–∞–±–æ—Ä–µ A.
    *   –ï—Å–ª–∏ –≤ –Ω–∞–±–æ—Ä–µ A –µ—Å—Ç—å {üçé, üçê, üçâ}, –∞ –≤ –Ω–∞–±–æ—Ä–µ B –µ—Å—Ç—å {üçé, üçê}, —Ç–æ A >= B.



```python
from typing import Set

def create_fruit_set(fruit_string: str) -> Set[str]:
    """
    –°–æ–∑–¥–∞–µ—Ç –º–Ω–æ–∂–µ—Å—Ç–≤–æ —Ñ—Ä—É–∫—Ç–æ–≤ –∏–∑ —Å—Ç—Ä–æ–∫–∏.

    Args:
        fruit_string: –°—Ç—Ä–æ–∫–∞ —Å —Ñ—Ä—É–∫—Ç–∞–º–∏ (üçé, üçê, üçâ, üß∫).

    Returns:
        –ú–Ω–æ–∂–µ—Å—Ç–≤–æ —É–Ω–∏–∫–∞–ª—å–Ω—ã—Ö —Ñ—Ä—É–∫—Ç–æ–≤.
    """
    if not all(fruit in ["üçé", "üçê", "üçâ", "üß∫"] for fruit in fruit_string):
        raise ValueError("–°—Ç—Ä–æ–∫–∞ –º–æ–∂–µ—Ç —Å–æ–¥–µ—Ä–∂–∞—Ç—å —Ç–æ–ª—å–∫–æ —Å–∏–º–≤–æ–ª—ã üçé, üçê, üçâ, üß∫")
    return set(fruit_string)  # –ò—Å–ø–æ–ª—å–∑—É–µ–º set() –¥–ª—è —Å–æ–∑–¥–∞–Ω–∏—è –º–Ω–æ–∂–µ—Å—Ç–≤–∞

def display_set(fruit_set: Set[str]) -> str:
  """
  –ü—Ä–µ–æ–±—Ä–∞–∑—É–µ—Ç –º–Ω–æ–∂–µ—Å—Ç–≤–æ —Ñ—Ä—É–∫—Ç–æ–≤ –≤ —Å—Ç—Ä–æ–∫—É –¥–ª—è –æ—Ç–æ–±—Ä–∞–∂–µ–Ω–∏—è.

    Args:
        fruit_set: –ú–Ω–æ–∂–µ—Å—Ç–≤–æ —Ñ—Ä—É–∫—Ç–æ–≤.

    Returns:
        –°—Ç—Ä–æ–∫–∞ –¥–ª—è –æ—Ç–æ–±—Ä–∞–∂–µ–Ω–∏—è.
  """
  return "{" + ", ".join(fruit_set) + "}"


# –°–æ–∑–¥–∞–µ–º –º–Ω–æ–∂–µ—Å—Ç–≤–∞ —Ñ—Ä—É–∫—Ç–æ–≤
fruits_set_A = create_fruit_set("üçéüçê")  # –ú–Ω–æ–∂–µ—Å—Ç–≤–æ A: {üçé, üçê}
fruits_set_B = create_fruit_set("üçêüçâ")  # –ú–Ω–æ–∂–µ—Å—Ç–≤–æ B: {üçê, üçâ}
fruits_set_C = create_fruit_set("üçéüçêüçâ")  # –ú–Ω–æ–∂–µ—Å—Ç–≤–æ C: {üçé, üçê, üçâ}
fruits_set_D = create_fruit_set("üß∫")  # –ú–Ω–æ–∂–µ—Å—Ç–≤–æ D: {üß∫}

# –í—ã–≤–æ–¥–∏–º –º–Ω–æ–∂–µ—Å—Ç–≤–∞
print(f"–ú–Ω–æ–∂–µ—Å—Ç–≤–æ A: {display_set(fruits_set_A)}")
print(f"–ú–Ω–æ–∂–µ—Å—Ç–≤–æ B: {display_set(fruits_set_B)}")
print(f"–ú–Ω–æ–∂–µ—Å—Ç–≤–æ C: {display_set(fruits_set_C)}")
print(f"–ú–Ω–æ–∂–µ—Å—Ç–≤–æ D: {display_set(fruits_set_D)}")

# –û–±—ä–µ–¥–∏–Ω–µ–Ω–∏–µ –º–Ω–æ–∂–µ—Å—Ç–≤ (Union)
union_result = fruits_set_A | fruits_set_B
print(f"A ‚à™ B: {display_set(union_result)}")  # –†–µ–∑—É–ª—å—Ç–∞—Ç: {üçé, üçê, üçâ}

# –ü–µ—Ä–µ—Å–µ—á–µ–Ω–∏–µ –º–Ω–æ–∂–µ—Å—Ç–≤ (Intersection)
intersection_result = fruits_set_A & fruits_set_B
print(f"A ‚à© B: {display_set(intersection_result)}")  # –†–µ–∑—É–ª—å—Ç–∞—Ç: {üçê}

# –†–∞–∑–Ω–æ—Å—Ç—å –º–Ω–æ–∂–µ—Å—Ç–≤ (Difference)
difference_result_AB = fruits_set_A - fruits_set_B
print(f"A - B: {display_set(difference_result_AB)}")  # –†–µ–∑—É–ª—å—Ç–∞—Ç: {üçé}
difference_result_BA = fruits_set_B - fruits_set_A
print(f"B - A: {display_set(difference_result_BA)}")  # –†–µ–∑—É–ª—å—Ç–∞—Ç: {üçâ}

# –°–∏–º–º–µ—Ç—Ä–∏—á–Ω–∞—è —Ä–∞–∑–Ω–æ—Å—Ç—å –º–Ω–æ–∂–µ—Å—Ç–≤ (Symmetric Difference)
symmetric_difference_result = fruits_set_A ^ fruits_set_B
print(f"A ^ B: {display_set(symmetric_difference_result)}")  # –†–µ–∑—É–ª—å—Ç–∞—Ç: {üçé, üçâ}

# –ü–æ–¥–º–Ω–æ–∂–µ—Å—Ç–≤–æ (Subset)
subset_result1 = fruits_set_A <= fruits_set_C
print(f"A <= C: {subset_result1}")  # –†–µ–∑—É–ª—å—Ç–∞—Ç: True (A –ø–æ–¥–º–Ω–æ–∂–µ—Å—Ç–≤–æ C)
subset_result2 = fruits_set_A <= fruits_set_B
print(f"A <= B: {subset_result2}") # –†–µ–∑—É–ª—å—Ç–∞—Ç: False (A –Ω–µ –ø–æ–¥–º–Ω–æ–∂–µ—Å—Ç–≤–æ B)

# –ù–∞–¥–º–Ω–æ–∂–µ—Å—Ç–≤–æ (Superset)
superset_result1 = fruits_set_C >= fruits_set_A
print(f"C >= A: {superset_result1}")  # –†–µ–∑—É–ª—å—Ç–∞—Ç: True (C –Ω–∞–¥–º–Ω–æ–∂–µ—Å—Ç–≤–æ A)
superset_result2 = fruits_set_B >= fruits_set_A
print(f"B >= A: {superset_result2}")  # –†–µ–∑—É–ª—å—Ç–∞—Ç: False (B –Ω–µ –Ω–∞–¥–º–Ω–æ–∂–µ—Å—Ç–≤–æ A)

# –ü—Ä–æ–≤–µ—Ä–∫–∞ –Ω–∞ –Ω–∞–ª–∏—á–∏–µ —ç–ª–µ–º–µ–Ω—Ç–∞
print(f"üçé in A: {'üçé' in fruits_set_A}")  # –†–µ–∑—É–ª—å—Ç–∞—Ç: True
print(f"üçâ in A: {'üçâ' in fruits_set_A}")  # –†–µ–∑—É–ª—å—Ç–∞—Ç: False
```

*   **`create_fruit_set(fruit_string)`:** –≠—Ç–∞ —Ñ—É–Ω–∫—Ü–∏—è —Å–æ–∑–¥–∞–µ—Ç –º–Ω–æ–∂–µ—Å—Ç–≤–æ –∏–∑ —Å—Ç—Ä–æ–∫–∏ —Å —Ñ—Ä—É–∫—Ç–∞–º–∏.
