# Модуль `hypotez/src/suppliers/aliexpress/api/models/currencies.py`

## Обзор

Этот модуль содержит класс `Currency` для определения валют, используемых в API AliExpress.  В нем перечислены различные валюты в виде констант.

## Классы

### `Currency`

**Описание**: Класс `Currency` содержит константы, представляющие различные валюты.

**Константы**:

- `USD`: Доллар США
- `GBP`: Фунт стерлингов
- `CAD`: Канадский доллар
- `EUR`: Евро
- `UAH`: Гривна
- `MXN`: Мексиканский песо
- `TRY`: Турецкая лира
- `RUB`: Российский рубль
- `BRL`: Бразильский реаль
- `AUD`: Австралийский доллар
- `INR`: Индийская рупия
- `JPY`: Японская иена
- `IDR`: Индонезийская рупия
- `SEK`: Шведская крона
- `KRW`: Южнокорейская вона
- `ILS`: Израильский шекель
