# Модуль `src.scenario`

## Обзор

Модуль `src.scenario` предоставляет функции для выполнения сценариев, связанных с поставщиками. Он включает в себя функции для запуска сценариев из файлов и непосредственно из словарей.

## Функции

### `run_scenario`

**Описание**: Выполняет одиночный сценарий для заданного поставщика.

**Параметры**:
- `supplier` (Supplier): Поставщик, для которого выполняется сценарий.
- `scenario` (dict): Словарь, представляющий сценарий.

**Возвращает**:
- `None`

### `run_scenarios`

**Описание**: Выполняет список сценариев для заданного поставщика.

**Параметры**:
- `supplier` (Supplier): Поставщик, для которого выполняются сценарии.
- `scenarios` (list[dict]): Список словарей, представляющих сценарии.

**Возвращает**:
- `None`

### `run_scenario_file`

**Описание**: Выполняет сценарии из файла для заданного поставщика.

**Параметры**:
- `supplier` (Supplier): Поставщик, для которого выполняются сценарии.
- `scenario_file` (str): Путь к файлу со сценариями.

**Возвращает**:
- `None`

### `run_scenario_files`

**Описание**: Выполняет сценарии из списка файлов для заданного поставщика.

**Параметры**:
- `supplier` (Supplier): Поставщик, для которого выполняются сценарии.
- `scenario_files` (list[str]): Список путей к файлам со сценариями.

**Возвращает**:
- `None`

### `insert_grabbed_data_to_prestashop`

**Описание**: Вставляет полученные данные в PrestaShop.

**Параметры**:
- `supplier` (Supplier): Поставщик, для которого выполняются сценарии.
- `data` (dict): Словарь с данными для вставки.

**Возвращает**:
- `None`
