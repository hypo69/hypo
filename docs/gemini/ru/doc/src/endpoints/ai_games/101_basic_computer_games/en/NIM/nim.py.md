# NIM

## Обзор

Этот модуль реализует игру NIM, в которой два игрока по очереди берут камни из нескольких кучек. Цель игры - забрать последний камень.

## Функции

### `display_piles`

**Описание**: Выводит текущее состояние кучек камней.

**Параметры**:
- `piles` (list): Список, представляющий кучки камней.

**Возвращает**:
- `None`

### `get_player_move`

**Описание**: Запрашивает у игрока номер кучки и количество камней для удаления.

**Параметры**:
- `piles` (list): Список, представляющий кучки камней.
- `player` (int): Номер текущего игрока (1 или 2).

**Возвращает**:
- `tuple`: Кортеж, содержащий номер кучки (int) и количество камней для удаления (int).

**Вызывает исключения**:
- `ValueError`: Возникает, если ввод не является целым числом.

### `play_nim`

**Описание**: Реализует игру NIM.

**Параметры**:
- `None`

**Возвращает**:
- `None`
