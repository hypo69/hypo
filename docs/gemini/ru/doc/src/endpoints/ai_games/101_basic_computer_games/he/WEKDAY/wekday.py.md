# WEKDAY

## Обзор

Этот модуль реализует игру WEKDAY, которая определяет день недели для введенной даты. Пользователь вводит месяц и день, а программа вычисляет и выводит соответствующий день недели.

## Функции

### `main`

**Описание**: Основная функция программы, которая реализует игру WEKDAY.

**Параметры**:
- Нет

**Возвращает**:
- Нет

**Вызывает исключения**:
- `ValueError`: Возникает, если пользователь вводит некорректные данные (не целые числа).

**Алгоритм**:

1.  Инициализирует переменную `day` значением 2, что соответствует понедельнику (1 января 1900 года).
2.  Запускает бесконечный цикл `while True`, позволяющий пользователю вводить несколько дат.
3.  Внутри цикла:
    -   Запрашивает у пользователя месяц (1-12) и день месяца.
    -   Обрабатывает исключение `ValueError`, если ввод не является целым числом.
    -   Проверяет корректность введенного месяца (1-12).
    -   Проверяет корректность введенного дня (должен быть больше 0).
    -   Инициализирует список `daysInMonth` с количеством дней в каждом месяце.
    -   Проверяет, что введенный день не выходит за пределы допустимого количества дней для указанного месяца.
    -   Вычисляет общее количество дней с начала года до введенной даты.
    -   Вычисляет день недели на основе общего количества дней.
    -   Инициализирует список `dayNames` с названиями дней недели.
    -   Выводит день недели.
    -   Предлагает пользователю сыграть еще раз или закончить игру.
4.  Если пользователь не хочет продолжать, программа завершается.
```python
def main():
    """
    Основная функция программы, которая реализует игру WEKDAY.

    Args:
        None

    Returns:
        None

    Raises:
        ValueError: Возникает, если пользователь вводит некорректные данные (не целые числа).
    """
    # יום בשבוע של 1/1/1900 (יום שני)
    day = 2

    # לולאה ראשית של המשחק
    while True:
        # קבלת קלט מהמשתמש: חודש ומספר יום
        try:
            month = int(input("הכנס חודש (1-12): "))
            dayNumber = int(input("הכנס יום בחודש: "))
        except ValueError as ex:
            print("קלט לא תקין, אנא הכנס מספרים שלמים.")
            continue

        # בדיקת תקינות החודש
        if month < 1 or month > 12:
            print("חודש לא תקין, אנא הכנס חודש בין 1 ל-12.")
            continue

        # בדיקת תקינות מספר היום
        if dayNumber < 1:
            print("מספר יום לא תקין, אנא הכנס מספר יום חיובי.")
            continue

        # מערך המכיל את מספר הימים בכל חודש
        daysInMonth = [31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31]

        # בדיקה האם מספר היום גדול ממספר הימים בחודש
        if (month == 2 and dayNumber > 28) or (month != 2 and dayNumber > daysInMonth[month - 1]):
             print("מספר יום לא תקין עבור החודש הזה.")
             continue

        # חישוב מספר הימים הכולל מתחילת השנה
        totalDays = sum(daysInMonth[0:month - 1]) + dayNumber

        # חישוב היום בשבוע
        day = (totalDays % 7) + 1

        # מערך שמות הימים
        dayNames = ['ראשון', 'שני', 'שלישי', 'רביעי', 'חמישי', 'שישי', 'שבת']

        # הדפסת היום בשבוע המתאים
        print(f"היום בשבוע הוא: {dayNames[day - 1]}")

        # אפשרות לסיים את המשחק
        again = input("האם תרצה לחשב תאריך נוסף? (כן/לא): ")
        if again.lower() != 'כן':
          print("תודה ושלום!")
          break
```
