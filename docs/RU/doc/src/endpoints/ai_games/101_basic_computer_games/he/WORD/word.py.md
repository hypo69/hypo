# FROG Game

## Обзор

Этот файл содержит реализацию игры "Лягушка", в которой игрок управляет лягушкой, пытающейся перебраться через реку, прыгая по камням. Цель игры - достичь противоположного берега (камня 12), избегая падения в воду.

## Функции

### `play_frog_game`

**Описание**: Запускает игру "Лягушка". Игрок должен перепрыгнуть через реку по камням, начиная с позиции 1 и заканчивая на позиции 12.

**Параметры**:
- Нет параметров.

**Возвращает**:
- Нет возвращаемого значения.

**Вызывает исключения**:
- `ValueError`: Возникает, если пользователь вводит не число при запросе на ввод номера камня.

## Пример Использования

```python
if __name__ == "__main__":
    play_frog_game()
```
Этот код запустит игру, если файл выполняется как основной скрипт.

## Подробное описание

1. **Импорт модуля `sys`**:
   - `import sys`: Импортирует модуль `sys`, который не используется напрямую в текущей версии игры.

2. **Функция `play_frog_game()`**:
   - `def play_frog_game():`: Определяет функцию, которая содержит логику игры.
   - `playerPosition = 1`: Инициализирует начальную позицию лягушки на 1.

3. **Вывод инструкций к игре**:
   - Выводит на экран объяснение правил игры, включая цель (достичь камня 12) и последствия неправильных прыжков.

4. **Игровой цикл `while True:`**:
   - Бесконечный цикл, который позволяет игре продолжаться, пока игрок не выиграет или не проиграет.

5. **Запрос ввода пользователя и обработка ошибок**:
   - Выводит текущую позицию лягушки.
   - Использует блок `try...except ValueError` для обработки некорректного ввода. Если пользователь вводит что-то, что не является целым числом, выводится сообщение об ошибке.
   - `jumpTo = int(input("לאיזו אבן אתה רוצה לקפוץ? "))`: Запрашивает у пользователя ввод номера камня и преобразует его в целое число.

6. **Проверка условия победы**:
   - `if jumpTo > 11:`: Проверяет, прыгнул ли игрок на камень 12 или дальше.
   - `print("הגעת לצד השני של הנהר! ניצחת!")`: Выводит сообщение о победе.
   - `break`: Завершает цикл, если игрок выиграл.

7. **Проверка условия проигрыша**:
   - `if jumpTo < 2:`: Проверяет, прыгнул ли игрок в воду.
   - `print("טבעת במים! הפסדת!")`: Выводит сообщение о проигрыше.
   - `break`: Завершает цикл, если игрок проиграл.

8. **Обновление позиции игрока**:
    -`playerPosition = jumpTo`: Обновляет позицию лягушки на номер камня, на который она прыгнула.

9.  **Проверка достижения конца реки**:
   -  `if playerPosition == 12:`: Проверяет, достигла ли лягушка камня 12 (победа).
   -  `print("הגעת לצד השני של הנהר! ניצחת!")`: Выводит сообщение о победе.
   -  `break`: Завершает цикл (и игру), если игрок выиграл.

10. **Запуск игры**:
    - `if __name__ == "__main__":`: Гарантирует, что код будет запущен только тогда, когда скрипт выполняется напрямую, а не когда он импортируется как модуль.
    - `play_frog_game()`: Вызывает функцию для запуска игры.
