# Модуль `hypotez/src/main.py`

## Обзор

Данный модуль предоставляет интерактивное меню для запуска предопределенных скриптов. Модуль принимает пользовательский ввод для выбора и выполнения скриптов 1 или 2.

## Функции

### `script1()`

**Описание**: Выполняет скрипт 1.

**Возвращает**:  Не имеет возвращаемого значения.


### `script2()`

**Описание**: Выполняет скрипт 2.

**Возвращает**:  Не имеет возвращаемого значения.


### `show_help()`

**Описание**: Выводит справку по доступным командам.

**Возвращает**:  Не имеет возвращаемого значения.


### `interactive_menu()`

**Описание**: Интерактивное меню для выбора и запуска скриптов.

**Возвращает**:  Не имеет возвращаемого значения.


### `main()`

**Описание**: Основная функция для обработки аргументов командной строки и запуска меню.

**Аргументы**:
- `--help` (булево):  Показывает доступные опции и справку.

**Возвращает**:  Не имеет возвращаемого значения.


## Использование

Пример использования модуля:

```bash
python main.py  # Запускает интерактивное меню
python main.py --help # Показ справки
```
