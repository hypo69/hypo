# Модуль `hypotez/src/logger/header.py`

## Обзор

Этот модуль отвечает за определение корневого пути к проекту `hypotez`. Он использует список маркеров файлов (например, `pyproject.toml`, `requirements.txt`, `.git`), чтобы найти корневую директорию проекта, начиная с текущего файла и перемещаясь вверх по директориям.  Найденный корневой путь добавляется в `sys.path`, что позволяет корректно импортировать модули из других частей проекта. Модуль также загружает настройки из файла `settings.json` и, если возможно, документацию из `README.MD`, сохраняя их в глобальные переменные для использования в других частях приложения.

## Функции

### `set_project_root`

**Описание**: Функция находит корневую директорию проекта, начиная с текущего файла.

**Параметры**:
- `marker_files` (tuple): Кортеж файлов или директорий, используемых в качестве маркеров для определения корневой директории проекта. По умолчанию содержит `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Объект `Path` к корневой директории проекта, или к текущей директории, если корневая директория не найдена.

**Вызывает исключения**:
- Нет


## Глобальные переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root`.

**Тип**: `Path`


### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `src/settings.json`.

**Тип**: `dict` или `None`


### `doc_str`

**Описание**: Строка с текстом документации, загруженная из файла `README.MD`.

**Тип**: `str` или `None`


### `__project_name__`

**Описание**: Название проекта, полученное из настроек (`settings`).  По умолчанию `hypotez`.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта, полученное из настроек (`settings`). По умолчанию пустая строка.

**Тип**: `str`


### `__doc__`

**Описание**: Текст документации проекта.

**Тип**: `str`


### `__details__`

**Описание**: Дополнительные детали о проекте (по умолчанию пустая строка).

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта, полученное из настроек (`settings`). По умолчанию пустая строка.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права на проект, полученное из настроек (`settings`). По умолчанию пустая строка.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка для поддержки разработчика.

**Тип**: `str`
