# Задача No 834
## Совершенные числа и последовательность Коллатца

## Условия

Совершенное число — это положительное целое число, которое равно сумме своих собственных делителей (т.е. делителей, исключая само число). Например, 28 — совершенное число, так как 1 + 2 + 4 + 7 + 14 = 28.

Последовательность Коллатца формируется следующим образом:
  - для четного `n`: `n → n/2`
  - для нечетного `n`: `n → 3n + 1`
Последовательность останавливается, когда `n = 1`.

Например, последовательность Коллатца, начинающаяся с 13:
13 → 40 → 20 → 10 → 5 → 16 → 8 → 4 → 2 → 1

Пусть `f(n)` будет количеством шагов, необходимых для того, чтобы последовательность Коллатца, начинающаяся с `n`, достигла 1.
Например, `f(13) = 9`.

Известно, что первые четыре совершенных числа: 6, 28, 496 и 8128.

Пусть `S(n)` будет суммой `f(p)` для всех совершенных чисел `p <= n`.
Например, `S(10) = f(6) = 8` и `S(10000) = f(6) + f(28) + f(496) = 8 + 18 + 263 = 289`.

Найдите `S(10^18)`.
