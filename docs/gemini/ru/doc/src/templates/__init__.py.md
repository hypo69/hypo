# Модуль `src.templates`

## Обзор

Модуль `src.templates` предназначен для управления шаблонами и версиями в рамках проекта. Он содержит информацию о версии проекта, а также включает другие необходимые компоненты.

## Переменные

### `MODE`

**Описание**:
Глобальная переменная, определяющая режим работы приложения. Может принимать значения `'dev'` (режим разработки) или `'prod'` (режим продакшена).

**Тип**:
`str`

**Значение по умолчанию**:
`'dev'`
