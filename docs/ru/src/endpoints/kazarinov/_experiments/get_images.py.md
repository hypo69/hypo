# Модуль для получения списка изображений, сгенерированных ИИ
## Обзор

Модуль предоставляет функциональность для получения списка изображений, сгенерированных искусственным интеллектом. Он сканирует указанный каталог, находит файлы изображений с определенными расширениями и возвращает их список.

## Подробнее

Этот модуль предназначен для использования в проекте `hypotez` для получения списка изображений, созданных ИИ, которые затем могут быть использованы для дальнейшей обработки или отображения. Он использует функции из других модулей проекта, таких как `recursively_get_filepath` для поиска файлов и `pprint` для вывода информации.

## Функции

### `recursively_get_filepath`

**Назначение**: Рекурсивно получает список путей к файлам, соответствующим заданным расширениям, в указанном каталоге.

**Параметры**:

-   `gs.path.external_data / 'kazarinov' / 'converted_images' / 'pastel'` (Path): Путь к каталогу, в котором нужно искать изображения.
-   `['*.jpeg','*.jpg','*.png']` (List[str]): Список расширений файлов для поиска.

**Возвращает**:

-   `List[Path]`: Список путей к найденным файлам изображений.

**Вызывает исключения**:

-   `FileNotFoundError`: Если указанный путь не существует.
-   `OSError`: Если возникает ошибка при доступе к файловой системе.

**Как работает функция**:

1.  Функция `recursively_get_filepath` принимает путь к директории и список расширений файлов в качестве аргументов.
2.  Она рекурсивно обходит указанную директорию и все ее поддиректории.
3.  Для каждой найденной директории она проверяет файлы на соответствие указанным расширениям.
4.  Все найденные пути к файлам, соответствующие заданным расширениям, добавляются в результирующий список.
5.  В конце функция возвращает список всех найденных путей к файлам.

**ASCII flowchart**:

```
Начало
  │
  │ Получение пути к директории и списка расширений
  │
  │ Рекурсивный обход директории и поддиректорий
  │
  │ Для каждой директории:
  │   │ Проверка файлов на соответствие расширениям
  │   │ Если файл соответствует:
  │   │   │ Добавление пути к файлу в результирующий список
  │
  │ Возврат списка путей к файлам
  │
Конец
```

**Примеры**:

```python
from pathlib import Path
from src.utils.file import recursively_get_filepath

# Пример использования функции
image_dir = Path('./images')  # Укажите путь к директории с изображениями
extensions = ['*.jpeg', '*.jpg', '*.png']
image_paths = recursively_get_filepath(image_dir, extensions)
print(image_paths)  # Вывод списка путей к найденным файлам
```

### `pprint`

**Назначение**: Выводит данные в удобочитаемом формате.

**Параметры**:

-   `images_path` (List[Path]): Список путей к изображениям для вывода.

**Возвращает**:

-   `None`: Функция ничего не возвращает.

**Как работает функция**:

1.  Функция `pprint` принимает список путей к изображениям.
2.  Она использует модуль `pprint` для форматированного вывода данных в консоль.
3.  В результате в консоли отображается удобочитаемый список путей к изображениям.

**ASCII flowchart**:

```
Начало
  │
  │ Получение списка путей к изображениям
  │
  │ Форматированный вывод списка в консоль
  │
Конец
```

**Примеры**:

```python
from src.utils.printer import pprint
from pathlib import Path

# Пример использования функции
image_paths = [Path('./images/image1.jpg'), Path('./images/image2.png')]  # Список путей к изображениям
pprint(image_paths)  # Вывод списка в удобочитаемом формате
