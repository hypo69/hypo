# Модуль hypotez/src/utils/convertors/ns.py

## Обзор

Модуль `ns.py` предоставляет функции для преобразования объектов `SimpleNamespace` в различные форматы: словари, JSON, CSV, XML и XLS.  Он предназначен для удобного преобразования данных, представленных в формате `SimpleNamespace`, в более подходящие для обработки структуры.


## Функции

### `ns2dict`

**Описание**: Преобразует объект `SimpleNamespace` в словарь Python.

**Параметры**:

- `ns_obj` (SimpleNamespace): Объект `SimpleNamespace`, который необходимо преобразовать.

**Возвращает**:

- `dict`: Словарь, представляющий данные из объекта `SimpleNamespace`.

### `ns2json`

**Описание**: Преобразует объект `SimpleNamespace` в формат JSON.

**Параметры**:

- `ns_obj` (SimpleNamespace): Объект `SimpleNamespace`, который необходимо преобразовать.
- `json_file_path` (str | Path, optional): Путь к файлу, в который необходимо сохранить JSON-данные. Если не указан, возвращает строку JSON.

**Возвращает**:

- `str | bool`: Если путь к файлу не указан, возвращает строку JSON. В противном случае, возвращает `True`, если файл успешно сохранен, или `False` при ошибке.

**Обрабатываемые исключения**:

- `Exception`: Общее исключение при выполнении операций с JSON.


### `ns2csv`

**Описание**: Преобразует объект `SimpleNamespace` в формат CSV.

**Параметры**:

- `ns_obj` (SimpleNamespace): Объект `SimpleNamespace`, который необходимо преобразовать.
- `csv_file_path` (str | Path): Путь к файлу, в который необходимо сохранить данные в формате CSV.

**Возвращает**:

- `bool`: `True`, если преобразование и сохранение в CSV успешно, `False` в противном случае.

**Обрабатываемые исключения**:

- `Exception`: Общее исключение при выполнении операций с CSV.


### `ns2xml`

**Описание**: Преобразует объект `SimpleNamespace` в формат XML.

**Параметры**:

- `ns_obj` (SimpleNamespace): Объект `SimpleNamespace`, который необходимо преобразовать.
- `root_tag` (str, optional): Имя тега корневого элемента XML (по умолчанию "root").

**Возвращает**:

- `str`: Строка XML, представляющая данные из объекта `SimpleNamespace`.

**Обрабатываемые исключения**:

- `Exception`: Общее исключение при выполнении операций с XML.


### `ns2xls`

**Описание**: Преобразует объект `SimpleNamespace` в формат XLS.

**Параметры**:

- `ns_obj` (SimpleNamespace): Объект `SimpleNamespace` для преобразования.
- `xls_file_path` (str | Path): Путь к файлу, в который нужно сохранить данные в формате XLS.

**Возвращает**:

- `bool`: `True`, если преобразование и сохранение в XLS успешно, `False` в противном случае.

**Обрабатываемые исключения**:

- `Exception`: Общее исключение при выполнении операций с XLS.
