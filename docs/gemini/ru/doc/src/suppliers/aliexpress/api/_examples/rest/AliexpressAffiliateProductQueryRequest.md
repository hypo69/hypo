# Модуль `hypotez/src/suppliers/aliexpress/api/_examples/rest/AliexpressAffiliateProductQueryRequest.py`

## Обзор

Модуль содержит класс `AliexpressAffiliateProductQueryRequest`, представляющий запрос к API AliExpress для получения информации о продуктах, связанных с партнерской программой. Класс наследуется от базового класса `RestApi` и предоставляет методы для настройки параметров запроса и получения имени API.

## Классы

### `AliexpressAffiliateProductQueryRequest`

**Описание**: Класс `AliexpressAffiliateProductQueryRequest` предназначен для формирования запросов к API AliExpress, чтобы получить данные о продуктах, участвующих в партнерской программе.  Он предоставляет набор свойств (аттрибутов) для настройки параметров запроса.

**Атрибуты**:

- `app_signature`:  Значение для подписи приложения.
- `category_ids`: Список идентификаторов категорий продуктов.
- `delivery_days`: Максимальное количество дней доставки.
- `fields`: Список полей, которые нужно вернуть в ответе.
- `keywords`: Список ключевых слов для поиска.
- `max_sale_price`: Максимальная цена продажи.
- `min_sale_price`: Минимальная цена продажи.
- `page_no`: Номер страницы результатов поиска.
- `page_size`: Размер страницы результатов поиска.
- `platform_product_type`: Тип продукта на платформе.
- `ship_to_country`: Страна доставки.
- `sort`: Сортировка результатов.
- `target_currency`: Целевая валюта.
- `target_language`: Целевой язык.
- `tracking_id`: Идентификатор отслеживания.

**Методы**:

#### `getapiname()`

**Описание**: Возвращает имя API для запроса.

**Возвращает**:
- `str`: Имя API запроса (в данном случае `aliexpress.affiliate.product.query`).


**Примечание**: Класс `RestApi` не описан в данном файле, поэтому его детали в документации не отображаются.  Предполагается, что он отвечает за базовые функции работы с API.
