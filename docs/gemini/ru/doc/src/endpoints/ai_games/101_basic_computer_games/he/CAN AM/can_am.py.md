# CAN AM

## Обзор

Данный модуль реализует игру CAN AM, в которой пользователь соревнуется с компьютером в гонке на машинах. Пользователь управляет скоростью своей машины, пытаясь достичь финиша первым.

## Функции

### `main_loop`

**Описание**: Основная логика игры CAN AM.

**Параметры**:
 - Нет параметров

**Возвращает**:
 - None: Функция ничего не возвращает явно.

**Вызывает исключения**:
 - ValueError: Возникает, если пользователь вводит некорректное значение скорости.

## Переменные

### `userDistance`

**Описание**: Расстояние, пройденное машиной пользователя.

### `userSpeed`

**Описание**: Скорость машины пользователя.

### `computerDistance`

**Описание**: Расстояние, пройденное машиной компьютера.

### `computerSpeed`

**Описание**: Скорость машины компьютера (постоянная).

### `speedChoice`
**Описание**: Выбор пользователя для изменения скорости.

## Описание кода

1. **Импорт модуля `time`**:
   - `import time`: Импортирует модуль `time`, используемый для добавления небольшой задержки между обновлениями экрана.

2.  **Инициализация переменных**:
    - `userDistance = 0`: Инициализирует расстояние, пройденное машиной пользователя, нулём.
    - `userSpeed = 0`: Инициализирует скорость машины пользователя нулём.
    - `computerDistance = 0`: Инициализирует расстояние, пройденное машиной компьютера, нулём.
    - `computerSpeed = 2`: Инициализирует скорость машины компьютера значением 2 (постоянная скорость).

3. **Главный игровой цикл `while userDistance < 500 and computerDistance < 500:`**:
   - Этот цикл выполняется, пока ни одна из машин не пересечёт финишную линию (500).
   - `print(f"מיקום שלך: {userDistance}, מיקום מחשב: {computerDistance}")`: Выводит текущее положение обеих машин.
   - **Получение ввода от пользователя**:
     - `while True:`: Бесконечный цикл, который продолжается, пока пользователь не введёт корректный ввод.
     - `try...except ValueError`: Обрабатывает возможные ошибки ввода.
     - `speedChoice = int(input("בחר מהירות (0=האט, 1=שמור, 2=האץ): "))`: Запрашивает ввод пользователя для выбора скорости (0, 1 или 2).
     - `if speedChoice in [0, 1, 2]:`: Проверяет, является ли ввод корректным (0, 1 или 2).
     - `break`: Выходит из внутреннего цикла, если ввод корректен.
     - `else`: Если ввод некорректен, выводит сообщение об ошибке пользователю.
   - `userSpeed += (speedChoice - 1)`: Обновляет скорость машины пользователя.
   - `if userSpeed < 0:`: Ограничивает скорость, чтобы она не стала отрицательной.
   - `userDistance += userSpeed`: Обновляет расстояние, пройденное машиной пользователя, на основе её скорости.
   - `computerDistance += computerSpeed`: Обновляет расстояние, пройденное машиной компьютера.
   - `time.sleep(0.5)`: Пауза 0.5 секунды между обновлениями экрана.

4. **Проверка победителя**:
   - `if userDistance >= 500:`: Проверяет, достигла ли машина пользователя финишной линии.
   - `print("ניצחת!")`: Выводит сообщение о победе пользователя.
   - `else`: Если машина пользователя не достигла финишной линии, то побеждает компьютер.
   - `print("המחשב ניצח!")`: Выводит сообщение о победе компьютера.
