# Модуль `login.py`

## Обзор

Модуль `login.py` предназначен для реализации процесса авторизации в системе C-Data с использованием веб-драйвера. Он содержит функцию `login`, которая выполняет вход в систему, используя предоставленные учетные данные.

## Подробней

Этот модуль является частью подсистемы `src.suppliers.cdata` проекта `hypotez` и отвечает за автоматизацию процесса входа в систему C-Data через веб-интерфейс. Он использует локаторы элементов веб-страницы для заполнения полей логина и пароля, а также для нажатия кнопки входа.
