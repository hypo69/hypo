# Модуль `src.suppliers.etzmaleh.graber`

## Обзор

Модуль `src.suppliers.etzmaleh.graber` представляет класс `Graber`, предназначенный для сбора данных о товарах с сайта `etzmaleh.co.il`. Класс наследует функциональность от родительского класса `Graber` и предоставляет возможность переопределять методы для специфической обработки данных, если это необходимо. Также модуль включает механизмы для выполнения предварительных действий перед запросом к веб-драйверу через декораторы.

## Классы

### `Graber`

**Описание**:
Класс `Graber` предназначен для операций сбора данных о товарах с сайта `etzmaleh.co.il`.

**Методы**:

- `__init__`: Инициализирует класс, устанавливает префикс поставщика и вызывает конструктор родительского класса.
    
    **Параметры**:
    - `driver` (`Driver`): Экземпляр веб-драйвера.

## Функции

### `close_pop_up`
   
   **Описание**:
    Создает декоратор для закрытия всплывающих окон перед выполнением основной логики функции.

    **Параметры**:
    - `value` (Any, optional): Дополнительное значение для декоратора. По умолчанию `None`.

    **Возвращает**:
    - `Callable`: Декоратор, оборачивающий функцию.
