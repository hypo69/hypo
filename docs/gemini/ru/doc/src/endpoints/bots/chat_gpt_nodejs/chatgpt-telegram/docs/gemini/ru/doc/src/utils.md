# Модуль utils

## Обзор

Данный модуль предоставляет функцию для удаления файла.

## Функции

### `removeFile`

**Описание**: Асинхронная функция для удаления файла по заданному пути.

**Параметры**:

- `path` (string): Путь к файлу, который необходимо удалить.

**Возвращает**:

- `void`: Функция не возвращает значения.

**Обрабатываемые исключения**:

- `e`: Если возникла ошибка при удалении файла, функция выводит сообщение об ошибке в консоль.
