# Документация для `cdata_categories_laptops_hp.json`

## Обзор

Файл `cdata_categories_laptops_hp.json` содержит JSON-структуру, определяющую сценарии для категорий ноутбуков HP. Каждый сценарий включает информацию о бренде, URL, состоянии, активности и связанных категориях PrestaShop.

## Структура JSON

### `scenarios`

Объект `scenarios` содержит набор сценариев, каждый из которых представляет собой категорию ноутбуков HP. Ключом является название сценария, а значением - объект с данными сценария.

#### HP 11.6 I3
   **Описание**: Сценарий для ноутбука HP с диагональю 11.6 дюймов и процессором Intel Core i3.
  
   **Параметры**:
     - `brand` (str): Бренд ноутбука, всегда "HP".
     - `url` (str): URL, связанный с этим сценарием.
     - `checkbox` (bool):  Флаг, указывающий, выбран ли сценарий.
     - `active` (bool): Флаг, указывающий, активен ли сценарий.
     - `condition` (str): Состояние продукта, всегда "new".
     - `presta_categories` (str): Строка, содержащая идентификаторы категорий PrestaShop, разделенные запятыми.
