# Модуль `story.py`

## Обзор

Этот модуль предоставляет механизмы для создания историй в TinyTroupe. Он позволяет описывать симуляцию, включая агентов и окружение, и генерировать истории на основе этой информации.

## Классы

### `TinyStory`

**Описание**: Класс `TinyStory` предназначен для создания и управления историями, связанными с симуляцией. История может быть сфокусирована на агенте или окружении.

**Методы**:

- `__init__`: Инициализирует объект `TinyStory`.
    - **Описание**: Инициализирует историю, принимая на вход окружение, агента, цель, контекст и другие параметры для настройки генерации истории.
    - **Параметры**:
        - `environment` (TinyWorld, необязательно): Окружение симуляции.
        - `agent` (TinyPerson, необязательно): Агент симуляции.
        - `purpose` (str, необязательно): Цель истории. По умолчанию "Be a realistic simulation.".
        - `context` (str, необязательно): Текущий контекст истории. По умолчанию "".
        - `first_n` (int, необязательно): Количество первых взаимодействий для включения в историю. По умолчанию 10.
        - `last_n` (int, необязательно): Количество последних взаимодействий для включения в историю. По умолчанию 20.
        - `include_omission_info` (bool, необязательно): Включать ли информацию о пропущенных взаимодействиях. По умолчанию True.
    - **Возвращает**:
        - `None`
    - **Исключения**:
        - `Exception`: Если не предоставлено ни окружение, ни агент.
        - `Exception`: Если предоставлены и окружение, и агент.


- `start_story`: Запускает создание истории.
    - **Описание**: Начинает создание новой истории с использованием предопределенных шаблонов и требований.
    - **Параметры**:
        - `requirements` (str, необязательно): Требования к истории. По умолчанию "Start some interesting story about the agents.".
        - `number_of_words` (int, необязательно): Желаемое количество слов в истории. По умолчанию 100.
        - `include_plot_twist` (bool, необязательно): Включать ли поворот сюжета. По умолчанию False.
    - **Возвращает**:
        - `str`: Созданный начальный фрагмент истории.
    - **Исключений**: Нет.

- `continue_story`: Продолжает историю.
    - **Описание**: Продолжает текущую историю с новыми требованиями.
    - **Параметры**:
        - `requirements` (str, необязательно): Требования к продолжению истории. По умолчанию "Continue the story in an interesting way.".
        - `number_of_words` (int, необязательно): Желаемое количество слов в продолжении истории. По умолчанию 100.
        - `include_plot_twist` (bool, необязательно): Включать ли поворот сюжета. По умолчанию False.
    - **Возвращает**:
        - `str`: Созданный фрагмент продолжения истории.
    - **Исключений**: Нет.

- `_current_story`: Возвращает текущую историю.
    - **Описание**: Возвращает текущий текст истории, включая информацию о взаимодействиях агентов или окружения.
    - **Параметры**: Нет
    - **Возвращает**:
        - `str`: Текущая история.
    - **Исключений**: Нет.
