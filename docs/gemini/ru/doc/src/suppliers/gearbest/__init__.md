# Модуль hypotez/src/suppliers/gearbest/__init__.py

## Обзор

Данный модуль является инициализатором для модуля `gearbest`, предоставляющего функции для работы с поставщиком Gearbest. Он определяет константу `MODE` и импортирует класс `Graber` из модуля `graber`.

## Константы

### `MODE`

**Описание**:  Константа, хранящая режим работы модуля (например, 'dev', 'prod'). Текущее значение: 'dev'.

## Импорты

### `from .graber import Graber`

**Описание**: Импортирует класс `Graber` из модуля `graber`.  Этот класс, вероятно, содержит методы для сбора данных с сайта Gearbest.
