# Модуль тестирования извлечения `test_extraction.py`

Этот модуль содержит модульные тесты для проверки функциональности классов `ArtifactExporter` и `Normalizer`, используемых для экспорта артефактов в различных форматах и нормализации текстовых данных.

## Фикстуры

### `exporter`

**Описание**:
Фикстура `exporter` создает экземпляр класса `ArtifactExporter` с базовой папкой вывода "./test_exports".

## Тесты экспорта

### `test_export_json`

**Описание**:
Тест проверяет корректность экспорта данных в формате JSON.

**Параметры**:
- `exporter`: Экземпляр `ArtifactExporter`, предоставляемый фикстурой.

**Проверяет**:
-   Создание файла JSON в указанной директории.
-   Соответствие экспортированных данных оригинальным.

### `test_export_text`

**Описание**:
Тест проверяет корректность экспорта текстовых данных в файл TXT.

**Параметры**:
- `exporter`: Экземпляр `ArtifactExporter`, предоставляемый фикстурой.

**Проверяет**:
-   Создание файла TXT в указанной директории.
-   Соответствие экспортированных данных оригинальным.

### `test_export_docx`

**Описание**:
Тест проверяет корректность экспорта данных с Markdown-форматированием в файл DOCX.

**Параметры**:
- `exporter`: Экземпляр `ArtifactExporter`, предоставляемый фикстурой.

**Проверяет**:
-   Создание файла DOCX в указанной директории.
-   Сохранение части оригинального контента в экспортированном файле.
-   Отсутствие Markdown-форматирования в экспортированном файле.
