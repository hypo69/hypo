# Модуль hypotez/src/suppliers/ebay/header.py

## Обзор

Этот модуль содержит функции для работы с настройками проекта, включая поиск корневой директории проекта и загрузку настроек из файла `settings.json`.  Он также предоставляет доступ к информации о проекте, такой как имя, версия, описание и автор.


## Функции

### `set_project_root`

**Описание**:  Находит корневую директорию проекта, начиная от текущей директории файла. Поиск происходит вверх по дереву директорий, останавливаясь на первой директории, содержащей указанные файлы-маркеры (например, `pyproject.toml`, `requirements.txt`, `.git`). Если корневая директория не найдена, возвращает директорию, где находится скрипт.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для определения корневой директории проекта.


**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена, иначе - путь к директории текущего файла.


**Вызывает исключения**:

- Нет.


### `settings`

**Описание**:  Переменная, содержащая настройки проекта, загруженные из файла `settings.json`.

**Параметры**:

- Нет


**Возвращает**:

- `dict`: Словарь с настройками проекта, если файл `settings.json` успешно загружен, и `None`, если файл не найден или содержит невалидные данные.


**Вызывает исключения**:

- `FileNotFoundError`: Если файл `settings.json` не найден.
- `json.JSONDecodeError`: Если файл `settings.json` содержит некорректный JSON.


### `doc_str`

**Описание**: Переменная, содержащая описание проекта (документация), загруженную из файла `README.MD`.

**Параметры**:

- Нет


**Возвращает**:

- `str`: Строка с описанием проекта, если файл `README.MD` успешно загружен, и `None`, если файл не найден или содержит невалидные данные.


**Вызывает исключения**:

- `FileNotFoundError`: Если файл `README.MD` не найден.
- `json.JSONDecodeError`: Если файл `README.MD` содержит некорректные данные.


## Постоянные переменные

### `__project_name__`

**Описание**: Имя проекта, полученное из настроек (`settings.json`). По умолчанию используется значение 'hypotez'.

**Параметры**:

- Нет


**Возвращает**:

- `str`: Имя проекта.

### `__version__`

**Описание**: Версия проекта, полученная из настроек (`settings.json`). По умолчанию пустая строка.

**Параметры**:

- Нет


**Возвращает**:

- `str`: Версия проекта.


### `__doc__`

**Описание**: Описание проекта, полученное из файла `README.MD`. По умолчанию пустая строка.

**Параметры**:

- Нет


**Возвращает**:

- `str`: Описание проекта.


### `__details__`

**Описание**: Дополнительные детали о проекте. По умолчанию пустая строка.

**Параметры**:

- Нет


**Возвращает**:

- `str`: Дополнительные детали.


### `__author__`

**Описание**: Автор проекта, полученный из настроек (`settings.json`). По умолчанию пустая строка.

**Параметры**:

- Нет


**Возвращает**:

- `str`: Автор проекта.


### `__copyright__`

**Описание**: Авторские права проекта, полученные из настроек (`settings.json`). По умолчанию пустая строка.

**Параметры**:

- Нет


**Возвращает**:

- `str`: Авторские права.

**Описание**: Ссылка на возможность поддержать разработчика чашкой кофе. По умолчанию ссылка указана.

**Параметры**:

- Нет


**Возвращает**:

- `str`: Ссылка на платёжную систему.
