# Инструкция по обработке кода

## Обзор

Данный документ содержит инструкции для обработки входного Python-кода. Модель должна сгенерировать документацию в формате Markdown, соответствующую заданным требованиям.

## Формат документации

Каждый файл должен содержать:

- Заголовок и краткое описание содержимого файла.
- Раздел с оглавлением (TOC), содержащий ссылки на все основные разделы документации.
- Документацию для классов, функций и методов в соответствии со следующим форматом:
```python
def функция(параметр: str, параметр1: Optional[str | dict | str] = None) -> dict | None:
    """
    Args:

        dict | None: Описание возвращаемого значения. Возвращает словарь или `None`.

    """
    # Тело функции
```

```python
try:

## Структура документации

Документация должна содержать следующие разделы:

- **Классы:**
    - Список всех классов в модуле.
    - Для каждого класса: описание, методы.
- **Функции:**
    - Список всех функций в модуле.
    - Для каждой функции: описание, параметры, возвращаемое значение, вызываемые исключения.
- **Переменные:**
    - Описание всех глобальных переменных, если таковые имеются.

## Пример документации для функции

### Функция `calculate_average`

**Описание**: Вычисляет среднее арифметическое списка чисел.

**Параметры**:
- `numbers` (list[int]): Список чисел.

**Возвращает**:
- `float`: Среднее арифметическое чисел в списке. Возвращает `None`, если список пуст.

**Вызывает исключения**:
- `TypeError`: Если входной параметр не является списком целых или дробных чисел.

```python
def calculate_average(numbers: list[Union[int, float]]) -> Union[float, None]:
    """
    Args:
        numbers (list[Union[int, float]]): Список чисел.

    Returns:
        Union[float, None]: Среднее арифметическое чисел в списке. Возвращает None, если список пуст.

    Raises:
        TypeError: Если входной параметр не является списком целых или дробных чисел.
    """
    if not isinstance(numbers, list):
        raise TypeError("Ввод должен быть списком.")
    if not all(isinstance(num, (int, float)) for num in numbers):
        raise TypeError("Все элементы списка должны быть числами.")

    if not numbers:
        return None

    return sum(numbers) / len(numbers)
```
