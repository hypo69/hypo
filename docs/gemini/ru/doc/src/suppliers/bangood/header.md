# Модуль hypotez/src/suppliers/bangood/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`. Он также загружает содержимое файла `README.MD`.  Модуль инициализирует переменные, содержащие информацию о проекте, такую как имя проекта, версия, документация, автор и прочие данные.

## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная с текущего каталога и продвигаясь вверх по дереву директорий. Останавливается на первой директории, содержащей один из указанных файлов или директорий.

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, используемых для определения корневого каталога проекта.  По умолчанию использует `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневому каталогу проекта, если найден. В противном случае возвращает путь к директории, в которой находится данный скрипт.

**Вызывает исключения**:

- Нет.


## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта.  Инициализируется вызовом функции `set_project_root()`.

**Тип**: `Path`


### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json`.

**Тип**: `dict`


### `doc_str`

**Описание**: Строка с содержимым файла `README.MD`.

**Тип**: `str`


### `__project_name__`

**Описание**: Имя проекта. Возвращает значение из настроек `settings`, если оно присутствует, иначе использует значение по умолчанию `'hypotez'`.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта. Возвращает значение из настроек `settings`, если оно присутствует, иначе использует значение по умолчанию `''`.

**Тип**: `str`


### `__doc__`

**Описание**: Документация проекта. Возвращает значение `doc_str`, если оно не пусто, иначе использует значение по умолчанию `''`.

**Тип**: `str`


### `__details__`

**Описание**: Дополнительные детали проекта.  Используется значение по умолчанию `''`.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта. Возвращает значение из настроек `settings`, если оно присутствует, иначе использует значение по умолчанию `''`.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права проекта. Возвращает значение из настроек `settings`, если оно присутствует, иначе использует значение по умолчанию `''`.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка на способ поддержания разработчика (например, оплатить кофе). Возвращает значение из настроек `settings`, если оно присутствует, иначе использует значение по умолчанию.

**Тип**: `str`
