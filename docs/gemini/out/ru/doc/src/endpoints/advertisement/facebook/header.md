# Модуль hypotez/src/endpoints/advertisement/facebook/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`.  Также он загружает описание проекта из `README.MD` для использования в других частях приложения.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная от текущей директории файла и идя вверх по дереву директорий.  Останавливается на первой директории, содержащей указанные маркерные файлы.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для определения корневой директории проекта. По умолчанию: (`'pyproject.toml'`, `'requirements.txt'`, `.git`).

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена. В противном случае возвращает директорию, где расположен текущий файл.

**Вызывает исключения**:

- Нет


### `__init__`


**Описание**:  Инициализирует переменные, необходимые для работы программы.  Получает корневую директорию проекта с помощью `set_project_root` и загружает настройки из `settings.json`, а также описание из `README.MD`


**Параметры**:

- Нет

**Возвращает**:

- Нет

**Вызывает исключения**:

- `FileNotFoundError`:  Если файл `settings.json` или `README.MD` не найдены.
- `json.JSONDecodeError`: Если содержимое `settings.json` некорректное JSON.


## Переменные


### `__root__`

**Описание**: Путь к корневой директории проекта. Инициализируется вызовом `set_project_root()`.

**Тип**: `Path`


### `settings`

**Описание**: Словарь с настройками проекта, загруженный из `settings.json`.


**Тип**: `dict` | `None`

### `doc_str`

**Описание**: Строка с описанием проекта, загруженная из `README.MD`.


**Тип**: `str` | `None`


### `__project_name__`

**Описание**: Название проекта, полученное из настроек (`settings`). По умолчанию `'hypotez'`.


**Тип**: `str`

### `__version__`

**Описание**: Версия проекта, полученная из настроек (`settings`). По умолчанию пустая строка.


**Тип**: `str`

### `__doc__`

**Описание**: Документация проекта, загруженная из `README.MD`.


**Тип**: `str`

### `__details__`


**Описание**: Подробная информация о проекте. По умолчанию пустая строка.


**Тип**: `str`

### `__author__`

**Описание**: Автор проекта, полученный из настроек (`settings`). По умолчанию пустая строка.


**Тип**: `str`

### `__copyright__`


**Описание**: Авторские права на проект, полученные из настроек (`settings`). По умолчанию пустая строка.


**Тип**: `str`

### `__cofee__`

**Описание**: Ссылка для поддержки разработчика. По умолчанию: "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69".

**Тип**: `str`
