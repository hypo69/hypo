# Модуль hypotez/src/suppliers/ivory/__init__.py

## Обзор

Данный модуль предоставляет начальную точку входа для работы с поставщиком данных Ivory. Он содержит конфигурационную переменную `MODE` и импортирует класс `Graber` из модуля `graber`.

## Конфигурация

### `MODE`

**Описание**: Переменная, определяющая режим работы модуля.  Текущее значение: `'dev'`

**Тип**: `str`

## Классы

### `Graber`

**Описание**: Класс для работы с поставщиком данных Ivory. Подробная документация находится в файле `graber.py`.
