# Предотвращение вредного контента в TinyTroupe

## Обзор

Этот модуль содержит функции для предотвращения генерации вредного контента в модели TinyTroupe. Он предоставляет инструменты для фильтрации и проверки текстовых данных на предмет потенциально вредных слов и фраз.

## Функции

### `is_harmful_content`

**Описание**:  Функция проверяет, содержит ли входной текст вредный контент, используя предопределенный список запрещенных слов и фраз.

**Параметры**:

- `text` (str): Текст для проверки.


**Возвращает**:

- `bool`: `True`, если текст содержит вредный контент, иначе `False`.

**Вызывает исключения**:

- `TypeError`: Если входной параметр `text` не является строкой.


```python
def is_harmful_content(text: str) -> bool:
    """
    Args:
        text (str): Текст для проверки.

    Returns:
        bool: True, если текст содержит вредный контент, иначе False.

    Raises:
        TypeError: Если входной параметр text не является строкой.
    """
    if not isinstance(text, str):
        raise TypeError("Входной параметр text должен быть строкой")
    
    #Здесь должен быть код для проверки на вредный контент
    # (например, сравнение с списком запрещенных слов/фраз).
    
    # Пример:
    harmful_keywords = ["убийство", "насилие", "война"]
    for keyword in harmful_keywords:
        if keyword in text:
            return True
    return False
```


### `filter_harmful_content`

**Описание**: Функция фильтрует входной текст, удаляя вредный контент, используя предопределенный список запрещенных слов и фраз.

**Параметры**:

- `text` (str): Текст для фильтрации.


**Возвращает**:

- `str`: Отфильтрованный текст, без вредного контента. Если вредный контент не найден, возвращается исходный текст.

**Вызывает исключения**:

- `TypeError`: Если входной параметр `text` не является строкой.


```python
def filter_harmful_content(text: str) -> str:
    """
    Args:
        text (str): Текст для фильтрации.

    Returns:
        str: Отфильтрованный текст, без вредного контента. Если вредный контент не найден, возвращается исходный текст.

    Raises:
        TypeError: Если входной параметр text не является строкой.
    """
    if not isinstance(text, str):
        raise TypeError("Входной параметр text должен быть строкой")
    
    # Здесь должен быть код для фильтрации вредного контента.
    
    # Пример:
    harmful_keywords = ["убийство", "насилие", "война"]
    for keyword in harmful_keywords:
        text = text.replace(keyword, "")
    return text
```
