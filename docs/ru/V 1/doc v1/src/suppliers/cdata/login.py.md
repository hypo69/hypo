# Модуль `login`

## Обзор

Модуль `login.py` расположен в `src/suppliers/cdata/` и предназначен для реализации интерфейса авторизации в системе C-Data с использованием веб-драйвера. Он содержит функцию `login`, которая выполняет вход пользователя на сайт `reseller.c-data.co.il`.

## Подробней

Этот модуль является частью подсистемы, отвечающей за взаимодействие с поставщиками данных C-Data. Функция `login` использует локаторы элементов веб-страницы для заполнения полей email и пароля, а также для нажатия кнопки входа. После успешного входа в систему, модуль логирует информацию о входе.
