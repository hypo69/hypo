# Модуль hypotez/src/endpoints

## Обзор

Этот модуль содержит определения конечных точек (endpoints) для взаимодействия с различными внешними сервисами. Он предоставляет классы для работы с данными PrestaShop, Kazarinov и другими, определенными константой `MODE`.


## Константные переменные

### `MODE`

**Описание**:  Переменная, определяющая режим работы (например, 'dev', 'prod').

**Значение**:  `'dev'`


## Модули

### `prestashop`

**Описание**: Содержит классы для взаимодействия с API PrestaShop.

### `kazarinov`

**Описание**: Содержит класс для взаимодействия с ботом Kazarinov Telegram.


## Классы

### `PrestaShop`

**Описание**:  Класс для работы с API PrestaShop.

**Подробности**:  Взаимодействие с основными ресурсами PrestaShop.

### `PrestaCategory`

**Описание**:  Класс для работы с категориями PrestaShop.

**Подробности**:  Получение, редактирование и управление категориями.

### `PrestaCustomer`

**Описание**: Класс для работы с клиентами PrestaShop.

**Подробности**:  Управление клиентской базой данных.

### `PrestaLanguage`

**Описание**:  Класс для работы с языками PrestaShop.

**Подробности**:  Управление языками магазина.


### `PrestaProduct`

**Описание**:  Класс для работы с товарами PrestaShop.

**Подробности**:  Получение, редактирование и управление продуктами.


### `PrestaShopShop`

**Описание**:  Класс для работы с магазинами PrestaShop.

**Подробности**:  Управление магазинами PrestaShop.


### `PrestaSupplier`

**Описание**:  Класс для работы с поставщиками PrestaShop.

**Подробности**:  Управление поставщиками.


### `PrestaWarehouse`

**Описание**:  Класс для работы с складами PrestaShop.

**Подробности**:  Управление складами.


### `PriceListRequester`

**Описание**:  Класс для запроса прайс-листов.

**Подробности**:  Выполнение запросов к API прайс-листов.


### `KazarinovTelegramBot`

**Описание**:  Класс для взаимодействия с ботом Kazarinov Telegram.

**Подробности**:  Отправка сообщений, получение ответов и другие взаимодействия с ботом.
