# Модуль example_pprint

## Обзор

Модуль `example_pprint` демонстрирует использование различных функций pretty print для форматирования вывода данных. Включает импорт и использование стандартной библиотеки `pprint`, а также пользовательской функции `pprint` из модуля `src.printer`.

## Подробней

Этот модуль предназначен для показа возможностей форматированного вывода, что может быть полезно для отладки и представления данных в удобном для чтения виде. Он иллюстрирует, как использовать `pprint` как из стандартной библиотеки, так и из пользовательского модуля.

## Функции

### `pprint`

```python
def pprint(object: object, stream: Optional[Any] = None, indent: int = 1, width: int = 80, depth: Optional[int] = None, *, compact: bool = False, sort_dicts: bool = True, underscore_numbers: bool = False) -> None:
    """ Функция выполняет форматированный вывод данных в удобочитаемом виде.
    Args:
        object (object): Объект для печати.
        stream (Optional[Any], optional): Поток вывода. По умолчанию `None` (используется стандартный вывод).
        indent (int, optional): Отступ для форматирования. По умолчанию 1.
        width (int, optional): Максимальная ширина строки. По умолчанию 80.
        depth (Optional[int], optional): Максимальная глубина для печати вложенных структур. По умолчанию `None` (без ограничений).
        compact (bool, optional): Флаг для компактного вывода. По умолчанию `False`.
        sort_dicts (bool, optional): Флаг для сортировки словарей по ключам. По умолчанию `True`.
        underscore_numbers (bool, optional): Флаг для разделения чисел подчеркиваниями. По умолчанию `False`.

    Returns:
        None: Функция ничего не возвращает.

    """
    ...
```

**Параметры**:

- `object` (object): Объект, который нужно вывести.
- `stream` (Optional[Any], optional): Поток, в который будет выводиться отформатированная строка. Если не указан, используется стандартный вывод (`sys.stdout`). По умолчанию `None`.
- `indent` (int, optional): Величина отступа для каждого уровня вложенности. По умолчанию `1`.
- `width` (int, optional): Максимальная ширина строки при форматировании. По умолчанию `80`.
- `depth` (Optional[int], optional): Максимальная глубина рекурсивного вывода. Если `None`, глубина не ограничена. По умолчанию `None`.
- `compact` (bool, optional): Если `True`, пытается уместить больше данных в каждой строке. По умолчанию `False`.
- `sort_dicts` (bool, optional): Если `True`, словари сортируются по ключам перед выводом. По умолчанию `True`.
- `underscore_numbers` (bool, optional): Если `True`, добавляет подчеркивания к большим числам для улучшения читаемости. По умолчанию `False`.

**Возвращает**:

- `None`: Функция ничего не возвращает, она только выводит данные.

**Как работает функция**:

1. Функция `pprint` используется для форматированного вывода объектов.
2. Она принимает различные параметры для настройки форматирования, такие как отступы, ширину, глубину и т.д.
3. Внутри она использует модуль `pprint` из стандартной библиотеки Python, но может быть переопределена для пользовательских нужд.

**Примеры**:

```python
pprint("Hello, world!")
```
