# Модуль PrestaShop

## Обзор

Данный модуль предоставляет инструменты для работы с API платформы PrestaShop. Он включает в себя функции для взаимодействия с различными ресурсами API, а также инструменты для управления данными, такими как категории, клиенты, языки, прайс-листы, продукты, магазины, поставщики и склады.  Модуль также содержит инструменты для работы со схемами API, позволяя проверять и создавать данные в соответствии с определёнными стандартами.  В нём также присутствуют примеры использования и структура для лучшего понимания и эффективной работы.

## Структура модуля

### Основной каталог (`PrestaShop`)

- `__init__.py`: Инициализация модуля.
- `category.py`: Управление категориями.
- `customer.py`: Управление клиентами.
- `language.py`: Управление языками.
- `pricelist.py`: Управление прайс-листами.
- `product.py`: Управление продуктами.
- `shop.py`: Управление магазинами.
- `supplier.py`: Управление поставщиками.
- `version.py`: Управление информацией о версии модуля.
- `warehouse.py`: Управление складами.

### Каталог Примеры (`_examples`)

- `__init__.py`: Инициализация модуля примеров.
- `header.py`: Пример скрипта заголовка.
- `version.py`: Пример скрипта версии.

### Каталог API (`api`)

- `__init__.py`: Инициализация модуля API.
- `_dot`: Каталог с файлами DOT для графических представлений.
- `_examples`: Каталог с примерами скриптов для использования API.
- `api.py`: Основной скрипт для взаимодействия с API PrestaShop.
- `version.py`: Управление информацией о версии модуля API.

### Каталог API Схемы (`api_schemas`)

- `__init__.py`: Инициализация модуля API схем.
- `api_resourses_list.py`: Список доступных ресурсов API.
- `api_schema_category.json`: JSON-схема для категорий.
- `api_schema_language.json`: JSON-схема для языков.
- `api_schema_product.json`: JSON-схема для продуктов.
- `api_schemas_buider.py`: Скрипт для построения схем API.
- `api_suppliers_schema.json`: JSON-схема для поставщиков.
- `csv_product_schema.json`: CSV-схема для продуктов.
- `PrestaShop_product_combinations_fields.json`: JSON-файл для полей комбинаций продуктов.
- `PrestaShop_product_combinations_sysnonyms_he.json`: JSON-файл для синонимов комбинаций продуктов на иврите.


### Каталог Домены (`domains`)

- `__init__.py`: Инициализация модуля доменов.
- `ecat_co_il`: Настройки для `ecat.co.il`.
    - `__init__.py`: Инициализация домена `ecat.co.il`.
    - `settings.json`: JSON-файл с настройками для `ecat.co.il`.
- `emildesign_com`: Настройки для `emildesign.com`.
    - `__init__.py`: Инициализация домена `emildesign.com`.
    - `settings.json`: JSON-файл с настройками для `emildesign.com`.
- `sergey_mymaster_co_il`: Настройки для `sergey.mymaster.co.il`.
    - `__init__.py`: Инициализация домена `sergey.mymaster.co.il`.
    - `settings.json`: JSON-файл с настройками для `sergey.mymaster.co.il`.
