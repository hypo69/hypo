# Задача No 533
## Задачи с числами Ландри
## Условия
Число Ландри — это число вида $a^b + b^a$, где $a$ и $b$ — целые положительные числа.
Когда $a \ge b$, первые несколько чисел Ландри:
$2^1 + 1^2 = 3$
$3^1 + 1^3 = 4$
$3^2 + 2^3 = 17$
$4^1 + 1^4 = 5$
$4^2 + 2^4 = 32$
$4^3 + 3^4 = 145$
$5^1 + 1^5 = 6$
$5^2 + 2^5 = 57$
$5^3 + 3^5 = 268$
$5^4 + 4^5 = 1249$

Пусть $L(n)$ — $n$-е число Ландри в возрастающем порядке, то есть $L(1) = 3, L(2) = 4, L(3) = 5, L(4) = 6, L(5) = 17$ и так далее.
Также, определим $S(n)$ как сумму $L(i)$ для $1 \le i \le n$.
Например, $S(5) = 3 + 4 + 5 + 6 + 17 = 35$.

Найдите $S(1000000)$.
