# Модуль `graber.py`

## Обзор

Модуль `graber.py` является частью проекта `hypotez` и предназначен для сбора данных о товарах с сайта `grandadvanse.co.il`. Он наследует функциональность от родительского класса `Graber` и переопределяет методы обработки полей, если требуется нестандартная обработка.

## Подробнее

Этот модуль предназначен для извлечения информации о товарах с сайта `grandadvanse.co.il`. Он использует веб-драйвер для взаимодействия с сайтом и извлекает данные о товарах, такие как название, описание, цена и другие характеристики. Модуль также предоставляет возможность предварительной обработки данных перед отправкой запроса к веб-драйверу с использованием декораторов.

## Классы

### `Graber`

**Описание**: Класс `Graber` наследуется от класса `Graber` (переименованного как `Grbr`) и предназначен для сбора данных о товарах с сайта `grandadvance.co.il`.

**Как работает класс**:
Класс инициализируется с веб-драйвером и индексом языка. Он загружает конфигурацию и локаторы, специфичные для сайта `grandadvance.co.il`, и устанавливает локатор для декоратора.

**Методы**:

- `__init__(self, driver: Driver, lang_index: int)`: Инициализирует класс `Graber`.

  **Параметры**:
    - `driver` (Driver): Экземпляр веб-драйвера для взаимодействия с сайтом.
    - `lang_index` (int): Индекс языка, используемого на сайте.

  **Как работает метод**:
    1. Загружает конфигурацию из `grandadvance.json` с использованием `j_loads_ns`.
    2. Загружает локаторы элементов продукта из `product.json` с использованием `j_loads_ns`.
    3. Вызывает конструктор родительского класса `Graber` с префиксом поставщика, драйвером и индексом языка.
    4. Устанавливает `Context.locator_for_decorator` для использования декоратора (если необходимо).
