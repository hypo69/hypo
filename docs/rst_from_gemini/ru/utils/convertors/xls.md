# xls.py - Преобразование файлов XLS в словари

**Расположение:** `C:\Users\user\Documents\repos\hypotez\src\utils\convertors\xls.py`

**Роль:** `doc_creator` (создание документации)

**Описание:**

Этот файл содержит функцию `xls2dict`, предназначенную для чтения файла Excel (формат XLS) и преобразования его содержимого в словарь Python.  Функция использует функции `read_xls_as_dict` и `save_xls_file` из модуля `src.utils.xls`.

**Функции:**

* **`xls2dict(xls_file: str | Path) -> dict | None`:**
    * Принимает на вход путь к файлу XLS (строка или объект `Path`).
    * Использует функцию `read_xls_as_dict` из модуля `src.utils.xls` для чтения файла в виде словаря.
    * Возвращает словарь, содержащий данные из файла Excel, или `None`, если произошла ошибка при чтении файла.


**Предполагаемый использование (пример):**

```python
import pathlib

# Путь к файлу XLS
xls_file_path = pathlib.Path("path/to/your/file.xls")

# Вызов функции
result_dict = xls2dict(xls_file_path)

if result_dict:
    print(result_dict)
else:
    print("Ошибка при чтении файла.")
```

**Модули, используемые в файле:**

* `pathlib`:  для работы с путями к файлам.
* `src.utils.xls`: Модуль, содержащий функции `read_xls_as_dict` и `save_xls_file` для работы с XLS-файлами.  Необходимо документация для этого модуля.
