# Модуль hypotez/src/suppliers/grandadvance/header.py

## Обзор

Данный модуль содержит функции для определения корневого каталога проекта и загрузки настроек из файла `settings.json`.  Также он загружает описание проекта из файла `README.MD` и предоставляет доступ к различным метаданным проекта (имя, версия, описание и др.).

## Оглавление

* [Функции](#функции)
    * [`set_project_root`](#set_project_root)

## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная с каталога текущего файла, проходя вверх по иерархии директорий и останавливаясь на первой директории, содержащей указанные маркерные файлы.

**Параметры**:

* `marker_files` (tuple): Кортеж имен файлов или каталогов, используемых для определения корневого каталога проекта. По умолчанию (`('pyproject.toml', 'requirements.txt', '.git')`).

**Возвращает**:

* `Path`: Объект `Path` представляющий корневой каталог проекта, если он был найден, иначе - каталог, где расположен текущий скрипт.


**Вызывает исключения**:

* Не вызывает исключений.

## Переменные

### `__root__`

**Описание**:  Путь к корневому каталогу проекта.  Инициализируется вызовом функции `set_project_root`.

### `settings`

**Описание**: Словарь с настройками проекта, загруженными из файла `src/settings.json`. Может быть `None`, если файл не найден или содержит невалидные данные.

### `doc_str`

**Описание**: Строка с описанием проекта, загруженная из файла `src/README.MD`. Может быть `None`, если файл не найден или содержит невалидные данные.

### `__project_name__`

**Описание**: Имя проекта, полученное из настроек (`settings`).  По умолчанию `hypotez`.

### `__version__`

**Описание**: Версия проекта, полученная из настроек. По умолчанию пустая строка.

### `__doc__`

**Описание**: Описание проекта (строка), полученная из файла `README.MD`. По умолчанию пустая строка.

### `__details__`

**Описание**: Дополнительные детали проекта. По умолчанию пустая строка.

### `__author__`

**Описание**: Автор проекта, полученный из настроек. По умолчанию пустая строка.

### `__copyright__`

**Описание**: Авторские права на проект, полученные из настроек. По умолчанию пустая строка.

### `__cofee__`

**Описание**: Ссылка на страницу сбора средств для разработчика (чашечка кофе). По умолчанию установлено значение "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69".
