# Игра в стихи

## Обзор

Данный модуль реализует простую игру в стихи, где компьютер выбирает случайное слово, а игрок должен ввести рифмующееся слово.

## Содержание

- [Обзор](#обзор)
- [Используемые переменные](#используемые-переменные)
- [Логика игры](#логика-игры)

## Используемые переменные

### `words`

- **Описание**: Список слов, используемых в игре.
- **Тип**: `List[str]`
- **Пример**: `["cat", "hat", "bat", "rat", "mat", "fat", "sat", "gnat"]`

### `selected_word`

- **Описание**: Слово, выбранное случайным образом из списка `words`.
- **Тип**: `str`

### `user_word`
- **Описание**: Слово, введенное пользователем.
- **Тип**: `str`

## Логика игры

### 1. Инициализация

- Импортируется модуль `random` для выбора случайного слова.
- Создается список слов `words`.
- Выбирается случайное слово из списка `words` и сохраняется в переменной `selected_word`.
- Выводится сообщение для пользователя с выбранным словом.

```python
import random

# רשימת מילים
words = ["cat", "hat", "bat", "rat", "mat", "fat", "sat", "gnat"]

# בחירת מילה אקראית מהרשימה
selected_word = random.choice(words)

# הדפסת המילה הנבחרת למשתמש
print(f"YOUR WORD IS {selected_word}")
```
### 2. Игровой цикл
- Запускается цикл на 5 итераций для 5 попыток игрока.
- Запрашивается ввод слова от пользователя, которое переводится в нижний регистр для сравнения без учета регистра.
- Проверяется, совпадают ли последние две буквы введенного слова и выбранного слова.
- Если совпадают, выводится сообщение "RIGHT!".
- Если не совпадают, выводится сообщение "WRONG!".

```python
# לולאה למשחק (5 ניסיונות)
for _ in range(5):
    # קבלת מילה מהמשתמש
    user_word = input("ENTER A RHYMING WORD: ").lower()

    # בדיקה אם המילה מתחרזת
    if user_word[-2:] == selected_word[-2:]: # בדיקת התחרזות של שתי אותיות אחרונות
        print("RIGHT!") # המילה מתחרזת
    else:
        print("WRONG!") # המילה לא מתחרזת
```

### 3. Завершение игры

- После завершения цикла выводится сообщение "Game Over".

```python
print("Game Over")
```
