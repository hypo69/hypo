# exceptions.py - Модуль пользовательских исключений для API AliExpress

```markdown
## Файл: hypotez/src/suppliers/aliexpress/api/errors/exceptions.py

**Описание:** Модуль содержит пользовательские классы исключений, предназначенные для обработки ошибок, возникающих при взаимодействии с API AliExpress.

**Комментарии:**

* Модуль содержит иерархию исключений, наследующихся от базового класса `AliexpressException`.
* Каждый класс исключения имеет свой `__init__`-метод, принимающий строку `reason` для пояснения причины возникновения ошибки.
* Метод `__str__` возвращает строковое представление исключения, содержащее причину (reason).
* Классы исключений предназначены для конкретных типов ошибок, например, неверные аргументы (`InvalidArgumentException`), отсутствие продукта (`ProductIdNotFoundException`), ошибки при запросе к API (`ApiRequestException`, `ApiRequestResponseException`) и т.д.


**Классы исключений:**

* **`AliexpressException`:** Базовый класс для всех исключений, связанных с API AliExpress.  Принимает строку `reason` для описания ошибки.

* **`InvalidArgumentException`:** Исключение, которое возникает, когда входные аргументы некорректны.

* **`ProductIdNotFoundException`:** Исключение, возникающее при попытке найти продукт по несуществующему идентификатору.

* **`ApiRequestException`:** Исключение для ошибок, связанных с выполнением запроса к API AliExpress (например, проблемы с подключением или таймаутом).

* **`ApiRequestResponseException`:** Исключение для ошибок, связанных с невалидным ответом от API AliExpress (например, ошибка в формате ответа).

* **`ProductsNotFoudException`:** Исключение, которое возникает, когда при поиске продуктов ничего не найдено.

* **`CategoriesNotFoudException`:** Исключение, которое возникает, когда при поиске категорий ничего не найдено.

* **`InvalidTrackingIdException`:** Исключение для случаев, когда идентификатор отслеживания недействителен или отсутствует.


**Пример использования:**

```python
try:
    # Ваш код, который может вызвать исключение
    result = aliexpress_api_call(...)
except AliexpressException as e:
    print(f"Ошибка API: {e}")
```

**Важность:** Определение собственных исключений позволяет улучшить структуру кода, сделать его более читаемым и организованным.  Это помогает точно локализовать и обрабатывать различные типы ошибок, связанных с API.
