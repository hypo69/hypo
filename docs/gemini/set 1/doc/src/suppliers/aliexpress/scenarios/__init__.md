# Модуль `hypotez/src/suppliers/aliexpress/scenarios/__init__.py`

## Обзор

Этот модуль содержит сценарии для взаимодействия с платформой AliExpress.  В нём определены константы и импорты для других сценариев, таких как сценарий логина.

## Переменные

### `MODE`

**Описание**:  Переменная, хранящая режим работы (например, 'dev', 'prod').

**Значение**: `'dev'` по умолчанию.


## Модули

### `login`

**Описание**: Модуль, содержащий сценарии и функции для авторизации на AliExpress.

## Функции

В этом модуле функций нет.


## Импорты

### `from .login import login`

**Описание**:  Импортирует функцию `login` из подмодуля `login`.  Это указывает на то, что этот модуль предоставляет доступ к функциям, необходимым для работы со сценариями логина на AliExpress.
