# Модуль `hypotez/src/suppliers/ivory/header.py`

## Обзор

Этот модуль содержит функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и получения информации о проекте (имя, версия, описание и т.д.).

## Функции

### `set_project_root`

**Описание**: Определяет корневую директорию проекта, начиная с директории текущего файла и ища вверх по дереву директорий.  Функция останавливается на первой директории, содержащей указанные файлы-маркеры.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, которые будут использованы для определения корневой директории проекта. По умолчанию: `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращает директорию, в которой расположен текущий скрипт.

**Вызывает исключения**:

- Нет.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученной с помощью функции `set_project_root`.

**Тип**: `Path`


### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json`.

**Тип**: `dict` или `None`.


### `doc_str`

**Описание**: Содержание файла `README.MD`, содержащего описание проекта.

**Тип**: `str` или `None`.


### `__project_name__`

**Описание**: Название проекта, полученное из настроек (`settings.json`). Если настройки не найдены или не содержат `project_name`, значение по умолчанию `'hypotez'`.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта, полученная из настроек (`settings.json`). Если настройки не найдены или не содержат `version`, значение по умолчанию `''`.

**Тип**: `str`


### `__doc__`

**Описание**: Описание проекта, полученное из файла `README.MD`. Если файл не найден, значение по умолчанию `''`.

**Тип**: `str`


### `__details__`

**Описание**: Дополнительные детали проекта, по умолчанию пустая строка.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта, полученный из настроек (`settings.json`). Если настройки не найдены или не содержат `author`, значение по умолчанию `''`.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права на проект, полученные из настроек (`settings.json`). Если настройки не найдены или не содержат `copyright`, значение по умолчанию `''`.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка для поддержки разработчика. Получена из настроек `settings.json`, в противном случае - ссылка по умолчанию.

**Тип**: `str`



## Обработка исключений

Обработка исключений `FileNotFoundError` и `json.JSONDecodeError` при чтении файлов `settings.json` и `README.MD` предотвращает остановку программы при отсутствии этих файлов или некорректном формате данных.  Пустая строка `...` используется в качестве маркера для обработки этих исключений.
