# Конфигурация Generative AI

## Обзор

Данный JSON файл содержит конфигурационные параметры для использования моделей генеративного AI, таких как Gemini. Он определяет место хранения данных и список доступных моделей.

## Структура JSON

Файл представляет собой JSON-объект со следующими ключами:

- `storage`: Строка, указывающая на текущий выбранный метод хранения данных.
- `storage_choices`: Список строк, представляющий доступные варианты хранения данных.
- `model`: Строка, указывающая на текущую выбранную модель генеративного AI.
- `model_choices`: Список строк, представляющий доступные варианты моделей генеративного AI.

## Параметры конфигурации

### `storage`

**Описание**: Определяет текущий выбранный метод хранения данных.

**Тип**: `str`

**Возможные значения**: Одно из значений, указанных в списке `storage_choices`.

**Пример**:
```json
"storage": "external_storage"
```

### `storage_choices`

**Описание**: Представляет список доступных вариантов хранения данных.

**Тип**: `list[str]`

**Возможные значения**:
- `"external_storage"`: Хранение данных во внешнем хранилище.
- `"data"`: Локальное хранение данных.
- `"google_drive"`: Хранение данных на Google Drive.

**Пример**:
```json
"storage_choices": [
    "external_storage",
    "data",
    "google_drive"
  ]
```

### `model`

**Описание**: Определяет текущую выбранную модель генеративного AI.

**Тип**: `str`

**Возможные значения**: Одно из значений, указанных в списке `model_choices`.

**Пример**:
```json
"model": "gemini-2.0-flash-exp"
```

### `model_choices`

**Описание**: Представляет список доступных моделей генеративного AI.

**Тип**: `list[str]`

**Возможные значения**:
- `"gemini-2.0-flash-exp"`: Модель Gemini 2.0 Flash experimental.
- `"gemini-1.5-flash-8b-exp-0924"`: Модель Gemini 1.5 Flash 8B experimental.
- `"gemini-1.5-flash"`: Модель Gemini 1.5 Flash.

**Пример**:
```json
"model_choices": [
    "gemini-2.0-flash-exp",
    "gemini-1.5-flash-8b-exp-0924",
    "gemini-1.5-flash"
  ]
```
