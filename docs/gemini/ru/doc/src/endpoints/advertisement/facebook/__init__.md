# Модуль `hypotez/src/endpoints/advertisement/facebook/__init__.py`

## Обзор

Этот модуль предоставляет интерфейс для работы с рекламной платформой Facebook. Он импортирует необходимые классы и функции из других модулей данного пакета, включая `Facebook`, `FacebookFields`, `FacebookPromoter` и `get_event_url`.  В нём также определён константный параметр `MODE` со значением 'dev'.

## Переменные

### `MODE`

**Описание**: Переменная, хранящая режим работы (например, 'dev', 'prod').

**Значение**: `'dev'`

Этот модуль импортирует следующие компоненты:
