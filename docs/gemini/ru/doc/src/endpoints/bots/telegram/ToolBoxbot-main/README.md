# Документация для README.md

## Обзор

Этот файл представляет собой HTML-код, который содержит текст приветствия и описание возможностей Toolbox AI. Toolbox AI - это универсальный помощник, который может генерировать контент для различных рабочих задач.

## Подробней

Этот код является частью Telegram-бота `ToolBoxbot`, который предоставляет пользователям инструменты для генерации текстов, идей и изображений с использованием нейронных сетей. Он отображает приветственное сообщение и объясняет основные функции бота, такие как генерация текстов для SMM, email-рассылок, SEO, рекламы, генерация идей, создание изображений по описанию и транскрипция аудио- и видеоматериалов. Также упоминается наличие 5 бесплатных генераций для ознакомления с сервисом.
