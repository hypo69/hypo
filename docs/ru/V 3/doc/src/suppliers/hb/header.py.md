# src.suppliers.hb.header

## Обзор

Модуль `header.py` предназначен для определения метаданных проекта, таких как имя, версия, авторские права и пути к корневой директории. Он также загружает настройки из файла `settings.json` и документацию из `README.MD`.

## Подробней

Этот модуль важен для инициализации основных параметров проекта и обеспечения доступа к ним из других модулей. Он определяет функцию `set_project_root`, которая автоматически определяет корневую директорию проекта на основе наличия определенных файлов-маркеров. Загрузка настроек и документации позволяет централизованно управлять конфигурацией и описанием проекта.

## Функции

### `set_project_root`

```python
def set_project_root(marker_files: tuple = ('__root__', '.git')) -> Path:
    """
    Args:
        marker_files (tuple): Filenames or directory names to identify the project root.
    
    Returns:
        Path: Path to the root directory if found, otherwise the directory where the script is located.
    """
```

**Описание**: Находит корневой каталог проекта, начиная с директории текущего файла, и останавливается на первой директории, содержащей любой из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или каталогов для идентификации корня проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена, иначе - директория, где расположен скрипт.

**Примеры**:

```python
from pathlib import Path
root_path = set_project_root()
print(f"Root path: {root_path}")
```

## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта.

### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json`.

### `doc_str`

**Описание**: Строка с содержимым файла `README.MD`, используемая как документация проекта.

### `__project_name__`

**Описание**: Имя проекта, полученное из настроек или значение по умолчанию 'hypotez'.

### `__version__`

**Описание**: Версия проекта, полученная из настроек или пустая строка.

### `__doc__`

**Описание**: Документация проекта, полученная из файла `README.MD` или пустая строка.

### `__details__`

**Описание**: Дополнительные детали проекта (в данном коде не присваивается значение).

### `__author__`

**Описание**: Автор проекта, полученный из настроек или пустая строка.

### `__copyright__`

**Описание**: Информация об авторских правах, полученная из настроек или пустая строка.

### `__cofee__`

**Описание**: Сообщение с предложением поддержать разработчика чашкой кофе, полученное из настроек или значение по умолчанию.
