# Модуль hypotez/src/endpoints/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и документации из файла `README.MD`, а также для установки пути к корневой директории в переменную `sys.path`.  Он использует модули `sys`, `json`, `pathlib` и `packaging.version`.

## Функции

### `set_project_root`

**Описание**:  Определяет корневую директорию проекта, начиная от текущего файла и идя вверх по дереву каталогов.  Поиск прекращается при обнаружении файлов или директорий, указанных в `marker_files`.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, по которым определяется корень проекта. По умолчанию это `('pyproject.toml', 'requirements.txt', '.git')`.


**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена, иначе путь к директории текущего файла.

**Вызывает исключения**:

- Нет


### `set_project_root`

**Описание**:  Определяет корневую директорию проекта, начиная от текущего файла и идя вверх по дереву каталогов.  Поиск прекращается при обнаружении файлов или директорий, указанных в `marker_files`.

**Параметры**:

- `marker_files` (tuple, optional): Кортеж имен файлов или директорий, по которым определяется корень проекта. По умолчанию это `('pyproject.toml', 'requirements.txt', '.git')`.


**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена, иначе путь к директории текущего файла.

**Вызывает исключения**:

- Нет


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта.  Получен с помощью функции `set_project_root()`.

**Тип**: `Path`


### `settings`

**Описание**: Словарь со значениями из файла `settings.json`.

**Тип**: `dict`


### `doc_str`

**Описание**: Строка с содержимым файла `README.MD`.

**Тип**: `str`



### `__project_name__`

**Описание**: Имя проекта, полученное из настроек или, по умолчанию, 'hypotez'.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта, полученное из настроек или, по умолчанию, пустая строка.

**Тип**: `str`


### `__doc__`

**Описание**:  Документация проекта, полученная из `README.MD` или, по умолчанию, пустая строка.

**Тип**: `str`


### `__details__`

**Описание**: Подробности о проекте. По умолчанию пустая строка.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта, полученное из настроек или, по умолчанию, пустая строка.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права проекта, полученное из настроек или, по умолчанию, пустая строка.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка на возможность поддержать разработчика. Полученное из настроек или, по умолчанию, ссылка по умолчанию.

**Тип**: `str`


## Обработка исключений

В модуле используются конструкции `try...except` для обработки `FileNotFoundError` и `json.JSONDecodeError` при чтении файлов `settings.json` и `README.MD`.  Это позволяет избежать аварийного завершения программы при отсутствии или некорректном формате этих файлов.
