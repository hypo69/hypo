# Задача No 391
## Последовательность Хофштадтера
## Условия
Последовательность Хофштадтера G определяется следующим образом:

```
G(0) = 0;
G(1) = 1;
G(n) = n - G(G(n-1)) для n > 1.
```
Вот первые несколько членов последовательности: 0, 1, 1, 2, 3, 3, 4, 5, 6, 6, 8, 8, 8, 10, 11, 12...

Назовем сумму  S(n) = Σ G(k) для k=0 до n.

Например: S(5) = 0 + 1 + 1 + 2 + 3 + 3 = 10.

Найдите S(10<sup>9</sup>) mod 10<sup>9</sup>.
