# Модуль `even.py`

## Обзор

Этот модуль представляет собой простую игру, которая проверяет, является ли введенное пользователем число четным или нечетным, и выводит соответствующее сообщение.

## Функции

### `main`

**Описание**: Основная функция программы. Она запрашивает у пользователя целое число, проверяет, является ли оно четным или нечетным, и выводит соответствующее сообщение.

**Параметры**:
- Нет параметров

**Возвращает**:
- Нет возвращаемого значения.

**Вызывает исключения**:
- `ValueError`: Возникает, если пользователь вводит некорректное значение, не являющееся целым числом.
