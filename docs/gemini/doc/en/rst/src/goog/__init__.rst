Module goog
==========

.. module:: goog
    :platform: Windows, Unix
    :synopsis:

    This module provides functionality related to Google Sheets.


Description
-----------

This module initializes the `goog` namespace.  It currently defines a constant `MODE` and imports the `Spreadsheet` class from the `spreadsheet` submodule.
