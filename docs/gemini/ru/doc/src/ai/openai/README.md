# Модуль `openai`

## Обзор

Этот модуль предоставляет инструменты для работы с API OpenAI. Он содержит классы и функции для взаимодействия с различными моделями, такими как генерация текста, ответы на вопросы и другие задачи.

## Классы

### `OpenAI`

**Описание**: Базовый класс для взаимодействия с API OpenAI. Предоставляет методы для выполнения различных запросов к API.

**Методы**:

- `get_completion(prompt: str, model: str = "text-davinci-003") -> str | None`: Выполняет запрос к модели OpenAI для получения ответа.
  
  **Описание**:  Возвращает ответ, сгенерированный выбранной моделью на основе предоставленного запроса.
  
  **Параметры**:
  - `prompt` (str): Запрос для модели OpenAI.
  - `model` (str, optional): Название модели OpenAI. По умолчанию `text-davinci-003`.
  
  **Возвращает**:
  - `str | None`: Сгенерированный текст или `None` в случае ошибки.
  
  **Вызывает исключения**:
  - `OpenAIError`: Возникает при проблемах с подключением к API OpenAI или при получении ошибочного ответа.

## Функции

### `get_answer(question: str, context: str, model: str = "text-davinci-003") -> str | None`

**Описание**: Получает ответ на вопрос на основе предоставленного контекста, используя модель OpenAI.

**Параметры**:
  - `question` (str): Вопрос для модели.
  - `context` (str): Контекст для ответа на вопрос.
  - `model` (str, optional): Название модели OpenAI. По умолчанию `text-davinci-003`.

**Возвращает**:
- `str | None`: Ответ на вопрос, сгенерированный моделью, или `None` в случае ошибки.

**Вызывает исключения**:
- `OpenAIError`: Возникает при проблемах с подключением к API OpenAI или при получении ошибочного ответа.
