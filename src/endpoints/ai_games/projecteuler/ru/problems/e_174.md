# Задача No 174
## Подсчет составных плиток

## Условия

Мы определяем составную плитку, как квадрат, который состоит из нескольких концентрических слоев, каждый из которых имеет толщину 1. Например, квадрат 8x8 может быть сделан из внутреннего квадрата 6x6, окружая его слоем плиток.

Таким образом, квадрат 8x8 можно представить как составной из одного слоя плиток, и он будет использовать 8<sup>2</sup> - 6<sup>2</sup> = 28 плиток.
Аналогично, квадрат 14x14 может быть представлен как составной из двух слоев:
* слой 1: 14<sup>2</sup> - 12<sup>2</sup> = 28 плиток
* слой 2: 12<sup>2</sup> - 10<sup>2</sup> = 44 плиток

Всего потребуется 28+44 = 72 плитки.

Имея данное определение, выведем что:

- Квадрат 4x4 нельзя представить в качестве составной плитки.
- Квадрат 5x5 можно представить в качестве составной плитки 5<sup>2</sup> - 3<sup>2</sup> = 16 плиток.
- Квадрат 12x12 можно представить как составную плитку с одним слоем: 12<sup>2</sup> - 10<sup>2</sup> = 44 плитки или с двумя слоями: 12<sup>2</sup> - 8<sup>2</sup> = 80 и 8<sup>2</sup> - 4<sup>2</sup> = 48 и всего 128 плиток (то есть: 12<sup>2</sup> - 8<sup>2</sup> + 8<sup>2</sup> - 4<sup>2</sup> = 12<sup>2</sup> - 4<sup>2</sup> = 128 = 144 - 16).

Пусть T(n) будет количеством квадратов, которые могут быть представлены в качестве составных плиток, где `n` - количество плиток.

Например, T(28)=1, T(44)=2, T(128)=1.

Сколько значений `n` ≤ 1000000 таких, что T(n) = 10?
