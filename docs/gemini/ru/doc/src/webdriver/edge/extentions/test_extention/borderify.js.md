# Документация для borderify.js

## Обзор

Данный файл `borderify.js` является скриптом, который добавляет красную рамку вокруг всего тела HTML-документа и вызывает `alert()`.

## Описание скрипта

Скрипт `borderify.js` выполняет следующие действия:
1. Устанавливает стиль рамки для элемента `body` документа. Рамка имеет толщину 5 пикселей, сплошную линию и красный цвет.
2. Вызывает функцию `alert()`, которая отображает стандартное модальное окно с сообщением. Сообщение по умолчанию зависит от браузера.

## Используемые функции

### `document.body.style.border = "5px solid red";`

**Описание**: 
Устанавливает CSS-свойство `border` для элемента `body` документа.

**Параметры**:
- Свойство border принимает значение `5px solid red` (5-пиксельная сплошная красная рамка).

**Возвращает**:
- Ничего, изменяет стиль элемента body.

### `alert()`

**Описание**: 
Вызывает модальное окно с сообщением, которое по умолчанию зависит от браузера.

**Параметры**:
- Отсутствуют.

**Возвращает**:
- Ничего.
