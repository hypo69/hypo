# Модуль `facebook_fields`

## Обзор

Модуль `facebook_fields` предназначен для хранения и предоставления полей, используемых в объявлениях и событиях Facebook. Он загружает эти поля из JSON-файла и делает их доступными через атрибуты класса.

## Классы

### `FacebookFields`

**Описание**: Класс для хранения и предоставления полей, используемых в объявлениях и событиях Facebook.

**Методы**:
- `__init__`: Инициализирует класс и загружает поля из файла.
- `_payload`: Загружает данные полей из JSON-файла и устанавливает их как атрибуты класса.

#### `__init__`

**Описание**: Инициализирует экземпляр класса `FacebookFields` и вызывает метод `_payload` для загрузки полей.

**Параметры**:
   - Нет параметров.

**Возвращает**:
    - Ничего не возвращает.

#### `_payload`

**Описание**: Загружает данные полей из JSON-файла `facebook_feilds.json` и устанавливает их как атрибуты экземпляра класса.

**Параметры**:
   - Нет параметров.

**Возвращает**:
   - `bool | None`: Возвращает `True` при успешной загрузке и установке полей, в противном случае `None`.

**Вызывает исключения**:
    - Нет исключений
