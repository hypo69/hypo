# Модуль hypotez/src/goog/quickstart.py

## Обзор

Этот модуль предоставляет функции для взаимодействия с Google Apps Script API. Он позволяет создавать новые проекты скриптов, загружать файлы в эти проекты и получать URL-адреса созданных скриптов. Модуль использует библиотеки `google-auth-oauthlib` и `googleapiclient` для аутентификации и взаимодействия с API.

## Функции

### `main`

**Описание**: Функция `main` является точкой входа модуля. Она выполняет аутентификацию, создает новый проект скрипта, загружает файлы в него и выводит URL-адрес созданного скрипта в консоль.

**Параметры**:
Нет параметров.

**Возвращает**:
- Нет значения возвращается.

**Обрабатывает исключения**:
- `errors.HttpError`: Обрабатывает возможные ошибки при взаимодействии с API. Выводит содержимое ошибки в консоль.

## Константы

### `SCOPES`

**Описание**: Список скопов авторизации, необходимых для доступа к Google Apps Script API.

**Значение**: `['https://www.googleapis.com/auth/script.projects']`


### `SAMPLE_CODE`

**Описание**: Пример кода для файла `hello.js`.

**Значение**:  `function helloWorld() { console.log("Hello, world!"); }`


### `SAMPLE_MANIFEST`

**Описание**: Пример манифеста приложения для файла `appsscript.json`.

**Значение**: `{"timeZone": "America/New_York", "exceptionLogging": "CLOUD"}`


### `MODE`

**Описание**: Переменная, хранящая текущий режим работы (например, 'dev').

**Значение**: `'dev'`

## Модули

### `google.auth.transport.requests`

**Описание**: Модуль для работы с запросами, необходимыми для аутентификации.

### `google.oauth2.credentials`

**Описание**: Модуль для работы с учетными данными OAuth2.

### `google_auth_oauthlib.flow`

**Описание**: Модуль для управления процессом авторизации приложения.

### `googleapiclient.discovery`

**Описание**: Модуль для построения объектов API.

### `googleapiclient`

**Описание**: Модуль, предоставляющий доступ к Google APIs.

### `header`

**Описание**: Не описан.

### `src.gs`

**Описание**: Не описан.

### `pathlib`

**Описание**: Модуль для работы с путями к файлам.

**Использование**: `from pathlib import Path`


## Использование

Для запуска модуля, выполните скрипт Python, который содержит функцию `main`.  Убедитесь, что файл `credentials.json` с ключами доступа к API расположен в той же директории, что и скрипт. Также, необходимо, чтобы утилита `python` была доступна, и файл `token.json` был создан при выполнении авторизации.
