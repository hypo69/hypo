# Модуль `hypotez/src/suppliers/aliexpress/api/_examples/rest/__init__.py`

## Обзор

Данный модуль предоставляет импортные функции для работы с API AliExpress, относящиеся к различным функциональным блокам, например, запросам продуктов, заказов, и т.д.

## Импортированные классы

### `AliexpressAffiliateProductSmartmatchRequest`

**Описание**:  Класс, представляющий запрос для поиска продуктов, похожих на заданный.

### `AliexpressAffiliateOrderGetRequest`

**Описание**: Класс, представляющий запрос для получения информации о конкретном заказе.


### `AliexpressAffiliateOrderListRequest`

**Описание**: Класс, представляющий запрос для получения списка заказов.

### `AliexpressAffiliateHotproductDownloadRequest`

**Описание**: Класс, представляющий запрос для загрузки популярных продуктов.

### `AliexpressAffiliateProductdetailGetRequest`

**Описание**: Класс, представляющий запрос для получения подробной информации о продукте.


### `AliexpressAffiliateHotproductQueryRequest`

**Описание**: Класс, представляющий запрос для получения информации о популярных продуктах.

### `AliexpressAffiliateFeaturedpromoProductsGetRequest`

**Описание**: Класс, представляющий запрос для получения списка продуктов с выделенными акциями.

### `AliexpressAffiliateFeaturedpromoGetRequest`

**Описание**: Класс, представляющий запрос для получения подробной информации об акциях.

### `AliexpressAffiliateProductQueryRequest`

**Описание**: Класс, представляющий запрос для поиска продуктов по различным критериям.

### `AliexpressAffiliateCategoryGetRequest`

**Описание**: Класс, представляющий запрос для получения информации о категориях продуктов.

### `AliexpressAffiliateOrderListbyindexRequest`

**Описание**: Класс, представляющий запрос для получения списка заказов с определенным индексом.

### `AliexpressAffiliateLinkGenerateRequest`

**Описание**: Класс, представляющий запрос для генерации ссылок на продукты.
