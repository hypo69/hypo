# Модуль alirequests

## Обзор

Модуль `alirequests.py` предназначен для экспериментов с запросами к сайту AliExpress. В текущей версии модуль инициализирует веб-драйвер Firefox и открывает главную страницу AliExpress.

## Подробней

Этот модуль является частью подпроекта `suppliers` проекта `hypotez` и предназначен для автоматизации взаимодействия с AliExpress. 
В текущем виде код инициализирует веб-драйвер Firefox и открывает главную страницу AliExpress.

## Функции

### `Driver`

**Назначение**: Класс для управления веб-драйвером.

**Параметры**:
- `browser` (Chrome | Firefox): Указывает, какой браузер будет использоваться (Chrome или Firefox).

**Возвращает**:
- Объект веб-драйвера, готовый к использованию.

**Как работает функция**:
1. Инициализирует драйвер Firefox.
2. Открывает главную страницу AliExpress.

**Примеры**:
```python
from src.webdriver.driver import Driver, Chrome, Firefox

d = Driver(Firefox)
d.get_url(r"https://www.aliexpress.com")
