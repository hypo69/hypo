# Модуль `header.py`

## Обзор

Модуль `header.py` предназначен для демонстрации использования `pprint` и настройки путей для импорта модулей. Он включает в себя добавление пути к корневой директории проекта в `sys.path`.

## Импорты

В модуле используются следующие импорты:

- `sys`: Для работы с путями и системными переменными.
- `os`: Для работы с операционной системой, в частности, для получения текущей директории.
- `pathlib.Path`: Для удобной работы с путями к файлам и директориям.

## Переменные

### `__root__`

**Описание**: Абсолютный путь к корневой директории проекта.

**Тип**: `pathlib.Path`

**Значение**: Вычисляется как срез текущей директории, полученной с помощью `os.getcwd()`, до строки 'hypotez' включительно.

**Пример**: Если текущая директория `/home/user/projects/hypotez/src/utils/powershell/examples/pprint`, то `__root__` будет `/home/user/projects/hypotez`.
