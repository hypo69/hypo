**Описание:**

Данный модуль отвечает за автоматическую отправку рекламных объявлений в группы Facebook.  Он использует драйвер для взаимодействия с веб-сайтом Facebook, загружает список управляемых групп из файла JSON и запускает рекламную кампанию по заданным группам и категориям.

**Логика работы:**

1. **Инициализация:**
   - Импортируются необходимые библиотеки: `header`, `copy`, `Driver`, `Chrome`, `FacebookPromoter`, `logger`.
   - Создается экземпляр драйвера веб-драйвера Chrome.
   - Получает доступ к странице Facebook.
   - Загружает список путей к файлам с данными о группах (`filenames`).
   - Загружает список рекламных кампаний (`campaigns`).
   - Создается экземпляр класса `FacebookPromoter`, который отвечает за непосредственную отправку объявлений в группы.  Передаются драйвер, пути к файлам с группами и флаг `no_video`.

2. **Циклическая отправка объявлений:**
   - Цикл `while True` обеспечивает непрерывную работу.
   - Внутри цикла функция `promoter.run_campaigns` запускает рекламную кампанию.  Обратите внимание на использование `copy.copy(campaigns)`. Это важно, чтобы не изменять исходный список `campaigns` во время выполнения. Если бы это было не так, то в будущем эта функция могла бы привести к ошибкам.

3. **Обработка прерывания:**
