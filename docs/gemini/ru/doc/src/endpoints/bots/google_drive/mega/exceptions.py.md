# Модуль исключений для работы с Mega

## Обзор

Модуль содержит определения пользовательских исключений, специфичных для работы с Mega. Эти исключения используются для обработки ошибок, возникающих при взаимодействии с API Mega, таких как неверные учетные данные или ошибки в запросах.

## Подробней

Данный модуль предоставляет набор исключений, которые могут быть использованы для более точной обработки ошибок при работе с сервисом Mega. Использование специализированных исключений позволяет улучшить читаемость кода и упростить отладку.

## Классы

### `MegaException`

**Описание**: Базовый класс для всех исключений, связанных с Mega.

**Как работает класс**:
Класс `MegaException` является родительским для всех остальных исключений, определенных в этом модуле. Он наследуется от стандартного класса `Exception` и служит для группировки всех исключений, специфичных для работы с Mega.

### `MegaIncorrectPasswordExcetion`

**Описание**: Исключение, возникающее при вводе неверного пароля или электронной почты.

**Как работает класс**:
Класс `MegaIncorrectPasswordExcetion` наследуется от `MegaException` и предназначен для обработки ситуаций, когда пользователь вводит неверные учетные данные (пароль или электронную почту) при попытке аутентификации в Mega.

### `MegaRequestException`

**Описание**: Исключение, возникающее при ошибке в запросе к API Mega.

**Как работает класс**:
Класс `MegaRequestException` наследуется от `MegaException` и используется для обработки ошибок, возникающих при выполнении запросов к API Mega. Это может быть связано с неверным форматом запроса, проблемами с сетью или другими ошибками на стороне сервера.
