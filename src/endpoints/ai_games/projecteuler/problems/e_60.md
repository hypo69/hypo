# Задача No 60
## Простые пары
## Условия
Простые числа 3, 7, 109 и 673 интересны тем, что для любых двух из этих чисел, если их конкатенировать в любом порядке, результат будет простым числом. Например, взяв 7 и 109, получаем как 7109, так и 1097, оба из которых являются простыми. Сумма этих четырех простых чисел, 792, является первым примером суммы набора из четырех простых чисел с таким свойством.

Найдите наименьшую сумму набора из пяти простых чисел, для которых для любых двух чисел их конкатенация в любом порядке будет простым числом.
