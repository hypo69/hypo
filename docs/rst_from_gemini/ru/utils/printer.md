# Модуль `printer.py`

Файл: `C:\Users\user\Documents\repos\hypotez\src\utils\printer.py`

Этот модуль предоставляет функции для красивой и цветной печати данных в консоли. Он поддерживает различные типы данных, включая словари, списки, строки (включая проверку на существование и тип файлов) и позволяет задавать цвет текста, фона и стиль шрифта.

## Функции:

* **`_color_text(text: str, text_color: str = "", bg_color: str = "", font_style: str = "") -> str`**: Внутренняя вспомогательная функция для применения цветовых и стилистических настроек к тексту. Возвращает отформатированную строку.

* **`pprint(print_data: Any = None, text_color: str = "white", bg_color: str = "", font_style: str = "") -> None`**: Функция для красивой печати данных в консоли.
    * Принимает данные любого типа `Any`.
    * Поддерживает различные форматы вывода данных:
        * **Словари:**  Выводит данные словаря в формате JSON с отступами.
        * **Списки:**  Выводит каждый элемент списка на новой строке.
        * **Файлы (.csv, .xls):**  Если передан путь к файлу, функция пытается прочитать и вывести содержимое файла, но поддерживаются только csv и xls файлы.  Если тип файла не поддерживается, выводится соответствующее сообщение.
        * **Другие типы данных:** Выводит данные в строковом формате.
    *  Использует ANSI escape-коды для изменения цвета и стиля текста.
    *  Управляет обработкой исключений, возникающих при работе с данными.
    *  Если `print_data` равно `None`, выводит сообщение об отсутствии данных.
    * **Параметры форматирования:**
        * `text_color`: Цвет текста (например, "red", "green", "blue", "white"). По умолчанию – белый.
        * `bg_color`: Цвет фона (например, "bg_red", "bg_green").
        * `font_style`: Стиль шрифта (например, "bold", "underline").

## Словари и списки

При выводе словарей и списков, функция автоматически их форматирует.


## Обработка ошибок:

Функция `pprint` использует блок `try...except`, чтобы обрабатывать возможные исключения при работе с разными типами данных.  Это предотвращает сбой программы при некорректном вводе.


## ANSI Escape Codes

Используются стандартные ANSI escape-коды для управления цветом и стилем текста в консоли.

## Пример использования:


```python
from utils.printer import pprint

pprint({"name": "Alice", "age": 30}, text_color="green")
pprint([1, 2, 3, 4], text_color="blue")
```


## Дополнительные замечания:


* Модуль предполагает наличие библиотек `json`, `csv`, `pandas`, `pathlib` и `pprint`.
* Важно, чтобы консоль поддерживала ANSI escape-коды. В противном случае цвета и стили могут не отображаться корректно.
* Добавлены комментарии, описывающие назначение функций и переменных.
