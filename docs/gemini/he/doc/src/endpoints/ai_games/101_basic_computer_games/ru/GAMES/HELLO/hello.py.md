## פונקציות

### `print`

**Description**: הפונקציה מובנית ב Python להצגת טקסט,משתנים, או שילוב של שניהם.

**Parameters**:
- `objects` (object): האובייקט או האובייקטים שיודפסו.
- `sep` (str, optional): המחרוזת שמפרידה בין האובייקטים, ברירת מחדל היא רווח.
- `end` (str, optional): המחרוזת שמסיימת את הפלט, ברירת מחדל היא שורה חדשה.
- `file` (file, optional): קובץ אליו יודפס הפלט, ברירת מחדל היא פלט סטנדרטי.
- `flush` (bool, optional): האם לשטוף את הפלט באופן מיידי, ברירת מחדל היא `False`.

**Returns**:
- `None`: הפונקציה לא מחזירה ערך.

### `input`

**Description**: הפונקציה מובנית ב Python לקבלת קלט מהמשתמש.

**Parameters**:
- `prompt` (str, optional): הטקסט שיוצג למשתמש כבקשה לקלט.

**Returns**:
- `str`: מחרוזת המכילה את הקלט שסופק על ידי המשתמש.

### `f-string`
 
**Description**: פורמט המאפשר לשלב משתנים ישירות בתוך מחרוזות.

**Parameters**:
-`variable`(variable): כל משתנה מוגדר ניתן לשימוש בתוך המחרוזת המעוצבת.
  
**Returns**:
-`str`: מחרוזת עם הערכים של המשתנים המשולבים.
