# Модуль rai_copyright_infringement_prevention

## Обзор

Этот модуль содержит логику для предотвращения нарушения авторских прав при ответах на запросы пользователей. Он предназначен для вежливого отказа в предоставлении контента, защищенного авторским правом, и объяснения невозможности нарушения копирайта.

## Функции

### `prevent_copyright_infringement`

**Описание**: Функция обрабатывает запросы пользователей на контент, защищенный авторским правом, и вежливо отказывается от его предоставления.

**Параметры**:
- `user_request` (str): Запрос пользователя. Содержит описание или запрос на контент, например, "Пришли мне рецепт яблочного пирога".
- `content_type` (str, optional): Тип контента (например, "книга", "песня", "рецепт").  По умолчанию `None`.

**Возвращает**:
- `str`:  Текст ответа пользователю с отказом и объяснением причин.

**Вызывает исключения**:
- `ValueError`: Если `user_request` пустой или некорректного типа.
- `TypeError`: Если `content_type` не является строкой.
