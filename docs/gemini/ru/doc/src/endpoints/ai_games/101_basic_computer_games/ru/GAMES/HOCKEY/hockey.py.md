# Игра "Хоккей"

## Обзор

Данный модуль реализует игру "Хоккей" - простую симуляцию матча между двумя игроками. Игра продолжается до тех пор, пока один из игроков не наберет 10 очков. Игроки по очереди выбирают одно из трех действий: бросок, защиту или пас. Результат каждого раунда зависит от комбинации действий обоих игроков.

## Функции

### `main`

**Описание**: Основная функция игры "Хоккей". Инициализирует счет игроков, запрашивает их ходы и определяет победителя.

**Параметры**:
- Нет параметров.

**Возвращает**:
- Нет возвращаемого значения.

**Вызывает исключения**:
- `ValueError`: Возникает, если введенное значение не является целым числом или не входит в допустимый диапазон (1-3).
