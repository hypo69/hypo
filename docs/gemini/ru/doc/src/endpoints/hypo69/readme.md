# Модуль hypo69

## Обзор

Модуль `hypo69` предоставляет набор конечных точек для разработчиков. Включает в себя следующие компоненты:

* **small_talk_bot**: чат-бот на основе ИИ.
* **code_assistant**: модуль для обучения модели кода проекта.
* **psychologist_bot**: начальный этап разработки модуля анализа диалогов.
