# `header.py`

## Обзор

Файл `header.py` содержит общие настройки и метаданные для проекта, включая определение корневой директории, загрузку настроек из файла `settings.json`, чтение документации из `README.MD` и определение базовых переменных проекта.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла и двигаясь вверх, останавливаясь на первой директории, содержащей один из маркеров.

**Параметры**:
- `marker_files` (tuple): Имена файлов или директорий, которые идентифицируют корень проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена, иначе путь к директории, где расположен скрипт.

## Переменные

### `MODE`
**Описание**: Режим работы приложения, по умолчанию установлен в `dev`.

### `__root__`

**Описание**: Путь к корневой директории проекта.

### `settings`

**Описание**: Словарь настроек, загруженный из файла `settings.json`.

### `doc_str`

**Описание**: Строка, содержащая документацию, загруженную из файла `README.MD`.

### `__project_name__`

**Описание**: Название проекта, по умолчанию `hypotez`, или из настроек.

### `__version__`

**Описание**: Версия проекта, по умолчанию пустая строка, или из настроек.

### `__doc__`

**Описание**: Документация проекта из `README.MD`, по умолчанию пустая строка.

### `__details__`

**Описание**: Детали проекта, по умолчанию пустая строка.

### `__author__`

**Описание**: Автор проекта, по умолчанию пустая строка, или из настроек.

### `__copyright__`

**Описание**: Информация об авторских правах, по умолчанию пустая строка, или из настроек.

### `__cofee__`

**Описание**: Строка с предложением поддержать разработчика, по умолчанию содержит ссылку на boosty.to, или из настроек.
