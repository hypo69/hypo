# Задача No 498
## Остаток от биномиального коэффициента

## Условия

Для любых целых неотрицательных чисел *n* и *k*, биномиальный коэффициент (n choose k), обозначаемый как C(*n*,*k*), определяется как:
C(*n*,*k*) = *n*! / (*k*! * (*n*−*k*)!)
Например, C(5,3) = 5! / (3! * 2!) = 10.

Дано целое неотрицательное число *n*, целое число *k*, и простое число *p*, определим *f*(*n*, *k*, *p*) как остаток от деления C(*n*,*k*) на *p*.
Так, *f*(5,3,7) = 10 mod 7 = 3.

Дано целое положительное число *N* и простое число *p*, определим *S*(*N*,*p*) как сумму всех значений *f*(*n*,*k*,*p*) для всех 0 ≤ *n*, *k* ≤ *N*.
Например, *S*(10,7) = 236.

Найдите *S*(10<sup>9</sup>, 1000000007).
