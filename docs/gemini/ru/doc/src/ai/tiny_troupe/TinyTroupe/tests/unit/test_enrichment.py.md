# Документация модуля `test_enrichment.py`

## Обзор

Данный модуль содержит модульные тесты для проверки функциональности класса `TinyEnricher` из модуля `tinytroupe.enrichment`. Тест `test_enrich_content` проверяет, что метод `enrich_content` корректно обогащает входной текст, делая его как минимум в 3 раза больше по объему символов.

## Функции

### `test_enrich_content`

**Описание**:
Тестовая функция, проверяющая корректность работы метода `enrich_content` класса `TinyEnricher`. Функция подготавливает исходный текст и требования к обогащению, после чего вызывает метод `enrich_content` и проверяет, что полученный результат не является `None` и что длина результата как минимум в три раза превышает длину исходного текста.

**Параметры**:
- Нет параметров.

**Возвращает**:
- Нет возвращаемого значения.

**Вызывает исключения**:
- Нет явных исключений. В случае ошибки тест завершится с ошибкой `AssertionError`.
