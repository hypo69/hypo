## Основные возможности

Бот для Telegram, написанный на Python, который позволяет загружать файлы по прямым ссылкам, а также с сервисов Mega.nz, Dropbox на Google Drive.

## Установка

1.  Установите необходимые модули из `requirements.txt` с помощью команды:
    ```bash
    sudo pip3 install -r requirements.txt
    ```

## Запуск

1.  Запустите бота с помощью команды:
    ```bash
    python3 bot.py
    ```

## Использование

1.  Для начала работы с ботом, используйте команду `/auth` для авторизации.
2.  После авторизации отправьте боту ссылку на файл, который нужно загрузить на Google Drive.

## Доступные команды

-   `/start`: Выводит приветственное сообщение.
-   `/auth`: Запускает процесс авторизации.
-   `/revoke`: Удаляет сохраненные учетные данные.
-   `/help`: Выводит справочную информацию.

## Поддерживаемые ссылки

-   Прямые ссылки.
-   Ссылки Mega.nz.
-   Ссылки Dropbox.

## Требования

-   [Google Drive API Credential](https://console.cloud.google.com/apis/credentials) (тип "Другие").
-   Telegram Bot Token (полученный через BotFather).
-   Openload ftp login и Key (опционально).
-   Mega Email и Password (опционально).

## Настройка

1.  Создайте [Google Drive api Credential](https://console.cloud.google.com/apis/credentials) (тип "Другие") и скачайте JSON-файл.
2.  Поместите скачанный JSON-файл в корневую директорию бота и переименуйте его в `client_secrets.json`.
3.  Замените токен вашего бота в файле [creds.py](./creds.py).

## Хостинг

Вы можете использовать Heroku для хостинга бота. Убедитесь, что вы изменили токен бота и Google Client API перед размещением.
