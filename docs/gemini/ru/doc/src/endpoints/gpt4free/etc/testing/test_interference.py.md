# Модуль для тестирования взаимодействия с OpenAI (GPT-3.5-turbo)
=============================================================

Модуль содержит функцию `main`, которая осуществляет взаимодействие с OpenAI API (в данном случае, с локальным сервером, эмулирующим OpenAI API) для генерации текста на основе заданной модели.

## Обзор

Этот модуль предназначен для проверки работоспособности и взаимодействия с OpenAI API, используя модель `gpt-3.5-turbo`. Он отправляет запрос на генерацию текста и выводит результат в потоковом или не потоковом режиме.

## Подробней

Модуль `test_interference.py` выполняет следующие действия:

1.  Импортирует библиотеку `openai`.
2.  Устанавливает ключ API и базовый URL для OpenAI.
3.  Определяет функцию `main`, которая отправляет запрос к OpenAI API и обрабатывает ответ.
4.  Выводит сгенерированный текст в зависимости от того, является ли ответ потоковым или нет.
