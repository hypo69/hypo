# Модуль hypotez/src/endpoints/advertisement/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`.  Он также загружает описание проекта из файла `README.MD` и предоставляет доступ к различным параметрам проекта, таким как имя, версия, описание, автор, права и информация для пожертвований.

## Функции

### `set_project_root`

**Описание**: Определяет корневую директорию проекта, начиная с текущего файла и проходя вверх по иерархии директорий.  Ищет файлы или папки, указанные в аргументе `marker_files`, и останавливается на первой найденной директории, содержащей хотя бы один из них.

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или папок, которые используются для определения корневой директории проекта. По умолчанию используется кортеж `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращает директорию, где расположен текущий файл.

**Вызывает исключения**:

- Нет.


### Загрузка настроек

**Описание**: Загружает настройки из файла `settings.json` в переменную `settings`. Обрабатывает `FileNotFoundError` и `json.JSONDecodeError`, если файл не найден или поврежден.

### Загрузка документации

**Описание**: Загружает текст документации проекта из файла `README.MD` в переменную `doc_str`.  Обрабатывает `FileNotFoundError` и `json.JSONDecodeError`, если файл не найден или поврежден.

## Глобальные переменные

### `__root__`

**Описание**: Путь к корневой директории проекта. Инициализируется функцией `set_project_root`.

### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json`.

### `doc_str`

**Описание**: Строка с текстом документации проекта, загруженная из файла `README.MD`.

### `__project_name__`

**Описание**: Имя проекта. Значение берется из настроек (settings), по умолчанию равно `'hypotez'`.

### `__version__`

**Описание**: Версия проекта. Значение берется из настроек (settings), по умолчанию равно пустой строке.

### `__doc__`

**Описание**: Документация проекта. Значение берется из `doc_str`, по умолчанию пустая строка.

### `__details__`

**Описание**: Подробное описание проекта (пока пусто).

### `__author__`

**Описание**: Автор проекта. Значение берется из настроек (settings), по умолчанию пустая строка.

### `__copyright__`

**Описание**: Авторские права проекта. Значение берется из настроек (settings), по умолчанию пустая строка.

### `__cofee__`

**Описание**: Ссылка для пожертвований на кофе разработчику. Значение берется из настроек (settings), по умолчанию ссылка на Boosty.
