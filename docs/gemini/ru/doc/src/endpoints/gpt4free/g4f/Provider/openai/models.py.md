# Модуль определения моделей OpenAI

## Обзор

Этот модуль определяет списки моделей, используемых OpenAI, включая текстовые, графические и vision-модели. Он задает модели по умолчанию, а также списки доступных моделей для использования в проекте `hypotez`.

## Подробнее

Модуль содержит перечни моделей, которые используются при взаимодействии с OpenAI. Важно поддерживать эти списки актуальными, чтобы обеспечить совместимость и правильное функционирование системы.

## Переменные

### `default_model`

**Описание**: Модель по умолчанию для текстовых задач.

**Назначение**: Указывает, какая модель будет использоваться, если не указана конкретная модель.

### `default_image_model`

**Описание**: Модель по умолчанию для задач генерации изображений.

**Назначение**: Определяет, какая модель используется, если не указана конкретная модель для генерации изображений.

### `image_models`

**Описание**: Список моделей, поддерживающих задачи генерации изображений.

**Назначение**: Перечисляет все доступные модели для генерации изображений.

### `text_models`

**Описание**: Список моделей, поддерживающих текстовые задачи.

**Назначение**: Перечисляет все доступные модели для текстовых задач.

### `vision_models`

**Описание**: Список моделей, поддерживающих задачи, связанные с vision (анализ и обработка изображений).

**Назначение**: Перечисляет все доступные модели для vision задач.

### `models`

**Описание**: Объединенный список всех доступных моделей (текстовых и графических).

**Назначение**: Предоставляет полный список моделей, доступных для использования в системе.
