# Модуль исключений для работы с Mega

## Обзор

Модуль содержит пользовательские исключения, используемые при работе с API Mega. Эти исключения позволяют более точно обрабатывать ошибки, возникающие в процессе взаимодействия с сервисом Mega.

## Подробней

Данный код определяет иерархию исключений для обработки ошибок, специфичных для работы с Mega API. `MegaException` является базовым классом для всех исключений, связанных с Mega. Остальные классы, такие как `MegaIncorrectPasswordExcetion` и `MegaRequestException`, наследуются от него и представляют конкретные типы ошибок. Использование пользовательских исключений позволяет улучшить читаемость и поддерживаемость кода, а также предоставляет возможность более точной обработки ошибок.

## Классы

### `MegaException`

**Описание**: Базовый класс для всех исключений, связанных с Mega.

### `MegaIncorrectPasswordExcetion`

**Описание**: Исключение, которое выбрасывается, когда введен неверный пароль или email.

### `MegaRequestException`

**Описание**: Исключение, которое выбрасывается при возникновении ошибки в запросе к Mega.
