Crawlee Header Module
=====================

.. module:: crawlee_python.header
   :platform: Windows, Unix
   :synopsis: Module for project initialization and configuration.


This module provides functions for finding the project root directory, loading settings, and retrieving project metadata.  It relies on `settings.json` and `README.MD` files within the project structure.  Error handling is implemented for missing or malformed files.
