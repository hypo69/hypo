# borderify.js

## Обзор

Этот скрипт добавляет красную границу толщиной 5 пикселей к телу документа.  Также выводит диалоговое окно alert().


## Функции

### `document.body.style.border = "5px solid red";`

**Описание**: Устанавливает свойство `border` для элемента `body` на странице.

**Описание параметров:**

* `"5px solid red"`:  Строковое значение, определяющее стиль границы. 5px - толщина, solid - тип, red - цвет.
