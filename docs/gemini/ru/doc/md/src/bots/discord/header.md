# Модуль hypotez/src/bots/discord/header.py

## Обзор

Данный модуль содержит функции для определения корневого каталога проекта и загрузки настроек из файла `settings.json`. Он также предоставляет доступ к информации о проекте, такой как имя, версия, описание и автор.

## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная с текущего каталога, и восходя по иерархии каталогов до нахождения файлов-маркеров (pyproject.toml, requirements.txt, .git).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или каталогов, используемых для определения корневого каталога проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневому каталогу проекта. Если корневой каталог не найден, возвращает путь к текущему каталогу.

**Вызывает исключения**:

-  Не вызывает исключения.

## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта, полученный с помощью функции `set_project_root()`.

**Тип**: `Path`

### `settings`

**Описание**: Словарь со значениями настроек, загруженных из файла `settings.json`.

**Тип**: `dict` или `None`

### `doc_str`

**Описание**: Строка с описанием проекта из файла `README.MD`.

**Тип**: `str` или `None`


### `__project_name__`

**Описание**: Имя проекта, полученное из настроек (`settings.json`) или установленное по умолчанию.

**Тип**: `str`

### `__version__`

**Описание**: Версия проекта, полученная из настроек (`settings.json`).

**Тип**: `str`


### `__doc__`

**Описание**: Документация проекта из файла `README.MD`.

**Тип**: `str`


### `__details__`

**Описание**: Детали проекта.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта, полученный из настроек (`settings.json`).

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права проекта, полученные из настроек (`settings.json`).

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка для поддержки разработчика.

**Тип**: `str`
