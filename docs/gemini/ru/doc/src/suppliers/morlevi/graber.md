# Модуль hypotez/src/suppliers/morlevi/graber.py

## Обзор

Модуль `graber.py` содержит класс `Graber`, предназначенный для сбора данных о товарах с сайта `morlevi.co.il`. Класс наследуется от базового класса `Graber` и предоставляет функции для извлечения различных полей товара, включая обработку изображений и закрытие всплывающих окон.  Класс реализует асинхронные методы для обработки данных и включает в себя обработку ошибок.

## Классы

### `Graber`

**Описание**: Класс `Graber` реализует логику сбора данных о товаре с сайта `morlevi.co.il`. Он расширяет функциональность базового класса `Grbr` и предоставляет специфические методы для извлечения полей товара.

**Атрибуты**:

- `supplier_prefix`: Строка, определяющая префикс поставщика (`morlevi`).

**Методы**:

#### `__init__(self, driver: Driver)`

**Описание**: Инициализирует экземпляр класса `Graber`.

**Параметры**:

- `driver` (Driver): Экземпляр класса `Driver`, используемый для взаимодействия с веб-драйвером.

#### `grab_page(self, driver: Driver) -> ProductFields`

**Описание**: Асинхронный метод для сбора данных о товаре.

**Параметры**:

- `driver` (Driver): Экземпляр класса `Driver`.

**Возвращает**:

- `ProductFields`: Объект, содержащий собранные данные о товаре.

#### `local_saved_image(self, value: Any = None)`

**Описание**: Метод для получения и сохранения изображения товара локально. Сохраняет скриншот изображения в директории `tmp` и сохраняет путь к файлу в поле `local_saved_image` объекта `ProductFields`.

**Параметры**:

- `value` (Any, optional): Дополнительное значение, которое можно передать при вызове. При наличии, используется вместо скриншота. По умолчанию `None`.

**Возвращает**:

- `bool`: `True` в случае успешного сохранения, `False` в противном случае.
