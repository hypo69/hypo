# Задача No 224
## Треугольные числа, которые являются суммой квадратов
## Условия
Треугольные числа определяются как Tn = n(n+1)/2.  Например, T1=1, T2=3, T3=6, T4=10 и т.д.

Сначала мы ищем треугольные числа, которые являются суммой двух квадратов и меньше одного миллиона.

Первые несколько из них:
1 = 1^2 + 0^2,
10 = 3^2 + 1^2,
325 = 18^2 + 1^2 = 15^2 + 10^2.

Затем мы ищем треугольные числа, которые являются суммой трех квадратов и меньше одного миллиона.

Первые несколько из них:
6 = 2^2 + 1^2 + 1^2,
2016 = 44^2 + 14^2 + 8^2,
2550 = 48^2 + 15^2 + 15^2.

Затем мы ищем треугольные числа, которые являются суммой четырех квадратов и меньше одного миллиона.

Первые несколько из них:
15 = 3^2 + 2^2 + 1^2 + 1^2,
105 = 9^2 + 4^2 + 2^2 + 0^2.

Найдите количество треугольных чисел, которые являются суммой не более чем четырех квадратов и меньше 10^10.
