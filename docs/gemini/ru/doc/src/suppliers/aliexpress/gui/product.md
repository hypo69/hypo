# Модуль hypotez/src/suppliers/aliexpress/gui/product.py

## Обзор

Данный модуль предоставляет класс `ProductEditor` для редактирования данных продуктов, полученных из JSON-файлов AliExpress. Класс позволяет загружать JSON-файлы, отображать информацию о продукте и асинхронно подготавливать его для дальнейшей обработки.

## Классы

### `ProductEditor`

**Описание**: Класс `ProductEditor` представляет собой виджет PyQt6 для редактирования данных продуктов AliExpress. Он позволяет загружать данные из JSON-файла, отображать основные характеристики продукта и запускать асинхронную подготовку.

**Атрибуты**:

- `data: SimpleNamespace`: Загруженные данные из JSON-файла.
- `language: str`: Язык отображения (по умолчанию 'EN').
- `currency: str`: Валюта (по умолчанию 'USD').
- `file_path: str`: Путь к загруженному JSON-файлу.
- `editor: AliCampaignEditor`: Экземпляр класса `AliCampaignEditor` для обработки данных продукта.
- `main_app`: Экземпляр главного приложения.

**Методы**:

#### `__init__(self, parent=None, main_app=None)`

**Описание**: Инициализирует виджет `ProductEditor`.

**Параметры**:

- `parent (Optional[QWidget], optional)`: Родительский виджет. По умолчанию `None`.
- `main_app`: Экземпляр главного приложения.

#### `setup_ui(self)`

**Описание**: Настраивает пользовательский интерфейс (UI).

#### `setup_connections(self)`

**Описание**: Устанавливает соединения между элементами UI и соответствующими обработчиками событий.

#### `open_file(self)`

**Описание**: Открывает диалоговое окно для выбора JSON-файла.

**Возвращает**:
    - `None`

#### `load_file(self, file_path: str)`

**Описание**: Загружает данные из JSON-файла.

**Параметры**:

- `file_path (str)`: Путь к JSON-файлу.


**Вызывает исключения**:

- `Exception`: При ошибке загрузки JSON-файла. Ошибка будет отображена в диалоговом окне.

#### `create_widgets(self, data: SimpleNamespace)`

**Описание**: Создает виджеты на основе загруженных данных.

**Параметры**:

- `data (SimpleNamespace)`: Данные продукта.

#### `prepare_product_async(self)`

**Описание**: Асинхронно подготавливает продукт.

**Вызывает исключения**:

- `Exception`: При ошибке подготовки продукта. Ошибка будет отображена в диалоговом окне.
