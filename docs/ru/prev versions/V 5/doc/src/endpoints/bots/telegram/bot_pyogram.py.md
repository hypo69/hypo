# Модуль для работы с Telegram ботом на Pyrogram

## Обзор

Данный модуль предоставляет простой Telegram бот, использующий библиотеку Pyrogram. Бот отвечает на команду `/start` приветственным сообщением и эхом повторяет все остальные текстовые сообщения, которые он получает.

## Подробнее

Модуль содержит основные компоненты для запуска и обработки сообщений в Telegram боте. Он использует переменные окружения для получения API ID, API HASH и токена бота, что позволяет избежать жесткого кодирования этих параметров в коде.
Основное назначение этого кода - служить базовым примером для создания более сложных ботов с использованием Pyrogram.
