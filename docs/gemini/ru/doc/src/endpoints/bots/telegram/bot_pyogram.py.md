# Модуль для работы с Telegram ботом на Pyrogram
## Обзор

Модуль предоставляет простой Telegram бот, созданный с использованием библиотеки Pyrogram. Бот умеет отвечать на команду `/start` приветственным сообщением и повторять все остальные текстовые сообщения, отправленные пользователем.

## Подробней

Этот код представляет собой минимальную реализацию Telegram бота, использующего библиотеку Pyrogram. Он демонстрирует основные принципы работы с Pyrogram, такие как создание клиента, регистрация обработчиков сообщений и запуск бота.

```python
def start_command(client, message):
    """
    Обработчик команды /start.

    Args:
        client: Экземпляр клиента Pyrogram.
        message: Объект сообщения, содержащий информацию о входящем сообщении.

    Returns:
        None

    Raises:
        None
    """
    ...
```

**Как работает функция**:
1.  Функция `start_command` является обработчиком команды `/start`, то есть она вызывается, когда пользователь отправляет боту сообщение, начинающееся с `/start`.
2.  При получении команды `/start` функция отправляет пользователю приветственное сообщение "Привет! Я простой бот на Pyrogram." с помощью метода `reply_text` объекта `message`.

**Параметры**:

*   `client`: Экземпляр клиента Pyrogram, используемый для взаимодействия с Telegram API.
*   `message`: Объект сообщения, содержащий информацию о входящем сообщении, такую как текст сообщения, идентификатор пользователя и т.д.

**Возвращает**:

*   None

**Вызывает исключения**:

*   None

**Примеры**:

```python
# Пример вызова команды /start в Telegram
/start
```
