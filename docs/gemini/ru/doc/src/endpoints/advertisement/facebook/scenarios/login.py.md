# Модуль `login.py`

## Обзор

Модуль `login.py` предназначен для выполнения сценария авторизации пользователя в Facebook. Он использует драйвер браузера для взаимодействия с веб-страницей, заполнения полей логина и пароля и нажатия кнопки входа.

## Импорты

Модуль импортирует следующие библиотеки:

-   `pathlib.Path`: Для работы с путями к файлам.
-   `typing.Dict`: Для определения типов данных.
-   `src.gs`: Для доступа к глобальным настройкам приложения.
-   `src.webdriver.driver.Driver`: Класс для управления веб-драйвером.
-   `src.utils.jjson`: Функции для работы с JSON.
-   `src.logger.logger`: Для ведения логов.

## Константы

-   `MODE`: Указывает режим работы приложения, по умолчанию установлен в `dev`.
-   `locators`: Словарь, содержащий локаторы для элементов страницы авторизации Facebook. Загружается из `login.json`.

## Функции

### `login`

**Описание**:
Выполняет вход на Facebook. Функция использует переданный `Driver` для выполнения авторизации на Facebook, заполняя логин и пароль, а затем нажимает кнопку входа.

**Параметры**:
-   `d` (Driver): Экземпляр драйвера для взаимодействия с веб-элементами.

**Возвращает**:
-   `bool`: `True`, если авторизация прошла успешно, иначе `False`.

**Вызывает исключения**:
-   `Exception`: Если возникает ошибка при вводе логина, пароля или нажатии кнопки.
