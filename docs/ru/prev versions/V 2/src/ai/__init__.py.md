# Модуль `ai`

## Обзор

Модуль `ai` предоставляет интерфейс для взаимодействия с моделями искусственного интеллекта. В настоящее время он включает интеграцию с Google Gemini через класс `GoogleGenerativeAI`.

## Импортированные модули

### `GoogleGenerativeAI`

- **Описание**: Класс, предоставляющий интерфейс для работы с моделью Google Gemini.
  - Данный класс импортируется из модуля `hypotez.src.ai.gemini`.
   ```python
    from .gemini import GoogleGenerativeAI
   ```
