# Модуль `hypotez/src/endpoints/advertisement/facebook/scenarios/__init__.py`

## Обзор

Данный модуль предоставляет функции для работы с рекламными сценариями на Facebook. Он импортирует функции из различных подмодулей, обеспечивая удобный доступ к функционалу для создания, публикации и управления рекламными объявлениями и постами.

## Импортированные модули

*   `login`: Функции для входа в аккаунт Facebook.
*   `post_message`: Функции для публикации сообщений. Подмодуль содержит функции для работы с заголовками, изображениями, подписями к изображениям и публикацией сообщений.
*   `switch_account`: Функции для переключения между аккаунтами Facebook.
*   `post_event`: Функции для публикации событий. Подмодуль включает функции для работы с заголовком, описанием, датой, временем и публикацией события.
*   `post_ad`: Функции для публикации рекламных объявлений.


## Функции

### `login`

**Описание**: Функция для входа в аккаунт Facebook.

**Параметры**:
*   `credentials` (dict): Словарь с данными для авторизации (логин и пароль).

**Возвращает**:
*   `bool`: `True`, если вход успешен, `False` в противном случае.

**Вызывает исключения**:
*   `AuthenticationError`: Возникает при ошибке авторизации.
*   `ConnectionError`: Возникает при проблемах с подключением.


### `post_message_title`

**Описание**: Функция для создания заголовка сообщения.

**Параметры**:
*   `title` (str): Текст заголовка.


**Возвращает**:
*   `str`: Возвращает созданный заголовок.


**Вызывает исключения**:
*   `ValueError`: Возникает при пустом или некорректном заголовке.

### `upload_post_media`

**Описание**: Функция для загрузки медиафайлов для поста.

**Параметры**:
*   `media_files` (list): Список файлов, которые необходимо загрузить.


**Возвращает**:
*   `list`: Возвращает список ссылок на загруженные медиафайлы.


**Вызывает исключения**:
*   `FileNotFoundError`: Возникает при отсутствии указанных файлов.
*   `TypeError`: Возникает при некорректном типе входных данных.

### `update_post_media_captions`

**Описание**: Функция для обновления подписей к изображениям поста.

**Параметры**:
*   `media_ids` (list): Список идентификаторов загруженных медиафайлов.
*   `captions` (list): Список подписей к изображениям.

**Возвращает**:
*   `bool`: `True`, если обновление подписей успешно, `False` в противном случае.


**Вызывает исключения**:
*   `ValueError`: Возникает при несоответствии длин списка идентификаторов и списка подписей.

### `message_publish`

**Описание**: Функция для публикации сообщения на Facebook.


**Параметры**:
*   `post_data` (dict): Словарь с данными поста (заголовок, медиа, подписи и т.д.).


**Возвращает**:
*   `bool`: `True`, если публикация успешна, `False` в противном случае.


**Вызывает исключения**:
*   `PublishError`: Возникает при ошибке публикации.
