# Модуль code_checker_he

## Обзор

Этот модуль предназначен для проверки кода Python, обеспечивая его соответствие определённым требованиям, включая использование RST для документации и обработку ошибок при загрузке настроек.

## Функции

### `j_loads`

**Описание**: Функция для загрузки данных из JSON-файла.

**Параметры**:

- `filepath` (Path): Путь к файлу JSON.

**Возвращает**:

- `dict | None`: Словарь, содержащий данные из файла JSON, или `None` в случае ошибки.

**Обрабатываемые исключения**:

- `JSONDecodeError`: Возникает при ошибках декодирования JSON.


### `j_loads_ns`

**Описание**: Функция для загрузки данных из JSON-файла и возврата объекта `SimpleNamespace`.

**Параметры**:

- `filepath` (Path): Путь к файлу JSON.

**Возвращает**:

- `SimpleNamespace | None`: Объект `SimpleNamespace` содержащий данные из файла JSON, или `None` в случае ошибки.

**Обрабатываемые исключения**:

- `JSONDecodeError`: Возникает при ошибках декодирования JSON.
