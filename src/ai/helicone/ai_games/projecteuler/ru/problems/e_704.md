# Задача No 704
## Сумма степеней простых чисел

## Условия
Определим функцию $S(n)$ как сумму всех различных степеней простых чисел, которые меньше или равны $n$.

Например,
$S(1) = 0$
$S(2) = 2 + 2^2 + 2^3 + ... = 2 + 4 + 8 + ... = 2/(1 - 1/2) = 4$
$S(3) = 2 + 3 + 2^2 + 2^3 + ... + 3^2 + 3^3 + ... = 4 + 4.5 = 8.5$
$S(5) = 2 + 3 + 5 + 2^2 + 2^3 + ... + 3^2 + 3^3 + ... + 5^2 + 5^3 + ... = 4 + 4.5 + 6.25 = 14.75$

Определим $T(n)$ как сумму $S(p)$ для всех простых чисел $p \leq n$.

Например:
$T(1) = 0$
$T(2) = S(2) = 4$
$T(3) = S(2) + S(3) = 4 + 8.5 = 12.5$
$T(5) = S(2) + S(3) + S(5) = 4 + 8.5 + 14.75 = 27.25$
$T(10) = S(2) + S(3) + S(5) + S(7) = 2 + 3 + 5 + 7 + 4 + 9 + 25 + 49 + ... = 120.6081109...$

Пусть $F(k)$ равно $T(10^k)$.

Известно, что $F(4) = T(10^4) = 17105869.9731...$

Найдите $F(8) = T(10^8)$.  Приведите ваш ответ с точностью до десяти знаков после запятой.
