# Модуль: Ассистент Программирования

Этот модуль содержит класс `CodeAssistant`, который используется для взаимодействия с различными ИИ моделями, такими как Google Gemini и OpenAI, для задач обработки кода.  Пример использования показан ниже.

## Пример использования

```python
assistant = CodeAssistant(role='code_checker', lang='ru', model=['gemini'])
assistant.process_files()
```

## Классы

### `CodeAssistant`

**Описание**: Класс `CodeAssistant` используется для взаимодействия с различными ИИ моделями, такими как Google Gemini, и предоставляет методы для анализа и генерации документации для кода.

**Атрибуты**:

- `role` (str): Роль ассистента (например, 'code_checker').
- `lang` (str): Язык, на котором будет работать ассистент (например, 'ru').
- `model` (list): Список используемых ИИ моделей (например, ['gemini']).

**Методы**:

#### `process_files`

**Описание**: Метод для обработки файлов с кодом.

**Параметры**:
- Нет

**Возвращаемое значение**:
- `None`: Метод не возвращает значения.
