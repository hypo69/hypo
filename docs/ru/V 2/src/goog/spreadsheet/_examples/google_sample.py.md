# Модуль `google_sample.py`

## Обзор

Модуль демонстрирует базовое использование Sheets API Google. Он считывает значения из образцовой таблицы и выводит их в консоль.

## Константы

### `SCOPES`
- **Описание**: Список областей доступа, необходимых для работы с Google Sheets API. В данном случае требуется доступ только для чтения.
- **Тип**: `List[str]`
- **Значение**: `['https://www.googleapis.com/auth/spreadsheets.readonly']`

### `SAMPLE_SPREADSHEET_ID`
- **Описание**: Идентификатор образцовой таблицы Google Sheets.
- **Тип**: `str`
- **Значение**: `'1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgvE2upms'`

### `SAMPLE_RANGE_NAME`
- **Описание**: Диапазон ячеек в таблице, которые будут прочитаны.
- **Тип**: `str`
- **Значение**: `'Class Data!A2:E'`

### `ROOT_DIRECTORY`
- **Описание**: Абсолютный путь к текущей рабочей директории.
- **Тип**: `Path`

### `path`
- **Описание**: Путь к файлу с учетными данными `client_secret`.
- **Тип**: `Path`

## Функции

### `main`

**Описание**:
Основная функция, демонстрирующая использование Google Sheets API для чтения данных из таблицы.

**Параметры**:
- Нет параметров.

**Возвращает**:
- `None`: Функция ничего не возвращает.

**Вызывает исключения**:
- `HttpError`: Возникает, если происходит ошибка при обращении к Google Sheets API.

**Принцип работы**:
1. **Инициализация**:
   - Загружает учетные данные из файла `token.json`, если он существует.
   - Если файл не существует или учетные данные недействительны, запускается процесс аутентификации с помощью файла `credentials.json`.
   - Сохраняет учетные данные в файл `token.json` для последующих запусков.
2. **Получение данных**:
   - Создает объект `service` для работы с Google Sheets API.
   - Вызывает метод `get` для получения данных из указанного диапазона таблицы.
   - Извлекает значения из ответа API.
3. **Обработка и вывод**:
   - Если данные не найдены, выводится сообщение "No data found."
   - Если данные есть, выводит в консоль значения из первого и пятого столбцов каждой строки.
4. **Обработка ошибок**:
   - Ловит исключение `HttpError` и выводит информацию об ошибке.
