# Модуль hypotez/src/suppliers/amazon/graber.py

## Обзор

Модуль `hypotez/src/suppliers/amazon/graber.py` содержит класс `Graber`, предназначенный для сбора данных о товарах с сайта amazon.com.  Класс наследуется от родительского класса `Graber` и предоставляет функции для извлечения различных полей товара.  Включает в себя механизм предварительной обработки перед запросом к веб-драйверу (через декоратор).

## Классы

### `Graber`

**Описание**: Класс `Graber` реализует функциональность для сбора данных о товарах с amazon.com. Наследуется от `Grbr` (Graber) и предоставляет методы для извлечения информации.

**Атрибуты**:

- `supplier_prefix`: Строка, определяющая префикс поставщика (в данном случае "amazon").

**Методы**:

#### `__init__(self, driver: Driver)`

**Описание**: Инициализирует экземпляр класса `Graber`.

**Параметры**:

- `driver` (Driver): Экземпляр класса `Driver` для взаимодействия с веб-драйвером.


#### `grab_page(self, driver: Driver) -> ProductFields`

**Описание**: Асинхронная функция для сбора полей товара.

**Параметры**:

- `driver` (Driver): Объект драйвера для взаимодействия с веб-страницей.

**Возвращает**:

- `ProductFields`: Объект, содержащий собранные данные о товаре.

**Внутренняя логика**:

Функция `grab_page` вызывает функцию `fetch_all_data` для извлечения данных о товаре.  Она инициализирует глобальную переменную `d`, которая хранит экземпляр драйвера.

## Функции

### `fetch_all_data(**kwards)`

**Описание**:  Вспомогательная асинхронная функция для извлечения всех полей товара из `Amazon`.  Обрабатывает вызовы к функциям, отвечающим за извлечение конкретных полей, используя аргументы `kwards`.

**Параметры**:

- `kwards`: Словарь произвольных параметров, содержащих имена полей, которые нужно извлечь, и их значения.

**Возвращает**:
- `None`: Данная функция не возвращает никаких значений, но вызывает другие методы для обработки запросов.


**Важно:**  Достаточно большая часть кода в функции `fetch_all_data` содержит вызовы методов `self.<метод>(...)`.  Это означает, что функции, такие как `self.id_product`, `self.description`, и т.д.  должны быть определены в классе `Graber` (или базовом классе `Grbr`).   Для каждого поля, которое требуется извлечь, должна быть функция в классе `Graber`.  В данном коде есть много не реализованных функций - их нужно реализовать, чтобы этот код работал.
