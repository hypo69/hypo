# Модуль gworksheets

## Обзор

Модуль `gworksheets` предоставляет инструменты для работы с электронными таблицами Google Sheets. Он позволяет создавать, получать доступ к и манипулировать листами (worksheet), управлять заголовками и категориями, а также устанавливать направление текста.

## Классы

### `GWorksheet`

**Описание**: Класс `GWorksheet` наследуется от `Worksheet` и предоставляет методы для работы с листами в Google Sheets.  Он позволяет добавлять, изменять и получать доступ к данным на листе.

**Атрибуты**:

- `sh`: Объект, представляющий книгу (spreadsheet).
- `ws`: Объект, представляющий лист (worksheet).
- `render`: Объект `GSRender`, используемый для форматирования и управления отображением данных.

**Методы**:

#### `__init__`

**Описание**: Конструктор класса `GWorksheet`.

**Параметры**:

- `self`: Текущий объект.
- `sh`: Объект, представляющий книгу (spreadsheet).
- `ws_title` (str = 'new'): Название листа. По умолчанию 'new'.
- `rows` (optional): Количество строк. По умолчанию `None`.
- `cols` (optional): Количество столбцов. По умолчанию `None`.
- `direcion` (str = 'rtl'): Направление текста. По умолчанию 'rtl'.
- `wipe_if_exist` (bool = True):  Флаг, указывающий на очистку данных на листе, если лист с таким именем уже существует.


**Возвращает**:
- `None`


#### `get`

**Описание**: Создает новый лист или открывает существующий лист в Google Sheets.

**Параметры**:

- `self`: Текущий объект.
- `sh`: Объект, представляющий книгу (spreadsheet).
- `ws_title` (str = 'new'): Название листа. По умолчанию 'new'.
- `rows` (int = 100): Количество строк. По умолчанию 100.
- `cols` (int = 100): Количество столбцов. По умолчанию 100.
- `direction` (str = 'rtl'): Направление текста. По умолчанию 'rtl'.
- `wipe_if_exist` (bool = True): Флаг, указывающий на очистку данных на листе, если лист с таким именем уже существует.


**Описание документации внутри метода**:
* Создаёт новую таблицу в книге, если `ws_title` равно 'new', иначе открывает существующий лист по `ws_title`.
* `ws_title` (str) — Название листа (worksheet) в книге (spreadsheet).
* `rows` (int) — Количество строк.
* `cols` (int) — Количество столбцов.
* `wipe_if_exist` (bool) — Очистить данные листа при открытии, если лист с таким именем уже существует.


**Возвращает**:
- `None`


#### `header`

**Описание**: Устанавливает заголовок для листа.

**Параметры**:

- `self`: Текущий объект.
- `world_title` (str): Заголовок.
- `range` (str = 'A1:Z1'): Диапазон ячеек для заголовка. По умолчанию A1:Z1.
- `merge_type` (str('MERGE_ALL') | str('MERGE_COLUMNS') | str('MERGE_ROWS') = 'MERGE_ALL'): Тип слияния ячеек для заголовка. По умолчанию MERGE_ALL.


**Возвращает**:
- `None`


#### `category`

**Описание**: Записывает название категории на листе.

**Параметры**:

- `self`: Текущий объект.
- `ws_category_title`: Название категории.


**Возвращает**:
- `None`


#### `direction`

**Описание**: Устанавливает направление текста на листе.

**Параметры**:

- `self`: Текущий объект.
- `direction` (str = 'rtl'): Направление текста. По умолчанию 'rtl'.


**Возвращает**:
- `None`
