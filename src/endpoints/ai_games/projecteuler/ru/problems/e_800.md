# Задача No 800
## Диофантовы тройки
## Условия

Тройка положительных целых чисел (a, b, c) называется диофантовой, если a<sup>2</sup> + b<sup>2</sup> = c<sup>2</sup>.

Например, (3, 4, 5) является диофантовой тройкой.

Пусть T(n) - количество диофантовых троек, где a, b и c не превосходят n.

T(10) = 1 (тройка (3, 4, 5))
T(100) = 16
T(1000) = 156
T(10000) = 1509

Найдите T(10<sup>18</sup>).
