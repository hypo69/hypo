# Задача No 247
## Площади квадратов
## Условия
Рассмотрим процесс, начинающийся с квадрата 1 x 1. Мы добавляем квадраты к правой стороне и сверху, которые удовлетворяют следующим условиям:

1. Новые квадраты должны иметь максимально возможную площадь.
2. Новые квадраты должны добавляться к правой стороне или сверху текущего набора квадратов, формируя прямоугольник.
3. Нет два квадрата не должны перекрываться.

Примером такого процесса является:
 
![Изображение последовательности квадратов](https://projecteuler.net/project/images/p247_squares.gif)

Начнем с квадрата размера 1, затем добавим квадрат размера 1 справа. Затем добавляем квадрат размера 2 сверху. Теперь мы добавляем квадрат размера 3 справа и, наконец, квадрат размера 5 сверху.

Пусть f(n) представляет количество квадратов, которые были созданы, если нарисовать такой процесс, пока самый левый нижний угол самого крайнего квадрата не пересечет линию y = x.

Таким образом, f(5) = 5, f(10) = 13, f(100) = 144.

Найдите f(1000000).
