# Задача No 446
## Retractions
## Условия
Определим последовательность *S* следующим образом:
  * *S*<sub>0</sub> = 290797
  * *S*<sub>*n*+1</sub> = *S*<sub>*n*</sub><sup>2</sup> mod 50515093

Пусть *T*<sub>*n*</sub> будет числом *i* ≤ *n* таким, что *S*<sub>*i*</sub> ≡ 0 (mod 1000).

Найдите ∑*T*<sub>*n*</sub> для 1 ≤ *n* ≤ 10<sup>11</sup>.

## Пример
*T*<sub>1000</sub> = 0.
*T*<sub>2000</sub> = 0.
*T*<sub>3000</sub> = 1.
*T*<sub>10000</sub> = 1.
