# Документация к файлу popup.js

## Обзор

Файл `popup.js` отвечает за взаимодействие с пользователем и отправку запросов к серверу для получения ответов от модели обработки естественного языка.  Он использует AngularJS для построения интерфейса и `$http` для выполнения HTTP запросов.

## Оглавление

* [Функции](#функции)
    * [`loadAssistants`](#loadassistants)
    * [`sendMessage`](#sendMessage)

## Функции

### `loadAssistants`

**Описание**: Функция загружает список доступных ассистентов с сервера.

**Параметры**:
Нет

**Возвращает**:
Нет

**Вызывает исключения**:
- Любое исключение, возникающее при выполнении HTTP запроса `$http.get`.

### `sendMessage`

**Описание**: Функция отправляет сообщение модели обработки естественного языка на сервер.

**Параметры**:
Нет. Внутри функции используется переменная `$scope.message` и данные из выбранного ассистента

**Возвращает**:
Нет

**Вызывает исключения**:
- Любое исключение, возникающее при выполнении HTTP запроса `$http.post`.


## Использование AngularJS

Этот код использует AngularJS для управления приложением. Он определяет модуль `openaiApp` и контроллер `MainController`.

## Важные замечания

- Необходимо убедиться, что сервер (FastAPI) работает по адресам `http://localhost:8000/assistants` и `http://localhost:8000/ask`.
- В коде используется `$scope.selectedAssistant.id` для передачи ID ассистента.  Это предполагает, что `$scope.selectedAssistant` содержит объект с полем `id`, содержащим идентификатор ассистента.  Обязательно обеспечьте корректную структуру данных ассистентов на стороне сервера.
- При возникновении ошибок в коде используется блок `catch` для обработки исключений и вывода сообщений об ошибках пользователю.
