# hypotez/src/webdriver/firefox/extentions/test_extention/html/popup.html

## Обзор

Файл `popup.html` содержит разметку HTML для всплывающего окна расширения браузера Firefox. Он отображает заголовок "Hypotez" и параграф с приветственным сообщением.

## Разметка HTML

Файл содержит стандартную разметку HTML документа:

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Hypotez</title>
</head>
<body>
    <h1>Hypotez</h1>
    <p>Привет, Это Давидка. Я обучаю модель</p>
</body>
</html>
```

Этот код создает основную структуру HTML документа, включая декларацию типа документа, мета-теги для кодировки и масштабирования, заголовок страницы, а также заголовок уровня 1 и параграф с текстом.
