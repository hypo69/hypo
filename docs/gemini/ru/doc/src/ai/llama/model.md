# Модуль hypotez/src/ai/llama/model.py

## Обзор

Этот модуль предоставляет константу `MODE`, которая определяет режим работы приложения.

## Константы

### `MODE`

**Описание**: Константа, определяющая режим работы.

**Значение**: `"dev"`.

**Использование**:  Эта константа может быть использована для управления различными аспектами приложения, например, включения или отключения функционала в режиме разработки.
