## Примеры подготовки кампаний

Этот модуль содержит примеры использования функций из модуля `prepare_campaigns`, демонстрирующие различные сценарии подготовки кампаний на AliExpress.

## Функции

### `process_campaign_category`

**Описание**: Обрабатывает категорию кампаний.

**Параметры**:
- `campaign_name` (str): Название кампании.
- `category` (str): Название категории.
- `language` (str): Язык.
- `currency` (str): Валюта.
- `force` (bool, optional): Принудительно пересчитать данные. По умолчанию `False`.

**Возвращает**:
- `None`: Данные обновляются непосредственно в базе данных.


### `process_campaign`

**Описание**: Обрабатывает конкретную кампанию.

**Параметры**:
- `campaign_name` (str): Название кампании.
- `categories` (list[str], optional): Список категорий. По умолчанию `None`.
- `language` (str, optional): Язык. По умолчанию `None`.
- `currency` (str, optional): Валюта. По умолчанию `None`.
- `force` (bool, optional): Принудительно пересчитать данные. По умолчанию `False`.


**Возвращает**:
- `None`: Данные обновляются непосредственно в базе данных.


### `process_all_campaigns`

**Описание**: Обрабатывает все кампании.

**Параметры**:
- `language` (str, optional): Язык. По умолчанию `None`.
- `currency` (str, optional): Валюта. По умолчанию `None`.
- `force` (bool, optional): Принудительно пересчитать данные. По умолчанию `False`.

**Возвращает**:
- `None`: Данные обновляются непосредственно в базе данных.


### `get_directory_names`

**Описание**: Возвращает список имен директорий.

**Параметры**:
- `directory_path` (Path): Путь к директории.

**Возвращает**:
- `list[str]`: Список имен директорий.


**Примечание**:  Подробная документация для функций `process_campaign_category`, `process_campaign`, `process_all_campaigns` и `get_directory_names` отсутствует в предоставленном коде.  Для более полной документации необходимы  дополнительные комментарии к исходному коду в соответствующих функциях.  Также неясно, что представляют переменные `gs` и `Path`, их нужно определить для корректного описания.
