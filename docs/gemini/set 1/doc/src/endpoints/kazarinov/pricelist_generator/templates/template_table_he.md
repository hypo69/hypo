# Шаблон HTML для генерации таблицы прайслиста (HE)

## Обзор

Данный HTML-шаблон предназначен для генерации таблицы прайслиста, ориентированной на пользователей из Израиля.  Он отображает список продуктов с их изображениями, названиями, описаниями и спецификациями.  Шаблон использует RTL (справа налево) расположение текста.

## Структура шаблона

### Тег `<html>`

Отвечает за общие настройки HTML документа.  Атрибут `dir="rtl"` задает направление текста справа налево.

### Тег `<head>`

Содержит мета-теги, задающие кодировку, масштабирование и заголовок страницы. Также содержит стили, определяющие отображение.

### Тег `<body>`

Содержит основную часть страницы.

#### Тег `<div class="container">`

Контейнер для всего содержимого.  Ширина задана в `90%`, для удобства отображения.

##### Тег `<h1>`

Заголовок страницы.  Текст выводится по центру.

##### Тег `<p class="lead">`

Краткое описание содержимого страницы, выводится по центру.

##### Тег `<table>`

Таблица, содержащая карточки продуктов.

###### Тег `<tr class="product-card">`

Строка таблицы, представляющая собой карточку продукта.

####### Тег `<td>`

Ячейка таблицы, содержащая информацию о продукте.

######## Тег `<h3>`

Название продукта.  Расположен внутри тега `<td>`.

######## Тег `<table>`

Вложенная таблица для более структурированного отображения изображения и текста продукта.

######### Тег `<img>`

Изображение продукта. Атрибуты `src`, `alt` важны для правильной работы.

######### Тег `<td class="product-info">`

Ячейка для описания продукта и спецификаций.

############ Тег `<p>`

Элементы, содержащие описание продукта и его спецификаций.

##### Тег `<div class="footer">`

Футер страницы, содержит итоговую информацию о цене.

###### Тег `<p>`

Содержит информацию о суммарной цене.  Использует класс `price-tag` для выделения цены.

## Стили

### `body`

Задает фон, цвет текста, шрифт, и направление текста.

### `h1`

Задает выравнивание и размер заголовка `<h1>`.

### `.lead`

Задает выравнивание и размер текста описания.

### `.container`

Определяет ширину контейнера для содержимого.

### `table`

Настраивает таблицу для отображения без отступов между ячейками.  Устанавливает RTL для таблицы.

### `.product-card`

Стиль для карточек продуктов, задает фон, рамку, радиусы, отступы, ширину и тип бокса.

### `.product-card h3`

Отменяет верхний отступ для заголовков продуктов.

### `.product-card img`

Устанавливает ширину и высоту изображений, сохраняет пропорции, добавляет радиусы, белый фон.

### `.product-info`

Задает пропорциональное заполнение пространства.

### `.price-tag`

Стиль для блока цены, задает цвет, отступы, радиус, жирный шрифт.

### `.footer`

Стиль для футера, задаёт выравнивание, отступ, и верхнюю границу.

## Использование шаблона

Шаблон использует Django шаблонизацию.  Внутри шаблона необходимо передавать данные о продуктах (например, `products`, `title`, `description`, `price`, `currency`).

**Пример использования:**

```html
{% extends 'base.html' %}
{% load static %}

{% block content %}
    {% include 'pricelist_generator/template_table_he.html' with products=product_list title='Прайс-лист' description='Список продукции' price=total_price currency='NIS' %}
{% endblock %}
```

В этом примере подставляются списки продуктов, заголовок страницы, описание, итоговая цена и валюта.

**Важные замечания:**
- Обратите внимание на использование `{{ product.image_local_saved_path }}` для пути к изображению.  Необходимо корректно обрабатывать путь для сохранения изображений.
- Проверьте, что переменные `product_list`, `total_price`, `currency`, `title` и `description` корректно заполняются в Django коде.
