# Модуль ogg.js

## Обзор

Этот модуль предоставляет класс `OggConverter` для преобразования файлов OGG в MP3 и загрузки аудио из URL в формат OGG.

## Класс `OggConverter`

**Описание**: Класс для конвертации и загрузки аудио файлов.

### Метод `toMp3`

**Описание**: Преобразует файл OGG в MP3.

**Параметры**:

- `input` (строка): Путь к файлу OGG.
- `output` (строка): Имя выходного файла MP3 (без расширения).

**Возвращает**:

- `Promise<string | null>`: Путь к созданному файлу MP3 или `null` в случае ошибки.


**Обработка исключений**:

- Блок `try...catch`: Обрабатывает возможные ошибки при выполнении конвертации. Если возникает ошибка, `console.log` выводит сообщение об ошибке.

**Пример использования**:

```javascript
ogg.toMp3('/path/to/input.ogg', 'output_file')
  .then(path => console.log('Файл успешно сохранен:', path))
  .catch(error => console.error('Ошибка преобразования:', error));
```

### Метод `create`

**Описание**: Загружает аудио из URL в формате OGG.

**Параметры**:

- `url` (строка): URL аудио файла.
- `filename` (строка): Имя файла OGG (без расширения).

**Возвращает**:

- `Promise<string | null>`: Путь к созданному файлу OGG или `null` в случае ошибки.

**Обработка исключений**:

- Блок `try...catch`: Обрабатывает возможные ошибки при загрузке файла. Если возникает ошибка, `console.log` выводит сообщение об ошибке.


**Пример использования**:

```javascript
ogg.create('https://example.com/audio.ogg', 'downloaded_audio')
  .then(path => console.log('Аудио загружено:', path))
  .catch(error => console.error('Ошибка загрузки:', error));
```


##  Использование


**Описание**:  Класс `OggConverter` используется для конвертации и загрузки файлов.  Создайте экземпляр класса, затем вызовите метод `toMp3` для преобразования или `create` для загрузки.


## Импорты

- `axios`: Библиотека для HTTP запросов.
- `fs`: Модуль для работы с файловой системой (в частности, `createWriteStream`).
- `path`: Модуль для работы с путями.
- `url`: Модуль для работы с URL.
- `ffmpeg`: Библиотека для конвертации видео/аудио.
- `@ffmpeg-installer/ffmpeg`:  Менеджер установки FFmpeg.
- `utils.js`:  Модуль с вспомогательными функциями (в примере указана функция `removeFile`).

```javascript
import axios from 'axios';
import { createWriteStream } from 'fs';
// ... (другие импорты)
```


##  Примечания

- Необходимо установить необходимые зависимости: `npm install axios fluent-ffmpeg @ffmpeg-installer/ffmpeg fs node-fetch path url`.
