# Модуль `hypotez/src/suppliers/morlevi/__init__.py`

## Обзор

Данный модуль предоставляет начальную точку входа для работы с поставщиком данных Morlevi.  Он содержит константу `MODE` и импортирует класс `Graber` из подмодуля `graber`.

## Переменные

### `MODE`

**Описание**: Константа, определяющая режим работы модуля. В данном случае значение `'dev'`.

## Классы

### `Graber`

**Описание**: Класс для работы с API поставщика данных Morlevi. Подробности реализации находятся в файле `graber.py`.  Этот класс скорее всего предоставляет методы для извлечения, обработки и хранения данных.  Подробное описание класса доступно в документации `graber.py`.
