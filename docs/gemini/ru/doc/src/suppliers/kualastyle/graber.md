# Модуль `hypotez/src/suppliers/kualastyle/graber.py`

## Обзор

Модуль `hypotez/src/suppliers/kualastyle/graber.py` содержит класс `Graber`, предназначенный для сбора данных с веб-страниц товаров на сайте `kualastyle.co.il`.  Класс наследуется от базового класса `Grbr` и предоставляет функции для обработки полей товаров.  Он позволяет выполнять предварительные действия перед отправкой запроса к веб-драйверу с помощью декоратора `@close_pop_up`.
