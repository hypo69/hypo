# Модуль src.webdriver.chrome.extentions

## Обзор

Модуль `src.webdriver.chrome.extentions` предназначен для управления расширениями Chrome в контексте автоматизированного тестирования веб-приложений. Он предоставляет функциональность для работы с версиями расширений и метаданными.

## Подробней

Этот модуль, вероятно, используется для определения и управления версиями расширений, используемых в Chrome, когда запускается автоматизированное тестирование веб-приложений. Он может включать функции для проверки совместимости версий расширений, установки и обновления расширений, а также для получения метаданных расширений. Он предоставляет информацию о версии расширения, подробное описание и детали. В проекте `hypotez` он служит для унификации версий используемых расширений.

## Функции

### `__version__`

```python
__version__: str
```

**Описание**: Строка, представляющая версию расширения.

**Параметры**:
- Нет

**Возвращает**:
- `str`: Версия расширения.

**Вызывает исключения**:
- Нет

**Примеры**:
```python
from src.webdriver.chrome.extentions.version import __version__
print(__version__)
```
