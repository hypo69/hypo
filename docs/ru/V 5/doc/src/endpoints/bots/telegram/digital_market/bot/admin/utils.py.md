# Модуль утилит для обработки сообщений об удалении

## Обзор

Модуль содержит функцию `process_dell_text_msg`, которая обрабатывает сообщения об удалении текста в Telegram боте. Он использует `aiogram` для работы с ботом и `FSMContext` для управления состоянием. Функция удаляет последнее сообщение пользователя и сообщение, вызвавшее функцию.

## Подробней

Этот модуль предоставляет утилиту для удаления сообщений в Telegram боте. Функция `process_dell_text_msg` получает идентификатор последнего сообщения из состояния и пытается удалить его и сообщение пользователя. Если идентификатор последнего сообщения не найден, в лог записывается предупреждение. В случае возникновения ошибки при удалении сообщения, информация об ошибке записывается в лог.

## Функции

### `process_dell_text_msg`

```python
async def process_dell_text_msg(message: Message, state: FSMContext):
    """
    Args:
        message (Message): Объект сообщения Telegram.
        state (FSMContext): Объект состояния FSM.

    Returns:
        None

    Raises:
        Exception: Если происходит ошибка при удалении сообщения.

    """
```

**Как работает функция**:

Функция `process_dell_text_msg` асинхронно обрабатывает сообщение об удалении текста. Сначала она получает данные из состояния FSM, чтобы получить `last_msg_id` (идентификатор последнего сообщения). Если `last_msg_id` существует, функция пытается удалить сообщение с этим ID из чата пользователя. Затем она удаляет сообщение пользователя, вызвавшее функцию. Если `last_msg_id` не найден в состоянии, функция логирует предупреждение. Если во время удаления сообщения возникает исключение, функция логирует ошибку.

**Параметры**:

- `message` (Message): Объект сообщения Telegram, содержащий информацию о сообщении, которое нужно обработать.
- `state` (FSMContext): Объект состояния FSM, содержащий данные о текущем состоянии пользователя.

**Возвращает**:

- `None`: Функция ничего не возвращает.

**Вызывает исключения**:

- `Exception`: Если происходит ошибка при удалении сообщения, информация об ошибке логируется.

**Примеры**:

Пример вызова функции:

```python
from aiogram import types
from aiogram.fsm.context import FSMContext

# Создаем объект Message (пример)
message = types.Message(message_id=123, from_user=types.User(id=456), chat=types.Chat(id=456), date=None, content_type=None)

# Создаем объект FSMContext (пример)
state = FSMContext(bot=None, chat_id=456, user_id=456)

# Вызываем функцию
await process_dell_text_msg(message, state)
