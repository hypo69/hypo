# Модуль hypotez/src/product/product_fields/product_fields.py

## Обзор

Данный модуль содержит класс `ProductFields`, предназначенный для работы с полями товаров в базе данных PrestaShop. Класс предоставляет методы для доступа и изменения различных полей товара, включая основные (id_product, цена, название и др.) и дополнительные поля, а также мультиязычные поля.  Модуль использует библиотеку `langdetect` для определения языка текста. Он обеспечивает структурированный и удобный способ работы с информацией о товарах, в том числе с учетом специфики базы данных PrestaShop.

## Классы

### `ProductFields`

**Описание**: Класс `ProductFields` предоставляет методы для работы с полями товаров в базе данных PrestaShop. Он хранит список полей, данные по умолчанию, и информацию о языках.

**Атрибуты**:
- `product_fields_list`: Список названий полей товаров из файла `fields_list.txt`.
- `language`: Словарь, сопоставляющий язык с его идентификатором в PrestaShop.
- `presta_fields`: Объект `SimpleNamespace` для хранения загруженных полей товаров.
- `assist_fields_dict`: Словарь для хранения вспомогательных данных, например, URL изображений.

**Методы**:

- `__init__(self)`: Инициализирует класс. Загружает список полей из файла `fields_list.txt`, значения полей по умолчанию из `product_fields_default_values.json` и заполняет атрибуты `product_fields_list`, `language`, `presta_fields`, `assist_fields_dict`.
- `_load_product_fields_list(self)`: Загружает список полей из файла `fields_list.txt`. Возвращает список строк.
- `_payload(self)`: Загружает значения полей по умолчанию из файла `product_fields_default_values.json`. Возвращает `True`, если загрузка прошла успешно, иначе `False`.

**Свойства (property)**:

Каждое поле товара представлено свойством (с `@property` и `@setter`) для доступа и изменения. Примеры:


- `id_product`: Возвращает/устанавливает идентификатор товара.
- `id_supplier`: Возвращает/устанавливает идентификатор поставщика.
- ... (все другие свойства)

**Обработка исключений**:
Каждый метод, изменяющий поля товара, обрабатывает исключение `ProductFieldException` с использованием `logger` для ведения журнала ошибок.
