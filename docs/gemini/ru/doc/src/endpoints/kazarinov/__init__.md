# Модуль hypotez/src/endpoints/kazarinov/__init__.py

## Обзор

Этот модуль содержит константы и импорты для работы с ботом KazarinovTelegramBot.

## Константы

### `MODE`

**Описание**:  Строковая константа, определяющая режим работы. В данном случае, `'dev'`.

## Импорты

### `KazarinovTelegramBot`

**Описание**: Импортирует класс `KazarinovTelegramBot` из файла `kazarinov_bot.py` в этом же каталоге.
