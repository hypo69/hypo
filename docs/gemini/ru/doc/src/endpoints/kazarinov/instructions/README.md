# ИНСТРУКЦИЯ ПО ИСПОЛЬЗОВАНИЮ ФАЙЛОВ

## Обзор

Этот директория содержит инструкции и системные подсказки для ИИ-модели.  Она включает два файла: `system_instruction_mexiron.md` и `command_instruction_mexiron.md`.


## Файлы

### `system_instruction_mexiron.md`

**Описание**: Этот файл содержит системную инструкцию, которая загружается во время инициализации модели и описывает поведение инициализированной модели.  Он определяет контекст и ожидания для модели.


### `command_instruction_mexiron.md`

**Описание**: Этот файл содержит инструкцию, которая является кратким руководством для модели, описывающим шаблон для ответа модели.  Он обеспечивает структуру и формат, в котором модель должна формировать свой ответ.


## Примеры использования

В каждом из этих файлов ожидается формализованный и структурированный текст, описывающий работу соответствующего файла.
