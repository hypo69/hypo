# Модуль process_data.py

## Обзор

Модуль `process_data.py` содержит константы и импорты, необходимые для обработки данных в контексте Fast API и HTML.  Он предназначен для предварительной обработки данных перед их использованием в других частях приложения.

## Константы

### `MODE`

**Описание**:  Переменная, хранящая режим работы приложения (`dev`).

**Значение**: `'dev'`


## Импорты

### `from .. import main`

**Описание**: Импортирует модуль `main` из родительского каталога.

### `from main import process_dataa`

**Описание**: Импортирует функцию `process_dataa` из модуля `main`.
