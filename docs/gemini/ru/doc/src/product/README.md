# product Module: Product Management

## Обзор

Модуль `product` отвечает за управление всеми аспектами данных продукта, включая обработку, валидацию и управление полями. Он состоит из следующих компонентов:

1. **product.py** — Управляет основной логикой обработки продуктов, включая операции по созданию, обновлению и удалению записей о продуктах. Предоставляет функции для обработки данных продуктов и обеспечивает соответствие бизнес-правилам для управления продуктами в приложении.

2. **product_fields.py** — Управляет логикой, связанной с полями продукта, обрабатывая валидацию полей, форматирование и управление. Этот модуль гарантирует, что поля продукта соответствуют необходимым критериям для последовательного ввода данных, что позволяет точно и эффективно обрабатывать информацию о продукте.

## product.py

### product.py

**Описание**: Этот файл содержит функции для управления продуктами.

**Функции**:

### `create_product`

**Описание**: Создает новую запись о продукте.

**Параметры**:
- `product_data` (dict): Словарь с данными о продукте.

**Возвращает**:
- `dict | None`: Словарь с данными о созданном продукте или `None` при ошибке.


**Вызывает исключения**:
- `ValueError`: Если данные продукта некорректны.
- `Exception`: При других ошибках.


### `update_product`

**Описание**: Обновляет существующую запись о продукте.

**Параметры**:
- `product_id` (int): Идентификатор продукта.
- `updated_data` (dict): Словарь с обновлёнными данными.


**Возвращает**:
- `bool`: `True`, если обновление прошло успешно, `False` в противном случае.


**Вызывает исключения**:
- `ValueError`: Если ID продукта некорректен или данные некорректны.
- `Exception`: При других ошибках.


### `delete_product`

**Описание**: Удаляет запись о продукте.

**Параметры**:
- `product_id` (int): Идентификатор продукта.


**Возвращает**:
- `bool`: `True`, если удаление прошло успешно, `False` в противном случае.


**Вызывает исключения**:
- `ValueError`: Если ID продукта некорректен.
- `Exception`: При других ошибках.
