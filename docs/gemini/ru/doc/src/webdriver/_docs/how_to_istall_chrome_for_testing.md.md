## Установка Chrome на Windows

Для установки Chrome на Windows следуйте этим шагам:

1. **Загрузка установщика:**
   - Перейдите на официальную страницу загрузки Chrome: [https://www.google.com/chrome/](https://www.google.com/chrome/).
   - Нажмите кнопку "Скачать Chrome".
   - Сохраните установочный файл `ChromeSetup.exe` в удобное для вас место.

2. **Запуск установщика:**
   - Запустите скачанный файл `ChromeSetup.exe`.
   - Следуйте инструкциям установщика, которые появятся на экране.
   - Установка, как правило, выполняется в автоматическом режиме, без необходимости дополнительных настроек.

3. **Завершение установки:**
    - После завершения установки, Chrome должен автоматически запуститься. Если этого не произошло, вы можете запустить его через ярлык на рабочем столе или в меню "Пуск".

## Установка Chrome на macOS

Для установки Chrome на macOS выполните следующие шаги:

1. **Загрузка DMG-файла:**
   - Откройте браузер Safari (или любой другой) и перейдите на страницу загрузки Chrome: [https://www.google.com/chrome/](https://www.google.com/chrome/).
   - Нажмите кнопку "Скачать Chrome".
   - Сохраните DMG-файл в папку "Загрузки" или другое удобное место.

2. **Монтирование образа:**
   - Откройте скачанный DMG-файл.
   - В открывшемся окне перетащите значок Chrome в папку "Программы".
   
3. **Запуск Chrome:**
   - Найдите Chrome в папке "Программы" и запустите его.
   - При первом запуске macOS может запросить разрешение на открытие приложения из неизвестного источника. Подтвердите, чтобы запустить Chrome.

## Установка Chrome на Linux

Установка Chrome на Linux может несколько отличаться в зависимости от дистрибутива. Вот инструкция для наиболее распространённых дистрибутивов:

### Debian/Ubuntu

1. **Загрузка DEB-пакета:**
   - Перейдите на страницу загрузки Chrome: [https://www.google.com/chrome/](https://www.google.com/chrome/).
   - Нажмите кнопку "Скачать Chrome".
   - Выберите DEB-пакет (для Debian/Ubuntu) и сохраните его.

2. **Установка пакета:**
    - Откройте терминал.
    - Перейдите в директорию, куда вы скачали DEB-файл.
    - Выполните команду для установки (возможно, потребуется `sudo`):
    ```bash
    sudo apt install ./google-chrome-stable_current_amd64.deb
    ```
    (Замените `google-chrome-stable_current_amd64.deb` на имя вашего файла).

3. **Завершение установки:**
   - После установки Chrome можно запустить из меню приложений или через команду в терминале `google-chrome`.

### Fedora/CentOS/RHEL

1. **Загрузка RPM-пакета:**
   - Перейдите на страницу загрузки Chrome: [https://www.google.com/chrome/](https://www.google.com/chrome/).
   - Нажмите кнопку "Скачать Chrome".
   - Выберите RPM-пакет (для Fedora/CentOS/RHEL) и сохраните его.

2. **Установка пакета:**
    - Откройте терминал.
    - Перейдите в директорию, куда вы скачали RPM-файл.
    - Выполните команду для установки (возможно, потребуется `sudo`):
    ```bash
    sudo rpm -ivh google-chrome-stable_current_x86_64.rpm
    ```
   (Замените `google-chrome-stable_current_x86_64.rpm` на имя вашего файла).

3. **Завершение установки:**
   - После установки Chrome можно запустить из меню приложений или через команду в терминале `google-chrome`.

## Проверка установки

Для проверки корректности установки Chrome выполните следующие действия:

1. Запустите браузер Chrome.
2. Введите в адресной строке `chrome://version/` и нажмите Enter.
3. Убедитесь, что информация о версии Chrome отображается корректно.
4. Если Chrome открывается и показывает информацию о версии, то установка прошла успешно.

Теперь вы можете использовать установленный Chrome для запуска автоматизированных тестов через Selenium WebDriver.
