# Модуль `header`

## Обзор

Модуль `header` определяет корневой путь к проекту, загружает настройки из `settings.json` и информацию из `README.MD`, а также предоставляет метаданные проекта.

## Функции

### `set_project_root`

**Описание**:
Находит корневой каталог проекта, начиная с каталога текущего файла, ищет вверх и останавливается в первом каталоге, содержащем любой из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple): Кортеж с именами файлов или каталогов для идентификации корневого каталога проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневому каталогу, если он найден, иначе каталог, где находится скрипт.

## Переменные

### `__root__`

**Описание**:
Путь к корневому каталогу проекта. Определяется функцией `set_project_root`.

### `settings`

**Описание**:
Словарь с настройками проекта, загруженный из файла `settings.json`.

### `doc_str`

**Описание**:
Строка с содержимым файла `README.MD`.

### `__project_name__`

**Описание**:
Имя проекта. По умолчанию 'hypotez', если не указано в `settings.json`.

### `__version__`

**Описание**:
Версия проекта. По умолчанию пустая строка, если не указано в `settings.json`.

### `__doc__`

**Описание**:
Документация проекта, загруженная из файла `README.MD`.

### `__details__`

**Описание**:
Детальная информация о проекте. В настоящее время не используется и является пустой строкой.

### `__author__`

**Описание**:
Автор проекта. По умолчанию пустая строка, если не указано в `settings.json`.

### `__copyright__`

**Описание**:
Информация об авторских правах. По умолчанию пустая строка, если не указано в `settings.json`.

### `__cofee__`

**Описание**:
Строка с информацией о возможности угостить разработчика кофе. Значение по умолчанию: "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69". Загружается из файла `settings.json` если присутствует.
