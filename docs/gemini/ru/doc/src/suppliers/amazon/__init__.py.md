# Модуль `src.suppliers.amazon`

## Обзор

Модуль `src.suppliers.amazon` предназначен для работы с данными от поставщика Amazon. Он включает в себя функциональность для сбора данных (граббинга) и выполнения сценариев, таких как получение списка товаров в категории.
