## Введение

Класс `ExecuteLocator` предназначен для работы с веб-элементами через Selenium WebDriver. Он предоставляет методы для поиска элементов по локаторам, получения атрибутов и отправки сообщений.  В этом руководстве описаны шаги по тестированию этих методов.


## Подготовка окружения

### Установка зависимостей

Убедитесь, что установлены все необходимые библиотеки:

```bash
pip install -r requirements.txt
```

Файл `requirements.txt` должен содержать следующие зависимости:

```text
pytest==7.4.0
selenium==4.16.1
```

### Настройка WebDriver

Установите WebDriver для браузера, который вы будете использовать для тестирования (например, ChromeDriver для Chrome).


## Написание тестов

### Структура тестов

Создайте файл `test_executor.py` в директории `tests`.  Примеры тестов для класса `ExecuteLocator` приведены в input_code.

### Реализация тестов

Ниже представлены примеры реализации тестов для методов класса `ExecuteLocator`.  Полные примеры тестов находятся в input_code.

#### `test_get_webelement_by_locator_single_element`

**Описание**: Тест проверяет, что метод `get_webelement_by_locator` возвращает правильный элемент, когда по локатору найден один элемент.

#### `test_get_webelement_by_locator_multiple_elements`

**Описание**: Тест проверяет, что метод `get_webelement_by_locator` возвращает список элементов, когда по локатору найдено несколько элементов.

#### `test_get_webelement_by_locator_no_element`

**Описание**: Тест проверяет, что метод `get_webelement_by_locator` возвращает `False`, когда по локатору не найдено элементов.

#### `test_get_attribute_by_locator`

**Описание**: Тест проверяет, что метод `get_attribute_by_locator` возвращает значение атрибута элемента.

#### `test_send_message`

**Описание**: Тест проверяет, что метод `send_message` отправляет сообщение на элемент, используя `send_keys`.

#### `test_send_message_typing_speed`

**Описание**: Тест проверяет, что метод `send_message` отправляет сообщение с заданной скоростью набора текста.

## Запуск тестов

Для запуска тестов выполните команду:

```bash
pytest tests/test_executor.py
```

## Проверка результатов тестирования

После запуска тестов `pytest` выведет результаты в терминале. Убедитесь, что все тесты прошли успешно.

## Обновление тестов

По мере изменений в коде класса `ExecuteLocator`, обновляйте тесты, чтобы они проверяли все новые или измененные функции.
