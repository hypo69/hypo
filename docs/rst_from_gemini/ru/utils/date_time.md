# Модуль `date_time.py`

Файл: `C:\Users\user\Documents\repos\hypotez\src\utils\date_time.py`

## Описание

Данный модуль предоставляет функцию для проверки, находится ли текущее время в заданном интервале, с опцией таймаута. Это полезно для запуска операций, которые должны выполняться только в определенные временные промежутки (например, ночные задачи обслуживания).  Модуль также содержит функции для получения ввода от пользователя с таймаутом.

## Функции

### `interval(self, start: time = time(23, 0), end: time = time(6, 0)) -> bool`

Проверяет, находится ли текущее время в заданном интервале.

* **Аргументы:**
    * `start` (time): Начало интервала (по умолчанию 23:00).
    * `end` (time): Конец интервала (по умолчанию 06:00).
* **Возвращаемое значение:**
    * `bool`: `True`, если текущее время находится в интервале, `False` в противном случае.


### `interval_with_timeout(self, timeout: int = 5, start: time = time(23, 0), end: time = time(6, 0)) -> bool`

Проверяет, находится ли текущее время в заданном интервале с таймаутом.

* **Аргументы:**
    * `timeout` (int): Время в секундах, которое следует ждать проверки интервала.
    * `start` (time): Начало интервала (по умолчанию 23:00).
    * `end` (time): Конец интервала (по умолчанию 06:00).
* **Возвращаемое значение:**
    * `bool`: `True`, если текущее время находится в интервале и проверка завершилась успешно до таймаута, `False` в противном случае (если таймаут истек или время не в интервале).


### `get_input(self)`

Запрашивает ввод от пользователя.

* **Возвращаемое значение:**
    *  Метод не возвращает значения, но изменяет внутреннее свойство `user_input`.


### `input_with_timeout(self, timeout: int = 5) -> str | None`

Ожидает ввода от пользователя с таймаутом.

* **Аргументы:**
    * `timeout` (int): Время ожидания ввода в секундах.
* **Возвращаемое значение:**
    * `str`: Введенные данные.
    * `None`: Если таймаут истек.


## Пример использования (в блоке `if __name__ == '__main__':`)

Этот пример демонстрирует, как использовать функцию `interval_with_timeout`:

```python
timeout_check = TimeoutCheck()
if timeout_check.interval_with_timeout(timeout=5):
    print("Current time is within the interval.")
else:
    print("Current time is outside the interval or timeout occurred.")
```
