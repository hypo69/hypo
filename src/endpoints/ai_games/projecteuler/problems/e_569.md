# Задача No 569
## Рабочие дни
## Условия
На планете Берляндия есть N различных типов рабочих дней. За каждым рабочим днем закреплено целое число от 1 до N.
В каждый из дней рабочего дня одного типа каждый из М работников выполняет одну единицу работы.

В Берляндии приняты интересные законы относительно трудовых будней:
* В течение одного рабочего дня каждого типа, все работники работают одновременно.
* По окончании каждого рабочего дня все работники обязаны отдохнуть.
* Время отдыха длится от одного до N дней включительно.
* В течение времени отдыха никакая работа не производится.
* Как только время отдыха заканчивается, работники возвращаются к работе на один рабочий день.
* Рабочие циклы могут начинаться в любой день.

Пусть W(N, M, K) будет количеством единиц работы, выполненных всеми M работниками за K дней.
Например, W(3, 4, 10) = 28, так как наилучшим графиком работ будет последовательность дней работы типов 1, 2, 3, а затем период отдыха 1 день, затем снова рабочий день типа 1, итого 4 рабочих дня. За 4 дня работы каждый из 4 работников сделает 4 единицы работы, поэтому в итоге 4 * 4 = 16 единиц работы. Последовательность типов рабочих дней 1, 1, 1, 1 дает в итоге 16 единиц работы за 10 дней. Но 1, 2, 3, 1, 2, 1, 2 дает 7 дней работы, что равно 7 * 4 = 28 единиц работы, а еще остается 3 дня для отдыха. 1, 2, 3, 1, 2, 3, 1 - 7 рабочих дней, 28 единиц работы, остается 3 дня - нельзя сделать больше.
Похожая схема для 1, 2, 3, 1, 2, 3, 1 дает 7 рабочих дней, что равно 7*4=28.
Из всех рабочих схем 1, 2, 3, 1, 2, 1, 2 является оптимальной для 10 дней.

Определим S(N, M, L) как сумму всех W(N, M, K) для 1 ≤ K ≤ L.

Например, S(3, 4, 10) = 28 + 56 + 84 + 112 + 140 + 168 + 196 + 224 + 252 + 280 = 1540.

Найдите S(10, 10000, 10<sup>15</sup>) mod 1000000007.
