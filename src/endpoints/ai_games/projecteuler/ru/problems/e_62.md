# Задача No 62
## Кубические перестановки
## Условия
Кубическое число 41063625 (345<sup>3</sup>) может быть переставлено, чтобы сформировать два других кубических числа: 56623104 (384<sup>3</sup>) и 66430125 (405<sup>3</sup>). Фактически, 41063625 - это наименьшее кубическое число, для которого существует ровно три перестановки кубов.

Найдите наименьшее кубическое число, перестановки которого образуют ровно пять кубов.
