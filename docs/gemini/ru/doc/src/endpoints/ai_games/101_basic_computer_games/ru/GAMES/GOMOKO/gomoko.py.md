# Гомоку

## Обзор

Данный модуль реализует игру "Гомоку" (также известную как "Пять в ряд") для двух игроков, где один игрок - человек, а другой - компьютер. Игра ведется на доске 15x15. Цель игры - первым выстроить непрерывную линию из пяти своих фишек по вертикали, горизонтали или диагонали.

## Функции

### `print_board`

**Описание**: Выводит текущее состояние игрового поля в консоль с нумерацией строк и столбцов.

**Параметры**:

- `board` (list): Двумерный список (матрица), представляющий игровое поле.

**Возвращает**:
- `None`: Функция ничего не возвращает, а только выводит данные в консоль.

### `check_winner`

**Описание**: Проверяет, есть ли победитель на игровом поле.

**Параметры**:

- `board` (list): Двумерный список (матрица), представляющий игровое поле.
- `player` (int): Номер текущего игрока (1 или 2).

**Возвращает**:

- `bool`: `True`, если есть победитель, иначе `False`.

### `check_full_board`

**Описание**: Проверяет, заполнено ли игровое поле.

**Параметры**:

- `board` (list): Двумерный список (матрица), представляющий игровое поле.

**Возвращает**:

- `bool`: `True`, если игровое поле полностью заполнено, иначе `False`.
