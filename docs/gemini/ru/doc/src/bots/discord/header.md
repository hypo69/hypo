# Модуль hypotez/src/bots/discord/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла settings.json. Он также предоставляет доступ к информации о проекте, такой как название, версия, документация и прочее.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с текущего файла. Поиск происходит вверх по директориям до тех пор, пока не будет найдена директория, содержащая один из указанных маркеров (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, используемых для определения корневой директории проекта. По умолчанию (`'pyproject.toml'`, `'requirements.txt'`, `.git`).

**Возвращает**:

- `Path`: Объект Path, представляющий корневую директорию проекта. Если корневая директория не найдена, возвращает директорию, в которой находится текущий файл.

**Возможные исключения**:

- Нет

## Переменные

### `__root__`

**Описание**: Переменная, содержащая путь к корневой директории проекта. Инициализируется вызовом функции `set_project_root()`.

**Тип**: `Path`

## Загрузка настроек

**Описание**: Модуль загружает настройки из файла `settings.json`, расположенного в корневой директории проекта.

**Возможные исключения**:

- `FileNotFoundError`: Если файл `settings.json` не найден.
- `json.JSONDecodeError`: Если содержимое файла `settings.json` не является валидным JSON.

## Загрузка документации

**Описание**: Модуль загружает текст документации из файла `README.MD`, расположенного в корневой директории проекта.

**Возможные исключения**:

- `FileNotFoundError`: Если файл `README.MD` не найден.
- `json.JSONDecodeError`: Если содержимое файла `README.MD` не является валидным текстом.

## Переменные проекта

### `__project_name__`

**Описание**: Название проекта. Значение берется из `settings.json` или устанавливается как `'hypotez'`, если значение не найдено.

**Тип**: `str`

### `__version__`

**Описание**: Версия проекта. Значение берется из `settings.json` или устанавливается как пустая строка, если значение не найдено.

**Тип**: `str`

### `__doc__`

**Описание**: Документация проекта. Значение берется из `README.MD` или устанавливается как пустая строка, если значение не найдено.

**Тип**: `str`

### `__details__`

**Описание**: Подробная информация о проекте. По умолчанию пустая строка.

**Тип**: `str`

### `__author__`

**Описание**: Автор проекта. Значение берется из `settings.json` или устанавливается как пустая строка, если значение не найдено.

**Тип**: `str`

### `__copyright__`

**Описание**: Авторские права проекта. Значение берется из `settings.json` или устанавливается как пустая строка, если значение не найдено.

**Тип**: `str`

### `__cofee__`

**Описание**: Ссылка для поддержки автора. Значение берется из `settings.json` или устанавливается как значение по умолчанию, если значение не найдено.

**Тип**: `str`
