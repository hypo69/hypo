# Модуль `src.ai.openai.bully`

## Обзор

Этот модуль предназначен для демонстрации того, как можно использовать OpenAI для генерации примеров грубого поведения с точки зрения агрессора. Он использует API OpenAI для создания ответов, имитирующих стиль и тон агрессивного человека на основе предоставленного контекста.

## Константы

### `MODE`
- **Описание**: Режим работы модуля, в данном случае установлен как `dev`.
- **Тип**: `str`

### `system_prompt`
- **Описание**: Системный запрос, задающий роль и задачу для модели OpenAI. Просит модель ответить в стиле агрессора, используя его точку зрения и отвечать в структурированном JSON с ключом "bully_response".
- **Тип**: `str`

## Функции

### `bully`
   
**Описание**: Функция отправляет запрос к API OpenAI для получения ответа в стиле агрессора.

**Параметры**:
- `user_message` (str, optional): Сообщение пользователя, которое будет использовано для запроса ответа от модели. По умолчанию `"Hello!"`.
- `messages` (list, optional): Список сообщений, включая системное сообщение, используемый для контекста в запросе к модели. По умолчанию `[{"system": "user", "content": system_prompt}]`.

**Возвращает**:
- `list`: Обновленный список сообщений, включающий ответ от модели.

**Пример использования**:
```python
messages = [{"role": "system", "content": system_prompt}]
user_message = "ты ничтожество"
response = bully(user_message=user_message, messages=messages)
print(response)
```
