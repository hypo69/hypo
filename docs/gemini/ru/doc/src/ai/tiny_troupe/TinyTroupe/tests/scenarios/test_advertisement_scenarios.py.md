# `test_advertisement_scenarios.py`

## Обзор

Этот файл содержит тесты для проверки различных сценариев, связанных с рекламой и потребительским поведением. Включает тесты для оценки рекламных объявлений, создания рекламных текстов и профилирования потребителей.

## Функции

### `test_ad_evaluation_scenario`

**Описание**: Тестирует сценарий оценки рекламных объявлений, где агенты выбирают наиболее убедительное объявление и объясняют свой выбор.

**Параметры**:
- `setup`: Фикстура pytest для настройки окружения тестирования.

**Возвращает**:
- `None`

**Вызывает исключения**:
- `AssertionError`: Возникает, если не удается извлечь результаты, отсутствует поле `ad_id` или `ad_title`, или значение `ad_id` не является одним из допустимых вариантов.

### `test_ad_creation_scenario`

**Описание**: Тестирует сценарий создания рекламного текста для сдачи квартиры в аренду с участием фокус-группы.

**Параметры**:
- `setup`: Фикстура pytest для настройки окружения тестирования.
- `focus_group_world`: Фикстура pytest для имитации фокус-группы.

**Возвращает**:
- `None`

**Вызывает исключения**:
- `AssertionError`: Возникает, если сгенерированный текст не содержит идей для рекламы квартиры.

### `test_consumer_profiling_scenario`

**Описание**: Тестирует сценарий профилирования потребителей с использованием кэширования для сохранения данных между сессиями.

**Параметры**:
- `setup`: Фикстура pytest для настройки окружения тестирования.

**Возвращает**:
- `None`

**Вызывает исключения**:
- `AssertionError`: Возникает, если файл кэша не был создан.
