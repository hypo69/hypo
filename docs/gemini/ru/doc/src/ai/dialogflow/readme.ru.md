# Модуль dialogflow

## Обзор

Данный модуль предоставляет интеграцию с Google Dialogflow для обработки естественного языка и создания разговорных ИИ-приложений. Он позволяет определять намерения пользователей, извлекать сущности, управлять контекстами и интегрироваться с различными платформами.  Модуль также поддерживает Webhook-интеграции для вызова внешних сервисов.

## Классы

### Класс `Dialogflow`

**Описание**: Класс `Dialogflow` предоставляет методы для взаимодействия с сервисом Dialogflow.

**Методы**:

- `detect_intent(text: str) -> dict | None`:
    **Описание**: Обнаруживает намерение пользователя на основе входного текста.
    **Параметры**:
    - `text` (str): Текст, введенный пользователем.
    **Возвращает**:
    - `dict | None`: Словарь с результатом распознавания намерения или `None`, если произошла ошибка.
    **Вызывает исключения**:
    - `ValueError`: Если входной параметр `text` имеет некорректный тип.
    - `Exception`: Общие ошибки.

- `list_intents() -> list | None`:
    **Описание**: Возвращает список доступных намерений.
    **Параметры**:
    -  нет
    **Возвращает**:
    - `list | None`: Список намерений или `None`, если произошла ошибка.
    **Вызывает исключения**:
    - `Exception`: Общие ошибки.

- `create_intent(display_name: str, training_phrases_parts: list, message_texts: list) -> dict | None`:
    **Описание**: Создает новое намерение в Dialogflow.
    **Параметры**:
    - `display_name` (str): Имя намерения для отображения.
    - `training_phrases_parts` (list): Список фрагментов фраз для обучения намерения.
    - `message_texts` (list): Список текстов ответов для намерения.
    **Возвращает**:
    - `dict | None`: Словарь с информацией о созданном намерении или `None`, если произошла ошибка.
    **Вызывает исключения**:
    - `TypeError`: Если типы входных параметров не соответствуют ожидаемым.
    - `ValueError`: Если входные параметры имеют некорректные значения.
    - `Exception`: Общие ошибки.

- `delete_intent(intent_id: str) -> bool | None`:
    **Описание**: Удаляет намерение с указанным `intent_id`.
    **Параметры**:
    - `intent_id` (str): Уникальный идентификатор намерения.
    **Возвращает**:
    - `bool | None`: `True`, если намерение было удалено, `False` если возникла ошибка. `None`, если произошла ошибка.
    **Вызывает исключения**:
    - `ValueError`: Если `intent_id` имеет некорректный формат.
    - `Exception`: Общие ошибки.
