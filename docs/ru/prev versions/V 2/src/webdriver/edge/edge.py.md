# Модуль `edge`

## Обзор

Модуль `edge` предоставляет пользовательский класс `Edge` для управления браузером Microsoft Edge с использованием Selenium WebDriver. Класс включает в себя дополнительные функции, такие как управление пользовательским агентом, профилем пользователя и различными режимами окна, а также предоставляет методы для выполнения JavaScript и поиска элементов на странице.

## Классы

### `Edge`

**Описание**: Пользовательский класс WebDriver для браузера Microsoft Edge с расширенным функционалом.

**Атрибуты**:
- `driver_name` (str): Имя используемого WebDriver, по умолчанию `'edge'`.

**Методы**:
- [`__init__`](#__init__): Инициализирует экземпляр `Edge` с заданными параметрами.
- [`_payload`](#_payload): Загружает исполнителей для локаторов и JavaScript-сценариев.
- [`set_options`](#set_options): Создает и настраивает параметры запуска для Edge WebDriver.

## Функции

### `__init__`

**Описание**: Инициализирует WebDriver Edge с указанным пользовательским агентом и опциями.

**Параметры**:
- `profile_name` (Optional[str], optional): Имя профиля, если требуется использовать другой профиль. По умолчанию `None`.
- `user_agent` (Optional[str], optional): Строка user-agent для использования. Если `None`, генерируется случайный user-agent. По умолчанию `None`.
- `options` (Optional[List[str]], optional): Список опций Edge, передаваемых при инициализации. По умолчанию `None`.
- `window_mode` (Optional[str], optional): Режим окна браузера (`windowless`, `kiosk`, `full_window` и т.д.). По умолчанию `None`.

**Возвращает**:
- `None`: Функция ничего не возвращает.

**Вызывает исключения**:
- `WebDriverException`: Вызывается в случае ошибки при запуске WebDriver.
- `Exception`: Вызывается в случае общей ошибки.

### `_payload`

**Описание**: Загружает исполнителей для локаторов и JavaScript-сценариев.

**Параметры**:
- Отсутствуют

**Возвращает**:
- `None`: Функция ничего не возвращает.

### `set_options`

**Описание**: Создает и настраивает параметры запуска для Edge WebDriver.

**Параметры**:
- `opts` (Optional[List[str]], optional): Список опций для добавления в Edge WebDriver. По умолчанию `None`.

**Возвращает**:
- `EdgeOptions`: Конфигурированный объект `EdgeOptions`.
