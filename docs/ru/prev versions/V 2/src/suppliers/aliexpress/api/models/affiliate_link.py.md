# Модуль `affiliate_link`

## Обзор

Модуль `affiliate_link` содержит класс `AffiliateLink`, представляющий партнерскую ссылку.

## Содержание

    - [`AffiliateLink`](#AffiliateLink)

## Классы

### `AffiliateLink`

**Описание**: Класс `AffiliateLink` представляет модель партнерской ссылки, содержащую ссылку на продвижение и источник значения.

**Атрибуты**:
- `promotion_link` (str): Ссылка на продвижение.
- `source_value` (str): Источник значения.
