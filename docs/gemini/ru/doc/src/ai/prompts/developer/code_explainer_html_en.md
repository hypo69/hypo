## Обзор

Этот файл содержит описание функции `add_numbers`, которая складывает два числа, используя функцию `calculate_sum` из модуля `src.utils.calculator`.

## Функции

### `add_numbers`

**Описание**: Функция `add_numbers` принимает два числа в качестве аргументов и возвращает их сумму, используя функцию `calculate_sum`.

**Параметры**:

- `a` (число): Первое слагаемое.
- `b` (число): Второе слагаемое.

**Возвращает**:

- `число`: Сумма чисел `a` и `b`.

**Вызывает исключения**:

Возможные исключения, которые могут быть подняты функцией `calculate_sum`, будут перехвачены и повторно выброшены этой функцией.  Полное описание исключений находится в документации для модуля `src.utils.calculator`.


```python
from src.utils.calculator import calculate_sum

def add_numbers(a: float, b: float) -> float:
    """
    Args:
        a (float): Первое слагаемое.
        b (float): Второе слагаемое.

    Returns:
        float: Сумма чисел a и b.

    Raises:
        TypeError: Если в качестве аргументов переданы не числа.
        ValueError: Если аргументы не являются числовыми значениями.
        Exception: В случае других ошибок в calculate_sum.
    """
    try:
        result = calculate_sum(a, b)
        return result
    except TypeError as ex:
        raise TypeError(f"Ошибка: Оба аргумента должны быть числами. {ex}")
    except ValueError as ex:
        raise ValueError(f"Ошибка: Аргументы должны быть числовыми значениями. {ex}")
    except Exception as ex:
        raise Exception(f"Произошла ошибка: {ex}")
```
