# Модуль `header`

## Обзор

Модуль `header.py` предназначен для определения основных настроек и метаданных проекта `hypotez`. Он отвечает за установку корневой директории проекта, загрузку настроек из файла `settings.json` и чтение документации из файла `README.MD`. Также модуль определяет глобальные переменные, содержащие имя проекта, версию, автора и другие метаданные.

## Подробней

Модуль выполняет следующие задачи:

1.  **Определение корневой директории проекта**: Функция `set_project_root` ищет корневую директорию проекта, начиная с текущей директории файла и двигаясь вверх по дереву каталогов. Поиск прекращается, когда найден каталог, содержащий один из маркеров (файлы или каталоги, такие как `__root__` или `.git`). Корневая директория используется для определения путей к другим файлам проекта.

2.  **Загрузка настроек из файла `settings.json`**: Модуль пытается загрузить настройки из файла `settings.json`, расположенного в корневой директории проекта. Если файл найден и успешно прочитан, его содержимое сохраняется в глобальной переменной `settings`.

3.  **Чтение документации из файла `README.MD`**: Модуль пытается прочитать содержимое файла `README.MD`, расположенного в корневой директории проекта. Если файл найден и успешно прочитан, его содержимое сохраняется в глобальной переменной `doc_str`.

4.  **Определение глобальных переменных**: На основе загруженных настроек и документации модуль определяет глобальные переменные, такие как `__project_name__`, `__version__`, `__doc__`, `__author__`, `__copyright__` и `__cofee__`. Эти переменные содержат метаданные проекта, которые могут быть использованы в других модулях.

## Функции

### `set_project_root`

```python
def set_project_root(marker_files: tuple = ('__root__', '.git')) -> Path:
    """
    Args:
        marker_files (tuple): Filenames or directory names to identify the project root.
    
    Returns:
        Path: Path to the root directory if found, otherwise the directory where the script is located.
    """
```

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла, и останавливается на первой директории, содержащей любой из файлов-маркеров.

**Параметры**:

*   `marker_files` (tuple): Кортеж имен файлов или каталогов для идентификации корня проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:

*   `Path`: Путь к корневой директории, если она найдена, иначе директория, где расположен скрипт.

**Примеры**:

```python
from pathlib import Path
# Пример вызова функции set_project_root
root_path = set_project_root()
print(f"Корневая директория проекта: {root_path}")

# Пример вызова функции set_project_root с пользовательскими файлами-маркерами
root_path = set_project_root(marker_files=('my_marker_file', '.svn'))
print(f"Корневая директория проекта с пользовательскими маркерами: {root_path}")
