# Модуль `hypotez/src/gui/context_menu/qt6/main.py`

## Обзор

Модуль `hypotez/src/gui/context_menu/qt6/main.py` предназначен для добавления или удаления пункта контекстного меню на рабочем столе и в папках. Он использует Windows Registry для модификации контекстного меню в Windows Explorer. Модуль предоставляет функции для добавления и удаления пункта меню с именем "hypo AI assistant".

## Классы

### `ContextMenuManager`

**Описание**: Класс `ContextMenuManager` представляет собой основное окно приложения для управления контекстным меню.

**Методы**:

- `__init__(self)`: Инициализирует интерфейс пользователя с кнопками для добавления, удаления и выхода.
- `initUI(self)`: Инициализирует пользовательский интерфейс с кнопками для добавления, удаления пункта меню и выхода из приложения.


## Функции

### `add_context_menu_item()`

**Описание**: Добавляет пункт контекстного меню на рабочий стол и в папки. Создает ключ реестра в `HKEY_CLASSES_ROOT\Directory\Background\shell` для добавления пункта меню "hypo AI assistant".  Запуск скрипта при выборе пункта меню настраивается в реестре.

**Параметры**:
Нет.

**Возвращает**:
- None.

**Вызывает исключения**:
- Выводит диалоговое окно с ошибкой, если скрипт не найден. Возвращает `None`.


### `remove_context_menu_item()`

**Описание**: Удаляет пункт контекстного меню "hypo AI assistant". Удаляет соответствующий ключ реестра.

**Параметры**:
Нет.

**Возвращает**:
- None.

**Вызывает исключения**:
- Выводит диалоговое окно с предупреждением, если пункт меню не найден.
- Выводит диалоговое окно с ошибкой, если операция удаления завершилась с ошибкой.


## Константы

### `MODE`

**Описание**: Строковая константа, вероятно, задающая режим работы модуля. В данном случае, значение `'dev'`.
