# Модуль для работы со складами PrestaShop
## Обзор

Модуль `warehouse.py` предназначен для взаимодействия с API PrestaShop с целью управления складами. Он расширяет функциональность класса `PrestaShop`, предоставляя методы для работы с сущностями складов в PrestaShop.
## Подробней

Этот модуль предоставляет инструменты для получения данных о складах из PrestaShop, что может быть полезно для автоматизации процессов управления запасами и синхронизации данных между PrestaShop и другими системами. Модуль использует классы и функции из других модулей, таких как `header`, `src.gs`, `src.utils.printer` и `src.logger.logger`.
## Классы
### `PrestaWarehouse`
**Описание**: Класс `PrestaWarehouse` расширяет класс `PrestaShop` и предоставляет функциональность для работы со складами PrestaShop.

 **Наследует**:
   - `PrestaShop`: Класс, предоставляющий базовые методы для взаимодействия с API PrestaShop.

**Принцип работы**:
  1.  Класс инициализируется с использованием параметров подключения к PrestaShop API.
  2.  Использует методы класса `PrestaShop` для выполнения запросов к API PrestaShop и получения данных о складах.
  3.  Предоставляет методы для получения, создания, обновления и удаления складов в PrestaShop.
