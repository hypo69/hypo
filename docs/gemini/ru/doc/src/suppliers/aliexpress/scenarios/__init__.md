# Модуль hypotez/src/suppliers/aliexpress/scenarios/__init__.py

## Обзор

Данный модуль содержит сценарии для работы с поставщиком AliExpress.

## Переменные

### `MODE`

**Описание**: Переменная, определяющая режим работы (в данном случае 'dev').

## Модули

### `login`

**Описание**: Модуль, содержащий функции для входа в систему AliExpress.
