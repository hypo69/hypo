# Модуль hypotez/src/utils/autodoc.py

## Обзор

Модуль `autodoc` содержит декоратор `autodoc`, который автоматически обновляет строку документации функции, добавляя время последнего вызова функции.  Этот декоратор полезен для отслеживания активности функций.

## Декоратор `autodoc`

### `autodoc(func)`

**Описание**: Декоратор `autodoc` оборачивает функцию `func` и обновляет её docstring перед вызовом, добавляя строку с текущим временем.

**Параметры**:

- Не принимает параметры.

**Возвращает**:

- Обёрнутую функцию `func`.

**Вызывает исключения**:

- Не вызывает никаких исключений.

## Функция `update_docstring(func)`

### `update_docstring(func)`

**Описание**: Функция `update_docstring` обновляет строку документации функции `func`, добавляя время последнего вызова.

**Параметры**:

- `func` (callable): Функция, чья строка документации должна быть обновлена.

**Возвращает**:

- Не возвращает значения.

**Вызывает исключения**:

- Не вызывает никаких исключений.

## Функция `example_function`

### `example_function(param1: int, param2: str) -> None`

**Описание**: Пример функции, использующей декоратор `autodoc`. Каждый раз при вызове её docstring обновляется с информацией о времени последнего вызова.

**Параметры**:

- `param1` (int): Первое значение.
- `param2` (str): Второе значение.

**Возвращает**:

- `None`

**Вызывает исключения**:

- Не вызывает никаких исключений.


## Постоянные значения

### `MODE`

**Описание**: Переменная `MODE` хранит значение конфигурации.

**Значение**: `'dev'`

**Тип**: `str`

## Пример использования

```python
@autodoc
def example_function(param1: int, param2: str) -> None:
    """Пример функции.

    Args:
        param1 (int): Первое значение.
        param2 (str): Второе значение.
    """
    print(f"Processing {param1} and {param2}")

example_function(1, "test")
print(example_function.__doc__)  # Вывод обновленного docstring
example_function(2, "another test")
print(example_function.__doc__)  # Вывод обновленного docstring
```

**Вывод**: Пример демонстрирует, как использовать декоратор `autodoc` для автоматического обновления строки документации.  Вы увидите, как docstring функции `example_function` будет обновляться при каждом вызове.
