## Описание полей

Каждый объект сценария содержит следующие поля:

-   `brand` (str): Бренд ноутбука (например, "LENOVO").
-   `template` (str): Шаблон серии (например, "THINKPAD X").
-   `url` (str): URL-адрес, связанный с данной конфигурацией (может быть ссылкой на товар или просто текстовым описанием).
-  `checkbox` (bool): Поле для чекбокса, которое всегда `false`.
-   `active` (bool): Указывает, активен ли данный сценарий (`true` или `false`).
-   `condition` (str): Состояние товара, в данном случае всегда `new`.
-   `presta_categories` (str): Строка, содержащая идентификаторы категорий PrestaShop, разделенные запятыми.
