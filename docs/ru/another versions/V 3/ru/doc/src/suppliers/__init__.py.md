# Модуль `src.suppliers`

## Обзор

Модуль `src.suppliers` предоставляет базовый класс `Supplier` и механизмы для расширения его функциональности специфичными методами для каждого поставщика. Это позволяет единообразно обрабатывать информацию, получаемую из разных источников (например, Amazon, AliExpress, Morlevi).

## Подорбней

Основная цель модуля — предоставить гибкую структуру для извлечения и обработки данных от различных поставщиков. Для этого используется механизм подключения специфичных функций через интерфейс `supplier.related_functions`. Методы каждого поставщика располагаются в отдельных директориях, что облегчает поддержку и расширение функциональности. Взаимосвязь между сущностями `Supplier`, `Driver` и `Product` иллюстрируется диаграммой `supplier-warehouse-client.png`.
