# BATNUM: משחק ניחוש מילים

## סקירה

קובץ זה מיישם את משחק המילים "בונקר", בו המחשב בוחר מילה בת 5 אותיות, והשחקן צריך לנחש אותה. לאחר כל ניסיון, המחשב מספק רמזים: "A" עבור אות במקום הנכון, "B" עבור אות קיימת במילה אבל לא במקום הנכון, ו-"-" עבור אות שאינה קיימת במילה.
