# Задача No 830
## Длиннейшая палиндромная подпоследовательность

## Условия
Рассмотрим последовательность целых чисел `A`, длиной `n`. Подпоследовательностью последовательности `A` является любая последовательность, которую можно получить из `A` путём удаления нуля или более элементов, не меняя порядок оставшихся элементов. Например, если `A` = {1, 2, 3, 4, 5}, то {1, 3, 5} является подпоследовательностью `A`, а {1, 5, 3} не является.
Палиндромная последовательность – это последовательность, которая читается одинаково в обоих направлениях. Например, {1, 2, 1} – палиндром, а {1, 2, 3} – нет.

Длина подпоследовательности – это количество элементов в подпоследовательности. Ваша задача – написать программу, которая для заданной последовательности целых чисел `A` находит длину её самой длинной палиндромной подпоследовательности.

Пусть `N` = 10<sup>7</sup>.
Пусть `s[0] = 290797`.
Пусть `s[n+1] = (s[n] * s[n] mod 50515093) mod 2000 + 1` для `n ≥ 0`.
Пусть последовательность `A` есть `s[0], s[1], ..., s[N-1]`.

Найдите длину самой длинной палиндромной подпоследовательности последовательности `A`.
