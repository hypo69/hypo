# Модуль pricelist_generator

## Обзор

Модуль предназначен для экспериментов с созданием PDF-отчетов на основе данных, полученных в формате JSON. Он использует класс `ReportGenerator` для генерации отчета в формате HTML и последующего преобразования в PDF.

## Подробней

Данный модуль является частью экспериментов, проводимых в рамках проекта `hypotez`. Он предназначен для автоматизации процесса создания PDF-отчетов на основе данных, полученных из JSON-файлов. Основная цель - упростить и ускорить процесс создания отчетов, предоставляя гибкий инструмент для преобразования данных в удобочитаемый формат.

## Классы

### `ReportGenerator`

**Описание**: Класс, отвечающий за генерацию отчетов на основе предоставленных данных.

**Как работает класс**:

Класс `ReportGenerator` инкапсулирует логику создания отчетов. Он принимает данные, файл HTML и файл PDF в качестве входных данных и использует их для генерации отчета. Для работы требуется наличие путей к данным, HTML-шаблону и конечному PDF-файлу.

**Методы**:
- `create_report`: Метод для создания отчета.

## Функции

### `create_report`

```python
def create_report(data: dict, html_file: Path, pdf_file: Path) -> None:
    """
    Создает отчет на основе предоставленных данных, HTML-файла и сохраняет его в PDF-файл.

    Args:
        data (dict): Данные для отчета.
        html_file (Path): Путь к HTML-файлу.
        pdf_file (Path): Путь к PDF-файлу.

    Returns:
        None

    Raises:
        Exception: Если происходит ошибка при создании отчета.

    """
```

**Как работает функция**:

Функция `create_report` принимает словарь `data`, путь к HTML-файлу `html_file` и путь к PDF-файлу `pdf_file`. Она использует эти данные для создания отчета.

**Параметры**:
- `data` (dict): Данные, используемые для генерации отчета.
- `html_file` (Path): Путь к HTML-файлу, который будет использован в качестве шаблона для отчета.
- `pdf_file` (Path): Путь к PDF-файлу, в который будет сохранен сгенерированный отчет.

**Возвращает**:
- `None`: Функция ничего не возвращает.

**Вызывает исключения**:
- `Exception`: Функция может вызвать исключение, если в процессе генерации отчета произойдет ошибка.

**Примеры**:

```python
from pathlib import Path
from src.endpoints.kazarinov.react import ReportGenerator

data = {'ключ': 'значение'}
html_file = Path('path/to/template.html')
pdf_file = Path('path/to/output.pdf')

report_generator = ReportGenerator()
report_generator.create_report(data, html_file, pdf_file)
