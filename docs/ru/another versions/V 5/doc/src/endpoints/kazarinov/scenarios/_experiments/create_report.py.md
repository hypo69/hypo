# Модуль для проверки генерации прайслиста

## Обзор

Модуль предназначен для проверки и генерации отчетов о прайс-листах на основе данных, полученных от различных поставщиков. Он включает в себя функциональность для извлечения, обработки и интеграции данных о продуктах, а также для создания HTML и PDF отчетов.

## Подробней

Этот модуль играет важную роль в процессе автоматической генерации прайс-листов для проекта `hypotez`. Он использует класс `ReportGenerator` для создания отчетов на разных языках (в данном случае, иврит и русский) и сохраняет их в форматах HTML и PDF. Модуль также взаимодействует с другими частями проекта, такими как модули для обработки данных и интеграции с внешними платформами.

## Переменные

- `report_generator`: Экземпляр класса `ReportGenerator`, используемый для создания отчетов.
- `html_file_he`: Путь к HTML-файлу для отчета на иврите.
- `pdf_file_he`: Путь к PDF-файлу для отчета на иврите.
- `html_file_ru`: Путь к HTML-файлу для отчета на русском.
- `pdf_file_ru`: Путь к PDF-файлу для отчета на русском.

## Функции

### `create_report`

```python
def create_report(data: dict, lang: str, html_file: Path, pdf_file: Path) -> None:
    """
    Создает отчет в формате HTML и PDF на основе предоставленных данных.

    Args:
        data (dict): Данные для отчета.
        lang (str): Язык отчета.
        html_file (Path): Путь для сохранения HTML-файла.
        pdf_file (Path): Путь для сохранения PDF-файла.

    Returns:
        None

    Raises:
        Exception: Если возникает ошибка при создании отчета.
    """
```

**Как работает функция**:
Функция `create_report` принимает данные, язык и пути для сохранения HTML и PDF файлов в качестве параметров. Она вызывает метод `create_report` объекта `report_generator` для генерации отчета на указанном языке и сохранения его в указанных файлах.

**Параметры**:
- `data` (dict): Данные для отчета.
- `lang` (str): Язык отчета.
- `html_file` (Path): Путь для сохранения HTML-файла.
- `pdf_file` (Path): Путь для сохранения PDF-файла.

**Возвращает**:
- `None`

**Примеры**:

```python
# Пример вызова функции create_report
report_generator = ReportGenerator()
html_file_he = test_directory / 'he.html'
pdf_file_he = test_directory / 'he.pdf'
report_generator.create_report(response_he_dict['he'], 'he', html_file_he, pdf_file_he)
