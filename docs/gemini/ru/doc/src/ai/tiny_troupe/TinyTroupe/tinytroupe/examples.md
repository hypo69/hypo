# Модуль examples

## Обзор

Этот модуль содержит примеры использования библиотеки tinytroupe.  Примеры демонстрируют создание агентов, настраивая их характеристики, такие как возраст, национальность, профессия, рутина, описание профессии, черты характера, профессиональные интересы, личные интересы, навыки и отношения с другими агентами.

## Функции

### `create_oscar_the_architect`

**Описание**: Функция создает агента "Oscar", представляющего архитектора.

**Возвращает**:
- `TinyPerson`: Объект агента Oscar с заданными параметрами.


### `create_lisa_the_data_scientist`

**Описание**: Функция создает агента "Lisa", представляющего специалиста по данным.

**Возвращает**:

### `create_marcos_the_physician`
