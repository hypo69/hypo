## Обзор

Модуль `src.suppliers.bangood.scenario` предназначен для сбора данных о товарах с сайта поставщика bangood.co.il с использованием веб-драйвера. Он включает в себя функции для сбора списка категорий, товаров в категориях и обработки страниц товаров.

## Функции

### `get_list_products_in_category`

**Описание**: Функция возвращает список URL-адресов товаров со страницы категории. Если необходимо пролистать страницы, функция должна это сделать (в текущей версии TODO).

**Параметры**:
- `s` (Supplier): Объект поставщика, содержащий необходимые данные (драйвер, локаторы).

**Возвращает**:
- `list[str, str, None]`: Список URL-адресов товаров или `None`, если не найдено ни одного товара.

**Пример использования**:

```python
list_of_products = get_list_products_in_category(supplier_object)
if list_of_products:
    for product_url in list_of_products:
        print(product_url)
```

**Детали реализации:**
1.  Получает драйвер из объекта поставщика `s`.
2.  Получает локаторы для категории из `s.locators['category']`.
3.  Закрывает баннер, используя локатор `s.locators['product']['close_banner']`.
4.  Проверяет наличие локаторов, и если их нет, выводит сообщение об ошибке и возвращает `None`.
5.  Прокручивает страницу.
6.  **TODO**: Добавить поддержку пролистывания страниц.
7.  Собирает ссылки на товары, используя локатор `l['product_links']`.
8.  Если ссылки не найдены, выводит предупреждение и возвращает `None`.
9.  Преобразует одиночную ссылку в список, если это необходимо.
10. Выводит информационное сообщение о количестве найденных товаров.
11. Возвращает список URL-адресов товаров.
