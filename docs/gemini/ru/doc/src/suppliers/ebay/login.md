# Модуль `hypotez/src/suppliers/ebay/login.py`

## Обзор

Этот модуль содержит реализацию логина на eBay для веб-драйвера.  Он определяет константу `MODE`, которая, вероятно, задаёт режим работы (например, разработка или производство).  Модуль предполагает наличие изображения `login.png`, которое может быть использовано для визуализации процесса логина.

## Константы

### `MODE`

**Описание**:  Переменная, определяющая режим работы.  Вероятно, принимает значения 'dev' или 'prod' (разработка или производство).

**Значение по умолчанию**: 'dev'
