# Модуль scenario_executor

## Обзор

Данный модуль отвечает за выполнение сценариев, считывая список файлов сценариев, загружая и исполняя каждый сценарий по очереди.  Каждый сценарий описывает ряд действий, направленных на взаимодействие с внешними ресурсами (в данном случае, скорее всего, веб-сайт PrestaShop).  Модуль обрабатывает как успешное выполнение сценариев, так и ошибки, предоставляя информацию об успехе/неуспехе и обновляя журнал.

## Функции

### `run_scenario_executor`

**Описание**: Функция запускает выполнение всех сценариев из предоставленного списка файлов.

**Параметры**:

- `scenario_files_list` (list[str]): Список путей к файлам сценариев.
- `journal_file_path` (str, optional): Путь к файлу журнала. По умолчанию None.


**Возвращает**:

- `bool`: `True`, если все сценарии успешно выполнены, `False` в противном случае.


**Вызывает исключения**:

- `ValueError`: Если `scenario_files_list` пуст или содержит некорректные пути к файлам.
- `FileNotFoundError`: Если какой-либо файл сценария не найден.
- `Exception`: Обрабатывает любые другие непредвиденные ошибки при выполнении сценариев.
