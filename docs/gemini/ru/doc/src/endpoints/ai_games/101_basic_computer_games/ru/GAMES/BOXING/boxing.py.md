# Игра "Бокс"

## Обзор

Этот файл содержит код игры "Бокс" - текстовой имитации боксерского поединка между двумя игроками. Игроки по очереди наносят удары, выбирая их силу (слабый, средний, сильный), и компьютер случайным образом определяет, попал ли удар. Игра продолжается, пока один из боксеров не будет нокаутирован.

## Импорт

```python
import random
```

Импортируется модуль `random` для генерации случайных чисел, используемых при определении попадания удара.

## Переменные

```python
player1Health = 10
player2Health = 10
currentPlayer = 1
```
- `player1Health`:  Целочисленная переменная, хранящая здоровье первого игрока. Изначально устанавливается в 10.
- `player2Health`: Целочисленная переменная, хранящая здоровье второго игрока. Изначально устанавливается в 10.
- `currentPlayer`: Целочисленная переменная, указывающая на текущего игрока (1 или 2). Изначально устанавливается в 1.

## Основной игровой цикл

```python
while player1Health > 0 and player2Health > 0:
    print(f"Здоровье игрока 1: {player1Health}, Здоровье игрока 2: {player2Health}")
    
    while True:
        try:
            attackType = int(input(f"Игрок {currentPlayer}, выберите силу удара (1-слабый, 2-средний, 3-сильный): "))
            if attackType in [1, 2, 3]:
                break
            else:
                print("Некорректный ввод. Пожалуйста, выберите 1, 2 или 3.")
        except ValueError as ex:
            print("Некорректный ввод. Пожалуйста, введите число.")
    
    randomNum = random.randint(1, 100)
    
    hit = False
    if attackType == 1 and randomNum <= 80:
        hit = True
    elif attackType == 2 and randomNum <= 60:
        hit = True
    elif attackType == 3 and randomNum <= 40:
        hit = True
    
    if hit:
        print("Удар достиг цели!")
        if currentPlayer == 1:
            player2Health -= 1
        else:
            player1Health -= 1
    else:
        print("Удар не достиг цели.")

    currentPlayer = 3 - currentPlayer
```
   
-   **Цикл `while player1Health > 0 and player2Health > 0:`**:
    -   Условие: Цикл продолжается, пока здоровье обоих игроков больше нуля.
    -   **Вывод текущего здоровья**:
        -   Выводит текущее здоровье обоих игроков.
    -   **Ввод силы удара**:
        -   **Цикл `while True:`**:
            - Бесконечный цикл для ввода данных, пока не будет введено корректное значение.
            -   **Обработка ошибок**:
                -   `try...except ValueError as ex`: Обрабатывает исключение `ValueError`, если пользователь ввел не число.
                -   Выводит сообщение об ошибке при некорректном вводе.
            -  Запрашивает у текущего игрока выбор силы удара (1 - слабый, 2 - средний, 3 - сильный).
            -   `if attackType in [1, 2, 3]:`: Проверяет, корректен ли ввод. Если ввод корректен, то выходит из цикла ввода, в противном случае запрашивает ввод снова.
    -   **Генерация случайного числа**:
         -   `randomNum = random.randint(1, 100)`: Генерирует случайное число от 1 до 100 для определения попадания.
    -   **Определение попадания**:
        - `hit = False`: Инициализирует переменную hit (попадание) со значением False.
        -   Условные операторы `if`, `elif` проверяют, попал ли удар в зависимости от выбранной силы удара и сгенерированного случайного числа.
    -   **Уменьшение здоровья**:
        -  Если удар достиг цели, уменьшает здоровье противника на 1.
    -   **Вывод результатов**:
         -   Выводит сообщение о том, попал удар или нет.
    -   **Переключение игрока**:
         - `currentPlayer = 3 - currentPlayer`: Переключает текущего игрока (с 1 на 2 или с 2 на 1).

## Определение победителя

```python
if player1Health <= 0:
    print("Игрок 2 победил!")
else:
    print("Игрок 1 победил!")
```

-   После выхода из основного игрового цикла (когда здоровье одного из игроков достигло нуля), определяется и объявляется победитель.
-   Проверяется, если здоровье игрока 1 меньше или равно нулю. Если это так, то побеждает игрок 2. В противном случае побеждает игрок 1.
