# Задача No 65
## Конвергенты e
## Условия
Дробь 27/10 представляет собой 10-ый конвергент непрерывной дроби для e:

```
e = [2; 1,2,1, 1,4,1, 1,6,1 , ... , 1,2k,1, ...]
```
Первые десять конвергентов для e:
```
2, 3, 8/3, 11/4, 19/7, 87/32, 106/39, 193/71, 1264/465, 1457/536
```
Сумма цифр числителя 10-го конвергента равна 1 + 4 + 5 + 7 = 17.

Найдите сумму цифр в числителе 100-го конвергента для непрерывной дроби для e.
