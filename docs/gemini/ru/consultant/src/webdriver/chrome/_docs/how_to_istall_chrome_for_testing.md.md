# ИНСТРУКЦИЯ по установке Chrome для тестирования

## Основные требования:

Для автоматизированного тестирования с использованием Selenium или других подобных инструментов требуется установка совместимой версии браузера Google Chrome и соответствующего ChromeDriver. Ниже приведена подробная инструкция.

## 1. Установка Google Chrome

### Шаг 1. Загрузка
Скачайте последнюю версию Google Chrome с официального сайта:
[https://www.google.com/chrome/](https://www.google.com/chrome/)

### Шаг 2. Установка

Запустите скачанный установщик и следуйте инструкциям на экране.
После установки убедитесь, что браузер корректно запускается.

### Шаг 3. Проверка версии
После установки проверьте версию установленного браузера. Для этого:
1. Запустите Chrome.
2. В адресной строке введите `chrome://settings/help` и нажмите Enter.
3. Запомните версию браузера. Она потребуется для дальнейшей установки ChromeDriver.

## 2. Установка ChromeDriver

ChromeDriver - это отдельный исполняемый файл, который позволяет Selenium и другим инструментам управлять браузером Chrome.

### Шаг 1. Определение версии ChromeDriver

1. Посетите сайт [https://chromedriver.chromium.org/downloads](https://chromedriver.chromium.org/downloads)
2. Найдите версию ChromeDriver, соответствующую версии вашего установленного Chrome (версию вы определили на шаге 3 из предыдущего раздела).

### Шаг 2. Загрузка ChromeDriver

1. Скачайте подходящий zip-архив ChromeDriver.
2. Распакуйте скачанный архив в любую удобную вам директорию. Рекомендуется добавить директорию, где расположен ChromeDriver, в системную переменную `PATH`, чтобы не указывать путь к нему каждый раз при запуске тестов.

### Шаг 3. Проверка установки
Вы можете проверить установку, запустив chromedriver.exe (или chromedriver в Linux/macOS).  При успешном запуске в терминале будет выведена информация о версии и порт, на котором запущен драйвер.

## 3. Настройка переменных среды (необязательно, но рекомендуется)

### Шаг 1. Windows

1. Нажмите Win+Pause/Break или перейдите в "Система" через "Панель управления".
2. Выберите "Дополнительные параметры системы".
3. Нажмите "Переменные среды".
4. В разделе "Системные переменные" выберите `Path` и нажмите "Изменить".
5. Нажмите "Создать" и добавьте путь к директории, где расположен файл `chromedriver.exe`.
6. Нажмите "ОК" во всех открытых окнах.

### Шаг 2. Linux/macOS

1.  Откройте терминал
2.  Отредактируйте файл `.bashrc` (или `.zshrc`, если вы используете zsh) и добавьте строку:

    ```bash
    export PATH=$PATH:/path/to/your/chromedriver/directory
    ```
    Замените `/path/to/your/chromedriver/directory` на фактический путь к директории, где находится ChromeDriver
3.  Сохраните файл и выполните в терминале команду: `source ~/.bashrc` (или `source ~/.zshrc`)

## 4. Заключение

Теперь у вас установлены Google Chrome и ChromeDriver, необходимые для запуска автоматизированных тестов. Убедитесь, что версии ChromeDriver и браузера Chrome совместимы.

## Примечание
Регулярно обновляйте Chrome и ChromeDriver для обеспечения стабильности и безопасности ваших тестов.
