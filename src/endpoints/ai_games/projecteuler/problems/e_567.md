# Задача No 567
## Sum of squares
## Условия

Пусть S(n) - сумма квадратов цифр числа n. Последовательность чисел определяется следующим образом: a0 = n и ak+1 = S(ak).

Например, если начать с n = 44, то последовательность будет 44, 32, 13, 10, 1, 1.
Последовательность, начинающаяся с n = 85, будет 85, 89, 145, 42, 20, 4, 16, 37, 58, 89.

Интересно, что каждая последовательность либо приходит к 1, либо входит в цикл, содержащий 89.

Назовем счастливым числом такое число n, для которого последовательность, начинающаяся с n, приходит к 1.
Назовем несчастливым числом такое число n, для которого последовательность, начинающаяся с n, входит в цикл, содержащий 89.

Назовем числом n «невезением», если ни n, ни n + 1 не являются счастливыми.
Например, 46 и 47 – оба несчастливые, поэтому 46 – число невезения.

Найдите количество чисел невезения, не превышающих 10 000 000.
