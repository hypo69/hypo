# Модуль `header.py`

## Обзор

Модуль `header.py` предназначен для настройки и инициализации основных параметров проекта, таких как корневая директория, конфигурации из `config.json`, данных из `README.MD`, а также основных метаданных проекта.

## Функции

### `set_project_root`

**Описание**:
Находит корневую директорию проекта, начиная с директории текущего файла и двигаясь вверх, останавливаясь на первой директории, содержащей один из маркеров.

**Параметры**:
- `marker_files` (tuple, optional): Список файлов или директорий, которые указывают на корень проекта. По умолчанию `('__root__','.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории проекта, если она найдена, иначе путь к директории, где расположен скрипт.

### Глобальные переменные

### `__root__`
**Описание**:
Хранит путь к корневой директории проекта, определенному с помощью функции `set_project_root`.

### `config`
**Описание**:
Словарь, содержащий конфигурации, загруженные из файла `config.json`.

### `doc_str`
**Описание**:
Содержит строку с содержимым файла `README.MD`.

### `__project_name__`
**Описание**:
Содержит имя проекта, полученное из конфигурации или по умолчанию `hypotez`.

### `__version__`
**Описание**:
Содержит версию проекта, полученную из конфигурации.

### `__doc__`
**Описание**:
Содержит описание проекта, полученное из файла `README.MD`.

### `__details__`
**Описание**:
Содержит дополнительные детали о проекте.

### `__author__`
**Описание**:
Содержит имя автора проекта, полученное из конфигурации.

### `__copyright__`
**Описание**:
Содержит информацию об авторских правах, полученную из конфигурации.

### `__cofee__`
**Описание**:
Содержит строку с призывом угостить разработчика чашечкой кофе.
