# Модуль KINEMA

## Обзор

Модуль `KINEMA` реализует текстовую игру "Кинема", в которой игрок управляет перемещением точки на плоскости, вводя команды для движения влево, вправо, вверх и вниз. Текущие координаты точки и ее путь отображаются в консоли.

## Переменные

- `x` (int): Координата X точки на плоскости. Изначально устанавливается в 0.
- `y` (int): Координата Y точки на плоскости. Изначально устанавливается в 0.
- `command` (str): Команда, введенная пользователем для перемещения точки.

## Основной цикл игры

Основной цикл игры управляет ходом игры, обрабатывает ввод команд пользователем и обновляет координаты точки.

**Описание**:

-   Цикл начинается с бесконечного `while True:`.
-   Запрашивает у пользователя ввод команды (`L/R/U/D` или `E` для выхода).
-   Если введена команда `E`, игра завершается.
-   В зависимости от введенной команды, изменяет координаты `x` и `y`.
-   Выводит текущие координаты.
-   Делает небольшую паузу.

```python
while True:
    command = input("Введите команду (L/R/U/D или E для выхода): ").upper()
    if command == "E":
        print("Игра окончена.")
        break
    # ...
```

## Обработка команд перемещения

В этом разделе происходит анализ команды, введенной пользователем, и изменяются координаты точки в соответствии с командой.

**Описание**:

-   Если команда `L`, уменьшает координату `x` на 1.
-   Если команда `R`, увеличивает координату `x` на 1.
-   Если команда `U`, увеличивает координату `y` на 1.
-   Если команда `D`, уменьшает координату `y` на 1.
-   Если команда не распознана, выводится сообщение об ошибке.

```python
    if command == "L":
        x -= 1
    elif command == "R":
        x += 1
    elif command == "U":
        y += 1
    elif command == "D":
        y -= 1
    else:
        print("Неверная команда. Пожалуйста, используйте L, R, U, D или E.")
        continue
```

## Вывод текущих координат

После каждого перемещения точки выводит ее текущие координаты на экран.

**Описание**:
- Выводит текущие координаты точки в формате `(x, y)`.
- Делает небольшую паузу.

```python
    print(f"Текущие координаты: ({x}, {y})")
    time.sleep(0.5)
```
