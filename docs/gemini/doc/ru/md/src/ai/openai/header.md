# Модуль `hypotez/src/logger/header.py`

## Обзор

Модуль `hypotez/src/logger/header.py` определяет корневой путь к проекту `hypotez`.  Все импорты строятся относительно этого пути.  В будущем этот путь планируется перенести в системную переменную.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла, проходя вверх по дереву директорий, пока не найдёт директорию, содержащую один из указанных файлов или папок.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, которые указывают на корневую директорию проекта.  По умолчанию это `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена. В противном случае возвращает директорию, в которой находится текущий файл.

**Вызывает исключения**:

- Нет


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта.

**Тип**: `Path`


## Дополнительные данные

### Загрузка настроек

Загружает настройки из файла `settings.json` в переменную `settings`. Обрабатывает ошибки `FileNotFoundError` и `json.JSONDecodeError`, если файл не найден или содержит некорректные данные.

### Загрузка документации

Загружает содержимое файла `README.MD` в переменную `doc_str`. Обрабатывает ошибки `FileNotFoundError` и `json.JSONDecodeError`.

### Проектные атрибуты

- `__project_name__`: Название проекта (из `settings.json` или по умолчанию 'hypotez').
- `__version__`: Версия проекта (из `settings.json` или по умолчанию пустая строка).
- `__doc__`: Документация проекта (из `README.MD` или по умолчанию пустая строка).
- `__details__`: Подробности о проекте (пустая строка по умолчанию).
- `__author__`: Автор проекта (из `settings.json` или по умолчанию пустая строка).
- `__copyright__`: Авторские права (из `settings.json` или по умолчанию пустая строка).
- `__cofee__`: Ссылка для пожертвований (из `settings.json` или по умолчанию ссылка).
