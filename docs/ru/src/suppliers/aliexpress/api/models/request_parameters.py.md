# Модуль: src.suppliers.aliexpress.api.models.request_parameters

## Обзор

Модуль содержит классы, определяющие параметры запросов к API AliExpress. Эти классы используются для указания типов продуктов, сортировки результатов и типов ссылок.

## Подробней

Этот модуль предоставляет набор констант, которые можно использовать для стандартизации и облегчения создания запросов к API AliExpress. Он определяет возможные значения для типов продуктов, способов сортировки и типов ссылок, что помогает обеспечить консистентность и избежать ошибок при формировании запросов.

## Классы

### `ProductType`

**Описание**: Класс, определяющий типы продуктов, которые можно запросить через API AliExpress.

**Принцип работы**:
Класс содержит статические атрибуты, представляющие различные типы продуктов: `ALL`, `PLAZA` и `TMALL`.

**Атрибуты**:
- `ALL` (str): Представляет все типы продуктов.
- `PLAZA` (str): Представляет продукты, доступные на площадке Plaza.
- `TMALL` (str): Представляет продукты, доступные на площадке Tmall.

**Примеры**:

```python
product_type = ProductType.ALL
print(product_type)  # Вывод: ALL
```

### `SortBy`

**Описание**: Класс, определяющий способы сортировки результатов запроса к API AliExpress.

**Принцип работы**:
Класс содержит статические атрибуты, представляющие различные способы сортировки: по возрастанию/убыванию цены, по объему продаж.

**Атрибуты**:
- `SALE_PRICE_ASC` (str): Сортировка по возрастанию цены.
- `SALE_PRICE_DESC` (str): Сортировка по убыванию цены.
- `LAST_VOLUME_ASC` (str): Сортировка по возрастанию объема продаж.
- `LAST_VOLUME_DESC` (str): Сортировка по убыванию объема продаж.

**Примеры**:

```python
sort_order = SortBy.SALE_PRICE_ASC
print(sort_order)  # Вывод: SALE_PRICE_ASC
```

### `LinkType`

**Описание**: Класс, определяющий типы ссылок, возвращаемых API AliExpress.

**Принцип работы**:
Класс содержит статические атрибуты, представляющие различные типы ссылок: `NORMAL` и `HOTLINK`.

**Атрибуты**:
- `NORMAL` (int): Обычная ссылка.
- `HOTLINK` (int): "Горячая" ссылка (например, партнерская).

**Примеры**:

```python
link_type = LinkType.NORMAL
print(link_type)  # Вывод: 0
```
