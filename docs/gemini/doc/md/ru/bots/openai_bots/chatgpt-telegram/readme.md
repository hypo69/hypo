```markdown
# Голосовой помощник chatgpt - telegram

## Обзор

Данный модуль предоставляет функциональность для взаимодействия с голосовым помощником ChatGPT через Telegram.  Он описывает логику работы и методы, необходимые для интеграции.


## Функции

### `get_message`

**Описание**: Функция для получения сообщения от пользователя в Telegram.

**Параметры**:
- `update` (telebot.types.Update): Объект с данными от Telegram.


**Возвращает**:
- `str`: Текст сообщения пользователя.

**Вызывает исключения**:
- `ValueError`: Если сообщение не получено или имеет неверный тип.
- `TypeError`: В случае некорректного типа входных данных.


### `process_message`

**Описание**: Функция для обработки полученного сообщения и отправки запроса к ChatGPT.

**Параметры**:
- `message` (str): Текст сообщения пользователя.


**Возвращает**:
- `str`: Ответ от ChatGPT.

**Вызывает исключения**:
- `ValueError`: Если сообщение не может быть обработано.
- `ConnectionError`: Если нет подключения к API ChatGPT.
- `APIError`: При проблемах с API ChatGPT.


### `send_message`

**Описание**: Функция для отправки ответа ChatGPT в Telegram.

**Параметры**:
- `bot` (telebot.TeleBot): Объект бота для отправки сообщений.
- `chat_id` (int): Идентификатор чата.
- `response` (str): Текст ответа ChatGPT.


**Возвращает**:
- `None`.

**Вызывает исключения**:
- `telebot.apihelper.ApiException`: При проблемах с Telegram API.
- `ValueError`: Если сообщение для отправки некорректно.


```
