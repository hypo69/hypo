# Модуль `header.py`

## Обзор

Модуль `header.py` предназначен для настройки проекта, определения его корневой директории, загрузки настроек из файла `settings.json`, чтения документации из `README.MD` и определения основных атрибутов проекта, таких как имя, версия, автор и т.д.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла, и добавляет её в `sys.path`. Поиск идет вверх по директориям, пока не будет найдена директория, содержащая один из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple, optional): Кортеж имен файлов или директорий, которые идентифицируют корневую директорию проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории проекта, если она найдена. Если корневая директория не найдена, возвращает директорию, где находится скрипт.

## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта. Инициализируется функцией `set_project_root`.
Тип: `Path`

### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженные из файла `settings.json`. Если файл не найден или не является валидным JSON, то принимает значение `None`.
Тип: `dict | None`

### `doc_str`

**Описание**: Строка, содержащая документацию проекта, загруженную из файла `README.MD`. Если файл не найден, то принимает значение `None`.
Тип: `str | None`

### `__project_name__`

**Описание**: Имя проекта. Получается из `settings` или по умолчанию равно `hypotez`.
Тип: `str`

### `__version__`

**Описание**: Версия проекта. Получается из `settings` или по умолчанию равна пустой строке.
Тип: `str`

### `__doc__`

**Описание**: Строка с документацией, загруженная из файла `README.MD` или по умолчанию пустая строка.
Тип: `str`

### `__details__`

**Описание**: Строка для хранения дополнительной информации о проекте. По умолчанию пустая строка.
Тип: `str`

### `__author__`

**Описание**: Автор проекта. Получается из `settings` или по умолчанию пустая строка.
Тип: `str`

### `__copyright__`

**Описание**: Информация об авторских правах. Получается из `settings` или по умолчанию пустая строка.
Тип: `str`

### `__cofee__`

**Описание**: Строка с приглашением поддержать разработчика чашкой кофе. Получается из `settings` или по умолчанию содержит ссылку на Boosty.
Тип: `str`
