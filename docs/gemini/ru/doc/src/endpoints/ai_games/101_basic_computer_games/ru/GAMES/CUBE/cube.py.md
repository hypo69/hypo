# Модуль CUBE

## Обзор

Этот модуль реализует игру-головоломку "Куб", в которой игрок должен собрать куб, перемещая его грани. Куб представлен в виде матрицы 3x3, где каждая ячейка представляет собой грань куба.

## Функции

### `generate_cube`

**Описание**: Генерирует случайный куб 3x3.

**Возвращает**:
- `list[list[int]]`:  Случайно сгенерированный куб в виде матрицы 3x3.

### `display_cube`

**Описание**: Выводит куб на экран.

**Параметры**:
- `cube` (`list[list[int]]`): Куб для вывода.

**Возвращает**:
- `None`: Функция ничего не возвращает, а выводит куб на экран.

### `move_cube`

**Описание**: Перемещает грани куба в зависимости от введенной команды.

**Параметры**:
- `cube` (`list[list[int]]`): Куб, грани которого нужно переместить.
- `move` (`str`): Команда перемещения (`U`, `D`, `L` или `R`).

**Возвращает**:
- `list[list[int]]`: Куб после перемещения граней.

### `is_solved`

**Описание**: Проверяет, собран ли куб.

**Параметры**:
- `cube` (`list[list[int]]`): Куб для проверки.

**Возвращает**:
- `bool`: `True`, если куб собран, `False` в противном случае.
