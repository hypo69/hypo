# Модуль hypotez/src/suppliers/hb/scenarios/__init__.py

## Обзор

Данный модуль предоставляет функции и классы для работы с поставщиком `hb.co.il`.  Он содержит функции для получения списка категорий, списка товаров в категории, скачивания страницы товара и авторизации.

## Постоянные переменные

### `MODE`

**Описание**: Переменная, хранящая режим работы (например, 'dev', 'prod').

**Значение**: `'dev'`


## Функции

### `get_list_products_in_category`

**Описание**: Возвращает список товаров в указанной категории.

**Параметры**:
- `category_id` (str): Идентификатор категории.

**Возвращает**:
- `list`: Список товаров.


### `get_list_categories_from_site`

**Описание**: Возвращает список категорий сайта.

**Возвращает**:
- `list`: Список категорий.


### `grab_product_page`

**Описание**: Скачивает страницу товара.

**Параметры**:
- `product_url` (str): URL страницы товара.

**Возвращает**:
- `str`: HTML-код страницы товара.

**Возможные исключения**:
- `Exception`: При возникновении ошибок при скачивании.


### `login`

**Описание**: Функция для авторизации.

**Параметры**:
-  `username` (str): Имя пользователя.
-  `password` (str): Пароль.

**Возвращает**:
- `bool`: `True`, если авторизация успешна, `False` иначе.

**Возможные исключения**:
- `Exception`: В случае ошибок авторизации.


## Модули

### `packaging.version`

**Описание**: Модуль для работы с версиями пакетов.


### `hb.scenarios.version`

**Описание**: Модуль, содержащий информацию о версии и документацию.

**Атрибуты**:
- `__version__`: Строка, представляющая версию.
- `__doc__`: Документация модуля.
- `__details__`: Подробные данные о версии.
