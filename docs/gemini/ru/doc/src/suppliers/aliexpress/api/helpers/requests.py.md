# Модуль `requests.py`

## Обзор

Модуль `requests.py` содержит вспомогательные функции для отправки API-запросов и обработки ответов от AliExpress API. Он включает в себя функцию `api_request`, которая выполняет запрос, обрабатывает возможные ошибки и возвращает результат.

## Функции

### `api_request`

**Описание**: Функция отправляет API-запрос, обрабатывает ответ и возвращает результат. Функция выполняет запрос, обрабатывает возможные ошибки и возвращает результат. В случае ошибки, она логирует информацию и возвращает `None`.

**Параметры**:
- `request` (object): Объект запроса, который имеет метод `getResponse` для получения ответа от API.
- `response_name` (str): Название ключа в ответе, содержащего нужные данные.
- `attemps` (int, optional): Количество попыток выполнения запроса. По умолчанию 1.

**Возвращает**:
- `object | None`: Объект с результатом, полученным из ответа API, или `None` в случае ошибки.

**Вызывает исключения**:
- `ApiRequestException`: Возникает, если во время выполнения запроса произошла ошибка, связанная с запросом.
- `ApiRequestResponseException`: Возникает, если во время обработки ответа произошла ошибка.
