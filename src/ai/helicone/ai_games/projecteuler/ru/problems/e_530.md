# Задача No 530
## Сумма Мобиуса

## Условия
Функция Мёбиуса μ(n) определяется следующим образом:
- μ(n) = 1, если n - свободное от квадратов положительное целое число с чётным количеством простых множителей.
- μ(n) = -1, если n - свободное от квадратов положительное целое число с нечётным количеством простых множителей.
- μ(n) = 0, если n имеет квадратный множитель, отличный от 1.

Например, μ(6) = 1, μ(30) = -1 и μ(12) = 0.

Пусть M(n) будет суммой μ(i) для i = 1 до n. То есть M(n) = ∑ μ(i) для 1 ≤ i ≤ n.
Дано, что M(100) = 1, M(1000) = -6, M(10000) = -49.

Пусть S(n) будет суммой M(i) для i = 1 до n. То есть S(n) = ∑ M(i) для 1 ≤ i ≤ n.
Дано, что S(100) = -41, S(1000) = -419, S(10000) = -3288.

Найдите S(10<sup>12</sup>).
