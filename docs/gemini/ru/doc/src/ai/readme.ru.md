# Модуль `src.ai`

## Обзор

Модуль `ai` представляет собой интерфейс для управления различными моделями искусственного интеллекта (ИИ). Он абстрагирует взаимодействие с внешними API и обработку конфигураций, необходимых для анализа данных и обработки естественного языка. Модуль содержит ряд подмодулей, каждый из которых отвечает за интеграцию с конкретной моделью ИИ или сервисом.

## Подробнее

Модуль `ai` предназначен для упрощения работы с различными моделями ИИ, предоставляя унифицированный интерфейс для их использования. Он позволяет легко переключаться между разными моделями и сервисами, а также настраивать их параметры для решения конкретных задач. Этот модуль является ключевым компонентом проекта `hypotez`, обеспечивая возможность использования передовых технологий ИИ для анализа и обработки данных.

## Подмодули

### 1. `anthropic`

Интеграция с моделями ИИ Anthropic для задач, связанных с пониманием языка и генерацией ответов.

[Перейти к модулю](https://github.com/hypo69/hypo/blob/master/src/ai/anthropic/readme.ru.md)

### 2. `dialogflow`

Интеграция с Google Dialogflow для обработки естественного языка (NLU) и создания интерактивных приложений.

[Перейти к модулю](https://github.com/hypo69/hypo/blob/master/src/ai/dialogflow/readme.ru.md)

### 3. `gemini`

Управление соединениями с моделями ИИ Gemini для приложений, требующих уникальных возможностей ИИ Gemini.

[Перейти к модулю](https://github.com/hypo69/hypo/blob/master/src/ai/gemini/readme.ru.md)

### 4. `helicone`

Подключение к моделям Helicone для настройки решений на базе ИИ.

[Перейти к модулю](https://github.com/hypo69/hypo/blob/master/src/ai/helicone/readme.ru.md)

### 5. `llama`

Интерфейс для LLaMA (Large Language Model Meta AI) для понимания и генерации естественного языка.

[Перейти к модулю](https://github.com/hypo69/hypo/blob/master/src/ai/llama/readme.ru.md)

### 6. `myai`

Кастомный подмодуль ИИ для специализированных конфигураций и реализации уникальных функций ИИ.

[Перейти к модулю](https://github.com/hypo69/hypo/blob/master/src/ai/myai/readme.ru.md)

### 7. `openai`

Интеграция с API OpenAI для генерации текста, классификации, перевода и других задач.

[Перейти к модулю](https://github.com/hypo69/hypo/blob/master/src/ai/openai/readme.ru.md)

### 8. `tiny_troupe`

Интеграция с моделями ИИ от Microsoft для обработки естественного языка и анализа данных с использованием маленьких моделей.

[Перейти к модулю](https://github.com/hypo69/hypo/blob/master/src/ai/tiny_troupe/readme.ru.md)

### 9. `revai`

Интеграция с моделью от rev.com для работы с аудиофайлами (записи переговоров, совещаний, звонков и др.).

[Перейти к модулю](https://github.com/hypo69/hypo/blob/master/src/ai/revai/readme.ru.md)

### 10. `prompts`

Системные и командные промпты в формате Markdown для моделей ИИ.
