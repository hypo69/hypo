# Модуль `hypotez/src/ai/openai`

## Обзор

Этот модуль предоставляет интерфейс для работы с моделью OpenAI. Он содержит функции для перевода текста и взаимодействия с моделью.

## Модули

### `translator`

Модуль `translator` содержит функцию для перевода текста.

### `model`

Модуль `model` содержит класс `OpenAIModel` для взаимодействия с моделью OpenAI.

## Функции

### `translate`

**Описание**: Функция для перевода текста.

**Параметры**:
- `text` (str): Текст для перевода.
- `target_language` (str): Целевой язык перевода.

**Возвращает**:
- `str`: Переведенный текст.
- `None`: В случае возникновения ошибки.

**Вызывает исключения**:
- `Exception`: В случае возникновения необработанной ошибки.
