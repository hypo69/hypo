# Модуль для работы с интернет-поиском

## Обзор

Модуль предоставляет функциональность для выполнения интернет-поиска и получения результатов. Он использует модуль `web_search` из `...tools` для поиска и форматирования результатов.

## Подорбней

Этот модуль предназначен для интеграции поиска в другие части проекта, например, для получения дополнительной информации или контекста на основе запросов пользователей. Он содержит импорты необходимые для поиска в интернете и обработки полученных результатов.

## Функции

### `search`

```python
from ...tools.web_search import search
```

**Назначение**: Функция для выполнения поискового запроса в интернете.

**Параметры**:
- Параметры определяются в модуле `...tools.web_search.search`.

**Возвращает**:
- Результаты поиска определяются в модуле `...tools.web_search.search`.

**Как работает функция**:
1. Функция принимает поисковый запрос.
2. Выполняет поиск в интернете с использованием указанного запроса.
3. Возвращает результаты поиска.

### `SearchResults`

```python
from ...tools.web_search import SearchResults
```

**Назначение**: Класс для представления результатов поиска.

**Параметры**:
- Параметры определяются в модуле `...tools.web_search.SearchResults`.

**Возвращает**:
- Возвращаемое значение определяется в модуле `...tools.web_search.SearchResults`.

**Как работает функция**:
1. Класс принимает результаты поиска.
2. Форматирует результаты поиска для удобного представления.
3. Возвращает отформатированные результаты поиска.

### `get_search_message`

```python
from ...tools.web_search import get_search_message
```

**Назначение**: Функция для получения сообщения с результатами поиска.

**Параметры**:
- Параметры определяются в модуле `...tools.web_search.get_search_message`.

**Возвращает**:
- Возвращаемое значение определяется в модуле `...tools.web_search.get_search_message`.

**Как работает функция**:
1. Функция принимает результаты поиска.
2. Форматирует результаты в виде сообщения.
3. Возвращает сообщение с результатами поиска.
