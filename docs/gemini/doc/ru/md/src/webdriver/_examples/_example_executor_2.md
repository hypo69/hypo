# Модуль hypotez/src/webdriver/_examples/_example_executor_2.py

## Обзор

Этот модуль содержит примеры использования класса `ExecuteLocator` для выполнения различных задач тестирования с использованием Selenium WebDriver.  Примеры включают получение данных из HTML, взаимодействие с элементами страницы (нажатия, ввод текста), и обработку потенциальных ошибок.  Модуль демонстрирует как работать с разными типами локаторов и атрибутов, а также как обрабатывать возможные исключения, связанные с выполнением локаторов.


## Использование

### Импорт необходимых библиотек

## Примеры использования `ExecuteLocator`

### `execute_locator`

**Описание**: Функция для выполнения локатора и получения результата.

**Параметры**:
- `locator_data` (dict): Словарь, содержащий информацию о локаторе.
- `continue_on_error` (bool, optional): Если `True`, выполнение не прерывается при ошибке. По умолчанию `False`.

**Возвращает**:
- `dict | None`: Результат выполнения локатора. Если ошибка и `continue_on_error`=False, возвращает `None`.

**Вызывает исключения**:
- `ExecuteLocatorException`: Если возникает ошибка при выполнении локатора и `continue_on_error`=False.

### `send_message`

**Описание**: Отправляет сообщение в элемент.

**Параметры**:
- `locator_data` (dict): Словарь, содержащий информацию о локаторе.
- `message` (str): Сообщение для отправки.
- `typing_speed` (float, optional): Скорость ввода текста. По умолчанию `0.05`.
- `continue_on_error` (bool, optional): Если `True`, выполнение не прерывается при ошибке. По умолчанию `False`.

**Возвращает**:
- `bool`: Успешность операции.

**Вызывает исключения**:
- `ExecuteLocatorException`: Если возникает ошибка при выполнении локатора и `continue_on_error`=False.



### `evaluate_locator`

**Описание**: Оценивает локатор и возвращает значение атрибута.

**Параметры**:
- `attribute_name` (str): Имя атрибута.

**Возвращает**:
- `str | None`: Значение атрибута.


**Вызывает исключения**:
- `ExecuteLocatorException`: Если возникает ошибка при выполнении локатора.
