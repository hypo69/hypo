# Файл desktop.ini

## Обзор

Данный файл представляет собой файл Windows desktop.ini, используемый для управления отображением и метаданными папок. Он содержит информацию о свойствах папки, таких как значок и состояние отображения. Формат файла - ini.

## Структура

Файл desktop.ini содержит секции, разделенные символом `[`.  В каждой секции хранятся определённые метаданные о папке. 

В данном конкретном случае присутствует две секции:

### `.ShellClassInfo`

Эта секция содержит информацию, связанную с отображением папки.  Например, `IconResource` определяет ресурс иконки для папки.

### `.ViewState`

Эта секция содержит данные о состоянии отображения папки.  Например, `Mode`, `Vid` и `FolderType`  могут описывать различные свойства отображения.
