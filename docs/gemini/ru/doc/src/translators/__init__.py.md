# Модуль `src.translators`

## Обзор

Модуль `src.translators` предназначен для определения и управления режимом работы приложения. В данном случае, модуль устанавливает переменную `MODE` для указания текущего режима работы (разработка или продакшн).

## Переменные

### `MODE`
**Описание**:
Строковая переменная, определяющая текущий режим работы приложения.

**Значение**:
`'dev'` - указывает на режим разработки.
