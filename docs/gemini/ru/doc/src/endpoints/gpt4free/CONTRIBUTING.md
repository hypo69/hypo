# Инструкция по внесению вклада в gpt4free

## Обзор

Этот файл содержит инструкции для разработчиков, желающих внести свой вклад в проект gpt4free. Он описывает шаги, необходимые для добавления поддержки новых веб-сайтов.

## Подробнее

Документ описывает процесс внесения вклада в проект `gpt4free`, включая шаги по реверс-инжинирингу веб-сайтов, добавлению их в список тестов и рефакторингу для интеграции в ядро `gpt4free`.

### Инструкции по внесению вклада

1.  **Реверс-инжиниринг веб-сайта**:
    *   Выберите веб-сайт из списка [sites-to-reverse](https://github.com/xtekky/gpt4free/issues/40).
    *   Проведите реверс-инжиниринг выбранного веб-сайта.

2.  **Добавление в список тестов**:
    *   Добавьте информацию о веб-сайте в директорию [./etc/unittest/](https://github.com/xtekky/gpt4free/tree/main/etc/unittest/).

3.  **Рефакторинг и интеграция**:
    *   Отрефакторите код и добавьте его в директорию [./g4f](https://github.com/xtekky/gpt4free/tree/main/g4f).
