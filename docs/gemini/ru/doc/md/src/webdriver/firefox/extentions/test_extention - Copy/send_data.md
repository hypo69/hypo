# Модуль `send_data.js`

## Обзор

Этот JavaScript-модуль предназначен для сбора данных о загруженной веб-странице (заголовок, URL, содержимое тела) и отправки их на сервер по адресу `http://127.0.0.1/hypotez.online/api/` используя метод POST.


## Функции

### `onPageLoad`

**Описание**: Функция `onPageLoad` собирает данные о странице и отправляет их на сервер.

**Описание работы**: 
1. Считывает заголовок страницы (`document.title`), URL страницы (`window.location.href`) и содержимое тела страницы (`document.body.innerHTML`).
2. Формирует объект `data` с собранными данными.
3. Использует `fetch` для отправки POST запроса на сервер `http://127.0.0.1/hypotez.online/api/` с данными в формате JSON.
4. Обрабатывает ответ сервера с помощью `.then`, обрабатывая успешный ответ и выводя полученный JSON в консоль, а при ошибке выводя сообщение об ошибке в консоль.

**Параметры**:
Нет входных параметров.

**Возвращает**:
Нет возвращаемого значения.

**Обрабатывает исключения**:
- `Error`: Если ответ сервера не успешный (например, при ошибке на сервере).  В этом случае выводится сообщение об ошибке в консоль.  


##  Использование


```javascript
// Добавляем обработчик события для загрузки страницы
window.addEventListener('load', onPageLoad);
```

Эта строка кода добавляет обработчик `onPageLoad` к событию `load` элемента `window`.  Это означает, что функция `onPageLoad` будет вызвана, как только страница будет полностью загружена.
