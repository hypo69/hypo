# Модуль: `src.suppliers.wallashop._experiments.JUPYTER_header`

## Функции

### `start_supplier`

```python
def start_supplier(supplier_prefix: str = 'aliexpress', locale: str = 'en') -> Supplier:
    """ Старт поставщика """
```

**Описание**: Запускает поставщика с заданными параметрами.

**Как работает функция**:
1. Функция принимает два параметра: `supplier_prefix` (префикс поставщика) и `locale` (локаль).
2. Создает словарь `params`, содержащий переданные параметры.
3. Инициализирует и возвращает объект класса `Supplier` с использованием параметров из словаря `params`.

**Параметры**:
- `supplier_prefix` (str, optional): Префикс поставщика. По умолчанию `'aliexpress'`.
- `locale` (str, optional): Локаль. По умолчанию `'en'`.

**Возвращает**:
- `Supplier`: Объект класса `Supplier`, инициализированный с заданными параметрами.

**Примеры**:

```python
supplier = start_supplier(supplier_prefix='wallashop', locale='ru')
```
```python
supplier = start_supplier()
```
```python
supplier = start_supplier(locale='fr')
```
```python
supplier = start_supplier(supplier_prefix='ozon', locale='en')
