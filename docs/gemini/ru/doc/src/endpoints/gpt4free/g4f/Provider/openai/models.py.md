# Модуль определения моделей OpenAI

## Обзор

Модуль содержит определения моделей, используемых в OpenAI, включая модели для обработки текста и изображений. Он задает списки моделей по умолчанию, а также предоставляет полный список всех доступных моделей.

## Подробнее

Этот модуль важен для конфигурации и выбора моделей, используемых в других частях проекта `hypotez`, особенно в компонентах, взаимодействующих с API OpenAI. Он позволяет легко управлять списком доступных моделей и выбирать нужную модель для конкретной задачи.

## Переменные

### `default_model`

**Описание**: Модель текста по умолчанию, используемая в OpenAI.

### `default_image_model`

**Описание**: Модель изображений по умолчанию, используемая в OpenAI.

### `image_models`

**Описание**: Список моделей, предназначенных для обработки изображений.

### `text_models`

**Описание**: Список моделей, предназначенных для обработки текста.

### `vision_models`

**Описание**: Список моделей, поддерживающих обработку как текста, так и изображений.

### `models`

**Описание**: Объединенный список всех доступных моделей, включающий как текстовые, так и графические модели.
