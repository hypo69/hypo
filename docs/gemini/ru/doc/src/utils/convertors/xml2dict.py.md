# Модуль `xml2dict`

## Обзор

Модуль `xml2dict` предоставляет утилиты для преобразования XML-данных в словари. Он включает функции для разбора XML-строк и преобразования деревьев XML-элементов в словарные представления.

## Функции

### `_parse_node`

**Описание**: Разбирает XML-узел в словарь.

**Параметры**:
- `node` (ET.Element): XML-элемент для разбора.

**Возвращает**:
- `dict | str`: Словарь, представляющий XML-узел, или строку, если у узла нет атрибутов или дочерних элементов.

### `_make_dict`

**Описание**: Создает новый словарь с тегом и значением.

**Параметры**:
- `tag` (str): Имя тега XML-элемента.
- `value` (any): Значение, связанное с тегом.

**Возвращает**:
- `dict`: Словарь с именем тега в качестве ключа и значением в качестве значения словаря.

### `xml2dict`

**Описание**: Разбирает XML-строку в словарь.

**Параметры**:
- `xml` (str): XML-строка для разбора.

**Возвращает**:
- `dict`: Словарь, представляющий XML.

### `ET2dict`

**Описание**: Преобразует дерево XML-элементов в словарь.

**Параметры**:
- `element_tree` (ET.Element): Дерево XML-элементов.

**Возвращает**:
- `dict`: Словарь, представляющий дерево XML-элементов.
