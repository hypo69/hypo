# Документация для Telegram-бота поиска фильмов

## Обзор

Этот документ описывает Telegram-бота, предназначенного для поиска ссылок на бесплатный просмотр фильмов и сериалов. Бот использует библиотеку Aiogram, сервисы Google для ассоциативного поиска, а также библиотеку Кинопоиска и сервис [w2.kpfr/wiki](https://w2.kpfr.wiki/). В боте реализована защита от флудинга с использованием middlewares, а для парсинга используется BeautifulSoup.

## Подробней

Этот код представляет собой Telegram-бота, который помогает пользователям находить ссылки на бесплатный просмотр фильмов и сериалов. Он использует несколько сторонних библиотек и сервисов для обеспечения функциональности, включая Aiogram для управления ботом, Google для поиска, Кинопоиск для информации о фильмах и BeautifulSoup для парсинга веб-страниц. Защита от флуда реализована с помощью промежуточного программного обеспечения (middlewares).

## Быстрый старт

### 1. Клонирование проекта

Склонируйте проект на свой компьютер, используя следующую команду:

```shell
git clone <адрес_репозитория>
```

### 2. Создание `.env` файла

Создайте файл `.env` в корневой директории проекта и заполните его в следующем формате:

```
TOKEN=123456789 # Токен вашего telegram бота
```

Замените `123456789` на токен вашего Telegram-бота.

### 3. Создание и активация виртуального окружения

Если у вас нет виртуального окружения, создайте его с помощью следующей команды:

```shell
python -m venv venv
```

Активируйте виртуальное окружение:

```shell
venv\Scripts\activate.bat
```

### 4. Установка зависимостей

Обновите `pip` и установите необходимые зависимости:

```shell
pip install --upgrade pip
```

```shell
pip install -r requirements.txt
```

### 5. Запуск бота

Запустите бота, выполнив скрипт `run.py`:

```shell
python run.py
```
