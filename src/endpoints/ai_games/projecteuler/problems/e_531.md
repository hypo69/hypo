# Задача No 531
## Китайская теорема об остатках, часть 3
## Условия
Пусть $CRT(a, n, b, m)$ - это наименьшее неотрицательное решение $x$ системы сравнений:
$x \equiv a \pmod{n}$
$x \equiv b \pmod{m}$.

Например, $CRT(2, 3, 3, 5) = 8$.
Также пусть $S(p, q, r) = \sum_{a=0}^{p-1} \sum_{b=0}^{q-1} CRT(a, p, b, q) \pmod{r}$.

Например, $S(3, 5, 100) = 260$.
Найдите $S(10^7 + 1, 10^7 + 2, 10^9)$.
