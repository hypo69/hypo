# hypotez/src/goog/quickstart.py

Этот скрипт демонстрирует базовый способ использования API Apps Script.  Он создает новый проект Apps Script, загружает в него файлы и выводит URL проекта.

**Описание:**

Скрипт использует Python API Google Apps Script, чтобы:

1. **Авторизоваться:**  Получает токен доступа от пользователя, сохраняя его в файле `token.json` для последующего использования.  Этот файл должен быть создан в первый раз при запуске.  Использует `google-auth-oauthlib` и `google-auth-httplib2` для удобной авторизации. Обратите внимание на `SCOPES` — это необходимые разрешения, запрошенные у пользователя.
2. **Создать проект:** Создает новый проект в Apps Script с заданным названием.
3. **Загрузить файлы:**  Загружает два файла: `hello.js` (JavaScript код) и `appsscript.json` (файл манифеста).
4. **Вывести URL:** Выводит URL проекта Apps Script, позволяя пользователю открыть созданный проект.

**Важные моменты:**

* **`credentials.json`:**  Файл `credentials.json` содержит секретные ключи приложения.  **Не размещайте этот файл в репозитории!**  Создайте его в отдельном месте, например, в папке `config`.
* **`token.json`:**  Файл `token.json` хранит токены доступа и обновления.  Он создается автоматически.
* **`SCOPES`:**  Определяет требуемые разрешения для доступа к API.
* **Обработка ошибок:**  Использует блок `try...except` для обработки возможных ошибок API (например, `HttpError`).
* **Загрузка кода:** Код и манифест Apps Script (функция `helloWorld` и файл манифеста) жестко закодированы в скрипте.  Можно изменить этот код, чтобы соответствовать вашим потребностям.
* **Использование `gs`:**  Скрипт использует объект `gs` из модуля `__init__.py`.  Это указывает на необходимость наличия данного модуля и соответствующей структуры папок в проекте.

**Как использовать:**

1. **Установка зависимостей:** Установите необходимые библиотеки:
   ```bash
   pip install google-api-python-client google-auth-httplib2 google-auth-oauthlib
   ```
2. **Создание `credentials.json`:** Получите ключи приложения Google Cloud Platform и создайте файл `credentials.json` с соответствием настройкам.
3. **Запуск:** Запустите скрипт из командной строки:
   ```bash
   python quickstart.py
   ```
4. **Авторизация:**  Скрипт попросит вас авторизоваться в приложении Google.
5. **Результат:** Вы увидите URL созданного проекта Apps Script в консоли.


**Возможные проблемы:**

* **Отсутствующий файл `credentials.json`:** Убедитесь, что файл `credentials.json` существует и содержит корректные ключи.
* **Неверные разрешения:** Проверьте, что в настройках приложения Google Apps Script разрешены требуемые `SCOPES`.
* **Ошибка API:** Проверьте сообщение об ошибке API для более точной диагностики проблемы.


**Улучшения:**

* **Управление ошибками:**  Более подробная обработка ошибок с выводом сообщений об ошибках.
* **Ввод данных:**  Возможность ввода имени проекта и других данных.
* **Модульность:** Вынести функции в отдельные модули для большей организованности.
* **Документация:** Добавить более подробную документацию к параметрам и функциям.
