# Модуль для авторизации поставщика HB

## Обзор

Модуль `login.py` предназначен для осуществления авторизации поставщика HB. Он содержит функции и классы, необходимые для выполнения входа в систему поставщика.

## Подробнее

Этот модуль является частью системы `hypotez` и отвечает за автоматизацию процесса входа в систему HB. Он использует модуль логирования `src.logger.logger` для записи информации о процессе входа в систему и возможных ошибках.

## Функции

### `login(s)`

```python
def login(s) -> bool:
    """ Функция логин. 
   @param
        s - Supplier
    @returns
        True if login else False

   """
    return True
```

**Назначение**: Осуществляет попытку входа в систему поставщика.

**Параметры**:
- `s`: Объект `Supplier`, содержащий информацию, необходимую для входа в систему.

**Возвращает**:
- `bool`: `True`, если вход в систему выполнен успешно, `False` в противном случае.

**Как работает функция**:
1. Функция `login` принимает объект `s` типа `Supplier` в качестве параметра.
2. Внутри функции происходит попытка входа в систему поставщика с использованием данных, содержащихся в объекте `s`.
3. В текущей реализации функция всегда возвращает `True`, что означает успешный вход в систему.

**Примеры**:

```python
from src.suppliers.hb.login import login

class Supplier:
    def __init__(self):
        pass

s = Supplier()
result = login(s)
print(result)  # Вывод: True
