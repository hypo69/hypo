# Модуль hypotez/src/utils/iso/__init__.py

## Обзор

Этот модуль предоставляет константу `MODE` для определения режима работы.

## Константы

### `MODE`

**Описание**: Константа, хранящая строковое значение режима работы (`'dev'` по умолчанию).

**Значение**: `'dev'`

**Примечание**:  Это значение может быть изменено для переключения между различными режимами работы.
