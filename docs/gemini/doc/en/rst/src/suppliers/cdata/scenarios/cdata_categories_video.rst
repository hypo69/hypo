cdata_categories_video.json
============================

This file defines scenarios for different types of NVIDIA graphics cards, including Tesla, Quadro, and general NVIDIA GPUs.  Each scenario has details like brand, URL, checkbox status, activity status, condition, and associated PrestaShop categories.
