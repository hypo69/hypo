# Модуль hypotez/src/utils/ftp.py

## Обзор

Модуль `ftp.py` предоставляет интерфейс для взаимодействия с FTP-серверами. Он включает функции для отправки, получения и удаления файлов с FTP-сервера.

## Функции

### `write`

**Описание**: Отправляет файл на FTP-сервер.

**Параметры**:
- `source_file_path` (str): Путь к файлу, который необходимо отправить.
- `dest_dir` (str): Целевой каталог на FTP-сервере.
- `dest_file_name` (str): Имя файла на FTP-сервере.

**Возвращает**:
- bool: `True`, если файл успешно отправлен, `False` в противном случае.

**Примеры**:
```python
>>> success = write('local_path/to/file.txt', '/remote/directory', 'file.txt')
>>> print(success)
True
```

**Обрабатываемые исключения**:
- Любые исключения при установлении соединения с FTP-сервером или передаче файла будут перехвачены и записаны в лог.


### `read`

**Описание**: Получает файл с FTP-сервера.

**Параметры**:
- `source_file_path` (str): Путь, куда будет сохранен файл локально.
- `dest_dir` (str): Каталог на FTP-сервере, где находится файл.
- `dest_file_name` (str): Имя файла на FTP-сервере.

**Возвращает**:
- Union[str, bytes, None]: Содержимое файла, если запрос успешен, `None` в противном случае.

**Примеры**:
```python
>>> content = read('local_path/to/file.txt', '/remote/directory', 'file.txt')
>>> print(content)
b'Some file content'
```

**Обрабатываемые исключения**:
- Любые исключения при установлении соединения с FTP-сервером или получении файла будут перехвачены и записаны в лог.


### `delete`

**Описание**: Удаляет файл с FTP-сервера.

**Параметры**:
- `source_file_path` (str): Путь к файлу локально (не используется).
- `dest_dir` (str): Каталог на FTP-сервере, где находится файл.
- `dest_file_name` (str): Имя файла на FTP-сервере.

**Возвращает**:
- bool: `True`, если файл успешно удален, `False` в противном случае.

**Примеры**:
```python
>>> success = delete('local_path/to/file.txt', '/remote/directory', 'file.txt')
>>> print(success)
True
```

**Обрабатываемые исключения**:
- Любые исключения при установлении соединения с FTP-сервером или удалении файла будут перехвачены и записаны в лог.

## Конфигурация соединения

Параметры подключения к FTP-серверу хранятся в словаре `_connection`.  Значения в данном словаре должны быть предварительно настроены.


## Модули и зависимости

- `src.logger`: Модуль для логирования ошибок и сообщений.
- `typing`: Модуль для указания типов данных.
- `ftplib`: Модуль для работы с протоколом FTP.
- `pathlib`: Модуль для работы с путями к файлам.
