<h1>Документация модуля</h1>

<h2>Обзор</h2>
<p>Этот модуль содержит описание формата документации для Python файлов, предназначенное для генерации HTML-документации. Он описывает структуру и требования к комментариям в Python коде для автоматической генерации документации.</p>

<h2>Формат документации</h2>
<p>Для каждого Python файла необходимо создать документацию в формате HTML, следуя указанным требованиям.</p>

<h3>Формат комментариев для функций и классов</h3>
<pre><code>python
def function(param: str, param1: Optional[str | dict | str] = None) -> dict | None:
    """
    Args:
        param (str): Описание параметра `param`.
        param1 (Optional[str | dict | str], optional): Описание параметра `param1`. По умолчанию `None`.

    Returns:
        dict | None: Описание возвращаемого значения. Возвращает словарь или `None`.

    Raises:
        SomeError: Описание ситуации, при которой возникает исключение `SomeError`.
    """
</code></pre>

<h3>Обработка исключений</h3>
<p>В блоках обработки исключений использовать <code>ex</code> вместо <code>e</code>.</p>

<h3>Структура документации</h3>
<p>Документация должна включать:</p>
<ul>
  <li>Заголовок модуля</li>
  <li>Краткое описание содержимого</li>
  <li>Раздел "Функции"</li>
  <li>Раздел "Классы"</li>
  <li>Для каждой функции и класса подробное описание, включая параметры, возвращаемые значения, исключения.</li>
  <li>Использовать уровни заголовков <code><h1></code>, <code><h2></code>, <code><h3></code>, <code><h4></code></li>
  <li>Список параметров (<code><ul></code>) в формате <code>параметр (тип): описание</code></li>
  <li>Список возвращаемых значений (<code><ul></code>) в формате <code>возвращаемое значение (тип): описание</code></li>
  <li>Список исключений (<code><ul></code>) в формате <code>исключение: описание</code></li>
  <li>Примеры использования (если применимо) в формате блоков кода <code><pre><code>python ... </code></pre></code></li>
</ul>

<p><strong>Возвращаемые значения</strong>:</p>

</ul>
