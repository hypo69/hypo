### `__init__`

**Описание**: Конструктор класса `Gemini`. Инициализирует инстанс модели Gemini.

**Параметры**:

- `api_key` (str): Ключ API для доступа к сервису Gemini.  Обязательный параметр.
- `model_name` (Optional[str], optional): Имя модели Gemini. По умолчанию `None`.
- `generation_config` (Optional[Dict], optional): Конфигурация для генерации. По умолчанию `None`.
- `system_instruction` (Optional[str], optional): Системная инструкция, которая будет передана в модель при ее инициализации и остается неизменной в течение всего сеанса работы с инстансом. По умолчанию `None`.

## Инструкции

### Системная инструкция (`system_instruction`)

**Описание**: Указание, которое передается в модель при ее инициализации и остается неизменным в течение всего инстанса.  Не может быть изменено во время работы с инстансом.

### Командная инструкция (`command_instruction`)

**Описание**: Указание, которое передается перед каждым запросом к модели.  Обычно дополняет системную инструкцию.  Инструкцию можно изменять от запроса к запросу.

**Местоположение**:

- Глобальные инструкции хранятся в папке `src/ai/prompts`.
- Инструкции для запросов обычно находятся в папке `instruction` у клиента.
- Клиенты прописываются в `src.endpoints`.
- Глобальные инструкции также могут храниться на клиенте.


**Примеры названий файлов инструкций**:

```
instruction_doc_writer_html_en.md
instruction_code_checker_en.md
instruction_code_checker_he.md
instruction_code_checker_ru.md
instruction_code_explainer_en.md
instruction_code_explainer_ru.md
```
