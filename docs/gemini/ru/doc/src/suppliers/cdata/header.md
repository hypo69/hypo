# Модуль hypotez/src/suppliers/cdata/header.py

## Обзор

Данный модуль содержит функции для определения корневого каталога проекта, загрузки настроек из файла `settings.json` и получения документации из файла `README.MD`.  Он также определяет переменные, хранящие информацию о проекте (название, версия, автор и т.д.).


## Функции

### `set_project_root`

**Описание**:  Находит корневой каталог проекта, начиная с текущего каталога и поднимаясь по дереву каталогов.  Останавливается на первом каталоге, содержащем один из указанных файлов-маркеров (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или каталогов, используемых для идентификации корневого каталога проекта.


**Возвращает**:
- `Path`: Путь к корневому каталогу проекта, если он найден, иначе - текущий каталог.

**Обрабатываемые исключения**:
- Нет.

## Переменные

### `__root__`

**Описание**:  Переменная, хранящая путь к корневому каталогу проекта.  Инициализируется функцией `set_project_root()`.

**Тип**: `Path`

### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженные из файла `src/settings.json`.

**Тип**: `dict`

### `doc_str`

**Описание**: Строка, содержащая текст документации проекта, загруженная из файла `src/README.MD`.

**Тип**: `str`


### `__project_name__`

**Описание**: Имя проекта, получаемое из настроек или устанавливаемое по умолчанию как `hypotez`.

**Тип**: `str`

### `__version__`

**Описание**: Версия проекта, получаемая из настроек или устанавливаемая по умолчанию как пустая строка.

**Тип**: `str`


### `__doc__`

**Описание**: Документация проекта, загруженная из файла `README.MD` или пустая строка по умолчанию.

**Тип**: `str`


### `__details__`

**Описание**: Подробности о проекте.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта, получаемый из настроек или устанавливаемый по умолчанию как пустая строка.

**Тип**: `str`

### `__copyright__`

**Описание**: Авторские права проекта, получаемый из настроек или устанавливаемый по умолчанию как пустая строка.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка на страницу для пожертвований на поддержку разработчиков.

**Тип**: `str`
