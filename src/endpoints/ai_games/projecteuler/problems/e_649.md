# Задача No 649
## Совершенные числа на интервале
## Условия

Натуральное число называется совершенным, если оно равно сумме своих собственных делителей (не включая само число). Например, 28 - совершенное число, поскольку 28 = 1 + 2 + 4 + 7 + 14.

Обозначим через S(n) количество совершенных чисел в интервале [1, n]. Известно, что S(10) = 1, S(1000) = 2, и S(10<sup>6</sup>) = 3.

Также обозначим через F(a, b) количество совершенных чисел в интервале [a, b]. Например, F(1, 10) = 1, F(10, 1000) = 1, и F(1000, 10000) = 1.

Обозначим через G(n) сумму F(1, k), для k = 1, 2, ..., n.
Например, G(10) = 1, G(100) = 2, G(1000) = 5, и G(10000) = 9.

Найдите G(10<sup>14</sup>).
