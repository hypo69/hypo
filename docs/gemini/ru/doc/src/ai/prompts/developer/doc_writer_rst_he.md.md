# Документация для `prompt`

## Обзор

Этот модуль содержит описание требований к документации кода. Описываются необходимые стили и структура документации в формате Markdown для Python-файлов.

## Формат документации

- Документация должна быть в формате Markdown (.md).
- Каждый файл должен начинаться с заголовка и краткого описания его содержимого.
- Для документирования классов и функций использовать стандартный формат, включая описание параметров, возвращаемых значений и вызываемых исключений.
- Использовать `ex` вместо `e` в блоках обработки исключений.

## Форматирование документации

Использовать правильный синтаксис Markdown для заголовков, списков и ссылок.

### Функции

#### `function_name`

**Описание**: Краткое описание функции.

**Параметры**:
- `param` (str): Описание параметра `param`.
- `param1` (Optional[str | dict | str], optional): Описание параметра `param1`. По умолчанию `None`.

**Возвращает**:
- `dict | None`: Описание возвращаемого значения.

**Вызывает исключения**:
- `SomeError`: Описание ситуации, в которой возникает исключение `SomeError`.

## Заголовки разделов

Использовать заголовки первого уровня (`#`), второго уровня (`##`), третьего уровня (`###`) и четвёртого уровня (`####`) последовательно.

## Пример файла

```markdown
# Название модуля

## Обзор

Краткое описание назначения модуля.

## Классы

### `ClassName`

**Описание**: Краткое описание класса.

**Методы**:
- `method_name`: Краткое описание метода.
- `method_name`: Краткое описание метода.
**Параметры**:
- `param` (str): Описание параметра `param`.
- `param1` (Optional[str | dict | str], optional): Описание параметра `param1`. По умолчанию `None`.


## Функции

### `function_name`

**Описание**: Краткое описание функции.

**Методы**:
- `method_name`: Краткое описание метода.   
- `method_name`: Краткое описание метода.

**Параметры**:
- `param` (str): Описание параметра `param`.
- `param1` (Optional[str | dict | str], optional): Описание параметра `param1`. По умолчанию `None`.

**Возвращает**:
- `dict | None`: Описание возвращаемого значения.

**Вызывает исключения**:
- `SomeError`: Описание ситуации, в которой возникает исключение `SomeError`.
```
