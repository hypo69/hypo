# Модуль hypotez/src/utils/ftp.py

## Обзор

Данный модуль предоставляет интерфейс для взаимодействия с FTP-серверами. Он включает функции для отправки, получения и удаления файлов с FTP-сервера.

**Назначение**:

Позволяет отправлять и получать медиафайлы (изображения, видео), электронные таблицы и другие файлы с/на FTP-сервер.

**Модули**:

- `helpers` (локальный): Локальные вспомогательные утилиты для операций с FTP.
- `typing`: Типы данных для параметров функций и возвращаемых значений.
- `ftplib`: Предоставляет возможности клиента протокола FTP.
- `pathlib`: Для работы с путями к файлам в файловой системе.

**Функции**:

- `write`: Отправка файла на FTP-сервер.
- `read`: Получение файла с FTP-сервера.
- `delete`: Удаление файла с FTP-сервера.


## Функции

### `write`

**Описание**: Отправляет файл на FTP-сервер.

**Параметры**:
- `source_file_path` (str): Путь к файлу, который нужно отправить.
- `dest_dir` (str): Путь к целевой директории на FTP-сервере.
- `dest_file_name` (str): Имя файла на FTP-сервере.

**Возвращает**:
- `bool`: `True`, если файл успешно отправлен, `False` в противном случае.

**Примеры**:
```python
success = write('local_path/to/file.txt', '/remote/directory', 'file.txt')
print(success)  # Вывод: True
```

**Обрабатывает исключения**:
- `Exception`: Если возникла ошибка при подключении к FTP-серверу или передаче файла. Сообщение об ошибке записывается в лог.


### `read`

**Описание**: Получает файл с FTP-сервера.

**Параметры**:
- `source_file_path` (str): Путь, куда будет сохранён файл локально.
- `dest_dir` (str): Директория на FTP-сервере, где находится файл.
- `dest_file_name` (str): Имя файла на FTP-сервере.

**Возвращает**:
- `Union[str, bytes, None]`: Содержимое файла, если получение прошло успешно, иначе `None`.

**Примеры**:
```python
content = read('local_path/to/file.txt', '/remote/directory', 'file.txt')
print(content)  # Вывод: b'Some file content'
```

**Обрабатывает исключения**:
- `Exception`: Если возникла ошибка при подключении к FTP-серверу или получении файла. Сообщение об ошибке записывается в лог.


### `delete`

**Описание**: Удаляет файл с FTP-сервера.

**Параметры**:
- `source_file_path` (str): Путь к файлу локально (не используется).
- `dest_dir` (str): Директория на FTP-сервере, где находится файл.
- `dest_file_name` (str): Имя файла на FTP-сервере.

**Возвращает**:
- `bool`: `True`, если файл успешно удалён, `False` в противном случае.

**Примеры**:
```python
success = delete('local_path/to/file.txt', '/remote/directory', 'file.txt')
print(success)  # Вывод: True
```

**Обрабатывает исключения**:
- `Exception`: Если возникла ошибка при подключении к FTP-серверу или удалении файла. Сообщение об ошибке записывается в лог.
