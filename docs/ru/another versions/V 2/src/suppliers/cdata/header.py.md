# Модуль `src.suppliers.cdata.header`

## Обзор

Модуль `header.py` предназначен для определения и установки корневой директории проекта, а также для загрузки основных настроек и документации. Он также определяет основные переменные, такие как название проекта, версия и автор.

## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная с каталога текущего файла, и ищет вверх, останавливаясь на первом каталоге, содержащем любой из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple, optional): Имена файлов или каталогов для определения корневого каталога проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневому каталогу, если он найден, иначе - каталог, где расположен скрипт.

## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта.

**Тип**: `Path`

### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженные из файла `settings.json`.

**Тип**: `dict` или `None`

### `doc_str`

**Описание**: Строка, содержащая документацию проекта, загруженную из файла `README.MD`.

**Тип**: `str` или `None`

### `__project_name__`

**Описание**: Название проекта. По умолчанию `hypotez`.

**Тип**: `str`

### `__version__`

**Описание**: Версия проекта. По умолчанию пустая строка `''`.

**Тип**: `str`

### `__doc__`

**Описание**: Документация проекта.

**Тип**: `str`

### `__details__`

**Описание**: Дополнительная информация о проекте. По умолчанию пустая строка `''`.

**Тип**: `str`

### `__author__`

**Описание**: Автор проекта. По умолчанию пустая строка `''`.

**Тип**: `str`

### `__copyright__`

**Описание**: Информация об авторских правах проекта. По умолчанию пустая строка `''`.

**Тип**: `str`

### `__cofee__`

**Описание**: Сообщение с предложением угостить разработчика чашкой кофе для повышения энтузиазма в разработке.

**Тип**: `str`
