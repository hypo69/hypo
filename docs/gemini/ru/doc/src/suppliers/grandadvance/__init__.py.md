# Модуль `src.suppliers.grandadvance`

## Обзор

Модуль `src.suppliers.grandadvance` предоставляет функциональность для работы с поставщиком Grand Advance. Он включает в себя граббер для сбора данных.

## Константы

### `MODE`

- **Описание**: Режим работы модуля. По умолчанию установлено значение `'dev'`.

## Импорты

- Из текущего пакета импортируется класс `Graber` из модуля `graber`.

## Классы

### `Graber`

- **Описание**: Класс `Graber` используется для сбора данных с веб-сайта поставщика Grand Advance.

- **Методы**:
  -  Подробное описание методов класса `Graber` смотрите в документации модуля `graber`.
