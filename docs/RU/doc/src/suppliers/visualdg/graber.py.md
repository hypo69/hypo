# Модуль `src.suppliers.visualdg.graber`

## Обзор

Модуль `src.suppliers.visualdg.graber` представляет собой класс `Graber`, предназначенный для сбора информации о товарах со страницы `visualdg.co.il`. Он наследует функциональность от родительского класса `Graber` и предоставляет возможность переопределения методов обработки полей для нестандартных ситуаций. Модуль также включает в себя механизмы для выполнения предварительных действий перед отправкой запроса к веб-драйверу через декораторы.

## Функции

### `close_pop_up`
```python
def close_pop_up(value: Any = None) -> Callable:
```

**Описание**: Создает декоратор для закрытия всплывающих окон перед выполнением основной логики функции.

**Параметры**:
- `value` (Any, optional): Дополнительное значение для декоратора. По умолчанию `None`.

**Возвращает**:
- `Callable`: Декоратор, оборачивающий функцию.

## Классы

### `Graber`

**Описание**: Класс для операций захвата данных со страницы товаров `visualdg.co.il`.

**Наследует**:
- `Grbr` (родительский класс `Graber` из модуля `src.suppliers.graber`)

**Атрибуты**:
- `supplier_prefix` (str): Префикс поставщика, используемый для идентификации.

#### `__init__`

```python
def __init__(self, driver: Driver):
```
**Описание**: Инициализирует класс `Graber` с указанным веб-драйвером.

**Параметры**:
- `driver` (`Driver`): Экземпляр веб-драйвера, используемый для взаимодействия с веб-страницей.

**Действия**:
- Устанавливает `supplier_prefix` в значение `visualdg`.
- Вызывает конструктор родительского класса `Grbr` с установленным `supplier_prefix` и `driver`.
- Устанавливает `Context.locator_for_decorator` в значение `None`. Это означает, что если декоратор `@close_pop_up` будет использоваться, по умолчанию он не будет пытаться закрыть всплывающее окно с помощью локатора.
