# Модуль `src.suppliers.aliexpress`

## Обзор

Модуль `src.suppliers.aliexpress` предоставляет функциональность для работы с платформой AliExpress. Он включает в себя классы для взаимодействия с API AliExpress, выполнения запросов и управления рекламными кампаниями.

## Классы

### `Aliexpress`

**Описание**: Этот класс обеспечивает взаимодействие с платформой AliExpress.

### `AliApi`

**Описание**: Класс для работы с API AliExpress.

### `AliRequests`

**Описание**: Класс для выполнения HTTP-запросов к API AliExpress.

### `AliCampaignEditor`

**Описание**: Класс для управления рекламными кампаниями AliExpress.

## HTML-генераторы
### `ProductHTMLGenerator`

**Описание**: Класс для генерации HTML-кода для отображения продуктов.

### `CategoryHTMLGenerator`

**Описание**: Класс для генерации HTML-кода для отображения категорий товаров.

### `CampaignHTMLGenerator`

**Описание**: Класс для генерации HTML-кода для отображения рекламных кампаний.
