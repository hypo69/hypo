# Задача No 120
## Остаток от деления (Square Remainders)
## Условия

Пусть r будет остатком от деления (a-1)^n + (a+1)^n на a^2.

Например, если a = 7 и n = 3, то 6^3 + 8^3 = 216 + 512 = 728. 728 mod 49 = 42.
При изменении n, наибольшее значение r для a = 7 составляет 42.

В общем случае, пусть r_max будет наибольшим значением r при изменении n.
Далее, пусть S = sum(r_max) для 3 ≤ a ≤ 1000.

Найдите S.
