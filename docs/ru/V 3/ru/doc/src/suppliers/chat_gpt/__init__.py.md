# Модуль `src.suppliers.chat_gpt`

## Обзор

Модуль `src.suppliers.chat_gpt` предоставляет интеграцию с моделью GPT через Google Sheets. Включает класс `GptGs`, предназначенный для работы с данными из Google Sheets и использования их в запросах к GPT.

## Подробней

Этот модуль обеспечивает возможность извлекать данные из Google Sheets, обрабатывать их и использовать в качестве контекста или параметров для взаимодействия с GPT моделями. Это может быть полезно для автоматизации процессов, требующих анализа данных из таблиц и генерации ответов на основе этих данных.

## Классы

### `GptGs`

**Описание**: Класс `GptGs` предназначен для интеграции с Google Sheets и моделью GPT. Он позволяет извлекать данные из Google Sheets, обрабатывать их и использовать для взаимодействия с GPT моделями.

**Методы**:
- `__init__`: Инициализирует экземпляр класса `GptGs`.

**Параметры**:
- Нет явных параметров, но подразумевается использование параметров конфигурации для доступа к Google Sheets и GPT.

**Примеры**
```python
# Пример создания экземпляра класса GptGs
gpt_gs = GptGs()
```
