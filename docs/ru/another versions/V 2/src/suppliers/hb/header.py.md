# Модуль `header`

## Обзор

Модуль `header.py` предназначен для настройки и получения основных параметров проекта, таких как корневая директория, настройки из файла `settings.json`, а также информации из файла `README.MD`. Он также предоставляет доступ к основным метаданным проекта, таким как название, версия, автор и т.д.

## Функции

### `set_project_root`

**Описание**:
Функция `set_project_root` находит корневую директорию проекта, начиная с директории текущего файла и двигаясь вверх по иерархии. Поиск останавливается на первой директории, содержащей один из указанных файлов-маркеров. Если корневая директория не найдена, возвращается директория, где находится скрипт.

**Параметры**:
- `marker_files` (tuple, optional): Кортеж имен файлов или директорий, используемых для идентификации корневой директории проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории проекта.

## Переменные

### `__root__`
**Описание**:
Путь к корневой директории проекта, полученный с помощью функции `set_project_root`.

### `settings`
**Описание**:
Словарь с настройками проекта, загруженный из файла `settings.json`. Если файл не найден или при его загрузке произошла ошибка, значение будет `None`.

### `doc_str`
**Описание**:
Содержимое файла `README.MD`. Если файл не найден или при его чтении произошла ошибка, значение будет `None`.

### `__project_name__`
**Описание**:
Название проекта, полученное из `settings.json` или `hypotez` по умолчанию.

### `__version__`
**Описание**:
Версия проекта, полученная из `settings.json` или пустая строка по умолчанию.

### `__doc__`
**Описание**:
Содержимое файла README.MD, загруженное в переменную `doc_str`.

### `__details__`
**Описание**:
Пустая строка. Предполагается для хранения дополнительной информации о проекте.

### `__author__`
**Описание**:
Имя автора проекта, полученное из `settings.json` или пустая строка по умолчанию.

### `__copyright__`
**Описание**:
Информация об авторских правах, полученная из `settings.json` или пустая строка по умолчанию.

### `__cofee__`
**Описание**:
Строка с призывом поддержать разработчика чашечкой кофе, полученная из `settings.json` или текст с ссылкой на Boosty по умолчанию.
