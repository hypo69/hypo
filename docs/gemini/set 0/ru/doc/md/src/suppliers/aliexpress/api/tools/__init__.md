## Обзор

Данный модуль предоставляет инструменты для работы с API AliExpress. В нём объединены функции для получения идентификаторов продуктов.

## Функции

### `get_product_id`

**Описание**: Функция `get_product_id` предназначена для получения идентификатора продукта AliExpress.


**Параметры**:

- `param` (str): URL страницы продукта AliExpress.


**Возвращает**:

- `str`: Идентификатор продукта (если найден) или `None`, если идентификатор не удалось получить.


**Вызывает исключения**:

- `ValueError`: Если передан некорректный URL.
- `Exception`:  В случае возникновения других непредвиденных ошибок.

from .get_product_id import get_product_id
