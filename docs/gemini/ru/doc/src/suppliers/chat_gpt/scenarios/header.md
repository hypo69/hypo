# Модуль `hypotez/src/suppliers/chat_gpt/scenarios/header.py`

## Обзор

Этот модуль содержит вспомогательные функции для работы с настройками проекта, включая определение корневой директории проекта, загрузку настроек из файла `settings.json` и чтение документации из файла `README.MD`.  Модуль также предоставляет доступ к переменным, хранящим информацию о проекте: имя проекта, версия, документация, детали проекта, автор, авторские права и ссылка на поддержку разработчика.

## Функции

### `set_project_root`

**Описание**:  Находит корневую директорию проекта, начиная с директории текущего файла. Поиск происходит вверх по дереву директорий, пока не будет найдена директория, содержащая один из файлов-маркеров (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, которые используются для определения корневой директории. По умолчанию используется кортеж `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращает путь к директории, где находится текущий файл.

**Вызывает исключения**:

- Нет.


### Функции (встроенные)

В модуле `header.py` используются несколько встроенных функций Python, в том числе `resolve`, `exists` из `pathlib.Path`.

## Переменные

### `MODE`

**Описание**:  Переменная, хранящая режим работы (`'dev'`).

### `__root__`

**Описание**: Путь к корневой директории проекта. Инициализируется функцией `set_project_root()`.

### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженные из файла `settings.json`.

### `doc_str`

**Описание**: Строка, содержащая текст документации, загруженная из файла `README.MD`.

### `__project_name__`

**Описание**: Имя проекта (получено из `settings.json` или установлено по умолчанию как 'hypotez').

### `__version__`

**Описание**: Версия проекта (получена из `settings.json` или установлена по умолчанию как пустая строка).

### `__doc__`

**Описание**: Документация проекта (получена из `README.MD` или установлена по умолчанию как пустая строка).

### `__details__`

**Описание**: Детали проекта (по умолчанию пустая строка).

### `__author__`

**Описание**: Автор проекта (получен из `settings.json` или установлен по умолчанию как пустая строка).

### `__copyright__`

**Описание**: Авторские права проекта (получен из `settings.json` или установлен по умолчанию как пустая строка).

### `__cofee__`

**Описание**: Ссылка для поддержки разработчика через систему "кофе" (получена из `settings.json` или установлена по умолчанию как ссылка на Boosty).


## Обработка исключений

Модуль использует блоки `try...except` для обработки `FileNotFoundError` и `json.JSONDecodeError`, которые могут возникнуть при чтении файлов `settings.json` и `README.MD`.  В случае ошибки, эти переменные (settings, doc_str) будут пустыми.
