# Модуль grab_lilnks_to_chats

## Обзор

Этот модуль предоставляет функцию для получения ссылок на чаты с использованием веб-драйвера.  Модуль использует данные из файла `chats_list.json` для поиска ссылок.

## Функции

### `get_links`

**Описание**: Функция извлекает ссылки на отдельные чаты.

**Параметры**:

- `d` (Driver): Объект веб-драйвера, необходимый для взаимодействия с веб-страницей.

**Возвращает**:

- `list[str]`: Список ссылок на чаты. Возвращает пустой список, если ссылки не найдены.

**Вызывает исключения**:

- `Exception`: Любые исключения, возникающие во время взаимодействия с веб-страницей или обработки данных.
