# Модуль `header.py`

## Обзор

Модуль `header.py` предназначен для определения и настройки переменных окружения и конфигурации проекта, включая путь к корневой директории, настройки из файла `settings.json`, а также информацию о проекте (название, версия, автор и т.д.).

## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная с каталога текущего файла, ища вверх и останавливаясь в первом каталоге, содержащем любой из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или каталогов для идентификации корневого каталога проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневому каталогу, если он найден. Если корневой каталог не найден, то возвращается путь к каталогу, где расположен скрипт.

## Переменные

### `MODE`

**Описание**: Режим работы приложения. По умолчанию установлен в `'dev'`.

### `__root__`

**Описание**: Путь к корневой директории проекта. Инициализируется функцией `set_project_root`.

### `settings`

**Описание**: Словарь с настройками, загруженными из файла `settings.json`. Если файл не найден или произошла ошибка декодирования JSON, то устанавливается в `None`.

### `doc_str`

**Описание**: Строка с содержимым файла `README.md`. Если файл не найден или произошла ошибка декодирования JSON, то устанавливается в `None`.

### `__project_name__`

**Описание**: Название проекта. По умолчанию `'hypotez'`. Значение берется из `settings`, если оно присутствует.

### `__version__`

**Описание**: Версия проекта. Значение берется из `settings`, если оно присутствует.

### `__doc__`

**Описание**: Содержание документации проекта из `README.md`.

### `__details__`
**Описание**: Детали проекта.

### `__author__`

**Описание**: Автор проекта. Значение берется из `settings`, если оно присутствует.

### `__copyright__`

**Описание**: Информация об авторских правах. Значение берется из `settings`, если оно присутствует.

### `__cofee__`

**Описание**: Сообщение о возможности угостить разработчика кофе. Значение берется из `settings`, если оно присутствует.
