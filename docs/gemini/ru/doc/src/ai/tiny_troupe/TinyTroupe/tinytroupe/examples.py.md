# examples.py

## Обзор

Этот модуль содержит примеры использования библиотеки tinytroupe для создания агентов. Эти примеры можно использовать как есть или немного модифицировать для создания собственных агентов.

## Функции

### `create_oscar_the_architect`

**Описание**:
Создает и возвращает агента по имени Оскар, который является архитектором.

**Возвращает**:
- `TinyPerson`: Объект агента `TinyPerson`.

### `create_lisa_the_data_scientist`

**Описание**:
Создает и возвращает агента по имени Лиза, которая является специалистом по анализу данных.

**Возвращает**:
- `TinyPerson`: Объект агента `TinyPerson`.

### `create_marcos_the_physician`

**Описание**:
Создает и возвращает агента по имени Маркос, который является врачом.

**Возвращает**:
- `TinyPerson`: Объект агента `TinyPerson`.

### `create_lila_the_linguist`

**Описание**:
Создает и возвращает агента по имени Лила, которая является лингвистом.

**Возвращает**:
- `TinyPerson`: Объект агента `TinyPerson`.
