# Модуль borderify.js

## Обзор

Этот скрипт изменяет стиль границы элемента `document.body`, устанавливая для него красную границу толщиной 5 пикселей. Также он выводит диалоговое окно с помощью `alert()`.

## Функции

### `document.body.style.border = "5px solid red";`

**Описание**: Устанавливает стиль границы для элемента `document.body`.

**Параметры**:

*  (Строковый литерал):  "5px solid red". Этот литерал задаёт красную границу толщиной 5 пикселей.


### `alert()`

**Описание**: Выводит диалоговое окно с предупреждением.

**Параметры**:  Функция не имеет явных параметров.


**Возвращает**:  Функция `alert()` не возвращает явных значений.


**Вызывает исключения**:  Возможные исключения не описаны, так как функция `alert()` является встроенной частью JavaScript.
