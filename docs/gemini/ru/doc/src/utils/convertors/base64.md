# Модуль hypotez/src/utils/convertors/base64

## Обзор

Модуль `base64` предоставляет функцию для декодирования Base64-кодированного содержимого и записи его в временный файл с указанным расширением.

## Функции

### `base64_to_tmpfile`

**Описание**: Функция конвертирует Base64-кодированное содержимое в временный файл. Она декодирует Base64-строку и записывает результат в временный файл, используя расширение из имени входного файла. Возвращает путь к созданному временному файлу.

**Параметры**:

- `content` (str): Base64-кодированное содержимое, которое необходимо декодировать и записать во временный файл.
- `file_name` (str): Имя файла, из которого извлекается расширение для временного файла.

**Возвращает**:

- str: Путь к временному файлу.

**Пример использования**:

```python
>>> base64_content = "SGVsbG8gd29ybGQh"  # Base64-кодированное содержимое "Hello world!"
>>> file_name = "example.txt"
>>> tmp_file_path = base64_to_tmpfile(base64_content, file_name)
>>> print(f"Временный файл создан по пути: {tmp_file_path}")
```

**Примечания**:

Функция использует `tempfile.NamedTemporaryFile` для создания временных файлов. Это гарантирует, что созданный файл будет удален после использования, если это не будет явно запрещено.


### `base64encode`

**Описание**: Функция кодирует изображение в Base64.

**Параметры**:

- `image_path` (str): Путь к изображению.

**Возвращает**:

- str: Base64-кодированное представление изображения.

**Примечания**:
- Требуется открыть файл в бинарном режиме ("rb").
- Возвращаемое значение кодируется в utf-8.
