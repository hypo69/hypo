# Модуль `category.py`

## Обзор

Модуль предназначен для сбора данных о категориях и товарах с сайта поставщика Kualastyle. Он включает функции для получения списка категорий, списка товаров в категории и обработки страниц товаров. Модуль использует веб-драйвер для взаимодействия с сайтом поставщика.

## Функции

### `get_list_products_in_category`

**Описание**: Возвращает список URL-адресов товаров со страницы категории. Если необходимо, пролистывает страницы категорий.

**Параметры**:
- `s` (Supplier): Объект поставщика.

**Возвращает**:
- `list[str, str, None]`: Список URL-адресов товаров или `None`, если список пуст.

**Вызывает исключения**:
- Отсутствуют.

### `paginator`

**Описание**: Выполняет пролистывание страниц категории.

**Параметры**:
- `d` (Driver): Объект веб-драйвера.
- `locator` (dict): Словарь с локаторами.
- `list_products_in_category` (list): Список текущих URL товаров.

**Возвращает**:
- `True`: Если пролистывание прошло успешно.
- `None`: Если пролистывание не удалось или не требуется.

**Вызывает исключения**:
- Отсутствуют.

### `get_list_categories_from_site`

**Описание**: Собирает актуальный список категорий с сайта поставщика.

**Параметры**:
- `s`: Объект поставщика.

**Возвращает**:
 - отсутствует

**Вызывает исключения**:
- Отсутствуют.
