# Задача No 591
## Fractured sequence

Рассмотрим последовательность целых чисел $a_n$, определенную как:
$a_1 = 1$
$a_{n+1} = a_n / 2$, если $a_n$ четно
$a_{n+1} = a_n + \lfloor \sqrt{a_n} \rfloor$, если $a_n$ нечетно.

Первые десять членов последовательности таковы: 1, 2, 1, 2, 1, 2, 1, 2, 1, 2.
Очевидно, последовательность в итоге зацикливается.

Пусть $s(N)$ - сумма различных значений в последовательности $a_n$ при $a_1 = N$ до того момента, как последовательность начнет зацикливаться.

Например:
- $s(1)=1$, потому что последовательность равна 1, 2, 1, 2, 1, 2,...
- $s(2)=3$, потому что последовательность равна 2, 1, 2, 1, 2, ...
- $s(3)=7$, потому что последовательность равна 3, 4, 2, 1, 2, 1, 2, ...
- $s(10)=19$, потому что последовательность равна 10, 5, 7, 9, 12, 6, 3, 4, 2, 1, 2, 1, 2,...

Найдите $\sum_{k=1}^{500} s(k^4)$.
