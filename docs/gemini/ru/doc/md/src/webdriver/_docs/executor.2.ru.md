# Модуль executor

## Обзор

Данный модуль предоставляет набор функций для взаимодействия с веб-элементами, основанный на предоставленных локаторах.  Функции позволяют выполнять действия, извлекать данные и отправлять сообщения веб-элементам.

## Оглавление

* [execute_locator](#execute-locator)
* [get_webelement_by_locator](#get-webelement-by-locator)
* [get_attribute_by_locator](#get-attribute-by-locator)
* [send_message](#send-message)
* [get_url](#get-url)

## Функции

### `execute_locator`

**Описание**: Выполняет действия на веб-элементе, используя указанный локатор.  Может выполнять поиск элемента, отправку сообщений или получение атрибутов.

**Параметры**:
- `locator` (dict): Словарь или объект с информацией о локаторе (например, тип поиска и селектор). Необходимый параметр.
- `message` (str, optional): Сообщение для отправки элементу (например, текст для ввода). По умолчанию пустая строка.
- `typing_speed` (float, optional): Скорость набора текста, если отправляется сообщение (в секундах между нажатиями клавиш). По умолчанию 0.0.
- `continue_on_error` (bool, optional): Флаг для продолжения выполнения в случае ошибки. По умолчанию `True`.

**Возвращает**:
- `any`: Результат выполнения локатора, который может быть веб-элементом, списком элементов, значением атрибута или результатом действия.

**Вызывает исключения**:
- Возможны различные исключения, связанные с работой с веб-драйвером (например, `NoSuchElementException`, `ElementNotInteractableException` и т.д.).


### `get_webelement_by_locator`

**Описание**: Находит и возвращает веб-элемент(ы) на странице, используя локатор.

**Параметры**:
- `locator` (dict): Словарь или объект с информацией о локаторе. Необходимый параметр.

**Возвращает**:
- `any`: Один или несколько веб-элементов, найденных по локатору. Может быть списком или отдельным элементом, в зависимости от спецификации локатора.

**Вызывает исключения**:
- Возможны различные исключения, связанные с работой с веб-драйвером (например, `NoSuchElementException` и т.д.).


### `get_attribute_by_locator`

**Описание**: Получает значение атрибута веб-элемента, найденного по локатору.

**Параметры**:
- `locator` (dict): Словарь или объект с информацией о локаторе. Необходимый параметр.
- `message` (str, optional): Сообщение для отправки элементу перед получением атрибута. По умолчанию пустая строка.

**Возвращает**:
- `any`: Значение атрибута веб-элемента. Возвращает `None` в случае ошибки.

**Вызывает исключения**:
- Возможны различные исключения, связанные с работой с веб-драйвером (например, `NoSuchElementException`, `AttributeError` и т.д.).


### `send_message`

**Описание**: Отправляет сообщение (например, текст) веб-элементу, используя локатор.

**Параметры**:
- `locator` (dict): Словарь или объект с информацией о локаторе. Необходимый параметр.
- `message` (str): Сообщение для отправки элементу. Необходимый параметр.
- `typing_speed` (float, optional): Скорость набора текста, если сообщение отправляется по частям (в секундах между нажатиями клавиш). По умолчанию 0.0.
- `continue_on_error` (bool, optional): Флаг для продолжения выполнения в случае ошибки. По умолчанию `True`.


**Возвращает**:
- `bool`: `True`, если сообщение успешно отправлено, или `False`, если произошла ошибка.

**Вызывает исключения**:
- Возможны различные исключения, связанные с работой с веб-драйвером (например, `ElementNotInteractableException`, `NoSuchElementException`, `TimeoutException` и т.д.).


### `get_url`

**Описание**: Загружает HTML-контент с указанного URL-адреса или локального файла.

**Параметры**:
- `url` (str): URL-адрес или путь к файлу для получения HTML-контента. Необходимый параметр.
- `protocol` (str, optional): Протокол для URL (по умолчанию `https://`).

**Возвращает**:
- `bool`: `True`, если контент успешно загружен, или `False`, если произошла ошибка.

**Вызывает исключения**:
- Возможны исключения, связанные с сетевыми ошибками, ошибками обработки файлов и т.д.
