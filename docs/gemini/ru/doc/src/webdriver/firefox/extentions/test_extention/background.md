# Модуль `background.js`

## Обзор

Данный файл JavaScript (`background.js`) служит обработчиком событий для расширения Firefox.  Он содержит код, который запускается в фоне и реагирует на клики по значку расширения в адресной строке. При клике выполняется скрипт `contentScript.js` в активной вкладке.

## Функции

### `browser.browserAction.onClicked.addListener`

**Описание**:  Устанавливает обработчик события `browser.browserAction.onClicked`.  Этот обработчик вызывается каждый раз, когда пользователь нажимает значок расширения.

**Параметры**:

- `(tab)`:  Объект `tab`, содержащий информацию об активной вкладке.

**Возвращает**:
-  Не имеет возвращаемого значения.


### `browser.scripting.executeScript`

**Описание**:  Выполняет скрипт `contentScript.js` в контексте активной вкладки.

**Параметры**:

- `{
    target: { tabId: tab.id },
    files: ["contentScript.js"],
  }`: Объект с настройками для выполнения скрипта.
    - `target.tabId`: Идентификатор активной вкладки, на которой нужно выполнить скрипт.
    - `files`: Массив путей к файлам скриптов, которые нужно выполнить.

**Возвращает**:
- Не имеет возвращаемого значения.

**Вызывает исключения**:
- `TypeError`: Возникает, если `tab.id` или `files` имеют неверный тип.
- `Error`: Возникает при других ошибках во время выполнения скрипта или при проблемах с доступом к вкладке.



## Замечания

Файл `background.js` напрямую зависит от файла `contentScript.js`, который должен быть доступен в расширении.  Отсутствие `contentScript.js` приведёт к ошибке.
