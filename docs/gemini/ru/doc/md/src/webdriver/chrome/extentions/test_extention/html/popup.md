# Файл hypotez/src/webdriver/chrome/extentions/test_extention/html/popup.html

## Обзор

Этот HTML-файл представляет собой всплывающее окно расширения `hypotez`. Он отображает заголовок "hypotez" и инструкцию по использованию расширения.

## Структура файла

Файл `popup.html` содержит:

*   Раздел `head`, где определяется заголовок страницы и стиль `css`.
*   Раздел `body`, где отображается контент страницы, в том числе заголовок "hypotez" и текст с инструкцией по использованию расширения.

## Стиль

CSS-стили в файле задают ширину и отступы для тела страницы.
