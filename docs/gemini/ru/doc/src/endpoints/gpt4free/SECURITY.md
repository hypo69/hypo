# Документация по безопасности

## Обзор

Этот документ содержит информацию о том, как сообщать об уязвимостях в проекте `hypotez`.

## Подробней

Этот файл предоставляет инструкции для пользователей и разработчиков о том, как сообщать о потенциальных проблемах безопасности. Он объясняет, куда обращаться и чего ожидать после отправки отчета.

## Содержание

- [Сообщение об уязвимости](#reporting-a-vulnerability)

## Сообщение об уязвимости

### `Reporting a Vulnerability`

Сообщайте о (предполагаемых) уязвимостях безопасности по адресу https://t.me/xtekky. Вы получите ответ в течение 48 часов. Если проблема будет подтверждена, мы выпустим исправление как можно скорее, в зависимости от сложности, но, как показывает практика, в течение нескольких дней.
