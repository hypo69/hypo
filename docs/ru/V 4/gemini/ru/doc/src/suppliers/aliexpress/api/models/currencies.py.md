# Модуль `currencies`

## Обзор

Модуль содержит класс `Currency`, который определяет константы для различных валют. Он предоставляет удобный способ для работы с валютами в рамках проекта, избегая использования строковых литералов напрямую.

## Подробней

Этот модуль предназначен для централизованного хранения кодов валют, используемых в API AliExpress. Это упрощает поддержку и обновление списка валют, а также обеспечивает согласованность использования валют во всем проекте. Использование констант вместо строковых литералов уменьшает вероятность ошибок и облегчает рефакторинг кода.

## Классы

### `Currency`

**Описание**: Класс, содержащий константы для различных валют.

**Методы**:
- Отсутствуют.

from src.suppliers.aliexpress.api.models.currencies import Currency
