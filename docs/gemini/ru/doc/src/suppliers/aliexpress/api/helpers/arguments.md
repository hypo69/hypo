# Модуль `hypotez/src/suppliers/aliexpress/api/helpers/arguments.py`

## Обзор

Этот модуль предоставляет вспомогательные функции для работы с аргументами, связанными с API AliExpress.  Он содержит функции для преобразования данных в удобный формат, а также для получения идентификаторов продуктов.

## Функции

### `get_list_as_string`

**Описание**: Преобразует список или строку в строку, разделенную запятыми.

**Параметры**:
- `value`: (любой тип) Значение, которое нужно преобразовать.

**Возвращает**:
- `str | None`: Возвращает строку, разделенную запятыми, если `value` является списком, или само значение, если `value` является строкой. Если `value` равно `None`, возвращает `None`.

**Вызывает исключения**:
- `InvalidArgumentException`: Если `value` имеет неподдерживаемый тип данных (не список и не строка).


### `get_product_ids`

**Описание**: Получает список идентификаторов продуктов из списка или строки.

**Параметры**:
- `values`: (str | list) Список или строка с идентификаторами продуктов, разделенными запятыми.

**Возвращает**:
- `list`: Список идентификаторов продуктов.

**Вызывает исключения**:
- `InvalidArgumentException`: Если `values` не является строкой или списком.
