# Модуль `get_relative_path`

## Обзор

Модуль `get_relative_path` демонстрирует использование функции `get_relative_path` для получения относительного пути от текущего файла до указанной директории.

- [Функции](#функции)

## Импорт
В данном модуле импортируются следующие модули:
*   `pathlib.Path` для работы с путями файловой системы.
*   `header` (предположительно кастомный модуль, импорт без использования)
*   `src.utils.path.get_relative_path` для получения относительного пути

## Пример использования

```python
from pathlib import Path
import header
from src.utils.path import get_relative_path

relative_path = get_relative_path(Path(__file__).resolve(), 'hypotez')
print(relative_path)
```
В данном примере:
- `Path(__file__).resolve()` получает абсолютный путь к текущему файлу.
- Функция `get_relative_path` вычисляет относительный путь от текущего файла до директории `hypotez`.
- Полученный относительный путь выводится на экран.

## Функции

### `get_relative_path`

**Описание**:
Функция для получения относительного пути между двумя директориями.
Более подробное описание функции смотрите в документации модуля `src.utils.path`.

**Параметры**:
- `from_path` (Path): Абсолютный путь до директории, относительно которой надо вычислить путь.
- `to_path` (str): Путь до директории, к которой надо вычислить путь.

**Возвращает**:
- `Path`: Относительный путь от `from_path` до `to_path`.
