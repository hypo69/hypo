# Задача No 22
## Имена и суммы
## Условия
Используя текстовый файл, содержащий более пяти тысяч имен, для начала отсортируйте его в алфавитном порядке. Затем, посчитав алфавитную ценность для каждого имени, умножьте ее на порядковый номер в отсортированном списке для определения результата.

К примеру, имя COLIN (алфавитная ценность 3 + 15 + 12 + 9 + 14 = 53) стоит под 938-м номером в списке. Результат для этого имени будет 938 × 53 = 49714.

Какова сумма всех полученных результатов для имен в файле names.txt?
