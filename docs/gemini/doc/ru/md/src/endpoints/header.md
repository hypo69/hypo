# Модуль hypotez/src/endpoints/header.py

## Обзор

Этот модуль отвечает за инициализацию проекта, включая определение корневой директории проекта, загрузку настроек из файла `settings.json`, а также загрузку документации из файла `README.MD`.  Он также определяет переменные, содержащие ключевые атрибуты проекта (имя, версия, описание и т.д.).

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла. Поиск выполняется вверх по дереву директорий, и процесс останавливается на первой директории, содержащей один из указанных маркеров (файлы или директории).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для определения корневой директории проекта.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена, иначе - путь к директории, где находится текущий файл.

**Вызывает исключения**:
- Не вызывает никаких исключений.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root`.

**Тип**: `Path`

### `settings`

**Описание**: Словарь настроек проекта, загруженный из файла `settings.json`.

**Тип**: `dict`

**Значение по умолчанию**: `None`

### `doc_str`

**Описание**: Строковое содержимое файла `README.MD`.

**Тип**: `str`

**Значение по умолчанию**: `None`

### `__project_name__`

**Описание**: Имя проекта. Получается из настроек (`settings`) или имеет значение по умолчанию `hypotez`.

**Тип**: `str`

### `__version__`

**Описание**: Версия проекта. Получается из настроек (`settings`) или имеет значение по умолчанию пустую строку.

**Тип**: `str`

### `__doc__`

**Описание**: Документация проекта (содержимое `README.MD`). Получается из переменной `doc_str`.

**Тип**: `str`

### `__details__`

**Описание**: Подробная информация о проекте (пустая строка по умолчанию).

**Тип**: `str`

### `__author__`

**Описание**: Автор проекта. Получается из настроек (`settings`) или имеет значение по умолчанию пустую строку.

**Тип**: `str`

### `__copyright__`

**Описание**: Авторские права на проект. Получается из настроек (`settings`) или имеет значение по умолчанию пустую строку.

**Тип**: `str`

### `__cofee__`

**Описание**: Ссылка на страницу для пожертвований на кофе разработчику.

**Тип**: `str`

**Значение по умолчанию**: Ссылка на страницу для пожертвований.


## Обработка исключений

В модуле используется обработка исключений `try...except`, чтобы справиться с потенциальными ошибками при чтении файлов `settings.json` и `README.MD`.  В случае `FileNotFoundError` или `json.JSONDecodeError`, соответствующие переменные (`settings`, `doc_str`) устанавливаются в значение по умолчанию.

## Импорты

Модуль использует следующие импорты:

- `sys`: Для доступа к системным переменным.
- `json`: Для работы с JSON-файлами.
- `packaging.version`: Для работы с версиями.
- `pathlib`: Для работы с путями к файлам.
- `src.gs`: Для доступа к переменным и функциям из модуля `gs`.
