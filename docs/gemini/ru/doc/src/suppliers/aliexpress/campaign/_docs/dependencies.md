# Модуль управления кампаниями AliExpress

## Обзор

Данный модуль предоставляет инструменты для управления кампаниями на платформе AliExpress. Он включает в себя функции для редактирования, подготовки и анализа данных кампаний, а также взаимодействие с внешними сервисами, такими как Google Sheets.

## Зависимости

* `gspread` - Для взаимодействия с Google Sheets.
* `pandas` - Для обработки данных.
* `src.settings.gs` - Настройки для доступа к Google Sheets.
* `src.suppliers.aliexpress.AliCampaignGoogleSheet` - Для работы с Google Sheets для кампаний AliExpress.
