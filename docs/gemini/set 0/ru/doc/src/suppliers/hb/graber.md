# Модуль hypotez/src/suppliers/hb/graber.py

## Обзор

Модуль `graber.py` содержит класс `Graber`, предназначенный для сбора данных о товарах с сайта `hb.co.il`. Класс наследуется от базового класса `Graber` из модуля `suppliers`, добавляя специфические методы для обработки данных с целевого сайта.  Реализует асинхронный захват данных с веб-страницы товара. Предоставляет методы для извлечения различных полей товара, таких как название, описание, цена и другие. Включает механизм предварительной обработки (через декоратор `close_pop_up`), позволяющий выполнять действия до основного запроса к веб-драйверу.
