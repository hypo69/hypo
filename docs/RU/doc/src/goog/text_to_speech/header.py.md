# `header.py`

## Обзор

Модуль `header.py` содержит основные настройки и информацию о проекте, включая версию, имя проекта, автора и пути к файлам настроек. Он также определяет функцию для определения корневой директории проекта и загрузки настроек из файла `settings.json`.

## Константы

### `MODE`
   - **Описание**: Режим работы приложения. По умолчанию `dev`.
   - **Тип**: `str`

## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная с каталога текущего файла, и останавливается на первом каталоге, содержащем любой из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple, optional): Кортеж с именами файлов или каталогов для определения корня проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневому каталогу, если он найден, иначе каталог, где расположен скрипт.

**Пример использования**:
```python
root_path = set_project_root()
print(root_path)
```

## Глобальные переменные

### `__root__`

-   **Описание**: Путь к корневой директории проекта.
-   **Тип**: `Path`

### `settings`
-   **Описание**: Словарь с настройками, загруженными из файла `settings.json`.
-   **Тип**: `dict` или `None`

### `doc_str`
-   **Описание**: Строка с содержимым файла `README.MD`.
-   **Тип**: `str` или `None`

### `__project_name__`
-   **Описание**: Имя проекта. Загружается из `settings.json` или устанавливается в `hypotez` по умолчанию.
-   **Тип**: `str`

### `__version__`
-   **Описание**: Версия проекта. Загружается из `settings.json` или устанавливается в пустую строку по умолчанию.
-   **Тип**: `str`

### `__doc__`
-   **Описание**: Содержимое документации проекта, загруженной из `README.MD`.
-   **Тип**: `str`

### `__details__`
-   **Описание**: Детальная информация о проекте. На данный момент пустая строка.
-   **Тип**: `str`

### `__author__`
-   **Описание**: Автор проекта. Загружается из `settings.json` или устанавливается в пустую строку по умолчанию.
-   **Тип**: `str`

### `__copyright__`
-   **Описание**: Информация об авторских правах. Загружается из `settings.json` или устанавливается в пустую строку по умолчанию.
-   **Тип**: `str`

### `__cofee__`
-   **Описание**: Сообщение о поддержке разработчика. Загружается из `settings.json` или устанавливается в значение по умолчанию.
-   **Тип**: `str`
