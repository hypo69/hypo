# BOUNCE

## סקירה כללית

משחק "הכדור הקופץ" הוא אנימציה פשוטה המדמה את תנועת הכדור הקופץ מהגבולות העליונים והתחתונים של המסך. המשתמש יכול ללחוץ על מקש כלשהו כדי לעצור את האנימציה. גובה הכדור משתנה בצורה סינוסואידית.
