# Описание полей продукта

## Обзор

Этот документ содержит список полей, которые используются для описания товаров в системе. Каждое поле представляет собой атрибут или характеристику продукта.

## Оглавление

- [Обзор](#обзор)
- [Список полей](#список-полей)

## Список полей

### `additional_delivery_times`
    
**Описание**: Дополнительное время доставки.

### `additional_shipping_cost`
    
**Описание**: Дополнительная стоимость доставки.

### `advanced_stock_management`
    
**Описание**: Управление запасами.

### `affiliate_short_link`
    
**Описание**: Короткая партнерская ссылка.

### `affiliate_summary`
    
**Описание**: Партнерская сводка.

### `affiliate_summary_2`
    
**Описание**: Партнерская сводка 2.

### `affiliate_text`
    
**Описание**: Партнерский текст.

### `affiliate_image_large`
    
**Описание**: Большое партнерское изображение.

### `affiliate_image_medium`
    
**Описание**: Среднее партнерское изображение.

### `affiliate_image_small`
    
**Описание**: Маленькое партнерское изображение.

### `associations`
    
**Описание**: Связанные товары.

### `available_date`
    
**Описание**: Дата доступности.

### `available_for_order`
    
**Описание**: Доступно для заказа.

### `available_later`
    
**Описание**: Будет доступно позже.

### `available_now`
    
**Описание**: Доступно сейчас.

### `cache_default_attribute`
    
**Описание**: Кеш атрибута по умолчанию.

### `cache_has_attachments`
    
**Описание**: Кеш наличия вложений.

### `cache_is_pack`
    
**Описание**: Кеш признака, является ли товар набором.

### `condition`
    
**Описание**: Состояние товара (новый, б/у и т.д.).

### `customizable`
    
**Описание**: Возможность кастомизации.

### `date_add`
    
**Описание**: Дата добавления.

### `date_upd`
    
**Описание**: Дата обновления.

### `delivery_in_stock`
    
**Описание**: Время доставки товара в наличии.

### `delivery_out_stock`
    
**Описание**: Время доставки товара, которого нет в наличии.

### `depth`
    
**Описание**: Глубина.

### `description`
    
**Описание**: Полное описание товара.

### `description_short`
    
**Описание**: Краткое описание товара.

### `ean13`
    
**Описание**: Штрихкод EAN13.

### `ecotax`
    
**Описание**: Экологический налог.

### `height`
    
**Описание**: Высота.

### `how_to_use`
    
**Описание**: Инструкция по применению.

### `specification`
    
**Описание**: Спецификация товара.

### `id_category_default`
    
**Описание**: ID категории по умолчанию.

### `id_default_combination`
    
**Описание**: ID комбинации по умолчанию.

### `id_default_image`
    
**Описание**: ID изображения по умолчанию.

### `locale`
    
**Описание**: Локаль (язык).

### `id_manufacturer`
    
**Описание**: ID производителя.

### `id_product`
    
**Описание**: ID товара.

### `id_shop_default`
    
**Описание**: ID магазина по умолчанию.

### `id_shop`
    
**Описание**: ID магазина.

### `id_supplier`
    
**Описание**: ID поставщика.

### `id_tax`
    
**Описание**: ID налога.

### `id_type_redirected`
    
**Описание**: ID типа редиректа.

### `indexed`
    
**Описание**: Индексирован для поиска.

### `ingredients`
    
**Описание**: Ингредиенты.

### `is_virtual`
    
**Описание**: Виртуальный товар.

### `isbn`
    
**Описание**: Номер ISBN.

### `link_rewrite`
    
**Описание**:  Ссылка для SEO.

### `location`
    
**Описание**: Местоположение на складе.

### `low_stock_alert`
    
**Описание**: Оповещение о низком запасе.

### `low_stock_threshold`
    
**Описание**: Порог низкого запаса.

### `meta_description`
    
**Описание**: Мета-описание.

### `meta_keywords`
    
**Описание**: Мета-ключевые слова.

### `meta_title`
    
**Описание**: Мета-заголовок.

### `minimal_quantity`
    
**Описание**: Минимальное количество для заказа.

### `mpn`
    
**Описание**: Номер производителя.

### `name`
    
**Описание**: Название товара.

### `online_only`
    
**Описание**: Только онлайн продажа.

### `on_sale`
    
**Описание**: На распродаже.

### `out_of_stock`
    
**Описание**: Статус отсутствия на складе.

### `pack_stock_type`
    
**Описание**: Тип управления запасами для комплектов.

### `price`
    
**Описание**: Цена товара.

### `product_type`
    
**Описание**: Тип товара.

### `quantity_discount`
    
**Описание**: Скидка от количества.

### `redirect_type`
    
**Описание**: Тип редиректа.

### `reference`
    
**Описание**: Артикул товара.

### `show_condition`
    
**Описание**: Показывать состояние.

### `show_price`
    
**Описание**: Показывать цену.

### `state`
    
**Описание**: Состояние товара.

### `supplier_reference`
    
**Описание**: Артикул поставщика.

### `text_fields`
    
**Описание**: Текстовые поля.

### `unit_price_ratio`
    
**Описание**: Коэффициент цены за единицу.

### `unity`
    
**Описание**: Единица измерения.

### `upc`

### `uploadable_files`
    
**Описание**: Доступные для загрузки файлы.

### `visibility`
    
**Описание**: Видимость товара.

### `volume`

### `weight`

### `wholesale_price`

### `width`
    
**Описание**: Ширина.
    
### `local_saved_image`
    
**Описание**: Локально сохраненное изображение.
    
### `local_saved_video`
    
**Описание**: Локально сохраненное видео.
