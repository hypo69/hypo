# Модуль hypotez/src/ai/dialogflow/header.py

## Обзор

Данный модуль отвечает за определение корневого пути к проекту `hypotez`. Он используется для корректной работы импортов, настроенных относительно этого пути. В будущем предполагается перенести информацию о корневом пути в системные переменные.

## Функции

### `set_project_root`

**Описание**:  Находит корневую директорию проекта, начиная с текущей директории файла. Поиск ведется вверх по иерархии директорий, и прекращается при обнаружении директории, содержащей один из указанных файлов-маркеров (например, `pyproject.toml`, `requirements.txt` или `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для определения корневой директории проекта. По умолчанию (`pyproject.toml`, `requirements.txt`, `.git`).

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращает директорию, в которой находится сам скрипт.

**Вызывает исключения**:

- Нет указанных исключений.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root`.

**Тип**: `Path`

### `settings`

**Описание**: Словарь со значениями настроек проекта, загруженный из файла `settings.json`.

**Тип**: `dict`

### `doc_str`

**Описание**: Текст файла README, загруженный из файла `README.MD`.

**Тип**: `str`


### `__project_name__`

**Описание**: Название проекта. Получает значение из `settings`, по умолчанию `hypotez`.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта. Получает значение из `settings`, по умолчанию пустая строка.

**Тип**: `str`


### `__doc__`

**Описание**: Документация проекта. Получает значение из `doc_str`, по умолчанию пустая строка.

**Тип**: `str`


### `__details__`

**Описание**: Дополнительные сведения о проекте. По умолчанию пустая строка.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта. Получает значение из `settings`, по умолчанию пустая строка.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права на проект. Получает значение из `settings`, по умолчанию пустая строка.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка на платежную систему для поддержки разработчиков. Получает значение из `settings`, по умолчанию адрес.

**Тип**: `str`


**Примечание**:  Обратите внимание на обработку исключений (`try...exept`) для файлов `settings.json` и `README.MD`.  Если эти файлы не найдены или некорректно отформатированы, соответствующие переменные будут содержать значение по умолчанию.
