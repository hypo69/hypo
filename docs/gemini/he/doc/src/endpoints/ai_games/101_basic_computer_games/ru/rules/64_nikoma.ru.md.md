## סקירה כללית

**NICOMA** הוא משחק אסטרטגיה שבו השחקן שולט בצי ספינות, במטרה להגן על כוחותיו ולהשמיד את ספינות האויב. שדה הקרב מוצג כרשת, שבה השחקן יכול למקם את ספינותיו, לבחור אסטרטגיות התקפה ולהגיב למהלכי היריב. מטרת המשחק היא להשמיד לחלוטין את צי היריב, תוך שמירה על כמה שיותר מספינותיו.

## הוראות שלב אחר שלב למימוש

### 1. אתחול המשחק
   - התוכנית מציגה הודעת פתיחה ומסבירה את הכללים:
     ```
     ברוכים הבאים ל-NICOMA - קרב ימי!
     אתם שולטים בצי ספינות. מטרת המשחק היא להשמיד את צי האויב.
     המשחק מתנהל על רשת בגודל 10x10.
     ```
   - השחקן בוחר את מצב המשחק:
     1. יחיד (שחקן נגד מחשב).
     2. שני שחקנים (שחקן נגד שחקן אחר).
   - התוכנית יוצרת שדה בגודל 10x10 לכל שחקן.
   - כל שחקן ממקם את ספינותיו (למשל, 5 ספינות באורך שונה) בשדה שלו, על ידי הגדרת קואורדינטות התחלתיות. התוכנית בודקת שהספינות לא מצטלבות ולא יוצאות מגבולות השדה.

### 2. תהליך המשחק העיקרי

#### 2.1. תור השחקן:
   - השחקן מזין קואורדינטות התקפה (למשל, `B5`), כדי לתקוף את התא המצוין בשדה היריב.
   - התוכנית בודקת האם הירי פגע בספינה או החטיא:
     - **פגיעה:** התוכנית מסמנת את הנזק בשדה היריב ומדווחת:
       ```
       פגיעה! ספינת האויב ניזוקה.
       ```
     - **החטאה:** התוכנית מדווחת שהירי החטיא:
       ```
       החטאה!
       ```

#### 2.2. תור היריב:
   - במצב יחיד, המחשב יוצר קואורדינטות אקראיות להתקפה.
   - במצב שני שחקנים, השחקנים מתחלפים בתורות.

#### 2.3. הטבעת ספינה:
   - אם כל התאים של ספינה ניזוקו, היא נחשבת לטבועה. התוכנית מדווחת לשחקן:
     ```
     הטבעתם ספינה של היריב!
     ```

#### 2.4. המשך המשחק:
   - המשחק ממשיך עד שלאחד השחקנים נשארו ספינות לא טבועות.

### 3. סיום המשחק
   - כאשר כל הספינות של אחד השחקנים מושמדות, התוכנית מסיימת את המשחק ומכריזה על המנצח:
     ```
     המנצח: שחקן 1! מזל טוב על הניצחון!
     ```
   - השחקנים יכולים לבחור:
     ```
     האם תרצו לשחק שוב? (כן/לא)
     ```

## דוגמה לתוכנית

1. **תחילת המשחק:**
   ```
   ברוכים הבאים ל-NICOMA!
   בחרו מצב משחק:
   1. יחיד
   2. שני שחקנים
   > 1
   מקמו את הספינות שלכם:
   הזינו קואורדינטות עבור הספינה הראשונה (למשל, A1, A2, A3):
   > A1, A2, A3
   ```

2. **תהליך המשחק:**
   ```
   תורכם! הזינו קואורדינטות להתקפה:
   > B5
   החטאה!

   תור המחשב:
   המחשב תוקף: C3
   פגיעה! הספינה שלכם ניזוקה.

   תורכם! הזינו קואורדינטות להתקפה:
   > C4
   פגיעה! ספינת האויב ניזוקה.
   ```

3. **סיום המשחק:**
   ```
   כל ספינות האויב הושמדו!
   המנצח: שחקן 1!
   האם תרצו לשחק שוב? (כן/לא):
   > לא
   תודה ששיחקתם!
   ```

## מגבלות אפשריות

- גודל השדה קבוע (10x10).
- הספינות ממוקמות בתוך השדה ולא יכולות לחצות אחת את השנייה.
- התוכנית צריכה לטפל בקלט שגוי (למשל, קואורדינטות מחוץ לשדה).

## מימוש

המשחק יכול להיות מיושם ב-Python באמצעות היכולות הבאות:
- **רשימות** לאחסון מידע על מיקום הספינות בשדה.
- **מודול `random`** ליצירת מהלכי המחשב.
- **לולאות ותנאים** לטיפול במהלך המשחק ובדיקת מצב הספינות.

## שיפורים מומלצים

- להוסיף אפשרות לבחור גודל שדה (למשל, 15x15).
- ליישם אלגוריתם מורכב יותר למהלכי המחשב (למשל, התקפה מכוונת לאחר פגיעה).
- להוסיף ממשק גרפי באמצעות `pygame` או ספרייה אחרת להמחשת השדה.
