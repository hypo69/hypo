# Модуль hypotez/src/suppliers/aliexpress/gapi

## Обзор

Данный модуль предоставляет функции и классы для взаимодействия с API AliExpress.  Включает в себя информацию о версии и конфигурацию режима работы (MODE).


## Конфигурация

### `MODE`

**Описание**:  Переменная, определяющая режим работы модуля.

**Значение**:  `'dev'` (по умолчанию).


## Версия

### `__version__`

**Описание**:  Переменная, содержащая текущую версию модуля.  Получена из файла `version.py`.


### `__doc__`

**Описание**:  Строковое описание модуля.  Получено из файла `version.py`.


### `__details__`

**Описание**:  Дополнительная информация о модуле.  Получена из файла `version.py`.


## Импорты

### `from packaging.version import Version`

**Описание**: Импортирует класс `Version` из модуля `packaging.version` для работы с версиями.


### `from .version import __version__, __doc__, __details__`

**Описание**: Импортирует переменные `__version__`, `__doc__` и `__details__` из модуля `version.py` внутри папки `gapi`.
