# Документация для файла `ksp_categories_headphones_hyperx.json`

## Обзор

Файл `ksp_categories_headphones_hyperx.json` содержит JSON-конфигурацию для сценариев обработки наушников HyperX типа "In-ear Bud" на сайте KSP. Он определяет URL, условия, соответствие категориям PrestaShop и другие параметры для извлечения данных о товарах.

## Структура JSON

### Раздел "scenarios"

Содержит сценарии для обработки различных категорий товаров.

#### Сценарий "In-ear Bud"

Описывает сценарий для обработки наушников HyperX типа "In-ear Bud".

##### Параметр "brand"

**Описание**: Бренд товара.

**Значение**: "HYPER-X".

##### Параметр "url"

**Описание**: URL страницы категории на сайте KSP.

**Значение**: "https://ksp.co.il/web/cat/242..1250..2767".

##### Параметр "checkbox"

**Описание**: Флаг для управления состоянием чекбокса.

**Значение**: `false`.

##### Параметр "active"

**Описание**: Флаг, указывающий, активен ли сценарий.

**Значение**: `true`.

##### Параметр "condition"

**Описание**: Условие, которому должен соответствовать товар.

**Значение**: "new".

##### Раздел "presta_categories"

Содержит соответствия категорий PrestaShop для данного сценария.

###### Раздел "template"

Определяет соответствие между именем бренда и категорией PrestaShop.

##### Параметр "hyper-x"

**Описание**: Категория PrestaShop для бренда HyperX.

**Значение**: "HEADPHONES BT In-ear Bud".
