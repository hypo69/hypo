# Модуль `hypotez/src/suppliers/aliexpress/campaign/html_generators.py`

## Обзор

Этот модуль предоставляет классы для генерации HTML-контента для рекламных кампаний AliExpress.  Он позволяет создавать HTML-страницы для отдельных продуктов, категорий продуктов и обзора всей кампании.  Модуль использует библиотеку `html` для экранирования HTML-символов и `pathlib` для работы с путями к файлам.

## Классы

### `ProductHTMLGenerator`

**Описание**: Класс для генерации HTML-страниц отдельных продуктов.

**Методы**

#### `set_product_html`

**Описание**: Создает HTML-файл для отдельного продукта.

**Параметры**:

- `product` (SimpleNamespace):  Данные о продукте, которые должны быть включены в HTML (объект SimpleNamespace).
- `category_path` (str | Path): Путь для сохранения HTML-файла.

**Возвращает**:
    -  Не имеет возвращаемого значения. Файл сохраняется в указанный путь.

**Обрабатываемые исключения**:
    - Нет.


### `CategoryHTMLGenerator`

**Описание**: Класс для генерации HTML-страниц категорий продуктов.

**Методы**

#### `set_category_html`

**Описание**: Создает HTML-файл для категории, включая список продуктов.

**Параметры**:

- `products_list` (list[SimpleNamespace] | SimpleNamespace): Список продуктов или отдельный продукт, который должен быть включен в HTML.
- `category_path` (str | Path): Путь к файлу категории.

**Возвращает**:
    -  Не имеет возвращаемого значения. Файл сохраняется в указанный путь.

**Обрабатываемые исключения**:
    - Нет.

### `CampaignHTMLGenerator`

**Описание**: Класс для генерации HTML-страницы обзора кампании.

**Методы**

#### `set_campaign_html`

**Описание**: Создает HTML-файл для обзора кампании, отображая все категории.

**Параметры**:

- `categories` (list[str]): Список названий категорий.
- `campaign_path` (str | Path): Путь для сохранения HTML-файла кампании.

**Возвращает**:
    -  Не имеет возвращаемого значения. Файл сохраняется в указанный путь.

**Обрабатываемые исключения**:
    - Нет.
