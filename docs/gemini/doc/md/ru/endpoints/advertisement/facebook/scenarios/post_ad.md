# Модуль post_ad

## Обзор

Этот модуль предоставляет функцию `post_ad` для публикации рекламного сообщения в группах Facebook. Он использует Selenium для взаимодействия с веб-сайтом Facebook и предоставляет обработку ошибок и логирование.

## Определения

### `MODE`

**Описание**: Переменная, определяющая режим работы (например, `'development'`).

### `locator`

**Описание**: Объект `SimpleNamespace`, содержащий локаторы элементов страницы Facebook, загруженные из файла JSON.

### `fails`

**Описание**: Счётчик неудачных попыток.

## Функции

### `post_ad`

**Описание**: Отправляет заголовок и описание рекламного сообщения на Facebook.

**Параметры**:
- `d` (Driver): Экземпляр драйвера, используемый для взаимодействия с веб-страницей.
- `message` (SimpleNamespace): Объект `SimpleNamespace`, содержащий данные сообщения, в том числе описание, путь к изображению (если нужно) и т.д.

**Возвращает**:
- `bool`: `True`, если сообщение успешно опубликовано; `None` в противном случае.

**Вызывает исключения**:
- Возможные исключения, возникающие при взаимодействии с драйвером или при обработке изображений.  (Специфичные исключения должны быть указаны с помощью `Raises`).

**Детали**:
Функция `post_ad` последовательно выполняет следующие шаги:
1. Отправляет заголовок сообщения (используя `post_message_title`).
2. Если указан путь к изображению, отправляет его на публикацию (используя `upload_post_media`).
3. Опубликовывает сообщение (используя `message_publish`).
4. Возвращает `True`, если все этапы завершились успешно. В противном случае, возвращает `None` или вызывает соответствующие исключения.


## Использование

Пример использования функции `post_ad`:

```python
from src.webdriver import Driver
from src.endpoints.advertisement.facebook.scenarios import post_ad
from types import SimpleNamespace

# Инициализация драйвера
driver = Driver(...)

# Создание объекта с данными сообщения
event = SimpleNamespace(title="Campaign Title", description="Event Description", image_path="path/to/image.jpg")

# Вызов функции post_ad
if post_ad(driver, event):
    print("Message posted successfully!")
else:
    print("Failed to post message.")

```

**Примечание**:  Этот пример предполагает, что у вас есть необходимые импорты и инициализация драйвера.  Необходимо также убедиться в корректности путей к файлам и валидности входных данных.
