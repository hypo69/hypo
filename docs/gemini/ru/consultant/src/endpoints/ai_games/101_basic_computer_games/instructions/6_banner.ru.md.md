Описание игры
--------------------
Программа "Banner" принимает текстовое сообщение от пользователя и отображает его на экране в виде большого баннера. Каждая буква в сообщении представлена соответствующим набором символов, формируя увеличенное изображение буквы.

Основные правила
--------------------
1. Пользователь вводит текстовое сообщение для преобразования в баннер.
2. Программа преобразует каждую букву сообщения в соответствующую матрицу символов.
3. Баннер отображается на экране.

Инструкция для модели ИИ
--------------------------

1. **Инициализация игры:**

   - Создайте массивы символов для каждой буквы алфавита (A-Z) и цифр (0-9).
   - Каждая буква должна быть представлена в виде матрицы символов (например, 5 строк на 5 столбцов).
     Например:
     .. code-block:: python

        BANNER_LETTERS = {
            'A': [
                '  *  ',
                ' * * ',
                '*****',
                '*   *',
                '*   *'
                ],
            'B': [
                '**** ',
                '*   *',
                '**** ',
                '*   *',
                '**** '
                ],
           ...
        }

2. **Основной цикл игры:**

   - Код запрашивает у игрока сообщение для преобразования в баннер.
   - Для каждой буквы в сообщении код находит соответствующую матрицу символов.
   - Код выводит каждую строку матрицы символов для всех букв сообщения.

3. **Обработка ввода игрока:**

   - Код проверяет, что ввод игрока содержит только буквы и цифры (без пробелов и специальных символов).
   - В случае некорректного ввода, код просит игрока повторить ввод.
   -  Если вводе содержит пробелы, то обрабатывать их, как пробелы между символами.

4. **Создание баннера:**

   - Код для каждой буквы в сообщении находит соответствующую матрицу символов.
   - Код выводит каждую строку матрицы символов для всех букв сообщения.
   -  Код добавляет пробелы между буквами для лучшей читаемости.

5. **Завершение игры:**

   - Код после вывода баннера спрашивает игрока, хочет ли он создать еще один баннер.
   - Если игрок соглашается, код повторяет процесс.

   Пример диалога:
   --------------------
   .. code-block::

    Компьютер: Введите сообщение для создания баннера: HELLO

    Компьютер:
    H   H EEEEE L     L     OOOO
    H   H E     L     L     O    O
    HHHHH EEEEE L     L     O    O
    H   H E     L     L     O    O
    H   H EEEEE LLLLL LLLLL OOOO

    Компьютер: Хотите создать еще один баннер? (да/нет): да

    WW WW O    O R R   L     D    D

    Компьютер: Спасибо за игру!

   Примечания
   --------------------
   - Модель ИИ должна корректно обрабатывать ввод пользователя и предоставлять понятные инструкции.
   - Модель должна уметь создавать матрицы символов для каждой буквы и цифры.
   - Модель должна быть готова к повторным запускам игры, если пользователь захочет создать еще один баннер.
