Module hypotez.src.ai.prompts.developer.improve_code_en
====================================================

This module provides instructions and guidelines for automatically generating documentation for Python code in reStructuredText (rst) format, specifically targeting Sphinx documentation generation. It outlines the expected input formats (Python code, Markdown, or dictionaries), output formatting requirements, project structure considerations, and best practices for creating clear and comprehensive Sphinx documentation.  The module aims to handle various input types, including Python code files, markdown files, or dictionaries.  When provided with a Python code file, the module will analyze imports and existing project structure for consistent documentation.
