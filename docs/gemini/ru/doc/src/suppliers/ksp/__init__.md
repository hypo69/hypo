# Модуль hypotez/src/suppliers/ksp/__init__.py

## Обзор

Этот модуль содержит инициализацию для поставщика данных KSP.  Он предоставляет доступ к классу `Graber` для сбора данных.

## Содержание

### Переменные

#### `MODE`

**Описание**:  Переменная, определяющая режим работы.  В данном случае, она установлена в 'dev'.

**Тип**: `str`

#### `Graber`

**Описание**: Класс для сбора данных.  Поддерживает различные методы работы с данными KSP.

**Тип**: `class`


## Классы

### `Graber`

**Описание**: Класс для сбора данных.  Подробная документация находится в файле `graber.py`.
