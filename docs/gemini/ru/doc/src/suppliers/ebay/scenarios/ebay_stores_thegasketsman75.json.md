## Обзор

Данный JSON-файл содержит конфигурационные данные для магазина eBay с идентификатором `thegasketsman75`, используемые в системе Hypotez. Он определяет настройки магазина, а также сценарии для обработки товаров, в частности, для наборов прокладок (Gasket KIT).
## Структура JSON

### store
Секция `store` содержит общие данные о магазине.

**Описание:** Общая информация о магазине на eBay.

**Поля:**

-   `store_id` (str): Идентификатор магазина на eBay (в данном случае "thegasketsman75").
-   `supplier_id` (int): Идентификатор поставщика в системе Hypotez (в данном случае 4534).
-  `get store banners` (bool): Флаг, указывающий, нужно ли получать баннеры магазина.
-   `description` (str): Краткое описание магазина ("thegasketsman75 Gasket KIT").
-   `about` (str): Информация "о магазине" - в данном случае пустая строка.
-   `url` (str): URL магазина на eBay ("https://www.ebay.com/str/thegasketsman75").
-   `shop categories page` (str): URL страницы категорий магазина (пустая строка).
-   `shop categories json file` (str): Путь к JSON-файлу с категориями магазина (пустая строка).

### scenarios
Секция `scenarios` содержит настройки для различных сценариев обработки товаров.

**Описание:** Настройки для сценариев обработки товаров магазина.

#### Gasket KIT
Сценарий для наборов прокладок (Gasket KIT).

**Описание**: Настройки для сценария обработки наборов прокладок.
**Поля**:

-   `url` (str): URL магазина на eBay ("https://www.ebay.com/str/thegasketsman75").
-   `active` (bool): Флаг, указывающий, активен ли сценарий (в данном случае `true`).
-   `condition` (str): Состояние товара (в данном случае "new").
-  `presta_categories` (dict): Словарь категорий для Prestashop.
    -   `template` (dict): Шаблон для категорий.
        -   `gasket KIT` (str): Категория, соответствующая наборам прокладок (в данном случае "GASKET KIT").
-   `checkbox` (bool): Флаг, указывающий на наличие чекбокса (в данном случае `false`).
-   `price_rule` (int): Правило ценообразования (в данном случае 1).
