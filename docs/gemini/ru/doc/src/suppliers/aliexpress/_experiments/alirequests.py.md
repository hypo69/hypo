# Модуль `alirequests`

## Обзор

Модуль `alirequests` предназначен для экспериментов с запросами к AliExpress. В текущей версии модуля выполняется открытие страницы AliExpress с использованием веб-драйвера Firefox.

## Подробней

Этот модуль является частью более крупного проекта `hypotez` и используется для автоматизации взаимодействия с платформой AliExpress. Он включает в себя настройку и запуск веб-драйвера для выполнения различных задач, таких как сбор данных, тестирование и автоматизация покупок.

## Функции

### `Driver`

```python
from src.webdriver.driver import Driver, Chrome, Firefox

d = Driver(Firefox)
d.get_url(r"https://www.aliexpress.com")
...
```

**Назначение**: Инициализация и настройка веб-драйвера Firefox для открытия страницы AliExpress.

**Как работает функция**:

1.  **Инициализация драйвера**: Создается экземпляр класса `Driver` с указанием типа браузера `Firefox`.
2.  **Открытие URL**: Используется метод `get_url` для открытия указанного URL (`https://www.aliexpress.com`) в браузере, управляемом драйвером.

Внутри функции происходят следующие действия и преобразования:

A.  Создание экземпляра драйвера `d`.
|
B.  Открытие URL в браузере.

**Примеры**:

```python
from src.webdriver.driver import Driver, Chrome, Firefox

# Инициализация драйвера Firefox и открытие страницы AliExpress
d = Driver(Firefox)
d.get_url(r"https://www.aliexpress.com")
