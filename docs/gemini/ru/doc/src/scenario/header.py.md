# Модуль `header`

## Обзор

Модуль `header.py` содержит основные настройки и метаданные проекта, такие как путь к корню проекта, имя проекта, версию, документацию и информацию об авторе. Он также загружает настройки из файла `settings.json` и документацию из `README.MD`.

## Функции

### `set_project_root`

**Описание**:
Находит корневую директорию проекта, начиная с директории текущего файла, поднимаясь вверх и останавливаясь на первой директории, содержащей любой из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или директорий, которые идентифицируют корень проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена, иначе путь к директории, где расположен скрипт.

## Переменные модуля

### `MODE`

**Описание**:
Режим работы проекта.

**Тип**:
- `str`

**Значение**:
-  `'dev'`

### `__root__`

**Описание**:
Путь к корневой директории проекта.

**Тип**:
- `Path`

**Значение**:
- Определяется функцией `set_project_root`.

### `settings`

**Описание**:
Словарь настроек проекта, загруженный из файла `settings.json`.

**Тип**:
- `dict | None`

**Значение**:
- Загружается из `settings.json` или `None`, если файл не найден или не удалось его декодировать.

### `doc_str`

**Описание**:
Строка документации проекта, загруженная из файла `README.MD`.

**Тип**:
- `str | None`

**Значение**:
- Содержимое `README.MD` или `None`, если файл не найден или не удалось прочитать.

### `__project_name__`

**Описание**:
Имя проекта.

**Тип**:
- `str`

**Значение**:
-  Получается из `settings.json`, в противном случае по умолчанию `'hypotez'`.

### `__version__`

**Описание**:
Версия проекта.

**Тип**:
- `str`

**Значение**:
- Получается из `settings.json`, в противном случае по умолчанию `''`.

### `__doc__`

**Описание**:
Документация проекта.

**Тип**:
- `str`

**Значение**:
- Содержимое `doc_str`, если оно существует, в противном случае `''`.

### `__details__`

**Описание**:
Детали проекта.

**Тип**:
- `str`

**Значение**:
-  Пустая строка `''`.

### `__author__`

**Описание**:
Автор проекта.

**Тип**:
- `str`

**Значение**:
-  Получается из `settings.json`, в противном случае по умолчанию `''`.

### `__copyright__`

**Описание**:
Авторские права проекта.

**Тип**:
- `str`

**Значение**:
- Получается из `settings.json`, в противном случае по умолчанию `''`.

### `__cofee__`

**Описание**:
Сообщение о возможности поддержать разработчика чашкой кофе.

**Тип**:
- `str`

**Значение**:
- Получается из `settings.json` или  по умолчанию  `"Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69"`
