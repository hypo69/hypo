# Модуль hypotez/src/endpoints/emil/header.py

## Обзор

Этот модуль содержит функции для определения корневого каталога проекта и загрузки настроек из файла `settings.json`.  Также он содержит переменные, хранящие информацию о проекте (название, версия, описание и т.д.).

## Функции

### `set_project_root`

**Описание**: Определяет корневой каталог проекта, начиная с текущего файла. Ищет файлы или каталоги-маркеры (например, `pyproject.toml`, `requirements.txt`, `.git`) в родительских директориях, пока не найдет каталог, содержащий один из указанных маркеров. Если корневой каталог не найден, возвращает текущую директорию.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или каталогов, по которым определяется корневой каталог.


**Возвращает**:

- `Path`: Объект `Path` представляющий путь к корневому каталогу проекта. Если корневой каталог не найден, возвращает директорию, где находится файл `header.py`.

**Вызывает исключения**:

-  Никаких исключений не генерирует.

### `get_project_settings`

**Описание**: Функция загрузки настроек из файла `settings.json`

**Параметры**:
-  Не принимает никаких параметров.


**Возвращает**:
-  `dict`: Словарь с настройками проекта, или None, если файл не найден или некорректен.


**Вызывает исключения**:
- `FileNotFoundError`: Если файл `settings.json` не найден.
- `json.JSONDecodeError`: Если содержимое файла `settings.json` не является валидным JSON.


## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта, полученный с помощью функции `set_project_root`.

**Тип**: `Path`


### `settings`

**Описание**: Словарь с настройками проекта, загруженный из `settings.json`.

**Тип**: `dict` или `None`


### `doc_str`

**Описание**: Текст из файла `README.MD`, если он существует.

**Тип**: `str` или `None`


### `__project_name__`

**Описание**: Название проекта, полученное из настроек или установленное по умолчанию.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта, полученная из настроек или установленная по умолчанию.

**Тип**: `str`


### `__doc__`

**Описание**: Документация проекта (из файла `README.MD`), если указана.

**Тип**: `str`


### `__details__`

**Описание**: Дополнительная информация о проекте.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта, полученный из настроек или установленный по умолчанию.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права, полученный из настроек или установленный по умолчанию.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка на спонсорскую платформу для поддержки разработчика.

**Тип**: `str`
