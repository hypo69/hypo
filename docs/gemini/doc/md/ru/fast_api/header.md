# Модуль header

## Обзор

Данный модуль содержит функции для получения корневой директории проекта и доступа к настройкам проекта, хранящимся в файле `settings.json`. Он также обрабатывает возможные ошибки, такие как отсутствие файла `settings.json` или некорректный формат JSON.


## Функции

### `get_project_root`

**Описание**: Находит корневую директорию проекта, начиная от текущего файла, ищет вверх по дереву директорий, и останавливается на первой директории, содержащей один из указанных файлов или директорий.

**Параметры**:
- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, которые используются для определения корневой директории проекта. По умолчанию: `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории проекта, если она найдена, иначе - путь к директории текущего файла.

**Вызывает исключения**:
- Нет


### `get_settings`

**Описание**: Загружает настройки проекта из файла `settings.json`, находящегося в корневой директории проекта.

**Параметры**:
- Нет

**Возвращает**:
- `dict`: Словарь настроек проекта, если файл `settings.json` найден и отформатирован корректно; в противном случае возвращает `None`.

**Вызывает исключения**:
- `FileNotFoundError`: Если файл `settings.json` не найден.
- `json.JSONDecodeError`: Если файл `settings.json` содержит некорректный JSON.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта.

**Тип**: `Path`


### `settings`

**Описание**: Словарь настроек проекта.


**Тип**: `dict | None`

### `doc_str`

**Описание**: Содержимое файла `README.MD`.

**Тип**: `str | None`

### `__project_name__`

**Описание**: Имя проекта, полученное из настроек.

**Тип**: `str`

### `__version__`

**Описание**: Версия проекта, полученная из настроек.

**Тип**: `str`

### `__doc__`

**Описание**: Документация проекта, полученная из файла `README.MD`.

**Тип**: `str`

### `__details__`

**Описание**: Дополнительные детали проекта (пустая строка по умолчанию).

**Тип**: `str`

### `__author__`

**Описание**: Автор проекта, полученный из настроек.

**Тип**: `str`

### `__copyright__`

**Описание**: Авторские права проекта, полученные из настроек.

**Тип**: `str`

### `__cofee__`

**Описание**: Ссылка для поддержки разработчика.

**Тип**: `str`
