# Модуль example_list

Этот модуль содержит примеры работы с списками.

### `example_list`

**Параметры**:

* `data` (list): Список, с которым будет производиться работа.


**Возвращает**:

* `list`: Измененный список.


**Вызывает исключения**:

* Нет.
