# Задача No 377
## Сумма цифр фибоначчи-чисел
## Условия
Числа Фибоначчи определяются как:

* F<sub>0</sub> = 0
* F<sub>1</sub> = 1
* F<sub>n</sub> = F<sub>n-1</sub> + F<sub>n-2</sub>

Сумма цифр числа - это сумма всех цифр в числе. Например, сумма цифр 314159 равна 3+1+4+1+5+9=23.
Пусть S(n) - сумма цифр n-го числа Фибоначчи.
Сумма S(n) для первых нескольких чисел Фибоначчи равна:

| n | F<sub>n</sub> | S(n) |
|---|---|---|
| 0 | 0 | 0 |
| 1 | 1 | 1 |
| 2 | 1 | 1 |
| 3 | 2 | 2 |
| 4 | 3 | 3 |
| 5 | 5 | 5 |
| 6 | 8 | 8 |
| 7 | 13 | 4 |
| 8 | 21 | 3 |
| 9 | 34 | 7 |
| 10 | 55 | 10 |
| 11 | 89 | 17 |
| 12 | 144 | 9 |

Пусть SS(n) - сумма S(k) для k от 0 до n.
Первые значения SS(n) равны:

| n | SS(n) |
|---|---|
| 0 | 0 |
| 1 | 1 |
| 2 | 2 |
| 3 | 4 |
| 4 | 7 |
| 5 | 12 |
| 6 | 20 |
| 7 | 24 |
| 8 | 27 |
| 9 | 34 |
| 10 | 44 |
| 11 | 61 |
| 12 | 70 |

Найдите SS(10<sup>18</sup>). Запишите ответ по модулю 10<sup>9</sup>+7.
