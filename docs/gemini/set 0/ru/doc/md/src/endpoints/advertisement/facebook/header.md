# Модуль hypotez/src/endpoints/advertisement/facebook/header.py

## Обзор

Данный модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`.  Он также предоставляет доступ к документации проекта (README.MD) и различным метаданным, таким как название проекта, версия, автор и т.д.


## Функции

### `set_project_root`

**Описание**:  Находит корневую директорию проекта, начиная с текущего файла и проходя вверх по иерархии директорий. Поиск останавливается на первой директории, содержащей один из файлов-маркеров (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для определения корневой директории проекта. По умолчанию включает ('pyproject.toml', 'requirements.txt', '.git').

**Возвращает**:

- `Path`: Путь до корневой директории проекта, если она найдена. В противном случае возвращается путь к директории, где расположен текущий скрипт.

**Вызывает исключения**:
Не вызывает исключений.


## Переменные

### `__root__`

**Описание**:  Путь к корневой директории проекта. Присваивается результатом вызова функции `set_project_root()`.


### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json`.

**Значение по умолчанию**: `None`

**Замечание**:  Файл `settings.json` должен быть расположен в корне проекта в папке `src`.


### `doc_str`

**Описание**: Строковое представление содержимого файла `README.MD`.


**Значение по умолчанию**: `None`


### `__project_name__`

**Описание**: Название проекта. Получается из словаря `settings`.  Если `settings` или ключ `project_name` отсутствуют, используется значение `'hypotez'`.


### `__version__`

**Описание**: Версия проекта. Получается из словаря `settings`. Если `settings` или ключ `version` отсутствуют, используется значение `''`.


### `__doc__`

**Описание**: Документация проекта. Получается из переменной `doc_str`.  Если `doc_str` не определен, используется пустая строка.


### `__details__`

**Описание**: Подробная информация о проекте. Используется пустая строка по умолчанию.


### `__author__`

**Описание**: Автор проекта. Получается из словаря `settings`. Если `settings` или ключ `author` отсутствуют, используется пустая строка.


### `__copyright__`

**Описание**: Авторские права. Получается из словаря `settings`. Если `settings` или ключ `copyright` отсутствуют, используется пустая строка.


### `__cofee__`

**Описание**: Ссылка для поддержки разработчика. Получается из словаря `settings`.  Если `settings` или ключ `cofee` отсутствуют, используется строка с ссылкой.


**Обработка исключений**:

Обработка `FileNotFoundError` и `json.JSONDecodeError` при чтении `settings.json` и `README.MD`.  В случае ошибки соответствующие переменные не изменяются, сохраняя значение по умолчанию.
