# revai

## Overview

This module, `revai`, is a Python module for interacting with the Rev AI API.  It currently provides basic configurations and constants, likely intended for setting up environment variables or modes (e.g., 'dev', 'prod') for future functionality.


## Constants

### `MODE`

**Description**: A string constant defining the current operating mode.  Currently set to 'dev'.


## Usage Examples

```python
# Example accessing the MODE constant:
from revai import MODE

print(MODE)
```
