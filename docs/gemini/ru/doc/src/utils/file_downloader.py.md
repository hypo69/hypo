# Модуль для скачивания файлов

## Обзор

Модуль содержит функцию `download_file`, которая позволяет скачивать файлы из интернета по указанному URL-адресу и сохранять их локально на диск. Модуль использует библиотеку `requests` для выполнения HTTP-запросов.

## Подробнее

Этот модуль предоставляет простую функцию для скачивания файлов, что может быть полезно для автоматизации загрузки данных или ресурсов из интернета. Он позволяет загружать файлы небольшими частями, что помогает избежать проблем с памятью при работе с большими файлами.

## Функции

### `download_file`

```python
def download_file(url: str, destination: str):
    """
    Функция для скачивания файла по указанному URL и сохранения его на диск.

    Args:
        url (str): URL файла для скачивания.
        destination (str): Путь для сохранения скачанного файла.

    Returns:
        None

    Raises:
        None

    Example:
        >>> file_url = 'https://example.com/path/to/file.txt'
        >>> save_as = 'downloaded_file.txt'
        >>> download_file(file_url, save_as)
        Файл успешно загружен!
    """
```

**Назначение**: Функция `download_file` скачивает файл по указанному URL-адресу и сохраняет его в указанном месте на диске.

**Параметры**:
- `url` (str): URL-адрес файла, который необходимо скачать.
- `destination` (str): Путь к файлу, в который будет сохранен скачанный файл.

**Возвращает**:
- None

**Вызывает исключения**:
- None

**Как работает функция**:
1. Отправляет GET-запрос на сервер с указанным URL.
2. Проверяет, успешен ли запрос (код ответа 200 означает успех).
3. Если запрос успешен, открывает файл для записи в бинарном режиме.
4. Скачивает файл по частям (по 1024 байта), чтобы избежать проблем с памятью при больших файлах.
5. Записывает каждую часть в файл.
6. Выводит сообщение об успешной загрузке или ошибке.

```
A: Отправить GET-запрос
|
B: Проверить статус ответа (200?)
|
C: Открыть файл для записи (в бинарном режиме) - D: Скачать файл по частям (1024 байта) - E: Записать часть в файл
|
F: Вывести сообщение об успехе или ошибке
```

**Примеры**:

```python
file_url = 'https://example.com/path/to/file.txt'
save_as = 'downloaded_file.txt'
download_file(file_url, save_as)
