# Документация для `category.json`

## Обзор

Файл `category.json` содержит JSON-структуру, определяющую локатор для извлечения ссылок на товары со страницы категории.
Этот файл используется для настройки парсера, который извлекает ссылки на товары с использованием указанных параметров.

## Структура JSON

```json
{
  "product_links": {
    "attribute": "href",
    "by": "XPATH",
    "selector": "//span[@data-component-type ='s-product-image']//a",
    "if_list":"first",
    "use_mouse": false, 
    "mandatory": true,
    "timeout":0,
    "timeout_for_event":"presence_of_element_located",
    "event": null
  }
}
```

## Описание полей

### `product_links`

Объект, содержащий параметры для поиска ссылок на товары.

#### `attribute`

**Описание**:
Атрибут HTML-элемента, который нужно извлечь.

**Тип**:
`string`

**Значение**:
`"href"`

#### `by`

**Описание**:
Метод поиска элемента.

**Тип**:
`string`

**Значение**:
`"XPATH"`

#### `selector`

**Описание**:
Строка-селектор, используемый для поиска элемента.

**Тип**:
`string`

**Значение**:
`"//span[@data-component-type ='s-product-image']//a"`

#### `if_list`

**Описание**:
Указывает, как обрабатывать список найденных элементов.

**Тип**:
`string`

**Значение**:
`"first"` - брать первый элемент из списка.

#### `use_mouse`

**Описание**:
Указывает, нужно ли использовать эмуляцию мыши при взаимодействии с элементом.

**Тип**:
`boolean`

**Значение**:
`false`

#### `mandatory`

**Описание**:
Указывает, является ли наличие элемента обязательным.

**Тип**:
`boolean`

**Значение**:
`true`

#### `timeout`

**Описание**:
Время ожидания в секундах перед тем, как поиск элемента будет считаться неудачным.

**Тип**:
`number`

**Значение**:
`0`

#### `timeout_for_event`

**Описание**:
Указывает тип ожидания перед поиском элемента.

**Тип**:
`string`

**Значение**:
`"presence_of_element_located"` - ожидать появления элемента на странице.

#### `event`

**Описание**:
Событие для ожидания (используется в связке с `timeout_for_event`). В данном случае не используется.

**Тип**:
`null`

**Значение**:
`null`
