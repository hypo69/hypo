# Задача No 223
## Почти египетские тройки

## Условия

Рассмотрим тройки целых чисел (a, b, c), удовлетворяющие условию a² + b² = c². Например, (3, 4, 5) - одна такая тройка.

Пусть мы добавим требование a² + b² = c² + 1.  Тройка (a, b, c) будет называться *почти египетской тройкой*. Самый маленький пример такой тройки - (1, 1, 1).  Следующие несколько примеров (сортированные по возрастанию c) следующие:

(1, 1, 1)
(1, 3, 3)
(1, 8, 8)
(2, 2, 3)
(1, 15, 15)
(1, 24, 24)
(2, 6, 7)
(1, 35, 35)
(3, 4, 5)

Сколько существует *почти египетских троек* (a, b, c), у которых c ≤ 25000000?
