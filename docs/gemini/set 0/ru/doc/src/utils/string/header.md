# Модуль `hypotez/src/logger/header.py`

## Обзор

Данный модуль определяет корневой путь к проекту `hypotez`.  Все импорты в проекте строятся относительно этого пути.  В будущем предполагается перенести эту логику в системные переменные.  Модуль предоставляет функцию `set_project_root` для поиска корневого каталога и функцию для загрузки настроек из файла `settings.json` и документации из `README.MD`.

## Функции

### `set_project_root`

**Описание**:  Находит корневой каталог проекта, начиная с каталога текущего файла. Поиск происходит вверх по дереву каталогов до тех пор, пока не будет найден каталог, содержащий один из указанных файлов-маркеров (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или каталогов, которые указывают на корневой каталог проекта. По умолчанию это `('pyproject.toml', 'requirements.txt', '.git')`.


**Возвращает**:

- `Path`: Путь к корневому каталогу проекта, если он найден. В противном случае возвращает каталог, где находится данный скрипт.


**Вызывает исключения**:
- Нет


### Загрузка настроек

**Описание**:  Загружает настройки из файла `settings.json` в корневом каталоге проекта.  Обрабатывает возможные исключения `FileNotFoundError` и `json.JSONDecodeError` в случае отсутствия файла или ошибки декодирования JSON.

**Возвращает**:

- `dict`: Словарь с настройками проекта, загруженный из `settings.json`.  Возвращает `None`, если файл не найден или содержит некорректный JSON.


**Вызывает исключения**:
- `FileNotFoundError`: Если файл `settings.json` не найден.
- `json.JSONDecodeError`: Если файл `settings.json` содержит некорректный JSON.

### Загрузка документации


**Описание**: Загружает текст документации из файла `README.MD` в корневом каталоге проекта. Обрабатывает возможные исключения `FileNotFoundError` и `json.JSONDecodeError` в случае отсутствия файла или ошибки декодирования.


**Возвращает**:

- `str`: Текст документации из `README.MD` или `None`.


**Вызывает исключения**:
- `FileNotFoundError`: Если файл `README.MD` не найден.
- `json.JSONDecodeError`: Если файл `README.MD` содержит некорректный формат.



## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта, полученный с помощью функции `set_project_root()`.

**Тип**: `Path`

### `settings`

**Описание**: Словарь со значениями настроек проекта.

**Тип**: `dict` или `None`

### `doc_str`

**Описание**:  Строка, содержащая текст документации из `README.MD`.

**Тип**: `str` или `None`


### `__project_name__`, `__version__`, `__doc__`, `__details__`, `__author__`, `__copyright__`, `__cofee__`

**Описание**:  Переменные, содержащие значения из загруженных настроек (если доступны). Предварительно заполнены значениями по умолчанию.
