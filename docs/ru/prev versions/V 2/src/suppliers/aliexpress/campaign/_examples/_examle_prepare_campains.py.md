## Примеры использования

### Пример 1: Обработка одной категории кампании

```python
process_campaign_category("SummerSale", "Electronics", "EN", "USD", force=True)
```

**Описание**: Этот пример демонстрирует, как обработать одну категорию ("Electronics") для кампании "SummerSale" на английском языке (EN) и в долларах США (USD), принудительно (force=True).

### Пример 2: Обработка конкретной кампании

```python
process_campaign("WinterSale", categories=["Clothing", "Toys"], language="EN", currency="USD", force=False)
```

**Описание**: В этом примере показано, как обработать конкретную кампанию "WinterSale" для категорий "Clothing" и "Toys" на английском языке (EN) в долларах США (USD), без принудительного режима (force=False).

### Пример 3: Обработка всех кампаний

```python
process_all_campaigns(language="EN", currency="USD", force=True)
```

**Описание**: Этот пример показывает, как обработать все доступные кампании на английском языке (EN) в долларах США (USD), принудительно (force=True).

## Вспомогательный код

```python
campaigns_directory = Path(gs.path.google_drive,'aliexpress','campaigns')
campaign_names = get_directory_names(campaigns_directory)
languages = {'EN': 'USD', 'HE': 'ILS', 'RU': 'ILS'}
```

**Описание**:
- `campaigns_directory`: Путь к директории с кампаниями на Google Drive.
- `campaign_names`: Список имен подкаталогов в директории кампаний.
- `languages`: Словарь, сопоставляющий языки с валютами.
