# Модуль `hypotez/src/suppliers/aliexpress/api/models/request_parameters.py`

## Обзор

Этот модуль содержит определения констант для параметров запросов API AliExpress.  Он определяет типы продуктов, способы сортировки и типы ссылок.

## Оглавление

- [Классы](#классы)
    - [ProductType](#producttype)
    - [SortBy](#sortby)
    - [LinkType](#linktype)

## Классы

### `ProductType`

**Описание**: Класс `ProductType` содержит константы для типов продуктов на AliExpress.

**Константы**:

- `ALL`: Представляет все типы продуктов.
- `PLAZA`: Представляет продукты из раздела PLAZA.
- `TMALL`: Представляет продукты из раздела TMALL.


### `SortBy`

**Описание**: Класс `SortBy` содержит константы для различных способов сортировки результатов поиска.

**Константы**:

- `SALE_PRICE_ASC`: Сортировка по цене продажи в порядке возрастания.
- `SALE_PRICE_DESC`: Сортировка по цене продажи в порядке убывания.
- `LAST_VOLUME_ASC`: Сортировка по последнему объёму продаж в порядке возрастания.
- `LAST_VOLUME_DESC`: Сортировка по последнему объёму продаж в порядке убывания.


### `LinkType`

**Описание**: Класс `LinkType` содержит константы для типов ссылок.

**Константы**:

- `NORMAL`: Обычный тип ссылки.
- `HOTLINK`: Тип ссылки "горячая ссылка".
