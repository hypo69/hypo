# Модуль `src.suppliers.aliexpress.api`

## Обзор

Модуль предоставляет API для работы с Aliexpress. Он включает в себя обертку для API Aliexpress, модели данных и информацию о версии.

## Подробней

Этот модуль является частью проекта `hypotez` и предназначен для упрощения взаимодействия с API Aliexpress. Он предоставляет классы и функции для выполнения запросов к Aliexpress и обработки ответов. Модуль включает в себя информацию о версии, что позволяет отслеживать изменения и зависимости в проекте.
