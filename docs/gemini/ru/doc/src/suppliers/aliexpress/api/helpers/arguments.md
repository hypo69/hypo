# Модуль `hypotez/src/suppliers/aliexpress/api/helpers/arguments.py`

## Обзор

Модуль `arguments.py` содержит вспомогательные функции для работы с аргументами, используемыми в API для AliExpress.  Он предоставляет инструменты для обработки строк, списков и получения идентификаторов продуктов.

## Функции

### `get_list_as_string`

**Описание**: Преобразует значение в строку, разделенную запятыми, если это список. Если значение `None`, возвращает `None`. В противном случае, если значение - строка, возвращает ее напрямую. В противном случае, генерирует исключение `InvalidArgumentException`.

**Параметры**:

- `value`: Значение, которое необходимо преобразовать (может быть списком или строкой, или `None`).

**Возвращает**:

- Строка, разделенная запятыми, если `value` - список.  Строковое значение `value`, если оно строка. `None` если `value` равно `None`.

**Вызывает исключения**:

- `InvalidArgumentException`: Если `value` не является ни списком, ни строкой, ни `None`.


### `get_product_ids`

**Описание**: Преобразует входные данные в список идентификаторов продуктов. Разделяет строку, если входные данные представляют собой строку. Если вход не является ни строкой, ни списком, выбрасывает исключение. Иначе, вызывает функцию `get_product_id` для каждого элемента входного списка и возвращает список полученных идентификаторов.

**Параметры**:

- `values`: Входные данные (строка или список).

**Возвращает**:

- Список идентификаторов продуктов (`list`).

**Вызывает исключения**:

- `InvalidArgumentException`: Если `values` не является ни строкой, ни списком.
