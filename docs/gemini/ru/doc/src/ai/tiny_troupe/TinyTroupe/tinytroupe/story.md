# Модуль story

## Обзор

Модуль `story` предоставляет инструменты для создания историй на основе симуляций. Он позволяет описывать окружения и агентов, формировать цели для генерации истории и включать контекст прошлых взаимодействий.

## Классы

### `TinyStory`

**Описание**: Класс `TinyStory` отвечает за создание и управление историей.  История может быть связана с окружением или агентом.

**Методы**:

- `__init__`: Инициализирует историю, принимая на вход окружение, агента, цель, контекст и параметры для отображения взаимодействия.
    - **Параметры**:
        - `environment` (`TinyWorld`, необязательно): Окружение, в котором происходит история. По умолчанию `None`.
        - `agent` (`TinyPerson`, необязательно): Агент в истории. По умолчанию `None`.
        - `purpose` (`str`, необязательно): Цель создания истории. По умолчанию `"Be a realistic simulation."`.
        - `context` (`str`, необязательно): Текущий контекст истории. По умолчанию `""`.
        - `first_n` (`int`, необязательно): Количество первых взаимодействий для включения в историю. По умолчанию `10`.
        - `last_n` (`int`, необязательно): Количество последних взаимодействий для включения в историю. По умолчанию `20`.
        - `include_omission_info` (`bool`, необязательно): Включать ли информацию об опущенных взаимодействиях. По умолчанию `True`.
    - **Возвращает**: `None`
    - **Исключения**:
        - `Exception`: Возникает, если предоставлены как `environment`, так и `agent`.
        - `Exception`: Возникает, если не предоставлены ни `environment`, ни `agent`.

- `start_story`: Запускает создание новой истории.
    - **Параметры**:
        - `requirements` (`str`, необязательно): Требования к истории. По умолчанию `"Start some interesting story about the agents."`.
        - `number_of_words` (`int`, необязательно): Желаемое количество слов в истории. По умолчанию `100`.
        - `include_plot_twist` (`bool`, необязательно): Включать ли повороты сюжета. По умолчанию `False`.
    - **Возвращает**: `str`: Созданная часть истории.
    - **Исключений**: Не указаны.

- `continue_story`: Продолжает существующую историю.
    - **Параметры**:
        - `requirements` (`str`, необязательно): Требования к продолжению истории. По умолчанию `"Continue the story in an interesting way."`.
        - `number_of_words` (`int`, необязательно): Желаемое количество слов в продолжении. По умолчанию `100`.
        - `include_plot_twist` (`bool`, необязательно): Включать ли повороты сюжета. По умолчанию `False`.
    - **Возвращает**: `str`: Созданная часть истории.
    - **Исключений**: Не указаны.

- `_current_story`: Получает текущую историю.
    - **Параметры**: Не принимает.
    - **Возвращает**: `str`: Текущее состояние истории, включая взаимодействия.
    - **Исключений**: Не указаны.
