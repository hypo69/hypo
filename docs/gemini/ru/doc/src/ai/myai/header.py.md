# Модуль `src.ai.myai.header`

## Обзор

Модуль `src.ai.myai.header` предназначен для определения основных параметров проекта, таких как корневой каталог, настройки из `settings.json`, а также версии и прочей информации, необходимой для работы проекта.

## Функции

### `set_project_root`

**Описание**:
Находит корневой каталог проекта, начиная с директории текущего файла, и останавливается на первой директории, содержащей маркерные файлы.

**Параметры**:
- `marker_files` (tuple, optional): Кортеж с именами файлов или директорий, которые идентифицируют корень проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена, или путь к директории, где находится скрипт.

## Переменные

### `__root__`
**Описание**: 
Путь к корневой директории проекта. Инициализируется функцией `set_project_root`.

### `settings`
**Описание**: 
Словарь с настройками проекта, загруженный из файла `settings.json`. Может быть `None`, если файл не найден или имеет неверный формат.

### `doc_str`
**Описание**: 
Строка с содержимым файла `README.MD`. Может быть `None`, если файл не найден или имеет неверный формат.

### `__project_name__`
**Описание**: 
Имя проекта, взятое из настроек (`settings.json`). По умолчанию `'hypotez'`.

### `__version__`
**Описание**: 
Версия проекта, взятая из настроек (`settings.json`). По умолчанию `''`.

### `__doc__`
**Описание**: 
Документация проекта, загруженная из `README.MD`. По умолчанию `''`.

### `__details__`
**Описание**: 
Детальная информация о проекте. По умолчанию `''`.

### `__author__`
**Описание**: 
Автор проекта, взятый из настроек (`settings.json`). По умолчанию `''`.

### `__copyright__`
**Описание**: 
Информация об авторских правах, взятая из настроек (`settings.json`). По умолчанию `''`.

### `__cofee__`
**Описание**: 
Сообщение о возможности угостить разработчика кофе, взятое из настроек (`settings.json`). По умолчанию `"Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69"`.
