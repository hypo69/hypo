# Описание локаторов магазина Morlevi

## Обзор

Файл `store.json` содержит JSON-объект с локаторами элементов страницы магазина Morlevi. Этот файл предназначен для использования в автоматизированных тестах или при парсинге веб-страницы. Он содержит информацию о CSS-селекторах, идентификаторах и других атрибутах, которые позволяют однозначно идентифицировать элементы на веб-странице.

## Структура файла

Файл `store.json` представляет собой JSON-объект, который содержит пары "ключ-значение". Ключи - это наименования локаторов, а значения - соответствующие им CSS-селекторы, XPath или другие идентификаторы элементов.
Объект можно представить в виде следующей структуры:

```json
{
  "locator_name_1": "locator_value_1",
  "locator_name_2": "locator_value_2",
  ...
  "locator_name_n": "locator_value_n"
}
```
- `locator_name_1` (str): Имя локатора, используемое для обращения к элементу.
- `locator_value_1` (str): Значение локатора, которое может быть CSS-селектором, XPath выражением или другим идентификатором.

## Примеры
Примеры локаторов, которые могут содержатся в файле:

```json
{
    "logo_image": "div.logo",
    "search_field": "#search",
    "cart_button": ".cart-button",
    "product_card": "div.product-card",
    "product_name": "h3.product-title",
     "product_price": "span.price",
     "button_add_to_card": "button.add-to-card-button"
}
```

- `logo_image`: Локатор для изображения логотипа.
- `search_field`: Локатор для поля поиска.
- `cart_button`: Локатор для кнопки корзины.
- `product_card`: Локатор для карточки товара.
- `product_name`: Локатор для заголовка товара.
- `product_price`: Локатор для цены товара.
- `button_add_to_card`: Локатор для кнопки добавления товара в корзину.
