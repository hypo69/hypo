# Задача No 137
## Фибоначчи, золотое сечение и степенные ряды
## Условия

Рассмотрим степенной ряд, определенный как:

$$F(x) = \frac{x}{1 - x - x^2} = x + x^2 + 2x^3 + 3x^4 + 5x^5 + \cdots$$

Коэффициенты являются числами Фибоначчи $F_n$, где $F_1 = 1, F_2 = 1, F_3 = 2, \ldots$

Для коэффициента $x^n$ мы можем определить соответствующее число Фибоначчи $F_n$.
Например, если $x=1/2$, то
$$F(1/2) = \frac{1/2}{1-1/2-1/4} = 2$$
или, что то же самое,
$$F(1/2) = 1/2+1/4+2/8+3/16+5/32+\cdots=2$$

Знаменатель $1 - x - x^2$ имеет корни в
$x = \frac{-1 \pm \sqrt{5}}{-2} = \frac{1 \pm \sqrt{5}}{2}$.

Один из корней равен $\phi = \frac{1 + \sqrt{5}}{2}$, золотому сечению. 

Мы также можем определить $G(x) = \frac{x}{1 - 3x - x^2}$ и $H(x) = \frac{x}{1 - 4x - x^2}$.
По-видимому, существуют целые значения $x$, для которых $F(x)$, $G(x)$ и $H(x)$ являются целыми числами. 

Пусть последовательность $a(n)$ является возрастающей последовательностью значений $x$, для которых $F(x)$ является целым числом.

Первыми пятью значениями являются:

$$a(1) = 1/2, a(2) = 1/3, a(3) = 2/5, a(4) = 8/21 \text{ и } a(5) = 5/13.$$

Какое 15-е значение в этой последовательности?
