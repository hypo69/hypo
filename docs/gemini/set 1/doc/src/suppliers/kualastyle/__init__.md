# Модуль `hypotez/src/suppliers/kualastyle/__init__.py`

## Обзор

Данный модуль `kualastyle` предоставляет функции и классы для работы с поставщиком данных Kualastyle.  В нём определена переменная `MODE`, которая задаёт режим работы (в данном случае `'dev'`). Также модуль импортирует класс `Graber` из подмодуля `graber`.
