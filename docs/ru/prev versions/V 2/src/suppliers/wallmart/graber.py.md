# Модуль `src.suppliers.wallmart.graber`

## Обзор

Модуль `src.suppliers.wallmart.graber` содержит класс `Graber`, предназначенный для сбора данных о товарах с сайта `wallmart.com`. Класс наследует функциональность из родительского класса `Graber` и предоставляет возможность переопределения методов для нестандартной обработки полей. Модуль также включает механизм для выполнения предварительных действий перед запросом к вебдрайверу через декоратор, который по умолчанию определен в родительском классе.

## Классы

### `Graber`

**Описание**: Класс для сбора данных о товарах с сайта `wallmart.com`. Наследует функциональность из родительского класса `Graber` и предоставляет возможность переопределения методов для нестандартной обработки полей.

**Методы**:
- `__init__`: Инициализация экземпляра класса `Graber`.

#### `__init__`

**Описание**: Инициализирует экземпляр класса `Graber`, устанавливая префикс поставщика и вызывая конструктор родительского класса.

**Параметры**:
- `driver` (`Driver`): Экземпляр вебдрайвера для взаимодействия со страницей.

**Возвращает**:
- `None`

## Функции

### `close_pop_up`
    
**Описание**: Создает декоратор для закрытия всплывающих окон перед выполнением основной логики функции.

**Параметры**:
    - `value` (Any, optional): Дополнительное значение для декоратора. По умолчанию `None`.

**Возвращает**:
    - `Callable`: Декоратор, оборачивающий функцию.
