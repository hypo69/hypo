# send_data.js

## Обзор

Данный JavaScript-файл содержит функцию `onPageLoad`, которая собирает информацию о загруженной странице (заголовок, URL, содержимое) и отправляет ее на сервер.  Функция использует `fetch` для асинхронной отправки POST-запроса.

## Функции

### `onPageLoad`

**Описание**: Данная функция собирает информацию о загруженной странице и отправляет ее на указанный серверный API.

**Логика**:
1. Считывает заголовок страницы (`document.title`), URL (`window.location.href`) и содержимое тела страницы (`document.body.innerHTML`).
2. Формирует объект `data` с полученной информацией.
3. Использует `fetch` для отправки POST-запроса на `http://127.0.0.1/hypotez.online/api/`.
4. Обрабатывает ответ от сервера,  выводя результат в консоль (`console.log`) или выводит ошибку в консоль (`console.error`).

**Возвращает**:
Не имеет возвращаемого значения.

**Обрабатывает исключения**:
- `Error`: Если запрос `fetch` неуспешен (например, ошибка сети или сервера).  В этом случае функция переходит к блоку `catch` и выводит сообщение об ошибке в консоль.

## Использование

Функция `onPageLoad` автоматически вызывается при загрузке страницы через `window.addEventListener('load', onPageLoad);`.  Необходимо гарантировать, что серверная часть (`http://127.0.0.1/hypotez.online/api/`) корректно обрабатывает POST-запросы, содержащие JSON-данные.
