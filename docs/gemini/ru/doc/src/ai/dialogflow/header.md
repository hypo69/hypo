# Модуль `hypotez/src/ai/dialogflow/header.py`

## Обзор

Этот модуль содержит функции для определения корневого пути проекта и загрузки настроек из файла `settings.json`. Он также обрабатывает потенциальные ошибки при чтении файла настроек и файла `README.MD`.  Модуль обеспечивает доступ к различным метаданным проекта, таким как имя проекта, версия, документация и т.д.

## Функции

### `set_project_root`

**Описание**:  Функция находит корневой каталог проекта, начиная с текущего каталога и двигаясь вверх по иерархии каталогов. Она ищет файлы, указанные в `marker_files`.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или каталогов, которые указывают на корневой каталог проекта. По умолчанию включает `'pyproject.toml'`, `'requirements.txt'`, и `'.git'`.


**Возвращает**:

- `Path`: Путь к корневому каталогу проекта, если найден, в противном случае путь к каталогу текущего файла.

**Вызывает исключения**:
- Нет.

### `__root__`


**Описание**: Переменная хранящая корневой путь к проекту.


##  Использование

```python
from hypotez.src.ai.dialogflow.header import set_project_root

root_path = set_project_root()
print(root_path)
```


## Загрузка настроек

**Описание**: Этот модуль загружает настройки из файла `settings.json` и извлекает из него различные данные, такие как `project_name`, `version`, `author` и др.

**Обработка ошибок**:

Функции обработки ошибок `try...except` предназначены для перехвата и обработки ошибок `FileNotFoundError` и `json.JSONDecodeError`, которые могут возникнуть при чтении файла `settings.json` или `README.MD`. Это предотвращает сбой программы при отсутствии или неправильном формате этих файлов.


## Переменные

### `__root__`

**Описание**:  Переменная, содержащая корневой путь к проекту. Значение устанавливается при вызове `set_project_root()`.


### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженный из файла `settings.json` в корневом каталоге проекта.


### `doc_str`

**Описание**: Строка, содержащая текст из файла `README.MD`, если он существует. В противном случае `doc_str` содержит пустую строку.

### `__project_name__`

**Описание**: Строковая переменная, хранящая имя проекта, полученное из настроек или устанавливаемое по умолчанию как 'hypotez'


### `__version__`

**Описание**: Строковая переменная, хранящая версию проекта, полученная из настроек или устанавливаемая по умолчанию как пустая строка.


### `__doc__`

**Описание**: Строковая переменная, хранящая документацию проекта, полученная из файла `README.MD` или устанавливаемая по умолчанию как пустая строка.


### `__details__`

**Описание**: Строковая переменная, хранящая подробную информацию о проекте. По умолчанию она пуста.

### `__author__`

**Описание**: Строковая переменная, хранящая имя автора проекта, полученная из настроек или устанавливаемая по умолчанию как пустая строка.

### `__copyright__`

**Описание**: Строковая переменная, хранящая авторские права проекта. По умолчанию устанавливается как пустая строка.

### `__cofee__`

**Описание**: Строковая переменная, содержащая ссылку для поддержки разработчика.  По умолчанию содержит ссылку на Boosty.
