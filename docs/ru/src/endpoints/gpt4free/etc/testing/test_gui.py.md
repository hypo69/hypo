# Модуль для запуска графического интерфейса g4f

## Обзор

Модуль предназначен для запуска графического интерфейса (GUI) для проекта g4f. Он добавляет в путь поиска модулей родительские директории, чтобы обеспечить доступ к модулю `g4f.gui`, и затем запускает GUI с помощью функции `run_gui()`.

## Подробнее

Этот модуль упрощает процесс запуска графического интерфейса, особенно когда модуль `g4f` установлен не в стандартном месте или разрабатывается локально. Путь к модулю добавляется динамически, чтобы избежать проблем с импортом.

## Функции

### `run_gui`

```python
def run_gui():
    """Запускает графический интерфейс g4f."""
    ...
```

**Назначение**: Запускает графический интерфейс приложения g4f.

**Как работает функция**:
1. Функция `run_gui()` вызывается для инициализации и отображения графического интерфейса пользователя.

**Примеры**:
```python
run_gui()
```
Этот вызов запустит графический интерфейс, позволяя пользователю взаимодействовать с g4f через визуальные элементы.
