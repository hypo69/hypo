# Модуль hypotez/src/gui/context_menu/header.py

## Обзор

Данный модуль содержит код для инициализации путей к бинарным файлам GTK, FFmpeg и Graphviz, а также для добавления этих путей в системный путь `sys.path`. Он также включает обработку предупреждений GTK, чтобы предотвратить вывод сообщений об ошибках в консоль.


## Функции

### `__init__`

**Описание**: Данная функция не определена в предоставленном коде.

### `__root__`


**Описание**:  Возвращает абсолютный путь к корневому каталогу проекта, определенный из файла `settings.json`.

**Возвращает**:
- `Path`: Абсолютный путь к корню проекта.

### `sys_path_env_var`


**Описание**: Устанавливает переменную окружения `WEASYPRINT_DLL_DIRECTORIES`, добавляя путь к бинарным файлам GTK в системный путь `sys.path`.

**Параметры**:
- Нет


**Возвращает**:
- Нет

### `load_project_name`


**Описание**: Загружает имя проекта из файла `settings.json`.

**Параметры**:
- Нет


**Возвращает**:
- `str`: Имя проекта.


## Константы

### `MODE`

**Описание**: Переменная `MODE` хранит значение 'dev'.


## Инициализация путей к бинарным файлам

**Описание**: Данный код инициализирует переменные, содержащие пути к бинарным файлам GTK, FFmpeg и Graphviz, на основе пути к корневому каталогу проекта. Далее, он добавляет эти пути в `sys.path`, если они отсутствуют.


## Обработка предупреждений GTK

**Описание**:  Код подавляет вывод предупреждений GTK в консоль, используя функцию `warnings.filterwarnings()`.
