## Файл chrome.py

### Класс `Chrome`

**Описание**:  Этот класс расширяет базовый класс `selenium.webdriver.Chrome`, предоставляя дополнительные методы для настройки и инициализации драйвера Chrome.

**Атрибуты**:

* `driver_name`: Название драйвера ('chrome').
* `d`: Экземпляр драйвера `webdriver.Chrome`.
* `options`: Объект `ChromeOptions` для настройки параметров.
* `user_agent`: Словарь с настройками user-agent.


**Методы**:

#### `__init__`

**Описание**: Инициализирует экземпляр класса `Chrome`.

**Параметры**:

* `user_agent` (dict, необязательно): Словарь настроек User-Agent. Используется по умолчанию `UserAgent().random`.

**Возвращает**:

* None.

**Вызывает исключения**:

* `Exception`: При возникновении ошибок при настройке WebDriver.


#### `find_free_port`

**Описание**: Находит свободный порт в заданном диапазоне.

**Параметры**:

* `start_port` (int): Начальный порт диапазона.
* `end_port` (int): Конечный порт диапазона.

**Возвращает**:

* int | None: Свободный порт, если найден, иначе None.

**Вызывает исключения**:

* `OSError`: Если порт занят.


#### `set_options`

**Описание**: Настраивает опции запуска для Chrome WebDriver.

**Параметры**:

* `settings` (list | dict | None, необязательно): Настройки для опций Chrome.

**Возвращает**:

* `ChromeOptions`: Объект `ChromeOptions` с заданными опциями запуска.

**Вызывает исключения**:

* `Exception`: При возникновении ошибок при настройке опций.

## Файл chrome.json

**Описание**: Файл конфигурации для Chrome WebDriver. Содержит информацию о путях к драйверу, бинарному файлу Chrome и других настройках.

**Структура**:

* **`profiles`**: Определяет директории для профилей Chrome.
* **`driver`**: Настройки пути к ChromeDriver и Chrome.
* **`headers`**: Настройки HTTP заголовков.
