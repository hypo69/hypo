# Модуль `hypotez/src/templates/header.py`

## Обзор

Этот модуль содержит функцию `set_project_root`, предназначенную для определения корневой директории проекта.  Функция ищет корневой каталог, начиная с текущей директории и поднимаясь вверх по иерархии директорий. В качестве маркеров корневой директории используются файлы или папки, перечисленные в аргументе `marker_files`. Если корневой каталог не найден, используется текущая директория.


## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с текущего файла.

**Параметры**:
- `marker_files` (tuple): Кортеж строк (имен файлов или директорий), которые используются для определения корневой директории проекта. По умолчанию включает `pyproject.toml`, `requirements.txt` и `.git`.

**Возвращает**:
- `Path`: Путь к корневой директории проекта.  Возвращает директорию текущего файла, если корневая директория не найдена.

**Примечания**:
- Функция добавляет корневую директорию в `sys.path`, если она еще не присутствует.

**Пример использования**:

```python
root_dir = set_project_root()
print(root_dir)
```

## Переменные

### `MODE`

**Описание**: Строковая переменная, хранящая значение режима (`dev`).

### `__root__`

**Описание**: Переменная, хранящая путь к корневой директории проекта.  Инициализируется функцией `set_project_root()`. Тип - `Path`.
