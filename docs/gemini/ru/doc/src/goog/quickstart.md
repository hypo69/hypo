# Модуль `hypotez/src/goog/quickstart.py`

## Обзор

Данный модуль предоставляет функции для взаимодействия с Google Apps Script API.  Он позволяет создавать новые проекты скриптов, загружать файлы в эти проекты и выводить URL скрипта пользователю. Модуль использует библиотеки `google-auth-oauthlib` и `googleapiclient` для аутентификации и взаимодействия с API.

## Функции

### `main`

**Описание**: Функция, содержащая основной логический блок для работы с Google Apps Script API.  Она выполняет аутентификацию, создание проекта, загрузку файлов и вывод URL проекта.

**Параметры**:
Нет входных параметров.

**Возвращает**:
-  Нет возвращаемых значений.

**Вызывает исключения**:
- `googleapiclient.errors.HttpError`: Возникает, если API возвращает ошибку HTTP.  В этом случае выводится содержимое ошибки.
