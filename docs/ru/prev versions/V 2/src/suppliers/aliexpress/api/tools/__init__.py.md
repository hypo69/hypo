# Модуль `src.suppliers.aliexpress.api.tools`

## Обзор

Модуль `src.suppliers.aliexpress.api.tools` предоставляет набор инструментов для работы с API AliExpress. В настоящее время он включает в себя функцию `get_product_id` для извлечения ID продукта.

## Функции

### `get_product_id`

**Описание**: Функция для извлечения ID продукта из строки.

**Параметры**:
- Отсутствуют

**Возвращает**:
-  Описание возвращаемого значения находится в описании функции `get_product_id`

**Вызывает исключения**:
- Описание исключений находится в описании функции `get_product_id`
