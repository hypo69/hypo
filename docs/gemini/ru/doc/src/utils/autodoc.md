# Модуль `hypotez/src/utils/autodoc.py`

## Обзор

Модуль `autodoc` содержит декоратор для автоматического обновления строки документации (docstring) функций.  Декоратор добавляет информацию о времени последнего вызова функции в docstring при каждом её вызове.

## Функции

### `autodoc`

**Описание**: Декоратор, который добавляет информацию о времени последнего вызова функции в её docstring.

**Параметры**:
- Нет входных параметров.

**Возвращает**:
- Оборачиваемую функцию.

**Вызывает исключения**:
- Нет указанных исключений.


### `update_docstring`

**Описание**: Функция, обновляющая docstring функции, добавляя информацию о времени последнего вызова.

**Параметры**:
- `func` (функция): Функция, docstring которой необходимо обновить.

**Возвращает**:
- Нет значения возвращается.

**Вызывает исключения**:
- Нет указанных исключений.


### `example_function`

**Описание**: Пример функции, использующей декоратор `autodoc`. При каждом вызове её docstring обновляется с информацией о времени последнего вызова.

**Параметры**:
- `param1` (int): Первое значение.
- `param2` (str): Второе значение.

**Возвращает**:
- `None`: Функция не возвращает значения.

**Вызывает исключения**:
- Нет указанных исключений.


## Использование

Декоратор `autodoc` применяется к функции с помощью синтаксиса:

```python
@autodoc
def my_function():
    # ... тело функции ...
```

После каждого вызова функции `my_function` её docstring будет содержать информацию о времени последнего вызова.


## Пример

```python
@autodoc
def example_function(param1: int, param2: str) -> None:
    """Пример функции.

    Args:
        param1 (int): Первое значение.
        param2 (str): Второе значение.
    """
    print(f"Processing {param1} and {param2}")

example_function(1, "test")
print(example_function.__doc__)  # Вывод обновленного docstring
example_function(2, "another test")
print(example_function.__doc__)  # Вывод обновленного docstring
```

Этот пример демонстрирует использование декоратора `autodoc`. При каждом вызове функции `example_function` её docstring будет обновляться, отображая время последнего вызова.
