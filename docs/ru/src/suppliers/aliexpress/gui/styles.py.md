# Модуль стилей для UI элементов

## Обзор

Этот модуль содержит общие функции стилизации для элементов пользовательского интерфейса (UI), разработанного на PyQt6. В частности, он предоставляет функцию для установки фиксированного размера виджета.

## Подробней

Модуль `styles.py` предназначен для централизации настроек стилей, что упрощает поддержку единообразного внешнего вида приложения. Использование этого модуля помогает избежать дублирования кода и облегчает изменение стилей в масштабе всего приложения.

## Функции

### `set_fixed_size`

```python
def set_fixed_size(widget: QtWidgets.QWidget, width: int, height: int):
    """ Устанавливает фиксированный размер для заданного виджета.

    Args:
        widget (QtWidgets.QWidget): Виджет, для которого необходимо установить фиксированный размер.
        width (int): Ширина виджета.
        height (int): Высота виджета.

    Returns:
        None

    Raises:
        TypeError: Если `widget` не является экземпляром класса `QtWidgets.QWidget`.

    Как работает функция:
     1. Функция принимает виджет, ширину и высоту в качестве аргументов.
     2. Вызывает метод `setFixedSize` виджета, передавая ему указанные значения ширины и высоты.
     3. Устанавливает фиксированный размер виджета, предотвращая его изменение пользователем или автоматической системой размещения.

    Примеры:
        >>> from PyQt6.QtWidgets import QWidget
        >>> widget = QWidget()
        >>> set_fixed_size(widget, 100, 50)
        >>> widget.size().width()
        100
        >>> widget.size().height()
        50
    """
    widget.setFixedSize(width, height)
