# Модуль `edge.py`

## Обзор

Модуль `edge.py` содержит класс `Edge`, который представляет собой пользовательский WebDriver для браузера Microsoft Edge. Он предоставляет упрощенную настройку, включая использование случайного `User-Agent` и возможность добавления пользовательских опций.

## Классы

### `Edge`

**Описание**:
Пользовательский класс WebDriver для браузера Edge с расширенными функциональными возможностями.

**Атрибуты**:
- `driver_name` (str): Имя используемого WebDriver, по умолчанию `edge`.

**Методы**:
- `__init__`: Инициализирует WebDriver Edge с заданным User-Agent и опциями.

#### `__init__`

**Описание**:
Инициализирует Edge WebDriver с заданным User-Agent и опциями.

**Параметры**:
- `user_agent` (Optional[str], optional): Строка User-Agent. Если `None`, будет сгенерирован случайный User-Agent. По умолчанию `None`.
- `options` (Optional[List[str]], optional): Список опций Edge, передаваемых во время инициализации. По умолчанию `None`.

**Возвращает**:
- `None`: Функция не возвращает значения.

**Вызывает исключения**:
- `WebDriverException`: Если не удается запустить WebDriver Edge.
- `Exception`: При возникновении общей ошибки в процессе запуска WebDriver.

### `set_options`

**Описание**:
Создает и настраивает опции запуска для Edge WebDriver.

**Параметры**:
- `opts` (Optional[List[str]], optional): Список опций для добавления в Edge WebDriver. По умолчанию `None`.

**Возвращает**:
- `EdgeOptions`: Настроенный объект `EdgeOptions`.
