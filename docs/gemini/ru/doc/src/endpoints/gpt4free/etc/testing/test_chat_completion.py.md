# Модуль для тестирования чат-завершения gpt4free

## Обзор

Модуль предназначен для тестирования функциональности чат-завершения с использованием библиотеки `gpt4free`. Включает синхронный и асинхронный тесты для проверки генерации текста на основе заданной модели и провайдера.

## Подробней

Этот модуль выполняет следующие действия:

1.  Добавляет путь к родительскому каталогу в `sys.path`, чтобы обеспечить доступ к модулям проекта `hypotez`.
2.  Импортирует библиотеки `g4f` и `asyncio`.
3.  Выполняет синхронный тест с использованием `g4f.ChatCompletion.create` для генерации стихов о дереве.
4.  Выполняет асинхронный тест с использованием `g4f.ChatCompletion.create_async` для генерации ответа на приветствие.

## Функции

### `run_async`

```python
async def run_async():
    """Асинхронно создает завершение чата и выводит результат.

    Args:
        None

    Returns:
        None

    """
```

**Назначение**:
Асинхронно создает завершение чата с использованием `g4f.ChatCompletion.create_async` и выводит результат в консоль.

**Как работает функция**:

1.  Вызывает `g4f.ChatCompletion.create_async` с моделью по умолчанию и сообщением "hello!".
2.  Печатает полученный ответ с префиксом "create\_async:".

**Примеры**:

```python
import asyncio
async def run_async():
    response = await g4f.ChatCompletion.create_async(
        model=g4f.models.default,
        messages=[{"role": "user", "content": "hello!"}],
    )
    print("create_async:", response)

asyncio.run(run_async())
