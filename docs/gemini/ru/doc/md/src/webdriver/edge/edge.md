# Модуль hypotez/src/webdriver/edge/edge.py

## Обзор

Этот модуль содержит класс `Edge`, представляющий собой кастомный драйвер WebDriver для браузера Edge. Он упрощает настройку драйвера с использованием `fake_useragent` и предоставляет дополнительные функции, такие как выполнение JavaScript-кода и работу с локаторами.

## Классы

### `Edge`

**Описание**: Кастомный класс `Edge`, наследуемый от `selenium.webdriver.Edge`.  Предназначен для расширения функциональности стандартного драйвера Edge.

**Атрибуты**:

- `driver_name` (str): Имя используемого WebDriver, по умолчанию 'edge'.

**Методы**:

#### `__init__`

**Описание**: Инициализирует экземпляр класса `Edge` с указанным user-agent и опциями.

**Параметры**:

- `user_agent` (Optional[dict], optional): Словарь для указания user-agent. Если `None`, генерируется случайный user-agent.


**Вызывает исключения**:

- `WebDriverException`: Исключение, возникающее при неудачном запуске WebDriver.
- `Exception`: Общее исключение, возникающее при непредвиденных ошибках.


#### `_payload`

**Описание**: Загружает исполнители для локаторов и сценариев JavaScript.

**Возвращает**:
- `None`

#### `set_options`

**Описание**: Создает и настраивает опции запуска для WebDriver Edge.

**Параметры**:

- `opts` (Optional[List[str]], optional): Список опций для добавления к Edge WebDriver. По умолчанию `None`.

**Возвращает**:
- `EdgeOptions`: Настроенный объект `EdgeOptions`.
