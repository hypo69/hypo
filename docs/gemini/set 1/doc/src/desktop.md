# Файл desktop.ini

## Обзор

Данный файл представляет собой файл Windows, используемый для задания свойств папок.  В частности, он определяет иконку, используемую для отображения папки в проводнике Windows.

## Структура

Файл `desktop.ini` имеет формат INI.  Он содержит секции, разделенные квадратными скобками, и их значения.  В этом случае, две секции: `[ShellClassInfo]` и `[ViewState]`.

### [ShellClassInfo]

**Описание:** Определяет иконку для папки.

**Параметры:**

- `IconResource=C:\\WINDOWS\\System32\\SHELL32.dll,300`: Указывает путь к файлу DLL (SHELL32.dll) и индекс ресурса иконки (300).

### [ViewState]

**Описание:**  Определяет состояние отображения папки.

**Параметры:**

- `Mode=`, `Vid=`, `FolderType=Generic`: Параметры, связанные с отображением и состоянием папки.  Их точное значение не указано в данной структуре.  Возможно, это значения по умолчанию или  неактивные в данном контексте.


## Использование

Файлы `desktop.ini` используются для управления отображением папок в проводнике Windows.  Их создание и редактирование происходит автоматически в большинстве случаев при работе с папками.  Пользователю, как правило, не требуется непосредственное взаимодействие с этим типом файла.
