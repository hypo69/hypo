# src.webdriver.edge.extentions.version

## Обзор

Файл `version.py` содержит информацию о версии и метаданные для модуля `src.webdriver.edge.extentions`. Он определяет переменные, такие как версия, имя, документация и автор модуля.

## Подробнее

Этот файл предназначен для хранения и предоставления информации о версии модуля `src.webdriver.edge.extentions`. Он включает в себя:

- `__version__`: Версия модуля или пакета.
- `__name__`: Имя модуля. Если скрипт запущен напрямую, значение будет `"__main__"`.
- `__doc__`: Строка документации модуля.
- `__details__`: Дополнительные детали о модуле.
- `__annotations__`: Типовые аннотации для переменных и функций в модуле.
- `__author__`: Имя автора(ов) модуля.

Эти метаданные могут быть использованы для различных целей, таких как проверка совместимости, ведение журнала и отображение информации о модуле.

## Переменные

### `__name__`

**Описание**: Содержит имя модуля.

**Тип**: str

### `__version__`

**Описание**: Содержит версию модуля или пакета.

**Тип**: str

### `__doc__`

**Описание**: Строка документации модуля.

**Тип**: str

### `__details__`

**Описание**: Дополнительные детали о версии модуля.

**Тип**: str

### `__annotations__`

**Описание**: Типовые аннотации для переменных и функций в модуле.

**Тип**: Any

### `__author__`

**Описание**: Имя автора(ов) модуля.

**Тип**: str
