# Модуль hypotez/src/endpoints/kazarinov/scenarios/header.py

## Обзор

Этот модуль содержит функцию `set_project_root` для поиска корневой директории проекта и установления пути к ней в переменную окружения `sys.path`.  Также он загружает настройки из файла `settings.json` и документацию из файла `README.MD` в глобальные переменные, если эти файлы существуют.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная от текущей директории, и поднимаясь по дереву директорий до тех пор, пока не найдёт директорию, содержащую один из указанных файлов или директорий.

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, которые будут использоваться для определения корневой директории проекта.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена, в противном случае — путь к текущей директории.

**Вызывает исключения**:

- Нет


## Глобальные переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root`.

**Тип**: `Path`


### `settings`

**Описание**: Словарь настроек, загруженный из файла `src/settings.json`.

**Тип**: `dict`


### `doc_str`

**Описание**: Строка документации, загруженная из файла `src/README.MD`.

**Тип**: `str`


### `__project_name__`

**Описание**: Название проекта, полученное из настроек (`settings`). По умолчанию `hypotez`.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта, полученная из настроек (`settings`). По умолчанию пустая строка.

**Тип**: `str`


### `__doc__`

**Описание**: Строка документации, полученная из `README.MD`. По умолчанию пустая строка.

**Тип**: `str`


### `__details__`

**Описание**: Строка дополнительной информации. По умолчанию пустая строка.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта, полученный из настроек (`settings`). По умолчанию пустая строка.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права, полученные из настроек (`settings`). По умолчанию пустая строка.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка для поддержки разработчика. Получена из настроек (`settings`). По умолчанию содержит ссылку на Boosty.

**Тип**: `str`


## Обработка исключений

Обработка ошибок `FileNotFoundError` и `json.JSONDecodeError` предотвращает падение скрипта при отсутствии или неверном формате файлов настроек и документации.
