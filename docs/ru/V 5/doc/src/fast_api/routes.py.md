# Маршруты Fast API

## Обзор

Модуль `routes.py` предназначен для манипулирования маршрутами в FastAPI-сервере. Он содержит класс `Routes`, который отвечает за обработку входящих сообщений от Telegram бота.

## Подробней

Этот модуль определяет обработчик для входящих сообщений от Telegram бота. Класс `Routes` инициализирует обработчик `BotHandler` и назначает метод `handle_message` обработчика для обработки входящих сообщений.

## Классы

### `Routes`

**Описание**: Класс, отвечающий за настройку обработчиков маршрутов для FastAPI.

**Как работает класс**:
Класс `Routes` содержит метод `tegram_message_handler`, который инициализирует обработчик `BotHandler` и назначает метод `handle_message` этого обработчика для обработки входящих сообщений от Telegram бота.

**Методы**:
- `tegram_message_handler`: Настраивает обработчик входящих сообщений от Telegram бота.
