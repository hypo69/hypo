Как использовать этот блок кода
=========================================================================================\n\nОписание
-------------------------
Данный блок кода представляет собой список строк.
\n\nШаги выполнения
-------------------------
1. Блок кода определяет список строк `["a","b","c"]`.
\n\nПример использования
-------------------------
.. code-block:: python

    my_list = ["a","b","c"]
    print(my_list)
