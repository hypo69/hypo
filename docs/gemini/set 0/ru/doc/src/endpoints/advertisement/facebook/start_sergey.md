# Модуль `hypotez/src/endpoints/advertisement/facebook/start_sergey.py`

## Обзор

Модуль `start_sergey.py` отвечает за запуск рекламных кампаний в Facebook для разных рекламодателей (в данном случае, Казаринов и AliExpress). Он использует драйвер для взаимодействия с веб-сайтом Facebook, загружает данные о группах и рекламных объявлениях из JSON-файлов, и выполняет запуск кампаний с учетом языка и валюты. Модуль также включает цикл для периодического запуска и повторного запуска кампаний с определёнными задержками.

## Определения

### `MODE`

**Описание**: Переменная, хранящая режим работы (в данном случае `'dev'`).

### `group_file_paths_ru`, `adv_file_paths_ru`, `group_file_paths_he`, `adv_file_paths_he`

**Описание**: Список путей к JSON-файлам с данными о группах и рекламных объявлениях для русского и иврита языков.

### `group_categories_to_adv`

**Описание**: Список категорий для таргетирования рекламных объявлений.

## Функции

### `run_campaign`

**Описание**: Функция для запуска рекламной кампании.

**Параметры**:

- `d` (Driver): Экземпляр драйвера для взаимодействия с браузером.
- `promoter_name` (str): Имя рекламодателя.
- `campaigns` (list): Список названий кампаний.
- `group_file_paths` (list): Список путей к файлам с данными о группах.
- `language` (str): Язык рекламной кампании.
- `currency` (str): Валюта рекламной кампании.

**Возвращает**:
- Не имеет возвращаемого значения.

**Вызывает исключения**:
- Любые исключения, которые могут быть вызваны при использовании `FacebookPromoter`.


### `campaign_cycle`

**Описание**: Функция для запуска цикла управления кампаниями.

**Параметры**:

- `d` (Driver): Экземпляр драйвера для взаимодействия с браузером.

**Возвращает**:
- `bool`: Значение `True` по завершении цикла.

**Вызывает исключения**:
- Любые исключения, которые могут быть вызваны внутри цикла.

### `main`

**Описание**: Основная функция для запуска программы.

**Параметры**:
- Не имеет параметров.

**Возвращает**:
- Не имеет возвращаемого значения.

**Вызывает исключения**:
- `KeyboardInterrupt`: Обрабатывает прерывание программы пользователем.
- Любые исключения, которые могут быть вызваны внутри цикла.

## Использование

Модуль `start_sergey.py` предназначен для автоматического запуска рекламных кампаний в Facebook. Для его использования необходимо:
1. Подключить необходимые библиотеки.
2. Создать экземпляр драйвера.
3. Настроить параметры рекламных кампаний.
4. Запустить функцию `main()`.

## Замечания

- Модуль использует сторонние библиотеки (`header`, `random`, `time`, `copy`, `Path`, `gs`, `get_directory_names`, `get_filenames`, `Driver`, `Chrome`, `FacebookPromoter`, `logger`, `interval`).
- В коде присутствуют задержки, чтобы избежать проблем с API Facebook.
- Необходимо настроить пути к файлам с данными о группах и рекламных объявлениях (`group_file_paths_ru`, `adv_file_paths_ru`, `group_file_paths_he`, `adv_file_paths_he`).
- Необходимо правильно настроить переменные `gs.path.google_drive` и все необходимые пути к файлам.
- Добавлен комментарий, объясняющий назначение `file_paths_ru.extend(adv_file_paths_ru)`.
- Добавлен блок обработки `KeyboardInterrupt`, для безопасного завершения.
