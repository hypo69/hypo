# Модуль миграции базы данных: `upd_table_purschases`

## Обзор

Этот модуль содержит скрипт миграции базы данных, используемый Alembic для обновления таблицы `purchases`. Он добавляет столбец `payment_type` и создает уникальное ограничение для столбца `payment_id`.
