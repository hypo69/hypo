# Модуль `hypotez/src/suppliers/aliexpress/gui/campaign.py`

## Обзор

Этот модуль предоставляет класс `CampaignEditor` для визуального редактирования кампаний AliExpress.  Класс обеспечивает загрузку данных кампании из JSON-файла, отображение основных свойств (название, описание, имя промоакции) и функцию подготовки кампании.


## Классы

### `CampaignEditor`

**Описание**: Класс `CampaignEditor` представляет собой виджет для редактирования данных кампании AliExpress. Он предоставляет интерфейс для выбора JSON-файла, загрузки данных, отображения основных свойств кампании и запуска подготовки кампании.

**Атрибуты**:

- `data: SimpleNamespace`:  Объект `SimpleNamespace`, содержащий загруженные данные кампании.
- `current_campaign_file: str`: Путь к загруженному файлу кампании.
- `editor: AliCampaignEditor`: Экземпляр класса `AliCampaignEditor` для подготовки кампании.

**Методы**:

#### `__init__(self, parent=None, main_app=None)`

**Описание**: Инициализирует виджет `CampaignEditor`.

**Параметры**:
- `parent` (Optional): Родительский виджет.
- `main_app` (Optional): Экземпляр главного приложения.

#### `setup_ui(self)`

**Описание**: Настраивает пользовательский интерфейс виджета. Создает элементы управления (кнопку открытия файла, поля ввода данных), устанавливает размеры и добавляет в макет.

#### `setup_connections(self)`

**Описание**: Устанавливает связи между элементами интерфейса и обработчиками событий.

#### `open_file(self)`

**Описание**: Открывает диалоговое окно для выбора JSON-файла с данными кампании.

**Возвращает**: Не имеет значения.

#### `load_file(self, campaign_file)`

**Описание**: Загружает JSON-файл и обрабатывает его содержимое.

**Параметры**:
- `campaign_file (str)`: Путь к файлу.

**Возвращает**:
- Не имеет значения.
**Вызывает исключения**:
- `Exception`: Возникает при ошибках при загрузке файла, например, если файл не существует или некорректно отформатирован.

#### `create_widgets(self, data)`

**Описание**: Создает поля ввода для отображения данных кампании.

**Параметры**:
- `data (SimpleNamespace)`: Данные кампании, загруженные из файла.

**Возвращает**: Не имеет значения.

#### `prepare_campaign(self)`

**Описание**: Асинхронно запускает подготовку кампании.

**Возвращает**: Не имеет значения.
**Вызывает исключения**:
- `Exception`: Возникает при ошибках подготовки кампании.
