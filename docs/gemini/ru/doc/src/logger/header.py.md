# src.logger.header

## Обзор

Модуль `header.py` определяет корневой путь к проекту и загружает основные настройки проекта. Все импорты в проекте строятся относительно этого корневого пути. Также модуль загружает файл `settings.json` и `README.MD`

## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная с каталога текущего файла, перемещаясь вверх до первого каталога, содержащего любой из указанных файлов-маркеров.

**Параметры**:
- `marker_files` (tuple, optional): Кортеж имен файлов или каталогов для идентификации корневого каталога проекта. По умолчанию `('__root__','.git')`.

**Возвращает**:
- `Path`: Путь к корневому каталогу, если он найден, иначе — каталог, где расположен скрипт.

## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта. Устанавливается функцией `set_project_root`.

### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json`. Если файл не найден или не удается прочитать, остается `None`.

### `doc_str`

**Описание**: Строка с содержимым файла `README.MD`. Если файл не найден или не удается прочитать, остается `None`.

### `__project_name__`

**Описание**: Название проекта, полученное из `settings.json`, или `'hypotez'`, если `settings` не загружены.

### `__version__`

**Описание**: Версия проекта, полученная из `settings.json`, или пустая строка, если `settings` не загружены.

### `__doc__`

**Описание**: Содержимое файла `README.MD`, загруженное в переменную `doc_str`.

### `__details__`

**Описание**: Детали проекта. В текущей версии не используется, имеет значение пустой строки.

### `__author__`

**Описание**: Автор проекта, полученный из `settings.json`, или пустая строка, если `settings` не загружены.

### `__copyright__`
**Описание**: Информация об авторских правах проекта, полученная из `settings.json`, или пустая строка, если `settings` не загружены.

### `__cofee__`

**Описание**: Сообщение с предложением поддержать разработчика чашечкой кофе, полученное из `settings.json`, или ссылка на boosty, если `settings` не загружены.
