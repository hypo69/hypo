## Подробней

Модуль предназначен для упрощения взаимодействия с API Helicone и OpenAI. Класс `HeliconeAI` предоставляет удобные методы для выполнения различных задач обработки текста, таких как генерация стихов, анализ тональности, создание кратких изложений и перевод текста. Модуль использует библиотеки `helicone` и `openai` для взаимодействия с соответствующими сервисами.

### `HeliconeAI`

**Описание**: Класс `HeliconeAI` интегрирует функциональность Helicone и OpenAI для обработки текста.

**Как работает класс**:
