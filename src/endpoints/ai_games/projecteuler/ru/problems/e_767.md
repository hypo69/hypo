# Задача No 767
## Наибольшее различие между двумя последовательными значениями n, для которых факториал n оканчивается на k нулей
## Условия

Обозначим через $Z(n)$ количество нулей, на которые заканчивается факториал $n$.
Например, $10! = 3628800$, поэтому $Z(10) = 2$.

Обозначим через $M(k)$ наибольшее различие между двумя последовательными значениями $n$, для которых $Z(n) = k$.

Например, $Z(5) = 1$, $Z(6) = 1$, $Z(7) = 1$, $Z(8) = 1$, $Z(9) = 1$ и $Z(10) = 2$, поэтому наименьшее значение $n$, для которого $Z(n) = 2$, равно $10$.

$Z(24) = 4$ и $Z(25) = 6$. Первые значения $n$, для которых $Z(n) = 4$, равны $20, 21, 22, 23$ и $24$.
Первые значения $n$, для которых $Z(n) = 6$, равны $25, 26, 27, 28$ и $29$.
Значит, $M(4) = 4$, а $M(5)$ не определено, так как $Z(n)$ не принимает значение $5$.
Также $M(6) = 4$.

Известно, что $M(1000) = 4$.

Найдите $\sum M(k)$ для всех значений $k$, для которых $1 \le k \le 10^5$ и $M(k)$ определено.
