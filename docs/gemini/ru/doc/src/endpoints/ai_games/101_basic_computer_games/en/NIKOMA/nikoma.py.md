# Документация модуля `nikoma.py`

## Обзор

Модуль `nikoma.py` реализует игру "Ракета", в которой игрок управляет ракетой, пытаясь достичь заданной высоты (100), расходуя топливо. Игрок на каждом шаге вводит количество топлива, которое он хочет потратить на подъем, а высота, на которую поднимается ракета, определяется случайным образом.

## Игровой процесс

Цель игры - достичь высоты 100, прежде чем закончится топливо. Игрок начинает со 100 единицами топлива. На каждом шаге игрок вводит количество топлива для сжигания. Высота, на которую поднимется ракета, рассчитывается как случайное число от 0 до удвоенного значения введенного топлива. Игра заканчивается при достижении высоты 100 или при исчерпании топлива.

## Функции

В данном модуле нет явно определенных функций, весь код игры выполняется в глобальной области видимости.

## Переменные

- `rocketHeight` (int): Текущая высота ракеты. Изначально установлена в 0.
- `fuel` (int): Количество топлива. Изначально установлено в 100.
- `fuelToBurn` (int): Количество топлива, которое игрок хочет сжечь на текущем шаге.
- `rise` (int): Значение высоты, на которую ракета поднимется на текущем шаге.

## Принципы работы

1. **Инициализация переменных**:
   - Устанавливает начальную высоту ракеты `rocketHeight` равной 0.
   - Устанавливает начальное количество топлива `fuel` равным 100.

2. **Основной игровой цикл**:
   - Цикл `while rocketHeight < 100 and fuel > 0:` выполняется, пока ракета не достигнет высоты 100 или пока не закончится топливо.
   - Выводит текущую высоту и количество топлива: `print(f"Высота: {rocketHeight}, Топливо: {fuel}")`.
   - **Ввод количества топлива**:
     - Запрашивает у пользователя количество топлива для сжигания с помощью `fuelToBurn = int(input("Сколько топлива сжечь?: "))`.
     - **Обработка исключения**:
       - Используется блок `try...except ValueError` для обработки ошибок ввода. Если пользователь введет не целое число, выводится сообщение об ошибке и цикл продолжается со следующей итерации.
   - **Проверка наличия топлива**:
     - Проверяет, достаточно ли топлива для сжигания: `if fuelToBurn > fuel:`.
     - Если топлива недостаточно, выводит сообщение об ошибке: `print("У вас нет столько топлива!")`.
   - **Сжигание топлива и расчет подъема**:
     - Если топлива достаточно, уменьшает количество топлива на введенное значение: `fuel -= fuelToBurn`.
     - Вычисляет случайный подъем в диапазоне от 0 до 2 * `fuelToBurn`: `rise = random.randint(0, 2 * fuelToBurn)`.
     - Увеличивает высоту ракеты на величину подъема: `rocketHeight += rise`.

3. **Проверка условий окончания игры и вывод сообщения**:
   - После завершения цикла проверяет, достигла ли ракета высоты 100: `if rocketHeight >= 100:`.
   - Если ракета достигла высоты 100, выводит сообщение о победе: `print("Поздравляю! Вы достигли высоты 100!")`.
   - Если ракета не достигла высоты 100, выводит сообщение о проигрыше: `print("Вы исчерпали топливо. Вы не достигли высоты 100!")`.
