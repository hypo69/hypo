hypotez.src.scenario
====================

.. module:: hypotez.src.scenario
    :platform: Windows, Unix
    :synopsis: Module containing scenario execution functions.

This module provides functions for executing scenarios for suppliers, handling data from various sources (like PrestaShop) and driving the execution process.  It facilitates the execution of scenarios defined in files or as dictionaries.
