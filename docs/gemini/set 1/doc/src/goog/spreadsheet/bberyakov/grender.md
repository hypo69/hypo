# Модуль hypotez/src/goog/spreadsheet/bberyakov/grender.py

## Обзор

Этот модуль предоставляет класс `GSRender` для форматирования и рендеринга данных в Google Таблицах.  Он содержит функции для создания и настройки форматов ячеек, объединения ячеек, установки направления текста и добавления заголовков/категорий в таблицу.

## Классы

### `GSRender`

**Описание**: Класс `GSRender` отвечает за рендеринг данных в Google Таблицах. Он предоставляет методы для форматирования ячеек, объединения ячеек, установки направления текста и добавления заголовков/категорий.


**Методы**:

- `__init__`: Инициализирует экземпляр класса.
- `render_header`: Рисует заголовок таблицы в первой строке, применяя форматирование и объединение ячеек.
- `merge_range`: Объединяет ячейки в указанном диапазоне по заданному типу объединения (MERGE_ALL, MERGE_COLUMNS, MERGE_ROWS).
- `set_worksheet_direction`: Устанавливает направление текста в листе Google Таблицы (LTR или RTL).
- `header`: Добавляет заголовок в таблицу.
- `write_category_title`: Добавляет заголовок категории в таблицу.
- `get_first_empty_row`: Получает номер первой пустой строки в таблице (или по заданной колонке).

### `CellFormat`

**Описание**:  Внутренний класс, представляющий формат ячейки.

**Атрибуты**:

- `numberFormat`: Формат числа (любой тип).
- `backgroundColor`: Цвет фона ячейки (любой тип).
- `borders`: Границы ячейки (любой тип).
- `padding`: Отступы ячейки (любой тип).
- `horizontalAlignment`: Горизонтальное выравнивание текста (любой тип).
- `verticalAlignment`: Вертикальное выравнивание текста (любой тип).
- `wrapStartegy`: Стратегия переноса текста по строкам (любой тип).
- `textDirection`: Направление текста (любой тип).
- `textFormat`: Формат текста (любой тип).
- `hyperlinkDisplayType`: Тип отображения гиперссылки (любой тип).
- `textRotation`: Поворот текста (любой тип).
- `backgroundColorStyle`: Стиль фона ячейки (любой тип).
