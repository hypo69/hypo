morlevi_categories_video.json
============================

This file defines video scenarios for Morlevi categories.  Each scenario maps a specific GPU model to its details like brand, URL, and other relevant attributes.
It's structured as a JSON object containing a "scenarios" key, which holds a dictionary where each key is a GPU model and each value contains properties about that model.
