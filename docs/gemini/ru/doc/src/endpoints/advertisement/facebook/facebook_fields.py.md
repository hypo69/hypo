# Модуль `facebook_fields`

## Обзор

Модуль `facebook_fields` предоставляет класс `FacebookFields`, который предназначен для загрузки и хранения полей, используемых в объявлениях и событиях Facebook. Поля загружаются из JSON-файла и доступны как атрибуты экземпляра класса.

## Классы

### `FacebookFields`

**Описание**: Класс для управления полями объявлений и событий Facebook. Поля загружаются из JSON файла и доступны как атрибуты экземпляра.

**Методы**:
*   `__init__`: Инициализирует экземпляр класса и вызывает метод `_payload` для загрузки полей.
*   `_payload`: Загружает поля из JSON-файла и устанавливает их как атрибуты экземпляра класса.

#### `__init__`

```python
def __init__(self):
    """
    Инициализирует экземпляр класса `FacebookFields`.
    """
```

**Описание**: Инициализирует экземпляр класса `FacebookFields`. Вызывает метод `_payload` для загрузки полей.

**Параметры**:
*   Нет

**Возвращает**:
*   Нет

#### `_payload`

```python
def _payload(self):
    """
    Загружает поля из JSON-файла и устанавливает их как атрибуты экземпляра класса.
    """
```

**Описание**: Загружает данные из JSON файла, который содержит поля для объявлений и событий Facebook, и устанавливает их как атрибуты экземпляра класса.

**Параметры**:
*   Нет

**Возвращает**:
*   `True` : В случае успешной загрузки и установки полей.
*   `None` : Если не удалось загрузить данные из файла.
