# Модуль hypotez/src/suppliers/aliexpress/gui/campaign.py

## Обзор

Этот модуль предоставляет класс `CampaignEditor`, предназначенный для редактирования кампаний AliExpress.  Класс предоставляет графический интерфейс пользователя (GUI) для загрузки, просмотра и подготовки данных кампании, загруженных из JSON файла.

## Классы

### `CampaignEditor`

**Описание**: Класс `CampaignEditor` является виджетом PyQt6, предоставляющим интерфейс для работы с данными кампаний AliExpress.

**Атрибуты**:

- `data: SimpleNamespace`: Содержит данные загруженной кампании.
- `current_campaign_file: str`: Путь к загруженному файлу JSON.
- `editor: AliCampaignEditor`: Объект класса `AliCampaignEditor` для подготовки кампании.

**Методы**:

#### `__init__(self, parent=None, main_app=None)`

**Описание**: Инициализирует виджет `CampaignEditor`.

**Параметры**:

- `parent` (Optional[QWidget], опционально): Родительский виджет. По умолчанию `None`.
- `main_app` (Optional[MainApp], опционально): Объект главного приложения.  По умолчанию `None`.

**Возвращает**:
-  None

#### `setup_ui(self)`

**Описание**:  Настраивает пользовательский интерфейс (UI). Создает виджеты для открытия файла, отображения имени файла и кнопки подготовки кампании.

**Возвращает**:
- None

#### `setup_connections(self)`

**Описание**:  Устанавливает связи между виджетами.

**Возвращает**:
- None

#### `open_file(self)`

**Описание**: Открывает диалог выбора файла JSON с данными кампании. Загружает данные в `CampaignEditor`.

**Параметры**:
- None

**Возвращает**:
- None

**Возможные исключения**:
-  `Exception`: Возникает при ошибке ввода данных.


#### `load_file(self, campaign_file)`

**Описание**: Загружает данные из файла JSON.

**Параметры**:

- `campaign_file (str)`: Путь к файлу JSON.

**Возвращает**:
- None

**Возможные исключения**:
- `Exception`: Возникает при ошибке при загрузке файла.  Сообщение об ошибке отображается пользователю.

#### `create_widgets(self, data)`

**Описание**: Создает виджеты для редактирования данных кампании на основе загруженных данных.

**Параметры**:
- `data (SimpleNamespace)`: Данные кампании.

**Возвращает**:
- None

**Возможные исключения**:
- None


#### `prepare_campaign(self)`

**Описание**: Асинхронно подготавливает кампанию.

**Параметры**:
- None

**Возвращает**:
- None

**Возможные исключения**:
- `Exception`: Возникает при ошибке при подготовке кампании. Сообщение об ошибке отображается пользователю.
