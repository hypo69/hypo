# Модуль `src.bots.discord`

## Обзор

Данный модуль предназначен для интеграции с Discord. Он содержит настройки и общую логику для работы с Discord API.

## Переменные

### `MODE`

**Описание**: Определяет режим работы бота (например, `dev` для разработки).

**Тип**: `str`

**Значение по умолчанию**: `dev`
