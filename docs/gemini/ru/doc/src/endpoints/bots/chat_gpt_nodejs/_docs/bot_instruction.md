# Инструкция для бота

## Обзор

Этот файл содержит инструкцию для бота, определяющую доступные команды и их параметры.

## Подробнее

Данная инструкция предназначена для предоставления информации пользователям о том, как взаимодействовать с ботом. Каждая команда описана с указанием ее синтаксиса и назначения.

## Функции

### `!hi`

**Назначение**: Приветствует пользователя.

**Как работает функция**:
1. Команда `!hi` используется для инициации приветствия от бота.
2. Бот отвечает пользователю приветственным сообщением.

**Примеры**:
- `!hi`

### `!train <data> <data_dir> <positive> <attachment>`

**Назначение**: Обучает модель с предоставленными данными.

**Параметры**:
- `<data>`: Файл с данными для обучения.
- `<data_dir>`: Директория с данными для обучения.
- `<positive>`: Положительные примеры для обучения.
- `<attachment>`: Файл, прикрепленный к сообщению, содержащий данные для обучения.

**Как работает функция**:
1. Команда `!train` используется для обучения модели.
2. В зависимости от предоставленных параметров (файла, директории или прикрепленного файла) модель обучается на соответствующих данных.

**Примеры**:
- `!train file.txt dir positive.txt attachment.txt`

### `!test <test_data>`

**Назначение**: Тестирует модель с предоставленными JSON-данными.

**Параметры**:
- `<test_data>`: JSON-данные для тестирования модели.

**Как работает функция**:
1. Команда `!test` используется для тестирования модели.
2. Модель тестируется на основе предоставленных JSON-данных.

**Примеры**:
- `!test test_data.json`

### `!archive <directory>`

**Назначение**: Архивирует файлы в указанной директории.

**Параметры**:
- `<directory>`: Директория для архивации.

**Как работает функция**:
1. Команда `!archive` используется для архивации файлов.
2. Файлы в указанной директории архивируются.

**Примеры**:
- `!archive /path/to/directory`

### `!select_dataset <path_to_dir_positive> <positive>`

**Назначение**: Выбирает набор данных для обучения из указанной директории.

**Параметры**:
- `<path_to_dir_positive>`: Путь к директории с положительными примерами.
- `<positive>`: Положительные примеры для обучения.

**Как работает функция**:
1. Команда `!select_dataset` используется для выбора набора данных для обучения.
2. Выбираются данные из указанной директории с положительными примерами.

**Примеры**:
- `!select_dataset /path/to/positive positive_examples.txt`

### `!instruction`

**Назначение**: Отображает инструкцию для пользователя.

**Как работает функция**:
1. Команда `!instruction` используется для отображения инструкции пользователю.
2. Бот предоставляет сообщение с описанием доступных команд и их параметров.

**Примеры**:
- `!instruction`
