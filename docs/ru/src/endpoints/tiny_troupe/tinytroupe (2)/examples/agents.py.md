# Модуль с примерами агентов

## Обзор

Модуль содержит примеры создания и загрузки спецификаций для агентов TinyTroupe. Эти примеры можно использовать как есть или модифицировать для создания собственных агентов.

## Подробнее

Модуль предоставляет функции для создания четырех разных агентов: Оскара-архитектора, Лизы-дата-сайентиста, Маркоса-врача и Лилы-лингвиста. Для каждого агента есть два способа создания: загрузка спецификации из файла и программное создание с помощью кода.

## Функции

### `create_oscar_the_architect`

```python
def create_oscar_the_architect():
    """
    Создает агента Оскара-архитектора, загружая спецификацию из файла.

    Returns:
        TinyPerson: Объект агента Oscar.
    """
    ...
```

**Назначение**: Создает агента Оскара-архитектора, загружая его спецификацию из файла.

**Возвращает**:

-   `TinyPerson`: Объект агента Oscar, созданный на основе загруженной спецификации.

**Как работает функция**:

1.  Вызывается функция `load_example_agent_specification("Oscar")`, которая загружает спецификацию агента Oscar из файла.
2.  Вызывается метод `TinyPerson.load_specification()`, который создает объект `TinyPerson` на основе загруженной спецификации.
3.  Возвращается созданный объект `TinyPerson`.

**Примеры**:

```python
oscar = create_oscar_the_architect()
print(oscar.name)  # Oscar
```
