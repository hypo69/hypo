# Модуль `hypotez/src/suppliers/visualdg/header.py`

## Обзор

Этот модуль содержит функции для определения корневого каталога проекта, загрузки настроек из файла `settings.json` и получения информации о проекте из `README.MD`.  Он также содержит переменные, хранящие информацию о проекте, например, имя проекта, версию, описание и автора.


## Функции

### `set_project_root`

**Описание**:  Находит корневой каталог проекта, начиная с текущего каталога и двигаясь вверх по дереву каталогов.  Ищет файлы или каталоги, указанные в аргументе `marker_files`.

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или каталогов, которые используются для определения корневого каталога проекта. По умолчанию: `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневому каталогу проекта, если найден, иначе путь к каталогу, где находится текущий файл.

**Вызывает исключения**:
- Нет.


## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта, полученный с помощью функции `set_project_root`.

**Тип**: `Path`


### `settings`

**Описание**: Словарь с настройками проекта, загруженными из файла `settings.json`.

**Тип**: `dict` или `None`.


### `doc_str`

**Описание**:  Строка документации, прочитанная из файла `README.MD`.

**Тип**: `str` или `None`.


### `__project_name__`

**Описание**: Имя проекта. Получается из `settings.json` или имеет значение по умолчанию `hypotez`.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта. Получается из `settings.json` или имеет значение по умолчанию пустая строка.

**Тип**: `str`


### `__doc__`

**Описание**: Описание проекта. Получается из `settings.json` или имеет значение по умолчанию пустая строка.

**Тип**: `str`


### `__details__`

**Описание**:  Подробное описание проекта. По умолчанию пустая строка.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта. Получается из `settings.json` или имеет значение по умолчанию пустая строка.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права. Получается из `settings.json` или имеет значение по умолчанию пустая строка.

**Тип**: `str`


### `__cofee__`

**Описание**:  Ссылка для пожертвования на чашку кофе автору проекта. Получается из `settings.json` или имеет значение по умолчанию ссылка.

**Тип**: `str`


## Обработка исключений

Файлы `settings.json` и `README.MD` открываются с использованием блоков `try...except`.  В случае `FileNotFoundError` или `json.JSONDecodeError` соответствующие переменные устанавливаются в `None` или пустые строки, и программа продолжает выполнение.
