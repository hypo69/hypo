# Шаблон таблицы прайслиста (he)

## Обзор

Этот файл представляет собой HTML-шаблон для генерации таблицы прайслиста на иврите. Шаблон отображает список продуктов с их изображениями, названиями, описаниями и спецификациями.  Также отображается общая цена.

## Структура

Шаблон использует табличную структуру для вывода информации о продуктах.  Каждый продукт представлен в виде отдельной строки таблицы.


## Элементы

### Заголовок

* `<h1>{{ title }}</h1>`:  Заголовок страницы, динамически подставляется.
* `<p class="lead">{{ description }}</p>`:  Краткое описание или вступление к таблице, динамически подставляется.

### Таблица продуктов

* `<table>`:  Основная таблица для отображения продуктов.
* `<tr class="product-card">`: Строка для каждого продукта, содержащая визуальную карточку.
    * `<h3>{{ product.product_title }}</h3>`: Название продукта.
    * `<table>`: Вложенная таблица, размещающая изображение и описание.
        * `<td><img src="{{ product.image_local_saved_path }}"></td>`: Изображение продукта.
        * `<td><p>{{ product.product_description }}</p><p>{{ product.specification }}</p></td>`: Описание и спецификация продукта.
* `{% for product in products %} ... {% endfor %}`: Цикл для итерации по списку продуктов.

### Общая цена

* `<div class="footer">`: Блок для отображения общей цены.
* `<p>מחיר כוךך הכל: <span class="price-tag">{{ price }} {{ currency }}</span></p>`:  Текст "Общая цена:" и динамические значения цены и валюты.


## Стили

Шаблон содержит CSS для форматирования элементов страницы, в том числе:

* Цвета и шрифты.
* Размеры и отступы.
* Выравнивание текста (справа налево).
* Дизайн карточек продуктов.
* Форматирование цены.


## Динамические данные

Шаблон использует шаблонизатор (например, Jinja2) для динамической подстановки следующих данных:

* `{{ title }}`: Заголовок страницы.
* `{{ description }}`: Описание.
* `{{ products }}`: Список продуктов.
* `{{ product.product_title }}`: Название продукта.
* `{{ product.image_local_saved_path }}`: Путь к изображению продукта.
* `{{ product.product_description }}`: Описание продукта.
* `{{ product.specification }}`: Спецификация продукта.
* `{{ price }}`: Общая цена.
* `{{ currency }}`: Валюта.
