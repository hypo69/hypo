## Обзор

Этот модуль содержит инициализацию для пакета `openai_trаigner`.  Он определяет константы и импорты, необходимые для работы графического интерфейса.

## Константы

### `MODE`

**Описание**: Переменная, определяющая режим работы приложения (например, 'dev' или 'prod').

**Значение**: 'dev' по умолчанию.

**Описание**: Импортирует информацию о версии, документации и других деталях из модуля `version.py` внутри пакета.
