# Модуль `src.logger.header`

## Обзор

Модуль `src.logger.header` определяет корневой путь к проекту и обеспечивает доступ к глобальным настройкам и документации проекта. Все импорты в проекте строятся относительно этого корневого пути.

## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная с каталога текущего файла, ища вверх до первого каталога, содержащего любой из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple): Кортеж с именами файлов или каталогов, которые идентифицируют корень проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневому каталогу, если он найден. В противном случае возвращается каталог, где расположен скрипт.

## Глобальные переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта.

### `settings`

**Описание**: Словарь с настройками проекта, загруженными из `settings.json`.

### `doc_str`

**Описание**: Строка с документацией проекта, прочитанная из `README.MD`.

### `__project_name__`

**Описание**: Название проекта, взятое из `settings.json` или по умолчанию `hypotez`.

### `__version__`

**Описание**: Версия проекта, взятая из `settings.json` или пустая строка.

### `__doc__`

**Описание**: Строка с документацией проекта, взятая из `README.MD` или пустая строка.

### `__details__`

**Описание**: Строка для дополнительных деталей о проекте. По умолчанию пустая строка.

### `__author__`

**Описание**: Автор проекта, взятый из `settings.json` или пустая строка.

### `__copyright__`

**Описание**: Авторские права проекта, взятые из `settings.json` или пустая строка.

### `__cofee__`

**Описание**: Сообщение для поддержки разработчика. По умолчанию "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69" или значение из `settings.json`.
