```python
def get_project_root(marker_files=('pyproject.toml', 'requirements.txt', '.git')) -> Path:
    """!
    Находит корневую директорию проекта, начиная от текущей директории, 
    ища вверх по дереву директорий и останавливаясь на первой директории, содержащей любой из указанных файлов.

    Args:
        marker_files (tuple): Имена файлов или каталогов для определения корня проекта.
    
    Returns:
        Path: Путь к корневой директории, если найдена, иначе — директория, где находится данный скрипт.
    """
    __root__:Path
    current_path:Path = Path(__file__).resolve().parent
    __root__ = current_path
    for parent in [current_path] + list(current_path.parents):
        if any((parent / marker).exists() for marker in marker_files):
            __root__ = parent
            break
    if __root__ not in sys.path:
        sys.path.insert(0, str(__root__))
    return __root__
```

**Описание ключевых частей кода:**

* **`marker_files`:**  Кортеж, содержащий имена файлов или каталогов, по наличию которых определяется корень проекта. Рекомендуется использовать `pyproject.toml`, `requirements.txt` и `.git`, чтобы гарантировать корректную работу в разных сценариях.
* **`__root__`:** Переменная, хранящая путь к корневой директории проекта.
* **`sys.path.insert(0, str(__root__))`:**  Добавляет корень проекта в `sys.path`. Это необходимо для импорта модулей из других директорий проекта. Важно, что он вставляется в начало списка (`sys.path.insert(0, ...)`).

**Использование:**

Код в файле получает корневую директорию проекта при помощи `get_project_root()`, а затем импортирует модули из корневой директории, как показано в примере ниже.
