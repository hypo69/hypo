# Модуль `gworksheets.py`

## Обзор

Модуль `gworksheets.py` предоставляет класс `GWorksheet`, который расширяет функциональность `Worksheet` для работы с Google Sheets. Он позволяет создавать, открывать и модифицировать листы в Google Sheets, а также настраивать их направление и заголовки.

## Классы

### `GWorksheet`

**Описание**:
Класс `GWorksheet` предоставляет интерфейс для работы с листами Google Sheets, наследуя функциональность от класса `Worksheet`.

**Наследует**:
- `Worksheet`: Предоставляет базовый функционал для работы с листами.

**Атрибуты**:
- `sh`: Ссылка на объект `Spreadsheet` (Google Spreadsheet)
- `ws`: Ссылка на объект `Worksheet` (Google Sheet)
- `render`: Экземпляр класса `GSRender` для рендеринга данных в Google Sheets.

**Методы**:

## Функции

### `__init__`

**Описание**:
Конструктор класса `GWorksheet`. Инициализирует объект, получая ссылку на Spreadsheet и название листа. Если листа с указанным именем не существует, он будет создан.

**Параметры**:
- `sh` : Экземпляр класса Spreadsheet, представляющий Google Spreadsheet
- `ws_title` (str, optional): Название листа. По умолчанию `new`.
- `rows` (int, optional): Количество строк для нового листа. По умолчанию `None`.
- `cols` (int, optional): Количество колонок для нового листа. По умолчанию `None`.
- `direcion` (str, optional): Направление листа. По умолчанию `rtl`.
- `wipe_if_exist` (bool, optional): Флаг, указывающий, нужно ли очистить лист, если он существует. По умолчанию `True`.
- `*args`: Произвольные позиционные аргументы
- `**kwards`: Произвольные именованные аргументы

**Возвращает**:
- `None`

### `get`

**Описание**:
Получает или создает лист в Google Sheets. Если лист с именем `ws_title` существует, он будет открыт, иначе будет создан новый лист.

**Параметры**:
- `sh`: Экземпляр класса Spreadsheet, представляющий Google Spreadsheet
- `ws_title` (str, optional): Название листа. По умолчанию `new`.
- `rows` (int, optional): Количество строк для нового листа. По умолчанию `100`.
- `cols` (int, optional): Количество колонок для нового листа. По умолчанию `100`.
- `direction` (str, optional): Направление листа. По умолчанию `rtl`.
- `wipe_if_exist` (bool, optional): Флаг, указывающий, нужно ли очистить лист, если он существует. По умолчанию `True`.

**Возвращает**:
- `None`

### `header`

**Описание**:
Устанавливает заголовок для листа в Google Sheets.

**Параметры**:
- `world_title` (str): Заголовок, который нужно установить.
- `range` (str, optional): Диапазон ячеек для заголовка. По умолчанию `A1:Z1`.
- `merge_type` (str, optional): Тип слияния ячеек заголовка. Может быть `MERGE_ALL`, `MERGE_COLUMNS` или `MERGE_ROWS`. По умолчанию `MERGE_ALL`.

**Возвращает**:
- `None`

### `category`

**Описание**:
Записывает заголовок категории в лист.

**Параметры**:
- `ws_category_title`: Заголовок категории.

**Возвращает**:
- `None`

### `direction`

**Описание**:
Устанавливает направление листа.

**Параметры**:
- `direction` (str, optional): Направление листа. По умолчанию `rtl`.

**Возвращает**:
- `None`
