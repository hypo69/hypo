# Модуль src.webdriver

## Обзор

Модуль `src.webdriver` предоставляет абстракцию для управления веб-драйверами различных браузеров. Он содержит классы для работы с драйверами Chrome, Firefox, Edge, а также вспомогательные классы для других задач (парсинг, использование Crawlee).

## Классы

### `Driver`

**Описание**: Базовый класс для работы с веб-драйверами.  Определяет общие методы для управления браузером.

### `Chrome`

**Описание**: Класс для работы с веб-драйвером Chrome. Наследуется от `Driver`.

### `Firefox`

**Описание**: Класс для работы с веб-драйвером Firefox. Наследуется от `Driver`.


### `Edge`

**Описание**: Класс для работы с веб-драйвером Edge. Наследуется от `Driver`.


### `BS`

**Описание**: Класс для работы с BeautifulSoup для парсинга HTML.


### `Playwrid`

**Описание**: Класс для работы с Playwright для автоматизации браузера.


### `CrawleePython`

**Описание**: Класс для работы с Crawlee Python для парсинга и сбора данных.
