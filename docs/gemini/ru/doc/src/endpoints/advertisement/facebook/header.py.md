# Модуль `header.py`

## Обзор

Модуль `header.py` предназначен для определения основных настроек и метаданных проекта, включая пути к файлам, имя проекта, версию, автора и т.д. Он также загружает настройки из файла `settings.json` и документацию из `README.MD`.

## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная с каталога текущего файла, ища вверх и останавливаясь в первом каталоге, содержащем любой из маркерных файлов.

**Параметры**:

*   `marker_files` (tuple): Имена файлов или каталогов, идентифицирующих корень проекта.

**Возвращает**:

*   `Path`: Путь к корневому каталогу, если он найден, иначе каталог, где расположен скрипт.

## Переменные

### `__root__`
**Описание**: Путь к корневому каталогу проекта.
**Тип**: `Path`

### `settings`
**Описание**: Словарь, содержащий настройки проекта, загруженные из файла `settings.json`.
**Тип**: `dict | None`

### `doc_str`
**Описание**: Строка, содержащая содержимое файла `README.MD`.
**Тип**: `str | None`

### `__project_name__`
**Описание**: Имя проекта, полученное из настроек или "hypotez" по умолчанию.
**Тип**: `str`

### `__version__`
**Описание**: Версия проекта, полученная из настроек или пустая строка по умолчанию.
**Тип**: `str`

### `__doc__`
**Описание**: Документация проекта, полученная из файла `README.MD` или пустая строка по умолчанию.
**Тип**: `str`

### `__details__`
**Описание**: Детали проекта (пока не используется).
**Тип**: `str`

### `__author__`
**Описание**: Автор проекта, полученный из настроек или пустая строка по умолчанию.
**Тип**: `str`

### `__copyright__`
**Описание**: Информация об авторских правах проекта, полученная из настроек или пустая строка по умолчанию.
**Тип**: `str`

### `__cofee__`
**Описание**: Сообщение с предложением поддержать разработчика чашкой кофе, полученное из настроек или текст по умолчанию.
**Тип**: `str`
