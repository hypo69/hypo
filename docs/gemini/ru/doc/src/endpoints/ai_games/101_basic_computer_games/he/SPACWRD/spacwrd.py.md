# SPACWRD

## Обзор

Этот модуль реализует игру "SPACWRD", в которой игрок должен угадать слово из 5 букв, загаданное компьютером. После каждой попытки компьютер предоставляет подсказки: `*` означает правильную букву на правильной позиции, `.` означает правильную букву на неправильной позиции, а пробел означает, что буквы нет в слове.

## Функции

### `None`

**Описание**:
Основной игровой процесс, где компьютер выбирает случайное слово из списка, а игрок пытается его угадать, получая подсказки.

**Параметры**:
-   Нет параметров

**Возвращает**:
-   `None`: Функция ничего не возвращает явно, но выводит сообщения в консоль.

**Вызывает исключения**:
-   `ValueError`: Возникает, если пользователь вводит некорректные данные (не слово или не 5 букв).

## Переменные

### `words`

**Описание**:
Список слов из пяти букв, из которого компьютер выбирает слово для угадывания.

**Тип**:
`list`

### `secretWord`

**Описание**:
Случайно выбранное слово из списка `words`, которое игрок должен угадать.

**Тип**:
`str`

### `numberOfGuesses`

**Описание**:
Счетчик количества попыток, сделанных игроком.

**Тип**:
`int`

### `maxGuesses`

**Описание**:
Максимальное количество попыток, разрешенных игроку.

**Тип**:
`int`
