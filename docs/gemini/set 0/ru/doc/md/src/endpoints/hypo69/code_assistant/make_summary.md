# Модуль make_summary

## Обзор

Модуль `make_summary` отвечает за создание файла `SUMMARY.md`, который используется для компиляции документации с помощью `mdbook`.  Он рекурсивно обходит заданную директорию, собирает имена всех `.md` файлов (исключая `SUMMARY.md`) и генерирует `SUMMARY.md` файл с гиперссылками на эти файлы.

## Функции

### `make_summary`

**Описание**: Создает файл `SUMMARY.md`, рекурсивно обходя папку.

**Параметры**:
- `docs_dir` (Path): Путь к исходной директории с `.md` файлами.

**Возвращает**:
- `None`: Функция не возвращает явного значения.

**Вызывает исключения**:
- Возможны различные исключения при работе с файлами (например, `FileNotFoundError`, `IOError`).


### `_make_summary`

**Описание**: Рекурсивно обходит папку и создает файл `SUMMARY.md` с главами на основе `.md` файлов.

**Параметры**:
- `src_dir` (Path): Путь к папке с исходниками `.md`.
- `summary_file` (Path): Путь для сохранения файла `SUMMARY.md`.

**Возвращает**:
- `bool`: `True`, если файл успешно создан, `False` - в противном случае.  

**Вызывает исключения**:
- Любые исключения, возникающие во время работы с файлами (например, `FileNotFoundError`, `IOError`).  Обработка исключений с использованием `try...except` блоков.


### `prepare_summary_path`

**Описание**: Формирует путь к файлу, заменяя часть пути `'src'` на `'docs'` и добавляя имя файла.

**Параметры**:
- `src_dir` (Path): Исходный путь с `'src'`.
- `file_name` (str, опционально): Имя файла, которое нужно создать. По умолчанию `'SUMMARY.md'`.

**Возвращает**:
- `Path`: Новый путь к файлу.

**Вызывает исключения**:
- Возможны исключения при работе с путями, например, если `src_dir` не валидный путь.

## Использование

Для использования модуля необходимо передать путь к директории с `.md` файлами в функцию `make_summary`.  Функция создаст или перезапишет файл `SUMMARY.md` в соответствующей подпапке `docs`.


```
from pathlib import Path
from make_summary import make_summary

# Пример использования
docs_dir = Path("./src/endpoints/hypo69/docs") # Путь к папке с .md файлами
make_summary(docs_dir)
```
