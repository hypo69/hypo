# Задача No 183
## Максимизация произведений

## Условия

Пусть N — целое положительное число, и пусть N разбивается на k равных частей, каждое размером N/k.

Пусть P(N,k) будет произведением этих частей, то есть, P(N,k) = (N/k)^k.

Для N=10,

| k | P(10,k) |
|---|---|
| 1 | 10       |
| 2 | 25       |
| 3 | 37.0370370... |
| 4 | 39.0625  |
| 5 | 32       |
| 6 | 21.433470... |
| 7 | 14.74455... |
| 8 | 10.000000 |
| 9 | 6.94873... |
| 10 | 1.00000000 |

Оказывается, что максимум P(10,k) достигается при k=4.

Пусть M(N) — значение k, для которого P(N,k) является максимумом. В вышеприведенном примере M(10) = 4.

Если мы исследуем значения M(N) для 5 <= N <= 100, то обнаружим, что их сумма равна 255.

Найдите сумму M(N) для 5 <= N <= 10000.
