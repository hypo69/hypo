# Модуль translate_product_fields

## Обзор

Данный модуль предоставляет функции для работы с переводами полей товара.  Он взаимодействует с базой данных PrestaShop, осуществляет перевод текстовых полей с помощью API и сохраняет результаты в базе данных.

## Функции

### `get_translations_from_presta_translations_table`

**Описание**: Функция извлекает переводы полей товара из таблицы переводов в базе данных PrestaShop.

**Параметры**:
- `product_reference` (str): Уникальный идентификатор товара.
- `credentials` (dict): Словарь с данными для авторизации и подключения к базе данных PrestaShop.
- `i18n` (str, optional): Код языка перевода (например, `en_EN`, `he_HE`, `ru-RU`). По умолчанию `None`.

**Возвращает**:
- `list`: Список словарей с переводами полей товара.  Возвращает пустой список, если записи с указанным `product_reference` не найдено.

**Вызывает исключения**:
- Возможны исключения, связанные с подключением к базе данных PrestaShop,  указанными в `ProductTranslationsManager`.


### `insert_new_translation_to_presta_translations_table`

**Описание**: Функция добавляет новую запись с переводом в таблицу переводов.


**Параметры**:
- `record` (dict): Словарь с данными для новой записи (перевод).
- `credentials` (dict): Словарь с данными для авторизации и подключения к базе данных PrestaShop.

**Возвращает**:
- `None`: Функция не возвращает значений.

**Вызывает исключения**:
- Возможны исключения, связанные с подключением к базе данных PrestaShop,  указанными в `ProductTranslationsManager`.


### `translate_record`

**Описание**: Функция переводит поля товара с одного языка на другой.

**Параметры**:
- `record` (dict): Словарь с полями товара, которые нужно перевести.
- `from_locale` (str): Код исходного языка (например, `en_EN`).
- `to_locale` (str): Код целевого языка (например, `he_HE`).


**Возвращает**:
- `dict`: Словарь с переведенными полями товара.


**Вызывает исключения**:
- Возможны исключения, связанные с API перевода (модуль `translate`).
