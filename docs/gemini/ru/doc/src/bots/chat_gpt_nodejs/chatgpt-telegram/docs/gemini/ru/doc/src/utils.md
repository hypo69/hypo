# Модуль `utils.js`

## Обзор

Этот модуль содержит функцию для удаления файла.

## Функции

### `removeFile`

**Описание**: Асинхронная функция для удаления файла по указанному пути.

**Параметры**:

- `path` (строка): Путь к файлу, который необходимо удалить.

**Возвращает**:

- `undefined`: Функция не возвращает никакого значения.

**Обрабатывает исключения**:

- `Error`: В случае ошибки при удалении файла, функция выводит сообщение об ошибке в консоль и переходит к следующей строке кода.


```javascript
import { unlink } from 'fs/promises'
export async function removeFile(path) {
    try {
        await unlink(path)
    } catch (e) {
        console.log('Error while removing file', e.message)
    }
}
```
