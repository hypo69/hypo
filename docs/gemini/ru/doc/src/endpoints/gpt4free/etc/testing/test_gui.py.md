# Модуль для запуска графического интерфейса g4f

## Обзор

Модуль предназначен для запуска графического интерфейса (GUI) библиотеки `g4f`. Он добавляет родительский каталог в `sys.path` для импорта модулей из проекта `hypotez` и запускает графический интерфейс с помощью функции `run_gui`.

## Подробней

Этот модуль обеспечивает простой способ запуска графического интерфейса библиотеки `g4f`, что позволяет пользователям взаимодействовать с функциональностью `g4f` через визуальный интерфейс. Добавление родительского каталога в `sys.path` необходимо для корректного импорта модулей внутри проекта `hypotez`.
