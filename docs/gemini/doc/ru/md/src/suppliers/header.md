# Модуль `hypotez/src/suppliers/header.py`

## Обзор

Этот модуль отвечает за инициализацию проекта, нахождение корневой директории проекта, загрузку настроек из файла `settings.json` и чтение документации из файла `README.MD`. Он также устанавливает корневую директорию в `sys.path` для корректного импорта модулей.

## Функции

### `set_project_root`

**Описание**: Функция находит корневую директорию проекта, начиная с текущего файла и проходя вверх по директориям.

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или директорий, по которым определяется корневая директория проекта. По умолчанию содержит `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории проекта, если найдена, иначе директория, в которой расположен скрипт.

**Примеры использования**:

```python
root_path = set_project_root()
print(root_path)
```

**Обрабатывает исключения**:
- Не обрабатывает исключения.


### `settings_file`

**Описание**: Функция загружает настройки из файла `settings.json`.

**Возвращает**:
- `dict`: Словарь с настройками проекта, если файл найден и корректный, иначе `None`.


### `doc_file`

**Описание**: Функция считывает документацию из файла `README.MD`.

**Возвращает**:
- `str`: Содержимое файла `README.MD`, если файл найден, иначе `None`.

**Обрабатывает исключения**:
- `FileNotFoundError`, `json.JSONDecodeError`: Если файл `settings.json` не найден или некорректен.
- `FileNotFoundError`, `json.JSONDecodeError`: Если файл `README.MD` не найден или некорректен.



## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта.

**Тип**: `Path`

**Значение**: Результат вызова функции `set_project_root`.

### `settings`

**Описание**: Словарь с настройками проекта.

**Тип**: `dict`

**Значение**: Загруженные настройки из файла `settings.json`. По умолчанию `None`.

### `doc_str`

**Описание**: Содержимое файла `README.MD`.

**Тип**: `str`

**Значение**: Загруженное содержимое документации из файла `README.MD`. По умолчанию `None`.


### `__project_name__`

**Описание**: Название проекта.

**Тип**: `str`

**Значение**: Значение поля `project_name` из настроек (`settings`), по умолчанию `'hypotez'`.


### `__version__`

**Описание**: Версия проекта.

**Тип**: `str`

**Значение**: Значение поля `version` из настроек (`settings`), по умолчанию пустая строка.


### `__doc__`

**Описание**: Документация проекта.

**Тип**: `str`

**Значение**: Содержимое файла `README.MD`, по умолчанию пустая строка.


### `__details__`

**Описание**: Дополнительные детали о проекте.

**Тип**: `str`

**Значение**: Поле `details` из настроек (`settings`), по умолчанию пустая строка.

### `__author__`

**Описание**: Автор проекта.

**Тип**: `str`

**Значение**: Значение поля `author` из настроек (`settings`), по умолчанию пустая строка.

### `__copyright__`

**Описание**: Авторские права.

**Тип**: `str`

**Значение**: Значение поля `copyright` из настроек (`settings`), по умолчанию пустая строка.


### `__cofee__`

**Описание**: Ссылка для поддержки автора.

**Тип**: `str`

**Значение**: Значение поля `cofee` из настроек (`settings`), по умолчанию ссылка на Boosty.
