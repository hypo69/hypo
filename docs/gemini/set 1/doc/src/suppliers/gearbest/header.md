# Модуль hypotez/src/suppliers/gearbest/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`. Он также предоставляет доступ к документирующей строке (`README.MD`) проекта и различным метаданным, таким как имя проекта, версия, автор и т.д.


## Функции

### `set_project_root`

**Описание**: Определяет корневую директорию проекта, начиная с текущего файла и идя вверх по иерархии директорий.  Использует заданный список файлов/директорий для определения корня проекта.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, которые должны присутствовать в пути к корню проекта. По умолчанию: `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена. В противном случае возвращает путь к директории, где расположен текущий файл.

**Примеры использования**:

```python
root_path = set_project_root()
print(root_path)
```


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root`.

**Тип**: `Path`

**Значение**: Значение зависит от расположения файла и структуры проекта.


### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json`.

**Тип**: `dict` | `None`

**Значение**: Словарь с настройками или `None`, если файл `settings.json` не найден или содержит недопустимый JSON.


### `doc_str`

**Описание**: Содержимое файла `README.MD`, если файл существует.

**Тип**: `str` | `None`

**Значение**: Строка с содержимым файла `README.MD` или `None`, если файл не найден или не содержит корректный текст.


### `__project_name__`

**Описание**: Имя проекта, полученное из настроек или значение по умолчанию.

**Тип**: `str`

**Значение**: Строка с именем проекта.


### `__version__`

**Описание**: Версия проекта, полученная из настроек или значение по умолчанию.

**Тип**: `str`

**Значение**: Строка с версией проекта.


### `__doc__`

**Описание**: Документирующая строка проекта, полученная из файла `README.MD`

**Тип**: `str`

**Значение**: Строка с документирующей строкой проекта


### `__details__`

**Описание**: Дополнительная информация о проекте.

**Тип**: `str`

**Значение**: Строка с дополнительной информацией о проекте.


### `__author__`

**Описание**: Автор проекта, полученный из настроек или значение по умолчанию.

**Тип**: `str`

**Значение**: Строка с именем автора.


### `__copyright__`

**Описание**: Авторские права, полученные из настроек или значение по умолчанию.

**Тип**: `str`

**Значение**: Строка с информацией об авторских правах.


### `__cofee__`

**Описание**: Ссылка на платёжную систему для пожертвований автору за чашечку кофе.

**Тип**: `str`

**Значение**: Строка со ссылкой на платформу для сбора средств.
