# Модуль hypotez/src/suppliers/aliexpress/campaign/gsheets_check_this_code.py

## Обзор

Данный модуль предоставляет класс `AliCampaignGoogleSheet` для работы с Google Таблицами в контексте рекламных кампаний AliExpress. Он наследует класс `SpreadSheet` и добавляет методы для управления листами, записи данных о категориях и продуктах, а также форматирования листов.  Модуль использует библиотеки `gspread`, `gspread-formatting` для взаимодействия с Google Таблицами, `src.webdriver.driver` для работы с браузерами, `src.utils` для вспомогательных функций и логирования.

## Классы

### `AliCampaignGoogleSheet`

**Описание**: Класс для работы с Google Таблицами в рамках кампаний AliExpress.  Предназначен для создания и управления рабочими листами, записи данных о категориях и продуктах, а также форматирования листов.

**Атрибуты**:

- `spreadsheet_id`: Идентификатор Google Таблицы.
- `spreadsheet`: Экземпляр класса `SpreadSheet`.
- `worksheet`: Экземпляр класса `Worksheet` (рабочий лист).
- `driver`: Экземпляр класса `Driver`, использующий браузер Chrome по умолчанию.

**Методы**:

#### `__init__`

**Описание**: Инициализирует экземпляр класса `AliCampaignGoogleSheet`.

**Параметры**:

- `campaign_name` (str): Название рекламной кампании.
- `language` (str | dict, optional): Язык кампании. По умолчанию None.
- `currency` (str, optional): Валюта кампании. По умолчанию None.

**Возвращает**:  None.

#### `clear`

**Описание**: Очищает содержимое Google Таблицы, удаляя листы продуктов и очищая данные на листах категорий и других.

**Возвращает**: None.

#### `delete_products_worksheets`

**Описание**: Удаляет все листы Google Таблицы, кроме 'categories' и 'product_template'.

**Возвращает**: None.


#### `set_campaign_worksheet`

**Описание**: Записывает данные рекламной кампании на лист Google Таблицы 'campaign'.

**Параметры**:

- `campaign` (SimpleNamespace): Объект с данными о кампании.

**Возвращает**: None.


#### `set_products_worksheet`

**Описание**: Записывает данные о продуктах категории на лист Google Таблицы.

**Параметры**:

- `category_name` (str): Название категории.

**Возвращает**: None.


#### `set_categories_worksheet`

**Описание**: Записывает данные о категориях на лист Google Таблицы 'categories'.

**Параметры**:

- `categories` (SimpleNamespace): Объект с данными о категориях.

**Возвращает**: None.


#### `get_categories`

**Описание**: Возвращает данные о категориях из листа Google Таблицы 'categories'.

**Возвращает**: list[dict]: Список словарей с данными о категориях.

#### `set_category_products`

**Описание**: Записывает данные о продуктах в новую таблицу Google Sheets.

**Параметры**:

- `category_name` (str): Название категории.
- `products` (dict): Данные о продуктах.

**Возвращает**: None.



#### `_format_categories_worksheet`

**Описание**: Форматирует лист 'categories'.

**Параметры**:

- `ws` (Worksheet): Рабочий лист.


**Возвращает**: None.


#### `_format_category_products_worksheet`

**Описание**: Форматирует лист с продуктами категории.

**Параметры**:

- `ws` (Worksheet): Рабочий лист.

**Возвращает**: None.
