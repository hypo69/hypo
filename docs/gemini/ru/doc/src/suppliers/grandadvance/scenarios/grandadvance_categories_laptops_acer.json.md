# Документация для `grandadvance_categories_laptops_acer.json`

## Обзор

Этот файл содержит JSON-конфигурацию для парсинга категорий ноутбуков Acer на сайте GrandAdvance. Конфигурация включает настройки для различных моделей ноутбуков, основываясь на типе процессора и размере экрана. Для каждой модели определены условия для фильтрации, URL страницы и соответствующие категории PrestaShop.

## Структура JSON

JSON-файл представляет собой словарь, где каждый ключ – это название модели ноутбука (например, "ACER 11.6 I3"), а значение – это словарь с настройками для этой модели.

## Описание параметров

### Ключи верхнего уровня (название модели)

-   **`"ACER 11.6 I3"`**, **`"ACER 11.6 i5"`**, и т.д.:
    -   `string`: Название модели ноутбука. Используется как ключ для идентификации модели и её конфигурации.

### Вложенный словарь (настройки модели)

-   **`brand`**:
    -   `string`: Производитель ноутбука, всегда `"ACER"` в данном файле.
-   **`url`**:
    -   `string`: URL страницы сайта GrandAdvance, откуда необходимо парсить товары.
-   **`checkbox`**:
    -   `dict`: Словарь, содержащий настройки фильтров для процессора и размера экрана.
    -   **`cpu`**:
        -   `dict`: Настройки для фильтрации по типу процессора.
            -   `class`: `string` CSS класс элемента для фильтрации.
            -   `by`: `string`: Тип селектора, всегда `"css selector"` в данном файле.
            -   `value`: `list`: Список допустимых значений для фильтрации процессоров.
    -   **`screensize`**:
        -   `dict`: Настройки для фильтрации по размеру экрана.
            -   `class`: `string`: CSS класс элемента для фильтрации.
            -   `by`: `string`: Тип селектора, всегда `"css selector"` в данном файле.
            -   `value`: `list`: Список допустимых значений для фильтрации размеров экрана.
-   **`active`**:
    -   `bool`: Указывает, активна ли данная конфигурация для парсинга. Всегда `true` в данном файле.
-   **`condition`**:
    -   `string`: Условие товара, всегда `"new"` в данном файле.
-   **`presta_categories`**:
    -   `string`: Строка, содержащая идентификаторы категорий PrestaShop, к которым относится данный товар. Идентификаторы разделены запятыми.
