**1. Настройка корневого каталога проекта:**

```python

    # ... (код для нахождения корневого каталога)

```

Функция `set_project_root` находит корневой каталог проекта, начиная с текущей директории и идя вверх по иерархии директорий. Она ищет файлы `pyproject.toml`, `requirements.txt` и `.git`.  Нахождение любого из этих маркеров означает, что текущая директория — корень проекта. Если корневой каталог не найден, возвращается директория, в которой находится исполняемый файл.  Эта функция добавляет найденный корневой каталог в `sys.path`, что позволяет импортировать модули из проекта.
