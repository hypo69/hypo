# `keyboard.py`

## Обзор

Модуль `keyboard.py` содержит определения inline-клавиатур для использования в Telegram-боте. Он включает в себя клавиатуры для поиска фильмов и выбора типа контента (фильм или сериал).

## Inline-клавиатуры

### `find_movie`

**Описание**:
Inline-клавиатура, содержащая кнопку "Найти", которая вызывает callback-запрос `new_movies`. Используется для инициации поиска фильмов.

### `choice`

**Описание**:
Inline-клавиатура, предлагающая пользователю выбор между сериалом и фильмом. Содержит кнопки "Сериал" (callback `series`) и "Фильм" (callback `film`).
