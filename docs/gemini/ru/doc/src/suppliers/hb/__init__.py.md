# Модуль `hypotez.src.suppliers.hb`

## Обзор

Модуль `hypotez.src.suppliers.hb` предназначен для работы с поставщиком HB. Он содержит функциональность для сбора данных (граббер) и настройки режима работы.

## Константы

### `MODE`

**Описание**: Режим работы модуля. По умолчанию установлен в `dev`.
