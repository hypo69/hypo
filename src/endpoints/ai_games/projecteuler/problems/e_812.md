# Задача No 812
## Сумма первых k простых чисел в ряде Фибоначчи

## Условия
Рассмотрим ряд Фибоначчи F(n), определенный как:

F(0) = 0
F(1) = 1
F(n) = F(n-1) + F(n-2) для n > 1

Рассмотрим последовательность простых чисел P(n) в ряде Фибоначчи, отсортированную по возрастанию. Первые несколько простых чисел в ряде Фибоначчи: 2, 3, 5, 13, 89, 233,...

Пусть S(k) - это сумма первых k простых чисел в ряде Фибоначчи.

Например, S(6) = 2 + 3 + 5 + 13 + 89 + 233 = 343.

Найдите S(100).
