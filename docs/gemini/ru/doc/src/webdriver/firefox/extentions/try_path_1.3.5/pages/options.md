# Модуль options.js

## Обзор

Данный JavaScript-файл отвечает за обработку настроек расширения "try_xpath_1.3.5" для Firefox.  Он получает настройки из хранилища расширения, отображает их в интерфейсе пользователя и позволяет сохранить изменённые настройки. Также файл включает валидацию введённых значений для предотвращения ошибок.

## Функции

### `isValidAttrName`

**Описание**: Проверяет, является ли переданное имя атрибута допустимым для установки на элемент DOM.

**Параметры**:

* `name` (string): Имя атрибута.

**Возвращает**:

* boolean: `true`, если имя атрибута допустимо, иначе `false`.

**Обрабатываемые исключения**:

* `Error`: Возможные ошибки при попытке установить атрибут.


### `isValidAttrNames`

**Описание**: Проверяет, являются ли все имена атрибутов в переданном объекте допустимыми.

**Параметры**:

* `names` (object): Объект, ключи которого представляют имена атрибутов.

**Возвращает**:

* boolean: `true`, если все имена атрибутов допустимы, иначе `false`.


### `isValidStyleLength`

**Описание**: Проверяет, является ли переданное значение длины стилей допустимым.

**Параметры**:

* `len` (string): Значение длины стилей.

**Возвращает**:

* boolean: `true`, если значение длины допустимо, иначе `false`.


### `loadDefaultCss`

**Описание**: Загружает CSS-стили по умолчанию из файла `/css/try_xpath_insert.css`.

**Возвращает**:

* Promise<string>: Promise, который разрешается с текстом загруженного CSS, или отклоняется с ошибкой.


### `extractBodyStyles`

**Описание**: Извлекает значения ширины и высоты из CSS-текста.

**Параметры**:

* `css` (string): CSS-текст.

**Возвращает**:

* object: Объект, содержащий значения `width` и `height`, или пустой объект, если значения не найдены.


### `createPopupCss`

**Описание**: Создаёт CSS-правило для body всплывающего окна.

**Параметры**:

* `bodyStyles` (object): Объект, содержащий значения `width` и `height` для body.

**Возвращает**:

* string: CSS-правило для body.


## Переменные (константы)

### `defaultAttributes`

**Описание**: Объект, содержащий значения атрибутов по умолчанию.

### `defaultPopupBodyStyles`

**Описание**: Объект, содержащий значения стилей по умолчанию для body всплывающего окна.
