# Модуль example_module

## Обзор

Этот модуль предоставляет функции для работы с данными.

## Функции

### `calculate_average`

**Описание**: Функция вычисляет среднее значение списка чисел.

**Параметры**:
- `numbers` (list[float]): Список чисел для вычисления среднего.

**Возвращает**:
- `float`: Среднее значение списка чисел. Возвращает `None`, если список пустой.

**Вызывает исключения**:
- `TypeError`: Если входной параметр `numbers` не является списком чисел.
