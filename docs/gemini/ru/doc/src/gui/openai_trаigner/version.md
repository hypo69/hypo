# Модуль hypotez/src/gui/openai_trаigner/version.py

## Обзор

Этот модуль содержит информацию о версии проекта, а также другие метаданные, такие как автор, лицензия и ссылки.

## Переменные

### `MODE`

**Описание**: Переменная, хранящая режим работы.

**Значение**: `'dev'`


### `__version__`

**Описание**: Строковая переменная, содержащая версию проекта.

**Значение**: `'v1.1'`


### `__doc__`

**Описание**: Строковая переменная, содержащая документацию к модулю.

**Значение**:  `''`


### `__details__`

**Описание**: Строковая переменная, содержащая подробную информацию о модуле.

**Значение**: `''`


### `__author__`

**Описание**: Строковая переменная, содержащая имя автора.

**Значение**: `'hypo69'`


### `__copyright__`

**Описание**: Строковая переменная, содержащая информацию о лицензии.

**Значение**:  (Текст лицензии, который будет отображаться в markdown)


### `__cofee__`

**Описание**: Строковая переменная, содержащая ссылку на поддержку разработчика.

**Значение**: "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69"


## Лицензия

Полный текст лицензии представлен в переменной `__copyright__`.
