# Модуль hypotez/src/suppliers/aliexpress/api/version.py

## Обзор

Модуль `version.py` содержит информацию о версии модуля или пакета, его имя, документацию, дополнительные детали, аннотации типов и имя автора.

## Переменные

### `__version__`

**Описание**: Переменная, хранящая версию модуля или пакета.

**Значение**: `"3.12.0.0.0.4"`


### `__name__`

**Описание**:  Имя модуля. Если скрипт запускается напрямую, значение будет `"__main__"`.

**Тип**: `str`


### `__doc__`

**Описание**: Строка документации модуля.

**Тип**: `str`


### `__details__`

**Описание**:  Переменная, содержащая дополнительные детали о модуле или классе.

**Тип**: `str`


### `__annotations__`

**Описание**: Содержит аннотации типов для переменных и функций в модуле.

**Тип**:  Не определен, но предполагается, что это словарь или другая структура, содержащая аннотации.


### `__author__`

**Описание**: Имя(на) автора(ов) модуля.

**Значение**: `"hypotez "`
