# Модуль hypotez/src/logger/header.py

## Обзор

Модуль `hypotez/src/logger/header.py` определяет корневой путь к проекту `hypotez`.  Все импорты строятся относительно этого пути. В дальнейшем планируется перенести определение корневого пути в системную переменную.  Модуль также загружает настройки из файла `settings.json` и описание проекта из `README.MD`, если они существуют.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная от директории текущего файла, перемещаясь вверх по иерархии директорий.  Останавливается на первой директории, содержащей один из указанных маркеров файлов.

**Параметры**:

- `marker_files` (tuple): Кортеж из имен файлов или директорий, используемых для идентификации корневой директории проекта. По умолчанию: `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории, если найдена, иначе путь к директории текущего файла.

**Вызывает исключения**:

- Не вызывает исключений.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root()`.

**Тип**: `Path`

## Загрузка настроек

**Описание**:  Загружает настройки из файла `settings.json` в переменную `settings`. Обрабатывает возможные ошибки (файл не найден, ошибка декодирования JSON).


## Загрузка документации

**Описание**: Загружает описание проекта из файла `README.MD` в переменную `doc_str`, если он существует. Обрабатывает возможные ошибки (файл не найден, ошибка декодирования).


##  Переменные проекта

**Описание**: Переменные, содержащие информацию о проекте, загруженные из файла `settings.json`.


- `__project_name__`: Название проекта. По умолчанию `hypotez`.
- `__version__`: Версия проекта. По умолчанию пустая строка.
- `__doc__`: Описание проекта. По умолчанию пустая строка.
- `__details__`: Дополнительные детали о проекте. По умолчанию пустая строка.
- `__author__`: Автор проекта. По умолчанию пустая строка.
- `__copyright__`: Авторские права. По умолчанию пустая строка.
- `__cofee__`: Ссылка на платежную систему, для пожертвований разработчику. По умолчанию ссылка на Boosty.


**Примечание**: Значения переменных могут быть `None`, если соответствующие поля отсутствуют в `settings.json` или файл `settings.json` не найден.
