# Модуль `boxing.py`

## Обзор

Этот модуль реализует игру "Бокс" между двумя игроками, где каждый игрок по очереди наносит удары, снижая здоровье противника. Игра заканчивается, когда здоровье одного из игроков достигает нуля или меньше.

В данном модуле нет объявленных функций.

## Переменные

### `player1Health`

**Описание**:
Количество здоровья игрока 1. Инициализируется значением 100.

### `player2Health`

**Описание**:
Количество здоровья игрока 2. Инициализируется значением 100.

### `currentPlayer`

**Описание**:
Игрок, чей ход сейчас. Инициализируется значением 1 (игрок 1).

### `attackDamage`

**Описание**:
Количество нанесенного урона во время текущего раунда. Рассчитывается случайно в диапазоне от 0 до 20.

## Подробное описание кода

1. **Импорт модуля `random`**:
   - `import random`: Импортирует модуль `random` для генерации случайных чисел, используемых для определения урона от удара.
2. **Инициализация переменных**:
   - `player1Health = 100`: Устанавливает начальное здоровье игрока 1 в 100.
   - `player2Health = 100`: Устанавливает начальное здоровье игрока 2 в 100.
   - `currentPlayer = 1`: Устанавливает текущего игрока на игрока 1.
3. **Основной игровой цикл `while`**:
   - Цикл продолжается, пока здоровье обоих игроков больше 0.
   - **Вывод текущего состояния здоровья**:
     - `print(f"שחקן 1 חיים: {player1Health}")`: Выводит текущее здоровье игрока 1.
     - `print(f"שחקן 2 חיים: {player2Health}")`: Выводит текущее здоровье игрока 2.
   - **Генерация урона**:
     - `attackDamage = random.randint(0, 20)`: Генерирует случайный урон от 0 до 20.
     - `print(f"שחקן {currentPlayer} הנחית מכה של {attackDamage} נקודות נזק.")`: Выводит сообщение о нанесенном уроне.
   - **Нанесение урона**:
     - `if currentPlayer == 1:`: Если текущий игрок - игрок 1, то:
       - `player2Health -= attackDamage`: Уменьшает здоровье игрока 2 на величину нанесенного урона.
     - `else:`: Если текущий игрок - игрок 2, то:
       - `player1Health -= attackDamage`: Уменьшает здоровье игрока 1 на величину нанесенного урона.
   - **Проверка условий завершения игры**:
     - `if player1Health <= 0 or player2Health <= 0:`: Если здоровье хотя бы одного игрока стало меньше или равно 0, игра заканчивается.
       - `break`: Выход из цикла.
   - **Смена игрока**:
     - `if currentPlayer == 1:`: Если текущий игрок - игрок 1, то:
       - `currentPlayer = 2`: Переключается на игрока 2.
     - `else:`: Если текущий игрок - игрок 2, то:
       - `currentPlayer = 1`: Переключается на игрока 1.
4. **Определение победителя**:
   - `if player1Health <= 0:`: Если здоровье игрока 1 меньше или равно 0, то:
     - `print("שחקן 2 ניצח!")`: Выводит сообщение о победе игрока 2.
   - `else:`: В противном случае (если здоровье игрока 2 меньше или равно 0), то:
     - `print("שחקן 1 ניצח!")`: Выводит сообщение о победе игрока 1.
