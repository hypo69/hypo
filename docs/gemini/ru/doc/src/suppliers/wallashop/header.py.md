## Функции

### `set_project_root`

**Описание**:
Находит корневой каталог проекта, начиная с каталога текущего файла,
просматривая вверх и останавливаясь на первом каталоге, содержащем любой из маркерных файлов.

**Параметры**:
- `marker_files` (tuple): Кортеж с именами файлов или каталогов, которые идентифицируют корень проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневому каталогу, если он найден, иначе - каталог, где находится скрипт.

## Переменные

### `MODE`

**Описание**:
Режим работы приложения, по умолчанию установлен в `dev`.

### `__root__`

**Описание**:
Путь к корневому каталогу проекта. Определяется функцией `set_project_root`.

### `settings`

**Описание**:
Словарь с настройками, загруженными из файла `settings.json`. Значение `None`, если файл не найден или произошла ошибка декодирования JSON.

### `doc_str`

**Описание**:
Строка, содержащая документацию из файла `README.MD`. Значение `None`, если файл не найден или произошла ошибка декодирования JSON.

### `__project_name__`

**Описание**:
Имя проекта, извлекается из настроек или по умолчанию `hypotez`.

### `__version__`

**Описание**:
Версия проекта, извлекается из настроек или по умолчанию пустая строка.

### `__doc__`

**Описание**:
Строка документации проекта, загруженная из файла `README.MD`.

### `__details__`

**Описание**:
Детальное описание проекта (пустая строка по умолчанию).

### `__author__`

**Описание**:
Имя автора проекта, извлекается из настроек или по умолчанию пустая строка.

### `__copyright__`

**Описание**:
Информация об авторских правах проекта, извлекается из настроек или по умолчанию пустая строка.

### `__cofee__`

**Описание**:
Текст с предложением угостить разработчика кофе, извлекается из настроек или по умолчанию "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69".
