# Модуль hypotez/src/suppliers/aliexpress/gui/styles.py

## Обзор

Данный модуль содержит функции для настройки размеров элементов пользовательского интерфейса (UI) в приложении.

## Функции

### `set_fixed_size`

**Описание**: Устанавливает фиксированный размер для заданного виджета PyQt6.

**Параметры**:
- `widget` (QtWidgets.QWidget): Виджет, для которого необходимо установить размер.
- `width` (int): Ширина виджета.
- `height` (int): Высота виджета.

**Возвращает**:
-  (None): Функция не возвращает значение.

**Вызывает исключения**:
- Нет.
