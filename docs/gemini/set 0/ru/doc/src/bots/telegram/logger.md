# Модуль logger для телеграм-ботов

## Обзор

Модуль `logger` предоставляет инструменты для логгирования событий в телеграм-ботах. Он предназначен для работы в режимах разработки (`dev`) и производства (`prod`), позволяя настраивать уровень подробности логов.

## Константы

### `MODE`

**Описание**: Константа, определяющая текущий режим работы (разработка или производство).

**Значение**: `"dev"`.


## Функции

### `log_message`

**Описание**: Функция для записи сообщений в лог.

**Параметры**:
- `message` (str): Текст сообщения, которое нужно записать в лог.
- `level` (str, optional): Уровень важности сообщения (например, `info`, `warning`, `error`). По умолчанию `info`.
- `additional_info` (dict, optional): Дополнительная информация, которая может быть включена в лог (например, данные о пользователе, или дополнительные контекстные данные).


**Возвращает**:
- `None`.

**Вызывает исключения**:
- `TypeError`: Если `message` не является строкой.
