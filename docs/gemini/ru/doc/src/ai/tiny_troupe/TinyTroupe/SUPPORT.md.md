## Как сообщать о проблемах и получать помощь

Этот проект использует GitHub Issues для отслеживания ошибок и запросов на новые функции. Пожалуйста, выполните поиск по существующим проблемам перед тем, как создавать новые, чтобы избежать дублирования. Для новых проблем, пожалуйста, оформите свою ошибку или запрос на новую функцию как новую Issue.

Для получения помощи и ответов на вопросы об использовании этого проекта, пожалуйста, обратитесь к нашей Wiki или опубликуйте сообщение на нашем форуме обсуждений, оба здесь, на GitHub.

## Политика поддержки Microsoft

Поддержка TinyTroupe ограничивается ресурсами, перечисленными выше.
