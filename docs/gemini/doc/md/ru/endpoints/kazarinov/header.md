# Модуль header

## Обзор

Данный модуль (`hypotez/src/endpoints/kazarinov/header.py`) содержит функции для определения корневой директории проекта и получения настроек проекта из файла `settings.json`. Он также предоставляет информацию о проекте, такую как название, версия, документация, автор и т.д.

## Оглавление

- [Функции](#функции)
    - [get\_project\_root](#get_project_root)


## Функции

### `get_project_root`

**Описание**: Находит корневую директорию проекта, начиная от текущей директории. Поиск выполняется вверх по иерархии директорий до тех пор, пока не будет найдена директория, содержащая один из файлов или папок из списка `marker_files`.

**Параметры**:
- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или папок, которые используются для определения корневой директории проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории проекта, если она найдена, иначе текущая директория.

**Пример использования**:
```python
root_path = get_project_root()
print(root_path)
```

**Примечания**:
- Функция добавляет корневую директорию проекта в `sys.path`, если она еще не присутствует. Это необходимо для импорта модулей из других директорий проекта.

### Функции работы с файлами

Несмотря на то, что в исходном коде присутствуют функции работы с файлами `settings.json` и `README.MD`, они не имеют отдельных разделов в данной документации, так как они не являются отдельными функциями, а представляют собой части кода, выполняющие чтение и обработку этих файлов.


## Переменные

### `MODE`

**Описание**: Строковая переменная, содержащая значение режима работы (в данном случае 'development').

### `__root__`

**Описание**: Путь к корневой директории проекта. Получена с помощью функции `get_project_root()`.

### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json`.  Может быть `None`, если файл не найден или не удалось его разобрать.

### `doc_str`

**Описание**: Строка с содержимым файла `README.MD`.  Может быть `None`, если файл не найден или не удалось его разобрать.

### `__project_name__`

**Описание**: Название проекта. Получено из настроек (`settings`). По умолчанию 'hypotez'.

### `__version__`

**Описание**: Версия проекта. Получена из настроек (`settings`). По умолчанию ''.

### `__doc__`

**Описание**: Документация проекта. Получена из файла `README.MD` . По умолчанию ''.

### `__details__`

**Описание**: Подробное описание проекта. По умолчанию ''.

### `__author__`

**Описание**: Автор проекта. Получен из настроек (`settings`). По умолчанию ''.

### `__copyright__`

**Описание**: Авторские права. Получен из настроек (`settings`). По умолчанию ''.

### `__cofee__`

**Описание**: Ссылка для поддержки разработчика. Получен из настроек (`settings`). По умолчанию ссылка на Boosty.to.
