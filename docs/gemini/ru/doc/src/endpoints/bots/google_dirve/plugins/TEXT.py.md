# Текстовые константы для Google Drive бота

## Обзор

Данный файл содержит текстовые константы, используемые в боте для Google Drive. Константы включают сообщения для пользователя, пути к файлам, настройки авторизации и другие текстовые данные.

## Константы

### Имя папки в Google Drive

#### `drive_folder_name`
**Описание**: Имя папки, используемой в Google Drive для хранения загруженных файлов.
    
**Значение**: "GDriveUploaderBot"

### Данные для Mega

#### `MEGA_EMAIL`
**Описание**: Электронная почта пользователя Mega.
  
**Значение**: "bearyan8@yandex.com"

#### `MEGA_PASSWORD`
**Описание**: Пароль пользователя Mega.
    
**Значение**: "bearyan8@yandex.com"

### Сообщения

#### `START`
**Описание**: Приветственное сообщение бота.
   
**Формат**: Приветственное сообщение с информацией о боте, ссылкой на канал обновлений и предложением сообщить об ошибках.

#### `HELP`
**Описание**: Сообщение с информацией о командах и возможностях бота.
    
**Формат**: Текст с инструкциями по авторизации, смене аккаунта, возможностям бота и поддерживаемым типам ссылок.

#### `DP_DOWNLOAD`
**Описание**: Сообщение о начале загрузки файла с Dropbox.
    
**Значение**: "Dropbox Link !! Downloading Started ..."

#### `OL_DOWNLOAD`
**Описание**: Сообщение о начале загрузки файла с Openload.
    
**Значение**: "Openload Link !! Downloading Started ... \n Openload Links Are Extremely Slow"

#### `PROCESSING`
**Описание**: Сообщение о начале обработки запроса.
    
**Значение**: "Processing Your Request ...!!"

#### `DOWNLOAD`
**Описание**: Сообщение о начале загрузки.
   
**Значение**: "Downloading Started ..."

#### `DOWN_MEGA`
**Описание**: Сообщение о начале загрузки файла с Mega.
   
**Значение**: "Downloading Started... \n  Mega Links are \n Extremely Slow :("

#### `DOWN_COMPLETE`
**Описание**: Сообщение об окончании загрузки.
  
**Значение**: "Downloading complete !!"

#### `NOT_AUTH`
**Описание**: Сообщение об отсутствии авторизации.
   
**Значение**: "You Are Not Authorised To Using this Bot \n\n Please Authorise Me Using /auth  \n\n @aryanvikash"

#### `REVOKE_FAIL`
**Описание**: Сообщение о неудачной попытке отзыва авторизации.
    
**Значение**: "You Are Already UnAuthorised \n. Please Use /auth To Authorise \n\n report At @aryanvikash "

#### `AUTH_SUCC`
**Описание**: Сообщение об успешной авторизации.
   
**Значение**: "Authorised Successfully  !! \n\n Now Send me A direct Link :)"

#### `ALREADY_AUTH`
**Описание**: Сообщение о повторной авторизации.
   
**Значение**: "You Are Already Authorised ! \n\n Wanna Change Drive Account? \n\n Use /revoke \n\n report At @aryanvikash "

#### `AUTH_URL`
**Описание**: Сообщение со ссылкой для авторизации.
    
**Формат**: Ссылка для перехода на страницу авторизации Google Drive.

#### `UPLOADING`
**Описание**: Сообщение о начале загрузки на Google Drive.
  
**Значение**: "Download Complete !! \n Uploading Your file"

#### `REVOKE_TOK`
**Описание**: Сообщение об успешном отзыве токена.
   
**Значение**: " Your Token is Revoked Successfully !! \n\n Use /auth To Re-Authorise Your Drive Acc. "

#### `DOWNLOAD_URL`
**Описание**: Сообщение об успешной загрузке файла на Google Drive.
    
**Формат**: Текст с информацией об имени файла, размере, и ссылкой для скачивания с Google Drive.

#### `AUTH_ERROR`
**Описание**: Сообщение об ошибке авторизации.
   
**Значение**: "AUTH Error !! Please  Send Me a  valid Token or Re - Authorise Me  \n\n report At @aryanvikash"

### Настройки

#### `DOWN_TWO`
**Описание**:  Константа для управление параллельными загрузками.
    
**Значение**: `True`

#### `DOWN_PATH`
**Описание**: Путь для сохранения загруженных файлов.
    
**Значение**: `"Downloads/"` (Linux path)

#### `OPENLOAD`
**Описание**: Флаг для включения/отключения поддержки Openload.
    
**Значение**: `True`

#### `DROPBOX`
**Описание**: Флаг для включения/отключения поддержки Dropbox.
    
**Значение**: `True`

#### `MEGA`
**Описание**: Флаг для включения/отключения поддержки Mega.
    
**Значение**: `True`

### Обновление

#### `UPDATE`
**Описание**: Сообщение об обновлении бота.
   
**Формат**: Текст с информацией об изменениях в боте и поддерживаемых типах ссылок.
