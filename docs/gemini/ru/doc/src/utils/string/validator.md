# Модуль hypotez/src/utils/string/validator.py

## Обзор

Модуль `validator.py` предоставляет набор статических методов для валидации строк, проверяя их на соответствие различным критериям, таким как формат цены, веса, артикула и URL.  Включает в себя функции для очистки строк от лишних символов, проверки на соответствие числовому формату и на валидность URL-адресов.


## Классы

### `ProductFieldsValidator`

**Описание**: Класс `ProductFieldsValidator` содержит статические методы для валидации различных полей продукта, таких как цена, вес и артикул.

**Статические методы**:

### `validate_price(price: str) -> bool`

**Описание**:  Проверяет, является ли переданная строка `price` корректным числовым значением, представляющим цену.

**Параметры**:

- `price` (str): Строка, представляющая цену.

**Возвращает**:

- `bool`: `True`, если строка `price` является валидной ценой, `None` в противном случае.


### `validate_weight(weight: str) -> bool`

**Описание**: Проверяет, является ли переданная строка `weight` корректным числовым значением, представляющим вес.

**Параметры**:

- `weight` (str): Строка, представляющая вес.

**Возвращает**:

- `bool`: `True`, если строка `weight` является валидным весом, `None` в противном случае.


### `validate_sku(sku: str) -> bool`

**Описание**: Проверяет, является ли переданная строка `sku` валидным артикулом (SKU). Проверяет минимальную длину и удаляет лишние символы.

**Параметры**:

- `sku` (str): Строка, представляющая артикул.

**Возвращает**:

- `bool`: `True`, если строка `sku` является валидным артикулом, `None` в противном случае.


### `validate_url(url: str) -> bool`

**Описание**: Проверяет валидность URL-адреса. Проверяет, что URL начинается с "http://" или "https://", и что в нём присутствует доменное имя.

**Параметры**:

- `url` (str): Строка, представляющая URL-адрес.

**Возвращает**:

- `bool`: `True`, если строка является валидным URL, `None` в противном случае.



### `isint(s: str) -> bool`

**Описание**: Проверяет, можно ли преобразовать строку `s` в целое число.

**Параметры**:

- `s` (str): Строка.

**Возвращает**:

- `bool`: `True`, если строка можно преобразовать в целое число, `None` в противном случае.  Возможны исключения при преобразовании.
