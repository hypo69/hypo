# Модуль миграции базы данных: initial_revision

## Обзор

Данный модуль содержит скрипт миграции базы данных, созданный с использованием Alembic. Он определяет изменения, необходимые для начальной структуры базы данных, используемой телеграм-ботом для цифрового рынка. В частности, он создает таблицы `categories`, `users`, `products` и `purchases`.

## Подробней

Этот файл является частью системы миграций Alembic и предназначен для автоматического изменения схемы базы данных. Он содержит функции `upgrade` и `downgrade`, которые определяют, как применить и отменить изменения, соответственно. Это позволяет управлять версиями базы данных и автоматически применять необходимые обновления.
