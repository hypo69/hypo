# Модуль `webdriver.executor`

## Обзор

Этот модуль предоставляет функции для взаимодействия с веб-элементами на странице.  Он позволяет выполнять различные действия, такие как поиск элементов, отправка сообщений, получение атрибутов и загрузка контента с URL.  Модуль ориентирован на гибкость и возможность настройки процесса взаимодействия.

## Функции

### `execute_locator`

**Описание**: Выполняет действия на веб-элементе, используя указанный локатор. Функция может выполнять различные задачи, включая нахождение элемента, отправку сообщений и получение атрибутов.

**Параметры**:
- `locator` (dict): Словарь или объект, содержащий информацию о локаторе (например, тип поиска, селектор). Обязательный параметр.
- `message` (str, optional): Сообщение для отправки элементу (например, текст для ввода). По умолчанию пустая строка.
- `typing_speed` (float, optional): Скорость набора текста, если отправляется сообщение (в секундах между нажатиями клавиш). По умолчанию 0.0.
- `continue_on_error` (bool, optional): Флаг, указывающий, продолжать ли выполнение, если произошла ошибка. По умолчанию `True`.

**Возвращает**:
- `any`: Результат выполнения локатора. Может быть веб-элементом, списком элементов, значением атрибута или результатом действия.

**Вызывает исключения**:
-  Возможны исключения, связанные с ошибками доступа к веб-элементу, ошибками парсинга или другими проблемами, возникающими при взаимодействии с веб-драйвером.


### `get_webelement_by_locator`

**Описание**: Находит и возвращает веб-элемент(ы) на странице, используя локатор.

**Параметры**:
- `locator` (dict): Словарь или объект, содержащий информацию о локаторе. Обязательный параметр.

**Возвращает**:
- `any`: Один или несколько веб-элементов, найденных по локатору. Может быть веб-элементом или списком веб-элементов, в зависимости от спецификации локатора.


**Вызывает исключения**:
-  Возможны исключения, связанные с отсутствием элемента по указанному локатору или ошибками поиска.


### `get_attribute_by_locator`

**Описание**: Получает значение атрибута веб-элемента, найденного по локатору.

**Параметры**:
- `locator` (dict): Словарь или объект, содержащий информацию о локаторе. Обязательный параметр.
- `message` (str, optional): Сообщение для отправки элементу перед получением атрибута. По умолчанию пустая строка.

**Возвращает**:
- `any`: Значение атрибута веб-элемента. Возвращает `None`, если произошла ошибка.

**Вызывает исключения**:
- Возможны исключения, связанные с ошибками доступа к атрибуту, отсутствием элемента или другими проблемами, связанными с получением атрибутов.


### `send_message`

**Описание**: Отправляет сообщение (например, текст) веб-элементу, используя локатор.

**Параметры**:
- `locator` (dict): Словарь или объект, содержащий информацию о локаторе. Обязательный параметр.
- `message` (str): Сообщение для отправки элементу. Обязательный параметр.
- `typing_speed` (float, optional): Скорость набора текста. По умолчанию 0.0.
- `continue_on_error` (bool, optional): Флаг для продолжения выполнения в случае ошибки. По умолчанию `True`.


**Возвращает**:
- `bool`: `True`, если сообщение отправлено успешно, `False` в противном случае.


**Вызывает исключения**:
-  Возможны исключения, связанные с ошибками взаимодействия с веб-элементом, например, ошибками отправки сообщения.


### `get_url`

**Описание**: Загружает HTML-контент с указанного URL-адреса или локального файла.

**Параметры**:
- `url` (str): URL-адрес или путь к файлу. Обязательный параметр.
- `protocol` (str, optional): Протокол для URL (по умолчанию `https://`).


**Возвращает**:
- `bool`: `True`, если контент успешно загружен, `False` в противном случае.


**Вызывает исключения**:
- Возможны исключения, связанные с ошибками сети, ошибками парсинга HTML или другими проблемами при доступе к ресурсу.
