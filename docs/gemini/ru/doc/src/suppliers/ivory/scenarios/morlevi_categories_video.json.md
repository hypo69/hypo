# morlevi_categories_video.json

## Обзор

Файл `morlevi_categories_video.json` содержит JSON-структуру, определяющую сценарии для сбора данных о видеокартах Gigabyte с сайта Morlevi. Каждый сценарий включает URL-адрес, параметры фильтрации, настройки активности и сопоставление с категориями PrestaShop.

## Сценарии

### `GIGABYTE RTX 4090`

**Описание**: Сценарий для сбора данных о видеокартах GIGABYTE RTX 4090.

**Параметры**:
- `brand` (str): Бренд "GIGABYTE".
- `url` (str): URL-адрес страницы с видеокартами RTX 4090 на сайте Morlevi.
- `checkbox` (bool): Флаг `false`, указывающий, что чекбокс не используется.
- `active` (bool): Флаг `true`, указывающий, что сценарий активен.
- `condition` (str): Условие "new", указывающее на новые товары.
- `presta_categories` (dict): Настройки категорий PrestaShop.
    - `template` (dict): Сопоставление шаблона для gigabyte с "RTX 4090".
