# Обзор модуля `revai`

## Обзор

Модуль `revai` предназначен для взаимодействия с API rev.com, сервисом, предоставляющим услуги транскрипции и анализа аудиофайлов. Этот модуль позволяет разработчикам интегрировать возможности работы со звуковыми файлами (переговоров, совещаний, звонков и т.п.) в свои приложения.
