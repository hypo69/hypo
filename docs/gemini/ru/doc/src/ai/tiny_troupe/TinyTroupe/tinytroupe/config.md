# Модуль config.ini

## Обзор

Этот файл содержит конфигурацию для взаимодействия с API OpenAI или Azure OpenAI. Он определяет типы API, параметры моделей, таймауты, количество попыток и другие важные настройки для работы приложения. Также он включает параметры для предотвращения вредоносного и нарушающего авторские права контента, а также для кеширования API-вызовов.

## Параметры

### `API_TYPE`

**Описание**: Тип используемого API (openai или azure).

**Значение по умолчанию**: `openai`

### `AZURE_API_VERSION`

**Описание**: Версия API Azure OpenAI.

**Значение по умолчанию**: `2023-05-15` (Требуется обновление документации Azure для актуальной версии)

### `MODEL`

**Описание**: Модель OpenAI для использования.

**Значение по умолчанию**: `gpt-4o`

### `MAX_TOKENS`

**Описание**: Максимальное количество токенов для модели.

**Значение по умолчанию**: `4000`

### `TEMPERATURE`

**Описание**: Температура модели (контроль случайности).

**Значение по умолчанию**: `0.3`

### `FREQ_PENALTY`

**Описание**: Наказание за повторяющийся текст.

**Значение по умолчанию**: `0.0`

### `PRESENCE_PENALTY`

**Описание**: Наказание за повторное появление слов в ответе.

**Значение по умолчанию**: `0.0`

### `TIMEOUT`

**Описание**: Таймаут для запроса API (в секундах).

**Значение по умолчанию**: `60`

### `MAX_ATTEMPTS`

**Описание**: Максимальное количество попыток запроса API.

**Значение по умолчанию**: `5`

### `WAITING_TIME`

**Описание**: Время ожидания между попытками запроса API (в секундах).

**Значение по умолчанию**: `1`

### `EXPONENTIAL_BACKOFF_FACTOR`

**Описание**: Фактор экспоненциального увеличения времени ожидания при повторных ошибках.

**Значение по умолчанию**: `5`

### `EMBEDDING_MODEL`

**Описание**: Модель для создания эмбеддингов.

**Значение по умолчанию**: `text-embedding-3-small`

### `CACHE_API_CALLS`

**Описание**: Флаг кеширования API вызовов.

**Значение по умолчанию**: `False`

### `CACHE_FILE_NAME`

**Описание**: Имя файла для кеширования.

**Значение по умолчанию**: `openai_api_cache.pickle`

### `MAX_CONTENT_DISPLAY_LENGTH`

**Описание**: Максимальная длина отображаемого контента.

**Значение по умолчанию**: `1024`

## Раздел "Simulation"

### `RAI_HARMFUL_CONTENT_PREVENTION`

**Описание**: Включение функции предотвращения вредоносного контента.

**Значение по умолчанию**: `True`

### `RAI_COPYRIGHT_INFRINGEMENT_PREVENTION`

**Описание**: Включение функции предотвращения нарушения авторских прав.

**Значение по умолчанию**: `True`

## Раздел "Logging"

### `LOGLEVEL`

**Описание**: Уровень ведения логов.

**Возможные значения**: `ERROR`, `WARNING`, `INFO`, `DEBUG`

**Значение по умолчанию**: `ERROR`
