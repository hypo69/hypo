# Модуль hypotez/src/utils/convertors/base64

## Обзор

Данный модуль предоставляет функцию для декодирования содержимого, закодированного в Base64, и записи его во временный файл с указанным расширением.

## Функции

### `base64_to_tmpfile`

**Описание**: Преобразует содержимое, закодированное в Base64, во временный файл.

**Параметры**:

- `content` (str): Содержимое, закодированное в Base64, которое необходимо декодировать и записать в файл.
- `file_name` (str): Имя файла, используемое для извлечения расширения файла для временного файла.


**Возвращает**:

- `str`: Путь к временному файлу.

**Пример использования**:

```python
>>> base64_content = "SGVsbG8gd29ybGQh"  # Base64 закодированное содержимое "Hello world!"
>>> file_name = "example.txt"
>>> tmp_file_path = base64_to_tmpfile(base64_content, file_name)
>>> print(f"Временный файл создан по пути: {tmp_file_path}")
```

**Описание поведения**: Функция декодирует Base64-закодированное содержимое и записывает его во временный файл с тем же расширением, что и переданное имя файла. Возвращает путь к созданному временному файлу. Использует `tempfile.NamedTemporaryFile` для создания временного файла и гарантирует, что файл будет удален после завершения работы функции.


### `base64encode`

**Описание**: Кодирует изображение в Base64.

**Параметры**:

- `image_path` (str): Путь к изображению, которое необходимо закодировать.

**Возвращает**:

- `str`: Строка, содержащая закодированное в Base64 изображение.


**Описание поведения**:  Функция открывает изображение по указанному пути в бинарном режиме, считывает его содержимое и кодирует в Base64. Возвращает закодированное в Base64 содержимое изображения в виде строки.  Обратите внимание, что эта функция предполагает работу с изображениями.


```python
# Пример использования для кодирования изображения
image_path = "путь/к/изображению.jpg"
encoded_image = base64encode(image_path)
print(encoded_image)
```


## Обработка исключений

Функции в модуле не содержат явной обработки исключений. Пользователь должен обработать потенциальные ошибки, такие как `FileNotFoundError` при работе с `image_path`.


```python

#Пример обработки FileNotFoundError
try:
    encoded_image = base64encode("путь/к/несуществующему/изображению.jpg")
    print(encoded_image)
except FileNotFoundError as ex:
    print(f"Ошибка: Файл не найден - {ex}")


```
