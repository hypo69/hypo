# Модуль banners_grabber

## Обзор

Модуль `banners_grabber.py` отвечает за получение баннеров с сайта KSP.

## Функции

### `get_banners`

**Описание**: Функция `get_banners` возвращает булево значение, указывающее на успешность получения баннеров.


**Возвращает**:
- `bool`: `True`, если баннеры получены успешно, `False` в противном случае.


```python
def get_banners() -> bool:
    """
    Возвращает True, если баннеры успешно получены.

    Returns:
        bool: True, если баннеры получены успешно.
    """
    return True
```
