## Функции

### `save_text_file`

**Описание**: Сохраняет данные в текстовый файл. Поддерживает различные типы данных (строка, список строк, словарь).

**Параметры**:
- `data` (str | list[str] | dict): Данные для записи.
- `file_path` (str | Path): Путь к файлу.
- `mode` (str, optional): Режим записи (`w` для записи, `a` для добавления). По умолчанию `'w'`.
- `exc_info` (bool, optional): Если True, регистрирует трассировку стека при ошибке. По умолчанию `True`.

**Возвращает**:
- bool: `True`, если файл был успешно сохранён, `False` в противном случае.

**Вызывает исключения**:
- Любые исключения, возникающие при работе с файлом.


### `read_text_file`

**Описание**: Читает содержимое файла. При чтении каталога возвращает список строк из всех файлов.

**Параметры**:
- `file_path` (str | Path): Путь к файлу или каталогу.
- `as_list` (bool, optional): Если True, возвращает содержимое файла в виде списка строк. По умолчанию `False`.
- `extensions` (list[str], optional): Список расширений файлов, которые нужно включить при чтении каталога. По умолчанию `None`.
- `exc_info` (bool, optional): Если True, регистрирует трассировку стека при ошибке. По умолчанию `True`.

**Возвращает**:
- str | list[str] | None: Содержимое файла как строка или список строк, или `None` при ошибке.

**Вызывает исключения**:
- Любые исключения, возникающие при работе с файлом.


### `get_filenames`

**Описание**: Возвращает список имён файлов в каталоге, опционально отфильтрованных по расширению.

**Параметры**:
- `directory` (str | Path): Каталог для поиска.
- `extensions` (str | list[str], optional): Расширения для фильтрации. По умолчанию `'*'`.
- `exc_info` (bool, optional): Если True, регистрирует трассировку стека при ошибке. По умолчанию `True`.

**Возвращает**:
- list[str]: Список имён файлов, найденных в каталоге.

**Вызывает исключения**:
- Любые исключения, возникающие при работе с файлом.


### `recursively_yield_file_path`

**Описание**: Рекурсивно генерирует пути к файлам, соответствующим заданным шаблонам.

**Параметры**:
- `root_dir` (str | Path): Корневой каталог для поиска.
- `patterns` (str | list[str], optional): Шаблоны для фильтрации файлов. По умолчанию `'*'`.
- `exc_info` (bool, optional): Если True, регистрирует трассировку стека при ошибке. По умолчанию `True`.

**Возвращает**:
- Generator[Path, None, None]: Итератор путей к файлам, соответствующим шаблонам.

**Вызывает исключения**:
- Любые исключения, возникающие при работе с файлом.
