# Модуль src.endpoints.hypo69.code_assistant.onela_bot

## Обзор

Модуль `onela_bot` предназначен для взаимодействия с моделью ассистента программиста через чат Telegram. Он обрабатывает текстовые сообщения и загруженные документы, отправляя запросы к модели и возвращая результаты пользователю.

## Классы

### `OnelaBot`

**Описание**: Класс `OnelaBot` представляет собой бота Telegram для взаимодействия с моделью ассистента программиста. Наследуется от класса `TelegramBot`.

**Атрибуты**:

- `model`: Объект класса `GoogleGenerativeAI`, используемый для взаимодействия с моделью.


**Методы**:

#### `__init__`

**Описание**: Инициализирует объект `OnelaBot`.

**Параметры**:

- `gs.credentials.telegram.onela_bot`:  Данные для авторизации Telegram бота.

**Возвращает**:
- `None`

#### `handle_message`

**Описание**: Обрабатывает текстовые сообщения, отправленные пользователю.

**Параметры**:

- `update` (Update): Данные обновления Telegram.
- `context` (CallbackContext): Контекст выполнения.


**Возвращает**:
- `None`

**Обрабатывает исключения**:

- `Exception`:  Возникает при ошибке обработки сообщения. Логгирует ошибку с помощью `logger`.


#### `handle_document`

**Описание**: Обрабатывает загруженные документы.

**Параметры**:

- `update` (Update): Данные обновления Telegram.
- `context` (CallbackContext): Контекст выполнения.

**Возвращает**:
- `None`

**Обрабатывает исключения**:

- `Exception`: Возникает при ошибке обработки документа. Логгирует ошибку с помощью `logger`.
