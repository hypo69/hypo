# HORSES

## סקירה כללית

משחק "סוסים" הוא משחק פשוט בו השחקן בוחר סוס, והמחשב קובע באופן אקראי איזה סוס ינצח. השחקן מנצח או מפסיד בהתאם לבחירתו.

## פונקציות

### `main`

**תיאור**:
פונקציית המשחק הראשית המדמה משחק סוסים. היא בוחרת מספר סוס מנצח באופן אקראי, מבקשת מהמשתמש לבחור סוס, ומודיעה אם המשתמש ניצח או הפסיד.

**פרמטרים**:
- אין פרמטרים.

**החזרות**:
- אין ערך מוחזר.

**הערות**:
- המשחק מקבל קלט מהמשתמש כדי לקבוע את בחירת הסוס.
- המשחק מטפל בשגיאות קלט של משתמש.

**הערות נוספות**:
- הקוד כולל טיפול בחריגות בעת קבלת קלט מהמשתמש.
- אם המשתמש מכניס קלט לא תקין, תודפס הודעת שגיאה, והמשחק יסתיים.
- מודול ה- `random` משמש ליצירת המספר המנצח.
