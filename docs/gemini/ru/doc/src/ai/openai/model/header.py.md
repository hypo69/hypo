# Модуль `src.logger.header`

## Обзор

Модуль `src.logger.header` предназначен для определения корневого пути проекта. Все импорты в проекте строятся относительно этого пути. Этот модуль также загружает основные настройки проекта из `settings.json` и документацию из `README.MD`.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла, поднимаясь вверх и останавливаясь на первой директории, содержащей любой из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple, optional): Кортеж имен файлов или директорий для идентификации корневой директории проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена, иначе - директория, где расположен скрипт.

## Переменные

### `MODE`
**Описание**: Режим работы приложения (по умолчанию 'dev').
- `str`: Значение режима.

### `__root__`
**Описание**: Путь к корневой директории проекта.
- `Path`: Путь к корневой директории проекта.

### `settings`
**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json`.
- `dict | None`: Словарь с настройками или `None`, если файл не найден или не является валидным JSON.

### `doc_str`
**Описание**: Строка с документацией проекта, загруженная из файла `README.MD`.
- `str | None`: Строка с документацией или `None`, если файл не найден.

### `__project_name__`
**Описание**: Название проекта, полученное из настроек или `hypotez` по умолчанию.
- `str`: Название проекта.

### `__version__`
**Описание**: Версия проекта, полученная из настроек или пустая строка по умолчанию.
- `str`: Версия проекта.

### `__doc__`
**Описание**: Строка с документацией проекта, загруженная из файла `README.MD`.
- `str`: Строка с документацией или пустая строка, если файл не найден.

### `__details__`
**Описание**: Детали проекта.
- `str`: Пустая строка.

### `__author__`
**Описание**: Автор проекта, полученный из настроек или пустая строка по умолчанию.
- `str`: Автор проекта.

### `__copyright__`
**Описание**: Информация об авторских правах проекта, полученная из настроек или пустая строка по умолчанию.
- `str`: Информация об авторских правах.

### `__cofee__`
**Описание**: Строка с приглашением угостить разработчика кофе, полученная из настроек или строка по умолчанию.
- `str`: Приглашение угостить разработчика кофе.
