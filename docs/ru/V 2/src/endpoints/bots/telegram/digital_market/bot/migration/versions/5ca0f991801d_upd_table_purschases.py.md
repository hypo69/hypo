# Обновление таблицы `purchases`

## Обзор

Этот файл содержит миграцию базы данных Alembic, которая добавляет столбец `payment_type` в таблицу `purchases` и создает уникальное ограничение для `payment_id`. Миграция также предоставляет возможность отката этих изменений.

## Оглавление

- [Функции](#Функции)
  - [`upgrade`](#upgrade)
  - [`downgrade`](#downgrade)

## Функции

### `upgrade`

**Описание**: Функция выполняет обновление базы данных, добавляя столбец `payment_type` в таблицу `purchases` и создавая уникальное ограничение для столбца `payment_id`.

**Параметры**:
- Нет параметров.

**Возвращает**:
- `None`

**Вызывает исключения**:
- Нет явных исключений

### `downgrade`

**Описание**: Функция выполняет откат изменений, внесенных функцией `upgrade`. Удаляет уникальное ограничение для `payment_id` и удаляет столбец `payment_type` из таблицы `purchases`.

**Параметры**:
- Нет параметров.

**Возвращает**:
- `None`

**Вызывает исключения**:
- Нет явных исключений
