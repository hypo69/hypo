# Конфигурационный файл `config.ini`

## Обзор

Данный файл содержит конфигурационные параметры для работы с OpenAI API и Azure OpenAI Service, а также параметры симуляции и логирования.

## OpenAI

### Общие параметры

- `API_TYPE` (str): Тип API. По умолчанию: `openai`. Возможные значения: `openai`, `azure`.
- `AZURE_API_VERSION` (str): Версия API для Azure. По умолчанию: `2023-05-15`. См. документацию Azure.

### Параметры модели

- `MODEL` (str): Используемая модель. По умолчанию: `gpt-4o`.
- `MAX_TOKENS` (int): Максимальное количество токенов в ответе. По умолчанию: `4000`.
- `TEMPERATURE` (float): Температура для генерации текста. По умолчанию: `0.3`.
- `FREQ_PENALTY` (float): Штраф за частоту токенов. По умолчанию: `0.0`.
- `PRESENCE_PENALTY` (float): Штраф за наличие токенов. По умолчанию: `0.0`.
- `TIMEOUT` (int): Максимальное время ожидания ответа от API в секундах. По умолчанию: `60`.
- `MAX_ATTEMPTS` (int): Максимальное количество попыток вызова API в случае неудачи. По умолчанию: `5`.
- `WAITING_TIME` (int): Время ожидания между попытками вызова API в секундах. По умолчанию: `1`.
- `EXPONENTIAL_BACKOFF_FACTOR` (int): Фактор экспоненциальной задержки для повторных попыток. По умолчанию: `5`.
- `EMBEDDING_MODEL` (str): Модель для встраивания текста. По умолчанию: `text-embedding-3-small`.

### Параметры кэширования

- `CACHE_API_CALLS` (bool): Флаг для включения/выключения кэширования вызовов API. По умолчанию: `False`.
- `CACHE_FILE_NAME` (str): Имя файла для хранения кэша. По умолчанию: `openai_api_cache.pickle`.
- `MAX_CONTENT_DISPLAY_LENGTH` (int): Максимальная длина отображаемого контента в логах. По умолчанию: `1024`.

## Simulation

- `RAI_HARMFUL_CONTENT_PREVENTION` (bool): Флаг для включения/выключения предотвращения вредоносного контента. По умолчанию: `True`.
- `RAI_COPYRIGHT_INFRINGEMENT_PREVENTION` (bool): Флаг для включения/выключения предотвращения нарушений авторских прав. По умолчанию: `True`.

## Logging

- `LOGLEVEL` (str): Уровень логирования. Возможные значения: `ERROR`, `WARNING`, `INFO`, `DEBUG`. По умолчанию: `ERROR`.
