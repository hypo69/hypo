# morlevi_categories_psu_antec.json

## Обзор

Данный файл содержит JSON-конфигурацию для сбора данных о блоках питания (PSU) марки ANTEC с сайта Morlevi. Он определяет сценарии для каждого блока питания, включая его мощность, URL страницы, активность, состояние и категории PrestaShop.

## Структура JSON

### `scenarios`

#### Описание

Словарь, содержащий сценарии для каждого блока питания. Каждый ключ представляет собой название модели блока питания, а значение - объект с деталями о данном блоке питания.

#### Пример структуры
```json
{
  "scenarios": {
    "ANTEC 450W": {
      "brand": "ANTEC",
      "name": "450W",
      "url": "https://www.morlevi.co.il/Cat/66?p_145=634&sort=datafloat2%2Cprice&keyword=",
      "checkbox": false,
      "active": true,
      "condition":"new",
      "presta_categories": "158,511,188,577"
    },
    ...
  }
}
```
#### Ключи

Каждый сценарий имеет следующие ключи:

-   `brand` (str): Бренд блока питания (например, "ANTEC").
-   `name` (str): Название модели блока питания (например, "450W").
-   `url` (str): URL страницы с блоком питания на сайте Morlevi.
-   `checkbox` (bool): Флаг, указывающий, что чекбокс не выбран (всегда `false`).
-   `active` (bool): Флаг, указывающий, что сценарий активен (всегда `true`).
-   `condition` (str): Состояние товара, в данном случае всегда "new".
-   `presta_categories` (str): Строка, содержащая ID категорий PrestaShop, к которым относится товар (например, "158,511,188,577").

## Использование

Этот JSON файл используется для настройки процесса парсинга веб-страниц сайта Morlevi с целью сбора данных о блоках питания ANTEC. Каждый сценарий определяет конкретную модель блока питания и параметры ее сбора.
