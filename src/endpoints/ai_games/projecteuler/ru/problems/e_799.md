# Задача No 799
## Уникальные последовательности сумм частичных сумм

## Условия

Рассмотрим последовательность целых чисел $s_1, s_2, s_3, ...$. 
Определим последовательность частичных сумм $a_n$ как:

$a_1 = s_1$
$a_2 = s_1 + s_2$
$a_3 = s_1 + s_2 + s_3$
...
$a_n = \sum_{k=1}^n s_k$

И, наконец, определим последовательность сумм частичных сумм $b_n$ как:

$b_1 = a_1$
$b_2 = a_1 + a_2$
$b_3 = a_1 + a_2 + a_3$
...
$b_n = \sum_{k=1}^n a_k$

Рассмотрим последовательность $s = \{1, 1, 1, 1, 1, ...\}$. Тогда:
$a = \{1, 2, 3, 4, 5, ...\}$
$b = \{1, 3, 6, 10, 15, ...\}$

Теперь рассмотрим последовательность $s = \{1, -1, 1, -1, 1, ...\}$. Тогда:
$a = \{1, 0, 1, 0, 1, ...\}$
$b = \{1, 1, 2, 2, 3, ...\}$

Теперь рассмотрим последовательность $s = \{1, 2, 3, 4, 5, ...\}$. Тогда:
$a = \{1, 3, 6, 10, 15, ...\}$
$b = \{1, 4, 10, 20, 35, ...\}$

Определим $U(n)$ как количество последовательностей $s$, состоящих из первых $n$ членов, где $s_k \in \{-1, 1\}$, таких, что все элементы $b_1, b_2, ..., b_n$ являются уникальными.
Например, $U(1) = 2$, поскольку есть две возможные последовательности $s$, $\{1\}$ и $\{-1\}$, и в обоих случаях $b_1$ будет уникальным.
Также $U(2) = 2$, поскольку есть четыре возможных последовательности $s$: $\{1, 1\}, \{1, -1\}, \{-1, 1\}$ и $\{-1, -1\}$. Для $\{1, 1\}$ $b = \{1, 3\}$, для $\{1, -1\}$ $b = \{1, 1\}$ (не уникальные), для $\{-1, 1\}$ $b = \{-1, -2\}$ и для $\{-1, -1\}$ $b = \{-1, -3\}$. Следовательно, только последовательности $\{1, 1\}$ и $\{-1, -1\}$ дают уникальные элементы $b$ вплоть до $b_2$.

Известно, что $U(4) = 12$, $U(8) = 304$ и $U(20) = 444896$.

Найдите $U(100) \pmod{1\,000\,000\,007}$.
