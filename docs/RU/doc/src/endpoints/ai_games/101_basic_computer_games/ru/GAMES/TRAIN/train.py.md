# Игра "Поезд"

## Обзор

Игра "Поезд" - это игра-симулятор, где игрок управляет поездом, разгоняясь и замедляясь, стараясь достичь определенной скорости на определенном расстоянии. Игрок вводит значения для ускорения и замедления, и игра выводит текущую скорость и расстояние, пройденное поездом. Цель - достичь целевой скорости на целевом расстоянии.

## Описание

Игра начинается с нулевой скорости и пройденного расстояния. Игрок вводит величину ускорения и замедления. Игра вычисляет новую скорость и пройденное расстояние, используя введенные значения. Игра показывает текущую скорость и пройденное расстояние. Игра продолжается до тех пор, пока пройденное расстояние не достигнет целевого расстояния.

## Функции

### Основной игровой цикл

**Описание**: Основной игровой цикл, в котором происходит взаимодействие с пользователем и обновление состояния игры.

**Параметры**:
- Нет параметров.

**Возвращает**:
- Нет возвращаемого значения.

**Вызывает исключения**:
- `ValueError`: Если пользователь введет не числовое значение.

## Переменные

### `speed`
- **Описание**: Текущая скорость поезда. Инициализируется значением `0`.

### `distance`
- **Описание**: Текущее пройденное расстояние. Инициализируется значением `0`.

### `targetDistance`
- **Описание**: Целевое расстояние, которое необходимо достичь. Установлено в `100`.
