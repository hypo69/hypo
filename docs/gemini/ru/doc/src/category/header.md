# Модуль hypotez/src/category/header.py

## Обзор

Этот модуль определяет корневой путь к проекту Hypotez. Все импорты строятся относительно этого пути. Он находит директорию проекта, начиная с текущего файла, ищет вверх по дереву директорий до тех пор, пока не найдет директорию, содержащую файлы-маркеры (`pyproject.toml`, `requirements.txt`, `.git`). Если корневая директория не найдена, или не найдена ни один из файлов-маркеров, возвращает директорию, где находится текущий файл.  В дальнейшем предполагается перенести логику определения корневого пути в системные переменные.

## Функции

### `set_project_root`

**Описание**: Функция находит корневую директорию проекта, начиная с текущего файла.

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, по которым происходит поиск корневой директории проекта. По умолчанию содержит ('pyproject.toml', 'requirements.txt', '.git').

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена, иначе возвращает директорию, где расположен текущий скрипт.


**Вызывает исключения**:

- Нет.


## Переменные

### `__root__`

**Описание**: Переменная, хранящая путь к корневой директории проекта.

**Тип**: `Path`.


## Дополнительные сведения

### Чтение настроек из settings.json

**Описание**: Модуль пытается загрузить настройки проекта из файла `settings.json` в корневой директории проекта.


**Обработка ошибок**:

- Возможные исключения `FileNotFoundError` и `json.JSONDecodeError` при чтении и декодировании файла обрабатываются, и переменная `settings` устанавливается в `None`, если файл не найден или имеет неверный формат.


### Чтение README.MD

**Описание**: Модуль пытается прочитать файл `README.MD` в корневой директории проекта для получения информации о проекте.

**Обработка ошибок**:

- Возможные исключения `FileNotFoundError` и `json.JSONDecodeError` при чтении файла обрабатываются, и переменная `doc_str` устанавливается в `None`, если файл не найден или имеет неверный формат.


### Переменные проекта

**Описание**:  Этот раздел содержит инициализацию переменных, связанных с проектом, на основе данных из `settings.json`.

**Переменные**:

- `__project_name__` (str): Название проекта (получено из `settings.json` или установлено по умолчанию).
- `__version__` (str): Версия проекта (получено из `settings.json` или установлено по умолчанию).
- `__doc__` (str): Текст из файла README.MD или пустая строка.
- `__details__` (str): Пустая строка.
- `__author__` (str): Автор проекта (получено из `settings.json` или установлено по умолчанию).
- `__copyright__` (str): Авторские права (получено из `settings.json` или установлено по умолчанию).
- `__cofee__` (str): Ссылка для поддержки разработчика (получено из `settings.json` или установлено по умолчанию).
