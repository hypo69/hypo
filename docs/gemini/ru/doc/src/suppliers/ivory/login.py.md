# Модуль `login.py`

## Обзор

Модуль `login.py` предназначен для выполнения операций, связанных с аутентификацией и входом в систему поставщика Ivory.  Включает в себя определение режима работы (разработка или продакшн) и импорты необходимых модулей.

## Константы

### `MODE`

**Описание**: Режим работы приложения. По умолчанию установлен в 'dev' (режим разработки).

**Значение**: `'dev'`
