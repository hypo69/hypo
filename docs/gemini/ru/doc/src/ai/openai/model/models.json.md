# Документация для файла `models.json`

## Обзор

Файл `models.json` содержит словарь, где ключами являются названия моделей OpenAI, а значениями - пустые словари.  Предназначен для хранения метаданных, связанных с моделями OpenAI, доступных в приложении.

## Структура файла

Файл представляет собой JSON-объект (словарь), где:
- **Ключи**: Названия моделей OpenAI (например, `"gpt-3.5-turbo-instruct"`, `"gpt-4o"`).
- **Значения**: Пустые словари (`{}`).
