## Обзор

Этот файл содержит пример JSON-данных, используемых для тестирования функциональности, связанной с артефактами. Он включает информацию о человеке, его возрасте, профессии и произвольное текстовое содержимое.

## Структура JSON

Файл `test_artifact.json` представляет собой JSON-объект, включающий следующие поля:

```json
{
    "name": "John Doe",
    "age": 30,
    "occupation": "Engineer",
    "content": "This is a sample JSON data."
}
```

## Поля JSON

- `name` (строка): Имя. В данном примере - "John Doe".
- `age` (число): Возраст. В данном примере - 30.
- `occupation` (строка): Профессия. В данном примере - "Engineer".
- `content` (строка): Произвольное текстовое содержимое. В данном примере - "This is a sample JSON data."
