# Модуль `header.py`

## Обзор

Модуль `header.py` содержит общие настройки и импорты для работы с контекстным меню в Tkinter. Он также определяет переменную `MODE`, которая указывает на режим работы приложения (разработка или продакшн).

## Содержание

1. [Обзор](#обзор)
2. [Переменные](#переменные)
3. [Импорты](#импорты)

## Переменные

### `MODE`
**Описание**: Определяет режим работы приложения. По умолчанию установлен в `dev` (режим разработки).
- Тип: `str`
- Значение по умолчанию: `'dev'`

## Импорты

### `sys`
**Описание**: Модуль `sys` предоставляет доступ к некоторым переменным и функциям, взаимодействующим с интерпретатором Python.

### `os`
**Описание**: Модуль `os` предоставляет доступ к операционной системе.

### `Path` из `pathlib`
**Описание**: Класс `Path` из модуля `pathlib` используется для представления путей к файлам и каталогам.

### `__root__`
**Описание**: Переменная `__root__` определяет корневую директорию проекта `hypotez`.

**Тип**: `Path`

**Значение**: Путь к корневой директории проекта, полученный из текущей рабочей директории.

**Пример**: Если текущая рабочая директория `/home/user/projects/hypotez/src/gui/context_menu/tkinter`, то `__root__` будет `/home/user/projects/hypotez`.

from pathlib import Path
