# Модуль `hypotez/src/suppliers/aliexpress/campaign/html_generators.py`

## Обзор

Этот модуль содержит классы для генерации HTML-контента для рекламной кампании AliExpress. Он предоставляет функции для создания страниц отдельных товаров, категорий и обзора всей кампании.  Модуль использует библиотеку `html` для экранирования HTML-данных и `pathlib` для работы с путями к файлам.

## Классы

### `ProductHTMLGenerator`

**Описание**: Класс для генерации HTML-страниц отдельных товаров.

**Статические методы**:

#### `set_product_html`

**Описание**: Создает HTML-файл для отдельного товара.

**Параметры**:
- `product` (SimpleNamespace): Объект, содержащий данные о товаре.
- `category_path` (str | Path): Путь к каталогу, в котором будет сохранен HTML-файл.

**Возвращает**:
-  Не имеет возвращаемого значения.

**Вызывает исключения**:
- Не указывает какие-либо исключения.


### `CategoryHTMLGenerator`

**Описание**: Класс для генерации HTML-страниц категорий.

**Статические методы**:

#### `set_category_html`

**Описание**: Создает HTML-файл для категории, содержащий список товаров.

**Параметры**:
- `products_list` (list[SimpleNamespace] | SimpleNamespace): Список товаров для включения в HTML. Может быть списком или одним объектом SimpleNamespace, если нужно сгенерировать HTML для одной категории.
- `category_path` (str | Path): Путь к каталогу, в котором будет сохранен HTML-файл.


**Возвращает**:
-  Не имеет возвращаемого значения.


**Вызывает исключения**:
- Не указывает какие-либо исключения.


### `CampaignHTMLGenerator`

**Описание**: Класс для генерации HTML-страницы обзора всей рекламной кампании.

**Статические методы**:

#### `set_campaign_html`

**Описание**: Создает HTML-файл для обзора всей кампании, перечисляя все категории.

**Параметры**:
- `categories` (list[str]): Список имен категорий.
- `campaign_path` (str | Path): Путь к каталогу, в котором будет сохранен HTML-файл.

**Возвращает**:
-  Не имеет возвращаемого значения.

**Вызывает исключения**:
- Не указывает какие-либо исключения.
