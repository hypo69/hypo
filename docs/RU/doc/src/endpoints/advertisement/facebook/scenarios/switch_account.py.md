# Модуль `switch_account.py`

## Обзор

Модуль `switch_account.py` содержит функции для переключения между аккаунтами в Facebook. Основная функция `switch_account` использует веб-драйвер для поиска и нажатия кнопки переключения аккаунта, если она присутствует на странице.

   - [`switch_account`](#switch_account)

## Функции

### `switch_account`

**Описание**: Выполняет переключение между аккаунтами, если кнопка "Переключить" доступна.

**Параметры**:
- `driver` (Driver): Экземпляр веб-драйвера для управления браузером.

**Возвращает**:
- `None`: Функция ничего не возвращает явно.

**Вызывает исключения**:
- Исключения, которые могут возникнуть при взаимодействии с веб-драйвером (например, если элемент не найден).
```python
def switch_account(driver: Driver):
    """ Если есть кнопка `Переключить` - нажимаю её  """
    driver.execute_locator(locator.switch_to_account_button)
```
