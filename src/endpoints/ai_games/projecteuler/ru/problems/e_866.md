# Задача No 866
## Игра Нима на графе
## Условия

Двое игроков играют в игру Нима на ациклическом ориентированном графе. Граф имеет *n* вершин, пронумерованных от 0 до *n-1*. Каждая дуга в графе идет от вершины с меньшим индексом к вершине с большим индексом. Иными словами, если существует дуга из вершины *u* в вершину *v*, то *u* < *v*.

Первоначально в каждой вершине графа лежит некоторое количество камней. На каждом ходу игрок выбирает любую вершину, содержащую хотя бы один камень, и перемещает ровно один камень вдоль любой исходящей из этой вершины дуги. Игроки делают ходы по очереди. Игрок, который не может сделать ход, проигрывает.

Пусть *W*(*n*, *k*) — количество графов с *n* вершинами, для которых существует выигрышная стратегия для второго игрока, если в каждой вершине изначально лежит *k* камней.

Например, *W*(2, 1) = 1, поскольку единственным графом с двумя вершинами, в котором существует дуга 0 → 1, является такой граф. В этом графе для второго игрока всегда существует выигрышная стратегия, так как первый игрок всегда переместит камень в вершину 1, и второй игрок не сможет сделать ход. Также *W*(3, 1) = 2.

Известно, что *W*(5, 1) = 12 и *W*(10, 1) = 7968.

Найдите *W*(100, 1) mod 1 000 000 007.
