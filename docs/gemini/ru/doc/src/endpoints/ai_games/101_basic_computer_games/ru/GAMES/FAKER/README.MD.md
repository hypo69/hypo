# Модуль FakeDataGenerator

## Обзор

Модуль `FakeDataGenerator` предоставляет класс для генерации случайных данных, таких как имена, адреса, номера телефонов, email-адреса и другие. Этот класс может быть полезен для тестирования, заполнения баз данных, создания демонстрационных данных и других задач, где требуется генерация случайных значений.

## Классы

### `FakeDataGenerator`

**Описание**: Класс для генерации фейковых (случайных) данных.

**Атрибуты:**
- `first_names`: Список имен.
- `last_names`: Список фамилий.
- `cities`: Список городов.
- `streets`: Список улиц.
- `domains`: Список доменов.

**Методы**:
- [`random_name`](#random_name): Генерирует случайное полное имя.
- [`random_email`](#random_email): Генерирует случайный email-адрес.
- [`random_phone`](#random_phone): Генерирует случайный номер телефона.
- [`random_address`](#random_address): Генерирует случайный адрес.
- [`random_string`](#random_string): Генерирует случайную строку заданной длины.
- [`random_int`](#random_int): Генерирует случайное целое число в заданном диапазоне.
- [`random_choice`](#random_choice): Выбирает случайный элемент из списка.

## Функции

### `random_name`

**Описание**: Генерация случайного полного имени.

**Параметры**:
- Нет

**Возвращает**:
- `str`: Полное имя, состоящее из случайного имени и фамилии.

### `random_email`

**Описание**: Генерация случайного email-адреса.

**Параметры**:
- Нет

**Возвращает**:
- `str`: Email-адрес в формате `имя.фамилия@домен`.

### `random_phone`

**Описание**: Генерация случайного номера телефона в формате `+1-XXX-XXX-XXXX`.

**Параметры**:
- Нет

**Возвращает**:
- `str`: Номер телефона.

### `random_address`

**Описание**: Генерация случайного адреса.

**Параметры**:
- Нет

**Возвращает**:
- `str`: Адрес в формате `номер_дома улица, город`.

### `random_string`

**Описание**: Генерация случайной строки заданной длины.

**Параметры**:
- `length` (int, optional): Длина строки. По умолчанию 10.

**Возвращает**:
- `str`: Случайная строка, содержащая буквы и цифры.

### `random_int`

**Описание**: Генерация случайного целого числа в заданном диапазоне.

**Параметры**:
- `min_value` (int, optional): Минимальное значение. По умолчанию 0.
- `max_value` (int, optional): Максимальное значение. По умолчанию 100.

**Возвращает**:
- `int`: Случайное целое число.

### `random_choice`

**Описание**: Выбор случайного элемента из списка.

**Параметры**:
- `options` (List[str]): Список значений для выбора.

**Возвращает**:
- `str`: Случайный элемент из списка.
