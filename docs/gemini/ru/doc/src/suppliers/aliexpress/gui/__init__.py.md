# Модуль `src.suppliers.aliexpress.gui`

## Обзор

Данный модуль предназначен для обеспечения графического интерфейса (GUI) для работы с поставщиком AliExpress. Он является частью более крупной системы `hypotez`. Модуль предоставляет функционал для разработчиков и может быть расширен в будущем.

## Переменные

### `MODE`

**Описание**:
Режим работы приложения. В данном случае установлен режим разработки (`dev`).

**Тип**: `str`
