# Модуль `hypotez/src/endpoints/prestashop/product.py`

## Обзор

Модуль `product.py` предоставляет класс `PrestaProduct` для работы с API PrestaShop, позволяя выполнять операции по проверке наличия товара, расширенному поиску и получению информации о товаре по ID.

## Классы

### `PrestaProduct`

**Описание**: Класс `PrestaProduct` наследуется от класса `PrestaShop` и предоставляет методы для работы с товарами в базе данных PrestaShop через API.

**Методы**:

- `check(product_reference: str)`:
    **Описание**: Проверка наличия товара в базе данных по `product_reference` (SKU, MKT).
    **Параметры**:
        - `product_reference` (str):  Строка, представляющая SKU или MKT товара.
    **Возвращает**:
        - dict: Словарь с информацией о товаре, если товар найден.
        - False: Если товар не найден.
    **Вызывает исключения**:
        - Возможны исключения, генерируемые базовым классом `PrestaShop`.

- `search(filter: str, value: str)`:
    **Описание**: Расширенный поиск в базе данных по фильтрам.
    **Параметры**:
        - `filter` (str):  Наименование фильтра для поиска.
        - `value` (str): Значение фильтра.
    **Возвращает**:
        - Результат поиска, возможно None.
    **Вызывает исключения**:
        - Возможны исключения, генерируемые базовым классом `PrestaShop`.


- `get(id_product)`:
    **Описание**: Возвращает информацию о товаре по ID.
    **Параметры**:
        - `id_product` (int): Идентификатор товара.
    **Возвращает**:
        - dict: Словарь с информацией о товаре.
        - Возможен возврат `None` при отсутствии товара с данным ID.
    **Вызывает исключения**:
        - Возможны исключения, генерируемые базовым классом `PrestaShop`.


### `__init__`

**Описание**: Конструктор класса `PrestaProduct`.

**Параметры**:
    - `credentials` (Optional[dict | SimpleNamespace], optional): Словарь или объект SimpleNamespace с параметрами `api_domain` и `api_key`. По умолчанию `None`.
    - `api_domain` (Optional[str], optional): Домен API. По умолчанию `None`.
    - `api_key` (Optional[str], optional): Ключ API. По умолчанию `None`.
    - `*args`, `**kwards`: Дополнительные аргументы и ключевые аргументы.

**Возвращает**:
    - Не имеет возвращаемого значения.

**Вызывает исключения**:
    - `ValueError`: Если оба параметра `api_domain` и `api_key` не заданы.
