# Модуль `wdl.py`

## Обзор

Модуль `wdl.py` предоставляет функцию `wget_dl` для скачивания файлов по URL-адресу с использованием утилиты `wget`. Функция обрабатывает процесс скачивания, перехватывает ошибки и возвращает имя скачанного файла или сообщение об ошибке.

## Подробней

Этот модуль предназначен для автоматизации скачивания файлов из интернета в рамках проекта `hypotez`. Он использует утилиту `wget`, которая должна быть установлена в системе, для выполнения загрузки. Функция `wget_dl` принимает URL файла в качестве аргумента, запускает процесс `wget` и возвращает имя файла после успешной загрузки или сообщение об ошибке в случае неудачи. Модуль находится в каталоге `hypotez/src/endpoints/bots/google_dirve/plugins/`, что указывает на его использование в контексте работы с Google Drive и, возможно, ботами.

## Функции

### `wget_dl`

```python
def wget_dl(url: str) -> str | tuple[str, str]:
    """
    Args:
        url (str): URL-адрес файла для скачивания.

    Returns:
        str | tuple[str, str]: Имя скачанного файла в случае успеха или кортеж ("error", filename) в случае ошибки.

    Raises:
        Exception: Возникает при ошибке во время выполнения команды wget.

    Example:
        >>> wget_dl('https://example.com/file.txt')
        Downloading Started
        Downloading Complete file.txt
        'file.txt'

        >>> wget_dl('https://example.com/nonexistent_file.txt')
        Downloading Started
        DOWNLAOD ERROR : ...
        ('error', 'nonexistent_file.txt')
    """
```

**Описание**: Функция скачивает файл по указанному URL-адресу с использованием утилиты `wget`.

**Параметры**:
- `url` (str): URL-адрес файла для скачивания.

**Возвращает**:
- `str`: Имя скачанного файла в случае успеха.
- `tuple[str, str]`: Кортеж `("error", filename)`, где `filename` - имя файла, который пытались скачать, в случае ошибки.

**Вызывает исключения**:
- `Exception`: Возникает при ошибке во время выполнения команды `wget`.
