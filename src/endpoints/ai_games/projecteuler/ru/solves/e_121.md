## Ответ на Задачу No 121: Призовая игра

### 1. Анализ задачи и решение
**Понимание задачи:**
* Имеется игра с `n` раундами, где `n` = 15.
* В каждом раунде игрок тянет жетон из сумки с `n` жетонами, пронумерованными от 1 до `n`.
* Выигрыш происходит при вытягивании жетона "1".
* После каждого раунда жетон возвращается в сумку.
* Нужно найти количество выигрышных исходов, когда игра идет до 15 раундов.

**Решение:**
1. **Вероятность выигрыша в раунде:** Вероятность вытянуть жетон "1" в каждом раунде равна 1/n. В нашем случае 1/15.
2. **Вероятность проигрыша в раунде:** Вероятность не вытянуть жетон "1" в каждом раунде равна (n-1)/n. В нашем случае 14/15.
3. **Расчет выигрышных исходов:** Игрок может выиграть в любом из 15 раундов.
   * 1-й раунд: Вероятность выигрыша 1/15.
   * 2-й раунд: Вероятность проигрыша в 1-м раунде * вероятность выигрыша во 2-м раунде = (14/15)*(1/15).
   * k-й раунд: Вероятность проигрыша в (k-1) раундах * вероятность выигрыша в k-м раунде = (14/15)^(k-1) * (1/15).
4. **Количество благоприятных исходов:** Для каждого раунда, количество исходов это произведение количества возможностей (вариантов) до этого раунда с учетом проигрыша на количество выигрышных исходов. Т.е., если до n-го раунда мы проигрывали, это означает, что в каждом предыдущем раунде мы вытянули один из 14 жетонов (не "1"), и в n-ом раунде вытянули "1" жетон.
5. **Сумма вариантов выигрыша:** Суммируем все варианты выигрыша в каждом раунде.
6. **Вычисление количества вариантов:**  
   - В первом раунде выигрышных исходов: 1.
   - Во втором раунде: 14 * 1 = 14.
   - В третьем раунде: 14 * 14 * 1 = 196
   - В n-ом раунде: 14^(n-1) * 1
   Таким образом общее кол-во вариантов выигрыша:
   sum(14^(k-1)) от k=1 до n, что равняется 1+ 14 + 14^2 + ... + 14^14
   Тогда общее кол-во выигрышных исходов: 1 + 14 + 14^2 + .... + 14^14. Это геометрическая прогрессия. Используем формулу суммы геометрической прогрессии: S = a * (q^n - 1) / (q - 1)
   где, а = 1 (первый член), q = 14 (знаменатель), n = 15 (количество членов)

### 2. Алгоритм решения
1. Начать
2. Инициализировать переменную `total_wins` = 0.
3. Инициализировать переменную `rounds` = 15.
4. Для каждого раунда `i` от 1 до `rounds`:
  - вычислить `wins_in_current_round` = 14^(i - 1)
  - прибавить `wins_in_current_round` к `total_wins`
5. Вернуть `total_wins`
6. Конец

### 3. Реализация на Python 3.12
```python
def calculate_total_wins(rounds):
  """
  Calculates the total number of ways a player can win the prize game.

  Args:
    rounds: The number of rounds played.

  Returns:
    The total number of winning outcomes.
  """
  total_wins = 0
  for i in range(1, rounds + 1):
      wins_in_current_round = 14**(i - 1)
      total_wins += wins_in_current_round
  return total_wins

# Пример использования:
rounds = 15
result = calculate_total_wins(rounds)
print(result)
```

### 4. Блок-схема в формате mermaid
```mermaid
flowchart TD
    Start((Начало)) --> InitializeWins((Инициализировать total_wins = 0))
    InitializeWins --> InitializeRounds((Инициализировать rounds = 15))
    InitializeRounds --> LoopStart((Начать цикл: i от 1 до rounds))
    LoopStart --> CalculateRoundWins((Вычислить wins_in_current_round = 14^(i-1)))
    CalculateRoundWins --> AddToTotalWins((Добавить wins_in_current_round к total_wins))
    AddToTotalWins --> LoopStart
    LoopStart --> ReturnTotalWins((Вернуть total_wins))
    ReturnTotalWins --> End((Конец))
```
**Легенда:**
* **Start, End:** Начало и конец алгоритма.
* **InitializeWins:** Инициализирует переменную `total_wins` для хранения общего количества выигрышных исходов, присваивая ей значение 0.
* **InitializeRounds:** Инициализирует переменную `rounds`, определяющую количество раундов игры, значением 15.
* **LoopStart, LoopEnd:** Определяют начало и конец цикла перебора раундов.
* **CalculateRoundWins:** Вычисляет количество выигрышных исходов в текущем раунде, как 14^(i-1), где i — номер текущего раунда.
* **AddToTotalWins:** Добавляет количество выигрышных исходов текущего раунда к общему количеству выигрышных исходов.
* **ReturnTotalWins:** Завершает выполнение и возвращает общее количество выигрышных исходов.
