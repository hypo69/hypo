# Модуль hypotez/src/scenario/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json` и документации из `README.MD`. Он также инициализирует переменные, хранящие информацию о проекте, такую как имя, версия, описание, автор и авторские права.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с текущей директории и поднимаясь вверх по дереву директорий. Останавливается на первой директории, содержащей указанные маркерные файлы.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для определения корневой директории проекта. По умолчанию (`('pyproject.toml', 'requirements.txt', '.git')`).


**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена, иначе директория, в которой находится скрипт.

**Вызывает исключения**:

- Нет.


### Загрузка настроек и документации

**Описание**: Модуль пытается загрузить данные из файлов `settings.json` и `README.MD` в соответствующие переменные.

**Возможные ошибки**:

- `FileNotFoundError`: Возникает, если файлы `settings.json` или `README.MD` не найдены.
- `json.JSONDecodeError`: Возникает, если файл `settings.json` имеет некорректный формат JSON.


## Переменные

**Описание**: Переменные, содержащие информацию о проекте, инициализируются значениями из загруженных данных или принимают значения по умолчанию.

- `__root__` (Path): Путь к корневой директории проекта.
- `settings` (dict): Словарь с настройками проекта (из `settings.json`).
- `doc_str` (str): Текстовое содержимое файла `README.MD`.
- `__project_name__` (str): Название проекта (из `settings` или по умолчанию).
- `__version__` (str): Версия проекта (из `settings` или по умолчанию).
- `__doc__` (str): Документация проекта (из `README.MD` или по умолчанию).
- `__details__` (str): Дополнительные детали о проекте.
- `__author__` (str): Автор проекта (из `settings` или по умолчанию).
- `__copyright__` (str): Авторские права (из `settings` или по умолчанию).
- `__cofee__` (str): Ссылка для поддержки разработчика.

**Примечания**: Значения по умолчанию указаны в коде.
