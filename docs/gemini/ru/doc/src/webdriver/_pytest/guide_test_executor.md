## Введение

Класс `ExecuteLocator` предназначен для взаимодействия с веб-элементами через Selenium WebDriver.  Он предоставляет методы для работы с элементами, такими как получение атрибутов и отправка сообщений. Данное руководство поможет вам написать и запустить тесты, чтобы убедиться в корректной работе этого класса.

## Подготовка окружения

### Установка зависимостей

Убедитесь, что у вас установлены необходимые библиотеки. Выполните следующую команду в терминале:

```bash
pip install -r requirements.txt
```

Файл `requirements.txt` должен содержать следующие зависимости:

```
pytest==7.4.0
selenium==4.16.1
```

### Настройка WebDriver

Убедитесь, что WebDriver для вашего браузера (например, ChromeDriver для Chrome) установлен и доступен в вашей системе.  Путь к исполняемому файлу должен быть задан корректно.

## Запуск тестов

Для запуска тестов выполните следующую команду в корневой директории проекта:

```bash
pytest tests/test_executor.py
```


## Проверка результатов тестирования

`pytest` выведет результаты в терминале. Убедитесь, что все тесты прошли успешно.  Если тесты не прошли, `pytest` укажет на ошибку, и вы сможете исправить проблемные участки в коде.
