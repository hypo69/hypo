# Модуль `src.webdriver.chrome.extentions`

## Обзор

Модуль `src.webdriver.chrome.extentions` предназначен для управления и версионирования расширений Chrome в рамках проекта. Он включает в себя информацию о версии, документации и деталях расширения.

## Импорт зависимостей

Модуль импортирует следующие зависимости:

-   `packaging.version`: Используется для работы с версиями пакетов.
-   `__version__`, `__doc__`, `__details__` из `./version`: Переменные, содержащие информацию о версии, документации и деталях расширения.

## Переменные модуля

### `__version__`

**Описание**: Версия текущего расширения.

**Тип**: `str`

### `__doc__`

**Описание**: Документация расширения.

**Тип**: `str`

### `__details__`

**Описание**: Детали расширения.

**Тип**: `str`
