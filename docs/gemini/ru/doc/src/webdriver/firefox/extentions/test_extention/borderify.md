# borderify.js

## Обзор

Этот JavaScript-файл содержит код, который добавляет красную границу толщиной 5 пикселей к телу документа и выводит диалоговое окно с помощью функции `alert()`.

## Функции

### `document.body.style.border = "5px solid red";`

**Описание**: Устанавливает для элемента `body` CSS-свойство `border`, задавая красную границу толщиной 5 пикселей.

**Параметры**:  Нет параметров.  Данный код напрямую манипулирует свойством `border`.

**Возвращает**: Нет возвращаемых значений.


### `alert()`

**Описание**: Выводит диалоговое окно с сообщением.

**Параметры**: Нет параметров.

**Возвращает**: Нет возвращаемых значений.  Функция `alert` не возвращает явное значение.


## Использование

Этот файл можно использовать для добавления красной границы к документу и вывода диалогового окна.  Он предназначен для непосредственного включения в HTML-страницу.
