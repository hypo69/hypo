# Модуль `story.py`

## Обзор

Модуль `story.py` предоставляет инструменты для создания историй, связанных с симуляциями. Он позволяет генерировать истории на основе окружения (`TinyWorld`) или агента (`TinyPerson`), задавая цель и контекст.

## Классы

### `TinyStory`

**Описание**: Класс `TinyStory` предназначен для создания и управления историями. Он хранит информацию об окружении, агенте, цели, контексте и других настройках для генерации истории.

**Методы**:

- `__init__`: Инициализирует объект `TinyStory`.
    - **Параметры**:
        - `environment` (`TinyWorld`, необязательно): Окружение симуляции.
        - `agent` (`TinyPerson`, необязательно): Агент в симуляции.
        - `purpose` (`str`, необязательно, по умолчанию "Be a realistic simulation."): Цель истории.
        - `context` (`str`, необязательно, по умолчанию ""): Текущий контекст истории.
        - `first_n` (`int`, необязательно, по умолчанию 10): Количество первых взаимодействий для включения в историю.
        - `last_n` (`int`, необязательно, по умолчанию 20): Количество последних взаимодействий для включения в историю.
        - `include_omission_info` (`bool`, необязательно, по умолчанию `True`): Включать ли информацию об опущенных взаимодействиях.
    - **Возвращает**: `None`
    - **Исключения**:
        - `Exception`: Если не предоставлен ни `environment`, ни `agent`, или предоставлены оба.


- `start_story`: Инициализирует генерацию истории.
    - **Параметры**:
        - `requirements` (`str`, необязательно, по умолчанию "Start some interesting story about the agents."): Требования к истории.
        - `number_of_words` (`int`, необязательно, по умолчанию 100): Ограничение по количеству слов в истории.
        - `include_plot_twist` (`bool`, необязательно, по умолчанию `False`): Включать ли сюжетный поворот.
    - **Возвращает**: `str`: Текст начальной части истории.
    - **Исключений**: Нет.

- `continue_story`: Продолжает генерацию истории.
    - **Параметры**:
        - `requirements` (`str`, необязательно, по умолчанию "Continue the story in an interesting way."): Требования к продолжению истории.
        - `number_of_words` (`int`, необязательно, по умолчанию 100): Ограничение по количеству слов в продолжении.
        - `include_plot_twist` (`bool`, необязательно, по умолчанию `False`): Включать ли сюжетный поворот.
    - **Возвращает**: `str`: Текст продолжения истории.
    - **Исключений**: Нет.

- `_current_story`: Возвращает текущий текст истории.
    - **Параметры**: Нет.
    - **Возвращает**: `str`: Текущий текст истории, включающий данные об окружении или агенте.
    - **Исключений**: Нет.
