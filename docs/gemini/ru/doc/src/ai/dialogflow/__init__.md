# Модуль hypotez/src/ai/dialogflow

## Обзор

Модуль `hypotez/src/ai/dialogflow` предоставляет инструменты и константы для работы с диалоговым интерфейсом Dialogflow.  Подробное описание функциональности модуля отсутствует в представленном коде.

## Константы

### `MODE`

**Описание**: Константа, определяющая режим работы (например, 'dev', 'prod'). В данном случае установлено значение `'dev'`.

**Значение**: `'dev'`
