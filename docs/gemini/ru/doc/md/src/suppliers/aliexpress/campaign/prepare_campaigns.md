# Модуль `hypotez/src/suppliers/aliexpress/campaign/prepare_campaigns.py`

## Обзор

Этот модуль предназначен для подготовки рекламных кампаний AliExpress. Он обрабатывает категории, данные о кампаниях и генерирует рекламные материалы. Модуль поддерживает обработку отдельных кампаний и всех кампаний в каталоге.  Поддерживается указание языка и валюты для обработки.

## Функции

### `process_campaign_category`

**Описание**: Обрабатывает конкретную категорию внутри кампании для заданного языка и валюты.

**Параметры**:
- `campaign_name` (str): Название рекламной кампании.
- `category_name` (str): Категория для кампании.
- `language` (str): Язык для кампании.
- `currency` (str): Валюта для кампании.

**Возвращает**:
- `List[str]`: Список названий продуктов в категории.

**Пример использования**:
```python
titles: List[str] = process_campaign_category("summer_sale", "electronics", "EN", "USD")
print(titles)
```

### `process_campaign`

**Описание**: Обрабатывает кампанию, управляя настройкой и обработкой кампании.

**Параметры**:
- `campaign_name` (str): Название рекламной кампании.
- `language` (Optional[str], optional): Язык для кампании. Если не указан, обрабатывается для всех локалей.
- `currency` (Optional[str], optional): Валюта для кампании. Если не указан, обрабатывается для всех локалей.
- `campaign_file` (Optional[str], optional): Опциональное путь к конкретному файлу кампании.

**Возвращает**:
- `bool`: True, если кампания обработана, иначе False.


**Пример использования**:
```python
res = process_campaign("summer_sale", "EN", "USD", "campaign_file.json")
```

### `process_all_campaigns`

**Описание**: Обрабатывает все кампании в каталоге 'campaigns' для указанного языка и валюты.

**Параметры**:
- `language` (Optional[str], optional): Язык для кампаний.
- `currency` (Optional[str], optional): Валюта для кампаний.


**Пример использования**:
```python
process_all_campaigns("EN", "USD")
```

### `main_process`

**Описание**: Основная функция для обработки кампании.

**Параметры**:
- `campaign_name` (str): Название рекламной кампании.
- `categories` (List[str]): Список категорий для кампании. Если пустой, обрабатывается вся кампания без конкретных категорий.
- `language` (Optional[str], optional): Язык для кампании.
- `currency` (Optional[str], optional): Валюта для кампании.

**Пример использования**:
```python
main_process("summer_sale", ["electronics"], "EN", "USD")
```

### `main`

**Описание**: Основная функция для парсинга аргументов и запуска обработки.

**Пример использования**:
```python
main()
```

## Модульные переменные

- `MODE`: строковая константа, хранящая режим работы (например, 'dev').
- `campaigns_directory`: переменная, определяющая путь к каталогу кампаний.
