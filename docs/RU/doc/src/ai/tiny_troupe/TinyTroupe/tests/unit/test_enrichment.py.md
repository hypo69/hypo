# Документация модуля `test_enrichment.py`

## Обзор

Данный модуль содержит unit-тесты для проверки функциональности класса `TinyEnricher`, в частности, его метода `enrich_content`. Тесты проверяют, что метод корректно расширяет входной текст согласно заданным требованиям, делая результат как минимум в три раза длиннее исходного текста.

## Функции

### `test_enrich_content`

**Описание**:
Тест проверяет корректность работы метода `enrich_content` класса `TinyEnricher`. Он сравнивает длину обогащенного текста с исходным, а также проверяет на `None`.

**Параметры**:
- Нет параметров.

**Возвращает**:
- Нет возвращаемого значения.

**Вызывает исключения**:
- Нет исключений.
