"""

### Название игры: **HELLO** (Приветствие)
---
#### Описание
**HELLO** — это простая игра, в которой игрок вводит имя, а программа приветствует его с персонализированным сообщением. Игрок может повторить игру, вводя новое имя, или завершить программу.

---
### Пошаговая инструкция для реализации

#### 1. **Инициализация игры**
   - При запуске игры программа приветствует игрока и объясняет правила:
     ```
     Привет! Введи своё имя, и я тебя поприветствую.
     ```
   - Программа ожидает, что пользователь введёт своё имя. После этого программа выводит персонализированное сообщение, например:
     ```
     Привет, [имя]! Рад познакомиться!
     ```

#### 2. **Основной процесс игры**
   - **Ввод имени:**
     1. Игрок вводит своё имя. Программа проверяет, чтобы имя было непустым.
     2. Если имя пустое или состоит только из пробелов, программа выводит ошибку и запрашивает ввод повторно.

   - **Приветствие:**
     1. После ввода имени программа выводит приветственное сообщение, используя имя пользователя.

#### 3. **Продолжение игры**
   - После того как программа поприветствовала игрока, она задаёт вопрос:
     ```
     Хотите ввести другое имя? (да/нет)
     ```
   - Если игрок выбирает "да", программа снова запрашивает ввод имени и приветствует его.
   - Если игрок выбирает "нет", программа завершает работу:
     ```
     Спасибо за игру! До свидания!
     ```

#### 4. **Завершение игры**
   - Когда игрок завершает игру, программа прощается с ним и завершает выполнение.

---
