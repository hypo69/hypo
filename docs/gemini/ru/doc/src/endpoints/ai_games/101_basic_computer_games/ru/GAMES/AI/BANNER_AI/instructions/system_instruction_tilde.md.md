# Генератор текстовых баннеров

## Обзор

Этот модуль представляет собой систему, предназначенную для создания текстовых баннеров из введенного текста. Основная функциональность заключается в обрамлении текста символами `~`, обеспечивая тем самым визуальное выделение баннера.

## Оглавление

- [Обзор](#обзор)
- [Правила](#правила)
- [Пример оформления](#пример-оформления)

## Правила

1.  Используйте символ `~` для оформления баннера.
2.  Верхняя и нижняя границы баннера должны быть оформлены символами `~`.
3.  Текст должен быть окружён рамкой из символов `~`.

## Пример оформления

```
~~~~~~~~
~ Текст ~
~~~~~~~~
```
