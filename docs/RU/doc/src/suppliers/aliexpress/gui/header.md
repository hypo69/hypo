# Модуль hypotez/src/suppliers/aliexpress/gui/header.py

## Обзор

Данный модуль содержит функцию `set_project_root` для определения корневой директории проекта и функцию загрузки настроек из файла `settings.json`. Он использует переменную `MODE` для обозначения режима работы, и включает импорты необходимых библиотек.

## Функции

### `set_project_root`

**Описание**: Функция находит корневую директорию проекта, начиная от текущей директории файла, поднимаясь вверх по дереву директорий и останавливаясь на первой директории, содержащей один из указанных файлов-маркеров.

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, используемых для определения корневой директории. По умолчанию содержит `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена. Иначе возвращает директорию, в которой находится скрипт.

**Вызывает исключения**:

- Не вызывает никаких исключений.

### Инициализация настроек

**Описание**: Функция загружает настройки из файла `settings.json` в переменную `settings` и обрабатывает возможные исключения при чтении и декодировании файла.

**Параметры**:

- Не принимает параметры.

**Возвращает**:

- Нет возвращаемого значения.

**Вызывает исключения**:

- `FileNotFoundError`: Если файл `settings.json` не найден.
- `json.JSONDecodeError`: Если содержимое файла `settings.json` не является валидным JSON.


## Переменные

### `MODE`

**Описание**: Переменная, хранящая текущий режим работы. В текущем примере, она имеет значение 'dev'.
