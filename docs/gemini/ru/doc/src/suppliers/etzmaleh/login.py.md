## Функции

### `login`

**Описание**: Функция для авторизации поставщика.

**Параметры**:
- `s` (Any): Объект поставщика, для которого выполняется вход в систему.

**Возвращает**:
- `bool`: Возвращает `True`, если вход в систему выполнен успешно, в противном случае возвращает `False`.

**Вызывает исключения**:
- `None`: Функция не вызывает исключений.
