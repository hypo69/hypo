# Документация для FreeGPT WebUI

## Обзор

Данный документ предоставляет информацию о FreeGPT WebUI, проекте, который обеспечивает доступ к ChatGPT с ПК без использования VPN. Описаны особенности, процесс установки для Windows, а также информация о портативной версии, поддержке, используемых проектах и юридическом уведомлении.

## Подробней

FreeGPT WebUI представляет собой приложение, позволяющее использовать ChatGPT 3.5/4 непосредственно с компьютера. Проект включает в себя веб-интерфейс для взаимодействия с ботом и джейлбрейки для расширения возможностей. В настоящее время поддерживается ChatGPT 3.5 Turbo.

## Установка (Windows)

### Предварительные шаги

1.  Скачайте и установите [Git](https://git-scm.com/download/win).
2.  Скачайте и установите [Python 3.10.X](https://www.python.org/downloads/). Убедитесь, что Python добавлен в PATH.
3.  При возникновении ошибок, связанных с Visual Studio, скачайте и установите [Visual Studio](https://visualstudio.microsoft.com/ru/downloads/) для компиляции библиотек.

### Основные шаги

1.  Скачайте репозиторий, открыв командную строку (терминал) в папке, где хотите разместить `freegpt-webui-ru`, и выполните команду:

    ```
    git clone https://github.com/Em1tSan/freegpt-webui-ru.git
    ```

2.  Откройте папку `freegpt-webui-ru` и запустите файл `install.bat`, а затем `start.bat`. Это создаст виртуальную среду, установит зависимости и запустит скрипт.
3.  Перейдите в браузере по адресу: `http://127.0.0.1:1338`.

Если возникают трудности с классической установкой, рекомендуется использовать портативную версию.

## Портативная версия

Портативная версия не требует установки Python. Её можно скачать в [релизах](https://github.com/Em1tSan/freegpt-webui-ru/releases).
