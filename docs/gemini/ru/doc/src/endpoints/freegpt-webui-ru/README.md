# Документация для разработчика: FreeGPT WebUI

## Обзор

Данный документ предоставляет информацию о проекте FreeGPT WebUI, предназначенном для доступа к ChatGPT с персонального компьютера без использования VPN. В настоящее время поддерживается ChatGPT 3.5 Turbo, веб-доступ для бота и джейлбрейки для расширения возможностей. Документация содержит инструкции по установке, информацию о портативной версии, а также сведения об использованных проектах и юридические аспекты.

## Подробней

Этот проект предоставляет удобный интерфейс для взаимодействия с ChatGPT, позволяя пользователям обходить географические ограничения и расширять функциональность бота с помощью джейлбрейков. Он включает в себя как классическую установку, так и портативную версию для упрощения процесса развертывания.

## Установка, Windows

1.  **Скачайте и установите [git](https://git-scm.com/download/win). Скачайте и установите [Python 3.10.X](https://www.python.org/downloads/) Не забудьте применить, что надо добавить его в PATH.**
    Если в процессе установки у вас будут сыпаться ошибки, с упоминанием Visuial Studio - скачайте [Visual Studio](https://visualstudio.microsoft.com/ru/downloads/) для компиляции библиотек.

2.  **Скачайте репозиторий**: Откройте командную строку (терминал) в папке, где хотите разместить freegpt-webui-ru, и выполните команду:

    ```
    git clone https://github.com/Em1tSan/freegpt-webui-ru.git
    ```

3.  **Запустите установку**: Откройте папку freegpt-webui-ru и запустите файл `install.bat`, а затем `start.bat`. Это создаст виртуальную среду, установит зависимости и запустит скрипт.

4.  **Перейдите в браузере по адресу**: `http://127.0.0.1:1338`.

Если возникают трудности с классической установкой, можно воспользоваться портативной версией.

## Портативная версия

Портативная версия не требует установки Python. Скачать её можно в [релизах](https://github.com/Em1tSan/freegpt-webui-ru/releases).
