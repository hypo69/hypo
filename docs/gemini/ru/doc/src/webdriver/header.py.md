# Модуль `header`

## Обзор

Модуль `header` содержит функции и переменные, необходимые для определения корневой директории проекта. В основном, он используется для настройки путей и импортов в проекте `hypotez`.

## Подорбней

Этот модуль предоставляет функцию `set_project_root`, которая автоматически определяет корневую директорию проекта, начиная с текущей директории файла и двигаясь вверх по иерархии каталогов. Поиск останавливается, когда обнаруживается каталог, содержащий один из заданных файлов-маркеров (например, `__root__` или `.git`). Это позволяет динамически настраивать пути к файлам и модулям внутри проекта, независимо от текущего рабочего каталога.

## Функции

### `set_project_root`

```python

    """
    Args:
        marker_files (tuple): Filenames or directory names to identify the project root.

    Returns:
        Path: Path to the root directory if found, otherwise the directory where the script is located.
    """

```

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла, и останавливается на первой директории, содержащей любой из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple): Имена файлов или директорий, используемые для идентификации корневой директории проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена, иначе директория, где расположен скрипт.

**Примеры**:

```python
from pathlib import Path
root_path = set_project_root()
print(f"Root path: {root_path}")
# Root path: /путь/к/корневой/директории
```

## Переменные

### `__root__`

```python
"""__root__ (Path): Path to the root directory of the project"""
```

**Описание**: Путь к корневой директории проекта. Определяется функцией `set_project_root`.
