# Модуль `src.suppliers.ivory.__morlevi__`

## Обзор

Модуль `src.suppliers.ivory.__morlevi__` предназначен для сбора данных о товарах с сайта поставщика Morlevi. Он включает в себя функции для входа на сайт, сбора информации о товарах с их страниц, а также получения списка товаров из категорий.

## Функции

### `login`

**Описание**:
Функция выполняет вход на сайт поставщика Morlevi. Она обрабатывает модальные окна и обеспечивает повторные попытки входа в случае неудачи.

**Параметры**:
- `supplier` (Supplier): Объект поставщика с необходимыми данными для входа.

**Возвращает**:
- `bool | None`: Возвращает `True`, если вход успешен, `None` в случае ошибки.

**Вызывает исключения**:
- `Exception`: Общее исключение при возникновении ошибки во время входа.

### `_login`

**Описание**:
Внутренняя функция для выполнения фактического входа на сайт Morlevi.

**Параметры**:
- `_s` (Supplier): Объект поставщика.

**Возвращает**:
- `bool | None`: Возвращает `True`, если вход выполнен, `None` в случае ошибки.

**Вызывает исключения**:
- `Exception`: Вызывает исключение, если вход не удался.

### `grab_product_page`

**Описание**:
Функция извлекает информацию о товаре со страницы товара на сайте Morlevi. Она использует локаторы для поиска элементов и заполняет поля объекта `Product`.

**Параметры**:
- `s` (Supplier): Объект поставщика.

**Возвращает**:
- `Product`: Возвращает объект `Product` с собранными данными о товаре.

### `list_products_in_category_from_pagination`

**Описание**:
Функция получает список ссылок на товары из категории, обрабатывая пагинацию, если она есть.

**Параметры**:
- `supplier` (Supplier): Объект поставщика.

**Возвращает**:
- `list`: Возвращает список ссылок на товары в категории.

### `get_list_products_in_category`

**Описание**:
Функция для получения списка товаров в категории.

**Параметры**:
- `s` (Supplier): Объект поставщика.
- `scenario` (JSON): JSON-объект со сценарием.
- `presath` (PrestaShopWebServiceDict): Объект для взаимодействия с PrestaShop.

**Возвращает**:
- `None`: Функция не возвращает значения.

### `get_list_categories_from_site`

**Описание**:
Функция для получения списка категорий с сайта.

**Параметры**:
- `s` (Supplier): Объект поставщика.
- `scenario_file` (str): Путь к файлу сценария.
- `brand` (str, optional): Бренд. По умолчанию `''`.

**Возвращает**:
- `None`: Функция не возвращает значения.
