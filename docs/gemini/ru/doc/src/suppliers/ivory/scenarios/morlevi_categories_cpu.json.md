## Обзор

Данный файл `morlevi_categories_cpu.json` содержит конфигурацию для сбора данных о процессорах (CPU) с сайта morlevi.co.il. Файл описывает различные сценарии для процессоров Intel, сгруппированные по моделям и поколениям. Каждый сценарий определяет URL для сбора данных, а также информацию о соответствии категориям PrestaShop.

## Структура JSON

### Сценарии

JSON-файл содержит основной объект `scenarios`, который представляет собой словарь. Каждый ключ этого словаря является названием сценария (например, "Intel CELERON LGA1200 Gen 10"). Значение каждого ключа - это словарь, описывающий параметры сценария:

- **brand**: (string) Бренд процессора, всегда "INTEL".
- **url**: (string) URL страницы на сайте morlevi.co.il, откуда нужно собирать информацию.
- **checkbox**: (boolean) Флаг, указывающий на наличие чекбокса в UI. Всегда `false`.
- **active**: (boolean) Флаг, указывающий, активен ли сценарий. Всегда `true`.
- **condition**: (string) Состояние товара, всегда "new".
- **presta_categories**: (object) Объект, содержащий соответствия для категорий PrestaShop.
    - **template**: (object) Объект, определяющий шаблон категории.
        - **cpu**: (string) Значение для категории CPU, например "INTEL CELERON LGA1200".

## Описание полей

### `scenarios`

Словарь, содержащий все сценарии сбора данных.

#### Ключ сценария

Название сценария, описывающего конкретную модель процессора, например `"Intel CELERON LGA1200 Gen 10"`.

#### `brand`

**Описание**: Бренд процессора.

**Тип**: `str`

**Значение**: Всегда `"INTEL"`.

#### `url`

**Описание**: URL страницы на сайте morlevi.co.il для сбора данных.

**Тип**: `str`

**Пример**: `"https://www.morlevi.co.il/Cat/337?p_134=584&sort=datafloat2%2Cprice&keyword="`

#### `checkbox`

**Описание**: Флаг, указывающий на наличие чекбокса в UI.

**Тип**: `bool`

**Значение**: Всегда `false`.

#### `active`

**Описание**: Флаг, указывающий, активен ли сценарий.

**Тип**: `bool`

**Значение**: Всегда `true`.

#### `condition`

**Описание**: Состояние товара.

**Тип**: `str`

**Значение**: Всегда `"new"`.

#### `presta_categories`

**Описание**: Объект, содержащий соответствия для категорий PrestaShop.

**Тип**: `object`

##### `template`

**Описание**: Объект, определяющий шаблон категории.

**Тип**: `object`

###### `cpu`

**Описание**: Значение для категории CPU.

**Тип**: `str`

**Пример**: `"INTEL CELERON LGA1200"`, `"I5 LGA1700"`
