# BOUNCE

## סקירה כללית

משחק "הכדור הקופץ" הוא אנימציה פשוטה המדמה את תנועת הכדור הקופץ מהגבולות העליונים והתחתונים של המסך. המשתמש יכול ללחוץ על מקש כלשהו כדי לעצור את האנימציה. גובה הכדור משתנה לפי סינוס.

## פונקציות

### `clear_screen`

**תיאור**: מנקה את מסך הטרמינל.

**פרמטרים**: אין

**החזרות**: אין

### `get_keypress`

**תיאור**: בודקת אם נלחץ מקש, מבלי לחסום את הביצוע.

**פרמטרים**: אין

**החזרות**:
- `bool`: מחזירה `True` אם נלחץ מקש, אחרת `False`.

## משתנים

- `Y`: גובה ההתחלתי של הכדור.
- `T`: ערך התחלתי עבור פונקציית הסינוס.
- `D`: צעד השינוי של פונקציית הסינוס.

## לולאה עיקרית

- מחשבת את מיקום הכדור החדש באמצעות הפונקציה `Y = 20 + 19 * math.sin(T)`.
- מנקה את המסך באמצעות הפונקציה `clear_screen()`.
- מדפיסה את הכדור במיקום החדש.
- מעדכנת את הערך של `T` עבור הפריים הבא.
- מחכה 0.1 שניות באמצעות הפונקציה `time.sleep(0.1)`.
- בודקת אם נלחץ מקש באמצעות הפונקציה `get_keypress()`.
- יוצאת מהלולאה אם נלחץ מקש.

## הסבר קוד

1. **ייבוא מודולים:**
   - `import math`: מייבא את המודול `math` לשימוש בפונקציה המתמטית `sin`.
   - `import time`: מייבא את המודול `time` לניהול עיכוב באנימציה.
   - `import os`: מייבא את המודול `os` לפעולות הקשורות למערכת ההפעלה.
   - `import sys`: מייבא את המודול `sys` לפרמטרים של מערכת.
2. **פונקציה `clear_screen()`:**
   - הפונקציה `clear_screen()` משמשת לניקוי מסך הטרמינל.
   - בהתאם למערכת ההפעלה (Windows או Unix-like) מופעלת פקודת ניקוי המסך המתאימה.
3. **פונקציה `get_keypress()`:**
    - הפונקציה `get_keypress()` בודקת אם נלחץ מקש כלשהו.
    - עבור Windows נעשה שימוש במודול `msvcrt`, כאשר `kbhit()` בודק אם נלחץ מקש, ו-`getch()` קורא את התו.
    - עבור מערכות Unix-like נעשה שימוש ב-`select.select` לקריאת תו לא חוסמת מ-stdin.
    - הפונקציה מחזירה `True` אם נלחץ מקש, ו-`False` אחרת.
4. **איתחול משתנים:**
   - `Y = 20`: הגובה האנכי ההתחלתי של הכדור.
   - `T = 0`: הערך ההתחלתי של פונקציית הסינוס.
   - `D = 0.2`: צעד השינוי של פונקציית הסינוס.
5. **לולאה אינסופית `while True:`:**
   - הלולאה ממשיכה עד שלוחצים על מקש כלשהו.
   - `Y = 20 + 19 * math.sin(T)`: חישוב המיקום האנכי החדש של הכדור באמצעות סינוס.
   - `clear_screen()`: ניקוי המסך לפני ציור המיקום החדש.
   - `print(" " * int(Y) + "O")`: הדפסת הכדור (`O`) על המסך.
   - `T += D`: הגדלת הערך של `T` עבור השלב הבא באנימציה.
   - `time.sleep(0.1)`: יצירת עיכוב של 0.1 שניות כדי להאט את האנימציה.
   - `if get_keypress():`: בדיקה אם נלחץ מקש.
      - `break`: סיום הלולאה אם נלחץ מקש.
