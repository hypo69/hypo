# Модуль hypotez/src/utils/convertors/base64

## Обзор

Этот модуль предоставляет функцию для декодирования кодированного в Base64 содержимого и записи его во временный файл с указанным расширением.

## Функции

### `base64_to_tmpfile`

**Описание**: Преобразует содержимое, закодированное в Base64, во временный файл.  Функция декодирует содержимое, закодированное в Base64, и записывает его во временный файл с таким же расширением, как и в предоставленном имени файла. Возвращает путь к временному файлу.

**Параметры**:
- `content` (str): Содержимое, закодированное в Base64, которое будет декодировано и записано в файл.
- `file_name` (str): Имя файла, используемое для извлечения расширения файла для временного файла.


**Возвращает**:
- str: Путь к временному файлу.

**Примеры**:
```python
>>> base64_content = "SGVsbG8gd29ybGQh"  # Base64 закодированное содержимое "Hello world!"
>>> file_name = "example.txt"
>>> tmp_file_path = base64_to_tmpfile(base64_content, file_name)
>>> print(f"Временный файл создан по адресу: {tmp_file_path}")
Временный файл создан по адресу: /tmp/tmpfile.txt
```

**Примечания**:
Временный файл создается с помощью `tempfile.NamedTemporaryFile`. Флаг `delete=False` гарантирует, что файл не будет автоматически удален после закрытия.


### `base64encode`

**Описание**: Функция для кодирования изображения в Base64.  

**Параметры**:
- `image_path` (str): Путь к изображению, которое необходимо закодировать.


**Возвращает**:
- str: Строка, содержащая закодированное изображение в Base64.

**Примечания**:
Функция открывает файл изображения в бинарном режиме (`"rb"`), читает его содержимое и кодирует в Base64. Результат кодирования возвращается в виде строки UTF-8.
