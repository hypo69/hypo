# Инструкция по генерации документации в формате Markdown

## Обзор

Данная инструкция описывает процесс генерации документации в формате Markdown для Python-файлов. Документация должна быть структурированной и содержать подробные описания функций, классов и методов.

## Требования к формату документации

* **Формат:** Используется стандарт Markdown (.md).
* **Заголовки:** Используются заголовки первого, второго, третьего и четвёртого уровней (#, ##, ###, ####).
* **Описание модуля:** Каждый файл начинается с заголовка и краткого описания его содержимого.
* **Комментарии к функциям/методам:** Для функций и методов используются стандартные комментарии docstring с секциями `Args`, `Returns`, `Raises`.
* **Обработка исключений:** Вместо `e` используется `ex` в блоках обработки исключений.
* **Оглавление (TOC):** В начале каждого файла документации добавляется раздел с оглавлением, содержащий ссылки на все основные разделы.
* **Форматирование:**  Используются списки, правильный синтаксис Markdown для заголовков, списков и ссылок.  Документация должна быть структурированной и чёткой. Примеры приведены в инструкции.

## Пример

```markdown
# Модуль `my_module`

## Обзор

Этот модуль содержит функции для работы с числами.


## Функции

### `add_numbers`

**Описание**: Функция складывает два числа.

**Параметры**:
- `a` (int): Первое число.
- `b` (int): Второе число.

**Возвращает**:
- int: Сумма `a` и `b`.

**Вызывает исключения**:
- `TypeError`: Если входные данные не являются числами.


```

## Обработка входных Python-файлов

Для каждого входного Python-файла необходимо сгенерировать соответствующую Markdown-документацию, следуя описанным требованиям.  Для генерации оглавления можно использовать инструменты или скрипты.  Важно прочитать и понять структуру и содержание входных файлов.

## Форматирование примеров

Примеры кода должны быть оформлены в блоках кода, например:

```python
def add_numbers(a: int, b: int) -> int:
    """
    Args:
        a (int): Первое число.
        b (int): Второе число.

    Returns:
        int: Сумма a и b.

    Raises:
        TypeError: Если входные данные не являются числами.
    """
    try:
        return a + b
    except TypeError as ex:
        print(f"Ошибка: {ex}")
        return None
```
