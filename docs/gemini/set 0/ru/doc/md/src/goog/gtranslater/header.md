# Модуль hypotez/src/goog/gtranslater/header.py

## Обзор

Данный модуль `hypotez/src/goog/gtranslater/header.py` содержит вспомогательные функции для работы с проектом, включая определение корневой директории проекта, чтение настроек из файла `settings.json` и документации из `README.MD`.  Он также инициализирует переменные, необходимые для работы других модулей проекта, например,  `__root__`,  `__project_name__`,  `__version__`, `__doc__`,  `__details__`, `__author__`, `__copyright__`, `__cofee__`.


## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с текущей директории и идя вверх по дереву директорий, пока не найдет директорию, содержащую один из указанных файлов или директорий.

**Параметры**:
- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, используемые для поиска корневой директории проекта. По умолчанию: ('pyproject.toml', 'requirements.txt', '.git').


**Возвращает**:
- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращается путь к директории, где находится текущий скрипт.


**Вызывает исключения**:
-  Не вызывает исключений.


###  `__init__`

**Описание**: Инициализирует переменные, которые будут использоваться другими модулями, такие как `__root__`, `__project_name__`, и т.д.,  из файла `settings.json` и `README.MD`.


**Параметры**:
-  Нет параметров.


**Возвращает**:
-  Нет значения возврата.


**Вызывает исключения**:
- `FileNotFoundError`: Если файл `settings.json` или `README.MD` не найден.
- `json.JSONDecodeError`: Если содержимое файла `settings.json` не является валидным JSON.


## Переменные

### `MODE`

**Описание**:  Переменная, определяющая режим работы приложения ('dev' или другие значения).


### `__root__`

**Описание**: Путь к корневой директории проекта, полученный из функции `set_project_root()`.


### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженные из файла `settings.json`.


### `doc_str`

**Описание**:  Строка, содержащая текст из файла `README.MD`.


### `__project_name__`

**Описание**: Название проекта.  Получается из настроек, если они доступны, иначе значение по умолчанию – 'hypotez'.


### `__version__`

**Описание**: Версия проекта.  Получается из настроек, если они доступны, иначе значение по умолчанию – ''.


### `__doc__`

**Описание**:  Документация проекта. Получается из настроек, если они доступны, иначе значение по умолчанию – ''.


### `__details__`

**Описание**: Подробная информация о проекте. Значение по умолчанию – ''.


### `__author__`

**Описание**: Автор проекта.  Получается из настроек, если они доступны, иначе значение по умолчанию – ''.


### `__copyright__`

**Описание**: Авторские права. Получается из настроек, если они доступны, иначе значение по умолчанию – ''.


### `__cofee__`

**Описание**: Ссылка на  поддержку разработчика. Получается из настроек, если они доступны, иначе значение по умолчанию - ссылка на Boosty.
