# Модуль hypotez/src/category/header.py

## Обзор

Данный модуль определяет корневой путь к проекту `hypotez`.  Все импорты строятся относительно этого пути. В будущем этот функционал планируется перенести в системную переменную.


## Функции

### `set_project_root`

**Описание**: Функция находит корневой каталог проекта, начиная с текущей директории и поднимаясь вверх по дереву директорий, пока не найдет директорию содержащую один из указанных файлов-маркеров (pyproject.toml, requirements.txt, .git). Если корневой каталог не найден, функция возвращает директорию, где находится сам скрипт.

**Параметры**:

- `marker_files` (tuple): Кортеж с именами файлов или директорий, по которым будет определяться корневой каталог. По умолчанию включает `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Объект `Path` представляющий путь к корневому каталогу проекта. Если корневой каталог не найден, возвращается текущая директория.


**Вызывает исключения**:

- Не вызывает исключений.


## Переменные

### `__root__`

**Описание**:  Переменная, содержащая объект `Path` с путем к корневому каталогу проекта, полученный с помощью функции `set_project_root()`.


## Дополнительные замечания

- Функция `set_project_root` добавляет корневой каталог проекта в `sys.path`, что позволяет импортировать модули из подкаталогов проекта.
- Модуль использует библиотеку `pathlib` для работы с путями.
- Модуль подключает модуль `src.gs`.
- Модуль пытается загрузить настройки из файла `src/settings.json` и документацию из файла `src/README.MD`.  Если файлы не найдены или не могут быть обработаны, соответствующие переменные остаются `None` или пустыми строками.

- `__project_name__`, `__version__`, `__doc__`, `__details__`, `__author__`, `__copyright__`, `__cofee__`:  Переменные, содержащие информацию о проекте, извлеченные из файла `settings.json`. Если файл не найден или данные отсутствуют, используются значения по умолчанию.
