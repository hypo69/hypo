# `grab_lilnks_to_chats.py`

## Обзор

Модуль предназначен для сбора ссылок на отдельные чаты с веб-страницы, используя Selenium. Модуль использует драйверы `Chrome` и `Firefox` для навигации по веб-странице и извлечения необходимых данных.

   - [`get_links`](#get_links)

## Функции

### `get_links`

**Описание**: Извлекает ссылки на отдельные чаты со страницы.

**Параметры**:
- `d` (`Driver`): Экземпляр драйвера браузера.

**Возвращает**:
- `list`: Список элементов, представляющих ссылки на чаты.

## Основной блок

**Описание**: Инициализирует драйвер Firefox, переходит по URL и получает ссылки на чаты.

**Детали**:

- Создается экземпляр драйвера `Firefox`.
- Загружается веб-страница `https://chatgpt.com/`.
- Вызывается функция `get_links` для извлечения ссылок.
- Выполняется дальнейшая обработка полученных ссылок (обозначено `...`).
