# Модуль PerplexityApi

## Обзор

Модуль `PerplexityApi` предназначен для взаимодействия с API Perplexity AI. Он наследует функциональность от класса `OpenaiTemplate` и предоставляет специфические параметры для работы с Perplexity AI, включая URL, модели и необходимость аутентификации.

## Подробней

Модуль определяет класс `PerplexityApi`, который предоставляет интерфейс для взаимодействия с API Perplexity AI. Он устанавливает базовый URL, список поддерживаемых моделей и флаг, указывающий на необходимость аутентификации.

## Классы

### `PerplexityApi`

**Описание**: Класс `PerplexityApi` предоставляет интерфейс для работы с API Perplexity AI.

**Наследует**:
- `OpenaiTemplate`: Класс `PerplexityApi` наследует от `OpenaiTemplate`, что позволяет повторно использовать общую логику для взаимодействия с API, подобными OpenAI.

**Атрибуты**:
- `label` (str): Метка для Perplexity API (значение: `"Perplexity API"`).
- `url` (str): URL веб-сайта Perplexity AI (значение: `"https://www.perplexity.ai"`).
- `login_url` (str): URL для настроек API, используемый для входа в систему (значение: `"https://www.perplexity.ai/settings/api"`).
- `working` (bool): Указывает, работает ли API в данный момент (значение: `True`).
- `needs_auth` (bool): Указывает, требуется ли аутентификация для использования API (значение: `True`).
- `api_base` (str): Базовый URL для API Perplexity AI (значение: `"https://api.perplexity.ai"`).
- `default_model` (str): Модель по умолчанию, используемая API (значение: `"llama-3-sonar-large-32k-online"`).
- `models` (List[str]): Список поддерживаемых моделей API.

**Методы**:
- Отсутствуют явно определенные методы, но класс наследует методы от `OpenaiTemplate`.

**Принцип работы**:
1. Класс `PerplexityApi` наследует от класса `OpenaiTemplate`, предоставляя готовый шаблон для взаимодействия с API, подобными OpenAI.
2. Устанавливаются значения атрибутов, специфичные для Perplexity API, такие как URL, необходимость аутентификации, базовый URL API и список поддерживаемых моделей.
3. Когда экземпляр класса используется для отправки запросов к API Perplexity AI, он использует эти атрибуты для настройки соединения и обработки ответов.

## Функции

В данном модуле не определены отдельные функции, он содержит только определение класса `PerplexityApi`.
