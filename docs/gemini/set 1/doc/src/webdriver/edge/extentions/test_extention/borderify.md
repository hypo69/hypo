# Модуль borderify.js

## Обзор

Данный скрипт JavaScript добавляет красную границу толщиной 5 пикселей к телу документа. Также выводит диалоговое окно alert().

## Функции

### `document.body.style.border = "5px solid red";`

**Описание**: Устанавливает стили для элемента body документа. Добавляет красную границу толщиной 5 пикселей.

**Параметры**:
- Отсутствуют

**Возвращает**:
- Отсутствует. Изменяет стили DOM-элемента.


### `alert()`

**Описание**: Выводит диалоговое окно с сообщением alert.

**Параметры**:
- Отсутствуют

**Возвращает**:
- Отсутствует.

**Вызывает исключения**:
- Отсутствуют.
