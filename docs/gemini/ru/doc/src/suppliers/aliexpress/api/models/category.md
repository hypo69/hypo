# Модуль `hypotez/src/suppliers/aliexpress/api/models/category.py`

## Обзор

Этот модуль определяет классы для работы с категориями на сайте AliExpress.  Он содержит базовый класс `Category` и дочерний класс `ChildCategory`, расширяющий функциональность для работы с родительскими категориями.


## Классы

### `Category`

**Описание**: Базовый класс для представления категории. Хранит идентификатор категории и её название.

**Атрибуты**:

- `category_id` (int): Идентификатор категории.
- `category_name` (str): Название категории.


### `ChildCategory`

**Описание**: Класс, представляющий дочернюю категорию. Наследует атрибуты класса `Category` и добавляет атрибут для идентификации родительской категории.

**Атрибуты**:

- `parent_category_id` (int): Идентификатор родительской категории.
- `category_id` (int): Идентификатор дочерней категории.
- `category_name` (str): Название дочерней категории.
