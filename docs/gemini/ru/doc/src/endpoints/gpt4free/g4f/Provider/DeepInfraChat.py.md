# Модуль DeepInfraChat для взаимодействия с платформой DeepInfra

## Обзор

Модуль `DeepInfraChat.py` предоставляет класс `DeepInfraChat`, который наследуется от `OpenaiTemplate` и предназначен для взаимодействия с платформой DeepInfra. Он определяет URL, базовый API, список поддерживаемых моделей и псевдонимы моделей, специфичные для DeepInfra.

## Подробней

Этот модуль является частью проекта `hypotez` и обеспечивает интеграцию с сервисом DeepInfra для использования различных моделей машинного обучения, включая модели для работы с текстом и изображениями. Он определяет настройки подключения и список поддерживаемых моделей, что позволяет легко использовать эти модели через интерфейс `OpenaiTemplate`.

## Классы

### `DeepInfraChat`

**Описание**: Класс для взаимодействия с платформой DeepInfra.

**Наследует**:

- `OpenaiTemplate`: Предоставляет базовый функционал для взаимодействия с API OpenAI-совместимых платформ.

**Атрибуты**:

- `url` (str): URL для доступа к DeepInfra Chat.
- `api_base` (str): Базовый URL для API DeepInfra.
- `working` (bool): Флаг, указывающий на работоспособность провайдера (в данном случае `True`).
- `default_model` (str): Модель, используемая по умолчанию (`deepseek-ai/DeepSeek-V3`).
- `default_vision_model` (str): Модель для работы с изображениями по умолчанию (`openbmb/MiniCPM-Llama3-V-2_5`).
- `vision_models` (list[str]): Список поддерживаемых моделей для работы с изображениями.
- `models` (list[str]): Список поддерживаемых моделей для работы с текстом.
- `model_aliases` (dict[str, str]): Словарь псевдонимов моделей для упрощения их использования.

**Методы**:
- Отсутствуют, используются методы родительского класса `OpenaiTemplate`.

**Принцип работы**:

Класс `DeepInfraChat` настраивает параметры подключения к платформе DeepInfra и определяет список доступных моделей. При инициализации класса устанавливаются значения URL, базового API, списка моделей и псевдонимов моделей. Эти параметры используются для взаимодействия с API DeepInfra через методы, унаследованные от `OpenaiTemplate`.

## Переменные класса `DeepInfraChat`

- `url`: URL для доступа к DeepInfra Chat.
- `api_base`: Базовый URL для API DeepInfra.
- `working`: Флаг, указывающий на работоспособность провайдера.
- `default_model`: Модель, используемая по умолчанию.
- `default_vision_model`: Модель для работы с изображениями по умолчанию.
- `vision_models`: Список поддерживаемых моделей для работы с изображениями.
- `models`: Список поддерживаемых моделей для работы с текстом.
- `model_aliases`: Словарь псевдонимов моделей.
