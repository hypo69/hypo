Description
----------

This file contains a question-answer pair.  The question asks why configuration files are named after the module (e.g., `suppliers.json`), suggesting an alternative name like `config.json`.  The answer explains that the current naming scheme is intended to improve understandability for AI models.


Question
-------

Why are configuration files named after the module (e.g., `suppliers.json`)? Wouldn't it be better to name them `config.json`?


Answer
------

File names are designed for improved comprehension by AI models.
