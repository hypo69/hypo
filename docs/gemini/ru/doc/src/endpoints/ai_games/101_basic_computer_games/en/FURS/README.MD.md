# FURS: Экспедиция по Торговле Мехом

## Обзор

Этот документ описывает игру FURS, в которой игрок выступает в роли лидера французской экспедиции по торговле мехом в 1776 году. Игрок покидает район Онтарио, чтобы продать меха и получить припасы на следующий год. В игре есть выбор из трех фортов для торговли, где стоимость припасов и цена на меха зависят от выбранного форта. Игрок также указывает типы мехов, которые у него есть для торговли. Игра продолжается до тех пор, пока игрок не решит прекратить торговлю.

## Описание Игры

### Введение
Игра моделирует экспедицию по торговле мехом, где игрок принимает решения о том, где и какие меха продавать, а также какие припасы покупать.

### Игровой процесс

Игровой процесс включает следующие этапы:

1. **Выбор Форта**: Игрок выбирает один из трех фортов для торговли.
2. **Торговля Мехом**: Игрок указывает, какие меха и в каком количестве он хочет продать.
3. **Покупка Припасов**: Игрок покупает необходимые припасы, опираясь на имеющиеся средства.
4. **Завершение Торговли**: Игрок может продолжить торговлю в других фортах или завершить игру.

### Цель

Цель игры - эффективно управлять ресурсами и получить максимальную прибыль от торговли мехом, чтобы обеспечить экспедицию необходимыми припасами на следующий год.

### Продолжение Игры

Игра продолжается, пока игрок не решит прекратить торговлю, то есть он может возвращаться в разные форты, если он еще не получил достаточно припасов.
