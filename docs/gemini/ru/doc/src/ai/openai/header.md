# Модуль `hypotez/src/logger/header.py`

## Обзор

Этот модуль определяет корневой путь к проекту `hypotez`.  Все импорты строятся относительно этого пути.  В будущем планируется перенести это определение в системную переменную.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с текущей директории файла и идя вверх по дереву директорий, пока не найдет директорию содержащую один из файлов-маркеров (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж из имен файлов или директорий, используемых для определения корневой директории проекта.  По умолчанию: `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Объект `Path` представляющий путь к корневой директории проекта, если она найдена. В противном случае возвращает директорию, где находится текущий скрипт.


## Глобальные переменные

### `__root__`

**Описание**: Путь к корневой директории проекта. Получается вызовом функции `set_project_root`.

**Тип**: `Path`

### `settings`

**Описание**: Словарь настроек проекта, загруженный из файла `src/settings.json`.

**Тип**: `dict` или `None`


### `doc_str`

**Описание**: Строка документации проекта, загруженная из файла `src/README.MD`.

**Тип**: `str` или `None`



### `__project_name__`

**Описание**: Название проекта, полученное из файла `settings.json`. По умолчанию `'hypotez'`.

**Тип**: `str`

### `__version__`

**Описание**: Версия проекта, полученная из файла `settings.json`. По умолчанию пустая строка.

**Тип**: `str`


### `__doc__`

**Описание**: Документация проекта. Загружается из файла `README.MD`. Если файл не найден, то значение по умолчанию - пустая строка.

**Тип**: `str`

### `__details__`

**Описание**: Подробная информация о проекте (пустая строка по умолчанию).

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта, полученное из файла `settings.json`. По умолчанию пустая строка.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права на проект, полученные из файла `settings.json`. По умолчанию пустая строка.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка на страницу для поддержки автора проекта (благодарность). Получается из файла `settings.json`. По умолчанию - предопределённая ссылка.

**Тип**: `str`
