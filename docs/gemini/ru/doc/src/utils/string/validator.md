# Модуль hypotez/src/utils/string/validator.py

## Обзор

Модуль `validator.py` предоставляет набор статических методов для валидации строк, таких как цены, веса, артикулы и URL-адреса.  Методы проверяют соответствие строк заданным шаблонам и критериям, возвращая `True`, если строка валидна, и `None` или `False` в противном случае.

## Классы

### `ProductFieldsValidator`

**Описание**: Класс `ProductFieldsValidator` содержит статические методы для валидации различных полей продуктов, таких как цена, вес и артикул.

**Статические методы**:

#### `validate_price(price: str) -> bool`

**Описание**: Проверяет корректность формата цены.

**Параметры**:

- `price` (str): Строка, представляющая цену.

**Возвращает**:

- `bool`: `True`, если цена валидна, `None` иначе.

#### `validate_weight(weight: str) -> bool`

**Описание**: Проверяет корректность формата веса.

**Параметры**:

- `weight` (str): Строка, представляющая вес.

**Возвращает**:

- `bool`: `True`, если вес валиден, `None` иначе.


#### `validate_sku(sku: str) -> bool`

**Описание**: Проверяет корректность формата артикула (SKU).

**Параметры**:

- `sku` (str): Строка, представляющая артикул.

**Возвращает**:

- `bool`: `True`, если артикул валиден, `None` иначе.

#### `validate_url(url: str) -> bool`

**Описание**: Проверяет корректность формата URL-адреса.

**Параметры**:

- `url` (str): Строка, представляющая URL-адрес.

**Возвращает**:

- `bool`: `True`, если URL валиден, `None` иначе.


#### `isint(s: str) -> bool`

**Описание**: Проверяет, является ли строка целым числом.

**Параметры**:

- `s` (str): Строка, представляющая число.

**Возвращает**:

- `bool`: `True`, если строка представляет целое число, `None` в противном случае.
