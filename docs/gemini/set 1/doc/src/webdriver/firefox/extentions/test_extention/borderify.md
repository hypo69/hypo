# Модуль borderify.js

## Обзор

Этот модуль добавляет красную границу толщиной 5 пикселей к телу документа.

## Функции

### `document.body.style.border = "5px solid red";`

**Описание**: Устанавливает свойство `border` элемента `document.body` на красную границу толщиной 5 пикселей.

**Возвращает**:  Не имеет возвращаемого значения.


### `alert()`

**Описание**: Вызывает диалоговое окно с сообщением.


**Возвращает**: Не имеет возвращаемого значения.
