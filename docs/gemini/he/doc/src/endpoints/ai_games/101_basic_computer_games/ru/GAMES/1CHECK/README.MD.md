## הסבר על הקוד

1. **אתחול לוח הדמקה**:
   - המחלקה `Checkerboard` מאתחלת רשת 8x8 עם 48 כלים המוצבים בשורות החיצוניות.

2. **בדיקת חוקיות הקפיצה**:
   - הפונקציה `is_valid_jump` בודקת אם הקפיצה חוקית על ידי הבטחת:
     - מיקום הנחיתה הוא בתוך הלוח.
     - מיקום הנחיתה ריק.
     - יש כלי לקפוץ מעליו.
     - הקפיצה היא אלכסונית.

3. **ביצוע קפיצות**:
   - הפונקציה `perform_jump` מעבירה את הכלי ומסירה את הכלי שקפצו מעליו, ומעדכנת את הלוח ואת ספירת הכלים שהוסרו.

4. **מציאת קפיצות אפשריות**:
   - הפונקציה `find_possible_jumps` סורקת את הלוח כדי למצוא את כל הקפיצות החוקיות.

5. **סימולציית המשחק**:
   - הפונקציה `play` מדמה את המשחק על ידי ביצוע קפיצות חוקיות שוב ושוב עד שלא ניתן לבצע עוד קפיצות.

6. **תצוגת הלוח**:
   - הפונקציה `display_board` מדפיסה את המצב הנוכחי של הלוח להדמיה.
