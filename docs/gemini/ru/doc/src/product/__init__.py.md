# Модуль `src.product`

## Обзор

Модуль `src.product` предназначен для управления продуктами. Он содержит классы и функции, необходимые для работы с данными продуктов, включая их поля и перевод.

## Содержание

- [Обзор](#обзор)
- [Константы](#константы)
- [Импортированные модули](#импортированные-модули)

## Константы

### `MODE`

**Описание**: Режим работы приложения. По умолчанию установлен в `dev`.

## Импортированные модули

### `Product`
   
**Описание**:  Класс для работы с продуктами. Детальное описание можно найти в `product.py`.

### `ProductFields`
   
**Описание**:  Класс для работы с полями продуктов. Детальное описание можно найти в `product_fields.py`.

### `translate_presta_fields_dict`
   
**Описание**:  Функция для перевода многоязычных полей `ProductFields`. Детальное описание можно найти в `product_fields_translator.py`.
