# Модуль hypotez/src/webdriver/playwright/__init__.py

## Обзор

Этот модуль предоставляет интерфейс для взаимодействия с веб-драйвером Playwright. Он определяет константу `MODE` и импортирует класс `Playwrid` из модуля `playwrid`.


## Константы

### `MODE`

**Описание**:  Переменная, хранящая режим работы (например, 'dev' или 'prod').

**Значение**: `'dev'` по умолчанию.


## Модули

### `playwrid`

**Описание**:  Модуль, содержащий класс `Playwrid`.


## Классы

### `Playwrid`

**Описание**:  Класс `Playwrid` представляет собой веб-драйвер, позволяющий взаимодействовать с веб-страницами.

**Подробная информация о классе будет добавлена по мере документирования соответствующего модуля `playwrid`.**
