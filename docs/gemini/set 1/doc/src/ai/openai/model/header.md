# Модуль `hypotez/src/logger/header.py`

## Обзор

Модуль `hypotez/src/logger/header.py` определяет корневой путь к проекту `Hypotez`.  Все импорты строятся относительно этого пути.  В будущем этот модуль планируется перенести в системную переменную для более гибкого управления путями.


## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная от текущей директории скрипта и идя вверх по дереву директорий. Останавливается на первой директории, содержащей указанные файлы-маркеры.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, которые используются для определения корневой директории проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.


**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена. Иначе возвращает директорию, в которой расположен скрипт.


**Пример использования**:

```python
root_path = set_project_root()
print(root_path)
```


## Глобальные переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root()`.

**Тип**: `Path`

**Значение**: Путь к корневой директории проекта.

### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `src/settings.json`.

**Тип**: `dict`

**Значение**: Словарь настроек или `None`, если файл `settings.json` не найден или содержит невалидные данные.


### `doc_str`

**Описание**: Строковое содержимое файла README.md.

**Тип**: `str`

**Значение**: Содержимое файла README.md или `None`, если файл не найден или содержит невалидные данные.

### `__project_name__`

**Описание**: Название проекта.

**Тип**: `str`

**Значение**: Значение поля `project_name` из файла `settings.json` или 'hypotez', если это поле отсутствует или файл `settings.json` не найден.

### `__version__`

**Описание**: Версия проекта.

**Тип**: `str`

**Значение**: Значение поля `version` из файла `settings.json` или '', если это поле отсутствует или файл `settings.json` не найден.


### `__doc__`

**Описание**: Строковое описание проекта.

**Тип**: `str`

**Значение**: Содержимое файла README.md или '' если файл не найден или содержимое некорректно.

### `__details__`

**Описание**: Дополнительные детали о проекте.

**Тип**: `str`

**Значение**: Пустая строка.

### `__author__`

**Описание**: Автор проекта.

**Тип**: `str`

**Значение**: Значение поля `author` из файла `settings.json` или '', если это поле отсутствует или файл `settings.json` не найден.


### `__copyright__`

**Описание**: Авторские права.

**Тип**: `str`

**Значение**: Значение поля `copyrihgnt` из файла `settings.json` или '', если это поле отсутствует или файл `settings.json` не найден.


### `__cofee__`

**Описание**: Ссылка для поддержки проекта.

**Тип**: `str`

**Значение**: Значение поля `cofee` из файла `settings.json` или строка с ссылкой на поддержку, если это поле отсутствует или файл `settings.json` не найден.
