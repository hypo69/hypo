# Модуль `src.suppliers.aliexpress._examples.affiliated_products_generator`

## Обзор

Данный модуль содержит пример использования класса `AliAffiliatedProducts` для генерации аффилированных ссылок на товары с AliExpress. В примере демонстрируется создание экземпляра класса с заданными параметрами, обработка списка URL-адресов продуктов и вывод результатов, включая аффилированные ссылки и пути к локальным файлам изображений и видео.
