# Модуль hypotez/src/webdriver/edge/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и получения строки документации из файла `README.MD`. Он также предоставляет переменные с информацией о проекте, такие как имя, версия, описание, автор и др.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с текущей директории. Поиск осуществляется вверх по директориям, пока не будет найдена директория, содержащая один из указанных файлов или директорий.

**Параметры**:

- `marker_files` (tuple): Кортеж строк (имен файлов или директорий), по которым определяется корневая директория проекта. По умолчанию это `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена, иначе - текущая директория, где находится скрипт.

**Вызывает исключения**:

- Нет


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта.

**Тип**: `Path`

### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженные из файла `settings.json`.

**Тип**: `dict`


### `doc_str`

**Описание**: Строка документации, загруженная из файла `README.MD`.

**Тип**: `str`


### `__project_name__`

**Описание**: Имя проекта, полученное из настроек. По умолчанию `hypotez`.

**Тип**: `str`

### `__version__`

**Описание**: Версия проекта, полученная из настроек. По умолчанию пустая строка.

**Тип**: `str`

### `__doc__`

**Описание**: Документация проекта, полученная из файла `README.MD`. По умолчанию пустая строка.

**Тип**: `str`

### `__details__`

**Описание**: Подробности о проекте. По умолчанию пустая строка.

**Тип**: `str`

### `__author__`

**Описание**: Автор проекта, полученный из настроек. По умолчанию пустая строка.

**Тип**: `str`

### `__copyright__`

**Описание**: Авторские права проекта, полученные из настроек. По умолчанию пустая строка.

**Тип**: `str`

### `__cofee__`

**Описание**: Ссылка на страницу сбора средств на чашку кофе для разработчика. По умолчанию ссылка на Boosty.

**Тип**: `str`


## Обработка исключений

### `FileNotFoundError`, `json.JSONDecodeError`

Функции `set_project_root`, и чтение файлов `settings.json`, `README.MD` содержат обработку исключений для ситуаций, когда файлы не найдены или содержат некорректные данные. При возникновении ошибки код проигнорирует ошибку, сохраняя корректность работы кода.
