# Модуль `battle.py`

## Обзор

Модуль `battle.py` реализует игру "Морской бой" между пользователем и компьютером. В игре используется поле 10x10, на котором компьютер размещает 5 кораблей (каждый размером 1 клетка). Игроки по очереди делают выстрелы, пока не пройдет 30 ходов.

### `display_board`
```python
def display_board(board: list[list[int]], is_computer: bool = False) -> None:
    """
    Отображает игровое поле в консоли, скрывая корабли компьютера, если `is_computer` равно `True`.

    Args:
        board (list[list[int]]): Игровое поле для отображения.
        is_computer (bool, optional): Флаг, определяющий, нужно ли скрывать корабли компьютера. По умолчанию False.
    """
```
**Описание**: Отображает игровое поле в консоли. Корабли компьютера (значение 1) скрываются, если `is_computer` равно `True`. Пустые клетки отображаются как "O", попадания как "X", а промахи как "-".

**Параметры**:
- `board` (list[list[int]]): Игровое поле для отображения.
- `is_computer` (bool, optional): Флаг, определяющий, нужно ли скрывать корабли компьютера. По умолчанию `False`.

**Возвращает**:
- `None`: Функция ничего не возвращает, выводит данные в консоль.
