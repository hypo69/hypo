# Модуль `banners_grabber.py`

## Обзор

Модуль `banners_grabber.py` предназначен для сбора баннеров с сайта KSP. Он содержит функцию `get_banners`, которая отвечает за получение баннеров.

## Подробней

Этот модуль является частью проекта `hypotez` и расположен в директории `src/suppliers/ksp`. Это указывает на его роль как поставщика данных для системы, специализирующегося на KSP (возможно, конкретном поставщике или платформе). Функция `get_banners` вероятно, взаимодействует с веб-сайтом KSP для извлечения рекламных баннеров.
