## Маршруты API

**GET /flights**

* **Описание:** Возвращает список доступных рейсов.
* **Параметры запроса:**
    * `origin`: Код аэропорта вылета (например, SVO).
    * `destination`: Код аэропорта прилета (например, JFK).
    * `departure_date`: Дата вылета (YYYY-MM-DD).
    * `return_date`: Дата возвращения (YYYY-MM-DD).
    * `passengers`: Количество пассажиров.
    * `class`: Тип класса (например, economy, business).
    * `page`: Номер страницы (для пагинации).
    * `per_page`: Количество элементов на странице (для пагинации).
* **Возвращаемые данные:**  JSON массив с объектами рейсов.  Каждый объект содержит:
    * `id`: ID рейса.
    * `airline`: Название авиакомпании.
    * `flight_number`: Номер рейса.
    * `departure_airport`: Код аэропорта вылета.
    * `arrival_airport`: Код аэропорта прилета.
    * `departure_time`: Время вылета.
    * `arrival_time`: Время прилета.
    * `price`: Цена.
    * `duration`: Длительность полета.
    * ... (другие нужные поля)
* **Возможные статусы ответа:**
    * 200 OK
    * 400 Bad Request (некорректные параметры запроса)
    * 404 Not Found (рейсы не найдены)
    * 500 Internal Server Error


**POST /booking**

* **Описание:** Создает бронирование.
* **Параметры запроса:**
    * `flight_id`: ID рейса.
    * `passengers`: Массив с данными пассажиров (ФИО, дата рождения и т.д.)
    * `class`: Тип класса.
    * ...(другие необходимые параметры для бронирования)
* **Возвращаемые данные:**  JSON объект с данными бронирования.
* **Возможные статусы ответа:**
    * 201 Created
    * 400 Bad Request (некорректные данные)
    * 404 Not Found (рейс не найден)
    * 409 Conflict (бронирование не создано из-за коллизий)
    * 500 Internal Server Error
