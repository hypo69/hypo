# Модуль `src.endpoints`

## Обзор

Модуль `src.endpoints` предназначен для организации работы с различными API и внешними сервисами. В данный момент он содержит заглушку и объявлена переменная `MODE`.

## Содержание

- [Обзор](#обзор)
- [Переменные](#переменные)

## Переменные

### `MODE`

**Описание**:
- Переменная `MODE` определяет режим работы приложения.

**Значение**:
- `dev`: Режим разработки.
