Этот модуль предоставляет базовые импорты и константы для работы с поставщиком данных GrandAdvance.

- `Graber`: Класс для работы с грабером данных GrandAdvance.


## Константы

### `MODE`

**Описание**: Строковая константа, определяющая режим работы (`dev`).


##  Документация импортированных модулей

### `Graber`

**Описание**: Класс, предоставляющий функционал для извлечения данных из GrandAdvance.  Подробная документация класса находится в файле `graber.py`.


```markdown
# Модуль hypotez/src/suppliers/grandadvance/graber.py

## Обзор

Данный модуль содержит реализацию класса `Graber`, предназначенного для работы с поставщиком данных GrandAdvance.

## Классы

### `Graber`

**Описание**: Класс для работы с поставщиком данных GrandAdvance.

**Методы**:

#### `get_data(self, params: dict) -> dict | None`

**Описание**: Метод извлечения данных из GrandAdvance.

**Параметры**:
- `params` (dict): Словарь параметров для запроса.

**Возвращает**:
- `dict | None`: Словарь с полученными данными или `None`, если запрос не удался.

**Вызывает исключения**:
- `GrandAdvanceError`:  Исключение, которое может быть вызвано при ошибках взаимодействия с сервером GrandAdvance.
```
