# Модуль `Groq`

## Обзор

Модуль `Groq` предназначен для взаимодействия с сервисом Groq через API OpenAI. Он наследует функциональность от класса `OpenaiTemplate` и предоставляет специфические параметры и настройки для работы с моделями Groq.

## Подробнее

Модуль определяет URL, базовые настройки API, флаги работоспособности и необходимости аутентификации, а также список поддерживаемых моделей и их псевдонимов. Это позволяет унифицировать взаимодействие с различными моделями Groq через единый интерфейс.

## Классы

### `Groq`

**Описание**: Класс `Groq` представляет собой адаптер для работы с API Groq, совместимый с OpenAI.

**Наследует**:
- `OpenaiTemplate`: Обеспечивает базовую функциональность для взаимодействия с API OpenAI.

**Атрибуты**:
- `url` (str): URL для доступа к playground Groq (`https://console.groq.com/playground`).
- `login_url` (str): URL для получения ключей API Groq (`https://console.groq.com/keys`).
- `api_base` (str): Базовый URL для API Groq (`https://api.groq.com/openai/v1`).
- `working` (bool): Флаг, указывающий на работоспособность провайдера (значение `True`).
- `needs_auth` (bool): Флаг, указывающий на необходимость аутентификации (значение `True`).
- `default_model` (str): Модель, используемая по умолчанию (`mixtral-8x7b-32768`).
- `fallback_models` (List[str]): Список резервных моделей для использования в случае недоступности основной модели.
- `model_aliases` (Dict[str, str]): Словарь псевдонимов моделей для удобства использования.

## Функции

В данном модуле нет отдельных функций, только атрибуты класса.
