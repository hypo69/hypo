# Модуль hypotez/src/endpoints/kazarinov/header.py

## Обзор

Этот модуль содержит функцию `set_project_root` для определения корневого каталога проекта и инициализирует переменные, представляющие имя проекта, версию, документацию, информацию об авторе и копиирайт, загружая данные из файла `settings.json` и `README.MD`.

## Функции

### `set_project_root`

**Описание**:  Находит корневой каталог проекта, начиная с текущей директории и перемещаясь вверх по дереву каталогов, пока не найдёт директорию, содержащую один из файлов или директорий в списке `marker_files`.

**Параметры**:

- `marker_files` (tuple): Кортеж с именами файлов или директорий, используемых для поиска корневого каталога. По умолчанию: `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Объект `Path` представляющий путь к корневой директории проекта. Если корневая директория не найдена, возвращается текущая директория.


**Вызывает исключения**:

- Не вызываются.


## Переменные

### `MODE`

**Описание**: Строковая переменная, хранящая режим работы приложения. По умолчанию установлено значение `'dev'`.

### `__root__`

**Описание**:  Объект `Path` хранящий путь к корневому каталогу проекта.  Инициализируется вызовом `set_project_root()`.

### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженный из файла `settings.json` в корневой директории проекта.

**Возможные ошибки**:

- `FileNotFoundError`: Если файл `settings.json` не найден.
- `json.JSONDecodeError`: Если содержимое файла `settings.json` не является валидным JSON.

### `doc_str`

**Описание**: Строка содержащая текст из файла `README.MD` в корневой директории проекта.

**Возможные ошибки**:

- `FileNotFoundError`: Если файл `README.MD` не найден.
- `json.JSONDecodeError`: Если содержимое файла `README.MD` не является допустимым форматом текста.

### `__project_name__`

**Описание**: Строка содержащая имя проекта. Загружается из `settings` или имеет значение по умолчанию "hypotez".

### `__version__`

**Описание**: Строка содержащая версию проекта. Загружается из `settings` или имеет значение по умолчанию ''.

### `__doc__`

**Описание**: Строка содержащая документацию проекта. Загружается из `doc_str` или имеет значение по умолчанию ''.

### `__details__`

**Описание**: Строка содержащая детали проекта. Имеет значение по умолчанию ''.

### `__author__`

**Описание**: Строка содержащая имя автора проекта. Загружается из `settings` или имеет значение по умолчанию ''.

### `__copyright__`

**Описание**: Строка содержащая копирайт проекта. Загружается из `settings` или имеет значение по умолчанию ''.

### `__cofee__`

**Описание**: Строка содержащая ссылку для поддержки разработчика. Загружается из `settings` или имеет значение по умолчанию.
