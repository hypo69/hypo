# Модуль _experiments

## Обзор

Модуль содержит экспериментальный код, связанный с поставщиком `ksp`. В основном предназначен для тестирования и отладки функциональности. Содержит импорты различных модулей и определения путей для работы с файловой системой.

## Подробней

Этот модуль, расположенный в директории `src/suppliers/ksp/_experiments`, предоставляет экспериментальную площадку для разработки и проверки новых идей, связанных с поставщиком `ksp`. В нем содержатся различные импорты модулей, используемых в проекте, а также определение путей к корневой директории и директории `src`.

## Функции

### `start_supplier`

```python
def start_supplier(supplier_prefix: str = 'aliexpress', locale: str = 'en'):
    """ Старт поставщика """
    ...
```

**Назначение**: Функция инициализирует и возвращает объект `Supplier` с заданными параметрами.

**Параметры**:

-   `supplier_prefix` (str, optional): Префикс поставщика. По умолчанию `'aliexpress'`.
-   `locale` (str, optional): Локаль поставщика. По умолчанию `'en'`.

**Возвращает**:

-   Объект класса `Supplier`.

**Как работает функция**:

1.  Формирует словарь `params`, содержащий параметры `supplier_prefix` и `locale`.
2.  Создает и возвращает экземпляр класса `Supplier`, передавая словарь `params` в качестве аргументов.

```
start_supplier
|
params = {'supplier_prefix': supplier_prefix, 'locale': locale}
|
Supplier(**params)
```

**Примеры**:

```python
supplier1 = start_supplier()
print(supplier1)  # Выведет объект Supplier с параметрами по умолчанию

supplier2 = start_supplier(supplier_prefix='amazon', locale='de')
print(supplier2)  # Выведет объект Supplier с указанными параметрами
