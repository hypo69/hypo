## Структура конфигурации
    
### `browser_type`

**Описание**: Определяет тип используемого браузера.

**Значение**:
    - `"chromium"`: Используется браузер Chromium.

### `headless`

**Описание**: Определяет, запускать ли браузер в безголовом режиме (без графического интерфейса).

**Значение**:
    - `true`: Браузер будет запущен в безголовом режиме.

### `options`

**Описание**: Список дополнительных параметров командной строки для запуска браузера.

**Значения**:
    - `"--disable-dev-shm-usage"`: Отключает использование `/dev/shm` для снижения использования памяти.
    - `"--no-sandbox"`: Отключает песочницу браузера.
    - `"--disable-gpu"`: Отключает использование GPU для рендеринга.

### `user_agent`

**Описание**: Строка User-Agent, которую браузер будет использовать при запросах.

**Значение**:
    - `"Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/96.0.4664.110 Safari/537.36"`: User-Agent, имитирующий браузер Chrome на Windows.

### `proxy`

**Описание**: Настройки прокси-сервера.

**Поля**:
    - `enabled` (bool): Определяет, использовать ли прокси.
        - `false`: Прокси не используется.
    - `server` (str): URL прокси-сервера.
        - `"http://proxy.example.com:8080"`: Пример URL прокси-сервера.
    - `username` (str): Имя пользователя для авторизации на прокси.
        - `"user"`: Пример имени пользователя.
    - `password` (str): Пароль для авторизации на прокси.
        - `"password"`: Пример пароля.

### `viewport`

**Описание**: Определяет размеры окна просмотра браузера.

**Поля**:
    - `width` (int): Ширина окна просмотра в пикселях.
        - `1280`: Ширина 1280 пикселей.
    - `height` (int): Высота окна просмотра в пикселях.
        - `720`: Высота 720 пикселей.

### `timeout`

**Описание**: Таймаут для операций Playwright в миллисекундах.

**Значение**:
    - `30000`: Таймаут 30 секунд.

### `ignore_https_errors`

**Описание**: Определяет, игнорировать ли ошибки сертификатов HTTPS.

**Значение**:
    - `false`: Ошибки сертификатов HTTPS не игнорируются.
