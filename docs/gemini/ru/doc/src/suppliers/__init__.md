# Модуль `hypotez/src/suppliers/__init__.py`

## Обзор

Модуль `hypotez/src/suppliers` предоставляет базовый класс `Supplier` для работы с поставщиками данных.  Он предназначен для организации взаимодействия с различными источниками данных (например, Amazon, AliExpress), предоставляя структуру для добавления специфичных методов извлечения информации для каждого поставщика.  Классы и функции, относящиеся к конкретному поставщику, хранятся в соответствующих подпапках.
