# Модуль `hypotez/src/webdriver/playwright/__init__.py`

## Обзор

Данный модуль определяет точку входа для подмодуля `playwright`, используемого для управления браузером через Playwright. Он содержит определение режима (`MODE`) и импортирует класс `Playwrid` из файла `playwrid.py`.

## Оглавление

- [Обзор](#обзор)
- [Переменные](#переменные)
- [Импорты](#импорты)

## Переменные

### `MODE`

**Описание**:
Режим работы. В данном случае установлен в `'dev'`.

## Импорты

### `from .playwrid import Playwrid`

**Описание**:
Импортирует класс `Playwrid` из файла `playwrid.py`, делая его доступным для использования в рамках данного модуля.
