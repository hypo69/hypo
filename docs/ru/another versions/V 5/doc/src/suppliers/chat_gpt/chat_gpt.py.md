# Модуль для работы с ChatGpt

## Обзор

Модуль `chat_gpt.py` предназначен для организации работы с ChatGpt, включая чтение HTML-файлов из директории с беседами.

## Подробней

Этот модуль предоставляет класс `ChatGpt`, который содержит методы для обработки HTML-файлов, содержащих данные бесед. В частности, метод `yeld_conversations_htmls` предназначен для поиска и обработки этих файлов.

## Классы

### `ChatGpt`

**Описание**: Класс для работы с ChatGpt.

**Как работает класс**:
Класс `ChatGpt` предназначен для управления и обработки данных, связанных с ChatGpt. В текущей версии класса реализован метод `yeld_conversations_htmls`, который отвечает за поиск HTML-файлов с беседами и их последующую обработку. В дальнейшем класс может быть расширен для включения других функций, таких как отправка запросов к ChatGpt, анализ ответов и сохранение данных.

**Методы**:
- `yeld_conversations_htmls`: Метод для поиска и обработки HTML-файлов с беседами.
