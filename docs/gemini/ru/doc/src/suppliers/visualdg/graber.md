# Модуль visualdg.graber

## Обзор

Модуль `visualdg.graber` содержит класс `Graber`, предназначенный для сбора данных с веб-страниц товаров на сайте `visualdg.co.il`. Класс наследуется от базового класса `Grbr` и предоставляет функции для обработки полей товаров.  Он позволяет переопределять стандартную обработку полей, добавляя специализированные логики для конкретных полей.  Перед выполнением основных операций с веб-драйвером, можно использовать декоратор `close_pop_up` для закрытия всплывающих окон.

## Классы

### `Graber`

**Описание**: Класс `Graber` отвечает за сбор данных с страниц товаров.  Он предоставляет методы для обработки информации с сайта, переопределяя или дополняя базовые функции.

**Атрибуты**:

- `supplier_prefix: str`: Префикс для идентификации поставщика данных.

**Методы**:

- `__init__(self, driver: Driver)`:
    **Описание**: Инициализирует экземпляр класса.
    **Параметры**:
        - `driver (Driver)`: Экземпляр класса драйвера, используемый для взаимодействия с веб-страницей.
    **Возвращает**:
        - `None`
