# Модуль hypotez/src/endpoints/prestashop/__init__.py

## Обзор

Этот модуль предоставляет интерфейсы для работы с API Престашоп. Он импортирует классы, реализующие взаимодействие с различными ресурсами Престашоп, такими как продукты, поставщики, категории, склады, языки, магазины, прайс-листы и клиенты. Модуль определяет константу `MODE`, которая, вероятно, задаёт режим работы API (например, `dev`, `prod`).

## Импортированные классы и функции

- `PrestaShop`: Класс, представляющий основной интерфейс для работы с API Престашоп.
- `PrestaProduct`: Класс для работы с продуктами.
- `PrestaSupplier`: Класс для работы с поставщиками.
- `PrestaCategory`: Класс для работы с категориями.
- `PrestaWarehouse`: Класс для работы со складами.
- `PrestaLanguage`: Класс для работы с языками.
- `PrestaShopShop`: Класс для работы с магазинами Престашоп.
- `PriceListRequester`: Класс для запроса прайс-листов.
- `PrestaCustomer`: Класс для работы с клиентами.

## Константы

### `MODE`

**Описание**: Строковая константа, вероятно, определяющая режим работы API (например, `dev` для разработки, `prod` для производства). Значение по умолчанию: `'dev'`.

## Поддерживаемые типы данных

* Типы данных, возвращаемые некоторыми классами, вероятно, соответствуют структурам данных, используемым в API Престашоп.
