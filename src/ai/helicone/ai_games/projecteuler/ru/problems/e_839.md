# Задача No 839
## Неравенства с треугольниками

## Условия

Рассмотрим треугольник со сторонами $a$, $b$ и $c$, где $a$, $b$ и $c$ являются целыми числами. Известно, что для существования такого треугольника должно выполняться неравенство треугольника, то есть:

*   $a + b > c$
*   $a + c > b$
*   $b + c > a$

Рассмотрим треугольник, для которого выполняется дополнительное условие $a \le b \le c$. Пусть $T(n)$ — количество различных треугольников, удовлетворяющих условиям $a \le b \le c$ и $a+b+c \le n$.

Например:
* $T(10) = 15$. Следующие треугольники удовлетворяют условиям:
    * (1, 1, 1)
    * (1, 2, 2)
    * (1, 3, 3)
    * (1, 4, 4)
    * (2, 2, 2)
    * (2, 3, 3)
    * (2, 4, 4)
    * (3, 3, 3)
    * (3, 3, 4)
    * (2, 3, 4)
    * (1, 2, 3)
    * (1, 3, 5)
    * (2, 2, 3)
    * (1, 4, 5)
    * (2, 4, 5)
* $T(100) = 2151$

Найдите $T(120000)$.
