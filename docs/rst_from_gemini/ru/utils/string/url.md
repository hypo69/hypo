# utils/string/url.py

Этот модуль предоставляет функции для работы с URL-строками, включая извлечение параметров запроса и проверку на валидность URL.

**Описание функций:**

* **`extract_url_params(url: str) -> dict | None`:**
    Извлекает параметры из строки URL.

    **Аргументы:**
    * `url` (str): Строка URL для парсинга.

    **Возвращаемое значение:**
    * `dict | None`: Словарь параметров запроса и их значений или `None`, если URL не содержит параметров.  Возвращает словарь, где ключи - имена параметров, а значения - их значения.  Если параметр встречается несколько раз, его значением будет список значений.

    **Пример использования:**

    ```python
    url = "https://example.com/page?param1=value1&param2=value2&param3="
    params = extract_url_params(url)
    if params:
        print(params)  # Output: {'param1': 'value1', 'param2': 'value2', 'param3': ''}
    ```

* **`is_url(text: str) -> bool`:**
    Проверяет, является ли переданный текст валидным URL с использованием библиотеки `validators`.

    **Аргументы:**
    * `text` (str): Строка для проверки.

    **Возвращаемое значение:**
    * `bool`: `True`, если строка является валидным URL, иначе `False`.

    **Пример использования:**

    ```python
    url1 = "https://www.example.com"
    url2 = "invalid_url"
    print(is_url(url1))  # Output: True
    print(is_url(url2))  # Output: False
    ```

**Обработка ошибок:**

* Функция `extract_url_params` возвращает `None`, если URL не содержит параметров.

**Использование в `if __name__ == "__main__":`:**

Этот блок кода предназначен для демонстрации работы модуля в интерактивном режиме.  Он запрашивает у пользователя URL, проверяет его валидность и, если URL корректный, выводит его параметры.

**Установка:**

Для использования функции `is_url` необходимо установить библиотеку `validators`:

```bash
pip install validators
```

**Рекомендации:**

*  Добавить комментарии для описания обработки случаев, когда параметр встречается несколько раз.
