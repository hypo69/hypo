# Модуль header

## Обзор

Данный модуль содержит функцию `get_project_root`, которая находит корневой каталог проекта, начиная с текущего файла.  Модуль также инициализирует переменные, относящиеся к проекту (название, версия, описание и т.д.), считывая информацию из файла `settings.json` и `README.MD`.

## Оглавление

- [get_project_root](#get_project_root)
- [Переменные проекта](#переменные-проекта)


## Функции

### `get_project_root`

**Описание**: Находит корневой каталог проекта.

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или каталогов, используемых для определения корневого каталога проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневому каталогу проекта, если найден, в противном случае путь к каталогу, в котором расположен текущий скрипт.

**Вызывает исключения**:
- Нет


## Переменные проекта

**Описание**: Переменные, представляющие информацию о проекте.

- `__root__` (Path): Путь к корневому каталогу проекта.  Инициализируется функцией `get_project_root()`.
- `settings` (dict): Словарь настроек проекта, загруженный из файла `settings.json`. Может быть `None`, если файл не найден или некорректный.
- `doc_str` (str):  Содержание файла `README.MD`. Может быть `None`, если файл не найден или некорректный.
- `__project_name__` (str): Название проекта. Получается из `settings`, если доступно, иначе используется значение по умолчанию 'hypotez'.
- `__version__` (str): Версия проекта. Получается из `settings`, если доступно, иначе пустая строка.
- `__doc__` (str): Документация проекта. Получается из `doc_str`, если доступно, иначе пустая строка.
- `__details__` (str): Дополнительные детали о проекте (пустая строка).
- `__author__` (str): Автор проекта. Получается из `settings`, если доступно, иначе пустая строка.
- `__copyright__` (str): Авторские права. Получается из `settings`, если доступно, иначе пустая строка.
- `__cofee__` (str): Ссылка для поддержки разработчика. Получается из `settings`, если доступно, иначе стандартная ссылка.
