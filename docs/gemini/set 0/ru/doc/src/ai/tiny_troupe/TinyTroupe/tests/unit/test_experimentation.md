# Модуль `test_experimentation.py`

## Обзор

Этот модуль содержит тесты для класса `ABRandomizer`, проверяя корректность рандомизации и дерандомизации вариантов A/B-тестирования.

## Функции

### `test_randomize`

**Описание**: Функция проверяет правильность работы метода `randomize` класса `ABRandomizer`. Проверяет случайную выборку вариантов "option1" и "option2" 20 раз,  используя  сгенерированный внутренний список `randomizer.choices`.

**Параметры**:

-  Не принимает параметры.

**Возвращает**:
-  Не возвращает значения.

**Вызывает исключения**:
-  `Exception`: Если для какого-либо элемента не найдена корректная рандомизация.

### `test_derandomize`

**Описание**: Функция тестирует метод `derandomize` класса `ABRandomizer`. Проверяет, что переданные значения a и b (результаты рандомизации) возвращают исходные значения "option1" и "option2".

**Параметры**:

-  Не принимает параметры.

**Возвращает**:
-  Не возвращает значения.

**Вызывает исключения**:
-  Не вызывает исключений.


### `test_derandomize_name`

**Описание**: Функция проверяет метод `derandomize_name` класса `ABRandomizer`, который определяет название варианта (control или treatment) по результатам рандомизации.

**Параметры**:
