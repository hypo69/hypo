# Модуль `login`

## Обзор

Модуль `login.py` предназначен для автоматизации процесса входа в аккаунт на сайте AliExpress с использованием WebDriver. Он содержит функцию `login`, которая принимает объект класса `Supplier` и использует его для взаимодействия с веб-страницей AliExpress через WebDriver.

## Подробней

Этот модуль является частью системы автоматизации для работы с поставщиками на AliExpress. Функция `login` использует предоставленные локаторы для поиска элементов на странице входа, ввода данных и выполнения входа в систему. В случае неуспешного выполнения какого-либо шага предусмотрена логика обработки ошибок (TODO).

```python
def login(s: object) -> bool:

```

**Как работает функция**:
Функция `login` принимает объект `s` класса `Supplier`, который содержит WebDriver и локаторы для элементов на странице AliExpress. Функция выполняет следующие шаги:

1.  Получает WebDriver из объекта `s`.
2.  Получает локаторы для элементов страницы входа из объекта `s`.
3.  Переходит на страницу AliExpress.
4.  Принимает cookies (если это необходимо).
5.  Открывает форму входа.
6.  Вводит email, пароль и нажимает кнопку входа.
7.  Устанавливает язык, валюту и страну доставки (если это необходимо).

**Параметры**:

*   `s` (object): Объект класса `Supplier`, содержащий WebDriver и локаторы.

**Возвращает**:

*   `bool`: `True`, если вход выполнен успешно (в текущей версии всегда `True` для отладки).
