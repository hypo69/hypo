# Задача No 438
## Сумма делений на рекурсивную последовательность

## Условия
Пусть $S(n)$ обозначает множество всех натуральных чисел, которые не превосходят $n$, чье представление в десятичной системе счисления состоит только из цифр 1 и 2.

Пусть $F(n)$ — сумма чисел из $S(n)$.

Пусть $G(n)$ — сумма $k$ при всех $k$ из $S(n)$, где $k$ делит $F(n)$.

Например:
$S(10) = \{1, 2\}$.
$F(10) = 1 + 2 = 3$.
$G(10) = 1 + 3 = 4$, поскольку только 1 и 3 делят 3.

$S(25) = \{1, 2, 11, 12, 21, 22\}$.
$F(25) = 1 + 2 + 11 + 12 + 21 + 22 = 69$.
Множество делителей 69 в S(25) равно {1, 3, 23}.  Следовательно, $G(25) = 1$  (только 1 делит 69).
Заметим, что 3 и 23 не являются членами $S(25)$.

Найдите $\sum_{n=1}^{30} G(10^n)$.
