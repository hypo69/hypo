Модуль `src.webdriver.javascript` содержит функции для работы с JavaScript в драйвере.


## Константы

* **MODE = 'debug'**: Режим работы модуля (в данном случае отладочный).

## Функции

* **`get_project_root(marker_files=('pyproject.toml', 'requirements.txt', '.git')) -> Path`**:
    Находит корневой каталог проекта, начиная с текущего каталога и поднимаясь вверх по иерархии.
    Останавливается на первом каталоге, содержащем один из файлов или папок в списке `marker_files`.

    **Аргументы:**
    * `marker_files`: Кортеж строк, представляющих имена файлов или каталогов, которые используются для поиска корневого каталога проекта.  По умолчанию: `('pyproject.toml', 'requirements.txt', '.git')`.

    **Возвращаемое значение:**
    * `Path`: Путь к корневому каталогу проекта, если найден, иначе путь к каталогу текущего файла.

    **Описание:** Функция рекурсивно проверяет родительские каталоги текущего файла, пока не найдёт каталог, содержащий один из указанных маркеров.  Если корневой каталог не найден в `sys.path`, он добавляется в `sys.path` для корректной импортации.


## Переменные

* **`__root__ (Path)`**:  Путь к корневому каталогу проекта, полученный с помощью функции `get_project_root()`.
