# Документация для jquery-3.5.1.slim.min.js

## Обзор

Данный файл содержит сжатый и минифицированный JavaScript код библиотеки jQuery версии 3.5.1.  Он предоставляет обширные возможности для работы с DOM, событиями, анимацией и AJAX-запросами.  Для получения полной документации рекомендуется обратиться к официальному сайту jQuery.

## Функции

### `jQuery.extend`

**Описание**: Расширяет существующий объект, добавляя или изменяя свойства из последующих объектов.

**Параметры**:

- `a` (объект): Объект, который будет расширен.
- `arguments[s]` (объект, optional, s>1): Объекты, свойства которых будут добавлены к `a`.
- `l` (bool, optional): Флаг, указывающий, следует ли использовать глубокое копирование (true), или только поверхностное (false). По умолчанию false.

**Возвращает**:
- `a`: Расширенный объект.

**Вызывает исключения**:
- Нет.

### `jQuery.each`

**Описание**: Циклически проходит по элементам массива или объекта, вызывая функцию `t` для каждого элемента.

**Параметры**:

- `e` (массив/объект): Массив или объект, по которому будет происходить итерация.
- `t` (функция): Функция, которая будет вызвана для каждого элемента. Возвращает `false` для прерывания цикла.


**Возвращает**:
- `e`: Массив или объект (без изменений).

**Вызывает исключения**:
- Нет.

### `jQuery.grep`

**Описание**: Фильтрует массив, возвращая новый массив, содержащий только элементы, удовлетворяющие условию `t`.

**Параметры**:

- `e` (массив): Массив, который нужно отфильтровать.
- `t` (функция): Функция, которая возвращает `true`, если элемент должен быть включен в новый массив, `false` иначе.
- `n` (bool, optional):  Если `true`, возвращает элементы, которые не удовлетворяют условию; если `false` (по умолчанию) — удовлетворяющие условию.

**Возвращает**:
- `r` (массив): Новый массив отфильтрованных элементов.

**Вызывает исключения**:
- Нет.

### `jQuery.map`

**Описание**: Применяет функцию к каждому элементу массива или объекта, создавая новый массив.

**Параметры**:

- `e` (массив/объект): Массив или объект, по которому будет происходить итерация.
- `t` (функция): Функция, которая будет применена к каждому элементу.  Должна возвращать значение для нового массива или `null`/`undefined` для пропуска.
- `n` (дополнительный аргумент, optional): Дополнительный аргумент, передаваемый в функцию `t`.


**Возвращает**:
- `r` (массив): Новый массив, полученный путем применения функции.

**Вызывает исключения**:
- Нет.


### `jQuery.parseHTML`

**Описание**: Парсит строку HTML и возвращает массив узлов DOM.


**Параметры**:

- `e` (строка): Строка HTML для парсинга.
- `t` (элемент DOM, optional):  Элемент документа, в котором будут созданы узлы.
- `n` (bool, optional):  Флаг. Если `true`, то обрабатываются теги `<script>` (считываются содержимое тега, но не выполняется).


**Возвращает**:
- Массив элементов DOM или пустой массив, если входная строка пустая или недопустима.

**Вызывает исключения**:
- Нет.
