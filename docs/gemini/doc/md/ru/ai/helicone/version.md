```markdown
# Модуль version.py

## Обзор

Этот модуль содержит константы, определяющие версию проекта `hypotez` и другие метаданные.  Данные берутся из файла `settings.json` в корне проекта, если он существует и корректно сформирован. В противном случае используется значения по умолчанию.

## Оглавление

- [Версия](#версия)
- [Название проекта](#название-проекта)
- [Документация](#документация)
- [Подробности](#подробности)
- [Автор](#автор)
- [Авторские права](#авторские-права)
- [Кофе](#кофе)
- [Настройки](#настройки)


## Настройки

### `settings`

**Описание**: Словарь, содержащий настройки проекта, считываемые из файла `settings.json`.

**Тип**: `dict`

**Значение по умолчанию**: `None`

**Обработка ошибок**: Если файл `settings.json` не найден или содержит некорректный JSON, то переменная `settings` сохранит значение `None`, и будут использоваться значения по умолчанию.

## Версия

### `__version__`

**Описание**: Строковое представление версии проекта.

**Тип**: `str`


## Название проекта

### `__project_name__`

**Описание**: Строковое представление названия проекта.

**Тип**: `str`

**Значение по умолчанию**: `hypotez`


## Документация

### `__doc__`

**Описание**: Строковое представление документации проекта.


**Тип**: `str`


## Подробности

### `__details__`

**Описание**: Строковое представление дополнительных подробностей о проекте.


**Тип**: `str`


## Автор

### `__author__`

**Описание**: Строковое представление имени автора проекта.

**Тип**: `str`

**Значение по умолчанию**: пустая строка


## Авторские права

### `__copyright__`

**Описание**: Строковое представление авторских прав проекта.

**Тип**: `str`

**Значение по умолчанию**: пустая строка


## Кофе

### `__cofee__`

**Описание**: Строковое представление ссылки для поддержки разработчика.

**Тип**: `str`

**Значение по умолчанию**:  Ссылка на Boosty.to для поддержки разработчика.


##  Функции

### `load_settings`

**Описание**:  Функция загрузки настроек из файла settings.json.

**Параметры**:

- Не принимает параметров.

**Возвращает**:

- `dict | None`: Возвращает загруженные настройки из файла, либо `None` в случае ошибок.

**Вызывает исключения**:
- `FileNotFoundError`: Если файл `settings.json` не найден.
- `json.JSONDecodeError`: Если файл `settings.json` содержит некорректный JSON.
```
