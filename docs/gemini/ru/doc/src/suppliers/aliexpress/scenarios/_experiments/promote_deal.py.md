# Модуль `promote_deal.py`

## Обзор

Модуль `promote_deal.py` является частью проекта `hypotez` и предназначен для создания рекламной кампании на AliExpress. Он использует модуль `AliPromoDeal` для получения информации о продуктах и их продвижения.

## Подробней

Модуль предназначен для автоматизации процесса создания рекламных кампаний на AliExpress. Он извлекает информацию о продуктах и использует ее для продвижения товаров. В текущей версии код не выполняет никаких действий, кроме импорта необходимых модулей и создания экземпляра класса `AliPromoDeal`, а также получения списка деталей о продуктах.

## Функции

### `AliPromoDeal`

```python
deal = AliPromoDeal('150624_baseus_deals')
```

**Назначение**: Создает экземпляр класса `AliPromoDeal` для работы с продуктами, участвующими в рекламной кампании.

**Параметры**:
- `deal_name` (str): Имя рекламной кампании (`'150624_baseus_deals'` в данном случае).

**Как работает функция**:
1. Создается экземпляр класса `AliPromoDeal` с указанным именем кампании.

### `get_all_products_details`

```python
products = deal.get_all_products_details()
```

**Назначение**: Функция получает детали всех продуктов.

**Параметры**:
Функция не принимает параметров

**Как работает функция**:
1. Функция `get_all_products_details` извлекает детальную информацию обо всех продуктах.
