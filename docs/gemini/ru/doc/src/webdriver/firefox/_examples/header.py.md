# Модуль `header.py`

## Обзор

Данный модуль определяет константу `MODE` и устанавливает корень проекта в системный путь.

## Оглавление

1. [Константы](#константы)
2. [Импорты](#импорты)

## Константы

### `MODE`
**Описание**: Определяет режим работы приложения. По умолчанию установлено значение `'dev'`.

## Импорты

### `sys`
**Описание**: Модуль `sys` используется для доступа к системным переменным и функциям, включая `sys.path`, который используется для добавления пути к корню проекта.

### `os`
**Описание**: Модуль `os` используется для работы с операционной системой, в частности для получения текущей директории и манипуляций с путями.

### `Path` из `pathlib`
**Описание**: Класс `Path` из модуля `pathlib` используется для представления файловых путей в объектно-ориентированном стиле.

### Установка корня проекта
**Описание**: Данный блок кода добавляет корень проекта в `sys.path`, позволяя импортировать модули из других директорий проекта.
