# Модуль hypotez/src/webdriver

## Обзор

Этот модуль предоставляет интерфейс для управления веб-драйверами, используемыми для автоматизации задач веб-скрапинга и взаимодействия с веб-сайтами.  Модуль предоставляет абстракцию над различными типами веб-драйверов, включая ChromeDriver, FirefoxDriver, EdgeDriver,  и другие.
