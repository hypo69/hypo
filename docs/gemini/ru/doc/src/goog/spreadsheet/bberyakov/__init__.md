# Модуль hypotez/src/goog/spreadsheet/bberyakov

## Обзор

Этот модуль предоставляет инструменты для работы с Google Spreadsheets. Он содержит классы для взаимодействия с рабочими листами и возможностями рендеринга.

## Константы

### `MODE`

**Описание**:  Переменная, хранящая текущий режим работы (например, 'dev' или 'prod').

**Значение**: `'dev'`


## Классы

### `GSpreadsheet`

**Описание**: Класс для взаимодействия с Google Spreadsheets.


### `GWorksheet`

**Описание**: Класс для работы с отдельными рабочими листами в Google Spreadsheets.


### `GSRenderr`

**Описание**: Класс для рендеринга данных из Google Spreadsheets.
