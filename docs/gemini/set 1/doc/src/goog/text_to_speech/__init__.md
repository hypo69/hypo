# Модуль `hypotez/src/goog/text_to_speech/__init__.py`

## Обзор

Этот модуль предоставляет интерфейс для работы с Google Text-to-Speech.  Он использует библиотеки `pyttsx3` и `gtts` для синтеза речи.

## Константы

### `MODE`

**Описание**:  Устанавливает режим работы модуля. В данном случае он имеет значение `'dev'`.

## Классы

### `TTS`

**Описание**: Класс, реализующий функциональность для Google Text-to-Speech.

**Методы**

- `__init__(*args, **kwards)`: Инициализирует экземпляр класса `TTS`.

**Параметры:**
- `*args`: Аргументы.
- `**kwards`: Именованные аргументы.

**Описание метода `__init__`:**

**Описание**: Инициализирует объект `pyttsx3.init` и выводит список доступных голосов.

```python
def __init__(self,*args,**kwards):
    """
    Инициализирует объект pyttsx3.init и выводит список доступных голосов.

    Args:
        *args: Аргументы.
        **kwards: Именованные аргументы.
    """
    tts = pyttsx3.init()
    voices = tts.getProperty('voices')
    for v in voices:
        print(v)
```
