Facebook Module
===============

.. module:: hypotez.src.endpoints.advertisement.facebook.facebook
   :platform: Windows, Unix
   :synopsis: Facebook advertisement module

   This module provides functionality for interacting with Facebook through a web driver.  It includes scenarios for login, posting messages, uploading media, and more.


Scenarios
--------
- login: Facebook login.
- post_message: Posting a text message into a form.
- upload_media: Uploading a file or a list of files.
