# Модуль hypotez/src/utils/convertors/ns.py

## Обзор

Модуль `ns.py` предоставляет функции для конвертации объектов `SimpleNamespace` в различные форматы: словарь, JSON, CSV, XML и XLS.

## Функции

### `ns2dict`

**Описание**: Преобразует объект `SimpleNamespace` в словарь. Рекурсивно обрабатывает вложенные структуры.

**Параметры**:

- `ns_obj` (SimpleNamespace): Объект `SimpleNamespace` для преобразования.

**Возвращает**:

- `Dict[str, Any]`: Преобразованный словарь с обработкой вложенных структур.

### `ns2json`

**Описание**:  Преобразует объект `SimpleNamespace` в формат JSON.  Не представлена в коде, но предполагается, что использует `ns2dict` для промежуточного преобразования.

**Параметры**:

- `ns_obj` (SimpleNamespace): Объект `SimpleNamespace` для преобразования.

**Возвращает**:

- `str`: Строка в формате JSON.


### `ns2csv`

**Описание**: Преобразует объект `SimpleNamespace` в формат CSV.

**Параметры**:

- `ns_obj` (SimpleNamespace): Объект `SimpleNamespace` для преобразования.
- `csv_file_path` (str | Path): Путь к файлу для сохранения CSV.


**Возвращает**:

- `bool`: `True`, если преобразование прошло успешно, `False` в противном случае.

**Обрабатывает исключения**:

- `Exception`:  Возникает при ошибках во время работы с CSV, например, при проблемах с файловой системой.


### `ns2xml`

**Описание**: Преобразует объект `SimpleNamespace` в формат XML.

**Параметры**:

- `ns_obj` (SimpleNamespace): Объект `SimpleNamespace` для преобразования.
- `root_tag` (str, опционально):  Имя корневого тега XML. По умолчанию "root".

**Возвращает**:

- `str`: Результирующая строка XML.


**Обрабатывает исключения**:

- `Exception`: Возникает при ошибках во время работы с XML.


### `ns2xls`

**Описание**: Преобразует объект `SimpleNamespace` в формат XLS (Excel).

**Параметры**:

- `data` (SimpleNamespace): Объект `SimpleNamespace` для преобразования.
- `xls_file_path` (str | Path): Путь к файлу для сохранения XLS.

**Возвращает**:

- `bool`: `True`, если преобразование прошло успешно, `False` в противном случае.

**Обрабатывает исключения**:

- `Exception`: Возникает при ошибках во время работы с Excel.


## Вспомогательные функции (из импортированных модулей)

Модуль использует функции из модулей `src.utils.csv`, `src.utils.xls`, `xml2dict` и `logger`.  Эти функции не документированы в данном файле, но предполагается, что они отвечают за сохранение CSV и XLS файлов, а также за работу с XML.

## Модульные переменные


**MODE**:
Переменная, хранящая текущий режим работы (например, 'dev' или 'prod').
