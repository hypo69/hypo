# SECURITY.md

## Обзор

Данный файл содержит политику безопасности для репозитория Microsoft. Он описывает процедуру сообщения о найденных уязвимостях и предоставляет контакты для связи.

## Сообщения о проблемах безопасности

**Не следует сообщать о проблемах безопасности через общедоступные GitHub-вопросы.**

Вместо этого отправляйте сообщения в Microsoft Security Response Center (MSRC) по адресу [https://msrc.microsoft.com/create-report](https://aka.ms/security.md/msrc/create-report).

Также можно отправить сообщение по электронной почте на адрес [secure@microsoft.com](mailto:secure@microsoft.com).  По возможности, зашифруйте сообщение с помощью нашего ключа PGP; его можно загрузить на странице [Страница ключа PGP Microsoft Security Response Center](https://aka.ms/security.md/msrc/pgp).

Вы должны получить ответ в течение 24 часов. Если по какой-либо причине этого не произойдет, свяжитесь с нами повторно по электронной почте, чтобы убедиться, что мы получили ваше первоначальное сообщение. Дополнительная информация доступна на сайте [microsoft.com/msrc](https://www.microsoft.com/msrc).

В сообщении о проблеме безопасности необходимо указать следующую информацию (по возможности):

* Тип проблемы (например, переполнение буфера, SQL-инъекция, межсайтовый скриптинг и т. д.)
* Полные пути к исходным файлам, связанным с проблемой
* Местоположение затронутого исходного кода (тег/ветвь/коммит или прямой URL)
* Любые специальные конфигурации, необходимые для воспроизведения проблемы
* Пошаговые инструкции по воспроизведению проблемы
* Прототип или код эксплоита (если это возможно)
* Воздействие проблемы, включая то, как злоумышленник может воспользоваться проблемой

Эта информация поможет нам быстрее обработать ваш отчет.

Если вы сообщаете об ошибке в рамках программы вознаграждений за обнаружение уязвимостей, более подробные отчеты могут способствовать получению более высокой награды. Подробнее о наших текущих программах вознаграждений вы можете узнать на странице [Программа вознаграждений Microsoft за обнаружение уязвимостей](https://aka.ms/security.md/msrc/bounty).

## Предпочтительные языки
