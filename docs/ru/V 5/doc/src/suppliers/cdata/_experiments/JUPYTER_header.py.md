# Модуль _experiments

## Обзор

Модуль `_experiments` предназначен для экспериментов, связанных с поставщиками данных (cdata). В данный момент он включает в себя настройку путей к директориям проекта, импорт необходимых модулей и классов, а также функцию для запуска поставщика. Модуль подготавливает окружение для экспериментов, связанных с парсингом и обработкой данных о товарах от различных поставщиков, таких как AliExpress.

## Подробнее

Этот модуль является частью подсистемы поставщиков (`suppliers`) и используется для проведения экспериментов с кодом, который взаимодействует с различными источниками данных. В частности, он содержит функцию `start_supplier`, которая создает и возвращает объект поставщика (`Supplier`). Это позволяет инициализировать процесс получения данных о товарах для последующей обработки и загрузки в систему, например, PrestaShop.

## Функции

### `start_supplier`

```python
def start_supplier(supplier_prefix: str = 'aliexpress', locale: str = 'en') -> Supplier:
    """ Старт поставщика """
    params: dict = {
        'supplier_prefix': supplier_prefix,
        'locale': locale
    }
    return Supplier(**params)
```

**Как работает функция**:

Функция `start_supplier` создает экземпляр класса `Supplier` с заданными параметрами. Она принимает префикс поставщика (`supplier_prefix`) и локаль (`locale`) в качестве аргументов. Эти параметры передаются в конструктор класса `Supplier` для инициализации объекта поставщика. Внутри функции формируется словарь `params`, который содержит переданные аргументы, а затем этот словарь распаковывается (`**params`) при создании экземпляра `Supplier`.

**Параметры**:

- `supplier_prefix` (str, optional): Префикс поставщика, используемый для идентификации поставщика. По умолчанию `'aliexpress'`.
- `locale` (str, optional): Локаль, используемая для поставщика. По умолчанию `'en'`.

**Возвращает**:

- `Supplier`: Объект поставщика, созданный с заданными параметрами.

**Примеры**:

```python
supplier = start_supplier(supplier_prefix='aliexpress', locale='ru')
# Создает объект поставщика AliExpress с русской локалью.
```

```python
supplier = start_supplier(supplier_prefix='amazon', locale='en')
# Создает объект поставщика Amazon с английской локалью.
```
```python
supplier = start_supplier()
#Создает объект поставщика aliexpress с английской локалью.
```
