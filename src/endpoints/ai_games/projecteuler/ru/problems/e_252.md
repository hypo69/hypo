# Задача No 252
## Concordant Tangents

## Условия

Заданы *n* точек (xᵢ, yᵢ) на плоскости.

Точка (xⱼ, yⱼ) называется *видимой* из точки (xᵢ, yᵢ), если прямая линия, соединяющая (xᵢ, yᵢ) и (xⱼ, yⱼ), не пересекается ни с одним из отрезков, соединяющих любые другие пары точек из набора *n*. Другими словами, точка j видна из точки i, если ни одна другая точка не находится на прямой, соединяющей i и j, и ни один отрезок, соединяющий другие точки, не перекрывает этот отрезок.

Например, из 4-х точек A(1,1), B(2,4), C(3,1) и D(6,2) точка D видима из A, но C невидима из A.

Обозначим V(p) - число видимых точек из p.

Для набора точек, полученных по следующему правилу:

s₀ = 290797
sₙ₊₁ = sₙ² mod 50515093

xₙ = s₂ₙ mod 2000 - 1000
yₙ = s₂ₙ₊₁ mod 2000 - 1000

Найдите максимальное значение V(p) для первых 500 точек.
