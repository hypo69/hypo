## Оглавление

- [ProductType](#producttype)

## Классы

### `ProductType`

**Описание**: Класс `ProductType` определяет типы продуктов на AliExpress.

**Константы**:

- `ALL`: Представляет все типы продуктов.
- `PLAZA`: Представляет тип продуктов с платформы PLAZA.
- `TMALL`: Представляет тип продуктов с платформы TMALL.


### `SortBy`

**Описание**: Класс `SortBy` определяет параметры сортировки для запросов.

**Константы**:

- `SALE_PRICE_ASC`: Сортировка по цене продажи в порядке возрастания.
- `SALE_PRICE_DESC`: Сортировка по цене продажи в порядке убывания.
- `LAST_VOLUME_ASC`: Сортировка по последнему объему продаж в порядке возрастания.
- `LAST_VOLUME_DESC`: Сортировка по последнему объему продаж в порядке убывания.


### `LinkType`

**Описание**: Класс `LinkType` определяет типы ссылок.

**Константы**:

- `NORMAL`: Обычный тип ссылки (значение 0).
- `HOTLINK`: Тип ссылки "горячая ссылка" (значение 2).
