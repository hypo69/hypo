# Модуль hypotez/src/endpoints/hypo69/small_talk_bot/header.py

## Обзор

Данный модуль содержит вспомогательные функции для работы с настройками проекта, определяет корневую директорию проекта и загружает информацию о проекте из файлов `settings.json` и `README.MD`.  Модуль также предоставляет доступ к глобальным переменным, содержащим информацию о проекте (имя, версия, описание, автор, копирайт и ссылку на пожертвования).

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная от текущего файла и идя вверх по директориям до тех пор, пока не найдет директорию, содержащую указанные файлы-маркеры (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, которые используются для определения корневой директории проекта. По умолчанию - (`'pyproject.toml'`, `'requirements.txt'`, `.git`).


**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена, иначе путь к текущей директории.

**Вызывает исключения**:

- Нет.


### Глобальные переменные

**Описание**: Модуль определяет ряд глобальных переменных, описывающих проект:


- `__root__` (Path): Путь к корневой директории проекта.
- `settings` (dict): Словарь, содержащий настройки проекта, загруженный из `settings.json`.
- `doc_str` (str): Строка с текстом из файла `README.MD`.
- `__project_name__` (str): Имя проекта, извлечённое из `settings.json`, по умолчанию 'hypotez'.
- `__version__` (str): Версия проекта, извлечённая из `settings.json`, по умолчанию пустая строка.
- `__doc__` (str): Текст документации проекта из `README.MD`, по умолчанию пустая строка.
- `__details__` (str): Дополнительная информация о проекте, по умолчанию пустая строка.
- `__author__` (str): Автор проекта, извлечённый из `settings.json`, по умолчанию пустая строка.
- `__copyright__` (str): Авторские права, извлечённые из `settings.json`, по умолчанию пустая строка.
- `__cofee__` (str): Ссылка для пожертвований, извлечённая из `settings.json`, по умолчанию ссылка на Boosty.

**Обработка исключений:**

При попытке чтения из `settings.json` или `README.MD` обрабатываются исключения `FileNotFoundError` и `json.JSONDecodeError`.  Если файл не найден или содержимое некорректно, соответствующие переменные устанавливаются в значения по умолчанию.
