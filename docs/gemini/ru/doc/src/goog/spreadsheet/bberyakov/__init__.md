# Модуль hypotez/src/goog/spreadsheet/bberyakov

## Обзор

Данный модуль предоставляет инструменты для работы с Google Spreadsheets.  Он содержит классы для взаимодействия с отдельными листами (GWorksheet) и всей электронной таблицей (GSpreadsheet), а также инструменты для рендеринга (GSRenderr).

## Содержание

### Модуль `gspreadsheet`

Класс `GSpreadsheet`: Представляет собой интерфейс для работы с Google Spreadsheets.  Не содержит прямых функций для работы с данными, зато предоставляет методы для доступа к отдельным листам.


### Модуль `gworksheets`

Класс `GWorksheet`: Представляет собой интерфейс для работы с отдельным листом Google Spreadsheets.  Обеспечивает методы для взаимодействия с данными листа.


### Модуль `grender`

Класс `GSRenderr`: Представляет инструменты для рендеринга данных Google Spreadsheets в различные форматы.  Подробности реализации и функциональности требуют дополнительной документации.


## Константы

### `MODE`

**Значение:**  'dev'

**Описание:**  Устанавливает режим работы (в данном случае - 'dev').  Более подробное описание режима должно быть доступно в дополнительных документах.
