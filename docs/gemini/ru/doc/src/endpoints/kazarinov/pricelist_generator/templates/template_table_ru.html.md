# Шаблон таблицы прайс-листа (Русский)

## Обзор

Этот HTML-шаблон используется для генерации прайс-листа с информацией о продуктах, включая изображения, описание и цену. Он предназначен для отображения списка товаров с подробной информацией о каждом из них.

## Заголовок

В начале HTML-документа определен заголовок, который включает в себя следующие метатеги:
- `charset="UTF-8"`: устанавливает кодировку символов UTF-8.
- `name="viewport"`: настраивает область просмотра для адаптивного отображения на различных устройствах.
- `<title>{{ title }}</title>`: устанавливает заголовок страницы, динамически заполняемый значением переменной `title`.

## Стили

В разделе `<style>` определены CSS-стили для оформления документа:

- `body`: общие стили для тела документа, включая фон, цвет текста и шрифт.
- `h1`: стили для заголовка первого уровня, выравнивание по центру, размер шрифта и отступы.
- `.lead`: стили для вводного текста, выравнивание по центру и размер шрифта.
- `.container`: стили для основного контейнера, определяющие ширину и отступы.
- `table`: стили для таблицы, определяющие ширину и направление текста.
- `.product-card`: стили для карточек продуктов, включая фон, рамку, отступы и поведение при разрыве страницы.
- `.product-card h3`: стили для заголовка продукта, убирающие верхний отступ.
- `.product-card img`: стили для изображений продуктов, включая размеры, пропорции, фон и отступы.
- `.product-info`: стили для текстовой информации о продукте, включая flexbox для заполнения оставшегося места.
- `.price-tag`: стили для отображения цены, включая цвет фона, цвет текста, отступы и скругление углов.
- `.footer`: стили для подвала страницы, включая выравнивание по центру и верхнюю рамку.

## Контейнер

Основной контент размещается внутри элемента `div` с классом `container`. Этот элемент центрирует контент на странице и ограничивает его ширину.
Внутри контейнера располагаются следующие элементы:
- Заголовок `h1` с динамическим содержимым `{{ title }}`.
- Параграф с вводным текстом `p` с классом `lead` и динамическим содержимым `{{ description }}`.
- Таблица `table` для отображения списка товаров.

## Продукты

Каждый продукт представлен в виде строки таблицы `tr` с классом `product-card`.
Внутри каждой строки:
- Заголовок третьего уровня `h3` с названием товара `{{ product.product_title }}`.
- Контейнер `div` с классом `product-card-content`, содержащий:
    - Изображение `img` продукта, с атрибутами `src` (путь к файлу изображения) и `alt` (альтернативный текст).
    - Контейнер `div` с классом `product-info`, содержащий:
        - Описание продукта `p` с содержимым `{{ product.product_description }}`.
        - Спецификация продукта `p` с содержимым `{{ product.specification }}`.
Используется цикл `{% for product in products %}` для итерации по списку продуктов и генерации соответствующего HTML для каждого товара.

## Цена и подвал

В нижней части контейнера находится подвал `div` с классом `footer`, который отображает общую цену, если она указана.
- Цена отображается в элементе `p` с классом `price-tag` и динамическим содержимым `{{ price }}` и `{{ currency }}`.
- Условный оператор `{% if price %}` проверяет наличие цены перед отображением.

## Медиа для печати

Раздел `@media print` содержит стили для печатной версии документа:

- `.product-card`: устанавливает разрыв страницы после каждой карточки продукта, что обеспечивает печать каждого продукта на отдельной странице.
- `.product-card:last-child`: отменяет разрыв страницы для последней карточки, что позволяет избежать пустой страницы после печати.
