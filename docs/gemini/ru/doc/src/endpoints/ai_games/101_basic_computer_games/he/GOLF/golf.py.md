# משחק גולף

## Обзор

קוד זה מיישם משחק גולף פשוט, בו המשתמש מנסה להגיע לחור עם מספר מינימלי של חבטות. המשחק מדמה מרחק לחור, ומחשב את המרחק שהכדור עובר בפועל עם סטייה אקראית.

## Оглавление

- [Обзор](#обзор)
- [Функции](#функции)
- [Переменные](#переменные)

## Функции

### `main`

**Описание**: פונקציה ראשית של המשחק.

**Параметры**:
- אין פרמטרים.

**Возвращает**:
- `None`

**Вызывает исключения**:
- `ValueError`: כאשר המשתמש מכניס קלט שאינו מספר שלם.

## Переменные

### `distanceToHole`

**Описание**: המרחק לחור, מאותחל ל-400.

**Тип**: `int`

### `numberOfHits`

**Описание**: מונה החבטות, מאותחל ל-0.

**Тип**: `int`

### `hitDistance`

**Описание**: מרחק החבטה שהמשתמש מכניס.

**Тип**: `int`

### `realDistance`

**Описание**: מרחק החבטה האמיתי, כולל סטייה אקראית.

**Тип**: `int`
