# Модуль `hypotez/src/category/category.py`

## Обзор

Данный модуль предоставляет классы и функции для работы с категориями товаров, в основном ориентированные на интеграцию с PrestaShop.  Модуль обеспечивает возможность асинхронного и синхронного обхода категорий, сбора информации о них и создания иерархического словаря категорий.


## Классы

### `Category`

**Описание**: Класс `Category` представляет собой категорию товара и наследуется от класса `PrestaCategory`. Он предоставляет методы для работы с родительскими категориями и рекурсивного обхода категорий.

**Атрибуты**:
- `credentials`: Словарь с учетными данными для доступа к API.

**Методы**:

#### `get_parents(self, id_category, dept)`

**Описание**: Получение списка родительских категорий для заданной категории.

**Параметры**:
- `id_category` (int): Идентификатор категории.
- `dept` (int): Глубина.

**Возвращает**:
- `List[int]`: Список идентификаторов родительских категорий.


#### `crawl_categories_async(self, url, depth, driver, locator, dump_file, id_category_default, category=None)`

**Описание**: Асинхронная рекурсивная функция для обхода категорий. Построение иерархического словаря категорий.

**Параметры**:
- `url` (str): URL страницы для обхода.
- `depth` (int): Глубина рекурсии.
- `driver` (webdriver): Экземпляр Selenium webdriver.
- `locator` (dict): Xpath локатор для поиска ссылок на категории.
- `dump_file` (Path): Путь к файлу для записи иерархического словаря.
- `id_category_default` (int): Идентификатор категории по умолчанию.
- `category` (dict, optional): Словарь, представляющий категорию, по умолчанию `None`.

**Возвращает**:
- `dict`: Иерархический словарь, представляющий категории и их URL.


#### `crawl_categories(self, url, depth, driver, locator, dump_file, id_category_default, category={})`

**Описание**: Рекурсивная функция для обхода категорий. Построение иерархического словаря категорий.

**Параметры**:
- `url` (str): URL страницы для обхода.
- `depth` (int): Глубина рекурсии.
- `driver` (webdriver): Экземпляр Selenium webdriver.
- `locator` (dict): Xpath локатор для поиска ссылок на категории.
- `dump_file` (Path): Путь к файлу для записи иерархического словаря.
- `id_category_default` (int): Идентификатор категории по умолчанию.
- `category` (dict, optional): Словарь, представляющий категорию, по умолчанию пустой словарь.

**Возвращает**:
- `dict`: Иерархический словарь, представляющий категории и их URL.


## Функции

### `check_duplicate_url(dictionary, url)`

**Описание**: Проверяет, существует ли URL в иерархическом словаре.

**Параметры**:
- `dictionary` (dict): Иерархический словарь для проверки.
- `url` (str): URL для проверки.

**Возвращает**:
- `bool`: `True`, если URL уже существует, иначе `False`.


### `compare_and_print_new_keys(current_dict, file_path)`

**Описание**: Сравнивает текущий словарь с данными из файла и выводит ключи, которые отсутствуют в текущем словаре.

**Параметры**:
- `current_dict` (dict): Текущий словарь для сравнения.
- `file_path` (str): Путь к файлу с данными для сравнения.

**Возвращает**:
- `None`.  (Функция выводит результат на консоль.)
