# Модуль hypotez/src/suppliers/aliexpress/gui/version.py

Этот модуль содержит константы и метаданные для графического интерфейса (GUI) поставщика AliExpress. В частности, он определяет режим работы (MODE) и предоставляет информацию о версии, имени, документации и других метаданных.

## Константы

### `MODE`

**Описание**: Переменная, определяющая режим работы GUI.

**Значение**: `'dev'`
