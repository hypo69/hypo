# Модуль hypotez/src/ai/gemini/header.py

## Обзор

Этот модуль отвечает за инициализацию и настройку проекта. Он находит корневую директорию проекта, загружает конфигурацию из файла `config.json`, а также считывает содержимое файла `README.MD`.  Он определяет переменные, хранящие информацию о проекте (название, версия, описание, авторы, и т.д.), для дальнейшего использования.


## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная от текущей директории файла. Ищет вверх по директориям, пока не найдет директорию, содержащую один из указанных файлов или директорий.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для определения корневой директории проекта.  По умолчанию - `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена, в противном случае - путь к директории, где расположен текущий скрипт.

**Вызывает исключения**:

-  Не вызывает исключений.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта. Инициализируется функцией `set_project_root`. Тип: `Path`.

### `config`

**Описание**: Словарь, содержащий конфигурацию проекта. Загружается из файла `config.json` в корневой директории проекта. Тип: `dict`.

### `doc_str`

**Описание**: Строка, содержащая содержимое файла `README.MD` в корневой директории проекта. Тип: `str`.

### `__project_name__`

**Описание**: Название проекта. Получается из словаря `config` или устанавливается по умолчанию как `'hypotez'`. Тип: `str`.

### `__version__`

**Описание**: Версия проекта. Получается из словаря `config` или устанавливается по умолчанию как пустая строка. Тип: `str`.

### `__doc__`

**Описание**: Документация проекта. Получается из переменной `doc_str` или устанавливается по умолчанию как пустая строка. Тип: `str`.


### `__details__`

**Описание**: Дополнительные детали проекта. Устанавливается по умолчанию как пустая строка. Тип: `str`.


### `__author__`

**Описание**: Автор проекта. Получается из словаря `config` или устанавливается по умолчанию как пустая строка. Тип: `str`.


### `__copyright__`

**Описание**: Авторские права проекта. Получается из словаря `config` или устанавливается по умолчанию как пустая строка. Тип: `str`.

### `__cofee__`

**Описание**: Ссылка на страницу поддержки разработчика (пожертвования кофе). Получается из словаря `config` или устанавливается по умолчанию. Тип: `str`.


**Обработка исключений**

Обработка ошибок `FileNotFoundError` и `json.JSONDecodeError` выполняется при чтении файлов `config.json` и `README.MD`. В случае ошибки переменные `config` и `doc_str` соответственно устанавливаются в `None` и пустую строку.
