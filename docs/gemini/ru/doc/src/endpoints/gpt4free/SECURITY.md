# Отчетность об уязвимостях

## Обзор

Этот документ описывает процесс сообщения об уязвимостях безопасности в проекте.

## Подробнее

Этот документ предназначен для предоставления информации о том, как сообщать об уязвимостях безопасности, чтобы они могли быть своевременно устранены. Анализируя предоставленный код, можно увидеть, что он описывает, как сообщать об уязвимостях безопасности в проекте.

## Отчетность об уязвимости

Пожалуйста, сообщайте о (предполагаемых) уязвимостях безопасности по адресу https://t.me/xtekky. Вы получите ответ в течение 48 часов. Если проблема подтвердится, мы выпустим исправление как можно скорее, в зависимости от сложности, но, как показывает практика, в течение нескольких дней.
