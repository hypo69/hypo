# Прокси-сервера (WebDriver)

## Обзор

Файл `proxies.txt` содержит список прокси-серверов в формате строка/строка, подходящих для использования с WebDriver.  Каждая строка представляет собой адрес прокси-сервера в формате `socks5://<IP-адрес>:<порт>` или `socks4://<IP-адрес>:<порт>`, либо `http://<IP-адрес>:<порт>`.
