# Блэкджек

## Обзор

Этот модуль реализует игру в блэкджек, где игрок соревнуется с дилером. Цель игры - набрать 21 очко или как можно ближе к 21, но не превысить это значение. В игре используется стандартная колода из 52 карт. Игрок может брать дополнительные карты (HIT) или остановиться (STAND). Дилер также берет карты до тех пор, пока сумма его очков не станет больше 16.

## Функции

### `deal_card`

**Описание**:
Выдает карту из колоды.

**Параметры**:
- `deck` (list): Колода карт.

**Возвращает**:
- `Any`: Последнюю карту из колоды.

### `calculate_hand_value`

**Описание**:
Вычисляет значение руки, учитывая, что туз может быть равен 1 или 11.

**Параметры**:
- `hand` (list): Рука игрока или дилера.

**Возвращает**:
- `int`: Суммарное значение карт в руке.

### `display_cards`

**Описание**:
Отображает карты игрока и дилера.

**Параметры**:
- `player_hand` (list): Рука игрока.
- `dealer_hand` (list): Рука дилера.
- `show_dealer_full` (bool, optional): Если `True`, показывает все карты дилера, иначе показывает только одну. По умолчанию `False`.

**Возвращает**:
- `None`

### `play_blackjack`

**Описание**:
Запускает игру в блэкджек.

**Параметры**:
- `None`

**Возвращает**:
- `None`
