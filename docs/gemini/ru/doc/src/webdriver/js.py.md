# Модуль `src.webdriver.js`

## Обзор

Модуль `src.webdriver.js` предоставляет утилиты на JavaScript для взаимодействия с веб-страницей. Он предназначен для расширения возможностей Selenium WebDriver путем добавления общих функций на основе JavaScript для взаимодействия с веб-страницами, включая манипуляции с видимостью, получение информации о странице и управление фокусом браузера.

Основные возможности:
    1. Делает невидимые DOM-элементы видимыми для взаимодействия.
    2. Получает метаданные, такие как состояние готовности документа, реферер или язык страницы.
    3. Программно управляет фокусом окна браузера.

## Классы

### `JavaScript`

**Описание**: Предоставляет утилиты на JavaScript для взаимодействия с веб-страницей.

**Методы**:
- `__init__`: Инициализирует помощник JavaScript с экземпляром Selenium WebDriver.
- `unhide_DOM_element`: Делает невидимый DOM-элемент видимым, изменяя его свойства стиля.
- `ready_state`: Возвращает статус загрузки документа.
- `window_focus`: Устанавливает фокус на окно браузера.
- `get_referrer`: Возвращает URL-адрес реферера текущего документа.
- `get_page_lang`: Возвращает язык текущей страницы.

#### `__init__`
**Описание**: Инициализирует объект `JavaScript` с экземпляром WebDriver.

**Параметры**:
- `driver` (WebDriver): Экземпляр WebDriver для выполнения JavaScript.

#### `unhide_DOM_element`

**Описание**: Делает невидимый DOM-элемент видимым, изменяя его свойства стиля.

**Параметры**:
- `element` (WebElement): Объект WebElement, который нужно сделать видимым.

**Возвращает**:
- `bool`: `True`, если скрипт выполнен успешно, `False` в противном случае.

**Вызывает исключения**:
- `Exception`: В случае ошибки при выполнении скрипта.

## Функции

### `ready_state`

**Описание**: Получает статус загрузки документа.

**Возвращает**:
- `str`: `'loading'`, если документ все еще загружается, `'complete'`, если загрузка завершена.

**Вызывает исключения**:
- `Exception`: В случае ошибки при получении `document.readyState`.

### `window_focus`

**Описание**: Устанавливает фокус на окно браузера с помощью JavaScript.

**Вызывает исключения**:
- `Exception`: В случае ошибки при выполнении `window.focus()`.

### `get_referrer`

**Описание**: Получает URL-адрес реферера текущего документа.

**Возвращает**:
- `str`: URL-адрес реферера или пустую строку, если недоступен.

**Вызывает исключения**:
- `Exception`: В случае ошибки при получении `document.referrer`.

### `get_page_lang`

**Описание**: Получает язык текущей страницы.

**Возвращает**:
- `str`: Код языка страницы или пустую строку, если недоступен.

**Вызывает исключения**:
- `Exception`: В случае ошибки при получении `document.documentElement.lang`.
