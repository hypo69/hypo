# Задача No 786
## Количество строк в графе Феррерса, когда количество ячеек равно n
## Условия
Граф Феррерса — это совокупность точек, расположенных в форме массивов, где количество точек в каждой строке не возрастает сверху вниз.

Например, возможные графы Феррерса, использующие 4 точки, следующие:

    XXXX
    XXX
    X
    XX
    XX
    X
    X
    X
    X
    X

Пусть f(n) — количество различных строк (то есть количество разных чисел) в графах Феррерса, содержащих n точек. Например, используя четыре точки, мы имеем количество строк 1, 2 и 4. Следовательно, f(4) = 3.

Вы также можете проверить, что f(5) = 3, f(10) = 8, f(100) = 1876, а f(1000) = 117874.

Найдите f(10^16). Выведите ответ по модулю 1 000 000 007.
