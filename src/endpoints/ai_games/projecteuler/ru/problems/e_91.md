# Задача No 91
## Прямоугольные треугольники с целочисленными координатами

## Условия
Три точки с целочисленными координатами (p, q), (r, s) и (t, u) образуют треугольник.

Задача № 91 ставит вопрос, сколько прямоугольных треугольников можно образовать с вершинами в точках с целочисленными координатами (0,0), (p, q) и (r, s), где 0 ≤ p, q, r, s ≤ 50?
