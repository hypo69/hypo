# Модуль hypotez/src/suppliers/aliexpress/api/version.py

## Обзор

Этот модуль содержит информацию о версии API AliExpress.  В нём определены переменные, описывающие версию, имя, документацию, дополнительные детали, аннотации и автора модуля.

## Переменные

### `__version__`

**Описание**: Переменная, содержащая версию модуля или пакета.

**Тип**: `str`

**Значение**: `"3.12.0.0.0.4"`


### `__name__`

**Описание**: Содержит имя модуля. Если скрипт выполняется напрямую, значение будет `"__main__"`.

**Тип**: `str`


### `__doc__`

**Описание**: Строка документации модуля.

**Тип**: `str`


### `__details__`

**Описание**: Эта переменная, вероятно, содержит дополнительные детали о модуле, но точное назначение зависит от конкретного модуля или пакета.

**Тип**: `str`

**Значение**: `"Details about version for module or class"`


### `__annotations__`

**Описание**: Содержит аннотации типов для переменных и функций в модуле.

**Тип**:  Любой тип, содержащий аннотации типов


### `__author__`

**Описание**: Имя(на) автора(ов) модуля.

**Тип**: `str`

**Значение**: `'hypotez '`
