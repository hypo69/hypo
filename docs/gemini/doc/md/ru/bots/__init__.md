# Модуль hypotez/src/bots

## Обзор

Данный модуль `hypotez/src/bots` содержит инициализацию и импорты для ботов.

## Модули

### `telegram`

Модуль содержит реализацию Telegram бота.


## Переменные

### `MODE`

**Описание**:  Переменная, определяющая режим работы (в данном случае 'development').

**Тип**: `str`

**Значение**: `'development'`


## Импорты

### `TelegramBot`

**Описание**:  Класс TelegramBot, вероятно, содержит логику работы с Telegram ботом.

**Модуль**: `hypotez.src.bots.telegram`
