# Модуль `header.py`

## Обзор

Модуль `header.py` предназначен для настройки и инициализации основных параметров проекта, таких как корневая директория, настройки из `settings.json`, версия проекта, авторские права и т.д. 
Этот модуль обеспечивает централизованный доступ к этим параметрам и упрощает их использование в других частях проекта.

## Константы

### `MODE`
```python
MODE = 'dev'
```
**Описание**: Указывает режим работы приложения (в данном случае - `dev`).

## Функции

### `set_project_root`

**Описание**:
Находит корневую директорию проекта, начиная с директории текущего файла, и добавляет её в `sys.path`.

**Параметры**:
- `marker_files` (tuple, optional): Кортеж с именами файлов или директорий, которые используются для определения корневой директории проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории проекта. Если не найдена, возвращает директорию, где расположен скрипт.

## Переменные модуля

### `__root__`

**Описание**:
Путь к корневой директории проекта. Вычисляется с помощью функции `set_project_root`.

### `settings`
**Описание**:
Словарь с настройками проекта, загруженный из файла `settings.json` . Если файл не найден или произошла ошибка декодирования, то `settings` будет равен `None`.

### `doc_str`
**Описание**:
Строка, содержащая содержимое файла `README.MD`, или `None`, если файл не найден.

### `__project_name__`
**Описание**:
Название проекта, взятое из `settings.json` или 'hypotez', если `settings` равно `None`.

### `__version__`
**Описание**:
Версия проекта, взятая из `settings.json` или пустая строка, если `settings` равно `None`.

### `__doc__`
**Описание**:
Документация проекта, содержащаяся в `doc_str`, или пустая строка, если `doc_str` равно `None`.

### `__details__`
**Описание**:
Строка с детальной информацией о проекте. В данный момент пустая строка.

### `__author__`
**Описание**:
Автор проекта, взятый из `settings.json` или пустая строка, если `settings` равно `None`.

### `__copyright__`
**Описание**:
Информация о копирайте проекта, взятая из `settings.json` или пустая строка, если `settings` равно `None`.

### `__cofee__`
**Описание**:
Сообщение о возможности поддержки разработчика, взятое из `settings.json` или стандартное сообщение, если `settings` равно `None`.
