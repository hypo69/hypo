# Модуль `prepare_all_campaigns`

## Обзор

Модуль предназначен для запуска обработки рекламных кампаний AliExpress для различных языков и категорий. Он использует функции из модуля `prepare_campaigns` для подготовки и запуска кампаний.

## Подробней

Этот модуль является частью процесса автоматизации рекламных кампаний AliExpress. Он предназначен для упрощения запуска и управления кампаниями для разных языковых рынков и категорий товаров. Модуль может создавать новые рекламные кампании, если они не существуют, и использует названия категорий из директорий для настройки кампаний.

## Функции

### `process_campaign`

```python
process_campaign(campaign_name: str, language: str, currency: str, campaign_file: Optional[str] = None) -> None:
    """
    Подготавливает и запускает рекламную кампанию для заданных параметров.

    Args:
        campaign_name (str): Название рекламной кампании.
        language (str): Язык кампании (например, 'EN', 'HE', 'RU').
        currency (str): Валюта кампании (например, 'USD', 'ILS').
        campaign_file (Optional[str], optional): Путь к файлу кампании (если существует). По умолчанию `None`.

    Returns:
        None

    Raises:
        Exception: Если возникает ошибка при обработке кампании.

    Example:
        >>> process_campaign(campaign_name='rc', language='EN', currency='USD')
    """
```

**Описание**: Функция `process_campaign` отвечает за подготовку и запуск рекламной кампании.

**Как работает функция**:
1. Принимает название кампании, язык, валюту и путь к файлу кампании (если есть).
2. Вызывает функцию `process_campaign` из модуля `prepare_campaigns` для обработки кампании с указанными параметрами.
3. Если файл кампании не указан, будет создана новая кампания.

**Параметры**:
- `campaign_name` (str): Название рекламной кампании.
- `language` (str): Язык кампании (например, 'EN', 'HE', 'RU').
- `currency` (str): Валюта кампании (например, 'USD', 'ILS').
- `campaign_file` (Optional[str], optional): Путь к файлу кампании (если существует). По умолчанию `None`.

**Возвращает**:
- `None`

**Вызывает исключения**:
- `Exception`: Если возникает ошибка при обработке кампании.

**Примеры**:

```python
process_campaign(campaign_name='rc', language='EN', currency='USD')
```

### `main_process`

```python
main_process(category_name: str, brands: list[str]) -> None:
    """
    Запускает основной процесс обработки рекламных кампаний для указанных категорий и брендов.

    Args:
        category_name (str): Название категории товаров.
        brands (list[str]): Список брендов для обработки.

    Returns:
        None

    Raises:
        Exception: Если возникает ошибка при обработке кампании.

    Example:
        >>> main_process('brands', ['mrgreen'])
    """
```

**Описание**: Функция `main_process` запускает основной процесс обработки рекламных кампаний для указанных категорий и брендов.

**Как работает функция**:
1. Принимает название категории и список брендов.
2. Вызывает функцию `main_process` из модуля `prepare_campaigns` для обработки кампаний с указанными параметрами.
3. Использует предоставленные категории и бренды для настройки и запуска кампаний.

**Параметры**:
- `category_name` (str): Название категории товаров.
- `brands` (list[str]): Список брендов для обработки.

**Возвращает**:
- `None`

**Вызывает исключения**:
- `Exception`: Если возникает ошибка при обработке кампании.

**Примеры**:

```python
main_process('brands', ['mrgreen'])
```
