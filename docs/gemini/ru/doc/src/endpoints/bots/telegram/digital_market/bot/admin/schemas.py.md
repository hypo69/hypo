# Модуль schemas.py

## Обзор

Модуль `schemas.py` содержит определения моделей данных, используемых для представления информации о продуктах в цифровом магазине Telegram-бота. Он определяет структуры данных для идентификации продукта, представления информации о продукте, включая его название, описание, цену, категорию, идентификатор файла и скрытый контент.

## Подробнее

Этот модуль предоставляет модели данных, основанные на Pydantic, для валидации и сериализации данных, связанных с продуктами. Он гарантирует, что данные о продуктах соответствуют определенным критериям, таким как минимальная длина названия и описания, а также положительная цена и идентификатор категории. Эти модели используются для обмена данными между различными компонентами бота, такими как API, база данных и пользовательский интерфейс.

## Классы

### `ProductIDModel`

**Описание**: Модель для представления идентификатора продукта.

**Параметры**:
- `id` (int): Идентификатор продукта.

### `ProductModel`

**Описание**: Модель для представления информации о продукте.

**Параметры**:
- `name` (str): Название продукта (минимальная длина 5 символов).
- `description` (str): Описание продукта (минимальная длина 5 символов).
- `price` (int): Цена продукта (должна быть больше 0).
- `category_id` (int): Идентификатор категории продукта (должен быть больше 0).
- `file_id` (str | None): Идентификатор файла продукта (может быть `None`).
- `hidden_content` (str): Скрытый контент продукта (минимальная длина 5 символов).
