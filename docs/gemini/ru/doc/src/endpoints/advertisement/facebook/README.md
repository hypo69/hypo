# Модуль `facebook`

## Обзор

Данный модуль содержит функции и классы для работы с рекламными объявлениями на платформе Facebook.

## Оглавление

* [Функции](#функции)


## Функции

### `create_ad_campaign`

**Описание**: Функция создания рекламной кампании на Facebook.

**Параметры**:
- `campaign_name` (str): Название рекламной кампании.
- `target_audience` (dict): Целевая аудитория кампании.
- `budget` (float): Бюджет кампании.
- `advertising_objectives` (list[str], optional): Цели рекламы. По умолчанию `None`.
- `ad_placement` (list[str], optional): Место размещения рекламы. По умолчанию `None`.

**Возвращает**:
- `dict | None`: Данные о созданной кампании или `None` в случае ошибки.


**Вызывает исключения**:
- `FacebookAPIError`: Возникает при ошибках доступа к API Facebook.
- `ValueError`: Возникает при некорректных входных данных.
- `Exception`: Возникает при возникновении других непредвиденных ошибок.

### `get_ad_performance`

**Описание**: Функция получения данных о результатах рекламной кампании.

**Параметры**:
- `campaign_id` (str): Идентификатор рекламной кампании.

**Возвращает**:
- `dict | None`: Данные о результатах кампании или `None` в случае ошибки.

**Вызывает исключения**:
- `FacebookAPIError`: Возникает при ошибках доступа к API Facebook.
- `ValueError`: Возникает при некорректных входных данных.
- `Exception`: Возникает при возникновении других непредвиденных ошибок.
