# Модуль `graber.py`

## Обзор

Модуль `graber.py` предназначен для сбора данных о товарах с сайта `grandadvance.co.il`. Он реализует класс `Graber`, который наследуется от класса `Graber` из модуля `src.suppliers.graber`. Класс `Graber` предоставляет методы для извлечения информации о товарах, при этом обеспечивая возможность переопределения стандартной логики обработки полей, а также позволяет использовать декораторы для выполнения предварительных действий перед отправкой запросов к веб-драйверу.

## Функции

### `close_pop_up`

**Описание**: Создает декоратор для закрытия всплывающих окон перед выполнением основной логики функции.
  **!!! Этот метод закомментирован в коде и не используется в данный момент.**

**Параметры**:
- `value` (`Any`, optional): Дополнительное значение для декоратора. По умолчанию `None`.

**Возвращает**:
- `Callable`: Декоратор, оборачивающий функцию.

**Принцип работы**:
Декоратор  перехватывает выполнение функции, перед ее запуском выполняется код закрытия всплывающего окна, затем выполняется обернутая функция.
Код перехвата всплывающего окна может быть переопределен.
