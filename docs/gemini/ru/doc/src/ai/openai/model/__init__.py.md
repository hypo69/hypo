# Модуль `src.ai.openai.model`

## Обзор

Модуль `src.ai.openai.model` содержит определения и функциональность для взаимодействия с моделями OpenAI, в частности, для обучения и использования моделей.

## Переменные

### `MODE`

- **Описание**: Режим работы модуля, по умолчанию установлен в `'dev'`.
- **Тип**: `str`
- **Значение**: `'dev'`

## Импортированные модули

### `training`

- **Описание**: Модуль, содержащий класс `OpenAIModel`.

## Классы

### `OpenAIModel`

- **Описание**: Класс для обучения и работы с моделями OpenAI. Подробная документация находится в [модуле training](training.md).
- **Импортируется из**: `hypotez/src/ai/openai/model/training.py`
