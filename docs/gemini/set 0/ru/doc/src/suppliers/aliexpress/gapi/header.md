# Модуль hypotez/src/suppliers/aliexpress/gapi/header.py

## Обзор

Модуль `header.py` определяет корневой путь к проекту `hypotez`. Все импорты в проекте строятся относительно этого пути.  В будущем планируется перенести определение корневого пути в системную переменную.

## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная от текущей директории, ищет вверх по дереву каталогов, пока не найдет каталог, содержащий один из указанных файлов или директорий.

**Параметры**:

- `marker_files` (tuple): Список файлов или каталогов, по которым определяется корневой каталог проекта.

**Возвращает**:

- `Path`: Путь к корневому каталогу проекта, если найден, иначе — текущая директория.

**Обрабатывает исключения**:

- Нет


## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта.

**Тип**: `Path`

##  Дополнительные замечания

Этот модуль используется для корректного импорта файлов из других модулей проекта `hypotez`. Он позволяет при необходимости изменить структуру проекта без необходимости изменения импорта во всех других файлах.

##  Инициализация

Функция `set_project_root()` устанавливает корневой путь проекта в переменную `__root__`. Далее, модуль пытается загрузить настройки из файла `settings.json` в переменную `settings`.  Если файл не найден или некорректно отформатирован, переменная `settings` остается `None`. Аналогично обрабатываются попытки загрузить информацию из файла `README.MD` в переменную `doc_str`.

```
__root__ = set_project_root()
```

Полученные значения, если они были успешно загружены, используются для определения имени проекта, версии, документации, автора и других метаданных проекта, иначе принимаются значения по умолчанию.
```
__project_name__ = settings.get("project_name", 'hypotez') if settings  else 'hypotez'
__version__: str = settings.get("version", '')  if settings  else ''
__doc__: str = doc_str if doc_str else ''
__details__: str = ''
__author__: str = settings.get("author", '')  if settings  else ''
__copyright__: str = settings.get("copyrihgnt", '')  if settings  else ''
__cofee__: str = settings.get("cofee", "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69")  if settings  else "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69"
