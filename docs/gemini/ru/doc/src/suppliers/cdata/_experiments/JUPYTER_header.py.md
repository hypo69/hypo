# Документация для модуля `src.suppliers.cdata._experiments.JUPYTER_header`

## Обзор

Данный модуль, расположенный по пути `src/suppliers/cdata/_experiments/JUPYTER_header.py`, представляет собой экспериментальный файл, используемый для работы с поставщиками данных в проекте `hypotez`. Он содержит импорты необходимых библиотек, настройки путей и функции для запуска поставщика.

## Подробнее

Модуль предназначен для инициализации и настройки окружения поставщика данных, такого как `aliexpress`, а также для интеграции с другими частями проекта, включая веб-драйвер, модели данных `Product` и `Category`, утилиты для работы со строками и эндпоинты `PrestaShop`.

## Функции

### `start_supplier`

```python
def start_supplier(supplier_prefix: str = 'aliexpress', locale: str = 'en') -> Supplier:
    """ Старт поставщика """
    ...
```

**Назначение**: Запускает поставщика данных с указанными параметрами.

**Параметры**:

-   `supplier_prefix` (str): Префикс поставщика (например, `'aliexpress'`). По умолчанию `'aliexpress'`.
-   `locale` (str): Локаль поставщика (например, `'en'`). По умолчанию `'en'`.

**Возвращает**:

-   `Supplier`: Объект поставщика, инициализированный с переданными параметрами.

**Как работает функция**:

1.  Функция принимает префикс поставщика и локаль в качестве входных параметров.
2.  Формирует словарь `params` с этими параметрами.
3.  Инициализирует объект `Supplier` с использованием переданных параметров.
4.  Возвращает созданный объект `Supplier`.

```
      start_supplier
      |
      params = {supplier_prefix, locale}
      |
      Supplier(**params)
      |
      return Supplier
```

**Примеры**:

```python
# Пример запуска поставщика aliexpress с английской локалью
supplier = start_supplier(supplier_prefix='aliexpress', locale='en')

# Пример запуска поставщика с русской локалью
supplier = start_supplier(supplier_prefix='aliexpress', locale='ru')
