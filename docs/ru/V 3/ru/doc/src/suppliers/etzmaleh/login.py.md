# Модуль `login`

## Обзор

Модуль `login.py` предназначен для реализации процесса авторизации поставщика `etzmaleh` с использованием веб-драйвера. Он содержит функцию `login`, которая выполняет вход в систему и возвращает результат операции.

## Подробней

Этот модуль является частью подсистемы поставщиков проекта `hypotez` и отвечает за автоматизацию входа в систему поставщика `etzmaleh`. Он использует модуль `logger` для логирования процесса входа и возвращает `True` в случае успешной авторизации. В текущей версии, функция `login` просто логирует факт входа и возвращает `True`.
