# Документация для Movie Bot

## Обзор

Этот документ предоставляет подробное описание телеграм-бота `Movie Bot`, предназначенного для поиска ссылок на бесплатный просмотр фильмов и сериалов. Бот разработан с использованием библиотеки `Aiogram`, сервисов Google для ассоциативного поиска, библиотеки `Кинопоиска` и сервиса `w2.kpfr/wiki`.

## Подробней

`Movie Bot` предоставляет пользователям возможность находить ссылки на бесплатный просмотр фильмов и сериалов через Telegram. Он использует несколько источников данных и технологий для обеспечения наиболее релевантных результатов поиска:

- **Aiogram**: Основная библиотека для создания Telegram-ботов.
- **Google Search**: Используется для ассоциативного поиска.
- **Кинопоиск API**: Предоставляет информацию о фильмах и сериалах.
- **w2.kpfr/wiki**: Дополнительный источник данных для поиска ссылок.
- **Middlewares**: Реализована защита от флуда.
- **BeautifulSoup**: Используется для парсинга HTML-страниц.

## Быстрый старт

### Клонирование проекта
Клонируйте репозиторий проекта на свой локальный компьютер.

### Настройка `.env` файла
Создайте файл `.env` в корневой директории проекта и заполните его необходимыми переменными окружения.

```
TOKEN=<Токен вашего telegram бота>
```

### Создание и активация виртуального окружения

Если у вас еще нет виртуального окружения, создайте его и активируйте.

```shell
python -m venv venv
```

```shell
venv\\Scripts\\activate.bat
```

### Установка зависимостей

Обновите `pip` и установите все необходимые зависимости из файла `requirements.txt`.

```shell
pip install --upgrade pip
```

```shell
pip install -r requirements.txt
```

### Запуск бота

Запустите бота, выполнив скрипт `run.py`.

```shell
python run.py
```
