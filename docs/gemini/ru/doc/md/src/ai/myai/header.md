# Модуль hypotez/src/ai/myai/header.py

## Обзор

Данный модуль содержит функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и получения документации из файла `README.MD`. Он также определяет переменные, содержащие информацию о проекте, такие как имя, версия, описание, автор и другие.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла, ищет вверх по дереву директорий и останавливается на первой директории, содержащей любой из указанных файлов-маркеров.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для определения корневой директории проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Объект `Path` с путем к корневой директории проекта, если она найдена. В противном случае возвращает директорию, где находится текущий скрипт.

**Вызывает исключения**:

- Не вызывает исключений.


## Переменные

### `__root__`

**Описание**:  Объект `Path` содержащий путь к корневой директории проекта.


### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженные из файла `src/settings.json`.  Может быть `None`, если файл не найден или некорректен.


### `doc_str`

**Описание**: Строка, содержащая текст документации, загруженная из файла `src/README.MD`. Может быть `None`, если файл не найден или некорректен.


### `__project_name__`

**Описание**:  Строка, содержащая имя проекта, полученная из настроек (`settings`). По умолчанию `'hypotez'`.


### `__version__`

**Описание**: Строка, содержащая версию проекта, полученная из настроек (`settings`). По умолчанию `''`.


### `__doc__`

**Описание**: Строка, содержащая описание проекта, полученная из документации (`doc_str`). По умолчанию `''`.


### `__details__`

**Описание**: Строка, содержащая дополнительные детали о проекте. По умолчанию `''`.


### `__author__`

**Описание**: Строка, содержащая имя автора проекта, полученная из настроек (`settings`). По умолчанию `''`.


### `__copyright__`

**Описание**: Строка, содержащая копирайт проекта, полученная из настроек (`settings`). По умолчанию `''`.


### `__cofee__`

**Описание**: Строка, содержащая ссылку для поддержки автора, полученная из настроек (`settings`). По умолчанию ссылка на Boosty.


## Обработка исключений

В модуле используются блоки `try...except` для обработки возможных исключений при работе с файлами `settings.json` и `README.MD`. В случае возникновения `FileNotFoundError` или `json.JSONDecodeError` соответствующие переменные (например, `settings`, `doc_str`)  получают значение `None`.
