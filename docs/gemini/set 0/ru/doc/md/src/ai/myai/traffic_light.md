# Модуль traffic_light

## Обзор

Модуль `traffic_light` содержит константу `MODE`, которая определяет режим работы системы.  В текущей реализации режим задаётся строкой 'dev'. Подробнее о назначении и использовании режима см. комментарии в коде.


## Константы

### `MODE`

**Описание**: Константа, определяющая режим работы системы. В текущей реализации принимает значение 'dev'.

**Значение**: `'dev'`
