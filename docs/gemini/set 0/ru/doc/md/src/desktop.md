# Файл desktop.ini

## Обзор

Файл `desktop.ini` используется для хранения метаданных о папке в операционной системе Windows. Он содержит информацию о визуальном представлении папки (например, значке) и других настройках.  Этот файл имеет специальный формат, использующий секции и ключи, для хранения данных.  Файл не содержит исполняемого кода и не предназначен для непосредственного использования Python-программами.  Он является дополнительным метаданными и используется для отображения папок.

## Формат файла

Файл `desktop.ini` имеет структуру, основанную на секциях.  Каждый элемент данных находится в пределах определённой секции. Пример:

```
[.ShellClassInfo]
IconResource=C:\\WINDOWS\\System32\\SHELL32.dll,300
[ViewState]
Mode=
Vid=
FolderType=Generic
```

Каждая секция имеет имя, заключённое в квадратные скобки.  Ключи внутри секций определяют конкретные атрибуты.  Например, ключ `IconResource` внутри секции `.ShellClassInfo` задаёт ресурс значка.
