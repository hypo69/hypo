Модуль `login.py` предназначен для выполнения входа в систему поставщика IVORY. Этот файл, вероятно, содержит логику для аутентификации и получения доступа к ресурсам IVORY.

Этот модуль, расположенный в `src/suppliers/ivory/login.py`, играет важную роль в процессе интеграции с поставщиком IVORY. Он может содержать функции для обработки учетных данных, установления сессий и управления доступом к API IVORY.
