# Tiny Utils

## Обзор

**Tiny Utils** — это библиотека утилит, предоставляющая набор легких функций для различных общих задач. Библиотека включает утилиты для преобразования форматов данных, обработки текста и файлов, операций со строками, форматирования дат и времени, обработки изображений и многое другое. Она организована по модулям для удобного доступа к конкретным функциональным возможностям.

## Установка

Для использования **Tiny Utils** клонируйте репозиторий и установите необходимые зависимости, как указано в файле `requirements.txt`.

```bash
git clone https://github.com/hypo69/tiny-utils.git
cd tiny_utils
pip install -r requirements.txt
```

## Обзор модулей

Эта библиотека содержит несколько подмодулей, каждый из которых отвечает за определенную задачу:

- **Преобразователи**: Модули для преобразования форматов данных, таких как текст в изображение, webp в png, JSON, XML, кодирование Base64 и многое другое.
- **Утилиты для строк**: Инструменты для продвинутой обработки строк.
- **Операции с файлами**: Функции для обработки и манипулирования файлами.
- **Утилиты для работы с датами и временем**: Инструменты для форматирования дат и времени.
- **Утилиты FTP**: Функции обработки файлов FTP.
- **Утилиты для изображений**: Базовые функции обработки изображений.
- **Утилиты для PDF**: Утилиты для работы с файлами PDF, включая преобразование, объединение, разделение и извлечение текста.
- **Утилиты для принтера**: Функции для отправки данных на принтер.

## Описание модулей

### Преобразователи

Модуль `convertors` содержит утилиты для преобразования данных между форматами. Эти модули могут обрабатывать различные типы данных, от CSV до JSON и текста до изображений.

#### Файлы:

- `text2png.py`: Преобразует данные текста в файл изображения PNG.
- `tts.py`: Преобразует текст в речь и сохраняет его как аудиофайл.
- `webp2png.py`: Преобразует изображения из формата WebP в формат PNG.
- `xls.py`: Обрабатывает преобразования и манипуляции с файлами XLS.
- `xml2dict.py`: Преобразует данные XML в словарь Python.
- `base64.py`: Кодирует или декодирует данные с использованием кодирования Base64.
- `csv.py`: Предоставляет инструменты для парсинга и манипулирования CSV.
- `dict.py`: Утилиты для работы со словарями Python.
- `html.py`: Преобразует содержимое HTML в различные форматы.
- `json.py`: Утилиты для парсинга и манипулирования JSON.
- `md2dict.py`: Преобразует Markdown-содержимое в словарь.
- `ns.py`: Специализированные утилиты для преобразования пространств имен.

## Примеры использования

### Преобразование текста в изображение PNG

```python
from tiny_utils.convertors import text2png

text = "Привет, мир!"
output_path = "выходное_изображение.png"
text2png.convert(text, output_path)
```

### Преобразование XML в словарь

```python
from tiny_utils.convertors import xml2dict

xml_data = "<root><item>Привет</item></root>"
словарь = xml2dict.convert(xml_data)
print(словарь)
```

### Парсинг и обработка JSON

```python
from tiny_utils.convertors import json

json_data = '{"имя": "Иван", "возраст": 30}'
обработанные_данные = json.parse(json_data)
print(обработанные_данные)
```
