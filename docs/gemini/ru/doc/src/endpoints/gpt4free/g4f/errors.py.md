# Модуль ошибок g4f

## Обзор

Модуль содержит определения пользовательских исключений, используемых в проекте `g4f` (gpt4free). Эти исключения предназначены для обработки различных ошибок, которые могут возникнуть при работе с провайдерами, моделями и API.

## Подробнее

Этот модуль предоставляет набор классов исключений, которые позволяют более конкретно обрабатывать ошибки, возникающие в процессе работы с различными сервисами и моделями. Использование специализированных исключений упрощает отладку и обработку ошибок в коде, делая его более надежным и понятным.

## Классы

### `ProviderNotFoundError`

**Описание**: Исключение, которое выбрасывается, когда провайдер не найден.

### `ProviderNotWorkingError`

**Описание**: Исключение, которое выбрасывается, когда провайдер не работает.

### `StreamNotSupportedError`

**Описание**: Исключение, которое выбрасывается, когда потоковая передача не поддерживается.

### `ModelNotFoundError`

**Описание**: Исключение, которое выбрасывается, когда модель не найдена.

### `ModelNotAllowedError`

**Описание**: Исключение, которое выбрасывается, когда модель не разрешена.

### `RetryProviderError`

**Описание**: Исключение, которое выбрасывается для повторной попытки с другим провайдером.

### `RetryNoProviderError`

**Описание**: Исключение, которое выбрасывается, когда не осталось провайдеров для повторной попытки.

### `VersionNotFoundError`

**Описание**: Исключение, которое выбрасывается, когда версия не найдена.

### `ModelNotSupportedError`

**Описание**: Исключение, которое выбрасывается, когда модель не поддерживается.

### `MissingRequirementsError`

**Описание**: Исключение, которое выбрасывается, когда отсутствуют необходимые зависимости.

### `NestAsyncioError`

**Описание**: Исключение, которое выбрасывается при проблемах с `nest_asyncio`.
**Наследует**: `MissingRequirementsError`

### `MissingAuthError`

**Описание**: Исключение, которое выбрасывается, когда отсутствует аутентификация.

### `PaymentRequiredError`

**Описание**: Исключение, которое выбрасывается, когда требуется оплата.

### `NoMediaResponseError`

**Описание**: Исключение, которое выбрасывается, когда отсутствует медиа-ответ.

### `ResponseError`

**Описание**: Исключение, которое выбрасывается при общей ошибке ответа.

### `ResponseStatusError`

**Описание**: Исключение, которое выбрасывается при ошибке в статусе ответа.
**Наследует**: `ResponseError`

### `RateLimitError`

**Описание**: Исключение, которое выбрасывается при превышении лимита запросов.
**Наследует**: `ResponseStatusError`

### `NoValidHarFileError`

**Описание**: Исключение, которое выбрасывается, когда HAR-файл недействителен.

### `TimeoutError`

**Описание**: Исключение, которое выбрасывается при ошибке времени ожидания во время запросов API.

### `ConversationLimitError`

**Описание**: Исключение, которое выбрасывается, когда превышен лимит разговоров при запросах API к конечной точке AI.
