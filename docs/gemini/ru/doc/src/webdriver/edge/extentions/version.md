# Модуль hypotez/src/webdriver/edge/extentions/version.py

## Обзор

Этот модуль содержит константы и переменные, связанные с версией расширения для браузера Edge.

## Переменные

### `MODE`

**Описание**: Переменная, хранящая режим работы, например, 'dev' или 'prod'.

**Значение**: `'dev'`


## Дополнительные сведения

### `__version__`

**Описание**:  Переменная, содержащая версию модуля или пакета.

**Значение**: `"3.12.0.0.0.4"`


### `__name__`

**Описание**: Имя модуля. Если скрипт выполняется напрямую, значение будет `"__main__"`.


### `__doc__`

**Описание**: Строка документации модуля.


### `__details__`

**Описание**:  Переменная, вероятно, содержит дополнительные детали о модуле, но точное назначение зависит от конкретного модуля или пакета.

**Значение**: `"Details about version for module or class"`


### `__annotations__`

**Описание**: Содержит типы данных для переменных и функций в модуле.


### `__author__`

**Описание**: Имя(я) автора(ов) модуля.

**Значение**: `"hypotez "`
