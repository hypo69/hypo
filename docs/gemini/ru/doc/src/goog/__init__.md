# Модуль goog

## Обзор

Этот модуль предоставляет инструменты для работы с Google Spreadsheets.

## Содержимое

- `MODE` (str): Переменная, определяющая режим работы (например, 'dev').
- `Spreadsheet`: Класс для работы с Google Spreadsheets.

## Классы

### `Spreadsheet`

**Описание**: Класс для взаимодействия с Google Spreadsheets.

**Методы**:

- Методы класса `Spreadsheet` отсутствуют в предоставленном коде.  Для их документирования необходимо знать их функциональность и сигнатуры.
  
## Переменные

### `MODE`

**Описание**: Переменная, определяющая режим работы.  Значение по умолчанию — 'dev'.

**Тип**: str

**Значение**: 'dev' (по умолчанию)


## Импорты

- `from .spreadsheet import SpreadSheet`: Импорт класса `Spreadsheet` из файла `spreadsheet.py` в текущем пакете.
