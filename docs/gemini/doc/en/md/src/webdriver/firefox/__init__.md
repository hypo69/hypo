# hypotez/src/webdriver/firefox/__init__.py

## Overview

This module provides initialization and import functionality for the Firefox webdriver, part of the `hypotez` project's web driver infrastructure. It currently only imports the `Firefox` class from the `firefox` submodule.


## Modules

### `firefox`

This submodule contains the implementation details for the Firefox webdriver.


## Variables

### `MODE`

**Description**: A string variable, set to 'dev' by default.  It's likely a configuration variable to define the mode of operation (e.g., development, production).

**Value**: `'dev'`


## Imports


### `from .firefox import Firefox`

Imports the `Firefox` class from the `firefox` submodule. This is the primary function of this module, enabling usage of the Firefox webdriver.
