# Модуль translate_product_fields

## Обзор

Этот модуль предоставляет функции для управления переводами полей товара. Он связывает словарь полей товара, таблицу переводов и переводчики, обеспечивая перевод полей товара на различные языки.

## Функции

### `get_translations_from_presta_translations_table`

**Описание**: Функция извлекает переводы полей товара из таблицы переводов PrestaShop.

**Параметры**:
- `product_reference` (str): Уникальный идентификатор товара.
- `credentials` (dict): Словарь с данными для аутентификации и подключения к базе данных переводов.
- `i18n` (str, optional): Код языка перевода в формате `en_EN`, `he_HE`, `ru-RU`. По умолчанию `None`.

**Возвращает**:
- `list`: Список словарей с переводами. Возвращает пустой список, если переводы не найдены.

**Обрабатывает исключения**:
- Возможные исключения при работе с базой данных должны быть обработаны в блоке `try...exept`.


### `insert_new_translation_to_presta_translations_table`

**Описание**: Функция для добавления новой записи перевода в таблицу переводов PrestaShop.

**Параметры**:
- `record` (dict): Словарь с данными для новой записи перевода.
- `credentials` (dict): Словарь с данными для аутентификации и подключения к базе данных переводов.

**Возвращает**:
- `None`.

**Обрабатывает исключения**:
- Возможные исключения при работе с базой данных должны быть обработаны в блоке `try...exept`.


### `translate_record`

**Описание**: Функция для перевода полей товара на другой язык.

**Параметры**:
- `record` (dict): Словарь с полями товара, которые необходимо перевести.
- `from_locale` (str): Исходный язык перевода в формате `en_EN`, `he_HE`, `ru-RU`.
- `to_locale` (str): Целевой язык перевода в формате `en_EN`, `he_HE`, `ru-RU`.

**Возвращает**:
- `dict`: Словарь с переведенными полями товара.

**Обрабатывает исключения**:
- Возможные исключения при работе с API перевода должны быть обработаны в блоке `try...exept`.
