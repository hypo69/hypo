# ZOOP

## Обзор

Этот модуль реализует простую игру в угадывание слов под названием "ZOOP". Компьютер выбирает случайное слово из предопределенного списка, а игрок пытается его угадать. Игроку предоставляется подсказка в виде количества букв в слове.

## Используемые библиотеки

- `random`: Используется для выбора случайного слова из списка.

## Переменные

- `wordList` (list): Список возможных слов для игры (`['ZOOP', 'BOOP', 'LOOP', 'GOOP']`).
- `targetWord` (str): Случайно выбранное слово из `wordList`, которое нужно угадать.
- `userGuess` (str): Ввод пользователя, его попытка угадать слово.

## Основной игровой цикл

### **Описание**:
Основной цикл игры, который продолжается до тех пор, пока игрок не угадает правильное слово.

### **Параметры**:
Отсутствуют.

### **Возвращает**:
Отсутствует.

### **Вызывает исключения**:
Отсутствуют.

#### **Логика цикла:**

1.  **Вывод длины слова:**
    - `print(f"אורך המילה הוא {len(targetWord)} אותיות")`: Выводит длину загаданного слова, предоставляя подсказку игроку.

2.  **Начало бесконечного цикла**:
    -  `while True:`: Бесконечный цикл, который продолжается до тех пор, пока игрок не угадает слово.

3.  **Получение пользовательского ввода**:
    -   `userGuess = input("נסה לנחש את המילה: ").upper()`: Запрашивает ввод от пользователя и преобразует его в верхний регистр для сравнения без учета регистра.

4.  **Проверка ввода**:
    -   `if userGuess == targetWord:`: Проверяет, совпадает ли ввод пользователя с загаданным словом.

5.  **Вывод сообщения о победе**:
    -   `print("מזל טוב! ניחשת נכון את המילה!")`: Выводит сообщение о победе, если пользователь угадал слово.
    -   `break`: Завершает цикл и игру.

6.  **Сообщение о неудачной попытке**:
    -   `else:`: Выполняется, если ввод пользователя не совпадает с загаданным словом.
    -   `print("נסה שוב.")`: Выводит сообщение с просьбой попробовать еще раз.
