# Модуль `header`

## Обзор

Модуль `header` содержит функции и настройки, необходимые для определения корневой директории проекта `hypotez` и загрузки настроек из файла `settings.json`. Он обеспечивает централизованную точку для доступа к конфигурационным данным и путям проекта.

## Подробней

Этот модуль предназначен для упрощения доступа к корневой директории проекта и настройкам приложения. Функция `set_project_root` автоматически определяет корень проекта на основе наличия определенных файлов-маркеров, таких как `__root__` или `.git`. Загрузка настроек из `settings.json` позволяет гибко конфигурировать приложение без необходимости изменения кода. Расположение файла: `hypotez/src/suppliers/aliexpress/header.py` указывает, что он используется в подсистеме, связанной с поставщиками AliExpress.

## Функции

### `set_project_root`

```python
def set_project_root(marker_files: tuple = ('__root__', '.git')) -> Path:
    """
    Args:
        marker_files (tuple): Filenames or directory names to identify the project root.
    Returns:
        Path: Path to the root directory if found, otherwise the directory where the script is located.
    """
```

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла, и останавливается на первой директории, содержащей любой из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или директорий для идентификации корня проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена, иначе - директория, где расположен скрипт.

**Примеры**:
```python
from pathlib import Path
root_path: Path = set_project_root()
print(f"Root path: {root_path}")
```

## Переменные

### `__root__`

```python
"""__root__ (Path): Path to the root directory of the project"""
```

**Описание**: Глобальная переменная, содержащая путь к корневой директории проекта.

### `settings`

```python
settings: dict = None
```

**Описание**: Глобальная переменная, содержащая словарь с настройками, загруженными из файла `settings.json`.
