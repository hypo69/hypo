# Модуль web_login

## Обзор

Модуль `web_login.py` предназначен для автоматизации процесса логина на сайт Aliexpress и сохранения cookies для дальнейшего использования. Это эксперимент, направленный на изучение возможности автоматического входа в систему для обхода ограничений, связанных с доступом к данным.

## Подробней

Этот модуль использует библиотеку `requests` для отправки HTTP-запросов и библиотеку `pickle` для сохранения и загрузки cookies. Он также использует класс `Supplier` из модуля `src.suppliers` для получения экземпляра драйвера браузера. Основная цель модуля - автоматизировать процесс входа на сайт Aliexpress, чтобы избежать ручного ввода логина и пароля при каждом запуске скрипта.

## Функции

### `Supplier`

```python
a = Supplier('aliexpress')
```

**Описание**: Создает экземпляр класса `Supplier` для работы с Aliexpress.

**Как работает функция**:
1.  Создается экземпляр класса `Supplier` с параметром `'aliexpress'`.
2.  Этот экземпляр используется для получения драйвера браузера, который будет использоваться для автоматизации входа на сайт Aliexpress.

**Параметры**:
-   `'aliexpress'` (str): Идентификатор поставщика, в данном случае Aliexpress.

**Возвращает**:
-   `None`

**Вызывает исключения**:
-   `Exception`: Если возникает ошибка при создании экземпляра класса `Supplier`.

**Примеры**:

```python
a = Supplier('aliexpress')
```

### `get_url`

```python
d.get_url('https://aliexpress.com')
```

**Описание**: Открывает указанный URL в браузере.

**Как работает функция**:
1.  Вызывается метод `get_url` драйвера браузера (`d`) с URL `'https://aliexpress.com'`.
2.  Браузер переходит по указанному URL.

**Параметры**:
-   `'https://aliexpress.com'` (str): URL, который необходимо открыть в браузере.

**Возвращает**:
-   `None`

**Вызывает исключения**:
-   `Exception`: Если возникает ошибка при открытии URL в браузере.

**Примеры**:

```python
d.get_url('https://aliexpress.com')
```
