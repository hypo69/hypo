# Модуль examples.py

## Обзор

Модуль содержит примеры использования библиотеки `tinytroupe`. В этих примерах показано, как создавать агентов (TinyPerson) с различными характеристиками и навыками, которые можно использовать непосредственно или модифицировать для создания собственных агентов.

## Подробнее

Этот модуль демонстрирует создание нескольких персонажей с разными профессиями, интересами и навыками. Каждый персонаж создается как экземпляр класса `TinyPerson`, и для каждого персонажа определяются различные атрибуты, такие как возраст, национальность, профессия, распорядок дня, описание профессии, черты личности, профессиональные и личные интересы, навыки и отношения с другими персонажами.

## Функции

### `create_oscar_the_architect`

```python
def create_oscar_the_architect():
  """
  Создает персонажа по имени Оскар, архитектора.

  Args:
      Нет

  Returns:
      TinyPerson: Объект TinyPerson, представляющий Оскара-архитектора.

  Raises:
      Нет
  """
```

**Как работает функция:**

1.  **Создание персонажа**: Создается экземпляр класса `TinyPerson` с именем "Oscar".
2.  **Определение атрибутов**: Определяются различные атрибуты Оскара, такие как возраст, национальность, профессия, распорядок дня, описание профессии, черты личности, профессиональные и личные интересы, навыки и отношения с другими персонажами.
3.  **Возврат персонажа**: Возвращается объект `TinyPerson`, представляющий Оскара-архитектора.

```mermaid
graph TD
    A[Создание персонажа TinyPerson("Oscar")] --> B{Определение атрибутов: возраст, национальность, профессия...}
    B --> C[Возврат объекта TinyPerson]
```

**Примеры:**

```python
oscar = create_oscar_the_architect()
print(oscar.name)  # Вывод: Oscar
print(oscar.age)   # Вывод: 30
```
