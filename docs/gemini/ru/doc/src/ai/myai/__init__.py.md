# Модуль `src.ai.myai`

## Обзор

Модуль `src.ai.myai` является пакетом и включает в себя различные инструменты и компоненты для работы с искусственным интеллектом. В данном модуле задается режим работы `MODE` по умолчанию, который установлен на `'dev'`.

## Переменные

### `MODE`

**Описание**: Режим работы модуля. По умолчанию установлен в `'dev'`.

**Тип**: `str`
