# Модуль hypotez/src/ai/gemini/__init__.py

## Обзор

Модуль `gemini` содержит настройки и импорты для работы с моделью генеративного искусственного интеллекта Gemini.

## Переменные

### `MODE`

**Описание**: Строковая переменная, определяющая режим работы (например, 'dev', 'prod'). В текущей реализации переменная настроена на значение 'dev'.

**Значение**: 'dev'


## Импорты

### `from .generative_ai import GoogleGenerativeAI`

**Описание**: Импортирует класс `GoogleGenerativeAI` из модуля `generative_ai`. Этот класс, вероятно, содержит реализацию взаимодействия с моделью Gemini.
