# Руководство для Тестера Модуля Рекламных Кампаний AliExpress

## Обзор

Данный документ предоставляет руководство для тестеров, отвечающих за проверку модуля, обрабатывающего рекламные кампании на платформе AliExpress. Модуль включает в себя три ключевых файла: `edit_campaign.py`, `prepare_campaigns.py` и `test_campaign_integration.py`.  Документация описывает функциональность каждого файла, предоставляя инструкции по тестированию и ожидаемое поведение.

## Файлы

### `edit_campaign.py`

**Описание**: Файл содержит класс `AliCampaignEditor`, унаследованный от `AliPromoCampaign`, отвечающий за управление рекламными кампаниями.

**Классы**:

#### `AliCampaignEditor`

**Описание**:  Класс для управления кампанией.

**Методы**:

-  `__init__`: Инициализация класса.  (Подробности требуют определения кода)
-  `update_campaign_data`: Обновление данных кампании. (Подробности требуют определения кода)
-  `run_campaign`: Запуск кампании. (Подробности требуют определения кода)

### `prepare_campaigns.py`

**Описание**: Файл содержит функции для подготовки материалов кампании, включая обновление категорий и обработку кампаний по категориям.

**Функции**:

#### `update_category`

**Описание**: Обновляет данные категории в JSON файле.

**Параметры**:
- `category_data` (dict): Данные о категории в формате словаря.
- `file_path` (str): Путь к JSON файлу с категориями.

**Возвращает**:
- bool: `True`, если обновление прошло успешно, `False` иначе.

**Вызывает исключения**:
- `FileNotFoundError`: Если файл не найден.
- `JSONDecodeError`: Если файл не валидный JSON.

#### `process_campaign_category`

**Описание**: Обрабатывает конкретную категорию в рамках кампании.

**Параметры**:
- `category_id` (int): Идентификатор категории.
- `campaign_data` (dict): Данные о кампании.

**Возвращает**:
- dict | None: Результат обработки категории, или `None` при ошибке.


#### `process_campaign`

**Описание**: Обрабатывает всю кампанию по всем категориям.

**Параметры**:
- `campaign_data` (dict): Данные о кампании.

**Возвращает**:
- list: Список результатов обработки каждой категории.


#### `main`

**Описание**: Асинхронная основная функция для обработки кампании.

**Параметры**:
- `campaign_data` (dict): Данные о кампании.

**Возвращает**:
- None


### `test_campaign_integration.py`

**Описание**:  Файл содержит тесты, проверяющие интеграцию компонентов модуля.


**Тесты**:

#### `test_update_category_success`

**Описание**: Проверка успешного обновления категории.

#### `test_update_category_failure`

**Описание**: Проверка обработки ошибки при обновлении категории.

#### `test_process_campaign_category_success`

**Описание**: Проверка успешной обработки категории.

#### `test_process_campaign_category_failure`

**Описание**: Проверка обработки ошибки при обработке категории.

#### `test_process_campaign`

**Описание**: Проверка обработки всех категорий в кампании.

#### `test_main`

**Описание**: Проверка основного сценария выполнения кампании.
