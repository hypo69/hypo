# Модуль утилит

## Обзор

Модуль содержит утилиту `removeFile` для удаления файлов.

## Содержание

- [Функции](#функции)
    - [`removeFile`](#removefile)

## Функции

### `removeFile`

**Описание**: Асинхронная функция для удаления файла по указанному пути.

**Параметры**:
- `path` (string): Путь к файлу, который необходимо удалить.

**Возвращает**:
- `Promise<void>`: Ничего не возвращает.

**Вызывает исключения**:
- `Error`: В случае ошибки при удалении файла выводит сообщение в консоль.
