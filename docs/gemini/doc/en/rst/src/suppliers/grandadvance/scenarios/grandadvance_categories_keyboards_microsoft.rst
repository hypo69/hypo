grandadvance_categories_keyboards_microsoft
==========================================

This file contains a JSON object defining various Microsoft keyboard and mouse products, along with their associated details.
It includes information such as brand, URLs, active status, and more.

.. rubric:: Data Structure

The data is organized as a dictionary where each key corresponds to a product name (e.g., "MICROSOFT WIRELESS KEYBOARD"), and each value is a dictionary containing specific product attributes.
