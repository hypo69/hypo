# Модуль hypotez/src/utils/iso/__init__.py

## Обзор

Данный модуль (`hypotez/src/utils/iso/__init__.py`) содержит конфигурацию режима работы.  В настоящий момент он определяет константу `MODE` со значением 'dev'.

## Конфигурационные параметры

### `MODE`

**Описание**:  Переменная, определяющая режим работы приложения.

**Значение**: `'dev'`

**Примечание**:  Данная константа может быть использована для выбора различных наборов настроек или поведения в зависимости от режима (например, `dev`, `prod`).
