# Документация для `amazon_categories_lighting.json`

## Обзор

Данный файл содержит JSON-конфигурацию для сценариев импорта товаров с Amazon, связанных с категорией "Освещение" (lighting). В частности, здесь определены настройки для импорта товаров из категории "Муранское стекло".

## Структура файла

### `scenarios`

**Описание**:
Корневой объект, содержащий сценарии импорта. Каждый ключ в этом объекте представляет собой название сценария.

#### `Murano Glass`

**Описание**:
Сценарий импорта товаров из категории "Муранское стекло" с Amazon.

##### `url`

**Описание**:
URL-адрес категории товаров на Amazon.

**Тип**: `str`

**Значение**: `"https://amzn.to/46PWVcs"`

##### `condition`

**Описание**:
Условие товаров, которые необходимо импортировать.

**Тип**: `str`

**Значение**: `"new"`

##### `presta_categories`

**Описание**:
Категории PrestaShop, в которые будут импортированы товары.

###### `default_category`

**Описание**:
Словарь, определяющий категорию PrestaShop по умолчанию.

**Тип**: `dict`

**Структура**: `{ "id_категории": "название_категории" }`

**Значение**: `{"11209": "MURANO GLASS"}`

###### `additional_categories`

**Описание**:
Список дополнительных категорий PrestaShop для импорта товаров.

**Тип**: `list`

**Значение**: `[""]`

##### `price_rule`

**Описание**:
Правило ценообразования, которое будет применено к товарам.

**Тип**: `int`

**Значение**: `1`
