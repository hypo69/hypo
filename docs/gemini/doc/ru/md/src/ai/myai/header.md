# Модуль hypotez/src/ai/myai/header.py

## Обзор

Данный модуль `header.py` отвечает за инициализацию проекта, находит корневую директорию проекта и загружает настройки из файла `settings.json`.  Он также загружает описание проекта из файла `README.MD` и предоставляет доступ к различным атрибутам проекта, таким как название, версия, описание и автор.

## Функции

### `set_project_root`

**Описание**:  Ищет корневую директорию проекта, начиная от текущего файла. Поиск происходит вверх по дереву директорий, останавливаясь на первой директории, содержащей один из указанных файлов-маркеров.

**Параметры**:

- `marker_files` (tuple): Кортеж с именами файлов или директорий, которые будут использоваться для определения корневой директории проекта. По умолчанию ('pyproject.toml', 'requirements.txt', '.git').

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена, иначе путь к директории, где расположен скрипт.

**Вызывает исключения**:

- Нет.


## Атрибуты

### `__root__`

**Описание**: Путь к корневой директории проекта.

**Тип**: `Path`

**Инициализация**: Вызывается функцией `set_project_root`.


### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json`.

**Тип**: `dict`

**Инициализация**: Загружается из файла `gs.path.root / 'src' / 'settings.json'`. Обрабатываются исключения `FileNotFoundError` и `json.JSONDecodeError`.


### `doc_str`

**Описание**: Строка с описанием проекта, загруженная из файла `README.MD`.

**Тип**: `str`

**Инициализация**: Загружается из файла `gs.path.root / 'src' / 'README.MD'`. Обрабатываются исключения `FileNotFoundError` и `json.JSONDecodeError`.


### `__project_name__`

**Описание**: Название проекта.

**Тип**: `str`

**Значение**: Получается из словаря настроек `settings`, по ключу `project_name`, если ключ отсутствует, то значение по умолчанию `hypotez`.


### `__version__`

**Описание**: Версия проекта.

**Тип**: `str`

**Значение**: Получается из словаря настроек `settings`, по ключу `version`, если ключ отсутствует, то значение по умолчанию `''`.


### `__doc__`

**Описание**: Документация проекта.

**Тип**: `str`

**Значение**: Получается из переменной `doc_str`, если она не пустая. В ином случае значение по умолчанию `''`.


### `__details__`

**Описание**: Дополнительные детали проекта.

**Тип**: `str`

**Значение**: По умолчанию `''`.


### `__author__`

**Описание**: Автор проекта.

**Тип**: `str`

**Значение**: Получается из словаря настроек `settings`, по ключу `author`, если ключ отсутствует, то значение по умолчанию `''`.


### `__copyright__`

**Описание**: Авторские права.

**Тип**: `str`

**Значение**: Получается из словаря настроек `settings`, по ключу `copyrihgnt`, если ключ отсутствует, то значение по умолчанию `''`.


### `__cofee__`

**Описание**: Ссылка на помощь автору проекта (например, ссылка на Бусти).

**Тип**: `str`

**Значение**: Получается из словаря настроек `settings`, по ключу `cofee`, если ключ отсутствует, то значение по умолчанию `'Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69'`.
