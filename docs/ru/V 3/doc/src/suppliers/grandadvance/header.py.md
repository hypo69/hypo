# Модуль `header.py`

## Обзор

Модуль `header.py` предназначен для определения основных параметров проекта `hypotez`, таких как имя проекта, версия, автор и пути к файлам конфигурации. Он также включает функцию для определения корневой директории проекта.

## Подробней

Этот модуль выполняет следующие задачи:

1.  Определение корневой директории проекта с помощью функции `set_project_root`.
2.  Чтение настроек из файла `settings.json` и документации из `README.MD`.
3.  Определение основных параметров проекта, таких как имя, версия, автор и т. д.

Этот модуль важен для правильной инициализации проекта и доступа к его основным настройкам.

## Функции

### `set_project_root`

```python
def set_project_root(marker_files: tuple[str, ...]=('__root__', '.git')) -> Path:
    """
    Args:
        marker_files (tuple): Filenames or directory names to identify the project root.
    
    Returns:
        Path: Path to the root directory if found, otherwise the directory where the script is located.
    """
```

**Описание**: Находит корневой каталог проекта, начиная с каталога текущего файла,
поиска вверх и остановки в первом каталоге, содержащем любой из маркерных файлов.

**Параметры**:

*   `marker_files` (tuple): Кортеж имен файлов или каталогов для идентификации корня проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:

*   `Path`: Путь к корневому каталогу, если он найден, в противном случае - каталог, в котором находится скрипт.

**Примеры**:

```python
from pathlib import Path
root_path = set_project_root()
print(f"Root directory: {root_path}")
```

## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта.

### `settings`

**Описание**: Словарь с настройками, загруженными из файла `settings.json`.

### `doc_str`

**Описание**: Строка с содержимым файла `README.MD`.

### `__project_name__`

**Описание**: Имя проекта. Значение по умолчанию: `hypotez`.

### `__version__`

**Описание**: Версия проекта. Значение по умолчанию: `''`.

### `__doc__`

**Описание**: Документация проекта, загруженная из файла `README.MD`. Значение по умолчанию: `''`.

### `__details__`

**Описание**: Дополнительные детали проекта. Значение по умолчанию: `''`.

### `__author__`

**Описание**: Автор проекта. Значение по умолчанию: `''`.

### `__copyright__`

**Описание**: Информация об авторских правах проекта. Значение по умолчанию: `''`.

### `__cofee__`

**Описание**: Сообщение о поддержке разработчика. Значение по умолчанию: `"Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69"`.
