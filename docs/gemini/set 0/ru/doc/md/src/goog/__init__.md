# Модуль hypotez/src/goog

## Обзор

Данный модуль содержит константу `MODE` и импортирует класс `Spreadsheet` из подмодуля `spreadsheet`.

## Константы

### `MODE`

**Описание**:  Переменная, хранящая строку 'dev'.  Вероятно, используется для обозначения режима работы.


## Модули

### `spreadsheet`

**Описание**: Содержит класс `Spreadsheet`.


## Классы

### `Spreadsheet`

**Описание**:  Класс `Spreadsheet`, вероятно, предназначен для работы с электронными таблицами. Подробное описание класса отсутствует в предоставленном коде.  Необходима дополнительная информация из файла `spreadsheet.py` для полной документации.
