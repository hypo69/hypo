dot Converter
============

.. module:: hypotez.src.utils.convertors.dot
   :platform: Windows, Unix
   :synopsis: Converts DOT files into PNG images using the Graphviz library


Description
-----------

This module provides a function to convert DOT files to PNG images using the Graphviz library. It allows users to specify the input DOT file and the desired output PNG file.
