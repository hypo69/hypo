# Модуль code_assistant

## Обзор

Этот модуль предоставляет инструменты для создания помощника по коду.

## Функции

### `get_code_completion`

**Описание**: Получает предложения по завершению кода.

**Аргументы**:
- `code_snippet` (str): фрагмент кода для анализа.
- `context` (Optional[str | dict | str], optional): контекст для анализа. По умолчанию `None`.

**Возвращает**:
- `list[str]`: список предложений по завершению кода.

**Вызывает исключения**:
- `ValueError`: если `code_snippet` не является строкой.
- `TypeError`: если `context` имеет неподходящий тип.


### `generate_documentation`

**Описание**: Генерирует документацию для Python-кода.

**Аргументы**:
- `filepath` (str): путь к файлу Python.
- `output_format` (str): формат выходных данных (например, 'rst').


**Возвращает**:
- `str`: сгенерированная документация.

**Вызывает исключения**:
- `FileNotFoundError`: если файл не найден.
- `SyntaxError`: если файл содержит синтаксические ошибки.
- `TypeError`: если `filepath` или `output_format` не являются строками.
