# Модуль `src.suppliers.morlevi._experiments`

## Обзор

Модуль содержит экспериментальный код, связанный с поставщиком Morlevi. Включает в себя импорты библиотек, определение путей и функцию для запуска поставщика.

## Подробней

Данный модуль, по-видимому, является частью экспериментального кода, предназначенного для тестирования или разработки новых функций, связанных с поставщиком Morlevi. Он включает в себя импорты различных модулей, настройку путей и функцию для запуска поставщика с определенными параметрами. Этот код может быть использован для отладки, тестирования или демонстрации функциональности, прежде чем она будет интегрирована в основную кодовую базу.

## Импорты

- `sys`: Используется для работы с системными параметрами и функциями.
- `os`: Используется для работы с операционной системой, например, для получения текущей директории.
- `pathlib.Path`: Используется для работы с путями к файлам и директориям.
- `json`: Используется для работы с JSON-данными.
- `re`: Используется для работы с регулярными выражениями.
- `src.webdriver.driver.Driver`: Класс для управления веб-драйвером.
- `src.suppliers.Supplier`: Класс поставщика.
- `src.product.Product`, `src.product.ProductFields`: Классы для работы с продуктами и их полями.
- `src.category.Category`: Класс для работы с категориями.
- `src.utils.StringFormatter`, `src.utils.StringNormalizer`: Классы для форматирования и нормализации строк.
- `src.utils.printer.pprint`: Функция для удобной печати данных.
- `src.endpoints.PrestaShop.Product as PrestaProduct`: Класс для работы с продуктами PrestaShop.
- `src.utils.files.save_text_file`: Функция для сохранения текста в файл.

## Переменные

- `dir_root (Path)`: Абсолютный путь к корневой директории проекта `hypotez`. Вычисляется путем определения текущей рабочей директории и обрезания пути до корневой папки проекта.
- `dir_src (Path)`:  Путь к директории `src` внутри корневой директории проекта. Используется для добавления в `sys.path`, чтобы обеспечить импорт модулей из этой директории.

## Функции

### `start_supplier`

```python
def start_supplier(supplier_prefix: str = 'aliexpress', locale: str = 'en') -> Supplier:
    """ Старт поставщика """
```

**Описание**: Функция для запуска поставщика с указанными параметрами.

**Как работает функция**:
1.  Принимает два аргумента: `supplier_prefix` (префикс поставщика, по умолчанию `'aliexpress'`) и `locale` (локаль, по умолчанию `'en'`).
2.  Создает словарь `params`, содержащий переданные аргументы.
3.  Инициализирует объект `Supplier` с использованием словаря `params` в качестве аргументов.
4.  Возвращает созданный объект `Supplier`.

**Параметры**:

*   `supplier_prefix` (str, optional): Префикс поставщика. По умолчанию `'aliexpress'`.
*   `locale` (str, optional): Локаль. По умолчанию `'en'`.

**Возвращает**:

*   `Supplier`: Объект поставщика, созданный с указанными параметрами.

**Примеры**:

```python
supplier = start_supplier(supplier_prefix='aliexpress', locale='en')
print(supplier)
```

```python
supplier = start_supplier(locale='ru')
print(supplier)
```
```python
supplier = start_supplier()
print(supplier)
