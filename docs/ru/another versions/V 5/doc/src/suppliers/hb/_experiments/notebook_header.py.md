# Модуль notebook_header

## Обзор

Модуль `notebook_header.py` предназначен для настройки окружения и импорта необходимых модулей для работы с проектом `hypotez`. Он добавляет корневую директорию проекта в `sys.path`, что позволяет импортировать модули из других частей проекта. Также модуль импортирует различные классы и функции, необходимые для работы с поставщиками, продуктами, категориями и другими компонентами проекта.

## Подробней

Этот модуль выполняет важную роль в настройке окружения для экспериментов и разработки. Он обеспечивает доступность необходимых модулей и классов, что упрощает процесс разработки и тестирования. В частности, он добавляет корневую директорию проекта в `sys.path`, что позволяет импортировать модули из других частей проекта без необходимости указывать относительные пути.

## Функции

### `start_supplier`

```python
def start_supplier(supplier_prefix: str, locale: str) -> Supplier | str:
    """ Старт поставщика """
    ...
```

**Как работает функция**:
Функция `start_supplier` инициализирует и возвращает объект поставщика (`Supplier`) на основе предоставленных префикса поставщика (`supplier_prefix`) и локали (`locale`). Если параметры не заданы, возвращает сообщение об ошибке.

**Параметры**:
- `supplier_prefix` (str): Префикс поставщика.
- `locale` (str): Локаль поставщика.

**Возвращает**:
- `Supplier | str`: Объект поставщика или сообщение об ошибке, если параметры не заданы.

**Примеры**:
```python
supplier = start_supplier('hb', 'ru')
if isinstance(supplier, Supplier):
    print('Поставщик успешно инициализирован')
else:
    print(supplier)
