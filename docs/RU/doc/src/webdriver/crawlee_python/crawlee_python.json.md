# crawlee_python.json

## Обзор

Этот файл представляет собой JSON-конфигурацию для веб-скрапинга с использованием Crawlee. Он определяет различные параметры, такие как максимальное количество запросов, тип браузера, настройки прокси, viewport и другие.

## Структура JSON

### `max_requests`

**Описание**: Максимальное количество параллельных запросов, которые может выполнять Crawlee.

**Тип данных**: `number`

**Пример**: `10`

### `headless`

**Описание**: Определяет, запускать ли браузер в безголовом режиме.

**Тип данных**: `boolean`

**Значения**:
- `true`: Браузер запускается в безголовом режиме.
- `false`: Браузер запускается с графическим интерфейсом.

**Пример**: `true`

### `browser_type`

**Описание**: Тип браузера, который будет использоваться для веб-скрапинга.

**Тип данных**: `string`

**Возможные значения**: `"chromium"`, `"firefox"`, `"webkit"`

**Пример**: `"chromium"`

### `options`

**Описание**: Список дополнительных параметров командной строки, передаваемых браузеру.

**Тип данных**: `array` of `string`

**Пример**:
```json
[
    "--disable-dev-shm-usage",
    "--no-sandbox",
    "--disable-gpu"
]
```

### `user_agent`

**Описание**: Строка User-Agent, используемая браузером для HTTP-запросов.

**Тип данных**: `string`

**Пример**: `"Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/96.0.4664.110 Safari/537.36"`

### `proxy`

**Описание**: Настройки прокси-сервера.

**Тип данных**: `object`

#### `enabled`

**Описание**: Включает или отключает использование прокси-сервера.

**Тип данных**: `boolean`

**Значения**:
- `true`: Прокси-сервер включен.
- `false`: Прокси-сервер отключен.

**Пример**: `false`

#### `server`

**Описание**: Адрес прокси-сервера в формате `http://<host>:<port>`.

**Тип данных**: `string`

**Пример**: `"http://proxy.example.com:8080"`

#### `username`

**Описание**: Имя пользователя для аутентификации на прокси-сервере.

**Тип данных**: `string`

**Пример**: `"user"`

#### `password`

**Описание**: Пароль для аутентификации на прокси-сервере.

**Тип данных**: `string`

**Пример**: `"password"`

### `viewport`

**Описание**: Настройки viewport браузера.

**Тип данных**: `object`

#### `width`

**Описание**: Ширина viewport в пикселях.

**Тип данных**: `number`

**Пример**: `1280`

#### `height`

**Описание**: Высота viewport в пикселях.

**Тип данных**: `number`

**Пример**: `720`

### `timeout`

**Описание**: Максимальное время ожидания в миллисекундах для выполнения запросов.

**Тип данных**: `number`

**Пример**: `30000`

### `ignore_https_errors`

**Описание**: Определяет, игнорировать ли ошибки SSL при выполнении запросов.

**Тип данных**: `boolean`

**Значения**:
- `true`: Ошибки SSL игнорируются.
- `false`: Ошибки SSL не игнорируются.

**Пример**: `false`
