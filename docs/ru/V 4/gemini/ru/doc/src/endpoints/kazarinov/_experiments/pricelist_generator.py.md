# Модуль pricelist_generator

## Обзор

Модуль `pricelist_generator.py` предназначен для экспериментов с созданием PDF-отчетов. Он использует класс `ReportGenerator` для генерации отчетов на основе JSON-данных и HTML-шаблонов. Модуль является частью подсистемы `kazarinov` и используется для создания отчетов в формате PDF.

## Подробней

Данный модуль предназначен для генерации PDF-отчетов на основе данных, полученных из JSON-файлов. Он использует класс `ReportGenerator` для преобразования данных в HTML и последующего создания PDF-файла. Расположение модуля в структуре проекта `hypotez` указывает на его использование в экспериментальных целях внутри подсистемы `kazarinov`. Основная задача модуля - автоматизация процесса создания отчетов, что позволяет упростить и ускорить процесс формирования документации на основе структурированных данных.

## Переменные

### `base_path`

- **Описание**: Определяет базовый путь к директории, содержащей данные для генерации отчета.
- **Тип**: `pathlib.Path`
- **Значение**: Путь к директории `external_data / 'kazarinov' / 'mexironim' / '24_11_24_05_29_40_543'`

### `data`

- **Описание**: Содержит данные, загруженные из JSON-файла.
- **Тип**: `dict`
- **Значение**: Данные, загруженные из файла `202410262326_he.json`, расположенного в `base_path`.

### `html_file`

- **Описание**: Определяет путь к HTML-файлу, который будет создан на основе данных.
- **Тип**: `pathlib.Path`
- **Значение**: Путь к файлу `202410262326_he.html`, расположенного в `base_path`.

### `pdf_file`

- **Описание**: Определяет путь к PDF-файлу, который будет сгенерирован.
- **Тип**: `pathlib.Path`
- **Значение**: Путь к файлу `202410262326_he.pdf`, расположенного в `base_path`.

### `r`

- **Описание**: Экземпляр класса `ReportGenerator`, используемый для создания отчета.
- **Тип**: `ReportGenerator`
- **Значение**: Объект класса `ReportGenerator`.
