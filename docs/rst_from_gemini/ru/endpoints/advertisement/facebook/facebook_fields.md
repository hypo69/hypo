Модуль `facebook_fields.py` отвечает за загрузку и хранение полей, необходимых для создания объявлений и событий в Facebook. Он загружает данные из файла `facebook_feilds.json`, расположенного в директории `gs.path.src/advertisement/facebok/`.  Полученные поля хранятся в виде атрибутов класса `FacebookFields`.

**Класс `FacebookFields`:**

Представляет собой контейнер для полей объявлений и событий Facebook.

* **`__init__(self)`:**
    * Инициализирует класс.
    * Вызывает метод `_payload()` для загрузки и обработки данных из файла.
