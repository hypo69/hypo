# Задача No 380
## Запутанные шахматы
## Условия

В шахматах на доске N x N ферзь может атаковать любую клетку в своей строке, столбце и обеих диагоналях.

Пусть Q(N) будет количеством способов размещения N ферзей на доске N x N так, чтобы ни один ферзь не атаковал другого. Например, Q(4) = 2, Q(5) = 10, Q(6) = 4.

Пусть F(N) будет количеством способов размещения N ферзей на доске N x N так, что для каждой пары ферзей на доске один ферзь может атаковать другого.
Например, F(4) = 4. Существует 4 способа размещения четырех ферзей так, чтобы каждый ферзь мог атаковать какого-нибудь другого ферзя.

Пусть G(N) = F(N)/Q(N).
Известно, что G(4) = 2, G(5) = 8.8, G(10) ≈ 25.7146.

Найдите G(1000). Округлите ответ до 4 знаков после запятой.
