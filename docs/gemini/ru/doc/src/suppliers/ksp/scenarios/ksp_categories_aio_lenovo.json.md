## Обзор

Файл `ksp_categories_aio_lenovo.json` содержит JSON-структуру, определяющую сценарии для категоризации All-in-One компьютеров Lenovo на сайте KSP. Каждый сценарий описывает конкретную модель, её URL на сайте KSP, состояние (новое), активность и соответствие категориям PrestaShop.

## Структура файла

Файл представляет собой JSON-объект со следующей структурой:
```json
{
  "scenarios": {
    "Название модели": {
      "brand": "LENOVO",
      "url": "URL на сайте KSP",
      "checkbox": false,
      "active": true,
      "condition": "new",
      "presta_categories": {
        "template": {
          "lenovo": "Название категории PrestaShop"
        }
      }
    }
  }
}
```

## Описание полей

### `scenarios`
- **Описание**: Корневой объект, содержащий все сценарии категоризации.
- **Тип**: `Object` (словарь, где ключ - название модели, значение - объект сценария)

### `Название модели`
- **Описание**: Ключ, представляющий название конкретной модели All-in-One компьютера Lenovo.
- **Тип**: `String`
- **Пример**: `"IdeaCentre AIO 3-24ITL"`

### `brand`
- **Описание**: Бренд производителя.
- **Тип**: `String`
- **Значение**: `"LENOVO"`

### `url`
- **Описание**: URL-адрес страницы товара на сайте KSP.
- **Тип**: `String`
- **Пример**: `"https://ksp.co.il/web/cat/1513..159..28794"`

### `checkbox`
- **Описание**: Флаг, указывающий на использование чекбокса (в данном случае всегда `false`).
- **Тип**: `Boolean`
- **Значение**: `false`

### `active`
- **Описание**: Флаг, указывающий на активность сценария.
- **Тип**: `Boolean`
- **Значение**: `true`

### `condition`
- **Описание**: Состояние товара.
- **Тип**: `String`
- **Значение**: `"new"`

### `presta_categories`
- **Описание**: Объект, содержащий информацию о категориях PrestaShop.
- **Тип**: `Object`
    -  `template`
       - **Описание**: Объект, содержащий соответствие между брендами и категориями PrestaShop.
       - **Тип**: `Object`
            -  `lenovo`
                - **Описание**: Название категории PrestaShop для Lenovo.
                - **Тип**: `String`
                - **Пример**: `"IdeaCentre 24"`
