# ИНСТРУКЦИЯ ПО ИСПОЛЬЗОВАНИЮ БОТ-ТРЕЙНЕРА ДЛЯ OPENAI

## Обзор

Данный документ описывает пошаговую инструкцию по использованию бота для обучения модели OpenAI через Discord.  Он включает в себя инструкции по подготовке данных, запуску обучения, мониторингу процесса и тестированию обученной модели.

## Команды бота

### `!train`

**Описание**: Команда для запуска процесса обучения модели.

**Параметры**:
- `"Your training data here"` (str):  Текстовые данные для обучения модели. Используется, если обучение осуществляется с использованием текстовых данных.
- `positive=True` (bool, optional): Признак, что данные относятся к положительной категории (по умолчанию `False`). Необязательно.
- `<file_path>` (str, optional):  Путь к файлу с данными для обучения. Используется, если обучение осуществляется с использованием файла. В этом случае параметр `positive` необязателен.


**Возвращает**:
- `Model training started. Job ID: <job_id>` (str): Сообщение об успешном запуске обучения и идентификатор задачи.

**Вызывает исключения**:
- `FileNotFoundError`: Если указан файл, но он не найден.
- `ValueError`: Если передан неверный тип данных.
- `APIError`: Если возникла ошибка при взаимодействии с API OpenAI.
- `OtherError`: Общее сообщение об ошибке при обработке команды.


### `!test`

**Описание**: Команда для тестирования обученной модели.

**Параметры**:
- `{"test_key": "test_value"}` (dict): Данные для тестирования. Должны быть переданы как JSON строка.

**Возвращает**:
- `Model prediction: <prediction_result>` (str):  Результат предсказания модели.


**Вызывает исключения**:
- `ValueError`: Если передан неверный тип данных.
- `APIError`: Если возникла ошибка при взаимодействии с API OpenAI.
- `OtherError`: Общее сообщение об ошибке при обработке команды.

### `!archive`

**Описание**: Команда для архивирования файлов.

**Параметры**:
- `<directory_path>` (str): Путь к каталогу, который нужно архивировать.

**Возвращает**:
- `Archiving completed.` (str): Сообщение об успешном архивировании.

**Вызывает исключения**:
- `FileNotFoundError`: Если указанный каталог не найден.
- `IOError`: Если возникли проблемы с чтением или записью файла.
- `OtherError`: Общее сообщение об ошибке при обработке команды.

### `!select_dataset`

**Описание**: Команда для выбора набора данных для обучения.

**Параметры**:
- `<path_to_dir_positive>` (str): Путь к каталогу с данными для положительной категории.
- `positive=True` (bool, optional): Признак, что данные относятся к положительной категории (по умолчанию `False`).

**Возвращает**:
- `Dataset selected successfully.` (str): Сообщение об успешном выборе набора данных.

**Вызывает исключения**:
- `FileNotFoundError`: Если указанный каталог не найден.
- `ValueError`: Если передан неверный тип данных.
- `OtherError`: Общее сообщение об ошибке при обработке команды.


### `!ask`

**Описание**: Команда для получения ответа от обученной модели.

**Параметры**:
- `<question>` (str): Вопрос для модели.

**Возвращает**:
- `Model response: <answer>` (str): Ответ модели на вопрос.

**Вызывает исключения**:
- `ValueError`: Если передан неверный тип данных.
- `APIError`: Если возникла ошибка при взаимодействии с API OpenAI.
- `OtherError`: Общее сообщение об ошибке при обработке команды.
