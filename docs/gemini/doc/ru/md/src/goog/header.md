# Модуль hypotez/src/goog/header.py

## Обзор

Данный модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`. Он также предоставляет доступ к информации о проекте, такой как имя, версия, описание и автор.

## Функции

### `set_project_root`

**Описание**: Определяет корневую директорию проекта, начиная с текущего файла. Поиск осуществляется вверх по директориям до тех пор, пока не будет найдена директория, содержащая один из файлов, указанных в параметре `marker_files`.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для определения корня проекта. По умолчанию используется кортеж `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Объект `Path` с путем к корневой директории проекта, если она найдена. В противном случае возвращает директорию, где расположен текущий файл.

**Вызывает исключения**:

- Нет.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root()`.

**Тип**: `Path`


## Загрузка настроек

**Описание**: Загружает настройки из файла `settings.json` в переменную `settings`.

**Обработка ошибок**:

- Используются блоки `try...except` для обработки ошибок `FileNotFoundError` и `json.JSONDecodeError` при чтении файла `settings.json`. В случае ошибки переменная `settings` остается `None`.

## Загрузка описания проекта

**Описание**: Загружает описание проекта из файла `README.MD` в переменную `doc_str`.

**Обработка ошибок**:

- Используются блоки `try...except` для обработки ошибок `FileNotFoundError` и `json.JSONDecodeError` при чтении файла `README.MD`. В случае ошибки переменная `doc_str` остается `None`.

## Переменные проекта

**Описание**: Переменные, содержащие информацию о проекте. Значения берутся из переменной `settings` или устанавливаются по умолчанию, если `settings` равен `None`.
