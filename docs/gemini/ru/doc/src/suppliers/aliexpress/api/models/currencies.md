# Модуль `hypotez/src/suppliers/aliexpress/api/models/currencies.py`

## Обзор

Модуль `currencies.py` содержит класс `Currency`, который определяет константы для различных валют.

## Классы

### `Currency`

**Описание**: Класс `Currency` содержит константы для различных валют.

**Атрибуты**:

- `USD`: Константа для валюты доллар США.
- `GBP`: Константа для валюты британский фунт.
- `CAD`: Константа для валюты канадский доллар.
- `EUR`: Константа для валюты евро.
- `UAH`: Константа для валюты гривна.
- `MXN`: Константа для валюты мексиканский песо.
- `TRY`: Константа для валюты турецкая лира.
- `RUB`: Константа для валюты российский рубль.
- `BRL`: Константа для валюты бразильский реал.
- `AUD`: Константа для валюты австралийский доллар.
- `INR`: Константа для валюты индийская рупия.
- `JPY`: Константа для валюты японская йена.
- `IDR`: Константа для валюты индонезийская рупия.
- `SEK`: Константа для валюты шведская крона.
- `KRW`: Константа для валюты южнокорейская вона.
- `ILS`: Константа для валюты израильский шекель.
