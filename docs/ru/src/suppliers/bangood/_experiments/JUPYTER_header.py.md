# Модуль _experiments (JUPYTER_header.py)

## Обзор

Модуль содержит набор экспериментов, связанных с поставщиком Banggood. Он включает в себя импорты необходимых библиотек и классов для работы с веб-драйвером, поставщиками, продуктами, категориями и другими утилитами. Также, модуль включает функцию для запуска поставщика.

## Подробнее

Этот модуль, по-видимому, является частью процесса исследования и разработки для интеграции с поставщиком Banggood. Он содержит импорты и функции, необходимые для взаимодействия с веб-сайтом Banggood, извлечения информации о продуктах и категориях, а также для сохранения этой информации в структурированном формате.  Содержит вспомогательные функции и классы, предназначенные для облегчения процесса сбора и обработки данных с сайта поставщика. Расположение модуля в директории `_experiments` указывает на то, что код находится в стадии разработки и экспериментов.

## Импортированные модули и классы

*   **sys**: Для работы с системными параметрами и функциями.
*   **os**: Для работы с операционной системой, например, для получения текущей директории.
*   **Path (pathlib)**: Для работы с путями к файлам и директориям.
*   **json**: Для работы с данными в формате JSON.
*   **re**: Для работы с регулярными выражениями.
*   **Driver (src.webdriver.driver)**: Класс для управления веб-драйвером.
*   **Supplier (src.suppliers)**: Класс, представляющий поставщика.
*   **Product (src.product)**: Класс, представляющий продукт.
*   **ProductFields (src.product)**: Класс, определяющий поля продукта.
*   **Category (src.category)**: Класс, представляющий категорию продукта.
*   **StringFormatter (src.utils)**: Класс для форматирования строк.
*   **StringNormalizer (src.utils)**: Класс для нормализации строк.
*   **pprint (src.utils.printer)**: Функция для "красивой" печати данных.
*   **PrestaProduct (src.endpoints.PrestaShop)**: Класс для работы с продуктами в PrestaShop.

## Переменные

## Функции

### `start_supplier`

```python
def start_supplier(supplier_prefix: str = 'aliexpress', locale: str = 'en') -> Supplier:
    """ Старт поставщика """
```

**Назначение**: Инициализирует и возвращает объект класса `Supplier` с заданными параметрами.

**Параметры**:

*   `supplier_prefix` (str): Префикс поставщика (например, 'aliexpress'). По умолчанию 'aliexpress'.
*   `locale` (str): Языковая локаль (например, 'en'). По умолчанию 'en'.

**Возвращает**:

*   `Supplier`: Объект класса `Supplier`, инициализированный с заданными параметрами.

**Как работает функция**:

1.  Создает словарь `params`, содержащий параметры `supplier_prefix` и `locale`.
2.  Инициализирует объект класса `Supplier` с использованием словаря `params` в качестве аргументов.
3.  Возвращает созданный объект `Supplier`.

**ASCII flowchart функции**:

```
Начало
  ↓
Создание словаря params
  ↓
Инициализация Supplier с params
  ↓
Возврат Supplier
  ↓
Конец
```

supplier2 = start_supplier(supplier_prefix='bangood', locale='ru')
