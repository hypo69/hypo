# Модуль hypotez/src/endpoints/advertisement/facebook/scenarios/switch_account.py

## Обзор

Модуль содержит функцию `switch_account`, предназначенную для переключения между учетными записями Facebook в веб-приложении.  Функция ищет и взаимодействует с элементом "Переключить аккаунт",  используя предопределенные локаторы.


## Функции

### `switch_account`

**Описание**: Функция переключения между аккаунтами Facebook. Ищет и нажимает кнопку "Переключить аккаунт", если она доступна.

**Параметры**:

- `driver` (Driver): Объект драйвера веб-драйвера, используемый для взаимодействия с веб-страницей.

**Возвращает**:
-  None: Функция не возвращает явных значений.


**Вызывает исключения**:
-  Возможные исключения, генерируемые методом `driver.execute_locator`. (Необходимо добавить конкретные типы исключений, которые может генерировать `execute_locator`.)


## Переменные

### `MODE`

**Описание**:  Строковая переменная, определяющая режим работы (например, 'dev', 'prod'). Текущее значение равно 'dev'.


### `locator`

**Описание**:  Объект `SimpleNamespace`, содержащий локаторы элементов страницы для переключения аккаунтов. Локаторы загружаются из файла `post_message.json` в папке `locators`.


## Импорты

- `Path`: для работы с путями к файлам.
- `SimpleNamespace`: для создания объекта с атрибутами.
- `gs`: (Скорее всего, модуль, который предоставляет пути к ресурсам).
- `Driver`: (Вероятно, класс из модуля `webdriver.driver` для взаимодействия с браузером).
- `j_loads_ns`: функция для загрузки данных из JSON-файла в структуру `SimpleNamespace`.
