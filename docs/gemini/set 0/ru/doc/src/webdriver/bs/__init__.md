# Модуль `hypotez/src/webdriver/bs/__init__.py`

## Обзор

Этот модуль предоставляет базовые функции и настройки для работы с веб-драйвером в контексте проекта Hypotez. Он импортирует классы и функции из модуля `src.webdriver.bs`.

## Переменные

### `MODE`

**Описание**: Переменная, определяющая режим работы (например, 'dev' или 'prod').

**Значение**: `'dev'`


## Импорты

### `from .bs import BS`

**Описание**: Импортирует класс `BS` из модуля `src.webdriver.bs`.


##  Содержание

Модуль `hypotez/src/webdriver/bs/__init__.py` содержит конфигурацию для веб-драйвера, а также импорт класса `BS` из подмодуля `bs`.


##  Список элементов


* Переменная `MODE`
* Импорт `from .bs import BS`


## Дополнительная информация

Этот модуль служит в качестве точки входа для использования компонентов веб-драйвера. Подробное описание класса `BS` и его методов содержится в документации соответствующего модуля `bs`.
