# Модуль `affiliate_links_shortener_scenario`

## Обзор

Модуль предназначен для сокращения ссылок через веб-браузер, используя драйвер браузера `Driver`.

## Функции

### `get_short_affiliate_link`

**Описание**:
Скрипт для генерации сокращенной партнерской ссылки.

**Параметры**:
- `d` (`Driver`): Экземпляр драйвера браузера.
- `url` (`str`): Полный URL для сокращения.

**Возвращает**:
- `str`: Сокращенный URL.

**Вызывает исключения**:
- `ValueError`: Возникает, если не удалось получить короткий URL или если получен неправильный URL.
