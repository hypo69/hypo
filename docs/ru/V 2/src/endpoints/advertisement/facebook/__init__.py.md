# Модуль `src.endpoints.advertisement.facebook`

## Обзор

Модуль `src.endpoints.advertisement.facebook` предоставляет функциональность для работы с Facebook API в контексте рекламы. Он содержит классы для управления рекламными кампаниями и получения данных, а также вспомогательные классы для работы с полями и промоушеном.
