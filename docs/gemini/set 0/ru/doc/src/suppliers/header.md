# Модуль `hypotez/src/suppliers/header.py`

## Обзор

Данный модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`.  Он также предоставляет доступ к информации о проекте, такой как название, версия, документация, автор и др.

## Функции

### `set_project_root`

**Описание**: Определяет корневую директорию проекта, начиная от директории текущего файла и идя вверх по иерархии директорий. Поиск останавливается на первой директории, содержащей файлы или папки, указанные в списке `marker_files`. Если корневая директория не найдена, возвращается директория, в которой расположен скрипт. Если корневая директория не находится в `sys.path`, она добавляется в начало.

**Параметры**:

- `marker_files` (tuple): Кортеж с именами файлов или директорий, используемых для определения корневой директории проекта. По умолчанию: (`'pyproject.toml'`, `'requirements.txt'`, `.git`).

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена; иначе путь к директории, где находится скрипт.

**Пример использования**:

```python
root_path = set_project_root()
print(root_path)
```

### `__root__`


**Описание**: Переменная, содержащая корневой путь проекта, который был получен вызовом `set_project_root()`.


**Значение**:
- `Path`: Путь к корневой директории проекта.



## Загрузка настроек

**Описание**: Модуль загружает настройки из файла `settings.json` в переменную `settings` и  информацию из файла `README.MD` в переменную `doc_str`.  Если файлы не найдены или некорректно отформатированы, переменные остаются `None`.

**Обработка ошибок**:

- `FileNotFoundError`: Если файлы `settings.json` или `README.MD` не найдены.
- `json.JSONDecodeError`: Если файл `settings.json` не содержит корректный JSON.

## Переменные проекта

**Описание**: Переменные, содержащие информацию о проекте. Эти переменные инициализируются данными из `settings.json`. Если `settings.json` отсутствует, используются значения по умолчанию.

**Переменные**:

- `__project_name__` (str): Название проекта.
- `__version__` (str): Версия проекта.
- `__doc__` (str):  Документация проекта.
- `__details__` (str): Подробная информация о проекте.
- `__author__` (str): Автор проекта.
- `__copyright__` (str): Авторские права.
- `__cofee__` (str): Ссылка для поддержки разработчика.


## Примечания

- Модуль использует `gs.path.root` для доступа к корневой директории, предполагая, что модуль `gs` уже импортирован и содержит соответствующую функциональность.
- Этот модуль предполагает, что `gs` и `pathlib` успешно импортированы.
- Модуль полагается на наличие файлов `pyproject.toml`, `requirements.txt`, `.git` в корне проекта, или же на имена директорий, указанные в `marker_files`, для поиска корневой директории.
- Обработка исключений (`ex`) используется для `FileNotFoundError` и `json.JSONDecodeError`.
