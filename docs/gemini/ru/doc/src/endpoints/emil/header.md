# Модуль `hypotez/src/endpoints/emil/header.py`

## Обзор

Этот модуль содержит функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и получения документации из файла `README.MD`. Он также инициализирует переменные, содержащие информацию о проекте (название, версия, автор, описание, и др.).


## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная с каталога текущего файла, ищет вверх по дереву каталогов, останавливаясь на первом каталоге, содержащем файлы или директории-маркеры.

**Параметры**:

- `marker_files` (tuple): Кортеж имён файлов или директорий, используемых для определения корневой директории проекта.

**Возвращает**:

- `Path`: Путь к корневому каталогу проекта, если он найден, в противном случае - путь к директории, где расположен текущий файл.

**Вызывает исключения**:

- Нет.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта.


### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `src/settings.json`.

**Значение по умолчанию**: `None`.

**Возможные типы исключений**:

- `FileNotFoundError`: Если файл `settings.json` не найден.
- `json.JSONDecodeError`: Если содержимое файла `settings.json` не является корректным JSON.

### `doc_str`

**Описание**: Строка с содержимым файла `README.MD`.

**Значение по умолчанию**: `None`.

**Возможные типы исключений**:

- `FileNotFoundError`: Если файл `README.MD` не найден.
- `json.JSONDecodeError`: Если содержимое файла `README.MD` имеет неверный формат.

### `__project_name__`

**Описание**: Название проекта.

**Значение по умолчанию**: `hypotez`.


### `__version__`

**Описание**: Версия проекта.

**Значение по умолчанию**: `''`.


### `__doc__`

**Описание**: Документация к проекту, полученная из файла `README.MD`.

**Значение по умолчанию**: `''`.


### `__details__`

**Описание**: Подробная информация о проекте.

**Значение по умолчанию**: `''`.


### `__author__`

**Описание**: Автор проекта.

**Значение по умолчанию**: `''`.


### `__copyright__`

**Описание**: Авторские права на проект.

**Значение по умолчанию**: `''`.


### `__cofee__`

**Описание**: Ссылка для поддержки разработчика.

**Значение по умолчанию**: "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69".
