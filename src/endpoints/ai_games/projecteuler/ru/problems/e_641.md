# Задача No 641
## Разложение на множители полиномов второго порядка
## Условия
Рассмотрим полином второго порядка P(x) = x² + ax + b, где a и b — целые числа.

Определим операцию факторизации F(P) как набор всех пар (c, d) целых чисел, таких что P(x) = (x + c)(x + d).

Например, F(x² + 2x - 15) = {(-5, 3), (3, -5), (-3, 5), (5, -3)}, так как x² + 2x - 15 = (x - 5)(x + 3) = (x + 3)(x - 5) = (x - 3)(x + 5) = (x + 5)(x - 3).

Пусть S(n) будет суммой всех |c| значений из всех пар (c, d) в наборах факторизаций F(P) для всех полиномов P(x) = x² + ax + b, где |a| ≤ n и |b| ≤ n.

Например:
* S(1) = 10.
* S(10) = 1704.
* S(100) = 182500.

Найдите S(10 000 000).
