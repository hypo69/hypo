# Модуль hypotez/src/suppliers/aliexpress/scenarios/login.py

## Обзор

Этот модуль содержит функцию `login`, предназначенную для входа на сайт AliExpress через Selenium WebDriver.  Функция взаимодействует с объектом `Supplier`, содержащим необходимые данные для входа, включая драйвер браузера и локаторы элементов на странице.

## Функции

### `login`

**Описание**: Функция осуществляет вход на AliExpress через Selenium WebDriver.

**Параметры**:
- `s` (`Supplier`): Объект класса `Supplier` с запущенным драйвером браузера и данными для входа.

**Возвращает**:
- `bool`: `True`, если вход успешен, иначе возвращает `False` (в текущей реализации возвращает `True` для отладки).


**Возможные исключения**:
- Возможны исключения, связанные с работой Selenium WebDriver (например, `NoSuchElementException`, `TimeoutException`).
- Не описаны возможные исключения, связанные с некорректным объектом `Supplier` или отсутствием локаторов элементов.

**Детали реализации**:
Функция получает доступ к драйверу браузера (`_d`) и локаторам элементов (`_l`) из объекта `Supplier`. Затем выполняет следующие действия:
1. Переходит на страницу AliExpress.
2. Принимает cookies.
3. Открывает форму входа.
4. Взаимодействует с полями для ввода email и пароля, используя локаторы элементов.
5. Нажимает кнопку входа.
6. Возвращает `True` для отладки.

**Примечания**:
-  Необходимо реализовать обработку возможных ошибок (например, `False` от `execute_locator`) и логику, которая будет возвращать `False` в случае неудачного входа.
-  Метод `set_language_currency_shipto` пока не используется и нуждается в реализации.
