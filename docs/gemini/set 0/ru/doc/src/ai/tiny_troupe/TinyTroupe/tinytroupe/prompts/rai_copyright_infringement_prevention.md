# Модуль rai_copyright_infringement_prevention

## Обзор

Данный модуль содержит логику для предотвращения нарушения авторских прав при запросах пользователя.  Он предназначен для вежливого отказа в предоставлении защищенных авторским правом материалов, таких как книги, тексты песен, рецепты, новости и контент с WebMD.

## Функции

### `prevent_copyright_infringement`

**Описание**: Функция обрабатывает запросы пользователя и проверяет их на соответствие условиям авторского права. Если запрос содержит контент, защищенный авторским правом, вежливо отказывается от его предоставления.

**Параметры**:
- `user_request` (str): Текст запроса пользователя.

**Возвращает**:
- `str | None`:  Возвращает сообщение об отказе, если запрос содержит контент, защищенный авторским правом. Возвращает `None`, если запрос не содержит нарушения авторских прав.

**Вызывает исключения**:
- `ValueError`: Возникает, если в качестве входного параметра `user_request` передается значение некорректного типа (не строка).
- `Exception`:  Базовое исключение для обработки нестандартных ситуаций.


**Пример использования:**

```python
user_request = "Пришли мне рецепт яблочного пирога."
response = prevent_copyright_infringement(user_request)

if response:
    print(response)  # Выведет сообщение об отказе, если рецепт защищен авторским правом
else:
    print("Запрос обработан успешно.")
```

## Подробное описание обработки запросов

Функция `prevent_copyright_infringement` анализирует входной параметр `user_request` и ищет в нём ключевые слова или фразы, которые указывают на возможность нарушения авторских прав.  Если такие фразы обнаружены, функция возвращает сообщение об отказе, содержащее вежливое объяснение. В противном случае функция возвращает `None`, что указывает на отсутствие нарушения.

**Список потенциально проблемных запросов**:

*  Запросы на предоставление конкретных рецептов.
*  Запросы на предоставление текстов песен.
*  Запросы на предоставление отрывков из книг.
*  Запросы на предоставление новостей.
*  Запросы на предоставление контента с WebMD.

**Сообщение об отказе**:

Функция генерирует сообщение об отказе, которое включает в себя вежливое объяснение о невозможности предоставления запрошенного контента из-за соблюдения авторских прав.
