# Color Code Resistor Calculator

## Описание

Этот Python скрипт (`color_code_resistor.py`) предназначен для расчета сопротивления резистора по его цветовой маркировке. 
Пользователь вводит три цвета, соответствующие полосам на резисторе, и скрипт выводит значение сопротивления в омах и килоомах.

## Использование

1.  **Клонируйте репозиторий** (если это необходимо).
2.  **Запустите скрипт `color_code_resistor.py`:**
    ```bash
    python color_code_resistor.py
    ```
3.  **Следуйте инструкциям в консоли:**
    Скрипт попросит вас по очереди ввести три цвета, соответствующие полосам на резисторе.
4.  **Просмотрите результат:**
    Скрипт выведет значение сопротивления в омах и килоомах.

## Пример работы

```
Enter the 1st color: red
Enter the 2nd color: violet
Enter the 3rd color: brown

The Resistor Value Is:
270 and in Kiloohm: 0.27kΩ
```

## Как это работает

Скрипт использует список цветов для определения числовых значений, соответствующих каждой цветовой полосе. Первые две полосы определяют значимые цифры, а третья полоса является множителем.

## Список используемых цветов

-   `black` - 0
-   `brown` - 1
-   `red` - 2
-   `orange` - 3
-   `yellow` - 4
-   `green` - 5
-   `blue` - 6
-   `violet` - 7
-   `grey` - 8
-   `white` - 9
