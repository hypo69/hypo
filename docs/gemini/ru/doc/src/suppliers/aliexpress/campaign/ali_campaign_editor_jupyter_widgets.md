# Модуль `hypotez/src/suppliers/aliexpress/campaign/ali_campaign_editor_jupyter_widgets.py`

## Обзор

Данный модуль предоставляет виджеты Jupyter для редактирования кампаний AliExpress.  Он позволяет взаимодействовать с кампаниями, категориями и языками, инициализировать редакторы, сохранять кампании и отображать продукты.

## Классы

### `JupyterCampaignEditorWidgets`

**Описание**: Класс `JupyterCampaignEditorWidgets` предоставляет виджеты для взаимодействия с редактором кампаний AliExpress в Jupyter Notebook. Он позволяет выбрать кампанию, категорию и язык, а также выполнять такие действия, как инициализация редактора, сохранение кампании и отображение продуктов.

**Атрибуты класса**:

- `language`: (str) Язык.
- `currency`: (str) Валюта.
- `campaign_name`: (str) Название кампании.
- `category_name`: (str) Название категории.
- `category`: (SimpleNamespace) Объект категории.
- `campaign_editor`: (AliCampaignEditor) Редактор кампании.
- `products`: (list[SimpleNamespace]) Список продуктов.

**Методы**:

#### `__init__(self)`

**Описание**: Инициализирует виджеты и настраивает редактор кампании. Устанавливает значения по умолчанию и обратные вызовы для виджетов.

#### `initialize_campaign_editor(self, _)`

**Описание**: Инициализирует редактор кампании.

**Параметры**:

- `_`: (без значения) Неиспользуемый аргумент, необходимый для обратного вызова кнопки.

**Действия**:
Настраивает редактор кампании на основе выбранной кампании и категории.

#### `update_category_dropdown(self, campaign_name: str)`

**Описание**: Обновляет выпадающий список категорий на основе выбранной кампании.

**Параметры**:

- `campaign_name` (str): Название кампании.

**Действия**:
Обновляет список категорий в выпадающем списке.

#### `on_campaign_name_change(self, change: dict[str, str])`

**Описание**: Обрабатывает изменения в выпадающем списке кампаний.

**Параметры**:

- `change` (dict[str, str]): Словарь смены, содержащий новое значение.

**Действия**:
Обновляет переменную `campaign_name` и вызывает `update_category_dropdown`. Также повторно инициализирует редактор кампании (`initialize_campaign_editor`).

#### `on_category_change(self, change: dict[str, str])`

**Описание**: Обрабатывает изменения в выпадающем списке категорий.

**Параметры**:

- `change` (dict[str, str]): Словарь смены, содержащий новое значение.

**Действия**:
Обновляет переменную `category_name` и повторно инициализирует редактор кампании (`initialize_campaign_editor`).

#### `on_language_change(self, change: dict[str, str])`

**Описание**: Обрабатывает изменения в выпадающем списке языка.

**Параметры**:

- `change` (dict[str, str]): Словарь смены, содержащий новое значение.

**Действия**:
Разделяет строку языка на язык и валюту и обновляет соответствующие переменные.  Повторно инициализирует редактор кампании (`initialize_campaign_editor`).

#### `save_campaign(self, _)`

**Описание**: Сохраняет кампанию и её категории.

**Параметры**:

- `_`: (без значения) Неиспользуемый аргумент, необходимый для обратного вызова кнопки.

**Действия**:
Сохраняет выбранную кампанию, категорию и язык в редакторе кампании.  Обрабатывает исключения при сохранении.

#### `show_products(self, _)`

**Описание**: Отображает продукты в выбранной категории.

**Параметры**:

- `_`: (без значения) Неиспользуемый аргумент, необходимый для обратного вызова кнопки.

**Действия**:
Создает редактор кампании с предоставленными данными и отображает продукты из выбранной категории. Обрабатывает исключения при отображении продуктов.


#### `open_spreadsheet(self, _)`

**Описание**: Открывает Google Spreadsheet в браузере.

**Параметры**:

- `_`: (без значения) Неиспользуемый аргумент, необходимый для обратного вызова кнопки.

**Действия**:
Открывает ссылку на Google Spreadsheet в браузере.

#### `setup_callbacks(self)`

**Описание**: Настраивает обратные вызовы для виджетов.

#### `display_widgets(self)`

**Описание**: Отображает виджеты для взаимодействия в Jupyter Notebook.
