# Модуль `background.js`

## Обзор

Этот файл содержит код обработчика события `onInstalled` для расширения Chrome, интегрированного с OpenAI.  Он выводит сообщение в консоль, когда расширение установлено.

## Функции

### `chrome.runtime.onInstalled.addListener`

**Описание**: Регистрирует обработчик события `onInstalled`.

**Параметры**:

- `callback`:  Функция, которая будет вызвана при установке расширения. В данном случае анонимная функция.

**Возвращает**:
- Не имеет возвращаемого значения.

**Вызывает исключения**:
- Не имеет описанных исключений.
