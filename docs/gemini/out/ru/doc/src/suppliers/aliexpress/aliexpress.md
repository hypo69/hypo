# Модуль aliexpress

## Обзор

Модуль `aliexpress` предоставляет класс `Aliexpress`, который интегрирует функциональность из классов `Supplier`, `AliRequests` и `AliApi` для работы с AliExpress. Он позволяет осуществлять взаимодействие с AliExpress, используя различные режимы работы (без веб-драйвера, с использованием Chrome, Mozilla, Edge или системного веб-драйвера) и настраивать языковые и валютные параметры.

## Классы

### `Aliexpress`

**Описание**: Базовый класс для работы с AliExpress. Объединяет функциональность классов `Supplier`, `AliRequests` и `AliApi`, обеспечивая взаимодействие с AliExpress.

**Методы**:

- `__init__`: Инициализирует класс `Aliexpress`.

## Методы класса `Aliexpress`

### `__init__`

**Описание**: Инициализирует класс Aliexpress.

**Параметры**:

- `webdriver` (bool | str, опционально): Режим работы с веб-драйвером. Допустимые значения:
    - `False` (по умолчанию): Нет веб-драйвера.
    - `'chrome'`: Использование веб-драйвера Chrome.
    - `'mozilla'`: Использование веб-драйвера Mozilla.
    - `'edge'`: Использование веб-драйвера Edge.
    - `'default'`: Использование системного веб-драйвера.
- `locale` (str | dict, опционально): Настройки языка и валюты для скрипта. По умолчанию `{ 'EN': 'USD' }`.
- `*args`: Дополнительные позиционные аргументы.
- `**kwargs`: Дополнительные именованные аргументы.

**Возвращает**:
  - Нет значения возврата.

**Вызывает исключения**:
  - Нет описания исключений.
