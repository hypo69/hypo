grandadvance_categories_keyboards_hp
===================================

This file contains a JSON object describing HP keyboards and mice, along with their associated information.  Each key represents a product, and the value is a dictionary containing details like brand, URL, checkbox status, activity, condition, and PrestaShop categories.
