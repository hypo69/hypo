# src.suppliers.aliexpress.api.models.languages

## Обзор

Модуль `src.suppliers.aliexpress.api.models.languages` содержит класс `Language`, который определяет константы для различных языков, используемых в API AliExpress.

## Подробней

Этот модуль предоставляет централизованный способ хранения и использования кодов языков, что упрощает поддержку и обновление списка поддерживаемых языков. Использование класса `Language` позволяет избежать дублирования строковых литералов и обеспечивает типобезопасность при работе с языковыми кодами.

## Классы

### `Language`

**Описание**: Класс, содержащий константы для представления кодов различных языков.

**Методы**:
- Нет

**Параметры**:
- Нет

**Примеры**:
```python
from src.suppliers.aliexpress.api.models.languages import Language

# Использование константы для английского языка
english_code = Language.EN
print(english_code)  # Вывод: EN

# Использование константы для русского языка
russian_code = Language.RU
print(russian_code)  # Вывод: RU
