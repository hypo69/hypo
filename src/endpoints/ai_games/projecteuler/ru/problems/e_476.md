# Задача No 476
## Точки внутри треугольника

## Условия
Пусть T(a, b, c) обозначает треугольник с координатами вершин (0,0), (a,0) и (b,c).

Пусть N(a, b, c) будет количеством точек с целочисленными координатами, находящихся строго внутри треугольника T(a, b, c) (т.е. не на сторонах треугольника).

Например, N(3, 2, 1) = 1.

Пусть M(n) будет максимумом N(a, b, c) для 0 < a, b, c ≤ n.

Например, M(20) = 96.

Найдите M(1000).
