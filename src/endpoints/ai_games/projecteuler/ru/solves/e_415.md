## Ответ на Задачу No 415: Игра с костями

### 1. Анализ задачи и решение
**Понимание задачи:**
*   Дано два множества, каждое из которых получено бросанием `n` игральных костей.
*   Необходимо найти ожидаемое количество совпадающих значений между этими двумя множествами.
*   `F(n)` - ожидаемое количество совпадений при бросании `n` костей дважды.
*   Нам нужно вычислить `F(10000)` и округлить результат до 10 знаков после запятой.

**Решение:**
1.  **Вероятность совпадения:** Для каждого значения от 1 до 6, вероятность того, что оно выпадет при броске кости, равна 1/6. Вероятность того, что конкретное значение `x`  появится хотя бы один раз в `n` бросках костей равна `1 - (5/6)^n`.
2.  **Независимость множеств:** Так как множества A и B формируются независимо, вероятность того, что значение `x` присутствует и в множестве A, и в множестве B, равна `(1 - (5/6)^n) * (1 - (5/6)^n)`.
3.  **Ожидание совпадений для одного значения:** Если два множества содержат число `x`, то  это является совпадением. Для всех `x` от 1 до 6, ожидаемое количество совпадений равно вероятности `x` встретиться в обоих множествах.
4.  **Суммарное ожидание:** Так как  все 6 значений независимы, то итоговое ожидание равно сумме ожиданий для каждого значения от 1 до 6, то есть `F(n) = 6 * (1 - (5/6)^n)^2`.

### 2. Алгоритм решения
1.  Начать
2.  Ввести значение `n = 10000`
3.  Вычислить вероятность `p = 1 - (5/6)^n`
4.  Вычислить ожидаемое количество совпадений `expected_matches = 6 * p * p`
5.  Округлить `expected_matches` до 10 знаков после запятой
6.  Вывести `expected_matches`
7.  Конец

### 3. Реализация на Python 3.12
```python
def calculate_expected_matches(number_of_dice):
    """
    Calculates the expected number of matches between two sets of dice rolls.

    Args:
        number_of_dice: The number of dice rolled in each set.

    Returns:
        The expected number of matches, rounded to 10 decimal places.
    """
    probability_of_at_least_one = 1 - (5/6) ** number_of_dice
    expected_matches = 6 * probability_of_at_least_one ** 2
    return round(expected_matches, 10)

# Пример использования:
n_value = 10000
result = calculate_expected_matches(n_value)
print(result)
```
