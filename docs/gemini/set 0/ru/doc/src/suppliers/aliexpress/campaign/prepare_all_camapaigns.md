# Модуль prepare_all_camapaigns

## Обзор

Модуль `prepare_all_camapaigns` предназначен для проверки и, при необходимости, создания affiliate для рекламных кампаний на AliExpress.  Если кампании не существует, то она создаётся.

## Функции

### `process_all_campaigns`

**Описание**: Функция обрабатывает все рекламные кампании.  Она выполняет проверку существования каждой кампании и, если необходимо, создает новые affiliate для них.

- Любые исключения, генерируемые функциями, используемыми внутри `process_all_campaigns`.
