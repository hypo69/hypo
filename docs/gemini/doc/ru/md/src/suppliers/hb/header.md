# Модуль hypotez/src/suppliers/hb/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и получения документации из файла `README.MD`.  Модуль использует библиотеку `packaging.version` для работы с версиями.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с текущей директории.  Ищет файлы или директории в `marker_files`, указанные в аргументе.

**Параметры**:
- `marker_files` (tuple): Кортеж с именами файлов или директорий, используемых для определения корневой директории проекта. По умолчанию используется кортеж ('pyproject.toml', 'requirements.txt', '.git').


**Возвращает**:
- `Path`: Путь к корневой директории проекта, если найдена. В противном случае возвращает путь к директории, где находится текущий скрипт.

**Вызывает исключения**:
-  Не вызывает исключений.


### Загрузка настроек

**Описание**: Загружает настройки из файла `settings.json` в переменную `settings`.

**Возвращает**:
- `dict`: Словарь со значениями настроек. Возвращает None если файл не найден или содержимое не в формате JSON.

**Вызывает исключения**:
- `FileNotFoundError`: Если файл `settings.json` не найден.
- `json.JSONDecodeError`: Если файл `settings.json` содержит некорректный JSON.


### Загрузка документации

**Описание**: Загружает содержимое файла `README.MD` в переменную `doc_str`.

**Возвращает**:
- `str`: Содержимое файла `README.MD`.  Возвращает None, если файл не найден или при ошибке чтения.

**Вызывает исключения**:
- `FileNotFoundError`: Если файл `README.MD` не найден.
- `json.JSONDecodeError`:  Если возникла проблема с чтением файла.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученной с помощью функции `set_project_root`.

**Тип**: `Path`

### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json`.

**Тип**: `dict` или `None`

### `doc_str`

**Описание**: Содержимое файла `README.MD`.

**Тип**: `str` или `None`

### `__project_name__`

**Описание**: Название проекта. Получено из настроек, если доступны. Иначе - 'hypotez'.

**Тип**: `str`

### `__version__`

**Описание**: Версия проекта. Получено из настроек, если доступны. Иначе - пустая строка.

**Тип**: `str`

### `__doc__`

**Описание**: Документация проекта.  Получена из файла `README.MD`, если доступна. Иначе - пустая строка.

**Тип**: `str`

### `__details__`

**Описание**: Подробная информация о проекте.

**Тип**: `str`

### `__author__`

**Описание**: Автор проекта. Получено из настроек, если доступны. Иначе - пустая строка.

**Тип**: `str`

### `__copyright__`

**Описание**: Авторские права на проект. Получено из настроек, если доступны. Иначе - пустая строка.

**Тип**: `str`

### `__cofee__`

**Описание**: Ссылка для поддержки разработчика. Получено из настроек, если доступны. Иначе - строка с умолчанием.

**Тип**: `str`


##  Использование

```python
# Пример использования функции set_project_root
root_dir = set_project_root()
print(f"Root directory: {root_dir}")

# Пример доступа к настройкам
if settings:
    print(f"Project name: {settings['project_name']}")

# Пример доступа к документации
if doc_str:
    print(f"First few lines of README: {doc_str[:100]}...")
```
