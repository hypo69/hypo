# Модуль hypotez/src/gui/context_menu/qt6/header.py

## Обзор

Этот модуль содержит константы и переменные, используемые в контекстном меню Qt6.  Он определяет константу `MODE` и добавляет путь к корневому каталогу проекта `hypotez` в системный путь `sys.path`.

## Константы и Переменные

### `MODE`

**Описание**:  Константа, хранящая режим работы приложения (`dev`, `prod`, etc.).

**Значение**: `'dev'` по умолчанию.
