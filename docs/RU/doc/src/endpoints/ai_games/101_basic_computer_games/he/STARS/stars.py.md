# STARS

## Обзор

Этот скрипт имитирует простую анимацию падающих звезд на экране. Звезды отображаются в случайных позициях и постепенно опускаются вниз, пока не исчезают за нижней границей экрана. Игра продолжается до тех пор, пока пользователь не нажмет любую клавишу.
