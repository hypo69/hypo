# Модуль `hypotez/src/suppliers/amazon/header.py`

## Обзор

Этот модуль содержит функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и чтения документации из файла `README.MD`.  Он также предоставляет переменные, содержащие информацию о проекте (название, версия, автор и др.).

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла, ищет вверх по дереву директорий, пока не найдет директорию, содержащую файлы-маркеры (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Список файлов или директорий, используемых для определения корневой директории проекта. По умолчанию включает (`'pyproject.toml'`, `'requirements.txt'`, `.git`).

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена, в противном случае путь к директории текущего файла.

**Обрабатываемые исключения**:

Нет


### Загрузка настроек проекта

**Описание**: Функция загружает настройки проекта из файла `settings.json` в корневой директории проекта.

**Параметры**:
- Нет

**Возвращает**:
- `dict`: Словарь с настройками проекта.
- `None`: Если файл настроек не найден или не может быть прочитан.

**Обрабатываемые исключения**:

- `FileNotFoundError`: Если файл `settings.json` не найден.
- `json.JSONDecodeError`: Если файл `settings.json` не является валидным JSON.



### Чтение документации

**Описание**: Функция читает содержимое файла `README.MD` из корневой директории проекта и сохраняет его в переменной.

**Параметры**:
- Нет

**Возвращает**:
- `str`: Содержимое файла `README.MD`.
- `None`: Если файл `README.MD` не найден или не может быть прочитан.

**Обрабатываемые исключения**:

- `FileNotFoundError`: Если файл `README.MD` не найден.
- `json.JSONDecodeError`: Если файл не является валидным JSON.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта. Инициализируется функцией `set_project_root`.

**Тип**: `Path`


### `settings`

**Описание**: Словарь с настройками проекта, загруженный из `settings.json`.

**Тип**: `dict` или `None`


### `doc_str`

**Описание**: Содержимое файла `README.MD`.

**Тип**: `str` или `None`



### `__project_name__`

**Описание**: Название проекта. Получается из настроек. Если `settings` или ключ "project_name" отсутствуют, то значение по умолчанию - `hypotez`.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта. Получается из настроек. Значение по умолчанию - пустая строка.

**Тип**: `str`


### `__doc__`

**Описание**: Документация проекта. Получается из `doc_str`. Значение по умолчанию - пустая строка.

**Тип**: `str`


### `__details__`

**Описание**: Подробная информация о проекте. Значение по умолчанию - пустая строка.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта. Получается из настроек. Значение по умолчанию - пустая строка.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права на проект. Получается из настроек. Значение по умолчанию - пустая строка.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка на возможность поддержать разработчика чашечкой кофе. Получается из настроек. Значение по умолчанию - ссылка.

**Тип**: `str`
