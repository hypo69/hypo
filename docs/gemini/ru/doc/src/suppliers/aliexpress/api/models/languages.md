# Модуль `hypotez/src/suppliers/aliexpress/api/models/languages.py`

## Обзор

Данный модуль содержит класс `Language` для определения языковых кодов, используемых в API AliExpress.

## Классы

### `Language`

**Описание**: Класс `Language` предоставляет константы для различных языков.

**Атрибуты**:

- `EN`: Код языка английский.
- `RU`: Код языка русский.
- `PT`: Код языка португальский.
- `ES`: Код языка испанский.
- `FR`: Код языка французский.
- `ID`: Код языка индонезийский.
- `IT`: Код языка итальянский.
- `TH`: Код языка тайский.
- `JA`: Код языка японский.
- `AR`: Код языка арабский.
- `VI`: Код языка вьетнамский.
- `TR`: Код языка турецкий.
- `DE`: Код языка немецкий.
- `HE`: Код языка иврит.
- `KO`: Код языка корейский.
- `NL`: Код языка нидерландский.
- `PL`: Код языка польский.
- `MX`: Код языка мексиканский (вероятно, обозначает регион/вариант языка).
- `CL`: Код языка чилийский (вероятно, обозначает регион/вариант языка).
- `IW`: Код языка иврит (возможно дублирование).
- `IN`: Код языка индийский (вероятно, обозначает регион/вариант языка).


**Примечание:**  Несмотря на то, что атрибуты определены как строки, их целесообразно рассматривать как константы, представляющие языковые коды.  В зависимости от контекста использования, может потребоваться соответствующее преобразование.
