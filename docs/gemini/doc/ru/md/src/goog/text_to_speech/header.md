# Модуль hypotez/src/goog/text_to_speech/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и получения документации из файла `README.MD`.  Он также устанавливает переменные для использования в других частях проекта.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная от текущего файла.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или папок, по которым определяется корневая директория проекта. По умолчанию включает `pyproject.toml`, `requirements.txt`, `.git`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена. В противном случае возвращает директорию, в которой находится текущий файл.

**Вызывает исключения**:

- Нет.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта. Инициализируется вызовом функции `set_project_root`.

**Тип**: `Path`


### `settings`

**Описание**: Словарь настроек, загруженный из файла `settings.json`.

**Тип**: `dict`

**Обработка ошибок**: Загрузка настроек может вызвать следующие исключения:
    - `FileNotFoundError`: Если файл `settings.json` не найден.
    - `json.JSONDecodeError`: Если файл `settings.json` содержит некорректный JSON.


### `doc_str`

**Описание**: Строка документации, загруженная из файла `README.MD`.

**Тип**: `str`

**Обработка ошибок**: Загрузка документации может вызвать следующие исключения:
    - `FileNotFoundError`: Если файл `README.MD` не найден.
    - `json.JSONDecodeError`: Если файл `README.MD` содержит некорректный JSON (в данном случае ошибка не ожидается).


### `__project_name__`

**Описание**: Название проекта. По умолчанию - `hypotez`. Получается из настроек `settings` (ключ `project_name`) или используется значение по умолчанию.


**Тип**: `str`

### `__version__`

**Описание**: Версия проекта. По умолчанию - пустая строка. Получается из настроек `settings` (ключ `version`) или используется значение по умолчанию.


**Тип**: `str`


### `__doc__`

**Описание**: Документация проекта. По умолчанию - пустая строка. Получается из переменной `doc_str` или используется значение по умолчанию.


**Тип**: `str`

### `__details__`

**Описание**: Подробная информация о проекте. По умолчанию - пустая строка.

**Тип**: `str`

### `__author__`

**Описание**: Автор проекта. По умолчанию - пустая строка. Получается из настроек `settings` (ключ `author`) или используется значение по умолчанию.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права на проект. По умолчанию - пустая строка. Получается из настроек `settings` (ключ `copyrihgnt`) или используется значение по умолчанию.

**Тип**: `str`

### `__cofee__`

**Описание**: Ссылка для поддержки разработчика через кофе. По умолчанию - ссылка на Boosty. Получается из настроек `settings` (ключ `cofee`) или используется значение по умолчанию.

**Тип**: `str`
