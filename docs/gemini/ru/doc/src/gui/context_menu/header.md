# Модуль hypotez/src/gui/context_menu/header.py

## Обзор

Этот модуль устанавливает пути к бинарным каталогам GTK, FFmpeg и Graphviz, а также добавляет их в `sys.path`, чтобы обеспечить доступность соответствующих библиотек.  Он также обрабатывает конфигурацию WeasyPrint, добавляя необходимые пути в переменную среды, и подавляет сообщения предупреждений GTK.

## Пути

### `__root__`

**Описание**: Абсолютный путь к корневому каталогу проекта.

**Получение**: Получается из имени проекта, содержащегося в `settings.json`, и текущего каталога.


### `gtk_bin_path`

**Описание**: Путь к каталогу бинарных файлов GTK.

**Формирование**: Формируется на основе `__root__` и подкаталогов `bin/gtk/gtk-nsis-pack/bin`.

### `ffmpeg_bin_path`

**Описание**: Путь к каталогу бинарных файлов FFmpeg.

**Формирование**: Формируется на основе `__root__` и подкаталогов `bin/ffmpeg/bin`.

### `graphviz_bin_path`

**Описание**: Путь к каталогу бинарных файлов Graphviz.

**Формирование**: Формируется на основе `__root__` и подкаталогов `bin/graphviz/bin`.
