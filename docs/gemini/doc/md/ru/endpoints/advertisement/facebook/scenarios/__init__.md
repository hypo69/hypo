# Модуль `hypotez/src/endpoints/advertisement/facebook/scenarios`

## Обзор

Данный модуль предоставляет функции для взаимодействия с Facebook Ads API. Он содержит сценарии для различных действий, таких как логин, публикация сообщений, событий и объявлений.

## Оглавление

* [Логирование](#логирование)
* [Публикация сообщения](#публикация-сообщения)
* [Переключение аккаунта](#переключение-аккаунта)
* [Публикация события](#публикация-события)
* [Публикация объявления](#публикация-объявления)

## Функции

### `login`

**Описание**: Функция для входа в аккаунт Facebook.

**Параметры**:
 - Нет

**Возвращает**:
 - `dict`: Данные пользователя после успешного входа.

**Вызывает исключения**:
 - `AuthenticationError`: Проблема с авторизацией.
 - `ConnectionError`: Проблемы с соединением.

### `switch_account`

**Описание**: Переключение между аккаунтами Facebook.

**Параметры**:
 - Нет

**Возвращает**:
 - `dict`: Данные текущего аккаунта после успешного переключения.

**Вызывает исключения**:
 - `AccountNotFoundError`: Аккаунт не найден.
 - `SwitchingError`: Проблема с переключением аккаунтов.


### `post_message_title`
**Описание**: Функция для создания заголовка сообщения.

**Параметры**:
 - Нет


**Возвращает**:
 - `str`:  Заголовок сообщения

**Вызывает исключения**:
 - `ValueError`: Неверные данные ввода.


### `upload_post_media`
**Описание**: Загрузка медиафайлов (изображений) для поста.

**Параметры**:
 - Нет


**Возвращает**:
 - `list[str]`: Список URL загруженных изображений

**Вызывает исключения**:
 - `ValueError`:  Неверные данные ввода.
 - `FileError`:  Проблемы с файлами


### `update_post_media_captions`
**Описание**: Обновление подписей к изображениям в посте.

**Параметры**:
 - Нет


**Возвращает**:
 - `dict`:  Данные об обновленных подписях.

**Вызывает исключения**:
 - `ValueError`: Неверные данные ввода.


### `message_publish`
**Описание**: Опубликовать сообщение.

**Параметры**:
 - Нет


**Возвращает**:
 - `dict`:  Данные об опубликованном сообщении.

**Вызывает исключения**:
 - `PublishingError`: Ошибка публикации.


### `post_message`
**Описание**:  Функция для публикации сообщения.

**Параметры**:
 - Нет


**Возвращает**:
 - `dict`:  Данные об опубликованном сообщении.

**Вызывает исключения**:
 - `PostingError`: Ошибка публикации.


### `post_event_title`
**Описание**: Функция для задания заголовка события.

**Параметры**:
 - Нет


**Возвращает**:
 - `str`: Заголовок события.

**Вызывает исключения**:
 - `ValueError`: Некорректный заголовок.


### `post_event_description`
**Описание**: Функция для задания описания события.

**Параметры**:
 - Нет


**Возвращает**:
 - `str`: Описание события.

**Вызывает исключения**:
 - `ValueError`: Некорректное описание.


### `post_date`
**Описание**: Функция для задания даты события.

**Параметры**:
 - Нет


**Возвращает**:
 - `date`: Дата события.

**Вызывает исключения**:
 - `ValueError`: Некорректная дата.


### `post_time`
**Описание**: Функция для задания времени события.

**Параметры**:
 - Нет


**Возвращает**:
 - `time`: Время события.

**Вызывает исключения**:
 - `ValueError`: Некорректное время.


### `post_event`
**Описание**: Функция для публикации события.

**Параметры**:
 - Нет


**Возвращает**:
 - `dict`: Данные о публикации события.

**Вызывает исключения**:
 - `PostingError`: Проблема с публикацией.


### `post_ad`
**Описание**: Функция для публикации объявления.

**Параметры**:
 - Нет


**Возвращает**:
 - `dict`: Данные об опубликованном объявлении.

**Вызывает исключения**:
 - `AdvertisingError`: Ошибка при публикации объявления.
