# Модуль `src.ai`

## Обзор

Модуль `src.ai` предоставляет интерфейс для взаимодействия с моделями ИИ, такими как Gemini и OpenAI. Он определяет константу `MODE` для обозначения режима работы и импортирует классы `GoogleGenerativeAI` и `OpenAIModel`.

## Константы

### `MODE`

**Описание**: Переменная, определяющая текущий режим работы модуля.

**Значение**: Строка, в данном случае 'dev'.


## Классы

### `GoogleGenerativeAI`

**Описание**:  Класс, предоставляющий интерфейс для работы с моделью ИИ Gemini.  Подробная информация о классах и функциях находится в файле `src/ai/gemini.py`.

### `OpenAIModel`

**Описание**: Класс, предоставляющий интерфейс для работы с моделью ИИ OpenAI. Подробная информация о классах и функциях находится в файле `src/ai/openai.py`.


## Модуль `__init__.py`

**Описание**:  Файл инициализации модуля `src.ai`. Он содержит импорты необходимых классов из подмодулей.
