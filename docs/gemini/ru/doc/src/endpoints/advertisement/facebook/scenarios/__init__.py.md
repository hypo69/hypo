# Модуль `hypotez/src/endpoints/advertisement/facebook/scenarios/__init__.py`

## Обзор

Данный модуль инициализирует пакет `scenarios` и предоставляет доступ к различным сценариям взаимодействия с Facebook, связанным с публикацией контента, переключением аккаунтов и т.д.

## Оглавление

- [Импортированные модули](#импортированные-модули)
- [Экспортированные функции и переменные](#экспортированные-функции-и-переменные)

## Импортированные модули

В этом модуле импортируются следующие модули:

- `.login`: Модуль для выполнения входа в Facebook.
- `.post_message`: Модуль для работы с публикацией сообщений.
- `.switch_account`: Модуль для переключения между аккаунтами.
- `.post_event`: Модуль для работы с публикацией событий.
- `.post_ad`: Модуль для публикации рекламных объявлений.

## Экспортированные функции и переменные

В этом модуле экспортируются следующие функции и переменные:

- `login`: Функция для входа в Facebook.
- `switch_account`: Функция для переключения между аккаунтами.
- `post_message_title`: Функция для публикации заголовка сообщения.
- `upload_post_media`: Функция для загрузки медиафайлов для поста.
- `update_post_media_captions`: Функция для обновления подписей к медиафайлам поста.
- `message_publish`: Функция для публикации сообщения.
- `post_message`: Функция для публикации сообщения.
- `post_event_title`: Функция для публикации заголовка события.
- `post_event_description`: Функция для публикации описания события.
- `post_date`: Функция для установки даты события.
- `post_time`: Функция для установки времени события.
- `post_event`: Функция для публикации события.
- `post_ad`: Функция для публикации рекламного объявления.
