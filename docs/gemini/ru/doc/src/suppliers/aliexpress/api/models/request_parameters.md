# Модуль `hypotez/src/suppliers/aliexpress/api/models/request_parameters.py`

## Обзор

Данный модуль определяет перечисления (enum) для параметров запросов API AliExpress. Он содержит константы, представляющие типы продуктов, способы сортировки и типы ссылок.

## Оглавление

* [ProductType](#product-type)
* [SortBy](#sort-by)
* [LinkType](#link-type)

## `ProductType`

**Описание**: Перечисление для типов продуктов.

**Константы**:

- `ALL`: Представляет все типы продуктов.
- `PLAZA`: Представляет тип продукта "PLAZA".
- `TMALL`: Представляет тип продукта "TMALL".


## `SortBy`

**Описание**: Перечисление для способов сортировки результатов поиска.

**Константы**:

- `SALE_PRICE_ASC`: Сортировка по цене (по возрастанию).
- `SALE_PRICE_DESC`: Сортировка по цене (по убыванию).
- `LAST_VOLUME_ASC`: Сортировка по последнему объему (по возрастанию).
- `LAST_VOLUME_DESC`: Сортировка по последнему объему (по убыванию).

## `LinkType`

**Описание**: Перечисление для типов ссылок.

**Константы**:

- `NORMAL`: Обычный тип ссылки.
- `HOTLINK`: Тип ссылки "HOTLINK".
