# Модуль для конвертации XLS файлов в словарь

## Обзор

Модуль предоставляет функциональность для конвертации файлов в формате `.xls` в словарь Python. Он использует функции из модуля `src.utils.xls` для чтения данных из XLS файла и возвращает их в виде словаря.

## Подробней

Данный модуль предоставляет удобный интерфейс для преобразования данных из устаревшего формата `.xls` в более современный и удобный для обработки формат словаря Python. Это может быть полезно для интеграции старых данных в новые системы или для анализа данных, хранящихся в формате `.xls`.

## Функции

### `xls2dict`

```python
def xls2dict(xls_file: str | Path) -> dict | None:
    """"""
```

**Назначение**: Преобразует XLS файл в словарь Python.

**Параметры**:

- `xls_file` (str | Path): Путь к XLS файлу.

**Возвращает**:

- `dict | None`: Словарь, представляющий данные из XLS файла, или `None` в случае ошибки.

**Как работает функция**:

1.  Функция `xls2dict` принимает путь к XLS файлу в качестве аргумента.
2.  Вызывает функцию `read_xls_as_dict` из модуля `src.utils.xls`, передавая ей путь к файлу.
3.  Возвращает результат, полученный от `read_xls_as_dict`.

```
XLS файл --> read_xls_as_dict() --> Словарь Python
```

**Примеры**:

```python
from pathlib import Path
from src.utils.convertors.xls import xls2dict

# Пример использования с указанием пути к файлу в виде строки
file_path_str = "example.xls"
data_from_xls = xls2dict(file_path_str)

if data_from_xls:
    print(f"Данные из XLS файла: {data_from_xls}")
else:
    print("Не удалось преобразовать XLS файл в словарь.")

# Пример использования с указанием пути к файлу в виде объекта Path
file_path_obj = Path("example.xls")
data_from_xls = xls2dict(file_path_obj)

if data_from_xls:
    print(f"Данные из XLS файла: {data_from_xls}")
else:
    print("Не удалось преобразовать XLS файл в словарь.")
