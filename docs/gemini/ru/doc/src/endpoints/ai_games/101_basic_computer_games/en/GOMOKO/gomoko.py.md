# Игра Гомоку

## Обзор

Данный модуль реализует классическую настольную игру Гомоку (также известную как "Пять в ряд"). Игра ведется между пользователем и компьютером на доске размером 15x15. Цель игры - первым выстроить непрерывную линию из пяти своих фишек по вертикали, горизонтали или диагонали.

## Функции

### `print_board`

**Описание**: Выводит текущее состояние игрового поля в консоль.

**Параметры**:

- `board` (list[list[str]]): Двумерный список, представляющий игровое поле.

**Возвращает**:

- `None`: Функция ничего не возвращает, а только выводит данные в консоль.

### `check_winner`

**Описание**: Проверяет, есть ли на игровом поле выигрышная комбинация для указанного игрока.

**Параметры**:

-   `board` (list[list[str]]): Двумерный список, представляющий игровое поле.
-   `player` (int): Номер игрока (1 или 2), для которого выполняется проверка.

**Возвращает**:

-   `bool`: `True`, если есть победитель, иначе `False`.

### `check_full_board`

**Описание**: Проверяет, заполнено ли игровое поле.

**Параметры**:

- `board` (list[list[str]]): Двумерный список, представляющий игровое поле.

**Возвращает**:

- `bool`: `True`, если доска заполнена (нет пустых ячеек), иначе `False`.

## Переменные

-   `board` (list[list[str]]): Двумерный список, представляющий игровое поле 15x15.
-   `game_on` (bool): Флаг, указывающий, продолжается ли игра (`True`) или нет (`False`).
-   `current_player` (int): Номер текущего игрока (1 или 2).
