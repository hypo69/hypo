# Модуль `src.webdriver.edge`

## Обзор

Модуль `src.webdriver.edge` предназначен для управления браузером Edge. Он предоставляет функциональность для взаимодействия с Edge через вебдрайвер.

## Содержание

- [Обзор](#обзор)
- [Переменные](#переменные)
- [Импорты](#импорты)

## Переменные

### `MODE`

**Описание**: Указывает режим работы модуля. В данном случае установлен режим `'dev'`.
   
## Импорты

### `from .edge import Edge`
**Описание**: Импортирует класс `Edge` из модуля `edge.py`
