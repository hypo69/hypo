# Модуль `src`

## Обзор

Модуль `src` является корневым модулем проекта. Он предоставляет структурированный обзор различных модулей внутри проекта, описывая их основные функциональные возможности и роли.

## Содержание

- [Обзор](#обзор)
- [Переменные](#переменные)
- [Импорты](#импорты)

## Переменные

### `MODE`

**Описание**: Определяет режим работы приложения.

**Значение**: `'dev'`

## Импорты

### `from .credentials import gs`

**Описание**: Импортирует модуль `gs` из подмодуля `credentials`.
