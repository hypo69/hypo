# Модуль rocket.py

## Обзор

Модуль `rocket.py` реализует игру "Ракета", в которой моделируется запуск ракеты в космос. Игрок вводит начальный импульс, а программа рассчитывает траекторию ракеты, отображая её высоту в зависимости от времени, а также время падения на землю.

## Функции

### `play_rocket_game`

**Описание**:
Моделирует запуск ракеты в космос и отображает высоту как функцию времени.

**Параметры**:
- Нет параметров.

**Возвращает**:
- `None`: Функция не возвращает значения.

**Вызывает исключения**:
- `ValueError`: Возникает, если ввод пользователя не является числом.

```python
def play_rocket_game():
    """
    Моделирует запуск ракеты в космос и отображает высоту как функцию времени.
    """
    try:
        # Запрашиваем начальный импульс у пользователя
        initialThrust = float(input("הזן את הדחף ההתחלתי: "))
    except ValueError as ex:
        print("קלט לא תקין, אנא הזן מספר.")
        return

    rocketHeight = 0  # Высота ракеты
    time = 0  # Время
    timeStep = 0.1  # Шаг времени

    print("\nשיגור הרקטה...")
    time.sleep(1)  # Небольшая задержка для симуляции
    # Основной цикл, пока ракета в воздухе (высота положительная)
    while rocketHeight >= 0:
        # Печатаем простой график, представляющий высоту ракеты в заданный момент времени
        print("*" * int(rocketHeight / 10 if rocketHeight > 0 else 0) + f" {time:.1f}")

        # Вычисляем новую высоту ракеты
        # Формула имитирует влияние тяги и гравитации на высоту
        rocketHeight = rocketHeight + initialThrust - 0.5 * 10 * time * time

        # Увеличиваем время
        time += timeStep
        # Короткая задержка между шагами симуляции
        time.sleep(0.05)

    # Печатаем время падения на землю
    print(f"\nהרקטה פגעה בקרקע בזמן {time:.1f} שניות.")
```
