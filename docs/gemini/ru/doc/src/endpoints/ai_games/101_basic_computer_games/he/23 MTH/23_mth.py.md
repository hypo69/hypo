## Обзор

Этот модуль реализует простую игру "Арифметика", в которой компьютер предлагает пользователю математические задачи (сложение, вычитание или умножение) и проверяет правильность ответов. Игра продолжается до тех пор, пока пользователь не даст неправильный ответ.

## Функции

В этом модуле нет отдельных функций, весь код находится в глобальной области видимости.

## Переменные

### `correctAnswers`
- **Описание**: Переменная для подсчета количества правильных ответов.
- **Тип**: `int`
- **Начальное значение**: `0`

### `operand1`
- **Описание**: Первое число для математической задачи, генерируется случайным образом.
- **Тип**: `int`
- **Значения**: Целое число от 1 до 10 включительно.

### `operand2`
- **Описание**: Второе число для математической задачи, генерируется случайным образом.
- **Тип**: `int`
- **Значения**: Целое число от 1 до 10 включительно.

### `operations`
- **Описание**: Список возможных математических операций.
- **Тип**: `list`
- **Значения**: `['+', '-', '*']`

### `operator`
- **Описание**: Случайная операция из списка `operations`.
- **Тип**: `str`
- **Значения**: `'+'`, `'-'` или `'*'`

### `question`
- **Описание**: Строка, содержащая математическое выражение для пользователя.
- **Тип**: `str`
- **Значение**: Например, `"5 + 3 = ?"`

### `userAnswer`
- **Описание**: Ответ пользователя, преобразованный в целое число.
- **Тип**: `int`
- **Значения**: Целое число, введенное пользователем.

### `correctAnswer`
- **Описание**: Правильный ответ на математическое выражение.
- **Тип**: `int`
- **Значения**: Результат вычисления выражения `operand1 operator operand2`.

## Описание кода

1. **Импорт модуля `random`**:
    ```python
    import random
    ```
   - Импортирует модуль `random` для генерации случайных чисел и операций.

2. **Инициализация счетчика правильных ответов**:
   ```python
   correctAnswers = 0
   ```
   - Устанавливает начальное значение переменной `correctAnswers` равным 0.

3. **Основной цикл игры**:
   ```python
   while True:
   ```
   - Начинает бесконечный цикл, который продолжается до тех пор, пока пользователь не даст неверный ответ.

4. **Генерация случайных чисел и операции**:
   ```python
    operand1 = random.randint(1, 10)
    operand2 = random.randint(1, 10)
    operations = ['+', '-', '*']
    operator = random.choice(operations)
   ```
   - Генерирует два случайных числа `operand1` и `operand2` в диапазоне от 1 до 10.
   - Определяет список возможных операций `operations`.
   - Выбирает случайную операцию из списка `operations` и сохраняет ее в переменной `operator`.

5. **Формирование и вывод вопроса**:
   ```python
    question = f"{operand1} {operator} {operand2} = ?"
    print(question)
   ```
   - Создает строку вопроса с использованием f-строки и выводит ее на экран.

6. **Получение ответа от пользователя**:
    ```python
    try:
        userAnswer = int(input("הזן את התשובה: "))
    except ValueError:
        print("אנא הזן מספר שלם.")
        break
    ```
    - Запрашивает ввод от пользователя и преобразует его в целое число.
    - Использует `try-except` для обработки `ValueError`, если пользователь ввел не число.

7. **Вычисление правильного ответа**:
   ```python
   if operator == '+':
        correctAnswer = operand1 + operand2
   elif operator == '-':
        correctAnswer = operand1 - operand2
   else: # operator == '*'
        correctAnswer = operand1 * operand2
   ```
   - Вычисляет правильный ответ в зависимости от выбранной операции.

8. **Проверка ответа пользователя**:
   ```python
   if userAnswer == correctAnswer:
        correctAnswers += 1
   else:
        break
   ```
   - Сравнивает ответ пользователя с правильным ответом.
   - Увеличивает счетчик правильных ответов, если ответ верный.
   - Прерывает цикл, если ответ неверный.

9. **Вывод результата игры**:
    ```python
    print(f"YOU GOT {correctAnswers} RIGHT")
    ```
    - Выводит итоговое количество правильных ответов после завершения игры.
