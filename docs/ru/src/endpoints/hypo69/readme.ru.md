# src.endpoints.hypo69

## Обзор

Документация для модуля `src.endpoints.hypo69`, который содержит эндпоинты для разработчика. Модуль включает в себя различные инструменты и боты, облегчающие процесс разработки и взаимодействия с AI-моделями.

## Подробней

Модуль `src.endpoints.hypo69` предоставляет набор эндпоинтов, предназначенных для упрощения разработки и интеграции с AI-моделями. В частности, он содержит:

- `small_talk_bot`: бот для общения с AI-моделью.
- `code_assistant`: модуль для обучения модели коду проекта.
- `psychologist_bot`: модуль для парсинга диалогов (ранняя разработка).

Этот модуль является частью проекта `hypotez` и предоставляет инструменты для взаимодействия с различными AI-моделями и выполнения задач обработки кода. Он расположен в подкаталоге `src/endpoints/hypo69` и предназначен для использования разработчиками.
