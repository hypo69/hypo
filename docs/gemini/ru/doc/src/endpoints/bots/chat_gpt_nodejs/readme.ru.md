# Модуль chat_gpt_nodejs

## Обзор

Данный модуль предоставляет реализацию чат-бота ChatGPT на основе Node.js.  Модуль позволяет взаимодействовать с моделью ChatGPT, отправляя запросы и получая ответы.  Подробная информация о реализации доступна по ссылке: [https://habr.com/ru/companies/selectel/articles/731692/](https://habr.com/ru/companies/selectel/articles/731692/).
