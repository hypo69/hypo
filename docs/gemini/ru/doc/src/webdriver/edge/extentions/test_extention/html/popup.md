# hypotez/src/webdriver/edge/extentions/test_extention/html/popup.html

## Обзор

Файл `popup.html` содержит HTML-код для всплывающего окна расширения. Он отображает заголовок "hypotez" и краткое описание, информирующее пользователя о функциональности расширения.

## Структура HTML

Этот HTML-файл имеет стандартную структуру веб-страницы с тегами `<html>`, `<head>` и `<body>`.

### Тег `<head>`

Тег `<head>` содержит заголовок страницы (`<title>`) и стили CSS (`<style>`), определяющие внешний вид страницы. Стиль `body` устанавливает фиксированную ширину и отступы для элемента `<body>`.

### Тег `<body>`

Тег `<body>` содержит основное содержимое страницы:

* **Заголовок (`<h1>`):**  Отображает название "hypotez".
* **Параграф (`<p>`):**  Содержит описание действия расширения, информируя пользователя о его функции.


## Файл popup.html

```html
<!DOCTYPE html>
<html>
<head>
    <title>hypotez</title>
    <style>
        body {
            width: 200px;
            padding: 10px;
        }
    </style>
</head>
<body>
    <h1>hypotez</h1>
    <p>Click the extension icon to collect data from the current webpage.</p>
</body>
</html>
```


## Дополнительные замечания

Этот HTML-файл не содержит JavaScript-кода.  Функциональность расширения реализуется в связанных JavaScript-файлах.  Этот файл предназначен только для отображения пользовательского интерфейса всплывающего окна.
