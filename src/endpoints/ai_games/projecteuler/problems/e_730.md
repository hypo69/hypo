# Задача No 730
## Задачи с числами Кармайкла

## Условия
Число Кармайкла - это составное число n, удовлетворяющее сравнению *b<sup>n-1</sup>* ≡ 1 (mod *n*) для всех целых чисел *b*, взаимно простых с *n*.
Известно, что 561 является наименьшим числом Кармайкла, а 1105 - следующим.

Пусть *C(N)* - количество чисел Кармайкла, не превышающих *N*. Например, *C*(1000) = 1, *C*(10000) = 3 и *C*(1000000) = 16.

Также пусть *S(N)* - сумма чисел Кармайкла, не превышающих *N*. Например, *S*(1000) = 561, *S*(10000) = 2821 и *S*(1000000) = 2163921.

Даны следующие значения:

* *C*(10<sup>8</sup>) = 105
* *S*(10<sup>8</sup>) = 194758720

Найдите *S*(10<sup>16</sup>) mod 10<sup>9</sup>.
