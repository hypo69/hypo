# Модуль Provider
## Обзор

Модуль определяет структуру и базовые параметры для провайдеров, используемых в проекте `hypotez`. Он содержит информацию о поддержке стриминга, необходимости аутентификации, а также базовые параметры для создания запросов к моделям.

## Подробней

Этот модуль служит основой для интеграции различных провайдеров (например, OpenAI, Google Gemini) в систему. Он задает общие параметры и структуру, которые используются для унификации работы с разными моделями.
