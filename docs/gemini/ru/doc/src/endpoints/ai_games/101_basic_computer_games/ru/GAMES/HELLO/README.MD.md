# Игра Hello

## Обзор

Этот файл содержит код игры "Hello", которая приветствует пользователя и просит ввести имя. Программа выводит персонализированное приветствие и дополнительное сообщение. Это одна из самых простых программ, которая демонстрирует базовые команды Python.

## Функции

### `print`

**Описание**: Выводит текст на экран.

### `input`

**Описание**: Получает данные от пользователя.

**Параметры**:

- Сообщение для пользователя (str): Текст, который будет отображен пользователю перед вводом.

**Возвращает**:

- Ввод пользователя (str): Текст, введенный пользователем.

###  Основные блоки программы

1.  **Вывод приветствия на экран**
    ```python
    print("Привет! Добро пожаловать в игру Hello!")
    ```
    -   Используется функция `print` для вывода приветствия пользователю.

2.  **Запрос имени пользователя**
    ```python
    name = input("Как тебя зовут? ")
    ```
    -   Используется функция `input` для получения данных от пользователя. Запрашивает имя и сохраняет его в переменной `name`.

3.  **Вывод персонализированного приветствия**
    ```python
    print(f"Привет, {name}! Приятно познакомиться!")
    ```
    -   Используется f-строка для подстановки имени пользователя в текст приветствия.

4.  **Дополнительное сообщение**
    ```python
     print("Спасибо за использование программы Hello!")
    ```
    - Выводит дополнительное сообщение после персонализированного приветствия.

## Пример выполнения программы

Привет! Добро пожаловать в игру Hello!

## Блок-схема

```mermaid
flowchart TD
    Start([Начало]) --> DisplayWelcome["Вывод: 'Привет! Добро пожаловать в игру Hello!'"]
    DisplayWelcome --> AskName["Вывод: 'Как тебя зовут?'"]
    AskName --> GetUserName[/"Получение имени пользователя"/]
    GetUserName --> StoreName["Сохранение имени в переменную 'name'"]
    StoreName --> DisplayGreeting["Вывод: 'Привет, {name}! Приятно познакомиться!'"]
    DisplayGreeting --> DisplayThanks["Вывод: 'Спасибо за использование программы Hello!'"]
    DisplayThanks --> End([Конец])
```

**Легенда:**

1.  **`Start`** – Начало программы.
2.  **`DisplayWelcome`** – Вывод приветствия пользователю.
3.  **`AskName`** – Вывод запроса имени пользователя.
4.  **`GetUserName`** – Получение имени от пользователя.
5.  **`StoreName`** – Сохранение имени в переменную `name`.
6.  **`DisplayGreeting`** – Вывод персонализированного приветствия с использованием переменной `name`.
7.  **`DisplayThanks`** – Вывод сообщения о завершении программы.
8.  **`End`** – Конец программы.
