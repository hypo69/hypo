# מודול `header`

## סקירה כללית

מודול זה מגדיר פונקציה `set_project_root` אשר מוצאת את ספריית השורש של הפרויקט על ידי חיפוש כלפי מעלה מהספרייה של הקובץ הנוכחי. הוא עוצר כאשר הוא מוצא ספרייה המכילה אחד מקבצי המרקר שצוינו. המודול גם מגדיר את משתנה `__root__` המכיל את הנתיב לספריית השורש של הפרויקט.

## פונקציות

### `set_project_root`

**תיאור**:
מוצא את ספריית השורש של הפרויקט החל מהספרייה של הקובץ הנוכחי, מחפש כלפי מעלה ועוצר בספרייה הראשונה המכילה אחד מקבצי המרקר.

**פרמטרים**:
- `marker_files` (tuple, אופציונלי): שמות קבצים או שמות ספריות המשמשים לזיהוי שורש הפרויקט. ברירת מחדל: `('__root__', '.git')`.

**החזרות**:
- `Path`: נתיב לספריית השורש אם נמצא, אחרת הנתיב לספרייה בה נמצא הסקריפט.

**הערות**:
אם ספריית השורש אינה קיימת ב-`sys.path`, היא תתווסף.
