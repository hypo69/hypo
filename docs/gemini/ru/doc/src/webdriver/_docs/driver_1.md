# Модуль driver_1

## Обзор

Данный модуль определяет метакласс `DriverMeta`, предназначенный для динамического создания класса `Driver`, наследующего как от базового класса `Driver`, так и от указанного класса Selenium WebDriver (например, `Chrome`, `Firefox` или `Edge`). Метакласс отвечает за создание корректной комбинации этих классов.

## Метаклассы

### `DriverMeta`

**Описание**: Метакласс `DriverMeta` контролирует создание класса `Driver`.

**Методы**:

#### `__call__`

**Описание**: Метод `__call__` вызывается при создании экземпляра класса.  Он создает новый класс `Driver`, наследующий от базового класса `Driver` и указанного класса WebDriver.

**Параметры**:

- `cls` (type): Класс, который вызывается (в данном случае, это `Driver`).
- `webdriver_cls` (type): Класс Selenium WebDriver, от которого наследоваться (`Chrome`, `Firefox`, или `Edge`).
- `*args`: Аргументы для конструктора класса `Driver`.
- `**kwargs`: Именованные аргументы для конструктора класса `Driver`.

**Возвращает**:
- `Driver`: Экземпляр динамически созданного класса `Driver`.


**Проверки**:

- `assert isinstance(webdriver_cls, type)`:  Проверяет, что `webdriver_cls` является классом.
- `assert issubclass(webdriver_cls, Chrome | Firefox | Edge)`:  Проверяет, что `webdriver_cls` является подклассом одного из разрешенных классов WebDriver (`Chrome`, `Firefox` или `Edge`).

**Вызывает исключения**:
- `AssertionError`: Возникает, если проверка типов не пройдена.

**Динамическое создание класса `Driver`**:

- Внутри метода `__call__` динамически создается новый класс с именем `Driver`. Этот новый класс наследуется от `cls` (базовый класс `Driver`) и `webdriver_cls` (указанный класс WebDriver).

#### `Driver`

**Описание**:  Динамически создаваемый класс, который наследует от базового `Driver` и указанного Selenium WebDriver.

**Методы**:

##### `__init__`

**Описание**: Конструктор динамически создаваемого класса `Driver`.

**Параметры**:
- `self`: Ссылка на экземпляр класса.
- `*args`: Аргументы для конструктора базового класса `Driver`.
- `**kwargs`: Именованные аргументы для конструктора базового класса `Driver`.

**Возвращает**:
- `None`

**Действия**:
- Логирует инициализацию WebDriver с его именем и аргументами.
- Вызывает конструкторы родительских классов с использованием `super()`.
- Вызывает метод `driver_payload`.


##### `driver_payload`

**Описание**: Метод, определенный внутри динамически создаваемого класса `Driver`.  Вызывает метод `driver_payload` из родительского класса `Driver`.

**Параметры**:
- `self`: Ссылка на экземпляр класса.

**Возвращает**:
- `None`


## Пример использования

```python
# Создание экземпляра Driver с Chrome в качестве WebDriver
chrome_driver = Driver(Chrome, *args, **kwargs)

# Создание экземпляра Driver с Firefox в качестве WebDriver
firefox_driver = Driver(Firefox, *args, **kwargs)
```
