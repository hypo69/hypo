# Модуль hypotez/src/webdriver/firefox/firefox.py

## Обзор

Этот модуль определяет подкласс `webdriver.Firefox`, называемый `Firefox`. Он предоставляет дополнительную функциональность, такую как запуск Firefox в режиме киоска и настройку профиля Firefox для WebDriver.  Модуль использует настройки из файла `firefox.json` для конфигурации драйвера.  Включает обработку исключений при запуске драйвера.  Инициализирует и настраивает необходимые объекты для взаимодействия с браузером.


## Классы

### `Firefox`

**Описание**: Подкласс `selenium.webdriver.Firefox`, добавляющий дополнительные возможности.

**Атрибуты**:

- `driver_name` (str): Имя используемого WebDriver, по умолчанию 'firefox'.


**Методы**:

### `__init__`

**Описание**: Инициализирует Firefox WebDriver со специфицированными параметрами запуска, профилем, версиями geckodriver и Firefox.

**Параметры**:

- `profile_name` (Optional[str], необязательно): Имя профиля Firefox для использования.
- `geckodriver_version` (Optional[str], необязательно): Версия geckodriver для использования.
- `firefox_version` (Optional[str], необязательно): Версия Firefox для использования.
- `user_agent` (Optional[dict], необязательно): Словарь с настройками user agent.

**Возвращает**:
- `None`

**Вызывает исключения**:

- `WebDriverException`: Исключение, которое возникает при проблемах с запуском драйвера.  (Проблемы с обновлением Firefox, отсутствием драйвера.)
- `Exception`: Общее исключение, возникающее при других ошибках инициализации.

**Описание**:  Модуль использует файл `firefox.json` в папке `src/webdriver/firefox` для загрузки параметров конфигурации WebDriver.  Этот файл должен содержать настройки для работы с Firefox WebDriver, включая пути к исполняемым файлам, параметры профиля, пользовательские агенты и т.д.
