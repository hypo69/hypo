## АНАЛИЗ КОДА: `bingo.py`

### <алгоритм>
1. **Начало игры:**
    - Инициализация:
        - Создается пустая матрица 5x5 `bingoCard`.
        - Формируется список чисел от 1 до 75, исключая числа, кратные 10.
        - Этот список перемешивается.
        - Матрица `bingoCard` заполняется случайными уникальными числами из перемешанного списка.

2. **Игровой цикл:**
    - Пока на карточке `bingoCard` есть неотмеченные числа (не равные 0):
        - Генерируется случайное число `randomNumber` от 1 до 75.
            - Пример: `randomNumber = 32`
        - Выводится `randomNumber` пользователю.
            - Пример: `Выпало число: 32`
        - Проверяется, есть ли `randomNumber` на карточке `bingoCard`:
            - Если `randomNumber` есть на карточке, то он заменяется на 0.
            - Если `randomNumber` нет на карточке, ход пропускается.
        - Выводится текущее состояние `bingoCard`.

3. **Конец игры:**
    - Когда все числа на карточке отмечены, выводится сообщение "BINGO!".
    - Завершение программы.

**Пример потока данных:**
- `create_bingo_card()`: Создает матрицу (карточку) и возвращает её.
- `play_bingo()`:
    - Вызывает `create_bingo_card()` для получения игровой карточки.
    - Циклически:
        - Генерирует случайное число.
        - Выводит число.
        - Вызывает `mark_number()` для отметки числа на карточке.
        - Вызывает `print_bingo_card()` для вывода карточки.
        - Вызывает `is_bingo()` для проверки, все ли числа отмечены.

### <объяснение>
**1. Импорты:**
   - `import random`: Импортирует модуль `random` для генерации случайных чисел и перемешивания списка чисел при создании карточки бинго.

**2. Функции:**

   - `create_bingo_card()`:
     - **Назначение:** Создает и возвращает карточку бинго 5x5.
     - **Логика:**
         - Создает пустой список `card`, который будет представлять собой матрицу 5x5.
         - Создает список `numbers` чисел от 1 до 75, исключая кратные 10.
         - Перемешивает список `numbers` с помощью `random.shuffle()`.
         - Заполняет матрицу `card` построчно, извлекая числа из `numbers` с помощью `numbers.pop()`.
     - **Возвращаемое значение:**  Список списков (матрица) `card` - карточка бинго.
     - **Пример:** `card = create_bingo_card()` возвращает матрицу 5x5 с числами, как описано выше.

   - `print_bingo_card(card)`:
     - **Назначение:** Выводит карточку бинго в консоль.
     - **Аргументы:** `card` - карточка бинго (список списков).
     - **Логика:**
       - Итерируется по строкам карточки.
       - Выводит каждую строку, выравнивая числа по правому краю и разделяя их пробелами.
       - Добавляет пустую строку в конце.
     - **Пример:** `print_bingo_card(card)` выведет текущее состояние игровой карточки.

   - `mark_number(card, number)`:
     - **Назначение:** Отмечает число на карточке, заменяя его на 0.
     - **Аргументы:** `card` - карточка бинго, `number` - число, которое нужно отметить.
     - **Логика:**
         - Итерируется по всем ячейкам карточки.
         - Если находит ячейку, равную `number`, заменяет ее значение на 0 и завершает функцию.
     - **Пример:** `mark_number(card, 25)` - если число 25 есть на карточке, оно будет заменено на 0.

   - `is_bingo(card)`:
     - **Назначение:** Проверяет, все ли числа на карточке отмечены.
     - **Аргументы:** `card` - карточка бинго.
     - **Логика:**
       - Итерируется по всем числам в карточке.
       - Если находит число, не равное 0, возвращает `False`.
       - Если все числа равны 0, возвращает `True`.
     - **Возвращаемое значение:** `True`, если все числа на карточке отмечены, иначе `False`.
     - **Пример:** `is_bingo(card)` - возвращает `True`, если все числа на карточке равны 0, иначе `False`.

   - `play_bingo()`:
      - **Назначение:** Запускает игру "Бинго".
      - **Логика:**
          - Создает карточку с помощью `create_bingo_card()`.
          - Выводит карточку в консоль с помощью `print_bingo_card()`.
          - Инициализирует множество `called_numbers` для отслеживания уже названных чисел.
          - Запускает цикл `while not is_bingo(bingo_card)`:
            - Генерирует случайное число `number` от 1 до 75.
            - Гарантирует, что число не было вызвано ранее.
            - Добавляет число в множество `called_numbers`.
            - Выводит выпавшее число.
            - Отмечает число на карточке с помощью `mark_number()`.
            - Выводит карточку.
          - Выводит сообщение "BINGO!" при завершении игры.

**3. Переменные:**
   - `card`: Представляет собой карточку бинго в виде списка списков (матрица 5x5).
   - `numbers`: Список чисел от 1 до 75, исключая кратные 10.
   - `row`: Список, представляющий строку в карточке бинго.
   - `i`, `j`: Индексы для итерации по строкам и столбцам карточки.
   - `number`: Случайное число, генерируемое в процессе игры.
   - `bingo_card`: Переменная, хранящая игровую карточку, сгенерированную `create_bingo_card()`
    - `called_numbers`: множество, хранящее уже выпавшие числа в текущей игре.
