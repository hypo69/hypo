# Модуль `hypotez/src/suppliers/aliexpress/api/models/__init__.py`

## Обзор

Данный модуль предоставляет набор моделей для работы с API AliExpress. Он содержит определения классов, представляющих различные сущности, такие как языки, валюты, параметры запроса, ссылки, горячие товары, продукты и категории.  Модуль импортирует классы из соответствующих файлов, обеспечивая единую точку доступа к ним.

## Оглавление

* [Классы](#классы)
* [Импорты](#импорты)


## Импорты

- `Language`: Модель для представления языков.
- `Currency`: Модель для представления валют.
- `ProductType`: Модель для представления типов продуктов.
- `SortBy`: Модель для представления способов сортировки.
- `LinkType`: Модель для представления типов ссылок.
- `AffiliateLink`: Модель для представления партнерских ссылок.
- `HotProductsResponse`: Модель для ответа на запрос горячих товаров.
- `Product`: Модель для представления продукта.
- `Category`: Модель для представления категории.
- `ChildCategory`: Модель для представления подкатегории.


## Классы

В данном модуле отсутствуют классы, которые определяются в других модулях, и импортируются в текущем модуле. Подробное описание этих классов находится в соответствующих файлах.
