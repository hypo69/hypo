# Модуль `header.py`

## Обзор

Модуль `header.py` расположен в `src/gui/context_menu/qt6` и предоставляет необходимые импорты и настройки для работы с контекстным меню в Qt6.

- [Обзор](#обзор)
- [Импорты](#импорты)
- [Глобальные переменные](#глобальные-переменные)

## Импорты

### `sys`
Модуль `sys` используется для доступа к некоторым переменным и функциям, взаимодействующим с интерпретатором Python.
### `os`
Модуль `os` предоставляет функции для взаимодействия с операционной системой, такие как работа с файловой системой.
### `pathlib`
Модуль `pathlib` предоставляет классы для представления путей к файлам и каталогам, делая работу с ними более удобной и объектно-ориентированной.

## Глобальные переменные

### `__root__`
**Описание**: `__root__` хранит корневой путь к проекту, который вычисляется на основе текущей директории, усеченной до директории с именем `hypotez`.
**Тип**: `pathlib.Path`
