# FURS: Меховые Торговцы

## Обзор

Игра "Меховые Торговцы" - это экономическая симуляция, где игрок выступает в роли торговца мехом, перемещающегося между тремя городами (А, Б и С) с целью получения максимальной прибыли. В каждом городе цены на мех меняются случайным образом, и игрок должен принимать решения о покупке, продаже или перемещении в другой город. Цель игры - накопить как можно больше денег за 10 ходов.

## Инициализация переменных

```python
playerMoney = 1000
playerFur = 10
currentCity = 'A'
turnNumber = 1
```

**Описание**:
Инициализация начальных значений переменных игры:
- `playerMoney`: Начальное количество денег у игрока.
- `playerFur`: Начальное количество меха у игрока.
- `currentCity`: Начальный город игрока.
- `turnNumber`: Номер текущего хода.

## Основной игровой цикл

```python
while turnNumber <= 10:
    priceA = random.randint(100, 500)
    priceB = random.randint(100, 500)
    priceC = random.randint(100, 500)

    print(f"\n--- Ход {turnNumber} ---")
    print(f"Цены на мех: A={priceA}, B={priceB}, C={priceC}")
    print(f"Деньги: {playerMoney}, Мех: {playerFur}, Город: {currentCity}")

    action = input("Действие (купить/buy, продать/sell, переместиться/move, ждать/wait): ").lower()

    if action == "купить" or action == "buy":
        try:
            amount = int(input("Сколько меха купить? "))
            if playerMoney >= amount * locals()[f'price{currentCity}']:
                playerMoney -= amount * locals()[f'price{currentCity}']
                playerFur += amount
                print("Покупка совершена.")
            else:
                 print("Недостаточно денег.")
        except ValueError as ex:
             print("Некорректный ввод.")
    elif action == "продать" or action == "sell":
        try:
             amount = int(input("Сколько меха продать? "))
             if playerFur >= amount:
                playerMoney += amount * locals()[f'price{currentCity}']
                playerFur -= amount
                print("Продажа совершена.")
             else:
                print("Недостаточно меха.")
        except ValueError as ex:
             print("Некорректный ввод.")
    elif action == "переместиться" or action == "move":
        city = input("В какой город переместиться (A, B, C)? ").upper()
        if city in ('A', 'B', 'C') and city != currentCity:
            cost = playerMoney * 0.1
            playerMoney -= cost
            currentCity = city
            print(f"Перемещение в {currentCity}. Стоимость: {cost}")
        else:
           print("Некорректный город или вы уже в нем находитесь.")
    elif action == "ждать" or action == "wait":
         print("Ожидание...")
    else:
        print("Неверное действие.")
    
    turnNumber += 1
```
**Описание**:
Основной игровой цикл, который выполняется 10 ходов.
- **Генерация цен**: Случайным образом генерируются цены на мех для каждого города.
- **Вывод информации**: Выводится текущий ход, цены на мех, количество денег и меха, а также текущий город игрока.
- **Запрос действия**: Игроку предлагается выбрать одно из действий: `купить`, `продать`, `переместиться`, `ждать`.
- **Обработка действия**:
    - **Покупка**: Запрашивается количество меха для покупки, проверяется наличие денег, производится покупка или выводится сообщение об ошибке.
    - **Продажа**: Запрашивается количество меха для продажи, проверяется наличие меха, производится продажа или выводится сообщение об ошибке.
    - **Перемещение**: Запрашивается город для перемещения, проверяется корректность ввода, списывается стоимость перемещения и меняется текущий город.
    - **Ожидание**: Выводится сообщение об ожидании.
- **Увеличение номера хода**: Переход к следующему ходу.
- **Обработка исключений**: Обработка `ValueError` при некорректном вводе количества меха при покупке или продаже.

## Подсчет и вывод итогового капитала

```python
finalCapital = playerMoney + (playerFur * locals()[f'price{currentCity}'])
print(f"\n--- Итого ---")
print(f"Общий капитал: {finalCapital}")
```

**Описание**:
- Подсчитывает общий капитал игрока, суммируя имеющиеся деньги и стоимость меха по цене текущего города.
- Выводится итоговый капитал игрока.
