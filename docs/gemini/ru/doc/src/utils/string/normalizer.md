# Модуль normalizer

## Обзор

Модуль `normalizer` предоставляет функции для нормализации строк, булевых значений, целых и чисел с плавающей точкой. Он содержит вспомогательные методы для обработки текста, включая удаление HTML-тегов и специальных символов.  Этот модуль призван упростить и стандартизировать входные данные, гарантируя их согласованность.

## Функции

### `normalize_boolean`

**Описание**: Нормализует входные данные в булево значение.

**Параметры**:
- `input_data` (Any): Данные, которые могут представлять булево значение (например, bool, строка, целое число).

**Возвращает**:
- `bool`: Булево представление входных данных. Возвращает исходное значение, если преобразование невозможно.

**Пример использования**:
```python
>>> normalize_boolean('yes')
True
>>> normalize_boolean(0)
False
>>> normalize_boolean(1)
True
```

**Обрабатывает исключения**:
- `Exception`:  Логирует ошибку, если произошла ошибка при преобразовании, и возвращает исходное значение.


### `normalize_string`

**Описание**: Нормализует строку или список строк.

**Параметры**:
- `input_data` (str | list): Входные данные, которые могут быть строкой или списком строк.

**Возвращает**:
- `str`: Очищенная и нормализованная строка в кодировке UTF-8. Возвращает исходное значение, если произошла ошибка.

**Пример использования**:
```python
>>> normalize_string(" Пример строки <b>с HTML</b> ")
'Пример строки с HTML'
>>> normalize_string(['Hello', '  World!  '])
'Hello World!'
```

**Обрабатывает исключения**:
- `TypeError`: Если `input_data` не является строкой или списком.
- `Exception`: Логирует ошибку и возвращает исходное значение в случае возникновения ошибки при обработке строки.


### `normalize_int`

**Описание**: Нормализует входные данные в целое число.

**Параметры**:
- `input_data` (str | int | float | Decimal): Входные данные, которые могут быть числом или его строковым представлением.

**Возвращает**:
- `int`: Целое числовое представление входных данных. Возвращает исходное значение, если преобразование невозможно.

**Пример использования**:
```python
>>> normalize_int('42')
42
>>> normalize_int(3.14)
3
```

**Обрабатывает исключения**:
- `ValueError`, `TypeError`, `InvalidOperation`: Логирует ошибку и возвращает исходное значение при возникновении ошибки преобразования.


### `normalize_float`

**Описание**: Безопасно преобразует входные значения в число с плавающей точкой или список чисел с плавающей точкой.

**Параметры**:
- `value` (Any): Входное значение для преобразования. Может быть одиночным значением (числом или строкой) или итерируемым объектом (списком/кортежем).

**Возвращает**:
- `float` | `List[float]` | `None`: Значение с плавающей точкой, список чисел с плавающей точкой или `None`, если преобразование не выполняется. Возвращает исходное значение, если преобразование невозможно.

**Пример использования**:
```python
>>> normalize_float("3.14")
3.14
>>> normalize_float([1, '2.5', 3])
[1.0, 2.5, 3.0]
```


**Обрабатывает исключения**:
- `ValueError`, `TypeError`: Логирует предупреждение и возвращает исходное значение, если преобразование в float невозможно.

### `normalize_sql_date`

**Описание**: Нормализует дату в формат SQL (YYYY-MM-DD).

**Параметры**:
- `input_data` (str): Данные, которые могут представлять дату (например, строка, объект datetime).

**Возвращает**:
- `str`: Нормализованная дата в формате SQL (YYYY-MM-DD) или исходное значение, если преобразование невозможно.

**Пример использования**:
```python
>>> normalize_sql_date('2024-12-06')
'2024-12-06'
>>> normalize_sql_date('12/06/2024')
'2024-12-06'
```

**Обрабатывает исключения**:
- `Exception`: Логирует ошибку и возвращает исходное значение при возникновении ошибки преобразования.

### `simplify_string`

**Описание**: Упрощает строку, оставляя только буквы, цифры и заменяя пробелы подчеркиваниями.

**Параметры**:
- `input_str` (str): Строка для упрощения.

**Возвращает**:
- `str`: Упрощенная строка. Возвращает исходную строку при ошибке.

### `remove_line_breaks`

**Описание**: Удаляет переводы строк из входной строки.

**Параметры**:
- `input_str` (str): Входная строка.

**Возвращает**:
- `str`: Строка без переводов строк.


### `remove_html_tags`

**Описание**: Удаляет HTML теги из входной строки.

**Параметры**:
- `input_html` (str): Входная HTML строка.

**Возвращает**:
- `str`: Строка без HTML тегов.


### `remove_special_characters`

**Описание**: Удаляет указанные спецсимволы из строки или списка строк.

**Параметры**:
- `input_str` (str | list): Входная строка или список строк.
- `chars` (list[str], optional): Список символов для удаления. По умолчанию - `['#']`.

**Возвращает**:
- `str` | `list`: Обработанная строка или список со удалёнными символами.
