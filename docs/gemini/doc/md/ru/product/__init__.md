# Модуль product

## Обзор

Модуль `product` содержит классы и функции для работы с продуктами.  Модуль предоставляет инструменты для работы с полями продуктов, их переводом и представлением в плоском формате.

## Содержание

* [Классы](#классы)
* [Функции](#функции)


## Классы

### `Product`

**Описание**: Класс `Product` предоставляет методы и атрибуты для работы с продуктами. Подробное описание в файле `product.py`.

### `ProductFields`

**Описание**: Класс `ProductFields` содержит информацию о полях продукта. Подробное описание в файле `product_fields.py`.


## Функции

### `translate_presta_fields_dict`

**Описание**: Функция `translate_presta_fields_dict` переводит многоязычные поля класса `ProductFields`.

**Возвращает**:
- словарь (dict): Переведенные поля продукта.

**Подробности**: Подробное описание функции, включая параметры и возможные исключения, находятся в файле `product_fields/product_fields_translator.py`.
