## Создание рекламной кампании

### 1. Инициализация кампании

**Описание**: Настройка базовых параметров кампании.

**Параметры**:
- `campaign_name` (str): Имя кампании.
- `language` (str): Язык кампании.
- `currency` (str): Валюта кампании.

**Пример использования**:

```python
campaign_name = 'example_campaign'
language = 'EN'
currency = 'USD'
```


### 2. Создание директорий для кампании

**Описание**: Создание директорий для хранения данных кампании и категорий.

**Параметры**:
- `campaign_name` (str): Имя кампании.
- `categories` (list[str]): Список категорий.

**Пример использования**:

```python
categories = ['electronics', 'fashion']
create_directories(campaign_name, categories)
```


### 3. Сохранение конфигурации кампании

**Описание**: Сохранение параметров кампании в конфигурационный файл.

**Параметры**:
- `campaign_name` (str): Имя кампании.
- `campaign_config` (dict): Словарь с конфигурацией кампании.

**Пример использования**:

```python
campaign_config = {'name': campaign_name, 'language': language, 'currency': currency}
save_config(campaign_name, campaign_config)
```


### 4. Сбор данных о продуктах

**Описание**: Сбор данных о продуктах для кампании.

**Параметры**:
- `product_urls` (list[str]): Список URL адресов продуктов.

**Возвращает**:
- `product_data` (dict): Данные о продуктах.

**Пример использования**:

```python
product_urls = ['https://www.aliexpress.com/item/123.html', 'https://www.aliexpress.com/item/456.html']
product_data = collect_product_data(product_urls)
```


### 5. Сохранение данных о продуктах

**Описание**: Сохранение собранных данных о продуктах.

**Параметры**:
- `campaign_name` (str): Имя кампании.
- `product_data` (dict): Данные о продуктах.

**Пример использования**:

```python
save_product_data(campaign_name, product_data)
```


### 6. Создание рекламных материалов

**Описание**: Создание рекламных материалов на основе собранных данных.

**Параметры**:
- `campaign_name` (str): Имя кампании.
- `product_data` (dict): Данные о продуктах.

**Пример использования**:

```python
create_promotional_materials(campaign_name, product_data)
```


### 7. Просмотр и публикация кампании

**Описание**: Просмотр и публикация созданной кампании.

**Параметры**:
- `campaign_name` (str): Имя кампании.

**Пример использования**:

```python
review_campaign(campaign_name)
publish_campaign(campaign_name)
```


## Редактирование рекламной кампании

### 1. Загрузка существующей конфигурации кампании

**Описание**: Загрузка параметров существующей кампании.

**Параметры**:
- `campaign_name` (str): Имя кампании.

**Возвращает**:
- `campaign_config` (dict): Данные о кампании.

**Пример использования**:

```python
campaign_name = 'example_campaign'
campaign_config = load_config(campaign_name)
```


### 2. Обновление конфигурации кампании

**Описание**: Обновление параметров кампании.

**Параметры**:
- `campaign_name` (str): Имя кампании.
- `campaign_config` (dict): Данные о кампании.


**Пример использования**:

```python
campaign_config['language'] = 'RU'
save_config(campaign_name, campaign_config)
```


### 3. Обновление категорий и директорий

**Описание**: Обновление списка категорий и соответствующих директорий.

**Параметры**:
- `campaign_name` (str): Имя кампании.
- `new_categories` (list[str]): Новый список категорий.


**Пример использования**:

```python
new_categories = ['home', 'beauty']
update_categories(campaign_name, new_categories)
```


### 4-5. Сбор и сохранение новых данных о продуктах

**Описание**: Сбор и сохранение обновленных данных о продуктах.  Подробности см. в разделе [Создание рекламной кампании](#создание-рекламной-кампании) пункты 4 и 5.


### 6. Обновление рекламных материалов

**Описание**: Обновление рекламных материалов на основе новых данных.  Подробности см. в разделе [Создание рекламной кампании](#создание-рекламной-кампании) пункт 6.


### 7. Просмотр и публикация обновленной кампании

**Описание**: Просмотр и публикация обновленной кампании.  Подробности см. в разделе [Создание рекламной кампании](#создание-рекламной-кампании) пункт 7.
