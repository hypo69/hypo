# Модуль `aliexpress_promote_deal.py`

## Обзор

Модуль предназначен для подготовки объявлений о скидках на товары с AliExpress в формате, пригодном для публикации в Facebook. Включает функциональность для получения информации о товарах и формирования рекламных материалов.

## Подробнее

Модуль является частью экспериментов по продвижению товаров с AliExpress. Он использует класс `AliPromoDeal` для подготовки данных о товарах и предназначен для автоматизации процесса создания рекламных объявлений.

- `deal_name` (str): Имя акции, используемое для идентификации набора товаров. В данном случае - `'150624_baseus_deals'`.
