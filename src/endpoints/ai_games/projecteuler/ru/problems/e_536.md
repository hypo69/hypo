# Задача No 536
## Функции с суммами цифр

## Условия

Пусть S(n) - сумма цифр натурального числа n в десятичной записи.

Пусть F(n) - количество натуральных чисел i таких, что i ≤ n и S(i) > S(i+1).

Например, F(100) = 9, потому что числа 9, 19, 29, 39, 49, 59, 69, 79, 89 удовлетворяют условию S(i) > S(i+1).

Также, F(1000) = 189 и F(10<sup>6</sup>) = 599 994.

Найдите F(10<sup>18</sup>).
