# Модуль hypotez/src/webdriver/firefox/header.py

## Обзор

Данный модуль содержит функции для определения корневой директории проекта, загрузки настроек из файла `settings.json`, и получения документации из файла `README.MD`.  Он также предоставляет доступ к переменным, описывающим проект (имя проекта, версия, описание, автор, авторские права и ссылка на поддержку разработчика).

## Функции

### `set_project_root`

**Описание**:  Находит корневую директорию проекта, начиная с директории текущего файла. Поиск производится вверх по иерархии директорий, пока не будет найдена директория, содержащая один из указанных маркеров файлов (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, которые используются для определения корневой директории проекта.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена; в противном случае - директория, где расположен текущий скрипт.

**Вызывает исключения**:

- Нет

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла. Поиск производится вверх по иерархии директорий, пока не будет найдена директория, содержащая один из указанных маркеров файлов (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple, optional): Кортеж имен файлов или директорий, которые используются для определения корневой директории проекта. По умолчанию (`('pyproject.toml', 'requirements.txt', '.git')`).

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена; в противном случае - директория, где расположен текущий скрипт.

**Вызывает исключения**:

- Нет


## Переменные

### `__root__`

**Описание**:  Путь к корневой директории проекта.  Инициализируется функцией `set_project_root()`.

**Тип**: `Path`


### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json`.

**Тип**: `dict` или `None`.


### `doc_str`

**Описание**: Строка с содержимым файла `README.MD`.

**Тип**: `str` или `None`.


### `__project_name__`

**Описание**: Имя проекта. Получается из словаря настроек `settings` или имеет значение по умолчанию `hypotez`, если `settings` не задан или пуст.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта. Получается из словаря настроек `settings` или имеет значение по умолчанию пустой строкой, если `settings` не задан или пуст.

**Тип**: `str`


### `__doc__`

**Описание**: Описание проекта (документация).  Получается из файла `README.MD` или имеет значение по умолчанию пустой строкой.

**Тип**: `str`


### `__details__`

**Описание**: Дополнительные подробности о проекте.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта. Получается из словаря настроек `settings` или имеет значение по умолчанию пустой строкой.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права на проект. Получается из словаря настроек `settings` или имеет значение по умолчанию пустой строкой.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка для поддержки разработчика. Получается из словаря настроек `settings` или имеет значение по умолчанию строкой с ссылкой.

**Тип**: `str`
