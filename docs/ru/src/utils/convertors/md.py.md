# Модуль `src.utils.convertors.md2dict`

## Обзор

Модуль `src.utils.convertors.md2dict` предназначен для конвертации строки Markdown в структурированный словарь, включая извлечение JSON содержимого, если оно присутствует. Модуль поддерживает работу как в Windows, так и в Unix системах.

## Оглавление

1. [Функции](#Функции)
   - [`md2html`](#md2html)
   - [`md2dict`](#md2dict)
 

## Функции

### `md2html`

**Описание**: Конвертирует строку Markdown в HTML.

**Параметры**:
- `md_string` (str): Строка Markdown для конвертации.
- `extras` (list, optional): Список расширений markdown2. По умолчанию `None`.

**Возвращает**:
- `str`: HTML-представление Markdown.

**Вызывает исключения**:
- `Exception`: Возникает в случае ошибки при преобразовании Markdown в HTML.

### `md2dict`

**Описание**: Конвертирует строку Markdown в структурированный словарь.

**Параметры**:
- `md_string` (str): Строка Markdown для конвертации.
- `extras` (list, optional): Список расширений markdown2 для md2html. По умолчанию `None`.

**Возвращает**:
- `Dict[str, list[str]]`: Структурированное представление Markdown содержимого, где ключи — заголовки, а значения — список строк содержимого раздела.

**Вызывает исключения**:
- `Exception`: Возникает в случае ошибки при парсинге Markdown в структурированный словарь.
