# Задача No 294
## Сумма цифр, кратная 23
## Условия
Пусть f(n) - сумма цифр целого числа n. Например, f(2023) = 2 + 0 + 2 + 3 = 7.
Пусть S(k) - количество целых чисел n, где 10<sup>k-1</sup> ≤ n < 10<sup>k</sup>, таких, что f(n) кратно 23.
Например, S(1) = 0, S(2) = 4 и S(3) = 55.

Найдите S(10<sup>15</sup>) mod 10<sup>9</sup>.
