# Модуль hypotez/src/suppliers/aliexpress/gui/styles.py

## Обзор

Модуль `styles.py` содержит функции для управления стилями пользовательского интерфейса (UI) для алиэкспресс-раздела приложения `hypotez`. Он определяет константу `MODE` и функцию `set_fixed_size` для изменения размеров виджетов.


## Константы

### `MODE`

**Описание**: Переменная, определяющая режим работы приложения.  В данном случае она имеет значение `'dev'`.  (Дополнительные детали о значении режима требуются для полной документации.)


## Функции

### `set_fixed_size`

**Описание**: Устанавливает фиксированный размер для заданного виджета.

**Параметры**:

- `widget` (QtWidgets.QWidget): Виджет, для которого необходимо установить размер.
- `width` (int): Ширина виджета.
- `height` (int): Высота виджета.

**Возвращает**:
  -  Не имеет возвращаемого значения.
