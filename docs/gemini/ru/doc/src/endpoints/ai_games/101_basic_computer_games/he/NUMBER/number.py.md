# Игра "Угадай число"

## Обзор

Этот модуль реализует игру "Угадай число", где компьютер загадывает случайное число от 1 до 100, а игрок пытается его угадать. Компьютер сообщает, является ли догадка слишком высокой или слишком низкой, пока игрок не угадает число.
