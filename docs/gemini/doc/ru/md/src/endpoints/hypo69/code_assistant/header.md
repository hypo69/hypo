# Модуль `hypotez/src/logger/header.py`

## Обзор

Модуль `hypotez/src/logger/header.py` определяет корневой путь к проекту `hypotez`. Все импорты строятся относительно этого пути.  В дальнейшем предполагается перенести определение корневого пути в системную переменную.

## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная с текущего каталога и двигаясь вверх по дереву каталогов.  Останавливается на первом каталоге, содержащем один из указанных файлов/каталогов.

**Параметры**:

- `marker_files` (tuple): Кортеж файлов или каталогов, по которым определяется корневой каталог проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневому каталогу проекта, если он найден. В противном случае возвращает директорию, где расположен текущий скрипт.


**Пример использования**:

```python
root_path = set_project_root()
print(root_path)
```


**Вызывает исключения**:

-  Не вызывает исключений.


## Глобальные переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта, полученный из функции `set_project_root`.


### `settings`

**Описание**: Словарь со настройками проекта, загруженный из файла `src/settings.json`.

**Возвращает**:

- `dict`: Словарь настроек или `None`, если файл не найден или содержит некорректный JSON.


### `doc_str`

**Описание**: Строка с содержимым файла `README.MD` в корне проекта.

**Возвращает**:

- `str`: Строка с содержимым файла `README.MD` или `None` при ошибке.



### `__project_name__`

**Описание**:  Название проекта, полученное из настроек (из `settings.json`) или заданное по умолчанию как `'hypotez'`.

**Возвращает**:

- `str`: Название проекта.

### `__version__`

**Описание**: Версия проекта, полученное из настроек (`settings.json`) или пустая строка, если не найдена в настройках.

**Возвращает**:

- `str`: Версия проекта.

### `__doc__`

**Описание**: Документация проекта (содержимое файла `README.MD`),  или пустая строка, если файл не найден.

**Возвращает**:

- `str`: Документация проекта.


### `__details__`

**Описание**: Дополнительные детали проекта (пустая строка по умолчанию).

**Возвращает**:

- `str`: Дополнительные детали проекта.


### `__author__`

**Описание**: Автор проекта, полученное из настроек (`settings.json`) или пустая строка, если не найдена в настройках.

**Возвращает**:

- `str`: Автор проекта.


### `__copyright__`

**Описание**: Авторские права проекта, полученное из настроек (`settings.json`) или пустая строка, если не найдена в настройках.

**Возвращает**:

- `str`: Авторские права проекта.


### `__cofee__`

**Описание**: Ссылка на страницу для поддержки разработчика (бонус).

**Возвращает**:

- `str`: Ссылка на страницу для поддержки.
