Hot Products Response
=======================

.. module:: hypotez.src.suppliers.aliexpress.api.models.hotproducts
   :synopsis: Module containing the HotProductsResponse class.


This module defines the `HotProductsResponse` class, which represents the response from the AliExpress API for hot product listings. It includes information about the current page, record count, and total products available.
