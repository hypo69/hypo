# Модуль hypotez/src/suppliers/etzmaleh/__init__.py

## Обзор

Этот модуль предоставляет начальную точку входа для поставщика данных Etzmaleh. Он импортирует класс `Graber` из подмодуля `graber`.  По умолчанию режим работы установлен в `dev`.


## Переменные

### `MODE`

**Описание**:  Переменная, определяющая режим работы. В текущей реализации она имеет значение `'dev'`.

## Импорты

### `from .graber import Graber`

**Описание**: Импортирует класс `Graber` из модуля `graber`.  Это указывает на то, что модуль `etzmaleh` использует функциональность, реализованную в `graber`.
