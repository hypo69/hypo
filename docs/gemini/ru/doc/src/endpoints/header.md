# Модуль hypotez/src/endpoints/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и документации из `README.MD`. Он также предоставляет переменные с метаданными о проекте, такие как имя, версия, автор и авторские права.


## Функции

### `set_project_root`

**Описание**:  Находит корневую директорию проекта, начиная с текущей директории файла и поднимаясь вверх по дереву директорий, пока не найдёт директорию, содержащую один из файлов-маркеров (pyproject.toml, requirements.txt, .git).

**Параметры**:
- `marker_files` (tuple): Кортеж имён файлов или директорий, по которым определяется корень проекта.

**Возвращает**:
- `Path`: Путь к корневой директории, если найдена, иначе - директория, в которой расположен скрипт.

**Вызывает исключения**:
- Нет

### Загрузка настроек


**Описание**: Загружает настройки из файла `settings.json` в корневой директории проекта.

**Параметры**:
- Нет


**Возвращает**:
- Нет

**Вызывает исключения**:
- `FileNotFoundError`: Если файл `settings.json` не найден.
- `json.JSONDecodeError`: Если содержимое файла `settings.json` не является корректным JSON.


### Загрузка документации


**Описание**: Загружает документацию из файла `README.MD` в корневой директории проекта.

**Параметры**:
- Нет


**Возвращает**:
- Нет

**Вызывает исключения**:
- `FileNotFoundError`: Если файл `README.MD` не найден.
- `json.JSONDecodeError`: Если содержимое файла `README.MD` не является корректным JSON.




## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root`.

**Тип**: `Path`

### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json`.

**Тип**: `dict`

### `doc_str`

**Описание**: Строка с содержимым файла `README.MD`.

**Тип**: `str`

### `__project_name__`

**Описание**: Имя проекта, полученное из настроек или заданное по умолчанию.

**Тип**: `str`

### `__version__`

**Описание**: Версия проекта, полученная из настроек или заданная по умолчанию.

**Тип**: `str`

### `__doc__`

**Описание**: Документация проекта, полученная из файла README.md.

**Тип**: `str`

### `__details__`

**Описание**: Детали проекта (по умолчанию пустая строка).

**Тип**: `str`

### `__author__`

**Описание**: Автор проекта, полученный из настроек или заданный по умолчанию.

**Тип**: `str`

### `__copyright__`

**Описание**: Авторские права на проект, полученные из настроек или заданные по умолчанию.

**Тип**: `str`

### `__cofee__`

**Описание**: Ссылка на страницу для поддержки разработчика.

**Тип**: `str`
