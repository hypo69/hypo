# Модуль profiling

## Обзор

Этот модуль предоставляет инструменты для анализа характеристик агентов, таких как распределение возрастов, типичные интересы и т.д.  Все методы построения графиков также возвращают Pandas DataFrame с данными, использованными для построения.

## Функции

### `plot_age_distribution`

**Описание**: Строит гистограмму распределения возрастов заданных агентов.

**Параметры**:

- `agents` (List[TinyPerson]): Список агентов, для которых необходимо построить распределение возрастов.
- `title` (str, optional): Заголовок графика. По умолчанию "Распределение возрастов".
- `show` (bool, optional): Флаг, указывающий, нужно ли отображать график. По умолчанию `True`.

**Возвращает**:

- `pd.DataFrame`: DataFrame с данными, использованными для построения графика (возрасты агентов).


### `plot_interest_distribution`

**Описание**: Строит круговую диаграмму распределения интересов заданных агентов.

**Параметры**:

- `agents` (List[TinyPerson]): Список агентов, для которых необходимо построить распределение интересов.
- `title` (str, optional): Заголовок графика. По умолчанию "Распределение интересов".
- `show` (bool, optional): Флаг, указывающий, нужно ли отображать график. По умолчанию `True`.

**Возвращает**:

- `pd.DataFrame`: DataFrame с данными, использованными для построения графика (интересы агентов).
