# Игра "Жизнь 2"

## Обзор

Данный модуль реализует симуляцию клеточного автомата "Жизнь 2", разработанную Джоном Конвеем.  Игра представляет собой поле клеток, где каждая клетка может быть либо живой, либо мертвой. Состояние клетки в следующем поколении зависит от количества живых соседей в текущем поколении.  Игрок может задавать начальную конфигурацию живых клеток, и наблюдать за эволюцией этой конфигурации с течением времени.

## Функции

### `initialize_grid`

**Описание**: Создает и инициализирует игровое поле (сетку) с мертвыми клетками.

**Параметры**:
- `num_rows` (int): Количество строк в сетке.
- `num_cols` (int): Количество столбцов в сетке.

**Возвращает**:
- `list of lists`: Игровое поле, представленное в виде списка списков.

### `place_alive_cells`

**Описание**: Размещает начальные живые клетки на игровом поле случайным образом.

**Параметры**:
- `grid` (list of lists): Игровое поле.
- `initial_alive_cells` (int): Количество начальных живых клеток.

### `display_grid`

**Описание**: Выводит текущее состояние игрового поля на экран.

**Параметры**:
- `grid` (list of lists): Игровое поле.

### `count_alive_neighbors`

**Описание**: Подсчитывает количество живых соседей у клетки.

**Параметры**:
- `grid` (list of lists): Игровое поле.
- `row` (int): Индекс строки клетки.
- `col` (int): Индекс столбца клетки.

**Возвращает**:
- `int`: Количество живых соседей.

### `compute_next_generation`

**Описание**: Вычисляет следующее поколение клеток на основе текущего состояния поля и правил игры "Жизнь".

**Параметры**:
- `grid` (list of lists): Текущее игровое поле.

**Возвращает**:
- `list of lists`: Новое игровое поле, представляющее следующее поколение.
