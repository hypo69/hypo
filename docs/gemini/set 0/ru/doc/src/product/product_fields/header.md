# Модуль hypotez/src/product/product_fields/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и получения документации из файла `README.MD`. Он также предоставляет доступ к глобальным переменным, представляющим имя проекта, версию, документацию, подробную информацию, автора, копирайт и ссылку на поддержку разработчиков.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная от текущей директории и проходя вверх по директориям до тех пор, пока не будет найдена директория, содержащая один из указанных файлов или директорий.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, которые используются для определения корневой директории проекта. По умолчанию: `('pyproject.toml', 'requirements.txt', '.git')`.


**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена, иначе путь к текущей директории.

**Вызывает исключения**:

- Нет


### Глобальные переменные

**Описание**: Модуль определяет несколько глобальных переменных, хранящих информацию о проекте.

- `__root__`: Путь к корневой директории проекта, полученный из функции `set_project_root`.
- `settings`: Словарь настроек, загруженный из файла `settings.json`.
- `doc_str`: Строка документации, загруженная из файла `README.MD`.
- `__project_name__`: Имя проекта, полученное из `settings` или значение по умолчанию 'hypotez'.
- `__version__`: Версия проекта, полученная из `settings` или пустая строка по умолчанию.
- `__doc__`: Строка документации, полученная из `doc_str` или пустая строка по умолчанию.
- `__details__`: Подробная информация о проекте (пустая строка по умолчанию).
- `__author__`: Автор проекта, полученный из `settings` или пустая строка по умолчанию.
- `__copyright__`: Копирайт проекта, полученный из `settings` или пустая строка по умолчанию.
- `__cofee__`: Ссылка на поддержку разработчика (по умолчанию).


**Обработка исключений**:

- При чтении файла `settings.json` или `README.MD` могут возникнуть исключения `FileNotFoundError` или `json.JSONDecodeError`. Эти исключения обрабатываются с помощью блоков `try...except`, и в случае возникновения исключения соответствующие переменные остаются с их значениями по умолчанию.
