# Модуль `hypotez/src/webdriver/chrome/_examples/driver.py`

## Обзор

Этот модуль предоставляет примеры использования классов `Driver` и `Chrome` из модуля `src.webdriver`.  Он демонстрирует различные методы, такие как навигация по URL, извлечение домена, сохранение куки, обновление страницы, прокрутка, получение языка страницы, установка пользовательского User-Agent, поиск элементов и получение текущего URL.
