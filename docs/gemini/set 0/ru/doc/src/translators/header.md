# Модуль hypotez/src/translators/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`, а также получения информации о проекте, например, название, версию, описание и прочее.  Модуль использует `gs` для доступа к файловой системе проекта и `json` для обработки данных настроек.


## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла и идя вверх по дереву директорий. Останавливается на первой директории, содержащей один из указанных маркеров файлов (например, `pyproject.toml`, `requirements.txt`, `.git`). Если корневая директория не найдена, возвращает директорию, в которой находится текущий файл.  Добавляет корневую директорию в `sys.path`, если она там еще не присутствует.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, используемых в качестве маркеров корневой директории проекта. По умолчанию содержит `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена, в противном случае – директорию текущего файла.


### `<Имя функции не определено>`

**Описание**: Функция, которая не имеет имени (описания). Похоже, вызывается в строке `__root__ = set_project_root()`

**Параметры**:
- Не определены.

**Возвращает**:
- `None` (На основе контекста)

**Вызывает исключения**:
- Не определены.


## Переменные

### `__root__`

**Описание**:  Переменная, содержащая путь к корневой директории проекта.  Получается вызовом функции `set_project_root()`.

**Тип**: `Path`


### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженные из файла `src/settings.json`.  Если файл не найден или некорректен, `settings` устанавливается в `None`.

**Тип**: `dict` | `None`


### `doc_str`

**Описание**: Строка с описанием проекта, загруженная из файла `src/README.MD`. Если файл не найден или некорректен, `doc_str` устанавливается в `None`.

**Тип**: `str` | `None`



### `__project_name__`

**Описание**: Название проекта, полученное из настроек (`settings`). По умолчанию равно `'hypotez'`.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта, полученная из настроек (`settings`). По умолчанию пустая строка.

**Тип**: `str`


### `__doc__`

**Описание**: Описание проекта, полученное из `doc_str`. По умолчанию пустая строка.

**Тип**: `str`


### `__details__`

**Описание**: Дополнительные детали проекта. По умолчанию пустая строка.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта, полученный из настроек (`settings`). По умолчанию пустая строка.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права проекта, полученные из настроек (`settings`). По умолчанию пустая строка.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка для поддержки разработчика через кофе. Получается из настроек (`settings`). По умолчанию – указанная ссылка.

**Тип**: `str`
