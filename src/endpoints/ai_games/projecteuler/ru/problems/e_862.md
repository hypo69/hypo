# Задача No 862
## Загадочный контур
## Условия
Рассмотрим целочисленную точку с координатами (x,y) на плоскости. Назовем контур, проведенный через целые точки, «загадочным», если он удовлетворяет двум условиям:

*   Контур образует замкнутую ломаную линию, состоящую из отрезков, которые параллельны осям координат.
*   Ломаная проходит через n различных точек (x₁, y₁), (x₂, y₂), ..., (xₙ, yₙ) в произвольном порядке.

Например, точки (1, 1), (1, 4), (3, 4), (3, 3), (2, 3), (2, 2), (1, 2) и (1, 1) образуют загадочный контур.

Обозначим как M(n) количество различных загадочных контуров, которые можно построить, используя n различных точек. Два контура считаются различными, если они не являются трансляциями друг друга.

*   M(4) = 1
*   M(6) = 2
*   M(10) = 22
*   M(100) = 165501088

Найдите M(20000).

**Примечание:** Контур не должен пересекаться сам с собой. То есть последовательность точек, образующих контур, может быть записана как (x₁, y₁), (x₂, y₂), ..., (xₙ, yₙ), (x₁, y₁), где каждая пара соседних точек либо имеет одну и ту же координату x, либо одну и ту же координату y.
