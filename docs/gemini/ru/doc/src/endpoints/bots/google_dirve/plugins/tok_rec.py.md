# Модуль определения токенов `tok_rec`

## Обзор

Модуль `tok_rec` содержит функции для идентификации токенов определенного формата. В частности, модуль предоставляет функцию `is_token`, которая проверяет, является ли входная строка токеном, соответствующим определенным критериям длины и формата.

## Подробней

Этот модуль предназначен для работы с токенами, которые предположительно используются для аутентификации или авторизации. Функция `is_token` проверяет, соответствует ли переданная строка формату токена (например, проверяет длину строки и наличие определенного символа на определенной позиции).

## Функции

### `is_token`

```python
def is_token(token) -> bool:
    """
    Args:
        token (str): Строка, которую необходимо проверить на соответствие формату токена.

    Returns:
        bool: `True`, если строка соответствует формату токена, `False` в противном случае.

    Example:
        >>> is_token('some token / some data')
        False
    """
```

**Описание**: Определяет, является ли переданная строка токеном. Токен должен соответствовать определенным критериям длины (62 символа) и формата (наличие символа "/" на второй позиции).

**Параметры**:
- `token` (str): Строка, которую необходимо проверить.

**Возвращает**:
- `bool`: `True`, если строка соответствует формату токена, `False` в противном случае.

**Примеры**:
```python
>>> is_token('AAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAA/AAAAAAAAAAAAAAAAAAAAAA')
True

>>> is_token('AAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAA-AAAAAAAAAAAAAAAAAAAAAA')
False

>>> is_token('AAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAA/AAAAAAAAAAAAAAAAAAAAAA')
False
