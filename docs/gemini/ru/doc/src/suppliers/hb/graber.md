# Модуль hypotez/src/suppliers/hb/graber.py

## Обзор

Модуль `hypotez/src/suppliers/hb/graber.py` содержит класс `Graber`, предназначенный для сбора данных о товарах с сайта `hb.co.il`. Он позволяет задавать префикс для поставщика и включает поддержку предварительных действий (например, закрытия всплывающих окон) перед отправкой запроса к веб-драйверу.

### `Graber`

**Описание**: Класс `Graber` отвечает за сбор данных с сайта `hb.co.il`. Он обрабатывает поля страницы товара и предоставляет функции для их извлечения.

**Инициализация**:

```python
def __init__(self, driver: Driver):
    """Инициализация класса сбора полей товара.

    Args:
        driver (Driver): Объект драйвера веб-драйвера.
    """
    self.supplier_prefix = 'hb'
    super().__init__(supplier_prefix=self.supplier_prefix, driver=driver)
    Context.locator_for_decorator = None
```

**Примечания:**

- Инициализирует префикс поставщика `self.supplier_prefix` как 'hb'.
- Вызывает конструктор родительского класса `Grbr`, передавая `supplier_prefix` и `driver`.
- Устанавливает `Context.locator_for_decorator` в `None`.  Это важно для корректной работы декоратора.
