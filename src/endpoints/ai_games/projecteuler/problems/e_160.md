# Задача No 160
## Факториал, сокращенный до последних пяти цифр

## Условия
Для натурального числа n, пусть f(n) будет результатом произведения всех цифр числа n. Например, f(123) = 1 * 2 * 3 = 6 и f(93) = 9 * 3 = 27.
Для натурального числа n, определим g(n) как:
g(n) = n! / (10^k), где k - наибольшая целая степень 10, на которую делится n!.

Например:
5! = 120
g(5) = 120 / 10^1 = 12.
10! = 3628800
g(10) = 3628800 / 10^2 = 36288.
Определим h(n) как последние пять цифр g(n).
Например:
h(5) = 00012
h(10) = 6288.
Вам дано, что h(1000) = 06048.

Найдите h(10<sup>12</sup>).
