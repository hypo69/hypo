# Модуль hypotez/src/goog/spreadsheet/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и чтения документации из файла `README.MD`.  Он инициализирует переменные, хранящие информацию о проекте, такие как имя, версия, документация, автор и прочее.

## Функции

### `set_project_root`

**Описание**:  Находит корневую директорию проекта, начиная с текущей директории и поднимаясь по иерархии директорий.  Останавливается на первой директории, содержащей один из файлов-маркеров (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж с именами файлов или директорий, которые используются для идентификации корневой директории проекта.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена, иначе путь к текущей директории.

**Вызывает исключения**:

- Нет


### `set_project_root`


**Описание**: Находит корневую директорию проекта, начиная с текущей директории и поднимаясь по иерархии директорий.  Останавливается на первой директории, содержащей один из файлов-маркеров (например, `pyproject.toml`, `requirements.txt`, `.git`). Если корневая директория не найдена, используется текущая.

**Параметры**:

- `marker_files` (tuple): Кортеж с именами файлов или директорий, которые используются для идентификации корневой директории проекта.

**Возвращает**:

- `Path`: Путь к корневой директории проекта.

**Вызывает исключения**:

- Нет


## Переменные

### `__root__`

**Описание**: Переменная, хранящая путь к корневой директории проекта. Инициализируется функцией `set_project_root()`.

**Тип**: `Path`

### `settings`

**Описание**: Переменная, хранящая словарь настроек проекта, загруженный из файла `settings.json`.

**Тип**: `dict` или `None`


### `doc_str`

**Описание**: Переменная, хранящая строку с текстом документации, прочитанной из файла `README.MD`.

**Тип**: `str` или `None`

### `__project_name__`

**Описание**: Переменная, хранящая имя проекта.  По умолчанию, если файл настроек не существует или не содержит `"project_name"`, значение устанавливается в "hypotez".

**Тип**: `str`

### `__version__`

**Описание**: Переменная, хранящая версию проекта.  По умолчанию, если файл настроек не существует или не содержит `"version"`, значение устанавливается в пустую строку.

**Тип**: `str`

### `__doc__`

**Описание**: Переменная, хранящая документацию проекта. По умолчанию, если файл `README.MD` не существует или не может быть прочитан, значение устанавливается в пустую строку.

**Тип**: `str`

### `__details__`

**Описание**: Переменная, хранящая дополнительные подробности о проекте. По умолчанию, значение пустая строка.

**Тип**: `str`

### `__author__`

**Описание**: Переменная, хранящая имя автора проекта. По умолчанию, если файл настроек не существует или не содержит `"author"`, значение устанавливается в пустую строку.

**Тип**: `str`

### `__copyright__`

**Описание**: Переменная, хранящая копирайт проекта. По умолчанию, если файл настроек не существует или не содержит `"copyright"`, значение устанавливается в пустую строку.

**Тип**: `str`


### `__cofee__`

**Описание**: Переменная, хранящая ссылку на спонсорскую страницу для поддержки разработчика. По умолчанию, значение устанавливается в ссылку на страницу boosty.to.

**Тип**: `str`
