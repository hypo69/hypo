## Конфигурация

### `supplier`

**Описание**: Название поставщика.

**Значение**: `"ivory"`

### `supplier_id`

**Описание**: Идентификатор поставщика.

**Значение**: `"11343"`

### `supplier_prefix`

**Описание**: Префикс поставщика.

**Значение**: `"ivory"`

### `start_url`

**Описание**: Начальный URL для сбора данных.

**Значение**: `"https://www.ivory.co.il/"`

### `login_url`

**Описание**: URL для авторизации на сайте поставщика.

**Значение**: `"https://www.ivory.co.il/"`

### `price_rule`

**Описание**: Правило расчета цены товара.

**Значение**: `"*1.43"`

### `collect_products_from_categorypage`

**Описание**: Указывает, нужно ли собирать товары со страниц категорий.

**Значение**: `false`

### `if_login`

**Описание**: Указывает, требуется ли вход в систему для работы.

**Значение**: `true`

### `scenario_files`

**Описание**: Массив файлов сценариев.

**Значение**: `[]` (пустой массив)

### `last_runned_scenario`

**Описание**: Последний запущенный сценарий.

**Значение**: `""` (пустая строка)

### `excluded`

**Описание**: Список исключений.

**Значение**: `[]` (пустой массив)
