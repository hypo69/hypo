# Модуль hypotez/src/goog/drive/drive.py

## Обзор

Данный модуль предоставляет базовый класс `GoogleDriveHandler` для взаимодействия с Google Диском.  Он позволяет загружать файлы в указанную папку. Модуль использует библиотеку `google-api-python-client` для работы с API Google Диска.  Включает в себя методы для получения и сохранения токенов доступа, что позволяет избежать повторного входа в систему.

## Классы

### `GoogleDriveHandler`

**Описание**: Класс `GoogleDriveHandler` обрабатывает взаимодействие с Google Диском. Он хранит имя папки и данные авторизации.

**Методы**:

- `__init__(self, folder_name: str)`
    **Описание**: Инициализирует обработчик.
    **Параметры**:
        - `folder_name` (str): Название папки на Google Диске, куда будут загружаться файлы.
    **Возвращает**:
        - None


- `_create_credentials(self)`
    **Описание**: Получает и сохраняет токен доступа для Google Диска.
    **Параметры**:
        - None
    **Возвращает**:
        - Credentials: Объект, содержащий данные авторизации.
    **Вызывает исключения**:
        - Возможные исключения при работе с авторизацией.


- `upload_file(self, file_path: Path)`
    **Описание**: Загружает файл на Google Диск в указанную папку.
    **Параметры**:
        - `file_path` (Path): Путь к файлу на локальном диске.
    **Возвращает**:
        - None
    **Вызывает исключения**:
        - Возможные исключения при работе с загрузкой файла.


## Функции

### `main()`

**Описание**: Функция `main` демонстрирует базовые возможности API Google Диска. Она запрашивает список файлов в корневом каталоге Google Диска, используя полученные ранее учетные данные.

**Параметры**:
    - None
**Возвращает**:
    - None
**Вызывает исключения**:
    - Возможные исключения при работе с API Google Диска.
