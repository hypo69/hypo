# Модуль `utils`

## Обзор

Модуль `utils` содержит вспомогательные функции для административной части Telegram-бота, в частности, функцию для удаления сообщений.

## Функции

### `process_dell_text_msg`

**Описание**: Асинхронная функция для обработки удаления текстовых сообщений. Она удаляет последнее сообщение пользователя, сохраненное в состоянии, а также само сообщение, вызвавшее функцию.

**Параметры**:
- `message` (Message): Объект сообщения от Telegram.
- `state` (FSMContext): Объект контекста FSM (конечного автомата), используемый для хранения и доступа к данным о состоянии пользователя.

**Возвращает**:
- `None`: Функция не возвращает значения явно.

**Вызывает исключения**:
- `Exception`: В случае ошибки при удалении сообщения, функция логирует ошибку и продолжает выполнение.
