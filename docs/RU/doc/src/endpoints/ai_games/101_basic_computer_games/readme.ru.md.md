## История

В 1973 году вышла книга [101 BASIC Computer Games](http://www.bitsavers.org/pdf/dec/_Books/101_BASIC_Computer_Games_Mar75.pdf), ставшая бестселлером. В ней были представлены 101 игра на языке BASIC. Книга была опубликована компанией Digital Equipment Corporation (DEC). В то время компьютеры были большими и дорогими, и BASIC был создан для облегчения обучения программированию. Книга была доступна по цене 7.50 долларов, что было значительным вкладом в образование и культуру. Для создания книги DEC объявила конкурс, приглашая программистов присылать свои игры на BASIC. Авторы лучших работ получали гонорар, а их игры включались в сборник.
DEC предъявляла строгие требования к оформлению присылаемых материалов: использование белой, нелинованной бумаги, свежей черной ленты для телетайпа, а также подробную информацию об авторе. Кроме основной книги, были изданы два дополнительных руководства. Книга выдержала три переиздания:
    - 1-е издание: июль 1973 года.
    - 2-е издание: апрель 1974 года.
    - 3-е издание: март 1975 года.
Авторские права принадлежат Digital Equipment Corporation, Maynard, Massachusetts 01754.

## О проекте

В этом репозитории представлена *интерпретация* оригинальных игр из книги "101 BASIC Computer Games", переписанных на языке Python. Каждая игра сопровождается исходным кодом и кратким пояснением. Проект создан для демонстрации истории программирования и обучения базовым алгоритмам.

## Как использовать репозиторий

1.  Выберите игру из списка.
2.  Откройте соответствующий файл с кодом.
3.  Запустите программу и попробуйте сыграть.
4.  Изучите код, чтобы понять, как он работает.
5.  Экспериментируйте: измените код, добавьте свои идеи или создайте новые игры на основе этих примеров.
