## Переменные

### `MODE`

**Описание**:  Строковая переменная, которая, вероятно, задаёт режим работы (например, 'dev', 'prod').

**Значение**:  'dev'.

### `__root__`

**Описание**: Путь к корню проекта `hypotez`.

**Тип**: `pathlib.Path`

**Получение**: Строка `os.getcwd()` обрезается до части, содержащей имя папки 'hypotez'.
