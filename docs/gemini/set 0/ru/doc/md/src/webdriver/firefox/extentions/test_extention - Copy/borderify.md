# borderify.js

## Обзор

Этот скрипт добавляет красную границу толщиной 5 пикселей к телу документа. Также выводит диалоговое окно alert().

## Функции

### `document.body.style.border = "5px solid red";`

**Описание**: Устанавливает свойство `border` элемента `document.body` на "5px solid red". Это добавляет красную границу толщиной 5 пикселей к телу документа.

**Параметры**:

* Не принимает параметров.

**Возвращает**:

*  Не возвращает значение.


### `alert()`

**Описание**: Выводит диалоговое окно с сообщением alert().

**Параметры**:

* Не принимает параметров.

**Возвращает**:

* Не возвращает значение.

**Вызывает исключения**:

* Не вызывает исключения.
