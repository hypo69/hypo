# Модуль openai

## Обзор

Этот модуль предоставляет инструменты для работы с API OpenAI. Он содержит классы и функции для взаимодействия с различными моделями обработки естественного языка и генерации текста.

## Функции

### `get_completion`

**Описание**: Функция для получения ответа от модели OpenAI.

**Параметры**:
- `prompt` (str): Запрос к модели.
- `model` (Optional[str], optional): Модель OpenAI для использования. По умолчанию использует модель по умолчанию.
- `temperature` (Optional[float], optional): Параметр температуры для модели. По умолчанию 0.7.
- `max_tokens` (Optional[int], optional): Максимальное количество токенов в ответе. По умолчанию 150.
- `n` (Optional[int], optional): Количество ответов, которые должна вернуть модель. По умолчанию 1.


**Возвращает**:
- `dict | None`: Словарь с результатом или `None` при ошибке.


**Вызывает исключения**:
- `OpenAIAPIError`: Возникает при ошибках в запросе к API OpenAI.
- `ValueError`: Возникает при некорректных параметрах.
