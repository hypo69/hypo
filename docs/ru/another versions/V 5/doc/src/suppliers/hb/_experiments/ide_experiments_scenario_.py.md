# Модуль ide_experiments_scenario_.py

## Обзор

Модуль `ide_experiments_scenario_.py` предназначен для экспериментов с сценариями наполнения полей продукта (`product_fields`) от поставщика `hb` (HB Dead Sea). Он содержит код для запуска сценариев, связанных с категорией товаров "טיפוח כפות ידיים ורגליים" (уход за руками и ногами) на сайте поставщика.

## Подробней

Этот файл является частью экспериментов, связанных с наполнением данными о продуктах от поставщика HB Dead Sea. Он использует различные модули и классы из проекта `hypotez`, такие как `Supplier`, `Product`, `ProductFields`, `Driver` и другие. Основная цель - автоматизировать процесс сбора и обработки информации о товарах с сайта поставщика для дальнейшей загрузки в систему.

## Классы

В данном файле напрямую не определены классы, но используются экземпляры классов из других модулей:

- `Supplier` (из `src.suppliers`): Представляет поставщика HB Dead Sea.
- `Product` (из `src.product`): Представляет продукт.
- `ProductFields` (из `src.product`): Представляет поля продукта.
- `Driver` (из `src.webdriver.driver`): Представляет веб-драйвер для взаимодействия с сайтом поставщика.

## Функции

### `run_scenarios`

```python
ret = run_scenarios(s, s.current_scenario)
```

**Описание**: Запускает сценарии для сбора данных о продуктах.

**Как работает функция**:
Функция `run_scenarios` принимает объект поставщика `s` и текущий сценарий `s.current_scenario` в качестве аргументов. Она выполняет набор действий, определенных в сценарии, для извлечения информации о продуктах с веб-сайта поставщика.

**Параметры**:
- `s` (Supplier): Объект поставщика, содержащий информацию о поставщике и его настройках.
- `s.current_scenario` (dict): Словарь, представляющий текущий сценарий, включая URL, название, условие и категории PrestaShop.

**Возвращает**:
- `None`: Возвращаемое значение не указано явно (`...`).

**Примеры**:

```python
s.current_scenario: dict =  {
      "url": "https://hbdeadsea.co.il/product-category/bodyspa/feet-hand-treatment/",
      "name": "טיפוח כפות ידיים ורגליים",
      "condition": "new",
      "presta_categories": {
        "default_category": 11259,
        "additional_categories": []
      }
    }

ret = run_scenarios(s, s.current_scenario)
