# Модуль enrichment.py

## Обзор

Этот модуль предоставляет класс `TinyEnricher`, предназначенный для обогащения контента с использованием LLM. Класс использует шаблоны для составления сообщений и отправки их в LLM, а затем обрабатывает полученный ответ.

## Классы

### `TinyEnricher`

**Описание**: Класс `TinyEnricher` расширяет `JsonSerializableRegistry` и отвечает за обогащение контента.  Он хранит контекст и использует его при запросе к LLM, если это задано.

**Методы**:

- `__init__`: Инициализирует объект `TinyEnricher`.
- `enrich_content`: Обогащает контент с помощью LLM.


#### `__init__`

**Описание**: Конструктор класса `TinyEnricher`.

**Параметры**:

- `use_past_results_in_context` (bool): Флаг, указывающий, следует ли использовать прошлые результаты в контексте запроса к LLM. По умолчанию False.


#### `enrich_content`

**Описание**: Метод обогащает контент с использованием LLM.

**Параметры**:

- `requirements` (str): Требования к обогащению.
- `content` (str): Контент для обогащения.
- `content_type` (str, optional): Тип контента. По умолчанию None.
- `context_info` (str, optional): Дополнительная контекстная информация. По умолчанию пустая строка.
- `context_cache` (list, optional): Список с контекстом. По умолчанию None.
- `verbose` (bool, optional): Флаг, указывающий на подробный вывод. По умолчанию False.


**Возвращает**:

- `dict | None`: Результат обогащения или `None`, если LLM вернул `None`.


**Вызывает исключения**:

- Возможны исключения, возникающие при взаимодействии с LLM или библиотекой `openai`.
