# Модуль crawlee_python

## Обзор

Этот модуль предоставляет класс `CrawleePython` для управления веб-скрапингом с использованием Playwright.  Он позволяет задавать максимальное количество запросов, режим работы (с графическим интерфейсом или без), и тип браузера. Модуль включает функции для настройки, запуска и экспорта данных, полученных в результате скрапинга.

## Классы

### `CrawleePython`

**Описание**: Класс `CrawleePython` управляет процессом веб-скрапинга.

**Методы**:

#### `__init__`

**Описание**: Инициализирует экземпляр класса `CrawleePython` с заданными параметрами.

**Параметры**:

- `max_requests` (int): Максимальное количество запросов, которые будет выполнять скрапер. По умолчанию 5.
- `headless` (bool): Флаг, указывающий, должен ли браузер работать в фоновом режиме без графического интерфейса. По умолчанию `False`.
- `browser_type` (str): Тип браузера для использования (например, 'firefox', 'chromium'). По умолчанию 'firefox'.

#### `setup_crawler`

**Описание**: Настраивает экземпляр `PlaywrightCrawler`.

**Возвращает**:  `None`.

#### `run_crawler`

**Описание**: Запускает скрапер с заданным списком URL-адресов.

**Параметры**:

- `urls` (list[str]): Список URL-адресов для запуска скрапинга.

**Возвращает**: `None`.

#### `export_data`

**Описание**: Экспортирует собранные данные в JSON-файл.

**Параметры**:

- `file_path` (str): Путь к файлу для сохранения данных.

**Возвращает**: `None`.

#### `get_data`

**Описание**: Возвращает собранные данные.

**Возвращает**: `dict`. Возвращает словарь с результатами парсинга.

#### `run`

**Описание**: Основной метод для запуска процесса скрапинга, настроенный для выполнения инициализации, запуска скрапинга, экспорта данных и вывода полученных данных.


**Параметры**:

- `urls` (list[str]): Список URL-адресов для начала скрапинга.


**Возвращает**: `None`.


## Пример использования

```python
# ... (import statements)

async def main():
    experiment = CrawleePython(max_requests=5, headless=False, browser_type='firefox')
    await experiment.run(['https://ksp.co.il'])

asyncio.run(main())
```

Этот пример показывает, как создать экземпляр `CrawleePython`, указать начальный URL и запустить скрапинг.  Обратите внимание на использование `asyncio.run()` для запуска асинхронного кода в основной программе.
