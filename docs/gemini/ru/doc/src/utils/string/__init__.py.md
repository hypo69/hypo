# Модуль `src.utils.string`

## Обзор

Модуль `src.utils.string` предоставляет набор утилит для работы со строками, включая валидацию и нормализацию данных. Он включает в себя валидатор для полей продукта, а также функции для нормализации строк, целых чисел, чисел с плавающей точкой, булевых значений и SQL-дат.
