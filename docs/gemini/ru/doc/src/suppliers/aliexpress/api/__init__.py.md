# Модуль `aliexpress.api`

## Обзор

Модуль `aliexpress.api` представляет собой обертку для API AliExpress, предназначенную для взаимодействия с сервисами AliExpress. Модуль включает в себя классы для работы с API и модели данных.

## Переменные

### `__version__`

**Описание**: Строка, представляющая версию текущего модуля.

### `__doc__`

**Описание**: Строка, содержащая основную документацию модуля.

### `__details__`

**Описание**: Строка, содержащая дополнительную информацию о модуле.

## Импорты

### `Version`

Импортируется из `packaging.version`. Используется для работы с версиями пакетов.

### `AliexpressApi`

Импортируется из `.api`. Представляет класс для работы с API AliExpress.

### `models`

Импортируется из `.`. Содержит модели данных, используемые в модуле.
