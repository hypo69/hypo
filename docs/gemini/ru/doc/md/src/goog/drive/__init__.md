# Модуль `hypotez/src/goog/drive/__init__.py`

## Обзор

Этот модуль предоставляет инициализацию для работы с Google Drive. Он импортирует класс `GoogleDrive` из подмодуля `drive`.

## Переменные

### `MODE`

**Описание**:  Устанавливает режим работы (например, 'dev', 'prod').  В данном случае он инициализирован со значением 'dev'.

**Тип**: `str`

## Модули

### `GoogleDrive`

**Описание**:  Представляет класс для работы с Google Drive.  Подробная документация доступна в модуле `drive.py`.


## Импорты

### `from .drive import GoogleDrive`

**Описание**: Импортирует класс `GoogleDrive` из модуля `drive.py`.  Это позволяет использовать класс в текущем модуле.
