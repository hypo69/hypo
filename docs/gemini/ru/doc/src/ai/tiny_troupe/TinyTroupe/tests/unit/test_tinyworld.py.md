# Документация модуля `test_tinyworld.py`

## Обзор

Этот модуль содержит набор тестов для проверки функциональности класса `TinyWorld`, включая запуск мира, отправку широковещательных сообщений, кодирование и декодирование полного состояния мира.

## Функции

### `test_run`

**Описание**: Тестирует запуск мира `TinyWorld` с и без агентов и проверяет целостность сообщений.

**Параметры**:
- `setup`: Фикстура pytest для настройки тестовой среды.
- `focus_group_world` (TinyWorld): Мир `TinyWorld` с агентами для тестирования.

**Возвращает**:
- `None`: Функция не возвращает значений.

**Вызывает исключения**:
- `AssertionError`: Если проверка целостности сообщений не пройдена.

### `test_broadcast`

**Описание**: Тестирует широковещательную рассылку сообщений в мире `TinyWorld` и проверяет, получили ли агенты сообщение.

**Параметры**:
- `setup`: Фикстура pytest для настройки тестовой среды.
- `focus_group_world` (TinyWorld): Мир `TinyWorld` с агентами для тестирования.

**Возвращает**:
- `None`: Функция не возвращает значений.

**Вызывает исключения**:
- `AssertionError`: Если сообщение не получено агентами.

### `test_encode_complete_state`

**Описание**: Тестирует кодирование полного состояния мира `TinyWorld` в словарь.

**Параметры**:
- `setup`: Фикстура pytest для настройки тестовой среды.
- `focus_group_world` (TinyWorld): Мир `TinyWorld` с агентами для тестирования.

**Возвращает**:
- `None`: Функция не возвращает значений.

**Вызывает исключения**:
- `AssertionError`: Если закодированное состояние равно `None` или не содержит имени мира или агентов.

### `test_decode_complete_state`

**Описание**: Тестирует декодирование состояния мира `TinyWorld` из словаря и проверяет восстановление данных.

**Параметры**:
- `setup`: Фикстура pytest для настройки тестовой среды.
- `focus_group_world` (TinyWorld): Мир `TinyWorld` с агентами для тестирования.

**Возвращает**:
- `None`: Функция не возвращает значений.

**Вызывает исключения**:
- `AssertionError`: Если декодированный мир равен `None` или не соответствует исходным данным (имя и количество агентов).
