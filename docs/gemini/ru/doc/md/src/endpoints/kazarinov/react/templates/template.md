# Шаблон HTML для отображения продуктов

## Обзор

Данный HTML-шаблон (`template.html`) предназначен для отображения списка продуктов с их изображениями, названиями и описаниями.  Он использует Bootstrap для стилизации и предоставляет удобный вид для пользователей.  Также шаблон позволяет отобразить общую стоимость продуктов.


## Структура

Шаблон состоит из нескольких основных блоков:

* **Заголовок (`<head>`)**:  Устанавливает кодировку, мета-теги для масштабирования и заголовок страницы. Также подключает стили Bootstrap.
* **Тело (`<body>`)**:  Включает контейнер (`<div class="container">`) для отображения контента.  Внутри:
    * **Заголовок (`<h1>`)**:  Выводит заголовок страницы.
    * **Описание (`<p>`)**:  Выводит описание продуктов.
    * **Карточки продуктов (`<div class="product-card">`)**:  Отображает информацию о каждом продукте в карточке.
        * **Изображение (`<img>`)**:  Отображает изображение продукта.
        * **Информация о продукте (`<div class="product-info">`)**: Выводит название и описание.
    * **Общая стоимость (`<div class="footer">`)**:  Отображает общую стоимость продуктов, используя тег `<span class="price-tag">`.


## Стиль

Шаблон использует CSS для стилизации отображения:

* **`body`**: Устанавливает белый фон и черный текст.
* **`.product-card`**:  Устанавливает отступы и светлый фон.
* **`.product-card img`**:  Устанавливает размер изображения, сохраняет пропорции, добавляет белый фон под картинкой.
* **`.product-card .product-info`**:  Устанавливает отображение информации справа от картинки.
* **`.price-tag`**:  Устанавливает стили для вывода цены продукта.
* **`.footer`**: Стиль для блока общей стоимости


## Переменные

Шаблон использует следующие переменные:

* `language`: Язык страницы.
* `title`: Заголовок страницы.
* `description`: Описание продуктов.
* `products`: Список продуктов (массив объектов).  Каждый объект должен содержать `product.image_local_saved_path`, `product.product_title`, `product.product_description`.
* `price`: Общая цена продуктов.
* `currency`: Валюта.

## Пример использования

Для использования шаблона, необходимо передать значения переменных в шаблон через шаблонизатор (например, Jinja2).  Пример передачи данных:

```python
# Примерный код для передачи данных
products = [
    {'image_local_saved_path': 'path/to/image1.jpg', 'product_title': 'Название продукта 1', 'product_description': 'Описание продукта 1'},
    {'image_local_saved_path': 'path/to/image2.jpg', 'product_title': 'Название продукта 2', 'product_description': 'Описание продукта 2'}
]

# ... передача данных в шаблон ...
```
