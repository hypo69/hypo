# Задача No 59
## Расшифровка XOR

## Условия

Компьютерный файл размером 6 КБ `cipher1.txt` содержит зашифрованное сообщение, где каждый символ был зашифрован с использованием простого ключа XOR. Ключ состоит из трех строчных букв. Используя файл `cipher1.txt`, расшифруйте сообщение и найдите сумму ASCII-кодов в оригинальном тексте.

[Текст файла cipher1.txt](https://projecteuler.net/project/resources/p059_cipher1.txt)

В качестве примера, если шифртекст `65, 120, 67, 97`, и ключ - `abc`, тогда, шифрование работает следующим образом:
```
65 XOR 97 = 32
120 XOR 98 = 26
67 XOR 99 = 34
97 XOR 97 = 0
```
где 65, 120, 67 и 97 - ASCII коды символов сообщения, а 97, 98 и 99 - ASCII коды `a`, `b` и `c` соответственно. Расшифрованный текст состоит из ASCII кодов 32, 26, 34 и 0, которые соответствуют символам пробела, символу замены, символу '"' и null символу.
