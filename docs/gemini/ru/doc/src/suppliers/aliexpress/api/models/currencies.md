# Модуль hypotez/src/suppliers/aliexpress/api/models/currencies.py

## Обзор

Данный модуль содержит класс `Currency` для определения поддерживаемых валют.

## Классы

### `Currency`

**Описание**: Класс `Currency` содержит константы, представляющие различные валюты.

**Атрибуты**:

- `USD`: Символ валюты USD.
- `GBP`: Символ валюты GBP.
- `CAD`: Символ валюты CAD.
- `EUR`: Символ валюты EUR.
- `UAH`: Символ валюты UAH.
- `MXN`: Символ валюты MXN.
- `TRY`: Символ валюты TRY.
- `RUB`: Символ валюты RUB.
- `BRL`: Символ валюты BRL.
- `AUD`: Символ валюты AUD.
- `INR`: Символ валюты INR.
- `JPY`: Символ валюты JPY.
- `IDR`: Символ валюты IDR.
- `SEK`: Символ валюты SEK.
- `KRW`: Символ валюты KRW.
- `ILS`: Символ валюты ILS.
