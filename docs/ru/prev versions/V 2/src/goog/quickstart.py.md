# Модуль `quickstart`

## Обзор

Модуль `quickstart` демонстрирует базовое использование Apps Script API. Он вызывает Apps Script API для создания нового скриптового проекта, загружает файл в проект и выводит URL скрипта для пользователя.
Модуль предназначен для работы в окружении Windows и Unix.

## Импорты

Модуль импортирует следующие библиотеки:
- `pathlib.Path` для работы с путями к файлам.
- `google.auth.transport.requests.Request` для работы с запросами Google API.
- `google.oauth2.credentials.Credentials` для управления учетными данными.
- `google_auth_oauthlib.flow.InstalledAppFlow` для авторизации OAuth 2.0.
- `googleapiclient.errors` для обработки ошибок API.
- `googleapiclient.discovery.build` для построения сервиса Google API.
- `header` - собственный модуль.
- `src.gs` - собственный модуль.

## Константы

- `SCOPES`: Список областей доступа к API, необходимых для работы скрипта.
- `SAMPLE_CODE`: Пример кода Apps Script.
- `SAMPLE_MANIFEST`: Пример манифеста Apps Script.

## Функции

### `main`

**Описание**: Вызывает Apps Script API.

**Параметры**:
- Нет параметров.

**Возвращает**:
- Нет возвращаемого значения.

**Вызывает исключения**:
- `googleapiclient.errors.HttpError`: Возникает в случае ошибки при вызове API.

**Детали реализации**:

1. **Настройка учетных данных**:
   - Пытается загрузить учетные данные из файла `token.json`.
   - Если файл не существует или учетные данные недействительны, запускает процесс авторизации.
   - Сохраняет полученные учетные данные в `token.json` для последующих запусков.
2. **Создание сервиса Apps Script API**:
   - Использует `googleapiclient.discovery.build` для создания сервиса Apps Script API.
3. **Создание нового проекта**:
   - Создает новый проект Apps Script с названием "My Script" с помощью метода `service.projects().create`.
4. **Загрузка файлов в проект**:
   - Загружает два файла в проект: `hello` с кодом JavaScript и `appsscript` с манифестом в формате JSON.
5. **Вывод URL скрипта**:
   - Выводит URL скрипта для доступа к проекту.
6. **Обработка ошибок**:
   - Перехватывает исключения `googleapiclient.errors.HttpError` и выводит сообщение об ошибке.
