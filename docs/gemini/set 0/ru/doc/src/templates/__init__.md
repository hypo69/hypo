# Модуль hypotez/src/templates/__init__.py

## Обзор

Данный модуль (`hypotez/src/templates/__init__.py`) предоставляет инициализацию для пакета `templates`. Он содержит константу `MODE` и импорты, необходимые для работы других модулей в пакете.

## Константы

### `MODE`

**Описание**: Переменная, хранящая режим работы (`'dev'` в данном случае).
