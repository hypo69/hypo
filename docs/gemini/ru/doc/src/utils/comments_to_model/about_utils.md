# Модуль: hypotez/src/utils/comments_to_model/about_utils

## Обзор

Этот модуль содержит описание служебных утилит, используемых для обработки комментариев и преобразования их в модель.  Модуль предназначен для минимального использования внутренних утилит, преимущественно используя стандартную библиотеку Python.

## Функции

### `process_comments`

**Описание**: Обрабатывает список комментариев, извлекая необходимую информацию.

**Параметры**:
- `comments` (list): Список комментариев в формате словарей. Каждый словарь должен содержать необходимые поля для преобразования.

**Возвращает**:
- `dict | None`: Словарь с результатами обработки комментариев или `None`, если входные данные некорректны.

**Вызывает исключения**:
- `ValueError`: Возникает, если формат комментариев в списке не соответствует ожидаемому.
- `TypeError`: Возникает, если тип входных данных не соответствует ожидаемому.
