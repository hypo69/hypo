### Название игры: **PIZZA** (Пицца)

---

#### Описание  
**PIZZA** — это игра, в которой игроки соревнуются за то, чтобы собрать наибольшее количество кусочков пиццы. Пицца разбита на 8 кусочков, и каждый игрок по очереди выбирает один из кусочков. Если игрок выбирает кусочек, который граничит с уже выбранными кусочками, он забирает их себе. Цель игры — собрать как можно больше кусочков пиццы.

---

### Пошаговая инструкция для реализации

#### 1. **Инициализация игры**
   - Программа выводит приветственное сообщение и объясняет правила:  
     ```
     Добро пожаловать в PIZZA!  
     Ваша задача — собрать как можно больше кусочков пиццы.  
     Пицца разбита на 8 кусочков. Каждый игрок по очереди выбирает кусочек.  
     Если вы выбираете кусочек, который граничит с уже выбранными кусочками, вы забираете их себе.  
     Удачи!  
     ```

   - Программа создаёт пиццу, состоящую из 8 кусочков, и расставляет их на доске.  
   - Игроки поочерёдно делают ходы.

---

#### 2. **Основной процесс игры**

##### **2.1. Ход игрока:**
   - Игрок выбирает кусочек пиццы, указывая его номер (от 1 до 8).  
   - Программа проверяет, является ли кусочек свободным:  
     - Если кусочек свободен, он захватывается игроком.  
     - Если кусочек уже захвачен, программа сообщает об ошибке и предлагает игроку повторить ход:  
       ```
       Кусочек уже захвачен. Попробуйте снова.  
       ```

   - Если игрок выбирает кусочек, который граничит с уже выбранными кусочками, он забирает их себе.  
   - Программа обновляет состояние пиццы и отображает текущую ситуацию:  
     ```
     Текущее состояние пиццы:  
     Кусочек 1: Игрок 1  
     Кусочек 2: Игрок 2  
     Кусочек 3: Свободен  
     ...  
     ```

##### **2.2. Захват кусочков:**
   - Если игрок захватывает кусочек, программа обновляет счётчик захваченных кусочков для этого игрока.  
   - Программа сообщает, сколько кусочков захвачено каждым игроком:  
     ```
     Игрок 1: 4 кусочка  
     Игрок 2: 3 кусочка  
     ```

##### **2.3. Проверка условий завершения игры:**
   - Игра заканчивается, когда все кусочки пиццы захвачены.  
   - Программа объявляет победителя:  
     ```
     Игра окончена! Победил Игрок 1 с 5 захваченными кусочками.  
     ```

---

#### 3. **Завершение игры**
   - После завершения игры программа предлагает сыграть снова:  
     ```
     Хотите сыграть снова? (да/нет)  
     ```

   - Если игрок выбирает "да", игра начинается заново с новой расстановкой кусочков.  

---

### Пример работы программы

1. **Начало игры:**  
   ```  
   Добро пожаловать в PIZZA!  
   Пицца разбита на 8 кусочков.  
   Игрок 1, ваш ход.  
   Введите номер кусочка, который хотите захватить (от 1 до 8):  
   > 3  
   Кусочек 3 захвачен Игроком 1.  
   ```

2. **Игровой процесс:**  
   ```  
   Игрок 2, ваш ход.  
   Введите номер кусочка, который хотите захватить:  
   > 4  
   Кусочек 4 захвачен Игроком 2.  

   Игрок 1, ваш ход.  
   Введите номер кусочка, который хотите захватить:  
   > 2  
   Кусочек 2 захвачен Игроком 1.  
   ```

3. **Завершение игры:**  
   ```  
   Игра окончена! Победил Игрок 1 с 5 захваченными кусочками.  
   Хотите сыграть снова? (да/нет):  
   > нет  
   Спасибо за игру!  
   ```

---

### Возможные ограничения  
- Игрок должен вводить номер кусочка в правильном диапазоне (от 1 до 8).  
- Программа должна обрабатывать неверный ввод и предлагать повторить попытку.  
- Кусочки не могут быть захвачены повторно.

---

### Реализация  
Игра может быть реализована на Python с использованием следующих возможностей:  
- **Массивы или списки** для представления пиццы и захваченных кусочков.  
- **Циклы и условия** для проверки ввода игрока и обработки его ходов.  
- **Функции** для проверки допустимости хода и обновления состояния пиццы.
