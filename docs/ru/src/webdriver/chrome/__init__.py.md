# Модуль `src.webdriver.chrome`

## Обзор

Модуль `src.webdriver.chrome` предоставляет функциональность для управления браузером Google Chrome через WebDriver. Он включает в себя классы и функции, необходимые для автоматизации действий в Chrome.
