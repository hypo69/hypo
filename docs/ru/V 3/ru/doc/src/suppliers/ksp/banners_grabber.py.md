# Модуль `banners_grabber.py`

## Обзор

Модуль `banners_grabber.py` предназначен для сбора баннеров с сайта KSP (предположительно, интернет-магазина). В текущей версии модуля реализована только заглушка функции `get_banners`, которая всегда возвращает `True`.

## Подробней

Этот модуль, вероятно, является частью системы, отвечающей за автоматизированный сбор и обработку рекламных материалов. Он расположен в подкаталоге `src/suppliers/ksp`, что указывает на его роль как компонента, специфичного для работы с поставщиком KSP. В дальнейшем модуль может быть расширен для выполнения реального сбора баннеров, их анализа и сохранения.
