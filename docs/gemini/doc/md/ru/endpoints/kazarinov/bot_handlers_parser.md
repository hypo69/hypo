# Модуль bot_handlers_parser

## Обзор

Этот модуль содержит класс `HandlersParser`, предназначенный для обработки команд, полученных телеграм-ботом.  Класс обрабатывает URL-адреса, извлекает данные о ценах и продуктах, и взаимодействует с другими модулями для выполнения сценариев.

## Классы

### `HandlersParser`

**Описание**: Класс `HandlersParser` отвечает за обработку команд от телеграм-бота. Он инициализируется с именем вебдрайвера.

**Атрибуты**:
- `mexiron` (`Mexiron`): Экземпляр класса `Mexiron`, используемый для выполнения сценариев.

**Методы**:

#### `__init__`

**Описание**: Конструктор класса.

**Параметры**:
- `webdriver_name` (Optional[str], optional): Название вебдрайвера ('firefox', 'chrome', 'edge'). По умолчанию 'firefox'.

#### `handle_url`

**Описание**: Обрабатывает URL, полученный от пользователя.

**Параметры**:
- `update` (`Update`): Объект `Update` содержащий информацию о сообщении пользователя.
- `context` (`CallbackContext`): Контекст вызова.

**Возвращает**:
- `Any`:  Возвращает значение, указывающее на успех или неудачу выполнения сценария.

#### `get_data_from_onetab`

**Описание**: Извлекает данные (цену, имя, список ссылок) из URL-адреса one-tab.

**Параметры**:
- `response` (str): URL-адрес страницы one-tab.

**Возвращает**:
- `list[int | float, str, list] | bool`: Возвращает кортеж из цены, имени и списка ссылок, или `False`, если произошла ошибка.

#### `handle_next_command`

**Описание**: Обрабатывает команду '--next' и связанные команды.

**Параметры**:
- `update` (`Update`): Объект `Update` содержащий информацию о сообщении пользователя.

**Возвращает**:
- `None`


#### `fetch_target_urls_onetab`

**Описание**: Извлекает целевые URL с указанного URL OneTab.

**Параметры**:
- `one_tab_url` (str): URL страницы OneTab для извлечения целевых URL.

**Возвращает**:
- `Tuple[int, str, List[str]] | bool`: Кортеж из цены, имени и списка URL или `False`, если произошла ошибка.


**Вызывает исключения**:
- `requests.exceptions.RequestException`: При ошибке запроса.
- `ValueError`: При ошибке преобразования цены в число.
