# Модуль onela_bot

## Обзор

Модуль `onela_bot` предназначен для взаимодействия с моделью ассистента программиста через чат Telegram. Он обрабатывает текстовые сообщения и загруженные документы, отправляя их на обработку модели и отправляя полученные ответы обратно пользователю.

## Классы

### `OnelaBot`

**Описание**: Класс `OnelaBot` наследуется от `TelegramBot` и реализует функциональность взаимодействия с моделью ассистента программиста.

**Методы**:

- `__init__`: Инициализация объекта `OnelaBot`.
- `handle_message`: Обработка текстовых сообщений.
- `handle_document`: Обработка загруженных документов.

**Параметры**:

- Не применимо.

**Возвращает**:

- Не применимо.

**Вызывает исключения**:

- Не применимо.

### `GoogleGenerativeAI`

**Описание**: Класс `GoogleGenerativeAI` не описан в данном фрагменте кода.


## Функции

### `handle_message`

**Описание**: Обрабатывает текстовые сообщения, полученные от пользователя через Telegram.

**Параметры**:
- `update` (Update): Объект, содержащий данные обновления Telegram.
- `context` (CallbackContext): Контекст выполнения.

**Возвращает**:
- `None`: Метод не возвращает значение.

**Вызывает исключения**:
- `Exception`: Обрабатывает общие ошибки при взаимодействии с моделью.  Ошибка выводится в лог.


### `handle_document`

**Описание**: Обрабатывает загруженные документы, полученные от пользователя через Telegram.

**Параметры**:
- `update` (Update): Объект, содержащий данные обновления Telegram.
- `context` (CallbackContext): Контекст выполнения.

**Возвращает**:
- `None`: Метод не возвращает значение.

**Вызывает исключения**:
- `Exception`: Обрабатывает общие ошибки при взаимодействии с моделью. Ошибка выводится в лог.
