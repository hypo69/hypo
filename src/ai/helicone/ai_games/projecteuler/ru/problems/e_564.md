# Задача No 564
## Ленивые коровы

## Условия

На бесконечной прямой расположено N пастбищ, каждое из которых представлено целым числом xᵢ. На этих пастбищах находятся M коров. Нас интересует, как далеко должна идти самая "ленивая" корова.

Формально, необходимо найти минимальное значение D такое, что все коровы могут быть распределены по пастбищам так, чтобы максимальное расстояние между любой коровой и назначенным ей пастбищем не превышало D.

Вы можете предположить, что все пастбища расположены в разных точках.

Рассмотрим два пастбища xᵢ и xⱼ и k коров. Найти минимальное D такое, что k коров могут быть распределены между xᵢ и xⱼ так, что каждая корова находится на расстоянии не более D от одного из двух пастбищ.

Например, если два пастбища находятся в точках 0 и 10, а количество коров равно 3, то минимальное значение D равно 2,5. Все три коровы должны быть в пределах 2,5 единиц от 0 или 10. Одно из возможных решений - 2 коровы на расстоянии 2,5 от 0, 1 корова на расстоянии 2,5 от 10.

Даны два массива X и C, где X содержит позиции пастбищ, а C - количество коров на каждом пастбище. Необходимо найти минимальное D такое, что все коровы могут быть распределены между всеми пастбищами так, чтобы каждая корова находилась на расстоянии не более D от какого-либо пастбища.

Например:

X = [0, 10]
C = [3]

минимальное значение D = 2.5

X = [0, 10]
C = [1, 2]

минимальное значение D = 2.5

X = [0, 10, 20]
C = [3, 4]

минимальное значение D = 2.5

X = [0, 10, 20]
C = [1, 1, 1]

минимальное значение D = 5

X = [0, 4, 10]
C = [2, 1, 1]

минимальное значение D = 2

X = [0, 4, 9, 11]
C = [3, 1, 2, 2]

минимальное значение D = 1

Найдите минимальное значение D для следующего набора пастбищ и коров:
X = [2, 4, 7, 10, 14, 16, 20, 22, 25, 27, 28]
C = [10, 2, 6, 9, 1, 3, 7, 5, 2, 8, 1]
