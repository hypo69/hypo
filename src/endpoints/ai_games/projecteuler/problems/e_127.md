# Задача No 127
## ABC-хиты
## Условия
Тройка положительных целых чисел (a, b, c) называется abc-хитом, если выполняются следующие условия:

1.  НОД(a, b) = НОД(a, c) = НОД(b, c) = 1
2.  a < b
3.  a + b = c
4.  rad(abc) < c

Здесь rad(n) это радикал n, определенный как произведение простых делителей n. Например, rad(12) = 2 * 3 = 6.

Рассмотрим abc-хиты для c ≤ 1000: 50 abc-хитов. Их сумма равна 12523.

Найдите сумму abc-хитов для c ≤ 120000.
