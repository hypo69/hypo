# Модуль для парсинга продуктов AliExpress

## Обзор

Модуль `products.py` содержит функции для обработки и преобразования данных о продуктах, полученных из API AliExpress. Он включает в себя функции для парсинга как отдельного продукта, так и списка продуктов, приводя их к нужному формату.

## Подробней

Этот модуль используется для предобработки данных о товарах, извлеченных из API AliExpress, перед их дальнейшей обработкой или сохранением в базе данных. Он обеспечивает единообразное представление данных, упрощая работу с ними на следующих этапах.

### `parse_products`
