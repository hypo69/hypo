# Модуль `hypotez/src/suppliers/aliexpress/header.py`

## Обзор

Данный модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`.  Он также обрабатывает случай отсутствия файла настроек, возвращая значения по умолчанию.  Для корректной работы требуется модуль `gs`.


## Функции

### `set_project_root`

**Описание**:  Определяет корневую директорию проекта, начиная с текущей директории файла и идя вверх по дереву директорий, пока не найдет директорию, содержащую один из указанных маркеров (файлы или директории).

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или директорий, по которым будет определяться корневая директория проекта. По умолчанию используются `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории проекта, если найдена, в противном случае - путь к текущей директории.

## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root()`.

**Тип**: `Path`

### `__project_name__`

**Описание**: Имя проекта, полученное из настроек или по умолчанию `hypotez`.

**Тип**: `str`

### `__version__`

**Описание**: Версия проекта, полученная из настроек или по умолчанию пустая строка.

**Тип**: `str`

### `__doc__`

**Описание**: Содержимое файла README.MD или пустая строка, если файл не найден или содержит некорректный формат.

**Тип**: `str`

### `__details__`

**Описание**: Дополнительные детали проекта (пустая строка по умолчанию).

**Тип**: `str`

### `__author__`

**Описание**: Автор проекта, полученный из настроек или по умолчанию пустая строка.

**Тип**: `str`

### `__copyright__`

**Описание**: Авторские права проекта, полученные из настроек или по умолчанию пустая строка.

**Тип**: `str`

### `__cofee__`

**Описание**: Ссылка на поддержку разработчика через платформу по сбору средств.

**Тип**: `str`
