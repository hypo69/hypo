# Модуль hypotez/src/ai/openai

## Обзор

Данный модуль предоставляет инструменты для работы с моделью OpenAI.  Он содержит функции для перевода текста и класс для работы с моделью OpenAI.

## Переменные

### `MODE`

**Описание**:  Переменная, определяющая режим работы (`dev`).

**Тип**: str

## Функции

### `translate`

**Описание**: Функция для перевода текста.

**Параметры**:
- `text` (str): Текст для перевода.
- `source_lang` (str): Язык источника.
- `target_lang` (str): Целевой язык.


**Возвращает**:
- str: Переведенный текст или сообщение об ошибке.

**Вызывает исключения**:
- `ValueError`: Если не указаны `source_lang` и `target_lang` или они не поддерживаются.
- `Exception`: Если произошла ошибка при взаимодействии с моделью.


## Классы

### `OpenAIModel`

**Описание**: Класс для работы с моделью OpenAI.

**Методы**:

- `__init__`: Конструктор класса.


**Параметры**:

- `api_key` (str):  API ключ для доступа к модели OpenAI.


**Возвращает**:
- None


**Вызывает исключения**:
- `ValueError`: Если API ключ не предоставлен или имеет неверный формат.
- `Exception`: Если произошла ошибка при инициализации.
