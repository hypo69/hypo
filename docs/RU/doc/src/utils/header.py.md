# Модуль `header`

## Обзор

Модуль `header` предназначен для определения корневого пути проекта и загрузки основных параметров проекта из файла `settings.json`, а также из README.MD. Все импорты в проекте строятся относительно этого корневого пути.

## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная с каталога текущего файла, ища вверх по дереву каталогов до первого каталога, содержащего любой из маркеров.

**Параметры**:
- `marker_files` (tuple): Кортеж с именами файлов или каталогов, которые идентифицируют корень проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневому каталогу, если он найден, иначе путь к каталогу, где расположен скрипт.

## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта. Устанавливается функцией `set_project_root`.

### `settings`

**Описание**: Словарь, загруженный из `settings.json`. Содержит настройки проекта, такие как имя проекта, версия и т.д.

### `doc_str`

**Описание**: Строка, содержащая текст из файла `README.MD`.

### `__project_name__`

**Описание**: Название проекта. Загружается из `settings.json` или по умолчанию `hypotez`.

### `__version__`

**Описание**: Версия проекта. Загружается из `settings.json` или по умолчанию пустая строка.

### `__doc__`

**Описание**: Документация проекта, загружается из файла `README.MD`, если он есть.

### `__details__`

**Описание**: Дополнительная информация о проекте. В данный момент пустая строка.

### `__author__`

**Описание**: Автор проекта. Загружается из `settings.json` или по умолчанию пустая строка.

### `__copyright__`

**Описание**: Информация об авторских правах. Загружается из `settings.json` или по умолчанию пустая строка.

### `__cofee__`

**Описание**: Строка с информацией о возможности поддержать автора. Загружается из `settings.json`, если есть, или имеет значение по умолчанию.
