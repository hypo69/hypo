# Модуль dict_scenarios

## Обзор

Модуль `dict_scenarios.py` содержит словарь `scenario`, который определяет параметры для сбора данных с сайта Amazon. Эти параметры включают URL, условие (новое), категории PrestaShop и правило цены.  Этот модуль используется для настройки сценариев сбора данных для товаров, таких как "Murano Glass".

## Подробней

Этот модуль предоставляет структуру данных, определяющую, как собирать информацию с Amazon и как связывать её с категориями в PrestaShop.  Он является частью экспериментов по автоматизации сбора данных и категоризации товаров для последующей обработки и загрузки в систему PrestaShop.

## Переменные

### `scenario`

```python
scenario: dict = {
    "Murano Glass": {
        "url": "https://www.amazon.com/s?k=Art+Deco+murano+glass&crid=24Q0ZZYVNOQMP&sprefix=art+deco+murano+glass%2Caps%2C230&ref=nb_sb_noss",
        "condition": "new",
        "presta_categories": {
            "default_category": {"11209": "MURANO GLASS"}
        },
        "price_rule": 1
    }
}
```

**Описание**: Словарь, содержащий настройки сценария для сбора данных о товарах.

**Принцип работы**:
Словарь `scenario` содержит конфигурацию для сбора информации о товарах с Amazon и их последующей категоризации в PrestaShop. Каждый ключ верхнего уровня (например, "Murano Glass") представляет собой название товара или категории, для которой определены настройки. Значение каждого ключа - это словарь с параметрами сбора данных:
- `"url"`: URL страницы Amazon, с которой собираются данные о товаре.
- `"condition"`: Условие товара (например, "new").
- `"presta_categories"`:  Словарь, определяющий соответствие между категориями Amazon и категориями в PrestaShop. В данном случае, категория "MURANO GLASS" в PrestaShop имеет ID "11209".
- `"price_rule"`: Правило для определения цены товара.

**Как работает переменная**:

1.  **Определение товара**: Определяется товар или категория товара, для которого будет настроен сценарий сбора данных (например, `"Murano Glass"`).
2.  **URL**: Указывается URL страницы Amazon, содержащей список товаров для данной категории.
3.  **Условие товара**: Определяется состояние товара, например, новый (`"new"`).
4.  **Соответствие категорий PrestaShop**: Настраивается соответствие между категориями Amazon и категориями PrestaShop для автоматической категоризации товара в PrestaShop.
5.  **Правило цены**: Указывается правило, по которому будет определяться цена товара.

**Примеры**:

```python
scenario = {
    "Murano Glass": {
        "url": "https://www.amazon.com/s?k=Art+Deco+murano+glass&crid=24Q0ZZYVNOQMP&sprefix=art+deco+murano+glass%2Caps%2C230&ref=nb_sb_noss",
        "condition": "new",
        "presta_categories": {
            "default_category": {"11209": "MURANO GLASS"}
        },
        "price_rule": 1
    }
}
```

В этом примере, для товара "Murano Glass" определены URL страницы Amazon, условие "new", соответствие категории PrestaShop с ID "11209" и правило цены 1.
