# Модуль для провоцирования грубости у AI моделей
## Обзор

Модуль предназначен для демонстрации того, как можно спровоцировать AI-модели, такие как GPT-3.5-turbo и GPT-4, на генерацию грубых высказываний. В основном используется для оценки устойчивости AI к нежелательному контенту.

## Подробней

Модуль содержит функцию `bully`, которая отправляет специально разработанные запросы в OpenAI API, чтобы заставить модель отвечать как хулиган. Это позволяет исследовать границы этичного поведения AI и выявлять возможные уязвимости.

## Функции

### `bully`

```python
def bully(user_message="Hello!", messages=[{"system": "user", "content": system_prompt}]) -> list:
    """
    Функция отправляет запрос в OpenAI API с целью получить ответ в стиле хулигана.

    Args:
        user_message (str, optional): Сообщение пользователя, которое будет отправлено в качестве запроса. По умолчанию "Hello!".
        messages (list, optional): Список сообщений, представляющих контекст диалога. По умолчанию список с системным промптом.

    Returns:
        list: Обновленный список сообщений, включающий ответ от AI модели.

    Raises:
        openai.error.OpenAIError: В случае ошибки при взаимодействии с OpenAI API.

    Example:
        >>> messages = bully(user_message="Ты мне не нравишься!")
        >>> print(messages[-1])  # Вывод последнего сообщения, содержащего ответ модели
        {'role': 'user', 'content': '{\n  "bully_response": "Да ты вообще никто, и звать тебя никак!"\n}'}
    """
```

**Как работает функция**:

1.  **Добавление сообщения пользователя**: Функция принимает сообщение пользователя (`user_message`) и добавляет его в список сообщений (`messages`).
2.  **Запрос к OpenAI API**: Функция отправляет список сообщений в OpenAI API, используя модель `gpt-3.5-turbo`.
3.  **Получение ответа**: Функция получает ответ от AI модели и добавляет его в список сообщений.
4.  **Возврат списка сообщений**: Функция возвращает обновленный список сообщений, включающий ответ от AI модели.

```text
Начало --> Добавление сообщения пользователя
         |
         v
         Отправка запроса в OpenAI API
         |
         v
         Получение ответа от AI модели
         |
         v
         Возврат списка сообщений --> Конец
```

**Примеры**:

```python
# Пример 1: Отправка простого сообщения
messages = bully(user_message="Ты глупый!")
print(messages[-1])

# Пример 2: Использование контекста диалога
messages = [{"role": "system", "content": system_prompt}]
messages = bully(user_message="Ты мне не нравишься!", messages=messages)
print(messages[-1])

# Пример 3: Получение ответа в стиле хулигана
messages = bully(user_message="Я лучше тебя!", messages=[{"system": "user", "content": system_prompt}])
print(messages[-1])
