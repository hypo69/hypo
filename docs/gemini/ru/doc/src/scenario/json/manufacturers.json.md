# manufacturers.json

## Обзор

Файл `manufacturers.json` содержит информацию о производителях, используемых в системе. В настоящее время файл содержит данные о производителе `pemco-hardware`.

## Содержание

- [Описание производителей](#описание-производителей)
  - [pemco-hardware](#pemco-hardware)

## Описание производителей

### `pemco-hardware`

**Описание**: Информация о производителе `pemco-hardware`.

**Поля**:
- `descripiton` (str): Описание сферы деятельности производителя. В данном случае - "heavy industry".
- `url` (str): URL-адрес веб-сайта производителя. В данном случае - "https://www.pemco-hardware.com".
