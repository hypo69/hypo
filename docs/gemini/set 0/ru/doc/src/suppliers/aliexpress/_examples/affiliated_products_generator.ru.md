# Модуль `affiliated_products_generator.py`

## Обзор

Этот модуль предоставляет инструменты для генерации аффилированных ссылок для продуктов с AliExpress. Он содержит класс `AliAffiliatedProducts`, который отвечает за загрузку данных о продуктах, формирование аффилированных ссылок и сохранение связанных медиа-файлов (изображений и видео).

## Класс `AliAffiliatedProducts`

**Описание**: Класс для работы с аффилированными продуктами с AliExpress.

**Атрибуты**:

- `campaign_name`: Название рекламной кампании (str).
- `campaign_category`: Категория рекламной кампании (str, optional).
- `language`: Язык для генерации ссылок (str).
- `currency`: Валюта для генерации ссылок (str).


**Методы**:

### `process_affiliate_products`

**Описание**: Обрабатывает список URL или ID продуктов, генерирует аффилированные ссылки, сохраняет изображения и видео.

**Параметры**:

- `prod_urls` (list): Список URL или ID продуктов (str).


**Возвращает**:

- `list[Product]`: Список объектов `Product`, содержащих аффилированные ссылки, ID продукта и пути к сохранённым медиа-файлам, или `None` при ошибке.


**Вызывает исключения**:

- `ConnectionError`: Возникает при проблемах с подключением к AliExpress.
- `ValueError`: Возникает при неверных входных данных.
- `HTTPError`: Возникает при ошибках HTTP-запроса.
- `IOError`: Возникает при проблемах с файловыми операциями (чтение/запись).
- `Exception`: Любые другие неожиданные ошибки.
