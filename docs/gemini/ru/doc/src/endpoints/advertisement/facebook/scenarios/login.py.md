# Модуль `src.endpoints.advertisement.facebook.scenarios.login`

## Обзор

Модуль `login.py` предназначен для выполнения сценария авторизации на Facebook. Он загружает локаторы элементов страницы из JSON-файла, использует драйвер браузера для взаимодействия с элементами и осуществляет вход, используя предоставленные учетные данные.

## Импорты

В данном модуле используются следующие импорты:

-   `pathlib.Path`: Для работы с путями файлов.
-   `typing.Dict`: Для аннотации типов.
-   `src.gs`: Глобальные настройки приложения.
-   `src.webdriver.driver.Driver`: Класс драйвера для управления веб-браузером.
-   `src.utils.jjson`: Модуль для работы с JSON.
-    `src.logger.logger`: Модуль для логирования.

## Переменные

### `locators`

-   **Описание**: Словарь локаторов веб-элементов для авторизации Facebook. Загружается из JSON-файла.
-   **Тип**: `dict`
-   **Значение**: Результат загрузки `login.json` из пути `src/endpoints/advertisement/facebook/locators/login.json`.
