# Модуль hypotez/src/suppliers/ksp/header.py

## Обзор

Данный модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`. Также он загружает документацию из файла `README.MD`.  Модуль инициализирует переменные, представляющие имя проекта, версию, описание, автора, копирайт и ссылку для пожертвований.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла и идя вверх по дереву директорий. Остановка происходит при обнаружении файлов `pyproject.toml`, `requirements.txt` или `.git`.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для определения корневой директории проекта. По умолчанию содержит `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращает директорию, в которой расположен скрипт.

**Вызывает исключения**:

- Нет.


## Переменные

### `__root__`

**Описание**: Переменная, хранящая путь к корневой директории проекта.  Инициализируется функцией `set_project_root()`.

**Тип**: `Path`

**Описание**: Путь к корневой директории проекта.

### `settings`

**Описание**: Словарь настроек, загруженный из файла `settings.json`.

**Тип**: `dict`

**Описание**: Словарь настроек проекта.

**Обработка ошибок**:
Файл `settings.json` может быть не найден или содержать некорректный JSON. В этом случае переменная `settings` будет установлена в `None`.

### `doc_str`

**Описание**: Строка документации, загруженная из файла `README.MD`.

**Тип**: `str`

**Описание**: Строка документации из файла `README.MD`


**Обработка ошибок**:
Файл `README.MD` может быть не найден или содержать ошибки. В этом случае переменная `doc_str` будет установлена в `None`.

### `__project_name__`

**Описание**: Имя проекта.

**Тип**: `str`

**Описание**: Имя проекта, полученное из настроек (`settings`). По умолчанию `hypotez`.

### `__version__`

**Описание**: Версия проекта.

**Тип**: `str`

**Описание**: Версия проекта, полученная из настроек (`settings`). По умолчанию пустая строка.


### `__doc__`

**Описание**: Документация проекта.

**Тип**: `str`

**Описание**: Документация проекта, загруженная из файла `README.MD`. По умолчанию пустая строка.



### `__details__`

**Описание**: Дополнительные сведения о проекте.

**Тип**: `str`

**Описание**: Дополнительные сведения о проекте, по умолчанию пустая строка.


### `__author__`

**Описание**: Автор проекта.

**Тип**: `str`

**Описание**: Автор проекта, полученный из настроек (`settings`). По умолчанию пустая строка.


### `__copyright__`

**Описание**: Авторские права.

**Тип**: `str`

**Описание**: Авторские права, полученные из настроек (`settings`). По умолчанию пустая строка.


### `__cofee__`

**Описание**: Ссылка для пожертвований.

**Тип**: `str`

**Описание**: Ссылка для пожертвований. По умолчанию ссылка на Boosty.
