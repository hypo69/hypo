## Функции

### Основная логика игры

**Описание**: 
Основная логика игры "Кролик". Инициализирует положение кролика, выполняет серию случайных прыжков, запрашивает у пользователя предположение о конечном положении, выводит фактическое положение кролика и определяет, угадал ли пользователь.

**Параметры**:
- Нет параметров

**Возвращает**:
- Нет возвращаемого значения.

**Вызывает исключения**:
- `ValueError`: Возникает, если пользователь вводит не целое число.

```python
import random

# אתחול מיקום הארנב
bunnyPosition = 0

# לולאה המבצעת 10 קפיצות
for _ in range(10):
    # יצירת מספר אקראי 1 או 2
    randomJump = random.randint(1, 2)
    # אם המספר האקראי הוא 1 - הארנב קופץ קדימה
    if randomJump == 1:
        bunnyPosition += 1
    # אם המספר האקראי הוא 2 - הארנב קופץ אחורה
    else:
        bunnyPosition -= 1

# קבלת ניחוש מהמשתמש לגבי מיקום הארנב
try:
    userGuess = int(input("נחש את מיקום הארנב: "))
except ValueError as ex:
    print("אנא הזן מספר שלם.")
    exit()

# הצגת המיקום האמיתי של הארנב
print(f"מיקום הארנב האמיתי: {bunnyPosition}")

# בדיקה האם הניחוש נכון
if userGuess == bunnyPosition:
    print("YOU GOT IT!")
# אם הניחוש לא נכון
else:
    print("TOO BAD!")
```

**Описание кода:**

1.  **Импорт модуля `random`**:
    *   Импортируется модуль `random`, который используется для генерации случайных чисел.
2.  **Инициализация положения кролика**:
    *   `bunnyPosition = 0`: Определяется переменная `bunnyPosition`, которая инициализируется значением 0. Эта переменная используется для хранения положения кролика на доске.
3.  **Цикл прыжков**:
    *   `for _ in range(10):`: Цикл `for`, который повторяется 10 раз, где каждая итерация представляет прыжок кролика. Переменная `_` используется как переменная цикла, которая не используется.
    *   `randomJump = random.randint(1, 2)`: Генерация случайного целого числа между 1 и 2 с помощью функции `randint` из модуля `random`. Число 1 представляет прыжок вперед, а число 2 представляет прыжок назад.
    *   `if randomJump == 1:`: Условный оператор `if`, проверяющий, равно ли случайное число 1. Если да, выполняется прыжок вперед.
        *   `bunnyPosition += 1`: Значение `bunnyPosition` увеличивается на 1, то есть кролик прыгает на один шаг вперед.
    *   `else:`: Если случайное число не равно 1, то оно равно 2 (поскольку диапазон только 1 или 2).
        *   `bunnyPosition -= 1`: Значение `bunnyPosition` уменьшается на 1, то есть кролик прыгает на один шаг назад.
4.  **Получение ввода от пользователя**:
    *   `try...except ValueError`: Блок `try-except` используется для обработки возможных ошибок во время получения ввода от пользователя. Он пытается преобразовать ввод в целое число, и если преобразование не удается, он выводит сообщение и завершает программу.
    *   `userGuess = int(input("נחש את מיקום הארנב: "))`: Запрос у пользователя ввести предположение о положении кролика, преобразование ввода в целое число и сохранение его в переменную `userGuess`.
5.  **Вывод положения кролика**:
    *   `print(f"מיקום הארנב האמיתי: {bunnyPosition}")`: Вывод конечного положения кролика после 10 прыжков.
6.  **Проверка предположения**:
    *   `if userGuess == bunnyPosition:`: Условный оператор `if`, проверяющий, равно ли предположение пользователя фактическому положению кролика.
        *   `print("YOU GOT IT!")`: Вывод сообщения пользователю, что он правильно угадал.
    *   `else:`: Если предположение пользователя отличается от положения кролика.
        *   `print("TOO BAD!")`: Вывод сообщения пользователю, что он не угадал.
