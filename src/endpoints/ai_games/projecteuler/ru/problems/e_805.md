# Задача No 805
## Сокрытие последовательностей

## Условия

Рассмотрим бесконечную последовательность целых чисел, `a(n)`, где `a(1) = 1`, и которая удовлетворяет рекуррентному соотношению:
`a(n) = a(n - a(n - 1)) + a(n - 1 - a(n - 2))` при `n > 2`,
где `a(0)` считается равным `0`.

Первые несколько членов этой последовательности:
`1, 1, 2, 3, 4, 4, 5, 6, 6, 8, 8, 8, 9, 10, 10, 11, 12, ...`

Также рассмотрим последовательность `b(n)`, которая является упорядоченным списком значений `a(n)`, встречающихся в первый раз.

Первые несколько членов этой последовательности:
`1, 2, 3, 4, 5, 6, 8, 9, 10, 11, 12, ...`

Пусть `S(n)` - сумма первых `n` членов последовательности `b(n)`.
Известно, что `S(10) = 69` и `S(1000) = 741500`.

Найдите `S(10^8)`.
