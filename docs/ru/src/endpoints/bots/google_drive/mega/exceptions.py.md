# Модуль исключений для Mega

## Обзор

Модуль содержит определения пользовательских исключений, специфичных для взаимодействия с Mega API. Эти исключения позволяют более точно обрабатывать ошибки, возникающие при работе с Mega, такие как неверные учетные данные или ошибки при выполнении запросов.

## Подробней

Этот модуль предоставляет иерархию исключений, начинающуюся с базового класса `MegaException`.  Производные классы, такие как `MegaIncorrectPasswordExcetion` и `MegaRequestException`, используются для обозначения конкретных типов ошибок, которые могут возникнуть при взаимодействии с сервисом Mega. Использование этих исключений позволяет улучшить обработку ошибок и отладку в модулях, взаимодействующих с Mega API.

## Классы

### `MegaException`

**Описание**: Базовый класс для всех исключений, связанных с Mega.

**Как работает класс**:
Этот класс служит родительским для всех пользовательских исключений, определенных для работы с Mega API. Он наследуется от стандартного класса `Exception` в Python и не добавляет никакой дополнительной функциональности. Его основная цель - обеспечить общую базу для всех исключений, связанных с Mega, что позволяет удобно перехватывать и обрабатывать их.

### `MegaIncorrectPasswordExcetion`

**Описание**: Исключение, возникающее при предоставлении неверного пароля или email.

**Как работает класс**:
Этот класс исключения используется для обозначения ситуации, когда пользователь ввел неверный пароль или адрес электронной почты при попытке аутентификации в Mega. Наследуется от `MegaException`.

### `MegaRequestException`

**Описание**: Исключение, возникающее при ошибке в запросе к Mega API.

**Как работает класс**:
Этот класс исключения используется для обозначения ошибок, возникающих при выполнении запросов к Mega API. Это может быть связано с проблемами сетевого соединения, неверным форматом запроса или другими проблемами на стороне сервера Mega. Наследуется от `MegaException`.
