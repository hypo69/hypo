# Задача No 896
## Вычисление максимального пути в графе
## Условия

Рассмотрим ориентированный граф, в котором каждая вершина имеет целочисленную метку, а каждое ребро имеет неотрицательную целочисленную метку.

Путь в графе — это последовательность вершин, соединенных ребрами, которые можно проходить в направлении ребер. Общая длина пути равна сумме меток ребер в пути.
Назовем значение пути максимальной меткой вершины на этом пути.

Например, рассмотрим граф со следующими вершинами и ребрами:

* Вершины: (1, 2, 3, 4, 5)
* Ребра: (1 -> 2: 7, 1 -> 3: 2, 2 -> 4: 3, 3 -> 4: 2, 3 -> 5: 8)

Здесь вершины имеют метки (1, 2, 3, 4, 5) соответственно, а ребра имеют указанные выше метки.

Путь 1 -> 2 -> 4 имеет длину 7 + 3 = 10 и значение max(1, 2, 4) = 4.

Путь 1 -> 3 -> 5 имеет длину 2 + 8 = 10 и значение max(1, 3, 5) = 5.

Путь 1 -> 3 -> 4 имеет длину 2 + 2 = 4 и значение max(1, 3, 4) = 4.

Путь 1 -> 2 -> 4 имеет общую длину 10, но его значение всего 4.

Для заданного графа найдите путь с максимальным значением. Если есть несколько путей с одинаковым максимальным значением, выберите путь с максимальной длиной.

Для заданного графа с 20000 вершинами, пронумерованными от 1 до 20000, где вершина *i* имеет метку *i*, ребра графа определяются следующим правилом:

Из каждой вершины *i* есть ребро в вершину *j*, если *j > i* и *j* является наименьшим целым числом таким, что 
( *i* + *j* ) является делителем *i* * *j*.

Длина ребра от *i* до *j* определяется как *i* + *j*.
 
Найдите значение и длину пути, удовлетворяющего вышеуказанным условиям.
Запишите значение и длину в виде «значение,длина».
