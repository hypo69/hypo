# Модуль `crawlee_python`

## Обзор

Модуль `crawlee_python` предоставляет функциональность для взаимодействия с Crawlee на Python. Он включает в себя класс `CrawleePython`, который позволяет запускать и управлять Crawlee-скриптами.
