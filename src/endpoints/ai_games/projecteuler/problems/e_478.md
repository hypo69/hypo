# Задача No 478
## Заполненные квадраты

## Условия

Пусть $M(n, k)$ - количество способов, которыми можно заполнить квадратную сетку $n \times n$ целыми числами от 1 до $k$, так, чтобы все числа в каждой строке и каждом столбце были строго упорядочены по возрастанию.

Например, $M(3, 5) = 70$ и $M(4, 8) = 15400$.

Пусть $S(n)$ будет суммой $M(n, k)$ для $1 \le k \le 2n$.

Например, $S(3) = M(3,1) + M(3,2) + M(3,3) + M(3,4) + M(3,5) + M(3,6) = 1 + 5 + 15 + 35 + 70 + 126 = 252$.

Известно, что $S(10) \text{ mod } 10^9 = 137226872$.

Найдите $S(10^6) \text{ mod } 10^9$.
