# Документация для `category.json`

## Обзор

Файл `category.json` содержит JSON-структуру, определяющую локаторы для элементов на странице категории Amazon. Эти локаторы используются для навигации и извлечения информации о продуктах.

## Описание структуры

Файл `category.json` представляет собой JSON-объект с двумя основными разделами: `pager` и `product_links`.

### Раздел `pager`

Этот раздел определяет локаторы и параметры для управления пагинацией на странице категории.

**Описание**:

  Параметры для прокрутки страницы

**Параметры**:

- `attribute` (null): Атрибут, используемый для поиска элемента. В данном случае `null`.
- `by` (str): Метод поиска элемента. В данном случае `"event"`.
- `selector` (null): Селектор для поиска элемента. В данном случае `null`.
- `if_list` (str): Условие для работы со списком. В данном случае `"first"`.
- `use_mouse` (bool): Использовать ли мышь для взаимодействия. В данном случае `false`.
- `mandatory` (bool): Является ли элемент обязательным. В данном случае `true`.
- `timeout` (int): Время ожидания элемента в секундах. В данном случае `0`.
- `timeout_for_event` (str): Событие, ожидаемое до истечения таймаута. В данном случае `"presence_of_element_located"`.
- `event` (str): Событие для взаимодействия. В данном случае `"scroll(5,'both')"`.

### Раздел `product_links`

Этот раздел определяет локаторы и параметры для извлечения ссылок на продукты на странице категории.

**Описание**:

Параметры для извлечения ссылок на продукты.

**Параметры**:

- `attribute` (str): Атрибут, содержащий ссылку на продукт. В данном случае `"href"`.
- `by` (str): Метод поиска элемента. В данном случае `"XPATH"`.
- `selector` (str): XPATH-селектор для поиска элемента. В данном случае `//span[@data-component-type ='s-product-image']//a`.
- `if_list` (str): Условие для работы со списком. В данном случае `"first"`.
- `use_mouse` (bool): Использовать ли мышь для взаимодействия. В данном случае `false`.
- `mandatory` (bool): Является ли элемент обязательным. В данном случае `true`.
- `timeout` (int): Время ожидания элемента в секундах. В данном случае `0`.
- `timeout_for_event` (str): Событие, ожидаемое до истечения таймаута. В данном случае `"presence_of_element_located"`.
- `event` (null): Событие для взаимодействия. В данном случае `null`.
