Module Description
------------------

This module contains functions for posting messages on Facebook, including sending titles and descriptions, uploading media, and publishing the post.  It leverages the `Driver` class and interacts with the Facebook web application through locators defined in a JSON file.  Locators are automatically loaded from a JSON file.
