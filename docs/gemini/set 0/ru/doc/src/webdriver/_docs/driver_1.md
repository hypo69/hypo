# Модуль `driver_1`

## Обзор

Данный модуль предоставляет метакласс `DriverMeta`, который динамически создаёт класс `Driver`, наследующий от базового класса `Driver` и заданного класса Selenium WebDriver (`Chrome`, `Firefox` или `Edge`). Метакласс отвечает за инстанцирование корректной комбинации этих классов.

## Метаклассы

### `DriverMeta`

**Описание**: Метакласс `DriverMeta` используется для управления созданием нового класса `Driver`.

#### `__call__`

**Описание**: Метод `__call__` вызывается при инстанцировании класса. Он создаёт новый класс `Driver`, наследующий как от базового класса `Driver`, так и от указанного класса Selenium WebDriver (`Chrome`, `Firefox`, или `Edge`).

**Параметры**:

- `cls` (type): Класс, который инстанцируется, в данном случае `Driver`.
- `webdriver_cls` (type): Класс Selenium WebDriver, от которого наследуется (`Chrome`, `Firefox`, или `Edge`).
- `*args`: Аргументы, передаваемые в конструктор класса `Driver`.
- `**kwargs`: Ключевые аргументы, передаваемые в конструктор класса `Driver`.

**Возвращает**:
- type: Созданный класс `Driver`.

**Проверяет**:

- `isinstance(webdriver_cls, type)`: Проверяет, является ли `webdriver_cls` классом.
- `issubclass(webdriver_cls, Chrome | Firefox | Edge)`: Проверяет, является ли `webdriver_cls` подклассом одного из разрешённых классов WebDriver (`Chrome`, `Firefox`, или `Edge`).

**Возможные исключения**:

- `AssertionError`: Возникает, если `webdriver_cls` не является классом или не является подклассом одного из разрешённых классов Selenium WebDriver.


## Пример использования

```python
# Пример создания драйвера Chrome
from selenium.webdriver.chrome.webdriver import WebDriver as Chrome
# ... (Импорты и определения других классов, если есть)
chrome_driver = Driver(Chrome, *args, **kwargs)

# Пример создания драйвера Firefox
from selenium.webdriver.firefox.webdriver import WebDriver as Firefox
firefox_driver = Driver(Firefox, *args, **kwargs)
```

**Примечание**: Для использования приведённого примера необходимо импортировать соответствующие классы Selenium WebDriver.
