## Функции

### `set_project_root`

**Описание**: 
Находит корневую директорию проекта, начиная с директории текущего файла, и поднимаясь вверх до первого каталога, содержащего любой из маркерных файлов.

**Параметры**:
- `marker_files` (tuple, optional): Имена файлов или каталогов для определения корня проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если найден, иначе - директория, где расположен скрипт.

## Переменные

### `__root__`

**Описание**:
Путь к корневой директории проекта.

### `settings`

**Описание**:
Словарь с настройками проекта, загруженный из `settings.json`.

### `doc_str`

**Описание**:
Строка с содержимым документации проекта, загруженная из `README.MD`.

### `__project_name__`

**Описание**:
Название проекта, полученное из `settings.json` или по умолчанию `hypotez`.

### `__version__`

**Описание**:
Версия проекта, полученная из `settings.json` или пустая строка.

### `__doc__`

**Описание**:
Содержание документации проекта.

### `__details__`

**Описание**:
Дополнительная информация о проекте. В текущей реализации - пустая строка.

### `__author__`

**Описание**:
Имя автора проекта, полученное из `settings.json` или пустая строка.

### `__copyright__`

**Описание**:
Авторские права проекта, полученные из `settings.json` или пустая строка.

### `__cofee__`

**Описание**:
Строка с призывом поддержать разработчика чашечкой кофе, полученная из `settings.json` или строка по умолчанию.
