# Модуль `version`

## Обзор

Модуль `version` содержит информацию о версии и другие метаданные, используемые в проекте. Он включает в себя переменные, такие как `__version__`, `__name__`, `__doc__`, `__details__`, `__annotations__` и `__author__`, которые предоставляют важные сведения о модуле.

## Переменные

### `MODE`

**Описание**:  Режим работы приложения (по умолчанию 'dev').

### `__name__`

**Описание**: Содержит имя модуля. Если скрипт запущен напрямую, значение будет `"__main__"`.

### `__version__`

**Описание**: Версия модуля или пакета.

### `__doc__`

**Описание**: Строка документации модуля.

### `__details__`

**Описание**: Дополнительные сведения о версии для модуля или класса.

### `__annotations__`

**Описание**: Содержит аннотации типов для переменных и функций в модуле.

### `__author__`

**Описание**: Имя(имена) автора(ов) модуля.
