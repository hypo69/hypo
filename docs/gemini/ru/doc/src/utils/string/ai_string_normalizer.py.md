# Модуль для нормализации ответа модели

## Обзор

Модуль `ai_response_normalizer.py` предоставляет функцию `normalize_answer`, предназначенную для очистки текстовых ответов, сгенерированных AI-моделями. Функция удаляет префиксы и суффиксы, часто используемые для обрамления кода или текста в ответах моделей, такие как ```md, ```html и т.д.

## Подробней

Этот модуль важен для предобработки ответов от AI, чтобы извлечь полезное содержимое без лишних обрамляющих конструкций. Он помогает приводить ответы к чистому виду, что упрощает их дальнейшую обработку и отображение.

    Args:
        text (str): Строка для нормализации.

        str: Нормализованная строка без префиксов и суффикса.

    1. Функция перебирает список префиксов `prefixes`.
    2. Для каждого префикса проверяется, начинается ли с него входная строка `text`.
    3. Если префикс найден, он удаляется из начала строки с использованием `text.removeprefix(prefix)`.
    4. После удаления префикса, функция удаляет суффикс '```' из конца строки с использованием `.removesuffix('```')`.
    5. Если ни один из префиксов не совпал с началом строки, возвращается исходная строка `text` без изменений.

    ASCII flowchart:

    Начало
     ↓
    Перебор префиксов
     ↓
    Начинается ли строка с префикса?
     ├── Да → Удаление префикса и суффикса '```'
     └── Нет → Следующий префикс
     ↓
    Возврат нормализованной строки

    Примеры:
        >>> normalize_answer("```html\\n<p>Hello</p>```")
        '<p>Hello</p>'

        >>> normalize_answer("Some text")
        'Some text'

        >>> normalize_answer("```md\\n# Заголовок```")
