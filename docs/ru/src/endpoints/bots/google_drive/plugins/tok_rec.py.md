# Модуль для определения токенов

## Обзор

Модуль содержит функцию `is_token`, которая используется для определения, является ли переданная строка токеном определенного формата.

## Подробней

Модуль `tok_rec.py` предназначен для верификации токенов. В текущей версии он проверяет, соответствует ли переданная строка определенному формату токена (длина 62 символа, второй символ `/`). Данный модуль может быть использован для валидации токенов, полученных из различных источников, например, из конфигурационных файлов или пользовательского ввода.

## Функции

### `is_token`

```python
def is_token(token: str) -> bool:
    """ Функция определяет, является ли переданная строка токеном заданного формата.

    Args:
        token (str): Строка, которую необходимо проверить на соответствие формату токена.

    Returns:
        bool: `True`, если строка соответствует формату токена, `False` в противном случае.

    Example:
        >>> is_token("some_string/with_62_symbols...................................")
        False
        >>> is_token("some_string /with_62_symbols...................................")
        False
        >>> is_token("1234567890/1234567890123456789012345678901234567890123456789012")
        True
    """
```

**Как работает функция**:

1. **Разделение строки**: Функция разделяет входную строку `token` на список подстрок, используя пробелы в качестве разделителей, и берет последний элемент этого списка.

2. **Проверка длины**: Проверяется, что длина полученной строки равна 62 символам.
3. **Проверка символа**: Если длина строки равна 62, проверяется, что второй символ в строке равен `/`.
4. **Возврат значения**: Если оба условия выполняются, функция возвращает `True`, иначе `False`.

```
Начало
   ↓
Разделение строки и получение последнего элемента
   ↓
Проверка длины строки == 62?
   ├── Да → Проверка второго символа == "/"?
   │           ├── Да → Возврат True
   │           └── Нет → Возврат False
   └── Нет → Возврат False
Конец
```

**Примеры**:

```python
>>> is_token("some_string/with_62_symbols...................................")
False
>>> is_token("some_string /with_62_symbols...................................")
False
>>> is_token("1234567890/1234567890123456789012345678901234567890123456789012")
True
