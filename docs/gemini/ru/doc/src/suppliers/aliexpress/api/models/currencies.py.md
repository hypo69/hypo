# Модуль `currencies`

## Обзор

Модуль `currencies.py` определяет класс `Currency`, содержащий константы для различных валют. Он предназначен для централизованного хранения и использования кодов валют, применяемых в API AliExpress.

## Подробней

Этот модуль предоставляет простой и удобный способ доступа к кодам валют, что позволяет избежать ошибок при их использовании в коде. Класс `Currency` содержит статические атрибуты, каждый из которых представляет собой код определенной валюты.

## Классы

### `Currency`

**Описание**: Класс, предназначенный для хранения кодов валют.

**Принцип работы**:
Класс `Currency` содержит статические атрибуты, представляющие коды валют. Это позволяет использовать коды валют как константы, что повышает читаемость и надежность кода.

**Атрибуты**:
- `USD` (str): Код доллара США.
- `GBP` (str): Код фунта стерлингов.
- `CAD` (str): Код канадского доллара.
- `EUR` (str): Код евро.
- `UAH` (str): Код украинской гривны.
- `MXN` (str): Код мексиканского песо.
- `TRY` (str): Код турецкой лиры.
- `RUB` (str): Код российского рубля.
- `BRL` (str): Код бразильского реала.
- `AUD` (str): Код австралийского доллара.
- `INR` (str): Код индийской рупии.
- `JPY` (str): Код японской иены.
- `IDR` (str): Код индонезийской рупии.
- `SEK` (str): Код шведской кроны.
- `KRW` (str): Код южнокорейской воны.
- `ILS` (str): Код израильского шекеля.
