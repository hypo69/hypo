# Модуль hypotez/src/utils/convertors/csv.py

## Обзор

Модуль `hypotez/src/utils/convertors/csv.py` предоставляет инструменты для конвертации данных в формате CSV в различные форматы, включая JSON и SimpleNamespace.  Модуль включает функции для чтения CSV-файлов в формате словарей и объектов `SimpleNamespace`, а также для записи данных в JSON-формат.

## Функции

### `csv2dict`

**Описание**: Функция преобразует данные из CSV-файла в словарь.

**Параметры**:
- `csv_file` (str | Path): Путь к CSV-файлу для чтения.

**Возвращает**:
- dict | None: Словарь, содержащий данные из CSV, преобразованные в JSON-формат, или `None`, если преобразование не удалось.

**Вызывает исключения**:
- `Exception`: Если не удаётся прочитать CSV-файл.


### `csv2ns`

**Описание**: Функция преобразует данные из CSV-файла в объекты `SimpleNamespace`.

**Параметры**:
- `csv_file` (str | Path): Путь к CSV-файлу для чтения.

**Возвращает**:
- SimpleNamespace | None: Объект `SimpleNamespace`, содержащий данные из CSV, или `None`, если преобразование не удалось.

**Вызывает исключения**:
- `Exception`: Если не удаётся прочитать CSV-файл.


### `csv_to_json`

**Описание**: Функция преобразует CSV-файл в JSON-формат и сохраняет результат в новый JSON-файл.

**Параметры**:
- `csv_file_path` (str | Path): Путь к CSV-файлу для чтения.
- `json_file_path` (str | Path): Путь к JSON-файлу для сохранения.
- `exc_info` (bool, optional): Если True, включает информацию об отладке в лог. По умолчанию True.

**Возвращает**:
- List[Dict[str, str]] | None: Данные в JSON-формате как список словарей, или `None`, если преобразование не удалось.

**Пример использования**:

```python
>>> json_data = csv_to_json('dialogue_log.csv', 'dialogue_log.json')
>>> print(json_data)
[{'role': 'user', 'content': 'Hello'}, {'role': 'assistant', 'content': 'Hi there!'}]
```

**Вызывает исключения**:
- `Exception`: Если возникает ошибка при чтении CSV или записи в JSON.  Ошибка логируется с помощью `logger`.


## Модульные переменные

### `MODE`

**Описание**:  Переменная, хранящая режим работы (например, 'dev', 'prod'). В данном случае это строка 'dev'.


## Замечания

Этот модуль использует функции из других модулей, таких как `src.logger`, `src.utils.csv`, для обработки CSV данных, что предполагает наличие этих функций.
