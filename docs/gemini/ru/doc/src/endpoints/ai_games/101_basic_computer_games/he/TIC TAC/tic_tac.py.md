# משחק איקס עיגול

## Обзор

קובץ זה מיישם את המשחק איקס עיגול, בו שחקן אחד מתחרה נגד המחשב. המשחק מתבצע על לוח 3x3, כאשר השחקן מסמן 'X' והמחשב מסמן 'O'. מטרת המשחק היא ליצור שורה של שלושה סימנים זהים, אופקית, אנכית או אלכסונית.

## Функции

### `print_board`

**Описание**:
מדפיסה את לוח המשחק הנוכחי למסוף. הלוח מוצג עם מספרים 1-9 כדי שהמשתמש יוכל לבחור את המשבצת

**Параметры**:
- `board` (list): לוח המשחק הנוכחי.

**Возвращает**:
- `None`: הפונקציה אינה מחזירה ערך.

### `check_win`

**Описание**:
בודקת אם שחקן מסוים ניצח. היא בודקת את כל השורות, העמודות והאלכסונים.

**Параметры**:
- `board` (list): לוח המשחק הנוכחי.
- `player` (str): השחקן שבודקים את הניצחון שלו ('X' או 'O').

**Возвращает**:
- `bool`: `True` אם השחקן ניצח, `False` אחרת.

### `is_board_full`

**Описание**:
בודקת האם כל המשבצות בלוח מלאות, כלומר אין יותר מקומות פנויים.

**Параметры**:
- `board` (list): לוח המשחק הנוכחי.

**Возвращает**:
- `bool`: `True` אם הלוח מלא, `False` אחרת.

### `computer_move`

**Описание**:
המחשב בוחר משבצת פנויה באופן אקראי. הוא ממשיך לבחור עד שהוא מוצא משבצת פנויה.

**Параметры**:
- `board` (list): לוח המשחק הנוכחי.

**Возвращает**:
- `tuple`: טופל המכיל את האינדקסים של השורה והעמודה שנבחרו.

### `play_tic_tac_toe`

**Описание**:
הפונקציה הראשית של המשחק איקס עיגול. מנהלת את כל התהליך של המשחק, משחקן נגד המחשב.

**Параметры**:
- `None`: הפונקציה לא מקבלת פרמטרים.

**Возвращает**:
- `None`: הפונקציה לא מחזירה ערך.
