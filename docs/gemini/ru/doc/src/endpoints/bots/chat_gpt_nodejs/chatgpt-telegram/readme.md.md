## Описание проекта

Проект представляет собой реализацию голосового помощника на основе ChatGPT, который интегрирован с Telegram. Пользователи могут взаимодействовать с ботом через Telegram, отправляя ему голосовые сообщения, которые затем будут обработаны и переданы в ChatGPT для получения ответа. Полученный ответ будет отправлен обратно пользователю в текстовом виде.

Данный проект демонстрирует, как можно использовать возможности ChatGPT для создания интеллектуальных ботов и голосовых ассистентов, а также интеграцию с популярными мессенджерами, такими как Telegram.
