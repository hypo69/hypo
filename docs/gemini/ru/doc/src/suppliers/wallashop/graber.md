## Декоратор (Комментированный)

### `close_pop_up`

**Описание**:
Этот декоратор (комментированный) предназначен для закрытия всплывающих окон перед выполнением основной логики функции. Он может быть переопределен в потомках для специфической обработки.


**Параметры**:
    * `value (Any)`: Дополнительные данные для декоратора. (По умолчанию `None`)

**Возвращает**:
    * `Callable`: Декоратор, оборачивающий функцию.


**Обрабатываемые исключения**:
    * `ExecuteLocatorException`: Исключение, выбрасываемое при ошибке выполнения локатора.



**Важно**: Данный декоратор закомментирован в коде, что означает, что он не активен по умолчанию. Для его использования, нужно раскомментировать код и переопределить логику внутри декоратора.
