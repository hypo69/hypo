# Модуль hypotez/src/goog/header.py

## Обзор

Этот модуль содержит функции для определения корневого каталога проекта и загрузки настроек из файла `settings.json` и документации из `README.MD`.  Он также инициализирует переменные для имени проекта, версии, документации, подробностей, автора, копирайта и ссылки на кофе.


## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная от текущего каталога и поднимаясь вверх по иерархии каталогов, пока не найдёт файл или каталог-маркер.

**Параметры**:
- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или каталогов, которые используются для определения корневого каталога проекта. По умолчанию: `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Объект `Path` представляющий корневой каталог проекта, если найден. В противном случае возвращает каталог, где расположен данный скрипт.

**Вызывает исключения**:
- Нет.


## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта. Инициализируется вызовом функции `set_project_root`.

**Тип**: `Path`.

### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженный из файла `src/settings.json`.

**Тип**: `dict` или `None`.

### `doc_str`

**Описание**: Строка, содержащая текст документации, загруженная из файла `src/README.MD`.

**Тип**: `str` или `None`.


### `__project_name__`

**Описание**: Имя проекта, полученное из настроек `settings`. По умолчанию: `'hypotez'`.

**Тип**: `str`.


### `__version__`

**Описание**: Версия проекта, полученная из настроек `settings`. По умолчанию: пустая строка.

**Тип**: `str`.


### `__doc__`

**Описание**: Документация проекта, полученная из `doc_str`. По умолчанию: пустая строка.

**Тип**: `str`.


### `__details__`

**Описание**: Подробности проекта. По умолчанию: пустая строка.

**Тип**: `str`.


### `__author__`

**Описание**: Автор проекта, полученный из настроек `settings`. По умолчанию: пустая строка.

**Тип**: `str`.


### `__copyright__`

**Описание**: Авторские права проекта, полученные из настроек `settings`. По умолчанию: пустая строка.

**Тип**: `str`.


### `__cofee__`

**Описание**: Ссылка на возможность поддержать разработчика чашкой кофе. По умолчанию: ссылка.

**Тип**: `str`.
