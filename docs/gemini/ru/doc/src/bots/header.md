# Модуль hypotez/src/bots/header.py

## Обзор

Данный модуль содержит функции для инициализации проекта. Он определяет корневую директорию проекта, загружает настройки из файла `settings.json` и загружает описание проекта из файла `README.MD`.

## Функции

### `set_project_root`

**Описание**:  Находит корневую директорию проекта, начиная от текущей директории файла и ищет вверх по дереву директорий, пока не найдет директорию, содержащую один из файлов-маркеров (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, по которым определяется корень проекта. По умолчанию содержит ('pyproject.toml', 'requirements.txt', '.git').

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена, иначе - путь к директории, в которой расположен скрипт.

**Вызывает исключения**:

- Нет


### <ins>Описание переменных</ins>

**Переменная**: `__root__`

**Описание**: Переменная, содержащая путь к корневой директории проекта. Ее значение устанавливается функцией `set_project_root`.

**Тип**: `Path`

**Переменная**: `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженные из файла `settings.json`.

**Тип**: `dict`

**Переменная**: `doc_str`

**Описание**: Строка, содержащая описание проекта, загруженная из файла `README.MD`.

**Тип**: `str`

**Переменная**: `__project_name__`

**Описание**: Название проекта, полученное из `settings.json`. По умолчанию `hypotez`.

**Тип**: `str`

**Переменная**: `__version__`

**Описание**: Версия проекта, полученная из `settings.json`. По умолчанию пустая строка.

**Тип**: `str`

**Переменная**: `__doc__`

**Описание**: Документация проекта, полученная из `README.MD`. По умолчанию пустая строка.

**Тип**: `str`

**Переменная**: `__details__`

**Описание**: Дополнительные детали проекта. По умолчанию пустая строка.

**Тип**: `str`

**Переменная**: `__author__`

**Описание**: Автор проекта, полученный из `settings.json`. По умолчанию пустая строка.

**Тип**: `str`

**Переменная**: `__copyright__`

**Описание**: Авторские права на проект, полученные из `settings.json`. По умолчанию пустая строка.

**Тип**: `str`

**Переменная**: `__cofee__`

**Описание**: Ссылка на платформ для подержки проекта.

**Тип**: `str`


##  Обработка исключений


Этот код использует конструкцию `try...except` для обработки потенциальных ошибок при чтении и парсинге файлов `settings.json` и `README.MD`:

- **FileNotFoundError:** Возникает, если файл не найден.
- **json.JSONDecodeError:** Возникает, если файл `settings.json` имеет неправильный формат JSON.

В блоке `except`  используется `...` что означает, что ошибки будут проигнорированы,  а программа продолжится выполнение. Это не лучший подход и стоит рассмотреть альтернативные способы обработки таких ошибок.
