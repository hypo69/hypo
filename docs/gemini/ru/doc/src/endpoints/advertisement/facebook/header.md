# Модуль hypotez/src/endpoints/advertisement/facebook/header.py

## Обзор

Данный модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`. Он также предоставляет доступ к информации о проекте, такой как название, версия, описание и другие детали, если они доступны.

## Функции

### `set_project_root`

**Описание**:  Находит корневой каталог проекта, начиная с текущей директории, ищет вверх по директориям, пока не найдет директорию, содержащую файлы маркеров (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:
- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, используемых для определения корневого каталога проекта. По умолчанию: (`'pyproject.toml'`, `'requirements.txt'`, `.git`).

**Возвращает**:
- `Path`: Путь к корневому каталогу проекта, если он найден. В противном случае возвращает директорию, в которой находится текущий файл.


**Вызывает исключения**:
- Нет

## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта. Инициализируется функцией `set_project_root()`. Тип: `Path`.

### `settings`

**Описание**: Словарь с настройками проекта, загруженными из файла `settings.json`. Тип: `dict`. Значение по умолчанию: `None`.

### `doc_str`

**Описание**: Строковое представление файла `README.MD`, если он существует.  Тип: `str`. Значение по умолчанию: `None`.


### `__project_name__`

**Описание**: Название проекта. Возвращает значение `project_name` из настроек или `hypotez` по умолчанию. Тип: `str`.

### `__version__`

**Описание**: Версия проекта. Возвращает значение `version` из настроек или пустую строку по умолчанию. Тип: `str`.

### `__doc__`

**Описание**: Описание проекта. Возвращает содержимое `README.MD` или пустую строку. Тип: `str`.

### `__details__`

**Описание**: Дополнительные детали проекта. По умолчанию пустая строка. Тип: `str`.

### `__author__`

**Описание**: Автор проекта. Возвращает значение `author` из настроек или пустую строку по умолчанию. Тип: `str`.

### `__copyright__`

**Описание**: Авторские права проекта. Возвращает значение `copyright` из настроек или пустую строку по умолчанию. Тип: `str`.

### `__cofee__`

**Описание**: Ссылка на спонсорскую страницу для поддержания разработчика. Возвращает значение `cofee` из настроек или строку с ссылкой по умолчанию. Тип: `str`.


## Обработка исключений

Файл использует блоки `try...except` для обработки возможных ошибок при чтении файла `settings.json` и `README.MD`:

- `FileNotFoundError`:  Ошибка, если файл не найден.
- `json.JSONDecodeError`: Ошибка, если файл `settings.json` не соответствует формату JSON.

В случае ошибки, соответствующие переменные остаются с установленным значением по умолчанию.
