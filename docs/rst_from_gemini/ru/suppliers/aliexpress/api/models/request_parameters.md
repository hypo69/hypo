**Описание:**

Этот файл определяет перечисления (enum) для параметров запросов API AliExpress, используемых в модуле `src.suppliers.aliexpress.api.models`.  Он задаёт возможные значения для различных параметров сортировки и типов ссылок.

**Классы:**

* **`ProductType`:**
    * `ALL`: Представляет все типы продуктов.
    * `PLAZA`: Представляет тип продуктов с платформы PLAZA.
    * `TMALL`: Представляет тип продуктов с платформы TMALL.

* **`SortBy`:**
    * `SALE_PRICE_ASC`: Сортировка по цене продажи в порядке возрастания.
    * `SALE_PRICE_DESC`: Сортировка по цене продажи в порядке убывания.
    * `LAST_VOLUME_ASC`: Сортировка по последнему объёму продаж в порядке возрастания.
    * `LAST_VOLUME_DESC`: Сортировка по последнему объёму продаж в порядке убывания.

* **`LinkType`:**
    * `NORMAL`: Обычный тип ссылки.
    * `HOTLINK`: Тип ссылки, помеченный как "горячая ссылка".


**Рекомендации по использованию:**

При работе с API AliExpress в коде проекта необходимо использовать эти перечисления для задания параметров сортировки и типов ссылок. Например:
