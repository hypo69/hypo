# Модуль `src.category.header`

## Обзор

Модуль `src.category.header` определяет корневой путь к проекту. Все импорты строятся относительно этого пути. 
В дальнейшем планируется перенести эту функциональность в системную переменную.

## Функции

### `set_project_root`

**Описание**:
Находит корневой каталог проекта, начиная с каталога текущего файла,
поиск ведется вверх по дереву каталогов и останавливается на первом каталоге, содержащем любой из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple, optional):  Имена файлов или каталогов для идентификации корня проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневому каталогу, если он найден, в противном случае - каталог, где расположен скрипт.

## Переменные

### `__root__`

**Описание**:
Путь к корневому каталогу проекта.

### `settings`

**Описание**:
Словарь с настройками проекта, загруженный из файла `settings.json`. Если файл не найден или произошла ошибка декодирования JSON, значение будет `None`.

### `doc_str`

**Описание**:
Содержимое файла `README.md` в виде строки. Если файл не найден или произошла ошибка декодирования, значение будет `None`.

### `__project_name__`

**Описание**:
Название проекта. Значение берется из `settings.json`, если файл существует и содержит поле `project_name`. В противном случае значение по умолчанию `'hypotez'`.

### `__version__`

**Описание**:
Версия проекта. Значение берется из `settings.json`, если файл существует и содержит поле `version`. В противном случае значение по умолчанию `''`.

### `__doc__`

**Описание**:
Строка документации проекта. Значение берется из переменной `doc_str` если она определена. В противном случае значение по умолчанию `''`.

### `__details__`

**Описание**:
Строка для дополнительных деталей. Значение по умолчанию `''`.

### `__author__`

**Описание**:
Автор проекта. Значение берется из `settings.json`, если файл существует и содержит поле `author`. В противном случае значение по умолчанию `''`.

### `__copyright__`

**Описание**:
Авторские права проекта. Значение берется из `settings.json`, если файл существует и содержит поле `copyrihgnt`. В противном случае значение по умолчанию `''`.

### `__cofee__`

**Описание**:
Сообщение для поддержки разработчика. Значение берется из `settings.json`, если файл существует и содержит поле `cofee`. В противном случае используется значение по умолчанию `"Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69"`.
