# Модуль hypotez/src/endpoints/header.py

## Обзор

Этот модуль содержит функцию `get_project_root`, которая находит корневой каталог проекта, а также переменные, содержащие информацию о проекте (имя, версия, описание и т.д.), полученные из файла `settings.json` и `README.MD`.

## Функции

### `get_project_root`

**Описание**:  Функция ищет корневой каталог проекта, начиная с текущего каталога и поднимаясь вверх по дереву каталогов, пока не найдет каталог, содержащий указанные файлы-маркеры.

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или каталогов, используемых для определения корневого каталога проекта. По умолчанию используется `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневому каталогу проекта, если он найден. В противном случае возвращает каталог, где находится скрипт.

**Вызывает исключения**:
  - Никаких исключений не генерирует.

### `settings_load`

**Описание**: Загрузка настроек проекта из файла settings.json.

**Параметры**:
-  Никаких параметров.

**Возвращает**:
- `dict`: Словарь с настройками проекта, если файл `settings.json` найден и корректен, иначе None

**Вызывает исключения**:
- `FileNotFoundError`: Если файл `settings.json` не найден.
- `json.JSONDecodeError`: Если содержимое файла `settings.json` не является корректным JSON.


## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта, полученный с помощью функции `get_project_root`.


### `settings`

**Описание**: Словарь настроек проекта. Загружается из файла `settings.json`. Может быть `None`, если файл не найден или содержит ошибки.


### `doc_str`

**Описание**: Содержимое файла `README.MD`. Загружается из файла `README.MD`. Может быть `None`, если файл не найден или содержит ошибки.


### `__project_name__`

**Описание**: Имя проекта, полученное из настроек. По умолчанию 'hypotez'.

### `__version__`

**Описание**: Версия проекта, полученная из настроек. По умолчанию пустая строка.


### `__doc__`

**Описание**: Документация к проекту, полученная из файла `README.MD`. По умолчанию пустая строка.


### `__details__`

**Описание**: Дополнительные детали проекта. По умолчанию пустая строка.


### `__author__`

**Описание**: Автор проекта, полученный из настроек. По умолчанию пустая строка.


### `__copyright__`

**Описание**: Авторские права проекта, полученные из настроек. По умолчанию пустая строка.


### `__cofee__`

**Описание**: Ссылка на страницу для пожертвований разработчику.


## Модули

### `gs`

**Описание**:  Модуль `gs`, который предоставляет функции для работы с файлами проекта. (Необходимость в подробной документации к модулю `gs` для корректной работы этой документации)
