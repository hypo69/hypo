# Модуль onela_bot

## Обзор

Модуль `onela_bot` предоставляет класс `OnelaBot` для взаимодействия с моделью ассистента программиста через чат Telegram. Он обрабатывает текстовые сообщения и загруженные документы, передавая их модели и возвращая ответы.

## Классы

### `OnelaBot`

**Описание**: Класс `OnelaBot` наследуется от `TelegramBot` и реализует взаимодействие с моделью ассистента программиста.

**Атрибуты**:

* `model`: Объект класса `GoogleGenerativeAI`, используемый для запросов к модели.

**Методы**:

#### `__init__`

**Описание**: Инициализирует объект `OnelaBot`.

**Параметры**:

* Нет

**Возвращает**:
  * Нет


#### `handle_message`

**Описание**: Обрабатывает текстовые сообщения от пользователя.

**Параметры**:

* `update` (Update): Данные обновления Telegram.
* `context` (CallbackContext): Контекст выполнения.

**Возвращает**:
  * `None`

**Обрабатывает исключения**:
* `Exception`: В случае возникновения ошибки при обработке сообщения.


#### `handle_document`

**Описание**: Обрабатывает загруженные документы от пользователя.

**Параметры**:

* `update` (Update): Данные обновления Telegram.
* `context` (CallbackContext): Контекст выполнения.

**Возвращает**:
  * `None`

**Обрабатывает исключения**:
* `Exception`: В случае возникновения ошибки при обработке документа.

## Использование

Для запуска бота необходимо импортировать модуль и создать экземпляр класса `OnelaBot`. Затем нужно запустить цикл обработки сообщений с помощью `asyncio.run(bot.application.run_polling())`.


```
```python
import asyncio
from src.endpoints.hypo69.code_assistant import onela_bot

# ...
bot = onela_bot.OnelaBot()
asyncio.run(bot.application.run_polling())
