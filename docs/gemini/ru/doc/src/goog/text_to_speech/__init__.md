# Модуль hypotez/src/goog/text_to_speech/__init__.py

## Обзор

Данный модуль предоставляет функции для синтеза речи с использованием Google Text-to-Speech API.  Модуль использует библиотеку `pyttsx3` для взаимодействия с API Google TTS.

## Константы

### `MODE`

**Значение:** `'dev'`

**Описание**:  Переменная, определяющая режим работы модуля (в данном случае, `dev`).

**Методы**:
