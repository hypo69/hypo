# Модуль `hypotez/src/suppliers/aliexpress/campaign/html_generators.py`

## Обзор

Этот модуль содержит классы для генерации HTML-контента для рекламных кампаний AliExpress.  Он предоставляет функции для создания HTML-страниц отдельных товаров, категорий товаров и обзора всей кампании.  Модуль использует Bootstrap для стилизации вывода.

## Оглавление

- [Генератор HTML для продуктов](#генератор-html-для-продуктов)
- [Генератор HTML для категорий](#генератор-html-для-категорий)
- [Генератор HTML для кампании](#генератор-html-для-кампании)


## Классы

### `ProductHTMLGenerator`

**Описание**: Класс для генерации HTML-страниц отдельных товаров.

**Методы**

#### `set_product_html`

**Описание**: Создает HTML-файл для отдельного продукта.

**Параметры**:

- `product` (SimpleNamespace): Объект SimpleNamespace, содержащий данные о продукте (например, `product_id`, `product_title`, `target_sale_price`, `local_image_path`, `promotion_link` и др.).
- `category_path` (str | Path): Путь к каталогу, в котором будет сохранен HTML-файл.

**Возвращает**:
  - `None`


### `CategoryHTMLGenerator`

**Описание**: Класс для генерации HTML-страниц категорий товаров.

**Методы**

#### `set_category_html`

**Описание**: Создает HTML-файл для категории товаров.

**Параметры**:

- `products_list` (list[SimpleNamespace] | SimpleNamespace): Список объектов SimpleNamespace, содержащих данные о продуктах в данной категории.  Может быть как список, так и одиночным объектом SimpleNamespace, если требуется сгенерировать HTML для единственного продукта в категории.
- `category_path` (str | Path): Путь к каталогу, в котором будет сохранен HTML-файл.

**Возвращает**:
  - `None`


### `CampaignHTMLGenerator`

**Описание**: Класс для генерации HTML-страницы обзора всей рекламной кампании.

**Методы**

#### `set_campaign_html`

**Описание**: Создает HTML-файл для обзора рекламной кампании, перечисляя все доступные категории.

**Параметры**:

- `categories` (list[str]): Список имен категорий товаров.
- `campaign_path` (str | Path): Путь к каталогу, в котором будет сохранен HTML-файл.

**Возвращает**:
  - `None`

### `MODE`
