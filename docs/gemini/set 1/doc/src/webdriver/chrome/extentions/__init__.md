# Модуль hypotez/src/webdriver/chrome/extentions/__init__.py

## Обзор

Этот модуль предоставляет начальную настройку для расширений браузера Chrome.  Он содержит конфигурационные переменные и импорты, необходимые для работы с расширениями.


## Переменные

### `MODE`

**Описание**:  Переменная, определяющая режим работы (например, 'dev', 'prod').

**Значение**:  `'dev'` по умолчанию.


## Модули

### `packaging.version`

**Описание**: Модуль для работы с версиями пакетов.


### `extentions.version`

**Описание**: Модуль для доступа к версии, документации и дополнительным данным расширения.

**Атрибуты**:
- `__version__`: Строка с версией.
- `__doc__`: Строка с документацией.
- `__details__`: Словарь с дополнительными деталями.
