# Модуль `xls`

## Обзор

Модуль `xls` предоставляет функции для конвертации данных из файлов Excel в формат словаря.

## Оглавление

- [Функции](#функции)
    - [`xls2dict`](#xls2dict)

## Функции

### `xls2dict`

**Описание**: Функция для чтения данных из xls файла и конвертации в словарь.

**Параметры**:
- `xls_file` (str | Path): Путь к xls файлу.

**Возвращает**:
- `dict | None`:  Словарь, полученный из xls файла, или `None` если произошла ошибка.
