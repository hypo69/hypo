# Документация для `morlevi_categories_network.json`

## Обзор

Данный JSON-файл содержит конфигурацию сценариев для категорий товаров "Сетевое оборудование" для поставщика Morlevi. Каждый сценарий определяет URL категории на сайте поставщика, состояние чекбокса, активность сценария, условие и соответствие категориям PrestaShop.

## Структура

### `scenarios`
Словарь, содержащий сценарии для различных категорий сетевого оборудования. Ключи словаря представляют собой названия категорий.

#### `routers`
Сценарий для категории "Роутеры".

**Описание**:
Описывает настройки для категории "Роутеры" на сайте поставщика Morlevi.

**Содержит**:
- `url` (str): URL страницы категории на сайте поставщика: "https://www.morlevi.co.il/Cat/111".
- `checkbox` (bool): Состояние чекбокса (всегда `false`).
- `active` (bool): Активность сценария (всегда `true`).
- `condition` (str): Условие сценария (всегда `new`).
- `presta_categories` (str): Список ID категорий PrestaShop, связанных с данной категорией (в данном случае "210,211").
