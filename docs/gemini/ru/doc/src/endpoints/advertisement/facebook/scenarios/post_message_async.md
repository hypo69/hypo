# Модуль hypotez/src/endpoints/advertisement/facebook/scenarios/post_message_async.py

## Обзор

Этот модуль содержит функции для асинхронной публикации сообщений в Facebook, связанных с рекламными кампаниями AliExpress. Он предоставляет инструменты для заполнения заголовка и описания поста, загрузки медиафайлов (изображений и видео) и добавления подписей к ним.

## Функции

### `post_title`

**Описание**: Функция отправляет заголовок и описание рекламной кампании в поле для публикации сообщения на Facebook.

**Параметры**:

- `d` (Driver): Экземпляр драйвера Selenium, используемый для взаимодействия с веб-страницей.
- `category` (SimpleNamespace): Объект, содержащий заголовок и описание кампании.

**Возвращает**:

- `bool`: `True`, если заголовок и описание отправлены успешно; `None`, если произошла ошибка.

**Обрабатывает исключения**:

- `Exception`: В случае возникновения любой ошибки при выполнении функции.


### `upload_media`

**Описание**: Функция загружает медиафайлы (изображения и видео) в раздел изображения и обновляет подписи.

**Параметры**:

- `d` (Driver): Экземпляр драйвера Selenium, используемый для взаимодействия с веб-страницей.
- `products` (List[SimpleNamespace]): Список объектов, содержащих пути к локальным медиафайлам.
- `no_video` (bool, опционально): Флаг, указывающий, что не нужно загружать видео. По умолчанию `False`.

**Возвращает**:

- `bool`: `True`, если медиафайлы загружены успешно; `None`, если произошла ошибка.

**Обрабатывает исключения**:

- `Exception`: В случае возникновения любой ошибки при загрузке или обновлении подписей.


### `update_images_captions`

**Описание**: Функция асинхронно добавляет описания к загруженным медиафайлам.

**Параметры**:

- `d` (Driver): Экземпляр драйвера Selenium, используемый для взаимодействия с веб-страницей.
- `products` (List[SimpleNamespace]): Список объектов, содержащих данные для обновления подписей.
- `textarea_list` (List[WebElement]): Список областей ввода, куда нужно добавить подписи.

**Обрабатывает исключения**:

- `Exception`: В случае возникновения ошибки при обновлении подписей.


### `promote_post`

**Описание**: Управляет процессом продвижения поста с заголовком, описанием и медиафайлами.

**Параметры**:

- `d` (Driver): Экземпляр драйвера Selenium, используемый для взаимодействия с веб-страницей.
- `category` (SimpleNamespace): Объект с деталями заголовка и описания поста.
- `products` (List[SimpleNamespace]): Список объектов, содержащих медиафайлы и детали для публикации.
- `no_video` (bool, опционально): Флаг, указывающий, что не нужно загружать видео. По умолчанию `False`.

**Возвращает**:

- `bool`: `True`, если пост опубликован успешно; `None`, если произошла ошибка.
