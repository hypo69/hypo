# Модуль hypotez/src/utils/path.py

## Обзор

Модуль `path` определяет корневой путь к проекту `hypotez`. Все импорты строятся относительно этого пути.  В дальнейшем планируется перенести определение пути в системные переменные.

## Переменные

### `MODE`

**Описание**: Строковая переменная, определяющая режим работы.  Текущее значение: 'dev'.

## Функции

### `get_relative_path`

**Описание**: Возвращает часть пути начиная с указанного сегмента и до конца.

**Параметры**:

- `full_path` (str): Полный путь.
- `relative_from` (str): Сегмент пути, с которого нужно начать извлечение.

**Возвращает**:

- `Optional[str]`: Относительный путь начиная с `relative_from`, или `None`, если сегмент не найден.

**Пример использования**:

```python
full_path = "/home/user/project/data/file.txt"
relative_from = "data"
relative_path = get_relative_path(full_path, relative_from)
print(relative_path)  # Вывод: file.txt
```

**Реализация**:

Функция преобразует строки `full_path` и `relative_from` в объекты `Path`. Затем находит индекс сегмента `relative_from` в списке компонентов пути. Если сегмент найден, формирует относительный путь, начиная с этого сегмента, используя метод `Path(*parts[start_index:])`. Иначе возвращает `None`.  Результат преобразуется в строку с помощью `as_posix()`.
