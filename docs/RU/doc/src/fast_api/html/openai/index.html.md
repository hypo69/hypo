# `index.html`

## Обзор

Этот HTML-файл представляет собой веб-интерфейс для взаимодействия с моделью OpenAI. Он позволяет пользователю отправлять запросы к модели, а также обучать модель, предоставляя данные в формате CSV.

## HTML Структура

Структура HTML-файла включает в себя следующие элементы:

- Заголовок (`<title>`): `OpenAI Model Interaction`
- Подключение CSS Bootstrap для стилизации
- Подключение AngularJS
- Форма для ввода сообщения и системной инструкции
- Кнопка для отправки запроса к модели
- Область для отображения ответа модели
- Форма для ввода данных для обучения модели
- Кнопка для запуска обучения модели
- Область для отображения ID задания обучения

## AngularJS Контроллер

### `MainController`

AngularJS контроллер `MainController` управляет взаимодействием с пользовательским интерфейсом и отправкой данных на сервер.

**Описание**: Контроллер обрабатывает ввод пользователя, отправляет запросы к серверу и обновляет пользовательский интерфейс.

**Методы**:

- [`askModel()`](#askmodel): Отправляет запрос к модели OpenAI.
- [`trainModel()`](#trainmodel): Запускает процесс обучения модели.

#### `askModel()`

**Описание**: Отправляет сообщение и системную инструкцию на сервер для получения ответа от модели OpenAI.

**Параметры**:
   - Нет явных параметров. Использует значения `vm.message` и `vm.systemInstruction` из модели представления.

**Возвращает**:
   - Нет явного возвращаемого значения. Обновляет `vm.response` в зависимости от ответа сервера.

#### `trainModel()`

**Описание**: Отправляет данные для обучения модели на сервер и получает ID задания обучения.

**Параметры**:
   - Нет явных параметров. Использует значение `vm.trainingData` из модели представления.

**Возвращает**:
   - Нет явного возвращаемого значения. Обновляет `vm.jobId` в зависимости от ответа сервера.
