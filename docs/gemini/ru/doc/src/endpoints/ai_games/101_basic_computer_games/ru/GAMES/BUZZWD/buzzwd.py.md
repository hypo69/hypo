# Игра BUZZWD

## Обзор

Данный модуль реализует игру "BUZZWD", в которой компьютер загадывает случайное число от 1 до 100, а игрок должен угадать это число. Программа предоставляет подсказки "СЛИШКОМ МАЛО" или "СЛИШКОМ МНОГО" после каждой попытки, пока игрок не угадает число.

## Описание игры
"BUZZWD" - это простая игра, где компьютер загадывает случайное число, а игрок пытается его угадать. Компьютер дает подсказки, является ли введенное число "слишком высоким" или "слишком низким".

## Используемые переменные
- `numberOfGuesses` (int): Счетчик количества попыток, которые сделал игрок.
- `targetNumber` (int): Случайное целое число от 1 до 100, которое нужно угадать.
- `userGuess` (int): Число, введенное игроком.

## Основной игровой цикл
- Цикл `while True` продолжает игру до тех пор, пока игрок не угадает загаданное число.
- `numberOfGuesses` увеличивается на 1 с каждой попыткой.
- Происходит ввод числа пользователем и его проверка.
- Если `userGuess` соответствует `targetNumber`, игра завершается.

## Обработка ввода пользователя
- Блок `try...except ValueError` обрабатывает ошибки ввода.
- Если пользователь вводит не целое число, выводится сообщение об ошибке.
- Ввод преобразуется в целое число и сохраняется в `userGuess`.

## Условия завершения игры
- Игра заканчивается, когда `userGuess` равен `targetNumber`.
- Сообщение "УГАДАНО" выводится на экран.
- Выводится общее количество попыток `numberOfGuesses`.
- Выполняется оператор `break` для выхода из цикла.
- Если `userGuess` меньше `targetNumber`, выводится "СЛИШКОМ МАЛО".
- Если `userGuess` больше `targetNumber`, выводится "СЛИШКОМ МНОГО".

## Функции

### `random.randint(a, b)`

**Описание**:
Генерирует случайное целое число N, такое что a <= N <= b.

**Параметры**:
- `a` (int): Нижняя граница диапазона.
- `b` (int): Верхняя граница диапазона.

**Возвращает**:
- `int`: Случайное целое число в заданном диапазоне.

### `input(prompt)`

**Описание**:
Отображает приглашение для ввода (`prompt`) и считывает строку введенную пользователем.

**Параметры**:
- `prompt` (str): Текст приглашения к вводу.

**Возвращает**:
- `str`: Строка, введенная пользователем.

### `int(x)`

**Описание**:
Преобразует число или строку в целое число.

**Параметры**:
- `x` (str | int): Число или строка, которую нужно преобразовать.

**Возвращает**:
- `int`: Целочисленное представление `x`.

**Вызывает исключения**:
- `ValueError`: Если `x` не может быть преобразовано в целое число.
