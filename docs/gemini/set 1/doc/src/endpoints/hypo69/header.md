# Модуль hypotez/src/endpoints/hypo69/header.py

## Обзор

Этот модуль содержит вспомогательные функции для работы с проектом, включая определение корневой директории проекта и загрузку настроек из файла `settings.json`. Он также предоставляет переменные, хранящие информацию о проекте, такие как название, версия, документация и автор.

## Функции

### `set_project_root`

**Описание**: Определяет корневую директорию проекта, начиная от директории текущего файла.

**Параметры**:

- `marker_files` (tuple): Кортеж из файлов или директорий, по которым определяется корневая директория проекта. По умолчанию - `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена, иначе - директория расположения текущего файла.

**Вызывает исключения**:

- Нет.


###  Описание переменных:


**__root__**:


- **Тип:** Path
- **Описание:** Путь к корневой директории проекта. Получается вызовом функции `set_project_root()`.

**settings**:

- **Тип:** dict
- **Описание:** Словарь настроек проекта, загруженный из файла `settings.json` в корневой директории проекта.  Возможен `None` при ошибке загрузки или отсутствии файла.

**doc_str**:

- **Тип:** str
- **Описание:** Строка, содержащая содержание файла README.md из корневой директории проекта. Возможен `None` при ошибке загрузки или отсутствии файла.

**__project_name__**:

- **Тип:** str
- **Описание:** Название проекта, полученное из настроек (settings). По умолчанию - 'hypotez'.

**__version__**:

- **Тип:** str
- **Описание:** Версия проекта, полученная из настроек (settings). По умолчанию - пустая строка.

**__doc__**:

- **Тип:** str
- **Описание:** Документация проекта (содержимое файла README.md), полученная из настроек (settings). По умолчанию - пустая строка.

**__details__**:

- **Тип:** str
- **Описание:** Подробная информация о проекте, полученная из настроек (settings). По умолчанию - пустая строка.

**__author__**:

- **Тип:** str
- **Описание:** Автор проекта, полученный из настроек (settings). По умолчанию - пустая строка.

**__copyright__**:

- **Тип:** str
- **Описание:** Авторские права проекта, полученные из настроек (settings). По умолчанию - пустая строка.

**__cofee__**:

- **Тип:** str
- **Описание:** Ссылка на поддержку разработчика (кофе), полученная из настроек (settings). По умолчанию - ссылка по умолчанию.
