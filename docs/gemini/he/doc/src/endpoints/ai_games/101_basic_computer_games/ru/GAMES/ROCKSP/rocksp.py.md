# rocksp.py

## סקירה כללית

מודול זה מיישם את המשחק הקלאסי "אבן, נייר ומספריים". המשתמש מתבקש לבחור בין אבן (1), נייר (2) או מספריים (3), והמחשב בוחר באופן אקראי אחת מהאפשרויות. המודול מכריע מי המנצח או אם יש תיקו ומציג את התוצאה למשתמש.
