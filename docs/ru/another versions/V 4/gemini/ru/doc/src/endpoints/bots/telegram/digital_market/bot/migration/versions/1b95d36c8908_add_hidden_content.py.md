# 1b95d36c8908_add_hidden_content

## Обзор

Модуль содержит Alembic миграцию для добавления столбца `hidden_content` в таблицу `products`. Alembic используется для управления изменениями схемы базы данных.

## Подробней

Этот файл содержит миграцию базы данных, которая добавляет новый столбец `hidden_content` типа `Text` в таблицу `products`. Этот столбец не может быть `NULL`.
Миграция также содержит функцию `downgrade`, которая удаляет столбец `hidden_content` из таблицы `products` при откате миграции.
