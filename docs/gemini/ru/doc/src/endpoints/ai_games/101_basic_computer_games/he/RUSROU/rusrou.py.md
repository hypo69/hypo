# Игра "Русская рулетка"

## Обзор

Этот модуль реализует игру "Русская рулетка", в которой игрок выбирает число от 1 до 6, и если это число совпадает со случайно выбранным числом, игрок "умирает".

## Функции

### `random.randint(a, b)`

**Описание**:
Функция `randint` из модуля `random` используется для генерации случайного целого числа в диапазоне от `a` до `b` включительно.

**Параметры**:
- `a` (int): Нижняя граница диапазона (включительно).
- `b` (int): Верхняя граница диапазона (включительно).

**Возвращает**:
- int: Случайное целое число в диапазоне от `a` до `b`.

### `input(prompt)`

**Описание**:
Функция `input` используется для получения ввода от пользователя.

**Параметры**:
- `prompt` (str): Сообщение, которое будет выведено пользователю для запроса ввода.

**Возвращает**:
- str: Строка, введенная пользователем.

### `int(x)`

**Описание**:
Функция `int` используется для преобразования значения `x` в целое число.

**Параметры**:
- `x` (str | int): Значение, которое нужно преобразовать в целое число.

**Возвращает**:
- int: Целое число, полученное из `x`.

**Вызывает исключения**:
- ValueError: Если `x` не может быть преобразовано в целое число.

## Переменные

### `numberOfTries`

**Описание**:
Переменная `numberOfTries` хранит количество попыток, сделанных игроком в текущей игровой сессии.

**Тип**:
- int

### `userNumber`

**Описание**:
Переменная `userNumber` хранит число, введенное пользователем.

**Тип**:
- int

### `randomNumber`

**Описание**:
Переменная `randomNumber` хранит случайное число, сгенерированное компьютером.

**Тип**:
- int

## Использование

1.  **Импорт модуля `random`**:

    ```python
    import random
    ```
    Импортируем модуль `random` для генерации случайных чисел.

2.  **Инициализация количества попыток**:

    ```python
    numberOfTries = 0
    ```
    Инициализируем переменную `numberOfTries` нулём. Эта переменная отслеживает количество попыток пользователя.

3.  **Основной игровой цикл**:
    ```python
    while True:
    ```
    Запускается бесконечный цикл, который продолжается до тех пор, пока игрок не проиграет.

4.  **Увеличение количества попыток**:
    ```python
    numberOfTries += 1
    ```
    Увеличиваем количество попыток на 1 перед каждым новым раундом.

5.  **Получение ввода от пользователя**:
    ```python
    try:
        userNumber = int(input("בחר מספר בין 1 ל-6: "))
        if userNumber < 1 or userNumber > 6:
             print("מספר לא תקין. אנא בחר מספר בין 1 ל-6.")
             continue
    except ValueError:
        print("קלט לא תקין. אנא הזן מספר שלם.")
        continue
    ```
    Пробуем получить ввод пользователя и преобразовать его в целое число. Если пользователь вводит неверное число, выводится сообщение об ошибке, и цикл начинается заново.

6. **Генерация случайного числа**:

    ```python
    randomNumber = random.randint(1, 6)
    ```
    Генерируем случайное число в диапазоне от 1 до 6.

7. **Проверка на совпадение**:
    ```python
    if userNumber == randomNumber:
        print(f"אתה מת! ב-{numberOfTries} נסיונות.")
        break
    ```
    Проверяем, совпадает ли число пользователя со случайным числом. Если совпадают, выводим сообщение о проигрыше с указанием количества попыток и выходим из цикла.
