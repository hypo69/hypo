# Документация для `template_table_he.html`

## Обзор

Файл `template_table_he.html` представляет собой HTML-шаблон для отображения прайс-листа на иврите (справа налево). Он предназначен для динамического наполнения данными о товарах, их описаниях, изображениях и общей стоимости. Шаблон использует стили CSS для оформления элементов и обеспечивает корректное отображение текста справа налево.

## Структура

Шаблон состоит из следующих основных частей:

*   `<head>`: Содержит метаданные, заголовок страницы и стили CSS.
*   `<body>`: Содержит основное содержимое страницы, включая заголовок, описание, карточки товаров и подвал.
*   `<div>` с классом `container`: Основной контейнер для всего содержимого.
*   `<h1>`: Заголовок прайс-листа, динамически задаваемый переменной `title`.
*   `<p>` с классом `lead`: Описание прайс-листа, динамически задаваемое переменной `description`.
*   `{% for product in products %}`: Цикл для перебора списка товаров.
*   `<div>` с классом `product-card`: Карточка товара, содержащая изображение и информацию о товаре.
*   `<img>`: Изображение товара, путь к которому динамически задается `product.image_local_saved_path`.
*   `<div>` с классом `product-info`: Контейнер для информации о товаре (название, описание, спецификация).
*   `<h3>`: Название товара, динамически задаваемое `product.product_title`.
*   `<p>`: Описание товара, динамически задаваемое `product.product_description`.
*   `<p>`: Спецификация товара, динамически задаваемая `product.specification`.
*   `<div>` с классом `footer`: Подвал страницы с итоговой ценой.
*   `<span>` с классом `price-tag`: Итоговая цена, динамически задаваемая переменными `price` и `currency`.

## Стили

Стили CSS определены внутри `<style>` и включают:

*   Общие стили для `body`: белый фон, черный текст, шрифт, направление текста справа налево.
*   Стили для `h1`: центрированный заголовок, размер шрифта и отступ сверху.
*   Стили для `.lead`: центрированный текст, размер шрифта и отступы.
*   Стили для `.container`: ограничение ширины и центрирование.
*   Стили для `.product-card`: оформление карточек товаров, включая светлый фон, границы, отступы и flexbox для выравнивания.
*   Стили для `product-card h3`: отступы для заголовка товара.
*   Стили для `product-card img`: размер, сохранение пропорций и отступы для изображений.
*   Стили для `.product-info`: заполнение оставшегося пространства flexbox.
*   Стили для `.price-tag`: оформление тега цены, включая цвет фона, цвет текста, скругление углов и жирный шрифт.
*   Стили для `.footer`: выравнивание текста, отступы и верхняя граница.

## Динамическое наполнение

Шаблон использует Jinja для динамического наполнения данными.

### Переменные Jinja

*   `title`: Заголовок прайс-листа.
*   `description`: Описание прайс-листа.
*   `products`: Список словарей, каждый из которых представляет информацию о товаре (название, описание, изображение и спецификацию).
*   `product.image_local_saved_path`: Путь к изображению товара.
*   `product.product_title`: Название товара.
*   `product.product_description`: Описание товара.
*   `product.specification`: Спецификация товара.
*    `price`: Общая стоимость.
*   `currency`: Валюта.

## HTML-элементы

Шаблон использует следующие HTML-элементы:

*   `<!DOCTYPE html>`: Объявление типа документа.
*   `<html>`: Корневой элемент документа.
*   `<head>`: Контейнер для метаданных.
*   `<meta>`: Метаданные (кодировка, viewport).
*   `<title>`: Заголовок страницы.
*   `<style>`: Стили CSS.
*   `<body>`: Основное содержимое страницы.
*   `<div>`: Контейнеры для элементов.
*   `<h1>`: Заголовок страницы.
*   `<p>`: Абзацы текста.
*   `<img>`: Изображение.
*   `<h3>`: Подзаголовки.
*   `<span>`: Строчный контейнер.
*    `{% for ... in ... %}`: Цикл Jinja.
*   `{{ ... }}`: Переменные Jinja.
