# Модуль `src.suppliers.hb.graber`

## Обзор

Модуль `src.suppliers.hb.graber` представляет собой класс, предназначенный для сбора данных о товарах с сайта `hb.co.il`. Он наследует функциональность от родительского класса `Graber` и предоставляет специализированную обработку полей товара, если это необходимо. Модуль также включает механизм для выполнения предварительных действий перед отправкой запроса к вебдрайверу через декоратор.

## Классы

### `Graber`

**Описание**: Класс для операций захвата данных о товарах с сайта `hb.co.il`.

**Наследует от**: `src.suppliers.graber.Graber`

**Атрибуты**:
- `supplier_prefix` (str): Префикс поставщика, устанавливается как 'hb'.

**Методы**:

#### `__init__`

**Описание**: Инициализирует класс `Graber`, устанавливает префикс поставщика и вызывает конструктор родительского класса. Также настраивает контекст для использования декораторов.

**Параметры**:
- `driver` (`src.webdriver.driver.Driver`): Экземпляр веб-драйвера для взаимодействия с сайтом.

**Возвращает**:
- `None`: Ничего не возвращает.
