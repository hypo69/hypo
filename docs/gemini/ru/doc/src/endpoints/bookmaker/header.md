# Модуль hypotez/src/endpoints/header.py

## Обзор

Данный модуль содержит функцию `set_project_root` для определения корневой директории проекта и переменные для хранения настроек проекта, версии, документации и других метаданных.  Модуль инициализирует переменные, связанные с проектом, на основе файла `settings.json` и `README.MD` в корневом каталоге проекта.

## Функции

### `set_project_root`

**Описание**: Функция определяет корневую директорию проекта, начиная с текущего файла. Она ищет указанные файлы или директории (pyproject.toml, requirements.txt, .git) в родительских каталогах до тех пор, пока не будет найден.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, по которым определяется корневой каталог проекта.

**Возвращает**:

- `Path`: Путь до корневой директории проекта, если она найдена, иначе – директория, в которой расположен текущий скрипт.

**Вызывает исключения**:

- Нет.


### `load_settings`

**Описание**: Несуществующая функция.


## Переменные

### `__root__`

**Описание**: Путь до корневой директории проекта, полученный с помощью функции `set_project_root`.

**Тип**: `Path`


### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json`.

**Тип**: `dict` | `None`


### `doc_str`

**Описание**: Строка с содержимым файла `README.MD`.

**Тип**: `str` | `None`


### `__project_name__`

**Описание**: Название проекта, полученное из файла `settings.json`.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта, полученная из файла `settings.json`.

**Тип**: `str`


### `__doc__`

**Описание**: Документация проекта, полученная из файла `README.MD`.

**Тип**: `str`


### `__details__`

**Описание**: Подробная информация о проекте (по умолчанию пустая строка).

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта, полученный из файла `settings.json`.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права проекта, полученные из файла `settings.json`.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка на поддержку разработчика через платформа Boosty.

**Тип**: `str`


**Обработка исключений**:

- Функции `load_settings`, и  `load_readme` обрабатывают потенциальные `FileNotFoundError` и `json.JSONDecodeError`,  игнорируя их,  если файл не найден или некорректно отформатирован.  Вместо `e`, используется `ex`,  как указано в инструкции.



## Использование

Модуль предназначен для инициализации переменных, связанных с проектом, для использования в других модулях. Переменные `__root__`, `__project_name__`,  `__version__`, и другие содержат метаданные о проекте.

```python
# Пример использования
from hypotez.src.endpoints import header

# Получаем корневой каталог проекта
project_root = header.__root__

# Доступ к переменным
project_name = header.__project_name__
project_version = header.__version__
```
