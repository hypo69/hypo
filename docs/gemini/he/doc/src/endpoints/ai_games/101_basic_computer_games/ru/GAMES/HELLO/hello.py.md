## פונקציות

### `print`

**Description**: מדפיסה את הטקסט "Привет! Добро пожаловать в игру Hello!" למסך.

**Parameters**:
- `objects` (object, optional): האובייקטים להדפסה. ברירת מחדל: `None`.
- `sep` (str, optional): המפריד בין האובייקטים. ברירת מחדל: `''`.
- `end` (str, optional): הסיום. ברירת מחדל: `'\n'`.
- `file` (object, optional): האובייקט אליו להדפיס. ברירת מחדל: `sys.stdout`.
- `flush` (bool, optional): האם לשטוף את התזרים. ברירת מחדל: `False`.

**Returns**:
- `None`: פונקציה זו אינה מחזירה ערך.

### `input`

**Description**: מבקשת מהמשתמש להזין את שמו ומחזירה אותו.

**Parameters**:
- `prompt` (str, optional): הודעת ההנחיה למשתמש. ברירת מחדל: `None`.

**Returns**:
- `str`: מחזירה את הקלט שהזין המשתמש.
