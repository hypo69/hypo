# Задача No 737
## Разделение числа на слагаемые с ограничением на делители

## Условия

Обозначим через $P(n)$ количество способов, которыми натуральное число $n$ может быть представлено как сумма целых положительных чисел, таких что каждый слагаемый является либо простым числом, либо равен 1.
Например, $P(4) = 6$ так как $4 = 1 + 1 + 1 + 1 = 1 + 1 + 2 = 1 + 2 + 1 = 2 + 1 + 1 = 1 + 3 = 3 + 1$.

Пусть $Q(n)$ - количество способов, которыми натуральное число $n$ может быть представлено как сумма целых положительных чисел, таких что каждый слагаемый является либо простым числом, либо числом, имеющим ровно два различных простых делителя, либо равен 1.
Например, $Q(10) = 53$

Обозначим через $S(n)$ сумму всех $Q(k)$ для $1 \le k \le n$.

Известно, что $S(10) = 213$.

Найдите $S(10^8) \pmod{10^9}$.
