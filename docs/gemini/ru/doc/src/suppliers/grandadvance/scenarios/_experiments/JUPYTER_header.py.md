# Модуль экспериментов GrandAdvance

## Обзор

Модуль содержит экспериментальный код для работы с поставщиками, продуктами и категориями в контексте проекта `hypotez`. 
Включает в себя импорты различных модулей, настройки путей и функцию для запуска поставщика.

## Подробней

Модуль предназначен для экспериментов и отладки функциональности, связанной с поставщиками, продуктами и категориями. 
Содержит импорты необходимых библиотек и модулей, а также функцию `start_supplier` для запуска поставщика с заданными параметрами.
Модуль использует различные утилиты для работы со строками, форматированием и нормализацией данных.
Также модуль взаимодействует с PrestaShop через класс `PrestaProduct`.

## Функции

### `start_supplier`

```python
def start_supplier(supplier_prefix: str = 'aliexpress', locale: str = 'en') -> Supplier:
    """ Старт поставщика """
```

**Назначение**: Запускает поставщика с заданными параметрами.

**Параметры**:
- `supplier_prefix` (str): Префикс поставщика. По умолчанию `'aliexpress'`.
- `locale` (str): Локаль поставщика. По умолчанию `'en'`.

**Возвращает**:
- `Supplier`: Объект поставщика, созданный с заданными параметрами.

**Как работает функция**:

1. **Определение параметров**: Функция принимает два параметра: `supplier_prefix` и `locale`, которые определяют префикс и локаль поставщика соответственно. Если параметры не указаны, используются значения по умолчанию: `'aliexpress'` и `'en'`.
2. **Создание словаря параметров**: Функция создает словарь `params`, содержащий параметры `supplier_prefix` и `locale`. Этот словарь используется для инициализации объекта `Supplier`.
3. **Создание объекта поставщика**: Функция создает и возвращает объект `Supplier`, используя параметры из словаря `params`.

```
        Определение параметров --> Создание словаря параметров --> Создание объекта поставщика
```

**Примеры**:

```python
supplier1 = start_supplier()
supplier2 = start_supplier(supplier_prefix='amazon', locale='de')
