# Модуль `header.py`

## Обзор

Модуль `header.py` предназначен для установки корневой директории проекта, загрузки настроек из файла `settings.json` и чтения документации из файла `README.MD`, а также для определения основных метаданных проекта, таких как имя проекта, версия, документация, автор, копирайт и сообщение для поддержки разработчика.

## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная с каталога текущего файла, ища вверх и останавливаясь в первом каталоге, содержащем любой из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple, optional): Имена файлов или каталогов для идентификации корня проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневому каталогу, если найден, иначе каталог, где расположен скрипт.

## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта.

### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json`. Может быть `None`, если файл не найден или произошла ошибка декодирования JSON.

### `doc_str`

**Описание**: Строка с содержимым документации, загруженная из файла `README.MD`. Может быть `None`, если файл не найден или произошла ошибка.

### `__project_name__`

**Описание**: Имя проекта. По умолчанию 'hypotez', если не найдено в настройках.

### `__version__`

**Описание**: Версия проекта. Пустая строка, если не найдено в настройках.

### `__doc__`

**Описание**: Документация проекта, загруженная из `README.MD`. Пустая строка, если файл не найден или произошла ошибка.

### `__details__`

**Описание**: Детали проекта. Изначально пустая строка.

### `__author__`

**Описание**: Автор проекта. Пустая строка, если не найдено в настройках.

### `__copyright__`

**Описание**: Копирайт проекта. Пустая строка, если не найдено в настройках.

### `__cofee__`

**Описание**: Сообщение с просьбой поддержать разработчика чашкой кофе. По умолчанию "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69", если не найдено в настройках.
