# Модуль `arguments`

## Обзор

Модуль `arguments` предназначен для обработки и валидации аргументов, используемых в API для работы с AliExpress. Он предоставляет функции для преобразования аргументов в нужный формат и проверки их допустимости. В частности, модуль содержит функции для преобразования списков в строки и извлечения идентификаторов продуктов из различных форматов.

## Подробней

Этот модуль обеспечивает стандартизацию входных данных, что упрощает дальнейшую обработку и взаимодействие с API AliExpress. Он используется для подготовки данных перед отправкой запросов к API, а также для обработки ответов от API. Он обрабатывает аргументы, такие как списки идентификаторов продуктов или другие параметры, которые могут быть переданы в виде строк или списков. Важность этого модуля заключается в обеспечении надежной и предсказуемой обработки аргументов, что снижает вероятность ошибок и повышает стабильность работы API.

## Функции

### `get_list_as_string`

```python
def get_list_as_string(value) -> str | None:
    """
    Преобразует переданное значение в строку, если это возможно.

    Args:
        value: Значение для преобразования.

    Returns:
        str | None: Строковое представление значения, если `value` является строкой или списком. Возвращает `None`, если `value` равно `None`.

    Raises:
        InvalidArgumentException: Если `value` не является строкой или списком.

    Example:
        >>> get_list_as_string(['a', 'b', 'c'])
        'a,b,c'
        >>> get_list_as_string('abc')
        'abc'
        >>> get_list_as_string(None) # doctest: +ELLIPSIS
    """
```

**Описание**: Преобразует входное значение в строку. Если значение является списком, элементы объединяются в строку, разделенную запятыми.

**Параметры**:
- `value`: Значение, которое необходимо преобразовать в строку.

**Возвращает**:
- `str | None`: Строковое представление значения, если это строка или список. Возвращает `None`, если входное значение равно `None`.

**Вызывает исключения**:
- `InvalidArgumentException`: Если входное значение не является строкой, списком или `None`.

**Примеры**:

```python
get_list_as_string(['a', 'b', 'c'])
```

Результат:

```
'a,b,c'
```

```python
get_list_as_string('abc')
```

Результат:

```
'abc'
```

```python
get_list_as_string(None)
```

Результат:

```
None
```

### `get_product_ids`

```python
def get_product_ids(values) -> list[str]:
    """
    Извлекает и возвращает список идентификаторов продуктов из переданного значения.

    Args:
        values: Значение, содержащее идентификаторы продуктов. Может быть строкой, разделенной запятыми, или списком.

    Returns:
        list[str]: Список идентификаторов продуктов.

    Raises:
        InvalidArgumentException: Если `values` не является строкой или списком.

    Example:
        >>> get_product_ids('123,456,789')
        ['123', '456', '789']
        >>> get_product_ids(['123', '456', '789'])
        ['123', '456', '789']
    """
```

**Описание**: Извлекает идентификаторы продуктов из входных данных, которые могут быть строкой (с разделителями-запятыми) или списком.

**Параметры**:
- `values`: Строка с идентификаторами, разделенными запятыми, или список идентификаторов продуктов.

**Возвращает**:
- `list[str]`: Список идентификаторов продуктов.

**Вызывает исключения**:
- `InvalidArgumentException`: Если `values` не является строкой или списком.

**Примеры**:

```python
get_product_ids('123,456,789')
```

Результат:

```
['123', '456', '789']
```

```python
get_product_ids(['123', '456', '789'])
```

Результат:

```
['123', '456', '789']
```
