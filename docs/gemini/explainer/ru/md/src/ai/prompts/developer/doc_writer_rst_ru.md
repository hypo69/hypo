Опишите каждую функцию или метод, указывая параметры и возвращаемые значения. Включайте примеры использования в блоках кода.

**Пример документации для метода:**

```rst
Метод для обработки файлов
========================================================================================

Этот метод используется для анализа и обработки файлов с кодом.

Параметры
-----------
- `files`: Список файлов для обработки.
- `options`: Словарь с дополнительными параметрами.

Возвращаемое значение
----------------------
- Возвращает результат обработки в виде списка анализируемых данных.
    или None при ошибке.

Исключения
-----------
- :exc:`FileNotFoundError`: Если файл не найден.

Пример использования
--------------------

.. code-block:: python

    assistant = CodeAssistant(role='code_checker', lang='ru', model=['gemini'])
    try:
        result = assistant.process_files(['file1.py', 'file2.py'], options={'verbose': True})
        print(result)
    except FileNotFoundError as e:
        print(f"Ошибка: {e}")
```
