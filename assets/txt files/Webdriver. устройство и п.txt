Webdriver. устройство и принцип работы.

Веб-драйвер разработан на базе библиотеки selenium, которая позволяет взаимодействовать напрямую HTML страницами.
Веб-драйвер запускает один из   браузеров: Edge, Firefox, Chrome, Opera etc.
Традиционно используется драйвер Firefox, который называется geckodgriver.exe и скачивается с гитхаба Firefox.
У меня он расположен в
 binnary/geckodgriver.exe. инициализации класса драйвер для него берутся параметры записанные в файле global_settings.ps
Он он предоставляет поставщику метод execute_locator посредством которого посылаются и понимаются данные от HTML страницы.

Подробно локатор описан в разделе локатор

Модуль веб-драйвер
Модуль веб-драйвер содержит в себе подмодули каждого конкретного веб-браузера. в каждом из них есть файл установок json где описаны особенности подключения. Там  же можно задать профиль пользователя.
