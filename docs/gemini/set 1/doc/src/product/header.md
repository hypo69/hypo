# Модуль hypotez/src/product/header.py

## Обзор

Этот модуль определяет корневой путь к проекту Hypotez. Он используется для построения правильных путей к файлам проекта, таких как `settings.json` и `README.MD`. Все импорты в других модулях проекта строятся относительно этого корневого пути.

## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная с текущего файла. Поиск происходит вверх по иерархии каталогов, пока не будет найден каталог, содержащий указанные маркерные файлы (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или каталогов, которые используются для определения корневого каталога проекта. По умолчанию используется кортеж `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Объект `Path` с путем к корневому каталогу проекта. Если корневой каталог не найден, возвращается путь к каталогу, где находится текущий скрипт.


**Вызывает исключения**:

- Нет.


## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта, полученный с помощью функции `set_project_root()`.

**Тип**: `Path`


### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json` в корневом каталоге.


**Тип**: `dict` или `None`

**Примечания**: Возможная ошибка `FileNotFoundError` или `json.JSONDecodeError` при чтении файла.


### `doc_str`

**Описание**: Строка с содержимым файла `README.MD` в корневом каталоге.


**Тип**: `str` или `None`

**Примечания**: Возможная ошибка `FileNotFoundError` или `json.JSONDecodeError` при чтении файла.



### `__project_name__`

**Описание**: Название проекта, полученное из настроек (`settings`). По умолчанию используется `'hypotez'`.


**Тип**: `str`

### `__version__`

**Описание**: Версия проекта, полученная из настроек (`settings`). По умолчанию используется пустая строка.


**Тип**: `str`

### `__doc__`

**Описание**: Документация проекта, полученная из файла `README.MD`. По умолчанию используется пустая строка.


**Тип**: `str`

### `__details__`

**Описание**: Подробная информация о проекте (по умолчанию пустая строка).


**Тип**: `str`

### `__author__`

**Описание**: Автор проекта, полученный из настроек (`settings`). По умолчанию используется пустая строка.


**Тип**: `str`

### `__copyright__`

**Описание**: Авторские права проекта, полученные из настроек (`settings`). По умолчанию используется пустая строка.


**Тип**: `str`

### `__cofee__`

**Описание**: Ссылка на помощь автору проекта (по умолчанию ссылка на Boosty).


**Тип**: `str`
