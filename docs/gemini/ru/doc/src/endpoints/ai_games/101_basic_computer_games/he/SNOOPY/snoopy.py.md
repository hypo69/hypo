# Игра SNOOPY

## Обзор

Этот модуль реализует простую игру-угадайку, в которой компьютер выбирает случайную букву из английского алфавита, а игрок должен её угадать. Игра продолжается до тех пор, пока игрок не угадает букву.

## Описание игры

Игра "SNOOPY" - это простая игра в угадывание, в которой компьютер выбирает случайную букву из списка 26 английских букв, а игрок должен угадать эту букву. Игра заканчивается, когда игрок правильно угадывает букву.

## Правила игры

1. Компьютер случайным образом выбирает одну букву из 26 букв английского алфавита (от A до Z).
2. Игрок вводит свою догадку буквы.
3. Компьютер проверяет, соответствует ли буква, введенная игроком, выбранной букве.
4. Если буквы совпадают, игра заканчивается и игрок побеждает.
5. Если буквы не совпадают, игрок продолжает угадывать до правильного предположения.

## Алгоритм игры

1. Определить список букв английского алфавита.
2. Выбрать случайную букву из списка букв.
3. Начать цикл "пока буква не угадана":
    3.1. Попросить игрока ввести догадку.
    3.2. Проверить, равна ли догадка выбранной букве:
        3.2.1. Если догадка верна, вывести сообщение о победе и завершить игру.
        3.2.2. В противном случае, перейти к следующей итерации цикла.
4. Завершить игру.

## Объяснение кода

1.  **Импорт модуля `random`**:
    -   `import random`: Импортируется модуль `random`, который используется для генерации случайных значений.

2.  **Определение списка букв**:
    -   `alphabet = ['A', 'B', 'C', ..., 'Z']`: Создается список, содержащий все буквы английского алфавита, позволяющий делать случайный выбор.

3.  **Выбор случайной буквы**:
    -   `targetLetter = random.choice(alphabet)`: Выбирается случайная буква из списка `alphabet` и сохраняется в переменной `targetLetter`, это значение, которое пользователь будет пытаться угадать.

4.  **Игровой цикл**:
    -   `while True:`: Бесконечный цикл, который продолжается до тех пор, пока пользователь не угадает правильную букву.

5.  **Получение ввода от пользователя**:
    -   `userGuess = input("נחש אות (A-Z): ").upper()`: Предлагает пользователю ввести букву и использует `.upper()`, чтобы преобразовать букву в верхний регистр, чтобы код игнорировал, является ли введенная буква большой или маленькой.

6.  **Проверка догадки**:
    -   `if userGuess == targetLetter:`: Сравнение догадки пользователя с загаданной буквой.
    -   `print("כל הכבוד! ניחשת נכון!")`: Вывод сообщения о победе, если догадка верна.
    -   `break`: Выход из цикла, завершение игры.
    -   `else: print("נסה שוב")`: Если догадка неверна, будет выведено "נסה שוב", и цикл продолжится.
