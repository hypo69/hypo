# Модуль `test_advertisement_scenarios.py`

## Обзор

Этот модуль содержит тесты для сценариев оценки и создания рекламных объявлений. Тесты проверяют работу агентов, их способность анализировать рекламные объявления и создавать рекламные кампании для аренды квартиры.

## Функции

### `test_ad_evaluation_scenario`

**Описание**:  Функция тестирует сценарий оценки рекламных объявлений.  Она генерирует несколько рекламных объявлений, описывает контекст оценки и запрашивает агентов выбрать наилучшее объявление, обосновав свой выбор.

**Параметры**:

- `setup`: Набор настроек для запуска сценария.

**Возвращает**:

-  Не имеет возвращаемого значения.

**Вызывает исключения**:

-  `AssertionError`: Если результаты оценки не соответствуют ожидаемым значениям (например, если не выбран ни один вариант или не предоставлено обоснование выбора).


### `test_ad_creation_scenario`

**Описание**:  Функция тестирует сценарий создания рекламной кампании для аренды квартиры.  Она предоставляет контекст фокус-группы, описание квартиры и запрос на разработку стратегии продвижения.

**Параметры**:

- `setup`: Набор настроек для запуска сценария.
- `focus_group_world`: Объект, представляющий среду фокус-группы.

**Возвращает**:

-  Не имеет возвращаемого значения.

**Вызывает исключения**:

-  `AssertionError`:  Если результат не соответствует ожиданиям (например, отсутствуют идеи для рекламы квартиры, или не выполняется проверка полученных данных).


### `test_consumer_profiling_scenario`

**Описание**: Функция тестирует сценарий профилирования потребителей. Она создает и интервьюирует группу потребителей, чтобы определить их потенциальный интерес к определенному продукту (бутилированному гаспачо).

**Параметры**:

- `setup`: Набор настроек для запуска сценария.

**Возвращает**:

-  Не имеет возвращаемого значения.

**Вызывает исключения**:

- `AssertionError`: Если не создан ожидаемый файл кэширования.


##  Дополнительные пояснения

- Функции используют pytest для запуска тестов.
- Функции `test_ad_evaluation_scenario` и `test_ad_creation_scenario` тестируют взаимодействие агентов с текстом, выполняя оценку и создание рекламных объявлений.
- Функция `test_consumer_profiling_scenario` тестирует взаимодействие агентов с пользователями, выявляя их интересы.
- В модуле используется  `ResultsExtractor`,  позволяющий извлекать данные из ответов агентов.
- В модуле используется `control.begin` и `control.end` для управления процессом моделирования.
-  Модуль использует файлы кэширования ("test_consumer_profiling_scenario.cache.json") для сохранения и загрузки состояния во время тестов.
