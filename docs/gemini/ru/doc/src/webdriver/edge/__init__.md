# Модуль hypotez/src/webdriver/edge/__init__.py

## Обзор

Данный модуль предоставляет инициализацию для драйвера Edge.

## Переменные

### `MODE`

**Описание**: Переменная, определяющая режим работы (например, 'dev', 'prod'). В данном случае используется значение 'dev'.

**Значение**: `'dev'`

## Импорты

### `from .edge import Edge`

**Описание**: Импортирует класс `Edge` из модуля `edge.py` внутри текущей директории.


## Модуль

**Описание**: Модуль содержит инициализацию для драйвера браузера Edge.
