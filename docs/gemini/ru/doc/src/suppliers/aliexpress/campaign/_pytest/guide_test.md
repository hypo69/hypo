# Руководство для Тестера - Модуль Подготовки Рекламных Кампаний AliExpress

## Обзор

Данный документ предоставляет руководство для тестеров, занимающихся проверкой модуля, отвечающего за подготовку материалов для рекламных кампаний на платформе AliExpress.  Документ описывает основные файлы модуля, их функциональность и инструкции по тестированию.

## Классы

### `AliCampaignEditor`

**Описание**: Класс `AliCampaignEditor`, наследующий от `AliPromoCampaign`, отвечает за управление рекламной кампанией.

**Методы**:

* **`__init__`**: Инициализация объекта `AliCampaignEditor`.  Подробности реализации требуют отдельной документации к исходному коду.

## Функции

### `update_category`

**Описание**: Функция обновляет данные категории в JSON-файле.

**Параметры**:
- `category_data` (dict): Словарь с данными обновляемой категории.
- `file_path` (str): Путь к JSON-файлу.

**Возвращает**:
- bool: `True`, если обновление прошло успешно, иначе `False`.

**Вызывает исключения**:
- `FileNotFoundError`: Если файл не найден.
- `json.JSONDecodeError`: Если содержимое файла не является корректным JSON.
- `ValueError`: Если данные категории имеют неправильный формат.


### `process_campaign_category`

**Описание**: Функция обрабатывает конкретную категорию в рамках рекламной кампании.

**Параметры**:
- `category_data` (dict): Данные категории.
- `campaign_id` (int): Идентификатор кампании.

**Возвращает**:
- dict | None: Словарь с результатами обработки категории или `None` при ошибке.

**Вызывает исключения**:
- `ValueError`: Если данные категории некорректны.
- `Exception`: Другие возможные ошибки при обработке категории.


### `process_campaign`

**Описание**: Функция обрабатывает всю кампанию по всем категориям.

**Параметры**:
- `campaign_data` (dict): Данные кампании, включая список категорий.

**Возвращает**:
- list: Список результатов обработки каждой категории.

**Вызывает исключения**:
- `ValueError`: Если данные кампании некорректны.
- `Exception`: Другие возможные ошибки при обработке кампании.


### `main`

**Описание**: Асинхронная функция для обработки всей кампании.

**Параметры**:
- `campaign_id` (int): Идентификатор обрабатываемой кампании.

**Возвращает**:
- None

**Вызывает исключения**:
- `Exception`: Другие возможные ошибки при асинхронной обработке.


## Тесты (модуль `test_campaign_integration.py`)

### `test_update_category_success`

**Описание**: Проверка успешного обновления категории.

### `test_update_category_failure`

**Описание**: Проверка обработки ошибки при обновлении категории.

### `test_process_campaign_category_success`

**Описание**: Проверка успешной обработки категории.

### `test_process_campaign_category_failure`

**Описание**: Проверка обработки ошибки при обработке категории.

### `test_process_campaign`

**Описание**: Проверка обработки всех категорий в кампании.

### `test_main`

**Описание**: Проверка основного сценария выполнения кампании асинхронно.


## Инструкции по тестированию

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Запуск тестов

```bash
pytest test_campaign_integration.py
```
