# Модуль `hypotez/src/suppliers/aliexpress/api/models/languages.py`

## Обзор

Данный модуль определяет перечисление (enum) поддерживаемых языков для API AliExpress.  Он предоставляет константы для различных языковых кодов.

## Классы

### `Language`

**Описание**: Класс, представляющий перечисление языков.  Содержит различные константы, представляющие языки.

**Атрибуты**:

- `EN`: Код языка английский.
- `RU`: Код языка русский.
- `PT`: Код языка португальский.
- `ES`: Код языка испанский.
- `FR`: Код языка французский.
- `ID`: Код языка индонезийский.
- `IT`: Код языка итальянский.
- `TH`: Код языка тайский.
- `JA`: Код языка японский.
- `AR`: Код языка арабский.
- `VI`: Код языка вьетнамский.
- `TR`: Код языка турецкий.
- `DE`: Код языка немецкий.
- `HE`: Код языка иврит.
- `KO`: Код языка корейский.
- `NL`: Код языка нидерландский.
- `PL`: Код языка польский.
- `MX`: Код языка мексиканский.  (Возможно, не стандартный язык, уточнить)
- `CL`: Код языка чилийский. (Возможно, не стандартный язык, уточнить)
- `IW`: Код языка иврит (возможно дублирование). (Уточнить)
- `IN`: Код языка индийский. (Возможно, не стандартный язык, уточнить)
