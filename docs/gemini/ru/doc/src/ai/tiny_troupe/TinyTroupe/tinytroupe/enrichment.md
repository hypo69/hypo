# Модуль enrichment.py

## Обзор

Этот модуль предоставляет класс `TinyEnricher`, предназначенный для обогащения контента с использованием LLM (Large Language Model).  Класс использует шаблоны `mustache` для формирования запросов к LLM и обрабатывает полученные результаты.

## Классы

### `TinyEnricher`

**Описание**: Класс `TinyEnricher` наследуется от `JsonSerializableRegistry` и отвечает за обогащение контента. Он хранит информацию о контексте и использует LLM для генерации результатов обогащения.

**Атрибуты**:

- `use_past_results_in_context`: Булевый параметр, определяющий, использовать ли предыдущие результаты в контексте.
- `context_cache`: Список, содержащий историю контекста.

**Методы**:

#### `__init__`

**Описание**: Конструктор класса.

**Параметры**:

- `use_past_results_in_context` (bool):  Указывает, использовать ли предыдущие результаты в контексте. По умолчанию `False`.

**Возвращает**:
- None


#### `enrich_content`

**Описание**: Метод обогащения контента с использованием LLM.

**Параметры**:

- `requirements` (str): Требования к обогащению контента.
- `content` (str): Контент, который нужно обогатить.
- `content_type` (str, optional): Тип контента. По умолчанию `None`.
- `context_info` (str, optional): Дополнительная информация для контекста. По умолчанию `""`.
- `context_cache` (list, optional): Список предыдущих результатов. По умолчанию `None`.
- `verbose` (bool, optional): Флаг для печати отладочной информации. По умолчанию `False`.


**Возвращает**:
- str | None: Результат обогащения (строка кода) или `None`, если запрос к LLM не удался.

**Вызывает исключения**:
- Возможны исключения, связанные с использованием `openai_utils` или `utils`.
