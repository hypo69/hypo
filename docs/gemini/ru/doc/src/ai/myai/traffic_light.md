# Модуль traffic_light

## Обзор

Этот модуль содержит константы и, возможно, функции, относящиеся к управлению светофором.  Подробное описание функциональности ожидается в оставшейся части документации. Ссылка на пример реализации алгоритмов управления светофором: [https://habr.com/ru/articles/849414/](https://habr.com/ru/articles/849414/).


## Константы

### `MODE`

**Описание**: Переменная, определяющая режим работы модуля. В данном случае, хранит строку 'dev'.


**Значение**: `'dev'`
