# Модуль hypotez/src/suppliers/chat_gpt/scenarios/header.py

## Обзор

Данный модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json` и документации из `README.MD`.  Он также содержит переменные, содержащие информацию о проекте, например, имя, версию, описание и автора.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла.  Поиск происходит вверх по иерархии директорий до тех пор, пока не будет найдена директория, содержащая один из файлов маркеров (pyproject.toml, requirements.txt, .git).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, которые используются для определения корневой директории проекта. По умолчанию (`pyproject.toml`, `requirements.txt`, `.git`).

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена. Иначе возвращает директорию, где находится текущий файл.

**Вызывает исключения**:

- Нет.

## Переменные

### `MODE`

**Описание**: Переменная, определяющая режим работы (в данном случае, 'dev').

### `__root__`

**Описание**:  Переменная, содержащая путь к корневой директории проекта. Она инициализируется вызовом функции `set_project_root`.

### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `src/settings.json`.

**Возвращаемое значение**:

- dict: Словарь с настройками проекта.


**Обработка ошибок**:
- `FileNotFoundError`: Возникает, если файл `settings.json` не найден.
- `json.JSONDecodeError`: Возникает, если содержимое `settings.json` не является валидным JSON.

### `doc_str`

**Описание**: Строка с текстовой документацией (читается из файла `README.MD`).

**Возвращаемое значение**:

- str: Текст документации из файла `README.MD`
- None: Если файл не найден или не удалось считать.

**Обработка ошибок**:
- `FileNotFoundError`: Возникает, если файл `README.MD` не найден.
- `json.JSONDecodeError`: Возникает, если содержимое `README.MD` не является валидным.


### `__project_name__`

**Описание**: Имя проекта, полученное из `settings` или установленное значение по умолчанию `'hypotez'`.

**Возвращаемое значение**:

- str: Строковое имя проекта.

**Обработка ошибок**:
- Нет.

### `__version__`

**Описание**: Версия проекта, полученная из `settings` или значение по умолчанию ''.

**Возвращаемое значение**:

- str: Строковая версия проекта.

**Обработка ошибок**:
- Нет.


### `__doc__`

**Описание**: Текст документации проекта, считываемый из файла `README.MD`.

**Возвращаемое значение**:

- str: Текст документации.

**Обработка ошибок**:
- Нет.


### `__details__`


**Описание**: Подробная информация о проекте.

**Возвращаемое значение**:

- str: Подробности проекта.


### `__author__`


**Описание**: Автор проекта.

**Возвращаемое значение**:

- str: Имя автора проекта.


### `__copyright__`

**Описание**: Авторские права на проект.

**Возвращаемое значение**:

- str: Авторские права.


### `__cofee__`


**Описание**: Ссылка для поддержки разработчика.

**Возвращаемое значение**:

- str: Ссылка для поддержки разработчика.
