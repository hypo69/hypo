# Модуль `start_posting_my_groups.py`

## Обзор

Этот модуль отвечает за запуск рекламных кампаний в группах Facebook. Он использует `FacebookPromoter` для публикации объявлений в указанных группах, основываясь на конфигурационных файлах. Модуль также обрабатывает прерывание кампании через `KeyboardInterrupt`.

## Импорты

В этом разделе описаны все импортированные модули и классы, используемые в данном файле.

- `header`: Пользовательский модуль. Его назначение в данном коде неясно.
- `copy`: Используется для создания копии списка `campaigns`.
- `src.webdriver.driver.Driver`, `src.webdriver.driver.Chrome`: Классы для управления веб-драйвером (Chrome).
- `src.endpoints.advertisement.facebook.promoter.FacebookPromoter`: Класс, отвечающий за запуск рекламных кампаний в Facebook.
- `src.logger.logger.logger`: Объект логгера для записи информации о выполнении программы.

## Глобальные переменные

В этом разделе описаны глобальные переменные, используемые в модуле.

- `d`: Экземпляр класса `Driver` с использованием драйвера Chrome, предназначенный для управления браузером.
- `filenames`: Список строк, представляющих имена файлов с данными о группах (в данном случае, 'my_managed_groups.json').
- `campaigns`: Список строк, представляющих названия рекламных кампаний.
- `promoter`: Экземпляр класса `FacebookPromoter`, который будет использоваться для проведения рекламных кампаний.

## Обработка исключений

В этом разделе описан блок обработки исключений.
-  `KeyboardInterrupt`: Обработка прерывания программы через `Ctrl+C` пользователем. В этом случае программа выведет сообщение о том, что кампания была прервана.

```python
try:
    while True:
        
        promoter.run_campaigns(campaigns = copy.copy(campaigns), group_file_paths = filenames)
        ...

        
except KeyboardInterrupt:
    logger.info("Campaign promotion interrupted.")
```
