# Модуль подготовки всех кампаний AliExpress

## Обзор

Этот модуль предназначен для запуска процесса подготовки всех рекламных кампаний на AliExpress. Он импортирует необходимые модули и функции для выполнения этой задачи.

## Подробнее

Модуль `prepare_all_campaigns.py` является отправной точкой для запуска процесса подготовки рекламных кампаний AliExpress. Он импортирует модуль `process_all_campaigns` и вызывает функцию `process_all_campaigns()`, которая содержит основную логику обработки кампаний. Расположение модуля в структуре проекта указывает на его роль в организации и подготовке рекламных кампаний поставщика AliExpress.
