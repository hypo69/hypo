# Модуль revai

## Обзор

Данный модуль (`src.ai.revai`) предоставляет базовые функции и переменные для работы с системой.

## Переменные

### `MODE`

**Описание**: Переменная, хранящая режим работы системы.

**Значение**:  `'dev'`

##  Документация отсутствует

В файле отсутствуют классы и функции для документации.
