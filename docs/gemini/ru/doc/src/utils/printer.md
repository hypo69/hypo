# Модуль `hypotez/src/utils/printer.py`

## Обзор

Этот модуль предоставляет функции для красивой печати данных с возможностью стилизации текста, включая цвет, фон и шрифт.  Он предназначен для удобной выдачи информации в читаемом формате в консоли.

## Функции

### `_color_text`

**Описание**: Функция применяет стили цвета, фона и шрифта к тексту. Использует ANSI escape коды для изменения визуального отображения.

**Параметры**:
- `text` (str): Текст, к которому необходимо применить стили.
- `text_color` (str, опционально): Цвет текста. По умолчанию пустая строка (без цвета).  Доступные цвета:  `red`, `green`, `blue`, `yellow`, `white`, `cyan`, `magenta`, `light_gray`, `dark_gray`, `light_red`, `light_green`, `light_blue`, `light_yellow`. (см. `TEXT_COLORS`)
- `bg_color` (str, опционально): Цвет фона. По умолчанию пустая строка (без фона). Доступные цвета фона: `bg_red`, `bg_green`, `bg_blue`, `bg_yellow`, `bg_white`, `bg_cyan`, `bg_magenta`, `bg_light_gray`, `bg_dark_gray`, `bg_light_red`, `bg_light_green`, `bg_light_blue`, `bg_light_yellow`. (см. `BG_COLORS`)
- `font_style` (str, опционально): Стиль шрифта. По умолчанию пустая строка (без стиля). Доступные стили: `bold`, `underline`. (см. `FONT_STYLES`)

**Возвращает**:
- `str`: Текст со применёнными стилями.

**Пример**:
```python
>>> _color_text("Hello, World!", text_color="green", font_style="bold")
'\033[1m\033[32mHello, World!\033[0m'
```


### `pprint`

**Описание**: Функция форматирует и печатает данные с опциональными стилями цвета, фона и шрифта. Поддерживает различные типы данных, включая словари, списки, строки и пути к файлам.

**Параметры**:
- `print_data` (Any, опционально): Данные для печати. Может быть `None`, `dict`, `list`, `str` или `Path`.
- `text_color` (str, опционально): Цвет текста. По умолчанию "white".
- `bg_color` (str, опционально): Цвет фона. По умолчанию пустая строка.
- `font_style` (str, опционально): Стиль шрифта. По умолчанию пустая строка.

**Возвращает**:
- `None`

**Вызывает исключения**:
- `Exception`: Возникает в случае ошибок при обработке данных или их печати.


**Пример**:
```python
>>> pprint({"name": "Alice", "age": 30}, text_color="green")
# Вывод:
#  \033[32m{\n    "name": "Alice",\n    "age": 30\n}\033[0m

>>> pprint(["apple", "banana", "cherry"], text_color="blue", font_style="bold")
# Вывод (на нескольких строках):
# \033[34m\033[1mapple\033[0m
# \033[34m\033[1mbanana\033[0m
# \033[34m\033[1mcherry\033[0m

```


## Константы

`TEXT_COLORS`, `BG_COLORS`, `FONT_STYLES`, `RESET`:  Хранят значения ANSI escape кодов для управления стилями текста.
