# Документация для `cdata_categories_gaming_desktops.json`

## Обзор

Файл `cdata_categories_gaming_desktops.json` содержит JSON-структуру, описывающую сценарии для игровых настольных компьютеров. Каждый сценарий включает информацию о бренде, URL-адресе, состоянии чекбокса, активности, состоянии товара и категориях PrestaShop. Этот файл используется для определения конфигураций и категорий товаров при парсинге данных с сайта C-Data.

## Структура JSON

### Ключ `scenarios`

Ключ `scenarios` содержит словарь, где каждый ключ является идентификатором сценария (например, `"GAMING DESKTOP ASUS I5"`), а значение - это словарь с информацией о данном сценарии.

### Сценарий (например, `GAMING DESKTOP ASUS I5`)

Каждый сценарий представляет собой словарь со следующими ключами:

#### `brand`

**Описание**:
Бренд игрового настольного компьютера.

**Тип**:
`str`

**Пример**:
`"ASUS"`, `"DELL"`

#### `url`

**Описание**:
URL-адрес страницы с товарами данного сценария на сайте C-Data.

**Тип**:
`str`

**Пример**:
`"https://reseller.c-data.co.il/%D7%9E%D7%97%D7%A9%D7%91%D7%99%D7%9D-%D7%A0%D7%99%D7%99%D7%97%D7%99%D7%9D#/specFilters=227m!#-!4634&manFilters=10"`,
`"-------------------------------------GAMING DESKTOP DELL AMD-----------------------------------"`
(может быть строкой-заглушкой)

#### `checkbox`

**Описание**:
Состояние чекбокса, указывающее, выбран ли данный сценарий.

**Тип**:
`bool`

**Пример**:
`false`

#### `active`

**Описание**:
Состояние активности сценария.

**Тип**:
`bool`

**Пример**:
`true`

#### `condition`

**Описание**:
Состояние товара (например, "новый").

**Тип**:
`str`

**Пример**:
`"new"`

#### `presta_categories`

**Описание**:
Строка, содержащая идентификаторы категорий PrestaShop, к которым относится товар. Идентификаторы разделены запятыми.

**Тип**:
`str`

**Пример**:
`"91,89,231,236,242"`
