# Модуль hypotez/src/gui/header.py

## Обзор

Данный модуль отвечает за определение корневого пути к проекту `hypotez`. Он находит директорию, содержащую файлы `pyproject.toml`, `requirements.txt` или `.git` и добавляет эту директорию в системный путь `sys.path`.  Также загружает настройки из файла `settings.json` и документацию из `README.MD`, если они существуют.  Модуль содержит глобальные переменные, хранящие информацию о проекте, такие как имя, версия, описание, автор, и т.д.

## Функции

### `set_project_root`

**Описание**:  Находит корневую директорию проекта, начиная от текущей директории. Поиск идет вверх по иерархии директорий, до тех пор пока не будет найдена директория, содержащая один из заданных файлов (или директорий).

**Параметры**:
- `marker_files` (tuple): Кортеж из строк, представляющих имена файлов или директорий, которые служат маркерами корневой директории проекта. По умолчанию включает `pyproject.toml`, `requirements.txt` и `.git`.

**Возвращает**:
- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращает путь к директории, из которой был вызван скрипт.

**Вызывает исключения**:
- Нет


## Глобальные переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный функцией `set_project_root()`.

**Тип**: `Path`


### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженные из файла `settings.json`.

**Тип**: `dict` | `None`

### `doc_str`

**Описание**: Строка, содержащая документацию, загруженную из файла `README.MD`.

**Тип**: `str` | `None`


### `__project_name__`

**Описание**: Имя проекта. Значение берется из настроек (`settings`) или имеет значение по умолчанию `'hypotez'`, если настройки не загружены.

**Тип**: `str`

### `__version__`

**Описание**: Версия проекта. Значение берется из настроек (`settings`) или имеет значение по умолчанию `''`, если настройки не загружены.

**Тип**: `str`

### `__doc__`

**Описание**: Документация проекта, полученная из `README.MD` или пустая строка, если файл не найден.

**Тип**: `str`

### `__details__`

**Описание**: Дополнительная информация о проекте (по умолчанию пустая строка).

**Тип**: `str`

### `__author__`

**Описание**: Автор проекта. Значение берется из настроек (`settings`) или имеет значение по умолчанию `''`, если настройки не загружены.

**Тип**: `str`

### `__copyright__`

**Описание**: Авторские права на проект. Значение берется из настроек (`settings`) или имеет значение по умолчанию `''`, если настройки не загружены.

**Тип**: `str`

### `__cofee__`

**Описание**: Ссылка на страницу сбора средств для поддержки разработчика. Значение берется из настроек (`settings`) или имеет значение по умолчанию, содержащее ссылку на Boosty.

**Тип**: `str`
