# Документация модуля `main.py`

## Обзор

Модуль `main.py` содержит основной код для запуска Telegram-бота Казарёнова. Он обрабатывает аргументы командной строки, загружает настройки из JSON-файла (если указано) и запускает бота в режиме `test` или `prod`.

## Функции

### `parse_args`

**Описание**: Парсит аргументы командной строки для запуска бота.

**Параметры**:
- Нет

**Возвращает**:
- `dict`: Словарь с параметрами запуска, включая путь к файлу настроек и режим работы бота.

### `main`

**Описание**: Главная функция, запускающая бота Казарёнова с параметрами, полученными из командной строки или файла настроек.

**Параметры**:
- Нет

**Возвращает**:
- Нет

**Вызывает исключения**:
- `Exception`: Если возникает ошибка при запуске бота, то она логируется.
