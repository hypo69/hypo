# Модуль `src.webdriver.firefox`

## Обзор

Модуль `src.webdriver.firefox` предназначен для работы с веб-браузером Firefox. Он содержит функциональность для управления Firefox через WebDriver.

## Содержание

- [Импорты](#импорты)

## Импорты

### `from .firefox import Firefox`

Импортирует класс `Firefox` из модуля `firefox`, который предоставляет функциональность для управления браузером Firefox через WebDriver.
