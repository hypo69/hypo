# Модуль для сбора данных о товарах с AliExpress (aliexpress/graber.py)

## Обзор

Модуль предназначен для сбора данных о товарах с веб-сайта AliExpress. Он содержит класс `Graber`, который наследуется от базового класса `src.suppliers.graber.Graber` и предоставляет методы для обработки различных полей товара на странице.

## Подробнее

Этот модуль является частью системы для автоматизированного сбора данных о товарах с различных онлайн-платформ. Класс `Graber` специализируется на сборе и обработке данных с AliExpress, используя веб-драйвер для взаимодействия с сайтом. Он позволяет настраивать и расширять логику сбора данных, переопределяя методы обработки полей товара.

## Классы

### `Graber`

**Описание**: Класс для сбора данных о товарах с AliExpress.

**Как работает класс**:
Класс `Graber` наследуется от класса `src.suppliers.graber.Graber`. Он инициализируется с использованием экземпляра веб-драйвера и индекса языка. В конструкторе устанавливается префикс поставщика (`aliexpress`) и вызывается конструктор родительского класса. Также устанавливается значение локатора для декоратора в `None`, что позволяет при необходимости выполнять действия по закрытию всплывающих окон перед выполнением основных операций.

**Методы**:
- `__init__`: Инициализация класса.

**Параметры**:
- `supplier_prefix` (str): Префикс поставщика (aliexpress).
- `driver` (Driver): Экземпляр веб-драйвера.
- `lang_index` (int): Индекс языка.

#### `__init__`

```python
def __init__(self, driver: Driver, lang_index:int):
    """
    Инициализация класса сбора полей товара.

    :param driver: Экземпляр веб-драйвера для взаимодействия с браузером.
    :type driver: Driver
    """
    self.supplier_prefix = 'aliexpress'
    # вызов конструктора родительского класса
    super().__init__(supplier_prefix=self.supplier_prefix, driver=driver, lang_index=lang_index)

    # устанавливает значение локатора для декоратора в `None`
    # если будет установленно значение - то оно выполнится в декораторе `@close_pop_up`
    Context.locator_for_decorator = None
```

**Описание**: Инициализирует экземпляр класса `Graber`.

**Как работает функция**:
Конструктор класса `Graber` принимает экземпляр веб-драйвера `driver` и индекс языка `lang_index`. Он устанавливает значение `supplier_prefix` равным `'aliexpress'`, затем вызывает конструктор родительского класса `Graber` (из модуля `src.suppliers.graber`) с переданными параметрами. В завершение устанавливает атрибут `Context.locator_for_decorator` в `None`.

**Параметры**:
- `driver` (Driver): Экземпляр веб-драйвера для взаимодействия с браузером.
- `lang_index` (int): Индекс языка.

**Примеры**:
```python
from src.webdriver.driver import Driver
from src.suppliers.aliexpress.graber import Graber

# Пример создания экземпляра класса Graber
driver_instance = Driver()
graber_instance = Graber(driver=driver_instance, lang_index=0)
```
