# Задача No 504
## Углы решетки

## Условия
Пусть $T(a, b, c, d)$ обозначает четырехугольник с вершинами в точках $(0, 0)$, $(a, 0)$, $(b, c)$ и $(0, d)$, где $a, b, c$ и $d$ являются целыми числами такими, что $1 \leq a, b, c, d \leq m$.
Пусть $N(a, b, c, d)$ будет количеством точек с целыми координатами, лежащих строго внутри $T(a, b, c, d)$.

Например, $N(4, 2, 1, 3) = 4$.

Для $m=4$, количество четырехугольников $T(a, b, c, d)$, для которых $N(a, b, c, d)$ является идеальным квадратом, равно 160.

Найдите количество четырехугольников $T(a, b, c, d)$, для которых $N(a, b, c, d)$ является идеальным квадратом, при $1 \leq a, b, c, d \leq 100$.
