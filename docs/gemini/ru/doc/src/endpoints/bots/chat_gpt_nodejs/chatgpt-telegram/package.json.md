# Описание файла `package.json`

## Обзор

Файл `package.json` используется для управления зависимостями и скриптами проекта `chatgpt-telegram`. Он содержит метаданные о проекте, такие как имя, версия, описание, а также информацию о зависимостях, используемых в проекте, и скрипты для запуска и разработки.

## Структура файла

### Основные поля
- `name`: `"chatgpt-telegram"` - Название проекта.
- `version`: `"1.0.0"` - Версия проекта.
- `description`: `""` - Описание проекта.
- `main`: `"index.js"` - Точка входа приложения.
- `keywords`: `[]` - Массив ключевых слов, связанных с проектом.
- `author`: `""` - Автор проекта.
- `license`: `"ISC"` - Лицензия проекта.
- `type`: `"module"` -  Тип проекта, используется для работы с ES модулями.

### `devDependencies`

Содержит список зависимостей для разработки проекта:
- `cross-env`: `"^7.0.3"` - Используется для установки переменных окружения кроссплатформенно.
- `nodemon`: `"^2.0.22"` - Используется для перезапуска сервера при изменениях кода.

### `scripts`

Содержит скрипты для запуска и управления проектом:
- `start`: `"cross-env NODE_ENV=production node ./src/main.js"` - Скрипт для запуска приложения в production окружении.
- `dev`: `"cross-env NODE_ENV=development nodemon ./src/main.js"` - Скрипт для запуска приложения в режиме разработки с помощью `nodemon`.

### `dependencies`

Содержит список зависимостей, необходимых для работы приложения:
- `@ffmpeg-installer/ffmpeg`: `"^1.1.0"` - Установщик `ffmpeg`.
- `axios`: `"^1.4.0"` - HTTP клиент на основе Promise.
- `config`: `"^3.3.9"` - Для работы с конфигурациями проекта.
- `fluent-ffmpeg`: `"^2.1.2"` - Node.js API для `ffmpeg`.
- `openai`: `"^3.2.1"` - Библиотека для взаимодействия с OpenAI API.
- `telegraf`: `"^4.12.2"` - Telegram Bot API для Node.js.

## Зависимости
В этом разделе описаны зависимости, которые используются в проекте:

### Зависимости разработки
- `cross-env`: Используется для установки переменных окружения кроссплатформенно.
- `nodemon`: Используется для перезапуска сервера при изменениях кода в режиме разработки.

### Основные зависимости
- `@ffmpeg-installer/ffmpeg`: Предоставляет установщик для `ffmpeg`.
- `axios`: Используется для выполнения HTTP-запросов.
- `config`: Обеспечивает управление конфигурацией приложения.
- `fluent-ffmpeg`: Предоставляет Node.js интерфейс для работы с `ffmpeg`.
- `openai`: Клиентская библиотека для работы с OpenAI API.
- `telegraf`: Библиотека для создания Telegram ботов.

## Скрипты

### `start`
- **Описание**: Скрипт запускает приложение в production-окружении.
- **Команда**: `cross-env NODE_ENV=production node ./src/main.js`
- **Использование**: Устанавливает переменную окружения `NODE_ENV` в `production` и запускает основной файл приложения (`./src/main.js`) с помощью Node.js.

### `dev`
- **Описание**: Скрипт запускает приложение в режиме разработки.
- **Команда**: `cross-env NODE_ENV=development nodemon ./src/main.js`
- **Использование**: Устанавливает переменную окружения `NODE_ENV` в `development` и запускает основной файл приложения с помощью `nodemon`, который перезапускает приложение при изменениях файлов.
