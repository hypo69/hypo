# Задача No 260
## Каменный треугольник
## Условия
Имеются три кучи камней, содержащих a, b и c камней соответственно. На каждом ходе игрок выбирает две кучи и убирает из каждой по одному камню. Игрок выигрывает, когда какая-либо куча становится пустой. 

Начальные количества камней (a, b, c) могут быть представлены в виде точки в трехмерном пространстве.

Пусть S(n) будет количеством выигрышных позиций (a, b, c) таких, что 0 ≤ a, b, c ≤ n.

Известно, что S(10) = 118, S(100) = 71496 и S(1000) = 50847414.

Найдите S(1000000).
