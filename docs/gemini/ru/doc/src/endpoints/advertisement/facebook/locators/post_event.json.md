# Локаторы для создания событий в Facebook

## Обзор

Этот файл содержит JSON-объект с локаторами элементов веб-страницы для создания событий в Facebook. Каждый локатор описывает конкретный элемент, необходимый для автоматизации процесса создания событий.

## Описание локаторов

### `event_title`

**Описание**: Локатор для поля ввода названия мероприятия.

**Параметры**:
- `attribute` (null): Атрибут элемента (не используется).
- `by` (str): Способ поиска элемента ("XPATH").
- `selector` (str): XPATH-селектор элемента ("//label[@aria-label = 'Название мероприятия']//input").
- `if_list` (str): Указывает, что нужно взять первый элемент из списка, если таких элементов несколько("first").
- `use_mouse` (bool): Указывает, нужно ли использовать мышь для взаимодействия с элементом(false).
- `timeout` (int): Таймаут для ожидания элемента(0).
- `timeout_for_event` (str): Тип таймаута для события("presence_of_element_located").
- `event` (str): Событие, которое нужно выполнить с элементом("click();%EXTERNAL_MESSAGE%").
- `mandatory` (bool): Указывает, является ли элемент обязательным для взаимодействия(true).
- `locator_description` (str): Описание локатора("поле ввода названия сообщения. При переходе по специально сконструированной ссылке откроется сразу. См код сценария").
