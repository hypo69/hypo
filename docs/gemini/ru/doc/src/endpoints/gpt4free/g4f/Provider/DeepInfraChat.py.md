# Модуль DeepInfraChat

## Обзор

Модуль `DeepInfraChat.py` предназначен для взаимодействия с платформой DeepInfra Chat через API OpenAI. Он определяет класс `DeepInfraChat`, который наследуется от класса `OpenaiTemplate` и предоставляет конфигурацию для использования различных моделей, включая языковые и визуальные модели. Модуль также задает параметры подключения к API DeepInfra.

## Подробней

Этот модуль является частью проекта `hypotez` и предназначен для интеграции с различными AI-моделями, предоставляемыми через DeepInfra. Он определяет URL-адрес, базовый URL API и список поддерживаемых моделей, а также их псевдонимы. Это позволяет пользователям легко переключаться между различными моделями и использовать их через единый интерфейс.

## Классы

### `DeepInfraChat`

**Описание**: Класс `DeepInfraChat` предназначен для настройки и подключения к платформе DeepInfra Chat через API OpenAI. Он наследует функциональность от класса `OpenaiTemplate`.

**Наследует**:
- `OpenaiTemplate`: Предоставляет базовую реализацию для работы с API OpenAI.

**Атрибуты**:
- `url` (str): URL-адрес для доступа к DeepInfra Chat. Значение: `"https://deepinfra.com/chat"`.
- `api_base` (str): Базовый URL API OpenAI для DeepInfra. Значение: `"https://api.deepinfra.com/v1/openai"`.
- `working` (bool): Указывает, является ли провайдер рабочим. Значение: `True`.
- `default_model` (str): Модель, используемая по умолчанию. Значение: `'deepseek-ai/DeepSeek-V3'`.
- `default_vision_model` (str): Визуальная модель, используемая по умолчанию. Значение: `'openbmb/MiniCPM-Llama3-V-2_5'`.
- `vision_models` (List[str]): Список поддерживаемых визуальных моделей.
- `models` (List[str]): Список поддерживаемых языковых моделей.
- `model_aliases` (Dict[str, str]): Словарь псевдонимов моделей для удобства использования.

**Принцип работы**:

Класс `DeepInfraChat` настраивает параметры, необходимые для взаимодействия с API DeepInfra. Он определяет URL-адрес, базовый URL API, список поддерживаемых моделей и псевдонимы моделей. При инициализации класса эти параметры используются для настройки соединения и выбора подходящей модели для генерации текста.
