## Функции

### `process_data`

**Описание**: Обрабатывает данные, полученные из HTML-формы, передает их скрипту Python и возвращает результат.

**Параметры**:

- `request: Request`: Объект запроса.
- `first_name: str = Form(...)`: Имя пользователя.
- `last_name: str = Form(...)`: Фамилия пользователя.

**Возвращает**:

- `dict | None`: Словарь с результатом выполнения скрипта Python или `None` при ошибке.


**Обрабатывает исключения**:

- `HTTPException(status_code=400, detail="First name and last name must be provided")`: Если имя или фамилия не предоставлены.
- `HTTPException(status_code=500, detail=f"Error executing the script: {stderr.decode()}")`: Если возникла ошибка при выполнении скрипта Python.


### `open_index`

**Описание**: Возвращает информацию о перенаправлении на главную страницу.

## Дополнительные заметки

- Модуль использует `uvicorn` для запуска приложения. Необходимо установить `uvicorn` и `python-dotenv` (если используются переменные окружения).
- `html` папка должна содержать начальную страницу (index.html) с формой.
- В папке `./src` должен быть файл `script.py`, который будет обрабатывать полученные данные.
- Присутствуют попытки запустить браузер на странице index.html, но они не реализованы оптимальным способом.
