# Модуль `src.webdriver.driver`

## Обзор

Этот модуль предоставляет базовый класс для работы с веб-драйверами. Он содержит реализацию общих методов для управления веб-драйверами, включая навигацию, взаимодействие с элементами, обработку исключений и работу с локальными настройками.

## Классы

### `Driver`

**Описание**: Основной класс для работы с веб-драйверами.  Используется в качестве обертки над различными типами драйверов (Chrome, Firefox, Edge)

**Конструктор**:


### `__init__`

**Описание**: Инициализирует экземпляр класса `Driver`.

**Параметры**:
- `*args`, `**kwargs`: Аргументы и ключевые аргументы для конструктора подкласса.


### `driver_payload`

**Описание**: Возвращает данные для драйвера.

**Возвращает**:
- dict: Словарь с данными драйвера.


### `DriverBase`

**Описание**: Базовый класс для работы с веб-драйверами.


#### Атрибуты

- `previous_url: str`: Предыдущий URL.
- `referrer: str`: Ссылка на предыдущую страницу.
- `page_lang: str`: Язык страницы.
- `ready_state`: Состояние загрузки страницы.
- `get_page_lang()`: Получение языка страницы.
- `unhide_DOM_element()`: Раскрытие элемента DOM (необходимо уточнить функционал).
- `get_referrer()`: Получение ссылки на предыдущую страницу.
- `window_focus()`: Приведение окна в фокус.
- `execute_locator()`: Выполнение локализации.
- `click()`: Клик по элементу.
- `get_webelement_as_screenshot()`: Получение скриншота элемента.
- `get_attribute_by_locator()`: Получение атрибута элемента по локатору.
- `send_message()`: Отправка сообщения.
- `send_key_to_webelement()`: Ввод текста в элемент.


#### Методы

##### `scroll`

**Описание**: Прокрутка страницы.

**Параметры**:
- `scrolls: int`: Количество прокруток.
- `frame_size: int`: Размер фрейма.
- `direction: str`: Направление прокрутки.
- `delay: float`: Задержка между прокрутками.

**Возвращает**:
- `None | bool`: Значение `None` или `bool` в зависимости от успешности операции.


##### `locale`

**Описание**: Установка языка.

**Параметры**:
- `self`: Экземпляр класса.

**Возвращает**:
- `None | str`: Значение `None` или строка языка, если успешно.


##### `get_url`

**Описание**: Получение текущего URL.

**Параметры**:
- `url: str`: URL.

**Возвращает**:
- `bool`: `True`, если URL получен успешно.


##### `extract_domain`

**Описание**: Извлечение домена из URL.

**Параметры**:
- `url: str`: URL.

**Возвращает**:
- `str`: Домен.


##### `_save_cookies_localy`

**Описание**: Сохранение куки в файл.

**Параметры**:
- `to_file: str | Path`: Путь к файлу.

**Возвращает**:
- `bool`: Успешность сохранения.


##### `page_refresh`

**Описание**: Обновление страницы.

**Возвращает**:
- `bool`: Успешность обновления.


##### `wait`

**Описание**: Ожидание загрузки страницы.

**Параметры**:
- `interval: float`: Интервал ожидания.


##### `delete_driver_logs`

**Описание**: Очистка логов драйвера.

**Возвращает**:
- `bool`: Успешность очистки.


##### `click`

**Описание**: Клик по элементу.


##### `get_webelement_as_screenshot`

**Описание**: Получение скриншота элемента.


##### `get_attribute_by_locator`

**Описание**: Получение атрибута элемента по локатору.


##### `send_message`

**Описание**: Отправка сообщения.


##### `send_key_to_webelement`

**Описание**: Ввод текста в элемент.


##### `unhide_DOM_element`

**Описание**: Раскрытие элемента DOM.


##### `get_referrer`

**Описание**: Получение ссылки на предыдущую страницу.


##### `get_page_lang`

**Описание**: Получение языка страницы.


### `DriverMeta`

**Описание**: Метакласс для класса `Driver`.


#### Методы

##### `__call__`

**Описание**: Вызов метакласса.

**Параметры**:
- `cls`: Метакласс.
- `webdriver_cls`: Класс веб-драйвера.
- `*args`: Аргументы.
- `**kwargs`: Ключевые аргументы.

**Возвращает**:
- `Driver`: Экземпляр класса `Driver`.
