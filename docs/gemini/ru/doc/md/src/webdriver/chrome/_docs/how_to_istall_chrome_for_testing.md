## Установка ChromeDriver

Для работы с Chrome WebDriver необходимо установить ChromeDriver, который является драйвером для взаимодействия с браузером Chrome.

## Выбор версии ChromeDriver

Необходимо выбрать версию ChromeDriver, соответствующую версии Chrome, которая используется для тестирования.  Рекомендуется использовать последнюю стабильную версию.

## Загрузка ChromeDriver

Скачать необходимую версию ChromeDriver можно с официального сайта [https://chromedriver.chromium.org/downloads](https://chromedriver.chromium.org/downloads).

## Установка ChromeDriver (1)

После загрузки, распакуйте скачанный архив в удобное для вас место.  Важно:  разместите исполняемый файл ChromeDriver (например, `chromedriver.exe` на Windows) в папке, которая добавлена в переменную окружения `PATH`.  Это позволит запускать ChromeDriver из любой точки вашей системы.

## Проверка установки

После установки, проверьте работоспособность ChromeDriver:
1. Откройте командную строку или терминал.
2. Введите команду `chromedriver --version`.
3. Если установка прошла успешно, вы увидите информацию о версии ChromeDriver.

Если команда не распознается или выдает ошибку, проверьте правильность установки и добавления папки с `chromedriver` в переменную `PATH`.
