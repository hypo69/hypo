# Модуль grender

## Обзор

Данный модуль предоставляет инструменты для форматирования и рендеринга Google Таблиц. Он содержит классы и функции для работы с ячейками, форматированием, слиянием ячеек и настройкой таблиц.

## Классы

### `GSRender`

**Описание**: Класс `GSRender` содержит методы для рендеринга Google Таблиц. Он отвечает за форматирование заголовков, слияние ячеек и настройку свойств таблиц.

**Методы**:

- `__init__`:
    **Описание**: Инициализирует объект `GSRender`.
    **Параметры**:
        - `self`: Экземпляр класса `GSRender`.
        - `*args`: Аргументы.
        - `**kwards`: Параметры.
    **Возвращает**:
        - `None`: Не возвращает значения.
- `render_header`:
    **Описание**: Рисует заголовок таблицы в первой строке.
    **Параметры**:
        - `self`: Экземпляр класса `GSRender`.
        - `ws` (Worksheet): Таблица в книге.
        - `world_title` (str): Заголовок Google таблицы.
        - `range` (str = 'A1:Z1'): Диапазон ячеек.
        - `merge_type` (str('MERGE_ALL') | str('MERGE_COLUMNS') | str('MERGE_ROWS') = 'MERGE_ALL'): Тип слияния ячеек.
    **Возвращает**:
        - `None`: Не возвращает значения.
- `merge_range`:
    **Описание**: Объединяет ячейки в заданном диапазоне.
    **Параметры**:
        - `self`: Экземпляр класса `GSRender`.
        - `ws` (Worksheet): Таблица.
        - `range` (str): Диапазон ячеек.
        - `merge_type` (str('MERGE_ALL') | str('MERGE_COLUMNS') | str('MERGE_ROWS') = 'MERGE_ALL'): Тип слияния.
    **Возвращает**:
        - `None`: Не возвращает значения.
- `set_worksheet_direction`:
    **Описание**: Устанавливает направление текста в Google таблице.
    **Параметры**:
        - `self`: Экземпляр класса `GSRender`.
        - `sh` (Spreadsheet): Google Таблица.
        - `ws` (Worksheet): Рабочий лист.
        - `direction` (str('ltr') | str('rtl') = 'rtl'): Направление текста.
    **Возвращает**:
        - `None`: Не возвращает значения.
- `header`:
    **Описание**: Добавляет строку заголовков в таблицу.
    **Параметры**:
        - `self`: Экземпляр класса `GSRender`.
        - `ws` (Worksheet): Рабочий лист.
        - `ws_header` (str | list): Заголовок.
        - `row` (int = None): Номер строки.
    **Возвращает**:
        - `None`: Не возвращает значения.
- `write_category_title`:
    **Описание**: Добавляет строку категории в таблицу.
    **Параметры**:
        - `self`: Экземпляр класса `GSRender`.
        - `ws` (Worksheet): Рабочий лист.
        - `ws_category_title` (str | list): Заголовок категории.
        - `row` (int = None): Номер строки.
    **Возвращает**:
        - `None`: Не возвращает значения.
- `get_first_empty_row`:
    **Описание**: Находит первую пустую строку в таблице.
    **Параметры**:
        - `self`: Экземпляр класса `GSRender`.
        - `ws` (Worksheet): Рабочий лист.
        - `by_col` (int = None): Номер столбца для поиска.
    **Возвращает**:
        - `int`: Номер первой пустой строки.
