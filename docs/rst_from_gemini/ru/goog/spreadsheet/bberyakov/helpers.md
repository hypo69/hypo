# helpers.py

**Расположение:** `C:\Users\user\Documents\repos\hypotez\src\goog\spreadsheet\bberyakov\helpers.py`

**Роль:** `doc_creator` (генерация документации для модуля)

**Описание:**

Модуль `helpers.py` предоставляет функции для перевода цветовых форматов. Поддерживаются преобразования между HEX, десятичным (DECIMAL) и RGB форматами.

**Функции:**

* **`hex_color_to_decimal(letters: str) -> int`:** Преобразует цвет из HEX (например, "A1") в десятичный формат.

    * **Вход:** Строка `letters` представляющая цвет в HEX формате (например, "A1").
    * **Выход:** Целое число, представляющее цвет в десятичном формате.
    * **Описание:** Функция использует сложные алгоритмы для преобразования HEX в DECIMAL, обрабатывая как односимвольные, так и двухсимвольные HEX значения.  В документации не хватает объяснения логики преобразования.
    * **Примеры:**
        ```python
        print(hex_color_to_decimal("A"))  # Output: 10
        print(hex_color_to_decimal("AA")) # Output: 27
        ```
        
* **`decimal_color_to_hex(number: int) -> str`:** Преобразует цвет из десятичного (DECIMAL) формата в HEX.

    * **Вход:** Целое число `number`, представляющее цвет в десятичном формате.
    * **Выход:** Строка, представляющая цвет в HEX формате.
    * **Описание:** Функция использует обратный алгоритм для преобразования DECIMAL в HEX, учитывая как односимвольные, так и двухсимвольные HEX представления.  Необходимо доработать документацию по граничным случаям (например, отрицательные числа, числа, которые не представляют валидный HEX).
    * **Примеры:**
        ```python
        print(decimal_color_to_hex(10))  # Output: "A"
        print(decimal_color_to_hex(27)) # Output: "AA"
        ```


* **`hex_to_rgb(hex: str) -> tuple`:** Преобразует цвет из HEX формата в RGB.

    * **Вход:** Строка `hex`, представляющая цвет в HEX формате (например, "#FF0000" или "FF0000").
    * **Выход:** Кортеж `(r, g, b)`, где `r`, `g`, `b` - целые числа, представляющие значения красного, зеленого и синего цветов в RGB формате.
    * **Описание:** Функция обрабатывает как строки с `#`, так и без него в качестве входного HEX формата.  Обработка некорректных входных данных нуждается в улучшении.
    * **Примеры:**
        ```python
        print(hex_to_rgb("#FF0000"))  # Output: (255, 0, 0)
        print(hex_to_rgb("FF0000"))  # Output: (255, 0, 0)
        ```
