# hypotez/src/gui/openai_trаigner/version.py

## Обзор

Файл `version.py` содержит информацию о версии, авторе, лицензии и другие метаданные проекта `openai_trаiner`.

## Содержание

- [Обзор](#обзор)
- [Переменные](#переменные)

## Переменные

### `MODE`
**Описание**: Определяет режим работы приложения. В текущей версии установлено значение `'dev'`.
    
**Тип**: `str`

### `__version__`
**Описание**: Версия проекта.
    
**Тип**: `str`

### `__doc__`
**Описание**:  Строка документации. В текущей версии пустая.
    
**Тип**: `str`

### `__details__`
**Описание**:  Детали проекта. В текущей версии пустая.
    
**Тип**: `str`

### `__author__`
**Описание**:  Автор проекта.
    
**Тип**: `str`

### `__copyright__`
**Описание**:  Информация о лицензии и авторских правах.
    
**Тип**: `str`

### `__cofee__`
**Описание**:  Сообщение с ссылкой на страницу поддержки разработчика.
    
**Тип**: `str`
