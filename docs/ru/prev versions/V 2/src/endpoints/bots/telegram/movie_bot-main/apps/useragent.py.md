# Модуль useragent

## Обзор

Модуль `useragent` предоставляет функцию для получения случайного User-Agent из заранее определенного списка. Это полезно для имитации различных браузеров при выполнении HTTP-запросов, чтобы избежать блокировок и ограничений.
