## Функции

### `login`

**Описание**: Функция для выполнения входа поставщика.

**Параметры**:
- `s` (any): Объект поставщика (Supplier).

**Возвращает**:
- `bool`: `True`, если вход выполнен успешно, в противном случае `False`.

**Вызывает исключения**:
- `NameError`: Возникает, если в коде используется переменная `Truee`, которая не определена.
