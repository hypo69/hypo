# Задача No 644
## Числа Рамануджана-Харди, которые также являются произведениями трех простых чисел

## Условия
Рассмотрим числа в виде n = a³ + b³, где a и b — положительные целые числа.

Если n может быть выражено двумя разными способами как a³ + b³, то n называется числом Рамануджана-Харди. Например, 1729 = 1³ + 12³ = 9³ + 10³ является наименьшим числом Рамануджана-Харди.

Пусть RH(m) — m-е число Рамануджана-Харди, при этом RH(1) = 1729.

Также, рассмотрим числа в виде n = p * q * r, где p, q и r — различные простые числа.

Пусть T(m) — m-е число, удовлетворяющее этому условию, при этом T(1) = 30 = 2 * 3 * 5.

Пусть RHT(m) — m-е число, которое одновременно является числом Рамануджана-Харди и произведением трех различных простых чисел. RHT(1) = 1729 = 7 * 13 * 19.

Найдите RHT(10), десятое число, которое одновременно является числом Рамануджана-Харди и произведением трех различных простых чисел.
