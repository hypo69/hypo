Модуль для обработки событий расширения OpenAI Model Interface в Chrome.
=========================================================================================

Этот модуль отслеживает событие установки расширения и выводит сообщение в консоль.

Пример использования
--------------------

Модуль автоматически активируется при установке расширения в Chrome.

"""

from src.logger.logger import logger

# Обработчик события установки расширения Chrome.
chrome.runtime.onInstalled.addListener(() => {
    try:
        # Код выводит сообщение об успешной установке расширения в консоль разработчика.

    except Exception as e:
         # Код логирует ошибку, если она возникает при попытке установки расширения

})
