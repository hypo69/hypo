* **Неэффективная обработка ввода:** Пользовательский ввод в интерактивном меню можно улучшить.  Сейчас пользователю нужно вводить `exit` для выхода. Можно сделать так, чтобы `'q'` или `'quit'` также выходили из программы.

* **Необходимость использования функции `input` в `show_help`:**  Функция `show_help` должна выводить меню из `input`.

* **Проверка на существование файла `credentials.py`:** Если файл `credentials.py` отсутствует, импорт `from .credentials import gs` вызовет ошибку.

Он содержит функции для запуска скрипта 1 и 2, а также для вывода справки.
Также обрабатывает аргументы командной строки, позволяя запускать программу с параметром --help для вывода справки.
"""
from packaging.version import Version

import argparse
import sys

    from .credentials import gs

def script1():

def show_help():
    """Выводит справку по доступным командам."""
    print("\nДоступные команды:")

    """Интерактивное меню для выбора и запуска скриптов."""
