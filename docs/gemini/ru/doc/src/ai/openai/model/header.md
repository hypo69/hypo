# Модуль hypotez/src/logger

## Обзор

Модуль `hypotez/src/logger` определяет корневой путь к проекту `hypotez`.  Все импорты строятся относительно этого пути.  В дальнейшем планируется перенести эту функциональность в системные переменные.

## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная с текущего каталога и идя вверх по дереву директорий, до тех пор, пока не найдет директорию, содержащую один из указанных маркеров (файлы или папки).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или каталогов, используемых для определения корневого каталога проекта.

**Возвращает**:

- `Path`: Путь к корневому каталогу проекта, если он найден; в противном случае - путь к текущей директории скрипта.


**Пример использования**:

```python
root_path = set_project_root()
print(root_path)
```

**Вызывает исключения**:
- Не вызывает исключений.


## Глобальные переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта, полученный вызовом функции `set_project_root`.

**Тип**: `Path`

**Значение**: Путь к корневому каталогу проекта.


### `settings`

**Описание**: Словарь настроек проекта, загруженный из файла `settings.json` в корне проекта.

**Тип**: `dict`

**Значение**: Словарь настроек или `None`, если файл `settings.json` не найден или содержит недопустимый JSON.


### `doc_str`

**Описание**: Текст документации, извлечённый из файла `README.MD` в корне проекта.

**Тип**: `str`

**Значение**: Строка документации или `None`, если файл `README.MD` не найден или при попытке чтения.


### `__project_name__`

**Описание**: Название проекта, полученное из настроек (`settings.json`).

**Тип**: `str`

**Значение**: Строка с именем проекта. По умолчанию `"hypotez"`.


### `__version__`

**Описание**: Версия проекта, полученное из настроек (`settings.json`).

**Тип**: `str`

**Значение**: Строка с версией проекта. По умолчанию `""`.


### `__doc__`

**Описание**: Текст документации проекта, извлечённый из файла `README.MD` в корне проекта.

**Тип**: `str`

**Значение**: Строка документации или пустая строка, если файл не найден.


### `__details__`

**Описание**: Подробная информация о проекте (из настроек или пустая строка).

**Тип**: `str`

**Значение**: Строка с дополнительной информацией. По умолчанию пустая строка.


### `__author__`

**Описание**: Автор проекта, полученный из настроек (`settings.json`).

**Тип**: `str`

**Значение**: Строка с именем автора. По умолчанию `""`.


### `__copyright__`

**Описание**: Авторские права на проект, полученные из настроек (`settings.json`).

**Тип**: `str`

**Значение**: Строка с авторскими правами. По умолчанию `""`.


### `__cofee__`

**Описание**: Ссылка для поддержки разработчиков (например, оплата кофе).

**Тип**: `str`

**Значение**: Строка с ссылкой. По умолчанию — ссылка на бусти.
