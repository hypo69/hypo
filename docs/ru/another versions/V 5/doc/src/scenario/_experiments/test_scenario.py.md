# Модуль `test_scenario.py`

## Обзор

Модуль `test_scenario.py` предназначен для тестирования сценариев взаимодействия с различными поставщиками (suppliers) в рамках проекта `hypotez`. Он включает в себя функциональность для запуска поставщика с определенным префиксом и выполнения сценариев для этого поставщика.

## Подробней

Этот модуль используется для проведения экспериментов и тестирования логики взаимодействия с поставщиками, такими как `aliexpress`, `amazon`, `kualastyle` и `ebay`. Он позволяет проверить корректность работы сценариев и убедиться, что взаимодействие с поставщиками происходит без ошибок.

## Функции

### `start_supplier`

```python
def start_supplier(supplier_prefix: str) -> Supplier:
    """
    Args:
        supplier_prefix (str): Префикс поставщика, определяющий, с каким поставщиком будет происходить взаимодействие.

    Returns:
        Supplier: Объект класса `Supplier`, инициализированный с указанным префиксом.

    Raises:
        Нет.

    Example:
        >>> supplier = start_supplier('aliexpress')
    """
```

**Описание**: Инициализирует и возвращает объект класса `Supplier` на основе предоставленного префикса поставщика.

**Как работает функция**:
1. Определяет словарь `params`, содержащий префикс поставщика.
2. Создает и возвращает объект класса `Supplier` с использованием этого словаря.

**Параметры**:
- `supplier_prefix` (str): Префикс поставщика (например, `'aliexpress'`).

**Возвращает**:
- `Supplier`: Объект класса `Supplier`, инициализированный с указанным префиксом.

**Примеры**:

```python
supplier = start_supplier('aliexpress')
```

## Переменные

- `supplier_prefix` (str): Префикс поставщика по умолчанию (в данном случае `'aliexpress'`).
- `s` (Supplier): Объект класса `Supplier`, созданный с использованием функции `start_supplier`.
- `scenario` (Scenario): Объект класса `Scenario`, созданный с использованием объекта `Supplier`.
