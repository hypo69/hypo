# Модуль hypotez/src/webdriver/edge/header.py

## Обзор

Этот модуль содержит функции для определения корневого каталога проекта и загрузки настроек из файла `settings.json`.  Он также загружает README.MD, если файл существует.  Модуль предоставляет переменные, содержащие информацию о проекте, такие как имя, версия, описание, автор и пр.

## Оглавление

* [Функции](#функции)
    * [`set_project_root`](#set_project_root)


## Функции

### `set_project_root`

**Описание**:  Находит корневой каталог проекта, начиная с текущего каталога и идя вверх по древу каталогов.  Ищет файлы, указанные в `marker_files`, чтобы определить корневой каталог. Если корневой каталог не найден, возвращает текущий каталог.


**Параметры**:

* `marker_files` (tuple): Кортеж с именами файлов или каталогов, которые указывают на корневой каталог проекта. По умолчанию (`pyproject.toml`, `requirements.txt`, `.git`).


**Возвращает**:

* `Path`: Путь к корневому каталогу проекта, если найден, иначе путь к каталогу текущего файла.


**Вызывает исключения**:

- Нет.


## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта, полученный вызовом функции `set_project_root`.

**Тип**: `Path`

### `settings`

**Описание**: Словарь со значениями настроек, загруженных из файла `settings.json`. Если файл не найден или содержит невалидные данные, `settings` будет равно `None`.

**Тип**: `dict` или `None`

### `doc_str`

**Описание**: Строка с содержимым файла README.MD, загруженного из файла `README.MD`. Если файл не найден, `doc_str` будет равно `None`.

**Тип**: `str` или `None`

### `__project_name__`

**Описание**: Имя проекта.  Получается из настроек (`settings`), по умолчанию равно 'hypotez'.

**Тип**: `str`

### `__version__`

**Описание**: Версия проекта.  Получается из настроек (`settings`), по умолчанию равно пустой строке.

**Тип**: `str`


### `__doc__`

**Описание**: Документация проекта.  Получается из файла `README.MD`, по умолчанию пустая строка.

**Тип**: `str`

### `__details__`

**Описание**: Дополнительные детали о проекте.  По умолчанию пустая строка.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта.  Получается из настроек (`settings`), по умолчанию пустая строка.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права.  Получается из настроек (`settings`), по умолчанию пустая строка.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка для поддержки разработчика.  Получается из настроек (`settings`), по умолчанию ссылка на Boosty.

**Тип**: `str`
