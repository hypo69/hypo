# Модуль `hypotez/src/suppliers/aliexpress/utils/locales.py`

## Обзор

Модуль `locales.py` предназначен для загрузки данных локализации из файла JSON. Он содержит функцию для загрузки и обработки данных локализации, необходимых для работы с данными из AliExpress.

## Функции

### `get_locales`

**Описание**: Загружает данные локализации из JSON файла.

**Параметры**:

- `locales_path` (Path | str): Путь к файлу JSON, содержащему данные локализации.


**Возвращает**:

- `list[dict[str, str]] | None`: Список словарей, где каждый словарь содержит пару "язык" (ключ) - "валюта" (значение). Возвращает `None`, если файл не найден или произошла ошибка при чтении.

**Примеры**:

```python
from src.suppliers.aliexpress.utils.locales import get_locales
from pathlib import Path

locales = get_locales(Path('/path/to/locales.json'))
print(locales)
```

**Примечания**: Использует функцию `j_loads_ns` для загрузки данных из файла, ожидаемого формата JSON. Возвращаемое значение - `None` для случая отсутствия данных в файле.


### `load_locales_data`

**Описание**: (Из документации в коде). Не существует в текущем файле. 


**Примечания**: Данная функция `load_locales_data` не определена в данном файле. Приведенный в документации пример ее использования является некорректным, так как эта функция отсутствует.


## Переменные

### `locales`

**Описание**: Список словарей, содержащий загруженные данные локализации (или `None`, если загрузка не удалась).

**Примечания**: Значение этой переменной определяется в момент исполнения, при выполнении вызова функции `get_locales`.


## Использование

Для использования модуля необходимо импортировать соответствующую функцию и передать путь к файлу JSON в качестве аргумента.


```python
from hypotez.src.suppliers.aliexpress.utils.locales import get_locales
from pathlib import Path

locales_path = Path("path/to/locales.json") # Замените на фактический путь
locales_data = get_locales(locales_path)

if locales_data:
    for locale_data in locales_data:
        print(locale_data)
else:
    print("Ошибка загрузки данных локализации.")
```
