# Модуль `src.suppliers.ivory.graber`

## Обзор

Модуль `src.suppliers.ivory.graber` предназначен для сбора данных о товарах с сайта `ivory.co.il`. Класс `Graber` наследуется от родительского класса `Graber` и переопределяет его методы для обработки специфических полей сайта `ivory.co.il`. Модуль включает механизмы для обработки всплывающих окон с использованием декораторов.

## Классы

### `Graber`

**Описание**: Класс для операций захвата данных о товарах с сайта `ivory.co.il`.

**Наследует от**: `src.suppliers.graber.Graber`

**Методы**:
- `__init__`: Инициализирует экземпляр класса `Graber`, устанавливает префикс поставщика и вызывает конструктор родительского класса.

#### `__init__`
**Описание**: Инициализация класса сбора полей товара.

**Параметры**:
- `driver` (`src.webdriver.driver.Driver`): Экземпляр веб-драйвера для управления браузером.

**Возвращает**:
- `None`

## Функции

### `close_pop_up`
**Описание**: Декоратор для закрытия всплывающих окон перед выполнением основной логики функции.

**Параметры**:
- `value` (`Any`, optional): Дополнительное значение для декоратора. По умолчанию `None`.

**Возвращает**:
- `Callable`: Декоратор, оборачивающий функцию.

**Пример использования:**
```python
@close_pop_up()
async def my_function():
    # Логика функции после закрытия всплывающего окна
    pass
```
