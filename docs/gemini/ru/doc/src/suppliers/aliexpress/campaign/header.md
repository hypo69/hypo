# Модуль hypotez/src/suppliers/aliexpress/campaign/header.py

## Обзор

Этот модуль содержит функцию `set_project_root` для определения корневого каталога проекта и инициализирует переменные, связанные с проектом, такими как `__root__`, `__project_name__`, `__version__`, `__doc__`, `__details__`, `__author__`, `__copyright__`, и `__cofee__`, считывая данные из файла `settings.json` и `README.MD` в каталоге проекта.


## Функции

### `set_project_root`

**Описание**: Определяет корневой каталог проекта, начиная с текущего каталога и поднимаясь вверх по дереву каталогов, пока не найдет каталог, содержащий файлы `pyproject.toml`, `requirements.txt` или `.git`.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или каталогов, используемых для определения корневого каталога. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Объект `Path` представляющий путь к корневому каталогу проекта. Если корневой каталог не найден, возвращает путь к каталогу, в котором находится текущий файл.

**Вызывает исключения**:
- Нет

## Переменные

**Описание**: Переменные, используемые для хранения информации о проекте.

### `__root__`

**Тип**: `Path`

**Описание**: Путь к корневому каталогу проекта, полученный функцией `set_project_root`.

### `__project_name__`

**Тип**: `str`

**Описание**: Название проекта. По умолчанию `'hypotez'`. Получается из файла `settings.json`.

### `__version__`

**Тип**: `str`

**Описание**: Версия проекта. По умолчанию `''`. Получается из файла `settings.json`.

### `__doc__`

**Тип**: `str`

**Описание**: Документация проекта. По умолчанию `''`. Получается из файла `README.MD`.

### `__details__`

**Тип**: `str`

**Описание**: Дополнительные данные о проекте. По умолчанию `''`.

### `__author__`

**Тип**: `str`

**Описание**: Автор проекта. По умолчанию `''`. Получается из файла `settings.json`.

### `__copyright__`

**Тип**: `str`

**Описание**: Авторские права на проект. По умолчанию `''`. Получается из файла `settings.json`.

### `__cofee__`

**Тип**: `str`

**Описание**: Ссылка для поддержки разработчика. По умолчанию  "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69". Получается из файла `settings.json`.

## Обработка исключений

**Описание**:  Модуль обрабатывает исключения `FileNotFoundError` и `json.JSONDecodeError`, возникающие при чтении файлов `settings.json` и `README.MD`. Если файл не найден или JSON невалиден, переменные, соответствующие этим файлам, устанавливаются в значения по умолчанию.
