# Документация для `visualdg_categories_video_asus.json`

## Обзор

Данный файл содержит JSON-конфигурацию для сценариев, связанных с видеокартами ASUS на платформе VisualDG. В частности, он определяет сценарий для видеокарт серии GForce от Nvidia.

## Структура JSON

JSON-файл содержит один главный объект `scenarios`, который является словарем, где ключи - это названия сценариев, а значения - их конфигурации.

## Сценарии

### `GForce`

**Описание**:
Сценарий для видеокарт серии GForce от производителя Nvidia, представленных на сайте VisualDG.

**Параметры**:
- `brand` (str): Производитель видеокарты. В данном случае "Nvidia".
- `url` (str): URL-адрес, ведущий на страницу с видеокартами ASUS на сайте VisualDG.
- `checkbox` (bool): Флаг, указывающий, отмечен ли чекбокс в пользовательском интерфейсе. В данном случае `false`.
- `active` (bool): Флаг, указывающий, активен ли сценарий. В данном случае `true`.
- `condition` (str):  Условие, определяющее состояние товара. В данном случае `"new"`.
- `presta_categories` (str): Список категорий PrestaShop, к которым относится данный товар. В данном случае `"108,524"`.

**Пример:**

```json
{
  "scenarios": {
    "GForce": {
      "brand": "Nvidia",
      "url": "https://www.visualdg.co.il/169411-%D7%9B%D7%A8%D7%98%D7%99%D7%A1%D7%99-%D7%9E%D7%A1%D7%9A-ASUS/243216-Asus",
      "checkbox": false,
      "active": true,
      "condition":"new",
      "presta_categories": "108,524"
    }
  }
}
```
