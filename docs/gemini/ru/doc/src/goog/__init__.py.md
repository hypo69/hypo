# Модуль `src.goog`

## Обзор

Модуль `src.goog` предоставляет функциональность для работы с Google Sheets. Он включает в себя класс `SpreadSheet`, который позволяет взаимодействовать с таблицами Google.

- [Переменные](#переменные)

    - [`SpreadSheet`](#spreadsheet)

## Переменные

### `MODE`
- **Описание**: Режим работы. По умолчанию установлен в `'dev'`.

## Импорты

### `SpreadSheet`

- **Описание**: Класс для взаимодействия с Google Sheets. Подробное описание и методы класса смотрите в документации `spreadsheet.py`.
