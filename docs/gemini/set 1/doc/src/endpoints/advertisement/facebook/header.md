# Модуль `hypotez/src/endpoints/advertisement/facebook/header.py`

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`. Также он загружает описание проекта из `README.MD`.


## Функции

### `set_project_root`

**Описание**:  Находит корневую директорию проекта, начиная с текущей директории файла, перемещаясь вверх по дереву директорий и останавливаясь на первой директории, содержащей один из указанных файлов-маркеров (например, `pyproject.toml`, `requirements.txt`, `.git`). Если корневая директория не найдена, возвращает директорию, в которой находится текущий скрипт.  Добавляет корневую директорию в `sys.path`, если ее там еще нет.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для определения корневой директории проекта. По умолчанию содержит ('pyproject.toml', 'requirements.txt', '.git').


**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае, возвращает путь к директории, в которой находится текущий скрипт.


## Переменные

### `__root__`

**Описание**:  Путь к корневой директории проекта, полученный вызовом функции `set_project_root`.

**Тип**: `Path`

### `settings`

**Описание**: Словарь настроек, загруженный из файла `settings.json` в корневой директории проекта.

**Тип**: `dict` или `None`

### `doc_str`

**Описание**: Строковое представление содержимого файла `README.MD` в корневой директории проекта.

**Тип**: `str` или `None`


### `__project_name__`

**Описание**: Название проекта. Получается из настроек `settings.json` или имеет значение `'hypotez'` по умолчанию.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта. Получается из настроек `settings.json` или имеет значение `''` по умолчанию.

**Тип**: `str`


### `__doc__`

**Описание**: Документация проекта, полученная из файла `README.MD`.

**Тип**: `str`

### `__details__`

**Описание**: Детали проекта.

**Тип**: `str`

### `__author__`

**Описание**: Автор проекта. Получается из настроек `settings.json` или имеет значение `''` по умолчанию.

**Тип**: `str`

### `__copyright__`

**Описание**: Авторские права на проект. Получается из настроек `settings.json` или имеет значение `''` по умолчанию.

**Тип**: `str`

### `__cofee__`

**Описание**: Ссылка на возможность поддержать разработчика. Получается из настроек `settings.json` или имеет значение по умолчанию.

**Тип**: `str`

**Обработка исключений**:

- `FileNotFoundError`: Возникает, если файл `settings.json` или `README.MD` не найден.
- `json.JSONDecodeError`: Возникает, если файл `settings.json` не имеет корректный JSON-формат.
В этих случаях переменные `settings` и `doc_str` устанавливаются в `None`, а программа продолжает выполнение.
