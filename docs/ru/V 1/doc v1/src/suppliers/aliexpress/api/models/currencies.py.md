# Модуль currencies

## Обзор

Модуль `currencies.py` содержит определение класса `Currency`, который представляет собой набор статических атрибутов, соответствующих кодам различных валют. Этот модуль предназначен для использования в других частях проекта `hypotez`, где требуется представление и обработка валютных кодов, используемых AliExpress.

## Подробней

Этот модуль предоставляет централизованный и удобный способ доступа к списку поддерживаемых валют, используемых AliExpress. Использование класса `Currency` позволяет избежать дублирования строковых значений валют по всему коду и обеспечивает консистентность в использовании валютных кодов. Он определяет класс `Currency`, который содержит статические атрибуты, представляющие коды валют, такие как USD, GBP, CAD и другие.

## Классы

### `Currency`

**Описание**: Класс `Currency` предоставляет набор статических атрибутов, представляющих собой коды различных валют.

**Методы**: Нет

**Параметры**: Нет

**Примеры**
```python
from src.suppliers.aliexpress.api.models.currencies import Currency

# Использование кодов валют
usd_code = Currency.USD
print(usd_code)  # Вывод: USD

eur_code = Currency.EUR
print(eur_code)  # Вывод: EUR
```
