# Модуль helpers

## Обзор

Модуль `helpers` содержит функции для работы с цветовыми кодами, преобразуя их между различными форматами (HEX, десятичный, RGB).

## Функции

### `hex_color_to_decimal`

**Описание**: Функция преобразует цвет из HEX-формата в десятичный формат.

**Параметры**:
- `letters` (str): HEX-код цвета (например, 'FF', 'AA').


**Возвращает**:
- `int`: Десятичный код цвета.


**Пример использования**:
```python
print(hex_color_to_decimal('A'))  # Вывод: 10
print(hex_color_to_decimal('AA')) # Вывод: 170
```

### `decimal_color_to_hex`

**Описание**: Функция преобразует цвет из десятичного формата в HEX-формат.

**Параметры**:
- `number` (int): Десятичный код цвета.

**Возвращает**:
- `str`: HEX-код цвета.


**Пример использования**:
```python
print(decimal_color_to_hex(10))  # Вывод: A
print(decimal_color_to_hex(170)) # Вывод: AA
```

### `hex_to_rgb`

**Описание**: Функция преобразует цвет из HEX-формата в RGB-формат.

**Параметры**:
- `hex` (str): HEX-код цвета (например, '#FF0000', 'FF0000').

**Возвращает**:
- `tuple`: Кортеж из трех целых чисел, представляющих значения R, G и B в RGB-формате.


**Пример использования**:
```python
print(hex_to_rgb('#FF0000'))  # Вывод: (255, 0, 0)
print(hex_to_rgb('FF0000')) # Вывод: (255, 0, 0)
```

### `letter_to_number`

**Описание**: Вспомогательная функция, преобразующая буквенный символ в соответствующее числовое представление в формате HEX (A-Z).

**Параметры**:
- `letter` (str): Буква из HEX-кода.

**Возвращает**:
- `int`: Числовое представление буквы в HEX-коде (1-26).

**Примечания:**

Функция `letter_to_number` используется внутри `hex_color_to_decimal` для обработки буквенной части HEX-кода.  В примерах использования показаны возможные вызовы `hex_color_to_decimal`,  которые демонстрируют применение `letter_to_number`.
