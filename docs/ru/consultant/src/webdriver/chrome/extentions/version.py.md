- **Плюсы**:

- **Минусы**:

"""
Модуль для хранения информации о версии
======================================

Модуль содержит информацию о версии, авторе и деталях модуля.

Пример использования:
---------------------
.. code-block:: python

    from src.webdriver.chrome.extentions.version import __version__, __author__, __details__

"""

__doc__: str = 'Модуль для хранения информации о версии'  #  определяем документацию модуля, меняем тип на str
__details__: str = 'Details about version for module or class' #  определяем детали модуля, меняем тип на str
__annotations__ = {} # определяем аннотации, меняем тип на dict # тип для аннотаций
