# DIAMND

## סקירה כללית

הקובץ DIAMND הוא משחק טקסט בו השחקן מנסה לצייר מעוין של כוכביות על ידי הגדרת הגובה שלו.
המשחק קודם כל מבקש מהמשתמש להזין גובה לא זוגי למעוין. לאחר מכן, אם המספר שהוזן הוא לא זוגי, המשחק בונה מעוין של כוכביות, תוך שימוש ברווחים כדי ליצור את הגיאומטריה הנכונה של הצורה. אם המספר שהוזן הוא זוגי או קטן מ-1, מוצגת הודעת שגיאה.
