# Модуль `_experiments`

## Обзор

Модуль `_experiments` содержит экспериментальный код, связанный с поставщиком `ivory`. В коде производятся импорты различных модулей и библиотек, необходимых для работы с поставщиками, веб-драйверами, продуктами, категориями и другими компонентами проекта. Он включает в себя функции для запуска поставщика с определенными параметрами.

## Подробней

Этот модуль, вероятно, предназначен для тестирования и экспериментов с различными функциями и интеграциями, связанными с поставщиком `ivory`. Расположение файла в директории `src/suppliers/ivory/_experiments` указывает на то, что это часть подсистемы, отвечающей за взаимодействие с конкретным поставщиком, и находится на стадии разработки или экспериментов.

## Функции

### `start_supplier`

```python
def start_supplier(supplier_prefix: str = 'aliexpress', locale: str = 'en') -> Supplier:
    """ Старт поставщика """
```

**Описание**: Запускает поставщика с указанными параметрами.

**Как работает функция**:
1.  Принимает `supplier_prefix` и `locale` в качестве параметров.
2.  Создает словарь `params` с этими параметрами.
3.  Инициализирует и возвращает экземпляр класса `Supplier` с использованием переданных параметров.

**Параметры**:

*   `supplier_prefix` (str, optional): Префикс поставщика. По умолчанию `'aliexpress'`.
*   `locale` (str, optional): Локаль поставщика. По умолчанию `'en'`.

**Возвращает**:

*   `Supplier`: Объект поставщика, инициализированный с переданными параметрами.

**Примеры**:

```python
supplier = start_supplier(supplier_prefix='my_supplier', locale='fr')
# Возвращает объект поставщика 'my_supplier' с локалью 'fr'
```

```python
supplier = start_supplier()
# Возвращает объект поставщика 'aliexpress' с локалью 'en'
