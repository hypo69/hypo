# Модуль `src.suppliers.aliexpress.api.models`

## Обзор

Модуль `src.suppliers.aliexpress.api.models` содержит определения моделей данных, используемых для взаимодействия с API AliExpress. Модели включают в себя языковые настройки, валюты, параметры запросов, партнерские ссылки, горячие товары, продукты и категории.

## Импортированные модули
В данном файле происходит импорт следующих моделей:
- `Language`
- `Currency`
- `ProductType`
- `SortBy`
- `LinkType`
- `AffiliateLink`
- `HotProductsResponse`
- `Product`
- `Category`, `ChildCategory`

## Модели

### `Language`

**Описание**: Модель, представляющая язык.

### `Currency`

**Описание**: Модель, представляющая валюту.

### `ProductType`

**Описание**: Модель, представляющая тип продукта.

### `SortBy`

**Описание**: Модель, представляющая способ сортировки.

### `LinkType`

**Описание**: Модель, представляющая тип ссылки.

### `AffiliateLink`

**Описание**: Модель, представляющая партнерскую ссылку.

### `HotProductsResponse`

**Описание**: Модель, представляющая ответ с горячими товарами.

### `Product`

**Описание**: Модель, представляющая продукт.

### `Category`

**Описание**: Модель, представляющая категорию.

### `ChildCategory`

**Описание**: Модель, представляющая дочернюю категорию.
