# Модуль управления кампаниями AliExpress

## Обзор

Этот модуль предоставляет инструменты для управления кампаниями на платформе AliExpress. Он включает в себя функции для редактирования кампаний, управления промо-кампаниями, подготовки данных и взаимодействия с Google Таблицами.


## Файлы

* **`ali_campaign_editor.py`**: Основной файл для редактирования кампаний AliExpress.
* **`ali_promo_campaign.py`**: Управление промо-кампаниями на AliExpress.  Зависит от `AliCampaignGoogleSheet`.
* **`gsheet.py`**: Обработка данных Google Таблиц, связанных с кампаниями. Зависит от библиотек `gspread`, `pandas` и файла настроек `src.settings.gs`.
* **`header.py`**: Общие функции и классы, используемые в модуле.
* **`prepare_campaigns.py`**: Подготовка и организация данных для кампаний.
* **`ttypes.py`**: Определяет типы и структуры, используемые в модуле.
* **`version.py`**: Файл с информацией о версии модуля.


## Функции (Подробно)

Список функций и методов с подробным описанием, параметрами, возвращаемыми значениями и возможными исключениями будет добавлен по мере наличия файлов с определением функций.


## Модули, используемые в этом проекте

* **`gspread`**: Для взаимодействия с Google Таблицами.
* **`pandas`**: Для работы с данными в формате DataFrame.
* **`src.settings.gs`**: Файл с настройками для работы с Google Таблицами.
* **`AliCampaignGoogleSheet`**: Класс для работы с данными кампаний в Google Таблицах (см. `ali_promo_campaign.py`).


## Зависимости

Этот модуль использует следующие внешние библиотеки:

* **`gspread`**: Библиотека для работы с Google Таблицами.
* **`pandas`**: Библиотека для работы с данными.
* **`src.settings.gs`**:  Файл настроек, необходимый для работы с Google Sheets.
