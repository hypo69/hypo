# Модуль `test_jupyter_examples.py`

## Обзор

Данный модуль содержит тесты для выполнения Jupyter ноутбуков. Он перебирает все `.ipynb` файлы в указанной папке, выполняет их с помощью `ExecutePreprocessor`, а затем сохраняет результат. Если во время выполнения возникает ошибка, тест завершается неудачей.

## Функции

### `get_notebooks`

**Описание**: Возвращает список путей к Jupyter ноутбукам в заданной папке.

**Параметры**:
- `folder` (str): Путь к папке, содержащей ноутбуки.

**Возвращает**:
- list[str]: Список путей к файлам `.ipynb`.

### `test_notebook_execution`

**Описание**: Выполняет Jupyter ноутбук и проверяет, что не произошло никаких исключений.

**Параметры**:
- `notebook_path` (str): Путь к файлу `.ipynb`.


**Вызывает исключения**:
- `pytest.fail`: Если во время выполнения ноутбука произошла ошибка.


## Постоянные переменные

### `NOTEBOOK_FOLDER`

**Описание**: Путь к папке, содержащей Jupyter ноутбуки. По умолчанию `../examples/`.


### `TIMEOUT`

**Описание**: Таймаут для долго выполняющихся ноутбуков. По умолчанию 600 секунд.


### `KERNEL_NAME`

**Описание**: Имя ядра Jupyter, используемого для выполнения ноутбуков. По умолчанию `"python3"`.


## Обработка исключений

В функции `test_notebook_execution` используется блок `try...except...finally`, для обработки возможных ошибок во время выполнения ноутбука и сохранения результатов. Блок `finally` гарантирует, что запущенный ноутбук всегда сохраняется, даже если произошла ошибка.  Используется  `pytest.fail`, для более информативной ошибки pytest.
