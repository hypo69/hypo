### Название модуля:
**`101_basic_computer_games`**

### Описание модуля:
"""
Модуль для реализации классических компьютерных игр по мотивам книги *101 BASIC Computer Games*
=========================================================================================

Этот модуль содержит 101 игру, вдохновленную классической книгой *101 BASIC Computer Games*. Все игры были переписаны на Python, сохраняя структуру и логику оригинальных версий, написанных на языке BASIC. Включает как текстовые, так и простые графические игры, с возможностью запуска и игры в командной строке или в окне терминала.

Пример использования
--------------------

Пример использования модуля для запуска одной из игр:

.. code-block:: python

    from _101_basic_computer_games import guess_the_number
    guess_the_number.play()
"""

Такой модуль будет включать в себя разнообразные игры, такие как *Guess the Number*, *Hangman*, *Tic-Tac-Toe* и другие, сохраняя дух оригинала, но с использованием возможностей Python для улучшенной функциональности.
