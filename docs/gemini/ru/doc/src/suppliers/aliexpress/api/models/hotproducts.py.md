# Модуль `hotproducts`

## Обзор

Модуль содержит класс `HotProductsResponse`, предназначенный для представления ответа API, содержащего информацию о популярных продуктах (hot products) с AliExpress. Этот класс включает в себя информацию о текущей странице, количестве записей на странице, общем количестве записей и списке продуктов.

## Подробней

Модуль используется для обработки данных, полученных из API AliExpress, которые содержат информацию о популярных продуктах. Класс `HotProductsResponse` предоставляет структуру для хранения и доступа к этим данным. Он упрощает работу с API, предоставляя типизированный способ доступа к различным частям ответа.

## Классы

### `HotProductsResponse`

**Описание**: Класс предназначен для представления ответа API, содержащего информацию о популярных продуктах с AliExpress.

**Принцип работы**: Класс содержит поля, соответствующие данным, возвращаемым API в ответ на запрос популярных продуктов. Это позволяет удобно хранить и получать доступ к данным, таким как номер текущей страницы, количество элементов на странице, общее количество элементов и список объектов `Product`.

**Атрибуты**:

- `current_page_no` (int): Номер текущей страницы в ответе API.
- `current_record_count` (int): Количество записей на текущей странице.
- `total_record_count` (int): Общее количество записей, доступных в API.
- `products` (List[Product]): Список объектов `Product`, представляющих информацию о продуктах на текущей странице.
