# Модуль hypotez/src/utils/string/formatter.py

## Обзор

Модуль `formatter.py` содержит набор функций для форматирования строк, включая удаление символов, HTML-тегов, линий переноса и нелатинских символов.  Он предоставляет инструменты для очистки данных перед использованием в различных частях приложения.

## Функции

### `remove_line_breaks`

**Описание**: Удаляет символы перевода строки (`\\n` и `\\r`) из входной строки.

**Параметры**:

- `input_str` (str): Входная строка.

**Возвращает**:

- `str`: Строка без символов перевода строки.

### `remove_htmls`

**Описание**: Удаляет HTML-теги из входной строки.

**Параметры**:

- `input_html` (str): Входная строка, содержащая HTML-теги.

**Возвращает**:

- `str`: Строка без HTML-тегов.

### `escape_html_tags`

**Описание**: Заменяет символы `<` и `>` на `&lt;` и `&gt;` соответственно в входной HTML-строке.

**Параметры**:

- `input_html` (str): Входная HTML-строка.

**Возвращает**:

- `str`: Экранированная HTML-строка.


### `escape_to_html`

**Описание**: Заменяет символы на их HTML-экранированные последовательности.

**Параметры**:

- `text` (str): Входной текст.

**Возвращает**:

- `str`: Текст с замененными символами на их HTML-экранированные последовательности.


### `remove_non_latin_characters`

**Описание**: Удаляет нелатинские символы из входной строки.

**Параметры**:

- `input_str` (str): Входная строка.

**Возвращает**:

- `str`: Строка, содержащая только латинские символы.

### `remove_special_characters`

**Описание**: Удаляет специальные символы, не разрешенные в определенных контекстах, из строки или списка строк.

**Параметры**:

- `input_str` (str | list): Входная строка или список строк.

**Возвращает**:

- `str | list`: Обработанная строка или список с удаленными специальными символами.


### `clear_numbers`

**Описание**: Очищает строку, оставляя только десятичные числа и точки.

**Параметры**:

- `input_str` (str): Входная строка.

**Возвращает**:

- `str`: Очищенная строка, содержащая только десятичные числа и точки.

**Пример**:

```python
>>> input_str = 'aaa123.456 cde'
>>> output_str = StringFormatter.clear_numbers(input_str)
>>> print(output_str)
123.456
```
