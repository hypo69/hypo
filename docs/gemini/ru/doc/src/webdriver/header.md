# Модуль hypotez/src/webdriver/header.py

## Обзор

Этот модуль содержит функции для определения корневого каталога проекта, загрузки настроек из файла `settings.json` и получения документации из файла `README.MD`.  Он также инициализирует важные переменные, такие как `__root__`, `__project_name__`, `__version__`, `__doc__`, `__details__`, `__author__`, `__copyright__`, и `__cofee__`.


## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная с текущей директории и поднимаясь вверх по дереву каталогов.  Останавливается на первой директории, содержащей один из указанных маркеров (файлы или директории).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или каталогов, используемых для определения корневого каталога.

**Возвращает**:

- `Path`: Объект Path, представляющий корневой каталог проекта, если он найден; в противном случае возвращает директорию, в которой находится текущий скрипт.

**Обрабатывает исключения**:

Не обрабатывает исключения.


## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта, полученный с помощью функции `set_project_root`.

**Тип**: `Path`


### `settings`

**Описание**: Словарь настроек, загруженный из файла `settings.json` в корневом каталоге проекта.

**Тип**: `dict` | `None`


### `doc_str`

**Описание**: Строка документации, загруженная из файла `README.MD` в корневом каталоге проекта.

**Тип**: `str` | `None`


### `__project_name__`

**Описание**: Имя проекта, взятое из настроек `settings` или установленное по умолчанию "hypotez".

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта, взятая из настроек `settings` или установленная по умолчанию пустой строкой.

**Тип**: `str`


### `__doc__`

**Описание**: Строка документации проекта, взятая из `doc_str` или установленная по умолчанию пустой строкой.

**Тип**: `str`


### `__details__`

**Описание**: Строка дополнительных данных о проекте. Изначально пустая строка.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта, взятый из настроек `settings` или установленный по умолчанию пустой строкой.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права проекта, взятые из настроек `settings` или установленные по умолчанию пустой строкой.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка на страницу, где можно поддержать разработчика. Взято из настроек `settings` или установлено по умолчанию.

**Тип**: `str`


## Обработка исключений

В коде присутствует обработка исключений `FileNotFoundError` и `json.JSONDecodeError` при чтении файлов `settings.json` и `README.MD`.  Это предотвращает сбой программы при отсутствии файлов или некорректном формате данных.
