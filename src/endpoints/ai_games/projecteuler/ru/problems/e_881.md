# Задача No 881
## Разворачивание числа
## Условия

Определим "разворачивание" числа как операцию перестановки цифр таким образом, чтобы получились два натуральных числа, *a* и *b*, в десятичной записи без ведущих нулей, таких что *a* + *b* является максимально возможным.

Например, рассмотрим число 1076.
Возможные разворачивания и их суммы:
1 + 076 = 77
10 + 76 = 86
107 + 6 = 113
1076 + 0 = 1076

Максимальная сумма 1076, достигается в случае разворачивания 1076 + 0, но в определении сказано, что числа должны быть натуральными, то есть не включать 0.
Максимальная сумма среди разворачиваний без 0 - 113, достигается в случае 107 + 6.
Назовем максимальную сумму, которую можно получить при разворачивании натуральных чисел, *М(n)*. Таким образом, *М(1076) = 113*.

Даны следующие значения *М(n)*:
*М(16) = 7*
*М(202) = 22*
*М(12345) = 9846*
*М(198919) = 108909*

Пусть *S(L)* - сумма всех *М(n)* для всех чисел *n* от 1 до 10<sup>L</sup> - 1, где все числа представлены в десятичном виде без ведущих нулей. 
Известно, что:
*S(2) = 45*
*S(4) = 4941*

Найдите *S(16)*.
