# FIPFOP

## Обзор

Игра "FIPFOP" - это простая игра в слова, где компьютер выбирает случайное слово из заданного массива слов, а игрок должен угадать буквы в слове. Игрок угадывает по одной букве за раз, и компьютер отображает слово с найденными буквами на своих местах, а остальные буквы заменяются символом "-". Игрок получает подсказку о длине слова.

## Алгоритм

1.  Инициализировать список возможных слов.
2.  Выбрать случайное слово из списка.
3.  Инициализировать отображаемое слово так, чтобы все буквы были заменены на "-".
4.  Пока слово не разгадано:
    4.1. Отобразить слово с найденными буквами и "-" на оставшихся местах.
    4.2. Попросить игрока ввести букву.
    4.3. Если введенная буква есть в слове:
        4.3.1. Раскрыть все места, где буква появляется в отображаемом слове.
5.  Вывести сообщение "YOU GOT IT!"
6.  Завершение игры.

```python
import random

# רשימת המילים האפשריות
words = ['python', 'computer', 'programming', 'algorithm', 'variable']

# בחירת מילה אקראית מהרשימה
targetWord = random.choice(words)

# יצירת מילה להצגה עם קווים תחתונים במקום האותיות
displayWord = ['-' for _ in targetWord]

# לולאת המשחק הראשית
while True:
    # הדפסת המילה הנוכחית עם האותיות שנמצאו והקווים התחתונים
    print(" ".join(displayWord))

    # קבלת קלט מהמשתמש (ניחוש אות)
    userGuess = input("נחש אות: ").lower()

    # בדיקה אם האות נמצאת במילה
    if userGuess in targetWord:
        # עדכון המילה המוצגת עם האותיות שנמצאו
        for i in range(len(targetWord)):
            if targetWord[i] == userGuess:
                displayWord[i] = userGuess
    else:
         print ("נסה שנית") # אם האות לא נמצאת במילה, לא עושים כלום

    # בדיקה אם כל האותיות נוחשו
    if '-' not in displayWord:
        print("YOU GOT IT!")
        break
```
