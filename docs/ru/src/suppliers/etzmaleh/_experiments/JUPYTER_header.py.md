# Модуль _experiments

## Обзор

Модуль содержит набор экспериментов, связанных с поставщиком Etzmaleh. Включает в себя импорты необходимых библиотек и определение базовых путей к проекту.

## Подробней

Этот модуль содержит экспериментальный код, который, вероятно, использовался для тестирования и разработки различных функций, связанных с поставщиком Etzmaleh. Он включает в себя импорт библиотек, необходимых для работы с файлами, JSON, регулярными выражениями, а также модулей для работы с веб-драйвером, поставщиками, продуктами, категориями и PrestaShop. Модуль также содержит функцию `start_supplier`, которая инициализирует объект поставщика.

## Функции

### `start_supplier`

```python
def start_supplier(supplier_prefix: str = 'aliexpress', locale: str = 'en'):
    """ Старт поставщика """
```

**Описание**: Запускает поставщика с указанным префиксом и локалью.

**Как работает функция**:
Функция принимает префикс поставщика и локаль в качестве аргументов. Создает словарь `params` с этими значениями и использует его для инициализации объекта `Supplier`.

**Параметры**:
- `supplier_prefix` (str, optional): Префикс поставщика. По умолчанию `'aliexpress'`.
- `locale` (str, optional): Локаль поставщика. По умолчанию `'en'`.

**Возвращает**:
- Объект `Supplier`: Объект поставщика, инициализированный с указанными параметрами.

**Примеры**:

```python
supplier = start_supplier(supplier_prefix='aliexpress', locale='en')
```
```python
supplier = start_supplier(supplier_prefix='amazon', locale='de')
```
```python
supplier = start_supplier()
