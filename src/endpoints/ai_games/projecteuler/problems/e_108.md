# Задача No 108
## Диофантовы уравнения типа 1/x + 1/y = 1/n
## Условия
В диофантовом уравнении типа

1/x + 1/y = 1/n

где x, y и n – положительные целые числа, нас интересует количество различных решений (x, y) для конкретного значения n.

Например, при n = 12 существует 15 различных решений:

1/13 + 1/156 = 1/12
1/14 + 1/84 = 1/12
1/15 + 1/60 = 1/12
1/16 + 1/48 = 1/12
1/18 + 1/36 = 1/12
1/20 + 1/30 = 1/12
1/21 + 1/28 = 1/12
1/22 + 1/26.4 = 1/12 (нецелое решение)
1/24 + 1/24 = 1/12
1/26 + 1/22 = 1/12
1/28 + 1/21 = 1/12
1/30 + 1/20 = 1/12
1/36 + 1/18 = 1/12
1/48 + 1/16 = 1/12
1/60 + 1/15 = 1/12
1/84 + 1/14 = 1/12
1/156 + 1/13 = 1/12

Заметим, что если (x, y) является решением, то (y, x) также является решением, поэтому достаточно считать решения, где x ≤ y. Так, для n = 12 существует 8 решений:

1/13 + 1/156 = 1/12
1/14 + 1/84 = 1/12
1/15 + 1/60 = 1/12
1/16 + 1/48 = 1/12
1/18 + 1/36 = 1/12
1/20 + 1/30 = 1/12
1/21 + 1/28 = 1/12
1/24 + 1/24 = 1/12

Какое наименьшее значение n, при котором число решений превышает одну тысячу?
