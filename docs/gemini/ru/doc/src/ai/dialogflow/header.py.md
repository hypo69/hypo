# Модуль `header`

## Обзор

Модуль `header` предназначен для определения основных настроек и констант проекта, таких как режим работы, корневая директория, версия, имя проекта и др. Он также загружает настройки из файла `settings.json` и документацию из `README.MD`.

## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная с каталога текущего файла и поднимаясь вверх до первого каталога, содержащего один из маркерных файлов.

**Параметры**:
- `marker_files` (tuple, optional): Кортеж имен файлов или каталогов, которые идентифицируют корень проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневому каталогу, если найден, иначе — каталог, в котором расположен скрипт.

## Переменные

### `MODE`

**Описание**: Режим работы. По умолчанию установлен в `dev`.

### `__root__`

**Описание**: Путь к корневому каталогу проекта.

### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json`. Может быть `None`, если файл не найден или произошла ошибка при загрузке.

### `doc_str`

**Описание**: Строка с содержимым файла `README.MD`. Может быть `None`, если файл не найден или произошла ошибка при чтении.

### `__project_name__`

**Описание**: Имя проекта. Значение берется из `settings.json` (ключ `"project_name"`), если `settings` определен, иначе используется значение `'hypotez'`.

### `__version__`

**Описание**: Версия проекта. Значение берется из `settings.json` (ключ `"version"`), если `settings` определен, иначе используется пустая строка `''`.

### `__doc__`

**Описание**: Содержимое документации проекта, взятое из переменной `doc_str`. Если `doc_str` равен `None` используется пустая строка `''`.

### `__details__`

**Описание**: Строка с детальной информацией о проекте. В данном случае всегда пустая строка.

### `__author__`

**Описание**: Автор проекта. Значение берется из `settings.json` (ключ `"author"`), если `settings` определен, иначе используется пустая строка `''`.

### `__copyright__`

**Описание**: Информация об авторских правах проекта. Значение берется из `settings.json` (ключ `"copyrihgnt"`), если `settings` определен, иначе используется пустая строка `''`.

### `__cofee__`

**Описание**: Приглашение угостить разработчика чашечкой кофе. Значение берется из `settings.json` (ключ `"cofee"`), если `settings` определен, иначе используется значение по умолчанию: "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69".
