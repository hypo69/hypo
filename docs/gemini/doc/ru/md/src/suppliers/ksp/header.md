# Модуль `hypotez/src/suppliers/ksp/header.py`

## Обзор

Данный модуль содержит функции для определения корневого каталога проекта и загрузки настроек из файла `settings.json`. Он также загружает описание проекта из файла `README.MD` и предоставляет доступ к различным переменным, связанным с проектом, таким как имя, версия, описание, автор и т.д.


## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная с текущей директории и идя вверх по дереву каталогов. Останавливается на первом каталоге, содержащем указанные файлы-маркеры.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или каталогов, используемых для определения корневого каталога проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.


**Возвращает**:

- `Path`: Путь к корневому каталогу проекта. Если корневой каталог не найден, возвращается путь к каталогу, где расположен текущий файл.

**Вызывает исключения**:

- Нет


## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта, полученный функцией `set_project_root()`.


### `settings`

**Описание**: Словарь со значениями настроек, загруженный из файла `settings.json` в корневом каталоге проекта. Может быть `None`, если файл не найден или содержит некорректные данные.


### `doc_str`

**Описание**: Строковое представление содержимого файла `README.MD` в корневом каталоге проекта. Может быть `None`, если файл не найден.


### `__project_name__`

**Описание**: Имя проекта. Получается из настроек (`settings.get("project_name", 'hypotez')`), если они доступны, иначе равно 'hypotez'.


### `__version__`

**Описание**: Версия проекта. Получается из настроек (`settings.get("version", '')`), если они доступны, иначе пустая строка.


### `__doc__`

**Описание**: Описание проекта. Получается из `doc_str`, если он доступен, иначе пустая строка.


### `__details__`

**Описание**: Дополнительные детали проекта. По умолчанию пустая строка.


### `__author__`

**Описание**: Автор проекта. Получается из настроек (`settings.get("author", '')`), если они доступны, иначе пустая строка.


### `__copyright__`

**Описание**: Авторские права проекта. Получается из настроек (`settings.get("copyright", '')`), если они доступны, иначе пустая строка.


### `__cofee__`

**Описание**: Ссылка для пожертвования на чашку кофе для поддержки разработчика. Получается из настроек (`settings.get("cofee", "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69")`), если они доступны, иначе указанное значение по умолчанию.
