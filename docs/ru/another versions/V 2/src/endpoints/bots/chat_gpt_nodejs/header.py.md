# Модуль `src.bots.openai_bots.header`

## Обзор

Модуль `header.py` предназначен для определения базовых настроек и констант проекта, а также для инициализации корневой директории проекта. Он загружает конфигурацию из `settings.json` и `README.md`, определяя имя проекта, версию, документацию и другую метаинформацию.

## Функции

### `set_project_root`

**Описание**:
Находит корневой каталог проекта, начиная с каталога текущего файла и двигаясь вверх, пока не будет найден каталог, содержащий один из маркеров (файлов или каталогов).

**Параметры**:
- `marker_files` (tuple, optional): Кортеж имен файлов или каталогов для идентификации корня проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневому каталогу, если он найден. Если не найден, возвращает каталог, в котором расположен скрипт.

**Пример**
```python
root_path = set_project_root()
print(f"Root path: {root_path}")
```

## Переменные

### `__root__`

**Описание**:
`Path` к корневому каталогу проекта. Инициализируется функцией `set_project_root`.

### `settings`
**Описание**:
Словарь с настройками, загруженными из файла `settings.json`.

### `doc_str`

**Описание**:
Строка с содержимым файла `README.MD`.

### `__project_name__`

**Описание**:
Имя проекта, загруженное из `settings.json` (ключ `project_name`), если `settings` не None, иначе "hypotez".

### `__version__`

**Описание**:
Версия проекта, загруженная из `settings.json` (ключ `version`), если `settings` не None, иначе пустая строка.

### `__doc__`

**Описание**:
Строка документации проекта, загруженная из файла `README.MD`, если `doc_str` не None, иначе пустая строка.

### `__details__`

**Описание**:
Строка с подробной информацией о проекте, по умолчанию пустая строка.

### `__author__`

**Описание**:
Автор проекта, загруженный из `settings.json` (ключ `author`), если `settings` не None, иначе пустая строка.

### `__copyright__`

**Описание**:
Информация об авторских правах проекта, загруженная из `settings.json` (ключ `copyrihgnt`), если `settings` не None, иначе пустая строка.

### `__cofee__`

**Описание**:
Строка с предложением угостить разработчика кофе, загруженная из `settings.json` (ключ `cofee`), если `settings` не None, иначе стандартное сообщение.
