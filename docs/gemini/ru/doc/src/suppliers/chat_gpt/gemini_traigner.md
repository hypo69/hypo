# Модуль gemini_traigner

## Обзор

Этот модуль содержит константы и, вероятно, функции для обучения модели Gemini.  Поддерживает режим разработки (`MODE = 'dev'`).


## Константы

### `MODE`

**Описание**:  Переменная, определяющая режим работы. Текущее значение равно `'dev'`.
