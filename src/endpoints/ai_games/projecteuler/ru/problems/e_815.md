# Задача No 815
## Башни Ханой с двумя стеками

## Условия
Классическая головоломка Ханойской башни состоит из трех стержней и нескольких дисков разного размера, которые можно перемещать с одного стержня на другой. Цель состоит в том, чтобы переместить стопку дисков с начального стержня на целевой стержень, используя промежуточный стержень, соблюдая следующие правила:
  * Одновременно можно перемещать только один диск.
  * Каждый ход заключается в том, чтобы взять верхний диск с одного стержня и поместить его на вершину другого стержня.
  * Ни один диск не может быть помещен поверх диска меньшего размера.

В этой задаче мы рассмотрим вариант головоломки, где используются два промежуточных стержня, а не один. Пусть n - количество дисков и h(n) - минимальное количество ходов, необходимых для решения головоломки, где у нас есть два промежуточных стержня.

Например, h(3) = 5, h(5) = 13, и h(10) = 49.

Найдите h(10000).
