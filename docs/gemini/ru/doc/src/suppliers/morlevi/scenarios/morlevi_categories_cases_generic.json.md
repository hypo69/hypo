# Документация для `morlevi_categories_cases_generic.json`

## Обзор

Файл `morlevi_categories_cases_generic.json` содержит конфигурации для сопоставления категорий товаров "computer cases" (компьютерные корпуса) от поставщика Morlevi с предопределенными шаблонами. Конфигурации включают в себя информацию о бренде, URL, активности, состоянии и соответствующих категориях PrestaShop.

## Структура файла

Файл представляет собой JSON-объект, содержащий поле `scenarios`. Поле `scenarios` содержит словарь, где ключи - это названия сценариев, а значения - объекты, описывающие конфигурацию каждого сценария.

## Описание сценариев

### `GENERIC MID TOWER`
**Описание**: Сценарий для корпусов типа "MID TOWER" от бренда "GENERIC".

**Параметры**:
- `brand`: "GENERIC" (строка) - название бренда.
- `template`: "" (строка) - шаблон (пустая строка).
- `url`: "https://www.morlevi.co.il/Cat/97" (строка) - URL, связанный со сценарием.
- `checkbox`: `false` (логическое значение) - флаг, указывающий, что чекбокс не выбран.
- `active`: `true` (логическое значение) - флаг, указывающий, что сценарий активен.
- `condition`: "new" (строка) - состояние товара.
- `presta_categories`: (словарь) - Словарь категорий PrestaShop
    - `template`: (словарь) - Шаблон категорий
        - `"computer cases"`: `"MID TOWER"` (строка) - Категория PrestaShop для компьютерных корпусов.
