# Модуль `src.webdriver.bs`

## Обзор

Модуль `src.webdriver.bs` предназначен для разбора HTML-контента с использованием `BeautifulSoup` и `XPath`. Он предоставляет класс `BS`, который позволяет загружать HTML-контент с URL-адресов или локальных файлов и выполнять поиск элементов с помощью XPath.

## Классы

### `BS`

**Описание**: Класс для разбора HTML-контента с использованием `BeautifulSoup` и `XPath`.

**Атрибуты**:

- `html_content` (str): HTML-контент, который будет разобран.

**Методы**:

- [`__init__`](#__init__)
- [`get_url`](#get_url)
- [`execute_locator`](#execute_locator)

#### `__init__`

**Описание**: Инициализирует парсер `BS` с необязательным URL.

**Параметры**:
- `url` (Optional[str], optional): URL или путь к файлу для загрузки HTML-контента. По умолчанию `None`.

#### `get_url`

**Описание**: Загружает HTML-контент из файла или URL и разбирает его с помощью `BeautifulSoup` и `XPath`.

**Параметры**:
- `url` (str): Путь к файлу или URL для загрузки HTML-контента.

**Возвращает**:
- `bool`: `True`, если контент был успешно загружен, `False` в противном случае.

#### `execute_locator`

**Описание**: Выполняет поиск элементов по XPath-локатору в HTML-контенте.

**Параметры**:
- `locator` (Union[SimpleNamespace, dict]): Объект-локатор, содержащий селектор и атрибут.
- `url` (Optional[str], optional): Необязательный URL или путь к файлу для загрузки HTML-контента. По умолчанию `None`.

**Возвращает**:
- `List[etree._Element]`: Список элементов, соответствующих локатору.
