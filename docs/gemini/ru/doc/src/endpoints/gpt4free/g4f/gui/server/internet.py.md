# Модуль для работы с интернет-поиском в g4f

## Обзор

Модуль предназначен для организации интернет-поиска и обработки результатов поиска. 
Включает функции для выполнения поисковых запросов и форматирования результатов в удобном виде.

## Подробнее

Модуль `internet.py` предоставляет функциональность для интеграции поисковых запросов в приложение `g4f`. Он использует модуль `web_search` для выполнения фактических поисковых запросов и обработки полученных результатов.

## Функции

### `search`

```python
from ...tools.web_search import search
def search(query: str, num_results: int = 5) -> SearchResults:
    """Выполняет поисковый запрос в интернете.

    Args:
        query (str): Поисковый запрос.
        num_results (int, optional): Количество результатов поиска. По умолчанию 5.

    Returns:
        SearchResults: Результаты поиска.

    Raises:
        Exception: В случае возникновения ошибки при выполнении поискового запроса.

    """
    ...
```

**Назначение**: Выполняет поисковый запрос в интернете и возвращает результаты.

**Параметры**:
- `query` (str): Строка поискового запроса.
- `num_results` (int, optional): Количество желаемых результатов поиска. По умолчанию равно 5.

**Возвращает**:
- `SearchResults`: Объект, содержащий результаты поиска.

**Вызывает исключения**:
- `Exception`: В случае возникновения ошибки при выполнении поискового запроса.

**Как работает функция**:
1. Функция принимает поисковый запрос и количество желаемых результатов.
2. Она вызывает модуль `web_search` для выполнения фактического поискового запроса.
3. Функция возвращает результаты поиска в виде объекта `SearchResults`.

**Примеры**:

```python
from ...tools.web_search import search
results = search("какая погода в Москве", num_results=3)
print(results)
```
