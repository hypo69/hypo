# Инструкции для модели AI

## Обзор

Данный модуль содержит инструкции и системные промпты для модели AI. Он предназначен для определения поведения модели при инициализации и предоставления шаблонов ответов.

## Файлы

### `system_instruction_mexiron.md`

**Описание**: Этот файл содержит системную инструкцию, загружаемую при инициализации модели. Она определяет поведение модели, ее ограничения и ожидаемое поведение в различных сценариях.

**Содержание**:  Подробное описание настроек и правил работы модели, включая возможные ограничения, обработку входных данных, формат вывода, и т.д.  

### `command_instruction_mexiron.md`

**Описание**: Данный файл содержит инструкцию для модели, представляющую собой шаблон ответа. Она описывает структуру и формат, которые модель должна использовать при формировании ответа.

**Содержание**: Описание ожидаемого формата ответа модели, включая ключевые слова, разметку, структуру данных и т.д.
