# Задача No 689
## Совершенные квадраты в факториалах
## Условия

Для целого положительного числа $n$ определим $f(n)$ как наименьшее целое число $m$, такое что $n! + m$ является полным квадратом.

Например:

* $1! + 0 = 1 = 1^2$, поэтому $f(1) = 0$.
* $2! + 2 = 4 = 2^2$, поэтому $f(2) = 2$.
* $3! + 3 = 9 = 3^2$, поэтому $f(3) = 3$.
* $4! + 4 = 28$ не является полным квадратом, $4! + 5 = 29$, ..., $4! + 21 = 45$, $4! + 22 = 46$, $4! + 23 = 47$, $4! + 24 = 48$, $4! + 25 = 49 = 7^2$, поэтому $f(4) = 25$.
* $5! + 1 = 121 = 11^2$, поэтому $f(5) = 1$.

Определим $S(N) = \sum_{n=1}^{N} f(n)$.

Известно, что $S(10) = 144$ и $S(1000) = 1170199$.

Найдите $S(10^7)$.
