# Модуль `hypotez/src/scenario/__init__.py`

## Обзор

Этот модуль содержит функции для выполнения сценариев, используемых для управления поставщиками. Он предоставляет интерфейс для запуска отдельных сценариев, файлов сценариев и списков сценариев.

## Функции

### `run_scenario_files`

**Описание**: Функция запускает сценарии, определенные в указанных файлах.

**Параметры**:
- `s` (Supplier): Объект поставщика, который будет использовать сценарии.
- `scenario_files` (list): Список путей к файлам сценариев.

**Возвращает**:
- `None`: Функция не возвращает значение.

**Вызывает исключения**:
- `FileNotFoundError`: Возникает, если один из файлов сценариев не найден.
- `TypeError`: Возникает, если `scenario_files` не является списком строк.
- `ValueError`: Возникает, если `scenario_files` пуст.
- `Exception`: Возникает при любых других непредвиденных ошибках.


### `run_scenarios`

**Описание**: Функция запускает сценарии, переданные в виде словаря или списка словарей.

**Параметры**:
- `s` (Supplier): Объект поставщика, который будет использовать сценарии.
- `scenarios` (dict | list):  Сценарий в формате словаря или список словарей, содержащих сценарии.

**Возвращает**:
- `None`: Функция не возвращает значение.

**Вызывает исключения**:
- `TypeError`: Возникает, если `scenarios` не является словарем или списком словарей.
- `Exception`: Возникает при любых других непредвиденных ошибках.
