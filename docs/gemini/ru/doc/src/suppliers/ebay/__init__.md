# Модуль hypotez/src/suppliers/ebay/__init__.py

## Обзор

Данный модуль представляет собой инициализацию для модуля `ebay` в пакете `suppliers`. Он содержит константу `MODE` и импортирует класс `Graber` из подмодуля `graber`.

## Константы

### `MODE`

**Описание**:  Переменная, определяющая режим работы модуля. В данном случае она имеет значение 'dev'.

## Модули

### `ebay.graber`

**Описание**: Подмодуль, содержащий класс `Graber`.
