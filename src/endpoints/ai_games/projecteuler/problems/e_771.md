# Задача No 771
## Суммы простых чисел с ограничением на разрядность
## Условия

Рассмотрим разбиения натурального числа *n* на простые слагаемые.

Пусть *p(n)* — это количество таких разбиений, при которых ни одно простое число не превышает количество разрядов *n*.

Например,
- *p(10) = 5*, поскольку существует 5 разбиений числа 10 на простые числа, не превышающие 2-х разрядное число (т.е. простые числа < 100):
    - 3 + 7
    - 5 + 5
    - 2 + 3 + 5
    - 2 + 2 + 3 + 3
    - 2 + 2 + 2 + 2 + 2

- *p(100) = 190*, поскольку существует 190 разбиений числа 100 на простые числа, не превышающие 3-х разрядное число (т.е. простые числа < 1000).

Обозначим через *S(N)* сумму *p(n)* для всех *2 <= n <= N*.
Например, *S(10) = 24*.

Найдите *S(10<sup>7</sup>)*.
