# Модуль `hypotez/src/endpoints/kazarinov/pricelist_generator/__init__.py`

## Обзор

Этот модуль предоставляет начальную точку доступа к функциональности генерации прайс-листов.  Он импортирует класс `ReportGenerator` из подмодуля `pricelist_generator`.  Основной настройкой является переменная `MODE`, которая в текущем варианте установлена в 'dev'.
