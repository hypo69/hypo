# Модуль `hypotez/src/goog/gtranslater/header.py`

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`.  Он также загружает информацию о проекте из `README.MD` для использования в других частях приложения.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с текущего файла и поднимаясь по директориям до тех пор, пока не будет найден файл или директория из `marker_files`. Если корневая директория не найдена, возвращает директорию, где находится текущий скрипт. Дополнительно добавляет корневую директорию в `sys.path`.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для определения корневой директории проекта. По умолчанию содержит `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращается путь к текущей директории.


## Переменные

### `__root__`

**Описание**:  Содержит путь к корневой директории проекта, полученный с помощью функции `set_project_root()`.

### `settings`

**Описание**: Словарь настроек, загруженный из файла `settings.json`.  Может содержать информацию о проекте, авторе и т.д.

**Возможные значения**:
- `dict`: Словарь со значениями настроек.
- `None`: Если файл `settings.json` не найден или содержит невалидный JSON.


### `doc_str`

**Описание**: Строка, содержащая содержимое файла `README.MD`.

**Возможные значения**:
- `str`: Строка с содержимым `README.MD`.
- `None`: Если файл `README.MD` не найден или не существует.


### `__project_name__`

**Описание**: Имя проекта, полученное из настроек. По умолчанию имеет значение `'hypotez'`.

**Тип**: `str`

### `__version__`

**Описание**: Версия проекта, полученная из настроек. По умолчанию пустая строка.

**Тип**: `str`

### `__doc__`

**Описание**: Документация проекта, полученная из `README.MD`. По умолчанию пустая строка.

**Тип**: `str`

### `__details__`

**Описание**: Подробности о проекте. По умолчанию пустая строка.

**Тип**: `str`

### `__author__`

**Описание**: Автор проекта, полученный из настроек. По умолчанию пустая строка.

**Тип**: `str`

### `__copyright__`

**Описание**: Авторские права на проект, полученные из настроек. По умолчанию пустая строка.

**Тип**: `str`

### `__cofee__`

**Описание**: Ссылка для поддержки разработчика. По умолчанию содержит ссылку на Boosty.

**Тип**: `str`
