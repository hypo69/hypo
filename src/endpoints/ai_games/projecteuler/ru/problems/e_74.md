# Задача No 74
## Цепочки факториальных сумм

## Условия
Сумма факториала цифр числа 145 равна 1! + 4! + 5! = 1 + 24 + 120 = 145.

Не каждое число может быть равно сумме факториалов своих цифр. Например,

169 → 363601 → 1454 → 169

Возможно начать с числа 169 и получить цепь длиной 3, в результате которой мы возвращаемся к 169. Эта цепь является циклической.

Пусть мы определим цепь, начинающуюся с числа, как последовательность чисел, которые получаются, пока мы не достигаем того же числа снова. Например, цепь, начинающаяся с числа 69, имеет длину 5:

69 → 363600 → 1454 → 169 → 363601

Оказывается, что большинство таких цепей начинаются с числа, которое в конечном счете приводит к циклическим цепям длиной 1 (например, 1! = 1), 2 (например, 145 → 145), или 3 (например, 169 → 363601 → 1454 → 169).

Нас интересуют только циклы, длина которых составляет ровно 60, из всех стартовых чисел меньше миллиона.

Сколько цепей начинаются с числа меньше миллиона, и имеют длину, равную 60?
