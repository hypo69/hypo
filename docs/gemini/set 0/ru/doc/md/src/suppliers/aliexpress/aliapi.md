# Модуль `hypotez/src/suppliers/aliexpress/aliapi.py`

## Обзор

Данный модуль предоставляет класс `AliApi` для работы с API AliExpress.  Он расширяет базовый класс `AliexpressApi`, добавляя дополнительные функции и взаимодействие с базами данных для управления категориями и рекламными кампаниями.

## Классы

### `AliApi`

**Описание**:  Класс `AliApi` расширяет функциональность базового класса `AliexpressApi`, предоставляя методы для взаимодействия с AliExpress API и работы с базами данных.

**Атрибуты**:

* `manager_categories`: Экземпляр класса `CategoryManager`, управляющий категориями продуктов.
* `manager_campaigns`: Экземпляр класса `ProductCampaignsManager`, управляющий рекламными кампаниями и предложениями.

**Методы**:

#### `__init__`

**Описание**: Инициализирует экземпляр класса `AliApi`.

**Параметры**:

* `language` (str, по умолчанию 'en'): Язык для запросов к API.
* `currency` (str, по умолчанию 'usd'): Валюта для запросов к API.

**Возвращает**:

*  None

#### `retrieve_product_details_as_dict`

**Описание**: Получает информацию о продуктах по списку идентификаторов и возвращает данные в формате словаря.

**Параметры**:

* `product_ids` (list): Список идентификаторов продуктов.

**Возвращает**:

* dict | None: Словарь с данными о продуктах. Возвращает `None`, если произошла ошибка.

**Пример кода (включен в документацию):**

```python
# Пример использования метода
product_ids = [123, 456, 789]
product_details = ali_api_instance.retrieve_product_details_as_dict(product_ids)
print(product_details)

```

#### `get_affiliate_links`

**Описание**:  Получает аффилированные ссылки для указанных ссылок на продукты.

**Параметры**:

* `links` (str | list):  Ссылки на продукты.
* `link_type` (int, по умолчанию 0): Тип аффилированной ссылки.
* `**kwargs`: Дополнительные аргументы для метода родительского класса.

**Возвращает**:

* List[SimpleNamespace]: Список объектов `SimpleNamespace` с аффилированными ссылками.
