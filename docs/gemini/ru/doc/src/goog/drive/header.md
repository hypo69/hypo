# Модуль hypotez/src/goog/drive/header.py

## Обзор

Этот модуль содержит функции для работы с настройками проекта, полученными из файла `settings.json`, и определения корневой директории проекта. Он также обрабатывает возможные ошибки при чтении файла настроек и файла `README.MD`.  В модуле определены константы для имени проекта, версии, документации и других метаданных.


## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла. Поиск выполняется вверх по иерархии директорий, пока не будет найдена директория, содержащая один из указанных файлов или директорий.

**Параметры**:

- `marker_files` (tuple): Кортеж из имен файлов или директорий, используемых для определения корневой директории проекта. По умолчанию используется кортеж ('pyproject.toml', 'requirements.txt', '.git').

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращает директорию, в которой расположен текущий файл.

**Вызывает исключения**:

- Нет


### `__get_project_root_dir` (скрытая функция - не документированная)

**Описание**: Эта функция (с именем, начинающимся с двух подчеркиваний) скрыта и не предназначена для прямого использования извне. Она реализует логику поиска корневой директории, которая использовалась в предыдущей версии.


## Переменные

### `MODE`

**Описание**: Строковая константа, хранящая режим работы (`'dev'`).

### `__root__`

**Описание**:  Экземпляр `Path` к корневой директории проекта, полученный с помощью функции `set_project_root`.

### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженные из файла `settings.json`. Если файл не найден или содержит некорректные данные, переменная `settings` равна `None`.

### `doc_str`

**Описание**: Строка, содержащая содержимое файла `README.MD`. Если файл не найден или содержит некорректные данные, переменная `doc_str` равна `None`.

### `__project_name__`

**Описание**: Имя проекта, взятое из настроек (`settings.json`) или установленное по умолчанию как 'hypotez'.

### `__version__`

**Описание**: Версия проекта, взятая из настроек (`settings.json`) или пустая строка по умолчанию.

### `__doc__`

**Описание**: Документация к проекту, взятая из файла `README.MD` или пустая строка по умолчанию.

### `__details__`

**Описание**: Подробное описание проекта (пустая строка по умолчанию).

### `__author__`

**Описание**: Автор проекта, взятый из настроек (`settings.json`) или пустая строка по умолчанию.

### `__copyright__`

**Описание**: Авторские права проекта, взятый из настроек (`settings.json`) или пустая строка по умолчанию.

### `__cofee__`

**Описание**: Ссылка на бонусную поддержку разработчика (пустая строка по умолчанию).


## Обработка исключений

Обработка `FileNotFoundError` и `json.JSONDecodeError` при чтении файлов `settings.json` и `README.MD` предотвращает падение скрипта при отсутствии или некорректном формате этих файлов. В случае ошибки соответствующая переменная (settings или doc_str) устанавливается в None.
