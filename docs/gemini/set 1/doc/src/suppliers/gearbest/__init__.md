Этот модуль предоставляет инициализацию для поставщика Gearbest.  Он импортирует класс `Graber` из файла `graber.py`.  Константа `MODE` установлена в 'dev'.

## Импорты

**Описание**: Импортирует класс `Graber` из подмодуля `graber` внутри текущего пакета.  Это предполагает, что файл `graber.py` содержит определение класса `Graber`.
