# Модуль hypotez/src/suppliers/morlevi/graber.py

## Обзор

Модуль `hypotez/src/suppliers/morlevi/graber.py` содержит класс `Graber`, предназначенный для сбора данных о товарах с сайта `morlevi.co.il`. Класс реализует асинхронный захват данных с помощью веб-драйвера и предоставляет функции для обработки различных полей товара.  Включает в себя обработку скриншотов изображений и их сохранение локально.

## Описание класса

### `Graber`

**Описание**: Класс `Graber` наследуется от базового класса `Grbr` и отвечает за сбор данных с сайта `morlevi.co.il`.  Он инициализируется экземпляром веб-драйвера и содержит методы для обработки разных полей товара.

**Атрибуты**:

- `supplier_prefix`: Строка, определяющая префикс поставщика.

**Методы**:

#### `__init__(self, driver: Driver)`

**Описание**: Инициализирует экземпляр класса `Graber`.

**Параметры**:

- `driver (Driver)`: Экземпляр класса `Driver` для работы с веб-драйвером.

#### `grab_page(self, driver: Driver) -> ProductFields`

**Описание**: Асинхронный метод для захвата данных страницы товара.

**Параметры**:

- `driver (Driver)`: Экземпляр класса `Driver` для работы с веб-драйвером.

**Возвращает**:

- `ProductFields`: Объект, содержащий собранные данные о товаре.

#### `local_saved_image(self, value: Any = None)`

**Описание**:  Метод для загрузки и сохранения изображения товара локально. Сохраняет путь к сохраненному изображению в поле `ProductFields.local_saved_image`.

**Параметры**:

- `value (Any, optional)`: Дополнительное значение, которое может быть передано при вызове метода в словаре `kwargs`. Если передан, его значение подставляется в поле `ProductFields.local_saved_image`.

**Возвращает**:

- `bool`: `True`, если изображение успешно сохранено, `False` в противном случае.

**Обрабатываемые исключения**:

- `Exception`: Обрабатывает любые возможные исключения при работе с веб-драйвером или сохранением изображения.
