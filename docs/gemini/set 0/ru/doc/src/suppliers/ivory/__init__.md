# Модуль hypotez/src/suppliers/ivory/__init__.py

## Обзор

Данный модуль предоставляет базовый интерфейс для работы с поставщиком данных Ivory. Он содержит конфигурацию режима работы (`MODE`) и импортирует класс `Graber` для непосредственного взаимодействия с источником данных.

## Оглавление

* [Обзор](#обзор)
* [Переменные](#переменные)
* [Модули](#модули)

## Переменные

### `MODE`

**Описание**: Строковая константа, определяющая режим работы. В данном случае, задан режим `dev`.

**Значение**: `'dev'`


## Модули

### `Graber`

**Описание**: Класс, отвечающий за сбор данных от поставщика Ivory. Подробная документация находится в файле `hypotez/src/suppliers/ivory/graber.py`.

**Примечание**:  Для получения полной информации о методах класса `Graber`, пожалуйста, обратитесь к документации соответствующего файла.
