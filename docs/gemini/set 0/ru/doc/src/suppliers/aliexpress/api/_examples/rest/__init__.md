# Модуль `hypotez/src/suppliers/aliexpress/api/_examples/rest/__init__.py`

## Обзор

Данный модуль предоставляет импортные функции для доступа к API AliExpress, предоставляя объекты запросов к различным конечным точкам.

## Импортированные классы

### `AliexpressAffiliateProductSmartmatchRequest`

**Описание**: Класс для запроса информации о продуктах AliExpress на основе алгоритма Smartmatch.

### `AliexpressAffiliateOrderGetRequest`

**Описание**: Класс для получения информации об определенном заказе AliExpress.

### `AliexpressAffiliateOrderListRequest`

**Описание**: Класс для получения списка заказов AliExpress.

### `AliexpressAffiliateHotproductDownloadRequest`

**Описание**: Класс для скачивания горячих продуктов AliExpress.

### `AliexpressAffiliateProductdetailGetRequest`

**Описание**: Класс для получения подробной информации о продукте AliExpress.

### `AliexpressAffiliateHotproductQueryRequest`

**Описание**: Класс для поиска горячих продуктов AliExpress.

### `AliexpressAffiliateFeaturedpromoProductsGetRequest`

**Описание**: Класс для получения данных о продуктах с акциями AliExpress.

### `AliexpressAffiliateFeaturedpromoGetRequest`

**Описание**: Класс для получения данных о рекламных акциях AliExpress.

### `AliexpressAffiliateProductQueryRequest`

**Описание**: Класс для поиска продуктов AliExpress по различным критериям.

### `AliexpressAffiliateCategoryGetRequest`

**Описание**: Класс для получения данных о категориях продуктов AliExpress.

### `AliexpressAffiliateOrderListbyindexRequest`

**Описание**: Класс для получения списка заказов AliExpress, используя индекс.

### `AliexpressAffiliateLinkGenerateRequest`

**Описание**: Класс для генерации ссылок на продукты AliExpress.
