# Модуль `wdl`

## Обзор

Модуль `wdl` предоставляет функцию для загрузки файлов по URL-адресу с использованием утилиты `wget`.

## Оглавление

- [Функции](#функции)
    - [`wget_dl`](#wget_dl)

## Функции

### `wget_dl`

**Описание**: Загружает файл по указанному URL-адресу, используя утилиту `wget`.

**Параметры**:
- `url` (str): URL-адрес файла для загрузки.

**Возвращает**:
- `str | tuple[str, str]`: В случае успешной загрузки возвращает имя загруженного файла. В случае ошибки возвращает кортеж `("error", filename)`, где `filename` - имя файла, которое планировалось сохранить.

**Вызывает исключения**:
- `Exception`: Возникает, если во время загрузки происходит ошибка. В этом случае выводится сообщение об ошибке, а функция возвращает кортеж `("error", filename)`.
