# Документация для `cdata_categories_workstatios.json`

## Обзор

Файл `cdata_categories_workstatios.json` содержит конфигурации для сценариев категорий рабочих станций, используемых при импорте данных из C-Data.  Представляет собой JSON-объект, включающий в себя разделы со сценариями рабочих станций, таких как HP и DELL.

## Структура файла

### Раздел `scenarios`
Раздел `scenarios` представляет собой объект, содержащий в себе наборы данных для каждого сценария импорта. Ключи этого объекта являются названиями сценариев.

#### Сценарий `"HP I7"`

**Описание**: Конфигурация для рабочих станций HP с процессором I7.

**Свойства**:
- `brand` (str): Производитель рабочей станции (`HP`).
- `url` (str): URL-адрес для получения данных с сайта C-Data.
- `checkbox` (bool): Флаг, указывающий, нужно ли использовать чекбокс (`false`).
- `active` (bool): Флаг, указывающий, активен ли сценарий (`true`).
- `condition` (str): Состояние товара (`new`).
- `presta_categories` (str): Список идентификаторов категорий PrestaShop через запятую (`91,242,246,237,89`).
