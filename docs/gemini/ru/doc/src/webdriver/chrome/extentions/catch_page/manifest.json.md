# Манифест расширения Chrome "URL Sender"

## Обзор

Этот файл `manifest.json` является манифестом для расширения Chrome "URL Sender". Расширение предназначено для отправки URL текущей страницы на локальный сервер.

## Структура

### `manifest_version`

**Описание**: Версия манифеста. Используется для указания версии формата манифеста.

**Значение**: `3`

### `name`

**Описание**: Название расширения.

**Значение**: `URL Sender`

### `version`

**Описание**: Версия расширения.

**Значение**: `1.0`

### `description`

**Описание**: Краткое описание функциональности расширения.

**Значение**: `Sends the URL of the current page to a local server.`

### `action`

**Описание**: Конфигурация действия расширения, в частности, поведение при нажатии на иконку расширения.

#### `default_popup`

**Описание**: HTML файл, который будет отображаться во всплывающем окне при нажатии на иконку расширения.

**Значение**: `popup.html`

### `background`

**Описание**: Настройки фонового скрипта расширения.

#### `service_worker`

**Описание**: Файл фонового скрипта, который будет запущен в качестве service worker.

**Значение**: `background.js`

### `permissions`

**Описание**: Список разрешений, которые необходимы расширению для работы.

**Значение**:
- `activeTab`:  Разрешение на доступ к активной вкладке.
- `storage`: Разрешение на использование API storage для сохранения данных.
