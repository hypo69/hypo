# Задача No 73
## Подсчет дробей в заданном диапазоне
## Условия
Рассмотрим множество дробей, в которых числитель и знаменатель — целые положительные числа. Если мы перечислим дроби меньше 1 с знаменателем, не превышающим 8, в порядке возрастания, то получим:
1/8, 1/7, 1/6, 1/5, 1/4, 2/7, 1/3, 3/8, 2/5, 3/7, 1/2, 4/7, 3/5, 5/8, 2/3, 5/7, 3/4, 4/5, 5/6, 6/7, 7/8

Видно, что между 1/3 и 1/2 имеется 3 дроби.

Сколько дробей содержится между 1/3 и 1/2 в перечне дробей, в которых знаменатель не превышает 12 000?

```python
"""
Модуль для решения задачи 73 Project Euler.
=========================================================================================

Этот модуль подсчитывает количество дробей между 1/3 и 1/2,
где знаменатель не превышает заданного значения.

Функция :func:`count_fractions` вычисляет это количество.

Пример использования
--------------------

.. code-block:: python

    result = count_fractions(12000)
    print(result)  # Выведет количество дробей

"""
from math import gcd
from src.logger.logger import logger # Импорт логгера

def count_fractions(limit: int) -> int:
    """
    Подсчитывает количество дробей между 1/3 и 1/2 с знаменателем не более `limit`.

    :param limit: Максимальное значение знаменателя.
    :return: Количество дробей между 1/3 и 1/2.
    """
    count = 0
    try:
        # Цикл перебирает все возможные знаменатели
        for denominator in range(2, limit + 1):
            # Цикл перебирает все возможные числители
            for numerator in range(denominator // 3 + 1, (denominator + 1) // 2):
                # Код проверяет, является ли дробь несократимой
                if gcd(numerator, denominator) == 1:
                     # Если условие выполняется, увеличивает счетчик
                    count += 1
    except Exception as e:
         # В случае ошибки логирует ее
        logger.error(f"Произошла ошибка при подсчете дробей: {e}")
        return 0
    # Возвращает общее количество подходящих дробей
    return count

if __name__ == "__main__":
    # Устанавливаем предел знаменателя
    limit = 12000
    # Вызываем функцию count_fractions и сохраняет результат
    result = count_fractions(limit)
    # Печатает результат
    print(f"Количество дробей между 1/3 и 1/2 с знаменателем не более {limit}: {result}")
```
