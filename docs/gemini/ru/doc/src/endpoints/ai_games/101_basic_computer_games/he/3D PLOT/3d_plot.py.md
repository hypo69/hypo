# 3D PLOT

## Обзор

Этот модуль реализует простую 3D-графику, рисуя линии между точками, заданными координатами X, Y и Z. Игрок вводит координаты для нескольких точек, которые затем соединяются линиями. График отображается на текстовом экране с использованием только координат X и Y.

## Функции

### `draw_3d_plot`

**Описание**:
Рисует простой трехмерный график, соединяя точки линиями на текстовом экране.

**Параметры**:
- Нет параметров.

**Возвращает**:
- Нет возвращаемого значения.

**Вызывает исключения**:
- `ValueError`: Если ввод пользователя не является числом.

**Детали реализации:**
1. **Инициализация экрана:**
   - Создается пустой текстовый экран размером `WIDTH` x `HEIGHT`.
2. **Ввод количества точек:**
   - Запрашивает у пользователя количество точек (от 1 до 10).
   - Проверяет ввод на корректность (целое число от 1 до 10).
3. **Инициализация списков координат:**
   - Создаются пустые списки `x_coords`, `y_coords`, `z_coords` для хранения координат.
4. **Ввод координат:**
   - В цикле запрашиваются координаты X, Y и Z для каждой точки.
   - Проверяет ввод на корректность (число).
5. **Рисование линий:**
   - В цикле проходит по каждой паре соседних точек.
   - Вычисляет разницу координат `delta_x` и `delta_y`.
   - Определяет количество шагов для рисования линии, берется максимальное значение из `abs(delta_x)` и `abs(delta_y)`.
   - Вычисляет приращения для `x_increment` и `y_increment` для каждого шага.
   - В цикле рисует линию, вычисляя текущие координаты и ставя символ `*` на экране, если координаты находятся в пределах экрана.
6. **Вывод экрана:**
   - Выводит текущее состояние экрана в консоль.

```python
def draw_3d_plot():
    """
    Args:
        None

    Returns:
        None

    Raises:
        ValueError: Если ввод пользователя не является числом.
    """
    # יצירת מסך טקסט ריק
    screen = [[' ' for _ in range(WIDTH)] for _ in range(HEIGHT)]

    # קבלת מספר הנקודות מהמשתמש
    while True:
        try:
            num_points = int(input("הזן את מספר הנקודות (עד 10): "))
            if 1 <= num_points <= 10:
                break
            else:
                print("מספר הנקודות צריך להיות בין 1 ל-10.")
        except ValueError:
            print("אנא הזן מספר שלם.")

    # אתחול רשימות לקואורדינטות X, Y ו-Z
    x_coords = []
    y_coords = []
    z_coords = []

    # קבלת קואורדינטות עבור כל נקודה
    for i in range(num_points):
        while True:
            try:
                x = float(input(f"הזן קואורדינטת X עבור נקודה {i + 1}: "))
                y = float(input(f"הזן קואורדינטת Y עבור נקודה {i + 1}: "))
                z = float(input(f"הזן קואורדינטת Z עבור נקודה {i + 1}: "))
                x_coords.append(x)
                y_coords.append(y)
                z_coords.append(z)
                break
            except ValueError:
                print("אנא הזן מספרים.")


    # ציור קווים בין הנקודות הסמוכות
    for i in range(num_points - 1):
        # חישוב ההפרש בין קואורדינטות ה-X וה-Y של הנקודות
        delta_x = x_coords[i+1] - x_coords[i]
        delta_y = y_coords[i+1] - y_coords[i]

        # קביעת מספר הצעדים לצורך ציור הקו
        steps = max(abs(delta_x), abs(delta_y))

        # חישוב השינוי בכל צעד עבור X ו-Y
        x_increment = delta_x / steps if steps != 0 else 0
        y_increment = delta_y / steps if steps != 0 else 0

        # ציור הקו צעד אחר צעד
        for step in range(int(steps) + 1):
            # חישוב קואורדינטות נוכחיות על הקו
            current_x = x_coords[i] + step * x_increment
            current_y = y_coords[i] + step * y_increment

            # בדיקה האם הנקודה בתוך גבולות המסך
            if 0 <= int(current_x) < WIDTH and 0 <= int(current_y) < HEIGHT:
                # ציור נקודה על המסך
                screen[int(current_y)][int(current_x)] = '*'


    # הדפסת המסך
    for row in screen:
        print("".join(row))
```
