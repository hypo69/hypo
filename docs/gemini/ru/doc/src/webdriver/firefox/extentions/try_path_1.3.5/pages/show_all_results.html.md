# Файл: show_all_results.html

## Обзор

Этот файл представляет собой HTML-страницу, которая используется для отображения результатов выполнения XPath-запросов в расширении Tryxpath для Firefox. Он содержит структуру для вывода информации о запросе, контексте и результатах, а также предоставляет возможность экспорта этих данных.

## Оглавление

1.  [Экспорт ссылок](#экспорт-ссылок)
2.  [Информация](#информация)
3.  [Контекстная информация](#контекстная-информация)
4.  [Основная информация](#основная-информация)

## Экспорт ссылок

Этот раздел содержит ссылки для экспорта данных в разных форматах.

*   **Plain text**: Экспорт данных в виде обычного текста.
*   **Some values are converted by JSON.stringify.**: Экспорт данных, где некоторые значения преобразованы с помощью `JSON.stringify`.

## Информация

В этом разделе представлена общая информация о текущем запросе.

*   **Message**: Сообщение о выполнении запроса.
*   **Title**: Заголовок страницы, на которой выполнялся запрос.
*   **URL**: URL страницы, на которой выполнялся запрос.
*   **frameId**: Идентификатор фрейма, в котором выполнялся запрос.

## Контекстная информация

Этот раздел отображает детали контекста, в котором выполнялся запрос.

*   **Method**: Метод, использованный для запроса.
*   **Expression**: XPath-выражение, использованное в запросе.
*   **Specified resultType**: Указанный тип результата.
*   **resultType**: Фактический тип результата.
*   **Resolver**: Используемый разрешитель.

### Контекстные детали

*   Таблица с дополнительными деталями контекста.

## Основная информация

В этом разделе представлена основная информация о результатах запроса.

*   **Method**: Метод, использованный для запроса.
*   **Expression**: XPath-выражение, использованное в запросе.
*   **Specified resultType**: Указанный тип результата.
*   **resultType**: Фактический тип результата.
*   **Resolver**: Используемый разрешитель.
*   **Count**: Количество найденных элементов.

### Основные детали

*   Таблица с дополнительными деталями основных результатов.
