# Модуль login.py

## Обзор

Модуль `login.py` содержит функцию `login`, предназначенную для входа на сайт AliExpress через веб-драйвер.  Функция взаимодействует с объектом класса `Supplier`,  получая от него драйвер браузера и локаторы для элементов страницы.

## Функции

### `login`

**Описание**: Функция для входа на AliExpress через веб-драйвер.

**Параметры**:

- `s` (`Supplier`): Объект класса `Supplier` с запущенным драйвером.

**Возвращает**:

- `bool`: `True`, если вход успешен, иначе возвращает `False` (в текущей реализации всегда возвращает `True`).


**Обработка исключений**:

- Не содержит блоков обработки исключений (`ex`).


**Детали**:

Функция `login` осуществляет следующие действия:

1. Получает веб-драйвер и локаторы из объекта `s`.
2. Переходит на страницу `https://www.aliexpress.com`.
3. Принимает куки (если необходимо).
4. Находит и кликает элемент для открытия формы входа.
5. Находит поля для ввода электронной почты и пароля.
6. Находит и нажимает кнопку входа.
7. Возвращает `True`.
