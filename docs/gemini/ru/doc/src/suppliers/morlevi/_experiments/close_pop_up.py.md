# Модуль `close_pop_up`

## Обзор

Модуль предназначен для проверки локатора закрытия всплывающего окна на сайте поставщика `Morlevi`.

## Подробней

Модуль `close_pop_up` используется для автоматизации процесса закрытия всплывающих окон на сайте `Morlevi`. Это позволяет продолжить работу с сайтом без ручного вмешательства для закрытия рекламных или иных окон. Модуль является частью экспериментов по автоматизации работы с поставщиками и может быть использован для улучшения стабильности и надежности сбора данных.
