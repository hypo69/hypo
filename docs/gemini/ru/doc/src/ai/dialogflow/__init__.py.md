# Модуль `src.ai.dialogflow`

## Обзор

Модуль `src.ai.dialogflow` предназначен для интеграции с сервисом Dialogflow.
Этот модуль содержит общую настройку и может включать в себя классы и функции для работы с Dialogflow API.

## Переменные

### `MODE`

**Описание**:
Глобальная переменная, определяющая режим работы модуля.

**Значение**:
- `'dev'`: Указывает на то, что модуль работает в режиме разработки.
