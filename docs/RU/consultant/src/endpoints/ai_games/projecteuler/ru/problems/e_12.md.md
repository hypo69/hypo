# Задача No 12
## Highly divisible triangular number
## Условия
Треугольные числа получаются путем сложения последовательных натуральных чисел. Так, 7-е треугольное число равно 1 + 2 + 3 + 4 + 5 + 6 + 7 = 28. Первые десять треугольных чисел:

1, 3, 6, 10, 15, 21, 28, 36, 45, 55, ...

Давайте перечислим делители первых семи треугольных чисел:

1: 1
3: 1,3
6: 1,2,3,6
10: 1,2,5,10
15: 1,3,5,15
21: 1,3,7,21
28: 1,2,4,7,14,28

Замечаем, что 28 - первое треугольное число, имеющее более пяти делителей.

Каково первое треугольное число, имеющее более пятисот делителей?

```python
"""
Модуль для решения задачи 12 Project Euler: Highly divisible triangular number.
==============================================================================

Этот модуль определяет функцию для нахождения первого треугольного числа,
имеющего заданное количество делителей.

Пример использования
--------------------

.. code-block:: python

    result = find_triangular_number_with_divisors(500)
    print(result)

"""
from src.logger.logger import logger
from math import sqrt


def count_divisors(number: int) -> int:
    """
    Подсчитывает количество делителей числа.

    :param number: Целое число, для которого необходимо подсчитать делители.
    :return: Количество делителей числа.
    """
    count = 0
    # Код выполняет итерацию от 1 до квадратного корня числа для подсчета делителей
    for i in range(1, int(sqrt(number)) + 1):
        if number % i == 0:
            count += 2
            # Проверка, является ли число квадратом
            if i * i == number:
                count -= 1
    return count

def find_triangular_number_with_divisors(num_divisors: int) -> int:
    """
    Находит первое треугольное число, имеющее заданное количество делителей.

    :param num_divisors: Минимальное количество делителей, которое должно иметь треугольное число.
    :return: Первое треугольное число, удовлетворяющее условию.
    """
    try:
        number = 1
        i = 1
        # Код выполняет поиск треугольного числа, пока количество делителей меньше требуемого
        while True:
            i += 1
            number += i
            if count_divisors(number) > num_divisors:
                return number
    except Exception as ex:
        logger.error('Произошла ошибка при вычислении треугольного числа', ex)
        return None
```
