# Оглавление для "101 Basic Computer Games"

## Обзор

Этот документ представляет собой оглавление для сборника "More Basic Computer Games", содержащего 101 игру, написанную на языке BASIC. Для каждой игры предоставляется краткое описание и ссылка на соответствующий файл на GitHub.
