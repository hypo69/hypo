# Задача No 845
## Числовые разделения

## Условия

Пусть s(n) - количество способов, которыми целое число n можно разбить на слагаемые, которые являются квадратами целых чисел.

Например,
* 1 = 1²
* 2 = 1² + 1²
* 3 = 1² + 1² + 1²
* 4 = 2² = 1² + 1² + 1² + 1²
* 5 = 2² + 1² = 1² + 1² + 1² + 1² + 1²
Таким образом, s(1) = 1, s(2) = 1, s(3) = 1, s(4) = 2 и s(5) = 2.

Пусть S(n) = Σ s(i) для 1 ≤ i ≤ n.

В частности, S(5) = 1 + 1 + 1 + 2 + 2 = 7.
Также, S(100) = 1040.

Найдите S(10<sup>7</sup>).
