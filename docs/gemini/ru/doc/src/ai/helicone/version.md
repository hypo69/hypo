# Модуль hypotez/src/ai/helicone/version.py

## Обзор

Этот модуль содержит константы и переменные, связанные с версией проекта и настройками. Он загружает данные из файла `settings.json` и возвращает значения для различных метаданных проекта.

## Переменные

### `MODE`

**Описание**: Строковая константа, хранящая режим работы проекта. В данном случае, значение `'dev'`.

**Значение**: `'dev'`


### `__project_name__`

**Описание**: Строковая переменная, содержащая имя проекта.  Значение берется из файла `settings.json` или имеет значение `'hypotez'` по умолчанию.

**Тип**: `str`

**Значение**: Значение из `settings.json` или `'hypotez'`


### `__version__`

**Описание**: Строковая переменная, содержащая версию проекта.  Значение берется из файла `settings.json` или пустая строка по умолчанию.

**Тип**: `str`

**Значение**: Значение из `settings.json` или пустая строка


### `__doc__`

**Описание**: Строковая переменная, содержащая документацию к проекту. По умолчанию пустая строка.

**Тип**: `str`

**Значение**: Пустая строка


### `__details__`

**Описание**: Строковая переменная, содержащая дополнительные детали проекта. По умолчанию пустая строка.

**Тип**: `str`

**Значение**: Пустая строка


### `__author__`

**Описание**: Строковая переменная, содержащая имя автора проекта. Значение берется из файла `settings.json` или пустая строка по умолчанию.

**Тип**: `str`

**Значение**: Значение из `settings.json` или пустая строка


### `__copyright__`

**Описание**: Строковая переменная, содержащая информацию об авторских правах. Значение берется из файла `settings.json` или пустая строка по умолчанию.

**Тип**: `str`

**Значение**: Значение из `settings.json` или пустая строка


### `__cofee__`

**Описание**: Строковая переменная, содержащая ссылку для поддержки разработчика. Значение берется из файла `settings.json` или строка со ссылкой по умолчанию.

**Тип**: `str`

**Значение**: Значение из `settings.json` или ссылка по умолчанию


### `settings`

**Описание**: Словарь, хранящий настройки проекта, загруженные из файла `settings.json`.

**Тип**: `dict`

**Значение**: Словарь, полученный из `settings.json` или `None`


## Функции

### `None`

В модуле нет функций.


## Обработка исключений

### `FileNotFoundError`

**Описание**: Исключение, которое возникает, если файл `settings.json` не найден.

### `json.JSONDecodeError`

**Описание**: Исключение, которое возникает, если файл `settings.json` содержит невалидный JSON.
