# Задача No 684
## Сумма S(n)

## Условия

Определим функцию *S(n)* как сумму цифр числа *n* в десятичной системе счисления.

Пусть *F<sub>i</sub>* - это *i*-е число Фибоначчи, где *F<sub>1</sub>* = 1, *F<sub>2</sub>* = 1 и *F<sub>i</sub>* = *F<sub>i-1</sub>* + *F<sub>i-2</sub>* для *i* > 2.

Найдите ∑ *S*(*F<sub>i</sub>*) для 3 ≤ *i* ≤ 90.
