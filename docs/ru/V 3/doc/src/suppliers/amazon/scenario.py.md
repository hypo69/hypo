# src.suppliers.amazon.scenario

## Обзор

Модуль `src.suppliers.amazon.scenario` предназначен для сбора информации о товарах с сайта Amazon, начиная со страниц категорий. Он содержит функции для извлечения списка категорий и товаров, а также для обработки страниц товаров и передачи данных о них.

## Подробней

Этот модуль является частью системы сбора данных о товарах от поставщиков, в частности, с Amazon. Он автоматизирует процесс получения информации о товарах, имитируя действия пользователя в браузере. Основная задача модуля - предоставить актуальные данные о товарах для дальнейшей обработки и интеграции с другими системами, например, PrestaShop.

Модуль включает в себя следующие шаги:
1.  Получение списка категорий товаров с сайта поставщика.
2.  Сбор списка товаров, представленных в каждой категории.
3.  Передача URL каждого товара в функцию `grab_product_page()` для обработки и извлечения детальной информации о товаре.
4.  Использование класса `Product` для организации и хранения полученных данных о товаре.
