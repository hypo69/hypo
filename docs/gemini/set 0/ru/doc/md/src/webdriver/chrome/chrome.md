# Модуль hypotez/src/webdriver/chrome/chrome.py

## Обзор

Модуль `hypotez/src/webdriver/chrome/chrome.py` предоставляет реализацию драйвера Selenium для браузера Chrome. Он интегрирует настройки, определенные в файле конфигурации `chrome.json`, такие как пользовательский агент и настройки профиля браузера, для гибкого и автоматизированного взаимодействия с браузером.

## Классы

### `Chrome`

**Описание**: Класс `Chrome` расширяет класс `webdriver.Chrome` и реализует сингултон-паттерн для обеспечения единственного экземпляра драйвера Chrome.

**Методы**:

#### `__new__`

**Описание**: Статический метод `__new__` гарантирует создание только одного экземпляра класса `Chrome`. Если экземпляр уже существует, вызывается метод `window_open()` для открытия нового окна.

**Возвращает**: Экземпляр класса `Chrome`.

#### `__init__`

**Описание**: Инициализирует драйвер Chrome с указанными опциями и профилем.

**Параметры**:

- `user_agent` (Optional[str]): Строка пользовательского агента, используемая для имитации браузера. По умолчанию генерируется случайный пользовательский агент.

**Вызывает исключения**:

- `Exception`: Возникает при ошибках во время инициализации драйвера. В этом случае выполняется логгирование ошибки, и метод возвращает None.


#### `_payload`

**Описание**: Загружает исполняемый модуль для локализации и сценариев JavaScript.


**Возвращает**: `None`

- str: Нормализованный путь, с замещенными переменными окружения.

- str: Пустая строка если входной путь является пустой строкой.
