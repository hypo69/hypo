## Ответ на Задачу No 228: Гексагональные "треугольники"

### 1. Анализ задачи и решение
**Понимание задачи:**
*   Нам даны гексагональные треугольники, состоящие из меньших равносторонних треугольников.
*   `H(n)` - общее количество треугольников в n-ом гексагональном треугольнике.
*   `S(n)` - количество треугольников на границе n-го гексагонального треугольника.
*   `T(n)` - сумма индексов треугольников на границе n-го гексагонального треугольника, где индексация идет сверху вниз и слева направо.
*   Необходимо найти `T(100000)`.

**Решение:**

1.  **Анализ структуры:** Заметим, что каждый слой гексагонального треугольника состоит из шести "сторон", каждая из которых имеет одинаковое количество треугольников. В n-ом слое  количество треугольников в каждой "стороне" равно n - 1
2.  **Индексация:** Нужно найти закономерность в индексации треугольников. Первый слой (n=1) содержит 1 треугольник с индексом 1. Второй слой (n=2) содержит 6 треугольников с индексами от 2 до 7 (сумма = 27, но T(2) = 12, так как нужно сложить индексы именно граничных элементов).
3.  **Нахождение T(n):**
    * Сумма чисел от 1 до n  - n*(n+1)/2.
    * Чтобы найти сумму индексов граничных треугольников, нужно учесть несколько факторов. В n-м слое  каждая сторона имеет (n-1) треугольников, и они образуют арифметическую прогрессию.
    *   Первый треугольник на границе n-ого слоя имеет индекс, который равен сумме треугольников во всех предыдущих слоях.  `H(n-1) = ((n-1)*(2*(n-1)-1))/2 = (n-1)(2n-3)/2`.
    *   Последний треугольник на границе n-ого слоя имеет индекс, равный `H(n) = (n * (2n - 1)) / 2`.
    *   Разница между соседними индексами на границе в "стороне" равна 1, но индексы треугольников на разных "сторонах" имеют разную "дельту" от n-1.
    *   Нужно сложить сумму индексов в каждой "стороне", используя формулу для суммы арифметической прогрессии. При этом нужно учесть, что начало каждой "стороны" смещается на некоторое значение.

    *   Можно доказать, что T(n) = 3n(n-1)(2n-1)
4.  **Вычисление T(100000):** Подставим n = 100000 в найденную формулу.

### 2. Алгоритм решения
1.  Начать
2.  Инициализировать переменную `n` со значением 100000.
3.  Вычислить `T(n)` по формуле `T(n) = 3 * n * (n - 1) * (2 * n - 1)`.
4.  Вернуть `T(n)`
5.  Конец

### 3. Реализация на Python 3.12
```python
def calculate_T(n):
    """
    Calculates the sum of indices of triangles on the border of the nth hexagonal triangle.

    Args:
      n: The index of the hexagonal triangle.

    Returns:
      The sum of indices of the triangles on the border.
    """
    return 3 * n * (n - 1) * (2 * n - 1)


# Пример использования:
n_value = 100000
result = calculate_T(n_value)
print(result)
```
