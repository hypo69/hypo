# Модуль `hypotez/src/suppliers/aliexpress/__init__.py`

## Обзор

Этот модуль предоставляет инициализацию для поставщика данных AliExpress. Он импортирует классы и функции из подмодулей, предоставляя доступ к функциональности для работы с данными AliExpress.

## Импорты

Этот модуль импортирует следующие элементы:

* `Aliexpress`: Класс, вероятно, представляющий основную сущность для работы с поставщиком.
* `AliApi`: Класс или функция для взаимодействия с API AliExpress.
* `AliRequests`: Класс или функция для управления запросами к API AliExpress.
* `AliCampaignEditor`: Класс для редактирования кампаний.
* `ProductHTMLGenerator`, `CategoryHTMLGenerator`, `CampaignHTMLGenerator`: Классы для генерации HTML-представлений продуктов, категорий и кампаний.


## Постоянные значения

Модуль определяет константу:

* `MODE`: Вероятно, строковая константа, определяющая режим работы (например, 'dev', 'prod').
