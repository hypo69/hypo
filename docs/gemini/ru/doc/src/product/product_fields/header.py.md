# Модуль `header.py`

## Обзор

Модуль `header.py` предназначен для определения настроек проекта, загрузки конфигурационных файлов (settings.json, README.MD) и предоставления базовых метаданных о проекте. Он также содержит функцию для определения корневой директории проекта.

## Константы
### `MODE`
- **Описание**: Определяет режим работы приложения (например, 'dev' для разработки).
- **Тип**: `str`
- **Значение**: `dev`

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла, и останавливается на первой директории, содержащей любой из указанных маркерных файлов.

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или каталогов для идентификации корневой директории проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена, иначе путь к директории, где расположен скрипт.

## Переменные

### `__root__`
- **Описание**: Путь к корневой директории проекта.
- **Тип**: `Path`
- **Инициализация**: Путь к корневой директории проекта, определенный функцией `set_project_root`.

### `settings`
- **Описание**: Словарь с настройками проекта, загруженными из `settings.json`.
- **Тип**: `dict | None`
- **Инициализация**: Загружается из файла `settings.json`, расположенного в корне проекта. Может быть `None`, если файл не найден или не может быть прочитан.

### `doc_str`
- **Описание**: Строка с содержимым файла документации `README.MD`.
- **Тип**: `str | None`
- **Инициализация**: Загружается из файла `README.MD`, расположенного в корне проекта. Может быть `None`, если файл не найден или не может быть прочитан.

### `__project_name__`
- **Описание**: Название проекта.
- **Тип**: `str`
- **Инициализация**: Значение из `settings.json` (ключ `project_name`), по умолчанию `'hypotez'`.

### `__version__`
- **Описание**: Версия проекта.
- **Тип**: `str`
- **Инициализация**: Значение из `settings.json` (ключ `version`), по умолчанию `''`.

### `__doc__`
- **Описание**: Содержимое файла `README.MD`.
- **Тип**: `str`
- **Инициализация**: Значение из переменной `doc_str`, по умолчанию `''`.

### `__details__`
- **Описание**: Детали проекта. В текущей версии всегда пустая строка.
- **Тип**: `str`
- **Значение**: `''`

### `__author__`
- **Описание**: Автор проекта.
- **Тип**: `str`
- **Инициализация**: Значение из `settings.json` (ключ `author`), по умолчанию `''`.

### `__copyright__`
- **Описание**: Информация об авторских правах.
- **Тип**: `str`
- **Инициализация**: Значение из `settings.json` (ключ `copyrihgnt`), по умолчанию `''`.

### `__cofee__`
- **Описание**: Сообщение с просьбой поддержать разработчика чашкой кофе.
- **Тип**: `str`
- **Инициализация**: Значение из `settings.json` (ключ `cofee`), по умолчанию "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69".
