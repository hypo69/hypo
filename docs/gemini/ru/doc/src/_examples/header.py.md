## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла и двигаясь вверх, останавливаясь на первой директории, содержащей любой из указанных файлов-маркеров.

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или директорий, которые идентифицируют корневую директорию проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена. В противном случае - путь к директории, где находится скрипт.

## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта.

### `settings`

**Описание**: Словарь с настройками проекта, загруженными из `settings.json`.

### `doc_str`

**Описание**: Строка с документацией проекта, загруженной из `README.MD`.

### `__project_name__`

**Описание**: Название проекта. По умолчанию `hypotez`.

### `__version__`

**Описание**: Версия проекта. По умолчанию `''`.

### `__doc__`

**Описание**: Документация проекта. По умолчанию `''`.

### `__details__`

**Описание**: Детали проекта. По умолчанию `''`.

### `__author__`

**Описание**: Автор проекта. По умолчанию `''`.

### `__copyright__`

**Описание**: Информация об авторских правах проекта. По умолчанию `''`.

### `__cofee__`

**Описание**: Сообщение с предложением поддержать разработчика чашкой кофе. По умолчанию `"Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69"`.
