# Модуль hypotez/src/suppliers/morlevi/header.py

## Обзор

Этот модуль содержит функции для определения корневого каталога проекта и загрузки настроек из файла `settings.json` и документации из файла `README.MD`.  Он также предоставляет переменные, содержащие информацию о проекте, такие как имя, версия, описание и автор.


## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная от текущей директории и идя вверх по древу директорий, пока не найдёт директорию, содержащую один из файлов/каталогов, указанных в параметре `marker_files`.

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или каталогов, которые указывают на корневой каталог проекта. По умолчанию содержит ('pyproject.toml', 'requirements.txt', '.git').


**Возвращает**:

- `Path`: Путь к корневому каталогу проекта, если он найден, в противном случае путь к директории, где находится скрипт.


**Вызывает исключения**:
- Нет

### Функции с использованием try...exept

**Описание**: Функции для чтения файла настроек `settings.json` и файла документации `README.MD` и обработки потенциальных исключений:

**Параметры**:
Не применимо

**Возвращает**:
Не применимо


**Вызывает исключения**:

- `FileNotFoundError`: Возникает, если файл настроек или файл документации не найден.
- `json.JSONDecodeError`: Возникает, если содержимое файла настроек не является валидным JSON.

## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта, полученному с помощью функции `set_project_root()`.

**Тип**: `Path`


### `settings`

**Описание**: Словарь настроек проекта, загруженный из файла `settings.json`. Может быть `None`, если файл не найден или повреждён.

**Тип**: `dict` или `None`


### `doc_str`

**Описание**: Строковое представление документации, загруженное из файла `README.MD`. Может быть `None`, если файл не найден или повреждён.

**Тип**: `str` или `None`


### `__project_name__`

**Описание**: Имя проекта, полученное из настроек, или по умолчанию 'hypotez'.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта, полученная из настроек, или по умолчанию пустая строка.

**Тип**: `str`


### `__doc__`

**Описание**: Строковое описание проекта, загруженное из файла `README.MD` или пустая строка.

**Тип**: `str`


### `__details__`

**Описание**: Подробное описание проекта. По умолчанию пустая строка.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта, полученный из настроек, или по умолчанию пустая строка.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права проекта, полученные из настроек, или по умолчанию пустая строка.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка на платформа для поддержки разработчиков (пожертвования на кофе)

**Тип**: `str`
