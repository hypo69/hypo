**Название урока:** Собираем информацию о компьютере

**Описание:**

Этот урок демонстрирует, как получить различную информацию о компьютере с помощью Python. В частности, он показывает, как получить следующие данные:

*   Операционная система
*   Версия операционной системы
*   Имя компьютера
*   Архитектура компьютера
*   Процессор
*   Количество ядер процессора
*   Общий объем оперативной памяти (RAM)
*   Доступный объем оперативной памяти (RAM)
*   Общий объем диска

**Используемые модули:**

*   **`platform`**: Предоставляет доступ к идентификаторам основной платформы. Этот модуль используется для получения такой информации, как операционная система, ее версия, имя компьютера и архитектура.
*   **`psutil`**:  "psutil" (process and system utilities) - это кросс-платформенный модуль Python для извлечения информации о запущенных процессах и использовании ресурсов системы (таких как процессор, память, диски, сеть и т. д.).
    Этот модуль используется для получения информации о процессоре (количество ядер), оперативной памяти и дисковом пространстве.

**Установка модуля `psutil`:**

Перед использованием модуля `psutil` необходимо его установить, выполнив команду:

```bash
pip install psutil
```

**Код:**

```python
import platform
import psutil

def get_system_info():
    info = {
        "Операционная система": platform.system(),
        "Версия ОС": platform.version(),
        "Имя компьютера": platform.node(),
        "Архитектура": platform.architecture(),
        "Процессор": platform.processor(),
        "Количество ядер": psutil.cpu_count(logical=True),
        "Объем оперативной памяти": round(psutil.virtual_memory().total / (1024 ** 3), 2),
        "Объем свободной памяти": round(psutil.virtual_memory().available / (1024 ** 3), 2),
        "Диск": psutil.disk_usage('/').total / (1024 ** 3)
    }
    return info


if __name__ == "__main__":
    system_info = get_system_info()
    for key, value in system_info.items():
        print(f"{key}: {value}")
```

**Объяснение кода:**

1.  **Импорт модулей:** Импортируем модули `platform` и `psutil`.
2.  **Функция `get_system_info()`:**
    *   Создается словарь `info`, который будет содержать собранную информацию.
    *   `platform.system()`, `platform.version()`, `platform.node()`, `platform.architecture()` и `platform.processor()` возвращают соответственно операционную систему, ее версию, имя компьютера, архитектуру и процессор.
    *   `psutil.cpu_count(logical=True)` возвращает количество логических ядер процессора.
    *   `psutil.virtual_memory().total` возвращает общий объем оперативной памяти в байтах.
    *   `psutil.virtual_memory().available` возвращает объем доступной оперативной памяти в байтах.
    *   `psutil.disk_usage('/').total` возвращает общий объем диска в байтах, начиная с корневого каталога ("/").
    *   Объемы памяти и диска делятся на `1024 ** 3` (1024 в третьей степени) для преобразования в гигабайты.
    *   `round(..., 2)` округляет значения до двух знаков после запятой.
    *   Функция возвращает словарь `info`.
3.  **Условие `if __name__ == "__main__":`:**
    *   Этот блок кода выполняется только при запуске скрипта непосредственно как основной программы, а не при импорте его как модуля.
    *   Вызываем функцию `get_system_info()` и сохраняем возвращаемый словарь в переменную `system_info`.
    *   Проходимся циклом `for` по элементам словаря `system_info`, используя метод `items()`, который возвращает пары ключ-значение.
    *   Для каждой пары ключ-значение печатаем информацию в формате "ключ: значение" с использованием f-строк.

**Замечания:**

*   Код использует f-строки для форматирования вывода (доступны с Python 3.6).
*   Конструкция `if __name__ == '__main__':` гарантирует, что код, связанный с получением и выводом системной информации, будет выполнен только при запуске скрипта напрямую, а не при его импорте.
*   В коде используется "/" в `disk_usage()` , который является корневым каталогом для большинства операционных систем, но может быть изменен для получения информации о другом диске.
*  Код конвертирует байты в гигабайты с помощью деления на `1024**3`, что соответствует `1024*1024*1024`.
