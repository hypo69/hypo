## Обзор

Этот файл содержит вопрос и ответ, относящийся к конфигурационным файлам. Он объясняет, почему конфигурационные файлы называются по имени модуля (например, `suppliers.json`) вместо общего имени `config.json`.

## Вопрос

### Почему конфигурационные файлы называются по имени модуля (например: `suppliers.json`)?

**Описание**: Задает вопрос о причинах именования конфигурационных файлов по имени модуля, а не общим именем `config.json`.

## Ответ

### Имена файлов сделаны для удобства понимания моделями ИИ.

**Описание**: Отвечает на поставленный вопрос, объясняя, что такое именование помогает моделям искусственного интеллекта лучше понимать и обрабатывать эти файлы.
