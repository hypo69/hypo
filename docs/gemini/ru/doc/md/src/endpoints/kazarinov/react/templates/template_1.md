# Шаблон HTML для отображения продуктов

## Обзор

Этот HTML-шаблон (`template_1.html`) предназначен для отображения списка продуктов с изображениями, названиями, описаниями и общей стоимостью. Он использует Bootstrap для стилизации и адаптирован под различные устройства.

## Стили

Шаблон использует внешний CSS-файл Bootstrap для стилизации элементов страницы.  Встроенные стили настраивают цветовую схему, размер шрифтов, отступы и визуальное оформление элементов (карточки продуктов, цен, подвала).

## Структура страницы

Страница состоит из следующих основных секций:

* **Заголовок страницы:**  Отображает заголовок, определенный переменной `title`.
* **Описание:**  Отображает описание, определенное переменной `description`.
* **Список продуктов:** Динамически генерирует карточки каждого продукта из списка `products`.  Каждая карточка содержит:
    * Название продукта (`product.product_title`).
    * Изображение продукта (`product.image_local_saved_path`).
    * Описание продукта (`product.product_description`).
* **Подвал страницы:** Отображает общую стоимость (`price`) в указанной валюте (`currency`).

## Переменные шаблона

Шаблон использует следующие переменные:

* **`language`**: Язык страницы.
* **`title`**: Заголовок страницы.
* **`description`**: Описание страницы.
* **`products`**: Список объектов продуктов, каждый с полями `product_title`, `image_local_saved_path`, `product_description`.
* **`price`**: Общая стоимость.
* **`currency`**: Валюта.

**Обратите внимание**:  Шаблон предполагает использование Django шаблонов и доступа к данным из соответствующего контекста.  Для отображения изображений необходимо указать правильный путь к локальным изображениям (`product.image_local_saved_path`).
