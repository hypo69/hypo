# Модуль `banners_grabber.py`

## Обзор

Модуль `banners_grabber.py` предназначен для сбора баннеров с сайта KSP (предположительно, поставщика KSP). В текущей версии модуль содержит только заглушку функции `get_banners`, которая всегда возвращает `True`.

## Подробней

Модуль является частью подсистемы, отвечающей за получение данных от поставщиков, в частности, от KSP. Он может быть использован для автоматического сбора рекламных баннеров с сайта поставщика с целью их дальнейшего использования в маркетинговых кампаниях или для анализа.
