# Модуль hypotez/src/webdriver/bs/bs.py

## Обзор

Модуль `bs.py` предоставляет класс `BS` для парсинга HTML-контента, полученного из файла или URL, с использованием библиотек `BeautifulSoup` и `lxml`.  Он позволяет искать элементы на странице по различным локаторам (например, по ID, CSS классу, тегу).

## Классы

### `BS`

**Описание**: Класс `BS` предназначен для работы с HTML-контентом. Он позволяет загружать контент из файлов и URL, а также выполнять поиск элементов на странице по заданным локаторам.

**Атрибуты**:

- `html_content` (str): Содержимое HTML-страницы.


**Методы**:

#### `__init__(self, url: str | None = None)`

**Описание**: Конструктор класса `BS`.  Инициализирует атрибут `html_content`.

**Параметры**:

- `url` (str | None, optional): URL или путь к файлу с HTML-контентом. По умолчанию `None`.


#### `get_url(self, url: str)`

**Описание**: Загружает HTML-контент из файла или URL.

**Параметры**:

- `url` (str): Путь к файлу или URL-адрес.


**Возвращает**:

- `bool`: `True`, если загрузка успешна; `False`, если возникла ошибка.

**Обрабатывает исключения**:

- `Exception`: Обрабатывает исключения при чтении файла. Выводит ошибку в лог.
- `requests.RequestException`: Обрабатывает исключения при получении данных с URL. Выводит ошибку в лог.
- Возвращает `None` при ошибках валидации пути и URL.


#### `execute_locator(self, locator: SimpleNamespace | dict, url: str = None)`

**Описание**: Выполняет поиск элементов на странице по заданному локатору.

**Параметры**:

- `locator` (SimpleNamespace | dict): Объект, содержащий данные локатора. Должен содержать поля `attribute`, `by`, `selector`
- `url` (str, optional): URL, если необходимо загрузить страницу. По умолчанию `None`.


**Возвращает**:

- `list`: Список найденных элементов (lxml.etree._Element). `None` при ошибке.

**Обрабатывает исключения**:
- `Exception`: Обрабатывает все другие возможные исключения. Выводит ошибку в лог.

## Примеры

```python
# Пример использования класса BS для загрузки страницы из файла
url = 'file:///path/to/your/file.html'
bs_object = BS(url)
success = bs_object.get_url(url)
if success:
  elements = bs_object.execute_locator(locator)

# Пример использования класса BS для загрузки страницы из URL
url = 'https://example.com'
bs_object = BS()
success = bs_object.get_url(url)
if success:
  elements = bs_object.execute_locator(locator, url)
```


**Примечание**: В примере заменивте `'path/to/your/file.html'` на фактический путь к файлу.  Обратите внимание на правильный способ задания пути к файлу, в зависимости от операционной системы. Для работы с локальными файлами, необходим префикс `file://`.  Также обратите внимание на использование `locator` как объекта `SimpleNamespace` или `dict`.  Этот пример демонстрирует базовые способы использования модуля.  В реальных приложениях требуется более сложная обработка результатов и ошибок.
