## משחק "קלט-פלט"

### קונספט

משחק "קלט-פלט" נועד כמשימה אינטראקטיבית המשתמשת ב"מכונה קסומה" שהופכת מספרי קלט למספרי פלט לפי כלל מוגדר. התלמידים צריכים לקבוע את הכלל הזה בהתבסס על סדרת נתוני קלט ופלט.

### מכניקה

1.  יוצרים שני עמודות על הלוח או המקרן: "קלט" ו"פלט".
2.  המורה מציב מספר בעמודת "קלט".
3.  ה"מכונה" מעבדת את המספר ומוציאה תוצאה לעמודת "פלט".
4.  התלמידים צריכים לנחש את כלל ההמרה על ידי התבוננות בתוצאות.

### דוגמאות

**1. כלל פשוט**

| קלט | מכונה = ?  | פלט |
| --- | ------------- | --- |
| 4   |               | 7   |
| 0   |               | 3   |
|     |               |    |
|     | תשובה: מכונה = +3 |    |

**2. כלל מורכב יותר**

| קלט | מכונה = ?      | פלט |
| --- | --------------- | --- |
| 4   |                 | 11  |
| 7   |                 | 17  |
| 10  |                 | 23  |
|    | תשובה: מכונה = 2x + 3 |    |

**3. כלל אלגברי**

| קלט | מכונה = ?        | פלט |
| --- | ----------------- | --- |
| 9   |                   | 4   |
| 25  |                   | 6   |
| 4   |                   | 3   |
|    | תשובה: מכונה = √x + 1 |    |

ניתן להתאים את המשחק למגוון ביטויים אלגבריים, המאפשרים לתלמידים לפתח חשיבה אלגברית בצורה משחקית.

## "מחשב 10 סנט"

### קונספט

"מחשב 10 סנט" הוא מודל פשוט ללימוד השיטה הבינארית. בעזרת קופסת קרטון ופילם שקוף, המורה יוצר מכשיר אינטראקטיבי המדמה את פעולת המחשב על בסיס קוד בינארי.

### מכניקה

1.  **יצירה:** ארבעה חורים נחתכים על קופסת קרטון. החורים מכוסים בפילם שקוף וממוספרים 1, 2, 4 ו-8 מימין לשמאל.
2.  **ייצוג בינארי:** חורים פתוחים מייצגים "1", וסגורים מייצגים "0". לכן, שילוב של חורים פתוחים וסגורים מתאים למספר בינארי.
3.  **למידה:** ילדים לומדים להתאים שילובים בינאריים למקבילות העשרוניות שלהם.

### שימוש

המורה פותח או סוגר חורים שונים, ומציג מספרי בינאריים שונים, תוך כדי שהוא מראה את המקבילות העשרוניות על הלוח. בדרך זו, הילדים רואים בצורה ויזואלית איך המערכת הבינארית עובדת, מה שעוזר להם להבין את עקרונות העבודה של מחשבים.

### מספרים בינאריים עבור "מחשב 10 סנט"

הייצוג של מספרים בינאריים עבור "מחשב 10 סנט" במקור מוצג בצורה של אלמנטים גרפיים. כאן נשחזר אותם בטקסט, כאשר `o` מציין חור סגור ו-`#` חור פתוח:

| ייצוג בינארי | מספר עשרוני | ייצוג בינארי | מספר עשרוני |
| ----------- | -------- | ----------- | -------- |
| `o o o o` | 0        | `# o o o`   | 8        |
| `# o o o`   | 1        | `# # o o`   | 9        |
| `o # o o`   | 2        | `o # # o`  | 10       |
| `# # o o`  | 3        | `# # # o`  | 11       |
| `o o # o`  | 4        | `o o # #`  | 12       |
| `# o # o`  | 5        | `# o # #`   | 13       |
| `o # # o`  | 6        | `o # # #`  | 14       |
| `# # # o`  | 7        | `# # # #` | 15       |

## יישום עם Google Gemini

בחלק הקודם של המאמר, סקרנו את מושגי המשחקים וההוראות למודל. כעת, נבחן כיצד זה מיושם באמצעות קוד.

### קוד עבור `GoogleGenerativeAI`

כדי להשלים את התמונה, נביא את הקוד של המחלקה `GoogleGenerativeAI` כפי שהוצג קודם:

```python
import google.generativeai as genai

class GoogleGenerativeAI:
    """
    מחלקה ליצירת אינטראקציה עם מודלים של Google Generative AI.
    """

    MODELS = [
        'gemini-1.5-flash-8b',
        'gemini-2-13b',
        'gemini-3-20b'
    ]

    def __init__(self, api_key: str, system_instruction: str, model_name: str = 'gemini-2-13b'):
        """
        אתחול מודל GoogleGenerativeAI.

        Args:
            api_key (str): מפתח API לגישה ל-Gemini.
            system_instruction (str): הוראה למודל (prompt מערכת).
            model_name (str, optional): שם מודל Gemini בו יש להשתמש. ברירת מחדל: 'gemini-2-13b'.
        """
        self.api_key = api_key
        self.model_name = model_name
        genai.configure(api_key=self.api_key)
        self.model = genai.GenerativeModel(model_name=self.model_name, system_instruction=system_instruction)

    def ask(self, q: str) -> str:
        """
        שליחת שאילתה למודל וקבלת תשובה.

        Args:
            q (str): טקסט השאילתה.

        Returns:
            str: תשובת המודל או הודעת שגיאה.
        """
        try:
            response = self.model.generate_content(q)
            return response.text
        except Exception as ex:
            return f'Error: {str(ex)}'
```

### יישום משחקים באמצעות `GoogleGenerativeAI`

כעת ניצור קוד לשימוש במחלקה זו ליישום המשחקים "קלט-פלט" ו"מחשב 10 סנט":

```python
# הוראות מערכת (כפי שתוארו קודם)

system_instruction_input_output = """
אתה מנוע משחק למשחק מתמטי "קלט-פלט". המשימה שלך היא להמיר מספר שמגיע ב"קלט" בהתאם לכלל נתון ( "מכונה" נסתרת) ולהוציא את התוצאה ב"פלט".

עליך לציית לכללים הבאים:
-   התחל עם כללים פשוטים (למשל, חיבור או חיסור עם מספר שלם).
-   ככל שהמשתמש מתמודד, אתה יכול בהדרגה לעבור לכללים מורכבים יותר (כפל, חילוק, שורש, שילוב של פעולות וכו').
-   אם המשתמש מבקש כלל ספציפי, עליך להשתמש בו.
-   בסוף כל תשובה, עליך לציין בקצרה את סוג הפעולה שבה השתמשת (למשל "מכונה +3" או "מכונה 2x+1").

**פורמט קלט/פלט:**
-   משתמש: `קלט: <מספר>`
-   אתה: `פלט: <תוצאה>, מכונה:<תיאור פעולה>`

**דוגמאות:**

משתמש: `קלט: 5`
אתה: `פלט: 8, מכונה +3`

משתמש: `קלט: 10`
אתה: `פלט: 21, מכונה 2x+1`

משתמש: `קלט: 9`
אתה: `פלט: 3, מכונה שורש ריבועי`

משתמש: `הגדר מכונה x * 2`
אתה: `אוקיי, מכונה: x*2`

**התחל את המשחק עם כלל פשוט כלשהו.**
"""

system_instruction_binary = """
אתה סימולטור "מחשב 10 סנט" ללימוד ילדים על המערכת הבינארית. המשימה שלך היא לייצג מספרים בצורה בינארית באמצעות 4 "נורות" (0 - נורה כבויה, 1 - נורה דולקת).

הנורות שלך מציינות 1, 2, 4 ו-8 במערכת הבינארית (מימין לשמאל).

עליך לציית לכללים הבאים:
-   כאשר משתמש מציין מספר עשרוני, אתה מראה אילו "נורות" צריך להדליק (1) ולכבות (0) כדי להציג את המספר הזה.
-   כאשר משתמש מציין אילו "נורות" דולקות, אתה אומר איזה מספר עשרוני הן מייצגות.
-   עליך להשתמש בפורמט `0000` לייצוג הבינארי, כאשר הביט הימני ביותר הוא 1, ואז 2, 4 ו-8 בהתאמה.
-   אם המשתמש מבקש להסביר מה עשית - הסבר.

**פורמט קלט/פלט:**
-   משתמש: `עשרוני: <מספר>`
-   אתה: `בינארי: <ייצוג בינארי>, הסבר:<מה עשית>`
-   משתמש: `נורות: <ייצוג בינארי>`
-   אתה: `עשרוני: <מספר>, הסבר:<מה עשית>`

**דוגמאות:**
משתמש: `עשרוני: 5`
אתה: `בינארי: 0101, הסבר: הנורות 1 ו-4 דולקות, 1+4=5`

משתמש: `נורות: 1010`
אתה: `עשרוני: 10, הסבר: הנורות 2 ו-8 דולקות, 2+8=10`

משתמש: `הסבר איך 7`
אתה: `בינארי: 1110, הסבר: הנורות 1, 2 ו-4 דולקות. 1+2+4 = 7`

**התחל את הסימולציה עם מספר פשוט כלשהו.**
"""
# החלף את "YOUR_API_KEY" במפתח ה-API שלך
api_key = "YOUR_API_KEY"

# אתחול משחק "קלט-פלט"
input_output_game = GoogleGenerativeAI(api_key=api_key, system_instruction=system_instruction_input_output)

# אתחול "מחשב 10 סנט"
binary_computer = GoogleGenerativeAI(api_key=api_key, system_instruction=system_instruction_binary)

# דוגמאות לשימוש:
print("משחק קלט-פלט:")
print(input_output_game.ask("קלט: 7"))
print(input_output_game.ask("קלט: 12"))
print(input_output_game.ask("הגדר מכונה x / 2"))
print(input_output_game.ask("קלט: 12"))

print("\nמחשב 10 סנט:")
print(binary_computer.ask("עשרוני: 6"))
print(binary_computer.ask("נורות: 0111"))
print(binary_computer.ask("הסבר איך 13"))
```

### ניתוח קוד

1.  **ייבוא ספרייה:**

    ```python
    import google.generativeai as genai
    ```

    ספריית `google.generativeai` מיובאת לצורך אינטראקציה עם מודלי Gemini.

2.  **מחלקה `GoogleGenerativeAI`:**

    *   `__init__` (בנאי): מאתחל את המחלקה עם מפתח API, הוראת מערכת ושם המודל. מגדיר את הספרייה ויוצר מופע של מודל Gemini.
    *   `ask` (שיטה): מקבלת שאילתה, שולחת אותה למודל ומחזירה תשובה טקסטואלית. מטפלת בשגיאות אפשריות.

3.  **הוראות מערכת:**

    *   `system_instruction_input_output`: מתארת את הכללים והפורמט של המשחק "קלט-פלט".
    *   `system_instruction_binary`: מתארת את הכללים והפורמט עבור "מחשב 10 סנט".

4.  **אתחול מופעים:**

    *   מופעים של המחלקה `GoogleGenerativeAI` נוצרים עבור כל משחק, תוך העברת ההוראות המתאימות.
    *   שימו לב: עליכם להחליף את `"YOUR_API_KEY"` במפתח ה-API האמיתי של Google Gemini.

5.  **דוגמאות לשימוש:**

    *   קריאות לשיטת `ask` מדגימות כיצד ניתן לשלוח שאילתות למודל ולקבל תשובות במסגרת שני המשחקים.
