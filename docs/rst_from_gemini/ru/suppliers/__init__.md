## Модуль поставщика. Класс `Supplier`

Этот модуль предоставляет инструменты для работы с поставщиками данных.  Класс `Supplier` является базовым классом, предоставляющим общий интерфейс.

Для каждого конкретного поставщика (например, Amazon, AliExpress, Morlevi) есть свои специфичные методы (функции) для извлечения информации. Эти функции дополняют базовый класс `Supplier` и подключаются через интерфейс `supplier.related_functions`.

Методы каждого конкретного поставщика находятся в своих собственных директориях, имена которых соответствуют префиксу имени поставщика (supplier_prefix). Например: `amazon`, `aliexpress`, `morlevi`.
- `supplier_prefix` определяется при создании нового поставщика в системе и обычно основывается на сокращении названия или сайта поставщика.
