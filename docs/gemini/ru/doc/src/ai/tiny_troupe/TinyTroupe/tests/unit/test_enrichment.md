# Модуль `test_enrichment.py`

## Обзор

Этот файл содержит тесты для класса `TinyEnricher` из модуля `tinytroupe.enrichment`.  Тесты проверяют корректность обогащения текста и соответствие результата заданным требованиям (размер результата должен быть не менее в 3 раза больше исходного).

## Функции

### `test_enrich_content`

**Описание**: Функция `test_enrich_content` проверяет метод `enrich_content` класса `TinyEnricher`.  Она использует тестовый входной текст (`content_to_enrich`) и требования (`requirements`) для обогащения текста.  Затем проверяет, что результат обогащения не `None` и что его длина не меньше в 3 раза длины исходного текста.

**Параметры**:

- `content_to_enrich` (str): Исходный текст, который будет обогащён.  В данном случае это текст документа.
- `requirements` (str): Требования к обогащению текста.  Описывают желаемый результат.
- `content_type` (str): Тип содержимого (`"Document"` в данном случае).
- `context_info` (str): Дополнительная информация для контекста обогащения.
- `context_cache` (Optional[str]): Кэш контекста, используется для кеширования. По умолчанию `None`.
- `verbose` (bool): Флаг, указывающий на включение подробной отладки.


**Возвращает**:

- `None`: Функция не возвращает значение, но осуществляет проверку результата.


**Вызывает исключения**:

- `AssertionError`: Возникает, если результат обогащения `None` или не удовлетворяет условию размера.
- `AttributeError`: Возникает, если у класса `TinyEnricher` отсутствует метод `enrich_content`.
