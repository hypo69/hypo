# Модуль `header`

## Обзор

Модуль `header.py` предназначен для определения метаданных проекта `hypotez`, таких как версия, автор, название и прочая информация, необходимая для идентификации и описания проекта. Он также содержит функцию для определения корневой директории проекта.

## Подробней

Этот модуль играет важную роль в инициализации основных параметров проекта. Он пытается прочитать настройки из файла `settings.json` и документацию из `README.MD`, чтобы установить значения для переменных, таких как имя проекта, версия, авторские права и описание. Если файлы не найдены или содержат ошибки, устанавливаются значения по умолчанию.

## Функции

### `set_project_root`

```python
def set_project_root(marker_files: tuple = ('__root__', '.git')) -> Path:
    """
    Args:
        marker_files (tuple): Filenames or directory names to identify the project root.
    
    Returns:
        Path: Path to the root directory if found, otherwise the directory where the script is located.
    """
```

**Описание**: Находит корневой каталог проекта, начиная с директории текущего файла, и останавливается на первой директории, содержащей любой из указанных файлов-маркеров.

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или директорий для идентификации корня проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена, иначе - директория, где расположен скрипт.

**Примеры**:

```python
from pathlib import Path
root_path = set_project_root()
print(f"Root path: {root_path}")
```

## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта.

### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json`.

### `doc_str`

**Описание**: Строка с содержимым файла `README.MD`, содержащая документацию проекта.

### `__project_name__`

**Описание**: Название проекта.

### `__version__`

**Описание**: Версия проекта.

### `__doc__`

**Описание**: Документация проекта.

### `__details__`

**Описание**: Дополнительные детали проекта.

### `__author__`

**Описание**: Автор проекта.

### `__copyright__`

**Описание**: Информация об авторских правах.

### `__cofee__`

**Описание**: Сообщение с предложением поддержать разработчика.
