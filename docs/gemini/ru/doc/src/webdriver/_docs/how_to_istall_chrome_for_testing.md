## Инструкция по установке ChromeDriver

1. **Найдите подходящую версию ChromeDriver:** Откройте веб-сайт [https://chromedriver.chromium.org/downloads](https://chromedriver.chromium.org/downloads) и найдите версию ChromeDriver, соответствующую вашей версии Chrome.  Важно выбрать версию, совместимую с вашей версией Chrome.
2. **Загрузите ChromeDriver:** Нажмите на ссылку для скачивания соответствующего файла ChromeDriver (обычно в формате `.exe` или `.zip`).
3. **Сохраните скачанный файл:** Сохраните загруженный файл в удобном для вас месте (например, на рабочем столе).


## Добавление в системный путь

1. **Распакуйте архив (если необходимо):** Если вы скачали архив, распакуйте его в удобную папку.
2. **Добавление в переменную окружения PATH:**  Откройте системные переменные среды:
   * На Windows: Нажмите Win + I, выберите "Система", затем "О программе", "Дополнительные параметры системы". Выберите вкладку "Дополнительно".  В разделе "Переменные среды" выберите "PATH" и нажмите "Редактировать...".
   * На Linux/macOS:  В зависимости от вашей системы используйте соответствующую команду для добавления в переменную окружения PATH.

3. **Добавление пути:**  Добавьте путь к директории, в которую вы сохранили ChromeDriver, в переменную окружения PATH.  Например, если вы сохранили ChromeDriver в папку `C:\chromedriver`, добавьте `C:\chromedriver` в переменную PATH.
4. **Сохраните изменения:** Сохраните изменения в переменных среды.
5. **Перезапустите командную строку/терминал:**  Это необходимо для того, чтобы изменения вступили в силу.


## Проверка установки

1. **Откройте командную строку/терминал:**
2. **Введите команду:**  Введите следующую команду в командную строку, чтобы проверить установку:
```
chromedriver --version
```
