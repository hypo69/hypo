# Модуль `hypotez/src/suppliers/aliexpress/gui/main.py`

## Обзор

Этот модуль предоставляет интерфейс главного окна для управления рекламными кампаниями. Он использует PyQt6 для построения графического интерфейса и позволяет редактировать JSON-файлы, связанные с кампаниями, продуктами и категориями. Модуль содержит классы для управления вкладками и обработкой событий (например, открытие/сохранение файлов).
