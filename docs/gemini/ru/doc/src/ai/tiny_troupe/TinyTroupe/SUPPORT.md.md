## Как подавать заявки на ошибки и получать помощь

Этот проект использует GitHub Issues для отслеживания ошибок и запросов на новые функции. Пожалуйста, поищите существующие проблемы, прежде чем подавать новые, чтобы избежать дубликатов. Для новых проблем, отправьте свою ошибку или запрос функции как новую Issue.

Для помощи и вопросов об использовании этого проекта, пожалуйста, обратитесь к нашей Wiki или разместите сообщение на нашем форуме обсуждений, оба прямо здесь, на GitHub.

## Политика поддержки Microsoft

Поддержка TinyTroupe ограничена ресурсами, указанными выше.
