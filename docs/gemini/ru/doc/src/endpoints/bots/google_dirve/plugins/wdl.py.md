# Модуль для загрузки файлов с использованием wget

## Обзор

Этот модуль предоставляет функцию `wget_dl` для загрузки файлов по URL-адресу с использованием утилиты `wget`.

## Подробней

Функция `wget_dl` использует `subprocess.check_output` для выполнения команды `wget` в оболочке. Это позволяет загружать файлы по указанному URL-адресу и сохранять их с указанным именем файла.

## Функции

### `wget_dl`

```python
def wget_dl(url):
    """
    Загружает файл по URL-адресу с использованием wget.

    Args:
        url (str): URL-адрес файла для загрузки.

    Returns:
        str: Имя файла, если загрузка прошла успешно, или "error", если произошла ошибка.
    """
```

**Описание**: Загружает файл по заданному URL, используя утилиту `wget`.

**Параметры**:

-   `url` (str): URL-адрес файла, который необходимо загрузить.

**Возвращает**:

-   `str`: Имя загруженного файла в случае успеха или `"error"` в случае возникновения ошибки.

**Примеры**:

```python
filename = wget_dl('https://example.com/file.txt')
if filename != 'error':
    print(f'Файл {filename} успешно загружен.')
else:
    print('Ошибка при загрузке файла.')
```
