# Модуль hypotez/src/translators/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`. Он также обеспечивает доступ к документации, хранящейся в файле `README.MD`.

## Функции

### `set_project_root`

**Описание**:  Определяет корневую директорию проекта, начиная с текущей директории и идя вверх по дереву директорий, пока не найдет директорию, содержащую один из указанных маркеров файлов (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или каталогов, используемых для определения корневой директории.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена. В противном случае, возвращается директория, в которой расположен текущий скрипт.


**Вызывает исключения**:

- Нет.


### `load_settings`

**Описание**: Загружает настройки из файла `settings.json` в корневой директории проекта.

**Параметры**:

- Нет.

**Возвращает**:

- `dict | None`: Словарь с настройками проекта или `None`, если файл не найден или невалидный JSON.


**Вызывает исключения**:

- `FileNotFoundError`: Если файл `settings.json` не найден.
- `json.JSONDecodeError`: Если содержимое файла `settings.json` не является валидным JSON.


### `load_documentation`

**Описание**: Загружает текстовую документацию из файла `README.MD` в корневой директории проекта.

**Параметры**:

- Нет.

**Возвращает**:

- `str | None`: Строка с текстом документации, если она найдена. В противном случае, возвращается `None`.


**Вызывает исключения**:

- `FileNotFoundError`: Если файл `README.MD` не найден.
- `UnicodeDecodeError`: Если файл `README.MD` содержит символы, не поддерживаемые кодировкой.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта.

**Тип**: `Path`

**Примечание**: Эта переменная инициализируется вызовом функции `set_project_root`.


### `settings`

**Описание**: Словарь с настройками проекта.

**Тип**: `dict | None`

**Примечание**: Загружается из файла `settings.json`.


### `__project_name__`
**Описание**: Имя проекта. Получается из настроек, по умолчанию 'hypotez'.

**Тип**: `str`


### `__version__`
**Описание**: Версия проекта. Получается из настроек, по умолчанию пустая строка.

**Тип**: `str`


### `__doc__`
**Описание**: Текстовая документация проекта. Получается из файла README.MD.

**Тип**: `str`


### `__details__`
**Описание**: Подробная информация о проекте (пустая строка по умолчанию).

**Тип**: `str`


### `__author__`
**Описание**: Автор проекта. Получается из настроек, по умолчанию пустая строка.

**Тип**: `str`


### `__copyright__`
**Описание**: Авторские права. Получается из настроек, по умолчанию пустая строка.

**Тип**: `str`


### `__cofee__`
**Описание**: Ссылка на пожертвование разработчику. Получается из настроек, по умолчанию ссылка на Boosty.

**Тип**: `str`

### `MODE`
