Module: hypotez.src.goog.spreadsheet._docs

=======================

This module provides documentation for interacting with Google Sheets using the Google Sheets API v4.  It includes a high-level wrapper class, `Spreadsheet`, for easier management of requests to the API.  The module details how to create, access, and modify spreadsheets, including formatting elements like cell widths, merges, formatting, and data population.
