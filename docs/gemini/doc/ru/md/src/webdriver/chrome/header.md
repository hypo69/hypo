# Модуль hypotez/src/webdriver/chrome/header.py

## Обзор

Этот модуль предоставляет функции для работы с настройками проекта, например, получения пути к корневой директории проекта и загрузки настроек из файла `settings.json`. Он также предоставляет доступ к информации о проекте (название, версия, описание, автор, и др.).


## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с текущей директории и идя вверх по дереву директорий, пока не найдет директорию, содержащую файлы-маркеры (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, которые используются для определения корневой директории проекта. По умолчанию использует кортеж `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращает директорию, где расположен текущий скрипт.

**Вызывает исключения**:

- Нет.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root`.


### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json`. `settings` может быть `None`, если файл не найден или содержит некорректные данные.


### `doc_str`

**Описание**: Строка с описанием проекта (README.MD). `doc_str` может быть `None`, если файл не найден или содержит некорректные данные.


### `__project_name__`

**Описание**: Название проекта, полученное из `settings`. По умолчанию `hypotez`.


### `__version__`

**Описание**: Версия проекта, полученная из `settings`. По умолчанию пустая строка.


### `__doc__`

**Описание**: Описание проекта, полученное из `doc_str`. По умолчанию пустая строка.


### `__details__`

**Описание**: Дополнительные данные о проекте (пустая строка по умолчанию).


### `__author__`

**Описание**: Автор проекта, полученный из `settings`. По умолчанию пустая строка.


### `__copyright__`

**Описание**: Авторские права проекта, полученные из `settings`. По умолчанию пустая строка.


### `__cofee__`

**Описание**: Ссылка на платежную систему для подержки разработчиков. По умолчанию содержит ссылку на Boosty.to.
