# Модуль `login.py`

## Обзор

Модуль `login.py` предоставляет функциональность для авторизации поставщиков, используя вебдрайвер. В данном случае, реализована заглушка, которая всегда возвращает `True`.

## Функции

### `login`

**Описание**: Функция выполняет процесс авторизации. В текущей реализации, она просто регистрирует сообщение в лог и возвращает `True`.

**Параметры**:
- `s` (any): Объект поставщика.

**Возвращает**:
- `bool`: Возвращает `True`, имитируя успешную авторизацию.

**Пример использования**
```python
from src.suppliers.etzmaleh.login import login

# Создаем экземпляр поставщика
class Supplier:
    def __init__(self, name):
        self.name = name

supplier = Supplier("SomeSupplier")

# Пробуем залогиниться
if login(supplier):
    print("Успешно залогинились!")
else:
    print("Не удалось залогиниться.")
```
