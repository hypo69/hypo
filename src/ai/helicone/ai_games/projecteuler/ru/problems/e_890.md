# Задача No 890
## Сумма длин отрезков
## Условия
Рассмотрим три точки на плоскости: *A*, *B* и *C*. Пусть *f*( *A*, *B*, *C* ) будет суммой длин отрезков *AB* + *BC* + *CA*.

Теперь рассмотрим набор *S*, содержащий *n* различных точек на плоскости. Для каждой точки *P* в *S* вычислим *f*( *P*, *X*, *Y* ), где *X* и *Y* — ближайшие к *P* точки в *S*. Если несколько точек находятся на одном минимальном расстоянии, выбираем точку с наименьшим индексом в списке *S*.

Определите сумму всех *f*( *P*, *X*, *Y* ) для всех точек *P* в наборе *S*. Результат округлите до 8 знаков после запятой.

Для определения расстояния между точками используйте формулу Евклида: √((x₁ - x₂)² + (y₁ - y₂)²).

**Например:**
Пусть S = [(0,0), (1,0), (0,1), (1,1)].

*   Для точки (0,0) ближайшие точки - (1,0) и (0,1). Тогда f((0,0), (1,0), (0,1)) = 1 + 1 + √2 ≈ 3.41421356.
*   Для точки (1,0) ближайшие точки - (0,0) и (1,1). Тогда f((1,0), (0,0), (1,1)) = 1 + 1 + √2 ≈ 3.41421356.
*   Для точки (0,1) ближайшие точки - (0,0) и (1,1). Тогда f((0,1), (0,0), (1,1)) = 1 + 1 + √2 ≈ 3.41421356.
*   Для точки (1,1) ближайшие точки - (1,0) и (0,1). Тогда f((1,1), (1,0), (0,1)) = 1 + 1 + √2 ≈ 3.41421356.

Сумма всех f() = 3.41421356 * 4 ≈ 13.65685424
