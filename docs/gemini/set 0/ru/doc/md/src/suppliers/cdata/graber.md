# Модуль hypotez/src/suppliers/cdata/graber.py

## Обзор

Модуль `hypotez/src/suppliers/cdata/graber.py` содержит класс `Graber`, предназначенный для сбора данных о товарах с сайта `cdata.co.il`. Класс наследуется от базового класса `Grbr` и предоставляет асинхронные методы для извлечения различных полей товара.  Он позволяет выполнять предварительные действия (например, закрытие всплывающих окон) до основного процесса сбора данных.

### `Graber`

**Описание**: Класс `Graber` реализует логику сбора данных о товаре с сайта `cdata.co.il`.  Он наследуется от `Grbr` и предоставляет асинхронные методы для извлечения различных полей товара.  Класс хранит префикс поставщика (`supplier_prefix`).

    **Параметры**:

    **Возвращает**:
        - None

- **`grab_page(self, driver: Driver) -> ProductFields`**:
    **Описание**: Асинхронный метод для сбора данных о товаре.
    **Параметры**:
        - `driver` (Driver): Экземпляр класса `Driver`, используемый для сбора данных.
    **Возвращает**:
        - `ProductFields`: Объект `ProductFields`, содержащий собранные данные.
