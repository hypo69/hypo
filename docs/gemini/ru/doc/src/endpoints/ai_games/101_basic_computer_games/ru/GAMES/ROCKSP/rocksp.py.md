# Камень, ножницы, бумага

## Обзор

Этот модуль реализует классическую игру "Камень, ножницы, бумага" для одного игрока против компьютера. Игрок делает выбор (камень, ножницы или бумага), компьютер делает случайный выбор, и затем определяется победитель по стандартным правилам игры.

## Функции

### `play_rock_paper_scissors`

**Описание**:
Функция реализует игровой процесс "Камень, ножницы, бумага". Запрашивает ввод пользователя, генерирует выбор компьютера и определяет победителя.

**Параметры**:
- Нет параметров.

**Возвращает**:
- Нет возвращаемого значения. Функция выводит результаты игры на экран.

**Вызывает исключения**:
- `ValueError`: Возникает, если пользователь вводит некорректное значение, не являющееся целым числом.
