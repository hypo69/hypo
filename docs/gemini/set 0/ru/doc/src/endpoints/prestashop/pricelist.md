# Модуль pricelist.py

## Обзор

Модуль `pricelist.py` предоставляет класс `PriceListRequester` для запроса и обновления цен продуктов в Престашоп API.  Этот модуль наследуется от класса `PrestaShop` из модуля `api.py` и отвечает за взаимодействие с API Престашоп для получения, обновления и модификации цен.

## Класс PriceListRequester

### Описание

Класс `PriceListRequester` предназначен для взаимодействия с API Престашоп для управления ценами товаров. Он наследуется от класса `PrestaShop`.

### Метод `__init__`

**Описание**: Инициализирует объект класса `PriceListRequester`.

**Параметры**:
- `api_credentials` (dict): Словарь с учетными данными для API, включая `'api_domain'` и `'api_key'`.

**Возвращает**:
-  None

### Метод `request_prices`

**Описание**: Запрашивает список цен для указанных товаров.

**Параметры**:
- `products` (list): Список товаров, для которых требуется получить цены.

**Возвращает**:
- dict | None: Словарь, где ключами являются товары, а значениями — их цены. Пример: `{'product1': 10.99, 'product2': 5.99}`. Возвращает `None` в случае ошибки.

**Вызывает исключения**:
-  (Возможные исключения из PrestaShop): Ошибки, которые могут возникнуть при взаимодействии с Престашоп API.


### Метод `update_source`

**Описание**: Обновляет источник данных для запроса цен.

**Параметры**:
- `new_source` (object): Новый источник данных.

**Возвращает**:
-  None

### Метод `modify_product_price`

**Описание**: Модифицирует цену указанного товара.

**Параметры**:
- `product` (str): Название товара.
- `new_price` (float): Новая цена товара.

**Возвращает**:
-  None

**Вызывает исключения**:
-  (Возможные исключения из PrestaShop): Ошибки, которые могут возникнуть при взаимодействии с Престашоп API, или при попытке модифицировать цену.
