# Модуль hypotez/src/ai/openai/model/__init__.py

## Обзор

Этот модуль предоставляет начальную точку входа для доступа к моделям OpenAI, предоставляя константу `MODE` и импортирует класс `OpenAIModel` из модуля `training`.

## Константы

### `MODE`

**Описание**:  Константа, определяющая режим работы модели (например, 'dev', 'prod'). В данном случае её значение равно 'dev'.

**Значение**: 'dev'


## Классы

### `OpenAIModel`

**Описание**:  Класс, представляющий модель OpenAI. Подробная информация о классе содержится в файле `training.py`.
