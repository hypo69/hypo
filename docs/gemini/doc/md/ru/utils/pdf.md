# Модуль utils.pdf

## Обзор

Модуль `utils.pdf` предназначен для преобразования HTML-контента или файлов в PDF-формат, используя библиотеки `pdfkit` и `fpdf`.  Модуль включает в себя методы для сохранения HTML-данных в PDF-файл и альтернативный метод для сохранения текста в PDF.  Он также содержит функции для поиска корневой директории проекта.


## Функции

### `get_project_root`

**Описание**: Находит корневую директорию проекта, начиная с текущей директории файла. Поиск выполняется вверх по иерархии директорий до тех пор, пока не будет найдена директория, содержащая файлы `pyproject.toml`, `requirements.txt` или `.git`.

**Параметры**:
- `marker_files` (tuple, опционально): Кортеж имен файлов или директорий, используемых для определения корневой директории проекта. По умолчанию (`('pyproject.toml', 'requirements.txt', '.git')`).

**Возвращает**:
- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращает директорию, где расположен текущий файл.

**Вызывает исключения**:
- Никаких исключений не вызывает.


### `save_pdf`

**Описание**: Сохраняет HTML-контент или HTML-файл в PDF-файл, используя библиотеку `pdfkit`.

**Параметры**:
- `data` (str | Path): HTML-контент или путь к HTML-файлу.
- `pdf_file` (str | Path): Путь к сохраняемому PDF-файлу.

**Возвращает**:
- `bool`: `True` если PDF успешно сохранен, `False` в противном случае.

**Вызывает исключения**:
- `pdfkit.PDFKitError`: Ошибка генерации PDF через `pdfkit`.
- `OSError`: Ошибка доступа к файлу.
- `Exception`: Любая другая непредвиденная ошибка.


### `save_pdf_v2`

**Описание**: Альтернативный метод сохранения текста в PDF с использованием библиотеки `FPDF`.

**Параметры**:
- `data` (str): Текст, который необходимо сохранить в PDF.
- `pdf_file` (str | Path): Путь к сохраняемому PDF-файлу.

**Возвращает**:
- `bool`: `True` если PDF успешно сохранен, `False` в противном случае.

**Вызывает исключения**:
- `Exception`: Любая другая непредвиденная ошибка.


## Классы

### `PDFUtils`

**Описание**: Класс для работы с PDF-файлами.  Предоставляет статические методы для сохранения HTML-контента в PDF-формат.


**Методы**:
- `save_pdf`: Сохраняет HTML-контент или файл в PDF.
- `save_pdf_v2`: Альтернативный метод сохранения текста в PDF.



##  Константы

### `MODE`

**Описание**: Строковая константа, хранящая режим работы (в данном случае "development").

### `wkhtmltopdf_exe`

**Описание**: Путь к исполняемому файлу `wkhtmltopdf.exe`. Используется библиотекой `pdfkit`.


### `configuration`

**Описание**: Конфигурация для библиотеки `pdfkit`. Устанавливает путь к `wkhtmltopdf.exe`.

### `options`

**Описание**: Параметры для библиотеки `pdfkit`.  В данном случае `{"enable-local-file-access": ""}` разрешает доступ к локальным файлам.

##  Примечания

- Необходимо убедиться, что исполняемый файл `wkhtmltopdf.exe` доступен по пути `wkhtmltopdf_exe`.  В противном случае произойдет ошибка.
- Модуль `fpdf` должен быть установлен для использования метода `save_pdf_v2`.
