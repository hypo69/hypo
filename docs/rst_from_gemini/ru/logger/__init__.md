# hypotez/src/logger/__init__.py

**Расположение:** `C:\Users\user\Documents\repos\hypotez\src\logger\__init__.py`

**Роль:** `doc_creator` (создание документации)

**Описание:**

Этот модуль инициализирует логгирование и предоставляет доступ к функциональности модуля `logger`. Он также определяет корневую директорию проекта и добавляет её в `sys.path` для корректного импорта других модулей.

**Функции:**

* **`get_project_root(marker_files=('pyproject.toml', 'requirements.txt', '.git')) -> Path`:**
    * Находит корневую директорию проекта, начиная от текущего файла.
    * Ищет в родительских директориях файлы, указанные в `marker_files` (например, `pyproject.toml`, `requirements.txt`, `.git`).
    * Если корневой каталог найден, добавляет его в `sys.path`, что позволяет импортировать модули из других каталогов проекта.
    * Возвращает путь к корневому каталогу проекта. Если корневой каталог не найден, возвращает текущую директорию.

**Переменные:**

* **`__root__ (Path)`:** Путь к корневому каталогу проекта, полученный функцией `get_project_root()`. Эта переменная используется для корректного импорта других модулей из проекта.

**Импорты:**

* `sys`
* `pathlib`
* `packaging.version` (для работы с версиями пакетов)
* `src.logger` (модуль, содержащий логику логгирования)
* `logger` (вероятно, объект логгера из модуля `src.logger`)
* `exceptions` (модуль, содержащий пользовательские исключения)

**Исключения (из модуля `exceptions`):**

* `ExecuteLocatorException`
* `DefaultSettingsException`
* `CredentialsError`
* `PrestaShopException`
* `PayloadChecksumError`
