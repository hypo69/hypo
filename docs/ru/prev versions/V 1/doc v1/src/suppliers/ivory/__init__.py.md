# Модуль `src.suppliers.ivory`

## Обзор

Модуль `src.suppliers.ivory` предназначен для работы с поставщиком Ivory. Он содержит классы и функции для сбора и обработки данных с использованием грабера.

## Подробней

Модуль предоставляет функциональность для взаимодействия с поставщиком Ivory, включая сбор данных. В настоящее время модуль включает класс `Graber`, который, вероятно, отвечает за парсинг и извлечение информации с веб-сайтов или API поставщика.

## Классы

### `Graber`

**Описание**:
Класс `Graber` предназначен для сбора данных с ресурсов поставщика Ivory.

**Методы**:
- `__init__`: Инициализирует объект грабера.
- `grab`: Выполняет сбор данных.

**Параметры**:
- Отсутствуют, если не указаны в конструкторе класса.

**Примеры**
```python
# Примеры определения класса и работы с классом
from src.suppliers.ivory.graber import Graber

graber = Graber()
data = graber.grab()
```
