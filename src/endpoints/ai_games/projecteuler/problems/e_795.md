# Задача No 795
## Расстояние Хэмминга на больших массивах

## Условия

Расстояние Хэмминга между двумя целыми числами — это количество позиций, в которых их двоичное представление отличается. Например, расстояние Хэмминга между 5 (101 в двоичном виде) и 3 (011 в двоичном виде) равно 2.

Пусть *D(x, y)* обозначает расстояние Хэмминга между *x* и *y*.

Рассмотрим два массива целых чисел *A* и *B* одинаковой длины *n*. Определим *H(A, B)* как сумму *D(A[i], B[i])* для всех *i* от 0 до *n-1*.

Например, если *A* = {1, 2, 3} и *B* = {2, 3, 4}, то *H(A, B)* = *D(1, 2) + D(2, 3) + D(3, 4)* = 2 + 1 + 2 = 5.

Пусть *S(n)* — множество всех массивов целых чисел длиной *n*, где каждый элемент находится в диапазоне от 0 до *2<sup>20</sup> - 1*. Другими словами, *S(n)* содержит все массивы, которые состоят из *n* целых чисел, каждое из которых может принимать значения от 0 до 1048575.

Назовем *T(n)* суммой всех значений *H(A, B)* для всех возможных пар массивов (*A, B*) из *S(n)*.

Например, *T(2) = 20971520*.

Найдите *T(10<sup>7</sup>) mod 1000000007*.
