# Задача No 816
## Сумма квадратов
## Условия
Дано множество целых чисел $S$, где $S_0 = 290797$ и $S_n = S_{n-1}^2 \text{ mod } 50515093$ для $n \ge 1$.

Далее, пусть $T_n = (S_n \text{ mod } 2000) - 1000$.

Например, первые десять элементов последовательности $T$ следующие:
$T_0 = -179$, $T_1 = -786$, $T_2 = -908$, $T_3 = 522$, $T_4 = -470$, $T_5 = -771$, $T_6 = 385$, $T_7 = -611$, $T_8 = -471$ и $T_9 = -160$.

Для двух целых чисел $a$ и $b$, определим $d(a, b)$ как расстояние между точками $(a,0)$ и $(b,0)$. Тогда $d(a,b) = |a-b|$.

Определим $D(n)$ как минимальное расстояние между всеми парами точек $(T_i, 0)$ и $(T_j, 0)$, где $0 \le i < j < n$.

Например, $D(10)=2$.

Найдите $D(2000000)$.
