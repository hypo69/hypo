# Модуль `example_pprint.py`

## Обзор

Модуль содержит примеры использования функций `pprint` и `pretty_print` для форматированного вывода данных.

## Подробнее

Этот модуль демонстрирует, как использовать функции `pprint` (из модуля `src.printer`) и `pretty_print` (из стандартной библиотеки `pprint`) для форматированного вывода данных. Он импортирует необходимые модули и выводит строку "Hello, world!" с использованием функции `pprint`.

## Функции

### `pprint`

```python
from src.printer import pprint
```

**Описание**: Функция для форматированного вывода данных.

**Как работает функция**:
Функция `pprint` из модуля `src.printer` используется для вывода строки "Hello, world!". Она предоставляет возможности для более читабельного и структурированного вывода данных по сравнению со стандартной функцией `print`.

**Примеры**:

```python
