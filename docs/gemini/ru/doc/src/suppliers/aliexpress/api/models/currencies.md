# Модуль `hypotez/src/suppliers/aliexpress/api/models/currencies.py`

## Обзор

Этот модуль содержит класс `Currency`, определяющий константы для различных валют.

## Классы

### `Currency`

**Описание**: Класс `Currency` содержит константы для различных валют.

**Атрибуты**:

- `USD`: Константа для американского доллара.
- `GBP`: Константа для британского фунта стерлингов.
- `CAD`: Константа для канадского доллара.
- `EUR`: Константа для евро.
- `UAH`: Константа для гривны.
- `MXN`: Константа для мексиканского песо.
- `TRY`: Константа для турецкой лиры.
- `RUB`: Константа для российского рубля.
- `BRL`: Константа для бразильского реала.
- `AUD`: Константа для австралийского доллара.
- `INR`: Константа для индийской рупии.
- `JPY`: Константа для японской йены.
- `IDR`: Константа для индонезийской рупии.
- `SEK`: Константа для шведской кроны.
- `KRW`: Константа для южнокорейской воны.
- `ILS`: Константа для израильского шекеля.
