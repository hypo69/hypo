# Модуль для работы с моделями генерации видео GPT4Free

## Обзор

Модуль предоставляет пример использования GPT4Free для генерации видео. Он использует клиент `Client` из библиотеки `g4f` для взаимодействия с провайдером `HuggingFaceMedia` и генерации видео на основе текстового запроса.

## Подробнее

Этот модуль демонстрирует, как использовать GPT4Free для работы с моделями генерации видео. Он показывает, как инициализировать клиент с использованием ключа API, получить список доступных моделей для генерации видео и сгенерировать видео на основе текстового запроса.

## Функции

### `Client`

```python
class Client:
    """
    Клиент для взаимодействия с провайдерами GPT4Free.

    Args:
        provider (g4f.Provider): Провайдер, которого следует использовать.
        api_key (str): Ключ API для доступа к провайдеру.

    """
```

**Описание**: Класс `Client` используется для установки соединения с провайдером `HuggingFaceMedia`.

**Параметры**:
- `provider` (`g4f.Provider`): Провайдер, которого следует использовать. В данном случае `g4f.Provider.HuggingFaceMedia`.
- `api_key` (`str`): Ключ API для доступа к провайдеру.

**Примеры**:

```python
client = Client(
    provider=g4f.Provider.HuggingFaceMedia,
    api_key="hf_***" # Your API key here
)
```

### `client.media.generate()`

```python
def generate(model: str, prompt: str, response_format: str = "url") -> object:
    """
    Генерирует медиа-контент на основе текстового запроса.

    Args:
        model (str): Модель для генерации контента.
        prompt (str): Текстовый запрос для генерации контента.
        response_format (str, optional): Формат ответа. По умолчанию "url".

    Returns:
        object: Объект с данными о сгенерированном контенте.
    """
```

**Описание**: Метод `generate()` используется для генерации видео на основе текстового запроса.

**Параметры**:
- `model` (`str`): Модель для генерации контента.
- `prompt` (`str`): Текстовый запрос для генерации контента.
- `response_format` (`str`, optional): Формат ответа. По умолчанию `"url"`.

**Возвращает**:
- `object`: Объект с данными о сгенерированном контенте.

**Как работает функция**:
1. Функция выполняет запрос к провайдеру для генерации видео на основе предоставленного текстового запроса.
2. Функция возвращает объект с данными о сгенерированном видео.

**Примеры**:

```python
result = client.media.generate(
    model=video_models[0],
    prompt="G4F AI technology is the best in the world.",
    response_format="url"
)

print(result.data[0].url)
