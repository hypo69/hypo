# Модуль hypotez/src/suppliers/aliexpress/api/models

## Обзор

Данный модуль содержит определения моделей для работы с API AliExpress.  Он предоставляет классы, представляющие различные типы данных, необходимые для взаимодействия с API, такие как типы валют, языков, параметры запроса, а также модели для товаров, категорий, горячих товаров и партнерских ссылок.

## Модули

### `languages`

Этот модуль содержит определение класса `Language`, представляющего языковые параметры для API запросов.


### `currencies`

Этот модуль содержит определение класса `Currency`, представляющего валютные параметры для API запросов.


### `request_parameters`

Этот модуль содержит определения перечислений для параметров запросов к API AliExpress:
- `ProductType`:  Перечисление типов продуктов.
- `SortBy`: Перечисление параметров сортировки продуктов.
- `LinkType`: Перечисление типов ссылок.


### `affiliate_link`

Этот модуль содержит определение класса `AffiliateLink`, представляющего модель партнерской ссылки.


### `hotproducts`

Этот модуль содержит определение класса `HotProductsResponse`, представляющего ответ API для горячих товаров.


### `product`

Этот модуль содержит определение класса `Product`, представляющего модель продукта.


### `category`

Этот модуль содержит определения классов `Category` и `ChildCategory`, представляющих категории и подкатегории продуктов.
