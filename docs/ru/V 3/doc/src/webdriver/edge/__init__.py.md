# Модуль `src.webdriver.edge`

## Обзор

Модуль `src.webdriver.edge` предоставляет функциональность для работы с браузером Edge в проекте `hypotez`. Он включает в себя необходимые классы и функции для управления экземплярами Edge WebDriver.

## Подробней

Этот модуль служит связующим звеном между основной логикой проекта и драйвером Edge, позволяя автоматизировать взаимодействие с браузером Edge для различных задач, таких как тестирование веб-приложений, сбор данных и выполнение других автоматизированных действий.
