# Модуль для работы с помощником по коду

## Обзор

Этот модуль предоставляет класс `CodeAssistant` для работы с различными моделями ИИ, такими как Google Gemini и OpenAI, для выполнения задач обработки кода.  Модуль предназначен для анализа и генерации документации по исходному коду.

## Примеры использования

```python
assistant = CodeAssistant(role='code_checker', lang='ru', model=['gemini'])
assistant.process_files()
```

## Классы

### `CodeAssistant`

**Описание**: Класс для работы с помощником по коду.  Он предоставляет методы для взаимодействия с моделями ИИ и обработки файлов кода.

**Атрибуты**:

- `role` (str): Роль помощника (например, 'code_checker').
- `lang` (str): Язык, на котором помощник будет работать (например, 'ru').
- `model` (list): Список моделей ИИ, используемых помощником (например, ['gemini']).

**Методы**:

- `process_files()`: Метод для обработки списка файлов кода.  Подробное описание внутри метода.
