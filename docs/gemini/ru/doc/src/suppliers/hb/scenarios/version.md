# Модуль hypotez/src/suppliers/hb/scenarios/version.py

## Обзор

Модуль `version.py` содержит константы и переменные, относящиеся к версии и другим параметрам сценариев.

## Переменные

### `MODE`

**Описание**: Переменная, хранящая режим работы (`dev`).

**Значение**: `'dev'`


## Документированные атрибуты

### `__version__`

**Описание**: Переменная, хранящая версию модуля или пакета.

**Тип**: Строка (`str`)

**Значение**: `"3.12.0.0.0.4"`


### `__name__`

**Описание**:  Имя модуля. Если скрипт запускается напрямую, значение будет `"__main__"`.

**Тип**: Строка (`str`)


### `__doc__`

**Описание**: Строка документации модуля.

**Тип**: Строка (`str`)

### `__details__`

**Описание**: Переменная, содержащая дополнительные детали о модуле.  Точное назначение зависит от конкретного модуля или пакета.

**Тип**: Строка (`str`)

**Значение**: `"Details about version for module or class"`


### `__annotations__`

**Описание**: Содержит аннотации типов для переменных и функций в модуле.

**Тип**: Специальный объект Python, хранящий аннотации.


### `__author__`

**Описание**:  Имя(на) автора(ов) модуля.

**Тип**: Строка (`str`)

**Значение**: `"hypotez "`
