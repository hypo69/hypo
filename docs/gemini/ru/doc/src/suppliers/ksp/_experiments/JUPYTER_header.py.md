# Модуль `src.suppliers.ksp._experiments.JUPYTER_header`

## Обзор

Модуль содержит набор экспериментов, связанных с поставщиком `ksp`. 
Он включает в себя импорты различных модулей и классов, необходимых для работы с поставщиками, продуктами, категориями, веб-драйверами, утилитами и конечными точками PrestaShop.
Также содержит функцию `start_supplier` для запуска поставщика с указанными параметрами.

## Подробней

Модуль используется для экспериментов и отладки функциональности, связанной с поставщиком `ksp`.
Он позволяет запускать поставщика, обрабатывать продукты и категории, а также взаимодействовать с веб-драйверами и API PrestaShop.
Этот код является частью проекта `hypotez` и предназначен для использования разработчиками для тестирования и отладки.

## Функции

### `start_supplier`

```python
def start_supplier(supplier_prefix: str = 'aliexpress', locale: str = 'en' ):
    """ Старт поставщика """
    ...
```

**Назначение**: Запускает поставщика с указанным префиксом и локалью.

**Параметры**:
- `supplier_prefix` (str, optional): Префикс поставщика. По умолчанию `'aliexpress'`.
- `locale` (str, optional): Локаль поставщика. По умолчанию `'en'`.

**Возвращает**:
- `Supplier`: Объект поставщика, созданный с указанными параметрами.

**Как работает функция**:
1. Функция принимает префикс поставщика и локаль в качестве аргументов.
2. Создает словарь `params` с этими параметрами.
3. Создает экземпляр класса `Supplier` с использованием словаря `params` в качестве аргументов.
4. Возвращает созданный экземпляр `Supplier`.

**Примеры**:

```python
supplier = start_supplier(supplier_prefix='ksp', locale='ru')
# Возвращает объект поставщика ksp с локалью ru
```
```python
supplier = start_supplier()
# Возвращает объект поставщика aliexpress с локалью en
```
```python
supplier = start_supplier(locale='de')
# Возвращает объект поставщика aliexpress с локалью de
