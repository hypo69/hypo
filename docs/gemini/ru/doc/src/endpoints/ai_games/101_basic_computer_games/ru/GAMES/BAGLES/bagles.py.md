# Игра Бейглы

## Обзор

Этот модуль реализует игру "Бейглы", в которой компьютер загадывает трехзначное число с неповторяющимися цифрами, а игрок пытается его угадать, получая подсказки "PICO", "FERMI" и "BAGELS".

## Функции

### `generate_secret_number`

**Описание**: Генерирует случайное трехзначное число с неповторяющимися цифрами.

**Параметры**:
- Нет параметров.

**Возвращает**:
- `str`: Сгенерированное трехзначное число в виде строки.

### `get_clues`

**Описание**: Генерирует подсказки "PICO", "FERMI" и "BAGELS" на основе сравнения загаданного числа и предположения игрока.

**Параметры**:
- `secret_number` (str): Загаданное трехзначное число.
- `user_guess` (str): Предположение игрока.

**Возвращает**:
- `list[str]`: Список подсказок. Если нет совпадений, возвращает `["BAGELS"]`.

### `play_bagels`

**Описание**: Основная логика игры "Бейглы". Инициализирует игру, обрабатывает ввод пользователя, генерирует подсказки и выводит результаты.

**Параметры**:
- Нет параметров.

**Возвращает**:
- `None`: Функция ничего не возвращает, только выводит сообщения в консоль.
