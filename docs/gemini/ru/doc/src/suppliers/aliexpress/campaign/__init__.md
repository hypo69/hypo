# Модуль hypotez/src/suppliers/aliexpress/campaign

## Обзор

Данный модуль содержит инструменты для управления рекламными кампаниями на Aliexpress. Он предоставляет классы и функции для редактирования кампаний, обработки данных, генерации отчетов и работы с данными в формате Google Sheets (хотя это не реализовано).

## Модули

### `ali_campaign_editor`

**Описание**: Класс для редактирования кампаний Aliexpress.  

### `prepare_campaigns`

**Описание**: Функции для подготовки данных рекламных кампаний.  

### `html_generators`

**Описание**: Классы для генерации HTML-отчетов о рекламных кампаниях.


## Классы

### `AliCampaignEditor`

**Описание**: Класс для редактирования кампаний Aliexpress.


## Функции

### `process_campaign`

**Описание**: Обрабатывает данные одной рекламной кампании.

**Параметры**:
- `campaign_data` (dict): Словарь с данными кампании.

**Возвращает**:
- `dict`: Словарь с обработанными данными кампании или `None`, если произошла ошибка.

**Вызывает исключения**:
- `ValueError`: Если входные данные некорректного формата.


### `process_campaign_category`

**Описание**: Обрабатывает данные по категориям рекламных кампаний.

**Параметры**:
- `category_data` (dict): Словарь с данными по категориям.

**Возвращает**:
- `dict`: Словарь с обработанными данными по категориям или `None`, если произошла ошибка.

**Вызывает исключения**:
- `ValueError`: Если входные данные некорректного формата.


### `process_all_campaigns`

**Описание**: Обрабатывает данные всех рекламных кампаний.

**Параметры**:
- `all_campaigns_data` (list): Список словарей с данными по всем кампаниям.

**Возвращает**:
- `list`: Список словарей с обработанными данными по всем кампаниям или `None`, если произошла ошибка.

**Вызывает исключения**:
- `ValueError`: Если входные данные некорректного формата.


### `CategoryHTMLGenerator`

**Описание**: Класс для генерации HTML-отчетов по категориям.

**Методы**:
- `generate_report`: Генерирует HTML-отчет по заданным данным.


### `ProductHTMLGenerator`

**Описание**: Класс для генерации HTML-отчетов по продуктам.

**Методы**:
- `generate_report`: Генерирует HTML-отчет по заданным данным.



## Константы

### `MODE`

**Описание**: Переменная, хранящая режим работы (например, 'dev' или 'prod').

**Значение**: `'dev'` по умолчанию.
