# Модуль валидации строк

## Обзор

Модуль `validator.py` предназначен для валидации строк на соответствие определенным критериям или форматам. Он включает функции для проверки цен, веса, артикулов и URL, а также для проверки, является ли строка целым числом.

## Классы

### `ProductFieldsValidator`

**Описание**: 
Класс `ProductFieldsValidator` предназначен для проверки строк на соответствие определенным критериям или шаблонам. Он предоставляет статические методы для валидации различных полей продукта, таких как цена, вес, артикул и URL.

## Функции

### `validate_price`

**Описание**: 
Функция `validate_price` проверяет, является ли строка допустимой ценой. Она удаляет все символы, кроме цифр и точки, заменяет запятую на точку и пытается преобразовать строку в число с плавающей точкой.

**Параметры**:
- `price` (str): Строка, представляющая цену для проверки.

**Возвращает**:
- `bool | None`: Возвращает `True`, если строка является допустимой ценой, иначе `None`.

### `validate_weight`

**Описание**: 
Функция `validate_weight` проверяет, является ли строка допустимым весом. Она удаляет все символы, кроме цифр и точки, заменяет запятую на точку и пытается преобразовать строку в число с плавающей точкой.

**Параметры**:
- `weight` (str): Строка, представляющая вес для проверки.

**Возвращает**:
- `bool | None`: Возвращает `True`, если строка является допустимым весом, иначе `None`.

### `validate_sku`

**Описание**: 
Функция `validate_sku` проверяет, является ли строка допустимым артикулом. Она удаляет специальные символы и переносы строк, обрезает пробелы и проверяет длину строки.

**Параметры**:
- `sku` (str): Строка, представляющая артикул для проверки.

**Возвращает**:
- `bool | None`: Возвращает `True`, если строка является допустимым артикулом, иначе `None`.

### `validate_url`

**Описание**: 
Функция `validate_url` проверяет, является ли строка допустимым URL. Она добавляет "http://" к URL, если он не начинается с "http", парсит URL и проверяет наличие схемы и сетевого расположения.

**Параметры**:
- `url` (str): Строка, представляющая URL для проверки.

**Возвращает**:
- `bool | None`: Возвращает `True`, если строка является допустимым URL, иначе `None`.

### `isint`

**Описание**: 
Функция `isint` проверяет, является ли строка целым числом. Она пытается преобразовать строку в целое число и возвращает `True`, если преобразование удалось, и `None` в противном случае.

**Параметры**:
- `s` (str): Строка, которую нужно проверить на целое число.

**Возвращает**:
- `bool | None`: Возвращает `True`, если строка является целым числом, иначе `None`.
