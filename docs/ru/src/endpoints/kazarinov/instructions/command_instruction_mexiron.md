# Инструкция по обработке JSON данных о компьютерных компонентах

## Обзор

Эта инструкция описывает процесс обработки JSON данных, содержащих информацию о компьютерных компонентах. Цель - классифицировать тип сборки (например, игровой, рабочая станция), предоставить заголовки и описания на иврите и русском языках, перевести детали компонентов и вернуть структурированный JSON-вывод. Необходимо поддерживать правильное форматирование, включать оценки достоверности и следовать подробным рекомендациям по описаниям и обработке компонентов.

## Терминология

В данной инструкции слова `product` (продукт) и `component` (компонент) являются синонимами и обозначают компонент для сборки компьютера.

## Структура JSON

- **`he`** (иврит): Содержит информацию на иврите.
    - **`title`**: Заголовок сборки на иврите.
    - **`description`**: Описание сборки на иврите.
    - **`build_types`**: Объект с типами сборки и оценками достоверности (например, `gaming`, `workstation`).
    - **`products`**: Массив объектов, представляющих компоненты.
        - **`product_id`**: Идентификатор продукта (оставляется как есть из входных данных).
        - **`product_title`**: Название продукта на иврите (генерируется).
        - **`product_description`**: Описание продукта на иврите (генерируется). Если не можете создать описание - оставьте поле пустым.
        - **`specification`**: Спецификация продукта на иврите (генерируется). Если не можете создать спецификацию - оставьте поле пустым.
        - **`image_local_saved_path`**: Путь к изображению продукта (оставляется как есть из входных данных).
- **`ru`** (русский): Содержит информацию на русском языке.
    - **`title`**: Заголовок сборки на русском языке.
    - **`description`**: Описание сборки на русском языке.
    - **`build_types`**: Объект с типами сборки и оценками достоверности (например, `gaming`, `workstation`).
    - **`products`**: Массив объектов, представляющих компоненты.
        - **`product_id`**: Идентификатор продукта (оставляется как есть из входных данных).
        - **`product_title`**: Название продукта на русском языке (генерируется).
        - **`product_description`**: Описание продукта на русском языке (генерируется). Если не можете создать описание - оставьте поле пустым.
        - **`specification`**: Спецификация продукта на русском языке (генерируется). Если не можете создать спецификацию - оставьте поле пустым.
        - **`image_local_saved_path`**: Путь к изображению продукта (оставляется как есть из входных данных).
