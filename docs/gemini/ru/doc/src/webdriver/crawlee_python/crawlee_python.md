# Модуль `crawlee_python`

## Обзор

Модуль `crawlee_python` предоставляет класс `CrawleePython` для выполнения веб-скрейпинга с использованием Playwright.  Он позволяет задать максимальное количество запросов, режим работы (с графическим интерфейсом или без) и тип браузера.  Модуль умеет собирать ссылки с веб-страниц, извлекать данные и сохранять их в JSON-файл.

## Классы

### `CrawleePython`

**Описание**:  Класс `CrawleePython` реализует логику веб-скрейпинга.  Он управляет настройкой и запуском PlaywrightCrawler, обработкой запросов, извлечением данных и сохранением результатов.

**Атрибуты**:

- `max_requests`: Максимальное количество запросов.
- `headless`: Режим работы (с графическим интерфейсом или без).
- `browser_type`: Тип браузера (например, 'firefox').
- `crawler`: Экземпляр `PlaywrightCrawler`.

**Методы**:

#### `__init__`

**Описание**: Инициализирует экземпляр `CrawleePython` с заданными параметрами.

**Параметры**:

- `max_requests` (int): Максимальное количество запросов.
- `headless` (bool): Режим работы (с графическим интерфейсом или без).
- `browser_type` (str): Тип браузера.

#### `setup_crawler`

**Описание**: Настраивает экземпляр `PlaywrightCrawler`.

**Возвращает**:
- None

#### `run_crawler`

**Описание**: Запускает скриптер на заданном списке URL.

**Параметры**:
- `urls` (list[str]): Список URL для запуска сбора.

**Возвращает**:
- None

#### `export_data`

**Описание**: Экспортирует собранные данные в JSON-файл.

**Параметры**:
- `file_path` (str): Путь к файлу для сохранения данных.

**Возвращает**:
- None

#### `get_data`

**Описание**: Получает собранные данные.

**Возвращает**:
- dict: Собранные данные в виде словаря.

#### `run`

**Описание**: Главный метод для запуска веб-скрейпинга.

**Параметры**:
- `urls` (list[str]): Список URL для запуска сбора.

**Возвращает**:
- None


## Пример использования

```python
async def main():
    experiment = CrawleeExperiment(max_requests=5, headless=False, browser_type='firefox')
    await experiment.run(['https://ksp.co.il'])

asyncio.run(main())
```
