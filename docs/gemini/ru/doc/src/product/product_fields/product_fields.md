# Модуль hypotez/src/product/product_fields/product_fields.py

## Обзор

Модуль `product_fields` содержит класс `ProductFields`, предназначенный для работы с полями товаров в системе управления контентом PrestaShop. Класс предоставляет методы для доступа и изменения различных полей товара, включая основные данные (ID, поставщик, бренд, категории), цены, характеристики, а также дополнительные поля, такие как изображения и текстовые поля.  В модуле реализована обработка многоязычных данных, а также логика загрузки и сохранения данных товаров.

## Классы

### `ProductFields`

**Описание**: Класс `ProductFields` содержит данные о полях товара в формате API PrestaShop.  Он предназначен для работы с данными товаров, хранящимися в базе данных PrestaShop.  Класс включает в себя свойства для доступа к различным полям товара, а также методы для их изменения.

**Атрибуты**:

* `product_fields_list`: Список названий полей товара. Загружается из файла `fields_list.txt`.
* `language`: Словарь, сопоставляющий языковые коды с их идентификаторами в PrestaShop.
* `presta_fields`: Объект `SimpleNamespace`, содержащий поля товара, загруженные из файла `product_fields_default_values.json`.
* `assist_fields_dict`: Словарь для вспомогательных полей (например, URL изображений).

**Методы**:

* `__init__(self)`: Инициализирует класс. Загружает список полей из файла `fields_list.txt`, загружает дефолтные значения полей из файла `product_fields_default_values.json`.
* `_load_product_fields_list(self) -> List[str]`: Загружает список полей из файла.
* `_payload(self) -> bool`: Загружает дефолтные значения полей из файла `product_fields_default_values.json`.
