# Модуль `test_control.py`

## Обзор

Данный модуль содержит тесты для модуля `control` из пакета `tinytroupe`.  Тесты проверяют корректность работы функций `begin`, `checkpoint`, и `end` в различных сценариях: с агентами, с миром и с фабрикой агентов. Тесты включают проверку статуса симуляции и существования кэшированных следов.
