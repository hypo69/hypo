# Модуль playwrid

## Обзор

Этот модуль содержит класс `Playwrid`, который представляет собой подкласс `PlaywrightCrawler`. Он предоставляет дополнительную функциональность, такую как возможность устанавливать пользовательские настройки браузера, профили и параметры запуска, используя Playwright.  Модуль загружает настройки из файла `playwrid.json` и, при необходимости, из файла с указанным именем.

## Классы

### `Playwrid`

**Описание**: Класс `Playwrid` наследуется от `PlaywrightCrawler` и предоставляет расширенные возможности для управления настройками и запуском Playwright.

**Методы**:

#### `__init__`

**Описание**: Инициализирует экземпляр `Playwrid` с заданными параметрами запуска, настройками и пользовательским агентом.

**Параметры**:

- `settings_name` (Optional[str], опционально): Имя файла настроек. Если указано, используется файл с указанным именем, если он существует. По умолчанию `None`.
- `user_agent` (Optional[Dict[str, Any]], опционально): Словарь с настройками пользовательского агента. По умолчанию `None`.


**Возвращает**:

- `None`

#### `_load_settings`

**Описание**: Загружает настройки для `Playwrid` из файла `playwrid.json` или файла с указанным именем.

**Параметры**:

- `settings_name` (Optional[str], опционально): Имя файла настроек. По умолчанию `None`.

**Возвращает**:

- `SimpleNamespace`: Объект `SimpleNamespace` содержащий настройки.

#### `_set_launch_options`

**Описание**: Конфигурирует параметры запуска для `PlaywrightCrawler`.

**Параметры**:

- `settings` (SimpleNamespace): Объект `SimpleNamespace` содержащий настройки запуска.

**Возвращает**:

- `dict`: Словарь с параметрами запуска для Playwright.

#### `start`

**Описание**: Запускает `Playwrid` и переходит на указанный URL.

**Параметры**:

- `url` (str): URL для перехода.

**Возвращает**:

- `None`

**Вызывает исключения**:

- Любой тип исключения (`Exception`): В случае возникновения ошибки во время работы.


#### `current_url`

**Описание**: Свойство для получения текущего URL.  (Необходимо документировать детали и тип возвращаемого значения)
