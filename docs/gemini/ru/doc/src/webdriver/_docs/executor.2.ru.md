# Модуль executor

## Обзор

Этот модуль предоставляет набор функций для взаимодействия с веб-элементами на основе локаторов. Функции позволяют выполнять различные действия, такие как нахождение элементов, отправка сообщений, извлечение атрибутов и получение URL.

## Функции

### `execute_locator`

**Описание**: Выполняет действия на веб-элементе, используя указанный локатор.  Возможность обработки ошибок позволяет продолжить выполнение скрипта, даже если часть операций не удалась.

**Параметры**:
- `locator` (dict): Словарь или объект, содержащий информацию о локаторе (например, тип поиска и селектор).  Обязательный параметр.
- `message` (str, необязательно): Сообщение для отправки элементу (например, текст для ввода). По умолчанию пустая строка.
- `typing_speed` (float, необязательно): Скорость набора текста, если отправляется сообщение (в секундах между нажатиями клавиш). По умолчанию 0.0.
- `continue_on_error` (bool, необязательно): Флаг, указывающий, следует ли продолжать выполнение, если произошла ошибка. По умолчанию `True`.

**Возвращает**:
- `any`: Результат выполнения локатора, который может быть веб-элементом, списком элементов, значением атрибута или результатом действия.

### `get_webelement_by_locator`

**Описание**: Находит и возвращает веб-элемент(ы) на странице, используя локатор.

**Параметры**:
- `locator` (dict): Словарь или объект с информацией о локаторе. Обязательный параметр.

**Возвращает**:
- `any`: Один или несколько веб-элементов, найденных по локатору. Возвращает `None`, если элемент не найден.

### `get_attribute_by_locator`

**Описание**: Получает значение атрибута веб-элемента, найденного по локатору.

**Параметры**:
- `locator` (dict): Словарь или объект с информацией о локаторе. Обязательный параметр.
- `message` (str, необязательно): Сообщение для отправки элементу перед получением атрибута. По умолчанию пустая строка.

**Возвращает**:
- `any`: Значение атрибута веб-элемента. Возвращает `None`, если атрибут не найден или произошла ошибка.


### `send_message`

**Описание**: Отправляет сообщение (например, текст) веб-элементу, используя локатор.

**Параметры**:
- `locator` (dict): Словарь или объект с информацией о локаторе. Обязательный параметр.
- `message` (str): Сообщение для отправки элементу. Обязательный параметр.
- `typing_speed` (float, необязательно): Скорость набора текста, если сообщение отправляется по частям. По умолчанию 0.0.
- `continue_on_error` (bool, необязательно): Флаг для продолжения выполнения в случае ошибки. По умолчанию `True`.

**Возвращает**:
- `bool`: `True`, если сообщение успешно отправлено, или `False`, если произошла ошибка.

### `get_url`

**Описание**: Загружает HTML-контент с указанного URL-адреса или локального файла.

**Параметры**:
- `url` (str): URL-адрес или путь к файлу для получения HTML-контента. Обязательный параметр.
- `protocol` (str, необязательно): Протокол для URL (по умолчанию `https://`). По умолчанию `https://`.

**Возвращает**:
- `bool`: `True`, если контент успешно загружен, или `False`, если произошла ошибка.
