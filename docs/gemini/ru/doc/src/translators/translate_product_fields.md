# Модуль translate_product_fields

## Обзор

Данный модуль отвечает за управление переводами полей товаров. Он обеспечивает связь между словарем полей товара, таблицей переводов и переводчиками.  Модуль предоставляет функции для получения переводов из таблицы переводов PrestaShop, добавления новых переводов и перевода полей товара.

## Функции

### `get_translations_from_presta_translations_table`

**Описание**: Функция получает переводы полей товара из таблицы переводов PrestaShop.

**Параметры**:
- `product_reference` (str):  Уникальный идентификатор товара.
- `credentials` (dict): Словарь с данными для подключения к базе данных переводов (например, логин, пароль, имя базы данных).
- `i18n` (str, optional): Код языка перевода (например, `en_EN`, `ru_RU`). По умолчанию `None`.

**Возвращает**:
- list: Список словарей, каждый из которых представляет перевод определённого поля товара. Возвращает пустой список, если переводы не найдены или произошла ошибка.

**Вызывает исключения**:
- Возможны исключения, связанные с базой данных (например, `DatabaseError`) при подключении или работе с таблицей переводов.


### `insert_new_translation_to_presta_translations_table`

**Описание**: Функция добавляет новый перевод в таблицу переводов PrestaShop.

**Параметры**:
- `record` (dict): Словарь, содержащий данные для нового перевода.
- `credentials` (dict): Словарь с данными для подключения к базе данных переводов (например, логин, пароль, имя базы данных).


**Возвращает**:
- Возвращает `None`.

**Вызывает исключения**:
- Возможны исключения, связанные с базой данных (например, `DatabaseError`) при подключении или работе с таблицей переводов.


### `translate_record`

**Описание**: Функция переводит поля товара.

**Параметры**:
- `record` (dict): Словарь с данными, которые необходимо перевести.
- `from_locale` (str): Код исходного языка.
- `to_locale` (str): Код языка перевода.


**Возвращает**:
- dict: Словарь с переведёнными данными.


**Вызывает исключения**:
- Возможны исключения, связанные с работой с переводчиком (например, `TranslationError`).
