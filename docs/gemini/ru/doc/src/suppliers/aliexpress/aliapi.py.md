# Модуль `aliapi.py`

## Обзор

Модуль `aliapi.py` представляет собой класс `AliApi`, который является расширением класса `AliexpressApi` и предоставляет специализированные методы для взаимодействия с API AliExpress. Он включает в себя функции для получения информации о товарах, создания партнерских ссылок и управления данными.

## Классы

### `AliApi`

**Описание**:
Класс `AliApi` предоставляет пользовательские методы для работы с AliExpress API, включая получение информации о товарах и создание партнерских ссылок.

**Параметры**:
   - `language` (str): Язык для запросов API. По умолчанию `'en'`.
   - `currency` (str): Валюта для запросов API. По умолчанию `'usd'`.
   - `*args`: Произвольные позиционные аргументы.
   - `**kwargs`: Произвольные именованные аргументы.

**Атрибуты**:
  - `manager_categories` (CategoryManager): Менеджер для категорий товаров.
  - `manager_campaigns` (ProductCampaignsManager): Менеджер для кампаний и акций.

**Методы**:

  - [`retrieve_product_details_as_dict`](#retrieve_product_details_as_dict)
  - [`get_affiliate_links`](#get_affiliate_links)

## Функции

### `retrieve_product_details_as_dict`

**Описание**:
Получает подробную информацию о товарах, используя их идентификаторы, и возвращает данные в виде списка словарей.

**Параметры**:
- `product_ids` (list): Список идентификаторов товаров.

**Возвращает**:
- `dict | None`: Список словарей с подробной информацией о товарах. Может вернуть `None` в случае ошибки.

### `get_affiliate_links`

**Описание**:
Получает партнерские ссылки для указанных товаров.

**Параметры**:
- `links` (str | list): Ссылка или список ссылок на товар.
- `link_type` (int, optional): Тип партнерской ссылки. По умолчанию `0`.
- `**kwargs`: Дополнительные именованные аргументы.

**Возвращает**:
- `List[SimpleNamespace]`: Список объектов `SimpleNamespace`, содержащих партнерские ссылки.
