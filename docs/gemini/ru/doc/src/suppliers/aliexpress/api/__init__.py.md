# Модуль `src.suppliers.aliexpress.api`

## Обзор

Модуль `src.suppliers.aliexpress.api` предоставляет API-обертку для работы с Aliexpress API.

## Содержание

- [Обзор](#обзор)
- [Переменные](#переменные)
- [Импорты](#импорты)

## Переменные
### `__version__`
**Описание**:  Переменная хранит информацию о версии модуля.

### `__doc__`
**Описание**: Переменная хранит  документацию модуля.

### `__details__`
**Описание**: Переменная хранит  детали модуля.
## Импорты

### `packaging.version.Version`
**Описание**: Импортирует класс `Version` из модуля `packaging.version` для работы с версиями пакетов.

### `AliexpressApi`
**Описание**: Импортирует класс `AliexpressApi` из модуля `.api`, предоставляющий основной API-интерфейс для взаимодействия с Aliexpress.

### `models`
**Описание**: Импортирует модуль `models` из текущего пакета, который вероятно содержит определения моделей данных, используемых в API.
