# Файл desktop.ini

## Обзор

Данный файл представляет собой файл системной информации Windows, используемый для настройки отображения папок.  Он содержит метаданные для папки, такие как имя, значок и атрибуты.

## Структура файла

Файл содержит ряд строк, каждая из которых представляет собой пару ключ-значение. Ключи обычно содержат имя атрибута, а значения - его значение. Примеры ключей включают `Mode`, `Vid`, `FolderType` и `Logo`.

## Значения в файле

* **`Mode`**: Определяет режим отображения папки.
* **`Vid`**: Идентификатор вида папки.
* **`FolderType`**: Тип папки (например, `Generic`).
* **`Logo`**: Путь к файлу изображения, используемому в качестве значка папки. В данном случае, путь `E:\\Users\\user\\images\\LOGOS\\R.png` указывает на изображение логотипа.
