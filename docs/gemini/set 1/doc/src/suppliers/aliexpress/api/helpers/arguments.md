# Модуль `hypotez/src/suppliers/aliexpress/api/helpers/arguments.py`

## Обзор

Модуль `arguments.py` содержит вспомогательные функции для обработки аргументов, используемых в API AliExpress.  Он предоставляет инструменты для преобразования и валидации входных данных, обеспечивая корректную работу других частей API.


## Функции

### `get_list_as_string`

**Описание**: Преобразует список или строку в строку, разделенную запятыми, или возвращает `None` для `None`.

**Параметры**:
- `value`: Значение, которое может быть списком или строкой.

**Возвращает**:
- `str | None`:  Строковое представление списка, разделенного запятыми, или `None`, если входное значение `None`.

**Вызывает исключения**:
- `InvalidArgumentException`: Если входное значение не является ни списком, ни строкой.


### `get_product_ids`

**Описание**: Преобразует строку или список идентификаторов продуктов в список идентификаторов, полученных с помощью `get_product_id`.

**Параметры**:
- `values`:  Входное значение, которое может быть списком или строкой идентификаторов.

**Возвращает**:
- `list`: Список идентификаторов продуктов.

**Вызывает исключения**:
- `InvalidArgumentException`: Если входное значение не является ни списком, ни строкой.
