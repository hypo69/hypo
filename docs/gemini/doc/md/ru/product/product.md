## Обзор

Модуль `src.product` определяет поведение продукта в проекте. Он взаимодействует с веб-сайтом, продуктом и API PrestaShop.  Этот модуль включает класс `Product`, который наследует атрибуты и методы от `ProductFields` и `PrestaShop`.

## Класс Product

### `Product`

**Описание**: Класс `Product` управляет манипуляциями с продуктом.  Изначально он запрашивает данные с страницы продукта, а затем работает с API PrestaShop.  Он наследует функциональность от классов `ProductFields` и `PrestaShop`.

## Функции

### `Product.get_parent_categories`

**Описание**:  Получает родительские категории для заданной категории.  Повторяет функциональность метода `get_parents` из класса `Category`.

**Параметры**:
- `id_category` (int): ID категории.
- `dept` (int, optional): Глубина категории. По умолчанию 0.

**Возвращает**:
- `list`: Список родительских категорий.

**Вызывает исключения**:
- `TypeError`: Если `id_category` не является целым числом.
