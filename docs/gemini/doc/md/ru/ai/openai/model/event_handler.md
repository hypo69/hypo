# Модуль `hypotez/src/ai/openai/model/event_handler.py`

## Обзор

Данный модуль определяет обработчик событий для потоковой передачи данных от модели OpenAI, в частности, для ассистентов. Он реализует класс `EventHandler`, который перехватывает и обрабатывает события, связанные с созданием текста, изменением текста, вызовом инструментов, а также выводом от инструментов, включая код интерпретатора.

## Классы

### `EventHandler`

**Описание**: Класс `EventHandler` наследуется от `AssistantEventHandler` и переопределяет методы для обработки различных событий, связанных с потоковой передачей данных от модели OpenAI.


**Методы**:

#### `on_text_created`

**Описание**: Обрабатывает событие создания нового текста в потоке.

**Параметры**:
- `text` (Text): Объект, содержащий созданный текст.


**Возвращает**:
- `None`


#### `on_text_delta`

**Описание**: Обрабатывает событие изменения текста в потоке.

**Параметры**:
- `delta` (TextDelta): Объект, содержащий изменения в тексте.
- `snapshot` (Text): Текущее состояние текста.


**Возвращает**:
- `None`


#### `on_tool_call_created`

**Описание**: Обрабатывает событие создания вызова инструмента.

**Параметры**:
- `tool_call` (ToolCall): Объект, содержащий информацию о вызове инструмента.


**Возвращает**:
- `None`


#### `on_tool_call_delta`

**Описание**: Обрабатывает событие изменения вызова инструмента.

**Параметры**:
- `delta` (ToolCallDelta): Объект, содержащий изменения в вызове инструмента.
- `snapshot` (ToolCall): Текущее состояние вызова инструмента.


**Возвращает**:
- `None`

**Дополнительные детали**:
- Обрабатывает вызовы инструмента типа "code_interpreter".
- Выводит ввод и вывод для интерпретатора кода.
- Выводит журналы (`output.logs`).
