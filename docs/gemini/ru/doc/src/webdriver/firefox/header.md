# Модуль hypotez/src/webdriver/firefox/header.py

## Обзор

Данный модуль содержит функции для работы с настройками проекта, получения пути к корневой директории проекта и доступа к дополнительной документации. Он использует `json` для работы с файлом настроек `settings.json` и содержит логику для обработки ошибок при чтении файла настроек и документации.

## Функции

### `set_project_root`

**Описание**: Функция находит корневую директорию проекта, начиная с текущего файла. Поиск происходит вверх по директориям, пока не будет найдена директория, содержащая один из указанных файлов-маркеров (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или каталогов, используемых для определения корневой директории проекта. По умолчанию используется кортеж `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращает путь к директории, где расположен текущий файл.

**Вызывает исключения**:

- Нет.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root()`.


### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json`. `None`, если файл не найден или некорректен.


### `doc_str`

**Описание**: Строка с документацией проекта, загруженная из файла `README.MD`. `None`, если файл не найден или некорректен.

### `__project_name__`

**Описание**: Название проекта, полученное из настроек. По умолчанию `hypotez`.

### `__version__`

**Описание**: Версия проекта, полученная из настроек. По умолчанию пустая строка.

### `__doc__`

**Описание**: Документация проекта, полученная из файла `README.MD`. По умолчанию пустая строка.

### `__details__`

**Описание**: Дополнительные детали проекта. По умолчанию пустая строка.

### `__author__`

**Описание**: Автор проекта, полученный из настроек. По умолчанию пустая строка.

### `__copyright__`

**Описание**: Авторские права, полученные из настроек. По умолчанию пустая строка.

### `__cofee__`

**Описание**: Ссылка для поддержки разработчика. По умолчанию ссылка на Boosty.


## Обработка исключений

Функции `set_project_root`, а также чтение данных из `settings.json` и `README.MD` содержат обработку исключений `FileNotFoundError` и `json.JSONDecodeError`. В случае возникновения ошибки, соответствующие переменные (settings, doc_str) инициализируются значением по умолчанию.
