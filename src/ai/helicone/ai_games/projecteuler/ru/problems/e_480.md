# Задача No 480
## Преобразования строк
## Условия
Дана строка `s` длины `n`, состоящая из строчных букв английского алфавита.

Мы можем применить следующую операцию к строке `s`:
Выбираем два индекса `i` и `j`, такие что `1 <= i < j <= n` и `s[i] != s[j]`, затем меняем `s[i]` и `s[j]` местами.

Для заданной строки `s`, пусть `f(s)` будет минимальным количеством операций, необходимых для преобразования `s` в строку, в которой все одинаковые символы стоят подряд.

Например, для строки `s` = `"aabbaa"`, `f(s)` = 2.
Один из возможных вариантов достижения этого значения:

- поменять местами s[1] и s[5], получив `"aabaaa"`
- поменять местами s[2] и s[4], получив `"aaaaba"`

Затем можно поменять местами s[4] и s[5] получив `"aaaaab"` или s[5] и s[6], получив `"aaaaab"` 
(В строке `"aaaaab"` все символы одного вида находятся рядом, `f(s) = 2`).

Рассмотрим все строки длины `n`, состоящие из первых `m` букв английского алфавита.
Определим `F(n, m)` как сумму `f(s)` для всех таких строк `s`.

Например, `F(3, 3) = 12`, `F(7, 3) = 73119`

Найдите `F(10^7, 30)` и приведите ответ по модулю `1000000007`.
