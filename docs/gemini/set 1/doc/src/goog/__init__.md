# Модуль goog

## Обзор

Этот модуль предоставляет инструменты для работы с Google Spreadsheets. Он содержит класс `Spreadsheet`, позволяющий взаимодействовать с электронными таблицами.  Модуль настроен на работу в режиме разработки (MODE = 'dev').
