# Модуль запуска приложения Juliana

## Обзор

Модуль является точкой входа для запуска веб-приложения Juliana. Он загружает конфигурацию из `config.json`, настраивает маршруты для веб-сайта и backend API, а затем запускает Flask-сервер.

## Подробней

Этот модуль играет центральную роль в инициализации и запуске всего приложения. Он отвечает за:

1.  Загрузку параметров конфигурации, необходимых для работы приложения (например, номер порта).
2.  Определение маршрутов, используемых как для клиентской части (веб-сайта), так и для backend API.
3.  Запуск Flask-сервера с заданной конфигурацией.

## Функции

### `__main__`

**Назначение**: Главная функция, выполняющаяся при запуске скрипта.

**Как работает функция**:

1.  Загружает конфигурацию из файла `config.json`.
2.  Создает экземпляры классов `Website` и `Backend_Api`, передавая им необходимые параметры.
3.  Добавляет маршруты для веб-сайта и backend API в Flask-приложение.
4.  Запускает Flask-сервер на указанном порту.

**ASCII flowchart**:

```
Загрузка конфигурации из config.json
     │
     │
Создание экземпляров Website и Backend_Api
     │
     │
Добавление маршрутов для веб-сайта
     │
     │
Добавление маршрутов для backend API
     │
     │
Запуск Flask-сервера
     │
     │
Закрытие Flask-сервера
```

**Примеры**:

При запуске модуля `run.py` из командной строки:

```bash
python run.py
