# Задача No 404
## Зеленые эллипсы
## Условия
Рассмотрим эллипс, заданный уравнением x²/a² + y²/b² = 1.
Пусть для эллипса с a > b, C1 является окружностью с центром (0,0) и радиусом a, а C2 является окружностью с центром (0,0) и радиусом b.

Определим E(a,b) как множество точек (x,y), таких что:
* x и y являются целыми числами
* x² / a² + y² / b² ≤ 1
* x² + y² > b²
* x² + y² < a²

Для примера, E(10,6) имеет 24 точки.
Вот их координаты:

(±8, ±3), (±8, ±4), (±7, ±5), (±6, ±6), (±3, ±8), (±4, ±8), (±5, ±7), (±6, ±6).

Пусть F(N) равно сумме количества точек в E(a, b) для всех пар (a, b) удовлетворяющих следующим условиям:
* a и b являются целыми числами
* 0 < b < a ≤ N

Известно, что F(10) = 252 и F(100) = 337048.

Найдите F(10<sup>7</sup>) mod 1000000007.
