# Модуль `src.webdriver.edge.extentions.version`

## Обзор

Модуль `version.py` определяет версию и другую метаинформацию для модуля `src.webdriver.edge.extentions`.

## Оглавление

- [Переменные](#переменные)
    - [`MODE`](#mode)
    - [`__name__`](#__name__)
    - [`__version__`](#__version__)
    - [`__doc__`](#__doc__)
    - [`__details__`](#__details__)
    - [`__annotations__`](#__annotations__)
    - [`__author__`](#__author__)


## Переменные

### `MODE`
**Описание**: Определяет режим работы модуля. По умолчанию `dev`.

### `__name__`
**Описание**: Содержит имя модуля.

### `__version__`
**Описание**: Содержит версию модуля.

### `__doc__`
**Описание**: Строка документации модуля.

### `__details__`
**Описание**: Содержит дополнительные сведения о версии модуля или класса.

### `__annotations__`
**Описание**: Содержит аннотации типов для переменных и функций в модуле.

### `__author__`
**Описание**: Содержит имя автора модуля.
