# Модуль hypotez/src/category/header.py

## Обзор

Данный модуль определяет корневой путь к проекту `hypotez`. Все импорты строятся относительно этого пути. В будущем планируется перенести определение корневого пути в системную переменную.

## Функции

### `get_project_root`

**Описание**: Находит корневой каталог проекта, начиная от текущего файла, проходя вверх по директориям, и останавливаясь на первом каталоге, содержащем указанные файлы или директории.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, которые будут использоваться для определения корневого каталога проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.


**Возвращает**:

- `Path`: Путь к корневому каталогу проекта, если найден. В противном случае возвращает директорию, в которой расположен скрипт.

**Вызывает исключения**:

- Нет.


## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта.


**Тип**:

- `Path`


## Замечания

Переменная `__root__` используется для определения корневого каталога проекта. Она инициализируется с помощью функции `get_project_root()`, и если данный корневой каталог не содержится в пути поиска модулей Python (`sys.path`), то он добавляется в начале этого пути (`sys.path.insert(0, str(__root__))`). Это позволяет импортировать модули из корневого каталога проекта.
