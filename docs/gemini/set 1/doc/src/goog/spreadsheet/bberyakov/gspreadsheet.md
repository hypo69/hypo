# Модуль gspreadsheet

## Обзор

Модуль `gspreadsheet` предоставляет инструменты для работы с Google Таблицами. Он позволяет создавать, открывать и управлять таблицами, используя учетные записи сервисов Google.

## Классы

### `GSpreadsheet`

**Описание**: Класс `GSpreadsheet` реализует интерфейс для взаимодействия с Google Таблицами. Он наследуется от класса `Spreadsheet`.

**Атрибуты**:

* `gsh`: Объект `Spreadsheet`, представляющий открытую таблицу Google.
* `gclient`: Объект для работы с клиентом Google Spreadsheets.

**Методы**:

#### `__init__`

**Описание**: Конструктор класса `GSpreadsheet`. Инициализирует объект `GSpreadsheet`.

**Параметры**:

* `s_id` (str, необязательно): Идентификатор таблицы.
* `s_title` (str, необязательно): Название таблицы.
* `*args`: Дополнительные аргументы.
* `**kwards`: Дополнительные ключевые аргументы.

**Возвращает**:
-  None

#### `get_project_spreadsheets_dict`

**Описание**: Возвращает словарь со списком таблиц.

**Параметры**:

* `self`: Экземпляр класса `GSpreadsheet`.


**Возвращает**:
- dict: Словарь с данными о таблицах.


#### `get_by_title`

**Описание**: Открывает таблицу по ее названию или создает новую, если ее нет.

**Параметры**:

* `self`: Экземпляр класса `GSpreadsheet`.
* `sh_title` (str, по умолчанию 'New Spreadsheet'): Название таблицы.

**Возвращает**:
- None

#### `get_by_id`

**Описание**: Открывает таблицу по ее идентификатору.

**Параметры**:

* `self`: Экземпляр класса `GSpreadsheet`.
* `sh_id` (str): Идентификатор таблицы.

**Возвращает**:
- Spreadsheet: Объект `Spreadsheet`, представляющий открытую таблицу.



#### `get_all_spreadsheets_for_current_account`

**Описание**: Возвращает список всех таблиц текущего аккаунта.

**Параметры**:

* `self`: Экземпляр класса `GSpreadsheet`.

**Возвращает**:
- list: Список объектов `Spreadsheet`, представляющих все открытые таблицы.

## Заметки

-  Этот модуль использует библиотеки `gspread` и `json` для взаимодействия с Google Таблицами.
-  Необходимо настроить соответствующую учетную запись сервиса Google для работы с API.
-  Используется файл `goog\\onela-hypotez-1aafa5e5d1b5.json` для аутентификации.
-  Методы `get_by_title` и `get_by_id` возвращают ссылку на таблицу.
-  Код в комментариях нуждается в доработке для большей ясности и корректности.
