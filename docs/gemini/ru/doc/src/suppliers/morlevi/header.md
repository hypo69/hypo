# Модуль hypotez/src/suppliers/morlevi/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`.  Он также содержит переменные, хранящие информацию о проекте, например, имя, версию, документацию и авторство.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла. Ищет вверх по директориям, пока не найдет директорию, содержащую один из указанных маркеров.

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, используемых для определения корневой директории проекта. По умолчанию содержит ('pyproject.toml', 'requirements.txt', '.git').

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена, иначе путь к директории текущего файла.

**Примечания**:
- Добавляет корневую директорию в `sys.path` для импорта модулей из проекта.

## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта.


### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `src/settings.json`.

**Возможные значения**:

* `dict`: Словарь настроек.
* `None`: Если файл `settings.json` не найден или содержит некорректный JSON.

### `doc_str`

**Описание**: Строка с содержимым файла `README.MD`.

**Возможные значения**:

* `str`: Содержимое файла `README.MD`
* `None`: Если файл `README.MD` не найден или содержит некорректный текст.


### `__project_name__`

**Описание**: Название проекта. По умолчанию 'hypotez'.


### `__version__`

**Описание**: Версия проекта. По умолчанию пустая строка.


### `__doc__`

**Описание**: Документация проекта (из файла `README.MD`). По умолчанию пустая строка.


### `__details__`

**Описание**: Дополнительные сведения о проекте. По умолчанию пустая строка.


### `__author__`

**Описание**: Автор проекта. По умолчанию пустая строка.


### `__copyright__`

**Описание**: Авторские права. По умолчанию пустая строка.


### `__cofee__`

**Описание**: Ссылка на платформа для поддержки разработчика. По умолчанию содержит ссылку на boosty.to.


## Обработка исключений

При загрузке настроек (`settings`) и документации (`doc_str`) обрабатываются исключения `FileNotFoundError` и `json.JSONDecodeError`.  В случае ошибки значения соответствующих переменных остаются `None`.
