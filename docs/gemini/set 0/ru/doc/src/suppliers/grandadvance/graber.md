## Обзор

Модуль `graber.py` содержит класс `Graber`, предназначенный для сбора данных о товарах с сайта `grandadvanse.co.il`. Класс наследуется от базового класса `Grbr` из модуля `src.suppliers`.  Он реализует асинхронную функцию `grab_page` для извлечения различных полей товара.  Для каждого поля товара определена соответствующая функция, которая может быть переопределена для специфической обработки.  Класс предоставляет механизм для выполнения предварительных действий (например, закрытие всплывающих окон) перед основным сбором данных.
