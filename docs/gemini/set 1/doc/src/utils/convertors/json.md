# Модуль `hypotez/src/utils/convertors/json.py`

## Обзор

Модуль `src.utils.convertors.json` предоставляет функции для конвертации данных в формате JSON в различные форматы: CSV, SimpleNamespace, XML и XLS.

## Функции

### `json2csv`

**Описание**: Преобразует данные в формате JSON или из файла JSON в CSV формат с разделителем запятая.

**Параметры**:
- `json_data` (str | list | dict | Path): Данные в формате JSON в виде строки, списка словарей или пути к файлу JSON.
- `csv_file_path` (str | Path): Путь к файлу CSV для записи.

**Возвращает**:
- bool: `True`, если преобразование прошло успешно, `False` - в противном случае.

**Вызывает исключения**:
- `ValueError`: Если тип `json_data` не поддерживается.
- `Exception`: Если произошла ошибка при разборе JSON или записи в CSV.


### `json2ns`

**Описание**: Преобразует данные в формате JSON или из файла JSON в объект `SimpleNamespace`.

**Параметры**:
- `json_data` (str | dict | Path): Данные в формате JSON в виде строки, словаря или пути к файлу JSON.

**Возвращает**:
- `SimpleNamespace`: Преобразованные данные JSON в виде объекта `SimpleNamespace`.

**Вызывает исключения**:
- `ValueError`: Если тип `json_data` не поддерживается.
- `Exception`: Если произошла ошибка при разборе JSON.


### `json2xml`

**Описание**: Преобразует данные в формате JSON или из файла JSON в XML формат.

**Параметры**:
- `json_data` (str | dict | Path): Данные в формате JSON в виде строки, словаря или пути к файлу JSON.
- `root_tag` (str, опционально): Имя тега корневого элемента XML. По умолчанию "root".

**Возвращает**:
- str: Результирующая строка XML.

**Вызывает исключения**:
- `ValueError`: Если тип `json_data` не поддерживается.
- `Exception`: Если произошла ошибка при разборе JSON или преобразовании в XML.


### `json2xls`

**Описание**: Преобразует данные в формате JSON или из файла JSON в XLS формат.

**Параметры**:
- `json_data` (str | list | dict | Path): Данные в формате JSON в виде строки, списка словарей или пути к файлу JSON.
- `xls_file_path` (str | Path): Путь к файлу XLS для записи.

**Возвращает**:
- bool: `True`, если преобразование прошло успешно, `False` - в противном случае.

**Вызывает исключения**:
- `ValueError`: Если тип `json_data` не поддерживается.
- `Exception`: Если произошла ошибка при разборе JSON или записи в XLS.
