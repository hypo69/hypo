# Задача No 384
## Руническое начертание
## Условия

Рассмотрим последовательность чисел, образованную по следующему правилу:
* $a_1 = 1$.
* Для $n>1$, $a_n = \text{sum}(a_{n-1})$ + наименьшее положительное число, которое не входит в $\{a_1, ..., a_{n-1}\}$.
    
Где sum($k$) обозначает сумму цифр числа $k$ в десятичной системе счисления.
Первые несколько чисел этой последовательности таковы: 1, 2, 4, 8, 16, 23, 28, 31, 35, 40, 45, 50, 56, 62, 68, 74, 80, 86, 93, ...
    
Пусть $S(n)$ - сумма первых $n$ членов этой последовательности.
Например, $S(5) = 1 + 2 + 4 + 8 + 16 = 31$.
    
Вычислите $S(10^{11})$.
