# Модуль `src.endpoints.bots`

## Обзор

Модуль `src.endpoints.bots` предназначен для управления и интеграции ботов различных платформ. В данный момент модуль содержит функциональность для работы с ботом Telegram.

## Содержание

- [Обзор](#обзор)
- [Импорт](#импорт)
- [Переменные](#переменные)
- [Модули](#модули)

## Импорт

В данном модуле импортируется класс `TelegramBot` из модуля `telegram`, который расположен в текущей директории.

## Переменные

### `MODE`

**Описание**:
Режим работы модуля. По умолчанию установлен в `'dev'`.

## Модули

### `telegram`

- **Описание**: Модуль содержит функциональность для работы с ботом Telegram. Включает класс `TelegramBot`.
