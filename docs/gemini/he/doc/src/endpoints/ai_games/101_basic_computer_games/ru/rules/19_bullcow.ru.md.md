# правила המשחק BULLCOW (Быки и коровы)

## סקירה כללית

המשחק **BULLCOW** הוא גרסה משופרת של המשחק "Bagels", שבו המטרה היא לנחש מספר בן 5 ספרות נסתר שחושב עליו היריב (מחשב או שחקן אחר). שחקן והמחשב מתחלפים בניסיונות לנחש את המספר. עבור כל ספרה במיקום הנכון ניתנת "שור" (BULL), ועבור כל ספרה נכונה במיקום לא נכון - "פרה" (COW). לדוגמה, אם המספר הנסתר הוא 51340, והתשובה שלך היא 21734, התוצאה תהיה: 1 שור ו-2 פרות.

## הוראות שלב אחר שלב ליישום

### 1. אתחול המשחק
   - אחד השחקנים (או המחשב) מייצר מספר אקראי בן 5 ספרות שישמש כ"חידה".
   - השחקן השני (או המחשב) מנסה לנחש את המספר הזה.
   - כל ניסיון של השחקן מוערך לפי מספר ה"שור" וה"פרות" שמוצג כרמז.

### 2. מחזור המשחק העיקרי
   - **תור השחקן:**
     1. השחקן מזין את התשובה שלו בת 5 הספרות.
     2. התוכנית משווה אותה למספר החידה ומציגה את מספר ה"שור" וה"פרות".
     3. השחקן ממשיך בניסיונות חדשים עד שהוא מנחש את המספר הנכון.
   
   - **תור המחשב:**
     1. המחשב מייצר מספר שישמש כ"חידה" עבור השחקן.
     2. המחשב מצמצם בהדרגה את טווח המספרים האפשריים על סמך רמזים.
     3. המשחק ממשיך עד שהמחשב או השחקן מנחשים את המספר.

### 3. ספירת המנצח
   - המנצח הוא זה שמנחש ראשון את המספר בן 5 הספרות.
   - התוכנית מודיעה על המנצח:
     ```
     ברכות, ניחשת את המספר!
     ```
   - אם שני השחקנים לא מצליחים לנחש את המספר במספר קבוע של ניסיונות, המשחק מסתיים בתיקו.

### 4. סיום המשחק
   - לאחר שאחד השחקנים מנחש את המספר או שמספר הניסיונות נגמר, התוכנית תציע לשחק שוב:
     ```
     האם תרצה לשחק שוב? (כן/לא)
     ```

## דוגמה לעבודה של התוכנית

1. **תחילת המשחק:**
   ```
   ברוכים הבאים למשחק שוורים ופרות!
   חשבו על מספר בן 5 ספרות.
   הניסיונות שלכם:
   ```

2. **תור השחקן:**
   ```
   הניסיון שלך: 21734
   תוצאה: 1 שור, 2 פרות
   ```

3. **תור המחשב:**
   ```
   הניסיון שלי: 6195
   התוצאה שלי: 0 שוורים, 4 פרות
   ```

4. **סיום המשחק:**
   ```
   ברכות, ניחשת את המספר!
   האם תרצה לשחק שוב? (כן/לא)
   > לא
   תודה על המשחק!
   ```

## מגבלות אפשריות
- שחקנים יכולים לתת רמזים בלתי אפשריים (לדוגמה, אם מספר ה"פרות" או "שוורים" לא יכול להיות נכון בהתאם למספרים), מה שעלול להוביל לשגיאה בספירה.
- המספר חייב להיות בן 5 ספרות וללא ספרות חוזרות.

## יישום
המשחק מיושם באמצעות יצירת מספרים אקראיים לחידה והערכת כל קלט של השחקן באמצעות ספירת שוורים ופרות.
