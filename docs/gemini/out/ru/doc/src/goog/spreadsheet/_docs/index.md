# Документация модуля hypotez/src/goog/spreadsheet/_docs

## Обзор

Модуль `hypotez/src/goog/spreadsheet/_docs` содержит документацию, описывающую процесс работы с Google Sheets API v4 для создания и управления электронными таблицами.  Документация охватывает создание сервисного аккаунта, установку необходимых библиотек, создание и настройку таблиц, включая формулы, форматирование ячеек и ширину столбцов.  Также представлено обсуждение важных аспектов работы с API, таких как квоты и различия в терминологии API v3 и v4.


## Классы

### `Spreadsheet`

**Описание**: Класс `Spreadsheet` представляет собой обёртку над Google Sheets API v4. Он предоставляет методы для подготовки и выполнения запросов к таблицам, упрощая процесс создания, редактирования и управления Google Sheets.


**Методы**:

- `prepare_setColumnWidth(col, width)`: Подготавливает запрос для изменения ширины столбца.
- `prepare_setColumnsWidth(startCol, endCol, width)`: Подготавливает запрос для изменения ширины нескольких столбцов.
- `prepare_mergeCells(range_str)`: Подготавливает запрос для объединения ячеек.
- `prepare_setValues(cellsRange, values, majorDimension="ROWS")`: Подготавливает запрос для заполнения ячеек. `majorDimension` может быть установлен как "ROWS" или "COLUMNS"
- `prepare_setCellsFormat(range_str, format_dict)`: Подготавливает запрос для применения форматирования к диапазону ячеек.
- `prepare_setCellsFormats(range_str, values)` : Подготавливает запрос для применения форматирования к ячейкам по переданным значениям `values`. `values` должен быть массивом массивов и соответствовать диапазону.
- `runPrepared(valueInputOption="USER_ENTERED")`: Выполняет подготовленные запросы к Google Sheets API v4.  `valueInputOption` определяет, как будут обрабатываться значения в ячейках.



## Функции


### `toGridRange(range_str)`

**Описание**:  Преобразует строковое представление диапазона ячеек (A1-нотация) в формат `GridRange` для использования в методах `Spreadsheet`.


**Параметры**:
- `range_str` (str): Строковое представление диапазона ячеек в A1-нотации.


**Возвращает**:
- dict: Объект `GridRange` в формате, ожидаемом API.
