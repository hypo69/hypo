# Модуль `login.py`

## Обзор

Модуль `login.py` предназначен для обработки логина поставщика. Он содержит функции для авторизации поставщика и использует модуль `logger` для логирования.

## Подробней

Этот модуль является частью подсистемы `suppliers` и отвечает за авторизацию поставщиков в системе `hypotez`. Функция `login` выполняет процесс аутентификации, возвращая `True` в случае успешной авторизации.
Этот модуль содержит функцию `login`, которая принимает объект поставщика (`Supplier`) и возвращает булево значение, указывающее на успешность авторизации.
