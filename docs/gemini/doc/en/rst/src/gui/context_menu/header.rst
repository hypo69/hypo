hypotez/src/gui/context_menu/header.py
=======================================

This module defines constants and paths related to the project's environment. It loads the project name from `settings.json`, determines the root path, and adds necessary paths to the system path.  It also handles potential missing paths and updates the WEASYPRINT_DLL_DIRECTORIES environment variable.  The module suppresses GTK log output.
