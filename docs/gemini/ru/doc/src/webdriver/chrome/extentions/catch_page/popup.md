# Модуль popup.js

## Обзор

Этот модуль содержит JavaScript-код для всплывающего окна расширения Chrome. Он обрабатывает нажатие кнопки "Отправить URL" и отправляет активный URL в фоновый скрипт.

## Функции

### `addEventListener`

**Описание**: Регистрирует обработчик события `click` для кнопки с ID "sendUrlButton".

**Параметры**:

- `event` (object): Объект события клика.

**Вызывает исключения**:

Нет


### `chrome.tabs.query`

**Описание**: Запрашивает данные о текущей активной вкладке.

**Параметры**:

- `query` (object): Объект с параметрами запроса (active: true, currentWindow: true).


**Вызывает исключения**:

Нет


### `runtime.sendMessage`

**Описание**: Отправляет сообщение в фоновый скрипт расширения.

**Параметры**:

- `message` (object): Объект с данными сообщения (action: "sendUrl", url: activeTabUrl).
- `callback` (function): Функция обратного вызова, которая получает ответ от фонового скрипта.

**Вызывает исключения**:

Нет


### Анонимная функция (обработчик события)

**Описание**: Обработчик события `click` для кнопки "Отправить URL". Выполняет следующие действия:
 - Выводит диалоговое окно с сообщением "Hello, world!".
 - Получает активный URL текущей вкладки.
 - Отправляет URL в фоновый скрипт с помощью `chrome.runtime.sendMessage`.
 - Обрабатывает ответ от фонового скрипта, выводит соответствующие сообщения об успехе или ошибке.

**Вызывает исключения**:

Нет
