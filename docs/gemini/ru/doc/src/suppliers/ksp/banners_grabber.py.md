# Модуль для сбора баннеров KSP

## Обзор

Модуль `banners_grabber.py` предназначен для сбора баннеров с сайта KSP. На данный момент содержит заглушку функции `get_banners`, которая всегда возвращает `True`.

## Подробней

Этот модуль, расположенный в `src/suppliers/ksp/`, является частью системы для работы с поставщиками (suppliers), в частности, с KSP. Он предназначен для автоматического сбора баннеров с сайта поставщика. В текущей версии, модуль содержит только определение функции `get_banners`, которая всегда возвращает `True`.
