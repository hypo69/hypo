# Модуль hypotez/src/suppliers/morlevi/header.py

## Обзор

Этот модуль содержит функции для определения корневого каталога проекта и загрузки настроек из файла `settings.json`. Он также загружает описание проекта из файла `README.MD`.  Модуль инициализирует переменные, содержащие имя проекта, версию, описание, авторов и другие метаданные.

## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная с текущего каталога. Проверяет наличие файлов-маркеров в родительских каталогах. Если корневой каталог не найден, возвращает текущий.

**Параметры**:

- `marker_files` (tuple): Кортеж с именами файлов или каталогов, по которым определяется корневой каталог проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневому каталогу проекта, или путь к текущему каталогу, если корневой каталог не найден.

**Вызывает исключения**:

- Нет

### `__root__`


**Описание**: Переменная, содержащая корневой каталог проекта, установленный функцией `set_project_root`.


## Инициализация переменных

**Описание**: Данный блок кода загружает настройки из файла `settings.json` и описание проекта из файла `README.MD`. ## Переменные

**Описание**: Следующие переменные содержат информацию о проекте:


* `__root__`: Path - Путь к корневому каталогу проекта.
* `settings`: dict - Словарь с настройками проекта из файла `settings.json`
* `doc_str`: str - Строка, содержащая описание проекта из файла `README.MD`.
* `__project_name__`: str - Имя проекта.
* `__version__`: str - Версия проекта.
* `__doc__`: str - Описание проекта.
* `__details__`: str - Дополнительная информация о проекте.
* `__author__`: str - Автор проекта.
* `__copyright__`: str - Авторские права.
* `__cofee__`: str - Ссылка на поддержку разработчика.

**Примечания**: Значения переменных, которые не загружаются из файла `settings.json`, имеют значения по умолчанию.
