# Модуль `tok_rec`

## Обзор

Модуль `tok_rec` содержит функцию для проверки, является ли строка токеном определенного формата.

## Содержание

- [Функции](#функции)
    - [`is_token`](#is_token)

## Функции

### `is_token`

**Описание**: Функция проверяет, является ли переданная строка токеном определенного формата. Токен должен состоять из 62 символов, где второй символ должен быть "/".

**Параметры**:
- `token` (str): Строка, которую необходимо проверить на соответствие формату токена.

**Возвращает**:
- `bool`: Возвращает `True`, если строка является токеном, и `False` в противном случае.
