## Классы

### `FacebookAdvertisement`

**Описание**:
Класс для взаимодействия с API Facebook и получения данных об объявлениях.

**Методы**:
- `__init__`: Инициализирует класс `FacebookAdvertisement` с токеном доступа Facebook.
- `get_ads_data`: Получает данные об объявлениях с помощью API Facebook.

#### `__init__`

**Описание**:
Инициализирует класс `FacebookAdvertisement` с токеном доступа Facebook.

**Параметры**:
- `access_token` (str): Токен доступа Facebook.

**Возвращает**:
- `None`: Метод не возвращает ничего.

#### `get_ads_data`

**Описание**:
Получает данные об объявлениях с помощью API Facebook.

**Параметры**:
- `account_id` (str): Идентификатор рекламного аккаунта Facebook.
- `fields` (str): Список полей для выборки из API Facebook. По умолчанию `None`.

**Возвращает**:
- `list | None`: Возвращает список данных об объявлениях, если запрос успешен, или `None` в случае ошибки.

**Вызывает исключения**:
- `Exception`: Возникает в случае ошибки при запросе к API Facebook.

## Функции

### `convert_date`

**Описание**:
Преобразует дату из формата Unix Timestamp в строку в формате YYYY-MM-DD.

**Параметры**:
- `unix_timestamp` (int): Дата в формате Unix Timestamp.

**Возвращает**:
- `str`: Дата в формате YYYY-MM-DD.

**Вызывает исключения**:
- `ValueError`: Возникает, если `unix_timestamp` не является целым числом.

### `get_statistic_for_all_ads`

**Описание**:
Получает статистику для всех объявлений из переданных данных.

**Параметры**:
- `ads_data` (list): Список данных об объявлениях.

**Возвращает**:
- `list`: Список словарей, где каждый словарь содержит статистику для одного объявления.

### `get_statistic_for_one_ad`

**Описание**:
Извлекает и форматирует статистику для одного объявления.

**Параметры**:
- `ad` (dict): Данные об одном объявлении.

**Возвращает**:
- `dict`: Словарь со статистикой для данного объявления.

**Вызывает исключения**:
- `KeyError`: Возникает, если в данных об объявлении отсутствует ожидаемый ключ.
