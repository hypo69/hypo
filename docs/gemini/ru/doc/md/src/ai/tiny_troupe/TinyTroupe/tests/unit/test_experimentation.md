# Модуль `test_experimentation.py`

## Обзор

Данный модуль содержит тесты для класса `ABRandomizer`, проверяя корректность рандомизации и дерандомизации.

## Функции

### `test_randomize`

**Описание**: Функция тестирует метод `randomize` класса `ABRandomizer`. Она выполняет рандомизацию 20 раз и проверяет, что результат соответствует ожидаемому, основываясь на значениях в `randomizer.choices`.

**Параметры**:
- Нет входных параметров.

**Возвращает**:
- Нет значения возврата.

**Вызывает исключения**:
- `Exception`: В случае, если результат рандомизации не соответствует ожидаемому.


### `test_derandomize`

**Описание**: Функция тестирует метод `derandomize` класса `ABRandomizer`. Она выполняет рандомизацию и затем дерандомизацию 20 раз, проверяя, что результат дерандомизации совпадает с исходными значениями.

**Параметры**:
- Нет входных параметров.

**Возвращает**:
- Нет значения возврата.

**Вызывает исключения**:
- Нет.


### `test_derandomize_name`

**Описание**: Функция тестирует метод `derandomize_name` класса `ABRandomizer`. Она выполняет рандомизацию и затем проверяет, что полученное имя соответствует ожидаемому (контроль или лечение) в зависимости от результата рандомизации.

**Параметры**:
- Нет входных параметров.

**Возвращает**:
- Нет значения возврата.

**Вызывает исключения**:
- `Exception`: В случае, если результат дерандомизации не соответствует ожидаемому.

### `test_passtrough_name`

**Описание**: Функция тестирует обработку метода `derandomize_name` в случае, когда в `ABRandomizer` передано значение `passtrough_name`. Она выполняет рандомизацию, проверяет, что если значение находится в `passtrough_name`, то метод `derandomize_name` возвращает исходное значение.

**Параметры**:
- Нет входных параметров.

**Возвращает**:
- Нет значения возврата.

**Вызывает исключения**:
- Нет.

### `test_intervention_1`

**Описание**: Функция тестирует обработку определённого вида вмешательства (в данном случае - `intervention_1`).

**Параметры**:
- Нет входных параметров.

**Возвращает**:
- Нет значения возврата.

**Вызывает исключения**:
- Нет. (Функция помечена как `pass`, поэтому здесь нет описания вызываемых исключений)
