# Модуль `schemas.py`

## Обзор

Модуль `schemas.py` содержит Pydantic модели для валидации данных, связанных с продуктами.
Включает модели для идентификации продукта и для создания/обновления данных о продукте.

## Содержание

- [Классы](#классы)
    - [`ProductIDModel`](#productidmodel)
    - [`ProductModel`](#productmodel)

## Классы

### `ProductIDModel`

**Описание**: Модель для представления идентификатора продукта.

**Параметры**:
- `id` (int): Идентификатор продукта.

### `ProductModel`

**Описание**: Модель для представления данных продукта.

**Параметры**:
- `name` (str): Название продукта. Минимальная длина - 5 символов.
- `description` (str): Описание продукта. Минимальная длина - 5 символов.
- `price` (int): Цена продукта. Должна быть больше 0.
- `category_id` (int): Идентификатор категории продукта. Должен быть больше 0.
- `file_id` (Optional[str]): Идентификатор файла продукта (может отсутствовать).
- `hidden_content` (str): Скрытый контент продукта. Минимальная длина - 5 символов.
