# Модуль `start_posting.py`

## Обзор

Модуль предназначен для автоматической отправки рекламных объявлений в группы Facebook. Он использует веб-драйвер для взаимодействия с Facebook и выполняет продвижение по заданным группам с использованием различных кампаний.

## Подробней

Этот модуль является частью системы автоматизации рекламы в Facebook. Он конфигурируется с использованием списка файлов, содержащих информацию о группах Facebook, и списка рекламных кампаний. Модуль инициализирует веб-драйвер, входит в Facebook и запускает рекламные кампании в заданных группах. Он выполняет цикл, пока не будет прерван вручную.

## Классы

### `FacebookPromoter`

**Описание**: Класс `FacebookPromoter` предназначен для управления процессом продвижения рекламы в Facebook.

**Принцип работы**:
Класс инициализируется с веб-драйвером и путями к файлам, содержащим информацию о группах Facebook. Он предоставляет методы для запуска рекламных кампаний в заданных группах.

**Методы**:

- `run_campaigns(campaigns: list, group_file_paths: list)`: Запускает рекламные кампании в заданных группах.
    - **Параметры**:
        - `campaigns` (list): Список названий кампаний для запуска.
        - `group_file_paths` (list): Список путей к файлам, содержащим информацию о группах Facebook.

## Функции

### Отсутствуют функции, описаны только методы класса `FacebookPromoter`.

## Переменные

- `d`: Инстанс веб-драйвера (Chrome), используемый для автоматизации действий в браузере.
- `filenames`: Список имен файлов, содержащих информацию о группах Facebook, в которых будет размещаться реклама.
- `excluded_filenames`: Список имен файлов, содержащих информацию об исключенных группах Facebook.
- `campaigns`: Список рекламных кампаний, которые будут запущены.
- `promoter`: Инстанс класса `FacebookPromoter`, используемый для управления процессом продвижения.

## Как работает модуль

1.  **Инициализация**:
    *   Создается инстанс веб-драйвера Chrome.
    *   Определяются списки файлов с информацией о группах (`filenames`) и исключенных группах (`excluded_filenames`).
    *   Определяется список рекламных кампаний (`campaigns`).
    *   Создается инстанс класса `FacebookPromoter` с веб-драйвером и списком файлов групп.
2.  **Цикл продвижения**:
    *   Запускается бесконечный цикл, в котором вызывается метод `run_campaigns` класса `FacebookPromoter` для запуска рекламных кампаний в заданных группах.
    *   После запуска кампаний скрипт засыпает на 180 секунд.
3.  **Прерывание**:
    *   Цикл прерывается при возникновении исключения `KeyboardInterrupt` (например, при нажатии Ctrl+C).
    *   При прерывании выводится сообщение в лог.

```
Инициализация веб-драйвера и списков групп и кампаний
↓
Запуск цикла
↓
→ Запуск рекламных кампаний (FacebookPromoter.run_campaigns)
↓
Сон на 180 секунд
↓
Повтор цикла или прерывание
```
