# Модуль gemini_traigner

## Обзор

Этот модуль предоставляет функции для обучения модели Gemini.  В настоящее время он содержит лишь константу MODE, определяющую режим работы.

## Константы

### `MODE`

**Описание**:  Константа, определяющая режим работы. В текущей реализации значение равно 'dev'.

**Значение**: 'dev'
