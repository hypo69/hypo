# Модуль `src.suppliers.aliexpress.api.helpers`

## Обзор

Модуль `src.suppliers.aliexpress.api.helpers` предоставляет набор вспомогательных функций для работы с API AliExpress. Он включает в себя функции для выполнения API-запросов, обработки аргументов, разбора продуктов и фильтрации категорий.
