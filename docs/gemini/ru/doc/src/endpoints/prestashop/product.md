# Модуль hypotez/src/endpoints/prestashop/product.py

## Обзор

Модуль `product.py` предоставляет класс `PrestaProduct`, который реализует взаимодействие с API Престашоп для работы с товарами.  Этот класс наследуется от базового класса `PrestaShop` и предоставляет методы для проверки наличия товара, расширенного поиска и получения информации о товаре по ID.

## Классы

### `PrestaProduct`

**Описание**: Класс `PrestaProduct` представляет собой сущность для работы с товарами в API Престашоп.  Он предоставляет методы для выполнения операций, связанных с товарами.

**Методы**:

- `check(product_reference: str)`: Проверяет наличие товара в базе данных по указанному `product_reference` (SKU, MKT). Возвращает словарь с информацией о товаре, если он найден, иначе `False`.
- `search(filter: str, value: str)`: Выполняет расширенный поиск товаров по указанному фильтру и значению.  Возвращает результат поиска.
- `get(id_product)`: Возвращает информацию о товаре по его `id_product`.


### `__init__`

**Описание**: Конструктор класса `PrestaProduct`. Инициализирует необходимые параметры для взаимодействия с API Престашоп, используя переданные аргументы или значения по умолчанию.

**Параметры**:

- `credentials (Optional[dict | SimpleNamespace], optional):` Словарь или объект SimpleNamespace с параметрами `api_domain` и `api_key`. По умолчанию `None`.
- `api_domain (Optional[str], optional):` Домен API. По умолчанию `None`.
- `api_key (Optional[str], optional):` Ключ API. По умолчанию `None`.

**Возвращает**:
- Нет возвращаемых значений


**Вызывает исключения**:
- `ValueError`:  Если не указаны оба параметра `api_domain` и `api_key`.
