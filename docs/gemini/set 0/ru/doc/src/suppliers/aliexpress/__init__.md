# Модуль `hypotez/src/suppliers/aliexpress/__init__.py`

## Обзор

Этот модуль предоставляет интерфейс для работы с поставщиком AliExpress. Он импортирует классы и функции для работы с API AliExpress, создания рекламных кампаний и генерации HTML-отчетов.

## Описание модуля

Модуль `aliexpress` содержит константу `MODE` и импорты, необходимые для взаимодействия с API AliExpress, создания рекламных кампаний и работы с генерацией HTML отчетов.

## Константы

### `MODE`

**Описание**: Строковая константа, определяющая режим работы модуля. В данном случае это `'dev'`.

## Импорты

### `Aliexpress`

**Описание**: Класс, представляющий работу с AliExpress.

### `AliApi`

**Описание**: Класс, представляющий работу с API AliExpress.

### `AliRequests`

**Описание**: Класс, представляющий работу с запросами к API AliExpress.

### `AliCampaignEditor`

**Описание**: Класс, отвечающий за создание и редактирование рекламных кампаний на AliExpress.

### `ProductHTMLGenerator`

**Описание**: Класс для генерации HTML-отчетов для продуктов.

### `CategoryHTMLGenerator`

**Описание**: Класс для генерации HTML-отчетов для категорий.

### `CampaignHTMLGenerator`

**Описание**: Класс для генерации HTML-отчетов для рекламных кампаний.


**Примечания:**  Для более подробной информации необходимо ознакомиться с определениями импортированных классов и функций из соответствующих файлов.
