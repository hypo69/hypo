Aliexpress Affiliated Products Generator
=======================================

.. module:: hypotez.src.suppliers.aliexpress.affiliated_products_generator
    :platform: Windows, Unix
    :synopsis: Module for generating affiliate products from AliExpress.

This module provides functionalities for collecting full product data, including affiliate links and saved images/videos, from AliExpress product URLs or IDs. It leverages the `AliApi` class and handles various aspects of data processing, image/video saving, and output generation. The module includes error handling and logging mechanisms.
