# Yahtzee

## Обзор

Данный модуль реализует игру Yahtzee (кости). Игрок бросает пять кубиков и пытается набрать очки, составляя различные комбинации. Игра состоит из 13 раундов, в каждом из которых игрок может перебросить кубики до трех раз, после чего выбирает категорию для записи очков.

## Функции

### `roll_dice`

**Описание**: 
Бросает 5 кубиков и возвращает результаты.

**Параметры**: 
Отсутствуют.

**Возвращает**:
- `list[int]`: Список из 5 целых чисел, представляющих результаты броска кубиков (от 1 до 6).

### `calculate_score`

**Описание**:
Вычисляет очки для выбранной категории.

**Параметры**:
-   `dice` (list[int]): Список из 5 целых чисел, представляющих результаты броска кубиков.
-   `category` (str): Строка, представляющая категорию, для которой нужно рассчитать очки.

**Возвращает**:
-   `int`: Целое число, представляющее количество очков, полученных за выбранную категорию.

### `play_yahtzee`

**Описание**:
Основная функция для игры в Yahtzee. Реализует игровой процесс, включая броски костей, выбор категорий и подсчет очков.

**Параметры**:
Отсутствуют.

**Возвращает**:
Отсутствует. Функция выводит результаты игры в консоль.

**Вызывает исключения**:
- `ValueError`: Возникает в случаях неверного ввода индексов кубиков или номера категории.
