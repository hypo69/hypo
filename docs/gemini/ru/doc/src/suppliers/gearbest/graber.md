# Модуль hypotez/src/suppliers/gearbest/graber.py

## Обзор

Модуль `hypotez/src/suppliers/gearbest/graber.py` содержит класс `Graber`, предназначенный для сбора данных со страниц товаров на сайте gearbest.com.  Класс наследуется от класса `Grbr` из родительского модуля `src.suppliers.graber`. Он реализует функции для обработки различных полей товара, а также предоставляет возможность переопределения стандартной обработки полей для специфических случаев.  Модуль использует веб-драйвер для взаимодействия с сайтом и логирует ошибки.  В классе реализована возможность использования декоратора для предварительных действий (закрытия всплывающих окон) перед выполнением основной функции сбора данных.
