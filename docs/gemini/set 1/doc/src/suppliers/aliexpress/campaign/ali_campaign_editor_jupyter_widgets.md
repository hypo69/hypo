# Модуль hypotez/src/suppliers/aliexpress/campaign/ali_campaign_editor_jupyter_widgets.py

## Обзор

Этот модуль содержит виджеты для управления кампаниями AliExpress в Jupyter Notebook.  Он позволяет пользователю выбирать кампании, категории и языки, инициализировать редакторы, сохранять кампании и отображать продукты.

## Классы

### `JupyterCampaignEditorWidgets`

**Описание**: Класс предоставляет виджеты для взаимодействия с и управления кампаниями AliExpress, включая выбор кампаний, категорий и языков, а также выполнение действий, таких как инициализация редакторов, сохранение кампаний и отображение продуктов.

**Атрибуты класса:**

- `language: str`: Язык кампании.
- `currency: str`: Валюта кампании.
- `campaign_name: str`: Имя выбранной кампании.
- `category_name: str`: Имя выбранной категории.
- `category: SimpleNamespace`: Объект, представляющий выбранную категорию.
- `campaign_editor: AliCampaignEditor`: Объект редактора кампании.
- `products: list[SimpleNamespace]`: Список продуктов выбранной категории.

**Методы:**

#### `__init__(self)`

**Описание**: Инициализирует виджеты и настраивает редактор кампании.

**Детали:**
- Настраивает виджеты для выбора кампаний, категорий и языков.
- Устанавливает значения по умолчанию и обратные вызовы для виджетов.
- Инициализирует директорию кампаний.

#### `initialize_campaign_editor(self, _)`

**Описание**: Инициализирует редактор кампаний.

**Параметры:**
- `_`: Неиспользуемый аргумент, необходимый для обратного вызова кнопки.

**Детали:**
- Устанавливает `campaign_name`, `category_name`, `language` и `currency` из выбранных значений виджетов.
- Создает экземпляр `AliCampaignEditor` с заданными параметрами.
- Если `category_name` задан, получает соответствующую категорию и продукты.
- Выводит предупреждение, если `campaign_name` не выбран.


#### `update_category_dropdown(self, campaign_name: str)`

**Описание**: Обновляет выпадающий список категорий на основе выбранной кампании.

**Параметры:**
- `campaign_name (str)`: Имя кампании.

**Детали:**
- Определяет путь к директории категорий для текущей кампании.
- Получает список имен категорий.
- Обновляет выпадающий список категорий.

#### `on_campaign_name_change(self, change: dict[str, str])`

**Описание**: Обрабатывает изменения в выпадающем списке имен кампаний.

**Параметры:**
- `change (dict[str, str])`: Словарь изменения, содержащий новое значение.

**Детали:**
- Устанавливает `campaign_name` на новое значение.
- Обновляет выпадающий список категорий.
- Переинициализирует редактор кампании.

#### `on_category_change(self, change: dict[str, str])`

**Описание**: Обрабатывает изменения в выпадающем списке категорий.

**Параметры:**
- `change (dict[str, str])`: Словарь изменения, содержащий новое значение.

**Детали:**
- Устанавливает `category_name` на новое значение.
- Переинициализирует редактор кампании.

#### `on_language_change(self, change: dict[str, str])`

**Описание**: Обрабатывает изменения в выпадающем списке языков.

**Параметры:**
- `change (dict[str, str])`: Словарь изменения, содержащий новое значение.

**Детали:**
- Разделяет значение языка/валюты.
- Переинициализирует редактор кампании.

#### `save_campaign(self, _)`

**Описание**: Сохраняет кампанию и ее категории.

**Параметры:**
- `_`: Неиспользуемый аргумент, необходимый для обратного вызова кнопки.

**Детали:**
- Сохраняет категории выбранной кампании в Google Таблицу. Обрабатывает потенциальные ошибки.

#### `show_products(self, _)`

**Описание**: Отображает продукты в выбранной категории.

**Параметры:**
- `_`: Неиспользуемый аргумент, необходимый для обратного вызова кнопки.

**Детали:**
- Отображает продукты выбранной категории. Обрабатывает потенциальные ошибки.


#### `open_spreadsheet(self, _)`

**Описание**: Открывает Google Таблицу в браузере.

**Параметры:**
- `_`: Неиспользуемый аргумент, необходимый для обратного вызова кнопки.

**Детали:**
- Открывает ссылку на Google Таблицу в браузере.

#### `setup_callbacks(self)`

**Описание**: Настраивает обратные вызовы для виджетов.

#### `display_widgets(self)`

**Описание**: Отображает виджеты для взаимодействия в Jupyter Notebook.
