# Модуль ROCKET

## Обзор

Этот модуль реализует текстовую игру "Ракета", в которой игрок пытается запустить ракету, вводя начальную скорость и угол наклона. Компьютер рассчитывает траекторию полета и сообщает, достигла ли ракета цели или нет. Игрок должен подобрать параметры запуска, чтобы достичь цели.

## Константы

### `GRAVITY`

**Описание**: Константа, представляющая ускорение свободного падения в футах в секунду в квадрате.

**Значение**: `32.2`

## Основной игровой цикл

Основной игровой цикл позволяет игроку многократно запускать ракету, пока он не решит выйти из игры.

### Цикл `while True`

**Описание**: Бесконечный цикл, который продолжается до тех пор, пока игрок не введет отрицательную скорость.

### Ввод данных

#### `input("Введите начальную скорость ракеты (футы/сек) (отрицательное значение для выхода): ")`

**Описание**: Запрашивает у пользователя начальную скорость ракеты.

**Параметры**:
- Нет.

**Возвращает**:
- `str`: Введенная пользователем строка.

#### `velocity = float(...)`

**Описание**: Преобразует введенную пользователем строку в число с плавающей точкой.

**Параметры**:
- Введенная пользователем строка.

**Возвращает**:
- `float`: Число с плавающей точкой.

#### `input("Введите угол наклона (в градусах): ")`

**Описание**: Запрашивает у пользователя угол наклона.

**Параметры**:
- Нет.

**Возвращает**:
- `str`: Введенная пользователем строка.

#### `angle = float(...)`

**Описание**: Преобразует введенную пользователем строку в число с плавающей точкой.

**Параметры**:
- Введенная пользователем строка.

**Возвращает**:
- `float`: Число с плавающей точкой.

### Обработка выхода из игры

#### `if velocity < 0:`

**Описание**: Проверяет, ввел ли пользователь отрицательную скорость, сигнализируя о желании выйти из игры.

#### `print("Игра завершена.")`

**Описание**: Выводит сообщение о завершении игры.

#### `break`

**Описание**: Завершает текущий цикл.

### Преобразование угла

#### `angle_radians = math.radians(angle)`

**Описание**: Преобразует угол из градусов в радианы.

**Параметры**:
- `angle` (`float`): Угол в градусах.

**Возвращает**:
- `float`: Угол в радианах.

### Вычисление дальности полета

#### `distance = (velocity**2 * math.sin(2 * angle_radians)) / GRAVITY`

**Описание**: Вычисляет дальность полета ракеты по формуле.

**Параметры**:
- `velocity` (`float`): Начальная скорость ракеты.
- `angle_radians` (`float`): Угол в радианах.
- `GRAVITY` (`float`): Ускорение свободного падения.

**Возвращает**:
- `float`: Дальность полета ракеты.

### Вывод результатов

#### `print(f"Дальность полета ракеты: {distance:.2f} футов")`

**Описание**: Выводит рассчитанную дальность полета с двумя знаками после запятой.

**Параметры**:
- `distance` (`float`): Дальность полета ракеты.

**Возвращает**:
- Нет.

### Проверка достижения цели

#### `if distance >= 1000:`

**Описание**: Проверяет, достигла ли ракета цели (дальность 1000 футов и более).

#### `print("GOOD SHOT")`

**Описание**: Выводит сообщение о том, что ракета достигла цели.

#### `else:`

**Описание**: Выполняется, если дальность полета меньше 1000 футов.

#### `print("SHORT SHOT")`

**Описание**: Выводит сообщение о том, что ракета не достигла цели.

## Обработка ошибок

### Блок `try...except ValueError`

**Описание**: Блок `try...except` используется для обработки возможных ошибок ввода пользователя. Если пользователь вводит нечисловое значение, программа не завершится с ошибкой, а выведет сообщение об ошибке.

#### `except ValueError as ex:`

**Описание**: Перехватывает исключение `ValueError`, которое может возникнуть, если пользователь вводит нечисловое значение.
**Параметры**:
- `ex` (`ValueError`): Объект исключения.

#### `print("Ошибка ввода. Пожалуйста, введите числовое значение.")`

**Описание**: Выводит сообщение об ошибке ввода.
