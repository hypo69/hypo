# Модуль `tok_rec.py`

## Обзор

Модуль `tok_rec.py` предназначен для проверки, является ли предоставленная строка токеном определенного формата. В частности, он проверяет, соответствует ли строка определенной длине и содержит ли определенный символ в определенной позиции.

## Подробней

Этот модуль содержит функцию `is_token`, которая принимает строку в качестве аргумента и возвращает `True`, если строка соответствует определенным критериям, позволяющим считать ее токеном. В противном случае функция возвращает `False`. Этот модуль, вероятно, используется для идентификации или валидации токенов, отправленных в систему, возможно, как часть процесса аутентификации или авторизации.

## Функции

### `is_token`

```python
def is_token(token):
    """
    Args:
        token: Проверяемая строка.

    Returns:
        bool: `True`, если строка является токеном, `False` в противном случае.
    """
```

**Описание**: Определяет, является ли предоставленная строка токеном.

**Параметры**:
- `token`: Строка, которую необходимо проверить.

**Возвращает**:
- `bool`: `True`, если строка соответствует критериям токена (длина 62 символа и символ "/" на второй позиции), иначе `False`.

**Примеры**:

```python
>>> is_token('some string /and some other text')
False

>>> is_token('very long token with / symbol and some other text')
False

>>> is_token('1/verylongtokenwithslash/symbolandthestringisexactly62symbols')
False

>>> is_token('1/verylongtokenwithslashsymbolandthestringisexactly62symbols')
True
