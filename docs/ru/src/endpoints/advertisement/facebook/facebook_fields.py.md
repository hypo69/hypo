# Модуль `facebook_fields`

## Обзор

Модуль `facebook_fields.py` предназначен для определения и загрузки полей, используемых в рекламных объявлениях и событиях Facebook. Он обеспечивает централизованное управление набором полей, необходимых для взаимодействия с API Facebook.

## Подробней

Этот модуль предназначен для упрощения работы с полями Facebook, используемыми в рекламных кампаниях и событиях. Он загружает поля из JSON-файла и делает их доступными через атрибуты класса `FacebookFields`. Это позволяет избежать жесткого кодирования полей в коде и упрощает их изменение и обновление. Класс использует модуль `logger` для логирования ошибок при загрузке данных.

## Классы

### `FacebookFields`

**Описание**: Класс `FacebookFields` предназначен для хранения и управления полями, используемыми в рекламных объявлениях и событиях Facebook.

**Как работает класс**:
1. При инициализации класса вызывается метод `_payload`, который загружает данные полей из JSON-файла.
2. Данные из JSON-файла преобразуются в атрибуты класса, что позволяет обращаться к ним через `self`.
3. В случае ошибки загрузки данных, информация об ошибке логируется с использованием `logger.debug`.

**Методы**:

- `__init__(self)`:
    *   **Описание**: Инициализирует класс `FacebookFields` и вызывает метод `_payload` для загрузки данных.
    *   **Как работает метод**:
        1.  Вызывается метод `_payload` для загрузки данных полей Facebook.
    *   **Параметры**:
        *   `self` (FacebookFields): Объект класса `FacebookFields`.
    *   **Возвращает**:
        *   None
    *   **Вызывает исключения**:
        *   Нет

- `_payload(self)`:
    *   **Описание**: Загружает данные полей Facebook из JSON-файла и устанавливает их как атрибуты класса.
    *   **Как работает метод**:
        1.  Формируется путь к JSON-файлу с использованием `gs.path.src` и других компонентов пути.
        2.  JSON-файл загружается с использованием функции `j_loads`.
        3.  Если загрузка не удалась (например, файл не найден или поврежден), в лог записывается сообщение об ошибке с использованием `logger.debug`, и функция возвращает `None`.
        4.  Если загрузка прошла успешно, данные из JSON-файла (в виде словаря) итерируются, и каждый ключ-значение становится атрибутом класса.
        5.  После успешной загрузки и установки атрибутов функция возвращает `True`.
    *   **Параметры**:
        *   `self` (FacebookFields): Объект класса `FacebookFields`.
    *   **Возвращает**:
        *   `True` в случае успешной загрузки и установки атрибутов, `None` в случае ошибки.
    *   **Вызывает исключения**:
        *   Нет

## Переменные

- `data`:
    *   **Описание**: Содержит данные, загруженные из JSON-файла.
    *   **Как используется**: Используется для итерации по полям Facebook и установки их как атрибутов класса.
- `name`:
    *   **Описание**: Содержит имя поля Facebook.
    *   **Как используется**: Используется в качестве ключа для установки атрибута класса.
- `value`:
    *   **Описание**: Содержит значение поля Facebook.
    *   **Как используется**: Используется в качестве значения атрибута класса.
