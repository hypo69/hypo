# Модуль hypotez/src/suppliers/visualdg/graber.py

## Обзор

Модуль `hypotez/src/suppliers/visualdg/graber.py` содержит класс `Graber`, предназначенный для сбора данных о товарах с сайта `visualdg.co.il`. Класс наследуется от базового класса `Graber` и предоставляет методы для извлечения различных полей товара.  Внутри класса реализованы асинхронные функции для обработки данных, а также декоратор для закрытия всплывающих окон (хотя декоратор и не используется).

## Классы

### `Graber`

**Описание**: Класс `Graber` расширяет базовый класс `Grbr` для сбора данных с сайта `visualdg.co.il`.  Он содержит методы для извлечения данных о товаре, включая поля из родительского класса, а также дополнительные методы (не реализованные) для обработки специфичных полей.

**Атрибуты**:

- `supplier_prefix`: Префикс поставщика (`visualdg`).

**Методы**:

#### `__init__(self, driver: Driver)`

**Описание**: Инициализирует экземпляр класса `Graber`.

**Параметры**:

- `driver (Driver)`: Экземпляр класса `Driver` для работы с веб-драйвером.

#### `grab_page(self, driver: Driver) -> ProductFields`

**Описание**: Асинхронная функция для сбора данных о товаре.

**Параметры**:

- `driver (Driver)`: Экземпляр класса `Driver` для работы с веб-драйвером.

**Возвращает**:

- `ProductFields`: Объект, содержащий собранные данные о товаре.
