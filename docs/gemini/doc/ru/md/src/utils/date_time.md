# Модуль `hypotez/src/utils/date_time.py`

## Обзор

Модуль `date_time.py` содержит функции для проверки текущего времени относительно заданного интервала с опциональным таймаутом.  Функции полезны для запуска операций, которые должны выполняться только в определенные временные интервалы (например, для выполнения задач поздней ночью).

## Классы

### `TimeoutCheck`

**Описание**: Класс `TimeoutCheck` предоставляет методы для проверки времени и обработки ввода от пользователя с таймаутом.

**Методы**:

#### `interval`

**Описание**: Проверяет, находится ли текущее время в заданном интервале.

**Параметры**:

- `start` (time): Начало интервала (по умолчанию 23:00).
- `end` (time): Конец интервала (по умолчанию 06:00).

**Возвращает**:

- `bool`: `True`, если текущее время находится в интервале, иначе `False`.


#### `interval_with_timeout`

**Описание**: Проверяет, находится ли текущее время в заданном интервале с заданным таймаутом.

**Параметры**:

- `timeout` (int): Время ожидания в секундах.
- `start` (time): Начало интервала (по умолчанию 23:00).
- `end` (time): Конец интервала (по умолчанию 06:00).

**Возвращает**:

- `bool`: `True`, если текущее время находится в интервале и проверка завершена успешно, иначе `False`. Возвращает `False`, если произошел таймаут.


#### `get_input`

**Описание**: Запрашивает ввод от пользователя.

**Возвращает**:

- Ничего. Сохраняет введенные данные в `self.user_input`


#### `input_with_timeout`

**Описание**: Ожидает ввод от пользователя с таймаутом.

**Параметры**:

- `timeout` (int): Время ожидания в секундах.

**Возвращает**:

- `str | None`: Введенные данные или `None`, если произошёл таймаут.


## Функции

В этом модуле нет отдельных функций, только методы класса `TimeoutCheck`.
