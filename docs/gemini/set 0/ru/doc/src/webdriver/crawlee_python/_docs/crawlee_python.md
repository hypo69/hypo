# Модуль crawlee_python

## Обзор

Этот модуль предоставляет класс `CrawleePython` для выполнения веб-скрейпинга с помощью библиотеки `crawlee` и Playwright. Класс позволяет настроить, запустить и экспортировать данные, полученные с веб-сайта. Поддерживается асинхронная работа, что делает процесс эффективным для обработки множества запросов.

## Классы

### `CrawleePython`

**Описание**: Класс `CrawleePython` используется для настройки, запуска и обработки результатов веб-скрейпинга с использованием Playwright.

**Методы**:

#### `__init__(self, max_requests: int, headless: bool = True, browser_type: str = 'chromium')`

**Описание**: Конструктор класса. Инициализирует `PlaywrightCrawler` с заданными параметрами.

**Параметры**:
- `max_requests` (int): Максимальное количество запросов во время выполнения.
- `headless` (bool, по умолчанию True): Флаг, указывающий на использование без графического интерфейса (headless).
- `browser_type` (str, по умолчанию 'chromium'): Тип браузера для использования (например, 'chromium', 'firefox').

#### `setup_crawler(self)`

**Описание**: Настройка обработчика запросов для `PlaywrightCrawler`. Обрабатывает каждый запрос, извлекает данные с страницы и добавляет ссылки в очередь для дальнейшего сканирования.

**Возвращает**:
- None

#### `run_crawler(self, initial_urls: list)`

**Описание**: Запускает процесс сканирования с заданным списком начальных URL.

**Параметры**:
- `initial_urls` (list): Список начальных URL для сканирования.

**Возвращает**:
- None

#### `export_data(self, data: list, filename: str = 'crawled_data.json')`

**Описание**: Экспортирует собранные данные в файл JSON.

**Параметры**:
- `data` (list): Список данных в формате словарей.
- `filename` (str, по умолчанию 'crawled_data.json'): Имя файла для экспорта.

**Возвращает**:
- None

#### `get_data(self)`

**Описание**: Возвращает собранные данные в формате словаря.

**Возвращает**:
- dict: Словарь, содержащий собранные данные.


#### `run(self, initial_urls: list)`

**Описание**: Основной метод для запуска процесса сканирования. Настраивает сканер, выполняет сканирование, экспортирует данные и выводит результат.

**Параметры**:
- `initial_urls` (list): Список начальных URL для сканирования.

**Возвращает**:
- None
