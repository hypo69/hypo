# Модуль `header.py`

## Обзор

Модуль `header.py` содержит основные настройки и метаданные проекта, а также утилиты для определения корневой директории проекта.

## Функции

### `set_project_root`

**Описание**:
Находит корневую директорию проекта, начиная с директории текущего файла,
поиска вверх и останавливаясь на первой директории, содержащей любой из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple): Имена файлов или каталогов для определения корневой директории проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена, в противном случае - директория, где находится скрипт.

## Переменные

### `__root__`
**Описание**: 
Путь к корневой директории проекта.

### `settings`
**Описание**: 
Словарь с настройками проекта, загруженный из `settings.json`.

### `doc_str`
**Описание**: 
Строка с содержимым файла `README.md`.

### `__project_name__`
**Описание**: 
Название проекта, по умолчанию `'hypotez'`.

### `__version__`
**Описание**: 
Версия проекта.

### `__doc__`
**Описание**: 
Содержимое документации проекта.

### `__details__`
**Описание**: 
Детали проекта (в настоящее время пустая строка).

### `__author__`
**Описание**: 
Автор проекта.

### `__copyright__`
**Описание**: 
Информация об авторских правах проекта.

### `__cofee__`
**Описание**: 
Сообщение с предложением угостить разработчика кофе.
