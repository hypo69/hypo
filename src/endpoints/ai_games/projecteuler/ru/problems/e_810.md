# Задача No 810
## Числа, составленные из 2 и 3
## Условия

Назовем число "составленным из 2 и 3", если оно является произведением неотрицательных целых степеней 2 и 3.

Первые несколько таких чисел:
1, 2, 3, 4, 6, 8, 9, 12, 16, 18, 24, 27, 32, 36, 48, 54, 64, 72, 81, 96, 108, 128, 144, 162, 192, 216, 243, 256, 288, 324, 384, 432, 486, 512, 576, 648, 729, 768, 864, 972, 1024, 1152, 1296, 1458, 1536, 1728, 1944, 2048, 2187, 2304, 2592, 2916, 3072, 3456, 3888, 4096, …

Дано число $n$. Пусть $f(n)$ будет количеством "составленных из 2 и 3" чисел, не превосходящих $n$.

Например, $f(100) = 19$ и $f(10^6) = 1727$.

Найдите $f(10^{18})$.
