# Модуль hypotez/src/webdriver/bs/bs.py

## Обзор

Модуль `hypotez/src/webdriver/bs/bs.py` предоставляет класс `BS` для парсинга HTML-страниц с использованием библиотек `BeautifulSoup` и `lxml`.  Он позволяет получать содержимое страниц как из файлов, так и из URL-адресов, а также искать элементы по различным локеторам (ID, CSS, текст).

## Классы

### `BS`

**Описание**: Класс `BS` предназначен для парсинга HTML-содержимого. Он хранит HTML-контент и предоставляет методы для его получения и поиска элементов.

**Методы**:

#### `__init__(self, url: str | None = None)`

**Описание**: Конструктор класса `BS`. Принимает URL-адрес (или путь к файлу) для последующего парсинга.

**Параметры**:

- `url` (str | None, optional): URL-адрес (или путь к файлу) для загрузки HTML-содержимого. По умолчанию `None`.

#### `get_url(self, url: str)`

**Описание**: Загружает HTML-содержимое из указанного URL или файла.

**Параметры**:

- `url` (str): Путь к файлу или URL-адрес, содержащий HTML.

**Возвращает**:

- bool: `True`, если загрузка прошла успешно, иначе `False`.

**Вызывает исключения**:

- `Exception`:  Возникает при проблемах с чтением файла.
- `requests.RequestException`:  Возникает при проблемах с запросом к URL.


#### `execute_locator(self, locator: SimpleNamespace | dict, url: str = None)`

**Описание**:  Ищет элементы на странице по заданному локетору.

**Параметры**:

- `locator` (SimpleNamespace | dict): Объект `SimpleNamespace` или словарь, содержащий данные локетора (атрибут, тип селектора и сам селектор).
- `url` (str, optional):  URL-адрес страницы для парсинга. Если указан, метод сначала загрузит содержимое страницы с помощью `get_url`. По умолчанию `None`.

**Возвращает**:

- list: Список элементов, соответствующих заданному локетору. Возвращает `None`, если локетор не найден или произошла ошибка.

**Вызывает исключения**:

- `Exception`:  Возникает при проблемах с поиском элементов.

## Пример использования

```python
# Создаем экземпляр класса BS
bs_parser = BS()

# Загрузка HTML из файла
if bs_parser.get_url('file:///path/to/your/file.html'):
  # Ищем элементы по ID
  locator = SimpleNamespace(attribute='my-element-id', by='id', selector='//*[@id="my-element-id"]')
  elements = bs_parser.execute_locator(locator)
  if elements:
    print(f"Найдено {len(elements)} элементов с ID='my-element-id'")
  else:
    print("Элементы с указанным ID не найдены")
```

**Примечания**:

- Код в блоке `if __name__ == "__main__":` используется для демонстрации работы модуля и должен быть удален из итоговой реализации.
- В примере указан несуществующий файл, подставьте правильный путь к файлу.
-  `locator` должен быть корректно сформирован с необходимыми значениями атрибута, типа локетора (`by` - 'id', 'css', 'text') и селектора.
- Обработка исключений (в `get_url`) должна быть расширена, чтобы более точно определять типы ошибок и принимать соответствующие действия.

-  Для корректного использования `execute_locator` обязательно передавайте `locator` типа `SimpleNamespace` или `dict`.
