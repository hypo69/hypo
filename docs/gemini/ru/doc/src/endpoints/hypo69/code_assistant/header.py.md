# `src.logger.header`

## Обзор

Модуль `src.logger.header` определяет корневой путь к проекту и предоставляет доступ к основным настройкам проекта, а также метаданным проекта. Все импорты в проекте строятся относительно этого пути.

## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная с каталога текущего файла, ища вверх по иерархии каталогов, пока не найдет каталог, содержащий один из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple, optional): Кортеж имен файлов или каталогов, используемых для идентификации корневого каталога проекта. По умолчанию `('__root__','.git')`.

**Возвращает**:
- `Path`: Путь к корневому каталогу проекта, если он найден. В противном случае возвращает каталог, где расположен скрипт.

## Глобальные переменные

- `__root__` (Path): Путь к корневому каталогу проекта.
- `settings` (dict | None): Словарь с настройками проекта, загруженными из `settings.json`.
- `doc_str` (str | None): Строка с содержимым файла `README.MD`.
- `__project_name__` (str): Название проекта. Значение берется из `settings.json` или 'hypotez' по умолчанию.
- `__version__` (str): Версия проекта. Значение берется из `settings.json` или '' по умолчанию.
- `__doc__` (str): Описание проекта, загруженное из `README.MD` или '' по умолчанию.
- `__details__` (str): Детальная информация о проекте. В данный момент не определена (равна '').
- `__author__` (str): Автор проекта. Значение берется из `settings.json` или '' по умолчанию.
- `__copyright__` (str): Информация об авторских правах. Значение берется из `settings.json` или '' по умолчанию.
- `__cofee__` (str): Сообщение с предложением поддержать разработчика. Значение берется из `settings.json` или строка по умолчанию.
