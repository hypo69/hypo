# Модуль graber

## Обзор

Модуль `graber.py` предназначен для сбора данных о товарах с сайта `visualdg.co.il`. Он наследуется от класса `Graber` из модуля `src.suppliers.graber` и переопределяет методы для нестандартной обработки полей, специфичных для данного сайта. Модуль использует декоратор для выполнения предварительных действий перед отправкой запроса к веб-драйверу, таких как закрытие всплывающих окон.

## Подробней

Этот модуль является частью системы сбора данных о товарах с различных онлайн-магазинов. Он специализируется на парсинге сайта `visualdg.co.il`. Для каждого поля товара создана функция обработки, при этом, если требуется нестандартная обработка, функция перегружается в данном классе. Перед отправкой запроса к веб-драйверу могут выполняться предварительные действия с использованием декоратора.

## Классы

### `Graber`

**Описание**: Класс `Graber` предназначен для операций захвата данных с сайта `visualdg.co.il`. Он наследуется от класса `Graber` (переименованного в `Grbr` для избежания конфликта имен) из модуля `src.suppliers.graber`.

**Методы**:
- `__init__`: Инициализирует экземпляр класса `Graber`, устанавливает префикс поставщика и вызывает конструктор родительского класса.

**Параметры**:
- `driver` (Driver): Экземпляр веб-драйвера для управления браузером.
- `lang_index` (int): Индекс языка.

**Примеры**
```python
from src.webdriver.driver import Driver

# Пример создания экземпляра класса Graber
driver = Driver()
graber = Graber(driver=driver, lang_index=0)
```
