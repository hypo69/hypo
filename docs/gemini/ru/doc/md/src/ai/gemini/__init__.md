# Модуль hypotez/src/ai/gemini/__init__.py

## Обзор

Этот модуль предоставляет интерфейс к модели Gemini.  Он содержит константу `MODE` для указания режима работы (в данном случае 'dev') и импортирует класс `GoogleGenerativeAI` из модуля `generative_ai`.

## Константа MODE

### `MODE`

**Описание**:  Константа, определяющая режим работы. В данном случае, это 'dev'.

**Значение**:  'dev'


## Импорты

### `from .generative_ai import GoogleGenerativeAI`

**Описание**: Импортирует класс `GoogleGenerativeAI` из подмодуля `generative_ai`.  Этот класс, скорее всего, содержит реализацию взаимодействия с моделью Gemini.
