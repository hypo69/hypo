# Задача No 495
## Совершенная перестановка
## Условия
Перестановка *p* множества {1, 2, ..., *n*} называется *совершенной*, если |*p<sub>i</sub>* - *i*| = 1 для всех *i*.
Например, перестановка (2, 1, 4, 3) является совершенной, а перестановки (1, 2, 3, 4) и (2, 3, 4, 1) не являются.

Пусть *P*(*n*) — количество совершенных перестановок множества {1, 2, ..., *n*}.

Вы можете проверить, что *P*(1) = 1, *P*(2) = 2, *P*(3) = 2 и *P*(4) = 4.

Пусть *S*(*n*) = Σ *P*(*i*) для *i* = 1 до *n*.
Вы можете проверить, что *S*(10) = 324.

Найдите *S*(10<sup>7</sup>) mod 1000000007.
