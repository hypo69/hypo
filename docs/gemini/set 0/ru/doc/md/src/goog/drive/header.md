# Модуль hypotez/src/goog/drive/header.py

## Обзор

Этот модуль предоставляет функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и документации из `README.MD`.  Он использует функцию `set_project_root()` для поиска корневой директории, а затем загружает данные.  Модуль также определяет переменные, содержащие информацию о проекте, такие как имя, версия, описание и другие метаданные.

## Функции

### `set_project_root`

**Описание**:  Находит корневую директорию проекта, начиная с текущей директории файла и поднимаясь вверх по иерархии директорий до тех пор, пока не будет найдена директория, содержащая один из указанных файлов или директорий.

**Параметры**:
- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, используемых для определения корневой директории проекта. По умолчанию это `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь до корневой директории проекта, если найдена. В противном случае возвращается путь к директории, в которой находится текущий файл.

**Вызывает исключения**:
- Нет


## Переменные

### `__root__`

**Описание**:  Путь к корневой директории проекта. Инициализируется функцией `set_project_root()`. Тип `Path`.


### `settings`

**Описание**: Словарь с настройками проекта, загруженными из файла `settings.json` в корневой директории проекта. Инициализируется с `None` и загружается из файла в случае успеха. Тип `dict`.

### `doc_str`

**Описание**: Строка документации, загруженная из файла `README.MD` в корневой директории проекта. Инициализируется с `None` и загружается из файла в случае успеха. Тип `str`.

### `__project_name__`

**Описание**: Имя проекта. Получается из настроек (`settings`) или устанавливается по умолчанию как `'hypotez'`. Тип `str`.

### `__version__`

**Описание**: Версия проекта. Получается из настроек (`settings`) или устанавливается по умолчанию как пустая строка. Тип `str`.

### `__doc__`

**Описание**: Документация проекта. Получается из настроек (`doc_str`) или устанавливается по умолчанию как пустая строка. Тип `str`.

### `__details__`

**Описание**: Дополнительная информация о проекте. По умолчанию пустая строка. Тип `str`.

### `__author__`

**Описание**: Автор проекта. Получается из настроек (`settings`) или устанавливается по умолчанию как пустая строка. Тип `str`.

### `__copyright__`

**Описание**: Авторские права. Получается из настроек (`settings`) или устанавливается по умолчанию как пустая строка. Тип `str`.

### `__cofee__`

**Описание**: Ссылка для поддержки разработчика. Получается из настроек (`settings`) или устанавливается по умолчанию. Тип `str`.

**Обработка исключений:**

В модуле присутствует обработка исключений `FileNotFoundError` и `json.JSONDecodeError` при чтении файлов `settings.json` и `README.MD`.  Это позволяет программе продолжать работу, даже если эти файлы не найдены или содержат некорректные данные.
