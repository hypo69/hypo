# Документация для `bathroom.json`

## Обзор

Файл `bathroom.json` содержит конфигурацию сценариев для обработки товаров категории "ванные комнаты" от поставщика Kualastyle. Включает в себя настройки для URL, условия добавления товаров, привязки к категориям PrestaShop и правила ценообразования.

## Структура

### `scenarios`

**Описание**: Основной раздел, содержащий настройки для различных сценариев обработки товаров.

### `Designed sinks`

**Описание**: Сценарий для обработки категории "Designed sinks".

#### `url`

**Описание**: URL-адрес страницы с товарами "Designed sinks" на сайте поставщика.

**Значение**:
```
https://kualastyle.com/collections/%D7%9B%D7%99%D7%95%D7%A8%D7%99%D7%9D-%D7%9E%D7%A2%D7%95%D7%A6%D7%91%D7%99%D7%9D
```

#### `condition`

**Описание**: Условие для добавления товаров.

**Значение**: `"new"` (новый товар).

#### `presta_categories`

**Описание**: Раздел для определения соответствий категорий PrestaShop.

##### `default_category`

**Описание**: Словарь, связывающий ID категории PrestaShop с названием категории.

**Значение**: `{"11005": "Designed sinks"}`.
- ID категории `11005` соответствует категории `Designed sinks`.

#### `price_rule`

**Описание**: Правило ценообразования.

**Значение**: `1`.
