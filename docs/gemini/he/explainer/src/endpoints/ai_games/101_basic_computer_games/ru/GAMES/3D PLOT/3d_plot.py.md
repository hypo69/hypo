3. **קבלת קלט משתמש עבור Y:**

4. **לולאה חיצונית (עבור X):**
   - אתחול `X` לערך `Xmin`. לדוגמה: `X = 1.0`.
   - כל עוד `X` קטן או שווה ל-`Xmax` (תנאי לולאה):

   - סיום הלולאה החיצונית כאשר `X` גדול מ-`Xmax`.

5. **סיום:** סיום ביצוע התוכנית.
