# Модуль background.js

## Обзор

Этот модуль содержит код для расширения браузера Firefox, отвечающий за обработку кликов по иконке расширения. При клике на иконку расширения выполняется скрипт `contentScript.js` в текущей вкладке.

## Функции

### `browser.browserAction.onClicked.addListener`

**Описание**: Функция слушает события клика по иконке расширения.

**Параметры**:

- `(tab)`: Объект, содержащий информацию о текущей вкладке.

**Возвращает**:

-  (Не применимо):  Функция не возвращает значение.  Выполняет асинхронную операцию.


**Детали**:

Функция `browser.browserAction.onClicked.addListener` регистрирует обработчик для событий клика по иконке расширения браузера. Когда пользователь кликает по иконке расширения, функция `(tab) => { ... }` вызывается с объектом `tab`, содержащим информацию о текущей вкладке.


### `browser.scripting.executeScript`

**Описание**: Выполняет скрипт в контексте текущей вкладки.

**Параметры**:

- `target`: Объект, содержащий параметры целевой вкладки.
    - `tabId`: Идентификатор вкладки, в которой нужно выполнить скрипт.
- `files`: Массив имён файлов со скриптами для выполнения.
    - `"contentScript.js"`:  Имя файла со скриптом, который нужно выполнить.


**Возвращает**:
- (Не применимо): Функция `browser.scripting.executeScript` не возвращает прямого значения. Запускается асинхронная операция выполнения скрипта в указанной вкладке.


**Детали**:

Функция `browser.scripting.executeScript` выполняет скрипт `contentScript.js` в контексте текущей вкладки (указанной параметром `tabId`).  Это позволяет расширению взаимодействовать с содержимым страницы.
