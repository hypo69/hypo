# Модуль `keyboard.py`

## Обзор

Модуль `keyboard.py` содержит определения встроенных клавиатур (inline keyboards) для Telegram-бота. Эти клавиатуры используются для предоставления пользователям интерактивных опций, таких как поиск фильмов и выбор типа контента (фильм или сериал).

## Подробнее

Этот модуль определяет две встроенные клавиатуры: `find_movie` и `choice`. `find_movie` предоставляет кнопку для поиска новых фильмов, а `choice` предоставляет пользователю выбор между фильмами и сериалами. Эти клавиатуры используются в обработчиках бота для навигации и взаимодействия с пользователем.

## Клавиатуры

### `find_movie`

**Описание**: Встроенная клавиатура с кнопкой "Найти" для запроса новых фильмов.

**Как работает клавиатура**:
Клавиатура `find_movie` представляет собой объект `InlineKeyboardMarkup`, содержащий одну кнопку `InlineKeyboardButton` с текстом "Найти". При нажатии на эту кнопку, боту отправляется callback-запрос с данными `new_movies`, который может быть обработан для запуска процесса поиска фильмов.

**Ключевые элементы**:
- `InlineKeyboardMarkup`: Класс, используемый для создания встроенных клавиатур в Telegram.
- `InlineKeyboardButton`: Класс, представляющий кнопку на встроенной клавиатуре.
- `text`: Текст, отображаемый на кнопке ("Найти").
- `callback_data`: Данные, отправляемые боту при нажатии на кнопку ("new_movies").

### `choice`

**Описание**: Встроенная клавиатура с кнопками "Сериал" и "Фильм" для выбора типа контента.

**Как работает клавиатура**:
Клавиатура `choice` также является объектом `InlineKeyboardMarkup` и содержит две кнопки: "Сериал" и "Фильм". При нажатии на кнопку "Сериал", боту отправляется callback-запрос с данными `series`. При нажатии на кнопку "Фильм" отправляется callback-запрос с данными `film`. Это позволяет боту определить, какой тип контента интересует пользователя.

**Ключевые элементы**:
- `InlineKeyboardMarkup`: Класс для создания встроенных клавиатур.
- `InlineKeyboardButton`: Класс, представляющий кнопку на клавиатуре.
- `text`: Текст, отображаемый на кнопках ("Сериал", "Фильм").
- `callback_data`: Данные, отправляемые боту при нажатии на кнопки ("series", "film").
