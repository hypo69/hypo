Tiny Utils
============

## Обзор

**Tiny Utils** — это библиотека утилит, предоставляющая набор лёгких вспомогательных функций для различных общих задач. Библиотека включает в себя утилиты для преобразования форматов данных, обработки текста и файлов, строковых операций, форматирования дат и времени, обработки изображений и др. Она организована в несколько модулей для удобного доступа к конкретным функциональным возможностям.

## Установка

Для использования **Tiny Utils** клонируйте репозиторий и установите необходимые зависимости, как указано в файле `requirements.txt`.
`requirements.txt` включает в себя необходимые пакеты для работы библиотеки.

```bash
git clone https://github.com/hypo69/tiny-utils.git
cd tiny_utils
pip install -r requirements.txt
```

## Обзор модулей

Библиотека содержит несколько подмодулей, каждый из которых обрабатывает определённую задачу:

- **Преобразователи**: Модули для преобразования форматов данных, таких как текст в изображение, WebP в PNG, JSON, XML, кодирование Base64 и др.
- **Утилиты для строк**: Инструменты для расширенной работы со строками, включая форматирование, обрезку, поиск и замену.
- **Утилиты для работы с файлами**: Функции для чтения, записи, создания и удаления файлов, а также для работы с каталогами.
- **Утилиты для работы с датами и временем**: Инструменты для форматирования дат и времени, вычисления разницы между датами, а также для работы с временными зонами.
- **Утилиты для работы с FTP**: Функции для загрузки и скачивания файлов с FTP-серверов, а также для управления каталогами на FTP.
- **Утилиты для работы с изображениями**: Базовые функции обработки изображений, такие как изменение размера, обрезка и преобразование форматов.
- **Утилиты для работы с PDF**: Утилиты для чтения и преобразования файлов PDF, а также для извлечения текста и изображений из PDF.
- **Утилиты для работы с принтером**: Функции для отправки данных на принтер, включая форматирование и печать текста и изображений.

## Описание модулей

### Преобразователи

Модуль `convertors` содержит утилиты для преобразования данных между форматами. Эти модули могут обрабатывать различные типы данных, от CSV до JSON и текста до изображений.

#### Файлы:

- `text2png.py`: Преобразует текстовые данные в изображение PNG.
- `tts.py`: Преобразует текст в речь и сохраняет его в виде аудиофайла.
- `webp2png.py`: Преобразует изображения из формата WebP в формат PNG.
- `xls.py`: Обрабатывает преобразования и манипуляции с файлами XLS.
- `xml2dict.py`: Преобразует данные XML в словарь Python.
- `base64.py`: Кодирует или декодирует данные с использованием кодирования Base64.
- `csv.py`: Предоставляет инструменты для парсинга и манипулирования CSV.
- `dict.py`: Утилиты для обработки словарей Python.
- `html.py`: Преобразует контент HTML в различные форматы.
- `json.py`: Утилиты для парсинга и манипулирования JSON.
- `md2dict.py`: Преобразует контент Markdown в словарь.
- `ns.py`: Специализированные утилиты для преобразования имён пространств.

### Утилиты для строк

Модуль `strings` содержит инструменты для расширенной работы со строками, включая форматирование, обрезку, поиск и замену.
Здесь представлены функции для манипулирования текстовыми данными.

### Утилиты для работы с файлами
Модуль `files` предоставляет функции для чтения, записи, создания и удаления файлов, а также для работы с каталогами.
Этот модуль упрощает взаимодействие с файловой системой.

### Утилиты для работы с датами и временем
Модуль `datetime` содержит инструменты для форматирования дат и времени, вычисления разницы между датами, а также для работы с временными зонами.
Этот модуль предназначен для работы с датами и временем в различных форматах.

### Утилиты для работы с FTP
Модуль `ftp` предоставляет функции для загрузки и скачивания файлов с FTP-серверов, а также для управления каталогами на FTP.
Этот модуль обеспечивает интеграцию с FTP-серверами.

### Утилиты для работы с изображениями
Модуль `images` включает в себя базовые функции обработки изображений, такие как изменение размера, обрезка и преобразование форматов.
Этот модуль позволяет выполнять основные операции с изображениями.

### Утилиты для работы с PDF
Модуль `pdf` содержит утилиты для чтения и преобразования файлов PDF, а также для извлечения текста и изображений из PDF.
Этот модуль предназначен для работы с PDF документами.

### Утилиты для работы с принтером
Модуль `printer` предоставляет функции для отправки данных на принтер, включая форматирование и печать текста и изображений.
Этот модуль позволяет взаимодействовать с принтерами.

## Примеры использования

### Преобразование текста в изображение PNG
Пример использования `text2png.py` для преобразования текста в PNG изображение.

```python
from src.utils.convertors import text2png

text = "Пример текста"
output_path = "output.png"
text2png.convert(text, output_path)
```
### Преобразование XML в словарь
Пример использования `xml2dict.py` для преобразования XML в словарь Python.

```python
from src.utils.convertors import xml2dict

xml_string = "<root><element>Значение</element></root>"
result = xml2dict.parse(xml_string)
print(result)
```

### Парсинг и манипуляции с JSON
Пример использования `json.py` для парсинга и манипуляций с JSON данными.
```python
from src.utils.convertors import json

json_string = '{"key": "value"}'
result = json.loads(json_string)
print(result)

```
