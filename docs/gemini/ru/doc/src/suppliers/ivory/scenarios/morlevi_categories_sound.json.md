# Документация для `morlevi_categories_sound.json`

## Обзор

Этот файл `morlevi_categories_sound.json` содержит JSON-структуру, определяющую сценарии для сбора данных о категориях товаров, связанных со звуком, с веб-сайта Morlevi. Каждый сценарий включает информацию о бренде, URL-адресе категории, активности, состоянии товара и соответствующих категориях PrestaShop.

## Структура JSON

JSON-файл содержит один корневой объект с ключом `"scenarios"`, значением которого является объект, содержащий набор сценариев. Каждый сценарий представляет собой объект, где ключ - это название сценария, а значение - объект с настройками для этого сценария.

## Описание полей

### `scenarios`
Объект, содержащий все сценарии для сбора данных. Ключом каждого сценария является его название (например, "Logitech speakers"), а значением - объект с настройками этого сценария.

#### `brand`
- **Описание**: Бренд товара или производителя.
- **Тип**: `str`
- **Пример**: `"LOGITECH"`, `"GENIUS"`, `"CREATIVE"`, `"MICROSOFT"`, `"Corsair"`, `"COOLER MASTER"`, `"GENERIC"`

#### `url`
- **Описание**: URL-адрес страницы категории на сайте Morlevi.
- **Тип**: `str`
- **Пример**: `"https://www.morlevi.co.il/Cat/161?p_315=29&sort=datafloat2%2Cprice&keyword="`

#### `checkbox`
- **Описание**: Флаг, указывающий, должен ли быть включен чекбокс для данного сценария. 
- **Тип**: `bool`
- **Значение**: `false` для всех сценариев.

#### `active`
- **Описание**: Флаг, указывающий, активен ли данный сценарий.
- **Тип**: `bool`
- **Значение**: `true` для всех сценариев.

#### `condition`
- **Описание**: Состояние товара.
- **Тип**: `str`
- **Значение**: `"new"` для всех сценариев.

#### `presta_categories`
- **Описание**: Категории PrestaShop, к которым относится товар.
- **Тип**: `str`
- **Пример**: `"520,521"`, `"520,522"`, `"520,523"`
