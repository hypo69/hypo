# Задача No 238
## Блуждающий генератор
## Условия
Рассмотрим последовательность чисел, генерируемую с помощью следующего генератора:

`s(0) = 290797`

`s(n+1) = s(n)^2 mod 50515093`

Пусть t(n) будет строкой, образованной конкатенацией строк, представляющих двоичное разложение s(i) (где i = 1, 2, ..., n).
Например, t(10) = "10100110011110110010111111001101110101000101100001".

Найдите такое n, для которого количество единиц в t(n) равно 3*10^6.
