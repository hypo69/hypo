# Модуль `google_sample`

## Обзор

Модуль `google_sample.py` демонстрирует базовое использование Google Sheets API. Он считывает данные из указанной таблицы и выводит значения из определенных столбцов.

## Оглавление

1. [Обзор](#обзор)
2. [Константы](#константы)
3. [Переменные](#переменные)
4. [Функции](#функции)
    - [`main`](#main)

## Константы

### `MODE`
- **Описание**: Режим работы приложения. По умолчанию установлен в `dev`.
- **Тип**: `str`

### `SCOPES`
- **Описание**: Список областей доступа к Google Sheets API. В данном случае, только для чтения.
- **Тип**: `list`

### `SAMPLE_SPREADSHEET_ID`
- **Описание**: Идентификатор примера Google-таблицы.
- **Тип**: `str`

### `SAMPLE_RANGE_NAME`
- **Описание**: Диапазон ячеек в примере Google-таблицы, которые будут прочитаны.
- **Тип**: `str`

## Переменные

### `ROOT_DIRECTORY`
- **Описание**: Абсолютный путь к текущей рабочей директории.
- **Тип**: `pathlib.Path`

### `path`
- **Описание**: Путь к файлу с учетными данными клиента Google API.
- **Тип**: `pathlib.Path`

## Функции

### `main`

**Описание**:
Основная функция, демонстрирующая использование Sheets API. Выполняет авторизацию, считывает данные из таблицы и выводит их.

**Параметры**:
- Нет параметров.

**Возвращает**:
- `None`

**Вызывает исключения**:
- `HttpError`: В случае ошибки при обращении к Google Sheets API.
