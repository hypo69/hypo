# Модуль `hypotez/src/webdriver/chrome/header.py`

## Обзор

Данный модуль содержит функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и получения документации из файла `README.MD`. Он также определяет переменные, содержащие информацию о проекте, такие как название, версия, описание, автор и пр.

## Функции

### `set_project_root`

**Описание**: Функция находит корневую директорию проекта, начиная от текущего файла и проходя вверх по директориям до тех пор, пока не найдёт папки содержащей один из файлов или директорий из заданного списка.

**Параметры**:

- `marker_files` (tuple): Кортеж, содержащий имена файлов или директорий, используемых для поиска корневой директории проекта. По умолчанию это `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Объект `Path` представляющий корневую директорию проекта. Если корневая директория не найдена, возвращает директорию, в которой расположен текущий файл.

**Вызывает исключения**:

- Нет исключений.


## Переменные

### `__root__`

**Описание**: Переменная, содержащая путь к корневой директории проекта. Инициализируется функцией `set_project_root()`.

**Тип**: `Path`

### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженные из файла `settings.json` в директории проекта.

**Тип**: `dict` | `None`


### `doc_str`

**Описание**: Строка, содержащая документацию, загруженную из файла `README.MD` в директории проекта.

**Тип**: `str` | `None`


### `__project_name__`

**Описание**: Имя проекта. Значение берется из `settings`, по умолчанию "hypotez".

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта. Значение берется из `settings`, по умолчанию пустая строка.

**Тип**: `str`


### `__doc__`

**Описание**: Документация проекта. Значение берется из `doc_str`, по умолчанию пустая строка.

**Тип**: `str`


### `__details__`

**Описание**: Подробная информация о проекте. По умолчанию пустая строка.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта. Значение берется из `settings`, по умолчанию пустая строка.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права. Значение берется из `settings`, по умолчанию пустая строка.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка на способ поддержки разработчика. Значение берется из `settings`, по умолчанию ссылка.

**Тип**: `str`


## Обработка исключений

Обработка исключений `FileNotFoundError` и `json.JSONDecodeError`  при чтении файлов `settings.json` и `README.MD` предотвращает ошибки при отсутствии или неверном формате этих файлов.  Пустая обработка исключений (`...`) означает, что в случае ошибки значение соответствующей переменной остается `None`.
