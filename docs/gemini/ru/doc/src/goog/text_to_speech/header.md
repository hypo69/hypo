# Модуль hypotez/src/goog/text_to_speech/header.py

## Обзор

Этот модуль содержит функции для работы с настройками проекта, такие как получение корневого каталога проекта и чтение настроек из файла `settings.json`. Он также обеспечивает доступ к документации, хранящейся в файле `README.MD`.


## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная с текущей директории и двигаясь вверх по дереву каталогов, пока не найдет директорию, содержащую указанные файлы-маркеры.

**Параметры**:

- `marker_files` (tuple): Кортеж с именами файлов или каталогов, по которым будет определяться корневой каталог проекта. По умолчанию (`pyproject.toml`, `requirements.txt`, `.git`).

**Возвращает**:

- `Path`: Путь к корневому каталогу проекта, если он найден. В противном случае возвращает путь к каталогу, в котором находится скрипт.

**Вызывает исключения**:

- Нет.


## Переменные

### `__root__`

**Описание**: Переменная, содержащая путь к корневому каталогу проекта.

**Тип**: `Path`.

### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json`.

**Тип**: `dict`.

**Значение по умолчанию**: `None`.

### `doc_str`

**Описание**: Строка с содержимым файла `README.MD`.

**Тип**: `str`.

**Значение по умолчанию**: `None`.


### `__project_name__`

**Описание**: Название проекта, полученное из настроек проекта.

**Тип**: `str`.

**Значение по умолчанию**: `hypotez`.

### `__version__`

**Описание**: Версия проекта, полученная из настроек проекта.

**Тип**: `str`.

**Значение по умолчанию**: ''.

### `__doc__`

**Описание**: Документация проекта, полученная из файла `README.MD`.

**Тип**: `str`.

**Значение по умолчанию**: ''.

### `__details__`

**Описание**: Подробная информация о проекте.

**Тип**: `str`.

**Значение по умолчанию**: ''.

### `__author__`

**Описание**: Автор проекта, полученный из настроек проекта.

**Тип**: `str`.

**Значение по умолчанию**: ''.

### `__copyright__`

**Описание**: Авторские права на проект, полученные из настроек проекта.

**Тип**: `str`.

**Значение по умолчанию**: ''.

### `__cofee__`

**Описание**: Ссылка на способ поддержать разработчика (например, купить ему кофе).

**Тип**: `str`.

**Значение по умолчанию**:  "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69".

## Обработка исключений

В модуле используются блоки `try...except` для обработки возможных ошибок при работе с файлами `settings.json` и `README.MD`. В случае `FileNotFoundError` или `json.JSONDecodeError` соответствующие переменные (например, `settings`, `doc_str`) не изменяются.
