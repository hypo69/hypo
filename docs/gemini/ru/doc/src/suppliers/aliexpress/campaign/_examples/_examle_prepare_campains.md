## Обзор

Этот модуль предоставляет примеры использования функций из модуля `prepare_campaigns` для подготовки кампаний на AliExpress.  Он демонстрирует различные способы обработки кампаний: по категории, по имени кампании и все кампании.  Примеры включают указание языка, валюты и опции `force` для принудительного обновления.


## Функции

### `process_campaign_category`

**Описание**: Функция для обработки кампаний по определенной категории.

**Параметры**:
- `campaign_name` (str): Название кампании.
- `category` (str): Название категории.
- `language` (str): Язык кампании.
- `currency` (str): Валюта кампании.
- `force` (bool, optional): Флаг для принудительного обновления. По умолчанию `False`.

**Возвращает**:
- `None`: Функция не возвращает значение.

**Вызывает исключения**:
- `Exception`:  Возникает при любых ошибках во время выполнения.



### `process_campaign`

**Описание**: Функция для обработки определенной кампании.

**Параметры**:
- `campaign_name` (str): Название кампании для обработки.
- `categories` (list[str], optional): Список категорий, которые необходимо обработать.
- `language` (str, optional): Язык кампании.
- `currency` (str, optional): Валюта кампании.
- `force` (bool, optional): Флаг для принудительного обновления. По умолчанию `False`.


**Возвращает**:
- `None`: Функция не возвращает значение.

**Вызывает исключения**:
- `Exception`:  Возникает при любых ошибках во время выполнения.


### `process_all_campaigns`

**Описание**: Функция для обработки всех кампаний.

**Параметры**:
- `language` (str, optional): Язык кампаний.
- `currency` (str, optional): Валюта кампаний.
- `force` (bool, optional): Флаг для принудительного обновления. По умолчанию `False`.

**Возвращает**:
- `None`: Функция не возвращает значение.

**Вызывает исключения**:
- `Exception`: Возникает при любых ошибках во время выполнения.


### `get_directory_names`

**Описание**: Функция для получения списка имён файлов или папок в заданном каталоге.

**Параметры**:
- `campaigns_directory` (Path): Путь к каталогу.

**Возвращает**:
- list[str]: Список имён файлов/папок в каталоге, или пустой список, если каталог пуст или не существует.

**Вызывает исключения**:
- `Exception`: Возникает при проблемах с чтением каталога или валидностью input данных.
