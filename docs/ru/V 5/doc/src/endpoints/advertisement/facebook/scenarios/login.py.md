# Модуль `login`

## Обзор

Модуль `login` предназначен для выполнения сценария авторизации в Facebook. Он использует веб-драйвер для автоматического заполнения учетных данных и нажатия кнопки входа.

## Подробней

Этот модуль является частью системы автоматизации взаимодействия с Facebook в проекте `hypotez`. Он предоставляет функцию `login`, которая принимает объект драйвера веб-браузера, заполняет поля логина и пароля на странице входа в Facebook, и выполняет вход в систему. Модуль использует JSON-файл с локаторами веб-элементов для поиска полей ввода и кнопок на странице.

## Функции

### `login`

```python
def login(d: Driver) -> bool:
    """ Выполняет вход на Facebook.

    Функция использует переданный `Driver` для выполнения авторизации на Facebook, заполняя
    логин и пароль, а затем нажимает кнопку входа.

    Args:
        d (Driver): Экземпляр драйвера для взаимодействия с веб-элементами.

    Returns:
        bool: `True`, если авторизация прошла успешно, иначе `False`.

    Raises:
        Exception: Если возникает ошибка при вводе логина, пароля или нажатии кнопки.
    """
```

**Как работает функция**:
1. Функция принимает объект `Driver`, который используется для взаимодействия с веб-браузером.
2. Загружает учетные данные Facebook из `gs.facebook_credentials`.
3. Пытается ввести логин, пароль и нажать кнопку входа, используя локаторы, загруженные из JSON-файла.
4. В случае возникновения исключений при вводе данных или нажатии кнопки, логирует ошибку и возвращает `False`.
5. В случае успешного выполнения всех действий возвращает `True`.

**Параметры**:
- `d` (Driver): Экземпляр драйвера, используемый для управления веб-браузером.

**Возвращает**:
- `bool`: Возвращает `True`, если авторизация прошла успешно, и `False` в противном случае.

**Вызывает исключения**:
- `Exception`: Возникает, если происходит ошибка при вводе логина, пароля или нажатии кнопки.

**Примеры**:

```python
from src.webdriver.driver import Driver
from selenium import webdriver

# Пример использования функции login
driver = Driver(webdriver.Chrome())  # Инициализация веб-драйвера Chrome
driver.get("https://www.facebook.com/")  # Открытие страницы Facebook

success = login(driver)  # Вызов функции login для выполнения авторизации

if success:
    print("Авторизация прошла успешно!")
else:
    print("Ошибка при авторизации!")

driver.quit()  # Закрытие веб-драйвера
