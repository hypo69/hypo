## Обзор

Данный файл содержит JSON-конфигурацию для определения параметров сбора данных о дронах Eachine с сайта Banggood. Он определяет условия поиска, марки и сопоставления с категориями в Prestashop.

## Структура файла

Файл имеет следующую структуру:

### Раздел "scenarios"
   - Этот раздел содержит все сценарии сбора данных.

### Раздел "Eachine RC Drones"

   - **Описание**: Данный раздел описывает конкретный сценарий сбора данных для дронов Eachine.

   #### Параметр "brand"

   - **Описание**: Указывает бренд дронов для сбора данных.

   #### Параметр "url"

    - **Описание**: Указывает URL для поиска дронов на сайте Banggood.

   #### Параметр "active"

   - **Описание**: Указывает, активен ли данный сценарий. (true/false).

   #### Параметр "condition"

    - **Описание**: Определяет условие товара (новое).

    #### Раздел "presta_categories"

    - **Описание**: Содержит информацию о сопоставлении категорий с Prestashop.

    #### Раздел "template"
    - **Описание**: Содержит сопоставление имени бренда с категорией.
    -  **Пример**: {"eachine": "RC Drones"}
