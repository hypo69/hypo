# Модуль для работы с помощником по программированию

## Обзор

Этот модуль содержит класс :class:`CodeAssistant`, используемый для взаимодействия с различными моделями ИИ, такими как Google Gemini и OpenAI, для задач обработки кода.  Он предназначен для анализа и генерации документации кода.


## Примеры использования

```python
from ai.prompts.developer import code_assistant  # Импортируем класс CodeAssistant

assistant = code_assistant.CodeAssistant(role='code_checker', lang='ru', model=['gemini'])
assistant.process_files(files=['file1.py', 'file2.py'], options={})
```


## Классы

### `CodeAssistant`

**Описание**: Класс для работы с помощником по программированию.  Он взаимодействует с моделями ИИ для анализа и генерации документации кода.

**Атрибуты**:

- `role` (str): Роль помощника (например, 'code_checker').
- `lang` (str): Язык, используемый помощником (например, 'ru').
- `model` (list): Список используемых моделей ИИ (например, ['gemini']).


**Методы**:

#### `process_files`

**Описание**: Метод для обработки файлов кода.

**Параметры**:
- `files` (list[str]): Список путей к файлам, которые необходимо обработать.
- `options` (dict, необязательно): Дополнительные параметры для настройки обработки.

**Возвращает**:
- `list[dict]`: Список словарей с результатами анализа.  Каждый словарь содержит данные об обработанном файле.


**Обрабатывает исключения**:
- `FileNotFoundError`: Если указанный файл не найден.
- `Exception`: В случае других ошибок при обработке файла.

## Исключения

### `FileNotFoundError`

**Описание**:  Это исключение возникает, когда файл, указанный в параметре `files`, не найден.

**Параметры**:
- `file` (str): Путь к файлу, который не найден.


