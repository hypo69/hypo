# Модуль myai

## Обзор

Данный модуль содержит константу `MODE`, определяющую режим работы приложения.

## Константы

### `MODE`

**Описание**:  Константа, определяющая режим работы приложения.

**Значение**: `'dev'`

**Примечание**:  В данном случае, режим `dev` предполагает, что приложение находится в режиме разработки.
