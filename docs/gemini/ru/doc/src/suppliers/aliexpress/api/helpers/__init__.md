# Модуль `hypotez/src/suppliers/aliexpress/api/helpers/__init__.py`

## Обзор

Этот модуль предоставляет набор функций и классов для работы с API AliExpress. Он содержит инструменты для обработки запросов, извлечения и анализа данных о продуктах, а также категориях.

## Функции

### `api_request`

**Описание**: Функция для выполнения запросов к API AliExpress.

**Параметры**:
- `url` (str): URL запроса.
- `params` (dict, optional): Параметры запроса. По умолчанию `None`.
- `headers` (dict, optional): Заголовки запроса. По умолчанию `None`.

**Возвращает**:
- `dict | None`: Результат запроса в формате словаря или `None`, если произошла ошибка.

**Вызывает исключения**:
- `requests.exceptions.RequestException`: Ошибки, связанные с выполнением запроса (например, проблемы с подключением к сети).
- `ValueError`: Ошибка, если входные данные не соответствуют ожидаемому формату.

### `get_list_as_string`

**Описание**: Функция для преобразования списка в строку, разделенную запятыми.

**Параметры**:
- `list_data` (list): Список значений.

**Возвращает**:
- `str`: Строка, содержащая значения списка, разделенные запятыми.
- `None`: Возвращает None, если на вход не передается список.

**Вызывает исключения**:
- `TypeError`: Ошибка, если введенный аргумент не является списком.


### `get_product_ids`

**Описание**: Функция для извлечения идентификаторов продуктов из списка.

**Параметры**:
- `products_data` (list): Список данных о продуктах.

**Возвращает**:
- `list[int]`: Список идентификаторов продуктов.
- `None`: Возвращает None, если на вход не передается список.


**Вызывает исключения**:
- `TypeError`: Ошибка, если введенный аргумент не является списком.


### `parse_products`

**Описание**: Функция для парсинга данных о продуктах.


**Параметры**:
- `data` (dict): Данные о продуктах в формате словаря.


**Возвращает**:
- `list[dict]`: Список словарей с данными о продуктах после парсинга.
- `None`: Возвращает None, если произошла ошибка.


**Вызывает исключения**:
- `ValueError`: Ошибка, если входные данные не соответствуют ожидаемому формату.


### `filter_parent_categories`

**Описание**: Функция для фильтрации родительских категорий.


**Параметры**:
- `categories` (list): Список данных о категориях.

**Возвращает**:
- `list[dict]`: Список словарей с родительскими категориями.


**Вызывает исключения**:
- `TypeError`: Ошибка, если введенный аргумент не является списком.


### `filter_child_categories`

**Описание**: Функция для фильтрации дочерних категорий.


**Параметры**:
- `categories` (list): Список данных о категориях.

**Возвращает**:
- `list[dict]`: Список словарей с дочерними категориями.


**Вызывает исключения**:
- `TypeError`: Ошибка, если введенный аргумент не является списком.
