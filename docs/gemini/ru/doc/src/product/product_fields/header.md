# Модуль hypotez/src/product/product_fields/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json` и документации из файла `README.MD`.  Он также предоставляет доступ к переменным, описывающим проект (имя, версия, описание, автор, права и т.д.).

## Функции

### `set_project_root`

**Описание**: Определяет корневую директорию проекта, начиная с директории текущего файла и идя вверх по древу директорий, пока не найдёт директорию, содержащую один из указанных файлов-маркеров (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, которые используются для поиска корневой директории. По умолчанию содержит `('pyproject.toml', 'requirements.txt', '.git')`.


**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращает директорию, где расположен текущий файл.


**Вызывает исключения**:

- Нет.

## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта. Инициализируется функцией `set_project_root`.


**Тип**: `Path`


### `settings`

**Описание**: Словарь, содержащий настройки проекта. Загружается из файла `settings.json` в корневой директории проекта.


**Тип**: `dict` или `None`


### `doc_str`

**Описание**: Строка, содержащая содержимое файла `README.MD` в корневой директории проекта.


**Тип**: `str` или `None`


### `__project_name__`

**Описание**: Имя проекта. Извлекается из `settings` или устанавливается по умолчанию как `'hypotez'`, если `settings` не содержит значение `project_name`.


**Тип**: `str`


### `__version__`

**Описание**: Версия проекта. Извлекается из `settings`, если доступен, иначе устанавливается как пустая строка.


**Тип**: `str`


### `__doc__`

**Описание**: Документация проекта.  Извлекается из `doc_str` или устанавливается как пустая строка, если `doc_str` не доступен.


**Тип**: `str`


### `__details__`

**Описание**: Подробности проекта (пустая строка по умолчанию).


**Тип**: `str`


### `__author__`

**Описание**: Автор проекта. Извлекается из `settings`, иначе устанавливается как пустая строка.


**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права. Извлекается из `settings`, иначе устанавливается как пустая строка.


**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка для поддержки разработчика (например, на Boosty). Извлекается из `settings`, иначе устанавливается по умолчанию.


**Тип**: `str`
