# src.scenario.header

## Обзор

Модуль `header.py` отвечает за инициализацию проекта, определение корневой директории, загрузку настроек из `settings.json` и документации из `README.md`, а также определяет основные метаданные проекта.

## Функции

### `set_project_root`

**Описание**: 
Находит корневую директорию проекта, начиная с директории текущего файла, просматривая вверх и останавливаясь на первой директории, содержащей один из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или директорий для идентификации корневой директории проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена, иначе директория, где расположен скрипт.

## Переменные

### `__root__`

**Описание**:
`Path` - Путь к корневой директории проекта.

### `settings`

**Описание**:
`dict` - Словарь с настройками проекта, загруженными из файла `settings.json`. Может быть `None`, если файл не найден или возникла ошибка при разборе JSON.

### `doc_str`

**Описание**:
`str` - Строка, содержащая содержимое файла `README.MD`. Может быть `None`, если файл не найден или возникла ошибка при чтении.

### `__project_name__`

**Описание**:
`str` - Название проекта, по умолчанию `hypotez`. Загружается из `settings.json` или используется значение по умолчанию, если `settings` равно `None`.

### `__version__`

**Описание**:
`str` - Версия проекта. Загружается из `settings.json` или остается пустой строкой, если `settings` равно `None`.

### `__doc__`

**Описание**:
`str` - Документация проекта, загруженная из `README.MD`. Может быть пустой строкой, если `doc_str` равно `None`.

### `__details__`

**Описание**:
`str` - Детальная информация о проекте. В текущей версии модуля это всегда пустая строка.

### `__author__`

**Описание**:
`str` - Автор проекта. Загружается из `settings.json` или остается пустой строкой, если `settings` равно `None`.

### `__copyright__`

**Описание**:
`str` - Информация об авторских правах. Загружается из `settings.json` или остается пустой строкой, если `settings` равно `None`.

### `__cofee__`

**Описание**:
`str` - Сообщение с предложением угостить разработчика кофе. Загружается из `settings.json` или используется значение по умолчанию, если `settings` равно `None`.
