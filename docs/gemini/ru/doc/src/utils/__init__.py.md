# Модуль `src.utils`

## Обзор

Модуль `src.utils` представляет собой коллекцию небольших утилит, предназначенных для упрощения часто выполняемых задач программирования. Он включает в себя инструменты для конвертации данных, работы с файлами и форматированного вывода. Этот модуль предоставляет набор переиспользуемых функций, которые могут ускорить процесс разработки.

## Импорты

Модуль `src.utils` импортирует множество подмодулей и функций, которые разбиты по категориям.

### Конверторы (`convertors`)

Этот раздел включает функции для конвертации различных форматов данных:

- `TextToImageGenerator`: Генератор изображений из текста.
- `base64_to_tmpfile`: Конвертация base64 в временный файл.
- `base64encode`: Кодирование в base64.
- `csv2dict`: Конвертация CSV в словарь.
- `csv2ns`: Конвертация CSV в `namedtuple`.
- `decode_unicode_escape`: Декодирование unicode-escape строк.
- `dict2csv`: Конвертация словаря в CSV.
- `dict2html`: Конвертация словаря в HTML.
- `dict2ns`: Конвертация словаря в `namedtuple`.
- `dict2xls`: Конвертация словаря в XLS.
- `dict2xml`: Конвертация словаря в XML.
- `dot2png`: Конвертация DOT в PNG.
- `escape2html`: Экранирование HTML.
- `html2dict`: Конвертация HTML в словарь.
- `html2escape`: Конвертация HTML в экранированный HTML.
- `html2ns`: Конвертация HTML в `namedtuple`.
- `html2text`: Конвертация HTML в текст.
- `html2text_file`: Конвертация HTML-файла в текстовый файл.
- `json2csv`: Конвертация JSON в CSV.
- `json2ns`: Конвертация JSON в `namedtuple`.
- `json2xls`: Конвертация JSON в XLS.
- `json2xml`: Конвертация JSON в XML.
- `md2dict`: Конвертация Markdown в словарь.
- `ns2csv`: Конвертация `namedtuple` в CSV.
- `ns2dict`: Конвертация `namedtuple` в словарь.
- `ns2xls`: Конвертация `namedtuple` в XLS.
- `ns2xml`: Конвертация `namedtuple` в XML.
- `replace_key_in_dict`: Замена ключа в словаре.
- `speech_recognizer`: Распознавание речи.
- `text2speech`: Конвертация текста в речь.
- `webp2png`: Конвертация WEBP в PNG.
- `xls2dict`: Конвертация XLS в словарь.

### CSV (`csv`)

Этот раздел содержит функции для работы с CSV файлами:

- `read_csv_as_dict`: Чтение CSV файла в словарь.
- `read_csv_as_ns`: Чтение CSV файла в `namedtuple`.
- `read_csv_file`: Чтение CSV файла.
- `save_csv_file`: Сохранение CSV файла.

### Дата и время (`date_time`)

Этот раздел содержит функции для работы с датой и временем:

- `TimeoutCheck`: Класс для проверки таймаутов.

### Файлы (`file`)

Этот раздел содержит функции для работы с файлами:

- `get_directory_names`: Получение имен директорий.
- `get_filenames`: Получение имен файлов.
- `read_text_file`: Чтение текстового файла.
- `recursively_get_file_path`: Рекурсивное получение пути к файлам.
- `recursively_read_text_files`: Рекурсивное чтение текстовых файлов.
- `recursively_yield_file_path`: Рекурсивный генератор путей к файлам.
- `remove_bom`: Удаление BOM из файла.
- `save_text_file`: Сохранение текста в файл.

### Изображения (`image`)

Этот раздел содержит функции для работы с изображениями:

- `save_image`: Сохранение изображения.
- `save_image_from_url`: Сохранение изображения по URL.
- `random_image`: Получение случайного изображения.

### JSON (`jjson`)

Этот раздел содержит функции для работы с JSON:

- `j_dumps`: Сериализация в JSON.
- `j_loads`: Десериализация из JSON.
- `j_loads_ns`: Десериализация из JSON в `namedtuple`.

### PDF (`pdf`)

Этот раздел содержит функции для работы с PDF:

- `PDFUtils`: Утилиты для работы с PDF.

### Принтер (`printer`)

Этот раздел содержит функции для форматированного вывода:

- `pprint`: Форматированный вывод.

### Строки (`string`)

Этот раздел содержит функции для работы со строками:

- `ProductFieldsValidator`: Валидатор полей продукта.
- `StringFormatter`: Форматировщик строк.
- `normalize_string`: Нормализация строки.
- `normalize_int`: Нормализация целого числа.
- `normalize_float`: Нормализация числа с плавающей точкой.
- `normalize_boolean`: Нормализация булевого значения.

### URL (`url`)

Этот раздел содержит функции для работы с URL:

- `extract_url_params`: Извлечение параметров из URL.
- `is_url`: Проверка, является ли строка URL.

### Видео (`video`)

Этот раздел содержит функции для работы с видео:

- `save_video_from_url`: Сохранение видео по URL.

### Пути (`path`)

Этот раздел содержит функции для работы с путями:

- `get_relative_path`: Получение относительного пути.
