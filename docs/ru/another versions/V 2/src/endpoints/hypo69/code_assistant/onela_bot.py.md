# Модуль `onela_bot`

## Обзор

Модуль `src.endpoints.hypo69.code_assistant.onela_bot` предназначен для взаимодействия с моделью ассистента программиста через чат Telegram. Он включает класс `OnelaBot`, который обрабатывает текстовые сообщения и документы, используя модели `GoogleGenerativeAI` для генерации ответов.

## Классы

### `OnelaBot`

**Описание**: Класс `OnelaBot` предназначен для обработки входящих сообщений и документов в Telegram. Он использует модель `GoogleGenerativeAI` для генерации ответов на текстовые запросы пользователей и обрабатывает загруженные документы.

**Методы**:
- [`__init__`](#__init__): Инициализация объекта `OnelaBot`.
- [`handle_message`](#handle_message): Обрабатывает текстовые сообщения, отправленные в чат.
- [`handle_document`](#handle_document): Обрабатывает загруженные документы.

#### `__init__`
**Описание**: Инициализирует объект `OnelaBot`, вызывая конструктор родительского класса `TelegramBot`.

**Параметры**:
- Нет.

**Возвращает**:
- `None`

#### `handle_message`
**Описание**: Обрабатывает текстовые сообщения, отправленные пользователями в чат. Запрашивает у модели `GoogleGenerativeAI` ответ на сообщение и отправляет его пользователю.

**Параметры**:
- `update` (Update): Данные обновления Telegram.
- `context` (CallbackContext): Контекст выполнения.

**Возвращает**:
- `None`

**Вызывает исключения**:
- `Exception`: Возникает при ошибке обработки текстового сообщения.

#### `handle_document`
**Описание**: Обрабатывает загруженные документы, скачивает их на локальный диск и отправляет пользователю.

**Параметры**:
- `update` (Update): Данные обновления Telegram.
- `context` (CallbackContext): Контекст выполнения.

**Возвращает**:
- `None`

**Вызывает исключения**:
- `Exception`: Возникает при ошибке обработки загруженного документа.
