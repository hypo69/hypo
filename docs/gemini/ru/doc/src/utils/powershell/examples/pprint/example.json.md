## Структура JSON

Файл представляет собой JSON-объект с одним ключом верхнего уровня `"keys"`. Значением этого ключа является другой JSON-объект, который содержит различные ключи, такие как `"key_hello"` и `"key_goodbye"`. Каждый из этих ключей имеет значением JSON-объект, содержащий массив сообщений `messages`, которые представлены на разных языках.

## Ключи

### `key_hello`

**Описание**: Содержит сообщения для приветствия.

#### Сообщения:

-   `en`: "Hello, World!"
-   `ru`: "Привет, Мир!"

### `key_goodbye`

**Описание**: Содержит сообщения для прощания.

#### Сообщения:

-   `en`: "Have a nice day!"
-   `ru`: "Всего хорошего!"

## Сообщения

**Описание**: Массив сообщений в формате JSON, каждое из которых представляет собой объект с языковыми кодами (например, `en` для английского, `ru` для русского) в качестве ключей и соответствующими сообщениями в качестве значений.
