# Battle

## Обзор

Этот модуль реализует игру "Морской бой" между игроком и компьютером. Игроки по очереди стреляют по полю противника, пытаясь потопить корабли. Игра заканчивается после 30 ходов.

**Возвращает**:
- `None`: Функция ничего не возвращает, выводит игровое поле в консоль.

## Дополнительная информация

### Игровой процесс

1.  Создаются два игровых поля: для игрока и для компьютера.
2.  На поле компьютера размещаются 5 кораблей случайным образом.
3.  Игроки по очереди делают ходы, пока не будет сделано 30 ходов.
4.  Игрок вводит координаты своего выстрела.
5.  Компьютер делает выстрел в случайную клетку.
6.  На экране отображается состояние поля игрока.
7.  После 30 ходов игра заканчивается.
