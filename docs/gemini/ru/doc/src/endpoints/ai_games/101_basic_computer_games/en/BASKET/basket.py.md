# Игра "Баскетбол"

## Обзор

Данный модуль реализует текстовую игру "Баскетбол", в которой игрок выбирает дистанцию броска, а компьютер определяет результат броска (попадание или промах) на основе вероятности, зависящей от дистанции.

## Функции

### `Основной игровой цикл`

**Описание**: Основной цикл игры, который запрашивает у пользователя дистанцию броска, проверяет ее корректность, рассчитывает вероятность попадания и выводит результат броска.

**Параметры**:
- нет

**Возвращает**:
- нет

**Вызывает исключения**:
- `ValueError`: Возникает, если пользователь вводит не целое число.
