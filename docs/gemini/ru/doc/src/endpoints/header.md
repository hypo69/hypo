# Модуль hypotez/src/endpoints/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и получения документации из файла `README.MD`.  Он также определяет и инициализирует переменные, содержащие информацию о проекте, такие как имя, версия, автор, копирайт и ссылку на пожертвования.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с текущей директории файла и идя вверх по иерархии директорий.

**Параметры**:

- `marker_files` (tuple): Кортеж файлов или директорий, которые используются для определения корня проекта. По умолчанию используются `pyproject.toml`, `requirements.txt` и `.git`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращает директорию, где находится данный скрипт.


**Вызывает исключения**:

- Нет

### Инициализация переменных проекта


**Описание**:  Инициализирует глобальные переменные, описывающие проект (`__root__`, `__project_name__`, `__version__`, `__doc__`, `__details__`, `__author__`, `__copyright__`, `__cofee__`), читая информацию из `settings.json` и `README.MD`.  Если файлы не найдены или содержат ошибки парсинга, значения переменных по умолчанию.

**Параметры**:

- Нет

**Возвращает**:

- Нет


**Вызывает исключения**:

- `FileNotFoundError`: Если файлы `settings.json` или `README.MD` не найдены.
- `json.JSONDecodeError`: Если содержимое файла `settings.json` не является корректным JSON.


## Переменные

### `MODE`

**Описание**:  Строковая переменная, хранящая режим работы (например, 'dev' или 'prod').

### `__root__`

**Описание**:  Переменная, хранящая путь к корневой директории проекта. Инициализируется функцией `set_project_root()`.


### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженные из файла `settings.json`.


### `doc_str`

**Описание**: Строковая переменная, содержащая содержимое файла `README.MD`.

### `__project_name__`

**Описание**: Имя проекта. Получается из `settings` или по умолчанию равно 'hypotez'.

### `__version__`

**Описание**: Версия проекта. Получается из `settings` или по умолчанию пустая строка.


### `__doc__`

**Описание**: Документация проекта. Получается из `doc_str` или по умолчанию пустая строка.

### `__details__`

**Описание**: Дополнительная информация о проекте. По умолчанию пустая строка.

### `__author__`

**Описание**: Автор проекта. Получается из `settings` или по умолчанию пустая строка.


### `__copyright__`

**Описание**: Авторские права. Получается из `settings` или по умолчанию пустая строка.


### `__cofee__`

**Описание**: Ссылка на пожертвования (на чашку кофе). Получается из `settings` или по умолчанию предустановленная ссылка.
