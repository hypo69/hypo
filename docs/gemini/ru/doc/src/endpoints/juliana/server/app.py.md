# Документация модуля Flask приложения

## Обзор

Данный модуль содержит минимальную конфигурацию Flask приложения. Он создает инстанс Flask приложения и указывает папку с шаблонами HTML.

## Подробней

Этот модуль является отправной точкой для создания веб-приложения с использованием Flask. Он инициализирует приложение и устанавливает путь к папке, где будут храниться HTML-шаблоны для рендеринга веб-страниц.

## Функции

### `Flask(__name__, template_folder='./../client/html')`

**Назначение**: Создает экземпляр Flask приложения.

**Параметры**:
- `__name__` (str): Имя текущего модуля. Flask использует его для определения местоположения ресурсов приложения.
- `template_folder` (str): Путь к папке, содержащей HTML-шаблоны. В данном случае, это папка `./../client/html`, расположенная на один уровень выше текущей директории и внутри папки `client`.

**Возвращает**:
- Объект `Flask`: Экземпляр Flask приложения.

**Как работает функция**:

1.  Инициализация: Функция `Flask()` вызывается с параметрами `__name__` и `template_folder`.
2.  Создание экземпляра: Создается экземпляр Flask приложения, который будет использоваться для обработки запросов и рендеринга веб-страниц.

```ascii
Инициализация Flask
│
├── Создание экземпляра Flask приложения
│
└── Установка папки с шаблонами
```

**Примеры**:

```python
from flask import Flask

app = Flask(__name__, template_folder='./../client/html')
```
```python
from flask import Flask

app = Flask(__name__, template_folder='templates')
```
```python
from flask import Flask

app = Flask(__name__)
