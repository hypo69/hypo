# Задача No 316
## Номер Фейнмана

## Условия
Рассмотрим функцию *f(n)*, где *n* - натуральное число, которая возвращает целое число, состоящее из цифр, полученных конкатенацией  десятичной записи *n*, повторенной бесконечное количество раз.

Например,
*   *f(1) = 111111111111111...*
*   *f(10) = 101010101010101...*
*   *f(101) = 101101101101101...*
 
Пусть *s<sub>n</sub>* - это строка, состоящая из первых *n* цифр строки *f(n)*.

Пусть *z(n)* - это наименьшее целое число *m*, такое, что *s<sub>m</sub>* содержит все последовательные подстроки, которые могут быть сформированы из *n* цифр.

Например, *z(3)* = 111, так как строка *s<sub>111</sub> = 333333333333333333333333333333333333333333333333333333333333333333333333333333333333333333333333333333333333333333333333333333333333333333333333333333*  содержит "333", которая является единственной последовательностью из трёх цифр в этой строке.

Также, *z(10) = 25* и *z(100) = 270*.

Найдите $\sum_{n=1}^{1000} z(n)$.
