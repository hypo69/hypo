# Модуль `src.ai`

## Обзор

Модуль `src.ai` предоставляет интерфейс для работы с различными моделями искусственного интеллекта. Он включает в себя классы для взаимодействия с моделями Google Gemini и OpenAI.

## Импортированные модули

В этом разделе перечислены модули, импортированные в данном файле.

```python
from .gemini import GoogleGenerativeAI
from .openai import OpenAIModel
```
## Модули

### `gemini`

Содержит класс `GoogleGenerativeAI` для работы с моделями Google Gemini.

### `openai`

Содержит класс `OpenAIModel` для работы с моделями OpenAI.
