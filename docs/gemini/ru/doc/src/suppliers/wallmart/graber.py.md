# Модуль `src.suppliers.wallmart.graber`

## Обзор

Модуль `src.suppliers.wallmart.graber` предназначен для сбора значений полей на страницах товаров веб-сайта `wallmart.com`. Он расширяет функциональность базового класса `Graber`, предоставляя возможность переопределения стандартных методов обработки полей для учета специфических особенностей структуры страниц `wallmart.com`.

## Подробней

Этот модуль является частью системы сбора данных о товарах с различных онлайн-площадок. Он использует веб-драйвер для навигации по сайту `wallmart.com` и извлечения необходимой информации. Модуль включает механизм для предварительной обработки запросов к веб-драйверу через декораторы, что позволяет выполнять такие действия, как закрытие всплывающих окон перед извлечением данных. Расположение файла в структуре проекта указывает на его роль как специализированного инструмента для работы с `wallmart.com` в рамках общей системы сбора данных.

## Классы

### `Graber`

**Описание**: Класс `Graber` предназначен для извлечения данных с сайта `wallmart.com`. Он наследуется от базового класса `Graber` (`Grbr`) и переопределяет некоторые его методы для адаптации к структуре страниц `wallmart.com`.

**Методы**:
- `__init__`: Инициализирует экземпляр класса `Graber`, устанавливает префикс поставщика и выполняет базовую инициализацию родительского класса.

**Параметры**:
- `driver` (`Driver`): Экземпляр веб-драйвера, используемый для взаимодействия с веб-страницами.
- `lang_index` (`int`): Индекс языка, используемый для локализации контента (если применимо).

**Примеры**

```python
from src.webdriver.driver import Driver
from src.suppliers.wallmart.graber import Graber

# Пример создания экземпляра класса Graber
driver = Driver()
graber = Graber(driver=driver, lang_index=0)
```
