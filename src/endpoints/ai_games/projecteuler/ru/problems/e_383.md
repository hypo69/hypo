# Задача No 383
## Соединения с помощью решета
## Условия
Пусть S(n) - количество чисел m, таких что 1 < m < n, и наибольший общий делитель m и n равен 1.

Далее пусть F(n) будет количеством целых чисел k, таких что 1 < k < n, и S(k) = S(n).

Пусть G(n) будет суммой всех чисел k, таких что 1 < k < n, и S(k) = S(n).

Например:
S(2) = 1
S(3) = 2
S(4) = 2
S(5) = 4
S(6) = 2

Итак:
F(2) = 1
F(3) = 2
F(4) = 2
F(5) = 0
F(6) = 3

G(2) = 2
G(3) = 7
G(4) = 7
G(5) = 0
G(6) = 17

Вычислите sum(G(n)) для 1 < n <= 10^8.
