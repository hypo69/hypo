# hypotez/src/utils/printer.py

## Обзор

Модуль `printer.py` предоставляет утилиты для форматированного вывода данных в консоль с возможностью настройки стилей текста, таких как цвет, фон и начертание. Он предназначен для улучшения читаемости выводимой информации при разработке и отладке.

## Константы

### `MODE`

- **Описание**: Режим работы модуля. По умолчанию установлен в `dev`.
- **Тип**: `str`

### `RESET`

- **Описание**: ANSI escape-код для сброса стилей текста.
- **Тип**: `str`

### `TEXT_COLORS`

- **Описание**: Словарь, содержащий ANSI escape-коды для различных цветов текста.
- **Тип**: `dict`

### `BG_COLORS`

- **Описание**: Словарь, содержащий ANSI escape-коды для различных цветов фона текста.
- **Тип**: `dict`

### `FONT_STYLES`

- **Описание**: Словарь, содержащий ANSI escape-коды для различных стилей шрифта (например, `bold`, `underline`).
- **Тип**: `dict`

## Функции

### `_color_text`

**Описание**: Применяет заданные стили (цвет текста, фона и начертание) к входной строке.

**Параметры**:

-   `text` (str): Строка, к которой будут применены стили.
-   `text_color` (str, optional): Цвет текста. По умолчанию пустая строка (без цвета).
-   `bg_color` (str, optional): Цвет фона. По умолчанию пустая строка (без фона).
-   `font_style` (str, optional): Стиль шрифта. По умолчанию пустая строка (без стиля).

**Возвращает**:

-   `str`: Строка с примененными стилями.

**Пример использования:**

```python
>>> _color_text("Hello, World!", text_color="green", font_style="bold")
'\033[1m\033[32mHello, World!\033[0m'
```

### `pprint`

**Описание**: Выводит данные в консоль в форматированном виде с возможностью настройки стилей текста.

**Параметры**:

-   `print_data` (Any, optional): Данные для вывода. Может быть любого типа, включая `None`, `dict`, `list`, `str` или `Path`. По умолчанию `None`.
-   `text_color` (str, optional): Цвет текста. По умолчанию `white`. Доступные цвета указаны в `TEXT_COLORS`.
-   `bg_color` (str, optional): Цвет фона. По умолчанию пустая строка (без фона). Доступные цвета указаны в `BG_COLORS`.
-   `font_style` (str, optional): Стиль шрифта. По умолчанию пустая строка (без стиля). Доступные стили указаны в `FONT_STYLES`.

**Возвращает**:

-   `None`

**Вызывает исключения**:

-   `Exception`: Если тип данных не поддерживается или возникает ошибка во время вывода.

**Примеры использования:**

```python
>>> pprint({"name": "Alice", "age": 30}, text_color="green")
\033[32m{
    "name": "Alice",
    "age": 30
}\033[0m

>>> pprint(["apple", "banana", "cherry"], text_color="blue", font_style="bold")
\033[34m\033[1mapple\033[0m
\033[34m\033[1mbanana\033[0m
\033[34m\033[1mcherry\033[0m

>>> pprint("text example", text_color="yellow", bg_color="bg_red", font_style="underline")
\033[4m\033[33m\033[41mtext example\033[0m
```
