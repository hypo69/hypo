# examples.py

## Обзор

Модуль содержит примеры использования библиотеки `tinytroupe` для создания агентов. Примеры можно использовать непосредственно или с небольшими изменениями для создания собственных агентов.

    - [`create_marcos_the_physician`](#create_marcos_the_physician)
    - [`create_lila_the_linguist`](#create_lila_the_linguist)

## Функции

### `create_oscar_the_architect`

**Описание**: Создает агента Оскара, архитектора.

**Возвращает**:
- `TinyPerson`: Объект агента Оскара.

### `create_lisa_the_data_scientist`

**Описание**: Создает агента Лизу, специалиста по данным.

**Возвращает**:
- `TinyPerson`: Объект агента Лизы.

### `create_marcos_the_physician`

**Описание**: Создает агента Маркоса, врача.

**Возвращает**:
- `TinyPerson`: Объект агента Маркоса.

### `create_lila_the_linguist`

**Описание**: Создает агента Лилу, лингвиста.

**Возвращает**:
- `TinyPerson`: Объект агента Лилы.
