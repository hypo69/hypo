# Модуль hypotez/src/webdriver/bs/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`. Он также загружает описание проекта из `README.MD`.


## Функции

### `set_project_root`

**Описание**: Определяет корневую директорию проекта, начиная с директории текущего файла. Ищет вверх по директориям, пока не найдет директорию, содержащую один из указанных файлов-маркеров.

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, которые используются для определения корневой директории проекта. По умолчанию это ('pyproject.toml', 'requirements.txt', '.git').

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена, иначе — директория, где расположен текущий скрипт.


### Загрузка настроек и документации

**Описание**: Загружает настройки из файла `settings.json` в глобальную переменную `settings`. Загружает описание проекта из `README.MD` в глобальную переменную `doc_str`.

**Вызывает исключения**:

- `FileNotFoundError`: Если файл `settings.json` или `README.MD` не найден.
- `json.JSONDecodeError`: Если содержимое `settings.json` не является валидным JSON.

**Глобальные переменные**:

- `__root__` (Path): Путь к корневой директории проекта, полученный с помощью `set_project_root()`.
- `settings` (dict): Словарь с настройками проекта, загруженный из `settings.json`. Значение по умолчанию `None`.
- `doc_str` (str): Строка с описанием проекта, загруженная из `README.MD`. Значение по умолчанию `None`.
- `__project_name__` (str): Название проекта, полученное из настроек или установленное по умолчанию.
- `__version__` (str): Версия проекта, полученная из настроек или установленная по умолчанию.
- `__doc__` (str): Описание проекта, полученное из `README.MD` или пустая строка, если файл не найден.
- `__details__` (str): Подробное описание проекта (пустая строка по умолчанию).
- `__author__` (str): Автор проекта, полученный из настроек или пустая строка по умолчанию.
- `__copyright__` (str): Авторские права, полученные из настроек или пустая строка по умолчанию.
- `__cofee__` (str): Ссылка для поддержки разработчика.  Значение по умолчанию `Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69`.
