# Модуль: src.utils

## Обзор

Этот документ описывает назначение и структуру модуля `src.utils`, который содержит служебные утилиты проекта. Основная цель модуля - предоставлять инструменты, которые могут быть использованы разными частями проекта, минимизируя зависимости от внутренних утилит.

## Инструкции

### Общие положения

Модуль `src.utils` содержит утилиты, которые должны быть максимально независимы от внутренних частей проекта.
Это означает, что данный модуль использует только стандартные библиотеки Python, такие как `json`, и не использует внутренние утилиты проекта.

### `j_loads`, `j_loads_ns` и `j_dumps`

Важно отметить, что в модулях `src.utils` и `src.utils.converters` **не используются** функции `j_loads`, `j_loads_ns` и `j_dumps`. Вместо них используется стандартная библиотека `json` для работы с JSON.

### Использование `import json`

Для работы с JSON-данными используйте стандартную библиотеку `json`, импортируя ее следующим образом:

```python
import json
```
