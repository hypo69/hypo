# Задача No 712
## Зависимые множества

## Условия

Назовем множество целых чисел *зависимым*, если один из его элементов является суммой других элементов (не обязательно всех).

Например, множество {5, 1, 4, 2} является зависимым, поскольку 5 = 1 + 4. Множество {1, 2, 3} также является зависимым, поскольку 3 = 1 + 2. Однако, множество {1, 5, 8} является независимым.

Пусть f(*n*) будет числом независимых подмножеств множества {1, 2, ..., *n*}.
Например, f(3) = 5, поскольку существует 5 независимых подмножеств: {}, {1}, {2}, {3}, {1, 2}.

Пусть F(*n*) будет суммой f(*i*) для 1 ≤ *i* ≤ *n*.
Известно, что F(6) = 142.

Найдите F(10<sup>7</sup>) mod 1 000 000 007.
