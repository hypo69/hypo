# Модуль: utils.comments_to_model.about_utils

## Обзор

Этот модуль содержит общую информацию об утилитах, используемых для преобразования комментариев в модель. Он предоставляет краткое описание целей и принципов работы этих утилит.

## Функции

### `get_comments_from_docstring`

**Описание**: Эта функция извлекает комментарии из docstring Python-функции.

**Параметры**:

- `docstring` (str): Текст docstring Python-функции.

**Возвращает**:

- `list[str]`: Список строк, содержащих комментарии из docstring. Возвращает пустой список, если docstring не найден.


### `extract_function_data`

**Описание**:  Эта функция анализирует docstring и извлекает информацию о параметрах, возвращаемых значениях и исключениях для функции.

**Параметры**:

- `docstring` (str): Текст docstring Python-функции.

**Возвращает**:

- `dict`: Словарь, содержащий информацию о параметрах, возвращаемых значениях и исключениях. Возвращает None, если docstring некорректен или пуст.


### `extract_class_data`

**Описание**:  Эта функция анализирует docstring и извлекает информацию о параметрах, возвращаемых значениях и исключениях для класса.

**Параметры**:

- `docstring` (str): Текст docstring Python-класса.

**Возвращает**:

- `dict`: Словарь, содержащий информацию о параметрах, возвращаемых значениях и исключениях. Возвращает None, если docstring некорректен или пуст.
