# Модуль hypotez/src/suppliers/hb/graber.py

## Обзор

Модуль `hypotez/src/suppliers/hb/graber.py` содержит класс `Graber`, предназначенный для сбора данных о товарах со страницы `hb.co.il`. Класс наследуется от `Grbr` (из модуля `src.suppliers`) и предоставляет методы для извлечения различных полей товара.  Каждый метод извлечения поля (например, `id_product`, `name`, `price`) может быть переопределён для специфичной обработки данных.  Класс также использует асинхронную обработку и может использовать декоратор `close_pop_up` для закрытия всплывающих окон.

## Классы

### `Graber`

**Описание**: Класс `Graber` отвечает за сбор данных о товарах с сайта `hb.co.il`. Он наследуется от базового класса `Grbr` и переопределяет методы для специфической обработки данных, если это необходимо.

**Атрибуты**:

* `supplier_prefix`:  Строка, представляющая префикс поставщика (`'hb'`).


**Методы**:

#### `__init__(self, driver: Driver)`

**Описание**: Инициализирует экземпляр класса `Graber`.

**Параметры**:
* `driver` (Driver): Экземпляр драйвера WebDriver.

#### `grab_page(self, driver: Driver) -> ProductFields`

**Описание**: Асинхронный метод для извлечения данных о товаре.

**Параметры**:
* `driver` (Driver): Экземпляр драйвера WebDriver.


**Возвращает**:
* `ProductFields`: Объект содержащий собранные данные о товаре.

## Дополнительная информация

* Модуль использует асинхронное программирование (asyncio).
* Для обработки ошибок используется блок `try...except` и обработка исключения `ExecuteLocatorException`.
* Внутри метода `grab_page` имеется функция `fetch_all_data`, которая вызывает другие методы для извлечения конкретных полей товара.
