# Конфигурация TinyTroupe

## Обзор

Данный файл (`config.ini`) содержит конфигурацию для модуля TinyTroupe, определяющую параметры взаимодействия с API OpenAI или Azure OpenAI.  Он задаёт тип API, модель, количество токенов, температуру, штрафы за частоту и присутствие слов, таймауты и другие настройки для оптимизации и управления работой API.

## Параметры

### `API_TYPE`

**Описание**: Тип API, используемый для взаимодействия с OpenAI.

**Значение по умолчанию**: `openai`

**Допустимые значения**: `openai`, `azure`

### `AZURE_API_VERSION`

**Описание**: Версия API Azure OpenAI.

**Значение по умолчанию**: `2023-05-15`

**Примечание**: Для получения актуальной информации о доступных версиях, см. документацию Azure OpenAI.


### `MODEL`

**Описание**:  Модель OpenAI, используемая для обработки запросов.

**Значение по умолчанию**: `gpt-4o`

**Допустимые значения**:  Смотри документацию OpenAI.


### `MAX_TOKENS`

**Описание**: Максимальное количество токенов, которое может быть обработано в одном запросе.

**Значение по умолчанию**: `4000`


### `TEMPERATURE`

**Описание**: Параметр, влияющий на случайность генерации текста. Значение от 0 до 1, где 0 - полностью детерминированная генерация, а 1 - максимальная случайность.

**Значение по умолчанию**: `0.3`


### `FREQ_PENALTY`

**Описание**: Штраф за частоту повторяющихся слов.

**Значение по умолчанию**: `0.0`


### `PRESENCE_PENALTY`

**Описание**: Штраф за присутствие повторяющихся слов.

**Значение по умолчанию**: `0.0`


### `TIMEOUT`

**Описание**: Максимальное время ожидания ответа от API.

**Значение по умолчанию**: `60` (секунд)


### `MAX_ATTEMPTS`

**Описание**: Максимальное количество попыток запроса к API.

**Значение по умолчанию**: `5`


### `WAITING_TIME`

**Описание**: Время ожидания между последовательными попытками запроса к API.

**Значение по умолчанию**: `1` (секунда)


### `EXPONENTIAL_BACKOFF_FACTOR`

**Описание**:  Множитель для экспоненциального увеличения времени ожидания между последующими попытками, в случае неудачи.

**Значение по умолчанию**: `5`


### `EMBEDDING_MODEL`

**Описание**: Модель для создания эмбеддингов текста.

**Значение по умолчанию**: `text-embedding-3-small`



### `CACHE_API_CALLS`

**Описание**: Флаг, определяющий использование кеширования результатов API-вызовов.

**Значение по умолчанию**: `False`


### `CACHE_FILE_NAME`

**Описание**: Имя файла для кеширования результатов API-вызовов.

**Значение по умолчанию**: `openai_api_cache.pickle`


### `MAX_CONTENT_DISPLAY_LENGTH`

**Описание**: Максимальная длина контента, отображаемого в программе.

**Значение по умолчанию**: `1024`


### `[Simulation]`
### `RAI_HARMFUL_CONTENT_PREVENTION`

**Описание**: Включить или выключить предотвращение генерации вредного контента.

**Значение по умолчанию**: `True`

### `RAI_COPYRIGHT_INFRINGEMENT_PREVENTION`

**Описание**: Включить или выключить предотвращение нарушения авторских прав.

**Значение по умолчанию**: `True`

### `[Logging]`
### `LOGLEVEL`

**Описание**: Уровень ведения логов.

**Значение по умолчанию**: `ERROR`
**Допустимые значения**: `ERROR`, `WARNING`, `INFO`, `DEBUG`
