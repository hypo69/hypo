# Модуль `header.py`

## Обзор

Модуль `header.py` предназначен для определения основных параметров и настроек проекта. Он устанавливает корневую директорию проекта, загружает конфигурацию из файла `config.json` и описание из `README.md`. Также определяет метаданные проекта, такие как имя, версию, автора и информацию об авторских правах.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла, и останавливается на первой директории, содержащей один из маркеров.

**Параметры**:
- `marker_files` (tuple, optional): Кортеж имен файлов или директорий, которые идентифицируют корневую директорию проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена, иначе путь к директории, где расположен скрипт.

## Переменные модуля

### `__root__`

**Описание**: Путь к корневой директории проекта.

### `config`

**Описание**: Словарь, содержащий конфигурацию проекта, загруженную из файла `config.json`.

### `doc_str`

**Описание**: Строка, содержащая описание проекта, загруженное из файла `README.md`.

### `__project_name__`

**Описание**: Имя проекта. По умолчанию 'hypotez', если не найдено в файле конфигурации.

### `__version__`

**Описание**: Версия проекта. Пустая строка, если не найдена в файле конфигурации.

### `__doc__`

**Описание**: Документация проекта из файла `README.md`, пустая строка, если файл не найден.

### `__details__`

**Описание**: Строка с детальной информацией о проекте. Изначально пустая.

### `__author__`

**Описание**: Автор проекта. Пустая строка, если не найден в файле конфигурации.

### `__copyright__`

**Описание**: Информация об авторских правах. Пустая строка, если не найдена в файле конфигурации.

### `__cofee__`
**Описание**: Призыв угостить разработчика чашкой кофе.  Ссылка на boosty, если переменная settings не найдена.
