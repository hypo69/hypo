# Модуль __init__.py

## Обзор

Этот модуль содержит инициализацию для TinyTroupe, включая импорт необходимых библиотек, чтение конфигурации, настройку логгера и добавление текущей директории в системный путь. Он также включает в себя вывод предупреждения об использовании ИИ и форматирование вывода для Jupyter.

## Импорты

- `os`: Для работы с операционной системой.
- `logging`: Для ведения журнала.
- `configparser`: Для чтения конфигурационных файлов.
- `rich`: Библиотека для форматированного вывода.
- `rich.jupyter`: Поддержка Jupyter Notebook для rich.
- `utils`: Модуль служебных функций из пакета `tinytroupe`.

## Функции

### `read_config_file`

**Описание**: Читает конфигурационный файл и возвращает объект `configparser.ConfigParser`.

**Параметры**:

-  Нет.


**Возвращает**:

- `configparser.ConfigParser`: Объект конфигурации.


**Возможные исключения**:

- `FileNotFoundError`: Если конфигурационный файл не найден.
- `configparser.Error`: Если файл конфигурации имеет неправильный формат.


### `pretty_print_config`

**Описание**: Выводит конфигурацию в красивом формате.

**Параметры**:

- `config (configparser.ConfigParser)`: Объект конфигурации для вывода.


**Возвращает**:

-  `None`.


**Возможные исключения**:

-  Нет.



### `start_logger`

**Описание**: Настраивает и запускает логгер.

**Параметры**:

- `config (configparser.ConfigParser)`: Объект конфигурации, содержащий настройки логгера.


**Возвращает**:

-  `None`.


**Возможные исключения**:

- Нет.


### `inject_html_css_style_prefix`

**Описание**: Добавляет префикс CSS стилей для предотвращения отступов в Jupyter Notebook.

**Параметры**:

- `html_format (str)`: Текущий формат HTML.
- `css_prefix (str)`: Префикс CSS.


**Возвращает**:

- `str`: Измененный формат HTML с добавленным префиксом CSS.


**Возможные исключения**:

- Нет.
