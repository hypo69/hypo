# Модуль `test_extraction.py`

## Обзор

Этот модуль содержит модульные тесты для проверки функциональности классов `ArtifactExporter` и `Normalizer`, используемых для экспорта артефактов в различные форматы и нормализации концепций соответственно.

## Функции

### `exporter`

**Описание**:
Фикстура Pytest для создания экземпляра `ArtifactExporter` с базовой папкой вывода "./test_exports".

**Возвращает**:
- `ArtifactExporter`: Экземпляр `ArtifactExporter`.

### `test_export_json`

**Описание**:
Тестирует экспорт артефактов в формате JSON с использованием класса `ArtifactExporter`.

**Параметры**:
- `exporter`: Экземпляр `ArtifactExporter`, предоставляемый фикстурой.

**Вызывает исключения**:
- `AssertionError`: Если JSON-файл не был экспортирован или экспортированные данные не совпадают с оригинальными.

### `test_export_text`

**Описание**:
Тестирует экспорт артефактов в текстовом формате с использованием класса `ArtifactExporter`.

**Параметры**:
- `exporter`: Экземпляр `ArtifactExporter`, предоставляемый фикстурой.

**Вызывает исключения**:
- `AssertionError`: Если текстовый файл не был экспортирован или экспортированные данные не совпадают с оригинальными.

### `test_export_docx`

**Описание**:
Тестирует экспорт артефактов в формате docx с использованием класса `ArtifactExporter`. Также проверяет, что форматирование Markdown преобразуется в docx.

**Параметры**:
- `exporter`: Экземпляр `ArtifactExporter`, предоставляемый фикстурой.

**Вызывает исключения**:
- `AssertionError`: Если docx-файл не был экспортирован или экспортированные данные не содержат оригинальный текст или содержат элементы форматирования markdown.

### `test_normalizer`

**Описание**:
Тестирует нормализацию концепций с использованием класса `Normalizer`.

**Вызывает исключения**:
- `AssertionError`: Если количество нормализованных элементов не соответствует заданному или если нормализующая карта не ведет себя должным образом.
