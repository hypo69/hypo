# NIM

## סקירה כללית

קובץ זה מיישם את המשחק "NIM", משחק מתמטי לשני שחקנים. המשחק מתחיל עם מספר ערימות של אבנים (או פריטים אחרים), ושחקנים לוקחים בתורות כל מספר של אבנים מערימה אחת. השחקן שלוקח את האבן האחרונה מנצח.

## פונקציות

### `display_piles`

**תיאור**:
מציג את המצב הנוכחי של ערימות האבנים.

**פרמטרים**:
- `piles` (list): רשימה המייצגת את ערימות האבנים.

**החזרות**:
- אין.

### `get_player_move`

**תיאור**:
מבקש מהשחקן מספר ערימה ומספר אבנים להסרה.

**פרמטרים**:
- `piles` (list): רשימה המייצגת את ערימות האבנים.
- `player` (int): מספר השחקן הנוכחי (1 או 2).

**החזרות**:
- `tuple`: טופל המכיל את מספר הערימה ואת מספר האבנים להסרה.

**Raises**:
- `ValueError`: אם הקלט אינו מספר שלם.
