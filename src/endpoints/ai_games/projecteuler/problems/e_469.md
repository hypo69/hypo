# Задача No 469
## Разноцветные шары

## Условия
Представьте себе ящик, содержащий n разноцветных шаров. Каждый раз, когда вы достаете случайный шар из ящика, вы помещаете его обратно, но заменяете его двумя шарами того же цвета.

Пусть E(n) будет ожидаемым количеством ходов, которые нужно сделать, чтобы получить по крайней мере один шар каждого цвета.

Например, E(1) = 1, E(2) = 3, E(3) = 11, и E(4) = 49.

Найдите E(10<sup>6</sup>) по модулю 1000000007.
