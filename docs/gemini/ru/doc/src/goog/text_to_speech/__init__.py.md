# Модуль `src.goog.text_to_speech`

## Обзор

Модуль `src.goog.text_to_speech` предоставляет функциональность для преобразования текста в речь, используя Google Text-to-Speech API и pyttsx3. Модуль предназначен для использования на платформах Windows и Unix.
