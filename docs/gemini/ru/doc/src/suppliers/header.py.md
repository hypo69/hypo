# Модуль `src.suppliers.header`

## Обзор

Модуль `src.suppliers.header` предназначен для определения корневой директории проекта, загрузки настроек из файла `settings.json`, чтения документации из `README.MD`, а также для хранения основных метаданных проекта.

## Функции

### `set_project_root`

**Описание**: 
Находит корневую директорию проекта, начиная с директории текущего файла, идя вверх по дереву каталогов до первого каталога, содержащего любой из маркерных файлов.

**Параметры**:
- `marker_files` (tuple): Кортеж с именами файлов или каталогов, по которым определяется корень проекта. По умолчанию (`'__root__'`, `'.git'`).

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена. Если корень не найден, возвращается директория, где расположен скрипт.

## Переменные

### `__root__`

**Описание**: 
Путь к корневой директории проекта, который определяется функцией `set_project_root`.

### `settings`

**Описание**: 
Словарь с настройками проекта, загруженный из файла `settings.json`. Если файл не найден или имеет неверный формат JSON, значение будет `None`.

### `doc_str`

**Описание**:
Содержимое файла `README.MD` в виде строки. Если файл не найден или не может быть прочитан, значение будет `None`.

### `__project_name__`

**Описание**:
Имя проекта, взятое из настроек `settings` или по умолчанию `'hypotez'`.

### `__version__`

**Описание**:
Версия проекта, взятая из настроек `settings` или пустая строка `''`.

### `__doc__`

**Описание**:
Строка документации проекта, загруженная из файла `README.MD`. Если файл не найден, значение будет пустой строкой `''`.

### `__details__`

**Описание**:
Строка с дополнительной информацией о проекте. В данном коде всегда инициализируется как пустая строка `''`.

### `__author__`

**Описание**:
Имя автора проекта, взятое из настроек `settings` или пустая строка `''`.

### `__copyright__`

**Описание**:
Информация о копирайте проекта, взятая из настроек `settings` или пустая строка `''`.

### `__cofee__`

**Описание**:
Сообщение с призывом поддержать разработчика чашкой кофе, взятое из настроек `settings` или строка по умолчанию "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69".
