## Класс `ProductFields`

**Описание**: Класс `ProductFields` предоставляет методы для работы с полями товаров в базе данных PrestaShop.  Он позволяет получать и устанавливать значения полей, а также управлять данными о товарах в соответствии со структурой таблиц PrestaShop.

**Атрибуты**:
- `product_fields_list`: Список наименований полей товаров.
- `language`: Словарь, сопоставляющий названия языков с их идентификаторами в PrestaShop.
- `presta_fields`: Объект `SimpleNamespace`, содержащий поля товара, соответствующие полям таблицы `ps_product`.
- `assist_fields_dict`: Словарь для хранения дополнительных полей, например, URL изображений.


### `__init__`

**Описание**: Конструктор класса `ProductFields`. Инициализирует атрибуты класса, загружает список полей из файла `fields_list.txt`, загружает значения полей из файла `product_fields_default_values.json` и создаёт вспомогательный объект `presta_fields`.
