# `src/webdriver/header.py`

## Обзор

Модуль `header.py` содержит общие настройки и переменные, используемые в проекте `hypotez`.
Он определяет путь к корневой директории проекта, загружает настройки из `settings.json` и `README.MD`, а также устанавливает основные метаданные проекта.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла, и останавливается на первой директории, содержащей один из маркерных файлов.

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или директорий для идентификации корневой директории проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена, в противном случае - путь к директории, где находится скрипт.

## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта. Инициализируется функцией `set_project_root`.

### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json`. Может быть `None`, если файл не найден или не может быть прочитан.

### `doc_str`

**Описание**: Строка с содержимым файла `README.MD`. Может быть `None`, если файл не найден или не может быть прочитан.

### `__project_name__`

**Описание**: Название проекта, загруженное из `settings.json` или по умолчанию `hypotez`.

### `__version__`

**Описание**: Версия проекта, загруженная из `settings.json`. Может быть пустой строкой, если не найдена в настройках.

### `__doc__`

**Описание**: Строка документации проекта, загруженная из `README.MD`.

### `__details__`

**Описание**: Строка с дополнительной информацией о проекте. В данный момент пустая строка.

### `__author__`

**Описание**: Автор проекта, загруженный из `settings.json`. Может быть пустой строкой, если не найдена в настройках.

### `__copyright__`

**Описание**: Информация о копирайте проекта, загруженная из `settings.json`. Может быть пустой строкой, если не найдена в настройках.

### `__cofee__`

**Описание**: Строка с ссылкой на поддержку разработчика, загруженная из `settings.json`. По умолчанию "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69".
