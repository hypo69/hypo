# Модуль конфигурации TinyTroupe

## Обзор

Данный файл (`config.ini`) содержит конфигурационные параметры для работы с API моделей OpenAI и Azure OpenAI. Он определяет тип API, параметры модели, временные ограничения, политику кэширования, уровень ведения логов и параметры моделирования.

## Параметры API

### `API_TYPE`

**Описание**: Тип используемого API. Возможные значения: `openai` или `azure`. По умолчанию `openai`.

**Значение по умолчанию**: `openai`

### `AZURE_API_VERSION`

**Описание**: Версия API для Azure OpenAI.

**Значение по умолчанию**: `2023-05-15`

## Параметры модели

### `MODEL`

**Описание**: Идентификатор модели OpenAI, используемой для генерации текста.

**Значение по умолчанию**: `gpt-4o`

### `MAX_TOKENS`

**Описание**: Максимальное количество токенов, которое может сгенерировать модель.

**Значение по умолчанию**: `4000`

### `TEMPERATURE`

**Описание**: Параметр, влияющий на степень случайности генерации. Значение от 0 до 1, где 0 – полностью детерминированная генерация, 1 – максимальная случайность.

**Значение по умолчанию**: `0.3`

### `FREQ_PENALTY`

**Описание**: Наказание за повторяющиеся токены в генерации.

**Значение по умолчанию**: `0.0`

### `PRESENCE_PENALTY`

**Описание**: Наказание за повторение целых фраз или понятий в генерации.

**Значение по умолчанию**: `0.0`

### `TIMEOUT`

**Описание**: Максимальное время ожидания в секундах для получения ответа от API.

**Значение по умолчанию**: `60`

### `MAX_ATTEMPTS`

**Описание**: Максимальное количество попыток запроса к API.

**Значение по умолчанию**: `5`

### `WAITING_TIME`

**Описание**: Время ожидания между попытками в экспоненциальном отставании (в секундах).

**Значение по умолчанию**: `1`

### `EXPONENTIAL_BACKOFF_FACTOR`

**Описание**: Коэффициент для экспоненциального отставания.

**Значение по умолчанию**: `5`


### `EMBEDDING_MODEL`

**Описание**: Модель для извлечения векторных представлений (встраиваний).

**Значение по умолчанию**: `text-embedding-3-small`

## Кэширование

### `CACHE_API_CALLS`

**Описание**: Флаг включения/выключения кэширования вызовов API.

**Значение по умолчанию**: `False`

### `CACHE_FILE_NAME`

**Описание**: Имя файла для сохранения кэша.

**Значение по умолчанию**: `openai_api_cache.pickle`

## Отображение содержания

### `MAX_CONTENT_DISPLAY_LENGTH`

**Описание**: Максимальная длина отображаемого контента.

**Значение по умолчанию**: `1024`


## Моделирование

### `RAI_HARMFUL_CONTENT_PREVENTION`

**Описание**: Включить/выключить предотвращение генерации вредоносного контента.

**Значение по умолчанию**: `True`

### `RAI_COPYRIGHT_INFRINGEMENT_PREVENTION`

**Описание**: Включить/выключить предотвращение нарушения авторских прав.

**Значение по умолчанию**: `True`


## Логирование

### `LOGLEVEL`

**Описание**: Уровень детализации логов. Возможные значения: `ERROR`, `WARNING`, `INFO`, `DEBUG`.

**Значение по умолчанию**: `ERROR`
