# Модуль `src.webdriver.edge._examples`

## Обзор

Модуль `src.webdriver.edge._examples` представляет собой набор примеров и вспомогательных данных для работы с веб-драйвером Edge. Он предназначен для использования в операционных системах Windows и Unix.
