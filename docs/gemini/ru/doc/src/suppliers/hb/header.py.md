# Модуль `header.py`

## Обзор

Модуль `header.py` отвечает за настройку и инициализацию базовых параметров проекта, включая поиск корневой директории проекта, загрузку настроек из `settings.json`, чтение документации из `README.MD` и определение основных атрибутов проекта, таких как имя, версия и автор.

## Функции

### `set_project_root`

**Описание**:
Находит корневую директорию проекта, начиная с директории текущего файла и двигаясь вверх, пока не найдет директорию, содержащую один из указанных маркерных файлов.

**Параметры**:
- `marker_files` (tuple, optional): Кортеж имен файлов или директорий для определения корневой директории проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена. В противном случае возвращает директорию, где расположен скрипт.

## Переменные модуля

### `MODE`

**Описание**:
Указывает текущий режим работы. По умолчанию установлен в значение `dev`.

### `__root__`

**Описание**:
`Path`: Путь к корневой директории проекта. Инициализируется функцией `set_project_root`.

### `settings`

**Описание**:
`dict`: Словарь, содержащий настройки проекта, загруженные из файла `settings.json`. Может быть `None`, если файл не найден или произошла ошибка при его загрузке.

### `doc_str`

**Описание**:
`str`: Строка, содержащая документацию проекта, загруженную из файла `README.MD`. Может быть `None`, если файл не найден или произошла ошибка при его загрузке.

### `__project_name__`

**Описание**:
`str`: Название проекта, извлекаемое из файла `settings.json`. Если настройка не найдена, по умолчанию установлено значение `'hypotez'`.

### `__version__`

**Описание**:
`str`: Версия проекта, извлекаемая из файла `settings.json`. Если настройка не найдена, по умолчанию установлено значение `''`.

### `__doc__`

**Описание**:
`str`: Строка, содержащая документацию проекта. Загружается из переменной `doc_str`, в противном случае равно `''`.

### `__details__`

**Описание**:
`str`: Дополнительные сведения о проекте. По умолчанию установлено значение `''`.

### `__author__`

**Описание**:
`str`: Автор проекта, извлекаемый из файла `settings.json`. Если настройка не найдена, по умолчанию установлено значение `''`.

### `__copyright__`

**Описание**:
`str`: Информация об авторских правах проекта, извлекаемая из файла `settings.json`. Если настройка не найдена, по умолчанию установлено значение `''`.

### `__cofee__`

**Описание**:
`str`: Сообщение с информацией о поддержке разработчика, извлекается из файла `settings.json`. Если настройка не найдена, по умолчанию установлено значение `"Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69"`.
