**Описание:**

Этот файл содержит инициализацию модуля `scenario`, предоставляющего функции для выполнения сценариев.  Модуль предназначен для управления сценариями выполнения поставщиков (например, AliExpress), которые взаимодействуют с драйвером (например, для интеграции с PrestaShop).

**Ключевые функции:**

* **`run_scenario_files(supplier, scenario_files)`:** Выполняет сценарии, определённые в переданных файлах.  Ожидается, что файлы содержат описание сценариев в формате JSON.
* **`run_scenarios(supplier, scenarios)`:** Выполняет сценарии, переданные в виде списка словарей. Каждый словарь представляет собой отдельный сценарий.
* **`run_scenario(supplier, scenario)`:** Выполняет отдельный сценарий.

**Структура выполнения сценариев:**

Файл иллюстрирует архитектуру, где сценарии определяют действия, исполнитель (executor) выполняет их, а поставщик (supplier) предоставляет данные для сценария, а драйвер (driver) предоставляет интерфейс для взаимодействия с целевой системой (например, PrestaShop).
