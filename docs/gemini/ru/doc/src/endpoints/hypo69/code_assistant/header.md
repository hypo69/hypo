# Модуль `hypotez/src/logger/header.py`

## Обзор

Модуль `hypotez/src/logger/header.py` определяет корневой путь к проекту `hypotez`. Все импорты строятся относительно этого пути. В дальнейшем путь к проекту будет перенесен в системную переменную.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла, перемещаясь вверх по директориям до тех пор, пока не будет найдена директория, содержащая один из указанных файлов-маркеров.

**Параметры**:

- `marker_files` (tuple): Кортеж из имен файлов или каталогов, используемых для определения корневой директории проекта.

**Возвращает**:

- `Path`: Путь к корневой директории, если найдена, иначе путь к директории, где находится текущий скрипт.

**Вызывает исключения**:

- Не вызывает исключений.

## Глобальные переменные

### `__root__`

**Описание**: Путь к корневой директории проекта.


## Обработка настроек

**Описание**: Чтение настроек из файла `settings.json` в корневой директории проекта.


**Возможные исключения**:

- `FileNotFoundError`: Если файл `settings.json` не найден.
- `json.JSONDecodeError`: Если файл `settings.json` содержит невалидные данные JSON.

## Обработка документации

**Описание**: Чтение документации из файла `README.MD` в корневой директории проекта.


**Возможные исключения**:

- `FileNotFoundError`: Если файл `README.MD` не найден.
- `json.JSONDecodeError`: Если файл `README.MD` содержит невалидные данные.


## Другие переменные

### `__project_name__`

**Описание**: Название проекта, полученное из настроек `settings.json` или значение по умолчанию `hypotez`.

### `__version__`

**Описание**: Версия проекта, полученная из настроек `settings.json` или пустая строка по умолчанию.

### `__doc__`

**Описание**: Документация проекта, полученная из `README.MD` или пустая строка по умолчанию.

### `__details__`

**Описание**: Подробности проекта, по умолчанию пустая строка.

### `__author__`

**Описание**: Автор проекта, полученный из настроек `settings.json` или пустая строка по умолчанию.

### `__copyright__`

**Описание**: Авторские права проекта, полученные из настроек `settings.json` или пустая строка по умолчанию.

### `__cofee__`

**Описание**: Ссылка для поддержки разработчика, полученная из настроек `settings.json` или значение по умолчанию.
