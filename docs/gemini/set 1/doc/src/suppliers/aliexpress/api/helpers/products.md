# Модуль hypotez/src/suppliers/aliexpress/api/helpers/products.py

## Обзор

Модуль `products.py` содержит функции для обработки данных о продуктах, полученных с API AliExpress.  В частности, он предоставляет функции для парсинга отдельных продуктов и списков продуктов.

- [Функции](#функции)


## Функции

### `parse_product`

**Описание**: Функция обрабатывает данные об одном продукте, извлекая URL маленьких изображений продукта.


**Параметры**:

- `product`: (Объект): Данные о продукте.  Предполагается, что `product` имеет атрибут `product_small_image_urls`, содержащий список URL-адресов изображений.


**Возвращает**:

- `product`: (Объект): Объект `product`, обновленный с URL маленьких изображений.


**Вызывает исключения**:

- Нет.



### `parse_products`

**Описание**: Функция обрабатывает список продуктов, применяя функцию `parse_product` к каждому элементу списка.


**Параметры**:

- `products`: (Список): Список данных о продуктах. Каждый элемент списка должен быть объектом, который можно передать в функцию `parse_product`.


**Возвращает**:

- `new_products`: (Список): Новый список обработанных продуктов.


**Вызывает исключения**:

- Нет.
