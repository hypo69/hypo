## Структура

### `product_links`

**Описание**:
  Содержит данные для определения ссылок на страницы продуктов на странице категории.

**Структура**:

- `attribute` (str): Атрибут, из которого извлекается ссылка на продукт. В данном случае "href".
- `by` (str): Способ поиска элемента. В данном случае "XPATH".
- `selector` (str): XPath-выражение для поиска элемента, содержащего ссылку на продукт: `//span[@data-component-type ='s-product-image']//a`.
- `if_list` (str): Указывает, что нужно взять первый элемент списка. В данном случае "first".
- `use_mouse` (bool): Указывает, нужно ли использовать мышь. В данном случае `false`.
- `mandatory` (bool): Указывает, является ли элемент обязательным. В данном случае `true`.
- `timeout` (int): Время ожидания элемента. В данном случае `0`.
- `timeout_for_event` (str): Событие, которое нужно ожидать. В данном случае `"presence_of_element_located"`.
- `event` (str | null): Событие, которое нужно ожидать. В данном случае `null`.
