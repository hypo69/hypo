# Задача No 775
## Сумма квадратов

## Условия
Сумма квадратов первых n натуральных чисел равна n(n+1)(2n+1)/6.

Определим функцию *s(n)*, как сумму квадратов целых чисел от 1 до n, которые делятся на 3. Например, s(9) = 3<sup>2</sup> + 6<sup>2</sup> + 9<sup>2</sup> = 126.

Далее, пусть S(N) будет суммой *s(n)* при 1 ≤ n ≤ N.
Например, S(10) = s(1) + s(2) + ... + s(10) = 0 + 0 + 3<sup>2</sup> + 3<sup>2</sup> + 3<sup>2</sup> + 3<sup>2</sup> + 3<sup>2</sup> + 6<sup>2</sup> + 3<sup>2</sup> + 3<sup>2</sup> + 6<sup>2</sup> + 9<sup>2</sup> = 243 + 81 + 126 = 450.

Дано, что S(1000) = 721873200.

Найдите S(123456789012345) mod 1 000 000 007.
