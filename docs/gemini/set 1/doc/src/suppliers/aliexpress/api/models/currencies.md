# Модуль `hypotez/src/suppliers/aliexpress/api/models/currencies.py`

## Обзор

Этот модуль определяет константы для валют, используемых API AliExpress.

## Классы

### `Currency`

**Описание**: Класс, содержащий константы для различных валют.

**Константы**:

- `USD`: Доллар США.
- `GBP`: Фунт стерлингов.
- `CAD`: Канадский доллар.
- `EUR`: Евро.
- `UAH`: Гривна.
- `MXN`: Мексиканский песо.
- `TRY`: Турецкая лира.
- `RUB`: Российский рубль.
- `BRL`: Бразильский реал.
- `AUD`: Австралийский доллар.
- `INR`: Индийская рупия.
- `JPY`: Японская йена.
- `IDR`: Индонезийская рупия.
- `SEK`: Шведская крона.
- `KRW`: Южнокорейская вона.
- `ILS`: Израильский шекель.
