# Модуль `src.endpoints.prestashop.product_fields`

## Обзор

Модуль `src.endpoints.prestashop.product_fields` предназначен для управления полями товара в контексте Prestashop. Он включает в себя классы и функции для обработки и перевода полей товара.

  - [`ProductFields`](#productfields)
  - [`translate_presta_fields_dict`](#translate_presta_fields_dict)

## Классы

### `ProductFields`

**Описание**:
Класс `ProductFields` предназначен для работы с полями товара. Детали реализации этого класса находятся в файле `product_fields.py`.

### `translate_presta_fields_dict`

**Описание**:
Функция `translate_presta_fields_dict` предназначена для перевода словаря полей Prestashop. Детали реализации этой функции находятся в файле `product_fields_translator.py`.
