# Модуль hypotez/src/suppliers/aliexpress/graber.py

## Обзор

Модуль `hypotez/src/suppliers/aliexpress/graber.py` предоставляет класс `Graber`, предназначенный для сбора данных о товарах с сайта aliexpress.com. Класс наследуется от базового класса `Graber` и переопределяет некоторые функции для специфической обработки данных с aliexpress.  Реализует асинхронный захват данных с помощью веб-драйвера.  Предоставляет функции для обработки различных полей товара.  Поддерживает предварительные действия перед запросом к веб-драйверу, используя декоратор (не реализован в данном случае).
