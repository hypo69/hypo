# Документация для модуля test_gui.py

## Обзор

Модуль `test_gui.py` предназначен для запуска графического интерфейса (GUI) для проекта g4f (GPT4Free). Он добавляет путь к родительскому каталогу в `sys.path` и вызывает функцию `run_gui` для запуска GUI.

## Подробней

Этот модуль упрощает запуск графического интерфейса пользователя, позволяя разработчикам и пользователям взаимодействовать с функциональностью GPT4Free через удобный интерфейс. Добавление родительского каталога в `sys.path` необходимо для корректной работы импортов внутри модуля `g4f.gui`.
