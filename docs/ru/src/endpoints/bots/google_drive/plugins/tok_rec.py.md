# Модуль для определения токенов

## Обзор

Модуль содержит функцию `is_token`, предназначенную для идентификации токенов определенного формата. В текущей версии модуля функция ориентирована на проверку токенов, заканчивающихся определенной последовательностью символов и имеющих определенную длину.

## Функции

### `is_token`

```python
def is_token(token: str) -> bool:
    """
    Функция проверяет, является ли входная строка токеном определенного формата.

    Args:
        token (str): Строка, которую необходимо проверить на соответствие формату токена.

    Returns:
        bool: `True`, если строка соответствует формату токена, `False` в противном случае.
    """
```

**Как работает функция**:

Функция `is_token` выполняет следующие шаги:

1. **Извлечение последнего слова**: Из входной строки `token` извлекается последнее слово, разделенное пробелами. Это делается с помощью `token.split()[-1]`.
2. **Определение длины токена**: Определяется длина полученного слова и сохраняется в переменной `TLEN`.
3. **Проверка длины и формата**: Если длина `TLEN` равна 62, выполняется проверка, что второй символ токена равен "/". Если оба условия выполняются, функция возвращает `True`, иначе - `False`. Если длина `TLEN` не равна 62, функция сразу возвращает `False`.

```
A: Получение последнего слова из входной строки
|
B: Определение длины полученного слова
|
C: Проверка длины слова (TLEN == 62)
|
D: Если TLEN == 62: Проверка второго символа слова (token[1] == "/")
|
E: Если TLEN != 62: Возврат False
|
F: Если второй символ слова == "/": Возврат True, иначе: Возврат False
```

**Примеры**:

```python
>>> is_token("Некий токен 1234567890123456789012345678901234567890123456789012345678901/")
False

>>> is_token("Некий токен 12345678901234567890123456789012345678901234567890123456789012/")
False

>>> is_token("1234567890123456789012345678901234567890123456789012345678901/ 1")
False
