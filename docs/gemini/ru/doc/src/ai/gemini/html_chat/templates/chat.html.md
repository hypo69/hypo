# Документация для `chat.html`

## Обзор

Файл `chat.html` представляет собой HTML-шаблон для веб-страницы чата Kazarinov AI. Он включает в себя структуру страницы, стили CSS и JavaScript для обработки пользовательского ввода и отображения сообщений чата.

## Структура HTML

HTML-структура страницы включает в себя следующие основные элементы:

-   `<head>`: Содержит метаданные страницы, ссылки на стили CSS (Bootstrap и пользовательские), а также скрипт jQuery.
-   `<body>`: Содержит основной контент страницы:
    -   `container`: Основной контейнер для контента, центрирующий его на странице.
    -   `h1`: Заголовок страницы "Kazarinov AI Chat".
    -   `chat-box`: Контейнер для отображения сообщений чата с фиксированной высотой и полосой прокрутки.
    -   `chat-log`: Элемент, в который добавляются сообщения чата.
    -   `form#chat-form`: Форма для ввода сообщений пользователя.
    -   `input#user-input`: Поле ввода для ввода сообщений.
    -   `button`: Кнопка для отправки сообщений.

## Стили CSS

Файл использует следующие стили CSS:

-   **Bootstrap CSS**: Подключается через CDN для базового оформления элементов.
-   **`css/styles.css`**: Подключается пользовательский файл стилей для дополнительных настроек. Стили `styles.css` задают внешний вид чата, положение и вид элементов.

## JavaScript

Скрипт JavaScript, встроенный в файл, выполняет следующие действия:

-   **Обработчик отправки формы**:
    -   Предотвращает стандартное поведение отправки формы.
    -   Получает текст из поля ввода `input#user-input`.
    -   Добавляет сообщение пользователя в `div#chat-log`.
    -   Отправляет AJAX-запрос на сервер по адресу `'/ask'` с текстом пользователя.
    -   Получает ответ от сервера и добавляет его в `div#chat-log`.
    -   Очищает поле ввода `input#user-input`.

-   **Использование jQuery**:
    -   Используется для обработки событий и манипуляций с DOM.
    -   `$(document).ready(...)`: Гарантирует, что скрипт выполняется после полной загрузки страницы.
    -   `$('#chat-form').submit(...)`: Назначает обработчик на отправку формы.
    -   `$.ajax(...)`: Выполняет AJAX-запрос.
    -   `$('#chat-log').append(...)`: Добавляет HTML-элементы в чат.
    -   `$('#user-input').val(...)`: Устанавливает или получает значение поля ввода.
