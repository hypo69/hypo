# Модуль tinytroupe.enrichment

## Обзор

Этот модуль предоставляет класс `TinyEnricher` для обогащения контента с использованием моделей больших языков. Класс использует шаблоны Mustache для формирования запросов к LLM и извлекает код из ответа.

## Классы

### `TinyEnricher`

**Описание**: Класс для обогащения контента с помощью моделей больших языков. Наследует от `JsonSerializableRegistry`.

**Методы**:

#### `__init__`

**Описание**: Конструктор класса.

**Параметры**:

- `use_past_results_in_context` (bool): Флаг, указывающий, нужно ли использовать предыдущие результаты в контексте. По умолчанию `False`.

**Возвращает**:
- None

#### `enrich_content`

**Описание**: Метод для обогащения контента.

**Параметры**:

- `requirements` (str): Требования для обогащения.
- `content` (str): Контент, который нужно обогатить.
- `content_type` (str, optional): Тип контента. По умолчанию `None`.
- `context_info` (str, optional): Дополнительная информация о контексте. По умолчанию пустая строка.
- `context_cache` (list, optional): Кэшированные контексты. По умолчанию `None`.
- `verbose` (bool, optional): Флаг для вывода дополнительных сообщений в консоль. По умолчанию `False`.


**Возвращает**:
- `str | None`: Обогащенный контент (в формате кода) или `None`, если запрос не успешен.


**Вызывает исключения**:
- Никаких исключений не описано.
