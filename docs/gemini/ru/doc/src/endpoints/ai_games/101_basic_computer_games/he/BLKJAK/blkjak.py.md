# בלק ג'ק

## Обзор

Этот модуль реализует игру в блекджек против дилера. Игрок и дилер получают по две карты, цель игры - набрать сумму очков, как можно более близкую к 21, не превышая ее. Значения карт: числовые карты соответствуют своему номиналу, картинки (J, Q, K) стоят 10, а туз (A) может стоить 1 или 11 в зависимости от того, что выгоднее для руки.

## Функции

### `card_value`

**Описание**: Возвращает числовое значение карты.

**Параметры**:
- `card` (str): Строка, представляющая карту (например, '2', 'K', 'A').

**Возвращает**:
- `int`: Числовое значение карты. Картинки возвращают 10, туз - 11 (может быть изменен позже), числовые карты - свой номинал.

### `calculate_hand_value`

**Описание**: Вычисляет общую стоимость карт в руке, учитывая, что туз может стоить 1 или 11.

**Параметры**:
- `hand` (list): Список строк, представляющих карты в руке.

**Возвращает**:
- `int`: Общая стоимость карт в руке.

### `create_deck`

**Описание**: Создает и перемешивает стандартную колоду из 52 карт.

**Параметры**:
- Нет

**Возвращает**:
- `list`: Перемешанный список строк, представляющих карты в колоде.

### `deal_card`

**Описание**: Раздает верхнюю карту из колоды.

**Параметры**:
- `deck` (list): Список строк, представляющих колоду карт.

**Возвращает**:
- `str`: Строка, представляющая карту, которая была сдана.

### `display_hand`

**Описание**: Отображает карты в руке.

**Параметры**:
- `hand` (list): Список строк, представляющих карты в руке.
- `hide_first_card` (bool, optional): Если `True`, первая карта будет скрыта. По умолчанию `False`.

**Возвращает**:
- `None`: Функция ничего не возвращает, а печатает карты в консоль.

### `play_blackjack`

**Описание**: Запускает игру в блекджек.

**Параметры**:
- Нет

**Возвращает**:
- `None`: Функция ничего не возвращает, а управляет ходом игры.
