# Модуль `header.py`

## Обзор

Модуль `header.py` предназначен для инициализации основных параметров и настроек проекта, таких как путь к корневой директории, загрузка настроек из файла `settings.json`, а также получение информации о проекте, такой как имя, версия, описание, авторские права и т.д.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла, поднимаясь вверх и останавливаясь на первой директории, содержащей любой из файлов-маркеров.

**Параметры**:

-   `marker_files` (tuple, optional): Кортеж имен файлов или директорий, идентифицирующих корень проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

-   `Path`: Путь к корневой директории, если она найдена, иначе директория, где расположен скрипт.

## Переменные

### `MODE`

**Описание**: Режим работы приложения. По умолчанию установлен в `'dev'`.

### `__root__`

**Описание**: Путь к корневой директории проекта.

### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json`.

### `doc_str`

**Описание**: Строка с содержимым файла `README.md`.

### `__project_name__`

**Описание**: Имя проекта. По умолчанию `'hypotez'`, или значение из `settings.json`, если он загружен.

### `__version__`

**Описание**: Версия проекта. По умолчанию `''`, или значение из `settings.json`, если он загружен.

### `__doc__`

**Описание**: Описание проекта, полученное из файла `README.md`, или пустая строка, если файл не найден или не прочитан.

### `__details__`

**Описание**: Детальное описание проекта (в настоящее время пустая строка).

### `__author__`

**Описание**: Автор проекта. По умолчанию `''`, или значение из `settings.json`, если он загружен.

### `__copyright__`

**Описание**: Авторские права проекта. По умолчанию `''`, или значение из `settings.json`, если он загружен.

### `__cofee__`

**Описание**: Строка с призывом угостить разработчика кофе.  По умолчанию "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69", или значение из `settings.json`, если он загружен.

## Импорты

-   `sys`: Модуль для работы с системными параметрами и функциями.
-   `json`: Модуль для работы с JSON.
-   `packaging.version.Version`: Класс для работы с версиями пакетов.
-   `pathlib.Path`: Класс для работы с путями к файлам и директориям.
-   `src.gs`: Модуль для глобальных настроек.
