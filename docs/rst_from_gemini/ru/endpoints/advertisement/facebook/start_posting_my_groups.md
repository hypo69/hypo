# Модуль `start_posting_my_groups.py`

Файл: `C:\Users\user\Documents\repos\hypotez\src\endpoints\advertisement\facebook\start_posting_my_groups.py`

**Роль:**  `doc_creator` (создание документации).

**Описание:**

Этот скрипт отвечает за автоматическую публикацию рекламных объявлений в группы Facebook. Он использует веб-драйвер для взаимодействия с Facebook и класс `FacebookPromoter` для управления кампаниями.

**Входные данные:**

*   Список названий кампаний (`campaigns`).
*   Список путей к файлам с данными о группах (`group_file_paths`). Файлы предполагается в формате JSON, содержащем информацию о группах, в которые нужно публиковать.

**Выходные данные:**

*   Отправка рекламных объявлений в указанные группы Facebook.


**Основные функции:**

*   **Инициализация:**
    *   Инициализация веб-драйвера (Chrome).
    *   Переход на страницу Facebook.
    *   Инициализация объекта `FacebookPromoter`, передавая веб-драйвер и пути к файлам с группами.  `no_video = True` указывает, что не будут загружаться видео.
*   **Цикл публикаций:**
    *   Бесконечный цикл (`while True`),  позволяющий запускать рекламные кампании до завершения работы скрипта.
    *   Метод `promoter.run_campaigns()` запускает выбранные рекламные кампании, используя информацию из переданных файлов с данными о группах.
    *   **Обработка прерывания:**  Блок `try...except KeyboardInterrupt` обрабатывает прерывание пользователем (например, нажатием Ctrl+C). При этом печатается информационное сообщение.

**Структура кода:**

*   Использует модули `header`, `copy`, `webdriver`, `FacebookPromoter`, `logger`.
*   Определяет переменные `MODE` и `filenames`.
*   Определяет список `campaigns` с именами кампаний.


**Технические детали:**

*   **`MODE = 'debug'`:**  Вероятно, используется для настройки уровня логирования (в зависимости от реализации `logger`).
*   **`group_file_paths`:** Список путей к файлам с группами.  Этот список, по-видимому, содержит имена файлов, содержащие информацию о целевых группах, необходимую для публикации.  Формат данных в этих файлах должен быть определён в документации класса `FacebookPromoter`.
*   **`no_video = True`:** Подавляет загрузку видео при работе с Facebook.
