# Модуль defaults.py

## Обзор

Модуль `defaults.py` определяет значения по умолчанию для заголовков HTTP-запросов, используемых в проекте `hypotez`. Он также включает поддержку сжатия Brotli, если библиотека `brotli` установлена. Модуль предназначен для централизованного хранения и управления стандартными значениями заголовков, что упрощает их изменение и поддержку в различных частях проекта.

## Подробней

Этот модуль содержит два набора заголовков: `DEFAULT_HEADERS` и `WEBVIEW_HAEDERS`. `DEFAULT_HEADERS` используются для обычных HTTP-запросов, а `WEBVIEW_HAEDERS` - для запросов, выполняемых из веб-интерфейса. Наличие поддержки Brotli определяется при импорте модуля и влияет на значение заголовка `accept-encoding`.

## Функции

### `has_brotli`

Эта переменная определяет, доступна ли библиотека `brotli`. Она устанавливается в `True`, если импорт `brotli` успешен, и в `False` в противном случае.

**Как работает переменная**:

1. Попытка импортировать библиотеку `brotli`.
2. Если импорт успешен, переменная `has_brotli` устанавливается в `True`.
3. Если импорт не удается (возникает исключение `ImportError`), переменная `has_brotli` устанавливается в `False`.

### `DEFAULT_HEADERS`

Словарь, содержащий заголовки HTTP-запроса по умолчанию.

**Атрибуты**:

-   `"accept"` (str): Типы контента, которые клиент принимает ("*/*" - принимает все типы).
-   `"accept-encoding"` (str): Методы сжатия, которые клиент поддерживает. Включает "gzip, deflate" и, если `has_brotli` равен `True`, добавляет ", br" (Brotli).
-   `"accept-language"` (str): Предпочитаемый язык клиента ("en-US").
-   `"referer"` (str): URL страницы, с которой был сделан запрос (пустая строка).
-   `"sec-ch-ua"` (str): Информация о браузере клиента.
-   `"sec-ch-ua-mobile"` (str): Указывает, является ли клиент мобильным устройством ("?0" - нет).
-   `"sec-ch-ua-platform"` (str): Платформа клиента ("Windows").
-   `"sec-fetch-dest"` (str): Назначение запроса ("empty").
-   `"sec-fetch-mode"` (str): Режим запроса ("cors").
-   `"sec-fetch-site"` (str): Сайт, с которого был сделан запрос ("same-origin").
-   `"user-agent"` (str): Строка user-agent клиента.

### `WEBVIEW_HAEDERS`

Словарь, содержащий заголовки HTTP-запроса по умолчанию для веб-интерфейса.

**Атрибуты**:

-   `"Accept"` (str): Типы контента, которые клиент принимает ("*/*" - принимает все типы).
-   `"Accept-Encoding"` (str): Методы сжатия, которые клиент поддерживает ("gzip, deflate, br").
-   `"Accept-Language"` (str): Предпочитаемый язык клиента (пустая строка).
-   `"Referer"` (str): URL страницы, с которой был сделан запрос (пустая строка).
-   `"Sec-Fetch-Dest"` (str): Назначение запроса ("empty").
-   `"Sec-Fetch-Mode"` (str): Режим запроса ("cors").
-   `"Sec-Fetch-Site"` (str): Сайт, с которого был сделан запрос ("same-origin").
-   `"User-Agent"` (str): Строка user-agent клиента (пустая строка).
