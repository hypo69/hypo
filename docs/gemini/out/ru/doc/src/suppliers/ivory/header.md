# Модуль hypotez/src/suppliers/ivory/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`. Он также предоставляет доступ к информации о проекте, такой как имя, версия, описание и авторы.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная от текущей директории.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для определения корневой директории проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена, иначе директорию, где расположен скрипт.


**Вызывает исключения**:

- Нет


### Загрузка настроек

**Описание**: Загружает настройки из файла `settings.json` в корневой директории проекта.

**Параметры**:

- Нет

**Возвращает**:

- Нет


**Вызывает исключения**:

- `FileNotFoundError`: Если файл `settings.json` не найден.
- `json.JSONDecodeError`: Если содержимое файла `settings.json` не является валидным JSON.


### Загрузка документации

**Описание**: Загружает документацию из файла `README.MD` в корневой директории проекта.

**Параметры**:

- Нет

**Возвращает**:

- Нет


**Вызывает исключения**:

- `FileNotFoundError`: Если файл `README.MD` не найден.
- `json.JSONDecodeError`: Если содержимое файла `README.MD` не является валидным JSON.

## Глобальные переменные

### `__root__`

**Описание**: Путь к корневой директории проекта.

**Тип**: `Path`

**Описание**: Путь к корневой директории проекта, полученный из функции `set_project_root()`.

### `settings`

**Описание**: Словарь со значениями настроек из файла `settings.json`.

**Тип**: `dict`

**Описание**: Словарь настроек или `None` если файл `settings.json` не найден или невалидный.


### `doc_str`

**Описание**: Строка, содержащая содержимое файла `README.MD`.

**Тип**: `str`

**Описание**: Строка с содержимым файла `README.MD`, или `None` если файл не найден или поврежден.



### `__project_name__`

**Описание**: Название проекта.

**Тип**: `str`

**Описание**: Название проекта, полученное из настроек или по умолчанию `'hypotez'`.


### `__version__`

**Описание**: Версия проекта.

**Тип**: `str`

**Описание**: Версия проекта, полученная из настроек или по умолчанию пустая строка.


### `__doc__`

**Описание**: Описание проекта.

**Тип**: `str`

**Описание**: Описание проекта, полученное из файла `README.MD` или пустая строка.

### `__details__`

**Описание**: Дополнительные детали проекта.

**Тип**: `str`

**Описание**: Дополнительные детали проекта (пустая строка по умолчанию).

### `__author__`

**Описание**: Автор проекта.

**Тип**: `str`

**Описание**: Автор проекта, полученный из настроек или пустая строка по умолчанию.

### `__copyright__`

**Описание**: Авторские права проекта.

**Тип**: `str`

**Описание**: Авторские права проекта, полученные из настроек или пустая строка по умолчанию.

### `__cofee__`

**Описание**: Ссылка на поддержку разработчика через кофе.

**Тип**: `str`

**Описание**: Ссылка на поддержку разработчика через кофе или строка по умолчанию.
