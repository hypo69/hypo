# Задача No 83
## Путь в матрице

## Условия

В файле [matrix.txt](https://projecteuler.net/project/resources/p083_matrix.txt) находится текстовый файл, содержащий матрицу 80 на 80.

Каждый элемент матрицы представляет собой целое неотрицательное число.
Ваша задача — найти путь от верхнего левого элемента матрицы к нижнему правому,
который минимизирует сумму элементов на пути.
Вы можете перемещаться только в четырех направлениях: вверх, вниз, влево и вправо.

Например, рассмотрим следующую матрицу 5 на 5:

```
131	673	234	103	18
201	96	342	965	150
630	803	746	422	111
537	699	497	121	956
805	732	524	37	331
```

Оптимальный путь, минимизирующий сумму, выделен жирным шрифтом.

**131** → 201 → 630 → 537 → 805 → 732 → 524 → 37 → **331**

Таким образом, минимальная сумма для этого примера равна 131 + 201 + 630 + 537 + 805 + 732 + 524 + 37 + 331 = 3828.

**Найдите минимальную сумму, двигаясь от верхнего левого угла до правого нижнего в матрице 80 на 80, содержащейся в файле matrix.txt.**
