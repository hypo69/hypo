# Модуль для работы с помощником по коду

## Обзор

Этот модуль предоставляет класс `CodeAssistant` для работы с различными моделями ИИ, такими как Google Gemini и OpenAI, для выполнения задач обработки кода.  Модуль предназначен для анализа и генерации документации по коду.

## Примеры использования

```python
# Пример использования класса CodeAssistant
assistant = CodeAssistant(role='code_checker', lang='ru', model=['gemini'])
assistant.process_files()
```


## Классы

### `CodeAssistant`

**Описание**: Класс для взаимодействия с моделями ИИ для обработки кода.

**Атрибуты**:

- `role` (str): Роль помощника (например, 'code_checker').
- `lang` (str): Язык, на котором будет работать помощник (например, 'ru').
- `model` (list): Список используемых моделей ИИ (например, ['gemini']).

**Методы**:

- `process_files()`: Метод для обработки файлов кода.


## Методы

### `process_files`

**Описание**: Метод обрабатывает список файлов, выполняя анализ и генерацию документации.

**Параметры**:

- `files` (list): Список путей к файлам для обработки.
- `options` (dict, optional): Дополнительные параметры для обработки файлов. По умолчанию пусто.


**Возвращает**:

- list: Список результатов обработки каждого файла.


**Возможные исключения**:

- `FileNotFoundError`: Возникает, если файл не найден.
- `TypeError`: Возникает, если введен некорректный тип данных.
- `Exception`: Возникает при других непредвиденных ошибках.


```python
# Пример использования
assistant = CodeAssistant(role='code_checker', lang='ru', model=['gemini'])
results = assistant.process_files(files=['file1.py', 'file2.py'], options={'verbose': True})
```

**Примечания**:


- Подробное описание параметров и возвращаемых значений для каждого метода будет добавлено в дальнейшем.
- Дополнительные методы и функциональность могут быть добавлены в будущем.
- Модуль может быть расширен для работы с другими языками программирования и моделями ИИ.
