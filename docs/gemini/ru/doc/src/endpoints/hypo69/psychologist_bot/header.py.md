# Модуль `header.py`

## Обзор

Модуль `header.py` содержит основные настройки и метаданные проекта, а также функции для определения корневой директории проекта.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла, поднимаясь вверх и останавливаясь на первой директории, содержащей любой из маркерных файлов.

**Параметры**:
- `marker_files` (tuple, optional): Список имен файлов или директорий для идентификации корневой директории проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена, иначе - директория, в которой находится скрипт.

## Переменные

### `MODE`

**Описание**: Режим работы приложения (например, `dev`). По умолчанию установлен в значение `'dev'`.

### `__root__`

**Описание**: Абсолютный путь к корневой директории проекта.

### `settings`

**Описание**: Словарь с настройками проекта, загруженными из файла `settings.json`. Может быть `None`, если загрузка не удалась.

### `doc_str`

**Описание**: Строка, содержащая содержимое файла `README.MD`. Может быть `None`, если файл не найден или возникла ошибка.

### `__project_name__`

**Описание**: Название проекта. По умолчанию `'hypotez'`.

### `__version__`

**Описание**: Версия проекта. По умолчанию `''`.

### `__doc__`

**Описание**: Документация проекта, взятая из файла `README.MD`.

### `__details__`

**Описание**: Детальная информация о проекте. В данный момент пустая строка.

### `__author__`

**Описание**: Автор проекта. По умолчанию `''`.

### `__copyright__`

**Описание**: Информация об авторских правах. По умолчанию `''`.

### `__cofee__`

**Описание**: Строка с предложением поддержать разработчика. По умолчанию "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69".
