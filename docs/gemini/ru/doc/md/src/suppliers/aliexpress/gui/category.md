# Модуль `hypotez/src/suppliers/aliexpress/gui/category.py`

## Обзор

Этот модуль предоставляет интерфейс для подготовки рекламных кампаний AliExpress. Он использует PyQt6 для создания графического интерфейса пользователя (GUI) и асинхронные операции для оптимизации производительности.  Модуль позволяет загружать JSON-файлы с настройками кампании, обрабатывать информацию о категориях и асинхронно подготавливать их для дальнейшей обработки.

## Классы

### `CategoryEditor`

**Описание**: Класс `CategoryEditor` представляет собой окно приложения для редактирования и подготовки категорий рекламных кампаний AliExpress.

**Атрибуты**:

* `campaign_name`: Название кампании (строка).
* `data`: Данные кампании (объект `SimpleNamespace`).
* `language`: Язык (строка).
* `currency`: Валюта (строка).
* `file_path`: Путь к файлу кампании (строка).
* `editor`: Объект `AliCampaignEditor` для выполнения асинхронной подготовки категорий.

**Методы**:

#### `__init__(self, parent=None, main_app=None)`

**Описание**: Инициализирует главное окно.

**Параметры**:

* `parent` (необязательно): Родительское окно.
* `main_app` (необязательно): Экземпляр главного приложения.

#### `setup_ui(self)`

**Описание**: Настраивает пользовательский интерфейс.

#### `setup_connections(self)`

**Описание**: Настраивает соединения сигналов и слотов.

#### `open_file(self)`

**Описание**: Открывает диалог выбора файла для загрузки JSON-файла.

#### `load_file(self, campaign_file)`

**Описание**: Загружает JSON-файл и инициализирует данные.

**Параметры**:

* `campaign_file` (строка): Путь к JSON-файлу.

**Возвращает**:

* `None`.

**Обрабатывает исключения**:

* `Исключение`: Обрабатывает возможные ошибки при загрузке JSON-файла и отображает сообщение об ошибке пользователю.

#### `create_widgets(self, data)`

**Описание**: Создает виджеты на основе загруженных данных из JSON-файла.

**Параметры**:

* `data` (объект `SimpleNamespace`): Данные кампании.

**Обрабатывает исключения**:

* `Исключение`: Ошибки при создании виджетов.

#### `prepare_all_categories_async(self)`

**Описание**: Асинхронная подготовка всех категорий.

**Возвращает**:

* `None`.

**Обрабатывает исключения**:

* `Исключение`: Обрабатывает возможные ошибки при асинхронной подготовке категорий.

#### `prepare_category_async(self)`

**Описание**: Асинхронная подготовка конкретной категории.

**Параметры**:

* `campaign_name` (строка): Название кампании.

**Возвращает**:

* `None`.

**Обрабатывает исключения**:

* `Исключение`: Обрабатывает возможные ошибки при асинхронной подготовке категории.
