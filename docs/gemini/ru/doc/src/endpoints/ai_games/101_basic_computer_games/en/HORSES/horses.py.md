# HORSES

## Обзор

Этот модуль реализует простую игру "Скачки", в которой игрок выбирает лошадь, а компьютер случайным образом определяет победителя. Игрок выигрывает, если угадал лошадь-победителя.

## Импорт

```python
import random
```
Модуль `random` используется для генерации случайного номера лошади-победителя.

## Логика игры

Игра работает следующим образом:

1.  **Генерация номера лошади-победителя:**
    -   Случайное целое число от 1 до 7 генерируется и сохраняется в переменной `winningHorse`.
2.  **Запрос ввода номера лошади от пользователя:**
    -   Пользователю предлагается ввести номер лошади (от 1 до 7).
    -   Происходит проверка корректности ввода: если пользователь ввел не целое число или число не в диапазоне 1-7, то программа завершается с сообщением об ошибке.
3.  **Проверка результата и вывод сообщения:**
    -   Если номер лошади, выбранный пользователем, совпадает с номером лошади-победителя, то выводится сообщение о победе.
    -   В противном случае выводится сообщение о проигрыше.

### Переменные

-   `winningHorse` (int): Случайный номер лошади-победителя (от 1 до 7).
-   `userHorse` (int): Номер лошади, выбранный пользователем (от 1 до 7).

## Пример работы
```python
import random

# Генерируем случайное число от 1 до 7 (номер лошади-победителя)
winningHorse = random.randint(1, 7)

# Запрашиваем у пользователя ввод числа (номер лошади, за которую он играет)
try:
    userHorse = int(input("Выберите номер лошади от 1 до 7: "))
    if userHorse < 1 or userHorse > 7:
      print("Пожалуйста, выберите номер лошади от 1 до 7.")
      exit()
except ValueError:
    print("Пожалуйста, введите целое число от 1 до 7.")
    exit()


# Проверяем, выиграл ли пользователь
if userHorse == winningHorse:
    print("УРА! ВАША ЛОШАДЬ ВЫИГРАЛА!") # Вывод сообщения о победе
else:
    print("К СОЖАЛЕНИЮ, ВЫ ПРОИГРАЛИ!") # Вывод сообщения о проигрыше
```
