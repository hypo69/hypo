# Документация для `hypotez/src/endpoints/advertisement/facebook/locators/switch_account.json`

## Обзор

Данный файл содержит JSON-объект, определяющий локатор для кнопки переключения аккаунта на Facebook. Локатор используется для автоматизации действий на странице Facebook.

## Оглавление

- [Описание локатора `switch_to_account_button`](#описание-локатора-switch_to_account_button)

## Описание локатора `switch_to_account_button`

### `switch_to_account_button`

**Описание**: Объект JSON, содержащий информацию о локаторе кнопки переключения аккаунта.

**Атрибуты**:

-   `attribute`: `null` - атрибут не задан.
-   `by`: `"XPATH"` - способ поиска элемента на странице (используется XPATH).
-   `selector`: `"//div[@aria-label = 'Переключить']"` - XPATH-выражение для поиска элемента.
-   `if_list`: `"first"` - если найдено несколько элементов, использовать первый.
-   `use_mouse`: `false` - не использовать мышь для действий (используется клик).
-   `timeout`: `0` - таймаут ожидания элемента равен 0.
-   `timeout_for_event`: `"presence_of_element_located"` - событие ожидания - наличие элемента.
-   `event`: `"click()"` - событие, которое необходимо выполнить (клик).
-   `mandatory`: `true` - элемент обязателен для выполнения действия.
-  `locator_description`: `"Переключение на аккаунт, если надо. Если есть кнопка - нажимаю"` - текстовое описание локатора.
