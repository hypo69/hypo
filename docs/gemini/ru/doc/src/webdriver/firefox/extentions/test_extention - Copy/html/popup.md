# Файл hypotez/src/webdriver/firefox/extentions/test_extention - Copy/html/popup.html

## Обзор

Данный HTML-файл представляет собой простую веб-страницу с заголовком "Hypotez" и параграфом с текстом.  Файл предназначен для отображения на странице расширения Firefox.

## Содержание

### HTML-структура

Файл содержит стандартный HTML-шаблон, включающий в себя:

*   `<head>`: Тег `head` с мета-данными (charset, viewport).
*   `<body>`: Тег `body` с содержимым страницы, в т.ч.:
    *   `<h1>`: Заголовок первого уровня "Hypotez".
    *   `<p>`: Параграф с текстом "Привет, Это Давидка. Я обучаю модель".

## Конфигурация

### Постоянные значения

`MODE`: Строковое значение `'debug'`.
