# Модуль Forefront.py
## Обзор

Модуль предоставляет функциональность для взаимодействия с провайдером Forefront, используя модель gpt-3.5-turbo. Он содержит функцию для создания запросов к API Forefront и получения ответов в потоковом режиме. Модуль предназначен для использования в проекте Hypotez.

## Подробней

Модуль взаимодействует с API Forefront для генерации текста на основе предоставленных сообщений. Он использует потоковую передачу данных для получения ответов в режиме реального времени. Модуль требует наличия библиотеки `requests` для выполнения HTTP-запросов.

## Функции

### `_create_completion`

```python
def _create_completion(model: str, messages: list, stream: bool, **kwargs):
    """
    Создает запрос к API Forefront и возвращает ответ в потоковом режиме.

    Args:
        model (str): Название используемой модели.
        messages (list): Список сообщений для отправки в API.
        stream (bool): Флаг, указывающий, нужно ли использовать потоковый режим.
        **kwargs: Дополнительные аргументы.

    Yields:
        str: Часть ответа, полученная в потоковом режиме.

    Как работает функция:
    1. Формирует JSON-данные для отправки в API Forefront, включая текст сообщения, параметры идентификации и настройки модели.
    2. Отправляет POST-запрос к API Forefront с использованием потоковой передачи данных.
    3. Итерирует по строкам ответа, извлекая полезные данные (токены) из каждой строки.
    4. Преобразует полученные токены из JSON-формата и декодирует их.
    5. Возвращает токены в потоковом режиме.

    Внутри функции происходят следующие действия и преобразования:
    Создание JSON-данных: Формируются данные для отправки в API Forefront.
    |
    Отправка POST-запроса: Запрос отправляется с параметром stream=True для потоковой передачи.
    |
    Обработка ответа: Итерация по строкам ответа и извлечение токенов.
    |
    Декодирование токенов: Преобразование JSON-формата и декодирование.
    |
    Генерация токенов: Возврат токенов в потоковом режиме.

    Примеры:
        _create_completion(model='gpt-4', messages=[{'content': 'Hello'}], stream=True)
        _create_completion(model='gpt-4', messages=[{'content': 'Translate to Russian: Hello'}], stream=True)
    """
```
