# Модуль определения моделей OpenAI

## Обзор

Модуль содержит определения моделей, используемых в OpenAI, включая текстовые и графические модели.

## Подробнее

В этом модуле задаются списки моделей, используемых для различных целей: генерации текста и обработки изображений. Это позволяет централизованно управлять доступными моделями и использовать их в других частях проекта.

## Переменные

### `default_model`

**Описание**: Модель, используемая по умолчанию для текстовых задач.

**Значение по умолчанию**: `"auto"`

### `default_image_model`

**Описание**: Модель, используемая по умолчанию для задач обработки изображений.

**Значение по умолчанию**: `"dall-e-3"`

### `image_models`

**Описание**: Список доступных моделей для работы с изображениями.

**Значение по умолчанию**: `[default_image_model]`

### `text_models`

**Описание**: Список доступных моделей для обработки текста.

**Значение по умолчанию**: `[default_model, "gpt-4", "gpt-4.5", "gpt-4o", "gpt-4o-mini", "o1", "o1-preview", "o1-mini", "o3-mini", "o3-mini-high"]`

### `vision_models`

**Описание**: Список моделей, поддерживающих обработку визуальной информации. В данном случае, он совпадает со списком текстовых моделей.

**Значение по умолчанию**: `text_models`

### `models`

**Описание**: Объединенный список всех доступных моделей (текстовых и графических).

**Значение по умолчанию**: `text_models + image_models`
