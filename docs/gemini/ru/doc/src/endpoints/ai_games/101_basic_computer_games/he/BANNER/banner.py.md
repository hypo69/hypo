# Bagels Game Documentation

## Обзор

Этот модуль реализует игру "Бейглс", где игрок пытается угадать секретное трехзначное число с уникальными цифрами. Компьютер предоставляет подсказки в виде "Fermi" (цифра верна и на правильном месте), "Pico" (цифра верна, но не на своем месте), и "Bagels" (нет совпадений).
