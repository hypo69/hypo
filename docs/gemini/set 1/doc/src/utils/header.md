# Модуль hypotez/src/logger/header.py

## Обзор

Данный модуль `header.py` отвечает за определение корневого пути к проекту `hypotez`.  Он также загружает настройки из файла `settings.json` и информацию из файла `README.MD`.  Все импорты в проекте строятся относительно этого корневого пути.


## Функции

### `set_project_root`

**Описание**:  Находит корневую директорию проекта, начиная от текущего файла. Поиск происходит вверх по дереву директорий, пока не будет найдена директория, содержащая указанные маркеры файлов.

**Параметры**:

- `marker_files` (tuple): Кортеж имён файлов или директорий, по которым определяется корень проекта. По умолчанию используются `pyproject.toml`, `requirements.txt`, `.git`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена, иначе - директория, где расположен текущий скрипт.


**Примеры использования**:

```python
root_path = set_project_root()
print(root_path)
```


**Вызывает исключения**:

-  Возможных исключений не описано.


###  Переменные


**Описание**:  Переменные, которые представляют корневой путь проекта, настройки, и документацию проекта.

**Описание переменных**:

- `__root__`: Путь к корневой директории проекта, полученный функцией `set_project_root()`.
- `settings`: Словарь с настройками проекта, загруженными из `settings.json`.
- `doc_str`: Текст из файла `README.MD`.
- `__project_name__`: Название проекта, полученное из настроек.
- `__version__`: Версия проекта, полученная из настроек.
- `__doc__`: Документация проекта, полученная из `README.MD`.
- `__details__`: Подробности проекта (пустая строка по умолчанию).
- `__author__`: Автор проекта, полученный из настроек.
- `__copyright__`: Авторские права, полученные из настроек.
- `__cofee__`: Ссылка на поддержку разработчика, полученная из настроек.




**Обработка ошибок**:


- Файлы `settings.json` и `README.MD` обрабатываются в блоках `try...except`, чтобы предотвратить ошибки при отсутствии файлов или некорректном формате JSON.  В случае ошибок, соответствующие переменные не инициализируются.
