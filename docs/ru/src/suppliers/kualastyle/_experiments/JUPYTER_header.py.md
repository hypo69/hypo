# Модуль _experiments

## Обзор

Модуль `_experiments` предназначен для экспериментальных задач, связанных с поставщиком Kualastyle. В нем содержатся различные импорты и функции, необходимые для работы с данными, веб-драйверами и другими компонентами системы.

## Подробней

Этот модуль, расположенный в директории `src/suppliers/kualastyle/_experiments`, играет роль экспериментальной площадки для новых идей и подходов, которые впоследствии могут быть интегрированы в основные модули проекта. Он включает в себя импорты библиотек, таких как `sys`, `os`, `pathlib`, `json`, `re`, а также пользовательских модулей, таких как `Driver`, `Supplier`, `Product`, `Category` и другие. Модуль предназначен для проведения экспериментов с различными компонентами системы, такими как взаимодействие с веб-драйверами, обработка данных о продуктах и категориях, а также интеграция с PrestaShop.

## Функции

### `start_supplier`

```python
def start_supplier(supplier_prefix: str = 'aliexpress', locale: str = 'en' ) -> Supplier:
    """ Старт поставщика """
    params: dict = \
    {
        'supplier_prefix': supplier_prefix,
        'locale': locale
    }
    
    return Supplier(**params)
```

**Описание**: Функция `start_supplier` предназначена для инициализации и запуска поставщика с заданными параметрами.

**Как работает функция**:
Функция принимает два параметра: `supplier_prefix` и `locale`, которые определяют префикс поставщика и локаль соответственно. Затем она создает словарь `params`, содержащий эти параметры, и использует этот словарь для инициализации объекта `Supplier`. В конечном итоге функция возвращает созданный объект `Supplier`.

**Параметры**:
- `supplier_prefix` (str, optional): Префикс поставщика. По умолчанию `'aliexpress'`.
- `locale` (str, optional): Локаль поставщика. По умолчанию `'en'`.

**Возвращает**:
- `Supplier`: Объект поставщика, инициализированный с заданными параметрами.

**Примеры**:
```python
supplier = start_supplier(supplier_prefix='my_supplier', locale='fr')
# supplier будет экземпляром класса Supplier, инициализированным с supplier_prefix='my_supplier' и locale='fr'
```
```python
supplier = start_supplier()
# supplier будет экземпляром класса Supplier, инициализированным с supplier_prefix='aliexpress' и locale='en'
```
