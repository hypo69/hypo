# Модуль hypotez/src/product/product_fields/header.py

## Обзор

Этот модуль содержит функции для определения корневого каталога проекта и загрузки настроек проекта. Он использует `gs` для работы с путями и загружает настройки из `settings.json` и `README.MD` файлов, находящихся в корне проекта.

## Функции

### `set_project_root`

**Описание**:  Находит корневой каталог проекта, начиная с каталога текущего файла, ищет вверх по дереву каталогов, пока не найдет каталог, содержащий один из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или каталогов, используемых для определения корневого каталога проекта. По умолчанию это ('pyproject.toml', 'requirements.txt', '.git').

**Возвращает**:
- `Path`: Путь к корневому каталогу проекта, если найден, иначе - каталог, где находится скрипт.

**Вызывает исключения**:
- Нет.


### Загрузка настроек проекта

Этот раздел описывает код, который загружает настройки проекта из `settings.json` и `README.MD`, обрабатывая возможные ошибки:

**Описание**: Загружает настройки проекта из файла `settings.json` и документацию из `README.MD`. Если файлы не найдены или содержат невалидные данные, используется значение по умолчанию.

**Обработка исключений**:
- `FileNotFoundError`: Возникает, если файл `settings.json` или `README.MD` не найден.
- `json.JSONDecodeError`: Возникает, если файл `settings.json` содержит некорректный JSON.

В этих случаях соответствующие переменные инициализируются значениями по умолчанию.

**Переменные**:
- `__root__` (Path): Путь к корневому каталогу проекта.
- `settings` (dict): Словарь с настройками проекта.
- `doc_str` (str): Текст документации из файла `README.MD`.


**Переменные проекта (глобальные):**
- `__project_name__` (str): Название проекта. Значение по умолчанию: 'hypotez'.
- `__version__` (str): Версия проекта. Значение по умолчанию: пустая строка.
- `__doc__` (str): Текст документации. Значение по умолчанию: пустая строка.
- `__details__` (str): Подробная информация о проекте. Значение по умолчанию: пустая строка.
- `__author__` (str): Автор проекта. Значение по умолчанию: пустая строка.
- `__copyright__` (str): Авторские права. Значение по умолчанию: пустая строка.
- `__cofee__` (str): Ссылка для поддержки разработчика. Значение по умолчанию: ссылка на Boosty.
