# Задача No 749
## Минимальная сумма чисел Фибоначчи

## Условия
Любое натуральное число можно представить в виде суммы различных чисел Фибоначчи, причем используется минимальное возможное количество чисел Фибоначчи. Например:
* 13 = 13
* 14 = 13 + 1
* 15 = 13 + 2
* 16 = 13 + 3
* 17 = 13 + 3 + 1
* 18 = 13 + 5
* 19 = 13 + 5 + 1
* 20 = 13 + 5 + 2
* 21 = 21

Пусть F(n) – количество чисел Фибоначчи в минимальном представлении числа n в виде суммы чисел Фибоначчи. Из примера выше: F(13) = 1, F(14) = 2, F(15) = 2, F(16) = 2, F(17) = 3, F(18) = 2, F(19) = 3, F(20) = 3 и F(21) = 1.

Пусть S(n) = ∑ F(k) для 2 ≤ k ≤ n.

Известно, что S(10) = 19.

Найдите S(10^18) mod 1234567891.

## Пример
S(10) = 19
