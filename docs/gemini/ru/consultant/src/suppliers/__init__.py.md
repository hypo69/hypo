Модуль поставщиков
========================

Этот модуль содержит базовый класс :class:`Supplier` и связанные с ним классы для
получения данных о товарах от различных поставщиков.

Основные компоненты:
    - :class:`Supplier`: Базовый класс для всех поставщиков.
    - :class:`Graber`: Класс для сбора данных.
    - :class:`Context`: Класс для контекста запроса.
    - `close_pop_up`: Функция для закрытия всплывающих окон.

Модуль предоставляет интерфейс для подключения специфических методов извлечения
информации для каждого поставщика. Эти методы находятся в директориях с
именем `<supplier_prefix>`, например: `amazon`, `aliexpress`, `morlevi`.
