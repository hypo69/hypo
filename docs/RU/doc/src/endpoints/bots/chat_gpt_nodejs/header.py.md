# Модуль `header.py`

## Обзор

Модуль `header.py` содержит базовые настройки и метаданные проекта, включая определение корневой директории, загрузку настроек из файла `settings.json`, чтение документации из `README.MD`, а также установку основных переменных, таких как название проекта, версия, автор и т.д.

## Функции

### `set_project_root`

**Описание**:
Находит корневую директорию проекта, начиная с директории текущего файла, и останавливается на первой директории, содержащей один из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple, optional): Кортеж имен файлов или директорий, которые идентифицируют корневую директорию проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена, в противном случае - путь к директории, где находится скрипт.

## Глобальные переменные

### `__root__`

**Описание**:
Путь к корневой директории проекта. Инициализируется с помощью функции `set_project_root`.

### `settings`

**Описание**:
Словарь, содержащий настройки проекта, загруженные из файла `settings.json`. Если файл не найден или содержит ошибку JSON, значение остается `None`.

### `doc_str`

**Описание**:
Строка, содержащая документацию проекта, прочитанную из файла `README.MD`. Если файл не найден или содержит ошибку, значение остается `None`.

### `__project_name__`

**Описание**:
Название проекта. По умолчанию `'hypotez'`. Если настройка `"project_name"` есть в `settings`, используется ее значение.

### `__version__`

**Описание**:
Версия проекта. По умолчанию `''`. Если настройка `"version"` есть в `settings`, используется ее значение.

### `__doc__`

**Описание**:
Строка, содержащая документацию проекта. Значение берется из `doc_str`. Если `doc_str`  `None`, остается `''`.

### `__details__`

**Описание**:
Детали проекта. По умолчанию `''`.

### `__author__`

**Описание**:
Автор проекта. По умолчанию `''`. Если настройка `"author"` есть в `settings`, используется ее значение.

### `__copyright__`

**Описание**:
Информация об авторских правах проекта. По умолчанию `''`. Если настройка `"copyrihgnt"` есть в `settings`, используется ее значение.

### `__cofee__`

**Описание**:
Сообщение с призывом угостить разработчика чашкой кофе. По умолчанию `"Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69"`. Если настройка `"cofee"` есть в `settings`, используется ее значение.
