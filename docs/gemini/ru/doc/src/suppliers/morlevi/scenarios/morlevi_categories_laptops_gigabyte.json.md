# Документация для `morlevi_categories_laptops_gigabyte.json`

## Обзор

Этот JSON-файл содержит конфигурацию сценариев для сопоставления моделей ноутбуков GIGABYTE с категориями в PrestaShop. Каждый сценарий определяет соответствие конкретной модели ноутбука (например, "GIGABYTE 15 I5") определенным категориям в PrestaShop, а также включает информацию о бренде, URL-адресе, статусе активности и состоянии товара.

## Структура файла
Файл содержит один основной объект JSON с ключом `scenarios`, который содержит в себе объекты, описывающие каждый сценарий. Ключом каждого сценария является строка, представляющая модель ноутбука.

```json
{
  "scenarios": {
    "Название сценария 1": {
      "brand": "Бренд",
      "url": "URL",
      "checkbox": true/false,
      "active": true/false,
      "condition": "new",
      "presta_categories": {
        "template": {
          "gigabyte": [
            "Категория 1",
            "Категория 2"
          ]
        }
      }
    },
    "Название сценария 2": {
      "brand": "Бренд",
      "url": "URL",
       "checkbox": true/false,
      "active": true/false,
      "condition": "new",
      "presta_categories": {
        "template": {
          "gigabyte": [
             "Категория 1",
            "Категория 2"
          ]
        }
      }
    },
    ...
  }
}
```

## Описание полей

### `scenarios`
**Описание**:  Объект, содержащий все сценарии для сопоставления моделей ноутбуков GIGABYTE с категориями PrestaShop. Ключи объекта являются названиями сценариев, например, "GIGABYTE 15 I5".

### `brand`
**Описание**: Строка, представляющая бренд ноутбука.
**Тип**: `str`
**Пример**: `"GIGABYTE"`

### `url`
**Описание**:  Строка, содержащая URL-адрес, связанный с данным сценарием. Может быть `null`.
**Тип**: `str | null`
**Пример**: `"https://www.example.com/notebook"` или `null`

### `checkbox`
**Описание**:  Логическое значение, указывающее, активен ли сценарий.
**Тип**: `bool`
**Пример**: `true` или `false`

### `active`
**Описание**:  Логическое значение, указывающее, активен ли сценарий.
**Тип**: `bool`
**Пример**: `true` или `false`

### `condition`
**Описание**:  Строка, представляющая состояние товара.
**Тип**: `str`
**Пример**: `"new"`

### `presta_categories`
**Описание**:  Объект, содержащий категории PrestaShop, к которым относится товар.
**Тип**: `object`

### `template`
**Описание**:  Объект, содержащий шаблон для категорий.
**Тип**: `object`

### `gigabyte`
**Описание**: Массив строк, представляющий категории PrestaShop, к которым следует отнести товар.
**Тип**: `list[str]`
**Пример**: `["LAPTOPS INTEL I5", "15"]`
