# Модуль для парсинга данных о сделках из XLS-файлов, сгенерированных в личном кабинете AliExpress

## Обзор

Модуль предназначен для извлечения информации о сделках из XLS-файлов, экспортированных из личного кабинета AliExpress. Он использует класс `DealsFromXLS` для парсинга данных и предоставляет возможность итерироваться по каждой сделке.

## Подробней

Этот модуль позволяет автоматизировать процесс получения данных о сделках, что особенно полезно для анализа эффективности рекламных кампаний и оптимизации стратегии продаж на AliExpress. Он облегчает задачу извлечения структурированной информации из табличных данных, представленных в формате XLS.
