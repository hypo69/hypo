# Модуль `hypotez/src/endpoints/advertisement/facebook/scenarios/__init__.py`

## Обзор

Данный модуль предоставляет функции и классы для управления рекламными сценариями на Facebook. Он импортирует функции из отдельных файлов сценариев, предоставляя единый интерфейс для работы с различными действиями, такими как вход в систему, публикация сообщений, переключение учетных записей и публикация рекламных объявлений.
