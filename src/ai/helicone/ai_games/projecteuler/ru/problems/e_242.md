# Задача No 242
## Нечетные элементы в подмножествах

## Условия

Дано множество *S* = {1, 2, 3, ..., *n*}.

Рассмотрим подмножества *S*, которые обладают следующим свойством:

Для каждого подмножества, сумма его элементов является нечетным числом.

Пусть *F*(*n*) - количество таких подмножеств.

Например, *F*(3) = 4. Множества, которые этому удовлетворяют: {1}, {1,2,3}, {2}, {3}.

Найдите *F*(10<sup>7</sup>) mod 10<sup>8</sup>.
