# Модуль `hypotez/src/goog/spreadsheet/bberyakov/__init__.py`

## Обзор

Данный модуль предоставляет инструменты для работы с Google Spreadsheets. Он импортирует классы `GSpreadsheet`, `GWorksheet`, и `GSRenderr` из соответствующих подмодулей, а также определяет константу `MODE`, вероятно, для выбора режима работы.


## Константы

### `MODE`

**Описание**:  Переменная, определяющая режим работы (например, `dev`, `prod`). Значение по умолчанию: `'dev'`.


## Модули

### `gspreadsheet`

**Описание**: Модуль, содержащий класс `GSpreadsheet`, вероятно, для работы с Google Spreadsheets в целом.


### `gworksheets`

**Описание**: Модуль, содержащий класс `GWorksheet`, вероятно, для работы с отдельными листами Google Spreadsheets.


### `grender`

**Описание**: Модуль, содержащий класс `GSRenderr`, вероятно, для обработки и рендеринга данных Google Spreadsheets.
