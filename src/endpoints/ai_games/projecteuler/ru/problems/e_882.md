# Задача No 882
## Простые числа и их цифры

## Условия

Определим `f(n)` как сумму цифр натурального числа `n`. Например, `f(123) = 1 + 2 + 3 = 6`.

Определим `g(n)` как сумму цифр числа, представляющего собой произведение цифр числа `n`. Например, `g(123) = f(1 * 2 * 3) = f(6) = 6`.

Определим `h(n)` как наименьшее простое число `p`, такое что `f(p) = f(n)`.

Для данного натурального числа `k` определим `h(k, i)` как *i-е* наименьшее простое число `p`, такое что `f(p) = f(k)`. Например, `h(10, 1) = 19`, а `h(10, 2) = 43`.

Далее определим `S(k, n) = Σ g(h(k, i))` для всех `1 ≤ i ≤ n`.

Например, `S(10, 1) = g(19) = f(1*9) = 9`.

`S(10, 5) = g(19) + g(43) + g(67) + g(89) + g(251) = f(9) + f(12) + f(42) + f(72) + f(10) = 9 + 3 + 6 + 9 + 1 = 28`.

Также `S(123, 10) = 71`.

Вычислите `S(2024, 5000)`.
