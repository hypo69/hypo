# Модуль hypotez/src/endpoints/kazarinov/scenarios/header.py

## Обзор

Этот модуль содержит функции для определения корневого каталога проекта и загрузки настроек из файла `settings.json` и документации из файла `README.MD`. Он использует переменные, чтобы получить информацию о проекте.

## Функции

### `set_project_root`

**Описание**: Определяет корневой каталог проекта, начиная с текущего каталога и идя вверх по иерархии директорий.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, которые указывают на корневой каталог проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Объект `Path` представляющий корневой каталог проекта, если он найден. В противном случае возвращает директорию, в которой находится скрипт.

**Вызывает исключения**:

- Нет.


### Функции, использующие `try...except`

**Обработка ошибок**:

Следующие функции используют блок `try...except` для обработки потенциальных ошибок при работе с файлами и JSON:


**Функция 1:** `set_project_root` - не вызывается в блоке `try...except`.


**Функция 2:** Загрузка настроек из `settings.json`

```python
try:
    with open(gs.path.root / 'src' / 'settings.json', 'r') as settings_file:
        settings = json.load(settings_file)
ex
except (FileNotFoundError, json.JSONDecodeError):
    ...
```
- Эта функция пытается загрузить данные из файла `settings.json`, используя модуль `json`.
- Если файл не найден или данные в нем некорректны, выполняется блок `except`, и `settings` будет иметь значение `None`.


**Функция 3:** Загрузка документации из `README.MD`

```python
try:
    with open(gs.path.root / 'src' / 'README.MD', 'r') as settings_file:
        doc_str = settings_file.read()
ex
except (FileNotFoundError, json.JSONDecodeError):
    ...
```

- Эта функция пытается загрузить содержимое файла `README.MD` в переменную `doc_str`.
- Если файл не найден или данные в нем некорректны, выполняется блок `except`, и `doc_str` будет иметь значение `None`.




## Переменные

### `__root__`

**Описание**: Объект `Path`, содержащий путь к корневому каталогу проекта. Инициализируется функцией `set_project_root()`.

### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженный из файла `settings.json`. Может быть `None`, если файл не найден или поврежден.

### `doc_str`

**Описание**: Строка, содержащая содержимое файла `README.MD`. Может быть `None`, если файл не найден или поврежден.


### `__project_name__`, `__version__`, `__doc__`, `__details__`, `__author__`, `__copyright__`, `__cofee__`

**Описание**: Строковые переменные, содержащие значения, полученные из словаря `settings` (или значения по умолчанию, если `settings` равен `None`)

- `__project_name__`: Название проекта.
- `__version__`: Версия проекта.
- `__doc__`: Документация проекта (из `README.MD`).
- `__details__`: Подробная информация о проекте.
- `__author__`: Автор проекта.
- `__copyright__`: Авторские права.
- `__cofee__`: Ссылка на пожертвования автору проекта.
