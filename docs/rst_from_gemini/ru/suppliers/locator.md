# Файл: hypotez/src/suppliers/locator.py

Расположение: `C:\Users\user\Documents\repos\hypotez\src\suppliers\locator.py`

**Описание:**

Модуль `locator.py` отвечает за загрузку и предоставление доступа к параметрам расположения (`locators`) для поставщиков.  Он позволяет получить конфигурацию для поиска нужных данных, хранящихся в структурированном формате JSON.

**Класс `Locator`:**

Представляет локетор с различными атрибутами выбора.

**Атрибуты:**

* `supplier_prefix` (строка): Префикс, идентифицирующий папку поставщика, где хранятся файлы конфигурации локетора.
* `locator` (объект `SimpleNamespace`): Объект, содержащий определения локетора, загруженные из файла JSON. Обеспечивает удобный доступ к атрибутам локетора.

**Методы:**

* `__init__(self, supplier_prefix: str)`:
    * Инициализирует экземпляр `Locator` загрузкой настроек из файла JSON на основе `supplier_prefix`.
    * **Аргументы:**
        * `supplier_prefix` (строка): Префикс, идентифицирующий папку поставщика.
    * **Исключения:**
        * `FileNotFoundError`: Если файл JSON с определениями локетора не найден.
        * `ValueError`: Если содержимое JSON некорректное.

* `_load_locator(self) -> SimpleNamespace`:
    * Загружает настройки локетора из файла JSON, основанного на `supplier_prefix`.
    * **Возвращаемое значение:**
        * Объект `SimpleNamespace`, содержащий загруженные настройки.
    * **Исключения:**
        * `FileNotFoundError`: Если файл JSON с определениями локетора не найден.
        * `ValueError`: Если содержимое JSON некорректное.


**Использование:**

Для использования класса `Locator`, необходимо создать экземпляр, передав ему `supplier_prefix`:

```python
from hypotez.src.suppliers.locator import Locator

try:
    locator = Locator("supplier_name")
    # Теперь можно использовать атрибуты из locator.locator, например:
    print(locator.locator.search_path) 
except (FileNotFoundError, ValueError) as e:
    print(f"Ошибка при загрузке локетора: {e}")
```

**Комментарии:**

* Используется `gs.path.src` для получения пути к папке `src`. Это предполагает, что `gs` (вероятно, `global_settings`) определен где-то в `__init__.py` и содержит информацию о путях.
* Использование `SimpleNamespace` улучшает читаемость кода, делая доступ к атрибутам локетора более интуитивно понятным.
* Обработка исключений `FileNotFoundError` и `ValueError` позволяет программе корректно обрабатывать ошибки, возникающие при работе с файлами JSON.
