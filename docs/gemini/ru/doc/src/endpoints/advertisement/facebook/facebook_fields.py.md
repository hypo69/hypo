# Модуль для работы с полями Facebook

## Обзор

Модуль `facebook_fields.py` предназначен для управления полями, используемыми в объявлениях и событиях Facebook. Он содержит класс `FacebookFields`, который загружает и предоставляет доступ к этим полям из JSON-файла.

## Подробней

Этот модуль облегчает настройку и использование различных полей, необходимых для создания и управления рекламными кампаниями в Facebook. Он использует JSON для хранения данных о полях, что обеспечивает гибкость и удобство в настройке.

## Классы

### `FacebookFields`

**Описание**: Класс для работы с полями объявлений и событий Facebook.

**Как работает класс**:
1.  При инициализации класса вызывается метод `_payload`, который загружает данные о полях из JSON-файла (`facebook_feilds.json`).
2.  Если загрузка данных прошла успешно, каждое поле из JSON-файла добавляется как атрибут класса.
3.  Это позволяет удобно получать доступ к полям через экземпляр класса.

**Методы**:

*   `__init__(self)`: Инициализирует класс и вызывает метод `_payload` для загрузки данных.
*   `_payload(self)`: Загружает данные о полях из JSON-файла и устанавливает их как атрибуты класса.

    **Как работает метод**:
    1.  Формируется путь к файлу `facebook_feilds.json` с использованием `gs.path.src`.
    2.  Вызывается функция `j_loads` для загрузки данных из JSON-файла.
    3.  Если данные не загружены, в лог записывается сообщение об ошибке.
    4.  Для каждого элемента в загруженных данных устанавливается атрибут класса с соответствующим именем и значением.
    5.  Возвращает `True` в случае успешной загрузки данных.

**Пример**:

```python
from src.endpoints.advertisement.facebook.facebook_fields import FacebookFields

fb_fields = FacebookFields()
# Теперь можно обращаться к полям как к атрибутам экземпляра класса
# Например: fb_fields.some_field
```
