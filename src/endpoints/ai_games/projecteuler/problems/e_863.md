# Задача No 863
## Минимальный размер матрицы, содержащей все пути

## Условия

Представьте, что у вас есть квадратная матрица, где каждый элемент является целым числом от 1 до n включительно. Из каждого элемента можно перемещаться в соседний элемент по горизонтали или вертикали (диагональные перемещения запрещены). Путь — это последовательность элементов, полученная в результате таких перемещений. Путь не может проходить через один и тот же элемент дважды. Путь может быть любой длины (больше или равной 1), и в нем могут быть элементы, которые не являются соседними.

Известно, что матрица размера n x n содержит `n^2` элементов. Матрица, содержащая все возможные пути, должна содержать все пути, которые могут быть найдены в матрице размера n x n. Эта матрица не обязательно должна быть квадратной.  Пусть m(n) — минимальный размер матрицы, которая содержит все пути, которые могут быть найдены в матрице размера n x n.

Например, матрица 2x2 содержит 16 возможных путей, показанных ниже:
```
1; 2; 3; 4
1-2; 2-1; 1-3; 3-1; 2-4; 4-2; 3-4; 4-3
1-2-4; 4-2-1; 1-3-4; 4-3-1
1-2-4-3; 3-4-2-1
```

Эти 16 путей могут быть найдены в матрице 4x3:
```
1 2 4
3 1 2
4 3 1
2 4 3
```

Таким образом, m(2) = 12, т.е. размер матрицы (4x3) = 12. 

Известно, что m(3) = 48.
Найдите m(100).
