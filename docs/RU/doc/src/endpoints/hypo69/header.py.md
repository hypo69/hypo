## Функции

### `set_project_root`

**Описание**:
Находит корневую директорию проекта, начиная с директории текущего файла, поднимаясь вверх и останавливаясь в первой директории, содержащей один из маркерных файлов.

**Параметры**:
- `marker_files` (tuple, optional): Кортеж имен файлов или каталогов для идентификации корня проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена. В противном случае возвращает директорию, где расположен скрипт.

```python
def set_project_root(marker_files: tuple = ('__root__', '.git')) -> Path:
    """
    Finds the root directory of the project starting from the current file's directory,
    searching upwards and stopping at the first directory containing any of the marker files.

    Args:
        marker_files (tuple): Filenames or directory names to identify the project root.
    
    Returns:
        Path: Path to the root directory if found, otherwise the directory where the script is located.
    """
```

## Переменные
### `__root__`
**Описание**: Путь к корневой директории проекта. Инициализируется функцией `set_project_root`.

### `settings`
**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json`.
Значение `None`, если файл не найден или произошла ошибка при разборе JSON.

### `doc_str`
**Описание**: Строка с содержимым файла `README.MD`.
Значение `None`, если файл не найден или произошла ошибка при чтении файла.

### `__project_name__`
**Описание**: Имя проекта, полученное из настроек (`settings.get("project_name")`) или значение по умолчанию `hypotez`, если настройки не загружены.

### `__version__`
**Описание**: Версия проекта, полученная из настроек (`settings.get("version")`) или пустая строка, если настройки не загружены.

### `__doc__`
**Описание**: Строка с документацией проекта, полученная из `doc_str`. Пустая строка, если `doc_str` не задан.

### `__details__`
**Описание**: Строка с дополнительными деталями проекта. По умолчанию - пустая строка.

### `__author__`
**Описание**: Автор проекта, полученный из настроек (`settings.get("author")`) или пустая строка, если настройки не загружены.

### `__copyright__`
**Описание**: Информация об авторских правах проекта, полученная из настроек (`settings.get("copyrihgnt")`) или пустая строка, если настройки не загружены.

### `__cofee__`
**Описание**: Сообщение с предложением угостить разработчика кофе, полученное из настроек (`settings.get("cofee")`) или сообщение по умолчанию со ссылкой на Boosty, если настройки не загружены.
