# Документация для модуля `xai`

## Обзор

Этот модуль предоставляет Python-клиент для взаимодействия с API xAI. Он упрощает процесс отправки запросов к API xAI, поддерживая как стандартные, так и потоковые запросы.
Модуль обеспечивает удобный интерфейс для аутентификации, генерации ответов (chat completion) и потоковой передачи ответов от моделей xAI.

## Подробней

Модуль `xai` предназначен для облегчения интеграции с API xAI, предоставляя инструменты для аутентификации запросов, генерации ответов и потоковой передачи данных.
Он позволяет разработчикам легко взаимодействовать с мощными моделями xAI, используя Python.

## Функции

В данном коде функции не представлены, так как предоставленный текст является инструкцией по использованию API клиента xAI.
Примеры кода демонстрируют, как инициализировать клиент `XAI`, выполнять запросы на завершение чата и потоковую передачу данных.

### `chat_completion`

**Назначение**: Отправляет запрос на завершение чата к API xAI и возвращает ответ.

**Параметры**:

- `messages` (list): Список сообщений для отправки в API. Каждое сообщение представляет собой словарь с ключами `role` и `content`.

**Возвращает**:

- `str`: Ответ от API xAI.

**Как работает функция**:

1. Формирует запрос к API xAI с использованием предоставленных сообщений.
2. Отправляет запрос и получает ответ.
3. Возвращает полученный ответ в виде строки.

**Примеры**:

```python
import json
from xai import XAI

api_key = "your_api_key_here"  # Замените на ваш реальный ключ API
xai = XAI(api_key)

messages = [
    {
        "role": "system",
        "content": "You are Grok, a chatbot inspired by the Hitchhikers Guide to the Galaxy."
    },
    {
        "role": "user",
        "content": "What is the answer to life and universe?"
    }
]

completion_response = xai.chat_completion(messages)
print("Non-streaming response:", completion_response)
```

### `stream_chat_completion`

**Назначение**: Отправляет запрос на потоковое завершение чата к API xAI и возвращает генератор, который предоставляет ответы по частям.

**Параметры**:

- `messages` (list): Список сообщений для отправки в API. Каждое сообщение представляет собой словарь с ключами `role` и `content`.

**Возвращает**:

- `generator`: Генератор, который возвращает строки из потокового ответа API xAI.

**Как работает функция**:

1. Формирует запрос на потоковую передачу данных к API xAI с использованием предоставленных сообщений.
2. Отправляет запрос и получает потоковый ответ.
3. Возвращает генератор, который предоставляет каждую строку из потока.

**Примеры**:

```python
import json
from xai import XAI

api_key = "your_api_key_here"  # Замените на ваш реальный ключ API
xai = XAI(api_key)

messages = [
    {
        "role": "system",
        "content": "You are Grok, a chatbot inspired by the Hitchhikers Guide to the Galaxy."
    },
    {
        "role": "user",
        "content": "What is the answer to life and universe?"
    }
]

stream_response = xai.stream_chat_completion(messages)
print("Streaming response:")
for line in stream_response:
    if line.strip():
        print(json.loads(line))
```
