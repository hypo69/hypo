# Модуль hypotez/src/suppliers/chat_gpt

## Обзор

Данный модуль предоставляет интерфейс для взаимодействия с поставщиком ChatGPT.  Он содержит вспомогательные классы и функции для работы с сервисом, например, для подключения к Google Sheet.

## Функции

### `MODE`

**Описание**: Переменная, хранящая режим работы (в данном случае 'dev').  Значение используется для настройки поведения модуля.


**Значение**: `'dev'`


### `GptGs`

**Описание**: Класс для взаимодействия с Google Sheet, связанный с ChatGPT.

**Описание**:  Подразумевает, что этот класс находится в подмодуле `gsheet`, а его детали описаны в соответствующем файле документации.
