# Модуль hypotez/src/webdriver/playwright/pw_executor.py

## Обзор

Модуль `pw_executor.py` содержит код для управления сессиями браузера Playwright.  Он определяет константу `MODE` для выбора режима работы (например, 'dev' для разработки, 'prod' для производства).

## Переменные

### `MODE`

**Описание**:  Константа, определяющая режим работы. На данный момент установлено значение `'dev'`.  Может быть изменено для переключения на другой режим (например, 'prod').

**Тип**: str

**Значение**: 'dev'
