# Документация для файла hypotez/src/fast_api/html/openai/index.html

## Обзор

Этот файл представляет собой страницу HTML с интерфейсом для взаимодействия с моделью OpenAI.  Он использует AngularJS для обработки ввода-вывода и AJAX-запросы к API для отправки сообщений и обучения модели.  Страница предоставляет поля для ввода сообщений, системных инструкций и обучающих данных, а также отображает ответы модели и ID задачи обучения.

## Взаимодействие с моделью

Данный раздел описывает функционал страницы для взаимодействия с моделью OpenAI.

### askModel()

**Описание**: Функция отправляет запрос к API для получения ответа от модели.

**Параметры**:

- `message` (str): Текстовое сообщение для модели.
- `system_instruction` (str, optional): Системные инструкции для модели (необязательно).


**Возвращает**:
 - (dict | None):  Ответ от сервера, содержащий поле `response` с результатом работы модели. Возвращает `None`, если произошла ошибка.

**Обрабатываемые исключения**:
- `HTTPError`: Ошибка при выполнении запроса. Подробнее об ошибке можно увидеть в поле `detail` ответа от сервера.


## Обучение модели

Этот раздел описывает функцию обучения модели OpenAI.

### trainModel()

**Описание**: Функция отправляет данные для обучения модели OpenAI на сервер.

**Параметры**:

- `data` (str): Строка с обучающими данными в формате CSV.
- `positive` (bool): Флаг, указывающий на тип обучения (положительный).


**Возвращает**:
 - (dict | None):  Ответ от сервера, содержащий поле `job_id` с ID задачи обучения. Возвращает `None`, если произошла ошибка.


**Обрабатываемые исключения**:
- `HTTPError`: Ошибка при выполнении запроса. Подробнее об ошибке можно увидеть в поле `detail` ответа от сервера.
