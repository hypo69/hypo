# Файл `hypotez/src/suppliers/aliexpress/gui/main.py`

## Описание

Этот файл содержит код главного окна приложения, управляющего рекламными кампаниями. Приложение использует PyQt6 для создания графического интерфейса с вкладками для редактирования JSON, кампаний и продуктов.  Включает в себя функциональность для открытия, сохранения файлов JSON, копирования и вставки текста, а также редактирования данных в соответствующих редакторах.

## Класс `MainApp`

* **`__init__`**:
    * Инициализирует главное окно приложения с вкладками.
    * Устанавливает заголовок окна и размеры.
    * Создает `QTabWidget` для организации вкладок.
    * Создает и добавляет вкладки для редактирования JSON, кампаний и продуктов, используя соответствующие классы (`CampaignEditor`, `ProductEditor`, `CategoryEditor`).  Ключевая информация: каждый редактор связан с отдельным виджетом `QWidget` на вкладке.
    * Создает меню `menubar` с пунктами `File` и `Edit`.
    * Функции `open_file`, `save_file`, `exit_application`, `copy`, `paste` встроены в меню.
    * `open_product_action` используется для открытия файла продукта, настроен для использования с редактором `ProductEditor`.

* **`create_menubar`**:
    * Создаёт основное меню приложения.
    * Добавляет пункты меню для `File` и `Edit`, с привязкой соответствующих обработчиков.

* **`open_file`**:
    * Открывает диалоговое окно для выбора файла JSON.
    * Загружает выбранный файл в редактор JSON, используя `load_file`.

* **`save_file`**:
    * Сохраняет текущие изменения в редактируемом файле. Логика сохранения различается в зависимости от активной вкладки: JSON-редактор, редактор продуктов.

* **`exit_application`**:
    * Закрывает приложение.

* **`copy`**:
    * Копирует выделенный текст в буфер обмена. Обработка ошибок отсутствия фокусированного текстового виджета.

* **`paste`**:
    * Вставляет текст из буфера обмена. Аналогично `copy` - обработка ошибки отсутствия фокуса на виджете.

* **`load_file`**:
    * Загружает JSON файл в соответствующий редактор (в данном случае, `promotion_app`). Обработка исключений при загрузке.

## Функция `main`

* Инициализирует `QApplication` и создаёт экземпляр `MainApp`.
* Создаёт `QEventLoop` для асинхронных операций.
* Запускает приложение (`main_app.show()`).
* Запускает цикл обработки событий (`loop.run_forever()`) для корректной работы Qt.

## Требования/Комментарии

* Код предполагает наличие классов `CampaignEditor`, `ProductEditor`, `CategoryEditor`, `AliCampaignEditor`, `ProductEditor`.
* `src.utils.j_loads_ns`, `src.utils.j_dumps` – предполагаются как вспомогательные функции для работы с JSON.
* Необходимо добавить классы `CampaignEditor`, `ProductEditor`, `CategoryEditor`, `AliCampaignEditor`, которые должны содержать методы для загрузки и сохранения данных, а также обработку событий.
* Возможно, стоит использовать `QLineEdit`, `QTextEdit`, или `QPlainTextEdit` для виджетов редактирования,  в зависимости от потребностей.
* Загрузка файла выполняется только в редактор JSON (`promotion_app`) в методе `open_file`.
* Необходимо улучшить функциональность сохранения, чтобы обрабатывать и другие вкладки (`ProductEditor`).


## Поддержка комментариев в коде

Код содержит много хороших комментариев, объясняющих назначение функций и переменных.  Однако, для повышения качества документации рекомендуется добавить пояснения к более сложным фрагментам кода, например, к обработке исключений при загрузке файлов или к взаимодействию с различными редакторами.
