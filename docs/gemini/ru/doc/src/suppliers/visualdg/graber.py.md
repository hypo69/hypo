# Модуль `src.suppliers.visualdg.graber`

## Обзор

Модуль `src.suppliers.visualdg.graber` представляет класс `Graber`, который предназначен для сбора значений полей со страницы товара на сайте `visualdg.co.il`.
Этот класс расширяет функциональность базового класса `Graber`, предоставляя специфические методы обработки полей, если это необходимо.
Перед отправкой запроса к веб-драйверу, класс может выполнять предварительные действия через декоратор. По умолчанию декоратор находится в родительском классе.

## Функции

### `close_pop_up`
   
**Описание**: Создает декоратор для закрытия всплывающих окон перед выполнением основной логики функции.

**Параметры**:
- `value` (Any): Дополнительное значение для декоратора.

**Возвращает**:
- `Callable`: Декоратор, оборачивающий функцию.
