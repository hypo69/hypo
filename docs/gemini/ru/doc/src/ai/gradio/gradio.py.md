# Модуль `gradio.py`

## Обзор

Модуль `gradio.py` предоставляет простой интерфейс для создания и запуска Gradio-приложения. Он содержит функцию `greet`, которая принимает имя и возвращает приветствие, а также запускает Gradio-интерфейс для этой функции.

## Функции

### `greet`

**Описание**: Функция, принимающая имя и возвращающая приветствие.

**Параметры**:
- `name` (str): Имя, которое нужно поприветствовать.

**Возвращает**:
- `str`: Приветствие в формате "Hello \[имя]!".

```python
def greet(name):
    return "Hello " + name + "!"
```
