# Модуль hypotez/src/ai/dialogflow/header.py

## Обзор

Данный модуль отвечает за определение корневого пути к проекту `hypotez`.  Все импорты в проекте строятся относительно этого пути. В дальнейшем планируется перенести эту логику в системные переменные.  Модуль также загружает настройки из файла `settings.json` и документацию из `README.MD`, если они существуют.

## Функции

### `set_project_root`

**Описание**:  Функция находит корневой каталог проекта, начиная с текущего каталога и поднимаясь вверх по иерархии каталогов. Поиск останавливается при обнаружении директории, содержащей один из указанных файлов-маркеров (pyproject.toml, requirements.txt, .git). Если корневой каталог не найден, возвращает директорию текущего файла.  В случае успешного нахождения корневого каталога, добавляет его в системный путь Python (`sys.path`).

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, используемых для определения корневого каталога.  По умолчанию используется кортеж `('pyproject.toml', 'requirements.txt', '.git')`.


**Возвращает**:

- `Path`: Путь к корневому каталогу проекта.


**Вызывает исключения**:

-  Не вызывает исключений.


## Переменные

### `__root__`

**Описание**: Переменная, хранящая корневой путь к проекту. Инициализируется вызовом функции `set_project_root()`.


### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженные из файла `src/settings.json`. Если файл не найден или содержит некорректные данные, значение устанавливается в `None`.


### `doc_str`

**Описание**: Строка, содержащая текст документации, загруженный из файла `src/README.MD`. Если файл не найден или содержит некорректные данные, значение устанавливается в `None`.


### `__project_name__`

**Описание**: Строка, содержащая имя проекта.  Получается из `settings['project_name']` или по умолчанию `'hypotez'`.


### `__version__`

**Описание**: Строка, содержащая версию проекта. Получается из `settings['version']` или по умолчанию `''`.


### `__doc__`

**Описание**: Строка, содержащая описание проекта. Получается из `doc_str` или по умолчанию `''`.


### `__details__`

**Описание**: Строка, содержащая дополнительные детали о проекте. По умолчанию пустая строка.


### `__author__`

**Описание**: Строка, содержащая имя автора проекта.  Получается из `settings['author']` или по умолчанию `''`.


### `__copyright__`

**Описание**: Строка, содержащая копирайт проекта. Получается из `settings['copyright']` или по умолчанию `''`.


### `__cofee__`

**Описание**: Строка, содержащая ссылку на страницу сбора пожертвований на кофе для разработчика.  Получается из `settings['cofee']` или по умолчанию ссылка на  `boosty.to/hypo69`.
