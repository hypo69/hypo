* **`get_project_root(marker_files=('pyproject.toml', 'requirements.txt', '.git'))`**:

    * **Аргументы:**
        * `marker_files`: Кортеж имен файлов или директорий, по которым определяется корень проекта.
    * **Возвращаемое значение:**
        * Объект `Path` к корневой директории. Возвращает директорию текущего файла, если корень не найден.
