# Игра "Пицца"

## Обзор

Этот модуль представляет собой простую игру, в которой игрок должен определить, сколько пицц нужно заказать, исходя из количества людей и размера пиццы. Программа вычисляет необходимое количество пицц и предоставляет пользователю возможность выбора между маленьким, средним и большим размерами пиццы. В конце выдается рекомендация о количестве пицц в соответствии с выбранным размером.

## Функции

### `calculate_pizza_amount`

**Описание**:
Эта функция запрашивает у пользователя количество человек и размер пиццы (маленькая, средняя или большая), после чего рассчитывает и выводит рекомендуемое количество пицц.

**Параметры**:
- Нет параметров.

**Возвращает**:
- `None`: Функция ничего не возвращает, а выводит результат в консоль.

**Вызывает исключения**:
- `ValueError`: Возникает, если введенное значение количества людей не является целым числом.

```python
def calculate_pizza_amount():
    """
    Args:
        None
    Returns:
        None
    Raises:
        ValueError: Если введенное значение количества людей не является целым числом.
    """
    # קבלת מספר האנשים מהמשתמש
    try:
        num_people = int(input("הכנס את מספר האנשים: "))
        if num_people <= 0:
             print("מספר האנשים חייב להיות חיובי.")
             return
    except ValueError as ex:
         print("קלט לא תקין. אנא הכנס מספר שלם.")
         return

    # קבלת גודל הפיצה מהמשתמש
    pizza_size = input("הכנס את גודל הפיצה (S, M, L): ").upper()

    # בדיקה איזה גודל פיצה נבחר וחישוב מספר הפיצות
    if pizza_size == "S":
        # פיצה קטנה - פיצה אחת לכל 3 אנשים
        num_pizzas = (num_people + 2) // 3
        print(f"מומלץ להזמין {num_pizzas} פיצות קטנות.")
    elif pizza_size == "M":
        # פיצה בינונית - פיצה אחת לכל 2 אנשים
        num_pizzas = (num_people + 1) // 2
        print(f"מומלץ להזמין {num_pizzas} פיצות בינוניות.")
    elif pizza_size == "L":
        # פיצה גדולה - פיצה אחת לכל אדם
        num_pizzas = num_people
        print(f"מומלץ להזמין {num_pizzas} פיצות גדולות.")
    else:
        print("גודל פיצה לא תקין. אנא הכנס S, M או L.")

```
