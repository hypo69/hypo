# Модуль show_all_results.js

## Обзор

Этот модуль отвечает за отображение результатов поиска, полученных из расширения `tryxpath`. Он обрабатывает данные результатов, заполняет соответствующие HTML-элементы и предоставляет возможности экспорта данных в текстовые файлы.

## Функции

### `showAllResults`

**Описание**: Функция отображает результаты поиска, получаемые из расширения. Она заполняет различные поля на странице HTML с результатами поиска, включая информацию о контексте поиска и список найденных элементов.

**Параметры**:

- `results` (объект): Объект, содержащий результаты поиска.

**Возвращает**:

-  Не имеет возвращаемого значения.

**Обрабатываемые исключения**:

- `fu.onError`: Обработчик исключений.

### `makeTextDownloadUrl`

**Описание**: Создает URL для загрузки текстового файла.

**Параметры**:

- `text` (строка): Текст, который будет сохранен в файле.

**Возвращает**:

- `строка`: URL для загрузки файла.

**Обрабатываемые исключения**:

- Не обрабатывает исключения.

### `makeInfoText`

**Описание**: Создает строку, содержащую информацию о результатах поиска в формате, подходящем для экспорта.

**Параметры**:

- `results` (объект): Объект, содержащий результаты поиска.

**Возвращает**:

- `строка`: Строка с информацией о результатах.

**Обрабатываемые исключения**:

- Не обрабатывает исключения.


### `makeConvertedInfoText`

**Описание**: Создает строку с информацией о результатах поиска, где значения "value" и "textContent" представлены в формате JSON.

**Параметры**:

- `results` (объект): Объект, содержащий результаты поиска.

**Возвращает**:

- `строка`: Строка с информацией о результатах.

**Обрабатываемые исключения**:

- Не обрабатывает исключения.

##  Взаимодействие с расширением

Функции `showAllResults` и другие функции модуля взаимодействуют с расширением браузера `tryxpath` для получения результатов.  Они используют API `browser.runtime.sendMessage` для обмена данными.
