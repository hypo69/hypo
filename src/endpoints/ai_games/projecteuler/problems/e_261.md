# Задача No 261
## Последовательности, определенные с помощью деления

## Условия
Рассмотрим последовательность, полученную многократным делением на основе следующих правил:

-   $n_0 = 12345$
-   $n_{i+1} = \left\lfloor \frac{n_i}{d(n_i)} \right\rfloor$, где $d(n_i)$ - количество делителей $n_i$.

Первые десять элементов этой последовательности будут:
12345, 1481, 185, 37, 7, 1, 0, 0, 0, 0, ...

Теперь рассмотрим другую последовательность, которая формируется по следующему правилу:
-   $s_0 = 1$
-   $s_{i+1}$ - сумма различных $n_k$, где $n_k$ - это элемент из первой последовательности (т.е.  $n_i = \left\lfloor \frac{n_{i-1}}{d(n_{i-1})} \right\rfloor$), и $n_k \ge 1$.

Таким образом, первые несколько членов $s_i$:
$s_0 = 1$
$s_1 = 12345 + 1481 + 185 + 37 + 7 + 1 = 14058$

Пусть $t$ будет количеством членов $s_i$ таких, что $s_i$ - палиндром, для $0 \le i \le 10^{15}$.
Найдите последние 9 цифр $t$.
