Campaign Testing Guide
=======================

This document provides guidance for testers verifying the module responsible for preparing advertising materials for AliExpress campaigns.  The module comprises three main files:

- `edit_campaign.py`: Campaign management.
- `prepare_campaigns.py`: Campaign category preparation and processing.
- `test_campaign_integration.py`: Integration tests for the entire module.
