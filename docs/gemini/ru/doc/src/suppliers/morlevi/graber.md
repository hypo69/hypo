# Модуль `hypotez/src/suppliers/morlevi/graber.py`

## Обзор

Модуль `hypotez/src/suppliers/morlevi/graber.py` содержит класс `Graber`, предназначенный для сбора данных с веб-страниц товара на сайте `morlevi.co.il`. Класс наследуется от базового класса `Graber` и предоставляет методы для обработки различных полей страницы товара.  Класс реализует механизм обработки полей, а также предоставляет возможность переопределения стандартной обработки для специфических полей.  В модуле также определены вспомогательные функции и переменные, необходимые для работы класса `Graber`.

## Классы

### `Graber`

**Описание**: Класс `Graber` наследует функциональность от базового класса `Grbr` и специализируется на сборе данных с сайта `morlevi.co.il`.  Он определяет предварительные действия перед выполнением основных операций сбора данных.

**Методы**:

- `__init__(self, driver: Driver)`:
    **Описание**: Инициализирует класс `Graber`.
    **Параметры**:
        - `driver` (Driver): Экземпляр класса `Driver`, представляющий веб-драйвер.
    **Возвращает**:
        -  None


- `local_saved_image(self, value: Any = None)`:
    **Описание**:  Загружает и сохраняет изображение продукта. Сохраняет скриншот изображения, получаемый из веб-драйвера, в временную директорию (`tmp`) и записывает путь к сохранённому изображению в поле `local_saved_image` объекта `ProductFields`.
    **Параметры**:
        - `value` (Any, необязательно): Дополнительное значение, которое можно передать при вызове функции. Если `value` передан, его значение используется в поле `ProductFields.local_saved_image`.
    **Возвращает**:
        - `bool`: `True`, если изображение успешно сохранено, `False` в противном случае.

**Примечания**:
* Метод `local_saved_image` содержит неполную реализацию, и необходимо дополнить обработку возможных ошибок.
* Код содержит закомментированный фрагмент, реализующий декоратор `close_pop_up`.
* Метод `local_saved_image` содержит заглушки для обработки исключений и возврата значений. Необходимо реализовать полную функциональность обработки ошибок и возвращаемых значений.
