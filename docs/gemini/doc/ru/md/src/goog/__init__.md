# Модуль goog

## Обзор

Данный модуль содержит константу `MODE` и импортирует класс `Spreadsheet` из подмодуля `spreadsheet`.

## Константы

### `MODE`

**Описание**:  Строковая константа, определяющая режим работы модуля. В данном случае, режим `'dev'`.
