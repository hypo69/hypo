# Модуль md2dict

## Обзор

Модуль `md2dict` предназначен для конвертации строк Markdown в структурированные словари Python.  Он умеет извлекать JSON-контент, если таковой присутствует в строке Markdown, а также парсить заголовки и текст, структурируя их в словарь с секциями.

## Функции

### `md2dict`

**Описание**: Конвертирует строку Markdown в структурированный словарь.

**Параметры**:

- `md_string` (str): Строка Markdown для конвертации.

**Возвращает**:

- `Dict[str, dict | list]`: Структурированное представление Markdown содержимого. Возвращает словарь с ключом "json", если найден JSON контент, или словарь с секциями Markdown.

**Вызывает исключения**:

- `Exception`: Любые ошибки при парсинге или извлечении JSON.  Подробная информация об ошибке логгируется в `logger`.

### `extract_json_from_string`

**Описание**: Извлекает JSON контент из строки, если он присутствует.

**Параметры**:

- `text` (str): Строка для извлечения JSON контента.

**Возвращает**:

- `dict | None`: Извлеченный JSON контент или `None`, если JSON не найден.

**Вызывает исключения**:

- `Exception`: Любые ошибки при парсинге JSON.  Подробная информация об ошибке логгируется в `logger`.


## Константы

### `MODE`

**Описание**: Текущий режим работы модуля (в данном случае 'dev').


## Модули

### `re`

**Описание**: Модуль для работы с регулярными выражениями. Используется для поиска JSON и заголовков.

### `markdown2`

**Описание**: Библиотека для парсинга Markdown в HTML. Используется для предварительной обработки строки Markdown.

### `src.logger`

**Описание**: Модуль для логирования.  Используется для записи ошибок.

### `typing`

**Описание**: Модуль для определения типов данных.
