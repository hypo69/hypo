# Модуль `src.endpoints.kazarinov.main.py`

## Обзор

Этот модуль является точкой входа для запуска скриптов, предназначенных для автоматизации процесса сбора информации о компьютерных комплектующих с сайтов поставщиков. Он координирует работу различных подмодулей, таких как `minibot`, `scenario` и `quotation_builder`, для создания итогового прайс-листа.

## Подробней

Модуль предназначен для автоматизации процесса сбора данных о комплектующих для компьютеров. Сначала пользователь собирает ссылки на нужные товары и отправляет их боту. Бот, в свою очередь, запускает сценарий сбора данных с этих веб-страниц. Собранные данные затем обрабатываются с помощью `quotation_builder` для формирования конечного прайс-листа.
