# Задача No 752
## Бинарный куб
## Условия
Рассмотрим двоичную строку длиной *n*, которую можно интерпретировать как целое число от 0 до 2<sup>*n*</sup> - 1. Строку можно разбить на несколько подстрок, которые могут быть интерпретированы как целые числа. Пусть *S*(*n*) - это сумма всех целых чисел, соответствующих всем разбиениям двоичной строки длиной *n*. Например, при *n* = 3 мы имеем следующие двоичные строки и все их разбиения:

*   "000" -> 0
*   "001" -> 0 + 1 = 1
*   "010" -> 0 + 10; 01 + 0 = 10 + 1 = 11
*   "011" -> 0 + 1 + 1; 01 + 1 = 2 + 1 = 3
*   "100" -> 1 + 0 + 0; 10 + 0; 1 + 00 = 1 + 10 + 1 = 12
*   "101" -> 1 + 0 + 1; 10 + 1; 1 + 01 = 2 + 11 + 2 = 15
*   "110" -> 1 + 1 + 0; 11 + 0; 1 + 10 = 2 + 3 + 11 = 16
*   "111" -> 1 + 1 + 1; 11 + 1; 1 + 11 = 3 + 12 + 12 = 27

Таким образом, S(3) = 0 + 1 + 11 + 3 + 12 + 15 + 16 + 27 = 85.

Пусть T(*n*) - сумма S(*k*) для всех *k* от 1 до *n*. Таким образом, T(3) = S(1) + S(2) + S(3) = 1 + 5 + 85 = 91.

Вычислите T(10<sup>8</sup>) по модулю 1 000 000 007.
