# Задача No 270
## Тетрамино
## Условия
Тетрамино – это фигура, состоящая из четырех соединенных квадратов. Существует семь различных тетрамино:

![Тетрамино](https://projecteuler.net/images/p270_tetrominoes.gif)

Если две тетрамино можно получить друг из друга с помощью поворота или отражения, они считаются одинаковыми.

Рассмотрим прямоугольную область размером 2 x n. Выложите эту область тетрамино так, чтобы никакие из них не перекрывались и не выходили за пределы области. Пусть T(n) – количество способов выложить прямоугольную область 2 x n с использованием тетрамино.

Например, T(1) = 1, T(2) = 3, T(3) = 11 и T(4) = 41.

![Примеры укладки](https://projecteuler.net/images/p270_examples.gif)

Вычислите T(10^12) mod 1000000007.
