# Категория "מראות" (Зеркала)

## Обзор

Данный JSON-файл содержит информацию о категории товаров "Зеркала".  Он определяет название категории, а также указывает, есть ли у нее подкатегории. В данном случае, подкатегорий нет, и поле `scenarios` в текущей версии пусто.

## Структура JSON

### `category name on site`
 
**Описание**: Название категории товаров на сайте.
 
**Тип**: `str`

**Пример**: `"מראות"` (Зеркала)

### `have subcategories`

**Описание**: Логическое значение, указывающее, есть ли у данной категории подкатегории.

**Тип**: `bool`

**Пример**: `false` (нет подкатегорий)

### `scenarios`

**Описание**: Словарь, предназначенный для хранения сценариев, связанных с категорией. В текущем файле пустой.

**Тип**: `dict`

**Пример**: `{}` (пустой словарь)
