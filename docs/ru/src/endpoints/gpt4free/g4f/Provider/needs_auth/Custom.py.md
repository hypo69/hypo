# Модуль Custom

## Обзор

Модуль `Custom` определяет класс `Custom`, который является подклассом `OpenaiTemplate`. Этот класс предназначен для настройки и использования пользовательского провайдера, совместимого с API OpenAI. Также содержит класс `Feature`, который является наследником класса `Custom`.

## Подробней

Модуль предоставляет базовую конфигурацию для пользовательских провайдеров, работающих с OpenAI API, и определяет параметры, такие как базовый URL API, необходимость аутентификации и другие настройки.

## Классы

### `Custom`

**Описание**: Класс `Custom` представляет собой шаблон для пользовательских провайдеров, совместимых с OpenAI API. Он наследуется от `OpenaiTemplate` и содержит основные настройки для работы с таким провайдером.

**Принцип работы**:
Класс `Custom` инициализирует основные параметры, необходимые для взаимодействия с пользовательским провайдером, совместимым с API OpenAI. Это включает в себя указание базового URL API, флага, указывающего на необходимость аутентификации, и других настроек.

**Аттрибуты**:
- `label` (str): Метка провайдера (в данном случае "Custom Provider").
- `working` (bool): Указывает, работает ли провайдер в данный момент (в данном случае `True`).
- `needs_auth` (bool): Указывает, требуется ли аутентификация для работы с провайдером (в данном случае `False`).
- `api_base` (str): Базовый URL API провайдера (в данном случае "http://localhost:8080/v1").
- `sort_models` (bool): Указывает, нужно ли сортировать модели (в данном случае `False`).

### `Feature`

**Описание**: Класс `Feature` является подклассом класса `Custom`. Он представляет собой провайдера `Feature Provider`
**Принцип работы**:
Класс `Feature` наследует основные параметры от класса `Custom`. Изменяет `label` и `working`

**Аттрибуты**:
- `label` (str): Метка провайдера (в данном случае "Feature Provider").
- `working` (bool): Указывает, работает ли провайдер в данный момент (в данном случае `False`).
- `needs_auth` (bool): Указывает, требуется ли аутентификация для работы с провайдером (в данном случае `False`).
- `api_base` (str): Базовый URL API провайдера (в данном случае "http://localhost:8080/v1").
- `sort_models` (bool): Указывает, нужно ли сортировать модели (в данном случае `False`).
