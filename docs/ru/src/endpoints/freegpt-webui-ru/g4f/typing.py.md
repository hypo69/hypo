# Модуль определения типов данных для g4f

## Обзор

Этот модуль определяет пользовательские типы данных, используемые в проекте g4f. Он включает определение типа `sha256` для хешей SHA256, а также импортирует необходимые модули для работы с типами, такие как Dict, NewType, Union, Optional, List и get_type_hints.

## Подробнее

Модуль предназначен для обеспечения строгой типизации и улучшения читаемости кода. Он использует `NewType` для создания различимого типа `sha256`, что позволяет избежать путаницы со строками общего назначения.

## Типы

### `sha256`

```python
sha256 = NewType('sha_256_hash', str)
```

**Описание**: Тип данных, представляющий собой SHA256 хеш. Это подтип строки, используемый для указания, что переменная должна содержать строку, представляющую SHA256 хеш.

**Принцип работы**:
1. Определяется новый тип `sha256` с использованием `NewType`.
2. `sha256` является подтипом `str`, что означает, что он ведет себя как строка, но позволяет проверять типы и делает код более понятным.

**Примеры**:

```python
from typing import NewType

sha256 = NewType('sha_256_hash', str)

my_hash: sha256 = 'a948904f2f0d479d8f81a1e035833dcb5cb8b286b0f25f30c978a9f3cf182363'
print(my_hash)
