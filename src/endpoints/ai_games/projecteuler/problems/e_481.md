# Задача No 481
## Комбинации блоков

## Условия

Рассмотрим способ расположения блоков размером 1×1×2. Можно составить столбцы высотой *n*, поместив блоки либо горизонтально, либо вертикально.

Например, при *n* = 2, можно сложить блоки тремя способами:
- две горизонтально расположенные единицы высотой 1
- две вертикально расположенные единицы высотой 2
- один блок, высотой 2, положенный горизонтально

Пусть *C(n)* – количество способов сложить блоки, чтобы получить столбец высотой *n*. Из вышесказанного следует, что *C(1)* = 1, *C(2)* = 3 и *C(3)* = 5.

Рассмотрим способ расположения блоков размером 1×1×3. Пусть *F(n)* – количество способов сложить блоки, чтобы получить столбец высотой *n*.
Можно убедиться, что *F(1)* = 1, *F(2)* = 1, *F(3)* = 4, *F(4)* = 6 и *F(5)* = 15.

Сколько способов можно сложить блоки размером 1×1×*k* для получения столбца высотой *n*?
Обозначим количество способов сложить блоки размером 1×1×*k* для получения столбца высотой *n* как *f(n, k)*.
Полагаем *f(n, k) = 0*, если *n < 0*.

Дано, что *f(7, 3) = 17*, *f(10, 3) = 89*, *f(100, 3) mod 1000000007 = 179323944*, *f(1000, 3) mod 1000000007 = 197827017*.

Найдите (*f*(10<sup>16</sup>, 10) mod 1000000007).

## Пример
Примеры из условия:
*   C(1) = 1
*   C(2) = 3
*   C(3) = 5
*   F(1) = 1
*   F(2) = 1
*   F(3) = 4
*   F(4) = 6
*   F(5) = 15
*   f(7, 3) = 17
*   f(10, 3) = 89
*   f(100, 3) mod 1000000007 = 179323944
*   f(1000, 3) mod 1000000007 = 197827017
