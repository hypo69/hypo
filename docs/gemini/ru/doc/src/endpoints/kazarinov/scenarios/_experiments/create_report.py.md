# Модуль для проверки генерации прайслиста

## Обзор

Модуль предназначен для проверки функциональности генерации отчетов о прайс-листах. Он использует класс `ReportGenerator` для создания HTML и PDF отчетов на основе данных о продуктах, полученных от разных поставщиков.

## Подорбней

Этот модуль является частью экспериментов по созданию прайс-листов. Он использует данные, полученные от AI-моделей, для генерации отчетов на разных языках. Основная цель - убедиться, что `ReportGenerator` работает корректно и создает отчеты в ожидаемом формате. Он использует `response_he_dict` и `response_ru_dict` для генерации отчетов на иврите и русском языках соответственно.

## Переменные

- `report_generator`: Экземпляр класса `ReportGenerator`, используемый для создания отчетов.
- `html_file_he`: Путь к HTML файлу для отчета на иврите.
- `pdf_file_he`: Путь к PDF файлу для отчета на иврите.
- `html_file_ru`: Путь к HTML файлу для отчета на русском.
- `pdf_file_ru`: Путь к PDF файлу для отчета на русском.
- `test_directory`:  Каталог, используемый для тестов, содержащий сгенерированные HTML и PDF файлы.
