# Модуль hypotez/src/endpoints/advertisement/header

## Обзор

Этот модуль содержит вспомогательные функции для работы с проектом, включая определение корневой директории проекта, загрузку настроек из файла `settings.json`, и получение информации о проекте из `README.md`.

## Функции

### `get_project_root`

**Описание**: Ищет корневую директорию проекта, начиная с текущей директории файла, ищет вверх по дереву каталогов, пока не найдет директорию, содержащую один из указанных файлов или каталогов. Добавляет найденную корневую директорию в `sys.path`, если она еще там не была.

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или каталогов, по которым определяется корневая директория проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории проекта, если она найдена, иначе путь к текущей директории.


**Пример использования**:
```python
root_path = get_project_root()
print(root_path)
```

### `get_settings`

**Описание**: Загружает настройки из файла `settings.json`.

**Возвращает**:
- `dict | None`: Словарь настроек, если файл успешно загружен и распарсен, иначе `None`.

**Обрабатывает исключения**:
- `FileNotFoundError`: Если файл `settings.json` не найден.
- `json.JSONDecodeError`: Если файл `settings.json` имеет некорректный формат JSON.

**Пример использования**:

```python
settings = get_settings()
if settings:
  print(settings['project_name'])
```

## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный функцией `get_project_root`.

**Тип**: `Path`


### `settings`

**Описание**: Словарь настроек проекта, загруженный из `settings.json`.

**Тип**: `dict | None`


### `doc_str`

**Описание**: Строка с содержимым файла `README.MD`.

**Тип**: `str | None`


### `__project_name__`

**Описание**: Имя проекта, полученное из настроек или имеющее значение по умолчанию `hypotez`.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта, полученная из настроек.

**Тип**: `str`


### `__doc__`

**Описание**: Документация проекта, полученная из `README.MD`.

**Тип**: `str`


### `__details__`

**Описание**: Дополнительная информация о проекте.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта, полученный из настроек.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права на проект, полученный из настроек.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка на страницу для поддержки разработчика (выплата за чашку кофе)

**Тип**: `str`
