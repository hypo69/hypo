# Модуль hypotez/src/suppliers/kualastyle/header.py

## Обзор

Этот модуль предоставляет функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и чтения документации из файла `README.MD`. Он также содержит переменные, хранящие информацию о проекте, например, имя, версию, описание, автора и информацию для поддержки разработчика.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла. Поиск происходит вверх по дереву директорий, пока не будет найдена директория, содержащая указанные файлы-маркеры (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, которые используются для определения корневой директории проекта. По умолчанию заданы значения `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращает директорию, где находится скрипт.

**Вызывает исключения**:
Нет.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root`.

**Тип**: `Path`

### `settings`

**Описание**: Словарь настроек, загруженный из файла `settings.json` в корневой директории проекта.

**Тип**: `dict` или `None`

### `doc_str`

**Описание**: Строка документации, загруженная из файла `README.MD` в корневой директории проекта.

**Тип**: `str` или `None`


### `__project_name__`

**Описание**: Имя проекта, полученное из настроек. По умолчанию `hypotez`.

**Тип**: `str`

### `__version__`

**Описание**: Версия проекта, полученная из настроек. По умолчанию пустая строка.

**Тип**: `str`

### `__doc__`

**Описание**: Описание проекта, полученное из настроек. По умолчанию пустая строка.

**Тип**: `str`

### `__details__`

**Описание**: Дополнительные подробности о проекте. По умолчанию пустая строка.

**Тип**: `str`

### `__author__`

**Описание**: Автор проекта, полученный из настроек. По умолчанию пустая строка.

**Тип**: `str`

### `__copyright__`

**Описание**: Авторские права, полученные из настроек. По умолчанию пустая строка.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка для поддержки разработчика чашкой кофе. По умолчанию ссылка на Boosty.

**Тип**: `str`
