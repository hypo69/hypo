# Модуль hypotez/src/suppliers/cdata/graber.py

## Обзор

Модуль `hypotez/src/suppliers/cdata/graber.py` содержит класс `Graber`, предназначенный для сбора данных о товарах со страницы `cdata.co.il`.  Класс наследуется от базового класса `Graber` и переопределяет методы для обработки специфичных данных с указанного сайта.  В классе реализованы асинхронные методы для извлечения различных полей товара, включая обработку потенциальных всплывающих окон.
