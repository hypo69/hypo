# Модуль hypotez/src/suppliers/cdata/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла settings.json. Он также загружает описание проекта (README.MD) и предоставляет доступ к информации о проекте (имя, версия, автор, права, сообщение с просьбой о поддержке).

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с текущей директории и поднимаясь вверх по дереву директорий, пока не найдет директорию, содержащую файлы-маркеры (pyproject.toml, requirements.txt, .git).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для идентификации корневой директории проекта. По умолчанию содержит ('pyproject.toml', 'requirements.txt', '.git').

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена. В противном случае, возвращает директорию, в которой расположен скрипт.

**Вызывает исключения**:
- Нет.


### `__root__` (переменная)

**Описание**: Переменная, содержащая путь к корневой директории проекта.  Значение присваивается в модуле через вызов функции `set_project_root()`.

**Тип**: `Path`


## Загрузка настроек

**Описание**: Загружает настройки из файла `settings.json` в переменную `settings`.

**Обработка исключений**:

- Обрабатывает `FileNotFoundError` и `json.JSONDecodeError` при ошибках доступа к файлу или ошибках декодирования JSON, устанавливая `settings` в `None` в случае ошибки.

## Загрузка описания проекта (README)

**Описание**: Загружает содержимое файла `README.MD` в переменную `doc_str`.

**Обработка исключений**:

- Обрабатывает `FileNotFoundError` и `json.JSONDecodeError` при ошибках доступа к файлу или ошибках декодирования, устанавливая `doc_str` в `None` в случае ошибки.



## Переменные проекта

**Описание**: Переменные, содержащие информацию о проекте (имя, версия, автор, права, просьба о поддержке). Значения берутся из переменной `settings` или задаются по умолчанию.

**Переменные**:

- `__project_name__` (str): Имя проекта.
- `__version__` (str): Версия проекта.
- `__doc__` (str): Содержимое файла README.MD.
- `__details__` (str): Дополнительные детали о проекте.
- `__author__` (str): Автор проекта.
- `__copyright__` (str): Авторские права.
- `__cofee__` (str): Ссылка для поддержки разработчика.
