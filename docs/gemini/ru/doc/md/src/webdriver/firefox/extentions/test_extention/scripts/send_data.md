# Модуль send_data.js

## Обзор

Этот скрипт JavaScript собирает информацию о загруженной странице (заголовок, URL, содержимое) и отправляет ее на сервер по POST запросу.

## Функции

### `onPageLoad`

**Описание**: Обработчик события загрузки страницы.  Собирает информацию о странице (заголовок, URL, содержимое) и отправляет её на сервер.

**Возвращает**:  Не имеет возвращаемого значения.

**Обрабатывает исключения**:
* `Error`: Вызывается, если серверный запрос не выполнен успешно (`response.ok` равен false).

**Детали**:
* С помощью `document.title`, `window.location.href` и `document.body.innerHTML` собирает заголовок, URL и содержимое загруженной страницы соответственно.
* Формирует объект `data` с собранными данными.
* Испоьзует `fetch` для отправки POST запроса на сервер по адресу `http://127.0.0.1/hypotez.online/api/`.  В запросе используется JSON-формат.
* Обрабатывает возможные ошибки при отправке и получении данных с сервера используя `.catch(error => {})` блок.
* Выводит результат ответа сервера в консоль (`console.log('Response:', json);`).
