# Модуль `hypotez/src/utils/printer.py`

## Обзор

Этот модуль предоставляет функции для красивой печати данных и форматирования текста, включая цвета, фоны и стили шрифта. Он предназначен для улучшения удобочитаемости вывода в консоль.  Модуль поддерживает печать различных типов данных, таких как словари, списки, строки и пути к файлам, а также позволяет задавать цвета и стили для улучшения визуализации.

## Функции

### `_color_text`

**Описание**:  Помощник для применения стилей цвета, фона и шрифта к тексту. Принимает текст и параметры для цвета, фона и стиля шрифта. Возвращает отформатированный текст.

**Параметры**:
- `text` (str): Текст, к которому необходимо применить стили.
- `text_color` (str, optional): Цвет текста. По умолчанию пустая строка (без цвета).  Возможные значения: "red", "green", "blue", "yellow", "white", "cyan", "magenta", "light_gray", "dark_gray", "light_red", "light_green", "light_blue", "light_yellow".
- `bg_color` (str, optional): Цвет фона. По умолчанию пустая строка (без фона). Возможные значения: аналогичны `text_color`.
- `font_style` (str, optional): Стиль шрифта. По умолчанию пустая строка (без стиля). Возможные значения: "bold", "underline".

**Возвращает**:
- `str`: Отформатированный текст с примененными стилями.


### `pprint`

**Описание**:  Красиво печатает данные с опциональными стилями цвета, фона и шрифта.  Обрабатывает различные типы данных (словари, списки, строки и пути к файлам).

**Параметры**:
- `print_data` (Any, optional): Данные для печати. Может быть `None`, `dict`, `list`, `str` или `Path`.
- `text_color` (str, optional): Цвет текста. По умолчанию "white".  См. :ref:`TEXT_COLORS`.
- `bg_color` (str, optional): Цвет фона. По умолчанию пустая строка (нет фона).  См. :ref:`BG_COLORS`.
- `font_style` (str, optional): Стиль шрифта. По умолчанию пустая строка (нет стиля). См. :ref:`FONT_STYLES`.

**Возвращает**:
- `None`

**Возможные исключения**:
- `Exception`:  В случае проблем с типом данных или при ошибках во время печати.

**Примеры использования**:  Примеры использования приведены в документации функции.


## Постоянные переменные

### `MODE`

**Описание**: Текстовая переменная, хранящая текущий режим работы (в данном случае "dev").


##  Словари констант

### `TEXT_COLORS`

**Описание**: Словарь, содержащий ANSI escape коды для различных цветов текста.


### `BG_COLORS`

**Описание**: Словарь, содержащий ANSI escape коды для различных цветов фона.


### `FONT_STYLES`

**Описание**: Словарь, содержащий ANSI escape коды для различных стилей шрифта.


##  Использование модуля

Модуль `printer` предоставляет функции `_color_text` и `pprint` для форматирования вывода в консоли.  `_color_text` - вспомогательная функция, а `pprint` – основная функция для печати данных с различными стилями форматирования.  Обратите внимание на использование методов `get()` для безопасного доступа к значениям в словарях констант.
