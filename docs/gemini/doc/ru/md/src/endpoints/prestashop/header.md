# Модуль hypotez/src/logger/header.py

## Обзор

Модуль `src.logger` определяет корневой путь к проекту `hypotez`.  Все импорты строятся относительно этого пути.  В дальнейшем планируется перенести получение корневого пути в системные переменные.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная от текущей директории файла. Поиск происходит вверх по дереву директорий, останавливаясь на первой директории, содержащей один из указанных маркеров (файлов или папок).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для определения корневой директории проекта. По умолчанию это ('pyproject.toml', 'requirements.txt', '.git').

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена, иначе — путь к директории, где расположен текущий скрипт.

**Вызывает исключения**:

- Нет.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root`.

**Тип**: `Path`

**Значение**: Путь к корневой директории проекта, найденный функцией `set_project_root`.

### `settings`

**Описание**: Словарь с настройками проекта.

**Тип**: `dict`

**Значение**: Содержимое файла `settings.json` в корневой директории проекта. Если файл не найден или содержит невалидные данные, `settings` будет равно `None`.


### `doc_str`

**Описание**: Строковое содержимое файла README.

**Тип**: `str`

**Значение**: Содержимое файла `README.MD` в корневой директории проекта. Если файл не найден или содержимое некорректно, `doc_str` будет равно `None`.



### `__project_name__`

**Описание**: Имя проекта, взятое из настроек `settings`.

**Тип**: `str`

**Значение**: Значение ключа `project_name` в `settings`. По умолчанию `hypotez`.

### `__version__`

**Описание**: Версия проекта, взятая из настроек `settings`.

**Тип**: `str`

**Значение**: Значение ключа `version` в `settings`. По умолчанию пустая строка.

### `__doc__`

**Описание**: Документация к проекту (из файла `README.MD`).

**Тип**: `str`

**Значение**: Содержимое файла `README.MD`. По умолчанию пустая строка.

### `__details__`

**Описание**: Подробное описание проекта (пустая строка по умолчанию).

**Тип**: `str`

**Значение**: Пустая строка по умолчанию.

### `__author__`

**Описание**: Автор проекта, взятый из настроек `settings`.

**Тип**: `str`

**Значение**: Значение ключа `author` в `settings`. По умолчанию пустая строка.

### `__copyright__`

**Описание**: Авторские права проекта, взятые из настроек `settings`.

**Тип**: `str`

**Значение**: Значение ключа `copyrihgnt` в `settings`. По умолчанию пустая строка.


### `__cofee__`

**Описание**: Ссылка для поддержки разработчика (кофе).

**Тип**: `str`

**Значение**: Значение ключа `cofee` в `settings`. По умолчанию ссылка для пожертвования на поддержку разработчика.
