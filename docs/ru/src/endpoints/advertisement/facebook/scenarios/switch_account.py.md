# Модуль `switch_account`

## Обзор

Модуль `switch_account` предназначен для автоматизации переключения между аккаунтами в Facebook с использованием веб-драйвера. Он использует локаторы, определенные в JSON-файле, для поиска и взаимодействия с элементами на веб-странице.

## Подробнее

Этот модуль является частью системы автоматизации для работы с Facebook и предназначен для упрощения процесса переключения между различными аккаунтами. Он загружает локаторы из файла `post_message.json` и использует их для поиска и нажатия на кнопку переключения аккаунта.

## Функции

### `switch_account`

```python
def switch_account(driver: Driver):
    """ Если есть кнопка `Переключить` - нажимаю её  """
```

**Назначение**: Выполняет переключение аккаунта в Facebook, нажимая на кнопку "Переключить", если она доступна.

**Параметры**:
- `driver` (Driver): Инстанс веб-драйвера, используемый для взаимодействия с веб-страницей.

**Возвращает**:
- `None`: Функция ничего не возвращает.

**Вызывает исключения**:
- Отсутствуют явные исключения, но могут возникнуть исключения, связанные с работой веб-драйвера, если элемент не найден или недоступен.

**Как работает функция**:
1. Функция получает инстанс веб-драйвера.
2. Использует метод `execute_locator` для поиска элемента, соответствующего кнопке "Переключить", используя локаторы, загруженные из JSON-файла.
3. Если элемент найден, функция выполняет клик по нему, тем самым переключая аккаунт.

```
    Начало
     ↓
  Наличие кнопки "Переключить"
     ↓
   Нажатие на кнопку
     ↓
    Конец
```

**Примеры**:
```python
from src.webdriver.driver import Driver, Firefox
from src.endpoints.advertisement.facebook.scenarios.switch_account import switch_account

# Пример использования функции switch_account
driver = Driver(Firefox)
driver.get("https://www.facebook.com")
# Предполагается, что пользователь уже авторизован и находится на странице, где доступна кнопка переключения аккаунта
switch_account(driver)
