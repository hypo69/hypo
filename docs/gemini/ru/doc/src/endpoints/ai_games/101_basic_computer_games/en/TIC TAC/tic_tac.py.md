# Игра "Крестики-нолики"

## Обзор

Этот модуль реализует классическую игру "Крестики-нолики" для двух игроков, где один игрок - человек, а второй - компьютер. Игроки по очереди ставят свои символы на поле 3x3, стремясь собрать три своих символа в ряд.

## Функции

### `print_board`

**Описание**: Выводит текущее состояние игрового поля в консоль.

**Параметры**:
- `board` (list): Список, представляющий игровое поле.

**Возвращает**:
- `None`

### `check_win`

**Описание**: Проверяет, есть ли выигрышная комбинация для данного игрока.

**Параметры**:
- `board` (list): Список, представляющий игровое поле.
- `player` (str): Символ игрока ('X' или 'O').

**Возвращает**:
- `bool`: `True`, если игрок выиграл, иначе `False`.

### `check_draw`

**Описание**: Проверяет, является ли текущая игра ничьей.

**Параметры**:
- `board` (list): Список, представляющий игровое поле.

**Возвращает**:
- `bool`: `True`, если ничья, иначе `False`.

### `play_tic_tac_toe`

**Описание**: Основная функция для запуска игры "Крестики-нолики".

**Параметры**:
- `None`

**Возвращает**:
- `None`
