# Модуль endpoints/advertisement/facebook

## Обзор

Данный модуль содержит API-эндпоинты для работы с рекламными кампаниями на Facebook.  Он предоставляет функции для создания, редактирования, и получения информации о рекламных кампаниях.

## Функции

### `create_campaign`

**Описание**: Создает новую рекламную кампанию на Facebook.

**Параметры**:
- `campaign_data` (dict): Словарь с данными для создания кампании.  Ожидается JSON-формат, соответствующий API Facebook. (Например, имя кампании, бюджет, целевая аудитория).
- `access_token` (str):  Токен доступа к API Facebook.

**Возвращает**:
- `dict | None`: Словарь с информацией о созданной кампании, если успешно, иначе `None`.


**Вызывает исключения**:
- `FacebookApiException`: Если возникла ошибка при обращении к API Facebook.
- `ValueError`: Если `campaign_data` не имеет корректного формата.
- `TypeError`: Если типы параметров не соответствуют ожидаемым.


### `get_campaign_details`

**Описание**: Возвращает подробную информацию о заданной рекламной кампании.

**Параметры**:
- `campaign_id` (str): Идентификатор рекламной кампании.
- `access_token` (str): Токен доступа к API Facebook.


**Возвращает**:
- `dict | None`: Словарь с деталями кампании, если успешно, иначе `None`.


**Вызывает исключения**:
- `FacebookApiException`: Если возникла ошибка при обращении к API Facebook.
- `ValueError`: Если `campaign_id` некорректен.
- `TypeError`: Если типы параметров не соответствуют ожидаемым.


### `update_campaign_status`

**Описание**: Обновляет статус заданной рекламной кампании.

**Параметры**:
- `campaign_id` (str): Идентификатор рекламной кампании.
- `new_status` (str): Новый статус кампании (например, "active", "paused").
- `access_token` (str): Токен доступа к API Facebook.


**Возвращает**:
- `bool`: `True`, если обновление прошло успешно, `False` в противном случае.


**Вызывает исключения**:
- `FacebookApiException`: Если возникла ошибка при обращении к API Facebook.
- `ValueError`: Если `new_status` некорректен.
- `TypeError`: Если типы параметров не соответствуют ожидаемым.
