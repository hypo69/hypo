# Омофоны

## Обзор

В этом файле содержится описание омофонов в английском языке. Омофоны — это слова, которые звучат одинаково, но имеют разное значение и, как правило, разное написание.

## Определение

**Омофоны** – это слова, которые звучат одинаково, но имеют разное значение и, как правило, разное написание.

## Примеры

Примеры омофонов:

- **two, too, to:** эти слова произносятся одинаково, но имеют разные значения: два, тоже, к.
- **know, no:** знать, нет.
- **here, hear:** здесь, слышать.
