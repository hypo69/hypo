# StringFormatter.py - модуль для форматирования строк

```markdown
## Модуль `hypotez/src/utils/string/formatter.py`

Этот модуль предоставляет функции для форматирования строк, включая удаление символов переноса строки, HTML-тегов, нелатинских символов и специальных символов. Он также содержит функции для преобразования строк в списки, извлечения значений из строк, содержащих скобки, и очистки URL-адресов.

**Функции:**

* **`remove_line_breaks(input_str: str) -> str`**: Удаляет символы переноса строки (`\\n`, `\\r`) из входной строки.
* **`remove_htmls(input_html: str) -> str`**: Удаляет HTML-теги из входной HTML-строки. Использует регулярное выражение.
* **`escape_html_tags(input_html: str) -> str`**: Заменяет символы `<` и `>` на `&lt;` и `&gt;` соответственно в входной HTML-строке. Использует функцию `html.escape()`.
* **`escape_to_html(text: str) -> str`**: Заменяет символы на их HTML-экранированные аналоги. Использует словарь `html_escapes`.
* **`remove_non_latin_characters(input_str: str) -> str`**: Удаляет все символы, не являющиеся латинскими буквами и пробелами, из входной строки.
* **`remove_special_characters(input_str: str | list) -> str | list`**: Удаляет все символы, не являющиеся буквами, цифрами и пробелами, из входной строки или списка строк. Если на вход поступает список, возвращает список обработанных строк.
* **`clear_numbers(input_str: str) -> str`**: Оставляет в строке только цифры и точки. Удаляет все символы, кроме цифр и точек.


**Преобразования данных:**

* **`convert_to_list(input: str | list[str, dict, list[dict]], delimiter: str = ',') -> list`**: Преобразует строку или список в список строк, используя разделитель (по умолчанию запятая).  (Описание этой функции находится в исходном коде, но код не предоставлен в примере).
* **`extract_value_from_parentheses_with_lead_dollar(input_str: str) -> str | list | bool`**: Извлекает значения из строки, заключённой в скобки, если строка начинается с `$`. Возвращает строку, список или `False`, если скобки не найдены. (Описание этой функции находится в исходном коде, но код не предоставлен в примере).
* **`clean_url_from_protocols(url: str) -> str`**: Очищает URL-адрес, удаляя протоколы (например, `http://`, `https://`). Использует функцию `urllib.parse.urlparse`. (Описание этой функции находится в исходном коде, но код не предоставлен в примере).

**Дополнительная информация:**

* Модуль использует регулярные выражения (`re`) для поиска и замены подстрок.
* Модуль использует функцию `html.escape()` для экранирования HTML-символов.
* Модуль импортирует класс `logger` из модуля `src.logger`.
* Модуль импортирует словарь `html_escapes`.


**Пример использования (для функции `clear_numbers`):**

```python
input_str = 'aaa123.456 cde'
output_str = StringFormatter.clear_numbers(input_str)
print(output_str)  # Выведет: 123.456
```

**Важно:**  Остальные функции (кроме `clear_numbers`) продемонстрированы в примере использования, но нет конкретных примеров вывода или обработки данных.
