# Модуль определения типов данных

## Обзор

Этот модуль определяет пользовательские типы данных и импортирует необходимые модули для аннотации типов.
Он используется для задания строгой типизации в проекте, что помогает улучшить читаемость и поддерживаемость кода.

## Подробнее

Модуль содержит определение типа `sha256`, который является новым типом на основе строки и представляет собой хэш SHA-256.
Также модуль импортирует различные типы из модуля `typing`, такие как `Dict`, `NewType`, `Union`, `Optional` и `List`.

## Типы

### `sha256`

**Описание**: Пользовательский тип, представляющий собой строку, содержащую хэш SHA-256.

**Принцип работы**:
Тип `sha256` создается с использованием `NewType` из модуля `typing`. `NewType` позволяет создать новый тип, который является подтипом существующего типа (в данном случае, `str`). Это полезно для повышения семантической ясности кода.

```python
sha256 = NewType('sha_256_hash', str)
```

**Примеры**:

```python
from typing import NewType

# Создание типа sha256
sha256 = NewType('sha_256_hash', str)

# Использование типа sha256
my_hash: sha256 = "e5b7b6a7c4a8b3b2e1e7a7b8c4a8b3b2e1e7a7b8c4a8b3b2e1e7a7b8c4a8b3b2"
print(my_hash)  # Вывод: e5b7b6a7c4a8b3b2e1e7a7b8c4a8b3b2e1e7a7b8c4a8b3b2e1e7a7b8c4a8b3b2
print(type(my_hash)) # Вывод: <class '__main__.sha_256_hash'>
```
