# Голосовой помощник chatgpt - telegram

## Обзор

Данный модуль предоставляет интеграцию между чат-ботом на базе модели ChatGPT и Telegram-ботом.  Он позволяет пользователям взаимодействовать с ChatGPT через Telegram-интерфейс.

## Модули

### `main.py`

**Описание**: Главный модуль, содержащий точки входа для Telegram-ботов.

**Функции**:

### `handle_message`

**Описание**: Обрабатывает входящие сообщения от пользователя в Telegram.

**Параметры**:
- `update`: Объект, содержащий информацию о входящем сообщении.

**Возвращает**:
- `None`: Функция не возвращает значение.


**Вызывает исключения**:
- `Exception`: Общий обработчик ошибок.


### `main`

**Описание**: Точка входа для запуска Telegram-бота.

**Параметры**:
- `token`: Токен доступа к Telegram-боту.

**Возвращает**:
- `None`: Функция не возвращает значение.


**Вызывает исключения**:
- `Exception`: Общий обработчик ошибок.
