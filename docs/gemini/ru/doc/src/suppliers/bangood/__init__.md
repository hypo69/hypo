# Модуль hypotez/src/suppliers/bangood/__init__.py

## Обзор

Данный модуль предоставляет функции и классы для работы с поставщиком Bangood.  Он содержит инструменты для получения данных о категориях и продуктах.


## Переменные

### `MODE`

**Описание**:  Переменная, определяющая режим работы (например, 'dev', 'prod').


## Классы

### `Graber`

**Описание**: Класс для извлечения данных с сайта Bangood.  Подробная информация о функциональности находится в файле `graber.py`.

## Функции

### `get_list_categories_from_site`

**Описание**: Возвращает список категорий с сайта Bangood.

**Возвращает**:
- `list`: Список словарей, где каждый словарь представляет категорию (идентификатор, имя и т.д.).

### `get_list_products_in_category`

**Описание**: Возвращает список продуктов, относящихся к определенной категории на сайте Bangood.

**Параметры**:
- `category_id` (str): Идентификатор категории.


**Возвращает**:
- `list`: Список словарей, где каждый словарь представляет продукт (идентификатор, название, цена и т.д.).
