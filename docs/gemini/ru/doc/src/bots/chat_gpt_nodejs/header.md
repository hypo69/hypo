# Модуль `hypotez/src/bots/openai_bots/header.py`

## Обзор

Этот модуль содержит вспомогательные функции для работы с проектом Hypotez, включая определение корневой директории проекта и загрузку настроек из файла `settings.json`.  Также он предоставляет доступ к документированию проекта из `README.MD`.

## Функции

### `set_project_root`

**Описание**:  Функция находит корневую директорию проекта, начиная от текущего файла. Она ищет файлы-маркеры (например, `pyproject.toml`, `requirements.txt`, `.git`) в родительских директориях текущего файла, и возвращает путь к найденной корневой директории. Если корневой каталог не найден, возвращает текущую директорию.

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, которые указывают на корень проекта. По умолчанию это `('pyproject.toml', 'requirements.txt', '.git')`.


**Возвращает**:

- `Path`: Путь к корневой директории проекта. Если корневая директория не найдена, возвращается путь к директории, где находится текущий файл.

**Вызывает исключения**:

- Нет


###  Другие атрибуты и переменные

* **`__root__`**: Путь к корневой директории проекта, получаемый вызовом функции `set_project_root`. Тип - `Path`.
* **`settings`**: Словарь настроек, загруженный из файла `settings.json` в корневой директории проекта. Тип - `dict`.
* **`doc_str`**: Строка документации, загруженная из файла `README.MD` в корневой директории проекта. Тип - `str`.
* **`__project_name__`**: Название проекта, полученное из настроек или значение по умолчанию `'hypotez'`. Тип - `str`.
* **`__version__`**: Версия проекта, полученная из настроек или пустая строка по умолчанию. Тип - `str`.
* **`__doc__`**: Документация проекта, полученная из `README.MD` или пустая строка по умолчанию. Тип - `str`.
* **`__details__`**: Дополнительная информация о проекте (по умолчанию пустая строка). Тип - `str`.
* **`__author__`**: Автор проекта, полученный из настроек или пустая строка по умолчанию. Тип - `str`.
* **`__copyright__`**: Авторские права, полученные из настроек или пустая строка по умолчанию. Тип - `str`.
* **`__cofee__`**: Ссылка на спонсорство разработчика (по умолчанию ссылка на Boosty). Тип - `str`.


**Примечания**:


Этот модуль предназначен для инициализации и работы с константами проекта, такими как его имя, версия и корневой каталог. Он загружает настройки и документацию, необходимые для работы других частей проекта. Обработка ошибок (например, `FileNotFoundError`, `json.JSONDecodeError`) позволяет программе продолжать выполнение, если необходимые файлы отсутствуют или повреждены.
