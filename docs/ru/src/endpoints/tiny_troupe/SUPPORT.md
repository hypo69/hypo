# Документация для разработчика: Файл SUPPORT.md

## Обзор

Этот файл содержит информацию о том, как сообщать об ошибках и получать помощь по проекту TinyTroupe.

## Подробней

Файл `SUPPORT.md` предоставляет пользователям инструкции о том, как правильно оформлять сообщения об ошибках (issues) и где искать ответы на вопросы по использованию проекта. Он также содержит информацию о политике поддержки Microsoft для TinyTroupe. Этот файл важен для поддержания качественной обратной связи с пользователями и обеспечения эффективной поддержки проекта.

## Разделы

### Как сообщать об ошибках и получать помощь

В этом разделе описывается использование GitHub Issues для отслеживания ошибок и запросов на новые функции.

**Принцип работы**:

1.  **Поиск существующих issues**: Перед созданием нового issue рекомендуется выполнить поиск среди уже существующих, чтобы избежать дублирования.
2.  **Создание нового issue**: Если проблема или запрос на функцию еще не зарегистрированы, создайте новый issue с подробным описанием.

Для получения помощи и ответов на вопросы по использованию проекта рекомендуется обращаться к Wiki или форуму обсуждений, расположенным на GitHub.
