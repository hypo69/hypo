# Модуль hypotez/src/utils/path.py

## Обзор

Модуль `hypotez/src/utils/path.py` определяет корневой путь к проекту. Все импорты строятся относительно этого пути.  В дальнейшем планируется перенести определение корневого пути в системные переменные.

## Переменные

### `MODE`

**Описание**:  Строковая переменная, хранящая режим работы (`dev`).

## Функции

### `get_relative_path`

**Описание**: Возвращает относительный путь, начиная с указанного сегмента.

**Параметры**:

- `full_path` (str): Полный путь.
- `relative_from` (str): Сегмент пути, с которого нужно начать извлечение.

**Возвращает**:

- `Optional[str]`: Относительный путь начиная с `relative_from`, или `None`, если сегмент не найден.

**Пример использования**:

```python
full_path = "/home/user/project/data/file.txt"
relative_from = "data"
relative_path = get_relative_path(full_path, relative_from)
print(relative_path)  # Выведет "data/file.txt"
```

**Реализация**:

1. Преобразует строки `full_path` и `relative_from` в объекты `Path`.
2. Разбивает `full_path` на составляющие (parts).
3. Находит индекс `relative_from` в `parts`.
4. Если `relative_from` найден:
   - Формирует `relative_path` из частей, начиная с найденного индекса.
   - Возвращает строковое представление `relative_path` с использованием `as_posix()`.
5. Если `relative_from` не найден:
   - Возвращает `None`.

**Возможные исключения**:

- Нет никаких обработанных исключений.
