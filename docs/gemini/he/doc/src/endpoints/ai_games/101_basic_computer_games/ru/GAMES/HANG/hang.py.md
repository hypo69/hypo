# `hang.py`

## סקירה כללית

מודול זה מממש את המשחק "איש תלוי", בו השחקן מנסה לנחש מילה נסתרת על ידי ניחוש אותיות. לכל ניחוש לא נכון, מתווסף חלק מציור האיש התלוי. השחקן מפסיד אם הציור הושלם לפני שהמילה נוחשה.
