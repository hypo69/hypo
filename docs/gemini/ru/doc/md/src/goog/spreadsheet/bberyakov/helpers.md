# Модуль helpers

## Обзор

Этот модуль предоставляет функции для работы с цветовыми форматами, включая преобразование между HEX, десятичным и RGB.

## Функции

### `hex_color_to_decimal`

**Описание**: Преобразует цвет из шестнадцатеричного (HEX) формата в десятичный.

**Параметры**:
- `letters` (str): Строка, представляющая цвет в HEX формате (например, "A1"). Допускается как один символ, так и два.

**Возвращает**:
- `int`: Десятичное значение цвета.


**Вызывает исключения**:
- Нет


### `decimal_color_to_hex`

**Описание**: Преобразует цвет из десятичного формата в шестнадцатеричный (HEX).

**Параметры**:
- `number` (int): Целое число, представляющее цвет в десятичном формате.

**Возвращает**:
- `str`: Строка, представляющая цвет в HEX формате (например, "A1").

**Вызывает исключения**:
- Нет


### `hex_to_rgb`

**Описание**: Преобразует цвет из шестнадцатеричного (HEX) формата в RGB.

**Параметры**:
- `hex` (str): Строка, представляющая цвет в HEX формате (например, "#FFFFFF" или "FFFFFF").

**Возвращает**:
- `tuple`: Кортеж, содержащий значения RGB (например, (255, 255, 255)).

**Вызывает исключения**:
- Нет
