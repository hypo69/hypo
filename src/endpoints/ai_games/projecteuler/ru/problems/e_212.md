# Задача No 212
## Соединенные кубы
## Условия
Рассмотрим набор кубов, которые были выровнены в 3D-пространстве таким образом, что их стороны параллельны осям координат. Каждый куб характеризуется координатами своих противоположных углов: *x*<sub>1</sub>, *y*<sub>1</sub>, *z*<sub>1</sub> и *x*<sub>2</sub>, *y*<sub>2</sub>, *z*<sub>2</sub>, где *x*<sub>1</sub> < *x*<sub>2</sub>, *y*<sub>1</sub> < *y*<sub>2</sub> и *z*<sub>1</sub> < *z*<sub>2</sub>.

Когда два куба имеют хотя бы одну общую точку, мы говорим, что они "соединены". Если куб A соединен с кубом B, а куб B соединен с кубом C, то куб A также соединен с кубом C через куб B (мы предполагаем, что соединение является транзитивным).

Мы генерируем набор кубов следующим образом:
Используем генератор случайных чисел с заданным начальным значением *s*<sub>0</sub> = 290797.

Для *n* ≥ 1, *s*<sub>n</sub> = (*s*<sub>n-1</sub><sup>2</sup>) mod 50515093.
Таким образом, мы можем определить координаты куба *n* следующим образом:
*x*<sub>1,n</sub> = *s*<sub>6n-5</sub> mod 10000
*y*<sub>1,n</sub> = *s*<sub>6n-4</sub> mod 10000
*z*<sub>1,n</sub> = *s*<sub>6n-3</sub> mod 10000
*x*<sub>2,n</sub> = *x*<sub>1,n</sub> + (*s*<sub>6n-2</sub> mod 2000) + 1
*y*<sub>2,n</sub> = *y*<sub>1,n</sub> + (*s*<sub>6n-1</sub> mod 2000) + 1
*z*<sub>2,n</sub> = *z*<sub>1,n</sub> + (*s*<sub>6n</sub> mod 2000) + 1

Таким образом, первые два куба задаются следующим образом:
C<sub>1</sub>: (9577, 7907, 7317) - (11577, 9707, 8317)
C<sub>2</sub>: (1556, 2894, 4379) - (2456, 4694, 5979)

Найдите общее количество соединенных групп после того, как сгенерируется 10000 кубов.
