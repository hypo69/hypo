# Модуль `header.py`

## Обзор

Этот модуль содержит общие настройки и метаданные для проекта, а также функцию для определения корневой директории проекта.

## Функции

### `set_project_root`

**Описание**:
Находит корневую директорию проекта, начиная с директории текущего файла и поднимаясь вверх по иерархии, останавливаясь на первой директории, содержащей любой из указанных маркерных файлов.

**Параметры**:
- `marker_files` (tuple, optional): Кортеж имен файлов или директорий для определения корневой директории проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена, иначе — путь к директории, где расположен скрипт.

## Глобальные переменные

### `MODE`

**Описание**:
Режим работы приложения, по умолчанию установлен в `'dev'`.

### `__root__`

**Описание**:
Путь к корневой директории проекта.

### `settings`

**Описание**:
Словарь, содержащий настройки проекта, загруженные из файла `settings.json`. Если файл не найден или не удается декодировать JSON, значение будет `None`.

### `doc_str`

**Описание**:
Строка, содержащая содержимое файла `README.MD`. Если файл не найден, значение будет `None`.

### `__project_name__`

**Описание**:
Имя проекта, по умолчанию `'hypotez'`, берется из `settings.json`, если доступен.

### `__version__`

**Описание**:
Версия проекта, берется из `settings.json`, если доступен.

### `__doc__`

**Описание**:
Строка документации проекта, загружается из файла `README.MD` или пустая строка если файл недоступен.

### `__details__`

**Описание**:
Дополнительная информация о проекте, на данный момент пустая строка.

### `__author__`

**Описание**:
Автор проекта, берется из `settings.json`, если доступен.

### `__copyright__`

**Описание**:
Информация об авторских правах, берется из `settings.json`, если доступен.

### `__cofee__`

**Описание**:
Сообщение с предложением поддержать разработчика чашкой кофе, берется из `settings.json`, если доступен, иначе задано значение по умолчанию.
