# Модуль `hypotez/src/endpoints/kazarinov/react/header.py`

## Обзор

Этот модуль содержит функции для определения корневого каталога проекта, загрузки настроек из файла `settings.json` и чтения документации из файла `README.MD`. Он предоставляет переменные, содержащие информацию о проекте, такие как имя, версия, описание, автор и информация о спонсорстве.

## Функции

### `set_project_root`

**Описание**: Определяет корневой каталог проекта, начиная с текущего каталога и ища вверх по дереву каталогов до тех пор, пока не найдет каталог, содержащий один из указанных маркеров (файлы или каталоги).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или каталогов, используемых в качестве маркеров для определения корневого каталога. По умолчанию это `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневому каталогу проекта, если он найден, иначе - путь к каталогу, в котором находится скрипт.

**Вызывает исключения**:

- Нет


### Загрузка настроек

**Описание**: Загружает настройки из файла `settings.json` в корневом каталоге проекта.

**Параметры**:
- Нет

**Возвращает**:
- Нет

**Вызывает исключения**:

- `FileNotFoundError`: Если файл `settings.json` не найден.
- `json.JSONDecodeError`: Если содержимое файла `settings.json` не является валидным JSON.

### Чтение документации

**Описание**: Читает содержимое файла `README.MD` в корневом каталоге проекта.

**Параметры**:
- Нет

**Возвращает**:
- Нет

**Вызывает исключения**:

- `FileNotFoundError`: Если файл `README.MD` не найден.
- `json.JSONDecodeError`: Если содержимое файла `README.MD` не является валидным JSON.


## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта. Инициализируется функцией `set_project_root()`.

**Тип**: `Path`


### `MODE`

**Описание**: Строковая переменная, содержащая режим работы. В данном случае это 'dev'.


### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженные из файла `settings.json`.

**Тип**: `dict`


### `doc_str`

**Описание**: Строка, содержащая содержимое файла `README.MD`.

**Тип**: `str`


### `__project_name__`

**Описание**: Имя проекта. Получается из настроек или имеет значение по умолчанию `hypotez`.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта. Получается из настроек или имеет значение по умолчанию пустая строка.

**Тип**: `str`


### `__doc__`

**Описание**: Документация проекта. Получается из файла `README.MD` или имеет значение по умолчанию пустая строка.

**Тип**: `str`


### `__details__`

**Описание**: Подробная информация о проекте.


**Тип**: `str`


### `__author__`

**Описание**: Автор проекта. Получается из настроек или имеет значение по умолчанию пустая строка.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права на проект. Получается из настроек или имеет значение по умолчанию пустая строка.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка на страницу для спонсорства разработчика.


**Тип**: `str`
