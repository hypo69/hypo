# Модуль `hypotez/src/fast_api/header.py`

## Обзор

Этот модуль содержит функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и получения документации из файла `README.MD`.  Он инициализирует глобальные переменные, необходимые для работы других частей приложения, такие как имя проекта, версия, описание и т.д.

    - [`set_project_root`](#set_project_root)

## Функции

### `set_project_root`

**Описание**:  Находит корневую директорию проекта, начиная от текущего файла и ища вверх по дереву директорий, пока не найдет директорию, содержащую один из указанных файлов-маркеров (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, по которым будет определяться корневая директория проекта. По умолчанию использует кортеж (`'pyproject.toml'`, `'requirements.txt'`, `.git`).

**Возвращает**:

- `Path`: Объект `Path` представляющий путь к корневой директории проекта. Возвращает директорию текущего файла, если корневая директория не найдена.

**Обрабатывает исключения**:

- Нет.


## Глобальные переменные

Этот модуль определяет следующие глобальные переменные:

- `__root__`: Путь к корневой директории проекта (`Path`).
- `settings`: Словарь с настройками проекта (`dict`).
- `doc_str`: Строка с содержанием файла `README.MD` (`str`).
- `__project_name__`: Имя проекта (`str`).
- `__version__`: Версия проекта (`str`).
- `__doc__`: Документация проекта (`str`).
- `__details__`: Подробная информация о проекте (`str`).
- `__author__`: Автор проекта (`str`).
- `__copyright__`: Авторские права (`str`).
- `__cofee__`: Ссылка на спонсорство разработчика (cup of coffee). (`str`).


**Обработка ошибок**:

Файлы `settings.json` и `README.MD` загружаются с использованием блоков `try...except`. При возникновении `FileNotFoundError` или `json.JSONDecodeError` эти переменные не инициализируются или устанавливаются в значения по умолчанию.
