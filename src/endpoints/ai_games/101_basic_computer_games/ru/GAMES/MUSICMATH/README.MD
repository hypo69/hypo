
# MusicMath: Демонстрация связи математики и музыки

## Описание
Модуль `MusicMath` предоставляет набор функций, демонстрирующих применение математических принципов в музыке. Этот модуль позволяет вычислять частоты нот, отношения между ними, длительности нот и ритмические паттерны.

## Функции

### `calculate_frequency(note_number, concert_a_freq=440.0)`
Вычисляет частоту ноты в герцах на основе её номера относительно A4 (440 Гц).

*   `note_number`: Номер ноты относительно A4 (A4 = 0).
*   `concert_a_freq`: Частота ноты A4 (по умолчанию 440.0 Гц).

### `calculate_interval_ratio(note1_number, note2_number)`
Вычисляет отношение частот между двумя нотами.

*   `note1_number`: Номер первой ноты.
*   `note2_number`: Номер второй ноты.

### `calculate_tempo_duration(bpm, beat_length, beats)`
Вычисляет продолжительность музыкального фрагмента на основе темпа и длительности ударов.

*   `bpm`: Темп в ударах в минуту.
*   `beat_length`: Длительность одного удара в секундах.
*   `beats`: Количество ударов в музыкальном фрагменте.

### `calculate_note_duration(tempo, note_value)`
Вычисляет длительность ноты в секундах.

*  `tempo`: Темп в ударах в минуту.
*   `note_value`: Длительность ноты относительно целой ноты (1.0 = целая, 0.5 = половинная, 0.25 = четвертная и т.д.).

### `calculate_rhythm_pattern(bar_length, note_values)`
Вычисляет общую продолжительность ритмического паттерна.
    
*   `bar_length`: Длительность такта в количестве целых нот.
*  `note_values`: Список длительностей нот (относительно целой ноты).

### `print_musical_examples()`
Выводит примеры использования функций модуля.

## Примеры использования
```python
import music_math

music_math.print_musical_examples()
```
