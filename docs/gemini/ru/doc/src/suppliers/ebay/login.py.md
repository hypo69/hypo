Модуль `src.suppliers.ebay.login` предоставляет интерфейс для авторизации на eBay с использованием веб-драйвера.

## Функции

### `login`

**Описание**: Функция выполняет авторизацию на eBay с использованием веб-драйвера.

**Параметры**:
- `driver`:  Веб-драйвер, используемый для взаимодействия с браузером.
- `user_name` (str): Имя пользователя eBay.
- `password` (str): Пароль пользователя eBay.
- `wait_time` (int, optional): Время ожидания для загрузки страницы. По умолчанию 10.

**Возвращает**:
- `bool`: `True`, если авторизация прошла успешно, `False` в противном случае.

**Вызывает исключения**:
- `TimeoutException`: Исключение, возникающее если время ожидания загрузки страницы истекло.
- `NoSuchElementException`: Исключение, возникающее если не удалось найти элемент на странице.
- `Exception`:  Исключение, возникающее при других ошибках авторизации.
