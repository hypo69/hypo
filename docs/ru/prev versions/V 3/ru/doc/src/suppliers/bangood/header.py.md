# Модуль `header.py`

## Обзор

Модуль `header.py` предназначен для определения основных настроек и параметров проекта `hypotez`. Он включает в себя функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и чтения документации из файла `README.MD`. Этот модуль играет важную роль в инициализации проекта, обеспечивая доступ к основным параметрам и путям.

## Подробней

Модуль выполняет следующие задачи:

- Определение корневой директории проекта на основе наличия определенных файлов-маркеров (`__root__`, `.git`).
- Загрузка настроек проекта из файла `settings.json`, включая имя проекта, версию, автора и т.д.
- Чтение документации проекта из файла `README.MD`.
- Определение глобальных переменных, таких как имя проекта, версия, автор и документация, которые используются в других модулях проекта.

## Функции

### `set_project_root`

```python
def set_project_root(marker_files: tuple[str, ...] = ('__root__', '.git')) -> Path:
    """
    Находит корневой каталог проекта, начиная с каталога текущего файла,
    поиск вверх и остановка на первом каталоге, содержащем любой из файлов-маркеров.

    Args:
        marker_files (tuple): Имена файлов или каталогов для идентификации корня проекта.

    Returns:
        Path: Путь к корневому каталогу, если он найден, в противном случае - каталог, где расположен скрипт.

    Example:
        >>> set_project_root()
        ...
    """
```

**Описание**: Функция `set_project_root` определяет корневой каталог проекта путем поиска файлов-маркеров в родительских директориях.

**Параметры**:

- `marker_files` (tuple, optional): Кортеж с именами файлов или каталогов, используемых для определения корня проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:

- `Path`: Объект `Path`, представляющий путь к корневому каталогу проекта.

### `__root__`

```python
__root__: Path
"""__root__ (Path): Path to the root directory of the project"""
```

**Описание**:  `__root__` хранит путь к корневой директории проекта.

**Параметры**:
   - Нет параметров.

**Возвращает**:
   - `Path`:  возвращает путь к корневой директории проекта.

## Переменные

### `settings`

**Описание**: Глобальная переменная `settings`, содержащая настройки проекта, загруженные из файла `settings.json`.

### `doc_str`

**Описание**: Глобальная переменная `doc_str`, содержащая документацию проекта, загруженную из файла `README.MD`.

### `__project_name__`

**Описание**: Глобальная переменная `__project_name__`, содержащая имя проекта.

### `__version__`

**Описание**: Глобальная переменная `__version__`, содержащая версию проекта.

### `__doc__`
