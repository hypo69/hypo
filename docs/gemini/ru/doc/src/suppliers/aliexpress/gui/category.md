# Модуль `hypotez/src/suppliers/aliexpress/gui/category.py`

## Обзор

Этот модуль предоставляет интерфейс для подготовки рекламных кампаний на AliExpress. Он основан на PyQt6 и использует асинхронные операции для повышения производительности. Модуль позволяет загружать JSON-файлы с конфигурацией кампаний, отображать информацию о кампании и категориях, а также асинхронно готовить категории к использованию.

## Классы

### `CategoryEditor`

**Описание**: Класс `CategoryEditor` представляет собой главное окно приложения, отвечающее за взаимодействие с пользователем при подготовке категорий рекламных кампаний.

**Атрибуты**:

- `campaign_name`: Имя кампании (строка).
- `data`: Данные кампании в формате `SimpleNamespace` (объект `SimpleNamespace`).
- `language`: Язык кампании (строка).
- `currency`: Валюта кампании (строка).
- `file_path`: Путь к загруженному файлу (строка).
- `editor`: Экземпляр класса `AliCampaignEditor` для работы с кампаниями.

**Методы**:

#### `__init__(self, parent=None, main_app=None)`

**Описание**: Инициализирует главное окно.

**Параметры**:
- `parent` (необязательно): Родительское окно.
- `main_app` (необязательно): Экземпляр главного приложения.

**Возвращает**:
-  None

#### `setup_ui(self)`

**Описание**: Настраивает пользовательский интерфейс окна.

**Возвращает**:
- None

#### `setup_connections(self)`

**Описание**: Устанавливает связи между элементами управления.

**Возвращает**:
- None

#### `open_file(self)`

**Описание**: Открывает диалог для выбора JSON-файла с данными кампании.

**Возвращает**:
- None

#### `load_file(self, campaign_file)`

**Описание**: Загружает JSON-файл с данными кампании.

**Параметры**:
- `campaign_file` (str): Путь к файлу.

**Возвращает**:
- None
**Возможные исключения**:
- `Exception`: В случае ошибки при загрузке файла.  Сообщение об ошибке будет выведено пользователю.

#### `create_widgets(self, data)`

**Описание**: Создает виджеты на основе загруженных данных из JSON-файла.

**Параметры**:
- `data` (SimpleNamespace): Данные кампании.

**Возвращает**:
- None


#### `prepare_all_categories_async(self)`

**Описание**: Асинхронно подготавливает все категории.

**Возвращает**:
- None
**Возможные исключения**:
- `Exception`: В случае ошибки при подготовке категорий.  Сообщение об ошибке будет выведено пользователю.

#### `prepare_category_async(self)`

**Описание**: Асинхронно подготавливает конкретную категорию.

**Возвращает**:
- None
**Возможные исключения**:
- `Exception`: В случае ошибки при подготовке категории.  Сообщение об ошибке будет выведено пользователю.
