# Модуль для конвертации XLS файлов в словарь
## Обзор

Модуль предоставляет функцию `xls2dict` для конвертации XLS файлов в формат словаря. Он использует функции `read_xls_as_dict` и `save_xls_file` из модуля `src.utils.xls`.

## Подробней

Этот модуль предназначен для упрощения процесса чтения данных из XLS файлов и представления их в виде словаря Python. Это может быть полезно для обработки и анализа данных, хранящихся в формате XLS.

## Функции

### `xls2dict`

```python
def xls2dict(xls_file: str | Path) -> dict | None:
    """Конвертирует XLS файл в словарь.
    
    Args:
        xls_file (str | Path): Путь к XLS файлу.

    Returns:
        dict | None: Словарь, представляющий данные из XLS файла, или None в случае ошибки.

    """
```

**Назначение**: Конвертирует XLS файл в словарь.

**Параметры**:
- `xls_file` (str | Path): Путь к XLS файлу, который нужно конвертировать.

**Возвращает**:
- `dict | None`: Словарь, представляющий данные из XLS файла. Возвращает `None` в случае ошибки.

**Как работает функция**:
1. Функция `xls2dict` принимает путь к XLS файлу (`xls_file`) в качестве аргумента.
2. Вызывает функцию `read_xls_as_dict` из модуля `src.utils.xls`, передавая ей `xls_file`.
3. Функция `read_xls_as_dict` читает данные из XLS файла и преобразует их в словарь.
4. Возвращает полученный словарь.

```
      Начало
        ↓
  Передача xls_file
  в read_xls_as_dict
        ↓
   read_xls_as_dict 
  обрабатывает xls_file
        ↓
      Возврат
```

**Примеры**:

```python
from pathlib import Path
# Пример 1: Преобразование XLS файла в словарь
file_path = Path('example.xls')
data = xls2dict(file_path)
if data:
    print(f'Данные из файла: {data}')
else:
    print('Не удалось преобразовать XLS файл в словарь.')

# Пример 2: Преобразование XLS файла по строковому пути
file_path_str = 'example.xls'
data = xls2dict(file_path_str)
if data:
    print(f'Данные из файла: {data}')
else:
    print('Не удалось преобразовать XLS файл в словарь.')
