# Модуль aliexpress/alirequests.py

## Обзор

Этот модуль предоставляет класс `AliRequests` для работы с API AliExpress. Он позволяет загружать куки из файлов, обновлять сессию и делать GET-запросы с учетом куки, а также генерирует короткие партнерские ссылки.

## Классы

### `AliRequests`

**Описание**: Класс `AliRequests` обрабатывает запросы к AliExpress, используя библиотеку `requests`.  Он хранит куки, сессионный идентификатор и заголовки.

**Методы**:

#### `__init__`

**Описание**: Инициализирует экземпляр класса `AliRequests`.

**Параметры**:

- `webdriver_for_cookies` (str, опционально, по умолчанию 'chrome'):  Имя вебдрайвера, из которого будут загружаться куки.

**Возвращает**:
- Нет.


#### `_load_webdriver_cookies_file`

**Описание**: Загружает куки из файла вебдрайвера.

**Параметры**:

- `webdriver_for_cookies` (str, опционально, по умолчанию 'chrome'): Имя вебдрайвера.

**Возвращает**:
- bool: `True`, если куки загружены успешно, `False` в противном случае.

**Обрабатывает исключения**:
- `FileNotFoundError`: Файл с куками не найден.
- `ValueError`: Произошла ошибка при разборе данных куки.
- `Exception`: Другие ошибки при загрузке куки.


#### `_refresh_session_cookies`

**Описание**: Обновляет куки сессии.

**Возвращает**:
- Нет.

**Обрабатывает исключения**:
- `requests.RequestException`: Ошибка при обновлении сессии.
- `Exception`: Другие ошибки при обновлении сессии.


#### `_handle_session_id`

**Описание**: Обрабатывает `JSESSIONID` в куках ответа.

**Параметры**:
- `response_cookies`: Куки ответа.

**Возвращает**:
- Нет.


#### `make_get_request`

**Описание**: Выполняет GET-запрос с использованием сохраненных кук.

**Параметры**:

- `url` (str): URL для запроса.
- `cookies` (List[dict], опционально): Список куки для запроса.
- `headers` (dict, опционально): Заголовки для запроса.

**Возвращает**:
- `requests.Response` или `False`: Объект `requests.Response` при успехе, `False` в противном случае.

**Обрабатывает исключения**:
- `requests.RequestException`: Ошибка при выполнении запроса.
- `Exception`: Другие ошибки при выполнении запроса.

#### `short_affiliate_link`

**Описание**: Создает короткую партнерскую ссылку.

**Параметры**:

- `link_url` (str): Исходный URL.

**Возвращает**:
- `requests.Response` или `False`: Объект `requests.Response` при успехе, `False` в противном случае.

**Обрабатывает исключения**:
- `requests.RequestException`: Ошибка при выполнении запроса.
- `Exception`: Другие ошибки при выполнении запроса.
