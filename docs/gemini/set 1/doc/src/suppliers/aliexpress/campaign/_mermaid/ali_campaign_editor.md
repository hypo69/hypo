# Модуль `ali_campaign_editor`

## Обзор

Этот модуль предоставляет инструменты для редактирования кампаний AliExpress. Он позволяет пользователю вносить изменения в список продуктов, описание кампании, категории и другие параметры, используя предоставленные данные и файлы. Модуль использует JSON-файлы для хранения данных о категориях и продуктах, что обеспечивает структурированное хранение информации.


## Описание диаграммы

Диаграмма `mermaid` описывает взаимодействие между классами и функциями `AliCampaignEditor` и `AliPromoCampaign`, а также последовательность действий при редактировании кампании. В ней показаны различные этапы:

- Ввод данных пользователем (название кампании, язык, валюта).
- Инициализация `AliCampaignEditor` и `AliPromoCampaign`.
- Чтение файла с продуктами (`sources.txt`) для проверки и удаления/переименования продуктов.
- Обновление данных продуктов.
- Обновление свойств кампании (описание).
- Обновление категорий в JSON-файлах.
- Получение категорий по имени.
- Получение продуктов для категории.
- Список всех категорий.


## Классы

### `AliCampaignEditor`

**Описание**: Класс `AliCampaignEditor` предоставляет методы для управления кампаниями AliExpress.  Он отвечает за обработку ввода пользователя, взаимодействие с файлами продуктов, обновление данных кампании и категорий.


**Методы**:

- `__init__`: Инициализирует экземпляр `AliCampaignEditor`.
- `delete_product`: Удаляет продукты из списка, если найдены совпадения.
- `update_product`: Обновляет информацию о продуктах.
- `update_campaign`: Обновляет параметры кампании (описание и другие).
- `update_category`: Обновляет информацию о категории в JSON файле.
- `get_category`: Возвращает данные о категории по имени.
- `list_categories`: Возвращает список всех категорий.
- `get_category_products`: Возвращает список продуктов для заданной категории.
- `Other methods`: Другие методы для работы с кампанией и продуктами.


## Обрабатываемые исключения:

- `FileNotFoundError`: Файл не найден.
- `JSONDecodeError`: Ошибка при декодировании JSON.
- `ValueError`: Проблемы с типом данных.
