# Модуль `make_summary.py`

## Обзор

Модуль предназначен для автоматического создания файла `SUMMARY.md`, который используется для компиляции документации с помощью инструментов, таких как `mdbook`. Модуль рекурсивно обходит указанную директорию с исходными файлами `.md` и генерирует оглавление, включая или исключая файлы в зависимости от указанного языка.

## Подробней

Модуль `make_summary.py` автоматизирует процесс создания оглавления (`SUMMARY.md`) для документации, генерируемой с помощью `mdbook`. Он выполняет рекурсивный обход указанной директории (обычно `src`) в поисках файлов с расширением `.md`. В зависимости от указанного языка (`ru` или `en`), модуль включает или исключает файлы с суффиксом `.ru.md`. Это позволяет поддерживать мультиязыковую документацию, где для каждого языка создается отдельный `SUMMARY.md`.

Скрипт полезен для проектов, требующих автоматизированную генерацию документации на нескольких языках. Использование путей относительно корня проекта обеспечивает устойчивость к изменениям в структуре директорий.
