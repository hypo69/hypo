# Модуль `emil_design.py`

## Обзор

Модуль `emil_design.py` предназначен для управления и обработки изображений, а также их продвижения на платформах Facebook и PrestaShop. Он включает в себя функциональность для описания изображений с использованием AI, публикации сообщений в Facebook и загрузки товаров в PrestaShop.

## Классы

### `EmilDesign`

**Описание**: Класс `EmilDesign` предназначен для проектирования и продвижения изображений через различные платформы.

**Атрибуты**:
- `ENDPOINT` (str): Константа, определяющая имя конечной точки ('emil').
- `gemini` (GoogleGenerativeAI): Экземпляр модели Google Gemini AI.
- `openai` (OpenAIModel): Экземпляр модели OpenAI.
- `base_path` (Path): Путь к базовой директории модуля.
- `config` (SimpleNamespace): Конфигурация модуля, загруженная из JSON-файла.
- `data_path` (Path): Путь к директории для хранения данных модуля.

**Методы**:
- [`__init__`](#__init__): Инициализирует класс `EmilDesign`.
- [`describe_images`](#describe_images): Описывает изображения, используя AI.
- [`promote_to_facebook`](#promote_to_facebook): Продвигает изображения и описания на Facebook.
- [`upload_to_prestashop`](#upload_to_prestashop): Загружает информацию о товарах в PrestaShop.
