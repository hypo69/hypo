# Модуль `src.endpoints.prestashop.api`

## Обзор

Модуль предоставляет интерфейс для взаимодействия с API PrestaShop, как в синхронном, так и в асинхронном режимах.

## Импорты

Модуль импортирует следующие классы из других модулей:

- `PrestaShop` из `.api`: Синхронный клиент API PrestaShop.
- `PrestaShopAsync` из `.api_async`: Асинхронный клиент API PrestaShop.
