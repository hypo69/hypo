# Модуль tinytroupe.__init__

## Обзор

Этот модуль инициализирует TinyTroupe, устанавливая необходимые библиотеки, читая конфигурацию, настраивая логгирование и корректируя вывод rich консоли для Jupyter.


## Импорты

- `os`: Для работы с операционной системой.
- `logging`: Для логгирования.
- `configparser`: Для работы с конфигурационными файлами.
- `rich`: Для красивой консольной отрисовки.
- `rich.jupyter`: Для интеграции rich с Jupyter.
- `sys`: Для работы со средой выполнения.
- `utils`: Для использования вспомогательных функций из модуля `utils`.


## Функции

### `read_config_file`

**Описание**: Читает конфигурационный файл и возвращает объект `configparser.ConfigParser`.

**Возвращает**:
- `configparser.ConfigParser`: Объект, содержащий конфигурацию.

**Вызывает исключения**:
- `FileNotFoundError`: Если конфигурационный файл не найден.
- `configparser.Error`: Если возникли проблемы при чтении файла.



### `pretty_print_config`

**Описание**: Выводит конфигурацию в красивом формате.

**Параметры**:
- `config (configparser.ConfigParser)`: Конфигурация, которую нужно вывести.

**Возвращает**:
- `None`


### `start_logger`

**Описание**: Инициализирует логгер на основе конфигурации.

**Параметры**:
- `config (configparser.ConfigParser)`: Конфигурация, которая содержит настройки логгера.

**Возвращает**:
- `None`



### `inject_html_css_style_prefix`

**Описание**: Добавляет префикс стилей CSS для отступов в HTML-выводе rich.

**Параметры**:
- `html_format (str)`: Формат HTML.
- `css_style_prefix (str)`: Префикс стилей CSS.

**Возвращает**:
- `str`: Формат HTML со стилями.



## AI Предупреждения

Модуль содержит предупреждение о том, что TinyTroupe использует модели ИИ, которые могут генерировать неточную или неуместную информацию. Пользователям рекомендуется проверять генерируемый контент перед использованием.

## Дополнительные заметки

- Модуль использует вспомогательный модуль `utils`.
- Добавляет текущий путь в `sys.path`, что позволяет импортировать модуль `utils`.
- Исправляет ошибку в библиотеке rich, чтобы избежать отступов в Jupyter.
