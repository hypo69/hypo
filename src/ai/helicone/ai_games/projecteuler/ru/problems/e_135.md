# Задача No 135
## Разность квадратов

## Условия
Дано уравнение:
x² − y² − z² = n

Найти количество значений n, 0 < n < 1000000, для которых существует ровно десять различных решений (x, y, z), где x, y и z — натуральные числа.

Пример:

Например, существует 5 решений уравнения x² − y² − z² = 27:

(x, y, z) = (17, 13, 5), (17, 5, 13), (15, 9, 3), (15, 3, 9) и (9, 7, 1).

В то время как для n = 11 существует только одно решение: (6, 5, 2).
