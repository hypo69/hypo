# Задача No 718
## Обобщенные числа трибоначчи
## Условия
Обобщенные числа Трибоначчи T(s, n) определяются следующим образом:

*   T(s, 0) = 0
*   T(s, 1) = 1
*   T(s, 2) = 1
*   T(s, n) = T(s, n-1) + T(s, n-2) + T(s, n-3)  если n > 2
*   T(s, n) = T(s, n - 1) + T(s, n - 2) + T(s, n - 3) + ... + T(s, n - s) если n > s

Примеры:

*   Обычная последовательность Трибоначчи получается при s = 3.
*   T(2, n) - это последовательность Фибоначчи.
*   T(4, 5) = T(4, 4) + T(4, 3) + T(4, 2) + T(4, 1) = 6 + 4 + 1 + 1 = 12

Пусть S(k) обозначает сумму T(n, n) для n от 0 до k.
Итак, S(5) = T(0,0) + T(1, 1) + T(2, 2) + T(3, 3) + T(4, 4) + T(5, 5) = 0 + 1 + 2 + 4 + 6 + 12 = 25.

Найдите S(10<sup>12</sup>) по модулю 1 000 000 007.
