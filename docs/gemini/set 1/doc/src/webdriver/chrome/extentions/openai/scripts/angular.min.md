##  Функции

### `ve`

**Описание**:  Функция, которая обрабатывает настройки Angular, устанавливая `objectMaxDepth` и `urlErrorParamsEnabled`.

**Параметры**:

* `a` (object): Объект с настройками, содержащими `objectMaxDepth` (целое число) и `urlErrorParamsEnabled` (boolean).

**Возвращает**:

* `object`:  Объект `Xb` с обновлёнными настройками. Возвращает `undefined` если `a` не является объектом.


### `Yb`

**Описание**:  Проверяет, является ли `a` числом, и, если да, то положительным.

**Параметры**:

* `a` (any): Любое значение.

**Возвращает**:

* `boolean`:  Возвращает `true`, если `a` положительное число, иначе `false`.


### `F`

**Описание**:  Функция-заготовка для создания функций обработки ошибок, принимающих произвольное количество аргументов.

**Параметры**:

* `a` (string, optional): Префикс для имени ошибки. По умолчанию `null`.
* `b` (class, optional):  Класс для создания ошибок. По умолчанию `Error`.

**Возвращает**:

* `function`:  Функция, создающая ошибки с деталями, предоставленными в качестве аргументов.


### `za`

**Описание**: Проверяет, является ли переданный аргумент массивом или коллекцией.

**Параметры**:

* `a` (any): Входной аргумент.

**Возвращает**:

* `boolean`:  Возвращает `true`, если аргумент является массивом или коллекцией (с свойством `length` и элементами), `false` в противном случае.


### `r`

**Описание**:  Функция для перебора элементов массива, объекта или итерируемых объектов.


**Параметры**:

* `a` (any): Массив, объект или итерируемый объект.
* `b` (function): Функция, которая будет вызываться для каждого элемента.
* `d` (any, optional):  Контекст, в котором будет вызываться функция `b`.


**Возвращает**:

* `any`:  Возвращает исходный объект `a`.
