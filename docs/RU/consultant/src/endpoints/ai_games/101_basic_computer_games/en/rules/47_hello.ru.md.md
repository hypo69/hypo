### Название игры: **HELLO** (Приветствие)
===================================================

#### Описание
**HELLO** — это простая игра, в которой игрок вводит имя, а программа приветствует его с персонализированным сообщением. Игрок может повторить игру, вводя новое имя, или завершить программу.

---

### Пошаговая инструкция для реализации
-------------------------------------

#### 1. **Инициализация игры**
   - При запуске игры программа приветствует игрока и объясняет правила:
     ```
     Привет! Введи своё имя, и я тебя поприветствую.
     ```
   - Программа ожидает, что пользователь введёт своё имя. После этого программа выводит персонализированное сообщение, например:
     ```
     Привет, [имя]! Рад познакомиться!
     ```

#### 2. **Основной процесс игры**
   - **Ввод имени:**
     1. Игрок вводит своё имя. Программа проверяет, чтобы имя было непустым.
     2. Если имя пустое или состоит только из пробелов, программа выводит ошибку и запрашивает ввод повторно.

   - **Приветствие:**
     1. После ввода имени программа выводит приветственное сообщение, используя имя пользователя.

#### 3. **Продолжение игры**
   - После того как программа поприветствовала игрока, она задаёт вопрос:
     ```
     Хотите ввести другое имя? (да/нет)
     ```
   - Если игрок выбирает "да", программа снова запрашивает ввод имени и приветствует его.
   - Если игрок выбирает "нет", программа завершает работу:
     ```
     Спасибо за игру! До свидания!
     ```

#### 4. **Завершение игры**
   - Когда игрок завершает игру, программа прощается с ним и завершает выполнение.

"""
Модуль для реализации игры HELLO (Приветствие).
================================================

Этот модуль реализует простую игру, в которой программа приветствует пользователя по имени.

Пример использования
--------------------

.. code-block:: python

    if __name__ == "__main__":
        play_hello_game()
"""
from src.logger.logger import logger

def play_hello_game():
    """
    Запускает игру "Приветствие".

    Функция инициализирует игру, запрашивает имя пользователя, приветствует его
    и предлагает повторить или завершить игру.
    """
    print("Привет! Введи своё имя, и я тебя поприветствую.")

    while True:
        name = input("> ")
        # Проверка ввода имени пользователя на пустоту или наличие пробелов
        if not name or name.strip() == "":
            print("Ошибка: Имя не может быть пустым. Пожалуйста, введите имя.")
            logger.error("Пользователь ввел пустое имя")
            continue

        print(f"Привет, {name}! Рад познакомиться!")

        while True:
            another_name = input("Хотите ввести другое имя? (да/нет): ").lower()
            # Проверка выбора пользователя: продолжать игру или нет
            if another_name == "да":
                print("Введи своё имя, и я тебя поприветствую.")
                break
            elif another_name == "нет":
                print("Спасибо за игру! До свидания!")
                return
            else:
                print("Пожалуйста, введите 'да' или 'нет'.")
                logger.error(f'Некорректный ввод пользователя: {another_name}')
                continue
if __name__ == "__main__":
    play_hello_game()
