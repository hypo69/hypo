# Модуль для работы с помощником по программированию

## Обзор

Этот модуль содержит класс `CodeAssistant`, предназначенный для взаимодействия с различными моделями ИИ, такими как Google Gemini и OpenAI, для задач обработки кода.  Он позволяет анализировать и генерировать документацию для кода.

## Примеры использования

```python
from ai.prompts.developer.code_assistant import CodeAssistant

assistant = CodeAssistant(role='code_checker', lang='ru', model=['gemini'])
result = assistant.process_files(files=['file1.py', 'file2.py'], options={'verbose': True})
print(result)
```

## Классы

### `CodeAssistant`

**Описание**: Класс для работы с помощником по программированию.

**Атрибуты**:

- `role` (str): Роль помощника (например, 'code_checker').
- `lang` (str): Язык, который будет использовать помощник (например, 'ru').
- `model` (list): Список используемых моделей ИИ (например, ['gemini']).

**Методы**:

#### `process_files`

**Описание**: Метод для обработки файлов кода. Анализирует и обрабатывает файлы кода, используя выбранные модели ИИ.

**Параметры**:

- `files` (list): Список файлов для обработки.
- `options` (dict, необязательно): Дополнительные параметры для настройки обработки.  По умолчанию `None`.


**Возвращает**:
- list: Список результатов обработки.  Возвращает список данных, полученных в результате анализа файлов.

**Вызывает исключения**:

- `FileNotFoundError`: Если файл не найден.
- `ValueError`: Если передан неверный тип данных или формат данных.
- `Exception`: Для других неожиданных ошибок.
