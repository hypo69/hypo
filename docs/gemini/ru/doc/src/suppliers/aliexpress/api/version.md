# Модуль hypotez/src/suppliers/aliexpress/api/version.py

## Обзор

Данный модуль содержит информацию о версии API AliExpress. Он определяет переменные `__version__`, `__name__`, `__doc__`, `__details__`, `__annotations__`, `__author__`.

## Переменные

### `__version__`

**Описание**: Переменная, хранящая версию модуля или пакета.

**Тип**: str

**Значение**: "3.12.0.0.0.4"


### `__name__`

**Описание**: Переменная, содержащая имя модуля. Если скрипт запускается напрямую, значение будет `"__main__"`.

**Тип**: str


### `__doc__`

**Описание**: Строка документации модуля.

**Тип**: str


### `__details__`

**Описание**: Переменная, вероятно, содержащая дополнительные детали о модуле, но точное назначение зависит от конкретного модуля или пакета.

**Тип**: str

**Значение**: "Details about version for module or class"


### `__annotations__`

**Описание**: Содержит аннотации типов для переменных и функций в модуле.

**Тип**: <Словарь типов>

**Примечание**:  В представленном коде нет примеров аннотаций типов.


### `__author__`

**Описание**: Имя(на) автора(ов) модуля.

**Тип**: str

**Значение**: 'hypotez'
