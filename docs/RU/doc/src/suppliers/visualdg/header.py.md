## Функции

### `set_project_root`

**Описание**:
Находит корневой каталог проекта, начиная с текущего каталога файла, ища вверх и останавливаясь на первом каталоге, содержащем любой из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple): Имена файлов или каталогов для идентификации корня проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневому каталогу, если он найден, иначе каталог, где находится скрипт.

## Переменные

### `MODE`

**Описание**:
Режим работы приложения. По умолчанию установлено значение `dev`.

### `__root__`

**Описание**:
Путь к корневому каталогу проекта. Инициализируется функцией `set_project_root`.

### `settings`

**Описание**:
Словарь настроек проекта, загружаемый из файла `settings.json`. Если файл не найден или не удается декодировать JSON, значение остается `None`.

### `doc_str`
**Описание**:
Строка, содержащая содержимое файла `README.MD`. Если файл не найден или не удается декодировать, значение остается `None`.

### `__project_name__`
**Описание**:
Имя проекта, извлекаемое из настроек или использующее значение по умолчанию `hypotez`.

### `__version__`

**Описание**:
Версия проекта, извлекаемая из настроек или использующая значение по умолчанию `''`.

### `__doc__`
**Описание**:
Строка документации проекта, извлекаемая из файла `README.MD` или использующая значение по умолчанию `''`.

### `__details__`
**Описание**:
Строка подробной документации проекта.  Используется значение по умолчанию `''`.

### `__author__`
**Описание**:
Автор проекта, извлекаемый из настроек или использующий значение по умолчанию `''`.

### `__copyright__`
**Описание**:
Авторские права проекта, извлекаемые из настроек или использующие значение по умолчанию `''`.

### `__cofee__`
**Описание**:
Текст с призывом угостить разработчика чашечкой кофе для повышения энтузиазма, извлекаемый из настроек или использующий значение по умолчанию `"Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69"`.
