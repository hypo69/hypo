# Модуль `start_event.py`

## Обзор

Модуль `start_event.py` предназначен для автоматической публикации мероприятий в группах Facebook. Он использует веб-драйвер для взаимодействия с Facebook и выполняет запланированные публикации в соответствии с заданными параметрами.

## Подробней

Этот модуль является частью системы автоматизации маркетинга `hypotez`. Он предназначен для автоматической публикации рекламных объявлений и мероприятий в группах Facebook.  
Модуль использует список файлов JSON, содержащих информацию о группах и событиях, и выполняет их публикацию с использованием класса `FacebookPromoter`.  
Основная цель модуля - автоматизировать процесс продвижения мероприятий в Facebook, экономя время и ресурсы маркетологов.

## Переменные

- `d`: Объект класса `Driver`, используемый для управления браузером Chrome.
- `filenames` (list[str]): Список имен файлов, содержащих информацию о группах Facebook.
- `excluded_filenames` (list[str]): Список имен файлов, которые следует исключить из обработки.
- `events_names` (list): Список имен событий для публикации.
- `promoter`: Объект класса `FacebookPromoter`, используемый для публикации событий в группах Facebook.

## Классы

### `FacebookPromoter`

**Описание**: Класс `FacebookPromoter` предназначен для автоматической публикации мероприятий в группах Facebook. Он использует веб-драйвер для взаимодействия с Facebook и выполняет запланированные публикации в соответствии с заданными параметрами.

**Как работает класс**:
1.  Инициализируется с драйвером веб-браузера и списком файлов, содержащих информацию о группах.
2.  Метод `run_events` выполняет публикацию событий в группах Facebook.

## Функции

### `time.strftime`

```python
time.strftime('%H:%M:%S')
```

**Описание**: Форматирует текущее время в строку в формате "ЧЧ:ММ:СС".

**Как работает функция**:
Функция `strftime` из модуля `time` используется для форматирования времени в заданном формате. В данном случае, она форматирует текущее время в формат "ЧЧ:ММ:СС".

**Параметры**:
- `format` (str): Формат, в котором требуется представить время. В данном случае, '%H:%M:%S'.

**Возвращает**:
- `str`: Строка с текущим временем в формате "ЧЧ:ММ:СС".

**Примеры**:

```python
current_time = time.strftime('%H:%M:%S')
print(f"Текущее время: {current_time}")
```

### `promoter.run_events`

```python
promoter.run_events(events_names = events_names, group_file_paths = filenames)
```

**Описание**: Запускает публикацию событий в группах Facebook.

**Как работает функция**:
Метод `run_events` класса `FacebookPromoter` выполняет публикацию событий в группах Facebook. Он принимает список имен событий и список файлов с информацией о группах.

**Параметры**:
- `events_names` (list): Список имен событий для публикации.
- `group_file_paths` (list[str]): Список путей к файлам, содержащим информацию о группах Facebook.

**Примеры**:

```python
promoter.run_events(events_names = ["event1", "event2"], group_file_paths = ["groups1.json", "groups2.json"])
```

time.sleep(7200)

**Параметры**:
- `seconds` (int): Количество секунд для приостановки выполнения программы.

time.sleep(3600)  # Приостановка на 1 час
