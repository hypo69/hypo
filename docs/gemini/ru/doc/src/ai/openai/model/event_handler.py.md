# Модуль `event_handler`

## Обзор

Модуль `event_handler` предоставляет класс `EventHandler`, который используется для обработки событий, поступающих из потока ответов OpenAI Assistant API. Этот модуль определяет, как следует обрабатывать различные типы событий, такие как создание текста, дельта текста, создание вызова инструмента и дельта вызова инструмента.

## Классы

### `EventHandler`

**Описание**: Класс `EventHandler` наследуется от `AssistantEventHandler` и переопределяет методы для обработки событий от OpenAI Assistant API.

**Методы**:
- [`on_text_created`](#on_text_created)
- [`on_text_delta`](#on_text_delta)
- [`on_tool_call_created`](#on_tool_call_created)
- [`on_tool_call_delta`](#on_tool_call_delta)

## Функции

### `on_text_created`

**Описание**: Метод вызывается при создании нового текстового блока.

**Параметры**:
- `text` (`Text`): Объект `Text`, представляющий созданный текстовый блок.

**Возвращает**:
- `None`: Метод ничего не возвращает.

### `on_text_delta`

**Описание**: Метод вызывается при получении дельты текстового блока.

**Параметры**:
- `delta` (`TextDelta`): Дельта текстового блока.
- `snapshot` (`Text`): Снимок текущего состояния текстового блока.

**Возвращает**:
- `None`: Метод ничего не возвращает.

### `on_tool_call_created`

**Описание**: Метод вызывается при создании нового вызова инструмента.

**Параметры**:
- `tool_call` (`ToolCall`): Объект `ToolCall`, представляющий созданный вызов инструмента.

**Возвращает**:
- `None`: Метод ничего не возвращает.

### `on_tool_call_delta`

**Описание**: Метод вызывается при получении дельты вызова инструмента.

**Параметры**:
- `delta` (`ToolCallDelta`): Дельта вызова инструмента.
- `snapshot` (`ToolCall`): Снимок текущего состояния вызова инструмента.

**Возвращает**:
- `None`: Метод ничего не возвращает.
