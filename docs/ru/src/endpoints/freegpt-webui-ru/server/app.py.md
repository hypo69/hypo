# Документация для модуля Flask приложения

## Обзор

Модуль инициализирует Flask-приложение с указанием папки для шаблонов.

## Подробнее

Этот модуль создает экземпляр Flask приложения, используемого для обработки веб-запросов. Важно отметить, что `template_folder` указывает на директорию, где хранятся HTML-шаблоны, используемые для рендеринга веб-страниц.

## Переменные

### `app`

-   **Описание**: Экземпляр Flask приложения.
-   **Тип**: `Flask`
-   **Параметры**:
    -   `__name__` (str): Имя текущего модуля.
    -   `template_folder` (str): Путь к папке с HTML-шаблонами ('./../client/html').
-   **Принцип работы**:
    -   Создается экземпляр класса `Flask`, который является основным объектом для веб-приложения.
    -   `__name__` используется Flask для определения корневой директории приложения.
    -   `template_folder` указывает на директорию, где Flask будет искать HTML-шаблоны.
