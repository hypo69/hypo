# Документация `package-lock.json`

## Обзор

Этот файл `package-lock.json` содержит информацию о конкретных версиях пакетов и их зависимостях, которые используются в проекте `chatgpt-telegram`. Этот файл гарантирует, что при установке пакетов в разных средах будут использоваться одни и те же версии зависимостей, что устраняет проблемы совместимости.
