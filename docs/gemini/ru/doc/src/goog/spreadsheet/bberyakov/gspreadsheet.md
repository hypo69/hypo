# Модуль gspreadsheet.py

## Обзор

Модуль `gspreadsheet.py` предоставляет класс `GSpreadsheet` для работы с таблицами Google Sheets. Он позволяет получать доступ к существующим таблицам по ID или названию, создавать новые таблицы, а также управлять доступом к ним. Модуль использует библиотеку `gspread` для взаимодействия с API Google Sheets.


## Классы

### `GSpreadsheet`

**Описание**: Класс `GSpreadsheet` наследуется от класса `Spreadsheet` и предоставляет методы для работы с таблицами Google Sheets.  Предназначен для взаимодействия с таблицами, создания новых и доступа к существующим.


**Методы**:

#### `__init__`

**Описание**: Инициализирует объект `GSpreadsheet`.

**Параметры**:
- `s_id` (str, опционально): ID таблицы Google Sheets.
- `s_title` (str, опционально): Название таблицы Google Sheets.
- `*args`: Дополнительные аргументы.
- `**kwards`: Дополнительные ключевые аргументы.

**Возвращает**:
- `None`


#### `get_project_spreadsheets_dict`

**Описание**: Возвращает словарь с информацией о таблицах Google Sheets.

**Параметры**:
- `self`: Экземпляр класса `GSpreadsheet`.

**Возвращает**:
- `dict`: Словарь с информацией о таблицах.


#### `get_by_title`

**Описание**: Возвращает объект таблицы по заданному названию. Если таблицы с таким названием не существует, создает новую.

**Параметры**:
- `self`: Экземпляр класса `GSpreadsheet`.
- `sh_title` (str, опционально): Название таблицы. По умолчанию "New Spreadsheet".

**Возвращает**:
- `None`


#### `get_by_id`

**Описание**: Возвращает объект таблицы по заданному ID.

**Параметры**:
- `self`: Экземпляр класса `GSpreadsheet`.
- `sh_id` (str): ID таблицы.

**Возвращает**:
- `Spreadsheet`: Объект таблицы.


#### `get_all_spreadsheets_for_current_account`

**Описание**: Возвращает список всех таблиц для текущего аккаунта.

**Параметры**:
- `self`: Экземпляр класса `GSpreadsheet`.

**Возвращает**:
- `None`


## Модули

### `global_settingspread`

- Данный модуль содержит определения для классов,  используемых в `gspreadsheet.py`, таких как `Spreadsheet` и `service_account`.

## Примечания

- Класс `GSpreadsheet` использует `service_account` для аутентификации.  Файл секрета  `goog\\onela-hypotez-1aafa5e5d1b5.json` должен быть в правильном каталоге.
- Методы `get_by_id` и `get_by_title` предполагают, что соответсвующие таблицы уже созданы, или что методы из `global_settingspread`  обеспечивают доступ к таким таблицам.
- Документация некоторых методов (`__init__`, `get_project_spreadsheets_dict`, `get_all_spreadsheets_for_current_account`) может быть дополнена более подробными объяснениями.
-  В коде присутствуют комментарии, которые можно использовать для улучшения документации.
- Код содержит примеры вызова методов, которые можно использовать для понимания использования.
