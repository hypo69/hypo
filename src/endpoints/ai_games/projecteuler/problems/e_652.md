# Задача No 652
## Взаимные числа Фибоначчи

## Условия

Рассмотрим последовательность Фибоначчи F(n), где F(0) = 0, F(1) = 1, и F(n) = F(n-1) + F(n-2) для n > 1.

Пусть gcd(a,b) обозначает наибольший общий делитель чисел a и b.

Определим последовательность A(n) следующим образом:
    * A(0) = 1
    * A(n) = gcd(F(A(n-1)), n) для n > 0

Первые несколько элементов последовательности A:
    1, 1, 1, 3, 1, 5, 1, 1, 3, 3, 1, 1, 1, 1, 15, 5, 1, 1, 9, 1, 1, 7, 1, 1, 1, 1, 1, 1, 3, 15, 1, 1, 13, 1, 1, 1, 1, 1, 1, 1, 1, ...

Определим S(n) как сумму первых n элементов последовательности A, то есть, 
$S(n) = \sum_{i=0}^{n-1} A(i)$.

Например, S(10) = 20.

Найдите S(10^18) mod 1 000 000 007.
