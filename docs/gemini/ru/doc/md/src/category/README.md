# Модуль категоризации данных

## Обзор

Этот модуль предназначен для категоризации данных о продуктах, полученных от поставщиков.  Он содержит класс `Product` для представления данных о продукте и класс `Supplier` для представления поставщика.

## Оглавление

* [Классы](#классы)
    * [Product](#product)
    * [Supplier](#supplier)

## Классы

### `Product`

**Описание**: Представляет данные о продукте.

**Атрибуты**:
- `product_id` (int): Идентификатор продукта.
- `name` (str): Название продукта.
- `category` (str): Категория продукта.
- `supplier_id` (int): Идентификатор поставщика.

**Методы**:
- `get_category()`: Возвращает категорию продукта.
  ```python
  def get_category(self) -> str:
      """
      Возвращает категорию продукта.

      Returns:
          str: Категория продукта.
      """
      return self.category
  ```

### `Supplier`

**Описание**: Представляет поставщика.

**Атрибуты**:
- `supplier_id` (int): Идентификатор поставщика.
- `name` (str): Название поставщика.
- `products` (list[Product]): Список продуктов, поставленных данным поставщиком.


**Методы**:
- `add_product(product: Product)`: Добавляет продукт к списку продуктов поставщика.
  ```python
  def add_product(self, product: Product) -> None:
      """
      Добавляет продукт к списку продуктов поставщика.

      Args:
          product (Product): Продукт, который нужно добавить.
      
      Raises:
          TypeError: Если переданный объект не является классом Product.
      """
      if not isinstance(product, Product):
          raise TypeError("Переданный объект не является Product")
      self.products.append(product)
  ```
- `get_products()`: Возвращает список продуктов поставщика.
  ```python
  def get_products(self) -> list[Product]:
      """
      Возвращает список продуктов, поставленных данным поставщиком.

      Returns:
          list[Product]: Список продуктов.
      """
      return self.products
  ```
