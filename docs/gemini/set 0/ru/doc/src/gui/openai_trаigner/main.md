# Модуль `hypotez/src/gui/openai_trаigner/main.py`

## Обзор

Данный модуль предоставляет интерфейс для взаимодействия с различными моделями искусственного интеллекта, такими как ChatGPT, Gemini и Claude. Он реализован с использованием PyQt6 и позволяет пользователю загружать URL-адреса, использовать Google-сервисы (Gmail, Google Docs и т.д.) и выбирать модели для работы. Модуль также поддерживает минимизацию окна в системном трее.

## Классы

### `AssistantMainWindow`

**Описание**: Главный класс приложения, реализующий графический интерфейс.

**Методы**:

#### `__init__`

**Описание**: Инициализирует главные элементы приложения: браузер, верхнюю панель инструментов, системный трей, меню сервисов Google и моделей.

**Параметры**:
- (self): Экземпляр класса.


#### `ask_for_browser`

**Описание**: Запрашивает у пользователя выбор браузера по умолчанию.

**Параметры**:
- (self): Экземпляр класса.

**Возвращает**:
- str: Имя выбранного браузера ('Chrome', 'Firefox', 'Edge'), или None при ошибке.


#### `load_url`

**Описание**: Загружает URL-адрес в браузере.

**Параметры**:
- (self, url: str = None): URL-адрес для загрузки. Если None, то используется значение из поля ввода.

**Возвращает**:
- None


#### `hide_to_tray`

**Описание**: Скрывает главное окно приложения в системном трее.

**Параметры**:
- (self): Экземпляр класса.

**Возвращает**:
- None


#### `quit_app`

**Описание**: Завершает работу приложения.

**Параметры**:
- (self): Экземпляр класса.

**Возвращает**:
- None

#### `closeEvent`

**Описание**: Обрабатывает событие закрытия окна. Предотвращает полное закрытие приложения, скрывая окно в трее.

**Параметры**:
- (self, event): Событие закрытия.

**Возвращает**:
- None



## Переменные

### `MODE`

**Описание**:  Устанавливает режим работы приложения. По умолчанию `'dev'`.

## Замечания

- Методы `ask_for_browser`, `load_url`, `hide_to_tray`, `quit_app` и `closeEvent` реализуют ключевые функции приложения, такие как выбор браузера, загрузка URL, скрытие в трее, выход из приложения и обработка закрытия окна.
- Приложение использует системный трей для минимизации и сохранения возможности взаимодействия с пользователем, даже когда приложение не отображается на экране.
- Код содержит примеры загрузки различных URL-адресов Google-сервисов. Необходимо заменить тестовые URL для Gemini и Claude на реальные адреса.
- Важно убедиться в корректности путей к профилям браузеров.
