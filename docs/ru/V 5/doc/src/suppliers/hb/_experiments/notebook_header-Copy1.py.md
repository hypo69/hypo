## Функции

### `start_supplier`

```python
def start_supplier(supplier_prefix, locale):
    """ Старт поставщика """
    if not supplier_prefix and not locale: return "Не задан сценарий и язык"
    
    params: dict = \
    {
        'supplier_prefix': supplier_prefix,
        'locale': locale
    }
    
    return Supplier(**params)
```

**Описание**: Функция для запуска поставщика.

**Как работает функция**:
Функция `start_supplier` принимает префикс поставщика и локаль в качестве аргументов и возвращает экземпляр класса `Supplier`, созданный с использованием этих параметров. Если префикс поставщика или локаль не заданы, функция возвращает сообщение об ошибке "Не задан сценарий и язык".

**Параметры**:
- `supplier_prefix` (str): Префикс поставщика.
- `locale` (str): Локаль.

**Возвращает**:
- `Supplier | str`: Экземпляр класса `Supplier` или строку с сообщением об ошибке, если параметры не заданы.

**Примеры**:
```python
start_supplier('hb', 'ru')
# Возвращает экземпляр класса Supplier с параметрами supplier_prefix='hb' и locale='ru'

start_supplier('', '')
# Возвращает строку "Не задан сценарий и язык"
