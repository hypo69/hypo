# Документация `morlevi_categories_memory.json`

## Обзор

Этот файл содержит JSON-конфигурацию для сценариев, связанных с категориями памяти (RAM) для поставщика Morlevi. Каждый сценарий описывает конкретный тип RAM (например, CORSAIR DIMM DDR4 16GB) и содержит информацию о бренде, URL-адресе, активном статусе, условиях и категориях PrestaShop.

## Структура JSON

Файл JSON представляет собой словарь, содержащий один ключ `scenarios`. Значением этого ключа является словарь, где каждый ключ – это название сценария (например, "CORSAIR DIMM DDR 4 16GB"), а значение – словарь с деталями этого сценария.

```json
{
  "scenarios": {
    "Название сценария": {
      "brand": "Бренд",
      "url": "URL",
      "checkbox": false,
      "active": true,
      "condition":"new",
      "presta_categories": {
        "template": {
          "ключ_бренда": "значение_категории"
        }
      }
    },
    "...": {
     "...": "..."
    }
  }
}
```

## Описание полей

### `scenarios`

Этот раздел содержит все сценарии для категорий памяти.

#### Название сценария

Каждый ключ в словаре `scenarios` представляет собой название конкретного сценария, например, `CORSAIR DIMM DDR 4 16GB`.

##### `brand`

**Описание**: Название бренда производителя RAM.

**Тип**: `str`

**Пример**: `"CORSAIR"`, `"CRUCIAL"`, `"G.SKILL"`, `"KINGSTON"`

##### `url`

**Описание**: URL-адрес страницы товара на сайте поставщика Morlevi.

**Тип**: `str`

**Пример**: `https://www.morlevi.co.il/Cat/149?p_315=20&p_45=100&p_44=93&sort=datafloat2%2Cprice&keyword=`

##### `checkbox`

**Описание**: Флаг, указывающий, отмечен ли данный сценарий. В текущей конфигурации всегда `false`.

**Тип**: `bool`

**Значение**: `false`

##### `active`

**Описание**: Флаг, указывающий, активен ли данный сценарий. В текущей конфигурации всегда `true`.

**Тип**: `bool`

**Значение**: `true`

##### `condition`
**Описание**: Условие товара. В текущей конфигурации всегда `new`

**Тип**: `str`

**Значение**: `new`

##### `presta_categories`

**Описание**: Раздел, содержащий информацию о категориях PrestaShop.

**Тип**: `dict`

###### `template`
**Описание**: Шаблон категорий PrestaShop, связывающий бренд с названием категории

**Тип**: `dict`

**Пример**:

```json
{
    "template": {
        "corsair": "DIMM DDR 4 16GB"
      }
}
```
