# Модуль `hypotez/src/suppliers/visualdg/__init__.py`

## Обзор

Модуль `hypotez/src/suppliers/visualdg` представляет собой пакет, включающий в себя функциональность для работы с поставщиком VisualDG. 
Этот модуль разработан для использования на платформах Windows и Unix.
