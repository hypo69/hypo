# Задача No 258
## Загадка битов Фибоначчи

## Условия
Рассмотрим последовательность битов, определенную следующим образом:

$s_0 = 1$

$s_1 = 1$

$s_n = (s_{n-1} + s_{n-2}) \bmod 2$, для $n > 1$

Таким образом, первые несколько элементов последовательности: 1, 1, 0, 1, 1, 0, 1, 1, 0, 1, 1, 0, ...

Теперь рассмотрим последовательность целых чисел, полученных путем интерпретации смежных групп битов из последовательности s в двоичном виде.

$g_1 = (s_1)$
$g_2 = (s_2, s_3)$
$g_3 = (s_4, s_5, s_6)$
$g_4 = (s_7, s_8, s_9, s_{10})$
...
Так,
$g_1 = 1_2 = 1$
$g_2 = 10_2 = 2$
$g_3 = 110_2 = 6$
$g_4 = 1101_2 = 13$

...

Обозначим через $F(n)$ n-ый член последовательности $g$.

Вам дано, что $F(10) = 4181$

Найдите $F(10^{18}) \bmod 10^9$.
