# Модуль `bot_pyogram.py`

## Обзор

Модуль предоставляет простой Telegram-бот, созданный с использованием библиотеки Pyrogram. Бот отвечает на команду `/start` приветственным сообщением и эхом повторяет все остальные текстовые сообщения, которые он получает. Модуль содержит функциональность для создания и запуска Telegram-бота.

## Подробней

Этот модуль является точкой входа для Telegram-бота. Он использует библиотеку Pyrogram для обработки входящих сообщений и отправки ответов. Бот инициализируется с использованием API ID, API Hash и токена, полученных из переменных окружения. Он обрабатывает команду `/start`, отправляя приветственное сообщение, и эхом повторяет все остальные текстовые сообщения.
