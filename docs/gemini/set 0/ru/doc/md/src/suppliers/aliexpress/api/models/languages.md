# Модуль `hypotez/src/suppliers/aliexpress/api/models/languages.py`

## Обзор

Данный модуль определяет класс `Language`, содержащий константы для различных языков.  Он предназначен для работы с языковыми параметрами при взаимодействии с API AliExpress.

## Классы

### `Language`

**Описание**: Класс `Language` содержит константы, представляющие различные языки.

- `TR`: Турецкий язык (`'TR'`).
- `DE`: Немецкий язык (`'DE'`).
- `HE`: Иврит (`'HE'`).
