# Задача No 363
## Бейсбольные эллипсы
## Условия

Бейсбольный эллипс представляет собой кривую, которая определяется четырьмя дугами окружностей. Например, эллипс со следующими параметрами:

-   `w` = 100
-   `h` = 50
-   `r` = 20

определяется следующими дугами:

-   Две четверти окружности с радиусом `r` вокруг точек `(r, h)` и `(w-r, h)`
-   Две половины окружности с радиусом `r` вокруг точек `(r, 0)` и `(w-r, 0)`
-   Две прямые линии, соединяющие соответствующие дуги.

Отрезок прямой линии между `(r, h)` и `(w-r, h)` имеет длину `w-2r`.
Отрезок прямой линии между `(r, 0)` и `(w-r, 0)` имеет длину `w-2r`.
Отрезок прямой линии между `(r, 0)` и `(r, h)` имеет длину `h`.
Отрезок прямой линии между `(w-r, 0)` и `(w-r, h)` имеет длину `h`.
Приведенный выше эллипс можно было бы описать с помощью координат точек `(20,0)`, `(80,0)`, `(20,50)` и `(80,50)`.

Пусть `E(w, h, r)` - площадь, окруженная эллипсом с параметрами `w`, `h` и `r`.

Например, `E(100, 50, 20)` = 4309.792919...

Пусть `S(a, b, c)` - сумма площадей `E(w, h, r)` для всех триплетов `(w, h, r)` целых чисел, таких, что:
-   `0 < w <= a`
-   `0 < h <= b`
-   `0 < r <= c`
-   `r <= w/2`
-   `r <= h`

Например:

`S(10, 5, 3) = 12139.766654...`
`S(100, 50, 20) = 10791573.76847...`

Найдите `S(10000, 5000, 2000)` и округлите результат до 10 десятичных знаков после запятой.
