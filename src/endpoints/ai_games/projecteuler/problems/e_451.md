# Задача No 451
## Круги, образуемые обратными числами по модулю
## Условия

Пусть $a$, $b$ и $n$ - целые числа.

Определим $R(a, b, n)$ как наименьшее целое число $k>0$ такое, что $a^k ≡ b \pmod n$. Если такого $k$ не существует, то $R(a, b, n)=0$.

Например, $R(3, 2, 7) = 6$.

Пусть $G(n)$ будет суммой всех значений $R(a, 1, n)$ для $1 \le a < n$ таких, что $\text{НОД}(a, n) = 1$.

Например, $G(7) = R(1, 1, 7) + R(2, 1, 7) + R(3, 1, 7) + R(4, 1, 7) + R(5, 1, 7) + R(6, 1, 7) = 1 + 3 + 6 + 3 + 6 + 2 = 21$.

Пусть $S(n)$ будет суммой значений $G(i)$ для $2 \le i \le n$.

Например, $S(5) = G(2) + G(3) + G(4) + G(5) = 1 + 2 + 6 + 4 = 13$.

Вычислите $S(10^8)$.
