# Модуль utils

## Обзор

Этот модуль содержит набор небольших, полезных утилит, предназначенных для упрощения повседневных задач программирования. Модуль включает инструменты для конвертации данных, работы с файлами и формата вывода. Это позволяет ускорить разработку, предоставляя простые и переиспользуемые функции.

## Утилиты конвертации

### `csv2dict`

**Описание**: Функция преобразует CSV-файл в словарь.

**Параметры**:

- `filepath` (str): Путь к CSV-файлу.

**Возвращает**:

- `dict`: Словарь, содержащий данные из CSV-файла. Возвращает `None`, если файл не найден или содержит ошибки.

**Вызывает исключения**:

- `FileNotFoundError`: Если файл не найден.
- `ValueError`: Если содержимое файла не соответствует ожидаемому формату.


### `dict2csv`

**Описание**: Функция преобразует словарь в CSV-файл.

**Параметры**:

- `data` (dict): Словарь данных.
- `filepath` (str): Путь к файлу CSV.
- `header` (Optional[list], optional): Заголовки CSV-файла. Если None, используется ключи словаря.

**Возвращает**:

- `bool`: True, если преобразование прошло успешно, False в противном случае.

**Вызывает исключения**:

- `TypeError`: Если входной параметр `data` не является словарем.


### `json2xls`

**Описание**: Функция преобразует JSON-файл в XLSX-файл.


**Параметры**:

- `filepath` (str): Путь к JSON-файлу.


**Возвращает**:

- `bool`: True, если преобразование прошло успешно, False в противном случае.


**Вызывает исключения**:

- `FileNotFoundError`: Если файл не найден.
- `json.JSONDecodeError`: Если содержимое файла не является валидным JSON.
- `ValueError`: Если содержимое файла не соответствует ожидаемому формату.


### `xls2dict`

**Описание**: Функция преобразует XLSX-файл в словарь.

**Параметры**:

- `filepath` (str): Путь к XLSX-файлу.

**Возвращает**:

- `dict`: Словарь, содержащий данные из XLSX-файла. Возвращает `None`, если файл не найден или содержит ошибки.

**Вызывает исключения**:

- `FileNotFoundError`: Если файл не найден.
- `ValueError`: Если содержимое файла не соответствует ожидаемому формату.



## Утилиты работы с файлами

### `read_text_file`

**Описание**: Функция считывает текст из файла.

**Параметры**:

- `filepath` (str): Путь к файлу.

**Возвращает**:

- `str`: Содержимое файла. Возвращает `None`, если файл не найден.

**Вызывает исключения**:

- `FileNotFoundError`: Если файл не найден.



### `save_text_file`

**Описание**: Функция сохраняет текст в файл.

**Параметры**:

- `filepath` (str): Путь к файлу.
- `text` (str): Текст для сохранения.

**Возвращает**:

- `bool`: True, если сохранение прошло успешно, False в противном случае.


**Вызывает исключения**:

- `TypeError`: Если `text` не является строкой.
