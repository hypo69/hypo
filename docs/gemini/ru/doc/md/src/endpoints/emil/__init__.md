# Модуль hypotez/src/endpoints/emil/__init__.py

## Обзор

Этот модуль предоставляет инициализацию для модуля `emil` в приложении `hypotez`. Он определяет константу `MODE` и импортирует класс `EmilDesign`.

## Константы

### `MODE`

**Описание**: Строковая константа, определяющая режим работы приложения.

**Значение**: `'dev'`


## Импорты

### `from .emil_design import EmilDesign`
