# Модуль `banners_grabber`

## Обзор

Модуль `banners_grabber` предназначен для сбора баннеров с сайта KSP. Он содержит функцию `get_banners`, которая должна выполнять эту задачу.

## Подробней

Этот модуль, расположенный в `src/suppliers/ksp/`, вероятно, является частью системы, отвечающей за сбор и обработку данных от поставщиков. В данном случае, он специализируется на KSP и, вероятно, используется для получения рекламных баннеров с их сайта. Функция `get_banners` должна реализовывать логику для парсинга сайта KSP и извлечения необходимых баннеров.

## Функции

### `get_banners`

```python
def get_banners():
    return True
```

**Описание**: Функция для получения баннеров с сайта KSP.

**Как работает функция**:
На данный момент функция `get_banners` просто возвращает `True`. В будущем здесь будет реализована логика для подключения к сайту KSP, извлечения баннеров и их обработки.

**Возвращает**:
- `bool`: Возвращает `True`. В будущем будет возвращать список баннеров или другую структуру данных, содержащую информацию о баннерах.

**Примеры**:
```python
result = get_banners()
print(result)  # Вывод: True
