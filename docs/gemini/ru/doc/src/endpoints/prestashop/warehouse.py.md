# Модуль warehouse для работы со складами в PrestaShop

## Обзор

Модуль `warehouse` предназначен для взаимодействия с API PrestaShop с целью управления информацией о складах. Он расширяет функциональность класса `PrestaShop`, предоставляя методы для работы со складами.

## Подробней

Этот модуль позволяет получать и управлять данными о складах в PrestaShop. Он использует API PrestaShop для выполнения запросов и обработки ответов, обеспечивая удобный интерфейс для работы со складскими данными.
