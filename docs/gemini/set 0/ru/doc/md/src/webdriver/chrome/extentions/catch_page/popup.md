# Модуль popup.js

## Обзор

Данный скрипт JavaScript обрабатывает событие клика на кнопку "Отправить URL" в всплывающем окне расширения Chrome. Он получает активную вкладку, извлекает ее URL и отправляет его в фоновый скрипт для дальнейшей обработки.

## Функции

### `addEventListener`

**Описание**:  Регистрирует обработчик события клика на элементе с id "sendUrlButton".

**Параметры**:
- `click` (function): Функция, которая выполняется при клике на элементе.

**Возвращает**:
- `null`: Данная функция не возвращает значение.


###  `chrome.tabs.query`

**Описание**:  Получает информацию об активной вкладке.

**Параметры**:
- `{ active: true, currentWindow: true }` (object): Объект с параметрами запроса. `active: true` означает запрос активной вкладки, а `currentWindow: true` - в текущем окне.

**Возвращает**:
- `tabs` (array): Массив объектов, содержащих информацию о найденных вкладках.


### `alert`

**Описание**:  Выводит диалоговое окно с сообщением.

**Параметры**:
- `"Hello, world!"` (string): Текст сообщения для отображения.


### `chrome.runtime.sendMessage`

**Описание**: Отправляет сообщение в фоновый скрипт расширения.

**Параметры**:
- `{ action: "sendUrl", url: activeTabUrl }` (object): Объект с данными для сообщения. `action` определяет действие, `url` —  передаваемый URL.
- `response` (function): Обработчик ответа от фонового скрипта.

**Возвращает**:
- `response` (object): Объект с данными, отправленными в ответ от фонового скрипта.


### Обработчик ответа (анонимная функция)

**Описание**: Обрабатывает ответ от фонового скрипта.

**Параметры**:
- `response` (object): Ответ от фонового скрипта.

**Возвращает**:
- `null`: Данная функция не возвращает значение.

**Обрабатываемые исключения**:
-  Не указаны, т.к. в данном примере нет явного блока обработки исключений (try-catch).


###  `if (response.status === "success")`

**Описание**: Проверяет статус ответа от фонового скрипта.

**Параметры**:
- `response.status` (string): Статус ответа от фонового скрипта.

**Возвращает**:
- `null`: Данная функция не возвращает значение.


### `alert("URL sent successfully!");`

**Описание**: Выводит диалоговое окно с сообщением о успешной отправке URL.


### `alert("Failed to send URL.");`

**Описание**: Выводит диалоговое окно с сообщением об ошибке при отправке URL.
