# Модуль try_xpath_check_frame.js

## Обзор

Этот JavaScript-файл (try_xpath_check_frame.js) определяет пространство имен `tryxpath`.  Он, скорее всего, является частью скрипта для проверки XPath в рамках расширения Firefox.

**Описание**:  Объект, в котором хранятся функции и переменные пространства имен `tryxpath`.
