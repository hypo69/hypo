# Документация компонента чата Gemini

## Обзор

Данный компонент реализует интерфейс чат-бота Gemini, позволяющий пользователю вводить запросы и получать ответы от модели.  Компонент использует React для отрисовки интерфейса и fetch для взаимодействия с API Gemini.

## Функции

### `App`

**Описание**:  Основной функциональный компонент, представляющий собой чат-ботовую форму.  Управляет состоянием ввода, списком сообщений и отправкой запросов.

**Состояние:**

- `input`: Текстовое поле для ввода сообщения пользователем.
- `messages`: Массив объектов, содержащих сообщения от пользователя и чат-бота. Каждый объект имеет поля `role` (роль: user или assistant) и `content` (содержание сообщения).

**Методы:**

- `sendMessage`: Асинхронная функция, обрабатывающая отправку сообщения.

    **Описание**: Отправляет сообщение на сервер (localhost:8000/api/chat) через POST запрос. Принимает пользовательский ввод, добавляет его в массив сообщений, отправляет запрос, обрабатывает ответ сервера и добавляет ответ чат-бота в массив сообщений.

    **Параметры**:
    - Нет явных параметров.

    **Возвращает**:
    - Нет значения возврата.

    **Обрабатывает исключения**:
    - `Error`: Выводит ошибку в консоль в случае сбоя запроса на сервер.


**Возвращает**:
- JSX-элемент, представляющий интерфейс чата, включающий область для отображения сообщений и форму для ввода сообщений.


## Использование

1.  Подключите React и ReactDOM в ваш проект.
2.  Создайте компонент `App` по приведенному коду.
3.  Создайте HTML-элемент с id `chat-app` на вашей странице, чтобы разместить компонент.
4.  Вызовите `ReactDOM.render(<App />, document.getElementById('chat-app'));` для отрисовки компонента.
5.  Настройте API на серверной стороне (localhost:8000/api/chat) для обработки запросов и получения ответов от модели Gemini.


## Структура

- `input`: Поле ввода для сообщений пользователя.
- `messages`: Массив отображаемых сообщений (пользователь/бот).
- `sendMessage`: Функция для отправки сообщений.
