# Модуль для создания клавиатур для Telegram Movie Bot

## Обзор

Модуль `keyboard.py` предназначен для создания и хранения inline-клавиатур, используемых в Telegram-боте для поиска фильмов. Он содержит определения для двух типов клавиатур: `find_movie` и `choice`. Клавиатуры создаются с использованием библиотеки `aiogram`.

## Подробней

Этот модуль предоставляет готовые объекты `InlineKeyboardMarkup`, которые могут быть использованы для добавления интерактивных кнопок в сообщения, отправляемые ботом. Это позволяет пользователям взаимодействовать с ботом, выбирая действия или указывая свои предпочтения.

## InlineKeyboardMarkup

### `find_movie`

**Описание**: Inline-клавиатура с кнопкой "Найти" для запуска поиска новых фильмов.

**Как работает класс**:
Клавиатура создается с одной кнопкой, текст которой "Найти". При нажатии на эту кнопку пользователь отправляет callback-запрос `new_movies`, который обрабатывается ботом для запуска процесса поиска фильмов.

**Параметры**:
- `inline_keyboard` (list): Список списков объектов `InlineKeyboardButton`, определяющий структуру клавиатуры.

**Примеры**:
```python
find_movie = InlineKeyboardMarkup(inline_keyboard=[
    [InlineKeyboardButton(text='Найти', callback_data='new_movies')]
])
```

### `choice`

**Описание**: Inline-клавиатура с кнопками "Сериал" и "Фильм" для выбора типа контента.

**Как работает класс**:
Клавиатура содержит две кнопки: "Сериал" и "Фильм". При нажатии на кнопку "Сериал" отправляется callback-запрос `series`, а при нажатии на кнопку "Фильм" - callback-запрос `film`. Это позволяет боту определить, какой тип контента интересует пользователя.

**Параметры**:
- `inline_keyboard` (list): Список списков объектов `InlineKeyboardButton`, определяющий структуру клавиатуры.

**Примеры**:
```python
choice = InlineKeyboardMarkup(inline_keyboard=[
    [InlineKeyboardButton(text='Сериал', callback_data='series'),
     InlineKeyboardButton(text='Фильм', callback_data='film')]
])
```
