# Модуль `hypotez/src/utils/string/html_escapes.py`

## Обзор

Модуль `html_escapes.py` содержит словарь, отображающий символы UTF-8 в их HTML-экранированные эквиваленты.  Он предназначен для безопасного отображения данных, содержащих специальные символы, в HTML-страницах, предотвращая возможные проблемы с парсингом.

## Словарь `html_escapes`

### `html_escapes`

**Описание**: Словарь, в котором ключами являются символы UTF-8, а значениями — их HTML-экранированные аналоги.

**Пример использования**:

```python
from hypotez.src.utils.string.html_escapes import html_escapes

escaped_character = html_escapes['<']  # Получим экранированный символ '<'
print(escaped_character)  # Выведет &lt;
```

**Содержание словаря**:

Словарь содержит большое количество пар "символ UTF-8 : HTML-экранированный символ", что позволяет обработать широкий спектр символов, включая символы пунктуации, специальные символы, латинские буквы с диакритическими знаками, греческие буквы и математические символы.

**Примечание**:  Список ключей (UTF-8 символы) слишком большой для удобного отображения в этом формате.  Если требуется детальный список, обратитесь к исходному коду файла.
```
