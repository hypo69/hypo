# Модуль `hypotez.src.suppliers.chat_gpt`

## Обзор

Модуль `hypotez.src.suppliers.chat_gpt` содержит код для интеграции с моделью GPT. 
Включает в себя функционал для работы с Google Sheets и запросами к API.
