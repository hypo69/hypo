## משתנים

### `board`
**תיאור**:
לוח המשחק, שהוא רשימה דו-ממדית (מטריצה) בגודל 8x8, שבה 0 מציין משבצת ריקה, 1 מציין חלק מספינה ו-2 מציין משבצת שנורתה בה פגיעה.

## הסבר קוד

1.  **ייבוא מודול `random`**:
    -   `import random`: מייבא את המודול `random`, שמשמש ליצירת מספרים אקראיים.
2.  **אתחול לוח המשחק**:
    -   `board = [[0 for _ in range(8)] for _ in range(8)]`: יוצר רשימה דו-ממדית (מטריצה) בגודל 8x8, המייצגת את לוח המשחק וממלא אותה באפסים.
3.  **פונקציה `place_ship()`**:
    -   `direction = random.randint(0, 1)`: בוחרת באקראי את כיוון הספינה: 0 - אופקי, 1 - אנכי.
    -   **מיקום אופקי**:
        -   `row = random.randint(0, 7)`: בוחרת שורה אקראית להתחלת הספינה.
        -   `col = random.randint(0, 4)`: בוחרת עמודה אקראית להתחלת הספינה, תוך התחשבות באורך הספינה כדי שלא תצא מגבולות הלוח.
        -   `for i in range(3): board[row][col + i] = 1`: ממקמת את הספינה על ידי הקצאת הערך 1 לשלוש משבצות בשורה שנבחרה.
    -   **מיקום אנכי**:
        -  `row = random.randint(0, 4)`: בוחרת שורה אקראית להתחלת הספינה, תוך התחשבות באורך הספינה כדי שלא תצא מגבולות הלוח.
        -   `col = random.randint(0, 7)`: בוחרת עמודה אקראית להתחלת הספינה.
        -   `for i in range(3): board[row + i][col] = 1`: ממקמת את הספינה על ידי הקצאת הערך 1 לשלוש משבצות בעמודה שנבחרה.
4.  **קריאה לפונקציה `place_ship()`**:
    -   `place_ship()`: קוראת לפונקציה כדי למקם את הספינה על הלוח.
5.  **לולאת המשחק הראשית `while True:`**:
    -   לולאה אינסופית, שנמשכת עד שהספינה טובעת.
    -   **הצגת לוח המשחק**:
        -   `for row in board: print(" ".join(map(str, row)))`: מציגה את המצב הנוכחי של לוח המשחק במסוף.
    -   **קלט קואורדינטות**:
        -  `try...except ValueError`: בלוק try-except מטפל בשגיאות קלט אפשריות. אם המשתמש יזין פורמט שגוי, תוצג הודעת שגיאה.
        -  `coordinates = input("Введите координаты выстрела (ряд, столбец): ")`: מבקשת מהשחקן להזין קואורדינטות בצורה "שורה,עמודה".
        - `row, col = map(int, coordinates.split(','))`: מפרידה את הקלט לשורה ועמודה וממירה אותם למספרים שלמים.
        -  `if not (0 <= row <= 7 and 0 <= col <= 7):`: בודקת שהקואורדינטות שהוזנו נמצאות בטווח המותר של לוח המשחק.
    -   **בדיקת פגיעה**:
        -  `if board[row][col] == 1:`: בודקת האם יש ספינה במשבצת בה ירה השחקן.
        -   `print("Попал!")`: מציגה הודעה על פגיעה.
        -   `board[row][col] = 2`: מסמנת את המשבצת כ"פגועה" כדי לא לספור פגיעות חוזרות.
        - **בדיקה האם הספינה טבעה:**
            -  `ship_sunk = True`: מגדירה את הדגל `ship_sunk` כ-`True`, בהנחה שהספינה טבעה.
            -   `for row_b in board:`: עוברת על כל השורות בלוח המשחק.
            -  `if 1 in row_b:`: אם יש בשורה כלשהי משבצת עם הערך `1`, אז הספינה עדיין לא טבעה.
            -  `ship_sunk = False`: במקרה זה, הדגל `ship_sunk` משתנה ל-`False`.
            - `if ship_sunk: print("Корабль потоплен! Вы выиграли!"); break`: אם הדגל נשאר `True`, אז הספינה טבעה, ומוצגת הודעת ניצחון והמשחק מסתיים.
    -   **בדיקת החטאה**:
         -   `elif board[row][col] == 0:`: בודקת אם הירי הוא החטאה.
         -   `print("Мимо!")`: מציגה הודעה על החטאה.
    -   **טיפול בפגיעה חוזרת**:
        -  `elif board[row][col] == 2:`: בודקת אם השחקן ירה בעבר במשבצת זו.
        - `print("Вы уже стреляли в эту клетку. Попробуйте другую.")`: מציגה הודעה שהשחקן כבר ירה במשבצת זו.
