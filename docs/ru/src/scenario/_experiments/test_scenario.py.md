# Модуль test_scenario.py

## Обзор

Модуль `test_scenario.py` предназначен для тестирования и демонстрации работы со сценариями, поставщиками (suppliers) и другими компонентами системы `hypotez`. Он включает в себя импорт необходимых модулей, настройку путей, создание экземпляров классов `Supplier` и `Scenario`, а также запуск сценариев.

## Подробней

Модуль используется для запуска и тестирования различных сценариев взаимодействия с поставщиками, такими как AliExpress, Amazon, Kualastyle и eBay. Код демонстрирует, как создать экземпляр класса `Supplier` с определенным префиксом, а затем использовать его для инициализации и запуска сценариев с помощью класса `Scenario`. Этот модуль играет важную роль в тестировании и проверке функциональности системы `hypotez` в контексте различных поставщиков.

## Функции

### `start_supplier`

```python
def start_supplier(supplier_prefix: str) -> Supplier:
    """
    Создает и возвращает экземпляр класса `Supplier` с заданными параметрами.

    Args:
        supplier_prefix (str): Префикс поставщика (например, 'aliexpress', 'amazon').

    Returns:
        Supplier: Объект класса `Supplier`, инициализированный с указанным префиксом.

    Example:
        >>> supplier = start_supplier('aliexpress')
        >>> print(supplier.supplier_prefix)
        aliexpress
    """
    params: dict = \
    {
        'supplier_prefix': supplier_prefix
    }
    
    return Supplier(**params)
```

**Как работает функция**:

Функция `start_supplier` принимает префикс поставщика в качестве аргумента и использует его для создания экземпляра класса `Supplier`. Она создает словарь `params`, содержащий префикс поставщика, и затем передает этот словарь в качестве аргументов для конструктора класса `Supplier`. В результате создается и возвращается объект класса `Supplier`, настроенный для работы с конкретным поставщиком.

Внутри функции происходят следующие действия:
A. Создается словарь `params` с префиксом поставщика.
B. Создается экземпляр класса `Supplier` с использованием `params`.
C. Возвращается созданный экземпляр класса `Supplier`.

**Параметры**:

- `supplier_prefix` (str): Префикс поставщика, используемый для инициализации объекта `Supplier`.

**Возвращает**:

- `Supplier`: Объект класса `Supplier`, настроенный для работы с указанным поставщиком.

## Переменные

- `supplier_prefix` (str): Определяет, с каким поставщиком будет работать сценарий. В коде закомментированы различные варианты (aliexpress, amazon, kualastyle, ebay), но активен только 'aliexpress'.

- `s` (Supplier): Экземпляр класса `Supplier`, созданный с использованием функции `start_supplier` и переменной `supplier_prefix`.

- `scenario` (Scenario): Экземпляр класса `Scenario`, созданный с использованием экземпляра `Supplier` (`s`).

## Запуск сценария

```python
scenario = Scenario(s)

scenario.run_scenarios()
```

После создания экземпляров классов `Supplier` и `Scenario`, вызывается метод `run_scenarios()` для запуска сценариев.
