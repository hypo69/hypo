# Модуль `login.py` для eBay

## Обзор

Модуль `login.py` предназначен для обеспечения интерфейса авторизации на платформе eBay с использованием веб-драйвера.

## Подробнее

Этот модуль, расположенный в `src/suppliers/ebay/`, предоставляет функциональность для автоматизированной авторизации на eBay. Он может использоваться для автоматизации задач, связанных с получением данных или управлением аккаунтом на eBay.
