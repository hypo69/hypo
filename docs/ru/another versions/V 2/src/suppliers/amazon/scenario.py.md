# Модуль `src.suppliers.amazon.scenario`

## Обзор

Модуль `src.suppliers.amazon.scenario` предназначен для сбора данных о товарах с сайта поставщика Amazon. Он включает в себя функционал для получения списка категорий товаров, списка товаров в каждой категории, и сбора детальной информации о каждом товаре. Модуль взаимодействует с веб-драйвером для навигации по сайту и извлечения необходимой информации.

## Функции

### `get_list_products_in_category`

**Описание**: Возвращает список URL-адресов товаров со страницы категории.
Функция также отвечает за прокрутку страницы категории если это необходимо.

**Параметры**:
- `s` (Supplier): Экземпляр поставщика (Supplier instance).

**Возвращает**:
- `list[str,str,None]`: Список URL-адресов товаров или `None`, если список не был получен или локаторы не найдены.

**Пример использования:**
```python
from src.suppliers.amazon.scenario import get_list_products_in_category

supplier_instance = ... # инициализируем экземпляр поставщика
list_of_products = get_list_products_in_category(supplier_instance)

if list_of_products:
    for product_url in list_of_products:
        print(product_url)
else:
    print("Не удалось получить список товаров.")
```

**Примечание:**
- Функция использует локаторы для поиска элементов на странице.
- Функция прокручивает страницу для загрузки всех товаров.
- Функция проверяет, является ли результат поиска строкой, если да, то оборачивает ее в список.
- Выводит в лог количество найденных товаров.
- Временно закоментирована логика проверки товара на наличие в базе данных.
