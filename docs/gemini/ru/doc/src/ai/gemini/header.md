# Модуль hypotez/src/ai/gemini/header.py

## Обзор

Этот модуль содержит вспомогательные функции для работы с проектом, включая определение корневой директории проекта, загрузку конфигурации из файла `config.json` и чтение документации из файла `README.MD`.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная от текущей директории файла. Поиск выполняется вверх по дереву директорий, пока не будет найдена директория содержащая один из файлов указанных в `marker_files`.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, по которым определяется корень проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена, иначе путь к директории, где находится скрипт.

**Вызывает исключения**:

- Нет исключений.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root()`.

**Тип**: `Path`

### `config`

**Описание**: Словарь с конфигурацией проекта, загруженный из файла `config.json` в директории `src`.

**Тип**: `dict`

**Примечания**: Значение по умолчанию `None`. В случае ошибки загрузки (например, файл не найден или некорректный формат JSON), значение остается `None`.

### `doc_str`

**Описание**: Строка с текстом документации, загруженная из файла `README.MD` в директории `src`.

**Тип**: `str`

**Примечания**: Значение по умолчанию `None`. В случае ошибки загрузки (например, файл не найден или некорректный формат текста), значение остается `None`.

### `__project_name__`

**Описание**: Имя проекта, полученное из конфигурации (config). В случае отсутствия ключа в config, возвращает 'hypotez'.

**Тип**: `str`

### `__version__`

**Описание**: Версия проекта, полученная из конфигурации (config). В случае отсутствия ключа в config, возвращает ''.

**Тип**: `str`

### `__doc__`

**Описание**: Текст документации, загруженный из файла `README.MD`. В случае отсутствия файла возвращает пустую строку.

**Тип**: `str`

### `__details__`

**Описание**: Дополнительные детали проекта. (пустая строка)

**Тип**: `str`

### `__author__`

**Описание**: Автор проекта, полученный из конфигурации (config). В случае отсутствия ключа в config, возвращает ''.

**Тип**: `str`

### `__copyright__`

**Описание**: Авторские права проекта, полученные из конфигурации (config). В случае отсутствия ключа в config, возвращает ''.

**Тип**: `str`

### `__cofee__`

**Описание**: Ссылка на способ поддержать разработчиков.

**Тип**: `str`
