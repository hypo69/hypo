# Модуль `src.utils`

## Обзор

Модуль `src.utils` представляет собой набор служебных утилит, предназначенных для использования в проекте. Он спроектирован так, чтобы минимизировать использование внутренних утилит. Важно отметить, что в этом модуле **не используются** функции `j_loads`, `j_loads_ns` и `j_dumps`. Вместо этого используется стандартная библиотека `import json`.

### Инструкции
1.  Модули `src.utils` и `src.utils.converters` — служебные утилиты проекта. Они спроектированы так, чтобы по минимуму использовать внутренние утилиты.  
2.  В этих модулях **НЕ используется** `j_loads`, `j_loads_ns` и `j_dumps`.  
3.  Используйте стандартную библиотеку `import json`.
