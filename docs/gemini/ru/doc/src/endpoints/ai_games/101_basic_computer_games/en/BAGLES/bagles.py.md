# Бейглы

## Обзор

Этот модуль реализует игру "Бейглы", в которой игрок пытается угадать трехзначное число, составленное из неповторяющихся цифр. После каждой попытки игрок получает подсказки: "PICO" означает, что одна из цифр угадана и находится на правильной позиции, "FERMI" означает, что одна из цифр угадана, но не на правильной позиции, "BAGELS" означает, что ни одна из цифр не угадана.

## Функции

### `generate_secret_number`

**Описание**: Генерирует случайное трехзначное число с неповторяющимися цифрами.

**Параметры**:
- Нет параметров.

**Возвращает**:
- `str`: Случайное трехзначное число в виде строки.

### `get_clues`

**Описание**: Генерирует подсказки PICO, FERMI и BAGELS на основе сравнения пользовательского предположения и секретного числа.

**Параметры**:
- `secret_number` (str): Секретное трехзначное число.
- `user_guess` (str): Предположение пользователя.

**Возвращает**:
- `list[str]`: Список подсказок. Возвращает `["BAGELS"]`, если нет совпадений.

### `play_bagels`

**Описание**: Основная логика игры Бейглы.

**Параметры**:
- Нет параметров.

**Возвращает**:
- `None`: Функция ничего не возвращает, но выводит результаты игры в консоль.
