# Модуль hypotez/src/ai/helicone/__init__.py

## Обзор

Этот модуль предоставляет конфигурацию для работы с сервисом Helicone.

## Переменные

### `MODE`

**Описание**:  Переменная, определяющая режим работы.

**Значение**: `'dev'`

**Описание**:  В текущей реализации это строка `'dev'`, указывающая на режим разработки.  Возможно, в дальнейшем этот параметр будет использоваться для переключения между режимами (например, `'prod'`).
