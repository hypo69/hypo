## Описание структуры JSON

JSON-файл содержит один объект с ключом `product_links`, который описывает локатор для извлечения ссылок на товары на странице категории.

## Раздел `product_links`

### Описание

Раздел `product_links` определяет параметры для поиска ссылок на товары.

### Структура

```json
{
  "product_links": {
    "attribute": "href",
    "by": "XPATH",
    "selector": "//span[@data-component-type ='s-product-image']//a",
    "if_list":"first",
    "use_mouse": false,
    "mandatory": true,
    "timeout":0,
    "timeout_for_event":"presence_of_element_located",
    "event": null
  }
}
```

### Параметры

- `attribute` (str): Атрибут HTML-элемента, значение которого нужно извлечь. В данном случае, это `href` для получения URL-адреса ссылки.
- `by` (str): Метод поиска элемента на веб-странице. Здесь используется `XPATH`, что означает поиск по XPath-выражению.
- `selector` (str): XPath-выражение для поиска элемента. В данном случае, `//span[@data-component-type ='s-product-image']//a` ищет все ссылки `<a>` внутри элементов `<span>` с атрибутом `data-component-type`, равным `s-product-image`.
- `if_list` (str): Указывает, как обрабатывать список найденных элементов. Значение `first` указывает на выбор первого элемента из списка.
- `use_mouse` (bool):  Указывает, нужно ли использовать эмуляцию мыши для взаимодействия с элементом. В данном случае `false`, что означает не использовать.
- `mandatory` (bool): Указывает, является ли элемент обязательным для поиска. `true` означает, что его отсутствие вызывает ошибку.
- `timeout` (int):  Время ожидания (в секундах) при поиске элемента. Здесь установлено `0`, что означает неявное ожидание.
- `timeout_for_event` (str): Указывает на ожидание конкретного события, прежде чем возвращать элемент. В данном случае, `presence_of_element_located`, что означает ожидание появления элемента в DOM.
- `event` (null):  Событие, которое нужно отслеживать при поиске. Здесь установлено значение `null`.
