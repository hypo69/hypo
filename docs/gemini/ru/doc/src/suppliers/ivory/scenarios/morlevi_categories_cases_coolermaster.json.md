# Документация для `morlevi_categories_cases_coolermaster.json`

## Обзор

Данный файл содержит JSON-конфигурацию сценариев для парсинга категорий корпусов Cooler Master с сайта Morlevi. Каждый сценарий определяет параметры для конкретного типа корпуса, включая его бренд, URL для парсинга, соответствие категориям PrestaShop, а также флаги активности и состояния.

## Структура JSON

### Ключ `scenarios`

`scenarios` - это основной ключ, который содержит объект с набором сценариев. Каждый ключ этого объекта является названием конкретного сценария.

### Сценарий

Каждый сценарий представляет собой объект со следующими ключами:

#### Ключ `brand`

- **Описание**: Указывает бренд товара.
- **Тип**: `str`
- **Пример**: `"COOLER MASTER"`

#### Ключ `template`

- **Описание**:  Используется для шаблонов. В данном файле всегда пустая строка.
- **Тип**: `str`
- **Пример**: `""`

#### Ключ `url`

- **Описание**: URL-адрес страницы сайта Morlevi для парсинга.
- **Тип**: `str`
- **Пример**: `"https://www.morlevi.co.il/Cat/285?p_315=74&p_124=540&sort=datafloat2%2Cprice&keyword="`
     
   **Примечание**: В одном из сценариев (`COOLERMASTER gaming full tower`) указан временный URL `"----------------------------COOLER MASTER gaming full TOWER--------------------------------"`.  Необходимо заменить его на корректный URL.

#### Ключ `checkbox`

- **Описание**: Флаг для работы с чекбоксами (не используется в данном случае).
- **Тип**: `bool`
- **Пример**: `false`

#### Ключ `active`

- **Описание**: Флаг, указывающий, активен ли сценарий.
- **Тип**: `bool`
- **Пример**: `true`

#### Ключ `condition`

- **Описание**: Условие товара.
- **Тип**: `str`
- **Пример**: `"new"`

#### Ключ `presta_categories`

- **Описание**: Объект, содержащий информацию о сопоставлении категорий Morlevi с категориями PrestaShop.
- **Тип**: `object`

##### Ключ `template` (внутри `presta_categories`)

- **Описание**:  Объект, где ключ - бренд (в нижнем регистре), а значение - соответствующая категория PrestaShop.
- **Тип**: `object`
- **Пример**: `{"cooler master": "MID TOWER"}`

## Заключение

Данный файл предназначен для конфигурации парсера, предоставляя ему необходимые данные для сбора информации о корпусах Cooler Master с сайта Morlevi. Каждый сценарий соответствует определенной категории корпуса и имеет связанные с ней параметры.
