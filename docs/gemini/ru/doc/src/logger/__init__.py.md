# Модуль `hypotez/src/logger/__init__.py`

## Обзор

Данный модуль инициализирует пакет `logger`, предоставляя доступ к основным компонентам, таким как логгер, и исключениям.

## Содержание

1.  [Переменные](#переменные)
2.  [Импорты](#импорты)

## Переменные

### `MODE`

- **Описание**: Определяет режим работы приложения (по умолчанию 'dev').
- **Тип**: `str`
- **Значение по умолчанию**: `'dev'`

## Импорты

### `logger`

- **Описание**: Импортирует настроенный логгер из модуля `logger.logger`.
- **Источник**: `from .logger import logger`

### Исключения

- **Описание**: Импортирует пользовательские исключения, используемые в модуле.
    - `ExecuteLocatorException`: Исключение, возникающее при проблемах с локаторами.
    - `DefaultSettingsException`: Исключение, возникающее при проблемах с настройками по умолчанию.
    - `CredentialsError`: Исключение, возникающее при проблемах с учетными данными.
    - `PrestaShopException`: Исключение, возникающее при проблемах, связанных с PrestaShop.
    - `PayloadChecksumError`: Исключение, возникающее при проблемах с контрольной суммой полезной нагрузки.
- **Источник**: `from .exceptions import ( ExecuteLocatorException, DefaultSettingsException, CredentialsError, PrestaShopException, PayloadChecksumError)`
