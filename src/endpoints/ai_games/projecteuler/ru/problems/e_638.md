# Задача No 638
## Совершенные кубы в интервалах

## Условия

Пусть $f(n)$ — количество совершенных кубов в интервале $[2, n]$.

Например, $f(10) = 2$, так как $2^3 = 8$ - это единственный совершенный куб в интервале $[2, 10]$.
Кроме того, $f(100) = 4$, так как $2^3 = 8$, $3^3 = 27$, $4^3 = 64$ и $5^3 = 125$ (но 125 выходит за пределы интервала) являются совершенными кубами в интервале $[2, 100]$.

Определим $G(n) = \sum_{k=2}^n f(k)$.
Таким образом, $G(10) = f(2) + f(3) + ... + f(10) = 0 + 0 + 0 + 0 + 0 + 0 + 1 + 1 + 1 + 2 = 6$.

Известно, что $G(100) = 415$ и $G(10^6) = 3528495$.

Найдите $G(10^{18})$.
