# Модуль `hypotez/src/suppliers/morlevi/header.py`

## Обзор

Этот модуль содержит функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и чтения документации из `README.MD`.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла, ищет вверх по дереву каталогов, пока не найдёт директорию, содержащую один из указанных маркеров файлов.

**Параметры**:

- `marker_files` (tuple): Кортеж с именами файлов или каталогов, которые используются в качестве маркеров для определения корневой директории проекта. По умолчанию (`('pyproject.toml', 'requirements.txt', '.git')`).

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена, в противном случае возвращает директорию, где расположен текущий скрипт.

**Вызывает исключения**:

- Нет.

## Переменные

### `__root__`

**Описание**: Содержит путь к корневой директории проекта, полученный с помощью функции `set_project_root`.
