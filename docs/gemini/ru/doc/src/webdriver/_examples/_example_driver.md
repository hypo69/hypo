# Модуль `hypotez/src/webdriver/_examples/_example_driver.py`

## Обзор

Этот модуль содержит примеры использования класса `Driver` для взаимодействия с веб-драйверами (Chrome, Firefox, Edge).  В примерах показано как навигация по URL, извлечение домена, прокрутка страницы и сохранение куки.  Важная особенность – гарантированное закрытие драйвера, даже при возникновении исключений,  что предотвращает зависание браузера.

### `main`

**Описание**: Функция `main` демонстрирует использование класса `Driver` с разными веб-браузерами (Chrome, Firefox, Edge). Используется блок `try...finally` для гарантированного закрытия браузера, даже если произойдет ошибка.
