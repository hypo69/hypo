# Задача No 521
## Наибольший простой делитель факториала
## Условия
Пусть $f(n)$ — наибольший простой делитель $n$.
Пусть $g(n) = \sum_{i=1}^n f(i!)$.

Например:
$f(1!) = f(1) = 1$,
$f(2!) = f(2) = 2$,
$f(3!) = f(6) = 3$,
$f(4!) = f(24) = 3$,
$f(5!) = f(120) = 5$.

Значит, $g(5) = 1 + 2 + 3 + 3 + 5 = 14$.

Также, $g(10) = 29$ и $g(100) = 687$.

Найдите $g(10^8)$.
