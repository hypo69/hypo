# Модуль `hypotez/src/logger/header.py`

## Обзор

Модуль `hypotez/src/logger/header.py` определяет корневой путь к проекту `hypotez`.  Все импорты в проекте строятся относительно этого пути.  В будущем, этот модуль планируется перенести в системные переменные.

## Оглавление

* [Функции](#функции)


## Функции

### `get_project_root`

**Описание**: Находит корневой каталог проекта, начиная с текущего файла. Поиск осуществляется вверх по директориям, до тех пор, пока не будет найдена директория, содержащая один из указанных файлов-маркеров.

**Параметры**:
- `marker_files` (tuple): Кортеж с именами файлов или директорий, которые используются для определения корневого каталога проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Объект `Path` с путем к корневому каталогу проекта. Если корневой каталог не найден, возвращается путь к директории, где расположен текущий файл.

**Вызывает исключения**:
- Нет.


## Глобальные переменные

**Описание**: Путь к корневому каталогу проекта.

**Тип**: `dict` | `None`
