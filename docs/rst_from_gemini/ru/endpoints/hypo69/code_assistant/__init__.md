**Описание:**

Этот модуль содержит функцию `get_project_root`, которая находит корневой каталог проекта, начиная с текущего файла. Он ищет файлы-маркеры (например, `pyproject.toml`, `requirements.txt`, `.git`)  в родительских каталогах и останавливается на первом, содержащем хотя бы один из этих маркеров.  Это позволяет импортировать модули из других частей проекта.


**Функции:**

* **`get_project_root(marker_files=('pyproject.toml', 'requirements.txt', '.git')) -> Path`**:
    * Находит корневой каталог проекта.
    * **Аргументы:**
        * `marker_files (tuple)`: Список файлов или каталогов, которые используются для определения корня проекта.
    * **Возвращает:**
        * `Path`: Путь к корневому каталогу проекта, если найден, иначе - каталог, в котором находится данный скрипт.
    * **Описание:** Функция рекурсивно поднимается по директориям от текущего файла, проверяя наличие указанных файлов-маркеров в каждом родительском каталоге.  Если такой каталог найден, функция возвращает его путь.  Если нет, возвращает каталог, где находится данный файл.  Также, если найденный корневой каталог не в `sys.path`, он добавляется в начало этого списка для правильного импорта модулей.


**Переменные:**

* **`__root__ (Path)`**: Переменная, хранящая путь к корневому каталогу проекта. Инициализируется вызовом функции `get_project_root()`.
    * **Описание:** Путь к корневому каталогу проекта, полученный с помощью `get_project_root()`.  Этот путь важен для импорта модулей из других частей проекта.
