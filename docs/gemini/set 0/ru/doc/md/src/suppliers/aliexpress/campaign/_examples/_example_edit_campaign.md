# Модуль hypotez/src/suppliers/aliexpress/campaign/_examples/_example_edit_campaign.py

## Обзор

Этот модуль содержит пример реализации редактора рекламных кампаний на AliExpress.  Он предоставляет базовый класс `AliCampaignEditor`, который наследуется от `AliPromoCampaign`, и иллюстрирует общие функции для работы с рекламными кампаниями.

## Классы

### `AliCampaignEditor`

**Описание**: Класс `AliCampaignEditor` предназначен для редактирования рекламных кампаний на AliExpress. Он наследуется от `AliPromoCampaign` и предоставляет расширенный функционал.

**Методы**:

- `__init__`:
    **Описание**: Конструктор класса. Инициализирует необходимые атрибуты, такие как имя кампании, категория, язык и валюта.
    **Параметры**:
        - `campaign_name` (str): Название рекламной кампании.
        - `category_name` (str): Название категории.
        - `language` (str, опционально, по умолчанию 'EN'): Язык кампании.
        - `currency` (str, опционально, по умолчанию 'USD'): Валюта кампании.
    **Возвращает**:
        - None


## Модули

Этот модуль использует следующие модули:

- `re`: для работы с регулярными выражениями.

- `pathlib`: для работы с путями к файлам.
- `typing`: для работы с типизацией.
- `types`: для работы с различными типами.
- `src.gs`: для работы с Google Sheets (если необходимо).

- `src.suppliers.aliexpress.affiliated_products_generator`: для работы с генерацией связанных продуктов.
- `src.suppliers.aliexpress.utils.extract_product_id`: для извлечения идентификаторов продуктов.
- `src.suppliers.aliexpress.utils.set_full_https`: для обеспечения HTTPS-ссылок.
