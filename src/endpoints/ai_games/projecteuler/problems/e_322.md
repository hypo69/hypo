# Задача No 322
## Биномиальные коэффициенты, делящиеся на простые числа
## Условия
Для неотрицательного целого числа *n* и целого числа *k* биномиальный коэффициент *C(n, k)* определяется как:

*C(n, k)* = *n*! / (*k*!(*n*−*k*)!)

Для заданного простого числа *p* мы определяем *N(n, p)* как число биномиальных коэффициентов *C(n, k)*, где *0 ≤ k ≤ n*, которые не делятся на *p*.

Например, *N(10, 2) = 6*, *N(20, 5) = 4* и *N(1000, 5) = 76*.

Пусть *f(n)* будет наименьшим целым числом, для которого *N(n, p) = n + 1*, что означает, что все биномиальные коэффициенты *C(n, k)*, где *0 ≤ k ≤ n*, не делятся на *p*.

Например, *f(10) = 11*, *f(100) = 107* и *f(1000) = 2089*.

Найдите Σ*f(10^k)* для *1 ≤ k ≤ 18*. Дайте в качестве ответа остаток от деления суммы на 1000000007.
