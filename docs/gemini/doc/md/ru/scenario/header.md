# Модуль scenario/header.py

## Обзор

Данный модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`. Он также предоставляет информацию о проекте, такую как имя, версия, документация, автор и другие детали.

## Оглавление

- [Функции](#функции)
    - [get_project_root](#get_project_root)


## Функции

### `get_project_root`

**Описание**: Функция находит корневую директорию проекта, начиная с директории текущего файла, ищет вверх по дереву директорий, останавливаясь на первой директории, содержащей один из указанных файлов-маркеров.

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или директорий, которые используются для определения корневой директории проекта.

**Возвращает**:
- `Path`: Путь к корневой директории проекта, если найдена, в противном случае - директория, в которой расположен скрипт.


**Вызывает исключения**:
- Нет.


## Переменные


### `__root__`

**Описание**: Путь к корневой директории проекта, полученной функцией `get_project_root`.

**Тип**: `Path`

### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json`.

**Тип**: `dict` или `None`

### `doc_str`

**Описание**: Текст документации проекта, загруженный из файла `README.MD`.

**Тип**: `str` или `None`


### `__project_name__`

**Описание**: Имя проекта. Получается из настроек, если они есть, иначе равно 'hypotez'.

**Тип**: `str`

### `__version__`

**Описание**: Версия проекта. Получается из настроек, если они есть, иначе пустая строка.

**Тип**: `str`

### `__doc__`

**Описание**: Документация проекта. Получается из настроек, если они есть, иначе пустая строка.

**Тип**: `str`

### `__details__`

**Описание**: Детали проекта.

**Тип**: `str`

### `__author__`

**Описание**: Автор проекта. Получается из настроек, если они есть, иначе пустая строка.

**Тип**: `str`

### `__copyright__`

**Описание**: Авторские права. Получается из настроек, если они есть, иначе пустая строка.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка для поддержки автора. Получается из настроек, если они есть, иначе ссылка по умолчанию.

**Тип**: `str`
