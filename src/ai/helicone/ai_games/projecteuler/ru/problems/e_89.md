# Задача No 89
## Римские числа

## Условия
Римские цифры записываются с использованием букв I, V, X, L, C, D и M.
```
Числа:  I = 1, V = 5, X = 10, L = 50, C = 100, D = 500, M = 1000.
```

Числа обычно записываются от наибольшего к наименьшему, например, 1666 записывается как MDCLXVI.

Однако римские цифры, представляющие 4 (IV), 9 (IX), 40 (XL), 90 (XC), 400 (CD) и 900 (CM), используют принцип вычитания:
* IV = 4
* IX = 9
* XL = 40
* XC = 90
* CD = 400
* CM = 900

В данной задаче нас интересуют только римские числа, которые можно записать с помощью минимального количества символов. Например, число 16 может быть записано как XVI, а не как XIIIIII, а число 15 может быть записано как XV, а не как VVV.

Предполагается, что все римские числа, содержащиеся в файле roman.txt, были записаны в неминимальной форме.

Найдите общее количество символов, сэкономленных при переводе каждого из чисел в файле roman.txt в его минимальную форму.

**Примечание:** Файл roman.txt содержит тысячу римских чисел, каждое из которых записано на новой строке.

**Пример:**

Рассмотрим следующие римские числа:
* VIII -> VIII (разницы нет)
* XVI -> XVI (разницы нет)
* XIIIIII -> XVI (экономия 3 символа)
* IIIIIIIIII -> X (экономия 8 символов)
* LVIIII -> LX (экономия 2 символа)

Общая экономия символов составит 3 + 8 + 2 = 13.
