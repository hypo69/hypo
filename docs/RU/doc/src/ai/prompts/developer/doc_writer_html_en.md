# Модуль: Помощник по коду

## Обзор

Данный модуль содержит класс `CodeAssistant`, предназначенный для взаимодействия с различными моделями ИИ, такими как Google Gemini и OpenAI, для задач обработки кода.  Модуль предоставляет инструменты для анализа кода и генерации документации.

## Примеры использования

Пример использования класса `CodeAssistant`:

```python
assistant = CodeAssistant(роль='проверяющий_код', язык='ru', модели=['gemini'])
результат = assistant.обработать_файлы(файлы=['файл1.py', 'файл2.py'], опции={})
print(результат)
```

## Поддерживаемые платформы

Данный модуль совместим с Python 3.x.

## Синтаксис

Модуль использует стандартный синтаксис Python.

## Атрибуты

### `роль`

Тип: `str`

Описание: Роль помощника (например, 'проверяющий_код').


### `язык`

Тип: `str`

Описание: Язык, который будет использовать помощник (например, 'ru').


### `модели`

Тип: `list[str]`

Описание: Список используемых моделей ИИ (например, `['gemini']`).


## Классы

### `CodeAssistant`

**Описание**: Класс `CodeAssistant` предназначен для взаимодействия с моделями ИИ, такими как Google Gemini, для анализа и генерации документации кода.

**Атрибуты**:

* `роль`: Роль помощника.
* `язык`: Язык, используемый помощником.
* `модели`: Список используемых моделей ИИ.

**Методы**:

* `обработать_файлы`: Метод для обработки кодовых файлов.

```python
def обработать_файлы(self, файлы: list[str], опции: dict = None) -> list | None:
    """
    Args:
        файлы (list[str]): Список файлов для обработки.
        опции (dict, optional): Дополнительные параметры для настройки обработки. По умолчанию None.

    Returns:
        list | None: Результат обработки в виде списка проанализированных данных. Возвращает None при ошибке.

    Raises:
        FileNotFoundError: Если файл не найден.
        ValueError: Если передан некорректный тип данных.
    """
```
