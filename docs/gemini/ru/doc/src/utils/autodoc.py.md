# Модуль `autodoc`

## Обзор

Модуль `autodoc` предоставляет декоратор `@autodoc`, который автоматически обновляет строку документации (docstring) функции, добавляя информацию о времени последнего вызова. Это полезно для отслеживания времени выполнения функций и отладки.

## Функции

### `autodoc`

**Описание**:
Декоратор для автоматического обновления docstring функции. Оборачивает функцию и обновляет ее docstring перед каждым вызовом, добавляя информацию о времени последнего вызова.

**Параметры**:
- `func` (function): Функция, которую необходимо обернуть декоратором.

**Возвращает**:
- `function`: Обернутая функция.

### `update_docstring`

**Описание**:
Обновляет docstring функции, добавляя информацию о времени последнего вызова. Если docstring не существует, создает его.

**Параметры**:
- `func` (function): Функция, чей docstring необходимо обновить.

**Возвращает**:
- `None`: Функция не возвращает значения. Обновление docstring происходит "на месте".

## Примеры использования

### `example_function`

**Описание**:
Пример функции, использующей декоратор `autodoc`. При каждом вызове функции, ее docstring обновляется, добавляя информацию о времени последнего вызова.

**Параметры**:
- `param1` (int): Первое значение.
- `param2` (str): Второе значение.

**Возвращает**:
- `None`: Функция не возвращает значения.

**Пример использования**:
```python
@autodoc
def example_function(param1: int, param2: str) -> None:
    """Пример функции.

    Args:
        param1 (int): Первое значение.
        param2 (str): Второе значение.
    """
    print(f"Processing {param1} and {param2}")

example_function(1, "test")
print(example_function.__doc__)
example_function(2, "another test")
print(example_function.__doc__)
```
