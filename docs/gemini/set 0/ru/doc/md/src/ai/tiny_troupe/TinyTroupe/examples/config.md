# Модуль конфигурации TinyTroupe

## Обзор

Данный файл (`config.ini`) содержит конфигурацию для приложения TinyTroupe, определяя параметры API, модели, временные ограничения и другие настройки.

## Параметры

### `API_TYPE`

**Описание**: Тип API, используемый приложением.

**Значение по умолчанию**: `openai`

**Возможные значения**: `openai`, `azure`.

### `AZURE_API_VERSION`

**Описание**: Версия API для Azure OpenAI.

**Значение по умолчанию**: `2023-05-15`

**Примечания**: Для получения актуальной информации о доступных версиях API обратитесь к документации Azure.

### `MODEL`

**Описание**: Модель языка, используемая для обработки запросов.

**Значение по умолчанию**: `gpt-4o`

### `MAX_TOKENS`

**Описание**: Максимальное количество токенов, которое может быть обработано моделью.

**Значение по умолчанию**: `4000`

### `TEMPERATURE`

**Описание**: Параметр, влияющий на креативность генерации текста моделью. Чем выше значение, тем более непредсказуем результат.

**Значение по умолчанию**: `0.3`

### `FREQ_PENALTY`

**Описание**: Штраф за частоту повторяющихся слов.

**Значение по умолчанию**: `0.0`

### `PRESENCE_PENALTY`

**Описание**: Штраф за повторение слов в разных частях текста.

**Значение по умолчанию**: `0.0`

### `TIMEOUT`

**Описание**: Максимальное время ожидания ответа от API (в секундах).

**Значение по умолчанию**: `60`

### `MAX_ATTEMPTS`

**Описание**: Максимальное количество попыток запроса к API в случае сбоя.

**Значение по умолчанию**: `5`

### `WAITING_TIME`

**Описание**: Время ожидания между попытками запроса к API в случае сбоя (в секундах).

**Значение по умолчанию**: `1`


### `EXPONENTIAL_BACKOFF_FACTOR`

**Описание**: Коэффициент экспоненциальной задержки между попытками запроса к API в случае сбоя.

**Значение по умолчанию**: `5`


### `EMBEDDING_MODEL`

**Описание**: Модель для создания векторных представлений текста.

**Значение по умолчанию**: `text-embedding-3-small`

### `CACHE_API_CALLS`

**Описание**: Флаг, определяющий, использовать ли кэш для результатов вызовов API.

**Значение по умолчанию**: `False`

### `CACHE_FILE_NAME`

**Описание**: Имя файла для кэша API вызовов.

**Значение по умолчанию**: `openai_api_cache.pickle`


### `MAX_CONTENT_DISPLAY_LENGTH`

**Описание**: Максимальная длина текста, который отображается перед обработкой.

**Значение по умолчанию**: `1024`


### [Simulation]

### `RAI_HARMFUL_CONTENT_PREVENTION`

**Описание**: Включение защиты от генерации вредного контента.

**Значение по умолчанию**: `True`

### `RAI_COPYRIGHT_INFRINGEMENT_PREVENTION`

**Описание**: Включение защиты от нарушения авторских прав.

**Значение по умолчанию**: `True`

### [Logging]

### `LOGLEVEL`

**Описание**: Уровень детализации логов.

**Значение по умолчанию**: `ERROR`

**Возможные значения**: `ERROR`, `WARNING`, `INFO`, `DEBUG`.
