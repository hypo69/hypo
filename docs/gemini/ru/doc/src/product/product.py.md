## Функции

### `get_parent_categories`

**Описание**: Собирает родительские категории из указанной категории. Дублирует функциональность `get_parents()` из класса `Category`.

**Параметры**:
- `id_category` (int): ID категории.
- `dept` (int, optional): Глубина категории. По умолчанию `0`.

**Возвращает**:
- `list`: Список родительских категорий.

**Вызывает исключения**:
- `TypeError`: Если `id_category` не является целым числом.
