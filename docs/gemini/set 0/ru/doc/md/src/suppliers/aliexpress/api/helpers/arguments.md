# Модуль `hypotez/src/suppliers/aliexpress/api/helpers/arguments.py`

## Обзор

Модуль `arguments.py` содержит вспомогательные функции для обработки аргументов, связанных с API AliExpress. Он предоставляет инструменты для преобразования входных значений в подходящий формат и валидации аргументов.

## Функции

### `get_list_as_string`

**Описание**: Функция преобразует входное значение в строку, разделяя элементы списка запятыми. Если входное значение является пустым или None, функция возвращает None. Если входное значение не является строкой или списком, возникает исключение `InvalidArgumentException`.

**Параметры**:
- `value`: Значение для преобразования. Может быть строкой или списком.

**Возвращает**:
- Строковое представление входного значения (с элементами списка, разделенными запятыми) или `None`, если входное значение равно `None`.

**Вызывает исключения**:
- `InvalidArgumentException`: Если входное значение не является ни строкой, ни списком.


### `get_product_ids`

**Описание**: Функция извлекает идентификаторы продуктов из входных данных, обрабатывая как строки, так и списки строк.

**Параметры**:
- `values`:  Входные данные, представляющие список идентификаторов продуктов. Может быть строкой или списком строк.

**Возвращает**:
- Список идентификаторов продуктов, полученных с помощью функции `get_product_id`.

**Вызывает исключения**:
- `InvalidArgumentException`: Если входные данные не являются ни строкой, ни списком.
