# Модуль `hypotez/src/suppliers/ksp/header.py`

## Обзор

Этот модуль содержит функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и чтения документации из файла `README.MD`.  Он также предоставляет переменные, содержащие информацию о проекте, например, имя проекта, версию, документацию и прочее.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с текущей директории и идя вверх по дереву директорий, пока не найдет директорию, содержащую один из указанных файлов-маркеров (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, которые используются для определения корня проекта. По умолчанию (`'pyproject.toml', 'requirements.txt', '.git'`).

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена, иначе — путь к текущей директории.

**Вызывает исключения**:

- Нет.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root()`. Тип: `Path`.

### `settings`

**Описание**: Словарь, содержащий настройки проекта. Загружается из файла `settings.json` в корневой директории проекта. Тип: `dict` или `None`.

### `doc_str`

**Описание**: Строка, содержащая текст документации из файла `README.MD` в корневой директории проекта. Тип: `str` или `None`.


### `__project_name__`

**Описание**: Имя проекта. Значение взято из настроек (`settings`) или имеет значение по умолчанию - `'hypotez'`.  Тип: `str`.

### `__version__`

**Описание**: Версия проекта. Значение взято из настроек (`settings`) или пустая строка по умолчанию. Тип: `str`.

### `__doc__`

**Описание**: Документация проекта. Значение взято из файла `README.MD` или пустая строка по умолчанию. Тип: `str`.


### `__details__`

**Описание**: Подробная информация о проекте. По умолчанию пустая строка. Тип: `str`.


### `__author__`

**Описание**: Автор проекта. Значение взято из настроек (`settings`) или пустая строка по умолчанию. Тип: `str`.


### `__copyright__`

**Описание**: Авторские права. Значение взято из настроек (`settings`) или пустая строка по умолчанию. Тип: `str`.

### `__cofee__`

**Описание**: Ссылка для поддержки разработчика. Значение взято из настроек (`settings`) или строка по умолчанию. Тип: `str`.
