## Примеры использования драйвера

### `main`

**Описание**: Главная функция, демонстрирующая примеры использования классов `Driver` и `Chrome`.

**Параметры**:
Нет.

**Возвращает**:
Нет.

**Вызывает исключения**:
Нет.

### `Driver(browser_type, user_agent=None)`

**Описание**: Конструктор класса `Driver`. Создаёт экземпляр веб-драйвера на основе переданного типа браузера.

**Параметры**:
- `browser_type` (класс): Класс, представляющий тип браузера (например, `Chrome`, `Firefox`).
- `user_agent` (dict, optional):  Словарь с пользовательским User-Agent. По умолчанию `None`.

**Возвращает**:
- Экземпляр класса `Driver`

**Вызывает исключения**:
- Возможны исключения, возникающие при создании драйвера (например, если не удалось загрузить драйвер браузера).

### `get_url(url: str) -> bool`

**Описание**:  Метод для навигации по заданному URL.

**Параметры**:
- `url` (str): URL адрес, на который нужно перейти.

**Возвращает**:
- `bool`:  `True`, если навигация успешна, `False` в противном случае.

**Вызывает исключения**:
- Возможны исключения, связанные с работой с веб-драйвером (например, проблемы с соединением, загрузка страницы не удалась).

### `extract_domain(url: str) -> str`

**Описание**: Метод для извлечения домена из URL.

**Параметры**:
- `url` (str): URL, из которого необходимо извлечь домен.

**Возвращает**:
- `str`: Доменная часть URL (например, `example.com`).

**Вызывает исключения**:
- `ValueError`: Если переданный `url` не имеет валидного формата.


### `_save_cookies_localy() -> bool`

**Описание**: Метод для сохранения куки в локальный файл.  (Метод помечен как приватный, _).

**Параметры**:
Нет.

**Возвращает**:
- `bool`: `True`, если сохранение успешное, `False` - если нет.


**Вызывает исключения**:
-  Исключения, связанные с работой с файлами (например, отсутствие доступа к файлу, проблема с форматом файла).


### `page_refresh() -> bool`

**Описание**: Обновляет текущую страницу.

**Параметры**:
Нет.

**Возвращает**:
- `bool`: `True`, если обновление успешно, `False` - если нет.


**Вызывает исключения**:
- Возможны исключения, связанные с обновлением страницы (например, проблемы с соединением, драйвер не смог обработать запрос).


### `scroll(scrolls: int, direction: str, frame_size: int, delay: int) -> bool`

**Описание**: Прокручивает страницу.

**Параметры**:
- `scrolls` (int): Количество прокруток.
- `direction` (str): Направление прокрутки ('forward', 'backward').
- `frame_size` (int): Размер прокрутки (пиксели).
- `delay` (int): Задержка между прокрутками (мс).

**Возвращает**:
- `bool`: `True`, если прокрутка прошла успешно, `False` - иначе.

**Вызывает исключения**:
- Возможны исключения, связанные с прокруткой (например,  проблемы с выполнением JavaScript кода).


### `locale`

**Описание**: Возвращает язык текущей страницы.

**Параметры**:
Нет.

**Возвращает**:
- `str`: Язык страницы (например, 'en', 'ru').

**Вызывает исключения**:
- Возможны исключения, связанные с анализом языка.


### `find_element(By, selector)`

**Описание**:  Находит элемент на странице по селектору.

**Параметры**:
- `By` (тип):  Константа из модуля `selenium.webdriver.common.by` (например, `By.ID`, `By.NAME`, `By.CSS_SELECTOR`).
- `selector` (str):  Селектор элемента.

**Возвращает**:
- Элемент (объект, представленный Selenium), если найден, иначе `None`.

**Вызывает исключения**:
- `NoSuchElementException`: Если элемент не найден.
- Возможны исключения, связанные с использованием Selenium.


### `current_url`

**Описание**: Возвращает текущий URL.

**Параметры**:
Нет.

**Возвращает**:
- `str`: Текущий URL.


**Вызывает исключения**:
- Возможны исключения, связанные с получением текущего URL.



### `window_focus()`

**Параметры**:
Нет.

**Возвращает**:
- Нет.

**Вызывает исключения**:
- Возможны исключения, связанные с изменением фокуса браузера.



## Примечания

- Убедитесь, что у вас установлены необходимые библиотеки (например, `selenium`).
- Необходимо установить соответствующий драйвер браузера (например, ChromeDriver для Chrome).
-  Пути к файлам настроек и других ресурсов должны быть настроены в глобальных настройках (`gs`).
