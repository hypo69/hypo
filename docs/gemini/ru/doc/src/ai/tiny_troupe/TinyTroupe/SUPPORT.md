# ИНСТРУКЦИЯ ПО ИСПОЛЬЗОВАНИЮ МОДУЛЯ SUPPORT

## Обзор

Данный модуль содержит информацию о том, как сообщать об ошибках, запрашивать новые функции и получать помощь по проекту TinyTroupe.

## Разделы

### Как сообщать об ошибках и получать помощь

Этот проект использует систему GitHub Issues для отслеживания ошибок и запросов новых функций.  Перед созданием новой проблемы, пожалуйста, проверьте существующие issue, чтобы избежать дублирования. Для новых проблем создавайте новый Issue.

Для получения помощи и ответов на вопросы по использованию этого проекта, пожалуйста, проконсультируйтесь с нашей Wiki или опубликуйте сообщение на нашем форуме обсуждений, оба находятся прямо здесь, на GitHub.

### Политика поддержки от Microsoft

Поддержка TinyTroupe ограничена ресурсами, указанными выше.
