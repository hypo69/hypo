# Игра WAR 2

## Обзор

Данный модуль реализует карточную игру "WAR 2" для двух игроков. Игра заключается в сравнении карт, и игрок с более старшей картой забирает карты со стола. Если карты равны, объявляется "война". Цель игры - собрать все карты колоды.

## Функции

### `create_deck`

**Описание**: Создает стандартную колоду из 52 карт.

**Параметры**:
- Нет

**Возвращает**:
- `list`: Список строк, представляющих карты в колоде.

### `deal_cards`

**Описание**: Раздает карты двум игрокам.

**Параметры**:
- `deck` (list): Список строк, представляющих колоду карт.

**Возвращает**:
- `tuple[list, list]`: Кортеж из двух списков, каждый из которых представляет карты игрока.

### `get_card_value`

**Описание**: Возвращает числовое значение карты (без учета масти).

**Параметры**:
- `card` (str): Строка, представляющая карту (например, "2C", "AH").

**Возвращает**:
- `int`: Числовое значение карты.

### `war`

**Описание**: Реализует логику "войны" в игре.

**Параметры**:
- `player_a` (list): Список карт игрока A.
- `player_b` (list): Список карт игрока B.
- `cards_on_table` (list): Список карт на столе.

**Возвращает**:
    -  `tuple[bool, bool, list, list, list]`: Кортеж, содержащий:
        -   `bool`: a_win_war - победа игрока A в войне (у игрока B меньше 4 карт)
        -   `bool`: b_win_war - победа игрока B в войне (у игрока A меньше 4 карт)
        -   `list`: cards_on_table - оставшиеся карты на столе
        -   `list`: player_a - колода игрока A после войны
        -   `list`: player_b - колода игрока B после войны

### `play_war`

**Описание**: Основная логика игры в войну.

**Параметры**:
- Нет

**Возвращает**:
- `None`: Функция не возвращает значения, но выводит результаты игры в консоль.
