Этот модуль содержит информацию о версии и другие метаданные для сценариев поставщика HB.

## Переменные

### `MODE`

**Описание**: Переменная, хранящая режим работы (например, 'dev', 'prod').


### `__version__`

**Описание**: Переменная, хранящая версию модуля.


### `__name__`

**Описание**: Переменная, хранящая имя модуля.


### `__doc__`

**Описание**: Документационная строка модуля.


### `__details__`

**Описание**: Переменная, содержащая дополнительные детали о модуле (например, описание версии).


### `__annotations__`

**Описание**: Содержит аннотации типов для переменных и функций в модуле.


### `__author__`

**Описание**: Имя(я) автора(ов) модуля.
