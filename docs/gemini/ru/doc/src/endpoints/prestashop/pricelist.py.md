# Модуль `pricelist`

## Обзор

Модуль `pricelist` предоставляет функциональность для взаимодействия с API PrestaShop с целью получения и обновления информации о ценах товаров. Он содержит класс `PriceListRequester`, который наследуется от класса `PrestaShop` и реализует методы для запроса цен, обновления источника данных и модификации цен товаров.

## Подробней

Модуль предназначен для использования в проекте `hypotez` для автоматизации процессов, связанных с управлением ценами в PrestaShop. Он обеспечивает абстракцию от деталей API PrestaShop и предоставляет удобный интерфейс для работы с ценами товаров.

## Классы

### `PriceListRequester`

**Описание**: Класс `PriceListRequester` предназначен для запроса и управления ценами товаров в PrestaShop. Он наследуется от класса `PrestaShop` и использует его для взаимодействия с API PrestaShop.

**Как работает класс**:
Класс `PriceListRequester` инициализируется с учетными данными API PrestaShop, которые используются для аутентификации при запросах к API. Он предоставляет методы для запроса цен товаров, обновления источника данных и модификации цен товаров.

**Методы**:

- `__init__(self, api_credentials)`: Инициализирует объект класса `PriceListRequester`.
- `request_prices(self, products)`: Запрашивает список цен для указанных товаров.
- `update_source(self, new_source)`: Обновляет источник данных для запроса цен.
- `modify_product_price(self, product, new_price)`: Модифицирует цену указанного товара.
