# Модуль hypotez/src/endpoints/prestashop/api/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`. Также он предоставляет информацию о проекте (название, версия, описание, автор и т.д.)

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с текущей директории, ищет вверх по дереву директорий, пока не найдет директорию, содержащую указанные файлы-маркеры.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для определения корневой директории проекта. По умолчанию включает `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта. Если корневая директория не найдена, возвращается путь к директории, где расположен текущий файл.

**Вызывает исключения**:

- Нет.


### `set_project_root`

**Описание**:  Найденная корневая директория проекта, используемая для построения путей к другим файлам и папкам.

**Параметры**: Нет.

**Возвращает**:

- `Path`: Путь к корневой директории проекта.


## Переменные

### `MODE`

**Описание**: Переменная, хранящая текущий режим работы приложения (например, 'dev', 'prod').  Значение по умолчанию: 'dev'.

### `__root__`

**Описание**:  Путь к корневой директории проекта, полученный с помощью функции `set_project_root()`.

### `settings`

**Описание**: Словарь настроек проекта, загруженный из файла `settings.json` в корневой директории проекта.  Значение по умолчанию: `None`.

**Обработка ошибок**: Если файл `settings.json` не найден или некорректно сформирован, переменная остается `None`, и используются значения по умолчанию.

### `doc_str`

**Описание**: Строковое представление документации, загруженное из файла `README.MD` в корневой директории. Значение по умолчанию: `None`.

**Обработка ошибок**: Если файл `README.MD` не найден или некорректно сформирован, переменная остается `None`, и используется значение по умолчанию.


### `__project_name__`

**Описание**: Название проекта, полученное из настроек (`settings`). Значение по умолчанию: `hypotez`.

### `__version__`

**Описание**: Версия проекта, полученная из настроек (`settings`). Значение по умолчанию: пустая строка.


### `__doc__`

**Описание**: Строковое представление документации проекта, полученное из файла README. Значение по умолчанию: пустая строка.

### `__details__`

**Описание**: Дополнительные детали о проекте. Значение по умолчанию: пустая строка.

### `__author__`

**Описание**: Автор проекта, полученный из настроек (`settings`). Значение по умолчанию: пустая строка.

### `__copyright__`

**Описание**: Авторские права, полученные из настроек (`settings`). Значение по умолчанию: пустая строка.

### `__cofee__`

**Описание**: Ссылка на спонсорскую страницу, предоставляющую возможность поблагодарить разработчика чашкой кофе. Значение по умолчанию: ссылка на Boosty.


## Использование

Модуль предоставляет функции для нахождения корневой директории проекта и загрузки настроек, что позволяет другим частям приложения использовать эти данные для выполнения своих задач.
