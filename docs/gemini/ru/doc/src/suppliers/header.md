# Модуль `hypotez/src/suppliers/header.py`

## Обзор

Данный модуль отвечает за инициализацию проекта, включая определение корневой директории проекта, загрузку настроек из файла `settings.json` и, опционально, документации из файла `README.MD`.  Он также определяет переменные, содержащие информацию о проекте (название, версия, описание, автор, копирайт, и т.д.).

## Функции

### `set_project_root`

**Описание**: Функция находит корневую директорию проекта, начиная с директории текущего файла. Поиск идет вверх по иерархии директорий, пока не будет найдена директория, содержащая один из указанных маркеров (файлы или директории).

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, используемых для определения корневой директории проекта. По умолчанию содержит `('pyproject.toml', 'requirements.txt', '.git')`.


**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращает директорию, в которой расположен текущий файл.

**Вызывает исключения**:

- Нет


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта.  Инициализируется функцией `set_project_root()`.


### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженные из файла `settings.json`.

**Значение по умолчанию**: `None`


### `doc_str`

**Описание**: Строка, содержащая содержимое файла `README.MD`.

**Значение по умолчанию**: `None`


### `__project_name__`

**Описание**: Название проекта. Значение взято из настроек `settings`. Если настроек нет, используется значение по умолчанию `'hypotez'`.


### `__version__`

**Описание**: Версия проекта. Значение взято из настроек `settings`. Если настроек нет, используется значение по умолчанию `''`.


### `__doc__`

**Описание**: Описание проекта. Значение взято из `doc_str`. Если `doc_str` не определен, используется значение по умолчанию `''`.


### `__details__`

**Описание**: Дополнительные детали о проекте. Значение по умолчанию `''`.


### `__author__`

**Описание**: Автор проекта. Значение взято из настроек `settings`. Если настроек нет, используется значение по умолчанию `''`.


### `__copyright__`

**Описание**: Копирайт проекта. Значение взято из настроек `settings`. Если настроек нет, используется значение по умолчанию `''`.


### `__cofee__`

**Описание**: Ссылка для пожертвования на чашечку кофе. Значение взято из настроек `settings`. Если настроек нет, используется значение по умолчанию.


## Обработка исключений

Обработка `FileNotFoundError` и `json.JSONDecodeError` в блоках `try...except` позволяет избежать аварийного завершения программы при отсутствии или некорректном формате файлов настроек и документации.
