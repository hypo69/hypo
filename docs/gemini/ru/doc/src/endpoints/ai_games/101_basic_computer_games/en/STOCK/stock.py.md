# STOCK

## Обзор

Игра "STOCK" представляет собой простую экономическую симуляцию, в которой игрок пытается заработать деньги, покупая и продавая акции в течение определенного количества раундов (дней). Цена акций изменяется случайным образом каждый раунд. Цель игры - максимизировать прибыль к концу игры.

## Функции
### `random.randint(a, b)`
**Описание**: Генерирует случайное целое число в диапазоне от `a` до `b` включительно.

**Параметры**:
- `a` (int): Нижняя граница диапазона.
- `b` (int): Верхняя граница диапазона.

**Возвращает**:
- `int`: Случайное целое число.

### `random.uniform(a, b)`
**Описание**: Генерирует случайное число с плавающей точкой в диапазоне от `a` до `b`.

**Параметры**:
- `a` (float): Нижняя граница диапазона.
- `b` (float): Верхняя граница диапазона.

**Возвращает**:
- `float`: Случайное число с плавающей точкой.

### `round(number, ndigits)`
**Описание**: Округляет число до указанного количества знаков после запятой.

**Параметры**:
- `number` (float): Число для округления.
- `ndigits` (int): Количество знаков после запятой.

**Возвращает**:
- `float`: Округленное число.

### `input(prompt)`
**Описание**: Выводит сообщение `prompt` и возвращает введенную пользователем строку.

**Параметры**:
- `prompt` (str): Сообщение для пользователя.

**Возвращает**:
- `str`: Введенная пользователем строка.

### `int(x)`
**Описание**: Преобразует значение `x` к целому числу.

**Параметры**:
- `x` (str): Значение для преобразования.

**Возвращает**:
- `int`: Целое число.

**Вызывает исключения**:
- `ValueError`: Если `x` не может быть преобразовано в целое число.

### `str.lower()`
**Описание**: Преобразует строку в нижний регистр.

**Параметры**:
- Строка для преобразования

**Возвращает**:
- `str`: Строка в нижнем регистре.

## Переменные
### `numberOfDays`
**Описание**: Общее количество дней игры.

**Тип**: `int`

**Значение**: 20

### `currentDay`
**Описание**: Текущий день игры.

**Тип**: `int`

**Значение**: 1

### `money`
**Описание**: Количество денег у игрока.

**Тип**: `int`

**Значение**: 1000

### `stockPrice`
**Описание**: Текущая цена акции.

**Тип**: `float`

**Значение**: Случайное значение от 5 до 50.

### `numberOfStocks`
**Описание**: Количество акций у игрока.

**Тип**: `int`

**Значение**: 0

## Игровой цикл

### `while currentDay <= numberOfDays:`
**Описание**: Основной игровой цикл, продолжающийся пока текущий день не превысит общее количество дней.

**Логика**:

1.  **Обновление цены акции:**
    -   `stockPrice` изменяется в пределах от 90% до 110% от текущей цены, значение округляется до двух знаков после запятой.
2.  **Вывод текущего состояния:**
    -   Выводится информация о текущем дне, цене акции, количестве акций и денег.
3.  **Запрос действия пользователя:**
    -   Пользователь вводит действие: купить (`buy`), продать (`sell`) или пропустить день, нажав Enter.

## Обработка действий пользователя

**Логика**:

1.  **Если действие `buy`:**
    -   Запрашивается количество акций для покупки.
    -   Проверяется, достаточно ли денег для покупки.
    -   Если достаточно, деньги уменьшаются на стоимость покупки, а количество акций увеличивается.
    -   Иначе, выводится сообщение о недостатке денег.
    -   Обрабатывается исключение `ValueError`, если пользователь вводит некорректное значение.

2.  **Если действие `sell`:**
    -   Запрашивается количество акций для продажи.
    -   Проверяется, достаточно ли акций для продажи.
    -   Если достаточно, деньги увеличиваются на стоимость продажи, а количество акций уменьшается.
    -   Иначе, выводится сообщение о недостатке акций.
    -   Обрабатывается исключение `ValueError`, если пользователь вводит некорректное значение.

3.  **Если действие не `buy` и не `sell`:**
    -   Игра переходит к следующему дню.

4.  **Увеличение номера текущего дня:**
    -  `currentDay` увеличивается на 1.
    -  Выводится разделитель `-----------------` для визуального разделения дней.
