# Задача No 751
## Почти золотое сечение

## Условия

Рассмотрим последовательность $a_n$, определенную следующим образом:
* $a_1 = 0$
* $a_{n+1} = \text{frac}(1/a_n)$, где $\text{frac}(x)$ представляет собой дробную часть числа $x$.

Например:
* $a_1 = 0$
* $a_2 = \text{frac}(1/0) = \text{frac}(+\infty) = 0$
* $a_3 = \text{frac}(1/0) = 0$
...
Не очень интересный случай!

Давайте изменим определение, добавив дополнительную константу $\phi$, золотое сечение, т.е.  $\phi = \frac{1 + \sqrt{5}}{2} \approx 1.6180339887$.

* $a_1 = 0$
* $a_{n+1} = \text{frac}(\phi/a_n)$

Теперь последовательность становится более интересной! 
* $a_1 = 0$
* $a_2 = \text{frac}(\phi/0) = 0$
* $a_3 = \text{frac}(\phi/0) = 0$
...

Все еще не очень интересно. Давайте введем следующее правило. Всякий раз, когда $a_n=0$, мы используем последнее ненулевое значение $a_i$, где $i<n$, чтобы вычислить следующий член.

Окончательное определение:

* $a_1 = \phi - 1$
* $a_{n+1} = \text{frac}(\phi/a_n)$, если $a_n \ne 0$
* $a_{n+1} = \text{frac}(\phi/a_k)$, где $k < n$ и $a_k \ne 0$, а $k$ наибольшее возможное, если $a_n = 0$.

* $a_1 = 0.6180339887\dots$
* $a_2 = 0.6180339887\dots$
* $a_3 = 0.6180339887\dots$
* $a_4 = 0.6180339887\dots$
* $a_5 = 0.6180339887\dots$
* $a_6 = 0.6180339887\dots$
* $a_7 = 0.6180339887\dots$
* $a_8 = 0.6180339887\dots$
* $a_9 = 0.6180339887\dots$
* $a_{10} = 0.6180339887\dots$
* $a_{11} = 0.6180339887\dots$
* $a_{12} = 0.6180339887\dots$
* $a_{13} = 0.6180339887\dots$
* $a_{14} = 0.6180339887\dots$
* $a_{15} = 0.6180339887\dots$

* $a_{20} = 0.6180339887\dots$
* ...
* $a_{100} = 0.6180339887\dots$

Пусть $s(n)$ - строка, составленная путем записи первых $n$ цифр после десятичной точки последовательности $a_i$.
Таким образом, $s(1) = "6"$, $s(2) = "61"$, $s(3) = "618"$, $s(10) = "6180339887"$ и так далее.
