# Модуль hypotez/src/webdriver/bs/header.py

## Обзор

Данный модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`. Он также загружает описание проекта из файла `README.MD`.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с текущей директории файла, и проходя вверх по иерархии директорий. Останавливается при обнаружении директории, содержащей один из указанных файлов-маркеров.

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, которые будут использованы для определения корневой директории проекта.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращает директорию, в которой расположен текущий скрипт.


**Вызывает исключения**:

- Нет


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта.

**Тип**: `Path`


##  Инициализация и обработка данных

**Описание**: Загрузка настроек из файла `settings.json` и документации из `README.MD`. Обрабатывает возможные исключения (файлы не найдены, некорректный JSON).


### `settings`

**Описание**: Словарь со значениями настроек.

**Тип**: `dict`

**Значение по умолчанию**: `None`


### `doc_str`

**Описание**: Строковое представление документации проекта.

**Тип**: `str`

**Значение по умолчанию**: `None`


##  Переменные проекта

**Описание**:  Переменные, содержащие информацию о проекте, полученную из `settings.json`.


### `__project_name__`

**Описание**: Название проекта.

**Тип**: `str`

**Значение по умолчанию**: `hypotez`


### `__version__`

**Описание**: Версия проекта.

**Тип**: `str`

**Значение по умолчанию**: `''`


### `__doc__`

**Описание**: Документация проекта (из `README.MD`).

**Тип**: `str`

**Значение по умолчанию**: `''`


### `__details__`

**Описание**: Подробности о проекте (пустая строка по умолчанию).

**Тип**: `str`

**Значение по умолчанию**: `''`


### `__author__`

**Описание**: Автор проекта.

**Тип**: `str`

**Значение по умолчанию**: `''`


### `__copyright__`

**Описание**: Авторские права.

**Тип**: `str`

**Значение по умолчанию**: `''`


### `__cofee__`

**Описание**: Ссылка на платёжную систему для поддержки разработчика.

**Тип**: `str`

**Значение по умолчанию**: `Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69`
