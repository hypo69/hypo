# Модуль hypotez/src/utils/convertors/csv.py

## Обзор

Модуль `hypotez/src/utils/convertors/csv.py` предоставляет утилиты для конвертации данных в формате CSV в формат JSON и обратно, а также для работы с данными в формате CSV.  Модуль содержит функции для чтения CSV в словарь, в объекты `SimpleNamespace`, а также для сохранения данных CSV в файл.

## Функции

### `csv2dict`

**Описание**: Функция конвертирует данные из CSV файла в словарь.

**Параметры**:
- `csv_file` (str | Path): Путь к файлу CSV.

**Возвращает**:
- `dict | None`: Словарь, содержащий данные из файла CSV, или `None`, если произошла ошибка чтения файла.

**Возможные исключения**:
- `Exception`: Возникает при проблемах с чтением файла CSV.


### `csv2ns`

**Описание**: Функция конвертирует данные из CSV файла в объекты `SimpleNamespace`.

**Параметры**:
- `csv_file` (str | Path): Путь к файлу CSV.

**Возвращает**:
- `SimpleNamespace | None`: Объект `SimpleNamespace`, содержащий данные из файла CSV, или `None`, если произошла ошибка чтения файла.

**Возможные исключения**:
- `Exception`: Возникает при проблемах с чтением файла CSV.


### `csv_to_json`

**Описание**: Конвертирует CSV-файл в JSON и сохраняет результат в файл JSON.

**Параметры**:
- `csv_file_path` (str | Path): Путь к файлу CSV.
- `json_file_path` (str | Path): Путь к файлу JSON для сохранения результата.
- `exc_info` (bool, optional): Если `True`, то в лог будут записаны подробности об ошибке (трейсбэк). По умолчанию `True`.

**Возвращает**:
- `List[Dict[str, str]] | None`: Список словарей с данными из JSON, или `None` в случае ошибки.

**Пример использования**:

```python
>>> json_data = csv_to_json('dialogue_log.csv', 'dialogue_log.json')
>>> print(json_data)
[{'role': 'user', 'content': 'Hello'}, {'role': 'assistant', 'content': 'Hi there!'}]
```


**Возможные исключения**:
- `Exception`:  Возникает при проблемах с чтением файла CSV или записи в файл JSON.  Информация об ошибке будет записана в лог.
