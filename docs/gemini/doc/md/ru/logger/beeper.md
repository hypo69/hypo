# Модуль beeper

## Обзор

Модуль `beeper` предоставляет инструменты для генерации звуковых сигналов (бипов) на основе уровня событий, используя библиотеки `asyncio`, `winsound`, `time` и `enum`.  Модуль позволяет настраивать частоту и продолжительность звукового сигнала, а также определяет разные звуковые мелодии для разных уровней событий.


## Классы

### `BeepLevel`

**Описание**: Класс перечислений `BeepLevel` определяет типы событий и соответствующие им мелодии.

**Перечислители**:
- `SUCCESS`
- `INFO`
- `ATTENTION`
- `WARNING`
- `DEBUG`
- `ERROR`
- `LONG_ERROR`
- `CRITICAL`
- `BELL`


### `BeepHandler`

**Описание**: Обработчик бипов, который реагирует на события, определяемые классом `BeepLevel`.

**Методы**:
- `emit(record)`:  Обрабатывает запись логов и воспроизводит соответствующий бип, основываясь на уровне события `record["level"]`.


## Функции

### `silent_mode`

**Описание**: Декоратор `silent_mode`, который позволяет управлять режимом "беззвучия" для функций, его использующих.

**Параметры**:
- `func` (функция): Функция, которую нужно декорировать.

**Возвращает**:
- Обернутая функция, которая пропускает вызов `beep` при включённом режиме "беззвучия".


### `Beeper.beep`

**Описание**: Функция для воспроизведения звукового сигнала (бипа) с указанием уровня события, частоты и продолжительности.

**Параметры**:
- `level` (`BeepLevel` или str, опционально `BeepLevel.INFO`): Уровень события (например, `BeepLevel.ERROR` или `"error"`). Доступные уровни: `info`, `attention`, `warning`, `debug`, `error`, `long_error`, `critical`, `bell` или соответствующие перечислители `BeepLevel`.
- `frequency` (int, опционально 400): Частота звукового сигнала.
- `duration` (int, опционально 1000): Продолжительность звукового сигнала в миллисекундах.

**Возвращает**:
- `None`

**Обрабатываемые исключения**:
- `Exception`: Выводится сообщение об ошибке при воспроизведении звука.


## Постоянные

### `MODE`

**Описание**:  Постоянная, определяющая режим работы бипера.


## Модульные зависимости

- `asyncio`
- `winsound`
- `time`
- `enum`
- `typing`
