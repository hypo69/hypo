# Модуль parse_data_types

## Обзор

Этот модуль предназначен для анализа входных данных различных типов (JSON, CSV, XLS, объекты Python) и преобразования их в структурированный формат для создания PDF-документов.  Он идентифицирует тип данных, предоставляет структурированное представление, включая таблицы, заголовки и иерархические списки, а также предлагает рекомендации по форматированию для создания профессионально выглядящих PDF-документов.

## Функции

### `analyze_data`

**Описание**:  Функция анализа входных данных, определяющая их тип и возвращающая структурированное представление.

**Параметры**:

- `data` (Любой тип данных): Входные данные для анализа.  Поддерживаются JSON, CSV, XLS и объекты Python.

**Возвращает**:

- `dict`:  Структурированные данные для создания PDF.  Ключи словаря отражают тип данных и структуру данных. Примеры:
    - Для JSON: `{"type": "JSON", "data": [{"key": "value"}]}`
    - Для CSV: `{"type": "CSV", "table": [{"header1": "value1", "header2": "value2"}]}`
    - Для XLS: `{"type": "XLS", "sheets": [{"name": "Sheet1", "data": [{"col1": "val1", "col2": "val2"}]}]}`
    - Для Python объекта: `{"type": "PythonObject", "object_type": "type_name", "data": {...}}`

**Вызывает исключения**:

- `TypeError`: Если тип входных данных не поддерживается.
- `ValueError`: Если входные данные некорректны.
- `IOError`: При ошибках ввода/вывода при обработке файлов (если это применимо).


## Пример использования

```python
# Пример вызова функции analyze_data
import json
input_data = {"name": "John Doe", "age": 30}
parsed_data = analyze_data(json.dumps(input_data))
# parsed_data будет содержать структурированные данные для создания PDF,
# например, для JSON, с указанием типа данных и структурированных данных.
```

## Рекомендации по форматированию

* **Таблицы:** Используйте табличный формат (`|---|---|`) для представления данных CSV и XLS.  Определите ширину колонок, шрифты и стили.
* **Списки:** Используйте маркированные списки (`*`, `-`, `+`) для иерархического отображения данных.
* **Заголовки:** Используйте заголовки разных уровней для структурирования данных.
* **Шрифты:**  Используйте читаемые шрифты, особенно для большого объема текста.
* **Цвета:**  Используйте цвета для выделения важных элементов данных.
