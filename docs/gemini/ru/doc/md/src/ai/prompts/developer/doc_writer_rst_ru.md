# Модуль для работы с ассистентом программиста

## Обзор

Этот модуль содержит класс `CodeAssistant`, предназначенный для взаимодействия с моделями ИИ (такими как Google Gemini и OpenAI) для выполнения задач по обработке кода.  Модуль предоставляет инструменты для анализа кода, создания документации и других задач.

## Примеры использования

```python
from ai.prompts.developer.code_assistant import CodeAssistant

assistant = CodeAssistant(role='code_checker', lang='ru', model=['gemini'])
assistant.process_files()
```


## Классы

### `CodeAssistant`

**Описание**: Класс `CodeAssistant` предоставляет интерфейс для взаимодействия с моделями ИИ.

**Атрибуты**

- `role` (str): Роль ассистента (например, 'code_checker').
- `lang` (str): Язык, на котором будет работать ассистент.
- `model` (list): Список используемых моделей ИИ.

**Методы**

#### `process_files`

**Описание**: Метод `process_files` анализирует файлы с кодом и выполняет необходимые действия в соответствии с заданной ролью.

**Параметры**

- `files` (list): Список путей к файлам для обработки.

**Возвращает**

- `list`: Список результатов обработки файлов.


**Исключения**

- `ValueError`: Возникает, если введены некорректные данные.
- `FileNotFoundError`: Возникает, если один или несколько файлов из списка `files` не найдены.

## Обработка исключений

### `FileNotFoundError`

**Описание**: Исключение `FileNotFoundError` возникает, когда один из файлов, указанный в списке `files` для обработки, не найден.

**Параметры**

- `file` (str): Путь к файлу, который не был найден.

**Пример использования**

```python
# Обработка исключения, чтобы продолжить выполнение, если файл не найден
try:
    process_file(file)
except FileNotFoundError as ex:
    handle_exception(ex)
```
