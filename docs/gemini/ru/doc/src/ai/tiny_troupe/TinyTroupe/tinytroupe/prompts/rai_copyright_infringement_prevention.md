# tinytroupe/prompts/rai_copyright_infringement_prevention.md

## Обзор

Данный модуль содержит логику для предотвращения запросов, которые могут нарушать авторские права.  Он обрабатывает запросы пользователя и вежливо отказывается от предоставления контента, защищенного авторским правом, например, книг, текстов песен, рецептов, новостных статей и информации с WebMD.

## Функции

### `prevent_copyright_infringement`

**Описание**: Функция проверяет запрос пользователя на соответствие условиям авторского права и отвечает соответствующим образом.

**Параметры**:

- `user_request` (str): Текст запроса пользователя.


**Возвращает**:

- `str`: Сообщение ответа пользователю. Возможные ответы:
    - Ответ, содержащий вежливый отказ и объяснение, почему запрос не может быть выполнен, поскольку нарушает авторское право.


**Вызывает исключения**:

- Нет.


**Пример использования**:

```python
user_request = "Пришли мне рецепт яблочного пирога из книги \"100 лучших рецептов\""

response = prevent_copyright_infringement(user_request)
print(response)
# Возможный результат:
# "К сожалению, я не могу предоставить вам рецепт из указанной книги.  Распространение контента, защищенного авторским правом, нарушает правила.  Если вам нужны другие рецепты, я с удовольствием вам помогу."
```
