# Модуль utils

## Обзор

Этот модуль содержит набор небольших, полезных утилит, предназначенных для упрощения повседневных задач программирования. Модуль включает инструменты для конвертации данных, работы с файлами и формата вывода. Это позволяет ускорить разработку, предоставляя простые и переиспользуемые функции.

## Функции

### `csv2dict`

**Описание**: Функция преобразует CSV-файл в словарь Python.

**Параметры**:
- `filepath` (str): Путь к CSV-файлу.

**Возвращает**:
- `dict | None`: Словарь, содержащий данные из CSV-файла, или `None`, если возникла ошибка.

**Вызывает исключения**:
- `FileNotFoundError`: Если файл не найден.
- `ValueError`: Если файл не является корректным CSV-файлом.


### `json2xls`

**Описание**: Функция преобразует JSON-файл в XLSX-файл.

**Параметры**:
- `filepath` (str): Путь к JSON-файлу.

**Возвращает**:
- `bool`: `True`, если преобразование выполнено успешно, `False` в противном случае.

**Вызывает исключения**:
- `FileNotFoundError`: Если файл не найден.
- `ValueError`: Если файл не является корректным JSON-файлом.


### `save_text_file`

**Описание**: Функция сохраняет текст в файл.

**Параметры**:
- `filepath` (str): Путь к файлу для сохранения.
- `text` (str): Текст, который необходимо сохранить.

**Возвращает**:
- `bool`: `True`, если сохранение выполнено успешно, `False` в противном случае.

**Вызывает исключения**:
- `IOError`: Если произошла ошибка ввода/вывода при работе с файлом.


### `read_text_file`

**Описание**: Функция считывает текст из файла.

**Параметры**:
- `filepath` (str): Путь к файлу.

**Возвращает**:
- `str | None`: Содержимое файла в виде строки или `None`, если файл не найден или при ошибке чтения.

**Вызывает исключения**:
- `FileNotFoundError`: Если файл не найден.
- `IOError`: Если произошла ошибка ввода/вывода.

## Классы

### `TextToImageGenerator`

**Описание**:  Класс для генерации изображений из текста.

**Методы**:
- `generate_image(text, output_path)`: Генерирует изображение из переданного текста и сохраняет его по указанному пути.


### `PDFUtils`

**Описание**: Класс для работы с PDF-файлами.

**Методы**:
- `extract_text(filepath)`: Извлекает текст из PDF-файла.


### `TimeoutCheck`

**Описание**: Класс для проверки таймаута.

**Методы**:
- `check_timeout(timeout)`: Проверяет таймаут, возвращает результат.


### и т.д.


## Подмодули

- `.convertors`: Содержит функции для конвертации данных (например, `csv2dict`, `json2xls`).
- `.csv`: Содержит функции для работы с CSV-файлами.
- `.date_time`: Содержит функции и классы для работы со временем и датой.
- `.file`: Содержит функции для работы с файлами (например, чтение, сохранение, получение списка файлов).
- `.image`: Содержит функции для работы с изображениями.
- `.jjson`: Содержит функции для работы с JSON-данными (например, `j_loads`, `j_dumps`).
- `.pdf`: Содержит класс `PDFUtils`.
- `.printer`: Содержит функцию `pprint` для красивой печати данных.
- `.string`: Содержит классы для работы с текстом.
- `.url`: Содержит функции для работы с URL-адресами.
- `.video`: Содержит функции для работы с видеофайлами.
- `.path`: Содержит функцию `get_relative_path`.
