# `store.json`

## Обзор

Файл `store.json` содержит JSON-объект с локаторами для элементов страницы магазина Walmart.
Эти локаторы используются для автоматизации взаимодействия с веб-страницей.

## Локаторы

### Поисковая строка

**Описание**: Локатор для поля ввода поисковой строки.

```json
"search_field": "#global-search-input"
```

### Кнопка поиска

**Описание**: Локатор для кнопки отправки поиска.

```json
"search_button": ".header-GlobalSearch-submit"
```

### Карточка товара

**Описание**: Локатор для контейнера карточки товара.

```json
"product_card": ".search-result-gridview-items > li:nth-child(1)"
```

### Заголовок товара

**Описание**: Локатор для заголовка товара в карточке.

```json
"product_title": "a > span > span:nth-child(1)"
```

### Цена товара

**Описание**: Локатор для цены товара в карточке.

```json
"product_price": "div > div > div > div:nth-child(1) > div > span > span > span"
```

### Кнопка добавления товара в корзину

**Описание**: Локатор для кнопки добавления товара в корзину.

```json
"add_to_cart_button": "div > div:nth-child(2) > div:nth-child(2) > button"
```

### Модальное окно корзины

**Описание**: Локатор для модального окна корзины, которое появляется после добавления товара.

```json
"cart_modal": "#cart-root"
```

### Кнопка перехода в корзину

**Описание**: Локатор для кнопки перехода в корзину из модального окна.

```json
"go_to_cart_button": "div > div > div > div > div > div > div > div > div:nth-child(3) > a"
```
