# Модуль `hypotez/src/suppliers/aliexpress/campaign/html_generators.py`

## Обзор

Этот модуль предоставляет классы для генерации HTML-контента для рекламных кампаний AliExpress. Он позволяет создавать HTML-страницы для отдельных продуктов, категорий и всей рекламной кампании.

## Классы

### `ProductHTMLGenerator`

**Описание**: Класс для генерации HTML-страниц для отдельных продуктов.

**Статические методы**:

#### `set_product_html`

**Описание**: Создаёт HTML-файл для отдельного продукта.

**Параметры**:

- `product` (SimpleNamespace): Объект содержащий детали продукта, которые нужно включить в HTML.
- `category_path` (str | Path): Путь для сохранения HTML-файла.

**Возвращает**:
-  Не используется.


### `CategoryHTMLGenerator`

**Описание**: Класс для генерации HTML-страниц для категорий продуктов.

**Статические методы**:

#### `set_category_html`

**Описание**: Создаёт HTML-файл для категории.

**Параметры**:

- `products_list` (list[SimpleNamespace] | SimpleNamespace): Список продуктов, которые нужно включить в HTML. Может быть списком или отдельным продуктом.
- `category_path` (str | Path): Путь для сохранения HTML-файла.

**Возвращает**:
- Не используется.


### `CampaignHTMLGenerator`

**Описание**: Класс для генерации HTML-страниц для рекламных кампаний.

**Статические методы**:

#### `set_campaign_html`

**Описание**: Создаёт HTML-файл для рекламной кампании, перечисляющий все категории.

**Параметры**:

- `categories` (list[str]): Список имён категорий.
- `campaign_path` (str | Path): Путь для сохранения HTML-файла.

**Возвращает**:
- Не используется.

## Модульные переменные

### `MODE`

**Описание**: Переменная, вероятно, определяющая режим работы (например, 'dev', 'prod').  Значение по умолчанию `'dev'`.

## Замечания

- Для создания HTML-файлов используются шаблоны с вставками данных о продуктах.
- Используется Bootstrap для стилизации HTML.
- Используется функция `save_text_file` из другого модуля для сохранения сгенерированного HTML в файл.
