## Установка

Для использования `Tiny Utils` клонируйте репозиторий и установите необходимые зависимости, как указано в файле `requirements.txt`.

```bash
git clone https://github.com/hypo69/tiny-utils.git
cd tiny_utils
pip install -r requirements.txt
```

## Обзор модулей

Библиотека содержит несколько подмодулей, каждый из которых выполняет определённую задачу:

- **Преобразователи**: Модули для преобразования форматов данных, таких как текст в изображение, webp в png, JSON, XML, кодирование Base64 и многое другое.
- **Утилиты для строк**: Инструменты для расширенной работы со строками.
- **Утилиты для файлов**: Функции для работы с файлами и манипуляции с ними.
- **Утилиты для даты и времени**: Инструменты для форматирования даты и времени.
- **Утилиты FTP**: Функции для работы с FTP.
- **Утилиты для изображений**: Основные функции обработки изображений.
- **Утилиты для PDF**: Утилиты для работы с файлами PDF, включая конвертацию, слияние, разделение и извлечение текста.
- **Утилиты для принтера**: Функции для отправки данных на принтер.

## Описание модулей

### Преобразователи

Модуль `convertors` содержит утилиты для преобразования данных между форматами. Эти модули могут обрабатывать различные типы данных, от CSV до JSON и текста до изображений.

#### Файлы:

- `text2png.py`: Преобразует текстовые данные в файл изображения PNG.
- `tts.py`: Преобразует текст в речь и сохраняет его в аудиофайл.
- `webp2png.py`: Преобразует изображения из формата WebP в формат PNG.
- `xls.py`: Обрабатывает преобразования и манипуляции с файлами XLS.
- `xml2dict.py`: Преобразует данные XML в словарь Python.
- `base64.py`: Кодирует или декодирует данные с помощью кодирования Base64.
- `csv.py`: Предоставляет инструменты для парсинга и манипуляций с CSV.
- `dict.py`: Утилиты для работы со словарями Python.
- `html.py`: Преобразует HTML-контент в различные форматы.
- `json.py`: Утилиты для парсинга и манипуляций с JSON.
- `md2dict.py`: Преобразует Markdown-контент в словарь.
- `ns.py`: Специализированные утилиты для преобразования пространств имён.
