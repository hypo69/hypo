# Модуль hypotez/src/utils/convertors/xls.py

## Обзор

Этот модуль предоставляет функции для работы с файлами Excel (.xls). Он содержит функции для чтения данных из файла Excel в формате словаря и записи данных в файл Excel.

## Функции

### `xls2dict`

**Описание**: Функция для чтения файла Excel и преобразования его содержимого в словарь.

**Параметры**:
- `xls_file` (str | Path): Путь к файлу Excel.

**Возвращает**:
- `dict | None`: Словарь, содержащий данные из файла Excel, или `None` в случае ошибки.


## Модуль `src.utils.xls`

**Описание**:  Данный модуль используется для работы с Excel-файлами (`.xls`).  Он содержит функции для чтения и записи данных в формате словаря.
