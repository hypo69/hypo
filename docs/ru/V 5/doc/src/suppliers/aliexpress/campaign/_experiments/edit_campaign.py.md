# Модуль `edit_campaign.py`

## Обзор

Модуль предназначен для редактирования рекламных кампаний на AliExpress. Он предоставляет функциональность для обработки и изменения параметров кампаний, а также для работы с категориями кампаний.

## Подробней

Этот модуль является частью системы управления рекламными кампаниями на AliExpress. Он включает в себя классы и функции для автоматизации процесса редактирования и обновления кампаний, а также для обработки различных категорий товаров. В модуле используются вспомогательные функции из других модулей проекта `hypotez`, таких как `src.utils` и `src.suppliers.aliexpress.campaign`.

### `AliCampaignEditor`

**Описание**: Класс предназначен для редактирования рекламных кампаний на AliExpress.
