# Модуль hypotez/src/webdriver/playwright/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`. Он также обрабатывает потенциальные ошибки при чтении файла настроек и файла `README.MD`.  Модуль инициализирует переменные, представляющие имя проекта, версию, документацию, дополнительные детали, автора, копирайт и ссылку на спонсорскую поддержку.

## Функции

### `set_project_root`

**Описание**:  Ищет корневую директорию проекта, начиная от текущего файла и двигаясь вверх по директориям. Останавливается на первой директории, содержащей один из указанных файлов или директорий (pyproject.toml, requirements.txt, .git).

**Параметры**:

- `marker_files` (tuple): Кортеж из имен файлов или директорий, которые используются для определения корневой директории проекта. По умолчанию (`pyproject.toml`, `requirements.txt`, `.git`).

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена, иначе директория, в которой расположен скрипт.

**Вызывает исключения**:

- Нет.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта. Инициализируется функцией `set_project_root`.


### `settings`

**Описание**: Словарь настроек, загруженный из файла `settings.json`.

**Возвращает**:

- `dict`: Словарь с настройками, или `None` если файл не найден или поврежден.

**Обработка исключений**:

- `FileNotFoundError`: Если файл `settings.json` не найден.
- `json.JSONDecodeError`: Если файл `settings.json` содержит некорректный JSON.

### `doc_str`

**Описание**: Строка с содержимым файла `README.MD`.

**Возвращает**:

- `str`: Содержимое файла `README.MD`, или `None` если файл не найден или поврежден.

**Обработка исключений**:

- `FileNotFoundError`: Если файл `README.MD` не найден.
- `json.JSONDecodeError`:  Если файл `README.MD` содержит некорректный формат.


### `__project_name__`

**Описание**: Имя проекта. Получается из настроек `settings` или имеет значение по умолчанию "hypotez".

**Тип данных**: `str`

### `__version__`

**Описание**: Версия проекта. Получается из настроек `settings` или имеет значение по умолчанию "".

**Тип данных**: `str`

### `__doc__`

**Описание**: Документация проекта. Получается из `doc_str` или имеет значение по умолчанию "".

**Тип данных**: `str`

### `__details__`

**Описание**: Дополнительные детали проекта. Имеет значение по умолчанию "".

**Тип данных**: `str`

### `__author__`

**Описание**: Автор проекта. Получается из настроек `settings` или имеет значение по умолчанию "".

**Тип данных**: `str`

### `__copyright__`

**Описание**: Авторские права. Получается из настроек `settings` или имеет значение по умолчанию "".

**Тип данных**: `str`

### `__cofee__`

**Описание**: Ссылка на спонсорскую поддержку разработчика (кофе). Получается из настроек `settings` или имеет значение по умолчанию.

**Тип данных**: `str`
