## תיאור המשחק

**חוקי המשחק:**

1. המחשב בוחר מספר שלם אקראי בין 1 ל-100.
2. השחקן מזין את הניחושים שלו לגבי המספר המסתורי.
3. לאחר כל ניסיון, המחשב מודיע אם המספר שהוזן היה נמוך מדי, גבוה מדי או נוחש.
4. המשחק ממשיך עד שהשחקן מנחש את המספר המסתורי.

**אלגוריתם:**

1. אתחל את מונה הניסיונות ל-0.
2. צור מספר אקראי בטווח של 1 עד 100.
3. התחל לולאה "כל עוד המספר לא נוחש":
    3.1. הגדל את מונה הניסיונות ב-1.
    3.2. בקש מהשחקן להזין מספר.
    3.3. אם המספר שהוזן שווה למספר המסתורי, הדפס את ההודעה "ניחשתי" ועבור לשלב 4.
    3.4. אם המספר שהוזן קטן מהמספר המסתורי, הדפס את ההודעה "נמוך מדי".
    3.5. אם המספר שהוזן גדול מהמספר המסתורי, הדפס את ההודעה "גבוה מדי".
4. הדפס את ההודעה "ניסיונות {מונה הניסיונות}"
5. סוף המשחק.

## קוד

```python
"""
BUZZWD:
=================
Сложность: 4
-----------------
Игра "BUZZWD" представляет собой простую числовую игру, где компьютер генерирует случайное число, а игрок должен угадать его, вводя числа по очереди. После каждого ввода программа сообщает, является ли введенное число "слишком высоким", "слишком низким" или же оно угадано. Игра завершается, когда игрок правильно угадывает число.

Правила игры:
1. Компьютер выбирает случайное целое число от 1 до 100.
2. Игрок вводит свои предположения о загаданном числе.
3. После каждой попытки компьютер сообщает, было ли введенное число слишком низким, слишком высоким или угадано.
4. Игра продолжается до тех пор, пока игрок не угадает загаданное число.
-----------------
Алгоритм:
1.  Установить счетчик попыток в 0.
2.  Сгенерировать случайное число в диапазоне от 1 до 100.
3.  Начать цикл "пока число не угадано":
    3.1 Увеличить счетчик попыток на 1.
    3.2 Запросить у игрока ввод числа.
    3.3 Если введенное число равно загаданному числу, вывести сообщение "УГАДАНО" и перейти к шагу 4.
    3.4 Если введенное число меньше загаданного числа, вывести сообщение "СЛИШКОМ МАЛО".
    3.5 Если введенное число больше загаданного числа, вывести сообщение "СЛИШКОМ МНОГО".
4. Вывести сообщение "ПОПЫТОК {счетчик попыток}"
5. Конец игры.
-----------------

"""
import random

# Инициализация счетчика попыток
numberOfGuesses = 0
# Генерируем случайное число от 1 до 100
targetNumber = random.randint(1, 100)

# Основной игровой цикл
while True:
    # Увеличиваем количество попыток
    numberOfGuesses += 1
    # Запрашиваем ввод числа у пользователя
    try:
        userGuess = int(input("Введите число: "))
    except ValueError:
        print("Пожалуйста, введите целое число.")
        continue

    # Проверяем, угадано ли число
    if userGuess == targetNumber:
        print("УГАДАНО")
        print(f"ПОПЫТОК {numberOfGuesses}")
        break  # Завершаем цикл, если число угадано
    elif userGuess < targetNumber:
        print("СЛИШКОМ МАЛО")  # Сообщаем, что загаданное число больше
    else:
        print("СЛИШКОМ МНОГО")  # Сообщаем, что загаданное число меньше

"""
Объяснение кода:
1.  **Импорт модуля `random`**:
   -  `import random`: Импортирует модуль `random`, который используется для генерации случайного числа.
2.  **Инициализация переменных**:
    -   `numberOfGuesses = 0`: Инициализирует переменную `numberOfGuesses` для подсчета попыток игрока.
    -   `targetNumber = random.randint(1, 100)`: Генерирует случайное целое число в диапазоне от 1 до 100 и сохраняет его в `targetNumber`.
3.  **Основной цикл `while True:`**:
    -   Бесконечный цикл, который продолжается до тех пор, пока игрок не угадает число (будет выполнена команда `break`).
    -   `numberOfGuesses += 1`: Увеличивает счетчик попыток на 1 при каждом новом витке цикла.
    -   **Ввод данных**:
        -   `try...except ValueError`: Блок try-except обрабатывает возможные ошибки ввода. Если пользователь введет не целое число, то будет выведено сообщение об ошибке.
        -   `userGuess = int(input("Введите число: "))`: Запрашивает у пользователя число и преобразует его в целое число, сохраняя результат в `userGuess`.
    -   **Условие победы**:
        -   `if userGuess == targetNumber:`: Проверяет, равно ли введенное число загаданному числу.
        -   `print("УГАДАНО")`: Выводит сообщение "УГАДАНО".
        -    `print(f"ПОПЫТОК {numberOfGuesses}")`: Выводит сообщение о количестве попыток.
        -   `break`: Завершает цикл (игру), если число угадано.
    -  **Подсказки**:
        - `elif userGuess < targetNumber:`: Проверяет, если введенное число меньше загаданного.
        - `print("СЛИШКОМ МАЛО")`: Выводит подсказку, что нужно ввести число больше.
        - `else:`: Если число не угадано и не меньше загаданного, то оно больше.
        - `print("СЛИШКОМ МНОГО")`: Выводит подсказку, что нужно ввести число меньше.
"""
```

## הסבר קוד

1.  **ייבוא מודול `random`**:
    -   `import random`: מייבא את המודול `random`, אשר משמש ליצירת מספר אקראי.
2.  **אתחול משתנים**:
    -   `numberOfGuesses = 0`: מאתחל את המשתנה `numberOfGuesses` לספירת ניסיונות השחקן.
    -   `targetNumber = random.randint(1, 100)`: מייצר מספר שלם אקראי בטווח שבין 1 ל-100 ושומר אותו ב-`targetNumber`.
3.  **לולאה ראשית `while True:`**:
    -   לולאה אינסופית שפועלת עד שהשחקן מנחש את המספר (הפקודה `break` מתבצעת).
    -   `numberOfGuesses += 1`: מגדיל את מונה הניסיונות ב-1 בכל מעבר חדש בלולאה.
    -   **קלט נתונים**:
        -   `try...except ValueError`: בלוק `try-except` מטפל בשגיאות קלט אפשריות. אם המשתמש מזין ערך שאינו מספר שלם, מוצגת הודעת שגיאה.
        -   `userGuess = int(input("Введите число: "))`: מבקש מהמשתמש להזין מספר וממיר אותו למספר שלם, תוך שמירת התוצאה ב-`userGuess`.
    -   **תנאי ניצחון**:
        -   `if userGuess == targetNumber:`: בודק אם המספר שהוזן שווה למספר המסתורי.
        -   `print("УГАДАНО")`: מדפיס את ההודעה "УГАДАНО".
        -   `print(f"ПОПЫТОК {numberOfGuesses}")`: מדפיס הודעה עם מספר הניסיונות.
        -   `break`: מסיים את הלולאה (המשחק) אם המספר נוחש.
    -   **רמזים**:
        - `elif userGuess < targetNumber:`: בודק אם המספר שהוזן קטן מהמספר המסתורי.
        - `print("СЛИШКОМ МАЛО")`: מדפיס רמז שצריך להזין מספר גדול יותר.
        - `else:`: אם המספר לא נוחש וגם לא קטן מהמספר המסתורי, אז הוא גדול ממנו.
        - `print("СЛИШКОМ МНОГО")`: מדפיס רמז שצריך להזין מספר קטן יותר.
