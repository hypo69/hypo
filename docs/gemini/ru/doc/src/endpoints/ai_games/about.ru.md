# 10 простых игр для обучения моделей машинного обучения

## Обзор

Данный документ описывает 10 простых игр, которые можно использовать для обучения моделей машинного обучения.  Игры разделяются по типам, учитывая сложность и типы моделей, которые они могут быть использованы.
