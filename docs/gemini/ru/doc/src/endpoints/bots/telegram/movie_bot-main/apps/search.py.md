# Модуль для поиска фильмов и сериалов на Kinopoisk

## Обзор

Этот модуль предоставляет функцию `search_query`, которая выполняет поиск фильмов или сериалов на сайте Kinopoisk через поисковую систему Google. Он использует библиотеки `requests` для выполнения HTTP-запросов, `BeautifulSoup` для парсинга HTML-ответов и `dotenv` для загрузки переменных окружения.

## Подробней

Модуль предназначен для поиска информации о фильмах и сериалах на сайте Kinopoisk. Функция `search_query` формирует поисковый запрос для Google, анализирует результаты поиска и возвращает информацию о первом найденном фильме или сериале, если таковой имеется. Этот модуль облегчает автоматический поиск нужной информации о фильмах и сериалах, предоставляя удобный интерфейс для получения ссылок, названий и описаний.

## Функции

### `search_query`

```python
def search_query(query: str, type_movie: str = 'series') -> dict | None:
    """
    Args:
        query (str): Поисковый запрос.
        type_movie (str, optional): Тип искомого контента ('series' или другой). По умолчанию 'series'.

    Returns:
        dict | None: Словарь с информацией о фильме/сериале, содержащий ключи 'link', 'title' и 'description',
                     или None, если ничего не найдено.

    Raises:
        Нет явных исключений.
    """
```

**Описание**: Выполняет поиск фильма или сериала на Kinopoisk через поисковую систему Google.

**Параметры**:
- `query` (str): Поисковый запрос.
- `type_movie` (str, optional): Тип искомого контента ('series' или другой). По умолчанию 'series'.

**Возвращает**:
- `dict | None`: Словарь с информацией о фильме/сериале, содержащий ключи 'link', 'title' и 'description', или `None`, если ничего не найдено.

**Примеры**:
```python
search_query('теория большого взрыва')
```
