# Модуль `header.py`

## Обзор

Модуль `header.py` содержит общие настройки и метаданные проекта, а также функцию для определения корневой директории проекта.

## Функции

### `set_project_root`

**Описание**:
Находит корневую директорию проекта, начиная с директории текущего файла, ища вверх до первого каталога, содержащего один из маркеров.

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или директорий для идентификации корня проекта. По умолчанию `('__root__','.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена, иначе директория, где находится скрипт.

## Глобальные переменные

### `__root__`

**Описание**:
Путь к корневой директории проекта.

### `settings`

**Описание**:
Словарь, содержащий настройки проекта, загруженные из `settings.json`.

### `doc_str`

**Описание**:
Строка, содержащая содержимое файла `README.MD` проекта.

### `__project_name__`

**Описание**:
Имя проекта, полученное из настроек или по умолчанию `hypotez`.

### `__version__`

**Описание**:
Версия проекта, полученная из настроек.

### `__doc__`

**Описание**:
Документация проекта, полученная из `README.MD`.

### `__details__`

**Описание**:
Детали проекта (в настоящее время пустая строка).

### `__author__`

**Описание**:
Автор проекта, полученный из настроек.

### `__copyright__`

**Описание**:
Авторские права проекта, полученные из настроек.

### `__cofee__`

**Описание**:
Сообщение для разработчиков с предложением угостить кофе, полученное из настроек или значение по умолчанию.
