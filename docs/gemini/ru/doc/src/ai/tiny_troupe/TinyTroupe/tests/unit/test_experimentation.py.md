# Документация модуля `test_experimentation.py`

## Обзор

Этот модуль содержит юнит-тесты для проверки функциональности класса `ABRandomizer`, который используется для A/B-тестирования. Тесты проверяют правильность рандомизации и дерандомизации вариантов, а также работу с именами вариантов.

## Функции

### `test_randomize`

**Описание**:
Тестирует метод `randomize` класса `ABRandomizer`. Проверяет, что при многократном вызове метода с разными индексами происходит корректная рандомизация вариантов.

**Параметры**:
- Нет параметров.

**Возвращает**:
- Нет возвращаемого значения.

**Вызывает исключения**:
- `Exception`: Если не найдена рандомизация для элемента с заданным индексом.

### `test_derandomize`

**Описание**:
Тестирует метод `derandomize` класса `ABRandomizer`. Проверяет, что дерандомизация возвращает исходные варианты после рандомизации.

**Параметры**:
- Нет параметров.

**Возвращает**:
- Нет возвращаемого значения.

**Вызывает исключения**:
- Нет.

### `test_derandomize_name`
