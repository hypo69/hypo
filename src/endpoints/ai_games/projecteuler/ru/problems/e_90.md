# Задача No 90
## Квадратные кубы
## Условия

Каждый из шести граней кубика пронумерован цифрой (от 0 до 9).

Используя два таких кубика, можно сформировать двузначные числа, просто расположив кубики рядом. Например, если на кубиках есть грани с цифрами {0, 1, 2, 3, 4, 5} и {1, 2, 3, 6, 7, 8}, можно сформировать число 28, используя второй кубик для десятков и первый для единиц.

Более того, изменяя расположение кубиков, можно сформировать 81, используя первый кубик для десятков и второй для единиц.

Известно, что в наборе из двух кубиков можно получить все девять двузначных чисел-квадратов: 01, 04, 09, 16, 25, 36, 49, 64 и 81.

Например, одно из решений - разместить цифры:
- {0, 5, 6, 7, 8, 9} на первом кубике и
- {1, 2, 3, 4, 8, 9} на втором кубике.

Из-за возможности поворота кубика, цифры 6 и 9 могут взаимозаменяться, и могут использоваться в любом положении.

Определите, сколько различных наборов из двух кубиков позволяют сформировать все девять двузначных чисел-квадратов.
