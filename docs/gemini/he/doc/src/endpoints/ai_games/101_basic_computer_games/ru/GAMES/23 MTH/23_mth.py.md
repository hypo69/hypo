# 23_mth.py

## סקירה כללית

קובץ זה מיישם את משחק ניחוש המספרים הקלאסי, בו המחשב בוחר מספר אקראי בין 1 ל-100, והמשתמש מנסה לנחש אותו, תוך שהוא מקבל רמזים "נמוך מדי" או "גבוה מדי" לאחר כל ניסיון. המשחק ממשיך עד שהמשתמש מנחש את המספר.
