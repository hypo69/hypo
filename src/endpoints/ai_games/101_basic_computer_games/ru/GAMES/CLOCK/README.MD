```python
"""
Digital Clock:
=================
Сложность: 2
-----------------
Игра "Digital Clock" - это простая программа, которая отображает текущее время в цифровом формате на экране. Она использует библиотеку Tkinter для создания графического интерфейса и модуль time для получения текущего времени.

Правила:
1. Программа отображает текущее время в формате ЧЧ:ММ:СС.
2. Время обновляется каждую секунду, создавая эффект цифровых часов.
3. Пользователь может закрыть окно, чтобы остановить программу.
-----------------
Алгоритм:
1. Импортировать необходимые библиотеки: tkinter для GUI и time для работы со временем.
2. Создать главное окно приложения Tkinter.
3. Задать заголовок окна.
4. Создать метку (label) для отображения времени с необходимыми стилями.
5. Разместить метку на окне.
6. Определить функцию для обновления времени:
    6.1. Получить текущее время в формате ЧЧ:ММ:СС.
    6.2. Обновить текст метки текущим временем.
    6.3. Запланировать следующее обновление метки через 1 секунду.
7. Вызвать функцию обновления времени один раз для начала работы часов.
8. Запустить главный цикл обработки событий Tkinter, чтобы окно оставалось открытым и реагировало на действия пользователя.
9. Конец программы.
-----------------

"""
import tkinter as tk
from time import strftime

def update_time():
    """
    Обновляет текст на метке времени, используя текущее время.
    """
    current_time = strftime("%H:%M:%S") # Получаем текущее время
    clock_label.config(text=current_time) # Обновляем текст метки
    clock_label.after(1000, update_time) # Планируем следующее обновление через 1 секунду

if __name__ == "__main__":
    root = tk.Tk() # Создаем главное окно
    root.title("Digital Clock Game") # Задаем заголовок окна

    clock_label = tk.Label(root, font=("Helvetica", 48), bg="black", fg="cyan") # Создаем метку для времени
    clock_label.pack(anchor="center", fill="both", expand=True) # Размещаем метку на окне

    update_time() # Запускаем обновление времени
    root.mainloop() # Запускаем главный цикл событий
```

**Разбор кода:**

1.  **Заголовок и описание:**
    *   В самом начале кода добавлен многострочный комментарий, который содержит:
        *   Название игры ("Digital Clock").
        *   Уровень сложности (2).
        *   Краткое описание игры, ее правила и алгоритм. Это помогает понять, что делает программа.

2.  **Импорт библиотек:**
    *   `import tkinter as tk` - импорт библиотеки Tkinter, используемой для создания GUI.
    *   `from time import strftime` - импорт функции strftime из модуля time, используемой для форматирования времени.

3.  **Функция `update_time()`**:
    *   Содержит описание назначения функции в docstring.
    *   `current_time = strftime("%H:%M:%S")`: Получает текущее время в формате "ЧЧ:ММ:СС".
    *   `clock_label.config(text=current_time)`: Обновляет текст на метке (label).
    *   `clock_label.after(1000, update_time)`: Планирует вызов этой же функции `update_time` через 1000 мс (1 секунду), что создает цикл обновления.

4.  **Условный запуск (`if __name__ == "__main__":`)**:
    *   Весь основной код помещен в блок `if __name__ == "__main__":`, что означает, что этот код будет выполнен только если файл запущен как самостоятельная программа.
    *   `root = tk.Tk()`: Создает главное окно приложения.
    *   `root.title("Digital Clock Game")`: Устанавливает заголовок окна.
    *   `clock_label = tk.Label(...)`: Создаёт метку для отображения времени, устанавливает шрифт, цвет фона и цвет текста.
    *   `clock_label.pack(...)`: Размещает метку на окне, позволяя ей расширяться и центрироваться.
    *   `update_time()`: Первый запуск обновления времени.
    *   `root.mainloop()`: Запускает главный цикл событий Tkinter для отображения и обновления окна.
