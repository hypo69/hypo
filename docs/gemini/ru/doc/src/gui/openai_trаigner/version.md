# Модуль hypotez/src/gui/openai_trаigner/version.py

## Обзор

Этот модуль содержит информацию о версии и метаданных для проекта. Он определяет константы `MODE`, `__version__`, `__doc__`, `__details__`, `__author__`, `__copyright__`, `__cofee__`.

## Переменные

### `MODE`

**Описание**: Строковая константа, хранящая режим работы (например, `dev`, `prod`).

**Значение**: `'dev'`

### `__version__`

**Описание**: Строковая константа, хранящая версию модуля.

**Значение**: `'v1.1'`

### `__doc__`

**Описание**: Строковая константа, хранящая строку документации модуля (пустая строка по умолчанию).

**Значение**: `''`

### `__details__`

**Описание**: Строковая константа, хранящая детали проекта (пустая строка по умолчанию).

**Значение**: `''`


### `__author__`

**Описание**: Строковая константа, содержащая имя автора.

**Значение**: `'hypo69'`

### `__copyright__`

**Описание**: Строковая константа, содержащая информацию о лицензии.


```markdown
**Лицензия:**

**Описание**:  Текст лицензии MIT.

**Ссылка**: [MIT License](https://opensource.org/licenses/MIT)


**Коммерческое использование запрещено** без предварительного разрешения авторов.

```

### `__cofee__`

**Описание**: Строковая константа, содержащая ссылку на возможность поддержать автора.

**Значение**: "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69"
