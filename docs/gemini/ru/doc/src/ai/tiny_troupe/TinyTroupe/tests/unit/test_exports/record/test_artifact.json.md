Данный файл содержит пример JSON-данных для тестирования. Он включает в себя информацию о пользователе, такую как имя, возраст, род занятий и произвольное текстовое содержимое.

## Структура JSON

###  `Структура данных`

**Описание**:

JSON-файл содержит следующие поля:
- `name` (str): Имя пользователя.
- `age` (int): Возраст пользователя.
- `occupation` (str): Профессия пользователя.
- `content` (str): Текстовое содержимое.

**Пример**:
```json
{
    "name": "John Doe",
    "age": 30,
    "occupation": "Engineer",
    "content": "This is a sample JSON data."
}
```
