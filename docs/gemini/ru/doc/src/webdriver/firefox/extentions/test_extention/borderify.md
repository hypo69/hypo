# Модуль borderify.js

## Обзор

Данный скрипт добавляет красную границу толщиной 5 пикселей к телу документа. Также выводит диалоговое окно с помощью `alert()`.

## Функции

### `document.body.style.border = "5px solid red";`

**Описание**: Устанавливает CSS-свойство `border` для элемента `document.body` на значение "5px solid red", добавляя красную границу толщиной 5 пикселей.


### `alert()`

**Описание**: Вызывает диалоговое окно `alert`, отображающее сообщение пользователю.


##  Обработка исключений


Этот скрипт не содержит блока обработки исключений (`try...catch`).
