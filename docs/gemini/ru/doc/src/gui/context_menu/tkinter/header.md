# Модуль hypotez/src/gui/context_menu/tkinter/header.py

## Обзор

Данный модуль `header.py` содержит константы и переменные, вероятно, определяющие режим работы и пути к ресурсам, используемые в модуле `context_menu.tkinter`.

## Переменные

### `MODE`

**Описание**: Переменная, определяющая режим работы приложения (`dev` в данном случае).

**Тип**: str

**Значение**: 'dev'
