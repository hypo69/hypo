# Модуль Glider

## Обзор

Модуль `Glider` предоставляет класс для взаимодействия с сервисом Glider.so, который предоставляет доступ к различным моделям машинного обучения, в основном семейства Llama и DeepSeek. Класс наследуется от `OpenaiTemplate` и содержит специфические настройки для работы с API Glider, такие как URL, endpoint и список поддерживаемых моделей.

## Подробней

Модуль предназначен для интеграции с проектом `hypotez` и предоставляет возможность использовать модели, предоставляемые Glider.so, через унифицированный интерфейс `OpenaiTemplate`. Это позволяет легко переключаться между различными провайдерами и моделями, не меняя основной код проекта.

## Классы

### `Glider`

**Описание**: Класс `Glider` наследуется от `OpenaiTemplate` и предназначен для взаимодействия с API сервиса Glider.so. Он определяет endpoint, список поддерживаемых моделей и их псевдонимы.

**Принцип работы**:
1. Класс наследуется от `OpenaiTemplate`, что обеспечивает совместимость с другими провайдерами и унифицированный интерфейс.
2. Определяются специфические для Glider.so параметры, такие как `label`, `url`, `api_endpoint`, `working`, `default_model` и `models`.
3. `model_aliases` предоставляет псевдонимы для моделей, что упрощает их использование и конфигурацию.

**Методы**:
- `__init__`: Конструктор класса, наследуется от `OpenaiTemplate`.

**Параметры**:
- `label` (str): Метка провайдера, "Glider".
- `url` (str): URL сервиса, "https://glider.so".
- `api_endpoint` (str): URL endpoint для API, "https://glider.so/api/chat".
- `working` (bool): Флаг, указывающий, работает ли провайдер, `True`.
- `default_model` (str): Модель по умолчанию, 'chat-llama-3-1-70b'.
- `models` (List[str]): Список поддерживаемых моделей.
- `model_aliases` (Dict[str, str]): Словарь псевдонимов моделей.

**Примеры**

```python
from src.endpoints.gpt4free.g4f.Provider.Glider import Glider

glider = Glider()
print(glider.label)  # Glider
print(glider.default_model) # chat-llama-3-1-70b
```

## Функции

В данном модуле нет отдельных функций, только класс `Glider` с его атрибутами. Однако, если `OpenaiTemplate` содержит функции, они будут унаследованы классом `Glider`.
