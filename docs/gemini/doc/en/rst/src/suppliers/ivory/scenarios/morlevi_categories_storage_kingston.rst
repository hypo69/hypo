Morlevi Categories Storage Kingston
=================================

This file contains a JSON representation of categories for Kingston products on the Morlevi website.  It details various SSD models (NVMe Gen4, SATA 3) and their corresponding information like brand, URLs, and PrestaShop category mappings.
