# Задача No 321
## Ступени
## Условия
Представьте себе лестницу, у которой каждая ступенька состоит из кубиков.
На рисунке ниже показана лестница из 5 ступенек:

```
      X
    XX
   XXX
  XXXX
 XXXXX
```

Такая лестница состоит из 15 кубиков.

Лестницу с *n* ступенями можно представить как треугольное число, чье значение равно t(*n*) = *n*(*n* + 1)/2.

Обозначим s(*n*) наименьшее количество кубиков, необходимых для создания лестницы с *n* ступенями, таким образом, что лестницу можно разделить на *k* полных квадратов, где *k* > 1.

Первые пять значений таковы:
s(1) = 1
s(2) = 3
s(3) = 6
s(4) = 10
s(5) = 15

Минимальное значение, для которого s(*n*) можно разделить на 2 квадрата это s(36) = 666, так как 666 = 2 * 18^2 = 6^2 + 24^2.

Найдите сумму всех s(*n*) <= 10^18, которые можно разделить на *k* полных квадратов, где *k* > 1.
