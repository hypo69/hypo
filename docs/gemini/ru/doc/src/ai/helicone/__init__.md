# Модуль `hypotez/src/ai/helicone/__init__.py`

## Обзор

Этот модуль предоставляет начальную настройку для работы с Helicone.  Он определяет константу `MODE` для выбора режима работы (например, 'dev', 'prod').


## Константы

### `MODE`

**Описание**:  Переменная, определяющая режим работы. Может принимать значения, такие как 'dev' или 'prod', влияя на конфигурацию или поведение последующих функций.

**Значение**: 'dev' по умолчанию.
