# Модуль `base64`

## Обзор

Модуль `base64` предоставляет функции для работы с Base64 кодированием. Он включает в себя функции для декодирования Base64 контента в временный файл и кодирования содержимого файла в Base64.

## Функции

### `base64_to_tmpfile`

**Описание**: Конвертирует Base64 закодированный контент во временный файл.

Эта функция декодирует Base64 закодированный контент и записывает его во временный файл с тем же расширением, что и предоставленное имя файла. Возвращается путь к временному файлу.

**Параметры**:
- `content` (str): Base64 закодированный контент, который нужно декодировать и записать в файл.
- `file_name` (str): Имя файла, используемое для извлечения расширения файла для временного файла.

**Возвращает**:
- `str`: Путь к временному файлу.

**Пример**:
```python
>>> base64_content = "SGVsbG8gd29ybGQh"  # Base64 encoded content "Hello world!"
>>> file_name = "example.txt"
>>> tmp_file_path = base64_to_tmpfile(base64_content, file_name)
>>> print(f"Temporary file created at: {tmp_file_path}")
Temporary file created at: /tmp/tmpfile.txt
```

### `base64encode`

**Описание**: Кодирует содержимое файла в Base64 строку.

Эта функция открывает файл, читает его содержимое, кодирует его в Base64 и возвращает закодированную строку.

- `image_path` (str): Путь к файлу, содержимое которого нужно закодировать в Base64.

- `str`: Base64 закодированная строка.
