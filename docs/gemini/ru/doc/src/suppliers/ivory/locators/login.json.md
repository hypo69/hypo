# Локаторы для страницы логина

## Обзор

Этот файл содержит JSON-объект, описывающий локаторы элементов для страницы логина. Локаторы используются для автоматического взаимодействия с элементами страницы в тестах.

## Структура JSON

### `login`

Объект, содержащий локаторы для различных элементов на странице логина.

#### `open_login_inputs`

**Описание**: Локатор для элемента, который открывает форму логина.

**Параметры**:
-   `attribute` (null): Атрибут элемента, значение которого нужно получить. В данном случае `null`.
-   `by` (string): Тип локатора, в данном случае `XPATH`.
-   `selector` (string): Значение локатора: `//span[@id='nav-link-accountList-nav-line-1']`.
-   `if_list` (string):  Указывает, как обрабатывать список элементов, в данном случае `first` - берется первый элемент списка.
-   `use_mouse` (boolean): Флаг, указывающий, использовать ли мышь для взаимодействия с элементом. В данном случае `false`.
-    `mandatory` (boolean): Флаг, указывающий, является ли наличие элемента обязательным. В данном случае `true`.
-    `timeout` (number): Время ожидания появления элемента, в данном случае `0`.
-    `timeout_for_event` (string): Событие, которое нужно ждать при таймауте. В данном случае `presence_of_element_located`.
-   `event` (string): Событие, которое нужно выполнить с элементом: `click()`.
-   `logic for action[AND|OR|XOR|VALUE|null]` (null): Логика для выполнения действия. В данном случае `null`.
