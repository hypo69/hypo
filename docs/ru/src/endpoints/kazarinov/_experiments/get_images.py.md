# Модуль для получения списка изображений

## Обзор

Модуль предназначен для получения списка изображений из указанной директории. Он использует функции для рекурсивного поиска файлов с заданными расширениями и вывода списка найденных файлов.

## Подробнее

Этот модуль является частью экспериментов Kazarinov и предназначен для работы с изображениями, сгенерированными ИИ. Он предоставляет простой способ получения списка файлов изображений, расположенных в определенной директории.

from src.utils.file import recursively_get_filepath
def recursively_get_filepath(folder: str | Path, patterns: List[str]) -> List[Path]:
    """Рекурсивно получает список файлов, соответствующих заданным шаблонам, в указанной папке.

    Args:
        folder (str | Path): Путь к папке, в которой производится поиск файлов.
        patterns (List[str]): Список шаблонов файлов для поиска (например, ['*.txt', '*.pdf']).

    Returns:
        List[Path]: Список путей к найденным файлам.

    Raises:
        FileNotFoundError: Если указанная папка не существует.

    Как работает функция:
     1.  Функция принимает путь к папке и список шаблонов файлов для поиска.
     2.  Она рекурсивно проходит по всем подпапкам указанной папки.
     3.  Для каждой подпапки функция проверяет, соответствуют ли файлы в ней заданным шаблонам.
     4.  Если файлы соответствуют шаблонам, их пути добавляются в список результатов.
     5.  В конце функция возвращает список путей ко всем найденным файлам.

    Внутри функции происходят следующие действия и преобразования:
    A. Проверка существования папки: Функция проверяет, существует ли указанная папка. Если папка не существует, вызывается исключение FileNotFoundError.
    |
    B. Рекурсивный поиск файлов: Функция рекурсивно проходит по всем подпапкам указанной папки и ищет файлы, соответствующие заданным шаблонам.

    D. Возврат списка результатов: В конце функция возвращает список путей ко всем найденным файлам.

## Переменные

### `images_path`

```python
images_path = recursively_get_filepath(gs.path.external_data / 'kazarinov' / 'converted_images' / 'pastel', ['*.jpeg','*.jpg','*.png'])
```

Содержит список путей к файлам изображений с расширениями `.jpeg`, `.jpg`, `.png`, расположенных в директории `gs.path.external_data / 'kazarinov' / 'converted_images' / 'pastel'`.  `gs.path.external_data`  - это корневая директория, где хранятся внешние данные. Путь указывает на подкаталог  `kazarinov/converted_images/pastel`, где, вероятно, хранятся обработанные или преобразованные изображения.
