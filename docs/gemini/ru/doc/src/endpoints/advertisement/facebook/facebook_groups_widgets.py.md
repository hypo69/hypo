## Обзор

Модуль `facebook_groups_widgets.py` предоставляет класс `FacebookGroupsWidget`, предназначенный для создания интерактивного выпадающего списка с URL-адресами групп Facebook. Данные для списка загружаются из JSON-файла. Модуль использует библиотеки `ipywidgets` для создания виджета и `IPython.display` для отображения.

## Классы

**Описание**: Класс для создания виджета выпадающего списка с URL-адресами групп Facebook.
