# Tiny Utils

## Обзор

Tiny Utils — это библиотека утилит, предоставляющая набор легких функций для различных общих задач. Эта библиотека включает утилиты для преобразования форматов данных, работы с текстом и файлами, строковыми операциями, форматированием дат и времени, обработкой изображений и т. д. Она организована в несколько модулей для удобного доступа к определенным функциям.

## Установка

Для использования Tiny Utils необходимо клонировать репозиторий и установить необходимые зависимости, указанные в файле `requirements.txt`.

```bash
git clone https://github.com/hypo69/tiny-utils.git
cd tiny_utils
pip install -r requirements.txt
```

## Обзор модулей

Библиотека содержит несколько подмодулей, каждый из которых выполняет определенную задачу:

- **Преобразователи**: Модули для преобразования форматов данных, таких как текст в изображение, webp в png, JSON, XML, кодирование Base64 и т. д.
- **Утилиты для строк**: Инструменты для расширенной обработки строк.
- **Утилиты для файлов**: Функции для работы с файлами и их манипулирования.
- **Утилиты для дат и времени**: Инструменты для форматирования дат и времени.
- **Утилиты для FTP**: Функции обработки файлов FTP.
- **Утилиты для изображений**: Базовые функции обработки изображений.
- **Утилиты для PDF**: Утилиты для работы с файлами PDF, включая преобразование, объединение, разделение и извлечение текста.
- **Утилиты для принтера**: Функции для отправки данных на принтер.

## Описание модулей

### Преобразователи

Модуль `convertors` содержит утилиты для преобразования данных между форматами. Эти модули могут обрабатывать различные типы данных, от CSV до JSON и текста до изображений.

#### Файлы:

- `text2png.py`: Преобразует текстовые данные в изображение PNG.
- `tts.py`: Преобразует текст в речь и сохраняет его как аудиофайл.
- `webp2png.py`: Преобразует изображения из формата WebP в формат PNG.
- `xls.py`: Обрабатывает преобразования и манипуляции с файлами XLS.
- `xml2dict.py`: Преобразует данные XML в словарь Python.
- `base64.py`: Кодирует или декодирует данные с использованием кодирования Base64.
- `csv.py`: Предоставляет инструменты для парсинга и манипулирования CSV.
- `dict.py`: Утилиты для работы со словарями Python.
- `html.py`: Преобразует HTML-контент в различные форматы.
- `json.py`: Утилиты для парсинга и манипулирования JSON.
- `md2dict.py`: Преобразует Markdown-контент в словарь.
- `ns.py`: Специализированные утилиты преобразования именных пространств.
