# `header.py`

## Обзор

Модуль `header.py` определяет корневой путь к проекту и загружает основные настройки проекта из `settings.json`, а также описание из `README.md`. Все импорты строятся относительно этого пути.

## Функции

### `set_project_root`

**Описание**:
Находит корневую директорию проекта, начиная с директории текущего файла, и поднимаясь вверх до первого каталога, содержащего один из маркерных файлов.

**Параметры**:
- `marker_files` (tuple, optional): Кортеж имен файлов или директорий для определения корня проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена, иначе директория, где расположен скрипт.

## Переменные

### `__root__`

**Описание**:
Путь к корневой директории проекта, определенный функцией `set_project_root`.

### `settings`

**Описание**:
Словарь с настройками проекта, загруженный из файла `settings.json`.

### `doc_str`

**Описание**:
Строка с описанием проекта, загруженная из файла `README.MD`.

### `__project_name__`

**Описание**:
Имя проекта, полученное из `settings.json` или `hypotez` по умолчанию.

### `__version__`

**Описание**:
Версия проекта, полученная из `settings.json` или пустая строка по умолчанию.

### `__doc__`

**Описание**:
Описание проекта, загруженное из `README.md` или пустая строка по умолчанию.

### `__details__`

**Описание**:
Детали проекта. В настоящее время пустая строка.

### `__author__`

**Описание**:
Автор проекта, полученный из `settings.json` или пустая строка по умолчанию.

### `__copyright__`

**Описание**:
Информация об авторских правах, полученная из `settings.json` или пустая строка по умолчанию.

### `__cofee__`

**Описание**:
Сообщение с предложением угостить разработчика кофе, полученное из `settings.json` или строка по умолчанию со ссылкой на boosty.to.
