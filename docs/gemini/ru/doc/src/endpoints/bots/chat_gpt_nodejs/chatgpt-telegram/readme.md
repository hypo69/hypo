# Голосовой помощник chatgpt - telegram

## Обзор

Данный документ описывает реализацию голосового помощника на базе ChatGPT, интегрированного с Telegram. Проект использует Node.js для взаимодействия с API ChatGPT и Telegram Bot API. 
Документ ссылается на статью на Хабре: [https://habr.com/ru/companies/selectel/articles/731692/](https://habr.com/ru/companies/selectel/articles/731692/), которая, вероятно, описывает подробности реализации данного помощника.

## Технологии

Проект реализован на следующих технологиях:

- **Node.js:** Основная среда выполнения для серверной части.
- **ChatGPT API:** Используется для получения ответов от модели ChatGPT.
- **Telegram Bot API:** Используется для интеграции с Telegram и обработки сообщений от пользователей.

## Установка и настройка

1.  **Клонирование репозитория:**
    ```bash
    git clone <ссылка_на_репозиторий>
    cd <название_репозитория>
    ```

2.  **Установка зависимостей:**
    ```bash
    npm install
    ```

3.  **Настройка переменных окружения:**
    - Создайте файл `.env` в корневой директории проекта.
    - Добавьте следующие переменные окружения, заменив значения на свои:
      ```env
      TELEGRAM_BOT_TOKEN=ваш_токен_бота_telegram
      OPENAI_API_KEY=ваш_api_ключ_openai
      ```
    -  `TELEGRAM_BOT_TOKEN` - токен вашего Telegram-бота, который можно получить через @BotFather.
    -  `OPENAI_API_KEY` - ваш API-ключ OpenAI.

4.  **Запуск проекта:**
    ```bash
    npm start
    ```
## Использование

После запуска проекта бот будет доступен в Telegram.

- **Отправка голосовых сообщений:** Бот преобразует голосовое сообщение в текст и передаст его в ChatGPT, после чего ответит текстовым сообщением.
- **Отправка текстовых сообщений:** Бот передаст текстовое сообщение в ChatGPT и ответит текстовым сообщением.
