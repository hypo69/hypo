## Переменные

### `MODE`

**Описание**: Константа, определяющая режим работы приложения (например, `dev`, `prod`).

**Значение**: `'dev'`


### `__root__`

**Описание**: Переменная, содержащая абсолютный путь к корневому каталогу проекта.

**Тип**: `Path`

**Описание**: Задает абсолютный путь к корневому каталогу, добавляя его в переменную окружения `sys.path`.
