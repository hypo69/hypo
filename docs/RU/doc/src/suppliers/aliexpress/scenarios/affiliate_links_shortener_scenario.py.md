## Функции

### `get_short_affiliate_link`

**Описание**: Скрипт для генерации сокращенной партнерской ссылки.

**Параметры**:
- `d` (`Driver`): Экземпляр веб-драйвера.
- `url` (`str`): Полный URL.

**Возвращает**:
- `str`: Сокращенный URL.

**Вызывает исключения**:
- `ValueError`: Если не удалось получить короткий URL или короткий URL некорректен.
