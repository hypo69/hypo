# Модуль gworksheets

## Обзор

Модуль `gworksheets` предоставляет классы и функции для работы с электронными таблицами Google. Он позволяет создавать, получать доступ и манипулировать листами (worksheets) внутри книги (spreadsheet).  Модуль использует библиотеки `global_settingspread`, `goog.grender` и др. для взаимодействия с Google Spreadsheets API.

## Классы

### `GWorksheet`

**Описание**: Класс `GWorksheet` наследуется от класса `Worksheet` и расширяет его функциональность, предоставляя методы для работы с отдельными листами в электронных таблицах Google.

**Методы**:

#### `__init__`

**Описание**: Конструктор класса `GWorksheet`.

**Параметры**:
- `self`: Экземпляр класса `GWorksheet`.
- `sh`: Экземпляр класса `Spreadsheet` представляющий книгу (spreadsheet).
- `ws_title`: (строка, по умолчанию 'new') Заголовок листа.
- `rows`: (целое число, по умолчанию None) Количество строк в листе.
- `cols`: (целое число, по умолчанию None) Количество столбцов в листе.
- `direction`: (строка, по умолчанию 'rtl') Направление отображения текста в листе ('rtl' или 'ltr').
- `wipe_if_exist`: (булево значение, по умолчанию True) Флаг, определяющий, нужно ли очищать лист при его существовании.
- `*args`:  Должны быть переданы аргументы.
- `**kwards`:  Должны быть переданы ключевые аргументы.


**Возвращает**:
- `None`

#### `get`

**Описание**: Создает новый лист или получает существующий лист в электронных таблицах.

**Параметры**:
- `self`: Экземпляр класса `GWorksheet`.
- `sh`: Экземпляр класса `Spreadsheet` представляющий книгу (spreadsheet).
- `ws_title`: (строка, по умолчанию 'new') Заголовок листа.
- `rows`: (целое число, по умолчанию 100) Количество строк.
- `cols`: (целое число, по умолчанию 100) Количество столбцов.
- `direction`: (строка, по умолчанию 'rtl') Направление отображения текста.
- `wipe_if_exist`: (булево значение, по умолчанию True) Очистить лист при его существовании.


**Возвращает**:
- `None`

#### `header`

**Описание**: Добавляет заголовок на лист.

**Параметры**:
- `self`: Экземпляр класса `GWorksheet`.
- `world_title`: Заголовок для добавления.
- `range`: (строка, по умолчанию 'A1:Z1') Диапазон ячеек для заголовка.
- `merge_type`: (строка, по умолчанию 'MERGE_ALL') Тип объединения ячеек (MERGE_ALL, MERGE_COLUMNS, MERGE_ROWS).

**Возвращает**:
- `None`


#### `category`

**Описание**:  Пишет заголовок категории в листе.

**Параметры**:
- `self`: Экземпляр класса `GWorksheet`.
- `ws_category_title`: Заголовок категории.

**Возвращает**:
- `None`

#### `direction`

**Описание**: Устанавливает направление текста в листе.

**Параметры**:
- `self`: Экземпляр класса `GWorksheet`.
- `direction`: (строка, по умолчанию 'rtl') Направление ('rtl' или 'ltr').

**Возвращает**:
- `None`
