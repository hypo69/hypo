# Задача No 422
## Последовательность последовательностей
## Условия
Пусть последовательность *S<sub>n</sub>* определяется следующим образом:
  * *S<sub>1</sub>* = "1"
  * *S<sub>n</sub>* получается, читая цифры в *S<sub>n-1</sub>* слева направо и описывая подряд идущие группы одинаковых цифр:
    - "1" записывается как "11" (одна единица)
    - "11" записывается как "21" (две единицы)
    - "21" записывается как "1211" (одна двойка, одна единица)
    - "1211" записывается как "111221" (одна единица, одна двойка, две единицы)

Первые десять членов последовательности таковы:

* S<sub>1</sub> = "1"
* S<sub>2</sub> = "11"
* S<sub>3</sub> = "21"
* S<sub>4</sub> = "1211"
* S<sub>5</sub> = "111221"
* S<sub>6</sub> = "312211"
* S<sub>7</sub> = "13112221"
* S<sub>8</sub> = "1113213211"
* S<sub>9</sub> = "31131211131221"
* S<sub>10</sub> = "132113213221133112132123"

Давайте определим функцию *f(n)* как количество цифр в *S<sub>n</sub>*. Например, *f(1) = 1, f(2) = 2, f(3) = 2, f(5) = 6*.

Также, пусть определена функция *g(n)* как *f(f(n))*. Например, *g(1) = f(f(1)) = f(1) = 1, g(2) = f(f(2)) = f(2) = 2, g(3) = f(f(3)) = f(2) = 2*.

Найдите значение Σ *g(n)* для 1 ≤ *n* ≤ 10<sup>7</sup>.
