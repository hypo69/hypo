# Локаторы для страницы входа

## Обзор

Этот файл содержит JSON-объект с локаторами для элементов страницы входа, используемыми для автоматизации процесса входа в систему. Локаторы организованы в логическую структуру, где каждый элемент имеет набор атрибутов, необходимых для взаимодействия с ним.

## Структура JSON

### `login`

Объект, содержащий локаторы для элементов страницы входа.

#### `open_login_inputs`

**Описание**: Локатор для кнопки или элемента, который открывает форму для ввода данных для входа.

**Атрибуты**:
- `attribute`: `null`. Атрибут элемента.
- `by`: `"XPATH"`. Тип локатора.
- `selector`: `"//span[@id='nav-link-accountList-nav-line-1']"`. XPath-селектор элемента.
- `if_list`: `"first"`. Указывает, что если найдено несколько элементов, то нужно выбрать первый.
- `use_mouse`: `false`. Указывает, нужно ли использовать мышь для взаимодействия.
- `mandatory`: `true`. Указывает, что элемент обязателен.
- `timeout`: `0`. Время ожидания элемента.
- `timeout_for_event`: `"presence_of_element_located"`. Тип ожидания элемента.
- `event`: `"click()"`. Событие, которое нужно вызвать при взаимодействии с элементом.
- `logic for action[AND|OR|XOR|VALUE|null]`: `null`. Логика действия.
