# Модуль `src.suppliers.aliexpress.utils`

## Обзор

Модуль `src.suppliers.aliexpress.utils` содержит набор утилит, используемых для работы с данными AliExpress. Он включает в себя функции для извлечения идентификаторов продуктов, обеспечения использования HTTPS, а также содержит словарь локалей.

## Переменные

### `MODE`

**Описание**: Режим работы приложения. По умолчанию установлено значение `dev`.
- **Тип**: `str`
- **Значение по умолчанию**: `'dev'`
