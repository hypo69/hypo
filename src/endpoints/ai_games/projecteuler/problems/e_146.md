# Задача No 146
## Исследование простых чисел с определенными разностями
## Условия

Простые числа 47, 53, 67 и 79 обладают замечательным свойством: каждое из чисел, полученных путем прибавления к ним 2, 4, 6, 8, 10, 12, а именно 49, 51, 53, 55, 57, 59, 61, 63, 65, 67, 69, 71, 73, 75, 77, 79, 81, 83, 85, 87, являются либо простыми, либо составными.

Пусть *n* – целое положительное число. Пусть *n*² + 1, *n*² + 3, *n*² + 7, *n*² + 9, *n*² + 13 и *n*² + 27 — последовательность, в которой все числа являются простыми.

Например, при n = 10, 10² + 1 = 101, 10² + 3 = 103, 10² + 7 = 107, 10² + 9 = 109, 10² + 13 = 113, и 10² + 27 = 127.
Однако есть числа n, при которых не все элементы последовательности являются простыми.
Например, при n = 11, 11² + 1 = 122, что не является простым числом.

Найдите сумму всех таких целых чисел n в интервале 0 < n < 150 000 000, для которых все шесть элементов последовательности являются простыми.
