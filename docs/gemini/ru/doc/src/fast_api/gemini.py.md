# Модуль `gemini.py`

## Обзор

Модуль `gemini.py` представляет собой веб-сервис на основе Flask, который предоставляет API для взаимодействия с моделью Google Gemini через класс `GoogleGenerativeAI`. Сервис принимает POST-запросы с текстовыми промптами и возвращает ответы от модели.

## Функции

### `ask`

**Описание**:
Обрабатывает POST-запросы, извлекает промпт из JSON-данных и передает его в модель Google Gemini для получения ответа.

**Параметры**:
- Нет явных параметров, данные извлекаются из тела POST-запроса.

**Возвращает**:
- `dict`: JSON-ответ, содержащий либо сгенерированный моделью текст в поле `reply`, либо сообщение об ошибке в поле `error`.
- `int`: HTTP-код ответа. 200 при успехе, 400 при отсутствии промпта, 500 при внутренней ошибке сервера.

**Вызывает исключения**:
- `Exception`: Любые исключения, которые могут возникнуть при обращении к модели Google Gemini, перехватываются и возвращаются клиенту в виде JSON-ответа.
