# Модуль hypotez/src/endpoints/kazarinov/scenarios/header.py

## Обзор

Этот модуль содержит функции для определения корневого каталога проекта и загрузки настроек из файла `settings.json`.  Он также обрабатывает возможные ошибки при чтении файла настроек.

## Функции

### `set_project_root`

**Описание**:  Находит корневой каталог проекта, начиная с текущего каталога и поднимаясь вверх по дереву каталогов, пока не найдет каталог, содержащий файлы-маркеры.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или каталогов, которые используются для идентификации корневого каталога проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневому каталогу проекта, если найден, иначе возвращает каталог, в котором расположен текущий скрипт.

**Вызывает исключения**:

- Нет

## Переменные

### `__root__`

**Описание**:  Путь к корневому каталогу проекта.  Получается при вызове функции `set_project_root()`.

**Тип**: `Path`

### `settings`

**Описание**: Словарь настроек, загруженный из файла `settings.json`.

**Тип**: `dict` или `None`

### `doc_str`

**Описание**: Текст из файла `README.MD`.

**Тип**: `str` или `None`


### `__project_name__`

**Описание**: Имя проекта, полученное из `settings.json` или заданное по умолчанию "hypotez".

**Тип**: `str`

### `__version__`

**Описание**: Версия проекта, полученное из `settings.json` или заданное по умолчанию пустой строкой.

**Тип**: `str`


### `__doc__`

**Описание**:  Документация проекта, полученная из `README.MD`, или пустая строка, если файл не найден.

**Тип**: `str`


### `__details__`

**Описание**:  Дополнительная информация о проекте, по умолчанию пустая строка.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта, полученный из `settings.json` или заданный по умолчанию пустой строкой.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права на проект, полученный из `settings.json` или заданный по умолчанию пустой строкой.

**Тип**: `str`

### `__cofee__`

**Описание**: Ссылка на поддержку разработчика. Получен из `settings.json` или задан по умолчанию.

**Тип**: `str`
