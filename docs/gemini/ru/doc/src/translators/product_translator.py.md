# Модуль `product_translator`

## Обзор

Модуль `product_translator` предназначен для управления переводами товаров. Он обеспечивает связь между словарем полей товара, таблицей переводов и переводчиками. Модуль позволяет получать переводы из базы данных PrestaShop, а также выполнять перевод полей товара с использованием AI.

## Функции

### `get_translations_from_presta_translations_table`

**Описание**: Функция возвращает список переводов полей товара из таблицы переводов.

**Параметры**:
- `product_reference` (str): Артикул товара, для которого требуется получить переводы.
- `i18n` (str, optional): Локаль перевода (например, 'en_US', 'ru-RU'). По умолчанию `None`.

**Возвращает**:
- `list`: Список словарей с переводами для указанного артикула товара.

### `insert_new_translation_to_presta_translations_table`

**Описание**: Функция добавляет новую запись перевода в таблицу переводов.

**Параметры**:
- `record` (dict): Словарь, содержащий данные для новой записи перевода.

**Возвращает**:
- `None`: Функция ничего не возвращает.

### `translate_record`

**Описание**: Функция для перевода полей товара.

**Параметры**:
- `record` (dict): Словарь, содержащий поля товара для перевода.
- `from_locale` (str): Локаль исходного текста.
- `to_locale` (str): Локаль, на которую требуется перевести текст.

**Возвращает**:
- `dict`: Словарь с переведенными полями товара.
