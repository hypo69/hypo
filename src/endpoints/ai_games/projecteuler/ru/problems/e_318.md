# Задача No 318
## Длиннейшие целые части

## Условия
Рассмотрим последовательность чисел s<sub>n</sub> = √(2) + √(3)<sup>n</sup>.

Пусть а<sub>n</sub> = ⌊ s<sub>n</sub> ⌋.
Пусть b<sub>n</sub> - наименьшее число, для которого выполняется условие a<sub>n</sub> = a<sub>n+b<sub>n</sub></sub>.

Первые несколько значений такие:

s<sub>1</sub> = 3.14626436994197234236...   a<sub>1</sub> = 3
s<sub>2</sub> = 9.89897948556635604252...   a<sub>2</sub> = 9
s<sub>3</sub> = 31.42696805867869709904...   a<sub>3</sub> = 31
s<sub>4</sub> = 99.89974949102826781797...   a<sub>4</sub> = 99
s<sub>5</sub> = 315.75925274417740161718...   a<sub>5</sub> = 315

Соответственно,

b<sub>1</sub> = 2, так как a<sub>1</sub> = 3, a<sub>2</sub> = 9, a<sub>3</sub> = 31 и a<sub>4</sub> = 99.
b<sub>2</sub> = 1, так как a<sub>2</sub> = 9, a<sub>3</sub> = 31 и a<sub>4</sub> = 99.
b<sub>3</sub> = 2, так как a<sub>3</sub> = 31, a<sub>4</sub> = 99, a<sub>5</sub> = 315 и a<sub>6</sub> = 999.
b<sub>4</sub> = 1, так как a<sub>4</sub> = 99, a<sub>5</sub> = 315 и a<sub>6</sub> = 999.
b<sub>5</sub> = 2, так как a<sub>5</sub> = 315, a<sub>6</sub> = 999, a<sub>7</sub> = 3161 и a<sub>8</sub> = 9999.

Найдите ∑b<sub>n</sub> для 1 ≤ n ≤ 2000.
