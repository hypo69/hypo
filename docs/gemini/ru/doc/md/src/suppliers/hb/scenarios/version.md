# Модуль hypotez/src/suppliers/hb/scenarios/version.py

## Обзор

Этот модуль содержит информацию о версии, а также другие метаданные.

## Переменные

### `MODE`

**Описание**:  Переменная, содержащая строковое значение `'dev'`.

**Тип**: `str`

### `__version__`

**Описание**:  Переменная, содержащая строковое представление версии модуля.

**Тип**: `str`

### `__name__`

**Описание**: Переменная, содержащая имя модуля. Если скрипт запускается напрямую, значение будет `"__main__"`.

**Тип**: `str`

### `__doc__`

**Описание**: Строка документации модуля.

**Тип**: `str`

### `__details__`

**Описание**: Переменная, содержащая дополнительные детали о модуле.

**Тип**: `str`

### `__annotations__`

**Описание**: Содержит аннотации типов для переменных и функций в модуле.

**Тип**: Любой


### `__author__`

**Описание**:  Имя (или имена) автора(ов) модуля.

**Тип**: `str`
