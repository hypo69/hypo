# Модуль для подготовки JSON-файла кампании AliExpress

## Обзор

Данный модуль предназначен для подготовки JSON-файлов, необходимых для создания и редактирования рекламных кампаний на AliExpress. Он включает в себя функции для обработки категорий кампаний, отдельных кампаний и всех кампаний в целом.

## Подробней

Модуль является частью пакета `src.suppliers.aliexpress.campaign._experiments` и предоставляет инструменты для автоматизации процесса создания и управления рекламными кампаниями на платформе AliExpress. Он использует другие модули проекта, такие как `AliCampaignEditor`, `gs`, `process_campaign_category`, `process_campaign` и `process_all_campaigns`, а также утилиты для работы с файлами и директориями.
