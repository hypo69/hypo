# Модуль `hypotez/src/ai`

## Обзор

Данный модуль предоставляет интерфейс для работы с моделями искусственного интеллекта, включая Google Gemini и OpenAI. Он определяет константу `MODE` для указания режима работы (в данном случае 'dev') и импортирует классы для взаимодействия с конкретными моделями.

## Константы

### `MODE`

**Описание**: Строковая константа, определяющая режим работы модуля.

**Значение**: `'dev'`

## Модели

### `GoogleGenerativeAI`

**Описание**: Класс, представляющий взаимодействие с моделью Google Gemini.

**Описание импорта**: Импортируется из подмодуля `gemini`.

### `OpenAIModel`

**Описание**: Класс, представляющий взаимодействие с моделью OpenAI.

**Описание импорта**: Импортируется из подмодуля `openai`.
