# Модуль revai

## Обзор

Модуль `revai` предназначен для взаимодействия с API сервиса Rev.com, который предоставляет услуги по транскрибации аудио и видео. Он включает в себя функциональность для работы со звуковыми файлами, такими как переговоры, совещания и звонки.
