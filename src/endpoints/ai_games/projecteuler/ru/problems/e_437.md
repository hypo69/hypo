# Задача No 437
## Гиперболическая группа чисел Фибоначчи

## Условия

Обозначим `F(n)` n-е число Фибоначчи (где `F(0) = 0`, `F(1) = 1`, `F(n) = F(n-1) + F(n-2)`).
Для целых чисел `a` и `b`, определим `G(a,b)` как последовательность, где `G(a,b,0) = a`, `G(a,b,1) = b` и `G(a,b,n) = G(a,b,n-1) + G(a,b,n-2)` для `n >= 2`.

Например:
- `G(1,2,n)` дает числа Фибоначчи, начинающиеся с 1 и 2.
- `G(3,1,n)` дает последовательность 3, 1, 4, 5, 9, 14, 23, ...
- `G(1,1,n)` дает стандартную последовательность Фибоначчи.

Определим `S(n)` как множество простых чисел `p` таких, что для некоторого целого числа `k` `G(1,1,k) mod p = 0` (то есть `p` делит некоторое число Фибоначчи).

Назовем простое число `p` членом гиперболической группы Фибоначчи если для некоторых целых чисел `a` и `b`, `G(a,b,k) mod p = 0` для некоторого `k`, где `a`, `b` могут принимать любые целочисленные значения.

Например, 2, 3, 5, 7 и 11 являются членами гиперболической группы Фибоначчи.
29 является первым простым числом, не являющимся членом гиперболической группы Фибоначчи.

Пусть `H(n)` — n-е простое число, не являющееся членом гиперболической группы Фибоначчи.
Известно, что `H(1)=29`, `H(10) = 1861` и `H(100) = 38207`.

Найдите `H(100000)`.
