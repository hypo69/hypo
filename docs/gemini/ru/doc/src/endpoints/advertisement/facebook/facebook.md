# Модуль facebook

## Обзор

Данный модуль предоставляет инструменты для взаимодействия с платформой Facebook, включая логин, отправку сообщений и загрузку медиа-контента.

## Классы

### `Facebook`

**Описание**: Класс `Facebook` предоставляет методы для работы с Facebook через веб-драйвер. Он позволяет выполнять различные действия, такие как вход в систему, продвижение постов и загрузка медиа-файлов.

**Атрибуты**:

- `d`: (Driver): Веб-драйвер для взаимодействия с Facebook. (Строковая аннотация типа для отложенного импорта).
- `start_page`: (str): Стартовая страница Facebook. По умолчанию `https://www.facebook.com/hypotez.promocodes`.
- `promoter`: (str): Идентификатор промоутера.

**Методы**:

#### `__init__`

**Описание**: Инициализирует объект `Facebook`.

**Параметры**:

- `driver` (Driver): Веб-драйвер.
- `promoter` (str): Идентификатор промоутера.
- `group_file_paths` (list[str]): Список путей к файлам групп.
- `*args`: Дополнительные аргументы.
- `**kwards`: Дополнительные ключевые аргументы.

**Возвращает**:
- Не имеет возвращаемого значения.


#### `login`

**Описание**: Выполняет вход в систему Facebook.

**Возвращает**:
- `bool`: `True`, если вход успешен, иначе `False`.

#### `promote_post`

**Описание**: Отправляет текст в форму сообщения.

**Параметры**:

- `item` (SimpleNamespace): Объект данных для продвижения поста.


**Возвращает**:
- `bool`: `True`, если продвижение успешно, иначе `False`.


#### `promote_event`

**Описание**: Пример функции для продвижения события.

**Параметры**:

- `event` (SimpleNamespace): Объект данных для продвижения события.

**Возвращает**:
- Не имеет возвращаемого значения.
