# Модуль `hypotez/src/endpoints/advertisement/facebook/scenarios/post_event.py`

## Обзор

Этот модуль содержит функции для публикации календарных событий в группах Facebook. Он использует Selenium WebDriver для взаимодействия с веб-сайтом Facebook. Модуль загружает локаторы из файла JSON и использует их для отправки заголовков, дат, времени, описаний событий.

## Функции

### `post_title`

**Описание**: Отправляет заголовок события на Facebook.

**Параметры**:
- `d` (Driver): Экземпляр драйвера Selenium WebDriver.
- `title` (str): Заголовок события.

**Возвращает**:
- `bool`: `True`, если заголовок успешно отправлен, иначе `None`.

**Вызывает исключения**:
- Возможны исключения, связанные с ошибками взаимодействия с веб-сайтом.

### `post_event`

**Описание**: Управляет процессом публикации события с заголовком, описанием и медиафайлами.

**Параметры**:
- `d` (Driver): Экземпляр драйвера Selenium WebDriver.
- `event` (SimpleNamespace): Объект, содержащий данные события (заголовок, описание, дата, время, ссылка на промо).

**Возвращает**:
- `bool`: `True`, если событие успешно опубликовано, иначе `None`.

**Вызывает исключения**:
- Возможны исключения, связанные с ошибками взаимодействия с веб-сайтом при отправке каждого элемента.
