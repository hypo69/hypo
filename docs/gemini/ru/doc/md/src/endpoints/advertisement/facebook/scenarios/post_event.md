# Модуль `hypotez/src/endpoints/advertisement/facebook/scenarios/post_event.py`

## Обзор

Модуль `post_event.py` содержит функции для публикации календарных событий в группах Facebook. Он использует Selenium WebDriver для взаимодействия с веб-сайтом Facebook и предоставляет методы для заполнения необходимых полей события (заголовок, дата, время, описание).

## Функции

### `post_title`

**Описание**: Отправляет заголовок события.

**Параметры**:

- `d` (Driver): Экземпляр драйвера, используемого для взаимодействия с веб-страницей.
- `title` (str): Заголовок события.

**Возвращает**:

- `bool`: `True`, если заголовок был отправлен успешно, иначе `None`.

**Вызывает исключения**:

- Возможны исключения, связанные с взаимодействием с веб-сайтом.  Модуль `logger` обрабатывает эти ошибки.


### `post_date`

**Параметры**:

- `date` (str): Дата события в формате "YYYY-MM-DD".

### `post_event`

**Описание**: Управляет процессом публикации события с заголовком, описанием и ссылками.

**Параметры**:

- `d` (Driver): Экземпляр драйвера, используемого для взаимодействия с веб-страницей.
- `event` (SimpleNamespace): Объект, содержащий данные события (заголовок, описание, дату, время).

**Возвращает**:

- `bool`: `True`, если событие было опубликовано успешно, иначе `None`.

**Вызывает исключения**:

- Возможны исключения, связанные с взаимодействием с веб-сайтом, передачей данных или проблемами с форматированием данных события. Модуль `logger` обрабатывает эти ошибки.

**Примечания**:

- Функция `post_event` последовательно вызывает функции для заполнения полей события.
- `event` должен содержать атрибуты `title`, `description`, `start` (в формате 'YYYY-MM-DD HH:MM').
- Добавлен обработчик ошибок для каждой функции.
- Добавлена задержка `time.sleep(30)` после отправки события для предотвращения быстрого закрытия браузера.
