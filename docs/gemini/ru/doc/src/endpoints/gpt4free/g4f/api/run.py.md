# Модуль запуска API g4f

## Обзор

Этот модуль предназначен для запуска API g4f (gpt4free). Он импортирует модуль `g4f.api` и запускает API с включенным режимом отладки.

## Подробней

Этот файл является точкой входа для запуска API g4f. Он использует функцию `run_api` из модуля `g4f.api` для запуска API-сервера. Включение режима отладки позволяет получать более подробные сообщения об ошибках и другую полезную информацию во время разработки и тестирования.

## Функции

### `run_api`

```python
def run_api(debug: bool = False) -> None:
    """
    Запускает API g4f.

    Args:
        debug (bool, optional): Флаг, определяющий, запускать ли API в режиме отладки. По умолчанию `False`.

    Returns:
        None: Функция ничего не возвращает.
    """
    ...
```

**Назначение**: Запускает API g4f.

**Параметры**:
- `debug` (bool, optional): Если установлено значение `True`, API запускается в режиме отладки, что позволяет выводить подробные сообщения об ошибках. По умолчанию `False`.

**Возвращает**:
- `None`: Функция ничего не возвращает.

**Как работает функция**:
1. Функция `run_api` вызывается из модуля `g4f.api`.
2. Если параметр `debug` установлен в `True`, API запускается в режиме отладки.

**Примеры**:

```python
import g4f.api

# Запуск API в режиме отладки
g4f.api.run_api(debug=True)

# Запуск API без режима отладки
g4f.api.run_api()
```
