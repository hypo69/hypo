Morlevi Sandisk Categories Storage Scenarios
==========================================

This file documents the scenarios for Morlevi Sandisk storage categories.  It details various storage options, including SSDs (SATA 3, M.2 SATA, NVMe, NVMe Gen4), HDDs, and external drives.  Each scenario contains information about the product, including brand, name, URL, and associated PrestaShop categories.
