# Модуль hypotez/src/templates/header.py

## Обзор

Модуль `header.py` содержит функцию `set_project_root`, которая определяет корневую директорию проекта, начиная с директории текущего файла. Эта функция ищет наличие определенных маркеров (файлов или каталогов) в родительских директориях текущего файла.  Если корневой каталог найден, он добавляется в переменную `sys.path`, что позволяет импортировать модули из этого каталога.

## Функции

### `set_project_root`

**Описание**:  Определяет корневую директорию проекта, начиная с директории текущего файла. Ищет наличие указанных файлов-маркеров в родительских директориях.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или каталогов, которые указывают на корневую директорию проекта. По умолчанию включает `pyproject.toml`, `requirements.txt` и `.git`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращает директорию, в которой находится скрипт.


**Вызывает исключения**:

-  Никаких исключений не генерирует.

## Переменные

### `MODE`

**Описание**: Строковая переменная, хранящая значение режима работы. По умолчанию `'dev'`.

### `__root__`

**Описание**: Переменная, хранящая путь к корневой директории проекта. Она инициализируется функцией `set_project_root`.  Тип `Path`.
