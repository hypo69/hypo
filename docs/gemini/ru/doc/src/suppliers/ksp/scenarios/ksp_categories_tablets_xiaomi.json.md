# Документация для `ksp_categories_tablets_xiaomi.json`

## Обзор

Файл `ksp_categories_tablets_xiaomi.json` содержит JSON-конфигурацию для сценариев парсинга планшетов Xiaomi с сайта KSP. Он определяет параметры для конкретной модели планшета, включая бренд, URL, настройки чекбокса, статус активности, состояние товара, а также соответствия категориям PrestaShop.

## Структура

### Сценарии
В этом разделе описываются различные сценарии для парсинга, представленные в виде объекта. Ключами объекта являются названия конкретных моделей или конфигураций.

#### Xiaomi Pad 5

Данный раздел определяет сценарий парсинга для планшета Xiaomi Pad 5.

**Описание**: Конфигурация для парсинга планшета Xiaomi Pad 5.

**Параметры**:

- `brand` (str): Бренд планшета, в данном случае "XIAOMI".
- `url` (str): URL-адрес страницы с планшетами Xiaomi на сайте KSP.
- `checkbox` (bool): Флаг, указывающий, нужно ли использовать чекбокс (в данном случае `false`).
- `active` (bool): Флаг, указывающий, является ли сценарий активным (в данном случае `true`).
- `condition` (str): Состояние товара, в данном случае "new" (новый).
- `presta_categories` (dict): Соответствие категорий PrestaShop, где ключами являются ID категорий PrestaShop, а значениями - наименования категорий.

## Описание полей

- `scenarios` (dict): Объект, содержащий сценарии для парсинга.
- `brand` (str): Бренд товара.
- `url` (str): URL-адрес для парсинга.
- `checkbox` (bool): Флаг использования чекбокса.
- `active` (bool): Флаг активности сценария.
- `condition` (str): Состояние товара (например, "new" или "used").
- `presta_categories` (dict): Соответствие категорий PrestaShop.
