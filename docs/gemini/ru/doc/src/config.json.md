# Конфигурационный файл `config.json`

## Обзор

Файл `config.json` содержит конфигурационные параметры для проекта `hypotez`. Он определяет автора проекта, его название, режим работы, доступные режимы, настройки git, пути к различным хранилищам данных, формат временных меток, релиз, версию, информацию об авторских правах и ссылку для поддержки разработчика.

## Структура JSON

### Основные параметры

- `author` (str): Автор проекта - "Hypo69".
- `project_name` (str): Название проекта - "hypotez".
- `mode` (str): Текущий режим работы проекта - "dev".
- `avaible_modes` (list): Список доступных режимов работы - `["dev", "debug", "test", "prod"]`.

### Настройки Git

- `git` (str): Имя репозитория git - "hypo".
- `git_user` (str): Имя пользователя git - "hypo69".

### Пути

- `path` (dict): Словарь, содержащий пути к различным директориям.
    - `external_storage` (str): Путь к внешнему хранилищу данных - "E:\\hypo69\\data".
    - `google_drive` (str): Путь к папке Google Drive - "H:\\My Drive\\hypotez".
    - `log` (str): Путь к директории для хранения логов - "E:\\hypo69\\log".
    - `tmp` (str): Путь к директории для временных файлов - "E:\\hypo69\\tmp".

### Формат временных меток

- `timestamp_format` (str): Формат для создания временных меток - "%y%m%d%H%M%S%f".

### Версия и авторские права

- `release` (str): Версия релиза - "0.1".
- `version` (str): Версия проекта - "0.11".
- `copyright` (str): Информация об авторских правах - "2024, hypo69".

### Поддержка разработчика
- `cofee` (str): Ссылка для поддержки разработчика - "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69".
