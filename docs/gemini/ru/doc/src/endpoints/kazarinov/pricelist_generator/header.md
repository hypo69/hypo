# Модуль hypotez/src/endpoints/kazarinov/scenarios/header.py

## Обзор

Этот модуль содержит вспомогательные функции для работы с настройками проекта, включая поиск корневой директории проекта и чтение файла настроек.  Он предоставляет переменные, содержащие имя проекта, версию, описание, подробности, автора, копирайт и ссылку на спонсорскую страницу.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная от текущей директории и ищет вверх по дереву директорий, пока не найдет директорию, содержащую один из файлов-маркеров (pyproject.toml, requirements.txt или .git).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для определения корневой директории проекта. По умолчанию: ('pyproject.toml', 'requirements.txt', '.git').


**Возвращает**:

- `Path`: Объект Path, представляющий путь к корневой директории проекта, если она найдена. В противном случае возвращает директорию, в которой расположен текущий скрипт.

**Вызывает исключения**:

- Не вызывает исключений.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта.  Инициализируется функцией `set_project_root`.


### `settings`

**Описание**: Словарь, содержащий настройки проекта.  Считывается из файла `src/settings.json` в корневой директории проекта.

**Инициализация**:

- Переменная `settings` инициализируется значением `None`.
- Внутри блока `try...except` происходит попытка чтения файла `src/settings.json` и загрузки его содержимого в переменную `settings` в формате JSON.
- В случае ошибок (FileNotFoundError или json.JSONDecodeError), блок `except` обрабатывает ошибку и не изменяет значение `settings`.



### `doc_str`

**Описание**: Строка, содержащая содержимое файла README.MD в корневой директории проекта.

**Инициализация**:

- Переменная `doc_str` инициализируется значением `None`.
- Внутри блока `try...except` происходит попытка чтения файла `src/README.MD` и чтения его содержимого в переменную `doc_str`.
- В случае ошибок (FileNotFoundError или json.JSONDecodeError), блок `except` обрабатывает ошибку и не изменяет значение `doc_str`.


### `__project_name__`

**Описание**: Имя проекта. Получается из настроек (settings) или имеет значение по умолчанию "hypotez", если настройки не найдены или некорректны.


### `__version__`

**Описание**: Версия проекта. Получается из настроек (settings) или имеет значение по умолчанию пустую строку, если настройки не найдены или некорректны.


### `__doc__`

**Описание**: Документация проекта. Получается из переменной `doc_str` или имеет значение по умолчанию пустую строку, если `doc_str` не определен.


### `__details__`

**Описание**: Подробности проекта. По умолчанию пустая строка.


### `__author__`

**Описание**: Автор проекта. Получается из настроек (settings) или имеет значение по умолчанию пустую строку, если настройки не найдены или некорректны.


### `__copyright__`

**Описание**: Авторские права проекта. Получается из настроек (settings) или имеет значение по умолчанию пустую строку, если настройки не найдены или некорректны.


### `__cofee__`

**Описание**: Ссылка на спонсорскую страницу для поддержки разработчика. Получается из настроек (settings) или имеет значение по умолчанию ссылку, если настройки не найдены или некорректны.
