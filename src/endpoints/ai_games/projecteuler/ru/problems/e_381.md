# Задача No 381
## (n, k)-последовательности
## Условия
Для натурального числа *n* определим *S(n)* как сумму факториалов *(n-1)! + (n-2)! + ... + 1! + 0!*.

Например, если *n* = 4, то *S(4) = 3! + 2! + 1! + 0! = 6 + 2 + 1 + 1 = 10*.

Определим *F(n)* как результат *S(n) mod n*.
Например, *F(4) = 10 mod 4 = 2*.

Также определим *SF(L, U)* как сумму *F(n)* для всех целых чисел *n* между *L* и *U*, включительно.
Например, *SF(3, 10) = F(3) + F(4) + F(5) + F(6) + F(7) + F(8) + F(9) + F(10) = 0 + 2 + 3 + 3 + 5 + 2 + 5 + 8 = 28*.

Найдите *SF(2500, 100000000)*.
