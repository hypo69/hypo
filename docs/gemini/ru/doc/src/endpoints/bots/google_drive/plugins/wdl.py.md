# Модуль для скачивания файлов с использованием wget

## Обзор

Модуль предоставляет функцию `wget_dl`, которая позволяет скачивать файлы по заданному URL-адресу с использованием утилиты `wget`. Функция обрабатывает исключения и возвращает имя скачанного файла или сообщение об ошибке.

## Функции

### `wget_dl`

```python
def wget_dl(url: str) -> str | tuple[str, str]:
    """
    Скачивает файл по заданному URL-адресу с использованием утилиты `wget`.

    Args:
        url (str): URL-адрес файла для скачивания.

    Returns:
        str | tuple[str, str]: Имя скачанного файла в случае успеха или кортеж ("error", filename) в случае ошибки.

    Raises:
        Exception: В случае возникновения ошибки при скачивании файла.

    Example:
        >>> wget_dl('https://example.com/file.txt')
        Downloading Started
        Downloading Complete file.txt
        'file.txt'

        >>> wget_dl('https://example.com/nonexistent_file.txt')
        Downloading Started
        DOWNLAOD ERROR : ...
        ('error', 'nonexistent_file.txt')
    """
    ...
```

**Назначение**:
Функция `wget_dl` предназначена для скачивания файлов по указанному URL-адресу с использованием утилиты `wget`.

**Параметры**:
- `url` (str): URL-адрес файла, который необходимо скачать.

**Возвращает**:
- `str`: Имя скачанного файла в случае успешного скачивания.
- `tuple[str, str]`: Кортеж, содержащий строку "error" и имя файла, в случае возникновения ошибки при скачивании.

**Вызывает исключения**:
- `Exception`: Возникает в случае, если процесс скачивания завершается с ошибкой.

**Как работает функция**:

1.  **Начало скачивания**: Выводит сообщение "Downloading Started" в консоль.
2.  **Определение имени файла**: Извлекает имя файла из URL-адреса с помощью `os.path.basename(url)`.
3.  **Выполнение команды wget**: Формирует и выполняет команду `wget` с использованием `subprocess.check_output`. Команда `wget` скачивает файл по указанному URL-адресу и сохраняет его под именем `filename`.
4.  **Обработка результата**: Если команда выполнена успешно, выводит сообщение "Downloading Complete" и имя файла в консоль, а затем возвращает имя файла.
5.  **Обработка ошибок**: Если в процессе выполнения команды возникает исключение, выводит сообщение об ошибке в консоль и возвращает кортеж `("error", filename)`.

**ASCII flowchart**:

```
Начало
  │
  │ URL
  ▼
Извлечение имени файла
  │
  │ Имя файла
  ▼
Формирование команды wget
  │
  │ Команда wget
  ▼
Выполнение команды wget
  │
  │ Успех/Ошибка
  ▼
Успех: Вывод сообщения об успехе и возврат имени файла
Ошибка: Вывод сообщения об ошибке и возврат кортежа ("error", filename)
  │
Конец
```

**Примеры**:

```python
# Пример успешного скачивания файла
url = 'https://www.easygifanimator.net/images/samples/video-to-gif-sample.gif'
result = wget_dl(url)
print(result)  # Вывод: video-to-gif-sample.gif

# Пример неуспешного скачивания файла (несуществующий URL)
url = 'https://www.easygifanimator.net/images/samples/nonexistent_file.gif'
result = wget_dl(url)
print(result)  # Вывод: ('error', 'nonexistent_file.gif')
