# Модуль borderify.js

## Обзор

Этот модуль содержит JavaScript-код, который добавляет красную рамку шириной 5 пикселей к элементу `body`. Также используется функция `alert()`.

## Функции

### `document.body.style.border = "5px solid red";`

**Описание**: Эта строка кода устанавливает свойство `border` элемента `body` на значение "5px solid red", добавляя красную рамку шириной 5 пикселей.

**Параметры**:

* Нет параметров.

**Возвращает**:

* Нет возвращаемого значения.  Изменяет свойство `style` элемента `body`.

**Вызывает исключения**:

* Нет.


### `alert()`

**Описание**: Эта функция отображает диалоговое окно со всплывающим сообщением `alert()`.

**Параметры**:

* Нет обязательных параметров. Если функция вызывается без аргументов, ничего не отображается. Если передаётся строковый аргумент, то он отображается как сообщение.

**Возвращает**:

* Нет возвращаемого значения.

**Вызывает исключения**:

* Нет.
