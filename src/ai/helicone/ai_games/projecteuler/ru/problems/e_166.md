# Задача No 166
## Обратные магические квадраты 4х4

## Условия
Заполните квадрат 4x4 числами от 0 до 9 так, чтобы суммы чисел в каждой строке, каждом столбце и двух главных диагоналях были равны. Например, один из возможных вариантов магического квадрата такой:

```
  0  1  2  3
  1  2  3  0
  2  3  0  1
  3  0  1  2
```
Назовем этот магический квадрат "нормальным", если сумма цифр равна 10. 

Однако, магические квадраты могут быть обратными. Квадрат 4x4 является обратным магическим квадратом, если суммы чисел в каждой строке и столбце равны, а суммы на главных диагоналях НЕ равны. Ниже приведен пример такого обратного магического квадрата:

```
  1 1 1 1
  3 0 0 1
  0 3 1 0
  0 0 3 1
```
Здесь каждая строка и каждый столбец имеет сумму 4, а диагональные суммы равны 5 и 7 соответственно.

Теперь, вместо поиска магических квадратов, нужно рассмотреть обратные магические квадраты, где числа находятся в диапазоне от 0 до 9.

Сколько всего существует обратных магических квадратов 4х4 с использованием чисел 0-9?
