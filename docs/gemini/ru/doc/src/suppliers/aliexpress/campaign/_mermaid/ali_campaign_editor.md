## Класс `AliCampaignEditor`

**Описание**: Класс `AliCampaignEditor` отвечает за взаимодействие с данными кампаний AliExpress. Он позволяет редактировать информацию о продуктах, категориях и самой кампании.

### `__init__`

**Описание**: Конструктор класса `AliCampaignEditor`.

**Параметры**:
- `campaign_name` (str): Название кампании.
- `language` (str): Язык кампании.
- `currency` (str): Валюта кампании.

**Возвращает**:
- `AliCampaignEditor`: Экземпляр класса `AliCampaignEditor` с проинициализированными данными.

**Вызывает исключения**:
- `ValueError`: Если любой из параметров не соответствует ожидаемому типу или формату.


### `delete_product`

**Описание**: Удаляет продукт из кампании.

**Параметры**:
- `product_id` (str): ID продукта для удаления.

**Возвращает**:
- `bool`: `True`, если продукт успешно удален, `False` иначе.

**Вызывает исключения**:
- `FileNotFoundError`: Если файл продукта не найден.
- `ValueError`: Если указан неверный формат `product_id`.


### `update_product`

**Описание**: Обновляет данные о продукте в кампании.

**Параметры**:
- `product_data` (dict): Данные для обновления продукта.


**Возвращает**:
- `bool`: `True`, если продукт успешно обновлен, `False` иначе.

**Вызывает исключения**:
- `TypeError`: Если `product_data` не является словарем.
- `ValueError`: Если `product_data` не содержит необходимых данных.


### `update_campaign`

**Описание**: Обновляет свойства кампании, например, описание.

**Параметры**:
- `campaign_updates` (dict): Словарь с обновляемыми свойствами кампании.

**Возвращает**:
- `bool`: `True`, если кампания успешно обновлена, `False` иначе.

**Вызывает исключения**:
- `TypeError`: Если `campaign_updates` не является словарем.
- `ValueError`: Если `campaign_updates` содержит некорректные данные.


### `update_category`

**Описание**: Обновляет данные о категории в файле JSON.

**Параметры**:
- `category_name` (str): Имя категории.
- `category_data` (dict): Данные для обновления категории.

**Возвращает**:
- `bool`: `True`, если категория успешно обновлена, `False` иначе.

**Вызывает исключения**:
- `FileNotFoundError`: Если файл категории не найден.
- `TypeError`: Если `category_data` не является словарем.


### `get_category`

**Описание**: Возвращает данные о категории по имени.

**Параметры**:
- `category_name` (str): Имя категории.

**Возвращает**:
- `SimpleNamespace`: Объект с данными о категории. Возвращает `None`, если категория не найдена.


**Вызывает исключения**:
- `TypeError`: Если `category_name` не является строкой.


### `list_categories`

**Описание**: Возвращает список имен всех категорий в кампании.


**Возвращает**:
- `list`: Список имен категорий.

**Вызывает исключения**:
- `FileNotFoundError`: Если файл с категориями не найден.



### `get_category_products`

**Описание**: Возвращает список продуктов для указанной категории.


**Параметры**:
- `category_name` (str): Имя категории.


**Возвращает**:
- `list`: Список продуктов для указанной категории.


**Вызывает исключения**:
- `FileNotFoundError`: Если файлы продуктов не найдены.
- `TypeError`: Если `category_name` не является строкой.
