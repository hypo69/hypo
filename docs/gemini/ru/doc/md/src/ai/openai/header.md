# Модуль hypotez/src/logger

## Обзор

Модуль `src.logger` определяет корневой путь к проекту `hypotez`.  Все импорты строятся относительно этого пути. В дальнейшем планируется перенести эту функцию в системные переменные.

## Функции

### `set_project_root`

**Описание**:  Находит корневую директорию проекта, начиная с директории текущего файла. Поиск идёт вверх по директориям, пока не найдёт директорию, содержащую один из указанных маркеров (файлов или папок).

**Параметры**:

- `marker_files` (tuple): Кортеж имён файлов или директорий, которые указывают на корневую директорию проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена. В противном случае возвращает путь к директории текущего файла.

**Вызывает исключения**:

- Никаких исключений не вызывает.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root`.


## Вспомогательные переменные

### `MODE`

**Описание**:  Строковая переменная, хранящая текущий режим работы (например, 'dev', 'prod').  Значение по умолчанию - 'dev'.

### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженный из файла `src/settings.json`. Может быть `None`, если файл не найден или невалидный.


### `doc_str`

**Описание**: Строка, содержащая содержимое файла `README.MD` проекта, если он существует. Может быть `None` если файл не найден.


### `__project_name__`

**Описание**: Название проекта. Получается из `settings.json` или устанавливается по умолчанию как `hypotez`.

### `__version__`

**Описание**: Версия проекта. Получается из `settings.json` или устанавливается по умолчанию как пустая строка.

### `__doc__`

**Описание**: Документация проекта, полученная из файла `README.MD`.

### `__details__`

**Описание**: Подробная информация о проекте (пустая строка по умолчанию).

### `__author__`

**Описание**: Автор проекта. Получается из `settings.json` или устанавливается по умолчанию как пустая строка.

### `__copyright__`

**Описание**: Авторские права. Получается из `settings.json` или устанавливается по умолчанию как пустая строка.


### `__cofee__`

**Описание**: Ссылка на боости сайт для пожертвований. Получается из `settings.json` или устанавливается по умолчанию как ссылка.
