# Модуль hypotez/src/goog/drive/header.py

## Обзор

Данный модуль содержит функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и чтения файла `README.MD`. Он также предоставляет переменные с информацией о проекте, такие как имя проекта, версия, документация, автор и т.д.

    - [`set_project_root`](#set_project_root)


## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с текущей директории и двигаясь вверх по дереву директорий. Поиск завершается, когда находится директория, содержащая один из указанных файлов или директорий.

**Параметры**:

- `marker_files` (tuple): Кортеж из имён файлов или директорий, используемых для определения корневой директории проекта. По умолчанию это (`pyproject.toml`, `requirements.txt`, `.git`).


**Возвращает**:

- `Path`: Объект `Path` представляющий путь к корневой директории проекта. В случае, если корневая директория не найдена, возвращает директорию, где находится текущий файл.

**Вызывает исключения**:

- Нет


## Инициализация

**Описание**: Этот раздел описывает, как происходит инициализация переменных проекта.

**Описание переменных**:

- `__root__` (Path): Путь к корневой директории проекта, полученный с помощью функции `set_project_root`.
- `settings` (dict): Словарь с настройками проекта, загруженный из файла `settings.json`. Если файл не найден или имеет неправильный формат, `settings` устанавливается в `None`.
- `doc_str` (str): Строка, содержащая содержимое файла `README.MD`. Если файл не найден или имеет неправильный формат, `doc_str` устанавливается в `None`.
- `__project_name__` (str): Имя проекта, полученное из настроек (`settings`). По умолчанию `hypotez`.
- `__version__` (str): Версия проекта, полученная из настроек (`settings`). По умолчанию пустая строка.
- `__doc__` (str): Документация проекта, полученная из файла `README.MD`. По умолчанию пустая строка.
- `__details__` (str): Подробная информация о проекте (по умолчанию пустая строка).
- `__author__` (str): Автор проекта, полученный из настроек (`settings`). По умолчанию пустая строка.
- `__copyright__` (str): Авторские права, полученные из настроек (`settings`). По умолчанию пустая строка.
- `__cofee__` (str): Ссылка для поддержки разработчика. По умолчанию ссылка на Boosty.


**Обработка ошибок**:

В коде используются блоки `try...exept` для обработки возможных ошибок:

- `FileNotFoundError`: Если файлы `settings.json` или `README.MD` не найдены.
- `json.JSONDecodeError`: Если содержимое файла `settings.json` не является корректным JSON.

В случае возникновения ошибки, соответствующие переменные устанавливаются в значения по умолчанию.
