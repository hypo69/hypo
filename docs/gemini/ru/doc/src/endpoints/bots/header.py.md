# Модуль `header.py`

## Обзор

Этот модуль содержит настройки и метаданные проекта, а также утилиту для определения корневой директории проекта. Он обеспечивает доступ к основным параметрам проекта, таким как имя, версия, описание и информация об авторе.

## Функции

### `set_project_root`

**Описание**:
Находит корневую директорию проекта, начиная с директории текущего файла, поднимаясь вверх по иерархии каталогов и останавливаясь на первом каталоге, содержащем один из заданных маркерных файлов.

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или каталогов, используемых для идентификации корневой директории проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращает директорию, где расположен скрипт.

## Переменные модуля

### `MODE`
**Описание**:
Режим работы проекта. Значение по умолчанию - `dev`.

### `__root__`
**Описание**:
Путь к корневой директории проекта.

### `settings`
**Описание**:
Словарь с настройками проекта, загруженный из файла `settings.json`. Может быть `None`, если файл не найден или не может быть распарсен.

### `doc_str`
**Описание**:
Строка с содержимым файла `README.MD`. Может быть `None`, если файл не найден или не может быть прочитан.

### `__project_name__`
**Описание**:
Имя проекта, полученное из настроек или значение по умолчанию `hypotez`.

### `__version__`
**Описание**:
Версия проекта, полученная из настроек или пустая строка.

### `__doc__`
**Описание**:
Описание проекта, полученное из файла `README.MD` или пустая строка.

### `__details__`
**Описание**:
Детальное описание проекта. На данный момент всегда пустая строка.

### `__author__`
**Описание**:
Автор проекта, полученный из настроек или пустая строка.

### `__copyright__`
**Описание**:
Информация об авторских правах проекта, полученная из настроек или пустая строка.

### `__cofee__`
**Описание**:
Строка с призывом угостить разработчика кофе. Значение по умолчанию `Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69` или значение из настроек.
