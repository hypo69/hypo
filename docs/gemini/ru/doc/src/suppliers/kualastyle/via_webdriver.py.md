# Модуль для парсинга Kualastyle через WebDriver

## Обзор

Модуль `via_webdriver.py` предназначен для парсинга данных с сайта Kualastyle с использованием WebDriver. Он содержит функции для получения списка URL продуктов из категории.

## Подробней

Модуль использует WebDriver для взаимодействия с веб-страницами, прокрутки страниц и извлечения информации о продуктах. Он предназначен для автоматизации процесса сбора данных о продуктах с сайта Kualastyle. В модуле используются логи и модуль `gs`.

## Функции

### `get_list_products_in_category`

```python
def get_list_products_in_category(s) -> list[str,str,None]:
    """ Returns list of products urls from category page
    Attrs:
        s - Suplier
    @returns
        list of products urls or None
    """
```

**Назначение**: Возвращает список URL продуктов со страницы категории.

**Параметры**:

-   `s`: Объект поставщика (Suplier), содержащий информацию о драйвере и локаторах.

**Возвращает**:

-   `list[str,str,None]`: Список URL продуктов или `None` в случае ошибки.

**Как работает функция**:

1.  **Инициализация**:
    -   Из объекта поставщика `s` извлекаются драйвер `d` и локаторы `l` для категории.

2.  **Прокрутка страницы**:
    -   Выполняется прокрутка страницы вниз (`direction = "forward"`) 10 раз (`scroll_count = 10`) с использованием метода `d.scroll()`.

3.  **Получение списка продуктов**:
    -   Извлекается функция `d.execute_locator` во временную переменную `_`.
    -   Выполняется поиск элементов, соответствующих локатору `l['product_links']` на странице, и результат сохраняется в `list_products_in_category`.

4.  **Возврат результата**:
    -   Возвращается список `list_products_in_category`, содержащий URL продуктов.
