# Документация для файла popup.js

## Обзор

Этот файл содержит JavaScript код для веб-расширения, взаимодействующего с сервером для получения и отправки сообщений.  Он использует AngularJS для управления пользовательским интерфейсом и `$http` для выполнения запросов к FastAPI серверу.

    * [loadAssistants](#loadassistants)
    * [sendMessage](#sendmessage)

## Функции

### `loadAssistants`

**Описание**: Загружает список ассистентов с сервера.

**Параметры**:
Нет.

**Возвращает**:
-  Список ассистентов (массив объектов).


**Обрабатывает исключения**:
-  Если произошла ошибка при загрузке данных (например, ошибка сети), выводится сообщение об ошибке в консоль.


### `sendMessage`

**Описание**: Отправляет сообщение на сервер для обработки моделью.

**Параметры**:
-  `message` (строка): Сообщение, которое нужно отправить.
-  `selectedAssistant` (объект): Объект выбранного ассистента, необходимый для передачи ID.  Если `selectedAssistant` не выбран (null), то поведение не определено.


**Возвращает**:
- Нет.

**Обрабатывает исключения**:
-  Если произошла ошибка при отправке или получении ответа (например, ошибка сети), выводится сообщение об ошибке в консоль и отображается сообщение "Произошла ошибка. Попробуйте позже." в области отображения ответа.
