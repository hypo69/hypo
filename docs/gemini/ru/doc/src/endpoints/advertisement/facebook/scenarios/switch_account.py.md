# Модуль `switch_account`

## Обзор

Модуль предназначен для переключения между аккаунтами в Facebook. Он использует веб-драйвер для поиска и нажатия на кнопку переключения аккаунта, если она доступна.

## Подробнее

Этот модуль является частью системы автоматизации задач, связанных с рекламой в Facebook. Он обеспечивает возможность переключения между различными аккаунтами для управления рекламными кампаниями. Модуль использует JSON-файл с локаторами для поиска элементов на странице.

## Функции

### `switch_account`

```python
def switch_account(driver: Driver):
    """ Если есть кнопка `Переключить` - нажимаю её  """
```

**Назначение**: Переключает аккаунт в Facebook, если доступна кнопка "Переключить".

**Параметры**:
- `driver` (Driver): Экземпляр веб-драйвера, используемый для взаимодействия с браузером.

**Возвращает**:
- None

**Вызывает исключения**:
- Отсутствуют явные исключения, но могут возникать исключения, связанные с работой веб-драйвера.

**Как работает функция**:

1. Функция `switch_account` принимает объект `driver` типа `Driver`, который используется для управления браузером.
2. Функция использует метод `execute_locator` объекта `driver` для поиска и выполнения действия (нажатия) на элементе, соответствующем локатору `locator.switch_to_account_button`.
3. Если элемент найден и действие выполнено успешно, происходит переключение аккаунта.

```
Начало
  ↓
driver.execute_locator(locator.switch_to_account_button)
  ↓
Конец
```

Пример вызова функции `switch_account` с передачей экземпляра драйвера:
