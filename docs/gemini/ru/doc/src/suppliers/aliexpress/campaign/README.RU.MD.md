# Модуль `campaign`

## Обзор

Модуль `campaign` предназначен для управления процессом создания и публикации рекламных кампаний на Facebook.
Он включает функционал для инициализации параметров кампании (название, язык, валюта),
создания структуры директорий, сохранения конфигураций для новой кампании,
сбора и сохранения данных о продуктах через `ali` или `html`,
генерации рекламных материалов, проверки кампании и публикации ее на Facebook.

## Подробней

Модуль предоставляет инструменты для автоматизации процесса создания и управления рекламными кампаниями.
Он позволяет определить основные параметры кампании, такие как название, язык и валюта,
создать необходимую структуру директорий для хранения данных и конфигураций,
а также собирать и сохранять информацию о продуктах для использования в рекламных материалах.
Этот модуль является ключевым компонентом в системе управления рекламными кампаниями,
обеспечивая гибкость и масштабируемость процесса.
