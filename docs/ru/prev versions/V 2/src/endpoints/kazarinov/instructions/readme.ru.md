# Инструкции и системные промпты для модели AI

## Обзор

Этот документ описывает файлы, содержащие инструкции и системные промпты для модели AI.

## Файлы

### `system_instruction_mexiron.md`

**Описание**: Этот файл содержит системный промпт, который загружается при инициализации модели. Он описывает поведение, которое должна демонстрировать модель после инициализации.

### `command_instruction_mexiron.md`

**Описание**: Этот файл содержит инструкцию, которая представляет собой короткое указание для модели. В инструкции описан шаблон, которому должна соответствовать модель при формировании ответа.
