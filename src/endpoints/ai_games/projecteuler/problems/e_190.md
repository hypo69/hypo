# Задача No 190
## Максимизация взвешенного геометрического среднего
## Условия

Пусть $S(n)$ будет последовательностью $n$ чисел $x_1, x_2, ..., x_n$, где $x_1 + x_2 + ... + x_n = n$, и $x_i > 0$ для всех $1 ≤ i ≤ n$.

Определим $P(n)$ как максимальное значение произведения $x_1 x_2^2 x_3^3 ... x_n^n$.

Например, если $n = 2$, то
$S(2)$ может быть: $\{0.5, 1.5\}$, $\{1, 1\}$ или $\{0.1, 1.9\}$.
Легко убедиться, что для $n = 2$ максимальное значение $x_1 x_2^2$ получается при $x_1 = \frac{2}{3}$ и $x_2 = \frac{4}{3}$, и в этом случае $P(2) = \frac{2}{3} \times (\frac{4}{3})^2 = \frac{32}{27}$.

Для $n = 3$, максимальное значение $x_1 x_2^2 x_3^3$ получается при $x_1 = \frac{3}{6}$, $x_2 = \frac{6}{6}$ и $x_3 = \frac{9}{6}$, и в этом случае $P(3) = \frac{3}{6} \times (\frac{6}{6})^2 \times (\frac{9}{6})^3 = \frac{729}{216} = \frac{27}{8}$.

Найдите $\sum P(n)$ для $2 ≤ n ≤ 15$.
