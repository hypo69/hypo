# Задача No 727
## Числа, выразимые как сумма двух квадратов и одного куба
## Условия
Натуральное число *n* называется выразимым как сумма двух квадратов и одного куба, если существуют целые неотрицательные числа *a*, *b* и *c* такие, что *n* = *a*² + *b*² + *c*³.

Пусть *S*(*N*) — количество чисел, не превосходящих *N*, которые выразимы как сумма двух квадратов и одного куба.

Например, следующие числа меньше или равны 100, и выразимы таким образом:

0 = 0² + 0² + 0³
1 = 1² + 0² + 0³
2 = 1² + 1² + 0³
3 = 1² + 1² + 1³
4 = 2² + 0² + 0³
5 = 2² + 1² + 0³
6 = 2² + 1² + 1³
8 = 2² + 2² + 0³
9 = 3² + 0² + 0³
10 = 3² + 1² + 0³
...
99 = 9² + 3² + 3³
100 = 10² + 0² + 0³

Таким образом, *S*(100) = 65.

Известно, что *S*(10⁴) = 3862.

Найдите *S*(10⁹).
