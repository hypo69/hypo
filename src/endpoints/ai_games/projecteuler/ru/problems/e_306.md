# Задача No 306
## Игра с монетами
## Условия

Два игрока, А и Б, играют в следующую игру:
Имеется куча из n монет. За один ход игрок забирает от 1 до m монет. 
Игрок, забравший последнюю монету, побеждает.
Определим через M(n, m) первого игрока в выигрыше, если оба игрока играют оптимально. 
Определим через S(m) количество значений n, для которых M(n, m) = истина, при 1 ≤ n ≤ 100.

Например:
S(1) = 100, потому что первый игрок выигрывает при всех n если m = 1.
S(2) = 67, потому что первый игрок проигрывает при n = 3k. 
S(3) = 76
S(4) = 67
S(5) = 84
S(6) = 67

Найдите S(10^6).
