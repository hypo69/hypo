# Модуль helicone.py

## Обзор

Этот модуль предоставляет класс `HeliconeAI` для взаимодействия с API Helicone и OpenAI, позволяющий генерировать стихотворения, анализировать тональность текста, создавать краткое изложение и переводить текст.  Модуль использует  библиотеки `helicone` и `openai`.

## Класс `HeliconeAI`

### Описание

Класс `HeliconeAI` представляет собой интерфейс для взаимодействия с API Helicone и OpenAI. Он инициализируется экземплярами `Helicone` и `OpenAI` и предоставляет методы для различных задач обработки текста.

### Методы

#### `generate_poem`

##### Описание

Генерирует стихотворение на основе заданного промпта.

##### Параметры

- `prompt` (str): Промпт для генерации стихотворения.

##### Возвращает

- `str`: Сгенерированное стихотворение.


#### `analyze_sentiment`

##### Описание

Анализирует тональность текста.

##### Параметры

- `text` (str): Текст для анализа.

##### Возвращает

- `str`: Результат анализа тональности.


#### `summarize_text`

##### Описание

Создает краткое изложение текста.

##### Параметры

- `text` (str): Текст для изложения.

##### Возвращает

- `str`: Краткое изложение текста.


#### `translate_text`

##### Описание

Переводит текст на указанный язык.

##### Параметры

- `text` (str): Текст для перевода.
- `target_language` (str): Целевой язык перевода.

##### Возвращает

- `str`: Переведенный текст.
