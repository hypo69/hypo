# Модуль `header.py`

## Обзор

Модуль `header.py` предназначен для инициализации основных настроек и переменных проекта. Он выполняет поиск корневой директории проекта, загружает настройки из файла `settings.json` и документацию из `README.md`. Также определяет основные метаданные проекта, такие как имя, версия, автор и т.д.

## Функции

### `set_project_root`

**Описание**:
Функция `set_project_root` находит корневую директорию проекта, начиная с директории текущего файла, двигаясь вверх по дереву каталогов. Поиск прекращается, когда найдена директория, содержащая один из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple): Набор имен файлов или каталогов, используемых для идентификации корневой директории проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории проекта, если она найдена. Если корневая директория не найдена, возвращается путь к директории, где расположен скрипт.

## Переменные

### `MODE`

**Описание**:
Определяет режим работы приложения. В данном случае установлен в `dev`.

### `__root__`

**Описание**:
Содержит путь к корневой директории проекта. Значение определяется функцией `set_project_root`.

### `settings`

**Описание**:
Словарь, содержащий настройки проекта, загруженные из файла `settings.json`. Если файл не найден или не может быть прочитан, значение будет `None`.

### `doc_str`

**Описание**:
Строка, содержащая текст документации, загруженной из файла `README.MD`. Если файл не найден или не может быть прочитан, значение будет `None`.

### `__project_name__`

**Описание**:
Имя проекта, извлекается из настроек. Если в настройках отсутствует, используется значение `hypotez`.

### `__version__`

**Описание**:
Версия проекта, извлекается из настроек. Если в настройках отсутствует, значение будет пустой строкой `''`.

### `__doc__`

**Описание**:
Строка с документацией проекта, загруженной из файла `README.MD`. Если документация не найдена, значение будет пустой строкой `''`.

### `__details__`

**Описание**:
Строка с дополнительной информацией о проекте. По умолчанию - пустая строка `''`.

### `__author__`

**Описание**:
Имя автора проекта, извлекается из настроек. Если в настройках отсутствует, значение будет пустой строкой `''`.

### `__copyright__`

**Описание**:
Информация о копирайте проекта, извлекается из настроек. Если в настройках отсутствует, значение будет пустой строкой `''`.

### `__cofee__`

**Описание**:
Строка с приглашением поддержать разработчика. По умолчанию "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69". Значение извлекается из настроек если доступно.
