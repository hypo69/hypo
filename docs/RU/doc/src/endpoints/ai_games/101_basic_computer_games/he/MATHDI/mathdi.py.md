# mathdi.py

## Обзор

Этот модуль реализует игру "Математическая викторина", в которой пользователю предлагается решить случайные математические задачи на сложение, вычитание, умножение и деление. Игра генерирует два случайных числа и случайную операцию, а затем просит пользователя ввести ответ. Игра предоставляет обратную связь, сообщая, был ли ответ правильным или нет.

- Нет параметров.

**Возвращает**:
