# Модуль `header.py`

## Обзор

Модуль `header.py` предназначен для установки корневой директории проекта, загрузки конфигурации из `config.json` и документации из `README.MD`, а также определения основных атрибутов проекта, таких как имя, версия, автор и т.д.

## Функции

### `set_project_root`

**Описание**:
Находит корневую директорию проекта, начиная с директории текущего файла, и ищет вверх по дереву, останавливаясь на первой директории, содержащей любой из маркерных файлов.

**Параметры**:
- `marker_files` (tuple, optional): Кортеж имен файлов или директорий для идентификации корня проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена, иначе путь к директории, где расположен скрипт.

### Переменные

### `__root__`

**Описание**:
Путь к корневой директории проекта. Определяется функцией `set_project_root`.

### `config`

**Описание**:
Словарь, содержащий конфигурацию проекта, загруженную из файла `config.json`. Если файл не найден или не может быть декодирован, остается `None`.

### `doc_str`

**Описание**:
Строка, содержащая документацию проекта, загруженную из файла `README.MD`. Если файл не найден или не может быть прочитан, остается `None`.

### `__project_name__`

**Описание**:
Имя проекта. Берется из `config.json` или устанавливается значением по умолчанию `'hypotez'`.

### `__version__`

**Описание**:
Версия проекта. Берется из `config.json` или устанавливается пустым значением.

### `__doc__`

**Описание**:
Документация проекта, загруженная из `README.MD` или устанавливается пустым значением.

### `__details__`

**Описание**:
Строка с дополнительной информацией о проекте. По умолчанию пустая строка.

### `__author__`

**Описание**:
Автор проекта. Берется из `config.json` или устанавливается пустым значением.

### `__copyright__`

**Описание**:
Информация об авторских правах. Берется из `config.json` или устанавливается пустым значением.

### `__cofee__`

**Описание**:
Сообщение с просьбой угостить разработчика чашкой кофе. По умолчанию: "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69".
