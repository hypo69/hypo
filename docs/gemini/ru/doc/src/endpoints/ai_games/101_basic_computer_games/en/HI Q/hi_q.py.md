# HI Q

## Обзор

Данный модуль реализует игру-головоломку "HI Q", в которой игрок должен убрать все шарики с доски, перепрыгивая через них, оставив в конце только один.

## Функции

### `init_board`

**Описание**: Инициализирует игровое поле с начальным расположением шариков.

**Возвращает**:
- `list[list[int]]`: Список списков, представляющий доску, где 1 - шарик, 0 - пустое место.

### `print_board`

**Описание**: Выводит текущее состояние игрового поля на экран.

**Параметры**:
- `board` (list[list[int]]): Список списков, представляющий доску.

**Возвращает**:
- `None`

### `get_move`

**Описание**: Запрашивает у пользователя координаты шарика и направление хода.

**Возвращает**:
- `tuple[int, int, str]`: Кортеж, содержащий строку, столбец и направление хода (`u`, `d`, `l`, `r`).

### `is_valid_move`

**Описание**: Проверяет, является ли ход допустимым.

**Параметры**:
- `board` (list[list[int]]): Список списков, представляющий доску.
- `row` (int): Строка выбранного шарика.
- `col` (int): Столбец выбранного шарика.
- `direction` (str): Направление хода (`u`, `d`, `l`, `r`).

**Возвращает**:
- `bool`: True, если ход допустим, иначе False.

### `make_move`

**Описание**: Выполняет ход, обновляя состояние игрового поля.

**Параметры**:
- `board` (list[list[int]]): Список списков, представляющий доску.
- `row` (int): Строка выбранного шарика.
- `col` (int): Столбец выбранного шарика.
- `direction` (str): Направление хода (`u`, `d`, `l`, `r`).

**Возвращает**:
- `list[list[int]]`: Обновленный список списков, представляющий доску.

### `has_moves`

**Описание**: Проверяет, есть ли еще возможные ходы.

**Параметры**:
- `board` (list[list[int]]): Список списков, представляющий доску.

**Возвращает**:
- `bool`: True, если есть возможные ходы, иначе False.

### `count_pegs`

**Описание**: Считает количество шариков на доске.

**Параметры**:
- `board` (list[list[int]]): Список списков, представляющий доску.

**Возвращает**:
- `int`: Количество шариков на доске.

### `play_hi_q`

**Описание**: Основная функция игры HI Q. Содержит игровой цикл, управление ходами и проверку на победу/проигрыш.

**Возвращает**:
- `None`
