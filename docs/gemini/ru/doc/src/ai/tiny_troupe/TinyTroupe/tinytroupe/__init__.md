# Модуль tinytroupe/__init__.py

## Обзор

Этот модуль инициализирует TinyTroupe, включая чтение конфигурационного файла, настройку логгера и обработку форматирования вывода. Он также содержит предупреждения об использовании AI.

## Импорты

Модуль импортирует следующие библиотеки:

- `os`: для взаимодействия с операционной системой.
- `logging`: для работы с логгированием.
- `configparser`: для парсинга конфигурационных файлов.
- `rich`: для красивого вывода в консоль.
- `rich.jupyter`: для поддержки Jupyter Notebook.
- `sys`: для управления системными переменными.
- `tinytroupe.utils`: для использования утилит.

## Функции

### `read_config_file`

**Описание**: Читает конфигурационный файл и возвращает объект `configparser.ConfigParser`.

**Параметры**:
- Нет

**Возвращает**:
- `configparser.ConfigParser`: Объект, содержащий конфигурацию.


### `pretty_print_config`

**Описание**: Выводит конфигурацию в красивом формате с помощью `rich`.

**Параметры**:
- `config (configparser.ConfigParser)`: Конфигурационный объект.

**Возвращает**:
- Нет

### `start_logger`

**Описание**: Инициализирует и настраивает логгер на основе предоставленной конфигурации.

**Параметры**:
- `config (configparser.ConfigParser)`: Конфигурационный объект.

**Возвращает**:
- Нет


### `inject_html_css_style_prefix`

**Описание**: Вставляет префикс CSS стилей для удаления отступов в Jupyter Notebook.

**Параметры**:
- `html_format (str)`: Строка HTML форматирования.
- `style_prefix (str)`: Префикс CSS стилей.


**Возвращает**:
- `str`: Измененная строка HTML с добавленным префиксом.
