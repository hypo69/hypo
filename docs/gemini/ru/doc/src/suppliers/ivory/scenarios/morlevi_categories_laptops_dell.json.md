# Документация для `morlevi_categories_laptops_dell.json`

## Обзор

Этот файл содержит JSON-конфигурацию сценариев для определения категорий товаров (ноутбуков) бренда DELL на сайте Morlevi. Сценарии определяют соответствие между различными моделями ноутбуков DELL (с разными процессорами и размерами экрана) и категориями PrestaShop, в которые эти товары должны быть отнесены.

## Структура JSON

JSON-файл имеет корневой объект с единственным ключом `"scenarios"`, значение которого - это объект, содержащий сценарии для различных моделей ноутбуков DELL.

```json
{
  "scenarios": {
    "SCENARIO_NAME_1": {
      "brand": "DELL",
      "url": null,
      "checkbox": false,
      "active": true,
      "condition": "new",
      "presta_categories": {
        "template": {
          "dell": [ "CATEGORY_1", "SIZE_1" ]
        }
      }
    },
    "SCENARIO_NAME_2": {
       "brand": "DELL",
      "url": "URL",
      "checkbox": false,
      "active": true,
      "condition": "new",
      "presta_categories": {
        "template": {
          "dell": [ "CATEGORY_2", "SIZE_2" ]
        }
      }
    },
    "...": {}
  }
}
```

## Сценарии

Каждый ключ в объекте `"scenarios"` представляет собой имя сценария, соответствующее определенной модели ноутбука DELL. Значение каждого сценария - это объект, содержащий настройки для определения категории товара.

### Примеры сценариев

- **`DELL 11.6 I3`**:  Ноутбук DELL с диагональю 11.6 дюймов и процессором Intel Core i3.
- **`DELL 15 I5`**: Ноутбук DELL с диагональю 15 дюймов и процессором Intel Core i5.
- **`DELL 17.3 AMD`**: Ноутбук DELL с диагональю 17.3 дюйма и процессором AMD.

## Ключи сценария

Каждый сценарий представлен объектом со следующими ключами:

-   `brand` (str): Всегда `"DELL"`, обозначает бренд ноутбука.
-   `url` (str | null): URL-адрес, используемый для фильтрации товаров на сайте Morlevi, или `null` если таковой отсутствует.
-   `checkbox` (bool): Всегда `false`. Возможно, используется в интерфейсе, но не имеет значения для логики.
-   `active` (bool): Всегда `true`, указывает, что сценарий активен.
-   `condition` (str): Всегда `"new"`, указывает, что товар новый.
-   `presta_categories` (dict): Объект, определяющий категории PrestaShop.

## `presta_categories`

Объект `presta_categories` содержит вложенный объект `template`, в котором находится список категорий для бренда `dell`.

-   `template` (dict): Объект, содержащий список категорий.
    -   `dell` (list): Список из двух элементов, первый указывает на название категории PrestaShop (например, `LAPTOPS INTEL I3`), второй - на размер экрана (например, `"11"` для 11-дюймовых ноутбуков).

## Заключение

Этот JSON-файл обеспечивает структурированное сопоставление моделей ноутбуков DELL с их категориями в PrestaShop, используя параметры процессора и размера экрана. Файл используется для автоматизации категоризации товаров на сайте Morlevi.
