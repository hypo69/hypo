# Модуль `hypotez/src/utils/cursor_spinner.py`

## Обзор

Этот модуль предоставляет утилиту для отображения вращающегося курсора в консоли, чтобы имитировать процесс загрузки или ожидания.

## Оглавление

- [Модуль `cursor_spinner`](#модуль-cursor_spinner)
    - [Функция `spinning_cursor`](#функция-spinning_cursor)
    - [Функция `show_spinner`](#функция-show_spinner)


## Функции

### `spinning_cursor`

**Описание**: Генератор для вращающегося курсора, который циклически меняет символы |, /, -, \\.

**Возвращает**:
- `str`: Следующий символ в последовательности курсора.


### `show_spinner`

**Описание**: Отображает вращающийся курсор в консоли на заданную продолжительность.

**Параметры**:
- `duration` (float): Время работы спиннера (в секундах). По умолчанию 5.0.
- `delay` (float): Задержка между каждой сменой символа (в секундах). По умолчанию 0.1.


**Пример использования**:
```python
show_spinner(duration=3.0, delay=0.2) # Отображает спиннер на 3 секунды
```

**Вызывает исключения**:
- Не вызывает никаких исключений.
