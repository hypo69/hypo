# Модуль `facebook_fields.py`

## Обзор

Модуль `facebook_fields.py` предназначен для загрузки и хранения полей, используемых в Facebook для объявлений и событий. Он загружает данные из JSON-файла и динамически добавляет их как атрибуты к экземпляру класса `FacebookFields`.

## Классы

### `FacebookFields`

**Описание**:
Класс `FacebookFields` предназначен для хранения полей, используемых в Facebook для объявлений и событий. Он загружает данные из JSON-файла и динамически добавляет их как атрибуты к экземпляру класса.

#### `__init__`
    
**Описание**:
Конструктор класса `FacebookFields`. Инициализирует экземпляр класса и вызывает метод `_payload` для загрузки данных.

**Параметры**:
   - Нет параметров.
    
#### `_payload`

**Описание**:
Метод `_payload` загружает данные из JSON-файла `facebook_feilds.json` и устанавливает их как атрибуты экземпляра класса `FacebookFields`.

**Параметры**:
   - Нет параметров.

**Возвращает**:
   - `True` в случае успешной загрузки данных, иначе возвращает `None`.
