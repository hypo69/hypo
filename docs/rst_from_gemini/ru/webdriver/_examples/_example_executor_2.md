# Файл `hypotez/src/webdriver/_examples/_example_executor_2.py`

Этот файл содержит примеры использования класса `ExecuteLocator` для различных сценариев тестирования веб-приложений с помощью Selenium.  Он демонстрирует, как выполнять различные действия с элементами веб-страницы, включая поиск элементов, отправку сообщений, клики и получение атрибутов.

## Функциональность

Файл демонстрирует следующие возможности `ExecuteLocator`:

* **Поиск элементов:**  Использование XPath для поиска различных элементов страницы.  Пример показывает поиск заголовка, ссылок на продукты, элементов пагинации.
* **Получение атрибутов:**  Извлечение значений атрибутов элементов (например, `textContent`, `href`).
* **Обработка событий:** Выполнение действий с найденными элементами, таких как клик (`event: "click()"`), ввод текста (`event: "%SEARCH%"`).
* **Обработка списков:**  Работа со списками элементов, получение первого или всех элементов (`if_list: "first"`).
* **Обработка ошибок:**  Использование `continue_on_error=True` для продолжения выполнения при возникновении ошибок.
* **Отправка сообщений:**  Метод `send_message` позволяет отправлять текст в текстовые поля с заданной скоростью набора.
* **Работа с несколькими локаторами:** Пример `multi_locator` демонстрирует как выполнять несколько локаторов одновременно.
* **Оценка локатора:** Использование метода `evaluate_locator` для получения значений атрибутов.
* **Обработка исключений:**  Демонстрация блока `try-except` для перехвата и обработки исключений `ExecuteLocatorException`.


## Примеры

Файл предоставляет разнообразные примеры:

* **Простой локатор:**  Демонстрация базового поиска и получения значения.
* **Локатор с различными событиями и атрибутами:**  Пример использования комбинаций событий и атрибутов.
* **Обработка ошибок и продолжение на ошибки:** Демонстрация как продолжить выполнение, даже если один из локаторов завершился ошибкой.
* **Отправка сообщений:** Показ метода `send_message` для ввода текста.
* **Локатор с множественными локаторами:** Работа с несколькими локаторами одновременно.
* **Использование `evaluate_locator`:** Получение значений атрибутов.
* **Обработка исключений:** Обработка возможных ошибок при выполнении локаторов.
* **Полный пример теста:** Использование всех ключевых функций `ExecuteLocator` в одном сценарии.


## Структура кода

Код организован в виде понятных блоков примеров, что облегчает понимание и использование различных возможностей `ExecuteLocator`.  Каждый пример сопровождается комментариями, описывающими его назначение и ожидаемое поведение.

## Важные замечания

*  Для запуска примеров требуется установить Selenium и драйвер браузера (в данном случае ChromeDriver).
*  Переменная `gs['chrome_driver_path']` должна быть корректно задана, указывая путь к исполняемому файлу ChromeDriver.
*  Использование `"%SEARCH%"` в качестве `event` для `send_message` предполагает, что в `ExecuteLocator` есть обработчик для подобных макросов.  Это необходимо для правильной работы примеров.

Этот файл является отличным руководством для понимания и использования `ExecuteLocator` в задачах веб-автоматизации.
