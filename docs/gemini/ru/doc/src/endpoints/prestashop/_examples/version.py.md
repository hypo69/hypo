# Модуль `version.py`

## Обзор

Модуль содержит информацию о версии, имени, документации, деталях, аннотациях и авторе текущего модуля.

## Переменные

### `__name__`

**Описание**:
Содержит имя модуля. Если скрипт запускается напрямую, значение будет `"__main__"`.

**Тип**:
`str`

### `__version__`

**Описание**:
Содержит версию модуля или пакета.

**Тип**:
`str`

**Значение**:
`"3.12.0.0.0.4"`

### `__doc__`

**Описание**:
Содержит строку документации модуля.

**Тип**:
`str`

### `__details__`

**Описание**:
Содержит дополнительные детали о модуле, но конкретное назначение зависит от модуля.

**Тип**:
`str`

**Значение**:
`"Details about version for module or class"`

### `__annotations__`

**Описание**:
Содержит аннотации типов для переменных и функций в модуле.

**Тип**:
`dict`

### `__author__`

**Описание**:
Содержит имя(имена) автора(ов) модуля.

**Тип**:
`str`

**Значение**:
`'hypotez'`
