# Поддержка

## Как сообщать о проблемах и получать помощь

В этом проекте для отслеживания ошибок и запросов на добавление новых функций используются GitHub Issues. Пожалуйста, выполните поиск по существующим проблемам, прежде чем создавать новые, чтобы избежать дубликатов. Для новых проблем, пожалуйста, сообщите об ошибке или запросе на добавление новой функции в виде новой Issue.

Для получения помощи и ответов на вопросы об использовании этого проекта, пожалуйста, обратитесь к нашей Wiki или разместите сообщение на нашем форуме обсуждений, оба из которых находятся здесь, на GitHub.
