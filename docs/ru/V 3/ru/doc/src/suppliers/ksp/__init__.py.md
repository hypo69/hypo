# Модуль `src.suppliers.ksp`

## Обзор

Модуль `src.suppliers.ksp` предназначен для организации и управления поставщиками данных, в частности, для работы с данными, получаемыми от KSP. Модуль содержит классы и функции, необходимые для извлечения и обработки данных из различных источников, связанных с KSP.

## Подробнее

Этот модуль предоставляет интерфейс для работы с грабером (Graber), который используется для сбора данных. Он импортирует класс `Graber` из подмодуля `.graber`, который, вероятно, содержит логику для взаимодействия с источниками данных KSP. Модуль служит точкой входа для функциональности, связанной с KSP, и обеспечивает абстракцию для упрощения использования грабера в других частях проекта.
