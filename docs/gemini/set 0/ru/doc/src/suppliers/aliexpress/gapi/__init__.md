# Модуль `hypotez/src/suppliers/aliexpress/gapi/__init__.py`

Данный модуль `hypotez/src/suppliers/aliexpress/gapi/__init__.py` служит для инициализации и импорта модулей, относящихся к API AliExpress. Он содержит константу `MODE` и импортирует информацию о версии из файла `version.py`.

**Описание**:  Переменная, хранящая режим работы (например, `'dev'` для разработки, `'prod'` для производства).  В данном случае, установлено значение `'dev'`.
