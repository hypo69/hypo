# Модуль `src.ai`

## Обзор

Модуль `src.ai` предоставляет интерфейс для взаимодействия с различными моделями искусственного интеллекта.
Он включает в себя классы для работы с моделями Google Gemini и OpenAI.

    - [GoogleGenerativeAI](#googlegenerativeai)

## Константы

### `MODE`

- **Описание**: Режим работы модуля.
- **Тип**: `str`
- **Значение по умолчанию**: `'dev'`

## Импортируемые модули

В модуле импортируются следующие классы:

-  `GoogleGenerativeAI` из `hypotez.src.ai.gemini`
-  `OpenAIModel` из `hypotez.src.ai.openai`

## Классы

### `GoogleGenerativeAI`

- **Описание**: Класс для работы с моделями Google Gemini.
-  **Расположение**: `hypotez/src/ai/gemini.py`
   (Подробная документация находится в файле `gemini.md`)

### `OpenAIModel`

- **Описание**: Класс для работы с моделями OpenAI.
- **Расположение**: `hypotez/src/ai/openai.py`
   (Подробная документация находится в файле `openai.md`)
