# Обновление заголовков файлов в проекте "hypotez"

Этот скрипт предназначен для обработки Python файлов в проекте "hypotez". Он добавляет или заменяет заголовки, строки интерпретаторов и строки документации в исходных кодах. Скрипт обходит все файлы в проекте и обновляет их, добавляя информацию о проекте, интерпретаторе и метаданных.

## Описание

Скрипт выполняет следующие задачи:

- Идентифицирует корневую папку проекта.
- Находит и добавляет строку кодировки.
- Добавляет строки с интерпретатором для Windows и Linux.
- Добавляет строку документации для модуля.
- Устанавливает значение режима работы проекта.

## Примеры запуска

### Стандартный запуск

Запускает обработку всех файлов проекта:

```bash
python update_files_headers.py
```

### Принудительное обновление файлов

Принудительно обновляет файлы, даже если заголовки уже присутствуют:

```bash
python update_files_headers.py --force-update
```

### Указание конкретной директории проекта

Обрабатывает файлы в указанной директории проекта:

```bash
python update_files_headers.py -p /path/to/project
```

### Исключение директории "venv"

Исключает директорию `venv` из обработки:

```bash
python update_files_headers.py --exclude-venv
```

## Логика работы скрипта

1. **Поиск корня проекта**  
   Скрипт ищет корень проекта, начиная с текущей директории. Если корень проекта не найден, выбрасывается ошибка.

2. **Добавление/замена заголовков**  
   Для каждого Python файла скрипт:
   - Добавляет строку кодировки `# -*- coding: utf-8 -*-`, если она отсутствует.
   - Добавляет строки с интерпретатором для Windows и Linux (в зависимости от операционной системы).
   - Добавляет строку документации модуля, если она отсутствует.
   - Устанавливает значение переменной `MODE`, если оно отсутствует.

3. **Режимы обновления**  
   - В обычном режиме обновления (`force-update` не установлен) заголовки добавляются только в случае их отсутствия.
   - В режиме `--force-update` заголовки и строки интерпретаторов обновляются даже если они уже присутствуют в файле.

4. **Исключение папок**  
   Папки, такие как `venv`, `tmp`, `docs`, `data` и другие указанные в переменной `EXCLUDE_DIRS`, исключаются из обработки.

## Зависимости

Скрипт использует стандартные библиотеки Python:

- `os`
- `argparse`
- `pathlib`
- `sys`
- `platform`

Убедитесь, что у вас установлен Python и зависимости, если это необходимо для вашего окружения.
