# Модуль `header.py`

## Обзор

Модуль `header.py` содержит основные настройки и импорты, необходимые для работы скриптов в рамках проекта `hypotez`. В частности, он определяет переменную `MODE` и добавляет корневую директорию проекта в `sys.path` для обеспечения доступа к модулям.

## Константы

### `MODE`

**Описание**: Режим работы скрипта. По умолчанию установлен в `'dev'`.

**Тип**: `str`

## Импорты

Модуль импортирует следующие библиотеки:

- `sys`: для работы с системными параметрами и функциями.
- `os`: для работы с операционной системой, включая пути к файлам.
- `pathlib.Path`: для удобной работы с путями к файлам и директориям.

## Настройка `sys.path`

В данном блоке кода выполняется добавление корневой директории проекта `hypotez` в список путей поиска модулей (`sys.path`). Это позволяет импортировать модули из проекта, даже если они не находятся в стандартных путях.

### `__root__`

**Описание**: Абсолютный путь к корневой директории проекта `hypotez`.

**Тип**: `pathlib.Path`

**Детали**:
  - Определяется путем поиска в текущей рабочей директории подстроки `hypotez`.
  - Обрезается текущая рабочая директория до найденной подстроки с учетом ее длины.

### `sys.path.append(__root__)`

**Описание**: Добавляет абсолютный путь к корневой директории проекта в `sys.path`.

**Детали**:
  - Позволяет импортировать модули из проекта без необходимости указывать их полные пути.
