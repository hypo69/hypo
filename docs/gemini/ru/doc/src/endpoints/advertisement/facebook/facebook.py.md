# Модуль рекламы на Facebook

## Обзор

Модуль `facebook.py` предназначен для автоматизации действий в социальной сети Facebook через веб-драйвер. Он включает в себя функции для логина, публикации сообщений, загрузки медиафайлов и продвижения публикаций. Модуль поддерживает работу как в среде Windows, так и в Unix.

## Классы

### `Facebook`

**Описание**: Класс для взаимодействия с Facebook через веб-драйвер.

**Параметры**:
-   `driver` (`Driver`): Экземпляр веб-драйвера.
-   `promoter` (`str`): Имя пользователя или идентификатор промоутера.
-   `group_file_paths` (`list[str]`): Список путей к файлам групп.

**Атрибуты**:
-   `d` (`Driver`): Экземпляр веб-драйвера.
-   `start_page` (`str`): URL начальной страницы Facebook.
-   `promoter` (`str`): Имя пользователя или идентификатор промоутера.

## Функции

### `login`

**Описание**: Функция выполняет вход в аккаунт Facebook.

**Параметры**:
    -   Нет

**Возвращает**:
    -   `bool`: `True`, если вход выполнен успешно, иначе `False`.

### `promote_post`

**Описание**: Функция отправляет сообщение в форму публикации и продвигает пост.

**Параметры**:
-   `item` (`SimpleNamespace`): Объект с данными для публикации.

**Возвращает**:
-   `bool`: `True`, если публикация выполнена успешно, иначе `False`.

### `promote_event`

**Описание**: Функция для продвижения события.

**Параметры**:
-   `event` (`SimpleNamespace`): Объект с данными о событии.
