* **Интеграция системного трея:** Окно приложения можно минимизировать в системный трей, сохраняя его работоспособность в фоновом режиме.  Пользователь может вернуть окно из трея.

* **Автоматическая установка "http" префикса:** Если пользователь вводит URL без префикса "http", приложение автоматически добавляет его.
