# Модуль goog.header

## Обзор

Этот модуль предоставляет функции для определения корневого каталога проекта, получения настроек из файла `settings.json` и извлечения документации из файла `README.MD`.

## Функции

### `get_project_root`

**Описание**: Находит корневой каталог проекта, начиная с текущей директории и идя вверх по иерархии директорий.  Останавливается на первой директории, содержащей один из указанных файлов-маркеров.

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для определения корневого каталога проекта.

**Возвращает**:
- `Path`: Путь к корневому каталогу проекта, если найден, иначе - путь к текущей директории.

**Вызывает исключения**:
- Нет


### `get_project_root`

**Описание**: Находит корневой каталог проекта, начиная с текущей директории и идя вверх по иерархии директорий.  Останавливается на первой директории, содержащей один из указанных файлов-маркеров.

**Параметры**:
- `marker_files` (tuple, optional): Кортеж имен файлов или директорий, используемых для определения корневого каталога проекта. По умолчанию используется кортеж `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневому каталогу проекта.

**Вызывает исключения**:
- Нет


## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта, полученный с помощью функции `get_project_root`.

**Тип**: `Path`

### `settings`

**Описание**: Словарь настроек, загруженный из файла `src/settings.json`.

**Тип**: `dict` или `None`

### `doc_str`

**Описание**: Строка с документацией, загруженная из файла `src/README.MD`.

**Тип**: `str` или `None`


### `__project_name__`

**Описание**: Название проекта. Получается из настроек из `settings.json`. По умолчанию  `hypotez`.


**Тип**: `str`

### `__version__`

**Описание**: Версия проекта. Получается из настроек из `settings.json`. По умолчанию пустая строка.

**Тип**: `str`


### `__doc__`

**Описание**: Документация проекта. Получается из файла `src/README.MD`. По умолчанию пустая строка.

**Тип**: `str`


### `__details__`

**Описание**: Дополнительные детали проекта. Пустая строка по умолчанию.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта. Получается из настроек из `settings.json`. По умолчанию пустая строка.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права на проект. Получается из настроек из `settings.json`. По умолчанию пустая строка.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка на возможность поддержать разработчика. Получается из настроек из `settings.json`. По умолчанию ссылка на Boosty.

**Тип**: `str`
