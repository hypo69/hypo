# Задача No 859
## Сумма простых множителей

## Условия

Пусть s(n) — сумма простых множителей n (с повторениями). Например, s(10) = 2 + 5 = 7 и s(100) = 2 + 2 + 5 + 5 = 14.

Пусть S(N) = ∑ s(n) для 2 ≤ n ≤ N. Например, S(10) = 7 + 5 + 11 + 7 + 15 + 11 + 17 + 7 + 19 = 110.

Пусть f(n) — количество натуральных чисел m таких, что S(m) = n. Например, f(110) = 1.

Пусть F(N) = ∑ f(n) для 2 ≤ n ≤ N. Например, F(1000) = 178.

Найдите F(10<sup>7</sup>).
