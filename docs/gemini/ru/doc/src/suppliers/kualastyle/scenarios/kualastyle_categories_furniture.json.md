# Документация для `kualastyle_categories_furniture.json`

## Обзор

Этот JSON-файл определяет категорию "ריהיטים" (мебель) для сайта Kualastyle. Он указывает, что эта категория имеет подкатегории и не имеет связанных сценариев.

- [Обзор](#обзор)
- [Структура JSON](#структура-json)
- [Описание полей](#описание-полей)

## Структура JSON

```json
{
  "category name on site": "רהיטים",
  "have subcategories": true,
  "scenarios": {}
}
```

## Описание полей

### `category name on site`

- **Описание**: Название категории на сайте.
- **Тип**: Строка.
- **Значение**: `"רהיטים"` (мебель).

### `have subcategories`

- **Описание**: Указывает, есть ли у данной категории подкатегории.
- **Тип**: Логическое значение (`boolean`).
- **Значение**: `true` (у категории есть подкатегории).

### `scenarios`

- **Описание**: Объект, содержащий сценарии, связанные с категорией.
- **Тип**: Объект (`object`).
- **Значение**: `{}` (в данном случае, сценарии отсутствуют).
