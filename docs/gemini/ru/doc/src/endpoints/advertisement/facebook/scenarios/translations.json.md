# Документация для `translations.json`

## Обзор

Этот файл `translations.json` содержит JSON-объект со словарем для различных языков (английский, русский и иврит), используемых в приложении для локализации текста. Он включает настройки направления текста, копирайт, а также переводы для различных терминов, связанных с ценами, скидками и продуктами.

## Структура JSON

### `LOCALE`

**Описание**: Объект, определяющий направление текста для каждого языка.

- `"EN"` (str): Направление текста слева направо (LTR) для английского языка.
- `"RU"` (str): Направление текста слева направо (LTR) для русского языка.
- `"HE"` (str): Направление текста справа налево (RTL) для иврита.

### `COPYRIGHT`

**Описание**: Объект, содержащий текст копирайта для каждого языка.

- `"EN"` (str): Текст копирайта на английском языке.
- `"RU"` (str): Текст копирайта на русском языке.
- `"HE"` (str): Текст копирайта на иврите.

### `original_price`

**Описание**: Объект, содержащий перевод термина "Цена" для каждого языка.

- `"EN"` (str): "Price" на английском языке.
- `"RU"` (str): "Цена" на русском языке.
- `"HE"` (str): "מחיר " на иврите.

### `sale_price`

**Описание**: Объект, содержащий перевод термина "Цена сегодня" для каждого языка.

- `"EN"` (str): "Sale Price" на английском языке.
- `"RU"` (str): "Цена сегодня" на русском языке.
- `"HE"` (str): " מחיר מבצע" на иврите.

### `discount`

**Описание**: Объект, содержащий перевод термина "Скидка" для каждого языка.

- `"EN"` (str): "Discount" на английском языке.
- `"RU"` (str): "Скидка" на русском языке.
- `"HE"` (str): "מבצע " на иврите.

### `promotion_link`

**Описание**: Объект, содержащий перевод термина "Ссылка на товар" для каждого языка.

- `"EN"` (str): "Link to Shop" на английском языке.
- `"RU"` (str): "Ссылка на товар" на русском языке.
- `"HE"` (str): "לינק " на иврите.

### `tags`

**Описание**: Объект, содержащий перевод термина "хэштеги" для каждого языка.

- `"EN"` (str): "tags" на английском языке.
- `"RU"` (str): "хэштеги" на русском языке.
- `"HE"` (str): " תגים" на иврите.

### `evaluate_rate`

**Описание**: Объект, содержащий перевод термина "Рейтинг товара" для каждого языка.
