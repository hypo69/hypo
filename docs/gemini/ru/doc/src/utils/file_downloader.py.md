# Модуль `file_downloader`

## Обзор

Модуль `file_downloader` предоставляет функциональность для скачивания файлов по указанному URL и сохранения их на диск.

## Содержание

- [Функции](#функции)
    - [`download_file`](#download_file)

## Функции

### `download_file`

**Описание**: Функция для скачивания файла по указанному URL и сохранения его на диск.

**Параметры**:
- `url` (str): URL файла для скачивания.
- `destination` (str): Путь и имя файла для сохранения.

**Возвращает**:
- `None`: Функция ничего не возвращает. Вместо этого она сохраняет файл на диск.

**Вызывает исключения**:
- `None`: Функция не вызывает исключений.
