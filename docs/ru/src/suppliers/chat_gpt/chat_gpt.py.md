# Модуль `chat_gpt`

## Обзор

Модуль `chat_gpt` предназначен для работы с моделью ChatGpt. В настоящее время содержит класс `ChatGpt`, который позволяет извлекать HTML-файлы бесед из определенной директории.

## Подробнее

Этот модуль предоставляет функциональность для обработки и извлечения данных из HTML-файлов, сгенерированных ChatGpt. Он использует библиотеку `pathlib` для работы с путями к файлам и директориям, а также модуль `gs` для доступа к глобальным настройкам путей. Расположение файла в структуре проекта `hypotez` указывает на его роль в качестве поставщика (supplier) данных, в данном случае, от ChatGpt.

## Классы

### `ChatGpt`

**Описание**: Класс `ChatGpt` предназначен для взаимодействия с данными, полученными из ChatGpt, в частности, для извлечения HTML-файлов бесед.

**Как работает класс**:
Класс содержит метод `yeld_conversations_htmls`, который находит все HTML-файлы в директории `conversations` и возвращает их содержимое. Класс использует глобальные настройки (`gs`) для определения пути к директории с данными ChatGpt.

**Методы**:
- `yeld_conversations_htmls`: Извлекает HTML-файлы бесед из директории.
