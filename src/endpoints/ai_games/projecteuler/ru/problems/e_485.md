# Задача No 485
## Последовательность Максимальных Делителей

## Условия

Пусть *d(n)* обозначает количество делителей числа *n*.

Пусть *M(n, k)* обозначает наибольшее значение *d(j)* для *n* ≤ *j* ≤ *n* + *k* - 1.

Например, если *k* = 3:
*M*(1, 3) = 3, потому что *d*(1) = 1, *d*(2) = 2, *d*(3) = 2.
*M*(2, 3) = 3, потому что *d*(2) = 2, *d*(3) = 2, *d*(4) = 3.
*M*(3, 3) = 3, потому что *d*(3) = 2, *d*(4) = 3, *d*(5) = 2.
*M*(4, 3) = 3, потому что *d*(4) = 3, *d*(5) = 2, *d*(6) = 4.

Пусть *S(n, k)* обозначает сумму *M(i, k)* для 1 ≤ *i* ≤ *n*.

Например, *S*(10, 3) = 59.

Дано, что *S*(10<sup>4</sup>, 10) = 16736.

Найдите *S*(10<sup>8</sup>, 10<sup>5</sup>).
