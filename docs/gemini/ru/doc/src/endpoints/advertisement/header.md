# Модуль hypotez/src/endpoints/advertisement/header.py

## Обзор

Данный модуль содержит функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и получения документации из файла `README.MD`.  Модуль предоставляет переменные, содержащие информацию о проекте (название, версия, описание, автор и т.д.).

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с текущего файла и идя вверх по дереву каталогов, пока не найдет директорию, содержащую один из указанных маркеров файлов (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или каталогов, используемых для определения корневой директории. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта. Если корневая директория не найдена, возвращается директория, содержащая текущий файл.

**Вызывает исключения**:

- Нет


### `set_project_root`

**Описание**: Находит корневую директорию проекта.

**Параметры**:

- `marker_files`:  Кортеж файлов/каталогов, которые указывают на корень проекта.


**Возвращает**:

- `Path`: Путь к корневой директории. Если корень не найден, возвращает путь к текущей директории.


**Вызывает исключения**:

- Нет


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root()`.

**Тип**: `Path`


##  Загрузка настроек

**Описание**: Загружает настройки из файла `settings.json`, расположенного в корневой директории проекта.

**Обработка ошибок**:

- В случае `FileNotFoundError` или `json.JSONDecodeError`, переменная `settings` остается `None`.


## Чтение документации

**Описание**: Читает содержимое файла `README.MD` из корневой директории проекта и сохраняет его в переменной `doc_str`.

**Обработка ошибок**:

- В случае `FileNotFoundError` или `json.JSONDecodeError`, переменная `doc_str` остается `None`.

##  Переменные проекта

### `__project_name__`

**Описание**: Название проекта.  Получается из `settings.json` или имеет значение по умолчанию `hypotez`.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта. Получается из `settings.json` или имеет значение по умолчанию пустая строка.

**Тип**: `str`

### `__doc__`

**Описание**: Документация из `README.MD`.  Получается из `README.MD` или имеет значение по умолчанию пустая строка.

**Тип**: `str`


### `__details__`

**Описание**:  Дополнительные детали проекта. По умолчанию пустая строка.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта. Получается из `settings.json` или имеет значение по умолчанию пустая строка.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права. Получается из `settings.json` или имеет значение по умолчанию пустая строка.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка для поддержки разработчика. Получается из `settings.json` или имеет значение по умолчанию ссылка.

**Тип**: `str`
