# Модуль `category`

## Обзор

Модуль `category` содержит определения классов для представления категорий товаров, полученных от AliExpress API. Он включает базовый класс `Category` и класс `ChildCategory`, наследующий от него и представляющий дочернюю категорию.

## Подробней

Этот модуль предназначен для работы с данными о категориях товаров, получаемыми из API AliExpress. Он определяет структуру данных для категорий, позволяя удобно хранить и обрабатывать информацию о них. Классы используются для типизации данных, возвращаемых API, что улучшает читаемость и поддерживаемость кода.

## Классы

### `Category`

**Описание**: Базовый класс для представления категории товара.

**Как работает класс**:
Класс `Category` содержит поля `category_id` и `category_name`, которые представляют идентификатор и название категории соответственно. Этот класс является базовым для других классов, представляющих категории.

**Методы**: Нет.

**Параметры**:
- `category_id` (int): Идентификатор категории.
- `category_name` (str): Название категории.

**Примеры**:

```python
category = Category()
category.category_id = 123
category.category_name = 'Электроника'
print(category.category_name)
```

### `ChildCategory`

**Описание**: Класс для представления дочерней категории товара, наследуемый от класса `Category`.

**Как работает класс**:
Класс `ChildCategory` наследует поля `category_id` и `category_name` от класса `Category` и добавляет поле `parent_category_id`, представляющее идентификатор родительской категории.

**Методы**: Нет.

**Параметры**:
- `category_id` (int): Идентификатор категории.
- `category_name` (str): Название категории.
- `parent_category_id` (int): Идентификатор родительской категории.

**Примеры**:

```python
child_category = ChildCategory()
child_category.category_id = 456
child_category.category_name = 'Смартфоны'
child_category.parent_category_id = 123
print(child_category.category_name)
```
