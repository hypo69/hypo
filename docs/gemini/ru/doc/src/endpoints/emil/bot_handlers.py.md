# Модуль `src.endpoints.emil.bot_handlers`

## Обзор

Модуль `src.endpoints.emil.bot_handlers` предназначен для обработки событий телеграм-бота `emil_bot`. Он обрабатывает команды, переданные боту, включая работу с ссылками OneTab и выполнение связанных сценариев.

## Классы

### `BotHandler`

**Описание**: Исполнитель команд, полученных ботом.

#### `__init__`

**Описание**: Инициализация обработчика событий телеграм-бота.

**Параметры**:
- `webdriver_name` (str): Название веб-драйвера для запуска.

## Функции

### `handle_message`

**Описание**: Обрабатывает текстовые сообщения, маршрутизируя их на основе наличия URL.

**Параметры**:
- `update` (Update): Объект обновления из телеграма.
- `context` (CallbackContext): Контекст выполнения.

**Возвращает**:
- `None`: Функция ничего не возвращает.

### `handle_url`

**Описание**: Обрабатывает URL, присланный пользователем.

**Параметры**:
- `update` (Update): Объект обновления из телеграма.
- `context` (CallbackContext): Контекст выполнения.

**Возвращает**:
- `Any`: Функция может возвращать данные любого типа.

### `handle_next_command`

**Описание**: Обрабатывает команду `--next` и ее аналоги, запрашивая случайный вопрос у модели и отправляя ответ.

**Параметры**:
- `update` (Update): Объект обновления из телеграма.

**Возвращает**:
- `None`: Функция ничего не возвращает.

**Вызывает исключения**:
- `Exception`: Обрабатывает исключения, возникающие при чтении вопросов.

### `fetch_target_urls_onetab`

**Описание**: Извлекает целевые URL из указанного URL OneTab.

**Параметры**:
- `one_tab_url` (str): URL страницы OneTab.

**Возвращает**:
- `list[str] | bool`: Список целевых URL или `False` в случае ошибки.

**Вызывает исключения**:
- `requests.exceptions.RequestException`: Возникает при ошибках во время выполнения HTTP-запроса.
