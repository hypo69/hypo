# Задача No 417
## Число делителей последовательных номеров
## Условия
Для натурального числа $n$ пусть $d(n)$ обозначает количество делителей $n$.
Например, $d(6) = 4$ так как 6 имеет 4 делителя: 1, 2, 3 и 6.

Пусть $S(n)$ обозначает сумму $d(k)$ для $k = 1$ до $n$.
Например, $S(10) = d(1) + d(2) + d(3) + d(4) + d(5) + d(6) + d(7) + d(8) + d(9) + d(10) = 1 + 2 + 2 + 3 + 2 + 4 + 2 + 4 + 3 + 4 = 27$.

Пусть $C(n)$ обозначает количество $k$ таких, что $1 \le k \le n$ и $d(k)$ является делителем $d(k+1)$.

Первые несколько значений $C(n)$ следующие:
$C(1) = 1$, $C(2) = 1$, $C(3) = 2$, $C(4) = 2$, $C(5) = 3$, $C(6) = 4$.

Найдите $C(100000000)$.
