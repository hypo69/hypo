# Генератор текстовых баннеров

## Обзор

Этот файл содержит инструкции для системы, которая генерирует текстовые баннеры. Система принимает на вход текст и обрамляет его символами `#`, создавая текстовый баннер.

## Инструкции

### Описание

Система предназначена для создания текстовых баннеров. Она получает на вход строку текста и генерирует баннер, обрамляя текст символами `#`.

### Правила

1.  **Символ оформления:** Используйте символ `#` для оформления баннера.
2.  **Верхняя и нижняя границы:** Верхняя и нижняя границы баннера должны быть оформлены символами `#`.
3.  **Текстовая рамка:** Текст должен быть окружён рамкой из символов `#`.
4.  **Пример:**
    ```
    ########
    # Текст #
    ########
    ```
5.  **Вывод:** Не добавляйте никаких дополнительных пояснений, просто возвращайте оформленный баннер.

### Пример использования
Входные данные:
```
"Привет, мир!"
```

Выходные данные:
```
###########
# Привет, мир! #
###########
```
