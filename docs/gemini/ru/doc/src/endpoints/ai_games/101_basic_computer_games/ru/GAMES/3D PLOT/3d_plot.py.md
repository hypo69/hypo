# 3D PLOT

## Обзор

Этот модуль реализует программу, которая строит трехмерный график функции Z = X^2 - Y^2. Пользователь вводит минимальные и максимальные значения для X и Y, а также шаг (приращение), с которым значения X и Y будут изменяться. Программа вычисляет значение Z для каждой пары (X, Y) и выводит координаты (X, Y, Z).

## Функции

### `main`

**Описание**: Основная функция программы, которая запрашивает у пользователя значения границ и шагов для X и Y, затем вычисляет и выводит значения Z в цикле.

**Параметры**:
- нет

**Возвращает**:
- нет

**Вызывает исключения**:
- `ValueError`: Если пользователь вводит нечисловые значения.

```python
def main():
    """
    Основная функция программы, которая запрашивает у пользователя значения границ и шагов для X и Y,
    затем вычисляет и выводит значения Z в цикле.

    Args:
        None

    Returns:
        None

    Raises:
        ValueError: Если пользователь вводит нечисловые значения.
    """
    try:
        Xmin = float(input("Введите минимальное значение X: "))
        Xmax = float(input("Введите максимальное значение X: "))
        Xincrement = float(input("Введите приращение для X: "))
        
        Ymin = float(input("Введите минимальное значение Y: "))
        Ymax = float(input("Введите максимальное значение Y: "))
        Yincrement = float(input("Введите приращение для Y: "))
    except ValueError as ex:
        print("Ошибка: Пожалуйста, введите числовые значения.")
        exit()

    X = Xmin
    while X <= Xmax:
        Y = Ymin
        while Y <= Ymax:
            Z = X * X - Y * Y
            print(f"X: {X}, Y: {Y}, Z: {Z}")
            Y += Yincrement
        X += Xincrement
```
