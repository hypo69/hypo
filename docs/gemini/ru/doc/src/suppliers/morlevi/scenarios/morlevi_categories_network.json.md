# Документация для `morlevi_categories_network.json`

## Обзор

Данный файл содержит JSON-конфигурацию сценариев для категорий сетевого оборудования Morlevi. Он определяет соответствия между URL-адресами категорий на сайте Morlevi и идентификаторами категорий PrestaShop, а также условия активации и фильтрации этих категорий.

## Структура JSON

### `scenarios`

Корневой объект, содержащий сценарии для различных категорий товаров.

#### `routers`

- **Описание**: Сценарий для категории "Маршрутизаторы".
- **Параметры**:
    - `url` (str): URL-адрес категории на сайте Morlevi.
    - `checkbox` (bool): Флаг, указывающий на наличие чекбокса (всегда `false`).
    - `active` (bool): Флаг, указывающий на активность сценария (всегда `true`).
    - `condition` (str): Условие для применения сценария (всегда `new`).
    - `presta_categories` (str): Строка, содержащая идентификаторы категорий PrestaShop, разделенные запятыми.
