# Модуль подготовки всех кампаний Aliexpress

## Обзор

Модуль `prepare_all_camapaigns.py` предназначен для подготовки рекламных кампаний на платформе Aliexpress. Он проверяет наличие существующих партнерских кампаний и создает новые, если таковые отсутствуют. Основная задача модуля - автоматизация процесса создания и подготовки рекламных кампаний для продвижения товаров с Aliexpress.

## Подробнее

Этот модуль выполняет запуск функции `process_all_campaigns` из модуля `src.suppliers.aliexpress.campaign`, которая отвечает за обработку всех кампаний. Импортируются необходимые библиотеки и модули, такие как `header` и `process_all_campaigns`.
