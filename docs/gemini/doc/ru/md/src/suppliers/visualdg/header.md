# Модуль hypotez/src/suppliers/visualdg/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`.  Он также инициализирует переменные, используемые для работы с проектом, такие как имя проекта, версия, документация, автор и другие метаданные.


## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла и поднимаясь вверх по иерархии директорий.

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, которые используются для идентификации корневой директории проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена, иначе — директория, в которой расположен текущий скрипт.

**Вызывает исключения**:

- Нет


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта.  Инициализируется функцией `set_project_root()`.

**Тип**: `Path`


## Загрузка настроек

### `settings`

**Описание**: Словарь настроек проекта, загруженный из файла `settings.json`.

**Тип**: `dict`

**Обработка ошибок**:

- Функция пытается загрузить данные из `settings.json` в директории проекта.
- Если файл не найден или содержит невалидные данные JSON, переменная `settings` остается `None`.



### `__project_name__`, `__version__`, `__doc__`, `__details__`, `__author__`, `__copyright__`, `__cofee__`

**Описание**: Переменные, содержащие значения, полученные из загруженных настроек, если они существуют; иначе — значения по умолчанию.


**Тип**: `str`

**Обработка ошибок**:

- Если файл настроек не существует или не содержит нужных ключей, значения этих переменных устанавливаются к значениям по умолчанию.

**Значения по умолчанию**:

- `__project_name__`: `'hypotez'`
- `__version__`: `''`
- `__doc__`: `''`
- `__details__`: `''`
- `__author__`: `''`
- `__copyright__`: `''`
- `__cofee__`: `Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69`
