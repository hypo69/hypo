# ИНСТРУКЦИЯ по установке ChromeDriver для тестирования

## Обзор

Данный документ описывает шаги по установке и настройке ChromeDriver для автоматизированного тестирования веб-приложений с использованием Selenium WebDriver.

## Оглавление

* [Установка ChromeDriver](#Установка-ChromeDriver)
* [Настройка среды](#Настройка-среды)
* [Проверка установки](#Проверка-установки)

## Установка ChromeDriver

1. **Скачивание:** Перейдите на официальный сайт проекта ChromeDriver [https://chromedriver.chromium.org/downloads](https://chromedriver.chromium.org/downloads).
2. **Выбор версии:** Выберите версию ChromeDriver, соответствующую вашей версии Chrome.  Важно! Выберите версию, точно соответствующую версии Chrome, которую вы используете для тестирования.  Несовпадение версий может привести к ошибкам.
3. **Загрузка:** Скачайте соответствующий исполняемый файл ChromeDriver.
4. **Размещение:** Сохраните скачанный файл в удобном для вас месте (например, в папке проекта или в отдельной папке для инструментов тестирования).

## Настройка среды

1. **Добавление в PATH:** Добавить путь к папке, где вы сохранили ChromeDriver, в переменную окружения PATH. Это позволит запускать ChromeDriver из любой точки вашей системы.  Подробные инструкции по настройке PATH зависят от вашей операционной системы.

## Проверка установки

1. **Открытие командной строки:** Откройте командную строку или терминал.
2. **Проверка версии:** Введите следующую команду, чтобы проверить установленный ChromeDriver:

```bash
chromedriver --version
```

Если установка прошла успешно, вы увидите номер версии ChromeDriver.

3. **Запуск в браузере:**  В браузере (Chrome) перейдите в настройки. Проверьте, что в разделе "Настройки" или "Дополнительные настройки" есть опция для запуска приложения в режиме разработчика, если таковая есть.

## Дополнительные рекомендации

* Для более удобной работы, рекомендуется сохранять ChromeDriver в папке, доступной из всех используемых проектов тестирования.
* Если у вас возникают сложности, обратитесь к документации ChromeDriver или к документации используемой тестовой среды.
* Если вы используете IDE, проверьте настройки запуска тестовых сценариев для корректного использования ChromeDriver.
