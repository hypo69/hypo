# Модуль hypotez/src/utils/path.py

## Обзор

Модуль `hypotez/src/utils/path.py` определяет корневой путь к проекту. Все импорты строятся относительно этого пути.

## Постоянные значения

### `MODE`

**Описание**:  Переменная, хранящая режим работы (`'dev'`).

## Функции

### `get_relative_path`

**Описание**: Возвращает часть пути начиная с указанного сегмента и до конца.

**Параметры**:
- `full_path` (str): Полный путь.
- `relative_from` (str): Сегмент пути, с которого нужно начать извлечение.

**Возвращает**:
- Optional[str]: Относительный путь начиная с `relative_from`, или `None`, если сегмент не найден.

**Подробное описание**:
Функция `get_relative_path` принимает полный путь (`full_path`) и сегмент пути (`relative_from`). Она преобразует входные строки в объекты `Path`. Затем ищет индекс `relative_from` в списке частей пути (`parts`). Если сегмент найден, функция формирует относительный путь, начиная с этого сегмента, используя `Path(*parts[start_index:])` и возвращает его в строковом формате с использованием `as_posix()`. В противном случае возвращает `None`.
