# Задача No 336
## Коробка плиток
## Условия

Представьте, что у вас есть набор прямоугольных плиток, каждая из которых имеет ширину 1 и высоту 2, а также набор квадратных плиток, каждая из которых имеет ширину 1 и высоту 1. Вы хотите покрыть этими плитками прямоугольную область шириной 2 и длиной *n*.

Пусть *T*(*n*) - количество различных способов покрытия прямоугольной области 2 x *n* с помощью этих плиток.

Например, *T*(1) = 1, *T*(2) = 3, *T*(3) = 5 и *T*(4) = 11.

Ниже представлены покрытия для *T*(3):

```
    ■■□
    ■■□
    
    ■□■
    ■□■
    
    □■■
    □■■
    
    ■■□
    □□■
    
    □■■
    ■□□
```

Обратите внимание, что повернутые варианты считаются различными.

Пусть *M*(*n*) - количество способов покрытия прямоугольной области 2 x *n* таким образом, чтобы использовалась как минимум одна прямоугольная плитка.

Очевидно, *M*(1) = 0, *M*(2) = 2, *M*(3) = 4 и *M*(4) = 10.

Пусть *S*(*n*) - сумма всех *M*(i) для 1 ≤ *i* ≤ *n*.

Известно, что *S*(5) = 40.

Найдите *S*(1 000 000) mod 100 000 007.
