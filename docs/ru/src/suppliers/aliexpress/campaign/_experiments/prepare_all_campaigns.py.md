# Модуль для подготовки рекламных кампаний AliExpress

## Обзор

Модуль `prepare_all_campaigns.py` предназначен для автоматической подготовки и обработки рекламных кампаний AliExpress. Он включает в себя функции для обработки всех кампаний на различных языках и для поиска названий категорий из директорий.

## Подробней

Этот модуль играет важную роль в проекте, так как автоматизирует процесс подготовки рекламных кампаний, что позволяет экономить время и ресурсы. Он использует другие модули, такие как `header` и `prepare_campaigns`, для выполнения своей задачи.
