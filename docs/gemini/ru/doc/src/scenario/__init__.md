# Модуль `hypotez/src/scenario/__init__.py`

## Обзор

Модуль `src.scenario` предоставляет функции для выполнения сценариев, предназначенных для поставщика (например, AliExpress). Он содержит функции для запуска сценариев, как по файлам, так и по списку словарей сценариев.

## Функции

### `run_scenario_files`

**Описание**: Функция для выполнения сценариев, заданных списком файлов.

**Параметры**:
- `supplier` (объект): Объект поставщика, для которого выполняются сценарии.
- `scenario_files` (список строк): Список путей к файлам со сценариями.

**Возвращает**:
- `None`: Функция не возвращает явных значений.

**Вызывает исключения**:
- `FileNotFoundError`: Если один из файлов в списке не найден.
- `ValueError`: Если в качестве аргумента `scenario_files` передан не список.
- `TypeError`: Если аргумент `supplier` не является объектом.


### `run_scenarios`

**Описание**: Функция для выполнения сценариев, заданных списком словарей.

**Параметры**:
- `supplier` (объект): Объект поставщика, для которого выполняются сценарии.
- `scenarios` (список словарей): Список словарей, представляющих сценарии.

**Возвращает**:
- `None`: Функция не возвращает явных значений.

**Вызывает исключения**:
- `TypeError`: Если аргумент `scenarios` не является списком или словари не имеют нужного формата.
- `TypeError`: Если аргумент `supplier` не является объектом.



### `run_scenario_file`

**Описание**: Функция для выполнения сценария из одного файла.

**Параметры**:
- `supplier` (объект): Объект поставщика, для которого выполняются сценарии.
- `scenario_file` (строка): Путь к файлу со сценарием.


**Возвращает**:
- `None`: Функция не возвращает явных значений.

**Вызывает исключения**:
- `FileNotFoundError`: Если файл не найден.
- `TypeError`: Если аргумент `supplier` не является объектом.
- `ValueError`: Если аргумент `scenario_file` не является строкой.


### `run_scenario`

**Описание**: Функция для выполнения одного сценария.

**Параметры**:
- `supplier` (объект): Объект поставщика, для которого выполняются сценарии.
- `scenario` (словарь): Словарь, представляющий сценарий.


**Возвращает**:
- `None`: Функция не возвращает явных значений.


**Вызывает исключения**:
- `TypeError`: Если аргумент `scenario` не является словарем.
- `TypeError`: Если аргумент `supplier` не является объектом.


### `execute_PrestaShop_insert`

**Описание**: Функция для выполнения операции вставки данных в PrestaShop.

**Параметры**:
- `data` (данные): Данные для вставки.


**Возвращает**:
- `None`: Функция не возвращает явных значений.


**Вызывает исключения**:
- `Exception`: Возможные исключения при работе с PrestaShop.


### `execute_PrestaShop_insert_async`

**Описание**: Функция для асинхронного выполнения операций вставки данных в PrestaShop.

**Параметры**:
- `data` (данные): Данные для вставки.


**Возвращает**:
- `None`: Функция не возвращает явных значений.


**Вызывает исключения**:
- `Exception`: Возможные исключения при работе с асинхронной обработкой PrestaShop.




## Константы

### `MODE`

**Описание**:  Переменная, определяющая режим работы (в данном случае 'dev').
