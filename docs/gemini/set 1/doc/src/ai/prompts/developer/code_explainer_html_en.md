# Документация к коду

## Обзор

Данный файл содержит функцию `add_numbers`, которая принимает два числа в качестве аргументов и возвращает их сумму, используя функцию `calculate_sum` из модуля `src.utils.calculator`.

## Функции

### `add_numbers`

**Описание**: Функция складывает два числа, используя функцию `calculate_sum`.

**Параметры**:

- `a` (number): Первое слагаемое.
- `b` (number): Второе слагаемое.


**Возвращает**:

- `number`: Сумма чисел `a` и `b`.

**Вызывает исключения**:

Возможные исключения, которые могут быть подняты из `calculate_sum`, если входные данные не являются числами.


```python
from src.utils.calculator import calculate_sum

def add_numbers(a: number, b: number) -> number:
    """
    Args:
        a (number): Первое слагаемое.
        b (number): Второе слагаемое.

    Returns:
        number: Сумма чисел a и b.

    Raises:
        TypeError: Если a или b не являются числами.
        Exception: Любое другое исключение, выброшенное функцией calculate_sum.
    """
    try:
        result = calculate_sum(a, b)
        return result
    excepts TypeError as e:
        raise TypeError(f"Ошибка: a или b не являются числами. Ошибка: {e}")
    except Exception as ex:
        raise Exception(f"Произошла ошибка при вычислении суммы: {ex}")
```

**Пример использования**:

```python
result = add_numbers(5, 3)
print(result)  # Вывод: 8
```
