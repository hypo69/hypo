# _example_driver.py

## Обзор

Данный модуль демонстрирует использование класса `Driver` с различными веб-браузерами (Chrome, Firefox, Edge). Он показывает, как инициализировать драйвер, переходить по URL, извлекать домен, прокручивать страницу, сохранять cookies и закрывать браузер.

## Функции

### `main`

**Описание**: Главная функция для демонстрации использования класса `Driver` с различными веб-браузерами.

**Параметры**:
- Нет параметров.

**Возвращает**:
- Нет возвращаемого значения.

**Вызывает исключения**:
- Нет явных исключений, но возможны исключения, связанные с работой веб-драйвера (например, `WebDriverException` если браузер не запускается, или `TimeoutException` если не получается загрузить страницу).
