# Модуль logger

## Обзор

Данный модуль предоставляет инструменты для логгирования событий телеграм-ботов.  Он конфигурируется в зависимости от значения переменной `MODE`.

## Переменные

### `MODE`

**Описание**: Строковая переменная, определяющая режим работы логгера.  Текущее значение: 'dev'.

## Функции

### `setup_logger`

**Описание**: Функция настраивает логгер в зависимости от значения переменной `MODE`.

**Параметры**:

-  Не принимает параметров.


**Возвращает**:

-  `logging.Logger`: Настроенный логгер.

**Вызывает исключения**:

- Нет.
