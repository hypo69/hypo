Templates Module
=================

.. module:: hypotez.src.templates.header
    :platform: Windows, Unix
    :synopsis: This module defines constants and paths related to the templates.

Module Description
------------------

This module defines a constant `MODE` and a path to the source directory. It also handles adding the source directory to the Python path, and imports the `gs` module from `hypotez.src.credentials`.
