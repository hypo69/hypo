# Модуль определения типов

## Обзор

Этот модуль определяет пользовательский тип `sha256` и импортирует необходимые типы из модуля `typing`.

## Подробней

Модуль служит для определения структуры данных, используемой в проекте. В частности, определяется тип `sha256` как псевдоним для строки, что позволяет улучшить читаемость и поддержку кода, явно указывая, что переменная должна содержать SHA256-хеш.

## Функции

### `NewType`

```python
sha256 = NewType('sha_256_hash', str)
```

**Назначение**: Создает новый пользовательский тип `sha256`, который является псевдонимом для типа `str`.

**Параметры**:
- `name` (str): Имя нового типа (`'sha_256_hash'`).
- `tp` (type): Тип, для которого создается псевдоним (`str`).

**Возвращает**:
- Новый тип данных `sha256`, представляющий собой строку.

**Как работает функция**:
1. Функция `NewType` вызывается с именем нового типа и базовым типом.
2. Создается новый тип, который ведет себя как базовый тип, но имеет другое имя для целей типизации.

**Примеры**:
```python
from typing import NewType

sha256 = NewType('sha_256_hash', str)

my_hash: sha256 = 'a948904f2f0d479d8f81a89448ee94f74585bca98d9f880e2974c92d8a8f965f'
print(type(my_hash))  # Output: <class 'str'>
print(sha256) # <class 'typing.NewType.<locals>.new_type'>
```
```python
from typing import NewType

sha256 = NewType('sha_256_hash', str)
bad_hash = 123
print(type(bad_hash))
```
## Импортированные типы

- `Dict`: Тип для словарей.
- `NewType`: Функция для создания новых типов.
- `Union`: Тип для объединения нескольких типов.
- `Optional`: Тип, указывающий на возможность отсутствия значения (None).
- `List`: Тип для списков.
- `get_type_hints`: Функция для получения аннотаций типов.
