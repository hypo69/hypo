# Модуль onela_bot

## Обзор

Модуль `onela_bot` предназначен для взаимодействия с моделью ассистента программиста через чат Telegram.  Он обрабатывает текстовые сообщения и загруженные документы, получая ответы от модели и отправляя их пользователю.

## Классы

### `OnelaBot`

**Описание**: Класс `OnelaBot` реализует взаимодействие с моделью ассистента программиста. Он наследуется от класса `TelegramBot` из модуля `src.bots.telegram`.

**Атрибуты**:

- `model`: Объект класса `GoogleGenerativeAI`, используемый для получения ответов от модели.


**Методы**:

#### `__init__`

**Описание**: Инициализирует объект `OnelaBot`.

**Параметры**:
- Нет

**Возвращает**:
- Не применимо


#### `handle_message`

**Описание**: Обрабатывает текстовые сообщения, получаемые от пользователя.

**Параметры**:
- `update` (Update): Объект, содержащий данные о полученном сообщении.
- `context` (CallbackContext): Объект, предоставляющий контекст выполнения.

**Возвращает**:
- `None`

**Обрабатывает исключения**:
- `Exception`: Обработка любых ошибок при получении ответа от модели. Логирует ошибку с использованием модуля `logger`.



#### `handle_document`

**Описание**: Обрабатывает загруженные документы, полученные от пользователя.

**Параметры**:
- `update` (Update): Объект, содержащий данные о полученном документе.
- `context` (CallbackContext): Объект, предоставляющий контекст выполнения.

**Возвращает**:
- `None`

**Обрабатывает исключения**:
- `Exception`: Обработка любых ошибок при загрузке и обработке документа. Логирует ошибку с использованием модуля `logger`.
