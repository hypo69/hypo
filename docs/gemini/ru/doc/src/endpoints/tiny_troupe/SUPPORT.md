# Документация для разработчика: SUPPORT.md

## Обзор

Файл `SUPPORT.md` содержит информацию о том, как сообщать об ошибках, получать помощь и политику поддержки проекта TinyTroupe.

## Подробнее

Этот файл предоставляет инструкции о том, как правильно сообщать о проблемах и искать помощь в использовании проекта. Он также описывает политику поддержки Microsoft для TinyTroupe.

## Разделы

### Как сообщать об ошибках и получать помощь

В этом разделе объясняется, как использовать GitHub Issues для отслеживания ошибок и запросов на новые функции. Прежде чем сообщать о новых проблемах, рекомендуется выполнить поиск по существующим проблемам, чтобы избежать дублирования.

### Политика поддержки Microsoft

В этом разделе указывается, что поддержка TinyTroupe ограничена ресурсами, перечисленными выше.
