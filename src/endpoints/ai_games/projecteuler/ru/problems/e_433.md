# Задача No 433
## Замечательная последовательность Фибоначчи
## Условия
Последовательность Фибоначчи F_n определяется как:

* F_0 = 0
* F_1 = 1
* F_n = F_{n-1} + F_{n-2} для n > 1.

Последовательность Фибоначчи начинается с 0, 1, 1, 2, 3, 5, 8, 13, ...

Также определим функцию g(n), как наибольший общий делитель (НОД) n-го числа Фибоначчи и (n+1)-го числа Фибоначчи, то есть g(n) = НОД(F_n, F_{n+1}).

Например:
* g(1) = НОД(1, 1) = 1
* g(2) = НОД(1, 2) = 1
* g(3) = НОД(2, 3) = 1
* g(4) = НОД(3, 5) = 1
* g(5) = НОД(5, 8) = 1

Очевидно, что g(n)=1 для всех n.

Теперь давайте определим функцию s(n) как сумму чисел g(i) для i от 1 до n, то есть s(n) = ∑ g(i) для 1 ≤ i ≤ n.

Итак, s(5) = 5.

Пусть функция F(k) будет суммой s(n) для n от 1 до k.
То есть F(k) = ∑ s(n) для 1 ≤ n ≤ k.

Например:
* F(5) = s(1) + s(2) + s(3) + s(4) + s(5) = 1 + 2 + 3 + 4 + 5 = 15.

Дано, что F(100) = 3528.

Найдите F(10^6).
