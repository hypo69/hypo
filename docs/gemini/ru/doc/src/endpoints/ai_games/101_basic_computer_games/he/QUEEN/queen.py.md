# QUEEN

## Обзор

Данный модуль реализует решение задачи о восьми ферзях на шахматной доске 8x8. Цель состоит в том, чтобы разместить восемь ферзей на доске таким образом, чтобы ни один из них не угрожал другому. Модуль предоставляет алгоритм для поиска одного из возможных решений этой задачи.

## Функции

### `is_position_safe`

**Описание**:
Функция проверяет, безопасно ли разместить ферзя в заданной позиции на шахматной доске.

**Параметры**:
- `board` (list): Двумерный массив, представляющий шахматную доску.
- `row` (int): Индекс строки для проверки.
- `col` (int): Индекс столбца для проверки.

**Возвращает**:
- `bool`: Возвращает `True`, если позиция безопасна для размещения ферзя, иначе `False`.

### `solve_queens`

**Описание**:
Функция пытается решить задачу о восьми ферзях, размещая ферзей на шахматной доске таким образом, чтобы ни один ферзь не угрожал другому.

**Параметры**:
- Нет параметров

**Возвращает**:
- `None`: Функция выводит решение на печать или сообщение, что решение не найдено, и ничего не возвращает.

**Вызывает исключения**:
- Нет явных исключений, но может завершить выполнение, если решение не найдено.
