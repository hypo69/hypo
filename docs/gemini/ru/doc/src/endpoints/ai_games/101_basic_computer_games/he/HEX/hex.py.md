# HEX

## Обзор

Этот модуль реализует игру "Гекс" для двух игроков, где один игрок (компьютер) играет против другого (человека). Цель игры - создать непрерывный путь из своих фишек от одной стороны доски до противоположной.

## Функции

### `create_board`

**Описание**: Создает и возвращает пустую игровую доску размером 10x10.

**Параметры**:
- Нет параметров.

**Возвращает**:
- `list[list[str]]`: Игровая доска, представленная в виде списка списков, где каждый элемент – пустая строка (`' '`).

### `print_board`

**Описание**: Выводит текущее состояние игровой доски на экран, включая координаты строк и столбцов.

**Параметры**:
- `board` (list[list[str]]): Игровая доска, представленная в виде списка списков.

**Возвращает**:
- `None`: Функция ничего не возвращает, а только выводит данные на экран.

### `choose_random_empty_cell`

**Описание**: Выбирает случайную пустую ячейку на игровой доске.

**Параметры**:
- `board` (list[list[str]]): Игровая доска, представленная в виде списка списков.

**Возвращает**:
- `tuple[int, int] | None`: Координаты выбранной пустой ячейки в виде кортежа `(row, col)` или `None`, если нет пустых ячеек.

### `get_player_input`

**Описание**: Запрашивает у игрока ввод координат ячейки, проверяет корректность ввода и возвращает координаты.

**Параметры**:
- `board` (list[list[str]]): Игровая доска, представленная в виде списка списков.

**Возвращает**:
- `tuple[int, int]`: Координаты выбранной игроком ячейки в виде кортежа `(row, col)`.

**Вызывает исключения**:
- `ValueError`: Если введенные данные не являются целыми числами.

### `is_winner`

**Описание**: Проверяет, есть ли победитель на доске. **В текущей версии всегда возвращает `False`**. Реализация проверки победителя отсутствует.

**Параметры**:
- `board` (list[list[str]]): Игровая доска, представленная в виде списка списков.
- `player` (str): Символ игрока (`'X'` или `'O'`).

**Возвращает**:
- `bool`: Всегда возвращает `False`.

### `is_board_full`

**Описание**: Проверяет, заполнена ли игровая доска.

**Параметры**:
- `board` (list[list[str]]): Игровая доска, представленная в виде списка списков.

**Возвращает**:
- `bool`: `True`, если доска полностью заполнена, `False` в противном случае.

### `play_hex`

**Описание**: Основная функция, управляющая игровым процессом. Инициализирует игру, обрабатывает ходы игроков и проверяет условия окончания игры.

**Параметры**:
- Нет параметров.

**Возвращает**:
- `None`: Функция ничего не возвращает, управляет ходом игры.
