# Модуль hypotez/src/suppliers/wallmart/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`.  Он также загружает строку документации из `README.MD`.  Модуль инициализирует переменные, содержащие информацию о проекте, такие как имя, версия, описание, автор и авторские права, и может предоставить ссылку на спонсорскую страницу для поддержания разработчика.

## Функции

### `set_project_root`

**Описание**: Определяет корневую директорию проекта, начиная с текущего файла и проходя вверх по иерархии директорий, пока не найдет директорию, содержащую один из файлов-маркеров (pyproject.toml, requirements.txt или .git).

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, по которым определяется корень проекта. По умолчанию - `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта. Если корневая директория не найдена, возвращается директория расположения текущего файла.

**Вызывает исключения**:

- Нет.

### Загрузка настроек из файла `settings.json`

Этот раздел описывает процесс загрузки и обработки настроек из файла `settings.json`.

**Описание**:

Функция пытается открыть файл `settings.json` в директории проекта и загрузить данные в формате JSON. Если файл не найден или содержит некорректные данные, переменная `settings` остается `None`.

**Возвращает**:

- `dict | None`: словарь с настройками проекта (если файл был успешно загружен), иначе None.


### Загрузка документации из `README.MD`

**Описание**:

Этот раздел описывает процесс загрузки строки документации из файла `README.MD`.

**Возвращает**:

- `str | None`: строка с содержанием `README.MD` (если файл был успешно загружен), иначе `None`.



## Использование

```python
# Пример использования
from hypotez.src.suppliers.wallmart.header import set_project_root
project_root = set_project_root()
print(f"Корневая директория проекта: {project_root}")
```

## Переменные

Этот раздел описывает глобальные переменные, определяемые в модуле.

- `__root__`: Путь к корневой директории проекта, полученный с помощью функции `set_project_root()`. Имеет тип `Path`.

- `settings`: Словарь с настройками проекта, загруженными из `settings.json`. Может быть `None` если файл не найден или некорректен. Тип `dict | None`.

- `doc_str`: Строка документации, загруженная из `README.MD`. Может быть `None` при ошибке. Тип `str | None`.

- `__project_name__`: Имя проекта, взятое из настроек. По умолчанию `hypotez`. Тип `str`.

- `__version__`: Версия проекта, взятая из настроек. По умолчанию пустая строка. Тип `str`.

- `__doc__`: Документация проекта (строка из `README.MD`). По умолчанию пустая строка. Тип `str`.

- `__details__`: Дополнительные детали проекта (по умолчанию пустая строка). Тип `str`.

- `__author__`: Автор проекта, взятый из настроек. По умолчанию пустая строка. Тип `str`.

- `__copyright__`: Авторские права, взятые из настроек. По умолчанию пустая строка. Тип `str`.

- `__cofee__`: Ссылка на страницу спонсорства (для поддержки разработчика). По умолчанию - ссылка на Boosty.  Тип `str`.
