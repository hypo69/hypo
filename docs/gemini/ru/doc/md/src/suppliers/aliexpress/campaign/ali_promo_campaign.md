# Модуль `hypotez/src/suppliers/aliexpress/campaign/ali_promo_campaign.py`

## Обзор

Модуль `ali_promo_campaign` предоставляет класс `AliPromoCampaign` для управления рекламными кампаниями на AliExpress. Он позволяет загружать, обрабатывать данные кампаний, работать с категориями и товарами, а также использовать ИИ для генерации описаний и другой информации. Модуль поддерживает различные языки и валюты.

## Классы

### `AliPromoCampaign`

**Описание**: Класс `AliPromoCampaign` предназначен для управления рекламными кампаниями. Он загружает данные кампаний, обрабатывает категории и товары, использует ИИ для генерации информации и сохраняет результаты в файлах.

**Атрибуты**:
- `language`: Язык кампании.
- `currency`: Валюта кампании.
- `base_path`: Базовый путь к данным кампании.
- `campaign_name`: Название кампании.
- `campaign`: Объект `SimpleNamespace` с данными о кампании из файла.
- `campaign_ai`: Объект `SimpleNamespace` с данными о кампании, сгенерированными ИИ.
- `gemini`: Модель ИИ Gemini.
- `openai`: Модель ИИ OpenAI (используется ли?).


**Методы**:

#### `__init__`

**Описание**: Инициализирует объект `AliPromoCampaign` для заданной кампании.

**Параметры**:
- `campaign_name` (str): Название кампании.
- `language` (Optional[str], optional): Язык кампании. По умолчанию None.
- `currency` (Optional[str], optional): Валюта кампании. По умолчанию None.
- `model` (str, optional): Тип модели ИИ (openai или gemini). По умолчанию 'openai'.

**Возвращает**:
- `SimpleNamespace`: Объект, представляющий кампанию.

**Обрабатывает исключения**:
- Возможны ошибки при загрузке файла кампании.


#### `process_campaign`

**Описание**: Обрабатывает все категории в кампании.

**Пример использования**:
```python
campaign.process_campaign()
```

#### `process_category_products`

**Описание**: Обрабатывает товары в указанной категории.

**Параметры**:
- `category_name` (str): Название категории.

**Возвращает**:
- `Optional[List[SimpleNamespace]]`: Список обработанных товаров, или `None`, если товары не найдены.


#### `process_ai_category`

**Описание**: Обрабатывает категорию с помощью ИИ.

**Параметры**:
- `category_name` (Optional[str], optional): Название категории. Если не указано, обрабатываются все категории.

**Возвращает**:
- None

**Обрабатывает исключения**:
- Возможны ошибки при взаимодействии с моделью ИИ.


#### `process_new_campaign`

**Описание**: Создаёт новую рекламную кампанию.

**Параметры**:
- `campaign_name` (str): Название кампании.
- `language` (Optional[str], optional): Язык кампании. По умолчанию None.
- `currency` (Optional[str], optional): Валюта кампании. По умолчанию None.

**Возвращает**:
- `List[Tuple[str, Any]]`: Список кортежей с именами категорий и обработанными результатами.

**Описание работы**:
Метод создает структуру для новой кампании, включая JSON-файлы для категорий и товаров.


#### `dump_category_products_files`

**Описание**: Сохраняет данные о товарах в JSON-файлы.

**Параметры**:
- `category_name` (str): Название категории.
- `products` (List[SimpleNamespace]): Список товаров.


#### `set_categories_from_directories`

**Описание**: Устанавливает категории из названий директорий в корне кампании.

**Возвращает**:
- `None`


#### `generate_output`

**Описание**: Генерирует выходные данные для рекламной кампании.


#### `generate_html`

**Описание**: Генерирует HTML-страницы для категории и главной страницы кампании.

#### `generate_html_for_campaign`

**Описание**: Генерирует HTML-страницы для всех категорий кампании.
