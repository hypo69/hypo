# Документация для `ksp_categories_notebooks_asus_by_model.json`

## Обзор

Данный JSON-файл содержит сценарии для категоризации ноутбуков ASUS по моделям на сайте KSP. Каждая модель имеет набор параметров, включая бренд, URL, флаги активности и соответствия категориям PrestaShop.

## Структура файла
Файл содержит один корневой объект JSON с ключом `scenarios`, значением которого является объект, содержащий пары "ключ-значение", где ключ - это название модели ноутбука, а значение - объект с описанием параметров сценария.
## Сценарии
Сценарии описывают конфигурации для каждой модели ноутбука ASUS, включая URL-адрес страницы KSP и соответствия категориям PrestaShop.

## Используемые ключи

### Ключи верхнего уровня

-   `scenarios` (object): Объект, содержащий все сценарии.

### Ключи сценария

-   `brand` (string): Бренд ноутбука.
-   `url` (string): URL-адрес страницы с товаром на сайте KSP.
-   `checkbox` (boolean): Флаг для чекбокса.
-   `active` (boolean): Флаг активности сценария.
-   `condition` (string): Состояние товара ("new").
-   `presta_categories` (object): Объект, содержащий соответствия категориям PrestaShop, либо шаблон для генерации.
    -   `ключ` (string): ID категории PrestaShop.
    -   `значение` (string): Название категории PrestaShop.
    -   `template` (object): Объект для шаблонизации категорий.
        -    `asus` (array): Массив, содержащий шаблон для генерации категорий.
          -  `0` (string) - шаблон для категорий процессоров Intel/AMD
          -  `1` (string) - шаблон для размера экрана


## Заключение
Этот документ предназначен для разработчиков, работающих с данными о ноутбуках ASUS. Он описывает структуру JSON-файла и ключи, используемые для категоризации товаров на сайте KSP. Данные соответствия используются для автоматической привязки товаров к соответствующим категориям PrestaShop.
