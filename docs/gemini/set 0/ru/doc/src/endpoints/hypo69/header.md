# Модуль hypotez/src/endpoints/hypo69/header.py

## Обзор

Этот модуль содержит вспомогательные функции для работы с проектом `hypotez`. Он находит корневую директорию проекта, загружает настройки из файла `settings.json` и читает README.md для получения документации.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла.  Ищет вверх по дереву директорий, пока не найдёт директорию, содержащую один из файлов в списке `marker_files`.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или каталогов, используемых для идентификации корневой директории проекта. По умолчанию содержит ('pyproject.toml', 'requirements.txt', '.git').


**Возвращает**:

- `Path`: Путь до корневой директории проекта, если найдена. В противном случае возвращает директорию, где расположен текущий скрипт.


**Вызывает исключения**:

- Не вызывает исключений.


### <ins>Переменные</ins>

#### `__root__`

**Описание**: Путь до корневой директории проекта.


## Переменные

### `MODE`

**Описание**: Строковая переменная, хранящая режим работы проекта.


### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженные из файла `settings.json`.  Значение по умолчанию – `None`.


### `doc_str`

**Описание**: Строковая переменная, содержащая текст файла `README.MD`. Значение по умолчанию - `None`.


### `__project_name__`

**Описание**: Имя проекта, полученное из настроек или установленное по умолчанию (`hypotez`).


### `__version__`

**Описание**: Версия проекта, полученная из настроек или установленная по умолчанию (пустая строка).


### `__doc__`

**Описание**:  Документация проекта, полученная из файла `README.md` или пустая строка.


### `__details__`

**Описание**: Подробное описание проекта. По умолчанию пустая строка.


### `__author__`

**Описание**: Автор проекта, полученный из настроек или установленный по умолчанию (пустая строка).


### `__copyright__`

**Описание**: Авторские права проекта, полученные из настроек или установленные по умолчанию (пустая строка).


### `__cofee__`

**Описание**: Ссылка для поддержки разработчика (чашка кофе).


## Обработка исключений

### Файл settings.json

Если файл `settings.json` не найден или содержит некорректный JSON, переменная `settings` остаётся `None`.

### Файл README.MD

Если файл `README.MD` не найден или содержит некорректные данные, переменная `doc_str` остаётся `None`.
