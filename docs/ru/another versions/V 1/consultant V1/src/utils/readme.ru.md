Tiny Utils
============

## Обзор

**Tiny Utils** — это библиотека утилит, предоставляющая набор лёгких вспомогательных функций для различных общих задач. Библиотека включает в себя утилиты для преобразования форматов данных, обработки текста и файлов, строковых операций, форматирования дат и времени, обработки изображений и другое. Она организована в несколько модулей для удобного доступа к конкретным функциональным возможностям.

## Установка

Для использования **Tiny Utils** клонируйте репозиторий и установите необходимые зависимости, как указано в файле `requirements.txt`.

```bash
git clone https://github.com/hypo69/tiny-utils.git # Уточнённый URL репозитория
cd tiny_utils
pip install -r requirements.txt
```


## Обзор модулей

Библиотека содержит несколько подмодулей, каждый из которых обрабатывает определённую задачу:

- **Преобразователи**: Модули для преобразования форматов данных, таких как текст в изображение, WebP в PNG, JSON, XML, кодирование Base64 и другие.
- **Утилиты для строк**: Инструменты для продвинутой работы со строками.
- **Утилиты для работы с файлами**: Функции для обработки и манипулирования файлами.
- **Утилиты для работы с датами и временем**: Инструменты для форматирования дат и времени.
- **Утилиты для работы с FTP**: Функции для работы с FTP-серверами.
- **Утилиты для работы с изображениями**: Базовые функции обработки изображений.
- **Утилиты для работы с PDF**: Утилиты для обработки и преобразования файлов PDF.
- **Утилиты для работы с принтером**: Функции для отправки данных на принтер.


## Описание модулей

### Преобразователи

Модуль `convertors` содержит утилиты для преобразования данных между форматами. Эти модули могут обрабатывать различные типы данных, от CSV до JSON и текста до изображений.

#### Файлы:

- `text2png.py`: Преобразует данные текста в изображение PNG.
- `tts.py`: Преобразует текст в речь и сохраняет его в виде аудиофайла.
- `webp2png.py`: Преобразует изображения из формата WebP в PNG.
- `xls.py`: Обрабатывает преобразования и манипуляции с файлами XLS.
- `xml2dict.py`: Преобразует данные XML в словарь Python.
- `base64.py`: Кодирует или декодирует данные с использованием кодирования Base64.
- `csv.py`: Предоставляет инструменты для парсинга и манипулирования CSV.
- `dict.py`: Утилиты для обработки словарей Python.
- `html.py`: Преобразует контент HTML в различные форматы.
- `json.py`: Утилиты для парсинга и манипулирования JSON.
- `md2dict.py`: Преобразует контент Markdown в словарь.
- `ns.py`: Специализированные утилиты для преобразования имён пространств.


### Утилиты для строк

Модуль `strings` предоставляет набор функций для работы со строками. Эти утилиты включают в себя функции для поиска, замены, форматирования и проверки строк.

### Утилиты для работы с файлами

Модуль `files` содержит функции для чтения, записи и управления файлами. Эти инструменты облегчают работу с файлами различных форматов и обеспечивают удобный доступ к их содержимому.

### Утилиты для работы с датами и временем

Модуль `datetime` предоставляет инструменты для работы с датами и временем, включая форматирование, преобразование и вычисление разницы между датами и временем.

### Утилиты для работы с FTP

Модуль `ftp` включает в себя функции для взаимодействия с FTP-серверами, включая загрузку и выгрузку файлов, а также управление каталогами.

### Утилиты для работы с изображениями

Модуль `images` содержит базовые функции для обработки изображений, включая изменение размера, обрезку и конвертацию форматов.

### Утилиты для работы с PDF

Модуль `pdf` предоставляет инструменты для работы с файлами PDF, включая извлечение текста, слияние и разделение PDF-документов.

### Утилиты для работы с принтером

Модуль `printer` включает в себя функции для отправки данных на принтер, позволяя программно распечатывать различные документы.

## Примеры использования

### Преобразование текста в изображение PNG

```python
from src.utils.convertors.text2png import text_to_png

text = "Пример текста"
image_path = "example.png"
text_to_png(text, image_path)
print(f"Изображение сохранено в {image_path}")
```

### Преобразование XML в словарь

```python
from src.utils.convertors.xml2dict import xml_to_dict

xml_string = "<root><element>данные</element></root>"
result = xml_to_dict(xml_string)
print(result)
```

### Парсинг и манипуляции с JSON

```python
from src.utils.convertors.json import j_loads

json_string = '{"key": "value"}'
data = j_loads(json_string)
print(data)
```
