# Модуль `pricelist_generator.py`

## Обзор

Модуль `pricelist_generator.py` предназначен для генерации HTML и PDF файлов с прайс-листами для мехиронов Казаринова. Он использует шаблоны Jinja2 для создания HTML-документов и `pdfkit` для преобразования HTML в PDF.

## Классы

### `ReportGenerator`

**Описание**: Класс для генерации HTML- и PDF-отчётов на основе данных из JSON.

**Методы**:
- [`generate_html`](#generate_html)
- [`create_report`](#create_report)

#### `generate_html`

**Описание**: Генерирует HTML-контент на основе шаблона и данных.

**Параметры**:
- `data` (dict): Словарь с данными для заполнения шаблона.
- `lang` (str): Язык отчёта ('ru' или 'he').

**Возвращает**:
- `str`: HTML-контент.

#### `create_report`

**Описание**: Полный цикл генерации отчёта. Загружает данные, генерирует HTML, сохраняет HTML в файл и конвертирует в PDF.

**Параметры**:
- `data` (dict): Словарь с данными для генерации отчёта.
- `lang` (str): Язык отчёта.
- `html_file` (str | Path): Путь к файлу для сохранения HTML.
- `pdf_file` (str | Path): Путь к файлу для сохранения PDF.

**Возвращает**:
- `bool`: `True`, если отчёт успешно сгенерирован, `False` в противном случае.

## Функции

### `main`

**Описание**: Основная функция для запуска генерации отчёта. Загружает данные из JSON, создает экземпляра `ReportGenerator` и запускает процесс генерации.

**Параметры**:
- `mexiron` (str): Название мехирона.
- `lang` (str): Язык отчёта.

**Возвращает**:
- `bool`: Результат выполнения генерации отчета.
