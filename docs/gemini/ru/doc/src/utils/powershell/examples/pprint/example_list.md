# Модуль example_list

## Обзор

Данный модуль содержит пример использования функций работы со списками.

## Функции

### `example_list`

**Описание**: Функция демонстрирует использование списков, выводит их элементы.

**Параметры**:
- `lst` (list): Список, элементы которого будут выведены.

**Возвращает**:
- `None`: Функция не возвращает значение.
