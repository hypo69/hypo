# Модуль для определения схем данных продуктов

## Обзор

Этот модуль определяет Pydantic-схемы для валидации и представления данных, связанных с продуктами. Он включает в себя модели для идентификации продукта (`ProductIDModel`) и для представления основных атрибутов продукта (`ProductModel`).

## Классы

### `ProductIDModel`

**Описание**: Эта модель используется для представления идентификатора продукта.

**Атрибуты**:
- `id` (int): Уникальный идентификатор продукта.

### `ProductModel`

**Описание**: Эта модель используется для представления основных атрибутов продукта, таких как имя, описание, цена, категория и прочее.

**Атрибуты**:
- `name` (str): Название продукта, минимальная длина - 5 символов.
- `description` (str): Описание продукта, минимальная длина - 5 символов.
- `price` (int): Цена продукта, должна быть больше 0.
- `category_id` (int): Идентификатор категории продукта, должен быть больше 0.
- `file_id` (str | None): Идентификатор файла, связанного с продуктом (может отсутствовать).
- `hidden_content` (str): Скрытое описание продукта, минимальная длина - 5 символов.
