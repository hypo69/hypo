# Модуль hypotez/src/webdriver

## Обзор

Данный модуль предоставляет инструменты для работы с веб-драйверами различных браузеров. Он включает в себя классы для управления драйверами Chrome, Firefox, Edge, а также вспомогательные классы для работы с различными библиотеками, такими как Beautiful Soup, Playwright, и Crawlee.

## Классы

### Класс `Driver`

**Описание**: Базовый класс для работы с веб-драйверами.  Предоставляет общие методы для взаимодействия с драйверами.


### Класс `Chrome`

**Описание**: Класс для управления веб-драйвером Chrome. Наследуется от `Driver`.


### Класс `Firefox`

**Описание**: Класс для управления веб-драйвером Firefox. Наследуется от `Driver`.


### Класс `Edge`

**Описание**: Класс для управления веб-драйвером Edge. Наследуется от `Driver`.


### Класс `BS`

**Описание**: Класс для работы с библиотекой Beautiful Soup.


### Класс `Playwright`

**Описание**: Класс для работы с библиотекой Playwright.


### Класс `CrawleePython`

**Описание**: Класс для работы с библиотекой Crawlee Python.


## Переменные

### `MODE`

**Описание**: Переменная, хранящая режим работы (например, 'dev' для разработки).
