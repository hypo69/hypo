# Модуль `src.bots.discord`

## Обзор

Данный модуль представляет собой инициализационный файл для пакета `src.bots.discord`. Он определяет константу `MODE`, которая может использоваться для настройки режима работы приложения.

## Переменные

### `MODE`

**Описание**: Константа, определяющая режим работы приложения.

**Тип**: `str`

**Значение**: `'dev'` (по умолчанию).
