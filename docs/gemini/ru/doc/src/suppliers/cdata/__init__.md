# Модуль hypotez/src/suppliers/cdata/__init__.py

## Обзор

Данный модуль `cdata` предоставляет базовые функции и классы для работы с поставщиками данных. В нем определен режим работы (`MODE`) и импортируется класс `Graber` из модуля `graber`.


## Переменные

### `MODE`

**Описание**: Строковая переменная, определяющая режим работы системы ('dev').

**Значение**:  'dev'


## Классы

### `Graber`

**Описание**: Класс `Graber` отвечает за получение данных от поставщиков.  Подробная информация о методах класса доступна в документации модуля `graber`.
