# Инструкция по внесению вклада в gpt4free

### Обзор

Этот документ предоставляет инструкции для разработчиков, желающих внести свой вклад в проект gpt4free. Он описывает шаги, необходимые для добавления поддержки новых веб-сайтов, а также для рефакторинга и интеграции кода в проект.

### Пошаговая инструкция

1. **Reverse-инжиниринг веб-сайта**:
   - Выберите веб-сайт из списка [sites-to-reverse](https://github.com/xtekky/gpt4free/issues/40), который вы хотите добавить в проект.
   - Проведите обратный инжиниринг этого веб-сайта, чтобы понять, как он работает и как взаимодействует с сервером.

2. **Добавление в unittest**:
   - Добавьте файлы для тестирования reverse-инжиниринга веб-сайта в директорию [./etc/unittest/](https://github.com/xtekky/gpt4free/tree/main/etc/unittest/).
   - Это позволит убедиться, что веб-сайт работает правильно и соответствует ожиданиям.

3. **Рефакторинг и добавление в g4f**:
   - Выполните рефакторинг кода и добавьте его в директорию [./g4f](https://github.com/xtekky/gpt4free/tree/main/g4f).
   - Убедитесь, что код соответствует стандартам проекта и хорошо интегрируется с существующей кодовой базой.

### Заключение
