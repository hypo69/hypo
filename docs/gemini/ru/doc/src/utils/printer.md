# Модуль hypotez/src/utils/printer.py

## Обзор

Модуль `hypotez/src/utils/printer.py` предоставляет функции для красивой печати данных в консоли с возможностью применения различных стилей текста (цвет, фон, шрифт).  Он позволяет форматировать данные различных типов (словари, списки, строки, пути к файлам) и выводить их в консоли с заданными стилями.  Модуль использует ANSI escape коды для форматирования вывода.

## Функции

### `_color_text`

**Описание**: Функция для применения цветовых и стилистических настроек к тексту.  Принимает на вход текст и параметры для изменения цвета текста, фона и шрифта, используя ANSI escape коды.

**Параметры**:

- `text` (str): Текст, к которому нужно применить стили.
- `text_color` (str, optional): Цвет текста. Доступные цвета: "red", "green", "blue", "yellow", "white", "cyan", "magenta", "light_gray", "dark_gray", "light_red", "light_green", "light_blue", "light_yellow". По умолчанию пустая строка (без цвета).
- `bg_color` (str, optional): Цвет фона текста. Доступные цвета: "bg_red", "bg_green", "bg_blue", "bg_yellow", "bg_white", "bg_cyan", "bg_magenta", "bg_light_gray", "bg_dark_gray", "bg_light_red", "bg_light_green", "bg_light_blue", "bg_light_yellow". По умолчанию пустая строка (без фона).
- `font_style` (str, optional): Стиль шрифта. Доступные стили: "bold", "underline". По умолчанию пустая строка (без стиля).

**Возвращает**:

- str: отформатированный текст с примененными стилями.

**Пример использования**:

```python
_color_text("Hello, World!", text_color="green", font_style="bold")
```


### `pprint`

**Описание**: Функция для красивой печати данных с поддержкой цветовых и стилистических настроек.  Функция обрабатывает различные типы данных (списки, словари, строки, пути к файлам) и форматирует их для вывода в консоли.

**Параметры**:

- `print_data` (Any, optional): Данные для печати. Может быть None, dict, list, str или Path.
- `text_color` (str, optional): Цвет текста. По умолчанию "white". См. [TEXT_COLORS](#TEXT_COLORS).
- `bg_color` (str, optional): Цвет фона. По умолчанию пустая строка (нет фона). См. [BG_COLORS](#BG_COLORS).
- `font_style` (str, optional): Стиль шрифта. По умолчанию пустая строка (нет стиля). См. [FONT_STYLES](#FONT_STYLES).

**Возвращает**:

- None

**Вызывает исключения**:

- `Exception`:  В случае проблем с обработкой данных или при неудачном чтении файла.

**Пример использования**:

```python
pprint({"name": "Alice", "age": 30}, text_color="green")
pprint(["apple", "banana", "cherry"], text_color="blue", font_style="bold")
pprint("my_file.txt", text_color="yellow", bg_color="bg_red", font_style="underline")
```

## Константы

### `MODE`

**Описание**:  Строковая константа, вероятно, определяющая режим работы модуля (например, 'dev', 'prod').

### `TEXT_COLORS`

**Описание**: Словарь, содержащий ANSI escape коды для различных цветов текста.

### `BG_COLORS`

**Описание**: Словарь, содержащий ANSI escape коды для различных цветов фона.

### `FONT_STYLES`

**Описание**: Словарь, содержащий ANSI escape коды для различных стилей шрифта.

### `RESET`

**Описание**: ANSI escape код для сброса всех стилей.


## Замечания

Обратите внимание на использование `TEXT_COLORS`, `BG_COLORS`, `FONT_STYLES` для получения доступных цветов и стилей.  Обработка файлов ('.csv', '.xls') ограничена.
