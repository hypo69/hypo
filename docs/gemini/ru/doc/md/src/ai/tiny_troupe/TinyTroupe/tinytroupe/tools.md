# Модуль tinytroupe.tools

## Обзор

Этот модуль предоставляет инструменты для агентов, позволяющие им выполнять специализированные задачи. Он содержит классы для создания, управления и обработки действий инструментов, включая экспорт и обогащение результатов.

## Классы

### `TinyTool`

**Описание**: Базовый класс для инструментов.  Представляет собой инструмент, который может быть использован агентами для выполнения задач.

**Методы**:

- `__init__(name, description, owner=None, real_world_side_effects=False, exporter=None, enricher=None)`: Инициализирует новый инструмент.
    - **Параметры**:
        - `name` (str): Название инструмента.
        - `description` (str): Краткое описание инструмента.
        - `owner` (str, опционально): Агент, владеющий инструментом. Если `None`, инструмент может быть использован любым агентом.
        - `real_world_side_effects` (bool, опционально): Флаг, указывающий, имеет ли инструмент реальные последствия в мире. По умолчанию `False`.
        - `exporter` (ArtifactExporter, опционально): Экспортер, который может использоваться для экспорта результатов действий инструмента. Если `None`, инструмент не сможет экспортировать результаты.
        - `enricher` (Enricher, опционально): Обогатитель, который может использоваться для обогащения результатов действий инструмента. Если `None`, инструмент не сможет обогащать результаты.
    - **Возвращает**:
        - Нет.
- `_process_action(agent, action: dict) -> bool`:  Обрабатывает действие инструмента.  Необходимо реализовать в подклассах. Поднимает исключение `NotImplementedError`.
    - **Параметры**:
        - `agent`: Агент, выполняющий действие.
        - `action` (dict): Словарь, описывающий действие инструмента.
    - **Возвращает**:
        - `bool`: Указывает, было ли действие успешно обработано.
- `_protect_real_world()`:  Обрабатывает потенциальные реальные последствия действия.
    - **Параметры**:
        - Нет.
    - **Возвращает**:
        - Нет.
- `_enforce_ownership(agent)`:  Проверяет, принадлежит ли инструмент текущему агенту.
    - **Параметры**:
        - `agent`: Выполняющий действие агент.
    - **Возвращает**:
        - Нет.
    - **Исключения**:
        - `ValueError`: Если агент не владелец инструмента.
- `set_owner(owner)`: Устанавливает владельца инструмента.
    - **Параметры**:
        - `owner`: Владелец инструмента.
    - **Возвращает**:
        - Нет.
- `actions_definitions_prompt() -> str`: Возвращает подсказку для определения действий инструмента. Необходимо реализовать в подклассах. Поднимает исключение `NotImplementedError`.
    - **Возвращает**:
        - str: Строка с подсказкой.
- `actions_constraints_prompt() -> str`: Возвращает подсказку с ограничениями для действий инструмента. Необходимо реализовать в подклассах. Поднимает исключение `NotImplementedError`.
    - **Возвращает**:
        - str: Строка с ограничениями.
- `process_action(agent, action: dict) -> bool`: Обрабатывает действие инструмента, включая проверку владельца и реальных последствий.
    - **Параметры**:
        - `agent`: Агент, выполняющий действие.
        - `action` (dict): Словарь, описывающий действие инструмента.
    - **Возвращает**:
        - `bool`: Указывает, было ли действие успешно обработано.
