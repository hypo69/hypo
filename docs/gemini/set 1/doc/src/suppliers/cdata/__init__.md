# Модуль hypotez/src/suppliers/cdata/__init__.py

## Обзор

Данный модуль предоставляет базовые инструменты и классы для работы с поставщиком данных cdata.  Включает импорт класса `Graber` из подмодуля `graber`.  Константа `MODE` имеет значение 'dev'.

## Классы

### `Graber`

**Описание**: Класс `Graber` предназначен для работы с данными, полученными от поставщика.  Он предоставляет функционал для взаимодействия с API поставщика, обработки данных и возвращения результатов. Подробное описание методов класса находится в файле `graber.py`.

## Константы

### `MODE`

**Описание**:  Переменная `MODE` содержит текущий режим работы, например, "dev", "prod", и т.д.
**Значение**: 'dev'

## Импорты

- `from .graber import Graber` : Импортирует класс `Graber` из модуля `graber`.
