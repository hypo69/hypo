# Модуль `hypotez/src/endpoints/prestashop/api/__init__.py`

## Обзор

Этот модуль предоставляет инициализацию для API Престашоп. Он содержит константу `MODE` и импортирует класс `PrestaShop` из подмодуля `api`.

## Содержимое

### Константа `MODE`

**Описание**: Константа `MODE` определяет режим работы API (в данном случае 'dev').

**Значение**: `'dev'`


### Импорт `PrestaShop`

**Описание**: Импортирует класс `PrestaShop` из подмодуля `api`.

**Модуль**: `api`

**Класс**: `PrestaShop`
