## Переменные

### `dir_root`

**Описание**: Путь к корневой директории проекта.

**Тип**: `pathlib.Path`

**Инициализация**: `Path(os.getcwd()[:os.getcwd().rfind('hypotez')+7])`

**Описание инициализации**:
Строка `os.getcwd()[:os.getcwd().rfind('hypotez')+7]` создает путь к корневой директории проекта, начиная с текущей рабочей директории (`os.getcwd()`) и обрезая ее до позиции, где встречается подстрока `hypotez`, добавляя 7 символов для корректного получения полного пути.


## Импорты

* `os`
* `sys`
* `pathlib`


## Код

```python
MODE = 'dev'

import os
import sys
from pathlib import Path

dir_root : Path = Path (os.getcwd()[:os.getcwd().rfind('hypotez')+7]) ## <- Корневая директория проекта
sys.path.append (str (dir_root) )  # Добавляю корневую директорию в sys.path
dir_src = Path (dir_root, 'src')
sys.path.append (str (dir_root) ) # Добавляю рабочую директорию в sys.path
```
