# Модуль `ali_campaign_editor.mmd`

## Обзор

Данный модуль содержит реализацию класса `AliCampaignEditor`, предназначенного для редактирования кампаний на AliExpress.  Класс предоставляет методы для управления продуктами, категориями и самой кампанией.

## Класс `AliCampaignEditor`

### Описание

Класс `AliCampaignEditor` отвечает за обработку входящих данных для редактирования кампании AliExpress, включая чтение и запись файлов, обновление данных о продуктах и категориях, а также взаимодействие с API AliExpress.

### Методы

#### `__init__`

**Описание**: Инициализирует объект `AliCampaignEditor`.

**Параметры**:
- `campaign_name` (str): Название кампании.
- `language` (str): Язык кампании.
- `currency` (str): Валюта кампании.

**Возвращает**:
- `AliCampaignEditor`: Объект класса `AliCampaignEditor`.

#### `delete_product`

**Описание**: Удаляет продукт из кампании, если он соответствует заданным критериям.

**Параметры**:
- `product_id` (str): Идентификатор продукта.

**Возвращает**:
- `None`.

**Вызывает исключения**:
- `ValueError`: Если продукт не найден.


#### `update_product`

**Описание**: Обновляет информацию о продукте в кампании.

**Параметры**:
- `product_data` (dict): Данные для обновления продукта.

**Возвращает**:
- `None`.

**Вызывает исключения**:
- `ValueError`: Если продукт не найден.

#### `update_campaign`

**Описание**: Обновляет свойства кампании, такие как описание.

**Параметры**:
- `campaign_updates` (dict): Обновляемые свойства кампании.

**Возвращает**:
- `None`.

**Вызывает исключения**:
- `ValueError`: Если кампания не найдена.


#### `update_category`

**Описание**: Обновляет данные о категории в файле JSON.

**Параметры**:
- `category_name` (str): Название категории.
- `category_data` (dict): Обновляемые данные о категории.

**Возвращает**:
- `None`.

**Вызывает исключения**:
- `FileNotFoundError`: Если файл JSON не найден.
- `ValueError`: Если данные о категории некорректны.


#### `get_category`

**Описание**: Возвращает информацию о категории по имени.

**Параметры**:
- `category_name` (str): Название категории.

**Возвращает**:
- `SimpleNamespace`: Объект, содержащий данные о категории, или `None` если категория не найдена.

**Вызывает исключения**:
- `ValueError`: Если категория не найдена.


#### `list_categories`

**Описание**: Возвращает список всех категорий в кампании.

**Параметры**:
-  None

**Возвращает**:
- `list[str]`: Список названий категорий.

**Вызывает исключения**:
- `ValueError`: Если список категорий пустой или некорректный.


#### `get_category_products`

**Описание**: Возвращает продукты, относящиеся к указанной категории.

**Параметры**:
- `category_name` (str): Название категории.

**Возвращает**:
- `list[SimpleNamespace]`: Список продуктов, относящихся к категории.
- `None`: Если продукты не найдены.

**Вызывает исключения**:
- `FileNotFoundError`: Если файл JSON не найден.
- `ValueError`: Если данные в файлах некорректны.
