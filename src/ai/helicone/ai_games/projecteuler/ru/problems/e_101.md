## Условия
Если мы возьмем последовательность кубических чисел, то есть n^3: 1, 8, 27, 64, 125, ..., можно легко получить «неправильную» формулу для последовательности, используя интерполяцию многочленами.

Например, если мы возьмем первые два члена, 1 и 8, мы можем получить простую линейную функцию f(n) = 7n - 6, что дает нам:

f(1) = 1
f(2) = 8

Точно так же, взяв первые три члена, 1, 8 и 27, мы можем получить квадратную функцию:

f(n) = 6n^2 - 11n + 6

В действительности, когда мы используем только первые k членов последовательности и ищем наименьшую степень многочлена, который проходит через эти k точек, мы получим уникальный многочлен f(n) степени k-1.

Этот многочлен затем может быть использован для генерации следующих членов последовательности.

Определим OP(k, n) как n-й член, генерируемый многочленом, который точно соответствует первым k членам последовательности.
Определим U(k, n) как n-й член последовательности кубических чисел.

Таким образом, в приведенном выше примере мы находим, что OP(2, 3) = 15 и OP(3, 4) = 58.

В качестве еще одного примера рассмотрим последовательность u(n) = n^3 - n^2 + n. Первые десять членов этой последовательности будут: 1, 3, 7, 13, 21, 31, 43, 57, 73, 91.

Многочлены наименьшей степени, которые точно соответствуют первым k членам этой последовательности, дадут следующие значения:
```
OP(1, n) = 1
OP(2, n) = 2n - 1
OP(3, n) = n^2 - n + 1
OP(4, n) = n^3 - n^2 + n
```
Исходя из вышесказанного, мы получим следующие "неправильные" значения:
```
OP(1, 1) = 1
OP(1, 2) = 1
OP(1, 3) = 1
OP(2, 2) = 3
OP(2, 3) = 5
OP(3, 3) = 7
OP(3, 4) = 13
OP(4, 4) = 13
OP(4, 5) = 31
```
Сумма первых десяти членов последовательности u(n) = n^3 - n^2 + n, является: 1 + 3 + 7 + 13 + 21 + 31 + 43 + 57 + 73 + 91 = 330.

Найдите сумму всех «неправильных» значений (т.е. тех, для которых OP(k, n) ≠ u(n)), для первых 10 членов последовательности, которая задается формулой u(n) = 1 - n + n^2 - n^3 + n^4 - n^5 + n^6 - n^7 + n^8 - n^9 + n^10.
