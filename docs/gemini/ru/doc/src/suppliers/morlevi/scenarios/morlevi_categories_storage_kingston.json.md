## Структура сценария

Каждый сценарий имеет следующую структуру:

*   `brand` (str): Бренд продукта.
*   `url` (str): URL-адрес, используемый для получения данных о продукте.
*   `checkbox` (bool): Флаг, определяющий нужно ли использовать чекбокс.
*  `active` (bool): Флаг, определяющий, активен ли сценарий.
*   `condition` (str): Состояние продукта (например, "new").
*   `presta_categories` (dict): Категории PrestaShop, к которым относится продукт.
    *   `template` (dict): Шаблон для категорий PrestaShop. Включает соответствие имени бренда и названия категории PrestaShop.
