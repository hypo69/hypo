# Модуль PrestaShop

## Обзор

Данный модуль предоставляет инструменты для взаимодействия с API платформы PrestaShop. Он содержит классы и функции для работы с различными ресурсами API, такими как категории, клиенты, языки, прайс-листы, продукты, магазины, поставщики, склады и т.д.  Модуль структурирован для удобного использования и содержит примеры кода в директории `_examples`.  Дополнительно, директория `api_schemas` содержит схемы данных для различных ресурсов API, что позволяет проверять данные перед отправкой запросов.

## Директории

### domains

Эта директория содержит поддиректории для различных доменов с настроенными параметрами и конфигурациями.

#### ecat.co.il

Настройки для домена `ecat.co.il`. Содержит файл `settings.json` с настройками.

#### emildesign.com

Настройки для домена `emildesign.com`. Содержит файл `settings.json` с настройками.

#### sergey.mymaster.co.il

Настройки для домена `sergey.mymaster.co.il`. Содержит файл `settings.json` с настройками.

###  Модуль API

Модуль `api` предоставляет интерфейс для взаимодействия с API PrestaShop.

###  Модуль API Schemas

Директория `api_schemas` содержит JSON-схемы для различных ресурсов API PrestaShop.  Это позволяет валидировать входящие данные перед отправкой запросов.  Схемы находятся в файлах JSON.

## Пример использования

```python
from PrestaShop.product import Product  # Пример, заменив на правильное имя модуля

# Инициализировать продукт
product = Product()

# Пример операции с продуктом
product_data = product.get_product_data(product_id="12345")

print(product_data)
```
