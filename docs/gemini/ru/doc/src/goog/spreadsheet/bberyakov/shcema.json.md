# Документация для файла `shcema.json`

## Обзор

Данный файл содержит JSON-схему для стилизации заголовка рабочего листа (worksheet title) в Google Sheets.  Она определяет параметры форматирования, такие как высота строки, цвет фона, горизонтальное выравнивание, направление текста и форматирование текста.

## Структура JSON

### Свойство `ws_title`

Объект, содержащий настройки стилей для заголовка рабочего листа.

#### Свойство `rowHeight`

- **Описание**: Высота строки заголовка в пикселях.
- **Тип**: `строка`
- **Пример**: `"35"`

#### Свойство `backgroundColor`

- **Описание**: Цвет фона заголовка в формате шестнадцатеричного кода.
- **Тип**: `строка`
- **Пример**: `"#FFAAAA"`

#### Свойство `horizontalAlignment`

- **Описание**: Горизонтальное выравнивание текста в заголовке.
- **Тип**: `строка`
- **Возможные значения**: `"RIGHT"`, `"LEFT"`, `"CENTER"`
- **Пример**: `"RIGHT"`

#### Свойство `textDirection`

- **Описание**: Направление текста.
- **Тип**: `строка`
- **Возможные значения**: `"RTL"` (справа налево), `"LTR"` (слева направо).
- **Пример**: `"RTL"`

#### Свойство `textFormat`

Объект, содержащий настройки форматирования текста заголовка.

##### Свойство `foregroundColor`

- **Описание**: Цвет текста заголовка в формате шестнадцатеричного кода.
- **Тип**: `строка`
- **Пример**: `"#888888"`

##### Свойство `bold`

- **Описание**: Определяет, должен ли текст быть жирным.
- **Тип**: `логическое`
- **Возможные значения**: `true`, `false`.
- **Пример**: `true`

##### Свойство `fontSize`

- **Описание**: Размер шрифта текста заголовка в пунктах.
- **Тип**: `строка`
- **Пример**: `"24"`
