# Модуль hypotez/src/endpoints/bots/telegram/__init__.py

## Обзор

Данный модуль предоставляет инициализацию и импорт для бота Telegram.

## Переменные

### `MODE`

**Описание**: Строковая переменная, определяющая режим работы бота (в данном случае, `'dev'` для разработки).

**Значение**: `'dev'`

## Импорты

### `TelegramBot`

**Описание**: Импортирует класс `TelegramBot` из файла `.bot`.


## Функции

(В данном файле нет функций)


```python
MODE = 'dev'

from .bot import TelegramBot
```
