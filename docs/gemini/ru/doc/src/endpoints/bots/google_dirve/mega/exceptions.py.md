# Модуль исключений для работы с Mega

## Обзор

Модуль содержит пользовательские исключения, используемые при работе с API Mega. Эти исключения позволяют обрабатывать специфические ошибки, которые могут возникнуть в процессе взаимодействия с сервисом Mega, такие как некорректный пароль или ошибки при выполнении запросов.

## Классы

### `MegaException`

**Описание**: Базовый класс для всех исключений, связанных с Mega.

**Методы**:
- Отсутствуют.

**Параметры**:
- Отсутствуют.

**Примеры**:
```python
raise MegaException('Произошла общая ошибка Mega')
```

### `MegaIncorrectPasswordExcetion`

**Описание**: Исключение, которое выбрасывается при вводе неверного пароля или email.

**Методы**:
- Отсутствуют.

**Параметры**:
- Отсутствуют.

**Примеры**:
```python
raise MegaIncorrectPasswordExcetion('Неверный пароль или email')
```

### `MegaRequestException`

**Описание**: Исключение, которое выбрасывается при возникновении ошибки в запросе к API Mega.

**Методы**:
- Отсутствуют.

**Параметры**:
- Отсутствуют.

**Примеры**:
```python
raise MegaRequestException('Ошибка при выполнении запроса к Mega')
```
