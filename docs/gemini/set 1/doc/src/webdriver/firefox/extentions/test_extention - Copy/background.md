# Модуль background.js

## Обзор

Этот модуль отвечает за обработку кликов по значку расширения в адресной строке браузера Firefox. При клике, скрипт `contentScript.js` выполняется в текущей вкладке.


## Функции

### `browser.browserAction.onClicked.addListener`

**Описание**: Функция `browser.browserAction.onClicked.addListener` регистрирует обработчик для событий клика по значку расширения.

**Параметры**:
- `(tab)` (object): Объект, содержащий информацию о вкладке, на которой был нажат значок.

**Возвращает**:
- (None): Функция не возвращает значений.

**Вызывает исключения**:
-  (None): В данном случае нет возможных исключений, так как это обработчик события.


## Скрипт `contentScript.js`

**Описание**: Скрипт `contentScript.js` (см. соответствующий файл).  Выполняется в текущей вкладке браузера при клике по значку расширения.
