# Модуль `hypotez/src/utils/string/validator.py`

## Обзор

Модуль `validator.py` предоставляет набор статических методов для валидации различных типов строк, таких как цена, вес, артикул и URL-адреса.  Он использует регулярные выражения, обработку исключений и методы из `urllib.parse` для обеспечения надежной валидации.

## Классы

### `ProductFieldsValidator`

**Описание**: Класс `ProductFieldsValidator` содержит статические методы для валидации различных полей, связанных с продуктами.

**Статические методы**:

#### `validate_price(price: str) -> bool`

**Описание**: Проверяет корректность строки, представляющей цену.

**Параметры**:

- `price` (str): Строка, представляющая цену.

**Возвращает**:

- `bool`: `True`, если строка является корректной ценой, иначе `None`.

**Обрабатываемые исключения**:

- `Exception`: Если при преобразовании строки в число возникает ошибка.

#### `validate_weight(weight: str) -> bool`

**Описание**: Проверяет корректность строки, представляющей вес.

**Параметры**:

- `weight` (str): Строка, представляющая вес.

**Возвращает**:

- `bool`: `True`, если строка является корректным весом, иначе `None`.

**Обрабатываемые исключения**:

- `Exception`: Если при преобразовании строки в число возникает ошибка.

#### `validate_sku(sku: str) -> bool`

**Описание**: Проверяет корректность строки, представляющей артикул (SKU).

**Параметры**:

- `sku` (str): Строка, представляющая артикул.

**Возвращает**:

- `bool`: `True`, если строка является корректным артикулом, иначе `None`.

**Обрабатываемые исключения**:

- Нет.


#### `validate_url(url: str) -> bool`

**Описание**: Проверяет корректность URL-адреса.

**Параметры**:

- `url` (str): Строка, представляющая URL-адрес.

**Возвращает**:

- `bool`: `True`, если строка является корректным URL-адресом, иначе `None`.

**Обрабатываемые исключения**:

- Нет.


#### `isint(s: str) -> bool`

**Описание**: Проверяет, является ли строка целым числом.

**Параметры**:

- `s` (str): Строка, которую нужно проверить.

**Возвращает**:

- `bool`: `True`, если строка является целым числом, иначе `None`.

**Обрабатываемые исключения**:

- `Exception`: Если при преобразовании строки в целое число возникает ошибка.
