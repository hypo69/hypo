# Модуль `src.suppliers.bangood`

## Обзор

Модуль `src.suppliers.bangood` предназначен для работы с поставщиком Banggood. Он включает в себя классы и функции для сбора данных с сайта Banggood, такие как категории товаров и сами товары.
