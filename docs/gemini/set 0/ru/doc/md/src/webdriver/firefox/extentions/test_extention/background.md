# Документация для background.js

## Обзор

Файл `background.js` отвечает за обработку событий клика по значку расширения. При нажатии значка, скрипт выполняет JavaScript-код из файла `contentScript.js` в текущей вкладке.

## Функции

### `browser.browserAction.onClicked.addListener`

**Описание**:  Устанавливает обработчик события клика по значку расширения.

**Параметры**:

- `(tab)` (object): Объект, содержащий информацию о вкладке, по которой был произведен клик.  

**Вызывает исключения**:

- (Не указано). Возможно возникновение различных ошибок, связанных с взаимодействием с API браузера.
