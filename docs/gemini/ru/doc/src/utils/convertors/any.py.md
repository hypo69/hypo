# Модуль `any`

## Обзор

Модуль `any` содержит утилиты для преобразования данных различных типов в словари. Он включает функцию `any2dict`, которая рекурсивно преобразует любые типы данных в словарь.

## Подробнее

Этот модуль предоставляет способ преобразования сложных структур данных, таких как объекты классов, списки, множества и другие типы данных, в словари. Это может быть полезно для сериализации данных, логирования или других задач, где требуется представление данных в виде словаря.

## Функции

### `any2dict`

```python
def any2dict(any_data: Any) -> dict | list | str | int | float | bool | None | list[Any] | False:
    """
    Рекурсивно преобразует любой тип данных в словарь.

    Args:
        any_data (Any): Любой тип данных.

    Returns:
        dict | list | str | int | float | bool | None | list[Any] | False: 
        Словарь, представляющий входные данные, список, базовый тип данных или `False`, если преобразование невозможно.

    Как работает функция:
    Функция `any2dict` рекурсивно преобразует входные данные любого типа в словарь.
    - Если входные данные являются экземпляром класса, функция пытается преобразовать его атрибуты (__dict__) в словарь.
    - Если входные данные являются словарем, функция рекурсивно преобразует его ключи и значения.
    - Если входные данные являются списком или кортежем, функция рекурсивно преобразует каждый элемент списка.
    - Если входные данные являются множеством, функция рекурсивно преобразует каждый элемент множества.
    - Если входные данные относятся к базовым типам данных (int, float, str, bool, None), функция возвращает их без изменений.
    - Если тип данных не поддерживается, функция возвращает `False`.
    """
```

**Параметры**:

- `any_data` (Any): Данные любого типа, которые требуется преобразовать в словарь.

**Возвращает**:

- `dict | list | str | int | float | bool | None | list[Any] | False`: 
  - Словарь, представляющий входные данные.
  - Список, если входные данные были списком, кортежем или множеством.
  - Базовый тип данных (int, float, str, bool, None), если входные данные относились к этим типам.
  - `False`, если преобразование невозможно.

**Примеры**:

```python
import types

# Пример 1: Словарь
data1 = {
    "name": "John",
    "age": 30,
    "address": {
        "city": "New York",
        "street": "Main St",
        "numbers": [1, 2, 3]
    },
    "phones": ["123-456-7890", "987-654-3210"],
    "skills": {"python", "java", "c++"}
}
print(any2dict(data1))
# Вывод: {'name': 'John', 'age': 30, 'address': {'city': 'New York', 'street': 'Main St', 'numbers': [1, 2, 3]}, 'phones': ['123-456-7890', '987-654-3210'], 'skills': ['python', 'java', 'c++']}

# Пример 2: Список
data2 = [1, 2, "three", {"key": "value"}]
print(any2dict(data2))
# Вывод: [1, 2, 'three', {'key': 'value'}]

# Пример 3: Целое число
data3 = 123
print(any2dict(data3))
# Вывод: 123

# Пример 4: Строка
data4 = "string"
print(any2dict(data4))
# Вывод: string

# Пример 5: None
data5 = None
print(any2dict(data5))
# Вывод: None

# Пример 6: Объект класса
class MyClass:
    def __init__(self, x):
        self.x = x

data6 = MyClass(10)
print(any2dict(data6))
# Вывод: {}

# Пример 7: SimpleNamespace
data7 = types.SimpleNamespace(a=1, b='hello', c=[1, 2, 3])
print(any2dict(data7))
# Вывод: {'a': 1, 'b': 'hello', 'c': [1, 2, 3]}

# Пример 8: Вложенные структуры
data8 = {'a': 1, 'b': types.SimpleNamespace(x=2, y=3)}
print(any2dict(data8))
# Вывод: {'a': 1, 'b': {'x': 2, 'y': 3}}

# Пример 9: Список с SimpleNamespace
data9 = [types.SimpleNamespace(x=2), 3, 'str']
print(any2dict(data9))
# Вывод: [{'x': 2}, 3, 'str']

# Пример 10: SimpleNamespace с объектом класса
data10 = types.SimpleNamespace(a=1, b=MyClass(3))
print(any2dict(data10))
# Вывод: {'a': 1, 'b': ''}

# Пример 11: Словарь с объектом класса
data11 = {"a": 1, "b": MyClass(10)}
print(any2dict(data11))
# Вывод: {'a': 1, 'b': ''}
