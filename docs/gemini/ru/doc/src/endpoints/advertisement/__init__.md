# Модуль hypotez/src/endpoints/advertisement/__init__.py

## Обзор

Данный модуль предоставляет базовый уровень для работы с рекламными объявлениями. Он импортирует необходимые классы и функции из других модулей, в частности, из `facebook.py`.

## Переменные

### `MODE`

**Описание**:  Переменная, содержащая режим работы (например, 'dev', 'prod'). В данном случае используется строка 'dev'.

**Значение**: `'dev'`

## Модули

### `facebook`

**Описание**: Модуль, содержащий функции и классы для работы с рекламными объявлениями Facebook.
