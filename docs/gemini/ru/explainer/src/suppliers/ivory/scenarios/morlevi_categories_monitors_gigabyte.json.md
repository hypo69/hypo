### <алгоритм>
1. **Чтение JSON:**  Загружаем JSON файл `morlevi_categories_monitors_gigabyte.json`.
2. **Разбор JSON:** Парсим JSON-содержимое в структуру данных Python (словарь).
3. **Доступ к сценариям:** Получаем доступ к разделу "scenarios" (словарь).
4. **Итерация по сценариям:** Проходим циклом по каждому ключу (название сценария) и его значению (словарь параметров сценария).
5. **Извлечение данных:** Для каждого сценария извлекаем следующие данные:
    * `brand` (строка): Бренд производителя, всегда "GIGABYTE".
    * `url` (строка): URL-адрес страницы с товарами или текстовая строка-разделитель.
    * `checkbox` (логическое): Булево значение, всегда `false`.
    * `active` (логическое): Булево значение, всегда `true`.
    * `condition` (строка): Состояние товара, всегда `"new"`.
    * `presta_categories` (строка): Строка с ID категорий, разделенных запятыми, например, "127,128,980".
6. **Обработка данных (пример):** 
    * При обработке сценария "GIGABYTE 24-25":
        * `brand` = "GIGABYTE"
        * `url` = "https://www.morlevi.co.il/Cat/8?p_315=2&p_350=1807&sort=datafloat2%2Cprice&keyword="
        * `checkbox` = `false`
        * `active` = `true`
        * `condition` = "new"
        * `presta_categories` = "127,129,980"
7. **Использование данных:**  Извлеченные данные могут использоваться для:
    *   Формирования запросов к веб-сайту для сбора данных.
    *   Настройки фильтров при парсинге товаров.
    *   Обновления информации в базе данных.
    *   Классификации товаров по категориям в `Prestashop`.
