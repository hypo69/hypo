### Блок кода 1

```rst
Как использовать блок кода
=========================================================================================

Описание
-------------------------
Этот блок кода выполняет проверку на валидность значения переменной `value` перед дальнейшей обработкой.

Шаги выполнения
-------------------------
1. Получение значения переменной `value` через вызов метода `get_value()`.
2. Проверка, является ли значение `value` пустым или `None`. Если да, то логируется сообщение об ошибке, и выполнение кода прекращается.
3. Проверка, является ли `value` корректным типом данных (например, целое число, строка). Если нет, то логируется сообщение об ошибке и выполнение кода прекращается.
4. Если значение `value` валидно, оно передается в следующую функцию для дальнейшей обработки.

Пример использования
-------------------------
Пример того, как можно использовать данный блок кода:

.. code-block:: python

    def get_value():
        # ... код для получения значения ...
        return 10

    def process_value(value):
        # ... код для обработки значения ...
        print(f"Обработанное значение: {value}")

    value = get_value()
    if value is not None:
        try:
            if not isinstance(value, int):
                raise TypeError("Некорректный тип данных")
            process_value(value)
        except TypeError as ex:
            print(f"Ошибка: {ex}")
    else:
        print("Ошибка: Значение отсутствует")
```

```rst
Как использовать блок кода
=========================================================================================

Описание
-------------------------
Этот блок кода выполняет отправку HTTP-запроса.

Шаги выполнения
-------------------------
1. Формирование HTTP-запроса с заданными параметрами.
2. Вызов функции `send_request` для отправки запроса.
3. Обработка ответа от сервера.

Пример использования
-------------------------
Пример того, как можно использовать данный блок кода:

.. code-block:: python

    import requests

    def send_request(url, params):
        try:
            response = requests.get(url, params=params)
            response.raise_for_status()  # Проверяет статус ответа
            return response.json()
        except requests.exceptions.RequestException as ex:
            print(f"Ошибка при отправке запроса: {ex}")
            return None


    url = "https://api.example.com/data"
    params = {"key": "value"}
    data = send_request(url, params)

    if data:
        # Обработка полученных данных
        print(data)
```
