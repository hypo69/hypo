# Задача No 452
## Расположение ребер
## Условия
Пусть C(n,k) обозначает биномиальный коэффициент "n выбрать k".

Считается, что k-мерный гиперкуб имеет 2<sup>k</sup> вершин, и из каждой вершины исходит k ребер.
Пусть Q(n,k) обозначает количество способов расположить n непересекающихся ребер в k-мерном гиперкубе.
Например, Q(4,2) = 24, Q(4,3) = 96, Q(5,2) = 480, Q(5,3) = 3840.

Пусть

$F(n,k) = \sum_{i=0}^{n} Q(i,k)$

Например, F(5,3) = 11760.

Найдите $F(10^6,10)$.
Ответ дайте по модулю 1000000007.
