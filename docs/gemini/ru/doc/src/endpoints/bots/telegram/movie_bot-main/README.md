# Инструкция по работе с Telegram ботом для поиска фильмов и сериалов

## Обзор

Этот документ предоставляет руководство для разработчиков по работе с Telegram ботом, предназначенным для поиска ссылок на бесплатный просмотр фильмов и сериалов. Бот создан с использованием библиотеки Aiogram, сервисов Google для ассоциативного поиска, библиотеки Кинопоиска и сервиса [w2.kpfr/wiki](https://w2.kpfr.wiki/). В боте реализована защита от флуда через middlewares, а для парсинга используется BeautifulSoup.

## Подробнее

Этот бот позволяет пользователям Telegram находить ссылки на бесплатный просмотр фильмов и сериалов. Он использует несколько внешних источников, таких как Google, Кинопоиск и w2.kpfr/wiki, для поиска релевантных ссылок. Защита от флуда реализована для предотвращения злоупотреблений.

## Быстрый старт

Следуйте этим шагам, чтобы запустить бота:

1.  Склонируйте проект на свой компьютер.
2.  Создайте файл `.env` и наполните его в предложенном формате:

```
TOKEN=123456789 # Токен вашего telegram бота
```

3.  Если у вас нет виртуального окружения, создайте и активируйте его:

```shell
python -m venv venv
```

```shell
venv\\Scripts\\activate.bat
```

4.  Обновите pip и установите необходимые зависимости:

```shell
pip install --upgrade pip
```

```shell
pip install -r requirements.txt
```

5.  Запустите `run.py`:

```shell
python run.py
```
