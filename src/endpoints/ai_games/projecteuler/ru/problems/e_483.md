# Задача No 483
## Сумма минимальных произведений
## Условия
Рассмотрим множество S(n) всех произведений чисел, взятых из {1, 2, ..., n} в не более чем двух экземплярах.
Например, S(4) = {1, 2, 3, 4, 2 * 3, 2 * 4, 3 * 4, 1 * 2, 1 * 3, 1 * 4}.
Определим f(n) как сумму всех элементов в S(n), отсортированных в порядке возрастания.
Например:
S(4) = {1, 2, 3, 4, 6, 8, 12, 2, 3, 4}, тогда f(4) = 1 + 2 + 2 + 3 + 3 + 4 + 4 + 6 + 8 + 12 = 45.

Пусть g(n) = Σ f(k) для 1 ≤ k ≤ n.
Так, g(4) = f(1) + f(2) + f(3) + f(4) = 1 + 5 + 18 + 45 = 69.

Найдите g(10^8) mod 1 000 000 007.
