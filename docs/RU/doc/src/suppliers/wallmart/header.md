# Модуль hypotez/src/suppliers/wallmart/header.py

## Обзор

Данный модуль содержит вспомогательные функции для работы с проектом, такие как определение корневой директории проекта и загрузку настроек из файла settings.json. Также он предоставляет доступ к переменным, описывающим проект (имя, версия, документация, автор, лицензия и т.д.).

## Функции

### `set_project_root`

**Описание**: Определяет корневую директорию проекта, начиная от текущего файла. Поиск осуществляется вверх по директориям до тех пор, пока не будет найдена директория, содержащая один из указанных маркеров (pyproject.toml, requirements.txt, .git). Если корневая директория найдена, то добавляется в `sys.path` для импорта модулей из других директорий.

**Параметры**:

- `marker_files` (tuple): Кортеж с названиями файлов или директорий, используемых для определения корневой директории проекта. По умолчанию содержит ('pyproject.toml', 'requirements.txt', '.git').

**Возвращает**:

- `Path`: Объект `Path` представляющий корневую директорию проекта. Если корневая директория не найдена, возвращает директорию, где находится текущий файл.

**Вызывает исключения**:

- Нет.


## Переменные

### `__root__`

**Описание**: Содержит путь к корневой директории проекта, полученный вызовом функции `set_project_root()`.

**Тип**: `Path`

**Значение**: Путь к корневой директории проекта.


### `settings`

**Описание**: Словарь с настройками проекта, загруженными из файла `src/settings.json`.

**Тип**: `dict`

**Значение**: Словарь с настройками. Может быть `None`, если файл `settings.json` не найден или содержит некорректный JSON.


### `doc_str`

**Описание**:  Содержит строковое представление документации проекта, загруженное из файла `README.MD`.

**Тип**: `str`

**Значение**: Строковое представление документации. Может быть `None`, если файл `README.MD` не найден или содержит ошибку.


### `__project_name__`

**Описание**: Название проекта.

**Тип**: `str`

**Значение**: Значение из `settings.json['project_name']` или 'hypotez', если ключ не найден или `settings` равен `None`.

### `__version__`

**Описание**: Версия проекта.

**Тип**: `str`

**Значение**: Значение из `settings.json['version']` или пустая строка, если ключ не найден или `settings` равен `None`.


### `__doc__`

**Описание**: Документация проекта.

**Тип**: `str`

**Значение**: Содержимое файла `README.MD` или пустая строка, если файл не найден или содержит ошибку.


### `__details__`

**Описание**: Подробные сведения о проекте.

**Тип**: `str`

**Значение**: Пустая строка.


### `__author__`

**Описание**: Автор проекта.

**Тип**: `str`

**Значение**: Значение из `settings.json['author']` или пустая строка, если ключ не найден или `settings` равен `None`.


### `__copyright__`

**Описание**: Авторские права.

**Тип**: `str`

**Значение**: Значение из `settings.json['copyright']` или пустая строка, если ключ не найден или `settings` равен `None`.


### `__cofee__`

**Описание**: Ссылка для поддержки разработчика.

**Тип**: `str`

**Значение**: Значение из `settings.json['cofee']` или строка с ссылкой, если ключ не найден или `settings` равен `None`.
