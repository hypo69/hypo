### `calculate_sum`

**Описание**: Функция суммирует два числа.

- `a` (number): Первое слагаемое.

**Возвращает**:

- `number`: Сумма чисел `a` и `b`.

**Вызывает исключения**:

- `TypeError`: Если входные данные `a` или `b` не являются числами.

### Алгоритм `calculate_sum`

1. Принимаются на вход два аргумента, `a` и `b`.
2. Проверяется, являются ли `a` и `b` числами. Если нет, генерируется исключение `TypeError`.
3. Вычисляется сумма `a` и `b`.
4. Возвращается вычисленная сумма.

**Пример**:

```
calculate_sum(5, 3) -> 8
```

```python
# Пример использования функций
def calculate_sum(a, b):
    """
    Args:
        a (number): Первое слагаемое.
        b (number): Второе слагаемое.

    Returns:
        number: Сумма чисел a и b.

    Raises:
        TypeError: Если входные данные a или b не являются числами.
    """
    if not isinstance(a, (int, float)) or not isinstance(b, (int, float)):
        raise TypeError("Входные данные должны быть числами")
    return a + b
```
