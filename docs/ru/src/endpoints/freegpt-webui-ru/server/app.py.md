# Модуль инициализации Flask приложения

## Обзор

Модуль содержит код для инициализации Flask приложения. В частности, создает экземпляр класса `Flask` и определяет путь к папке с HTML-шаблонами.

## Подробнее

Этот код является отправной точкой для веб-приложения, созданного с использованием фреймворка Flask. Он создает экземпляр Flask, который затем будет использоваться для определения маршрутов, обработки запросов и рендеринга веб-страниц. Указание `template_folder` позволяет Flask находить HTML-шаблоны для рендеринга.

## Функции

### `Flask`

```python
from flask import Flask

app = Flask(__name__, template_folder='./../client/html')
```

**Описание**: Инициализация Flask приложения.

**Параметры**:
- `__name__` (str): Специальная переменная Python, которая указывает на имя текущего модуля.
- `template_folder` (str): Путь к папке, содержащей HTML-шаблоны. В данном случае `./../client/html` указывает на папку `html`, расположенную на один уровень выше текущей папки (`server`) в директории `client`.

**Возвращает**:
- `Flask`: Объект Flask приложения.

**Как работает**:

1.  Импортирует класс `Flask` из библиотеки `flask`.
2.  Создает экземпляр класса `Flask`, передавая имя текущего модуля (`__name__`) и путь к папке с шаблонами (`./../client/html`).

**Примеры**:

```python
from flask import Flask

app = Flask(__name__, template_folder='./../client/html')
```
В этом примере создается базовое Flask приложение, готовое к настройке маршрутов и обработке запросов.
