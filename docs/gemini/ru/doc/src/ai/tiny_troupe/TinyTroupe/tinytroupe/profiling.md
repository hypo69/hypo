# Модуль `profiling`

## Обзор

Модуль `profiling` предоставляет функции для анализа характеристик агентов, таких как распределение возраста, типичные интересы и т.д.  Методы построения графиков возвращают Pandas DataFrame с данными, использованными для построения.

## Функции

### `plot_age_distribution`

**Описание**: Строит гистограмму распределения возраста агентов.

**Параметры**:
- `agents` (List[TinyPerson]): Список агентов, для которых нужно построить распределение возраста.
- `title` (str, optional): Заголовок графика. По умолчанию "Распределение возраста".
- `show` (bool, optional): Показывать ли график. По умолчанию True.

**Возвращает**:
- `pd.DataFrame`: DataFrame с данными, использованными для построения графика.

### `plot_interest_distribution`

**Описание**: Строит круговую диаграмму (pie chart) распределения интересов агентов.

**Параметры**:
- `agents` (List[TinyPerson]): Список агентов, для которых нужно построить распределение интересов.
- `title` (str, optional): Заголовок графика. По умолчанию "Распределение интересов".
- `show` (bool, optional): Показывать ли график. По умолчанию True.

**Возвращает**:
- `pd.DataFrame`: DataFrame с данными, использованными для построения графика.
