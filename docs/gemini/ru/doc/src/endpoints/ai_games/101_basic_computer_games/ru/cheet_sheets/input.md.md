# Функция `input()`

## Обзор

Этот документ описывает использование функции `input()` в Python для получения пользовательского ввода через текстовый интерфейс. Функция приостанавливает выполнение программы и возвращает введенное значение в виде строки.

## Синтаксис

```python
input([prompt])
```

**Параметры:**

- `prompt` (str, optional): Строка, отображаемая пользователю перед вводом. Может содержать инструкцию, например: `"Введите ваше имя: "`. По умолчанию `None`.

## Пример использования

```python
# Запрос имени у пользователя
name = input('Введите ваше имя: ')
print(f'Привет, {name}!')
```

**Результат выполнения:**

```
Введите ваше имя: Алексей
Привет, Алексей!
```

## Особенности

### Возвращает строку

Все данные, введённые через `input()`, интерпретируются как строки. Если требуется число, необходимо преобразовать строку в число:

```python
age = int(input('Введите ваш возраст: '))
print(f'Ваш возраст: {age}')
```

### Обработка ошибок

Для избежания ошибок преобразования (например, если пользователь ввёл текст вместо числа) можно использовать блок `try-except`:

```python
try:
    number = int(input('Введите число: '))
    print(f'Вы ввели число {number}')
except ValueError:
    print('Ошибка: необходимо ввести число.')
```

### Использование в циклах

Часто функция `input()` используется в циклах для повторного запроса данных:

```python
while True:
    text = input('Введите что-то (или "выход" для завершения): ')
    if text.lower() == 'выход':
        print('Завершение программы.')
        break
    print(f'Вы ввели: {text}')
```

## Советы для начинающих

- Убедитесь, что тип данных соответствует вашим ожиданиям (например, преобразуйте ввод в число, если нужно).
- Всегда проверяйте данные для предотвращения ошибок ввода.
- Используйте понятные и чёткие `prompt`-сообщения, чтобы пользователь понимал, что от него требуется.
