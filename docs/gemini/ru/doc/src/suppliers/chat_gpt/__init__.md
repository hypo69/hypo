# Модуль `hypotez/src/suppliers/chat_gpt/__init__.py`

## Обзор

Этот модуль предоставляет инициализацию для поставщика данных ChatGPT. Он содержит константу `MODE` и импортирует класс `GptGs` из модуля `gsheet`.

## Константы

### `MODE`

**Описание**: Переменная, определяющая режим работы, вероятно, 'dev' по умолчанию.

**Значение**: `'dev'`


## Модули

### `gsheet`

**Описание**: Модуль, содержащий класс для работы с Google Sheets.


## Импорты

### `GptGs`

**Описание**: Импортирует класс `GptGs` из модуля `gsheet`, который, вероятно, реализует логику взаимодействия с ChatGPT через Google Sheets.
