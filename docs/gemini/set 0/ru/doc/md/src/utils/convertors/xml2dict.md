# Модуль hypotez/src/utils/convertors/xml2dict.py

## Обзор

Модуль `xml2dict` предоставляет функции для конвертации XML-данных в словари.  Он включает в себя функции для парсинга XML-строк и преобразования XML-элементных деревьев в представления словарей.

## Функции

### `_parse_node`

**Описание**: Парсит XML-узел в словарь.

**Параметры**:
- `node` (ET.Element): XML-элемент для парсинга.

**Возвращает**:
- `dict | str`: Словарное представление XML-узла или строку, если у узла нет атрибутов или дочерних элементов.


### `_make_dict`

**Описание**: Создаёт новый словарь с именем тега и значением.

**Параметры**:
- `tag` (str): Имя тега XML-элемента.
- `value` (любой тип): Значение, связанное с тегом.

**Возвращает**:
- `dict`: Словарь с именем тега в качестве ключа и значением в качестве значения.


### `xml2dict`

**Описание**: Парсит XML-строку в словарь.

**Параметры**:
- `xml` (str): XML-строка для парсинга.

**Возвращает**:
- `dict`: Словарное представление XML-данных.


### `ET2dict`

**Описание**: Преобразует XML-элементное дерево в словарь.

**Параметры**:
- `element_tree` (ET.Element): XML-элементное дерево.

**Возвращает**:
- `dict`: Словарное представление XML-элементного дерева.


## Обработка исключений

Модуль обрабатывает `ImportError` при импорте `xml.etree.cElementTree`, переключаясь на `xml.etree.ElementTree` в случае необходимости.
