# Задача No 310
## Игра в ним с кучами
## Условия

Алиса и Боб играют в игру с кучами камней. Есть *n* куч, и они по очереди выбирают одну кучу и удаляют из нее любое ненулевое количество камней. Алиса ходит первой. Проигрывает игрок, который убирает последний камень.

Обозначим *f(n)* количество стартовых позиций, при которых Алиса имеет выигрышную стратегию, если *n* количество куч и размер каждой кучи не превышает *n*.

Например:
* f(1) = 1, поскольку единственная стартовая позиция - это 1 камень в 1 куче, и Алиса побеждает, убрав этот камень.
* f(2) = 7, потому что следующие 7 стартовых позиций являются выигрышными для Алисы:
    * [1,1]
    * [1,2]
    * [2,1]
    * [2,2]
    * [1]
    * [2]
    * []
* f(5) = 187.

Найдите *f(10<sup>7</sup>)*. Выведите результат по модулю 1234567891011.
