# Модуль `date_time`

## Обзор

Модуль `date_time` предоставляет функциональность для проверки, находится ли текущее время в заданном интервале, а также поддерживает проверку с таймаутом. Основная цель модуля - обеспечить возможность выполнения операций только в определенные периоды времени (например, для задач обслуживания в ночное время).  По умолчанию используется временной интервал с 23:00 до 06:00, и функция способна обрабатывать интервалы, пересекающие полночь. Также, модуль реализует функционал ожидания ввода от пользователя с таймаутом.

## Классы

### `TimeoutCheck`

**Описание**: Класс `TimeoutCheck` предназначен для проверки текущего времени на принадлежность заданному интервалу, а также для ожидания ввода пользователя с таймаутом.

**Методы**:
- `__init__`: Инициализирует экземпляр класса `TimeoutCheck`.
- `interval`: Проверяет, находится ли текущее время в заданном интервале.
- `interval_with_timeout`: Проверяет, находится ли текущее время в заданном интервале с таймаутом.
- `get_input`: Запрашивает ввод от пользователя.
- `input_with_timeout`: Ожидает ввод с таймаутом.

## Функции

### `interval`

**Описание**: Проверяет, находится ли текущее время в заданном интервале.

**Параметры**:
- `start` (time, optional): Время начала интервала. По умолчанию `time(23, 0)`.
- `end` (time, optional): Время конца интервала. По умолчанию `time(6, 0)`.

**Возвращает**:
- `bool`: `True`, если текущее время находится в интервале, иначе `False`.

### `interval_with_timeout`

**Описание**: Проверяет, находится ли текущее время в заданном интервале с таймаутом.

**Параметры**:
- `timeout` (int, optional): Время ожидания проверки в секундах. По умолчанию `5`.
- `start` (time, optional): Время начала интервала. По умолчанию `time(23, 0)`.
- `end` (time, optional): Время конца интервала. По умолчанию `time(6, 0)`.

**Возвращает**:
- `bool`: `True`, если текущее время находится в интервале и ответ получен в течение таймаута, иначе `False`.

### `get_input`

**Описание**: Запрашивает ввод от пользователя.

**Параметры**:
- Нет.

**Возвращает**:
- None. Сохраняет введенные данные в `self.user_input`.

### `input_with_timeout`

**Описание**: Ожидает ввод от пользователя с таймаутом.

**Параметры**:
- `timeout` (int, optional): Время ожидания ввода в секундах. По умолчанию `5`.

**Возвращает**:
- `str | None`: Введенная строка от пользователя или `None`, если произошел таймаут.
