# Модуль `hypotez/src/endpoints/prestashop/pricelist.py`

## Обзор

Модуль `hypotez/src/endpoints/prestashop/pricelist.py` предоставляет класс `PriceListRequester` для запроса и модификации цен товаров из Престашоп API. Класс наследуется от `PrestaShop`, что подразумевает использование базовых функций для работы с API.

## Классы

### `PriceListRequester`

**Описание**: Класс `PriceListRequester` предназначен для взаимодействия с API Престашоп для получения и изменения цен товаров. Он наследуется от базового класса `PrestaShop`, обеспечивая доступ к базовым функциям API.

**Методы**:

#### `__init__`

**Описание**: Инициализирует объект `PriceListRequester`.

**Параметры**:

- `api_credentials` (dict): Словарь, содержащий учетные данные для API, включая `api_domain` и `api_key`.

#### `request_prices`

**Описание**: Запрашивает список цен для указанных товаров.

**Параметры**:

- `products` (list): Список наименований товаров, для которых требуется получить цены.

**Возвращает**:

- dict | None: Словарь, где ключами являются названия товаров, а значениями - их цены. Возвращает `None` в случае ошибки.

#### `update_source`

**Описание**: Обновляет источник данных для запроса цен.

**Параметры**:

- `new_source` (object): Новый источник данных.


#### `modify_product_price`

**Описание**: Модифицирует цену указанного товара.

**Параметры**:

- `product` (str): Название товара.
- `new_price` (float): Новая цена товара.
