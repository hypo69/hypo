# Модуль `facebook`

## Обзор

Данный модуль содержит функции для работы с рекламными кампаниями на Facebook.

## Оглавление

* [Функции](#функции)


## Функции

### `create_campaign`

**Описание**: Создаёт новую рекламную кампанию на Facebook.

**Параметры**:
- `campaign_data` (dict): Словарь с данными для создания кампании.  Обязательные поля: `name`, `objective`, `budget`.  Дополнительные поля зависят от типа кампании.

**Возвращает**:
- `dict | None`: Словарь с результатами создания кампании, или `None` в случае ошибки.

**Вызывает исключения**:
- `FacebookAPIError`: Возникает при ошибках API Facebook.
- `ValueError`: Возникает при некорректных входных данных.


### `get_campaign_stats`

**Описание**: Получает статистику по рекламной кампании.

**Параметры**:
- `campaign_id` (str): Идентификатор кампании.

**Возвращает**:
- `dict | None`: Словарь со статистикой кампании, или `None` в случае ошибки.

**Вызывает исключения**:
- `FacebookAPIError`: Возникает при ошибках API Facebook.
- `ValueError`: Возникает при некорректных входных данных.


### `update_campaign_budget`

**Описание**: Обновляет бюджет рекламной кампании.

**Параметры**:
- `campaign_id` (str): Идентификатор кампании.
- `new_budget` (float): Новый бюджет кампании.

**Возвращает**:
- `dict | None`: Словарь с результатами обновления бюджета, или `None` в случае ошибки.

**Вызывает исключения**:
- `FacebookAPIError`: Возникает при ошибках API Facebook.
- `ValueError`: Возникает при некорректных входных данных.
