# Задача No 466
## Изометрические множества

## Условия
Обозначим через F(n) количество различных наборов из n точек на плоскости, таких что расстояние между любыми двумя точками является целым числом.

Наборы, которые можно получить один из другого при помощи вращения или отражения, считаются одинаковыми.

Например, F(3) = 4, наборы показаны ниже.
```
      (0,0)     (0,0)     (0,0)    (0,0)
      (3,0)     (4,0)     (4,0)    (5,0)
      (0,4)     (0,3)     (3,4)    (3,4)
```

Известно, что F(4) = 13, F(5) = 63 и F(100) mod 10<sup>9</sup> = 368028652.

Найдите F(10000) mod 10<sup>9</sup>.
