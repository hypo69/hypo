# Модуль hypotez/src/goog/drive/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и чтения документации из файла `README.MD`. Он предоставляет переменные, содержащие информацию о проекте, такие как имя, версия, описание и контактную информацию.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с текущей директории и двигаясь вверх по иерархии директорий. Останавливается на первой директории, содержащей файлы-маркеры (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для определения корневой директории проекта. По умолчанию включает ('pyproject.toml', 'requirements.txt', '.git').

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена, иначе — путь к директории, где находится текущий файл.

**Вызывает исключения**:

- Нет


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный функцией `set_project_root()`.

**Тип**: `Path`

## Загрузка настроек и документации

**Описание**: Загружает настройки из файла `settings.json` и читает содержимое файла `README.MD` в переменные `settings` и `doc_str` соответственно. Обрабатывает возможные исключения `FileNotFoundError` и `json.JSONDecodeError`.

**Переменные**:

- `settings`: Словарь настроек проекта.
- `doc_str`: Строка с содержимым файла `README.MD`.


## Переменные проекта

**Описание**: Переменные, содержащие информацию о проекте, такие как имя, версия, описание и контактная информация. Значения берутся из `settings`, если доступно, иначе используются значения по умолчанию.

**Переменные**:

- `__project_name__` (str): Имя проекта.
- `__version__` (str): Версия проекта.
- `__doc__` (str): Описание проекта (из README.MD).
- `__details__` (str): Дополнительная информация.
- `__author__` (str): Автор проекта.
- `__copyright__` (str): Авторские права.
- `__cofee__` (str): Ссылка для пожертвования разработчику.
