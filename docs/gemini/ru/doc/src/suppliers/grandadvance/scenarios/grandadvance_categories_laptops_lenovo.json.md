# Документация для `grandadvance_categories_laptops_lenovo.json`

## Обзор

Этот файл содержит JSON-конфигурацию для категорий ноутбуков Lenovo, предназначенную для парсинга сайта Grand Advance.
Он определяет различные модели ноутбуков Lenovo на основе их процессоров и размеров экрана, а также предоставляет информацию о том, как эти модели должны быть отфильтрованы на веб-сайте. 
Каждая запись в JSON определяет модель ноутбука, соответствующие параметры для фильтрации на сайте, а также  категории PrestaShop, к которым должна принадлежать модель.

## Структура JSON

JSON-файл представляет собой словарь, где каждый ключ является названием модели ноутбука Lenovo (например, "LENOVO 11.6 I3"). Значение каждого ключа – это словарь, содержащий параметры для парсинга и фильтрации.

## Описание ключей

### Общие ключи

-   `brand` (str): Марка ноутбука, всегда "LENOVO".
-   `url` (str): URL-адрес страницы, с которой нужно парсить ноутбуки.
-   `active` (bool): Флаг, указывающий, активна ли данная модель для парсинга (`true` - активна).
-    `condition` (str): Условие товара, всегда "new".
-   `presta_categories` (str): Список ID категорий PrestaShop, к которым принадлежит модель (разделены запятыми).

### Ключ "checkbox"

Этот ключ содержит параметры для фильтрации на веб-сайте Grand Advance.

#### Ключ "cpu"

-   `class` (str): CSS-класс для элементов, представляющих процессор.
-   `by` (str): Метод поиска элементов (всегда "css selector").
-   `value` (list): Список возможных значений процессора для фильтрации.

#### Ключ "screensize"

-   `class` (str): CSS-класс для элементов, представляющих размер экрана.
-   `by` (str): Метод поиска элементов (всегда "css selector").
-   `value` (list): Список возможных значений размера экрана для фильтрации.

## Примеры

```json
  "LENOVO 11.6 I3": {
    "brand": "LENOVO",
    "url": "https://www.grandadvance.co.il/default.aspx?g=products&a=list&tieId=654&manId=49",
    "checkbox": {
      "cpu": {
        "class": ".fSel",
        "by": "css selector",
        "value": [
          "CORE I3",
          "CORE I 3",
          "CORE i3",
          "CORE i 3",
          "Core I3",
          "Core I 3",
          "Core i3",
          "Core i 3",
          "I3",
          "I 3",
          "i3",
          "i 3"
        ]
      },
      "screensize": {
        "class": ".fSel",
        "by": "css selector",
        "value": [ "10.1", "10.2", "10.3", "10.4", "10.5", "10.9", "11", "10,1", "10,2", "10,3", "10,4", "10,5", "10,9" ]
      }

    },
    "active": true, "condition":"new","presta_categories": "2,3,4,989,48"
  },
```

В данном примере показана конфигурация для модели ноутбука "LENOVO 11.6 I3". 
- Он указывает, что бренд - "LENOVO". 
-  URL для парсинга указан как `https://www.grandadvance.co.il/default.aspx?g=products&a=list&tieId=654&manId=49`. 
-  Фильтрация происходит по значению процессора, указанным в `checkbox.cpu.value`, и размеру экрана, указанному в `checkbox.screensize.value`.
-  Также указано, что модель активна (`active: true`), условие `new` и она должна быть добавлена в PrestaShop категории `2,3,4,989,48`.
