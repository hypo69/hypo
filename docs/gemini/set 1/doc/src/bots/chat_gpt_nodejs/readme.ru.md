## Классы

### `ChatGPTBot`

**Описание**: Основной класс, представляющий бота ChatGPT. Он отвечает за обработку запросов пользователя и формирование ответов.

**Методы**:

* `start()`: Запускает бота, устанавливает слушатель событий и инициализирует необходимые ресурсы.
* `handleMessage(message: string): string | null`: Обрабатывает входное сообщение пользователя и генерирует ответ. Возвращает ответ бота или `null` в случае ошибки.
