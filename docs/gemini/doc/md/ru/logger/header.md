# Модуль `hypotez/src/logger/header.py`

## Обзор

Этот модуль отвечает за получение корневого пути проекта, а также зачитывает настройки из файла `settings.json` и описание из файла `README.MD` для использования в других модулях.  Он также определяет переменные, содержащие информацию о проекте (имя, версия, описание, автор и т.д.).

## Получение корневого пути проекта

### `get_project_root`

**Описание**: Функция находит корневой каталог проекта, начиная с текущей директории и идя вверх по дереву каталогов, пока не найдет директорию, содержащую один из файлов, указанных в параметре `marker_files`.

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или каталогов, которые используются для определения корневого каталога проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Объект `Path` представляющий путь к корневому каталогу проекта. В случае если корневой каталог не найден, возвращает путь к текущей директории.

**Вызывает исключения**:
- Нет.


## Чтение настроек

**Описание**: Данный раздел описывает процесс загрузки настроек проекта из файла `settings.json`.

**Детали**:
Файл настроек (`settings.json`) должен находиться в корневом каталоге проекта в подкаталоге `src`.  Функция пытается загрузить данные из этого файла используя `json.load()`. При возникновении ошибок (например, файл не найден или некорректный JSON), значение `settings` остаётся `None`.


## Чтение описания проекта

**Описание**:  Этот раздел описывает загрузку описания проекта из файла `README.MD`.

**Детали**:
Файл `README.MD` должен находиться в корневом каталоге проекта в подкаталоге `src`.  Функция пытается загрузить текст из этого файла. Если файл не найден или при чтении возникла ошибка, то `doc_str` сохраняет значение `None`.


## Переменные проекта

**Описание**:  Этот раздел описывает переменные, содержащие информацию о проекте.


### `__root__`

**Описание**: Переменная, хранящая объект `Path` представляющий корневой путь к проекту. Вычисляется с помощью функции `get_project_root`.

**Тип**: `Path`


### `MODE`

**Описание**: Переменная, хранящая строку, представляющую режим работы приложения. В данном случае, это строка `'development'`.

**Тип**: `str`


### `__project_name__`

**Описание**:  Имя проекта. Получается из `settings.json`, или имеет значение `'hypotez'` по умолчанию.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта. Получается из `settings.json`, или пустая строка по умолчанию.

**Тип**: `str`


### `__doc__`

**Описание**: Описание проекта, загруженное из `README.MD`


**Тип**: `str`


### `__details__`

**Описание**:  Дополнительные детали проекта.


**Тип**: `str`


### `__author__`

**Описание**: Автор проекта.


**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права проекта.


**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка на донат для автора.


**Тип**: `str`

### `settings`

**Описание**: Словарь с настройками, загруженный из `settings.json`.


**Тип**: `dict` или `NoneType`
