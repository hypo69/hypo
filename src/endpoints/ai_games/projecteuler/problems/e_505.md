# Задача No 505
## Генератор случайных чисел Дирихле
## Условия

Определим последовательность случайных чисел Дирихле, как описано ниже:

Пусть $a_0, a_1, a_2, \dots$ будет последовательностью положительных целых чисел. 
Определим последовательность случайных чисел $x_n$ следующим образом:
* $x_0 = 0$
* $x_n = x_{n-1} + \frac{1}{a_n}$ для $n>0$

Пусть $P_N(x)$ будет вероятностью того, что $x_N \le x$.
Известно, что $P_N(x)$ сходится к непрерывной функции $P(x)$ при $N\to\infty$.

Сгенерируем $a_n$ следующим образом:
* $a_0 = 1$
* $a_n = \left(a_{n-1}^2 \mod 1000000 + 3\right)$ для $n>0$

Например, первые 10 значений $a_i$ равны: 1, 4, 19, 364, 132499, 462967, 487999, 132736, 115753, 76776.

Найдите $P(4)$. Дайте ваш ответ в виде строки с точностью 10 знаков после запятой.

**Пример:**
Для $N=1$ вероятность $x_1 \le 1$ равна $1$, так как $x_1 = 0 + \frac{1}{a_1} = \frac{1}{4} < 1$. Для $N=1$ вероятность $x_1 \le 0.1$ равна $0$, так как $x_1 = \frac{1}{4} > 0.1$.
