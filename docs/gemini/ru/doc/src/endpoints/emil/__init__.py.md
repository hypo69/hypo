# Модуль `src.endpoints.emil`

## Обзор

Модуль `src.endpoints.emil` предоставляет функциональность, связанную с эндпоинтами для работы с `Emil`. В настоящее время содержит определение константы `MODE` и импортирует класс `EmilDesign`.

## Константы

### `MODE`

- **Описание**: Определяет режим работы модуля.
- **Значение**: `'dev'` (режим разработки).

## Импорты

### `EmilDesign`

- **Описание**: Импортирует класс `EmilDesign` из модуля `emil_design`, который предположительно используется для работы с эндпоинтами.
