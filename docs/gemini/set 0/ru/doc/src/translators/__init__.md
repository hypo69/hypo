# Модуль hypotez/src/translators

## Обзор

Данный модуль содержит инициализацию для подмодулей переводчиков.

## Константы

### `MODE`

**Описание**: Переменная, определяющая режим работы системы (например, 'dev', 'prod').

**Значение**: `'dev'`

##  Структура

Этот модуль не содержит классов или функций.  Он инициализирует переменные и/или импортирует другие модули, необходимые для работы остальных переводчиков.
