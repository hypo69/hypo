# Модуль header

## Обзор

Модуль `header.py` предназначен для определения и инициализации основных параметров проекта `hypotez`. Он включает в себя функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и чтения документации из файла `README.MD`.

## Подорбней

Этот модуль играет важную роль в инициализации проекта, обеспечивая доступ к основным настройкам и информации о проекте. Он определяет корневую директорию проекта, загружает настройки из `settings.json`, а также считывает документацию из `README.MD`. Это позволяет другим модулям легко получать доступ к информации о проекте, такой как имя проекта, версия, автор и т. д.

## Функции

### `set_project_root`

```python
def set_project_root(marker_files: tuple = ('__root__', '.git')) -> Path:
    """
    Args:
        marker_files (tuple): Filenames or directory names to identify the project root.
    
    Returns:
        Path: Path to the root directory if found, otherwise the directory where the script is located.
    """
```

**Описание**: Определяет корневую директорию проекта, начиная поиск с директории текущего файла и двигаясь вверх до первого каталога, содержащего один из указанных маркерных файлов.

**Параметры**:
- `marker_files` (tuple): Список имен файлов или каталогов, используемых для определения корневой директории проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена, иначе - директория, в которой расположен скрипт.

**Примеры**:

```python
from pathlib import Path
root_path = set_project_root()
print(f'Root path: {root_path}')
```

## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта.

### `settings`

**Описание**: Словарь с настройками проекта, загруженными из файла `settings.json`.

### `__project_name__`

**Описание**: Имя проекта, полученное из настроек или значение по умолчанию `hypotez`.

### `__version__`

**Описание**: Версия проекта, полученная из настроек.

### `__doc__`

**Описание**: Строка документации проекта, считанная из файла `README.MD`.

### `__details__`

**Описание**: Дополнительная информация о проекте. В данном коде всегда пустая строка.

### `__author__`

**Описание**: Имя автора проекта, полученное из настроек.

### `__copyright__`

**Описание**: Информация об авторских правах проекта, полученная из настроек.

### `__cofee__`

**Описание**: Сообщение с предложением поддержать разработчика чашкой кофе, полученное из настроек или значение по умолчанию.
