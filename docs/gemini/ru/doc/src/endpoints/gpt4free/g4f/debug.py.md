# Модуль отладки g4f

## Обзор

Модуль предоставляет функции для отладки и логирования в библиотеке g4f. Он позволяет включать/выключать логирование, проверять версии, обрабатывать логи и ошибки.

## Подробнее

Этот модуль предоставляет инструменты для упрощения отладки и мониторинга работы библиотеки g4f. Он позволяет настраивать логирование, обрабатывать ошибки и проверять версии, что может быть полезно при разработке и отладке приложений, использующих g4f.

## Функции

### `log`

```python
def log(*text: Any, file: Optional[Any] = None) -> None:
    """Log a message if logging is enabled."""
```

**Назначение**: Логирует сообщение, если включено логирование.

**Параметры**:
- `*text` (Any): Переменное количество аргументов, представляющих сообщение для логирования.
- `file` (Optional[Any], optional): Файловый объект, в который будет записано сообщение. По умолчанию `None`.

**Возвращает**:
- `None`

**Как работает функция**:
1. Проверяет, включено ли логирование с помощью переменной `logging`.
2. Если логирование включено, вызывает функцию `log_handler` с сообщением и файлом (если указан).

**Примеры**:

```python
logging = True
log("Это сообщение будет залогировано")

logging = False
log("Это сообщение не будет залогировано")
```

### `error`

```python
def error(*error: Any, name: Optional[str] = None) -> None:
    """Log an error message to stderr."""
```

**Назначение**: Логирует сообщение об ошибке в stderr.

**Параметры**:
- `*error` (Any): Переменное количество аргументов, представляющих сообщение об ошибке.
- `name` (Optional[str], optional): Имя ошибки. По умолчанию `None`.

**Возвращает**:
- `None`

**Как работает функция**:
1. Преобразует все элементы `error` в строки. Если элемент не является строкой, он форматируется как `"{type(e).__name__ if name is None else name}: {e}"`.
2. Вызывает функцию `log` с отформатированным сообщением об ошибке и `sys.stderr` в качестве файла.

**Примеры**:

```python
error("Произошла ошибка")
error("Ошибка", name="MyError")
try:
    raise ValueError("Неверное значение")
except ValueError as ex:
    error(ex)
