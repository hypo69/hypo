# `store.json`

## Обзор

Файл `store.json` содержит определения локаторов для различных элементов веб-страницы, используемых для взаимодействия с магазином поставщика KSP.

2. [Структура](#Структура)

## Объекты

### `store`

**Описание**: Объект `store` содержит в себе локаторы для элементов, относящихся к магазину.

#### `store > main_header`

**Описание**: Локаторы для главного заголовка магазина.

  - `logo_link`: Локатор для ссылки на логотип магазина.

#### `store > product_list`

**Описание**: Локаторы для списка продуктов.

   - `product_item`: Локатор для отдельного элемента продукта в списке.
   - `product_name`: Локатор для названия продукта.
   - `product_price`: Локатор для цены продукта.
   - `product_card_link`: Локатор для ссылки на карточку продукта.
   - `next_page_button`: Локатор для кнопки "следующая страница" в списке продуктов.
   - `current_page_button`: Локатор для текущей страницы в списке продуктов.
   - `previous_page_button`: Локатор для кнопки "предыдущая страница" в списке продуктов.

#### `store > product_card`

**Описание**: Локаторы для карточки товара.

   - `product_title`: Локатор для заголовка товара.
   - `product_price`: Локатор для цены товара в карточке.
   - `product_attributes`: Локаторы для атрибутов товара.
      - `item`: Локатор для отдельного атрибута.
      - `item_name`: Локатор для имени атрибута.
      - `item_value`: Локатор для значения атрибута.
   - `product_image`: Локатор для изображения товара.
    - `add_to_cart_button`: Локатор для кнопки добавления в корзину.

#### `store > cart`

**Описание**: Локаторы для корзины.

   - `cart_items`: Локатор для всех элементов корзины.
     - `item`: Локатор для отдельного элемента корзины.
     - `item_title`: Локатор для заголовка товара в корзине.
     - `item_price`: Локатор для цены товара в корзине.
     - `item_quantity`: Локатор для количества товара в корзине.
   - `cart_total`: Локатор для общей суммы корзины.
   - `clear_cart_button`: Локатор для кнопки очистки корзины.
    - `checkout_button`: Локатор для кнопки оформления заказа.
    - `cart_popup`: Локатор для попапа корзины.

#### `store > checkout`

**Описание**: Локаторы для страницы оформления заказа.

   - `checkout_form`: Локатор для формы оформления заказа.
     - `name`: Локатор для поля имени.
     - `last_name`: Локатор для поля фамилии.
     - `email`: Локатор для поля электронной почты.
     - `phone`: Локатор для поля номера телефона.
     - `shipping_address`: Локатор для поля адреса доставки.
     - `comment`: Локатор для поля комментария к заказу.
     - `payment_method`: Локатор для поля выбора метода оплаты.
     - `delivery_method`: Локатор для поля выбора метода доставки.
     - `submit_button`: Локатор для кнопки подтверждения заказа.
   - `checkout_success_message`: Локатор для сообщения об успешном оформлении заказа.
   - `checkout_error_message`: Локатор для сообщения об ошибке при оформлении заказа.
