# `store.json`

## Обзор

Файл `store.json` содержит JSON-объект с локаторами для элементов страницы магазина Gearbest. Локаторы используются для автоматизации взаимодействия с элементами веб-страницы при помощи Selenium.

## Локаторы

### Главная страница

#### Кнопка 'Принять'

**Описание**: Локатор для кнопки "Принять" всплывающего окна cookie.

```json
"home_page_accept_button": {
  "by": "xpath",
  "value": "//div[@class='cookies-policy-button']/a"
}
```

#### Всплывающее окно выбора региона

**Описание**: Локатор для всплывающего окна выбора региона.

```json
"home_page_region_popup": {
  "by": "xpath",
  "value": "//div[@class='region-popup-wrap']"
}
```

#### Кнопка 'Отмена' всплывающего окна

**Описание**: Локатор для кнопки "Отмена" во всплывающем окне выбора региона.

```json
"home_page_region_popup_cancel": {
  "by": "xpath",
  "value": "//div[@class='region-popup-wrap']//a[@class='region-popup-close']"
}
```

#### Кнопка 'Мой аккаунт'

**Описание**: Локатор для кнопки "Мой аккаунт" в шапке сайта.

```json
"home_page_my_account_button": {
  "by": "xpath",
  "value": "//a[@class='my_account']"
}
```

#### Кнопка 'Вход'

**Описание**: Локатор для кнопки "Вход" в выпадающем меню "Мой аккаунт".

```json
"home_page_login_button": {
  "by": "xpath",
  "value": "//div[@class='my_account_menu']//a[@data-type='login']"
}
```
