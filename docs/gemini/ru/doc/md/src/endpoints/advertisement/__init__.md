# Модуль `hypotez/src/endpoints/advertisement/__init__.py`

## Обзор

Этот модуль предоставляет инициализацию для подмодулей, связанных с рекламой.  Он содержит константу `MODE` и импорты для подмодуля `Facebook`.

## Константы

### `MODE`

**Описание**:  Переменная, определяющая режим работы приложения (`dev`).

## Подмодули

### `facebook`

**Описание**: Подмодуль, содержащий функции и классы, связанные с рекламной платформой Facebook.


## Импорты

### `from .facebook import Facebook`

**Описание**: Импортирует класс `Facebook` из подмодуля `facebook`.
