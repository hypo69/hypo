# Модуль example_list

## Обзор

Этот модуль содержит примеры работы с списками в Python.

## Функции

### `pprint_list`

**Описание**: Выводит список в удобочитаемом формате с помощью `pprint`.

**Параметры**:

- `input_list` (list): Входной список, который нужно отформатировать.


**Возвращает**:

- `None`: Данная функция не возвращает значений. Вывод осуществляется в `stdout`.


```
```python
def pprint_list(input_list: list) -> None:
    """
    Выводит список в удобочитаемом формате с помощью pprint.

    Args:
        input_list (list): Входной список, который нужно отформатировать.

    Returns:
        None: Данная функция не возвращает значений. Вывод осуществляется в stdout.
    """
    import pprint
    pprint.pprint(input_list)
