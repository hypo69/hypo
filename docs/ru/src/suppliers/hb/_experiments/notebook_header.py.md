# Модуль notebook_header

## Обзор

Модуль содержит вспомогательные импорты, переменные и функции, необходимые для запуска и работы с поставщиками (suppliers). Он предназначен для настройки окружения, импорта необходимых модулей и запуска поставщика с заданными параметрами.

## Подробней

Этот код используется для инициализации окружения, добавления необходимых путей в `sys.path`, импорта нужных модулей и запуска поставщика с указанными параметрами. Он обеспечивает возможность запуска поставщиков с определенными настройками, такими как префикс поставщика и локаль, что позволяет гибко управлять процессом сбора данных.

## Переменные

### `dir_root`

```python
dir_root : Path = Path (os.getcwd()[:os.getcwd().rfind('hypotez')+7])
```

**Описание**: Определяет корневую директорию проекта `hypotez` на основе текущей рабочей директории.
- Извлекает путь до корневой директории проекта, обрезая текущую рабочую директорию до символа "hypotez" (включительно).

### `dir_src`

```python
dir_src = Path (dir_root, 'src')
```

**Описание**: Определяет директорию `src` внутри корневой директории проекта.
- Формирует путь к директории `src`, объединяя корневую директорию `dir_root` и строку 'src'.

## Функции

### `start_supplier`

```python
def start_supplier(supplier_prefix, locale):
    """ Старт поставщика """
```

**Назначение**: Функция для запуска поставщика с заданным префиксом и локалью.

**Параметры**:
- `supplier_prefix` (str): Префикс поставщика.
- `locale` (str): Локаль поставщика.

**Возвращает**:
- `Supplier | str`: Возвращает экземпляр класса `Supplier` или сообщение об ошибке, если не заданы префикс и локаль.

**Как работает функция**:

1. **Проверка входных параметров**:
   - Проверяет, заданы ли `supplier_prefix` и `locale`. Если оба параметра отсутствуют, возвращает сообщение "Не задан сценарий и язык".
2. **Создание словаря параметров**:
   - Создает словарь `params`, содержащий `supplier_prefix` и `locale`.
3. **Инициализация и возврат экземпляра `Supplier`**:
   - Создает и возвращает экземпляр класса `Supplier` с переданными параметрами.

**Внутри функции происходят следующие действия и преобразования**:

A
|
-- B
|
C

Где:

- `A`: Проверка входных параметров.
- `B`: Создание словаря параметров.
- `C`: Инициализация и возврат экземпляра `Supplier`.

**Примеры**:
```python
supplier = start_supplier('hb', 'ru_RU')
```
```python
start_supplier(None, None) # Возвращает: "Не задан сценарий и язык"
```
