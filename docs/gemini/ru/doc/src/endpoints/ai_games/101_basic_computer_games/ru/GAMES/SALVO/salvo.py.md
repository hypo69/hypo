# SALVO

## Обзор

Игра "SALVO" представляет собой реализацию морского боя, в котором игрок пытается потопить корабли противника, размещенные на сетке 10x10. Игрок вводит координаты для стрельбы, и игра сообщает о попадании, промахе или потоплении корабля. Цель игры - потопить все корабли противника за минимальное количество ходов.

## Функции

### `create_board`

**Описание**: Создает игровое поле заданного размера, заполненное нулями.

**Параметры**:
- `size` (int): Размер игрового поля (например, 10 для поля 10x10).

**Возвращает**:
- `list[list[int]]`: Двумерный список (матрица), представляющий игровое поле. Все ячейки инициализированы значением `0`.

### `place_ships`

**Описание**: Размещает корабли на игровом поле случайным образом.

**Параметры**:
- `board` (list[list[int]]): Игровое поле, на котором размещаются корабли.
- `ships_lengths` (list[int]): Список длин кораблей, которые нужно разместить.

**Возвращает**:
- `list[tuple[int, int, str, int]]`: Список кортежей, где каждый кортеж представляет данные о корабле: (row, col, orientation, length).
    - `row` (int):  начальная строка корабля.
    - `col` (int): начальный столбец корабля.
    - `orientation` (str):  ориентация корабля (`'horizontal'` или `'vertical'`).
    - `length` (int): длина корабля.

### `is_sunk`

**Описание**: Проверяет, потоплен ли корабль.

**Параметры**:
- `board` (list[list[str | int]]): Игровое поле.
- `ship` (tuple[int, int, str, int]): Кортеж с данными о корабле: (row, col, orientation, length).
    - `row` (int):  начальная строка корабля.
    - `col` (int): начальный столбец корабля.
    - `orientation` (str):  ориентация корабля (`'horizontal'` или `'vertical'`).
    - `length` (int): длина корабля.

**Возвращает**:
- `bool`: `True`, если корабль потоплен (все его части помечены как 'hit'), иначе `False`.

### `print_board`

**Описание**: Выводит игровое поле в консоль, скрывая расположение кораблей.

**Параметры**:
- `board` (list[list[str | int]]): Игровое поле для вывода.

**Возвращает**:
- `None`: Функция ничего не возвращает, а выводит данные в консоль.

### `play_salvo`

**Описание**: Основная функция игры Salvo, управляющая игровым процессом.

**Параметры**:
- `None`: Функция не принимает параметров.

**Возвращает**:
- `None`: Функция ничего не возвращает, но выводит результаты игры в консоль.
