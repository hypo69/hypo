# Модуль `hypotez/src/suppliers/aliexpress/scenarios/__init__.py`

## Обзор

Данный модуль содержит сценарии для работы с сайтом AliExpress. В нём определены константы и импортированы необходимые функции из других файлов.

## Константы

### `MODE`

**Описание**:  Переменная, определяющая режим работы (например, 'dev', 'prod'). В данном случае она имеет значение 'dev'.


## Модули
