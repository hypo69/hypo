# Модуль `src.suppliers.morlevi.graber`

## Обзор

Модуль `src.suppliers.morlevi.graber` содержит класс `Graber`, который предназначен для сбора данных о товарах со страниц сайта `morlevi.co.il`. Он наследуется от класса `Graber` из `src.suppliers.graber`, и переопределяет его функциональность для специфических нужд. Класс `Graber` предоставляет методы для обработки полей на странице товара, а также возможность использования декораторов для выполнения предварительных действий перед запросом к веб-драйверу.

## Классы

### `Graber`

**Описание**: Класс для операций захвата данных с сайта Morlevi.

**Наследуется от**: `src.suppliers.graber.Graber`

**Методы**:
- `__init__`: Инициализирует класс, устанавливая префикс поставщика и вызывая конструктор родительского класса.
- `local_image_path`: Метод для получения и сохранения изображения товара локально.

#### `__init__`
```python
def __init__(self, driver: Driver):
    """Инициализация класса сбора полей товара."""
```
**Описание**: Инициализирует экземпляр класса `Graber`. Устанавливает префикс поставщика и вызывает конструктор родительского класса.

**Параметры**:
- `driver` (`src.webdriver.driver.Driver`): Экземпляр веб-драйвера для взаимодействия с веб-страницей.

#### `local_image_path`
```python
async def local_image_path(self, value: Any = None):
    """Fetch and save image locally."""
```

**Описание**: Получает изображение товара как скриншот, сохраняет его в локальный файл во временной директории `tmp` и сохраняет путь к локальному файлу в поле `local_image_path` объекта `ProductFields`.

**Параметры**:
- `value` (`Any`, optional): Значение, которое можно передать через kwargs. Если значение передано, оно будет использовано в поле `ProductFields.local_image_path`. По умолчанию `None`.

**Возвращает**:
    - `bool`: `True` в случае успешного сохранения изображения, иначе `None`.

**Вызывает исключения**:
- `Exception`: В случае ошибки при сохранении изображения.
