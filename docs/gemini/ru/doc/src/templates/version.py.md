# Модуль `version.py`

## Обзор

Модуль `version.py` предназначен для хранения и управления информацией о версии, названии проекта, авторе и других метаданных приложения. Он загружает эти данные из файла `settings.json` и предоставляет их в виде переменных.

- [Переменные](#переменные)
- [Импорты](#импорты)
- [Обработка файла настроек](#обработка-файла-настроек)

## Переменные

### `MODE`

- **Описание**: Определяет текущий режим работы приложения (по умолчанию `dev`).
- **Тип**: `str`
- **Значение по умолчанию**: `'dev'`

### `settings`

- **Описание**: Словарь, содержащий настройки, загруженные из файла `settings.json`.
- **Тип**: `dict | None`
- **Значение по умолчанию**: `None`

### `__project_name__`

- **Описание**: Название проекта. Загружается из `settings.json` или устанавливается по умолчанию `hypotez`.
- **Тип**: `str`
- **Значение по умолчанию**: `'hypotez'`

### `__version__`

- **Описание**: Версия проекта. Загружается из `settings.json` или устанавливается пустой строкой по умолчанию.
- **Тип**: `str`
- **Значение по умолчанию**: `''`

### `__doc__`

- **Описание**: Строка документации проекта. По умолчанию пустая строка.
- **Тип**: `str`
- **Значение по умолчанию**: `''`

### `__details__`

- **Описание**: Дополнительная информация о проекте. По умолчанию пустая строка.
- **Тип**: `str`
- **Значение по умолчанию**: `''`

### `__author__`

- **Описание**: Автор проекта. Загружается из `settings.json` или устанавливается пустой строкой по умолчанию.
- **Тип**: `str`
- **Значение по умолчанию**: `''`

### `__copyright__`

- **Описание**: Информация об авторских правах проекта. Загружается из `settings.json` или устанавливается пустой строкой по умолчанию.
- **Тип**: `str`
- **Значение по умолчанию**: `''`

### `__cofee__`

- **Описание**: Сообщение с информацией для поддержки разработчика. Загружается из `settings.json` или устанавливается по умолчанию.
- **Тип**: `str`
- **Значение по умолчанию**: `"Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69"`

## Импорты

### `json`

- **Описание**: Модуль для работы с JSON-данными.

## Обработка файла настроек

### Попытка загрузки настроек

- **Описание**: Попытка загрузить настройки из файла `../settings.json`.
- **Обработка исключений**:
    - `FileNotFoundError`: Возникает, если файл `../settings.json` не найден.
    - `json.JSONDecodeError`: Возникает, если файл `../settings.json` содержит некорректные JSON-данные.
    - В случае возникновения исключения, настройки не загружаются, и используются значения по умолчанию.
