# Модуль hypotez/src/_examples/get_relative_path.py

## Обзор

Этот модуль содержит пример использования функции `get_relative_path` для получения относительного пути к директории 'hypotez' из текущего файла.  Он демонстрирует взаимодействие с функцией `get_relative_path` из модуля `src.utils.path`.

## Функции

### `get_relative_path`

**Описание**: Функция, возвращающая относительный путь к заданной директории относительно текущего файла.

**Параметры**:

- `current_path` (Path): Путь к текущему файлу.
- `target_path` (str): Путь к целевой директории.

**Возвращает**:

- `Path`:  Относительный путь к целевой директории или `None`, если путь не найден.

**Вызывает исключения**:

- Нет


## Использование

Этот модуль демонстрирует использование функции `get_relative_path` с конкретными входными данными:

```python
from pathlib import Path
import header
from src.utils.path import get_relative_path

relative_path = get_relative_path(Path(__file__).resolve(), 'hypotez')
print(relative_path)
```

В данном примере:

- `Path(__file__).resolve()`:  Получает абсолютный путь к текущему файлу.
- `'hypotez'`:  Искомая целевая директория.
- `get_relative_path()`: Вычисляет и возвращает относительный путь.
- `print()`: Выводит результат на консоль.

## Модули

- `pathlib`: Использование для работы с путями.
- `header`:  Используемый модуль, но нет деталей о нём.
- `src.utils.path`: Модуль, содержащий функцию `get_relative_path`.


## Константны
