# Модуль `date_time`

## Обзор

Модуль `date_time` содержит класс `TimeoutCheck` с функциями для проверки, находится ли текущее время в заданном интервале, с возможностью установки таймаута. Это полезно для выполнения операций, которые должны происходить только в определенные периоды времени. Интервал по умолчанию — с 23:00 до 06:00, и функция может обрабатывать интервалы, которые охватывают полночь. Также предоставляет функциональность для ожидания ответа с таймаутом.

## Классы

### `TimeoutCheck`

**Описание**: Класс для проверки временных интервалов и ожидания ввода с таймаутом.

**Методы**:
- [`__init__`](#__init__): Инициализирует экземпляр класса `TimeoutCheck`.
- [`interval`](#interval): Проверяет, находится ли текущее время в указанном интервале.
- [`interval_with_timeout`](#interval_with_timeout): Проверяет, находится ли текущее время в интервале с таймаутом.
- [`get_input`](#get_input): Запрашивает ввод от пользователя.
- [`input_with_timeout`](#input_with_timeout): Ожидает ввод с таймаутом.

#### `__init__`

**Описание**: Инициализирует экземпляр класса `TimeoutCheck`.

**Параметры**:
-   `self` (TimeoutCheck): Ссылка на экземпляр класса.

#### `interval`

**Описание**: Проверяет, находится ли текущее время в заданном интервале.

**Параметры**:
- `start` (time): Начало интервала (по умолчанию 23:00).
- `end` (time): Конец интервала (по умолчанию 06:00).
- `self` (TimeoutCheck): Ссылка на экземпляр класса.

**Возвращает**:
- `bool`: `True`, если текущее время находится в интервале, `False` в противном случае.

#### `interval_with_timeout`

**Описание**: Проверяет, находится ли текущее время в интервале, с таймаутом.

**Параметры**:
-   `self` (TimeoutCheck): Ссылка на экземпляр класса.
-   `timeout` (int): Время ожидания в секундах (по умолчанию 5).
-   `start` (time): Начало интервала (по умолчанию 23:00).
-   `end` (time): Конец интервала (по умолчанию 06:00).

**Возвращает**:
- `bool`: `True`, если текущее время находится в интервале и получен ответ в течение таймаута, `False`, если нет или произошел таймаут.

#### `get_input`

**Описание**: Запрашивает ввод от пользователя.

**Параметры**:
-   `self` (TimeoutCheck): Ссылка на экземпляр класса.

#### `input_with_timeout`

**Описание**: Ожидает ввод от пользователя с таймаутом.

**Параметры**:
- `timeout` (int): Время ожидания ввода в секундах (по умолчанию 5).
- `self` (TimeoutCheck): Ссылка на экземпляр класса.

**Возвращает**:
- `str | None`: Введенные данные или `None`, если произошел таймаут.
