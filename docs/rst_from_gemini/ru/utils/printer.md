# Модуль `printer.py`

Файл: `hypotez/src/utils/printer.py`

Данный модуль предоставляет функции для красивой и цветной печати данных в консоли.  Он поддерживает форматирование текста с помощью ANSI escape кодов для изменения цвета, фона и стилей шрифта.  Поддерживается вывод данных различных типов, включая словари, списки и файлы CSV/XLS.

## Функции

* **`_color_text(text, text_color="", bg_color="", font_style="")`:**
    * Вспомогательная функция, которая форматирует строку `text` с помощью переданных параметров цвета, фона и стиля шрифта. Использует ANSI escape коды для управления цветовой схемой.
    * Возвращает отформатированную строку.
* **`pprint(print_data, text_color="white", bg_color="", font_style="")`:**
    * Основная функция для печати данных.
    * Принимает данные `print_data` любого типа (словари, списки, строки, пути к файлам).
    * Поддерживает изменение цвета текста, фона и стиля шрифта.
    * Для словарей печатает отформатированный JSON.
    * Для списков печатает каждый элемент на отдельной строке.
    * Для файлов CSV/XLS пытается прочитать и вывести данные. Если тип файла не поддерживается, выводит сообщение об ошибке.
    * Если `print_data`  равно `None`, выводит сообщение об отсутствии данных.
    * Обрабатывает возможные исключения во время работы с данными и выводит сообщение об ошибке.

## Использование

```python
from hypotez.src.utils.printer import pprint

# Печать словаря в зелёном цвете:
pprint({"name": "Alice", "age": 30}, text_color="green")

# Печать списка:
pprint(["apple", "banana", "orange"])

# Попытка печати файла CSV:
pprint("data.csv")

# Обработка ошибки (если файла нет или тип не поддерживается):
pprint("nonexistent_file.txt")

# Печать None:
pprint(None)
```

## Поддерживаемые типы данных

* Словари (JSON вывод)
* Списки
* Строки (включая пути к файлам)
* Файлы CSV и XLS

## ANSI Escape Коды

Модуль использует ANSI escape коды для форматирования вывода.


## Замечания

*  Функция `pprint` старается определить тип файла по расширению, но не делает глубокую валидацию.
* Для корректной работы требуется наличие библиотек `json`, `csv`, `pandas`.

* `MODE`:  переменная, представляющая режим работы модуля (например, 'debug', 'release').  Пока не используется.
* `TEXT_COLORS`, `BG_COLORS`, `FONT_STYLES`: словари, хранящие ANSI escape коды для различных цветов и стилей.
