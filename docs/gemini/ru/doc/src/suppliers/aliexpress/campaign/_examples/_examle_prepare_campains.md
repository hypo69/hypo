# Модуль hypotez/src/suppliers/aliexpress/campaign/_examples/_examle_prepare_campains.py

## Обзор

Этот модуль содержит примеры использования функций из модуля `prepare_campaigns` для подготовки кампаний AliExpress. Примеры включают обработку одной категории кампаний, конкретной кампании и всех кампаний.

## Функции

### `process_campaign_category`

**Описание**: Обрабатывает кампанию по заданной категории.

**Параметры**:
- `campaign_name` (str): Название кампании.
- `category` (str): Название категории.
- `language` (str): Язык кампании.
- `currency` (str): Валюта кампании.
- `force` (bool, optional): Принудительно переобрабатывать кампанию, если True. По умолчанию False.

**Возвращает**:
- `None`: Функция не возвращает явное значение.

**Вызывает исключения**:
- `Exception`:  Возможные ошибки при работе с кампаниями.


### `process_campaign`

**Описание**: Обрабатывает конкретную кампанию.

**Параметры**:
- `campaign_name` (str): Название кампании.
- `categories` (list[str], optional): Список категорий кампании. По умолчанию None.
- `language` (str, optional): Язык кампании. По умолчанию None.
- `currency` (str, optional): Валюта кампании. По умолчанию None.
- `force` (bool, optional): Принудительно переобрабатывать кампанию, если True. По умолчанию False.

**Возвращает**:
- `None`: Функция не возвращает явное значение.

**Вызывает исключения**:
- `Exception`:  Возможные ошибки при работе с кампаниями.


### `process_all_campaigns`

**Описание**: Обрабатывает все кампании.

**Параметры**:
- `language` (str, optional): Язык кампаний. По умолчанию None.
- `currency` (str, optional): Валюта кампаний. По умолчанию None.
- `force` (bool, optional): Принудительно переобрабатывать все кампании, если True. По умолчанию False.

**Возвращает**:
- `None`: Функция не возвращает явное значение.

**Вызывает исключения**:
- `Exception`:  Возможные ошибки при работе с кампаниями.


### `get_directory_names`

**Описание**: Получает имена файлов в директории.

**Параметры**:
- `campaigns_directory` (Path): Путь к директории с кампаниями.


**Возвращает**:
- `list[str]`: Список имен файлов в директории.


**Вызывает исключения**:
- `FileNotFoundError`: Если директория не найдена.
- `Exception`: Другие ошибки при работе с файлами.
