# Модуль grab_lilnks_to_chats

## Обзор

Этот модуль предназначен для извлечения ссылок на отдельные чаты. Он использует драйвер веб-драйвера (например, Chrome или Firefox) для взаимодействия с веб-сайтом и извлечения необходимых ссылок.

## Функции

### `get_links`

**Описание**: Функция извлекает ссылки на отдельные чаты.

**Параметры**:

- `d` (Driver): Объект веб-драйвера, используемый для взаимодействия с веб-сайтом.

**Возвращает**:

- `list[str]`: Список ссылок на отдельные чаты. Возвращает `None` в случае ошибок.

**Вызывает исключения**:

- `Exception`: Возникает в случае ошибок при взаимодействии с веб-драйвером или при парсинге данных.

## Использование

**Пример**:

```python
from src.webdriver.driver import Driver
from src.webdriver.firefox import Firefox
# ... (Импорты других необходимых модулей)

# Создаем экземпляр драйвера Firefox
d = Driver(Firefox)

# Загружаем страницу
d.get_url('https://chatgpt.com/')

# Извлекаем ссылки
links = get_links(d)

# Обработка полученных ссылок (если links не None)
if links:
    for link in links:
        # ... (Обработка каждой ссылки)
        print(link)
```

**Примечания**:

- Модуль использует конфигурационный файл `chats_list.json` для определения локаторов элементов страницы.  
- Для корректной работы необходимо установить и импортировать необходимые библиотеки, такие как `selenium`, `requests`, и др.
- Этот пример предполагает, что у вас есть соответствующие импорты и инициализация драйвера.  Убедитесь, что вы настроили необходимые параметры, такие как `gs.path` для правильного пути к файлам.

## Вспомогательные данные

- Используется `j_loads_ns` для загрузки данных из JSON файла `locators/chats_list.json`
- `locator`:  Переменная, содержащая загруженные локаторы из JSON файла.
