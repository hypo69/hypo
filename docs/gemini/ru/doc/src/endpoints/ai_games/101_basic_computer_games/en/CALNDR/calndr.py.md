# CALNDR: Календарь

## Обзор

Модуль `calndr.py` представляет собой игру "Календарь", которая выводит на экран календарь на месяц, основываясь на номере дня недели, с которого начинается месяц, и количестве дней в месяце. Пользователь вводит эти два параметра, а программа строит календарь.

## Функции

### `Основной блок кода`

**Описание**:
Основной блок кода отвечает за ввод данных, вычисление отступов и вывод календаря в консоль.

**Параметры**:
- Отсутствуют.

**Возвращает**:
- Ничего. Функция выводит календарь в консоль.

**Вызывает исключения**:
- `ValueError`: Возникает, если пользователь вводит некорректное значение (не целое число) при запросе дня начала недели или количества дней в месяце.
