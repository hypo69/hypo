# Модуль `src.bots.discord`

## Обзор

Модуль содержит инициализационный файл для подмодуля `discord`, предназначенного для интеграции с Discord ботами. Файл не содержит исполняемого кода и используется для обозначения директории как пакета Python.

## Детали

Данный файл не содержит классов или функций, поэтому нет разделов "Классы" и "Функции". Он служит только для объявления директории `discord` как пакета, что позволяет импортировать модули и подмодули внутри этой директории как часть Python пакета `src.bots`.
