# Модуль `grab_lilnks_to_chats`

## Обзор

Модуль предназначен для извлечения ссылок на отдельные чаты с веб-страницы, используя веб-драйвер. В настоящее время не поддерживается работа через драйверы Chrome и Firefox.

## Функции

### `get_links`

**Описание**: Извлекает ссылки на отдельные чаты, используя веб-драйвер.

**Параметры**:
- `d` (Driver): Объект веб-драйвера.

**Возвращает**:
- `list`: Список веб-элементов, представляющих ссылки на чаты.

```python
def get_links(d:Driver):
    """Ссылки на отдельные чаты """
    ...
    links = d.execute_locator(locator.link)
    return links
```
