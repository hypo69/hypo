# Модуль hypotez/src/webdriver/_pytest/test_executor.py

## Обзор

Этот модуль содержит тесты для класса `ExecuteLocator`, отвечающего за взаимодействие с веб-элементами. Тесты проверяют различные сценарии, включая поиск элементов, получение атрибутов и отправку сообщений.

## Фикстуры

### `driver_mock`

**Описание**: Создаёт фиктивный объект веб-драйвера, используемый для тестирования.

**Возвращает**:
- `MagicMock`: Фиктивный объект веб-драйвера.


### `execute_locator`

**Описание**: Создаёт экземпляр класса `ExecuteLocator` с фиктивным веб-драйвером.

**Параметры**:
- `driver_mock`: Объект фиктивного веб-драйвера.

**Возвращает**:
- `ExecuteLocator`: Экземпляр класса `ExecuteLocator`.


## Тесты

### `test_get_webelement_by_locator_single_element`

**Описание**: Проверяет получение одного элемента по локатору.

**Параметры**:
- `execute_locator`: Экземпляр класса `ExecuteLocator`.
- `driver_mock`: Фиктивный объект веб-драйвера.

**Возвращает**:
- `WebElement`: Найденный веб-элемент.


### `test_get_webelement_by_locator_multiple_elements`

**Описание**: Проверяет получение нескольких элементов по локатору.

**Параметры**:
- `execute_locator`: Экземпляр класса `ExecuteLocator`.
- `driver_mock`: Фиктивный объект веб-драйвера.

**Возвращает**:
- `list[WebElement]`: Список найденных веб-элементов.


### `test_get_webelement_by_locator_no_element`

**Описание**: Проверяет случай, когда элемент не найден по локатору.

**Параметры**:
- `execute_locator`: Экземпляр класса `ExecuteLocator`.
- `driver_mock`: Фиктивный объект веб-драйвера.


**Возвращает**:
- `False`: Значение, указывающее на отсутствие элемента.


### `test_get_attribute_by_locator`

**Описание**: Проверяет получение атрибута элемента по локатору.

**Параметры**:
- `execute_locator`: Экземпляр класса `ExecuteLocator`.
- `driver_mock`: Фиктивный объект веб-драйвера.

**Возвращает**:
- `str`: Значение атрибута.


### `test_send_message`

**Описание**: Проверяет отправку сообщения элементу.

**Параметры**:
- `execute_locator`: Экземпляр класса `ExecuteLocator`.
- `driver_mock`: Фиктивный объект веб-драйвера.

**Параметры (в функции `send_message`)**:
- `locator`: Словарь с локатором элемента.
- `message`: Текст сообщения.
- `typing_speed`: Скорость набора (по умолчанию 0).
- `continue_on_error`: Флаг продолжения выполнения при ошибке. По умолчанию `True`.


**Возвращает**:
- `True`: Успешное выполнение.


### `test_send_message_typing_speed`

**Описание**: Проверяет отправку сообщения элементу с заданной скоростью.

**Параметры**:
- `execute_locator`: Экземпляр класса `ExecuteLocator`.
- `driver_mock`: Фиктивный объект веб-драйвера.

**Параметры (в функции `send_message`)**:
- `locator`: Словарь с локатором элемента.
- `message`: Текст сообщения.
- `typing_speed`: Скорость набора.
- `continue_on_error`: Флаг продолжения выполнения при ошибке. По умолчанию `True`.

**Возвращает**:
- `True`: Успешное выполнение.

**Использует**:
- `patch`: Для подмены функции `time.sleep`.
