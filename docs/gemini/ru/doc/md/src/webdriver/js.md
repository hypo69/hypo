# Модуль hypotez/src/webdriver/js.py

## Обзор

Модуль `js.py` предоставляет функции для взаимодействия с веб-страницей через JavaScript, расширяя возможности Selenium WebDriver. Он включает в себя функции для отображения невидимых элементов DOM, получения метаданных страницы (состояние загрузки, referrer, язык) и управления фокусом браузера.

## Класс `JavaScript`

**Описание**: Класс `JavaScript` предоставляет методы для выполнения JavaScript кода в браузере.

### `unhide_DOM_element`

**Описание**: Делает невидимый DOM-элемент видимым, изменяя его свойства стиля.

**Параметры**:

- `element` (WebElement): Объект WebElement, который нужно сделать видимым.

**Возвращает**:

- `bool`: `True`, если скрипт выполнен успешно, `False` в противном случае.

**Вызывает исключения**:

- Любые исключения, возникающие при выполнении JavaScript кода.


### `ready_state`

**Описание**: Возвращает текущее состояние загрузки документа.

**Возвращает**:

- `str`: `'loading'`, если документ загружается, `'complete'`, если загрузка завершена.

**Вызывает исключения**:

- Любые исключения, возникающие при выполнении JavaScript кода.


### `window_focus`

**Описание**: Устанавливает фокус на окно браузера с помощью JavaScript.  Попытка вывести окно браузера на передний план.

**Вызывает исключения**:

- Любые исключения, возникающие при выполнении JavaScript кода.


### `get_referrer`

**Описание**: Возвращает URL ссылки-источника текущего документа.

**Возвращает**:

- `str`: URL ссылки-источника, или пустую строку, если данные недоступны.

**Вызывает исключения**:

- Любые исключения, возникающие при выполнении JavaScript кода.


### `get_page_lang`

**Описание**: Возвращает язык текущей страницы.

**Возвращает**:

- `str`: Код языка страницы, или пустую строку, если данные недоступны.

**Вызывает исключения**:

- Любые исключения, возникающие при выполнении JavaScript кода.
