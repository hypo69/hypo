# Модуль hypotez/src/utils/header.py

## Обзор

Этот модуль содержит функцию `set_project_root`, которая находит корневой каталог проекта, начиная с текущего файла.  Функция используется для корректного импорта файлов и модулей, находящихся в различных подпапках проекта.  Модуль также инициализирует переменные, представляющие информацию о проекте, такую как имя, версия, документация и другие детали, загружая данные из файла `settings.json` в корневой директории проекта.

## Функции

### `set_project_root`

**Описание**:  Функция находит корневой каталог проекта, начиная с текущего файла и просматривая родительские директории.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для определения корневого каталога. По умолчанию содержит `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневому каталогу проекта, если найден, иначе путь к каталогу, в котором находится текущий скрипт.

**Вызывает исключения**:

- Нет.


## Переменные

### `__root__`

**Описание**:  Путь к корневому каталогу проекта, полученный из функции `set_project_root`.

**Тип**: `Path`

**Значение**: Путь к корневому каталогу проекта.

### `settings`

**Описание**: Словарь с настройками проекта, полученный из файла `settings.json`.

**Тип**: `dict`

**Значение**: Словарь настроек проекта или `None`, если файл `settings.json` не найден или содержит невалидные данные.

### `doc_str`

**Описание**: Строка с описанием проекта (README.MD).

**Тип**: `str`

**Значение**: Строка с описанием проекта (README.MD) или `None`, если файл `README.MD` не найден или содержит невалидные данные.

### `__project_name__`

**Описание**: Имя проекта.

**Тип**: `str`

**Значение**: Имя проекта, полученное из файла `settings.json`, по умолчанию `hypotez`.

### `__version__`

**Описание**: Версия проекта.

**Тип**: `str`

**Значение**: Версия проекта, полученная из файла `settings.json`, по умолчанию пустая строка.

### `__doc__`

**Описание**: Документация проекта.

**Тип**: `str`

**Значение**: Документация проекта, полученная из файла README.md, по умолчанию пустая строка.

### `__details__`

**Описание**: Дополнительная информация о проекте.

**Тип**: `str`

**Значение**: Пустая строка.

### `__author__`

**Описание**: Автор проекта.

**Тип**: `str`

**Значение**: Автор проекта, полученный из файла `settings.json`, по умолчанию пустая строка.

### `__copyright__`

**Описание**: Авторские права проекта.

**Тип**: `str`

**Значение**: Авторские права проекта, полученный из файла `settings.json`, по умолчанию пустая строка.

### `__cofee__`

**Описание**: Ссылка для поддержки разработчика.

**Тип**: `str`

**Значение**: Ссылка для поддержки разработчика, полученная из файла `settings.json`, по умолчанию ссылка на Boosty.
