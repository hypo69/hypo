# Модуль `background.js`

## Обзор

Данный скрипт содержит обработчик события `onInstalled` для расширения Chrome, связанного с OpenAI.  При установке расширения выводится сообщение в консоль.


## Обработчики событий

### `chrome.runtime.onInstalled.addListener`

**Описание**: Функция-обработчик события установки расширения.

**Параметры**:
- `callback`: функция, которая будет вызвана при установке расширения. В данном случае это `callback` функция, которая выводит сообщение в консоль.

**Возвращает**:
- Не применимо (Void).


## Функции


### `onInstalled`

**Описание**:  Обрабатывает событие установки расширения, выводя сообщение в консоль.

**Параметры**:
-  Нет параметров.

**Возвращает**:
- Нет возвращаемого значения.

**Вызывает исключения**:
-  Нет.
```javascript
chrome.runtime.onInstalled.addListener(() => {
    console.log('OpenAI Model Interface Extension Installed');
});
```
