# Scenario Executor

## Обзор

Модуль `scenario_executor` отвечает за выполнение сценариев, загруженных из файлов. Он итерирует по списку файлов, загружает сценарии, и выполняет каждый из них, включая навигацию по URL, получение списков продуктов, навигацию по страницам продуктов, извлечение данных о продуктах, создание объектов продуктов и их добавление в PrestaShop.  В случае успешного выполнения сценария, обновляется журнал, а в случае ошибки, происходит обработка ошибок.  В итоге возвращается булево значение, отражающее результат выполнения всех сценариев.
