## Функции

### `get_allowed_actions`

```python
def get_allowed_actions(state: str) -> list[str]:
    """Возвращает список разрешенных действий для заданного состояния светофора.

    Args:
        state (str): Текущее состояние светофора.

    Returns:
        list[str]: Список разрешенных действий.

    Raises:
        ValueError: Если передано неизвестное состояние светофора.

    Example:
        >>> get_allowed_actions('red')
        ['go', 'stop']
    """
```

**Описание**: Возвращает список разрешенных действий для заданного состояния светофора.

**Как работает функция**:
Функция <описание работы функции. Например: принимает текущее состояние светофора в качестве аргумента и возвращает список действий, которые разрешены в этом состоянии.  Внутри функция содержит условные операторы, которые определяют список действий в зависимости от входного состояния. Например, если состояние "красный", функция может вернуть `['stop']`, а если состояние "зеленый" - `['go']`. Если передано неизвестное состояние, функция может вызывать исключение `ValueError`>.

**Параметры**:
- `state` (str): Текущее состояние светофора.

**Возвращает**:
- `list[str]`: Список разрешенных действий.

**Вызывает исключения**:
- `ValueError`: Если передано неизвестное состояние светофора.

**Примеры**:
