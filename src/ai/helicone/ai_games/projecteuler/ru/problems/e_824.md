# Задача No 824
## Гипербинарная запись

## Условия
Дано положительное целое число $n$. Гипербинарная запись $n$ - это выражение $n$ в виде суммы степеней 2, каждая из которых может быть взята с кратностью 0, 1 или 2.

Например:

$3 = 2^1 + 2^0$
$7 = 2^2 + 2^1 + 2^0$
$12 = 2^3 + 2^2$
$12 = 2^3 + 2^1 + 2^1$
$12 = 2^3 + 2^0 + 2^0 + 2^1$
$12 = 2^2 + 2^2 + 2^2$

Пусть $f(n)$ будет количеством различных гипербинарных записей для $n$. Например, $f(3) = 1, f(7) = 1, f(12) = 4$.
В задаче необходимо найти $\sum_{n=1}^{{10}^{16}} f(n) \pmod{10^9+7}$.
