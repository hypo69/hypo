## Обзор

Модуль `sergey_mymaster_co_il` предназначен для работы с сайтом `sergey_mymaster.co.il` в рамках API PrestaShop.

## Переменные

### `MODE`

**Описание**:
Устанавливает режим работы приложения. Может принимать значения `dev` или `prod`.

**Значение по умолчанию**:
`dev`
