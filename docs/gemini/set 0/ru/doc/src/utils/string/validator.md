# Модуль hypotez/src/utils/string/validator.py

## Обзор

Модуль `validator.py` предоставляет статические методы для валидации строк, используемых в приложении.  Включает в себя валидацию цен, весов, артикулов (SKU) и URL-адресов.  Методы возвращают булево значение, показывающее результат валидации.

* [Валидация URL](#валидация-url)

## Классы

### `ProductFieldsValidator`

**Описание**: Класс `ProductFieldsValidator` предоставляет статические методы для валидации различных полей, связанных с продуктами.

**Статические методы**:

### `validate_price`

**Описание**: Метод проверяет корректность формата значения цены.

**Параметры**:

- `price` (str): Значение цены в строковом формате.

**Возвращает**:

- `bool`: `True`, если цена валидна, иначе `None`.

**Обрабатываемые исключения**:

- `ValueError`: При неудачной попытке преобразования строки в число с плавающей точкой.


### `validate_weight`

### `isint`

**Описание**: Метод проверяет, является ли строка целым числом.
