# Модуль pricelist_generator

## Обзор

Этот модуль предоставляет класс `ReportGenerator` для генерации HTML и PDF отчетов на основе данных из JSON-файлов. Он использует Jinja2 для рендеринга HTML шаблонов и `wkhtmltopdf` для конвертации HTML в PDF.

## Класс `ReportGenerator`

### Описание

Класс `ReportGenerator` отвечает за генерацию HTML и PDF отчетов. Он загружает данные из JSON, рендерит шаблон с данными, сохраняет результат в HTML файл и преобразует его в PDF.

### Методы

#### `generate_html`

##### Описание

Генерирует HTML-контент на основе шаблона и данных.

##### Параметры

- `data` (dict): Словарь с данными для рендеринга шаблона.

##### Возвращает

- `str`: Сгенерированный HTML-контент.


#### `create_report`

##### Описание

Выполняет полный цикл генерации отчета.

##### Параметры

- `data` (dict): Словарь с данными для генерации отчета.
- `html_file` (str или Path): Путь к файлу для сохранения HTML.
- `pdf_file` (str или Path): Путь к файлу для сохранения PDF.

##### Возвращает

- `None`

##### Вызывает исключения

- Возможны ошибки при работе с файлами или `wkhtmltopdf`.


## Функция `main`

### Описание

Функция `main` запускает процесс генерации отчета для заданного мехирона и языка.

##### Параметры

- `mexiron` (str): Идентификатор мехирона.
- `lang` (str): Язык отчета.

##### Возвращает

- `bool`:  Результат выполнения (True, если успешно, иначе False).

##### Дополнительная информация

Функция загружает данные из JSON-файла, генерирует HTML и PDF отчёт и сохраняет их в файлы.
