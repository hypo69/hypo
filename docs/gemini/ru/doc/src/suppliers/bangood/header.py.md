# Модуль `header.py`

## Обзор

Этот модуль содержит общие настройки и переменные для проекта, а также функции для определения корневой директории проекта и загрузки настроек. Он также устанавливает переменные, содержащие информацию о проекте.

## Функции

### `set_project_root`

**Описание**:
Находит корневую директорию проекта, начиная с директории текущего файла, просматривая вверх и останавливаясь в первой директории, содержащей любой из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или директорий для определения корня проекта.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена, иначе директория, где расположен скрипт.

## Переменные

### `MODE`

**Описание**:
Указывает режим работы приложения. По умолчанию установлен в 'dev'.

### `__root__`

**Описание**:
Путь к корневой директории проекта. Инициализируется функцией `set_project_root`.

### `settings`

**Описание**:
Словарь настроек проекта, загруженный из файла `settings.json`. Значение может быть `None`, если файл не найден или произошла ошибка декодирования.

### `doc_str`

**Описание**:
Строка, содержащая содержимое файла `README.MD`. Значение может быть `None`, если файл не найден или произошла ошибка декодирования.

### `__project_name__`

**Описание**:
Название проекта, полученное из настроек. По умолчанию 'hypotez'.

### `__version__`

**Описание**:
Версия проекта, полученная из настроек. По умолчанию пустая строка.

### `__doc__`

**Описание**:
Содержание документации проекта, взятое из файла `README.MD`. По умолчанию пустая строка.

### `__details__`

**Описание**:
Дополнительная информация о проекте. На данный момент пустая строка.

### `__author__`

**Описание**:
Автор проекта, полученный из настроек. По умолчанию пустая строка.

### `__copyright__`

**Описание**:
Информация об авторских правах проекта, полученная из настроек. По умолчанию пустая строка.

### `__cofee__`

**Описание**:
Сообщение с призывом поддержать разработчика, полученное из настроек. По умолчанию - сообщение с ссылкой на boosty.
