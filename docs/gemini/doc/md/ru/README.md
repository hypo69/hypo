# Модули проекта

## Обзор

Этот документ содержит описание модулей проекта.  Каждый модуль имеет краткое описание и предназначение.

## advertisement

### Обзор

Управление рекламными кампаниями и взаимодействием с рекламными платформами. Поддержка создания, управления и анализа производительности рекламных объявлений.

## ai

### Обзор

Управление AI-моделями, включая создание, обучение и развертывание. Поддержка различных алгоритмов для анализа и обработки данных.

## bots

### Обзор

Управление интеграцией ботов (Telegram, Discord). Обеспечивает обмен сообщениями, обработку команд и взаимодействие с пользователями через мессенджеры.

## category

### Обзор

Управление структурой и иерархией категорий продуктов. Поддержка добавления, обновления, удаления и поиска категорий.

## db

### Обзор

Обработка подключений к базе данных и взаимодействие с ней, включая запросы и безопасную обработку данных.


## docs

### Обзор

Содержит документацию проекта, включая описания функций, классов, модулей, спецификаций API и инструкций по использованию.


## endpoints

### Обзор

Модуль для обработки конечных точек потребителей, интеграции с внешними сервисами и системами.


## fast_api

### Обзор

Предоставляет интеграцию с фреймворком FastAPI для создания API-интерфейсов и управления запросами сервера с высокой производительностью благодаря асинхронному выполнению.


## goog

### Обзор

Интеграция с Google-сервисами, такими как Google Таблицы и Google Диск, предоставляя инструменты для работы с облачными данными.


## gui

### Обзор

Модуль графического интерфейса пользователя (GUI) для взаимодействия с пользователем через визуальные элементы, такие как окна, кнопки и элементы управления.


## launcher

### Обзор

Инициализирует и настраивает все необходимые сервисы и модули перед запуском приложения.


## logger

### Обзор

Предоставляет структурированное ведение журнала и обработку ошибок для отслеживания ошибок, предупреждений и информационных сообщений.


## product

### Обзор

Управление данными о продуктах с функциональностью для добавления, обновления, удаления и обработки информации о продуктах.


## scenario

### Обзор

Управление конфигурируемыми рабочими процессами и автоматизированными последовательностями для работы с поставщиками данных.


## templates

### Обзор

Управление шаблонами для различных компонентов, таких как интерфейсы, электронные письма и сообщения, поддерживая стандартизированное форматирование.


## translator

### Обзор

Предоставляет автоматический перевод текста на несколько языков, используя внешние API или встроенные алгоритмы.


## translators

### Обзор

Включает дополнительные классы и функции для расширенного управления переводом, поддерживая несколько источников и типов переводов.


## utils

### Обзор

Содержит вспомогательные функции и классы, используемые в проекте, включая форматирование данных, разбор и обработку строк.


## webdriver

### Обзор

Управление задачами автоматизации браузера, совместимо с браузерами, такими как Chrome и Firefox, поддерживает веб-скрейпинг и тестирование интерфейса.
