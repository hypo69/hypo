# Модуль `hypotez/src/utils/convertors`

## Обзор

Этот модуль предоставляет функции для конвертации различных форматов данных, включая CSV, JSON, XML, HTML, MD, Base64, изображения и текст. Он включает утилиты для преобразования данных в словари, списки, форматы для работы с таблицами и т.д., а также для работы с аудио (речь в текст и наоборот).

## Доступные функции

### `csv2dict`

**Описание**: Преобразует данные из файла CSV в словарь.

**Параметры**:

- `filename` (str): Путь к файлу CSV.

**Возвращает**:

- `dict`: Словарь, содержащий данные из файла CSV. Возвращает `None`, если произошла ошибка.


### `csv2ns`

**Описание**: Преобразует данные из файла CSV в пространство имен.

**Параметры**:

- `filename` (str): Путь к файлу CSV.

**Возвращает**:

- `dict`: Словарь, представляющий пространство имен. Возвращает `None`, если произошла ошибка.


### `dict2ns`

**Описание**: Преобразует словарь в пространство имен.

**Параметры**:

- `data` (dict): Входной словарь.

**Возвращает**:

- `dict`: Словарь, представляющий пространство имен. Возвращает `None`, если произошла ошибка.


### `dict2csv`

**Описание**: Преобразует словарь в формат CSV.

**Параметры**:

- `data` (dict): Входной словарь.
- `filename` (str, optional): Путь к файлу для сохранения CSV. Если не указан, данные выводятся в консоль.

**Возвращает**:

- `None`.


### `dict2html`

**Описание**: Преобразует словарь в HTML.

**Параметры**:

- `data` (dict): Входной словарь.
- `filename` (str, optional): Путь к файлу для сохранения HTML. Если не указан, данные выводятся в консоль.

**Возвращает**:

- `None`.


### `dict2xls`

**Описание**: Преобразует словарь в формат XLSX.

**Параметры**:

- `data` (dict): Входной словарь.
- `filename` (str, optional): Путь к файлу для сохранения XLSX. Если не указан, данные выводятся в консоль.

**Возвращает**:

- `None`.


### `dict2xml`

**Описание**: Преобразует словарь в XML.

**Параметры**:

- `data` (dict): Входной словарь.
- `filename` (str, optional): Путь к файлу для сохранения XML. Если не указан, данные выводятся в консоль.

**Возвращает**:

- `None`.


### `dot2png`

**Описание**: Генерирует изображение PNG из файла `dot`.

**Параметры**:

- `filename` (str): Путь к файлу `dot`.
- `output_filename` (str, optional): Путь к выходному файлу PNG. Если не указан, используется имя файла `dot` с расширением `png`.

**Возвращает**:

- `None`.


### `html2escape`

**Описание**: Экранирует HTML-символы.

**Параметры**:

- `html_string` (str): Входная HTML-строка.

**Возвращает**:

- `str`: Экранированная HTML-строка. Возвращает `None`, если произошла ошибка.


### `html2ns`

**Описание**: Преобразует HTML в пространство имен.

**Параметры**:

- `html_string` (str): Входная HTML-строка.

**Возвращает**:

- `dict`: Словарь, представляющий пространство имен. Возвращает `None`, если произошла ошибка.


### `html2dict`

**Описание**: Преобразует HTML в словарь.

**Параметры**:

- `html_string` (str): Входная HTML-строка.

**Возвращает**:

- `dict`: Словарь, содержащий данные из HTML. Возвращает `None`, если произошла ошибка.


### `escape2html`

**Описание**: Преобразует экранированные символы обратно в HTML.

**Параметры**:

- `escaped_string` (str): Входная экранированная строка.

**Возвращает**:

- `str`: Строка без экранирования.


### `html2text` / `html2text_file`


**Описание**: Преобразует HTML в текст.


**Параметры**:


- `html_string` (str): Входной HTML.

**Возвращает**:


- `str`: Выходной текст.
