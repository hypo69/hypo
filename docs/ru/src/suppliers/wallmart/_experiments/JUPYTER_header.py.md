# Модуль для экспериментов с поставщиком Wallmart

## Обзор

Этот модуль предназначен для экспериментов, связанных с поставщиком Wallmart. Он содержит набор импортов, настроек путей и функцию для запуска поставщика.

## Подробней

Модуль `JUPYTER_header.py` служит для предварительной настройки окружения и импорта необходимых библиотек и модулей для экспериментов с данными от поставщика Wallmart. Он определяет пути к корневой директории проекта, добавляет их в `sys.path`, а также импортирует модули для работы с веб-драйвером, поставщиками, продуктами, категориями и утилитами для обработки строк. Кроме того, модуль содержит функцию `start_supplier` для инициализации поставщика с заданными параметрами.

**Назначение**: Запускает поставщика с указанными параметрами.

**Параметры**:

- `supplier_prefix` (str): Префикс поставщика. По умолчанию `'aliexpress'`.

**Возвращает**:
- `Supplier`: Объект поставщика, созданный с заданными параметрами.

**Как работает функция**:

1. Функция принимает префикс поставщика и локаль в качестве аргументов.
2. Создает словарь `params`, содержащий эти параметры.
3. Инициализирует и возвращает объект `Supplier` с использованием переданных параметров.
