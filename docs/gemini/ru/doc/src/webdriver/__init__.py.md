# Модуль `src.webdriver`

## Обзор

Модуль `src.webdriver` предоставляет инструменты для управления браузерами и скрапинга веб-сайтов. Он поддерживает различные браузеры и библиотеки для автоматизации, такие как Chrome, Firefox, Edge, BeautifulSoup, Playwright и Crawlee.
