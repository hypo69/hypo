# Модуль `hypotez/src/utils/convertors/xml2dict.py`

## Обзор

Этот модуль предоставляет функции для конвертации данных XML в словари. Он включает в себя функции для парсинга строк XML и преобразования деревьев XML элементов в представления словарей.

## Функции

### `_parse_node`

**Описание**: Парсит узел XML в словарь.

**Параметры**:

- `node` (ET.Element): Элемент XML для парсинга.

**Возвращает**:

- `dict | str`: Словарное представление узла XML или строка, если у узла нет атрибутов или дочерних элементов.


### `_make_dict`

**Описание**: Создает новый словарь с именем тега и значением.

**Параметры**:

- `tag` (str): Имя тега XML-элемента.
- `value` (any): Значение, связанное с тегом.

**Возвращает**:

- `dict`: Словарь с именем тега в качестве ключа и значением в качестве значения словаря.


### `xml2dict`

**Описание**: Парсит строку XML в словарь.

**Параметры**:

- `xml` (str): Строка XML для парсинга.

**Возвращает**:

- `dict`: Словарное представление XML.


### `ET2dict`

**Описание**: Преобразует дерево XML-элементов в словарь.

**Параметры**:

- `element_tree` (ET.Element): Дерево XML-элементов.

**Возвращает**:

- `dict`: Словарное представление дерева XML-элементов.


## Обработка исключений

Этот модуль обрабатывает возможную ошибку `ImportError` при импорте `xml.etree.cElementTree`. В случае возникновения ошибки, используется `xml.etree.ElementTree`.
