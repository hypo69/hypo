# Модуль `src.suppliers.kualastyle`

## Обзор

Модуль `src.suppliers.kualastyle` предназначен для работы с поставщиком Kualastyle. Он включает в себя класс `Graber` для сбора данных.

## Классы

### `Graber`

**Описание**: Класс `Graber` отвечает за сбор данных с сайта поставщика Kualastyle.

**Импорт**: `from .graber import Graber`

Для получения подробной информации о методах и параметрах класса `Graber`, смотрите [документацию для `graber.py`](./graber.md).
