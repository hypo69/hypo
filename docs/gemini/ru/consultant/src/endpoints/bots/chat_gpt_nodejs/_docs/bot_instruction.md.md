## Описание команд:
    
### `!hi`
- Назначение: Команда приветствует пользователя.
- Пример использования: `!hi`

### `!train <data> <data_dir> <positive> <attachment>`
- Назначение: Команда обучает модель с предоставленными данными.
- Аргументы:
    - `<data>`: Путь к файлу с данными для обучения.
    - `<data_dir>`: Путь к директории с данными для обучения.
    - `<positive>`: Путь к директории с позитивными примерами.
    - `<attachment>`: Файл для обучения, переданный как вложение.
- Пример использования: `!train data.txt data_dir positive_dir attachment.txt`

### `!test <test_data>`
- Назначение: Команда тестирует модель с предоставленными JSON данными.
- Аргументы:
    - `<test_data>`: JSON данные для тестирования.
- Пример использования: `!test {"key": "value"}`

### `!archive <directory>`
- Назначение: Команда архивирует файлы в указанной директории.
- Аргументы:
    - `<directory>`: Путь к директории, которую необходимо заархивировать.
- Пример использования: `!archive /path/to/directory`

### `!select_dataset <path_to_dir_positive> <positive>`
- Назначение: Команда выбирает набор данных для обучения из указанной директории.
- Аргументы:
    - `<path_to_dir_positive>`: Путь к директории с позитивными примерами.
    - `<positive>`: Уточняющий параметр.
- Пример использования: `!select_dataset /path/to/positive/ positive`

### `!instruction`
- Назначение: Команда выводит это сообщение с инструкцией.
- Пример использования: `!instruction`
