# Модуль `header.py`

## Обзор

Модуль `header.py` предназначен для инициализации основных параметров проекта, таких как корневая директория, настройки из `settings.json`, а также для загрузки документации из `README.MD`.

## Функции

### `set_project_root`

**Описание**: 
Функция для определения корневой директории проекта, начиная с директории текущего файла. Поиск ведется вверх по дереву директорий до первой директории, содержащей один из указанных маркерных файлов.

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для определения корневой директории. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена. В противном случае возвращается директория, в которой расположен текущий скрипт.

## Переменные

### `MODE`

**Описание**:
Режим работы приложения. По умолчанию установлен в `'dev'`.
-   `MODE` (str): строка `dev`

### `__root__`

**Описание**:
Путь к корневой директории проекта, полученный с помощью функции `set_project_root`.
-   `__root__` (Path): путь к корневой директории

### `settings`

**Описание**:
Словарь с настройками проекта, загруженный из файла `settings.json`. Если файл не найден или содержит ошибку JSON, значение переменной будет `None`.
-   `settings` (dict | None): словарь с настройками проекта или `None` если файла нет.

### `doc_str`

**Описание**:
Строка с содержимым файла `README.MD`. Если файл не найден или не может быть прочитан, значение переменной будет `None`.
-   `doc_str` (str | None): Строка с документацией или `None` если файла нет.

### `__project_name__`

**Описание**:
Имя проекта, полученное из настроек (если есть) или `'hypotez'` по умолчанию.
-   `__project_name__` (str): Имя проекта

### `__version__`

**Описание**:
Версия проекта, полученная из настроек (если есть) или `''` по умолчанию.
-  `__version__` (str): Версия проекта

### `__doc__`

**Описание**:
Строка с документацией из файла `README.MD` или `''` по умолчанию.
-   `__doc__` (str): Строка документации

### `__details__`

**Описание**:
Детали проекта. По умолчанию `''`.
-  `__details__` (str): Детали проекта

### `__author__`

**Описание**:
Автор проекта, полученный из настроек (если есть) или `''` по умолчанию.
-   `__author__` (str): Автор проекта.

### `__copyright__`

**Описание**:
Информация об авторских правах, полученная из настроек (если есть) или `''` по умолчанию.
-   `__copyright__` (str): Информация об авторских правах.

### `__cofee__`

**Описание**:
Сообщение для пожертвования разработчику, полученное из настроек (если есть), или стандартное сообщение с ссылкой на boosty.
-   `__cofee__` (str): Сообщение с ссылкой для пожертвования.
