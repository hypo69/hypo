# Задача No 230
## Fibonacci Words

## Условия
Определим последовательность слов Fi, используя следующие правила:

F(0) = "a"
F(1) = "b"
F(n) = F(n-1) + F(n-2) для n > 1

Таким образом, первые слова последовательности:
F(0) = "a"
F(1) = "b"
F(2) = "ba"
F(3) = "bab"
F(4) = "baba"
F(5) = "babab"
F(6) = "bababab"
F(7) = "babababa"
F(8) = "babababab"
F(9) = "bababababa"
...

Заметим, что конкатенация слов F(n) + F(n-1) также образует слово F(n+1).

Пусть C(n) — n-й символ в бесконечном слове, полученном путем конкатенации всех слов Fi, то есть, в слове:

ababaababbababaabaababbabab...

C(1) = a, C(2) = b, C(3) = a, C(4) = b, C(5) = a, C(6) = a, C(7) = b и т.д.

Найдите C(10<sup>18</sup>).
