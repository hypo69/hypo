# Модуль _experiments

## Обзор

Модуль содержит экспериментальный код, связанный с поставщиком Kualastyle. Включает в себя импорты различных модулей и библиотек, необходимых для работы с веб-драйвером, поставщиками, продуктами и категориями. Также содержит функцию `start_supplier` для запуска поставщика с определенными параметрами.

## Подробней

Этот модуль предназначен для экспериментов и отладки кода, связанного с поставщиком Kualastyle. Он содержит различные импорты и настройки, необходимые для работы с веб-драйвером, поставщиками, продуктами и категориями. Функция `start_supplier` позволяет запускать поставщика с заданными параметрами, такими как префикс поставщика и локаль.

## Функции

### `start_supplier`

```python
def start_supplier(supplier_prefix: str = 'aliexpress', locale: str = 'en'):
    """ Старт поставщика """
```

**Назначение**: Запускает поставщика с заданными параметрами.

**Параметры**:
- `supplier_prefix` (str): Префикс поставщика. По умолчанию `'aliexpress'`.
- `locale` (str): Локаль поставщика. По умолчанию `'en'`.

**Возвращает**:
- `Supplier`: Объект поставщика, созданный с заданными параметрами.

**Как работает функция**:

1.  Определяет параметры поставщика в виде словаря, используя переданные аргументы `supplier_prefix` и `locale`.
2.  Создает и возвращает экземпляр класса `Supplier` с использованием определенных параметров.

```ascii
    start_supplier
    │
    ├── Создание словаря параметров
    │   params = {'supplier_prefix': supplier_prefix, 'locale': locale}
    │
    └── Создание и возврат объекта Supplier
        Supplier(**params)
```

**Примеры**:

```python
# Запуск поставщика с префиксом 'aliexpress' и локалью 'en'
supplier = start_supplier()

# Запуск поставщика с префиксом 'kualastyle' и локалью 'ru'
supplier = start_supplier(supplier_prefix='kualastyle', locale='ru')
```
