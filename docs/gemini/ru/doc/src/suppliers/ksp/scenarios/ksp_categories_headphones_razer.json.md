## Обзор

Файл `ksp_categories_headphones_razer.json` содержит JSON-конфигурацию для определения сценариев парсинга категорий наушников бренда RAZER в интернет-магазине KSP. В файле описан один сценарий для наушников типа "In-ear Bud".

## Структура JSON

### `scenarios`

Этот раздел содержит описание сценариев парсинга. Ключи этого раздела соответствуют названиям типов наушников.

#### `In-ear Bud`

**Описание**: Сценарий парсинга для наушников типа "In-ear Bud" бренда RAZER.

**Поля**:
- `brand` (str): Название бренда, "RAZER".
- `url` (str): URL-адрес страницы с категорией товара на сайте KSP.
- `checkbox` (bool): Флаг, указывающий на использование чекбокса (в данном случае `false`).
- `active` (bool): Флаг, указывающий на активность сценария (в данном случае `true`).
- `condition` (str): Условие товара, в данном случае "new" (новый).
- `presta_categories` (dict): Раздел для настройки категорий PrestaShop.
    - `template` (dict): Шаблон для сопоставления категорий.
        - `razer` (str): Категория PrestaShop для бренда Razer, в данном случае "In-ear Bud".
