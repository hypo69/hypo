# Документация для `visualdg_categories_cases_asus.json`

## Обзор

Данный файл содержит JSON-структуру с описаниями сценариев для категорий товаров бренда ASUS, используемых в VisualDG. Каждый сценарий содержит информацию о бренде, шаблоне, URL, активности, состоянии товара и соответствующих категориях PrestaShop.

## Структура JSON

### Сценарии

Секция `scenarios` содержит объекты, представляющие собой конкретные сценарии.

#### `ASUS CASE`

**Описание**: Сценарий для товаров ASUS.

**Параметры**:

- `brand` (str): Бренд товара - `"ASUS"`.
- `template` (str): Шаблон (пустая строка).
- `url` (str): URL товара - `"https://www.visualdg.co.il/169420-%D7%9E%D7%90%D7%A8%D7%96%D7%99-ASUS/243216-Asus"`.
- `checkbox` (bool): Статус чекбокса - `false`.
- `active` (bool): Статус активности сценария - `true`.
- `condition` (str): Состояние товара - `"new"`.
- `presta_categories` (str): Категории PrestaShop, к которым относится товар - `"195,534"`.
