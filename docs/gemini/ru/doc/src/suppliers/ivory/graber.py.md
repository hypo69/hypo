# Модуль `graber.py`

## Обзор

Модуль `graber.py` предназначен для сбора информации о товарах с веб-сайта `ivory.co.il`. Он содержит класс `Graber`, который наследуется от родительского класса `Graber` (Grbr) и переопределяет некоторые методы для обработки специфичных полей на странице товара. Модуль также включает функциональность для работы с веб-драйвером и логирование ошибок.

## Подробней

Основная цель данного модуля - автоматизированный сбор данных о товарах с сайта `ivory.co.il`. Он использует веб-драйвер для навигации по страницам товаров и извлечения необходимой информации. Для обработки нестандартных ситуаций, связанных с особенностями разметки сайта, в классе `Graber` перегружаются методы родительского класса.

Модуль также предоставляет возможность выполнения предварительных действий перед отправкой запроса к веб-драйверу с помощью декораторов. Декоратор по умолчанию находится в родительском классе. Для его использования необходимо передать значение в `Context.locator`. При необходимости можно переопределить поведение декоратора, раскомментировав соответствующие строки кода.

## Классы

### `Graber`

**Описание**: Класс `Graber` предназначен для захвата информации о товарах с сайта `ivory.co.il`. Он наследуется от класса `Graber` (Grbr) и содержит методы для обработки полей товаров.

**Методы**:
- `__init__`: Инициализирует экземпляр класса `Graber`, устанавливает префикс поставщика и вызывает конструктор родительского класса.

**Параметры**:
- `driver` (Driver): Экземпляр веб-драйвера для управления браузером.
- `lang_index` (int): Индекс языка.

**Примеры**:

```python
from src.webdriver.driver import Driver
from src.suppliers.ivory.graber import Graber

# Пример создания экземпляра класса Graber
driver = Driver()
graber = Graber(driver, lang_index=0)
```
