# Задача No 621
## Красивые числа

## Условия

Положительное целое число называется красивым, если все его цифры различны и сумма его цифр равна 10. Например, 1234 является красивым, а 1333 и 124 не являются красивыми.
Пусть $F(n)$ будет количеством красивых чисел, которые меньше чем $10^n$.
Например, $F(1) = 0, F(2) = 0, F(3) = 0, F(4) = 2, F(5) = 8, F(6) = 44, F(7) = 194$.
Пусть $S(n) = \sum_{k=1}^n F(k)$.
Например, $S(7) = 248$.
Найдите $S(2023) \bmod 1234567891$.
