# Модуль `test_jupyter_examples.py`

## Обзор

Этот модуль содержит тесты для выполнения Jupyter ноутбуков. Он ищет все `.ipynb` файлы в указанной папке, выполняет их и сохраняет результаты.

## Функции

### `get_notebooks`

**Описание**: Возвращает список путей к Jupyter ноутбукам в заданной папке.

**Параметры**:
- `folder` (str): Путь к папке, содержащей ноутбуки.

**Возвращает**:
- list[str]: Список путей к файлам ноутбуков.

**Вызывает исключения**:
- `FileNotFoundError`: Возникает, если заданная папка не найдена.


### `test_notebook_execution`

**Описание**: Выполняет Jupyter ноутбук и проверяет, что не возникло никаких исключений.

**Параметры**:
- `notebook_path` (str): Путь к файлу Jupyter ноутбука.

**Возвращает**:
- None

**Вызывает исключения**:
- `Exception`:  Возникает, если при выполнении ноутбука произошла ошибка. Сообщение об ошибке будет использовано для отображения.
- Другие исключения Python.


## Константы

### `NOTEBOOK_FOLDER`

**Описание**: Путь к папке, содержащей Jupyter ноутбуки. Значение по умолчанию: "../examples/".

**Тип**: str


### `TIMEOUT`

**Описание**: Таймаут для долго выполняющихся ноутбуков. Значение по умолчанию: 600 секунд.

**Тип**: int


### `KERNEL_NAME`

**Описание**: Имя ядра для выполнения ноутбука. Значение по умолчанию: "python3".

**Тип**: str


## Модули

### `os`

**Описание**: Модуль для работы с операционной системой.

### `nbformat`

**Описание**: Модуль для работы с форматом Jupyter Notebook.

### `nbconvert.preprocessors`

**Описание**: Модуль для предобработки Jupyter ноутбуков.

### `pytest`

**Описание**: Фреймворк для написания модульных тестов.

### `sys`

**Описание**: Модуль для работы с системой.


## Использование

Для использования модуля необходимо:

1. Убедиться, что в папке, где запускается скрипт, есть папка `../examples/` с вашими Jupyter ноутбуками.
2. Запустить скрипт с помощью pytest.


```
pytest test_jupyter_examples.py
```

Это выполнит все Jupyter ноутбуки в указанной папке и выведет информацию о статусе выполнения каждого из них. Если произошла ошибка, то pytest отобразит соответствующее сообщение.
