# Задача No 348
## Числа-палиндромы, которые могут быть записаны как сумма квадратов и кубов
## Условия

Некоторые целые числа могут быть записаны как сумма квадратов и кубов. Первые несколько таких чисел:

```
1 = 0^2 + 1^3
2 = 1^2 + 1^3
3 = 1^2 + 2^3
4 = 2^2 + 0^3
5 = 1^2 + 2^3
8 = 2^2 + 2^3
9 = 0^2 + 9^3 = 3^2 + 0^3
...
```

Некоторые из этих чисел также являются палиндромами. Первые пять палиндромных чисел, которые могут быть записаны как сумма квадратов и кубов, имеют вид:

```
1 = 0^2 + 1^3
2 = 1^2 + 1^3
3 = 1^2 + 2^3
4 = 2^2 + 0^3
5 = 1^2 + 2^3
```

Обратите внимание, что `1 = 1^2 + 0^3 = 0^2 + 1^3` на самом деле считается только один раз.

Сколько составляет сумма пятерки наименьших палиндромных чисел, которые можно записать как сумму квадрата и куба, где квадрат и куб – строго положительные целые числа?
