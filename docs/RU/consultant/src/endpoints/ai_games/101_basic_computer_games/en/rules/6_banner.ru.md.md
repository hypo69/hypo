Описание
========

### Инструкция для модели ИИ: Реализация игры "Banner"

#### Описание игры:
"Banner" — это программа, которая создает большой баннер из текста, введенного пользователем. 
Игрок вводит сообщение, а компьютер выводит его в виде больших букв, состоящих из символов. 
Каждая буква может быть нарисована с использованием определенных символов, таких как `*`, `#`, или других.

#### Основные правила:
1. Игрок вводит сообщение, которое хочет преобразовать в баннер.
2. Компьютер преобразует каждую букву сообщения в соответствующий набор символов, создавая большие буквы.
3. Результат выводится на экран в виде баннера.

#### Инструкция для модели ИИ:
1. **Инициализация игры:**
   - Создайте словарь, где ключи - это буквы алфавита (A-Z) и цифры (0-9), а значения - это списки строк, представляющие матрицу символов для каждой буквы.
   - Каждая буква должна быть представлена в виде матрицы символов (например, 5 строк на 5 столбцов).
   - Определите константы `LETTER_HEIGHT` и `LETTER_WIDTH` для размера матрицы символов.

2. **Основной цикл игры:**
   - Запросите у игрока сообщение, которое нужно преобразовать в баннер.
   - Для каждой буквы в сообщении, найдите соответствующую матрицу символов.
   - Выведите каждую строку матрицы символов для всех букв сообщения, обеспечив пробелы между буквами.
   - Реализуйте повторный запуск игры, если пользователь захочет создать еще один баннер.
   - Используйте функции для разделения логики (например, `get_user_input`, `create_letter_banner`, `display_banner`).

3. **Обработка ввода игрока:**
    - Убедитесь, что ввод игрока содержит только буквы и цифры (регистр не должен влиять на результат). Пробелы можно игнорировать или использовать для разделения слов.
    - Если ввод некорректен, попросите игрока повторить ввод.
    - Примените приведение к верхнему регистру для обработки ввода.

4. **Создание баннера:**
    - Для каждой буквы в сообщении, найдите соответствующую матрицу символов в словаре.
    - Выведите каждую строку матрицы символов для всех букв сообщения, добавив пробел между буквами для читаемости.
    - Выведите весь баннер, объединив все строки.

5.  **Завершение игры:**
    - После вывода баннера, спросите игрока, хочет ли он создать еще один баннер.
    - Если игрок соглашается, повторите процесс.
    - При завершении игры выведите сообщение о благодарности.
