# Модуль `hypotez/src/suppliers/aliexpress/api/models/hotproducts.py`

## Обзор

Этот модуль определяет класс `HotProductsResponse`, который представляет собой ответ API для получения горячих продуктов с AliExpress.  Класс содержит информацию о текущей странице, количестве записей на странице и общее количество записей, а также список продуктов (`List[Product]`).


## Классы

### `HotProductsResponse`

**Описание**: Класс `HotProductsResponse` представляет собой структуру данных для хранения ответа API AliExpress о горячих продуктах. Он содержит информацию о текущей странице, количестве записей на странице, общее количество записей и список продуктов.


**Атрибуты**:

- `current_page_no` (int): Номер текущей страницы в результатах.
- `current_record_count` (int): Количество записей на текущей странице.
- `total_record_count` (int): Общее количество записей.
- `products` (List[Product]): Список объектов `Product`, представляющих горячие продукты.


**Примечание**: Класс `Product` предполагается определённым в другом модуле (например, `hypotez/src/suppliers/aliexpress/api/models/product.py`).  Его использование в этом модуле предполагает, что `Product` корректно импортирован.
