# CONTRIBUTING.md

## Обзор

Этот файл содержит инструкции для внесения вклада в проект gpt4free. Он описывает шаги, которые необходимо выполнить для добавления поддержки новых веб-сайтов, использующих GPT-4.

## Подробней

Этот файл предоставляет руководство для разработчиков, желающих расширить функциональность gpt4free, добавляя поддержку новых веб-сайтов. Он содержит информацию о том, как начать работу, какие шаги необходимо выполнить и где разместить файлы.

## Инструкции по внесению вклада

### 1. Реверс-инжиниринг веб-сайта

Выберите веб-сайт из списка [sites-to-reverse](https://github.com/xtekky/gpt4free/issues/40), который вы хотите добавить в gpt4free. Проведите реверс-инжиниринг этого веб-сайта, чтобы понять, как он взаимодействует с GPT-4.

### 2. Добавление в unittest

Добавьте информацию о реверс-инжиниринге веб-сайта в каталог [./etc/unittest/](https://github.com/xtekky/gpt4free/tree/main/etc/unittest/). Это необходимо для автоматизированного тестирования и проверки правильности работы добавленной поддержки.

### 3. Рефакторинг и добавление в g4f

Отрефакторите код и добавьте его в каталог [./g4f](https://github.com/xtekky/gpt4free/tree/main/g4f). Убедитесь, что код соответствует стандартам проекта и хорошо документирован.

## Благодарность
