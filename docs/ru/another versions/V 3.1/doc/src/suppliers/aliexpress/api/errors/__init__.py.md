# Модуль `src.suppliers.aliexpress.api.errors`

## Обзор

Модуль `src.suppliers.aliexpress.api.errors` предназначен для обработки ошибок, возникающих при взаимодействии с API AliExpress. Он импортирует все исключения, определенные в подмодуле `exceptions`, чтобы обеспечить централизованный доступ к ним.

## Подробней

Этот модуль играет важную роль в обработке ошибок, возникающих при взаимодействии с API AliExpress. Он предоставляет набор исключений, которые могут быть использованы для обработки различных типов ошибок, таких как сетевые ошибки, ошибки аутентификации и ошибки, связанные с данными.
