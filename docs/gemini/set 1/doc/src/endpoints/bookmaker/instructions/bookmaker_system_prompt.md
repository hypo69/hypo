# ИНСТРУКЦИЯ ПО ИСПОЛЬЗОВАНИЮ ПРОМПТА ДЛЯ МОДЕЛИ МАШИННОГО ОБУЧЕНИЯ

## Обзор

Данная документация описывает промпт для модели машинного обучения, предназначенный для анализа входных данных и определения возможности их использования для прогнозирования. Промпт анализирует различные типы входных данных (временные ряды, табличные данные, текстовые данные) и определяет, можно ли на основе этих данных сделать прогноз. Если это возможно, промпт выполняет анализ прогноза, включая выявление паттернов, трендов и определение периода прогнозирования. Результат возвращается в формате JSON.

## Структура промпта

### Входные данные

Промпт принимает различные типы входных данных, включая:

* Временные ряды чисел (например, `[12, 14, 15, 20, 25, ...]`).
* Табличные данные с несколькими столбцами (например, даты, продажи, категории).
* Текстовые описания с метаданными.

### Промпт

**Задача:**

1. **Проанализировать входные данные** и определить, возможно ли их использовать для прогнозирования.
2. Если данные подходят для прогнозирования, выполните **анализ прогноза**, включая:
    * Определение возможных зависимостей или трендов.
    * Оценку значимости данных для прогноза.
    * Пример прогноза, если это применимо.

### Формат ответа

Результат предоставляется в формате JSON:

```json
{
  "is_forecastable": true,
  "analysis": {
    "patterns_detected": "описание выявленных паттернов",
    "trend": "описание тренда (если применимо)",
    "forecast_period": "период, на который можно прогнозировать (если применимо)"
  },
  "forecast": "примерный прогноз, если возможно, иначе null"
}
```

### Обработка случаев, когда прогноз невозможен

Если входные данные недостаточны для прогнозирования, промпт возвращает:

```json
{
  "is_forecastable": false,
  "analysis": {
    "reason": "Недостаточно данных или структура данных не подходит для прогнозирования"
  },
  "forecast": null
}
```

## Примеры входных данных и результатов

**Пример 1 (временной ряд):**

**Входные данные:** `[12, 14, 15, 20, 25, ...]`

**Возможный результат (если прогноз возможен):**

```json
{
  "is_forecastable": true,
  "analysis": {
    "patterns_detected": "возрастающая тенденция",
    "trend": "постоянный рост",
    "forecast_period": "следующие 5 дней"
  },
  "forecast": "[28, 31, 34, 37, 40]"
}
```

**Пример 2 (недостаточно данных):**

**Входные данные:** `[12, 14]`

**Возможный результат:**

```json
{
  "is_forecastable": false,
  "analysis": {
    "reason": "Недостаточно данных для определения тренда"
  },
  "forecast": null
}
```

## Важные замечания

* Промпт не должен делать предположений, если входные данные явно не содержат временной или прогностической структуры.
* Промпт использует статистический и контекстуальный анализ для проверки возможности прогнозирования.
* При недостаточности входных данных для прогнозирования, промпт аргументирует это.
