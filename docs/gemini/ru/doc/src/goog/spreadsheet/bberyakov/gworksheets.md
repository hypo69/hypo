# Модуль gworksheets

## Обзор

Модуль `gworksheets` предоставляет инструменты для работы с электронными таблицами, интегрированными с библиотекой Google Sheets API. Он позволяет создавать, изменять и получать доступ к рабочим листам (worksheet), управлять их заголовками, категориями и направлением текста.

## Классы

### `GWorksheet`

**Описание**: Класс `GWorksheet` наследуется от `Worksheet` и предоставляет методы для работы с отдельными рабочими листами в книге Google Sheets.  Он обеспечивает абстракцию для создания, получения и управления листами.

**Атрибуты**:

- `sh`: Объект, представляющий книгу Google Sheets.
- `ws`: Объект, представляющий текущий рабочий лист.
- `render`: Объект `GSRender` для управления форматированием и отображением данных на листе.

**Методы**:

#### `__init__(self, sh, ws_title='new', rows=None, cols=None, direcion='rtl', wipe_if_exist=True, *args, **kwards)`

**Описание**: Инициализирует объект `GWorksheet`.

**Параметры**:
- `sh` (объект): Объект, представляющий книгу Google Sheets.
- `ws_title` (строка, по умолчанию 'new'): Название рабочего листа.
- `rows` (целое число, по умолчанию None): Количество строк на листе.
- `cols` (целое число, по умолчанию None): Количество столбцов на листе.
- `direcion` (строка, по умолчанию 'rtl'): Направление текста на листе ('rtl' или 'ltr').
- `wipe_if_exist` (булево, по умолчанию True): Удалять данные на листе, если он уже существует.
- `*args`, `**kwards`: Дополнительные аргументы и ключевые аргументы.


**Возвращает**:
- `None`


#### `get(self, sh, ws_title='new', rows=100, cols=100, direction='rtl', wipe_if_exist=True)`

**Описание**: Создает или открывает рабочий лист.

**Параметры**:
- `sh` (объект): Объект, представляющий книгу Google Sheets.
- `ws_title` (строка, по умолчанию 'new'): Название рабочего листа.
- `rows` (целое число, по умолчанию 100): Количество строк на листе.
- `cols` (целое число, по умолчанию 100): Количество столбцов на листе.
- `direction` (строка, по умолчанию 'rtl'): Направление текста на листе ('rtl' или 'ltr').
- `wipe_if_exist` (булево, по умолчанию True): Удалять данные на листе, если он уже существует.


**Возвращает**:
- `None`


#### `header(self, world_title, range='A1:Z1', merge_type='MERGE_ALL')`

**Описание**: Устанавливает заголовок для рабочего листа.


**Параметры**:
- `world_title` (строка): Заголовок.
- `range` (строка, по умолчанию 'A1:Z1'): Диапазон ячеек для заголовка.
- `merge_type` (строка, по умолчанию 'MERGE_ALL'): Тип слияния ячеек ('MERGE_ALL', 'MERGE_COLUMNS', 'MERGE_ROWS').


**Возвращает**:
- `None`


#### `category(self, ws_category_title)`

**Описание**: Записывает заголовок категории на рабочий лист.

**Параметры**:
- `ws_category_title` (любой тип): Название категории.

**Возвращает**:
- `None`


#### `direction(self, direction='rtl')`

**Описание**: Устанавливает направление текста на рабочем листе.

**Параметры**:
- `direction` (строка, по умолчанию 'rtl'): Направление текста ('rtl' или 'ltr').


**Возвращает**:
- `None`
