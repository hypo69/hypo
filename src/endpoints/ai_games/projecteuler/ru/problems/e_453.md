# Задача No 453
## Гибкие четырёхугольники
## Условия
На плоскости задан набор из *n* точек. Пусть F(*n*) - количество четырёхугольников, которые могут быть образованы из этих *n* точек, причём:

  - все четыре вершины находятся среди данных *n* точек
  - никакие три точки из четырёх не лежат на одной прямой
  - никакие две стороны не пересекаются
  - все углы не являются прямыми углами и не равны 180 градусам

Например, при *n* = 4, существует только один такой четырёхугольник: выпуклый четырёхугольник, образованный четырьмя точками. А для *n* = 5, существуют 5 таких четырёхугольников (как выпуклые, так и невыпуклые).

Определим G(*n*) как количество троек (*a*,*b*,*c*), удовлетворяющих 4 ≤ *a* < *b* < *c* ≤ *n*, таких что F(*a*) + F(*b*) = F(*c*).

Можно проверить, что G(10) = 1.

Найдите G(1000).
