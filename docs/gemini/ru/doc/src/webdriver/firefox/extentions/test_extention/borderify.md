# borderify.js

## Обзор

Этот скрипт изменяет стиль границы элемента `body`. Он устанавливает границу в 5 пикселей красного цвета.  Также выводит диалоговое окно с помощью `alert()`.


## Код

```javascript
document.body.style.border = "5px solid red";
alert()
```
