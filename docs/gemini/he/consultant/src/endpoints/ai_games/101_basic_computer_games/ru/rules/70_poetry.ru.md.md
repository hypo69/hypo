# Игра: POETRY (Поэзия)

---

#### Описание
**POETRY** — это игра, в которой игроки соревнуются в создании стихотворений. Каждый игрок по очереди добавляет строку к общему стихотворению. Цель игры — создать законченное стихотворение, которое будет звучать гармонично и соответствовать определённым правилам. Игра заканчивается, когда стихотворение будет завершено, и игроки голосуют за лучшую строку.

---

### Пошаговая инструкция для реализации

#### 1. **Инициализация игры**
   - Программа выводит приветственное сообщение и объясняет правила:
     ```
     Добро пожаловать в POETRY!
     Ваша задача — создать стихотворение, добавляя строки по очереди.
     Каждый игрок добавляет одну строку.
     Игра заканчивается, когда стихотворение будет завершено.
     Удачи!
     ```

   - Программа предлагает игрокам ввести свои имена.
   - Игроки поочерёдно делают ходы, добавляя строки к стихотворению.

---

#### 2. **Основной процесс игры**

##### **2.1. Ход игрока:**
   - Игрок добавляет строку к стихотворению.
   - Программа проверяет, соответствует ли строка правилам (например, длина строки, рифма, тематика).
   - Если строка соответствует правилам, она добавляется к стихотворению.
   - Если строка не соответствует правилам, программа сообщает об ошибке и предлагает игроку повторить ход:
     ```
     Строка не соответствует правилам. Попробуйте снова.
     ```

   - Программа отображает текущее состояние стихотворения:
     ```
     Текущее стихотворение:
     1. В лесу родилась ёлочка,
     2. В лесу она росла.
     ```

##### **2.2. Завершение стихотворения:**
   - Игра заканчивается, когда стихотворение будет завершено (например, после 4 строк).
   - Программа объявляет стихотворение и предлагает игрокам проголосовать за лучшую строку.
   - Игроки вводят номера строк, которые они считают лучшими.
   - Программа подсчитывает голоса и объявляет победителя:
     ```
     Победил Игрок 1 с строкой "В лесу родилась ёлочка".
     ```

---

#### 3. **Завершение игры**
   - После завершения игры программа предлагает сыграть снова:
     ```
     Хотите сыграть снова? (да/нет)
     ```

   - Если игрок выбирает "да", игра начинается заново с нового стихотворения.

---

### Пример работы программы

1. **Начало игры:**
   ```
   Добро пожаловать в POETRY!
   Введите ваши имена:
   Игрок 1: Алиса
   Игрок 2: Борис
   Алиса, ваш ход.
   Введите строку:
   > В лесу родилась ёлочка,
   Строка добавлена.
   ```

2. **Игровой процесс:**
   ```
   Борис, ваш ход.
   Введите строку:
   > В лесу она росла.
   Строка добавлена.

   Алиса, ваш ход.
   Введите строку:
   > Зимой и летом стройная,
   Строка добавлена.
   ```

3. **Завершение игры:**
   ```
   Стихотворение завершено:
     1. В лесу родилась ёлочка,
     2. В лесу она росла.
     3. Зимой и летом стройная,
     4. Зелёная была.

   Голосуйте за лучшую строку (введите номер строки):
   Алиса: 1
   Борис: 1

   Победил Алиса с строкой "В лесу родилась ёлочка".
   Хотите сыграть снова? (да/нет):
   > нет
   Спасибо за игру!
   ```

### Возможные ограничения
- Игрок должен вводить строку в правильном формате (например, длина строки не должна превышать 50 символов).
- Программа должна обрабатывать неверный ввод и предлагать повторить попытку.
- Строки должны соответствовать определённым правилам (например, рифма, тематика).
