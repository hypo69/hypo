## Описание локатора

### `link`

**Описание**:
Объект `link` определяет параметры локатора для поиска ссылок на состоявшиеся беседы.

**Параметры**:

- `attribute` (str): Атрибут, который следует использовать для поиска элемента. В данном случае - `href`.
- `by` (str): Метод поиска элемента, в данном случае - `XPATH`.
- `selector` (str): XPATH-селектор для поиска элемента. В данном случае - `//a[@data-discover = 'true']`.
- `timeout` (int): Время ожидания элемента в секундах. В данном случае - `0`.
- `timeout_for_event` (str): Тип события для таймаута. В данном случае - `presence_of_element_located`.
- `event` (None): Событие для ожидания (не используется в данном случае).
- `if_list` (str): Определяет, как обрабатывать список найденных элементов. В данном случае - `all`.
- `use_mouse` (bool): Указывает, нужно ли использовать мышь для взаимодействия с элементом. В данном случае - `false`.
- `mandatory` (bool): Указывает, является ли наличие элемента обязательным. В данном случае - `true`.
- `locator_description` (str): Описание локатора, например, "Ссылки на состоявшиеся беседы".
