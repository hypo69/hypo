# Модуль `hypotez/src/suppliers/hb/category.py`

## Обзор

Модуль `hypotez/src/suppliers/hb/category.py` отвечает за сбор информации о товарах с сайта поставщика `hb.co.il`. Он содержит функции для получения списка категорий, списка товаров в категории и обработки страниц товаров.  Этот модуль реализует специфичный для поставщика `hb.co.il` сценарий работы с категориями.

## Функции

### `get_list_products_in_category`

**Описание**: Функция собирает список URL-адресов товаров, находящихся на странице категории.  Она обрабатывает возможную навигацию по страницам с товарами.

**Параметры**:

- `s` (Supplier): Экземпляр класса `Supplier`, содержащий необходимые данные для работы с сайтом поставщика.

**Возвращает**:

- `list[str, str, None]`: Список URL-адресов товаров или `None`, если на странице не обнаружено ссылок на товары.  Возвращаемый список может содержать как отдельные URL, так и списки URL (если обнаружено несколько страниц с товарами).

**Вызывает исключения**:

- Возможны исключения, генерируемые методами `Driver`, если возникают проблемы с взаимодействием с веб-драйвером.

### `paginator`

**Описание**: Функция обрабатывает навигацию по страницам товаров, если таковая имеется.

**Параметры**:

- `d` (Driver): Экземпляр класса `Driver`, управляющий веб-драйвером.
- `locator` (dict): Словарь с локаторами для элементов на странице.
- `list_products_in_category` (list): Список URL-адресов товаров.

**Возвращает**:

- `bool`: `True`, если страница товаров содержит элементы для перехода к следующей странице. `False` если  на странице нет элементов для перехода к следующей странице, или нет элементов для перехода на предыдущую страницу.

**Вызывает исключения**:

- Возможны исключения, генерируемые методами `Driver`, если возникают проблемы с взаимодействием с веб-драйвером.
