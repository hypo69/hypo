# Задача No 423
## Капли воды
## Условия
Назовем последовательность чисел $a_1, a_2, a_3, \dots$ *последовательностью капель воды* (water drop sequence), если выполняются следующие условия:

*   $a_1 = 1$
*   Для всех $n > 1$, $a_n = a_{n-1} + \text{minDigit}(a_{n-1})$, где $\text{minDigit}(x)$ это наименьшая ненулевая цифра десятичного представления числа $x$.

Например, первые пять элементов последовательности: 1, 2, 4, 8, 16.

Назовем *суммой капель* $S(n)$ сумму первых $n$ элементов последовательности.

$S(5) = 1+2+4+8+16=31$.

Также определим функцию $T(n) = \sum_{i=1}^n S(i)$.

$T(5) = S(1) + S(2) + S(3) + S(4) + S(5) = 1 + 3 + 7 + 15 + 31 = 57$.

Найдите $T(2 \cdot 10^{16}) \bmod 10^9$.
