# Модуль hypotez/src/gui/openai_trаigner/version.py

## Обзор

Этот модуль содержит информацию о версии проекта, а также другие метаданные.

## Переменные

### `MODE`

**Описание**: Строковая переменная, хранящая режим работы приложения (например, 'dev' или 'prod').

**Значение**: 'dev'

### `__version__`

**Описание**: Строковая переменная, содержащая версию приложения.

**Значение**: 'v1.1'


### `__doc__`

**Описание**: Строковая переменная, содержащая документацию модуля.

**Значение**: "" (пустая строка)

### `__details__`

**Описание**: Строковая переменная, содержащая дополнительные детали о модуле.

**Значение**: "" (пустая строка)

### `__author__`

**Описание**: Строковая переменная, содержащая имя автора модуля.

**Значение**: 'hypo69'

### `__copyright__`

**Описание**: Строковая переменная, содержащая информацию о копирайтах.

**Значение**: Строка с информацией о лицензии MIT и запрете на коммерческое использование без разрешения.

### `__cofee__`

**Описание**: Строковая переменная, содержащая ссылку на поддержку разработчика через платформа boosty.to.
