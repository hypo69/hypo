# Модуль `xls`

## Обзор

Модуль `xls` предоставляет функции для конвертации Excel (xls) файлов в словари Python и обратно.

## Оглавление

- [Функции](#функции)
    - [`xls2dict`](#xls2dict)

## Функции

### `xls2dict`

**Описание**: Преобразует xls файл в словарь.

**Параметры**:
- `xls_file` (str | Path): Путь к xls файлу.

**Возвращает**:
- `dict | None`: Словарь, полученный из xls файла, или `None` в случае ошибки.
