# Модуль `hypotez/src/suppliers/aliexpress/gui/campaign.py`

## Обзор

Данный модуль предоставляет класс `CampaignEditor` для редактирования кампаний AliExpress. Он позволяет загружать JSON-файлы, отображать их содержимое в виде виджетов и запускать асинхронную подготовку кампании.

## Классы

### `CampaignEditor`

**Описание**: Класс `CampaignEditor` представляет собой виджет PyQt для редактирования данных кампании AliExpress. Он содержит интерфейс пользователя для выбора файла, загрузки данных, отображения и редактирования полей.

**Атрибуты**:

- `data`: Объект `SimpleNamespace`, содержащий загруженные данные кампании.
- `current_campaign_file`: Путь к текущему загруженному файлу.
- `editor`: Объект `AliCampaignEditor` для подготовки кампании.

**Методы**:

#### `__init__(self, parent=None, main_app=None)`

**Описание**: Инициализирует виджет `CampaignEditor`.

**Параметры**:
- `parent` (optional): Родительский виджет.
- `main_app` (optional): Объект главного приложения.

#### `setup_ui(self)`

**Описание**: Настраивает пользовательский интерфейс виджета.

#### `setup_connections(self)`

**Описание**: Устанавливает связи между сигналами и слотами.

#### `open_file(self)`

**Описание**: Открывает диалог выбора файла для загрузки JSON-файла.

**Возвращает**:  Не имеет возвращаемого значения.

#### `load_file(self, campaign_file)`

**Описание**: Загружает JSON-файл и отображает данные в виджете.

**Параметры**:
- `campaign_file` (str): Путь к загружаемому файлу.

**Возвращает**: Не имеет возвращаемого значения.

**Обрабатываемые исключения**:
- Любые исключения при загрузке JSON: Выводится сообщение об ошибке.

#### `create_widgets(self, data)`

**Описание**: Создает виджеты на основе данных, загруженных из JSON-файла.

**Параметры**:
- `data` (SimpleNamespace): Данные кампании.

**Возвращает**: Не имеет возвращаемого значения.


#### `prepare_campaign(self)`

**Описание**: Асинхронно запускает подготовку кампании.

**Возвращает**: Не имеет возвращаемого значения.

**Обрабатываемые исключения**:
- Любые исключения при подготовке кампании: Выводится сообщение об ошибке.
