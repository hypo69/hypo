# Модуль src.utils.autodoc

## Обзор

Модуль `src.utils.autodoc` содержит декоратор `autodoc`, который автоматически обновляет строку документации функции, добавляя время последнего вызова.  Декоратор оборачивает функцию, обновляя её docstring перед каждым вызовом, чтобы отразить актуальное время.

## Функции

### `autodoc`

**Описание**: Декоратор, добавляющий информацию о времени последнего вызова функции в её строку документации.

**Параметры**:
- `func` (callable): Функция, которая будет обернута декоратором.


**Возвращает**:
- `wrapper` (callable): Обёрнутая функция с обновлённым docstring.


**Вызывает исключения**:
-  Нет.

### `update_docstring`

**Описание**: Функция, обновляющая строку документации функции, добавляя в неё время последнего вызова.

**Параметры**:
- `func` (callable): Функция, чья docstring будет обновлена.

**Возвращает**:
- Нет.


**Вызывает исключения**:
- Нет.


### `example_function`

**Описание**: Пример функции, использующей декоратор `autodoc`.  При каждом вызове функция печатает сообщение и обновляет свой docstring с информацией о времени вызова.

**Параметры**:
- `param1` (int): Первое значение.
- `param2` (str): Второе значение.

**Возвращает**:
- None

**Вызывает исключения**:
- Нет.

## Пример использования

```python
@autodoc
def example_function(param1: int, param2: str) -> None:
    """Пример функции.

    Args:
        param1 (int): Первое значение.
        param2 (str): Второе значение.
    """
    print(f"Processing {param1} and {param2}")

example_function(1, "test")
print(example_function.__doc__)  # Вывод обновленного docstring
example_function(2, "another test")
print(example_function.__doc__)  # Вывод обновленного docstring
```

Этот пример демонстрирует, как использовать декоратор `autodoc` для автоматического обновления docstring функции. Каждый раз, когда вызывается функция `example_function`, её docstring обновляется с указанием времени последнего вызова.
