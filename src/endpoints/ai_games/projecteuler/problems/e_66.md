# Задача No 66
## Диофантовы уравнения

## Условия

Рассмотрим квадратичное диофантово уравнение вида:

x² – Dy² = 1

Например, когда D=13, минимальным решением x является 649² – 13×180² = 1.

Заметьте:

3² – 2×2² = 1
17² – 2×12² = 1
197² – 13×54² = 1
661² – 23×130² = 1

И так как x² – Dy² = 1, то не существуют целые значения x и y при D является квадратом.

Найдя минимальное решение x для D = {2, 3, 5, 6, 7}, мы получим следующие значения x:

3, 2, 9, 5, 8

Соответственно, наибольшее значение x получается когда D = 5.

Найдите значение D ≤ 1000, для которого получено наибольшее значение x в минимальном решении x.
