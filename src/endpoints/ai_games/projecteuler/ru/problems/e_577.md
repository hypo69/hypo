# Задача No 577
## Расположение точек на многоугольнике

## Условия
Рассмотрим правильный многоугольник с $n$ вершинами, пронумерованными от $1$ до $n$ по часовой стрелке.
Пусть $T(n)$ будет количеством треугольников, которые можно построить, выбрав три вершины этого многоугольника, так, что треугольник содержит центр многоугольника.
Например, $T(3) = 1$, $T(4) = 4$, и $T(6) = 16$.

Найдите $\sum_{3 \le n \le 12345} T(n)$.

## Пример
$T(3) = 1$, $T(4) = 4$, $T(6) = 16$
