# ИНСТРУКЦИЯ БOTA CHAT_GPT_NODEJS

## Обзор

Этот документ содержит инструкцию для бота, написанного на Node.js, использующего модель ChatGPT. Он описывает команды, которые может выполнять бот, а также ожидания относительно формата и содержания входных данных и выходных результатов.

## Команды

### `!hi`

**Описание**: Приветствует пользователя.

**Параметры**: Нет.

**Возвращает**:  Строку приветствия.

**Вызывает исключения**: Нет.


### `!train`

**Описание**: Обучает модель с предоставленными данными.

**Параметры**:
- `data` (строка или ссылка): Путь к файлу данных или ссылка на файл.
- `data_dir` (строка, опционально): Путь к каталогу с данными.
- `positive` (строка, опционально):  Название или метка позитивного набора данных.
- `attachment` (строка, опционально): Путь к файлу-приложению.


**Возвращает**: Сообщение об успехе или ошибке обучения.

**Вызывает исключения**:
- `ValueError`: В случае неверных входных данных (неправильный формат, отсутствие обязательных параметров).
- `IOError`: При проблемах с доступом к файлам или каталогам.
- `RuntimeError`: При возникновении необработанных ошибок в процессе обучения.


### `!test`

**Описание**: Тестирует модель с предоставленными тестовыми данными в формате JSON.

**Параметры**:
- `test_data` (строка или ссылка): Путь к файлу с тестовыми данными в формате JSON или ссылка на него.


**Возвращает**: Результаты тестирования в формате JSON.

**Вызывает исключения**:
- `ValueError`: Если входные данные не в формате JSON или содержат некорректные данные.
- `FileNotFoundError`: Если файл с тестовыми данными не найден.
- `JSONDecodeError`: Если файл не может быть декодирован как JSON.


### `!archive`

**Описание**: Архивирует файлы в указанном каталоге.

**Параметры**:
- `directory` (строка): Путь к каталогу, который нужно заархивировать.


**Возвращает**: Сообщение об успехе или ошибке архивирования.

**Вызывает исключения**:
- `ValueError`: Если путь к каталогу некорректен.
- `IOError`: Если не удаётся получить доступ к каталогу или создать архив.
- `RuntimeError`: Если произошла ошибка при выполнении архивации.


### `!select_dataset`

**Описание**: Выбирает набор данных для обучения из указанного каталога.

**Параметры**:
- `path_to_dir_positive` (строка): Путь к каталогу, содержащему положительные примеры данных.
- `positive` (строка):  Метка или название набора данных.


**Возвращает**: Сообщение об успехе или ошибке выбора набора данных.


**Вызывает исключения**:
- `ValueError`: Некорректные пути или метки.
- `IOError`:  Ошибка при работе с файлами или каталогами.
- `RuntimeError`:  Ошибка выполнения.


### `!instruction`

**Описание**: Выводит эту инструкцию.

**Параметры**: Нет.

**Возвращает**: Текст инструкции.

**Вызывает исключения**: Нет.
