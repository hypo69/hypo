# Модуль hypotez/src/_examples/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек проекта. Он использует `pathlib` для работы с путями, а также `json` для парсинга настроек.  Модуль предполагает наличие файлов `pyproject.toml`, `requirements.txt` и `.git` для определения корневой директории проекта.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с текущей директории файла и идя вверх по дереву директорий.

**Параметры**:

- `marker_files` (tuple): Кортеж из имен файлов или директорий, которые используются для определения корневой директории проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Объект `Path` представляющий корневую директорию проекта, или директорию текущего файла, если корневая не найдена.  Если корневой директории не найдено - возвращается текущая директория.

**Вызывает исключения**:

- Нет.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root()`.


### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `src/settings.json` в корневой директории проекта.

**Возвращает**:

- `dict`: Словарь настроек, либо `None`, если файл `src/settings.json` не найден или некорректно отформатирован.


### `doc_str`

**Описание**: Строка с содержимым файла `README.MD` в корневой директории проекта.

**Возвращает**:

- `str`: Строка с содержимым файла, либо `None`, если файл `README.MD` не найден или некорректно отформатирован.


### `__project_name__`

**Описание**: Название проекта, полученное из настроек проекта.

**Возвращает**:

- `str`: Название проекта, либо строка `'hypotez'` если ключ `project_name` отсутствует или некорректно отформатирован в файле настроек.


### `__version__`

**Описание**: Версия проекта, полученная из настроек проекта.

**Возвращает**:

- `str`: Версия проекта, либо пустая строка, если ключ `version` отсутствует или некорректно отформатирован в файле настроек.


### `__doc__`

**Описание**: Документация проекта, полученная из файла `README.MD` в корневой директории.

**Возвращает**:

- `str`: Документация проекта, либо пустая строка, если файл `README.MD` отсутствует или некорректно отформатирован.


### `__details__`

**Описание**: Подробная информация о проекте, по умолчанию пустая строка.

**Возвращает**:

- `str`: Подробная информация о проекте.


### `__author__`

**Описание**: Автор проекта, полученный из настроек проекта.

**Возвращает**:

- `str`: Автор проекта, либо пустая строка, если ключ `author` отсутствует или некорректно отформатирован в файле настроек.


### `__copyright__`

**Описание**: Авторские права, полученные из настроек проекта.

**Возвращает**:

- `str`: Авторские права, либо пустая строка, если ключ `copyrihgnt` отсутствует или некорректно отформатирован в файле настроек.

### `__cofee__`

**Описание**: Ссылка для поддержки автора через платёжную систему.

**Возвращает**:

- `str`: Ссылка для поддержки автора.
