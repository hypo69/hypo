# Модуль questions

## Обзор

Данный модуль содержит вопросы и ответы, предназначенные для облегчения понимания конфигурационных файлов.  В нем рассматривается проблема наименования файлов конфигурации и предлагается альтернативное решение.

## Вопросы

### Почему конфигурационные файлы называются по имени модуля (например: `suppliers.json`)?

**Описание**: Этот вопрос задает причину использования имён файлов, связанных с модулями, для конфигураций.

**Ответ**:  Имена файлов сделаны для удобства понимания моделями ИИ.

### Не лучше ли было их именовать `config.json`?

**Описание**: В этом вопросе предлагается альтернатива текущей системе именования файлов конфигурации.

**Ответ**: Имена файлов сделаны для удобства понимания моделями ИИ.
