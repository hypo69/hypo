# Модуль `hypotez/src/utils/convertors/dot.py`

## Обзор

Модуль `dot.py` предоставляет функцию `dot2png` для конвертации файлов в формате DOT в изображения PNG с помощью библиотеки Graphviz.  Функция обрабатывает входной DOT-файл, генерирует изображение PNG и сохраняет его в указанном месте.

## Функции

### `dot2png`

**Описание**: Функция `dot2png` конвертирует файл в формате DOT в изображение PNG. Она принимает путь к входному DOT-файлу и путь к выходному PNG-файлу в качестве аргументов.

**Параметры**:

- `dot_file` (str): Путь к входному файлу в формате DOT.
- `png_file` (str): Путь, куда будет сохранён выходной PNG-файл.

**Возвращает**:

- `None`: Функция ничего не возвращает, но выполняет сохранение выходного файла.

**Возможные исключения**:

- `FileNotFoundError`: Исключение возникает, если входной DOT-файл не найден.
- `Exception`: Исключение возникает при других ошибках во время конвертации (например, проблемы с Graphviz или некорректный формат DOT).

**Пример использования**:

```python
dot2png('example.dot', 'output.png')
```


## Использование из командной строки

Модуль `dot2png` можно использовать из командной строки. Для этого необходимо передать два аргумента: путь к входному файлу DOT и путь к выходному файлу PNG.

```bash
python dot2png.py input.dot output.png
```

В этом случае `input.dot` будет конвертирован в `output.png`.
