# hypo69: Эндпоинты для разработчика

## Обзор

Этот модуль содержит эндпоинты, предназначенные для облегчения работы разработчиков. Он включает в себя различные инструменты, такие как бот для чата с моделью ИИ (`small_talk_bot`), модуль для обучения модели коду проекта (`code_assistant`), и модуль для анализа диалогов (`psychologist_bot`).

## Описание модулей

### `small_talk_bot`

**Описание**: Бот с чатом модели ИИ. Позволяет вести диалог с обученной моделью искусственного интеллекта.

### `code_assistant`

**Описание**: Модуль обучения модели коду проекта. Помогает обучить модель ИИ на основе кодовой базы проекта, предоставляя разработчикам возможность взаимодействовать с моделью, понимающей структуру и семантику их кода.

### `psychologist_bot`

**Описание**: Ранняя разработка модуля парсинга диалогов. Предназначен для анализа и понимания диалогов, что может быть использовано в различных приложениях, например, для оценки настроения или выделения ключевых тем обсуждения.
