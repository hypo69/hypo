# Задача No 551
## Числа Элемака
## Условия
Для неотрицательного целого *n* пусть *s(n)* обозначает сумму цифр *n* в десятичной записи.

Последовательность *a(n)* определяется следующим образом:
*a(1) = 1* и 
*a(n) = a(n-1) + s(a(n-1))* для *n* > 1.

Первые члены последовательности таковы:
2, 4, 8, 16, 23, 28, 38, 49, 62, 70, 77, 91, 101, 103, 107, ...

Определим *b(n)* как наименьшее целое число *m* такое, что *a(m) = n*.
Например, *b(2) = 2*, *b(4) = 3*, *b(8) = 4*, *b(62) = 9*.
Если нет такого *m*, то *b(n) = 0*.

Определим *B(n)* как сумму всех *b(k)* для *1 ≤ k ≤ n*.
Например, *B(10) = 1+2+3+4+0+0+0+0+0+9=19*.

Найдите *B(2 000 000)*.
