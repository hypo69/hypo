**Общая структура:**
Этот JSON-файл представляет собой конфигурационный файл для проекта, вероятно, связанного с парсингом или каталогизацией товаров. Он содержит сценарии (в данном случае, только один) для обработки данных от поставщика.

**Поля:**

- `"scenarios"`: Объект, содержащий набор сценариев. Ключи объекта представляют собой названия сценариев.

- `"GForce"`: Название конкретного сценария. Предположительно, это может быть связано с товарами NVIDIA GeForce.
    - `"brand"`: Строка, представляющая бренд товара, в данном случае `"Nvidia"`.
    - `"url"`: Строка, содержащая URL-адрес, вероятно, ведущий на страницу с товарами Asus (возможно, определенной модели, так как это видеокарта) на сайте поставщика `"https://www.visualdg.co.il/169411-%D7%9B%D7%A8%D7%98%D7%99%D7%A1%D7%99-%D7%9E%D7%A1%D7%9A-ASUS/243216-Asus"`.
    - `"checkbox"`: Логическое значение, указывающее, выбран ли чекбокс, `false` в данном случае.
    - `"active"`: Логическое значение, определяющее, активен ли сценарий, `true` в данном случае.
     -  `"condition"`: Строка, определяющая состояние товара, `"new"` в данном случае.
    -  `"presta_categories"`: Строка, содержащая идентификаторы категорий PrestaShop,  `"108,524"` в данном случае.
