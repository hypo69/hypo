# Модуль `hypotez/src/suppliers/visualdg/__init__.py`

## Обзор

Данный модуль содержит начальную инициализацию для поставщика данных `visualdg`.  Он импортирует класс `Graber` из модуля `graber`.  Константа `MODE` устанавливает режим работы (в данном примере `dev`).


## Константы

### `MODE`

**Описание**:  Переменная, определяющая режим работы. В данном случае установлено значение `'dev'`.


## Модули

### `graber`

**Описание**: Модуль, содержащий класс `Graber`, который, предположительно, отвечает за сбор данных от поставщика `visualdg`.
