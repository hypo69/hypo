# Задача No 813
## Суммы степеней палиндромов
## Условия
Натуральное число является палиндромом, если его десятичное представление читается одинаково как слева направо, так и справа налево. Например, числа 5, 121, 3443 и 12321 являются палиндромами, а 123, 50 и 2425 не являются.

Определим функцию $P(n)$ как сумму $n$-х степеней всех палиндромов, которые меньше $10^n$.
Например,
$P(1) = 0^1 + 1^1 + 2^1 + 3^1 + 4^1 + 5^1 + 6^1 + 7^1 + 8^1 + 9^1 = 45$.
$P(2) = 0^2 + 1^2 + 2^2 + ... + 9^2 + 11^2 + 22^2 + 33^2 + ... + 99^2 = 4500 + 10*11^2*4500= 4500+30250+ 2025000 = 2059750$.
Вычислите $P(10^6)$ по модулю 1 000 000 007.
