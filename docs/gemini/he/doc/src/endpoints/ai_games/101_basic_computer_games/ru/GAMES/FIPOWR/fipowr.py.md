## פונקציות

### `fibonacci`

**תיאור**: מחשבת את המספר פיבונאצ'י ה-n.

**פרמטרים**:
- `n` (int): האינדקס של מספר פיבונאצ'י הנדרש.

**מחזירה**:
- `int`: המספר פיבונאצ'י ה-n.

```python
def fibonacci(n):
    if n <= 0:
        return 0
    elif n == 1:
        return 1
    else:
        a, b = 0, 1
        for _ in range(2, n + 1):
            a, b = b, a + b
        return b
```

## משתנים גלובליים

- `numberOfGuesses` (int): מונה את מספר הניסיונות שהמשתמש ביצע.
- `targetNumber` (int): המספר האקראי שנוצר בתחילת המשחק בין 1 ל-100.

```python
numberOfGuesses = 0
targetNumber = random.randint(1, 100)
```

## לולאה ראשית

לולאה זו מאפשרת למשתמש לנחש את המספר. היא ממשיכה עד שהמשתמש מנחש נכון.

1. **עדכון מספר הניסיונות**: `numberOfGuesses` גדל ב-1 בכל סיבוב.
2. **חישוב מספר פיבונאצ'י**: באמצעות הפונקציה `fibonacci`.
3. **חישוב מספר בחזקה**: `targetNumber` מועלה בחזקת המספר פיבונאצ'י.
4. **קבלת קלט מהמשתמש**: המשחק מבקש מהמשתמש לנחש את המספר.
    - **טיפול בשגיאות**: אם הקלט של המשתמש אינו מספר שלם, מוצגת הודעת שגיאה והלולאה ממשיכה.
5. **בדיקה אם הניחוש נכון**: אם הניחוש של המשתמש שווה לתוצאת החזקה, מוצגת הודעת ניצחון והמשחק מסתיים.
6. **המשך המשחק**: אם הניחוש שגוי, מוצגת הודעה והלולאה ממשיכה.

```python
while True:
    numberOfGuesses += 1
    fibonacciNumber = fibonacci(numberOfGuesses)
    poweredNumber = targetNumber ** fibonacciNumber

    try:
        userGuess = int(input(f"Попытка {numberOfGuesses}: Введите число: "))
    except ValueError as ex:
         print("Пожалуйста, введите целое число.")

    if userGuess == poweredNumber:
        print(f"ПОЗДРАВЛЯЮ! Вы угадали число за {numberOfGuesses} попыток!")
