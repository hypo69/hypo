# Модуль hypotez/src/logger/beeper.py

## Обзор

Этот модуль предоставляет классы и функции для воспроизведения звуковых сигналов (бипов) в зависимости от уровня события.  Модуль предназначен для использования в системах логгирования, обеспечивая звуковое оповещение о различных событиях (успех, информация, внимание, предупреждение, отладка, ошибка, критическая ошибка и т.д.).

## Перечисление BeepLevel

### `BeepLevel`

**Описание**: Перечисление, определяющее типы звуковых сигналов. Каждому типу соответствует своя мелодия.

**Уровни событий:**
- SUCCESS
- INFO
- ATTENTION
- WARNING
- DEBUG
- ERROR
- LONG_ERROR
- CRITICAL
- BELL

**Значения:**
Каждое значение BeepLevel представляет собой список кортежей, где каждый кортеж содержит имя ноты и длительность воспроизведения ноты.

## Класс BeepHandler

### `BeepHandler`

**Описание**: Обработчик звуковых сигналов.  Вызывает функцию воспроизведения звука в зависимости от уровня события в логе.

**Методы:**

- `emit(self, record)`:
    **Описание**: Обрабатывает запись лога и воспроизводит соответствующий звуковой сигнал.

    **Параметры**:
    - `record` (dict):  Запись лога, содержащая уровень события (например, 'level').

    **Возвращает**:
    -  (None):  Не возвращает значения.

    **Обрабатываемые исключения**:
    - `Exception`: Выводит сообщение об ошибке воспроизведения звука.


## Функция silent_mode

### `silent_mode`

**Описание**: Функция-декоратор, которая позволяет временно выключить воспроизведение звуковых сигналов.

**Параметры**:
- `func` (function): Функция, которую нужно декорировать.

**Возвращает**:
-  function: Обернутая функция, контролирующая режим "беззвучия".

**Внутренняя функция-обертка `wrapper`**:
- `args`: Позиционные аргументы, переданные в оригинальную функцию.
- `kwargs`: Именованные аргументы, переданные в оригинальную функцию.


## Класс Beeper

### `Beeper`

**Описание**: Класс, отвечающий за воспроизведение звуковых сигналов.

**Атрибуты**:
- `silent` (bool): Флаг, определяющий, включен ли режим "беззвучия". По умолчанию `False`.

**Методы**:

- `beep(self, level: BeepLevel | str = BeepLevel.INFO, frequency: int = 400, duration: int = 1000) -> None`:
    **Описание**: Воспроизводит звуковой сигнал, соответствующий указанному уровню события.

    **Параметры**:
    - `level` (BeepLevel | str, optional): Уровень события (например, `BeepLevel.INFO` или `'info'`). По умолчанию `BeepLevel.INFO`.
    - `frequency` (int, optional): Частота звука. По умолчанию 400.
    - `duration` (int, optional): Длительность звука в миллисекундах. По умолчанию 1000.

    **Возвращает**:
    - (None): Не возвращает значения.

    **Обрабатываемые исключения**:
    - `Exception`: Выводит сообщение об ошибке воспроизведения звука.


## Словарь `note_freq`

**Описание**: Словарь, содержащий частоты различных нот.

**Ключи**:
- Ноты (например, `'C3'`, `'C#3'`).

**Значения**:
- Соответствующие частоты нот в Герцах.
