# Модуль `arguments.py`

## Обзор

Модуль `arguments.py` содержит вспомогательные функции для обработки и проверки аргументов, используемых в API AliExpress. В частности, он предоставляет функции для преобразования списков в строки и извлечения идентификаторов продуктов.

## Функции

### `get_list_as_string`

**Описание**: Преобразует список или строку в строку, разделенную запятыми. Если входное значение `None`, возвращает `None`.

**Параметры**:
- `value` (str | list | None): Значение для преобразования. Может быть строкой, списком или `None`.

**Возвращает**:
- `str | None`: Строка, полученная путем объединения элементов списка через запятую, если входное значение - список. Исходная строка, если входное значение - строка. `None`, если входное значение `None`.

**Вызывает исключения**:
- `InvalidArgumentException`: Если входное значение не является строкой, списком или `None`.

### `get_product_ids`

**Описание**: Извлекает и возвращает список идентификаторов продуктов из строки или списка.

**Параметры**:
- `values` (str | list): Строка с идентификаторами продуктов, разделенными запятыми, или список идентификаторов продуктов.

**Возвращает**:
- `list`: Список идентификаторов продуктов, полученных после обработки каждого элемента.

**Вызывает исключения**:
- `InvalidArgumentException`: Если аргумент `values` не является строкой или списком.
