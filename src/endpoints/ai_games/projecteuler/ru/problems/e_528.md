# Задача No 528
##  Исключение повторений
## Условия
Определим функцию $f(n)$ следующим образом:
- Запишем $n$ в двоичном виде.
- Если двоичное представление содержит последовательные нули, вернем 0.
- Иначе, вернем целое число, полученное путем удаления всех повторений подряд идущих цифр.

Например:
- $f(3) = f(11_2) = 1$.
- $f(6) = f(110_2) = 0$.
- $f(7) = f(111_2) = 1$.
- $f(16) = f(10000_2) = 0$.
- $f(27) = f(11011_2) = 101_2 = 5$.

Определим $S(N) = \sum_{n=0}^{N} f(n)$.

Известно, что $S(10^3) = 2475$.

Найдите $S(10^{18})$.
