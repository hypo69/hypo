## Примеры подготовки кампаний

Этот модуль предоставляет примеры использования функций для подготовки кампаний на AliExpress.  Он демонстрирует, как обработать отдельные категории кампаний, конкретные кампании и все кампании, используя различные параметры.


## Функции

### `process_campaign_category`

**Описание**: Функция для обработки категории кампаний.

**Параметры**:
- `campaign_name` (str): Название кампании.
- `category_name` (str): Название категории.
- `language` (str): Язык.
- `currency` (str): Валюта.
- `force` (bool, optional): Принудительно переобработать данные. По умолчанию `False`.

**Возвращает**:
- `None`: Функция не возвращает значение.

**Вызывает исключения**:
- `ValueError`: Если переданные параметры не соответствуют ожидаемому типу или формату.
- `Exception`: В случае других ошибок.


### `process_campaign`

**Описание**: Функция для обработки конкретной кампании.

**Параметры**:
- `campaign_name` (str): Название кампании.
- `categories` (list[str], optional): Список категорий кампании.
- `language` (str, optional): Язык.
- `currency` (str, optional): Валюта.
- `force` (bool, optional): Принудительно переобработать данные. По умолчанию `False`.

**Возвращает**:
- `None`: Функция не возвращает значение.

**Вызывает исключения**:
- `ValueError`: Если переданные параметры не соответствуют ожидаемому типу или формату.
- `Exception`: В случае других ошибок.


### `process_all_campaigns`

**Описание**: Функция для обработки всех кампаний.

**Параметры**:
- `language` (str, optional): Язык.
- `currency` (str, optional): Валюта.
- `force` (bool, optional): Принудительно переобработать данные. По умолчанию `False`.

**Возвращает**:
- `None`: Функция не возвращает значение.

**Вызывает исключения**:
- `ValueError`: Если переданные параметры не соответствуют ожидаемому типу или формату.
- `Exception`: В случае других ошибок.


### `get_directory_names`

**Описание**: Функция для получения списка имён файлов в указанной директории.

**Параметры**:
- `campaigns_directory` (Path): Путь к директории.

**Возвращает**:
- list[str]: Список имён файлов в директории.

**Вызывает исключения**:
- `FileNotFoundError`: Если указанная директория не найдена.
- `Exception`: В случае других ошибок.


**Примечание**:

Данный модуль предполагает наличие функций `process_campaign_category`, `process_campaign`, `process_all_campaigns`, `get_directory_names` из модуля `..prepare_campaigns`.  Эти функции должны быть импортированы и определены в другом файле, чтобы данный пример мог работать.  Также предполагается, что переменная `gs.path.google_drive`  имеет правильное значение.
