# Модуль helper

## Обзор

Модуль `helper` предоставляет вспомогательные функции для работы с провайдерами, куки и асинхронными HTTP-запросами. Он импортирует функции из модуля `providers.helper`, модуль для получения куки `cookies` и модуль `aiohttp` для асинхронных запросов.

## Подробней

Этот модуль служит для импорта необходимых компонентов, которые затем используются в других частях проекта для обеспечения функциональности, связанной с провайдерами, управлением куки и выполнением асинхронных HTTP-запросов. Импорт `get_connector` позволяет использовать асинхронные соединения для повышения производительности при взаимодействии с сетью.

## Функции

### `get_cookies`

```python
def get_cookies() -> dict:
    """
    Возвращает словарь куки.

    Args:
        Нет аргументов.

    Returns:
        dict: Словарь куки.

    Raises:
        Нет исключений.

    Пример:
        >>> cookies = get_cookies()
        >>> print(cookies)
        {'cookie_name': 'cookie_value'}
    """
```

**Назначение**:

Функция `get_cookies` предназначена для получения куки.

**Параметры**:

- Функция не принимает параметров.

**Возвращает**:

- `dict`: Функция возвращает словарь куки.

**Вызывает исключения**:

- Функция не вызывает исключений.

**Как работает функция**:

1. Функция вызывает функцию `get_cookies` из модуля `..cookies`.
2. Функция возвращает словарь куки.

**Примеры**:

- Пример вызова функции:

```python
cookies = get_cookies()
print(cookies)
```
