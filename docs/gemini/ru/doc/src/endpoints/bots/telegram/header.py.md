# Модуль `header.py`

## Обзор

Модуль `header.py` определяет корневой путь к проекту и предоставляет общую информацию о проекте, такую как имя, версию, документацию и авторство. Он также загружает настройки из файла `settings.json`.

## Константы

### `MODE`

-   **Описание**: Режим работы приложения (по умолчанию `'dev'`).
-   **Тип**: `str`

## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная с каталога текущего файла, и останавливается на первом каталоге, содержащем любой из маркерных файлов.

**Параметры**:
- `marker_files` (tuple, optional): Кортеж с именами файлов или каталогов для идентификации корневого каталога проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневому каталогу, если он найден, в противном случае - каталог, в котором находится скрипт.

## Переменные

### `__root__`
**Описание**: Путь к корневому каталогу проекта.
**Тип**: `Path`

### `settings`

**Описание**: Словарь с настройками, загруженный из `settings.json`.
**Тип**: `dict | None`

### `doc_str`

**Описание**: Строка документации, загруженная из `README.MD`.
**Тип**: `str | None`

### `__project_name__`

**Описание**: Имя проекта. По умолчанию `'hypotez'`.
**Тип**: `str`

### `__version__`

**Описание**: Версия проекта. По умолчанию `''`.
**Тип**: `str`

### `__doc__`

**Описание**: Документация проекта, загруженная из файла `README.MD`.
**Тип**: `str`

### `__details__`

**Описание**: Детали проекта. В настоящее время пустая строка.
**Тип**: `str`

### `__author__`

**Описание**: Автор проекта. По умолчанию `''`.
**Тип**: `str`

### `__copyright__`

**Описание**: Авторские права проекта. По умолчанию `''`.
**Тип**: `str`

### `__cofee__`

**Описание**: Сообщение для поощрения разработчика чашкой кофе.
**Тип**: `str`
