# Модуль `header.py`

## Обзор

Модуль `header.py` предназначен для определения основных настроек проекта, таких как корневая директория, версия, имя проекта, а также загрузки настроек из файла `settings.json` и документации из `README.MD`.

## Константы

### `MODE`

Указывает режим работы приложения. В данном случае установлено значение `'dev'`.

## Функции

### `set_project_root`

**Описание**:

Находит корневую директорию проекта, начиная с директории текущего файла, и останавливается на первой директории, содержащей любой из указанных маркерных файлов.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для идентификации корневой директории проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории, если она найдена. В противном случае возвращается директория, где находится скрипт.

## Переменные

### `__root__`

**Описание**:
Содержит путь к корневой директории проекта, который определяется функцией `set_project_root`.

### `settings`

**Описание**:
Словарь, содержащий настройки проекта, загруженные из файла `settings.json`. Если файл не найден или возникает ошибка декодирования JSON, значение остается `None`.

### `doc_str`

**Описание**:
Строка, содержащая содержимое файла `README.MD`. Если файл не найден или возникает ошибка декодирования JSON, значение остается `None`.

### `__project_name__`

**Описание**:
Содержит имя проекта, полученное из настроек `settings.json` или значение по умолчанию `'hypotez'`, если настройки не загружены.

### `__version__`

**Описание**:
Содержит версию проекта, полученную из настроек `settings.json` или значение по умолчанию `''`, если настройки не загружены.

### `__doc__`

**Описание**:
Содержит строку документации, считанную из `README.MD`, или пустую строку, если файл не найден.

### `__details__`

**Описание**:
Содержит строку с деталями проекта. В данном случае всегда пустая строка `''`.

### `__author__`

**Описание**:
Содержит имя автора проекта, полученное из настроек `settings.json` или значение по умолчанию `''`, если настройки не загружены.

### `__copyright__`

**Описание**:
Содержит информацию об авторских правах проекта, полученную из настроек `settings.json` или значение по умолчанию `''`, если настройки не загружены.

### `__cofee__`

**Описание**:
Содержит строку с призывом угостить разработчика чашечкой кофе. Значение по умолчанию "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69", но может быть перезаписано значением из `settings.json` если таковое имеется.
