# Модуль `header.py`

## Обзор

Этот модуль содержит настройки и импорты, необходимые для работы с API PrestaShop. Он настраивает путь к корневой директории проекта, добавляет его в `sys.path`, импортирует необходимые библиотеки и модули, а также определяет константу `MODE`.

## Импорты

Модуль импортирует следующие библиотеки и модули:

- `sys`: Для работы с путями и системными переменными.
- `os`: Для работы с операционной системой, в частности, для определения текущей директории.
- `pathlib.Path`: Для работы с путями в объектно-ориентированном стиле.
- `json`: Для работы с JSON данными.
- `re`: Для работы с регулярными выражениями.
- `src.gs`: Пользовательский модуль.
- `src.suppliers.Supplier`: Класс для работы с поставщиками.
- `src.product.Product`, `src.product.ProductFields`, `src.product.ProductFieldsLocators`: Классы для работы с продуктами.
- `src.category.Category`: Класс для работы с категориями.
- `src.utils.jjson.j_dumps`, `src.utils.jjson.j_loads`, `src.utils.jjson.pprint`, `src.utils.jjson.save_text_file`: Функции для работы с JSON данными.
- `src.logger.logger`: Модуль для логирования.
- `src.logger.logger.StringNormalizer`, `src.logger.logger.ProductFieldsValidator`: Классы для валидации и нормализации данных.
