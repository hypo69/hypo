# Модуль hypotez/src/logger

## Обзор

Модуль `src.logger` определяет корневой путь к проекту `hypotez`. Все импорты строятся относительно этого пути.  В дальнейшем планируется перенести определение корневого пути в системную переменную.

## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная с текущего файла и перемещаясь вверх по директориям, пока не найдет директорию, содержащую один из указанных маркеров файлов (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или каталогов, которые используются в качестве маркеров для определения корневого каталога проекта. По умолчанию содержит `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневому каталогу проекта, если он найден. В противном случае возвращает путь к директории, где находится скрипт.

**Вызывает исключения**:

- Нет.


## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта.  Инициализируется вызовом функции `set_project_root()`.

**Тип**: `Path`


## Обработка файла настроек

**Описание**: Чтение файла `settings.json` в каталоге `src` для получения настроек проекта.

**Параметры**:

- Нет

**Возвращает**:

- `dict`: Словарь настроек проекта, если файл `settings.json` найден и корректен.  `None` в противном случае.

**Вызывает исключения**:

- `FileNotFoundError`: Если файл `settings.json` не найден.
- `json.JSONDecodeError`: Если содержимое файла `settings.json` не является корректным JSON.


## Обработка файла документации

**Описание**: Чтение файла `README.MD` в каталоге `src` для получения строки документации.

**Параметры**:

- Нет

**Возвращает**:

- `str`: Содержимое файла `README.MD`, если он найден. `None` в противном случае.

**Вызывает исключения**:

- `FileNotFoundError`: Если файл `README.MD` не найден.
- `json.JSONDecodeError`: В данном контексте исключение `json.JSONDecodeError` не имеет смысла, так как мы читаем строку, а не JSON.



## Переменные проекта (global variables)

**Описание**: Переменные, содержащие данные о проекте, полученные из файла `settings.json` и файла `README.md`.  Если файл `settings.json` или `README.md` не найден или содержит некорректные данные, соответствующие переменные получат значения по умолчанию.

**Переменные**:

- `__project_name__` (str): Название проекта. По умолчанию `hypotez`.
- `__version__` (str): Версия проекта. По умолчанию пустая строка.
- `__doc__` (str): Текст документации из `README.md`. По умолчанию пустая строка.
- `__details__` (str): Дополнительные детали. По умолчанию пустая строка.
- `__author__` (str): Автор проекта. По умолчанию пустая строка.
- `__copyright__` (str): Авторские права. По умолчанию пустая строка.
- `__cofee__` (str): Ссылка на возможность поддержать автора. По умолчанию URL для пожертвований.
