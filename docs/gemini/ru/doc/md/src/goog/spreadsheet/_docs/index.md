# Документация модуля hypotez/src/goog/spreadsheet/_docs

## Обзор

Данный модуль предоставляет инструменты для работы с Google Spreadsheets API v4. Он содержит класс-обертку `Spreadsheet`, упрощающий взаимодействие с API, позволяя создавать, изменять и заполнять данные в Google таблицах. Модуль включает функции для настройки различных параметров таблиц, таких как ширина столбцов, объединение ячеек, форматирование, цвет фона и другие.  Документация описывает процесс работы с Google Spreadsheets API v4 и предоставляет примеры использования класса `Spreadsheet`.

## Классы

### `Spreadsheet`

**Описание**: Класс `Spreadsheet` представляет собой обертку над Google Sheets API v4, предоставляя удобный интерфейс для работы с Google Таблицами. Он позволяет выполнять пакетные операции над таблицей, управлять шириной столбцов, заполнять ячейки данными, изменять форматирование и другие функции.

**Атрибуты**:

* `service`: Объект `apiclient.discovery.build`, предоставляющий доступ к Google Sheets API.
* `spreadsheetId`: Идентификатор Google таблицы.
* `sheetId`: Идентификатор листа в таблице.
* `sheetTitle`: Название листа в таблице.
* `requests`: Список запросов для пакетной обработки таблицы.
* `valueRanges`: Список запросов для заполнения ячеек данными.

**Методы**:

* `__init__(service, spreadsheet_id, sheet_id, sheet_title)`: Инициализирует экземпляр класса `Spreadsheet`.
* `prepare_setColumnWidth(col, width)`: Подготавливает запрос для изменения ширины столбца.
* `prepare_setColumnsWidth(startCol, endCol, width)`: Подготавливает запрос для изменения ширины нескольких столбцов.
* `prepare_setValues(cells_range, values, major_dimension="ROWS")`: Подготавливает запрос для заполнения ячеек значениями.
* `prepare_mergeCells(range)`: Подготавливает запрос для объединения ячеек.
* `prepare_setCellsFormat(range, format)`: Подготавливает запрос для форматирования ячеек.
* `prepare_setCellsFormats(range, formats)`: Подготавливает запрос для форматирования нескольких ячеек.
* `toGridRange(a1_range)`: Преобразует A1-нотацию в GridRange.
* `runPrepared(value_input_option="USER_ENTERED")`: Выполняет подготовленные запросы пакетно. Возвращает кортеж из результатов для `spreadsheets.batchUpdate` и `spreadsheets.values.batchUpdate`.



## Функции


### `prepare_setDimensionPixelSize`
**Описание**: Подготавливает запрос для обновления свойств измерения (столбцов или строк) в таблице.

**Параметры**:
* `dimension` (str):  Тип измерения (`COLUMNS` или `ROWS`).
* `startIndex` (int): Индекс начального элемента измерения.
* `endIndex` (int): Индекс конечного элемента измерения (не включительно).
* `pixelSize` (int): Размер в пикселях.

**Возвращает**:
- None.


### `prepare_setColumnsWidth`
**Описание**: Подготавливает запрос для изменения ширины нескольких столбцов.

**Параметры**:
* `startCol` (int): Индекс первого столбца.
* `endCol` (int): Индекс последнего столбца.
* `width` (int): Ширина столбцов в пикселях.

**Возвращает**:
- None.


### `prepare_setColumnWidth`
**Описание**: Подготавливает запрос для изменения ширины одного столбца.

**Параметры**:
* `col` (int): Индекс столбца.
* `width` (int): Ширина столбца в пикселях.

**Возвращает**:
- None.


### `prepare_setValues`
**Описание**: Подготавливает запрос для заполнения ячеек таблицы данными.

**Параметры**:
* `cellsRange` (str): Диапазон ячеек в формате A1-нотации.
* `values` (list): Список значений для заполнения.
* `majorDimension` (str, optional): Направление заполнения (по умолчанию "ROWS").

**Возвращает**:
- None.


###  `runPrepared`

**Описание**: Выполняет подготовленные запросы пакетно.

**Параметры**:
* `value_input_option` (str, optional): Тип ввода значений (по умолчанию "USER_ENTERED").

**Возвращает**:
- Кортеж из результатов `spreadsheets.batchUpdate` и `spreadsheets.values.batchUpdate`.
