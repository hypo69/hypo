# Модуль hypotez/src/suppliers/aliexpress/campaign/_examples/_example_edit_campaign.py

## Обзор

Этот модуль предоставляет пример реализации редактора рекламных кампаний на AliExpress.  Он основан на классе `AliPromoCampaign` и предоставляет дополнительные методы для редактирования и управления кампаниями.

## Классы

### `AliCampaignEditor`

**Описание**: Класс `AliCampaignEditor` расширяет функциональность класса `AliPromoCampaign` для редактирования рекламных кампаний.

**Наследование**: `AliPromoCampaign`

**Методы**:

- `__init__`:  Инициализирует экземпляр класса `AliCampaignEditor`.

**Параметры конструктора `__init__`**:
- `campaign_name` (str): Название рекламной кампании.
- `category_name` (str): Название категории.
- `language` (str, по умолчанию 'EN'): Язык кампании.
- `currency` (str, по умолчанию 'USD'): Валюта кампании.

## Импорты

Этот модуль использует следующие импорты:

- `re`: Для работы с регулярными выражениями.
- `shutil`: Для работы с файлами и каталогами.
- `pathlib`: Для работы с путями к файлам.
- `typing`: Для типов данных.
- `types`: Для работы с типами данных.
- `gs`:  Для работы с Google Sheets (необходимо уточнить функцию).
- `AliPromoCampaign`: Класс для работы с рекламными кампаниями.
- `AliAffiliatedProducts`: Для работы с аффилированными продуктами.
- `extract_prod_ids`: Для извлечения идентификаторов продуктов.
- `ensure_https`: Для обеспечения HTTPS-ссылок.
- `j_loads_ns`, `j_loads`, `j_dumps`:  Функции для работы с JSON (необходимо уточнить функционал).
- `list2string`, `csv2dict`: Функции конвертирования (необходимо уточнить функционал).
- `pprint`: Для красивой печати данных.
- `logger`: Для работы с логами.
- `read_text_file`, `get_filenames`: Функции для работы с файлами.
- `utils.interface`:  Интерфейсные функции для работы с файлами.
