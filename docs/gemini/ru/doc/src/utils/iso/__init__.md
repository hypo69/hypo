# Модуль hypotez/src/utils/iso/__init__.py

## Обзор

Этот модуль содержит константы, определяющие режим работы приложения.

## Константы

### `MODE`

**Описание**: Константа, определяющая режим работы приложения.

**Значение**: 'dev'

**Комментарии**:  Эта константа используется для настройки различных аспектов поведения приложения. В данном случае, значение 'dev' указывает на режим разработки.
