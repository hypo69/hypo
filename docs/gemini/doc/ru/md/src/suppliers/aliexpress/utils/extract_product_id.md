# Модуль `extract_product_id`

## Обзор

Модуль `extract_product_id` предназначен для извлечения идентификаторов товаров из URL-адресов или списка URL-адресов. Он поддерживает как списки URL-адресов, так и одиночные URL-адреса, а также непосредственное предоставление идентификаторов товаров.  Если входной параметр представляет собой список, а ни один URL не содержит действительного идентификатора, возвращается `None`.


## Функции

### `extract_prod_ids`

**Описание**: Функция извлекает идентификаторы товаров из списка URL-адресов или возвращает предоставленный идентификатор, если он был передан.

**Параметры**:
- `urls` (str | list[str]): URL, список URL-адресов или идентификатор товара.

**Возвращает**:
- str | list[str] | None: Список извлечённых идентификаторов товаров, одиночный идентификатор или `None`, если не найден ни один действительный идентификатор.

**Примеры**:
```python
>>> extract_prod_ids("https://www.aliexpress.com/item/123456.html")
'123456'

>>> extract_prod_ids(["https://www.aliexpress.com/item/123456.html", "7891011.html"])
['123456', '7891011']

>>> extract_prod_ids(["https://www.example.com/item/123456.html", "https://www.example.com/item/abcdef.html"])
['123456']

>>> extract_prod_ids("7891011")
'7891011'

>>> extract_prod_ids("https://www.example.com/item/abcdef.html")
None
```

**Вызывает исключения**:
- Никаких исключений не вызывает.


### `extract_id`

**Описание**: Функция извлекает идентификатор товара из URL-адреса или проверяет, является ли входной строкой валидным идентификатором.

**Параметры**:
- `url` (str): URL-адрес или идентификатор товара.

**Возвращает**:
- str | None: Извлечённый идентификатор товара или входной параметр, если он является валидным идентификатором, или `None`, если не найден ни один действительный идентификатор.


**Примеры**:
```python
>>> extract_id("https://www.aliexpress.com/item/123456.html")
'123456'

>>> extract_id("7891011")
'7891011'

>>> extract_id("https://www.example.com/item/abcdef.html")
None
```

**Вызывает исключения**:
- Никаких исключений не вызывает.
```
