# Модуль `emil_design`

## Обзор

Модуль `emil_design` предназначен для обработки изображений, извлечения описаний с помощью моделей искусственного интеллекта и дальнейшей публикации на Facebook и PrestaShop. Он использует данные из файлов, хранящихся в папке `emil` в Google Drive, а также API Facebook и PrestaShop для продвижения.

## Класс `EmilDesign`

### Описание

Класс `EmilDesign` предоставляет методы для работы с изображениями, их описанием и публикацией на Facebook и PrestaShop.

### Методы

#### `describe_images`

**Описание**: Метод описывает изображения, используя модель искусственного интеллекта.

**Параметры**:
- `from_url` (bool, опционально): Если `True`, то использует URL для описания изображений. По умолчанию `False`.

**Возвращает**:
- Не имеет возвращаемого значения.  Метод работает с файлами и сохраняет результаты в файле JSON.

**Обрабатываемые исключения**:
- Не описаны конкретные исключения.


#### `promote_to_facebook`

**Описание**: Метод публикует изображения и их описания на Facebook.

**Параметры**:
- Не имеет параметров.

**Возвращает**:
- Не имеет возвращаемого значения. Метод работает с файлами, загружает их на Facebook и сохраняет данные.


**Обрабатываемые исключения**:
- Не описаны конкретные исключения.


#### `upload_to_PrestaShop`

**Описание**: Метод загружает информацию о продуктах на PrestaShop.

**Параметры**:
- Не имеет параметров.

**Возвращает**:
- Не имеет возвращаемого значения. Метод создает экземпляры класса `Product` и `PrestaShop` для загрузки данных.


**Обрабатываемые исключения**:
- Не описаны конкретные исключения.



## Переменные

#### `MODE`

**Описание**: Переменная, определяющая режим работы модуля.

**Тип**: str

**Значение**: 'development' (по умолчанию)

#### `base_path`

**Описание**: Путь к папке с данными модуля в Google Drive.


**Тип**: `pathlib.Path`


**Значение**: Путь к папке `emil` в Google Drive.
