# Модуль questions

## Обзор

Данный модуль содержит вопросы и ответы для обработки и анализа. Файлы конфигурации (например, `suppliers.json`) организованы по имени модуля для лучшей семантической связи с соответствующими данными.

## Вопросы

### `Почему конфигурационные файлы называются по имени модуля (например: suppliers.json)?  Не лучше ли было их именовать config.json?`

**Вопрос**: Почему конфигурационные файлы называются по имени модуля (например: `suppliers.json`)?  Не лучше ли было их именовать `config.json`?

**Ответ**: Имена файлов сделаны для удобства понимания моделями ИИ.  Использование имени модуля (`suppliers.json`) позволяет непосредственно ассоциировать конфигурацию с данным модулем, что может улучшить интерпретацию информации моделями машинного обучения. Использование общего имени (`config.json`) может привести к потере контекста и сложности в понимании предназначения конфигурации для конкретного модуля.
