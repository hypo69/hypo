# Модуль hypotez/src/webdriver/playwright/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и чтения документации из файла `README.MD`.  Он также определяет переменные, хранящие информацию о проекте, такие как имя, версия, описание и другие детали.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла, ищет вверх по дереву директорий, пока не найдет директорию, содержащую один из указанных файлов-маркеров.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для определения корневой директории. По умолчанию заданы `pyproject.toml`, `requirements.txt` и `.git`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена. В противном случае, возвращает директорию, где находится текущий скрипт.

**Примечание**:  Функция добавляет корневую директорию в `sys.path`, что позволит импортировать модули из проекта.

## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root()`.

### `settings`

**Описание**: Словарь настроек, загруженный из файла `src/settings.json`. Возможна ситуация, когда `settings` будет `None`, если файл `settings.json` не найден или содержит невалидный JSON.

### `doc_str`

**Описание**: Строка документации, загруженная из файла `src/README.MD`. Возможна ситуация, когда `doc_str` будет `None`, если файл `README.MD` не найден.

### `__project_name__`

**Описание**: Имя проекта. По умолчанию - `hypotez`. Значение берется из `settings.json`, если доступно.

### `__version__`

**Описание**: Версия проекта. По умолчанию - пустая строка. Значение берется из `settings.json`, если доступно.

### `__doc__`

**Описание**: Документация проекта. По умолчанию - пустая строка. Значение берется из `README.MD`, если доступно.

### `__details__`

**Описание**: Подробное описание проекта. По умолчанию - пустая строка.

### `__author__`

**Описание**: Автор проекта. По умолчанию - пустая строка. Значение берется из `settings.json`, если доступно.

### `__copyright__`

**Описание**: Авторские права проекта. По умолчанию - пустая строка. Значение берется из `settings.json`, если доступно.

### `__cofee__`

**Описание**: Ссылка на страницу сбора средств для поддержки разработчика. По умолчанию - ссылка на Boosty. Значение берется из `settings.json`, если доступно.


## Обработка исключений

В коде используются `try...except` блоки для обработки возможных исключений:

- `FileNotFoundError`: Если файл `settings.json` или `README.MD` не найден.
- `json.JSONDecodeError`: Если файл `settings.json` содержит некорректный JSON.

В блоках `except` используется `...` для обозначения того, что обработчик ничего не делает, но это может быть изменено в зависимости от необходимости.
