Amazon Laptop Categories (HP)
============================

This file contains a JSON structure defining Amazon categories for HP laptops.  It includes details for different processor types (Intel Celeron, Intel i3, Intel i5, Intel i7, AMD Ryzen 3, AMD Ryzen 5, AMD Ryzen 7, AMD Athlon) and screen sizes (11, 13, 14, 15, 17 inches).  Each scenario corresponds to a specific combination of these parameters.
