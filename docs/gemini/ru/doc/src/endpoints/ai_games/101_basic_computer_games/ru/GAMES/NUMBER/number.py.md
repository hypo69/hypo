# Игра "ACE"

## Обзор

Этот модуль реализует игру "ACE", в которой игрок бросает кости и набирает очки, пока не выпадет 1 или не решит остановиться. Цель игры - набрать как можно больше очков, не проиграв из-за выпадения 1.

## Функции

В данном модуле функции не используются.

## Переменные

### `totalScore`

**Описание**:
Хранит общее количество очков игрока. Инициализируется значением 0.

**Тип**:
`int`

### `roll_again`

**Описание**:
Хранит ответ игрока на вопрос, хочет ли он бросить кость еще раз.

**Тип**:
`str`

### `diceRoll`

**Описание**:
Хранит результат броска кости (случайное число от 1 до 6).

**Тип**:
`int`
