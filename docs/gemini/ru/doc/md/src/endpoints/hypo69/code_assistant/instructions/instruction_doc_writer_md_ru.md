# Модуль example

## Обзор

Этот модуль предоставляет базовые функции для работы с данными.

## Функции

### `greet`

**Описание**: Функция приветствия.

**Параметры**:

- `name` (str): Имя пользователя.

**Возвращает**:

- `str`: Строка приветствия.


```python
def greet(name: str) -> str:
    """
    Args:
        name (str): Имя пользователя.

    Returns:
        str: Строка приветствия.
    """
    return f"Привет, {name}!"
```

### `calculate_sum`

**Описание**: Функция вычисления суммы двух чисел.

**Параметры**:

- `a` (int): Первое число.
- `b` (int): Второе число.

**Возвращает**:

- `int`: Сумма чисел `a` и `b`.


```python
def calculate_sum(a: int, b: int) -> int:
    """
    Args:
        a (int): Первое число.
        b (int): Второе число.

    Returns:
        int: Сумма чисел a и b.
    """
    return a + b
```
