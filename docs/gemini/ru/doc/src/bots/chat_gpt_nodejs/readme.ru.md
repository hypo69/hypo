# Боты чат-бота ChatGPT на Node.js

## Обзор

Данный модуль предоставляет реализацию бота чат-бота ChatGPT, разработанного с использованием Node.js.  Он включает в себя необходимую логику для взаимодействия с API ChatGPT и обработки запросов пользователей. Подробные инструкции и примеры использования доступны в данном документе.

## Функции

### `getChatResponse`

**Описание**: Данная функция выполняет запрос к API ChatGPT для получения ответа на заданный вопрос.

**Параметры**:
- `prompt` (str): Текст запроса к API ChatGPT.
- `apiKey` (str):  Ключ API для доступа к ChatGPT.


**Возвращает**:
- `Promise<object | null>`:  Объект с ответом ChatGPT или `null` в случае ошибки.

**Вызывает исключения**:
- `Error`:  Возникает при возникновении сетевых проблем или других ошибок API.


### `processUserMessage`

**Описание**: Обрабатывает входящее сообщение пользователя, генерирует запрос к API ChatGPT и возвращает сгенерированный ответ.

**Параметры**:
- `userMessage` (str): Текст сообщения пользователя.
- `apiKey` (str): Ключ API для доступа к ChatGPT.


**Возвращает**:
- `Promise<string | null>`: Сгенерированный ответ от ChatGPT или `null` в случае ошибки.

**Вызывает исключения**:
- `Error`: Возникает при возникновении сетевых проблем или других ошибок API.



## API Ключи

Для корректной работы бота необходимо указать ключ API ChatGPT.  Храните его в безопасном месте,  не вставляйте в публичный код. Используйте переменные окружения для хранения API ключей.


## Использование

Пример использования функций:

```javascript
// Пример использования функции getChatResponse
const getResponse = getChatResponse("Какой сегодня день?", "YOUR_API_KEY");
getResponse.then(response => {
  if (response) {
    console.log(response.choices[0].message.content); // Вывод ответа
  } else {
    console.error("Ошибка получения ответа");
  }
}).catch(error => {
  console.error("Ошибка: ", error);
});

// Пример использования функции processUserMessage
processUserMessage("Привет!", "YOUR_API_KEY")
  .then(response => {
    console.log(response);
  })
  .catch(ex => {
    console.error("Ошибка: ", ex);
  });
```

**Примечание**: Замените `"YOUR_API_KEY"` на ваш реальный ключ API.
