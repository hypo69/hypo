# Модуль для экспериментов с созданием PDF-отчета

## Обзор

Модуль `pricelist_generator.py` предназначен для экспериментов с созданием PDF-отчетов. Он использует класс `ReportGenerator` для генерации отчета на основе предоставленных данных в формате JSON и HTML-шаблона.

## Подробней

Этот модуль является частью экспериментов в проекте `hypotez` и расположен в директории `src/endpoints/kazarinov/_experiments`. Он предназначен для генерации PDF-отчета на основе данных, полученных из JSON-файла, и HTML-шаблона. В частности, он использует данные из файла `202410262326_he.json`, HTML-шаблон из `202410262326_he.html` и сохраняет результат в `202410262326_he.pdf`.

## Импортированные модули

- `pathlib.Path`: Используется для работы с путями к файлам.
- `header`:  Импортируется модуль `header`. **Необходимо предоставить подробную информацию о модуле `header` для его документирования**.

- `src.utils.jjson`: Модуль для работы с JSON-файлами (`j_loads`, `j_loads_ns`, `j_dumps`).

## Переменные

- `base_path (pathlib.Path)`: Путь к директории с данными.
- `data (dict)`: Данные, загруженные из JSON-файла.
- `html_file (pathlib.Path)`: Путь к HTML-файлу.
- `pdf_file (pathlib.Path)`: Путь к PDF-файлу, который будет сгенерирован.
- `r (src.endpoints.kazarinov.react.ReportGenerator)`: Объект класса `ReportGenerator`.

## Использование

1.  Определяется базовый путь к данным:

```python
base_path = gs.path.external_data / 'kazarinov' / 'mexironim' / '24_11_24_05_29_40_543'
```

2.  Загружаются данные из JSON-файла:

```python
data: dict = j_loads(base_path / '202410262326_he.json')
```

3.  Определяются пути к HTML и PDF файлам:

```python
html_file: Path = base_path / '202410262326_he.html'
pdf_file: Path = base_path / '202410262326_he.pdf'
```

4.  Создается экземпляр класса `ReportGenerator`:

```python
r = ReportGenerator()
```

5.  Генерируется PDF-отчет:

```python
r.create_report(data, html_file, pdf_file)
```
