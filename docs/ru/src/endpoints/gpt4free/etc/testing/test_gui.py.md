# Модуль запуска графического интерфейса g4f

## Обзор

Модуль предназначен для запуска графического интерфейса пользователя (GUI) для проекта g4f. Он добавляет путь к родительскому каталогу в `sys.path` и затем запускает GUI с помощью функции `run_gui`.

## Подробней

Этот модуль является точкой входа для запуска графического интерфейса приложения g4f. Он использует модуль `g4f.gui` для отображения интерфейса, позволяющего пользователю взаимодействовать с функциональностью g4f через графические элементы управления. Подключение `sys.path` необходимо для того, чтобы модуль `g4f.gui` был правильно импортирован, если он находится вне стандартных путей поиска модулей Python.
