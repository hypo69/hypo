# Задача No 102
## Треугольники, содержащие начало координат
## Условия
Три точки на плоскости заданы своими координатами: (x1, y1), (x2, y2), и (x3, y3).

Считается, что треугольник, образованный этими точками, содержит начало координат (0, 0), если начало координат находится внутри треугольника, на его стороне или совпадает с одной из вершин.

В файле `p102_triangles.txt` содержатся координаты тысячи треугольников.

Найдите, сколько из этих треугольников содержат начало координат.

**Примечание:** Приведенный ниже файл `p102_triangles.txt` состоит из пар координат x,y для трех точек, разделенных запятыми. Например:

-259,493,-745,-74,779,588
-175,-594,-421,-714,574,-645
-222,-256,-617,-570,369,586

(Этот файл уже включен в условия задач Project Euler)
