# Модуль тестирования базовых сценариев `test_basic_scenarios`

## Обзор

Этот модуль содержит тесты для основных сценариев работы с `tinytroupe`, включая запуск и остановку симуляций, определение атрибутов агентов и использование инструментов. Он также проверяет экспорт артефактов, созданных агентами.

## Подробнее

Модуль тестирует основные функции библиотеки `tinytroupe`, такие как создание агентов, определение их атрибутов, запуск и остановку симуляций, а также использование инструментов агентами для выполнения задач. Он также проверяет, что результаты работы агентов (например, созданные документы) экспортируются в файлы.
Тесты охватывают базовый сценарий с одним агентом и сценарий использования инструментов для создания документов.

## Функции

### `test_basic_scenario_1`

```python
def test_basic_scenario_1():
    """
    Тестирует базовый сценарий с одним агентом, включая запуск и остановку симуляции,
    определение атрибутов агента и создание чекпоинтов.
    """
```

**Как работает функция**:

1.  **Сброс состояния**: Начинается со сброса текущего состояния симуляции, чтобы убедиться, что нет активных симуляций.
2.  **Запуск симуляции**: Запускается новая симуляция с именем "default" и проверяется её статус.
3.  **Создание агента**: Создается агент `Oscar the Architect` с использованием фабрики агентов.
4.  **Определение атрибутов**: Определяются атрибуты агента, такие как возраст и национальность.
5.  **Проверка трассировки**: Проверяется наличие кэшированной и исполняемой трассировки.
6.  **Создание чекпоинта**: Создается чекпоинт симуляции.
7.  **Действие агента**: Агент выполняет действие, отвечая на вопрос.
8.  **Определение атрибута**: Агент определяет новый атрибут, такой как профессия.
9.  **Создание чекпоинта**: Создается еще один чекпоинт симуляции.
10. **Остановка симуляции**: Симуляция завершается.

**Примеры**:

```python
test_basic_scenario_1()
```

### `test_tool_usage_1`

```python
def test_tool_usage_1():
    """
    Тестирует сценарий использования инструментов агентом для создания документа.
    """
```

**Как работает функция**:

1.  **Настройка окружения**: Определяется папка для экспорта данных, создаются экземпляры `ArtifactExporter` и `TinyEnricher`, а также `TinyToolUse` с инструментом `TinyWordProcessor`.
2.  **Создание агента**: Создается агент `Lisa the Data Scientist` с использованием фабрики агентов.
3.  **Добавление факультета**: Агенту добавляется факультет `tooluse_faculty`, который позволяет использовать инструменты.
4.  **Действие агента**: Агент выполняет действие, получая задание на написание резюме.
5.  **Проверка действия**: Проверяется, что в списке действий агента есть действие типа `WRITE_DOCUMENT`.
6.  **Проверка файла**: Проверяется, что документ был создан и сохранен в файл в формате `.docx`, `.json` и `.md`.

**Примеры**:

```python
test_tool_usage_1()
```
