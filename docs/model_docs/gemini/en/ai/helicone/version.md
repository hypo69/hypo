Module helicone.version
=======================

.. module:: hypotez.src.ai.helicone.version
    :platform: Windows, Unix
    :synopsis: This module contains versioning information for the Helicone AI system.


Description
-----------

This module provides access to various versioning information, such as project name, version, author, copyright, and more.  It attempts to load settings from a `settings.json` file located in the `src` directory. If the file is not found or cannot be parsed, default values are used.
