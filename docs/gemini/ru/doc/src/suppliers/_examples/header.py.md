# Модуль `header.py`

## Обзор

Этот модуль содержит настройки и импорты, необходимые для работы с проектом `hypotez`. Он определяет корневую директорию проекта, добавляет ее в `sys.path`, что позволяет импортировать модули из любой точки проекта. Также модуль устанавливает режим работы проекта (по умолчанию `dev`).

## Переменные

### `MODE`

**Описание**: Указывает режим работы приложения. По умолчанию установлено значение `dev`.
    
### `dir_root`

**Описание**: Представляет собой объект `Path`, который указывает на корневую директорию проекта. Вычисляется на основе текущей директории.

## Импорты

Модуль импортирует следующие модули:

-   `os`: Для работы с операционной системой, например, для получения текущей директории.
-   `sys`: Для работы с системными настройками, например, для добавления директорий в `sys.path`.
-   `pathlib.Path`: Для работы с путями к файлам и директориям в объектно-ориентированном стиле.

## Константы
### `dir_root`

**Описание:** Корневая директория проекта, вычисляемая относительно текущей рабочей директории.
**Тип:** `pathlib.Path`
### `dir_src`

**Описание:** Путь к директории `src` проекта.
**Тип:** `pathlib.Path`
