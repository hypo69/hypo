# Yahtzee

## Обзор

Данный модуль реализует игру "Yahtzee" (Яцзы) - игра в кости, где игрок бросает пять кубиков и пытается набрать очки, составляя комбинации. Игра состоит из 13 раундов.

## Функции

### `roll_dice`

**Описание**: Бросает 5 кубиков и возвращает результаты.

**Возвращает**:
- `list`: Список из 5 случайных чисел от 1 до 6, представляющих результаты броска кубиков.

### `calculate_score`

**Описание**: Вычисляет очки для выбранной категории.

**Параметры**:
- `dice` (list): Список результатов броска кубиков.
- `category` (str): Выбранная категория для подсчета очков.

**Возвращает**:
- `int`: Количество очков для выбранной категории.

### `play_yahtzee`

**Описание**: Основная функция для игры в Yahtzee. Управляет игровым процессом, включая броски костей, выбор категорий и подсчет очков.

**Параметры**:
- Нет.

**Возвращает**:
- Нет.
