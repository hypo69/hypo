# Документация модуля `Google Drive Uploader Bot`

## Обзор

Этот модуль представляет собой Telegram-бота, написанного на Python, который позволяет загружать файлы по прямым и поддерживаемым ссылкам в Google Drive. Он был вдохновлен ботом [Google Drive Uploader](https://telegram.dog/driveuploadbot) от [CyberBoySumanjay](https://github.com/cyberboysumanjay).

## Подробней

Этот бот предоставляет возможность пользователям Telegram загружать файлы в Google Drive, используя различные типы ссылок, такие как прямые ссылки, ссылки Mega.nz, Dropbox и другие. Для работы с ботом требуется авторизация через команду `/auth`, после чего можно отправлять поддерживаемые ссылки боту для загрузки.

## Функциональность

Бот обладает следующими возможностями:

- Загрузка файлов в Google Drive по прямым ссылкам.
- Поддержка загрузки с Mega.nz.
- Поддержка загрузки с Dropbox.
- Возможность авторизации пользователя для доступа к Google Drive.
- Удаление сохраненных учетных данных пользователя.
- Поддержка Teamdrive (требует хардкодинга).

## Команды бота

- `/start` - Отображает стартовое сообщение.
- `/auth` - Запускает процесс авторизации пользователя.
- `/revoke` - Удаляет сохраненные учетные данные пользователя.
- `/help` - Отображает справочное сообщение.

## Настройка и использование

1.  Создайте учетные данные Google Drive API (тип "Другие") и скачайте JSON-файл.
2.  Поместите JSON-файл в корневой каталог бота и переименуйте его в "client_secrets.json".
3.  Замените токен бота в файле `creds.py`.
4.  Установите необходимые модули с помощью `sudo pip3 install -r requirements.txt`.
5.  Запустите бота с помощью `python3 bot.py`.

## Поддерживаемые ссылки

- Прямые ссылки
- Mega.nz
- Dropbox

## Требования

- Google Drive API credentials (тип "Другие") - Обязательно
- Telegram Bot Token (полученный через BotFather) - Обязательно
- Openload ftp login и Key - Опционально
- Mega Email и Password - Опционально
