# Модуль `src.webdriver.bs`

## Обзор

Модуль `src.webdriver.bs` предназначен для работы с BeautifulSoup. Он импортирует класс `BS` из модуля `bs.py`, предоставляя его для использования в других частях проекта.

## Подробней

Этот модуль служит точкой входа для функциональности BeautifulSoup, используемой в проекте `hypotez`. Он упрощает импорт и использование класса `BS`, который, вероятно, предоставляет методы для парсинга HTML или XML и извлечения данных.

## Функции

```python
from .bs import BS
```

- Нет возвращаемого значения.

**Вызывает исключения**:
- Нет известных исключений.
