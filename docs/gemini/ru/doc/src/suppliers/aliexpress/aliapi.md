# Модуль `hypotez/src/suppliers/aliexpress/aliapi.py`

## Обзор

Этот модуль предоставляет класс `AliApi`, расширяющий базовый класс `AliexpressApi`, для работы с API AliExpress. Он реализует методы для получения деталей о продуктах, аффилированных ссылок и других операций.  Модуль использует библиотеки `requests`, `json`, `asyncio`, и другие для взаимодействия с API и работы с данными. Модуль интегрирован с базами данных для управления категориями и рекламными кампаниями.

## Классы

### `AliApi`

**Описание**: Класс `AliApi` расширяет `AliexpressApi` и предоставляет дополнительные методы для взаимодействия с API AliExpress.  Включает методы для работы с данными о продуктах и аффилированными ссылками.

**Инициализация (`__init__`)**:

**Описание**: Инициализирует экземпляр класса `AliApi`.

**Параметры**:
- `language` (str): Язык для запросов к API. По умолчанию `'en'`.
- `currency` (str): Валюта для запросов к API. По умолчанию `'usd'`.

**Методы**:

#### `retrieve_product_details_as_dict`

**Описание**: Преобразует список объектов `SimpleNamespace` с информацией о продуктах в список словарей.

**Параметры**:
- `product_ids` (list): Список идентификаторов продуктов.

**Возвращает**:
- dict | None: Список словарей с информацией о продуктах. Возвращает None, если произошла ошибка.

**Пример использования (в коде):**
```python
prod_ids = [123, 456, 789]
prod_details = api_instance.retrieve_product_details_as_dict(prod_ids)
print(prod_details)
```

#### `get_affiliate_links`

**Описание**: Возвращает аффилированные ссылки для указанных ссылок продуктов.

**Параметры**:
- `links` (str | list): Ссылки на продукты.
- `link_type` (int, optional): Тип аффилированной ссылки. По умолчанию 0.


**Возвращает**:
- List[SimpleNamespace]: Список объектов `SimpleNamespace` с аффилированными ссылками.
