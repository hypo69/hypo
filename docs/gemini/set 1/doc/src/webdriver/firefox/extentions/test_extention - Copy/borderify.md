# Модуль borderify.js

## Обзор

Данный скрипт изменяет стиль элемента `document.body`, устанавливая ему границу толщиной 5 пикселей красного цвета. Также выводит диалоговое окно с помощью функции `alert()`.

## Функции

### `document.body.style.border = "5px solid red";`

**Описание**: Устанавливает границу элемента `document.body` с толщиной 5 пикселей и красным цветом.

**Параметры**: Нет.

**Возвращает**:  Не имеет возвращаемого значения.


### `alert()`

**Описание**: Отображает диалоговое окно с сообщением.

**Параметры**: Нет.

**Возвращает**:  Не имеет возвращаемого значения.
