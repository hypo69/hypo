# Модуль `hypotez/src/logger/header.py`

## Обзор

Модуль `hypotez/src/logger/header.py` определяет корневой путь к проекту `hypotez`. Все импорты строятся относительно этого пути.  В дальнейшем этот функционал планируется перенести в системную переменную.


## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная от текущего файла. Поиск выполняется вверх по директориям до тех пор, пока не будет найдена директория, содержащая один из указанных файлов-маркеров (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или каталогов, используемых для определения корневого каталога проекта. По умолчанию содержит ('pyproject.toml', 'requirements.txt', '.git').

**Возвращает**:

- `Path`: Путь к корневому каталогу проекта, если он найден. В противном случае возвращает путь к каталогу, в котором находится данный скрипт.


**Пример использования**:

```python
root_path = set_project_root()
print(root_path)
```

## Переменные

### `__root__`

**Описание**:  Путь к корневому каталогу проекта. Инициализируется функцией `set_project_root()`.

**Тип**: `Path`

### `settings`

**Описание**: Словарь настроек проекта, загруженный из файла `settings.json`.

**Тип**: `dict` или `None`

### `doc_str`

**Описание**: Строка, содержащая содержимое файла `README.MD`.

**Тип**: `str` или `None`


### `__project_name__`

**Описание**: Название проекта, полученное из файла настроек `settings.json`. По умолчанию 'hypotez'.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта, полученная из файла настроек `settings.json`. По умолчанию пустая строка.

**Тип**: `str`

### `__doc__`

**Описание**: Документация проекта, полученная из файла `README.MD`. По умолчанию пустая строка.

**Тип**: `str`


### `__details__`

**Описание**: Дополнительные детали проекта. По умолчанию пустая строка.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта, полученный из файла настроек `settings.json`. По умолчанию пустая строка.

**Тип**: `str`

### `__copyright__`

**Описание**: Авторские права проекта, полученные из файла настроек `settings.json`. По умолчанию пустая строка.

**Тип**: `str`

### `__cofee__`

**Описание**: Ссылка для поддержки разработчика. Получена из настроек файла `settings.json`. По умолчанию указана ссылка.

**Тип**: `str`

**Обработка исключений**:

- `FileNotFoundError`: Если файл `settings.json` или `README.MD` не найден.
- `json.JSONDecodeError`: Если содержимое файла `settings.json` не является валидным JSON.
