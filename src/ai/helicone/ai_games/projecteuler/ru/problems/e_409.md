# Задача No 409
## Сумма разбиений

## Условия

Пусть s(n,k) — количество способов представить целое число n как сумму положительных целых чисел, не превышающих k. 
Например, s(4,3) = 4, так как есть 4 способа представить 4 как сумму целых чисел, не превышающих 3:
* 1+1+1+1
* 1+1+2
* 1+3
* 2+2

Пусть S(n) = ∑ s(n,k) для 1 ≤ k ≤ n.
Например, S(4) = s(4,1) + s(4,2) + s(4,3) + s(4,4) = 1 + 2 + 4 + 5 = 12.

Также, пусть F(n) = ∑ S(i) для 1 ≤ i ≤ n.
Например, F(5) = S(1) + S(2) + S(3) + S(4) + S(5) = 1 + 3 + 7 + 12 + 23 = 46.

Найдите F(10^15) mod 1,000,000,007.
