# Настройки домена emildesign_com

## Обзор

Файл `settings.json` содержит настройки для домена `emildesign_com`, включая языковые настройки.

## Оглавление

1.  [Языковые настройки](#языковые-настройки)

## Языковые настройки

### `langs`

**Описание**:
Словарь, определяющий поддерживаемые языки и их числовые идентификаторы.

**Параметры**:
-  Отсутствуют явные параметры. Данные определены в JSON-структуре.

**Содержание**:

-   `en`: (int) Идентификатор английского языка, равный 1.
-   `he`: (int) Идентификатор иврита, равный 2.
-   `ru`: (int) Идентификатор русского языка, равный 3.

**Пример**
```json
{
  "langs": {
    "en": 1,
    "he": 2,
    "ru": 3
  }
}
```
