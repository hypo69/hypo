# Модуль `src.ai.gemini.header`

## Обзор

Модуль `header.py` предназначен для управления конфигурацией и определения основных переменных проекта. Он загружает конфигурацию из `config.json`, считывает документацию из `README.MD` и устанавливает корневую директорию проекта.

## Функции

### `set_project_root`

**Описание**:
    Находит корневую директорию проекта, начиная с директории текущего файла, и останавливается на первой директории, содержащей любой из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или директорий для идентификации корневой директории проекта. По умолчанию (`__root__`,`.git`).

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена, иначе - директория, где находится скрипт.

## Глобальные переменные

### `__root__`
**Описание**:
  Путь к корневой директории проекта. Инициализируется функцией `set_project_root`.

### `config`
**Описание**:
  Словарь с настройками проекта, загруженный из файла `config.json`. Может быть `None`, если файл не найден или имеет некорректный формат.

### `doc_str`
**Описание**:
  Строка, содержащая документацию проекта, загруженную из файла `README.MD`. Может быть `None`, если файл не найден.

### `__project_name__`
**Описание**:
  Имя проекта. По умолчанию - `hypotez`. Загружается из `config.json`.

### `__version__`
**Описание**:
  Версия проекта. Загружается из `config.json`.

### `__doc__`
**Описание**:
   Документация проекта, загруженная из файла `README.MD`.

### `__details__`
**Описание**:
   Детали проекта. На данный момент всегда пустая строка.

### `__author__`
**Описание**:
   Автор проекта. Загружается из `config.json`.

### `__copyright__`
**Описание**:
   Авторские права проекта. Загружаются из `config.json`.

### `__cofee__`
**Описание**:
   Сообщение о возможности поддержать разработчика чашечкой кофе. По умолчанию - `Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69`.
