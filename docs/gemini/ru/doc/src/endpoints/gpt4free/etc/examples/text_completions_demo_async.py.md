# Модуль асинхронного примера завершения текста с использованием g4f

## Обзор

Этот модуль демонстрирует, как использовать асинхронный клиент `g4f` для выполнения текстовых завершений с использованием модели `gpt-4o`. Он показывает основной пример использования `AsyncClient` для взаимодействия с API и получения ответа от модели.

## Подробнее

Модуль предоставляет пример асинхронного взаимодействия с моделью `gpt-4o` для генерации текста на основе заданных сообщений. В коде используется библиотека `g4f` для асинхронного выполнения запросов к модели.
Он иллюстрирует создание асинхронного клиента, отправку запроса на завершение текста и обработку полученного ответа.

        Exception: Если возникает ошибка при выполнении запроса к API.

```

**Назначение**:
Асинхронная функция `main` демонстрирует, как использовать `AsyncClient` для запроса завершения текста у модели `gpt-4o`. Она создает клиент, отправляет запрос с сообщениями и выводит полученный ответ.

**Как работает функция**:

1. **Создание асинхронного клиента**: Создается экземпляр `AsyncClient`.
2. **Отправка запроса на завершение текста**: Используется метод `client.chat.completions.create` для отправки запроса к модели `gpt-4o` с заданными сообщениями.
3. **Обработка ответа**: Полученный ответ извлекается и выводится в консоль.

```
Создание асинхронного клиента --> Отправка запроса к модели --> Обработка ответа и вывод в консоль
```

**Примеры**:

```python
import asyncio
from g4f.client import AsyncClient

async def main():
    client = AsyncClient()
    
    response = await client.chat.completions.create(
        model="gpt-4o",
        messages=[
            {"role": "system", "content": "You are a helpful assistant."},
            {"role": "user", "content": "how does a court case get to the Supreme Court?"}
        ]
    )
    
    print(response.choices[0].message.content)

asyncio.run(main())
