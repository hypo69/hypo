# Задача No 729
## Обобщённые двоичные представления

## Условия
Обобщенное двоичное представление натурального числа $n$ — это выражение $n$ в виде суммы степеней 2, где каждый показатель степени встречается не более двух раз.

Например:

$3 = 2^1 + 2^0$,
$7 = 2^2 + 2^1 + 2^0$,
$10 = 2^3 + 2^1$,
$13 = 2^3 + 2^2 + 2^0$,
$17 = 2^4 + 2^0$,
$23 = 2^4 + 2^2 + 2^1 + 2^0$.

Число $10$ имеет два обобщенных двоичных представления: $10 = 2^3 + 2^1$ и $10 = 2^3 + 2^0 + 2^0$ (показатель 0 встречается дважды).

Пусть $f(n)$ — число различных обобщенных двоичных представлений числа $n$.
Из примеров выше следует, что $f(3) = 1, f(7) = 1, f(10) = 2, f(13) = 1, f(17) = 1$ и $f(23) = 1$.

Пусть $S(n) = \sum_{k=1}^{n} f(k)$.
Известно, что $S(10) = 10$, $S(100) = 690$ и $S(1000) = 104020$.

Найдите $S(10^7)$.
