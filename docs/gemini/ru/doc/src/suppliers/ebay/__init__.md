# Модуль hypotez/src/suppliers/ebay/__init__.py

## Обзор

Этот модуль предоставляет начальную точку входа для работы с поставщиком eBay.  Он импортирует класс `Graber` из подмодуля `graber`.  Константа `MODE` задаёт режим работы, в данном случае `'dev'`.

## Содержание

### Модульные переменные

#### `MODE`

**Описание**: Переменная, определяющая режим работы. В данном случае, `'dev'`.

**Тип**: `str`

### Импорты

#### `Graber`

**Описание**: Импортирует класс `Graber` из модуля `graber`.
