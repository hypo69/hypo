# Генератор текстовых баннеров

## Обзор

Этот документ описывает работу системы генерации текстовых баннеров. Система предназначена для создания баннеров из введенного текста, используя символ `*` для оформления.

## Оглавление

1. [Обзор](#обзор)
2. [Правила](#правила)
3. [Пример оформления](#пример-оформления)

## Правила

1.  Используй символ `*` для оформления баннера.
2.  Верхняя и нижняя границы баннера должны быть оформлены символами `*`.
3.  Текст должен быть окружён рамкой из символов `*`.

## Пример оформления

```
*********
* Текст *
*********
```
