# `src.suppliers.hb.locators`

## Обзор

Модуль `src.suppliers.hb.locators` предназначен для управления и хранения локаторов элементов веб-страниц, используемых в контексте поставщика HB (Hotelbeds) в проекте `hypotez`. Он обеспечивает централизованное хранение и версионирование локаторов, а также облегчает их обновление и поддержку.

## Подорбней

Этот модуль играет важную роль в автоматизации тестирования и скрапинга данных с веб-сайта Hotelbeds. Локаторы используются для поиска и взаимодействия с элементами на веб-странице. Модуль позволяет абстрагироваться от конкретной структуры HTML и упрощает поддержку тестов при изменениях на сайте Hotelbeds.

## Импортированные модули

### `packaging.version`
Используется для управления версиями локаторов.

### `src.suppliers.hb.locators.version`
Импортирует переменные, связанные с версией, такие как `__version__`, `__doc__`, `__details__`.

### `src.suppliers.hb.locators.locator`
Импортирует все (*) сущности из модуля `locator`.

### `__version__`

**Описание**: Версия локаторов.

### `__doc__`

**Описание**: Документация модуля.

### `__details__`

**Описание**: Детальная информация о модуле.
