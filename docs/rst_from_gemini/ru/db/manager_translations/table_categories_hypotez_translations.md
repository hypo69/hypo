### Описание

Модуль `table_categories_hypotez_translations.py` предоставляет класс `CategoriesHypotezFullListManager`, предназначенный для работы с базой данных, содержащей информацию о категориях.  Класс использует SQLAlchemy для взаимодействия с MySQL базой данных.

### Класс `CategoriesHypotezFullListManager`

Этот класс отвечает за управление записями в таблице `categories_hypotez`.  Он предоставляет методы для добавления, выборки и обновления данных.


#### Конструктор `__init__(self, credentials, *args, **kwargs)`

Инициализирует объект класса.

* **`credentials`**: Словарь с данными для подключения к базе данных (логин, пароль, хост, порт, имя базы данных). Эти данные предполагается получить из переменной `gs.db_translations_credentials`. 
* **`*args, **kwargs`**: Дополнительные аргументы.

Метод строит строку подключения к базе данных с использованием данных из `credentials` и инициализирует SQLAlchemy движок `self.engine`, сессионный менеджер `self.Session`, базу данных `self.Base` и сессию `self.session`. Определяет модель таблицы и создает таблицу в базе данных.

#### Метод `define_model(self)`

Определяет модель таблицы `CategoryManager` как подкласс `Base` из SQLAlchemy.  Определяет поля таблицы, соответсвующие столбцам БД. Необходимо обращать внимание на типы данных.

#### Метод `create_table(self)`

Создает таблицу в базе данных, используя метаданные модели.

#### Метод `insert_record(self, ...)`

Добавляет новую запись в таблицу `categories_hypotez`.

*  Принимает на вход значения для всех полей таблицы.
*  Создает объект `CategoryManager` с переданными данными.
*  Добавляет объект в сессию и сохраняет изменения в базе данных.
*  Выводит сообщение о успешном добавлении записи.


#### Метод `select_record(self, **kwargs)`

Выбирает записи из таблицы `categories_hypotez` по заданным условиям.

*  Принимает на вход аргументы в виде ключевых слов (`**kwargs`). Каждый ключ должен соответствовать имени столбца в таблице, а значение - значению для фильтрации.
*  Создает запрос SQLAlchemy с помощью `self.session.query(self.CategoryManager)`.
*  Строит фильтр из переданных аргументов, учитывая случай отсутствия значения. Если значение `None` - условие не применяется.
*  Выполняет запрос и выводит информацию о каждой найденной записи.


#### Метод `update_record(self, id_category, id_parent, id_shop_default, date_upd, new_description)`

Обновляет запись в таблице `categories_hypotez` с заданными id и обновляет дату обновления.
