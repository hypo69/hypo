# Шаблон HTML для генерации прайс-листа

## Обзор

Этот шаблон HTML (`template.html`) предназначен для генерации HTML-страниц с прайс-листом товаров. Шаблон динамически заполняется информацией о товарах, включая изображения, названия, описания и цены.

## Структура

Шаблон состоит из основных секций, оформленных с использованием CSS для удобочитаемости и визуальной привлекательности.

## Разметка

### Заголовок (`<h1>`)

Отображает название прайс-листа.

### Краткое описание (`<p class="lead">`)

Представляет собой краткое описание прайс-листа.

### Карточки товаров (`<div class="product-card">`)

Каждая карточка товара содержит:

*   Изображение товара (`<img>`).
*   Название товара (`<h3>`).
*   Описание товара (`<p>`).
*   Технические характеристики товара (`<p>`).

### Цены (`<div class="footer">`)

Отображает общую стоимость всех товаров с указанием валюты.

## Стиль

CSS-стили обеспечивают:

*   Удобный визуальный дизайн.
*   Яркий цвет для значений цены.
*   Удобное отображение карточек товаров в одной строке.
*   Отступ между карточками товаров.
*   Оформление общей стоимости товара.
*   Отображение данных товаров в упорядоченном формате.

## Переменные

Шаблон использует следующие переменные:

*   `{{ title }}`: Заголовок прайс-листа.
*   `{{ description }}`: Краткое описание прайс-листа.
*   `{{ products }}`: Список объектов `product`, каждый из которых содержит:
    *   `product.image_local_saved_path`: Путь к изображению товара.
    *   `product.product_title`: Название товара.
    *   `product.product_description`: Описание товара.
    *   `product.specification`: Технические характеристики товара.
*   `{{ price }}`: Общая стоимость товаров.
*   `{{ currency }}`: Валюта.

## Обработка данных

Шаблон использует Django шаблоны для итерирования по списку продуктов (`{% for product in products %}...{% endfor %}`) и отображения информации о каждом продукте.

## Примеры использования

```html
<!-- Пример использования переменных -->
<h1>{{ title }}</h1>
<p class="lead">{{ description }}</p>
```

```html
<!-- Пример отображения списка продуктов -->
<div class="row">
    {% for product in products %}
        <div class="product-card">
            <img src="{{ product.image_local_saved_path }}" alt="{{ product.product_title }}" />
            <div class="product-info">
                <h3>{{ product.product_title }}</h3>
                <p>{{ product.product_description }}</p>
                <p>{{ product.specification }}</p>
            </div>
        </div>
    {% endfor %}
</div>
```

```html
<!-- Пример отображения итоговой стоимости -->
<div class="footer">
    <p>Общая стоимость: <span class="price-tag">{{ price }} {{ currency }}</span></p>
</div>
```

## Заключение

Данный шаблон предоставляет эффективный и удобный способ представления прайс-листа в HTML формате, обеспечивая аккуратный и наглядный вывод информации о товарах.
