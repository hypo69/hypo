# Модуль `hypotez/src/endpoints/prestashop/api/header.py`

## Обзор

Этот модуль содержит вспомогательные функции для работы с проектом, включая определение корневой директории проекта, загрузку настроек из файла `settings.json` и чтение файла `README.MD`.

## Функции

### `get_project_root`

**Описание**: Находит корневую директорию проекта, начиная от директории текущего файла.

**Параметры**:
- `marker_files` (tuple): Список файлов или директорий, которые используются для определения корневой директории проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории проекта, если найдена, иначе директория текущего файла.

**Вызывает исключения**:
- Нет.


### `get_project_settings`

**Описание**: Чтение настроек из файла `settings.json`.

**Параметры**:
- Нет.

**Возвращает**:
- `dict | None`: Словарь с настройками, если файл `settings.json` найден и успешно загружен; иначе `None`.

**Вызывает исключения**:
- `FileNotFoundError`: Если файл `settings.json` не найден.
- `json.JSONDecodeError`: Если файл `settings.json` поврежден или содержит некорректный JSON.



### `get_project_readme`

**Описание**: Чтение содержимого файла `README.MD`.

**Параметры**:
- Нет.

**Возвращает**:
- `str | None`: Содержимое файла `README.MD`, если файл найден и успешно прочитан; иначе `None`.

**Вызывает исключения**:
- `FileNotFoundError`: Если файл `README.MD` не найден.
- `json.JSONDecodeError`: Если файл `README.MD` поврежден или содержит некорректный JSON.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта.

**Тип**: `Path`

**Значение**: Результат вызова функции `get_project_root()`.


### `__project_name__`

**Описание**: Название проекта, полученное из файла `settings.json`.

**Тип**: `str`

**Значение**: Значение параметра `project_name` из `settings.json`, или `'hypotez'` по умолчанию.


### `__version__`

**Описание**: Версия проекта, полученная из файла `settings.json`.

**Тип**: `str`

**Значение**: Значение параметра `version` из `settings.json`, или пустая строка по умолчанию.


### `__doc__`

**Описание**: Содержимое файла `README.MD`.

**Тип**: `str`

**Значение**: Результат вызова функции `get_project_readme()`.


### `__details__`

**Описание**: Подробности о проекте.

**Тип**: `str`

**Значение**: Пустая строка.


### `__author__`

**Описание**: Автор проекта, полученный из файла `settings.json`.

**Тип**: `str`

**Значение**: Значение параметра `author` из `settings.json`, или пустая строка по умолчанию.


### `__copyright__`

**Описание**: Авторские права на проект, полученные из файла `settings.json`.

**Тип**: `str`

**Значение**: Значение параметра `copyright` из `settings.json`, или пустая строка по умолчанию.


### `__cofee__`

**Описание**: Ссылка на бонусную поддержку.

**Тип**: `str`

**Значение**: Значение параметра `cofee` из `settings.json`, или строка с ссылкой на поддержку по умолчанию.
