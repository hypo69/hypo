# Документация для `brands.json`

## Обзор

Файл `brands.json` содержит информацию о различных брендах, включая их активность, состояние и категории PrestaShop. Данные структурированы в виде JSON-объекта, где ключом является `"brand"`, а значением - объект с названиями брендов как ключами. Каждое название бренда связано с объектом, содержащим параметры `active`, `condition` и `presta_categories`.

## Структура файла

Файл имеет следующую структуру JSON:

```json
{
  "brand": {
    "BRAND_NAME": {
      "active": true,
      "condition": "new",
      "presta_categories": CATEGORY_ID
    },
    ...
  }
}
```
## Описание полей

### `brand`

- **Описание**: Корневой объект, содержащий все бренды.
- **Тип**: `object`
- **Содержание**: Объект, где ключи - это названия брендов.

### `BRAND_NAME`
- **Описание**: Название бренда, например, `"ACER"`, `"AMD"`, `"APPLE"` и т.д.
- **Тип**: `string`
- **Содержание**: Уникальный ключ для каждого бренда.

### `active`
- **Описание**: Показывает, активен ли бренд.
- **Тип**: `boolean`
- **Содержание**: `true`, если бренд активен, `false` в противном случае.

### `condition`
- **Описание**: Условие товаров бренда.
- **Тип**: `string`
- **Содержание**: `"new"`, что означает, что товары бренда новые.

### `presta_categories`
- **Описание**: ID категории PrestaShop, к которой относится бренд.
- **Тип**: `integer`
- **Содержание**: Числовой идентификатор категории.
