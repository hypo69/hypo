# Модуль `header.py`

## Обзор

Модуль `header.py` содержит общие настройки и константы для проекта, включая определение корневой директории проекта, загрузку настроек из файла `settings.json`, чтение документации из `README.MD`, а также определение основных атрибутов проекта, таких как название, версия, автор и прочее.

## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная с каталога текущего файла, перемещаясь вверх и останавливаясь на первом каталоге, содержащем любой из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple): Кортеж с именами файлов или каталогов, используемых для определения корневого каталога проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневому каталогу, если он найден. В противном случае возвращает путь к каталогу, в котором находится скрипт.

## Переменные

### `MODE`

**Описание**: Режим работы приложения. По умолчанию установлен в `dev`.

### `__root__`

**Описание**: Путь к корневому каталогу проекта. Определяется с помощью функции `set_project_root`.

### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json`. Если файл не найден или возникает ошибка при чтении JSON, значение переменной остается `None`.

### `doc_str`

**Описание**: Строка, содержащая содержимое файла `README.MD`. Если файл не найден или возникает ошибка при чтении, значение переменной остается `None`.

### `__project_name__`

**Описание**: Название проекта. Загружается из файла настроек. По умолчанию `"hypotez"`.

### `__version__`

**Описание**: Версия проекта. Загружается из файла настроек. По умолчанию пустая строка `''`.

### `__doc__`

**Описание**: Документация проекта, загруженная из `README.MD`.

### `__details__`

**Описание**: Детали проекта. По умолчанию пустая строка `''`.

### `__author__`

**Описание**: Автор проекта. Загружается из файла настроек. По умолчанию пустая строка `''`.

### `__copyright__`

**Описание**: Информация об авторских правах проекта. Загружается из файла настроек. По умолчанию пустая строка `''`.

### `__cofee__`

**Описание**: Сообщение о поддержке разработчика. Загружается из файла настроек. По умолчанию `"Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69"`.
