# Модуль config.ini

## Обзор

Этот файл содержит конфигурацию для взаимодействия с сервисами OpenAI или Azure OpenAI.  Он определяет типы API, параметры моделей, таймауты, лимиты и другие важные настройки.  Файл конфигурации использует синтаксис `.ini` и разделен на секции для различных параметров.

## Описание настроек

### `API_TYPE`

**Описание**: Тип API для взаимодействия с сервисом. Может принимать значения `openai` или `azure`.

**Значение по умолчанию**: `openai`


### `AZURE_API_VERSION`

**Описание**: Версия API для Azure OpenAI Service.

**Значение по умолчанию**: `2023-05-15` (Обратитесь к документации Azure OpenAI для актуальной версии).


## Параметры модели

### `MODEL`

**Описание**: Тип используемой модели OpenAI.

**Значение по умолчанию**: `gpt-4o`


### `MAX_TOKENS`

**Описание**: Максимальное количество токенов, которое может обработать модель.

**Значение по умолчанию**: `4000`


### `TEMPERATURE`

**Описание**: Параметр температуры, контролирующий случайность генерируемых текстов.

**Значение по умолчанию**: `0.3`


### `FREQ_PENALTY`

**Описание**: Наказание за частоту слов в генерируемом тексте.

**Значение по умолчанию**: `0.0`


### `PRESENCE_PENALTY`

**Описание**: Наказание за повторение слов в генерируемом тексте.

**Значение по умолчанию**: `0.0`


### `EMBEDDING_MODEL`

**Описание**: Модель для создания эмбеддингов текста.

**Значение по умолчанию**: `text-embedding-3-small`


## Параметры времени ожидания и попыток

### `TIMEOUT`

**Описание**: Максимальное время ожидания в секундах для ответа API.

**Значение по умолчанию**: `60`


### `MAX_ATTEMPTS`

**Описание**: Максимальное количество попыток запроса API.

**Значение по умолчанию**: `5`


### `WAITING_TIME`

**Описание**: Время ожидания между попытками запроса API в секундах.

**Значение по умолчанию**: `1`


### `EXPONENTIAL_BACKOFF_FACTOR`

**Описание**: Фактор экспоненциального увеличения времени ожидания между попытками.

**Значение по умолчанию**: `5`


## Параметры кэширования

### `CACHE_API_CALLS`

**Описание**: Включить/выключить кэширование вызовов API.

**Значение по умолчанию**: `False`


### `CACHE_FILE_NAME`

**Описание**: Имя файла для кэширования вызовов API.

**Значение по умолчанию**: `openai_api_cache.pickle`


## Параметры отображения содержимого

### `MAX_CONTENT_DISPLAY_LENGTH`

**Описание**: Максимальная длина выводимого содержимого в символах.

**Значение по умолчанию**: `1024`


## Настройки Simulation

### `RAI_HARMFUL_CONTENT_PREVENTION`

**Описание**: Включить защиту от генерации вредного контента.

**Значение по умолчанию**: `True`


### `RAI_COPYRIGHT_INFRINGEMENT_PREVENTION`

**Описание**: Включить защиту от нарушения авторских прав.

**Значение по умолчанию**: `True`


## Настройки Logging

### `LOGLEVEL`

**Описание**: Уровень подробности логов. Может принимать значения `ERROR`, `WARNING`, `INFO`, `DEBUG`.

**Значение по умолчанию**: `ERROR`
