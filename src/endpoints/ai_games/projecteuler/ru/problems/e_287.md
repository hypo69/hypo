# Задача No 287
## Квадратичное разделение

## Условия

Мы определяем `f(x,y)` как квадратное расстояние от точки `(x,y)` до начала координат. То есть, `f(x,y) = x² + y²`.

Далее мы определяем `g(x,y)` как 1, если `f(x,y)` ≤ 1000, и 0, если `f(x,y)` > 1000.

Рассмотрим точки на сетке с целыми координатами `(x,y)` в диапазоне `-N ≤ x, y ≤ N`.
Рассмотрим последовательность из 4N+1 точек вдоль линии `y = t`, `-N ≤ t ≤ N`, каждая точка имеет координаты `(x, t)` для `-N ≤ x ≤ N`.

Пусть `C` - число изменений значения `g(x,y)` когда x идет от -N к N.

Например, для N=100:
- `C` = 126 при `t = 0`
- `C` = 210 при `t = 50`
- `C` = 130 при `t = 100`

Мы также определяем `T(N)` как общее число изменений `g(x,y)`, когда `t` идет от `-N` до `N`. Таким образом, `T(N)` - это сумма значений `C` для каждого значения `t` от `-N` до `N`.

Оказывается, что `T(100) = 15150` и `T(1000) = 1501500`.

Найдите `T(2 000 000)`.
