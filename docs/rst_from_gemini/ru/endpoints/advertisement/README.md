## Определение рекламного объявления

Рекламное объявление описывается следующими полями:

* `id` (integer, уникальный идентификатор)
* `title` (string, заголовок объявления)
* `description` (string, описание объявления)
* `image_url` (string, URL изображения объявления)
* `target_audience` (string, целевая аудитория)
* `price` (float, цена за показ)
* `status` (string, статус объявления, например: "active", "draft", "pending")
* `created_at` (timestamp, дата создания объявления)
* `updated_at` (timestamp, дата последнего обновления)
* ... (добавьте другие поля)


## Получение списка объявлений

**Запрос:**

```
GET /advertisements
```

**Параметры запроса (опционально):**

* `status` (string): Фильтрация по статусу объявления.
* `target_audience` (string): Фильтрация по целевой аудитории.
* `page` (integer): Номер страницы результатов (для пагинации).
* `per_page` (integer): Количество объявлений на странице (для пагинации).

**Ответ (JSON):**

```json
{
  "advertisements": [
    {
      "id": 1,
      "title": "Объявление 1",
      "description": "Описание объявления 1",
      // ... другие поля
    },
    {
      "id": 2,
      "title": "Объявление 2",
      // ... другие поля
    }
  ],
  "total_count": 10, // общее количество объявлений
  "page": 1,
  "per_page": 10
}
```


## Получение конкретного объявления

**Запрос:**

```
GET /advertisements/{id}
```

**Ответ (JSON):**

```json
{
  "id": 1,
  "title": "Объявление 1",
  "description": "Описание объявления 1",
  // ... все поля объявления
}
```
