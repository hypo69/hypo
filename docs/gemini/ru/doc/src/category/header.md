# Модуль `hypotez/src/category/header.py`

## Обзор

Модуль `header.py` определяет корневой путь к проекту `hypotez`. Все импорты строятся относительно этого пути. В дальнейшем планируется перенести эту логику в системные переменные.

## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная с текущей директории и поднимаясь по иерархии каталогов, пока не найдет директорию, содержащую один из указанных маркеров (файлов или каталогов).

**Параметры**:

- `marker_files` (tuple): Кортеж с именами файлов или каталогов, используемых для определения корневой директории проекта. По умолчанию: (`'pyproject.toml'`, `'requirements.txt'`, `'.git'`).


**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена; в противном случае - директория, в которой находится текущий скрипт.

**Вызывает исключений**:
- Нет


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root`.


### `settings`

**Описание**: Словарь со настройками проекта.

**Тип**: `dict`

**Значение по умолчанию**: `None`

**Получение**:
Чтение файла `settings.json` из директории `src` в корневой директории проекта. Если файл не найден или некорректный, то переменная остается None.



### `doc_str`

**Описание**: Строка с текстом документации проекта (README.MD).

**Тип**: `str`

**Значение по умолчанию**: `None`

**Получение**:
Чтение файла `README.MD` из директории `src` в корневой директории проекта. Если файл не найден или некорректный, то переменная остается None.


### `__project_name__`

**Описание**: Название проекта, полученное из файла `settings.json`.

**Тип**: `str`

**Значение по умолчанию**: `hypotez`

**Получение**:
Извлекается из словаря `settings` по ключу `project_name`. Если ключ не найден или `settings` равно `None`, то используется значение по умолчанию.

### `__version__`

**Описание**: Версия проекта, полученная из файла `settings.json`.

**Тип**: `str`

**Значение по умолчанию**: `""`

**Получение**:
Извлекается из словаря `settings` по ключу `version`. Если ключ не найден или `settings` равно `None`, то используется значение по умолчанию.

### `__doc__`

**Описание**: Текст документации проекта.

**Тип**: `str`

**Значение по умолчанию**: `""`

**Получение**:
Получение из переменной `doc_str`, либо пустой строка.

### `__details__`

**Описание**: Подробная информация о проекте.

**Тип**: `str`

**Значение по умолчанию**: `""`

### `__author__`

**Описание**: Автор проекта, полученный из файла `settings.json`.

**Тип**: `str`

**Значение по умолчанию**: `""`

**Получение**:
Извлекается из словаря `settings` по ключу `author`. Если ключ не найден или `settings` равно `None`, то используется значение по умолчанию.


### `__copyright__`

**Описание**: Авторские права проекта, полученные из файла `settings.json`.

**Тип**: `str`

**Значение по умолчанию**: `""`

**Получение**:
Извлекается из словаря `settings` по ключу `copyright`. Если ключ не найден или `settings` равно `None`, то используется значение по умолчанию.
