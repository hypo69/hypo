# Задача No 880
## Разложения в виде суммы квадратов
## Условия
Любое натуральное число может быть представлено в виде суммы квадратов натуральных чисел. Рассмотрим для примера число 17:

17 = 1² + 4²
17 = 2² + 2² + 3²

Обозначим *s*(n) наименьшее количество квадратов натуральных чисел, сумма которых равна n.
Тогда *s*(17) = 2.
Обозначим *S*(n) сумму всех *s*(k) для k от 1 до n.

Известно, что *S*(10) = 19.

Найдите *S*(10^7).
