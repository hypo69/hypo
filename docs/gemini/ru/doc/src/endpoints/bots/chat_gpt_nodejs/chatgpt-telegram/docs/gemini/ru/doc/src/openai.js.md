# OpenAI Модуль

## Обзор

Этот модуль предоставляет класс `OpenAI` для взаимодействия с API OpenAI, включая функциональность чата и транскрипции аудио.

## Классы

### `OpenAI`

**Описание**: Класс для взаимодействия с API OpenAI.

**Конструктор**:
- `constructor(apiKey: string)`: Инициализирует класс с переданным ключом API.
    - **Параметры**:
        - `apiKey` (string): Ключ API для доступа к OpenAI.
   
**Методы**:
- `chat(messages: Array<object>) -> object | undefined`
   
    **Описание**: Отправляет запрос на чат в OpenAI.

    **Параметры**:
    - `messages` (Array<object>): Массив сообщений для чата, где каждое сообщение представляет собой объект с ключами `role` (роль отправителя, например, `'user'`, `'assistant'`, `'system'`) и `content` (текст сообщения).

    **Возвращает**:
    - `object | undefined`: Возвращает объект сообщения от ассистента или `undefined` в случае ошибки.

    **Вызывает исключения**:
    - Отсутствуют явные исключения, но может произойти ошибка при обращении к API OpenAI. Ошибки логируются в консоль.

- `transcription(filepath: string) -> string | undefined`

    **Описание**: Отправляет аудиофайл на транскрипцию в OpenAI.

    **Параметры**:
        - `filepath` (string): Путь к аудиофайлу для транскрипции.

    **Возвращает**:
    - `string | undefined`: Возвращает текст транскрипции или `undefined` в случае ошибки.

   **Вызывает исключения**:
    - Отсутствуют явные исключения, но может произойти ошибка при обращении к API OpenAI. Ошибки логируются в консоль.

## Константы

### `openai`

**Описание**: Экземпляр класса `OpenAI`, инициализированный с ключом API из конфигурации.
