# Модуль `hypotez/src/suppliers/hb/header.py`

## Обзор

Этот модуль содержит функции для работы с настройками проекта, определения корневой директории проекта и загрузки дополнительных данных. Он определяет переменные, хранящие имя проекта, версию, описание, авторов и другие детали, считывая их из файла `settings.json` и `README.MD`.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с текущего файла, ищет вверх по директориям, пока не найдет директорию, содержащую файлы-маркеры (pyproject.toml, requirements.txt, .git). Если корневая директория не найдена, возвращает директорию, где расположен текущий скрипт.

**Параметры**:

- `marker_files` (tuple): Список файлов или директорий, используемых для определения корневой директории проекта. По умолчанию включает ('pyproject.toml', 'requirements.txt', '.git').

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена, иначе путь к директории текущего скрипта.

**Вызывает исключения**:
- Нет.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта. Задаётся функцией `set_project_root()`.

**Тип**: `Path`


### `settings`

**Описание**: Словарь настроек проекта, считанный из файла `settings.json`.

**Тип**: `dict` или `None`

### `doc_str`

**Описание**: Строка с описанием проекта, считанная из файла `README.MD`.

**Тип**: `str` или `None`

### `__project_name__`

**Описание**: Имя проекта.  Получается из словаря настроек `settings`, по ключу `project_name`. Если ключ не найден, или словарь `settings` равен `None`, значение устанавливается в 'hypotez'.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта. Получается из словаря настроек `settings`, по ключу `version`. Если ключ не найден, или словарь `settings` равен `None`, значение устанавливается в ''.

**Тип**: `str`


### `__doc__`

**Описание**:  Описание проекта. Получается из переменной `doc_str`. Если `doc_str` равен `None`, значение устанавливается в ''.

**Тип**: `str`


### `__details__`

**Описание**:  Подробное описание проекта. По умолчанию пустая строка.

**Тип**: `str`

### `__author__`

**Описание**: Автор проекта. Получается из словаря настроек `settings`, по ключу `author`. Если ключ не найден, или словарь `settings` равен `None`, значение устанавливается в ''.

**Тип**: `str`

### `__copyright__`

**Описание**: Авторские права. Получается из словаря настроек `settings`, по ключу `copyright`. Если ключ не найден, или словарь `settings` равен `None`, значение устанавливается в ''.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка на поддержку разработчика через платёжную систему. Получается из словаря настроек `settings`, по ключу `cofee`. Если ключ не найден, или словарь `settings` равен `None`, значение устанавливается в строку с ссылкой.

**Тип**: `str`


## Обработка исключений

**Описание**: Модуль содержит блоки `try...except`, предназначенные для обработки `FileNotFoundError` и `json.JSONDecodeError` при чтении файлов настроек (`settings.json`) и описания (`README.MD`). Если файлы не найдены или некорректно отформатированы, соответствующие переменные (settings, doc_str) устанавливаются в `None` или пустые строки, не вызывая ошибку.
