=========================
Игра "Угадай число" (NUMBER)

.. contents:: Содержание
   :depth: 2

Введение
========
   Игра "Угадай число" (NUMBER) - это классическая игра, в которой компьютер загадывает случайное число, а игрок пытается его угадать. 
   После каждой попытки компьютер дает подсказку, больше или меньше загаданное число, чем введенное. Цель игры - угадать число за минимальное количество попыток.

Описание игры
=============

   Игра начинается с приветственного сообщения и объяснения правил. Затем программа предлагает игроку выбрать диапазон чисел, в котором будет загадано число.
   После ввода диапазона, компьютер генерирует случайное число и начинается основной игровой процесс.

Пошаговая инструкция
====================

1.  **Инициализация игры**
    
    - Вывод приветственного сообщения и объяснение правил.
    - Предложение выбрать диапазон чисел (минимум и максимум).
    - Генерация случайного числа в выбранном диапазоне.
    
    .. code-block::
    
        Добро пожаловать в NUMBER!
        Я загадаю случайное число, а вы должны его угадать.
        После каждого ответа я подскажу: больше или меньше ваше число.
        Выигрывает тот, кто угадает число за минимальное количество попыток. Удачи!

        Введите минимальное значение диапазона: 1
        Введите максимальное значение диапазона: 100
2.  **Основной процесс игры**
    
    - **Ввод числа игроком:** Игрок вводит предполагаемое число.
    - **Проверка числа:** Программа проверяет введенное число и дает подсказку:
        - Если число больше загаданного, сообщение: "Ваше число больше загаданного. Попробуйте ещё раз."
        - Если число меньше загаданного, сообщение: "Ваше число меньше загаданного. Попробуйте ещё раз."
        - Если число совпадает, игра завершается с поздравлением и количеством попыток.
    - **Подсчет попыток:** Подсчет количества попыток, которые понадобились игроку.
    
        .. code-block::
        
            Введите ваше число:
            > 50
            Ваше число больше загаданного. Попробуйте ещё раз.
            
            Введите ваше число:
            > 25
            Ваше число меньше загаданного. Попробуйте ещё раз.

    - Предложение сыграть ещё раз (да/нет).

Реализация на Python
====================
   
    .. code-block:: python
    
        """
        Модуль реализует игру "Угадай число".
        ====================================
    
        """
        import random
    
        from src.logger.logger import logger
    
    
        def play_number_game():
            """
            Запускает игру "Угадай число".
    
            Функция запрашивает у пользователя диапазон чисел,
            генерирует случайное число в этом диапазоне и позволяет
            пользователю угадывать его, предоставляя подсказки.
    
            """
            try:
                # Запрашивает у пользователя ввод минимального и максимального значений диапазона
                min_range = int(input('Введите минимальное значение диапазона: '))
                max_range = int(input('Введите максимальное значение диапазона: '))
                # Генерация случайного числа в заданном диапазоне
                secret_number = random.randint(min_range, max_range)
                # Инициализация счетчика попыток
                attempts = 0
                print('Я загадал число. Удачи!')
    
                # Основной цикл игры
                while True:
                    try:
                        # Запрашивает у пользователя ввод числа
                        guess = int(input('Введите ваше число: '))
                        attempts += 1
    
                        # Проверка введенного числа
                        if guess < secret_number:
                            print('Ваше число меньше загаданного. Попробуйте ещё раз.')
                        elif guess > secret_number:
                            print('Ваше число больше загаданного. Попробуйте ещё раз.')
                        else:
                            print(f'Поздравляем! Вы угадали число {secret_number} за {attempts} попыток!')
                            break
                    except ValueError as ex:
                        logger.error('Некорректный ввод. Введите целое число.', ex)
                        print('Пожалуйста, введите целое число.')
    
            except ValueError as ex:
                logger.error('Некорректный ввод диапазона. Введите целые числа.', ex)
                print('Пожалуйста, введите корректный диапазон чисел.')
                return
    
            # Предлагает сыграть еще раз
            while True:
                play_again = input('Хотите сыграть снова? (да/нет): ').lower()
                if play_again == 'да':
                    play_number_game()
                    return
                elif play_again == 'нет':
                    print('Спасибо за игру! До новых встреч!')
                    return
                else:
                   print('Пожалуйста, введите "да" или "нет".')
    
        if __name__ == '__main__':
            print("Добро пожаловать в NUMBER!")
            play_number_game()
