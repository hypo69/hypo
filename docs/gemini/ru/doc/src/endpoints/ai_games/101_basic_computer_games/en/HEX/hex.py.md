# HEX

## Обзор

Игра HEX - это логическая игра, в которой два игрока поочередно размещают свои фишки на шестиугольной сетке, стремясь соединить противоположные стороны доски. Цель каждого игрока — создать непрерывную цепь своих фишек, соединяющую их противоположные стороны доски. Игроки ставят свои фишки на шестиугольные клетки, пока один из них не сможет соединить свои стороны. В данной версии игра реализована для двух игроков: '1' и '2'

## Функции

### `print_board`

**Описание**: Выводит текущее состояние доски.

**Параметры**:
- `board` (list[list[str]]): Двумерный массив (список списков) представляющий доску.

**Возвращает**:
- `None`: Функция ничего не возвращает, а только выводит доску в консоль.

### `get_move`

**Описание**: Запрашивает у игрока координаты для хода.

**Параметры**:
- `board` (list[list[str]]): Двумерный массив (список списков) представляющий доску.
- `player` (str): Символ текущего игрока ('1' или '2').

**Возвращает**:
- `tuple[int, int]`: Кортеж из двух целых чисел, представляющих строку и столбец хода.

**Вызывает исключения**:
- `ValueError`: Возникает, если ввод не соответствует формату (например, не "A1", "B5" и т.д.) или координаты выходят за пределы доски.

### `check_win`

**Описание**: Проверяет, выиграл ли игрок.

**Параметры**:
- `board` (list[list[str]]): Двумерный массив (список списков) представляющий доску.
- `player` (str): Символ текущего игрока ('1' или '2').

**Возвращает**:
- `bool`: `True`, если игрок выиграл, `False` в противном случае.

### `play_hex`

**Описание**: Основная функция игры.

**Параметры**:
- `None`: Функция не принимает никаких параметров.

**Возвращает**:
- `None`: Функция ничего не возвращает, а запускает игру.
