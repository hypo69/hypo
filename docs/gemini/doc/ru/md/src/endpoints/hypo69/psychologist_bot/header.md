# Модуль `hypotez/src/endpoints/hypo69/psychologist_bot/header.py`

## Обзор

Этот модуль содержит вспомогательные функции для работы с проектом `hypotez`, такие как определение корневой директории проекта и загрузка настроек из файла `settings.json`. Он также предоставляет доступ к информации о проекте, например, его имени, версии и описанию.


## Функции

### `set_project_root`

**Описание**: Определяет корневую директорию проекта, начиная с текущей директории и поднимаясь вверх по иерархии директорий, пока не найдет директорию, содержащую один из указанных файлов (например, `pyproject.toml`, `requirements.txt`).

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, которые используются для определения корневой директории проекта. По умолчанию включает `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращает текущую директорию.

**Вызывает исключения**:

- Не вызывает никаких исключений.


## Переменные

### `MODE`

**Описание**: Переменная, содержащая значение режима работы (например, `dev`, `prod`).

### `__root__`

**Описание**: Путь к корневой директории проекта. Задаётся функцией `set_project_root`.

### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженные из файла `settings.json`.

**Тип**: `dict` | `None`


### `doc_str`

**Описание**: Текст, содержащий описание проекта, загруженный из файла `README.MD`.

**Тип**: `str` | `None`


### `__project_name__`

**Описание**: Название проекта. Получается из `settings`. По умолчанию `hypotez`.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта. Получается из `settings`. По умолчанию `''`.

**Тип**: `str`


### `__doc__`

**Описание**: Описание проекта. Получается из `doc_str`. По умолчанию `''`.

**Тип**: `str`


### `__details__`

**Описание**: Дополнительная информация о проекте.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта. Получается из `settings`. По умолчанию `''`.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права на проект. Получается из `settings`. По умолчанию `''`.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка на страницу для поддержки разработчика.

**Тип**: `str`
