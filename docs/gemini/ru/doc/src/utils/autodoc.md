# Модуль hypotez/src/utils/autodoc.py

## Обзор

Этот модуль содержит декоратор `autodoc`, который обновляет строку документации функции с добавлением времени последнего вызова функции. Декоратор используется для автоматического обновления docstring функции при ее вызове, добавляя информацию о времени последнего вызова в конец docstring.

## Функции

### `autodoc`

**Описание**: Декоратор, который обновляет docstring функции перед ее вызовом, добавляя в него строку с текущим временем.

**Параметры**:
- Нет входных параметров.

**Возвращает**:
- Оборачиваемую функцию.

**Вызывает исключения**:
- Нет.


### `update_docstring`

**Описание**: Функция, которая обновляет docstring функции, добавляя информацию о времени последнего вызова.

**Параметры**:
- `func` (функция): Функция, чья строка документации должна быть обновлена.

**Возвращает**:
- Нет.

**Вызывает исключения**:
- Нет.

### `example_function`

**Описание**: Пример функции, использующей декоратор `autodoc`. Каждый раз при ее вызове ее docstring обновляется, и в него добавляется информация о времени последнего вызова функции.

**Параметры**:
- `param1` (int): Первое значение.
- `param2` (str): Второе значение.

**Возвращает**:
- None

**Вызывает исключения**:
- Нет.


## Пример использования

```python
@autodoc
def example_function(param1: int, param2: str) -> None:
    """Пример функции.

    Args:
        param1 (int): Первое значение.
        param2 (str): Второе значение.
    """
    print(f"Processing {param1} and {param2}")

example_function(1, "test")
print(example_function.__doc__)  # Вывод обновленного docstring
example_function(2, "another test")
print(example_function.__doc__)  # Вывод обновленного docstring
```

Вывод примера:

```
Processing 1 and test
Last called at: 2024-01-01 12:00:00

Processing 2 and another test
Last called at: 2024-01-01 12:00:00
```

Этот вывод демонстрирует, как декоратор `autodoc` изменяет docstring функции `example_function` каждый раз при вызове, добавляя в него информацию о времени последнего вызова.
