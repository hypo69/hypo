# Модуль `src.suppliers.cdata.login`

## Обзор

Модуль `src.suppliers.cdata.login` предоставляет интерфейс для авторизации пользователя через веб-драйвер на сайте C-data.

## Оглавление

- [Обзор](#обзор)
- [Функции](#функции)
    - [`login`](#login)

## Функции

### `login`

**Описание**:
Функция выполняет вход пользователя на сайт C-data, используя данные, хранящиеся в `self`.

**Параметры**:
- `self` (object): Объект, содержащий информацию о веб-драйвере, локаторах и учетных данных пользователя.

**Возвращает**:
- `bool`: Возвращает `True`, если вход выполнен успешно, в противном случае, если вход не удался, может вызвать исключение.

**Вызывает исключения**:
- `Exception`: В случае ошибки при выполнении входа, например, если не найден элемент, не удалось нажать на кнопку.
