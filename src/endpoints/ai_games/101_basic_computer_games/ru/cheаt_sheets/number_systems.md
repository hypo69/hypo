### 2.1. –î–≤–æ–∏—á–Ω–∞—è (–±–∏–Ω–∞—Ä–Ω–∞—è) —Å–∏—Å—Ç–µ–º–∞ (–æ—Å–Ω–æ–≤–∞–Ω–∏–µ 2)

*   **–ò—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –≤ –∫–æ–º–ø—å—é—Ç–µ—Ä–∞—Ö:** –í—Å–µ –¥–∞–Ω–Ω—ã–µ –≤ –∫–æ–º–ø—å—é—Ç–µ—Ä–∞—Ö –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω—ã –≤ –¥–≤–æ–∏—á–Ω–æ–º –∫–æ–¥–µ (–±–∏—Ç–∞—Ö).

*   –ß–∏—Å–ª–æ `1011‚ÇÇ` (—á–∏—Ç–∞–µ—Ç—Å—è –∫–∞–∫ "–æ–¥–∏–Ω –Ω–æ–ª—å –æ–¥–∏–Ω –æ–¥–∏–Ω –ø–æ –æ—Å–Ω–æ–≤–∞–Ω–∏—é 2"). –ü–µ—Ä–µ–≤–æ–¥ –≤ –¥–µ—Å—è—Ç–∏—á–Ω—É—é —Å–∏—Å—Ç–µ–º—É:

    """
    –ü—Ä–µ–æ–±—Ä–∞–∑—É–µ—Ç –¥–≤–æ–∏—á–Ω–æ–µ —á–∏—Å–ª–æ (—Å—Ç—Ä–æ–∫—É) –≤ –¥–µ—Å—è—Ç–∏—á–Ω–æ–µ.

    return binary  # –í–æ–∑–≤—Ä–∞—â–∞–µ–º –¥–≤–æ–∏—á–Ω—É—é —Å—Ç—Ä–æ–∫—É
