# Модуль запуска Telegram-бота для поиска фильмов

## Обзор

Этот модуль является точкой входа для запуска Telegram-бота, предназначенного для поиска фильмов. Он инициализирует и запускает бота, используя библиотеку `aiogram` и другие необходимые компоненты, такие как middlewares для управления запросами.

## Подробней

Модуль выполняет следующие задачи:

1.  Загружает переменные окружения из файла `.env`, содержащие токен бота.
2.  Инициализирует `Dispatcher`, который обрабатывает входящие обновления от Telegram.
3.  Регистрирует middlewares для обработки запросов, такие как `ThrottlingMiddleware` для предотвращения спама.
4.  Подключает обработчики (handlers) из модуля `apps.hendlers.router` для обработки различных типов сообщений и команд.
5.  Запускает бота в режиме polling, что позволяет боту постоянно опрашивать сервер Telegram на наличие новых обновлений.
6.  Настраивает логирование для отслеживания работы бота и записи отладочной информации.
