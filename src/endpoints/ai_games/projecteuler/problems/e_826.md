# Задача No 826
## Иерархическая последовательность

## Условия
Определим последовательность $a_n$ следующим образом:

*   $a_1 = 1$
*   $a_{2n} = a_n$
*   $a_{2n+1} = a_n + a_{n+1}$

Первые 10 членов последовательности следующие:
1, 1, 2, 1, 3, 2, 3, 1, 4, 3.

Пусть $S(n) = \sum_{i=1}^n a_i$.

Дано, что $S(10^3) = 1683$ и $S(10^6) = 1666833$.

Найдите $S(10^{15})$.

<Пример>
Первые 10 членов последовательности: 1, 1, 2, 1, 3, 2, 3, 1, 4, 3.
