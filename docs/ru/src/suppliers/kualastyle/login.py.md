# Модуль `src.suppliers.kualastyle.login`

## Обзор

Модуль содержит функции для авторизации поставщика Kualastyle. Он предоставляет функции для входа в систему и закрытия всплывающих окон.

## Подробней

Этот модуль предназначен для автоматизации процесса входа в систему на веб-сайте Kualastyle. Он использует драйвер веб-браузера для взаимодействия с веб-страницей и выполнения необходимых действий для входа в систему. Модуль также включает функцию для закрытия всплывающих окон, которые могут мешать процессу входа в систему.
В проекте `hypotez` этот модуль используется для автоматизации процесса получения данных от поставщика Kualastyle.

2. Возвращает `True`, предполагая, что вход в систему выполнен успешно.

def close_pop_up(s) -> bool:
    """ Функция логин
   @param
        s - Supplier
    @returns
        True if login else False

   """
    ...
```

**Назначение**: Закрывает всплывающее окно на веб-странице Kualastyle.

**Параметры**:
- `s` (Supplier): Объект поставщика, содержащий необходимую информацию для закрытия всплывающего окна, включая драйвер веб-браузера и локаторы элементов.

**Возвращает**:
- `bool`: `True`, если всплывающее окно закрыто успешно, иначе `False`.

**Как работает функция**:

1.  **Извлечение драйвера и локатора**: Извлекает драйвер веб-браузера (`_d`) и локатор элемента для закрытия всплывающего окна (`_l`) из объекта поставщика `s`.
2.  **Навигация к URL**: Открывает URL `https://www.kualastyle.com` в браузере с помощью `_d.get_url('https://www.kualastyle.com')`.
3.  **Фокус на окне**: Переключает фокус на текущее окно браузера с помощью `_d.window_focus(_d)`.
4.  **Ожидание**: Ждет 5 секунд, чтобы убедиться, что всплывающее окно загрузилось, с помощью `_d.wait(5)`.
5.  **Выполнение локатора**: Пытается выполнить локатор, чтобы закрыть всплывающее окно, с помощью `_d.execute_locator(_l)`.
    *   Если локатор выполнен успешно, всплывающее окно закрывается.
    *   Если возникает исключение, регистрируется предупреждение с помощью `logger.warning(f"Не закрыл попап")`.
