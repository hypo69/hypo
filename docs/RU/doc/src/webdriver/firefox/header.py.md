# `hypotez/src/webdriver/firefox/header.py`

## Обзор

Данный модуль определяет базовые настройки и константы для проекта, включая путь к корневой директории, загрузку настроек из файла `settings.json` и общей информации из `README.MD`, а также устанавливает значения по умолчанию для основных переменных проекта.

## Функции

### `set_project_root`

**Описание**:
Находит корневую директорию проекта, начиная от директории текущего файла, и останавливается на первой директории, содержащей любой из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple, optional): Кортеж с именами файлов или директорий, идентифицирующих корень проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена, иначе - директория, где расположен скрипт.

## Переменные

### `MODE`

**Описание**:
Режим работы, по умолчанию `dev`.

### `__root__`

**Описание**:
Путь к корневой директории проекта.

### `settings`

**Описание**:
Словарь с настройками, загруженными из файла `settings.json`. Если файл не найден или содержит некорректный JSON, значение остается `None`.

### `doc_str`

**Описание**:
Строка, содержащая содержимое файла `README.MD`. Если файл не найден, значение остается `None`.

### `__project_name__`

**Описание**:
Название проекта, по умолчанию `hypotez`, берется из `settings.json` если таковой существует.

### `__version__`

**Описание**:
Версия проекта, по умолчанию `''`, берется из `settings.json` если таковой существует.

### `__doc__`

**Описание**:
Описание проекта, берется из `README.MD` если таковой существует.

### `__details__`

**Описание**:
Дополнительная информация о проекте, по умолчанию `''`.

### `__author__`

**Описание**:
Автор проекта, по умолчанию `''`, берется из `settings.json` если таковой существует.

### `__copyright__`

**Описание**:
Информация об авторских правах проекта, по умолчанию `''`, берется из `settings.json` если таковой существует.

### `__cofee__`

**Описание**:
Ссылка на чаевые для разработчика, по умолчанию "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69", берется из `settings.json` если таковой существует.
