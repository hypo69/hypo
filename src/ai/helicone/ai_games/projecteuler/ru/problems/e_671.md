# Задача No 671
## Сумма в квадрате

## Условия

Определим `s(n,k)` как сумму всех чисел, полученных путем вычитания `k` чисел из набора чисел от `1` до `n` и сложения абсолютных значений результатов.

Например, для `n=5` и `k=2`, мы рассматриваем все `C(5,2) = 10` способов выбора двух чисел `x` и `y` из набора {1, 2, 3, 4, 5}. Затем мы находим абсолютную разницу между ними и суммируем эти результаты.
1. `|1-2| = 1`
2. `|1-3| = 2`
3. `|1-4| = 3`
4. `|1-5| = 4`
5. `|2-3| = 1`
6. `|2-4| = 2`
7. `|2-5| = 3`
8. `|3-4| = 1`
9. `|3-5| = 2`
10. `|4-5| = 1`

Следовательно, `s(5,2) = 1+2+3+4+1+2+3+1+2+1 = 20`.

Теперь рассмотрим общую задачу: у нас есть набор чисел от 1 до n.  Выбираем k чисел из этого набора, вычитаем их в произвольном порядке, и берем абсолютное значение результата. Повторяем это для всех возможных сочетаний k чисел. Затем суммируем все эти абсолютные значения.
Мы определяем `S(n,k)` как сумму `s(n,j)` для всех `j` от 1 до `k`.

Например, `S(5,2) = s(5,1) + s(5,2) = 10 + 20 = 30`.

Далее, определим `F(n)` как сумму квадратов `S(n,k)` для всех `k` от 1 до `n`.

Например, `F(5) = S(5,1)^2 + S(5,2)^2 + S(5,3)^2 + S(5,4)^2 + S(5,5)^2 = 10^2 + 30^2 + 50^2 + 50^2 + 20^2 = 100 + 900 + 2500 + 2500 + 400 = 6400`.

Вы можете проверить, что `F(10) = 31633600`.

Найдите `F(10^8)`.  Ответ дайте по модулю 1000000007.
