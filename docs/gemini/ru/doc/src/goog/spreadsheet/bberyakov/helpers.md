# Модуль helpers

## Обзор

Модуль `helpers` содержит вспомогательные функции для работы с цветовыми кодами, предоставляя средства перевода между шестнадцатеричным (HEX) и десятичным (DECIMAL) форматами, а также из HEX в RGB.

## Функции

### `hex_color_to_decimal`

**Описание**: Преобразует шестнадцатеричный цвет (HEX) в десятичный (DECIMAL). Поддерживает ввод как односимвольных, так и двухсимвольных HEX кодов.

**Параметры**:

- `letters` (str): Шестнадцатеричное представление цвета (например, 'A', 'FF', '1A').

**Возвращает**:

- `int`: Десятичное значение цвета.

**Вызывает исключения**:

- Нет


### `decimal_color_to_hex`

**Описание**: Преобразует десятичное (DECIMAL) число в шестнадцатеричный (HEX) цвет.

**Параметры**:

- `number` (int): Десятичное значение цвета.

**Возвращает**:

- `str`: Шестнадцатеричное представление цвета (например, 'A', 'FF', '1A').

**Вызывает исключения**:

- Нет


### `hex_to_rgb`

**Описание**: Преобразует шестнадцатеричный (HEX) цвет в RGB.

**Параметры**:

- `hex` (str): Шестнадцатеричное представление цвета (например, '#FF0000', 'FF0000').

**Возвращает**:

- `tuple`: Кортеж из трёх целых чисел, представляющих значения R, G и B в RGB.

**Вызывает исключения**:

- Нет
