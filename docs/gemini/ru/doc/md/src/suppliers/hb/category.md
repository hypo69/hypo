# Модуль `hypotez/src/suppliers/hb/category.py`

## Обзор

Данный модуль отвечает за сбор информации о товарах со страниц категорий поставщика hb.co.il. Он реализует логику получения списков категорий и товаров, а также обработки страниц товаров. Модуль использует веб-драйвер для взаимодействия с сайтом и логику обработки страниц.

**Параметры**:

- `d` (Driver): Экземпляр класса `Driver`, представляющий веб-драйвер.
