# Справочник по файлам запросов для модели (prompts)

Этот файл содержит описание файлов, используемых для генерации запросов к модели.  Он предназначен для разработчиков и помогает в понимании структуры и логики работы с запросами.

**Структура каталога:**

Файлы запросов расположены в папке `src/ai/prompts`.  Каждый файл `.md` представляет собой отдельный запрос.

**Формат файла запроса:**

Каждый файл должен содержать следующую информацию:

* **Заголовок (Heading):**  Краткое и ясное описание запроса.  Используйте заголовки первого уровня (`#`).
* **Описание (Description):**  Подробное описание запроса.  Включайте информацию о данных, которые будут использованы в запросе, ожидаемом формате ответа, и любые другие важные детали.  Используйте заголовки второго уровня (`##`) для структурирования описания.
* **Пример запроса (Example Prompt):**  Конкретный пример запроса, который будет отправлен модели.
* **Пример ответа (Example Response):**  Пример ожидаемого ответа модели.  Укажите желаемый формат данных, например, JSON, список, или текст.
* **Комментарии/Примечания (Comments):** (Необязательно) Любые дополнительные комментарии или пояснения.

**Примеры файлов:**

Файл `example_request.md`:

```markdown
# Запрос на вывод списка продуктов

## Описание

Этот запрос предназначен для получения списка продуктов из базы данных.  В ответ ожидается массив JSON объектов, каждый из которых описывает отдельный продукт.  Запрос использует фильтрацию по категории.

## Пример запроса
```json
{
  "query": "Получить список продуктов из категории 'Овощи'",
  "category": "Овощи"
}
```

## Пример ответа
```json
[
  {"id": 1, "name": "Помидоры", "price": 2.5},
  {"id": 2, "name": "Огурцы", "price": 1.8}
]
```

## Примечания

Запрос может быть расширен для поддержки дополнительных параметров фильтрации.


```

```
