## <алгоритм>

1.  **Начало**: Процесс начинается с JSON-файла, который содержит данные о различных моделях оперативной памяти (RAM) от разных производителей.

2.  **Чтение JSON**: Данные из JSON-файла загружаются в память как объект Python (словарь). Ключом верхнего уровня является `"scenarios"`, который содержит в себе словарь с ключами-названиями моделей RAM (например, `"CORSAIR DIMM DDR 4 16GB"`).

3.  **Итерация по моделям RAM**: Происходит итерация по ключам словаря `"scenarios"`. На каждой итерации обрабатываются данные одной модели RAM.

4.  **Извлечение данных модели RAM**: Для каждой модели RAM извлекаются следующие данные:
    *   `brand` (строка): Производитель RAM (например, `"CORSAIR"`).
    *   `url` (строка): URL-адрес страницы продукта на сайте "morlevi.co.il".
    *   `checkbox` (логическое): Флаг, указывающий, нужно ли использовать чекбокс.
    *   `active` (логическое): Флаг, указывающий, активна ли модель.
    *    `condition` (строка): Состояние товара, в данном случае "new".
    *   `presta_categories` (словарь): Содержит информацию о категориях для PrestaShop.

5.  **Обработка `presta_categories`**: Внутри `presta_categories` есть словарь `template`, который содержит информацию о соответствии модели RAM категории. Ключи в `template` - это бренды в нижнем регистре, а значения - названия категории для каждого бренда.

6.  **Пример**: Для модели `"CORSAIR DIMM DDR 4 16GB"`:
    *   `brand` = `"CORSAIR"`
    *   `url` = `"https://www.morlevi.co.il/Cat/149?p_315=20&p_45=100&p_44=93&sort=datafloat2%2Cprice&keyword="`
    *   `presta_categories` = `{'template': {'corsair': 'DIMM DDR 4 16GB'}}`
    *    `condition` = `"new"`

7. **Конец цикла**: После обработки всех моделей RAM, цикл заканчивается.
8.  **Конец**: Процесс завершается.
