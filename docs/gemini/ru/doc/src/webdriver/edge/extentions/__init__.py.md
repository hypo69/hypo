# Модуль `src.webdriver.edge.extentions`

## Обзор

Модуль `src.webdriver.edge.extentions` предназначен для управления расширениями в веб-драйвере Edge. Он содержит информацию о версии и документацию.

## Переменные

### `__version__`

**Описание**: Версия текущего модуля.

### `__doc__`

**Описание**: Документация для текущего модуля.

### `__details__`

**Описание**: Детали текущего модуля.

## Импорты

### `packaging.version.Version`

**Описание**: Импортируется класс `Version` из модуля `packaging.version`, используется для работы с версиями пакетов.

###  `__version__`, `__doc__`, `__details__` from `.version`

**Описание**: Импортируются переменные версии, документации и деталей из модуля `.version`.
