# Модуль `hypotez/src/endpoints/advertisement/facebook/scenarios/switch_account.py`

## Обзор

Модуль содержит функцию `switch_account`, предназначенную для переключения между учетными записями в Facebook.  Функция взаимодействует с элементом "Переключить" на странице.

## Переключение между аккаунтами

### Функция `switch_account`

**Описание**:  Функция ищет и нажимает кнопку "Переключить аккаунт" на странице.

**Параметры**:

* `driver` (Driver): Объект драйвера веб-драйвера, предоставляющий методы для взаимодействия с веб-страницей.


**Возвращает**:

* Нет возвращаемого значения.


**Вызывает исключения**:

* Любые исключения, которые могут быть возбуждены методами класса `Driver`.


```python
def switch_account(driver: Driver):
    """ Если есть кнопка `Переключить` - нажимаю её  """
    driver.execute_locator(locator.switch_to_account_button)
```

**Примечание**:  В коде используется предварительно загруженный локетор `locator` для кнопки "Переключить аккаунт" из файла `post_message.json` в папке `locators`.  Этот локетор должен содержать необходимые данные для нахождения элемента на странице.
