# Модуль исключений для AliExpress API

## Обзор

Модуль содержит пользовательские исключения, используемые для обработки ошибок, возникающих при взаимодействии с AliExpress API. Он предоставляет иерархию исключений, начиная с базового класса `AliexpressException`, от которого наследуются все остальные исключения.

## Подорбней

Этот модуль определяет набор исключений, специфичных для AliExpress API, которые позволяют более точно обрабатывать различные типы ошибок, возникающие при работе с API. Использование пользовательских исключений облегчает отладку и обработку ошибок, делая код более надежным и понятным. Модуль предназначен для использования в других частях проекта, которые взаимодействуют с AliExpress API.

## Классы

### `AliexpressException`

**Описание**: Базовый класс для всех исключений AliExpress API.

**Как работает класс**:
Этот класс является родительским для всех остальных исключений, специфичных для AliExpress API. Он принимает строку `reason` в качестве аргумента, которая описывает причину возникновения исключения. Метод `__str__` переопределен для возврата этой причины в виде строки.

**Методы**:
- `__init__(self, reason: str)`: Инициализирует объект исключения с указанной причиной.
    - `reason` (str): Строка, описывающая причину исключения.
- `__str__(self) -> str`: Возвращает строковое представление исключения, содержащее причину.

**Параметры**:
- `reason` (str): Причина возникновения исключения.

**Примеры**
```python
try:
    raise AliexpressException('Произошла ошибка при обращении к API AliExpress')
except AliexpressException as ex:
    print(ex)
```

### `InvalidArgumentException`

**Описание**: Исключение, которое выбрасывается, когда аргументы неверны.

**Как работает класс**:
Этот класс является подклассом `AliexpressException` и используется для обозначения ошибок, связанных с некорректными аргументами, переданными в API.

**Примеры**
```python
try:
    raise InvalidArgumentException('Неверный формат аргумента')
except InvalidArgumentException as ex:
    print(ex)
```

### `ProductIdNotFoundException`

**Описание**: Исключение, которое выбрасывается, если ID продукта не найден.

**Как работает класс**:
Этот класс является подклассом `AliexpressException` и используется для обозначения ситуаций, когда запрошенный ID продукта не найден в базе данных AliExpress.

**Примеры**
```python
try:
    raise ProductIdNotFoundException('ID продукта не найден')
except ProductIdNotFoundException as ex:
    print(ex)
```

### `ApiRequestException`

**Описание**: Исключение, которое выбрасывается, если запрос к AliExpress API завершается неудачей.

**Как работает класс**:
Этот класс является подклассом `AliexpressException` и используется для обозначения ошибок, связанных с самим запросом к API (например, сетевые проблемы).

**Примеры**
```python
try:
    raise ApiRequestException('Ошибка при выполнении запроса к API')
except ApiRequestException as ex:
    print(ex)
```

### `ApiRequestResponseException`

**Как работает класс**:
Этот класс является подклассом `AliexpressException` и используется для обозначения ошибок, связанных с невалидным ответом, полученным от API (например, неожиданный формат).
