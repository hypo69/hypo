# Задача No 309
## Равнобедренные треугольники с целочисленными сторонами и высотами
## Условия
Известно, что равнобедренный треугольник имеет две стороны одинаковой длины. Также известно, что если стороны треугольника имеют целочисленные длины, то не обязательно его высоты также должны иметь целочисленные длины. Тем не менее, существуют равнобедренные треугольники с целочисленными сторонами, у которых, по крайней мере, одна высота также является целым числом.

Рассмотрим равнобедренный треугольник ABC с |AB| = |AC|. Пусть B(a,b) — наименьшее значение |BC| для всех равнобедренных треугольников ABC с |AB| = |AC| = a и высотой из A целого числа b.
Например, B(3, 12) = 5, а B(8, 3) = 10.

Вам дано, что sum(B(n,n)) для 1 ≤ n ≤ 1000 = 21579131.

Найдите sum(B(n,n)) для 1 ≤ n ≤ 10^7.
