# Модуль hypotez/src/webdriver/playwright/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и чтения документации из файла `README.MD`. Он также предоставляет доступ к переменным, содержащим информацию о проекте.

## Функции

### `set_project_root`

**Описание**:  Определяет корневую директорию проекта, начиная с директории текущего файла и идя вверх по дереву директорий.  Поиск прекращается при обнаружении директории, содержащей файлы, указанные в аргументе `marker_files`.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для определения корневой директории проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена. В противном случае возвращает директорию, в которой находится текущий файл.

**Вызывает исключения**:

- Не вызывает исключений.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root`.

**Тип**: `Path`

### `settings`

**Описание**: Словарь настроек, загруженный из файла `src/settings.json`.

**Тип**: `dict` или `None`

### `doc_str`

**Описание**: Строка документации, прочитанная из файла `README.MD`.

**Тип**: `str` или `None`


### `__project_name__`

**Описание**: Название проекта, полученное из файла настроек `settings.json`. По умолчанию `'hypotez'`.

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта, полученная из файла настроек `settings.json`. По умолчанию пустая строка.

**Тип**: `str`


### `__doc__`

**Описание**: Документация проекта, полученная из файла `README.MD`. По умолчанию пустая строка.

**Тип**: `str`

### `__details__`

**Описание**: Подробное описание проекта. По умолчанию пустая строка.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта, полученный из файла настроек `settings.json`. По умолчанию пустая строка.

**Тип**: `str`

### `__copyright__`

**Описание**: Авторские права проекта, полученные из файла настроек `settings.json`. По умолчанию пустая строка.

**Тип**: `str`

### `__cofee__`

**Описание**: Ссылка на страницу для поддержки разработчика (пожертвование). Получено из файла настроек `settings.json`. По умолчанию указан стандартный адрес.

**Тип**: `str`


## Обработка исключений

В коде используется конструкция `try...except`, чтобы обрабатывать возможные ошибки при работе с файлами:

- `FileNotFoundError`: Возникает, если файл `settings.json` или `README.MD` не найден.
- `json.JSONDecodeError`: Возникает, если содержимое файла `settings.json` не является валидным JSON.

В блоках `except` используются `...`, чтобы указать, что эти ошибки просто игнорируются, а переменные оставляются со значением `None`.
