# Модуль `hypotez/src/suppliers/aliexpress/utils`

## Обзор

Этот модуль содержит утилиты для работы с данными AliExpress. Он предоставляет функции для извлечения идентификаторов продуктов, перехода к HTTPS и работы с локалями.

## Функции

### `extract_prod_ids`

**Описание**: Функция извлекает идентификаторы продуктов из заданного входного значения.

**Параметры**:

- `input_data` (str): Входные данные, из которых нужно извлечь идентификаторы продуктов.


**Возвращает**:

- `list[str]`: Список извлеченных идентификаторов продуктов. Возвращает пустой список, если идентификаторы не найдены.


### `ensure_https`

**Описание**: Функция проверяет, что URL использует протокол HTTPS, и перенаправляет на него, если необходимо.

**Параметры**:

- `url` (str): URL-адрес, который нужно проверить и, при необходимости, перенаправить на HTTPS.


**Возвращает**:

- `str`: URL-адрес в формате HTTPS. Возвращает исходный URL, если он уже использует HTTPS.

### `locales`

**Описание**: Функция предоставляет информацию о поддерживаемых локалях.

**Возвращает**:
- `list[str]`: Список поддерживаемых локалей.


##  Список импортированных модулей

- `extract_prod_ids`
- `ensure_https`
- `locales`
