# Документация для `amazon_categories_desktops_dell_ref.json`

## Обзор

Этот файл содержит JSON-конфигурацию для сценариев парсинга Amazon, связанных с восстановленными (refurbished) настольными компьютерами Dell. Каждый сценарий определяет правила для поиска и категоризации товаров на основе определенных характеристик, таких как процессор (Intel i5, i7) и состояние товара (refurbished).

## Структура JSON

### `scenarios`

Словарь, содержащий различные сценарии парсинга. Каждый ключ в этом словаре представляет собой уникальный идентификатор сценария.

#### `REF DELL DESKTOP INTEL I5`
   
   **Описание**: Сценарий для поиска восстановленных настольных компьютеров Dell с процессором Intel i5.
    
   **Параметры**:
     - `brand` (str): Бренд товара - "DELL".
     - `url` (str): URL-адрес для поиска товаров на Amazon.
     - `active` (bool): Флаг, указывающий, активен ли сценарий (true - активен).
     - `condition` (str): Состояние товара - "ref" (восстановленный).
     - `presta_categories` (dict): Словарь с категориями PrestaShop.
       - `template` (dict):  Шаблон категорий PrestaShop. Ключ "dell" имеет значение "DESKTOPS INTEL I5".
     - `checkbox` (bool): Флаг, определяющий необходимость применения чекбокса (false - не применяется).
     - `price_rule` (int): Правило ценообразования.

#### `REF DELL DESKTOP INTEL I7`
   
   **Описание**: Сценарий для поиска восстановленных настольных компьютеров Dell с процессором Intel i7.

   **Параметры**:
     - `brand` (str): Бренд товара - "DELL".
     - `url` (str): URL-адрес для поиска товаров на Amazon.
     - `active` (bool): Флаг, указывающий, активен ли сценарий (true - активен).
     - `condition` (str): Состояние товара - "ref" (восстановленный).
     - `presta_categories` (dict): Словарь с категориями PrestaShop.
       - `template` (dict):  Шаблон категорий PrestaShop. Ключ "dell" имеет значение "DESKTOPS INTEL I5" (обратите внимание, что тут ошибка, и значение должно быть "DESKTOPS INTEL I7").
     - `checkbox` (bool): Флаг, определяющий необходимость применения чекбокса (false - не применяется).
     - `price_rule` (int): Правило ценообразования.
