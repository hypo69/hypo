# Модуль для нормализации ответа модели `ai_response_normalizer.py`

## Обзор

Модуль предоставляет инструменты для очистки и нормализации текстовых ответов, полученных от AI-моделей. Основная цель - удаление нежелательных префиксов и суффиксов, таких как Markdown или HTML-теги, чтобы получить чистый и полезный контент.

## Подробней

Этот модуль содержит функцию `normalize_answer`, которая принимает строку и пытается удалить известные префиксы и суффиксы, чтобы привести ответ AI к более чистому виду. Это особенно полезно, когда AI генерирует контент с использованием Markdown или HTML, и требуется извлечь только текст.

## Функции

### `normalize_answer`

```python
def normalize_answer(text:str) -> str:
    """"""
    for prefix in prefixes:
        if text.startswith(prefix):
            return text.removeprefix(prefix).removesuffix('```')

    return text
```

**Описание**: Нормализует текстовый ответ, удаляя известные префиксы и суффиксы.

**Как работает функция**:
Функция `normalize_answer` принимает строку `text` в качестве аргумента. Она итерируется по списку известных префиксов `prefixes`. Если строка `text` начинается с одного из этих префиксов, функция удаляет этот префикс, а также удаляет суффикс `````, если он присутствует. Если ни один из префиксов не совпадает, функция возвращает исходную строку без изменений.

**Параметры**:
- `text` (str): Строка, которую необходимо нормализовать.

**Возвращает**:
- `str`: Нормализованная строка, из которой удалены префиксы и суффиксы, если они были найдены.

**Примеры**:

```python
from src.utils.string.ai_response_normalizer import normalize_answer

# Пример 1: Удаление префикса '```html\n' и суффикса '```'
text1 = "```html\nПример строки <b>с HTML</b> ```"
normalized_text1 = normalize_answer(text1)
print(normalized_text1)  # Пример строки <b>с HTML</b>

# Пример 2: Удаление префикса '```md\n' и суффикса '```'
text2 = "```md\nПример строки с Markdown```"
normalized_text2 = normalize_answer(text2)
print(normalized_text2)  # Пример строки с Markdown

# Пример 3: Строка без префиксов и суффиксов
text3 = "Простая строка"
normalized_text3 = normalize_answer(text3)
print(normalized_text3)  # Простая строка

# Пример 4: Удаление префикса '```' и суффикса '```'
text4 = "```Пример строки```"
normalized_text4 = normalize_answer(text4)
print(normalized_text4) # Пример строки
```
