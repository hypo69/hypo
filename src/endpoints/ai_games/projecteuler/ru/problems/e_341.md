# Задача No 341
## Сумма последовательности Вието-Фибоначчи

## Условия

Рассмотрим последовательность  *a(n)*, определяемую как:
*   *a(1) = 1*
*   *a(2) = 2*
*   *a(n) = a(n-1) + a(n-a(n-1)) для n > 2*.

Первые десять членов этой последовательности будут 1, 2, 3, 4, 6, 8, 11, 15, 20, 26.

Пусть *S(n)* - сумма первых *n* членов последовательности. Например, *S(10) = 96*.

Найдите *S(10<sup>16</sup>) mod 10<sup>8</sup>*.
