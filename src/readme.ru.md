Глоссарий
================

### 1. **webdriver**
   - **`Driver`**: Объект, который управляет браузером (например, Chrome, Firefox) и выполняет действия, такие как навигация по веб-страницам, заполнение форм и т.д.
   - **`Executor`**: Интерфейс или класс, который выполняет команды или скрипты в контексте веб-драйвера.
   - **`Chrome`, `Firefox`, ...**: Конкретные браузеры, которые могут быть управляемы с помощью веб-драйвера.
   - **`locator`**: Механизм для поиска элементов на веб-странице (например, по ID, CSS-селектору, XPath).

### 2. **`Supplier`**
   - **list of suppliers (`Amazon`, `Aliexpress`, `Morlevi`, ...)**: Список компаний или платформ, которые предоставляют товары или услуги.
   - **`Graber`**: Инструмент или модуль, который автоматически собирает данные с веб-сайтов поставщиков (например, цены, наличие товаров).

### 3. **`Product`**
   - **`Product`**: Объект, представляющий товар или услугу, который может быть доступен на различных платформах.
   - **`ProductFields`**: Поля или атрибуты, которые описывают характеристики продукта (например, название, цена, описание, изображения).

### 4. ** `ai`
    - **`Model Prompt`**: Указание модели как обрабатывать входящуюю информацию и возвращать ответ. Загуэается при инициализации модели
    - **`Command Instruction`**: Небольшая команда - инструкция отправляемая с каждым запросом.
