# Документация для Movie Bot

## Обзор

Этот модуль представляет собой Телеграм-бот для поиска ссылок на бесплатный просмотр фильмов и сериалов. Он разработан на базе библиотеки Aiogram и использует сервисы Google для ассоциативного поиска, а также библиотеку Кинопоиска и сервис [w2.kpfr/wiki](https://w2.kpfr.wiki/). В боте реализована защита от флудинга с использованием middlewares, а для парсинга веб-страниц применяется BeautifulSoup.

## Подробнее

Этот бот предназначен для предоставления пользователям Telegram возможности находить ссылки на бесплатный просмотр фильмов и сериалов. Он интегрируется с различными сервисами для обеспечения наиболее релевантных результатов поиска. Защита от флудинга реализована для предотвращения злоупотребления ботом и обеспечения стабильной работы.

## Быстрый старт

### 1. Клонирование проекта

Склонируйте проект на свой компьютер, используя следующую команду:

```shell
git clone <repository_url>
```

### 2. Создание файла `.env`

Создайте файл `.env` в корневой директории проекта и заполните его в следующем формате:

```
TOKEN=123456789  # Токен вашего Telegram бота
```

Замените `123456789` на токен, полученный от BotFather в Telegram.

### 3. Создание и активация виртуального окружения

Если у вас нет виртуального окружения, создайте его с помощью следующей команды:

```shell
python -m venv venv
```

Активируйте виртуальное окружение:

```shell
venv\Scripts\activate.bat  # Для Windows
# или
source venv/bin/activate   # Для Linux/macOS
```

### 4. Установка зависимостей

Обновите `pip` и установите необходимые зависимости:

```shell
pip install --upgrade pip
pip install -r requirements.txt
```

### 5. Запуск бота

Запустите бота, выполнив следующую команду:

```shell
python run.py
```
