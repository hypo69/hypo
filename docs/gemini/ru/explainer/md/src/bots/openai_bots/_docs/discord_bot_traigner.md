## <algorithm>

Алгоритм работы бота состоит из последовательности шагов, описанных в документации:

1. **Запуск бота:** Программа запускается и ожидает сообщений от пользователя.
2. **Подготовка данных:** Пользователь готовит данные для обучения (текст или файл) и данные для тестирования (JSON-строка).
3. **Команда `!train`:**
    * Принимается команда `!train` с текстовыми данными или файлом в качестве вложения.
    * Данные сохраняются.
    * Запускается процесс обучения модели.
    * Отправляется уведомление о запуске обучения с уникальным идентификатором.
4. **Мониторинг обучения:** Бот следит за статусом процесса обучения.
5. **Команда `!test`:**
    * Принимается команда `!test` с JSON-строкой данных для тестирования.
    * Модель обрабатывает данные и выдает прогноз.
    * Отправляется ответ пользователю с прогнозом.
6. **Команды `!archive` и `!select_dataset`:**
    * Принимаются команды для управления данными.
    * Бот обрабатывает соответствующие запросы.
7. **Команда `!ask`:**
    * Принимается команда `!ask` с вопросом.
    * Модель отвечает на вопрос.
    * Отправляется ответ пользователю.
