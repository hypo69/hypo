# Модуль `hypotez/src/gui/openai_trаigner/main.py`

## Обзор

Этот модуль предоставляет пользовательский интерфейс для взаимодействия с моделями OpenAI. Он использует PyQt6 для создания графического окна, включающего веб-браузер для отображения страниц.  Модуль позволяет загружать URL, свертывать приложение в трей, и включает функциональность для выбора браузера по умолчанию.
