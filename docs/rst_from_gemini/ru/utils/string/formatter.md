# Модуль `formatter.py`

Файл: `hypotez/src/utils/string/formatter.py`

Этот модуль предоставляет функции для форматирования строк, включая удаление ненужных символов, HTML-тегов, нелатинских символов и др.

## Функции

**Обработка строк:**

* **`remove_line_breaks(input_str: str) -> str`**: Удаляет символы перевода строки (`\n`, `\r`) из входной строки.  Возвращает строку без символов перевода строки.
* **`remove_htmls(input_html: str) -> str`**: Удаляет HTML-теги из входной строки. Возвращает строку без HTML-тегов.
* **`escape_html_tags(input_html: str) -> str`**: Заменяет символы `<` и `>` на их HTML-экранированные аналоги (`&lt;` и `&gt;`) в входной HTML-строке. Важно для безопасности при отображении HTML в веб-странице.
* **`escape_to_html(text: str) -> str`**: Экранирует все символы в строке, чтобы они отображались корректно в HTML. Использует словарь `html_escapes`.
* **`remove_non_latin_characters(input_str: str) -> str`**: Удаляет все символы, не являющиеся латинскими буквами и пробелами.
* **`remove_special_characters(input_str: str | list) -> str | list`**: Удаляет все символы, не являющиеся латинскими буквами, цифрами и пробелами.  Принимает как строку, так и список строк и обрабатывает их соответственно.
* **`clear_numbers(input_str: str) -> str`**: Оставляет только цифры и точки (`.`) в строке, удаляя все другие символы.  Полезно для извлечения числовых значений из текста.


**Преобразование данных:**

* **`convert_to_list(input: str | list[str, dict, list[dict]], delimiter: str = ',') -> list`**: Преобразует строку или список в список строк, используя разделитель. Необходимо дополнить реализацию.
* **`extract_value_from_parentheses_with_lead_dollar(input_str: str) -> str | list | bool`**: Извлекает значение из строки, заключенной в скобки, с префиксом '$'. Необходимо дополнить реализацию.
* **`clean_url_from_protocols(url: str) -> str`**: Очищает URL-адрес, удаляя протокол (http://, https://, и т.д.). Необходимо дополнить реализацию.


## Класс `StringFormatter`

Класс `StringFormatter` предоставляет статические методы для форматирования строк.  Это улучшает организацию и читабельность кода.

## Дополнения и улучшения

* **`convert_to_list`**:  Необходимо добавить реализацию преобразования строки в список строк, используя `delimiter`. Должно быть обработано исключение, если `input` не является ни строкой, ни списком.
* **`extract_value_from_parentheses_with_lead_dollar`**: Необходимо описать поведение при отсутствии скобок или префикса '$' в строке.  Нужно рассмотреть разные варианты:
    * Возврат `None`
    * Поднятие исключения
    * Возврат пустой строки
* **`clean_url_from_protocols`**:  Необходимо реализовать очистку URL от протокола.
* **Документация**: Добавить примеры использования каждой функции.
* **Типизация**: Проверить корректность типов в возвращаемых значениях.
* **Обработка ошибок**: Добавить обработку возможных исключений (например, если передана неподходящая строка в функцию).

from hypotez.src.utils.string import formatter

string_with_breaks = "Строка\nс\nпереводами\nстрок"
formatted_string = formatter.remove_line_breaks(string_with_breaks)
print(formatted_string)  # Вывод: Строка с переводами строк

html_string = "<p>Пример</p> <b>текста</b>"
cleaned_string = formatter.remove_htmls(html_string)
print(cleaned_string)  # Вывод: Пример текста

number_string = "abc123.456def"
cleaned_number = formatter.clear_numbers(number_string)
print(cleaned_number) # Вывод: 123.456
