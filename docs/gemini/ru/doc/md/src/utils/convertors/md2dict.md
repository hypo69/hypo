# Модуль md2dict

## Обзор

Модуль `md2dict` предназначен для конвертации строк Markdown в структурированные словари Python.  Он поддерживает извлечение JSON содержимого, если оно присутствует в строке Markdown.

## Функции

### `md2dict`

**Описание**: Конвертирует строку Markdown в структурированный словарь, извлекая JSON контент, если он присутствует.

**Параметры**:

- `md_string` (str): Строка Markdown для конвертации.


**Возвращает**:

- `Dict[str, dict | list]`: Структурированное представление Markdown содержимого. Возвращает словарь с ключом "json", если найден JSON контент, или словарь с секциями Markdown.

**Вызывает исключения**:

- `Exception`: Возникает при ошибках при парсинге или извлечении JSON.  Подробности ошибки логгируются.


### `extract_json_from_string`

**Описание**: Извлекает JSON контент из строки, если он присутствует.

**Параметры**:

- `text` (str): Строка для извлечения JSON контента.

**Возвращает**:

- `dict | None`: Извлеченный JSON контент или `None`, если JSON не найден.

**Вызывает исключения**:

- `Exception`: Возникает при ошибках при парсинге JSON.  Подробности ошибки логгируются.


## Подробности

Этот модуль использует библиотеку `markdown2` для обработки Markdown.  Он ищет JSON-объекты в строке Markdown, используя регулярное выражение.  Если JSON найден, он возвращает словарь с ключом "json" и содержанием JSON. В противном случае возвращается словарь, содержащий заголовки Markdown и соответствующий им текст.

## Примеры использования

```python
import json
from hypotez.src.utils.convertors.md2dict import md2dict

markdown_string = """
# Заголовок 1
Текст после заголовка 1.

## Заголовок 2
```json
{"key": "value"}
```
Текст после заголовка 2.
"""

result = md2dict(markdown_string)
if 'json' in result:
    print("Найден JSON:")
    print(json.dumps(result['json'], indent=2))
else:
    print("JSON не найден:")
    print(result)
```

Этот пример демонстрирует, как использовать функцию `md2dict` и как проверить, был ли найден JSON.
