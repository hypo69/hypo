# Модуль hypotez/src/webdriver/edge/edge.py

## Обзор

Данный модуль предоставляет класс `Edge`, который расширяет функциональность стандартного WebDriver для браузера Edge. Он упрощает настройку драйвера, используя библиотеку `fake_useragent` для генерации случайных User Agent, и содержит полезные методы для работы с JavaScript и локаторами элементов.

## Классы

### `Edge`

**Описание**:  Класс `Edge` наследуется от `selenium.webdriver.Edge`. Он предоставляет расширенный функционал для работы с WebDriver Edge.

**Атрибуты**:

- `driver_name` (str): Имя используемого WebDriver, по умолчанию 'edge'.

**Методы**:

### `__init__`

**Описание**: Инициализирует экземпляр `Edge` WebDriver.

**Параметры**:

- `user_agent` (Optional[dict], optional): Словарь для указания User Agent. Если `None`, случайный User Agent генерируется автоматически.


**Возвращает**:
- None

**Обрабатываемые исключения**:

- `WebDriverException`: Возникает, если не удалось запустить Edge WebDriver.
- `Exception`: Возникает при других общих ошибках.


### `_payload`

**Описание**: Загружает исполнители для локаторов и сценариев JavaScript.

**Параметры**:
- Нет

**Возвращает**:
- None

**Обрабатываемые исключения**:
- Нет


### `set_options`

**Описание**: Создаёт и настраивает опции запуска для Edge WebDriver.

**Параметры**:

- `opts` (Optional[List[str]], optional): Список опций, которые необходимо добавить к Edge WebDriver. По умолчанию `None`.

**Возвращает**:
- `EdgeOptions`: Настроенный объект `EdgeOptions`.
