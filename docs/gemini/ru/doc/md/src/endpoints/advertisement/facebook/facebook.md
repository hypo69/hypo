# Модуль `hypotez/src/endpoints/advertisement/facebook/facebook.py`

## Обзор

Этот модуль предоставляет инструменты для взаимодействия с платформой Facebook через веб-драйвер.  Он включает функции для входа в аккаунт, публикации сообщений, загрузки медиа и управления рекламными постами.  Модуль использует сценарии, определенные в подпапке `scenarios`.

## Классы

### `Facebook`

**Описание**: Класс для взаимодействия с Facebook через веб-драйвер.  Обеспечивает методы для основных операций, таких как вход, публикация сообщений и продвижение постов.

**Атрибуты**:

- `d: Driver`: Объект веб-драйвера для управления сессией.
- `start_page:str`: Начальная страница Facebook.
- `promoter:str`:  Имя промоутера.


**Методы**:

#### `__init__(self, driver:Driver, promoter:str, group_file_paths: list[str], *args, **kwards)`

**Описание**: Инициализирует экземпляр класса.

**Параметры**:
- `driver (Driver)`: Объект веб-драйвера.
- `promoter (str)`: Имя промоутера.
- `group_file_paths (list[str])`: Список путей к файлам.


**Возвращает**:
- Не имеет возвращаемого значения.


**Примечания**:
- Метод принимает уже запущенный инстанс драйвера.

#### `login(self) -> bool`

**Описание**: Функция осуществляет вход в аккаунт.

**Возвращает**:
- `bool`: `True`, если вход успешен, иначе `False`.


#### `promote_post(self, item:SimpleNamespace) -> bool`

**Описание**: Функция публикует сообщение.

**Параметры**:
- `item (SimpleNamespace)`: Объект содержащий данные для публикации поста.


**Возвращает**:
- `bool`: `True`, если публикация успешна, иначе `False`.

**Примечания**:
- Параметр `message` содержит текст сообщения.  Символы `;` будут заменены на `SHIFT+ENTER`.


#### `promote_event(self,event:SimpleNamespace)`

**Описание**: Функция для продвижения события.

**Параметры**:
- `event (SimpleNamespace)`: Объект содержащий данные для продвижения события.

**Возвращает**:
- Не имеет возвращаемого значения.
