Модуль для проверки загрузки контента во фрейме
=========================================================================

Использует пространство имен tryxpath.

Пример использования
--------------------

.. code-block:: javascript

    if (tryxpath.isContentLoaded) {
      // Код выполняется, если контент загружен

// namespace
if (!tryxpath) {
    var tryxpath = {};
}

/**
