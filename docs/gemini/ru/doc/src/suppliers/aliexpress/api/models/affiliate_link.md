# Модуль `hypotez/src/suppliers/aliexpress/api/models/affiliate_link.py`

## Обзор

Модуль `affiliate_link.py` содержит определение класса `AffiliateLink`, предназначенного для представления данных об аффилированной ссылке с AliExpress.  Класс содержит поля для хранения ссылки и источника данных.

## Классы

### `AffiliateLink`

**Описание**: Представляет аффилированную ссылку с AliExpress.

**Атрибуты**:

- `promotion_link` (str): Ссылка на промоакцию.
- `source_value` (str): Источник данных о ссылке.
