# Задача No 302
## Гиперкубические числа

## Условия
Определим числа *h(n)* как целые числа, которые могут быть представлены в виде суммы кубов целых чисел, где каждое слагаемое является кубом целого числа, абсолютное значение которого не превышает *n*. Например, 
*h*(1) = {-1, 0, 1}, 
*h*(2) = {-9, -8, ..., 0, ..., 8, 9}

Пусть *H*(n) обозначает количество различных *h(n)*. Тогда *H*(1) = 3, *H*(2) = 19.

Определим *S(N)* как сумму всех *H(n)* для *1 <= n <= N*.
Тогда *S*(10) = 819.

Найдите *S(10^6)*.
