# Модуль hypotez/src/goog/spreadsheet/__init__.py

## Обзор

Данный модуль предоставляет интерфейсы для работы с Google Spreadsheets. Он импортирует необходимые классы из модулей `spreadsheet` и `reach_spreadsheet`.

## Переменные

### `MODE`

**Описание**: Переменная, хранящая режим работы модуля.  В текущей реализации хранит строку 'dev'.

## Импорты

### `from .spreadsheet import SpreadSheet`

**Описание**: Импортирует класс `SpreadSheet` из модуля `spreadsheet`.


### `from .reach_spreadsheet import ReachSpreadsheet`

**Описание**: Импортирует класс `ReachSpreadsheet` из модуля `reach_spreadsheet`.
