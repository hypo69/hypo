# Модуль utils

## Обзор

Модуль `utils` содержит набор полезных утилит для различных задач программирования, включая обработку данных, файлов, форматирование вывода, работу с датами и временем, изображениями, JSON, CSV, HTML, PDF и видео.  Он предоставляет инструменты для преобразования данных, работы с файлами и упрощения общих задач.

## Функции

### `base64_to_tmpfile`

**Описание**: Преобразует данные в формате Base64 в временный файл.

**Параметры**:
- `base64_data` (str): Данные в формате Base64.

**Возвращает**:
- `str`: Путь к временному файлу.


### `base64encode`

**Описание**: Кодирует данные в формате Base64.

**Параметры**:
- `data` (str): Данные для кодирования.

**Возвращает**:
- `str`: Данные в формате Base64.


### `csv2dict`

**Описание**: Преобразует CSV-данные в словарь.

**Параметры**:
- `csv_data` (str): CSV-данные.

**Возвращает**:
- `list[dict]`: Список словарей, представляющих CSV-данные.


### `csv2ns`

**Описание**: Преобразует CSV-данные в список словарей с именованными столбцами.


**Параметры**:
- `csv_data` (str): CSV-данные.


**Возвращает**:
- `list[dict]`: Список словарей, представляющих CSV-данные с именованными столбцами.

(и так далее для остальных функций)
