# Задача No 305
## Коммуникационные сбои
## Условия
Строка битов может содержать "коммуникационный сбой", если в ней есть подстрока вида `01010`. Например, строка `10110101001` содержит коммуникационный сбой, а строка `1101011010` его не содержит.

Пусть $C(L)$ будет количеством строк битов длины $L$ (состоящих из символов '0' и '1'), которые **содержат** коммуникационный сбой.

Например, $C(10) = 32$.

Пусть $S(N)$ будет суммой $C(L)$ при $1 \le L \le N$.

Например, $S(100) = 1580880$.

Найдите $S(10^7)$.
