### Название игры: **DIAMND** (Рисование ромба)

---

#### Описание  
**DIAMND** — это игра, в которой пользователь задаёт размер ромба, а программа создаёт его с использованием символов, отображая результат на экране. Ромб состоит из последовательности строк, где каждая строка формируется из пробелов и символов, формируя симметричный узор.

---

### Пошаговая инструкция для реализации

#### 1. **Инициализация игры**  
   - При запуске игры программа приветствует пользователя и объясняет её правила.  
   - Пользователь вводит размер ромба, задающий высоту его верхней половины (от центральной строки до верхнего угла). Размер должен быть положительным целым числом.  

#### 2. **Создание ромба**  
   - Программа строит ромб по следующим правилам:
     1. Ромб состоит из двух частей:
        - Верхняя часть, которая увеличивается в ширине на каждой строке.
        - Нижняя часть, которая уменьшается в ширине на каждой строке.
     2. Для каждой строки рассчитываются:
        - Количество пробелов слева от символов.
        - Количество символов (`*` или другого символа, если выбрано пользователем).  

#### 3. **Вывод результата**  
   - После расчёта всех строк ромб выводится на экран.
   - Пользователю предлагается возможность нарисовать новый ромб с другим размером или завершить игру.

#### 4. **Обработка ошибок**  
   - Если пользователь вводит некорректное значение (например, отрицательное число, строку или нечисловое значение), программа выводит сообщение об ошибке и запрашивает ввод повторно.  

---

### Пример работы программы  

1. **Начало игры:**  
   ```  
   Добро пожаловать в игру DIAMND!  
   Введите размер ромба (положительное целое число):  
   > 4  
   ```  

2. **Вывод ромба:**  
   ```  
      *
     ***
    *****
   *******
    *****
     ***
      *  
   ```  

3. **Продолжение игры:**  
   ```  
   Хотите создать новый ромб? (да/нет)  
   > да  
   Введите размер ромба (положительное целое число):  
   > 2  
     *
    ***
     *  
   Хотите создать новый ромб? (да/нет)  
   > нет  
   Спасибо за игру!  
   ```  

---

### Возможные ограничения  
- Размер ромба должен быть положительным целым числом. Ввод некорректных данных обрабатывается с повторным запросом.  
- Символ ромба фиксирован (`*`), но может быть изменён в реализации.  

---

### Реализация  
Игра может быть реализована на Python с использованием базовых циклов, условий и работы с выводом на экран.  
Рекомендуется:
- Создать отдельные функции для ввода данных, расчёта строк ромба и его отображения.
- Добавить обработку исключений для некорректного ввода.
