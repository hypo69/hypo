# Модуль `affiliated_products_generator.en.md`

## Обзор

Этот модуль предоставляет класс `AliAffiliatedProducts`, предназначенный для сбора данных о продуктах с AliExpress и создания аффилированных ссылок.  Он позволяет получить ссылки на продукты, а также сохранить изображения и видео, связанные с ними, в локальную систему.

## Классы

### `AliAffiliatedProducts`

**Описание**: Класс `AliAffiliatedProducts` используется для получения информации о продуктах с AliExpress и создания аффилированных ссылок. Он принимает параметры кампании, такие как название, категория, язык и валюта.

**Методы**:

- `process_affiliate_products`: Обрабатывает список URL или ID продуктов и возвращает список объектов `Product` с аффилированными ссылками и сохранёнными изображениями/видео.


## Функции

### `main`

**Описание**: Функция `main` демонстрирует пример использования класса `AliAffiliatedProducts`. Она настраивает параметры кампании, создаёт экземпляр класса, обрабатывает список продуктов, а затем выводит полученные данные.

**Параметры**:

- Нет явных параметров, но функция использует переменные, заданные внутри неё для параметров кампании.


**Возвращает**:
- Нет возвращаемого значения.

**Вызывает исключения**:
- Любые исключения, возникающие в методе `process_affiliate_products` класса `AliAffiliatedProducts`.
