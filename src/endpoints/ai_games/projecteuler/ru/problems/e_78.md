# Задача No 78
## Монеты

## Условия

Пусть p(n) представляет количество способов, которыми n монет могут быть разбиты на кучки. Например, есть ровно пять способов разбить пять монет на кучки:

     OOOOO
     OOOO   O
     OOO   OO
     OOO   O   O
     OO   OO   O
     OO   O   O   O
     O   O   O   O   O

Найдите наименьшее значение n, для которого p(n) делится на миллион.
