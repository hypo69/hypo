# Игра "Ферзь"

## Обзор

Данный модуль реализует игру "Ферзь" для двух игроков. Игроки по очереди перемещают своего ферзя по шахматной доске, стремясь достичь противоположного края доски первым.

## Функции

### `print_board`

**Описание**:
Выводит на экран шахматную доску с указанием текущих позиций ферзей.

**Параметры**:
- `player1_x` (int): X-координата ферзя игрока 1.
- `player1_y` (int): Y-координата ферзя игрока 1.
- `player2_x` (int): X-координата ферзя игрока 2.
- `player2_y` (int): Y-координата ферзя игрока 2.

**Возвращает**:
- `None`: Функция ничего не возвращает.

### `is_valid_move`

**Описание**:
Проверяет, является ли ход ферзя допустимым.
Ход допустим, если ферзь двигается по горизонтали, вертикали или диагонали.

**Параметры**:
- `current_x` (int): Текущая X-координата ферзя.
- `current_y` (int): Текущая Y-координата ферзя.
- `next_x` (int): Следующая X-координата ферзя.
- `next_y` (int): Следующая Y-координата ферзя.

**Возвращает**:
- `bool`: `True`, если ход допустим, `False` в противном случае.

### `get_player_move`

**Описание**:
Запрашивает у игрока ввод координат для перемещения ферзя.
Проверяет допустимость введенных координат.
Возвращает новые координаты.

**Параметры**:
- `player_number` (int): Номер игрока (1 или 2).
- `current_x` (int): Текущая X-координата ферзя игрока.
- `current_y` (int): Текущая Y-координата ферзя игрока.

**Возвращает**:
- `tuple[int, int]`: Кортеж из двух целых чисел, представляющих следующие координаты (`next_x`, `next_y`).

**Вызывает исключения**:
- `ValueError`: Если ввод не является допустимым (не два числа через запятую).
