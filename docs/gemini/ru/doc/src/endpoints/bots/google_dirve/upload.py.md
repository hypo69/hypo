# Модуль для загрузки файлов в Google Drive

## Обзор

Модуль предоставляет функциональность для загрузки файлов в Google Drive с использованием PyDrive. Он поддерживает загрузку как в личные папки, так и в Team Drive, а также создание папок при необходимости.

## Подробней

Этот модуль предназначен для интеграции с ботами и позволяет пользователям загружать файлы непосредственно в Google Drive. Он использует учетные данные пользователей для аутентификации и загрузки файлов. Модуль также обеспечивает возможность создания папок в Google Drive, если они не существуют.

## Функции

### `upload`

```python
def upload(filename: str, update, context, parent_folder: str = None) -> None:
    """
    Args:
        filename (str): Имя файла для загрузки.
        update: Объект update от Telegram Bot API.
        context: Контекст от Telegram Bot API.
        parent_folder (str, optional): Имя родительской папки в Google Drive. По умолчанию `None`.

    Returns:
        None: Функция ничего не возвращает.

    Raises:
        Exception: Если возникает ошибка при загрузке файла.

    Example:
        >>> upload('example.txt', update, context, 'MyFolder')
    """
```

**Описание**: Загружает указанный файл в Google Drive. Если указана родительская папка, файл будет загружен в эту папку. Если родительская папка не существует, она будет создана.

**Параметры**:
- `filename` (str): Путь к файлу, который нужно загрузить.
- `update`: Объект, представляющий входящее обновление от Telegram Bot API. Используется для получения ID пользователя.
- `context`: Контекст, переданный из Telegram Bot API.
- `parent_folder` (str, optional): Название папки в Google Drive, в которую нужно загрузить файл. Если не указана, файл будет загружен в корень диска. По умолчанию `None`.

**Возвращает**:
- `str`: Ссылка на загруженный файл в Google Drive.

**Вызывает исключения**:
- `Exception`: Возникает, если происходит ошибка при загрузке файла в Google Drive.

**Примеры**:

Пример вызова функции для загрузки файла 'example.txt' в папку 'MyFolder':

```python
upload('example.txt', update, context, 'MyFolder')
```
