# Инструкция по безопасности

## Обзор

Этот документ содержит информацию о том, как сообщать об уязвимостях в проекте `hypotez`.

## Подробней

Этот документ описывает процесс сообщения о потенциальных проблемах безопасности. Пожалуйста, используйте указанный канал для предоставления информации об уязвимостях, чтобы команда разработчиков могла оперативно реагировать и устранять проблемы.

## Как сообщить об уязвимости

Пожалуйста, сообщайте о (предполагаемых) уязвимостях безопасности по адресу https://t.me/xtekky. Вы получите ответ в течение 48 часов. Если проблема будет подтверждена, мы выпустим исправление как можно скорее, в зависимости от сложности, но исторически в течение нескольких дней.
