# Инструкция для бота

## Обзор

Этот файл содержит инструкции для бота, определяющие доступные команды и их параметры. Каждая строка представляет собой команду с описанием ее функциональности и требуемых аргументов.

## Подробнее

Файл предназначен для определения команд, которые бот может понимать и выполнять. Каждая команда начинается с символа `!`, за которым следует название команды и список аргументов.

## Команды

### `!hi`

**Описание**: Приветствует пользователя.

### `!train`

**Описание**: Обучает модель предоставленными данными.

**Параметры**:
- `data` (str): Файл с данными для обучения.
- `data_dir` (str): Каталог с данными для обучения.
- `positive` (str): Положительные данные.
- `attachment` (file): Файл, прикрепленный к сообщению.

### `!test`

**Описание**: Тестирует модель с предоставленными тестовыми данными в формате JSON.

**Параметры**:
- `test_data` (str): JSON данные для тестирования.

### `!archive`

**Описание**: Архивирует файлы в указанном каталоге.

**Параметры**:
- `directory` (str): Каталог для архивации.

### `!select_dataset`

**Описание**: Выбирает набор данных для обучения из указанного каталога.

**Параметры**:
- `path_to_dir_positive` (str): Путь к каталогу с положительными данными.
- `positive` (str): Положительные данные.

### `!instruction`

**Описание**: Отображает это сообщение с инструкциями.
