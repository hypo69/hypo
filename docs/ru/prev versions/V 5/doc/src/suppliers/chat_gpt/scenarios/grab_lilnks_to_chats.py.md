# Модуль `grab_lilnks_to_chats.py`

## Обзор

Модуль предназначен для извлечения ссылок на отдельные чаты с веб-страницы, используя Selenium WebDriver. В настоящее время функциональность ограничена из-за проблем с драйверами Chrome и Firefox.

## Подробней

Этот модуль является частью проекта `hypotez` и предназначен для автоматизации сбора данных. В частности, он пытается получить ссылки на чаты с сайта `chatgpt.com`. Из-за проблем с драйверами, указанными в коде, текущая реализация может быть нерабочей.

## Классы

В данном модуле классы не используются.

## Функции

### `get_links`

```python
def get_links(d:Driver):
    """Ссылки на отдельные чаты """
    ...
```

**Описание**: Функция для извлечения ссылок на отдельные чаты.

**Как работает функция**:
Функция `get_links` принимает объект драйвера `d` в качестве аргумента. Затем она использует метод `execute_locator` объекта `d` для поиска элементов, соответствующих локатору `locator.link`. Предполагается, что `locator.link` содержит информацию о том, как найти ссылки на чаты на веб-странице. Функция возвращает найденные ссылки.

**Параметры**:
- `d` (Driver): Объект драйвера, используемый для взаимодействия с веб-страницей.

**Возвращает**:
- `links`: Список ссылок на отдельные чаты, найденные на странице.

**Примеры**:

```python
from src.webdriver.driver import Driver
from src.webdriver.firefox import Firefox
from src.utils.jjson import j_loads_ns
from src import gs
locator = j_loads_ns(gs.path.src / 'suppliers' / 'chat_gpt' / 'locators' / 'chats_list.json')

d = Driver(Firefox)
d.get_url('https://chatgpt.com/')
links = get_links(d)
if links:
    print(f'Найдено {len(links)} ссылок на чаты.')
else:
    print('Ссылки на чаты не найдены.')
```

## Main

В блоке `if __name__ == '__main__':` выполняется следующий код:

- Создается экземпляр драйвера `Driver` с использованием `Firefox`.
- Открывается URL `https://chatgpt.com/`.
- Вызывается функция `get_links` для получения ссылок на чаты.
