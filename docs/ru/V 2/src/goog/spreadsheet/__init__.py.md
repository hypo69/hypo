# Модуль `src.goog.spreadsheet`

## Обзор

Модуль `src.goog.spreadsheet` предоставляет классы для работы с Google Sheets API, включая класс `SpreadSheet` для основных операций с таблицами и класс `ReachSpreadsheet` для более сложных задач.

## Оглавление

1. [Классы](#классы)
    - [`SpreadSheet`](#spreadsheet)
    - [`ReachSpreadsheet`](#reachspreadsheet)
2. [Импорты](#импорты)

## Классы

### `SpreadSheet`

**Описание**:
Класс для работы с Google Sheets API. Предоставляет базовые функции для чтения, записи и манипулирования данными в таблицах.

*Для подробной информации о методах класса см. файл `spreadsheet.py`*

### `ReachSpreadsheet`

**Описание**:
Класс для расширенного взаимодействия с Google Sheets API. Включает более продвинутые функции для обработки данных, запросов и интеграций.

*Для подробной информации о методах класса см. файл `reach_spreadsheet.py`*

## Импорты

Модуль импортирует следующие классы из файлов:
- `SpreadSheet` из `spreadsheet.py`
- `ReachSpreadsheet` из `reach_spreadsheet.py`
