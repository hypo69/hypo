# 101 игра на Python

&nbsp;&nbsp;&nbsp;
Добро пожаловать!

Этот репозиторий создан специально для тех, кто только начинает знакомство с программированием. 
Здесь представлены игры из культовой книги "101 BASIC Computer Games", адаптированные для языка Python. 
Каждая игра — это пример того, как простыми средствами можно реализовать сложную логику. 
Код написан с учетом начинающих программистов: легко читаемый, с понятными комментариями и примерами, 
чтобы вы могли быстро понять принципы работы и начать экспериментировать.

Канал на dzen.ru [публикацию](https://dzen.ru/hypo69), где я разбираю код репозитория. Подпишись, будет интересно

Я хочу показать тебе, как всего несколько строк кода могут воплотить сложные идеи, чтобы ты мог без труда разобраться в логике написания программы, 
вдохновиться и начать свои эксперименты.
