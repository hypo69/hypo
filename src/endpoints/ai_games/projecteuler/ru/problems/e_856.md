# Задача No 856
## Проверка целостности
## Условия

Рассмотрим последовательность S = {s_1, s_2, ..., s_n}, где каждый элемент s_i представляет собой целое число.  
Мы можем выполнить два вида операций с последовательностью S:
 * **Срез:** выберите непрерывный подсписок {s_i, ..., s_j}, и удалите все элементы, кроме s_i и s_j.
 * **Слияние:** выберите два соседних элемента s_i и s_{i+1}, и замените их на s_i + s_{i+1}.

После выполнения некоторого (возможно, нулевого) количества таких операций, мы получаем последовательность из одного элемента.

Мы говорим, что последовательность S является последовательностью **проверки целостности**, если, применяя некоторые операции из набора разрешенных, можно получить один элемент, равный 0.

Например, S = {1, -1, 2, -2} является последовательностью проверки целостности:
{1, -1, 2, -2} -> {0, 2, -2} -> {0, 0} -> {0}

S = {2, 4, -5} также является последовательностью проверки целостности:
{2, 4, -5} -> {6, -5} -> {1} - не приводит к 0, поэтому это не последовательность проверки целостности.
{2, 4, -5} -> {2, -1} -> {1} - не приводит к 0, поэтому это не последовательность проверки целостности.
{2, 4, -5} -> {2, 4, -5} - не приводит к 0, поэтому это не последовательность проверки целостности.

S = {2, 2, -2} является последовательностью проверки целостности:
{2, 2, -2} -> {4, -2} -> {2} - не приводит к 0, поэтому это не последовательность проверки целостности.
{2, 2, -2} -> {2, 0} -> {2} - не приводит к 0, поэтому это не последовательность проверки целостности.
{2, 2, -2} -> {2, 2, -2} - не приводит к 0, поэтому это не последовательность проверки целостности.

S = {1, 2, 3} не является последовательностью проверки целостности:
{1, 2, 3} -> {3, 3} -> {6}
{1, 2, 3} -> {1, 5} -> {6}
{1, 2, 3} -> {1, 2, 3} - и так далее

Рассмотрим последовательность A(n), полученную как:
A(n) = 1, 2, 3, ..., n.

Пусть C(n) будет количеством непрерывных подсписков A(n), которые являются последовательностями проверки целостности. Например, C(4) = 6, эти последовательности таковы:
 {1, -1} ,
 {1, 2, -3},
 {1, 2, 3, -6},
 {2, -2},
 {2, 3, -5},
 {3, -3}.
Также, C(10)= 147.
Найдите C(1000).
