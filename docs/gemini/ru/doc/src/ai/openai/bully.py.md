# Модуль для разведения chatGPT на грубости

## Обзор

Модуль `bully.py` предназначен для демонстрации того, как можно спровоцировать модель ChatGPT на генерацию грубых высказываний. Он использует OpenAI API для взаимодействия с моделью и предоставляет функцию `bully`, которая отправляет сообщения пользователю с определенным системным запросом, чтобы вызвать нежелательный ответ.

## Подробней

Этот модуль показывает, как можно манипулировать AI-моделями для получения ответов, которые нарушают этические нормы. В данном случае, модуль использует специально разработанный системный запрос, чтобы заставить модель выдать примеры буллинга.

## Функции

### `bully`

```python
def bully(user_message="Hello!", messages=[{"system": "user", "content": system_prompt}]) -> list:
    """
    Генерирует ответ от ChatGPT на основе заданного сообщения пользователя и истории сообщений,
    провоцируя модель на грубость.

    Args:
        user_message (str, optional): Сообщение пользователя. По умолчанию "Hello!".
        messages (list, optional): Список сообщений для контекста беседы. По умолчанию
            [{"system": "user", "content": system_prompt}].

    Returns:
        list: Список сообщений, включающий сообщения пользователя и ответы модели.
    """
```

**Как работает функция**:

Функция `bully` принимает сообщение пользователя и список сообщений, добавляет сообщение пользователя к списку, отправляет запрос в OpenAI API, получает ответ от модели, добавляет ответ модели к списку сообщений и возвращает обновленный список сообщений. Функция использует модель `gpt-3.5-turbo` для генерации ответа.

Внутри функции происходят следующие действия:
A. Добавление сообщения пользователя к истории сообщений.
B. Отправка запроса в OpenAI API с использованием истории сообщений.
C. Получение ответа от модели.
D. Добавление ответа модели к истории сообщений.
E. Возврат обновленной истории сообщений.

**Параметры**:

- `user_message` (str, optional): Сообщение пользователя, которое будет отправлено модели. По умолчанию "Hello!".
- `messages` (list, optional): Список сообщений, представляющих контекст разговора. По умолчанию содержит системный запрос для провоцирования модели.

**Возвращает**:

- `list`: Список сообщений, включающий сообщения пользователя и ответы модели.

**Пример**:

```python
