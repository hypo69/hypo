# Модуль для работы с Telegram ботом на Pyrogram
## Обзор
Этот модуль содержит простой Telegram бот, созданный с использованием библиотеки Pyrogram. Бот отвечает на команду `/start` приветственным сообщением и эхом повторяет все остальные текстовые сообщения, которые он получает.

## Подробнее

Этот код предоставляет базовую функциональность для создания и запуска Telegram бота. Он использует библиотеку Pyrogram для обработки входящих сообщений и отправки ответов.

## Функции

### `start_command`

```python
def start_command(client, message):
    """
    Обработчик команды /start. Отправляет приветственное сообщение в ответ.

    Args:
        client: Экземпляр клиента Pyrogram.
        message: Объект сообщения, представляющий входящее сообщение.

    Returns:
        None

    """
```

**Как работает функция**:

1.  Функция `start_command` является обработчиком команды `/start`.
2.  Когда бот получает сообщение с командой `/start`, эта функция вызывается автоматически.
3.  Функция отправляет ответное сообщение "Привет! Я простой бот на Pyrogram." пользователю, который отправил команду.

**Параметры**:

*   `client`: Экземпляр клиента Pyrogram, используемый для взаимодействия с Telegram API.
*   `message`: Объект сообщения, содержащий информацию о входящем сообщении (например, текст сообщения, отправитель, чат).

**Возвращает**:

*   `None`: Функция ничего не возвращает явно.

**Примеры**:

```python
# Пример вызова команды /start пользователем
/start

# Ответ бота
Привет! Я простой бот на Pyrogram.
```

```python
        message: Объект сообщения, представляющий входящее сообщение.

    Returns:
        None

    """
```

**Как работает функция**:

1.  Функция `echo_message` является обработчиком всех текстовых сообщений, которые не являются командами.

**Возвращает**:

*   `None`: Функция ничего не возвращает явно.

# Пример отправки текстового сообщения боту

# Ответ бота
