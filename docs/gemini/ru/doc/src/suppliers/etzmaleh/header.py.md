# Модуль `src.suppliers.etzmaleh.header`

## Обзор

Модуль `header.py` предназначен для определения и установки корневой директории проекта, а также загрузки настроек и документации из соответствующих файлов. Он используется для инициализации проекта и обеспечения доступа к общим ресурсам.

## Функции

### `set_project_root`

**Описание**:
Находит корневой каталог проекта, начиная с каталога текущего файла, ища вверх и останавливаясь на первом каталоге, содержащем любой из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple, optional): Имена файлов или каталогов, используемые для идентификации корневого каталога проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневому каталогу, если он найден, в противном случае — путь к каталогу, где расположен скрипт.

## Глобальные переменные

### `__root__`
**Описание**:
Путь к корневой директории проекта. Определяется функцией `set_project_root`.

### `settings`
**Описание**:
Словарь с настройками проекта, загруженный из файла `settings.json`. Может быть `None`, если файл не найден или произошла ошибка декодирования JSON.

### `doc_str`
**Описание**:
Строка с документацией проекта, загруженная из файла `README.MD`. Может быть `None`, если файл не найден или произошла ошибка декодирования JSON.

### `__project_name__`
**Описание**:
Имя проекта. По умолчанию "hypotez", или значение из `settings.json`, если оно есть.

### `__version__`
**Описание**:
Версия проекта. По умолчанию пустая строка, или значение из `settings.json`, если оно есть.

### `__doc__`
**Описание**:
Строка с документацией проекта. Загружается из файла `README.MD` или пустая строка.

### `__details__`
**Описание**:
Строка с дополнительными деталями проекта. По умолчанию пустая строка.

### `__author__`
**Описание**:
Автор проекта. По умолчанию пустая строка, или значение из `settings.json`, если оно есть.

### `__copyright__`
**Описание**:
Авторские права проекта. По умолчанию пустая строка, или значение из `settings.json`, если оно есть.

### `__cofee__`
**Описание**:
Строка с сообщением о поддержке разработчика. По умолчанию "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69", или значение из `settings.json`, если оно есть.
