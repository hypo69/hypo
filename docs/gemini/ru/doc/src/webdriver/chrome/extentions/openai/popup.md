# Документация для файла hypotez/src/webdriver/chrome/extentions/openai/popup.html

## Обзор

Файл `popup.html` представляет собой интерфейс пользователя для взаимодействия с моделью OpenAI через веб-приложение.  Он содержит две вкладки: "Chat" для обмена сообщениями с моделью и "Model" для управления обучением модели.

## Оглавление

- [Обзор](#обзор)
- [Вкладка "Chat"](#вкладка-chat)
- [Вкладка "Model"](#вкладка-model)


## Вкладка "Chat"

### Описание

Вкладка "Chat" позволяет пользователю общаться с обученной моделью OpenAI.  Пользователь вводит сообщение, а модель генерирует ответ.

### Элементы управления

- **Выпадающий список помощников (Assistants):** Позволяет выбрать помощника для взаимодействия.
- **Поле для ввода сообщения:** Текстовое поле для ввода запроса к модели.
- **Кнопка "Отправить":** Отправляет сообщение модели.
- **Поле отображения ответа:** Отображает ответ модели.


## Вкладка "Model"

### Описание

Вкладка "Model" предоставляет инструменты для управления и отслеживания состояния обучаемой модели.

### Элементы управления

- **Поле для ввода данных обучения (Training Data):** Текстовое поле для ввода данных, используемых для обучения модели.
- **Кнопка "Обучить (Train)":** Запускает процесс обучения модели на предоставленных данных.
- **Поле отображения статуса обучения (Training Status):** Отображает текущий статус процесса обучения модели.


## Скрипты

Файл `popup.html` использует внешние скрипты `angular.min.js`, `jquery-3.5.1.slim.min.js`, и `popup.js` для функциональности приложения. Файлы `style.css` содержит стили для отображения страницы.
