# Модуль `hypotez/src/suppliers/hb/category.py`

## Обзор

Данный модуль отвечает за сбор информации о товарах с сайта поставщика hb.co.il. Он реализует логику получения списка категорий, списка товаров в категории и парсинга данных о товарах. Модуль использует веб-драйвер для взаимодействия с сайтом и обрабатывает возможные ошибки, такие как отсутствие ссылок на товары или проблемы с навигацией по страницам.

## Функции

### `get_list_products_in_category`

**Описание**: Функция собирает ссылки на товары на странице категории.  Она обрабатывает возможные случаи отсутствия ссылок на товары и использует функцию `paginator` для обработки страниц с товарами.

**Параметры**:

- `s` (Supplier): Экземпляр класса `Supplier`, содержащий информацию о поставщике, веб-драйвере и локаторах.

**Возвращает**:

- `list[str, str, None]`: Список URL-адресов товаров или `None`, если ссылок на товары не найдено.  Список может содержать один URL, если на странице всего один продукт.

**Вызывает исключения**:

- Возможны исключения, генерируемые веб-драйвером или другими зависимостями, если возникают проблемы с доступом к странице или обработкой данных.


### `paginator`

**Описание**: Функция обрабатывает пагинацию на страницах категорий.

**Параметры**:

- `d` (Driver): Экземпляр веб-драйвера.
- `locator` (dict): Словарь с локаторами элементов для пагинации.
- `list_products_in_category` (list): Список URL-адресов товаров.

**Возвращает**:

- `bool`: `True`, если были найдены элементы для пагинации, `False` в противном случае.

**Вызывает исключения**:

- Возможны исключения, генерируемые веб-драйвером или другими зависимостями, если возникают проблемы с доступом к странице или обработкой данных.


### `get_list_categories_from_site`

**Описание**: Функция для сбора списка категорий с сайта.

**Параметры**:

- `s`: Экземпляр класса `Supplier`.

**Возвращает**:

- Результат выполнения.

**Вызывает исключения**:

- Возможны исключения, генерируемые веб-драйвером или другими зависимостями, если возникают проблемы с доступом к странице или обработкой данных.
