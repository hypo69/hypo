# Задача No 129
## Repunit Divisibility
## Условия
Репьюнит, R(k), это число, состоящее только из единиц. Например, R(6) = 111111.

Обозначим через A(n) наименьшее значение k, для которого R(k) делится на n.
Например, A(7) = 6 и A(41) = 5.

Если n не является взаимно простым с 10, то  R(k) никогда не будет делиться на n.

Наименьшее значение n, для которого A(n) > 10, равно 17.

Найдите наименьшее значение n, для которого A(n) > 10^6.
