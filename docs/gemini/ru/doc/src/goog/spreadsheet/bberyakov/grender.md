# Модуль grender

## Обзор

Данный модуль предоставляет инструменты для форматирования и рендеринга данных в Google Таблицах. Он содержит классы и функции для работы с Google Spreadsheets API, включая форматирование ячеек, объединение ячеек, установку свойств строк, добавление строк и управление данными.

## Классы

### `GSRender`

**Описание**: Класс `GSRender` содержит методы для рендеринга и форматирования Google Таблиц.

**Методы**:

- `__init__(*args, **kwards)`:
    **Описание**: Конструктор класса.
    **Параметры**:
        - `self`: Экземпляр класса.
        - `*args`: Аргументы.
        - `**kwards`: Именованные аргументы.
    **Возвращает**:
        - `None`:  Не возвращает значений.
    **Примечание**:  Этот метод пока содержит placeholder (`...`).

- `render_header(ws: Worksheet, world_title: str, range: str = 'A1:Z1', merge_type: str('MERGE_ALL') | str('MERGE_COLUMNS') | str('MERGE_ROWS') = 'MERGE_ALL') -> None`:
    **Описание**: Рисует заголовок таблицы в первой строке.
    **Параметры**:
        - `self`: Экземпляр класса.
        - `ws` (Worksheet): Объект Worksheet для работы с таблицей.
        - `world_title` (str): Заголовок Google таблицы.
        - `range` (str, опционально): Диапазон ячеек (по умолчанию 'A1:Z1').
        - `merge_type` (str('MERGE_ALL') | str('MERGE_COLUMNS') | str('MERGE_ROWS'), опционально): Тип объединения ячеек (по умолчанию 'MERGE_ALL').
    **Возвращает**:
        - `None`: Не возвращает значений.

- `merge_range(self, ws: Worksheet, range: str, merge_type: str('MERGE_ALL') | str('MERGE_COLUMNS') | str('MERGE_ROWS') = 'MERGE_ALL') -> None`:
    **Описание**: Объединяет ячейки в указанном диапазоне.
    **Параметры**:
        - `self`: Экземпляр класса.
        - `ws` (Worksheet): Объект Worksheet.
        - `range` (str): Диапазон ячеек.
        - `merge_type` (str('MERGE_ALL') | str('MERGE_COLUMNS') | str('MERGE_ROWS')): Тип объединения.
    **Возвращает**:
        - `None`: Не возвращает значений.
    **Примечание**: Документируйте возможные значения `merge_type`.

- `set_worksheet_direction(self, sh: Spreadsheet, ws: Worksheet, direction: str('ltr') | str('rtl') = 'rtl')`:
    **Описание**: Устанавливает направление текста в листе.
    **Параметры**:
        - `self`: Экземпляр класса.
        - `sh` (Spreadsheet): Объект Spreadsheet.
        - `ws` (Worksheet): Объект Worksheet.
        - `direction` (str('ltr') | str('rtl')): Направление текста.
    **Возвращает**:
        - `None`: Не возвращает значений.


- `header(self, ws: Worksheet, ws_header: str | list, row: int = None)`:
    **Описание**: Добавляет заголовок в таблицу.
    **Параметры**:
        - `self`: Экземпляр класса.
        - `ws` (Worksheet): Объект Worksheet.
        - `ws_header` (str | list): Заголовок (строка или список).
        - `row` (int, опционально): Номер строки для вставки заголовка.
    **Возвращает**:
        - `None`: Не возвращает значений.


- `write_category_title(self, ws: Worksheet, ws_category_title: str | list, row: int = None)`:
    **Описание**: Добавляет заголовок категории в таблицу.
    **Параметры**:
        - `self`: Экземпляр класса.
        - `ws` (Worksheet): Объект Worksheet.
        - `ws_category_title` (str | list): Заголовок категории.
        - `row` (int, опционально): Номер строки для вставки заголовка.
    **Возвращает**:
        - `None`: Не возвращает значений.


- `get_first_empty_row(self, ws: Worksheet, by_col: int = None) -> int`:
    **Описание**: Возвращает номер первой пустой строки.
    **Параметры**:
        - `self`: Экземпляр класса.
        - `ws` (Worksheet): Объект Worksheet.
        - `by_col` (int, опционально): Номер столбца для поиска.
    **Возвращает**:
        - `int`: Номер первой пустой строки.
    **Примечание**: Документировать поведение при `by_col = None`.
