# Модуль _experiments (JUPYTER_header.py)

## Обзор

Модуль `JUPYTER_header.py` предназначен для экспериментов с данными, касающимися поставщиков, в частности, для работы с данными, полученными из `cdata`. Он включает в себя импорты различных модулей и классов, необходимых для выполнения задач, связанных с поставщиками, продуктами, категориями и веб-драйверами.

## Подробней

Этот файл, расположенный в каталоге `src/suppliers/cdata/_experiments`, содержит код, необходимый для запуска поставщика с определенным префиксом и локалью. Он импортирует модули для работы с путями, JSON, регулярными выражениями, веб-драйверами, поставщиками, продуктами, категориями, форматированием строк, нормализацией строк, а также для взаимодействия с PrestaShop.
