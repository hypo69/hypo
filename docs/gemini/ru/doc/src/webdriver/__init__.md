# Модуль hypotez/src/webdriver/__init__.py

## Обзор

Этот модуль предоставляет интерфейс для работы с веб-драйверами.  Он содержит константы и импорты для выбора и инициализации различных драйверов (Chrome, Firefox, Edge и др.).

## Константы

### `MODE`

**Описание**:  Переменная, определяющая режим работы. В данном случае, принимает значение 'dev'.

**Значение**: `'dev'`
