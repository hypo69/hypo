Gearbest Header Module
=======================

.. module:: hypotez.src.suppliers.gearbest.header
    :platform: Windows, Unix
    :synopsis: This module contains functions for setting the project root and loading project settings.

Module Description
------------------

This module provides a function to find the root directory of the project. It also loads project settings from a JSON file and handles potential errors during loading.  It defines several module-level variables, storing project name, version, documentation, details, author, copyright, and a coffee link, fetched from the settings file.
