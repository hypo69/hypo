# Документация модуля `whatsapp_sender.py`

## Обзор

Модуль `whatsapp_sender.py` предназначен для автоматической отправки сообщений в WhatsApp с использованием библиотеки `pywhatkit` и эмуляции нажатий клавиш с помощью `pyautogui` и `pynput`. Он предоставляет функцию `send_whatsapp_message` для отправки сообщений по указанному номеру телефона в заданное время. Также в модуле реализован пример использования функции для отправки случайных сообщений из предопределенного списка в заданном временном интервале.

## Подробней

Этот модуль автоматизирует процесс отправки сообщений в WhatsApp, что может быть полезно для различных целей, таких как автоматические уведомления, поздравления или просто для развлечения. Модуль использует библиотеки `pywhatkit`, `pyautogui` и `pynput` для управления браузером и эмуляции действий пользователя.

## Функции

### `send_whatsapp_message`

```python
def send_whatsapp_message(phone_number, message, hour, minutes):
    """Sends a WhatsApp message using pywhatkit and pyautogui."""
```

**Описание**: Отправляет сообщение WhatsApp с использованием `pywhatkit` и `pyautogui`.

**Параметры**:

-   `phone_number` (str): Номер телефона получателя с кодом страны.
-   `message` (str): Текст сообщения для отправки.
-   `hour` (int): Час отправки сообщения (0-23).
-   `minutes` (int): Минута отправки сообщения (0-59).

**Возвращает**:

-   `None`: Функция ничего не возвращает.

**Вызывает исключения**:

-   `Exception`: В случае возникновения ошибки при отправке сообщения.

**Примеры**:

```python

```
