# Шаблон чата Gemini

## Обзор

Этот шаблон (`chat.html`) представляет собой HTML-страницу для веб-чата, использующего модель Gemini.  Он предоставляет интерфейс для ввода пользователя и отображения ответов модели.  Страница использует Bootstrap для стилизации и jQuery для обработки событий.

## Структура страницы

Страница состоит из следующих основных элементов:

* **Заголовок**: Заголовок "Kazarinov AI Chat".
* **Область чата**: Дивизион с классом `chat-box` для отображения истории диалога. Он имеет ограниченную высоту и использует `overflow-y: scroll` для прокрутки при необходимости.
* **Форма ввода**: Форма для ввода пользователем сообщений. Она содержит поле ввода (`user-input`) и кнопку "Отправить".
* **Подключение Bootstrap и jQuery**: Страница импортирует стили Bootstrap и библиотеку jQuery.
* **Дополнительные стили**: Подключает файл `styles.css` для кастомизации внешнего вида страницы.

## Обработка ввода пользователя

Обработчик событий `submit` для формы `chat-form` обрабатывает отправку сообщений пользователем.

### Функция обработки отправки

```javascript
$(document).ready(function() {
    $('#chat-form').submit(function(event) {
        event.preventDefault();

        let userInput = $('#user-input').val();
        $('#chat-log').append('<p><strong>Вы:</strong> ' + userInput + '</p>');

        $.ajax({
            url: '/ask',
            method: 'POST',
            data: { user_input: userInput },
            success: function(response) {
                $('#chat-log').append('<p><strong>AI:</strong> ' + response.response + '</p>');
                $('#user-input').val('');  // Очистка поля ввода
            }
        });
    });
});
```

**Описание**: Эта функция обрабатывает отправку сообщений. Она получает ввод пользователя, добавляет его в историю чата и отправляет запрос на сервер с помощью AJAX.

**Параметры**:

* `event`: Объект события.
* `userInput`: Текст сообщения пользователя.


**Возвращает**: Ничего (void).


## Взаимодействие с сервером

JavaScript код использует AJAX для отправки запроса на сервер `/ask` с данными пользователя. Сервер обрабатывает запрос и возвращает ответ, который отображается в чате.
