# Шаблон чата Gemini

## Обзор

Данный шаблон HTML (`chat.html`) представляет собой интерфейс для чата с моделью Gemini. Он использует Bootstrap для стилизации и AJAX для асинхронного обмена данными с сервером. Шаблон позволяет пользователю вводить сообщения и отображает ответы модели.

## Структура HTML

Шаблон содержит следующие основные элементы:

- **Заголовок:**  `<h1>Kazarinov AI Chat</h1>` отображает название чата.
- **Область чата:** `div.chat-box` содержит историю сообщений. Используется `overflow-y: scroll;` для прокрутки, когда количество сообщений превышает высоту области.
- **Поле ввода:**  `input#user-input` для ввода пользователем сообщений.
- **Кнопка отправки:** `button[type="submit"]` для отправки сообщения.
- **Форма:** `form#chat-form` собирает данные для отправки на сервер.
- **Стили Bootstrap:** Подключение стилей Bootstrap для адаптивного оформления.
- **Дополнительные стили:** Подключение внешних CSS-стилей из файла `styles.css`.


## JavaScript-логика

JavaScript-код в блоке `<script>` обрабатывает отправку сообщений и обновление чата.

### Обработка отправки сообщения

Функция `$(document).ready()` выполняется после полной загрузки страницы. Функция `$(\'#chat-form\').submit()` реагирует на отправку формы, предотвращая стандартное действие отправки страницы.

- Получает введенное пользователем сообщение из поля `#user-input`.
- Добавляет сообщение пользователя в `#chat-log` с разметкой `<strong>Вы:</strong>`.
- Отправляет AJAX-запрос на сервер с `POST`-методом к `/ask` с данными из поля ввода.

### Обработка ответа сервера

- Функция `success` в AJAX-запросе обрабатывает ответ сервера.
- Добавляет ответ модели AI в `#chat-log` с разметкой `<strong>AI:</strong>`.
- Очищает поле ввода `#user-input` после успешной отправки и получения ответа.

Приложение взаимодействует с сервером через AJAX.

- **Метод:** `POST`.
- **Адрес:** `/ask`.
- **Данные:** Содержит `user_input`.
