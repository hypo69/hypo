# Файл background.js

## Обзор

Этот файл содержит скрипт, который запускается в фоновом режиме расширения браузера. Он реагирует на клики по значку расширения и выполняет JavaScript код в текущей вкладке.

## Функции

### `browser.browserAction.onClicked.addListener`

**Описание**: Эта функция служит обработчиком для события клика по значку расширения. При клике на значок расширения она выполняет функцию, переданную в качестве аргумента.

**Параметры**:

- `(tab)` (object): Данные о вкладке, по которой был нажат значок.

**Возвращает**:

- `None`: Функция не возвращает явных значений.

**Вызывает исключения**:

- Не указаны.


### `browser.scripting.executeScript`

**Описание**:  Эта функция выполняет JavaScript код в текущей вкладке.

**Параметры**:

- `{ target: { tabId: tab.id }, files: ["contentScript.js"] }` (object): Объект с настройками выполнения кода.
    - `target.tabId` (number): Идентификатор вкладки, в которой нужно выполнить код.
    - `files` (array): Массив путей к файлам JavaScript, которые нужно выполнить.


**Возвращает**:

- `None`: Функция не возвращает явных значений.

**Вызывает исключения**:

- Не указаны.
