## Примеры использования `ExecuteLocator`

Примеры демонстрируют использование класса `ExecuteLocator` для поиска и взаимодействия с веб-элементами, включая обработку различных сценариев.

## Простой пример

Этот пример демонстрирует базовое использование `ExecuteLocator` для получения значения элемента по XPath.

### `execute_locator`

**Описание**: Выполняет локатор для получения значения элемента по указанным параметрам.

**Параметры**:
- `locator (dict)`: Словарь, содержащий параметры локатора.  Требуется, чтобы соответствовать схеме:
   - `by` (str): Способ локализации (XPATH, CSS, ID и т.д.).
   - `selector` (str):  Селектор для поиска элемента.
   - `attribute` (str, optional): Атрибут для получения значения (например, `textContent`, `value`).
   - `event` (str, optional):  Событие для выполнения (например, `click()`, `send_keys()`).
   - `if_list` (str, optional): "first" или "all". "first" - возвращает только первый найденный элемент, "all" - список всех найденных.
   - `use_mouse` (bool): Указывает, использовать ли `ActionChains` для имитации нажатия мыши.
   - `mandatory` (bool): Определяет, является ли локатор обязательным.
   - `locator_description` (str): Описание локатора.

- `continue_on_error (bool, optional)`: Продолжать выполнение при ошибке. По умолчанию `False`.


**Возвращает**:
- `str | list | None`: Значение атрибута, если локатор найден, `None` - если не найден. Возвращает список, если `if_list` = "all".

**Вызывает исключения**:
- `ExecuteLocatorException`: Если локатор не найден или произошла ошибка во время выполнения.



**Пример использования**:
```python
simple_locator = {
    "by": "XPATH",
    "selector": "//h1",
    "attribute": "textContent",
    "event": None,
    "if_list": "first",
    "use_mouse": False,
    "mandatory": True,
    "locator_description": "Получение заголовка страницы"
}

result = locator.execute_locator(simple_locator)
print(f"Результат выполнения простого локатора: {result}")
```
