# Модуль jquery-3.5.1.slim.min.js

## Обзор

Данный файл содержит сжатый и минимизированный код библиотеки jQuery версии 3.5.1.  Он предоставляет обширный набор функций для работы с DOM, обработкой событий, анимацией и другими аспектами JavaScript-разработки на веб-страницах.  Этот файл не содержит явных классов или функций,  но предоставляет функции, доступные через глобальный объект `$` или `jQuery`.

## Функции (внутренние)

Этот раздел содержит список функций, имеющихся в jQuery.  Из-за большого размера кода, подробные описания параметров, возвращаемых значений и исключений невозможно предоставить.

Примечание:  Описания в данном случае являются весьма общими, так как детальная информация о каждой функции из сжатого кода получить невозможно.


###  `jQuery.extend`

**Описание**: Расширяет объект.  Принимает произвольное количество аргументов, которые представляют собой объекты или функции.  Возвращает измененный объект.


### `jQuery.each`

**Описание**: Циклический метод, перебирающий элементы массива или объекта.


### `jQuery.map`

**Описание**: Преобразует один массив в другой.


###  `jQuery.grep`

**Описание**: Фильтрует массив на основе переданного условия.


### `jQuery.merge`

**Описание**: Объединяет два массива.


### `jQuery.isArray`

**Описание**: Проверяет, является ли переданный объект массивом.


### `jQuery.isPlainObject`

**Описание**: Проверяет, является ли переданный объект простым объектом (не массивом, не функцией и т.д.).


### `jQuery.isEmptyObject`

**Описание**: Проверяет, пуст ли переданный объект.


### `jQuery.globalEval`

**Описание**: Выполняет код JavaScript в контексте глобальной области видимости.


### `jQuery.error`

**Описание**:  Выбрасывает исключение с указанным сообщением.


### `jQuery.noop`

**Описание**: Пустая функция, не выполняющая никаких действий.


###  `jQuery.support`

**Описание**:  Объект, содержащий данные о поддержке различных функций браузером.


### `jQuery.each`

**Описание**: Циклический метод, перебирающий элементы массива или объекта.


### `jQuery.Deferred`

**Описание**:  Класс, представляющий собой объект для отслеживания и выполнения асинхронных операций.


### `jQuery.when`

**Описание**:  Ожидает завершения нескольких асинхронных операций.


### `jQuery.parseHTML`

**Описание**: Парсит строку HTML и возвращает массив узлов DOM.


### `jQuery.cleanData`

**Описание**: Очищает данные узлов DOM.



### `jQuery.clone`

**Описание**: Создает глубокую копию узла DOM.


### `jQuery.htmlPrefilter`

**Описание**: Фильтрует строку HTML перед парсингом.



### `jQuery.queue`

**Описание**: Управление очередью задач для объекта.


### `jQuery.dequeue`

**Описание**: Выполняет следующую задачу из очереди.


### `jQuery.cssHooks`

**Описание**: Набор хуков для работы со стилями.


### `jQuery.cssNumber`

**Описание**: Список свойств CSS, которые должны быть числами.



### `jQuery.cssProps`

**Описание**: Маппинг между свойствами CSS и их соответствующими именами.


### `jQuery.style`

**Описание**: Устанавливает или возвращает стиль элемента.


### `jQuery.css`

**Описание**:  Возвращает значение свойства CSS элемента.




## Методы jQuery (частично)


### `jQuery.fn.each`

**Описание**: Перебирает элементы в коллекции jQuery.


### `jQuery.fn.map`

**Описание**: Преобразует коллекцию jQuery в новую коллекцию.


### `jQuery.fn.get`

**Описание**: Возвращает элементы коллекции jQuery как массив DOM-узлов.


### `jQuery.fn.find`

**Описание**: Находит элементы внутри выбранных элементов.


### `jQuery.fn.filter`

**Описание**: Фильтрует элементы коллекции jQuery.


### `jQuery.fn.not`

**Описание**:  Удаляет элементы из коллекции jQuery, удовлетворяющие условию.

### `jQuery.fn.add`

**Описание**: Объединяет коллекции jQuery.


### `jQuery.fn.addClass`

**Описание**: Добавляет классы CSS к элементам.


### `jQuery.fn.removeClass`

**Описание**: Удаляет классы CSS у элементов.


### `jQuery.fn.toggleClass`

**Описание**: Добавляет или удаляет класс CSS, в зависимости от переданного значения.



## Использование

Для использования методов из данного файла необходимо загрузить его в проект.  Обратитесь к документации jQuery для более подробного ознакомления с доступными методами и их детальным описанием.  Подробности реализации методов и  параметров отсутствуют из-за минимизации.
