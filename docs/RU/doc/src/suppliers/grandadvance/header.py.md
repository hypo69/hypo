# Модуль `header`

## Обзор

Модуль `header.py` предназначен для определения основных настроек и переменных проекта, таких как корневой каталог, имя проекта, версия, автор и др. Он также загружает настройки из файла `settings.json` и документацию из `README.md`.

## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная с директории текущего файла, и останавливается на первом каталоге, содержащем один из указанных маркерных файлов.

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или каталогов, которые идентифицируют корень проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневому каталогу, если он найден, иначе директория, где расположен скрипт.

## Переменные

### `MODE`

**Описание**: Режим работы, по умолчанию `dev`.

### `__root__`

**Описание**: Путь к корневой директории проекта.

### `settings`

**Описание**: Словарь настроек, загруженный из файла `settings.json`. Может быть `None`, если файл не найден или при ошибке декодирования JSON.

### `doc_str`

**Описание**: Строка документации, загруженная из файла `README.md`. Может быть `None`, если файл не найден или при ошибке декодирования.

### `__project_name__`

**Описание**: Имя проекта. По умолчанию `'hypotez'`, или значение из `settings.json`, если он доступен.

### `__version__`

**Описание**: Версия проекта. По умолчанию пустая строка, или значение из `settings.json`, если он доступен.

### `__doc__`

**Описание**: Строка документации проекта, загруженная из файла `README.md`.

### `__details__`

**Описание**: Детальная информация о проекте. По умолчанию пустая строка.

### `__author__`

**Описание**: Автор проекта. По умолчанию пустая строка, или значение из `settings.json`, если он доступен.

### `__copyright__`

**Описание**: Информация об авторских правах. По умолчанию пустая строка, или значение из `settings.json`, если он доступен.

### `__cofee__`

**Описание**: Строка с информацией о поддержке разработчика. По умолчанию "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69", или значение из `settings.json`, если он доступен.
