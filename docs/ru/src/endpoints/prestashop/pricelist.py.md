# Модуль для работы со списком цен PrestaShop

## Обзор

Модуль `pricelist.py` предназначен для работы с API PrestaShop, в частности, для запроса и обновления списка цен товаров. Он содержит класс `PriceListRequester`, который наследуется от базового класса `PrestaShop` и предоставляет методы для получения цен товаров, обновления источника данных и изменения цен отдельных товаров.

## Подробней

Этот модуль позволяет взаимодействовать с API PrestaShop для получения актуальной информации о ценах товаров. Он может быть использован для автоматического обновления цен в системе, мониторинга цен конкурентов или других задач, связанных с управлением ценами.

## Классы

### `PriceListRequester`

**Описание**: Класс для запроса списка цен в PrestaShop. Он наследуется от класса `PrestaShop` и использует его методы для взаимодействия с API.

**Наследует**:

- `PrestaShop`: Базовый класс для работы с API PrestaShop.

**Аттрибуты**:

- Нет явных атрибутов, кроме тех, что наследуются от `PrestaShop`.

**Методы**:

- `__init__`: Инициализирует объект класса `PriceListRequester`.
- `request_prices`: Запрашивает список цен для указанных товаров.
- `update_source`: Обновляет источник данных для запроса цен.
- `modify_product_price`: Модифицирует цену указанного товара.
