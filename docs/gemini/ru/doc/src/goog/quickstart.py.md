# Модуль `quickstart`

## Обзор

Модуль `quickstart` предназначен для демонстрации базового использования Apps Script API. Он позволяет создавать новый проект скрипта, загружать файлы в этот проект и выводить URL скрипта для пользователя.

## Подробней

Этот модуль предоставляет пример взаимодействия с Google Apps Script API с использованием Python. Он автоматизирует процесс создания и обновления скриптов Google Apps Script, что может быть полезно для автоматизации задач, связанных с Google Workspace.

## Функции

### `main`

```python
def main():
    """Calls the Apps Script API."""
```

**Описание**:
Выполняет основные шаги взаимодействия с Apps Script API: аутентификацию, создание проекта, загрузку файлов и вывод URL скрипта.

**Как работает функция**:

1.  **Аутентификация**:
    *   Проверяет наличие файла `token.json` с сохраненными учетными данными.
    *   Если файл существует, пытается загрузить учетные данные из него.
    *   Если учетные данные отсутствуют или недействительны, запускает процесс аутентификации через `InstalledAppFlow`.
    *   Сохраняет полученные учетные данные в файл `token.json` для последующего использования.
2.  **Создание проекта**:
    *   Инициализирует сервис Apps Script API с использованием полученных учетных данных.
    *   Создает новый проект скрипта с названием "My Script".
3.  **Загрузка файлов**:
    *   Загружает два файла в проект:
        *   `hello` (SERVER\_JS) с кодом `SAMPLE_CODE`.
        *   `appsscript` (JSON) с манифестом `SAMPLE_MANIFEST`.
    *   Обновляет содержимое проекта скрипта с использованием загруженных файлов.
4.  **Вывод URL**:
    *   Выводит URL созданного скрипта.
5.  **Обработка ошибок**:
    *   Обрабатывает возможные ошибки API (`errors.HttpError`) и выводит их содержимое.

**Примеры**:

```python
if __name__ == '__main__':
    main()
```

## Переменные

### `SCOPES`

**Описание**:
Список областей доступа (scopes), необходимых для работы с Apps Script API.

### `SAMPLE_CODE`

**Описание**:
Пример кода JavaScript для загрузки в скрипт.

### `SAMPLE_MANIFEST`

**Описание**:
Пример манифеста (JSON) для загрузки в скрипт.
