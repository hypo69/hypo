# Модуль hypotez/src/suppliers/header.py

## Обзор

Данный модуль содержит функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и получения документации из файла `README.MD`. Он также инициализирует переменные, хранящие информацию о проекте (имя, версия, описание, автор, копирайт и др.).

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла и двигаясь вверх по дереву директорий, пока не найдет директорию, содержащую один из указанных файлов-маркеров (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж, содержащий имена файлов или директорий, по которым определяется корневая директория проекта. По умолчанию включает ('pyproject.toml', 'requirements.txt', '.git').

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращает директорию, в которой находится скрипт.

**Вызывает исключения**:

- Нет.

**Возвращает**:

**Вызывает исключения**:

## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root`. Тип `Path`.


### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженные из файла `settings.json`. Тип `dict` или `None`, если файл не найден или некорректно отформатирован.


### `doc_str`

**Описание**: Строка, содержащая содержимое файла `README.MD`, если он существует. Тип `str` или `None`, если файл не найден.


### `__project_name__`

**Описание**: Имя проекта. Строка. Значение взято из настроек `settings` или установлено по умолчанию ('hypotez').


### `__version__`

**Описание**: Версия проекта. Строка. Значение взято из настроек `settings` или установлено по умолчанию ("").


### `__doc__`

**Описание**: Документация проекта (содержимое `README.MD`). Строка. Значение взято из `doc_str` или установлено по умолчанию ("").


### `__details__`

**Описание**: Дополнительные детали о проекте. Строка (пустая по умолчанию).


### `__author__`

**Описание**: Автор проекта. Строка. Значение взято из настроек `settings` или установлено по умолчанию ("").


### `__copyright__`

**Описание**: Авторские права на проект. Строка. Значение взято из настроек `settings` или установлено по умолчанию ("").


### `__cofee__`

**Описание**: Ссылка на поддержку разработчика. Строка. Значение взято из настроек `settings` или установлено по умолчанию (ссылка на Boosty).

## Обработка исключений

Обработка возможных ошибок (FileNotFoundError, json.JSONDecodeError) при чтении файлов `settings.json` и `README.MD`.
