# Модуль hypotez/src/suppliers/morlevi/header.py

## Обзор

Этот модуль содержит функции для определения корневого каталога проекта, загрузки настроек из файла `settings.json` и документации из файла `README.MD`. Он также предоставляет переменные, содержащие информацию о проекте, такие как имя, версия, документация и прочие детали.

## Функции

### `set_project_root`

**Описание**: Определяет корневой каталог проекта, начиная с текущего файла и просматривая родительские каталоги до тех пор, пока не найдет директорию, содержащую один из файлов/каталогов, указанных в аргументе `marker_files`.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или каталогов, используемых для определения корневого каталога. По умолчанию включает `pyproject.toml`, `requirements.txt` и `.git`.

**Возвращает**:

- `Path`: Путь к корневому каталогу проекта. Если корневой каталог не найден, возвращает директорию, где находится текущий файл.

**Вызывает исключения**:

- Нет.


## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта.  Инициализируется вызовом функции `set_project_root()`.

**Тип**: `Path`

### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json` в корневой директории проекта.

**Тип**: `dict` | `None`

### `doc_str`

**Описание**: Строка с текстом документации из файла `README.MD` в корневой директории проекта.

**Тип**: `str` | `None`

### `__project_name__`

**Описание**: Имя проекта. Получается из словаря `settings`, если он существует, иначе используется значение по умолчанию `'hypotez'`.

**Тип**: `str`

### `__version__`

**Описание**: Версия проекта. Получается из словаря `settings`, если он существует, иначе используется значение по умолчанию `''`.

**Тип**: `str`

### `__doc__`

**Описание**: Документация проекта. Получается из переменной `doc_str`, если она существует, иначе используется значение по умолчанию `''`.

**Тип**: `str`

### `__details__`

**Описание**: Дополнительные детали проекта. По умолчанию пустая строка.

**Тип**: `str`

### `__author__`

**Описание**: Автор проекта. Получается из словаря `settings`, если он существует, иначе используется значение по умолчанию `''`.

**Тип**: `str`

### `__copyright__`

**Описание**: Авторские права проекта. Получается из словаря `settings`, если он существует, иначе используется значение по умолчанию `''`.

**Тип**: `str`

### `__cofee__`

**Описание**: Ссылка на поддержку разработчика. Получается из словаря `settings`, если он существует, иначе используется значение по умолчанию.

**Тип**: `str`
