# Модуль для тестирования чат-завершения gpt4free

## Обзор

Модуль `test_chat_completion.py` предназначен для тестирования функциональности чат-завершения, предоставляемой библиотекой `gpt4free`. Он включает в себя примеры использования как синхронного, так и асинхронного способов создания чат-завершений с использованием `g4f`.

## Подробней

Этот модуль демонстрирует, как использовать библиотеку `g4f` для создания чат-завершений. Он содержит пример использования `g4f.ChatCompletion.create` для синхронного создания чат-завершения и `g4f.ChatCompletion.create_async` для асинхронного создания чат-завершения. Модуль позволяет проверить работоспособность библиотеки `gpt4free` и ее интеграцию с различными моделями.

## Функции

### `run_async`

```python
async def run_async() -> None:
    """Асинхронно создает чат-завершение и выводит результат.

    Args:
        None

    Returns:
        None

    Как работает функция:
        1. Асинхронно вызывает `g4f.ChatCompletion.create_async` с моделью по умолчанию и сообщением "hello!".
        2. Выводит полученный ответ в консоль.

    Внутри функции происходят следующие действия и преобразования:
        Получение ответа от g4f.ChatCompletion.create_async
        |
        Вывод результата в консоль

    Примеры:
        >>> asyncio.run(run_async())
        create_async: Hello! How can I help you today?
    """
```

**Параметры**:
- Отсутствуют

**Возвращает**:
- `None`

**Как работает функция**:

1.  **Асинхронный вызов**: Функция асинхронно вызывает метод `g4f.ChatCompletion.create_async`, который создает чат-завершение. В качестве параметров передаются модель по умолчанию (`g4f.models.default`) и сообщение `"hello!"`.
2.  **Вывод результата**: Полученный ответ от чат-завершения выводится в консоль с префиксом `"create_async:"`.

**Примеры**:

```python
import asyncio

async def run_async():
    response = await g4f.ChatCompletion.create_async(
        model=g4f.models.default,
        messages=[{"role": "user", "content": "hello!"}],
    )
    print("create_async:", response)

asyncio.run(run_async())
