# Инструкция по установке ChromeDriver для тестирования

## Обзор

Данный документ описывает шаги по установке ChromeDriver, необходимого для работы тестов с веб-драйвером Selenium.


## Необходимые действия

1.  **Скачайте ChromeDriver:** Перейдите на сайт [https://chromedriver.chromium.org/downloads](https://chromedriver.chromium.org/downloads) и скачайте соответствующую версию ChromeDriver для вашей версии Chrome.  Обратите внимание на совместимость с вашей версией Chrome.
2.  **Разместите ChromeDriver:** Скопируйте скачанный исполняемый файл ChromeDriver в каталог, удобный для вашего проекта.  Например, в папку `hypotez/src/webdriver`.
3.  **Добавьте путь к ChromeDriver в переменные среды:**  В вашей операционной системе добавьте путь к папке, где вы сохранили ChromeDriver, в переменные среды.  Это необходимо, чтобы Selenium мог найти ChromeDriver при запуске тестов.  В зависимости от операционной системы, шаги могут отличаться.

## Примеры конфигурации для различных операционных систем

### Windows

1.  Открыть "Изменение системных параметров": "Этот компьютер" -> "Свойства" -> "Дополнительные параметры системы" -> "Переменные среды".
2.  Создать переменную `PATH`, если ее нет, или отредактировать существующую.
3.  Добавить в переменную `PATH` путь к папке с ChromeDriver, например,  `C:\\path\\to\\chromedriver`.


### macOS

1.  Открыть "Системные настройки" -> "Пользователи и группы" -> "Пользователи" -> [ваш пользователь] -> "Настройки пользователя"
2.  Найти "Переменные среды"
3.  Добавить переменную `PATH` и указать путь к папке с ChromeDriver. Например `~/path/to/chromedriver`

### Linux

1.  Открыть терминал.
2.  Выполнить команду для добавления пути к папке с ChromeDriver в переменную `PATH` (например, для `bash`):
    ```bash
    export PATH="$PATH:/path/to/chromedriver"
    ```
    Замените `/path/to/chromedriver` на фактический путь.
3.  Для постоянного применения необходимо добавить эту строку в ваш файл `.bashrc`, `.bash_profile` или аналогичный, чтобы изменения вступили в силу после перезагрузки терминала.


## Проверка установки

После добавления пути к ChromeDriver в переменные среды, запустите тесты, использующие Selenium и WebDriver. Если установка прошла успешно, то ChromeDriver будет найден и тесты будут работать без ошибок.


## Важные замечания

*  Необходимо использовать версию ChromeDriver, соответствующую вашей версии Chrome.
*  Если у вас возникнут ошибки при запуске тестов, убедитесь, что ChromeDriver установлен правильно и путь к нему указан корректно.
*  Обратите внимание на то, как именно ваш проект использует Selenium и WebDriver.  Проверьте, что конфигурация соответствует инструкциям проекта.
