# Модуль hypotez/src/webdriver/firefox/__init__.py

## Обзор

Этот модуль предоставляет базовые функции для взаимодействия с браузером Firefox. Он импортирует необходимый класс из подмодуля `firefox.py`.

## Конфигурационные переменные

### `MODE`

**Описание**: Переменная, определяющая режим работы.

**Значение**:  `'dev'` (по умолчанию).  Определяет режим разработки.


## Импорты

### `from .firefox import Firefox`

**Описание**: Импортирует класс `Firefox` из файла `firefox.py` в текущем пакете.
