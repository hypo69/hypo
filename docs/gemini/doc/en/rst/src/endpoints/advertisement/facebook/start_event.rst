Facebook Event Promotion
=========================

.. module:: hypotez.src.endpoints.advertisement.facebook.start_event
    :platform: Windows, Unix
    :synopsis: Sends events to Facebook groups.


Module Description
------------------

This module contains code for sending events to Facebook groups.  It utilizes the FacebookPromoter class to manage the process, handling the loading of event data and posting events.  Error handling and logging are implemented for robustness. The script runs in a loop, posting events at intervals.
