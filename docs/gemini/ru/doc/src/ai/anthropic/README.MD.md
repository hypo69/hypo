# Claude Anthropic Client

## Обзор

Этот модуль Python предоставляет простой интерфейс для взаимодействия с языковой моделью Claude от Anthropic. Он включает в себя основные функции для генерации текста, анализа тональности и перевода текста.

## Подробнее

Этот модуль позволяет легко интегрировать возможности Claude в ваши Python-приложения. Он предоставляет удобный способ генерации текста на основе заданных подсказок, анализа тональности текста и перевода текста между различными языками.

## Установка

Чтобы использовать этот модуль, вам необходимо установить библиотеку `anthropic`:

```bash
pip install anthropic
```

## Использование

### Инициализация

Сначала инициализируйте `ClaudeClient` с вашим API-ключом Anthropic:

```python
from claude_client import ClaudeClient

api_key = "your-api-key"
claude_client = ClaudeClient(api_key)
```

### Генерация текста

Сгенерируйте текст на основе данной подсказки:

```python
prompt = "Напиши короткий рассказ о роботе, учащемся любить."
generated_text = claude_client.generate_text(prompt)
print("Сгенерированный текст:", generated_text)
```

### Анализ тональности

Проанализируйте тональность данного текста:

```python
text_to_analyze = "Я сегодня очень счастлив!"
sentiment_analysis = claude_client.analyze_sentiment(text_to_analyze)
print("Анализ тональности:", sentiment_analysis)
```

### Перевод текста

Переведите текст с одного языка на другой:

```python
text_to_translate = "Hello, how are you?"
source_language = "en"
target_language = "es"
translated_text = claude_client.translate_text(text_to_translate, source_language, target_language)
print("Переведенный текст:", translated_text)
```

## Пример кода

Вот полный пример того, как использовать `ClaudeClient`:

```python
from claude_client import ClaudeClient

api_key = "your-api-key"
claude_client = ClaudeClient(api_key)

# Сгенерировать текст
prompt = "Напиши короткий рассказ о роботе, учащемся любить."
generated_text = claude_client.generate_text(prompt)
print("Сгенерированный текст:", generated_text)

# Анализировать тональность
text_to_analyze = "Я сегодня очень счастлив!"
sentiment_analysis = claude_client.analyze_sentiment(text_to_analyze)
print("Анализ тональности:", sentiment_analysis)

# Перевести текст
text_to_translate = "Hello, how are you?"
source_language = "en"
target_language = "es"
translated_text = claude_client.translate_text(text_to_translate, source_language, target_language)
print("Переведенный текст:", translated_text)
```

## Методы

### `generate_text(prompt, max_tokens_to_sample=100)`

Генерирует текст на основе данной подсказки.

**Назначение**:
Этот метод позволяет генерировать текст на основе заданной подсказки (prompt). Он использует модель Claude от Anthropic для создания текста, который соответствует стилю и содержанию подсказки.

**Как работает функция**:
Функция принимает текстовую подсказку и использует API Claude для генерации текста. Она отправляет подсказку в модель Claude, которая анализирует её и генерирует соответствующий текст. Затем сгенерированный текст возвращается пользователю.

**Параметры**:
- `prompt` (str): Подсказка, на основе которой генерируется текст.
- `max_tokens_to_sample` (int): Максимальное количество токенов для генерации (по умолчанию 100).

**Возвращает**:
- `str`: Сгенерированный текст.

### `analyze_sentiment(text)`

Анализирует тональность данного текста.

**Назначение**:
Этот метод позволяет определить эмоциональную окраску текста, то есть является ли текст положительным, отрицательным или нейтральным.

**Как работает функция**:
Функция принимает текст и использует API Claude для анализа его тональности. Модель Claude анализирует текст и определяет его эмоциональную окраску. Результат анализа возвращается пользователю.

**Параметры**:
- `text` (str): Текст для анализа.

**Возвращает**:
- `str`: Результат анализа тональности.

### `translate_text(text, source_language, target_language)`

Переводит данный текст с исходного языка на целевой язык.

**Назначение**:
Этот метод позволяет переводить текст с одного языка на другой. Он использует модель Claude от Anthropic для выполнения перевода.

**Как работает функция**:
Функция принимает текст, код исходного языка и код целевого языка. Она отправляет эту информацию в модель Claude, которая выполняет перевод текста. Затем переведенный текст возвращается пользователю.

**Параметры**:
- `text` (str): Текст для перевода.
- `source_language` (str): Код исходного языка.
- `target_language` (str): Код целевого языка.

**Возвращает**:
- `str`: Переведенный текст.
