# Модуль pprint_example_list

## Обзор

Этот модуль содержит примеры использования функции `pprint` для вывода списка.

## Функции

### `pprint_list`

**Описание**: Функция выводит переданный список с помощью `pprint`.

**Параметры**:

- `data` (list): Список, который нужно вывести.

**Возвращает**:

- `None`: Функция не возвращает значения.

**Вызывает исключения**:

- `TypeError`: Если входной параметр `data` не является списком.
