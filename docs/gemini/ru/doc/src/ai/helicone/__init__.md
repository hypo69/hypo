# Модуль helicone

## Обзор

Этот модуль предоставляет конфигурацию для работы с сервисом Helicone.  В данном случае, он определяет константу `MODE`, которая указывает на режим работы (в данном случае 'dev').

## Константы

### `MODE`

**Описание**: Константа, определяющая режим работы приложения.

**Значение**: `'dev'`

**Примечание**:  В этом примере константа `MODE` имеет значение 'dev'.  В реальном использовании, этот параметр может использоваться для переключения между различными средами (например, 'prod', 'staging').
