# Модуль hypotez/src/suppliers/aliexpress/campaign

## Обзор

Этот модуль содержит инструменты для управления рекламными кампаниями на Aliexpress. Он предоставляет классы и функции для редактирования, обработки и генерации отчетов о кампаниях. Модуль также включает инструменты для работы с данными в Google Sheets (хотя этот функционал может быть неактивным).
