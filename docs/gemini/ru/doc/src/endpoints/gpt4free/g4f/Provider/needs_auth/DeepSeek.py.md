# Модуль DeepSeek

## Обзор

Модуль `DeepSeek` представляет собой класс для взаимодействия с API DeepSeek, сервисом для обработки и генерации текста. Он наследует функциональность от класса `OpenaiAPI` и предоставляет методы для аутентификации и выполнения запросов к API DeepSeek. Этот модуль предназначен для интеграции с платформой DeepSeek, предоставляя удобный интерфейс для работы с их моделями, включая поддержку потоковой передачи и истории сообщений.

## Подробнее

Модуль `DeepSeek` упрощает взаимодействие с API DeepSeek, предоставляя готовый класс с настроенными параметрами подключения и методами аутентификации. Расположение файла в проекте указывает на его роль как одного из провайдеров API, требующих аутентификации.

## Классы

### `DeepSeek`

**Описание**: Класс `DeepSeek` предназначен для взаимодействия с API DeepSeek. Он наследует базовую функциональность от класса `OpenaiAPI` и расширяет ее, чтобы обеспечить специфические параметры и методы для работы с DeepSeek.

**Наследует**:

- `OpenaiAPI`: Класс `DeepSeek` наследует методы и атрибуты от `OpenaiAPI`, такие как обработка API ключей, выполнение запросов к API и управление сессиями.

**Атрибуты**:

- `label` (str): Метка, идентифицирующая провайдера как "DeepSeek".
- `url` (str): URL платформы DeepSeek ("https://platform.deepseek.com").
- `login_url` (str): URL для получения API ключей DeepSeek ("https://platform.deepseek.com/api_keys").
- `working` (bool): Указывает, что провайдер в настоящее время работоспособен (`True`).
- `api_base` (str): Базовый URL API DeepSeek ("https://api.deepseek.com").
- `needs_auth` (bool): Указывает, требуется ли аутентификация для доступа к API (`True`).
- `supports_stream` (bool): Указывает, поддерживает ли API потоковую передачу данных (`True`).
- `supports_message_history` (bool): Указывает, поддерживает ли API историю сообщений (`True`).
- `default_model` (str): Модель, используемая по умолчанию ("deepseek-chat").
- `fallback_models` (List[str]): Список резервных моделей, используемых в случае недоступности основной модели.
