# Модуль: Programming Assistant

Этот модуль содержит класс `CodeAssistant`, который используется для взаимодействия с различными моделями ИИ, такими как Google Gemini и OpenAI, для обработки кода.

## Примеры использования

Пример использования класса `CodeAssistant`:

```python
assistant = CodeAssistant(role='code_checker', lang='ru', model=['gemini'])
assistant.process_files()
```

## Плаформы

Этот модуль совместим с платформами, поддерживающими Python.

## Краткое описание

Модуль предоставляет инструменты для обработки и анализа кода с помощью ИИ.


## Классы

### `CodeAssistant`

**Описание**: Класс `CodeAssistant` предназначен для взаимодействия с различными моделями ИИ, такими как Google Gemini, и предоставляет методы для анализа и генерации документации для кода.

**Атрибуты**:

- `role`: Роль помощника (например, 'code_checker').
- `lang`: Язык, который будет использовать помощник (например, 'ru').
- `model`: Список используемых моделей ИИ (например, ['gemini']).

**Методы**:

#### `process_files`

**Описание**: Метод для обработки файлов кода.

**Параметры**:

- `files` (list): Список файлов для обработки.
- `options` (dict, необязательно): Дополнительные параметры для настройки обработки. По умолчанию `None`.

**Возвращает**:

- `list`: Результат обработки в виде списка проанализированных данных. Возвращает `None`, если произошла ошибка.

**Вызывает исключения**:

- `FileNotFoundError`: Если файл не найден.
- `ValueError`: Если входные данные некорректны.

**Пример использования**:

```python
assistant = CodeAssistant(role='code_checker', lang='ru', model=['gemini'])
result = assistant.process_files(files=['file1.py', 'file2.py'], options={})
```
