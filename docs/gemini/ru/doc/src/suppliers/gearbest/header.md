# Модуль hypotez/src/suppliers/gearbest/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла settings.json. Он также предоставляет доступ к информации о проекте, такой как имя, версия, документация и прочее.

## Функции

### `set_project_root`

**Описание**: Определяет корневую директорию проекта, начиная с директории текущего файла.  Ищет директории, содержащие файлы-маркеры (pyproject.toml, requirements.txt, .git).

**Параметры**:

- `marker_files` (tuple): Кортеж с именами файлов или директорий, которые указывают на корневую директорию проекта. По умолчанию используется кортеж `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращает директорию, где находится текущий скрипт.

**Вызывает исключения**:

- Нет


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта.  Инициализируется вызовом функции `set_project_root`.

**Тип**: `Path`


### `settings`

**Описание**: Словарь настроек проекта, загруженный из файла `settings.json`.

**Тип**: `dict`

**Примечание**: `settings` может быть `None`, если файл `settings.json` не найден или содержит недопустимый JSON.


### `doc_str`

**Описание**: Строка с текстом документации, загруженная из файла `README.MD`.

**Тип**: `str`

**Примечание**: `doc_str` может быть `None`, если файл `README.MD` не найден или не существует.


### `__project_name__`

**Описание**: Имя проекта. Получается из настроек. По умолчанию "hypotez".

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта. Получается из настроек. По умолчанию пустая строка.

**Тип**: `str`


### `__doc__`

**Описание**: Документация проекта. Получается из файла README.MD. По умолчанию пустая строка.

**Тип**: `str`


### `__details__`

**Описание**: Дополнительные детали проекта. По умолчанию пустая строка.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта. Получается из настроек. По умолчанию пустая строка.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права проекта. Получается из настроек. По умолчанию пустая строка.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка на бонусную поддержку разработчиков (копилка для чашки кофе). Получается из настроек. По умолчанию ссылка.

**Тип**: `str`
