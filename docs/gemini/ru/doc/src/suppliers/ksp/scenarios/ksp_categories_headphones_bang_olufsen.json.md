## Обзор

Этот файл содержит конфигурационные данные в формате JSON, определяющие сценарии для парсинга и обработки данных о наушниках бренда Bang & Olufsen (B&O) с веб-сайта KSP. Каждый сценарий определяет параметры для конкретной категории наушников, включая URL-адрес, условия и соответствия категориям PrestaShop.

## Описание структуры JSON

Файл JSON состоит из одного корневого объекта, содержащего ключ `scenarios`, который, в свою очередь, является объектом. Ключи этого объекта — это названия сценариев (например, `"In-ear Bud"`, `"Overear Headphones"`). Каждый сценарий имеет набор параметров, определяющих его поведение.

## Раздел `scenarios`

Объект `scenarios` содержит конфигурации для различных сценариев обработки категорий наушников B&O.

### `In-ear Bud`

**Описание**: Конфигурация для парсинга и обработки данных о внутриканальных наушниках B&O.

**Параметры**:

-   `brand` (str): Название бренда наушников (`"B&O"`).
-   `url` (str): URL-адрес страницы с внутриканальными наушниками на сайте KSP (`"https://ksp.co.il/web/cat/242..1250..5030"`).
-   `checkbox` (bool): Флаг, указывающий на необходимость использования чекбокса (в данном случае `false`).
-   `active` (bool): Флаг, указывающий, активен ли сценарий (`true`).
-   `condition` (str): Состояние товара (`"new"`).
-   `presta_categories` (dict): Объект, определяющий соответствие категорий PrestaShop.
    -   `template` (dict): Шаблон соответствия категорий. Ключ – строковое представление бренда (`"b & o"`), значение - название категории в PrestaShop (`"In-ear Bud Headphones"`).

### `Overear Headphones`

**Описание**: Конфигурация для парсинга и обработки данных о накладных наушниках B&O.

**Параметры**:

-   `brand` (str): Название бренда наушников (`"B&O"`).
-   `url` (str): URL-адрес страницы с накладными наушниками на сайте KSP (`"https://ksp.co.il/web/cat/242..1250..5030"`).
-   `checkbox` (bool): Флаг, указывающий на необходимость использования чекбокса (в данном случае `false`).
-   `active` (bool): Флаг, указывающий, активен ли сценарий (`true`).
-   `condition` (str): Состояние товара (`"new"`).
-   `presta_categories` (dict): Объект, определяющий соответствие категорий PrestaShop.
    -   `template` (dict): Шаблон соответствия категорий. Ключ – строковое представление бренда (`"b & o"`), значение - название категории в PrestaShop (`"Overear Headphones"`).
