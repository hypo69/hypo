# Описание локаторов для страницы Deals AliExpress

## Обзор

Данный файл `deals.json` содержит JSON-объект с описанием локатора для поиска элементов на странице deals AliExpress. Он предназначен для использования в автоматизированных тестах или скрейпинге данных.

## Описание локаторов

### `product_links`

**Описание**: Локатор для нахождения ссылок на товары со страницы deals.

**Параметры**:

-   `attribute` (null): Атрибут, который не используется в данном локаторе.
-   `by` ("XPATH"): Метод поиска элемента на странице, в данном случае XPATH.
-   `selector` ("//a[contains( @class ,'productContainer')]"):  XPATH выражение для поиска ссылок на товары.
-   `if_list` ("first"):  Указание, что возвращать нужно первый элемент из списка, если их несколько.
-   `use_mouse` (false): Флаг, указывающий, что не требуется использование мыши.
-   `mandatory` (false): Флаг, указывающий, что наличие данного элемента не является обязательным.
-   `timeout` (0): Время ожидания элемента, 0 - означает не ожидать.
-   `timeout_for_event` ("presence_of_element_located"):  Событие, которое нужно ждать для определения, найден ли элемент. В данном случае наличие элемента в DOM.
-   `event` (null):  Уточняющее событие, в данном случае не требуется.
-    `locator_description` ("Товары со страницы deals"): Описание локатора.

**Возвращает**:

    -  Объект с параметрами для поиска ссылок на товары.
