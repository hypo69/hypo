# Модуль маршрутизации FastAPI

## Обзор

Модуль `routes.py` предназначен для манипулирования маршрутами в сервере FastAPI. Он содержит класс `Routes`, который предоставляет методы для обработки различных запросов, включая обработку сообщений от Telegram-ботов.

## Подробней

Этот модуль обеспечивает связующее звено между входящими HTTP-запросами и обработчиками, которые выполняют необходимую логику приложения. В частности, он настраивает обработку сообщений, поступающих от Telegram-ботов, что позволяет приложению взаимодействовать с пользователями через Telegram.

## Классы

### `Routes`

**Описание**:
Класс `Routes` предназначен для управления маршрутами в FastAPI-приложении.

**Принцип работы**:
Класс инициализирует обработчики для различных типов запросов. В текущей версии реализована поддержка обработки сообщений от Telegram-ботов.

**Методы**:

- `tegram_message_handler()`: Метод для обработки сообщений, поступающих от Telegram-ботов.
