# Задача No 442
## Гиперболические игральные кости

## Условия

Рассмотрим две идеальные игральные кости, с гранями, помеченными числами от 1 до *n*. Пусть *S*<sub>*n*</sub> будет количеством способов, которыми можно расположить *n* таких игральных костей, чтобы сумма чисел на верхней грани была ровно *n* (мы считаем, что кости различимы).

Например, для *n* = 3 имеем:

*  1+1+1
*  1+2
*  2+1
*  3
и поэтому *S*<sub>3</sub> = 4. Более того, *S*<sub>10</sub> = 614.

Теперь рассмотрим гиперболические игральные кости. Для гиперболических игральных костей с *n* гранями грани помечены числами 1, 2, 2<sup>2</sup>, ..., 2<sup>(*n*−1)</sup>.

Пусть *H*(*n*) будет количеством способов расположить *n* гиперболических игральных костей так, чтобы сумма чисел на верхней грани была равна *n*.

Например, для *n* = 3 имеем:

*  1+1+1
и поэтому *H*(3) = 1.
Далее, *H*(10) = 51.

Пусть *T*(*n*) будет суммой *H*(*k*) для *k* от 1 до *n*.

Вам дано *T*(10<sup>6</sup>) = 174597244.

Найдите *T*(10<sup>9</sup>) mod 7<sup>10</sup>.
