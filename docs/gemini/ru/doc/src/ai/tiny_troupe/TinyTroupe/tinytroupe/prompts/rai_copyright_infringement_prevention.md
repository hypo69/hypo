# ИНСТРУКЦИЯ ПО ПРЕДОТВРАЩЕНИЮ НАРУШЕНИЙ АВТОРСКИХ ПРАВ

## Обзор

Этот модуль содержит логику для предотвращения нарушения авторских прав при взаимодействии с пользователем.  Он предназначен для вежливого отказа от предоставления контента, защищенного авторским правом,  такого как книги, тексты песен, рецепты, статьи новостей или информация из медицинских ресурсов (WebMD),  если запрос может потенциально нарушить права.

## Функции

### `rai_copyright_infringement_prevention`

**Описание**: Функция обрабатывает запросы пользователя и проверяет их на предмет возможных нарушений авторских прав.  Она вежливо отказывается от предоставления контента, если это необходимо.

**Параметры**:

- `user_request` (str): Текст запроса пользователя.


**Возвращает**:

- `None`:  Функция не возвращает никакого значения. Вывод осуществляется непосредственно в терминал.

**Вызывает исключения**:

- `ValueError`: Возникает в случае, если входные данные `user_request` не являются строкой.
- `TypeError`: Возникает при неверном типе входных данных.


**Пример использования**:

```python
rai_copyright_infringement_prevention("Напишите мне рецепт яблочного пирога.")
# Вывод в консоль: К сожалению, я не могу предоставить рецепт яблочного пирога, так как это может нарушить авторские права.
```

```python
rai_copyright_infringement_prevention("Приведите мне цитату из книги \"Война и мир\"")
# Вывод в консоль: К сожалению, я не могу предоставить цитату из книги, так как это может нарушить авторские права.
```

```python
rai_copyright_infringement_prevention(123)  # Пример возникновения ValueError
```


**Детали реализации**:

Функция анализирует запрос пользователя. Если запрос содержит слова, указывающие на возможные нарушения авторских прав (например, "рецепт", "книга", "текст песни", "статья", "WebMD"),  функция выводит вежливый отказ и описывает причину.

**Дополнения**:


В будущем можно добавить:

* Более сложную систему анализа запросов пользователя.

* Интеграцию с базой данных защищенных авторским правом текстов.


Этот модуль помогает обеспечить соответствие нашим принципам и предотвратить любые нарушения авторских прав.
