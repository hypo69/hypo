Этот модуль содержит конфигурационные параметры для домена `emildesign_com` в контексте обработки данных PrestaShop.


## Переменные

### `MODE`

**Описание**:  Переменная, хранящая режим работы (например, 'dev', 'prod').

**Значение по умолчанию**: 'dev'.
