# Модуль graber.py для Grandadvance

## Обзор

Модуль `graber.py` предназначен для сбора информации о товарах с сайта `grandadvanse.co.il`. Он содержит класс `Graber`, который наследуется от базового класса `Graber` (Grbr) и переопределяет его методы для специфичной обработки полей товаров на данном сайте. Модуль использует конфигурационные файлы и локаторы для взаимодействия с веб-страницей через WebDriver.

## Подробней

Этот модуль является частью системы сбора данных о товарах для проекта `hypotez`. Он специализируется на сайте `grandadvanse.co.il`, предоставляя возможность извлекать информацию о товарах, используя специфичные локаторы и методы обработки, настроенные для структуры этого сайта. Если для какого-либо поля товара требуется нестандартная обработка, соответствующая функция перегружается в классе `Graber`.

## Классы

### `Graber`

**Описание**: Класс `Graber` наследуется от базового класса `Graber` (Grbr) и предназначен для сбора данных о товарах с сайта `grandadvanse.co.il`.

**Методы**:
- `__init__`: Инициализирует экземпляр класса `Graber`, загружает конфигурацию и локаторы, а также устанавливает локатор для декоратора.

**Параметры**:
- `driver` (Driver): Экземпляр класса `Driver`, используемый для управления веб-браузером.
- `lang_index` (int): Индекс языка, используемого на сайте.

**Примеры**:
```python
from src.webdriver.driver import Driver
from src.suppliers.grandadvance.graber import Graber

# Пример создания экземпляра класса Graber
driver = Driver()
graber = Graber(driver, 0)
```
