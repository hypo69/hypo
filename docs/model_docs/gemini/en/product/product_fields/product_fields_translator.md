    :platform: Windows, Unix

Module Description
------------------

Этот модуль предоставляет функции для перевода полей товара на различные языки, используя схему языков клиентской базы данных.  Он обрабатывает поля товара, полученные от внешнего поставщика, и обновляет их идентификаторы языка в соответствии со схемой клиентской базы данных.
