# Задача No 879
## Скрученные битовые суммы

## Условия
Для заданного натурального числа $n$ определим $s(n)$ как сумму битов в двоичном представлении числа $n$. Например, $s(5) = s(101_2) = 2$ и $s(10) = s(1010_2) = 2$.

Также, для натурального числа $n$, определим $t(n)$ как число натуральных чисел $k \leq n$, таких что $s(k)$ является четным числом. Например, $t(1) = 0$, $t(2) = 1$ и $t(10) = 5$.

Наконец, определим последовательность $a_i$ следующим образом: $a_0 = 1$ и $a_i = t(a_{i-1} + i)$ для $i \geq 1$.
Первые несколько членов этой последовательности равны: $a_0 = 1$, $a_1 = t(2) = 1$, $a_2 = t(3) = 2$, $a_3 = t(5) = 2$, $a_4 = t(9) = 5$, $a_5 = t(14) = 7$, $a_6 = t(20) = 10$.

Пусть $S(n) = \sum_{i=0}^n a_i$. Найдите $S(10^{14}) \mod 10^9$.
