# Задача No 289
## Euler's Zigzag Path

## Условия

Рассмотрим бесконечную сетку точек с целочисленными координатами. Из точки (0,0) можно начать движение вправо в точку (1,0). Затем можно двигаться либо вверх, либо вниз. Повторяя движение вправо, а затем либо вверх, либо вниз, можно создать путь через точки сетки.

Назовем зигзагообразным путем путь, который начинается в (0,0), совершает n ходов вправо, и который не пересекает сам себя.

Кроме того, пусть a(n) будет количеством различных зигзагообразных путей из n ходов вправо.
Например, a(3) = 4, возможные пути:
- (0,0),(1,0),(2,1),(3,2)
- (0,0),(1,0),(2,1),(3,0)
- (0,0),(1,0),(2,-1),(3,0)
- (0,0),(1,0),(2,-1),(3,-2)

Последние две точки не должны быть одинаковыми.

Пусть z(n) будет количеством различных зигзагообразных путей из n ходов вправо, которые заканчиваются на оси x.
Например, z(3) = 2:
- (0,0),(1,0),(2,1),(3,0)
- (0,0),(1,0),(2,-1),(3,0)

Вы можете проверить, что a(10) = 1032 и z(10) = 512.

Найдите z(10<sup>10</sup>) mod 10<sup>9</sup>.
