# Модуль src.ai

## Обзор

Данный модуль предоставляет интерфейс к моделям искусственного интеллекта, включая Google Gemini и OpenAI.  Он позволяет работать с моделями через соответствующие API.

## Модули

### `gemini`

Этот модуль содержит класс для работы с моделями Google Gemini.

### `openai`

Этот модуль содержит класс для работы с моделями OpenAI.

## Классы

### `GoogleGenerativeAI`

**Описание**: Класс для взаимодействия с моделями Google Gemini.

**Методы**:

- `generate(prompt: str, max_tokens: int = 256, temperature: float = 0.7) -> str`: Генерирует текст на основе предоставленного запроса.
    - **Описание**: Генерирует текст, используя модель Google Gemini.
    - **Параметры**:
        - `prompt` (str): Запрос для генерации текста.
        - `max_tokens` (int, optional): Максимальное количество генерируемых токенов. По умолчанию 256.
        - `temperature` (float, optional): Температура генерации (вероятность выбора более необычных токенов). По умолчанию 0.7.
    - **Возвращает**:
        - `str`: Сгенерированный текст.
    - **Вызывает исключения**:
        - `APIError`: Возникает при ошибке взаимодействия с API Google Gemini.


### `OpenAIModel`

**Описание**: Класс для взаимодействия с моделями OpenAI.

**Методы**:

- `generate(prompt: str, max_tokens: int = 256, temperature: float = 0.7) -> str`: Генерирует текст на основе предоставленного запроса.
    - **Описание**: Генерирует текст, используя модель OpenAI.
    - **Параметры**:
        - `prompt` (str): Запрос для генерации текста.
        - `max_tokens` (int, optional): Максимальное количество генерируемых токенов. По умолчанию 256.
        - `temperature` (float, optional): Температура генерации (вероятность выбора более необычных токенов). По умолчанию 0.7.
    - **Возвращает**:
        - `str`: Сгенерированный текст.
    - **Вызывает исключения**:
        - `APIError`: Возникает при ошибке взаимодействия с API OpenAI.


## Переменные

### `MODE`

**Описание**: Переменная, определяющая режим работы (например, 'dev' или 'prod').


**Примечание**:  Данная документация предполагает наличие классов `GoogleGenerativeAI` и `OpenAIModel` в соответствующих модулях (`gemini.py` и `openai.py`).  В случае отсутствия этих классов, или их другой реализации,  соответствующая часть документации должна быть переписана.
