# Модуль `hypotez/src/suppliers/aliexpress/utils/locales.py`

## Обзор

Модуль `locales.py` предоставляет функции для загрузки и обработки данных о локализациях из файла JSON.  Он используется для получения информации о валютах, соответствующих различным языковым локализациям.


## Функции

### `get_locales`

**Описание**: Загружает данные о локализациях из файла JSON.

**Параметры**:
- `locales_path` (Path | str): Путь к файлу JSON, содержащему данные о локализациях.

**Возвращает**:
- `list[dict[str, str]] | None`: Список словарей, где каждый словарь содержит пары "язык" (ключ) - "валюта" (значение). Возвращает `None`, если файл не найден или данные некорректно отформатированы.

**Примеры**:
```python
>>> from src.suppliers.aliexpress.utils.locales import get_locales
>>> locales = get_locales(Path('/path/to/locales.json'))
>>> print(locales)
[{'EN': 'USD'}, {'HE': 'ILS'}, {'RU': 'ILS'}, {'EN': 'EUR'}, {'EN': 'GBR'}, {'RU': 'EUR'}]
```


## Постоянные переменные

### `locales`

**Описание**: Список словарей, содержащих данные о локализациях.
**Тип**: `list[dict[str, str]] | None`
**Значение**: Содержит данные, загруженные из файла `locales.json`.  `None`, если загрузка не удалась.


## Замечания

Этот модуль предполагает определенный формат данных в файле `locales.json`.  Он должен быть валидным JSON-объектом, содержащим поле `locales` со списком словарей.  Каждый словарь в списке должен содержать пары язык-валюта.


```python
# Пример структуры данных в файле locales.json
{
  "locales": [
    {"EN": "USD"},
    {"HE": "ILS"},
    {"RU": "ILS"},
    {"EN": "EUR"},
    {"EN": "GBR"},
    {"RU": "EUR"}
  ]
}
```
