# Модуль запуска GPT4Free

## Обзор

Этот модуль является точкой входа для запуска API GPT4Free. Он использует аргументы командной строки для настройки и запуска API.

## Подробнее

Модуль анализирует аргументы командной строки, предоставленные пользователем, и запускает API GPT4Free с этими аргументами. Если аргумент `gui` не указан, он устанавливается в `True` по умолчанию.

## Функции

### `get_api_parser`

```python
def get_api_parser() -> argparse.ArgumentParser:
    """
    Получает парсер аргументов командной строки.

    Returns:
        argparse.ArgumentParser: Парсер аргументов командной строки.
    """
```

**Как работает функция**:

1.  Функция `get_api_parser` вызывается для получения настроенного парсера аргументов командной строки.

**Примеры**:

```python
parser = get_api_parser()
```

### `run_api_args`

```python
def run_api_args(args: argparse.Namespace) -> None:
    """
    Запускает API с заданными аргументами.

    Args:
        args (argparse.Namespace): Аргументы командной строки.
    """
```

**Как работает функция**:

1.  Функция `run_api_args` вызывается с аргументами командной строки, чтобы запустить API GPT4Free.

**Примеры**:

```python
run_api_args(args)
```

## Основной блок кода

В основном блоке кода выполняются следующие действия:

1.  Получение парсера аргументов командной строки: `parser = get_api_parser()`.
2.  Разбор аргументов командной строки: `args = parser.parse_args()`.
3.  Если аргумент `gui` не указан, установка его в `True`:
    *   Проверка `if args.gui is None:`.
    *   Установка `args.gui = True`.
4.  Запуск API с заданными аргументами: `run_api_args(args)`.
