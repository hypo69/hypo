# Модуль `src.suppliers.hb.scenarios`

## Обзор

Модуль `src.suppliers.hb.scenarios` предоставляет набор инструментов для взаимодействия с поставщиком `hb.co.il`. Он включает функции для получения списка товаров в категории, списка категорий с сайта, сбора данных со страницы товара и авторизации. Модуль также содержит информацию о версии и детальное описание.

## Переменные

**Описание**: Версия модуля.

### `__doc__`

**Описание**: Документация модуля.

### `__details__`

**Описание**: Детальная информация о модуле.

## Импортированные модули

### `packaging.version`

**Описание**: Модуль `packaging.version` используется для работы с версиями. Импортируется `Version`.

### `categories`

**Описание**: Модуль `categories` содержит функции для работы с категориями товаров. Импортируются `get_list_products_in_category` и `get_list_categories_from_site`.

### `grabber`

**Описание**: Модуль `grabber` содержит функцию для сбора данных со страницы товара. Импортируется `grab_product_page`.

### `login`

**Описание**: Модуль `login` содержит функцию для авторизации. Импортируется `login`.

## Функции

### `get_list_products_in_category`

**Описание**: Функция для получения списка товаров в заданной категории.

### `get_list_categories_from_site`

**Описание**: Функция для получения списка категорий с сайта.

### `grab_product_page`

**Описание**: Функция для сбора данных со страницы товара.

### `login`

**Описание**: Функция для авторизации.
