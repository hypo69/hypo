# Задача No 548
## Идеальная цифровая инвариантность

## Условия
Натуральное число *n* называется идеальным цифровым инвариантом (ПЦИ), если *n* равно сумме своих цифр, возведенных в некоторую фиксированную степень *p*.

Давайте рассмотрим примеры:
- 1634 является ПЦИ, поскольку 1<sup>4</sup> + 6<sup>4</sup> + 3<sup>4</sup> + 4<sup>4</sup> = 1 + 1296 + 81 + 256 = 1634.
- 8208 является ПЦИ, поскольку 8<sup>4</sup> + 2<sup>4</sup> + 0<sup>4</sup> + 8<sup>4</sup> = 4096 + 16 + 0 + 4096 = 8208.
- 4150 является ПЦИ, поскольку 4<sup>5</sup> + 1<sup>5</sup> + 5<sup>5</sup> + 0<sup>5</sup> = 1024 + 1 + 3125 + 0 = 4150.

Пусть *S(p)* - сумма всех ПЦИ при заданной степени *p*. 
Так, например, *S(3)* = 153 + 370 + 371 + 407 = 1301.

Найдите сумму всех ПЦИ *n* таких, что *n* < 10<sup>16</sup>.
