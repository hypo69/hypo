# Модуль `options.js`

## Обзор

Этот модуль JavaScript отвечает за обработку и сохранение настроек расширения. Он предоставляет интерфейс для пользователя для изменения параметров, таких как атрибуты и CSS стили.  После сохранения настроек, модуль отправляет уведомление о необходимости переприложения стилей в всплывающем окне.

## Функции

### `isValidAttrName`

**Описание**: Проверяет, является ли имя атрибута допустимым.

**Параметры**:

- `name` (string): Имя атрибута.

**Возвращает**:

- boolean: `true`, если имя атрибута допустимо, `false` в противном случае.


**Вызывает исключения**:

- `Error`: В случае возникновения ошибки при добавлении атрибута к элементу.


### `isValidAttrNames`

**Описание**: Проверяет, являются ли все имена атрибутов в массиве допустимыми.

**Параметры**:

- `names` (object): Объект, содержащий имена атрибутов.


**Возвращает**:

- boolean: `true`, если все имена атрибутов допустимы, `false` в противном случае.


**Вызывает исключения**:

- Нет.

### `isValidStyleLength`

**Описание**: Проверяет, является ли значение длины стиля допустимым.

**Параметры**:

- `len` (string): Значение длины стиля.

**Возвращает**:

- boolean: `true`, если значение длины стиля допустимо, `false` в противном случае.


**Вызывает исключения**:

- Нет.

### `loadDefaultCss`

**Описание**: Загружает дефолтный CSS из файла.

**Возвращает**:

- Promise: Promise, содержащий ответ от запроса.  Содержит текст CSS.


**Вызывает исключения**:

- Нет.

### `extractBodyStyles`

**Описание**: Извлекает стили ширины и высоты из строки CSS.

**Параметры**:

- `css` (string): Строка CSS.

**Возвращает**:

- object: Объект с полями `width` и `height`, содержащими значения стилей. Возвращает пустой объект, если стили не найдены.


**Вызывает исключения**:

- Нет.


### `createPopupCss`

**Описание**: Создаёт строку CSS для тела всплывающего окна.

**Параметры**:

- `bodyStyles` (object): Объект, содержащий стили ширины и высоты.

**Возвращает**:

- string: Строка CSS для тела всплывающего окна.


**Вызывает исключения**:

- Нет.
