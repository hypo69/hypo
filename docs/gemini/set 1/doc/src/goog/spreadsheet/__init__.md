# Модуль `hypotez/src/goog/spreadsheet/__init__.py`

## Обзор

Данный модуль предоставляет интерфейс для работы с Google Spreadsheets. Он импортирует классы `Spreadsheet` и `ReachSpreadsheet`, обеспечивая доступ к основным функциям работы с электронными таблицами.


## Модульные переменные

### `MODE`

**Описание**: Переменная, определяющая режим работы модуля. В данном случае, значение `MODE` установлено в `'dev'`.  Это может указывать на режим разработки, тестирования или другой режим работы.

### `from .spreadsheet import SpreadSheet`

**Описание**: Импортирует класс `Spreadsheet` из модуля `spreadsheet`. Этот класс, вероятно, предоставляет базовые функции работы с Google Spreadsheets.

**Описание**: Импортирует класс `ReachSpreadsheet` из модуля `reach_spreadsheet`. Этот класс, вероятно, предоставляет расширенные функции работы с Google Spreadsheets, например, взаимодействие с определенными областями или API.
