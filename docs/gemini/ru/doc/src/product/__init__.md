# Модуль `hypotez/src/product`

## Обзор

Модуль `src.product` предоставляет инструменты для работы с продуктами. Он содержит классы для работы с продуктами и их характеристиками, а также функции для преобразования данных. Подробное описание классов и функций можно найти в соответствующих файлах (`product.py`, `product_fields.py`).

## Атрибуты

### `MODE`

**Описание**:  Переменная, определяющая режим работы модуля (например, `dev`, `prod`).  Текущее значение `'dev'`.

## Классы

### `Product`

**Описание**:  Класс, представляющий продукт. Подробное описание методов и атрибутов см. в файле `product.py`.

### `ProductFields`

**Описание**:  Класс, представляющий поля продукта. Подробное описание см. в файле `product_fields.py`.

## Функции

### `translate_presta_fields_dict`

**Описание**: Функция для перевода многоязычных полей `ProductFields`.

**Аргументы**:

* `data` (dict): Словарь данных, содержащий поля `ProductFields`.

**Возвращает**:

* `dict`: Словарь с переведенными полями.


## Внешние атрибуты

### `record`

**Описание**:  Словарь, содержащий поля продукта в плоском формате (без вложенности).

## Модули

### `product`

**Описание**: Модуль, содержащий класс `Product`. Подробное описание см. в файле `product.py`.

### `product_fields`

**Описание**: Модуль, содержащий класс `ProductFields` и функцию `translate_presta_fields_dict`. Подробное описание см. в файлах `product_fields.py` и `product_fields_translator.py`.

### `product_fields.product_fields`

**Описание**: Модуль, содержащий класс `ProductFields`. Подробное описание см. в файле `product_fields.py`.

### `product_fields.product_fields_translator`

**Описание**: Модуль, содержащий функцию `translate_presta_fields_dict`. Подробное описание см. в файле `product_fields_translator.py`.
