# Модуль `hypotez/src/suppliers/ksp/__init__.py`

## Обзор

Модуль `src.suppliers.ksp` предназначен для работы с поставщиком KSP. Он включает в себя функциональность для сбора данных (граббинга) с ресурсов KSP. 
Модуль поддерживает работу на платформах Windows и Unix.
