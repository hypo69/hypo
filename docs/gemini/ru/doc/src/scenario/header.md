# Модуль hypotez/src/scenario/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и чтения файла `README.MD`. Он также инициализирует переменные, представляющие имя проекта, версию, документацию, подробности, автора и авторские права.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная от текущей директории и проходя вверх по дереву директорий, пока не найдёт директорию содержащую указанные файлы.

**Параметры**:
- `marker_files` (tuple): Кортеж имён файлов или директорий, которые будут использоваться для определения корневой директории проекта.

**Возвращает**:
- `Path`: Объект `Path` представляющий путь к корневой директории проекта, если она найдена, иначе текущую директорию.

**Вызывает исключения**:
- Нет.


### `set_project_root`

**Описание**: Ищет корневую директорию проекта, начиная с текущего файла.

**Параметры**:
- `marker_files` (tuple, optional): Кортеж файлов или каталогов, которые используются для определения корневой директории. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории проекта.


## Переменные

### `__root__`

**Описание**:  Переменная, содержащая путь к корневой директории проекта, установленная функцией `set_project_root`.

**Тип**: `Path`


## Обработка файла `settings.json`

**Описание**: Загружает настройки из файла `settings.json` в переменную `settings`. Обрабатывает возможные исключения: `FileNotFoundError` (если файл не найден) и `json.JSONDecodeError` (если файл не валидный JSON).

## Обработка файла `README.MD`

**Описание**: Читает файл `README.MD` и сохраняет его содержимое в переменной `doc_str`. Обрабатывает возможные исключения: `FileNotFoundError` (если файл не найден) и `json.JSONDecodeError` (если файл повреждён).

## Переменные проекта

**Описание**:  Определяет и инициализирует переменные, представляющие имя проекта, версию, документацию, подробности, автора, авторские права и ссылку на пожертвования.


### `__project_name__`

**Описание**:  Имя проекта. Значение из настроек, по умолчанию `hypotez`.

**Тип**: `str`

### `__version__`

**Описание**: Версия проекта. Значение из настроек, по умолчанию пустая строка.

**Тип**: `str`

### `__doc__`

**Описание**: Документация проекта. Значение из файла `README.MD`, по умолчанию пустая строка.

**Тип**: `str`

### `__details__`

**Описание**: Подробности проекта. По умолчанию пустая строка.

**Тип**: `str`

### `__author__`

**Описание**: Автор проекта. Значение из настроек, по умолчанию пустая строка.

**Тип**: `str`

### `__copyright__`

**Описание**: Авторские права. Значение из настроек, по умолчанию пустая строка.

**Тип**: `str`

### `__cofee__`

**Описание**: Ссылка для пожертвования. Значение из настроек, по умолчанию ссылка на Boosty.

**Тип**: `str`
