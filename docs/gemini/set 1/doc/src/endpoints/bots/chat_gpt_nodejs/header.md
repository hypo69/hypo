# Модуль `hypotez/src/bots/openai_bots/header.py`

## Обзор

Этот модуль содержит вспомогательные функции для работы с проектом, включая определение корневой директории проекта и загрузку настроек из файла `settings.json`. Он также предоставляет доступ к переменным, содержащим информацию о проекте (имя, версия, описание и т.д.).

## Функции

### `set_project_root`

**Описание**:  Находит корневую директорию проекта, начиная с текущего файла и ища вверх по дереву директорий.  Останавливается на первой директории, содержащей указанные файлы маркеров.

**Параметры**:

- `marker_files` (tuple): Список файлов или директорий, по которым определяется корневая директория проекта. По умолчанию включает `'pyproject.toml'`, `'requirements.txt'` и `'.git'`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена, иначе директория, в которой расположен текущий скрипт.

**Вызывает исключения**:

- Нет



## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта.  Инициализируется функцией `set_project_root`.


### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json` в корневой директории проекта.

**Тип**: `dict`

**Значение по умолчанию**: `None`


### `doc_str`

**Описание**: Строка с текстом документации проекта, загруженная из файла `README.MD` в корневой директории проекта.

**Тип**: `str`

**Значение по умолчанию**: `None`


### `__project_name__`

**Описание**: Имя проекта, полученное из настроек (`settings.json`). Если ключ `project_name` не найден или `settings` не задано, по умолчанию используется "hypotez".

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта, полученная из настроек (`settings.json`). Если ключ `version` не найден или `settings` не задано, по умолчанию используется пустая строка.

**Тип**: `str`


### `__doc__`

**Описание**: Документация проекта, полученная из файла `README.MD`. Если файл не найден, используется пустая строка.

**Тип**: `str`

### `__details__`

**Описание**: Дополнительные детали проекта. По умолчанию пустая строка.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта, полученный из настроек (`settings.json`). Если ключ `author` не найден или `settings` не задано, по умолчанию используется пустая строка.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права проекта, полученные из настроек (`settings.json`). Если ключ `copyright` не найден или `settings` не задано, по умолчанию используется пустая строка.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка на страницу, где можно поддержать автора чашечкой кофе, полученная из настроек (`settings.json`). Если ключ `cofee` не найден или `settings` не задано, по умолчанию используется ссылка.

**Тип**: `str`

## Обработка исключений

Обработка `FileNotFoundError` и `json.JSONDecodeError` при чтении файлов `settings.json` и `README.MD` предотвращает аварийную остановку скрипта при отсутствии или некорректном формате этих файлов.
