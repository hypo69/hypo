# Модуль hypo69

## Обзор

Модуль `hypo69` содержит API-эндпоинты для разработчиков.  Включает в себя функциональность для работы с моделями чат-ботов (small_talk_bot),  тренировки кодовой модели (code_assistant) и раннего этапа разработки модуля парсинга диалогов (psychologist_bot).

## Модули

### `small_talk_bot`

**Описание**: Модуль реализует функциональность для взаимодействия с моделью чат-бота small_talk_bot.  Обеспечивает API для запуска диалогов, получения ответов и управления сессиями.

### `code_assistant`

**Описание**: Модуль предоставляет инструменты для обучения и использования кодовой модели.  Включает методы для загрузки данных, подготовки данных для обучения, запуска процесса обучения и использования обученной модели для генерации кода.

### `psychologist_bot`

**Описание**: Модуль находится на стадии раннего развития и содержит функции для парсинга и анализа диалогов.  На данном этапе предоставляет базовые инструменты для обработки текстовых данных.
