# Задача No 790
## Нечётные делители
## Условия
Определим функцию $d(n)$ как количество делителей числа $n$.

Например, $d(6) = 4$, так как делителями 6 являются 1, 2, 3 и 6.
Определим функцию $o(n)$ как количество нечётных делителей числа $n$.
Например, $o(6) = 2$, так как нечётными делителями 6 являются 1 и 3.
Далее, определим функцию $S(n)$ как сумму значений $o(k)$ для всех $1 \le k \le n$.
Например:
$S(1) = o(1) = 1$,
$S(2) = o(1) + o(2) = 1 + 1 = 2$,
$S(3) = o(1) + o(2) + o(3) = 1 + 1 + 2 = 4$,
$S(4) = o(1) + o(2) + o(3) + o(4) = 1 + 1 + 2 + 2 = 6$,
$S(5) = o(1) + o(2) + o(3) + o(4) + o(5) = 1 + 1 + 2 + 2 + 2 = 8$,
$S(10) = 18$

Пусть $F(k)$ будет $k$-м числом Фибоначчи (с $F(1) = 1$ и $F(2) = 1$).

Найдите $S(F(10^{18})) \mod 10^9$.
