# Модуль `hypo69`: Эндпоинты для разработчиков

## Обзор

Этот модуль содержит эндпоинты, разработанные специально для нужд разработчиков, включая инструменты для работы с ИИ.

## Оглавление

    - [`small_talk_bot`](#small_talk_bot)
    - [`code_assistant`](#code_assistant)
    - [`psychologist_bot`](#psychologist_bot)

## Эндпоинты

### `small_talk_bot`

**Описание**: Бот с чатом модели ИИ. Предназначен для ведения диалога с пользователем и демонстрации возможностей модели.

### `code_assistant`

**Описание**: Модуль для обучения модели ИИ коду текущего проекта. Помогает в интеграции модели с кодовой базой.

### `psychologist_bot`

**Описание**: Ранняя разработка модуля для парсинга диалогов. Предназначен для анализа и структурирования текстовых диалогов.
