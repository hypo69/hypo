# Модуль для модульного тестирования GPT4Free
=================================================

Модуль предназначен для запуска набора модульных тестов, проверяющих различные аспекты библиотеки `gpt4free`. Он импортирует необходимые модули и запускает тесты.

## Обзор

Модуль является точкой входа для запуска всех модульных тестов, связанных с библиотекой `gpt4free`. Он импортирует все тестовые модули и запускает их с помощью `unittest.main()`.

## Подробнее

Этот модуль выполняет следующую последовательность действий:

1.  Импортирует модуль `unittest`.
2.  Отключает проверку версии в `g4f.debug`, устанавливая `g4f.debug.version_check = False`.
3.  Импортирует все тестовые модули:
    *   `.asyncio`
    *   `.backend`
    *   `.main`
    *   `.model`
    *   `.client`
    *   `.image_client`
    *   `.include`
    *   `.retry_provider`
    *   `.thinking`
    *   `.web_search`
    *   `.models`
4.  Запускает модульные тесты с помощью `unittest.main()`.

## Функции

### `unittest.main()`

```python
unittest.main()
```

**Назначение**: Запускает модульные тесты, импортированные в данном модуле.

**Параметры**:
*   `Нет`.

**Возвращает**:
*   `None`.

**Вызывает исключения**:
*   `Нет`.

**Как работает функция**:

1.  Функция `unittest.main()` автоматически обнаруживает все классы, наследующие `unittest.TestCase`, и запускает все тестовые методы (методы, имена которых начинаются с `test_`).
2.  Она использует стандартный runner для запуска тестов и выводит результаты в консоль.

**ASCII flowchart**:

```
Начало
  ↓
Обнаружение тестовых классов и методов
  ↓
Запуск тестов
  ↓
Вывод результатов
  ↓
Конец
```

**Примеры**:

```python
import unittest

# Включаем тестовый метод
class MyTest(unittest.TestCase):
    def test_example(self):
        self.assertEqual(1, 1)

unittest.main()
```
```python
import unittest

# Создаем другой тестовый метод
class AnotherTest(unittest.TestCase):
    def test_another_example(self):
        self.assertTrue(True)

unittest.main()
```
