# Модуль `hypotez/src/suppliers/ivory/graber.py`

## Обзор

Этот модуль содержит класс `Graber`, предназначенный для сбора данных с веб-страницы товара на сайте `ivory.co.il`. Класс наследуется от `Grbr` (из модуля `src.suppliers.graber`) и предоставляет методы для обработки различных полей товара. В классе реализована возможность применения декоратора для предварительных действий перед отправкой запроса к веб-драйверу.  Возможность переопределения стандартной обработки полей.

## Классы

### `Graber`

**Описание**:  Класс `Graber` предназначен для сбора данных с веб-страницы товара на сайте `ivory.co.il`.  Наследуется от `Grbr` и предоставляет методы для обработки различных полей товара. Поддерживает возможность переопределения стандартной обработки полей.  В нём также реализована возможность применения декоратора для предварительных действий перед запросом к веб-драйверу.

**Атрибуты**:

- `supplier_prefix: str`: Префикс для идентификации поставщика данных (`ivory`).

**Методы**:

- `__init__(self, driver: Driver)`: Инициализирует экземпляр класса.
    - `driver (Driver)`: Экземпляр веб-драйвера.
    - `supplier_prefix (str)`: Префикс для поставщика.  
    - Устанавливает `supplier_prefix`.  
    - Вызывает конструктор родительского класса `super().__init__(supplier_prefix=self.supplier_prefix, driver=driver)`.
