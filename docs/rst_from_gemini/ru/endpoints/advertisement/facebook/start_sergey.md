# hypotez/src/endpoints/advertisement/facebook/start_sergey.py

## Модуль: src.endpoints.advertisement.facebook

**Описание:**

Модуль `start_sergey.py` отвечает за запуск рекламных кампаний в группах Facebook. Он загружает данные о группах, рекламных объявлениях и кампаниях, а затем отправляет объявления в указанные группы. Модуль использует асинхронные операции для повышения эффективности и гибкости запуска кампаний. Поддерживает несколько языков и валют.

**Константы:**

* `MODE`: `'debug'`.  Устанавливает режим работы.  Важно для отладки и логов.

**Импорты:**

* `header`: Вероятно, модуль, содержащий общие заголовки и настройки.
* `random`, `time`, `copy`, `pathlib`: Стандартные библиотеки для генерации случайных чисел, работы со временем, копирования данных и управления путями.
* `gs`: Модуль для работы с Google Drive.
* `get_directory_names`, `get_filenames`: Функции из модуля `src.utils.file` для работы с файлами и каталогами.
* `Driver`, `Chrome`: Классы для работы с веб-драйвером (вероятно, Selenium).
* `FacebookPromoter`: Класс для управления рекламными кампаниями на Facebook.
* `logger`: Модуль для ведения логов.
* `interval`: Функция из `src.utils.date_time` для определения интервалов времени.


**Функции:**

* **`run_campaign(d, promoter_name, campaigns, group_file_paths, language, currency)`:**
    * Запускает рекламную кампанию.
    * Принимает драйвер `d`, имя рекламодателя `promoter_name`, список кампаний `campaigns`, пути к файлам с группами `group_file_paths`, язык `language`, валюту `currency`.
    * Создает экземпляр `FacebookPromoter` и запускает метод `run_campaigns`.

* **`campaign_cycle(d)`:**
    * Цикл для управления запуском кампаний.
    * Принимает драйвер `d`.
    * Загружает данные о группах и объявлениях для разных языков (русский и иврит).
    * Перебирает пары "язык-валюта" (`language_currency_pairs`).
    * Для каждого языка и валюты запускает `run_campaign` для различных кампаний (`kazarinov_ru`, `kazarinov_he`, ...).
    * Дополнительно загружает кампании из каталога `aliexpress/campaigns` на Google Drive.

* **`main()`:**
    * Основная функция для запуска рекламных кампаний.
    * Создает драйвер `d`.
    * Начинает бесконечный цикл:
        * Выполняет `campaign_cycle`.
        * Проверяет интервал времени с помощью `interval()`.
        * Ведёт логи с помощью `logger`.
        * Засыпает на случайное время с помощью `time.sleep()`.
    * Обрабатывает прерывание `KeyboardInterrupt` для корректного завершения.

**Основные переменные:**

* `group_file_paths_ru`, `adv_file_paths_ru`, `group_file_paths_he`, `adv_file_paths_he`:  списки путей к файлам с данными о группах и объявлениях для разных языков (русский и иврит).
* `group_categories_to_adv`: Список категорий групп, к которым необходимо отправить объявления.
* `language_currency_pairs`: Список словарей, содержащих пары язык-валюта для кампаний.
* `campaigns`: Список кампаний для запуска.
* `d`: экземпляр драйвера.
