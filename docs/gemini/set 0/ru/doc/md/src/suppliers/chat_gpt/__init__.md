# Модуль hypotez/src/suppliers/chat_gpt/__init__.py

## Обзор

Данный модуль предоставляет интерфейс для работы с поставщиком данных ChatGPT, используя Google Sheets.  Он определяет константу `MODE` и импортирует класс `GptGs` из модуля `gsheet`.

## Константы

### `MODE`

**Описание**:  Переменная, определяющая режим работы. В данном случае значение установлено как 'dev'.


## Импорты

### `from .gsheet import GptGs`

**Описание**: Импортирует класс `GptGs` из модуля `gsheet`.  Этот класс, предположительно, содержит функциональность для взаимодействия с Google Sheets и ChatGPT.
