# Схема полей комбинаций товаров PrestaShop

## Обзор

Данная документация описывает структуру JSON-схемы, определяющей поля для импорта и экспорта комбинаций товаров в PrestaShop. Схема представляет собой JSON-объект, где ключи - это названия полей, а значения - их описания. 

## Содержание

- [Обзор](#обзор)
- [Поля](#поля)

## Поля

### `Product ID`
**Описание**: Идентификатор товара.
### `Attribute (Name:Type:Position)`
**Описание**: Атрибут товара, включающий имя, тип и позицию.
### `Value (Value:Position)`
**Описание**: Значение атрибута товара, включающее значение и позицию.
### `Supplier reference`
**Описание**: Ссылка на поставщика.
### `reference`
**Описание**: Внутренняя ссылка.
### `EAN13`
**Описание**: Штрихкод EAN13.
### `UPC`
**Описание**: Штрихкод UPC.
### `Wholesale price`
**Описание**: Оптовая цена.
### `Impact on price`
**Описание**: Влияние на цену.
### `Ecotax`
**Описание**: Экологический налог.
### `Quantity`
**Описание**: Количество товара.
### `Minimal quantity`
**Описание**: Минимальное количество для заказа.
### `Low stock level`
**Описание**: Уровень низкого запаса.
### `Impact on weight`
**Описание**: Влияние на вес.
### `Default (0/1)`
**Описание**: По умолчанию (0 или 1).
### `Combination available date`
**Описание**: Дата, когда комбинация становится доступной.
### `Image position`
**Описание**: Позиция изображения.
### `Image URLs(x,y,z)`
**Описание**: URL-адреса изображений (x, y, z).
### `Image Alt Text(x,y,z)`
**Описание**: Альтернативный текст для изображений (x, y, z).
### `shop`
**Описание**: Магазины, к которым относится комбинация (список через запятую).
### `Advanced Stock Mangment`
**Описание**: Расширенное управление запасами (0 или 1).
### `Depends On Stock`
**Описание**: Зависит от наличия на складе (0 или 1).
### `Warehouse`
**Описание**: Идентификатор склада.
