# hypotez/src/scenario/header.py

```markdown
## Файл hypotez/src/scenario/header.py

Этот файл отвечает за инициализацию проекта, нахождение корневой директории и загрузку настроек.

**Описание:**

Файл содержит функцию `set_project_root`, которая ищет корневую директорию проекта, начиная с текущей директории и поднимаясь вверх по дереву директорий, пока не найдёт директорию, содержащую указанные маркерные файлы (`pyproject.toml`, `requirements.txt`, `.git`).  Если корневая директория найдена, то она добавляется в `sys.path`, что позволяет импортировать модули из этой директории.

Далее, файл загружает настройки из файла `src/settings.json` и документацию из `src/README.MD`. Если файлы не найдены, обрабатываются исключения `FileNotFoundError` и `json.JSONDecodeError` (используется `...`).

Затем, он задаёт переменные, представляющие имя проекта, версию, документацию, детали, автора, копирайт и ссылку на поддержку разработчика, извлекая данные из загруженных настроек.  Если какие-то настройки не найдены, используются значения по умолчанию.

**Функции:**

* **`set_project_root(marker_files=...)`:** Ищет корневую директорию проекта.
    * **Аргументы:** `marker_files` (кортеж): список файлов/директорий, используемых для определения корневой директории.
    * **Возвращает:** `Path`: путь к корневой директории или текущей директории, если корень не найден.

**Переменные:**

* **`MODE`:** Строковая переменная, вероятно, хранит режим работы.
* **`__root__`:** `Path`: Путь к корневой директории проекта.
* **`settings`:** Словарь (`dict`): Загруженные настройки из файла `settings.json`.
* **`doc_str`:** Строка (`str`): Содержимое файла `README.MD`.
* **`__project_name__`:** Строка (`str`): Название проекта (получается из настроек или имеет значение по умолчанию).
* **`__version__`:** Строка (`str`): Версия проекта (получается из настроек или имеет значение по умолчанию).
* **`__doc__`:** Строка (`str`): Документация проекта (получается из файла `README.MD` или имеет значение по умолчанию).
* **`__details__`, `__author__`, `__copyright__`, `__cofee__`:** Строки (`str`): Подробные сведения о проекте, авторе, копирайте и ссылке для поддержки (из настроек или значения по умолчанию).

**Особенности:**

* Использование `Path` для работы с путями, что обеспечивает кроссплатформенность.
* Обработка исключений (`try...except`) для файла настроек и файла документации.
* Значения по умолчанию для переменных, если соответствующие данные не найдены в `settings.json`.
* Вставка корневой директории в `sys.path`, что позволяет импортировать модули из неё.


**Важно:** Файл предполагает наличие директорий и файлов `src`, `settings.json` и `README.MD` в структуре проекта.
```
