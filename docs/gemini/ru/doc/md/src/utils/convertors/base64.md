# Модуль hypotez/src/utils/convertors/base64

## Обзор

Модуль `base64` предназначен для преобразования данных, закодированных в Base64, в временный файл. Он предоставляет функцию `base64_to_tmpfile` для декодирования Base64-строки и сохранения результата в файл с указанным расширением.

## Функции

### `base64_to_tmpfile`

**Описание**: Преобразует Base64-строку в временный файл. Функция декодирует Base64-строку и записывает результат в временный файл с расширением, извлеченным из имени входного файла. Возвращает путь к созданному временному файлу.

**Параметры**:

- `content` (str): Base64-закодированная строка.
- `file_name` (str): Имя файла, из которого извлекается расширение для временного файла.

**Возвращает**:

- `str`: Путь к созданному временному файлу.

**Пример использования**:

```python
base64_content = "SGVsbG8gd29ybGQh"  # Base64 encoded content "Hello world!"
file_name = "example.txt"
tmp_file_path = base64_to_tmpfile(base64_content, file_name)
print(f"Temporary file created at: {tmp_file_path}")
```

**Обрабатываемые исключения**:
Нет указанных исключений.

### `base64encode`

**Описание**: Функция кодирует изображение в формате Base64.

**Параметры**:

- `image_path` (str): Путь к изображению.


**Возвращает**:
- str: Строка, закодированная в Base64.


**Пример использования**:


```python
image_path = "image.jpg"
encoded_image = base64encode(image_path)
print(encoded_image)
```

**Обрабатываемые исключения**:
Нет указанных исключений. Возможны исключения связанные с чтением файла (FileNotFoundError), которые не обрабатываются явно в коде.
