# Модуль hypotez/src/templates

## Обзор

Этот модуль содержит константы и импорты, необходимые для работы шаблонов.

## Константы

### `MODE`

**Описание**:  Переменная, хранящая режим работы приложения (например, 'dev', 'prod').

**Значение**: `"dev"`

## Модули

### `header`

**Описание**:  Модуль, содержащий общие заголовки или настройки.

### `packaging.version`

**Описание**:  Модуль для работы с версиями пакетов.

### `templates.version`

**Описание**:  Модуль, содержащий информацию о версии шаблонов.

**Атрибуты**:

- `__version__`: Строка, содержащая версию шаблонов.
- `__doc__`: Строка, содержащая описание документации.
- `__details__`:  Дополнительная информация о шаблонах.
