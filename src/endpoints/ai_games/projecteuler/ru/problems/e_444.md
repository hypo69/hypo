# Задача No 444
## Покерная игра

## Условия
Рассмотрим покерную игру, в которой используется колода из 16 карт: туз, король, дама и валет каждой из четырех мастей. В игре раздаются 5 карт из колоды.

Среди всех возможных раздач (16 выбираем 5), сколько из них содержат по крайней мере одну пару, т.е. две карты одного ранга?

По аналогии, пусть P(n, c) будет вероятностью, что раздача из c карт из колоды n рангов (4 карты каждого ранга) содержит хотя бы одну пару.

Можно показать, что P(2, 2) = 6 / 11.

Вычислите P(13, 5) и дайте ответ в виде строки, где числитель и знаменатель разделены символом "/". Сократите дробь.
