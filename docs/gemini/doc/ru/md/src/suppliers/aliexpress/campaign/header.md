# Модуль `hypotez/src/suppliers/aliexpress/campaign/header.py`

## Обзор

Этот модуль содержит функции для определения корневого каталога проекта и загрузки настроек проекта из файла `settings.json`. Он также загружает содержимое файла `README.MD` для документации проекта.

## Функции

### `set_project_root`

**Описание**:  Находит корневой каталог проекта, начиная от текущего каталога файла, поднимаясь вверх по иерархии каталогов и останавливаясь на первом каталоге, содержащем указанные файлы-маркеры.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или каталогов, по которым определяется корневой каталог проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневому каталогу проекта, если найден, иначе путь к каталогу, где расположен данный скрипт.

**Вызывает исключения**:

- Нет


### Загрузка настроек


**Описание**: Загружает настройки из файла `settings.json` в переменную `settings`.


**Возвращает**:

- `dict`: Словарь настроек или `None`, если файл не найден или содержит недопустимый JSON.

**Вызывает исключения**:

- `FileNotFoundError`: Если файл `settings.json` не найден.
- `json.JSONDecodeError`: Если файл `settings.json` содержит недопустимый JSON.


### Загрузка документации


**Описание**: Загружает содержимое файла `README.MD` в переменную `doc_str`.


**Возвращает**:

- `str`: Содержимое файла `README.MD` или `None`, если файл не найден или содержит ошибки.

**Вызывает исключения**:

- `FileNotFoundError`: Если файл `README.MD` не найден.
- `json.JSONDecodeError`:  (Необходимо уточнить, есть ли вероятность ошибки декодирования в случае чтения файла `.md`)


## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта.

**Тип**: `Path`


### `__project_name__`

**Описание**: Имя проекта, полученное из файла настроек `settings.json`. По умолчанию "hypotez".

**Тип**: `str`


### `__version__`

**Описание**: Версия проекта, полученная из файла настроек `settings.json`. По умолчанию пустая строка.

**Тип**: `str`


### `__doc__`

**Описание**: Содержимое файла `README.MD`, или пустая строка, если файл не найден.

**Тип**: `str`


### `__details__`

**Описание**: Подробная информация о проекте, по умолчанию пустая строка.

**Тип**: `str`


### `__author__`

**Описание**: Автор проекта, полученный из файла настроек `settings.json`. По умолчанию пустая строка.

**Тип**: `str`


### `__copyright__`

**Описание**: Авторские права на проект, полученные из файла настроек `settings.json`. По умолчанию пустая строка.

**Тип**: `str`


### `__cofee__`

**Описание**: Ссылка для поддержки разработчика. Получена из файла настроек `settings.json`. По умолчанию ссылка для пожертвований.

**Тип**: `str`


## Использование

Этот модуль предназначен для инициализации переменных, необходимых для работы других модулей проекта.  Он устанавливает `__root__`, а также загружает и сохраняет необходимые параметры проекта.  Ключевым аспектом является определение корневого каталога проекта, что позволяет избежать жестко заданных путей к файлам.
