# Документация модуля `test_tinyworld.py`

## Обзор

Этот модуль содержит набор unit-тестов для класса `TinyWorld` и его методов, проверяя основные функции, такие как запуск мира, широковещательные сообщения, кодирование и декодирование состояния мира.

## Функции

### `test_run`

**Описание**:
Проверяет корректность запуска мира `TinyWorld` с различными конфигурациями (пустой мир и мир с агентами).
Также проверяет целостность сообщений в памяти агентов, убеждаясь, что агенты не имеют сообщений, направленных на самих себя.

**Параметры**:
- `setup`: Фикстура pytest для настройки тестовой среды.
- `focus_group_world`: Фикстура pytest, представляющая мир `TinyWorld` с агентами.

**Возвращает**:
- `None`

**Вызывает исключения**:
- `AssertionError`: Если проверки не проходят.

### `test_broadcast`

**Описание**:
Проверяет, что метод `broadcast` правильно отправляет сообщения всем агентам в мире `TinyWorld`.
Гарантирует, что каждый агент получает отправленное сообщение и сохраняет его в своей эпизодической памяти.

**Параметры**:
- `setup`: Фикстура pytest для настройки тестовой среды.
- `focus_group_world`: Фикстура pytest, представляющая мир `TinyWorld` с агентами.

**Возвращает**:
- `None`

**Вызывает исключения**:
- `AssertionError`: Если проверки не проходят.

### `test_encode_complete_state`

**Описание**:
Проверяет, что метод `encode_complete_state` корректно кодирует состояние мира `TinyWorld`, включая его имя и список агентов.
Гарантирует, что возвращаемое состояние не является `None` и содержит ожидаемые атрибуты.

**Параметры**:
- `setup`: Фикстура pytest для настройки тестовой среды.
- `focus_group_world`: Фикстура pytest, представляющая мир `TinyWorld` с агентами.

**Возвращает**:
- `None`

**Вызывает исключения**:
- `AssertionError`: Если проверки не проходят.

### `test_decode_complete_state`

**Описание**:
Проверяет, что метод `decode_complete_state` корректно декодирует состояние мира `TinyWorld`.
Тест проверяет, что после декодирования мир имеет исходное имя и содержит исходное количество агентов.
Также проверяется, что метод возвращает корректный объект мира.

**Параметры**:
- `setup`: Фикстура pytest для настройки тестовой среды.
- `focus_group_world`: Фикстура pytest, представляющая мир `TinyWorld` с агентами.

**Возвращает**:
- `None`

**Вызывает исключения**:
- `AssertionError`: Если проверки не проходят.
