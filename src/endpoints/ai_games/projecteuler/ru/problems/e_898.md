# Задача No 898
## Репликации треугольников

## Условия

Рассмотрим прямоугольный треугольник с длинами сторон a, b и c, где a ≤ b < c и все три числа являются целыми.
С каждой из сторон этого треугольника можно построить прямоугольник, у которого другая сторона равна длине меньшей стороны треугольника.
Таким образом, для прямоугольного треугольника (3, 4, 5) можно построить три прямоугольника: 3×3, 3×4 и 3×5.
Определим как S(a, b, c) общее количество прямоугольных треугольников, которые можно построить внутри этих прямоугольников так, что длины всех сторон являются целыми и гипотенуза лежит вдоль другой стороны прямоугольника. 
Например, S(3, 4, 5) = 24, показано на рисунке ниже.
 
![Пример репликации треугольника](https://projecteuler.net/files/p898_triangle_replication_1.png)

Сумма всех S(a, b, c) для прямоугольных треугольников, у которых c ≤ 100, равна 68657776.
Найдите сумму всех S(a, b, c) для прямоугольных треугольников, у которых c ≤ 10<sup>8</sup>.

<br>
**Примечание:** Рисунок, упомянутый в тексте задачи, показывает, как из прямоугольного треугольника (3, 4, 5) можно получить 24 новых треугольника, рассматривая возможные прямоугольники, построенные на его сторонах. Он также иллюстрирует концепцию расчета S(a, b, c).
