Telegram Bot Trainer
=====================

This module defines a simple Telegram bot for interacting with an OpenAI model. It allows users to send text messages, voice notes, and documents, and receives responses from the model. The bot uses the python-telegram-bot library for Telegram interaction, and OpenAI models for processing user input.
