show_all_results.js
====================

This JavaScript file handles displaying search results from TryXPath.  It dynamically updates the UI with information from the provided `results` object, including context details, main results, and related details, allowing users to download the information in text format.  It also implements event listeners for clicking buttons, enabling actions such as focusing on context items or main details.
