# Модуль `hypotez/src/endpoints/prestashop/api/__init__.py`

## Обзор

Этот модуль инициализирует API для взаимодействия с PrestaShop. Он определяет константу `MODE`, которая, вероятно, указывает режим работы API (например, "dev", "prod"), и импортирует класс `PrestaShop` из подмодуля `api`.

## Константы

### `MODE`

**Описание**: Строковая константа, определяющая режим работы API. По умолчанию, вероятно, устанавливается в 'dev'.

**Значение**: `'dev'`


## Импорты

### `from .api import PrestaShop`

**Описание**: Импортирует класс `PrestaShop` из модуля `api`. Этот импорт позволяет использовать класс в других частях проекта.


```python
MODE = 'dev'

from .api import PrestaShop
```
