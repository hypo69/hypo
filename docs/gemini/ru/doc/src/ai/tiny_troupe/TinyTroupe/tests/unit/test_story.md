# Модуль `test_story.py`

## Обзор

Этот модуль содержит тесты для класса `TinyStory`.  Тесты проверяют корректность инициализации, старта и продолжения истории, используя объекты `TinyWorld`.

## Функции

### `test_story_start`

**Описание**: Тестирует функцию `start_story` класса `TinyStory`.  Проверяет, что сгенерированный старт истории соответствует заданным критериям правдоподобия.

**Параметры**:
- `setup`:  Данные, необходимые для настройки.
- `focus_group_world`: Объект `TinyWorld`, содержащий контекст истории.

**Возвращает**:
-  `None`

**Вызывает исключения**:
- `AssertionError`:  Если сгенерированная строка старта истории не проходит проверку правдоподобия.


### `test_story_start_2`

**Описание**: Тестирует функцию `start_story` класса `TinyStory` с дополнительными требованиями.  Проверяет, что сгенерированный старт истории соответствует заданным критериям правдоподобия, включая специфическое требование к истории.


**Параметры**:
- `setup`:  Данные, необходимые для настройки.
- `focus_group_world`: Объект `TinyWorld`, содержащий контекст истории.

**Возвращает**:
-  `None`

**Вызывает исключения**:
- `AssertionError`:  Если сгенерированная строка старта истории не проходит проверку правдоподобия, связанную с критерием "очень сумасшедшая".

### `test_story_continuation`

**Описание**: Тестирует функцию `continue_story` класса `TinyStory`.  Проверяет, что продолжение истории логично связывается с начальным фрагментом.

**Параметры**:
- `setup`:  Данные, необходимые для настройки.
- `focus_group_world`: Объект `TinyWorld`, содержащий контекст истории.

**Возвращает**:
- `None`

**Вызывает исключения**:
- `AssertionError`: Если продолжение истории не соответствует ожиданиям по связанности с начальным фрагментом.


##  Используемые модули

- `pytest`
- `logging`
- `tinytroupe` и его подмодули (`agent`, `environment`, `factory`, `extraction`, `story`, `examples`, `control`)
- `testing_utils`
