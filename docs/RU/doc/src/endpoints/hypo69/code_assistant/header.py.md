# Модуль `header`

## Обзор

Модуль `header.py` определяет корневой путь к проекту и загружает основные настройки проекта из файла `settings.json`. Все импорты в проекте строятся относительно этого корневого пути. Также модуль загружает описание проекта из файла `README.MD`.

## Функции

### `set_project_root`

**Описание**:
Находит корневую директорию проекта, начиная с директории текущего файла, идя вверх по структуре каталогов до первого каталога, содержащего один из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или каталогов, используемых для идентификации корневой директории проекта. По умолчанию (`'pyproject.toml'`, `'requirements.txt'`, `'.git'`).

**Возвращает**:
- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращается директория, где находится скрипт.

## Переменные

### `__root__`

**Описание**:
Путь к корневой директории проекта. Определяется с помощью функции `set_project_root`.

### `settings`

**Описание**:
Словарь с настройками проекта, загруженными из файла `settings.json`. Если файл не найден или имеет неверный формат JSON, переменная останется `None`.

### `doc_str`
**Описание**:
Строка с описанием проекта, загруженная из файла `README.MD`. Если файл не найден или не удается прочитать, переменная останется `None`.

### `__project_name__`

**Описание**:
Название проекта. Извлекается из настроек (`settings.json`), если доступен, иначе по умолчанию `'hypotez'`.

### `__version__`

**Описание**:
Версия проекта. Извлекается из настроек (`settings.json`), если доступна, иначе по умолчанию пустая строка `''`.

### `__doc__`

**Описание**:
Строка документации проекта. Загружается из `README.md`.

### `__details__`
**Описание**:
Дополнительная информация о проекте. На данный момент всегда пустая строка `''`.

### `__author__`

**Описание**:
Автор проекта. Извлекается из настроек (`settings.json`), если доступен, иначе по умолчанию пустая строка `''`.

### `__copyright__`

**Описание**:
Информация об авторских правах. Извлекается из настроек (`settings.json`), если доступна, иначе по умолчанию пустая строка `''`.

### `__cofee__`

**Описание**:
Строка с призывом угостить разработчика чашкой кофе. Извлекается из настроек (`settings.json`), если доступна, иначе используется значение по умолчанию `"Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69"`.
