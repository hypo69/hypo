# Задача No 253
## Многообразие мусорных баков
## Условия

Представьте себе ряд из n мусорных баков.

Сначала в каждом баке находится по одному шару.

Затем шары случайным образом перераспределяются между баками следующим образом:
*   Для каждого шара выбирается случайный бак (с равной вероятностью).
*   Затем шар помещается в выбранный бак.
*   Эти два действия выполняются для каждого из n шаров.

Пусть M(n) будет математическим ожиданием числа пустых баков после перераспределения шаров.

Например, M(3) = 1.

Вы можете проверить, что M(10) ≈ 3.602348673 и M(100) ≈ 36.60323913.

Найдите M(10<sup>7</sup>), с точностью до 10 знаков после запятой.
