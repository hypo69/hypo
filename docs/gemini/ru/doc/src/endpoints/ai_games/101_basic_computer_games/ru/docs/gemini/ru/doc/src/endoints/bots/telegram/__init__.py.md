# Модуль `__init__.py`

## Обзор

Модуль `__init__.py` инициализирует пакет `src.bots.telegram` и определяет константу `MODE`. Он также импортирует класс `TelegramBot` из модуля `bot.py`.

## Содержание

- [Обзор](#обзор)
- [Константы](#константы)
- [Импорт](#импорт)

## Константы

### `MODE`

- **Описание**: Константа, определяющая режим работы модуля. В данном случае установлена в `dev`.
- **Тип**: `str`
- **Значение**: `'dev'`

## Импорт

### `from .bot import TelegramBot`

- **Описание**: Импортирует класс `TelegramBot` из модуля `bot.py`, который находится в той же директории.
