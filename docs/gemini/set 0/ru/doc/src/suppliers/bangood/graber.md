# Модуль hypotez/src/suppliers/bangood/graber.py

## Обзор

Модуль `graber.py` содержит класс `Graber`, предназначенный для сбора данных о товарах с сайта bangood.com. Класс наследуется от родительского класса `Graber` из `src.suppliers`, обеспечивая базовый функционал для работы с веб-драйвером и предварительные действия.  В этом классе переопределены методы для обработки специфических полей.

## Классы

### `Graber`

**Описание**: Класс `Graber` отвечает за сбор данных о товарах с сайта bangood.com.  Он использует веб-драйвер для взаимодействия с сайтом и предоставляет методы для извлечения различных полей товара.

**Методы**:

- `__init__(self, driver: Driver)`:
    **Описание**: Инициализирует экземпляр класса `Graber`. Устанавливает `supplier_prefix` и передаёт `driver` экземпляру родительского класса. Также устанавливает значение `Context.locator_for_decorator`.
    **Параметры**:
    - `driver` (Driver): Экземпляр класса `Driver` для взаимодействия с веб-драйвером.
    
- `grab_page(self, driver: Driver) -> ProductFields`:
    **Описание**: Асинхронная функция для сбора данных о товаре.
    **Параметры**:
    - `driver` (Driver): Экземпляр веб-драйвера.
    **Возвращает**:
    - `ProductFields`: Объект, содержащий собранные данные о товаре.

## Функции

### `fetch_all_data(**kwards)`

**Описание**:  Асинхронная функция, которая вызывает другие функции для извлечения данных о товаре. Функция принимает ключевые аргументы и вызывает соответствующие методы из класса `Graber`, передавая необходимые значения.

**Параметры**:

- `**kwards`:  Ключевые аргументы для передачи в функции, вызываемые внутри.
