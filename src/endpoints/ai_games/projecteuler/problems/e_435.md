# Задача No 435
## Метод итерации

## Условия

Пусть $f(x) = \sum_{k=0}^{n} a_k x^k$ - многочлен степени $n$ с целыми коэффициентами.

Мы будем называть последовательность $\{ x_i \}$ итерациями $x$ по $f$, если $x_0 = x$ и $x_{i+1} = f(x_i)$ для $i \ge 0$.

Для целого $x$ и многочлена $f$ определим $C(f, x)$ как наименьшее значение $i$, такое что $x_i = x_j$ для некоторого $j < i$.

Для заданного $x$ и многочлена $f$ последовательность итераций по $f$ от $x$ будет периодической после $C(f, x)$ итераций.

Пусть $g_n(x) = x^n + x^{n-1} + \dots + x + 1$.

Например, $g_4(2) = 2^4 + 2^3 + 2^2 + 2 + 1 = 31$.
$g_n(x)$ можно записать как $(x^{n+1} - 1) / (x-1)$.

Пусть $S(n) = \sum_{x=1}^{100} C(g_n, x)$.

Известно, что $S(5) = 1973$.

Найдите $S(10^15) \mod 1 000 000 007$.
