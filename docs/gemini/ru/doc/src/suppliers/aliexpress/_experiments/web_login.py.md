# Модуль `web_login.py`

## Обзор

Модуль `web_login.py` предназначен для автоматизации процесса входа в систему Aliexpress, работы с cookies и выполнения других проверок, связанных с авторизацией. Этот модуль является частью экспериментов, связанных с поставщиком Aliexpress в проекте `hypotez`.

## Подробней

Этот модуль предназначен для автоматизации процесса входа в систему Aliexpress, работы с cookies и выполнения других проверок, связанных с авторизацией.

Модуль использует библиотеки `header`, `pathlib`, `pickle`, `requests` и классы `Supplier` и `Driver` из других модулей проекта `hypotez`. Он инициализирует поставщика Aliexpress и получает доступ к драйверу веб-браузера для выполнения действий на сайте Aliexpress.

## Импортированные модули и классы

- `header`: Модуль, содержащий заголовки запросов.
- `pathlib.Path`: Класс для работы с путями к файлам и директориям.
- `pickle`: Модуль для сериализации и десериализации объектов Python.
- `requests`: Библиотека для выполнения HTTP-запросов.
- `src.gs`: Глобальные настройки проекта.
- `src.suppliers.Supplier`: Базовый класс для работы с поставщиками.
- `src.utils.printer.pprint`: Функция для красивого вывода данных.

## Переменные модуля

- `a`: Экземпляр класса `Supplier` для Aliexpress.
- `d`: Драйвер веб-браузера, полученный из экземпляра `Supplier`.

## Инструкции по работе

1.  Импортируйте необходимые модули и классы.
2.  Создайте экземпляр класса `Supplier` для Aliexpress.
3.  Получите драйвер веб-браузера из экземпляра `Supplier`.
4.  Используйте драйвер для выполнения действий на сайте Aliexpress, таких как переход по URL.

Пример использования:

```python
import header
from pathlib import Path
import pickle
import requests

from src import gs
from src.suppliers import Supplier
from src.utils.printer import pprint

a = Supplier('aliexpress')

d = a.driver
d.get_url('https://aliexpress.com')
```
