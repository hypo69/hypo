# Модуль `hypotez/src/suppliers/aliexpress/api/tools/get_product_id.py`

## Обзор

Этот модуль предоставляет функцию `get_product_id` для извлечения идентификатора продукта из входного текста.

## Функции

### `get_product_id`

**Описание**: Возвращает идентификатор продукта, извлеченный из входного текста. В случае неудачного извлечения, вызывает исключение `ProductIdNotFoundException`.

**Параметры**:
- `raw_product_id` (str): Исходный текст, содержащий идентификатор продукта.

**Возвращает**:
- str: Идентификатор продукта в виде строки. Возвращает значение, полученное от функции `extract_prod_ids`.

**Вызывает исключения**:
- `ProductIdNotFoundException`: Если идентификатор продукта не найден в входном тексте.  Этот эквивалентен исключению из `extract_prod_ids` в случае неудачи.  Важно проверить документацию `extract_prod_ids` для конкретных условий, когда это исключение выбрасывается.

**Примечание**:  Текущая реализация использует функцию `extract_prod_ids`. Необходимо просмотреть реализацию функции `extract_prod_ids` для понимания точного алгоритма извлечения идентификатора продукта.  Комментарии в коде, касающиеся поиска по регулярному выражению, сейчас неактуальны.
