# Задача No 738
## Плитки Фермера Игнаца
## Условия
Фермер Игнац хочет покрыть плитками прямоугольную область размером *n* × *m*. У него есть плитки двух типов:

*   Прямоугольники 1 × 2.
*   L-образные плитки, состоящие из трех единичных квадратов.

На рисунке ниже показаны примеры L-образных плиток.

```
    ■  ■    ■ ■    ■    ■
    ■        ■    ■  ■    ■
```

Пусть F(*n*, *m*) — количество способов покрыть прямоугольник *n* × *m* этими двумя типами плиток.

Например, F(2, 3) = 5, F(2, 4) = 11, F(3, 2) = 13, F(4, 2) = 24 и F(2, 10) = 2881.

Определим S(*n*) = ∑ F(*n*, *m*) для 1 ≤ *m* ≤ 10.
Например, S(2) = 5 + 11 + 21 + 36 + 60 + 99 + 163 + 268 + 440 + 721 = 1824.

Также известно, что S(5) = 56822.

Найдите S(10000).
