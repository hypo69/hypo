# Модуль `hypotez/src/webdriver/playwright/playwrid.py`

## Обзор

Этот модуль определяет подкласс `PlaywrightCrawler`, называемый `Playwrid`. Он предоставляет дополнительную функциональность, такую как возможность устанавливать пользовательские настройки браузера, профили и параметры запуска с помощью Playwright.  Модуль загружает настройки из файла `playwrid.json` и опционально из файлов с именем, переданным в конструктор.  Модуль также управляет параметрами запуска Playwright, включая `headless` режим и пользовательские аргументы.

## Классы

### `Playwrid`

**Описание**: Подкласс `PlaywrightCrawler`, предоставляющий расширенные возможности.

**Методы**:

#### `__init__`

**Описание**: Инициализирует экземпляр `Playwrid` с указанными параметрами запуска, настройками и пользовательским агентом.

**Параметры**:

- `settings_name` (str, необязательно): Имя файла настроек для использования.
- `user_agent` (dict, необязательно): Словарь, содержащий настройки пользовательского агента.

**Возвращает**:
- None

**Вызывает исключения**:
- None

#### `_load_settings`

**Описание**: Загружает настройки для `Playwrid` из файла `playwrid.json`, или из переданного в конструктор файла.

**Параметры**:

- `settings_name` (str, необязательно): Имя файла настроек для использования.

**Возвращает**:
- SimpleNamespace: Объект SimpleNamespace, содержащий загруженные настройки.

**Вызывает исключения**:
- None


#### `_set_launch_options`

**Описание**: Настраивает параметры запуска для `Playwrid`.

**Параметры**:

- `settings` (SimpleNamespace): Объект SimpleNamespace, содержащий настройки запуска.

**Возвращает**:
- dict: Словарь с параметрами запуска для Playwright.

**Вызывает исключения**:
- None


#### `start`

**Описание**: Запускает `Playwrid` и переходит на указанный URL.

**Параметры**:

- `url` (str): URL для перехода.

**Возвращает**:
- None

**Вызывает исключения**:
- Любой тип исключения, которое может быть вызвано `super().run()`. Ошибка обработки записывается в лог с подробным описанием.

#### `current_url`

**Описание**: Свойство, возвращающее текущий URL.  (Не реализовано в данном фрагменте кода, но добавлено для полноты описания.)

**Возвращает**:
- str: Текущий URL.

**Вызывает исключения**:
- None
