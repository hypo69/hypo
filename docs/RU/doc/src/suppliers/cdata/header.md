# Модуль `hypotez/src/suppliers/cdata/header.py`

## Обзор

Этот модуль содержит функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и документации из `README.MD`, а также инициализирует переменные, хранящие информацию о проекте.

## Функции

### `set_project_root`

**Описание**: Функция находит корневую директорию проекта, начиная с текущей директории и поднимаясь вверх по дереву директорий, пока не найдет директорию, содержащую один из указанных файлов или папок.

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или директорий, по которым определяется корневая директория.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена, иначе - путь к директории, в которой находится текущий скрипт.

**Пример использования**:

```python
root_path = set_project_root(('pyproject.toml', 'requirements.txt'))
```

### `__root__`

**Описание**: Переменная, хранящая корневую директорию проекта, полученную с помощью функции `set_project_root`.

**Тип**: `Path`


## Переменные

### `MODE`

**Описание**: Строковая переменная, хранящая значение режима работы (в данном случае `dev`).


### `settings`

**Описание**: Переменная, хранящая словарь настроек, загруженный из файла `src/settings.json`.

**Тип**: `dict` | `None`


### `doc_str`

**Описание**: Переменная, хранящая строку документации, загруженную из файла `README.MD`.

**Тип**: `str` | `None`

### `__project_name__`

**Описание**: Переменная, содержащая имя проекта, взятая из настроек (`settings`).  Использует значение по умолчанию `hypotez` при отсутствии в настройках.

**Тип**: `str`

### `__version__`

**Описание**: Переменная, содержащая версию проекта, взятая из настроек (`settings`).  Использует значение по умолчанию `''` при отсутствии в настройках.

**Тип**: `str`

### `__doc__`

**Описание**: Переменная, содержащая строку документации проекта, взятая из `doc_str`.  Использует значение по умолчанию `''` при отсутствии в файле.

**Тип**: `str`

### `__details__`

**Описание**: Переменная, хранящая дополнительные детали проекта.

**Тип**: `str`

### `__author__`

**Описание**: Переменная, содержащая имя автора проекта, взятая из настроек (`settings`).  Использует значение по умолчанию `''` при отсутствии в настройках.

**Тип**: `str`

### `__copyright__`

**Описание**: Переменная, содержащая авторские права проекта, взятая из настроек (`settings`).  Использует значение по умолчанию `''` при отсутствии в настройках.

**Тип**: `str`

### `__cofee__`

**Описание**: Переменная, содержащая ссылку на платёжную систему для спонсорства разработчика. Значение по умолчанию взято из комментария в коде.

**Тип**: `str`
