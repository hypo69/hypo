# Модуль hypotez/src/suppliers

## Обзор

Этот модуль содержит базовые классы и интерфейсы для работы с поставщиками данных.  Он предоставляет абстрактный класс `Supplier` для взаимодействия с различными источниками данных (например, Amazon, AliExpress, Morlevi). Конкретные методы извлечения данных для каждого поставщика размещаются в соответствующих подпапках (например, `amazon`, `aliexpress`).  Модуль использует механизм подключения дополнительных функций через интерфейс `supplier.related_functions`.


## Классы

### `Supplier`

**Описание**: Базовый абстрактный класс для работы с поставщиками.  Этот класс определяет общий интерфейс, который реализуют конкретные поставщики, такие как AmazonSupplier, AliExpressSupplier и т.д.

**Методы** (Подробные описания методов находятся в файлах соответствующих поставщиков):

- `get_products(self, query: str, page: int = 1) -> list[dict] | None`: Получает список продуктов, соответствующих запросу.

- `get_product_details(self, product_id: str) -> dict | None`: Получает подробную информацию о продукте по его идентификатору.

- `get_prices(self, product_id: str) -> list[dict] | None`: Получает историю цен для продукта.

- `close(self) -> None`: Закрывает соединение с поставщиком.
