# `src.logger.header`

## Обзор

Модуль `src.logger.header` определяет корневой путь к проекту и загружает основные настройки проекта из файлов `settings.json` и `README.MD`. Он обеспечивает централизованное определение корня проекта и предоставляет доступ к основным параметрам проекта.

## Функции

### `set_project_root`

**Описание**:
Функция `set_project_root` находит корневую директорию проекта, начиная с директории текущего файла, и двигаясь вверх по дереву директорий, останавливаясь на первой директории, содержащей один из маркеров.

**Параметры**:
- `marker_files` (tuple, optional): Кортеж имен файлов или каталогов, которые используются для идентификации корневой директории проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращает путь к директории, где расположен скрипт.

## Глобальные переменные

### `__root__`

**Описание**:
`__root__` - это глобальная переменная, хранящая путь к корневой директории проекта. Значение переменной устанавливается функцией `set_project_root`.

### `settings`

**Описание**:
`settings` - это глобальная переменная, хранящая словарь с настройками проекта, загруженными из файла `settings.json`. Если файл не найден или не может быть прочитан, значение переменной остается `None`.

### `doc_str`

**Описание**:
`doc_str` - это глобальная переменная, хранящая содержимое файла `README.MD` в виде строки. Если файл не найден или не может быть прочитан, значение переменной остается `None`.

### `__project_name__`

**Описание**:
`__project_name__` - это глобальная переменная, хранящая имя проекта. По умолчанию значение 'hypotez', но может быть получено из файла `settings.json`.

### `__version__`

**Описание**:
`__version__` - это глобальная переменная, хранящая версию проекта. Значение может быть получено из файла `settings.json`. Если значение не указано, то по умолчанию будет пустая строка.

### `__doc__`

**Описание**:
`__doc__` - это глобальная переменная, хранящая документацию проекта. Значение устанавливается на основе содержимого `doc_str`.

### `__details__`

**Описание**:
`__details__` - это глобальная переменная, хранящая детальную информацию о проекте. По умолчанию пустая строка.

### `__author__`

**Описание**:
`__author__` - это глобальная переменная, хранящая имя автора проекта. Значение может быть получено из файла `settings.json`. Если значение не указано, то по умолчанию будет пустая строка.

### `__copyright__`

**Описание**:
`__copyright__` - это глобальная переменная, хранящая информацию о копирайте проекта. Значение может быть получено из файла `settings.json`. Если значение не указано, то по умолчанию будет пустая строка.

### `__cofee__`

**Описание**:
`__cofee__` - это глобальная переменная, хранящая сообщение о поддержке автора проекта. Значение может быть получено из файла `settings.json`, либо будет использоваться значение по умолчанию.
