# `header.py`

## Обзор

Модуль `header.py` предназначен для настройки и загрузки основных параметров проекта, таких как корневая директория, настройки из файла `settings.json`, а также информации из `README.MD`. Он также предоставляет глобальные переменные для доступа к этим параметрам.

## Функции

### `set_project_root`

**Описание**:
Находит корневой каталог проекта, начиная с каталога текущего файла, ища вверх и останавливаясь в первом каталоге, содержащем любой из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple): Имена файлов или каталогов, идентифицирующие корень проекта.

**Возвращает**:
- `Path`: Путь к корневому каталогу, если он найден, иначе каталог, где расположен скрипт.

## Глобальные переменные

### `__root__`

**Описание**:
Путь к корневому каталогу проекта. Определяется функцией `set_project_root`.

### `settings`

**Описание**:
Словарь настроек, загруженных из файла `settings.json`. Если файл не найден или имеет ошибку JSON, значение будет `None`.

### `doc_str`

**Описание**:
Содержимое файла `README.MD` в виде строки. Если файл не найден или имеет ошибку, значение будет `None`.

### `__project_name__`

**Описание**:
Имя проекта, полученное из настроек или по умолчанию 'hypotez'.

### `__version__`

**Описание**:
Версия проекта, полученная из настроек. Может быть пустой строкой.

### `__doc__`

**Описание**:
Содержимое документации из `README.MD` или пустая строка.

### `__details__`

**Описание**:
Дополнительная информация о проекте. В текущей версии всегда пустая строка.

### `__author__`

**Описание**:
Автор проекта, полученный из настроек или пустая строка.

### `__copyright__`

**Описание**:
Информация об авторских правах, полученная из настроек или пустая строка.

### `__cofee__`

**Описание**:
Сообщение с ссылкой для поддержки разработчика чашкой кофе, полученное из настроек или значение по умолчанию.
