# Модуль Tower

## Обзор

Этот модуль представляет собой текстовую игру "Башня", в которой игрок строит башню, вводя положительные числа. Каждое введенное число добавляется к высоте башни. Цель игры - построить башню как можно выше. Игра заканчивается после 10 ходов или когда игрок вводит неотрицательное число.
