# Модуль для работы с ChatGpt
## Обзор

Модуль `chat_gpt.py` предназначен для организации работы с ChatGpt, в частности, для обработки HTML-файлов, содержащих историю переписок.

## Подробней

Модуль `chat_gpt.py` предназначен для работы с данными, полученными из ChatGpt. На данный момент модуль содержит класс `ChatGpt`, который умеет обрабатывать HTML-файлы с историей переписок, находящиеся в определенной директории. Расположение файлов определяется через обращение к объекту `gs.path.data / 'chat_gpt' / 'conversations'`.

## Классы

### `ChatGpt`

**Описание**: Класс `ChatGpt` предназначен для обработки данных, связанных с ChatGpt.

**Принцип работы**:
Класс содержит метод `yeld_conversations_htmls`, который предназначен для поиска и обработки HTML-файлов с историей переписок. Файлы должны находиться в директории `data/chat_gpt/conversations`.

**Методы**:
- `yeld_conversations_htmls()`: Метод для поиска HTML-файлов с историей переписок и их обработки.
