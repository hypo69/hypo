# Задача No 123
## Наименьшее значение n для которого остаток r превышает 10^10
## Условия

Пусть *p<sub>n</sub>* обозначает *n*-е простое число. Найдите наименьшее значение *n*, для которого остаток при делении (*p<sub>n</sub>*-1)<sup>*n*</sup> + (*p<sub>n</sub>*+1)<sup>*n*</sup> на *p<sub>n</sub>*<sup>2</sup>  превышает 10<sup>10</sup>.

Например, если *n*=3, то *p<sub>3</sub>*=5, и остаток от (5-1)<sup>3</sup> + (5+1)<sup>3</sup> при делении на 5<sup>2</sup> равен 12, то есть 4<sup>3</sup> + 6<sup>3</sup> = 64 + 216 = 280 = 11 * 25 + 5.

Таким образом, r (остаток) = 12.
