# Модуль `test_experimentation.py`

## Обзор

Данный модуль содержит тесты для класса `ABRandomizer`, проверяющие корректность рандомизации и дерандомизации вариантов А/Б-тестирования.

## Функции

### `test_randomize`

**Описание**: Тестирует метод `randomize` класса `ABRandomizer`.  Функция запускает метод `randomize` 20 раз с различными входными данными и проверяет, что возвращаемые пары соответствуют ожидаемому рандомизированному результату.

**Параметры**:
-  `i` (int): Индекс элемента для рандомизации.
-  `"option1"` (str): Первый вариант.
-  `"option2"` (str): Второй вариант.

**Возвращает**:
-  None

**Вызывает исключения**:
- `Exception`: Если рандомизация не соответствует ожидаемому результату.


### `test_derandomize`

**Описание**: Тестирует метод `derandomize` класса `ABRandomizer`. Функция запускает метод `randomize` и `derandomize` 20 раз с различными входными данными и проверяет, что возвращаемые пары соответствуют исходным парам.


**Параметры**:
- `i` (int): Индекс элемента для рандомизации.
- `"option1"` (str): Первый вариант.
- `"option2"` (str): Второй вариант.

**Возвращает**:
- None

**Вызывает исключения**:
- Нет


### `test_derandomize_name`

**Описание**: Тестирует метод `derandomize_name` класса `ABRandomizer`.  Функция запускает метод `randomize` и `derandomize_name` 20 раз с различными входными данными и проверяет, что возвращаемое значение имени соответствует ожидаемому результату рандомизации.

**Параметры**:
- `i` (int): Индекс элемента для рандомизации.
- `"A"` (str): Первый вариант.
- `"B"` (str): Второй вариант.

**Возвращает**:
- None

**Вызывает исключения**:
- `Exception`: Если рандомизация не соответствует ожидаемому результату.


### `test_passtrough_name`

**Описание**: Тестирует поведение метода `derandomize_name` при наличии списка `passtrough_name`. Функция запускает метод `randomize` и `derandomize_name` с заданным списком и проверяет, что для значения из списка возвращается соответствующее имя.

**Параметры**:
- `passtrough_name` (list[str]): Список имен, которые должны быть возвращены без изменения.
- `"option1"` (str): Первый вариант.
- `"option2"` (str): Второй вариант.

**Возвращает**:
- None

**Вызывает исключения**:
- Нет


### `test_intervention_1`

**Описание**:  Тест, показывающий, что функция `intervention_1` не вызывает никаких исключений.

**Параметры**:
-  Нет.

**Возвращает**:
-  None

**Вызывает исключения**:
-  Нет. (Замечание: функция `pass` в коде означает, что эта функция пока не реализована, но это не ошибка.)
