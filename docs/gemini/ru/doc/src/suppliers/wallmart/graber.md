# Модуль `hypotez/src/suppliers/wallmart/graber.py`

## Обзор

Модуль `hypotez/src/suppliers/wallmart/graber.py` содержит класс `Graber`, предназначенный для сбора данных о товарах с сайта `wallmart.com`.  Класс наследуется от родительского класса `Graber` (из модуля `src.suppliers.graber`), расширяя функциональность для обработки специфических данных сайта `wallmart.com`.  Включает в себя обработку всплывающих окон (используя декоратор `close_pop_up`, который по умолчанию используется из родительского класса, но может быть переопределён) и инициализацию для работы с веб-драйвером.
