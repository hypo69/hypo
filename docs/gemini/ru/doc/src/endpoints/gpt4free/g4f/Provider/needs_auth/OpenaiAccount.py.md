# Модуль OpenaiAccount

## Обзор

Модуль `OpenaiAccount` является частью проекта `hypotez` и предназначен для работы с учетными записями OpenAI. Он расширяет функциональность класса `OpenaiChat`, добавляя специфические атрибуты, такие как необходимость аутентификации и использование веб-драйвера.

## Подробнее

Модуль определяет класс `OpenaiAccount`, который наследует класс `OpenaiChat`. Он устанавливает атрибут `needs_auth` в `True`, что указывает на необходимость аутентификации для использования данной модели. Также указывается родительский класс `OpenaiChat` и отключается использование драйвера (`use_nodriver = False`).

## Классы

### `OpenaiAccount`

**Описание**: Класс `OpenaiAccount` предназначен для работы с учетными записями OpenAI. Он наследует функциональность от класса `OpenaiChat` и добавляет атрибуты, указывающие на необходимость аутентификации и использование веб-драйвера.

**Наследует**:

- `OpenaiChat`: Класс, предоставляющий базовую функциональность для взаимодействия с OpenAI.

**Атрибуты**:

- `needs_auth` (bool): Указывает на необходимость аутентификации для использования данной модели. Значение по умолчанию: `True`.
- `parent` (str): Указывает на родительский класс. Значение: `"OpenaiChat"`.
- `use_nodriver` (bool): Определяет, нужно ли использовать драйвер. Если `False` - драйвер нужен. Значение по умолчанию: `False`.

## Функции

В данном модуле нет отдельных функций, но класс `OpenaiAccount` наследует методы от класса `OpenaiChat`.
