# Модуль header

## Обзор

Модуль `header.py` предназначен для определения корневой директории проекта и загрузки настроек из файла `settings.json`. Он содержит функции для поиска корневой директории на основе наличия определенных файлов-маркеров и обеспечивает доступ к настройкам проекта.

## Подробней

Этот модуль важен для определения абсолютного пути к корневой директории проекта, что необходимо для правильной работы с файлами конфигурации и другими ресурсами. Функция `set_project_root` ищет корневую директорию, начиная с текущего файла, и останавливается, когда находит один из файлов-маркеров (например, `__root__` или `.git`). После этого модуль пытается загрузить настройки из файла `settings.json`, расположенного в директории `src` относительно корневой директории проекта.

## Функции

### `set_project_root`

```python
def set_project_root(marker_files: tuple = ('__root__', '.git')) -> Path:
    """
    Args:
        marker_files (tuple): Filenames or directory names to identify the project root.
    
    Returns:
        Path: Path to the root directory if found, otherwise the directory where the script is located.
    """
```

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла и двигаясь вверх по дереву каталогов. Поиск прекращается при обнаружении каталога, содержащего один из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или каталогов, используемых для идентификации корневой директории проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена, иначе - директория, где расположен скрипт.

**Примеры**:
```python
from pathlib import Path
#Предположим, что текущий файл находится в /path/to/project/src/module.py
#И в /path/to/project/ находится файл .git

root_path = set_project_root()
print(root_path)
#Вывод: /path/to/project
```

## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта. Инициализируется функцией `set_project_root`.

### `settings`

**Описание**: Словарь, содержащий настройки, загруженные из файла `settings.json`. Если файл не найден или содержит ошибки, переменная остается `None`.
```python
settings:dict = None
