# try_xpath_check_frame.js

## Обзор

Файл `try_xpath_check_frame.js` является частью расширения `tryxpath` и предназначен для управления и проверки состояния загрузки контента в контексте iframe. Он содержит переменную `tryxpath.isContentLoaded`, которая используется для отслеживания, был ли загружен контент в iframe.

## Переменные

### `tryxpath.isContentLoaded`
    
**Описание**:
  Переменная, которая используется для отслеживания, был ли загружен контент в iframe. Изначально не имеет значения.
