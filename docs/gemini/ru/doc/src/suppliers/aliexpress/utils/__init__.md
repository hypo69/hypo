# Модуль `hypotez/src/suppliers/aliexpress/utils`

## Обзор

Этот модуль предоставляет утилиты для работы с данными AliExpress. Он содержит функции для извлечения идентификаторов продуктов, перевода URL-адресов в HTTPS и работы с локализациями.

## Оглавление

- [Функции](#функции)
    - [`extract_prod_ids`](#extract_prod_ids)
    - [`ensure_https`](#ensure_https)
    - [`locales`](#locales)


## Функции

### `extract_prod_ids`

**Описание**: Функция для извлечения идентификаторов продуктов из входных данных.

**Параметры**:
- Нет данных в функции. (Зависит от реализации)


**Возвращает**:
- `list[str]`: Список идентификаторов продуктов.


**Вызывает исключения**:
- `ValueError`: Возникает, если входные данные не в корректном формате.


### `ensure_https`

**Описание**: Функция для перевода URL-адреса в HTTPS-формат.

**Параметры**:
- `url` (str): URL-адрес.

**Возвращает**:
- `str`: URL-адрес в HTTPS-формате.

**Вызывает исключения**:
- `ValueError`: Возникает, если `url` не является корректным URL.


### `locales`

**Описание**: Функция для получения локализации. (Возможно, содержит словарь или список локалей).

**Параметры**:
- Нет данных в функции. (Зависит от реализации)

**Возвращает**:
- `dict` | `list`: Словарь или список локалей.


**Вызывает исключения**:
- `TypeError`: Возникает если аргумент имеет неправильный тип.
