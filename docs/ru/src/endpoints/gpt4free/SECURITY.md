# Документация по безопасности

## Обзор

Этот файл содержит информацию о том, как сообщать об уязвимостях в проекте `hypotez`.

## Подробней

В этом файле описан процесс сообщения об уязвимостях безопасности, обнаруженных в проекте. Он предоставляет контактную информацию и ожидания по времени ответа и выпуска исправлений.

## Разделы

### Сообщение об уязвимости

**Описание**: Инструкция по сообщению об уязвимостях в проекте.

**Принцип работы**:

1.  **Сообщение об уязвимости**: Пользователям предлагается сообщать о подозрительных уязвимостях безопасности в Telegram по указанной ссылке.
2.  **Время ответа**: Указано, что ответ будет предоставлен в течение 48 часов.
3.  **Выпуск исправления**: В случае подтверждения проблемы будет выпущен патч, сроки которого зависят от сложности, но обычно в течение нескольких дней.
