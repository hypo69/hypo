# Модуль `hypotez/src/bots/telegram/bot.py`

## Обзор

Данный модуль предоставляет класс `TelegramBot` для взаимодействия с Telegram ботом.  Он отвечает за регистрацию обработчиков команд и сообщений, обработку различных типов сообщений (текстовые, голосовые, документы), а также за отправку PDF-файлов.

## Класс `TelegramBot`

### `__init__`

**Описание**: Инициализирует Telegram бот.

**Параметры**:

- `token` (str): Токен Telegram бота.

**Возвращает**:
-  (None): Не возвращает значение.


### `register_handlers`

**Описание**: Регистрирует обработчики команд и сообщений.


**Возвращает**:
- (None): Не возвращает значение.


### `start`

**Описание**: Обработчик команды `/start`.

**Параметры**:

- `update` (Update): Объект с данными обновления.
- `context` (CallbackContext): Контекст текущего разговора.

**Возвращает**:
- (None): Не возвращает значение.


### `help_command`

**Описание**: Обработчик команды `/help`.

**Параметры**:

- `update` (Update): Объект с данными обновления.
- `context` (CallbackContext): Контекст текущего разговора.

**Возвращает**:
- (None): Не возвращает значение.


### `send_pdf`

**Описание**: Обработчик команды `/sendpdf` для отправки PDF-файла.

**Параметры**:

- `pdf_file` (str | Path): Путь к файлу PDF.

**Возвращает**:
- (None): Не возвращает значение.

**Обрабатываемые исключения**:

- `Exception`: Обрабатывает любые ошибки при отправке PDF-файла и выводит сообщение об ошибке пользователю.


### `handle_voice`

**Описание**: Обработчик голосовых сообщений.

**Параметры**:

- `update` (Update): Объект с данными обновления.
- `context` (CallbackContext): Контекст текущего разговора.

**Возвращает**:
- (None): Не возвращает значение.

**Обрабатываемые исключения**:

- `Exception`: Обрабатывает любые ошибки при обработке голосового сообщения и выводит сообщение об ошибке пользователю.


### `transcribe_voice`

**Описание**:  Метод распознавания речи. Заглушка, требует замены на реализацию распознавания речи.

**Параметры**:

- `file_path` (Path): Путь к файлу голосового сообщения.

**Возвращает**:
- `str`: Распознанный текст.


### `handle_document`

**Описание**: Обработчик документов.

**Параметры**:

- `update` (Update): Объект с данными обновления.
- `context` (CallbackContext): Контекст текущего разговора.

**Возвращает**:
- str: Содержимое текстового документа.


### `handle_message`

**Описание**: Обработчик текстовых сообщений.

**Параметры**:

- `update` (Update): Объект с данными обновления.
- `context` (CallbackContext): Контекст текущего разговора.

**Возвращает**:
