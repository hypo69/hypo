# Модуль hypotez/src/goog/spreadsheet/bberyakov

## Обзор

Этот модуль предоставляет классы для работы с Google Spreadsheets. Он содержит классы для работы с отдельными листами (GWorksheet) и с целым документом (GSpreadsheet). Также предоставляет базовый класс для визуализации данных (GSRenderr).
