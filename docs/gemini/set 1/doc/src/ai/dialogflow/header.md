# Модуль hypotez/src/ai/dialogflow/header.py

## Обзор

Этот модуль содержит вспомогательные функции для работы с проектом Hypotez, включая определение корневой директории проекта, загрузку настроек из файла `settings.json` и чтение документации из файла `README.MD`.  Он также предоставляет глобальные переменные, содержащие информацию о проекте, такие как имя, версия, описание, автор и т.д.


## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с текущего файла и двигаясь вверх по директориям до тех пор, пока не найдёт директорию, содержащую файлы-маркеры (pyproject.toml, requirements.txt, .git).

**Параметры**:

- `marker_files` (tuple): Кортеж строк, представляющих имена файлов или папок, по которым ищется корневая директория. По умолчанию: `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена. В противном случае возвращает директорию, где расположен текущий скрипт.

**Вызывает исключения**:

- Нет


###  (Глобальная переменная) `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root()`.

**Тип**: `Path`


## Загрузка настроек

**Описание**: Загрузка настроек проекта из файла `settings.json` в глобальную переменную `settings`.

**Обработка ошибок**:

- Если файл `settings.json` не найден или содержит некорректные данные, переменная `settings` остаётся `None`.  Используются блоки `try...except` для обработки исключений `FileNotFoundError` и `json.JSONDecodeError`.


## Чтение документации

**Описание**: Чтение документации проекта из файла `README.MD` в глобальную переменную `doc_str`.

**Обработка ошибок**:

- Если файл `README.MD` не найден или содержит некорректные данные, переменная `doc_str` остаётся `None`. Используются блоки `try...except` для обработки исключений `FileNotFoundError` и `json.JSONDecodeError`.

## Глобальные переменные

**Описание**: Глобальные переменные, хранящие информацию о проекте. Значения инициализируются из переменной `settings` (если она не None) или имеют значения по умолчанию.

**Переменные**:

- `__project_name__` (str): Имя проекта (по умолчанию 'hypotez').
- `__version__` (str): Версия проекта (по умолчанию '').
- `__doc__` (str): Документация проекта (по умолчанию '').
- `__details__` (str): Дополнительная информация о проекте (по умолчанию '').
- `__author__` (str): Автор проекта (по умолчанию '').
- `__copyright__` (str): Авторские права (по умолчанию '').
- `__cofee__` (str): Ссылка для поддержки разработчика (по умолчанию).
