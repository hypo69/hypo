# Модуль `hypotez/src/suppliers/locator.py`

Файл: `C:\Users\user\Documents\repos\hypotez\src\suppliers\locator.py`

## Описание

Модуль `locator.py` предоставляет класс `Locator` для загрузки и представления настроек локатора для поставщиков. Локаторы хранятся в JSON файлах, что обеспечивает гибкость и удобство доступа к настройкам.

## Класс `Locator`

Класс `Locator` предназначен для работы с данными локатора, загруженными из JSON файла.

**Атрибуты:**

* `supplier_prefix (str)`: Префикс, идентифицирующий папку поставщика, в которой хранятся файлы конфигурации локатора.
* `locator (SimpleNamespace)`: Объект `SimpleNamespace`, содержащий определения локатора, загруженные из JSON файла. Обеспечивает удобный доступ к атрибутам локатора.

**Методы:**

* `__init__(self, supplier_prefix: str)`: Инициализирует экземпляр `Locator`. Загружает настройки локатора из JSON файла, используя переданный `supplier_prefix`.

    **Аргументы:**
    * `supplier_prefix (str)`: Префикс, идентифицирующий папку поставщика.

    **Исключения:**
    * `FileNotFoundError`: Если JSON файл с определениями локатора не найден.
    * `ValueError`: Если содержимое JSON файла имеет неверный формат.


* `_load_locator(self) -> SimpleNamespace`: Загружает настройки локатора из JSON файла.

    **Возвращаемое значение:**
    * `SimpleNamespace`: Объект, содержащий данные локатора, загруженные из файла.

    **Исключения:**
    * `FileNotFoundError`: Если JSON файл не найден.
    * `ValueError`: Если содержимое JSON файла не соответствует ожидаемому формату.

## Пример использования

```python
from hypotez.src.suppliers.locator import Locator
from pathlib import Path

# Предполагается, что у вас есть переменная gs.path.src, содержащая путь к директории src
supplier_prefix = "example_supplier"
locator = Locator(supplier_prefix)

# Теперь вы можете получить доступ к данным локатора через locator.locator
print(locator.locator.field1)
print(locator.locator.field2)

# Обратите внимание на использование try-except для обработки потенциальных ошибок
try:
    locator = Locator("invalid_supplier")
except FileNotFoundError as e:
    print(f"Ошибка: {e}")
except ValueError as e:
    print(f"Ошибка: {e}")
```

## Комментарии

* Используется объект `gs.path.src` для построения пути к файлу JSON с определениями локатора.  Важно понимать, как именно определяется `gs.path.src` в вашем коде.
* `SimpleNamespace` используется для удобного доступа к атрибутам локатора.
* Обработка исключений `FileNotFoundError` и `ValueError` обеспечивает надёжность кода.
* Документирование `_load_locator` делает код более понятным и поддерживаемым.
* Добавлен пример использования и обработки ошибок для улучшения практической ценности документации.
