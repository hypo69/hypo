# Модуль `facebook`

## Обзор

Модуль `facebook` предназначен для автоматизации взаимодействия с платформой Facebook через веб-драйвер. Он включает в себя функциональность для входа в систему, продвижения постов и управления медиа-контентом.

## Подробней

Этот модуль предоставляет класс `Facebook`, который позволяет взаимодействовать с Facebook через веб-драйвер. Он поддерживает такие сценарии, как вход в аккаунт, публикация сообщений и загрузка медиафайлов.

## Классы

### `Facebook`

**Описание**: Класс для взаимодействия с Facebook через веб-драйвер.

**Как работает класс**:
Класс `Facebook` инициализируется с драйвером веб-браузера и именем пользователя. Он предоставляет методы для выполнения различных действий на Facebook, таких как вход в систему и продвижение постов.

**Методы**:
- `__init__`: Инициализирует экземпляр класса `Facebook`.
- `login`: Выполняет вход в аккаунт Facebook.
- `promote_post`: Продвигает пост на Facebook.
- `promote_event`: Продвигает событие на Facebook.

#### `login`

```python
def login(self) -> bool:
    return login(self)
```

**Описание**: Выполняет вход в аккаунт Facebook.

**Как работает класс**:
Метод `login` вызывает функцию `login` из модуля `.scenarios.login`, передавая ей текущий экземпляр класса `Facebook`.

**Возвращает**:
- `bool`: `True`, если вход выполнен успешно, иначе `False`.
