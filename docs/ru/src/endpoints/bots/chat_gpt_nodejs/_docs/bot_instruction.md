## Команды

### `!hi`

**Описание**: Приветствует пользователя.

**Как работает функция**: Отправляет приветственное сообщение пользователю, инициируя взаимодействие с ботом.

**Параметры**: Нет параметров.

**Возвращает**: Нет возвращаемого значения.

**Примеры**:
- `!hi`

### `!train`

**Описание**: Обучает модель на основе предоставленных данных.

**Как работает функция**: Использует предоставленные данные для обучения модели. Данные могут быть представлены в виде файла, директории или вложения.

**Параметры**:
- `data` (str): Путь к файлу с данными.
- `data_dir` (str): Путь к директории с данными.
- `positive` (str): Метка положительного класса для обучения.
- `attachment` (file): Файл, прикрепленный к сообщению, содержащий данные.

**Возвращает**: Нет возвращаемого значения.

**Примеры**:
- `!train data.txt data_dir positive`
- `!train attachment attachment_dir positive attachment_file`

### `!test`

**Описание**: Тестирует модель на основе предоставленных тестовых данных в формате JSON.

**Как работает функция**: Применяет обученную модель к тестовым данным и оценивает ее производительность.

**Параметры**:
- `test_data` (str): JSON строка, содержащая тестовые данные.

**Возвращает**: Нет возвращаемого значения.

**Примеры**:
- `!test {"feature1": "value1", "feature2": "value2"}`

### `!archive`

**Описание**: Архивирует файлы, находящиеся в указанной директории.

**Как работает функция**: Создает архив (например, zip) файлов, расположенных в указанной директории.

**Параметры**:
- `directory` (str): Путь к директории, которую необходимо архивировать.

**Возвращает**: Нет возвращаемого значения.

**Примеры**:
- `!archive /path/to/directory`

### `!select_dataset`

**Описание**: Выбирает набор данных для обучения из указанной директории.

**Как работает функция**: Позволяет выбрать определенный набор данных для обучения модели, указывая путь к директории с данными и метку положительного класса.

**Параметры**:
- `path_to_dir_positive` (str): Путь к директории, содержащей положительные примеры для обучения.
- `positive` (str): Метка, обозначающая положительный класс.

**Возвращает**: Нет возвращаемого значения.

**Примеры**:
- `!select_dataset /path/to/positive/data positive_label`

### `!instruction`

**Описание**: Отображает данное справочное сообщение.

**Как работает функция**: Выводит список доступных команд и их краткое описание для пользователя.

**Параметры**: Нет параметров.

**Возвращает**: Нет возвращаемого значения.

**Примеры**:
- `!instruction`
