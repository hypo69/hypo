# Модуль hypotez/src/webdriver/edge/header.py

## Обзор

Этот модуль содержит вспомогательные функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и чтения документации из файла `README.MD`. Он также предоставляет доступ к настройкам проекта, версии, описанию, автору и информации об авторских правах.

## Функции

### `set_project_root`

**Описание**:  Находит корневую директорию проекта, начиная от директории текущего файла и двигаясь вверх по дереву директорий. Поиск останавливается, когда находится директория, содержащая один из файлов-маркеров (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, по которым определяется корневая директория проекта. По умолчанию содержит `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена. В противном случае возвращает директорию текущего файла.


**Вызывает исключения**:

-  Не вызывают исключений.


###  Загрузка настроек


**Описание**: Загружает настройки из файла `settings.json` в переменную `settings`.

**Параметры**:  Нет.

**Возвращает**: Нет.

**Вызывает исключения**:

- `FileNotFoundError`: Если файл `settings.json` не найден.
- `json.JSONDecodeError`: Если содержимое `settings.json` не является валидным JSON.


### Чтение документации


**Описание**: Читает содержимое файла `README.MD` и сохраняет его в переменную `doc_str`.

**Параметры**: Нет.

**Возвращает**: Нет.

**Вызывает исключения**:

- `FileNotFoundError`: Если файл `README.MD` не найден.
- `json.JSONDecodeError`:  Если содержимое файла `README.MD` не является корректным текстом.


## Переменные

### `__root__`

**Описание**: Переменная, содержащая путь к корневой директории проекта, полученная с помощью функции `set_project_root()`.

**Тип**: `Path`.

### `settings`

**Описание**: Словарь, содержащий настройки проекта, загруженный из файла `settings.json`. Может быть `None`, если файл не найден или некорректен.

**Тип**: `dict` или `None`.

### `__project_name__`

**Описание**: Имя проекта, взятое из настроек или установленное по умолчанию как `hypotez`.

**Тип**: `str`.

### `__version__`

**Описание**: Версия проекта, взятая из настроек или установленное по умолчанию как пустая строка.

**Тип**: `str`.

### `__doc__`

**Описание**: Документация проекта, взятая из файла `README.MD` или установленное по умолчанию как пустая строка.

**Тип**: `str`.


### `__details__`

**Описание**: Дополнительные детали, пока не используется.

**Тип**: `str`.

### `__author__`

**Описание**: Автор проекта, взятый из настроек или установленное по умолчанию как пустая строка.

**Тип**: `str`.

### `__copyright__`

**Описание**: Авторские права, взятый из настроек или установленное по умолчанию как пустая строка.

**Тип**: `str`.


### `__cofee__`

**Описание**: Ссылка для поддержки автора чашкой кофе.

**Тип**: `str`.
