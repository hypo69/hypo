# Модуль `hypotez/src/suppliers/bangood/graber.py`

## Обзор

Модуль содержит класс `Graber`, предназначенный для сбора данных с веб-страниц товаров на сайте `bangood.com`. Класс наследуется от класса `Grbr` из родительского модуля `src.suppliers.graber`.  Он предоставляет методы для обработки различных полей товара, а также возможность переопределять стандартную обработку для конкретных полей.  В классе реализован декоратор для закрытия всплывающих окон до выполнения основной функции.

## Классы

### `Graber`

**Описание**: Класс `Graber` предназначен для сбора данных с веб-страниц товаров на сайте bangood.com. Он расширяет функциональность родительского класса `Grbr`, предоставляя специализированные методы обработки данных.

**Атрибуты**:

- `supplier_prefix`: Строка, представляющая префикс поставщика данных (`bangood`).

**Методы**:

- `__init__`: Инициализирует экземпляр класса `Graber`.
    - **Параметры**:
        - `driver` (`Driver`): Экземпляр класса драйвера веб-браузера, используемого для взаимодействия с веб-страницей.
    - **Возвращает**:  None.
    - **Описание**: Метод инициализирует атрибут `supplier_prefix` и вызывает конструктор родительского класса `Grbr`. Устанавливает глобальные настройки через `Context`
    - **Вызывает исключения**: Возможны исключения, связанные с инициализацией веб-драйвера.


## Функции

**Описание**:  В этом модуле нет независимых функций, все функциональности реализованы в классе.
