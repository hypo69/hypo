# Модуль для определения токенов

## Обзор

Модуль содержит функцию `is_token`, которая определяет, является ли переданная строка токеном определенного формата.

## Подробней

Данный модуль предназначен для идентификации токенов, которые используются в проекте `hypotez`. Функция `is_token` проверяет, соответствует ли переданная строка формату токена (состоит из 62 символов, второй символ - `/`).

## Функции

### `is_token`

```python
def is_token(token):
    """
    Проверяет, является ли переданная строка токеном.

    Args:
        token (str): Строка, которую необходимо проверить на соответствие формату токена.

    Returns:
        bool: `True`, если строка является токеном, `False` в противном случае.
    """
```

**Как работает функция**:

Функция `is_token` выполняет следующие действия:

1. **Извлечение последнего слова**: Извлекает последнее слово из переданной строки `token`, разделяя строку по пробелам. Это необходимо для того, чтобы проверить только токен, даже если он является частью более длинной строки.

2. **Определение длины токена**: Определяет длину извлеченного токена и сохраняет ее в переменной `TLEN`.

3. **Проверка длины и формата**:
    - Если длина токена равна 62, проверяет, является ли второй символ токена `/`. Если это так, функция возвращает `True`, иначе - `False`.
    - Если длина токена не равна 62, функция возвращает `False`.

**Параметры**:

- `token` (str): Строка, которую необходимо проверить на соответствие формату токена.

**Возвращает**:

- `bool`: `True`, если строка является токеном, `False` в противном случае.

**Примеры**:

```python
# Пример 1: Проверка строки, являющейся токеном
token1 = "Some string with token XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX/XXXXXXXXXX"
result1 = is_token(token1)
print(result1)  # Вывод: True

# Пример 2: Проверка строки, не являющейся токеном (неверная длина)
token2 = "Invalid token"
result2 = is_token(token2)
print(result2)  # Вывод: False

# Пример 3: Проверка строки, не являющейся токеном (неверный формат)
token3 = "Some string with token XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX-XXXXXXXXXX"
result3 = is_token(token3)
print(result3)  # Вывод: False
