# Модуль `main.py`

## Обзор

Модуль `main.py` представляет собой главный файл графического интерфейса (GUI) для управления рекламными кампаниями, товарами и категориями в рамках поставщика AliExpress. Он включает в себя основные элементы интерфейса, такие как главное окно, вкладки для различных редакторов (JSON, кампании, товары), а также меню с опциями для работы с файлами и редактирования.

## Подробней

Этот модуль является отправной точкой для запуска GUI приложения. Он инициализирует главное окно приложения, создает вкладки для редакторов JSON, кампаний и товаров, а также настраивает меню с командами для открытия, сохранения, копирования и вставки данных. GUI позволяет пользователям управлять рекламными кампаниями, редактировать информацию о товарах и категориях, а также взаимодействовать с данными в формате JSON. Модуль использует библиотеку PyQt6 для создания интерфейса и qasync для интеграции асинхронных операций.

## Классы

### `MainApp`

**Описание**: Главное окно приложения с вкладками для редакторов JSON, кампаний и товаров.

**Методы**:
- `__init__`: Инициализирует главное окно, создает вкладки и добавляет их в виджет вкладок.
- `create_menubar`: Создает строку меню с опциями для работы с файлами и редактирования.
- `open_file`: Открывает диалоговое окно выбора файла для загрузки JSON файла.
- `save_file`: Сохраняет текущий файл в зависимости от выбранной вкладки.
- `exit_application`: Завершает работу приложения.
- `copy`: Копирует выделенный текст в буфер обмена.
- `paste`: Вставляет текст из буфера обмена.
- `load_file`: Загружает JSON файл.

**Параметры**:
- Нет параметров.

**Примеры**
```python
app = QtWidgets.QApplication(sys.argv)
main_app = MainApp()
main_app.show()
```

## Функции

### `main`

```python
def main():
    """ Initialize and run the application """
    app = QtWidgets.QApplication(sys.argv)

    # Create an event loop for asynchronous operations
    loop = QEventLoop(app)
    asyncio.set_event_loop(loop)

    main_app = MainApp()
    main_app.show()

    # Run the event loop
    with loop:
        loop.run_forever()
```
