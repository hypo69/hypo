# Модуль hypotez/src/suppliers/amazon/graber.py

## Обзор

Модуль `graber.py` содержит класс `Graber`, предназначенный для сбора данных с веб-страниц Amazon. Класс наследуется от `Graber` из родительского модуля, предоставляя возможность переопределять методы обработки полей для специфической обработки данных Amazon.  Модуль также содержит декоратор для закрытия всплывающих окон перед выполнением функций.

## Классы

### `Graber`

**Описание**: Класс `Graber` расширяет функциональность родительского класса `Graber` для работы с данными Amazon.  Он предоставляет методы для работы с веб-драйвером и обработкой полей.

**Методы**:

- `__init__(self, driver: Driver)`:
    **Описание**: Инициализирует экземпляр класса.
    **Параметры**:
        - `driver` (Driver): Экземпляр класса `Driver` для взаимодействия с веб-драйвером.
    **Возвращает**:
        - `None`
