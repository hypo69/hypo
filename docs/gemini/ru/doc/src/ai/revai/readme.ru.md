# revai (rev.com)

## Обзор

Модуль предоставляет документацию по работе с API rev.com, сервисом для автоматической транскрибации аудио и видео. Он содержит информацию о конечных точках API, параметрах запросов и форматах ответов.

## Подробней

Данный модуль предназначен для разработчиков, желающих интегрировать возможности rev.com по транскрибации в свои приложения. Он содержит ссылки на официальную документацию API rev.com, а также примеры кода на Python.
