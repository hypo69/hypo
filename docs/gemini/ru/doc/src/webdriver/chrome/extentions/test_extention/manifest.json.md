# `manifest.json`

## Обзор

Файл `manifest.json` описывает расширение Hypotez для веб-браузера Chrome. Он определяет метаданные расширения, необходимые разрешения, фоновый скрипт, всплывающее окно, значки и разрешения хоста.

## Структура

### `manifest_version`
**Описание**: Версия манифеста расширения.

**Тип**: `number`
**Значение**: `3`

### `name`
**Описание**: Название расширения.

**Тип**: `string`
**Значение**: `"Hypotez"`

### `version`
**Описание**: Версия расширения.

**Тип**: `string`
**Значение**: `"1.0"`

### `description`
**Описание**: Краткое описание назначения расширения.

**Тип**: `string`
**Значение**: `"Collects data from web pages and sends it to a server."`

### `permissions`
**Описание**: Массив разрешений, необходимых расширению.

**Тип**: `array`
**Значения**:
   -  `"activeTab"`: Разрешение на доступ к текущей активной вкладке.
   -  `"storage"`: Разрешение на использование хранилища браузера.
   -  `"scripting"`: Разрешение на внедрение скриптов.

### `background`
**Описание**: Описание фонового скрипта расширения.

**Тип**: `object`
**Свойства**:
    - `"service_worker"`: Путь к фоновому скрипту.
        - **Тип**: `string`
        - **Значение**: `"borderify.js"`

### `action`
**Описание**: Описание всплывающего окна расширения.

**Тип**: `object`
**Свойства**:
  - `"default_popup"`: Путь к HTML файлу всплывающего окна.
        - **Тип**: `string`
        - **Значение**: `"html/popup.html"`
  - `"default_icon"`: Пути к значкам расширения.
        - **Тип**: `object`
        - **Свойства**:
          - `"16"`: Путь к значку размером 16x16 пикселей.
                - **Тип**: `string`
                - **Значение**: `"icons/icon.png"`
           - `"48"`: Путь к значку размером 48x48 пикселей.
                - **Тип**: `string`
                - **Значение**: `"icons/icon.png"`
           - `"128"`: Путь к значку размером 128x128 пикселей.
                - **Тип**: `string`
                - **Значение**: `"icons/icon.png"`

### `icons`
**Описание**: Пути к значкам расширения.

**Тип**: `object`
**Свойства**:
  - `"16"`: Путь к значку размером 16x16 пикселей.
        - **Тип**: `string`
        - **Значение**: `"icons/icon.png"`
  - `"48"`: Путь к значку размером 48x48 пикселей.
        - **Тип**: `string`
        - **Значение**: `"icons/icon.png"`
  - `"128"`: Путь к значку размером 128x128 пикселей.
        - **Тип**: `string`
        - **Значение**: `"icons/icon.png"`

### `host_permissions`
**Описание**: Разрешения хоста для расширения.

**Тип**: `array`
**Значения**:
  - `"https://*/*"`: Разрешение на доступ ко всем HTTPS ресурсам.
  - `"http://*/*"`: Разрешение на доступ ко всем HTTP ресурсам.
