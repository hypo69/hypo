# Модуль для работы с URL строками

## Обзор

Модуль `src.utils.string.url` предназначен для работы с URL-адресами, включая извлечение параметров запроса, проверку URL на валидность и сокращение длинных ссылок.

## Подробнее

Этот модуль предоставляет функции для обработки URL-адресов, которые могут быть полезны в различных приложениях, где требуется работа с веб-ссылками. Он включает в себя методы для извлечения параметров из URL, проверки URL на соответствие стандартам и сокращения длинных URL с использованием сервиса TinyURL.

## Функции

### `extract_url_params`

```python
def extract_url_params(url: str) -> dict | None:
    """ Извлекает параметры из строки URL.

    Args:
        url (str): Строка URL для парсинга.

    Returns:
        dict | None: Словарь параметров запроса и их значений или `None`, если URL не содержит параметров.
    """
    ...
```

**Назначение**: Извлечение параметров запроса из URL.

**Как работает функция**:
Функция принимает URL в качестве входных данных, использует `urlparse` для разбора URL и `parse_qs` для извлечения параметров запроса. Затем она преобразует значения параметров из списка в строку, если параметр имеет только одно значение. Если параметры отсутствуют, возвращается `None`.

**Параметры**:
- `url` (str): URL для парсинга.

**Возвращает**:
- `dict | None`: Словарь, содержащий параметры запроса и их значения. Если URL не содержит параметров, возвращается `None`.

**Примеры**:

```python
url = "https://example.com?param1=value1&param2=value2"
params = extract_url_params(url)
print(params)  # Вывод: {'param1': 'value1', 'param2': 'value2'}

url = "https://example.com"
params = extract_url_params(url)
print(params)  # Вывод: None
```

**Как работает функция**:
Функция использует библиотеку `validators` для проверки, является ли переданная строка валидным URL. Она возвращает `True`, если строка является валидным URL, и `False` в противном случае.

**Параметры**:
- `text` (str): Строка, которую необходимо проверить на валидность URL.

**Возвращает**:
- `bool`: `True`, если строка является валидным URL, иначе `False`.

**Примеры**:

```python

print(is_valid)  # Вывод: True

url = "not a url"
is_valid = is_url(url)
print(is_valid)  # Вывод: False
```

**Назначение**: Сокращение длинного URL с использованием сервиса TinyURL.

**Как работает функция**:
Функция принимает длинный URL, отправляет запрос к сервису TinyURL для его сокращения и возвращает сокращенный URL. Если запрос успешен (код состояния 200), возвращается сокращенный URL. В случае ошибки возвращается `None`.

**Параметры**:
- `long_url` (str): Длинный URL для сокращения.

**Возвращает**:
- `str | None`: Сокращенный URL или `None`, если произошла ошибка.

**Примеры**:

```python

if short_url:
    print(f"Сокращенный URL: {short_url}")
else:
    print("Ошибка при сокращении URL.")
