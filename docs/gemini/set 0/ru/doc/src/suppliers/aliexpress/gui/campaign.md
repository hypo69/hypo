# Модуль hypotez/src/suppliers/aliexpress/gui/campaign.py

## Обзор

Данный модуль предоставляет класс `CampaignEditor` для редактирования кампаний AliExpress. Класс позволяет открывать JSON-файлы с данными кампании, загружать их и отображать в виде пользовательского интерфейса.  Также, предоставляет возможность подготовить кампанию.


## Классы

### `CampaignEditor`

**Описание**: Класс `CampaignEditor` представляет собой виджет PyQt для редактирования данных кампаний AliExpress.  Он позволяет загружать JSON-данные, отображать их в удобном формате, а также выполнять подготовку кампании.

**Атрибуты**:

- `data`: `SimpleNamespace`.  Данные загруженной кампании.
- `current_campaign_file`: `str`.  Путь к файлу с данными кампании.
- `editor`: `AliCampaignEditor`.  Инстанс класса для подготовки кампании.


**Методы**:

#### `__init__(self, parent=None, main_app=None)`

**Описание**: Инициализирует виджет `CampaignEditor`.

**Параметры**:
- `parent` (Optional): Родительский виджет.
- `main_app` (Optional):  Ссылка на приложение.


#### `setup_ui(self)`

**Описание**: Настраивает пользовательский интерфейс. Создает необходимые виджеты (кнопки, метки, поля ввода), добавляет их в макет.

#### `setup_connections(self)`

**Описание**: Настраивает соединения сигналов и слотов.

#### `open_file(self)`

**Описание**: Открывает диалоговое окно для выбора JSON-файла с данными кампании.

#### `load_file(self, campaign_file)`

**Описание**: Загружает JSON-файл с данными кампании.

**Параметры**:
- `campaign_file` (str): Путь к файлу.

**Обрабатывает исключения**:
- `Exception`: Обрабатывает ошибки при загрузке JSON-файла и выводит соответствующее сообщение об ошибке.

#### `create_widgets(self, data)`

**Описание**: Создает виджеты для отображения данных кампании.

**Параметры**:
- `data` (SimpleNamespace): Данные загруженной кампании.


#### `prepare_campaign(self)`

**Описание**: Асинхронно выполняет подготовку кампании.

**Обрабатывает исключения**:
- `Exception`: Обрабатывает ошибки при подготовке кампании и выводит соответствующее сообщение об ошибке.
