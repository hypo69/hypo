# src.endpoints.hypo69

## Обзор

Этот модуль содержит конечные точки (endpoints) для разработчиков в проекте `hypotez`. Он включает в себя подмодули для различных целей, таких как бот для небольших разговоров с моделью ИИ, модуль для обучения модели коду проекта и ранняя разработка модуля для анализа диалогов психолога.

## Подробней

Модуль `src.endpoints.hypo69` предоставляет набор инструментов и API для разработчиков, работающих над проектом `hypotez`. Он разделен на несколько подмодулей, каждый из которых выполняет определенную функцию:

- `small_talk_bot`: Бот, предназначенный для ведения небольших разговоров с использованием модели ИИ. Может использоваться для тестирования и демонстрации возможностей модели.
- `code_assistant`: Модуль, предназначенный для обучения модели коду проекта. Помогает модели понимать структуру и логику кодовой базы `hypotez`.
- `psychologist_bot`: Находящийся на ранней стадии разработки модуль, который предназначен для анализа диалогов психолога.
