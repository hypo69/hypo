# Модуль запуска API g4f

## Обзор

Модуль предназначен для запуска API g4f (gpt4free) с использованием аргументов командной строки. Он использует парсер аргументов для обработки входных параметров и запуска API с заданными настройками.

## Подробней

Данный модуль является точкой входа для запуска API g4f. Он использует модуль `cli` для получения парсера аргументов, обрабатывает аргументы командной строки и запускает API с соответствующими параметрами.

## Функции

### `get_api_parser`

```python
def get_api_parser() -> argparse.ArgumentParser:
    """Получает парсер аргументов командной строки.

    Args:
        -

    Returns:
        argparse.ArgumentParser: Парсер аргументов командной строки.

    Raises:
        -

    Example:
        >>> parser = get_api_parser()
        >>> args = parser.parse_args(['--host', '0.0.0.0', '--port', '8000'])
    """
```

**Как работает функция**:

1.  Функция `get_api_parser()` из модуля `cli` возвращает настроенный парсер аргументов командной строки.
2.  Парсер используется для обработки аргументов, переданных при запуске скрипта.

**ASCII flowchart**:

```
Получение парсера аргументов из cli
↓
Возврат парсера
```

**Примеры**:

```python
from .cli import get_api_parser
parser = get_api_parser()
args = parser.parse_args(['--host', '0.0.0.0', '--port', '8000'])
```

### `run_api_args`

```python
def run_api_args(args: argparse.Namespace) -> None:
    """Запускает API с использованием аргументов командной строки.

    Args:
        args (argparse.Namespace): Аргументы командной строки.

    Returns:
        None

    Raises:
        -

    Example:
        >>> parser = get_api_parser()
        >>> args = parser.parse_args(['--host', '0.0.0.0', '--port', '8000'])
        >>> run_api_args(args)
    """
```

**Как работает функция**:

1.  Функция `run_api_args(args)` из модуля `cli` запускает API, используя переданные аргументы командной строки.

**ASCII flowchart**:

```
Получение аргументов командной строки
↓
Запуск API с использованием аргументов
```

**Примеры**:

```python
from .cli import get_api_parser, run_api_args
parser = get_api_parser()
args = parser.parse_args(['--host', '0.0.0.0', '--port', '8000'])
run_api_args(args)
```

## Логика работы модуля

1.  Получение парсера аргументов командной строки с помощью `get_api_parser()`.
2.  Разбор аргументов командной строки с использованием парсера.
3.  Если аргумент `gui` не указан, устанавливается значение `True`.
4.  Запуск API с использованием обработанных аргументов с помощью `run_api_args(args)`.
