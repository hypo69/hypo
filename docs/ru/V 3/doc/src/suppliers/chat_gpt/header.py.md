# Модуль `header`

## Обзор

Модуль `header.py` предназначен для определения корневого каталога проекта, загрузки настроек из файла `settings.json` и чтения документации из файла `README.MD`. Он также инициализирует основные переменные проекта, такие как имя, версия, автор и т.д.

## Подробней

Модуль выполняет следующие задачи:

1.  Определение корневого каталога проекта на основе наличия файлов-маркеров (например, `__root__` или `.git`).
2.  Чтение настроек проекта из файла `settings.json`. Если файл не найден или содержит ошибки, переменная `settings` остается `None`.
3.  Чтение содержимого файла `README.MD` для получения документации проекта. Если файл не найден или содержит ошибки, переменная `doc_str` остается `None`.
4.  Инициализация основных переменных проекта (`__project_name__`, `__version__`, `__doc__`, `__details__`, `__author__`, `__copyright__`, `__cofee__`) на основе данных из файла `settings.json` или значений по умолчанию.

Этот модуль важен для инициализации проекта и получения основных параметров, необходимых для работы других модулей.

## Функции

### `set_project_root`

```python
def set_project_root(marker_files: tuple = ('__root__', '.git')) -> Path:
    """
    Args:
        marker_files (tuple): Filenames or directory names to identify the project root.
    
    Returns:
        Path: Path to the root directory if found, otherwise the directory where the script is located.
    """
```

**Описание**: Находит корневой каталог проекта, начиная с каталога текущего файла, и останавливается на первом каталоге, содержащем любой из файлов-маркеров.

**Параметры**:

*   `marker_files` (tuple): Имена файлов или каталогов, используемые для определения корня проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:

*   `Path`: Путь к корневому каталогу, если он найден, в противном случае - каталог, в котором расположен скрипт.

**Примеры**:

```python
from pathlib import Path
# Пример использования функции set_project_root
root_path = set_project_root()
print(f'Root path: {root_path}')
```

## Переменные

### `__root__`

**Описание**: Путь к корневому каталогу проекта.

### `settings`

**Описание**: Словарь с настройками проекта, загруженными из файла `settings.json`.

### `doc_str`

**Описание**: Строка с документацией проекта, загруженной из файла `README.MD`.

### `__project_name__`

**Описание**: Имя проекта.

### `__version__`

**Описание**: Версия проекта.

### `__doc__`

**Описание**: Документация проекта.

### `__details__`

**Описание**: Дополнительная информация о проекте (в данном коде всегда пустая строка).

### `__author__`

**Описание**: Автор проекта.

### `__copyright__`

**Описание**: Информация об авторских правах проекта.

### `__cofee__`

**Описание**: Сообщение с предложением поддержать разработчика.
