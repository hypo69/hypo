# Модуль `hypotez/src/suppliers/ksp/banners_grabber.py`

## Обзор

Модуль `banners_grabber.py` предназначен для сбора баннеров от поставщика KSP.  Он содержит функцию `get_banners` для получения данных баннеров.

## Функции

### `get_banners`

**Описание**: Функция `get_banners` возвращает `True`, предполагая успешный сбор баннеров.


**Возвращает**:
- `bool`: `True`, если сбор баннеров успешно завершен.


**Вызывает исключения**:
- Нет.


```python
def get_banners() -> bool:
    """
    Args:
        Нет.

    Returns:
        bool: True, если сбор баннеров успешно завершен.

    Raises:
        Нет.
    """
    return True
```
