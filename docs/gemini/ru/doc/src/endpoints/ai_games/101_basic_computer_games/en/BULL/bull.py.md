# Игра "BULL"

## Обзор

Данный модуль реализует числовую логическую игру "BULL", в которой компьютер генерирует случайное четырехзначное число с уникальными цифрами, а игрок пытается его угадать. После каждой попытки компьютер сообщает количество "быков" (цифра угадана и находится на правильном месте) и "коров" (цифра угадана, но находится на неправильном месте).

## Функции

### `generate_target_number`

**Описание**: Генерирует случайное четырехзначное число с уникальными цифрами.

**Параметры**:
- Нет параметров.

**Возвращает**:
- `str`: Строка, представляющая четырехзначное число со всеми уникальными цифрами.

### `get_bulls_and_cows`

**Описание**: Подсчитывает количество "быков" и "коров" на основе предположения пользователя и загаданного числа.

**Параметры**:
- `user_guess` (str): Предположение пользователя в виде строки.
- `target_number` (str): Загаданное компьютером число в виде строки.

**Возвращает**:
- `tuple[int, int]`: Кортеж, где первый элемент - количество "быков", а второй - количество "коров".
