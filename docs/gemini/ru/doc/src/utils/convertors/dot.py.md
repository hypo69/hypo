# Модуль `dot`

## Обзор

Модуль предназначен для конвертации файлов в формате DOT в изображения PNG с использованием библиотеки Graphviz.

## Подробнее

Модуль предоставляет функцию `dot2png`, которая принимает путь к DOT-файлу и путь для сохранения PNG-файла. Он использует библиотеку `graphviz` для выполнения преобразования. Этот модуль полезен для визуализации графов, представленных в формате DOT.

## Функции

### `dot2png`

```python
def dot2png(dot_file: str, png_file: str) -> None:
    """ Конвертирует DOT-файл в PNG-изображение.

    Args:
        dot_file (str): Путь к входному DOT-файлу.
        png_file (str): Путь, по которому будет сохранен выходной PNG-файл.

    Raises:
        FileNotFoundError: Если DOT-файл не существует.
        Exception: При других ошибках во время конвертации.

    Example:
        >>> dot2png('example.dot', 'output.png')
        
        Это преобразует DOT-файл 'example.dot' в PNG-изображение с именем 'output.png'.
        
        Пример содержимого DOT-файла 'example.dot':
        
        ```dot
        digraph G {
            A -> B;
            B -> C;
            C -> A;
        }
        ```
        
        Чтобы запустить скрипт из командной строки:
        
        ```bash
        python dot2png.py example.dot output.png
        ```

        Эта команда создаст PNG-файл с именем 'output.png' из графа, определенного в 'example.dot'.
    """
```

**Как работает функция**:

1.  **Чтение DOT-файла**: Открывает и считывает содержимое DOT-файла.
2.  **Создание объекта Source**: Создает объект `Source` из содержимого DOT-файла, используя библиотеку `graphviz`.
3.  **Рендеринг в PNG**: Устанавливает формат выходного файла как PNG и выполняет рендеринг графа в указанный PNG-файл.
4.  **Обработка исключений**: Перехватывает исключения, такие как `FileNotFoundError`, если файл не найден, и общие исключения для обработки ошибок во время преобразования.

```
Чтение DOT-файла --> Создание объекта Source --> Рендеринг в PNG --> Обработка исключений
```

```
A: Открытие DOT-файла и чтение содержимого
|
B: Создание объекта Source из содержимого DOT-файла
|
C: Установка формата вывода как PNG
|
D: Рендеринг графа в PNG-файл
|
E: Обработка исключения FileNotFoundError (файл не найден)
|
F: Обработка общих исключений при конвертации
```

**Примеры**:

```python
dot2png('example.dot', 'output.png')
```

В этом примере DOT-файл с именем `example.dot` будет преобразован в PNG-изображение с именем `output.png`.
