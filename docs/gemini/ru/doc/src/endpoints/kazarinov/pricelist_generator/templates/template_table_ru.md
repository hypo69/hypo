# Шаблон таблицы прайслиста (ru)

## Обзор

Данный HTML-шаблон предназначен для генерации таблицы прайслиста.  Шаблон динамически заполняется данными о продуктах, включая название, описание, спецификацию, изображение и цену.  Результат –  красиво оформленный прайс-лист с информацией о каждом продукте.

## Структура

Шаблон использует HTML-структуру с таблицей для отображения карточек товаров.  Каждая карточка продукта содержит изображение, заголовок, краткое описание и цену.  В нижней части отображается общая стоимость прайслиста.


## Теги и элементы

### `<html>`

Основной элемент HTML-документа.

### `<head>`

Содержит мета-теги, заголовок страницы, и стили.

### `<style>`

Описание стилей для отображения таблицы. Включает стили для фона, текста, заголовков, карточек товаров, изображений, описаний, цен и подписи общей стоимости.

### `<body>`

Содержит контент страницы.

### `<div class="container">`

Контейнер, содержащий элементы, для организации и форматирования контента.

### `<h1>`

Заголовок страницы, выровненный по центру.

### `<p class="lead">`

Краткое описание прайслиста, выровненное по центру.

### `<table>`

Таблица для отображения карточек товаров.

### `<tr>`

Строка таблицы, представляющая карточку продукта.

### `<td>`

Ячейка таблицы.  Используется для размещения изображения, заголовка, описания и спецификации товара.

### `<h3>`

Заголовок продукта, размещённый в ячейке таблицы.

### `<table>` Внутри `<tr>`

Вложенная таблица для размещения изображения слева и описания справа от него.

### `<img>`

Тег для отображения изображения продукта.

### `<p>`

Теги для отображения описания продукта и его спецификации.

### `<span class="price-tag">`

Элемент для отображения цены товара.  Используется для выделения цены.

### `<div class="footer">`

Элемент для отображения общей стоимости прайслиста в нижней части страницы.

## Переменные шаблона (Вставляемые данные)

Шаблон использует следующие переменные, которые должны быть переданы при рендеринге:

* `{{ title }}`: Заголовок прайслиста.
* `{{ description }}`: Краткое описание прайслиста.
* `{{ products }}`: Список продуктов.  Каждый продукт должен иметь следующие поля:
    * `product_title`: Название продукта.
    * `image_local_saved_path`: Путь к изображению продукта на сервере.
    * `product_description`: Описание продукта.
    * `specification`: Спецификация продукта.
* `{{ price }}`: Общая стоимость.
* `{{ currency }}`: Валюта.

## Пример использования

```html
<!-- Пример использования шаблона -->
{% extends "base.html" %}

{% block content %}
    {% load static %}

    {% include "pricelist_generator/template_table_ru.html" with title="Прайс-лист на 2024 год" description="Список товаров с ценами" products=products price=total_price currency="руб." %}
{% endblock %}
```

Этот пример показывает как использовать шаблон `template_table_ru.html`, передавая необходимые данные, такие как список продуктов `products`, общую стоимость `total_price` и прочие значения.

##  CSS-стили

Подробное описание стилей можно найти в теге `<style>`.  Стиль отвечает за визуальное форматирование таблицы прайслиста, включая оформление карточек товаров, заголовков, описаний и цены.
