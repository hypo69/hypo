# Модуль `hypotez/src/utils/convertors`

## Обзор

Модуль `src.utils.convertors` предоставляет набор функций для конвертации различных форматов данных, включая работу с CSV, JSON, XML, HTML, Markdown, Base64, изображениями и текстом. Он содержит утилиты для преобразования данных в словари, списки, форматы для работы с таблицами и т.д.  Модуль поддерживает конвертацию в различные форматы, включая CSV, JSON, XLSX, XML, а также работу с изображениями (PNG, WebP) и текстовой речью.

## Функции

### `csv2dict`

**Описание**: Преобразует данные из файла CSV в словарь.

**Параметры**:
- `filepath` (str): Путь к файлу CSV.

**Возвращает**:
- `dict`: Словарь, содержащий данные из файла CSV. Возвращает `None`, если файл не найден или возникла ошибка.

**Вызывает исключения**:
- `FileNotFoundError`: Если файл CSV не найден.
- `Exception`: При других ошибках при чтении или обработке данных.


### `csv2ns`

**Описание**: Преобразует данные из файла CSV в пространство имен.

**Параметры**:
- `filepath` (str): Путь к файлу CSV.

**Возвращает**:
- `dict`: Словарь, содержащий данные из файла CSV в формате пространства имен. Возвращает `None`, если файл не найден или возникла ошибка.

**Вызывает исключения**:
- `FileNotFoundError`: Если файл CSV не найден.
- `Exception`: При других ошибках при чтении или обработке данных.


### `dict2ns`

**Описание**: Преобразует словарь в пространство имен.

**Параметры**:
- `data` (dict): Словарь, который нужно преобразовать.

**Возвращает**:
- `dict`: Словарь, содержащий данные в формате пространства имен. Возвращает `None`, если входные данные не являются словарем или возникла ошибка.


### `dict2csv`

**Описание**: Преобразует словарь в данные в формате CSV.

**Параметры**:
- `data` (dict): Словарь, который нужно преобразовать.
- `filepath` (str): Путь к файлу, куда будут записаны данные CSV.


**Возвращает**:
- `bool`: True, если запись прошла успешно, False в противном случае.


**Вызывает исключения**:
- `Exception`: При других ошибках при записи.


### `dict2html`

**Описание**: Преобразует словарь в HTML.

**Параметры**:
- `data` (dict): Словарь, который нужно преобразовать.

**Возвращает**:
- `str`: Строка с HTML-представлением данных.


**Вызывает исключения**:
- `Exception`: При других ошибках при преобразовании.


### `dict2xls`

**Описание**: Преобразует словарь в данные в формате XLSX.

**Параметры**:
- `data` (dict): Словарь, который нужно преобразовать.
- `filepath` (str): Путь к файлу, куда будут записаны данные XLSX.


**Возвращает**:
- `bool`: True, если запись прошла успешно, False в противном случае.


**Вызывает исключения**:
- `Exception`: При других ошибках при записи.


### `dict2xml`

**Описание**: Преобразует словарь в XML.

**Параметры**:
- `data` (dict): Словарь, который нужно преобразовать.

**Возвращает**:
- `str`: Строка с XML-представлением данных.


**Вызывает исключения**:
- `Exception`: При других ошибках при преобразовании.


### `replace_key_in_dict`

**Описание**: Заменяет ключ в словаре.

**Параметры**:
- `data` (dict): Словарь, в котором нужно заменить ключ.
- `old_key` (str): Старое имя ключа.
- `new_key` (str): Новое имя ключа.


**Возвращает**:
- `dict`: Словарь с измененным ключом.


**Вызывает исключения**:
- `KeyError`: Если ключ не найден.
