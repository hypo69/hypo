# Модуль `hypotez/src/endpoints/advertisement/facebook/scenarios/login.py`

## Обзор

Данный модуль содержит сценарий авторизации на Facebook. Он предоставляет функцию `login`, которая использует переданный объект драйвера для заполнения формы входа и нажатия кнопки входа.

## Функции

### `login`

**Описание**: Выполняет вход на Facebook. Использует переданный драйвер для взаимодействия с веб-элементами и выполняет вход с использованием предоставленных учетных данных.

**Параметры**:

- `d` (Driver): Экземпляр драйвера для взаимодействия с веб-элементами.

**Возвращает**:

- `bool`: `True`, если авторизация прошла успешно, иначе `False`.

**Вызывает исключения**:

- `Exception`: Если возникает ошибка при вводе логина, пароля или нажатии кнопки.

**Детали реализации**:

1. Получает учетные данные из `gs.facebook_credentials[0]`.
2. Посылает логин в элемент `locators.email`.
3. Ожидает 1.3 секунды.
4. Посылает пароль в элемент `locators['password']`.
5. Ожидает 0.5 секунды.
6. Нажимает кнопку входа (`locators['button']`).
7. Возвращает `True`, если все шаги прошли успешно, иначе `False`.
8. Обрабатывает возможные исключения с помощью блоков `try...except`.

## Локаторы

Модуль использует локаторы, загруженные из файла `login.json` в папке `locators`.  Файл `login.json` должен содержать необходимые локаторы для элементов формы входа Facebook.  Если файл не найден или некорректно сформирован, выводится соответствующее сообщение в лог.

## Использование

Для использования функции `login` необходимо:

1. Импортировать модуль `login`.
2. Создать экземпляр `Driver`.
3. Вызвать функцию `login`, передав созданный экземпляр драйвера.

Пример:

```python
from hypotez.src.endpoints.advertisement.facebook.scenarios import login
from src.webdriver.driver import Driver

# ... (Создание экземпляра Driver) ...

if login(driver):
    print("Вход выполнен успешно!")
else:
    print("Ошибка входа.")
```
