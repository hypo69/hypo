## Функции

### `set_project_root`

**Описание**: Находит корневой каталог проекта, начиная с каталога текущего файла, ища вверх по дереву каталогов, останавливаясь на первом каталоге, содержащем любой из файлов-маркеров.

**Параметры**:

-   `marker_files` (tuple, optional): Кортеж имен файлов или каталогов, используемых для идентификации корня проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

-   `Path`: Путь к корневому каталогу, если он найден, иначе — каталог, где расположен скрипт.

## Переменные

### `MODE`

**Описание**: Определяет режим работы приложения. По умолчанию установлено значение `'dev'`.

### `__root__`

**Описание**: Путь к корневой директории проекта. Инициализируется функцией `set_project_root`.

### `settings`

**Описание**: Словарь с настройками, загруженный из файла `settings.json`. Может быть `None`, если файл не найден или возникла ошибка при его чтении.

### `doc_str`

**Описание**: Строка с содержимым файла `README.MD`. Может быть `None`, если файл не найден или возникла ошибка при его чтении.

### `__project_name__`

**Описание**: Название проекта, полученное из файла `settings.json`. Если файла или ключа нет, по умолчанию используется `'hypotez'`.

### `__version__`

**Описание**: Версия проекта, полученная из файла `settings.json`. Если файла или ключа нет, по умолчанию используется пустая строка `''`.

### `__doc__`

**Описание**: Строка с содержимым файла `README.MD`. Если файл не найден, то используется пустая строка `''`.

### `__details__`

**Описание**: Детали проекта. В данном случае это пустая строка `''`.

### `__author__`

**Описание**: Имя автора проекта, полученное из файла `settings.json`. Если файла или ключа нет, по умолчанию используется пустая строка `''`.

### `__copyright__`

**Описание**: Информация об авторских правах проекта, полученная из файла `settings.json`. Если файла или ключа нет, по умолчанию используется пустая строка `''`.

### `__cofee__`

**Описание**: Сообщение с предложением поддержать разработчика, полученное из файла `settings.json`. Если файла или ключа нет, по умолчанию используется "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69".
