# Модуль hypotez/src/suppliers/hb/login.py

## Обзор

Данный модуль содержит функцию для авторизации поставщика (supplier).

## Функции

### `login`

**Описание**: Функция осуществляет логин поставщика.

**Параметры**:

- `s` (Supplier): Объект, представляющий поставщика.

**Возвращает**:

- `bool`: `True`, если логин успешен, иначе `False`.
