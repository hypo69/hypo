# ISO Country Codes

## Обзор

Файл `iso-country-codes.json` содержит массив объектов, представляющих страны и их соответствующие ISO-коды, регионы, подрегионы и другие географические данные. Этот файл используется для сопоставления стран с их уникальными идентификаторами и региональной принадлежностью.

## Содержание

- [Структура данных](#структура-данных)
- [Описание полей](#описание-полей)

## Структура данных

Файл представляет собой JSON-массив, где каждый элемент массива является объектом, содержащим данные о конкретной стране.

```json
[
  {
    "name": "Afghanistan",
    "alpha-2": "AF",
    "alpha-3": "AFG",
    "country-code": "004",
    "iso_3166-2": "ISO 3166-2:AF",
    "region": "Asia",
    "sub-region": "Southern Asia",
    "intermediate-region": "",
    "region-code": "142",
    "sub-region-code": "034",
    "intermediate-region-code": ""
  },
  {
    "name": "Åland Islands",
    "alpha-2": "AX",
    "alpha-3": "ALA",
    "country-code": "248",
    "iso_3166-2": "ISO 3166-2:AX",
    "region": "Europe",
    "sub-region": "Northern Europe",
    "intermediate-region": "",
    "region-code": "150",
    "sub-region-code": "154",
    "intermediate-region-code": ""
  },
  ...
]
```

## Описание полей

Каждый объект в массиве имеет следующие поля:

### `name`

**Описание**: Полное название страны.
**Тип**: `str`

### `alpha-2`

**Описание**: Двухбуквенный ISO 3166-1 альфа-2 код страны.
**Тип**: `str`

### `alpha-3`

**Описание**: Трехбуквенный ISO 3166-1 альфа-3 код страны.
**Тип**: `str`

### `country-code`

**Описание**: Цифровой ISO 3166-1 код страны.
**Тип**: `str`

### `iso_3166-2`

**Описание**: ISO 3166-2 код страны.
**Тип**: `str`

### `region`

**Описание**: Название региона, к которому относится страна.
**Тип**: `str`

### `sub-region`

**Описание**: Название подрегиона, к которому относится страна.
**Тип**: `str`

### `intermediate-region`

**Описание**: Название промежуточного региона, к которому относится страна (если применимо).
**Тип**: `str`

### `region-code`

**Описание**: Цифровой код региона.
**Тип**: `str`

### `sub-region-code`

**Описание**: Цифровой код подрегиона.
**Тип**: `str`

### `intermediate-region-code`
**Описание**: Цифровой код промежуточного региона (если применимо).
**Тип**: `str`
