# Модуль `drive.py`

## Обзор

Модуль `drive.py` предоставляет минимальную библиотеку для работы с Google Drive. Он позволяет загружать файлы в указанную папку на Google Drive, используя API Google Drive v3.

    *   [main](#main)

## Классы

### `GoogleDriveHandler`

**Описание**: Класс для управления взаимодействием с Google Drive.

**Методы**:

*   `__init__(self, folder_name: str)`: Конструктор класса.
*   `_create_credentials(self)`: Получение действительных учетных данных пользователя из хранилища.
*   `upload_file(self, file_path: Path)`: Загрузка файла в указанную папку на Google Drive.

#### `__init__`
   
**Описание**: Инициализирует экземпляр `GoogleDriveHandler`.

**Параметры**:

*   `folder_name` (str): Имя папки в Google Drive, в которую будут загружаться файлы.

**Возвращает**:
 -  `None`

#### `_create_credentials`
   
**Описание**: Получает действительные учетные данные пользователя из хранилища или создает их при необходимости.

**Параметры**:

*   `self` (GoogleDriveHandler): Экземпляр класса.

**Возвращает**:
* `Credentials`:  Объект учетных данных.

#### `upload_file`

**Описание**: Загружает файл в указанную папку на Google Drive.

**Параметры**:

*   `file_path` (Path): Путь к файлу, который нужно загрузить.
*   `self` (GoogleDriveHandler): Экземпляр класса.

**Возвращает**:
 - `None`
