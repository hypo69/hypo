# Модуль `openai`

## Обзор

Модуль `openai` предоставляет функциональность для работы с моделью OpenAI.
Он включает в себя возможности для перевода текста и использования моделей OpenAI.

## Функции

### `translate`

**Описание**:
Функция для перевода текста с использованием модели OpenAI.

**Параметры**:
- `text` (str): Текст, который нужно перевести.
- `target_language` (str): Язык, на который нужно перевести текст.
- `source_language` (Optional[str], optional): Исходный язык текста. По умолчанию `None`.

**Возвращает**:
- `str | None`: Переведенный текст или `None`, если перевод не удался.

## Классы

### `OpenAIModel`

**Описание**:
Класс для работы с моделями OpenAI.
Предоставляет интерфейс для взаимодействия с API OpenAI.

**Методы**:
- `__init__`: Метод инициализации класса.
- `generate`: Метод для генерации текста с использованием модели OpenAI.
- `get_model_info`: Метод для получения информации о модели.
- `get_model_list`: Метод для получения списка доступных моделей.
