# Модуль `hypotez/src/category/header.py`

## Обзор

Модуль `hypotez/src/category/header.py` определяет корневой путь к проекту `hypotez`.  Все импорты строятся относительно этого пути. В дальнейшем этот модуль планируется перенести в системную переменную.


## Функции

### `set_project_root`

**Описание**:  Находит корневую директорию проекта, начиная с директории текущего файла. Поиск происходит вверх по древу директорий, пока не будет найдена директория, содержащая один из указанных файлов-маркеров.

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для определения корневой директории проекта. По умолчанию: `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если она найдена. В противном случае возвращает директорию, где расположен скрипт.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта. Определяется функцией `set_project_root()`.

**Тип**: `Path`


## Обработка файлов настроек

### Чтение файла настроек `settings.json`

**Описание**: Читает файл `settings.json` в директории `src` корневого каталога проекта, загружая настройки в словарь `settings`.

**Вызывает исключения**:

- `FileNotFoundError`: Если файл `settings.json` не найден.
- `json.JSONDecodeError`: Если файл `settings.json` содержит некорректный JSON.


### Чтение файла `README.MD`

**Описание**: Читает файл `README.MD` в директории `src` корневого каталога проекта, сохраняя его содержимое в переменной `doc_str`.

**Вызывает исключения**:

- `FileNotFoundError`: Если файл `README.MD` не найден.
- `json.JSONDecodeError`: Если файл `README.MD` содержит некорректный формат.


## Переменные проекта

### `__project_name__`

**Описание**: Название проекта, полученное из настроек. По умолчанию `hypotez`.


### `__version__`

**Описание**: Версия проекта, полученная из настроек. По умолчанию пустая строка.


### `__doc__`

**Описание**: Документация проекта, полученная из файла `README.MD`. По умолчанию пустая строка.


### `__details__`

**Описание**: Дополнительные подробности о проекте (значение по умолчанию пустое).


### `__author__`

**Описание**: Автор проекта, полученный из настроек. По умолчанию пустая строка.


### `__copyright__`

**Описание**: Авторские права проекта, полученные из настроек. По умолчанию пустая строка.


### `__cofee__`

**Описание**: Ссылка на поддержку разработчика через платёжную систему boosty.to/hypo69.
