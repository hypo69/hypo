# Документация для `kualastyle_categories_lighting.json`

## Обзор

Этот файл содержит JSON-представление категории товаров "תאורה" (Освещение) для Kualastyle. Он определяет, что категория имеет подкатегории и не содержит сценариев.

## Оглавление

- [Обзор](#обзор)
- [Описание структуры JSON](#описание-структуры-json)

## Описание структуры JSON

### `category name on site`

**Описание**: Название категории на сайте.

**Тип**: `string`

**Значение**: `"תאורה"` (Освещение)

### `have subcategories`

**Описание**: Флаг, указывающий на наличие подкатегорий у данной категории.

**Тип**: `boolean`

**Значение**: `true` (категория имеет подкатегории)

### `scenarios`

**Описание**: Объект, содержащий сценарии для данной категории. В данном случае, он пуст.

**Тип**: `object`

**Значение**: `{}` (пустой объект, нет сценариев)
