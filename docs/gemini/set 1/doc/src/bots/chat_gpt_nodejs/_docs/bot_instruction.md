# ИНСТРУКЦИЯ для бота ChatGPT NodeJS

## Обзор

Этот документ содержит инструкции для бота ChatGPT NodeJS. Он описывает доступные команды бота, их параметры и ожидаемые результаты.

## Команды

### `!hi`

**Описание**: Приветствует пользователя.

**Синтаксис**: `!hi`

**Возвращает**: Текстовое приветствие.


### `!train`

**Описание**: Обучает модель с предоставленными данными.

**Синтаксис**: `!train <data> <data_dir> <positive> <attachment>`

**Параметры**:

- `<data>` (строка): Путь к файлу с данными для обучения.
- `<data_dir>` (строка): Путь к директории с данными для обучения.
- `<positive>` (строка): Флаг для обозначения положительного класса данных (требуется для задач классификации).
- `<attachment>` (строка): Путь к прикрепленному файлу с данными для обучения.

**Возвращает**: Сообщение об успешном/неудачном выполнении обучения.


### `!test`

**Описание**: Тестирует модель с предоставленными данными.

**Синтаксис**: `!test <test_data>`

**Параметры**:

- `<test_data>` (строка): Путь к JSON файлу с тестовыми данными.

**Возвращает**: Результаты тестирования модели в формате JSON.


### `!archive`

**Описание**: Архивирует файлы в указанной директории.

**Синтаксис**: `!archive <directory>`

**Параметры**:

- `<directory>` (строка): Путь к директории с файлами для архивирования.

**Возвращает**: Сообщение об успешном/неудачном выполнении архивирования.


### `!select_dataset`

**Описание**: Выбирает набор данных для обучения из указанной директории.

**Синтаксис**: `!select_dataset <path_to_dir_positive> <positive>`

**Параметры**:

- `<path_to_dir_positive>` (строка): Путь к директории с положительными примерами данных.
- `<positive>` (строка): Флаг для выбора положительного класса (требуется для задач классификации).

**Возвращает**: Сообщение об успешном выборе набора данных.


### `!instruction`

**Описание**: Выводит это сообщение с инструкциями.

**Синтаксис**: `!instruction`

**Возвращает**:  Это сообщение с инструкциями.
