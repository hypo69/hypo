# Модуль `hypotez/src/goog/spreadsheet/bberyakov/helpers.py`

Этот модуль содержит вспомогательные функции для работы с цветами, представленными в различных форматах (HEX, десятичное, RGB).

## Функции

**`hex_color_to_decimal(letters: str) -> int`**

Преобразует цвет из шестнадцатеричного (HEX) формата (например, "A", "AB", "FF") в десятичный формат.

* **`letters`:** Строка, представляющая цвет в HEX формате (например, "A", "AB", "FF").  Необходимо указать, что входные данные строкового типа.
* **Возвращаемое значение:** Целое число, представляющее цвет в десятичном формате.

**Пример использования:**

```python
print(hex_color_to_decimal("A"))  # Output: 10
print(hex_color_to_decimal("AB")) # Output: 171
```

**`decimal_color_to_hex(number: int) -> str`**

Преобразует цвет из десятичного формата в шестнадцатеричный (HEX).

* **`number`:** Целое число, представляющее цвет в десятичном формате.
* **Возвращаемое значение:** Строка, представляющая цвет в HEX формате.


**Пример использования:**

```python
print(decimal_color_to_hex(10))  # Output: A
print(decimal_color_to_hex(171)) # Output: AB
```

**`hex_to_rgb(hex: str) -> tuple`**

Преобразует цвет из шестнадцатеричного (HEX) формата в RGB формат.

* **`hex`:** Строка, представляющая цвет в HEX формате (например, "#FFFFFF", "FFFFFF").
* **Возвращаемое значение:** Кортеж из трех целых чисел, представляющих значения красного, зеленого и синего цветов в RGB формате (например, (255, 255, 255)). Добавлены проверка на наличие символа "#" и обработка случая, когда он отсутствует.

**Пример использования:**

```python
print(hex_to_rgb("#FFFFFF")) # Output: (255, 255, 255)
print(hex_to_rgb("FFFFFF")) # Output: (255, 255, 255)
```


##  Комментарии

* Код содержит многочисленные недописанные docstrings. Необходимо добавить более подробные описания параметров, возвращаемых значений и возможных исключений.
* Необходимо уточнить, какой формат входных данных ожидается в функциях. (Например, в `hex_color_to_decimal` предполагается, что входной HEX-код состоит из одной или двух букв,  однако нет явного указания на это в документации).
* В `decimal_color_to_hex`,  функция предполагает ввод числа от 1 до 26, но не делает это явным.
* Необходимо добавить обработку ошибок. Например, в `hex_to_rgb`, если введенная строка не является допустимым HEX-кодом, функция должна генерировать исключение.
