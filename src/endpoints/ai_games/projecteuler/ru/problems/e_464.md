# Задача No 464
## Множество странных чисел

## Условия

Определение:

Назовем положительное целое число странным, если сумма его собственных делителей (делителей, не включая само число) строго больше самого числа.

Первыми несколькими странными числами являются 12, 18, 20, 24, 30, 36, ...

Последующие странные числа могут быть найдены путем генерации всех чисел, которые могут быть записаны в виде суммы по крайней мере двух различных странных чисел, но эта процедура не может генерировать все странные числа.

Пусть S(n) - множество всех странных чисел, не превышающих n.
Пусть A(n) - множество всех целых чисел, которые могут быть записаны в виде суммы по крайней мере двух различных странных чисел из S(n).

Например, S(100) = {12, 18, 20, 24, 30, 36, 40, 42, 48, 54, 56, 60, 66, 70, 72, 78, 80, 84, 88, 90, 96}.
A(100) содержит, например, 24 = 12 + 12, 30 = 12 + 18, 32 = 12 + 20, 36 = 12 + 24, 38 = 18 + 20, ...
Некоторые целые числа, такие как 1, 2, 3, ... 11, 13, ... 17, ... не могут быть записаны как суммы по крайней мере двух различных странных чисел.

Пусть B(n) - множество всех положительных целых чисел, не превышающих n, которые не могут быть записаны как суммы по крайней мере двух различных странных чисел из S(n).

Например, B(100) содержит 1, 2, 3, ... 11, 13, 14, 15, 16, 17, 19, 21, 22, 23, 25, 26, 27, 28, 29, 31, 33, 34, 35, 37, 39, 41, 43, 44, 45, 46, 47, 49, 50, 51, 52, 53, 55, 57, 58, 59, 61, 62, 63, 64, 65, 67, 68, 69, 71, 73, 74, 75, 76, 77, 79, 81, 82, 83, 85, 86, 87, 89, 91, 92, 93, 94, 95, 97, 98, 99.

Пусть Sum(n) - сумма всех элементов в B(n).

Например, Sum(100) = 1 + 2 + 3 + ... + 11 + 13 + 14 + ... + 99 = 4050.
Также известно, что Sum(1000) = 316072.

Найдите Sum(10000).
