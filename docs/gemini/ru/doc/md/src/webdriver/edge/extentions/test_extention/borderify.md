# Модуль borderify.js

## Обзор

Этот скрипт изменяет стиль границы (`border`) элемента `body` на красную линию толщиной 5 пикселей и выводит всплывающее окно `alert()`.

## Функции

### `(anonymous)`

**Описание**: Анонимная функция, которая, скорее всего, выполняется при загрузке страницы.

**Действия**:
- Устанавливает свойство `border` элемента `document.body` со значением "5px solid red".
- Вызывает функцию `alert()`, которая отображает всплывающее окно с сообщением.

**Всплывающие окна**:
 - Функция `alert()` отображает всплывающее окно с сообщением.


**Примечания**:  Код не содержит явных параметров или возвращаемых значений, но выполняет заданные действия.
