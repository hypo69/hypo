# Модуль `hypotez/src/suppliers/wallashop/graber.py`

## Обзор

Модуль `hypotez/src/suppliers/wallashop/graber.py` содержит класс `Graber`, предназначенный для извлечения данных о товарах с сайта `wallashop.co.il`. Класс наследуется от базового класса `Grbr` и предоставляет асинхронные методы для извлечения различных полей товара.  Класс реализует  обработку полей страницы товара, предоставляя возможность переопределения стандартной обработки для нестандартных случаев.  В модуле также определены предварительные действия, которые можно совершить перед запросом к веб-драйверу с помощью декоратора `close_pop_up`.
