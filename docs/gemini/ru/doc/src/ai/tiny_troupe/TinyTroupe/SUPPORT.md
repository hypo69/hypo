# Модуль SUPPORT

## Обзор

Данный модуль содержит информацию о том, как отправить запрос поддержки, получить помощь и политике поддержки Microsoft для проекта TinyTroupe.

## Инструкции по подаче отчетов об ошибках и получению помощи

Этот проект использует GitHub Issues для отслеживания багов и запросов на новые функции. Перед тем, как создать новый отчет об ошибке, пожалуйста, проверьте существующие отчеты, чтобы избежать дублирования. Для новых отчетов об ошибках или запросов на новые функции создайте новую Issue.

Для получения помощи и ответов на вопросы об использовании этого проекта, пожалуйста, обратитесь к нашей Википедии или опубликуйте вопрос на нашем форуме обсуждений, оба расположены здесь же, на GitHub.

## Политика поддержки Microsoft

Поддержка TinyTroupe ограничена ресурсами, перечисленными выше.
