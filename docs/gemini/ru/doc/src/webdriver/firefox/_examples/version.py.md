# Модуль `version.py`

## Обзор

Модуль содержит информацию о версии, имени, документации, деталях, аннотациях и авторе для модуля.

## Оглавление

   - [`__version__`](#__version__)
   - [`__doc__`](#__doc__)
   - [`__details__`](#__details__)
   - [`__annotations__`](#__annotations__)
   - [`__author__`](#__author__)

## Переменные

### `__name__`

**Описание**: Содержит имя модуля. Если скрипт запускается напрямую, значение будет `"__main__"`.

### `__version__`

**Описание**: Содержит версию модуля или пакета.

**Тип**: `str`

**Значение**: `"3.12.0.0.0.4"`

### `__doc__`

**Описание**: Строка документации модуля.

**Тип**: `str`

### `__details__`

**Описание**: Содержит дополнительные сведения о модуле.

**Тип**: `str`

**Значение**: `"Details about version for module or class"`

### `__annotations__`

**Описание**: Содержит аннотации типов для переменных и функций в модуле.

**Тип**: `dict`

### `__author__`

**Описание**: Содержит имя или имена авторов модуля.

**Тип**: `str`

**Значение**: `'hypotez'`
