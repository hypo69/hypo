# Модуль hypotez/src/gui/openai_trаigner/version.py

## Обзор

Этот модуль содержит информацию о версии проекта, а также лицензию и контактную информацию.

## Переменные

### `MODE`

**Описание**: Переменная, хранящая режим работы приложения ('dev' в данном случае).

### `__version__`

**Описание**: Строка, представляющая версию проекта.

### `__doc__`

**Описание**: Строка, содержащая документацию к модулю (пустая в данном случае).

### `__details__`

**Описание**: Строка, содержащая дополнительные детали (пустая в данном случае).

### `__author__`

**Описание**: Строка, содержащая имя автора.

### `__copyright__`

**Описание**: Строка, содержащая информацию о лицензии проекта. Подробная информация о лицензии доступна по ссылке.

### `__cofee__`

**Описание**: Строка, содержащая ссылку на страницу для пожертвований на кофе разработчику.
