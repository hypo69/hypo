# Модуль `header.py`

## Обзор

Модуль `header.py` предназначен для определения корневой директории проекта, загрузки настроек из файла `settings.json`, чтения документации из `README.MD` и инициализации основных переменных проекта.

## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с директории текущего файла, и останавливается на первой директории, содержащей маркерные файлы.

**Параметры**:
- `marker_files` (tuple): Кортеж имен файлов или директорий для идентификации корневой директории проекта. По умолчанию `('pyproject.toml', 'requirements.txt', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена. В противном случае, путь к директории, где находится скрипт.

## Переменные

### `__root__`
**Описание**: Путь к корневой директории проекта. Инициализируется функцией `set_project_root`.

### `settings`
**Описание**: Словарь с настройками проекта, загруженными из `settings.json`. Если файл не найден или не является корректным JSON, значение будет `None`.

### `doc_str`
**Описание**: Строка с содержимым файла `README.MD`. Если файл не найден, значение будет `None`.

### `__project_name__`
**Описание**: Название проекта. Берется из `settings.json` под ключом `project_name`, если `settings` не `None`, иначе по умолчанию `hypotez`.

### `__version__`
**Описание**: Версия проекта. Берется из `settings.json` под ключом `version`, если `settings` не `None`, иначе по умолчанию пустая строка.

### `__doc__`
**Описание**: Документация проекта, загруженная из `README.MD`. Если `doc_str` не `None`, то значение  `doc_str`, иначе по умолчанию пустая строка.

### `__details__`
**Описание**: Дополнительные сведения о проекте. По умолчанию пустая строка.

### `__author__`
**Описание**: Автор проекта. Берется из `settings.json` под ключом `author`, если `settings` не `None`, иначе по умолчанию пустая строка.

### `__copyright__`
**Описание**: Информация об авторских правах проекта. Берется из `settings.json` под ключом `copyrihgnt`, если `settings` не `None`, иначе по умолчанию пустая строка.

### `__cofee__`
**Описание**: Сообщение с предложением поддержать разработчика. Берется из `settings.json` под ключом `cofee`, если `settings` не `None`, иначе по умолчанию "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69".
