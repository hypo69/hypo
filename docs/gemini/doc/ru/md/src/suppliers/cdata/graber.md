# Модуль hypotez/src/suppliers/cdata/graber.py

## Обзор

Модуль `hypotez/src/suppliers/cdata/graber.py` содержит класс `Graber`, предназначенный для сбора данных о товарах со страницы `cdata.co.il`.  Класс наследуется от базового класса `Grbr` и предоставляет асинхронные методы для извлечения различных полей товара.  Реализована обработка полей товара, а также возможность использования пользовательских декораторов для предварительной обработки (закрытия всплывающих окон).

## Классы

### `Graber`

**Описание**:  Класс `Graber` отвечает за сбор данных о товаре с сайта `cdata.co.il`. Он содержит методы для извлечения различных полей товара, используя веб-драйвер.

**Атрибуты**:
- `supplier_prefix`: Префикс поставщика (`cdata`).

**Методы**:

#### `__init__(self, driver: Driver)`

**Описание**: Инициализирует экземпляр класса `Graber`.

**Параметры**:
- `driver (Driver)`: Экземпляр класса `Driver`, используемый для взаимодействия с веб-драйвером.

#### `grab_page(self, driver: Driver) -> ProductFields`

**Описание**: Асинхронный метод для сбора данных о товаре.

**Параметры**:
- `driver (Driver)`: Экземпляр класса `Driver` для взаимодействия с веб-драйвером.

**Возвращает**:
- `ProductFields`: Объект содержащий собранные данные о товаре.
