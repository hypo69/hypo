# הוראות למודול יצירת באנר טקסט

## סקירה כללית

מודול זה נועד ליצור באנר טקסט מעוצב מקלט טקסט נתון. המודול משתמש בכוכביות (`*`) כדי ליצור גבולות מסביב לטקסט.

## הוראות

### תיאור

המודול פועל לפי הכללים הבאים:

1.  השתמש בתו `*` לעיצוב הבאנר.
2.  הגבולות העליונים והתחתונים של הבאנר צריכים להיות מעוצבים באמצעות התו `*`.
3.  הטקסט צריך להיות מוקף במסגרת של תווים `*`.
4.  דוגמה:
    ```
    *********
    * טקסט *
    *********
    ```
5.  אין להוסיף הסברים נוספים, יש להחזיר רק את הבאנר המעוצב.
