# Модуль hypotez/src/suppliers/aliexpress/campaign/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта, загрузки настроек из файла `settings.json` и получения документации из `README.MD`. Он также предоставляет переменные, содержащие информацию о проекте (имя, версия, автор и др.).


## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная с текущей директории и двигаясь вверх по дереву директорий. Поиск останавливается на первой директории, содержащей файлы-маркеры (`pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или директорий, используемых для определения корневой директории.

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена, иначе - директория, в которой находится текущий скрипт.

**Вызывает исключения**:

- Нет.


## Переменные

### `__root__`

**Описание**: Путь к корневой директории проекта, полученный с помощью функции `set_project_root`.

**Тип**: `Path`

### `settings`

**Описание**: Словарь с настройками проекта, загруженный из файла `settings.json`.

**Тип**: `dict`

### `doc_str`

**Описание**: Строка с текстом документации, загруженная из файла `README.MD`.

**Тип**: `str`

### `__project_name__`

**Описание**: Имя проекта. Значение берется из `settings.json` (ключ `project_name`), по умолчанию - `hypotez`.

**Тип**: `str`

### `__version__`

**Описание**: Версия проекта. Значение берется из `settings.json` (ключ `version`), по умолчанию - пустая строка.

**Тип**: `str`

### `__doc__`

**Описание**: Документация проекта. Значение берется из `README.MD`, по умолчанию - пустая строка.

**Тип**: `str`

### `__details__`

**Описание**: Дополнительная информация о проекте (пустая по умолчанию).

**Тип**: `str`

### `__author__`

**Описание**: Автор проекта. Значение берется из `settings.json` (ключ `author`), по умолчанию - пустая строка.

**Тип**: `str`

### `__copyright__`

**Описание**: Авторские права. Значение берется из `settings.json` (ключ `copyrihgnt`), по умолчанию - пустая строка. Обратите внимание на опечатку в имени ключа в файле.

**Тип**: `str`

### `__cofee__`

**Описание**: Ссылка на страницу для поддержки разработчика. Значение берется из `settings.json` (ключ `cofee`), по умолчанию - ссылка на Boosty.

**Тип**: `str`
