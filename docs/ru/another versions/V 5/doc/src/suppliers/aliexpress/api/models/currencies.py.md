# Модуль: src.suppliers.aliexpress.api.models.currencies

## Обзор

Модуль определяет класс `Currency`, который содержит константы для различных валют.

## Подробнее

Этот модуль предоставляет простой способ определения и использования кодов валют, поддерживаемых AliExpress. Он содержит класс `Currency`, атрибуты которого представляют собой строковые константы, соответствующие кодам валют. Этот модуль облегчает использование кодов валют в других частях проекта, связанных с AliExpress API.

## Классы

### `Currency`

**Описание**: Класс, содержащий константы для представления кодов валют.

**Как работает класс**:
Класс `Currency` предназначен для хранения кодов валют в виде статических атрибутов. Каждый атрибут класса представляет собой строковую константу, соответствующую определенной валюте. Это позволяет удобно использовать коды валют в других частях проекта, просто обращаясь к атрибутам класса.

**Методы**: Нет

**Параметры**: Нет

**Примеры**:

```python
# Импорт класса Currency
from src.suppliers.aliexpress.api.models.currencies import Currency

# Использование кодов валют
usd = Currency.USD
eur = Currency.EUR
print(f"Код доллара США: {usd}")
print(f"Код евро: {eur}")
```
