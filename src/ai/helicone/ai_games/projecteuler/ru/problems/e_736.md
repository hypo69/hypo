# Задача No 736
## Пути в графе

## Условия

Определим граф следующим образом:
*   Множество вершин — это все целые числа, которые строго больше 0.
*   Вершины *a* и *b* соединены ребром тогда и только тогда, когда *a* + *b* — это простое число.

Путь в графе — это последовательность вершин, где каждая пара соседних вершин соединена ребром.
Длина пути равна количеству ребер в нем.
Назовем *L*(*n*) минимальную длину пути между вершинами 1 и *n*.
Например, *L*(2) = 1, *L*(3) = 1, *L*(4) = 2, *L*(5) = 2, *L*(6) = 3.

Пусть *S*(*n*) = ∑ *L*(*k*) для 2 ≤ *k* ≤ *n*.
Например, *S*(10) = 1 + 1 + 2 + 2 + 3 + 2 + 3 + 3 + 4 = 21.

Найдите *S*(10<sup>7</sup>).
