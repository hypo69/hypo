# Модуль `currencies`

## Обзор

Модуль `currencies.py` определяет класс `Currency`, который содержит константы для различных валют, используемых в API AliExpress. Он предоставляет удобный способ ссылаться на валюты, используя атрибуты класса, вместо строк.

## Подробнее

Этот модуль используется для определения списка поддерживаемых валют. Каждая валюта представлена как статическая переменная внутри класса `Currency`. Это позволяет избежать использования "магических строк" в коде и обеспечивает централизованное место для управления списком валют.  Список валют может быть использован в других модулях проекта `hypotez` для валидации валютных значений, конвертации валют или для отображения списка поддерживаемых валют в пользовательском интерфейсе.

## Классы

### `Currency`

**Описание**: Класс `Currency` содержит статические атрибуты, представляющие собой коды валют.

**Принцип работы**: Класс предназначен для хранения статического набора констант, представляющих собой коды различных валют.  Он не имеет методов экземпляра и используется только для доступа к атрибутам класса, которые определяют коды валют.

**Атрибуты**:
- `USD` (str): Код доллара США ('USD').
- `GBP` (str): Код британского фунта ('GBP').
- `CAD` (str): Код канадского доллара ('CAD').
- `EUR` (str): Код евро ('EUR').
- `UAH` (str): Код украинской гривны ('UAH').
- `MXN` (str): Код мексиканского песо ('MXN').
- `TRY` (str): Код турецкой лиры ('TRY').
- `RUB` (str): Код российского рубля ('RUB').
- `BRL` (str): Код бразильского реала ('BRL').
- `AUD` (str): Код австралийского доллара ('AUD').
- `INR` (str): Код индийской рупии ('INR').
- `JPY` (str): Код японской йены ('JPY').
- `IDR` (str): Код индонезийской рупии ('IDR').
- `SEK` (str): Код шведской кроны ('SEK').
- `KRW` (str): Код южнокорейской воны ('KRW').
- `ILS` (str): Код израильского шекеля ('ILS').

**Примеры**
```python
# Импорт класса Currency
from src.suppliers.aliexpress.api.models.currencies import Currency

# Использование кодов валют
usd_code = Currency.USD
eur_code = Currency.EUR

print(f"USD Code: {usd_code}")
print(f"EUR Code: {eur_code}")
