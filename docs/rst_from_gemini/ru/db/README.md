## Связи между таблицами

* Пользователи могут создавать гипотезы (`user_id` в таблице `hypotheses`).
* Комментарии относятся к конкретным гипотезам (`hypothesis_id` в таблице `comments`).


## Типы данных

Используемые типы данных в базе данных (SQL):

* `INTEGER`: Целые числа.
* `VARCHAR`: Строковые значения.
* `TEXT`: Длинные текстовые значения.
* `DATETIME`: Дата и время.
* `FOREIGN KEY`: Связывает записи между таблицами.


## Особенности

* База данных разработана для хранения информации о гипотезах и комментариях.
