# Модуль hypotez/src/goog/spreadsheet/header.py

## Обзор

Этот модуль содержит функции для определения корневой директории проекта и загрузки настроек из файла `settings.json`. Он также предоставляет доступ к документации проекта (README.MD), если она существует.


## Функции

### `set_project_root`

**Описание**: Находит корневую директорию проекта, начиная от текущей директории и двигаясь вверх по дереву каталогов.  Останавливается на первой директории, содержащей один из указанных файлов-маркеров (например, `pyproject.toml`, `requirements.txt`, `.git`).

**Параметры**:

- `marker_files` (tuple): Кортеж имен файлов или каталогов, используемых для определения корневой директории. По умолчанию: (`'pyproject.toml'`, `'requirements.txt'`, `'.git'`).

**Возвращает**:

- `Path`: Путь к корневой директории проекта, если найдена, иначе путь к текущей директории.

**Вызывает исключения**:

- Нет


### (Встроенная документация для глобальных переменных)

**Описание**:
Эти глобальные переменные представляют собой свойства проекта, полученные из файла `settings.json` или имеющие значения по умолчанию.  Они должны быть заполнены в файле `settings.json`.

**Переменные:**

- `__root__` (Path): Путь к корневой директории проекта, полученный из функции `set_project_root`.
- `__project_name__` (str): Название проекта, полученное из `settings.json`. По умолчанию: `'hypotez'`.
- `__version__` (str): Версия проекта, полученная из `settings.json`. По умолчанию: `''`.
- `__doc__` (str): Текст документации проекта (README.MD), если файл найден. По умолчанию: `''`.
- `__details__` (str): Подробности о проекте (пустая строка по умолчанию).
- `__author__` (str): Автор проекта, полученный из `settings.json`. По умолчанию: `''`.
- `__copyright__` (str): Авторские права, полученные из `settings.json`. По умолчанию: `''`.
- `__cofee__` (str): Ссылка для пожертвований на чашку кофе разработчику. По умолчанию: ссылка.
