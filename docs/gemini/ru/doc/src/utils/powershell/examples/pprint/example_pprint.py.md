# Модуль `src.utils.powershell.examples.pprint`

## Обзор

Данный модуль предоставляет пример использования функций `pprint` для форматированного вывода данных.

## Содержание

- [Обзор](#обзор)
- [Импорты](#импорты)
- [Константы](#константы)
- [Использование функций pprint](#использование-функций-pprint)

## Импорты

Модуль импортирует следующие библиотеки:

- `header`
- `pprint` (переименованный в `pretty_print`) из стандартной библиотеки `pprint`
- `pprint` из `src.printer`

## Константы

### `MODE`

- **Описание**: Константа, устанавливающая режим работы модуля.
- **Значение**: `'dev'`

## Использование функций pprint

Пример использования функций форматированного вывода:

### `pprint("Hello, world!")`

- **Описание**: Выводит строку "Hello, world!" с помощью функции `pprint` из модуля `src.printer`.
