# Документация для background.js

## Обзор

Этот скрипт служит бэкграунд-скриптом для расширения браузера. Он обрабатывает клик по иконке расширения и запускает скрипт контентной страницы.

## Функции

### `browser.browserAction.onClicked.addListener`

**Описание**: Этот обработчик реагирует на клик по иконке расширения.

**Параметры**:

- `tab` (object): Объект, содержащий информацию о вкладке, на которой был нажат клик.

**Возвращает**:
-  Не применимо. Обработчик выполняет действия асинхронно.

**Описание работы**: При нажатии на иконку расширения, функция запускает скрипт контентной страницы с помощью `browser.scripting.executeScript`.
