# Модуль NIM

## Обзор

Данный модуль реализует игру NIM для двух игроков. Игроки по очереди берут камни из кучек, и выигрывает тот, кто возьмет последний камень.

    - [display_piles](#display_piles)
    - [get_player_move](#get_player_move)
    - [play_nim](#play_nim)

## Функции

### `display_piles`

**Описание**: Выводит текущее состояние кучек камней.

**Параметры**:
- `piles` (list): Список, представляющий кучки камней.

**Возвращает**:
- `None`

### `get_player_move`

**Описание**: Запрашивает у игрока номер кучки и количество камней для удаления.

**Параметры**:
- `piles` (list): Список, представляющий кучки камней.
- `player` (int): Номер текущего игрока (1 или 2).

**Возвращает**:
- `tuple`: Кортеж, содержащий номер кучки и количество камней для удаления.

**Вызывает исключения**:
- `ValueError`: Возникает, если пользователь вводит некорректные данные (не целые числа).

### `play_nim`

**Описание**: Реализует игру NIM.

**Параметры**:
- `None`

**Возвращает**:
- `None`
