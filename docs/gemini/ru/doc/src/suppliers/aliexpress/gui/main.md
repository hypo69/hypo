# Модуль hypotez/src/suppliers/aliexpress/gui/main.py

## Обзор

Данный модуль предоставляет основной интерфейс приложения для управления рекламными кампаниями на AliExpress. Он построен с использованием PyQt6 и организован в виде вкладок, каждая из которых отвечает за редактирование определенных данных: JSON-файлов, кампаний, продуктов и категорий. Модуль также реализует функциональность открытия, сохранения и копирования/вставки данных, обеспечивая удобство работы с приложением.

## Классы

### `MainApp`

**Описание**: Главный класс приложения, содержащий в себе основные элементы пользовательского интерфейса и логику работы с вкладками.

**Методы**:

- `__init__(self)`:
    **Описание**: Инициализирует основное приложение с вкладками для редактирования JSON, кампаний, продуктов и категорий. Устанавливает заголовок окна, размеры и создает виджет QTabWidget для вкладок.
    **Параметры**:
    - `self`: ссылка на экземпляр класса.
    **Возвращает**:
    - None
    
- `create_menubar(self)`:
    **Описание**: Создает меню приложения с пунктами "Файл" и "Правка". Пункты "Открыть", "Сохранить" и "Выход" отвечают за управление файлами. Пункты "Копировать" и "Вставить" реализуют стандартные функции буфера обмена.
    **Параметры**:
    - `self`: ссылка на экземпляр класса.
    **Возвращает**:
    - None

- `open_file(self)`:
    **Описание**: Открывает диалоговое окно для выбора JSON-файла. Загружает выбранный файл в соответствующую вкладку.
    **Параметры**:
    - `self`: ссылка на экземпляр класса.
    **Возвращает**:
    - None

- `save_file(self)`:
    **Описание**: Сохраняет изменения в активной вкладке.
    **Параметры**:
    - `self`: ссылка на экземпляр класса.
    **Возвращает**:
    - None
- `exit_application(self)`:
    **Описание**: Закрывает приложение.
    **Параметры**:
    - `self`: ссылка на экземпляр класса.
    **Возвращает**:
    - None
    
- `copy(self)`:
    **Описание**: Копирует выделенный текст в буфер обмена.
    **Параметры**:
    - `self`: ссылка на экземпляр класса.
    **Возвращает**:
    - None

- `paste(self)`:
    **Описание**: Вставляет текст из буфера обмена.
    **Параметры**:
    - `self`: ссылка на экземпляр класса.
    **Возвращает**:
    - None

- `load_file(self, campaign_file)`:
    **Описание**: Загружает JSON-файл в приложение.
    **Параметры**:
    - `self`: ссылка на экземпляр класса.
    - `campaign_file` (str): Путь к загружаемому файлу.
    **Возвращает**:
    - None
    **Возможные исключения**:
    - `Exception`: В случае ошибок при загрузке файла.


## Функции

### `main()`

**Описание**: Инициализирует приложение и запускает его главный цикл.

**Параметры**:

- None

**Возвращает**:

- None
