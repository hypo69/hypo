# Модуль `hypotez/src/endpoints/advertisement/facebook/scenarios/switch_account.py`

## Обзор

Этот модуль содержит функцию `switch_account`, предназначенную для переключения между аккаунтами в Facebook. Она взаимодействует с веб-драйвером для поиска и нажатия кнопки "Переключить".

## Оглавление

* [Переключение между аккаунтами](#переключение-между-аккаунтами)


## Функции

### `switch_account`

**Описание**: Функция находит и нажимает кнопку "Переключить аккаунт" в интерфейсе Facebook.

**Параметры**:

- `driver` (Driver): Объект веб-драйвера, используемый для взаимодействия с браузером.

**Возвращает**:

-  Не имеет возвращаемого значения.

**Вызывает исключения**:

-  Возможные исключения, связанные с взаимодействием с веб-драйвером, например, `NoSuchElementException` в случае отсутствия кнопки.
-  Возможны исключения, связанные с файлами локейторов, например при проблемах с чтением json-файла.


## Переключение между аккаунтами

Функция `switch_account` предназначена для выполнения следующего действия:
* Искать элемент на странице по ключу `switch_to_account_button` из файла `post_message.json`
* Если элемент найден, нажать на него.

**Примечание**:  Данная функция предполагает наличие корректного файла `post_message.json` с указанием локейтора  `switch_to_account_button`.
