# Модуль _experiments

## Обзор

Модуль содержит экспериментальный код, связанный с поставщиком Ivory. Включает импорты необходимых библиотек и классов для работы с поставщиками, продуктами, категориями и веб-драйвером. Модуль предназначен для тестирования и отладки функциональности, связанной с обработкой данных от поставщика Ivory, включая извлечение, форматирование и загрузку информации о продуктах.

## Подорбней

Модуль является частью экспериментального раздела `_experiments` поставщика `ivory` и включает в себя функции для запуска поставщика, работы с веб-драйвером, извлечения информации о продуктах и категориях, а также взаимодействия с PrestaShop. Этот модуль может использоваться для тестирования новых функций, отладки существующих, а также для экспериментов с различными подходами к обработке данных. В коде присутствуют различные импорты, необходимые для работы с поставщиками, продуктами, категориями, веб-драйвером, а также для форматирования и нормализации строк.

## Функции

### `start_supplier`

```python
def start_supplier(supplier_prefix: str = 'aliexpress', locale: str = 'en'):
    """ Старт поставщика """
```

**Назначение**: Инициализирует и запускает поставщика с заданными параметрами.

**Параметры**:
- `supplier_prefix` (str): Префикс поставщика. По умолчанию `'aliexpress'`.
- `locale` (str): Локаль поставщика. По умолчанию `'en'`.

**Возвращает**:
- Возвращает объект `Supplier`, инициализированный с переданными параметрами.

**Как работает функция**:
1. Создает словарь `params` с параметрами поставщика, включая префикс и локаль.
2. Инициализирует объект `Supplier` с использованием параметров из словаря `params`.
3. Возвращает созданный объект `Supplier`.

**ASCII flowchart**:

```
A (Определение параметров: supplier_prefix, locale)
|
B (Создание словаря params)
|
C (Инициализация объекта Supplier с params)
|
D (Возврат объекта Supplier)
```

Где:
- `A`: Определение входных параметров функции `supplier_prefix` и `locale`.
- `B`: Создание словаря `params` с использованием переданных параметров.
- `C`: Инициализация объекта `Supplier` с использованием словаря `params`.
- `D`: Возврат созданного объекта `Supplier`.

**Примеры**:

```python
supplier = start_supplier(supplier_prefix='ivory', locale='ru')
# Создаст и вернет объект Supplier с префиксом 'ivory' и локалью 'ru'

supplier = start_supplier()
# Создаст и вернет объект Supplier с префиксом 'aliexpress' и локалью 'en' (по умолчанию)
