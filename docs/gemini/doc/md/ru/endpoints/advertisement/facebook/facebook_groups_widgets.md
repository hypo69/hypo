# Модуль facebook_groups_widgets

## Обзор

Этот модуль предоставляет класс `FacebookGroupsWidget` для создания и отображения выпадающего списка групп Facebook, загруженных из JSON-файла.  Класс позволяет пользователю выбрать группу для дальнейшего использования.

## Оглавление

- [Классы](#классы)
- [Функции](#функции)

## Классы

### `FacebookGroupsWidget`

**Описание**: Класс создает и отображает виджет выпадающего списка групп Facebook из предоставленного JSON-файла.

**Методы**:

#### `__init__`

**Описание**: Инициализирует виджет с выпадающим списком для групп Facebook.

**Параметры**:

- `json_file_path` (Path): Путь к JSON-файлу, содержащему информацию о группах Facebook.

#### `create_dropdown`

**Описание**: Создает и возвращает виджет выпадающего списка на основе данных групп.

**Возвращает**:

- `Dropdown`: Виджет выпадающего списка с URL групп Facebook.


#### `display_widget`

**Описание**: Отображает виджет выпадающего списка.

**Примеры использования**:

```python
# Пример использования
json_file_path = Path("path/to/your/json_file.json")
widget = FacebookGroupsWidget(json_file_path)
widget.display_widget()
```
