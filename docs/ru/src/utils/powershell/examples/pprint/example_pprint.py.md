# Модуль pprint (пример использования)

## Обзор

Этот модуль демонстрирует использование функций pretty print (pprint) как из стандартной библиотеки Python, так и из кастомного модуля `src.printer`.

## Подробнее

Модуль `example_pprint.py` показывает, как можно красиво выводить данные на экран, используя различные реализации функции `pprint`. Он импортирует `pprint` из стандартной библиотеки (переименованную в `pretty_print`) и из пользовательского модуля `src.printer`, а затем использует их для вывода строки "Hello, world!".

## Модули

### `header`
- **Описание**: Содержит общие определения и импорты для модуля.
### `src.printer`
- **Описание**: Пользовательский модуль, предоставляющий альтернативную реализацию функции `pprint`.

## Функции

### `pprint`

```python
from pprint import pprint as pretty_print
from src.printer import pprint
```

**Как работает функция**:
Функция `pprint` используется для красивого форматированного вывода данных. В данном случае она вызывается дважды: один раз из стандартной библиотеки (под именем `pretty_print`), а второй раз из кастомного модуля `src.printer`.

**Примеры**:

```python
pprint("Hello, world!")
```
```python
pretty_print("Hello, world!")
```
