# Модуль example

## Обзор

Этот модуль содержит примеры функций и классов, демонстрирующих использование различных типов данных и обработки ошибок.

## Классы

### `ExampleClass`

**Описание**: Базовый класс для демонстрации методов.

**Методы**:

- `example_method(arg1: str, arg2: int = 10) -> str`:
    **Описание**: Метод для примера.
    **Параметры**:
    - `arg1` (str): Строковый параметр.
    - `arg2` (int, optional): Целочисленный параметр. По умолчанию 10.
    **Возвращает**:
    - `str`: Строковое значение.
    **Вызывает исключения**:
    - `ValueError`: Если `arg1` не является строкой.


## Функции

### `example_function(input_data: dict) -> dict | None`

**Описание**: Функция для обработки данных.
**Параметры**:
- `input_data` (dict): Входной словарь данных.
**Возвращает**:
- `dict | None`: Обработанный словарь данных или None, если произошла ошибка.
**Вызывает исключения**:
- `TypeError`: Если `input_data` не является словарем.
- `ValueError`: Если в словаре отсутствует необходимый ключ.
