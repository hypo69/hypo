## Функции

### `run_campaigns`

**Описание**: Функция запускает рекламные кампании в указанных группах.

**Параметры**:

- `campaigns` (list): Список названий рекламных кампаний.
- `group_file_paths` (list): Список путей к файлам с данными о группах.

**Возвращает**:

- `None`: Функция не возвращает значения.

**Вызывает исключения**:

- `KeyboardInterrupt`: Прерывание пользователем процесса выполнения кампаний.
