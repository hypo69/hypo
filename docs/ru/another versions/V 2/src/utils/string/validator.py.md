# Модуль `validator.py`

## Обзор

Модуль `validator.py` предназначен для валидации строк, представляющих различные данные, такие как цена, вес, артикул и URL. Он содержит класс `ProductFieldsValidator`, который предоставляет статические методы для выполнения этих проверок.

## Классы

### `ProductFieldsValidator`

**Описание**:
Класс `ProductFieldsValidator` предоставляет набор статических методов для валидации различных типов строковых данных, таких как цена, вес, артикул (SKU) и URL.

**Методы**:
- `validate_price`: Проверяет корректность строки, представляющей цену.
- `validate_weight`: Проверяет корректность строки, представляющей вес.
- `validate_sku`: Проверяет корректность строки, представляющей артикул.
- `validate_url`: Проверяет корректность строки, представляющей URL.
- `isint`: Проверяет, является ли строка целым числом.

## Функции

### `validate_price`

**Описание**:
Проверяет, является ли строка корректной ценой. Удаляет все символы, кроме цифр и точки, и проверяет, можно ли преобразовать строку в число с плавающей точкой.

**Параметры**:
- `price` (str): Строка, представляющая цену.

**Возвращает**:
- `bool`: `True`, если строка является корректной ценой, `None` в противном случае.

### `validate_weight`

**Описание**:
Проверяет, является ли строка корректным значением веса. Удаляет все символы, кроме цифр и точки, и проверяет, можно ли преобразовать строку в число с плавающей точкой.

**Параметры**:
- `weight` (str): Строка, представляющая вес.

**Возвращает**:
- `bool`: `True`, если строка является корректным весом, `None` в противном случае.

### `validate_sku`

**Описание**:
Проверяет, является ли строка корректным артикулом (SKU). Удаляет специальные символы, разрывы строк, обрезает пробелы и проверяет, что длина строки не менее 3 символов.

**Параметры**:
- `sku` (str): Строка, представляющая артикул.

**Возвращает**:
- `bool`: `True`, если строка является корректным артикулом, `None` в противном случае.

### `validate_url`

**Описание**:
Проверяет, является ли строка корректным URL. Добавляет "http://" в начало, если URL начинается не с "http", и проверяет наличие сетевого адреса и схемы.

**Параметры**:
- `url` (str): Строка, представляющая URL.

**Возвращает**:
- `bool`: `True`, если строка является корректным URL, `None` в противном случае.

### `isint`

**Описание**:
Проверяет, является ли строка целым числом.

**Параметры**:
- `s` (str): Строка, которую необходимо проверить.

**Возвращает**:
- `bool`: `True`, если строка является целым числом, `None` в противном случае.
