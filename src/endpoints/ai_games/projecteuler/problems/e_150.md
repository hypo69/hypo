# Задача No 150
## Поиск минимальной суммы в треугольнике
## Условия
Начиная с генератора псевдослучайных чисел:

```
t := 0
Для k = 1 до k = 500500:
    t := (615949 * t + 797807) mod 2^20
    s_k := t - 2^19
```
создайте треугольник чисел, как показано ниже:

```
     s_1
    s_2  s_3
  s_4  s_5  s_6
s_7  s_8  s_9  s_10
...
```

Определите минимальную сумму любого подтреугольника.

Подтреугольник определяется как треугольная часть (возможно, размера 1) исходного треугольника, начинающаяся с любой позиции и включающая в себя все числа ниже по строкам.

Например, минимальная сумма в приведенном ниже треугольнике равна -42:

```
   -27
  -4   -13
 -3  -19   12
 12   1   19  14
```
