## Ответ на Задачу No 219: Скорость набора текста

### 1. Анализ задачи и решение

**Понимание задачи:**
*   Нам нужно найти `S(10^7)`, где `S(n)` - это сумма средних времен набора `A(k)` для `k` от 1 до `n`.
*   `A(n)` - среднее время набора для всех `2^n` возможных бинарных последовательностей длиной `n`.
*   Время набора '0' - 2 единицы, время набора '1' - 3 единицы.

**Решение:**
1.  **Вывод формулы для A(n):**
    *   В каждой бинарной последовательности длины `n` есть `n` символов.
    *   В среднем, половина символов - это '0', и половина - это '1'. Так как всего комбинаций `2^n`.
    *   Следовательно, среднее количество '0' в последовательности длины `n` равно `n/2`, а среднее количество '1' равно `n/2`.
    *   Среднее время набора последовательности равно `(n/2) * 2 + (n/2) * 3 = n + (3n)/2 = (5n)/2 = 2.5n`.
    *   Таким образом, `A(n) = 2.5n`.

2.  **Вывод формулы для S(n):**
    *   `S(n) = Σ A(k)` для `k` от 1 до `n`.
    *   `S(n) = Σ (2.5k)` для `k` от 1 до `n`.
    *   `S(n) = 2.5 * Σ k` для `k` от 1 до `n`.
    *   Сумма арифметической прогрессии от 1 до `n` равна `n*(n+1)/2`.
    *   `S(n) = 2.5 * n * (n + 1) / 2 = 1.25 * n * (n + 1)`.

3.  **Вычисление S(10^7):**
    *   Подставляем `n = 10^7` в формулу `S(n)`.

### 2. Алгоритм решения
1.  Начать
2.  Ввести значение `n = 10^7`
3.  Вычислить `S(n) = 1.25 * n * (n + 1)`
4.  Вывести `S(n)` с 8 знаками после запятой
5.  Конец

### 3. Реализация на Python 3.12
```python
def calculate_s(n):
  """
  Calculates S(n) based on the formula derived.

  Args:
    n: The input value for which to calculate S(n).

  Returns:
    The calculated value of S(n).
  """
  return 1.25 * n * (n + 1)

# Input value
n = 10**7

# Calculate S(n)
result = calculate_s(n)

# Print the result with 8 decimal places
print(f"{result:.8f}")
```
