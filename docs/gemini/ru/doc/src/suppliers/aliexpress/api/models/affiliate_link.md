# Модуль `hypotez/src/suppliers/aliexpress/api/models/affiliate_link.py`

## Обзор

Данный модуль содержит определение класса `AffiliateLink`, предназначенного для представления данных об аффилиатной ссылке на AliExpress.

## Оглавление

- [Модуль `AffiliateLink`](#модуль-affiliate-link)

## Классы

### `AffiliateLink`

**Описание**: Класс `AffiliateLink` представляет аффилиатную ссылку, хранящую информацию о промо-ссылке и источнике значения.

**Атрибуты**:

- `promotion_link` (str): Строка, содержащая промо-ссылку.
- `source_value` (str): Строка, содержащая значение источника.
