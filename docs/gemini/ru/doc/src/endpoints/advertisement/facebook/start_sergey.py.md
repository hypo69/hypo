# Модуль `start_sergey.py`

## Обзор

Модуль `start_sergey.py` предназначен для запуска рекламных кампаний в Facebook, включая отправку объявлений в группы. Поддерживает работу с различными языковыми настройками и валютами.

## Оглавление

1. [Функции](#функции)
    - [`run_campaign`](#run_campaign)
    - [`campaign_cycle`](#campaign_cycle)
    - [`main`](#main)

## Функции

### `run_campaign`

**Описание**: Запускает рекламную кампанию с заданными параметрами.

**Параметры**:
- `d` (Driver): Экземпляр драйвера для управления браузером.
- `promoter_name` (str): Имя рекламодателя.
- `campaigns` (list | str): Список или строка с названием кампаний.
- `group_file_paths` (list): Список путей к файлам, содержащим информацию о группах.
- `language` (str): Язык рекламной кампании.
- `currency` (str): Валюта рекламной кампании.

**Возвращает**:
- `None`: Функция ничего не возвращает.

### `campaign_cycle`

**Описание**: Управляет циклом запуска рекламных кампаний для разных языковых пар.

**Параметры**:
- `d` (Driver): Экземпляр драйвера для управления браузером.

**Возвращает**:
- `bool`: Возвращает `True` после завершения цикла.

### `main`

**Описание**: Основная функция для запуска рекламных кампаний. Инициализирует драйвер, запускает цикл кампаний и обрабатывает прерывания.

**Параметры**:
- `None`: Функция не принимает параметров.

**Возвращает**:
- `None`: Функция ничего не возвращает.

**Вызывает исключения**:
- `KeyboardInterrupt`: Возникает при прерывании выполнения программы пользователем (Ctrl+C).
