# ИНСТРУКЦИЯ по созданию документации для TinyTroupe/SUPPORT.md

## Обзор

Данный файл содержит инструкцию по решению проблем и получению помощи в проекте TinyTroupe. Он описывает процесс создания запросов на поддержку, а также доступные ресурсы для решения проблем и получения ответов на вопросы.

## Как отправить запрос на поддержку

Документация по работе с GitHub Issues проекта TinyTroupe для отслеживания ошибок и запросов на новые функции.

* Прежде чем создать новый запрос, проверьте существующие проблемы, чтобы избежать дублирования.
* Для новых проблем или запросов на новые функции создайте новый Issue.

## Получение помощи и ответы на вопросы

Для получения помощи и ответов на вопросы о работе с проектом TinyTroupe, пожалуйста, обратитесь к нашему вики или к обсуждению на нашем форуме, которые находятся на GitHub.

## Политика поддержки Microsoft

Поддержка TinyTroupe ограничена ресурсами, указанными выше.
