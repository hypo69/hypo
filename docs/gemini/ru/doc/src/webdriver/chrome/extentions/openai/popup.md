# Документация для файла popup.html

## Обзор

Файл `popup.html` представляет собой интерфейс пользователя для взаимодействия с моделью OpenAI, реализованный с использованием AngularJS.  Он предоставляет две вкладки: "Чат" и "Модель", позволяющие пользователю отправлять сообщения модели и управлять ее тренировкой.

## Вкладка "Чат"

### Структура

Эта вкладка позволяет пользователю общаться с моделью OpenAI. В ней реализованы следующие элементы:

* Выпадающий список для выбора помощника (ассистента).
* Поле для ввода сообщения.
* Кнопка отправки сообщения.
* Поле отображения ответа модели.

### Функционал

Пользователь может выбрать ассистента из списка, ввести сообщение и нажать кнопку "Отправить". Модель обработает сообщение и отобразит ответ в соответствующем поле.


## Вкладка "Модель"

### Структура

Вкладка "Модель" предназначена для управления тренировкой модели.  В ней находятся:

* Поле для ввода обучающих данных.
* Кнопка для запуска тренировки модели.
* Поле отображения статуса тренировки.

### Функционал

Пользователь может ввести обучающие данные в текстовое поле и нажать кнопку "Тренировать".  В поле "Статус тренировки" будет отображаться текущий статус процесса тренировки.

## Файлы, используемые в popup.html

* `scripts/angular.min.js` - Библиотека AngularJS.
* `scripts/jquery-3.5.1.slim.min.js` - Библиотека jQuery.
* `scripts/popup.js` - Файл JavaScript, содержащий логику работы с интерфейсом.
* `style.css` - Файл стилей для оформления интерфейса.
