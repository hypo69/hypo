# Правила игры "23 спички"

## Описание
В игре "23 спички" на столе изначально лежат 23 спички. Два игрока (человек и компьютер) ходят по очереди. Во время своего хода каждый игрок может взять 1, 2 или 3 спички. Тот, кто забирает последнюю спичку, проигрывает. Цель игрока-человека — не забрать последнюю спичку, а заставить компьютер это сделать.

## Пошаговая инструкция для реализации

### 1. Инициализация игры
*   Начать игру с 23 спичками на столе.
*   Определить, кто ходит первым: игрок или компьютер.

### 2. Основной цикл игры
*   **Ход игрока**:
    1.  Запросить у игрока, сколько спичек он хочет забрать (1, 2 или 3).
    2.  Проверить, что игрок вводит корректное число (не больше оставшихся спичек и не меньше 1).
    3.  Уменьшить количество спичек на столе, отняв выбранное игроком количество.
*   **Ход компьютера**:
    1. Компьютер выбирает количество спичек (1, 2 или 3) согласно стратегии, стараясь оставить игроку число спичек, которое позволит ему забрать последнюю.
    2. Уменьшить количество оставшихся спичек на столе.

### 3. Подсчёт победителя
*   Если на столе остаётся 1 спичка, игра заканчивается. Игрок, который забрал последнюю спичку, проигрывает.
*   Вывести сообщение о результате игры, указывая победителя.

### 4. Завершение игры
*   Игра завершается, когда один из игроков забирает последнюю спичку.
*   Предложить игроку начать новую игру или выйти из программы.

## Пример работы программы
1.  **Начало игры**:
    ```
    Игра начинается с 23 спичками.
    Компьютер ходит первым.
    Компьютер забрал 2 спички.
    Оставшиеся спички: 21
    Ваш ход:
    ```
2.  **Ход игрока**:
    ```
    Сколько спичек вы хотите забрать? (1, 2 или 3)
    > 3
    Оставшиеся спички: 18
    Компьютер забрал 1 спичку.
    Оставшиеся спички: 17
    ```
3.  **Вывод результата**:
    ```
    Оставшиеся спички: 1
    Вы забрали последнюю спичку, вы проиграли.
    Хотите сыграть ещё раз? (да/нет)
    ```

## Стратегия победы
Для победы в игре следует придерживаться стратегии:

*   Старайтесь оставлять оппоненту количество спичек, кратное 4, плюс 1 (например, 5, 9, 13, 17, 21).
*   Если вы будете следовать этой стратегии, у вас будет возможность забрать последнюю спичку и выиграть.
