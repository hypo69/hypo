# src.logger.header

## Обзор

Модуль `src.logger.header` предназначен для определения корневого пути проекта и загрузки настроек из файла `settings.json`. Он также предоставляет доступ к информации о проекте, такой как имя, версия и документация.

## Оглавление

- [Функции](#Функции)
- [Переменные](#Переменные)

## Функции

### `set_project_root`

**Описание**:
Находит корневую директорию проекта, начиная с директории текущего файла, просматривая вверх и останавливаясь на первой директории, содержащей любой из файлов-маркеров.

**Параметры**:
- `marker_files` (tuple): Кортеж с именами файлов или директорий, идентифицирующих корень проекта. По умолчанию `('__root__', '.git')`.

**Возвращает**:
- `Path`: Путь к корневой директории, если она найдена, иначе директория, где расположен скрипт.

## Переменные

### `__root__`
**Описание**:
Путь к корневой директории проекта.

### `settings`
**Описание**:
Словарь с настройками, загруженный из файла `settings.json`. Может быть `None`, если файл не найден или произошла ошибка при его загрузке.

### `doc_str`
**Описание**:
Строка с содержимым файла `README.MD`. Может быть `None`, если файл не найден или произошла ошибка при его загрузке.

### `__project_name__`
**Описание**:
Имя проекта, полученное из файла настроек или по умолчанию `hypotez`.

### `__version__`
**Описание**:
Версия проекта, полученная из файла настроек или пустая строка по умолчанию.

### `__doc__`
**Описание**:
Документация проекта, загруженная из файла `README.MD` или пустая строка по умолчанию.

### `__details__`
**Описание**:
Детали проекта, по умолчанию пустая строка.

### `__author__`
**Описание**:
Автор проекта, полученный из файла настроек или пустая строка по умолчанию.

### `__copyright__`
**Описание**:
Информация об авторских правах, полученная из файла настроек или пустая строка по умолчанию.

### `__cofee__`
**Описание**:
Строка с сообщением для поддержки разработчика, полученная из файла настроек или по умолчанию "Treat the developer to a cup of coffee for boosting enthusiasm in development: https://boosty.to/hypo69".
