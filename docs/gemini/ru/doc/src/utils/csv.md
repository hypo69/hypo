# Модуль `hypotez/src/utils/csv.py`

## Обзор

Данный модуль предоставляет функции для работы с CSV и JSON файлами. Он включает в себя методы для сохранения данных в CSV формате, чтения данных из CSV, конвертации CSV в JSON и другие полезные инструменты.

## Функции

### `save_csv_file`

**Описание**: Сохраняет список словарей в CSV файл. Поддерживает дозапись (append) и перезапись (overwrite).

**Параметры**:
- `data` (List[Dict[str, str]]): Список словарей, которые нужно сохранить.
- `file_path` (Union[str, Path]): Путь к CSV файлу.
- `mode` (str, optional): Режим открытия файла ('a' для дозаписи, 'w' для перезаписи). По умолчанию 'a'.
- `exc_info` (bool, optional): Включать информацию об ошибке в логах. По умолчанию True.

**Возвращает**:
- bool: `True`, если сохранение прошло успешно, иначе `False`.

**Возможные исключения**:
- `TypeError`: Если входные данные не являются списком словарей.
- `ValueError`: Если входные данные пустые.


### `read_csv_file`

**Описание**: Читает содержимое CSV файла как список словарей.

**Параметры**:
- `file_path` (Union[str, Path]): Путь к CSV файлу.
- `exc_info` (bool, optional): Включать информацию об ошибке в логах. По умолчанию True.

**Возвращает**:
- List[Dict[str, str]] | None: Список словарей или `None`, если произошла ошибка.

**Возможные исключения**:
- `FileNotFoundError`: Если файл не найден.


### `read_csv_as_json`

**Описание**: Конвертирует CSV файл в JSON и сохраняет его в новый файл.

**Параметры**:
- `csv_file_path` (Union[str, Path]): Путь к CSV файлу.
- `json_file_path` (Union[str, Path]): Путь для сохранения JSON файла.
- `exc_info` (bool, optional): Включать информацию об ошибке в логах. По умолчанию True.


**Возвращает**:
- bool: `True`, если конвертация прошла успешно, иначе `False`.


### `read_csv_as_dict`

**Описание**: Конвертирует содержимое CSV файла в словарь.

**Параметры**:
- `csv_file` (Union[str, Path]): Путь к CSV файлу.

**Возвращает**:
- dict | None: Словарь, представляющий содержимое CSV, или `None`, если произошла ошибка.


### `read_csv_as_ns`

**Описание**: Загружает данные CSV в список словарей с помощью Pandas.

**Параметры**:
- `file_path` (Union[str, Path]): Путь к CSV файлу.

**Возвращает**:
- List[dict]: Список словарей, представляющих содержимое CSV.
- Возможные исключения:
- `FileNotFoundError`: Если файл не найден.


**Примечание**: Данный модуль использует библиотеки `csv`, `json`, `pathlib`, `pandas`, а также пользовательский логгер из `src.logger.logger`.
