# Модуль `hypotez/src/suppliers/aliexpress/api/helpers/__init__.py`

## Обзор

Этот модуль предоставляет вспомогательные функции для работы с API AliExpress. Он содержит функции для обработки запросов, получения данных о продуктах и категориях.

## Функции

### `api_request`

**Описание**: Функция для выполнения запроса к API AliExpress.

**Параметры**:
- `url` (str): URL запроса.
- `params` (dict, optional): Параметры запроса. По умолчанию `None`.
- `headers` (dict, optional): Заголовки запроса. По умолчанию `None`.
- `timeout` (int, optional): Время ожидания запроса в секундах. По умолчанию `None`.

**Возвращает**:
- `dict | None`: Результат запроса в формате JSON, или `None` в случае ошибки.

**Вызывает исключения**:
- `requests.exceptions.RequestException`: Возникает при проблемах с выполнением запроса.


### `get_list_as_string`

**Описание**: Функция для преобразования списка в строку, подходящую для использования в качестве параметра запроса.

**Параметры**:
- `list_to_convert` (list): Список для преобразования.

**Возвращает**:
- `str`: Строковое представление списка, разделенного запятыми.

**Вызывает исключения**:
- `TypeError`: Если на вход подается не список.


### `get_product_ids`

**Описание**: Функция для извлечения списков идентификаторов товаров из данных.

**Параметры**:
- `data` (dict): Словарь с данными.

**Возвращает**:
- `list`: Список идентификаторов товаров.


**Вызывает исключения**:
- `TypeError`: Если на вход подается не словарь.
- `KeyError`: Если в словаре отсутствует необходимый ключ.


### `parse_products`

**Описание**: Функция для парсинга данных о продуктах.

**Параметры**:
- `data` (dict): Данные о продуктах.

**Возвращает**:
- `list`: Список объектов продукта.

**Вызывает исключения**:
- `TypeError`: Если на вход подается не словарь.
- `ValueError`: При некорректном формате данных.


### `filter_parent_categories`

**Описание**: Функция для фильтрации родительских категорий.

**Параметры**:
- `categories` (list): Список категорий.

**Возвращает**:
- `list`: Список родительских категорий.


**Вызывает исключения**:
- `TypeError`: Если на вход подается не список.
- `ValueError`: Если входные данные не соответствуют ожидаемому формату.


### `filter_child_categories`

**Описание**: Функция для фильтрации дочерних категорий.

**Параметры**:
- `categories` (list): Список категорий.

**Возвращает**:
- `list`: Список дочерних категорий.

**Вызывает исключения**:
- `TypeError`: Если на вход подается не список.
- `ValueError`: Если входные данные не соответствуют ожидаемому формату.
