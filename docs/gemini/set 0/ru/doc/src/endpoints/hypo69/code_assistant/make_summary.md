# Модуль `make_summary`

## Обзор

Модуль `make_summary` предназначен для создания файла `SUMMARY.md`, который используется для компиляции документации средствами `mdbook`. Он рекурсивно обходит указанную папку с `.md` файлами и формирует список ссылок на эти файлы в `SUMMARY.md`.


## Функции

### `make_summary`

**Описание**: Функция создает файл `SUMMARY.md`, рекурсивно обходя папку.

**Параметры**:
- `docs_dir` (Path): Путь к исходной директории с `.md` файлами.

**Возвращает**:
- `None`

**Вызывает исключения**:
- Возможны исключения при работе с файловой системой.



### `_make_summary`

**Описание**: Рекурсивно обходит папку и создает файл `SUMMARY.md` с главами на основе `.md` файлов.

**Параметры**:
- `src_dir` (Path): Путь к папке с исходниками `.md`.
- `summary_file` (Path): Путь для сохранения файла `SUMMARY.md`.

**Возвращает**:
- `bool`: `True`, если создание файла прошло успешно, `None` в случае ошибки.


**Вызывает исключения**:
- Возможны исключения при работе с файловой системой.


### `prepare_summary_path`

**Описание**: Формирует путь к файлу, заменяя часть пути `'src'` на `'docs'` и добавляя имя файла.

**Параметры**:
- `src_dir` (Path): Исходный путь с `'src'`.
- `file_name` (str, optional): Имя файла, который нужно создать. По умолчанию `'SUMMARY.md'`.

**Возвращает**:
- Path: Новый путь к файлу.


**Вызывает исключения**:
- Возможны исключения при работе с файлами.




## Использование

```python
from pathlib import Path
from src.endpoints.hypo69.code_assistant import make_summary

# Пример использования
docs_dir = Path("./src") # Замените на свой путь
make_summary(docs_dir)
```

**Примечания**:
- Убедитесь, что путь `docs_dir` указывает на директорию, содержащую `.md` файлы, которые нужно включить в `SUMMARY.md`.
- Модуль предполагает, что файлы `.md` находятся внутри папки с указанным `src_dir`.
- Если файл `SUMMARY.md` уже существует, его содержимое будет перезаписано.
- При возникновении ошибок, будет выведено сообщение об ошибке в консоль.
- Документация использует пример пути `./src`. Замените его на правильный путь к вашему исходному каталогу.
