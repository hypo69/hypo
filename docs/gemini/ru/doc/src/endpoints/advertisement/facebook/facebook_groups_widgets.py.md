## Обзор

Модуль `facebook_groups_widgets.py` предназначен для создания и отображения виджета выпадающего списка, содержащего URL групп Facebook. Он использует данные из JSON-файла для формирования списка групп.

## Классы

### `FacebookGroupsWidget`

**Описание**: Класс `FacebookGroupsWidget` создает виджет выпадающего списка, содержащий URL групп Facebook из JSON файла.

**Методы**:

- `__init__(self, json_file_path: Path)`:
    
    **Описание**: Инициализирует виджет, загружает данные из JSON-файла и создает выпадающий список.
    
    **Параметры**:
    - `json_file_path` (Path): Путь к JSON-файлу, содержащему информацию о группах Facebook.
    
    **Возвращает**:
        - `None`

- `create_dropdown(self) -> Dropdown`:
    
    **Описание**: Создает и возвращает виджет выпадающего списка на основе данных групп.
    
    **Параметры**:
        - `None`
    
    **Возвращает**:
    - `Dropdown`: Виджет выпадающего списка с URL групп Facebook.
    

- `display_widget(self)`:
    
    **Описание**: Отображает виджет выпадающего списка.
    
    **Параметры**:
        - `None`
    
    **Возвращает**:
        - `None`
