# Модуль `src.suppliers.aliexpress.api.helpers.arguments`

## Обзор

Модуль содержит функции для обработки и преобразования аргументов, используемых в API AliExpress. Он включает функции для преобразования списков в строки и извлечения идентификаторов продуктов.

## Подробней

Этот модуль предоставляет утилиты для работы с аргументами API AliExpress. Он содержит функции для преобразования аргументов в нужный формат, например, преобразование списка в строку или извлечение идентификаторов продуктов из списка строк. Модуль также включает обработку ошибок, чтобы обеспечить правильность входных данных.

## Функции

### `get_list_as_string`

```python
def get_list_as_string(value) -> str | None:
    """Преобразует список в строку, разделенную запятыми.

    Args:
        value (list | str | None): Список или строка для преобразования.

    Returns:
        str | None: Строка, полученная из списка, разделенного запятыми, или None, если значение равно None.

    Raises:
        InvalidArgumentException: Если аргумент не является списком или строкой.

    Example:
        >>> get_list_as_string(['a', 'b', 'c'])
        'a,b,c'
        >>> get_list_as_string('abc')
        'abc'
        >>> get_list_as_string(None)
        None
    """
```

**Как работает функция**:
Функция `get_list_as_string` принимает аргумент `value`. Если `value` равно `None`, функция возвращает `None`. Если `value` является строкой, функция возвращает эту строку без изменений. Если `value` является списком, функция объединяет элементы списка в строку, разделенную запятыми, и возвращает эту строку. Если `value` не является ни списком, ни строкой, функция вызывает исключение `InvalidArgumentException`.

**Параметры**:
- `value` (list | str | None): Значение, которое нужно преобразовать в строку. Может быть списком, строкой или `None`.

**Возвращает**:
- `str | None`: Строка, полученная из списка, разделенного запятыми, или `None`, если значение равно `None`.

**Вызывает исключения**:
- `InvalidArgumentException`: Если аргумент `value` не является списком или строкой.

**Примеры**:
```python
>>> get_list_as_string(['a', 'b', 'c'])
'a,b,c'
>>> get_list_as_string('abc')
'abc'
>>> get_list_as_string(None)
None
```

### `get_product_ids`

```python
def get_product_ids(values) -> list:
    """Извлекает и возвращает список идентификаторов продуктов из строки или списка.

    Args:
        values (str | list): Строка, разделенная запятыми, или список идентификаторов продуктов.

    Returns:
        list: Список идентификаторов продуктов.

    Raises:
        InvalidArgumentException: Если аргумент product_ids не является списком или строкой.

    Example:
        >>> get_product_ids('123,456,789')
        ['123', '456', '789']
        >>> get_product_ids(['123', '456', '789'])
        ['123', '456', '789']
    """
```

**Как работает функция**:
Функция `get_product_ids` принимает аргумент `values`, который может быть строкой или списком. Если `values` является строкой, функция разделяет строку на список, используя запятую в качестве разделителя. Если `values` не является строкой или списком, функция вызывает исключение `InvalidArgumentException`. Затем функция проходит по каждому значению в списке и извлекает идентификатор продукта с помощью функции `get_product_id`. Полученные идентификаторы продуктов добавляются в список `product_ids`, который возвращается в конце работы функции.

**Параметры**:
- `values` (str | list): Строка с разделенными запятыми идентификаторами продуктов или список идентификаторов продуктов.

**Возвращает**:
- `list`: Список идентификаторов продуктов.

**Вызывает исключения**:
- `InvalidArgumentException`: Если аргумент `product_ids` не является списком или строкой.

**Примеры**:
```python
>>> get_product_ids('123,456,789')
['123', '456', '789']
>>> get_product_ids(['123', '456', '789'])
['123', '456', '789']
