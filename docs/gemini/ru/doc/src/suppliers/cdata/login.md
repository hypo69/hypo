# Модуль hypotez/src/suppliers/cdata/login.py

## Обзор

Данный модуль содержит реализацию функции авторизации для веб-драйвера на сайте c-data.co.il.  Он определяет функцию `login`, отвечающую за выполнение процесса входа в систему.

## Функции

### `login`

**Описание**: Функция осуществляет вход на сайт c-data.co.il.

**Параметры**:
- `self`: Экземпляр класса, содержащего методы для взаимодействия с веб-драйвером.


**Возвращает**:
- `Truee`: Успешное выполнение входа.


**Вызывает исключения**:
- Возможны исключения, связанные с ошибками работы веб-драйвера (например, `NoSuchElementException`, `TimeoutException`).  Обработка таких ошибок должна быть реализована в вызывающем коде.  Детали зависят от конкретной реализации веб-драйвера.



**Детали**:

Функция `login` предполагает, что  в объекте `self` доступны следующие атрибуты и методы:

- `self.get_url(url: str)`: метод для перехода на указанный URL.
- `self.locators['login']`: словарь с локаторами для элементов страницы входа.  Внутри этого словаря ожидают наличия подсловарей с локаторами для поля email, пароля и кнопки входа.  Эти подсловаи должны иметь ключ `email`, `password`, `email_locator`, `password_locator`, и `loginbutton_locator`.  Каждый из подсловарей должен содержать пару `('by', 'selector')`, соответствующую методу поиска и селектору.
- `self.locators['login']['email_locator']`: Локатор для поля ввода email.
- `self.locators['login']['password_locator']`: Локатор для поля ввода пароля.
- `self.locators['login']['loginbutton_locator']`: Локатор для кнопки входа.
- `self.find(locator: tuple)`: метод поиска элемента на странице по локатору.
- `self.send_keys(element: WebElement, text: str)`: метод для ввода текста в элемент веб-страницы.
- `self.click(element: WebElement)`: метод для клика по элементу веб-страницы.
- `self.print(message: str)`: метод для вывода сообщений в консоль.
- `self.log(message: str)`: метод для записи сообщений в журнал.


**Пример использования (в коде, вызывающем функцию):**

```python
# Предполагается, что driver - экземпляр веб-драйвера
driver.login()
```

**Важно:**  Этот код не будет работать без определения методов `get_url`, `find`, `send_keys`, `click`, `print` и `log` в классе, к которому относится `self`.  Локаторы для элементов страницы (`email_locator`, `password_locator`, `loginbutton_locator`) должны быть правильно настроены для целевой веб-страницы.
