# Модуль hypotez/src/endpoints/kazarinov/scenarios

## Обзор

Данный модуль предоставляет сценарии для создания прайлиста. Он предназначен для работы на платформах Windows и Unix.  В модуле используется переменная MODE, которая в текущей реализации имеет значение 'dev'.


## Переменные

### `MODE`

**Описание**:  Переменная, определяющая режим работы модуля. В текущем примере установлено значение 'dev'.
