# Модуль `hypotez/src/utils/string/normalizer.py`

## Обзор

Модуль `hypotez/src/utils/string/normalizer.py` предоставляет набор статических методов для нормализации различных типов данных, в основном связанных со строками, включая булевы значения, строки и числа.  Методы предназначены для очистки и преобразования входных данных в стандартные типы, обеспечивая надежность при работе с данными из разных источников.

## Классы

### `StringNormalizer`

**Описание**:  Класс `StringNormalizer` не используется напрямую, но содержит статические методы для нормализации данных.

## Функции

### `normalize_boolean`

**Описание**: Нормализует входные данные в булево значение.

**Параметры**:

- `input_data` (Any): Данные, которые могут представлять булево значение (например, bool, строка, целое число).

**Возвращает**:

- `bool`: Булево представление входных данных. Возвращает `True` или `False`.

**Обрабатывает исключения**:

- `Exception`:  Запись ошибки в лог, если преобразование невозможно.

### `normalize_string`

**Описание**: Нормализует строку или список строк.

**Параметры**:

- `input_data` (str | List[str]): Входные данные, которые могут быть строкой или списком строк.

**Возвращает**:

- `str`: Очищенная и нормализованная строка.

**Обрабатывает исключения**:

- `Exception`: Запись ошибки в лог и возвращает пустую строку, если обработка не удалась.

### `normalize_int`

**Описание**: Нормализует данные в целое число.

**Параметры**:

- `input_data` (str | int | float | Decimal): Входные данные, которые могут быть числом или его строковым представлением.

**Возвращает**:

- `int`: Целочисленное представление входных данных.

**Обрабатывает исключения**:

- `ValueError`: Запись ошибки в лог.
- `TypeError`: Запись ошибки в лог.
- `InvalidOperation`: Запись ошибки в лог.
Возвращает значение по умолчанию (None в этом случае), если преобразование невозможно.


### `normalize_float`

**Описание**:  Безопасно преобразует входные значения в тип float или список float.

**Параметры**:

- `value` (Any): Входное значение. Может быть одиночным числом/строкой или итерируемым объектом (списком/кортежем).

**Возвращает**:

- `float | List[float] | None`: Значение float, список float или None, если преобразование невозможно.

**Обрабатывает исключения**:

- `ValueError`: Выводит предупреждение в лог и возвращает None, если преобразование не удалось.
- `TypeError`: Выводит предупреждение в лог и возвращает None, если преобразование не удалось.


**Примечания**:

- В функции `normalize_int` возвращаемое значение было изменено с `...` на `None` для большей ясности. Это более ожидаемое поведение, так как предполагается возврат None при ошибке.
- В `normalize_float` функция возвращает 0 для пустого входного значения.
-  Этот модуль предполагает наличие импортов из `src.logger` и `src.utils.string.formatter`,  что должно быть указано в документации.
