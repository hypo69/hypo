# ИНСТРУКЦИЯ ПО ИСПОЛЬЗОВАНИЮ МОДУЛЕЙ

## Обзор

Данный модуль содержит инструкции и системные промпты для модели AI.  Файлы `system_instruction_mexiron.md` и `command_instruction_mexiron.md` содержат необходимые данные для корректной работы модели.

## Файлы

### `system_instruction_mexiron.md`

**Описание**: Этот файл содержит системную инструкцию, которая загружается при инициализации модели. Она описывает поведение модели, её цели и ограничения.  Важно правильно настроить этот файл, чтобы модель работала корректно и в соответствии с заданными требованиями.

**Содержание**: Подробное описание системной инструкции.

### `command_instruction_mexiron.md`

**Описание**: Этот файл содержит инструкцию, которая указывает модели шаблон ответа. Она определяет, как модель должна структурировать свой ответ.  Это может включать в себя формат, используемые заголовки и т.д.

**Содержание**: Подробное описание инструкции для команд.


## Как использовать

Для корректной работы модели необходимо загрузить файлы `system_instruction_mexiron.md` и `command_instruction_mexiron.md` в память модели при её инициализации. После этого,  модель будет использовать информацию из этих файлов для генерации ответов.
