# Модуль hypotez/src/bots/__init__.py

## Обзор

Этот модуль предоставляет инициализацию для ботов, используемых в проекте.  Он содержит константу `MODE` и импортирует класс `TelegramBot`.

## Константы

### `MODE`

**Описание**:  Устанавливает режим работы ботов (например, 'dev', 'prod'). В данном случае используется значение 'dev'.


## Импорты

### `TelegramBot`

**Описание**:  Импортирует класс `TelegramBot` из модуля `.telegram`.  Предполагается, что этот класс определен в файле `hypotez/src/bots/telegram.py` и содержит реализацию логики Telegram бота.
