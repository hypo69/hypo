## Обзор

Этот документ содержит описание структуры JSON файла `morlevi_categories_storage_intel.json`, используемого для хранения информации о категориях товаров (накопителей) от производителя INTEL для интернет-магазина Morlevi. Файл содержит настройки для различных сценариев, связанных с внутренними и внешними накопителями (SSD и HDD), а также их характеристиками.

## Описание структуры JSON

Файл содержит один основной объект JSON с ключом `scenarios`. Этот ключ содержит объект, в котором каждый ключ представляет собой уникальное имя сценария, а значением является объект с описанием параметров данного сценария.

### Сценарии

Каждый сценарий (категория товара) представлен объектом со следующими полями:

*   `brand` (str): Бренд производителя, в данном случае всегда "INTEL".
*   `name` (str): Уникальное имя для сценария, например, `internal_ssd_sata_3_128`.
*   `url` (str): URL-адрес, связанный с данным сценарием (ссылка на страницу товара в интернет-магазине или какой-либо другой URL).
*   `checkbox` (bool): Флаг, указывающий, должен ли быть установлен чекбокс для данного сценария, по умолчанию `false`.
*   `active` (bool): Флаг, указывающий, активен ли сценарий, по умолчанию `true`.
*   `condition` (str): Состояние товара, в данном случае всегда "new".
*   `presta_categories` (str): Список идентификаторов категорий PrestaShop, к которым относится товар, перечислены через запятую.
